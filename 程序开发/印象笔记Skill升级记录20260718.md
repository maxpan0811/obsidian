# 印象笔记 Skill 升级记录

> 序列号：`20260718-182250`

## 摘要

用户看到印象笔记公众号文章（2026-07-18），要求核实本地 `evernote-yinxiang` skill 是否有新版。核实发现官方 v1.0.4（7月15日打包）对比本地 v1.0.3（7月8日）有大量新增能力，包括更新笔记、创建标签、创建笔记本、高级搜索等。用户确认升级，手动下载官方 zip 覆盖升级。

## 发现流程

1. 用户转发的印象笔记公众号文章宣称新版 Skill 新增"更新笔记、创建标签、创建笔记本、按时间/标签/笔记本搜索"等能力
2. 本地 `evernote-yinxiang` skill 版本 1.1.0（_meta.json 中为 1.0.3），官方最新为 v1.0.4（7月15日打包）
3. `references/api-commands.md` 从 3.2KB 暴涨至 24KB
4. 新增 3 个 API：`updateNoteFromMCP`、`createTagFromMCP`、`createNotebookFromMCP`

## 官方升级包信息

- 下载地址：`https://cdn.yinxiang.com/ai/yinxiang-skill-v1.0.4.zip`
- 官方文档：`https://guide.yinxiang.com/docs/yxai/skill.html`
- 打包时间：2026-07-15

## 新增能力清单

| 能力 | API 端点 | 说明 |
|------|---------|------|
| **更新笔记** | `updateNoteFromMCP` | 改标题/正文/笔记本/标签 |
| **创建标签** | `createTagFromMCP` | 创建独立标签 |
| **创建笔记本** | `createNotebookFromMCP` | 支持批量创建 |
| **搜索增强** | `searchNotesByFilter` | keyword + tagNames + notebookGuid + startTime/endTime + title 组合过滤 |
| **创建笔记增强** | `createNoteFromMCP` | 支持 `tagNames` 参数，一键创建带标签的笔记 |
| **最近3天搜索** | — | 用户说"最近"固定按 3 天筛选 |
| **批量操作** | — | 批量创建笔记本、批量移动笔记、批量管理标签 |

## 升级操作

1. **先备份**：`cp -r ~/.claude/skills/evernote-yinxiang ~/.Trash/evernote-yinxiang.bak`
2. **下载官方 zip**：从 `https://cdn.yinxiang.com/ai/yinxiang-skill-v1.0.4.zip` 下载到 `/tmp/`
3. **覆盖升级** 14 个文件，逐个 `cp -f` 覆盖：
   - `SKILL.md`（22KB, 场景从 7→10）
   - `references/api-commands.md`（24KB, bash+PowerShell 双平台）
   - 11 个 shell 脚本（新增 `create-notebook.sh`、`create-tag.sh`、`update-note.sh`、`search-notes.sh` 增强版）
   - `save-token.sh`、`_common.sh`
4. **全量校验**：14 个文件字节数与官方 zip 一致 ✅
5. **清理**：备份移入 Trash

## SKILL.md 关键变化
