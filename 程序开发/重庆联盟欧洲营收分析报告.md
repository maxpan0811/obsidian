### 问题与修复过程

1. **v1（子串匹配）**：宽泛子串匹配 → 36 条全部错配到"新南路门市"
2. **v2（品牌过滤）**：标记非携程系跳过 → 漏配严重，匹配率从 36→10%
3. **v3（复合关键词）**：4-8 字地点复合词匹配 → 太严，几乎无匹配
4. **v4（子串重叠）**：3-8 字全枚举重叠 → 匹配质量大幅提升，但仍有误配（人和春天→东和春天、沙坪正街→杨家坪正街）
5. **v5（唯一个关键词交集）**：剔除通用词，只比唯一个地点关键词 → 正确匹配率最高，误配几乎为零

### 最终采用的匹配方法

```
提取订单系统门店名和联盟条目的独有地点关键词（3-6字，剔除通用词）
→ 取交集 → 按长度×2 + 数量×10 加权评分
→ 阈值 score ≥ 20 → Top 60% → 去重输出 Top 3
```
