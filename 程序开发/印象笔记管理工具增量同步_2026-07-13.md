## 技术细节

### 管线选择

- 弃用：`sync_favorites.py`（Thrift API 云端管线）— step_list/reconcile 已被注释"use local_sync.py instead"
- 当前：`local_sync.py` — 直接从 Evernote 本地 SQLite 读取，无需 API 调用
- 优势：17 秒处理 47k 笔记，零网络依赖，支持原子写入（_pending/ → atomic rename）

### 核心文件

| 文件 | 作用 |
|------|------|
| local_sync.py | 主管线：读本地 SQLite → 对比 DB → 导出 → 清理 |
| local_db.py | local_state.db 的 SQLite 状态管理 |
| export_note.py | 单篇笔记导出（Thrift API，已不常用） |
| verify.py | SHA256 验证 |
| local_state.db | 跟踪已导出笔记的 GUID/哈希/时间戳 |
