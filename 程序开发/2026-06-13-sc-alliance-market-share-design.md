# 四川省四大联盟华程双品牌预订市占率分析工具

## 概述

分析四川省四大门店联盟（叮叮旅游、周一联盟、北斗联盟、悠途联盟）在携程欧洲渠道的**预订口径**双品牌市占率。以上一个完整自然周为目标周期，按门店维度展示双品牌（华程 + 国旅自营）营收占比，带环比（上周）和同比（去年同周）对比。输出为单 Excel 文件，含联盟汇总和按联盟展开的门店明细。

### 门店清单（内置）

脚本内置四大联盟的门店清单字典，每条记录含：`{联盟名, 品牌, 门店短名, 对接销售}`。

初始数据来源：现有的四个联盟分析文件（`周一联盟门店分析20260609.xlsx`、`叮叮旅游门店分析20260609.xlsx`、`北斗联盟门店分析20260609.xlsx`、`悠途联盟门店分析20260609.xlsx`）中 Sheet `2026年*联盟` 的所有门店行。

### 门店名匹配规则

联盟清单中为短名（如 `武侯区航空路门市`），订单 Col 31 中为全名（如 `成都携程武侯区航空路门市`）。匹配方式：短名作为子串查找（`short_name in full_name`）。

特殊映射（硬编码在脚本中）：
- `名山县门市` → `雅安携程名山区名山门市`

无订单的联盟门店仍列出，标注 0 订单。

### 品牌分类

| 品牌 | 规则（Col 60 订单归属） |
|------|------------------------|
| **华程** | 含"华程"（广州/上海/北京/成都/武汉/厦门/其他华程订单）或含"账号订单" |
| **国旅自营** | = "国旅订单" |
| **双品牌合计** | 华程 ∪ 国旅自营 |
| **双品牌占比** | (`华程收入` + `国旅收入`) / 该门店本周期总营收 |

## 时间周期

### 目标周计算

自动计算：**上一个完整的自然周（周一 00:00 ~ 周日 23:59）**。

- 如 2026-06-13（周六）运行 → 目标周 = 2026-06-08（周一）~ 2026-06-14（周日）
- Python 逻辑：`datetime.now()` → 上周一的 00:00:00 和上周日的 23:59:59

### 环比周期

目标周的上一周（大上周一 ~ 大上周日）。

### 同比周期

去年同周期编号（ISO 周编号相同），如今年第 23 周 → 2025 年第 23 周同一日期范围。

### 文件选择

- 2026 年数据 → 从 2026 年订单文件读取
- 2025 年数据 → 从 2025 年订单文件读取（短信不存在返回 None 而非报错）

### 脚本流程图

```
1. 配置参数加载
   ├── 时间计算（目标周/环比周/同比周起止日期）
   └── 文件检测（找最新2026/2025订单文件）

2. 订单数据加载
   ├── 读取2026年订单文件（筛选欧洲 + 目标周 + 环比周）
   └── 读取2025年订单文件（筛选欧洲 + 同比周）

3. 数据聚合
   ├── 按门店名匹配 → 分配到联盟
   ├── 计算每家门店：华程收入、国旅收入、总营收
   ├── 计算 TOP3 供应商及占比
   └── 汇总联盟级数据

4. Excel 输出
   ├── Sheet1: 联盟汇总对比
   ├── Sheet2-5: 各联盟门店明细
   ├── 条件格式应用
   └── 文件保存到 DATA_DIR

5. 打印摘要
   └── 控制台输出关键结果
```

## 输出格式规范

同现有其他 Excel 报表的一致标准：

- 数值：千分号 `#,##0`，无小数位
- 百分数：`0.00%`

## 错误处理

| 场景 | 行为 |
|------|------|
| 找不到 2026 订单文件 | 报错退出，打印 DATA_DIR 下文件列表 |
| 找不到 2025 订单文件 | 打印警告，同比列填 "-" |
| 目标周跨年 | 周编号规则使用 ISO 周号 |
| 某联盟本周期 0 订单 | 门店明细 Sheet 仍生成，显示 "本周无订单" |
| 门店名匹配不到任何订单 | 该门店行显示 0，标注无数据 |

## 限制与后续优化

- 当前仅支持**欧洲**业务板块，如需其他板块修改 `REGION_FILTER`
- 门店清单手动维护，推荐季度更新
- 同比需要 2025 年文件存在，若跨年（2027 年分析 2026 年同比）需调整文件匹配逻辑
