# 微信转发到 Codex 收件箱 · 2026-07-14

**系统序列号**：`019f56b2-620a-7d43-9679-da800d34bab6`

## 背景

保存公众号文章到印象笔记时，需要手动复制链接 → 切到 Codex 粘贴。想实现微信文章直接转发到 Codex 自动保存。

## 限制

微信分享菜单基于系统 Share Sheet。Codex/Claude Code 都是终端工具，没有注册 Share Extension，不会出现在分享菜单里。

## 架构方案

最终方案：**iOS Shortcut → iCloud Drive → Mac 端脚本处理**

```
iPhone WeChat → Share Sheet → Shortcut("转发到 Codex")
  → 追加到 iCloud Drive/CodexInbox/wechat-articles.md
  → iCloud 同步到 Mac → process_wechat_inbox.py
  → CDP 保存 → 印象笔记工作篮
```

### 优势

- 零延迟（iCloud Drive 实时同步）
- 零服务器（纯 iCloud 中转）
- 零轮询（用户触发处理）
- 不占用额外存储（只传 URL，非全文）

### 被排除的方案

| 方案 | 排除原因 |
|------|----------|
| 邮件中转 | 需轮询 IMAP，有 1-5 分钟延迟 |
| 飞书机器人 | 微信不能直接转发到飞书 |
| Get 笔记中转 | 文章存两份，数据冗余 |
| 微信读书剪藏 | 非所有文章可读 |

## 组件

### 1. iPhone Shortcut「转发到 Codex」

4 个操作：
1. 共享表单（接收 URL）
2. 获取 URL（提取地址）
3. 储存文件（追加到 iCloud Drive/CodexInbox/wechat-articles.md）
4. 显示通知（"✅ 已转发到 Codex"）

### 2. iCloud Drive 收件箱

文件位置：`~/Library/Mobile Documents/com~apple~CloudDocs/CodexInbox/wechat-articles.md`
iPhone Shortcut 写入 → iCloud 同步 → Mac 可见

### 3. Mac 端处理脚本

`~/.claude/skills/印象笔记管理工具/scripts/process_wechat_inbox.py`

支持命令：
- `python3 process_wechat_inbox.py` — 处理全部待处理
- `--dry-run` — 预览
- `--status` — 只看状态

### 4. Codex 触发

在会话中说「处理微信收件箱」触发处理流程。

## 使用流程

1. 微信打开公众号文章 → 右上角 ⋯ → 分享
2. 滚动找到「转发到 Codex」并点击
3. 看到通知「✅ 已转发到 Codex」
4. 下次在 Codex 中说「处理微信收件箱」
5. 文章自动保存到印象笔记工作篮
