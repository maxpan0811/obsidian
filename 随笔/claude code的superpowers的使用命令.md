---
调用方式：在对话框中输入 /<command> 即可激活对应的 skill。例如 /brainstorming 启动头脑风暴流程。

这些技能默认按 Process first（头脑风暴/调试/计划） → Implementation second（TDD/SDD等） 的优先级顺序使用。不确定用哪个时，用 /brainstorming 开始总没错。
