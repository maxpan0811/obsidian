# 上海中考相关细节

上海中考现行总分 **750分**，共7门计分科目，2025年政策与2024年保持一致（即延续2022年改革后的750分方案）。

## 各科满分明细

|科目|满分|考试形式|
|---|---|---|
|语文|**150**​|闭卷笔试，100分钟|
|数学|**150**​|闭卷笔试，100分钟|
|外语|**150**​|笔试140（含听力25）+ 听说测试10，人机对话|
|综合测试|**150**​|闭卷笔试135（物理70+化学50+跨学科案例分析15）+ 理化实验15|
|道德与法治|**60**​|开卷笔试30 + 日常考核30|
|历史|**60**​|开卷笔试30 + 日常考核30|
|体育与健身|**30**​|统一测试15 + 日常考核15|

> 💡 几个容易混的点：
> 
> - 综合测试的"跨学科案例分析"是15分单独一块（一般是地理+生物交叉），不算在物化生里
>     
> - 理化实验可以考2次取较好成绩，物理实验10分、化学实验5分
>     
> - 道法、历史都是**开卷**，但30分来自日常考核（平时表现+作业等），不是全靠那场40分钟笔试
>     
> - 外语听说测试、理化实验在5月中旬考，其余笔试在6月14-15日
>
