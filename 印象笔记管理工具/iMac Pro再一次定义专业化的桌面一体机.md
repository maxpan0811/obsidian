# iMac Pro再一次定义专业化的桌面一体机

---

![](.evernote-content/92EC6E5B-68E7-4924-8E1D-BF34CB5100D8/D8CFAD44-3505-4682-A237-9ABB81522B15.png)

过去几年大多数时间里，很多果粉曾抱怨苹果对 Mac 的关心确实不够，尤其是 iMac。毕竟每次在网上搜索苹果的 Mac 电脑哪一款适合桌面使用时，基本上所有搜索结果给出的答案都是 iMac 一体机电脑。因为 iMac 不仅美观占空间少，而且性能上已经足够日常家用。然而，苹果只是对 iMac 简单的更新配置，并没有考虑其新的方向，就拿去年款 iMac 来说，虽然有 4K 或 5K 分辨率屏幕，但性能上应付如此高的分辨率都有些困难。

因此，对于苹果的下一代 iMac 桌面电脑，很多人非常好奇，究竟苹果今后的方向在哪？考虑到太久没有发生十分显著的变化了，iMac 是否真的要做些什么，以吸引更多一些特定的人群呢？没错！在如此之多的一问之下，iMac Pro 诞生了。

**迄今最强大的 Mac**

就在本届 WWDC 2017 大会上，原本我们很多人以为，苹果可能还只是按照预期更新一下 iMac 产品线。但最意想不到的事情是，苹果竟然还让大家提前目睹了令人难以置信的新款 iMac Pro 产品。重点是，用苹果官方的宣传来说，iMac Pro 是迄今为止最性能最强大的 Mac，重新定义了 Mac 的强大。

![](.evernote-content/92EC6E5B-68E7-4924-8E1D-BF34CB5100D8/5D08B486-565A-40AB-9ABA-D27A7749C1AB.jpg)

对于普通的 iMac，苹果为其更新了 500 尼特可显示 10 亿色彩的 P3 显示屏，换上英特尔 Kaby Lake 微架构的第七代酷睿处理器，标配 Fusion Drive 和 Thunderbolt 3 接口，还升级了最新的 Radeon Pro 500 系列显卡。在这一系列升级下，顶配 27 英寸至少提升了 50% 性能，超高分辨率屏下应付多个 4K 视频编辑或者游戏已经不是问题。

，作为桌面电脑兴许还是有很多专业用户不太满意，他们太期待 Mac Pro 级别的电脑，所以苹果通过 iMac Pro 一次性解决问题。或许可以这么说，iMac Pro 填补了自  2013 年以来  Mac Pro 没有更新的空白，它无论在那一方面都比普通 iMac 更加出色，而且性能也超过了 Mac Pro。

你可以看到 iMac Pro 拥有比普通 iMac 更强大的处理器（Xeon），还有大容量的专业内存（ECC 128GB），更高端的显卡（Radeon Pro Vega），而这些正是大家去年一直希望 Mac Pro 能够更新的内容。如今 iMac Pro  比 Mac Pro 更据吸引力，相比老款 Mac Pro 最高可选的 12 核老架构  Xeon 处理器，全新 iMac Pro 可选更高 18 核新架构的 Xeon 处理器。至于图形运算性能，老 Mac Pro 原本只是 7 Teraflops 的 FirePro 显卡，但新 iMac Pro 的 Radeon Pro Vega 半精度浮点性能最高达到了 22 Teraflops。

![](.evernote-content/92EC6E5B-68E7-4924-8E1D-BF34CB5100D8/5FB29D7B-C962-4EDA-8436-2B35CEEE3D3B.jpg)

换句话说，苹果用迄今为止最强大的 Mac 来宣传 iMac Pro 并没有问题，甚至有些保守，因为按照 AMD 在 5 月底 Computex 2017 电脑展上公布 Vega 专业显卡数据，在 64 个计算单位，4096 个流处理器 和 16GB HBM2 的组合下，显存带宽可达 480GB/s，半精度浮点性能可以达到 25 Teraflops 。当然了，那是在水冷散热辅助下发挥的峰值性能，而 iMac Pro 仅通过双风扇、大面积的散热片和通风构造，能做到  22 Teraflops 和  400GB/s 带宽已经十分不错了。

![](.evernote-content/92EC6E5B-68E7-4924-8E1D-BF34CB5100D8/24968F2C-B655-4B6A-BD4C-FFF613749C9E.jpg)

**iMac Pro 让 Windows 阵营又慢了一步**

那么，与微软标杆级的一体机 Surface Studio 相比如何呢？去年微软秋季发布会首次发布的这款产品，当时被人为是让 iMac 无地自容仅剩屏幕的最强一体机，因为微软宣称他们将“一体机工作站”转化为了性能强劲的“数字画布”，同时还有强大的英特尔 Core i7 处理器和 Nvidia GeForce GTX 980M 显卡作为性能基础。

作为一体机来说，Surface Studio 这样的配置已经足够豪华，足以满足大量设计师、绘图师、漫画家、工程师各类内容创作者和专业人群。不过，在 iMac Pro 面前 Surface Studio 在性能上还是落败了，至强的定位已经在酷睿之上，更不用已经落后 GTX 1000 系列一代的 980M，完全无法像 iMac Pro 那样驱动两个 5K60Hz 的显示器。关键是，iMac Pro 所搭载是桌面级的 Vega 架构 GPU 核心，11 Teraflops 的单精度浮点运算性能已匹敌 GTX 1080 Ti  显卡（11.5 Teraflops）。

![](.evernote-content/92EC6E5B-68E7-4924-8E1D-BF34CB5100D8/5375081F-4689-44D9-9A64-4AD322D2FB4A.jpg)

不只是性能，Surface Studio 在行业目标布局上也明显比 iMac Pro 更有局限性。Surface Studio “零重力铰链”设计配合全球最薄 LCD 触控屏而生的“工作室模式”，再加上完美融合的输入设备 Surface Dial 和 Surface Pen 触控笔，其定位已经十分鲜明，更多还是专注于在桌面上绘画和设计的内容创作者或专业人士。

对比之下，iMac Pro 仅惊人的 GPU 配置，就能够满足目前大量专业人士所涉及到的图形处理工作，不只是 3D 渲染和更逼真的特效游戏表现，还包括即将在行业引领新风潮的高帧率的 VR 和机器学习任务。相比定义“数字画布”的 Surface Studio 而言，iMac Pro 才真正是“工作站级的一体机”，再加上强大的多核多线程 CPU、可扩展的 Thunderbolt 3 接口和 10Gb 太网接口等，无论是 4K 视频剪辑师、3D 动画师、音乐人、VR 软件开发者还是科研人员，iMac Pro 都能满足需求。

![](.evernote-content/92EC6E5B-68E7-4924-8E1D-BF34CB5100D8/708BA859-A8AA-4D7F-9EB6-8DC98C60988A.jpg)

更重要的是，苹果对一体机的紧凑设计又提升到了一个新的高端。因为将如此之强大多个硬件如何浓缩到一体机之中是一大工程设计问题，在 Windows 阵营很多强大的一体机要么只有至强 E3 级别的处理器，要么就是在塞进 GTX 1080 显卡之后，屏幕和底座部分显得十分突兀。但 iMac Pro 依然保持了标准 iMac 大小的设计，高度还是 51.6 厘米，宽度也是 65 厘米，而且重量也只是从 9.44kg 增加到了 9.7kg 而已。

在 Windows 阵营为了实现高性能和依然保持轻薄的工作站，目前也找到了一些解决方案，那就是模块化设计。例如联想的 ThinkCentre 微型多功能模块化一体机 Tiny-in-One，允许客户通过简单地更换内藏主要部件的模块，轻松做到设备升级换代，无需支出一比巨大的费用购买全新的一体机。不过，大家所看到造型如下图：

![](.evernote-content/92EC6E5B-68E7-4924-8E1D-BF34CB5100D8/0ECC96FD-C2EF-4C26-AC94-AA3B957F7F48.jpg)

很显然，顺应潮流可能不是苹果最想做的事情，反而是引领潮流，苹果更希望提供自家更成熟、更出色的方案保持“少即是美”的设计语言。

**那为何会有 iMac Pro 产品线？目标专业领域！**

其实去年这个问题就有答案了，当时苹果宣布 Mac 电脑的获得多家顶级企业的热捧，尤其是蓝色巨人 IBM。IBM 称 Mac 是最划算、最省成本的产品，并且称未来还会大量投资 Mac 电脑。因此毫无疑问，Mac 还在不断被更多的企业和机构认可，并将其作为标配商务办公设备。

![](.evernote-content/92EC6E5B-68E7-4924-8E1D-BF34CB5100D8/079DC5B8-9EFA-45EB-BC32-5CCD03294BFD.jpg)

从那一刻起，苹果 Mac 业务相当于迎来巨大的机遇，尤其是 iMac 桌面电脑产品线，苹果需要借此寻求新的手段以吸引和满足更多企业用户，挽回之前抱怨 Mac Pro 的专业用户，并满足那些大量等待采购苹果 Mac 设备的特定用户群，而配置更强大的 iMac Pro 自然应运而生。

此外，近年来苹果似乎十分热衷于将自家的产品线全面转向“专业级”定位的产品，以此推动自家产品线的吸引力，尤其是 iPad 产品线中多了 iPad Pro 这样的机型。因此，如今处理器更换为英特尔至强系列处理器和  AMD Vega Pro 专业显卡的 iMac Pro，所做的工作同样如此，更有助于吸引专业的消费者人群，因为此类人群对“专业”硬件和软件往往更加敏感。

![](.evernote-content/92EC6E5B-68E7-4924-8E1D-BF34CB5100D8/A69DE942-E690-4F7A-BF95-2CDF1AD8FB21.jpg)

当然了，iMac Pro 是可以预想中的事情。今年早些时候，苹果全球营销副总裁菲尔·席勒曾透露过一些 Mac 的未来规划，他声称苹果依然致力于为需求严苛的专业用户提供最高端、性能最佳的 Mac 桌面产品，未来的 Mac 会“充分考虑专业用户体验”。同时，他也证实了全新一代专业化的 iMac 一体机（即 iMac Pro）今年晚些时候正式上市。

**iMac Pro 更让人期待 Mac Pro**

很多人担心 Mac Pro 定位难保，事实上无需担心，iMac Pro 是一次惊人的升级，但其实他与 Mac Pro 是两种不同的产物。只要再拿菲尔·席勒透露的 Mac 未来规划就能说清楚事实了，他坦言 Mac Pro 的定位依然会是 Mac 阵营中最高端的桌面系统。这就表示，Mac Pro 还会拥有比 iMac Pro 更强大的规格，例如不局限于 4TB 的储存和有限的接口扩展，GPU 采用多核配置也是可以预想的事情。

按照席勒的说法，明年 Mac Pro 可能与酷无关，而是更专业，为此他还提到了模块化架构设计的解决方案，以此满足专业用户对于最高端及高吞吐量系统的需求。这同样也说明了，明年登场的 Mac Pro 拥有比 iMac Pro 更多的扩展性、兼容性灵活性和更高的可升级性，而不是像上面我们看到的 Tiny-in-One 那样在一体机上玩出突兀的模块化外观。简而言之，iMac Pro 可以保持很酷，但 Mac Pro 要做的是如何的伟大。

总之，苹果官方在预览 iMac Pro 时明确 iMac Pro 是目前最强的 Mac，但 Mac Pro 不会放弃，并证实了公司内部还在研发一款彻底重新设计的新一代 Mac Pro。另外，苹果确认还有一款新的高端专业显示器也在研发中，对此我们可以继续保持期待。
