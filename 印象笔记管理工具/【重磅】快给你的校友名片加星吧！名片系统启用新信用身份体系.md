【重磅】快给你的校友名片加星吧！名片系统启用新信用身份体系
-----------------------------

*2014-09-25* *清华校友总会* [清华校友总会](#)

日前，清华校友名片系统迎来重大升级，进行了全新改版。​

为提升“校友名片”的便捷性和可靠性，进一步改善用户体验，增强互动性和趣味性，增强活跃度与用户粘性，名片系统正式启用新的信用认证体系：由原先的学号自动认证模式，转变为“人工星级认证”模式。​

新的认证模式增加了对“校友回馈”、“校友服务”等维度的考量，为校友的信用身份提供新维度的认证支持，也为校友之间的名片换牌提供不同角度的参考。这种校友信用身份的差异性，将通过不同的“星级”来体现（如图）。同时，新的认证模式下，2-3星级用户的审核，将采用人工方式进行，进一步提升了系统的安全性。​

![](.evernote-content/88BAAB46-852A-4ECD-8B44-733AAB754B33/64219BBB-CE62-47DB-8D08-AA311170079F.jpg) ​

星级含义及权限请参加下表：​

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **用户角色**​ | **星级显示**​ | **名片搜索浏览**​ | **名片详情**​ | **加星操作**​ |
| 未注册用户/访客​ | 无星​ | 姓\*\*，年级院系​ | 不可见​ | 无法点亮​ |
| 名片系统注册用户​ | 1颗星​ | 姓\*\*，年级院系​ | 交换后可见​ | 自动点亮​ |
| **认证校友**​ | **2****颗星**​ | **姓名，年级院系**​ | **交换后可见**​ | **人工审核**​ |
| **星级校友**​ | **3****颗星**​ | **姓名，年级院系**​ | **交换后可见**​ | **人工审核**​ |

​

目前，名片加星工作已正式启动，现面向三类校友提供“审核加星”服务。​

**一是为校友服务的校友总会工作人员、地方校友会工作人员；二是捐赠校友；三是校友期刊订阅校友。**​

请符合条件的校友，将您的**“姓名+手机+校友工作信息/捐赠项目信息/期刊订阅信息”**在此微信公号留言给我们，或者发送邮件到：**school@tsinghua.org.cn****，并请注明“加星申请”****。**我们将会对您的加星申请进行审核并执行相关操作。​

此外，本次名片升级还对“互助汇”、“校友活动”等模块进行了优化，欢迎大家体验。让我们拉近全球各地清华校友之间的距离，一起构建天涯若比邻的沟通体验。​
