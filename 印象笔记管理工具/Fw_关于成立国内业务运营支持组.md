关于成立国内业务运营支持组

各位好：

      为解放产品经理生产力，提高对出发地销售单元的响应速度及解决出发地对大交通的需求，我统筹成立国内业务运营支持组，成员如下。

 

|  |  |  |
| --- | --- | --- |
| 国内业务运营支持组 | | |
| 区域 | 产品支持 | 票务支持 |
| 北京 | 王玮 | 潘莹 |
| 上海 | 徐晓涵 | 王庆（郎霄） |
| 广州 | 李艳 | 麦燕飞 |
| 武汉 |  | 王莺 |
| 成都 | 黄文婷 | 黄梅 |

 

产品支持的主要工作内容如下：

1、  协助销售单元向项目组反馈销售区域市场产品需求。

2、  推动销售区域完成当地出发产品竞争力分析。

3、  协助项目组完成销售区域出发地产品的设计。

4、  协助项目组在销售区域进行产品培训。

 

票务支持的主要工作内容如下：

1、  负责当地项目组票务数据整理工作。

2、  协助其它项目组在所在销售区域出发机位申请及登记，与产品经理确定付定人数，ERP中建立机票产品。

3、  负责所在销售区域出发机位订位出票，跟进财务付款，确认开票人数进名单、票价税金核对、反馈航司水单等。

 

 

近期各项目组有比较多的大交通需求，项目组所在地票务统计项目组整体数据，操作上请项目组直接联系出发地所在票务，正式需求除了发给出发地票务，请抄送项目组所在地票务。（举例：三亚项目组所属北京公司，三亚项目组对成都—三亚机票需求直接发给黄梅，抄送潘莹）。
