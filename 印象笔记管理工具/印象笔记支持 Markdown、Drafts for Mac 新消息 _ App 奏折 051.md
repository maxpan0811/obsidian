Drafts for Mac 新消息
------------------

**@Hum：**上星期（8 月 17 日）Drafts 的开发者在 [论坛](https://forums.getdrafts.com/t/drafts-for-mac-status-update/2276) 释出了 macOS 版的开发进度[视频](https://vimeo.com/285326696)。

![](.evernote-content/1891C432-69D0-4F31-810F-176C06FA3818/804FF45E-47C9-4AC1-992C-E9DDA08E6E92.png)

Drafts for macOS 界面

从视频上看，开发进度已经非常完整了：

* 三栏布局
* 支持 inbox、Archive、Trash 的文档分类
* 支持标签、工作区（Workspace）
* 支持搜索
* 支持语法高亮
* 支持暗夜模式

根据作者在帖子中的描述，Drafts for Mac 不会贸然使用苹果提供的 Marzipan 工具来做 iOS to macOS 的简单移植，而是会看看这种方法是否能提供更加合适和原生的体验。Drafts 团队将以体验为移植考量的主要标准，并尽量在 2018 年末发布。

在价格方面，Drafts for Mac 并非单独售价，而是和 Drafts for iOS 使用同样的销售策略：

* 免费版本可以用基础功能；
* 进阶功能和 iOS 使用同一个订阅，Drafts 称其为 Drafts Pro。

#### **Drafts for Mac 的定位**

Drafts 的发迹是由于它的两个特点：

1. 启动速度快，打开即写。
2. 乘上了 URL Schemes 的风潮。

这两个特点造就了它的独一无二，也让它具备了话题性，并在初期在 iOS 上积累一波不算大的狂热人群。在之后的每次更新，都是在保全基础功能易用程度不受影响的前提下，不断更新进阶功能。

但是这两个特点似乎在 macOS 上发挥不了太大的作用：

1. 打开即写其它 App 不难做到，只是因为想加广告或者其它考量。加上 macOS 上不少比较工具都有直接开写的快捷键，所以这一点不再是优势。
2. URL Schemes 主要用于突破 iOS 沙盒限制传递分发内容，macOS 上这个功能似乎用处不大，这边复制那边粘贴操作成本也没那么高。

所以我对 Drafts for Mac 并没有过高的期待。希望在 Mac 上用上 Drafts，主要是因为 iOS 上对 Drafts 依赖程度太高，Mac 上没有一个可以直接处理 iOS 上 Drafts 内容的工具。对于我来说，只要 Drafts for Mac 能把同步做好，和 iOS 上的 Drafts 无缝衔接，我其实不求它有什么新的特别亮眼的功能。

印象笔记 Mac 版支持 Markdown
---------------------

![](.evernote-content/1891C432-69D0-4F31-810F-176C06FA3818/CBB46B5A-55F4-4867-A734-9AA50C15B776.png)

**印象笔记 for Mac 8.2.0 Beta**

[下载链接](https://www.yinxiang.com/download/)

#### **应用简介**

自今年 6 月印象笔记与 Evernote 独立后，印象笔记已经成为一款完全独立的国产应用，拥有 Markdown、密码锁等专为中国用户设计的功能。

#### **支持 Markdown、数学公式、流程图和甘特图**

**@Minja：**Mac 版印象笔记终于支持 **Markdown** 了，这次不是洋葱新闻！

印象笔记的 Markdown 模式采用了双栏布局，一侧写、一侧预览，想在印象笔记里写 Markdown 的用户不必在借助马克飞象等第三方工具。

![](.evernote-content/1891C432-69D0-4F31-810F-176C06FA3818/D556B7DE-177B-4250-A2DA-8301B40BDCB2.png)

印象笔记支持 Markdown

不过，单是基础的 Markdown 语法，还不算特别厉害。这次 Beta 真正让人兴奋的是支持了丰富的扩展语法，**包括图表、数学公式、流程图、时序图和甘特图**。

![](.evernote-content/1891C432-69D0-4F31-810F-176C06FA3818/A9FF60D8-3116-4DDE-BF9F-BCAE45937736.png)

拓展语法

用 Markdown 来画图是一件很酷的事情。画图本身就很酷，可以把原来唠唠叨叨一大段话清晰扼要地展示出来；而用 Markdown 语法来画图就更酷了，几行简单的代码就能变成图、表，不用打开庞大的绘图软件。最近和 Hum 一起做写作课的 PPT，就经常要配柱状图、饼图等等图表来表示一些抽象关系，如果印象笔记早一些推出 Beta 版，说不定一些简单的图表就可以直接在写作工具里完成了。

![](.evernote-content/1891C432-69D0-4F31-810F-176C06FA3818/EC47C331-E18B-468C-BC8B-DDA82EB802F3.png)

流程图的语法和效果

在印象笔记之前，主要有 MWeb、Typora 等比较 Geek 风格的 Markdown 编辑器支持拓展语法，印象笔记后来居上，一下子把常见图表的语法一网打尽，可以说是非常让人意外的。

印象笔记独立后，我曾经担心过它会往社交化发展，但从目前的更新情况来看，它仍然是一款专注打磨核心功能的笔记工具。

![](.evernote-content/1891C432-69D0-4F31-810F-176C06FA3818/74EF2C76-722B-4CA7-99B1-AA2288FE20E4.png)

**GIF Brewery 3.9（2018–08–21）**

[下载链接](https://itunes.apple.com/hk/app/gif-brewery-3-by-gfycat/id1081413713?l=en&mt=12)

#### **应用简介**

GIF Brewery 是 Mac 上一款全能的 GIF 制作工具，可以一站式解决 GIF 录制、转换、编辑和压缩的需求。GIF Brewery 被 Gfycat 收购后就可以免费使用，不喜欢折腾其他工具的话，直接装一个 GIF Brewery 是很省心的选择。

#### **支持添加 PNG 和 GIF 格式图层**

**@Minja：**动图的表现力非常强大。不管是斗图时甩几个动态表情包，还是在文章里插几个操作演示的动图，都能够很好地表情达意。现在做动图的工具那么多，动图谁都能做，但想要做得比别人好，就需求一些技巧了。这次 GIF Brewery 更新的两个功能就不得不提：

1. 支持加入 PNG 图层
2. 支持加入 GIF 图层

可能有人觉得，图层功能不就是自定义贴纸嘛，有什么好写的？下面我就介绍两个关于图层的实用技巧。

**利用 PNG 图层背景透明的特性，可以给动图加个性水印。**以前我加水印的方法比较笨，是用 Workflow 一帧一帧加上去、再拼成一张 GIF，后来手机上的 [GIF Toaster](https://sspai.com/post/42583) 让我告别了原始的操作，这次 GIF Brewery 更新让我在电脑上加水印也更轻松了。

![](.evernote-content/1891C432-69D0-4F31-810F-176C06FA3818/CE83A8FE-FCCF-464B-A508-D3C7DAEB435C.gif)

用 PNG 图片自定义水印

**GIF 图层则可以拿来给动图加标注。**标注静态图片很轻松，比如像告诉读者「点击这里」，那加个箭头就好了，但是动图呢？当然也可以用 GIF Brewery 加个带箭头图案的图层，但是我希望最后的效果更生动一些，比如下面这个会动的手指：

![](.evernote-content/1891C432-69D0-4F31-810F-176C06FA3818/4DE0E7BE-3415-4E0F-9EF2-6D6D93C1E1C7.gif)

标注动图中的元素

上面这就是 GIF Brewery 图层功能的两个应用场景，很简单，但是可以让你的动图看起来更更棒。

Trello 4.5：支持表情回复
-----------------

![](.evernote-content/1891C432-69D0-4F31-810F-176C06FA3818/75038E3C-B598-4F69-8363-5CEFCC744980.png)

**Trello 4.5（2018–08–24）**

[下载链接](https://itunes.apple.com/cn/app/trello/id461504587?mt=8&ign-mpt=uo%3D4)

#### **应用简介**

Trello 是一款在线协作服务，常用在团队之间作为项目管理工具。它最大的特点是将「看板」的概念融进工具中：每项任务都是一张卡片，卡片之上的层级是列表，多个列表则组成一个看板。

#### **值得注意的更新内容**

**@文刀漢三：**Trello 在新版加入了表情回复的功能，我们可以用 Emoji 表情对评论进行快速回复。

![](.evernote-content/1891C432-69D0-4F31-810F-176C06FA3818/0C56A94A-C0CC-4701-B095-A0A15AAE122D.jpg)

Trello 中的表情回复

比较早采用了这项功能并将之发扬光大的软件应该是 Slack。表情回复在聊天中的作用很明显，比如很多人会在收到消息之后回复「收到」以告知对方自己已经收到并阅读了消息，但其实不是所有人都喜欢收到「收到」的回复，因为这条消息往往会给对方弹一条通知。比如老板在群内说了一声「下午 2 点开会」，如果全公司的人都回复「收到」，那么老板就会收到一大堆「收到」，此外也会造成刷屏。

像这种「告知对方我已收到 / 已读」的消息，我们可以使用表情回复来代替，既不会对接收方造成干扰，也能留下已读的印记。

![](.evernote-content/1891C432-69D0-4F31-810F-176C06FA3818/8D7A3121-CFB4-4737-9446-90FC76536B14.png)

下午 5 点开会

而且 Slack 做得更厉害的是，允许用户上传自定义表情，比如我们编辑部就制作了一系列审稿的表情来进行快速标记。

![](.evernote-content/1891C432-69D0-4F31-810F-176C06FA3818/AB375697-2F8F-48E9-A65F-D08C1E4BA955.png)

审稿用的表情

说回 Trello，它并不支持上传自定义表情，而且在桌面版中支持的表情数量非常少，所以这项功能对 Trello 的帮助也可能非常有限。

Day One 3.0：全新编辑器和暗色模式
----------------------

![](.evernote-content/1891C432-69D0-4F31-810F-176C06FA3818/56E47AFB-2445-446B-A11F-350C31902C3C.png)

**Day One 3.0（2018–08–22）**

[下载链接](https://itunes.apple.com/cn/app/day-one-journal/id1044867788?mt=8&at=10lJSw)

#### **应用简介**

Day One 是一款以简洁美观著称的日记应用，在同类应用中长期霸占榜首。除了最基本的用文字记录生活，还可以通过插入图片的方式快速添加当天的位置、天气等信息。良好的用户体验收获了大批好评。

#### **值得注意的更新内容**

**@文刀漢三：**这次 Day One 3.0 更新最引起我注意的功能其实是暗色模式，因为我回想了一下自己写日记的时间点，往往都是在临睡前，周围灯光都处于比较昏暗的状态。我们从 @Abe 的《[深色模式对视疲劳和效率的影响](https://sspai.com/article/45713?series_id=9)》可以了解到：在昏暗环境下，保持适宜亮度设置的深色模式能降低视疲劳程度，但需要避免打开白色背景的网页、文件。

而 Day One 以及它的使用场景，恰恰好就符合了昏暗环境 + 白色背景，会加重我们的视疲劳。因此在这种场景下，暗色模式显得尤为重要。

另一个引起我注意的新功能是录音功能。这让我想起了不少影视剧中的情景，男主角手拿着录音笔，一句一句地记录下了当时的心态，用录音作为自己生活的记录。

我们的手机已经能记录非常多信息了，文字自不用说，图片视频也拍得越来越多，唯独我们比较少尝试的是单独记录下自己的声音。当然我并不觉得语音日记会成为我之后常用的功能，但目前来看也是增加日记趣味性的新功能之一，也许我偶尔也会试一试用语音记录下当时的心情。
