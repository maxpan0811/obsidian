想当年，我们还在犹豫，到底是选北大好，还是选清华好？要不要弃北清，就留在上海？长大后，才发觉完全想多了。在经过幼升小，小升初，中考，层层选拔，过五关斩六将，千辛万苦进入重点中学后，才发觉要考入北清复交，路途还是很遥远。   
  
2016年北清复交的详细数据全部公布了。同时，我们也收集了上海各高中的具体人数。每个学校考取北清复交的比率就一目了然。那么上海孩子考取顶级大学的概率有多大呢？

1

  

考入北清有多难

![](.evernote-content/3ECD39A9-BBE0-4671-9047-E227D0B7B10A/8110B8D5-42C1-4E75-9D48-57E50DA92D56.png)  
  
上面是各高中考入北清的百分比排名。表里的人数，以2013年各校招入的学生为基数。没有排除各校在三年里，陆续出国留学的人数。各顶级中学出国的人数差异并不太大，不影响排名。  
  
从表里可以看出，前2名的高中，包揽了60%的清北名额，与后面的高中远远拉开了差距。而前6名高中包揽了83%的清北人数。进入前6名的高中，是冲击北清的最好方式。  
  
从第6名起，拼清华，都有点拼彩票校的感觉，需要的是孩子超常的发挥和突破极限的努力，以及爆表的人品。奇迹也会发生，只是概率低了点。  
  
那么整个上海幼儿入学的数量在2017年将会达到35万。进入这6个顶级中学的行列，概率为0.64%。进入清北的整体概率是0.059%。真是路漫漫其修远兮。

1

  

考入北清复交的难度

![](.evernote-content/3ECD39A9-BBE0-4671-9047-E227D0B7B10A/46B182BE-CF6E-424A-91F7-F0EBBFB0DE4D.jpg)

这是上海前13名高中的北清复交录取数据。录取比率超过了10%。如果目标是北清与复交，那么进入前13名的高中，就有了过10%的录取概率。  
  
这13个高中每年的总招生人数约为：4000人。以2017年幼儿的入学数作为基础，能进入这13个高中的概率是1.13%。而要考上北清复交的概率是：0.45%。

![](.evernote-content/3ECD39A9-BBE0-4671-9047-E227D0B7B10A/91C05297-506F-4309-87A0-F2299194EA5A.gif)

![](.evernote-content/3ECD39A9-BBE0-4671-9047-E227D0B7B10A/D282475D-F510-47D9-A9D4-670D7BC9BC30.png)

有5%以上的概率考入北清复交的有26个学校。也就是进了这26个学校，概率可以刷刷冲到5%。

![](.evernote-content/3ECD39A9-BBE0-4671-9047-E227D0B7B10A/91C05297-506F-4309-87A0-F2299194EA5A.gif)

![](.evernote-content/3ECD39A9-BBE0-4671-9047-E227D0B7B10A/97E2E3FB-7415-4916-96AD-37502037AB52.jpg)

不管怎样，想要进入北清复交，上海的学生必须进入前50名的高中。否刚按理论数据得出的0.45%的概率，直接就变为无限接近于零。只能眼睁睁看着它越走越远，越走越远。

4

  

这些高概率的初中

根据每个初中考入以上这些高中的具体数字。大致估算了一下上海TOP初中最终入北清复交的概率。大家可以在择校过程中，作为参考，增大去北清复交的机会。

![](.evernote-content/3ECD39A9-BBE0-4671-9047-E227D0B7B10A/58A5879E-27EC-4DF4-831B-A47FF74DF6FF.png)  
  
前8名的顶级初中，作为一梯队，可以有10%的概率进入北清复交。而二梯度，就一下子降了很多。为了加大进入北清复交的概率，每年顶层孩子们小升初的拼抢尤为激烈。

5

  

艰难的幼升小

![](.evernote-content/3ECD39A9-BBE0-4671-9047-E227D0B7B10A/ED96186E-B90E-49B1-81AD-300838C3E471.png)

从小升初与中考的数据，综合算出来各小学能够考入各层次大学的比例。  
  
顶级小学的北清复交率大致都在10%以上，如上外附小，逸夫、盛大、明珠、七外，福外等。这些学校的人数大约在3000人左右。 顶级的小学与一二梯队比起来，就不如初中及高中那样差距大。  
  
正如马拉松比赛，越到后面，人群越少，冲到终点时，就成了寥寥数人。幼升小只要进入二梯队，希望都会非常大。  
  
对于不同梯队的小学，父母的战略计划可以按以下方向进行。

|  |  |
| --- | --- |
| 级别 | 战略计划 |
| 顶级小学 | 跟着大部队，抢占前排，向着北清复交挺进吧，梦想就在前方！ |
| 一梯队 | 如果努力奋进，也能进北清复交。稍一松懈，连上大也难保。 |
| 二梯队 | 目标就是上大。如有更多追求，就看娃与父母对自身环境的超越。 |
| 三梯队 | 虽然学校整体不好，但重在个人。只要把目光放远，目标定高，奋发图强，逆袭的机会也不少。 |
| 菜场 | 须有十二分的努力，保持在全校前几名，目标是一本。如最终把北清复交揽入怀中，绝对可以写一篇励志文。一切事在人为！ |

![](.evernote-content/3ECD39A9-BBE0-4671-9047-E227D0B7B10A/91C05297-506F-4309-87A0-F2299194EA5A.gif)

这就是上海孩子们需要踏上的拼抢北清复交的荆棘路途。有一句：“不想是否能够成功，只需要风雨兼程。不去想身后会不会袭来寒风冷雨，既然目标是地平线。”  
祝愿孩子们在追求梦想的过程中，披荆斩棘，笑在山巅！
