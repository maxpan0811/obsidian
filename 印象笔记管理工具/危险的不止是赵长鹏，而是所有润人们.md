# 危险的不止是赵长鹏，而是所有润人们

当年艾跃进教授说，“你们这些富人存进美国的钱是取不出来的”，还引来网上一片嘲讽，但凡辩解，张口就是一句“私有财产神圣不可侵犯”。

所以你们真没搞清楚吗？当年鸦片战争事情，鬼子为什么拿着枪炮贩毒，难道在西方文明里，他们不懂吸食鸦片是堕落罪行？人家那英伦强盗自己穿上三件套的西装装绅士，还忘不了给你贴个野蛮人的标，这一下子品种都划分开了，干啥不要脸的事都没有心理负担了。

所以私有财产当然神圣不可侵犯，但你确认他们有把你当成同一物种吗？

你不过是野蛮人，属于打死爆金币的那种，还还不往死里打！

上个月，美国司法部刚割了赵长鹏和币安40亿美元，还有人很傻很天真的跑去给美帝辩白，说罚款是好事，40个亿买个平安，不算贵，我特别在《[好狠！华人前首富被美国一把割了](http://mp.weixin.qq.com/s?__biz=MzUyNzk4MTQ2NA==&mid=2247501674&idx=1&sn=c5ba6e3200ed8db7d795048bd8ae9cab&chksm=fa75c99dcd02408be13b23776ea8ef0b1ad2a97a64ae63d8ea96b00fd23b33d5fe986de45845&scene=21#wechat_redirect)》里说，40亿不过是第一刀，后面刀法且有呢。

没想到老美在搞钱方面效率这么高，不到一个月，又让赵老割交了28.5亿美元，你以为这就结束了？呵呵，这才到哪啊。

之前币安被迫交出了自己的客户名单，名单交易记录都被老美拿走，想想看，美帝这是打算干啥呢？

听说国内币圈富豪们都集中迁居新加坡，大约一个个还以为自己带着钱跑到海外了，实现了财富自由，那叫一个嘚瑟啊，可是亲们，你们不怕被老美顺着网线一个个寻摸出来，一起打劫了吗？

坡县那个小破地方，可没有胆量跟老美叫板，到时候一纸公文，老美就能直接全球追索你的资产。你的财富飞向了自由的阿美莉卡，这叫财富自由，可跟你没有半毛钱的关系了。

别说我危言耸听，钓鱼执法这一手，美国人玩得很溜的，就是论起法制精神，人家也能不亏，虚拟币交易从来没有合法过，至于管辖权这件事嘛，只要你赚的钱够多，抓起来勒索油水足够，那么全球范围内搜罗走你的财产，无非就是找个合适的借口罢了，别忘了，人家那里三权分立，立法也就是国会山开个会就能解决的事，不要太简单。

想想看，这波赵长鹏是以洗钱罪被美国给抓起来的，比特币从诞生没多久以后，最重要的用途就是洗钱，这特么就是原罪，之前一直装作没事人一样，默许甚至纵容你们洗钱，转移财富，一旦发现这个口子被封上了，没办法从中国大陆搞出更多的财富，很好，赵长鹏就被送上被告席，罪名就是洗钱。

笑死了，洗钱这玩意，跨国银行们都在干，平时不叫个事，关键时候就拿它说事，你就说你有没有洗钱，有吧，那么治你的罪是正义之举吧，吃了哑巴亏你还不能说啥。

最好笑的是润人们，没什么钱润到美国去，其实问题不大。

最近这段时间我还天天在抖音上看丁胖子金牌讲师呢，这哥们走线走到美国，去了就开始整治华人圈的职场文化，让华裔老板们提起黑工就瑟瑟发抖，整完了，哥们躺平了要饭，坚决不给美国贡献GDP。

像这种人，在美国没有大肆参与零元购，就已经算是对得起美利坚了，老美也不太可能出手收拾他，最多就给你遣送回国。

只要起点够低，直接躺平了，老美还真只能把你当大爷供着。

再有一种，润去干个白领工作，有学历，有技能，勤勤恳恳，老老实实，刚好做美国牛马，还是最主要的纳税人群体，就这种美国政府又没失心疯，干嘛去动你？真捏着鼻子动了你，一大票类似的润人就要物伤其类了，大家都不润了，美帝可就要找不到干活的了。

真正该害怕的是那些拿着从国内赚来的大量的钱，通过各种手段润去美国的“精英”润人们。

他们为什么会被盯上？不是因为曾经干过什么坏事，或者有悖于资本主义价值观。

只是因为他们够有钱。

这让我想起很多年前，我一个朋友读EMBA，结果被一个美女给缠上了，那时候他还梗着脖子说，比我有钱的也有，比我帅的也有，她谁也没找就找到了我，肯定是因为爱。

当时我们都用怜悯的眼光看着他，这跟爱情无关，只是你恰好有钱，又恰好到了人家的狩猎区，又恰好够软弱，被人给围猎了呗，之所以加个爱情，无非是涂抹一层保护色，总不好搞得太赤裸裸的。

有钱，并且把钱弄到了阿美莉卡，肯定不太可能走公开合法的途径。

老美玩钓鱼执法的秘诀在于先让你把钱弄出来，然后让你人来，人跟那么多钱在一起，肯定有说不清楚的地方，这就是所谓的原罪。

可笑的是，很多人之所以从国内把钱挪出来，为的就是资本积累的原罪在，心里总担心会出点啥事，非得把钱弄到鬼子的地界心里才算舒坦了。

所以，在国内的原罪到国外就能被洗白了？

老美一旦想收网，人家立刻就要让你解释钱是怎么赚来的，又是怎么到美国来的，一旦无法解释，或者你人跟钱都来了，但中间有一点纳税的事没说清楚，那么人就要进去，钱就要被没收。

你以为资本主义国家里资本家们能跟你共情？联合起来帮你保护所谓“私有财产”？

这就未免有点太天真了。

老美的富豪们为了保护自己的财产，越是有钱，越是要拿出大把钱去贿赂各路政客，给党派们当幕后金主爸爸，即便如此，还是得豢养高价的律师团队。

一个个又不是吃饱撑了，非要花那么多不必要花的钱。

你一个带着巨额财产润过来的，天生就是个移动靶子，死道友不死贫道，刚好你们被盯上了，人家才无暇顾及去搞他们的钱。

想想看，为啥川宝在位的时候无论干啥就要大叫一句China？这叫祸水东引，让老百姓把注意力都放在东方大国身上，自己干的许多屁事就没人多想了。

所以一旦美国国内贫富差距加大，民众憎恨富豪们，你猜富有润人们被集中办了，会不会引发民众同情呢？

呵呵，同情你才有鬼呢，这就跟当年中世纪的猎巫行动一样，底层民众们集体陷入到狂欢中，所以被抓起来的到底是不是女巫男巫重要吗？重要的是大家因此都很快乐。

你去看看现如今美国人是怎么看待赵长鹏被抓起来勒索，有人帮他喊冤吗？大家恨不得鼓掌叫好，都以为罚款太少，没有把赵老割罚得倾家荡产。

润出去鸽了几次巴菲特的孙割现在不知道咋样，买了个外交身份又如何？美国人狠起来连自己的大使都下得了黑手，何况是个不入流小国的外交人员。

之前跑得老快的李嘉诚花了许多钱，终于又跑回HK了。

李老板还买了个英国爵士的头衔，但那又如何，一旦丧失了统战价值，英伦的镰刀一刀刀就割过来，老李人老成精，一看不好，撒丫子跑回HK，别的不敢说，起码能落个善终。

润人们，但凡脑子好用点，这时候就要多想想，赵老割如今的下场是为了啥，李老板高调回港又是为了啥。

醒目点吧，这已经不是财富自由不自由的问题， 而是你未来还有没有可能有点自由的问题。
