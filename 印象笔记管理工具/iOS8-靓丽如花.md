# iOS8-靓丽如花

---

![](.evernote-content/7024C1FB-D1B9-4433-BD42-D23942527361/AAB36685-2E09-403C-BC9C-C97C3F6346C9.jpg)

今天凌晨一点苹果公司正式发布并推送了 iOS8 的更新。

昨晚一群人在朋友圈叨逼叨 iOS8 升级的事情，还有人宣称要凌晨升级完再睡觉。早晨起来之后发现大家依然谈性甚浓，很多朋友已经升级到了最新的系统，有的朋友还在等待，还有人说，整个公司的网络似乎都变慢了，大家都在升级 iOS8 么？

没有公关和热点炒作，但每次产品发布或系统升级都会形成焦点。iOS 和 OS X 的升级率应该是同类型的操作系统里最高的。看看 Windows XP 就知道了，人家微软已经不再维护这个版本了，国内依然有大量的用户用得热火朝天和死皮赖脸。

苹果为什么能够做到这一点？可以引用我之前文章里的一句话做个概括：

> 苹果通过一套良好的思维体系和工业制度，把自己喜欢的东西精确的设计并制造出来，送到用户手上。优秀的用户体验会让用户与产品建立情感上的连接，并确保用户会需要这个东西。情感的注入是保证产品具备长久生命力的关键因素。

这次 iOS8 的 UI 虽然和上一个版本大同小异，但新增了大大小小一百多个功能，底层框架也做了规模性的优化，这就导致升级 iOS8 需要将近 6G 的空余空间。而大部分iPhone 用户几乎没有清理空间的习惯，总是满了再删一点，再满再删，所以大家有600M 空余空间就已经是富翁了，去哪偷那 6G 剩余空间呢？

于是大家不得不非常苦逼的删掉自己心爱的音乐和视频，删掉那些现在不用将来可能也不用的 App，直到删出5.8G 空间，升级完 iOS8 之后再蚂蚁搬家恢复原状。其实完全没必要这样。如果你在一家软件公司，如果你不是产品经理，如果你没有下死手欺负过程序员，那么，找一个 iOS 的开发者，让他去 [ https://developer.apple.com ]下载一份对应你iPhone 或 iPad 型号的 iOS8 固件，直接用 iTunes 升级安装即可。记住，点击「检查更新」的时候按住 Option 键，在弹出窗口中选择下载好的固件，这种更新方式没有空间要求，剩下的就是等待。

升级成功之后，你会惊喜的发现，靠，和没升级一样！界面和之前的程序是一样一样的啊，谁在玩我？没人玩你，滑动解锁，进入手机，去探寻 iOS8 的靓丽之花。美好和光洁的沙砾，永远躺在深水河的河床底部，等着你去探索和发现。

进入手机界面之后，先去找一个叫做「提示」的 App，这是苹果为 iOS 用户制作的功能引导提示，这次提供了9项，包括快速回应通知、发送音频、管理邮件、Siri 熟知曲目、获得回复通知、最近使用联系人、查看信息发送时间、与家人共享、借助计时器自拍等。这个 App 后续会推出更多提示，大家可以据此学习使用 iOS8 的功能。

今天说说我关注的几个功能。

首先，由于我们无敌的购买力，苹果公司对天朝上国极为看重，这次为车迟国人民做了很多改进，包括农历历法，支持中文输入预测的 QuickType，天气 App 增加了更为详尽的数据，包括日出日落降雨概率湿度风速体感温度气压能见度等等。看到这些功能如实出现我充满了车迟国的荣誉感。结果地图把这种荣誉感破坏殆尽，传说中要增加的矢量图、3D 效果和逐向导航都去哪了？都特么去哪了？我充满了森森的被欺骗感。

![](.evernote-content/7024C1FB-D1B9-4433-BD42-D23942527361/904E5D3A-7CE4-4161-9A89-6C4DDB90D3FD.jpg)

其次，iOS 的锁屏壁纸实现了一种非常奇妙的效果。如题图所示，把锁屏壁纸设置成黑底红花，当点亮屏幕的时候，光线从屏幕四周向中心聚拢，那朵花就像在黑暗中盛开。关掉屏幕时，花儿又会慢慢的隐没到黑暗中。非常有情怀有木有？不过苹果什么都没说。另外，这次的壁纸选择很有讲究，尤其是地球那一张，非常带感。我估计可能是前阵子我吐槽锤子 T1 的壁纸甩 iOS 几条街导致了这样一个结果，嗯。

![](.evernote-content/7024C1FB-D1B9-4433-BD42-D23942527361/3583B9A7-3AD0-49F4-A264-92E5E80766F8.jpg)

第三，第三方输入法终于开放了，大家可以去 App Store 根据自己的使用习惯下载搜狗或百度输入法，我试用了搜狗输入法，全键盘滑行输入，感觉还不错。

第四，照相和图片有了大幅改进，比如延时摄影模式、拍摄计时器、调整曝光、亮度、对比度、高光和阴影等，大家自行体会吧，卡片相机渐行渐远了。

第五，也就是最具备科技感的 Handoff。如果你的 iOS 设备和 Mac 同在一个无线局域网内，那么支持 Handoff 的应用可以无缝在 iOS 和 OS X 中进行切换。如果你正在 Mac 上编辑一个邮件，旁边的 iPhone 会感知到这个操作并在屏幕左下角出现一个邮件的图标。撇开 Mac 直接打开 iPhone，就可以在 iPhone 上继续编辑刚才那份邮件了。

如果你正在 Mac 上打开 Safari 浏览某个重要的网页，突感尿急，抄起 iPhone 就走，打开，网页依然在手。据二爷鉴书描述，这个功能比云标签方便大概10倍。我说你不是产品经理么怎么会有这么精确的度量数据呢？他说四舍五入了，可能9.8倍左右。然后他抄起手机就去厕所了。

这个 Handoff 让我想起了科幻电影中随着手势挥动在不同的设备之间穿梭的虚拟屏幕。我想，这一天不会太远了。

最后，我还设置了一下家人共享，终于 App 不用买双份了，你可以和家人共享 iTunes、iBooks 和 App Store 的购买项目，还可以共享家庭照片、家庭日历等信息。当然，你也可以共享自己的位置，关于这一点，我觉得还是自觉一些比较好，真闹到群众举报的地步就不好了，北京人民的眼睛，是全世界最雪最亮的……

一百多个功能，我就不去讲了，大家可以慢慢体验。iOS8 的发布，让我更加期待 iPhone 6 Plus，另外，OS X 也该升级了……
