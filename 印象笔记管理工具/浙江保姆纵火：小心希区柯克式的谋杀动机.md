# 浙江保姆纵火：小心希区柯克式的谋杀动机

浙江保姆纵火：小心希区柯克式的谋杀动机
-------------------

原创*2017-06-23**陈岚*[女拳文化](http://mp.weixin.qq.com/s?__biz=MzA5NDMyNzY0Mg==&mid=2650117006&idx=1&sn=f0dd2c9ad9b701cdc7a600286a0778aa&chksm=885175bbbf26fcad00d22187fb83014f7394c1794564bf971bdb61e6f0e8de373a464d71c292&mpshare=1&scene=1&srcid=0623QtUjF2xjzWk3nwKjhP2g##)

希区柯克是我见过的人心高手，

他比巫师更洞察人们最隐秘诡异的动念，

从无常、无由、无厘头里看到规律、真相和真实需求。

先说几个匪夷所思的希式故事。

一个通缉犯逃到了小镇附近，藏身时遇到了一个小姑娘。小姑娘镇定自若，聪明伶俐，冒着危险和杀人如麻的通缉犯周旋。最后不仅让通缉犯信任她，还带着通缉犯找到了一个秘密的地窖，乖乖地按照她的吩咐藏进地窖，“当”地一声，小姑娘把地窖顶上的木板扣上，锁好。然后告诉通缉犯：“现在我得到你了。”

“向警方举报你，只能得到25美元。”“而抓到活的你，可以得到50美元。”

通缉犯简直无语了。一个小孩冒着生命危险，和他周旋了几天，最后的动机竟然是——**“镇上店里那个黄裙子，要50美元，那条黄裙子太漂亮了，我一定要得到它。”**

![](.evernote-content/3BDDA6DD-2F86-496B-A97F-A11921CB4F8C/2ED90799-2572-4E06-8675-66DE2CCF3D0A.png)

一个年轻女孩，她的父亲死了。她悲伤地参加了葬礼。亲友们都来吊唁，哪怕是多年不见的亲友。葬礼过后没多久，女孩的母亲也突然去世，又一场葬礼。再这之后不久，女孩的哥哥也去世……

这样非正常的死亡，显然不可能是疾病或偶然了……凶手最终被抓住了。就是女孩。她制造了后来的两次死亡。

她谋杀亲人的动机是什么呢？图谋遗产？泄愤报复？

都不是。

**希区柯克式的答案：在父亲的葬礼上，她见到了一位英俊的远方堂兄。她对他一见钟情。于是，她尽一切可能，制造机会，以便于再见到他。**

![](.evernote-content/3BDDA6DD-2F86-496B-A97F-A11921CB4F8C/2ED90799-2572-4E06-8675-66DE2CCF3D0A.png)

一个老姑娘去墨西哥旅游，她是个非常富有的女继承人，可是她除了巨额财富，什么也没有。身边的男人都是为了钱和她在一起，于是这次旅行，她刻意把自己伪装得很低调很朴素。手指上戴着在旅游景点买的那颗蓝宝石戒指——一颗不值钱的仿制品，又大又亮又美丽，边上那颗她一直戴着的母亲遗留的钻戒，虽然暗淡，价值却是很高。

她和闺蜜在路上雇佣了一个英俊的男青年做导游。

男青年羞涩而腼腆，对年长的她们有着毫不虚假的殷勤，照顾她们没有任何对年老色衰女人的嫌弃。这让她们俩很高兴。

在郊游的一个湖泊里，男青年制造了翻船事故，杀死两个女客人。

**他在湖泊里割下女客人的手指，拿走那颗不值钱的蓝宝石戒指****——****它好美啊。值得去送给自己的母亲，她已经白发苍苍了，可是他还是很爱她，很爱她，除了她，他谁也不会入眼。**

昨天纵火案第一条新闻出来，很多人从新闻中就读出了异常。

失火案里只有保姆一人逃生，女主人和孩子都留在了火场里，而保姆却称是女主人把自己叫醒让自己逃跑报警的。——这明显是一个反人性的描述，没有一个母亲会在失火的时候不是先去抱孩子。

几小时后，滚动新闻爆出，凶手确实是保姆，已经被警方控制。

![](.evernote-content/3BDDA6DD-2F86-496B-A97F-A11921CB4F8C/5E9FA94E-EB11-4857-B73B-991255722BB7.jpg)

动机呢？

**一个小时后，受害家庭男主人接受采访说，他们一家跟保姆相处得还挺好的，不久前还借了****10****万元给保姆家里买房。**

10个小时后，大家都关注的纵火动机，终于爆出。

就是保姆曾经多次盗窃财物，想纵火掩盖痕迹，一了百了。她大概先后盗窃了一块名表，和一些金银首饰在内的财物。至少男主人至案发前，都没有确定保姆盗窃。

——四条人命，和本身价值就在5000万以上的房子，似乎完全不在她的考虑之列。

又有消息爆出，这个保姆曾因欠债，并涉及高利贷，被告上法庭。

这就是典型的“为了煮熟自己一个鸡蛋，放火点别人家房子”的恶行。很早之前我就写过，最大的邪恶不是恶，而是因为愚蠢与贪婪行的恶，他们在作恶时，根本不会意识到、也不会在意自己所做的是什么。塔利班炸掉巴米扬大佛、ISIS毁坏尼尼微古城时，根本意识不到也不在乎，这样伟大的遗迹、记载历史的文物，是举世难觅的珍宝。

我的朋友在单位也遇到过这样的反社会人格的病人。

朋友做义工时认识一个无业人员，一时间怜悯心打动，觉得很可怜，看他挺老实，都穷逼成这样了还做义工，有心帮一把。正好自己单位招人，机会给谁都是给，不如给一个穷途末路一个机会是不是？也是功德一件。于是网开一面招他进了单位。

但谁也没想到的是，在这个大学生找工作都困难、失业率狂高的今天，朋友正觉得自己做了好人好事呢，这个入职一年的家伙，刚转正，先诬告同事，最后诬告上级，最后连介绍人也一并控告。

因为平时他真的看起来老实本分，说话咬文嚼字，而且看起来真的无法解释什么动机，控告介绍人这个举动，就显得说服力：“某某某招我我进来的，按理说对我有恩，我不该揭发他……我内心也很矛盾的。之所以要揭发他，是因为他们做的事太丧尽天良了，揭发他们对我又没好处对不对啦？”——这个逻辑真的竟然说服很多人相信他。

恰好碰上常见的两派系人事斗争，得到另一派一个虚幻的加薪升职的许诺后，他就主动积极地把自己出卖给了魔鬼。非常忠实地充当其中一派的打手，率队撕咬，乌贼一样四处喷污，咬得真诚而热血，连他自己都感动了，都信了。

构陷毕竟是构陷，现在也不是文革时期了。风波平定，最终受害人恢复名誉，争斗两派在老大弹压下握手言和，反对派也是相信他之前说得“有证据”，才把牌打到明面，最后发现丫的是纯粹胡喷，没有证据，害自己当众被打脸，一怒之下立即表态要从重处理这种人。开除、立即开除。

于是，兔子未死，狗已烹。被开掉。而且被整个系统拉黑名单。更被这个世界上可能是唯一真心善待过他的人永久拉了黑名单。

朋友惊魂未定，变成一个祥林嫂，把这个农夫与蛇的故事讲给他认识的每一个人听。——之所以要这样哀怨，是因为他很长时间想不通，怎么会有这样的人？人怎么能这么恩将仇报、信口雌黄？

那只白眼狼真实的隐蔽动机是这样的，和这个纵火保姆一样——这个动机只能说脑壳坏掉、蠢透，是任何一个情商智商正常的现代职场人士无法理解的：“他工作后谈了个女朋友，他自知条件差，为了笼络关系，便想安排他的女朋友工作。他也没有别的资源，所有的资源就是他天上掉馅饼的这份工作。因为他进单位很容易，他就想当然认为这一切都应该很容易，只要敲掉别人的饭碗，必然有空缺，就能近水楼台先得月介绍女友进单位。

白眼狼并不去想（他的智商不够想到），他轻易有工作，这个工作机会背后是介绍人半生的社会资历、人脉背景做背书，他的小心眼认为这个是他自己本事。而且，可以凭借自己的本事攫取更多。

……“他以为挤走了别人就能安排他女朋友吗？”

……“他以为诬告了上司自己就能升职吗？”

……故事三里的女孩杀了自己的母亲哥哥，新的葬礼上，那个陌生的英俊男子就能爱上她吗？一，这往往不在反社会人格的人考虑之列。二，就算考虑到了，她也顾不得了——她活在一个只有自己的世界里。

长期情感与物质的双重饥饿，他们活在一个“得不到就生不如死”的崩溃状态里。在忽然接触到巨额（或他们认为的巨额财富时）**，这样一类人会对财富或权力产生不切实际的幻想，和狂热贪欲。而欲望受阻，也不会自省或寻求合乎规则的途径去实现，而是陷入暴力攻击破坏的**疯狂**中。**

还有一个疑问是：他为什么不去斗别人？比如去斗反对派？就像那纵火保姆，你让她去抢劫杀人，她肯定不敢，遇到黑社会高利贷敲诈，她也一定跪地求饶。

因为他们这样的人，本质上胆小虚弱，只敢去咬善待他们的人，你打开了门，他才有了机会，而且是唯一的机会，他必利用充分、敲骨吸髓，也不在乎明天还过不过。

近期还有一个案子，也是这样的巨婴式的心理表达。毕节一男子因“取款时卡上没钱，就烧柜员机”，

![](.evernote-content/3BDDA6DD-2F86-496B-A97F-A11921CB4F8C/89E41BDE-9ACB-4B17-8886-012526499162.jpg)

没有明天，没有长远规划，也没有道德和温情。本质上，这些人和刚刚两岁的“得不到糖果就用脚踩碎、跺烂、吐口水”的婴儿没有区别，“如果这个世界不能满足我，就让世界去死好了。”这是巨婴心态，也是流民心态。

“我死后哪怕洪水滔天”的绝对自私。

前几天在和朋友们讨论，转型中的中国，象一个被巨手晃荡过的瓶子，瓶中之水正在重新沉淀，慢慢地完成新的阶级固化和新的社会相处方式。美国早已经走完了这条路。所以，很少会有美国家庭主动认识陌生人，更不可能把不是自己社会阶层的人带进自己的家庭生活。尽管在这样的固化表面，充满了其乐融融的平等意识、平权观念，不同阶层的人却绝不会在一个社区生活？更不会带回家中。

贫富差距存在于每一个社会里，但很少社会象当下的中国，有这么多天然仇富，而且理直气壮仇富的声音。原因当然很多，无法展开细数。

但是，这样的结果，只能提醒大家，远离垃圾人，多做背景调查，不要把没有充分了解的人带进自己的生活和工作圈子。

**因为你真的不知道，他的背景、他的经历会藏着怎么样的吊诡动机，又会在什么设定里诱发怎么样的疯狂。**

**为逝者志哀。**
