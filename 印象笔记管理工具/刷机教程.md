# 刷机教程

---

下载必要文件

下载ROM文件与刷机工具，为刷机过程做准备。

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/8376075D-82BE-4775-99D1-B4DF9167AE8F.gif)

1.本教程刷机过程需要在Windows操作系统环境下进行。  
2.要求型号为SAMSUNG Galaxy Note II GT-N7100国际版，韩版和中国运营商定制版未经过测试不推荐刷机。  
3.刷机有风险，务必要小心；因刷机造成的任何问题，小米公司不承担任何责任。  
4.本教程内容摘自互联网资源并经官方整理，如有版权问题可联系管理员。更多内容详见[免责声明](http://www.miui.com/a-69.html)。

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/E7BA5E8F-1D75-43B8-8B62-29B04EEC7521.png)

下载 MIUI ROM for Galaxy\_Note 2最新安装包 [立即下载](http://www.miui.com/download.html)  
如果已经在下载页面下载过最新安装包，无需重新下载。

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/6984D829-1B5F-4C11-B2AD-8966E8213D48.png)

下载N7100刷机工具包 N7100\_tools.zip [立即下载](http://bigota.d.miui.com/tools/N7100_tools.zip)（大小1.05G，MD5：17042b145c99d0ab939d34944ce5f989)  
下载完成后放置桌面并解压。

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/B627A873-2BD7-4989-BDAA-69E30FA15448.png)

请确保手机内存空间剩余大于2GB。

不要解压MIUI ROM安装包，将其直接拷贝到内置SD卡根目录下。

安装USB驱动

准备工作：安装必要的驱动,为后续刷机做准备。本步骤安装USB驱动。

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/8376075D-82BE-4775-99D1-B4DF9167AE8F.gif)

如已经安装过USB驱动,可略过本步骤，[前往下一步](http://www.miui.com/a-279.html)

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/E7BA5E8F-1D75-43B8-8B62-29B04EEC7521.png)

在电脑上操作  
  
打开N7100\_tools/Drive目录下SAMSUNG\_USB\_Driver\_for\_Mobile\_Phones.exe。  

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/C9182DCB-15F3-4F69-A13F-379D2271E206.gif)

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/B763C7D9-B882-478A-83C4-CD83419D3CFA.png)

[![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/6767FF1B-A03E-4415-AE3D-ADFD1F71BA00.png)](http://www.miui.com/a-277.html)

恢复三星官方4.1ZSALJ7系统

将N7100恢复至官方系统ZSALJ7以兼容本教程。

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/8376075D-82BE-4775-99D1-B4DF9167AE8F.gif)

进行本步骤会清空所有用户数据并会失去质保，请慎重考虑，做好相应的数据备份，确保手机电量充足。  
没有备份过EFS文件请先前往备份EFS教程 [点击进入](http://www.miui.com/thread-728527-1-1.html)  
如果已经已经是三星官方N7100OZSALJ7请前进[下一步](http://www.miui.com/a-280.html)。

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/E7BA5E8F-1D75-43B8-8B62-29B04EEC7521.png)

在手机上操作  
进入设置——应用程序——开发人员选项——USB调试 打勾 √  
完成后将手机开机状态下连接电脑，双击刷机工具包中ADB/adb Download.bat进入刷机模式。  
也可以通过按键方式进入刷机模式，方法：关机状态下，同时按住音量下键+HOME+电源键，出现选择画面之后按音量上键。  
屏幕出现绿色机器人图案，显示Downloading...表示已经成功进入刷机模式。

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/6984D829-1B5F-4C11-B2AD-8966E8213D48.png)

在电脑上操作  
运行刷机工具包中的Odin3 v3.04.exe  
勾选Auto Reboot/F.Reset Time两项；  
点击PDA按钮，从刷机工具包中选择N7100ZSALJ7\_N7100OZSALJ7\_N7100XXALJ2\_TGY.TAR；  
将手机连接至电脑，如图位置变成深绿色的才表示电脑连接手机成功，如果没有没变深绿色请重新插拔USB数据线；  
确认手机连接成功后，点击Start，等待完成，出现下图说明刷机成功，手机会自动重启并开机,之后关闭电脑上的Odin刷机工具。

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/C9182DCB-15F3-4F69-A13F-379D2271E206.gif)

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/AE17A7B9-148A-4EB4-BA06-370CF81961D5.jpg)

[![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/6767FF1B-A03E-4415-AE3D-ADFD1F71BA00.png)](http://www.miui.com/a-278.html)

置入RECOVERY

置入第三方Recovery以兼容本教程

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/E7BA5E8F-1D75-43B8-8B62-29B04EEC7521.png)

在手机上操作  
设置——应用程序——开发人员选项——USB调试 打勾 √  
之后将手机连接至电脑，双击工具包中/ADB/adb Download.bat进入刷机模式。  
也可以通过按键方式进入刷机模式，方法：关机状态下，同时按住音量下键+HOME+电源键，出现选择画面之后按音量上键。

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/6984D829-1B5F-4C11-B2AD-8966E8213D48.png)

在电脑上操作  
  
运行刷机工具包中的Odin3 v3.04.exe  
勾选Auto Reboot、F,Reset Time两项；  
点击PDA按钮，从刷机工具包中选择cwm6-root-n7100.tar；  
将手机连接至电脑，如图位置变成深绿色的才表示电脑连接手机成功，如果没有没变深绿色请重新插拔USB数据线；  
确认手机连接成功后，点击Start，等待完成，出现下图说明刷机成功，完成后会自动重启。  

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/C9182DCB-15F3-4F69-A13F-379D2271E206.gif)

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/E4AF4E3D-2F20-42C9-9266-0E25DD599B77.jpg)

[![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/6767FF1B-A03E-4415-AE3D-ADFD1F71BA00.png)](http://www.miui.com/a-279.html)

置入MIUI完整包

清除缓存数据置入MIUI完整包

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/E7BA5E8F-1D75-43B8-8B62-29B04EEC7521.png)

在手机上操作  
  
设置——应用程序——开发——USB调试 打勾 √  
之后将手机开机状态下连接至电脑，双击工具包中/ADB/adb Recovery.bat进入恢复模式，浅蓝色英文界面。  
也可以通过按键方式进入恢复模式，方法：关机状态下，同时按住音量上键+HOME+电源键。  
依次执行（音量键选择，电源键确认）  
wipe data/factory reset——Yes——delete all user data.  
wipe cache partition——Yes-Wipe Cache.  

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/BA868A41-F107-4A66-B318-947F5EC6C07C.png)

![](.evernote-content/C78920B2-E691-4006-9BD2-4E5E00844CCA/6984D829-1B5F-4C11-B2AD-8966E8213D48.png)

在手机上操作  
  
依次执行（音量键选择，电源键确认）  
install zip from sdcard.  
choose zip from internal sdcard.  
miui\_miui\_Note2\_2.X.X\_gdiqrbqkgr\_4.0.zip  
Yes-install miui XXXXX.   
之后点击Back键，执行reboot system now.  
等待完成后手机会自动重启。
