不知道你有没有这样的体验：工作或学习不得不需要面对着电脑屏幕，心里本就一万个不情愿，这时再看到乱糟糟的桌面，心思更难静下来了。

将文件需要按照场景、类目分门别类地进行整理，越来越不是表面工程，而成为我们的刚需。

我个人的文件归类习惯是：

1. 工作、生活、学习分开
2. 高频文件夹放在 Dock 栏
3. 四象限式的桌面用来暂时存放无法归类的文件，稍后整理

macOS 上的文件管理在系统本身和优质应用的加持下已经十分便利了，但当开始进行归类的时候，还是会碰到不可避免的效率问题。比如要将文件分别归类到 3 个文件夹，最快的方式自然是打开 4 个 Finder 窗口，而不是在多个窗口间来回切换，找不准目标。

这个用户痛点，其实是显而易见的，但是因为手动能搞定，所以长久以来少有开发商挖掘这个方向。最近，一款名为 [QSpace](https://apps.apple.com/cn/app/qspace/id1469774098?mt=12) 的多视图文件管理器上架 Mac App Store ，正是专为解决这个痛点而来。在上架后不久，QSpace 就跃升至付费榜第一名。

![](.evernote-content/2E36F771-A61F-448F-842F-DDE383FC75F9/C8866594-57C7-4560-BCDC-A746BE693060.png)

多视图工作区
------

初次打开，QSpace 会帮你新建一个由四个窗口组成的多视图工作区，这四个窗口有着和系统访达（Finder）相一致的操作习槚和界面呈现，同时还支持更多灵活而实用的特色功能。

![](.evernote-content/2E36F771-A61F-448F-842F-DDE383FC75F9/90EAA0A7-73FC-4350-B3DB-345290883AAE.png)

QSpace 支持列表、图标等 3 种文件展示方式，共有 12 种排布视图。当我需要对桌面进行整理时，我一般会选择默认的四宫格视图，左上角放置「桌面」文件夹，其余三个分别放置「工作」「学习」和「生活」文件夹。而当需要对诸如旅拍等照片进行整理时，我会选择左右两半的视图，如此排列可以有足够的空间显示照片缩略图，方便进行整理。

当然，你可以根据需要选择合适的视图组合，搭建独一无二的工作区。

![](.evernote-content/2E36F771-A61F-448F-842F-DDE383FC75F9/42B2FCE1-1696-40D7-875B-BA4F4ADE79A4.png)

QSpace 工作区所提供的 Finder 窗口，右击文件的菜单并没有原生的选择多，但保留了关键的用于整理的功能，更一目了然。

![](.evernote-content/2E36F771-A61F-448F-842F-DDE383FC75F9/9A265E16-84E3-45D0-90FB-A8F9D6B031C3.png)

左：QSpace 提供的右击菜单 右：系统原生右击菜单

列表视图支持 2 级缩放，图标视图 4 级缩放，按「 Command + 加号 / 减号」即可调整图标大小。

QSpace 还支持创建多个工作区，你可以为每一类文件归纳保存一个工作区，再次打开即可显示对应的视图及目录。所有工作区状态会自动保存，再次打开即可恢复。

比如当我需要整理桌面时，我会新建一个由「桌面」+「工作」+「学习」+「生活」的工作区，便于我对文件进行归档，将这一工作区进行保存后，再次打开就不用一一设置了。

![](.evernote-content/2E36F771-A61F-448F-842F-DDE383FC75F9/01DEF308-1AFE-4685-9942-3E5473F97924.png)

整理「桌面」工作区

增强的文件管理器
--------

QSpace 不仅是多视图工作区，更是一个增强的「文件管理器」。

它采用了浏览器式的可编辑式地址栏，你可以像使用 Finder 一样快速入门使用 QSpace，比如你可以在任意位置创建新文件、文件夹；在工作区视图之间、工作区视图和其他应用之间进行剪切、拷贝、粘贴、拖拽等操作；前进、后退、前往上级、拷贝或直接输入路径等。

![](.evernote-content/2E36F771-A61F-448F-842F-DDE383FC75F9/79228B56-40F8-4B85-AFAF-1852556A035E.png)

你还可以在「偏好设置」中自定义新文件模板，自定义文件名和文件内容。

![](.evernote-content/2E36F771-A61F-448F-842F-DDE383FC75F9/0A40F7C9-0C82-4DEA-9058-370F68CBBFC1.png)

尤其适合开发者
-------

一直以来，macOS 的 Finder 和终端直接的切换并不方便，没有便捷的路径复制。而且有时候，一个项目会涉及到文档、设计稿、资源、代码等多个目录，只能重复打开多个 Finder 窗口来回切换。

除了上述适用于全体用户的功能，QSpace 还考虑了开发人员的需要，在路径右侧放置了两个自定义图标，分别是「终端」和「编辑器」，分别实现一键打开当前目录的功能。

* 终端快捷键：一键在指定终端打开已选择目录（支持 Terminal、iterm)
* 编辑器怏捷键：一键在指定编辑器打开已选择的文件或目录（支持 VSCode、Atom、Sublime）

你同样可以在「偏好设置」中进行相关设置的更改。

![](.evernote-content/2E36F771-A61F-448F-842F-DDE383FC75F9/627EFE48-BC81-4223-A8F6-5D5675F57996.png)

正如 QSpace 的开发者所说，**「开发一款带有 Chrome 式地址栏，直接输入地址、拷贝地址，并且能把窗口切分成 N 个视图区域，最好还能一直保持工作区状态的小工具。」**

于是，QSpace 上线了，它看似只是对 finder 的小修小补，但正是这看似简单的新功能，让原本 finder 多开难、整理难的问题得到了解决，

无论你是追求便利的开发者还是有着文件整理需求的普通用户，相信 QSpace 都能让你在文件管理上更上一层楼。
