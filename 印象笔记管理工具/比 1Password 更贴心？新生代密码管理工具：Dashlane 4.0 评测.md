# 比 1Password 更贴心？新生代密码管理工具：Dashlane 4.0 评测

---

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/6D9326B4-D137-4A23-ACA8-A2A06E76D68A.jpg)

账号安全一直都是互联网时代中的黑天鹅，稍有闪失，就可能造成不可预料的后果，由此引发的资金被盗、小三被爆、祼照被套事件层出不穷。

想当年，少数派在[密码安全的科普文](http://sspai.com/31659)中洋洋洒洒地介绍了盗号的原理、设置密码的要领、密码管理的工具，只可惜文章 TL;DR，文章里介绍了一些密码管理的工具，如 [KeePass](http://www.keepass.info)、[1Password](http://sspai.com/tag/1password)、[LastPass](http://sspai.com/tag/lastpass) 等，作为一个重视信息安全的人，你可能在不同的站点上使用了不同的密码，要想靠脑子记住这些密码不是一件简单事。而这些工具的存在就是为了解决这个问题，它们的本质思路其实并没有太大区别，你可以往其中添加记录许许多多的密码，在这些密码之上，会有一个主密码，只有掌握了这个主密码，你就能解锁访问其中存储的所有密码。

这些工具之中，[1Password](http://sspai.com/tag/1Password) 可能是许多不差钱用户的首选，毕竟它在各方面都非常优秀，无论是口碑、跨平台性、实用性，都首屈一指。不过，今天要给大家介绍的 Dashlane 丝毫不比其逊色，甚至在很多方面比 1Password 更加贴心、实用。

贴心之一：免费模式，采取跨平台的订阅收费策略
----------------------

其实少数派之前也[介绍过 Dashlane 的 Android 版](http://sspai.com/29048)，和 1Password 很像，Dashlane 同样可以在不同的平台上使用，包括 iOS、Android、Mac 和 Windows。然而非常不同的是，Dashlane 的定价策略和 1Password 有着非常大的不同。

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/0120F7F9-0918-4670-9ECA-5C41F4EC90D5.jpg)

众所周知，1Password 采用的是单用户、单平台分别授权，这意味着，即使你想要使用 1Password，在 Mac 和 Windows 平台上你需要分别购买，其 iOS 和 Android 应用虽然在去年采用了免费 + 增值服务的定价策略，但想要使用完整功能，还是必须内购。

而 Dashlane 提供了免费模式，并且可以使用大多数基本功能，如密码管理、自动填充、电子钱包等，并且在此基础上采用订阅付费模式，也就是说只要你购买了 Dashlane 的服务，在各个不同的平台上都可以直接使用。付费版主要增加了账户备份至云端、多设备同步、不限密码分享个数、网页版访问等特性。

从定价上来看，Dashlane 一年 258 元，三年 518 年，五年 648 元，而 1Password 的售价则是 Windows 和 Mac 版本分别需要 326 元（打包购买可节省 180 元），移动版本解锁内购 68 元，在某些大版本升级时还可能需要再次付费购买。这样算下来，1Password 各平台的加总售价差不多相当于 Dashlane 三年的服务费用。

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/E5675196-A410-477E-9643-456120D90C17.jpg)

贴心之二：核心功能的细节之处
--------------

从功能上来说，Dashlane 最核心的功能就是帮你存储不同网站和服务的密码，相比 1Password，它在一些细节之处更加贴近人心。

**1. 支持按站点快速添加：**虽然 1Password 也有类似的功能，但是在界面排版上，Dashlane 的排布更加合理，一是采用了网格布局，而不是像 1Password 使用列表；二是在站点的流行度方面，Dashlane 提供的选项显然更加合理。

**2. 自动适配网站标识与底色：**使用 1Password 的话，经常会发现添加一些国内站点并不能很好地识别 Logo，只会显示其默认的统一标识。而 Dashlane 这一点则要智能得多，会拉取网站的 Logo，并且在详情页上，还会根据 Logo 颜色自动配色。

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/C62698C4-9952-455B-BC86-3F436D7A232F.jpg)

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/2BA20E7D-6120-4056-A138-3E9417A2CD6F.jpg)

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/8A2D9124-FA63-4369-BD56-342B5673D010.jpg)

**3. 按名称、类别或最近使用排序：**1Password 的排序方式比较死板，它甚至使用了三个界面：收藏、类别与整理来梳理不同的排序要求，而 Dashlane 的处理方式则更优雅些，它提供了按名称、类别或最近使用的排序方式，更贴近实际使用需求，而不需要用户的大量手工整理。

**4. 支持自定义类别：**1Password 移动端的 Pro 功能之一就是获取更多的密码类别，而在 Dashlane 中，这不仅是免费的，而且还支持你自定义根据自己的需求添加更多的类别。

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/38D5356D-671B-4222-A231-52570297CDC7.jpg)

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/4F6CE247-2792-4B62-9A77-ECC87CF4160C.jpg)

贴心之三：密码生成器与一键更改密码
-----------------

除了存储密码的核心功能外，Dashlane 还围绕密码提供了一些实用的小功能，其中之一就是密码生成器。在这里，你可以设定需要密码的长度，是否包含数字、字母、符号，甚至还有一个选项「Pronounceable」，如果勾中了这个选项，生成的密码会尽量使用可拼读的音节组合在一起。

另一项实用功能就是一键更改密码，Dashlane 可以自动检测你的密码安全程度，并提供一键更换密码的服务，只需要你点击两下，它会自动连接第三方站点的 API 对密码进行修改，并支持 200 多个站点。不过不知道是否出于安全的考虑，在支持的服务列表上，除了 Pinterest、Hulu、Netflix 等之外，没有看到其它主流的站点。感兴趣的话，你可以自行查看[所有支持的站点列表](https://www.dashlane.com/password-changer-list)。

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/70A40831-0E89-4B74-8884-31472CD8CD88.jpg)

贴心之四：自由度更高的安全笔记
---------------

除了密码之外，Dashlane 还提供了笔记管理，并且提供了一些默认模板，如财务数据、法律文件、会员身份、服务信息、软件授权、应用、数据库、WiFi 密码等 8 个模板，方便你记录和存储相关信息。在 1Password 中，这些类目可以作为密码中不同类型创建，而在 Dashlane 中，显然它将这些字段和自定义更丰富的信息单独成类，作为笔记的形式，更加具有灵活性。

除此之外，你还可以给笔记打上颜色分类标签，并且，如果你勾选上「password protected」，每次查看笔记时还需要额外输入主密码，最大程度上保障了你的小秘密，都可以记录在这些笔记中不被人发现。

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/24E79B88-C2C2-42B9-B018-BBBFEAAD902E.jpg)

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/5E8573C6-7691-402B-8D3F-2077A941036F.jpg)

贴心之五：省心省事的电子钱包
--------------

在网上购物时，如果你使用银联服务，每次在不同站点都需要重新绑定，是不是很麻烦？Dashlane 支持添加你的借记卡、信用卡、PayPal 账号、银行信息，这样，遇到需要填写银行卡信息的地方，它都可以帮你自动填充。甚至，它还会智能地分析当前是否支持信用卡并为你自动匹配相应的选项。

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/9FE1E4A7-354E-4580-A9FE-C2009A11C999.jpg)除此之外，当你使用 Dashlane 提供的电子钱包后，它还会自动存储消费凭证，并有对应的页面截图；把卡片和账单地址、有效期关联，当你的信用卡有效期快到时，还会提前通知你换卡。

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/3174E2B8-6AB1-4ED3-88BC-AC82CCF578FE.jpg)

贴心之六：细节之处，处处是惊喜
---------------

除了上面提到的这五点，Dashlane 的细节之处的考虑还有许多，例如在 Android 第三方应用中的展现更为明显，它会以悬浮球的形式出现在输入框旁，你只需要点击选择要使用的账号密码，然后等待登录即可。而在 iOS 平台上，已经有 180 多个应用集成了调用 Dashlane 的服务，而在 Safari 中，只需要通过 Action Sheet 便可以在网页上无障碍使用。

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/CED69966-EFCE-4B94-BE12-89D299997BE4.jpg)

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/C72F78AE-1EA5-4EE5-8A14-46DFB9531C1D.jpg)

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/8894E356-BA8B-4B15-A17E-5F5421FFE27E.jpg)

而在解锁方式上，除了 Touch ID 和主密码解锁，Dashlane 还提供了便捷的四位数 PIN 码解锁 。另外，Password History 功能还可以保存所有使用过的历史密码记录，特别是在密码申诉时查询之前使用过的密码非常有用。

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/96FB2A10-42FF-4950-9A5B-AD3E2A046243.jpg)

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/2122F2C3-D5D3-46CF-91DF-77BCD8669C24.jpg)

除此之外，Dashlane 还支持编辑个人信息和证件信息、在确保密码不可见的同时，将密码分享给他人使用、添加紧急联系人（对方可以请求获取你的所有密码，如果你在一段时间内不予拒绝的话）等一系列功能，在 Mac 版本上，还有看起来高端大气上档次的安全表盘，展现你的安全评分、各账号密码的安全指数与行动建议等信息。

![](.evernote-content/E543F914-BC39-43D9-AF83-E0035003A62F/1038BD97-BD8B-42F6-8C61-B42F6E831260.jpg)

总的来说，如果你之前没有尝试过类似的密码安全管理工具，或者因为 1Password 的售价望而却步没有坚持使用下去，那么你可以试试看 Dashlane，它更容易上手，在细节方面也更友好，在功能上也有许多贴心和实用的亮点。

目前 iOS 用户请前往 App Store 下载，Android 用户也可在 [Google Play](https://play.google.com/store/apps/details?id=com.dashlane) 免费下载 Dashlane。
