# 航班为何要“换季”

---

| --- | --- |
| 航班为何要“换季”？航空旅客出行小贴士  [南海网 http://www.hinews.cn](http://www.hinews.cn/)时间：2012-03-24 10:35 来源： 民航资源网 作者：     |  | | --- | | 我国民航系统每年都要进行两次航班计划调整，每年3月的最后一个星期日至10月的最后一个星期六执行夏秋季航班计划，10月的最后一个星期日开始到第二年3月的最后一个星期六执行冬春季航班计划，称为“航班换季”。航班换季主要涵盖增加新航线、增加新航班、调整航班时刻、调整航班线路，取消航线、航班等项工作。  　　那为什么航班会有季节之分呢？  　　航班换季一方面是考虑到旅游业的淡旺季、人们出行的周期性。根据航季的不同，各航空公司将参照执行的航班收益情况，在新航季里对现有航班进行调整。例如春季换季后三亚航班减少，因为夏季通常是海南旅游淡季，游客少，所以航空公司相应减少航班。而冬季北方寒冷，凭借温暖怡人的气候，三亚在冬季更受欢迎，航班数量会随之增加。  　　另一方面，根据我国的实际情况，春夏和秋冬之间全国天气、气压都有很大的差异，导致飞行标准会有很大的不同。航空公司对航班进行调整，一方面是为了安全，一方面是为了方便旅客。 | |
