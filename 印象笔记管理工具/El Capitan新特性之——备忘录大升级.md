# El Capitan新特性之——备忘录大升级

---

[![](.evernote-content/9A60E45C-69FA-43EE-B20F-C7B16DD1B125/36C4093D-18EB-42FE-BCC7-DF3AD51CFDCB.jpg)](http://www.feng.com/apple/tutorial/2015-10-23/El-Capitan----memo-big-upgrade-of-new-features_627988.shtml)

　相信在 iOS 9 和 OS X El Capitan 发布之前，对于许多 Mac 和 iOS 设备用户来说，备忘录应用基本上没有什么用处，他们基本不会使用这款应用。但是在 iOS 9 和 OS X El Capitan 中，一切都变化了，如今的备忘录应用已经非常强大了，今天我们就来了解一下 OS X El Capitan 中的备忘录是什么样的。

[![](.evernote-content/9A60E45C-69FA-43EE-B20F-C7B16DD1B125/36C4093D-18EB-42FE-BCC7-DF3AD51CFDCB.jpg)](http://resource.feng.com/resource/h060/h60/img201510232221260.png)

**更新升级**

要使用备忘录新的特性和功能，用户需要首先更新 iCloud 中的备忘录。在 iCloud 账户的备忘录侧边栏中用户可以看到有一个提示更新的按键。用户更新备忘录的内容之后就可以在其他登陆了同一个 iCloud 账号的、运行 OS 9 或 OS X El Capitan 的设备上看到自己更新的备忘录。但是如果用户没有更新自己所有的 Mac 或者其他设备，那么新的备忘录内容就不会出现在运行旧的操作系统版本的设备上，而在旧操作系统版本设备上创建的备忘录内容也不会出现在新版本中。用户只有将所有的设备都更新之后，你的备忘录和相关联的数据才会出现在其所有设备上。

**共享**

大部分苹果应用和一些第三方 OS X 应用中的共享菜单中现在都都多了一个备忘录选项。所以如果你是在浏览一个网站的图片、在 PDF 中阅读文本等，你都可以通过点击这个共享按键，选择备忘录把网页链接、照片、文本等内容保存下来，文件夹能够帮助用户进一步进行管理，通过缩略图用户可以快速了解到备忘录内有什么媒体内容。备忘录已经成为一个非常宝贵的搜索工具，不管用户身处何地，它都能够在你所有的设备之间进行同步。它也是 Mac 和移动设备之间最快的内容共享工具。

**核对清单**

很多用户在日常生活和工作中会有各种清单列表——购物清单、想法列表、支出明细等。但是这些清单最终很多都是没有能够好好地完整记录下来。如今在新版本备忘录中就不会有这样的问题。在 OS X El Capitan 的新版本备忘录中，用户只要一个点击就能创建清单列表。用户还可以给文本选择一个自己喜欢格式，包括标题、段落标题和项目符号和编号列表。还可以对文本继续加粗或者斜体操作，或者直接将照片应用中的照片添加进去。

**查找**

当然了，因为你可以在备忘录中添加附件，所以有时候你难免想不起来哪个附件是添加到哪个备忘录中。不过这不是什么问题，因为所有你添加到备忘录中的附件都会被放在附件浏览器之中，在这里只要你浏览一下就可以找到你需要的附件了。如果你在编辑新的备忘录时，需要将文件、URL、照片、地图位置、PDF、视频等附件内容添加其中，那么你就会发现它有多么重要了。使用附件浏览器的用户还可以类型和日期为基础来查找你需要的附件。

**照片**

你可通过 "照片" 按钮直接访问照片图库，轻松将照片和视频拖放至备忘录。

**轻扫**

用户可以通过向左轻扫的方式来删除备忘录，虽然这只是一个很小的特性，但是这会让从 iOS 平台上过来的用户迅速熟悉 Mac 及其用户界面。
