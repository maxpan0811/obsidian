有关去哪儿手机客户端活动总结：

一、基本情况分析：

1、400电话数：

开售前（12月6日—12月7日20：00）：

13个，有效10个

问题整理如下：

1、签证问题：签证如何办理及拒签损失如何计算……                    5人提问

2、产品延伸服务问题问题： 如酒店预定，景点包含，接送等问题。       1人提问

3、产品理解问题：分不清团队和自由行概念，以为产品是团队游          2人提问

4、团期问题：确认团期                                              2人提问

 

活动期间：12月月7日20：00—12：00

16个（包括回拨），有效14个

问题整理如下：

1、签证问题：签证好不好办，多长时间……                            2人提问

2、产品细节问题：包含项目，延伸服务……                            4人提问

3、产品理解问题：分不清团队和自由行概念，以为产品是团队游          3人提问

4、关于百程产品对比问题                                            5人提问

 

2、活动情况分析：

整体不理想。

|  |  |  |
| --- | --- | --- |
|  | 流量（产品点击量） | 成单数 |
| 华远 | 8.9w | 4 |
| 平均 | 13w |  |
| 最高（百程） | 18w | 30(全部) |

 

流量原因：（结合李岩反馈）

1、按照去哪儿的历史数据，周末流量工作日普遍较低。

2、和竞争对手的产品相似度很大，价格也比对手高。

3、竞争对手的产品在我们之前做活动，而且时间很近，冲掉了一部分的需求，也形成了比较明显的对比。

成单：

1、个人错误：录团期时由于自己的不认真和对系统的不了解，一个团期的录入是错误的（虽然在活动前改正了），但对成单一定有影响。

2、客观原因：

竞争对手的产品比我们的产品力强，而且时间很近，客人很容易形成对比而不平衡。

3、延伸服务这一块是劣势。（我们：自备签；百程：签证做为延伸服务并给予优惠）

 

三、收获：

有关产品：

1、以后的产品中应该涉及延伸服务这一块，有的客人不愿意多家采购，觉得很麻烦。这一块比较影响转化。

2、签证是比较重要的一块延伸服务，客人倾向于在哪里跟团，在哪里办签证，觉得产品供应商在团期和目的地的把握上，比较可靠。

3、客人对产品的误解，也说明，去哪儿的平台上有跟团游的受众（有客户明确说，下次有跟团游产品一定联系他）。

 

有关店铺运营（活动参与）的收获：

1、参与活动或者上产品之前要对市场有了解，充分沟通，仔细对比，知道自己产品的优劣势。保证产品力。不能因为活动的力度，就弱化对产品力的重视。

2、在明确产品的产品力和对手相比没有优势时，要更加重视活动的安排，谨慎考虑市场需求，在活动时间安排和库存量上设置上都要更用心。）

2、订单的跟进很重要，及时跟进订单，会对成交有影响（有一单是因为及时回拨电话，消除客户疑虑成交的）。

3、在沟通时，即使不成单，也要留心客户的直接和潜在需求，也要做记录和客户分析，要在后台做潜在客户的备注，方便二次销售。（有适合的产品时，可以通过微信对需求一样的客户作统一的推送）

4、即使不成单，也要给客户真诚的建议，并介绍自己的店铺（后期可以请客户关注我们的服务号），对店铺的形象比较重要。

5、产品信息输入一定要仔细准确，关键的信息要反复核查。产品的图文呈现也很重要。

6、面对客户，专业度很重要，影响到客人对店铺的信赖感。
