# All-in-One Messenger，把主流聊天服务聚合在一个 Chrome 扩展里丨App+1

---

![](.evernote-content/3F4D3EEE-45C8-4BA2-897E-888A5686AC67/75DCFFD8-79F9-46ED-B330-79D8F94626C1.jpg)

少数派此前曾介绍过 [Franz](http://sspai.com/34004) 这款通讯聚合类的应用，它可以把常用的通讯应用聚合在一个应用窗口内，免去来回切换窗口的麻烦，用了一阵之后，我个人觉得还算顺手，但是唯一有一个问题一直让我耿耿于怀：为什么一个基本由网页封装组成的应用，在安装之后会达到 100 多 MB 之巨？这对于我这种 128 GB MacBook 用户来说，还算有些敏感的。

后来，我发现了 [All-in-One Messenger](https://chrome.google.com/webstore/detail/all-in-one-messenger/lainlkmlgipednloilifbppmhdocjbda)，它是一款轻量的 Chrome 扩展，作用与 Franz 基本相同，只需要打开一个窗口，所有的通讯服务会以网页标签栏的形式呈现，方便你自如切换，可以大大提高了沟通的效率。但是和「庞大」的 Franz 相比，[All-in-One Messenger](https://chrome.google.com/webstore/detail/all-in-one-messenger/lainlkmlgipednloilifbppmhdocjbda) 仅有 1.4 MB 大小。

应用支持的通讯服务的种类也不算少，基本涵盖了国外主流的服务，对于国内用户，里面内置了微信、QQ、钉钉和 BearyChat，应该也能满足一部分的沟通需求。

![](.evernote-content/3F4D3EEE-45C8-4BA2-897E-888A5686AC67/4EF0060C-C1D7-4FC0-8BCF-A1A0BA9B44AA.jpg)

不幸的是，由于采用了网页封装的方式，你获得的体验如何完全取决于开发商对于网页版的态度，对于 Telegram、Slack 或 WhatsApp 这类网页版体验较好的服务，用起来你会感到幸福感满满。

假如你碰上了相反的情况，体验会很糟糕。比如 QQ 的网页版就是一个已经两年没有更新过的服务，几乎处于不可用的状态。

如果你像我一样，经常需要同时打开 Telegram、Slack、微信等多款聊天应用，又不想来回切换窗口，All-in-One Messenger 是一个不错的选择。尤其是对于 Mac 用户来说，疏于更新的 Mac 版微信，使用体验可能还不如网页版。

你可以前往 [Chrome 网上应用店](https://chrome.google.com/webstore/detail/all-in-one-messenger/lainlkmlgipednloilifbppmhdocjbda)免费下载 All-in-One Messenger。
