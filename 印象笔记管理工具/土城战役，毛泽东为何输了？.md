# 土城战役，毛泽东为何输了？

提起毛委员，很多人第一反应往往就是政治家和军事家。

他一生指挥无数次战斗，凭借高超的战略天才和对战争本质的深刻洞察，平生鲜有败绩。

而遵义会议，也被称为我党我军生死攸关的转折点，让人觉得毛委员掌权后，我军主力基本就没什么败仗了。

但鲜为人知的是，遵义会议后的第一场大仗，毛委员就打输了。

![](.evernote-content/A95F0D60-6D0A-4D6F-8F55-F60943400335/FD3C2150-3384-40C2-A244-9FBCBBC43EE5.png)

****01****

1956年9月10日，在八大预备会议第二次全体会议上，毛委员谈到经济建设要戒骄戒躁时，就举了打仗的例子：

骄傲不得，我也是犯过错误的。

比如打仗，高兴圩打了败仗，那是我指挥的；南雄打了败仗，是我指挥的；长征时候的土城战役是我指挥的，茅台那次打仗也是我指挥的。

而这四次败仗中，以土城战役规模最大，影响最深，因为土城战败，红军甚至扭转了长征的行军方向。

1935年1月，红军占领遵义后，一共有两种选择。

一是北渡长江，与红四方面军会师。

二是东出湘西，与红二、六军团会师，这也是最开始的打算。

此时中央也在犹豫，到底走哪条路，关键时刻老蒋替中央下了决心：别走了，留下来吧！

老蒋集中四十万大军，向遵义地区进逼，而红军不到四万人，敌我兵力对比异常悬殊，红军命悬一线。

老蒋大兵压境，和贺老总会师困难重重，中革军委只能见招拆招，下令北渡长江，进入川西北，同红四方面军会合，寻机创立新的根据地。

要北渡长江，就要经过赤水城，但川军已经个旅，先于红军到达赤水城，而且川军郭勋祺部，也紧跟在红军背后。

川军打红军并不积极，他们虽然讨厌红军，但更担心老蒋的中央军。

唯恐老蒋借口“追剿红军”，大军进入四川，来个一吃二（吃完红军吃川军）

只要红军不去四川，川军也乐意摸鱼，但现在红军摆明了要去四川，川军也开始紧张了。

****02****

毛委员在土城行军途中，观察了沿途地形，发现道路两边是山谷地带，是打伏击的好地方。

川军咬得很紧，但正因为紧跟红军，没有和中央军主力在一起，在战略上属于孤军深入。

红军可以利用土城沿路的有利地形，集中优势兵力，围歼川军郭勋祺师。

只要打疼了川军，就能震慑其他尾随的国军，让他们不敢离红军太近，从而摆脱背腹受敌的困境。

何况土城是赤水河东岸的重要渡口，只有拿下土城，才能北渡长江、进入四川和红四方面军进行战略会师。

更重要的是，这也是遵义会议后的第一仗，直接关系全军的士气。

一旦打输了，不但广大红军指战员会对前途失去信心，先前被迫下台的博古，恐怕也要反攻倒算。

鉴于“三人团”集权过重，将红军家底赔个底掉，遵义会议后，红军又走向另一个极端，在战事上实行民主表决。

任何军事提议，都要全体政治局委员讨论通过，并周公最后拍板。

战场瞬息万变，战机稍纵即逝，很多局委完全不懂指挥，这种决策机制显然是矫枉过正，没多久被废除了。

但打土城镇的时候，还是这个表决机制，因此当政治局通过毛委员的提议后，土城战役也就打响了。

1935年1月28日凌晨，细雨蒙蒙，寒风刺骨。

红三、红五军团及干部团，在彭老总、尚昆同志的指挥下，从南北两面向进占枫村坝、青杠坡的川军模范师、独立第4旅各3个团发起突袭。

****03****

尽管遭遇夜袭，敌情不明，但郭勋祺部训练有素，且官兵多为老乡，内部凝聚力强，并没有一溃而散，而是借助地形，顽强抵抗。

而红军战士也是拼死冲锋，战斗很快就进入白热化，双方在黑灯瞎火中冒雨厮杀，枪炮的闪光时不时照亮天空。

经过激战，红军击溃川军一部，但川军战斗力比较强，其主力仍在顽抗。

郭勋祺甚至跳进战壕，亲自鼓舞士气：我们现在有钱有粮，有吃有喝，全靠手里的枪，仗打败了，枪端不稳，就分文不值，连叫花子都不如！

显然，老郭是懂国军心理建设的，发言直击要害：仗打输了，荣华富贵都得丢，从大老爷变成要饭的！

由于敌军顽抗，战事胶着，为了加速战斗，按照中革军委的命令，红一军团第二师和干部团，也投入战斗。

干部团在红军序列中，属于一支特殊武装：他们负责保卫党中央和军委，同时为部队储备、培训和输送干部。

由于该部所有人员全是干部，属于军队的火种，不到万不得已，不得参加战斗。

当时毛委员和周公亲自去干部团作动员，鼓励干部团奋勇杀敌，可见形势何等紧张？

战至黄昏，红军虽给予川军重大杀伤，但始终未能全歼，双方形成拉锯对峙。

更严重的是，红军从抓获俘虏的番号中发现，之所以始终打不动，原来的情报有误！

敌军不是4个团6000多人，而是6个团1万多人，还有后续部队，且装备精良，战斗力很强。

据当年在军委总部任作战参谋的孔石泉回忆：

我们在土城那一仗没有打好，因为对敌人估计不足。

敌人的发报我们收到了，但把‘旅'翻译成了‘团'，因此估计敌人是两个团的兵力。

如果知道是旅，就不会打。

（开战）以后伤亡很大，不能不走了，是我们自己撤退的，只打了个击溃战。

“旅”和“团”这一字之差，导致情报失误，战事失利，让毛委员痛心疾首，心如刀割。

但毛的领袖能力，也就体现在这里：面对失败，他既不埋怨，也不纠结，而是迅速从不利局面中寻找有利因素。

****04****

既然敌军兵力超过红军，战斗力又强，伏击显然行不通，一旦天亮，敌军极有可能凭借优势兵力进行反扑。

于是他急令红一军团红二师火速返回增援，变被动为主动。

果然如他所料，天亮后敌军疯狂反扑，在增援部队尚未赶到的几个小时里，敌军步步进逼，甚至打到了中革军委的指挥部前沿。

指挥部前面是敌军，背后是赤水河，既无险可守，又无路可退，战局十分不利。

在这紧急关头，毛委员没有选择固守待援，反而命令陈赓率军委纵队干部团，发起反冲锋。

临危受命的干部团奇兵突出，猛打猛冲，打得敌人措手不及，很快就被打退下去。

毛委员在山上用望远镜看到这个情景，兴奋地对身边人员说：打得好！陈赓可以当军长。

当晚，毛委员提议召集政治局几个领导人开会。

会议根据各路国军正奔集而来，进行围堵的新情况，判明原定在这里北渡长江的计划已不能实现，决定迅速撤出战斗，渡赤水河西进。

这次战役打得并不好，是一场败仗，因此凯丰在会议上说，“老毛也不行”。

博古也跃跃欲试，觉得自己受了骗，老毛也不会打仗，自己不该让位，在会议上说：看起来，狭隘经验论者指挥也不成。

事后，他又对周公说：你是党内对军事指挥下最后决心的拍板人，不要都听毛的，他也不灵。

而郭勋祺也成了极少数，能让毛吃亏的国军将领，更有意思的是，后来的十大元帅，有七个参与了土城战役，想必解放后郭勋祺回忆此事，必定百感交集。

土城战役因为一字之差失败了，但也正是没打赢，才让红军才西渡赤水河，从而变被动为主动，由挫折走向胜利，拉开了四渡赤水的战幕。

最终成就了毛委员军事生涯中的“得意之笔”，创造了中外战争史上的奇迹。

毛曾如此点评：

土城这一仗，由于及时渡过了赤水，摆脱了尾敌，改变了被动局面。

部队果断地变为轻装，甩掉了包袱(笨重物资)，行动更自由了，更能打运动、游击战了。

这又为后来三次渡赤水打胜仗，创造了有利条件。

但改开后，也有人对此持否定态度。

认为土城战役不该打，更不该与川军决战，将本来放水的川军逼成了强敌。

因为郭勋祺本人一直对红军态度友善，他和我军多位高层将领关系密切，“四一二”反革命政变后，还保护过许多共产党员，是国军有名的“亲共分子”，后来还因为“通共”被老蒋软禁。

他虽然尾随红军，但不想为老蒋卖命，在土城之前从未攻击过红军，不该和他彻底翻脸。

这个观点该如何评价，就看怎么理解历史了。
