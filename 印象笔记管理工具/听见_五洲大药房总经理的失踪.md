# 听见|五洲大药房总经理的失踪

听见|五洲大药房总经理的失踪
==============

原创

上海宝山

上海宝山
*1月27日*

**馆藏故事之五洲大药房总经理项松茂** 音频： *00:00* / *13:21*

一位公司总经理的失踪，成了上海滩的大新闻。它和“一·二八”事件的爆发，同时登上了第二天上海所有报纸的头版。他是谁，他是一个怎样的人，究竟发生了什么事，如此牵动着所有上海人的心？

![](.evernote-content/864C4567-FEB4-40B2-9F75-5298E09E313B/EEA80A7D-E590-40FB-B7D3-FC35CD46DDCC)

图：项松茂（来源：上海淞沪抗战纪念馆）

他叫项松茂，时年52岁，仪表堂堂，是上海最负盛名的五洲大药房总经理。14岁，项松茂在苏州学徒。因处处留心，勤奋好学，三年学徒期满店主就提升他负责店里的账务，这段履历为他日后投身医药业铺平了道路。

1900年，在上海中英药房担任经理的舅父介绍项松茂进入中英药房担任会计。

![](.evernote-content/864C4567-FEB4-40B2-9F75-5298E09E313B/2B811AE4-71E6-423D-BCEF-CED41FCC1213)

图：项松茂生平介绍，上海淞沪抗战纪念馆馆藏（此系项松茂其自出生至1930年的生平事迹介绍手稿。）

![](.evernote-content/864C4567-FEB4-40B2-9F75-5298E09E313B/9BF0F7DD-553B-4E88-99CD-CC9F82567E6E)

讲解员

中英药房在上海医药界地位显要，有繁忙的药品及原料进出口及转销业务。这使项松茂眼界大开，在做会计工作的同时，对于药品的经销流程也有了进一步了解。项松茂工作极其认真，又善于思考，中英药店到汉口创设分店，就让他顺理成章出任经理。项松茂上任以后，深入调查汉口医药市场，积极开拓业务，使汉口分店成为中英药房新的利润源，他本人也成为汉口医药业中的佼佼者、武汉三镇工商界的头面人物。

31岁之际，项松茂受聘担任五洲大药房总经理。这五洲大药房，全称为“五洲大药房股份有限公司”，1907年由著名实业家夏粹芳、黄楚九及药剂师谢瑞卿在福州路所创办，是一家知名的集药品、化妆品和保健食品生产与销售于一体的大型企业。

![](.evernote-content/864C4567-FEB4-40B2-9F75-5298E09E313B/35861373-027B-44EB-BAE4-AED7B7758059)

![](.evernote-content/864C4567-FEB4-40B2-9F75-5298E09E313B/C53A22C5-1B70-4B24-AC03-4BB543FA639B)

图：民国时期五洲大药房部分药品的宣传价目表，上海淞沪抗战纪念馆馆藏。

![](.evernote-content/864C4567-FEB4-40B2-9F75-5298E09E313B/9BF0F7DD-553B-4E88-99CD-CC9F82567E6E)

讲解员

用今天的观点来看，项松茂在五洲建立了现代意义上的企业管理制度：在董事会、总经理之下，有店务、厂务、店厂联席会议和技术会议，又按生产、营业、财务、管理四个系统设置层级部门。公司产权明晰，管理严格、制度完善。资本总额一再增长，职工由最初的30余人逐年递增至400余人。

项松茂把“勤俭”作为办企业的方针。先将店里的豪华陈设变卖，充为营业资金，又邀请和钱庄熟悉的朋友担任副经理，取得金融资本的支持。项松茂对自己和家人也秉持节俭之风，他的内衣总是补丁又补丁，每餐饭只有一个荤菜。一条吃剩的鱼，他都会关照家里留着，下顿再吃。

项松茂的梦想不仅是建立一流的医药贸易企业，更要创建中国自己的制药企业。他说，“贩售外货，实仰人鼻息，不过拾其余汤残羹。必须振兴实业，发展我国的民族制药工业，自制新药，与外货抗衡，始能达到堵塞漏卮，挽回利权的目的。”

![](.evernote-content/864C4567-FEB4-40B2-9F75-5298E09E313B/2250FA16-62E1-41F4-A7F5-ECA68E98800C)

图：五洲大药房店铺分布情况

![](.evernote-content/864C4567-FEB4-40B2-9F75-5298E09E313B/9BF0F7DD-553B-4E88-99CD-CC9F82567E6E)

讲解员

在到国外联系业务时，他对于药厂考察格外仔细。经过他和五洲同仁们的不懈努力，中国在西药上开始不再单纯依靠进口。五洲按照各国药典的不同规格，用中草药炼制酊剂软膏；自制牛痘疫苗、醚精、硫酸低铁、纳夫他林、柠檬酸等化学药用原料；仿制德国“蛇牌”外科手术器械和医院设备，成为我国外科手术器械制造的鼻祖。

五洲在项松茂的主持经营下，还对药房原有经营模式、销售方式等进行了较大改进。他将药房迁址至福州路和河南路口的闹市中心，加强店面形象展示和商品的宣传销售。他特别注重商标名称和图样设计，五洲旗下的商标如五洲牌爽身粉、荷叶荷花牌肥皂、固本肥皂等都深受国人喜爱，是当时的潮牌好物无疑了。

![](.evernote-content/864C4567-FEB4-40B2-9F75-5298E09E313B/D71DB3F5-9A57-4379-A25F-ED0169BEE967)

图：固本肥皂，上海淞沪抗战纪念馆馆藏。（此系1922年项松茂斥巨资盘进原德商“上海固本肥皂厂”，改名“上海五洲固本皂药厂”后生产的国产固本肥皂。）

项松茂看得远，见得广，想得透彻。他深知，一个商人在国家有难时应尽的责任是什么。他从商近二十年，但他个人的经济情形也仅够一家温饱而已。而对于爱国事业，则无不竭力帮助着，比千万百万富翁所尽的力量更多、更大！

早在第一次世界大战爆发和“五四运动”期间，项松茂任总经理的五洲公司与中英、中法、华英、太和等药房联名在报纸上公开发表不进日货的声明。1930年，我国多地发生水灾，灾情严重，项松茂倡仪筹款救灾，并以五洲药房一星期门市营业收入总额的10%提作赈济之用。

1931年“九·一八”事变爆发后，全国人民同仇敌忾。项松茂加入了抗日救国会，成为该会会员之一，并在厂内编组义勇军一营，自任营长，聘请交通大学军事教官每日施以军事训练，积极备战。面对蒋介石“攘外必先安内”的不抵抗政策，项松茂十分愤慨，在生生美术公司印刷的抗日月历上写下“煮豆燃萁，内争可耻”八个大字。爱憎分明的立场溢于言表，表现了强烈的民族意识。

1932年“一·二八”淞沪抗战爆发，十九路军奋力抗击入侵日军，伤亡极大。项松茂接受下生产军用药品的任务，亲自督促，日夜加班赶制药品，努力供应前线急需。

![](.evernote-content/864C4567-FEB4-40B2-9F75-5298E09E313B/23E3A2DC-38E2-4D21-8D52-42880879C291)

图：五洲大药房被日军蛮横搜查，11位五洲大药房店员被全员带走

![](.evernote-content/864C4567-FEB4-40B2-9F75-5298E09E313B/9BF0F7DD-553B-4E88-99CD-CC9F82567E6E)

讲解员

当时，五洲大药房位于北四川路老靶子路（今武进路）口的第二支店，毗邻战区。“一·二八”事变当天傍晚的时候，有日本军车行驶经过这家店附近的街面，遭到我方爱国志士的狙击。第二天上午，一群日军和日本浪人不由分说地包围住药店，强行闯入搜查，发现不少义勇军的制服和抗日宣传品。他们以此为由，将11位留守的店员全部带走。

项松茂闻讯义愤填膺，赶到二支店。一名日军站岗士兵上前招呼，原来是相识的日本商人小山。项松茂寄以希望，赶紧请小山帮忙探问被捕店员下落，设法协助营救。小山答应了，两人约好次日原地见面。他的同事们都劝说他别去太危险了。他说：“11位同事危在旦夕，我不去营救，如何对全公司负责？我若贪生怕死，还算什么总经理？”  
    当夜，项松茂召集工人，报告十九路军胜利的消息和调查同仁踪迹情况，让工人不要慌张。他说：“我全家在此，不迁一物，明日本厂照常开工。”

第二天，也就是30日下午，项松茂带着职员朱灿如再去二支店。车走到北四川路时街上戒严了，项松茂推门下车，走到二支店却没见到小山。日本便衣尾随进店，项松茂没多想什么就出示了自己的名片，要求保释被捕店员。日本便衣答应了，就让朱灿如先回总店找人修玻璃，自己留店等候。

但等朱灿如等人返回时，店门已关闭。他们想进去却被日军阻止，并被告知项松茂已去司令部了。他们回到总店，把情况告知了项家长子项绳武。项绳武当即走访很多关押华人的地方都没找到父亲。31日，项绳武与公共租界巡捕房商量好，与日籍警探去日本领事馆交涉。日籍警探单独进了领事馆办公室，过了很长一段时间，失望地走出来，还埋怨说：“你父亲何故提倡抵制日货，组织义勇军？领事馆不得干涉军部，实无能为力。”

2月1日，项绳武找到较有声望的日本朋友桑野去日军俱乐部打听。看见日本军官声色俱厉地问桑野，桑野接连鞠躬，汗流满面。出来后，桑野急忙拉着项绳武往外走。上车才说，日军查明父亲是抗日委员会成员、义勇军营长，店中藏军服，自己送上门来，已押送到江湾司令部去了。他又说：“刚才军官问话，我不敢说你是项松茂的儿子，只说是五洲药房派来探听消息的人。”

这位中国的大实业家，就此在“一·二八”淞沪会战的战火之中失踪。家人和朋友四处打听，杳无音讯。数年后，他的家人才通过可靠渠道获知，项松茂与那11位店员于1月31日那天清晨被枪杀于日军军营，并被毁尸灭迹。他在生命的最后一刻，慷慨从容。日本军官审问时对着他咆哮怒吼：“你，还敢抵抗我们吗？谁若抵抗，我们就杀了谁！”他威武不屈，一字一顿地说道：“杀便杀！中国人，不爱中国爱什么！你们自己清楚地想一想，日本和中国同文同种，不好好想些睦邻友好的方法，倒以军队占领我国土，屠杀我民众。咳！我看你们日本不会有什么好下场！”

遇难那一天 ，是1932年1月31日。五洲董事会、店全体员工为永远记住项松茂先生和11名店员在1月31日的英勇献身，在店徽、厂徽上加刻“131”字样，并把试制出的牙膏也用“131”作为商标。这个商标含义深刻，不仅表达中国人对日寇侵略中国的愤怒，也表达出五洲全体职工团结一致反对日本军国主义暴行的决心。

四年后，一幢“远东第一”的、十层楼高的药业大厦，在原来五洲总店的地基上落成。人们在大堂中为项松茂举行公葬，设立了他的衣冠冢。

后来项家长子项绳武继承父业，他在整理遗物时发现父亲生前写的一幅自勉联：“**平居宜寡欲养身，临大节则达生委命；治家须量入为出，徇大义当芥视千金。**”令人唏嘘。黄炎培专门作《项松茂先生传》，记述下那激奋人心的最后一幕：“**敌盛怒问：若店曷为藏服械，若敢抗御，杀无赦。先生直立愤然答：死则死，中国人爱中国，份也。**”
