在阅读、摘录和批注过文档后，我们得到的是包含了知识重点的笔记卡片。然而这还只是学习的第一步，接下来我们要将这些零散的笔记卡片，以思维导图的形式，构建成一套完整的知识体系架构。

思维导图是由托尼·博赞发明，作为辅助思考、表达思维的工具受到各个群体的喜爱。作为学习者，思维导图以一个主题为中心，辐射所有相关的知识点。而在 MarginNote 中，每张笔记卡片就代表着一个知识点，将它们组织成思维导图的过程，也就是梳理、理解这些知识点的过程。

创建脑图笔记本
-------

```
本节所涉及的功能：

* 新建脑图笔记本

* 管理关联文档
```

在文档模式中，每一份文档被放在特定文件夹中，而笔记卡片是与文档绑定在一起的，即每份文档中的笔记卡片，是不能统一起来整理的。但是在实际的学习中，如果是一门课程，往往会有好几份甚至上十份与之相关的文档，甚至这些文档会被放在不同的文件夹中。这时就需要一种工具，可以将多份文档放在一起，方便将其中的笔记卡片汇集起来，这就是**脑图笔记本**。

![](.evernote-content/99CBFBD0-C6AD-4E97-ACAC-8CF045AB0C52/4AF86212-9A10-464D-B1A3-B302856CD56D.png)

多份文档组成的脑图笔记本

在任何一份你想创建思维导图的文档中，点击右上角菜单中「转到学习模式」，MarginNote 就会提醒你「笔记本学习升级」，创建一个新的脑图笔记本。创建成功后，就可以在学习模式的界面中，看到相应的笔记本。转到学习模式后，可以看到应用界面变为两栏，左侧是思维导图的操作区，右侧是熟悉的文档阅读界面（你也可以调换两者的位置，点击右上角菜单中的「文档置于左侧」）。

![](.evernote-content/99CBFBD0-C6AD-4E97-ACAC-8CF045AB0C52/40B1ED03-9CD2-4618-887A-6881D5E34CF4.gif)

新建脑图笔记本

在正式创建思维导图之前，最好先**将与主题相关的文档添加到笔记本之中**。点击右上角的「管理」，会列出已关联文档（即已添加到脑图笔记本中的文档）。在「文档库」列表中列举了 MarginNote 内所有的文档，点击右侧的加号，就可以将文档加入脑图笔记本。在管理界面中，也可以选择导入新的文档。

添加笔记卡片
------

```
本节所涉及的功能：

* 添加到脑图

* 自动添加到脑图
```

当我们创建好脑图笔记本，并把文档加入其中，却发现在左侧的界面中，并没有显示我们之前在阅读文档时创建的笔记卡片，这是因为我们还没将笔记卡片加入脑图中。

![](.evernote-content/99CBFBD0-C6AD-4E97-ACAC-8CF045AB0C52/B9A0DA42-C8B7-433B-B990-6C61B2096EE6.gif)

添加卡片到脑图

点击`摘录设置 > 显示未加入脑图的摘录`，就会显示出所有尚未加入脑图的笔记卡片。选择你想加入到脑图中的卡片，会自动定位到卡片在文档中的位置，再单击右上角的`添加`，就可以将笔记加入脑图。

如果文档里的笔记卡片数量不多，逐张添加的方式尚可以接受，但是卡片的数量一多，恐怕就有些繁琐。

![](.evernote-content/99CBFBD0-C6AD-4E97-ACAC-8CF045AB0C52/1E3F240D-DE16-4E12-AE8C-F9ABCDF2D446.gif)

自动添加到脑图

其实在学习模式中的摘录设置中，可以启用「自动添加到脑图」的选项，启用这个选项之后，在文档中进行摘录，自动生成笔记卡片之后，都会自动被添加到脑图中。所以为了方便脑图的整理，你也可以选择在学习模式中阅读文档，这样在笔记卡片数量较多的情况下，可以节约不少添加卡片的操作。

给卡片分组
-----

```
本节所涉及的功能：

* 组

* 大纲
```

当笔记卡片出现在脑图中时，默认会将同一文档内的笔记卡片，自动排列在一个框内，左上角还有文档的名称。这个框是用来做什么的呢？

在脑图界面中，一个框代表一个组。「组」是 MarginNote 中方便整理的工具，在一个组内的卡片，会根据卡片的数量和大小，自动进行对齐，保证卡片不会显得凌乱。

![](.evernote-content/99CBFBD0-C6AD-4E97-ACAC-8CF045AB0C52/661BB693-5B41-4B70-8312-21DA40414090.png)

组在脑图中是一个节点

除了视觉上的作用，其实一个组和一张笔记卡片一样，都是**脑图上的一个节点**。点击「搜索和大纲」，会以大纲的形式展示脑图的层级。可以看到，组和笔记卡片一样，可以作为节点，移动到脑图的任意位置。

![](.evernote-content/99CBFBD0-C6AD-4E97-ACAC-8CF045AB0C52/D8A6147F-69FC-44A5-B32D-870E68A9BBB4.png)

自动插入位置设置

在添加卡片到脑图时，会自动生成组。点击`摘录设置 > 自动插入位置`，可以看到默认是按文档分组。除此之外还有两种排式：分组（按文档目录）和选定位置。「分组（按文档目录）」和默认设置「分组（按文档）」类似，只有对于有目录的文档，卡片才会按照目录的层级进行排列，目录中的层级对应成为不同的组。而启用「选定位置」选项，添加笔记卡片到脑图时不会生成任何组，而是直接显示在界面中央，成为一个根结点。

![](.evernote-content/99CBFBD0-C6AD-4E97-ACAC-8CF045AB0C52/8E7B1905-0D5E-49A7-AB35-4A31149AA269.gif)

重组

当然你也可以手动创建组。点击「选择」，选中你想放在一组的笔记卡片，然后点击「重组」，就创建了一个新的组，双击左上角，就可以为组重命名。

理解了组的本质和作用，就可以灵活运用它，来合理构建脑图。

我们知道，脑图一般都是以一个关键词为中心，其他脑图应用也会默认创建这个中心关键词，而在 MarginNote 中并没有默认的关键词，这就需要自己去梳理。

以我自己的使用习惯为例，会先选中所有的卡片，使用上文说到的重组功能，新建一个组，而这个组就成为了关键词。再以这个词为中心，将笔记卡片分为不同的组，例如「毕业论文」是关键词，下一个层级就是「论文」和「大纲」组，前一个组用来梳理参考文献中摘录的笔记卡片，后一个组用来写自己论文的大纲。

组织笔记卡片
------

```
本节所涉及的功能：

* 合并

* 概要

* 链接
```

经过给笔记卡片分组的过程后，我们就可以把目光聚焦在一个组内，来看看如何将笔记卡片排成一个我们熟悉的脑图结构。

在 MarginNote 中组建脑图的操作与其他应用基本一致：选中笔记卡片按回车，可以创建同级的节点；选中笔记卡片按 Tab 键，则可以创建子节点，拖动笔记卡片到另一张卡片上，会成为它的子节点。与其他应用稍有不同的是，将笔记卡片拖动到父节点上时，会将这两张笔记卡片合并起来。经过这些操作，就能将笔记卡片以最常见的脑图形式组织起来。、

![](.evernote-content/99CBFBD0-C6AD-4E97-ACAC-8CF045AB0C52/5579844B-BB3A-4B4A-A19C-CCC8BF8D881B.gif)

完成基本的组织后，我们来了解下在 MarginNote 的脑图中，有哪些进阶的功能，让脑图的结构更加丰富。

首先是前文提到过的**合并**功能，不局限于父子节点，任何两张笔记卡片，甚至是两个组，都可以进行重组。点击「选择」，选中要合并的组或卡片，再点击「合并」即可。需要注意的是，两张笔记卡片合并会形成新的笔记卡片，而两个组的合并会创建出新的脑图。

![](.evernote-content/99CBFBD0-C6AD-4E97-ACAC-8CF045AB0C52/BDAE2017-75F7-477B-A91D-DA6B4925CC0A.gif)

合并

脑图的结构是严格的树形结构，笔记卡片以及组之间也只能是同级或父子级的关系，如果我们想给几张有联系的笔记卡片写一些概述，描述它们的共性，无论是重组成新的组和还是合并都不太合适，会影响到脑图的结构。这时就可以使用**概要**功能。

![](.evernote-content/99CBFBD0-C6AD-4E97-ACAC-8CF045AB0C52/65891471-127E-463A-83DC-A764DA3E4B9F.gif)

概要

点击「选择」，选中多张卡片，再点击「概要」，可以看到有新的卡片被创建出来，但它是以虚线的形式与卡片进行连接。打开脑图大纲，会看到它其实是这些卡片的同级节点。所以我们就以一种不改变脑图层级的方式，新建了一张卡片，用来总结某几张卡片的内容。

脑图的结构表明了卡片间的关系，但两张在脑图上相隔很远的卡片，可能互相有关联。如果要将这种关系在脑图中表现出来的话，就要用到**链接**功能。

![](.evernote-content/99CBFBD0-C6AD-4E97-ACAC-8CF045AB0C52/614CA3C3-C573-499D-B681-0FB9312DC7C0.gif)

链接

点击「选择」，选中一张卡片，再点击「链接」，卡片上就会出现链接的图标，将图标拖动到目标卡片，两张卡片之间就会有一条虚线连接，同时在卡片上会出现「链接卡片到快捷方式」，点击它就会立刻跳转到目标卡片。

这种方式不会对脑图结构做任何改变，却可以丰富卡片之间的关系。

其它功能

经过前面几步，一份层次清晰、分类合理的脑图应该已经形成。最后，我们来看看 MarginNote 里为脑图准备的功能。

### 分支样式

前文说到过，组其实也是脑图上的一个节点，组中的卡片其实可以视为组的子节点。在脑图窗口中，可以随时切换组和树形结构。选中一个组，点击工具条上的「分支样式」，就能选择是以组或者树形结构来展示卡片。

![](.evernote-content/99CBFBD0-C6AD-4E97-ACAC-8CF045AB0C52/8025261B-6170-4757-A7EB-403EE3E966EB.gif)

不同的分支样式

### 焦点

脑图是帮助我们梳理知识点的工具，但有时候笔记卡片太多，导致思维导图的层级太多，反倒会干扰我们查看卡片内容。

![](.evernote-content/99CBFBD0-C6AD-4E97-ACAC-8CF045AB0C52/A5EEFC6E-F2CD-48C7-AAAE-57EDFEF82030.gif)

切换焦点

所以在需要查看某一层级脑图时，就可以使用「焦点」功能。选择组或笔记卡片，点击工具条上的「焦点」，MarginNote 就会只将选中的卡片或组以及它的子节点显示出来。这样，我们就可以缩小范围，方便我们阅读笔记卡片的内容。

进入焦点模式后，脑图窗口的左上方的节点窗口就会显示出目前的焦点，及其焦点的父节点，方便我们随时切换到其他层级中。点击「主脑图」，即可算作推出焦点模式。

### 标题与重点

在上一章中，我们介绍了如何编辑笔记卡片，丰富它的内容。其实在学习模式中，还有两个功能可以编辑卡片。

在摘录文档内容的时候，文本会成为笔记的评论，此时的卡片是没有标题的，而很多时候，摘录文本内就有适合作为标题的内容。此时就可以先点击整体摘录文本，在选中其中的部分内容，在弹出的工具条中，就会出现「设为标题」的选项。

![](.evernote-content/99CBFBD0-C6AD-4E97-ACAC-8CF045AB0C52/942CA0FC-9C7E-4C84-BA9D-861EE1D501EF.gif)

设为标题和划重点

在工具条中，还有一个「划重点」选项，将一部分文本设为重点，就会改变它在笔记卡片中的样式。不仅如此，在复习模式中也会起作用，在下一章我们再来具体了解。

小结
--

在这一章中，我们以脑图笔记本-组-笔记卡片的线索，梳理了一遍在 MarginNote 中组织脑图的流程。将笔记卡片整理成脑图的过程，正是我们理解笔记内容时必不可少的一步。

然而，想要真正吃透书本中的内容，必须花时间将它们实实在在地记在我们脑中，变成我们自己的知识。下一章，我们就来了解复习模式，实现这最后一步。
