# "中国疫苗，让我们看见一个远离美国胁迫的世界"

---

"中国疫苗，让我们看见一个远离美国胁迫的世界"
-----------------------

原创 鞠峰 观察者网

![](.evernote-content/3678A49B-EA1B-4E44-9FB1-BCF8FCE601D7/E08FBC0E-08A6-492C-9069-15FBE78990A0.png)

**►****文 观察者网 鞠峰**

“中国的‘疫苗战略’让我们得以瞥见一个多极的世界，一个远离美国令人窒息的欺凌胁迫的世界。”

印度的自由媒体人麦特雷亚·巴卡尔（Maitreya Bhakal）7月7日在今日俄罗斯（RT）发表评论，指出中国和西方国家在为欠发达世界提供疫苗援助上采取完全相反的政策。在西方国家忙着囤积疫苗时，中国积极援助发展中国家，国际影响力与日俱增。

![](.evernote-content/3678A49B-EA1B-4E44-9FB1-BCF8FCE601D7/051E6568-A4FD-4FC2-90E3-976E18137D6E.png)

印度自由时评作家 麦特雷亚·巴卡尔

中国为开创一个更好的世界秩序提供了可能性。作者开篇写道，有两种提升国际影响力的方式。一种是西方的方式（Western way），是通过暴力、帝国主义和硬实力，建立殖民地，摧毁当地文化，掠夺财富，炮舰外交等等。

而另一种方式，是用思想和价值观的引力，来吸引其他国家，帮助需要帮助的人。怎么做比怎么说更重要。这也被称为“软实力”。

巴卡尔写道，中国和印度数千年来都是采用这种方式提升影响力。作为一个印度作者，他用自己的国家举例称，印度在非洲留下良好声誉的原因之一，是因为印度为那里的民众提供了负担得起的药物。印度制造的仿制药比西方制药公司产的要便宜得多。

而提到中国，文章称中国在这次疫情中创造了一条“疫苗丝绸之路”。对于世界上大多数国家、特别是发展中国家来说，中国为他们提供了拯救生命的疫苗，以及医疗设备，价值不可估量。而富裕的西方国家则忙着囤积疫苗。几个月以来，西方国家自私地拒绝与发展中国家分享疫苗，即使本国库存已经够用。

目前，低、中收入国家中使用的疫苗大多是中国疫苗。“中国对各国的供应和援助，让我们得以瞥见一个多极的世界应该是什么样的。”

“数字比西方的政治宣传更具说服力。（Numbers talk louder than Western propaganda.）”文章指出，据权威数据网站OurworldinData统计，**世界范围目前注射了约32.9亿剂次疫苗，其中13亿剂是在中国接种的。还有约100个国家准备或已经接种累计4.8亿剂的中国疫苗。**

不过需要指出的是，作者虽然对中国的疫苗援助给予肯定，但也难免同部分西方国家一样，将中国的对外援助归为所谓“疫苗外交”。事实上，中国从来不搞“疫苗外交”，外交部多次指出，中方向国际社会提供、捐赠疫苗的目的，是促进疫苗在全球的公平分配和使用，争取让更多人受益。中方正努力践行新冠肺炎疫苗作为全球公共产品的宣示，根据其他国家特别是发展中国家的需要，提供力所能及的支持与帮助。

巴卡尔指出，中国在控制国内疫情方面可能是世界上做得最出色的国家，而且正在帮助一些南亚国家对抗病毒。

文章称，在南亚，大多数国家已经无法收到印度生产的疫苗，他们也同情印度政府的处境，印度软性禁止疫苗出口的做法可以理解。这些国家开始从别的地方获取疫苗——中国正在逐渐填补空缺，而且可能是惟一一个有能力这么做的国家。

印度的多个邻国正在考虑在本地生产中国疫苗。“印度的第二波疫情正在缓慢收束，国家正不可避免地迎来第三波，因此我们伸出‘疫苗友谊之手’还得等一段时间。”巴卡尔最后写道。

麦特雷亚·巴卡尔是印度自由时评作家，现在为今日俄罗斯、CGTN、中国日报等媒体撰写英文评论，长期关注中国议题。他在推特等多个平台上的个人资料中都放着一句话，“如果你要宣传什么事，找印度；如果要办成什么事，看中国。（If you want something said, ask India. If you want something done, ask China.）”

![](.evernote-content/3678A49B-EA1B-4E44-9FB1-BCF8FCE601D7/216B7F40-C668-452F-B71A-FFE0FE942C8C.png)

截自巴卡尔推特资料

外交部国际经济司副司长郭学军9日在接受采访时说，截至目前，中国已向全球100多个国家和国际组织提供了5亿剂新冠疫苗和原液，相当于当前全球新冠疫苗总产量的六分之一。
