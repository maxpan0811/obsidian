这些方法，让 Windows 自动更新不再烦人

Windows 的更新总是让人深恶痛绝，网上搜索「如何关闭 Windows 更新」，百度和谷歌都有好几千万条结果，用英文搜更夸张，竟然有二十几亿条。可见这个 Windows 的更新策略一般人都受不了。不过即便这样，我并不建议完全关闭更新，毕竟它的确可以修复 bug，堵漏洞，让电脑更安全。其实大家有意见主要是因为会被强制自动重启，所以或者，我们可以先试试这样。

![](.evernote-content/9886745B-0FE7-4EBC-911A-A2E150654F5E/AF6C130D-F1B7-4F8B-803C-E0742CBFC0F8.png)

设定允许重启的时段
---------

其实我们可以设定允许重启的时间段，在设置的 更新和安全 这里，有一个 更改使用时间段，可以告诉系统，早上八点到晚上十一点，是我正常使用电脑的时间，你不要乱来给我突然重启。或者你也可以开启这个选项，系统会根据你每天的实际使用时间设置一个相对智能的时间段。

![](.evernote-content/9886745B-0FE7-4EBC-911A-A2E150654F5E/6B994093-906A-4550-9FF9-FA347DF806AA.png)

在这些设置好的时间内，Windows 不会自动重启设备。

如果你还是没什么安全感，担心某一天熬夜玩游戏没注意，还是被自动重启了。那么可以再到 高级选项 这里，开启 更新通知。以及将上面这一条给关掉。做完这几个步骤，Windows 就不会在未经你同意的情况下自动重启了。

![](.evernote-content/9886745B-0FE7-4EBC-911A-A2E150654F5E/0979DEBF-04A4-45F2-80D5-E8C4A440A174.png)

临时暂停更新七天
--------

然后还有这个，可以完完全全暂停更新七天，可以让你保证外出或者参加重要活动时，电脑不会出幺蛾子。如果七天不够，还可以在高级选项里面自定义一个最长为 35 天的期限。不过当这个期限用完了之后，你得至少更新一次才能继续暂停。

![](.evernote-content/9886745B-0FE7-4EBC-911A-A2E150654F5E/F05943FB-3124-46E2-8774-99017D552C90.png)

使用第三方软件完全关闭
-----------

最后，如果你实在想要完全禁止 Windows 更新，也有方法。一是专门的工具，比如我们介绍过的全能优化工具 [DISM++](https://www.chuyu.me/zh-Hans/index.html)，在 系统优化 这部分，就有着相关的设置，你可以设置为 从不检查。

![](.evernote-content/9886745B-0FE7-4EBC-911A-A2E150654F5E/45D0AD7E-B14E-4AC7-B306-0329FEF87989.png)

到服务中关闭更新
--------

如果不想用第三方工具，可以直接到系统的 服务 中关闭 自动更新。方法是在左下角 运行 这里输入 services 然后回车，就能打开 服务管理器。然后在一堆服务中找到 Windows Update 字样，右键查看属性，设置为 手动 或 禁用，就可以了。

![](.evernote-content/9886745B-0FE7-4EBC-911A-A2E150654F5E/A42A2A8B-94F2-497D-87DC-B68DFA6E97F8.png)

小结
--

Windows 自动更新自动重启很烦，但是对于保护系统安全是很必要的。如果你确实不希望被打扰，我们可以在系统中设定允许更新和重启的时间段，避开正常使用的时间。也可以指定某段日期暂时禁止更新。甚至还能通过第三方工具或系统服务级别的设置，完完全全关闭自动更新。希望大家掌握了这些技巧后，Windows 用的更舒心。
