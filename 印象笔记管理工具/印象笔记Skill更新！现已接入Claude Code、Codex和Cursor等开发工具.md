💡「印象笔记Skill」更新啦！

下载地址及安装说明在👇下方链接：

https://guide.yinxiang.com/docs/yxai/skill.html

以下，是本次印象笔记Skill更新详情：

1、新接入AI工具

![](.evernote-content/77CBE144-7C36-458F-8A66-2281EA44FCF0/A7A35855-87EB-4809-9D40-B7A75D6CC14C.png)

新增支持 Claude Code、Codex、Cursor等开发工具。

加上已支持的 OpenClaw 龙虾系列，目前主流的 Agent 和开发工具已基本覆盖。

无论你用哪个工具写分析、写方案、写代码设计，产出内容可以直接写入印象笔记，几乎不再需要中间复制粘贴。

2、笔记写入能力增强

这次更新同步升级了印象笔记Skill的笔记写入能力，无论用哪个工具调用，写入体验都会比之前更好：

* Markdown格式完整保留

![](.evernote-content/77CBE144-7C36-458F-8A66-2281EA44FCF0/53105F5A-B2E0-4467-89B8-022273DA67E9.jpg)![](.evernote-content/77CBE144-7C36-458F-8A66-2281EA44FCF0/25EB4B24-2022-405A-975F-84FCFC0ACB12.jpg)

用Markdown写的笔记内容，存进印象笔记后能正确转换格式。

标题、列表、代码块、引用、加粗、表格——写进去什么样，存下来就是什么样。

Claude Code 分析完代码结构、或者 Cursor 写完一篇技术方案，都可以直接写入印象笔记指定笔记本，「带格式」。

这些在印象笔记Skill更新后，均可“一步到位”。

已是印象笔记会员的用户，只需在常用 AI 工具中将「印象笔记Skill」升级至最新版本，即可体验本次更新；新开通会员的用户，安装「印象笔记Skill」并按指引完成配置后，即可开始使用。
