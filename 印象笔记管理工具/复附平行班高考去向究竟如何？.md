# 复附平行班高考去向究竟如何？

最近有家长问我，能不能聊一下四校平行班的情况，理科班自然是很不错的，平行班去向究竟怎么样呢？

![](.evernote-content/3C9C5490-F7B5-4F22-99D7-B8869E583B97/3BD0714C-5387-4AFE-8BE0-F09D90CC36FF.jpg)

1. 04届某平行班46个人，5个清北，25个复交

2. 19届某平行班40人左右，1个北大，8个复交，3个复交医学院，1个港大，同济华师6个，南大1个

3. 18届某平行班清北2个，15+复交，绿色通道走了多伦多，早稻田，韩国高丽等

4. 24届某平行班38人，无清北，复交录取1/5,其他985 1/5，海外留学1/5

5. 24届某平行班，北大一个，复旦4个，交大7个

了解下来整体情况大概如下：

1. 大多平行班不如理科班

2. 有的平行班也是比较强的，比理科班低不了太多

3. 有些平行班稍许差一些，不过985院校以及强211院校录取率也还是可以的，走绿色通道的一些，去的也都是QS排名不错的学校

![](.evernote-content/3C9C5490-F7B5-4F22-99D7-B8869E583B97/E419D71C-D9B3-4751-9D33-6A0D328F857C.jpg)

24届复附高考数据也是到达了天花板级别，25届作为新政名额分配到校出现的第一届，让我们拭目以待。

另外插播一条小道消息，静安很快要落地一所四校分校，应该就是复附分校了，如果是这样，对静安区绝对是利好的所在，毕竟，市西作为静安区目前最好的一所市重点，还是不太鸡，高考成绩也不算亮眼。
