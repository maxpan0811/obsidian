各位：

国航系列团出新规

一、适用范围：第一国际航段旅行日期在2016年8月15日-2017年03月31日期间的欧洲、美洲、非洲航线以及旅行日期在2016年8月15日-12月31日的澳新航线。

二、定金扣罚标准调整如下：

淡平季时段：除7.13-8.25/9.28-10.3之外的时段

|  |  |  |  |
| --- | --- | --- | --- |
| **淡平季** | 成行率考核标准 | 未达到成行率考核标准相应扣罚金额 | 原政策 |
| 座位取消在 | 欧洲/美洲 |
| 航班起飞前60天以外 | 无 | 无 | 提前35天取消无损 |
| 航班起飞前60天（含）至航班起飞前35天之间 | 70% | 500 RMB/人 | 提前35天取消无损 |
| 航班起飞前35天（含）以内 | 70% | 1000 RMB/人 | 55%，不足的损机票全款 |

举例，原9.18号团100个机位，允许在7.18前免费取消任何数量的机位，如取消50个保留50个，则以50个机位开始计算使用率；

在8.14前可以再免费取消30%，即可免费取消50\*30%=15个机位，超过部分罚500rmb/人；

如8.14号保留35个机位，出票前还可以再免费取消30%的机位，即35\*30%=10个机位，超过部分罚1000rmb/人。

旺季时段：2016.7.13-8.25/9.28-10.3

|  |  |  |  |
| --- | --- | --- | --- |
| **旺季** | 成行率考核标准 | 未达到成行率考核标准相应扣罚金额 | 原政策 |
| 座位取消在 | 欧洲 |  |
| 航班起飞前60天以外 | 无 | 无 | 提前56天取消每取消1个均损1000rmb |
| 航班起飞前60天（含）至航班起飞前35天之间 | 85% | 1000RMB/人 | 56天以内不足85%的损机票全款 |
| 航班起飞前35天（含）以内 | 85% | 2000RMB/人 |

举例：9.28号团原定机位100人，允许在7.28号前免费取消任意数量的机位，如取消50个保留50个，则以50个机位开始计算使用率；

在8.24前可以再免费取消15%，即可免费取消50\*15%=7个机位，超出部分罚1000rmb/人；

如8.24号保留43个机位，出票前可以再免费取消15%，即43\*15%=6个，超过部分罚2000rmb/人。

对我们而言，受影响的是8.15-8.20及9.28-10.3期间原政策属旺季的团队，政策变松了；但其余时段的政策变紧了。

国航的这个新政从2016年7月18日起正式开始实施，7.18日前可以重新确认一遍8.15号之后所有机位，@阿鲍，请组织时间大家做一次讨论吧。如有必要开了培训会也可以，你来安排时间吧。
