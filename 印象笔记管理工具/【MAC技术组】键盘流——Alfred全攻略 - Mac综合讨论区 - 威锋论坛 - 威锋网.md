Results：这个设定你可以添加或删除你想要搜索到的文件，Essentials，Extras都是些定制功能（你想让Alfred搜到就勾选复选框），当然，如果在Essentials和Extras里都没有你期望能够显示的格式（例如dmg格式文件），Advanced就可以解决你的问题，你只要拖拽你想让Alfred搜索到的文件格式（例如dmg格式，rtf格式，你只要拖拽随便一个dmg或者rtf格式的文件就可以设置成功），还有一个Unintelligent，勾选Search all file types可以搜索到所有东西，这个就=Spotlight了，只是这个会降低搜索的效率，容易混淆，不建议勾选。那么，Default Results到底要怎么用呢？你只要Option+空格直接输入你想要搜索的东西即可，你可以在不输入任何keywords即可打开应用程序或搜索文件或文件夹等等（不同于File Search下你要输入空格或者&#39;或者Open或者find才会显示你要搜索的文件，但是Default Results在某种程度上与File Search是有联系的）最下面的Fallbacks：  Setup fallback results指的是，如果Alfred找不到其他搜索结果了，它会自动显示你所设置好的提示你使用网络搜索，例如谷歌，维基百科等。  File Search：先来说说我在上面提到的DR和FS的联系吧：如果你想用Alfred使用File Search——find或者open或者in，你就需要在Default Results里的Search Scope设置可以索引到的路径才能搜索到你想要的文件（默认路径应该足够，如果没有你可自行添加）。也许你没有添加对应的路径，但是你不知道，以致于找不到文件，你可以在输入文字后按住Option键再按回车可以进入Spotlight搜索；按住CMD键加回车会进入Finder中查看此文件；按住Control加回车进入你设定的fallbacks进行搜索。  File Search的重头戏就是文件操作了  在选定自己想要操作的文件后，按Fn或者Control或者Shift都可以进入上图界面。  如果你想多文件操作，你就可以选中第一个文件后按Option+↑或者↓，选完按Option+→同样进入上图，即可进行操作。可以拷贝，移动等等。如果选错文件，按Option+←可以去除刚才选错的文件。按Option+CMD+/进入先前的路径，/和~符号可以进入用户主目录和系统根目录，还有其他定制功能可以自己研究。  在Advanced里还有一个设置使用Shift键可以Quicklook，很贴心的功能~  Web Search：你可以用它快速搜索网络上的东西，当然，你也可以用来快捷上网。  先看如果要添加一个快捷搜索的话，点右下角Add Custom Search  我这里用淘宝搜索举个例子：    Search
