# LyricsX，同时支持 iTunes 和 Spotify 的 Mac 歌词工具丨App+1

---

切换到 Spotify 后，失去了歌词显示功能让我感到很不习惯。辗转几款开源的歌词插件，它们的歌词呈现方式都不够优雅，直到我偶然遇到了 [XQS6LB3A](https://github.com/XQS6LB3A)开发的 LyricsX，一款优雅实用的开源 Mac App。与此前我们介绍过的另一款[同名应用](https://sspai.com/post/36884)相比， 它添加了对 Spotify 的支持，且功能更为整齐精炼。

一切为了优雅
------

LyricsX 有三种歌词呈现方式：歌词窗口、桌面上悬浮的逐行歌词、菜单栏歌词。多样的歌词呈现方式，为你的个人偏好提供了任性的可能。对我来说，菜单栏歌词可谓是「天生优雅」，这样的呈现方式恰到好处地利用菜单栏的空白界面，又不阻碍正常工作活动的界面。

![](.evernote-content/8AA546A6-6FCF-4A1C-9A24-D8B0A63E2188/89CA7500-CBA7-4355-96C7-7480216F53D4.png)菜单栏歌词

当然，对于另外两种呈现方式，开发者也给予也有贴心的设计之处，比如歌词窗口可以选择是否置于顶层，悬浮的逐行歌词可以自定义距离屏幕底部的高度来避免遮挡播放器的操作区等。

![](.evernote-content/8AA546A6-6FCF-4A1C-9A24-D8B0A63E2188/6BC3FA87-E051-4327-A504-0CF1B1F6EB51.png)歌词窗口![](.evernote-content/8AA546A6-6FCF-4A1C-9A24-D8B0A63E2188/FF0A94B6-D3D1-4346-9866-9281A8EB5286.gif)桌面上悬浮的逐行歌词

从实用出发
-----

LyricsX 在实用性上也是值得我夸赞的。从它的 Github 介绍来看，它的歌词源来自六家国内音乐平台，不用担心水土不服的问题。歌词支持手动选择搜索结果，有效解决了 Spotify 中文歌的英文歌名对搜索的误导。

支持 Spotify、iTunes 和 VOX，支持随播放器一同启动和退出，支持繁简转换，支持在截图时自动隐藏歌词，支持自定义歌词……种种「支持」，都是开发者对这款 App 的精心打磨。

![](.evernote-content/8AA546A6-6FCF-4A1C-9A24-D8B0A63E2188/D81E710C-4246-474D-990E-7326B750156A.png)

让我感到惊喜的重磅功能是歌词过滤。在展示歌词的时候，可以自动去掉歌曲名和艺人的名字，还可以去掉盗版歌词源中乱七八糟的网址。过滤内容也允许用户进行自定义，可以说是撒手锏了。

![](.evernote-content/8AA546A6-6FCF-4A1C-9A24-D8B0A63E2188/B704ED2D-7D0F-4A3E-BBB3-CF7A42437FC0.png)歌词过滤
