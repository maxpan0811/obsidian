HyperDrive Thunderbolt 3 USB-C Hub for MacBook Pro
--------------------------------------------------

* 优点：造型美观、接口丰富、速度稳定
* 缺点：会发热、插在电脑右侧时造型有些奇怪、价格不是所有人都能接受
* 售价：Indiegogo $79 / 淘宝 ￥688

**nerd\_du**：自从去年买了新款 MacBook Pro 后，就经常要面对使用转接器的问题，HyperDrive 的这款转接器其实很早就注意到了，但是因为价格缘故迟迟没有剁手。

后来随着需要用到的接口类型越来越多，除了 USB 和 HDMI，还会用到 SD 读卡器，仅仅购买一款官方转接器是不够的。

再加上我本身就很讨厌乱糟糟的线缆，现在经常把自己的 MacBook Pro 插成螃蟹 ，实在是不能忍。所以一狠心，悬在手上的刀还是落下了。

从美观程度上来说，HyperDrive 这款转接器绝对高于官方产品，它提供了和 MacBook Pro 一致的星空灰和银白两款配色，而且色差控制得也不错，看上去就像是 MacBook Pro 的一部分。

![](.evernote-content/AB813B0B-551E-494D-A9FE-6242A0CF710C/FDE045B4-891E-4C98-8FF8-E170DB38C0BE.jpg)

转接器上配有 6 种接口：HDMI、Thunderbolt 3、USB-C、MicroSD 和 SD 卡槽各一个，此外还有两个 USB 3.1 接口，基本上满足了大部分人的需求，由于同时调用了 MacBook Pro 上的两个 Thunderbolt 3 接口，所以速度和稳定性都不错，虽然没试过把所有的接口都插满，不过在同时使用 HDMI、Thunderbolt 3 充电、SD 读卡器和两个 USB 接口的情况下，2K 分辨率的显示器可以正常使用，文件的传输速度也没有受到影响。

我对这款转接器已经非常满意了，真要是说有什么会让人在意的地方，可能就是：

1. 使用时会发热，温度和 MacBook Pro 高速运行时的温度差不多；
2. 官方默认的用法是插在电脑左侧，鉴于两侧都是双 Thunderbolt 3 接口设计，你想插在右侧也不是不可以，但是就会比电脑高出来一截；
3. 根据 Kickstarter 上的网友反馈，有人遇到过传输速度不稳定的情况。

小米盒子 3S
-------

* 优点：价格容易接受，支持大部分国内视频网站，投屏体验较稳定
* 缺点：广告太多
* 售价：小米商城 ￥299

**Oscar Liu**：最近新购置了 iPad Pro，家中有一个戴尔 U2515 显示器，于是就想怎样配合两个设备一起使用。一开始想购买一个 Lightning 转 HDMI 的转换器，后来看了一些购买评论后发现两个问题：一个是显示比例不一致，另外一个是只有 1080p（我的显示器最高支持 2K 分辨率），所以就放弃了这个想法。

我仔细思考了一下使用场景，发现实际上我更多是用显示器来看视频，其实只需要一个能无线投屏的设备就可以了。尽管 Apple TV 是个不错的方案，但考虑到国内的情况，于是就想到了国产的电视盒子，就下手了一个小米盒子 3s。

![](.evernote-content/AB813B0B-551E-494D-A9FE-6242A0CF710C/529BDEC9-BDB9-4660-952E-140D64462840.jpg)

图 / 小米

这个黑色的小盒子有一个 HDMI 接口和一个 USB 3.0 接口，它投屏的方式是使用 DLNA 协议，大部分国内的视频应用都支持（一般会在播放器上有「TV」的标志）。虽然偶尔会遇到无法成功的情况，大部分的时候都还是比较稳定的。

不过，烦人的广告是国产电视盒子无法避免的问题。开机的广告、无法删除的内置应用、打开应用时弹出的广告…… 各种广告让整个使用体验变得很糟糕。投屏播放视频尽管避免了在电视上加载时的广告，但 iPad 视频应用里的广告也只有购买了会员才能跳过。

播放体验方面，虽然视频投到了显示器上，但要想改变清晰度只能回到 iPad 上去操作，并且会重新加载播放，不能继续读取之前的播放进度，这个确实有些麻烦。

在日本使用 Amazon Dash Button
------------------------

**JailbreakHum**：Amazon Dash Button 是 Amazon 出品的一个小玩意儿。一个 Dash Button 对应一款 Amazon 上的特殊商品，按一下 Dash 上的按钮就对这款商品执行了一次「一键购买」。

![](.evernote-content/AB813B0B-551E-494D-A9FE-6242A0CF710C/6D4E1398-ECA9-4D54-8FA5-2D36AAC3D6B7.jpg)

「又没纸了」、「又没洗发水了」、「又没洗衣液了」……

真的很奇怪，现在网购已经这么方便，我们每天都会花大把时间在网上，而家附近也都会有超市。但即便这样，一些生活的基本物品依然是会在特别寸的时候用完，要用的时候叫天不应叫地不灵。

Amazon Dash Button 不能让你在要用卫生纸的时候变出来一卷儿，但是能让你在发现快没纸的时候，按一下第二天就把纸给你送过来。

在日本，Amazon Dash Button 几乎已经覆盖了各方面的日用品。从厨房到浴室，从洗洁精到卫生纸…… 甚至包括食品、饮料，都有雀巢、Evian、Ariel、Elleair 等知名品牌入驻，产品目前 [已经包括 74 种](https://www.amazon.co.jp/gp/search/ref=sr_hi_3?rh=n%3A2250738051%2Cn%3A2408695051%2Cn%3A4752863051%2Ck%3Adash+button&keywords=dash+button&ie=UTF8&qid=1498703891)。

![](.evernote-content/AB813B0B-551E-494D-A9FE-6242A0CF710C/4DD44849-4F8C-43C6-ACAB-CE137BFD09C4.png)

Amazon Dash Button 的价格是 500 日元，这 500 日元算是充值在你的账户里，可以支付你用 Dash Button 进行的消费。

Amazon Dash Button 设计得可挂、可贴、防水。因为它设计的初衷就是发现东西不够的时候马上可以订购一样产品，那么它理想的位置就是在各个生活消耗品的位置，顺手就买。

它的电池不可充电与更换，但是可以支撑 Button 按 1000 次所需的电量。一星期买一次的话，要 7000 天，19 年多电量才能被用完。

除了正常的使用方式，Amazon Dash 也有 Geek 一点的玩法。比如 [可以用它绑定你的 IFTTT 账户](https://medium.com/@welovejeff/hacking-the-amazon-dash-button-to-work-with-ifttt-77fe87520ac3)，这样每次消费完，都可以在我的 Todoist 里添加一个任务，提醒我记得记账。

除了作为生活用品的订购器，国外还有极客将 Amazon Dash 改装成了门铃，当别人按了这个门铃，屋里并不会发出响声，但是会给你发一条短信，让你知道现在门口有人。这样不管你是否在家，你都知道有人来过。

有兴趣折腾的可以看 [这篇文章](https://github.com/initialstate/silent-doorbell/wiki/Part-1.-Setting-Up-the-Dash-Button)。

Blue Yeti USB 麦克风
-----------------

**文刀漢三：**我们编辑部每天都要开早会，审核前一天的稿件以及定下当天要发布的文章。由于 Hum 是远程工作，所以我们需要在开会的时候跟他进行语音通话。最开始的时候我是直接用 iPhone 跟他打微信，然后外放。听他的声音没什么问题，主要问题在于他听不清我们的声音。因为只要讲话的人稍微离麦克风远一点，比如坐在桌对面的位置，大概一米半距离，声音就很难被麦克风收进去。

所以当时需要买一款能在开会时使用的麦克风，也就是带有「全指向模式」的麦克风。「全指向」的意思是，面对各个方向的声音，它都会保持同等的灵敏度，全部都收进麦克风里。像有的麦克风其实是带有降噪功能的，比如当方向 A 有人在讲话时，方向 B 的声音会被进行降噪处理，音量会小很多，这种效果一般被称为「心型指向模式」，也就是主要拾取单方向（麦克风正面）的声音。

![](.evernote-content/AB813B0B-551E-494D-A9FE-6242A0CF710C/C1C777E2-DCA2-4F9F-84DA-1175CAD6CCF7.jpg)

Blue Yeti 这款麦克风有四种模式，在机身上有一个开关进行调整，其中包括了全指向模式和心型指向模式。同时拥有这两种模式的好处是既能满足开会的需求，也能满足一些额外的录制需求，比如录播客或者录制讲解视频。

![](.evernote-content/AB813B0B-551E-494D-A9FE-6242A0CF710C/CDD3C345-15A1-478B-B9A1-F0E19B39C511.jpg)

Blue Yeti 的四种模式

Blue Yeti 是一款用 USB 连接的麦克风 1 ，所以我可以直接将它插在 MacBook 上，然后将输入音源设成 Blue Yeti，输出音源设成 MacBook，这样就能在会议上正常使用了，并且能均衡收到各个方向的讲话声。

![](.evernote-content/AB813B0B-551E-494D-A9FE-6242A0CF710C/2C3EF5BF-6075-4609-B643-2EEB2699CCE8.png)

开会时的音频设置

Blue Yeti 曾被广泛认为是录制播客的最佳入门设备 2 ，音质是公认的好，不过得在录制环境还不错的前提下。为什么这么说，因为 Blue Yeti 虽然有心型指向模式，但降噪效果并不是特别出色。像一般的播客节目或者视频节目，观众对背景噪音的容忍度是很低的，Blue Yeti 容易把马路声、空调声、风扇声录进去，所以我会推荐大家进行一次降噪处理，推荐使用免费的 [Audacity](http://www.audacityteam.org/)。

另外 Blue Yeti 容易喷麦，如果你是录制播客的话，建议最好配备一个防喷罩。虽然说你可以稍微离麦克风远一点来避免喷麦的问题，但一般来说大家都不喜欢有距离感的说话声，还容易带有回音。另外 Blue Yeti 还容易把桌子的震动声也收进去，这个大家尽量避免敲到桌子就好。

Blue Yeti 的体积不小，买的时候要有心里准备，它一点都不便携。

![](.evernote-content/AB813B0B-551E-494D-A9FE-6242A0CF710C/777130FF-C2EC-4EDA-ABE1-7776839AA2F3.jpg)

Blue Yeti 的体积不小

Blue Yeti 的价格在专业麦克风里面算是比较便宜的，经常会在亚马逊上降价，如果你不着急购买的话，我建议可以等到一个好时机再出手。
