# Skype beta 版（Mac 及 Windows）增加 Facebook 间视频通话功能

---

分类: [互联网络](http://cn.engadget.com/category/internet/)

[![](.evernote-content/0A06146E-7257-42F9-A70B-D8A65C900F51/E01AF925-D0EF-4917-BDCF-8B2ACED16D96.jpg)](http://www.engadget.com/2011/11/17/skype-beta-for-mac-and-windows-adds-facebook-to-facebook-calling/)

因为朋友就是你 [Facebook](http://cn.engadget.com/2011/10/14/microsoft-finalizes-acquisition-of-skype-tony-bates-shares-his/) 的一切，因此 Skype 积极与 [Facebook 靠同一边站](http://www.engadget.com/2011/07/06/skype-comes-to-facebook-and-facebook-comes-to-skype/)，让你使用 Skype 就可以和 Facebook 上的朋友进行视讯通话。Skype 今年夏天才推出 [Facebook 视讯通话整合服务](http://cn.engadget.com/2011/07/29/skype-5-5-for-windows-now-available-complete-with-deeper-facebo/)，到现在更是已经在这方面出了两次的更新。最新的 Skype 5.7 Beta （Windows）和 Skype 5.4 Beta（Mac）能够让使用者连结自己的 Facebook 账号，然后在 Skype 中就可以和 Facebook 的好友进行视讯通话。Mark Zuckerberg 当初创立 Facebook （一开始叫 Facemesh）的时候只是要躲在计算机屏幕后看常春藤名校的漂亮女生而已，他大概万万想不到现在大家只要通过 Skype 视频通话就可以见到他的庐山真面目了吧？（前提是他要接电话啦...）
