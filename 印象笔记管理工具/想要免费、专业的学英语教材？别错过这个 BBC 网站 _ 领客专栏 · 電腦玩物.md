今天要介绍的这个英语教学网站：www.bbc.co.uk/learningenglish/chinese，来自于英国历史悠久的广播公司 BBC，据称他们从 1943 年开始就提供全球英语学习的单元。而进入网路时代后，这个「BBC Learning English」线上服务发展至今，仍然**每天持续更新**，**有大量专业的英语课程和学习测验，甚至提供练习英语听力的 MP3 让使用者免费下载**。

如果你想认识最新英文单字、练习从日常到职场的英语用法、学习英文文法中常见变化，甚至是强化自己的英文听力与口语能力。那么在「BBC Learning English」中提供了各种免费线上课程，这些课程**有文章、影片，还有可以下载的 PDF 教材，或是听力训练 MP3**！

并且「BBC Learning English」**不需注册**，里面的英语学习资源都能**免费使用**。网站本身还有中文版，除了一些重要功能有中文化外，某些单元的练习也都有中文辅助解释。

接下来，就让我来介绍「BBC Learning English」网站上的几个主要功能与使用方式。

1. 切换中文版网站

「BBC Learning English」有不同语言的版本，而且不同语言在教学单元上会有些微不同。你可以点击「BBC Learning English 中文站」连结直接打开中文版教学网站，或是在网站右上方的选单进行语言的切换。

![](.evernote-content/7785BCC8-F06A-4E02-AFE8-B8E49731387A/CA4F1E77-43C2-48AC-81F6-F155B9E0C207.jpg)

「BBC Learning English」的中文版网站，虽然不是所有的内容都有中文对照，但是基本的选项与功能会有中文版，让找到自己需要的内容相对容易。

而许多单元课程也提供了中英对照的解答，例如「双语看世界」单元，就用时事新闻进行英文与中文的双字幕练习。

![](.evernote-content/7785BCC8-F06A-4E02-AFE8-B8E49731387A/53C08154-9888-4933-8E92-3B221A7FA15E.jpg)

2. 规划好的英文学程

用影片学英文的网站很多，他们大多把现成的演讲或新闻影片变成教材，但是「BBC Learning English」并非只是把 BBC 制播的新闻片段拿来教你学英文，而是**全部重新制作的真正英语教学影片与练习**。

想要从头认真学习英语语法、口说与听力的朋友，建议可以进入「课程」页面。这里有 BBC 设计好的循序渐进课程，带领我们从一段段影片、一次次练习里，学会基础会话、职场用法或特殊语法。

![](.evernote-content/7785BCC8-F06A-4E02-AFE8-B8E49731387A/183414A9-7FC6-4F2F-B144-A0766EA92046.jpg)

3. 各种特制的学习专题

而想要比较随兴学习，或是当作课余补充英语知识的朋友，则可以进入「专题」页面。在特别设计的简短几分钟英语练习中，学习一个新的单字，或是学会一个正确的文法。

这些特制单元包含了：

* **随身英语**：最新潮流的英语新用法或新单字。
* **地道英语**：在日常生活与职场工作中的道地英语用法。
* **英语大破解**：用有趣的世界奇闻影片来学英语（文章版）。
* **你问我答**：解答你遇到的文法、词汇与文化问题。
* **今日短语**：中英搭配每天学会一个日常英语用法。
* **媒体英语**：了解专业新闻里的英文用法。
* **英语小测验**：线上测验自己的英文文法与单字。
* **每周一报**：用有趣的世界奇闻影片来学英语（影片版）。
* **双语看世界**：中英双语字幕看世界重大新闻影片。

![](.evernote-content/7785BCC8-F06A-4E02-AFE8-B8E49731387A/672EEE64-6B87-4D79-9991-78DE4F8D0EF3.jpg)

4. 各种练习测验题库

**最重要的是「BBC Learning English」上述的课程与单元都不会只是让你读英文、看英文，还会延伸设计课后练习**，可能是选择题、填空题、问答题，或是单字与文法的延伸教学。这些教材还都提供 PDF 免费下载，让我们可以带在身上随时练习。

![](.evernote-content/7785BCC8-F06A-4E02-AFE8-B8E49731387A/AD2CC04A-F39D-42C7-A183-A2CE66C88CFB.jpg)

5. 下载 MP3 练习英文听力

更棒的是，无论是影片或是文章，「BBC Learning English」都提供了朗读整篇内容的 MP3 文件可以免费下载。我们可以把这些 ，MP3 装到手机上，随时带在身上就能复习与加强英文听力。

![](.evernote-content/7785BCC8-F06A-4E02-AFE8-B8E49731387A/C3BB76B5-4986-4F5D-9B95-6D61AF9F4A87.jpg)

6. 中文版与英文版的不同

最后，如果你觉得「BBC Learning English」的中文站还不能满足你，也可以把语言切换回英文。这时候你可以找到很有名的 BBC 6 分钟学英文的不同单元，获得更多的学习资源。

**整体来说，「BBC Learning English」是一个有规划、有系统、深入浅出的英语学习网站。**它有适合中文用户的课程规划，也提供大量的阅读、听力练习资源下载，非常值得想要学习或强化英文的朋友参考。
