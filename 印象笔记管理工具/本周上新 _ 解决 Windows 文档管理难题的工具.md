本周上新 | 解决 Windows 文档管理难题的工具
---------------------------

数码荔枝

在使用电脑的过程中，我们总是会堆积各种各样的文件，其中就有不少是图片、Office 文档、笔记、电子书等。这些资料散落如果在硬盘各个角落不加以利用，就只是一个个冰冷的文件，而如果能够整合起来、随时查阅，就可以化为知识，消化吸收。因此，如何将电脑上的文件整理为资料库，就成为了数字时代非常重要的一门功课。

tagLyst 正是一款 Windows 平台备受好评的资料管理神器。tagLyst 不会转换或修改文件内容，通过给文件贴上标签，来实现对文件的多维度组织整理，进而形成你个人的专属知识库。无论源文件散布在哪个角落，知识都能立刻跃然眼前。

![](.evernote-content/19DA09C5-0A54-4664-87F0-B2D21BC3D5F3/59F202AB-5919-41D7-B543-AD87EC6A8A28.jpg)

tagLyst 已上架数码荔枝正版软件商城，点击文末「阅读原文」前往了解详情。

**通过「贴标签」整理文件**

tagLyst 的使用非常简单，将资料文件拖入默认或新建的资料库，或者将原有的文件夹转为资料库，仅此一步，所有资料便能够立刻在软件中查阅。然后就可以给文件贴标签，进行多维度的组织整理。

软件提供了非常丰富的标签功能与样式，每个资料库都可以有一套自己的标签，标签可以设置不同自定义分组、7 种颜色和超过 150 种图标。这些设置让标签变得井井有条，文件贴上多个标签依然一丝不乱。

![](.evernote-content/19DA09C5-0A54-4664-87F0-B2D21BC3D5F3/FF2B69B8-FBA9-4951-8061-680728C98955.jpg)

**强大的全文关键字检索**

检索是资料管理类软件最重要的功能之一。tagLyst 支持多种检索方式：按自定义标签、按文件目录、按文件类型、按时间范围等等。

更强大的是，tagLyst 对多种文档格式、如 txt / pdf / docx / xlsx / pptx 等实现了全文关键字检索，文件内部的信息也能够被聚合在一起，大幅降低了文献资料的查找难度。

![](.evernote-content/19DA09C5-0A54-4664-87F0-B2D21BC3D5F3/ECC6BDCC-8C2C-4128-B4A9-71A8953634DF.png)

**更多贴心功能**

tagLyst 不仅只是简单的加标签和检索，还提供了诸多好用的贴心功能，让文件管理更加轻松、易用。软件支持多种主流文档格式预览，包括 txt / md / pdf / docx / xlsx / pptx 等，不用打开文件就能查看大致内容。

![](.evernote-content/19DA09C5-0A54-4664-87F0-B2D21BC3D5F3/A0870E38-E0A6-4449-92EF-DC8E2BEF1DA4.jpg)

对于图片，tagLyst 提供了灵感流（瀑布流）布局，无需多余操作即可快速浏览全部图片。

![](.evernote-content/19DA09C5-0A54-4664-87F0-B2D21BC3D5F3/394FB542-3912-408A-A6AC-24ECB71CDB03.jpg)

此外，软件还支出标签云统计、悬浮窗、拖动归档、文件备注等实用功能。

![](.evernote-content/19DA09C5-0A54-4664-87F0-B2D21BC3D5F3/4C51F9F9-A2DB-42EE-909D-45A54EC71D26.jpg)

作为一款功能强大的资料管理工具，tagLyst 仍然保持了足够的简洁和流畅，数千文件的资料库也能够瞬间完成检索。tagLyst 不会破坏文件内容，也不会上传文件，所有操作都只在本地执行——当然，也可以将资料库文件夹放入信任的云同步服务，实现备份和团队协作。
