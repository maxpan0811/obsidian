# 后阮富仲时代的三大看点

---

在解说越南政坛权力的游戏之前，关于阮富仲总书记的去世，我需要给大家严肃地补充一个关键信息——在我们党的评价体系里，马克思主义者的称号是至高荣誉，细分又可以分为伟大的(10人)，杰出的(1人)，坚定的(7人)，忠诚的(3人)和不加定语的马克思主义者(1人)。而7月18日的公告又强调，苏林只是指导工作，不是主持工作。中国两次帮助西哈努克，终于让他重返王座，却被洪森这个越南带路党掌握了柬埔寨的实权。

今年春节，越南时任公安部长苏林抢在武文赏前面，越俎代庖地慰问了退休的国家主席们。（在儒家文化圈里，这就叫楚王问鼎）。

也是在今年春节，沉寂多年的南方派（走资派）领袖——前总理阮晋勇一反常态地高调慰问了公安部。

武文赏的辞职公告中提到了“根据越共中央检查委员会和其他部门提供的报告”。这里“其他部门”很可能指的就是公安部——是苏林把武文赏拉下了马。

《越南政坛大戏第二幕，主角换成了公安部》中，我们介绍了越南公安部是一个堪比当年苏联克格勃的超级部门，以及苏林升任国家主席，但被戏剧性地临时免去公安部长职务。

系列第三篇《普京访问朝鲜越南，有哪些猛料》中，则介绍了在越南女首富张美兰以招供换取免死之后，手握“百官行述”的苏林权倾朝野。

亲近苏林的原公安部第二副部长，取代了主持工作的常务副部长扶正。另一位亲近苏林的原公安部第三副部长荣升越共中央办公厅主任。

考虑到手握最高人事权的阮富仲如此支持苏林和他的小伙伴儿们，我分析认为越共当前的工作重心是保政权，防止颜色革命和南北分裂。如此一来，就必须要有一个能大刀阔斧收拾走资派的核心。

本文作为系列第四篇，将以阮富仲总书记的去世为转折点，讲述后阮富仲时代，以越南国家主席苏林和总理范明政为男一，男二，且有柬埔寨的洪森亲王作为特别出演的新一轮大戏。

**一:权力的游戏**
-----------

在解说越南政坛权力的游戏之前，关于阮富仲总书记的去世，我需要给大家严肃地补充一个关键信息——在我们党的评价体系里，马克思主义者的称号是至高荣誉，细分又可以分为伟大的（10人），杰出的（1人），坚定的（7人），忠诚的（3人）和不加定语的马克思主义者（1人）。迄今为止，被中国授予马克思主义者称号的外国人有且仅有阮富仲一位，而且一上来给的就是“坚定的马克思主义者”，和叶帅等同。

另外，我们的高层亲自去越南驻华大使馆吊唁，全国政协主席去河内吊唁，上海广东广西云南的书记各自到越南驻华领事馆吊唁，就连驻朝鲜大使都去了越南驻朝鲜大使馆吊唁……

这固然是发自肺腑地敬重阮富仲同志，同时也是给越南的男一号和男二号指明方向——不用你们倒向中国，能坚持阮富仲同志的平衡外交，就是顶好的同志。严肃的部分说完了，接下来要起飞了哈。

先说男一号苏林。

7月18日，越共中央通报:（后面我会拆解这段公文的玄机，先大致扫一眼就行。）

“截至目前，因需为阮富仲总书记集中接受积极治疗优先安排时间，确保越共中央委员会、政治局和书记处的总体指导调控工作的要求，根据越共第十三届中央委员会、政治局和书记处的工作制度，目前，政治局分工越共中央政治局委员、国家主席苏林按照政治局规定的责任和权限，负责指导越共中央委员会、政治局和书记处的工作。”

7月17日，越南在联合国提交了新的南海划界提案，中国外交部坚决反对。7月18日的这份公告说阮富仲已经到了需要集中治疗的程度。那么问题来了，这份南海划界提案到底经没经过阮富仲点头？

根据越通社《阮富仲总书记在108军队中央医院的最后日子里》可知，他老人家早在7月13日下午就已经需要上呼吸机来维持生命体征。当时医院的工作人员已经意识到了这是不可挽救的一刻，开始哭了。

那么从7月13日下午到7月18日发公告由苏林指导工作之间，这几天的越南是谁在做主?要知道，7月13日下午，苏林才从柬埔寨回国。其他几位大佬，有没有可能在阮富仲失去意识之后，苏林返回河内之前的时间里，先商量一下国家大事呢——比如给苏林限定职权。

如果有，那么两大男神之一的范明政必有参与。

前面两条关键信息简直是在玩薛定谔的猫，让我们无法确定那份不上道儿的提案是谁的责任。

而7月18日的公告又强调，苏林只是指导工作，不是主持工作。之所以让他指导，只是根据政治局的分工。也就是说，他连个代理总书记的名分都没有，不仅没有名分，权限也是经过政治局限定的。

试想，如果苏林在局里是众望所归，公告里怎么可能是这么个措辞?

即使无名无分，权力受限的指导权，也仅限定在阮富仲接受治疗期间。严格来说从公布阮富仲去世那一刻起，这份授权就终止了。即使按照东亚社会的人情世故，最多也就是办完阮富仲的身后事。

问题来了，中央委员会和政治局的工作就是开会。苏林如果不敢开会，那么他怎么指导？（如果开会选总书记，他因为手握公安部已经很强势了，大家不喜欢，大概率上位不了，所以对他来说，能不开就不开是最好的）

书记处倒是有日常工作，可主持日常工作的有常务书记。会都不敢开的苏林能不能指挥得动这位常务书记呢？

如果苏林敢开会，选举新的总书记，问题又来了。是他自己身兼两个职务呢？还是让出来一个呢？而且阮富仲去世后，留下的职务除了总书记还有两个关键的兼职——反腐败反消极指导委员会主任，以及越共十四大筹委会干部小组组长。

一个是最高反腐指挥权，一个是最高人事权。如果苏林只是当上了总书记，但没有得到这两个职位，那么他的权力还比不上2018-2021年的阮富仲。

这样的话，苏林最多就是把越共的四巨头制度再一次变成临时性的三巨头，无法顺利成为越共的新核心。

可是苏林之前已经在上位的路上，放倒了太多的人，把各方得罪得死死的，如果不能顺利成为核心，还想在这场权力的游戏里不出局，那么他大致还剩下两条路:

第一条，团结越南国内唯一和他资历相当的总理范明政，以及团结和越南有着特殊关系的老挝、柬埔寨的高层，同舟共济。

第二条，效仿南方派领袖，前总理阮晋勇，挑起南海争端，煽动民族主义，背弃阮富仲的平衡外交，剑走偏锋赌一把。

目前来看，第一条路已经出了岔子，第二条路已经有了苗头。第一条为什么出了岔子呢？答案就在7月12日，苏林对柬埔寨的访问里。

**二:列王的纷争**
-----------

2024年7月12日，越南国家主席苏林访问了柬埔寨，和柬埔寨枢密院院长洪森亲王举行了会谈，关于这次会谈的信息量有多大，有必要了解一下现代越南和柬埔寨的恩恩怨怨。

而要串起这些恩恩怨怨，就需要请出中国人民的老朋友——诺罗敦·西哈努克国王。

他老人家的故事轻轻松松就能写一篇万字长文，这里只拣最要紧的来说明问题。

众所周知，西哈努克因为常年流亡中国，将中国称之为第二祖国。但很多人不知道的是，他的流亡其实是分为两段的。

第一段是1970年他被亲美派政变推翻。于是他在中国的帮助下和柬埔寨的共产主义者（红色高棉）团结在了一起。18岁的辍学农村小伙洪森，正是在听了西哈努克这位柬埔寨国父的广播之后才决定去投红色高棉。

1975年这个王室+红色高棉的奇怪组合也确实推翻了美帝的狗腿子政权。问题是红色高棉掌权之后的所作所为，令人不忍直视。1976年，西哈努克由于拒不配合红色高棉，遭到了软禁。

1977年已经当上了师长的洪森因为担心被清洗，率领部队叛逃越南，成为柬埔寨三大带路党之一。

西哈努克能结束被红色高棉软禁的日子，开始第二段流亡生涯，是因为洪森带领着越南人民军朝着首都金边杀过来了。红色高棉在万不得已之下，本着试试看的心态，把西哈努克送到了国外，以寻求支援。

1979—1991，西哈努克在中美的支持下，成为联合国承认的柬埔寨国家元首，洪森则在苏联和越南的支持下一步步成为柬埔寨最有权势的人。

在1991年的和平谈判中，洪森亲自来到北京迎接西哈努克，并拜他为义父。柬埔寨各方势力在联合国的监管下成立了联合政府。

之后在一系列政治斗争中，西哈努克和他两个久经考验的王子，都不是洪森的对手，被迫过上了安度晚年的生活。洪森在20年前拥立了西哈努克最佛系的儿子成为国王，并在去年把首相的位置传给了自己的长子洪玛奈，他自己就简简单单地当个枢密院院长、参议院主席、国王顾问委员会主席什么的。

提炼一下这些恩恩怨怨，可以得出关键信息:

中国两次帮助西哈努克，终于让他重返王座，却被洪森这个越南带路党掌握了柬埔寨的实权。虽然洪森掌权后对华友好，但假如哪天洪森不在了，洪家和王室展开新一轮权力的游戏，一切就不好说。

而越南和柬埔寨本来就有极深的历史宿怨。越南最富裕的湄公河三角洲就是从柬埔寨手里抢的，至今很多柬埔寨人还管越南的最南方叫“下柬埔寨”。加上1979-1991年越南还在柬埔寨建立了傀儡政权，新仇旧恨很容易激发民族主义情绪。

还有就是，洪森的执政合法性问题，他有三大合法性：王室背书、曾经推翻红色高棉的功劳和柬埔寨的经济增长。

这三大合法性都有隐患。

有王室背书——但哪天王室和他闹翻了怎么办?

推翻红色高棉暴政——经历过红色高棉时代的人越来越少（去世了），年轻人用民族主义，要求清算带路党洪家怎么办？

经济增长——哪天经济停滞了咋整？

这些隐患的存在，让洪森害怕被清算。

要想未来不被清算，除了需要向北方大国证明他们家比王室还可靠，还需要得到民族主义的支持，让柬埔寨和越南平起平坐，最起码，也得用实际行动证明他是“杨康”，而不是“完颜康”，也就是逐渐减少柬埔寨经济对越南的依赖。

相信你已经想到了，这说的不就是在中国帮助下，即将于于今年八月份动工的“德崇富南运河”吗？

柬埔寨经济发展有一个大问题，就是内陆和沿海之间，没有河流联通，如果走一段陆地再到港口，就会导致出口产品成本高昂，所以只能走湄公河，借助越南的港口出海，降低成本，等于被越南卡了脖子。

而解决之道，就是修一条运河，德崇富南运河。运河将把柬埔寨内陆和沿海连接起来，以后出口不用再走越南了。

越南当然会想方设法阻止这条运河开通。拿湄公河的环境保护当理由只是基本操作。宣称要和菲律宾举行海岸警卫队联合演习，而且演习日期正好是柬埔寨运河动工的同一时期，摆明了是要拿南海问题和中国讨价还价。

访问柬埔寨的时候，苏林颇有些居高临下的姿态。

苏林强调，越柬是友好邻邦，回首历史两国曾并肩作战，共同对抗敌人。苏林重申，越南坚定不移奉行对柬埔寨友好、信任与合作政策。

洪森指出，柬埔寨不会忘记越南帮助柬埔寨推翻波尔布特政权，而越南领导人也回顾柬埔寨向越南提供的支持和帮助，使越南实现民族统一。（这话递的得有多硬？）

两国领导人特别强调了越柬关系的历史价值，并高度同意推动和多样化宣传、教育方式，让两国人民，特别是年轻一代对两党两国和两国人民传统友谊、团结和全面合作更加理解和正确认识；（说明柬埔寨年轻人有浓厚的反越情绪。）

注意，在苏林访问柬埔寨，与洪森会谈的7月12号这一天，阮富仲虽然住院，但还是能继续工作的。这时候苏林显然不可能知道阮富仲这么快就会过世，不然他不可能在这种时候还出去搞什么访问。

从洪森亲王的视角来看，苏林这人嘿，上一次见面还是公安部长，这次当上了主席就敢这么和本王聊天儿，等他当上了越共的核心，那还得了？

7月19日，阮富仲去世的消息被公开。

7月20日，中国驻柬埔寨大使汪文斌高调宣称柬埔寨与巴基斯坦并称为巴铁，柬钢。

7月22日，洪森长子洪玛奈首相和汪文斌大使对两国深厚的传统友谊表示赞赏，同时也对继续扩大两国国防、安全、能源等各方面的合作交换意见，而柬埔寨之前在国防领域的合作，是比较依赖越南的。

这个态度表面，中国不会因为越南有意见而放弃或搁置富南运河。

问题再一次摆在了我们面前——如何摆平越南，让他们别在南海和菲律宾联动呢？二号男神、总理范明政，该你上场了。

**三:冰雨的风暴**
-----------

阮富仲逝世后，苏林是二号人物，范明政是三号，他们上位一号的希望，比其他人都大，所以互为最大竞争对手。

范明政最让我佩服的地方在于，他可以在锦心绣口和大白话接地气之间无缝切换。比如他6月访华时，提出请中国投资越南高铁，不要时速250公里的，要350公里一步到位，而且要我们培训人员，转让技术。

是的，诸位没有看错。是投资，不是贷款，看看人家范大才子替我们着想的多细啊。这样就不会被人抹黑成债务陷阱了。我们只需要投资4000亿人民币（起步价），帮越南人学会怎么管理、维修和制造高铁和铁路，然后慢慢收车票钱就行了。

中越友谊地久天长，朝夕太短，一万年总能收回本儿来。

玩笑归玩笑，其实仔细想想，范大才子正是靠着这种厚如城墙的脸皮，才能在一次次权力的游戏中游刃有余，在列王的纷争中到处都吃得开。

比如，7月15日越南总理范明政日前表示，越方愿同中方加紧落实两党最高领导人重要共识，不断增进政治互信，深化务实合作，筑牢民意基础，加强多边协作，妥善处理分歧，推动具有战略意义的越中命运共同体落地生根、开花结果。

这番突如其来的告白，当时我们看不明白什么意思。

其实需要结合7月13日阮富仲的失去意识和7月17日的越南南海划界提案来看，才能知道范明政是在特殊时期，向我们示好，并撇清他的干系。

如此了得的功夫……越南接下来继续竹竿外交的问题应该是不大了。

之前我盘算的是苏林作为一个强力部门出身，在英国留下金箔牛排黑历史还不让人说，在德国不顾当地法律抓逃犯的“刺猬”，投美无门。相对于范明政这样和谁都过得去，刀切豆腐两面光，苏林当核心，对我们来说更靠得住。

但是当我注意到，我们国家给予搞竹子外交的阮富仲史无前例的哀荣，以及苏林连跟柬埔寨的洪森都能聊崩的时候，我现在更倾向于另一种可能:

一个唱红脸，一个唱白脸。

苏林气势汹汹，而范明政到哪里都笑嘻嘻，是一对不错的组合。

**尾声**
------

7月25日，美国商务部发言人证实，由于技术问题，拜登政府把原定于7月26日宣布是否将越南列为“市场经济国家”的决定推迟至8月2日。

我给翻译翻译，就是拜登政府如果承认越南市场经济地位，一怕越南成为中国出口的马甲，二怕得罪铁锈带工人；不承认越南市场经济地位，又怕越南怨恨美国，不配合美国的战略。

越南高层曾经承诺过，要解决给中国当马甲的问题，给美国一个交代。不过当时做出承诺的那位，早就在越南的剧烈人事变动中下台了。现在民主党忙着换新总统候选人，越南忙着“分享”阮富仲的政治遗产。

不确定性就很大了。

后阮富仲时代，苏林继承了阮富仲对内眼里不揉沙子的风格，范明政继承了阮富仲对外和谁都能笑嘻嘻的本领。理论上，他们俩很容易打起来。但目前来看，这俩人很可能不会开打，反倒是非常不错的组合，互相制衡。

很多事甚至不需要我们操心，越南这对儿神仙组合自己也能想办法摆平。

当然，最重要的是得等到8月2号，看看越南到底会不会拿出一个能取信于美国的交代。看看在中越南海闹起来的情况下，美国会不会免去越南倒向中国的担心。看看在民主党眼里，到底是拉拢越南遏制中国重要，还是笼络铁锈带的选票要紧。
