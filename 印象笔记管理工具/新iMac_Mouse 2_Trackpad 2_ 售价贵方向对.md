# 新iMac/Mouse 2/Trackpad 2: 售价贵方向对

---

[![](.evernote-content/10301680-9254-4953-A644-E95B8500AA46/2448EC57-2776-46AA-A628-2CE58DEFA899.jpg)](http://www.feng.com/iPhone/news/2015-10-14/New-iMacMouseTrackpad-2-the-price-for-your-direction_627104.shtml)

　苹果昨天晚上帮我们实现了几个小愿望，发布 2007 年古董款 Wireless Keyboard 和 2010 年 Magic Trackpad 的接班人，这两款配件我们已经使用多年。去年的 Retina 5K iMac 也获得了更新。此外，2009 年的 Magic Mouse 迎来接班人，苹果还首次将全新 Retina 4K 显示屏引入 21.5 英寸 iMac。

![](.evernote-content/10301680-9254-4953-A644-E95B8500AA46/2448EC57-2776-46AA-A628-2CE58DEFA899.jpg)

虽然我们知道这些升级是必然的，但它们真正到来的时候，我们还是感觉到惊喜。

比如，谁曾想过 Magic Trackpad 2 成为了苹果的第一款只支持蓝牙 4.0 的 Mac 配件？ iOS Lightning 连接器会进入 Mac 配件？或者 Retina 5K iMac 会在短短 1 年内迎来重大屏幕升级？

苹果昨天晚上更新的每一款产品都重申了苹果的承诺，苹果承诺要提供优质的使用体验，这是苹果的 DNA。

![](.evernote-content/10301680-9254-4953-A644-E95B8500AA46/7F620ECA-469D-4B05-81A0-86D2E344D4E1.jpg)

**Magic Keyboard**

我使用上一代苹果 Wireless Keyboard 已经多年，它出色的打字表面让我们感到惊叹，但需要更换 AA 电池最终让我们有些厌烦。

今年年初的时候，我勉强的用回 50 美元的苹果 Wired Keyboard，说实话，除了有线之外，我喜欢这款键盘的一切。

使用苹果 Wired Keyboard 有线键盘，我再也不用看到让人心烦气躁的 OS X“低电量”警告。这款键盘的按键也很舒服，但是，我一直都希望苹果推出一款可充电的无线蓝牙键盘。

当罗技发布一款让我们眼前一亮的蓝牙 Easy-Switch K811 键盘时，我们也暗自希望苹果也发布一款类似的键盘。

直到昨天晚上，这个小心愿终于能够实现。

苹果最新发布的 Magic Keyboard 配备可充电电池，承诺提供 1 个月以上的续航时间。这个时间也许无法和其他键盘的半年甚至 1 年续航相比，但它从其他方面进行了弥补，比如用户可以得到 Lightning 转 USB 线，如果你喜欢，你还可以使用 Lightning 接口连接键盘，这样很方便，就像许多拥有 Micro USB 端口的蓝牙键盘一样。

从另一方面来说，苹果将 12 英寸 Macbook 上的输入体验带到了Magic Keyboard 上，这意味着，键盘的键程变短了一些，虽然被假以“稳定性提升超过33%”之名。

和罗技不同，苹果没有在 Magic Keyboard 身上设任何专用的多设备配对按键，所以你无法像使用Easy-Switch K811 键盘那样毫不费力的在 Mac、iPad 以及 Apple TV 之间切换。

苹果最新发布的 Magic Keyboard 是我一直想要的产品，但是如果你已经拥有罗技 Easy-Switch K811 的话，你可以暂时不考虑购买，苹果 Magic Keyboard 售价为 99 美元。

![](.evernote-content/10301680-9254-4953-A644-E95B8500AA46/6D7AF42F-6CBB-4DE9-9E46-B7BC76CDF462.jpg)

**Magic Mouse 2 + Magic Trackpad 2**

我们使用第一代 Magic Mouse 的时候，觉得它又漂亮又能干，直到第二年 Magic Trackpad 的到来。在这里，我不是要和还在使用鼠标的人分立阵营，只是对我来说，换到 Magic Trackpad 之后，我仿佛打开了一个新世界的大门。我很难再回去使用任何鼠标，即使它是一款经过重新设计的鼠标。

![](.evernote-content/10301680-9254-4953-A644-E95B8500AA46/EE3F6D9C-C96B-493C-A3E4-098C2F7DA388.png)

苹果昨天晚上带来的 79 美元 Magic Mouse 2 还不算彻底的重新设计，它看起来与第一代相似，不过把 AA 电池换成了可充电的锂电池，该鼠标还没有支持 Force Touch 功能。 Magic Mouse 2 的售价增长了 10 美元，可能是因为它支持了 Lightning 线缆。

和 Magic Mouse 2 对比， Magic Trackpad 2 更有吸引力，其触控面积较上一代增加了 29%，虽然我觉得没有太大必要，但至少它给了苹果往里面添加可充电电池的空间和添加 Force Touch 的空间。Force Touch 的添加是一件令人惊喜的好事。

Magic Trackpad 2 的表面是白色玻璃，这个配色让我们有点意外，因为 Mac 的设计哲学是“黑和银”，此外，Magic Trackpad 2 触控板白边到边玻璃顶的下方还设有四个力传感器。

Magic Trackpad 2 是唯一一款需要Mac 支持蓝牙 4.0 的新配件。另外两款配件Magic Keyboard 和 Magic Mouse 2 都可向后兼容，不支持蓝牙 4.0 的 Mac 也可以使用，只要它运行的是 OS X 10.11 操作系统，

我希望 Magic Trackpad 2 可以兼容我的 2011 年款 iMac，但是，这好像有点痴心妄想。除非我升级到最新款的 iMac，不然的话，我可能还是会继续使用70美元的第一代 Magic Trackpad，我不太愿意购买 Magic Mouse 2，原因上面已经说过，就不重复了。

另外，对于部分用户来说，Magic Trackpad 2 的价格可能有点偏高，达到了 129 美元。

**Retina 4K 和 5K iMacs**

![](.evernote-content/10301680-9254-4953-A644-E95B8500AA46/11A0EAF9-B7B1-46C6-AB55-90815E8186CE.jpg)

去年 Retina 5K iMac 发布的时候就心痒痒想买一款，但我 Hold 住了，原因有三：

第一，我并不是真的很需要一台新 iMac。

第二，价格比常规 iMac 贵。

第三，明年苹果会带来更好的。

果然，苹果昨晚上就更新了 iMac 产品线，27 英寸 iMac 已经全部支持 Retina 5K 显示屏。

今天，苹果的 iMac 新闻稿上写道：

新的 Retian 屏幕拥有更广的色域，能够提供更生动、真实的色彩。在标准sRGB屏幕上，许多现实生活中的色彩无法在屏幕上重现。

Retina 5K 和 4K 屏幕拥有更广的 P3 色域，色彩空间增加了25%，色彩呈现更加丰富，图像更加生动，细节表现更佳，也更加真实。

我很庆幸去年忍住了没买，那么今年便可以买到最新款的 Retina iMac。那款 1499 美元的 21.5 英寸 4K 版本看起来好像很诱人，要不，那款 1799 美元的 5K iMac 好像也很不错，它有英特尔第六代 Skylake 处理器，更大的屏幕，更好的 GPU.......好像还蛮值。
