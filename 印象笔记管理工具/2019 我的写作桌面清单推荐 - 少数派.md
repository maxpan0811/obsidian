2019 年有很多尝试和改变，其中开始写作应该是对我收获最大的一个改变。之所以在 19 年想写点什么，一是女儿出生后初为人父的我想留点什么给她，于是尝试用文字去记录她的成长；二是现在是信息爆炸的年代，也是信息分享的年代，文字输出能强迫我去更多地输入，提升自己。

工欲善其事，必先利其器！2019 年为了打造舒适的写作环境，在预算有限的情况下我尝试添置一些新设备。为此我还专门列了一个清单，这个清单既有硬件，又有软件，下面我就来分享一下写作桌面清单中最值得推荐的设备和软件。

还要多说一句，因为预算有限，所以我在选择产品时会有取舍，这个在具体的产品推荐时会提到。

硬件推荐
----

其实如果开始写作非常简单，笔记本、手机、电脑都能够完成。但如果想要获得更加高效的写作也许你最需要下面这三款设备。

### 键盘：Filco Minila Air 蓝牙无线键盘

我开始写作时一直在使用 MacBook 的内置键盘，手感一般，但胜在方便。长时间去码字手指在告诉我也许你应该试一试外置键盘，再加上想给 iPad Pro 配置一块键盘，这样便开始选购我的第一个机械键盘。

在选购键盘时的条件非常明确：

1. 支持蓝牙（iPad 上可以使用）
2. 67 键的键盘即可（数字键盘不常用，喜欢小键盘）

因为我对机械键盘一窍不通，在论坛上转了一圈发现 Filco 的键盘评价还不错，在亚马逊店庆的时候将这款键盘入手。

![](.evernote-content/CCBA69F5-D071-41B8-8E5F-984C48B48868/27039FE3-7F3C-47BA-B388-53A69CD5E93A.jpg)

实际使用下来的感受我非常满意，每一次「嗒嗒嗒」的键盘敲击都是对指尖的一次欢愉，也就是这块键盘让我爱上打字的感觉。每次坐到电脑前即使不写什么也想敲击两下，这也许就是一款优秀键盘应该具有的素质吧。

总结一下优点：

1. 手感舒适，回弹迅速，节奏感强烈
2. 蓝牙连接稳定
3. 符合人体工程学，长时间打字不累

当然也有缺点，键帽容易打油，对多设备没有一键切换，不过这些都不影响我对它的喜爱。对于想选购打字键盘的朋友非常推荐这款。

![](.evernote-content/CCBA69F5-D071-41B8-8E5F-984C48B48868/D5AEAD19-7272-45EF-8980-3446DF171C3F.jpg)

### 屏幕：小米 34 英寸曲面显示器

如果你和我一样，长期是用笔记本电脑写作，那么你可以试一试外接显示器。多一个屏幕给你带来的是 1+1>2 的效率提升。

当你需要多窗口操作时，一个更大的屏幕或者两个屏幕给你带来的便捷操作能让你效率倍增，同时能提升你的写作专注度。

同样和键盘一样选购显示器之前明确条件：

1. 预算 2500 之内
2. 分辨率 2K 以上
3. 屏幕比例最好是 21:9，看电影是我为数不多的爱好之一，21:9 的屏幕能带来更好的沉浸感。

目的明确后再去选择显示器就没有这么的难了，小米这款显示器在双十一 1999 元的价格真的非常的香，所以不出意外的选择了这款。

![](.evernote-content/CCBA69F5-D071-41B8-8E5F-984C48B48868/2F394564-559C-425C-B15D-B9A9D93CD311.jpg)

实际使用感受：显示器整体做工不错，特别是在线材收纳方面做得非常棒。屏幕非常大，得益于 21:9 的屏幕比例，配合多窗口管理软件可以轻松地打开三个窗口，这样你在写作时查阅资料就变得非常简单。MacBook 在外接 2K 屏幕时会有字体发虚，可以通过软件强制开启 HiDPI 解决。21:9 的屏幕在影音娱乐方面给你提供更加的视觉观感。

![](.evernote-content/CCBA69F5-D071-41B8-8E5F-984C48B48868/70410069-7FE9-4AA8-84C7-E34BED56130B.jpg)

多屏任务

这款显示器困扰我的不是分辨率，而是色彩。虽然显示器支持 121% sRGB 宽广色域，但是和 MacBook 内屏相比颜色显示还是有偏差，所以在涉及到修图时我往往会使用 MacBook 内屏修图，以获得更加准确的色彩显示。

![](.evernote-content/CCBA69F5-D071-41B8-8E5F-984C48B48868/A856259A-2EB8-4397-B977-69515FCFA91B.jpg)

### 桌面：乐歌 E1 电动升降桌

长时间的伏案工作加上年龄的增长，不得不让我更加重视身体的反应。在入手显示器之后第一时间购买了支架来解决低头办公的问题。颈椎问题解决了，腰椎也要注意，所以便有了第三件推荐的数码产品：乐歌 E1 电动升降桌。

当然你也可以通过设定闹钟定时起来活动一下来解决腰椎的问题。但工作状态是需要连续性的，一旦打断重新进入状态又需要时间，电动升降桌便能很好地解决这个问题。

实际使用体验：乐歌在电动升降桌领域一直做得都不错，桌面尺寸可以根据自己的实际情况而定。E1 是双电机，这样桌子承重更大一些。我这款桌板也是乐歌的，复合板材，第一次打开也没有闻到异味，打开就能使用。升降过程比较稳，噪音也不大，自带时间管理（45 分钟），让你在坐、站中从容切换。

![](.evernote-content/CCBA69F5-D071-41B8-8E5F-984C48B48868/3A82FAC9-BBC0-4781-8789-E868B5BFE239.jpg)

### 小结

数码产品永远是越贵的越好，但是更贵的数码产品并不一定适合你。与其在纠结硬件的升级，不如把一部分预算投入到软件，一款好的软件带给你的便利绝对不能忽视。下面我们再来说一说软件方面我的推荐。

软件推荐
----

### Trello x 微软待办

起初写作并不多时仅仅通过简单的「待办事项」管理即可，但是随着发布平台的增多，要求各不相同，需要一个展板去管理这些选题，使用 Trello 的需求应运而生。

Trello 负责对选题进行动态跟踪，从计划、确定选题、素材整理、写作到最后发布都能通过展板中的卡片进行展示，一目了然。卡片建立后可以通过左右拖拽在各个状态切换，让你明确写作状态。如果是多人协作，可以将任务分配到具体个人。

配合 Trello 使用的是微软待办，微软待办中「一天」的概念可以和 Trello 无缝衔接，根据展板中的选题的状态来分配这一天的任务。把选题细化，再细化后，方便在一天中进行管理。

![](.evernote-content/CCBA69F5-D071-41B8-8E5F-984C48B48868/542BD24C-5A91-4BC0-BA1D-7A68363BF95D.jpg)

### Bear

在没有接触 Markdown 之前，使用 Office 始终没有找到行云流水的写作感觉，在内容和格式之间来回切换，思路不连贯。接触了 Markdown 之后，我就爱上了它。

需要说明的是 Markdown 的语法非常简单，稍微学习之后便能上手。使用 Markdown 编辑器再去写作时你只需关注内容即可，思路能保持一贯性，让你沉浸在文字之中。

Bear 是一款优秀的 Markdown 软件，通过订阅可以实现在 macOS、iPadOS 以及 iOS 无缝体验。颜值即正义，Bear 设计简洁，上手非常简单，让你更多的精力专注于内容本身。

Bear 区别于其他 Markdown 软件的还有一些小特色。首先文内支持文章的链接，可以方便地把散乱的文章整理到一起；其次 Bear 也支持待办事项，在列表中有一个进度条，你可以将文章划分为几块分别写作，完成一块内容勾选一个，让你快速掌握文章的写作状态。这点还是前面提到的将任务细化、再细化。

![](.evernote-content/CCBA69F5-D071-41B8-8E5F-984C48B48868/90129652-1AC5-43AF-8DED-3C1FD0DB0A80.gif)

### Pixelmator Pro

在写作中避免不了需要图片素材处理，简单的处理我通过 Mac 上的「照片」自带的编辑功能便能实现，同时「照片」也是我素材管理中心。但是负责的修图我都会交给 Pixelmator Pro 处理。

如果你之前使用过 Photoshop，那么上手 Pixelmator Pro 并不难。Pixelmator Pro 提供的功能和 Photoshop 相当，对于我处理文章图片素材已经绰绰有余。从图片素材的选取、拼接、调色、修复、添加文字、排版都能够胜任，让你在一款软件中完成素材的处理。
