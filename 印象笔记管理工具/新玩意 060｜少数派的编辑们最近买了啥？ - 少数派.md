新玩意 060｜少数派的编辑们最近买了啥？
=====================

#### 关于栏目

很多读者都会好奇少数派的编辑们到底平时都「买了啥」。我们希望通过「编辑部的新玩意」介绍编辑部成员们最近在用的新奇产品，让他们自己来谈谈这些新玩意的使用体验究竟如何。

@北鸮：Xikii 1800mini 机械键盘客制化套件
----------------------------

* 入手渠道：二手购入
* 参考价格：全新团购价 2499-2599

在自己没有接触键盘之前，我曾经在另外一个网站看过一把奇形怪状的键盘。作者把键盘砍到只剩下 72 个键，第一排的 F 区只留下 F1-F5。那个独特的设计给我留下了极其深刻的印象，以至于当我开始想购买键盘时，第一反应是搜索这把像刀一样的家伙，可惜当时我才发现，这东西不仅远远超过我的预算，还是限量发售的，市面上已经基本找不到了。

![](.evernote-content/2127235E-A08D-43E1-897D-15FED0AC89DA/CEF6E913-C93B-4723-A5F6-D6A6146A30A4)

（图源：外设堂 Weistang.com）

键盘的作者叫拆机狂魔小西西，那把键盘叫做 DIYer72，后来他成立了自己的客制化品牌 Xikii，出品了一系列设计独特的客制化键盘套件。Xikii 的产品大多采用了热插拔结构，同时他们开发了一套有专利的全包裹式定位板，能够更好地让轴体嵌入，保证热插拔结构的稳定性。

后来我开始接触机械键盘，其间我也知道了，Xikii 的产品对于很多专注客制化的人来说定位非常奇怪，因为它标配就是热插拔、蓝牙双模、一体式全包裹定位板，无论手感和声音都没什么调整的空间，外形张扬，到手插轴装帽即用——这基本是一把精致的半成品量产键盘。

但由于我的兴趣更多地落在寻找最适合自己的轴体上，所以一直对热插拔结构情有独钟~~，绝对不是因为我的焊工太烂~~。这时我想到了 Xikii。是时候给当时的自己圆梦了。Xikii72 已经售罄多年，在现有的配列中，我选择了 1800mini，一把 40% 配列为基准的 60% 键盘。

![](.evernote-content/2127235E-A08D-43E1-897D-15FED0AC89DA/82741D34-E2F4-422A-AFF8-CCAE6022A840)

键盘的基础配列非常小，数字区没有删掉但是被隔开，在这个紧凑的空间中，小键盘 Pad 还是保留了下来。这把键盘的配列灵感来自于 Cherry 的老型号 G80-1800，它 1800mini 的名字也是由此而来。之所以二手购入，是因为当时 Xikii 说电泳色绝版了，所以原本打算等最后一轮 1800mini 团购的我，为了这个电泳黄，提前收了一把二手。（结果刚才我才发现最后一轮团购又开放了电泳色，虽然我也没那么心痛，毕竟早买早享受，何况还便宜了点。）

这种配列其实没有看起来那么难用，习惯了之后甚至有点回不去。40% 的主键区配列可以在双手几乎不移动的情况下按到每一个键，当我需要数字和功能键，我既可以选择直接在字母区按组合键，也可以挪一点点手去按现成的键位。键值的自定义程度也非常高，在自己设置了一圈之后，这把键盘我从上手到习惯只花了两段文章的时间。而且考虑到目前我还没有找到最心仪的轴体，1800mini 拿来日常换轴就非常方便。

![](.evernote-content/2127235E-A08D-43E1-897D-15FED0AC89DA/C4CA15B0-910D-4870-A034-F519958F2F86)

这把键盘的问题主要就是键帽难配，1.75u 长度的回车和两个 2.75u 的空格对一般的键帽套组来说就是胡闹，而且我又常常换轴，不打算配贵的键帽用，日用原厂高度时就只能用手里的散碎键帽拼起来凑合。因为采用了非常厚实的一体化定位板，1800mini 的手感也没有非常优秀，整体来说比较硬朗，用段落感很强的轴时稍稍震手，所以我配了一个压力克数不高的线性轴（91 轴，触底 58.5gf）。对我这种黄色爱好者来说，这已经是最合理的配置了。

如果需要的话，这把键盘正处在最后一次团购阶段，本月底结束，大家可以自行搜索相关信息。最后还是提醒一句，选购键盘一定要严格按照自己的需求购买，好好读我的 [选购指南](https://sspai.com/post/64972)！

@Microhoo：Belkin 雷雳 3 扩展坞专业版
----------------------------

* 入手渠道：Apple Store
* 入手价格：2398 元

我是 M1 芯片的 MacBook Pro 用户，同时因为在家办公并且外接显示器所以它俨然成为了一台座机，于是乎就非常需要多多的接口了。又因为曾是第一代搓搓条 MacBook Pro 用户，所以手上已经有了官方的 USB-C 至 USB 转换器和网易的便携式多口转换头，就不再考虑小型的转换器，想要拥有一个大一点更多口且能独立供电的扩展坞。

![](.evernote-content/2127235E-A08D-43E1-897D-15FED0AC89DA/37128C78-7EE6-4A6C-9018-517EDC2BE621)

对于数码配件，我一向信任能被 Apple Store 钦点的品牌，而 Belkin 从各种线缆、移动电源到无线充电器我都曾购买过，品质一向靠谱，所以选择 Belkin 雷雳 3 扩展坞专业版也是水到渠成的事儿了。这款拓展坞不便宜，所以做工非常实在，除了前后接口板是用黑色塑料，四周被无接缝的一体式铝金属包裹，非常 Apple。

![](.evernote-content/2127235E-A08D-43E1-897D-15FED0AC89DA/F31FFDE4-02A7-4D7F-84E3-FEA087CEBDA9)

拓展坞的正面非常简洁，只有四个接口分别是：

1. 1 个适用于 USB-A 外围设备的 USB-A 3.1 端口
2. 1 个适用于 USB-C 设备的 USB-C 3.1 端口
3. SD 卡读取器无需适配器即可连接 SD 卡
4. 音频输入/输出支持麦克风和扬声器

背面则要复杂些：

1. 170 瓦 PSU（电源设备）不仅可以为扩展坞供电，另外还以高达 85 瓦的功率为连接的笔记本电脑充电，以便在您工作时保持笔记本电脑电量充足
2. Thunderbolt 3 端口通过随附的 Thunderbolt 3 线缆连接到笔记本电脑
3. Thunderbolt 3 端口连接到高分辨率监视器/显示器和 Thunderbolt 3/USB-C 外围设备
4. DisplayPort™ 连接到高分辨率监视器/显示器
5. Gb 级以太网可实现可靠的硬连线以太网连接
6. 4 个 USB-A 3.0 端口可用于通过 USB-A 连接的设备，例如鼠标、键盘或打印机

![](.evernote-content/2127235E-A08D-43E1-897D-15FED0AC89DA/FD4BDE32-1704-4C96-9454-E4E9DC3633EC)

不过在实际使用中，前面板除了经常插拔 SD 卡外，很少会使用到，常接的显示器和几个不同接口的数据线我都是插在后面，所以整体观感非常简洁。

![](.evernote-content/2127235E-A08D-43E1-897D-15FED0AC89DA/EF747EB4-E178-432E-82E1-89C61A95846C)

因为这台扩展坞有独立电源，所以能够提供最大 85W 的电力，不仅应付 MacBook Pro 充电绰绰有余，对于接入的其它数码设备也能提供足够马力的充电速率，我在用的显示器是 LG UltraFine 5K，连接使用一切顺利，官方数据是可以同时支持至多两台 4K 显示器。

![](.evernote-content/2127235E-A08D-43E1-897D-15FED0AC89DA/F419F867-09E3-4426-A7DE-B0B8D7EB3DD9)

最后，祝大家都能拥有一个快乐够用的「数码」桌面。

@化学心情下2：口香糖电池和 CD Walkman 线控
----------------------------

### **倍量口香糖电池充电套装**

* 入手渠道：京东自购
* 参考价格：42 元

前段时间和首个工作单位的领导喝咖啡聊天，作为当年「叱咤中山电子城」的「新奇特数码产品」的领导自然有着自己的一套数码产品「折腾学」，当然作为我非常敬仰的前辈，能够和其共事自然是我的荣幸。

那天喝咖啡的半途，领导神神秘秘的掏出来一个铁方块，打开卡槽开关之后弹出的磁光盘以及夏普标志性的音频产品 Logo 让我一下子兴奋起来，要知道这可是我上大学前最想拥有的数码产品之一：MD 播放器，而且当时无论是音质还是设计都是顶尖的 1bit 系列 夏普 MD 单放机。

实际上最让我感到惊奇的是我的领导是怎么让这个古董启动起来了——作为深入玩音频数码的老鸟，自然明白这么纤薄的机身必然不是传统的五号电池，而当年锂电池也并不会内置在 MD 这个本世纪初的数码产品上，那么只有一种可能：领导买到了全新的口香糖电池！

其实玩过这种上古音频数码的老鸟都知道，当年大部分设计精美纤薄的卡带机、CD Walkman 或者是 MD 播放器都采用了口香糖电池，也就是所谓的「条电」，细长的造型却采用了和五号充电电池相同的电压和容量，这样机器就可以非常便于携带，要知道如果没有口香糖电池，那么机身上就需要外挂「尿袋」——实际上就是使用外挂的五号电池盒来供电，虽说通用性更好但就非常不便携了。

![](.evernote-content/2127235E-A08D-43E1-897D-15FED0AC89DA/BD3A8E43-3193-4E85-8B27-1EBCD5E6262E)

所以在领导的「蛊惑」下，我下单了同款的口香糖电池充电套装，这种十多年前都算是小众的玩意竟然还有国内厂商生产，更让我觉得惊奇的电池工艺有了很大的提高，以往的口香糖电池存在记忆效应，电池电量如果没用完就充电会造成容量降低，而新的口香糖电池虽然依旧采用了镍氢。但却不存在记忆效应，更适合现在这种数码产品的随用随充的使用习惯。

![](.evernote-content/2127235E-A08D-43E1-897D-15FED0AC89DA/A588D073-1D32-415C-93A5-45A14E0E3D1F)

而买的充电器也非常适合我这种使用索尼 EJ 系列 CD Walkman 的老玩家，相比后期的 CD Walkman 仅需要一颗口香糖电池就可以驱动，EJ 系列需要一次使用两颗口香糖电池，但以前的口香糖电池充电器只能一颗颗的充电，而倍量的这款充电器可以一次充两颗口香糖电池，并且采用了 microUSB 接口，所以也可以使用现在的手机充电器进行充电。

总的来说，如果你跟我一样是一个老 CD Walkman 玩家并且里面也使用的口香糖电池，那么这个上古时代的「新玩意」或许能让你的老机器复活。

### **索尼 Walkman 线控 RM-MC33EL**

* 入手渠道：淘宝
* 参考价格：150 元

在搞定了我的 CD Walkman 的口香糖电池之后，让 CD Walkman 更方便控制的线控更新也提上日程，实际上无论是 CD Walkman 还是 MD 单反机，为了做的更加的薄，索尼将很多高级控制功能都交给了线控——比起现在耳机上的简单的播放盒音量条件，CD Walkman 的线控还可以实现音效更改，播放选歌等更为复杂的操作。而不同版本的线控实现的功能也不尽相同。

而说到线控其实就是一个棒状控制器，从现在的角度来看，这种 CD Walkman 上的线控其实影响了后期很多音频播放器的设计，而索尼之后推出的 MP3 产品的设计灵感也来源于线控。早期的线控没有屏幕，而后期的较为高阶机型的线控则采用了点阵单色液晶的小屏幕，甚至还带有背光方便晚上进行操作。

![](.evernote-content/2127235E-A08D-43E1-897D-15FED0AC89DA/EC0F345E-DFE2-454A-A649-0D3B22FF677E)

![](.evernote-content/2127235E-A08D-43E1-897D-15FED0AC89DA/28CB94F7-6748-4A53-BB73-91835E7B07C6)

我的 CD Walkman 之前搭配的是后配的 RM-MC38EL 线控，这款线控除了带有液晶有背光小屏幕之外，还支持半角字符，有些 CD 碟的曲目名称采用了日文或者中文文本的歌曲信息，这款线控就可以将这些文字显示出来。不过这么多年用下来线控的通病就出来了——串键，有时候播放按键会变成快进，非常的影响使用。

不过线控实在是太少了，最后好不容易搜索到了一个近似的型号的线控： RM-MC33EL，考虑到这玩意现在买一根少一根，加上必须解决原来线控的串键问题，所以用近乎收藏品的价格抓紧入手了一根。

![](.evernote-content/2127235E-A08D-43E1-897D-15FED0AC89DA/73DE803F-2B46-4A89-BA9E-33EA3EE4FF18)

实际上功能上对比后，放在 EJ985 这种早期的超薄 CD 机上，两款线控功能其实都差不多：拨杆形式的选歌，都带有背光的点阵液晶显示屏以及一键切换音效（只有重低音音效），唯一的区别就是估计就是颜色——原先的 RM-MC38EL 线控是银色，而 RM-MC33EL 线控是黑色的。

至于文件夹选择等功能就只能在后期的 NE 系列 CD Walkman 以及 Net MD 上可用（因为那时候的机器开始可以直接解码音频文件了，类似将音乐文件刻录到光盘/磁光盘上然后播放），所以在 EJ 系列机型上会映射成其他功能键，当然最高兴的还是解决了原来线控的串键问题，这样我背包出街的时候可以将线控放在裤子口袋里面，然后在口袋里面继续盲操机器听歌了。
