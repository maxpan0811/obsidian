# 苹果助力，软件订阅制抬头，现在的 Setapp 已经足以引起你的重视

---

在苹果宣布更改分成政策不久，Clean My Mac 的开发商 MacPaw 就推出了 [Setapp](https://setapp.com)。它当时内置了近 50 多款软件（现在已经增加到 90 多个），每个月收费 9.99 美元，可以理解为小型的「订阅制版的 Mac App Store」。

![](.evernote-content/C5BBBAA8-781F-4F29-9BEE-1092DB2B616D/1DBA9459-C248-4455-B9D6-6D1E5DDD8E02.jpg)Setapp 上的部分生产力工具

起初我是不看好 Setapp 的。因为对于老的 Mac 用户来说，大多数我们要用的软件都已经买过了，Setapp 只能对那些刚买 Mac 的人有经济上的吸引力，但实际上这样的人又很有可能不会接触到 Setapp。

**但现在我这个看法动摇得非常厉害。**

首先是 [Ulysses 转为订阅制](https://sspai.com/post/40361)，这**让 Setapp 的性价比大大提升**。Setapp 已经支持了[学生优惠](https://setapp.com/educational-discount)，价格是 $4.99/月，这就是 Ulysses 一个月的价钱1 ，其它软件几乎可以视为白送2 …

其次，最近任务管理工具巨头之一 2Do 入驻 Setapp，补上了这一短板。何况 2Do 也[在制作 Web App](https://twitter.com/2DoApp/status/840457097531817984)，想必也是有意向移步云端，届时应该也会动订阅制的心思。

![](.evernote-content/C5BBBAA8-781F-4F29-9BEE-1092DB2B616D/8F32BC36-B59D-4EFF-948B-1C5FCB9F7F6D.jpg)2Do 与 Setapp 联姻

诚然，Setapp 里有一些混 Bundle 的货，但也有够吃的硬菜：

![](.evernote-content/C5BBBAA8-781F-4F29-9BEE-1092DB2B616D/F474E8CF-6A3B-46A7-9DA0-AD297D7251CE.jpg)

* CleanMyMac 估计是 macOS 新手的必备工具，很多老手不想手动清理估计电脑上也会常备一个；
* iStat Menus 在监控系统信息方面，美观、易用、全面三者综合起来仍无人出其右；
* Downie 是口碑最好的视频下载工具之一，Youtube 甚至能检测列表一并下载，对国内的支持得也不错；
* Ulysses 虽然改了收费方式挨骂，但在苹果环境下，论功能完备与精致，依然是写作工具的王者；
* Gemini 是不会整理文件的小白的福音，能帮你找到系统中重复的文件（Setapp 中的另一款软件 Disk Drill 也有同样的功能）；
* 2Do 已经说过，是任务管理工具的巨头之一，其亮点在于过滤规则全面、iOS 交互设计趁手（层级少），拥趸颇多；
* MoneyWiz 是全平台记账工具中的佼佼者，功能完备报表丰富，少数派有专门的教程；
* iMazing 是 macOS 上的 iOS 文件管理工具中设计得最出色的一个，也是跟进更新最及时的；
* Marked 是 macOS 上 Markdown 文件的预览工具，特长在于可以自定义 CSS 并且导出单页的长 PDF，分享给人十分舒服；
* TaskPaper 是主打用文本输入的形式做任务管理的工具，没有图形界面所以输入起来就像写作一样顺手，任务管理方面又支持灵活的标签、项目与语法过滤，[Marco Arment 这样的人也在用它](https://twitter.com/marcoarment/status/712701293266149376)；
* ……

除了一哥级的软件，Setapp 中也不乏第二梯队的工具：

![](.evernote-content/C5BBBAA8-781F-4F29-9BEE-1092DB2B616D/24A8DDA9-EA01-4F98-8A0C-C60921A56C0C.jpg)

* 脑图阵营人气最旺的 MindNode 虽然不在，但 Setapp 有 iThoughtsX 和 XMind；
* 功能一般，颜值最高的剪切板工具 Paste 也在 Setapp 里；
* OmniOutliner 短期内不用想了，但是 Setapp 里有做大纲也很顺手的 Cloud Outliner；
* Timing、Mosaic、Unclutter、ForkLift、Alternote、Folx、泼辣修图、Screens…… 都是能够满足用户基本乃至进阶需求的好工具。

随着 macOS 的大版本更新降至，我们又将硬来付费升级的季度。Ulysses 这个浓眉大眼的叛变后，打订阅制主意的软件恐怕也会多起来，毕竟苹果都用分成这么露骨的招儿来引诱开发者了。

里外算算，说不定有人真是走 [Setapp](https://setapp.com) 更划算，特别是学生。
