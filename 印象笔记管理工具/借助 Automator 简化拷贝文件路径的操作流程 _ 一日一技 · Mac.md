# 借助 Automator 简化拷贝文件路径的操作流程 | 一日一技 · Mac

---

![](.evernote-content/1F0E303A-4FCD-4312-AA7D-FCE19902D23B/4B02DB97-DFF5-4D9C-A4C4-7E74D5DB6A76.jpg)

在 Mac 上，想要查看文件所在路径不是什么难事，你可以按下 `Command ＋ I` 在文件的详细信息中查看，或是勾选 Finder 中的「显示路径」选项。不过，如果你想把文件路径以文本格式拷贝到剪贴板，就会比较麻烦了。

一个可行的方案是：打开「终端」，将文件拖拽到开启的「终端」窗口中，然后将终端中显示的文件路径拷贝到剪贴板。但这个方法的操作步骤还是比较多的，实际使用时很不方便。

今天向大家介绍一个更快捷的方法：在右键菜单中添加一个「拷贝文件路径」选项。以后需要拷贝文件路径时，只需右键单击文件即可。下面的图片展示了最终的效果：

![](.evernote-content/1F0E303A-4FCD-4312-AA7D-FCE19902D23B/1A33578D-CB7C-4405-8966-EEB7D836C4EE.jpg)

准备工作
----

操作过程中，我们需要用到 Automator。如果你对 Automator 不熟悉，你可以在应用程序文件夹中中找到 Automator 或是直接在 Spotlight 中搜索。

![](.evernote-content/1F0E303A-4FCD-4312-AA7D-FCE19902D23B/A9EB76EB-2014-4E5F-902F-8B9BFE0053D3.jpg)

![](.evernote-content/1F0E303A-4FCD-4312-AA7D-FCE19902D23B/48FDC2B1-55B7-4E91-8BD4-20CD3EC7A264.jpg)

操作方法
----

1. 打开 Automator 后，选择新建文稿，并在弹出的界面中选择「服务」；

![](.evernote-content/1F0E303A-4FCD-4312-AA7D-FCE19902D23B/6AA12C31-EECB-48E7-93E9-49C200219480.jpg)

2. 创建文稿后，在左侧边栏资源库下的「实用工具」中找到「拷贝至剪贴板」并双击添加；

![](.evernote-content/1F0E303A-4FCD-4312-AA7D-FCE19902D23B/886EF71B-9441-4CD7-B1E4-B68210C9E34E.jpg)

3. 按照下图设置服务的应用范围；

![](.evernote-content/1F0E303A-4FCD-4312-AA7D-FCE19902D23B/8D41FA11-1DCF-43CA-8892-90F9969DAC73.jpg)

4. 按下`Command + S`保存文稿，在弹出的对话框中输入「拷贝文件路径」或是其他你想要的内容，然后关闭 Automator；

![](.evernote-content/1F0E303A-4FCD-4312-AA7D-FCE19902D23B/4364CE50-69AC-4387-BB5C-E2300EE8B96D.jpg)

5. 现在试着右键单击任意文件，看看是否已经有了「拷贝文件路径」选项，如果没有的话，继续按照下面的步骤操作；

6. 打开系统偏好设置，选择「键盘 - 快捷键 - 服务」，找到我们刚才添加的「拷贝文件路径」并勾选（如果找不到这一项的话，尝试通过勾选或取消勾选其他项目刷新列表，然后就可以看到新添加的服务了）。

![](.evernote-content/1F0E303A-4FCD-4312-AA7D-FCE19902D23B/0588E377-6237-4F7C-8A47-5087BE8938C7.jpg)

类似的，停用这一服务只需取消勾选即可。如果你希望彻底删除服务文件，可以右键单击希望删除的服务，在弹出的列表中选择「在 Finder 中打开」，然后删除对应的文件。

现在，右键单击任意一个文件或文件夹，就可以看到我们刚刚添加的「拷贝文件路径」选项了，你可以将拷贝的内容粘贴到任意应用程序中。你也可以同时选择多个文件，这种情况下，这几个文件的路径会同时拷贝到剪贴板中。

注：我是从 Topher Kessler 的 [《How to copy a file path in OS X》](http://www.cnet.com/how-to/how-to-copy-a-file-path-in-os-x/) 中了解到这一技巧的，技巧的原作者不详。
