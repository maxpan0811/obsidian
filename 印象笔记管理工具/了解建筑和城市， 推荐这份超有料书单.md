那么，想让孩子去触碰这个精彩的世界，你需要这份**具有 “独特气质” 的建筑书单。**当然，这份书单不仅仅是对孩子，对于想**了解各地建筑和建筑师行业**的大人来说，下列的书单也颇有帮助。

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/A44E8E9F-A060-45EF-9787-6576183B3FEB)

**让建筑来讲它自己的故事**

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/3385770C-6E25-460E-B5C1-30F3B414DFB4)

**《建筑的故事》**

[英] 帕特里克・狄龙 著

一部手绘全球建筑史，剖面图大师图解 16 座传世建筑

这是一本集建筑历史、建筑技艺、建筑表现手法于一体的科普故事书。

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/7AF42F1C-B94A-45BA-AB92-4BEB0F8B4BFA)

人们常常关注发生在建筑里的故事，殊不知**建筑它本身，也是故事。**

普利策新闻奖获奖者、英国建筑师与历史学家帕特里克・狄龙为你讲述建筑的故事。

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/F56C746C-D61C-4BAB-A042-B717B76EF7CA)

**本书通过等比例超精细手绘剖面图为人们揭秘历史上重要建筑背后不同寻常的故事，构建对建筑价值的认识。**波谲云诡的大时代背景、建筑流派的螺旋式演化、16 座传世建筑的前世今生、文艺复兴、美学革命和工业革命对建筑影响都被一一解说。所有这些细节都将带领读者进入熟悉而又陌生的建筑世界。

推荐理由：
-----

1. 普利策奖获奖作者与剖面图大师联袂创作。

2. 俯瞰建筑 3000 年，了解人类磅礴历史的另一面。

3. 宽幅全彩剖面图图解建筑，用建筑的语言去读懂建筑。

4. 在宏观视角中穿针引线，呈现关于建筑和建筑史的惊人细节。

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/6B841A0A-2BE2-4965-9A71-EBF000DB8A2E)

**建筑的形式是多种多样的**

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/175BFB5B-C6A3-4127-8768-90CD84AE6970)

**《如果你住在这里》**

[美] 吉尔斯・拉沃什 著

走进 16 栋世界各地的奇妙房子

一本轻松的建筑科普书，精妙恢弘的浮雕剪纸拼贴画，带你走进**世界各地的房子，探索它们的生活和秘密。**

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/F1087247-7716-4E4C-AF47-9583A672468B)

如果你住在威尼斯的水上宫殿里，你得划船才能去上学；

如果你住在荷兰的漂浮屋里，你可以旋转你的房子，在同一扇窗看日出与日落；

如果你住在覆盖着厚厚积雪的牧人小屋里，冬天也不会觉得寒冷……

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/FB2C3A3D-C66F-4494-8E76-F935121345E9)

推荐理由：
-----

1.“堪称艺术品” 的科普绘本，以精妙恢弘的手工拼贴画展现世界各地 16 座细节丰富、宏伟壮观的特色建筑。

2. 以 “这是什么房屋类型？”“它是由什么材质建造的？”“它坐落在哪里？”“它是什么时候建造的？”“关于它的一个有趣的事实？” 五个问题对世界各地的房子进行介绍，纸上浮雕艺术的表现形式从另一个角度表达出建筑除了人类的居所同时也是艺术的容器。

3. 不仅可以了解建筑结构，欣赏纸上浮雕艺术，同时能激起你对世界的理解、想象与热爱。

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/FE1F4388-2F4A-407F-B605-916317C3E037)

**建筑是文明的产物**

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/66A9CD22-258A-44F6-8232-AD9BF4DA83BD)

**《鸟瞰古文明》**

[法] 让・克劳德・戈尔万 著

130 幅城市复原图，重现古文明全景

一座座建筑物构成建筑群，然后逐渐形成了城市，历史上每一座伟大的城市都淋漓尽致地体现了所在区域的**地域特点和人文风貌。**

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/71289FCB-1EED-468E-B51D-2B95F9E9B47F)

这本书为人们复原了地中海区域的城市在被现代文明整理前的模样。

**130 幅城市复原图，让你发现建筑在历史上发挥的作用、感受城市肌理之美。**

推荐理由：
-----

1.130 幅复原图呈现古地中海文明三千纪的繁华与梦幻。

2. 古代城市复原图制作领域巨擘率领的专家团队，用纸笔精妙重构历史发生之地。

3. 图文结合，详解城市历史与其建筑之间错综复杂的关系。

4. 地中海古城版 “清明上河图”，用艺术的手法呈现历史之美。

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/1B88EB94-F3FC-4180-B059-DDBD0B213FAF)

**并非关于建筑而是关于人生**

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/FFB1609E-EE80-4FD5-97E1-DC7535F37A6C)

**《建筑师》**

[美] 大卫・马祖凯利 著

融叙事象征主义、色彩和视觉隐喻于一体的史诗级作品

如果人生是一张建筑图纸，属于你的那栋房屋会是什么模样？

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/74BDE02E-1C5C-46F3-AA75-41C65E8ED645)

这是一部关于风格、设计和视觉语言的**图像小说**，作者以一种极富实验性的方式来展现空间和色彩。建筑、雕塑、舞蹈、音乐、神话……

大卫・马祖凯利可谓图像叙事大师，将如此多的元素有条不紊地编织成一张巨大的叙事之网，从来没有一个人能将人性、灵魂探索、平面设计、哲学和幽默等关于漫画可承载的所有伟大内容都塞进了一本 340 页的书里。

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/2A858895-C767-40FA-98B2-F2A84B97C3AE)

推荐理由：
-----

1. 一次中年危机，一场自我放逐，一个 “奥德赛式” 的史诗故事。

2. 一场视觉实验，一部艺术之书，被誉为 “年度最华丽漫画”。

3. 每个人都是生活的建筑师，设计也由人出发，然后回归到人。

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/D093C399-FFCA-4234-B019-751EB24B4B38)

**相似的建筑，多样的表情**

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/B97C8D96-2051-432D-8517-BFFFD856F7BF)

**《东京老铺》**

[波兰] 乌尔巴诺维斯 著

发现一个有味道的老派东京

插画师乌尔巴诺维斯的插画作品集。他游历东京近两千家老铺，从中选出有美学个性的五十家，用近百幅门面全景、细节分镜、店内透视图，**手绘出如新海诚动画般唯美浪漫的昭和风情。**

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/F889BB0A-2ED6-491A-95A4-AC1640414572)

你可以从这些铺面插画中发现材质、色彩配上合适的比例所营造出的建筑表情如此丰富，平凡的建筑也变得精彩起来。

推荐理由：
-----

1. 新海诚动画《你的名字》、《天气之子》背景的美术绘师作品。

2. 在 50 家长寿企业的铺面表现中，感受日本文化和民族精神，体会到政治、经济、文化在民用建筑上留下的痕迹。

3. 清新的手绘画风对手绘练习有所帮助。

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/88C9B4F7-8D7A-4D33-8700-CCDD5A854D8E)

**建筑的诞生之路**

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/CAD7D40A-9109-46C7-8373-CD31E530C364)

**《伟大建筑手稿》**

[英] 海伦・托马斯 著

一部手绘全球建筑史，剖面图大师图解 16 座传世建筑

灵感迸发时需要及时记录下来，这是许多建筑师随身携带笔记本和钢笔的原因。

《伟大建筑手稿》是**横跨 5000 年文明发展史的设计美学巨著**，一部凝聚人类最高灵感的集大成之作。

本书收藏 276 张罕见的经典建筑手稿，剖视了 260 多位艺术巨匠的设计个性。

![](.evernote-content/DF825686-7429-4F58-A140-A5939CE0C1F1/E9546784-476A-4D85-978A-5CA344BAA449)

第一次把建筑按文化渊源进行关联编排，每一对开页都将不同时代、风格和地点的建筑之间建立联系和对比，华丽铺陈建筑最初的细节之美，娓娓道出不为人知的发展过程，讲述了人与建筑、建筑与时代的传奇故事。

推荐理由：
-----

1.276 张经典建筑手稿直观地展现了设计师们是如何记录灵感并完成创作的。

2. 在找寻上下图中关联因素的过程中，发觉建筑设计的层次感，深入理解建筑是如何被设计出来的。

3. 用于膜拜。

建筑是一个复杂的容器，它或许蕴含某种文化、承载着某种精神、表达着某种情绪，**每个人的理解都不同，所以希望这个书单为大家打开建筑世界的大门！**
