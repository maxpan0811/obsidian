# macOS Mojave发布：比起iOS 12 它才是惊喜

---

在 WWDC 开始前，人们都表示 macOS 10.14 将会迎来一个大升级，比如 iOS 的应用将能够在 macOS 上运行、又比如黑暗模式。但是今天的发布会下来，我们发现除了以上两个重大更新之外，macOS 10.14 还有很多亮点。

[![](.evernote-content/FA3499F9-7F09-4546-8D4D-BF7C9529027F/E6F6AF8D-007F-4FB8-B89B-6897CD0E3CCB.png)](https://resource.feng.com/resource/h062/h31/img201806050444330.png)

**黑暗模式**

最开始我们还是来看看命名，苹果方面也表示从六年前开始 macOS 就开始以地名来命名了，比如优胜美地、酋长岩 和 High Sierra 等等，但是此前这些地名都是山峰的名字，今年苹果将会换换口味，采用一个新的景点来命名 macOS——莫哈韦沙漠（Mojave）。

[![](.evernote-content/FA3499F9-7F09-4546-8D4D-BF7C9529027F/B87DE727-B1AD-4121-ACEC-0E7BF625BBE3.jpg)](https://resource.feng.com/resource/h062/h31/img201806050442070.jpg)

先来看看黑暗模式，这不仅仅是色调变成黑色的所谓“夜间模式”，苹果对所有的 UI 都相应做出了调整，全部都和黑色做出适配，并且对于程序员来说 Xcode 的黑色模式显然是最完美的。甚至 macOS Mojave 的壁纸还能够根据时间来变化，毕竟沙漠中每天的光影都是非常美妙的。

**Sneak Peak**

除了黑暗模式，最重要的功能肯定是 iOS 应用“搬家”到 macOS 了，这一功能被称作 Sneak Peak。苹果将 iOS 的 UIKit 框架的一部分加入到了 macOS 的 AppKit 上，这意味着 iOS 的 App 将能够在 macOS 环境运行了。苹果表示在第一阶段，股票、新闻、录影机等本属于 iOS 的 App 就是通过 Sneak Peak 移植到到 macOS 中的。

[![](.evernote-content/FA3499F9-7F09-4546-8D4D-BF7C9529027F/0C92A122-A30E-41E3-901B-0F44F4CA2739.jpg)](https://resource.feng.com/resource/h062/h31/img201806050440110.jpg)

第二阶段会在 2019 年开始，Sneak Peak 届时将会向第三方开发者开放，所以未来 iOS 应用都将能够在 macOS 上运行。**同时苹果也第一次坚定地声称绝对不会把 iOS 和 macOS 进行融合！**

**[![](.evernote-content/FA3499F9-7F09-4546-8D4D-BF7C9529027F/EC9D3CFF-B2C9-4B23-9E58-BA1F9E97F2C3.jpg)](https://resource.feng.com/resource/h062/h31/img201806050443060.jpg)**

**桌面、Finder 大升级**

本次 macOS Mojave 也有一些非常好用的小功能升级，比如桌面就有一个非常重要的功能——自动整理，macOS 用户常常喜欢把文件都放在桌面，现在自动整理能够把相应的文件进行分类，用户能够以文件类型、标签、日期等方式进行分类，拯救你杂乱的桌面，并且文件拖拽到桌面之后也会自动归类到相应的文件堆中。

[![](.evernote-content/FA3499F9-7F09-4546-8D4D-BF7C9529027F/5E4BD8AC-659C-4E07-B9A5-97C4DFD0D1D7.jpg)](https://resource.feng.com/resource/h062/h31/img201806050438290.jpg)

Finder 文件管理器采用了新的设计，特别是新的图片浏览模式更加好用了，相册模式中能够快速查看图片信息，还在右下角内置了快速编辑功能。并且在一次性选择多个图片文件时，有下降的快捷按钮能够让你一键创造一个 PDF 文件。并且文件预览中也集成了编辑功能，比如 PDF 文件的预览模式下就能改动文字，预览视频文件也能够按帧截图。

**[![](.evernote-content/FA3499F9-7F09-4546-8D4D-BF7C9529027F/8FBF3046-7E8A-48AA-BCB5-D08190AD3E27.jpg)](https://resource.feng.com/resource/h062/h31/img201806050442340.jpg)**

**新功能：截图、iPhone 摄像头**

macOS 的截图功能有更新，现在的截图并不会直接保存，而是会在左下角显示缩略图，用户能够点开它做一些简单的处理，功能和 QQ 截图十分相似。但是 macoS 的截图现在也新增了录像功能，录屏之后甚至能够直接将录屏视频拖到 Keynote 里，很方便。

[![](.evernote-content/FA3499F9-7F09-4546-8D4D-BF7C9529027F/4ADC60EB-1C89-49C0-B5A7-1DFDCFC9799F.jpg)](https://resource.feng.com/resource/h062/h31/img201806050442190.jpg)

而 Continuity Camera 可以在 macOS 中控制 iPhone 拍照，然后 iPhone 拍出的照片会自动进入到 Keynote 中，也就是说把 iPhone 变成了一个 Mac 自带的摄像头，拍一些文件什么的插入到文档、PPT 中会很方便。

[![](.evernote-content/FA3499F9-7F09-4546-8D4D-BF7C9529027F/E0549513-8BBC-44AE-8D1F-CB88E03D55CB.jpg)](https://resource.feng.com/resource/h062/h31/img201806050438480.jpg)

**App 更新**

新闻、股票和录音机 App 进入了 macOS 了，设计和 iPad 版本非常相似，多人 Facetime 功能自然也出现在了新版 macOS 中。并且 HomeKit 也同样支持 Mac 了，现在 macOS Majave 中也能够通过 Mac 控制家里的智能家居设备。

[![](.evernote-content/FA3499F9-7F09-4546-8D4D-BF7C9529027F/07472EDB-815A-48D8-B85D-E01AB41D268F.jpg)](https://resource.feng.com/resource/h062/h31/img201806050439190.jpg)

已经在155个国家普及的 Mac App Store 也重新设计了，新设计下侧边栏分类更加明确，评论更加直观。同时微软的 Office 365 、Adobe 的 Lightroom CC 都将被带到 Mac App Store。

**隐私再次进步**

macOS Majave 上用户能够控制更多权限可，除了相机、地理位置、照片、浏览器和日历等权限之外，用户现在也能够通过 macOS 限制应用获取邮件、短信、系统 Cookie 等敏感信息了。

[![](.evernote-content/FA3499F9-7F09-4546-8D4D-BF7C9529027F/10BE07FD-0573-4EF1-A020-73CE49782E4C.jpg)](https://resource.feng.com/resource/h062/h31/img201806050439570.jpg)

同时苹果也强调，去年 Safari 上线的 ITP（Intelligent Tracking Prevention）功能会禁止广告商获得 cookie 信息然后向你推送广告，从那以后 Safari 用户就脱离了广告的苦海。今年这一功能将会在 Mac 上全面普及，你的 Mac 在外界看来将不会有任何特别之处，广告商会对你无从下手。

**Metal 2 、Creat ML**

外置显卡也支持 Metal 2 了，通过外接显卡，iMac Pro 的性能甚至还能够快三倍以上。而 Creat ML 工具能够让开发者创造自己的机器学习、人工智能应用，通过 Creat ML ，一款 AI 应用的训练时间能够从24小时缩短到18分钟，这无疑是一个大跨越。

[![](.evernote-content/FA3499F9-7F09-4546-8D4D-BF7C9529027F/4A24399C-1885-49AF-A307-CBE4C0F9454F.jpg)](https://resource.feng.com/resource/h062/h31/img201806050439070.jpg)

以上就是 macOS 10.14 ，也就是 Mojave 的全部新功能了，这无疑是史上最易用的一个 macOS。并且一定程度上也反映了未来 macOS 的一大趋势：虽然不会和 iOS 融合，但会越来越多地依赖 iOS 的资源。

现在 macOS Mojave 的第一个测试版已经向开发者开放，大家能够前往开发者中心下载，但是考虑到这是早期版本，兼容性或许并不是很好，建议不要在主力机型上安装。
