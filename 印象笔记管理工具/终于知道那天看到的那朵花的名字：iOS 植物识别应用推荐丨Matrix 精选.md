iOS 发展至今，APP 数量早已数不胜数，应用功能也相当丰富，各类小众爱好者几乎都能在 iOS 平台上找到自己想要的应用，「植物」也算是其中的一个小众分类。

相信大多数植物爱好者的好奇心总是满格，去任何地方都少不了观察植物。你走在旁边和他说话，经常会发现根本像在和空气说话，因为他们的注意力都在路旁的植物上。

早在诺基亚 Symbian OS 时代，我就非常期盼有一款应用能帮助识别植物。遇到陌生物种，除了拍下来，还有别的办法么？回去用电脑网页版的图片搜索功能，请教植物达人，实在不行靠自己查植物图谱，反正就是不搞清楚绝不罢休。但不管怎样都还是不如当下就了解它来得痛快。

拥有这款叫作 LikeThat Garden 的 APP 你会有如神助！此前也有极个别中文 APP 称能识别植物，但都有个通病就是功能臃肿，识别效果还不尽人意。LikeThat Garden 的 UI 非常简洁，仅有三个主要功能——识别植物或蝴蝶，查看附近和自己识别成功的植物；在 iPhone 6 上安装使用，除了在进入拍摄搜索时稍微卡顿，其他表现都较出色。

* 查看附近和自己匹配成功的植物，可参照数据丰富

![](.evernote-content/785302F8-83AD-46B5-A781-C73DC66D88CD/90B258EA-54B5-4A5C-9AAF-66C3E632487C.jpg)

* 确认识别成功后查看更多信息，跳转维基百科「 需爬梯」

![](.evernote-content/785302F8-83AD-46B5-A781-C73DC66D88CD/5A7F47C1-F29D-47B4-9312-EFF64FBD40B9.jpg)

* **为了让匹配数据越来越准确，我倡议广大爱好者多一些研究精神，谨慎匹配。**例如下图有两种匹配建议，「Johnny JumpUps」是什么暂且不谈，第二建议是「报春花」，去任何搜索网站反向验证都是不匹配的。一个人的错误配对，会导致后来人的盲目相信，久而久之会对应用的准确性和用户体验造成伤害。

![](.evernote-content/785302F8-83AD-46B5-A781-C73DC66D88CD/8D2FAFE3-5627-4E27-A280-BC0FF6E96153.jpg)

*科普时间：「Johnny Jump Ups」加州金堇，是堇菜属的一个种，「Viola pedunculata」是拉丁文学名；而我识别的花其实是三色堇「Viola tricolor」，是堇菜属的另一个种。应用给出的建议不一定是正确答案哦，但能帮你更快找到答案。*

这家叫 JustVisual 的公司同时开发有其他 4 个可视化搜索的应用，能识别宠物、服饰、家具等，官网上显示它们还给企业提供可视化搜索的 API 接入服务。

移动互联网兴起至今，产生了众多曾经想象不到的用户需求，比起玩乐消遣，我更愿意为这类能展现了不起的技术力量并解决实际问题的公司点赞，在商业和技术的平衡上，它们需要付出更多。
