# 利用 Automator 在 Mac 上快速批量压缩图片丨一日一技

---

![](.evernote-content/D4F4CB64-E0B5-45CC-BD33-9FCB3F49B339/5EF843BD-3832-4A57-947B-42F22F790B6F.jpg)

大家都遇到过各种论坛的坑爹限制，比如单图最大不能大于 X MB，这时候，无需借助第三方软件，利用 Automator 就可以快速进行批量图片压缩。

1. 打开 Automator。新建一个 Workflow，选择 Folder Action（文件夹操作）。

![](.evernote-content/D4F4CB64-E0B5-45CC-BD33-9FCB3F49B339/A5DB0A43-0BAC-4042-92EA-88F645FC4369.jpg)

2. 然后把要压缩的图片拖进 Automator 右侧的工作框。

![](.evernote-content/D4F4CB64-E0B5-45CC-BD33-9FCB3F49B339/350C6A59-85F4-4D28-B55A-B1C733AB8575.jpg)操作完成后如下图所示：

![](.evernote-content/D4F4CB64-E0B5-45CC-BD33-9FCB3F49B339/559014C3-3153-42F5-97A0-4982E2234800.jpg)3. 然后选择 Photos（照片）-  Scale Images（缩放图片），把它拖动到右侧的 Workflow 中。这时会让你选择是否对原文件进行一次拷贝，看每个人需要不同自行选择。

![](.evernote-content/D4F4CB64-E0B5-45CC-BD33-9FCB3F49B339/06950946-22F5-442A-8817-3523A989BC24.jpg)

4. 随后可以选择「缩放到指定像素」或者「按百分比缩放」。

![](.evernote-content/D4F4CB64-E0B5-45CC-BD33-9FCB3F49B339/AD7AE543-4ADF-4FA9-B1C1-8F3F20F5AAAD.jpg)

5. 确定配置完成后，点击右上角的 Run（运行），任务完成！

![](.evernote-content/D4F4CB64-E0B5-45CC-BD33-9FCB3F49B339/2C2A8853-224B-40D2-A404-A2BEE7D08251.jpg)

你可以保存一下这个 Workflow 方便以后运行。
