DFX 5: 玩转打开/保存窗口，提高文件操作效率 | 领客专栏·Mac 玩儿法
----------------------------------------

*2016-01-19* *应用推荐第一媒体*

Default Folder X 这款老牌 Finder 增强软件在近日正式发布了 5.0 版本，进行了代码重写，完美支持 El Capitan（最低支持 Yosemite），而且不需要用户再去关闭 SIP 就可正常使用，2015 年 6 月 1 日之前购买的用户需付费 14.95 刀购买升级，6 月 1 日之后的可免费升级。

作为经常「混迹」于各大 Bundle 的老兵，Default Folder X（DFX）的特色似乎并没有受到大多数用户的关注，但如果你有频繁的文件打开/保存操作需求，DFX 会满足你所有的愿望，而且在 5.0 上新功能还在支持添加。

首先，你要知道 DFX 并不是一个像 TotalFinder 那样的独立型软件，它只在文件「打开」与「存储」窗口内出现，帮助用户快速打开文件或为文件快速选择一个位置保存，在这个过程里力求省去寻找目录（特别是常用目录）的时间，同时也提供对文件进行就地处理，比如像拷贝路径值、重命名、定位到 Finder、删除、压缩等操作。

新款 DFX 让我遗憾的是 Logo 都不更新一下，真是老旧的一塌糊涂，竟然还用玻璃反光特效，放在 OS X 里显得特别不合时宜，在「打开」&「存储」窗口上仍然是原来的布局，左侧是一个 「Drag Zone」，可以将常用的文件/文件夹拖入到「+」按钮上，这些文件/文件夹就会常驻在左侧栏，方便用户频繁打开。

右侧是一个由 6 个功能按钮组成的菜单，分别是文件操作菜单，目录定位菜单，收藏目录菜单，最近打开目录菜单，最近打开文件菜单以及打开状态下 Finder 目录跳转菜单。底部是文件的处理 Tab 菜单，提供了预览、信息查看、Tags、Comments 以及权限修改操作。看下图：

如果你觉得默认界面有些臃肿，你可以在后台选择 Minimal 界面：

DFX 还有一个挺有意思的功能，就是当你把 DFX 窗口与已打开的 Finder 窗口同时放置于主屏幕时，你可以直接将光标放在 Finder 窗口，此时 Finder 窗口就会处于灰色选中状态，单击一下即可迅速将 DFX 窗口内的目录同步为 Finder 窗口内的目录。

DFX 除了「寄居」在 Finder 「打开/保存窗口」之外，还在 Finder 内安插了两个插件，一个是类似于 Menubar 上的目录跳转菜单，还有一个 Drag Zone，作用和主体 DFX 功能一致，只是不能直接双击文件进行查看。

其实我觉得 DFX 的名字来历并不是上述介绍的窗口设计，而是在后台提供了 Default Folder 设置功能，简单的说就是你可以为每个应用程序设置默认的「打开/存储」目录，就像下图这样：

由于我们操作每个应用程序时，特别是在保存文件的环节，不同的程序往往需要保存到不同的目录位置，像我来说，Chrome 我喜欢将下载的文件保存到「下载」目录，Ulysses 文本保存在「文稿」目录，以此类推，非常好用，省去了许多频繁切换目录的操作，此外还提供了「只有保存指定格式后缀的文件」才自动归入默认目录的选择。

Default Folder X 的个性化设置还有很多，还有像为每个操作设置热键、排除 DFX 应用范围等功能，我们在这里就不一一祥述，大家可到官网下载试用版好好体验一番，试用期间没有功能限制。
