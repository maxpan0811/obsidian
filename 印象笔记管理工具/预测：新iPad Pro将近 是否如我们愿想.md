# 预测：新iPad Pro将近 是否如我们愿想

---

2018 年新款 iPad Pro 可能在 WWDC 发布，而分析师们已经预测，该机型可在 6 月份后期出货。  
  
说到平板电脑，市场上就只存在两种，一种叫 iPad，另一种是其他平板。在高价格区间，iPad 主导了整个平板电脑市场，平板机为了坚持自己的产品地位，苹果一直在使用新机型、新外设、新技术来使 iPad 保持新鲜感，包括 12.9 英寸 iPad Pro，Apple Pencil，120Hz 的屏幕刷新率和 ProMotion 触屏技术，正是这样持续不断的更新，在平板行业日渐低迷的年代，iPad 依旧能吸引众多消费者，也吸引着爱好者的目光。  
  
iPad 的基本设计在几年里都没有发生改变，除了变得更为轻薄，材质更高级，外观更具时尚感，iPad 从它诞生起到现在始终是我们记忆中的样子。2018 年，iPad 的老样子可能会由最新成员来改变，尽管苹果在很长时间里没有对 iPad 做出非常“惊人”的改变，但换句话说，苹果也在这段时间里琢磨着如何设计一款全新的 iPad 产品。  
  
**新改变**  
  
根据罗森布拉特证券（Rosenblatt Securities）的分析备忘录，一款带有 TrueDepth 传感器的新 iPad Pro（配有 Face ID）可能会在 6 月份的 WWDC 大会上公布，并将在当月晚些时候发布。预测这个时机是有一定道理的，因为第二代 iPad Pro 平板也是在去年 WWDC 上发布。  
  
目前还不知道新款 iPad 是否用跟 iPhone X 的“齐刘海”那样预留出顶部位置来放置 TrueDepth 传感器和前摄像头，因为传闻中它很有可能是全面屏——在笔记本都已经推出全面屏的当今，平板电脑采用全面屏似乎也不是什么令人惊讶的事了。  
  
这意味着 2018 年新 iPad 至少从外形上带来了新鲜感，和 iPhone X 保持风格一致的无 Home 键，窄边框，会让机子前面板整洁一体，提升美感。

**两型号 双尺寸**  
  
欧亚经济委员会的批准文件显示，苹果要销售两款我们从未见过的新 iPad，它们的型号是 A1893 和 A1954。虽然我们没办法从型号上获知什么具体的消息，这两串数字无论跟规格还是功能都没有直接联系，但是按照以往 iPad 的型号规律，同尺寸和同世代的 iPad 通常仅在最后两个数字上有所不同，而 A1893 和 A1954 则暗示新出的两款 iPad 不仅仅是同一个 iPad 的 Wi​​-Fi 和 Cellular 版本，而是两个不同的尺寸的新 iPad Pro。  
  
那么问题就来了，苹果今年里到底要发几台 iPad？一般来说，春季发布会和 WWDC 都是新 iPad 宣传的时机，而春季发布会偏向于常规产品线的升级换代，恰好 9.7 英寸的普通版 iPad 和 iPad mini 4 都需要更新一下，如果有的话，这个是它们的主场。而预测中的全面屏 TrueDepth 相机模块的新款高端 iPad 发布时机应该会选择在更为重要的 WWDC 上亮相。

**苹果的去指纹之路**  
  
苹果花大力气设计出了基于 TrueDepth 深度相机的 Face ID，用以取代指纹识别，从技术进步的角度讲，Face ID 大部分时候优于指纹识别，尽管 Touch ID 也同样优秀，但这抵挡不了苹果去指纹，推面容的决心。iPhone X 包括 iPhone X 后续机型都会把 Face ID 普及下去，那么 iPad 也不会放任不管。  
  
目前来自多方的消息称新款 iPad Pro 会采用和 iPhone X 一致的 Face ID + TrueDepth 相机不无道理，据称 Animoji 功能在新款 iPad Pro 上也有。  
  
另外，彭博社文章中还表示，今年 WWDC 上发布的 iOS 12 将出现让新 iPad Pro 生产力飙升的功能——标签式应用（tabbed apps）。这一功能将让 iPad 上的应用操作变得和在 Mac 上相似，同时打开多个 App 并且可以随时在它们之间切换，标签界面就有点像 Safari 的多个标签页一样。如果这个功能可以实现，iPad 的“电脑梦”才算真正有望实现。

**比A11更牛逼的A11X**  
  
我们都知道 iPhone X 以及 iPhone 8 都搭载了性能强劲的 A11 仿生芯片，从它们的跑分数据来看，性能已经远远超过安卓阵营。目前的 10.5 英寸和 12.9 英寸 iPad Pro 采用了基于 TSMC 10nm 制程的 A10X 仿生芯片，而 2018 年 iPad Pro 将跨代进化，它的核心是 A11X。  
  
A11X 的存在传闻已久，根据 MyDrivers 引用苹果供应链内的消息来源，这款芯片采用台积电最新的 7nm 工艺，而且是整合扇出型晶圆级封装（InFO WLP）。架构上，A11X 采用八核心设计，配置 3 颗高性能大核，5 颗节能小核，同时内建 M11 协处理器和 NPU（神经计算单元）。  
  
新 iPad Pro 大幅升级核心是必然的，如果前面所述的 iOS 12 标签式应用属实，又或者苹果想达到让 iPad 系统更加趋向于 macOS 的话，它的处理器性能需要足够强，也只有更高端的 A11X 能够承载这样的任务。

**还是LCD 没有OLED**  
  
显而易见，OLED 面板的产能决定了 iPad 系列短时间内都无法进行显示屏变革，今年传闻中两款 OLED 屏的新 iPhone X 就要收割掉大部分供应商订单，没有什么剩余产能可以分给平板电脑（即便有，OLED 大屏的成本也相当昂贵），今年的 iPad 依旧还是 LCD 屏，不过有 ProMotion 技术在，加上有可能进一步提升的屏幕刷新率，玩大型游戏和处理任务方面已经足以让 iPad 展现自己的所长了。  
  
总而言之，到现在为止不管是市场还是用户，都对苹果的新 iPad Pro 充满期待，平板行业更需要一针强心剂，这款集苹果最新黑科技，初步媲美笔记本电脑的 iPad Pro 到底能不能如大家所愿，相信在等待春季新品和 WWDC 这段日子，会有更多消息印证它的存在和强大，我们拭目以待。
