开源免费！面向PM的ClaudeCode交互式教程
-------------------------

翻到一个开源的 Claude Code 交互式入门教程

目前 800+ star，近期有更新，质量还不错

⬇️ 以下是简单介绍

一、 为什么 PM 也需要关注 Claude Code？

很多 PM 对 AI 的使用还停留在网页对话框。但真正的“AI 协同”应该是 AI 走进你的工作流。 claude-code-pm-course 这个项目，就是要把 Claude 变成你的“数字分身”。

二、 5W2H 全方位拆解

What (项目核心)： 一个交互式的、专为产品场景设计的命令行工具使用指南。

Why (解决痛点)： 解决 PM 在处理复杂 PRD、分析海量用户反馈、或者修改前端样式时，频繁切换窗口、上下文丢失的问题。

Who (受众)： 想提升技术理解力的 PM、追求极致效率的工作狂、以及想尝试“AI 原生开发”的朋友。

Where (场景)： 本地开发环境。它甚至能和你常用的 Obsidian 或 VS Code 联动，实现“视觉化”办公。

When (时机)： 这是一个 4-6 小时的进阶课程，适合周末花一个下午彻底打通 AI 办公的任督二脉。

How (上手路径)： 1. Git Clone 仓库。 2. 进入 course-materials。 3. 运行 claude 开启互动指令。

How Much (价值)： 免费的顶级教案。你学到的不仅是几个指令，而是如何像工程师一样与 AI 协作。

三、 课程亮点：不只是写代码

Agent 模式： 教你如何调用并行 Agent 处理复杂工作流。

自定义评审员： 创建你自己的“首席技术官”或“交互专家”子代理。

项目记忆： 利用 CLAUDE.md 让 AI 永远记得你的业务逻辑。

⬆️ 以上是简单介绍
