# 疯狂的Xeon芯新iMac 上ECC内存为更专业?

---

![](.evernote-content/9BBF4D1E-003A-4E2E-8C3D-2C198C3C2E8E/D360F75F-DA5A-46B2-9211-AAE9CBD20A7A.png)

随着苹果对 Mac Pro 发布计划的确定，有关另一款桌面苹果电脑 iMac 的相关细节也开始浮出水面。近日，大量新一代 iMac 的规格和升级方案被 macrumors 放出，有意思的是，这些信息均来自于一个不太知名的博客 “Pike's Universum”，声称 iMac 今年下半年临近 10 月底的时候将正式登场。

不得不说，iMac 是最值得期待的苹果电脑之一，毕竟不像 Mac Pro，此产品在普通消费家庭中更加普及。虽然昨天大多数锋友可能已经看到 iMac 传闻中的规格了，但是关于这份硬件列表依然有不少值得讨论的地方，尤其是全新主处理器对新特性的支持，下面就来一同唠叨一番。

**iMac 电脑不再提供酷睿系列处理器了？**

众所周知，苹果的 iMac 电脑产品线，传统所搭载的都是英特尔的酷睿系列处理器，包括 Core i3、Core i5 和 Core i7 各个型号。不过今天大多从英特尔 Core i5 芯片起步，现售最低配是搭载 1.6GHz 双核 Core i5 处理器的  21.5 英寸 iMac，最高则是 4.0GHz 四核  Core i7 处理器的 27 英寸 5K iMac。

![](.evernote-content/9BBF4D1E-003A-4E2E-8C3D-2C198C3C2E8E/66026BC8-DBF8-4C87-AB6A-B77D861693CA.png)

如果按照传闻所述，iMac 在英特尔芯片选择上跃进一大步，从酷睿系列直接升级到至强系列。虽然只是选择了入门级的至强  Xeon E3 系列，但是相比传统酷睿系列多了一些额外的企业级功能，主要有两点，第一是支持 ECC 内存，第二是支持 vPro 博锐技术。

所谓 ECC 内存也就是“纠错码”类型的内存，与普通的内存模块不同，ECC 模块提供额外的功能，允许其检查并控制数据的流动，进而纠正错误。 由于具有纠错码，此种类型的内存模块可以检测和纠正单比特错误，以及检测多比特错误。这与所谓的"奇偶校验位"有很大差别，后者仅识别错误，但不纠正错误。

简单的说，ECC 内存有助于维护数据的完整性，防止数据损坏，以及防止系统崩溃等，由于具有更高的可靠性，通常是面向服务器和工作站。

如果新一代 iMac 处理器确实真的从酷睿系列芯片转向至强系列，那么就个人而言，苹果主要希望提供的就是对 ECC 服务器内存的支持了，毕竟到 vPro 功能基本只在 Windows 阵营应用，与 Mac 关系不大。

**iMac 上 ECC 内存的意义在哪里？**

如果苹果希望改变 iMac 的定位，重新将其定位为“专业”计算平台，那么选择支持 ECC 服务区内存的至强 E3 处理器，必然是一件有意义的事情，相当于将原本在 Mac Pro 上至强 E5处理器所支持的技术特征，渐渐搬运到 iMac 上来。

但值得一提的是，尽管至强 E3 处理器能够支持 ECC 内存，但这些功能对普通消费者并无意义，英特尔在对至强芯片支持 ECC 内存的介绍中提到，ECC 功能支持下的嵌入式计算机模块是要求极高的医疗技术、服务器系统和航空业等领域应用的理想选择，也就是面向专业人士而非普通个人用户。

![](.evernote-content/9BBF4D1E-003A-4E2E-8C3D-2C198C3C2E8E/B3236718-C6DE-4892-9FC0-8F5A9FA1A377.jpg)

那么，苹果为何还要为 iMac 搭配至强处理器和 ECC 内存呢？其实道理很简单，苹果就是希望这些全新更高级的特征，能够简单抬高 iMac 电脑的平均售价，就如同 Mac Pro 本身就很高的定价一样，苹果从中赚到的利润更高。

问题是，苹果会强制涨价让所有对新一代 iMac 有所期待的买家负担成本吗？或者说苹果是否还会同时提供“酷睿”和“至强”处理器两个选项？鉴于苹果 iMac 电脑往往价格都不便宜，增加更多机型将会使产品线更复杂，并且还需要供应商准备更多组件，照此情况来看，苹果非常有可能直接全线标配 Xeon 至强系列处理器。另外，更换内存虽然会稍微拉高的定价，但是突显更“专业”，大多数消费者也愿意为此买单。

**小结：在“专业”路线上前行**

近年来，苹果似乎十分热衷于将自家的产品线全面转向“专业级”定位的产品，以此推动自家产品线的吸引力，尤其是 iPad 产品线。如今处理器更换为英特尔至强系列处理器的 iMac 同样如此，提供 ECC 服务区内存的支持或许只是其中之一属于硬件上的特征而已，软件上的改动尚不清楚。但无论如何，至强处理器将更有助于吸引专业的消费者人群，此类人群对“专业”硬件和软件往往更加敏感。同时，至强芯片对普通消费相比酷睿同样有一定的吸引力。

不久之前，苹果全球营销副总裁菲尔·席勒曾透露过 Mac Pro 的未来规划，在这其中他强调“充分考虑专业用户体验”，声称苹果依然致力于为需求严苛的专业用户提供最高端、性能最佳的 Mac 桌面产品。另外，席勒还透露了全新一代 iMac 一体机将在今年晚些时候正式上市，对此我们拭目以待。

![](.evernote-content/9BBF4D1E-003A-4E2E-8C3D-2C198C3C2E8E/81FDA2BA-25B2-46B4-A74A-E06B1A970FB3.png)

最后来回顾一下传闻中 iMac 的规格。谣言称，新一代 iMac 将搭载英特尔暂时未发布的 Xeon E3-1285 v6 处理器，内存可以选择 16GB、32GB 和 64GB 的 ECC 内存，并提供更快的 NVM Express PCIe 闪存存储，容量高达 2TB。在图形处理器方面，有可能采用全新一代 AMD 北极星架构的显卡。另外，新一代 iMac 还会提供 Thunderbolt 3 接口，或许会配备融入 Touch Bar 和 Touch ID 的独立键盘配件。

话说回来，目前还不清楚 iMac 的传闻是真是假，但如果是真的，你的看法又是什么呢？你会为涨价但配备至强芯片的的 iMac 买单吗？
