# 用一张图片，同时搞定微信和支付宝的付款码

---

有时候看到班长拿着手机逐个问每一个同学：微信还是支付宝支付班费的时候，我就在想能不能把支付宝和微信的收款码放在一个图片里，这样就不需要怕同学没有这个或者没有那个了！曾经把两张图放在一起，但总觉得突兀。后来才发现了这个制作方法，可以轻松地把两个二维码合并，再也不用问别人是微信支付还是支付宝支付了。

制作方法
----

1. 把自己的微信和支付宝的收款码保存到本地；
2. 打开 [制作网页](https://www.hotapp.cn/shouqian "制作网页")；  
   ![](.evernote-content/6023E72E-0821-4A29-B3B6-10334AFF805A/8011E8A0-0640-4660-BAD8-ED7624995B71.png)准备好收款码
3. 按照提示上传自己的微信和支付宝的收款码并合成。  
   ![](.evernote-content/6023E72E-0821-4A29-B3B6-10334AFF805A/B8795D59-072F-4A11-AD01-461FA1E5B434.png)上传收款码
4. 点击合成就制作完自己的收款码了，还可以自定义添加 Logo 等。  
   ![](.evernote-content/6023E72E-0821-4A29-B3B6-10334AFF805A/81A95FF6-E707-4B14-B3C3-FC5E591DF763.jpg)合成收款码

最后想说
----

* 另外，其公众号（zhima128）还支持 QQ 付款码的添加；
* 微信接口未开放权限，无法直接调起微信转账页面，所以需要长按识二维码别进行转账；
* 合并后的二维码无限制扫码次数，永久可用。​
