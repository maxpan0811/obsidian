# Mac 上稳定高效的 FTP 工具，除了 Transmit 还有它：ForkLift 3

---

前一段时间 macOS 的知名文件管理工具 [Transmit](https://panic.com/transmit/) 更新到 5.0 大版本，引发了大家的关注。但是同样推出了大版本更新的，我个人非常喜欢的另一 FTP 工具  [ForkLift 3](http://binarynights.com/forklift/index) 却似乎鲜有讨论。

那不妨就让我来介绍一波。

![](.evernote-content/867925E6-DE3C-40E0-9649-D7072CE2B9BD/5A4491AF-6616-480E-94B0-4469494E065B.png)

此前我很少用专门的文件管理或 FTP 工具，因为那时的需求不复杂，轻量的 [TotalFinder](https://totalfinder.binaryage.com/) 便已够用——TotalFinder 不是一个独立运行的软件，而是可以给原生 Finder 加点「特效」的「插件/扩展」，如下图的双窗口（dual pane）。

![](.evernote-content/867925E6-DE3C-40E0-9649-D7072CE2B9BD/7B308213-80E4-4958-AFB9-4E0CCCB97521.png)

所以如果你的需求也不强，TotalFinder 就够了，不需要继续往下阅读。如果还有兴趣，那就继续了解 ForkLift 3 多一点吧。

外观
--

ForkLift 3 的界面是我觉得最好看的，即使加了很多复杂的功能按钮，也保持着宛若原生的界面，也没有刻意变的五颜六色。这样的好处是，新用户几乎无需改变原有的使用习惯，上手零压力。

![](.evernote-content/867925E6-DE3C-40E0-9649-D7072CE2B9BD/D7FB600A-AE18-4D96-925E-F35A7DF450D8.png)图片来自官网

双窗口让文件管理更高效
-----------

双窗口是第三方文件管理器的必备卖点，ForkLift 的双窗口同时提供了「水平」和「垂直」两种模式，切换迅速还支持快捷键——**事实上 ForkLift 的所有功能均能自定义快捷键**。

![](.evernote-content/867925E6-DE3C-40E0-9649-D7072CE2B9BD/915D0F17-659F-4210-9824-495E7DCBC434.png)顶部红框处按钮可以切换窗口模式

双窗口也让本地和远程之间的文件管理、同步更方便。还可以将站点添加到左侧收藏夹，下次再访问直接点击即可。除了 FTP，ForkLift 也支持 Amazon S3 等多种文件传输协议。

![](.evernote-content/867925E6-DE3C-40E0-9649-D7072CE2B9BD/A68D25E0-BF09-4FB1-9FAA-E702E482ABCC.png)

如果你需要同时管理多个远程服务器，那么软件内置的贴心小工具 ForkLift mini 可不要错过。如下图，它可以将多个服务器添加到菜单栏，让远程文件管理像本地磁盘一样便捷。

![](.evernote-content/867925E6-DE3C-40E0-9649-D7072CE2B9BD/8F8CE4F2-F645-4B6E-A3A6-B15480ED5438.gif)

增强的文件预览
-------

ForkLift 3 的文件预览也是一大卖点。

除了大家都有的图片、视频等多媒体预览，ForkLift 还支持预览 PDF 时直接点击文档内的链接、复制文字等简单操作；还能直接编辑 TXT、HTML/PHP 等多种文本格式，对于艺高人胆大的程序员来说应该挺方便的。

![](.evernote-content/867925E6-DE3C-40E0-9649-D7072CE2B9BD/F7A5F407-0242-475C-B57E-96FA2FE2D8C6.png)

其它高效功能
------

ForkLift 3 内置了一个类似 Alfred/LaunchBar 的功能，可调用软件内的所有命令，也可用于快速执行自动化动作。如下图——批量重命名所有图片，并将文件名中的大写字母变成小写。如果动作较为复杂或是经常用到，还可以保存为 Presets，一键调用。

![](.evernote-content/867925E6-DE3C-40E0-9649-D7072CE2B9BD/8D5CEE52-ADDF-465E-9B12-D83B898D1890.gif)

话说 ForkLift 的收藏夹是真贴心，除了上面讲的，还能将当前打开的标签栏、文件夹路径、界面布局等保存为 WorkSpace（菜单栏：Favorite -> Save WorkSpace），无论何时，一键还原当时的工作场景。

其它的功能就不一一讲了，有类似需求的小伙伴一试便知。

软件目前在 [官网](http://www.binarynights.com/forklift/) 提供全功能试用及购买，价格比 PathFinder 和 Transmit 都便宜。试用之后如果实在喜欢的紧，还可将 ForkLift 设置为默认文件管理器完全取代 Finder，方法可 [参考官方说明](http://binarynights.com/forklift/manual#fileviewer)。不过实测在 Alfred 中打开文件夹，还是会弹出 Finder，开发商说在稍后的更新中会修正。
