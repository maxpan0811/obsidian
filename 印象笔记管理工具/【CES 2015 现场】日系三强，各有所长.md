# 【CES 2015 现场】日系三强，各有所长

---

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/B30CE4E2-1945-4483-B925-44DDC87470DE.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3801.jpg)

在 CES 中，按照展商实力来划分，日系的松下、东芝和夏普当属紧随三星索尼之后的第二集团军，虽然不是像三星索尼那样的全产品线策略，不过场馆也有些看头，基本上各家都有独门绝技。

更多的，这些日系厂商在大众消费市场的影响力不算太强，这可能和整个日系电子产品在世界上的失势有联系，不过在企业市场，或者供应链上，这几家日本企业还是占有比较重要的位置。

### 松下关键词：电池

可能有人会感到困惑，为什么前面说到的是松下，东芝和夏普，但是题图却是 Tesla Model X 呢？因为在松下展区，最引人注目的就是一辆还没有上市的 Tesla Model X，因为现场车不能开，所以也不知道是一比一仿真车，还是真的可以上路的原型车。

不过刚好也可以借松下之地来了解下这款备受期待的车型，首先因为电池组件集中在地盘上，所以动力部分就腾出了许多空间。除了后备箱之外，Model X 也有前备箱，和 Model S 一样。车标和前脸设计也是 Model S 一脉相承，辨识度很高。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/57E1F1AE-A0F6-4A1E-87AC-EDB4ACD69968.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3788.jpg)

中控台依然是一整块触控屏幕，仍是 Tesla 的特色。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/8FDDA920-43F4-4B67-B771-F12C930FCA61.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3791.jpg)

后备箱前有备用座椅，整辆 Model X 可以乘坐 7 人。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/6B88FAFA-E6DA-42ED-9D6D-642550F66826.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3793.jpg)

Model X 采用了鸥翼门设计，这是 Tesla 首次尝试这种设计。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/C531A9C1-5C0D-465F-A642-962D7E3923CE.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3796.jpg)

侧面来看车前灯设计比较犀利，有些鹰眼的感觉。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/30BFFB06-FF02-4996-9E53-FA82BB0E9556.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3807.jpg)

松下和 Tesla 合作紧密的地方就是 Tesla 的关键部分，电池，不管是 Tesla Model S，还是即将上市的 Model X。采用的都是松下提供的 18650 锂电池，虽然许多厂商都在生产这款 18650 电池，包括百十来块的充电宝也在使用这种 18650 电池，不过松下生产的相对能量密度和安全性属于上乘，这也是 Tesla 使用它的原因。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/738EEA9C-84A6-45DD-91EC-99A3D660A866.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3810.jpg)

电池，和相对清洁的电池能源成了松下展区的重点，并且松下也正在和 Tesla 建造“千兆工厂”，试图成为电动车能源供应的领导者，所以，松下在展区展示 Model X，并着重宣传“千兆工厂”也就在情理之中了。

除了把电池用在电动汽车上，电动小摩托，以及电动自行车也是松下正在推广的目标产品，下面这个电动机车想必会受到台湾人民的欢迎，非常适合拍台湾小清新电影。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/1B1F13D4-BFE4-4382-B623-45A5C1CEEC82.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3812.jpg) [![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/1C8561D2-3C0E-4BD0-987F-6E6F31CB7CAE.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3815.jpg)

### “不务正业”的东芝

这次展会上，东芝带来的一些展品非常有日本特色，比如日本目前处于领先地位的机器人。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/1786A823-05C2-4A09-83C1-C1D6108C1F74.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3821.jpg)

这个真人比例的机器人吸引了许多目光，能说会唱，也能交流，具有比较丰富的表情，只是在我看来，因为机器人的皮肤处理不够仿真，而且现场灯光比较杂乱，略显僵硬的表情还是显得有些惊悚。东芝的想法是，将这类机器人用在服务行业和家庭之中，现场只是演示了它（她）的语言歌唱能力，尚不知机动力和灵敏度如何。

这一次参加 CES 的观众中仍然有一些人带着 Google Glass，正好东芝也展示了一款眼镜，和 Google Glass 不同的是，这款眼镜没有在前面突出一块做显示用，而是将投影组件藏在镜片背后，一侧镜片镀有特殊的膜，能够在保证一定透光率的同时，在镜片上显示眼镜投影内容，这是另一种眼镜显示方案，相对自然一些。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/61595FEF-A949-4309-9817-6123B3B475F0.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3825.jpg)

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/CA4E1019-223F-4631-97C1-44BB26744975.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3827.jpg)

现场也体验了一下这个眼镜，简单来讲，用处大概就是通过图像和声音提醒，比如做菜的时候放多少盐，还有多少分钟就可以起锅；旅游的时候会显示景点信息，基本上可以替代导游了。更广阔的用途应该会在工业和企业领域，比如库存管理、安全检查等等方面。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/879D573F-027E-4651-93B1-8D27ED5391ED.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3887.jpg)

这个是东芝的黑科技之一，免脱衣试衣服，上图这位女性穿的是灰色大衣，但是“镜子”中显示的是她穿着橱窗里的羽绒服，因为拍照角度问题，所以显得略微不合身，略调整下后就能达到比较满意的状态，逼真度非常不错，背后的图像识别以及调整算法立功了。

### 夏普：我的屏幕最炫酷

历年来夏普都是以屏幕技术为骄傲，这一次他们在 4K 概念之后，推出了 Beyond 4K 的概念，一般 4K 电视提供了 2400 万个像素，而 夏普的 AQUOS Beyond 4K 超高清达到了 6600 万个子像素，现场人员表示，Beyond 4K 的分辨率接近了 8K（7680×4320），并且结合 Quattron 四原色技术，夏普电视会呈现出更真实鲜活的色彩。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/9358AB6B-64DC-4C98-818B-87BD68DE0F1D.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3831.jpg)

除了 Beyond 4K 电视之外，8K 电视被站夏普展区出现，预计明后年 8K 电视将会成为 CES 的电视主角，毕竟今年的 CES 上，不是 4K 电视都不好意思拿出手了。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/205CEF33-51D4-43E0-8596-EF799C53C1A2.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3833.jpg)

量子点技术是除了 4K、8K、OLED 之外，今年 CES 上电视技术的另一个关键词，夏普也展示了旗下的量子点技术电视。本次展会上，LG、TCL 也展示了自家的量子点电视，相较普通 LCD 电视而言，它可以使电视亮度有效提升 30%-40%，同时背光源色彩转换效率也大幅度提升，色彩更亮丽，色域更广阔。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/36EDD577-21CC-423A-9C25-7D0EF9766347.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3830.jpg)

Beyond 4K、8K、量子点电视这个概念之后，接下来就是夏普独门绝技 IGZO 显示技术，这一次给加了个前缀，称之为 MEMS-IGZO 技术，该技术和高通联合开发，能实现 LCD 显示器 2～3 倍的光学效率，而且色域更广。不过据称这种“薄省亮艳”的屏幕得到明后年才能出货。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/8FA08146-577A-4581-A256-32A496E44EB4.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3836.jpg)

下面夏普继续开启嘲讽模式，前面和普通 LCD 屏幕比完色域之后，又来比功耗了，下方能量槽状的显示屏就是两种屏幕功耗对比，IGZO 完胜。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/6356EE5B-F8C4-4639-89AC-DDAADEC826C2.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3839.jpg)

在智能手机市场，夏普去年最引人注目的作品当初无边框手机 Aquos Crystal，这一次这是他们大力推广的产品，在此就用这款神机作为日系三强的结束之作。

[![](.evernote-content/D949970C-23A7-4208-9C6E-5BB0B0A864C9/8672C5FD-F985-493C-AB12-C17545D335EF.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/DSCF3841.jpg)
