# 她深深的迷恋上了搭配罗技键盘的 iPad Pro

---

罗技的这款智能键盘不会让你的 iPad Pro 外观变得漂亮，不过，它依然可以让你的 iPad Pro 变得更加的优秀。

![](.evernote-content/7BEBA0C2-79D9-404A-AED4-B8DE1B7C02F1/273C4D54-D323-4491-A155-9ABA14E27CF6.jpg)

Serenity Caldwell 在过去 7 个月的时间里一直都坚持将 iPad Pro 作为自己的主力工作设备，在这 7 个月时间里，iPad Pro 与她发生了许多有趣起伏的故事，她也尝试了多个智能键盘，但是最终，罗技 Create 还是成功的征服了她。对了，这款键盘还是首款第三方iPad Pro配件，它可是相当有江湖地位的。

“我很喜欢智能键盘的想法，我感谢苹果做出的尝试与改进，新的 iPad Pro 确实很棒！我非常讨厌直接在屏幕上进行输入方面的工作，这与我的使用习惯有关，但是当我使用智能键盘之后，我感觉我的工作状态进入了一个新的节奏。”

![](.evernote-content/7BEBA0C2-79D9-404A-AED4-B8DE1B7C02F1/B45AFD68-6D43-46B0-A845-D69FB3CBCB3B.jpg)

**爱上打字**

罗技在业内早已经是声名在外了，罗技 Create 采用了苹果的智能连接技术，不需要电池，不需要蓝牙，让你不用担心这款键盘的续航能力，而其全尺寸和便携式的设计让我们没有理由不将其带在身边。Serenity Caldwell 对这款产品赞不绝口。“我可以用最快的速度打字，这一瞬间我以为 iPad Pro 变成了 MacBook Air 一样。”

其背光灯的设计同样令 Serenity Caldwell 欣喜若狂。而其外表设计的好处是手感和防滑性也令她感到非常的满意，即便是 iPad Pro 的机身比较宽，但是装在 Create 键盘中也能获得不错的触感。

“它的按键布局基本与 MacBook 一样，有截屏、音量调节、返回桌面、多任务管理等多个快捷键，不过我也要承认，并不是所有的人都需要用到这些功能，不过它们的存在肯定也为我们的使用提供了非常多的便利。”

![](.evernote-content/7BEBA0C2-79D9-404A-AED4-B8DE1B7C02F1/318A6C25-0905-4D08-B89A-9556395F23E8.jpg)

**就像是房间里的大象**

是的，这款键盘有一个让大家都很郁闷的缺点，就是重量，Serenity Caldwell 也不例外，它的重量和一部iPad Pro差不多，坦率的说，这必须要让你在重量和方便之间做出一个抉择，而 Serenity Caldwell 选择了方便！

**底线**

对于 Serenity Caldwell 来说，iPad Pro 就是她的工作设备，她希望iPad Pro 可以为她做任何的事情，她希望可以在自己的腿上利用 iPad Pro 打字，可以及时编辑图像，所以说，虽然这款键盘并不是十全十美的，但是它完全能符合 Serenity Caldwell 所有的需求。

iPad Pro 固然非常的强大，不过有时候，强大的设备，也需要一些合适的配件来衬托。
