# El Capitan新特性之——全新Spotlight搜索

---

[![](.evernote-content/AF452E8B-10C6-40DB-B88B-9FE58FAC45E4/420C5474-415C-47A8-984F-B9B86A99D0B6.jpg)](http://www.feng.com/apple/tutorial/2015-10-20/El-Capitan----new-Spotlight-search-of-new-features_627682.shtml)

　如果你认为曾经的 Spotlight 比较鸡肋的话，那么在 El Capitan 中你一定会对这个功能刮目相看。在 OS X El Capitan 中，Spotlight 变得更加智能。现在，你可以用自己的话在 Spotlight 中查找文件。Spotlight 还变得更加灵活，你既能调整窗口大小查看更多结果，也能在桌面上将它随意移动。

[![](.evernote-content/AF452E8B-10C6-40DB-B88B-9FE58FAC45E4/420C5474-415C-47A8-984F-B9B86A99D0B6.jpg)](http://resource.feng.com/resource/h060/h59/img201510202205090.png)

用户可以通过屏幕右上角的“放大镜”图标，或者是 control+Space 来呼出 Spotlight 搜索栏。

[![](.evernote-content/AF452E8B-10C6-40DB-B88B-9FE58FAC45E4/98EEBBA5-94EA-46B0-BF7D-1758DEDEFBF4.jpg)](http://resource.feng.com/resource/h060/h59/img201510202205430.png)

**支持自然语言**

支持自然语言或许是 OS X El Capitan Spotlight 获得的最好的更新之一。也就是说用户现在可以直接使用自然语言来提问，以便获得解答。比如你可以这么问，“明天的天气怎么样？”或者是“我上个星期二打开的图片”。Spotlight 就会根据你给出的这些指令来提供相对应的答案。

**移动**

在OS X El Capitan 中用户现在可以在桌面窗口中随意移动或者调整 Spotlight 窗口的大小。

**天气和体育**

在 OS X El Capitan 中苹果已经将天气信息整合到Spotlight之中，所以不管什么时候只要你想了解一个地方的天气，比如北京，你就可以发出非常简单的指令，“中国北京的天气”，你就能够看到北京的天气预报。

体育比赛的分数也是如此。用户只要说出参赛的队名+分数，参赛队伍的分数（或者是赛程）就会出现在 Spotlight 之中。

**网页视频**

网页视频搜索对于 Spotlight 来说也是一个全新的特性，使用起来方便简单，只要你知道视频的名字你就可以进行搜索。用户只要在 Spotlight 中输入视频名称或者其他搜索细节，然后在末尾添加“视频”一词即可得到你想要的结果。

**股票**

OS X 用户当中真正炒股的人有多少我们不知道，不过 Spotlight 还针对股票完善了，这让我们觉得有点不可思议。但是如果你有需要的话，你可以在Spotlight中输入股票代码或者是“苹果股价”、“纳斯达克价格”等词语来获得最新的股市动态，股票数据均由 Yahoo Finance 提供。

**拖拽**

你是否知道在 Spotlight 的搜索结果列表中，用户可以拖动列表中的项目到Dock 图标上，从而在应用中打开该项目。或者是将其从项目列表中拖放到桌面，或在 Finder 窗口中打开，创建副本。（你还可以选择搜索结果，使用Command & C组合键将其复制到文件或文件夹中）。这并非 OS X El Capitan 的新特性，但是知道这项功能并利用其的人或许不多。

**其他**

有些用户可能表示自己并不能使用所有新的Spotlight 功能。如果你遇到这种情况，先打开 Spotlight Settings，关闭应用然后再重新打开即可解决问题。**需要指出的是，有些指令并非在所有语言中都可用。**
