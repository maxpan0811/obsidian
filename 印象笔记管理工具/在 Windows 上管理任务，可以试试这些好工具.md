# 在 Windows 上管理任务，可以试试这些好工具

---

此前，我很少在 Windows 上接触任务管理类的工具，更倾向使用跨平台的笔记类应用，比如有道云笔记和 Google Keep。不过，自从成为少数派签约作者后，这一习惯有了变化，经常遇到需要快速记录选题灵感或者写作思路，并且根据时间制定写作计划、进度跟踪，逐渐感觉到笔记类应用已经不能高效地满足我在任务管理上的需求，所以逐渐开始接触了解任务管理工具。

最早接触任务管理概念是在 Android 移动平台，第一款深入使用的任务管理应用是 [Any.do](https://www.any.do/) ，在我的使用体验中，我认为任务管理工具应该满足以下两个需求：

* 列表式事项管理；
* 到期提醒功能。

带着这两个需求，让我们正式开始寻找 Windows 平台上好用的任务管理工具：

Cortana
-------

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/8BA2ADBE-76C3-4DFF-933D-684BF4B01276.gif)

[Cortana](https://www.microsoft.com/zh-cn/windows/cortana) 作为最具 Windows 10 系统特色的功能之一，Cortana 支持手动或者语音方式添加任务事项，只需键入「提醒我...」或者说「你好小娜，提醒我...」就可以开始使用。Cortana 可以提供基于时间、位置、照片的方式提醒，其中照片提醒功能，用户将照片添加到提醒中，例如孩子最喜爱的游戏，这样 Cortana 就会在下次购物时作出提醒。

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/D7A34586-0311-4BA1-B805-FCD0C1C72183.jpg)

Cortana 支持 Windows、Android、iOS 平台，有着跨平台的优势，支持简单的任务内容、时间提醒功能，语音录入任务备忘事项有不错的识别准确度。

Microsoft To-Do
---------------

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/E3F7F2EA-205A-40BC-AF55-05EA32A1A3DC.gif)

[Microsoft To-Do](https://todo.microsoft.com/) 应用提供了轻量化的任务管理，由著名任务管理工具奇妙清单团队打造，支持 Web、Android、iOS、Windows UWP 平台，几大平台的体验保持了很好的一致性，UWP 应用界面可以视为移动端的放大版本。

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/600F22B8-F717-44B8-AEBF-7B204888C073.jpg)

Microsoft To-Do 的优势是：

* 我的一天。打开应用主界面，率先聚焦的是「我的一天」界面，我的一天是 To-Do 应用默认内置的任务清单，显示了今天待办的任务，帮助用户快速定位今天当天需要办理的任务事项；
* 重复提醒。用户可以对任务设置重复提醒的时间，提供了每 1 天、工作日、每 1 周、每1 年或者自定义重复周期，避免多次手动添加需要重复提醒的任务；
* 简单清爽的界面。我在 Windows 和 Android 端都有使用 Microsoft To-Do 应用，简单流畅的操作方式留下了不错的印象，To-Do 还支持不同清单设置不同的皮肤，增强明显的视觉效果。

Microsoft To-Do 还支持 Outlook 任务中同步事项、清单，有简单的到期提醒、搜索待办事项和笔记内搜索的功能，推荐给深度使用微软系列产品的用户。

扩展阅读：[奇妙清单原班人马打造的轻量任务管理：Microsoft To-Do](https://sspai.com/post/38880)

Todolist
--------

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/35A28E05-4D92-4C38-9755-1DDCE3E53E17.jpg)

[Todolist](https://todoist.com/) 是一款相当「标准」的任务管理工具，支持的平台也最为全面，包括了 Web、Mac、Windows、iOS、Android 以及 Chrome/Safari/Firefox，甚至还可在 Apple Watch、Android Wear 以及 Outlook/Gmail 电邮端使用。

Todolist 有专门为 Windows 10 系统准备了 [UWP 安装版本](https://todoist.com/getWindowsApp)，也有 [exe 版本](http://dateist.todoist.com.s3.amazonaws.com/Todoist_for_Windows_Desktop_2_7_6.exe)，很好地满足了不同 Windows 系统平台的需求。

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/5D3BD3A3-6651-47D0-B0A8-E2DA973E7D33.jpg)

经过这段时间的使用，Todolist 具备以下优势：

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/36DAC69B-6659-4E7C-898E-69B63F5293FC.gif)

* 自然语义添加任务。除了「正儿八经」地输入任务、设置到期时间、提醒时间、添加标签/优先级外，用户还可以直接通过自然语音的输入方式添加任务，Todolist 将会自动帮助你添加到期时间的任务事项；
* 支持项目、标签和过滤器三大形式管理任务内容。项目管理方式类似文件夹，标签代表着任务的属性/定义，而过滤器被视为 Todolist 的一大特色，可按照任务分配给我、优先级别、没有到期的条件快速定位不同的任务；
* 更详尽的任务描述。Todolist 不仅支持文字来描述创建的任务，还可让用户自行添加 PDF、数据表格和照片附件来完善任务细节。

Todolist 提供了多个不同的付费方案，分别有免费基本版、高级个人版（$28.99/ 年）、商务团队版（$28.99/用户/年），其中多形式提醒任务到期、为任务细节添加附件、标签和过滤器都属于高级版提供的附件功能。

Todolist 应用本身具备完善的功能，推荐给追求实用、全面，同时又有多平台同步数据需求的用户。

滴答清单
----

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/B5C3676B-B799-4CEF-B632-3044B1583F68.jpg)

[滴答清单](https://www.dida365.com/)所支持的平台与 Todolist 类似，可谓是跨平台的「巨无霸」。滴答清单更专注于多方式创建任务事项和任务提醒的环节上，其优势主要体现在：

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/F7D16E43-C8ED-491A-868F-3739B5157BAE.jpg)

* 多方式创建任务。滴答清单支持以自然语言，语音输入，Siri 助手创建任务，微信端便捷创建任务的方式。应用还专门提供了专属邮箱，类似`todo+XXX@mail.dida365.com`的电邮地址，或者使用你的注册邮箱发送任务到 [todo@mail.dida365.com](mailto:todo@mail.dida365.com)，即可创建任务到滴答清单的收集箱。
* 智能提醒。滴答清单具有**多闹钟提醒、自定义重复功能和稍后提醒**功能，比如多闹钟提醒，用户可以分别设置提前一天、提前 5 小时、提前 1 小时等不同时间点的闹钟；自定义重复提醒，有助于较少用户频繁在固定时间设置同样的任务，譬如每周六下午有固定的足球友谊赛，即可设置为每周六下午 2 点提醒有足球赛事，而不需要每周六前专门设置重复的任务内容。
* 双向订阅。滴答清单目前支持订阅其他日历的事件到滴答清单，反过来，也支持将滴答清单以 URL 链接方式添加到 Google Calendar, Outlook 等日历应用。

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/B314E1AD-115E-424E-8755-5FF5959EB3B1.jpg)

不仅如此，滴答清单还针对 Windows 平台开发了特色功能，有打开/隐藏主程序、添加任务的快捷键，同时还提供了任务列表和日历视图的桌面小部件，帮助用户快速创建和查看任务。

除了以上特点外，滴答清单还提供了统计、摘要的选项帮助对创建的任务进行一次回顾和复盘的特色功能。不过诸如订阅日历、日历视图、自定义智能清单、高级搜索、清单和任务动态等功能均属于高级付费版（139 元/年），普通用户版本在清单数量、任务数量、提醒数量、子任务数量、共享成员数量、附件数量均有所「缩水」。

Trello
------

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/319A3FFE-BF80-4AD5-A312-4A411DE5870E.jpg)

加入少数派后，首次接触了团队协作工具 [Trello](https://trello.com/)，卡片组成的看板形式成为极具特色的设计风格。Trello 本身具备到期提醒的功能。

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/0179EF59-231A-4FCD-84B6-675F30D058A1.jpg)

对于很多习惯了主动通知提醒，和列表管理任务的用户，Trello 在消息通知和看板管理形式上存在不足的地方。而我运用 Trello 进行任务管理上会专门建立一个 Todo 的任务板（关于写作计划类），任务板下会有六个列表，分别是：

* 历史选题箱：存放除了今天收集的选题，再使用标签分类优先级别；
* 今天收集选题：存放今天当天收集的写作选题；
* 正在开展准备：存放已经开展素材收集的选题，并且在选题描述中添加大纲内容；
* 已动笔：存放已经开始写作的选题，将文章内容的一二级标题以清单形式展示，并标准哪部分已经完成了；
* 推迟完成：存放由于其他因素影响了写作进度需要推迟完成的选题；
* 归档箱：顾名思义就是存放已经完成的选题。

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/1D7C4FD9-808A-42A9-9877-F25349EF5002.jpg)

在使用 Trello 大半年的时间里，深刻感受到其看板管理写作上的优势，使用 Trello 搭建一套很完整的任务管理系统，将有别于其他任务管理工具以列表型进行展示管理，Trello 以看板作为核心特点，结合本身在团推协作、标签、添加到期时间、与第三方软件联动等功能，推荐给围绕 Trello 打造一套自己工作流的用户。

扩展阅读：[用 Trello 搭建任务系统](https://sspai.com/post/39284)

Moodo
-----

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/E26C5FAE-4A7D-4866-A65E-48F44A76EDD9.jpg)

[Moo.do](https://www.moo.do/) 是一个类似于 [Workflowy](https://workflowy.com/) 的无限层级的纯文本任务管理工具，拥有无限层级的树形结构，但它也拥有任务管理、多面板查看、日历等多种增强功能，就像是一个增强版的 Workflowy。[Moo.do](http://Moo.do) 除了类似 Workflowy 的无限层级以外，[Moo.do](http://Moo.do) 的多面板、通知、日历、优先级等功能让它也可用于任务管理，其创新性的地方体现在：

* 无线层级的列表功能。区别于 Workflowy [只能拥有一个文档，Moo.do](http://%E5%8F%AA%E8%83%BD%E6%8B%A5%E6%9C%89%E4%B8%80%E4%B8%AA%E6%96%87%E6%A1%A3%EF%BC%8CMoo.do) 支持创建多个 Document 文件夹，用户新建一个新文档时，将呈现一张白纸，让用户无限层级地创建个人任务事项；
* 多功能用途的 Pane 面板。Pane [是一个平铺展示的窗格，用户无需多次跳转即可查看多个子目录内容。除此之外，Moo.do](http://%E6%98%AF%E4%B8%80%E4%B8%AA%E5%B9%B3%E9%93%BA%E5%B1%95%E7%A4%BA%E7%9A%84%E7%AA%97%E6%A0%BC%EF%BC%8C%E7%94%A8%E6%88%B7%E6%97%A0%E9%9C%80%E5%A4%9A%E6%AC%A1%E8%B7%B3%E8%BD%AC%E5%8D%B3%E5%8F%AF%E6%9F%A5%E7%9C%8B%E5%A4%9A%E4%B8%AA%E5%AD%90%E7%9B%AE%E5%BD%95%E5%86%85%E5%AE%B9%E3%80%82%E9%99%A4%E6%AD%A4%E4%B9%8B%E5%A4%96%EF%BC%8CMoo.do) 还提供了被称为 Calendar Pane、Archived Pane 的多功能用途面板，比如 Calendar Pane 是一个专用于日程安排的 Pane，可将用户在任务事项中的内容按时间显示出来，也可以手动拖动条目到 Calendar Pane，快速为任务条目插入日期；
* 高度整合 Google [服务。Moo.do](http://%E6%9C%8D%E5%8A%A1%E3%80%82Moo.do) 支持 Google Calendar、Gmail、Google Drive、Google Contacts 之间无缝同步数据。

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/4F284BA3-88FF-4F6B-BBB6-62F027AD4C05.gif)

[Moo.do](http://Moo.do) 极具创新性地使用类似 Workflowy 的功能，围绕无限层级列表，多功能 Pane、`@` 和 `#` 结合使用，帮助用户构建一个更加详尽的任务安排系统。推荐给不需要华丽界面，更愿意自己「把握」任务安排的用户。

Remember The Milk
-----------------

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/E5BCBCF5-323F-4AA0-86EA-BE37864E4603.jpg)

[Remember The Milk](https://www.rememberthemilk.com/) 作为有着十年历史的任务管理工具，支持 macOS、Windows、Linux、iOS、Android、BB10、Apple Watch、Outlook，以及包括 Chrome、Firefox、Safari、Edge 浏览器平台，其优点包括了强大的快捷键、高效的日期识别、支持子任务，用户可以通过手机，邮件，IM 等很多方式添加任务，后期则支持电子邮件、IM 提醒、谷歌日历、手机软件、手机短信等方式提醒用户。

![](.evernote-content/22A225C2-E3EE-4A09-878A-43FEB60083F1/023BFBD3-6CE5-45DB-813D-1CF713F1D623.jpg)

Remember The Milk 的优势：

* 高级搜索。支持使用标识符进行搜索，搜索你的任务和笔记，并保存你喜欢的搜索内容作为智能列表；
* 强大的任务提醒。支持通过电子邮件、IM 软件、短信、Twitter、手机或者系统桌面通知的方式提醒任务；
* 电邮通知每日任务摘要。用户可以将今天到期或者已经开始的任务、明天/未来 7 天开展的任务等定制的任务设置成每天发送邮件的方式。
