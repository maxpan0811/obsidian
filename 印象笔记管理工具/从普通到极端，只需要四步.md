奥兰多夜店枪击案在成为各大媒体的头条的同时，两种截然不同的声音在社交媒体上打得热火朝天。一方主张爱能够战胜仇恨，另一方驳斥这种说法纯属给恐怖分子洗白，是没用的鸡汤。

案件发生之后，民主党的领先总统候选人希拉里在演讲中呼吁美国民众向穆斯林群体伸出双手、团结起来共同战胜恐怖主义。而共和党的候选人川普在演讲中把矛头直接对准美国的穆斯林群体，指责他们没有揭露那些他们知道是恐怖分子的穆斯林，从而纵容了袭击和惨剧。

![](.evernote-content/99F78046-B7D1-47C2-B68A-54EC5107C0D3/D09FBCD7-1C61-40A7-84CA-D3D05576107F.jpg)

两位候选人截然不同的观点代表了面临威胁的美国人必须做出的**两种选择**：

通过包容、团结和爱来坚守美国的价值观，还是通过排斥外来移民、散播怀疑和恐惧来保障安全？

奥兰多枪击案件值得我们深思的还有这样一个事实：枪击案的凶手Omar Siddiqui Mateen是土生土长的美国公民，而不是成长于受恐怖组织控制地区的外来移民。根据警方揭露，他在俱乐部里射杀了多人之后，打电话给当地警察局并告诉警方他效忠于伊斯兰国ISIS。虽然美国官方尚未证实凶手的宣称，这场惨剧指向了一个不能逃避的问题：**为什么在美国长大、生活在西方文明社会的Mateen会受到极端主义、甚至极端伊斯兰主义的影响，变成屠杀自己同胞的杀手？**

根据*New American Foundation*的数据，自911事件之后在美国发生了28起严重的恐怖主义袭击案件，而**其中80%都是由美国公民制造**。在2001-2016年之间因为恐怖主义被逮捕的497人中，**64%**都是出生在美国的公民；**在323个和极端伊斯兰主义有关的恐怖分子中，接近一半是出生在美国的公民**。

![](.evernote-content/99F78046-B7D1-47C2-B68A-54EC5107C0D3/23275CC6-1257-42FF-9085-A7813C6DE7D7.png)

这样的研究告诉我们，即使是西方国家的普通公民也容易受到极端主义和恐怖组织的影响。不幸的是，当今世界上最残暴极端的恐怖组织ISIS极其擅长利用网络和社交媒体从世界各地招募普通人。根据统计，一共有**来自超过100个国家的2万名志愿者离开了自己的家乡，成为了ISIS的战士**。在这些人中，不乏青少年和梦想成为ISIS战士妻子的年轻女孩。

![](.evernote-content/99F78046-B7D1-47C2-B68A-54EC5107C0D3/8563C488-E82A-4834-A950-D60F136AEF32.jpg)

图：2014年，15岁的Sabina Selimovic和17岁的 Samra Kesinovic从奥地利离家出走，成为了ISIS战士的妻子。

**那么，ISIS是如何使普通人演变为手持武器的杀手的？**

### 分步骤洗脑：ISIS的招募

**第一阶段：**

```
“如果吃下这一粒药丸你便能看清世界的本来面目，你愿意吗？” ——将未成年人和他／她们的社会环境相隔离
```

招募者最开始是向青少年目标人群发放**关于这个世界的“真相”**：疏于管理的食品安全、药品滥用、被曲解历史和政治事件，并且**散布谣言**称这个世界已经被“有钱的大佬”买下来了（其中包括Zionist conspiracy, the Illuminati, the Freemasons)。 而在阴谋论得以生存的网络世界，年轻的被招募者很容易便能找到支撑这些理论的“证据”。

ISIS的招募网站极其狡猾地利用类似于《黑客帝国》的电影来支持自己的论点。ISIS利用黑客帝国中Neo吃下药丸发现世界已被Matrix控制这个桥段来映射现实世界，暗示这些招募目标人群“**如果你是Neo，你是愿意吃下药丸觉醒，然后拯救世界，还是继续沉睡，一无所知？**”

![](.evernote-content/99F78046-B7D1-47C2-B68A-54EC5107C0D3/138AD3D8-1167-4886-9F54-3A83AC5F3088.jpg)

图：“吞下蓝色药丸，继续相信你原本的世界；吞下红色药丸，我将带你去见识真相。”

在这个阶段，被招募者会突然中断与家人，朋友的联系。不再出现在任何社交场合，不去上班，不见朋友，甚至会自行和家人隔离。这也是ISIS所鼓励的“**通往觉醒**”的步骤之一：身边的人不是被蒙在鼓里的梦中人就是祸害世界的阴谋家，与他们交往将会让自己软化，甚至被同化成服从于系统的“沉睡者” （sleeper）。

**第二阶段：**

```
“只有‘真正’的伊斯兰主义才能唤醒你” “你是注定的拯救者！” ——加强极端思想灌输
```

被隔离的年轻人很快会被组织吸收。招募者会向这些年轻人灌输“你是被选择的那个人”、“只有真正的伊斯兰主义（极端主义）才能让你获得救赎”等观点。

这些已经被隔离出来的年轻人为了寻求群体感和社交支持（social support）会很快被ISIS吸收，个人身份被抹去，个体记忆被删除。逐渐身边人们的任何举止都会让他们草木皆兵（比如LGBT群体在公共场所亲吻会被视为违反伊斯兰教义，对于这些极端者来说，LGBT的存在是世界末日的征兆之一）。

![](.evernote-content/99F78046-B7D1-47C2-B68A-54EC5107C0D3/5610B7A2-765C-47EE-8335-8ABAA65983EE.jpg)

**第三阶段：**

```
“不信仰‘真正’伊斯兰的人都是被传染的，都该死。” ——加强被招募者和极端伊斯兰组织的连接
```

第三阶段可以被看成是第二阶段的进阶版，被招募者此时已经完全相信自己在拯救世界，任何不相信真正“伊斯兰”的人都是被蒙在鼓里的邪恶化身。这也是迈向第四阶段的过渡期。

**第四阶段：**

```
“我不是人，他们也不是人，我是战士，他们是魔鬼。” ——去人性化
```

只有完成了这个阶段的被招募者才真正的成为了ISIS制造恐怖的武器。这些所谓的ISIS战士在完成了四个阶段的洗脑过程后已经不再相信非信仰者是有血有肉的人。因此对于这些没有人性的“东西”施暴不仅不是犯罪，而是净化世界的过程，是觉醒的必经之路。去人性化由不把对立者当作人看开始，到最终不把自己当作血肉之躯为止，导致了自杀式袭击的频频发生。ISIS的追随者中每一个人都把自杀及杀人当作是在参与圣战的使命。

![](.evernote-content/99F78046-B7D1-47C2-B68A-54EC5107C0D3/43B05C8A-08DD-4B2E-8ADF-8A57CA94F0F6.jpg)

### 针对弱势群体的欺骗：ISIS的谎言

**A. “被侵害”组**

针对被家暴、性侵的女性，ISIS会向他们承诺，追随ISIS之后的他们能够嫁给一位英雄，从此可以不用接触其他男人，终生受到保护 。

**B. “内疚”组**

主要针对有创伤记忆的人群。利用他们心中的内疚将她们引诱到ISIS，称效忠ISIS可以让她们得到内心的安宁。

**C. “人道主义”组**

例：发布叙利亚儿童死难照片，吸引人道主义工作者前往，后发现其实是ISIS组织。

### ISIS的招募手段为什么有效？

**我们 VS. 他们／群体边缘化：**

![](.evernote-content/99F78046-B7D1-47C2-B68A-54EC5107C0D3/B51CC7F5-6260-419C-B0F1-480EF348517C.jpg)  
图：我们 VS. 他们

这些看似一般人不会被洗脑的步骤，其实并没有我们想像得那么简单。ISIS能够在世界各地像魔鬼之花一样生根发芽，除了他们传销一样的招募技巧，外部的社会环境也起到了推波助澜的作用。

**恐怖极端主义并非在真空状态下平地起波澜，而是吸食着各个群体内部的暗流**。作为在社会、政治和经济上被边缘化了的弱势群体，如果足够多的穆斯林被美国主流社会视为具有嫌疑的对象，并且被待以排斥和敌意，那么久而久之那些曾经不认同极端伊斯兰的穆斯林就更有可能被发展成为潜在的极端分子。

社会心理学的两大经典实验（Zimbardo的斯坦福大学监狱实验，以及Milgram的电击服从实验）都曾经证明过群体边缘化（group polarization)的力量可以迅速将一个理智的普通人催化成手拿武器的暴徒。

具体来说，普通人在群体中的行为往往不象我们想像中的那样受理性控制。斯坦福大学监狱实验中，两组受试者被人为地分为囚徒与狱警，两组原本是普通人的受试者在长期的封闭环境中逐渐演变成为恶魔：狱警被权力冲昏头脑，囚徒被困境拿走了理智。两组受试者最终互相以暴力相待，甚至导致实验被迫提前结束。

![](.evernote-content/99F78046-B7D1-47C2-B68A-54EC5107C0D3/57AC3B22-1BF9-457D-898A-EC6810C1275C.jpg)  
图：电影《斯坦福监狱实验》再现了这个经典的心理学实验

Reicher 和 Haslam在一个斯坦福实验的演化版中采访了一个扮演囚徒的受试者。该名受试者曾经是模范囚徒，并协助狱警管理其他囚徒。在受到其他囚徒的嘲笑时，这名受试者因为立志要成为一名狱警而丝毫不受影响。然而在“选拔”的时候，他的努力被忽视，而另一名他认为能力不足的人被提名。这名受试者的行为转变非常戏剧化。他几乎立刻开始认同曾经被他所排斥的囚徒们，开始了一系列的暴力行为。

**这个实验的结果或许映射了当下的现实：**

> 一个不被认可的温和穆斯林，在受到“同伴”（极端穆斯林）嘲笑的情况下，并不会受到影响；而当主流社会采用排挤甚至采仇恨暴力的方式对待他们的时候，他们的认知和行为将会受到极大的影响，反而开始认可他们曾经排斥的极端穆斯林成员。

### 反思：如何抵抗恐怖主义？

**被社会接纳**对于一个人和一个少数群体来说非常重要。**被社会边缘化，甚至被隔离、被仇恨，很容易让人丧失理智**。人的意志和环境都不能单独决定一个人的行为，其行为永远都是人和环境共同作用的后果。也就是说，在防范ISIS的洗脑和招募之外，也要十分注意我们的社会环境释放的信息是什么。

而现在美国社会的环境不容乐观，**高呼“驱逐所有穆斯林移民”的人，从政客到键盘俠，实际是典型的反恐猪队友，恐怖主义神助攻**。他们的狭隘不仅没有提供有效的解决问题的方式，反而是在推波助澜。川普式的排外和敌意恰恰正中了极端穆斯林和恐怖分子的下怀，他宣扬的政策和价值观将一步步把温和的穆斯林边缘化，甚至可能迫使他们成为恐怖主义的武器。

![](.evernote-content/99F78046-B7D1-47C2-B68A-54EC5107C0D3/BEB41A21-3FFE-4E63-9E06-4D00EA4EF070.jpg)  
图：Credit to Rupak De Chowdhuri

爱能战胜仇恨，并不是说我们要爱那些杀人的魔鬼，或给恐怖分子洗白，而是因为在面对恐惧和威胁时，用包容、信任来舒缓社会不同群体之间的矛盾，让每一个群体都能感觉到被接纳和被信任，才能真正抵抗恐怖主义的侵蚀。而盲目和歧视性地排斥某个特定群体非但不能解决问题，反而会让情况恶化。

人类的历史以及社会科学的研究和证据都指明了这一点，选择爱和宽容不是圣母白莲花，也不是虚伪的政治正确，而是战胜恐怖主义和极端主义的明智选择。
