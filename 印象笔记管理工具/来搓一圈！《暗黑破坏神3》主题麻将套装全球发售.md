# 来搓一圈！《暗黑破坏神3》主题麻将套装全球发售

---

![](.evernote-content/3735BC2D-7063-4D29-B64F-CE411A163D16/EAE20BE1-4170-4C98-A2F0-7CAAE5677C18.jpg)

随着中国市场在暴雪运营中占据越来越重要的地位，暴雪曾为《魔兽世界：熊猫人之谜》推出麻将周边。现在，他们又带来了《暗黑破坏神3（Diablo III）》主题麻将套装，其在中国区的售价为1588元，而在全球发售的价格为300美元，大概面向华人暴雪迷用户，精美华丽拥有非常标准的144枚麻将牌（包括花牌）以及暗黑风格的筹码、桌布和骰子。

这款麻将套装给人的印象便是充满了暗黑的元素，每一枚筹码、每一把牌尺，甚至麻将盒的锁扣，无一不采用了游戏元素的精心设计。黑灵魂石方向钟，庇护之地世界地图桌布，营造充满悬念与惊喜的游戏氛围。

无论的黑灵魂石还是桌布上的地图，都为我们营造出了游戏中的氛围。如果能够在闲暇之余凑上四个人一起来一桌暗黑麻将，也是游戏粉丝们的一大乐事。

《暗黑破坏神3》主题麻将套装包括一个麻将盒、144个麻将牌、2个空白麻将牌，共计800点的筹码，暗黑世界主题桌布，4个牌尺，6个骰子和方向钟。
