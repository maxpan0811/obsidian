Re:回复： 20190225会议纪要

在11月的18年飞鹰班培训和12月底的福冈管理会上，我曾跟大家分享过我对“旅游运营商”的理解：

•旅游运营商：以市场分析、产品研发为核心，直接为终端游客提供服务，并向上掌控交通和目的地等资源、向下协同分销商

的，有较高市场份额、经营规模的在旅游生态圈里具有重要地位的企业。

应该说，我们现在和“旅游运营商”的定位还有不小差距，更不用说“最优秀的旅游运营商”（严苛地说，我们目前更像一个承担库存的代客加工厂，在产业链里没有身份感、没有价值主张，对分销商逆来顺受、委曲求全）。但同时，我认为在行业里我们最有条件成为一个优秀的“出境游运营商”，也应有信心率先成为优秀的出境游运营商。为此，我们除了产品、服务和ERP运营系统的不断优化与变革，还要有志于在整个出境游业务链中发挥主导性的作用，对上、下游都起到引导、牵动的作用，体现和输出我们在生态圈里重要的、不可或缺的价值。其中，特别重要的就是要形成我们对“渠道”（分销代理商）的影响力、网络力、牵动力；要形成这三力，就绕不开“品牌”的打造——品牌不仅代表产品力，还代表着“信用”（大大降低双方的“交易成本”），代表着双方双赢共进的持续合作。这是一个要花几年时间的系统工程（国内的旅游服务业界尚无先例，但在欧美日和国内其他行业已经验证），初始会有各种质疑、挑战，但一定要想通、并开始去规划。

我的发问，核心不在于两件事具体怎么安排，而是要表达我对不少同事对公司定位不够清晰、对渠道无原则妥协的担忧。我们要明确几个概念，一是我们是做B2B的，我们要按规矩做，不仅不能抢代理商的生意，还要真金白银地支持诚意合作的代理商（比如导游维护的复购仍然算到代理商头上）；二是我们要主导产品和服务的理念——游客满意至上，代理商要认可并与我们共同维护“翔龙”的品牌；三是公平交易，有规矩。一个不认可我们的品牌和理念、不尊重我们（包括我们的任何一个员工）、不讲规矩的代理商，你怎么能想象他能给我们带来价值和收益。

具体到这两件事，何勇的意见我赞同，薛薛、之明说得也很中肯；潘渤关于建立品牌意识的呼吁我很赞赏。同时，我相信潘渤有智慧和能力把这两件事做好，我们要给他（以及其他单元老总）尝试的空间。

> > > > > > > > > 郭总，我来谈一下我的看法：
> > > > > > > > >
> > > > > > > > > 1、关于领队收客的问题：这件事海超是有些误解，所以我们周一会后又沟通了一下。我的想法是把我们目前系统内无人维护的客户（目前系统内2/3客户一年无产量）给到领队们去维护，销售可能因为这些客户产量小不愿花时间去维护，但领队因为不领工资没有成本，所以维护下来即使每个客户每年只收10个人，对公司也是有边际收益的。而领队们的激励来自于收到一定客人数量后（比如100人/年），明年可以不交选团费就参与选团。这一点跟海超是达成共识的，毕竟现有的客户资源如果销售没时间维护，让别人去维护也是合理的，也是对销售的一种合理竞争。关于收直客这一点，目前阶段对外宣称肯定是这样，但原则上不引起客户投诉的话就先不管了，这块也很难查实。
> > > > > > > > >
> > > > > > > > > 2、领队和客人的群肯定要建立，否则行程中无法沟通。山东有些包团的客户如果有意见，要求回程后解散群，可以一事一议。
> > > > > > > > >
> > > > > > > > > 再多说一些，我一直认为我们必须建立我们的产品品牌，销售咨询和具体交易（物流、钱流）可以由代理商去完成，但与客人的品牌互动（信息流）必须我们来主导。类似其他行业的厂商，虽然通过渠道销售，但品牌推广一定是厂家主导，目前大品牌里走直销渠道的也只有苹果比较成功。而没有产品品牌就不会有品牌忠诚度和品牌溢价，我们就价值不大。
> > > > > > > > >
> > > > > > > > > 我认为目前的行程助手要好好利用起来，目前通过这2、3年的使用，应该已经有十几万客人的关注和使用了，这一块我们要坚决用起来，而不能受代理商意见的左右。通过行程助手，我们可以想办法给客人提供：通知按指纹时间和地点、推送送签进度信息、出行须知提示信息、奢侈品购买信息查询和留货、行程中意见反馈（上传图片）、自动导出行程游记等等服务功能，提高客人的满意度和粘度。这些服务功能可以理解为我们产品服务的自然延伸，既提高了客人的品牌认知，又不容易引起客户反弹，我们可以有个说法给到客户。
