# 终于让你没有理由不爱它：Ulysses for iOS [送码]

---

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/7D14F98D-BB49-48ED-AACE-6122F46BF8F6.jpg)

在诸多的 Markdown 应用中，[Byword](http://sspai.com/search/?q=byword)、[iA Writer](http://sspai.com/search?q=ia+writer) 与 [Ulysses](http://sspai.com/search?q=ulysses) 这三款软件，一直以来都广受赞誉；尤其是 Ulysses，无论是媒体推荐还是用户好评，都让它在近两年逐渐成为了 Mac 平台上 Markdown 写作应用的首选。

作为一款由德国人开发的应用，精工细造的同时也不免显得有些谨慎。在 Byword、iA Writer 纷纷发布了全 iOS 平台版许久之后，Ulysses 才慎重的发布了 iPad 版本。

虽然 iPhone 上的写作需求并没有那么大，但是对于希望可以全平台写作的人来说，不支持 iPhone 成为了他们选择 Ulysses 的一大顾虑。

而今天，这个顾虑被彻底消除了。在经历了长达 7 个月的准备，3 个月的 Beta 版测试，Ulysses for iPad 终于更新为通用版 Ulysses for iOS。至此 Ulysses 终于在 Apple 的全平台上线，对于不少人来说，他们也终于能将写作完全迁移到 Ulysses 上了。

一脉相承的优雅界面
---------

初次打开 Ulysses for iOS，它会自动连接 iCloud 配置你的 Ulysses 资料库，各地不同的联网情况会让进程各有快慢，如果你在 Ulysses 的 iCloud 文档库中存有不少文章的话，那么等待时间还会进一步加长。[1](http://sspai.com/33171#fn1)

在等待的时间中，Ulysses 也为我们贴心的准备了类似 Mac 版的介绍文档，详细展示了 Ulysses 的功能与设计。

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/1C3B9A1B-1012-4934-AAC0-A7A4F4D35E42.jpg)

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/A5C98400-C5AB-46C4-96B0-5F2792892588.jpg)

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/BF536C30-D183-4843-8A71-E2A8EC16D0C8.jpg)

应用的层级设计依旧延用了电脑版的逻辑，除了没有本地文库以外，其它均与电脑版一致，分为 iCloud 文库与外部文件文库，默认具有「全部、最近7天、喜爱、废纸篓、收件箱」的层级结构，同时你还可以在文档库中随意新建文档组与「筛选」[2](http://sspai.com/33171#fn2)

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/5F0B0C27-9CB7-4F6B-B7E2-64202231DA48.jpg)

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/70CC4B6D-BC1C-445E-9A50-9DDC1217F734.jpg)

在文章列表界面与编辑界面上，iPhone 屏幕虽远不及 iPad 与 Mac 的大小，但由于排版与字体安排的精细设计，可以说是完全复刻了 Mac 版的优雅与简约。

同时，Ulysses for iOS 和 Mac 版一样支持自定义界面，无论是字体还是文字布局，都是可以更改的。

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/47D4CCAE-8B31-4FC1-8ED8-E7569E2C9623.jpg)

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/3A9F6C55-A914-4BAF-8D87-7E15A7024B60.jpg)

Ulysses 一直久负盛名的写作主题在 iOS 版上依然是可以自由选择的，通过 iCloud 你可以把在 Mac 上安装的所有主题都同步到手机上，并且你还可以通过访问 Ulysses 官网直接在 iOS 版上添加写作主题。

重新设计的工具键盘
---------

Ulysses for iOS 一大令人惊艳之处在于其全新设计的键盘工具栏。

对大部分中文输入用户来说，输入 Markdown 的英文语法符号是一件非常麻烦的事情，需要多步反复跳转，这让手机本就不快的输入速度变得更是缓慢，而大多数 Markdown 应用的输入工具条都只有很少的几个常用符号快捷键，并不足够好用。

同时，对很多人来说 Markdown 的某些标记不是完全能够记住的，而对于 Ulysses 这样的对 Markdown 语法小有修改的应用来说更是如此。[3](http://sspai.com/33171#fn3)

开发者同样也考虑到了这一点，应用将所有常用的 Markdown 标签进行了整理，分为了三类，只需要点击相应的标签类型就能快速找到需要的 Markdown 标记，免去了记忆字符与手动重复输入的麻烦。

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/489426C2-E3DE-49EB-AC30-63CAE7B8F092.jpg)

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/1D66F996-D527-4503-AD2D-3B9580F23F8F.jpg)

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/1A226520-C3FC-4512-8035-4A2B5CD27AF8.jpg)

其中，图片标记的设计最有特点。当你选择 [img] 或手动输入相同字符后，你就会进入插入图片的界面，你既可以选择网络图片填写网址，亦可以直接从照片库中选择一张图片置入。

置入之后的图片并不会直接显示在文档中，而是显示一个小小的标记，只有当你点击后才会在弹出的窗口中看到图片的样子，对显示面积紧张的手机来说可谓十分有用。

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/158F9646-5610-4007-83B3-C1F237D5EED5.jpg)

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/EE3D0748-A297-4623-945C-3BD4A8B1B80B.jpg)

除了格式符号工具之外，Ulysses for iOS 的文章内搜索功能也设计的很好，无论是按钮的位置，还是搜索的速度，都让人感到便捷，并且其对中文的搜索的优化做的也十分到位，这对于显示内容有限的 iPhone 来说无疑是能够极大的提高效率的。

另外，Ulysses 还支持了 TextExpander 的片段展开功能，能大大提高写作效率。

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/8E36AAF9-0700-4847-93D1-C7014A58B406.jpg)

同样强大的导出模版
---------

对于文章的导出，iOS 版也保留了和 Mac 版一样的多种文本格式和文件格式的操作。

在手机上也能直接导出 HTML、ePub、PDF、DOCX 格式，并通过诸如微信、Telegram 等方式分享给你的朋友。

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/9233FBBF-61AD-4A70-8017-31D498232EE2.jpg)

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/49DF8E7B-913F-4BE1-9816-743B46B42198.jpg)

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/9C3BFC50-C541-4483-832C-2FCF9F96D561.jpg)

同时，在每种格式大类之下，还有若干样式可供选择。Mac 版的样式模版会自动同步到 iOS 端上来，同时你也可以直接通过访问 Ulysses 的官网下载你所需的样式模版。

除此以外，Ulysses 还支持直接将文章发到 Medium 平台上，而你的帐号授权也可以通过 iCloud 同步过来。

![](.evernote-content/E413E19B-FCF6-44C3-BC9F-60BA53A4206A/C6E6A326-6AFF-415C-8433-2C96CC88E647.jpg)

顺畅无比的同步写作
---------

Ulysses for iOS 内置的是 iCloud 文档库，利用 iCloud 在大多数情况下同步自然是无往不利。可以想象这样的一天——早晨，起床在 iPad 上写出文章的提纲，出门在地铁上或在工作的间隙，打开手机都能立刻就能继续对大纲进行填充，晚上回家后打开 Mac 为文章做最后的润色修改，全部过程一气呵成。

Ulysses for iOS 还支持了 Handoff 功能，只要你的 iPhone 和 Mac 上都安装有 Ulysses 并打开蓝牙，你放下手机立刻就能在 Mac 上继续自己的写作。

Ulysses 在 iCloud 同步问题上作出过不少的努力，甚至在 beta 测试期为 iOS 版专门设计了 Mac beta 版，测试对 iCloud 同步的优化。长期使用下来，可以说是非常的顺滑，同步总是非常的及时，只遇到过一次同步缓慢的问题。

不过在这里我还是想要提醒一句，iCloud 服务在中国始终是有一些同步问题，不一定能够一直保持非常好的同步效果，首先你要有一个足够稳定的网络，才能保证 Ulysses 在各平台之间使用的顺畅。[4](http://sspai.com/33171#fn4)

不足
--

作为一款经过开发组精心打磨的应用，Ulysses for iOS 本身已经没有什么设计上的硬伤，可以说，它满足了大部分 Ulysses 重度用户对它的幻想，甚至在不少地方还大大超出了期待。

倘若真的要说它的不足的话，则主要出现在一些细节的问题上 ——

* 启动速度偏慢，往往需要 2 秒左右才能完成启动。
* iCloud 同步没有同步按钮，需要等待 iCloud 自动完成同步，过程往往颇有坎坷。
* 仍不支持 Dropbox 上的文稿作为外部文稿进行编辑同步。

相信 Ulysses 的开发商会逐步解决这些问题。

总结
--

Ulysses for iOS 的这次大更新，可谓是打通了 Ulysses 的任督二脉，写作者的写作体验在 iPhone、iPad、Mac 三者间从此不再割裂，文档的无缝同步，编辑界面、Markdown 符号与导出模版均保持一致。同时对于图片视频这类现今更多存于手机上的文件，也能不用通过手机传至电脑后才能添加了，手机端即可以自由的添加。

无论是从设计，还是使用体验上来说，Ulysses for iOS 都可以说是复刻了其 Mac 版的完整功能与特色，简洁而强大，让人惊艳。

应用这次更新还支持了 iPad Pro ，只是非常遗憾我手上并未有 iPad Pro 来观察其为 iPad Pro 特别设计的布局。但我相信在诸多 Markdown 软件中率先支持了 iPad Pro 的 Ulysses for iOS 一定会成为 iPad Pro 上的首批高实用性生产力应用，而分屏模式必将与其相得益彰。

这款刚刚在 2015 年被 Apple 评为「2015 年度精选应用」的应用，一直都在不断的自我进化中，从未停歇，而今除了那个高昂的售价之外，我们还有什么理由不爱它？

对于这款应用的售价，开发商表示，应用新上架时会和 iPad 版的价格一致，售价 128 元，但是这并不是应用的最终定价，而是「首发优惠价」，之后应用的价格将会上调至 163 元。对于此前已经购买过 iPad 版的用户，则可以直接升级到通用版。
