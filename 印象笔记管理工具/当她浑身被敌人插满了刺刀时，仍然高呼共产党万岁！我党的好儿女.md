# 当她浑身被敌人插满了刺刀时，仍然高呼共产党万岁！我党的好儿女

---

当她浑身被敌人插满了刺刀时，仍然高呼共产党万岁！我党的好儿女
==============================

历史悠久传

要政同志，原名要秀森，生于1920年的晋县鲁家庄。是一名优秀的共产党员，是人民的好儿女。她为了党的事业，为了民族解放，献出了宝贵的生命，她的英名，刻在晋县革命烈士纪念碑上，也永远镌刻在全中国人民的心中。

![](.evernote-content/AA3ACA58-9710-4BC4-8B1C-6F95251800EB/8CC1DF9F-4947-42DF-91D7-4C8B289A17DB.jpg)

1931年，日本侵略者占领了东北全境，随之逐步向河北进犯，老蒋的不抵抗政策，拱手把祖国的大好河山让给猖狂的日本鬼子。在国难当头之际，民族存亡的生死关头，中国共产党领导全国各族人民举起反抗的义旗，领导全国人民和各界人士奋起抗日。

年仅11岁的要政经常让她的父亲要志国给她讲日本帝国主义侵略中国的罪行；讲共产党人英勇抗日的故事；讲冯玉祥将军组成察哈尔民众抗日救国的故事。父亲的每一遍讲述都深深地印在了要政的心灵里，幼小的要政从此埋下了对日本侵略者的刻苦仇恨。

从此，无论她学纺织、学织布、挖野菜、拾柴火，自学文化，练习武术，都是怀着一颗报国志，发誓：“一定要报阶级仇，民族恨！”自幼勤奋的要政，也练就了一身的好体质。

![](.evernote-content/AA3ACA58-9710-4BC4-8B1C-6F95251800EB/E76E0D1F-193C-48AF-9155-424FF87064BA.jpg)

“七七”事变后，抗日的星星之火，燎㷧于全国各地，鲁庄也于1938年组织起了各种抗日群众团体。满怀阶级仇恨的要政，积极地投入到了抗日队伍中，也终于实现了她亲临抗日活动的前线。由于她对抗日工作的热情和努力，做出了许多有成绩的工作。组织上对她积极地表现，不久就让她到县妇救会举办的训练班学习，她在学习期间加入了中国共产党。学习结业后，她被组织安排到二区开辟妇救会工作，不久便担任了二区妇救会主任。

已经十八岁的要政，自从担任妇救会主任一职后，积极带领二区妇救会的干部们，风餐露宿，跋山涉水，走乡串村，深入千家万户，积极发动妇女，宣传抗日活动。她还组织了妇女识字班，组织妇女做军鞋、送军粮、搞募捐，支援前线，站岗放哨。使二区的妇女工作开展得轰轰烈烈。

要政和长工出身的我党的干部张英同志于1941年结婚了。婚后夫妻二人互为帮手，齐心协力地做抗日工作，使她的工作越来越出色，不仅受到上级组织的多次表扬，还受到了当地妇女们的爱戴，成为了党在二区的抗日代表。

![](.evernote-content/AA3ACA58-9710-4BC4-8B1C-6F95251800EB/15C992AD-B8AD-4D6F-867B-4F99E6BE6CFF.jpg)

1942年春，由于敌人越来越频繁地出来扫荡，便根据的环境越来越恶化。要政为了鼓舞当地民众的抗日热情和积极性，她不顾危险地担任起了全区的抗日宣传鼓动工作，她怀着身孕，和男同志一样，白天隐藏在青纱帐里，晚上绕过林立的日军岗楼，继续走村串户，坚持带领群众进行抗日斗争。

一天的天黑以后，要政和区长张卓民依旧带上随身的小包裹，踏上了熟悉的乡间夜路，她们向宋家庄走去。她们到了宋庄后，马上召集了群众参加抗日誓师大会，会场上站满了群众。要政满怀激情大步走向主席台，并高声地向与会群众讲道：“父老乡亲们，我们的祖国是地大物博的国家，使得日本侵略者早就垂涎三尺，他们企图想吞并我们的大好山河，我们决不能当亡国奴，要用鲜血和生命保卫我们的国土！”

会场下群情激愤，响起“打倒日本帝国主义！”“消灭一切反动派！”“还我大好河山！”的口号。随着要政那宏亮、铿镪有力的演讲，场下掌声雷动。只见儿童团代表，青年学生代表，青救会代表，以及妇救会代表，纷纷上台发言，深表抗日的决心。

![](.evernote-content/AA3ACA58-9710-4BC4-8B1C-6F95251800EB/928CC2D7-47FE-4F18-82FA-DC53348CD757.jpg)

随着日军的不断加强围剿，使我抗日组织大部分遭到破坏，许多的抗日干部被日军杀害。我党为了保守有生的革命力量，及时调整部署，决定化整为零，转入隐蔽的地下开展工作。要政隐蔽在一位干娘的家里，随时接受党的指示秘密开展对敌斗争。

一天早饭后，她和干娘装扮成走亲戚的样子，为了开辟新区工作。她们路过永丰村敌人的据点时小心谨慎地绕过敌人的据点继续前行。当她们已经走出一里多路的时候，突然发现有几个背大枪的鬼子远远地跟在她们后面，她们都感觉到了危险的来临，随即加快了脚步。

她们紧走了一段路，发现敌人仍然紧紧地跟在后面，急中生智，她们迅速来到要政的婆家村，她两进村后为了甩掉敌人，所以就分开行动。要政绕了几个圈子，就走进了婆家门，进屋后看到婆婆正在织布，要政随即向婆婆示意外面有敌人。婆婆心领神会，走出门去望风，要政换了衣服和假发，坐在织布机前织起布来。

![](.evernote-content/AA3ACA58-9710-4BC4-8B1C-6F95251800EB/7BE8F6D6-E5A3-467B-BE7B-3E0A76337D7A.jpg)

此时，胡同里传来敌人的脚步声，随着鬼子：“抓共产党呀！”的嚎叫声，弄得全村鸡飞狗跳，乱成一团。片刻，鬼子杀气腾腾地闯进院子，几个端着大枪的鬼子直接就冲向织布机面前的要政。一个歪带帽子的洋鬼子学着半生不熟的中国话：“喂，共产党的，跑来了没有？”要政从容地拉着长音：“哟，共产党？谁知道他们是什么样？他们今天真是要跑到我家来，我可真想看看他们到底什么样？”

另一个鬼子接着说：“女的，女的”。要政假装不懂地说：“什么？女的？共产党不都是男的吗？”领头的鬼子不耐烦地说：“别他妈的瞎啰嗦，共产党地交不出来，死啦死啦地！”要政神情自若地说：“噢，闹了半天是不想信我呀，那就请便吧。”说完，手中的织布梭子飞快地运转起来，头也不抬地织起布来。鬼子看不出什么漏洞，便乱翻了一气，抓了几只鸡，灰溜溜地走了。

鬼子走了，要政婆媳两长出了一口气，婆婆对要政说：“可真险啊？”要政却说：“有什么可怕的，干革命还能没有一点风险，流血牺牲都是正常的事。”婆婆用钦佩的目光望着要政，布满皱纹的脸上露出了欣慰的笑容。

![](.evernote-content/AA3ACA58-9710-4BC4-8B1C-6F95251800EB/681C3961-9B27-4E21-AA97-9C9B5D726E85.jpg)

日军对根据地残酷的“五一”扫荡，给人们带来了更加深重的灾难，抗日干部越来越少了，在日军的严密封锁下，剩下的干部也失去了联系。要政一面积极寻找组织，一面组织发动群众工作，虽然工作的难度增大了，但要政却一点也不灰心，仍然信心十足，克服重重困难，一直到分娩她才稍事休息。

1943年3月7日，要政舍下刚出生的儿子张锁成继续开展工作。同年4月18日，随着夜幕降临，要政和县妇救会干部薛机农、区抗联主任杨森同志带着发放谷种的任务又出发了。她们一路上一边观察敌情，一边挨村挨户地开展工作。当夜深人静时，她们经群众的再三挽留，就住在了一农户家里。

凌晨三时左右，“咣！咣！咣！”一阵急促的敲门声把她们在睡梦中惊醒了，她们急忙翻身下床，拉开门闩，只听村长气喘吁吁地说：“快！快走，敌人来了。”要政和小薛随即化妆向村口走去，刚出村口就与敌人相遇。要政知道已经无法躲避了，只好大声地咳嗽了两声，通知后面的小薛有情况，小薛收到信号后及时转身躲开了，可敌人却向要政扑来，要政被捕了。

![](.evernote-content/AA3ACA58-9710-4BC4-8B1C-6F95251800EB/864A97F0-D9B7-47EB-B7CB-47A92783516D.jpg)

敌人把要政速带到晋县城监狱中。要政从被捕的群众中得知小薛脱了险，也使她放了心。但她看到这些无辜的群众心如刀绞。几天过去了，任凭敌人皮鞭、棍棒的严刑拷打，她咬紧牙关，始终没暴露身份。一天，号称杀人魔王的伪刘司令把要政拉出去审问，要政又受到了一顿酷刑，使鲜血淋漓的要政几度昏死过去，又几次被敌人用冷水冲醒过来。

当要政再一次被冷水冲醒过来时，睁开眼睛看到站在面前的是苑四（叛变投敌的原妇救会干部），要政马上明白了一切，她满腔怒火，忍着伤痛强撑起身子大骂：“汉奸！走狗！卖国贼！”敌人得意地说：“不要发火，你就是要政，二区妇救会主任，不错吧”？要政怒视着敌人，坚定地说：“对，我是要政，二区妇救会主任，中国共产党党员！”“痛快！那么你的同党是谁？他们都在什么地方？坚壁的粮食都藏在哪里？说出来不会亏待你的！”

“呸！想从我嘴里得到什么，办不到！”敌人恼羞成怒，把要政吊在树上轮流毒打，而敌人所得到的只是痛骂。要政被捕后，我党曾组织过多次营救，但均未成功。心狠手辣的敌人企图利用要政的儿子威胁要政，但由于党组织的及时采取措施，使敌人的阴谋没能得逞。

![](.evernote-content/AA3ACA58-9710-4BC4-8B1C-6F95251800EB/431B3BC1-A700-4C08-B87A-FE089A6E07A6.jpg)

同年4月，鬼子用木棍夹住了要政的全身，绑赴到刑场。伪军刘司令用手枪指着要政的脑门说：“年轻的女共产党员，难道你真的不怕死吗？”“怕死？怕死就不革命，怕死就不是共产党员！”“别忘了你还有心爱的儿子，难道你不管吗？”要政从容地大笑几声说：“哼，你也别忘了中国还有千千万万的人民，他们是决不会饶过你们的，血债一定要用血来偿！”

敌人面对这位宁死不屈的女共产党员，束手无策，只好动了杀机，敌军官歇斯底里地狂叫：“我要把你剁成肉泥！”随即，残暴的敌人对要政前一刀、后一刀、左一刀、右一刀，敌人转圈轮流地向要政刺去，每刺一刀，要政就高呼一声：“抗日必胜！日寇必败！当汉奸没有好下场！”

要政的宁死不屈，吓破了敌人，他们胆颤心惊地倒退了几步，惨无人道的伪军刘司令指使敌人蜂拥而上，从四面八方一齐刺向了要政，使要政浑身插满了刺刀。英雄的要政凭借最后的力气高呼：“打倒日本帝国主义！共产党万岁！”我们的要政停止了心跳，为抗日大业而牺牲了。

![](.evernote-content/AA3ACA58-9710-4BC4-8B1C-6F95251800EB/7A5D0C24-C56A-4F70-A93E-46EA234F0B1A.jpg)

要政牺牲后，所有的汉奸、叛徒都没能逃过人民的惩罚，在新中国成立后，统统被枪毙。但是，我们的英雄要政却永远也活不过来了？让我以悲痛的心情向大家讲述革命历史英烈们的事迹。当我们遇到不快的时候，一定要多想想那些在为了民族解放事业英勇献身的英烈们，愿他（她）们在天有灵，永垂千古！
