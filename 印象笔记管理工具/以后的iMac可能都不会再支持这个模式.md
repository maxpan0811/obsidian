# 以后的iMac可能都不会再支持这个模式

---

![](.evernote-content/3DCE9E07-2C3A-4C08-B4FC-237C6460C554/369F2D74-9F5B-4C28-92F0-24F3134A6510.jpg)

苹果已经明确表示，2017 年 iMac 将不支持 Target Display Mode 模式。此前 2009-2014 年 iMac 在该模式下，支持用户通过 Mini Display Port 或 Thunderbolt 连接，将其内置的屏幕来作为其他 Mac 设备的外部显示器。

该功能最早是在 2009 年晚些时候发布的 27 英寸 iMac 上亮相，让 iMac 成为其他 Mac 设备的外显既方便了用户，又延长了设备的使用寿命。如果一台iMac它的规格太落后不适合用来处理大型任务等，可它还是可以当外显，充分发挥余热。但设备要是部件除了问题导致 Mac 无法正常工作的话，那么 Target Display Mode 模式也就无法正常使用。

后来在 2010 年中、2011 年中和 2014 年中更新的 iMac 产品仍然该模式，可是到 2014 年晚些时候更新的 5K iMac 中，因为设备仅配备了一个 Thunderbolt 2 接口，无法以全分辨驱动新的超高分辨率显示屏。此后 iMac 再也不支持这个模式。

最近随着新款支持 Thunderbolt 3 的 4K 和 5K iMac 的机型出现，不少人又抱有期待，希望苹果能够重新推出这个模式。

可是外媒已经通过苹果方面发言人了解到，2017 iMac 将不会支持 Target Display Mode，并表示苹果没有计划重新启用这个模式。苹果相关支持网页（最新更新是在 2017 年 3 月）同样指出：iMac（2014 年晚些时候 27 英寸 Retina 5K）及以后的 iMac 机型均不能用作 Target Display Mode 显示屏。
