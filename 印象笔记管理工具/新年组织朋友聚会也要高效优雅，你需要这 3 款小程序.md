# 新年组织朋友聚会也要高效优雅，你需要这 3 款小程序

---

春节假期除了与家人团圆外，与朋友聚会也是一件相当重要的事情。可在绝大多数情况下，一次聚会总会遇到以下几个问题：

* 组织聚会时，群聊中讨论聚会/吃饭地点各方意见不一；
* 聚会经费总是一个人先买单，再 AA 制或者先缴纳费用，但却没有明细；
* 聚会后的照片不能统筹一起，各自发各自的图片。

难得一次的聚会，难道就不能把这些问题简化或解决，让聚会的重点回归到老友间的交流吗？答案或许去年没有，但今年这 3 个小程序的出现，让问题有了一个让人满意的答案。

**组织能手：群里有事**
-------------

**「群里有事」**主要由发布通知、发布活动、群发起投票 3 个功能组成，利用它们让组织聚会更高效。

### **发布高效的群通知**

微信群聊组织聚会已是近几年的常态，可当群聊聊天内容或聊天人数增多时，完全就不能好好谈完一件事情。常常一个通知（群公告）瞬间就被新的聊天信息覆盖，通知会被多少成员有效阅读不得而知，甚至极有可能部分成员完全不知情。但现在，借助小程序「群里有事」的发布通知功能便能解决。

在该小程序中点击下方「发布」选择发布通知，编辑界面除了常规的标题、内容、图片添加外，还包括更多选项功能中的署名、文号、模板选择。新年期间，「群里有事」还为用户准备了各种新年节日模版。

![](.evernote-content/03CB684A-D80C-4947-A222-6C31540D6077/A286FE30-31FA-44CD-9A4E-8C972BE481DB.png)

将编辑完成的通知发布于群聊后，群聊会提示收到来自小程序的信息通知，当成员打开后，「群里有事」便会自动将成员浏览记录下来，时间精确到几时几分。这样一来，通知发起者就可以清楚看到发起的通知被多少成员或哪些成员有效阅读。

![](.evernote-content/03CB684A-D80C-4947-A222-6C31540D6077/525E8D0F-2574-4C29-8FA1-4C29C2CC2867.png)

### **发起聚会/吃饭地点投票**

每次讨论聚会/吃饭地点时，总会有人存在意见分歧，虽然最后结果是以少数服从多数，但结果出来前在群聊中统计成员意见却是一件令人抓狂的事情。而利用「群里有事」可成功发起的利于统计的投票，让投票完成后能清楚看到各个意见的所占比例。

![](.evernote-content/03CB684A-D80C-4947-A222-6C31540D6077/CC2C9DDC-0EC9-4695-9168-85AACA59FFE7.png)

### **搭建聚会报名入口**

当聚会参与人数可能过多时（如班级聚会），就需要收集确认报名的人数以及具体的联络方式，而这些利用「群里有事」的发布活动功能便可解决。

当你将编辑好的活动发布群聊后，成员打开时便可以清楚地知道活动的详细内容、地址；点开地址时，支持跳转至微信自带的地图功能，不仅可以直接获取导航路线，还可以将该地点用手机安装好的地图应用打开，让那些不熟知聚会地点的成员也可以轻松找到具体地址。

![](.evernote-content/03CB684A-D80C-4947-A222-6C31540D6077/76E2235C-EB63-4A17-B869-1146BBB4C234.png)

成功报名后，活动下方还会显示报名人信息以及联系电话，相当于临时电话薄的存在，这对于组织聚会者来说确实能提供不小的便利。

**经费更透明：腾讯小经费**
---------------

每一次聚会产生的费用并不容易厘清。虽然彼此间都为朋友，但遇到收费原因不透明、收支不全、余额不清楚、自掏腰包等情况时，难免会破坏一份好心情。这种情况下，腾讯官方出品的社群活动经费管理小程序「腾讯小经费」便不容错过。

「腾讯小经费」主要针对打卡挑战、活动经费、校园经费以及情侣经费管理的场景开发，其中的 **活动经费**便能解决我们组织聚会时产生的费用问题。当你创建活动经费圈并填写金额和收费原因后，你可以发布到指定群组或邀请任意微信好友，用更好的形式收集经费。

![](.evernote-content/03CB684A-D80C-4947-A222-6C31540D6077/715857F8-F20F-49F9-BD55-6DAE364451D7.png)

同时，每个经费圈成员都能点击小程序中的「余额」查看每一笔费用的支出和提取记录。成员若有自费时，也可以在该小程序选择提取经费并留下提取原因或票据，等待审核员审核便可提取成功。

**用照片留住美好时光：群相册服务**
-------------------

聚会时，小伙伴都会拿出手机记录这场难得的见面，久久未曾见面的大家也常常会提出合影要求。这些照片对大家来说不仅值得讨论，而且更值得保存并记忆，但现实中成员间分享图片都是直接发送至群聊之中，并不易于讨论和整理，甚至还会遇到照片过期被自动清理的情况。

倘若你遇到以上情况，那么不妨试一试由腾讯相册管家官方出品的小程序「群相册服务」。借助它，我们可以创建群聊的专属群相册，发送至群聊后，每个微信群成员都能点击浏览并点赞和评论，同时也可以上传自己手机拍下的照片，让群相册的内容更加丰富多彩，更利于整理和回顾。「群相册服务」还支持智能人脸识别，易于浏览。

![](.evernote-content/03CB684A-D80C-4947-A222-6C31540D6077/79C426B6-F6A8-4794-ACA5-6B34FF2E0DA2.png)

更为重要的是，上传至「群相册服务」的照片将长久保存，不会出现过期被清理的现象。

希望这 3 款小程序能一定程度上解决部分人在组织聚会过程中遇到的繁琐事情，让聚会更好地进行。毕竟，朋友间难得的聚会最为重要的是彼此间的交流，而不是如何解决这类繁琐事情。
