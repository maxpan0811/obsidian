# 向别人展示 Demo 时，巧用「辅助功能」增强演示效果 | 一日一技 · Mac

---

### *关于栏目*

「一日一技」是少数派的全新栏目，我们将会介绍各种简单又实用的小技巧。这些技巧可能是你知道的，也可能是你还未注意到的；它可能是一个系统的操作技巧，也可能是某个 App 里的细节功能或用法……我们希望通过这个栏目，让你更好了解手中的设备和 App，能更充分去利用它们的特性，以此一点点改善与提升你的数字生活。

 「辅助功能」算得上是 OS X 「系统偏好设置」中项目最多，却又最少被人关注的一项面板了（与之相比，iOS 的「辅助功能」面板恐怕也有过之而不及 :-D ）。

这一点非常容易理解，毕竟「辅助功能」是帮助那些视觉、听力、肢体活动能力或认知与读写功能有缺陷的用户，顺利使用 OS X 的重要手段。但事实上，普通的 Mac 用户，也可以用其去实现一些需求。

![](.evernote-content/4D0AF01A-2920-488E-BD1B-541CC2E719F2/4EA867E1-3768-49F8-9F4D-BF4516ADECBC.jpg)

需求
--

在利用 Mac 向他人展示部分操作的 Demo 时，您可能会遇到下面两个十分常见的问题：

1. 在制作部分软件使用的视频演示时，放大效果可以更好地向他人展现操作的细节，如具体菜单项、按钮等。但人们最为常用的 QuickTime Player 「新建屏幕录制」，却无法实现这一点。
2. 出于环境以及投影设备自身亮度等条件的限制，将 Mac 的屏幕内容镜像到投影设备中进行演示时，在下的听众往往不容易看清光标在屏幕中的位置，从而使得演示效果大打折扣。

方法
--

在第三方付费应用的帮助下，前述需求当然很容易解决 — 例如，利用 [ScreenFlow](http://www.telestream.net/screenflow/overview.htm)的后期编辑可以方便地给视频和图片添加放大效果。但对于需求尚低的普通用户而言，付出高昂的资金和学习成本只为解决一个小的问题，则无异于「大炮打苍蝇」。

在「辅助功能」-「缩放」中勾选「使用键盘快捷键来缩放」，随后利用 `Option + Command + 8`（⌥⌘）快捷键，即可实现简单的「局部放大」功能（注：也勾选上「以键盘的焦点为中心来缩放」来调整缩放效果）。

![](.evernote-content/4D0AF01A-2920-488E-BD1B-541CC2E719F2/FC1D6EE2-6914-47DA-99B3-1CCB34F5B0E1.jpg)

在「显示器」中调大鼠标指针光标大小，即可轻松调整光标大小。

![](.evernote-content/4D0AF01A-2920-488E-BD1B-541CC2E719F2/D006C506-CC0C-4B1B-B619-A31D8E8472BC.jpg)
