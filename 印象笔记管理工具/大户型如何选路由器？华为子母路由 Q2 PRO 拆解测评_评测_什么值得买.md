随着[智能手机](https://www.smzdm.com/fenlei/zhinengshouji/)等移动设备的快速发展，无线[路由器](https://www.smzdm.com/fenlei/luyouqi/)在近十年的时间内迅速普及，可以说已经成为当今家庭中必备的 IT 设备之一，但是对于大多数的普通消费者而言，在选购路由器时其实很迷茫，往往会将 “天线多”、“价格实惠” 等作为选购的标准，而对于居住在复式 / 跃层 / 多层等大户型或复杂户型内的用户来说，更头疼的是如何解决 WiFi 信号覆盖面积的问题，是无线中继 / 还是[电力猫](https://www.smzdm.com/fenlei/dianlixianshipeiqi/) / 还是 Mesh 网络系统？本文我们就来了解一款大户型解决方案路由产品 —— 华为子母路由 Q2 Pro，顺带着也会聊一聊路由器选购中的一些问题。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/C653D402-7E5D-4B32-B3A9-6F210F432845.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_2/)

华为路由 Q2 Pro 子母装由一个母路由和一个子路由组成，不同于常见路由器扁平盒体加多根天线的造型，华为路由 Q2 Pro 采用了椭圆形柱体的外观设计，并将天线内置在了乳白色的塑料外壳中，简约优雅的造型也能更好的和时下流行的简约风家居风格相融合，而且也更具[智能家居](https://www.smzdm.com/fenlei/zhinengjiaju/)的科技感。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/161E6E05-0CF5-4310-A2DA-385B50C12E06.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_3/)

母路由的外形尺寸为 112 mm x 73.3 mm x 183.5 mm，重量约 459g，乍看会让人以为体积比较大，但其实和传统造型的路由器相比，它占用的空间要小很多。在路由器的正面非常简洁，只有一个 H 键，通过 H 按键，也可以一键将支持 HiLink 智联的华为盒子等[智能家居产品](https://www.smzdm.com/fenlei/zhinengjiaju/)连上路由器的 WiF，H 按键同时具有 WPS 功能。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/F3A9E31C-16F8-43A8-A575-04B19855FB30.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_4/)

正面和底面的连接处设计有类似于 “海岸灯” 的条状状态指示灯，未接入网络时为红色，成功连接网络后为蓝色，观感上还是比较有科技感和设计感的。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/04954329-4E8C-4720-B81F-9BD97EA57392.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_5/)

路由器的背面配置有一个三脚的电源接口和三个 10M/100M/1000M 网线接口，这三个接口支持网口盲插，不分 WAN 口和 LAN 口，入户宽带随便接入哪个网口都可以使用，对于小白用户非常友好。接口数量偏少，而且有一点缺憾的是网口没有指示灯，无法显示各网口当前状态。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/72E525C8-91D0-44D5-888D-F7362E632522.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_6/)

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/1D2AC6AE-DB75-4794-9E3C-B13AC0B636E1.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_7/)

在设备底部为路由器的基本信息，RESET 孔，以及围绕底部一周的大面积防滑脚垫，并且开有大栅格散热孔，而设备的顶部非常的简洁，除了暗刻的华为 LOGO 外没有任何其他元素。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/359025B1-D513-487B-91DA-4F9E8D73B01F.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_8/)

另外不同于常见路由采用的外置电源适配器，华为路由 Q2 Pro 将电源适配模块配置在了机身内，所以随机配备的是一条 1.6 米的三孔电源线以及一条 1.6 米防缠绕扁平 RJ45 网线。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/164D819C-93C3-4246-BCAF-1E70FCD8E92F.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_9/)

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/F95376E1-4413-4380-BC39-F85ADB915ADE.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_10/)

子路由外形尺寸为 75 mm x 46.3 mm x 103 mm，重量约 200g，正面为一颗状态指示灯，未接入网络时为红色，成功连接网络后为蓝色，背面为三孔插角，底部为一个 10M/100M/1000M LAN 口和 RESET 孔，而顶部同样为非常简洁的华为 LOGO。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/A02239B1-186A-451E-814D-F6E02166179B.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_11/)

从这款路由上其实可以顺便解释一下，“天线越多，信号越强” 的误区，首先天线可以内置也可以外置，这款产品就是内置的，而无线路由器的发射范围是由 IEEE802.11 无线标准决定的，而不是看天线。另外信号的强度和 “穿墙” 能力，则取决于路由器的发射功率，但是出于健康考虑工信部对无线路由器的发射功率是有限制的，不得高于 20dBm。当然也不排除少数山寨品牌会不顾法规刻意调高发射功率，有些玩家还还会通过刷新第三方固件，强行突破发射功率的限制，所以天线只是 “介质”，起到一定的增益左右，并非决定信号强弱的关键。而更关键的是内部 CPU 和 WIFI 芯片的配置，它们才是传输速率的关键，包括天线的数量也和其相关，这些我们下面再聊。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/6B582F17-D9A4-4B5D-81CA-80E5EF981016.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_12/)

从拆解中可以看到，华为路由 Q2 Pro 母路由的内部有两层构成，上层主要为电源适配模块，电源模块的背面是用于散热的均热板，下层为 PCB 电路版，电路版正面和背面的主要器件都被金属屏蔽罩所覆盖，所以看不到 CPU 和 WiFi 芯片的型号，只看到闪存来自 winbond (华邦电子)，大小 128M，另外母路由内没有独立的天线模块，完全依靠板载发射，在电路版上只能看到两个 MMCX 接口。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/EE4A6F3E-2AEF-4539-8507-9B354DBF980A.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_13/)

通过官方查询我们得到了母路由的完整配置方案，CPU 是来自 Realtek（瑞昱）的 RTL8197FS ，单核，主频 1GHz，内存为 128M DDR2，CPU 内置 2.4GWiFi 芯片，5G WiFi 芯片为 Realtek（瑞昱）的 RTL8812BRH，另外还配有电力线通信 PLC 芯片，型号为来自华为自家的 Hisilicon Hi5630S。所以它和子路由的通信模式其实就是我们常说的电力猫的模式。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/6E33DFBB-79F8-43AD-B672-3AB74C9E7EEF.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_14/)

从这些配置中我们就能得到这款路由的主要性能参数，首先 CPU 是支持全千兆有线网口的，而 WiFi 芯片支持的传输速率为：2.4G WiFi 300Mbps，5G WiFi 867Mbps，两者相加为 1167Mbps。并且支持 2\*2 MU-MIMO (多用户多入多出技术)，而这种模式下两根天线就够了。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/9C8E47B5-76A2-4985-BF25-2358D2121EAB.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_15/)

所以通过以上我们可以了解在选购路由时一些知识，首先千兆网口需要内部芯片的支持，而且真正的全千兆网口应该是 WAN 口和 LAN 口都是千兆，而很多所谓的千兆 WiFi 其实都是 2.4G 和 5G 频段相加的结果，并不是单一频段的传输速率，而且这种 300Mbps+ 867Mbps 的 WiFi 芯片（2\*2 MIMO）2 根天线基本就够用了。当然一些高端产品确实能到达单频段千兆传输速率，而且支持 4\*4 MU-MIMO，这时就需要 4 根以上的天线。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/888CCCEE-B231-42D0-98FC-10E980CD5F9C.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_16/)

再来看子路由的内部构造，和主路由类似上层为电源适配模块和均热板，下层为 PCB 点电路版，主要器件同样被屏蔽罩所覆盖，电路版左右为两个突出的天线，闪存同样来自 winbond (华邦电子)，大小 16M。经查询其 CPU 是和主路由对应的 Hisilicon Hi5630H，双核，主频 500MHz，内置 PLC 芯片，2.4GWiFi 和 5G WiFi 都是华为自家的 Hisilicon Hi1151。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/6E612029-272C-42BD-BB13-9F105BC71DF0.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_17/)

可以看到华为路由 Q2 Pro 子母路由之间采用的其实还是电力猫的方案，通俗来说就是通过家里的电线进行电力载波通信，子母路由在同一电力网络中即可，也可以理解为统一电表下的电路，它最大的优势在于避免了 “穿墙” 的困扰，而且可以做到信号哪里不好插哪里，非常方便。不过电力猫也是有先天缺陷，因为主要依托电网零火线传输数据，容易受家里大功率电器齐干扰，影响传输效率。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/1725F109-E94B-4855-92BE-EF4D4F051AA3.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_18/)

而华为路由 Q2 Pro 的解决办法是采用华为自研的千兆电力猫芯片凌霄 5630，并开发了 PLC Turbo 技术，硬件上从传统的 2 脚插头升级到了 3 脚插头，引入干扰更少的地线，增强了信噪比，使得组网性能得到提升，并且最多可以做到 1 个母路由拖 15 个子路由。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/6B8A0C49-9DB4-4D62-BF62-F2B437B976FD.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_19/)

当然它也不可能完全避免干扰，所以最好使用墙体上的市电插座，尽量不要使用插排，而且尽量远离空调、微波炉等大功率电器。那除了电力猫的方案，大户型 WiFi 扩展方案，还可以使用常见的无线中继器或者放大器的方案，以及 Mesh WiFi 方案，也就是多个路由之间采用网状 Mesh 拓扑结构。其实华为路由 Q2 Pro 除子母套装外，还有一个旗舰版，也就 3 母装。3 母装每两个路由间采用的是千兆电力线和 Mesh Wi-Fi 双通道混合组网技术，也就是每两个路由器之间同时建立了一条电力线连接通道和一条 Wi-Fi 连接通道，组网速度做到叠加，达到更高速稳定的扩展。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/89C087AD-6C51-4953-A9FE-61A73CB13F58.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_20/)

除了硬件配置，配套 APP 和连接的便捷性也是衡量一款路由器好坏的重要因素，华为路由 Q2 Pro 没有独立的 APP，而是被整合在华为 “智能家居” APP 中，再加上 IoT 专属通道以及路由器一键智联的 H 键，这其实都体现了华为致力于打造智能家居物联生态的决心。

将华为路由 Q2 Pro 连接电源和入户宽带后，手机连接到名为 HUAWEI-XXX 开放的 WiFi，打开手机 “智能家居” APP，输入宽度账号和密码即可接入网络。另外华为路由 Q2 Pro 还提供了从旧路由获取账号密码的功能，为不记得账号密码的用户免去的寻找的麻烦，使用方法也比较简单，将旧路由接通电源，用一根网线连接旧路由的 WAN 口和华为路由 Q2 Pro 的任意接口，再点击手机连接界面中的 “从旧路由获取” 按钮即可。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/2A3C13F4-85BF-4A06-91F1-F9B2E4D1207A.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_21/)

APP 中除了路由器常见功能外，还可以进行儿童上网和客人 WiFi 等设置，在 WiFi 设置中可以看到，华为路由 Q2 Pro 拥双频合一功能，将 2.4G 和 5G 双频合并显示为同一个无线网络名称，路由器会自动选择最优频段，无须用户手动切换。

而子路由的连接更加便捷，只需将子路由接入插座即可，无需进行任何设置便可和主路由建立通信，而且华为路由 Q2 Pro 的子路由和母路由会自动统一成一个 WiFi 名称。而且具有 “无缝漫游 “功能，整个子母路由系统会根据设备的实时位置，自动切换连接信号更好的 WiFi 热点。也就是如果你感觉哪个房间或位置信号不好，就可以把子路由安装在这个位置，而且路由器会为根据信号的强弱自动为你切换到子路由或母路由，非常的方便。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/9CF3EC16-89A1-48E1-9F77-EFCC3AE4F31C.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_22/)

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/BA701D2E-92B9-4A8B-9273-DAE9B4BB510F.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_23/)

在实际的使用感受，我们的办公室正好是复式的 110㎡的户型，而且由于是办公楼，WIFI 信号干扰非常复杂，我们通过相关软件测试不同位置的型号强度。可以看到，在同一房间距离主路由 3m 左右信号强度 - 25dBm 左右，同层隔一堵墙信号强度 - 40dBm 左右。楼下号强度也在 - 40dBm 之间，楼下再一堵墙信号强度 - 55dBm，再此房间处安装子路由后信号强度 - 40dBm~-45dBm 之间，楼下再两堵墙信号强度 - 60 dBm 左右，再此房间处安装子路由后信号强度 - 50dBm，可以看到子路由对信号的提升还是比较明显的。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/E13DABA9-A891-4E7C-8340-E3A9CC04531A.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_24/)

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/EEAA238F-9901-479E-87FB-9999EE910F82.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_25/)

在实际游戏的体验上来看，不管是《王者荣耀》或者《绝地求生》整个游戏对局中网络也比较稳定，没有出现因路由网络延迟而造成的卡顿现象。

[![](.evernote-content/0DFCE4C3-48BC-4040-800A-266FD616A1ED/8E1067F8-22FC-4439-AE13-A99F83113494.jpg)](https://post.smzdm.com/p/a83dxnpl/pic_26/)

总体来看华为路由 Q2 Pro 这款产品，很多朋友会说它在硬件配置上并不突出，算不上高端或电竞级的路由产品，那是因为它的侧重点并不在这，它的性能重点在于解决大户型或复杂户型家庭的 WiFi 信号覆盖问题，而且在传统电力猫方案的基础，通过软、硬件的优化，让子母路由达到了更高速稳定的信号扩展，如果有更高要求的话，还可以通过多个母路由组成 Mesh WiFi 和 PLC 的混合组网。 另外从设备产品设计、“H” 一键物联以及配套 APP 上也可以看到，华为在智能家居生态上的布局，旨在将华为旗下的路由打造成智能家居中心的。
