注：如无特殊说明，本文所述的「色彩管理」，并不是指显示器的颜色校准，而是指将颜色作为信息实体进行收集、整理等。

---

对于设计师和开发者来说，**色彩**是一种值得「管理」的信息。

早在影像技术发明以前，人们就开始尝试收集和整理色彩了。 19 世纪初出版的《维尔纳色彩命名法》（Werner's Nomenclature of Colours），就是一本著名的色彩分类指南，每种颜色都有对应的名称和实物样本，广泛服务于当时的艺术家和科学家。

著名的博物学家达尔文进行生物学研究时，就随身携带着一本《命名法》。因此，他可以通过比对色卡来确认他眼前生物的颜色术语，并以纯文本的方式，给千里之外的研究者精准地描述色彩。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/8DD7C0B9-ED9B-4BF4-BF97-E589AD3EC8FF)

达尔文：「我发现一种蛇，它的花纹浅如樱草黄，深如胆石黄，间杂墨黑线条」引用自公众号「几册」，《[维尔纳命名法：达尔文的色彩指南](https://mp.weixin.qq.com/s/ou5fQOIzYVsu99rB7UsbjA)》

到了今天，色彩管理的意义不仅仅在于建立共识和交流，更在于构建能提升色彩感知的知识储备。很多设计师们、画师们都有收集色卡进行配色的习惯，在培养审美能力的同时，也让他们能用自己的积累，更加从容地创作作品。

而本文要介绍的由 Ola Brothers 出品的 **Sip**，就是管理色彩的绝佳工具，且广泛服务于众多著名互联网企业。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/A715321D-AC8A-4EF7-8965-B0D6450D8E78)

作为一款老牌软件，经过多次版本迭代，拥有众多忠实用户

有了 Sip 的帮助，构建一个类似的色彩管理闭环并不复杂，我们可以把整个过程抽象为**收集**、**整理**、**运用**和**输出**色彩，四个步骤。

收集：令人愉悦的取色体验
------------

收集，是色彩管理的第一步。

数字化技术出现前，人们会通过固定比例的配方调制出想要的颜色。到了今天，只需要取色工具便可轻松获知目标色彩的通用代码。而 Sip 具有优秀的取色功能，你甚至可以完全将其只当作一个取色器来使用。

只需按下快捷键或点击菜单栏图标即可打开取色器，点击想要取色的地方，就会复制点选像素的颜色代码到剪贴板，且所有取到的颜色都会收录到历史记录中。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/D152F27C-DB29-44ED-8E58-C18A210D0EB0)

相比 macOS 自带的「数码测色计」，Sip 支持更高精度地移动取色选框。你可以通过 `滚轮` 或 `滚轮 + Command (⌘)` 键，来分别调整选区视野和缩放。通过方向键对选区进行像素级移动。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/C691A381-338E-4877-AEBA-F0F636540160.gif)

为了方便连贯地提取一片颜色，只需按下 `Control(⌃)` 键再点选，即可实现批量取色。批量取到的颜色会单独集合成一个调色盘。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/5CD25EAC-A06C-40BA-9EE9-0A1084B5E3DC.gif)

当然，除了使用取色器，你也可以使用内置的编辑器手动输入或选取颜色。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/D6181313-66F3-4177-82C1-4D863A4A31C6)

和专业软件中的类似，不仅提供 Hex 代码支持，还支持分别输入 RGBA 或使用无级滑块

此外，Sip 还支持自动识别你剪贴板中的颜色代码，并将其菜单栏图标更改为识别到的颜色，在收集的同时帮助你快速获知一串代码究竟是什么颜色。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/74913783-C44E-4AE7-B874-A93E2CB8A0A9)

同时，Sip 支持超过 24 种主流颜色格式，你可以在设置中预先选好想要的颜色格式，也可以指定某种格式来复制已保存的颜色，甚至可以开启智能格式（Smart Formats）功能，根据所开启的软件智能适配相应的格式。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/5A235F34-14A5-41CB-B76A-F37F3244610E)

支持单独设置大部分开发和设计软件的颜色格式

比如给 Xcode 设置为 Swift UlColor RGB，给 Visual Studio Code 设置为 CSS Hex，当你选定软件窗口时再打开取色框，并按住 `Command (⌘)` 取色即可智能粘贴对应的颜色代码。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/5A235F34-14A5-41CB-B76A-F37F3244610E)

可以看到 Xcode 中粘贴了 UIColor，而 VS code 中粘贴了 Hex，并不需要额外的操作

比如给 Xcode 设置为 Swift UlColor RGB，给 Visual Studio Code 设置为 CSS Hex，当你选定软件窗口时再打开取色框，并按住 Command (⌘) 取色即可智能粘贴对应的颜色代码。

你也可以通过这种方式和主流设计软件进行互动，比如**设置和获取** Photoshop 的前景色和背景色。省去了取色再单独设置的步骤，对设计师来说方便许多。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/9C787982-B7F0-4327-ACBC-7BE8A3261D3A)

打开取色框后，按住 Command (⌘) 再点击左键即可直接设置 Photoshop 前景色

Sip 现代化的取色器，让色彩管理系统拥有了一个方便、快捷的输入端，而这恰恰是大量收集的开始。有无数的信息管理崩溃于繁琐的输入，而优秀的取色器给 Sip 的色彩管理打下了坚实的基础。

整理：基于「调色盘」，构建你自己的色彩指南
---------------------

搞定了收集部分，接下来就是整理了。

Sip 拥有一个基于调色盘（Palette）的整理体系，它是多个色彩的集合，你可以按照需求进行调整、命名、喜爱或锁定你的调色盘。

同时，Sip 会在顶部显示一个「颜色历史」调色盘，它会收录所有你取过的颜色。默认的历史记录是无上限的，为了更实用，你可以在设置中自行限制历史记录的数量。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/E47DE8C9-BB26-43EA-AB6B-2490033E22CD)

我就限制了 25 个颜色历史，这样看起来会比较清晰

你可以通过两种方式创建自定义调色盘：一种是通过上文提到的批量取色，另一种是直接创建空调色盘。而对于每个调色盘，你同样有多种方式输入色彩，只需勾选调色盘前的小圆点，就可以通过取色器或颜色编辑器给选定的调色盘录入颜色，更加方便的是，你可以**通过拖动色块**的方式，将一种颜色在不同调色板中移动或调换顺序。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/FA8A383E-E7BE-48C8-BD3B-266DD932B8B8)

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/4D900924-1CAA-4E95-A508-4D87B9E57C69.gif)

勾选多个圆点即可批量录入到多个调色盘中

对于已创建的调色盘，只需单击，就可以以列表的形式显示更详细的颜色信息，包括颜色参考名和你选择的颜色代码。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/D6F76E00-FA4C-488A-8319-8C3D4FDDCF2F)

基于调色盘功能，你可以自由创建自己喜爱的颜色收藏、收集不同网站的配色或收集项目所需的主题色等等。比如下图就展示了三个主流浏览器的配色：Firefox、Safari 和 Chrome，当我们把 logo 高度抽象为色块时，logo 通过色彩所传达出的情感取向就十分明显了。通过这种方式，学习、收集优秀作品的配色思维，也对完善自己的作品有一定的参考价值。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/2596D45A-FCFB-4D4D-B4E1-568109E4960E)

从颜色上来看 Firefox 显然是最热情的浏览器

总之，基于调色盘的 Sip 具有一个包容且易用的整理系统，相信你可以更灵活地使用它来帮助自己梳理色彩信息。

运用：小工具，大用途
----------

当我们完成了收集和整理，一份专属的色彩指南便已成型。

为了能让你更好地运用、内化收集到的色彩，Sip 还内置了一些实用小工具，帮助你提高效率或解决一些小「痛点」。

Dock
----

为了让你对收集的颜色更触手可及，你可以在 Sip 的设置中启用颜色坞（Dock），并选择你喜爱的调色盘贴边常驻，方便你日常工作时调取，就像真的调色盘一样。同时，当你鼠标滑过，调色盘中的颜色会横向展开，这样就做到了不用时不占用空间，用到时又可以以合适的宽度浏览、调取颜色，十分人性化。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/226D2903-C116-47B4-9655-A4F9958CCA14)

顺滑的交互，值得一试

查看对比度
-----

「查看对比度」工具对于需要制作网站或文字海报的设计师或开发者来说非常实用，它允许你查看两种颜色互为底色或正文时的显示效果。换句话说，就是看一种颜色在另一种颜色上时，对比度是否明显，是否容易辨认文字内容。

你不仅能看到对比效果，更能看到对比度评分和建议，甚至一键自动修复糟糕的对比度。Sip 十分巧妙地将这些文字和颜色信息融合在了一起，让这个小工具看起来十分简洁。当你选定两种颜色后，也可以一键添加成调色盘。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/80CD2C23-338F-412A-9C43-C72FB978CED9.gif)

自定义颜色格式
-------

「自定义颜色格式」工具是一个适合开发者团队或企业内部使用的功能，它允许你创建自己独一无二的颜色格式。该工具提供了丰富的文档和帮助组件，有更高需求的用户可以自行学习。下图是一个十分简单的样例，它会在标准的 Hex 色码前增加 sspai 字样。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/9973EA1F-45A7-4AAF-8558-F50105CC57C3)

当然你也可以通过这个工具创建更加复杂的格式

快照与同步
-----

好不容易收集、整理了自己的调色盘，可不能让它们因意外丢失。

幸好，Sip 提供了本地快照和云端同步的功能，你可以手动或每分钟/小时/天自动创建你的调色盘快照备份，并在必要的时候回溯或导出所有调色盘。至于云端同步，只需你在软件内部填写简单的注册信息，即可免费开启自动同步。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/CA493CE2-68C4-4A4E-ABBA-A11C4A4B0365)

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/4999E666-0F6E-4B4A-BE10-C445F5233A5A)

国内的用户开启云端同步功能可能会使 Sip 卡顿或不稳定

加餐：使用 Aquarelo 进行混色
-------------------

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/680A2A3A-05E7-46A8-B2C4-23DC377564F3)

你也可以使用同样收录于 Setapp 的 Aquarelo 来增强你的工具流。Aquarelo 是一个专注混色的软件，你可以使用它来获得两种颜色之间的过渡色，支持生成最多 12 种过渡色。

只需将 Sip 调色盘导出为 Apple Color List 格式，即可导入 Aquarelo。接下来只要选择两个颜色即可生成，通过加减号调整过渡色数量。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/A50D2F8B-C894-4DF0-BB00-A6C258A006B6)

这里选择了 Firefox 的红色和黄色进行混色

如果你在构建一套统一的视觉系统不妨尝试使用 Aquarelo 配合 Sip 获取过渡色。

输出：多种格式，自由分享
------------

终于来到了色彩管理系统的最后一步，输出。

就像开篇达尔文将色彩名称分享给千里之外的研究员一样，当你完成了你的调色盘，你也可以选择和团队或朋友分享它们。

只需点击分享按钮，即可选择导出格式。除了自有格式 `.palette` 外，Sip 支持超过 14 种常见的文件格式导出调色盘，方便导入到合适的应用中。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/0AD1DCC3-8409-4260-AB5F-2B4846A86A18)

值得一提的是，当你导出的格式支持命名颜色时，Sip 也会帮你自动生成颜色名，比如 `.sass` 下，变量名即是色彩参考名。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/C2C8783A-79FE-4C9C-B9D8-813C7B22B875)

免去了起名的麻烦

除了开发场景，Sip 也支持更加通用的导出方式，比如 PDF 和 HTML。它会以专业报表的形式呈现你导出的调色盘。下图展示了分享为 HTML 格式的样例，可以看到不仅排版精美、颜色信息清晰，而且整个页面还做了自适应以保证在手机和宽屏设备上都有良好的阅读体验。此外，在每个颜色的右上角还提供了 Sip 的导入链接，方便你一键导入想要的颜色或整个调色盘。

![](.evernote-content/03C747D9-4E09-4341-A543-4F50471E8D34/491A5E34-A6A8-4799-8A37-B2B7E3DAEC9A)

从此开始，认真对待你的色彩
-------------

至此，一个逻辑自洽的色彩管理系统就算构建完毕啦。感谢 Sip 让**收集**、**整理**、**运用**和**输出色彩**这四个步骤如此简单。设计师、开发者们，不如从今天开始，尝试使用 Sip 建立起自己的色彩指南，更加认真地对待这些构成缤纷世界的颜色吧！

另外，对于需要色彩管理（指显示器的颜色空间校准）的专业人士，Sip 同样支持切换主流的 sRGB、Adobe RGB、Display P3 等颜色描述文件，别忘了提前在设置中选择你需要的描述文件再取色，这样才能收集到「正确」的色彩。
