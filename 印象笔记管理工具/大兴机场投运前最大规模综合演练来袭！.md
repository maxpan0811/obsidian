# 大兴机场投运前最大规模综合演练来袭！

大兴机场投运前最大规模综合演练来袭！
------------------

北京大兴国际机场

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/269CEA74-191D-4CED-9299-1A5DD627D48B.jpg)

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/0E071B7D-FCCD-4608-BE88-0DE296AE26CC.jpg)

就在刚刚，

伴随着8868名模拟旅客陆续退场，

大兴机场结束了第5次综合演练。

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/1F6E3EB1-94CC-4861-9156-680F56C150E7.jpg)

回首过往，

之前4场演练的盛况，

还历历在目！

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/26FDAF6B-480C-4F5A-9D19-EE48A922B8A8.gif)

看，这次小编成功集齐了5张演练邀请函！

请问现在可以召唤“神龙”了吗？

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/34BD857B-E006-444F-B151-A1BEB0230B2E.gif)

9时许，

8868名模拟旅客携带5600件模拟行李，

从大兴机场三、四层出发大厅正式开启本次演练。

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/94E17127-3AB9-4EF7-9F0F-24F1F6D2A934.jpg)

经过值机、行李托运、安检、

海关申报、边检、候机、登机

以及到达后行李提取等正常乘机环节，

对8个主流程进行了测试。

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/BA858358-3977-4EBC-8499-914DB740D4B2.jpg)

这次演练又有什么样的亮点呢？

 小编来给你说道说道~

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/23124421-4468-447F-BC19-E4411D0E0634.gif)

**No.1**

**规模最大：近9000人**

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/41EF684F-3760-42CA-BC19-27D99F50CFD1.jpg)

**8868** 名模拟旅客，

**5600**件模拟行李，

**140**架次航班，

又一次刷新了演练数据~

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/43354818-8C35-477F-A8CC-4083E0DF4B5B.jpg)

本次演练共涉及航空器进出港、

国内、国际进出港旅客保障、

国内进出港托运行李处理、

公共交通工具运送旅客保障、

货物进出港保障等在内的 **129**个演练科目。

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/6EAF3FD8-D5C5-4F65-ABF1-9FD04798862C.jpg)

覆盖航站楼全部区域，

开放跑道 **4**条，

值机柜台 **208**个，

行李转盘 **42**个，

登机口 **51**个，安检通道 **31**条。

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/29951171-F744-470B-B61C-8A5E4D1BEE78.gif)

**No.2**

**首次加入空轨联运流程测试科目**

首次加入大兴机场线草桥站空轨联运流程测试科目。

空轨联运科目完成模拟旅客在草桥站办理值机、

行李托运、行李安检等8个测试科目。

 走，跟小编一起去草桥站可以看，

 体验一下地铁出行吧~

草桥站共开放值机柜台 **4**个

（国际 **1** 个、国内 **3** 个）

安检系统 **2** 条，

列车间隔 **8.5** 分钟。

**No.3**

**首次进行2个时段航班高峰压力测试**

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/BF1CC804-A57C-4542-9901-94F197D692A7.jpg)

本次演练加入两个时段的航班高峰压力测试，

在午间11:00-12:00和13:00-14:00两个时段，

分别集中了50、60个进出港航班，

全过程仿真模拟机场运行。

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/08F91464-2F80-4F9E-AEAA-79AEF0F63313.jpg)

厦门航空首次加入大兴机场综合演练

共有两个班次的国内进出港航班

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/CDF41D44-A0C7-4F72-9103-AEABD203025B.jpg)

**No.4**

**首次进行安检信息系统故障处置**

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/89DC7D1C-FB56-4923-AFED-BA16988A811F.jpg)

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/543371A0-EAE9-46E6-AD64-57DC35369F3B.jpg)

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/5FA39BA4-7572-4A58-A74E-47CDCDF46E4F.jpg)

**No.5**

**星巴克挂牌开业！**

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/6E890545-406F-4735-94A5-30E6FD81189C.jpg)

刚给大家介绍完大兴机场网红饮品店，

好消息就来了！

这次演练，

4层公共区星巴克正式挂牌开业啦！！！

这可是首家开业的餐饮店哦~

看看队伍的长度就知道是有多火爆了

从今天起，这家店面就会正常营业啦

营业时间是早八点到晚五点哦！！！

店面位置在四层公共区东侧

小伙伴们快来打卡鸭！！！

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/58B7084E-1CA5-4474-A17F-A5392A4B508E.gif)

此外，本次演练，

还招募了来自北京市大兴区、

廊坊市安次区、广阳区

近200名残障模拟旅客参演，

实测大兴机场特殊旅客乘机

全流程的“无障碍”科目。

![](.evernote-content/351C837A-36EA-4616-80FC-76F7B74B8225/281653A0-25CC-459C-A9AC-ABFB4C910C69.gif)

随着大兴机场正式投运的临近，

演练覆盖范围和参演规模的增大，

大兴机场“以演促建、以演促练”的

整体效果不断增强。
