让 AirPods 在 Windows 上支持摘下自动暂停：MagicPods
=======================================

![](.evernote-content/AF04EAA7-AAD0-4D61-B488-28AD38B9B6C0/5F2CD72C-F5A3-422C-86D7-9A1BDA80329A)

App+1 | 让 AirPods 在 Windows 上支持摘下自动暂停：MagicPods

少数派之前介绍过两款让你可以在 Android 上使用 AirPods 更方便的小工具（[AndPods](https://sspai.com/post/55234)、[AirBattery](https://sspai.com/post/46147)）。作为一个 Windows 用户，我经常工作时把 AirPods Pro 连接我的电脑听歌，于是我也想在 Windows 上让 AirPods Pro 实现显示耳机电量、摘下自动暂停这些功能。在进行了一番搜索后，我找到了 [MagicPods](https://www.microsoft.com/store/productId/9P6SKKFKSHKM)。

第一次使用 MagicPods 前，你首先需要在系统的蓝牙设置里将电脑和 AirPods 进行配对。配对完成后，打开 MagicPods， 你可以在界面上看到目前 AirPods 和充电盒的电量。MagicPods 适配了暗色模式，会根据你系统目前的主题自动切换。

![](.evernote-content/AF04EAA7-AAD0-4D61-B488-28AD38B9B6C0/5AA593E9-0E6B-4916-AD25-F41659D9BCEB.png)

之后你每次打开电池盖准备使用 AirPods 时，Windows 都会弹出一个小窗口显示耳机所剩的电量，就和 iOS 一样（如果你不喜欢这个功能，可以在 MagicPods 里关闭 「AirPods splash screen」选项）。

![](.evernote-content/AF04EAA7-AAD0-4D61-B488-28AD38B9B6C0/C87F50A1-756D-44BE-8585-A2A3848ADF89.png)

除了在界面中显示电量，MagicPods 还提供了动态磁贴，在开始菜单就能看到耳机的电量。另外，当电量下降至 10% 时，MagicPods 会推送系统通知进行提醒。

![](.evernote-content/AF04EAA7-AAD0-4D61-B488-28AD38B9B6C0/FBD5BE7F-0828-40C4-A747-A319749C15FF.png)

MagicPods 还能让 AirPods 在 Windows 上实现摘下自动暂停的功能。目前你可以在几个主流浏览器（Chrome、Opera、Firefox、Microsoft Edge）、Spotify 以及 Yandex Music 里使用（如果你希望支持更多的应用，可以在开发者的 GitHub 页面提交 [Request](https://github.com/steam3d/MagicPods-Windows/issues)）。

当你看视频或者听歌时，只需要摘下耳机，MagicPods 就会自动停止。而当你重新戴上耳机，视频和音乐则会继续播放。你还可以在 MagicPods 里自定义摘下后的动作，除了播放 / 暂停，你还可以把它设定为降低播放音量。

![](.evernote-content/AF04EAA7-AAD0-4D61-B488-28AD38B9B6C0/61EAB88C-95C4-4F8D-B71C-94BF4D4E2E75.png)

在 Windows 上使用 AirPods 时，偶尔会觉得会存在一定的声音延时，这个问题在 Premiere Pro 或 Audition 等专业工具里比较明显。MagicPods 提供了一个低延迟的模式，开启后耳机会消耗更多的电量，但一定程度上能降低延迟的影响。

![](.evernote-content/AF04EAA7-AAD0-4D61-B488-28AD38B9B6C0/7A24D59D-352D-49B6-8315-3B889952E863.png)

使用 MagicPods 时，如果你发现无法检测到 AirPods，可以在设置里调整信号搜索的强度，或是开启「快速搜索」模式。另外，根据开发者的 [说明](https://steam3d.github.io/MagicPods-www/)，MagicPods 仅支持蓝牙 4.0 及以上协议的设备，同时配对时请确认自己的 AirPods 设备名内含有「AirPods」，例如「某某某的 AirPods」。

![](.evernote-content/AF04EAA7-AAD0-4D61-B488-28AD38B9B6C0/EF4D795A-95D4-40CC-9BFC-1C2DC871A6B6.png)

目前 MagicPods 已经能够实现大部分 AirPods 在 iOS 和 macOS 上的体验（唯一明显的功能缺失是不能在 AirPods 上实现两次敲击耳机暂停 / 播放），应用支持 AirPods 一代、二代以及 AirPods Pro。如果你需要在 Windows 上经常使用 AirPods，希望实现显示电量、摘下自动暂停这些功能，MagicPods 是一款值得一试的小工具。

你可以在 [Microsoft Store](https://www.microsoft.com/store/productId/9P6SKKFKSHKM) 里购买 MagicPods，售价 14 元人民币，软件提供全功能试用。
