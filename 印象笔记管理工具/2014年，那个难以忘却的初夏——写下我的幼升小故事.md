| |  | | --- | | 【写在正文前】  2014年夏季，嘿姆考入了上海市实验学校——传说中那个神秘的学校。  直到今天，我依然不同意别人口中所说的“聪明”2个字，我始终觉得，嘿姆是个很普通的孩子，他能被上实选中，不是因为智商高，而是他的学习习惯和学习热情。  其实，面对择校的孩子里，九成的孩子都是智商普通的孩子，我们不得不理智的承认：由我们这些普通的父母，普通的基因之下，诞生的孩子，也只能是普通的孩子。他们之中有些人进了名校，有些人没有，差别不在于先天智商，很大程度上在于后天的努力。  我想说，每个孩子都可以进名校。  【最重要的习惯是阅读】             曾经因为工作原因，有机会向儿童心理师讨教，关于如何建立习惯和秩序的问题。对方告诉了我一个数字“21”，说是21天的时间能够让孩子养成一个习惯——不管好的还是坏的。       我是文科生，最擅长的学科是语文，我深知这门学科要学得好，必需从小有阅读习惯才能日积月累。于是，我开始尝试那一个又一个的21天。       在嘿姆5、6个月大的时候，我买各种材质的书给他“玩”。真的是玩：捏、撕、扯、甚至啃！我相信他是爱它们的，不然为啥放那么多精力在上面？       过了周岁后，我们开始了亲子阅读。最初，我读得辛苦，他在边上各种摸爬滚打，完全不听；慢慢地，开始能被书上的图案吸引坐下来看一会儿或听那么一小段；再后来，终于愿意听完一整本了（其实也就5、6页）。       在他进幼儿园之后，开始变得非常热衷于听故事，总是要求我讲完一本又是一本。对我来说，最最悲催的是，当初为了吸引他听故事，我一直是采用分角色朗读的形式来变着声音读，效果确实不错，可是当他迷上《贝贝熊》系列的时候，我简直快疯了，一本念完已经喉咙发干了，他却经常不依不饶地期待我继续念第二本、第三本……恩，我很肯定，他是爱它们的，不然不会这么折腾他妈。       我们有了读书约定，每天仅限1本，如果妈妈读累了，他可以帮忙一起读（因为大量阅读和学了拼音，让他认识了不少字），或者我来负责旁白，他负责角色的部分。一起看书，就变成了睡前有意思的一段时光，虽然我常常需要加班，当总是能够赶上他睡前那十来分钟，一起念完故事，看着他满足地睡去。       期间，我做了我自己最自豪的事：那就是在每一本书上，他可能不认识的字上，我都标上了拼音。目的很简单，巩固拼音顺便认字。       中班开始了，他发现妈妈变得“懒惰”了，总是不愿意给他讲故事。终于，有一天，我告诉他：“你长大了，该轮到你给妈妈讲故事哄妈妈睡觉了。”他虽然有些不情愿，但还是给我讲了一次他熟悉的故事书。我用了他所能理解的所有赞美之词表扬他，那天，我看见他脸上带着成就感的笑容。我想，他是爱它们的，不然，他不会花那么多时间和口水，把它们念给他妈听，哄他妈睡觉。整个中班，他基本上把之前我给他念过的各种绘本都给我念了一遍，虽然经常念错字，跳字、漏字，但是念得越来越棒，也开始带个人感情阅读了，识字量的突飞猛进也是读故事书的附带衍生物。他也越来越享受阅读带来的快乐，也会挑选自己喜欢的类型来重点阅读。       到了大班，我又给他出了有意思的题目，和妈妈一起写自己的童话故事。我们两个有了属于自己的故事，虽然很简单，语言很平淡，三言两语，完全不精彩，但是他每一个奇怪的念头都让我觉得很惊喜。这么做并不是真的指望他成为一个幼儿级别的小作家，只是希望他能学会表达自己内心的想法，习惯和擅长表达是一种很重要的个人能力。我们一起编了几个童话故事，故事的主人公是一只叫“嘿姆”的小黄狗，他把自己的愿望，自己的挫折，自己的奇思妙想都加到了故事里，慢慢的，曾经老大难的看图说话也有了不小的进步。  【表扬他，是父母的义务】       朋友家养了两条训练有素的大狗，对人亲切、守规矩、懂秩序，出门从不会越过主人，切切实实是“遛狗”而不是“狗遛人”。我曾好奇她是怎么把狗狗教育的如此乖巧听话，她告诉我秘诀是“表扬”——你只要告诉它，什么是对的，做对了就使劲表扬，它为了那声表扬以后就只做对的，不做错的。       打个不恰当或者说很欠扁的比喻，其实孩子跟狗狗是一样的，他们都很单纯，也都很信任大人。       所以，我尽力去表扬嘿姆，他做对了，就夸，做错了就沉默，慢慢的，他能认真刷牙了，他能自己穿衣服了，他能到家及时先完成我布置的那些家庭作业了。他享受表扬，我享受表扬带来的良好效果，我们双赢。       但，表扬也不是完全无往不利的武器，无论如何表扬，他始终不爱吃饭，或者说吃不好饭，在吃饭问题上，我们的路还非常长，加油，嘿姆妈！  【请你让我做作业】       我的工作经常涉及到一些市场促销活动，目前，国内最容易让人引起购买欲的是什么促销活动呢？打折？直降？满减？还是买赠？都不是，是限量发售！特斯拉电动汽车在中国首发就10辆，70几万，瞬间抢没了。都是什么人买的？首批车主包括：新浪CEO曹国伟、汽车之家总裁李想、力帆足球俱乐部董事长尹喜地、合一资本董事长许亮、央视电视制作人张涵、云游控股董事会主席兼CEO汪东风、时代集团总裁王小兰、UC优视董事长兼CEO俞永福等。为啥这些名流会买？因为稀有，于是稀罕！饥饿营销，就是这么来的。       饥饿营销，同样适用孩子。       小班开始，我告诉嘿姆，你是幼儿园的学生了，要开始做作业了！我买了一系列适合他年龄的多湖辉游戏书，以每天只能做2页的要求，要求他每天放学回来先做“作业”。有趣的题目，鲜艳可爱的内页，所有解题都通过贴贴纸，涂颜色来完成，可想而知他有多喜欢。可是2页就是2页，每天就这点量，今天多做了，明天就少做或者没得做。所以，嘿姆很珍惜每天做作业的那一时刻，认真安静地完成作业，因为稀有，所以稀罕。       中班题目的量慢慢增加了，但是因为早就习惯了每天回家做作业的流程，他觉得做作业就是理所当然的，所以每天都能主动坐下来做作业。我也逐步开始让他尝试做需要用笔来解答的题目，尽量挑选有趣的类型，让他始终认为做作业是一件愉快的事，加上表扬逻辑贯彻到底，他对“作业”的认识还算美好。       到了大班，我忍不住还是功利了起来，把作业变成了百分之百用笔解答的类型，作业量也有所增加，除了开始时候抱怨了几次之外，他还是坚持下来了，持续了一阵子后，也就彻底变成习惯了。以至于考学结束后，他说：把这些书昨完吧，不然好浪费啊。  【计划赶不上变化】       啰嗦了很多，我知道大家最期待的是考学经历部分。嘿姆的考学经历说顺利其实很波折，说波折其实也蛮顺利的。顺利是因为他考了第一个学校，就直接录取了；波折是因为他妈选择困难症，反反复复彷徨犹豫纠纠结结——代替孩子做决定总是那么充满矛盾和恐慌。好在，虽然计划赶不上变化，但这个变化还是个好结局。       大班刚开始，我开始关注考学信息，参考了13年考学团经验，制定了考学计划：打算把浦东几个能考的民办都考一圈，如果时间不冲突，就再加个世外和上外打酱油，至于上实，因为它招生人数少，过程神秘，完全就没想过。按照计划，嘿姆会在3月考浦外，顺利的话，三月底拿到通知，然后淡定地面对4月中下旬平和的面试，再然后是五月初正达等学校的一波波面试海选。       可是，12月底，一条新政策让我晴天霹雳！每个孩子，只能在本区内选报两所民办学校，并统一在网上填写志愿，而且支援还有1、2区分，也就是说，对于很多名校而言，二志愿的孩子，很可能就不会考虑，甚至面试机会都未必会给。同时，也确定了，世外这种没有住宿条件的学校，我们这些外区的压根没资格报考，连酱油瓶都不给摸一下。       所以，我的计划迅速变了。把一志愿定为浦外附小，理由是浦外历年来看比正达题型容易，而且浦外还有晚托班，适合我们这种下班晚的父母。可是很快的，群里的妈妈告诉我，正达肯定比浦外好，不如浦外二志愿，正达一志愿。加上同事的女儿13年进了正达，常听她对学校赞美有加，加深了我对学校的好感度，慢慢滴，第一志愿真的变成了正达。       3月学而思的一场模拟面试，嘿姆成绩不错，但英语方面还是有扣分，而正达要考英语朗读这一点也让我打了退堂鼓，第一志愿又默默地变回了浦外附小只考跟读，不考朗读的学校。       3月底，平和开了介绍会，听得我心花怒放，狂长草，恨不得马上就把娃塞进学校。结果最后一刻，校长大人扔出一把快刀，直接把刚长出来的草连根一起断了——平和只招第一志愿，而且 面试还是抽签进行。这场博弈太大太残忍，果断放弃——不过事实证明，平和一志愿的今年都面了，所以，妈妈们还是需要有点勇气的，不要像我这样最后连志愿都不敢填。       浑浑噩噩又过了一个月，4月底的时候，还是在纠结浦外还是正达，跟着群里妈妈以酱油党的心态填报了上实，或许我的上实个人介绍是最短的，200出头的文字，大部分还是通过他学习情况来谈我们的教育理念。那点字足足花了我两个半小时，10177的报名号足以证明我大半夜不睡觉的努力了吧？或许是教育理念对胃口，也或许是觉得我这浓缩的是精华，反正网筛第一轮过了，让我很是欢喜——好歹我儿也有了去上实打酱油见世面的机会啊！       机考完，我自认是没有机会了，因为6种变化，他才做到D，做了多少题，自己也不记得。我默默地想，肯定没戏了，全力准备5月10日面试吧。又投入了到底是正达还是浦外的纠结之中——最后时刻，我还是决定正达了，理由有点诡异，因为正达校服好看太多，浦外校服居然是番茄颜色的运动服，我是外貌协会，一想到儿子要穿5年，接受无力，直接放弃。       5月6日跟着群里大部队在网上查上实机考分数，居然过了，原谅我用“居然”两个字，不是因为我不信任儿子的能力，而是我觉得我们压根就没想过要考这学校，意外！       意外跟着意外来，5月8日面试完，当天晚上一起面试的同学内部查分确认被淘汰了。因为二面那个孩子表现非常出色，回答问题的事后反馈看来是非常完美的表现，所有家长老师都觉得这娃应该能进上实，结果，意外来了，这样的娃都淘汰了！嘿姆没头没脑的几个问答，让我对他更不报希望了。我承认，5月8日那天，我失眠了。       5月9日，焦虑笼罩着我，直到9点半，报考页面刷新了，简单的“已通过”三个字，直接把我的眼泪逼出来了。坐在办公室里哭了几分钟后，才电话到处报喜。心情像什么呢？像做梦，像云霄飞车。那一天的那半个小时，是我人生中最最激动的半小时了。       虽然之后正达也向我们伸出了橄榄枝，但最终我们还是决定选择上实这条不平凡的路，因为徐校长在二面家长会上说的那句“取长容短”让我们很受鼓舞，坚定地选择了上实这条或许有些吃力有些辛苦的路。希望嘿姆能够理解爸爸妈妈替他做出的选择，也希望他能够在这条路上走的平稳、愉快。  【我们只是这一段的同行者】       走完了幼升小这一段路，回头看看，很多感慨，除了一开始说的“普通孩子也是能上名校”外，还有一句“即使上不了名校，只要坚持努力，也会变成优秀的人”。幼升小、小升初、中考、高考、考研、考博、申博士后，都不是人生的终点，每天，我们都要面对新的挑战，孩子也一样。或许今天是一道数学题，明天就是一个英语单词，后天变成了语文主谓宾句型改写。每一天，对他们来说都是新的，作为父母，我只想陪着他走这一段，帮助他更好地养成学习习惯，思考习惯，坚持的习惯，我只做这段路上的同行者，陪着他一起成长，一起坚持，直到有一天，放开手看他飞向远方。       我想对每一位妈妈爸爸说：不管你的孩子幼升小最终去向是哪里，他只要努力过，就是大赢家。请你，请你们一定要耐心地陪着他，继续走下去，人生还有那么多的机会，谁敢保证，他不会笑到最后？！加油！ | |
