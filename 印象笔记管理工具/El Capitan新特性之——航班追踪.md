# El Capitan新特性之——航班追踪

---

每一代 OS X 系统可能都隐藏着一些彩蛋，这些隐藏特性往往需要一定的时间才能为用户们所发现。有用户就在 OS X El Capitan 中发现了一个很实用的功能——航班追踪，有了这个功能用户就可以在 Mac 设备上查看航班状态。  
  
　在 El Capitan 中，航班号会被自动转换成可进一步操作的链接，当你点击这个链接并长按的时候，你就可以选择复制文本又或者是预览航班状态。

航班预览页面可以为你提供如航班状态（是否准时、提前、延迟或者是已降落）、离港时间、到达时间甚至是离港和达到航站楼的信息。除此之外它还可以显示航班路线的实时地图查看。  
  
　根据测试，航班号被转换成可用的链接需要一定的条件，该文本需包含国际航空运输协会（IATA）的前缀（例如 UA 代表美联航 United Airline）和航班号，一旦被识别 El Capitan 就会将其转换为可操作的链接。  
  
　航班追踪的功能适用于备忘录、邮件和信息应用中出现的航班号。
