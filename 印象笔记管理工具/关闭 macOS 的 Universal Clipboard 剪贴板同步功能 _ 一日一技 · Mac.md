# 关闭 macOS 的 Universal Clipboard 剪贴板同步功能 | 一日一技 · Mac

---

![](.evernote-content/CB488021-CCA5-4CBA-B001-C9B136BF64C0/420F3A5B-35D8-4A71-A8F0-CABF2FBE996E.jpg)

> Universal Clipboard 又称 Erwin Schrödinger's Clipboard，中文名薛定谔的剪贴板。
>
> *—— [@StackOverflowError](http://weibo.com/1765732340/EetJFeArC)*

Universal Clipboard（通用剪贴板）是 macOS Sierra 10.12 系统新增的一个功能，它能让你便捷地在 macOS 和 iOS 设备之间即时同步剪贴板内容，实现在这一台设备复制，另一台设备粘贴的无缝体验。

此前就有不少第三方 App 实现这个功能，比如 [Copied](http://sspai.com/tag/Copied)、[Command-C](http://sspai.com/tag/Command-C)、[Pasteasy](http://sspai.com/tag/Pasteasy) 等，但第三方 App 的体验并不算好，不论它们是通过 iCloud、蓝牙、局域网同步，都或多或少会在同步速度、配对稳定性方面有所不足。

苹果能让系统原生支持这一功能，对于经常有需求在设备间同步剪贴板的人来说，本是一件大好事。然而，从之前 10.12 Beta 测试版，直到现在 10.12.1 正式版更新，Universal Clipboard 的体验始终不够稳定，所以才有了叫它「薛定谔的剪贴板」的调侃。

诚然，有不少人表示 Universal Clipboard 用起来很顺畅、没有遇到问题，但也有一批人因为这个功能，系统出现了这样那样的问题，甚至影响到了正常的使用。当一个功能到了影响效率的时候，就有必要想解决办法了——在这里，最简单直接的，当然是关掉 Universal Clipboard。

虽然系统设置里没有直接的开关，允许我们关掉这个功能（相比之下，关掉「用 Apple Watch 解锁 Mac」就方便很多），但其实关闭方法也很简单。

经了解发现，Universal Clipboard 其实与 Handoff 功能是相连的，你只需要在系统偏好设置的「通用」菜单里去掉「允许在这台 Mac 和 iCloud 设备之间使用 Handoff」勾选，Universal Clipboard 就被关掉了。

![](.evernote-content/CB488021-CCA5-4CBA-B001-C9B136BF64C0/32444B96-175C-46EC-BACD-E3D6B2A6F1A7.jpg)

如你所见，这个方法带来的副作用就是，我们不能在设备间用 Handoff 了，但如果 Universal Clipboard 在你电脑上已经到了影响使用的情况时，这个方法总是帮你解决了问题。
