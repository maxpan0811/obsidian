提起国产 Windows 软件，你可能首先想到的是捆绑安装、弹窗广告、卸载残留等关键词。尽管一些所谓「大厂」的确致力于拉低业界的下限，但依然有开发者坚守底线，为改变整个生态圈而默默努力。今天，少数派就为大家带来了 10 款免费好用的国产软件，它们有的是个人作品，也有的已经走向正轨，希望能给你带来不一样的体验。

Snipaste
--------

作为 Windows 平台上最有特色的截图软件之一，Snipaste 自诞生之日起就受到了众多关注。除了自动检测边界及文本、形状、箭头、涂鸦、高亮、马赛克等强大标注功能外，Snipaste 还能让你将截图或剪贴板内容直接固定在屏幕上，方便随时查看。

![](.evernote-content/F8C4140F-FEA3-4F87-840B-65FC654AC56F/EB1AEE5C-CAD4-4F82-84B2-A84EFD7D870C.png)

  
Snipaste 由博士研究生 levie 历经三年开发，目前所有功能免费且无广告，同时提供商店版及绿色免安装版，macOS 及 Linux 版也正在路上，欢迎 [点此](https://zh.snipaste.com/ "点此") 下载体验。  
若想深入了解 Snipaste 的强大之处，你可以阅读少数派之前的详细评测文章：[Windows 上最好的免费截图标注工具：Snipaste](https://sspai.com/post/34962 "Windows 上最好的免费截图标注工具：Snipaste")  
此外，少数派也曾采访过 Snipaste 的开发者，你可以在此阅读：[幕后丨他做了最强免费「截图」工具 Snipaste 后，还有上万字的话想说](https://sspai.com/post/35097 "幕后丨他做了最强免费「截图」工具 Snipaste 后，还有上万字的话想说")

Listary
-------

Windows 文件资源管理器饱受诟病，其中一条「罪状」就是恼人的搜索功能，而 Listary 就是为了改变这一点而生。  
你可以在任意文件资源管理器窗口通过键盘输入直接调用 Listary 在本目录下进行搜索，并通过右键单击执行各种命令。

![](.evernote-content/F8C4140F-FEA3-4F87-840B-65FC654AC56F/FE83AC58-34AF-44EE-A5EA-99E85954C987.png)

  
Listary 还可以出现在文件选择对话框下，方便快速搜索指定路径，并快速切换目录。  
此外，你还可以用 Listary 搜索网页、打开应用等，当做快捷启动器毫无压力。  

![](.evernote-content/F8C4140F-FEA3-4F87-840B-65FC654AC56F/72D590D9-D4B4-40EF-927A-AFB4A0CF5D18.png)

  
Listary 由国人 Channing 开发，你可以在 [这里](http://www.listary.com/ "这里") 下载体验。免费版已囊括大部分功能，购买专业版可以解锁模糊搜索、项目管理、自定义动作等进阶功能，售价 19.95 美元。你也可以在数码荔枝以 59 元人民币的优惠价格购买。

Wox
---

如果你在寻找一款类似 Alfred 的快捷启动器，Wox 可以说是你的不二之选。它可以调用 Everything 进行快速全局搜索，也可以通过关键词唤起网络搜索引擎。你还可以安装各种插件，实现翻译、天气、命令行等进阶功能。

![](.evernote-content/F8C4140F-FEA3-4F87-840B-65FC654AC56F/C382CEAF-BEFA-49C9-ADA9-20D8F2BBBE88.png)

  
你可以在 GitHub 下载 Wox，应用完全免费。  
要想更全面了解 Wox，不妨阅读少数派的这篇文章：[Windows 上的 Alfred，免费开源的效率启动器：Wox](https://sspai.com/post/33460 "Windows 上的 Alfred，免费开源的效率启动器：Wox")  
除了 Wox 外，同样由国人开发的 [Rolan](https://sspai.com/post/www.irolan.com "Rolan") 也是快捷启动器佳选，少数派也发表过[介绍文章](https://sspai.com/post/41931 "介绍文章")，在这里就不多谈了。

WGestures
---------

除了键盘操作外，鼠标手势也是提高效率的重要手段。借助 WGestures，你可以在任何窗口通过鼠标右键或滚轮等绘制手势来执行各种命令，实现多任务处理、媒体控制、网页导航、命令行操作等功能。你也可以根据需要按应用设置手势或添加黑名单，防止误操作。

![](.evernote-content/F8C4140F-FEA3-4F87-840B-65FC654AC56F/A2A9F963-1AAF-4F49-89EF-639F50AF1331.png)

  
WGestures 由国人应元东开发，你可以在 [这里](http://www.yingdev.com/projects/wgestures "这里") 下载。他的其它一些作品包括模拟机械键盘声音的 Tickeys、轻量级词典 QingDict 和应用切换器增强 CmdTap 等。

Yu Writer
---------

相比 macOS 的百花齐放，在 Windows 上用 Markdown 进行写作一直没有太多选择，直到 Yu Writer 横空出世。你可以利用它自由写作，并在统一的文档库中进行管理，Yu Writer 会自动保存你的写作进度及操作记录，随时回滚至以前版本。  
写作完成之后，你可以将文章导出为 PDF、Docx、HTML 和微信公众号等多种格式，并选择主题配色。Yu Writer 的界面同样支持更换主题，功能区也可以自由定制。

![](.evernote-content/F8C4140F-FEA3-4F87-840B-65FC654AC56F/C264D298-E249-4E04-B0EF-156CC540033B.png)

  
你可以在 [官网](https://ivarptr.github.io/yu-writer.site/ "官网") 免费下载 Yu Writer，或者戳这里进一步了解→ [Yu Writer：漂亮好用的 Markdown 写作工具](https://sspai.com/post/40449 "Yu Writer：漂亮好用的 Markdown 写作工具")  
除了 Yu Writer 外，Windows 平台上还有[为知笔记](https://sspai.com/post/www.wiz.cn "为知笔记")和 [Leanote](https://leanote.com/ "Leanote") 等支持 Markdown 的国产笔记应用，欢迎大家下载体验。  
如果你只是需要一款更强大的文本编辑器，可以试试 [EverEdit](http://www.everedit.net/ "EverEdit") 或 [Notepad++](https://notepad-plus-plus.org/ "Notepad++")（台湾同胞开发）等软件，相信不会让你失望。

XMind
-----

如果你曾经做过思维导图，想必听说过 XMind 的大名。它可以帮助你实现自动布局、主题设计、演示批注、大纲查看等操作。你还可以利用它强大的编辑功能创造出鱼骨图、树状图、甘特图、矩阵等多种结构的思维导图，并存储为模板或导出为图片、PDF、HTML 等各种格式。  
如果你觉得 XMind 的界面太古板，不妨试一下其最近推出的 XMind ZEN，从底层全面焕新的它定会博得你的喜爱。

![](.evernote-content/F8C4140F-FEA3-4F87-840B-65FC654AC56F/E7C770A7-8E5E-42D0-9354-47941508F05F.png)

  
你可以在 [这里](https://www.xmind.cn/ "这里") 免费下载 XMind，应用同时提供增强版和专业版，售价分别为 559 元及 699 元人民币。XMind ZEN 目前处于公测阶段，可以免费使用，正式版价格未知。  
如果你对思维导图不甚了解，可以购买少数派出品的 [这样画思维导图最有用](https://sspai.com/series/12/list "这样画思维导图最有用") 付费教程，提高工作效率就靠它了。

iSlide
------

无论怎么努力都做不好 PPT？在彻底放弃之前不妨先试一下 iSlide 这款 PowerPoint 插件。它针对 PPT 设计过程中的痛点，提供了诸多贴心功能。  
你可以从它的诸多主题、配色、图标、图片库中寻找最适合自己的素材，并直接使用。你还可以利用它制作补间动画、创建智能图表、进行高效布局等，让 PPT 瞬间高大上起来。此外，iSlide 还提供了一键优化功能，帮助你统一文字间距、规范段落排列等。

![](.evernote-content/F8C4140F-FEA3-4F87-840B-65FC654AC56F/78DA5E3E-747B-4F67-9E84-F2A36CDCE64A.png)

  
你可以在 [这里](https://www.islide.cc/download "这里") 下载 iSlide，应用基础版免费，同时提供高级付费版本。  
深入了解 iSlide，当然还是看我派评测→ [谁都能在简单有趣的操作中完成惊艳的 PPT：iSlide](https://sspai.com/post/39708 "谁都能在简单有趣的操作中完成惊艳的 PPT：iSlide")

文电通 PDF 阅读器
-----------

对于文电通这个名字，你可能不太熟悉，但在海外市场，专注 PDF 领域二十余年的它早已打下一片江山，而文电通 PDF 阅读器就是它为个人用户准备的优质免费软件。你可以利用它，你可以为 PDF 文件添加注释、图章、文本和手写签名等，还可以直接发送至邮件、印象笔记和 Dropbox 等。此外，你还可以利用文电通旗下的 [加加 PDF](https://www.addpdf.cn/ "加加 PDF") 网站将 PDF 在线转换为其它格式，十分方便。

![](.evernote-content/F8C4140F-FEA3-4F87-840B-65FC654AC56F/B55AD7F3-F405-4265-852C-2B11D7113F6A.png)

  
你可以在文电通 [官网](http://pdf.gaaiho.com/index.php/zh-cn/ "官网") 免费下载其 PDF 阅读器。  
相较于文电通，你更有可能听说过福昕（Foxit），其在 PDF 领域确实实力更为雄厚，但我之前测试时发现其 PDF 阅读器捆绑了网页浏览器、导航广告等，因此决定将其评级下调一档。  
除了这两家之外，国内还有万兴科技深耕于 PDF 领域及海外市场，其主要产品有 [PDFelement](https://pdf.wondershare.cn/ "PDFelement") ，受篇幅所限，在这里就不细谈了。

Dism++
------

Windows 是个好系统，但很多操作未免过于复杂，如果有个软件能将各类常用优化清理功能集合在一起就好了——如果你曾这么想过，Dism++ 就是为你准备的。它基于微软官方系统修复工具 Dism 的底层技术 CBS 实现，同时提供系统更新修复、内存占用优化、热备份还原等功能。  
由于 Dism++ 大部分功能都基于 Windows 原生命令实现，Dism++ 主程序大小不过 1 MB 左右，同时完全免安装，解压即用，可以说是系统优化应用中的瑞士军刀。

![](.evernote-content/F8C4140F-FEA3-4F87-840B-65FC654AC56F/C7D9EBCC-11AE-44B4-A8D2-92382BA4DC49.png)

  
Dism++ 由初雨团队独立研发，你可以在其 [官网](https://www.chuyu.me/zh-Hans/ "官网") 免费下载。  
关联阅读：[Windows 下优化软件推荐—— Dism++ | 强迫症晚期患者福音](https://sspai.com/post/41992 "Windows 下优化软件推荐—— Dism++ | 强迫症晚期患者福音")
