# 我知道的胖东来

最近胖东来刷网，大家为胖东来鼓掌，一片叫好的原因基本是两个：①员工待遇高，一年休假多，这让很多在商业流通业上班族羡慕不已。②商场服务好，在胖东来购物，客户真正感觉自己是上帝。我作为在商业零售业从业者，在众人喝彩声中心下狐疑：胖东来真的那么好？

**盈利能力？人力成本？**

对于任何企业，服务意味着成本，尤其商业流通业，如果要把客户满意度提高到90%以上，估计企业要付出120%的努力。

于东来在某活动论坛上分享自己的经营理念时提到，上班时间现在是7个小时，想象将来会是6个小时；员工不想上班，想请假的话，不允许不批假，不让员工为了请假去找理由编理由；管理层一年190天年假，普通员工135天；在许昌，最基层员工工资6000+；还有5000块钱的委屈奖。没有一家商业企业能够做到，这样的休假，**企业的盈利能力能否覆盖这高昂的人力成本？**我不禁质疑，是不是两头榨取价值：一头压供货商的采购价格，转嫁自己的经营成本，另一头提高商品销售价格，割顾客的韭菜？

**调研与取经**

我带着上述疑问走访了业内人士，有的和胖东来的企业高管很熟悉，也多次去取经，也请胖东来的高管来自己企业分享，他们对胖东来的了解远比我们从媒体看到的多，他们是内行看门道。总结起来有这样一些特质：

**服务好**

其它家没有

**01**

胖东来的服务确实好，在胖东来购物，顾客能够感受到**其它家没有的服务**，这个媒体已经报道很多了。我不再展开细数。

**独具特色**

独特经营之道

**02**

胖东来的超市经营独具特色，业绩卓越，有自己独特经营之道。

**①****结合本地特点，选择电商不擅长的领域。产地直采、品牌直营。**胖东来超市70%产品都是自有品牌，这样保证质量也降低了经营成本。比如：馒头等熟食，本地人有到超市购买回家食用的习惯，胖东来抓住本地消费习惯把销量放大。如果在北方其它地区，顾客还是习惯回家自己做面食等熟食，只愿意在超市买定型包装食品。

**②****胖东来从来不卖假货。**胖东来是1995年建立，大家知道九十年代甚至上个世纪，那家超市不卖假货？胖东来在假货泛滥的时代不卖假货，树立了商业伦理门槛。

**③销售规模大。**胖东来因为销售产品品质好，服务好，不卖假货，这样建立很高品牌势能，周围的商超无法和他竞争，它的单店的销售规模是同样面积的同行的几倍，集采的优势就有了，供货商愿意把好产品用好的价格提供给胖东来，无形中提高了企业的毛利率，一般经营业绩好的超市的毛利率在13%-15%左右。胖东来估计要做到20%以上。这样的高毛利率可以轻松覆盖人力成本。这样可以保证高质量的服务。

**对员工好**

坚守企业文化

**03**

**胖东来确实对员工好**，这也是很多企业做不到的，员工有困难找到公司公司都能伸出援手。胖东来的员工休假是多，但是不休假的员工必须把休假员工都担当下来，一个要干两个或者三个人的活。不能出现岗位脱节。

据说胖东来收购一个超市的时候非常简单粗暴，先把员工工资大比例提高，然后提出要遵守胖东来的企业文化，如果你不同意或者不适应可以辞职，这些年也有不少的员工和高管辞职离开胖东来，这非常正常，**企业文化不能因为员工而改变。**

![](.evernote-content/AAF88A30-F788-433A-ADDB-FC8C5AD23355/491B67EA-5B94-49BF-8F35-317DE524F8AC.png)

**总结：坚守底线**

**满足员工自我价值实现**

胖东来的成功之道总结起来非常简单，胖东来从建立之初就**坚守自己的底线：不卖假货、顾客至上**。

在现在这个“内卷”时代，人人追求自我的时候，它满足了员工自我价值实现：上班可以多挣钱，一年可以多休假。**胖东来是盛和塾塾生企业，也是践行稻盛经营学的标杆企业，一家努力让员工实现物质和精神双幸福的优秀企业。**用高薪换来真心、真情和激情，增加了销售提高了服务质量。可谓：**攻心之上**。在经营方面做到了：**成本最小化，销售最大化**。

这些做法在外人看来不难，甚至有的企业也努力在做。但是要把这些文化和经营理念入脑、入心、入魂很难。胖东来做到了。所以中国只有一家“胖东来”。
