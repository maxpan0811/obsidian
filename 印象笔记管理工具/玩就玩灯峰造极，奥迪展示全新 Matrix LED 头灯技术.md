# 玩就玩灯峰造极，奥迪展示全新 Matrix LED 头灯技术

---

[![](.evernote-content/B859762B-7105-421B-BC3E-59A6895B05DB/A805073E-E967-4575-B393-2ABCA661D67C.jpg)](http://www.ifanr.com/485586/11-268)

奥迪近日公布了一则视频，展示了该公司全新 [Matrix LED](http://www.motorauthority.com/news/1031621_audi-demonstrates-effects-of-matrix-led-headlights-video) 头灯技术。

Matrix LED 正努力推动业界更好地治疗远光灯患者。该技术可以在对向来车时，自动调节远光灯，使其避开对向来车，但同时依然可以照亮对向来车的周围区域。简而言之，对向来车变成了远光灯盲点。

而结合导航系统的数据，Matrix LED 能够对弯道提前做出预判，并及时调整远光灯照射方向，使远光灯往弯道内侧方向偏移。该功能甚至能在司机转动方向盘前就发挥作用。

从视频来看，效果很不错。

[youku](http://v.youku.com/v_show/id_XODc2MzM2OTc2.html)
能实现视频里的效果，奥秘在于，汽车的每个头灯都由 25 个独立的 LED 灯泡构成，而这些 LED 灯泡都能根据实际场景单独受控制。那么这些 LED 灯如何调整的呢？车头安装了摄像头，用于捕获道路图像信息并反馈至计算机，然后由计算机给 LED 灯组发出指令。Matrix LED 当前只会在司机开启了远光灯并选择“自动调节”时发挥作用。

除了视频里演示的两项功能外，Matrix LED 还有一项贴心功能， 它可以与汽车的夜视辅助系统结合起来，当检测到前方有行人时，头灯会闪烁三次，以提醒行人注意，同时帮助司机看清行人。

[![](.evernote-content/B859762B-7105-421B-BC3E-59A6895B05DB/78772554-A726-403B-9B8F-407113577CE3.jpg)](http://www.ifanr.com/485586/22-119)

值得一提的是，这项技术并没有出现在美国市场的汽车上。原因在于，美国法规规定汽车远光灯和近光灯必须独立设置。而 Matrix LED 的智能远光灯实际上也干了近光灯的活。目前奥迪和其他一些研发类似技术的厂商，正努力游说相关方，以改变这些规定。

奔驰在 2015 款 CLS 车型上搭载的多光束 LED 头灯（MULTIBEAM LED）与 Matrix LED 有着类似的效果。前者的 LED 发光单元可以根据道路状况自动开闭，从而控制远光灯的照射区域、亮度等。同时奔驰头灯也会自动调整光照偏向弯道方向，最大偏转角度可达 12 度，方便司机更清晰地了解前方路况。
