　　相信从昨天开始，已经有一大波机友将自己的电脑升级到了 Windows 10 操作系统。对于一些自己电脑硬盘空间不多的用户，也提前做好了清理工作，目的就是为了迎接 Windows 10 的升级，比如删除 Windows 10 旧文件，清理Internet Explorer 临时文件，清空回收站，利用清理工具等等。但是有意思的是，当他们升级完成之后却发现，留给他们 C 盘的宝贵空间实际上更少了。  
  
　　其实他们发现，一个称之为 Windows.old 的文件才是造成硬盘空间不足的罪魁祸首。除非格式化硬盘重新安装 Windows 10 操作系统，否则直接升级的用户你都会看到系统 C 盘目录下多了 Windows.old 文件夹，那就是你的旧系统了，而且从属性查看大小之后，你可能还会发现此文件夹非常大，具体视之前使用的系统而异，不过小编这里达到了 53GB。更重要的是，你想直接按下 DELETE 键来删除，那是不可能的事情，因为系统默认保留就是为了你还能回滚到以前的版本。  
  

[![](.evernote-content/3BFFE917-3C0E-46EE-9526-AF5A5BC646B8/3F794EC3-C72A-4F4F-B48C-80332DA08623.jpg)](http://resource.feng.com/resource/h059/h01/img201507302126511.jpg "升级Win 10后如何删除旧Win 7和Win 8系统?")

  
　　那么，这时候该怎么做呢？小编来说一说最简单的方法，其实也就是简单的清理 C 盘空间：  
  
　　第一步很简单，也就是通过“磁盘清理”来完成，直接点击任务栏上的 Cortana 搜索栏，输入“磁盘清理”，你就看到有个“通过删除不需要的文件释放磁盘空间”的选项（如下图）。话说 Cortana 的确挺不错，支持搜索本地和互联网上任何内容，她还能检索你的应用程序和设置。  
  

[![](.evernote-content/3BFFE917-3C0E-46EE-9526-AF5A5BC646B8/058128B6-ABFC-4495-95C5-CFF900BC8D5C.jpg)](http://resource.feng.com/resource/h059/h01/img201507302126512.jpg "升级Win 10后如何删除旧Win 7和Win 8系统?")

  
　　点击“通过删除不需要的文件释放磁盘空间”之后会弹出个提示，要求你需要释放空间的磁盘，直接选 C 盘即可。其实进入到这一步，你也可以直接在“此电脑”中通过鼠标右键 C 盘，选择“属性”，再点击“磁盘清理”按钮。  
  

[![](.evernote-content/3BFFE917-3C0E-46EE-9526-AF5A5BC646B8/4C24F95F-104F-4E47-B53B-6B8570430DD1.jpg)](http://resource.feng.com/resource/h059/h01/img201507302126513.jpg "升级Win 10后如何删除旧Win 7和Win 8系统?")

  
　　此时你会看到一个磁盘清理的窗口，在“要删除的文件”中，你会看到各种混乱的数字，比如“Internet 临时文件”和“系统存档的 Windows 错误报告文件”等，我们先别关心这些。在整个窗口的下方，相信你也发现有一个要求系统管理员权限的按钮“清理系统文件”，我们继续点击这一项。  
  

[![](.evernote-content/3BFFE917-3C0E-46EE-9526-AF5A5BC646B8/5552A96E-0749-42E2-8B8D-A144312B2D73.jpg)](http://resource.feng.com/resource/h059/h01/img201507302126524.jpg "升级Win 10后如何删除旧Win 7和Win 8系统?")

  
　　这时候，“磁盘清理”会重新计算你的 C 盘可以释放多少空间，而且扫描过程中，你还会发现它把“以前的 Windows 安装”给计算在内了，而这正是我们要达到的目的。  
  

[![](.evernote-content/3BFFE917-3C0E-46EE-9526-AF5A5BC646B8/A1C082C5-C4A0-4F3D-9DB1-6749587520BC.jpg)](http://resource.feng.com/resource/h059/h01/img201507302126525.jpg "升级Win 10后如何删除旧Win 7和Win 8系统?")

  
　　最后，磁盘清理的窗口又回来了，不过多了不少可清理的项目，对于“以前的 Windows 安装”这一项，我们直接勾选，然后点击确定，系统会自动帮助我们删除掉了。  
  

[![](.evernote-content/3BFFE917-3C0E-46EE-9526-AF5A5BC646B8/8AF552F6-F85B-4115-8999-EBFE2167BA7A.jpg)](http://resource.feng.com/resource/h059/h01/img201507302126526.jpg "升级Win 10后如何删除旧Win 7和Win 8系统?")

  
　　删除之前安装的 Windows 7 或 Windows 8.1 就是这么简单。不过，请记住，一旦你选择删除 Windows.old 文件夹，那么将意味着你没有滚回之前系统版本的可能性。  
  
　　我们建议你先使用 Windows 10 一个月的时间，之后再做决定，就当做试用 Windows 10 体验全新的功能，包括 Cortana 小娜助手、Microsoft Edge、Continuum 模式和操作中心等更多，如果你真心无法真正上手，还是可以选择滚回一个月前你还在用的 Windows 系统。
