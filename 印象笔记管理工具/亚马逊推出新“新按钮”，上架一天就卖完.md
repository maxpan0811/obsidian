# 亚马逊推出新“新按钮”，上架一天就卖完

---

[爱范儿 亚马逊推出新“新按钮”，上架一天就卖完](http://www.ifanr.com/656636?utm_source=rss&utm_medium=rss&utm_campaign=)

![](.evernote-content/09D462F5-1DA8-46CA-8982-D01F5493AB68/B67A46DF-DF0E-4084-8CB5-6C6581088B44.jpg)

灯泡坏了，去污粉用完的时候，按一下[爱范儿曾介绍过的](http://www.ifanr.com/610596)这款亚马逊 Dash 按钮，它就能自动下单完成购买。

这是亚马逊进军“物联网”领域的第一步。现在，这个万物商店又打算用另外一个按钮 AWS IoT 来完成他们的“万物互联”大计。

![](.evernote-content/09D462F5-1DA8-46CA-8982-D01F5493AB68/AF1474F1-18A0-4927-AAA9-C09BE6738B88.jpg)

亚马逊 AWS IoT 按钮与之前的 Dash 按钮看上去差不多。不过，AWS IoT 除了能一键购买厕纸和去污粉之外，它还能用来控制所有接入的联网设备，比如启动你的音箱或者开关智能灯等。

另外，通过一定的设置后， AWS IoT 也可以与在线服务或社交网络对接，完成一键打车、一键分享 Twitter 的操作。有些类似实体版的 IFTTT 。

![](.evernote-content/09D462F5-1DA8-46CA-8982-D01F5493AB68/7A7A12CD-3CA8-49CB-8C06-6AA62BD7245F.jpg)

所以，有了这个 AWS IoT 按钮后，你可以：

> 按一下叫辆 Uber 回家，按一下智能门自动打开，按一下客厅灯亮了，按一下在 Netflix 上选部美剧，按一下一键叫了个披萨，按一下发条 Twitter……

不过你得节省着用。就像仙女只给你 3 个愿望一样，这个按钮只有 1000 次的使用机会。亚马逊称，按完 1000 多次，AWS IoT 就会没电，而且不能充电或换电池，只能重新购买。

2015 年亚马逊推出 Dash 按钮时，很多用户曾表示，希望 Dash 按钮不止适用于亚马逊特定的一些商品，而是能用来购买更多东西，甚至可以自定义功能。

现在 AWS IoT 就是这样一款产品，只不过很多功能需要开发者去设置或接入才能有上面提到的那些“一键完成”。

亚马逊 AWS IoT 按钮的售价为 19.95 美元（约合人民币 130 元），开售一天后现在商品页面已经显示的是“缺货”状态。不知道是不是真的那么受欢迎，还是就只拿出了几个先卖卖看。
