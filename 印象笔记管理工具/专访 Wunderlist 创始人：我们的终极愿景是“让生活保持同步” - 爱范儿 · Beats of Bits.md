专访 Wunderlist 创始人：我们的终极愿景是“让生活保持同步”
===================================

[![](.evernote-content/47EF2762-A3AD-4393-9177-0A2D4F93BF98/9C57A50E-87B1-4AE7-B340-D163A7736237.jpg)](http://cdn.ifanr.cn/wp-content/uploads/2014/07/ChristianReber.jpg)

今天，在全球拥有超过 800 万用户的事项管理应用 Wunderlist，正式在中国发布了全新版本 Wunderlist 3（奇妙清单 3）。该版本有全新的 UI 界面设计，重写了整个软件架构，带来了全面提速的实时同步以及全新的“大众清单”功能。

Wunderlist 这个产品来自德国初创企业 6Wunderkinder，Christian Reber 是这家公司的创始人兼 CEO。他 12 岁时就学会了七种编程语言，在 2010 年创建 6Wunderkinder 并持续推动其发展。正值 Wunderlist 3 发布之际，我们采访了 Christian Reber，以下为采访详细内容。

**ifanr：Wunderlist 3 准备了很长时间，是不是有什么困难耽误了进度？**

我们的目标是打造最好的产品，助您工作生活保持同步。这个产品要具备反应迅速、轻便易用、运行稳定的特点，能够让用户体验到每天使用的乐趣。

为了实现这一目标，我们需要重新设想整个服务器架构，定义一个既能实现实时同步、又能和老用户帐户兼容的系统，这需要重写所有的 API 和每一个用户信息。

此外，我们还重新设计了 UI 界面，增加了许多新功能。所以现在看到的奇妙清单 3 是一个经过完全重新编辑、设计而成的软件，是我们迄今为止最大的更新。截止目前，我们已在全球拥有 800 万用户、创建了 4.5 亿条清单。我们期待新版本的发布将带来更多的用户和市场使用。

**ifanr：你们对 Wunderlist 3 最满意的地方在哪里？**

借助奇妙清单 3，我们为全世界的清单创建了一个共享平台，你可以在这个平台上发现、探索他人的清单并从中得到启发、获取信息。长久以来，我们的用户都非常热衷于分享清单，希望能够有这样一个分享平台。所以我们对这次发布的 60 多项更新，尤其是新功能“大众清单”充满期待，也希望听到大家的反馈。

“大众清单”鼓励人们主动分享自己所热爱、了解的一切事物，去启发、教育、并影响身边的世界。无论是发微博推荐上海周边最佳避暑胜地，还是在豆瓣推荐“十大恐怖电影”，你都可以到奇妙清单——这个全球“清单加工厂”来撰写、管理、寻找你所需要的清单。清单发布后，你可以通过微博、微信、QQ、短信和朋友分享，或者将清单植入个人网站或博客，与世界分享。

**ifanr：Wunderlist 3 采用了全新的设计，这个版本有怎样的设计理念？**

从一开始，对奇妙清单来说，整体的设计就至关重要，关乎应用最后的成功。然而，设计对于我们来说绝不仅仅是应用界面的美化。我们的愿景是让用户无论身处何处，都能以最方便、最有趣的方式来撰写、管理任务清单。因此在设计奇妙清单 3 时，我们并没有急功近利，而是花时间精雕细琢，重新思考、设计和调试奇妙清单，让用户能够享受清单管理的简单乐趣。

首先，我们尝试以一种全新的视角审视奇妙清单，并大刀阔斧地删去了所有我们认为不属于核心体验的部分。这样的尝试让我们取得了一些突破性的改进。从奇妙清单的图标开始，从图标上的红丝带到星星，再到打开应用后的三折图层导航、手绘象形图标以及全新的背景图库，这些方面都有了明显的改进。

[![](.evernote-content/47EF2762-A3AD-4393-9177-0A2D4F93BF98/04F42A16-9993-47DB-B63F-F097CAD25A30.jpg)](http://cdn.ifanr.cn/wp-content/uploads/2014/08/unnamed1.jpg)

除去视觉效果，我们更深入思考那些用户最关注、对他们最重要的部分，比如易操作性、产品性能与反应速度。改进后的奇妙清单 3 能够满足用户的这些要求，他们将在使用过程中体验到前所未有的简单、快速和流畅。

奇妙清单 3 的 iOS 和 Android 操作系统上都引入了全新主页面，用户打开界面就能在屏幕上即刻看到所有的清单、动态与对话提醒。此外，我们还改进了清单分享的方式，让亲朋好友或者同事之间的分享交流更加便捷。总体来看，超过 60 项的更新与改进将为用户最终呈现一个崭新的奇妙清单 3。

新的设计理念让奇妙清单 3 更为快速、简洁，与此同时，我们的设计团队也投入了无数时间来巩固并改进奇妙清单的既有优势和特点。这样的设计理念将带领奇妙清单在不断更新完善其功能和版本的同时，始终保持它广受喜爱的极简特性。

**ifanr：GTD 工具领域竞争一直十分激烈，Wunderlist 如何让自己脱颖而出？你们又如何看待这个领域陆续不断的后来者？**

市场上有很多产品都只单为企业或个人的某一方提供服务。奇妙清单的与众不同之处在于它可以为每一个人提供服务。我们的目标是持续创造深受用户喜爱的产品，不仅要简洁、有格调、有特点，还要足够强大，能满足各类不同需求。

奇妙清单绝不仅仅是一个应用，更是一个能帮助人们高效管理生活的平台。借助这个平台，人们可以更为高效的利用自己的时间，创造更多价值。这就是我们优势。

奇妙清单致力于为用户提供一个操作简单、便捷，极易上手的平台。我们的设计团队夜以继日、投入大量精力来确保产品的直观简洁，任何人在任何设备上都可以轻易操作。

我们另一大优势在于全面覆盖的系统平台：奇妙清单 3 在各大平台同步发布，在iPhone、iPad、Mac、Android、Kindle Fire、Chrome OS 以及网页上均有合适版本可以使用，并即将推出可供 Windows 8 以及 Windows Phone 8 使用的版本。

[![](.evernote-content/47EF2762-A3AD-4393-9177-0A2D4F93BF98/C3D61F13-29FC-45EE-B26A-B43525EBD51A.jpg)](http://cdn.ifanr.cn/wp-content/uploads/2014/08/unnamed.jpg)

**ifanr：GTD 最重要的理念是什么？反映在 Wunderlist 产品上如何呈现？**

对于某些群体来说，GTD 的确是一种很实用的管理理念。然而在设计之初我们就发现，GTD 只有在使用者本身有强大的动力支撑，并且能够严格遵守这个方法来执行的情况下才能发挥作用。同时，GTD 这个概念本身也比较费解。这样的双重因素使得该方法在大众中难以得到推广。对于某些人来说，GTD 的确很好，但我们希望做的是一种更为大众化，能够为每个人带来高效和乐趣的产品。

不过，我们也从 GTD 中汲取了一些可以运用于奇妙清单的通用理念。最明显的就是奇妙清单中的收件箱。收件箱能够让用户随时记录下想做的事情。你不用在意自己在某刻的想法背景如何，或是是否连贯，只要将想法记录下来即可。同时，这样做还可以帮助大脑释放更多的记忆空间。这两点对于人们达到与生活保持同步来说是至关重要的。

**ifanr：Wunderlist 采用的商业模式是“增值付费”，目前公司的盈利状况如何？付费的比例有多少？**

奇妙清单致力于提供免费的产品。每个人都可以在任意设备上免费下载使用奇妙清单，包括 iPhone、iPad、Android、Mac、Windows 以及网页。只需每月支付 30 美金或每年 300 美金，就可以使用奇妙清单的其他高级功能，享受更好的团队管理服务。奇妙清单企业版专为企业用户开发，是高效管理团队的最佳版本，企业内的每个成员都能享受到高级账户的一切功能，同时管理层能够进行统一管理。

作为私企，我们不方便透露具体用户或者付费用户的数据。但我能说的是，目前，高级账户付费用户的比例相当可观。每天，全球各地都有数以万计的商业团体在使用奇妙清单。许多我们所敬仰钦佩的企业都在使用我们的产品来管理日常工作，比如 Apple、Cisco、PayPal 等等。

**ifanr：据我们所知，中国用户对 App 及服务的付费意愿并不是很强。你认为中国用户会为 Wunderlist 付费吗？有怎样的营收预期？有没有针对中国地区的特殊方案？**

中国市场对我们来说非常重要，奇妙清单在中国已经取得了很大的成功，我们也期望能够更加深入中国市场。

为此，我们决定专门聘请了一名中国市场经理管理中国市场，在德国柏林帮助我们制定本土化的中国市场战略，包括负责沟通本土合作伙伴、管理社交媒体等。

**ifanr：我们注意到 Wunderlist 3 增加了“大众清单”功能，Wunderlist 是否有打算做成一个平台级产品，就像 Evernote 那样？**

奇妙清单为全世界的清单创建了一个共享平台，你可以在这个平台上通过发现、探索他人的清单，从中获益。其实我们每天都在和各种清单打交道，购物清单、待办事项清单、愿望清单、电影清单或者音乐播放清单等等。

现在“大众清单”让分享成为可能，人们可以通过社交软件、博客、个人网站等各种途径，和全世界分享他们最爱的清单。这样，我们不仅创建了一个在线的清单信息库，更是提供了一个共享平台让世界各地的人们都可以从这些清单中寻找信息，汲取精华。

**ifanr：Wunderlist 的终极愿景是什么？你们希望它最后是什么样子的？**

奇妙清单的终极愿景是帮助用户保持生活同步，让所有人都可以通过电话、平板电脑或计算机和工作生活保持同步。

奇妙清单 3 遵循新的设计理念，实时更新的信息系统的创建以及新推出的大众清单让我们离这个终极目标更近一步。我们已经在帮助数以百万计的人们管理他们的待办事项，帮助数以万计的企业轻松而有趣地管理团队。现在，有了大众清单，我们则能创建一个强大的共享平台。
