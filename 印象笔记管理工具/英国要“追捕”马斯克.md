# 英国要“追捕”马斯克.......

英国威胁说他们要追捕和指控“全球网民”，甚至还要抓马斯克。

在英国对全国范围内的所谓“仇恨言论”进行打击之际，伦敦大都会警察局局长马克·罗利通过天空新闻台警告那些在网上“煽动仇恨的外国人”将会受到追捕指控，马斯克也不例外。

鞑英还以为自己是“日不落”呢，还警告“全球网民”……实际上，它治不了全世界的“反移民网民”，只能治本国的“网民”。

自由派圣经之一的《1984》，虚构了一个独裁的“大洋国“，监控所有民众的一言一行乃至于思想，书中有句名言，叫做“老大哥正在看着你”。

什么叫“老大哥正在看着你”？就是利用技术手段，监听你的电话，监视你的网络聊天，读取你的各种社交信息，然后看你到底是个什么人？如果你讲的话和鞑英帝国价值观不一致，那就要把你扼杀在萌芽之中。换个说法，这也叫——“文字狱”、“思想罪”！

![](.evernote-content/0CE141B4-0B08-416C-8F5D-8DC01D04A332/3B83990C-47B3-485D-ADF9-EE1C6239CB57.jpg)

高压统治、警察治国、掌控舆论、发动战争、制造苦难、洗脑人民，所有人都被它监视和奴役……自己造的孽，自己制造的难民，却甩锅给人民，使得底层互害、种族仇杀，什么叫做“自由即奴役，战争即和平，无知即力量”啊？

英国的烂事儿，怪不得白人右翼，也怪不得黑人和穆斯林移民，首先人家黑人和穆斯林没同意你跟着美国去发动战争，炸烂非洲中东伊拉克叙利亚阿富汗，制造那么多的“难民”……其次，英国本土老百姓也没同意你英国政府“引进难民”。

当年伊拉克、叙利亚战乱的时候。一船一船的难民涌入欧洲，那个躺在海滩上的小男孩的尸体，引发了无数欧洲白左的同情，他们要求法国政府、英国政府、德国政府尽可能地接受难民，后来证实，那个海滩上的小男孩，是他们“白头盔”组织的摆拍……

这在当年是一门“生意”，衰落的老欧洲需要劳动力，需要大量年轻、价格低廉的劳动力，欧洲本土人，享受高福利已经习惯了，这些人不愿意工作，这些人也没有制造业所需要的劳动技能和职业水平，所以，引进难民，成为欧洲补充劳动力的有效途径。

它们甚至有一整套接受难民的“产业链”，蛇头、当地的雇佣公司、房地产公司、公益组织，都会通力合作，他们建立安置点，按人头从政府领取补助预算，多报几个人，就可以多赚一笔钱，房地产开发商可以针对难民建立简易安置房，欧洲开发商还大言不惭的表达过，难民永远不嫌少。最后，羊毛出在羊身上，所有的费用，都会转嫁到难民身上，这些人不但要一家十几口挤在一个破房子里，还得以极为低廉的价格去当地的雇佣公司打工。这样，欧洲的上层精英们，就可以继续享受“贵族”生活了。

至于中东国家的难民们，给欧洲带来了强奸、谋杀、抢劫、治安混乱、以及当地底层民众的被剥夺和被伤害，欧洲的贵族老爷们管不着，反正火烧不到他们身上。

在这群管杀不管埋的官僚和贵族的“治理”下，难民和本土居民的冲突和仇恨终于愈演愈烈，完全不可收拾，英国政府是不管三七二十一，直接镇压。

![](.evernote-content/0CE141B4-0B08-416C-8F5D-8DC01D04A332/A3B057E1-E33A-4666-8AA8-C7AA1782B630.png)

面对问题，他们选择解决人，头疼医嘴，脚疼也医嘴。

鞑英帝国真的在疯狂抓人，“美丽的风景线”刚刚满十天，鞑英警方就抓了741，效率极高，很多人其实不是在冲突中抓的，而是“顺着网线”抓的，其中还有“破门抓人”的，就因为屋主发表了“与官方叙事不一致的言论”。

![](.evernote-content/0CE141B4-0B08-416C-8F5D-8DC01D04A332/BBBB5935-E756-41CC-A0E0-BCC69C3B80DB.jpg)

在英国，现在只要在网络上抱怨“移民政策”、“黑人”、“穆斯林”，就会被当做“煽动种族仇恨”抓起来，就像不能在美国网络上指责犹太人一样……甚至还有两个人因为在网络上发表“煽动性言论”被判刑了，一个判了20个月，另一个判了38个月——资产阶级铁拳不是闹着玩的。

![](.evernote-content/0CE141B4-0B08-416C-8F5D-8DC01D04A332/DC931404-942E-49AC-BC7D-6331EA435FB7.png)

![](.evernote-content/0CE141B4-0B08-416C-8F5D-8DC01D04A332/F68959D8-074E-4BD7-895F-D7065F26956F.png)

这个伦敦警察局长点名马斯克，是因为马斯克连发两条X，阴阳英国，说英国司法系统已经完全被黑人和穆斯林控制，还说如果“在英国发这张图，起码坐牢三个月”……一下子激怒了英国警方，于是威胁要“跨国制裁马斯克”。

![](.evernote-content/0CE141B4-0B08-416C-8F5D-8DC01D04A332/C4741340-80C0-453A-8513-604A51ACB055.jpg)

很多人活得短、见识浅薄、被现代资本主义洗脑严重，已经全然忘记了历史上的欧美帝国主义是个什么鸟样，忘记了他们是怎么镇压人民杀得血流成河的……更是坐井观天，不知道真正的“警察国家”是什么样子，“白色恐怖”对我们来说是书本上的历史，但是对于英美帝国主义来说，就是随时随地都存在的东西，人家都习惯了。

建议伦敦警察局长不要哔哔，直接去美国把马斯克抓了，毕竟他辱英太甚……不是不相信英国，就是想开开眼界，看看鞑英是怎么长臂管辖，跨国抓人的？

对了，我也反非法移民。
