作为滴答清单的重度用户，我一直致力于用它来实现个人任务笔记的 all-in-one（之前[文章中](https://sspai.com/post/51159)介绍过我自己的使用方法），但它相对不足的笔记功能一直让我觉得有点束手束脚。

而在上个月底，滴答清单上线了支持 Markdown 标记语言的编辑器！这着实让我激动不已。这一功能结合不久前加入的模版功能，让我能用滴答清单做的事情增加了许多，下面就给大家介绍一下功能更新以及一些我的使用场景。

功能更新：Markdown 和模板
-----------------

### 支持 Markdown 标记语言的「所见即所得」编辑器

Markdown 相信少数派的大家应该都很熟悉了，我就不多做介绍。

现在点击任务的详情，你会发现输入框底下就有了这些个操作按钮，支持标题、加粗斜体下划线、高亮、链接、无序、有序列表等主流的 Markdown 语法。

在移动端输入文字时，你可以点击相应的字体样式按钮或者直接输入相应的标记。这篇的文章大纲我就是在滴答里写作完成的，使用效果流畅、沉浸，很是不错。

![](.evernote-content/0CF71B17-7682-4702-8DB2-063DE4293017/B99395B6-FA23-41DD-9269-B85095B4A3CC.gif)

可选择的样式

截至目前，滴答清单全平台都可以使用 Markdown 功能（安卓端仍在审核中）。值得一提的是，电脑端和网页端还提供展开纸张的沉浸模式，可以帮你更好地进入书写状态。

![](.evernote-content/0CF71B17-7682-4702-8DB2-063DE4293017/7ABC32F6-8993-47AC-8B15-B0E50C54E03D.gif)

点击工具栏最左侧的矩形框，可以进入沉浸模式

目前滴答清单提供的 Markdown 功能和子任务是二选一的关系，一旦选择 Markdown 格式，当前任务就只能拿来作为笔记而不能再细分子任务（虽然有待办项，但无法添加时间，不是「任务」的用法）。此外，目前滴答清单的 Markdown 功能不支持表格和图文混排。

### 模版功能

在新建一个笔记时，点击右上角「...」即可保存模版；而在新建笔记时点击右上角的矩形框，即可选择调用已有模板。

![](.evernote-content/0CF71B17-7682-4702-8DB2-063DE4293017/2782B2C0-25DE-43A4-A7C6-169CC5390F03.gif)

新建笔记时调用模板

### 使用新功能的新场景

对我来说，之前的滴答清单笔记功能只能是作为随手记录和小型备忘的临时储存地，或者说偶尔写写文字记录想法的中转站，而有了 Markdown 和模板之后，我可以写出排版不错的文章，还可以随时调用这样的模板，因此将滴答清单用在了更多的使用场景中。

### 九宫格日记

我利用了待办项和分割线，做出了适合自己的九宫格日记模版。设置好提醒时间，在晚上睡前填完当天相关的内容就能完成一个九宫格日记了。

![](.evernote-content/0CF71B17-7682-4702-8DB2-063DE4293017/8E14A322-EE4B-479D-9126-5A88EDF9CA62.gif)

九宫格日记

### 影评模板

我只用滴答清单管理自己的追电影 / 剧清单，得益于看板模式，已经看完的电影和剧并不会「消失」，而可以在「已完成」中回顾。

支持 Markdown 功能后，我能用更美观丰富的样式记录剧评 / 影评。借助分享功能，你还能将完成的评论生成图片或者一键分享到豆瓣等平台。

![](.evernote-content/0CF71B17-7682-4702-8DB2-063DE4293017/CD7DAAF6-150D-40AA-AFEF-B60E5B88C79D.gif)

影评模板及分享

### 读书笔记

和电影一样，我也用滴答清单管理自己的阅读清单。在缺少 Markdown 格式时，我只能用普通的任务描述来做记录。

但支持 Markdown 语法后，我可以将章节要点用列表形式记录在每本书下，这样会让我对书中的观点更清晰。待办和无序都基本可以分解到 12 个左右，拿来用来做章节的重点记录等基本上也是够了的。

由于书籍结构没有规律可循，所以我制作的模板中只包括作者、书籍类目、大致内容。

打开滴答清单，配合番茄钟、进入沉浸模式、边阅读边记录，这绝对是最适合我个人的读书记录形式。

![](.evernote-content/0CF71B17-7682-4702-8DB2-063DE4293017/68EEEC7C-BE6A-451C-A9B7-781EC2664611.gif)

读书笔记

### 出行 / 搬家清单检查

每次出行都要打包一堆的东西，如果没有一个 checklist，很难做到不遗漏，而大多时候要带的物品都是重复的。

所以我制作了一个包含常用物品的出行清单模版。因为是可调用模板，可以在装箱前调用一次模板进行确认，再设置提醒在回程当日现场确认。

对于外出旅行来说，能有效避免在酒店遗漏东西。实在是居家旅行糊涂患者必备良品！

![](.evernote-content/0CF71B17-7682-4702-8DB2-063DE4293017/05A9603D-9106-4A0C-BF45-5D37F337519A.gif)

出行确认清单

结语
--

滴答清单此次的功能更新主要有两个优点。**第一就是彻底升级了任务详情文字单一的痛点**，比如无法排序文字和下层分类拆解等，替换成了 MD 模式方便了记录，甚至可以写文章。**第二就是结合模版功能，彻彻底底成为你私人的 「随笔记」**1 **。**不管是拿来记录或者检查，都能帮我更好地满足自己的生活场景。

当然前面提到的小问题也还需要改进，但毕竟滴答清单主要的功能还是待办，Markdown 只是辅助你更好地检查或者记录任务，一个待办软件能做成这样还要啥自行车啊。后期肯定也还会有改进，毕竟人本身就是笔记软件出身不是～

作为个人来说，我很满意滴答清单这次的更新，它作为**「备忘」**的功能更强了，更好地实现了我的 all-in-one 设想。所以，还不赶紧更新你的滴答，制作属于你自己的「随笔记」吗？
