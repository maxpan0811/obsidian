有人曾说：当习惯了 OS X ，就很难再回到 Windows。此话虽然过于绝对，但却也说明了两者间有着巨大的不同。

要我来说，所谓的「很难」与其说是因为两个系统间迥异的操作方式，不如说是因为 OS X 中许多贴心的细节设计，毕竟前者是可以花时间去习惯的，后者却是两家公司自身设计理念和文化底蕴所造就的，而这也正是苹果对于完美设计和极致体验的一贯追求。

本期精选了十项 OS X 中的细节案例展现给大家，也算是为不久后即将正式发布的 10.11 EI Capitan 做个预热吧。

### **1.**

![](.evernote-content/B84B63B4-FB5B-4435-82C4-E482E4A20743/DF1F67A1-664C-4756-91E4-D16D25F58749)

网络共享中的 PC 机图标是一台显示着经典 Windows 蓝屏的古董显示器。这一彩蛋存在了很久，不知道 Windows 10 的发布会不会让腹黑的苹果在未来的版本中去掉它。

### **2.**

![](.evernote-content/B84B63B4-FB5B-4435-82C4-E482E4A20743/DE95DC5A-7820-4EF0-A9A5-5CD1606E009C)

在 Finder 中对未知类型文件进行重命名时会默认选中「文件名+扩展名」，而对于已知类型文件进行重命名时只会默认选中「文件名」，这样做可以防止因误修改「扩展名」而导致文件不可用。

### **3.**

![](.evernote-content/B84B63B4-FB5B-4435-82C4-E482E4A20743/C4E93E17-7296-4147-BD36-2E9636765266)

在 OS X Yosemite「系统偏好设置」中的「节能器」图标由之前的老式节能灯改换成了顺应潮流的 LED 灯。

### **4.**

![](.evernote-content/B84B63B4-FB5B-4435-82C4-E482E4A20743/EF48C15F-38EA-4408-BEAD-ADCE525C26C8)

为了彰显 Retina 屏幕的优势，OS X 也对此做了相应的设计。比如系统英文输入法的星条旗图标在非 Retina 屏幕中只能看到三行星星，而在 Retina 屏幕上则可以看到六行星星。

### **5.**

![](.evernote-content/B84B63B4-FB5B-4435-82C4-E482E4A20743/576ABEE8-A311-4684-A180-143388685142)

Finder 会在列表视图模式中跟随「Date」栏的宽度自动调整日期的显示格式。

### **6.**

![](.evernote-content/B84B63B4-FB5B-4435-82C4-E482E4A20743/501BA337-1DA9-4E86-9133-55FB4ADB0249)

在「系统偏好设置」中搜索设置项时，会以聚光灯效果对搜索结果进行标识，匹配度越高亮度也越高。

### **7.**

![](.evernote-content/B84B63B4-FB5B-4435-82C4-E482E4A20743/FDFCD45A-309F-49D6-98E1-8E5E6D65FB85)

系统的窗口滚动条默认只在检测到触摸板的滚动操作时才会显示，但如果是外接鼠标且该设备又不支持触控操作，系统便会自动显示所有窗口中的滚动条。

### **8.**

![](.evernote-content/B84B63B4-FB5B-4435-82C4-E482E4A20743/B4B7897F-8B9F-4F91-9698-7EF5C5158FCF)

在任何应用的「帮助」菜单中进行搜索时，鼠标移入搜索结果时便会自动显示其在对应菜单中的位置。

### **9.**

![](.evernote-content/B84B63B4-FB5B-4435-82C4-E482E4A20743/58E42306-36DE-4501-BE78-588D8562CAB3)

使用 Dictation 进行语音输入时，系统会暂时让散热风扇停转以减少噪音干扰，待输入完毕后会再次恢复风扇运转。

### **10.**

![](.evernote-content/B84B63B4-FB5B-4435-82C4-E482E4A20743/E3D8395C-F74E-4C6B-ACD8-4253C286D4E8)

使用 QuickTime 录制 iPhone 屏幕时会默认清理状态栏杂项，并自动补满 WIFI 和电量的百分比；而最有情怀的是你会发现手机的时间变成了帮主发布初代 iPhone 的时间 —— 9:41 AM。
