今天主要说几点：

1. 欧美引领全球疫情2.0，比中日韩新严重许多，且仍未看见拐点。
2. 印度为代表的第三世界国家虽然表面数字还可以，但是实际上可能更加危险，也许会带来全球疫情3.0。
3. 全球经济损失可能非常严重，欧美未来政策会继续在抗疫和拯救经济之间摇摆，最后可能使得疫情和经济损失都更加严重。
4. 中国经济纵向比可能会非常、非常困难。但是横向比可能会非常、非常出色。

**1.    全球疫情2.0：欧美战场**

老规矩，上图。再回顾一下历史。

![](.evernote-content/D00A36E8-6842-45BF-A65F-F9620150D8BA/3A157FE2-4EF2-4688-9B90-139F17D33104.png)

这是[新冠观察（2）](http://mp.weixin.qq.com/s?__biz=MzU3OTUzNDYyNQ==&mid=2247484123&idx=1&sn=686de749ccb33c5b4687b39b2007d9bf&chksm=fd65e895ca126183b4dec2aea9b0957c89555f33ec5d875a943cb2e9cec4ce5e1c50e5479dd6&scene=21#wechat_redirect)里引用的WHO在3月17日的报告。其中每日新增最高的记录仍然是中国（绿色，源于2月中统计标准的变化），但是欧洲（橙色）在快速增长。

![](.evernote-content/D00A36E8-6842-45BF-A65F-F9620150D8BA/0995FC90-4F16-45D5-820C-2224179985E5.png)

这是[新冠观察（3）](http://mp.weixin.qq.com/s?__biz=MzU3OTUzNDYyNQ==&mid=2247484149&idx=1&sn=58ca157f7656e5b2271810b1434010f7&chksm=fd65e8bbca1261ad4a38755368c37edb5349a5201f00d51712f8148c7a9e16d01b5dcfee94ca&scene=21#wechat_redirect)里引用的WHO在3月20日的报告。其中欧洲每日新增已经超过了中国（深黄色，就是上图中的绿色+深黄色）最高的记录，而美国（浅黄色）的问题开始从数字上暴露（虽然大家都知道以前只是“不检测所以没有确诊”）。

![](.evernote-content/D00A36E8-6842-45BF-A65F-F9620150D8BA/0B60EE6E-6A98-4571-898E-AE08FEF6595D.png)

这是WHO在3月26日的报告。欧洲每日新增仍然在不断增长，但是在各国（包括英国）采取强度不一的隔离措施（这是我们在[新冠观察（2）](http://mp.weixin.qq.com/s?__biz=MzU3OTUzNDYyNQ==&mid=2247484123&idx=1&sn=686de749ccb33c5b4687b39b2007d9bf&chksm=fd65e895ca126183b4dec2aea9b0957c89555f33ec5d875a943cb2e9cec4ce5e1c50e5479dd6&scene=21#wechat_redirect)强调过的最重要最重要的政策措施）之后，不再是之前恐怖的指数增长（这里没有考虑瑞典等国家放弃检测轻症对数字准确性和完整性的影响）。

但是美国（浅黄色）的数字，开始爆发了。每日的新增超过了中国最高峰，且没有任何拐点出现的迹象。请注意这不代表疫情仅仅在这几天爆发，因为之前的检测数量实在是太少了。

![](.evernote-content/D00A36E8-6842-45BF-A65F-F9620150D8BA/7CFC8046-7A8B-43EC-A360-ECB3455F4C45.png)

WHO已经警告：美国可能成为全球新冠疫情新的震中。

![](.evernote-content/D00A36E8-6842-45BF-A65F-F9620150D8BA/73B45AA2-C341-4632-92BC-01BBDBDE18E6.png)

如果我们再看看每日新增的疫情图，有个很明显的发现，即**可以把全球至今的疫情分为两个阶段**：

* **全球疫情1.0**：新增确诊病人主要分布在中日韩新等东亚国家；
* **全球疫情2.0**：新增确诊病人主要分布在欧美发达国家，并且感染率（确诊人数占总人口的比例）远超中国。

这里多说一句，虽然大规模确诊病例是从中国开始，但是这并不代表病毒是从中国起源的。病毒起源这是一个需要科学家们去研究解决的问题。病毒到底是哪国起源的重要么？如果起源国没有像疯狗一样的乱咬人并且给他国带来巨大的损失，的确不太重要。

**新增病例，还只是问题的一部分。更大的问题在于死亡率和治愈率。**

[兔主席的这篇文章](https://mp.weixin.qq.com/s?__biz=MzA3MDIzODIwMg==&mid=2458089459&idx=1&sn=f6bd984e6bc9c5bd32582cb76b205ac4&scene=21#wechat_redirect)里面有详细的数据。在此借用一些关键数据。

|  |  |  |  |
| --- | --- | --- | --- |
| **主要指标** | **中国** | **欧盟** | **美国** |
| 每百万人口确诊总人数 | 56 | 546 | 227 |
| 死亡率（死亡人数 / 确诊总人数） | 4.0% | 6.1% | 1.4% |
| 治愈率（治愈人数 / 确诊总人数） | 91.1% | 11.6% | 13.1% |

可见欧美人口感染的比例已经远超中国。

**欧盟至今的死亡率（死亡人数 / 确诊总人数），都已经超过了中国。英法德意西这欧洲五大强国里，除了德国之外四国的死亡率都已经超过了中国。**

**更大的差距在治愈率**。中国的病人九成都已治愈，所以死亡率将来的增加会很有限。而欧美正好反过来，九成的病人尚未治愈（当然这跟确诊人数仍然在不断快速增长有关）。随着时间推移，随着现有病人不断去世，这个比例很可能还会持续增加。

我们在[新冠观察（2）](http://mp.weixin.qq.com/s?__biz=MzU3OTUzNDYyNQ==&mid=2247484123&idx=1&sn=686de749ccb33c5b4687b39b2007d9bf&chksm=fd65e895ca126183b4dec2aea9b0957c89555f33ec5d875a943cb2e9cec4ce5e1c50e5479dd6&scene=21#wechat_redirect)里估算过，2%的人口感染率是发达国家医疗资源被击穿的绝对上限。在意大利和西班牙，感染率仅仅是0.12-0.13%，就已经出现了**医疗资源挤兑的明显症状。这是纸面上人均医疗资源更好的欧盟死亡率超过中国最大的原因之一**。

作为对比，中国内地的感染率是0.0056%，湖北是0.11%，武汉是0.30%。可见武汉的封城是非常非常必要的。如果当时没有当机立断采取果断措施并且成功执行，我们没有可能集全国之力来援助湖北和武汉，我们更可能看到是全国皆武汉。

欧美的疫情将来会怎么发展？最高峰和拐点会在什么时候到来？现在仍然非常不清楚。

**最大的变数，我依旧认为，在于欧美在多快和多成功的执行隔离政策，从而降低感染率，避免医疗资源的挤兑**。

疫苗很难在2020年之内出现（一二三期临床实验以及半年一年安全性观察是很难避免的，不然不安全的疫苗可能会给民众带来更大的危害），而特效药是否会出现没有人能保证。

![](.evernote-content/D00A36E8-6842-45BF-A65F-F9620150D8BA/0CF570CC-C9EC-4F0D-946B-4567AEAFF4A7.png)

全球疫情2.0开始的另一个信号，就是中国终于开始无差别的对所有国外入境的人员进行14天的隔离了。这说明，在中国以外的这个世界，已经没有一个安全的、基本无毒的地方了。

但是从整个人类的角度来说，疫情2.0可能仍然不是最高点。

**2.    可能的全球疫情3.0：发展中国家**

关于世界人口和收入的分布，我们来复习一下《[世界秩序系列](http://mp.weixin.qq.com/s?__biz=MzU3OTUzNDYyNQ==&mid=2247483786&idx=1&sn=bf7c6e8df7f02d713dac2058e272c2ef&chksm=fd65ebc4ca1262d2f4de5d37296325abb23cedd69e2b978d623d10e7745a6fac0f348a44a59a&scene=21#wechat_redirect)》里的现行究极世界经济蛋糕分配图。

![](.evernote-content/D00A36E8-6842-45BF-A65F-F9620150D8BA/FA17BC1B-E22B-42DA-9CFE-8B35FA033698.png)

上图数据来源是世界银行。横轴是各国人口的比例，纵轴是2018年美元人均GDP的数值。

**这次疫情，在现有确诊数字上，似乎是富人病，因为影响的都是世界上最富裕（也就是图上最左边）的三个部分**：

* 美国；
* 其他高收入国家（西北欧、加澳新、中东石油国、以及日韩新）；
* 中国。

中国是全球疫情1.0，欧美是全球疫情2.0。**但是占世界人口绝大多数的其他中高收入国家（比如巴西、俄罗斯、墨西哥）、中低收入国家（比如印度）是更加危险和脆弱的地方**。因为那里的人均医疗资源，远远远远的低于中国和欧美。

在中国疫情最严重的时候，大家惊叹于恒河水锻炼出的逆天免疫力，比如九边这篇当时刷屏的文章《[面对疫情，印度为啥能躺赢](https://mp.weixin.qq.com/s?__biz=MzUzMjY0NDY4Ng==&mid=2247486646&idx=1&sn=7e38014dcdf75f2f95fe21ed7a67ed87&scene=21#wechat_redirect)？》。

但当时其实大家都知道，恒河水没有那么神奇。**印度没有多少确诊数字，那真的是因为没有数字，不是因为没有**。这里面的区别大家自行体会一下。

在送别了特朗普之后，最近印度毫无征兆的封国了，严厉程度紧追大家眼中最反动最落后最奇葩的北朝鲜。九边[最新的这篇文章](https://mp.weixin.qq.com/s?__biz=MzUzMjY0NDY4Ng==&mid=2247486916&idx=1&sn=fdde7fbcdefb1eb82da6b96ffb26ec7f&scene=21#wechat_redirect)里有更多的细节。这说明印度政府虽然对疫情的具体情况没有数，但是对自己国家是什么样还是有点B数的。作为参考，马来西亚最新的确诊人数已经有将近2000例。

即使是算作中高收入国家的巴西和墨西哥（话说，国内有多少人真把他们当中高收入国家看么？有多少人知道他们的美元人均GDP和中国其实是差不多的？），也情况堪忧。

在巴西，政府还不如黑帮。

![](.evernote-content/D00A36E8-6842-45BF-A65F-F9620150D8BA/700EAAE8-1C8E-4EBF-8D1B-9E61892F0AB9.jpg)

在墨西哥，我们都知道，政府的确打不过黑帮。

![](.evernote-content/D00A36E8-6842-45BF-A65F-F9620150D8BA/8D23264A-CA06-4D3D-94C6-05C02A7018AA.png)

在全球医疗条件最好、国家能力远强于巴西墨西哥印度等发展中国家的欧美，疫情2.0都如此猛烈，当发展中国家病毒开始蔓延的疫情3.0，最终会发展成什么样？

我们都不知道。但大概率不会很乐观。大概率会比现在欧美的疫情2.0的感染率和死亡率还差。

**3.    经济影响**

我们在[本系列第二篇](http://mp.weixin.qq.com/s?__biz=MzU3OTUzNDYyNQ==&mid=2247484123&idx=1&sn=686de749ccb33c5b4687b39b2007d9bf&chksm=fd65e895ca126183b4dec2aea9b0957c89555f33ec5d875a943cb2e9cec4ce5e1c50e5479dd6&scene=21#wechat_redirect)就说过，**这次新冠疫情的经济影响很可能是战争级别的**，即会影响GDP至少几个到几十个百分点的范围。

![](.evernote-content/D00A36E8-6842-45BF-A65F-F9620150D8BA/77005BC6-313C-453B-96E0-528355B48F86.png)

现在越来越多的经济分析人士开始认同这点了。比如伦敦政经学院的金刻羽教授[这篇访谈](https://mp.weixin.qq.com/s?__biz=MzI3ODYzMzc5NA==&mid=2247491461&idx=1&sn=59294afc1b91077dac454004ce0ef765&scene=21#wechat_redirect)里认为：

“全球面临的有可能是经济大萧条，而不是经济衰退，是1931年的情况，而不是2009年的情况。有可能面临的是大的国家40%产出的损失，而不是5%的产出的损失。”

而离“八熔八耻”只差三次的美股，也用真金白银在一定程度上表达了最真诚的预期。

![](.evernote-content/D00A36E8-6842-45BF-A65F-F9620150D8BA/24E9C51C-C999-4146-809C-E181483293D2.png)

全世界最知名的网红投资人达里奥同学，也认为新冠疫情终将改变世界秩序。

![](.evernote-content/D00A36E8-6842-45BF-A65F-F9620150D8BA/9A4351A4-6D08-4399-B615-BE767B60AC63.png)

PS说到这里我要再为我的《[剧变中的世界秩序](http://mp.weixin.qq.com/s?__biz=MzU3OTUzNDYyNQ==&mid=2247484101&idx=1&sn=85b650e748942fd176c0995ed88c9062&chksm=fd65e88bca12619d6944887d322bfc4835abb1196924af59fff8402c5251ab3e0ac8a9146c18&scene=21#wechat_redirect)》系列打call。本公众号于武汉封城第二天开始推送，现在持续连载中。

所以全世界各国都在用史无前例的货币和财政工具来拯救经济。

![](.evernote-content/D00A36E8-6842-45BF-A65F-F9620150D8BA/E77F97AE-0D5D-4627-8E8B-263A97E9D525.png)

这次疫情对经济影响巨大的原因在[前面章节](http://mp.weixin.qq.com/s?__biz=MzU3OTUzNDYyNQ==&mid=2247484123&idx=1&sn=686de749ccb33c5b4687b39b2007d9bf&chksm=fd65e895ca126183b4dec2aea9b0957c89555f33ec5d875a943cb2e9cec4ce5e1c50e5479dd6&scene=21#wechat_redirect)已经分析清楚了。最重要的因素，就是抗疫需要隔离。在人类尚未能够把所有经济活动远程化的今天，这就意味着极大规模的经济停摆。

这样的痛苦不是谁都能忍受的。所以特朗普开始吹风“14天内解除封城”，所以一开始英国选择躺倒策略，所以德克萨斯州副州长鼓吹经济比疫情重要以及老人应该自我牺牲。

就像我们在前面一再预测的：**欧美的抗疫政策一定会反复摇摆，在隔离的抗疫需要和民众的经济损伤之间反反复复**。这样的后果，可能不是理中客们预期的完美妥协，而是两边都实现不了的竹篮打水一场空。

在这个时候，中国在一开始付出巨大代价控制住了疫情，就显得那么宝贵。因为我们可以较为放心的回复经济活动。这与欧美的情况形成的巨大的对比。

有很多人正确的指出了，如果欧美的经济不好，中国也会受影响。**纵向与过去相比，或者与如果没有这次疫情相比，中国的经济一定会有相当大的损失。甚至2020年都有可能出现破天荒的GDP负增长**。

但是**横向来比，相比别的国家，中国的经济会是非常、非常的鹤立鸡群**。换句话说，如果中国一开始没有采取决然的措施，现在的情况，会差很多。

**一开始完全不考虑经济而只考虑生命的措施，到头来反而对经济最好**。正如最好的企业，都是一心为客户创造价值而不是把股东回报放在第一位的企业。这个世界，就是这么奇妙。

**在未来，我们可能都会过一段苦日子。一段我们这代人从未体会过的苦日子。一段我们这代人可能从未想象过的苦日子**。

可能会有很多企业倒闭，可能会有很多人失业。我们很难预计苦日子会持续多久，也许只是两三个月，也许会是两三年甚至更长。我们也很难预计最苦的时候会有多苦。

但是，我又想起了贸易战刚开始的时候，[本公众号第一篇文章](http://mp.weixin.qq.com/s?__biz=MzU3OTUzNDYyNQ==&mid=2247483655&idx=1&sn=ab4ae045d9a017231591e9aef6dc4d9a&chksm=fd65eb49ca12625f388cbb1b5d65d0c5d840c79ecf1e3b9baa99749d3c3d6abbdabc2e2d5810&scene=21#wechat_redirect)里的一段话：

“*二十世纪上半叶的美国发生了史上最严重的经济衰退：大萧条。大萧条的时候有多悲惨呢？GDP下跌了25%（差不多连续5年6%的GDP负增长），失业率达到了25%（相当于中国几亿人口的失业），股市下跌了90%（相当于上证300点而不是2700点）。*

*这个刺激不？劲爆不？嗨不嗨？*

*对比一下我们刚刚分析的中国经济的噩梦模式，相比之下太太太太太太温柔了。如果说现在的中国经济药丸，我们刚才分析的噩梦模式是地基价，那大萧条的美国就是永世不得翻身的三十六层地狱。另外要知道在大萧条前二十年的1907年，美国才经历过史上最恐怖之一的金融风暴和经济衰退。*

*然而，但是，不过，大家都知道，从大萧条开始不到二十年之后，美国就成为了无可争议的世界第一，且实际GDP总量是大萧条谷底的三倍还多。*

*是不是瞬间觉得未来没那么可怕了？是不是觉得又有了活下去的勇气？果然是没有比较就没有伤害。*”

历史当然不会简单重复，但是韵律都是相似的。

**所以，未来是光明的，但是道路是曲折的**。

* 欧美引领全球疫情2.0，比中日韩新严重许多，且仍未看见拐点。
* 印度为代表的第三世界国家虽然表面数字还可以，但是实际上可能更加危险，也许带来全球疫情3.0。
* 全球经济损失可能非常严重，欧美未来政策会继续在抗疫和拯救经济之间摇摆，最后可能使得疫情和经济损失都更加严重。
* 中国经济纵向比可能会非常、非常困难。但是横向比可能会非常、非常出色。
