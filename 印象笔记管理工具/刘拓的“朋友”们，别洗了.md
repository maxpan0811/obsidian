# 刘拓的“朋友”们，别洗了

刘拓的“朋友”们，别洗了
============

原创申鹏平原公子

收录于话题#刘拓2个内容

青年考古学者刘拓意外去世后，互联网上舆论遭遇了极大的反转，一方面是某些媒体大肆“悼念”，另一方面是普通网友劝他“下辈子投胎印度”。

如果了解了刘拓本人的思想和言论，你就会理解双方立场上的差异。

但是，很多所谓“刘拓”的朋友们就跳出来了，站在人性的角度怒骂网友，甚至给刘拓的言论洗地，说这些都是“污蔑”。

![](.evernote-content/390F540B-5C2E-4ED4-A77B-E6920FFD6C61/1B73B431-3480-4107-AAD1-FA4E26E9C696.png)

我想说，刘拓的“朋友”们，别洗了。

你们口口声声说“刘拓的言论是P图”，是“断章取义”，是“迫害”，那么你们解释解释下面这几张图呗，自己在微博公开发表的言论，这也是“仇家陷害”？“被仇家盗了号”？

![](.evernote-content/390F540B-5C2E-4ED4-A77B-E6920FFD6C61/224A3FCB-FEA6-40ED-8E31-EF46CCF11A77.jpg)

![](.evernote-content/390F540B-5C2E-4ED4-A77B-E6920FFD6C61/5984E1AD-FF80-4EE9-AAD1-41B226C93A3B.jpg)

刘拓本人都承认了“就是他说的”，你们比刘拓本人还懂他的思想？是的，刘拓死了，所以一切的罪过都可以推给“郑XX”，刘拓只是个“单纯的青年”……这不是“造谣式辟谣”么？

你们觉得网友们批评刘拓“不人性”，那么刘拓公然支持农奴制，希望西藏保留“原生态”就“人性”了？

讲实话，大家并不关心你们小圈子的“爱恨情仇”、“X咬X”，大家只知道刘拓确实说过那些话！互联网有记忆，微博历史圈都记得刘拓是个什么人。

有些人复读机一样反复说要考虑“刘拓在什么语境下说的话”，笑死我了，他都要把阿里地区划给印度了，他都妄图分裂国家了，我还要考虑他说话的“语境”？

你们“小圈子”在群里兴高采烈讨论围观沙特斩首的事情，被人截图批判了，你们不反思自己的冷血、麻木不仁、把别人的悲惨当做乐趣，却怒斥揭露你们的人“不道德”？这就是你们“自由知识分子”的风骨？

![](.evernote-content/390F540B-5C2E-4ED4-A77B-E6920FFD6C61/589B603B-F585-4682-8BE4-6A48942C1375.jpg)

你们口口声声吹嘘刘拓“完成了普通人一辈子都完成不了的事业”，说所有批评他的人都是“嫉妒”！好家伙，请问刘拓作为北大考古的研究生，除了穷游了中东，写了本“游记”（非学术论文），肆意妄为被伊拉克抓了还要动用国家资源去营救，他还有什么了不起的贡献？还有什么学术成就？

![](.evernote-content/390F540B-5C2E-4ED4-A77B-E6920FFD6C61/1E7C2214-7111-452F-B9D6-B6624F58F92C.jpg)

![](.evernote-content/390F540B-5C2E-4ED4-A77B-E6920FFD6C61/ED1E465B-B60F-4E9F-98A7-D0D2AA773263.jpg)

国家对他仁至义尽，该营救营救，全力保护他的安全，事情结束后甚至没有半点批评和怨言，对他自己作死只字未提，没有任何对不起他的地方……然而他回国后在社交网络上却大放厥词，否定西藏解放，支持“印度占有阿里地区”，眼里只有他本专业的“文物”，却没有国家统一、民族解放和西藏人民的生存权。

![](.evernote-content/390F540B-5C2E-4ED4-A77B-E6920FFD6C61/D590B5DD-8297-4848-9976-F61BE21C16D0.jpg)

![](.evernote-content/390F540B-5C2E-4ED4-A77B-E6920FFD6C61/1EDB2A39-2554-4188-AEEE-8CF77E1C0844.jpg)

其实我不懂你们强行为他洗什么？刘拓这路人在高校圈、学术圈可以说是“满坑满谷”，根本不稀奇，要不是这次某些媒体非要高调给他写悼词、唱赞歌，搞得全网哄堂大孝，咱们普通人才懒得掰扯他到底是个什么人。

他那个《阿富汗仿古行记》早就写清楚了他自己是个什么人，不尊重当地法律和文化，随意闯进村庄，随意拍照，不经允许摘村民的桑椹，被村民扭送警察局，在警察局不说话像一个巨婴一样大哭，最后警察手足无措放了他，他还自鸣得意；他还故意丢下司机去军事区拍照，置同伴安危于不顾……顺便说一句，他在伊拉克被抓，不是被XX组织抓的，而是被伊拉克政府抓的。

![](.evernote-content/390F540B-5C2E-4ED4-A77B-E6920FFD6C61/1B06559C-2A09-4E91-A959-07A7340DBAC7.jpg)

总的来说，他就是一个把自己的兴趣、爱好、专业凌驾于普通人的生存和尊严之上的小资产阶级知识分子，“我即是世界”。

他们最大的问题，是只在乎自己的感受，只在乎自己的利益，而没有把广大的人民当人，他们只在乎他们喜欢的“西藏原生态”，却没有问过当年在农奴制下西藏人民的感受。

你喜欢考古，喜欢文物，喜欢旧社会的西藏的艺术品，喜欢农奴制的西藏文化，喜欢人皮唐卡和阿姐鼓……问题来了，西藏人民是不是人？有没有人权？

这群人说白了就是“躲进小楼成一统”，把自己的世界当做全世界，把自己的感受当做全宇宙的感受。他只关心他的西藏农奴制下的“人皮唐卡”、“头骨法器”、“阿姐鼓”，一来他就喜欢这些玩意儿，二来他吃的就是这碗饭，所以他宁可国家不解放，民族不独立，人民不翻身，继续保留那个半封建半殖民地的“原生态”，这样他才能一辈子风花雪月搞他所热爱的“学问”，然后教出一大群弟子余孽，把他和他的理念，吹成“学术自由”、“大师之风”。

就如鲁迅先生说的：“他们活有余力，则以考古，但考古尚可，帮同保古就更可怕了。有些外人，很希望中国永是一个大古董以供他们的赏鉴，这虽然可恶，却还不奇，因为他们究竟是外人。而中国竟也有自己还不够，并且要率领了少年，赤子，共成一个大古董以供他们的赏鉴者，则真不知是生着怎样的心肝。”

所谓刘拓的“朋友”、“同学”，不要以为大家不了解你们学术圈那点“同门故旧”的小心思，不就是近亲繁殖、抱团取暖、互相吹捧、花花轿子人抬人吗？平日里傲慢、癫狂、自以为精英贵族，今天看到刘拓被揭穿了真面目，一个个兔死狐悲了是不是？

你们非要洗，只会让人觉得你们蛇鼠一窝、沆瀣一气，根子上烂透了，你们不但对国家民族没有认同，对社会主义道路没有认同，**你们甚至缺乏人类的正常情感，真把自己当做高高在上不食人间烟火的火星生物了。**

现在他死了，本来没有人会关心，但你们非要把他全网炒作，吹成献身考古、内心清澈的“圣人”……就不要怪舆论反弹了。

![](.evernote-content/390F540B-5C2E-4ED4-A77B-E6920FFD6C61/FFB09FD8-5DFE-477A-89A3-D4811F8CB430.jpg)
