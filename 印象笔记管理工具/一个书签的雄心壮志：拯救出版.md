# 一个书签的雄心壮志：拯救出版

---

[![](.evernote-content/C5439EBE-1445-4225-AC5F-A6A6F2F49DE4/1A953B52-53D7-4BCC-9825-CA73867986EC.jpg)](http://cdn.ifanr.cn/wp-content/uploads/2013/01/tumblr_m5bl2igIRF1qafc06o1_1280.jpg)

我一直相信这句老话，“你有一种思想（idea），我有一种思想，大家交换就各有两种思想。”在网络上，看到好文章，有趣的东西，我都会分享给朋友们看看。现在的分享动作也变得简单了，通过网络上的服务，可以一次性将一段话，一篇文章的网址分发到不同的社交网站中。

也由于社交网站的兴起，人们的阅读行为发生了变化——很大程度上，是否阅读一篇文章，不单单要看内容如何，标题如何，还要看分享出去的微博是否足够精彩，否则在信息洪流中，读者很难第一眼就判断出文章的价值。

既是程序员，又是作家的 [Paul Ford](http://www.ftrain.com/) 用 CoffeScript 开发了一个名为“[SavingPublishing](http://radar.oreilly.com/2012/12/save-publishing.html?utm_source=feedburner&amp;utm_medium=feed&amp;utm_campaign=Feed%3A+oreilly%2Fradar%2Fatom+%28O%27Reilly+Radar%29)”的书签工具，打算对现状做出小小的改变。所谓书签工具，英文是 Bookmarklet，直接拖到书签工具栏之后，直接点击便可使用的便捷工具。

SavingPublishing 的作用是对一整片文章进行分析，找出其中值得分享的片段，将字体变为红色高亮标识出来，然后我们就可以选择一句值得分享的话，分享出去。效果如下图：

[![](.evernote-content/C5439EBE-1445-4225-AC5F-A6A6F2F49DE4/B7BA4DB7-AD8F-4C41-9A7C-231DFFC56B7D.png)](http://cdn.ifanr.cn/wp-content/uploads/2013/01/Screen-Shot-2012-12-31-at-12.12.36-PM.png)

如果你直接点击高亮标识出来的一句话，就能够直接分享到 Twitter 上。由于是外国人开发的工具，所以出现这样的情况也在预料之内。

我在中文网页上启动了 SavingPublishing，发现它连中文文章都能分析，这就是意外之喜。我用 SavingPublishing 分析了好几篇中文文章，长短都有，发现它在一定程度上还能帮助我更快地抓住原文的观点——除了是帮助人们分享的工具以外，它还能帮助我们阅读。

[![](.evernote-content/C5439EBE-1445-4225-AC5F-A6A6F2F49DE4/E1F6BCD4-BEA3-4FC5-A86A-A0636168A9C9.jpg)](http://cdn.ifanr.cn/wp-content/uploads/2013/01/sp.jpg)

如果通过 SavingPublishing 就能把将文章中的精华句子分享到中文微薄服务中去，那就好了。

后来经过了解，发现 SavingPublishing 是一个开源项目，Forde 已经为它在 GitHub 上面创建了一个项目。如果中文开发者打算改造一下这个工具，打通它与中文微博服务的通道，善莫大焉。

未来，还会有多少帮助我们阅读，帮我们分享的工具出现呢？
