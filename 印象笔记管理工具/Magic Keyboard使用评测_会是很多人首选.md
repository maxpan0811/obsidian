# Magic Keyboard使用评测:会是很多人首选

---

[![](.evernote-content/475A1375-9A87-4C90-A0E2-13D13E5F3DFE/901296EC-CFD8-4F22-BC6B-F0D7FBBF914F.jpg)](http://www.feng.com/iPhone/news/2015-10-27/Magic-the-rid-device-info-keyboard-usage-review-would-be-a-lot-of-people-preferred_628340.shtml)

　日前，伴随着新款 iMac 的到来，苹果也给我们带来了新款配件——Magic Keyboard ，这款键盘的使用体验如何，我们来看看。  
  
　对于很多用户来说，键盘是他们输入数据的最主要工具，是人和设备交互的主要接触点，所以它应该是一款使用起来让人觉得容易上手，舒适轻便的工具，同时它也应该具有坚固稳定的结构。

![](.evernote-content/475A1375-9A87-4C90-A0E2-13D13E5F3DFE/901296EC-CFD8-4F22-BC6B-F0D7FBBF914F.jpg)

相比造型和设计，我通常更注重它的功能。我的大部分工作都是在 Mac mini 上完成的，使用的是有点旧的 IBM Type 102 键盘。毫不掩饰地说，我已经越来越喜欢这款键盘的触觉反馈，它能给我带来非常快速、精准的输入体验。

根据苹果的介绍，它采用的是剪刀脚键盘键帽设计所以我想这款键盘使用起来可能不会有什么特别好的感觉，但是真正使用过后它却让我改变了自己的想法。

**设计**

![](.evernote-content/475A1375-9A87-4C90-A0E2-13D13E5F3DFE/910FF1BE-C7DF-427D-8864-E4241E4DF719.jpg)

当把 Magic Keyboard 键盘从盒子里取出来的时候，我的头脑中闪过的第一个想法就是这东西怎么那么小，它的外观造型真的非常具有欺骗性。这款键盘长 10.98 英寸，高 4.52 英寸，但是剪刀脚键盘键帽下方设计有一X型支架，同键帽下方的硅胶碗共同对键帽起到支撑和固定作用，让我在真正使用的时候感觉键盘还是挺大的。

这款键盘符合我们一贯以来对苹果产品的认识，那就是它的质量确实是好到没得说，坚硬的塑料键帽，稳稳地“坐”在楔形的铝制外壳上。白色的塑料底能够有效减少键盘整体重量，而四角的四个橡胶垫脚则能够有效防止这个 0.51 磅重的键盘随意滑动。

键盘上有各种常见的功能键，方便用户进行快速的系统访问，以打开媒体、控制音量、调节屏幕亮度以及 Mission Control 等，Caps Lock下还藏着一个倍显孤独的状态灯。Magic Keyboard 的背部还有一个实体电源键、用于充电的 Lightning 接口，考虑到方便蓝牙信号的传输，铝制机身上还预留有一个小开口。

![](.evernote-content/475A1375-9A87-4C90-A0E2-13D13E5F3DFE/73002DB6-BA4E-441C-91FD-11ECA590CCD5.jpg)

如果你想要省电，或者延长电池续航，那么在不用的时候，你也可以把 Magic Keyboard 的电源关了。如果没有关闭电源键，同时又不在使用键盘，那么它便可智能感知闲置状态，进入低能耗待机模式，高效管理电量。根据苹果的介绍，这款 Magic Keyboard 键盘大约 2 小时就能完全充满，并可使用长达一个月左右。

一致性是苹果公司设计的主要原则之一，所以他们在设计这款键盘的同时也会考虑它和其他配件产品互相的视觉效果。Magic Keyboard 雪白的外壳可以说和 Magic Trackpad 2 等苹果配件风格一致，相得益彰。

**设置和使用**

如果用户是单独购买 Magic Keyboard，它的标配就是一条用于充电和配对的 Lightning 数据线。当用户使用这条数据线将设备连接到 Mac 上，OS X 就会弹出通知提示用户键盘已经可用，全程无需用户进行任何设置。如果你是那种不想去考虑各种 Mac 、蓝牙和键盘是否兼容的麻烦事的人，你想要的是“只要它能用”，那这款键盘简直就是为你而生的。

![](.evernote-content/475A1375-9A87-4C90-A0E2-13D13E5F3DFE/36AEADE6-6DBE-4E9E-A536-D476B4A2EBDC.jpg)

根据苹果的介绍，每一个键帽下都有一个剪刀机制，和以前的设计相比这种机制的稳定性更强，增加了 33%。实际使用过程中我们发现按键的平台确实坚若磐石，虽然这种设计本身会限制到按键的活动，一开始使用的时候我还有点不习惯，但是后来用得越久感觉就越自然了。

总的来说，按键的大小已经足够用户使用，打字体验还是很舒服的，不会感觉太狭窄，束手束脚。更重要的是我可以迅速而准确地打字，声音很小，触觉反馈却是实实在在的舒服。我用起来感觉不顺手的就是上下箭头键，和以前键盘上的按键相比，它们长度增加了大约一半，一时半会还没法习惯。

在 Mac 平台 Magic Keyboard 是一次增量更新，但是当这款新键盘和 iPad、iPhone 或者 Apple TV 匹配使用时，它那轻巧的机身和 Lightning 接口就显得尤为重要，特别适合旅行时使用。如今我出门在外都会选择使用 iPhone 6s Plus 来作为首要的打字和数据输入工具，虽然和笔记本还有一定的距离，但是 iPhone 6s Plus 和 Magic Keyboard 的组合也可以说是极好的。

这款键盘的质量在一般情况下可正常使用几年，如果你外出的话它还可以化身便携键盘。而且因为它支持 Ligitning 充电，所以你出门的时候就只要多带上这个键盘就行了。

![](.evernote-content/475A1375-9A87-4C90-A0E2-13D13E5F3DFE/B9F8F1A5-0438-40E6-AA1C-8A285DDC29D6.jpg)

**结论**

Magic Keyboard 是一个不错的增量更新，不过对于桌面用户来说它并非必买品，特别是现在已经有旧款苹果无线键盘的用户，你们可以不用太过纠结（Magic Keyboard 键盘的售价为 728 元）。对于马路勇士、空中飞人或者 iOS 深度用户来说，这个键盘不管是设计和质量，还有它的本身的便携性和内置电池，都值得你考虑。

我个人实在是太习惯于使用旧款的机械键盘了，所以现在还没有办法全天候使用 Magic Keyboard 键盘。不过我倒是很喜欢将它与 iOS 设备搭配使用。
