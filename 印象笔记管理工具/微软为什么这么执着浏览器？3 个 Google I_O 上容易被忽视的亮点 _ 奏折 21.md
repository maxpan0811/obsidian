### 微软为什么这么执着于浏览器

**@沨沄极客：**在本周的微软 Build 2019 大会上，微软正式发布了这款基于 Chromium 的 Edge 浏览器。

在 PC 浏览器领域，Chrome 是毫无疑问的王者，占据了超过半数的市场份额。[在 StatCounter 最新的统计数据](http://gs.statcounter.com/)中，Chrome 一直都稳居榜首，并远远超出 FireFox 和 Safari。微软的 Edge 和 IE 都不及 Chrome 的十分之一。

![](.evernote-content/504ABFEE-D291-4BB1-AC32-1A5D5731FD12/FCB8D6B0-2745-4FC9-AD58-EA56CDD5B761.png)

2019 年 5 月桌面浏览器市场份额数据

为何在这种局面下，微软不惜采用竞争对手的 Chromium 核心，也要让自家的 Edge 重新打开浏览器市场？微软为什么这么执着浏览器？

#### 微软浏览器面临最低谷

微软曾经是行业标准的制定者，它的 IE 打败了网景，并在 1996 年到 2004 年间凭借捆绑在 Windows 的特性迅速占据市场。曾经的 IE 也如同今天的 Chrome 一样为多数用户接受。

尽管如此，IE 的很多特性并不受大家待见，比如 JS 性能糟糕、响应速度慢、不遵守规范、漏洞众多。国内用户对前些年的银行安全控件、支付宝安全控件之类的 IE 插件应该并不陌生，不仅只能在 IE 上使用，本身的安全性也不高。

在表现糟糕的 IE 6、7 两代口碑崩坏之际，Chrome 异军突起，IE 的市场份额应声下跌。其实微软并非没有危机意识，它们推出了 IE 9 作为弥补，但没有解决根本上的问题，反而由于需要适配更多的 IE 版本让开发人员苦不堪言。

![](.evernote-content/504ABFEE-D291-4BB1-AC32-1A5D5731FD12/89FA21FD-18BD-424D-AB7A-0645EF2ECE67.png)

十年间桌面浏览器市场份额折线图

IE 的市场份额在 2012 年间被 Chrome 一举攻下，此后推出的 IE 10 和 IE 11 更新了标准和界面，但由于强迫升级、固有印象、Bug 依旧等问题，IE 的颓势已经难以改变。

外媒 ComputerWorld 在一篇评论《[How Microsoft lost the web](https://www.computerworld.com/article/3330239/how-microsoft-lost-the-web.html)》中给出了一系列理由说明为什么微软失去了竞争力：微软从未跟上市场节奏、IE 强迫升级、微软停止更新 IE 功能、Edge 仅提供给 Win 10、移动端体验极差…… 微软犯的每一个错误几乎都是 Chrome 的优点。

摆在微软面前的是浏览器市场的最低谷。

#### Web 才是未来

随着 5G 网络的出现，更多的功能会被 Web 服务所取代。早年一些受限于网速的设想已经在逐步实现了，比如[乔布斯在十年前就希望在手机上做纯 Web 应用](http://techland.time.com/2014/02/14/5-biggest-flip-flops-from-apple/)，而不是做一个 App Store。如今的小程序、PWA 应用已经显示出这种倾向。

微软在自己的研究中发现，[用户将 70% 的时间花在 Web 浏览器上](https://mashable.com/2015/07/29/microsoft-edge-preview/)。在这次 Build 2019 大会中，微软的重心也在 AI、Azure、云计算上，而浏览器正是这些 Web 服务的入口。只有掌握入口的人才有发言权，这是一块不可放弃的阵地。

2015 年，微软推出了 IE 的进化版 Edge。为了让浏览器更具竞争力，微软想了很多办法改进 Edge：Windows 10 内置、着重改善兼容性、推出 UWP 版插件、加入应用市场。但由于且平台受限，如今 Edge 和 IE 的市场份额加起来仅有 5%，依然反响平平。

举一个例子，Edge 早期仅提供 Win 10 版，Win 7、8 都无法使用，而且至今都没有 macOS 客户端。而 Chrome 早已实现了包括 macOS、Linux 在内的全平台客户端。这让微软很难在短时间内赶上 Chrome。

可以说，**在快速迭代的 Chrome 面前，Edge 并没有实际竞争力**，仅凭目前的 Edge，是无法在浏览器市场站住脚跟的。继续与 Chrome 竞争也无利可图。

相信微软自己也迫切的希望改变现状，**想在短期内实现重返市场的目标，兼容性强又是开源的 Chromium 内核自然是首选。**

#### 可能出现的改变

在 Edge 宣布改用 Chromium 内核后，一定会做出许多改变。

首先是利用 Chromium 的跨平台特性，把 Edge 带到其他平台上。Edge 官网的**即将发布 Win 7、Win 8、macOS 版本**印证了这一猜想。

![](.evernote-content/504ABFEE-D291-4BB1-AC32-1A5D5731FD12/357E1B01-A76F-4C62-A746-8A0B6D581F9F.png)

即将推出多平台客户端

接着微软会将 Edge 本身的特性带到新版本中，比如 Cortana、对 IE 的高兼容性。未来还有可能在省电方面做出改进和优化，以此和其他采用 Chromium 核心的浏览器拉开差距，保持自身的竞争力。

这项改变对国内用户而言也有着重大意义。新版本可能会带来诸如「书签同步、插件同步、设置同步」等一系列云端同步功能，这些功能会通过微软账号实现，国内用户使用的障碍也会更少。

多数人使用 Chrome 的一大障碍是无法直接通过 Chrome 应用商店下载插件。**如今可能会获得一个由微软运营的插件商店**（可能是 Microsoft Store）。由于和 Chrome 内核相同，所以原有的大量 Chrome 插件很有可能做一些简单修改就加入市场，让更多人用上插件。

![](.evernote-content/504ABFEE-D291-4BB1-AC32-1A5D5731FD12/E4186A84-D26A-4F52-9E7B-42C7DFCF6254.png)

换上 Chromium 内核的 Edge

可能大多数人仍然会像我一样继续使用 Chrome，但 Edge 如果能做到更省电、更省内存、又兼容所有 Chrome 插件，那也是广大用户乐于看到的一件事。

#### 微软仍需警惕

尽管微软在官网宣称，新一代的 Edge 是和谷歌团队密切合作一同打造的，公司之间并不是死对头的关系。但相信双方有着各自的考虑。即便采用相同的内核，Edge 和 Chrome 的使用体验仍然会有区别，依然是竞争对手关系。**谷歌欢迎微软使用自家内核，并不代表谷歌就会心甘情愿把 Chrome 的市场份额拿出来和微软共享。**

谷歌并非慈善家，它曾经[在一些网页搜索结果中疑似给 Firefox 使绊子](https://www.oschina.net/news/97898/firefox-battle-with-google)，被人解读为暗中打压 Firefox 浏览器。也很难保证谷歌不会在 Edge 崛起后翻脸不认人。

在微软去年年底宣布采用 Chromium 内核时，Firefox 背后的非盈利组织 Mozilla 脸色颇为难看，曾发表声明《[Goodbye, EdgeHTML](https://blog.mozilla.org/blog/2018/12/06/goodbye-edge/)》称微软把控制权交给了谷歌，Mozilla 认为「将网络世界的基础控制权交给一家公司是非常糟糕的」，它们会坚持抵制浏览器垄断。

![](.evernote-content/504ABFEE-D291-4BB1-AC32-1A5D5731FD12/6280650B-B329-4B54-AF3F-73C441726DD0.png)

Goodbye, EdgeHTML

不仅是垄断问题，Windows 这种量级的操作系统中，内置的浏览器采用了由竞争对手主导的开源核心。如果将来双方出现利益冲突，微软会陷入相当被动的境地。当然，现在说这些还为时尚早，但并不代表未来不会出现这种情况，这是微软需要警惕的。

不管怎么说，从用户的角度来看，这次微软是实实在在地做了件好事，切实地改进了浏览器体验。**终于让自家浏览器走上快车道，和其他浏览器同台竞技。**

在前几次奏折中，我们就给出过 Edge 的泄露版本和官方版本，你可以通过 [Edge 浏览器官网下载地址](https://www.microsoftedgeinsider.com/en-us/download/) 下载试用。

### 3 个 Google I/O 2019 上容易被忽视的亮点

**@Fairyex：**谷歌 2019 年的 I/O 大会如期而至，同往年一样，谷歌会在大会上宣布自己在软件上的最新突破，发布最新硬件，一般还会随着新 Android 系统特性的放出。今年的大会也不例外，少数派首页文章[「Google I/O 2019 开幕，这是首日发布会的四大亮点」](https://sspai.com/post/54601)进行了详细的介绍。

最近几年的 I/O 大会，谷歌都专注于将人工智能更好的应用在生活当中。今年谷歌的目标是「**将数字信息与现实世界结合起来**」，这意味着会有更多的指导型技术被公布，也就是一些我们现在还不能用上，未来会给所有科技公司起发现方向指导的技术。这些技术没有酷炫的演示，但是却能更深入影响到我们的生活，这也是媒体报道 I/O 大会时经常忽略的部分。

#### 1. 用自然语言与语音助手对话

虽然现在所有语音助手都宣称能够使用自然语言对话，其实绝大部分时候并不是真正的自然语言：比如我们一般会问家里人说「旅游的航班什么时候起飞呀？」而不是「查询从北京到上海的 XXXX 航班起飞时间」。这次大会谷歌就宣布了给 Google Assistant 加上 Picks for you 功能，实现真正自然语言对话。

简单来讲就是用户**主动同意**后，Google Assistant 将会通过各种应用（如邮件，日历等）提取用户相关的信息，当用户询问语音助手却没有给出信息时，Google Assistant 就会自动调用这些信息。**而且用户可以随时停止提供信息给语音助手。**

```
展示小明的图片；导航去上星期预定的菜馆；提醒我周年庆的时候买巧克力；
```

这些句子比起公式化的查询语句，使我们用起语音助手来更加惬意，也能令语音助手更加自然地融入我们的生活。

![](.evernote-content/504ABFEE-D291-4BB1-AC32-1A5D5731FD12/340AC37F-87A1-4E3B-BE70-C4D06EB027A1.png)

Picks for you 功能演示

#### 2. 数个新的保护隐私功能

说实在话，谷歌原本就是靠用户信息画像投放广告赚钱的。但是最近几年谷歌对用户个人信息隐私越来越重视，单这次大会首日就有一小半的内容和隐私相关。

首先是谷歌自家所有应用都会加上隐私设置快捷到达功能，长按用户上传的图片或者链接弹出的选项中就包括隐私设置。除去访问原有设置外，还新增自动删除隐私信息的功能，**可以自己手动确定相关数据保留多久**（3，18 个月，永久），比如地图上设置的是搜索历史、地理位置信息、路线信息等，不同应用可以单独设置。

更直接的是谷歌将浏览器的隐身模式带到了地图，打开后将不会保存用户的地理位置，导航历史，地点搜索等地理位置相关信息。今年还会将这个「隐身」功能扩展到 Youtube，搜索等应用。

![](.evernote-content/504ABFEE-D291-4BB1-AC32-1A5D5731FD12/8ED85A8C-7AAF-4851-AC9F-67E5C665A079.png)

地图隐私模式演示

在 Android 系统上谷歌也对隐私保护进行了改进。在新的 Android Q 系统中，隐私设置被放到了设置里的一级菜单；应用后台访问地理位置信息时会有一个很大很明显的提醒，点开就可以轻松查看设置所有应用的位置信息历史与权限；最后谷歌表示所有的隐私信息将尽量保存在本地，相关的数据处理和计算也尽量在本地进行。

宁可影响自己的收益也要保护用户的权益，这一举动证明了谷歌对用户信息的重视。

#### 3. 机器学习加持的幕后技术

前台演员的表演，往往需要无数个幕后演员的默默工作。谷歌今年也发布了几种幕后技术，可以以我们感受不到的方式让服务更加好用。

Live Relay 让有语言障碍的残疾人在任何时候将文字变为语音音频流，包括拨打电话的时候与对方通话；而 Live Captions 则反过来，可以将手机上的任何语音音频流变成实时字幕，就像我们在 Youtube 上看到的实时 CC 字幕一样。这两个原本的无障碍功能其实对于普通人来讲也是非常实用的，想象一下以后我们视频通话的时候也能有字幕了，视频也能在静音的情况下打字「通话」。

![](.evernote-content/504ABFEE-D291-4BB1-AC32-1A5D5731FD12/174B13E8-FD39-461C-875B-A33F3B3C35F6.png)

Live Captions 功能演示

另外，在机器学习火遍全世界时谷歌却开始考虑人工智能带来的隐患了。就像这次大会谷歌公布的 TCAV（使用概念激活向量进行测试）技术就是用来防止机器学习强化现实世界里已经存在的偏见与歧视。

机器学习本身不知道什么是偏见，你给它什么训练样本就能得出什么样的算法。但人提供的训练样本却有可能存在偏见，如果我们在训练识别性别算法的时候把男性图片全部换成亚洲人，最后机器很大可能会认为亚洲人 = 男性。TCAV 可以降低算法对单一判断标准的依赖，让最后的结果更加准确与公正。

### Mac 真的需要第三方搜索工具吗：ProFind

![](.evernote-content/504ABFEE-D291-4BB1-AC32-1A5D5731FD12/25F5DB76-A341-497F-BF19-E308488BBB32.png)

**ProFind 1.6.5**

[下载链接](http://www.zeroonetwenty.com/profind/#DLD)

**@Minja：**批评 Finder 的简陋、难用似乎已经成了一种正确方向，只要抓住一个「缺点」，就会有一堆第三方增强软件冒出来。最近开始受到关注的 ProFind，瞄准的就是 Finder 貌似羸弱的搜索功能。其实，包括在 [奏折 017](https://sspai.com/post/54004) 中介绍过的老牌搜索工具 HoudahSpot，依靠的还是系统自己的 Spotlight 索引，和 Finder 没有本质差别。

实际上，不是 Finder 简陋，而是搜索设置 —— 搜索语法、智能文件夹 —— 藏得太深，一般人可能发现不了。算上这些高级功能就可以很负责地说，Finder 搜不到的东西，那些吃 Spotlight 索引这碗饭的工具照样搜不出来（除非有一种病毒专门揪着 Finder 打，不过这种事情几乎不可能发生）。就电脑端而言，ProFind 除了**能够忽略不需要的文件夹**，其他功能基本逃不出 Finder 的五指山。相比之下，它的移动端做得就很有差异化，**支持在手机或 iPad 上检索电脑内的文件**，并且搜到后能直接使用。

![](.evernote-content/504ABFEE-D291-4BB1-AC32-1A5D5731FD12/D4DDCB46-EC86-428A-8BE3-4BE9F883A67A.png)

在手机上搜电脑文件

这意味着如果你正捧着 iPad 在沙发上休息，忽然同事让你发点文件过去，靠 ProFind 就能原地完成搜索、发送，不需要开电脑。可惜抛开这些非常少见的搜索场景不说，多数情况下我们还是不需要第三方搜索工具，至少不需要一个「换皮」版的 Finder。

### Android 新技术 FIDO2 让用户实现无密码登录

**@Fairyex：**虽然现在第三方登录及二次验证等登录方法已经让整个登录流程比以前方便不少，但无密码登录仍然是所有公司和用户想要达到的登录流程最终形态。

Google 也同样在这方面努力。现在，新的 FIDO2 技术使得用户可以把他们的手机 / 平板作为类似 U 盾的安全确认装置。目前只支持 Chrome 登录 Google 自家的服务，未来会有越来越多的网站及浏览器支持（Firefox 与 Edge 也有基本支持）。

想要体验无密码登录非常简单，一台 Android 7.0+ 的手机，最新版本的 Chrome 浏览器与开启两步认证的 Google 账户即可。当你登录支持 FIDO2 的网站时，浏览器会提示你在手机上确认登陆，接着手机会自动弹出确认登录提示，只需要按下「是」即可完成无密码登录。

加载过程演示

那么 FIDO2 是如何保证用户能够安全登录的呢？首先手机会通过蓝牙，无线网络，移动网络，GPS 等确认用户手机与要登录的设备处于同一位置。接着 Chrome 会确认登录地址是否为钓鱼地址，最后用户按下确认按钮，手机（已经登录 Google 账户）会发送一个一次性加密的登录确认信息到 Googke 服务器，这样就完成了无密码登录。

实际上 Graphviz 全名叫「Graph Visualization」，是由 AT&T 开发的开源图像绘制库（详见官网 [Graphviz](https://graphviz.gitlab.io/)）。其底层依赖语言叫做 Dot，上面提到的插件底层上也就是为 VS Code 提供了这种语言的支持。下面是 VS Code 中使用 Graphviz 所需要的插件：

### 视频：关于冥想的作用讲解

：[The Meditating Brain - YouTube](https://www.youtube.com/watch?v=dH-ZA6WGSBY)

****@Fairyex：****无论你觉得这是迷信或者传统习俗，冥想都有着悠久的历史。很多人都会在很多时候用类似冥想的方法放空自己，暂时远离喧闹的世界，但是背后的科学原理却很少有人主动研究。

BrainFact.org 的 YouTube 视频[「The Meditating Brain 」](https://www.youtube.com/watch?v=dH-ZA6WGSBY)简单介绍了冥想背后的科学原理。

视频中介绍思想平静并不意味着大脑正在休息，实际上冥想会对大脑的活动有造成可测量的影响。新手冥想的时候大脑会转移活跃思考时产生的 β 波到较慢的 α 波，经验丰富的冥想者还能产生更强的 Gamma 波，大脑的不同区域开始调节注意力，并控制你对自己身体思想和情绪的认识，就像在健身房锻炼身体一样。

且冥想可以实际上使大脑区域得到锻炼，冥想者的大脑皮层实际上比非冥想者更厚并有着更多的褶皱，它也能减轻压力、改善情绪甚至缓解焦虑和抑郁症状。

所以冥想是一种神奇的疗法？目前科学家仍需要更多的研究来确认冥想对心理症状有多少的帮助，但是尝试冥想并没有太大的坏处，只需要坐下来，深呼吸，集中注意力。
