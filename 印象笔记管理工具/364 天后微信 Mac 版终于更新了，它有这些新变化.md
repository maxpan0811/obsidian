# 364 天后微信 Mac 版终于更新了，它有这些新变化

---

![](.evernote-content/4319A3B4-2C8C-4B0B-9197-168484D03350/0EE45144-31DA-4769-83AD-1662CC3F58DA.jpg)

今天中午，「活久见」系列又加入了一个新成员——在沉寂了 364 天之后，Mac 版微信终于迎来了 2.0 版的更新。对多数人来说，这次更新最大的两个变化应该就是「长得像 Windows 了」和「可以在电脑上斗图了」（收藏功能都要往后排）。

![](.evernote-content/4319A3B4-2C8C-4B0B-9197-168484D03350/1F9CB033-03A5-4302-B89C-4A02B0509893.jpg)

不过外观上看起来和 Windows 版几乎一样的 Mac 版也并不是简单地把 Windows 版的功能搬了过来，它还「首发」了 Mac 锁定、收藏分类等几个 Windows 版上也没有的「独占」功能。

而对于已经在上一个版本停留了一年的 Mac 用户来说，也有一些 Windows 用户早就享受到的实用小功能也值得你在更新之后去尝试一下。

5 个 Mac 上独有的功能
--------------

### 锁定功能

这个是 Mac 版微信的首创功能，不少人临时离开电脑但是又觉得把微信退出再登录很麻烦，这时你就可以考虑使用锁定 Mac 微信功能单独锁定微信，此后只要通过手机解锁即可。

![](.evernote-content/4319A3B4-2C8C-4B0B-9197-168484D03350/3BA2A0EB-1CE2-4F9E-9FFD-F744698745E4.jpg)

### 分类查看收藏项目

可以直接查看收藏内容也是本次更新加入的一个重要功能，对于经常使用收藏功能的人来说，Mac 版加入的分组功能显得非常实用，它可以让你快速通过文件类型定位到你收藏夹中的内容。

![](.evernote-content/4319A3B4-2C8C-4B0B-9197-168484D03350/EEC647E2-8FCB-4DA6-833F-BB03AD77E2D4.jpg)

### 视频保存

可能因为新版调用了 Mac 的分享功能，所以 Mac 版还多了一个保存小视频的功能，好友给你发送过来的小视频在打开之后可以另存为 MP4 文件。

![](.evernote-content/4319A3B4-2C8C-4B0B-9197-168484D03350/F6E5EFBB-BDE5-4A7B-B31D-8D90EA037F06.jpg)

### 修改群公告

Mac 微信也是第一个把群成员列表从对话框的上方移到右侧的，对于人数过多的群组来说这样的查看方式显然比之前友好一些，而且群主也可以直接在里面修改群公告了。

![](.evernote-content/4319A3B4-2C8C-4B0B-9197-168484D03350/78614CBE-117E-4831-94CC-AC52226D72A3.jpg)

### 分组折叠

好友列表可以按照群聊、公众号和联系人三个分组进行分类折叠，虽然现在不少人已经放弃了直接在好友列表找人，而是采用搜索功能，但是想想每次点进好友列表都要面对一长串的群聊和好友名单，折叠起来至少……美观点。

![](.evernote-content/4319A3B4-2C8C-4B0B-9197-168484D03350/59505DC3-17A1-453C-9FE9-67FFEBC6EE52.jpg)

4 个容易被忽略的小功能
------------

### 下个表情雨

除了支持收发自定义表情之外，手机上常玩的「表情雨」功能现在也到了电脑上，快去试试满屏的 😂 往下落是一种什么样的体验？

![](.evernote-content/4319A3B4-2C8C-4B0B-9197-168484D03350/2FBF1B5C-5331-4CFA-9E03-9CEEE8635D71.jpg)

### 撤回 & 引用消息

以前在电脑上发错一条消息急急忙忙地掏出手机撤回的情况终于不存在了，这次更新之后你也可以直接在 Mac 版的微信上撤回消息。

![](.evernote-content/4319A3B4-2C8C-4B0B-9197-168484D03350/E213541F-C14F-4311-89CA-D59C488F74A9.jpg)

同时经常群聊的用户呼声很高的引用消息功能也加入了新版，不过和 Telegram 的引用相比，微信这种直接把引用信息再复制一遍的方式就显得太过简单粗暴，在实用性上也差了很多。

![](.evernote-content/4319A3B4-2C8C-4B0B-9197-168484D03350/A6D7C7DD-4D94-45A9-9475-0FCF831FB030.jpg)

### 消息免打扰功能

暂时不想被好友或群组的信息打扰，直接在对话列表点击右键就可以设置免打扰功能。

### 在收藏中添加本地文件

虽然你在聊天界面也能看到搜索框旁边的 + 号，但是这两个加号的功能是不一样的，在收藏界面你可以在这里直接把本地文件添加到收藏中，直接使用拖拽功能也能达到同样的效果。

![](.evernote-content/4319A3B4-2C8C-4B0B-9197-168484D03350/46118AAE-8FD7-4E79-A7AA-980288C55DEC.jpg)

在下一个 364 天，我们可能会看到……
--------------------

### 各种 Bug 的修复：闪退、信息延迟等

闪退是今天升级过后用户反映最多的一个问题，应用会在使用自定义表情，或者首次登录是遇到连续闪退的情况，而且新版在接收消息时候还会出现一定几率的延迟现象。（不过作为一款新应用，遇到各种 bug 也是可以理解的）

### 视频通话 & 语音聊天

这是微信官方表示已经在开发中的功能了。

### 更加完整的公众号体验

公众号体验在新版上虽然得到了改善，将此前显示在左侧的公众号详情移到了对话窗口，不过公众号下面的服务栏仍然没有出现在这次更新中，目前这个功能在 Windows 版上应该已经可以使用。

![](.evernote-content/4319A3B4-2C8C-4B0B-9197-168484D03350/0B99ADB0-90EE-4745-8E97-C68ECC22D48F.jpg)

同时，在公众号查看历史文章的时候，如果选择「用浏览器」打开，浏览器并不会直接打开文章，而是出现需要验证。

![](.evernote-content/4319A3B4-2C8C-4B0B-9197-168484D03350/D5B74031-C7D1-4FCA-B5BB-53D8D4E98DD6.jpg)

### 编辑收藏内容的标签

新版虽然引入了收藏页面，还率先添加了分类查看功能，但是收藏页面右侧的标签列表只能查看已经添加的标签，却没有办法对已收藏项目的标签进行编辑，有一种顾此失彼的感觉。

对于已经等待到绝望的 Mac 用户来说，在尝试了网页版以及各种第三方网页封装版的微信之后，能够看到微信官方还没有放弃 Mac 用户已经是一个很不错的消息了，不过对于新版的这些问题，还是希望 Mac 微信团队的更新能再稍微「勤快」一点，下次更新不要再是 364 天之后。
