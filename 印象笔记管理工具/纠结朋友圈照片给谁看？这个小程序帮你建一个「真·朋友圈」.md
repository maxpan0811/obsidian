# 纠结朋友圈照片给谁看？这个小程序帮你建一个「真·朋友圈」

---

在我的生活圈子里，总会因为朋友聚会、公司活动等场景需要而在微信建立群聊交流，在活动结束的时候，群聊的成员都会先后发出自己所拍摄的照片，但由于微信简单的时间线和恼人的操作方式让许多用户并不能如意地看到发在群聊中的所有照片，如：

* 朋友 / 同事 A 发了自己拍的照片引起成员讨论，讨论一旦过多，照片便不能够及时被浏览；
* 朋友 / 同事 B 邀请 C 发一些他拍的照片，C 发出了，但并不是原图；需要原图的话需要重新发，而且需要「经过照片 - 选择照片 - 原图」的路径操作；
* 一段时间后，去浏览群聊这些照片时，发现文件已过期；
* 想要查看群聊的所有图片，需要先点击一张图片，再点击右上方的功能键才能显示，但这显示的内容不仅图片，还有各种小视频、链接等。

不过现在，得益于小程序「多人相册」，上述的状况终于得到了解决。

使用介绍
----

小程序「多人相册」可以根据任意主题创建相簿，而且图片以原图保存，永不过期。

![](.evernote-content/44B7B5A2-6489-4C0B-A871-37CBEE3143AC/6B8ABCA5-5158-4A1C-ACC9-2CEB0F36EE0A.png)

进入该小程序后即可见到界面中的主要键位，具体实现功能如下：

* **新建多人相册**：建成后可邀请好友加入；
* **创建群相册**：即分享到微信群中的群相册，只允许成员进入，确保私密安全；
* **圆形对话按钮**：点击即可跳转到客服咨询界面。

点击后新建相册后，输入相册名称即可创建成功。

![](.evernote-content/44B7B5A2-6489-4C0B-A871-37CBEE3143AC/1632D7ED-F018-47C1-AB9F-626AA178DFB9.png)

在新界面中，你除了可以邀请好友加入、分享至朋友圈、添加照片外，还可以对创建的相册进行相关设置，包括但不限于相册名称、群昵称、是否允许成员添加照片、红包相册。

![](.evernote-content/44B7B5A2-6489-4C0B-A871-37CBEE3143AC/725D4701-1E7B-4942-8B1F-8B808A82CDF2.png)

这些功能很大程度上为我们解决了开篇所述的问题：

#### **方便被搜索和浏览**

相册是一个整体，利于被搜索和浏览。在聊天详情界面，点击聊天小程序，选择「多人相册」即可浏览该聊天窗口的所有相册内容。同时，邀请好友加入后，上传的照片便存在于同一个相册之中，方便整理和浏览。

![](.evernote-content/44B7B5A2-6489-4C0B-A871-37CBEE3143AC/7E691F9E-7EB9-443D-B03B-6A166AD6E428.png)

#### **相簿以原图保存，永不过期**

这意味着无论是谁，每一次在相簿上传的照片都以原图的形式存储在云端，永不过期。

有意思的功能
------

### 红包相册

小程序「多人相册」最为特别且让人印象深刻的功能是可设置「红包相册」。开启后，可以设置金额大小，新成员需要支付一定金额才能进入相册查看，原有的成员不受影响。

![](.evernote-content/44B7B5A2-6489-4C0B-A871-37CBEE3143AC/62A238AB-0054-4CE2-967F-653AF81D6660.png)

当其他人收到时，会以对话窗口展现，并标明「只需要￥？，*相册名称*随便看」，点击后即可跳转至微信支付界面，支付成功即可浏览该相册。不过略显遗憾的是，由于技术原因限制，红包相册的提现需要关注公众号时光相册。

![](.evernote-content/44B7B5A2-6489-4C0B-A871-37CBEE3143AC/85758AF9-DF14-4AC7-BA27-3B8C2CF3A9C6.png)

### 在相册点赞和评论

在群聊中浏览和讨论照片是一种非常不明智的行为，因为有时候会不知道讨论的具体对象是哪一张，或者讨论过多就难以浏览图片本身。相比之下，通过「多人相册」创办的相册就没有这个问题。

![](.evernote-content/44B7B5A2-6489-4C0B-A871-37CBEE3143AC/1E0565E4-9071-42BB-B997-B71EF10BDC28.png)

相册就如同朋友圈，成员间可以浏览；而每一张照片就像是一条朋友圈状态，成员们都可以为其点赞和评论，甚至还可以针对某条评论进行回复，非常方便。

需要注意的是，这个小程序由「时光相册」App 团队开发，因此你的照片有可能会被保存到第三方的服务器中，所以在使用时请注意保护好自己的隐私。

面对微信群聊图片的糟糕体验，「多人相册」无疑是最佳的拯救者，想要体验这款好用且高效的小程序，请直接在微信搜索即可。
