答疑汇总 04
=======

教程的倒数第二章，终于开始了解 Ulysses 的「压箱之宝」—— Ulysses Style Sheet！

有了 USS 之后，我们有了改造 Ulysses 排版样式的能力，原先一些我们只能希望开发者来做的事，通过 USS 就可以自己动手，丰衣足食。

---

Ulysses 作为一款诞生于海外的工具，在排版风格上自然与国内国情略有不符。例如读者 @ 一碌葛 就问道：

> 怎么为中文文章设置首段空两格？

为首段空两格，其实就是**调整段落文字的首行缩进**。我们来一步步分析：

1. 既然是调整段落，那么应该去找段落的样式选择器 `paragraph`；
2. 在[文档](https://www.ulyssesapp.com/styles/ulss-reference/reference.html#paragraph)中找到了相应的选择器，仔细看可用的属性，其实 `first-line-indent` 正好是用来调整首行缩进；
3. 「两格」意味着两个字的宽度，不能简单的给出「X pt」这样的值，而是必须根据样式中对字体的设计来调整首行缩进的值。

其实在第七章中被用来举例的 Papers，就已经设置了首行缩进：

```
paragraph {  
    style-title:            "Paragraph"  
    first-line-indent:      $indent  
}
```

而缩进的值，它用的是变量 `$indent`：

```
$indent = 0.5in
```

很明显我们不能这样来设置缩进的值，而是去找字体的设置。其中有一个变量：

```
$font-size = 12pt
```

正好设定了字号，那么剩下的事就简单了，我们利用乘法表达式设置缩进：

```
first-line-indent: 2*$font-size
```

---

USS 的用法非常灵活，一个需求可能会有好几个实现。有时甚至可以打破官方文档给出的定义，用各种曲线救国的方式来实现我们的需求。例如读者 @Xvinzhou 问道：

> 我是学法语的，经常需要涉及到中文和法语同时出现的情况。这种时候不知可否给中文和法语单独设置不同的字体？

本来在一种元素（例如段落）内是只能设置一种字体，但是有没有什么办法可以实现中法文用不同的字体呢？

这个需求的难点在于，如果正常地书写，中法文都被视为段落内的普通文字。那么关键就在于，如何将法语设定为另一种元素。这时我们就可以利用 Ulysses 定义的行内元素，例如行内代码、行内原始源，**将法语用一种行内元素的语法标记包裹起来**，再去修改这种元素的字体，就能实现中法文使用不同字体的效果了。

还是以 Papers 为例，默认的字体为：

```
defaults {  
    font-family:            "Times New Roman"  
    font-size:              $font-size  
    line-height:            $line-height  
}
```

我们将行内原始源分配给法语文字。回到 USS：

```
inline-raw {  
    visibility:             hidden  
}
```

行内原始源被设定为不可见，我们参照正常段落的文字样式，但用 Zapfino 这种字体来显示法语：

```
inline-raw {  
    font-family:            "Zapfino"  
    font-size:              0.7*$font-size  
    line-height:            0.5*$line-height  
}
```

因为 Zapfino 是手写体风格的字体，我们将字号和行高相应地缩小。在写作时，用原始源的符号包裹住法语部分：

从这两个实例可以看出，USS 不仅给了我们调整样式的空间，通过一些选择器和属性的混搭，说不定还能实现开发者都没有料想到的需求！
