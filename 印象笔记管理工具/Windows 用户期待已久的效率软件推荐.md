**1Clipboard**
--------------

支持双平台的免费剪贴板工具
-------------

![](.evernote-content/E333033C-32BF-46D6-AC18-6EF4FE83739B/71D2C9DC-DD3B-4CEC-917C-E4D3913341A3.png)

剪贴板工具可以帮你保存数百甚至更多条复制内容，包括文本、图片或文件，尤其当需要粘贴较多内容时就会非常有用，让你事半功倍。

1Clipboard 是一款简洁易用的免费剪贴板工具，支持 Windows 和 macOS 双平台，还可通过 Google Drive 来同步数据，作为一款免费软件，功能已足够满足大多数用户需要。

![](.evernote-content/E333033C-32BF-46D6-AC18-6EF4FE83739B/EFF1E2E8-B55A-49C4-AE71-EE8918F064A9.jpg)

**Vimium**
----------

用键盘控制 Chrome 浏览器
----------------

![](.evernote-content/E333033C-32BF-46D6-AC18-6EF4FE83739B/CF5A039D-D296-4948-89B7-694834488CE4.png)

浏览器或许是大多数人最常用的电脑软件之一，而 Vimium 这款 Chrome 扩展可以让你的浏览器更加好用。Vimium 可以让 Chrome 支持 Vim 编辑器的快捷键，例如按下 t 打开新标签页，按下 d 往下翻页，「Shift + J 或 K」则可以跳转到下/上一个标签页，让你浏览网页更加快速、顺手。

**XMind Zen**
-------------

高颜值的思维导图工具
----------

![](.evernote-content/E333033C-32BF-46D6-AC18-6EF4FE83739B/546E97E1-7E44-4800-8BA0-6B6B070D48DB.png)

思维导图可以帮助我们整理思路、规划工作和学习，或是捕捉每一个灵感瞬间。XMind Zen 是一款简洁美观、功能强大的效率工具，你可以用它快速绘制各种思维导图、脑图、鱼骨图、甘特图等图形结构，让思维清晰可见，或是随手记录点滴灵感，会议记录。

XMind Zen 支持 Mac & Win 双平台，可免费使用基础功能。

![](.evernote-content/E333033C-32BF-46D6-AC18-6EF4FE83739B/41530215-24BD-4A76-8FBB-1E690517C75C.jpg)

**Listary**
-----------

Windows 上最好用的文件搜索工具
-------------------

![](.evernote-content/E333033C-32BF-46D6-AC18-6EF4FE83739B/5B6C8001-208A-4762-B4CA-575638F6D24E.png)

Listary 是 Windows 上一款超高效率的本地文件搜索工具。也许你的文件散落在电脑的各个角落，难以管理和查阅，但 Listary 可以帮你快速搜索找到。不同于其他搜索工具，Listary 并没有一个固定的「搜索框」，只需要在文件管理器中任意位置输入文件名，它就能立刻进行文件搜索。

![](.evernote-content/E333033C-32BF-46D6-AC18-6EF4FE83739B/2A5F1D17-A0BA-49C9-B240-A4EA40EEA017.jpg)

> Listary 个人用户免费，功能更多的高级版「数码荔枝」有上架，特惠只需 59 元，比官网便宜 78 元，欢迎点击「**阅读原文**」，去免费下载试用或入手。

**Mathpix**
-----------

多平台的图片转 LaTeX 公式工具
------------------

![](.evernote-content/E333033C-32BF-46D6-AC18-6EF4FE83739B/3523AB44-897B-4EA5-B46F-6204FE6C4B53.jpg)

对于学生党、写论文的朋友，很可能会经常和 LaTex 打交道，那么 Mathpix 这款小工具一定不要错过，它可以帮你将截图等图片转换为 LaTex 公式，还可以进行多种变换。软件支持 Mac、Windows、Ubuntu、iOS 等多平台，均可免费下载。

![](.evernote-content/E333033C-32BF-46D6-AC18-6EF4FE83739B/46ABB20F-AA90-4982-A989-588D73EE3C12.gif)

**QuickLook**
-------------

让 Windows 支持 macOS 的快速查看
------------------------

![](.evernote-content/E333033C-32BF-46D6-AC18-6EF4FE83739B/E75ADC90-2DAF-43B2-8610-6C31FC6E60D0.png)

「快速查看」可谓 macOS 的杀手锏功能，只需轻点空格键，就能快速浏览图片、各种文档，甚至是音乐、视频内容，还可以拖动进度条，相比打开文件查看、再关闭，节省了大量操作。

![](.evernote-content/E333033C-32BF-46D6-AC18-6EF4FE83739B/16C59516-8B8B-4066-9529-74225F7EE444.gif)

而 QuickLook 这款开源免费的小工具，终于让 Windows 10 也获得了这一强大功能，再也不用羡慕 Mac 用户。它支持多种文件格式单击空格键预览，还可以用方向键切换不同文件。
