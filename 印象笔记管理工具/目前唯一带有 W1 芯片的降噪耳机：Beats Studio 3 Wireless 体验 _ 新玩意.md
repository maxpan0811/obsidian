# 目前唯一带有 W1 芯片的降噪耳机：Beats Studio 3 Wireless 体验 | 新玩意

---

Beats 在去年更新了旗下多款无线蓝牙耳机，还推出了搭载苹果 W1 芯片新品 [BeatsX](https://sspai.com/post/37935)，但唯独少了旗舰耳机 Beats Studio 的身影。上个月，在时隔三年之后，Beats 终于推出了搭载苹果 W1 芯片的新一代蓝牙降噪耳机：Beats Studio 3 Wireless。

![](.evernote-content/C081EAA4-F833-4B42-AC82-73964C15156C/01B9CE6F-FBE7-4119-B779-E032960768F5.jpg)Beats 四款带有 W1 芯片的无线耳机

在正式发售后，少数派第一时间收到了 Beats 提供的样机。究竟姗姗来迟的新旗舰表现如何？加入 W1 芯片的 Beats Studio 又有何不同？我们希望通过这篇文章，尽可能为你还原真实的使用体验。

外观和佩戴
-----

对于绝大部分用户而言，外观往往决定了他们对一款产品的第一印象。在被苹果收购后，Beats 的耳机在设计上也受到了苹果「设计美学」的影响，向着更加极简的路线靠拢。Beats Studio 3 Wireless 也是如此。

耳机基本上沿用了上一代 Beats Studio 的设计，哑光的材质看起来很有质感。第一眼看去，除了右侧单元底部的电源键，整个机身找不到其他按键和指示灯。和 Beats Solo 3 一样，播放控制键被藏在了左侧单元的 Beats Logo 上，电量指示灯也只有在按下电源键时才会依次亮起。这些设计上的细节再搭配上黑色的机身，让耳机显得很有一体感。

![](.evernote-content/C081EAA4-F833-4B42-AC82-73964C15156C/088EC20E-F2D0-454D-97A8-56BED5DE5D4B.jpg)很有一体感的设计

对戴眼镜的人来说，头戴式耳机在佩戴时「夹头」是一个不小的问题。Beats Studio 3 Wireless 的耳罩采用了较为柔软的人工皮革材质，顶部有橡胶材质的缓冲，虽然一定程度上有缓解「夹头」的问题，但还是会有一些压耳感，皮质的耳罩也有点闷，再加上它不算太轻的重量（260g），不太适合长时间佩戴。

![](.evernote-content/C081EAA4-F833-4B42-AC82-73964C15156C/079167C3-5204-4772-816F-0AEE3D8067D0.jpg)被「藏」在左侧 logo 上的播放控制键

降噪
--

Beats Studio 3 Wireless 搭载了名为 Pure ANC 的自适应降噪技术，根据官方的说法，它能够针对不同环境噪音以及播放的音乐类型进行每秒 50,000 次音频校准。这让我们在拿到 Beats Studio 3 Wireless 之前对它的降噪效果一开始充满了期待。

提到目前市面上的主动式降噪蓝牙耳机，很多人都会想到 BOSE QC35 和索尼 MDR-1000X。前者是降噪耳机市场的领先者，后者则拥有语音模式和 LDAC 等各种「黑科技」。作为两者的直接竞争对手，Beats Studio 3 Wireless 的降噪效果在实际体验中并没有明显的优势，但它有一些自己独有的功能。

开启降噪模式时，不同于 QC35 以及 MDR-1000X 带上耳机立刻进入降噪模式的体验，Beats Studio 3 Wireless 会经历一个噪音逐渐降低的过程，持续 3 秒左右。据 Beats 称，这样的设计是为了让用户能够在佩戴耳机时确认身边环境的安全。

![](.evernote-content/C081EAA4-F833-4B42-AC82-73964C15156C/53233291-DA11-4D62-AF3E-F42DDD676B28.jpg)双击电源键开启降噪模式

在日常生活中，降噪耳机的一个最常见的使用场景是通勤路上。在使用 Beats Studio 3 Wireless 通勤的几天里，在公交车上的感受是：在正常的播放音量下（30-40%），它能较好地消除行驶过程中的引擎声，同时不会让你错过报站声。这样的降噪效果还是让人很满意的。

几天的使用下来，少数派发现 Beats Studio 3 Wireless 的一个优势场景是针对风声的降噪。当风迎面吹来时，在使用开启降噪模式的 QC35 时，你还是能够听到耳边传来「呼呼」的声音。但是使用 Beats Studio 3 Wireless 时，它能够在迎风时慢慢地过滤掉这种声音。不过，当你转头或者改变方向时，风声会重新明显起来，需要等待几秒后噪音才会被消弱。

![](.evernote-content/C081EAA4-F833-4B42-AC82-73964C15156C/A018DC29-E68E-448C-99AA-0B630271239F.jpg)针对风声特别优化的降噪

总体而言，Beats Studio 3 Wireless 这种「渐进式」的降噪虽然有自己的一些亮点，但是体验上还有许多可以优化的地方。另外，在使用了几天后，我们这只耳机的右单元在开启降噪模式后会听见明显的底噪，只有关闭降噪模式才能消除。尽管这有可能是个别产品问题，但至少说明在降噪技术上，Beats 依然有许多可以提升的空间。

W1 芯片
-----

去年苹果首次在自家的的蓝牙耳机 AirPods 上使用了自家研发的 W1 芯片。作为被苹果收购的 Beats，也是近水楼台先得月，在 Beats Solo 3、PowerBeats 3 Wireless 和新品 BeatsX 上使用了 W1 芯片，这次 Beats Studio 3 Wireless 自然也不例外。

在之前少数派对 [AirPods](https://sspai.com/post/36653) 和 [BeatsX](https://sspai.com/post/37935) 的体验文章中，我们都提到 W1 芯片对于优化连接体验和提升续航起到了非常重要的作用。这句话同样适用于 Beats Studio 3 Wireless。第一次配对时，长按电源键 5 秒，靠近打开蓝牙功能的 iOS 设备，即可看到自动弹出的匹配界面，就这样快速完成了匹配。

![](.evernote-content/C081EAA4-F833-4B42-AC82-73964C15156C/64182A60-7093-4E81-B389-2BBCF2801E34.jpg)第一次配对

需要切换设备时，如果你的设备在同一个 Apple ID 下，你可以在 iOS 的控制中心和 macOS 的菜单栏音频设置中看到已经通过 iCloud 账户同步的 Beats Studio 3 Wireless，点击设备一键完成切换，整个流程非常无缝。

另外，除了双击电源键开启降噪模式，你还可以在手机和电脑上 Beats Studio 3 Wireless 的蓝牙设置中进行调节，而 QC35 和 MDR-1000X 都需要第三方 App 来实现这个功能。

![](.evernote-content/C081EAA4-F833-4B42-AC82-73964C15156C/A2FADE9E-125F-48A3-B3AA-367F1361D7A4.jpg)在系统设置里可以控制是否开启降噪

Beats Studio 3 Wireless 仍使用了 Mirco USB 充电口，而不是像 Beats X 一样使用 Lightning 接口。不过好在续航非常不错。根据官方说法，在开启降噪功能的情况下，Beats Studio 3 Wireless 的电池能够提供 22 小时的续航，而在关闭降噪功能的情况下，续航可以达到 40 个小时。

实际的体验情况基本和官方所称一致，每天听歌 6 小时左右的情况下，一次充满电的 Beats Studio 3 Wireless 坚持了 4 天的时间，这个续航成绩在我们使用过的蓝牙降噪耳机里算是数一数二的。不仅如此，配合上 Fast Fuel 快充功能，你可以在不到一个小时的时间内完成 0% 到 100% 的充电。

听感
--

尽管 Beats Studio 3 Wireless 的名字里有 Studio（工作室），但它很明显并不是针对专业用户的耳机。随着高编码率蓝牙音频传输协议的出现，蓝牙耳机的音质在近几年开始有了非常明显的进步。不过，Beats Studio 3 Wireless 并不像索尼 MDR-1000X 一样支持超高编码率的 LDAC 协议，因此它的音质上限其实比较有限。

![](.evernote-content/C081EAA4-F833-4B42-AC82-73964C15156C/7A157F75-931A-4D93-B465-FE1C04CB2205.jpg)

如果放下 Hi-Fi 的要求，只考虑日常使用场景的话，那 Beats Studio 3 Wireless 在 Beats 的无线蓝牙耳机里，可以说是声音最出色的一款。它整体的声音依旧趋向低频，但声音细节还原方面表现还不错。对于日常听流行音乐为主的用户来说，算是一款比较合适的耳机。不过，它的声音有一点点浑浊，解析力并不是很出色，当碰到像交响乐这样比较复杂编制的音乐时感觉明显力不从心。

总结
--

对于一款售价￥2888 的耳机，Beats Studio 3 Wireless 值得购买吗？对于任何一款 Beats 的产品，如果单从音质的角度作比较，你总能找到同价位声音素质更好、性价比更高的选择。在 Beats 的产品属性里，品牌价值是非常重要的一环，如果你不认同它的品牌价值，那么￥2888 价格可能很难打动你。

但是，如果你同时使用多款苹果产品，需要一款拥有 W1 芯片的蓝牙降噪耳机，你会发现 Beats Studio 3 Wireless 是目前市面上你唯一的选择。W1 芯片带来的连接体验和超长续航是它的独有优势，￥2888 你得到的是目前其他蓝牙降噪耳机都无法提供的体验。
