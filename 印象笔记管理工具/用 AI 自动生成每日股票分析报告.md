# 用 AI 自动生成每日股票分析报告

---

来源：[打开原文](https://mp.weixin.qq.com/s/VfZkliOXVHi1Ctgogl3vGw)

用 AI 自动生成每日股票分析报告

支持 A 股、港股、美股和 ETF，每只股票出一份报告，包含综合评分、买卖点位建议、主力资金流向、筹码分布、近期新闻舆情、风险警报和操作检查清单。

把行情数据、技术指标、资金流、公告和新闻整合在一起，生成一份有结构的参考报告。

AI 模型支持很全，DeepSeek、千问、Claude、OpenAI 兼容接口都能接，也支持 Ollama 跑本地模型。

新闻搜索源同样可配置，多个搜索服务可以按需组合，A 股场景下中文搜索的覆盖效果明显好于英文源。

部署简单，零成本。 Fork 仓库后配置 GitHub Actions，填好 Secrets 里的 API Key 和股票代码，每个工作日 18 点自动触发，不需要服务器，零成本跑起来。

配置完成之后，每天收盘后自动推送一份决策仪表盘到企业微信、飞书或 Telegram。

也配套了Web 工作台，可以手动触发分析、查历史报告、管理持仓，支持浅色深色主题切换。

内置Agent 问股功能，支持均线、缠论、波浪等十几种内置策略，可以多轮追问，类似一个会调取实时行情数据的股票助手。

推送效果看仓库里的示例，格式包括评分 + 方向 + 核心逻辑 + 具体风险点等等。

当然，市场信息嘈杂，AI 给的建议不能替代自己的判断，仓位和止损还是要自己把控。
