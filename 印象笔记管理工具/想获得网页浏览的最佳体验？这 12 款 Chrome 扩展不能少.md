想获得网页浏览的最佳体验？这 12 款 Chrome 扩展不能少
================================

2016 年 01 月 17 日

[![](.evernote-content/A44CFDDF-A480-483D-820D-F2883AE97CF0/0A5FF05D-3129-4F47-952D-A0F5AFEEF44A)](https://sspai.com/user/78)

#### [化学心情下 2](https://sspai.com/user/78)

相信不少朋友和我一样，每安装新的操作系统之后，都会使用 Safari/IE 下载 Chrome 浏览器，然后将后者作为主力上网浏览器，而将前者束之高阁。毕竟从体验上说，Chrome 拥有快速流畅的浏览体验，当然强大丰富的扩展也是让我一见倾心的关键。

此前我们向各位介绍了数款颇受欢迎的 Chrome 扩展，而今天笔者我起一回私心，给大家安利一些我正在用的 Chrome 扩展，相信常用 Chrome 你一定可以在其中淘到不少好货。

划词翻译：强大的多引擎划词翻译工具
-----------------

虽说 Chrome 浏览器内置有 Google 自家的翻译全文至 xx 语言的工具，但这个工具并不能逐句翻译，显得并不是那么「人性化」。而这款名为「划词翻译」的 Chrome 扩展除了可以对选中段落进行翻译之外，还可以在翻译时选择多个翻译引擎，除了我们熟知的「Google 翻译（分为 Google 翻译国内和国外两种引擎）」，还支持有道、百度以及必应引擎，弹出的翻译结果迷你框还可以固定在网页中与原文进行对照，非常方便。

在 [Chrome Web Store](https://chrome.google.com/webstore/detail/%E5%88%92%E8%AF%8D%E7%BF%BB%E8%AF%91/ikhdkkncnoglghljlkmcimlnlhkeamad) 下载 划词翻译。

![](.evernote-content/A44CFDDF-A480-483D-820D-F2883AE97CF0/45A49527-F28D-4C7B-A790-B29DB8305D9F.png)

![](.evernote-content/A44CFDDF-A480-483D-820D-F2883AE97CF0/04E30500-7227-42B2-B702-AFA1D7D4DE50.png)

印象笔记・剪藏：收集好文章，在印象笔记中珍藏
----------------------

当我在浏览网页时发现好的内容，就会将其通过「印象笔记・剪藏」这款 Chrome 扩展发送到我的印象笔记中。在「印象笔记・剪藏」中你即可以通过「隐藏广告」选项将网页内容提炼出来，方便你在印象笔记应用中进行「稍后读」。也可以通过「网页正文」、「整个页面」选项将网页以原格式保存，当然如果你仅仅只是给网页截图，那么就让「屏幕截图」的选项即可。同时你还可以对截取的内容打上相应的注释方便二次归纳与整理。

在 [Chrome Web Store](https://chrome.google.com/webstore/detail/pioclpoplcdbaefihamjohnefbikjilc) 下载 印象笔记・剪藏。

![](.evernote-content/A44CFDDF-A480-483D-820D-F2883AE97CF0/06BC7397-4D4F-49B6-B767-81C9631A2AE2.png)

![](.evernote-content/A44CFDDF-A480-483D-820D-F2883AE97CF0/F535C08C-4B5B-430D-9311-D8E2619318FD.png)

User-Agent Switcher：迅速变身「移动版」Chrome
-----------------------------------

简单来说，User-Agent Switcher for Chrome 这款小工具非常适合网页开发人员，通过自定义浏览器的 User-Agent，骗过网站以为你的 Chrome 浏览器是「移动版」，从而可以在 PC /Mac 端看到移动端才会呈现的页面与排版。

在 [Chrome Web Store](https://chrome.google.com/webstore/detail/user-agent-switcher-for-c/djflhoibgkdhkhhcedjiklpkjnoahfmg) 下载 User-Agent Switcher for Chrome。

![](.evernote-content/A44CFDDF-A480-483D-820D-F2883AE97CF0/95401B91-3CF1-43F3-81F0-BB7CC39790E1.png)

Push to Kindle：适用于国内环境的「Send to Kindle」
---------------------------------------

「Push to Kindle」是一款适用于国内 kindle 账户的「Send to Kindle」，弥补了国内 Kindle 账户用户无法使用 Amazon 官方「Send to Kindle」扩展的缺憾。在使用这款扩展之前，需要先将「kindle@fivefilters.org」这个邮箱地址加入到你的「已认可的发件人电子邮箱列表」中才可以正常收到推送。当你看到感兴趣或者希望在 Kindle 上阅读的内容时，点击「Push to Kindle」，在特殊的「发送至」页面就可以预览最终推送的排版效果。填写自己的 Kindle 邮箱地址，点击「Send！」就可以发送内容至 Kindle 设备了，实测国内使用无延时。

在 [Chrome Web Store](https://chrome.google.com/webstore/detail/push-to-kindle/pnaiinchjaonopoejhknmgjingcnaloc) 下载 Push to Kindle。

![](.evernote-content/A44CFDDF-A480-483D-820D-F2883AE97CF0/2BCBD6FE-13FC-4D37-BBF4-E5D418EDBCA3.png)

HTTPS Everywhere：开启全局 HTTPS 加密浏览
--------------------------------

如果你非常看重网络访问的安全性，那么你一定很清楚如果访问的网上银行不支持加密的后果：显然你几乎所有的敏感信息都可能被泄露，事实上有不少存储敏感信息的网站竟然不是默认开启全站加密，因此你需要这个名为「HTTPS Everywhere」的 Chrome 扩展进行强制开启。默认安装之后就立即生效，需要说明的是你可能在访问某些网站时会遇到「不安全的内容」，只要选择信任并完全加载即可。

在 [Chrome Web Store](https://chrome.google.com/webstore/detail/https-everywhere/gcbommkclmclpchllfjekcdonpmejbdp?utm_source=chrome-app-launcher-info-dialog) 下载 HTTPS Everywhere。

![](.evernote-content/A44CFDDF-A480-483D-820D-F2883AE97CF0/65318BD3-0932-472B-95E2-EDA2C0B4A62B.png)

Checker Plus for Gmail™：快速预览 Gmail 邮件
-------------------------------------

如果你和笔者我一样，习惯于在打开 Chrome 浏览网页的同时可以收到 Gmail 的实时推送，那么这款「Checker Plus for Gmail™」的扩展一定是最适合的，相比 Google 官方推出的「Checker for Gmail」扩展，这款第三方扩展可以实时查收邮件并提示新邮件数目，实时桌面推送提醒且无需你打开 Gmail 网页端，只需要通过当前页面中弹出的迷你窗口就可以查看新邮件，浏览邮件内容甚至对邮件进行快捷回复。

在 [Chrome Web Store](https://chrome.google.com/webstore/detail/oeopbcgkkoapgobdbedcemjljbihmemj?utm_source=chrome-app-launcher-info-dialog) 下载 Checker Plus for Gmail™。

![](.evernote-content/A44CFDDF-A480-483D-820D-F2883AE97CF0/C0C4EBB3-ACC3-462A-97D9-DCA0F81E7275.png)

Feedly Notifier：快速查看自己的 RSS 订阅
------------------------------

我因为每天都需要采编大量的国内外资讯，因此最离不开的工具就是 RSS 订阅工具了。而笔者所有的 RSS 订阅托管在 Feedly，这个名为「Feedly Notifier」第三方应用无需打开 Feedly 网站，在当前页面直接通过弹出的迷你窗口就中查看订阅的 feed 条目。此外这款扩展还可以进一步自定义，例如设定条目刷新间隔，控制弹出窗口大小以及新条目的通知方式等等。

在 [Chrome Web Store](https://chrome.google.com/webstore/detail/feedly-notifier/egikgfbhipinieabdmcpigejkaomgjgb?utm_source=chrome-app-launcher-info-dialog) 下载 Feedly Notifier。

![](.evernote-content/A44CFDDF-A480-483D-820D-F2883AE97CF0/5615D7E3-E5D6-440A-8F12-54002DE52BEF.png)

股票小助手：办公室的看盘利器
--------------

最近一年股票市场异常火热，但要想在工作上网时候「不经意」瞄一下大盘和自己的自选股，这个名为「股票小助手」的 Chrome 扩展应该最为合适：只需要点击扩展栏中「股票小助手」图标，就可以在弹出的迷你窗口看到当前大盘的涨跌情况、最新价、涨跌幅以及换手率，在搜索框中输入代码或者股票拼音还可以自动加入你的自选股中，鼠标移动到个股上还可以显示该个股的走势曲线。不过这个扩展也只能看盘，交易还是需要用手机或者交易软件搞定……

在 [Chrome Web Store](https://chrome.google.com/webstore/detail/%E8%82%A1%E7%A5%A8%E5%B0%8F%E5%8A%A9%E6%89%8B/pbhnnkmgakdodiillkoacilnkgonfkpe?utm_source=chrome-app-launcher-info-dialog) 下载 股票小助手。

![](.evernote-content/A44CFDDF-A480-483D-820D-F2883AE97CF0/1844DF7F-8C63-4B97-80BA-C8ADF97AD0B9.png)

AcFun Fix / 哔哩哔哩助手：让你追番更爽快
--------------------------

针对 AcFun 糟糕的用户体验，「AcFun Fix」的黑科技可以对被屏蔽的而无法播放的视频采取自动更改视频来源的方式顺利开启播放，其他诸如网页只显示头部、评论区不显示、播放器位置一片空白 / 黑屏等等这些怪毛病也统统都可以解决！不过目前这个小扩展还存在一些小 Bug：窗口模式播放时无法让播放器全屏、进度条有时会出现无法拖动的情况。如果影响不大大可无视。

在 [Chrome Web Store](https://chrome.google.com/webstore/detail/acfun-fix/mogaeppnobmjakbilmhpkklpfnljejfa?utm_source=chrome-app-launcher-info-dialog) 下载 AcFun Fix。

![](.evernote-content/A44CFDDF-A480-483D-820D-F2883AE97CF0/A360E1DD-2089-4105-BF30-9229D8729795.png)

针对 B 站同样也有与 AcFun Fix 类似的专用黑科技：「哔哩哔哩助手」，首先就能解决不少人遇到的 error 2048 视频播放错误：视频播放模式强制 HTML 5 优先。此外哔哩哔哩助手还加入获取视频下载地址格式、检查「我的关注」更新、播放器切换等小功能。如果你是番组重度癌患者，这款小助手确实可以帮助提升幸福感。

在 [Chrome Web Store](https://chrome.google.com/webstore/detail/%E5%93%94%E5%93%A9%E5%93%94%E5%93%A9%E5%8A%A9%E6%89%8B%EF%BC%9Abilibilicom-%E7%BB%BC%E5%90%88%E8%BE%85%E5%8A%A9%E6%89%A9%E5%B1%95/kpbnombpnpcffllnianjibmpadjolanh?utm_source=chrome-app-launcher-info-dialog) 下载 哔哩哔哩助手：bilibili.com 综合辅助扩展。

![](.evernote-content/A44CFDDF-A480-483D-820D-F2883AE97CF0/B10DDC47-DFF7-4FF7-B571-5FE2CFB6ADD7.png)

Pixlr Editor：试试这款「在线版」Photoshop
-------------------------------

Pixlr Editor 是由专业图像应用服务商 Autodesk 开发的在线图片编辑器，界面几乎就是 PhotoShop 的翻版，支持 PhotoShop 的图层以及 PSD 文件导入，而我们常用的套索工具、选择工具、魔棒工具也都统统支持，习惯 Photoshop 操作的朋友应该能很快上手，比较让人感动的是 Pixlr Editor 无需注册就可以免费使用。

在 [Chrome Web Store](https://chrome.google.com/webstore/detail/pixlr-editor/icmaknaampgiegkcjlimdiidlhopknpk?utm_source=chrome-app-launcher-info-dialog) 下载 Pixlr Editor。

![](.evernote-content/A44CFDDF-A480-483D-820D-F2883AE97CF0/C000E4E9-2A00-49E5-89E2-5DF2CD0D33A2.png)

LastPass: Free Password Manager：免费的密码管理工具
-----------------------------------------

就浏览器密码保存这项服务，笔者依旧选择了 LastPass，一个是相对配置比较简单，在默认表单的基础上可以几乎不做太多改动就可以通用，第二个是方便的密码填选功能，可以针对性在某一个站点存储多个密码，毕竟在目前的网络环境中，指望 Chrome 中 Google 账户的保存密码是不现实的（万一哪一天再也访问不了……），所以 LastPass 只能是我其中备选项之一。需要说明的是 LastPass 跨平台使用也是需要另外付费的。

在 [Chrome Web Store](https://chrome.google.com/webstore/detail/lastpass-free-password-ma/hdokiejnpimakedhajhdlcegeplioahd?utm_source=chrome-app-launcher-info-dialog) 下载 LastPass: Free Password Manager。
