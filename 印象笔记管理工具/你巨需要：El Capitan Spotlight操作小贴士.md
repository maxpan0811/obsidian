# 你巨需要：El Capitan Spotlight操作小贴士

---

[![](.evernote-content/114D4525-A6D9-4922-B399-4ECAC9F63824/27FB9FCD-7267-481E-BD2E-8F65CFFA8A25.png)](http://www.feng.com/iPhone/news/2015-10-13/You-need-huge-El-Capitan-Spotlight-operating-tips_626971.shtml)

　OS X El Capitan 中的 Spotlight 搜索得到进一步完善，增加了许多新的特性。你是否了解这些新特性对你日常使用 Spotlight 的方式会产生什么影响吗？下面就一起来了解这些操作小贴士吧。

![](.evernote-content/114D4525-A6D9-4922-B399-4ECAC9F63824/27FB9FCD-7267-481E-BD2E-8F65CFFA8A25.png)

**支持自然语言**

支持自然语言或许是 OS X El Capitan Spotlight 获得的最好的更新之一。也就是说用户现在可以直接使用自然语言来提问，以便获得解答。比如你可以这么问，“明天的天气怎么样？”或者是“我上个星期二打开的图片”。Spotlight 就会根据你给出的这些指令来提供相对应的答案。

**移动**

在OS X El Capitan 中用户现在可以在桌面窗口中随意移动或者调整 Spotlight 窗口的大小。

**天气和体育**

在 OS X El Capitan 中苹果已经将天气信息整合到Spotlight之中，所以不管什么时候只要你想了解一个地方的天气，比如北京，你就可以发出非常简单的指令，“中国北京的天气”，你就能够看到北京的天气预报。

体育比赛的分数也是如此。用户只要说出参赛的队名+分数，参赛队伍的分数（或者是赛程）就会出现在 Spotlight 之中。

**网页视频**

网页视频搜索对于 Spotlight 来说也是一个全新的特性，使用起来方便简单，只要你知道视频的名字你就可以进行搜索。用户只要在 Spotlight 中输入视频名称或者其他搜索细节，然后在末尾添加“视频”一词即可得到你想要的结果。

**股票**

OS X 用户当中真正炒股的人有多少我们不知道，不过 Spotlight 还针对股票完善了，这让我们觉得有点不可思议。但是如果你有需要的话，你可以在Spotlight中输入股票代码或者是“苹果股价”、“纳斯达克价格”等词语来获得最新的股市动态，股票数据均由 Yahoo Finance 提供。

**拖拽**

你是否知道在 Spotlight 的搜索结果列表中，用户可以拖动列表中的项目到Dock 图标上，从而在应用中打开该项目。或者是将其从项目列表中拖放到桌面，或在 Finder 窗口中打开，创建副本。（你还可以选择搜索结果，使用Command & C组合键将其复制到文件或文件夹中）。这并非 OS X El Capitan 的新特性，但是知道这项功能并利用其的人或许不多。

**其他**

有些用户可能表示自己并不能使用所有新的Spotlight 功能。如果你遇到这种情况，先打开 Spotlight Settings，关闭应用然后再重新打开即可解决问题。

Spotlight 键盘快捷键

-长按 Option 在 Spotlight 搜索结果窗口右下角查看用户当前选择项目所在的位置。

-Cmd+R或Cmd+Enter 将会在 Finder 或者合适的应用中打开当前高亮的搜索结果。

-Cmd+L 直接跳转到搜索结果的字典定义中。

-Cmd+B 在默认浏览器中搜索项目，默认使用任何一个选定的搜索引擎。

-Cmd+Backspace 可撤销错误输入的内容。
