# El Capitan新特性之——邮件手势操作

---

[![](.evernote-content/787C219E-7A7E-461B-BFD3-2F3B39301DAA/E4FB26B0-136A-4901-840C-D19395C41CDA.jpg)](http://www.feng.com/apple/tutorial/2015-10-22/El-Capitan-gestures---mail-of-new-features_627891.shtml)

除了支持全屏视图之外，El Capitan 中的邮件应用还新增了一个“给力”的功能-手势支持。

[![](.evernote-content/787C219E-7A7E-461B-BFD3-2F3B39301DAA/E4FB26B0-136A-4901-840C-D19395C41CDA.jpg)](http://resource.feng.com/resource/h060/h59/img201510222216460.png)

使用 iOS 设备的用户大多数都知道，iOS 端的邮件应用是支持手势操作的。在 El Capitan 中，苹果终于为 OS X 的邮件应用也带来了这一方便的操作。只需向右轻扫将邮件标记为已读或未读，或向左轻扫删除邮件即可。当然，前提是你要有 Magic Mouse 或者是触控板，否则无法实现。

特别是对于存有大量邮件的用户来说，手势操作可以更快速高效地对邮件进行整理和分拣，这样你便能第一时间关注到那些重要信息。操作和 iOS 设备无异，提高了桌面系统邮件应用的易用性。

这个功能既简化了操作，又提高了效率，是 El Capitan 注重稳定性提升的体现之一。
