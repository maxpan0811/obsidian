AWS 业务退出中国？其实这才是亚马逊作价 20 亿卖掉资产的原因
---------------------------------

来自雷锋网（leiphone-sz）的报道

11 月 13 日，光环新网宣布，该公司董事会已经同意以不超过人民币 20 亿元的金额（以分期付款的方式），收购亚马逊通技术服务（北京）有限公司基于 AWS 云服务的特定经营性资产（包括不限于服务器等 IT 设备），并利用这些资产在中国境内提供并运营基于亚马逊云技术的云服务。

光环新网给出的原因是：**为了遵守我国法律法规，进一步提升公司运营的 AWS 云服务的安全性和服务品质。**

而针对网上传闻的“AWS 出售中国业务”的说法，亚马逊 AWS 也做了官方回应：

> AWS 没有出售中国业务。我们将全心承诺，确保中国客户继续收到 AWS 业界领先的云服务。中国法律不允许外国公司拥有或运营特定技术、提供云服务。为此，为了遵守中国法律，AWS 向光环新网——AWS 的长期中国合作伙伴和 AWS 中国（北京）区域的服务运营商——出售特定的物理基础设施资产。AWS 继续在全球拥有 AWS 服务的知识产权。我们对在中国拥有这一重要的业务，以及它在未来若干年的增长前景都感到非常兴奋。

![](.evernote-content/80E67230-5EAB-491C-A53E-FA994A851897/D25C965E-45F6-40D4-8399-BFF60CE92BF8)

### 

**◆****◆  ◆**

亚马逊与光环新网的一段往事

### 

相对于大名鼎鼎的亚马逊而言，光环新网的认知度其实不算太高。不过据雷锋网了解，其实光环新网成立于 1999 年，主要业务是提供宽带接入、IDC（Internet Data Center）及其增值服务等；它与亚马逊之间也合作已久。

2013 年 12 月 18 日，亚马逊 AWS 宣布正式进入中国市场，并于 2014 年初开始有限预览；由于监管政策的要求，AWS 通过“前店后厂”的方式落地，以北京为前店，宁夏中卫为后厂，分别与光环新网和网宿科技合作；光环新网和网宿科技提供 IDC 和 ISP，包括基础架构、带宽和网络功能。

![](.evernote-content/80E67230-5EAB-491C-A53E-FA994A851897/FB3F0021-1DC1-4A02-8683-AAB548129242.jpg)

然而到了 2016 年 8 月，光环新网却又发布公告称，与亚马逊达成了新的运营协议，在北京地区（北京及周边地区）进行云服务基础设施和云服务平台的运营，并向中国客户提供 AWS 云服务；而光环新网根据约定向亚马逊付费，作为亚马逊提供服务和支持及授权的对价。

换句话说，最初也曾经与亚马逊达成合作协议的网宿，已经完全出局；而光环新网成为亚马逊 AWS 北京区域的唯一运营商。

值得一提的是，亚马逊之所以选择将北京区域的 AWS 来运营，是因为国内监管环境下的政策问题。之前在 2013 年底宣布入华之后，虽然有与国内 IDC 公司的合作，但亚马逊也在 2014 年被曝出涉嫌在华打政策擦边球，规避政策限制来扩大云服务销售。有知情人士表示，亚马逊一直坚持开展市场经营活动，只是把原来客户开的发票改成光环新网。

甚至在 2016 年 5 月，北京市通信管理局还曾对亚马逊下达《整改通知书》，通知中指出，亚马逊没有取得相关资质，要求亚马逊云服务在华停业整改。于是在经过一番基于监管要求的整改之后，2016 年 8 月，**光环新网得以成为亚马逊 AWS 在北京区域的唯一运营商。**

### 

**◆****◆  ◆**

不过云数据依然是一个问题

### 

合作协议的达成，似乎意味着亚马逊 AWS 真正拿到入华的通行证了。毕竟，基于网络信息安全的考虑，国家规定 经营增值电信业务的外商投资电信企业外资占股不得超过 50%，而且还必须得拥有牌照；而目前的 IDC 牌照拥有者，都只是国内的互联网中心服务企业；而光环新网就是这些企业中的一个。

换句话说，与光环新网的合作，似乎是亚马逊 AWS 的无奈之举。

![](.evernote-content/80E67230-5EAB-491C-A53E-FA994A851897/03BCA9BB-F162-415B-846C-824CE929D964.jpg)

不过据知情人士透露，虽然与光线新网达成了合作关系，但是持有 IDC 牌照的北京光环新网只提供机房，不直接参与实际运营，用户使用亚马逊中国服务费用，90% 支付给亚马逊，剩下10% 作为机房租金付给光环新网。

换句话说，提供云技术服务的所有数据，依然掌握在亚马逊这样的外资企业手中；这种情况是与国家基于安全考虑提出的相关规定相悖的，也存在一定的法律风险。但对于亚马逊 AWS 来说，能够落地已经是求之不得了。

![](.evernote-content/80E67230-5EAB-491C-A53E-FA994A851897/C391FAC1-976C-486B-B095-E0DAFCFB19AB.jpg)

对于光环新网来说，与亚马逊的合作带来的除了风险，还有收益。根据国信证券在 2017 年 7 月 20 日发布的研究报告，2016 年第四季度，AWS 中国区的收入环比增速达到了 10%，单月营收接近 1 亿元；而光环新网从中获取的收入主要来自基础设施和增值服务两个部分。

**但问题在于，即使是 10% 的环比增速，亚马逊 AWS 也没有能够在国内实现盈利。**它与光线新网的合作更多的是基于市场规则，光线新网为 AWS 提供基础资源和基础配置，亚马逊付钱给光线新网，因此后者拥有一定的利润空间。根据国信证券此前的预计，光线新网能够在 2017 年从 AWS 获取 12 亿元的收入，净利润为五到六千万元。

### 

**◆****◆  ◆**

AWS 在中国发展的真正阻碍

### 

2017 年 11 月 1 日，著名研究机构 IDC 发布了 2017 年上半年中国公共云 IaaS 的市场份额调查报告，报告显示，中国公共云 IaaS 市场排名前五的分别是阿里云、腾讯云、金山云、中国电信和 Ucloud，而亚马逊 AWS 却榜上无名。换句话说，**亚马逊 AWS 的 IaaS 在中国上半年的营收甚至没能达到 Ucloud 的 5774 万美元，份额也低于后者的 5.5%。**

PaaS 领域同样也是如此，亚马逊 AWS 在中国区榜上无名，份额低于第五名 IBM 的 4.6%。

![](.evernote-content/80E67230-5EAB-491C-A53E-FA994A851897/517C9803-3510-46D8-9222-9DC563A4722E.jpg)

与之形成鲜明对比的是，亚马逊 AWS 在全球市场一个季度的营收就是 40 多亿美元[（戳蓝色字体参看雷锋网此前对亚马逊财报的解读）](http://mp.weixin.qq.com/s?__biz=MTM2ODM0ODYyMQ==&mid=2651430790&idx=1&sn=b7dd4bd84552583a978bfefd00920d41&chksm=624d9f24553a16328865cf5450dc3cce0a2bccbf3734bf98e745518b7ab3b4dc2445ce4528de&scene=21#wechat_redirect)；而且它占据了全球市场 40% 的份额，是 Google、微软、IBM 三家公司加起来的两倍。如此鲜明的对比背后，实际上是亚马逊 AWS 在中国市场的无奈和尴尬。

据雷锋网了解，虽然亚马逊 AWS 在中国市场的用户需求增长很快，单月环比一度达到了 10%，但是受到政策限制，亚马逊在 2017 年基本没有扩展新用户，只有一些技术方面的推广。而亚马逊方面也就数据海外隔离等问题与工信部进行了反复沟通。

另外一个问题是，为了推动 AWS 对中国的覆盖，亚马逊曾经在宁夏投资了三个数据中心，分别位于中卫工业园、迎水桥和宣和寺子口三个地方，而且已经在 2016 年完成建设。但是由于相关监管政策仍未落地，宁夏数据中心仍未开展运营。**而光环新网为了争取 AWS 在宁夏 IDC 的运营权，在宁夏也成立了子公司。**

有消息称，亚马逊方面曾提出让光环新网作为宁夏中卫数据中心的运营合作方，这让宁夏相关政府部门大为不满。这也是原本于 2017 年 3 月开业的宁夏中卫以及中国 AWS 业务至今音讯全无的原因。

另外，2017 年 6 月 1 日，《网络安全法》正式执行，这部法律对于网络基础设施、网络信息和网络运营企业进行了严格监管，原则上禁止外国企业在华经营数据中心和网络新闻服务。对于亚马逊 AWS 来说，也许是其在中国市场发展的一道根本无法逾越的阻碍。

![](.evernote-content/80E67230-5EAB-491C-A53E-FA994A851897/A4E53B51-F219-4A9F-A6E4-755FF523AE18.jpg)

到了 2017 年 7 月的可信云大会上，云服务作为新的服务模式加入到了新版《电信业务分类目录》中，因此对于原本持有 IDC 牌照的企业，如果事实上已经从事了云服务或 CDN 服务，工信部要求以书面承诺的形式承诺在年底之前达到相应的经营许可要求，并获得相应的经营许可证。如果未按期承诺的，从 2017 年 4 月 1 号起，不得经营相应的业务。如果已承诺，按照承诺期限，年底之前未持证的企业，2018 年 1 月 1 日起，不得经营相应的业务。

**也就是说，如果要想运行 AWS 业务，光环新网还需要在 2018 年 1 月 1 日之前获得云服务牌照。**

雷锋网了解到，实际上光环新网已经在 2017 年 9 月之前向工信部提交了材料，目前正处于工信部的审批流程中。眼下，亚马逊将基于 AWS 云服务的特定经营性资产作价 20 亿元卖给光环新网，也可以说是为了获得监管部门的认可而再次做出的妥协之举；但从长远来看，要想在中国站稳脚跟，这可能是亚马逊 AWS 不得不做出的选择。

毕竟，20 亿元人民币对于亚马逊来说是个小数字，但也基本上抵得上 AWS 过去在中国区一年的营收了。
