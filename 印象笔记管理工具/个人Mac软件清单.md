说明
--

1. 1. 部分为付费APP，对应也有很多免费开源产品选择，但APP最终是看输出价值，这些APP对我的工作/生活有很大的帮助，买单还是值得的。
2. 2. 本着能省则省，部分付费APP，采用开源License申请解决。
3. 3. 在经济可接受的范围内，能买则买，支持正版。
4. 4. 不定期更新，So部分APP会被替代
5. 5. 在不断的选择和打磨APP时，我有一点感受，APP之于我们，就如同宝剑之于剑客，不在多，而在顺手和锋利。所以挑选出合适自己的，深入了解学习，最终提升自己的效率，让工作生活因之变得更美好

清单
--

### Alfred

* • **功能**：效率神器，系统主入口
* • **资费方案**：付费
* • **备注**：推荐购买 Mega 版，作者持续更新，社区活跃

### Dash

* • **功能**：文档检索，API 查询神器
* • **资费方案**：付费
* • **备注**：升级价格较高，建议无特殊需求时不升级,v7开始Dash已更新为订阅模式。

### Moom

* • **功能**：窗口管理
* • **资费方案**：付费
* • **备注**：无

### Bartendar

* • **功能**：菜单栏应用图标管理工具
* • **资费方案**：付费
* • **备注**：无

### Fantastical

* • **功能**：日历应用，支持自然语言输入和日历文件订阅
* • **资费方案**：付费
* • **备注**：无

### 1Password

* • **功能**：密码管理
* • **资费方案**：付费
* • **备注**：在密码管理方面表现强大

### Telegram

* • **功能**：安全通信，迄今为止最棒的IM工具
* • **资费方案**：免费
* • **备注**：机器人/频道都很强大

### 微信

* • **功能**：办公IM
* • **资费方案**：免费
* • **备注**：无

### 微信读书

* • **功能**：电子书阅读
* • **资费方案**：免费/付费
* • **备注**：搭配ZLibrary，可以弥补微信读书资源不足的缺陷

### Chrome

* • **功能**：浏览器及前端开发调试神器
* • **资费方案**：免费
* • **备注**：无

### Typora

* • **功能**：Markdown 编写，目录功能
* • **资费方案**：付费/买断
* • **备注**：搭配iCloud实现多设备同步

### keka

* • **功能**：解压缩，支持加密码
* • **资费方案**：免费
* • **备注**：官网免费，AppStore付费

### Things

* • **功能**：GTD，好用的todolist
* • **资费方案**：付费
* • **备注**：无

### PostMan

* • **功能**：网络调试
* • **资费方案**：免费
* • **备注**：无

### iTerm2

* • **功能**：终端
* • **资费方案**：免费
* • **备注**：无

### Snipaste

* • **功能**：截图
* • **资费方案**：免费
* • **备注**：无

### Affinity Photo 2

* • **功能**：平面设计，图片编辑
* • **资费方案**：付费/买断
* • **备注**：官网购买即可，限制平台，不限制设备数，对比PS还是弱些，但够用

### Windows App（原Microsoft Remote Desktop）

* • **功能**：远程桌面
* • **资费方案**：免费
* • **备注**：国区提供，不需要访问外区商店也可下载

### Eudic 欧路词典

* • **功能**：词典
* • **资费方案**：付费/买断
* • **备注**：无

### FileZila

* • **功能**：FTP
* • **资费方案**：免费
* • **备注**：选择第一个版本tar.bz2格式下载后解压即可

### Karabiner-Elements

* • **功能**：改键器
* • **资费方案**：免费
* • **备注**：无

### Spotify

* • **功能**：听歌App
* • **资费方案**：订阅
* • **备注**：免费版也可以听，功能受限且有广告

### iMazing

* • **功能**：iOS设备管理/文件管理
* • **资费方案**：付费
* • **备注**：无

### WebStorm

* • **功能**：IDE，利用插件进一步提升生产力
* • **资费方案**：付费
* • **备注**：开源项目 License

### DataGrip

* • **功能**：DataBase，对比navicat更为高效，功能丰富，也支持插件开发，拓展性强。同时与IDEA设计体验一致，学习成本低
* • **资费方案**：付费
* • **备注**：开源项目 License

### Cursor

* • **功能**：轻量级文本编辑,AI辅助
* • **资费方案**：付费
* • **备注**：如果没AI，完全可以使用VSC。

### Docker

* • **功能**：管理容器应用
* • **资费方案**：免费
* • **备注**：无

### Draw.io

* • **功能**：绘制流程图
* • **资费方案**：免费
* • **备注**：也有WEB版

### Screen Studio

* • **功能**：录屏
* • **资费方案**：付费
* • **备注**：有免费平替，相对差点。
