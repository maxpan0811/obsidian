# 如「 Alt + Tab 」般得心应手：Pintasking

---

![](.evernote-content/CABFB9CE-842A-4042-9672-DDA9170D0F0F/A25BDF91-7AE2-45BD-9B09-29C32C008625.jpg)

得益于硬件水平的不断提升，我们的移动设备似乎已能胜任大多数情况下的多任务处理工作。但内存大并不代表体验佳，尽管我们几乎能在任何时候便捷地呼出多任务管理界面，当后台任务堆积过多时，不手动进行一番清理我们依旧很难在诸多任务中进行快速定位和切换。

如何才能专注于某几项特定的任务而不受多余的干扰，Pintasking 或许能给我们一个完美的解决方案。

对「悬浮窗」的又一次「再创作」
---------------

我曾经向 [@Jeffery.Du](http://matrix.sspai.com/user/693015)推荐过一款名叫[BuMP Music Player](http://sspai.com/31726)的音乐播放器，这款播放器利用了悬浮窗的特性，在保证应用自身体积轻量、小巧的同时还能在最大程度上降低我们在阅读、工作以及浏览网页时的注意力分散状况（类似的应用还有[Link Bubble](http://sspai.com/24985)以及 [Flynx](http://sspai.com/31440) ）。

Pintasking 则更加「会玩儿」一点，利用悬浮窗，Pintasking 可以将我们的屏幕边缘变成两条灵活的「任务栏」。

![](.evernote-content/CABFB9CE-842A-4042-9672-DDA9170D0F0F/C34F4492-7555-456B-BB17-56E3FBE77D24.jpg)

![](.evernote-content/CABFB9CE-842A-4042-9672-DDA9170D0F0F/C08A22D2-06CC-4411-9076-A82803B13BED.jpg)

在系统设置的「辅助功能」（ Lollipop 5.1 中又叫「无障碍」）中开启 Pintasking 服务后（已 root 用户可安装内置的 System Helper 来获得最佳体验），Pintasking 就会开始在后台保持运行。**在使用过程中，我们只用把那些需要集中注意力解决的任务通过特定操作****「钉」（ pin ）在屏幕边缘即可，次要任务则可以忽略不管**。

这里的「特定的操作」有三种：

* 点击下拉通知栏中常驻的「 Tap to pin 」；
* 在设定好的区域（如状态栏中部）滑动触发（ SwipePad ）；
* 长按 Home 键选择「 Pin this task 」（在带有虚拟屏幕按键的设备上为按住 Home 键上滑）。

![](.evernote-content/CABFB9CE-842A-4042-9672-DDA9170D0F0F/66E0C673-5F84-4345-B18E-73860E062EB9.gif)

![](.evernote-content/CABFB9CE-842A-4042-9672-DDA9170D0F0F/8FF91FC1-8FA7-4A95-AF5F-F38B2E495E61.jpg)

以具体的使用场景为例：当我在 Feedly 看到一条值得翻译过来并推送至微博的消息时，只需在 Feedly 、需要用到的翻译工具以及微博这三个应用中把它们「钉」到屏幕边缘，然后便可以通过悬浮窗在这三个任务间快速切换。

![](.evernote-content/CABFB9CE-842A-4042-9672-DDA9170D0F0F/8B6EF395-7E32-4ED9-99F2-811D5EA426A1.gif)

![](.evernote-content/CABFB9CE-842A-4042-9672-DDA9170D0F0F/FA5EABD9-E1EA-4E34-A44A-077FAC7C564C.gif)

这样的多任务处理方式的好处是显而易见的——一方面，将需要处理的任务以悬浮窗的形式放在屏幕一侧，我们就无须在系统最近任务中进行查找和定位，节省时间、提高效率；另一方面，由于悬浮窗默认是全局置顶的，即便我们的注意力不得不暂时被其他应用分散和打扰、亦或是熄屏离开后重新点亮屏幕，悬浮窗的存在都能时刻提醒我们还有待处理的任务。

![](.evernote-content/CABFB9CE-842A-4042-9672-DDA9170D0F0F/A67C6FA7-B3A7-44D4-8523-BADC0B2E5604.jpg)

![](.evernote-content/CABFB9CE-842A-4042-9672-DDA9170D0F0F/73A76717-F329-4B55-A955-7586B444DEC7.jpg)

用细节定义完美
-------

在上手一段时间后，我最终没忍住选择了内购解锁 Pintasking Pro ，专业版强大而细致的设置功能让 Pintasking 成为了我手机里又一款不可或缺的效率工具。下面就简单罗列一些：

* **自定义悬浮窗大小以及悬浮窗被屏幕边缘遮挡的范围**；
* **自定义悬浮窗任务的自动消失条件**。可以设置「熄屏后自动关闭所有悬浮窗任务」，也可以勾选「只允许手动关闭悬浮窗任务」；
* **设置悬浮窗任务自动停靠区域**。可禁用上、下边缘以避免误操作；
* ……

浏览器插件：「 Link Bubble 什么的通通下岗吧。」
------------------------------

不得不提的是，除了多任务处理，Pintasking 还提供了一个叫做「浏览器扩展」（ Browser Extension ）的功能。

该功能看似与 Link Bubble 、Flynx 相类似，实则有很大的区别—— Link Bubble 与 Flynx 可以看作是独立的浏览器，**而 Pintasking 则需要一款其他的浏览器作为其扩展对象**：当我们点击第三方应用中的链接时依旧需要其他浏览器来打开这个链接，随后 Pintasking 会在浏览器右下方显示一个工具栏，点击工具栏上的图钉按钮，这个网页就会被「 Pin 」到悬浮窗并自动切换到第三方应用中方便我们继续浏览。

![](.evernote-content/CABFB9CE-842A-4042-9672-DDA9170D0F0F/6A30DC0B-F9FC-42C7-8D56-41A431AD11B5.jpg)

![](.evernote-content/CABFB9CE-842A-4042-9672-DDA9170D0F0F/9263D2C9-FD54-499A-9874-4FD9E9AA9454.gif)

**我们也可以这个「浏览器扩展」功能看作是一个「网页收集器」**，将我们在使用其它应用的过程中所打开的网页集中起来在后台打开以保证体验的流畅性。这一点与 Link Bubble 有异曲同工之处。

不仅能高效地帮你处理好多任务，还能「另辟蹊径」抢下 Link Bubble 的饭碗。在 Android 悬浮窗这个领域里，Pintasking 算是会玩儿的了。

你可以在 [Google Play Store](https://play.google.com/store/apps/details?id=com.phinxapps.pintasking) 上获取这款应用。
