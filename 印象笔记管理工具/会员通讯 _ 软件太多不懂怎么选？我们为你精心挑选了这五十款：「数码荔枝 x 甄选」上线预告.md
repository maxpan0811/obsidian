本文首发于「数码荔枝」
-----------

复制本段文字并打开淘宝 App，使用￥lizhi4you￥访问「数码荔枝」商城浏览我们在售的全部正版软件。

众所周知，「数码荔枝」长期专注于分享最新最优质的正版软件。经过数年的发展与努力，「数码荔枝」已经拥有了涵盖 macOS 和 Windows 平台在内数百款国内外优质软件的销售代理权，为近十万用户提供了更为便捷的正版软件购买体验。

但是，随着「数码荔枝」在售软件的数量和品类不断增加，我们强烈感受到，除了部分资深的软件玩家，大部分用户的需求其实很简单：找到一款工具，以便快速解决生活或工作中的实际问题。而非「深度对比多个同类工具之后再选出更适合自己的那款」。

于是，我们开始思索：如何为更多用户群体提供更具针对性的服务？显然，仅仅是一个「数码荔枝正版软件商城」，已无法满足各个细分人群的需求。

全新的「数码荔枝 x 甄选」
--------------

因此，我们决定，将现有的「数码荔枝正版软件商城」更改为全新的「数码荔枝 x 甄选」，我们会从「数码荔枝」所有代理的优质软件中，再精心挑选 50 款左右的软件，通过结合几种最常见的使用场景，打包呈现给大家。不管你是忙碌的上班族、还是有选择困难症，又或者是 Mac / Win 新手，我们希望，「数码荔枝 x 甄选」都能让你在最短的时间通过正版软件，体会到「解决问题」的快乐。

同时，我们还会创建新的「数码荔枝 x 集市」商城，这里会延续你熟悉的「数码荔枝」的风格：我们谈下的所有优质软件均会在「数码荔枝 x 集市」出售，并保持「数码荔枝」原有每月 5 – 10 款软件的上新频率。如果你不愿意被预设的场景束缚，想要发现更多有趣实用的软件，来逛逛我们的集市店，准错不了。

上述提到的「数码荔枝 x 甄选」和「数码荔枝 x 集市」，都会在一周左右的时间内完成准备并正式上线，请各位关注「数码荔枝」的朋友们拭目以待。

好软件，我们为你甄选
----------

不同的是，「数码荔枝 x 甄选」不仅仅只是在软件数量和质量上有变化的「浓缩版数码荔枝商城」，考虑到不同职业、不同人群之间的不同需求，我们还为你精心挑选了 macOS 和 Windows 2 个平台大类，7 个生活或工作中常见使用场景下，我们认为最优质的那一款软件，方便你在最快的时间内使用它们解决问题。

**买了新 Mac / 电脑**

第一次接触 macOS 或 Windows，相信很多人都为不知道要装什么软件发过愁，也有很多 Windows 用户感叹应用没有 macOS 下的精美，而同样还有 macOS 用户认为 Windows 更有生产力……

这个板块就是为了让你不再羡慕其它平台下那些或精美、或强大的软件，这里有 macOS 上优秀的清理工具 CleanMyMac，也有 Windows 系统优化利器 Process Lasso，我们之前介绍过的下载工具 Folx 和 IDM 也包含在其中。买了新电脑的你，一定不要错过这个板块。

![](.evernote-content/50115CBC-5F83-4E52-B127-909727CF7858/6C080415-37EB-42FD-9075-76CCAE3BBB1D)

**常和文字打交道**

不管你是专业的文字工作者，还是因为兴趣经常写文章，挑一个好的写作工具准没错。这个板块里有 macOS 知名的 Markdown 写作工具 Ulysses 以及国内开发者作品 MWeb，也有强大的双平台长文写作工具 Scrivener。经常和文字打交道的你，一定能在这里找到适合自己的那款写作工具。

![](.evernote-content/50115CBC-5F83-4E52-B127-909727CF7858/511F52AF-7C0B-4352-8A8D-37919D8AE281)

**还是个学生**

顾名思义，如果你还是个学生，或热爱学习，你不仅能在这里找到全能的 PDF 阅读、编辑工具 PDFelement、MarginNote 以及专业的数学公式软件 AxMath 等，也能找到读书管理利器 Knotes。当然，除了学习，生活也别耽误，MoneyWiz 能帮你厘清收支，合理安排预算，让你真正做到学习、生活两不误。

![](.evernote-content/50115CBC-5F83-4E52-B127-909727CF7858/F02F12F5-39A0-42B9-93AC-72CF02C9C970)

**需要处理图像 & 音视频**

你可能是设计师，对图片素材的收集和整理有需求，这个板块中的 Zoommy、Eagle 都是不错的选择；如果你需要标注图片，Snagit 能帮你大大提高效率。对于经常要处理音视频的你，Downie、Permute 也能助你一臂之力。

![](.evernote-content/50115CBC-5F83-4E52-B127-909727CF7858/4C120976-FC2E-4D3E-89F6-D2A48455C27E)

**更高效使用电脑**

做一件事，不仅要做成，而且还要做好。这个板块的目的就是让你在使用电脑的过程中，能够以更高效的方法和更少的时间来达到目的，比如让你更方便快捷同步文件的 Good Sync、macOS 上的解压缩利器 BetterZip 以及 Windwos 文件搜索工具 Listary Pro 等，我们相信，这些工具能够让你在使用电脑时事半功倍。

![](.evernote-content/50115CBC-5F83-4E52-B127-909727CF7858/5BCE471B-B53C-4209-9FE5-C215384B03CC)

**让电脑更安全流畅**

让电脑更安全流畅，想必是人人都希望实现的事。要实现这个目的，除了上文提到的 CleanMyMac 和 Process Lasso，广告拦截工具 Adguard、网络调试工具 Proxifier 也能有所帮助。

![](.evernote-content/50115CBC-5F83-4E52-B127-909727CF7858/AFA1DA5B-AAC1-4BEC-B314-D82F75E379F4)

**如果你是产品、开发岗位**

当然，「数码荔枝 x 甄选」的板块怎能少的了各位产品经理和开发者，我们也为你精心挑选了开发中可能会用到的利器：Git 管理工具 Git Tower、专业抓包工具 Charles 以及 iOS 界面调试利器 Reveal，你只需负责代码，它们会为你最好地呈现你的成果。

![](.evernote-content/50115CBC-5F83-4E52-B127-909727CF7858/15DE52F5-798D-48B1-9F72-1BAE1A90C170)
