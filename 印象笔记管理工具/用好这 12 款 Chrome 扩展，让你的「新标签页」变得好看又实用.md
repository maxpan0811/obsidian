# 用好这 12 款 Chrome 扩展，让你的「新标签页」变得好看又实用

---

打开 Chrome 浏览器的新标签页，或者在浏览器地址栏输入`chrome://newtab`，我们可以看到这样的默认新标签页页面：

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/728DD40A-510A-46FF-BD32-59C695DF5927.jpg)

如果你厌倦了官方新标签页页面仅显示最常访问网页和搜索栏，希望更高效利用好这块「一亩三分地」，让浏览器新标签页实现：

* 快速记录备忘
* 显示日历、天气等实用信息
* 单纯显示时间信息
* 每天更换壁纸图片
* 快速访问你最常访问的网页、书签和历史记录
* 实现网页导航页面
* ......

那么，这份 Chrome 新标签页扩展合辑能够满足以上需求，让你在新标签页玩出一片天。

Speed Dial 2
------------

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/8C821609-1C12-4258-80D2-3A9BB6935F14.jpg)

Speed Dial 2 作为目前个人使用的新标签页扩展之一，初始安装好之后重新打开新标签页，可以看到默认 3×2 排列的书签列表，以精美的图标呈现出来。

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/BA789287-C693-4E1A-AB7E-33750A75247F.jpg)

用户可自行调整书签列表的列数、间距、宽度等排列方式，如果需要有更好看的图标显示方式，Speed Dial 2 提供了多种自定义网站缩略图的设置选项，包括自动检测网站 ico、主页快照、外部图片链接或者本地上传图片（我派网站的很醒目喔）。甚至在自定义外观的设置页面，用户可以对书签缩略图的圆角、背景色、书签标题的文字样式、是否显示访问次数等有更详细的控制。

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/7C2AD0AB-CFC1-4C54-80C6-8052D85948B0.jpg)

进入 Speed Dail 2 的设置页面，可以挖掘更多玩法。

* 创建组的管理方式。Speed Dail 2 提供了组的方式来管理不同类型的书签，用户点击右键会看到创建新小组的入口，即可创建不同主题的书签合辑，方便添加不同类型的网站。
* 丰富的侧边栏信息。用户可设置 Speed Dail 2 侧边栏显示最近关闭的标签、浏览器书签、已安装 Chrome Apps 的快捷方式。
* 有趣的访问统计。Speed Dail 2 可统计用户最常访问网页占用的时间，并以饼图的形式呈现数据。
* 备份和恢复。在设置好 Speed Dail 2 各种选项后，扩展本身提供了导入导出设置，方便以后恢复，用户还可付费解锁（4.99 美元）同步和备份的功能，包括云同步 Speed Dail 2 书签和设置、在线访问书签、移动设备访问以及更多自定义选项。

下载地址：[Chrome Store](https://chrome.google.com/webstore/detail/speed-dial-2-new-tab/jpfpebmajhhopeonhlcgidhclcccjcik) | [Crx4Chrome](https://www.crx4chrome.com/extensions/jpfpebmajhhopeonhlcgidhclcccjcik/)

homenewtab
----------

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/584D146B-37FA-408B-98F9-43DDB9BA0C4E.jpg)

homenewtab 扩展默认设置会在页面显示多个功能模块，提供了提醒通知、快速备忘记录、天气时间、本地已安装的 Google Apps、搜索栏，homenewtab 还支持自行添加 New App、快速显示最近关闭的网页、更换壁纸，初上手会有点略微臃肿的感觉。不过经过使用之后发现了 homenewtab 三大特色功能：

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/F2245749-9E94-4B42-885C-E3AC9E95CD32.jpg)

* 自定义添加 New App。用户将需要添加的网页地址复制粘贴后，homenewtab 可以自动检测到 Icon 图标，添加后会以 Apps 形式显示在扩展右侧区域。

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/32A2E112-02C2-4E5D-A953-793E55CE3990.jpg)

* 多个网络服务通知。目前，homenewtab 支持 Google Calendar、Yahoo Mail、Hotmail (Outlook)、Facebook Messeage 服务的通知，有新邮件或者新信息将会在对应 Icon 显示通知角标，并且在左侧提醒通知区域预览对应信息。

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/504B1995-F5C3-4D85-BA5A-6ECBF0F4399D.jpg)

* 聚焦清爽模式。如果不喜欢 homenewtab 默认的显示效果，可以尝试 Focus mode 清爽模式，开启后界面仅显示时间日期以及搜索栏（设置选项可打开或者关闭）。

下载地址：[Chrome Store](https://chrome.google.com/webstore/detail/home-new-tab-page/ehhkfhegcenpfoanmgfpfhnmdmflkbgk) | [Crx4Chrome](https://www.crx4chrome.com/extensions/ehhkfhegcenpfoanmgfpfhnmdmflkbgk/)

Momentum
--------

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/FEF1D26B-C0EE-41A1-B36F-D46784F38465.jpg)

Momentum 是个人安装的新标签页扩展之二（另外一款就是上面提到的 Speed Dial 2），喜爱的原因就是官方精选的壁纸图片，用户还可以付费解锁实现自定义壁纸图片。

另外，个人经常使用的是 Momentum 提供的 Todo 备忘录功能，方便在浏览网页的时候，如果遇到优秀的素材做临时的记录；或者个人定下需要完成的事项，每次打开浏览器的时候将会第一时间看到任务清单。

Momentum 的其他小功能包括了经典名句、显示系统时间、搜索栏、天气、自定义网站链接、设定今天的目标。

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/79C45DDB-467C-41D7-A353-B8A010D87AEE.jpg)

当然，如果用户付费解锁 Momentum 高级功能的话（年付方案每月 2.5 美元），Momentum 提供的功能将会更加丰富。提供了多项功能：同步第三方任务管理软件（Asana、Trello、Todoist、Wunderlist、Google Tasks）、创建多个 Todo 集合等等。

下载地址：[Chrome Store](https://chrome.google.com/webstore/detail/momentum/laookkfknpbbblfpciffpaejjkokdgca) | [Crx4Chrome](https://www.crx4chrome.com/extensions/laookkfknpbbblfpciffpaejjkokdgca/)

远方 New Tab
----------

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/1B609DEC-F32C-438C-A07F-44177C42E45B.jpg)

喜欢更加丰富背景图片的用户，远方 New Tab 扩展会是不错的选择。远方 New Tab 提供了 7 个不同的图片源，包括远方官方源、Google Earth、Google Art、必应、500px、flicker、Unsplash，支持设置图片源显示顺序、更新间隔时间，提供直接保存图片到本地的功能。

其他包括显示天气时间、搜索栏、浏览历史的功能，远方 New Tab 扩展一个也没落下。

下载地址：[Chrome Store](https://chrome.google.com/webstore/detail/dream-afar-new-tab/henmfoppjjkcencpbjaigfahdjlgpegn) | [Crx4Chrome](https://www.crx4chrome.com/extensions/henmfoppjjkcencpbjaigfahdjlgpegn/)

Earth View from Google Earth
----------------------------

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/43487C0E-698A-4BEE-904E-A0A12C147C86.jpg)

来自 Google 官方出品的扩展，Earth View 最大的特色就是提供了相当宏观的地球卫星图，每次打开新标签页都被这样的图片所震撼，2.0 版本新增加了 500 张地球卫星图。

Earth View 提供了下载壁纸、更换壁纸的历史记录、复制壁纸链接或者分享至 Google+、Twitter 和 Facebook 的社交平台，页面右下角小地球还会自动显示壁纸拍摄地的信息。

下载地址：[Chrome Store](https://chrome.google.com/webstore/detail/earth-view-from-google-ea/bhloflhklmhfpedakmangadcdofhnnoh) | [Crx4Chrome](https://www.crx4chrome.com/extensions/bhloflhklmhfpedakmangadcdofhnnoh/)

Wallcat
-------

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/5365FBD7-355F-46DA-A90A-ED29B7A97A1D.jpg)

喜欢纯粹将新标签页换成漂亮的壁纸，Wallcat 就是一个相当好的选择。Wallcat 官方提供了四组不同的精选图片合辑：Strcture、Fresh Air、Gradients、Northern Perspective。另外 Wallcat 支持显示用户最常访问的八个网页列表。

下载地址：[Chrome Store](https://chrome.google.com/webstore/detail/wallcat/gfnglnppkpcidjclgbokfckkijgmjhib) | [Crx4Chrome](https://www.crx4chrome.com/extensions/gfnglnppkpcidjclgbokfckkijgmjhib/)

Currently
---------

如果用户喜欢将电脑屏保设置成经典的时钟翻页屏保 Fliqlo，相信 Currently 也会是你的菜。安装好扩展之后，你的新标签页是这样显示的:

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/0628FC7D-525B-4D65-B539-8F5EBC417A10.jpg)

Currently 采用了纯色块背景+扁平化天气图标的显示效果，新标签页简洁地显示了时间（读秒效果相当不错）、日期、天气，整个界面没有任何的干扰元素。在 Currently 后台的主题更换界面，提供了两套免费的主题，其他四套主题需要付费解锁。

下载地址：[Chrome Store](https://chrome.google.com/webstore/detail/currently/ojhmphdkpgbibohbnpbfiefkgieacjmh) | [Crx4Chrome](https://www.crx4chrome.com/extensions/ojhmphdkpgbibohbnpbfiefkgieacjmh/)

New Tab Clock
-------------

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/5EC773E6-FA99-4A66-BA22-91B100E7F2CF.jpg)

New Tab Clock 可以满足用户更简单看时间的方式，虽然没有 Currently 读秒的显示效果，但是它的界面没有其他元素的干扰，「老老实实」在新标签页面显示系统时间和日期，用户可以更改（通过设置选项或者右键）背景颜色、字体样式以及 12/24 小时制。

下载地址：[Chrome Store](https://chrome.google.com/webstore/detail/new-tab-clock/ljpapphpgkmigobbbakmnfoohclifanm) | [Crx4Chrome](https://www.crx4chrome.com/extensions/ljpapphpgkmigobbbakmnfoohclifanm/)

Dayboard: New Tab & Site Blocker
--------------------------------

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/9F33D66B-332B-4553-97C6-42F6743906E1.jpg)

Dayboard 的功能设计主要是围绕任务完成，用户可以在页面中添加目标任务内容、链接、任务日期。为了帮助用户专注完成任务，Dayboard 提供了 Blocked Sites、Refocus Tabs（免费试用 30 天或者付费永久解锁）、Goofy Mode 三大功能选项。

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/85179B21-1FBC-4648-8271-3AD5FD212346.jpg)

其中有意思的就是 Blocked Sites 禁止访问功能，用户可以自行添加禁止网站列表，如果 Dayboard 中有未完成的任务清单，禁止网站每天仅有默认 10 分钟的浏览时间，直至任务完成。

下载地址：[Chrome Store](https://chrome.google.com/webstore/detail/dayboard-new-tab-site-blo/kimodcegbhclamjcbifgfaldeengbgij) | [Crx4Chrome](https://www.crx4chrome.com/extensions/kimodcegbhclamjcbifgfaldeengbgij/)

iChrome
-------

希望将新标签页打造成功能齐全的页面，iChrome 会是很不错的选择。

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/08EB8368-C0D9-4E56-8EDC-F38DB32FF6C0.jpg)

iChrome 更像是 Start 扩展的高级版，用户可以自由定制不同的主题和功能小挂件，官方提供了天气、经典名句、新闻列表、体育赛事比分、股票、待办事项、时钟提醒、翻译、顶栏搜索条以及快捷应用入口（这里有大量可定制化的第三方小部件）。经过一段时间的体验之后，iChrome 在这几方面给我留下了深刻印象：

* 添加多个定制化的标签页。用户可自行增加多个标签页，每个标签页都有自己的小部件，彼此独立不影响使用，只需点按屏幕两侧的按钮或者使用键盘左右方向键即可切换不同标签页。
* 支持 OK Google 语音搜索方式。iChrome 提供的搜索栏可以使用 OK Google 语音启动，解放键盘输入的方式。
* 多个强大的小部件。iChrome 与其他新标签页扩展相比，强大的地方就是提供 43 个相当实用的小部件，比如 RSS 订阅、Pocket 订阅列表、Pushbullet、Gmail 未读邮件列表、货币转换、计算器、系统状态等等，用户只需将部件拖拽到页面中即可。

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/1F55F24F-F3DC-49A4-862C-03EE5A10D093.jpg)

用户还可以付费（2 美元 / 月或者 20 美元 / 年）解锁 iChrome 动态背景、最大化小部件、自定义右栏链接、精确到小时的天气预测等高级功能。稍微不好的体验是，iChrome 会在底部显示横条广告，用户需要点击关闭直至下次 Chrome 浏览器打开，或者付费去广告。

下载地址：[Chrome Store](https://chrome.google.com/webstore/detail/ichrome-new-tab-a-fast-pr/iccjgbbjckehppnpajnmplcccjcgbdep) | [Crx4Chrome](https://www.crx4chrome.com/extensions/iccjgbbjckehppnpajnmplcccjcgbdep/)

Infinity
--------

![](.evernote-content/2EC4A41B-7330-4FEE-B50D-481CE589BCAC/CE830C07-3A9E-4060-9E20-2FBDBAEF4F3F.jpg)

Infinity 官方介绍称这是一款基于 HTML5 的 Chrome 扩展程序，扩展的界面与导航类网站比较相似，除了常规功能（显示天气、待办事项、书签、历史记录)，Infinity 还重绘了大量的网站图标（虽然看上去图标样式还不是很同意），集成了 200 多个热门常用图标，自定义网站列表有着简洁的显示效果，支持用户本地上传图标、调节图标的圆角度数和透明度、9 种不同的布局。

Infinity 特色功能还包括了 Gmail 邮件提醒、定制不同的搜索引擎、Chrome Apps 扩展管理。Infinity 还提供了高级付费解锁功能，提供备份、自动同步数据等付费功能。

下载地址：[Chrome Store](https://chrome.google.com/webstore/detail/infinity-new-tab/dbfmnekepjoapopniengjbcpnbljalfg) | [Crx4Chrome](https://www.crx4chrome.com/extensions/nnnkddnnlpamobajfibfdgfnbcnkgngh/)

其实你可以拥有不一样的新标签页
---------------

Chrome 浏览器强大之处在于用户可定制安装各类实用的扩展，而我们每次打开的新标签页也拥有很多相当不错的 Chrome 插件，这篇 Chrome 新标签页扩展合辑，将帮助你拥有不一样的小标签页，实现显示漂亮的壁纸、整合更多小部件等定制化功能，或许合辑中推荐的扩展还会有「漏网之鱼」，希望大家在评论区留言补充，推荐优秀 Chrome 扩展「人人有责」。
