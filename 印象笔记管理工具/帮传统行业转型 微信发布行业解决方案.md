# 帮传统行业转型 微信发布行业解决方案

---

昨天，微信发布了微信行业解决方案，具体体现在以微信公众号和微信支付为基础，帮助传统行业将原有的商业模式转移到微信平台上，并形成服务线上和线下的闭环。

![](.evernote-content/0FE7B22E-B43F-4067-8A34-F7AF84DB567D/BCAE09DC-25D2-4682-94CF-A46396234C6E.jpg)

　　微信所发布的微信行业解决方案涉及到多个行业，其中针对线下的解决方案涉及行业包括快递、售货机、百货、餐厅、便利店、超市、票务、酒店、景区、医院、停车场，针对线上的解决方案则包括B2C电商。

　　其实早在2014年8月，微信就推出了“微信智慧生活”全行业解决方案，并在全国范围内得到了广泛应用。

**以下是微信行业解决方案的案例举例：**

　　智慧票务：以微信公众平台和微信支付为基础，提供包括“微信扫码购票+微信支付+公众账号服务+社交传播”的能力，解决排队购票、取票耗时多，销售渠道单一，忠实用户少，运营成本高的问题。该解决方案可以实现微信电子票核销，消费者无需排队直接扫码入场，基于微信订阅、红包、Wifi等提供全方位的解决方案，用智能化管理降低人力、物力成本。

　　智慧百货：以微信公众平台和微信支付为基础，提供包括“微信公众平台 + 会员卡 + 微信支付 +虚拟货架 + 社交传播 + 扫一扫wifi + 多客服 + 企业号”的能力，帮助企业与顾客建立更好的连接平台，同时通过支付建立联系，积累顾客数据。

　　智慧餐厅：顾客通过微信自助完成排号点餐、催菜、买单等环节，节约成本;引导顾客将部分沟通工作通过微信完成，获得更高效的反馈;社交化的营销方式加上微信支付，利于提前锁定用户，促成交易。

　　智慧医院：通过线上线下相结合，优化现有医疗资源配置，完善医疗服务体系，提升医院内部管理效率，做到医疗对患者、对医院管理的智慧化升级。包括微信挂号，候诊提醒;微信支付诊间费用，电子报告微信实时送达;微信导航和指引;停诊消息即时微信推送;票据电子化并微信通知。

　　智慧停车场：将停车系统、商场、服务及用户连通互动，实现线上和线下的真正无缝融合。微信随时随地缴纳停车费，无需排队等待;通过微信自助完成查车、寻车、缴费，大量释放人力;知道顾客信息，可获取顾客造访状态，与顾客实时交流;社交化能力加上微信支付，提供多种异业合作可能。

**连接“人”与“服务”**

　　在今年的十二届全国人大三次会议上，全国人大代表、腾讯公司董事会主席兼首席执行官马化腾(微博)带来了四个建议，其中《关于运用移动互联网推进智慧民生发展的建议》格外引人瞩目。

　　马化腾认为，移动互联网将“人”与“公共服务”通过数字化的方式全面连接，将带动和促进交通、医疗、环境保护、公共安全等民生领域信息化的跨越式发展，对于优化社会资源配置、创新公共服务供给模式、提升均等服务水平、实现信息普惠全民具有重要促进作用。

　　微信行业解决方案的连接能力整合起来主要体现在以下三个方面：

　　1.移动互联网化。即以公众账号为入口，通过开放接口功能和微信支付，让商户在原有商业模式基础上实现信息化。同时在线下行业中，通过微信支付撬动商业闭环中的关键点，帮助商户沉淀消费者。

　　2.数据化。通过微信连接，商户可以了解到用户订单、产品销售情况、会员信息、付款金额等数据，能够借此降低成本。同时，通过系统化的收集用户消费习惯和偏好、客户满意度等数据，商户还可以找到新的突破口和增长点。

　　3.社交化。公众号可以对用户数据进行分析、梳理，为特定的用户群推送其需要的商品信息、服务内容、信息反馈，实现精准营销。同时，用户在朋友圈里的分享还可以直接形成口碑传播，带动新用户。

**发力移动支付**

　　从今年春节发起的微信红包就可看出微信对于移动支付市场的布局，此次发布的微信行业解决方案更是有利于移动支付的发展。

　　业内人士指出，对于微信来说，移动支付是基础能力，后续会建立起相关的生态模式。而在未来接入更多的商业模式之后，支付能力可以发挥出重要的作用。

　　从线上支付的无缝对接，到线下可广泛应用的场景，微信支付作为移动互联网时代的支付工具，可以整合微信的社交能力、大数据能力以及微信产品平台。此外，随着更多行业平台和第三方开发者的加入，微信行业解决方案的场景也会逐渐丰富普及。
