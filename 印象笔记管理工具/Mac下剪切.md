# Mac下剪切

---

方法一：用Finder的插件TotalFinder来实现剪切功能，并且还有windows的特性（此软件还有更多优秀的功能）这样非Lion系统一样支持剪切功能

方法二：就是把系统升级到10.7 +系统就自带剪切功能  
先 cmd + C 复制文件，然后 cmd + option + V 粘贴，文件就移动过来了，相当于剪切。  
按图操作也可（先 cmd + C 复制文件后，在目的地按住 Option 点击右键）
