Windows 10 官方「外挂」更新，这 3 个实用小功能你或许用得上

![](.evernote-content/A5551F13-3E0B-4825-95DA-174BF1063B04/DC2C94FC-8CA5-45E5-97B7-C631D15E07DE)

Windows 10 官方「外挂」更新，这 3 个实用小功能你或许用得上

在 Windows 原生功能还很薄弱的时代，一个名为 PowerToys 的工具曾给不少追求效率的 Windows 用户留下了深刻的印象。多年之后，PowerToys 再度 [复活](https://sspai.com/post/56719)，凭借其官方开发背景成为可能是最了解 Windows 10 效率用户需求的增强工具。

**关联阅读：**[微软出品的窗口管理工具，还能让你不再苦记快捷键：PowerToys](https://sspai.com/post/56719)

此前我们已经介绍过 PowerToys 早期版本中的快捷键提示和桌面窗口管理等功能，而在最近的 0.16 版本中，PowerToys 再次迎来了数个核心功能更新。让我们一起来看看它又解决了 Windows 用户的哪些痛点。

让你的资源管理器更好用
-----------

针对大家几乎每天都要用到的资源管理器，PowerToys 增加了两项非常实用的功能：**SVG 图片和 Markdown 文本内容预览。**

尽管 Windows 资源管理器针对一些常见文件类型（如 jpg、png 等格式的图片）提供了文件预览，对常常需要跟 SVG 图片和 Markdown 文件打交道的人来说，我们依然无法直接通过资源管理器窗口直接查看这两种文件的预览效果，设计师朋友和文字工作者只有在特定应用中打开这两种文件才能一睹真容，未免有些麻烦。

如果你也需要和这两种文件打交道，不妨在下载安装 PowerToys 0.16 后在应用设置中找到资源管理器设置（File Explorer），然后勾选 SVG Previewer 和 Markdown Previewer。至此我们便开启了 Windows 自带资源管理器的 SVG 图片和 Markdown 文件预览功能。

![](.evernote-content/A5551F13-3E0B-4825-95DA-174BF1063B04/A3D510C5-2128-41B3-8A87-706BFEFA6887.png)

回到资源管理器，在顶部 Ribbons 选项中找到并启用「查看 > 预览窗格」选项，究竟在点击选中 SVG 图片和 Markdown 文件后直接在「预览窗格」中看到文件内容了。

![](.evernote-content/A5551F13-3E0B-4825-95DA-174BF1063B04/B7B78BE3-8BAF-4747-A62C-3625AB7E3578.png)

预览效果

不过目前除了通过「预览窗格」外，PowerToys 还无法直接让 SVG 图片和 Markdown 文件像其它拥有预览的文件一样直接在文件图标上以缩略图方式展示预览，直观性还是差了些，不知道后续会不会有解决方案，同时也期待 PowerToys 为 Windows 资源管理器加入更多类型的文档预览。

更简单快速地窗口切换
----------

熟悉快捷键操作的 Windows 用户一般会通过 `Alt+Tab` 进行多窗口切换，但 Windows 默认提供的窗口切换 UI 由缩略图和窗口标题组成，缩略图占据了大部分视觉空间，窗口数量过多时定位起来其实不太方便。

PowerToys 的新功能 Window Walker 给出了一个很有意思的思路：**直接输入软件的名字来进行准确定位**。

![](.evernote-content/A5551F13-3E0B-4825-95DA-174BF1063B04/09868AC2-639F-43ED-A452-E1DF4B57900E.png)

举个例子，当我打开 Chrome、Visual Studio Code、微信、TIM、Axure 等一大堆软件后，通过快捷键 `Ctrl+Win` 徽标键即可打开 Window Walker，此时输入法光标会自动定位至 Window Walker 的搜索框中，在这个搜索框输入需要切换到前台的软件名称或者字母即可快速完成窗口定位和拉起。

相比在一大堆窗口缩略图中找软件，通过搜索软件名来实现显然要快捷得多。

![](.evernote-content/A5551F13-3E0B-4825-95DA-174BF1063B04/49FA8EA0-77CC-4234-BCFB-E34D554FBAE4.gif)

不过 Window Walker 目前似乎还存在进程索引不到的问题，部分应用（比如我这里的 Todoist 和微软商店）无法通过搜索直接拉起。考虑到它的高频更新节奏，这个问题应该很快就能得到修复。

批量调整图片尺寸
--------

最后这项新功能对很多新媒体编辑而言也许是刚需 —— 批量调节图片尺寸。

Windows 10 系统原生的照片查看器可以通过单张编辑的形式来调节图片的尺寸，但如果是一批照片需要一次性进行尺寸调节处理，就只能转而求助于第三方工具。

PowerToys 0.16 新加入的 Image Resizer Settings 让这个工作变得简单了许多：右键选中需要调节尺寸的图片，然后选择重设图片效果大小，在弹出的设置框中选择需要缩放的尺寸、填充方式等参数后即可一键完成图片尺寸调整。

![](.evernote-content/A5551F13-3E0B-4825-95DA-174BF1063B04/3393C6D4-885D-478A-A9DB-ED87CEDB16B8.gif)

除此之外，在我们把多张图片移动到新文件夹的过程中，也可以通过选中图片用右键拖动到新文件夹，然后直接重设图片大小完成移动。这样一来我们就直接生成了符合尺寸要求的图片了，如果你想进一步省掉删除原图的步骤，也可以勾选上「缩放原图」的选项。

![](.evernote-content/A5551F13-3E0B-4825-95DA-174BF1063B04/5A8722CA-237D-4D25-A4EC-1BE9AAE78212.gif)

这个功能最大的好处就是不需要通过更专业的第三方软件调整，而且如果批量处理的图片存在多个格式，生成出来的图片也会保留原来的格式。
