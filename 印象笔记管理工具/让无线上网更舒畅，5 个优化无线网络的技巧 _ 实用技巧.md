有句话是这么说的：「无线网络，就跟水和电一样，成为了人们生存的基本需求」。如同我们想喝到干净的水，用上稳定的电，一个低延迟高速度的无线网络能极大提升生活质量。

可无论我们买了多贵的路由器，用上多快的光纤，总是会在某个时刻遇到速度突然变慢、延迟突然变高甚至断流等问题。很多时候并不是设备的问题，通过一些优化方法，我们可以减少这些问题出现的频率，还有几率完全消除这些问题。

改变信道：解决 Wi-Fi 信号拥堵
------------------

在你的周围有很多路由器和其他无线网络设备的时候，发生信号拥堵导致延迟而断流的次数会明显变多，这是因为信号发生了拥堵。

路由器和手机之间的交流不是通过专有通道的。当路由器和你手机交流的时候，路由器都会接受周围所有设备发出的信息，再筛选出与手机约定好的特定信道的数据，最后从信道中提取来自手机发送的数据。

### 信道简单科普

我们可以简单的把信道理解为 Wi-Fi 高速公路上的车道，每条车道上的车越少，那么那条车道上的车就开得越快。根据我国法律规定，在 2.4GHz 无线网络下有 13 条信道，除了单车道上的车，2.4GHz 无线网络还有邻频干扰的问题，就相当于每一条车道都没有绿化带，有一小部分车会突然变道影响其他车的速度。

切换到 5GHz 的无线网络可以解决干扰问题，因为 5GHz 的无线网络相当于有绿化带的 20 车道，而且目前上面的车比较少（支持设备数量还不够多）。但是 5GHz 无线网络有穿墙能力弱的问题，而且大部分旧路由器只支持 2.4GHz 无线网络。

现代路由器都提供了自动切换使用最少信道的功能，但这带来了另一个问题。如果你周围有很多路由器，你的路由器检测到 13 信道使用的设备比较少然后切换了过去，接着邻居的路由器也检测到 13 信道使用的设备少也切换了过去，很快 13 信道又被占满了…… 所以有时候我们感到无线网络互快忽慢就是这个问题（如果你周围的无线热点比较多的话）。

![](.evernote-content/F82DB5CB-E014-4A73-9BED-EA746B7D80EB/D0A3BF20-F406-45A3-80FE-98E8669928C4.png)

查看信道使用情况

使用支持 Android、Windows 和 macOS 的软件 [Netspot](https://www.netspotapp.com/) 可以轻松像上图查看自己与周围 Wi-Fi 的信道情况，再在路由器里选择一个比较少设备用的频道就可以很大缓解信号拥堵的情况（一般在路由器后台的「无线设置」里面可以找到相关设置）。

使用 QoS（Quality of Service）
--------------------------

除了外部因素，局域网里面的设备也会影响你使用无线网络质量。宿舍里面如果有一个人下电影其他人玩游戏就会特别卡；如果你在家里在线看片，那么你的女朋友可能连淘宝都打不开……

那是因为普通模式路由器是按照设备要求分配网络的，如果设备上使用的比较「贪婪」的网络请求，那么就可以榨干路由器的每一字节网速，不给其他设备活路。但实际上看片、玩游戏、下电影和淘宝都不需要那么多的网速（比如下电影时只要能留出几十 Kb 网速给游戏就行了）。

这个时候就需要 QoS 出马，打开路由器后台，一般在无线设置里找到 QoS 相关设置，填入家里宽带的实际上传和下载速度。注意是实际上传和下载速度，不是宽带签约速度，比如 100M 光纤下载速度就是 12.5MB/s 左右，可以自己在测速网站测试，一些路由器也会在设置旁边自带测速按钮。填好上传下载速度后打开 QoS 开关即可。

打开 QoS 以后，路由器就会智能根据每个设备的需求分配上传下载速度，比如游戏经常走的 UDP 协议就会变成稳定优先速度次之，BT 下载的端口则会再满足其他设备速度要求后尽量分配最大速度。这样路由器连接设备再多，也不会出现「一人使用，他人卡死」的情况了。

![](.evernote-content/F82DB5CB-E014-4A73-9BED-EA746B7D80EB/9EF11045-E747-465F-802A-DD2FFF9A1F8C.png)

一些路由器还能设置每个设备的优先级

清理 Wi-Fi 列表
-----------

相信大部分人偶尔也有蹦出过这个念头：**把周围的 Wi-Fi 从扫描列表中隐藏掉。**

听起来有一点古怪，但我们有时候的确想把除了自己家以外的其他 Wi-Fi 都隐藏起来。除了看起来更加干净以外，这一措施也有它的实际意义：

* 比如有时候你想连接自己的 Wi-Fi 却发现邻居的热点排在了第一位；
* 只留下一个 Wi-Fi，每次从打开 Wi-Fi 到连上 Wi-Fi 的速度会快一点；
* 不用频繁地与其他多个 Wi-Fi 通讯也能让无线网络的效能稍微高一点点。

通过下面的对比图可以看到，隐藏其他 Wi-Fi 之前，Wi-Fi 扫描列表非常混乱，隐藏其他 Wi-Fi 之后，只会显示你想要显示的 Wi-Fi：

![](.evernote-content/F82DB5CB-E014-4A73-9BED-EA746B7D80EB/180134CF-AA5D-439A-B47F-909DECC4F4D3.png)

隐藏前后的 Wi-Fi 列表

### 实现与原理

接下来我们就看看如何来实现这个效果。在说明具体的实现步骤之前有两点需要注意：

1. 这个方法只适用于 Windows，目前还没有发现适用于 macOS 和手机上的方法。
2. 如果周围的 Wi-Fi 改了名字，那么下面介绍的第一种方法就会失效。

#### 方法一：黑名单模式

首先，右键点击系统左下角的开始图标，选择「Windows PowerShell（管理员）」。

![](.evernote-content/F82DB5CB-E014-4A73-9BED-EA746B7D80EB/B01A3897-55A3-4DB5-91AF-77787C883E99.png)

这一步是为了打开控制台窗口

接着输入命令：

```
netsh wlan show networks
```

并回车，会显示所有的 Wi-Fi 列表，将它们复制到一个新建文本文件里。

![](.evernote-content/F82DB5CB-E014-4A73-9BED-EA746B7D80EB/3ED8E459-7C3B-4067-9463-CBDE26CD3784.png)

SSID 后面就是 Wi-Fi 名字

输入下面的命令，即可将它从扫描列表中隐藏：

```
netsh wlan add filter permission=block ssid="这里填上你想要隐藏的 Wi-Fi 名字" networktype=infrastructure
```

![](.evernote-content/F82DB5CB-E014-4A73-9BED-EA746B7D80EB/21A6A581-D44A-418E-B1F9-339B5D7519C8.png)

命令执行效果

你也可以分别通过下面两条命令，来查看已经隐藏的 Wi-Fi 和取消隐藏相应的 Wi-Fi。

```
netsh wlan show filters
netsh wlan delete filter permission=block ssid="这里填已经隐藏的 Wi-Fi 名字" networktype=infrastructure
```

![](.evernote-content/F82DB5CB-E014-4A73-9BED-EA746B7D80EB/1CD25284-5B12-4F44-97B4-82603FE8F9A4.png)

命令执行效果

使用上面的命令就可以轻松地隐藏或者显示 Wi-Fi 名字，但是这种方法有个缺点，那就是**如果你想要一次性隐藏很多 Wi-Fi 热点，必须一个一个地使用命令添加过滤器**，非常不方便。

如果你需要隐藏的 Wi-Fi 热点数量较多，那么，就可以使用下面的第二种方法来一次性隐藏大量 Wi-Fi。

#### 方法二：白名单模式

这种方法就非常简单了，只需要两步：

1. 输入以下命令，添加自家的 Wi-Fi 到白名单：

   ```
   netsh wlan add filter permission=allow ssid="自家的 Wi-Fi 名字" networktype=infrastructure
   ```
2. 输入以下命令，将所有 Wi-Fi 都添加到黑名单，这样子就只有白名单上的 Wi-Fi 热点会显示在列表中了：

   ```
   netsh wlan add filter permission=denyall networktype=infrastructure
   ```

![](.evernote-content/F82DB5CB-E014-4A73-9BED-EA746B7D80EB/631F83D6-4701-4717-A0B0-07BC8E8ED06B.png)

执行效果

这种方法非常快捷，但是只适用于放在固定地点的设备，比如家里的台式电脑。**如果你需要在不同地方连接不同的新 Wi-Fi 网络，就千万不要使用这种方法**，因为你根本看不到除了白名单外的其他 Wi-Fi 热点。

![](.evernote-content/F82DB5CB-E014-4A73-9BED-EA746B7D80EB/7636E336-B89C-4072-B94D-8F09E12BA807.png)

前后显示效果对比

你可能注意到所有命令开头都是 `netsh wlan`，`netsh` 是「Network Shell」的缩写，从 Windows 2000 开始便内置在系统中的一个命令行工具，帮助用户从本地或远程执行查看修改网络配置、网卡信息等网络相关的命令。

而 `netsh wlan` 则是 `netsh` 的无线网络版，使用它我们就能对无线网络配置进行查看修改了，包括修改 Wi-Fi 列表的显示。

注：这条技巧的灵感及方法来源于 Lifehacker 的文章《[How Do I Hide My Neighbors' Wifi Networks?](https://lifehacker.com/how-do-i-hide-my-neighbors-wifi-networks-1833806819)》

注意路由器天线的摆放方向
------------

可能大家都知道路由器上的天线越多信号就越好，但是天线的摆放方向也很重要。大部分人拿到路由器都会把天线摆成强迫症摆法，就是所有天线平行排列：

![](.evernote-content/F82DB5CB-E014-4A73-9BED-EA746B7D80EB/DBBBD2FD-FDB2-44E8-A65D-9243628F105F.png)

平行排列的路由器天线

根据 [TP-link 官方支持页面](https://service.tp-link.com.cn/detail_article_3283.html)，单天线的路由器只需要摆成与地面垂直 90 度就能获得最广的信号水平覆盖范围，而天线的上方下方为信号覆盖的薄弱点。

![](.evernote-content/F82DB5CB-E014-4A73-9BED-EA746B7D80EB/EF7D9A0B-F7E7-4A2E-949A-A5C26B8E850C.png)

路由器单天线覆盖效果

而且官方帮助特别用红字注明了**路由器天线紧凑在一起会互相干扰**。加上现在路由器大多不止一根天线，所以事实上平行排列的方法会使信号产生干扰，只是看起来比较美观。

那么多天线的路由器天线应该怎么摆放呢？如果是两根天线的路由器，可以让两根天线错开（即一根竖着放，一根斜着或者 90 度放），不仅可以避免干扰，还可以在相同功率下实现更均匀的覆盖范围。另外，有些朋友可能觉得两根都竖着放可以增加水平覆盖范围，但因为国家规定的路由器最大功率已经固定，所以其实平行排列也不会增加水平覆盖范围。

![](.evernote-content/F82DB5CB-E014-4A73-9BED-EA746B7D80EB/B2408107-03D9-4528-81D0-8CA7F217B552.jpg)

路由器天线的摆放

多根天线的路由器也是如此，尽量让每根天线都错开以达到最好的抗干扰效果。

![](.evernote-content/F82DB5CB-E014-4A73-9BED-EA746B7D80EB/6CF25B33-FA11-495F-BE6B-0744E01B0823.png)

TP-link 官方的建议

定时重启路由器
-------

为了节能，路由器所使用的处理器与内存配置相对它们做的工作都比较低（即使是现在的智能路由器也是一样）。作为 24 小时运行的设备，它的运行内存会逐渐被各种东西填满：系统泄漏的内存，其他地方产生的一点点内存垃圾等，现代路由器已经优化得很好了，但长时间工作仍会使性能下降。

隔一段时间重启路由器，清空运行内存，就可以让路由器重新回到最佳工作状态。甚至很多路由器都提供了自动在每天凌晨没有设备使用的时候重启的选项，如果路由器没有自动重启选项，那么每隔一个星期左右手动重启一次也不会太花功夫。

![](.evernote-content/F82DB5CB-E014-4A73-9BED-EA746B7D80EB/6FA3AA7B-CC19-4451-A7D0-9F540ED33022.png)

路由器提供的自动重启选项

结语
--

无线网络已经发展这么多年，普及到甚至让人觉得它已经是理所应当甚至是凭空出现的了。但耐用如汽车也要保养年检，路由器也并不是一个买回来就能开着不管的东西，缺少日常维护也会使无线网络质量下降。使用上面简单的优化方法，就能让路由器重新提供更高质量的无线网络。
