超级笔记功能上线，印象笔记客户端迎来「模块化」更新 - 少数派
===============================

7 月 2 日，印象笔记的官方微信推送 [动态](https://mp.weixin.qq.com/s/2tDYa0GPb-ErKAKwgcj2Ag)，宣布印象笔记「超级笔记」功能正式上线，在这篇动态中，官方甚至使用了「重新定义笔记」来形容这次更新。

相比之前网页端的新版模块化编辑器（以下简称 2019 版网页编辑器），这次印象笔记更新的超级笔记功能有哪些提升？如果你是印象笔记用户，又该如何发挥它的最大潜力？

作为印象笔记的资深用户，超级笔记功能甫一推出我便进行了上手体验。所以在这篇文章中，我们就围绕上述问题聊一聊印象笔记编辑器的新功能。

模块化已经成为了一种趋势
------------

什么是「超级笔记」？简单地讲就是「模块化」进客户端，或者说印象笔记的客户端进行了模块化。

在当前的网络服务中，我们经常能看到采用模块化设计的编辑器。虽说模块化编辑器依然没有一个统一的定义或标准，但印象笔记此前也已经就这一趋势进行过初步探索，将文字、段落、引用、待办事项、图片、视频、表格等等元素都拆散作为一个个文档「模块」。

![](.evernote-content/BF2AAD48-976E-4789-B4D0-936F3F042AEE/0F33FD88-0D32-46E9-8AF6-2CECF547DCCC.gif)

印象笔记此前更新加入的表格操作

使用模块化编辑器进行文档创作就像搭积木，使用恰当它可以让图文组织变得更简单也更有效率。

在提升文档编辑效率的同时，模块化也在重塑着其他产品的形态，现在我们在很多 App 和服务上都能看到模块化的影子：我们熟悉的 Todoist、WordPress 的 Gutenberg 编辑器、微软在 Build 2020 大会上提到的 [Fluid Framework](https://sspai.com/post/60652)……

![](.evernote-content/BF2AAD48-976E-4789-B4D0-936F3F042AEE/21AAC875-3E18-49B2-B017-A8F3DB760B23.gif)

Todoist 的模块拖拽

![](.evernote-content/BF2AAD48-976E-4789-B4D0-936F3F042AEE/2AF088CC-1F39-4924-A58D-608F64F4684F.gif)

微软的 Fluid Framework

模块化不仅是一种趋势，随着这种设计应用的不断推广，它更成为了互联网服务的「基础设施」。而对于印象笔记而言，2020 年恰好也是压力比较大的一年，这当中有来自新冠疫情的影响，也有竞争对手带来的压力。

就在今年 5 月，Notion 推出了新的付费计划，其后不久，另一款热门应用 Roam Research 也正式开放注册，无论是一般用户还是专业市场，新产品、新服务的出现都对印象笔记的市场提出了挑战；与此同时，尽管 2019 版网页版编辑器已经上线将近一年，仍有大量的印象笔记用户选择停留在原有客户端应用上 —— 自然而然地，为现有客户端升级、让更多印象笔记的现有用户用上新功能对印象笔记而言就有了特别的意义。

主要特性盘点
------

通过上面的对比不难看出，在「模块化」这一潮流之下，新版印象笔记客户端的超级笔记功能显然也带来了不少独具特色的体验和实用特性，总结起来包括：

### 模块化写作

超级笔记推出之后，使用印象笔记客户端码字变得更方便。以往通过口述语音转成文字之后，再次进行文字整理很麻烦，现在通过模块化的 4 个按键可以快速对文稿进行整理和排版。

1. 回车 - 将原来段落拆分成为模块。
2. 左键 - 拖拽调整新的次序。
3. End - 定位到段落末尾。
4. Delete - 段落合并。

![](.evernote-content/BF2AAD48-976E-4789-B4D0-936F3F042AEE/7A33FF74-70AC-4275-9603-2170ABDC3C22.gif)

4 键写作

很快我们就能完成对一段文字的调整。

### 悬浮笔记目录

对笔记内目录这个特性的支持也是呼声已久，因为在 2019 版网页编辑器和此前印象笔记客户端都没有笔记内目录这一功能（某段时间借助 Markdown 可以短暂实现）。

![](.evernote-content/BF2AAD48-976E-4789-B4D0-936F3F042AEE/0A2253B5-EC48-498C-8333-C190653C5903.gif)

悬浮目录

超级笔记中新增的悬浮笔记目录平时折叠隐藏在编辑器的右上方，最大化显示笔记内容；需要时则可以通过点击右上角的目录 icon 进行呼出。

设计上，悬浮目录左侧的竖线能够帮助我们清楚地看到相关目录之间的层级关系。有了它的加持也可以快速定位到笔记内部的相应内容，写作长文时非常方便。

### 跨笔记模块拖拽

微软在 Fluid Framework 中设想过的最令人印象深刻的场景，莫过于把文档模块从一个文件中拖到另一个文件。

在超级笔记中我们也能看到这样一个有些类似的特性。我们可以把一个模块从一条笔记拖到另一条笔记之中，这样笔记与笔记之间的内容迁移就变得简单了许多。

![](.evernote-content/BF2AAD48-976E-4789-B4D0-936F3F042AEE/03315FB1-53C2-40E6-978C-D0BDF90E1B3C.gif)

这一点，飞书也好、Notion 也罢，都是没有的。

写文章时，平常我们通过剪藏、微信公众号、微博等方式积累在印象笔记当中的案例、故事、名家语句等等素材顿时就能「活用」起来了，只需要拖拽这些素材就可以快速地合体形成一份文章框架。

因为基于客户端，只要屏幕足够大，我们甚至也可以同时打开好几条笔记的窗口进行拖拽。

它有哪些不一样
-------

和 Notion 这类以模块化编辑为特色的服务不同，印象笔记在加入超级笔记功能的同时并没有选择 Web 应用或 Electron 打包的形式，相反，新版印象笔记客户端依然保留了传统客户端体验，并在此基础之上进行了模块化更新。

对印象笔记而言，推出超级笔记这个功能的同时需要面对的首要问题是承接：承接之前的笔记内容和体验，承接新的网页版模块化编辑器，承接网页版模块化编辑器推出后不久、在 Mac 上推出的模块化测试版本。

所以即便此次推出的超级笔记和此前已经推出的网页版编辑器体验相近，当中还是有不少结合用户反馈进行的修正。接下来我会着重地从几个方面进行一下对比。

### 与旧版印象笔记客户端对比

整体而言超级笔记这一版的客户端流畅性在之前的客户端的基础之上有了很大提升，但凡用过印象笔记 Windows 客户端的用户应该也都明白这里说的「卡顿」是怎样的一回事情…… 以我自己在用的 Windows 平台版本为例，新版几乎没有可以明显感知到的卡顿现象，增加新功能的同时，可以看出印象笔记在客户端优化和处理之前的用户反馈上还算用心。

通过点击「新建超级笔记」即可进入到超级笔记编辑器的界面，因为格式化工具栏得到了重新设计，整体界面更加清爽、简洁，视觉元素的精简对于经常码字的人来讲更加友好，同时超级笔记的基本操作则与此前的笔记体验基本一致，上手成本较低。

![](.evernote-content/BF2AAD48-976E-4789-B4D0-936F3F042AEE/12A50E3B-47D4-4163-B380-218BCFE4F274.png)

界面对比

如果我们把旧版印象笔记的编辑器看成是平面的，那新版印象笔记编辑器就是立体的。

这种立体让印象笔记客户端的编辑器显得更加现代：比如终于能够将 URL 显示为一个网页标签而不是链接文本、支持以内嵌代码的形式插入视频而不是通过附件上传视频文件。虽然还不支持自动解析，但优酷、腾讯、B 站等主流视频网站其实都直接提供了内嵌代码可以直接复制插入。

![](.evernote-content/BF2AAD48-976E-4789-B4D0-936F3F042AEE/03E08AA5-3282-4BC9-80C0-8EB69D497014.gif)

边看边写的场景还是有的

在这样的多媒体混排之下，视频、笔记终于可以完全融合在一条笔记之中了（上班摸鱼也更方便了），音频方面也拥有类似的实现，这里不做过多展开。

最后，超级笔记也为新版客户端带来了智能表格支持。此前印象笔记曾多次对表格功能进行过升级，但整体而言印象笔记的表格功能一直停留在简单的文本内容阶段，旧版客户端也一样把表格当成文字来处理，甚至不能像 Excel 那样对表格内容进行运算，进度条什么的就更不用说了……

![](.evernote-content/BF2AAD48-976E-4789-B4D0-936F3F042AEE/58F889F9-AF34-433A-8FDD-97AB6B3B3983.gif)

智能表格还是很实用的

超级笔记的智能表格则完全支持表格内公式、进度条等功能，借助于超级笔记进行简单的项目管理自然也可以更方便一点。

![](.evernote-content/BF2AAD48-976E-4789-B4D0-936F3F042AEE/A5B82F0A-305E-41B1-AC6D-34D611CB9C1D.gif)

智能表格中直接插入进度条

在超级笔记中我们也可以更方便地进行看板管理，所有操作都可以在客户端内直接实现，不必依托于 2019 版网页编辑器或借助于 Kanbanote 这类第三方 App。

关联阅读：[利用看板思维，快速在印象笔记中完成一份职场报告](https://sspai.com/post/56652)

### 与新版网页版编辑器的对比

既然提到了 2019 年印象笔记推出的网页编辑器，当我们看到「超级笔记」的时候，自然也会联想到它。

关联阅读：[印象笔记编辑器大改版，模块化设计让排版更自由](https://sspai.com/post/55729)

超级笔记在功能上要比 2019 版网页编辑器更加强大，以之前文章中提到的网页版编辑器的万能「+」号为例，超级笔记在 2019 版网页编辑器的基础之上进行了升级，除了通过「+」号添加的各种文档模块之外，还新增了可以选中进行快速格式调整、复制粘贴的模块化按钮；同时我们也可以使用快捷键 / 来打开模块化扩展功能，这是网页版编辑器所不支持的。

![](.evernote-content/BF2AAD48-976E-4789-B4D0-936F3F042AEE/816B5C2A-0048-4FB4-8EA9-62A6C013349D.gif)

万能的「+」

其他体验方面，使用 2019 版网页编辑器时经常会遇到拖拽与鼠标手势插件相冲突的情况，在编辑器支持超级笔记功能之后，这个顾虑自然也就没有了。

### 与 Evernote 对比

尽管印象笔记与 Evernote 分家已经有一段时间了，每当看见印象笔记更新，不少人也还是会想起 Evernote 并不自觉地将二者进行对比。

这一次印象笔记的超级笔记更新走在了 Evernote 的前面，目前来讲，我们并未看到 Evernote 会有所跟进。在细分客户群体上，Evernote 的 2019 版网页编辑器只针对高级账户开放；而对于印象笔记而言无论是 2019 版网页编辑器还是此次的超级笔记，都是对全部用户进行开放的。

### 与飞书文档对比

经此一「疫」，飞书文档也成为了不少读者常用的文档写作工具，这里我也将新版印象笔记客户端的超级笔记功能和飞书在线文档进行了简单对比。

尽管整体上来讲飞书文档与超级笔记功能并无太大差别，刚刚上线的超级笔记相比飞书还是有着不小的改进空间，比如在内容上飞书文档支持 Airtable、地图等更多形式，在细节上飞书文档则支持通过直接插入视频网站的网址的形式插入视频，而「超级笔记」则需要通过获取内嵌代码再进行插入，显然还不够方便快速。

另外，飞书文档还可以通过 @ 进行快速调用，比如关联文档、协作成员、其他表格等等，这一点也要好用不少，尤其是在进行文档协作的时候。

在表格的问题上，撇开普通表格不谈，飞书文档支持电子表格的形式，这种电子表格的操作可以简单看成是一个内嵌的 Excel 表，可以进行相应的操作比如 Excel 中的公式；而「超级笔记」中的智能表格则另有特色，上面已有表述，这里不过多展开，只能说二者侧重场景不同。

### 与 Notion 对比

在 Notion 我们也可以通过「+」「/」来快速调用各种模块，可插入调用的具体内容方面，Notion 作为业界一强自然要比超级笔记丰富不少，包括行内公式、看板、地图等等，这些都体现出了 Notion 基于 database 的特性优势。

![](.evernote-content/BF2AAD48-976E-4789-B4D0-936F3F042AEE/40A27268-9E1E-4233-A6A9-3F73E5163424.gif)

Notion 的模块化菜单

值得一提的是，虽然二者都支持目录功能，但是印象笔记的超级笔记更进一步实现了笔记内悬浮目录；模块支持方面，超级笔记也支持跨笔记的模块拖拽，实现了模块在笔记与笔记之间的打通，这个我们在上面已经提到，这里不再展开。

最后，印象笔记客户端的超级笔记功能相比包括 Notion 在内的、上述大部分在线文档服务而言，最大的一个优势其实在于形式：客户端会给用户带来一种安全感，毕竟所有笔记可以下载到本地，而 Web 应用一旦断网就无法访问，场景还是十分受限的。

一点问题与猜想
-------

超级笔记功能推出的同时，也采取了一种与旧有笔记共存的方式 —— 也就是说在新版客户端中两种编辑器并存。考虑到之前印象笔记尚未从 Evernote 独立时多次因为编辑器升级导致表格翻车的事故，此番印象笔记如此谨慎的操作其实也可以理解。

不过这种做法依然带来了一种体验上的割裂感，旧的笔记无法使用超级笔记进行编辑，官方目前也尚未提供转换工具。

同时，尽管此次印象笔记的超级笔记功能在 Windows 和 Mac 平台上都有推出，网页版、桌面客户端之间的缝隙正在逐渐弥合，之前反馈的平台割裂感也依然存在。主要表现在同为模块化的网页版与桌面客户端之间的体验并不一致，以及移动端对应功能的缺席 —— 在手机上使用印象笔记的用户可不在少数。

整体上看，印象笔记超级笔记功能的推出顺应了编辑器模块化这个大趋势，我们甚至也可以借此期待一下 —— 在各方的共同努力之下，业内是否会就模块化达成某种共识，乃至于出现一个模块化编辑器的通用标准呢？让我们拭目以待。
