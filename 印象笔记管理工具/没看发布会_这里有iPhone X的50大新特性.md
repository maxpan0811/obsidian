# 没看发布会?这里有iPhone X的50大新特性

---

威锋网讯，苹果已经推出了十周年款的 iPhone X，这款代表着苹果未来的手机拥有一系列的新特性，这款设备要到 11 月才会发售，那么趁着现在有时间，我们就来看看这款旗舰设备的 50 大新特性。

![](.evernote-content/1723BCBD-8830-49A5-8285-1DFAEC684278/8B771E0D-F98A-446F-98FE-DBE4A4E64FCE.jpg)

**设计**

1、玻璃：iPhone X 的前后都采用了全玻璃的设计，强化层的深度为 50％。苹果公司表示，七层墨水工艺带来了精确的色调和不透明度，而反射光学层可以增强颜色，疏油涂层有助于减少污迹和指纹。

2、不锈钢中框：手术级的不锈钢带缠绕在 iPhone X 的边缘。这是一种由苹果设计的合金。

![](.evernote-content/1723BCBD-8830-49A5-8285-1DFAEC684278/A670FB07-B9F4-46D5-8F49-CE36DF20F3D7.jpg)

3、全面屏：iPhone X 具有几乎边缘到边缘的屏幕，在顶部仅有一个微小的凹口用于原深感摄像头系统。窄边框围绕着屏幕，主屏幕按钮已被取消。

4、侧边按钮：苹果加长了设备上的锁定键，并在 iPhone X 上重新命名为“侧边按钮”。长按可调用 Siri，双击调出 Apple Pay。

5、新的加速计

6、新的陀螺仪

**超视网膜显示屏**

7、5.8 英寸显示屏：iPhone 史上最大的显示屏。尽管如此，全面屏设计使它的整机尺寸在 iPhone 8 和 iPhone 8 Plus 之间。因此，iPhone X 是最适合单手握持但又具有最大尺寸显示屏的 iPhone 机型。

8、OLED：iPhone X 是首款配备 OLED 显示屏的 iPhone，OLED 可以带来更佳的色准、高对比度和更深邃的黑色。

![](.evernote-content/1723BCBD-8830-49A5-8285-1DFAEC684278/C20C693A-65DA-4456-9294-1FF6766A79E1.jpg)

9、原彩显示（True Tone）：iPhone X 可自动调整显示屏的色温和强度，使其与周围环境中的光线色温相匹配。例如，如果您正在开着白炽灯的昏暗房间，显示屏将显得更暖和更黄。如果您在阴天的时候出外，显示屏会显得更冷和更蓝。

10、HDR：iPhone X 实现了真正的 HDR (高动态范围) 显示，让你可以欣赏到符合杜比视界和 HDR10 标准的影片和节目，HDR 照片的显示效果也更加惊艳。另外，你还能观看其他视频网站提供的精彩 HDR 内容。

11、2436×1125：iPhone X 拥有 iPhone 史上分辨率最高的显示屏，ppi 达到了 458。

12、1,000,000:1 对比度

13、点按来唤醒

**前置摄像头**

14、原深感摄像头：这一前置摄像头系统包括了红外镜头、泛光感应元件、距离感应器、环境光传感器、扬声器、麦克风、700 万像素摄像头和点阵投影器。这个系统将被应用于Face ID脸部识别、人像模式自拍和动话表情（Animoji）。

15、Face ID：苹果在 iPhone X 上用 Face ID 取代了 Touch ID。只需拿起设备，看着它就可以完成设备解锁，又或者是将其用于验证 Apple Pay 付款。苹果表示，面部识别系统被陌生人破解的机会是百万分之一。

16、人像模式自拍：在原深感摄像头系统的帮助下，iPhone X 的前置摄像头也支持人像模式。

![](.evernote-content/1723BCBD-8830-49A5-8285-1DFAEC684278/3F5692C4-3A75-4AC5-9DAC-15CA4A221D09.jpg)

17、动话表情：动话表情（Animoji）是苹果一套全新的表情符号体系，它基于 iPhone 用户的面部表情。动话表情充分利用了 iPhone X 的原深感摄像头系统，该系统拥有几个新的 3D 传感器来实时检测您的面部表情。

**后置摄像头**

18、更大且更快的 1200 万像素传感器

19、垂直排列的双摄

![](.evernote-content/1723BCBD-8830-49A5-8285-1DFAEC684278/9FCC95ED-DA07-4010-819A-346DA4A72490.jpg)

20、双镜头光学图像防抖：两个后置镜头都具有光学图像防抖功能，而且反应飞快，即使在弱光下，也能拍出效果出众的照片和视频。

21、支持慢速同步的 4-LED 原彩闪光灯：慢速同步闪光将较低的快门速度与一组短促的闪光相结合。这种闪光方式在弱光条件下非常有用，可以在照亮前景主体的同时，让背景也得到适当的曝光。此外，4-LED 原彩闪光灯的光照均匀度最高也提升至原来的两倍，有助于减少亮斑现象。

22、全新图像信号处理器：苹果表示，其设计的图像信号处理器非常智能，可以侦测场景中的各种要素，例如人物、运动状态和光线条件，从而在你按下快门之前就对照片进行优化。另外，它还具备先进的像素处理能力、广色域拍摄、更快的自动对焦速度，并能生成效果更好的 HDR 照片。

23、苹果设计的视频编码器：苹果设计的视频编码器会实时地处理影像，优化影像质量。而 HEVC 压缩技术可在保持画质的同时，将视频文件压缩到原来的一半大小。

24、最高可达 60fps 的 4K 视频

25、最高可达 240fps 的 1080p 慢动作视频

![](.evernote-content/1723BCBD-8830-49A5-8285-1DFAEC684278/282B0B53-1B50-46AE-85F5-B005B03F18B7.jpg)

26、改进的视频防抖

27、长焦镜头更大的光圈：iPhone X 的长焦镜头现在拥有更大的 ƒ/2.4 光圈，作为对比，iPhone 7 Plus 上的长焦镜头光圈为 ƒ/2.8。

28、更好的低光变焦

29、改进的人像模式

30、人像光效：人像光效功能根据摄影布光的基本原则，结合运用深度感应摄像头和面谱绘制等复杂的软硬件，生成逼真的影棚级光效。

31、为增强现实专门调校

32、更深的像素

**电源**

33、无线充电：iPhone X 支持基于 Qi 标准的无线充电，iPhone X 可以放置于感应式充电板上进行无线充电，例如苹果推出的 AirPower 充电枕，又或者是选购一些第三方厂商推出的无线充电配件。

![](.evernote-content/1723BCBD-8830-49A5-8285-1DFAEC684278/7F56EB22-01D6-4CF5-92AB-E05286C439AE.jpg)

34、快速充电：iPhone X 支持快充，可以使用苹果的 29W、61W 和 87W USB-C 充电器为 iPhone X 充电，30 分钟可以充满 50% 的电量。这些充电器需要单独购买，不过 12 英寸 MacBook 和 2016 款或更新 MacBook Pro 也使用同样的充电器，可以通用。

35、更长的电池续航：比 iPhone 7 多两个小时的续航时间。

**性能**

36、A11 仿生：全新中央处理器的四个能效核心速度比 A10 Fusion 最高提升 70%，两个性能核心也有了最高达 25% 的速度提升。

37、M11 移动协处理器

![](.evernote-content/1723BCBD-8830-49A5-8285-1DFAEC684278/47F02565-4C6D-4576-BB2B-5AD4D1719C59.jpg)

38、神经网络引擎：这款 iPhone 上有史以来最强大、最智能的芯片，拥有一个每秒运算次数最高可达 6000 亿次的神经网络引擎。并为一系列功能提供驱动力，例如 Face ID 和 Animoji。

39、更快的 GPU：苹果设计的新款三核心图形处理器，与 A10 Fusion 相比，速度提升幅度最高可达 30% 之多。

40、NFC 读卡器模式：苹果之前推出了 Core NFC 框架，这个 iOS 11 的全新框架可以让应用检测到 NFC 标签。

41、蓝牙 5：与 4.2 版本相比，蓝牙 5 可以提供四倍的有效范围、两倍的速度以及八倍的广播信息能力。

**定位**

42、伽利略全球定位支持：伽利略是欧洲全球卫星导航系统（GNSS），它是美国zheng fu 拥有的全球定位系统（GPS）系统的一个竞争对手。

43、准天顶卫星系统支持：准天顶卫星系统是一种基于卫星的 GPS 增强系统，可在日本接收。

**iPhone X 上的 iOS 11**

44、简化的状态栏：时钟在左。Wi-Fi 强度、蜂窝信号条、电池余量在右。

![](.evernote-content/1723BCBD-8830-49A5-8285-1DFAEC684278/41BDA674-2BFA-4800-8199-5D0E7CE08E76.jpg)

45、长按侧边按钮调出 Siri

46、按下侧边按钮+音量加键进行截图

47、向上滑动来关闭应用

48、向上滑动并悬停来打开多任务界面

49、iPad 式的 Dock 设计

![](.evernote-content/1723BCBD-8830-49A5-8285-1DFAEC684278/DD4A92FC-DCEC-4F9A-8E66-007923B5D255.jpg)

50、Clips 的自拍场景：当您在 iPhone X 上使用 Clips 时，Clips 会新增一个“自拍场景”功能。iPhone X 的自拍场景可利用原深感摄像头将你置身于 360° 的动画背景中，让 Clips 玩起来更带劲。

苹果将 iPhone X 视作一款未来的手机，官网上写道：一直以来，我们都心存一个设想，期待着能够打造出这样一部 iPhone：它有整面的屏幕，能让你在使用时完全沉浸其中，仿佛忘记了它的存在。它是如此智能，你的一触、一碰、一言、一语，哪怕是轻轻一瞥，都会得到它心有灵犀的回应。而这个设想，终于随着 iPhone X 的到来成为了现实。现在，就跟未来见个面吧。

新增一系列特性的 iPhone X 将从 10 月 27 日开始预购，11 月 3 日开始发售。
