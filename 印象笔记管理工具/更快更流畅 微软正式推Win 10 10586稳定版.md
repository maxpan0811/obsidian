# 更快更流畅 微软正式推Win 10 10586稳定版

---

[![](.evernote-content/F16B43FE-C839-41EA-A1CD-3C4A9631B1C4/6780221E-39BE-4C95-B49D-5971311FD29F.jpg)](http://tech.feng.com/2015-11-10/More-quickly-and-smoothly-Microsoft-officially-Win-10-10586-stable-version_629692.shtml)

![](.evernote-content/F16B43FE-C839-41EA-A1CD-3C4A9631B1C4/6780221E-39BE-4C95-B49D-5971311FD29F.jpg)

威锋网 11 月 10 日消息， 今天微软通过 Windows 官方博客证实，上周主要针对“快速分支（Fast Ring）”Windows 10 测试会员推送的 BUILD 10586 版本更新，如今已经为“慢速分支（Slow Ring）”推送稳定版本，而所有 Windows 10 用户正式推送的时间将定在近期。除了博客证实之外，微软大部分高管也在 Twitter 认证账户上公布了此消息。

微软表示，Windows 10 Build 10586 更新的 ISO 镜像文件正在准备当中，有望在近日上线提供下载。因此，理论上现在还未获得更新的用户，也不用等太久就能获得这个版本的更新推送，迫切抢先体验的用户，可随时在 Windows Updae 中检查版本更新的推送情况。

BUILD 10586 完全可以确定为今年秋季 Windows 10 最稳定的一个正式版本更新：“Fall Update”，也就是传说中的“Windows 10 Threshold 2 累积更新的版本”，对于会员计划的测试用户而言该版本并没有太多更新，只是 BUG 修复和解决已知的问题。但对于普通 Windows 10 用户而言，他们将获得新的激活机制、Cortana 改进、Edge 收藏夹、阅读列表、密码同步等多项更新内容，还包括新的 Skype 应用，系统响应更灵敏，运行也更快更顺畅。

![](.evernote-content/F16B43FE-C839-41EA-A1CD-3C4A9631B1C4/0C84358D-0159-4766-B925-EBE859AA2ACD.jpg)

对于慢速分支用户 BUILD 10586 究竟更新了什么呢？

在 Bug 修复方面：

1、修复了上一个预览版播放音频时(例如来自 Groove 的音乐、来自 Movies&TV 应用的视频)在“操作中心（Action Center）”有信息弹出的时候导致音量降低至 75% 的问题；

2、Surface Pro 3 的电源键可以让设备正常睡眠而不是像此前那样直接关机了；

3、修复了在系统闲置时一个 CMD 窗口（显示为 disksnapshot.exe）会随机出现的 bug；

4、Windows 系统能够记住用户上一次的登录类型，下次登录会直接使用记住的类型，例如，当你使用 PIN 码进行登陆，那么下次登陆的时候依然会优先推荐 PIN 码进行登陆；

5、修复了在 Edge 浏览器预览非当前标签页时，若该标签网页未滚动至顶部会导致预览变黑的问题；该问题同样会在使用 Web 笔记功能时遇到；

6、修复了小尺寸设备（包括戴尔 Venue 8 Pro）在启动时可能出现屏幕尺寸大于实际物理尺寸而导致循环重启直到蓝屏的问题；

7、从 Windows 商店下载应用和游戏更稳定；

![](.evernote-content/F16B43FE-C839-41EA-A1CD-3C4A9631B1C4/9C84A8EA-8107-4832-96D5-8FF819462FF0.jpg)

当然了，BUILD 10586 并不是十全十美，即便作为稳定版本也还存在着一些已知的问题，主要针对参与测试的 Windows 计划会员，微软将其进行了一一列举。比如 ，Messaging+Skype 应用中会消失 Skype 短信和联系人丢失，还有会员中心可能无法使用的情况等等，对此微软也在官方博客中提供了一些解决办法。

桌面版 Windows 10 系统更新了，那其他微软“大统一”的设备呢？根据相关消息了解，BUILD 10586 的 ISO 镜像文件上线之时（或许是 11 月 12 日），微软将针对 Xbox One 推送 Windows 更新，届时包括 Windows 10 手机也可能在同一天 获得重大的更新，这几天一波相关截图流传在网络上。不过，按照另一种说法，手机版本还是会推迟至 11 月底或 12 月初。
