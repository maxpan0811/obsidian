# 索尼的汽车梦，碎在了一个最坏的年景

---

Original亚洲通讯社社长徐静波静说日本

3月25日，索尼·本田移动出行公司（SHM）发出了一份措辞平静的声明——旗下电动汽车品牌AFEELA的第一款量产车“AFEELA 1”，以及尚在开发中的第二款车型，全部停止开发与销售。已在美国加州完成预约的用户，预付金200美元将予全额退还。

这意味着，一个被寄予厚望的汽车梦，在正式亮相还不到一年半之后，悄悄落幕了。

![](.evernote-content/4B8CDCEF-A347-4743-B2BB-0D5B7D1B5FCA/1E45F461-D182-45E8-84DE-C961D3FB19B1.jpg)

我写过许多篇有关“索尼汽车”的介绍文章，这一辆汽车曾被视为索尼从一家电子公司向“生活公司”转型的一个典范，更是日本电子企业与汽车名厂强强合作的一个榜样。听到这辆研发了整整5年的车停产停售的消息，真的非常震惊与失望。

2020年，在美国CES消费电子展上，索尼发布了一辆叫做”VISION-S”的概念车。那一刻，全场哗然。没有人想到，以游戏机和随身听闻名于世的索尼，会造出一辆看起来如此认真、如此完美的汽车。

那辆车配备了车内外共40个传感器，搭载了高达800TOPS算力的计算机，车舱里是索尼最引以为傲的娱乐体验。它不只是一个展台上的道具，它更像是一封宣言：索尼要用自己对科技与娱乐的理解，重新定义“坐在车里”的感觉与概念。

![](.evernote-content/4B8CDCEF-A347-4743-B2BB-0D5B7D1B5FCA/225FD597-2DEB-4255-A77E-5EE7FC617FDC.jpg)

但索尼只会研发，不会造车。

两年后的2022年，索尼找到了本田。两家同样诞生于日本、同样以技术立身的昭和时代的工业巨头，宣布成立合资公司“索尼·本田移动出行”（SHM）。分工明确：本田负责车体、底盘、制造；索尼负责软件、娱乐、智能驾驶开发。

2023年CES，品牌名“AFEELA”正式发布。2025年CES，第一款量产车AFEELA 1正式亮相，开放预约。

起售价8万9900美元，折合人民币约65万元。这不是一辆普通人的代步工具，而是一辆对标特斯拉Model S高端区间的旗舰电动轿车。计划2026年中在美国开始交付，2027年前半年进入日本市场。

一切看起来，都在有条不紊地推进。

然后，本田在2026年3月12日发表了一份公告：到2026年3月为止的2025年度，本田预计将出现6900亿日元（约300亿元人民币）的巨额亏损——这是本田自1957年上市以来，历史上第一次出现年度最终赤字。若包含2026年度在内的累计损失，赤字额最高可能触及2.5万亿日元（约1082亿元人民币）。

同时，本田还宣布将停止三款电动车的开发与销售：Honda 0 SUV、Honda 0 Saloon、Acura RSX。

本田社长三部敏宏在发布会上说：“我们穷尽了所有办法，但收益依然极为严峻。电动车一旦进入量产与销售阶段，未来只会带来更大的损失。”

这句话，说的是Honda 0系列，但也彻底宣判了AFEELA的命运。

![](.evernote-content/4B8CDCEF-A347-4743-B2BB-0D5B7D1B5FCA/FEED5337-3F11-4635-98D0-E785807E2D2E.jpg)

AFEELA的整个商业命运，是建立在本田提供车辆平台、生产设备和制造技术的前提之上。生产地点，正是本田在美国俄亥俄州的工厂。本田一旦收缩电动化战略，AFEELA赖以为生的地基，也在一夜之间被抽空。

仅仅过了13天，AFEELA的停产停售公告随之而来。索尼人想骂本田人，但是，只能说，当初自己找错了对象。

但若只是把这件事归结为索尼与本田的“战略失误”，未免太过简单。

AFEELA停产的背后，是整个世界电动车市场在过去两年间经历的一场剧烈震荡。

先说美国。特朗普政府上台之后，拜登时代的电动车购置补贴政策被逐条修改，每辆车7500美元的联邦补贴几近消失。加州长期实施的严苛排放法规，也在联邦政策的压力下被迫松动。市场端的结果是：美国电动车需求急剧降温，原本预期的换购潮迟迟未至。

再看竞争格局。中国的新能源车企这几年以令人震惊与羡慕的速度向全球扩张，价格、技术与产品力全面提升。在欧洲、东南亚、中东，来自中国的电动车已经大量铺货。AFEELA这类定价在9万美元量级的产品，不仅要面对特斯拉的正面竞争，还要应对整体市场信心的下滑。

在这样的大环境里，一辆尚未量产、品牌认知度几乎为零、定价高达65万人民币的日本合资电动车，究竟能卖给谁？

这个问题，大概从一开始就没有一个令人信服的答案。

平心而论，AFEELA 1的技术构想，在研发当初是相当超前的。40个高级传感器、800TOPS算力、Level 2+高阶辅助驾驶、91度电的电池组、超过482公里的EPA续航——这些参数，放在今天依然不算落后。

索尼带来的车舱娱乐体验，更是业界无人能及的想象力。索尼PS的游戏生态、索尼的音乐与影视内容、与索尼音乐串流平台Spotify的深度整合——如果这套系统真的落地，坐在AFEELA里的体验，会与坐在任何一辆特斯拉或比亚迪里截然不同。

这辆车死掉，不是因为它不好。而是因为时机错了，结构错了，赶上了一个最坏的年景。

![](.evernote-content/4B8CDCEF-A347-4743-B2BB-0D5B7D1B5FCA/DFBA5A67-B022-4EC3-AB17-B11CBEE6BA98.jpg)

现在，两家公司说，会重新协商未来的方向。

可能缩小规模，可能重组合作，也可能彻底解散这段合作关系。具体答案，暂时还没有。

本田停产电动车，也将重新定义它在中国的电动车产业的布局，重回燃油车的老路。

索尼呢？这家公司在游戏、音乐、影像传感器领域依然稳健，汽车这条路走不通，也未必会伤筋动骨。只是那个“用索尼的方式重新定义汽车”的梦想，大概要搁置很长一段时间了。

索尼·本田移动出行公司（SHM）在我办公室的对面——东京中城的办公楼里，推开窗，就能望见他们，直线距离100米。因为对AFEELA有太多的期待，所以，也有了太强的失落感。也许过不了几天，SHM的办公室将被清空，买午餐盒饭的队列里，再也见不到那些员工的身影。

本田和索尼，都是从战后废墟里一砖一瓦创建起来的公司，骨子里有那种不服输的劲儿。但时代变了。电动车的竞争，不再是汽车行业内部的游戏，而是一场横跨科技、能源、政策与地缘政治的全域战争。

在这场战争里，光有情怀和技术是不够的。你还需要规模、供应链，以及一个愿意买单的市场。不仅需要至臻的完美，更需要速度！而过于的精益求精，等于是在为自己挖坑。

AFEELA最终输掉的，不是技术比拼，而是时间窗口。真的，花了5年的时间，黄花菜都凉了。
