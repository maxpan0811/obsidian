# 我身上的教训

---

智能摘要

工作后经常会被塞很多的事情，但是又不会争取时限上的宽裕，最后只能不停的加班，损害身心。这个方法需要坚持比较长的时间，但是好处是不容易复发。抑郁症也是一种非常容易复发的疾病，跟戒毒的复吸率差不多。我就很多地方没能好好坚持，罗翔说过，人的痛苦就是无法跨越知道和做到的鸿沟。我想算是一个孤独的抑郁症患者的倾诉欲吧，把自己的剖开，把那些平常难堪的不能开口的东西写给那些素不相识的人听，这样会让我有一个宣泄，就像心灵上被洗涤了一次。

作为当年高考全省前十（具体第几不知道，但是拿到了清华新生一等奖学金），选了清华当初最好的专业之一。本硕都是清华出身的人，一直混在互联网这个猪都能飞上天的风口上，40多岁了仍然是最基层的开发人员，title也不过是高级软件开发工程师，混过多家头部大厂，两次因为绩效被开。还得了抑郁症。

写一写自己的教训，算是为年轻人提个醒吧。

1

性格

为了更好的教育孩子，所以看了一些育儿和心理学的书，也听了一些课，回想自己的人生，真的百感交集。

父母的教育对于孩子一生的影响真的很大，所以奉劝所以想成为父母或者孩子还小的年轻人一定要好好学习一下在怎么当父母。

我的父母文化水平不高，但是望子成龙。他们在我小时候包办了我除了学习外的一切事情，并且严厉的监督我的学习。所以我从小学习成绩不错，得以高分考入top2。但是当我独自面对生活的事后，这种教育让我缺失了很多重要的能力：

1.  时间统筹的能力

2.  制定目标的能力

3.  自我管理的能力

4.  日常生活的能力

这些能力的缺失让我的大学生活一团糟，最后差点不能顺利毕业。感谢当时网络游戏并不发达，不然我很可能就被退学了。

同时我的性格也有很多缺陷：

1.  害怕和人发生冲突，所以不会拒绝。工作后经常会被塞很多的事情，但是又不会争取时限上的宽裕，最后只能不停的加班，损害身心。

2.  不喜欢当面沟通，如果需要我及时的对对方的言语做出反应，会让我压力很大。所以和他人对话的时候总是想快点结束，这样的结果要么是对方觉得我不停的在敷衍（为了这个没少跟老婆吵架），要么就是没把事情沟通清除，事后要很多的补救。不喜欢沟通的另一个问题是遇到了困难都想着能自己搞定尽量自己搞定，不会去通过询问整合资源，工作效率很受影响。

3. 害怕跳出舒适区。总喜欢做自己习惯和擅长的事情，避免学习和挑战，造成同样是工作10年，人家是10年的工作经验，我是10个一年的工作经验，自然就没有竞争力可言。

4. 逃避问题。遇到问题总有逃避心理，所以就自然而然的拖着。可是大部分问题拖着只会让问题更糟或者错过机会。

5. 缺乏热情。因为什么都是被安排的，所以对什么都没有热情，对事情总抱着一种得过且过的态度，不断找理由劝自己做到这样应该够了。可是如果什么事情的目标都是刚刚合格而已，那么很可能会有很多不合格的情况发生。我想自己两次被fire都是这种原因吧。

6. 讨好型人格。严厉的父母经常会培养出讨好型人格的孩子，表现就是努力配合对方的期望，崇拜权威，不会拒绝，以他人的评价作为自己的锚点，来确立自己的价值。

7. 不会取悦自己。幼时家贫，所以各种额外的需要很难被满足，长大后就习惯了不为自己花钱。日子过得很简朴，唯一拿来让自己开心一下的就是吃，结果就变成了个200斤的胖子。

所以如果你想将来有所成就，一定要避开我身上的这几个坑。

2

工作

对于工作和事业，总体来说选择比努力更重要。

我的父母只是一个小型国营工厂的普通干部，一直到退休都没有什么头衔。算是稍微有些文化，但是实际上眼光不高的类型。由于家族的原因，他们都很看重我的学习，所以对我的学习抓得很紧。但是因为眼光的问题，他们并不知道应该怎么正确的教育孩子，仍然停留在多花时间多努力的初级阶段。

由于这样的童年，再加上我的亲戚中并没有什么出色人物，我的眼界被学习隔离在了一个很小的区域内。幸运的是我的天赋和性格都比较适合读书，所以一路顺风顺水的，作为一个知名的”别人家的孩子“，考入了清华。

但是当我的人生第一次重大的选择，也就是专业选择的过程中，我并没有任何主见，父母也没有任何考虑。不知道幸运还是不幸，我的成绩基本上可以选任何专业，除了没有任何绘画功底选不了建筑。我就只是随大流的报名了当时清华收分最高的专业。现在想想真是一身冷汗，如果我当初想去的是北大，很可能就会掉进生化那个大坑，毕竟当年生化被陈章良搞得很火。

由于前面说的能力缺陷，我的大学生活并不愉快，总是徘徊在中等偏下。最后保研时只有一个研究计算机在本领域应用的偏门组愿意要我，我就稀里糊涂的去搞了计算机。

毕业的时候因为曾经的项目跟搜索沾边，我也拿到了百度的offer。但是当时一个很照顾我的师兄在创业，希望我去帮他。我这种不会拒绝的性格就放大了师兄那边的权重，最后选择了去师兄的创业公司，一呆就是四年。

师兄对我一直还不错，但是他的合伙人是从国有背景的网站出来的，弄得这个公司的管理和目标都很混乱，一手好牌也是打的乱七八糟，尤其是后两年，我在这个小公司基本上学不到什么东西，每天干着重复的事情。

但是即使是这种情况，我也没有赶快辞职。这让我很后悔，我耽误了自己最宝贵的时间，磨灭了工作的热情，养成了很多不好的工作习惯。当我在老婆的催促下，下定决心换个工作时，已经白白耽误了很多时间。

这段时间里，我既没有积攒到好的工作经验，也没有积攒到买房子的钱，可谓失败的开始。后来不得不付出几倍的钱才在通州买到一个小房子。

从师兄的公司出来，我因为漂亮的学历和面试的超水平发挥，幸运的被知名外企录取。但是在这个外企工作的过程中，我的缺点逐渐的暴露了出来。

首先是不主动，总是等着manager将要做的事情分配给我，而不是主动发现哪些可以做的技术方向。

其次是不合群，因为沟通障碍，不喜欢和别人打交道，包括一起去吃饭。因为缺少这些交流，我对公司内部的各种事情和动向完全不知道，对各个其他的team几乎一无所知。造成当我的工作遇到问题时，我都不知道该去找谁。当我需要决定做什么的时候，完全不知道现在manager们关心的是什么事情。

最后是完美主义，因为自己其他方面不行，那么就把精力都投放在工作的细节上，希望通过努力把事情做的漂亮来获得业绩。但是如果方向不对，越努力越失败。也许这个模块只需要做到70分，那我努力把他做到90分有什么意义呢？因为不会自己找前进的方向，我的manager估计会觉得管理我特别费劲吧。

虽然这个公司比较nice，我通过不停的低效加班也能跟上进度，但是公司也从没有给我涨级。收入自然也就一直上不去。随着婚后买房的压力越来越大，我就跳槽到了一个互联网的头部大厂。

我在这个大厂的时间很短，主要是我这种书呆子实在是适应不了这个大厂里面狂野的做事作风。我进去的时候感兴趣的部分在做了三个月后就因为组织变化，被别的team拿走了，而我所在的组被派去写windows的客户端。即使我几次沟通说自己不想写这个，但还是被按着头去写。最后不到一年的时候我跟我的manager的manager商量，正好我想走，他也有末位淘汰制的需要，就这样，我第一次被fire了。

因为提前有准备，我无缝衔接的进入了另外一个大厂。现在想想，这个大厂里面4年的工作经历对我来讲是个噩梦。9106的工作状态，KPI的高压，让我在这个期间得了抑郁症。在这个公司的经历仍然是连滚带爬，不善于沟通，不能自己想出需要做的方向，花太多时间在细节上。这样磕磕绊绊的在第二年的时候，被赶到了一个下属公司，算是又被fire一次。但是搞笑的是半年后这个下属公司被解散，我们这些技术人员又被接了回来。后面就是各种咬牙坚持，又有了被fire的经历和恐惧，我在这个期间得了抑郁症。在确诊之后，医生建议我要脱离以前的高压环境，不然对我的病情非常不利。所以我就又辞职了。

现在的工作是一家外企，非常幸运到遇到了一个很好的manager，虽然跟我同岁，但是经验和眼界都很高。他发现了身上很多的问题，并且耐心的给我指出。我是通过他才真的明白了自己工作中真正的问题在哪里，而不是总觉得自己的问题只是不够努力。虽然很多性格问题不一定能够改正，但是至少能让自己死个明白吧。

啰哩啰嗦的说了很多，总结一下给年轻人的经验教训吧：

1. 一定要扩张自己的视野，主动选择自己感兴趣而且有前途的行业，不要像我一样随波逐流被推着走。

2. 学会沟通，一群人的能力总是强过你一个人的，通过沟通你能够减少很多不必要的麻烦。

3. 多思考，至少是先思考再动手，事情没选对，多做多错。

4. 管理老板的预期，在项目的前期中期后期都要和manager好好沟通，让他知道你的进度。

5. 毕业后尽量去大公司，小公司工作的难度普遍低于大公司，所以对你的技术能力没什么帮助，而且会让你养成很多坏习惯。

6. 该决断的时候不要拖泥带水，还是那句话，拖延大部分时候只会让事情越来越差。

7. 跟一个好的manager非常重要，我第一份工作就是个反面典型，好好的方向和资源，最后做的稀烂，就是CEO私心太重。

8. 多和同事一起吃饭或者其他活动，很多信息是这个过程里面得到的，很多想法也是在这个过程中萌发的。

9. 早睡早起别熬夜，每天精精神神的。

3

健康

身体健康什么的我想大家都懂，我就不啰嗦了，我着重讲一下抑郁症。

很多没有经历也没有文化的人总觉得抑郁症只是简单的想不开的问题，比如我的父母就这么认为。

当你痛苦的只想窝在一个角落让自己慢慢恢复点能量的时候，他们总是说些个听起来好像正确，实际上什么用都帮不上的廉价安慰。

这种安慰只会让我觉得更烦燥，忍不住想和他们吵架，最后我的抑郁症就成了家里不能提的事情。

抑郁症的原理其实很简单，人的思考习惯会影响大脑的物理结构。比如实验就表明出租车司机大脑中负责处理地图相关的部分就会更发达。

抑郁症患者相当于在大脑的回路中建立了一条负能量快速通路，这个通路让你更习惯去思考负面的东西，从而让你习惯于低落的情绪中。

抑郁症的成因医学上一直没什么定论，在我身上的表现就是：

1. 心情低落，总是感觉心里空荡荡的，什么事情都提不起劲来

2. 意志减退，完全无法控制自己有效率的工作生活，不想说话，不想吃饭也不想工作

3. 暴躁易怒，容易因为一点小事突然爆发

4. 思维迟缓，记忆力和思考能力都下降

5. 腹背疼痛，有时候会觉得身上疼，尤其是遇到难过的事情时

6. 焦虑，有时候会感觉心里很慌，嘴唇止不住的抖

我第一次抑郁症发作是在上一家公司，那个时候我刚换了一个领导。应该说我之前就有抑郁倾向，这个领导只是加重了我的病情。这个领导年龄比我小，是从另外一个大厂跳槽过来的。可能是因为我比他年龄大让他不舒服了吧，感觉他在工作中总是针对我的样子。你事情做好了他没赞赏，你事情出状况就会在组会上当着所有人的面批评你。那段时间搞得我心情非常低落，每天都是带着一种上坟的心情去上班，再加上早出晚归的不规律生活，终于压垮了我的心态。我在家里总是把自己关在小房间，和家人交流的时候也容易不耐和发怒。做事很极端，曾经因为一点小事就跪下求父母闭嘴。

感谢老婆受过良好的教育，发觉我的问题后带我去看了心理咨询，虽然很贵，但是让我找到了自己的问题。随后去北医六院确诊了重度抑郁症，也被告知我现在的工作环境不利于我的病情，随后就是辞职在家吃了三个月药，初步控制了住后找了一个传统行业的外企，能够965的那种。吃了一年零三个月药后觉得病情差不多控制住了，就在医生同意下停了药。

第二次发作是因为家庭矛盾，主要是因为育儿理念的不同造成了父母和我们夫妇之间的矛盾。作为夹在中间的我承受了很大的压力，可是偏偏我是一个最不喜欢跟人发生冲突的人。当两边开始闹情绪而不是讲道理的时候，我就成了两边喷的对象。结果就是要么我就得早点回到在家里无论和哪个家人单独说话，用不了几句就会拐到雷区的感觉。于是我的抑郁症就又犯了。这次吃了半年的药，直到疫情让我没法去医院。可是疫情把所有人都困在家里，那种感觉更让人崩溃。

下面说说心理咨询师对我的抑郁症的一些建议，分享给大家吧。

1. 正确地自我评价。当咨询师跟我说明不顺心是生活的常态时，我心理莫名的轻松了很多。她的客户包括一些老总，也有一些红墙大院里的孩子。在表面的光鲜之下，都有自己的烦恼和痛苦。后来我想了想，确实即使我认识的几个在我看来很成功的朋友，也都有自己的麻烦和苦恼。所以我不是特别差特别倒霉的那个。

2. 对自己好一些。原生家庭经济条件不好的孩子，会习惯于自己的非必要需求被压制。即使长大后经济条件变好，也不习惯在取悦自己的地方花钱。她问我：你有没有给自己买过什么非必要的贵一点的东西？我想了想，什么都没有。为什么？因为觉得不值得。想想多可悲，自己的需要和快乐竟然感觉不值得。在我停止了哭泣后，咨询师建议我花5千块钱，给自己买些自己喜欢的东西。然后我就买了自己童年梦寐以求的游戏机，虽然后来因为忙还有孩子的原因不怎么玩，但是当时开箱时的幸福感真的让我现在想想都感动。

3. 去看医生，并遵循医嘱。抑郁症是一种病，而且不是单纯的心理疾病，千万不要想着我只要想开了就好。你的大脑回路时时刻刻的想把你拉向深渊，在新的回路建立之前，你需要药物的帮助。

4. 改变认知。现在一种比较流行的治疗抑郁症的方法是认知疗法。这种方法认为你改变不了事物，但是可以改变对于事物的认知，而认知影响情绪。这个方法需要坚持比较长的时间，但是好处是不容易复发。

抑郁症是一种被忽视但是却很常见的心理疾病，在中国的发病率大约4%-4.4%。抑郁症也是一种非常容易复发的疾病，跟戒毒的复吸率差不多。我就很多地方没能好好坚持，罗翔说过，人的痛苦就是无法跨越知道和做到的鸿沟。

最后，如果你的老板PUA你，还是尽快辞职吧，不值当的把自己的人生搭进去。CNM的李XX

4

家庭

家庭的事情不想写了，各家有各家的问题，就说几个我的经验教训吧：

1. 尽量别和父母同住，麻烦比方便多

2. 多看育儿书，即是学习经验，也是一个重新认识自我的过程

3. 尽量住的离上班的地方近，每天长时间的通勤让人崩溃，也让家庭崩溃

4. 找老婆尽量找学历高的，三观正的

5. 年轻人要把控好自身，不要一时贪图身体的欢愉最后落入无尽的悔恨，这是我高中时代一个好哥们的悲惨教训

6. 孩子要自己养，别扔给父母，不然会有很多的心理问题。这是我一个大学同学离婚后跟我哭诉他老婆得来的经验。

PS：有人在我的总结里说xb、装逼、凡尔赛，我就不回应了。鲁迅说，人的悲喜并不相通。每个人的艰辛只有自己知道，你的担忧、你的恐惧和你的愤恨只有你和你最亲近的人在乎。我想算是一个孤独的抑郁症患者的倾诉欲吧，把自己的剖开，把那些平常难堪的不能开口的东西写给那些素不相识的人听，这样会让我有一个宣泄，就像心灵上被洗涤了一次。

感谢你们听我这些絮叨，希望能有一两句对你们有用，谢谢。
