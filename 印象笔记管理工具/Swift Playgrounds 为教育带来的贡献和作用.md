# Swift Playgrounds 为教育带来的贡献和作用

---

Michael Cohen 是一名杰出的苹果教育家，Michael Cohen 在担任一个科技栏目嘉宾时，专门着重谈及了苹果 Swift Playgrounds 为教育所带来的贡献和作用。

![](.evernote-content/C6D53D7A-E3D4-4323-AEF3-C71B73438D24/9BFE7113-8E90-444E-90EB-BA77EB3F9495.jpg)

苹果公司在 2014 年带来 Swift Programming 的时候，当时他们的目标是那些开发人员。Swift Programming 作为一种更具有弹性和灵活性的语言，它被采用，并且在大多数的情况下被众多的开发者所接受。而不到 3 年之后，苹果又重新定义和改变了针对开发者们的方案，而这次除了开发者以外，苹果更是希望打开那些 9-16 岁学生编程的大门。

很显然，通过 Swift Playgrounds ，苹果有机会重新定义并且重新定位一股不可阻挡的教育技术力量。

![](.evernote-content/C6D53D7A-E3D4-4323-AEF3-C71B73438D24/5412E94A-950E-457F-81C4-C5A1CA8276D8.jpg)

Michael Cohen 认为，Swift Playgrounds 的灵感来源于当前教育领域的编程趋势，而苹果也旨在通过 Swift Playgrounds 这个较新的教育平台，展现自己强大的教育技术力量。

相信很多朋友都已经了解什么是 Swift Playgrounds 了。可以说 Swift Playgrounds 在编码教育世界里是独一无二的，因为大多数的编程平台都缺乏真正的程序语言交互，它们都缺少一些引人入胜的视觉效果。

我们都知道，编程在不少人的心目中，都是非常枯燥乏味的。但是用苹果的话来说，他们希望所有人“以真正有趣的方式”，在 iPad 上学习那些严肃的代码。

![](.evernote-content/C6D53D7A-E3D4-4323-AEF3-C71B73438D24/92B45B1A-15EB-413F-8C0E-1AB9BD8F910A.jpg)

和世界上任何一种语言一样，我们都需要通过学习去掌握编程，而 Swift Playgrounds 用更具有交互性的体验，来将编码学习融入其中。它通过一些具有吸引力和令人兴奋的游戏体验，为年轻、充满灵感和雄心勃勃的学生们打开了编程的世界。而那些更年轻的开发者们，终于能通过 Swift Playgrounds 来实现自己的梦想。

可以说，通过 Swift Playgrounds ，苹果并不是要创建一代开发人员，他们是要影响一代人。它会去了解和评估开发者们，然后编程和设计应用程序可以成为他们能实现的目标。这些学习者可以继续成为 CEO、医生、经理、教师或者其他职业，这些职业虽然不一定会与“编程”直接相关联，但是，他们能用新的技术、洞察力和进步的愿景去进入各行各业。

毫无疑问这一举措也能巩固苹果公司如今的成就，苹果正在为教室中大量采用编码铺平了道路，这也会使得更多学生准备创建属于自己的 App ，即使微软和谷歌现在也开始在这方面起步，但从目前的情况来看，微软和谷歌已经远远落后了。

通过超越数字体验，编程可以帮助学习者在批判性思维，复杂的程序解决和创造力方面获得更多的技能。这些技能在未来充满自动化、机器人和人工智能的世界里，至关重要。

![](.evernote-content/C6D53D7A-E3D4-4323-AEF3-C71B73438D24/AF33FCDB-20F9-473C-8F59-23986ACD9305.jpg)

而对于教育来说，如今的 Swift Playgrounds 等编程应用程序已经超越了数字体验，它们用创造性的方式去开拓我们的技能。即使学生们没有将编程当作是自己的专业目标，但是这些编程技能，同样会对他们未来所从事的专业和行业产生积极的影响，并且让他们冲击更高的高度。

Swift Playgrounds 看起来已经为未来做好了准备。编程是所有新兴和尖端技术的一部分。我们不需要削减目前尚未填补的，成千上万的编程和计算机科学相关的工作。在个人层面上，我们生活的社会，情感和经济因素直接受到不同的程序设计所影响。它超越了技术功能，成为了人们去思考，设想和思考编程可以做什么来影响和增强我们周边世界的催化剂。而这一切，只是一个开始。

教育依然需要发展，至于未来的教育会变成什么样，很多情况都是未知的，不过，苹果正在重新改变，苹果也希望在生产技术和教育方面，做出更多的贡献。
