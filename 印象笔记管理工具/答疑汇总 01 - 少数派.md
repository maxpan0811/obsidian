答疑汇总 01
=======

在开始第一期答疑汇总之前，首先感谢购买这份教程的各位。各位的支持是我写作的重要动力之一。

在第一章中的《调整到合适的版面》这一小节中，关于字号我写道：

> 不过对于字号，根据我的经验建议，无论是在 Macbook 上还是外接显示器，**字号最好不要小于 14 号**，否则字会太小不易阅读。

这一句放在对界面板式的讲解后，误导部分读者的理解，以为在「偏好设置 - 一般」中可以直接设置编辑界面内的字体字号。其实在 Ulysses 中的偏好设置，修改字体时只能选择字重（粗体、细体等），而不能选择字号。

如果想改变编辑界面内的字体字号，点击菜单栏中的「视图 - 缩放」就可以进行调整。

缩放字号

读者 @jjsgrl 提问道：

> 想问一下插入图片后在导出的时候能调节大小和添加图注吗？

在 Ulysses 编辑器内，图片尺寸是没法调整的。这种设置是符合 Ulysses 一以贯之的设计哲学的，写作时尽量不让其它元素抢占你的注意力。所以图片尺寸和位置都没法调整。而预览导出的时候，可以通过样式控制图片尺寸。

插入图片弹出窗口中的「说明」一栏，就可以为图片添加图片说明（图注），在预览的时候会显示出来（同样有样式控制呈现形式）。

细心的你可能会发现，在插入图片和链接时，都会有一栏「标题」。其实它对应着 HTML 中链接和图片对象的「title」属性，在导出时是不可见的。对它的使用需要一定的 HTML 知识，而且对普通用户用处不大，故而不做详细介绍了。

第一章发布会后，很多读者都来询问关于 Ulysses 中的 CSS 部分何时讲解。我会在后续更新中，花大篇幅为大家讲解 CSS 和 Ulysses Style Sheet。

很高兴看到大家有这么高涨的积极性，如果你想在之后的教程中学习一些特定的内容和用法，就像给应用开发者提需求一样，大家可以直接私信我。如果是普适性的内容，我就会将它加入正文之中。
