# 伤痕文学为何让人厌恶？

---

来源：[打开原文](https://mp.weixin.qq.com/s/Nb4AQWElGuylyGJIgqTxxQ)

↑↑↑点击上方蓝色“东亚男士”，关注后了解更多精彩内容
---------------------------

最近这些年，民间舆论这一块有了很大的反转。
---------------------

过去，人们对伤痕文学是非常热衷的，认为其“深刻”、“清醒”、“独立”，甚至将其视为“良心”。

可现在，人们对伤痕文学越来越厌恶了。
------------------

这到底是为什么呢？
---------

本人认为，这源于一个根本性问题：谁的伤痕？
---------------------

而由这个根本性问题，本人想到了其中的两方面。
----------------------

【01】
----

特权阶层的矫情。
--------

关于此，本人想到了某部不记得名字的电视剧当中的一句台词。
----------------------------

其中的剧情是，一位干部的女儿在特殊时期被下放到了农场，干的是挤牛奶的活儿。
-------------------------------------

干了一段时间以后，她撕心裂肺地失声痛哭，喊出了那句台词：

我的手是弹钢琴的，不是挤牛奶的。

说真的，听完这句台词以后，本人对那部剧的这个角色非但没有产生半点同情，反而感到一阵剧烈的恶心，以至于巴不得把八辈子吃的东西都要吐出来。

这句台词，真的太恶臭了！
------------

包括挤牛奶在内的一切劳动，是普通民众的日常。
----------------------

他们没有抱怨一句，都在埋头苦干，并早已习以为常。
------------------------

而你一个干部家的女儿，过一下普通民众的日常生活，就成伤痕了？
------------------------------

这也太矫情了吧？
--------

而那句“我的手是弹钢琴的”，充满了一种令人生厌的阶级傲慢，仿佛他们处在高位、不事生产是理所应当的一样。

这放在如今的语境当中，就相当于那些拧开矿泉水瓶盖、把自己的箱子抬起来放在高铁的行李架上，就自称“新时代独立XX”的人群一样：在其潜意识里，其就该被他人伺候着。

一介草民的本人跟这种人共情，那简直太尴尬了，一如月薪三千、身背房贷的打工族给大导演“走个面儿”一样。

【02】
----

模糊正邪的庸俗人性论。
-----------

所谓人性论，大致意思就是“人是立体的、多面的，没有绝对的好人与坏人”。
-----------------------------------

这本身没错，而且还成功塑造过一个非常具有代表性的角色：《亮剑》里的李云龙。
-------------------------------------

李云龙作为剧中的一号主角、正面人物，却也被安排了不少缺点。
-----------------------------

但这些缺点非但没有减弱他的光辉形象，反而强化了。
------------------------

因为，我们从中感受到的是一个有血有肉、和我们非常贴近的英雄人物。
--------------------------------

李云龙这个角色之所以受到追捧，其中一个很重要的原因就是

对过去的角色的脸谱化塑造的反感。

曾经的影视作品，正面人物一定是高大全的形象，但却给人的感觉是假大空，而反面人物则一定是又坏又蠢。

于是，人物便显得过于扁平与单调。
----------------

南斯拉夫经典电影《桥》与阿尔巴尼亚经典电影《宁死不屈》在中国上映以后能够引发热潮，其中的一个原因，也是因为这两部电影中的反派德国法西斯虽然邪恶，但是奸诈、聪明，所以击败这样的对手，才凸显出了保家卫国的真正价值。

此处也插一句，《宁死不屈》中的地下工作者米拉这个角色，是当时的中国男青年心中的“外国美女”的样板。

![](.evernote-content/E5FE7CB6-6D93-46A7-8FA4-DF8C6635299A/14F13ADB-E25F-4720-ACD2-347E64BA3DDD.png)

↑↑↑阿尔巴尼亚电影《宁死不屈》中的米拉：扮演者库麦
--------------------------

正因为脸谱化的角色塑造显得扁平、单调，人们才会对人性论指导下塑造的多面、立体、复杂的角色产生好感。

但是，我们可以明显感受到，人性论已经走向了庸俗，甚至走向了另一个方向的脸谱化。
---------------------------------------

在庸俗人性论的教条之下，正面人物一定要被强行安排各种缺点甚至污点，反面人物一定要被强行安排所谓“人性闪光点”、“心路历程”、“成长创伤”、“童年阴影”、“事出有因”等等。

除此之外，正面人物的一切崇高也必须被解构。
---------------------

他们做任何事情的动因必须被安排成利己的，不可以被安排成利他的。
-------------------------------

比如说，科学家回国是因为“美国种族歧视”、平民参与抗战是因为家人被侵略者杀害了等等。

并不否认，确实不少崇高都来自于利己的动因。
---------------------

但是，痴迷于庸俗人性论教条的导演想表达的却是“世界上根本没有崇高”、“要不是被某个条件触发，他们就会自私自利一辈子”，这就属于以己度人了。

与之对应，反面人物的一切罪恶必须被洗白。
--------------------

他们作恶，永远不是自己的错，永远是国家限制了他、体制迫害了他、时代辜负了他、社会冷落了他。

因此，庸俗人性论达到的实际效果就是让善恶、正邪的界限变得模糊，甚至让善恶、正邪彻底颠倒。

而这也是伤痕文学的惯用套路。
--------------

我们发现，伤痕文学当中的政府人员、军队政委一定是令人反感的，反倒是侵略者、流氓、地痞、街溜子、汉奸都是充满人性光辉的。

讨论热度很高，但票房惨淡的冯小刚的最新电影《抓特务》就是“庸俗人性论”的典型：潜伏特务被塑造成了人畜无害、有情有义的白莲花，公安干警反而被塑造成了敏感多疑、疑神疑鬼的偏执狂。

而这部剧的原名，则让冯小刚的用意昭然若揭：《比窦娥还冤》。
-----------------------------

冯小刚的电影走下坡路，也正是从伤痕文学叙事开始。
------------------------

![](.evernote-content/E5FE7CB6-6D93-46A7-8FA4-DF8C6635299A/69ADF40E-15E5-488B-BADD-B9A06174DF45.jpg)

由以上内容可知，伤痕文学之所以让人厌恶，是因为所谓的伤痕，是特权阶层与人渣的伤痕。

而所谓伤痕的讲述者均是春登群体（←点击阅读）。
-----------------------

换句话说，掌握话语霸权的春登群体

根本不站普通良善民众一边，而是站在特权阶层与人渣一边。

曾经，网络社交媒体没有那么发达，信息传播是中心化的。
--------------------------

春登群体利用报纸、杂志、电视、广播等媒介，讲述他们的所谓伤痕。
-------------------------------

再加上那个时候，民众也确实因为种种原因，普遍有反体制的想法。
------------------------------

所以，伤痕文学因为迎合了那时的人们的想法而广受欢迎。
--------------------------

如今，网络社交媒体发达了，信息传播是去中心化的了。
-------------------------

在“人人都可以是媒体”的时代，春登群体的话语霸权遭到了动摇。
------------------------------

人们逐渐在去中心化的信息传播与交流中明白，春登群体的所谓伤痕，不过是让他们过普通人的日常生活，顺带着抹黑正面人物、洗白反面人物。

这下，他们的矫情与险恶便昭然若揭了。
------------------

那么，伤痕文学也就自然不再有市场了。
------------------

而这一切，又牵扯到了一个极为重要的问题：立场。
-----------------------

所谓“文艺无立场”，就与“体育无国界”一样，都是骗人的谎言。
------------------------------

大道理，本人不必说。
----------

各位就只看一点：那些声称“文艺无立场”的人，是不是永远站在汉族无产阶级顺直男性的反面？没错，这就是他们非常明确的立场。

在有些人那里，在工地挥汗如雨、在车间加班加点、为避免超时而逆行导致重伤、手指被机器压断，不叫“伤痕”。

在有些人那里，地铁上被“下头男”看了一眼、脱产考公连续九年没上岸、相亲被“抠搜男”要求AA、网约车司机没帮我自己拿行李、高考结束后父母不拿出10万元给自己潇洒、高考不到300分却硬要复读而遭父母拒绝、家庭并不富裕却硬要留学而被父母阻止，叫“伤痕”。

后者的所谓“伤痕”，在前者眼中完全就是小题大做。
------------------------

身为前者的你，去共情后者，您没事儿吧？
-------------------
