# 龙某人，真的“反战”吗？

Original申鹏平原公子

台湾的龙女士，最近几年有点慌不择路、口不择言。

她开始翻来覆去地讲“反战”，车轱辘话反复说，口口声声就是“无论如何，我反战”。

她四月份纽约的一场新书分享会上，面对台下听众说了这么一句话：“要和平，不要战争，发生战争双方都有责任，但是更大的责任在强的一方”。

谁都知道她在阴阳什么，但当时刚好是美国正在入侵伊朗，所以她的意思是“美国责任更大”吗？

她4月18日在东京大学发表的《当战争不结束》的演讲，三观更加逆天，不谈是非对错，不谈历史和现实，开口就是什么“焦土视角”、“白骨视角”……意思还是一个，就是“反战”，要“和平”。

废话，谁不反战？谁不追求和平？

然而从去年到今年，美国和以色列发动了多场战争，手段包括暗杀、恐怖袭击、空袭、偷袭、武装入侵，甚至还屠杀平民，制造了无数焦土和白骨……龙女士是一个字都没提。历史上真正的“反战”，都有具体的对象，当年美国大学生反战，矛头直指发动越南战争的美国政府。请问龙女士用中文发言，用简体中文发帖，你反战的对象是谁？

她还专门发帖指出，她在东京的大学的演讲，就是针对“简体中文用户”的，意思是，专门讲给我们听的……这就离谱了，近代以来，我们是被侵略被压迫的一方，我们只是在反抗，在自卫 ，我们从未主动发起过对外战争，轰炸伊拉克、叙利亚、伊朗的从来不是我们，凭什么要对我们喊话？凭什么要我们反思？

她还说——历史上“真和平”的到来，往往不是通过炮火，而是通过“强国的谦卑”。战争的代价远超过和平，因此“强势者”必须记得和平最大的责任在“强国”，而且，和平不是慈善，不是让利，而是强国对自己最有利的投资。

不是，这“强国”，指的是美国吗？我可看不到美国有半点的谦卑。

大家有没有发现，最近这种“反战”的声音特别多，天天像个苍蝇一样在耳边嗡嗡嗡……讲难听点，你不能只在美国霸权衰落、美国走狗朝不保夕的时候喊“和平”、叫“反战”。早些年你们的口号可不是这些，而是“石要过刀，茅要过火，人要换种”，“宁可错杀一千，不可放过一个”。

当年你怎么不劝反动派、帝国主义走狗“爱好和平”呢？

龙女士所代表的某些知识分子、旧文人一贯如此，她们总是把自己看成跳出三界外不在五行中的自由人，只关心自己的小情调小情怀，她们嘴上喊着远离政治，实际上却从头至尾都要依赖政治而生存。

她这种人，连自己到底是什么人都搞不清楚，她在台湾长大，赴美留学，嫁给了一个德国人，生了几个混血孩子，写了几本文青小书，就以为自己是“世界公民”了，可以站在“人类”的制高点，对国家和民族等宏观命题指手画脚，贬低自己的故乡，推崇欧美日本，整天说一些似是而非小确幸的鸡汤废话，就真的自以为是大师，有文人风骨了。

李敖骂的非常好：“龙某人最拿手的是写“现象”，她最蹩脚的是只会写“现象”——瞎子摸象式的“现象”。她不会解读“原因”，也不会阐扬正义，她还喜欢说风凉话，怪“中国人你为什么不生气”，其实有良知的中国人早被这个洋人弃妇气死了。她只会写“现象”，不会写“原因”，因为她不知道“原因”。更严重的是，把“残山剩水”看成“大江大海”，这就连“现象”都看走眼了。明明是“残山剩水”，又何来“大江大海”？”

龙女士有句名言，叫做“我不要大国崛起，只要小民尊严”，我不知道是何等愚蠢的脑袋，才会把“大国崛起”和“小民尊严”对立起来？在近现代史上，通常只有大国崛起，才有小民尊严！

龙女士的父母在民国生活过，在那个租界遍地，洋人享有“治外法权”的时代，请问他们有“小民尊严”吗？哦，我忘了，他们不算“小民”，民国遍地的饿殍，才是“小民”，他们有尊严吗？“景明楼事件”中被美军凌辱的民国名媛们，被国民党枪杀的左联作家，还有被青帮、蓝衣社套麻袋丢进黄浦江的进步学生……他们有尊严吗？

龙女士曾经说她外公家是名门望族，纵横长江盐帮，抗战时期龙女士的母亲自带着自己家于运盐船队在长江畅通无阻。所以李敖才会——抗战时期你家能纵横长江水道贩卖私盐，你家和汪伪是什么关系!

龙女士假装不关心政治，实际上她就是个披着文青皮的政治生物，甚至一言一行全部和政治有关。

2016年，龙女士跑到香港大学开讲座，面对香港的高校师生，大谈音乐与政治无关，却又鼓吹《绿岛小夜曲》等政治味道浓厚的歌曲，台下一片沉默，龙女士又问在场的香港师生“哪首歌是你的童年启蒙歌？”结果有人回答：“是《我的祖国》。”

龙女士很尴尬，问现场的同学：“《我的祖国》怎么唱？”结果全场师生给她来了个大合唱——“一条大河波浪宽，风吹稻花香两岸”。这人龙女士很尴尬，她一来低估了普通民众的家国情怀，二来太虚伪，一边口口声声说着“音乐和政治无关”，一边又别有用心地贩卖自己的政治观点。双重标准的嘴脸露出来，文人的清高和傲骨，就都喂了狗了。

比如她在纽约的演讲，同样让所有人都不舒服，她说她在一本书里写了个丛林里的日本老鬼子，浓墨重彩讲鬼子写情书写家信……结果一帮抗战老兵非常生气，问她为什么要“和鬼子共情”，然而她还觉得自己很委屈。

她永远装着一副客观公正、悲天悯人的嘴脸，实际上永远站在加害者一边，永远站在绝大多数人的对立面。

在龙某人的笔下，她会同情国民党反动派，同情落魄的国民党高官，甚至同情某些日本侵略者，但她就是不能同情1949年以来的新中国和新中国的人民，她无法理解我们70年波澜壮阔的历程，不能理解我们筚路蓝缕、艰苦奋斗、为了富强繁荣作出的努力，她无法与我们大多数人共情。她是国家崛起、民族复兴的局外人，所以只能阴阳怪气，贬低人民的成就，满足自己孤芳自赏的矫情。

看到龙某人，你就会知道当年那个旧世界为什么会被摧毁？那些反动派为什么会被打败？因为他们永远站在人民的对立面，哪怕装得温文尔雅、宽厚仁慈，他们都改不了冷血自私的本性。

没有人喜欢战争，但莫名其妙不讲道理的“反战”，更是居心叵测。如果有人勾结帝国主义，时刻威胁我们、敌视我们、对我们置之死地而后快……说得明白一点，有人拿枪指着我们，你还劝我们反战、和平、“有强者的谦卑”，你是不是有病？

真要反战，可以劝帝国主义及其走狗放下武器、早日投降。
