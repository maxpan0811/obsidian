# 用 Mac 的 USB-C 接口，只是充电的话真的有点可惜了

有时候因为这事儿，我的背包里一下子多了好多转接头、适配器、扩展坞这种零零碎碎的小东西，出门忘带一个就得现找，特别麻烦。以前那种专用的 HDMI 接口没了，SD 卡槽也一度被砍掉了——虽然后来 Apple 把它加回来算是“有点良心发现”了；再加上电脑充电也得通过那个 USB-C 接口，同时还要负责外接显示器、传文件、接设备，用起来就感觉挺别扭的，像是被限制住了手脚一样。

可后来我慢慢发现，USB-C 好像并不只是用来充电的，它能做的事情，其实多得超出我之前的想象。

![](.evernote-content/51FAE433-EE7E-4621-A7A8-A6E3876EAEF9/523E1FC2-482C-4AB6-B77F-9D71CF1373FE.png)

尤其是用 Mac 的人，像我现在用的是配了 Thunderbolt 3 或 4 的新款 MacBook Pro（当然 Air 也一样），你会发现这个看起来不起眼的小接口，其实能把整台笔记本变成一个全功能的工作平台。

所以呢，我就想借这个机会，把我平时用得最多、而且觉得特别好用的一些 USB-C 的玩法分享出来，一共有 10 个，可能有些你早就试过了，但也可能有一些你没想到它还能这么用。总之，都是我用下来觉得真的很实用的功能，希望能帮你把 Mac 用得更顺手、更强大。

### 1. 连接外接显示器（最高可达 6K）

先从最直观的开始说起：**视频输出**。

你的 Mac 其实可以通过 USB-C 直接驱动高达 **6K 分辨率的显示器**（比如 Apple 自家的 Pro Display XDR），而且完全不需要又粗又笨的 HDMI 线。

![](.evernote-content/51FAE433-EE7E-4621-A7A8-A6E3876EAEF9/A4DF2DE8-F0C8-4BFE-8917-1B7C766880B5.jpg)

我自己平常用一根 **USB-C 转 DisplayPort 线**，就能把 MacBook Pro 接到 4K 显示器上。无需扩展坞，也不用额外电源，**一根线同时搞定视频、音频和充电**，非常干净。

如果你用的是支持 USB-C 输入的显示器，那更是绝配 —— 桌面上只有一根线，真正实现“桌面整洁控”的梦想。

**小贴士：** 想要用双屏？M 系列芯片虽然在双屏上有限制，但搭配支持 DisplayLink 的 USB-C 扩展坞，可以轻松破解这个难题，特别适合在家/公司多地办公的人。

### 2. 连接高品质音频设备

别一提到 USB-C 就只想到给电脑充电或者听个 Apple Music，其实它能干的事儿可多了去了，尤其在音频这块，真的是很能打。

我自己就用 USB-C 接过不少东西，比如那种能直接录人声、录乐器的专业音频接口，也接过用来提升耳机音质的 DAC 转换器，效果立马能听出来不一样；还有像 Shure MV7 或 Blue Yeti X 这种专业麦克风，插上就能录播客或者直播，声音干净又清晰，根本不用再折腾一堆设备。

### 3. 相机直连导入照片

还记得以前要找 SD 卡读卡器、再插电脑导照片的日子吗？

现在只要一根 USB-C 线，就能直接把现代相机（特别是无反）接到 Mac 上。

我用的是佳能相机，一接上电脑，Lightroom 就自动打开并开始导入 RAW 原图，整个过程流畅自然。

有些相机还支持“**实时连拍预览**”，摄影师在棚里拍照时，图像能马上显示在 Mac 上，非常适合专业场景。

### 4. 反向给 iPhone、AirPods 充电

其实这个功能很多人平时可能都没太留意到，Mac 其实是可以给其他设备反向充电的，而且有时候真的能派上大用场。

像我有时候坐飞机，座位那儿根本找不到能插头的地方，结果手机快没电了，这时候我就直接把 iPhone 插在 Mac 上，它就能开始充电，瞬间稳住局面。还有一次是开会前才发现 Magic Mouse 电量居然红了，来不及找充电器，就直接插在 Mac 上充了几分钟，也能撑一阵；有一次出门，AirPods 突然没电了，也是直接拿数据线连在电脑上补了一点电，真的是随插随充。

这些小场景听起来好像不是什么大事儿，但真碰上那种“差一点点就掉链子”的时刻，Mac 能变身移动充电器，真的特别救急，很实用。

### 5. 从外部 SSD 启动 macOS 系统

你可能会想，干嘛还要多装一个系统？其实理由挺多的，我自己就碰到过几次：

一方面，有时候我想试试 macOS 的测试版（Beta），但又不想拿主系统冒险，毕竟这些测试版本 bug 一堆，用外接 SSD 装一个就完全不怕崩；另一方面，我平常工作和个人用电脑的需求不太一样，所以我会把一个系统专门用来写代码办公，另一个留给日常生活用，分开之后整个人都清爽了不少，也不用担心资料混在一起；还有一个特别实用的点是，我平时用的那块 SSD 上装着完整系统，只要随身带着，去哪儿都能直接插在别人的 Mac 上继续用，像带着一个“移动的我”一样，非常方便。

![](.evernote-content/51FAE433-EE7E-4621-A7A8-A6E3876EAEF9/E8ABC5C0-194F-441E-BE68-B964E8EAAD6F.jpg)

我现在用的是三星 T9，这块 SSD 的速度说实话已经跟内置的没太大区别了，系统跑起来特别顺，体验感非常接近装在机器里面的状态。

### 6. 用 USB-C 网线转接头接入有线网络

无线虽然方便，但在视频会议、上传大型文件、远程 Xcode 调试等场景下，**稳定、快速的网络才是真刚需**。

![](.evernote-content/51FAE433-EE7E-4621-A7A8-A6E3876EAEF9/C7AFEC0F-56B4-43C5-9D6E-94C94956F198.png)

只要配一个 USB-C 到网线的转接头，立马拥有千兆甚至 2.5G 的网速。我平常在上传 4K 视频或构建大型项目时，都会插上线，效率高很多。

### 7. 扩展坞打造完整桌面工作站

有了 USB-C 扩展坞或 Thunderbolt 集线器，Mac 直接“变身台式机”。例如，我给自己家的扩展坞同时接了：

* 两台外接显示器
* USB 鼠标键盘
* SD 卡读卡器
* 外接硬盘
* 有线网络
* 还能给 Mac 充电

最妙的是 —— **只需一根 USB-C 线**！出门一拔线，回家一插上，所有设备瞬间恢复连接，效率满分。

### 8. 安全密钥或身份认证设备

如果你重视隐私安全（其实大家都应该），可以用 USB-C 接入硬件级别的安全密钥，比如 YubiKey。那么可以简单地实现：

![](.evernote-content/51FAE433-EE7E-4621-A7A8-A6E3876EAEF9/ED7787A3-8305-4976-A7AF-B708FA09911C.jpg)

* 登录 macOS 或 iCloud 时的硬件验证
* 给 Gmail、GitHub、银行账号添加物理二次验证
* 甚至支持无密码登录（基于 FIDO2） 对比短信验证码或 App 验证，这种方式安全性更高，也更省心。

### 9. 与开发板或硬件设备交互

如果你是搞开发的，或者爱折腾电子硬件，那么 USB-C 简直就是你进入“创客世界”的钥匙。

它可以用于：

* 调试 Arduino 或 Raspberry Pi
* 刷写固件到开发板
* 与实验室级别仪器通信

配合 Terminal、Homebrew、VS Code 等工具，你的 Mac 可以轻松变身为强大的“嵌入式开发中心”。

![](.evernote-content/51FAE433-EE7E-4621-A7A8-A6E3876EAEF9/C76D7DCD-884A-48AE-B701-0F5B1DD7A6A5.jpg)

### 10. 数据迁移与数据恢复

像是你刚换了新的 Mac，或者原来的老设备出了点故障、系统打不开了什么的，这时候 USB-C 就不只是个普通接口了，它完全能变成一个非常好用的“数据搬运工”。

一是我之前用 USB-C 线连着新旧两台 Mac 的时候，打开系统自带的迁移助理，像用户账号啊、安装过的软件啊、平时的文件啊，还有各种设置，都能一股脑儿地全搬过来，不用担心会漏掉什么，而且速度比用 Wi-Fi 快太多了，省事儿还省时间

![](.evernote-content/51FAE433-EE7E-4621-A7A8-A6E3876EAEF9/B5EAD386-8A5E-4EC3-8D74-5116BB903EF9.jpg)

二是我遇到过几次老 Mac 忽然坏了开不了机，但硬盘还能用，我就用 USB-C 把它接到另一台 Mac 上，开启目标磁盘模式（Target Disk Mode），这时候旧 Mac 变成了个外接硬盘，我就能把重要资料都一口气拷出来，这事儿救了我好几回，不然数据没了那真是哭都来不及

三是还有就是，有时候你手头只有一根 USB-C 线，这条线不光能传资料，有的还能直接当外接硬盘启动用，甚至能跑另一个系统，拿来测试或者分开工作和生活用系统也特别方便，毕竟不想主系统被搞乱的时候，能用这个也算很靠谱了

总之这个功能虽然不常用，假如真碰上突发情况，那么它就会像应急包一样帮你脱困，谁用谁知道有多香。😭又到了告别的时刻，本期的分享到这了，下期我们再见！！！🚀
