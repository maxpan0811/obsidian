Anki for iOS 大更新
----------------

前天（4 月 23 日），iOS 版的 Anki 又更新了，此次更新内容特别多，作者对用户界面进行了大改，好多同学都不适应，我熟悉了一天之后，挑选了一部分改动较大的地方和大家分享，也让尚未更新的同学云体验一把新版

### 主题更新

此次大改最直观的地方是在用户界面上

* • 原来蓝白色调的主题升级为新的灰底蓝字主题；
* • 取消原先的蓝 / 灰 / 黑三色主题；
* • 新的全局夜间模式；
* • iOS11 + 的系统上，顶部和底部的状态栏改为半透明；

### 功能

新版还在功能和按钮布局上也做了大量改动：

* • 移除了原来的主题选择，只在刷卡界面提供夜间模式
* • 移除了原来刷卡界面的 Tools，改由新的齿轮按钮调出菜单
* • 新版菜单可以通过在卡片界面向左滑动唤出
* • 长按新版 Tools 菜单的按钮还能拖拽排序

![](.evernote-content/9919674D-6C59-466E-A323-C36F4921192C/3285CC7D-287F-404C-AB4A-1CF4D9806C7A.jpg)

### 其它

还有些小细节也更新了，顺便也跟大家一起看看吧。

* • 顶部工具条由原来的 Quickbar 改为了 Top Bar；
* • Top Bar 可以在新 Tools 菜单里快捷关闭，关闭 Top Bar 之后，刷卡时会进入全屏模式，将无法看到 iOS 顶部状态栏；
* • 设置通知提醒

![](.evernote-content/9919674D-6C59-466E-A323-C36F4921192C/755FA44B-9E40-4399-82D7-D0345607965D.jpg)

左旧右新；老版移除了 Tools Buttons 和 Quickbar；分别改成了：Top Bar【顶部工具条】；Tools Overlay Button【卡片界面齿轮开关】；Tools Overlay Position【齿轮放置位置】；

![](.evernote-content/9919674D-6C59-466E-A323-C36F4921192C/6DCC5192-2381-40A7-B3D2-13AF346CC014.jpg)

点击 Top Bar 之后，就能进入全屏模式

![](.evernote-content/9919674D-6C59-466E-A323-C36F4921192C/44A97BFE-2831-48E2-ADA1-4D7EC73858C1.jpg)

定时发送通知提示复习；在【首页右上角齿轮进入设置 - Review-Notifications】里打开或者关闭
