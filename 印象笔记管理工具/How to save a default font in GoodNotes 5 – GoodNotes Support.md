How to save a default font in GoodNotes 5
=========================================

![Avatar](https://secure.gravatar.com/avatar/af594a71a57b53f643fdea6afcd81d60?default=https://assets.zendesk.com/hc/assets/default_avatar.png&r=g)

[Follow](https://support.goodnotes.com/hc/en-us/articles/360000954435-How-to-save-a-default-font-in-GoodNotes-5/subscription "Opens a sign-in dialog")

Saving your favorite font as default is available in version 5.0.32 or later.  Once you click on the Text tool, you will see a new Styling icon on the right side of the toolbar.

*Screenshot 1: GoodNot**es**5* Toolbar

![](.evernote-content/225AFF22-4D9B-4096-990C-0B030F3CB545/F751EAA1-E258-49C8-BA21-89A550D68F6D.jpg)

Two ways to save a font as default 

Option 1: Setting a default font from the start

Click on the Text tool > Styling icon > adjust the font size and style to your liking and click on **Save as Default** before exiting. 

Option 2: Changing the existing default font

Click on the textbox > use Select All and the text will be selected as shown in the picture.  Then click on the Styling icon to change the font size and style to your liking. Click **Save as Default** before exiting.

*Screenshot 2: Styling Settings*

![](.evernote-content/225AFF22-4D9B-4096-990C-0B030F3CB545/5622B6FF-2AB1-4683-B846-3409EA4A519F.png)

![](.evernote-content/225AFF22-4D9B-4096-990C-0B030F3CB545/981DC29C-D1FF-424F-9E07-154428F41194.jpg)

Changing a font temporarily

The changes you made in the font Settings are universal throughout your notebooks once you save it as default. To change the font style temporarily, you can select the whole sentence or a portion of the sentence and make the adjustment without saving it as default.

*Screenshot 3: Styling a portion of the sentence*

![](.evernote-content/225AFF22-4D9B-4096-990C-0B030F3CB545/67FA8776-8EEB-4CA0-99FA-19BD3F5988FC.jpg)

\*Please note: If you want to copy an existing text document and paste it as the original font style, you can use the lasso tool ![](.evernote-content/225AFF22-4D9B-4096-990C-0B030F3CB545/29236A9C-D806-4916-9441-1A59FBCDFF1B.png)to make the copy and paste it in place.
