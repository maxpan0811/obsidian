阅读很重要，但你知道怎么读才重要么？

这两天，大家的朋友圈都被“天才滑雪少女”谷爱凌刷屏，我们看到了她光环的一面，却不知道她是付出99.9%努力，才有了今天的成就。

小编看到一个细节，疫情期间，古爱凌也在持续阅读，她特别提到阅读的书籍给了她很多帮助。

我们都知道，阅读很重要。但怎么读，更重要！

持续、有效、有技巧地阅读，才能充分发挥阅读的力量。

今天，我们和大家透彻解析，如何持续、有效、有技巧地阅读。

**1**

**如何持续？**

**一年四季持续地读**

很多家长反映，小朋友在家里阅读，今天兴致好了，就拿出一本书，一口气读完；明天兴致没有了，就扔在一边不读了。如此反反复复、断断续续，良好的阅读习惯自然无法养成，阅读量也无法保证。

世界上最公平的事情，就是每个人的时间都一样。怎么用好自己的每年、每月、每周，甚至每一天、每一个小时，形成人与人之间最大的差别。

能够持续每天都在阅读的孩子，自然会有不一样的人生。如果做不到每天，能坚持每周都阅读，是为持续阅读迈出的第一步。

**逐渐提升难度持续地读**

英文阅读难度，有两个方向。

一个是语言难度，这一块由英文词汇数量、词汇难度、句式复杂度等决定。

一个是内容深度，这一块需要孩子匹配的心智，和年龄、经验及阅历相关。

读得太难，孩子不愿读；读得太简单，也不愿读。

为孩子持续选择匹配的阅读书籍，让他们在语言上得到提升的同时，又获得共鸣又不失深度的思考探索，这是阅读能否持续进行的另一个关键。

**一本书持续地读**

也有不少家长反映，孩子经常兴致勃勃打开一本书，读到一半，就放下了，然后就一直放下了，再没打开过。

很多英国和美国学校的老师，会用SSR（Sustained Silent Reading持续默读能力），来衡量一个孩子的阅读能力。能否将一本书，持续读完，正考验了一个孩子的SSR持续默读能力。

一个能持续读完一本又一本书的孩子，一定有着极强的专注度和投入度。未来不管在学业上，还是在工作中，必定会有出色的表现。

**2**

**如何有效？**

现在英文阅读的工具有很多，不论是RAZ，还是AR ，每次读完书，都会有几道阅读理解选择题，来测评一下孩子们读得到底怎么样。这样是不是有效阅读了呢？

说到有效阅读，如果有套标准，就再好不过了。说到标准，我们可以来看下CCSS(the Common Core State Standards)。

**什么是CCSS**

CCSS是指美国共同核心州立标准(the Common Core State Standards)。

它美国教育局，联合专家教授和老师，按照年级K-12，规定了每个级别的英语语言和数学教学内容，建立统一的英文和数学教学标准。

CCSS非常强调培养学生的批判性思维(critical-thinking)、解决问题(problem-solving)和分析能力(analytical skills)三大核心素养。

它将英文学习，分成了阅读(Reading)、写作(Writing)、语言(Language)、听说(Speaking and Listening)四部分,通过这四部分对美国幼儿园到12年级的标准进行了详细阐述。

在英美学校，老师经常带学生们，进行分组讨论，引导学生思考，每个人有不同的思考角度，又可以激发不同的思维火花，让阅读更有效。

**3**

**如何有技巧？**

**有技巧地学**

每个侦探都需要一个放大镜，每个神枪手都需要一把枪。

一个会读书的人，一定有自己的“放大镜”和“枪”。

要读，就读好书。

从优秀的读物中，孩子们可以学习作者的表达方式、写作手法等，这些就是一个读者的“放大镜”和“枪”。

越优秀的作者，越会运用好的写作表达方式和方法。孩子们学习到这些技能，就能更轻松地阅读其他书籍，甚至能激发自主探索其他读物的方式方法，掌握更好的自学能力。

**有技巧地用**

从优秀读物中，获得的写作手法，不断思维和强化，还可以运用到自己的表达和写作中。

不断积累，不断提升，不断收获。

正如苏斯博士说的那句话：

The more that you read, the more things you will know. The more that you learn, the more places you'll go

读的越多，懂的越多，学的越多，走的越远。

英文阅读是一个长期积累的过程，而持续、有效、有技巧地积累，将提高孩子们的英文阅读兴趣，建立长久良好的英文阅读习惯和思考表达习惯。
