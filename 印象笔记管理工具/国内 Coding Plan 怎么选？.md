从每月9.9元到200元，国内 Coding Plan 看起来都在卖“AI编程额度”，但实际上很难只按价格直接排名。

因为不同平台的计量口径并不一样：

有的按模型请求次数

有的按 prompts

有的按独立额度倍数

有的把文本、图像、语音等资源放进统一额度池

一次用户提问，也可能触发多次模型调用。所以“18000次请求”，不等于能连续对话18000轮。

截至2026年7月1日，主流方案大致可以这样理解：

火山方舟

Lite和Pro长期价分别为40元、200元，当前前两个月活动价为9.9元、49.9元，第三个月恢复原价。优势是价格门槛低、支持多模型切换，适合低成本尝鲜。

腾讯云 TokenHub

Lite和Pro分别为40元、200元，每月约18000次和90000次请求。额度展示直观，兼容Claude Code、Cursor、Cline、Codex等工具，适合希望长期稳定算账的人。

阿里云百炼

目前主推Pro套餐，200元/月，每月约90000次请求；Lite已经停止新购。优势是千问生态和模型池丰富，但存在限量抢购，购买前要确认库存。

智谱GLM Coding Plan

分Lite、Pro、Max三档，每5小时约80、400、1600个prompts，每周约400、2000、8000个。支持GLM-5.2、GLM-5-Turbo、GLM-4.7，并附带联网搜索、网页读取、开源仓库等MCP额度。高阶模型还存在不同时间段的额度倍率。

MiniMax Token Plan

Plus、Max、Ultra分别为20、50、120美元/月。它不只覆盖Coding，文本、图像、语音、音乐等资源共享统一额度，更适合Coding＋Agent＋多模态工作流。

Kimi会员 / Kimi Code

会员月费从19美元到199美元，Kimi Code额度分别为1×、5×、15×、30×，并与其他会员能力形成组合，适合已经长期使用Kimi Agent和资料工具的人。

可以简单这样选：

低成本尝鲜：先看火山活动价

希望额度透明：优先看腾讯云

想用最新GLM和MCP：看智谱原厂

想做Coding＋Agent＋多模态：看MiniMax

已经重度使用Kimi：直接看Kimi会员

已在千问和阿里生态：看百炼Pro

真正的性价比，不是“月费除以次数”。

而是完成同一个开发任务，要花多少钱、等多久、返工几次。

价格、活动和模型池调整很快，实际购买前仍要以官方下单页为准。
