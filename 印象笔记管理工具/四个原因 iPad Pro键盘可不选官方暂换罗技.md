# 四个原因 iPad Pro键盘可不选官方暂换罗技

---

[![](.evernote-content/41A72350-D4AC-4C94-BC4A-9E3A47A74F14/DE14E1F6-E783-4F2C-BE3A-FC98A50D6B98.gif)](http://www.feng.com/iPhone/news/2015-11-13/Four-reasons-why-the-Pro-don-t-choose-the-official-temporarily-in-logitech-keyboard_630067.shtml)

![](.evernote-content/41A72350-D4AC-4C94-BC4A-9E3A47A74F14/E994E940-ADD3-4359-9C5E-88D1E9FDBECB.jpg)

大多数用户买了 iPad Pro 之后，若有一定的生产力要求的购买配件是比不可少的事情。鉴于目前 iPad Pro 的相关配件还很少，所以不少人还是选择直接购买苹果官方 1288 元人民币的 Smart Keyboard。不过，现在你不得不继续等待，因为 Smart Keyboard 预计发货时间需要等 4 到 5 周时间。

在此期间，其实你可以稍微考虑一下第三方键盘，或许会更符合你的要求，比如罗技率先推出的 CREATE 背光键盘保护壳，它也可以和苹果 Smart Keyboard 一样通过 Smart Connector 磁性连接和供电，无需再进行蓝牙配对，也无需充电。现在有那么四个原因，它比 Smart Keyboard 更加值得购买一些。

![](.evernote-content/41A72350-D4AC-4C94-BC4A-9E3A47A74F14/D8CD89C4-D6FB-4C44-BBB5-197479E4C2B6.jpg)

1、第一个原因自然是因为罗技的 CREATE 背光键盘保护壳更快上市。这里并不是给罗技打什么广告，毕竟在那么多苹果配件商里面算比较用心了，甚至比苹果还用心。iPad Pro 9 月份就已经面向全球公布了，很多人希望可以将其作为一款代替笔记本电脑的产品，但两个月时间过去了，苹果并没有做到充分备货。

现在大家在苹果官方商店看看，不仅 iPad Pro 部分机型要等至　12 月初才能发货，而且 Apple Pencil 和 Smart Keyboard 的发货时间也需要 4 至 5 周时间。而在同样是苹果官方商店，CREATE 背光键盘保护壳显示有现货，部分的市区在 20 号左右就能拿到货，而且罗技官方也会提供销售，不必等太长时间。

![](.evernote-content/41A72350-D4AC-4C94-BC4A-9E3A47A74F14/DE14E1F6-E783-4F2C-BE3A-FC98A50D6B98.gif)

2、背光设计。相信很多果粉都是背光键盘的忠实粉丝，苹果之前在发布 MacBook 时曾表示“着手打造一款力求纤薄的键盘时，我们首要考虑的问题之一是：怎样设计按键背光，一切精心设计都是为了尽力实现高能效”，说明苹果还是十分注重用户夜间使用时“清晰、舒适和分别”的观感，但是这款“专为 iPad Pro 量身设计 ”并不具备。背光这种东西早就成为了大多数键盘吸引用户的功能之一，不清楚苹果是否担心背光更快耗电的问题。

3、多种配色可选。目前国内仅提供黑色、蓝色和经典红三种配色，不过实际上 CREATE 背光键盘有五种配色，还有太空灰加黑、蓝银这两种配色款。而苹果官方的 Smart Keyboard 只有炭灰色一种配色。

![](.evernote-content/41A72350-D4AC-4C94-BC4A-9E3A47A74F14/0AF6040B-0568-4942-AB47-3C0CEBA4620C.jpg)

4、最后一点是价格优势。罗技的 CREATE 背光键盘建议零售价格仅 998 元人民币，相比之下，苹果 Smart Keyboard 的售价高达 1299 元人民币。平心而论，两款键盘都摸过的人少之又少，手感如何不做评价，但罗技这款键盘至少在功能上更胜一筹，主屏幕、搜索 iPad 和网络、调节亮度、控制音频或视频播放等多种快捷操作一个不少，也提供打字和观看两种模式，以及自动唤醒和自动休眠。

鉴于 iPad Pro 刚刚正式上市，很多第三方配件可能还要一段时间才能跟上，不过目前在苹果官方商店也就两款键盘配件可选，如果在乎以更便宜的价格更快拿货，并中意背光功能的话目前罗技确实是更好的选择。
