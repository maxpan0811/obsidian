| |  | | --- | | *本帖最后由 wakasone 于 2016-6-16 00:39 编辑*      "**我正在参加 [家长帮"2016·与梦同行"小升初有奖征文活动](http://jzb.com/bbs/thread-4554929-1-1.html)，快来一起聊聊吧！"**    ***本周初中分班考告一段落，今年的XSC工作终于圆满结束了。关注家长帮也有一年多， 从大家分享的信息中获益匪浅，特别是小范老师精华帖里整理的各档名校清单非常精确。也非常赞同很多家长提到的孩子定位准确+择校思路清楚，就已经胜利了一半的观点。其实可以把十二年中小学教育看做一场马拉松，小升初只是第一个十公里，择校成功意味着可以进入第一军团，有机会在高考冲刺时挑战金牌。这里整理一下简单的经历，供大家参考和批评。***    **1.****学区房。06年公司派驻回国来上海工作时对这个大城市还一无所知，幸好有父亲的一位老同事指点在浦东内环内买了三居新房。儿子是05年底出生的，两岁来上海开始上民办幼儿园的托班，在这个幼儿园一读四年后上了对口浦东名校XX外国语小学。记得2011年拿着招办的录取通知书去小学报名时，还被招生老师以孩子未满六周岁为由拒之门外，电话投诉到招办才顺利入学。虽然由于入学后在班级里年龄最小，但是有幸结识了一群优秀的同学和鸡血的家长，今年小升初班里三分之二的学生择校成功，十多名同学考上A档、B档名校初中。**    **2.****奥数。学校三门主课里面数学是班级TOP3水平、语文是中等偏上、英语是后十名，典型的小青蛙一个。学习习惯还比较好，在学校可以基本把作业都做完，回家后做半小时课外功课就去玩了。三年级和班里的好基友结伴在学而思提高班学习奥数，同时也参加学校沙龙班学习思维课程到四年级末。儿子对奥数还比较喜欢，但是反感参加杯赛，我们本着快乐教育的原则，也不勉强他了。**    **3.****择校定位。四年级上学期开始考虑小升初的问题，保底的对口公办JCB属于C档，中考成绩不错，离家走路一分钟。小区邻居孩子有读SZD、XZY、JZ、HY等名校，但是报考民办包装简历需要的各种证书我们一个也没有。儿子从大班开始学习国际象棋四年级已经是三级棋士，偶然得知国际象棋一级棋士可以报考PW。百度一下了解到PW是全市招生B档公办校（TOP20），网上报名不需要简历。于是和儿子确认过他也愿意住宿，就把PW作为目标学校。另外有朋友小孩在SS读书，了解到SS是全市招生的A档公办校，学制六年也是网上报名不需要简历，上下学乘地铁六站路也还方便，于是把SS作为挑战学校。**    **4.****准备。四年级寒假期间强化国际象棋的训练，每天在chess.com上下四盘棋；四年级下学期春节后三月四月两次升级赛顺利突破，拿到一级棋士证书获得PW的敲门砖（娃没有校优）。之后加入PW家长QQ群结识了K老师， 五年级上学期开始每周一次去K老师家一对一辅导两个小时针对PW机考（英语听说和英语奥数）以及分班考的奥数/英语等内容。**    **5.****QRK****。今年三月末突然接到学校班主任老师的电话，说有一个SS的推优考机会，这时感觉就是天上掉馅饼了。4月1日这次推优考是72人录取30人，可惜遗珠了，清醒地明白了娃依然是小青蛙一枚。四月初老老实实地网报了PW和SS，两校都顺利通过了初审。4月23日参加SS的千人考不出所料再次落第，4月24日参加PW机考被第一批录取五一前顺利签约。**    **【总结】**   **2016年这场没有硝烟的战争，绝大多数家长目标是一流或者二流民办学校，希望在中考的战场上获得优势。通常的套路是至少要在三年级开始进XES尖子班，四五年级拿到四大杯赛的三等奖以上；四年级暑假开始上各种XW班和机构（获得推优机会）；五年级开始广发豪华简历争取面试机会，然后再参加笔试、体育测试、面试获得预录取；孩子考上民办后还要在附近租房，家长四年每天上下班要两个小时，初三为了中考全家还要辛苦一年多…**   **我们夫妻俩工作都比较忙，老人也不在上海，实在没有精力做以上这些事情。而且男孩在班级里比同学要小一岁，目前不可能拼得过早熟的女孩（班级里录取120、LS、HY、SW、ZJ、PH等A档名校的都是女孩…）。个人认为如果把长期目标定位在北清复交，90%取决于孩子的智商和学习习惯，而且男孩在高中后会快速发力，高二成绩就和女孩不相上下了。**   **于是选择了低调的半仙中学PW（娃住校后我俩也轻松了），同学大多是优秀少先队员上来的，学习风气会比较好，又可以锻炼娃的生活自理能力。争取初三努力可以进入前50%直升本校高中躲过中考，充分积聚力量高三全力挑战北清复交!** | |
