# 会自动更新的地球卫星照片桌面壁纸来了

会自动更新的地球卫星照片桌面壁纸来了
------------------

原创 *2016-04-03* *青小蛙*

DesktopEarth 是一款使用「实时地球卫星照片」作为桌面壁纸的应用，其照片基于 NASA 卫星照片及城市灯光照片，另外，支持多显示器，还有火星视角。@Appinn

*Windows / OS X / Android*

![](.evernote-content/527BE8E3-71E5-486A-BD58-C0C068DDADE8/9C9CCC27-E88B-43DB-8CC0-340B14276903.jpg)

之前青小蛙有介绍过使用「向日葵8号」卫星的 馒头桌面，不过其只有普通的卫星照片。地球上最牛逼的神秘组织 NASA 有更加令人心动的城市灯光照片。

以 Windows 版本的 **DesktopEarth** 设置页面为例，可以让地球以各种角度在你的桌面呈现：

![](.evernote-content/527BE8E3-71E5-486A-BD58-C0C068DDADE8/15BE09A4-BFFF-4971-BC2E-ED31B97BC0BB.jpg)

地球局部区域的照片要比整个地球更加震撼：

![](.evernote-content/527BE8E3-71E5-486A-BD58-C0C068DDADE8/8AFBE539-4CFE-4EE1-A159-AE0BDF7CF356.jpg)

Mac 上面还有基于 DesktopEarth 的 3D 屏保功能。

火星视角倒是蛮有趣的，不知道《火星救援》里的视角是不是这样呢？

![](.evernote-content/527BE8E3-71E5-486A-BD58-C0C068DDADE8/20D07FDF-259C-41E6-BEC5-0FB6BDC8625B.jpg)

![](.evernote-content/527BE8E3-71E5-486A-BD58-C0C068DDADE8/D001FAEC-BB1F-4934-A850-E6C19950D6B4.png)

对了，Mac 和 Android 下都属于付费应用。

![](.evernote-content/527BE8E3-71E5-486A-BD58-C0C068DDADE8/13040489-217B-4B32-841F-4C0A1E812AE7.jpg)
