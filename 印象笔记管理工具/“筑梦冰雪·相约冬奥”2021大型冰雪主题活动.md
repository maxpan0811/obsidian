“筑梦冰雪·相约冬奥”2021大型冰雪主题活动
=======================

北京天骄快乐家族 北京天骄快乐家族幼儿运动课程

![](.evernote-content/D4D42B19-014D-4AAD-A095-4CC094645386/73F5A52A-2F50-43C9-8AE3-72BE12340F94.gif)

**筑梦冰雪·相约冬奥**

—— 2021大型冰雪主题活动 ——

**活动背景**

第24届北京冬季奥运会将在2022年2月4日至2022年2月20日在中华人民共和国北京市和河北省张家口市联合举行。这是中国历史上第一次举办冬季奥运会，北京、张家口同为主办城市。这是我们的荣耀，我们为之呐喊，为之自豪！为迎接2022冬奥会，培养幼儿体育兴趣，加深幼儿爱国情感，锻炼幼儿身体，增强幼儿体质。特别策划了本次“筑梦冰雪·相约冬奥”主题运动会活动。

![](.evernote-content/D4D42B19-014D-4AAD-A095-4CC094645386/C4E2B47E-03B6-420C-B869-8DB43E6AD189.jpg)

**活动目标**

“筑梦冰雪·相约冬奥”主题运动会是营造一个冰雪冬奥会的氛围，让参与者在仪式感和趣味运动中感受运动的快乐，了解冬奥会的小知识及运动规则，让幼儿喜欢运动，并通过各项运动的特点促进小朋友体能和体格的发展，有一个健康的体魄，健康成长。

**活动流程**

|  |  |
| --- | --- |
| 1 | 主持人开场 |
| 2 | 国旗班、各班级入场落位 |
| 3 | 火炬传递（篮球场4角，每个角安排1名教师，依次进行火炬传递到达主席台后交给园长，园长高举火炬示意5秒后转身将火炬递下，接过话筒讲话） |

|  |  |
| --- | --- |
| 4 | 园长致辞 |
| 5 | 运动员代表宣誓 |
| 6 | 幼儿律动操及与游戏互动（天骄主持人负责） |

|  |  |
| --- | --- |
| 7 | 项目竞赛（轮转式组织，每个项目一个天骄教练） |

![](.evernote-content/D4D42B19-014D-4AAD-A095-4CC094645386/A5E510FC-6076-46D8-AF11-3B7A49CB2E54.jpg)

**活动准备**

|  |  |
| --- | --- |
| 1 | 假设活动9点整开始 幼儿园提前1.5——2个小时到达 |
| 2 | 舞台、及背景布 |
| 3 | 各班级：负责签到班级人员、组织家长小朋友到各自班级的候场区排队 |

|  |  |
| --- | --- |
| 4 | 活动开始之前半小时、各班级提前通知家长带着幼儿上厕所。 |
| 5 | 主班配班及保育员在活动中随机应变。 |
| 6 | 活动结束：主持人宣布结束、合影留念 |

![](.evernote-content/D4D42B19-014D-4AAD-A095-4CC094645386/A12722C3-7FD6-4645-A126-A988E952B462.jpg)

**活动保障**

|  |  |
| --- | --- |
| 1 | **教练系统：**  师生配比1:30；主训教练均经严格选拔及培训，全场把控，反复演练；应急团队：专人负责，随时在线。 |
| 2 | **组织管理系统：**  严格按照方案流程组织活动,每小队选举责任心强的人担任队长、副队长以及安全员，明确责任义务以达到有效管理； |
| 3 | **装备品质系统：**  活动配备的所有装备，都是经过国际品质体系认证过的名牌产品，以保证活动安全万无一失； |

|  |  |
| --- | --- |
| 4 | **后勤保障系统：**  为每位学员购买人身意外及医疗保险，备有急救箱应付突发事件，并配备专业后勤教练精心服务。 |

**温馨提示**

1.本活动经过实地详尽考察，独家策划活动环节，并对活动项目作出安全评估；

2.建议给孩子穿宽松的秋季衣物、运动鞋；

3.随身物品：背包/单肩包、水杯、个人专用药品（如有医疗史，报名时请提前告知）；

4.请勿携带大量现金、贵重物品、手机等电子产品；

5.活动中请听从老师的安排，严禁私自离队，参观时请爱护公物，未经允许不得随意触碰。
