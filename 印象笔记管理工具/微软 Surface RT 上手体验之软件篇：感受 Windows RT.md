# 微软 Surface RT 上手体验之软件篇：感受 Windows RT

---

[![](.evernote-content/10E574DB-1E33-425A-BA40-78D873E9B875/3C97E1A4-CBCC-44AC-B502-3070665CC402.jpg)](http://www.ifanr.com/178654/img_5701-2)

今天爱范儿在上海全程直播了微软 Windows 8 中国发布会。在发布会结束之后的 Demo 环节中，我们对 Surface RT 进行了充分的体验。这次中国发布会也是今年 6 月 19 日 Surface 首次亮相之后的第一次正式展示。

发布会上，微软全球 Windows 与 Windows Live 事业部总裁 Steven Sinosky 强调，笔记本电脑和平板电脑同样都是 PC，除了操作方式不一样，微软要为他们提供同样的体验。对于微软自家推出的平板电脑 Surface 系列来说，统一 Surface RT 和 Surface Pro 上的体验，就需要在完全不同的硬件上，统一 Windows 8 与 Windows RT 的操作感受。

Windows 是当之无愧的生产力工具，这个世界很大一部分都依靠它运转。在 Windows 8 开放预览版下载之后，很多用户对它表达了“失望”、“失败”这样的感受。在两个月的 Windows 8 RTM 使用之后，我认为这样的描述甚为偏颇。

[![](.evernote-content/10E574DB-1E33-425A-BA40-78D873E9B875/0CD45C39-5EC2-40A6-BCB0-CF037552C6D0.jpg)](http://www.ifanr.com/178654/img_5627)

全新的 Microsoft design style 开始界面是用户们吐槽的重点，大部分都集中在“割裂”、“学习成本高”、“无意义”这些点上。但不能否认的是 ，全新的开始菜单确实非常方便触控操作。任何新概念都需要学习成本，新开始菜单的核心概念是从边缘滑动，这并不复杂，相反我认为很人性，很直观。Steven Sinosky 今天在发布会上说，微软希望平板与传统 PC 不再分裂。发布会上，展示了众多厂商即将推出的 “变形本”、“组合本”、触控一体机等专为 Windows 8 设计、优化的新设备。配合这些新概念 PC 能够让同一台 Windows 8 计算机同时满足工作和娱乐。举例来说，“变形本” 360 度打开折叠，就变成一台平板；“组合本”安装键盘底座即从平板变身笔电。今天在现场的体验证明，某些品牌新设备优秀的做工、轻薄的程度都足以胜任这样的定位。

Windows 8 的传统桌面似乎没有过多引来大家的批评，总体体验和过去的 Windows 没有很大的不同是重要的原因。但是，众多的细节改进让使用更加方便。除去 Aero 界面这样见仁见智的视觉效果调整，文件管理器加入 Ribbon 菜单，任务管理器大幅增强等都无疑让 Windows 8 作为生产力工具更为称职。当然，崩溃、漏洞等等出现在预览版本上的问题在正式版中当然不会成为困扰。和过去版本相比，开、关机难以置信的快速更是让人心情舒畅。

所有这些，都完整的继承到了 Windows RT。

![](.evernote-content/10E574DB-1E33-425A-BA40-78D873E9B875/23CDAE41-7327-4D45-8D33-D6E1949257DC.jpg)

Windows RT 运行在基于 ARM 架构的 NVIDIA Tegra 3 芯片，内核与传统 PC 的 x86 架构完全不同。微软的技术人员对于如何实现这样的跨平台移植讳莫如深，但是乍玩起来，没有感觉到非常大的不同。和 Windows 8 一样，Windows RT 同样支持键盘与鼠标，同样提供 Microsoft design style 开始菜单和标准的 Windows 桌面。得益于 Microsoft design style UI，操作便捷、流畅。尝试从 Windows 应用商店搜索、下载、安装软件的过程迅速、直观。

随着正式发售日越来越近，应用商店中的软件数量快速增长，一些在预览版本中提供的软件开始大规模的更新，优秀软件也是可圈可点。桌面模式下，预装了 Office 2013  家庭和学生版的预览版本，包括 Word、Excel、PowerPoint 与 OneNote 四个组件。软件的操作和功能与 Windows 8 平台上的 Office 软件并无不同。在 26 日之后，它们会自动通过 Windows Update 自动更新到正式版本。

Windows 应用商店在 Windows 8 及 Windows RT 中均仅提供 Microsoft design style 应用下载，但是 Windows RT 只能通过应用商店安装程序。今天我们得到消息，微软倾向于永远不允许第三方开发 Windows RT 桌面应用。也就是说，很有可能未来 Surface RT 与其他 Windows RT 设备的桌面环境永远只会有预装的 Office 组件存在。Windows RT 无法安装所有 x86 平台的软件，按照目前微软的态度，未来即使 ARM 芯片的性能能够满足需要，诸如 Photoshop 这样的生产力工具将不会在出现 WIndows RT 的桌面上，它们只可能以 Microsoft design style 应用的形式通过应用商店部署。

[![](.evernote-content/10E574DB-1E33-425A-BA40-78D873E9B875/AD0895C4-2B96-45C7-8FB6-9B1E8F9EBCAA.jpg)](http://www.ifanr.com/178654/img_5695)

Surface 还有基于 x86 平台的 Pro 版本。真机目前还没有公开亮相，但是通过配置，我们可以推断，Intel i5 Ivy Brige 处理器及 4 GB 内存将会为我们提供与传统 PC 无差别的完整体验。但是对于一台平板电脑，散热口和风扇的设计确实有一些“先锋”。以我的经验判断，热量控制会是个不小的难题。而基于 Windows 平台的字体渲染方式， 10.6 英寸的全高清屏幕上的阅读体验也无法乐观。当然，也许 90 天之后上市的 Surface Pro 会解决这些问题。在这之前，我对它持谨慎态度。

从体验来看，基于 Windows RT 的 Surface RT 能够满足移动办公和娱乐的需要，虽然我们无法测试，但是标称的 8 小时电池电量也能够支持外出使用，同时 3688 元人民币的官方售价也让我对 Surface RT 的市场前景保持乐观。
