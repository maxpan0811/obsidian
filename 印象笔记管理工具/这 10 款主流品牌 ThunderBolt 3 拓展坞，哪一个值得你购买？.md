# 这 10 款主流品牌 ThunderBolt 3 拓展坞，哪一个值得你购买？

---

貌似因为之前吐槽 [为 XPS 购买 Thunderbolt 3 拓展坞的文章](https://sspai.com/post/40273) 还过得去，收到 Hum 的邀请为大家推荐合适的 TB3 拓展坞，于是有了本文的出现。作为如今非常流行的 USB Type-C/TB3 接口有多么好这里就不再赘述，考虑到 Type-C 接口的 Hub 已经相当普及，所以这一次依旧是聚焦在 TB3 拓展坞身上，看看目前市场上都有哪些产品可以让我们选择。

原厂 TB3 配件
---------

笔者曾经因为 Windows 这个系统踩过不少坑，这里先提醒大家要注意根据自己的操作系统做出选择，原厂的配件首先要纳入眼中。毕竟各大 PC 厂商没有苹果那么强的号召力，为了满足消费者的需求，当然会身先士卒为自家产品推出相应的配件。

![](.evernote-content/61D0AB64-7B53-49A8-B6A3-016E80042469/77F564E1-3AA8-4335-A0A6-FB3AB7122F4A.png)

在已发售的原厂配件中，联想、惠普以及戴尔都为自家产品推出了 TB3 拓展坞，售价在 250——300 美元区间，折合人民币约 1700——2000 元。从表格中可以看到戴尔比较坑，在提供的 USB 接口中有两个是 2.0，联想和惠普则全都是 3.0 的。视频输出上三者都提供了 3 个以上的接口，惠普缺少了 HDMI 而已，理论上都能实现双屏 4K 或三屏 FHD，但惠普并没有在官网标明。

![](.evernote-content/61D0AB64-7B53-49A8-B6A3-016E80042469/47B1CE36-9DC9-4AF3-B710-914370080ADA.jpg)

联想是在近期才推出 ThinkPad Thunderbolt Dock 一款产品，从表格中能看到接口相对戴尔和惠普来说算是最优秀的一个，并且售价也还算合理。同时在官网的介绍中得知还有企业级的相关支持，例如 PXE boot 和 Wake-On LAN 等功能。要说缺点的话就是该款产品暂时未在国内发售（国内官网只有 Type-C 版本），并且没有列明支持的设备清单（类似 ThinkPad X1 Carbon 新产品应该无压力），还有能否为连接的设备供电这点官方也没有说明。

![](.evernote-content/61D0AB64-7B53-49A8-B6A3-016E80042469/94E15135-21BE-4741-B5B8-9A7AB64C4FEF.jpg)

惠普的产品线相对丰富一些，一共有三款设备，分别是 Elite 90W Thunderbolt 3 Dock、ZBook 150W Thunderbolt 3 Dock 和 ZBook 200W Thunderbolt 3 Dock。接口方面三者是一致的（前面表格中使用的是 ZBook 200W Thunderbolt 3 Dock 作为对比），重点在于电源适配器的功率不同，相关的设备支持官网有说明，并且都能为支持的设备直接供电，所以购买时参考自己手上的机器即可。不过和联想一样，惠普也没有在国内推出这三款产品。

![](.evernote-content/61D0AB64-7B53-49A8-B6A3-016E80042469/3223CCF5-2C7F-45A9-A38E-06E95F7D579D.jpg)

戴尔的 TB16 已经算是第二代 TB3 的产品，首代的 TB15 早已回收成了洋垃圾，吸取经验后 TB16 起码是可用了（关于 TB15 的坑，在笔者上一篇文章中有详细说明）。接口方面相比联想和惠普没什么优势，但该有的都有，优点就是戴尔将 TB16 引进了国内。不过和海外不同，行货只提供 240W 一个版本（其实拓展坞是一样的，就是附属的电源适配器不同），对于 XPS、Precision 的用户来说可以提供最大 130W 的供电，能将原装电源适配器解放是个独有优势，但行货售价高达 2999 元也是没谁了。

除联想外，惠普和戴尔都将支持的设备在官网中列明了，有需求的可以自行查阅看看自己的设备能否使用。鉴于笔者此前踩过的坑，这里提醒一下要想正常使用这类拓展坞的话一般都需要安装对应的驱动，请认真参考说明书按照指示操作，虽然可能很烦，但为了以后的幸福，还是辛苦一下吧。

Only For Mac
------------

提及了原厂配件，肯定有第三方配件厂的相关产品，由于苹果的光环，为只有 TB3 接口的新 MBP 开发配件的也不在少数，不少还只针对 Mac 开发了产品，这里就列举一下给 Mac 用户参考，Windows 用户可以直接跳过到下一部分。

![](.evernote-content/61D0AB64-7B53-49A8-B6A3-016E80042469/D8A17BEE-350A-4171-8A52-7ADB8AA3FADE.png)

表格中的三款产品都是 For Mac 的，Windows 用户就不要奢望了，就算你对它们的造型设计如何流口水都改变不了你无法使用的结果，解决方法就是买一台 Mac（笑）。不同于上面原厂配件追求大而全来应付各种使用场景，第三方配件厂常常只是针对常见的使用场景而已，所以在接口上相对会少一点。不过在性能上不需要担心，三者表示通过组合接口的用法能够完美应对双屏 4K 或者单屏 5K。

![](.evernote-content/61D0AB64-7B53-49A8-B6A3-016E80042469/F660D604-C254-4E04-B34F-7C8B536E0386.jpg)

对于便携性要求高的用户，HyperDrive 无疑是最合适的选择。不需要像其它竞品一样在外接电源适配器，插上即能使用，接口方面也能满足大部分的场景，特别是 HDMI 接口明确表示能够支持 4K@30Hz 的视屏输出，算是它的一个优势吧。双读卡器的设置也是相当周到，即使是非众筹价的 99 刀售价在 TB3 产品中也算性价比突出了。

![](.evernote-content/61D0AB64-7B53-49A8-B6A3-016E80042469/F6406214-9148-475D-9372-C9D0FFB4C771.jpg)

而 OWC 推出的 Thunderbolt 3 Dock 在接口上相当周全：5 个 USB 3.0 能与联想、戴尔的产品匹敌，S/PDIF 和火线 800 两个接口对于部分特殊用途的消费者来说可能是必须的，虽然视频输出因为体积选择了 miniDP，但对于从上一代 Mac 升级的用户来说旧线材或许还能派上用场，299 刀的售价不算便宜，只是你一对比 Belkin 真觉得良心。

![](.evernote-content/61D0AB64-7B53-49A8-B6A3-016E80042469/CCDC0C56-F7A5-4C8D-BC5C-04A203CAEA4B.jpg)

在介绍完 HyperDrive 和 OWC 之后，笔者个人觉得卖 350 刀的 Belkin 毫无优势，接口不如 OWC 丰富，便携性不如 HyperDrive，价格还高，真的想不到为什么要选择它。要说优点的话就是可以为 Mac 供电，而且 Belkin 作为「御用」苹果配件厂，兼容性应该是一流的，然而还是改变不了曾经「欺诈」过笔者的事实。

![](.evernote-content/61D0AB64-7B53-49A8-B6A3-016E80042469/AAFEFEEE-C3DA-4175-851B-FF10800743EE.jpg)

其实 For Mac 的还有一款 Kensington 推出的 SD5000T，功能方面与 Belkin 没啥区别，价格也一样。不推荐的理由其实很简单，就是和上面的产品对比，外观实在是「不堪入目」，当然有兴趣的可以直接搜索就能找到了。

Windows/Mac 兼容
--------------

在笔者上一篇文章中已经介绍过 CalDigit 和 elgato 两家厂商，没想到谷歌后来又给推送了广告介绍 Plugable 和 StarTech 的拓展坞，并且都是兼容 Windows 和 Mac 的产品，似乎再一次证明了卖信息给谷歌好处还是有的。

![](.evernote-content/61D0AB64-7B53-49A8-B6A3-016E80042469/DC8CC80C-CB24-4A95-A648-BDF45F4BF291.png)

相信从表格中大家也都看出来了，兼容 Windows/Mac 的拓展坞在接口上几乎都是一致的，USB 3.0+DP 是标准的配备，那么在从中挑选的条件自然是其独有的特性以及很重要的价格了。

![](.evernote-content/61D0AB64-7B53-49A8-B6A3-016E80042469/977F0E43-0A3C-4D2D-B62E-67F4DCE65B58.jpg)

说到价格，不得不提及的就是 Plugable 家的 TBT3-UD1-83，189 刀的价格是本文中除 HyperDrive 外最便宜的设备了。兼容性方面官网也是很上心，列出了所有能支持的设备以及无法使用的产品。要说遗憾的就是 Plugable 该款产品不支持为笔记本供电，好消息是 Plugable 将在迟些推出升级款的 TBT3-UDV，不仅支持最高 60W 的供电，USB 3.0 接口还会增加到 5 个。

![](.evernote-content/61D0AB64-7B53-49A8-B6A3-016E80042469/BFFA7143-4B2F-436C-B370-0A0819F4BB1F.jpg)

接着就是里面对 Windows 最良心的 StarTech，虽然接口都一致，但它一共推出了三个版本，分别是 Windows Only（其它家只有 Mac Only，就它家对巨硬情有独钟）、Mac and Windows 和 Mac and Windows - 85W Power Delivery（表格中为该款）。售价依次为 245.99、263.99 和 404.99 美元（美亚的价格似乎会低一些），差别就是对于系统的支持和最高端的版本提供 85W 的供电，大家可以根据自己实际的需求按需选购。

![](.evernote-content/61D0AB64-7B53-49A8-B6A3-016E80042469/9A0FFF9D-9CCE-469D-BBA2-985D93D3F30B.png)

剩下的两款中，CalDigit 的 TS3 特别提供了两个 eSATA 的接口，这是全文里唯一一款提供该接口的产品，对于部分有工作需求的消费者来说，这也许是最好的选择了，并且 TS3 直接提供 85W 的供电，只要不是像 XPS 15 这样的产品供电完全无压力。嫌贵的话，CalDigit 也提供弱化版的 TS3 Lite，价格降到 199.99 美元，但是供电以及 eSATA 接口都取消了，和 Plugable 的基本没有差别。

![](.evernote-content/61D0AB64-7B53-49A8-B6A3-016E80042469/DCEAF9DF-5C68-4561-B6F0-0F397376E44B.jpg)

最后 elgato 的 Thunderbolt 3 Dock 实话说在功能上与其它家相比没什么优势，价钱与 CalDigit 的 TS3 一档但缺少 eSATA 这个卖点，供电也只说明 For Mac 而已，入手的理由个人觉得就只有「好看」。毕竟它也是获得过红点奖的产品，和其它产品相比外型设计的确显得几分优雅。

尾巴
--

上面就是目前笔者能找到的所有 TB3 拓展坞产品了，大家若知道其它产品的话欢迎留言讨论。在结尾给一下个人的意见，选购 TB3 拓展坞的话，如果是 Mac 用户相对会比较轻松，大部分的配件厂应该都会做好兼容。但对于 Windows 用户，笔者推荐首选原厂的配件，虽然价格不便宜，设置也颇为麻烦，可兼容性起码是有保证的；实在没有原厂配件可选，购买时请确保能够退货，避免因为兼容性问题无法使用，TB3 拓展坞最低的差不多也要 200 刀，这配件不便宜。

![](.evernote-content/61D0AB64-7B53-49A8-B6A3-016E80042469/0E909FAF-EE73-4A3E-858E-7A7B5C338B63.jpg)

再来个 One More Thing，若不是追求 TB3 强大的拓展性，只是为笔记本找一个好看好用的 Hub 的话，超级配件大厂田牌公司曾经为 Lumia 950/950XL 推出过的 Microsoft Display Dock 真的很帅很帅的（笑）。
