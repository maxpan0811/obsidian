# 千呼万唤始出来 苹果终于发布OS X 10.8.3

---

[![](.evernote-content/6A3391F7-1932-4239-AE2F-B628E72386C1/829A1458-9ED4-48BF-A1CD-7FAB991EC44C.jpg)](http://www.weiphone.com/apple/osx/2013-03-15/The_long-awaited__548487.shtml)

在历经了 3 个多月的测试，共发布了 13 个测试版之后，苹果今天终于发布了正式版的 OS X 10.8.3，添加了许多新功能，并且改善了许多内容。其中一个最值得注意的新功能就是支持使用 Mac 内建的摄像头在 Mac App Store 中兑换 iTunes 礼品卡。该功能首次出现在去年 11 月发布的 iTunes 11中。　　其它的改进包括 Boot Camp 可支持 Windows 8 的安装、Boot Camp 可支持具有 3TB 硬盘驱动器的Mac等。修复的问题包括可能导致文件 URL 意外结束应用程序的问题、可能导致 Logic Pro 在使用某些插件时变得无响应的问题，可能导致音频在 2011 年推出的 iMac 上播放时断断续续的问题等等。　　此外，本次更新包括了 Safari 6.0.3，提高了性能、稳定性和安全性。苹果建议所有 OS X Mountain Lion 用户都安装此更新。下面是 OS X 10.8.3 的直接下载地址，大家还可以通过 Mac App Store 进行升级：　　- OS X Mountain Lion Update v10.8.3 (540.46 MB)  　　- OS X Mountain Lion Update v10.8.3 (Combo)(793.69 MB) 　　此外，苹果还为 OS X Lion 和雪豹用户发布了安全升级补丁，大家可以通过系统更新或下面的地址升级。　　- Security Update 2013-001 (Lion) (31.42 MB) 　　- Security Update 2013-001 Server (Lion) (79.33 MB) 　　- Security Update 2013-001 (雪豹) (316.63 MB) 　　- Security Update 2013-001 Server (雪豹) (391.63 MB)
