# 新一代MacBook Pro：M4系列能否颠覆你的使用体验？

最近，苹果推出了全新的 MacBook Pro M4、M4 Pro 和 M4 Max，这让我感到无比兴奋。![](.evernote-content/E6DA9996-D73D-46CE-BB7A-06963DED2CB3/54352237-F57B-42D4-938A-02F364208768)作为一名长期使用 MacBook Pro 的用户，这个升级绝对值得期待。

升级的诱惑
-----

之前有传闻说 M4 Max 会非常出色，结果果然没有让我失望。![](.evernote-content/E6DA9996-D73D-46CE-BB7A-06963DED2CB3/430688EC-1F46-44F8-90E3-EDE369975008)新款 MacBook Pro 一共有五个版本：14 英寸和 16 英寸。14 英寸的版本有 M4、M4 Pro 或 M4 Max 芯片，而 16 英寸则只能选择 M4 Pro 或 M4 Max。

M4 Max 芯片的强大
------------

说到 M4 Max，它的性能真的是非常强大，基本上可以看作是两个 M4 Pro 芯片的结合。![](.evernote-content/E6DA9996-D73D-46CE-BB7A-06963DED2CB3/FF895A77-1256-40A1-90C6-2275860BCCBF)苹果每次升级都把性能翻倍，M4 Pro 的性能是 M4 的两倍，而 M4 Max 又是 M4 Pro 的两倍。这种提升简直让人惊叹。

mac版本最强的单核CPU
-------------

M4、M4 Pro 和 M4 Max 芯片都是mac版本上最强的单核 CPU。![](.evernote-content/E6DA9996-D73D-46CE-BB7A-06963DED2CB3/C0D051D7-41EF-4971-9C87-00AF6478862B)发布会上，苹果也公布了这一点。

Thunderbolt 5 的强大
-----------------

M4 Pro 和 M4 Max 芯片都支持 Thunderbolt 5，数据传输速度高达每秒 120GB。虽然我并不需要这么快的速度，但对一些视听专业人士来说，这将是一次重大的改变。

仍在使用 Wi-Fi 6E
-------------

不过，要注意的是，M4 系列仍然只支持 Wi-Fi 6E，而不是 Wi-Fi 7。![](.evernote-content/E6DA9996-D73D-46CE-BB7A-06963DED2CB3/EBA3ACC8-909D-476A-8A22-DD087A7B52AC)这个可能要等到 M5 芯片才会更新。有趣的是，iPhone 16 Pro 已经支持 Wi-Fi 7，但其他设备依然停留在 Wi-Fi 6。

显示能力提升
------

M4 Max 允许连接四个外部显示器，三台通过 Thunderbolt 连接，可以达到 6K@60Hz，第四台通过 HDMI 连接，可以达到 4K@144Hz。对于需要多屏幕工作的人来说，这真是太惊人了。

电池续航能力
------

在电池续航方面，14 英寸和 16 英寸的 MacBook Pro 续航时间是一样的，具体取决于芯片。![](.evernote-content/E6DA9996-D73D-46CE-BB7A-06963DED2CB3/31059076-82DF-4B94-AA25-371DE58B2141)M4 Max 提供 18 小时的视频播放时间，而 M4 Pro 是 22 小时，标准的 M4 芯片则是 24 小时。完全可以满足我一天的使用需求，随便观看电影系列都没问题。

12 兆像素的摄像头
----------

新款 MacBook Pro 配备了 12 兆像素的摄像头，支持 Center Stage。之前是 1080p，现在的像素数量提高了四倍，希望画质能跟上 iPhone 的水准。

新特性吸引人
------

新特性中，我觉得纳米纹理屏幕很不错，亮度提升了，14 英寸和 16 英寸的屏幕都更亮。![](.evernote-content/E6DA9996-D73D-46CE-BB7A-06963DED2CB3/36F151E5-7964-4E32-ACC2-AD133273973D)XDR HDR 格式依然保持在 1000 nits 的持续亮度和 1600 nits 的峰值亮度。而对于日常使用的 SDR 内容，亮度从 500 nits 提升到了 1000 nits。

结尾
--

总之，这次MacBook Pro M4、M4 Pro 和 M4 Max在性能和使用体验上都有不小的进步，是Apple最新一代的进化力作。好的，本期我们就到这里啦，感谢观看！我们下期再见！
