| |  | | --- | | (转自上海小学网站)  上海市徐汇区上海小学始创于清光绪二十九年（公元1903年），曾名为廿二铺小学堂，龙门师范附属小学、江苏省立第二师范学校附属小学、江苏省立上海中学附属实验小学、江苏省立上海实验小学等。解放后，学校改名上海市上海小学，1984年随上海市行政区划的调整，划入徐汇区管辖，改名上海市徐汇区上海小学。学校现是徐汇区内的一所大型公办小学，是上海教科院普教所教育科研实验基地之一，也是中国创造学会教育专业委员会实验基地。   学校坐落于上海市西南部的上中路，与文明遐迩的上海中学毗邻。校园占地约 30亩，建筑面积11000余平方米。校园内古树参天，小桥流水，鸟语花香，环境怡人。学校现有36个教学班， 1200余名学生，97名教职员工。   学校共有在岗中学高级教师7人，小学高级教师 63 人，共占在岗教师数的80%；35岁以下青年教师31人，占在岗教师数的35%，是一支充满朝气与活力的队伍。全校在岗教师中具有大学本科及以上学历的有78人，大专学历的9人，共占教师数的99%。   学校历史积淀厚重： 一幢和校史同龄的教学楼“行知堂”见证着学校百十年来的历程。从三四十年代学校校长——毕业于原东南大学教育系的盛朗西提倡的“在做中学，从学中做”的办学理念到现任校长提出“创上海教育品牌”的办学目标，从杜威的教育观到陶行知的教学思想再到目前上海市二期课改的新理念，学校始终以高标准的管理，高质量的教育教学站在徐汇教育的前沿。   一百多年来，学校培养了一批又一批的优秀毕业生，桃李遍布在各行各业，四海内外：中科院于光远院士，东南大学自动化研究所博士生导师、中国科学院冯纯伯院士，著名表演艺术家秦怡等等，都毕业于这座声望极高的学校。学校学生在体育、艺术、科技等比赛中屡屡荣获全国、市、区团体、个人第一名。 | |
