# 这里推荐的 Windows 应用，都经过了严格地人工挑选

---

我们生活在一个几乎万事都可「自动化」的时代，但这同时也让「人工挑选」成为了有温度、高质量和高品位的代名词。就像每天少数派更新的与数字生活相关的文字、每周五 Cuto 为大家呈上的编辑精心挑选的壁纸。

当朋友问起「上哪儿去找好玩、好用的 iOS / Mac / Android 应用」这个问题时，我总会有些自豪地向他们推荐少数派。而面对 Windows 用户，我还会向他们推荐另一个不错的地方，[「Windows 绝赞应用」项目](https://emlvirus.gitbooks.io/windows-apps-that-amaze-us/content/)。

### 什么是「Windows 绝赞应用」

「Windows 绝赞应用」项目由 @[EMLVIRUS](https://github.com/EMLVIRUS) 发起，旨在帮用户提供优秀的 Windows 应用以免除繁琐的搜寻工作。**我们姑且可以将其理解为一份人工维护的优质 Windows 应用推荐榜单**，前往 [该项目的 GitBook 页面](https://emlvirus.gitbooks.io/windows-apps-that-amaze-us/content/) 即可在线进行阅读和索引。

![](.evernote-content/3AC3D6D4-4A22-41A4-9D2D-64578C430C48/405A3ECD-45F8-415C-9F93-BD8C20625B5C.png)GitBook 的阅读体验简明清晰

截至目前，榜单已经已经收录了 **203** 个 Windows 应用，覆盖了音频文件、即时聊天、压缩处理、数据恢复、开发工具、隐私安全、系统增强等 **23** 个常见应用类别（俨然一个应用市场），以及「优秀国产软件名录」「黑名单」这两个极具特色的应用榜单。

我们可以在这当中找到 [Typora](https://typora.io/)、[Snipaste](https://sspai.com/post/35097)、[EagleGet](http://www.eagleget.com/)、[火绒](http://www.huorong.cn/) 等大家所熟知的 Windows 应用，也特别收录了 QTTabBar 这种久未更新但已经稳定的老牌 Windows Explorer 增强工具。

如果你是初涉 Windows 平台的萌新，它没准能让你在探索的道路上少走弯路；如果你是长年以 Windows 为主力操作系统的老用户，它也或多或少能在已有基础上进一步改善你的使用体验……总体而言，对于正在 Windows 生态中进行探索和发现的用户来说，这份榜单具有不小的参考价值。

### 榜单里的应用都是怎么挑选出来的

根据作者的筛选标准，被「Windows 绝赞应用」所推荐的 Windows 应用都满足以下特性，大家可以放心体验：

1. **特性精选**
2. **无侵入式广告**。存在非侵入式广告的，软件本身应当免费。若用户要求付费，则不应存在任何广告
3. **尊重隐私**：此处定义为：除非出于你或法律要求，否则该应用或应用提供商不会与任何第三方分享你的隐私信息
4. **维护良好**。若应用不再维护，则从此处移除
5. **易于学习掌握**：只要了解计算机基础操作即可完全掌握，需要简单学习的应有较高回报
6. **完全兼容 Windows 7 / 8.1 / 10**

值得称赞的是，**兼容性不高、有严格地区限制、****修改版和破解版破解版应用不会出现在该项目当中**。

另一方面，「Windows 绝赞应用」没有为名单中的应用标明具体的「流氓行为」和拉黑原因，但总体采用了以下拉黑标准：

1. 在自身安装包中捆绑其它软件的安装包（对于联网下载其它软件安装包且目标软件无明显恶意的行为的，本列表会额外注明但不会加入黑名单）
2. 收集敏感信息但是隐私权协议未说明或隐私权协议缺失的
3. 明显弱化系统安全性的应用，如后台植入、破坏系统文件等
4. 存在其它已知严重损害用户利益行为的

最后
--

作为一个人工维护的应用推荐项目，「Windows 绝赞应用」的后续补充和完善肯定离不开用户。如果你觉得这个项目对你有帮助，可以向身边的 Windows 用户多多推荐，注意声明 [GitBook项目来源](https://www.gitbook.com/book/emlvirus/windows-apps-that-amaze-us)；如果你想成为这个项目的一部分，作者也欢迎各位通过邮件或 Github 进行反馈。

另外，「Windows 绝赞应用」其实也受到了另一个 Github 项目 [Awsome Windows](https://github.com/Awesome-Windows/Awesome) 的启发，而 Awsome Windows 则是受 [Awsome macOS](https://github.com/iCHAIT/awesome-macOS) 的启发……这些项目涵盖了各自平台的应用推荐、系统安装设置流程等一系列参考资料，如果你的英语水平过关，稍加研读应该也能有所收获。
