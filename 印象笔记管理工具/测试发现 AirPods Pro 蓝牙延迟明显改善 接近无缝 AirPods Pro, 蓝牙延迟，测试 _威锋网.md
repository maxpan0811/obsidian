快过年了，想给自己换耳机了。

[![](.evernote-content/BA5F1F88-226F-4A51-85F7-075D1B5DBB8B/42F2FE6A-F093-4ED6-A834-11E1A24B9F49.png)](https://resource.feng.com/resource/h075/h53/img201912232208580.png "测试发现AirPods Pro蓝牙延迟明显改善 接近无缝")

苹果在数月前推出 AirPods Pro，可能有人感觉 AirPods Pro 和 AirPods 的区别只是变成了入耳式，加多了一个降噪功能，但其实最近的测试发现，AirPods Pro 在无线耳机延迟问题上有显著的改进。  
  
音乐家和软件开发人员 Stephen Coyle 在自己的博客文章中探讨了 AirPods Pro 在蓝牙延迟时间方面的改进。他进行了一系列测试，测试结果显示，AirPods Pro 的延迟是「接近无缝连接」体验。  
  
Stephen Coyle 分别用了第一代、第二代 AirPods、Beats Studio 3、索尼 WH-CH700N 和设备扬声器与 AirPods Pro 进行对比。  
  
测试环节分为两部分，一个是系统的键盘声反馈，另一个是测试者自己开发的游戏《Tapt》。前者键盘声并不连贯的，在使用键盘时，系统需要先唤醒蓝牙模块，然后播放声音，所以我们敲打键盘时的第一声往往会比之后的反应明显要慢一些；而后者由于是游戏，因此会持续唤醒蓝牙模块，使耳机会第一时间响应每个声音，同时这个游戏本身低频延迟表现也比较不错。

[![](.evernote-content/BA5F1F88-226F-4A51-85F7-075D1B5DBB8B/7AB0B207-5E57-4937-BB30-08E480F35C9B.jpg)](https://resource.feng.com/resource/h075/h53/img201912232209180.jpg "测试发现AirPods Pro蓝牙延迟明显改善 接近无缝")

对于所有设备和软件组合，Stephen Coyle 均使用游戏 90bpm 的节拍器触发 10 种声音，并通过敲打系统键盘，将 19 个随机测试结果进行平均计算后得出如下结果（在键盘测试时，Stephen Coyle 没有记录第一声的反应时间）：  
  
AirPods（第一代）：274 毫秒  
AirPods（第二代）：178 毫秒  
AirPods Pro：144 毫秒  
Beats Studio 3：258 毫秒  
索尼 WH-CH700N：230 毫秒  
设备扬声器：69 毫秒  
  
测试结果可以看出，使用苹果 H1 芯片的新 AirPods 和 AirPods Pro，相比于之前的 W1 芯片的第一代 AirPods 在延迟时间上大大改善，缩减了 30%。而 AirPods Pro 特比同为 H1 的 AirPods 第二代出色，延时时间缩减到 144 毫秒，缩减幅度接近初代 AirPods 的 50%，达到人耳几乎分辨不出延迟的水准。  
  
因此 Stephen Coyle 认为苹果似乎已经找到在不变换硬件的前提下缩减耳机延迟时间的办法。

[![](.evernote-content/BA5F1F88-226F-4A51-85F7-075D1B5DBB8B/823A0220-97BB-4057-A046-646E3661817A.jpg)](https://resource.feng.com/resource/h075/h53/img201912232209330.jpg "测试发现AirPods Pro蓝牙延迟明显改善 接近无缝")

另外，配备 W1 芯片的 Beats Studio 3 和第一代 AirPods 结果接近，为 258 毫秒；索尼 WH-CH700N 为 230 毫秒。另外 Stephen Coyle 还测试了第三代 Amazon Echo 和 JBL 的蓝牙耳机，结果都和 Beats、索尼耳机的测试结果接近。  
  
相比于过去的 AirPods 和其他无线耳机，作为音乐人的 Stephen Coyle 认为 AirPods Pro 在延迟方面的提升还是相当可观的，尽管 144 毫秒在游戏和部分音频制作中仍然还在人耳能察觉到的延迟范围内，不过它确实有足够的理由成为 Stephen Coyle 的首选无线耳机。
