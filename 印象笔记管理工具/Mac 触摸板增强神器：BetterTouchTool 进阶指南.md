Mac 触摸板增强神器：BetterTouchTool 进阶指南
================================

伴随着个人电脑的发明，键盘和鼠标也作为其辅助设备相继出现，这句话虽然合理但又不严谨，因为键盘早在打字机时代就已经存在了，而鼠标对于绝对的效率用户来说，基本上只属于一个可有可无的东西。也就是说，最好的操作工具其实是「手」（咳，不要想歪……）。

如何充分发挥「手」的作用？最重要的一点，就是让硬件与其能完美融合，也就是所谓的「人性化」，而出于对利益和生态发展的谋略，没有一家大众消费品厂商愿意提供近乎完美的方案，因此这就给 BTT 这类强大的辅助应用带来了空间。本文的目的也就随之产生了：通过对 BTT 功能的深入了解，让你拥有更细致、更易用的「人性化」操作方案。

让 Command 更好地掌控全局
-----------------

它的作用好比手动档汽车里的离合器，光用脚踩不会有任何反应，但换挡、起步、怠速等重要操作都离不开它，Command 键（简称「CMD」）在 OS X 系统下的重要性也是如此。它的独特优势在于，当双手自然放置在键盘上时，你的大拇指总是不经意地触碰到这个键。

我们可以利用 **BTT 修改系统快捷键**的特性，通过 BTT 合理地将 CMD 代替其他不太好按的组合按键，那么操作效率定会大幅增加。举些例子：

* Emoji 表情输入：⌃ ⌘ Space（默认） → ⌘ [（自定）
* 增加 Safari 标签页：⌘ T（默认） → ⌘ 右方向键（自定）
* 关闭 Safari 标签页：⌘ W（默认） → ⌘ 左方向键（自定）
* 全屏窗口：⌃ ⌘ F（默认） → ⌘ ]（自定）
* 隐藏当前窗口：⌘ F（默认） → ⇧ ⌘ D（自定）

或直接创造一些实用的操作方式：

* 隐藏所有窗口：⇧ ⌘ 数字键1
* 清除废纸篓：⇧ ⌘ 数字键2
* ......

关于 CMD 键的玩法有很多，读者们可以随意发挥自己的想象力，不过遗憾的是，目前版本的 BTT 不支持双击作为快捷键（可能是由于双击不算组合键的原因吧）。但即使如此，这些也不过是 BTT 强大功能的冰山一角，还有更有趣的。

别低估了鼠标 / 触控板
------------

相信有很多人买 Mac 电脑的理由，一定少不了对多点触控（Multi-Touch）设备的喜爱，苹果官方标配的 Magic Mouse / Magic Trackpad 就是最好的榜样，因为无论是在 OS X 系统，还是 BTT 应用本身，触控板在外设中绝对占有举足轻重的地位。真的有这么玄乎？

举些例子，我们可以使用 BTT 将「按键操作」设为「手势操作」：

* 增加系统音量：F12 → TipTap Right（双指，左指按住，右指点击）
* 减小系统音量：F11 → TipTap Left（双指，右指按住，左指点击）
* 打开 Finder 窗口：⌘ N → Four Finger Tap（四指轻拍）
* 睡眠 / 登出：Four Finder Click（四指点击）
* 开启手写输入：⇧ ⌃ Space → Single Finger Tap Left Side Middle（单击左侧中）

新的手势控制方法是不是离「人性化」更进一步了呢？！在享受便捷操作的同时，也要考虑到这些 Gesture 对其他应用的影响，如果存在冲突，或是共存引起了连续激发现象，那么笔者建议采纳下节的方法。

尴尬时，请选择修饰键
----------

顾名思义，修饰键是起修饰作用的，而且修饰的对象是鼠标，所以也可以这么认为：这是项整合了 Mouse 与 Keyboard 的功能。BTT 支持的修饰键为 ⇧ Shift、fn、⌃ Control、⌥ Option、⌘ CMD 等，可以单选也可以多选。个人建议，这里的设置要尽可能地利用剩余的 Key，因为高效率是基于理性的分配，而不是没理由的忽略。

软硬件结合，怎么「硬」？
------------

Apple Remote 是针对 Apple TV 无线遥控器开发的特殊功能，开启后用户可以通过  遥控器同步 BTT 实现对 Mac 的自定义操控，除此之外，它还支持与 iOS、Apple TV 设备联动，实现更多丰富的功能。

![](.evernote-content/D5EDD0CE-7E12-4D7F-B369-A01191DF1480/D925C2F8-9839-45C4-83F2-F03861A101E0)

Wacom tablets 是由 Wacom 开发的专业绘图平板电脑，也称作数位平板电脑，旗下还有触控笔等创意产品，这里就不在叙述了，具体内容详见 [Wacom](http://www.wacom.com.cn/) 官网。当然，有这方面需求的用户可以在 BTT 设置里开启体验。

所有 BTT 支持的硬件外设中，最火爆的当属 Leap Motion 无疑了。这是款小巧的 USB 体感控制设备，能够感知双手在空气中的自然移动，精确跟踪手和手指的运动。在与 BTT 连接后，还可实现更多特殊的手势操控方式，比如：转圈、提拉、伸缩、滚动等，而这一切都是在空气中完成的，非常酷炫！

BTT Remote
----------

这是 BTT 官方开发的伴随应用，属于控制型工具，在 [App Store](http://itunes.apple.com/us/app/btt-remote/id561676304?mt=8) 的售价为 ¥12 元人民币，虽说功能比较偏向于趣味性，而且远程连接的方式是基于端口的（不够安全），但用来尝鲜还是可以接受的。

![](.evernote-content/D5EDD0CE-7E12-4D7F-B369-A01191DF1480/2268FAB1-6340-4CC3-9EEF-F0B74E8AA123)

目前 BTT Remote 支持的特殊操控方式有：

* Shake Device 摇晃手机
* Press Volume Button Up 按音量加键
* Press Volume Button Down 按音量减键
* Three Finger Swipe Left 三指左划
* Three Finger Swipe Right 三指右划
* Three Finger Swipe Up 三指上划
* Three Finger Swipe Down 三指下划

还有......
--------

BTT 的设置项非常非常多，远不是一篇文章能介绍得全的，但其中大部分选项都没必要修改，因为它们几乎都是针对细微操作的设定，比如：触摸压力、触摸时间、触摸位置（正中、正左、正右等）。给我的感觉是，这些完全是为机器手臂定制的，对普通大众用户而言，选择忽略最佳，否则还会导致莫名其妙的报错。

注意，本文提到的所有设置都是通过 BTT 实现的（BTT 的作用相当于联动，即通过其它组合键去激发系统默认组合键或功能，而不是直接修改）因此，使用时别忘了将其后台运行。
