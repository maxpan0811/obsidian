你是否苦恼于撰写论文时插入引文繁琐步骤，在导师修改文章后你又要一字一句地重新插入引文，每次都被文献折磨得死去活来，那么你一定是没有掌握文献管理的正确姿势。

是否一定需要把全文下载后才可以导入文献？换了投稿杂志如何一键更改符合编辑要求的引文格式？纷繁杂乱的教程看完还是不知所云？

step 实现无痛入门。

如何选择文献管理软件
----------

市场上有众多的文献管理软件，这里列出一些可供选择的软件。

| 软件名称 | 当前最新版本 | 适用平台 | 费用 |
| --- | --- | --- | --- |
| [EndNotes](https://endnote.com/) | EndNote X9 | Windows/Mac/[iPad](https://itunes.apple.com/cn/app/endnote-for-ipad/id593994211?mt=8&elq=415a266a2b374ca1b03bc5c1d3109245) | [￥1818.00](https://item.jd.com/10009154059.html) |
| [Mendeley](https://www.mendeley.com/) | Mendeley 1.19.2 | Windows/Mac/Linux/[Web](https://www.mendeley.com/library/)/[iOS](https://itunes.apple.com/cn/app/mendeley-reference-manager/id380669300?mt=8)/[Android](https://play.google.com/store/apps/details?id=com.mendeley&hl=en_GB) | 免费 |
| [NoteExpress](http://www.inoteexpress.com/aegean/) | NoteExpress v3.2.0.7103 | Windows/iOS/Android/Web插件 | ¥998 |
| [E-Study](http://elearning.cnki.net/) | CNKI E-Study 3.3 | Windows/Mac/[Chrome插件](https://chrome.google.com/webstore/detail/cnki-e-study/eiphfejdlidfknjiockmkfpkfmnheljl) | 免费 |

（请使用正版软件，若你是在校学生，鼓励你咨询校方（图书馆）是否有提供正版的文献管理软件）

不同的软件操作逻辑略不相同，但是功能相似，本教程以 **Mendeley** 和 **E-Study** 为例子（演示文献管理的基本步骤（视频版请[查看此处](https://www.bilibili.com/video/av35566918/)），若需要进一步了解使用方法请查阅各软件的帮助/说明文档。

注意：Mendeley 对中文支持一般，若你查阅的文献中**包含大量中文词条，推荐使用 E-Study 软件**。

### 1. 安装文献管理软件

登陆 [Mendeley](https://www.mendeley.com/) 或 [E-Study](http://elearning.cnki.net/) 网址下载适合的客户端并安装。

### 2. 登陆账号

部分文献管理软件支持同步功能，建议在使用前注册并登陆账号，进行同步备份。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/C531A08C-B0A0-43D1-83B4-FB73E63300C2.png)

Mendeley Desktop 和 CNKI E-Study 客户端登陆界面

查找引文的四种方法
---------

### 方法一：使用内建搜索器导入文献

Mendeley 软件**内置了搜索器**，可直接按关键词**搜素互联网中的文献**，并添加到指定的文件夹中。Mendeley 文献分类逻辑以文件夹为单位，与操作系统文件分类管理逻辑相似。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/AA17C1A7-A164-4548-BB74-D186354B4BF0.png)

可以使用 Mendeley 中的 Literature Search 功能搜索文献。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/1C003FFA-56B1-4C13-A09A-356E8045FEF9.png)

### 方法二：结合泛用型搜索引擎导入文献

建议使用 [Google Scholar](https://scholar.google.com/)、[百度学术](http://xueshu.baidu.com/)、 [Bing 学术](https://www.bing.com/academic)等泛用型**文献搜索引擎**查找文献，再将词条导入文献管理软件。

#### GOOGLE SCHOLAR 使用简介

在这里打开 [Google Scholar](https://scholar.google.com/) 。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/EC69E70F-D719-4E25-87C3-569C1EB44D8B.png)

Google Scholar 主页

输入关键字（如题目）进行文献查找，可点击含 PDF 关键字的链接下载文献**全文**，也可以点击`"`图标选择复制或下载相应格式的**引文条目**。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/F7594C0D-0B17-4B2A-9820-FB855C8B2F3E.png)

选择 BibTex 格式（鼠标右键点击超链接 - 选择「链接另存为...」 - 保存为后缀是`bib`的文件）保存。

注：[BibTex](http://www.bibtex.org/) 是文献数据格式的一种，通常情况下与 LaTeX 文档结合使用，大部分的文献管理软件都支持导入这种格式。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/F506EF16-8210-4A32-AB32-EBFA4138A1F3.png)

其他的文献搜索引擎平台操作类似，此外还可以使用**文献数据库平台**查询下载文献，如[Pubmed](https://www.ncbi.nlm.nih.gov/pubmed)（主要收录生物医学文献）、[中国知网](http://www.cnki.net/)、[万方数据](http://wanfangdata.com.cn/index.html)、[维普网](http://www.cqvip.com/)等。

#### 中国知网使用简介

首先来到[中国知网首页](http://www.cnki.net/)。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/0F34B36A-5FDE-4E6F-8D23-F0D22CB2041B.png)

中国知网首页

找到文献后，选择目标文献，点击「导出/参考文献」。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/E323715A-00B9-43E9-90B4-44F47A8DA1DC.png)

选择适合的文献格式导出。图中第一个箭头所指的是比较常用国标格式。第二个箭头所指的是 E-Study 软件专属的格式，文件后缀为`eln`。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/516B8892-2E32-45DD-8CA0-BCCE2CA85B28.png)

#### 文献条目导入软件

导入 Mendeley 的步骤：「File → Import.. → BibTaX」，选择文档后导入。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/04082563-B20A-428A-8263-40BC083E290D.png)

导入 E-Study 的步骤略有不同，在学习单元处右键 → 新建学习单元。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/420E374E-9CF8-4842-B379-EFB9E4BA3E87.png)

E-Study 文献分类逻辑以「学习单元」为单位，下属两个子目录：**文献库**和**笔记库**，点击「添加文献」添加已下载的条目。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/70971B11-E48A-49C5-8C72-E87BF05B3568.png)

由于软件**识别信息不完整**（图示标题未能识别成功），可通过鼠标右键点击选中的条目，选择「更新题录信息」让软件从互联网中抓取匹配的条目信息。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/79A7EF0B-CEF6-44BA-B64D-F4B3C49971A4.png)

在检查抓取的内容无误后，勾选相应的选项进行条目信息更新。一般从互联网抓取的信息都比较准确，可以勾选「不在提醒更新字段的方式」以「更新全部字段」为默认更新方式。

### 方法三：添加全文导入文献

若本地已保存了全文文件，一般是后缀为 PDF 的文件，可通过添加文件导入文献。

依然是选择 Add Files 将全文导入 Mendeley。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/394BBD08-A35F-4FBB-BD21-A0AE44BBC375.png)

通过添加全文的方式导入的文献会关联本地的 PDF 文件，点击箭头所指图标，可直接打开对应的文件。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/058ECAB4-C9E9-4B6C-A077-BC0E7FAB3452.png)

### 方法四：手动编辑导入文献

当上述的方法无法导入目标文献时，可以通过手动键入文献信息，导入文献条目。

点击 「Add Entry Manually..」 来新建一个文档。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/1DF3B4A2-E44E-4107-A1FB-3188A59909E7.png)

在新界面中你需要为文献手动编辑文献信息。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/7CE7C80C-993B-4D06-8EC5-907CBA6A60BE.png)

插入引文
----

### 1. 安装 Word 插件（Mendeley）

将引用文献插入到论文中时，需要先在文本编辑器（如Word中编写）安装相应的文献管理软件的**插件**。

一般在安装文献管理软件前，若系统中已安装了文本编辑器，则会自动关联安装插件，也可以手动安装插件。

在 Mendeley 的 Tools 中选择 Install MS Word Plugin。将 Mendeley 插件安装到 Microsoft Word。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/47D470CA-2B51-41CB-9D80-ABD6308F425E.png)

安装完毕后，Mendeley 插件会显示在 Word 的引用模块中。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/93729C0D-8858-4ED2-B2C0-847D23B493F9.png)

E-Study 插件的安装方法类似，安装完毕后会在单独的 CNKI E-Study 标签中显示。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/6771B7C8-953C-410A-B8A0-B397C21306B8.png)

### 2. 选择插入格式

#### MENDELEY 插件改变引文格式

在 Mendeley 插件中的`Style`选项中选择所需的引文格式。如果所示的选项不满足的要求，可点击`More Styles...`选择更多的格式，若本地建的数据库没有你需要的格式，可到[在线仓库](https://github.com/citation-style-language/styles)下载更多的引文格式，也可以手动修改格式。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/161099ED-0868-42D1-9C4A-144388AA2497.png)

#### E-STUDY 插件改变引文格式

在 E-Study 插件中的`更改引文样式`选项中选择所需的引文格式：

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/531138FA-ABEA-4CC8-A43C-74251BCFDB40.png)

### 3. 插入文献

将**光标**移至需要插入文献（标为黄色处）的位置，在搜索框内输入**需要插入文献的关键字**搜索，选中目标文献后点击`OK`即可插入**数字上标**。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/B6A987D7-2E43-49C6-8710-4EB4B1984D03.png)

将光标定位到放置文献的位置（如**文章末尾**），点击「Insert Bibliography」插入具体引文：

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/999784BD-B8A7-4485-8EEC-0DF8CCEBD172.png)

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/F9EC9DF5-9F04-4AB2-A5EE-CE29D1171A26.png)

E-Study 插入文献的方法与 Mendeley 方法类似。

![](.evernote-content/A1D1E5AB-E808-4328-A506-0D34E6884FE6/0767D7EE-9B2D-4B92-BAF2-716FCBA8BCA4.png)

使用文献管理软件漫谈
----------

每年的学期末、毕业季，不少都被论文伤透了心，其中如何查询文献、整理文献，以及最后如何按照要求插入引文，这些最不起眼的小问题却是最难解决的，看着舍友复制、粘贴一晚上的引文，其实按照上述的流程可以在几分钟内解决插入引文的工作。

与文件夹管理文献相比，使用软件更方便查阅与整理文献，云同步存储功能有效地避免数据丢失，极大提高了工作学习效率，单是这一点我觉得就十分值得花时间学习文献管理软件。

结合文献管理插件与文本编辑软件（如 Word ），可以十分方便地一键修改文献格式、增删文献（自动重编序号）。

应该注意文献管理软件对于中文支持的缺陷，容易造成乱码等问题，可以参阅帮助文档和网络教程。

文献管理软件常使用专属的**文件格式**储存数据，对于在不同软件和用户间**共享**、**迁移**文献数据库极不方便，用户可以考虑使用通用型的格式存储或导出（如
BibTex）数据。

文献管理软件一般指的是**文献「条目」管理**，对于全文格式的文献一般以 PDF 格式存储，阅读时标记在文章中的笔记，依然是大部分软件管理的弱点，无法快速地检索和整理，而 E-Study 和一些 PDF
阅读器（如 [LiquidText](https://www.liquidtext.net/)、[MarginNote](https://www.marginnote.com/) 等）在笔记管理功能更值得借鉴。
