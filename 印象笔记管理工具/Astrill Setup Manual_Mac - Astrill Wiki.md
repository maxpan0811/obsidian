Getting Started with Astrill application setup on Macintosh OS X
----------------------------------------------------------------

### Downloading and Installing Astrill application

1. Download the setup installer from [Download](https://www.astrill.com/download.php) page. Do note that you need to click on any one of the Mac OS X link and download the file to your Mac computer. It should look like this after been downloaded on to your computer.

[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/62F99CF1-F744-4594-8530-C0E8A5232B63.png)](http://wiki.astrill.com/index.php/File:Mac_OS_Setup_Icon.png "Mac OS Setup Icon")

2. Double-click the "astrill-setup-mac.dmg" file that should be on your computer in your desktop, or downloads folders. If you are using Mac OS 10.8 or above, you need to press Ctrl+click on the file icon and then select Open.

[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/DE4ECD12-D2B9-475E-92F3-F001D0A1C2AB.png)](http://wiki.astrill.com/index.php/File:Astrill_Setup_Screen.png "Astrill Setup Screen")

3. If for some reason, while trying to run the setup you get a popup warning like this

[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/8DE7AD4F-4DD9-4E2E-A7B2-B0B7A5072C7C.png)](http://wiki.astrill.com/index.php/File:Warning1.png "Astrill Warning Prompt on Mac")

you can run the installer again by pressing Ctrl+click to Open the file and then choose to allow the following prompts to start the installation.

[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/4563156E-0ABA-4BA4-B451-4455B09543A9.png)](http://wiki.astrill.com/index.php/File:Solution.png "Astrill Setup Screen")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/0FE498A0-92DA-49FB-A263-1F5AA03A66F7.png)](http://wiki.astrill.com/index.php/File:Warning2.png "Astrill Warning Prompt on Mac")

4. You can now proceed with the installation of Astrill application on your computer. You can refer to these screenshots of installation process as a guide.

[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/19FB8A7F-C79F-459A-A3B7-1628DF32885E.png)](http://wiki.astrill.com/index.php/File:Intro.png "Installation Step 1")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/BE22C530-B6E9-4EC6-8BE0-8C08AF732FFE.png)](http://wiki.astrill.com/index.php/File:License_Agreement.png "Installation Step 2")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/D4EE17FB-0362-4644-A880-1745B36CC58E.png)](http://wiki.astrill.com/index.php/File:License_Prompt.png "Installation Step 3")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/E78355A7-5B36-4D14-83AA-66CF44F84E2C.png)](http://wiki.astrill.com/index.php/File:Destination.png "Installation Step 4")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/CCEA90F8-C117-4AD2-8895-EC124E6F9EE0.png)](http://wiki.astrill.com/index.php/File:User_Prompt.png "Installation Step 5")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/39916ECA-80D4-4E68-B867-918BB104F966.png)](http://wiki.astrill.com/index.php/File:Summary.png "Installation Step 6")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/BCF40CD2-80B7-421E-B2ED-95B9CB4673D9.png)](http://wiki.astrill.com/index.php/File:Eject.png "Installation Step 7")

In any case, if you still face any issues or have any additional queries, feel free to [contact us](https://www.astrill.com/contact-us.php) for help.

### Running Astrill application

1. Once Astrill application has been installed successfully, you can now run the application. Open Finder and then go to Applications folder. Find Astrill application (looks as grey star icon) and double click on it to open.

[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/93B29FD2-C570-4426-9A7C-9BEA04A0829D.png)](http://wiki.astrill.com/index.php/File:Finder.png "Mac Finder Screen")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/CF254385-F9BA-4108-AB22-D4F33AE2AC19.png)](http://wiki.astrill.com/index.php/File:Applications.png "Mac Applications Folder")

2. You can now login into Astrill application window by using your Astrill account login details (check your email inbox for "Welcome to Astrill" email you got from us). Once logged in you would be connected in Openweb mode by default. You can select any server of your choice from the drop down list of application. If you want to turn Astrill proxy off for some reason temporarily, you can do that by pressing the ON button once. It would toggle to OFF state and your real IP would be exposed again in the browser. You can switch back between ON/OFF real-time with no restrictions on switches whatsoever!

[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/CD1F8C7B-3988-4E69-9944-3F73F54C0B82.png)](http://wiki.astrill.com/index.php/File:Login.png "Astrill Login Screen")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/B8E28BCD-A4AD-4B7E-926F-0FE61E53A5E6.png)](http://wiki.astrill.com/index.php/File:Openweb_ON.png "Astrill Openweb ON")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/0B91C0A9-FF31-4B13-912F-0EC85D4F0009.png)](http://wiki.astrill.com/index.php/File:Servers.png "Astrill Servers List")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/AC2BCCC9-0F7A-4E97-B6D5-65D98D18623C.png)](http://wiki.astrill.com/index.php/File:Openweb_OFF.png "Astrill Openweb OFF")

3. You can even switch to Openvpn mode within Astrill application if you want to tunnel internet traffic from all your applications via VPN. Look for Openweb/Openvpn drop down menu in title bar and change to Openvpn mode. Now press on OFF button once to start connecting to the selected server of your choice. Wait a minute and you should be connected when you see a ON button with lock icon against the server name. If you want to disconnect, simply press the ON button again and it should disconnect from the server.

[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/A07B442F-FD83-4FF5-A2BC-ABF395F738C5.png)](http://wiki.astrill.com/index.php/File:Modes.png "Astrill Mode Selection Screen")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/99797838-9C5D-4657-B7EF-2CDC6E3C0BDE.png)](http://wiki.astrill.com/index.php/File:Openvpn_OFF.png "Astrill Openvpn OFF")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/9F94BF90-29BD-4972-89DD-13D4D204BB6B.png)](http://wiki.astrill.com/index.php/File:Connecting.png "Astrill Openvpn connecting phase")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/02A7C6E5-B885-4693-A1A6-54745F5F535F.png)](http://wiki.astrill.com/index.php/File:Openvpn_ON.png "Astrill Openvpn ON")

4. Once you are connected in Astrill application, you are ready to surf! Just check your IP address by clicking on Astrill icon and selecting "Check My IP" option. You can allow any firewall prompt you may get and it should open a page to confirm your IP address. If you see Astrill IP, you are good to go!

[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/00D25855-5FAF-4025-BEA2-D04C900B6600.png)](http://wiki.astrill.com/index.php/File:CheckIP.png "Check My IP address")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/8E35247E-3A12-46CC-89F3-DAE48D8B239F.png)](http://wiki.astrill.com/index.php/File:Firewall.png "Firewall Prompt")[![](.evernote-content/FB92A500-F6FB-4CB6-9278-30DFED76BF3F/0D8BF8E8-7B23-4020-B9C1-2AE3796F12A8.png)](http://wiki.astrill.com/index.php/File:ConfirmIP.png "Astrill IP address")

You can even take a tour of our application by visiting [How it works](https://www.astrill.com/how-it-works.php) page. We hope you enjoy Astrill - the fastest VPN service on the internet!
