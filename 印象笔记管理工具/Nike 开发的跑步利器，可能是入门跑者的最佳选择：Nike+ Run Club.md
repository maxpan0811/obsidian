# Nike 开发的跑步利器，可能是入门跑者的最佳选择：Nike+ Run Club

---

跑步已然成为大多数人首选的运动方式，除了拥有一双合适的跑鞋，找到一款好用的跑步 App 也成了许多跑者的需求。从最基本的跑步记录功能出发，出身运动名门的 [Nike+ Run Club](https://itunes.apple.com/cn/app/nike-run-club/id387771637?mt=8&uo=4&at=10lJSw) 因其功能上的纯粹和恰到好处，可以成为入门跑者的首选。

核心功能：跑步记录
---------

作为一个跑步 App，最基本也是最核心的功能便是对跑步的记录。一打开 NRC（即 Nike+ Run Club），默认的初始位置便是位于 App 功能栏最中心的「即刻开跑」界面。

![](.evernote-content/C3454722-E7E1-440B-968A-3C95B032F760/ABC29283-7D45-4211-80B9-73A6CBE2F9A3.jpg)开始跑步界面

在第一次开跑之前我们可以到设置中去进行一些简单的配置，其中包括自动暂停、语音提醒和显示方式等。如果你在跑步时会佩戴第三方的心率监测配件，NRC 也提供了相关的联动功能，可以让你在手机屏幕上看到所有跑步相关的数据。

![](.evernote-content/C3454722-E7E1-440B-968A-3C95B032F760/14811A9E-100E-4FF3-9C02-1D0C50C55981.jpg)跑步设置

NRC 在界面上方提供了四种跑步记录方式，分别是基本、距离、时长和疾速跑。

* **基本**，不设任何目标地开始跑步。
* **距离**，设定一个距离目标，跑步时会有一个进度条实时显示完成进度，达到目标时会有提醒。
* **时长**，设定一个时间目标，同上。
* **疾速跑**，适合搭配训练计划使用，可以在多次疾跑间歇中标记停顿和次数。

![](.evernote-content/C3454722-E7E1-440B-968A-3C95B032F760/81105B5C-55AF-43A8-A24C-111A3F510D4E.jpg)不同的跑步模式

开始跑步后，NRC 会和你一样进入「跑步状态」，界面开始只显示与你当前跑步相关的数据：配速、距离、时间。（有硬件支持才会显示心率，否则为空）

![](.evernote-content/C3454722-E7E1-440B-968A-3C95B032F760/F4281E10-91B0-47D3-9C03-A2C09CEBBFAA.png)跑步状态的 NRC

左上角的坐标图标的颜色代表着当前 GPS 信号的好坏，如果想要数据更加准确，自然需要选择 GPS 信号好的地方进行跑步。右上角的锁子图标可以「锁住」当前界面，防止误触，如果你跑步时习惯把手机放在口袋里，建议开启屏幕锁，以防止误操作手机。暂停键旁的相机图标可以快速打开相机，方便跑者记录眼前的风景。此外，NRC 也可以和 Music 联动，直接播放音乐库里的音乐。

当跑步结束后，NRC 会让你对本次跑步的难度感受和使用的跑步鞋款进行选择，最后和其他数据一并记录在内，方便日后查阅。

回顾表现：数据汇总
---------

跑步完成后的工作自然是查看本次或以往跑步的数据。在 「活动」界面，NRC 会在上方显示你跑步以来的数据和，在下方的列表展示从你第一次跑步开始所有的跑步记录，点击其中任意一次记录即可进入查看详细数据。右上角的加号允许用户自己添加跑步数据。

![](.evernote-content/C3454722-E7E1-440B-968A-3C95B032F760/095AB7DF-53D6-49B1-9CF0-D698E89BA470.jpg)「活动界面」展示了跑步数据

进入具体某一次的跑步记录后，可以查看该次跑步的所有数据，除了地图和基本的跑步数据外，还会有跑步后你选择的难度，以及你所使用跑鞋的里程总数，方便你评估自己的训练强度和跑鞋的磨损状况。不过鞋款的数据库只有 Nike 自家的鞋款，若你使用的是非 Nike 跑鞋，可以随意选择一个代替，只看里程数据就好。

点击「查看详情及分段记录」按钮后，便可以查看你整个跑步过程中具体到某一个位置，某一个时间点的配速和心率数据，还有分段成绩，一并帮助你回顾这次跑步的每一个细节，为更好的表现提供了数据基础。

![](.evernote-content/C3454722-E7E1-440B-968A-3C95B032F760/D5FC14C8-3B41-4942-8EB8-8503C1D768E0.jpg)可拖动进度条式的跑步数据

制定训练计划：我的教练
-----------

对于初入门的跑者，如果想靠一些简单的计划提高自己的跑步表现，NRC 提供的「我的教练」不失为一个好选择。在回答几个问题后，NRC 会自动根据你的信息生成一个定制的跑步计划，并提供相应的日程表，这个日程表很灵活，如有需要的话可以自行随意调整甚至终止。

![](.evernote-content/C3454722-E7E1-440B-968A-3C95B032F760/AA583166-29AA-4EC8-A0DC-C03AD17F708B.jpg)

在每一次跑步完成后，都可以选择是否把该次跑步纳入训练计划中去。并且每周都会有一个训练报告被推送给你，供你查看自己的计划完成情况。

成就系统：拿徽章！
---------

成就系统其实也可以叫做徽章系统，当你完成一些目标时，系统会自动给你发放徽章，以鼓励你继续运动，突破自我。

![](.evernote-content/C3454722-E7E1-440B-968A-3C95B032F760/4FFA7CC0-146C-4551-9104-EF9A9FD312CA.jpg)各式成就徽章

其中也不乏一些有趣的徽章，例如：

![](.evernote-content/C3454722-E7E1-440B-968A-3C95B032F760/D999DF89-ECD0-4208-8A8D-316BDCF93BE4.jpg)意外收获的徽章

当时得到这个徽章时我还挺意外，不过转而便感到了一丝欣喜，这样一个暖心的细节，更让人产生了坚持跑步的信心。

NRC 基础的跑步记录功能做的非常到位，整个 App 的主次功能十分分明，因而即便是成就和社交元素的加入，也并不影响其实用性。也是由于在大陆的水土不服，NRC 的社交元素略显鸡肋，故在此不做介绍。

Apple Watch 支持
--------------

有 Apple Watch Series 2 内建 GPS 的支撑，摆脱手机手机成为了现实。NRC 也提供 watchOS 应用，可以在跑步时独立于手机运作。

在 Apple Watch 上打开 NRC 后，可以通过滑动切换三个界面，分别是当月的跑步总结、开始跑步和设置。在这里需要注意一下，「所有指标」这个选项默认为关闭状态，只有打开后才能在跑步过程中看到包括配速、时间、距离和心率的所有指标，个人建议是保持这个选项开启，以便在跑步过程中得到更详细的信息。「自动暂停」这个选项不消说，开启后便可以在你等红灯或是停下歇息这类场景自动暂停跑步，以免干扰有效的跑步数据的记录。

![](.evernote-content/C3454722-E7E1-440B-968A-3C95B032F760/7B5EC8B3-6D53-4C18-A87E-8ABBD852C7FD.jpg)NRC 在 Apple Watch 上的三个开始界面

开始跑步后，和手机 App 一样，NRC 会在屏幕上主要显示配速、时间和距离这三个数据，默认为高亮显示距离，但是通过转动 Digital Crown 可以改变高亮显示的指标，所以你可以根据自己的需求来得到自己感兴趣的信息的高亮。

![](.evernote-content/C3454722-E7E1-440B-968A-3C95B032F760/45437799-DF71-4005-B0CA-9C881F23FFB6.jpg)分别高亮显示：距离，时间和配速

完成一次跑步后，NRC 会在屏幕上显示本次跑步的概要，用线条勾勒出跑步路线的大致轮廓，展示给用户，并且每次都会送上一段随机的激励话语，鼓励你坚持进行跑步。

![](.evernote-content/C3454722-E7E1-440B-968A-3C95B032F760/66725F3C-9D76-4668-A0C4-44C4C7DE6ABD.jpg)完成跑步后的界面

最后，当你完成跑步回到家中后，只需要在手机上打开 NRC，即可把跑步信息从手表中同步到手机和云端，一次完整的不需要手机的跑步体验就此完成。
