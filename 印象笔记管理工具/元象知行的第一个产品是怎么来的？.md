# 元象知行的第一个产品是怎么来的？

---

速读摘要

上海这座城市本身也极有代表性，它的发展脉络不仅代表了上海自身，而且在一定程度上反映了全中国乃至全球的经济脉络，可以成为极好的研学样本。今天的上海成为中国最重要的长三角经济区的龙头，国家四大科学城之一，未来它又将走向何方?只有在良好的生活氛围中，在生活中学习和创造，才能最大限度地发展学生的聪明才智。以上三者分别是教育的"术法道"，挑战式学习把学习变成了打怪升级的游戏体验，现象式学习则告诉我们真正有效的学习要来自与自身息息相关的各种现象，形成跨学科解决问题的能力，而生活教育理念则是告诉我们，只有人拥抱生活，热爱生活，才能成为真正的学习者。

缘   起

2023年3月的一天，大女儿子衿问我夫人一个问题：

为什么上海是中国经济排名第一的城市？未来还会继续保持第一吗？

如今回顾往事，女儿的这句话，不仅促成了这次令人难忘的五一之行，也间接导致了“元象知行”的诞生。

![](.evernote-content/FFB2376D-0F6E-4273-B9AA-4AD3815E715D/C2AF08B3-6C69-4AEF-A33E-CED1C5B77D14.png)

夫人跟我闲聊时提起女儿的问题，我突然意识到，孔子云：“十五有志于学”，孩子今年十二岁，能问出这种问题，说明她已经开始对这个世界产生浓厚的探究兴趣。如果以此为切入点进行深度探究，不仅可以探讨城市经济兴衰背后的规律，进而理解世界运行的一些基本规则，还可以理论指导实践，运用这些规则让自己生活得更好。

更重要的是，通过上述过程，她可以渐渐形成基本的跨学科思维方法并从中感受到乐趣，成为引领她成长最好的阶梯，并自发地将其运用于自己真正感兴趣的研究领域，未来她将从中受益无穷。

于是我和夫人商量，五一节快到了，反正也要带孩子出去玩，不如就此策划一场非正式的深度研学活动。从一开始，这场研学就被设计为基于问题的学习（Problem Based Learning，PbBL）。

上海这座城市本身也极有代表性，它的发展脉络不仅代表了上海自身，而且在一定程度上反映了全中国乃至全球的经济脉络，可以成为极好的研学样本。

站在全球史的角度上，上海所在的江南地区，不仅是中国最富庶的地区（没有之一），而且也代表了农业时代全球经济发展的最高成就，它成功的秘诀到底什么？

![](.evernote-content/FFB2376D-0F6E-4273-B9AA-4AD3815E715D/D0609F0D-3EC6-4898-ACF6-6C3A0052E137.jpg)

著名的李约瑟之问

![](.evernote-content/FFB2376D-0F6E-4273-B9AA-4AD3815E715D/918EB266-7472-45D2-BF3D-A7C2979FE618.jpg)

字面上问的是中国，其实发问对象就是江南。因为江南地区集中了那个时代中国最多的人才，产出了最多的科技成果，为什么农业时代取得如此高成就的江南，难以自发跨入工业时代？或者过去的优势其实恰恰就是阻止其迈入新时代的沉重枷锁？

更有意思的是，上海这这座城市本身，恰恰是完成了工业革命的欧洲，侵入中国并推动中国开始进行近现代化转型的过程中形成的。东西方的历史脉络在这里发生交汇，融合，它既是醉生梦死的十里洋场，也是中国近代工业发源地，近代革命运动的温床，新中国工业建设的基石，这一切进程在上海这座城市留下了怎样的痕迹，塑造了其与众不同的独特气质？今天的上海成为中国最重要的长三角经济区的龙头，国家四大科学城之一，未来它又将走向何方？

![](.evernote-content/FFB2376D-0F6E-4273-B9AA-4AD3815E715D/AE12A47C-4570-4124-978F-9749BFC6BE96.png)

本图出自《上海打造未来产业创新高地发展壮大未来产业集群行动方案》

于是，我把女儿问出的问题，拆解成如下四个问题，作为本次上海之行的研学主题：

1. 什么样的历史进程在上海留下了怎样的痕迹，塑造了其与众不同的独特气质？
2. 上海所在的江南地区，代表了农业时代全球经济发展的最高成就，它成功的秘诀到底是什么？
3. 为何江南难以自发跨入工业时代？过往优势为何构成阻止其迈入新时代的沉重枷锁？
4. **今天的上海成为中国最重要的长三角经济区的龙头，国家四大科学城之一，未来它又将走向何方？**

策  划

当然，短短几天的研学旅行不可能完全解答上述问题，甚至可能只是初步涉及到皮毛而已。但是如果就此给孩子心中种下探究的种子，初步品尝到探索的乐趣，形成探索型学习的习惯，那么对她的终身的学习成长一定大有裨益。

既然打算认真策划，本着“搂草打兔子”的原则，我俩分别通知了身边的朋友同事，组成了一个小小的亲子研学团。

我本身就对文史地理政治经济这些话题颇感兴趣，没事写一写公众号文章，如今又加上培养孩子这个目的，动力自然也是很足。但是真要策划起来，我发现还是面临巨大挑战：

1、由于是亲朋的孩子，年龄跨度很大，最小的孩子是我家二宝，不到七岁，最大的孩子则已经上了高中。要让这些孩子都能有兴趣参与，还能互动起来，那么门槛就要放到足够低，同时又要有足够的上限，让每个人都能找到自己感兴趣的点。

2、我平时写的文章虽然已经尽量向浅显易懂的方向努力，但是梳理一个区域的经济发展脉络，不是简简单单套“公民权力”、“小政府”、“自由市场”等几个名词就完事的，往往涉及多个学科，而且要打破学科之间长期存在的壁垒，建立跨学科的研究体系，这也是我的文章往往让成人读者都感到信息量很大，甚至有读者直呼看不懂。如何将其尽量简化，还要融入到旅行过程中，同时还能进行一些深度的讨论，也要花很多心思。

3、孩子平时上学写作业，难免对于学习这件事带有一定抵触心理。好不容易出去旅游，结果又布置一堆学习任务，恐怕会进一步引发对学习的反感。社会上很多研学项目，要么是学习和玩两张皮，要么就是打着研学的旗号去旅游，如何真正把学融入玩，让孩子乐在其中，真正实现“快乐教育”，这是困扰整个教育界的难题。

不过，我并不是第一个提出这个问题并给出解决方案的人，可以从前人的智慧中寻找解决方案。

![](.evernote-content/FFB2376D-0F6E-4273-B9AA-4AD3815E715D/EA0949BA-68F7-4D0B-9CBA-73A696285122.jpg)

瑞士教育家裴斯泰洛齐（1746--1827）

中国古人就说到“读万卷书，行万里路”，只有这样才能做到知行合一。瑞士教育家裴斯泰洛齐则早在两个世纪前更是进行了系统的教育实践。他早年的学生、后来成为知名教授的弗由伊曼回忆道：

“为了学习地理的基础知识，我们被带到户外，向伊佛东附近人烟稀疏的比由仑河谷步行。我们首先观察河谷的整体与各个部分，对这个河谷渐渐有了正确的认识。然后挖取河谷的粘土，把它包在一张大纸里面。回到学校，大家就站在一张大桌子边，把带回来的粘土捏造成河谷的模型。次日我们观察较高的地方，继续扩展我们的研究范围。最后，我们终于把整个伊佛东盆地研究完毕，把它做成鸟瞰的彩雕模型。然后，我们才开始学习地图。”这怎么可能不令学生印象深刻，并爱上学习？

两百年来，中外教育家对于如何实现更好的学习进行了无数研究，成果汗牛充栋。我通过反复筛选，从中挑选出三大法宝：

一是美国人提出的基于心流理论的挑战式学习（CBL，Challenge Based Learning）。游戏之所以让人沉迷，是因为它具有1.清晰的目标；2.即时反馈；3. 技能与挑战相匹配，能够引发心流，从而让人沉迷如此，欲罢不能。理论上说，学习完全可以和游戏一样引发心流，让人上瘾。这就是CBL的底层逻辑。

![](.evernote-content/FFB2376D-0F6E-4273-B9AA-4AD3815E715D/B02333F9-57A3-4882-BC83-FF1B40C71433.png)

二是欧洲人提出的调动全脑智能的现象式学习，从现象入手，强调整体性的学习方法，综合了来自不同学科的知识，学生们通过协作研究某个现实中的具体现象，其目的不是取代科目学习，而是将其置于更广阔的视角之下，发展学生解决问题的能力。该方法充分调动了人的三重脑（也可以理解为感知、认知、情绪与本能、逻辑推理、控制行动等大脑模式的多模态复合），充分调动多元智能不同类型个体的长处，并进行有效协作，解决实际问题，是最符合人类大脑运作原理的学习方法。

三是中国教育家陶行知的生活教育理念，生活即教育，社会即学校，教学做合一。“创造教育是以生活为教育，就是生活中才可求到教育。”只有在良好的生活氛围中，在生活中学习和创造，才能最大限度地发展学生的聪明才智。

![](.evernote-content/FFB2376D-0F6E-4273-B9AA-4AD3815E715D/9C94063E-4B39-4972-A49F-B6A21B821DE9.jpg)

我认为，以上三者分别是教育的“术法道”，挑战式学习把学习变成了打怪升级的游戏体验，现象式学习则告诉我们真正有效的学习要来自与自身息息相关的各种现象，形成跨学科解决问题的能力，而生活教育理念则是告诉我们，只有人拥抱生活，热爱生活，才能成为真正的学习者。

**![](.evernote-content/FFB2376D-0F6E-4273-B9AA-4AD3815E715D/F97C1371-FBCF-483E-BCCD-78B9461B068A.png)**

我要做的事情就简单了，通过旅程规划和活动设计，做出承接以上“术法道”的”器”——研学产品。

旅程规划主要是通过线上搜集信息，确定能够承载研学活动主线的场所。我将本次研学旅行定位为探秘上海经济发展的昨天，今天和明天，从中揭示从具体的事件和现象中探索背后隐藏的规律，并用它来解释当今的经济现象。

说到上海的过去，大部分第一反应是外滩和租界，但我觉得这是远远不够的，英国人当年为什么要把租界定在这里一定有更深层次的原因，我将目光放到了体现上海地区千年历史的朱家角，这里在明朝万历年间就成为一个典型的江南贸易大镇，英国人之所以选择上海外滩，因为这里跟他们熟悉的伦敦金融城实在是有太多相似之处（大河入海口前的河湾，背靠工商业发达的广阔平原）。

实际落到计划中，发现上海主城区和朱家角之间的交通实在是不太方便，反而是与上海相邻的苏州直接可以高铁互通。索性不如直接将第一站放在苏州。

相传苏州城最初由两千五百年前的名将伍子胥所建，是中国唯一一座城址没有变化的名城。苏州府原本就是东汉三国时期的吴郡，代表了大半个江南，上海这片区域原就属于苏州府之下的华亭县。坐拥太湖水系以及大运河，苏州成为明清时期江南的财富中心。上海在近代的真正发展，也是太平天国的战火烧到苏州，大批财富阶层和平民涌入上海租界的结果。

因此，苏州才是真正代表了上海的昨天。

关于元象知行：

![](.evernote-content/FFB2376D-0F6E-4273-B9AA-4AD3815E715D/047C5851-4D3C-4589-8314-804D1D3BCE4A.png)

**“元”“象”二字均出自《易经》。**“元”指万物初始与本源，“象”代表万物演化形成的复杂现象。****元象知行不仅是一个教育机构，更是一个梦想的摇篮。我们深知，现行的教育体系已难以满足孩子们在AI时代的需求。我们的孩子需要的不仅仅是知识的灌输，更是能力的培养、情感的激发和个性的发展。

在元象知行，我们致力于构建基于问题式学习和基于项目式学习相结合的场景化研究性学习体验，让孩子们在探索世界的同时，点燃他们内心的激情，发现学习的乐趣，培养终身受益的核心素养。

在“元象知行”，我们主要做如下两方面的工作：

**一方面，建立通用的“元认知”体系，这是一套基于复杂系统世界观的通用理论、方法和工具；**

**另一方面，接触千变万化的具体问题场景（被称为“象”），在场景中发现问题，分析问题，解决问题，持续加深对这些场景所涉及领域的理解，并基于这些理解实现对于“元认知”掌握的精进。**

**在元象知行没有“毕业”，因为创新永无止境，学习也永无止境，整个世界都是学习创新的课堂。我们相信每个孩子都有无限的潜能，我们的任务是激发这些潜能，让他们在创新的道路上不断前行。**
