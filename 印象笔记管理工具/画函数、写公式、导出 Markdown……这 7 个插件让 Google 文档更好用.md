画函数、写公式、导出 Markdown……这 7 个插件让 Google 文档更好用
==========================================

14小时前

[![](.evernote-content/E889691E-FBC1-4B09-8FBF-3CBF1EC430AD/D5A02589-C9C0-4810-BFF8-2F3B05B44E3F)](https://sspai.com/user/727446)

#### [i9NGbgNq](https://sspai.com/user/727446)

Google 文档一直是在线文档类工具中的佼佼者，编辑器能够自动保存文档，还能多人即时协作编辑。但是 Google 文档的排版功能稍弱，面对论文、报告这类讲究格式的写作文体时，就需要使用到 Word。

好在，Google 提供了插件商店，让第三方插件来提供那些 Google 文档尚不具备的新功能。这样一来，Google 文档虽然是轻量化的文档工具，但在也能完成很多 Word 这样的专业软件才能完成的工作。

本文将为大家介绍 7 个有关排版样式的 Google 文档插件，让你的文档变得美观好看。

Yob Graph Editor
----------------

要想文章看起美观，就要图文并茂。无论是演示操作步骤，还是表达数据信息，图表都是最直观的形式。图表也分简单和复杂，一般文档处理工具都支持插入像柱状图、折线图、饼图这类简单的图表。而函数图这种复杂的图表就需要其他工具来绘制。

![](.evernote-content/E889691E-FBC1-4B09-8FBF-3CBF1EC430AD/8BDBEE3D-CE49-41AC-8368-133886A9C662)

Yob Graph Editor

[Yob Graph Editor](https://chrome.google.com/webstore/detail/yob-graph-editor/doghjhjgnmiikbjphdcdeehhkfdembpf?utm_source=permalink) 是一款绘制函数图插件。尽管制作函数图会涉及坐标轴、单位、函数公式、数据，但 Yob Graph Editor 让这些都不再是问题。在 Google 文档启用插件后，你可以在侧边栏里调整函数图相关的参数，然后点击「Insert」就能完成制作函数图。

Table Formatter
---------------

[Table Formatter](https://chrome.google.com/webstore/detail/table-formatter/nhnodgfnhajblionllcpmgndadjgdeee?utm_source=permalink) 是一款表格样式美化插件，你可以用它在 Google 文档中选择各种表格配色，让表格看起来不那么单调。插件默认的配色模版有数十种，还支持自定义模版。在自定义模版中，你可以设置表格的每一条边框、每一排、每一栏的背景颜色和字体颜色，满足你的各种需求。

![](.evernote-content/E889691E-FBC1-4B09-8FBF-3CBF1EC430AD/571180CF-C26C-4440-866B-FD394349DF3A)

Table Formatter

Auto-Latex Equations
--------------------

虽然 Google 文档也能插入公式，但每次都要反反复复地输入运算符。[Auto-Latex Equations](https://chrome.google.com/webstore/detail/auto-latex-equations/iaainhiejkciadlhlodaajgbffkebdog?utm_source=permalink) 的作用就是将使用了 MathJax 和 LaTeX 语法的文字转换为数学公式。使用 Auto-Latex Equations 很简单，将 MathJax 和 LaTeX 语法的文字复制到文档以后，在侧边栏中点击「Render Equations」 即可完成转换。

![](.evernote-content/E889691E-FBC1-4B09-8FBF-3CBF1EC430AD/732D95D0-9C53-440C-AA80-E493F0991E82)

Auto-Latex Equations

Doc Tools
---------

英文写作中时常面临着需要统一格式的问题，比如标题的每一个单词字母要大写，文章出现的「ten」都要统一成数字「10」等等。如果你需要进行这样的操作，那么 [Doc Tools](https://chrome.google.com/webstore/detail/doc-tools/gjbhiglgnpikoggompgjflkoeacenbdo?utm_source=permalink) 就非常适合。它里面预设了多个常用的文本操作，帮助你可以快速地将英文字母和数字进行统一格式。

![](.evernote-content/E889691E-FBC1-4B09-8FBF-3CBF1EC430AD/52ED3141-C2E8-4BE8-9810-DB396CADF2FD)

Docs Tools

MindMeister
-----------

[MindMeister](https://chrome.google.com/webstore/detail/mindmeister/gknkengmjmcdgefkhdindnnljbhadikn?utm_source=permalink) 是一款思维导图转换插件，它能够把无序或有序列表转换成思维导图。转换后的脑图会根据列表的层级进行排列，以更直观地方式进行展示。

![](.evernote-content/E889691E-FBC1-4B09-8FBF-3CBF1EC430AD/7E4D63BA-5F2D-4AC4-B061-134924A8F47B)

MindMeister

Footnote Style
--------------

学术写作都需要进行引用，因此脚注的格式就非常重要。[Footnote Style](https://chrome.google.com/webstore/detail/footnote-style/hlcpimnopaolddjgmgjeibnhgkgbnbai?utm_source=permalink) 是一款脚注样式的插件，它能为文档插入的脚注提供统一的样式，让你的作业和论文看起来更加整洁好看。

![](.evernote-content/E889691E-FBC1-4B09-8FBF-3CBF1EC430AD/3579BA7C-871F-4D6D-869F-D459F1F58530)

Footnote Style

GD2md-html
----------

[GD2md-html](https://chrome.google.com/webstore/detail/gd2md-html/igffnbdfnodiaphfmfaiiaegmoljbghf?utm_source=permalink) 能把 Google 文档转换成 Markdown 或者 HTML 格式。由于 Google 文档采用富文本编辑器，很多情况下把内容直接复制到其他编辑器里会出现板式问题。这时你就可以使用 GD2md-html 转换 Markdown 或者 HTML 格式的文本，从侧边栏的插件里复制粘贴到其他编辑器中。

![](.evernote-content/E889691E-FBC1-4B09-8FBF-3CBF1EC430AD/B17ECD08-8407-44EC-9E7D-9D74B561378F)
