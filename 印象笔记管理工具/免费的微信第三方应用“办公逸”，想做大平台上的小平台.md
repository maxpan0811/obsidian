# 免费的微信第三方应用“办公逸”，想做大平台上的小平台

---

![](.evernote-content/2258BB64-BFD2-4D59-8A92-B3E9232EE9F9/E6DB4E03-C776-4EC3-99B7-27216B7D2D86.png)

统计考勤是企业“大姨妈”式会呼吸的痛。每到月底，人事就要吭哧吭哧的从考勤机里拷出数据，对员工出勤情况进行统计然后发工资。中间还要考虑谁请假了，谁出差了，谁是调休，谁有加班...

微信企业号的第三方应用[“办公逸”](http://www.bangongyi.com/)初步想解决的就是这样一个问题，他们把企业所有与考勤相关的服务集中在“办公逸”移动办公（OA）套件中。企业管理员可以在套件的帮助下通过微信企业号对员工的日常出勤、请假、加班、调休、出差、外出等活动进行统一管理，员工也能在微信上查看到自己每月的出勤状况。除了考勤服务，“办公逸”的移动办公（OA）套件还配有企业办公常用的“审批”、“红包”、“内部通讯录”等功能。之后“办公逸”也会自然加入薪酬管理、工资发放等服务。

![](.evernote-content/2258BB64-BFD2-4D59-8A92-B3E9232EE9F9/E6D3A3AD-DCF4-48AA-992B-D4DCC8AAD8BE.png)

“统计考勤”只是企业运转中的众多难题之一。除了“移动办公”和前两天开放的“企业消息”，目前微信第三方应用平台上还有“客户关系”、“文化建设”、“团队协同”、“人力资源”几个大的板块。“办公逸”在“客户关系”、“文化建设”、“团队协同”板块都推出了相应的套件。与“办公逸”玩法相同的还有“企微云平台”、“微加”、“iWork365”等公司，随着微信企业号逐渐发展，后期会有更多玩家入场。

![](.evernote-content/2258BB64-BFD2-4D59-8A92-B3E9232EE9F9/8C4AF7DE-CE0F-4A54-AD1B-590FD95900C2.png)

守着同一个入口，企业号第三方之间的竞争就自然变成了淘宝上商家们的竞争。现实的来说，早期对于大多数的企业，“办公逸”、“企微云平台”、“微加”们之间的区别一是微信的推荐位、二是价格、三才是产品。

“办公逸”目前是微信企业号第三方应用中排在第二的服务商，第一是“企微云平台”。据创始人刘建华提供的数据，7月6日微信第三方应用平台上线后，给他们带来了 3000 多家企业用户。

价格上“办公逸”选择了完全免费。这个决定是创始人刘建华在做了 10 多年软件生意后做出的，说到这个事情他有些无奈

> “中国的中小企业没有为软件付费的习惯。”

产品上，刘建华认为，很多传统厂商转做移动办公产品时最容易出现的状况就是直接把 PC 端的产品打包到移动端上。“办公逸”这方面选择了完全基于微信使用习惯的产品开发，例如员工打卡是用微信“摇一摇”实现，流程审批时员工直接“@”上级。

商业模式上，“办公逸”想做的是一个微信大平台里的小平台，作为企业流量的入口。后端接一些如机票预订、名片印刷、文具及其他可能的企业采购供应商，与供应商分成。这套商业模式成立的前提是，“办公逸”需要获取足够多的企业用户，但首页推荐位的美好只是暂时的。“办公逸”目前在招募加盟商，按他们的计划加盟商的收入会包括两部分，一是辅导企业上手使用应用的服务费用，二是在“办公逸”开放的 API 上针对企业需求定制开发获取收入。

创始人刘建华曾创立“[数字星空](http://www.pcstars.com/)”，公开资料[显示](http://www.baike.com/wiki/%E6%95%B0%E5%AD%97%E6%98%9F%E7%A9%BA)，该公司是卡巴斯基安全软件个人版中国区独家发行商。
