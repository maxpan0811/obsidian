# 【独家】印象笔记成为微信第一家系统级 API 使用者

---

[【独家】印象笔记成为微信第一家系统级 API 使用者](http://www.ifanr.com/245180)
---------------------------------------------------------

作者：黄龙中

[![](.evernote-content/2CD33BC3-AF0D-4BC9-AFB6-3EA7DE556109/25847A36-F565-4A18-94B4-9F2FA89725C9.jpg)](http://www.ifanr.com/245180/yinxiangbiji-weixin)

微信刚刚在 2 月 5 日放出 4.5 版本的更新，吸引众人眼球。除了在 App Store 里描述的新功能——语音群聊、群聊二维码、搜索聊天记录、迁移聊天记录、语音提醒公众号——外，微信在手机客户端的 About 页面和[网页版](http://weixin.qq.com/cgi-bin/readtemplate?uin=&amp;stype=&amp;promote=&amp;fr=&amp;lang=zh_CN&amp;ADTAG=&amp;check=false&amp;nav=faq&amp;t=weixin_faq_ios_45)新功能介绍页，讲到更多新功能，有些是很实用的：

* 撤消语音消息
* 在微信聊天界面自动抓取截图内容
* 位置信息可以直接用来导航
* 发送多张图片

需要特别关注的是，腾讯生活电商部的“微生活会员卡”账号仍然没有获得底层绑定的权限；相反，微信继续在产品功能上创新，推出“语音提醒”工具——据搜狗 CEO 王小川分析这是科大讯飞的技术。这意味着张小龙的微信团队顶住内部各方势力觊觎，在马化腾的支持下，不急于变现，还能在功能上不断推陈出新。这实在非常难得。

除了向自家的“语音提醒”工具开放底层 API 外，据**爱范儿**获得的独家消息，微信也向第三方应用开发者首次开放了系统底层 API，这位幸运儿是印象笔记，Evernote 中国版笔记应用。印象笔记也是目前唯一一家可以调用微信系统 API 的第三方开发者。显然，微信走出这一步是极其谨慎的。

据**爱范儿**了解到的独家消息，此次印象笔记与微信沟通的时间“并不长”，印象笔记方面由中国区总经理谷懿（@[谷艾米](http://weibo.com/201212316)）亲自与微信方面沟通。由于印象笔记的工具属性，及它与腾讯长期的合作，印象笔记得以成为第一家微信系统级 API 使用者。据印象笔记内部人士介绍，从 2012 年 5 月印象笔记进入中国以来，印象笔记团队便与腾讯保持密切合作，“最初是 Q+，后来是 QQ 手机浏览器，到现在的微信”。对于印象笔记与与 QQ 手机浏览器的合作，这位人士介绍 QQ 浏览器是印象笔记去年 12 月 1 日发布百宝箱后的入驻的第一批合作伙伴（爱范儿目前也是百宝箱一款应用，支持在 App 里将内容存储到印象笔记）。

我详细体验了微信平台的印象笔记。除了一些小瑕疵外，整体来说印象笔记保持了其以往的优秀品质。不过，比较遗憾的一点是，目前微信平台的印象笔记只支持国内的“印象笔记”账号，不支持 Evernote 国际账号的登录授权。以下为微信版印象笔记上手初体验。

**1. 如何激活微信平台的印象笔记？**

添加“我的印象笔记”（微信号为“myyxbj”），点击授权链接，输入印象笔记账号，授权。需要注意的，一是目前仅支持“印象笔记”账号，这对于国内早期习惯使用 Evernote 国际账号的用户可能不太方便；二是根据印象笔记 API 协议，印象笔记账号最长只能授权一年，另外可选 30 天、1 周、1 天。

[![](.evernote-content/2CD33BC3-AF0D-4BC9-AFB6-3EA7DE556109/7843F356-7726-43AC-9EFE-22E669B5DA08.jpg)](http://www.ifanr.com/245180/yxbj-weixin-api)

**2. 如何取消印象笔记对微信的授权？**

对“我的印象笔记”账号取消关注即可。如果需要重新授权，重新关注“我的印象笔记”（myyxbj），点击跳往授权页面即可。在重新授权页面，可以重新修改授权时间。

[![](.evernote-content/2CD33BC3-AF0D-4BC9-AFB6-3EA7DE556109/EB6075EA-2D9F-4EE4-92BA-70370DC8C712.jpg)](http://www.ifanr.com/245180/yxbj-weixin-api-2)

**3. 微信平台的印象笔记能做什么？**

微信平台印象笔记能做的事情非常有限：创建笔记、笔记本和标签，列出笔记本和标签。在阅读环境下，这两个功能也足够了。其他诸如更新笔记、删除笔记、账户信息等，在印象笔记 App 或 Web 端完成即可。

[![](.evernote-content/2CD33BC3-AF0D-4BC9-AFB6-3EA7DE556109/72039FDD-459C-4128-86C7-71602DA1A7A1.jpg)](http://www.ifanr.com/245180/yxbj-weixin-api-3)

**4. 用什么方式实现“微信版印象笔记”？**

向“我的印象笔记”发起对话，就将内容保存到印象笔记了。这里有两种方式：一是在与“我的印象笔记”聊天页面，直接输入文字、图片、语音、位置信息、视频等内容；二是在别处长按需要保存的内容，出现“复制”“转发”“删除”“公众号”后，点击“公众号”，调出印象笔记图标，进行保存。后者也是跳转到给“我的印象笔记”发信息的界面。

[![](.evernote-content/2CD33BC3-AF0D-4BC9-AFB6-3EA7DE556109/2F603FB8-CB9A-45F5-A5A1-4E0E19AC1562.jpg)](http://www.ifanr.com/245180/yxbj-weixin-api-4)

**5. 有哪些内容不可以使用微信平台的印象笔记？**

两种情况不能使用微信平台印象笔记：一是公众账号推送的专题内容（非“图文内容”），二是富媒体内容，如微信语音提醒中的小卡片。两种情况下，长按需要保存的内容，无法调出 API——只有“删除”选项。

[![](.evernote-content/2CD33BC3-AF0D-4BC9-AFB6-3EA7DE556109/D0B5D5FB-77E9-4162-8CDB-E51D7AB15202.jpg)](http://www.ifanr.com/245180/yxbj-weixin-api-5)

**6. 使用印象笔记保存微信内容后，它在网页端好不好用？**

网页端存在比较大的一个问题是：格式没了，文字挤成密密麻麻一片，甚至没有换行。另外，印象笔记在微信保存图文信息的时候，只能存取一二级页面的内容。比如点击打开图文信息后，第二层概述文字，需要点击第三次才能真正打开原文；那么，对不起，印象笔记寻址只寻到第二层，三级页面（也是公众账号自己的页面）不能被抓取。

[![](.evernote-content/2CD33BC3-AF0D-4BC9-AFB6-3EA7DE556109/CFC8AE4C-D492-42F2-869E-8C0BAD8EDF63.png)](http://www.ifanr.com/245180/yxbj-weixin-api-6)
