[![](.evernote-content/2243A0BD-06F8-4E1E-B5FE-B4E041CEA175/945F269B-C75C-4EB2-AB02-262AECDFA345)](https://sspai.com/user/706656)

#### [沨沄极客](https://sspai.com/user/706656)

这几天有一位读者问了我一个问题：「Windows 上有没有类似 macOS 的 [Yoink](https://eternalstorms.at/yoink/) 这种应用？」可惜我找了一圈并没有发现 Windows 下的文件暂存软件。

为此我找了一个替代方案，可以解决文件暂存的需求。

![](.evernote-content/2243A0BD-06F8-4E1E-B5FE-B4E041CEA175/53302FFE-5D21-4508-8106-FE26719CA3EA.gif)

Windows 10 下方的任务栏默认包含几个主要部分——开始按钮、搜索框、软件栏、状态栏、时间。**在软件栏和状态栏之间，我们还可以自定义添加「工具栏」。**今天要讲的文件暂存就是靠工具栏实现的。

文件暂存软件的本质就是一个放置文件的地方，而工具栏的本质是将一个文件夹固定在菜单栏中，两者在形式上是类似的。

首先在「软件栏」和「状态栏」之间的任意空白处右键，「工具栏 → 新建工具栏」，然后选择任意一个文件夹作为工具栏，工具栏就会被添加到任务栏。

![](.evernote-content/2243A0BD-06F8-4E1E-B5FE-B4E041CEA175/55006AC9-0A26-4678-8273-7841C9437E28.png)

接着我们对这个工具栏做一些调整，让它更像一个暂存文件的地方。「鼠标右键 → 取消勾选锁定所有任务栏」，「鼠标右键 → 查看 → 小图标」，并取消显示标题、取消显示文本。

![](.evernote-content/2243A0BD-06F8-4E1E-B5FE-B4E041CEA175/57B6ED2A-C17A-4C58-BD5B-1AFC6EA6845B.png)

这样，一个临时存放文件的地方就 OK 了。实现的效果和 Yoink 类似，你可以把一些文件拖放进这个文件暂存箱。

使用起来也非常方便，拖放时按住 Shift 移动文件，按住 Ctrl 复制文件。右键图标时是正常的右键菜单目录，可以打开文件所在位置，或者对文件进行更多的操作。

不想用了就直接关闭工具栏，不会占用任何系统资源，真正做到「文件暂存」。
