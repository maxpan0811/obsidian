# 美化右键菜单 Win 10第二个更新10532公布

### [美化右键菜单 Win 10第二个更新10532公布](http://fullrss.net/r/http/tech.feng.com/2015-08-28/Beautify-the-right-click-menu-Win-10-10532-announced-a-second-update_622901.shtml)

![](.evernote-content/4667F0E8-0F11-4129-BAD9-1AF19354808C/2D8F50DC-C7E1-485D-B30B-40839E52910A.jpg)

  
　　参与 Windows 10 会员计划的用户，现在有新玩意可以捣鼓了。威锋网 8 月 28 日消息，日前，微软正式公布了 Windows 10 新版本更新 BUILD 10532。已经开启“Fast”快速获取会员版本更新的用户，目前已经可以收到来自微软的更新推送。  
  
　　微软在最新的 BUILD 10532 版本中进行了一些小的调整，整体相比之前看起来不会有太大区别，但视觉上确实有一些很酷的变化，目的是为了吸引更多潜在的用户。  
  
　　其中，第一个比较酷的变化就是“右键菜单（上下文菜单）”，现在提供了与系统“现代风格”外观更相似的设计，显得更加一致了，而不像之前在某些区域点击鼠标右键时，呼出的菜单仍然是老旧的设计。虽然不是非常精美的设计，但看它的确是 Windows 10 独有的风格。更重要的是，这是微软耐心听取 Windows 测试用户心声的表现，此前很多用户在反馈应用中抱怨太多地方的右键菜单太丑。另外，微软表示 Windows 10 中的黑白主题的界面统一性也得到了增强，但更多优化工作还在继续。  
  

![](.evernote-content/4667F0E8-0F11-4129-BAD9-1AF19354808C/3EBF8A2E-9AFD-4843-84B6-A1989BCF14C8.png)

  
　　另一个变化不是特别显著，而且这个更新只与参与 Windows 会员计划的用户有关。之前 Windows 反馈应用程序只能直接向微软提出建议和反馈，并查看投票热门程度如何。现在，全新的 Windows 反馈应用程序中，新增加了一个分享按钮，方便用户将自己或他人反馈的问题分享到 Facebook 或 Twitter 等社交平台上，形成更激烈的讨论。  
  
　　反馈应用程序的内容支持网络分享对微软而言相当有意义，因为除了 Windows 测试会员之外，更多的普通用户也可以团结到一起讨论新的变化，或提供未来新版的建议，更广泛的引起微软的注意。  
  

![](.evernote-content/4667F0E8-0F11-4129-BAD9-1AF19354808C/6E03A1D0-B6B0-418F-8B98-5A68A46000F4.png)

  
　　其他更显著的变化微软暂时没有提及，有可能只是非常小的改进，比如 BUG 修复，或者一些很小的细节调整。如今，微软对每一个更新的描述越来越少了，甚至表示未来只要不是大更新就不会提供更新日志。我们不清楚为何微软要这么做，有可能是更新的内容不足以长篇大论，但不提供更新日志令不少用户感到不满，他们希望知晓微软会对新系统做出了哪些变化。  
  
　　最后需要说明的是，该版本只针对开启快速更新的 Windows insider 会员计划用户推送，用于测试和收集反馈、建议，系统存可能会存在很多不稳定因素，甚至影响系统性能造成错误，比如之前 64 位谷歌 Chrome 浏览器崩溃的问题还未解决，Windows Hello 暂时无法用于登陆等等。   
　  
　　至于普通用户何时获得 BUILD 10532 更新微软并未透露，若你也想体验最新更新，可以自行到“更新和安全”中加入会员计划，但一定要三思而后行。
