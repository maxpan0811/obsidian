| |  | | --- | | *本帖最后由 airjordanqhj 于 2014-5-10 22:10 编辑*   请录取上实的爸爸妈妈们注意一下，我们打算开个微信圈给今年入学的宝贝们，方便以后交流学习。如果有兴趣可以短我，谢谢。   今早看到儿子上实的录取通知时，激动的眼泪都出来了，开贴记录下过程，希望给有需要的爸爸妈妈点帮助，同时为明天所有的孩子加油!   上实第一步还是一如既往的网筛，看了以前前辈的攻略，我比较相信是电脑随机抽的。但我还是很认真的填写了报名表，个人简介和特长爱好也在报名前已写好。报名那天填一填就好。主要还是写了广泛的兴趣和强烈的求知欲，并且懂事礼貌乖巧。因为我家儿子其实真没什么出彩，从小一直玩到大班，什么都没学，去年11月我才惊觉要考小学了，可他连10以内加减都要用手扳。爸爸很认真的接下了这教学任务，真的很感谢爸爸这半年的付出，考进上实他功不可没啊。网投后其实也没在意，因为觉得还是靠运气，所以从没有准备上实的任何考题。   28号出的网筛结果，3300选1200，很幸运的过了，才开始重视，上网当了资料，1号那天在家帮他复习了下，学习了小键盘的运用，做了3遍，还想留3遍2号考试前做，谁知2号网都上不去，还好算教过了，希望大家吸取这个教训。   5月2号下午机考，考前关照了仔细做题，不要乱猜，不会的宁愿跳过。但他平时动作很慢，又关照要快点，其实也矛盾。出来告诉我做到106题e的两个组合，听到别的小朋友说都是150几题f的三个组合，还是有点担心的，但想他尽力了也没多问，带他去吃饭买玩具去了。   6号出结果，通过，那一刻很兴奋，觉得我们离上实又进了一步。马上开始了解这个学校还有历届面试题，又听说是按机考为主，所以也不知道该如何准备。就关照了礼貌问好，声音响亮就去了。   8号下午面试，160个小朋友，我们a组20人，我排队检查材料看证书，看着别人一叠叠的证书我就一张这次综测的一等奖，也不知道会不会对他有影响。然后小朋友去面试，家长听了校长讲座。校长真的很和蔼，讲话很中肯，对上实的感觉一下好了很多，觉得从浦西搬家也值了。谁知儿子出来说考的不好，又一滴汗。大概问了下题目：   1 看动画回答问题，好像就是大家说的猴子过生日的故事。去了另外一个教室回答问题，问了山谷是什么，小松鼠送什么礼物，小白兔送什么礼物，小乌龟送什么没答出来，小猴如何救小猪的，如果你是小猴怎么办。   2 跟老师做操，原地踏步，向左向右转。   3 拼雪花片，主题是拼个开心的东西。儿子拼了个老奶奶坐在车里看外面风景很开心，我实在很诧异他是怎么拼的，因为这是他第一次玩雪花片。   4 看电脑学习线段直线射线，然后做了3道题，我估计他错了1道。   5最后游戏，夹球走，还掉了一次，但说马上捡回来了继续。然后拍球原地转，再运球。   面试估计就这些 可能不全 孩子也记不清了。大家有什么的问题可以跟帖咨询，希望对大家有帮助。   也祝愿明天所有的小朋友们一切顺利， 加油! | |
