# 性感而专注，Inbox by Gmail 第三方客户端：Boxy for Mac

---

邮箱是我们生活中最重要的工具之一，不管是求职、沟通或是注册一个网站的会员等等，都需要用到邮箱。可是，在 OS X 平台上的各种邮件应用，不是速度太慢就是功能不足，唯一两款口碑颇为不错的邮件应用 [Sparrow](https://www.gmail.com/intl/en_us/mail/help/sparrow.html) 和 [Mailbox](https://www.mailboxapp.com) 还不凑巧，一前一后都与消费者说再见。这时侯，找到一款界面稍微过得去并且使用趁手的邮件应用，成了很多人着急的事情。

少数派前段时间刚做了一个比较全面的 [邮件 App 专题](http://sspai.com/32229)，今天要介绍的是其中一款，也是最新的一款：Boxy。

如果你跟我一样，选择将自己主要的帐号以及邮件往来放在了 Gmail 上面，并且是 Inbox 邮箱的重度用户，那非常建议你试试 Boxy，这款 OS X 平台上的第三方 Inbox by Gmail 客户端。它界面风格的高度还原、不错的性能，是我很快喜欢上它的理由。

更有效率地使用 Inbox，Boxy 是一种方法
------------------------

### 1. 键盘党、效率党福音：快捷键支持

Boxy 完整支持了 Inbox Web 端的快捷键操作。和很多本地应用一样，如果你熟悉 Mac 上的基本快捷键，那么你能很快熟悉 Boxy 的快捷键。⌘ ＋ N 新建邮件，⌘ ＋ F 检索邮件，⌘ ＋ 1、2、3 能在收件箱、延迟邮件以及已完成邮件三个功能页间快速切换。

![](.evernote-content/F1624CAD-3374-40AD-8F16-FB6373B972DD/3AC5D549-944E-4A34-AD5A-3445C291663D.jpg)

### 2. 有邮件了？放心，不让你错过一封邮件

Boxy 同时支持 OS X 系统通知。除了接收新邮件时的横幅，Boxy 同样会常驻在你的 Menubar 和 Dock 中。

![](.evernote-content/F1624CAD-3374-40AD-8F16-FB6373B972DD/8A93ACC0-987D-495B-A7BF-B1FBDC33B14A.jpg)

![](.evernote-content/F1624CAD-3374-40AD-8F16-FB6373B972DD/6CAA259D-957B-4471-B752-CA7C8756D74C.jpg)

### 3. 阅读模式：长邮件，更舒适

也许你经常使用 Safari 的阅读模式来浏览网页，不过如果阅读的是一封长邮件，一般也只能跳转到网页查看。Boxy 在应用内支持将邮件使用阅读模式浏览，没有了其它元素的干扰，舒适阅读的同时也提高了效率。

![](.evernote-content/F1624CAD-3374-40AD-8F16-FB6373B972DD/400FD1F9-C3B6-476D-8ED4-68A7649AD7E2.jpg)

与功能同样优秀的，还有设计
-------------

Boxy 虽然是由 Web 应用封装而来，在细节方面还是能看出开发者的用心程度的。

作为一款邮箱客户端，Boxy 在高度还原了 Inbox By Gmail 整体整体界面的同时，还去除了 Web 右上角的环聊、谷歌快捷方式以及谷歌通知等按键，减少了不必要的元素干扰。

而对于侧边栏的邮箱功能区，在收起时，Boxy 会以小图标的形式展示它们，展开以后才会有像 Web 应用一样的具体文字说明。

![](.evernote-content/F1624CAD-3374-40AD-8F16-FB6373B972DD/E3347921-DAA2-419E-933B-88D710EE0BDF.jpg)

![](.evernote-content/F1624CAD-3374-40AD-8F16-FB6373B972DD/769C9D4C-1B1C-4B25-9DD6-B8E0BBD9C549.jpg)

![](.evernote-content/F1624CAD-3374-40AD-8F16-FB6373B972DD/9985DA2F-B23A-4DD7-AAC6-DA027A003A31.jpg)

![](.evernote-content/F1624CAD-3374-40AD-8F16-FB6373B972DD/A3BE1AA2-C972-47C2-92A7-E10AABB95B1B.jpg)

在 Menubar 上，Boxy 会以淡淡的浅蓝色与黑色区分，告诉你有未读邮件，如果能够加入未读数量显示就更好了。

![](.evernote-content/F1624CAD-3374-40AD-8F16-FB6373B972DD/635F893E-A892-4795-AB80-5E54337C8D14.jpg)

![](.evernote-content/F1624CAD-3374-40AD-8F16-FB6373B972DD/E24C34B7-2EA5-48CF-996E-5AC31D2C2364.jpg)

Boxy 目前也有夜间模式，不过只能让侧边栏的颜色改变。

![](.evernote-content/F1624CAD-3374-40AD-8F16-FB6373B972DD/35D775D6-050C-49A8-A51B-F8CE545B4BA8.jpg)

小结
--

可以说，专注、简洁与实用，是 Boxy 能在众多邮件应用中脱颖而出的一大关键。反过来，专注也算是它的缺点之一。大多国内用户都有使用多个邮箱的需求，那这时 Boxy 可能就不是一个好的选择。

对于非重度用户，Boxy 的售价也会让人望而却步。如果你觉得 Boxy 售价稍高而又想在 OS X 平台上使用类似应用的话，使用 Fluid 封装一个 Web App 倒是一个不错的选择，具体可以参考之前我派对于 Fluid 的 [相关文章](http://sspai.com/29160) 进行了解。

目前你可以在 [Mac App Store](https://itunes.apple.com/cn/app/boxy.-inbox-by-gmail-email/id1053031090?l=it&ls=1&mt=12&uo=4&at=10lJSw) 下载 Boxy，售价 30 元。
