![](.evernote-content/7CA30713-4F5E-4421-B727-7694E57A806F/2AF160AC-9B35-44D1-A229-B7A8ACF6603D)

Windows 平台的 Pop Clip，可以自定义的划词「加速器」：Pantherbar

Internet Explorer 时代过来的朋友也许对「加速器」这个概念也许还有印象，在 Internet Explorer 8 中，微软为浏览器右键菜单引入了一项新功能，当我们选中网页中的文本或图像后，可以直接通过与浏览器右键菜单整合的各种「加速器」完成不同的快速操作，比如搜索、在地图上查看选中地址等等。

![](.evernote-content/7CA30713-4F5E-4421-B727-7694E57A806F/5A12B4AE-24FB-4B7C-9C1D-976A12023C50.jpg)

Internet Explorer 的「加速器」整合

而后历史的车轮将 Internet Explorer 无情碾过，这些在 PC 端也能给我们「一步（One Step）」般体验的小功能也很快被我们忘记了。但这并不意味着对类似功能的需求并不存在 —— 对我来说，在给导师或者国外的同事发邮件时灵感迸发，写了几句话以后才发现火速敲下的文字全是大写！接下来每次按下退格键都像是对刚刚几乎进入的心流状态的一次嘲讽。

![](.evernote-content/7CA30713-4F5E-4421-B727-7694E57A806F/A4D555F1-74C9-42F8-A4F8-D9C3B1F57735.png)

一键转换为小写

而如果有了 Pantherbar，这种续上灵感火花的时刻就只需简单地选中、然后选择 Lowercase 按钮。接下来我们就可以继续创作了。

当然，选词、操作的需要不只会出现在这种极端的应用场景中：搜索关键字相关的资讯、查找类似的高分辨率图片、遇到不知道释义的生词、收到需要放进待办的任务等等。你可以将 Pantherbar 看作是 macOS 平台的 [Pop Clip](https://sspai.com/post/25483)、Android 平台的 [文本工具条](https://sspai.com/post/55311)、Win98 时代的划词翻译、Internet Explorer 8「加速器」的全局版…… 那些往往需要我们进行复制粘贴、在不同应用间手动跳转的场景，我们都可以选择 Pantherbar 来轻松化解。

![](.evernote-content/7CA30713-4F5E-4421-B727-7694E57A806F/687ADEF7-D975-4571-8475-D73D457318D8.png)

Pantherbar 的自定义选项

Pantherbar 拥有多种配色可选且支持自定义，开启后在任意界面中选中文字都可以直接弹出工具条（当然应用内也提供了「排除项」设置）；除此之外，我们还可以手动调整 Pantherbar 文本选择工具条的出现位置。

![](.evernote-content/7CA30713-4F5E-4421-B727-7694E57A806F/FD339DCF-7ABE-48CE-ACED-A70209E7644F.png)

Pantherbar 的样式自定义（限高级版）

除了基本的复制、剪切、粘贴、搜索和分享外，Pantherbar 提供了在线下载和手动制作两种方式来帮助我们进行自定义。其中在线下载仓库中包含了搜图、划词翻译等常见划词扩展，直接双击下载而来的 `.pbar` 文件即可进行安装和使用。

以英文文献阅读为例，Pantherbar 在线扩展仓库提供的翻译扩展包括 QTranslate、Bing Translator、Google Translate、Urban Dictionary 等在内的多种划词翻译服务，下载安装后便能直接划词并调用对应的服务进行词语释义和句子翻译。

![](.evernote-content/7CA30713-4F5E-4421-B727-7694E57A806F/F087B80F-739D-4F6E-8423-FF44510F8CC1.png)

你也可以将选项从文字变成图标

除此之外在线仓库中还有一些用于其他场景的扩展，我在用的包括：

* Wox：没错，它可以和 Wox 联动，在你的 Windows 里面搜索你想要的内容
* Run in Powershell/Run in Cmd：爬贴的时候会用到
* Say：随时随地享受如 Edge 般的文本朗读体验
* Taobao/JD：看到心仪的东西想了解下价格？只需一步即可拔草

还有很多非常实用的拓展，比如把链接转为短链、直接在 Google Image 里找图、在印象笔记 / Word 中创建一个相应的新页面、去掉空格（并拷贝）等等。各位可以在下载安装后先去 Pantherbar 的在线扩展仓库进行一番探索。

![](.evernote-content/7CA30713-4F5E-4421-B727-7694E57A806F/9F4290C3-3DAC-4B9E-9B3D-CC14B026B15B.png)

创建 URL 扩展

另一方面，你也参考 Pantherbar 给出的语法示例自行创造适合自己的划词拓展，如在 B 站搜索相关视频、在少数派查看相关文章等。

![](.evernote-content/7CA30713-4F5E-4421-B727-7694E57A806F/1306FA11-C406-41DD-8B13-35295C77BF75.png)
