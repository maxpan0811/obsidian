# Detexify，在触控板上快速手写 LaTeX 符号 | App+1

---

使用 LaTeX 工作和学习的朋友一定经历过这样的烦恼：当你试图通过 LaTeX 来输入公式的时候，总是会想不起一些符号所对应的代码，如果到 [symbols-a4.pdf](http://www.ctan.org/tex-archive/info/symbols/comprehensive/) 文档中去寻找， 也会浪费大量的时间。更糟糕的是，如果你连这个符号叫什么都不知道，那么你不但在文档中找不到你想要的代码，就算是用 Google 也会束手无策，因为你根本不知道用什么关键词来搜索。

其实在 iOS 设备上我们有一个非常方便的解决办法，那就是使用 [MathPad](https://itunes.apple.com/au/app/mathpad/id435087400?mt=8) 这款 App。你只需要手写输入数学公式，MathPad 就会把你手写的内容直接识别出标准的公式，并且可以在导出选项中选择 LaTeX，这样你就可以看到你手写的数学公式所对应的所有代码了。

但是，绝大多数的人都会选择在桌面平台上来写 LaTeX，那么对于 macOS 用户，又该如何解决这个痛点呢？今天我就向大家介绍一款简单好用的 App，叫做 Detexify，可以帮助你在 Mac 上识别你的手写符号，并输出相应的 LaTeX 代码。

Detexify 只有非常简单的一个界面，左边是手写板，右边是代码候选区。你只要在左边的区域通过鼠标指针或者触控板画出一个符号，右边的区域就会识别出符号相对应的代码，并且根据相似度排列出近似的选项。

![](.evernote-content/CE4AAB73-CA3E-4FDE-A339-93295CBD37DD/12D4D4DC-4387-45EB-BF87-7626EFE862B6.gif)

在右边的候选区选择你想要的代码后，它就会自动拷贝到你的剪贴板中。但是，你需要通过付费购买软件来消除粘贴文本中提示购买的信息。

![](.evernote-content/CE4AAB73-CA3E-4FDE-A339-93295CBD37DD/2D69BF3E-174E-4AC2-8983-318045FFB930.png)

Detexify 除了操作简单和识别精确，它还可以通过一系列快捷键来提高使用效率。首先，它可以通过一个自定义的快捷键来实现全局呼出主界面；其次，它还可以使用按键 C 来清除手写板，使用 esc 来退出主界面。这样一来，你不需要鼠标，就可以完成所有的操作。

如果看完我前面的介绍后，你还有不清楚的地方，你也可以观看 Detexify 官方的介绍视频。

Detexify 除了 Mac 客户端，还有 [网页版](http://detexify.kirelabs.org/classify.html)，你也可以在网页版上找到下载链接和购买途径，每份 license 的价格为 8 欧元，购买后可以消除粘贴文本中提示购买的信息。
