既然Mac上没有Siri，那就体验一下微软的Cortana呗。

![](.evernote-content/0FA05ABD-FE65-41A4-AC13-0163DB39E905/21AC5914-FED4-4C88-A761-AF136045DE58.jpg)

  
　　威锋网讯，虽然Mac的用户现在还不能在 OS X 上用上 Siri 语音助手，但多亏了 Parallels 11 的更新，我们可以在 Mac 上使用微软的 Cortana，它可以让你在 Mac 上运行 Windows 10 和 Windows 应用。  
  
　　Parallels 和很多虚拟机应用一样，可以让你在 Mac 上运行其它系统，但它不需要在 Boot Camp 中启动进入 Windows，你可以直接打开 Parallels 并打开任何你需要的 Windows 应用。  
  
　　Parallels 最出色和最独特的功能在于其 Coherence 模式，它可以让 Windows 应用和你的 Mac 应用一同运行，而不是出现在独立的窗口。这意味着用户可以直接在 Mac 的桌面中使用微软的语音助手。  
  
　　OS X 的 Parallels 11 今天正式推出，它在提升了性能的同时还增加了对 Windows 10 的支持，它同样包含了一个全新的“旅行模式”，可以有效地减低电量的消耗。  
  
　　Parallels 11 定价 80 美元，但 Parallels 9 和 Parallels 10 的用户可以享受以 50 美元的优惠价格进行购买。相比于其它的虚拟机软件，Parallels 11 可以直接在 OS X 的前端运行，相当于 Cortana 被内嵌在 OS X 中。
