> 服务器达人注意了，支持 Yosemite 的 OS X Server 已经上架了，可以开始折腾了。

以下为文章全文：

![](.evernote-content/A78C7F26-7D35-4727-8B34-743C86EAAEBA/861A36D2-939A-4FCC-AFE6-F07B3EAB8F69.jpg)

  
  
　　近段时间科技媒体人士都被一年一度的 CES 展会吸引住了，可能这是没人会注意到苹果放出一个 OS X Server 重大更新的原因。威锋网消息，苹果近日正式发布了 OS X Server 4.0.3，这一个版本可为服务器和企业用户带来完整的 OS X Yosemite 和 iOS 8 支持。与以往相比，今年苹果发布相同重大更新的时间稍晚。  
  
　　OS X Server 4.0.3 如今已上架 Mac App Store，售价 19.99 美元（中国区 128 元），可让任何 OS X 设备转为服务器，其它主要功能还有：  
  
-针对 OS X Yosemite 和 iOS 8 进行了更新。  
-SMB3 是 OS X Yosemite 中用于共享文件的新默认协议。SMB3 可即时对数据进行加密或签名，防止篡改或窃取数据。  
-高速缓存服务器支持 IP 地址范围注册，用于在非 NAT 网络中缓存内容。  
-描述文件管理器支持 iOS 8 和 OS X Yosemite 的新功能，包括 Handoff 以及 OS X 企业 app 和 PDF、ePub、iBooks Author 文件等 iOS 媒体文件的推送安装功能。  
-经过重新设计的邮件服务器界面，便于为多个域名设置邮件。  
-会议室地址可存储在日历服务器中，以便与客户端的地图位置和旅行时间建立关联。  
-使用 Server 应用软件和配置描述文件来简化 Xsan 宗卷和客户端的设置。  
-Server 现在可执行网络诊断测试，验证是否可从互联网访问你的服务器。网络诊断测试通过检查服务器的主机名、服务端口和 DNS MX 记录来验证可访问性。  
-现在可在 "Server" 面板的 "访问" 标签中定义防火墙和基于用户/群组的服务访问控制功能。访问限制功能可应用于所有网络、本地网络、本台 Mac、自定义网络以及特定用户和群组等多个级别。  
-Xcode 服务器兼容 Xcode 6。  
-经过优化的图形和可视化统计数据
