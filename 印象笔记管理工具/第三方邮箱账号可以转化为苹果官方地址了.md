# 第三方邮箱账号可以转化为苹果官方地址了

---

![](.evernote-content/DECE6999-896A-4CC0-9E1A-8B1F06C6CFDA/5B623858-EB0B-4662-951D-87014EE40163.jpg)  
  
　苹果今天对 Apple ID 的工作方式进行了小小的改动，苹果用户终于可以将之前使用的第三方电子邮件地址更新为 Apple ID 的官方邮箱格式。意思是假如你之前使用的是网易或腾讯等邮箱注册了 Apple ID，那么现在如果你需要的话，可以将其转换成苹果 @icloud.com, @me.com 或者是 @mac.com 等电子邮件地址。  
  
　在今天之前，使用第三方电子邮件地址的苹果ID可以被更改为另一个第三方电子邮件地址，但没有一个选项可以让其转换成苹果账号创建时的官方电子邮件账户。  
  
　这一变化是由国外的苹果用户 Dillon 概述的，他在本月早些时候给苹果几位高管发了一封电子邮件，要求改变这个问题。苹果高管上周联系了 Dillon，并告知其苹果的工程团队将研究这一问题。他今天接到了第二个电话，让他知道这个问题已经解决了。  
  
　以下内容来自 Dillon 的报告:很长一段时间内，如果你注册了一个苹果 ID，但是使用第三方电子邮件地址作为你的苹果 ID，你就无法将它更改为苹果官方的电子邮件地址……即使之后注册的苹果邮箱地址是在同一个账户上。  
  
　几个星期前，我给蒂姆•库克、克雷格•费德里吉、菲尔•席勒和埃迪•库伊发了一封邮件。我解释了自己的想法，并问他们能否解决这个问题。上周，我收到了一封来自苹果高管关系部门的电子邮件和电话。与我交谈过的女性职员告诉我，这个问题将会被发送给一个工程团队，并将得到解决。今天，我接到了另一个电话和邮件，通知我这个问题已经解决了。

![](.evernote-content/DECE6999-896A-4CC0-9E1A-8B1F06C6CFDA/C872A5B0-A1A6-4E4B-9E06-3408481E1A74.jpg)

我试过了，我终于可以把我后面注册的苹果电子邮件设置为 Apple ID 了！苹果的“改变你的 Apple ID”支持文件今天更新，以反映对 Apple ID 变更的修改，现在它包含了一个确认将第三方电子邮件地址更改为 @icloud.com 的部分，你也可以选择 @me.com 或 @mac.com 等结尾的电子邮件地址。

如果用户执行了将第三方的苹果 ID 电子邮件地址切换到以 @icloud.com, @me.com 或者是 @mac.com 结尾的电子邮件地址。苹果警告称，用户将无法通过新的操作再次让其更改为第三方电子邮件地址账户。如果你输入一个新的苹果 ID，那账号的结尾地址就会变成 @icloud.com, @me.com 或者是 @mac.com，用户可以看到一条确认信息。

你的前苹果 ID 以第三方电子邮件的形式结束，成为你的 Apple ID 账户的额外电子邮件地址。对于所有希望将自己的苹果 ID 地址改为苹果官方电子邮件地址的苹果用户来说，这应该是一个受欢迎的改变。那些想要进行更换的人应该阅读苹果的[支持文档](https://support.apple.com/en-us/HT202667)，并遵循所有的步骤，其中包括在做出改变之前在所有 iOS 设备上签名。
