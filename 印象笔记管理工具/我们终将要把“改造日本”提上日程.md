# 我们终将要把“改造日本”提上日程

Original明叔杂谈明叔杂谈

1）今天，日本一些政客，乃至日本社会，整体上在历史问题上是非不分、真相不明。这些都可以归结到一个原因——二战结束后，作为日本事实上的占领国美国，对日本军国主义的清算非常不彻底。

2）美国之所以对日本军国主义清算不彻底，主要原因还是在于，二战后，随着全球地缘政治局势发生重大变化，尤其是在冷战爆发后，美国对日本的战略需求发生了变化，美国为了遏制苏联势力和共产主义在亚洲的传播，选择了不再彻底清算日本军国主义，转而扶持日本。

3）首先，美国保留了日本的天皇体制，让作为事实上战犯的裕仁天皇逃过了被送上绞刑架、被彻底清算的命运。裕仁这个糟老头子1926年即位，他是日本发动血腥的、残酷的对外侵略战争的罪魁祸首之一。就是这样一个恶贯满盈的家伙，最终没有被枪毙、没有被绞死，一直活到了1989年。这本来就是对所有被日本在二战中侵略过、伤害过的国家和人民的最大不公。如果没有美国的包庇和纵容，裕仁的结局本应该跟希特勒一样自杀，或者是跟墨索里尼一样，被枪毙后尸体吊在架子上示众。小日本的一些政客，以及那些对着中国人出言不逊、整天叫嚣的日本人，你们不要以为我们忘了这一点。

4）美国放过恶贯满盈的裕仁，不过是它包庇、纵容日本战犯的大规模行为的冰山一角。美国还对大量日本甲级战犯等进行了有选择性的赦免，让这些人重新担任日本政府重要官职。从某种程度上来说，今天的日本政坛，依然与犯下滔天罪行的日本军国主义政权有着千丝万缕的联系。了解了这一点，你对今天日本一些政客拼命否认、洗白和美化日本侵略历史，就不会感到任何奇怪了，因为，这些人本来就是一家子。

5）在这里，我以岸信介为例，讲一讲日本军国主义政权与战后日本政府千丝万缕的联系。岸信介曾是东条英机内阁成员，1945年9月11日，美国占领当局指定岸信介为甲级战犯并将其收押。1948年，美国看中了岸信介亲美、反共的立场，在东条英机等7名甲级战犯被绞死的第二天，就将岸信介释放。此后，岸信介重返日本政坛，并在1957年至1960年两次担任日本首相。

6）岸信介绝非日本战后政坛的孤例。而且，岸信介不是一个人在从政，而是带动了他的整个家族在从政，对日本政坛的影响一直持续到今天。岸信介的胞弟佐藤荣作，曾连任四届日本首相，共七年零八个月。美国容忍佐藤政府，后者则投桃报李，奉行亲美、反共的政策，对新中国非常敌视。

7）岸信介的外孙安倍晋三，在2006年9月至2007年9月、2012年12月至2020年9月，两次担任日本首相，前后近10年之久。此外，岸信介的另外一个外孙岸信夫，是安倍晋三的弟弟，曾担任日本防卫大臣。岸信夫之子岸信千世，1992年生，曾在富士电视台当记者，2020年也进入日本政坛。

8）通过岸信介的例子我们就可以看到，美国是怎么放过日本那些战犯的，又是怎么在当时的地缘政治环境下利用这些战犯的，以及这些战犯及其后代是如何继续在日本政坛保持活跃的。

9）我在这里再补充另外一个例子——小泉纯一郎。在1978年10月17日靖国神社偷偷将东条英机等14个甲级战犯的灵位纳入供奉之后，小泉纯一郎是亲自参拜靖国神社次数最多的日本首相。小泉纯一郎2001年4月26日至2006年9月26日担任日本首相，在任期间，曾经六次亲自参拜靖国神社，另外至少六次向靖国神社敬献供品。2001年10月，小泉纯一郎曾经访问中国，他参观了卢沟桥和中国人民抗日战争纪念馆，并鞠躬、献花圈、默哀。但是，看看他此后屡次参拜靖国神社的恶劣表现，谁会相信，他对日本侵华历史的反省是真诚的、是深入的？总有人说，日本已经道过歉了，但是，你什么时候看到过杀人犯的后代一边道歉、一边又大肆纪念杀人犯的？这样的道歉有什么意义？小泉纯一郎的外祖父小泉又次郎一生狂热支持军国主义，1942年参加右翼组织大政翼赞会，还曾担任日本国会视察团团长到前线慰问日军。战后，小泉又次郎被开除公职。但是，小泉纯一郎的父亲小泉纯也，后来成为了日本自民党内的鹰派人物，并在1964年出任日本防卫厅长官。小泉纯一郎的次子小泉进次郎目前也在从政，1982年出生的他，曾担任日本环境大臣、复兴大臣等职务。小泉进次郎立场保守，是当前日本自民党内重要年轻派人物之一。2025年8月15日，就在日本战败纪念日这一天，小泉进次郎作为石破茂内阁成员，亲自参拜靖国神社。这不是朝着中国人民的历史伤口撒盐吗？你说从小泉纯一郎到小泉进次郎，这些日本政客真的在反省吗？鬼才信！

10）看到这里，相信大家已经非常清楚了，正是因为美国战后对日本军国主义清算不彻底，从裕仁天皇到岸信介这样的战犯，很多人都逃过了正义的惩罚。不仅如此，他们及其家族成员还继续在日本从政，并在战后日本政坛发挥了重要作用。这些战犯的后代，由于家庭教育和个人政治立场等原因，不可能对日本在二战中的侵略历史做出客观、公正的评价。在目前美国再次借助日本遏制和打压中国的时候，美国也不可能迫使日本去建立正确的历史观。正因如此，中国才有责任、有义务施加外力，对日本进行改造，因为，如果日本不能建立起正确的历史观，它就不可能跟中国等周边国家建立起良好的关系，它还有可能在未来再次重复侵略的罪恶历史。

未来，中国对日本社会进行全方位改造的场景包括：

第一，在中国彻底解放台湾时，日本和美国联手干预，或者，日本在美国挑动下，因钓鱼岛问题等与中国发生战争，中国彻底打败美国和日本，将美国从日本赶出去，并由中国对日本进行新的军事占领，然后，对日本政府、教育、媒体、学术等进行全方位改造，帮助日本人建立起正确的历史观，剔除日本国民性中的侵略因素。

第二，如果中日之间不直接爆发战争，中国则需要利用美国衰败的历史时机，逐步削弱美国在日本的影响力，同步扩大中国在日本的影响力。中国要及早筹划，在日本国内支持有良知的政治人物、媒体领袖、学术权威和青年领袖等，让日本国内“东风逐步压倒西风”。在2035年、2049年之后，中国改造日本的任务将会日渐明显和紧迫。

第三，我们一定要认识到彻底改造日本的重要性，绝对不要因为到时候随着美国衰败、日本一些政客对中国表面上说软话，就放松了对日本军国主义的彻底清算。不把日本人的“脑子”改过来，这个国家早晚还会再次对中国刀兵相向。已经有日本政客公开表示要“卧薪尝胆”，等到中国衰落时再跟中国一战。这就是一个明确无误的信号——日本根本不会进行反思，它完全有可能再次加害中国。

第四，改造日本，说到底就是对日本实施“去军国主义化”，就是让日本拥抱和学习优秀的中华文明，就是让日本在真正走向文明的基础上去掉侵略成性的戾气和暴虐，就是让日本和中国等周边国家永久地和平相处。

当然，在现阶段，中国面临的主要矛盾依然是美国纠集其跟班儿对中国进行遏制和打压，中国的主要任务还是完成国家统一、实现民族复兴。在这个阶段，我们要用强大的舆论攻势和军事威慑，让日本被牢牢地钉在历史的耻辱柱上，不敢轻举妄动。但与此同时，我们也没有必要立即跟日本做彻底的清算，毕竟，日本只是一条美国手中的狗，跟狗纠缠，远不如直接对狗主人下手更有意义。这是中国人的卧薪尝胆，这也是中国人的战略耐心。正所谓，九世之仇，犹可报乎？答案是，虽百世可也！
