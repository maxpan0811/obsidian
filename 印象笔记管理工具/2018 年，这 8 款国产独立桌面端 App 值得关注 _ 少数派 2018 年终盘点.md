Eagle
-----

![](.evernote-content/08001A0D-A1D0-4564-B0B9-84A86AB252A8/C9D5EF74-0575-42AA-B848-7ED2CC42339C.png)

无论你是专业设计师还是普通用户，都可能会面临图片素材管理的难题，而 Eagle 可以担当从收集、整理到检索的大任。

在网页上看到了一组创意图片集？Eagle 的浏览器扩展可以帮助你一键收集它们；总是需要在各个文件夹和软件间切换？Eagle 支持子文件夹、标签、智能过滤和众多专业文件格式；面对数千张设计素材一筹莫展？Eagle 还可以通过颜色、星级、比例、注释及来源网址等高效筛选。

你可以访问 [Eagle 官网](https://eagle.cool/ "Eagle 官网") 下载 [macOS](https://eagle.cool/downloads/macOS "macOS") 和 [Windows](https://eagle.cool/downloads/windows "Windows") 试用版本，并在 [少数派正版软件商城](https://sspai.com/item/19 "少数派正版软件商城") 购买其授权。

MarkEditor 2.0
--------------

![](.evernote-content/08001A0D-A1D0-4564-B0B9-84A86AB252A8/BAB3188F-DA47-4002-823F-8B0EBB4DF478.png)

少数派曾推荐过许多 Markdown 编辑器，如专业高效的 [Ulysses](https://sspai.com/series/32 "Ulysses")、所见即所得的 [Typora](https://sspai.com/post/30292 "Typora") 和跨平台的 [Yu Writer](https://sspai.com/post/46205 "Yu Writer") 等。而 MarkEditor 2.0 可能是最适合国人的那一个。

为了让你的写作体验更舒心，MarkEditor 2.0 将图床和发布服务结合，并做了本地化改进。你可以直接从 Unsplash 搜索并插入图片，并一键将文章发布至 [少数派](https://sspai.com/post/47278 "少数派")、Bitcron 或复制为微信公众号格式，当然你也可以选择导出为 PDF 或幻灯片。

精于中文写作，MarkEditor 2.0 支持在中西文间自动插入空格，你还可以在中文输入状态下按快捷键直接输入英文标记符，免去频繁切换输入法的烦恼。此外，MarkEditor 2.0 还可以对中文进行词性分析，帮助你精炼文章。

你可以访问 [MarkEditor 官网](https://www.markeditor.com/ "MarkEditor 官网") 下载其 macOS 试用版本，并在 [少数派正版软件商城](https://sspai.com/item/43 "少数派正版软件商城") 购买其授权。

MWeb 3.0
--------

![](.evernote-content/08001A0D-A1D0-4564-B0B9-84A86AB252A8/18889841-CD5E-401F-A311-F0D87C07B815.png)

Mweb 3.0 是另一款强大的 Markdown 编辑器，本文亦是用它写作完成。与 MarkEditor 相比，MWeb 3.0 更加注重于文档库管理及静态博客的生成与发布。

在 MWeb 3.0 更新中，你可以在菜单栏快速记录灵感，并选择将一定周期内的记录整理为同一篇文档。而 `⌘Command-O` 快捷键能够调出快速搜索界面，并用于沉浸模式下切换打开的文档。

此外，MWeb 3.0 还自带了数种网站主题，可以一键发布文章至 WordPress、Medium、Blogger、印象笔记等服务。你也可以将写好的文章导出为图片、PDF、Epub、Docx 等格式，并上传包含图片至七牛云、Imgur、Google Photos 等平台。

你可以访问 [MWeb 官网](https://zh.mweb.im/index.html "MWeb 官网") 下载其 macOS 试用版本，并在 [Mac App Store](https://itunes.apple.com/cn/app/mweb/id1403919533?l=zh&ls=1&mt=12 "Mac App Store") 或 [少数派正版软件商城](https://sspai.com/item/13 "少数派正版软件商城") 购买其授权。MWeb 3.0 同时提供 iOS 版本，你可以在 [App Store](https://itunes.apple.com/cn/app/mweb-pro-markdown-writing/id1183407767?l=cn&mt=8 "App Store") 获取。

CC 助手
-----

![](.evernote-content/08001A0D-A1D0-4564-B0B9-84A86AB252A8/E4E55FB1-74E3-45B0-A6AC-6AB52D8813A3.png)

遇到需要暂存的碎片信息时，你会怎么处理？印象笔记剪藏有点兴师动众，剪贴板记录工具又未免不够直观。而有了 CC 助手，你就能按两下 `⌘Command-C / ⌃Ctrl-C` 将它丢进收集箱，留待稍后处理。

既然利用了系统的复制快捷键，CC 助手自然能够收集几乎任意可复制的内容。无论是微信刚收到的文件，还是网页看到的文字，你都能轻松收集，并添加 Todo 或编辑文本。CC 助手还支持自定义插件，从而实现 OCR、翻译、生成二维码等进阶功能。

你可以访问 [CC 助手官网](https://ccyixia.com/ "CC 助手官网") 免费下载其 [macOS](https://ccyixia.com/updates/download?platform=mac "macOS") 和 [Windows](https://ccyixia.com/updates/download?platform=windows&arch=x64 "Windows") 版本。

Snipaste 2.0
------------

![](.evernote-content/08001A0D-A1D0-4564-B0B9-84A86AB252A8/F3922995-C2FA-44B3-88F3-E59C8ACC9FE5.png)

Snipaste 创造性地将截图与贴图结合在一起，并在去年就登上了本榜单。今年，其发布了 Windows 2.0 版更新，还推出了功能更强大的专业版。

在截图时，对齐边缘是件令人头痛的事。而 Snipaste 专业版可以通过鼠标滚轮自动框选父 / 子元素，并在编辑时按右键选中界面元素，快速添加标注。你也可以手动输入截图尺寸，在拖拽调整大小也能移动选区。

贴图后，你也可以右键框选设置缩略图，并裁剪、多选、置顶及分组贴图。你还可以设置贴图背景色和灰度显示等属性，或者从浏览器直接拖拽至已有贴图。如果贴图是颜色卡片，你可以手动更改其色值。

此外，Snipaste 专业版还允许你二次编辑标注、独立记忆每种形状颜色及调整透明度，甚至用屏幕触发角执行指定操作。

你可以访问 [Snipaste 官网](https://zh.snipaste.com/ "Snipaste 官网") 下载其 [Windows 2.0 免费版](https://dl.snipaste.com/win-x64-cn "Windows 2.0 免费版") 及 [macOS 公测版](https://dl.snipaste.com/mac-beta-cn "macOS 公测版")，或在 [Microsoft Store](https://www.microsoft.com/zh-cn/p/snipaste/9p1wxpkb68kx "Microsoft Store") 以 99 元的价格购买专业版。

![](.evernote-content/08001A0D-A1D0-4564-B0B9-84A86AB252A8/14FE8B0B-9527-454E-8B08-77EC3C373B9B.png)

苹果生态圈内有很多大家都很熟悉的 Todo App ，像 OmniFocus、Things 和 2Do 等。而说起 Windows 上的 Todo App，也有 Microsoft To-Do、Todoist、滴答清单等软件供你选择。但是，如果你只想要便利贴一样的基础功能，轻量简洁的小黄条或许会打动你的心。

正如其名，小黄条可以像报事贴那样「贴」在 Windows、Android 桌面及 iOS 通知中心上。其交互也十分直观，点击空白处新建任务，左滑或长按设置提醒，右滑和双击则可以标记为已完成，并在所有设备间同步。你还可以自定义 Windows 版的字体、颜色、间距等参数，更好地搭配壁纸。

你可以访问[小黄条官网](http://www.6fcsj.com/ "小黄条官网") 免费下载其 Windows 版本，并在 [App Store](https://itunes.apple.com/cn/app/id1175974945?mt=8 "App Store") 和 [Google Play](https://play.google.com/store/apps/details?id=com.xiaohuangtiao "Google Play") 获取移动 App。

RIME | 中州韻
----------

![](.evernote-content/08001A0D-A1D0-4564-B0B9-84A86AB252A8/80152751-279D-42CE-BA5C-D1A74D3CFF55.png)

RIME | 中州韻是一个跨平台的输入法算法框架，并拥有 macOS、Windows、Linux 和 Android 等前端实现。

作为开源项目，RIME 相比搜狗、百度等最大的优势，除了隐私安全外，我想就属强大且可高度定制的输入方案了。除了常规的全拼、双拼、五笔之外，RIME 还支持笔画、注音、仓颉、粤拼、吴语等几乎所有输入方案。如果还不满意，RIME 允许你设计一套属于自己的方案，把输入法变成理想的模样。

今年，RIME 还适配了 Windows 10，对于 PC 上的效率爱好者来说无疑是一个好消息。你可以访问 [RIME | 中州韻官方网站](https://rime.im/ "RIME | 中州韻官方网站") 或 [GitHub 主页](https://github.com/rime "GitHub 主页") 获取更多信息。

RSSHub
------

![](.evernote-content/08001A0D-A1D0-4564-B0B9-84A86AB252A8/C7C1804A-21DC-447B-A830-0D261C47B7FF.png)

在算法推荐大行其道的今天，RSS 平均每 1/π 天就会在各路文章中死一次，而 RSSHub 想帮助大家延续对 RSS 的热爱。
