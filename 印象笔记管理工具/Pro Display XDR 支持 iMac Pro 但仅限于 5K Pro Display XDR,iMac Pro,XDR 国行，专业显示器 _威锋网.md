暗示 iMac Pro 接口该更新了？

[![](.evernote-content/C0AB8CEB-D509-40D7-B6FF-9ECB55D6A971/B4EA5D30-7811-4FE1-B044-11CCB55BC2A4.jpg)](https://resource.feng.com/resource/h074/h55/img201912141211590.jpeg "Pro Display XDR支持iMac Pro 但仅限于5K")

  
  
苹果高端专业显示器 Pro Display XDR 本周开售，官方也为购买者列出了与 Pro Display XDR 兼容的 Mac 型号，值得注意的是，iMac Pro 不在列表中。但是，今天我们确认 iMac Pro 可以连接 XDR 工作，不过它只能达到 5K 分辨率。  
  
在 Pro Display XDR 的技术规格中，苹果仅列出了一些能够驱动 32 英寸 6K 显示器的 Mac，其中包括 2019 年的 iMac，16 英寸的 MacBook Pro 和 2018 年的 15 英寸 MacBook Pro（以及任何与 Blackmagic eGPU 配对的 Thunderbolt 3 Mac）。  
  
视频编辑师托马斯・格罗夫・卡特（Thomas Grove Carter）已经在新的 Mac Pro 和 Pro Display XDR 上进行了测试，并在 Twitter 上分享了测试结果，幸运的是，iMac Pro 确实可以与苹果的新专业显示器一起使用，不过它仅限于 5K 分辨率显示，无法利用完整的 6K 分辨率。据称这可能是因为 iMac Pro 使用 Intel 较旧的 Alpine Ridge Thunderbolt 3 控制器，没有足够的带宽来驱动 6K 显示器。  
  
而国外有用户准备开始下一步测试：Pro Display XDR 是否可以与 iPad Pro 一起使用。我们可以期待一下这个结果，虽然希望不大，但凡事贵在折腾。
