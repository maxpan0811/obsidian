苹果Mac系统如何安装pip命令？
=================

[青衫隐灬](https://author.baidu.com/home?from=bjh_article&app_id=1667019693481040)

发布时间: 03-1017:25

最近打算简单学习下python，但是由于自己的电脑是Mac，因此需要安装pip命令。小编将Mac电脑安装pip的方法分享给大家。

1.键盘同时按“command+空格”，弹出搜索框，在搜索框中输入“终端”，打开终端。

![](.evernote-content/B139D773-1A40-456A-965D-92B7DD1417E4/B592CF61-4822-406E-8250-742766AD5F8B.jpg)

![](.evernote-content/B139D773-1A40-456A-965D-92B7DD1417E4/F1A1FFEA-7598-4BA9-96F6-46D682CA928D.jpg)

2.在终端上输入 sudo easy\_install pip 点击回车，系统会让你输入密码，密码是当前登录电脑的用户名的密码

代码示例：

appledeiMac:~ apple$ sudo easy\_install pip

3.安装后在终端上输入 pip -V （V必须大写），如果显示以下代码，则代表安装成功。

代码示例：

appledeiMac:~ apple$ pip -Vpip 20.2b1 from /Library/Python/2.7/site-packages/pip-20.2b1-py2.7.egg/pip(python 2.7)

以上就是苹果Mac os系统安装pip命令的方法，希望对您有所帮助，想了解更多关于Mac相关知识，请关注macz.com吧！
