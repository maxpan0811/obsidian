# NeXTSTEP

---

**NeXTSTEP** is a discontinued [object-oriented](https://en.wikipedia.org/wiki/Object-oriented_programming "Object-oriented programming"), [multitasking](https://en.wikipedia.org/wiki/Computer_multitasking "Computer multitasking")[operating system](https://en.wikipedia.org/wiki/Operating_system "Operating system") developed by [NeXT](https://en.wikipedia.org/wiki/NeXT "NeXT"), the technology company founded by [Steve Jobs](https://en.wikipedia.org/wiki/Steve_Jobs "Steve Jobs"). Based on the [Mach kernel](https://en.wikipedia.org/wiki/Mach_kernel "Mach kernel") and the [UNIX](https://en.wikipedia.org/wiki/UNIX "UNIX") -derived [BSD](https://en.wikipedia.org/wiki/BSD "BSD"), it was introduced in 1989 for the company's computers before being ported to several other [computer architectures](https://en.wikipedia.org/wiki/Computer_architecture "Computer architecture"). Following NeXT's transition away from hardware, version 4 was released as **OPENSTEP for Mach**, named after the company's [OpenStep](https://en.wikipedia.org/wiki/OpenStep "OpenStep") application programming interface (API).

Although relatively unsuccessful at the time, NeXTSTEP is notable for its influence on software development and computing history. [Tim Berners-Lee](https://en.wikipedia.org/wiki/Tim_Berners-Lee "Tim Berners-Lee") developed the first [web browser](https://en.wikipedia.org/wiki/Web_browser "Web browser") and [web server](https://en.wikipedia.org/wiki/Web_server "Web server") on a NeXTSTEP workstation, while [id Software](https://en.wikipedia.org/wiki/Id_Software "Id Software") used the operating system during development of [*Doom*](https://en.wikipedia.org/wiki/Doom_(1993_video_game) "Doom (1993 video game)") and [*Quake*](https://en.wikipedia.org/wiki/Quake_(video_game) "Quake (video game)").[^1] [^2]

In 1996, [Apple Computer](https://en.wikipedia.org/wiki/Apple_Computer "Apple Computer") acquired NeXT and used NeXTSTEP and OPENSTEP as the foundation for [Mac OS X](https://en.wikipedia.org/wiki/Mac_OS_X "Mac OS X") (later [macOS](https://en.wikipedia.org/wiki/MacOS "MacOS")). Its technologies continue to form the basis of [Darwin](https://en.wikipedia.org/wiki/Darwin_(operating_system) "Darwin (operating system)"), the core operating system underlying macOS, [iOS](https://en.wikipedia.org/wiki/IOS "IOS"), [iPadOS](https://en.wikipedia.org/wiki/IPadOS "IPadOS"), [watchOS](https://en.wikipedia.org/wiki/WatchOS "WatchOS"), and [tvOS](https://en.wikipedia.org/wiki/TvOS "TvOS").

Overview
--------

NeXTSTEP (also stylized as NeXTstep, NeXTStep, and NEXTSTEP) [^3] [^4] is a combination of several parts:

* a [Unix](https://en.wikipedia.org/wiki/Unix "Unix") operating system based on the [Mach kernel](https://en.wikipedia.org/wiki/Mach_kernel "Mach kernel"), plus [BSD](https://en.wikipedia.org/wiki/Berkeley_Software_Distribution "Berkeley Software Distribution")
* [Display PostScript](https://en.wikipedia.org/wiki/Display_PostScript "Display PostScript") and a proprietary windowing engine
* the [Objective-C](https://en.wikipedia.org/wiki/Objective-C "Objective-C") language and runtime
* an [object-oriented](https://en.wikipedia.org/wiki/Object-oriented_programming "Object-oriented programming") (OO) application layer, including several "kits"
* development tools for the OO layers.

NeXTSTEP is an implementation of the latter three components. Its toolkits serve as the primary development system for software on the platform.

It introduced the idea of the [Dock](https://en.wikipedia.org/wiki/Dock_(macOS) "Dock (macOS)") (carried through [OpenStep](https://en.wikipedia.org/wiki/OpenStep "OpenStep") and into [macOS](https://en.wikipedia.org/wiki/MacOS "MacOS")) and the [Shelf](https://en.wikipedia.org/wiki/Shelf_(computing) "Shelf (computing)"). NeXTSTEP originated or innovated many other [GUI](https://en.wikipedia.org/wiki/GUI "GUI") concepts which became common in other operating systems: 3D chiseled widgets, large full-color [icons](https://en.wikipedia.org/wiki/Icon_(computing) "Icon (computing)"), system-wide drag and drop of a wide range of objects beyond file icons, system-wide piped [services](https://en.wikipedia.org/wiki/Services_menu "Services menu"), real-time scrolling and window dragging, properties dialog boxes called "inspectors", and window modification notices (such as the saved status of a file). The system is among the first general-purpose user interfaces to handle publishing color standards, transparency, sophisticated sound and music processing (through a [Motorola 56000](https://en.wikipedia.org/wiki/Motorola_56000 "Motorola 56000")[DSP](https://en.wikipedia.org/wiki/Digital_signal_processor "Digital signal processor")), advanced [graphics primitives](https://en.wikipedia.org/wiki/Graphics_primitives "Graphics primitives"), internationalization, and modern [typography](https://en.wikipedia.org/wiki/Typography "Typography"), consistently across all applications.

Additional kits were added to the product line. These include Portable Distributed Objects (PDO), which allow easy [remote invocation](https://en.wikipedia.org/wiki/Remote_invocation "Remote invocation"), and [Enterprise Objects Framework](https://en.wikipedia.org/wiki/Enterprise_Objects_Framework "Enterprise Objects Framework"), an [object-relational](https://en.wikipedia.org/wiki/Object-relational "Object-relational")[database](https://en.wikipedia.org/wiki/Database "Database") system. The kits made the system particularly interesting to custom application programmers, and NeXTSTEP had a long history in the financial programming domain.[^3]

History
-------

NeXTSTEP was built on the [Mach kernel](https://en.wikipedia.org/wiki/Mach_(kernel) "Mach (kernel)") and the [Unix](https://en.wikipedia.org/wiki/Unix "Unix") -derived [BSD](https://en.wikipedia.org/wiki/Berkeley_Software_Distribution "Berkeley Software Distribution"), initially [4.3BSD-Tahoe](https://en.wikipedia.org/wiki/4.3BSD-Tahoe "4.3BSD-Tahoe"). A preview release (version 0.8) was shipped with the [NeXT Computer](https://en.wikipedia.org/wiki/NeXT_Computer "NeXT Computer") on October 12, 1988, and the first commercial release, NeXTSTEP 1.0, was made available on September 18, 1989.[^5] Later releases incorporated newer BSD code, including [4.3BSD-Reno](https://en.wikipedia.org/wiki/4.3BSD-Reno "4.3BSD-Reno") in NeXTSTEP 3.0.

NeXTSTEP was initially released for NeXT's [Motorola 68000](https://en.wikipedia.org/wiki/Motorola_68000_series "Motorola 68000 series") -based workstations. As the company transitioned away from hardware, NeXTSTEP 3.3, released in early 1995, was ported to [x86](https://en.wikipedia.org/wiki/X86 "X86"), [SPARC](https://en.wikipedia.org/wiki/SPARC "SPARC"), and [PA-RISC](https://en.wikipedia.org/wiki/PA-RISC "PA-RISC").

As part of this transition, NeXT partnered with [Sun Microsystems](https://en.wikipedia.org/wiki/Sun_Microsystems "Sun Microsystems") to develop the [OpenStep](https://en.wikipedia.org/wiki/OpenStep "OpenStep") specification, separating the application frameworks from the underlying operating system so they could be implemented on multiple platforms.

With the introduction of the OpenStep specification, NeXT rebranded NeXTSTEP as OPENSTEP for Mach, its implementation of the specification that retained the existing Mach-based operating system and compatibility with NeXTSTEP applications. Versions 4.0, 4.1, and 4.2 were released between 1996 and 1997.

Following Apple's acquisition of NeXT in 1996, OPENSTEP for Mach became the basis for [Rhapsody](https://en.wikipedia.org/wiki/Rhapsody_(operating_system) "Rhapsody (operating system)"), which evolved into [Mac OS X](https://en.wikipedia.org/wiki/Mac_OS_X "Mac OS X") (later [macOS](https://en.wikipedia.org/wiki/MacOS "MacOS")). Its technologies continue to underpin [Darwin](https://en.wikipedia.org/wiki/Darwin_(operating_system) "Darwin (operating system)"), the foundation of macOS, [iOS](https://en.wikipedia.org/wiki/IOS "IOS"), [iPadOS](https://en.wikipedia.org/wiki/IPadOS "IPadOS"), [watchOS](https://en.wikipedia.org/wiki/WatchOS "WatchOS"), and [tvOS](https://en.wikipedia.org/wiki/TvOS "TvOS").

Legacy
------

The first [web browser](https://en.wikipedia.org/wiki/Web_browser "Web browser"), [WorldWideWeb](https://en.wikipedia.org/wiki/WorldWideWeb "WorldWideWeb"), and the first [app store](https://en.wikipedia.org/wiki/App_store "App store") [^6] were all invented on the NeXTSTEP platform.

1990 CERN: A Joint proposal for a hypertext system is presented to the management. Mike Sendall buys a NeXT cube for evaluation, and gives it to [Tim Berners-Lee](https://en.wikipedia.org/wiki/Tim_Berners-Lee "Tim Berners-Lee"). Tim's prototype implementation on NeXTSTEP is made in the space of a few months, thanks to the qualities of the NeXTSTEP software development system. This prototype offers WYSIWYG browsing/authoring! Current Web browsers used in "surfing the Internet" are mere passive windows, depriving the user of the possibility to contribute. During some sessions in the CERN cafeteria, Tim and I try to find a catching name for the system. I was determined that the name should not yet again be taken from Greek mythology. Tim proposes "World-Wide Web". I like this very much, except that it is difficult to pronounce in French...

—  [Robert Cailliau](https://en.wikipedia.org/wiki/Robert_Cailliau "Robert Cailliau"), 2 November 1995 [^7]

Some features and [keyboard shortcuts](https://en.wikipedia.org/wiki/Keyboard_shortcut "Keyboard shortcut") now common to web browsers originated in NeXTSTEP conventions. The basic layout options of [HTML](https://en.wikipedia.org/wiki/HTML "HTML") 1.0 and 2.0 are attributable to those features of NeXT's Text class.[^8]

[Lighthouse Design Ltd](https://en.wikipedia.org/wiki/Lighthouse_Design "Lighthouse Design"). developed *Diagram!*, a drawing tool, originally called BLT (for Box-and-Line Tool) in which objects (boxes) are connected together using "smart links" (lines) to construct diagrams such a [flow charts](https://en.wikipedia.org/wiki/Flow_chart "Flow chart"). This basic design can be enhanced by the simple addition of new links and new documents, located anywhere in the local area network, that foreshadowed Tim Berners-Lee's initial prototype that was written on NeXTSTEP in October–December 1990.

In the 1990s, the pioneering PC games [*Doom*](https://en.wikipedia.org/wiki/Doom_(1993_video_game) "Doom (1993 video game)"), [*Doom II*](https://en.wikipedia.org/wiki/Doom_II "Doom II"), [*Quake*](https://en.wikipedia.org/wiki/Quake_(video_game) "Quake (video game)"), and their respective level editors were [developed](https://en.wikipedia.org/wiki/Development_of_Doom#Programming "Development of Doom") by [id Software](https://en.wikipedia.org/wiki/Id_Software "Id Software") on NeXT machines. Other games based on the [*Doom* engine](https://en.wikipedia.org/wiki/Doom_engine "Doom engine") such as [*Heretic*](https://en.wikipedia.org/wiki/Heretic_(video_game) "Heretic (video game)") and its sequel [*Hexen*](https://en.wikipedia.org/wiki/Hexen:_Beyond_Heretic "Hexen: Beyond Heretic") by [Raven Software](https://en.wikipedia.org/wiki/Raven_Software "Raven Software"), and [*Strife*](https://en.wikipedia.org/wiki/Strife_(1996_video_game) "Strife (1996 video game)") by [Rogue Entertainment](https://en.wikipedia.org/wiki/Rogue_Entertainment "Rogue Entertainment") were developed on NeXT hardware using id's tools.[^9]

Although relatively unsuccessful at the time, it attracted interest from computer scientists and researchers. It hosted the original development of the Electronic AppWrapper,[^10] invented by Jesse Tayler [^11] in the early 1990s, which was the first commercial software distribution catalog to collectively manage encryption and provide digital rights for [application software](https://en.wikipedia.org/wiki/Application_software "Application software") and [digital media](https://en.wikipedia.org/wiki/Digital_media "Digital media") (a forerunner of the modern " [app store](https://en.wikipedia.org/wiki/App_store "App store") " concept).

[Altsys](https://en.wikipedia.org/wiki/Altsys "Altsys") made the NeXTSTEP application Virtuoso. Version 2 was later ported to the [classic Mac OS](https://en.wikipedia.org/wiki/Classic_Mac_OS "Classic Mac OS") and Windows, becoming [Macromedia FreeHand](https://en.wikipedia.org/wiki/Macromedia_FreeHand "Macromedia FreeHand") version 4. The modern "Notebook" interface for [Mathematica](https://en.wikipedia.org/wiki/Mathematica "Mathematica"), and the advanced spreadsheet [Lotus Improv](https://en.wikipedia.org/wiki/Lotus_Improv "Lotus Improv"), were developed using NeXTSTEP. The software that controlled [MCI](https://en.wikipedia.org/wiki/MCI_Communications "MCI Communications") 's Friends and Family calling plan program was developed using NeXTSTEP.[^12] [^13]

About the time of the release of NeXTSTEP 3.2, NeXT partnered with [Sun Microsystems](https://en.wikipedia.org/wiki/Sun_Microsystems "Sun Microsystems") to develop [OpenStep](https://en.wikipedia.org/wiki/OpenStep "OpenStep"). It is the product of an effort to separate the underlying operating system from the higher-level object libraries to create a cross-platform object-oriented API standard derived from NeXTSTEP. OpenStep was released for Sun's [Solaris](https://en.wikipedia.org/wiki/Oracle_Solaris "Oracle Solaris"), [Windows NT](https://en.wikipedia.org/wiki/Windows_NT "Windows NT"), and NeXT's [Mach kernel](https://en.wikipedia.org/wiki/Mach_kernel "Mach kernel") -based operating system. NeXT's implementation is called " [OPENSTEP for Mach](https://en.wikipedia.org/wiki/OpenStep#OPENSTEP_for_Mach "OpenStep") " and its first release (4.0) superseded NeXTSTEP 3.3 on NeXT, Sun, and Intel [IA-32](https://en.wikipedia.org/wiki/IA-32 "IA-32") systems.

Following an announcement on December 20, 1996,[^14] [Apple Computer](https://en.wikipedia.org/wiki/Apple_Computer "Apple Computer") acquired NeXT on February 4, 1997, for $429 million. Based upon the "OPENSTEP for Mach" operating system, and developing the OpenStep API to become [Cocoa](https://en.wikipedia.org/wiki/Cocoa_(API) "Cocoa (API)"), Apple created the basis of [Mac OS X](https://en.wikipedia.org/wiki/Mac_OS_X "Mac OS X"),[^15] and eventually of [iOS](https://en.wikipedia.org/wiki/IOS "IOS"), [iPadOS](https://en.wikipedia.org/wiki/IPadOS "IPadOS"), [watchOS](https://en.wikipedia.org/wiki/WatchOS "WatchOS"), and [tvOS](https://en.wikipedia.org/wiki/TvOS "TvOS").

[GNUstep](https://en.wikipedia.org/wiki/GNUstep "GNUstep") is a [free software](https://en.wikipedia.org/wiki/Free_software "Free software") implementation of the OpenStep standard.[^16]
