# 中国人该不该质疑西方历史？正规史书中的内容，早已给出答案

最近一段时期，对于西方历史，国内舆论场上非常热闹，与前几年不同的是，以前沉默不语的一些专家教授名人，如今竟然公开轮番为西方历史站台，大力批判质疑西方历史的行为。

其中，虽然某些专家教授文章标题批判的是“西方伪史论”，但实际上行文中不难看到，只要是质疑西方历史的行为几乎都在他们批判的范围，甚至上升到对任何质疑的“零容忍”。

问题在于：专家教授之所以是专家教授，是因为在某个领域掌握了更多的知识，理应为国人解惑，给出西方历史的真相，既然他们相信西方历史为真，那么难道中国人真的就不该质疑西方历史吗？

其实，到底该不该质疑，不妨看一看关于西方历史的正规史书中的一些内容，比如古罗马历史的三个案例。在质疑西方历史的文章中，通常会遇到“多看几本西方书，就不会如何如何”，以下都是西史书上的，真伪如何自行判断吧。

![](.evernote-content/6FB8F0BE-F786-4027-85C1-AE687C57363D/1B692C76-5A63-4F0E-9D6A-31038616E382.jpg)

**首先，罗马帝国城市与人口**

18世纪末，英国历史学家爱德华•吉本，在不引证任何考古资料和历史文献资料的情况下，写出了厚厚的几大本“罗马帝国衰亡史”，比“三国演义”厚多了，反正没人知道他怎么知道古罗马那么多活灵活现的历史。如今，这本书成为国内很多学者研究古罗马的不容置喙的金科玉律。

这本书非常“神奇”，笔者仅谈一谈其中的两个数据：

其一说：罗马帝国光在东部亚洲行省就有5000座人口众多的城市。

看到这样的数据，笔者真是倒吸一口凉气，因为别说秦汉肯定不能相比，就是明清中国也比不了。根据史料记载，明朝全国共有140个府、193个州、1138个县，以及493个卫和359个所，依然被罗马帝国碾压。

![](.evernote-content/6FB8F0BE-F786-4027-85C1-AE687C57363D/B0356780-3DC9-4FA9-9D51-6F2D04B62AE0.png)

这个数据可不可信，大家自有判断，但偏偏这样的数据还被学者引用到了正规史书中。比如，国内学者撰写的《罗马社会经济研究》，其中就引用了这一数据，“在阿非利加主教区有650多座城市，在希腊则共有城市900座，东部亚洲行省则共有5000座人口众多的城市”。

当然，国内的一些老学者，因为过去中国太过落伍，也高估西方学者品行，故而没能对西方文献辨析，这是可以理解的，没必要对他们求全责备，但新世纪的学者还如此的话，就匪夷所思了。

![](.evernote-content/6FB8F0BE-F786-4027-85C1-AE687C57363D/57ECB7B6-4E8C-4C13-8E60-40DD50E671C0.png)

其二说：按这一不很严密的估算来看，那罗马帝国的总人数便将升至一亿二千万。

因为1.2亿过于庞大，连西方学者都不敢相信，于是罗马帝国的人口被“调整”为峰值6000万以上，即从人口估值4600万至1.2亿间取中间值。但需要注意的是，虽然西方学者不敢相信1.2亿这个数据，但依然相信这种说法有道理，所以才会在此基础上“打折”。

问题在于：西汉人口6300万、东汉约6500万，这是葛剑雄根据中国历史数据推测的结论，更建立在中国铁器普及、深耕犁的出现、几千年发展出的农作技术、庞大的国土、水稻小麦多种主粮等基础之上，那么缺乏汉朝这些客观基础的罗马帝国，靠什么养活6000万人？

![](.evernote-content/6FB8F0BE-F786-4027-85C1-AE687C57363D/E2409A88-3E89-445B-8717-8BD86805DDCC.jpg)

**其次，罗马帝国的道路**

近代以来，西方一句话在中国也颇为盛行，这句话叫“条条大道通罗马”，而从西方史书上看，罗马帝国的道路能让现代很多国家都为之汗颜。

美国查理.萨缪尔斯的《探秘古代科学技术：最坚固的路·罗马》，其中指出罗马拥有8万公里长的官道。德国史学家赫尔穆特.施耐德的《古希腊罗马技术史》中，说有超过8万公里的铺石长途公路。

英国历史学者莱斯莉·阿德金斯与罗伊·阿德金斯合著的《古代罗马社会生活》，被评价为“是较好的历史普及读物，适合历史专业师生及历史爱好者阅读”，其中指出在戴克里先统治时期，罗马拥有8.5万公里的主干道。如果加上支路，那么最起码得30万公里。

![](.evernote-content/6FB8F0BE-F786-4027-85C1-AE687C57363D/A5A102A3-2849-44CF-94E6-744498845815.jpg)

英国菲利普.马蒂塞克，牛津大学的罗马史博士，书名叫《古罗马穿越指南》，别看叫“穿越”，实质是介绍古罗马历史，其中指出“罗马帝国拥有29万公里的官方道路，以及无数的羊肠小道”。

以上这些数据，自然也公然被引用，甚至还进入了相关教科书中，比如百度教育中有一道题中提到“罗马帝国统治时期，帝国境内主要道路有372条，总长度8.5万公里。公路的每一段有精确的里程碑，主干道每隔五六英里就有一个驿站，常年供养着40匹马”。

![](.evernote-content/6FB8F0BE-F786-4027-85C1-AE687C57363D/CD3AFB69-4415-421D-A2FE-04254F0EDFD9.jpg)

**第三，罗马都城的人口**

谈这个话题之前，先问个问题：全球第一座人口百万的城市在哪？答案不在中国，而在罗马！

斯塔夫里阿诺斯的《全球通史》，关于罗马都城的描述，有一段这么写道：“当然，帝国最伟大的城市是罗马。它占地5000英亩，其人口据估计，公元二世纪时有100多万。”占地5000英亩的罗马都城人口百万，也不是斯塔夫里阿诺斯编造，而是引用西方的其他罗马史书。

其中，一英亩大约等于4047平方米，1平方千米等于100万平方米，因此5000英亩相当于20.235平方公里。因为“公元二世纪时有100多万”，换算下来每平方公里人口密度则超过5万。

![](.evernote-content/6FB8F0BE-F786-4027-85C1-AE687C57363D/32D33E80-C8C5-4264-B072-059A3147992B.png)

需要注意的是，罗马帝国还建造了大量休闲、娱乐等配套，比如700多个水池、500多个喷泉，无数面包房，宽阔的马路，无数个澡堂，可以容纳25万人的竞技场，还有若干购物中心、娱乐场所等，因此留给居住的面积很小。据此，只能说古罗马已经普及了超高层建筑，但可能吗？

显然，仅20平方公里的都城，人口却超过百万，这种叙事可不可信，相信大家都有基本常识，但问题是，国内学者依然将之引用到了城市规划专业的大学教材中：“图拉真在位时，罗马城的建设达到了顶峰，突破了13.86平方公里。奥古斯丁时期的罗马有人口近百万。”

![](.evernote-content/6FB8F0BE-F786-4027-85C1-AE687C57363D/E8F77720-E77A-4DAC-8774-5E7DB709B996.jpg)

除此之外，关于古罗马历史，还有很多问题，比如国内出版社还大力推荐《古罗马人的阅读》一书，书中谈到“说起罗马的图书业要上溯到公元前1世纪。书商与发行商一起承担图书的印刷与销售工作。书商雇佣数量不等的奴隶负责翻印图书。发行商把当时受欢迎的作家的作品摆在书店里”等，但古罗马拿什么印刷、怎么印刷？

古埃及、古希腊的历史，同样有很多让人吐槽之处，可谓数不胜数。比如关于古埃及的城市，希罗多德《历史》记录“公元前6世纪有人居住的城市有2万座”，狄奥多罗斯认为“公元1世纪时有1.8万座以上的重要乡村和城市”，还有认为托勒密王朝巅峰时期古埃及有3万座以上的城市！如果一座城市1000人，古埃及粮食能满足如此需求吗？

![](.evernote-content/6FB8F0BE-F786-4027-85C1-AE687C57363D/5BACF031-AC24-42C0-8DA7-B3DD15E82CF6.jpg)

总之，仅从西方史书来看，其中很多内容就比三国演义还三国演义，严重违背常理、违背常识，自然应该质疑。更重要的是，以上这些历史叙事，本身既无史书可信流传，也没有相应考古证据，这更给了质疑的正当理由。

按理说，西方史书中的这些明显问题，本不需要、也轮不到民间学者质疑，因为专家教授本该早就质疑，早就该做去伪存真的工作了，但问题是，本该做这件事的某些专家学者，不仅没有做这些工作，反而对这些显眼问题还坚信不疑、还将之引用、同时还严厉批判质疑者，自己不做还阻止别人做，这就是问题了。
