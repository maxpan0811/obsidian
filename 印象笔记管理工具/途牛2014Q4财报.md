> >
> > 3月4日，途牛网公布了截至2014年12月31日未经审计的第四季度财报。
> >
> >  
> >
> > -        途牛的收入和业务量增长超过华尔街预期、EPS低于华尔街预期达$0.11。但是途牛2015年Q1的增长预期比华尔街预期要好，也表示2015年收入将会持续加速增长；另外据分析师报告，途牛管理层表示利润率2015年Q1见底，2015年Q2开始将会好转。
> >
> > -        3月4日途牛股价基本保持稳定，略有下跌（-1.3%）； 第二天股价继续下跌4.7%，但是交易量不大，略低于3个月平均值。

> > **赢利能力**
> >
> > l  **毛利率**（团队游佣金率和自由行/其它业务利润率）为6.6%，相比13年Q4为3.3%
> >
> > ü  毛利率大幅提升主要由于规模扩大导致边际成本下降、直采业务量占比增加、马尔代夫价格战放缓（2013年Q4的毛利率特别低）
> >
> > ü  15年随着直采产品占比增加，毛利率可能会进一步提升。
> >
> > ü  各类产品毛利率相对稳定，团队游佣金率（不含周边游）为7-8%；自由行佣金率为5-6%。
> >
> > ü  直采产品比平台产品毛利率/佣金率 高2-3个百分点。
> >
> > •       **营业费用：营业费用同比增长211%****。**
> >
> > ü  产品研发费同比增长177%。不计股权报酬，产品研发费用占总营业费用的17%，同比增长173%，主要投入于移动平台，新产品（如自助打包Dynamic Package）以及服务（区域服务中心）。
> >
> > ü  市场营销费用同比增长273%。不计股权报酬，市场营销费用占总营业费用的63%，同比增长273%。营销费用增长主要投入于品牌和移动相关的广告宣传。
> >
> > ü  管理费用同比增长141%。不计股权报酬，管理费用占总营业费用的22%，同比增长99%。管理费用增长主要由于支持业务发展的新增管理人员成本以及上市相关费用。
> >
> > •       **运营亏损：**Q4运营亏损为1.8亿元人民币，约为13年同期亏损的3倍。相比Q3亏损继续扩大6400万。
> >
> > •       **净亏损：**Q4净亏损为1.7亿元人民币，约为13年同期亏损的3倍；相比上季度亏损额扩大6400万。
> >
> > •       **现金：**
> >
> > ü  截至2014年12月31日，途牛持有的现金及现金等价物、受限制现金和短期投资为19.7亿元人民币(3.18亿美元)。
> >
> > ü  Q4现金周转周期为（-44）天，相比去年同期为（-54）天；2014年全年现金周期为（-34）天，相比2013年为（-40）天。
> >
> > ü  Q4资本支出为3090万元人民币；2014年全年资本支出6440万人民币。

> > ü  收入增长95%-100%。预期2015年全年收入增长加速。
> >
> > ü  另外，分析师报告称途牛管理层表示2015年Q1利润率见底，Q2开始利润率提升。

> >
> > **电话会议：**
> >
> > -        **市场份额：**途牛引用了艾瑞咨询的报告，称他们在跟团游市场份额达到20.4%，相比2013年同期为17.8%。在线旅游市场份额由2013年的16%提高到2014年的20%。*（艾瑞为途牛付费报告，我们虽然在正式发布前和他们有数轮沟通，但是他们仍然大幅低估了我们**14**年的交易额，季报正式发布后我们会继续和他们沟通修正数据；另外度假**PR**正在联系易观做一份更客观的报告。）*
> >
> > -        **重点目的地：**
> >
> > ü  欧洲和日韩目的地增长迅速
> >
> > ü  泰国和东南亚市场在恢复。
> >
> > ü  途牛在中国赴马尔代夫旅游市场份额占17.4%。
> >
> > ü  途牛日本市场份额（在线+线下）由3%提高到4%。
> >
> > ü  异地成团帮助国内游迅速增长。
> >
> > -        **公司投资/****发展5****大领域：**
> >
> > ü  **区域覆盖:**
> >
> > **i.**2014年Q4增加45个区域服务中心。截至14年底达到75个区域服务中心，覆盖73个城市。
> >
> >                    ii.            2014年新设区域服务中心的58个城市在Q4贡献了总交易额的5.7-8.7%；在15年前两个月贡献超过12%的交易额；相比2013年Q4只占1.8%。
> >
> > ü  **供应链管理：**
> >
> > **i.**加强和关键供应商的合作关系；
> >
> > **ii.**增加直采产品占比
> >
> > -          目前国内游约50%是直采的；2014年出境游直采产品占比较小，将来会增加。
> >
> > -          长期直采和第三方平台供应可能会做到一半/一半。
> >
> > -          牛人专线GMV占10%，希望能提高到20%（近期？）。另外牛人专线的毛利率也较高。
> >
> > -          牛人专线等直采产品偏中高端，和第三方平台产品定位有差异，直接竞争不大。
> >
> > ü  **丰富产品结构**
> >
> > **i.**直接和当地地接社合作，异地成团：例如，原来三亚成团需要出发地客人达到20人；现在可以从各地挖掘客人，大幅提高成团率。
> >
> > -          这是2014年国内游大幅成长的主要原因，计划在2015年扩展到出境游
> >
> > -          团队游SKU由上市时的10万增加到30万；同期自由行SKU由10万增加到25万。
> >
> >                    ii.            增加中高端产品（牛人专线）进入中高端旅游市场。
> >
> >                   iii.            自助打包系统：帮助客人把不同供应商的产品打包，自动定价。
> >
> > ü  **品牌营销**
> >
> > **i.**平均订单价格较高，需要通过大规模品牌营销增强客户对途牛的信任。
> >
> > **ii.**营销效果很好，流量增加200%以上，途牛品牌的百度搜索指数也大幅提高。
> >
> >                   iii.            品牌营销投资期预计为两年（2014和2015年），14年营销费用的40%多用于线下广告费用，预计2015年仍然保持差不多的占比。
> >
> >                   iv.            在线营销费用里，14年前三个季度PC营销费用较高，Q4开始Mobile费用高于PC营销费用。15年Mobile营销费用占比会继续增加。
> >
> > ü  **客户服务**
> >
> > **i.**截至2014年底，途牛有625个产品咨询师，大部分在南京，少部分粤语咨询师在深圳；
> >
> > **ii.**另外呼叫中心员工150人。
> >
> > -        **竞争：**
> >
> > ü  确实看到一些新的竞争者，采取非常激进的竞争策略；但是途牛会坚决反击，如果必要，可能采取更激烈的价格战。
> >
> > ü  规模持续扩大，不排除有M&A
