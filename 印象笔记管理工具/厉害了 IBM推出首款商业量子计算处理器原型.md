# 厉害了 IBM推出首款商业量子计算处理器原型

---

IBM 近日宣布已经成功构建和测试了其最强大的通用量子计算处理器，这也是 IBM 的第三代量子处理器。IBM 的第一代处理器供开发人员、研究人员和程序员使用，通过 IBM Cloud 系统使用处理器进行量子计算；第二代则是商业量子处理器的原型，它是第一个“IBM Q”系统早期访问商业系统的核心，而近日 IBM 带来的第三代 16 量子位的处理器将允许更复杂的计算。

![](.evernote-content/ABE1D6A7-D7CB-4502-95F8-5ADE54BAD474/4DDF3229-A48C-4DFB-A1FE-02EF73D0F937.jpg)

今年3月，IBM 推出了业界首个商业通用量子计算平台，即为“IBM Q”系统。该系统将通过 IBM Cloud 平台交付。而在一年前，IBM 的量子处理器只作为一项科学研究工具启用，为大学提供资源、催化剂领域方面的研究，迄今为止，用户已经在 IBM Cloud 上运行了超过 30 万个量子试验。

![](.evernote-content/ABE1D6A7-D7CB-4502-95F8-5ADE54BAD474/6DBE9962-AB79-4EBD-8495-D05B861ABF93.jpg)

近日 IBM 为 IBM Q 推出两款新的处理器在处理能力方面获得了巨大的成绩，IBM宣布到目前为止功能最强大的两台量子计算机已经完成构建和测试成功，据此次测试结果表明，该系统的量子位分别达到 16 和 17 位，相比之前的 5 位量子位已经有了巨大的飞跃。通过一系列的测试和运行，“IBM Q”平台已经可以解决传统计算机无法处理的各种问题，比如药物和材料研究、供应链和物流、人工智能、金融服务以及云安全问题。

此外，为了让初学者更好地理解这些概念，IBM 还发布了一个针对 “Quantum Experience” 的 API 接口，以便更多的开发者和程序员能参与其中。2017 年上半年，IBM 还计划在这一平台上发布一个完整的 SDK工具包，供用户构建简单的量子应用和软件程序。未来几年，IBM 还计划推出50位或更大量子位的计算机。
