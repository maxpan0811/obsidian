# 众信旅游2017年大事记

---

众信旅游2017年大事记
------------

*2017-12-29*[众信旅游发言人](http://mp.weixin.qq.com/s?__biz=MzI0ODE5NTg2Mg==&mid=2649375994&idx=1&sn=55c696fdd4397e2279a08fb2f15a3db9&chksm=f1ba9a83c6cd13951dff2599fa33b9a9c50eb9715343fcb827e8a8ae1aed11879cf770a44229&mpshare=1&scene=1&srcid=12295m0vjF6XaPz9QZdYWVAR##)

**关键词：**

**创新发展 能力聚焦 合作共赢 责任担当**

2017年，众信旅游在上年度营业收入破百亿元的基础上，进一步升级在全产业链的资源布局，全面提升核心竞争力。

以不断创新的产品形态引领全行业供给侧优化发展，以标准化的科学管理体系聚焦服务水平的全方位提升，以卓越的整合服务能力承接了多个具有较大规模和社会影响力的项目和活动，凭借出众的运营优势与全球各大旅游目的地、多个异业优秀品牌展开了更深度的合作。

同时，牢固树立正确的大局观，积极务实地围绕“一带一路”、“旅游外交”等重大主题，持续构建企业的国际化合作格局，实现多方共赢；努力践行企业社会责任，在重大事件和使命面前担当有为，进一步夯实行业领军地位，并再次跻身中国旅游集团20强。

创新 发展

1、顺应旅游消费升级和供给侧改革大趋势，深度挖掘新市场新诉求，产品推陈出新，精彩纷呈。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/6FE5D167-B1F6-4E72-BEBE-92760403BABE.jpg)

● 敏锐洞察出境游一线市场主流人群的升级需求，全年度共独家研发推广了十大主题系列产品——“一家一团”、“随心自由行”、“我和春天有个约会”、“达人带路”、“神奇节日在哪里”、“全家人的暑假”、“五天年假”、“惠玩季”、“元旦跨年倒计时”、“2018狗年全家GOGOGO”。这些独具特色的品牌产品开创行业风气之先，直击出境游消费痛点，服务细分的消费客群，为人们日益增长的美好生活需要提供了优质的出境游产品供给。

2、牢牢抓住二三线城市爆发的出境游消费需求，开启国内多个区域中心城市的出境游市场新纪元。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/3003AA67-12F5-49F1-AA53-045DAB4716CC.jpg)

● 众信旅游推出从济南、南京等地出发以意大利米兰为枢纽去往欧洲各大旅游目的地的大型定期航班产品。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/592427B0-3DD4-45C6-97F1-00805E26F188.jpg)

● 全景旅游独家首发哈尔滨、长春、鄂尔多斯直航欧洲，鄂尔多斯直航以色列，天津直航日本青森等多条定期航班旅游产品，并推出“万人游新加坡”“千人游西澳”等年度大型项目。

● 上海众信旅游携手印度尼西亚狮航集团开通上海至印尼美娜多定期直航，与菲律宾国家航空、吉祥航空合作推出多款长滩、宿雾定期航班旅游产品等。

3、零售渠道布局积极稳健

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/A6140C86-7396-4016-99A2-D8D12444DE55.jpg)

● 众信旅游线下直营门店突破100家，坚持打造线上线下融合发展的新零售运营模式。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/8E382697-0B7C-42D0-97FB-3A9DD7D98BE1.jpg)

● 上海众信旅游战略并购苏州星舟国旅，与上海青旅共建线下生态圈、门店互相售卖对方的产品等举措充分体现出线下发展“新常态”下的强强联合与共赢发展之道。

4、出境综合服务板块成绩斐然

● 2017年5月，悠联货币被国家外汇管理局授予全国范围内经营个人本外币兑换特许业务资质，成为我国第九家获得该业务全国运营资质的特许机构。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/7D35695C-EF35-4551-8C17-A33DC3E58B01.jpg)

● 众信移民经过又一年的发展，已发展成为行业中的旗舰品牌，业务范围覆盖全球几十个热门国家，更开设了全国独家的奥地利配额移民，圣基茨也成为精品项目推向市场，为客户提供更多的出境服务选择。

5、资本运作持续推进

● 2017年，公司完成公开发行7亿元可转换公司债券融资，募集资金将投入出境游业务平台和“出境云”大数据管理分析平台建设。

能力 聚焦

1、2017年，在业务总量不断扩张的同时，众信旅游不忘初心，以科学化、标准化的管理体系进一步聚焦服务品质。

● 自年初开始，众信旅游以标准化管理为抓手，全方位梳理并提升各条业务线、各个服务环节的工作规范，通过制定标准、贯彻实施标准和对标准实施情况的监督检查，形成了行业领先的服务标准和服务品牌。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/297A2DEC-216D-4EDD-A26C-E87BA3A001A3.jpg)

● 众信旅游建立了更加科学系统的员工培训机制，通过全行业独树一帜的立体培训课程，整体提升了一线员工的综合服务能力。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/48373310-ECB6-4C2B-AF38-7394E23663A5.jpg)

● 全景旅游出台《全景领队服务新标准》，开启全新的领队管理模式，打造“服务为王”的强大领队集体，吸引了众多优秀领队人才，进一步提升了服务水平。

2、在锻造服务能力的基础上，面向国内外知名企业和机构客户的整合服务优势更加明显，承接了多个具有较大规模和社会影响力的项目和活动。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/2DD59244-18E8-4E3E-9F4C-1C98B3D9CC27.jpg)

● 2017年1月，众信集团旗下高端品牌奇迹旅行、浙江众信共同承接了全球浙商总会大健康委员会200 名浙商精英南极健康之行“南极论‘健’”活动。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/CAAC39DE-5B2D-4580-91D4-CA0EED6E7008.jpg)

● 5月，众信博睿承接服务的 “俞你同行”新东方2000名优秀教师航海之旅，成为中国教育界备受瞩目的一大盛事。

● 7月，众信博睿承接第三届“东亚峰会清洁能源论坛”，为来自10余个国家或国际机构的官员、专家学者提供了专业周到的会议策划和会务接待服务。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/DF7660F9-987B-4A16-B697-C2DD18EED34C.jpg)

● 8月，众信游学承接了阿斯塔纳世博会最大规模的中国青少年交流团体，为上合组织国家间文化交流增添了亮丽一笔。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/31834C3C-5FBF-4774-8E19-12DD883C6838.jpg)

● 11月，众信博睿宣布承接罗麦科技万人海外邮轮项目，该项目将为菲律宾带来史上最大规模的旅游团。

合作 共赢

1、2017年，众信旅游整合线上线下资源与渠道，与全球各大旅游目的地进一步加深伙伴关系，并与多领域优秀企业展开跨界合作，为企业发展创造更多动能。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/05206347-6313-4E57-9B6A-72F4F5F15858.jpg)

● 2017年7月，芬兰总理访华期间，众信旅游集团受邀作为旅游行业优秀代表与芬兰贸易投资旅游促进总署签署了战略合作谅解备忘录。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/79C91DF0-8DA9-404B-8F35-DF54155CA49A.jpg)

● 众信旅游、全景旅游与法荷航集团签署中国区合作协议，享有后者赴欧洲多个航点的采购优先权益。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/EB535F3C-F2EA-451F-9F69-14B62E69FE42.jpg)

● 众信旅游集团还与新加坡、加拿大、澳大利亚、新西兰、马来西亚、美国加利福尼亚州、澳大利亚昆士兰州、新南威尔士州、维多利亚州等30余个国家和地区的官方旅游推广机构，以及ClubMed、国泰航空、公主邮轮等分别签署了合作协议，入选澳大利亚旅游局升级版2017/2019年度“优选合作伙伴”，上海众信旅游与美国旅游推广局、美国联合航空联合打造的沪上首家美国主题馆成为城中潮流地。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/57F67D8D-48A5-4AF3-A15A-A67FFCB673E4.jpg)

● 众信旅游与艾威联合旅游顾问机构、华泰证券联合发布《2017年春节黄金周出境旅游价格指数报告》，为业内人士及大众游客提供了权威科学的决策参考。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/BA9077B7-508C-43E8-B0B6-701F30815301.jpg)

● 与足坛名将孙继海创立的嗨球科技达成战略合作伙伴关系，策划并制造具备独家竞争优势的精品体育旅游产品。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/412D83D7-F291-4EAD-879A-FD3DC4C0DFFA.jpg)

● 与唐山市旅游局签署战略合作协议，以打造政企合作示范样板，在市场开发、客源交流等方面实现合作共赢。

● 与热门电影《至爱梵高》中国片方携手新媒体营销推广，与国家大剧院、摩拜单车等知名机构和品牌展开会员及市场营销合作。

2、2017年，众信旅游牢固树立正确的大局观，积极务实地围绕“一带一路”、“旅游外交”等重大主题，充分发挥市场主体作用，促进多方共赢。

● 众信旅游积极响应2017年全国旅游工作会议上提出的世界旅游强国“三步走战略”，牢牢把握“一带一路”建设以及中国瑞士旅游年、中国丹麦旅游年、中国澳大利亚旅游年、中国东盟旅游合作年等活动为中国旅游连通世界带来的巨大机遇，以深受市场欢迎的沿线产品助推中国旅游行业在世界范围提升话语权。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/1D31226B-7D03-4A1D-89CA-8F6A25ED2C5F.jpg)

● 众信旅游承接了北京市旅游发展委员会赴哈萨克斯坦世博会、黑山16+1首都市长论坛及塞尔维亚“北京日”开幕式三大外事项目，为响应“一带一路”倡议，增加北京历史文化古都的海外知名度做出了应有的努力。

● 众信旅游作为创始会员单位，参与了由中国发起成立，第一个全球性、综合性、非政府、非营利的国际旅游组织——世界旅游联盟的成立。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/2F859D86-4830-426A-9E93-6E33621492BD.jpg)

● 众信旅游董事长及公司高管还参与了世界旅游城市联合会(WTCF)2017洛杉矶香山峰会、2017年欧洲消费市场灯塔峰会等国际交流活动，见证了全球最大旅展ITB落户中国，与多个国际组织和世界领先企业进行了友好互访和深入交流，企业的国际化视野和格局也随之不断拓展。

责任 担当

2017年，众信旅游积极践行企业社会责任，踊跃参与精准扶贫和社会公益事业，在重大突发事件中勇于担当，充分发挥了行业领军企业的引领示范作用。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/D0CE2590-26C1-4799-B338-02B96A639B51.jpg)

● 充分发挥典型带动效应——众信旅游作为北京市旅游委选树的“2017北京旅游行业榜样”，积极参与了主管部门组织的重大旅游安全突发事件演练、“带走一公斤”暑期文明旅游主题宣传等活动。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/6EAC74A8-1D25-471D-9FF2-7A25CA6B6D0F.jpg)

● 继续投身公益慈善事业——众信博睿应邀前往新华社对口扶贫单位——贵州省石阡县进行精准扶贫项目考察，众信旅游集团与众信博睿共同捐资10万元，并以旅游扶贫的方式研发红色生态之旅向市场推广。上海众信旅游联合中华义工联合会举行义卖活动，义卖所得全部捐赠给自闭症群体和公益基金会。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/05182383-EA4A-4627-91C8-B0CA45A61099.jpg)

● 积极促进就业创业——今年以来，众信旅游与河北民族师范学院开设了“众信旅游定制班”，还与北京城市学院、北京联合大学、北京农学院、北京财贸职业学院、北京第二外国语学院以及全国各地十余所高等院校展开了校企合作,为旅游专业毕业生提供了丰富的实习、培训及就业机会。

![](.evernote-content/FB6D1434-912C-4EF8-94C1-1B1B995A2781/6404AE00-AD36-4F14-999F-5EEE8B5D9FEC.jpg)

● 勇于担当、责任为先的企业风范——面对俄罗斯维姆航空破产、巴厘岛火山爆发等重大突发事件，众信旅游反应及时、处理果断，积极协调各方面资源保障客人安全如期返回祖国，赢得了行业内外交口称赞，更收获了客人的美誉和信赖。
