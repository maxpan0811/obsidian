# 网友们没发现董袭莹的爷爷，但发现冶金行业存在一个董家军

关于董袭莹这事，虽然这两天的热度下去了，但网友们一直没有放弃，他们一直想知道董袭莹的爷爷会是谁，但很遗憾的是，到目前为止，网友们依然没有找到她爷爷的姓名。

但正所谓有心栽花花不开，无心插柳柳成荫，网友们却由此发现了另一条线，那就是我们的治金系统或许存在一个董家军，因为董姓专家和领导真的好多好多。

在聊这个问题之前，我们先来看下董姓在全国的人数，明朝时是42万。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/325CC16D-E467-4A5C-91B0-8286128C696F.png)

到了现在，则是590万人，在百家姓里排第三十九。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/77FCDC9B-0F60-4B3F-A1DA-08F2C68185C5.png)

但在钢铁和治金行业，网友们发现了这些董姓专家和领导，比如凌源钢铁的董香君。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/AE178620-F578-48A6-A71C-6A05190EE7F0.png)

凌源钢铁还有董新民。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/36FE3D2D-E16A-4C06-B2CB-C1EA64D0D70B.png)

刚好与第八波中的内幕对应上。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/6E018A3C-0988-4794-862A-A58A9AAE2462.png)

董洪标则在南钢任职。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/93CE1ABA-2E6F-4598-B048-AB2FF22C8051.png)

钢铁线上任职的还有董翰。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/CACE8567-E04E-460B-B8A0-8DF98E2FEDD2.png)

这是钢铁行业的董姓专家，接下来我们看治金行业的董姓专家和管理者。

原冶金部有董贻正。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/0A7FB9AA-2D47-4B09-82DC-08ED2FF568A3.png)

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/20C942B5-932B-4570-8ABD-87444D4A50ED.png)

董贻正退休后，还担任协会副理事长。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/9498E688-60E5-4CF7-9EB2-0BFFFDCDBBB6.png)

接下来是董元篪，他也是出生于上海。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/5454ADAC-C9EA-4A7D-BEC4-D0E413137D68.png)

也是毕业于北京科技大学。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/B812C769-05A0-4485-945D-97197C2A404C.png)

治金行业还有个董朝望。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/B879A391-7666-49E0-93BA-34C655A181E7.png)

在北京科大任职的还有董凯。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/49FB9134-222D-4117-A277-9B3A2D696C2E.png)

董超芳也在北科大任职。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/93AA0411-F2B7-43FD-A0F1-41DDD332FD71.png)

这是董超芳的相关介绍。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/1A77859E-0902-4B4E-B6E9-2BEF7DB5A756.png)

董建新也在北科大任职。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/3516617E-C6C0-4EC2-8936-7E0EC35EAE7F.png)

治金副院长级别的还有董伯臣。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/B3D37D83-9327-499E-9E69-2316116FAFEC.png)

与董贻正同期在冶金部任职的还有董稼祥。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/2CED3087-83E5-4391-90CA-9A7AD707FF46.png)

除了董嫁祥外，董海也与董贻正同时在冶金部任职。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/089DFDB3-0F2E-4702-9454-6003F51882FF.png)

在冶金研究院工作的还有董炎武。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/11AFBE92-BEE5-48C6-A433-69A0B15CCF80.png)

在研究院任职的还有董君伟。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/FA59FF29-1FCE-4A2C-A835-75DC63130878.png)

其团队成员还有董艳伍。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/B22EC24B-6688-460F-BFB5-EDD61358A77F.png)

还有个冶金专家叫董朔。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/0867A439-6634-493A-92A6-8C624AF58CA3.png)

在冶金部的还有个董坎良。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/97E68332-5DFD-4A7D-84EF-71DE5CFE38D5.png)

有这很多董姓专家和管理者在吃冶金这碗饭 ，那么董晓晖担任中冶建研完院长也就不算奇怪了。

![](.evernote-content/850884F0-DFED-4D99-BA7D-E6317E29E9BB/5B450391-B458-432D-B673-873E3E7A308A.png)

这就是网友们发现的中国冶金行业中的董家军们，可以肯定他们并非一个家族的，也未必与董袭莹、董晓晖有关系，但因为都姓董，又同吃冶金这碗饭 ，还有好几个专家同在北科大任职，于是就被网友们联想到一起了。

写这个董家军，也只是因为网友们刚好发现了，于是我想整理出来，让大家看到，还是那句话，多知道一点总不是坏事。
