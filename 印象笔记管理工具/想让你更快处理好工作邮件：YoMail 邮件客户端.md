# 想让你更快处理好工作邮件：YoMail 邮件客户端

---

![](.evernote-content/161CE2B1-C0F7-4BDC-8157-88BB767817EC/20E56EF1-5981-4FF7-A47E-2B7CD1D478CA.jpg)

邮件作为一项古老的计算机技术[1](http://sspai.com/35154#fn1)相应的邮件客户端也是层出不穷。有 [Mailbox](http://sspai.com/tag/Mailbox) 这类带有创新的交互方式的也有 [Inbox](http://sspai.com/tag/Inbox)、[Spark](http://sspai.com/tag/Spark) 这类智能整理收件箱的[Airmail](http://sspai.com/tag/Airmail) 这类把邮件和其他软件串联的设计更是让你重新审视邮件的意义。

不过这些试图「重新发明邮件」的客户端并没有特别针对国内环境做设计。如果你不太热衷于尝试这些国外的邮件客户端又不满足于国内邮件服务商的官方网页或客户端想在工作中更好处理邮件可以试试 YoMail。

为办公而生
-----

没有国外企业常用的 ExchangeYoMail 在登陆时支持的都是**国内企业邮箱服务**节约了可能花费在配置邮箱上的时间。另外支持直接收发 Gmail 邮件相信也会戳中不少人痛点。

![](.evernote-content/161CE2B1-C0F7-4BDC-8157-88BB767817EC/42473231-7A1F-4C4C-BDD3-30FD5374CF87.jpg)  
*可直接登录国内各企业邮箱*

Dropbox 在国内不可用国产网盘也没有哪家能做到全民级别的渗透程度国内不少用户在工作中还是会经常通过邮件附件发送文件。YoMail 有单独的**附件中心**可以方便在一处查看所有邮件里的附件此外 YoMail 还提供了自己的云盘服务对体积较大的文件这是另一个选择。

![](.evernote-content/161CE2B1-C0F7-4BDC-8157-88BB767817EC/85FEC843-F8F5-4C07-8559-6B1C6DB3F8C7.jpg)  
*统一展示在附件管理*

想让你更快处理邮件
---------

对于办公人士而言邮件更多是工作信息的汇聚。如何处理这些信息也是考验产品设计的要点。虽然 YoMail 收件箱仍是传统的邮件列表形式但你可以切换到和 [简信](http://sspai.com/tag/%E7%AE%80%E4%BF%A1) 和 [Unibox](http://sspai.com/tag/Unibox) 类似的 IM **对话模式**它把与同一联系人的邮件往来以对话气泡的形式组织让你方便的查看与对方间相隔了几个月的完整对话记录。

工作中还有一些邮件需要转成待办事项。YoMail 内置日历包含了「事件」和「提醒」的功能可以将邮件直接添加为待办事项再加上通过邮件的方式「发起会议」不用再去其他的日历应用折腾一遍缩短了用户处理邮件的步骤。

![](.evernote-content/161CE2B1-C0F7-4BDC-8157-88BB767817EC/D5633D4C-13E8-4F03-9CC6-1DACAF54C969.jpg)  
*对话模式via 官方博客*

与 Gmail 类似YoMail 也支持根据发件人自动将邮件划分为社交、账单、旅行、通知、推广几个类别虽然目前地址分类名单不是很全但实际还是可以分出不少邮件来。另外YoMail 允许用户手动把发件人添加进对应分类自行扩充完善分类规则。

YoMail 还内置了「应用商店」目前整合了微信、Slack、Skype 等服务的网页端插件。将 IM 集成到邮件客户端而不是试图用邮件去取代 IM这个思路是否可行目前我还是持谨慎态度。有意思的是YoMail 开发商表示后续会开放提交自制插件的功能到时候说不定能看到一些有意思的东西。

![](.evernote-content/161CE2B1-C0F7-4BDC-8157-88BB767817EC/01EDDC3D-4672-4880-999C-76D8ED75FAE4.jpg)*官方主打的「应用商店」近期陆续整合了微信网页版和 Slack*

交互创新是遗憾
-------

在交互上的不足是目前 YoMail 最大的短板。不奢望 Airmail 般能高度定制的操作YoMail 在 Mac 端甚至对一些基本的快捷键比如「⌘ + W」关闭窗口都不支持 无疑也是一种降低操作效率。其次对于现在日渐成为主流模式的将邮件和其他效率工具如 Todo 类软件连接起来的设计YoMail 也是没有支持的倾向。可以说对于追求新颖交互的用户来说YoMail 还不太适合。

但值得期待的是在 Mac 版上架后的短短半个月时间YoMail 就已更新了多个版本迭代速度很快目前现存的诸多问题或许可以在之后的更新中解决。

经过一段时间的使用可以发现YoMail 实际上以邮件为切入点重新打造一个办公消息聚合软件。和 [Airmail](http://sspai.com/tag/Airmail) 或者 [Outlook](http://sspai.com/tag/Outlook) 这类软件相比YoMail 照顾了国内用户的习惯和实际需求这值得肯定。虽然初上架的 Mac 版由于移植等原因体验欠佳但 YoMail 专为国内邮件用户的一套处理邮件思路是不错的。也因此当前 Windows 版本的 YoMail 相对能获得更好体验。

YoMail 目前有 Windows 和 Mac 版本你可以 [前往官网](http://www.yomail.com) 免费下载据透露 iOS 版正在开发中。
