# 小娜识歌更简单 Win 10会员14267版本更新

---

[![](.evernote-content/CF889E9C-941C-4F82-B070-1386B74692CF/7EFDE16F-144A-426D-B55F-3A0F1F39E720.jpg)](http://tech.feng.com/2016-02-19/XiaoNa-general-songs-are-simple-Win-10-member-14267-version-update_638844.shtml)

![](.evernote-content/CF889E9C-941C-4F82-B070-1386B74692CF/7EFDE16F-144A-426D-B55F-3A0F1F39E720.jpg)

威锋网 2 月 19 日消息，今天凌晨，微软正式面向 Windows 10 insider 测试会员推送了最新的预览版系统更新：Windows Build 14267。

很显然，微软加快了新版本更新推送的速度，尤其是对于开启了“快速更新”的测试会员。对此微软表示，有很多用户希望能够更快获取最新的功能，尽早获得应用程序和驱动的更新，以便更快的查找当前版本的问题，并提供有效的建议和想法，促使 Windows 10 系统获得更出色的体验。

![](.evernote-content/CF889E9C-941C-4F82-B070-1386B74692CF/7CDAE68C-8D5D-4C9E-BE64-634A98B2F4E8.jpg)

Windows Insider 负责人 Gabriel Aul 说：“我们调整了 Windows Insider 测试会员接收快速版新版本的频率。”他接着表示，尽管意味着更多 BUG，但如果你更乐意稳定，也可以将更新速度的选项调整为慢速通道。

**增加了更人性化的小功能**

据微软在 Windows 博客中公布的内容显示，Build 14276 版本增加了几个新的小功能，比如 Cortana 的音乐搜索入口、Edge 收藏栏的管理、Edge 退出时清除浏览历史、改进 Edge 下载提升和功能，以及“消息 + Skype”应用程序的改进。

Cortana 语音助手搜索音乐更简单了：微软在 Cortana 界面的右上角加入了一个音乐搜索的按钮，通过点击该按钮，小娜便进入监听状态，识别你正在播放的歌曲，并为你搜索出相关的网络结果来。

![](.evernote-content/CF889E9C-941C-4F82-B070-1386B74692CF/6814761C-763A-460C-88D3-5EF503CDD3C0.png)

Microsoft Edge 浏览器收藏夹栏改进：在该版本中，如果你开启了 Edge 浏览器的收藏了，那么只要右键收藏栏就能够发现两个新的小功能。第一是可以选择是否显示已收藏网站的图标，关闭则以名称显示。第二个功能是支持在收藏栏创建新的文件夹，管理收藏的网站更方便了。

![](.evernote-content/CF889E9C-941C-4F82-B070-1386B74692CF/29E58458-06D2-4FB6-AFE8-17386C5CA8E9.jpg)

当退出 Edge 浏览器时清除浏览数据：升级之后，将支持设置退出 Edge 浏览器时自动清除浏览数据。具体操作很简单，点击浏览器右上方的“…”标识进入设置，然后点击“选择要清除的内容”按钮，在“清除浏览数据”中选择“关闭浏览器之后总是清除”。

![](.evernote-content/CF889E9C-941C-4F82-B070-1386B74692CF/91468ABA-31C9-477F-A829-AA1BE3087E9D.png)

改进 Edge 下载提示和功能：现在 Edge 浏览器开始下载时将可弹出下载提示栏，供选择下载文件到什么位置进行保存。只要打开设置，在“下载设置”这一项中打开选项即可。

![](.evernote-content/CF889E9C-941C-4F82-B070-1386B74692CF/1F1638B2-99E4-4218-B92E-DC95FB83BF6F.png)

“消息 + Skype”应用程序改进：新版中直接点击 Skype 消息左下角的回形针图标，即可以将图片当做消息附件发送了，此外，你还可以调用摄像头拍下照片当附件，消息中还可以发送自己的具体地理位置。

**修复 BUG 但仍存已知问题**

微软最后表示，Build 14267 这个版本修复了一些问题，包括“重置 PC ”功能、WSClient.dll 错误问题、Windows Hello 使用 RealSense 摄像头问题、存储容量显示错误问题。但已知问题依然存在，不过只有一两个，如果你使用 Hyper-V 并升级到了这个版本，而且有超过 1 个 vswitch 或多个虚拟网卡，则会失去网络连接。
