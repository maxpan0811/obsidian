NeXTSTEP 风格长什么样——给非设计师看的指南

Claude Code 整理 | 2026-07-19

你是不是见过那种很有老电脑感觉的界面？按钮摸起来像真的能按下去，窗口边上有立体感，标题栏是一条一条的条纹？那就是 NeXTSTEP 风格。

它长什么样？5 秒辨认法
------------

* 标题栏是条纹的 — 深灰色底上有一条条细横线，不是纯色块
* 按钮有立体感 — 看起来像从面板上凸出来或凹进去，不是平的
* 灰色是主色调 — 整个界面都是各种深浅的灰，没有花哨颜色
* 边框有方向 — 凸的东西左上亮、右下暗；凹的东西反过来
* 选中态是反色 — 黑底白字，像老式终端

具体怎么看每个元素
---------

### 标题栏（窗口最上面那条）

不是纯黑色！是深灰色底 + 细细的水平条纹。文字白色居中。看起来像金属拉丝的感觉。

### 按钮

两种状态：

* 普通按钮：浅灰色，左上有一条亮边（白色），右下有一条暗边（深灰），看起来微微凸起，像真的能按
* 按下去/选中：反过来，左上暗右下亮，看起来凹进去了，文字反白

### 表格/数据区域

背景浅灰，表头比背景稍深一点。表格线是细灰线。整体看起来像刻在面板里的感觉。

### 分区标题（小标题条）

和窗口标题栏一样的条纹风格，深色底白字。

### 滚动条

细长的灰色条，滑块也是灰色有立体感。

和我们看板的关系
--------

我们看板之前用的是 Classic Mac OS 的 Platinum 风格（90年代苹果电脑），和 NeXTSTEP 是同时代的东西，长得很像。主要区别：

|  | Platinum（之前） | NeXTSTEP（现在） |
| --- | --- | --- |
| 标题栏 | 灰色条纹 | 深灰/黑条纹，对比更强 |
| 按钮 | 灰色凸起 | 更明显的立体凿刻感 |
| 整体 | 柔和的灰 | 对比度更高，更硬 |
| 窗口 | 有阴影漂浮感 | 贴边，更像嵌入屏幕 |
