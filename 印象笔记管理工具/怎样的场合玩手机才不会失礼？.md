# 怎样的场合玩手机才不会失礼？

---

[![](.evernote-content/F56A188E-0521-4EF5-A156-B00DEB85A61B/8E58C8F0-6D16-4FCA-A893-328566FAA6DF.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/08/maxresdefault8.jpg)

哪怕是抱着手机长大的一代，总有那么些时候，也会发现看手机会给人“没礼貌”的感觉。玩手机，既是现代的沟通方式，又是娱乐方式，说到底是个人取悦自己的一种行为。而与这种个人小爱好相冲突的是，旁人是否介意看到你玩？

### 社交压强决定使用时机

[![](.evernote-content/F56A188E-0521-4EF5-A156-B00DEB85A61B/4996CB94-8ABA-4873-9E36-281E77E5D230.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/08/461.jpg)

图问：社交场合中的手机使用会影响对话吗？

82% 的调查者认为，社交场合的手机使用或多或少影响了正常对话。我们应该拥抱新的技术，也应该尊重身边的人，如何掌握最恰当的手机使用方式？这里需要引入一个“社交压强”的概念。

> 社交压强：在一个**相对封闭**的社交空间中，群体对个人行为“守规矩”的期望值。期望值越高，则社交压强越高，反之同理。

[![](.evernote-content/F56A188E-0521-4EF5-A156-B00DEB85A61B/AD62E9EF-D593-4C47-B074-5BB8B3F41087.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/08/491.jpg)

QZ 做了一个小范围调查（上图），问题是“什么时候用手机才是 OK 的？”我们可以把上图拆分成三组来看。

第一组是“普遍 OK”（>75%），涉及场合有走在大街上、交通工具上、排队时。

大家都认为这个场合没问题，因为这三个场合都不是社交空间，基本是自顾自的一段无聊时间。换言之，是社交强度最低的场合。

第二组是“还算 OK”，我把餐馆单独列为一类。

注意下面还有个家人聚餐的选项，因此这个餐馆多指独自就餐、情侣就餐、朋友小范围聚餐。独自就餐不必说，玩你的手机谁都不影响，但桌子上有其他人就不一定了。

饭桌是一个相对封闭的社交空间，饭桌上朋友之间存在较强的互动可能。根据圈子的不同，有的圈子会重视无手机的精神交流，有的圈子更重视多媒体的“新鲜事”交流，因此综合评定社交强度为中。

第三组是“不 OK”（<12%），涉及场合有家人聚餐、会议、电影院、宗教场合。这几个场合的都有着较强的群体文化/规则，身处其中需要按照规矩行事。正式会议中用手机自然是不妥，更有许多公司会在重要会议时要求寄存手机。

至于电影院和教堂，罔顾旁人感受使用手机就是个人素质问题了。综合评定社交强度为高。

[![](.evernote-content/F56A188E-0521-4EF5-A156-B00DEB85A61B/F318C480-3ECC-45A9-9CBA-75377A85DAAC.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/08/ap218126617161.jpg)

可见，社交强度与手机使用 OK 性成反比，社交强度约高，掏手机就越要谨慎。当然这些操作法则的大前提是，你想当一个被众人接受的好公民，否则上述皆为废话。

### 从肢体语言到“新比特沟通”

人类语言出现之前，沟通，就是眼神、动作和喉咙发出的哼哼声。语言的出现，让人与人之间信息传递效率上升了 N 个数量级，但眼神与肢体语言仍然扮演着重要的角色，这些下意识的动作仍是人类最诚实的语言。

后来有了信件和电话，独立于肢体语言的沟通出现了；直到有了网络，独立于一切物理个性的“比特沟通”出现了；后来有了移动互联，手机又以多媒体形式构建了当今的“新比特沟通”。

最早，人类的沟通基于个体的思想；如今，互联网构建了人类的思维之网，成为了人类沟通的新基础。不过不能不提的是，新的基础下，旧的礼法规则仍然适用，最真诚的信息依然要靠眼神和话语来传递。

“朋友围成一个圈刷着朋友圈”，在人类沟通的语境下，是进？是退？

基本的建议是，刷新鲜事之前先权衡社交压强，不要轻易地掏出那台手机。
