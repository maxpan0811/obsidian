# 中国最长的跨海大桥，要来了

文|凯风

杭州湾，正在上演大“缝补术”。

国家十五五规划纲要草案提出，推进沪甬跨海通道工程。

![](.evernote-content/0192FFDE-899D-42F8-A754-251C5C0D2D48/6EA232F3-960C-4A88-A402-58F645694FAF.png)

这意味着，又一个投资超1000亿的超级工程，被纳入国家规划，离正式落地只有一步之遥。

目前，在上海、浙江之间的杭州湾，已有多条跨海大桥横亘其上，一旦沪甬通道建成，杭州湾将从“C形”路网变为“O形”闭环。

杭州湾为何还要新建跨海通道？呼唤已久的琼州海峡、渤海海峡、台湾海峡跨海通道，还要等到何时？

******01******
--------------

南有珠江口，东有杭州湾。

在珠江口，一条“倒V字形”的出海口，将珠三角分为东西两岸，深港与珠中澳隔海相望。

在杭州湾，一条“喇叭形”的海湾，将沪上与浙东南隔开，上海、嘉兴与宁波、舟山各据一方。

![](.evernote-content/0192FFDE-899D-42F8-A754-251C5C0D2D48/1BCACC46-73D8-4094-BB49-02DA6839D541.png)

杭州湾，并非地处杭州，而是上海与宁波之间。

两地直线距离不足100公里，却因杭州湾阻隔，公路、铁路不得不绕道杭州，平白多出一两个小时。

2008年，全长36公里的杭州湾跨海大桥建成，上海到宁波通行时间从4-5小时缩短到2个多小时。

然而，10多年过去，杭州湾大桥车流量趋于饱和，节假日堵车已成常态。

更关键的是，这座大桥为纯公路桥，高铁仍要继续绕行杭州。

为此，通苏嘉甬铁路应运而生，杭州湾跨海铁路大桥开建。

根据规划，通苏嘉甬铁路有望于2027年建成通车，届时上海到宁波高铁将从1.5小时以上缩短到50分钟左右。

两条跨海通道，固然解决多数地区的通行问题，但仍难以应对长三角日益增长的车流，以及上海与宁波直连直通的需求。

沪甬跨海通道，就是由此而来。

根据规划，沪甬通道全长70公里，超过港珠澳大桥，建成后将成为我国最长的跨海大桥。

![](.evernote-content/0192FFDE-899D-42F8-A754-251C5C0D2D48/A9AE5BBE-08E9-4B8A-8C64-AC1EADC8C648.png)

这条新的跨海通道，为公铁两用，既有高速通道，也有高铁或城际通道，形成立体交通布局。

新通道将绕开杭州湾大桥，直连上海与宁波，节省数十公里，高速通行时间缩短到1个多小时，高铁则进入“半小时交通圈”。

这还不是终点。沪甬通道之外，另一条同样投资超千亿的超级工程正在路上——沪舟甬跨海通道。

![](.evernote-content/0192FFDE-899D-42F8-A754-251C5C0D2D48/CD68208D-D9AF-488A-A026-69B3E29BD658.png)

沪舟甬通道全长160公里，规模更为庞大，难度更高。

根据规划，沪舟甬从上海临港经大洋山、舟山直达宁波北仑，连接两大万亿城市，贯穿中国最大的两个港口。

目前，沪甬通道纳入国家“十五五”规划，进展将会加速，而沪舟甬还需要更长时间。

******02******
--------------

上海大都市圈，越来越像一个“省”。

沪甬跨海通道提速，不无上海大都市圈同城化战略的深层驱动。

日前，上海大都市圈规划草案印发，覆盖长三角14座地市，堪称中国最大都市圈。

有别于传统都市圈，上海大都市圈从内到外分为三个圈层：通勤圈、同城圈、产业圈。

![](.evernote-content/0192FFDE-899D-42F8-A754-251C5C0D2D48/0033FBBC-5348-440C-97BD-073B41A63C33.jpg)

安徽宣城、江苏盐城等边缘城市，虽然得以“傍”上上海，但只是最外围的产业协作圈。

与之对比，宁波、舟山则位于相对核心的“同城功能圈”，宁波下辖的慈溪市更被纳入最内层的“通勤圈”。

这一布局，不只出于地缘相近、人文相亲的考量，更重要的是宁波经济实力不容低估。

根据《[中国50强城市](https://mp.weixin.qq.com/s?__biz=MzUyMTI0MDkxMg==&mid=2247511969&idx=1&sn=851a74a05e446ffb7349beec786ff6af&scene=21#wechat_redirect)》，2025年宁波GDP高达1.87万亿，工业、外贸均跻身全国前十。

更重要的是，宁波还是“隐形冠军”第一城，坐拥众多制造业单项冠军，且与上海形成产业融合之势。

打通沪甬，既是两大经济强市高频互动的现实需要，也是上海大都市圈向浙东南地区纵深辐射的战略布局。

![](.evernote-content/0192FFDE-899D-42F8-A754-251C5C0D2D48/5BE35930-3DFC-4A2F-B5C8-54E300221BC1.jpg)

过去，上海更多只是“遥领”长三角，凭借经济实力、行政地位和产业能级，占据长三角龙头之位。

如今，随着跨海大桥开通，高铁连线成环，上海与周边形成“1小时交通圈”。

做大周边就是做大自己，做强联系就是做强自身，这是上海的长远发展之道。

******03******
--------------

三大超级跨海工程，何时落地？

台湾海峡、琼州海峡、渤海海峡跨海工程，被誉为三大世纪工程，投资都在数千亿规模。

东部的台湾海峡，随着“京台高铁”相关规划横空出世，“坐着高铁到台北”的说法引人遐想。

北方的渤海海峡，山东与辽宁隔海相望，中间是“C”形的渤海湾，最短处仅有100多公里。

南方的琼州海峡，广东与海南之间为何不建跨海大桥，几乎每年春节都会反复发酵。

然而，在国家十五五规划中，只有琼州海峡有所涉及，但仅提及建设“湛江至海口跨海轮渡及相关线路”。

![](.evernote-content/0192FFDE-899D-42F8-A754-251C5C0D2D48/356D90EF-85F5-4C46-B3D6-4732F82F9894.jpg)

湛海高铁虽是跨海高铁，实现海南与全国高铁网直连，选择的是高铁轮渡过海方案，即“人乘船过海、列车不过海”，而非跨海大桥。

根据《[7万亿投资！超级大基建，来了](https://mp.weixin.qq.com/s?__biz=MzUyMTI0MDkxMg==&mid=2247512124&idx=1&sn=5c1157a2fb2ae97cc35296498b7e01d0&scene=21#wechat_redirect)》，当前，传统基建告别大扩张时代，但超级工程仍有落地可能。

但超级工程能否推进，始终受到三大因素制约：战略上是否重要，技术上是否可行，经济上是否有性价比。

战略上势在必行的，技术上目前或许存在难度。

技术可行的，经济性价比或许不高，战略必要性也没那么大。

三重门槛横亘在前，三大世纪工程何时落地，仍然悬而未决。
