# If Alarm Clock：“更智能”的闹钟应用

---

[![](.evernote-content/A39F85C2-3397-4EB9-B8D9-C7E3DEF87844/1C5EC8C4-9074-44D9-B81B-770E7E620F71.png)](http://www.ifanr.com/wp-content/uploads/2012/04/If-Alarm-Clock-A-Smarter-Alarm_thumb1.png?9d7bd4)现在有很多人已经让智能手机代替了传统的闹钟，而各大应用商店里的闹钟应用也可以说是不计其数。但如果一款闹钟应用没有自己的杀手锏，单单换上几个漂亮的主题，添加几个平庸的功能，是很难在应用商店中脱颖而出的。

[If Alarm Clock](http://itunes.apple.com/us/app/if-alarm-clock/id505375688?mt=8) 属于可以脱颖而出的那一类。要是让我给它取个中文名字，我会叫它“条件闹钟”，因为它就是为你生活中的那些“条件”设计的一个智能闹钟，或者应该说——“更智能”的闹钟。

“如果明早不下雨的话，我就去晨跑。”“如果明天天气状况很糟糕的话，我或许应该早一点起床去上班避免塞车。”“如果我今晚睡眠质量好的话，我明天就早一点起床。”……

在你的生活中，肯定会有类似上面所提到的这些场景。普通的闹钟应用显然没办法做到这么“有人性”，它们只会在你设定的时间点不管三七二十一把你吵醒。但 If Alarm Clock 在这方面确实让人印象深刻：

你可以让它根据天气状况（天气、气温等）再决定要不要叫你起床，比如说“明早七点要是下雨的话就不要响闹钟”。If Alarm Clock 还拥有类似 Sleep Cycle Alarm Clock 记录睡眠周期和睡眠质量的功能（有的人觉得确实管用，有的人觉得只是噱头），因此你也可以让它根据你的睡眠质量，在合适的时间叫醒你。你也可以设置在符合组合条件（睡眠质量好又不下雨等）的情况下才响闹钟。

[![](.evernote-content/A39F85C2-3397-4EB9-B8D9-C7E3DEF87844/518ACB03-233E-459C-8B35-CD73B198FB55.jpg)](http://www.ifanr.com/wp-content/uploads/2012/04/mzl-1.dydecpfd.320x480-75.jpg?9d7bd4)

不过这款应用跟 [Sleep Cycle Alarm Clock](http://itunes.apple.com/us/app/sleep-cycle-alarm-clock/id320606217?mt=8&amp;ign-mpt=uo%3D2) 之类的应用一样有个严重的缺陷，就是必须让手机整晚保持非锁屏状态，这就意味着你必须插着电源使用才能确保手机不会没电而自动关机。

不过瑕不掩瑜，这款 If Alarm Clock 确实让人眼前一亮。

应用商店的时代来临后，智能移动系统的软件也迎来了爆发式的发展，但随之而来的是软件同质化严重的问题。开发者当然可以用出彩的 UI 设计来吸引眼球，但真正能够获得用户青睐、经得住时间考验的还是有杀手锏的特色应用，而“智能化”“人性化”都是方向之一，If Alarm Clock 就是非常好的例子。
