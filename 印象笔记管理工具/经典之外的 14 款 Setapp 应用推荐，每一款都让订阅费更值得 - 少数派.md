4 款图片处理工具
---------

本节先要推荐四款图片处理工具，它们分别是：快捷去除图片元素的 TouchRetouch、解除套壳之忧的 Mockuuups Studio、升级后更强的截图工具 CleanShot X、批量修改元数据的 MetaImage。

### TouchRetouch：只做好去线条、去水印这一件事

@JohnHarrod：Mac 上一键去除图片元素和线条的 app 很多，但无论是 Photoshop 还是 Pixelmator Pro，总给人一种「高射炮打蚊子」的感觉 —— 启动成本不低，也不能直接覆盖图片修改而 TouchRetouch 可以说是专门为了解决这几个痛点所诞生的，它并不全能，而只干好一件事：去图片元素、去水印、去线条，顺带附赠你一些简单的修图和裁剪功能。也正是因为轻量化，它不需要像上面的这些软件一样需要长时间的 Loading 过程，直接拖一张图片进去，即开即用，用完保存图片修改，就能直接关闭。

TouchRetouch 支持几种模式，有类似 Pixelmator Pro 的 Quick Repair 功能，也有专门用于移除线条的 Line Removal，也有可以一键完成多个去水印操作的 Object Removal 以及仿制图章工具。选择合适的工具，对着你要去除的图片部分一涂，剩下的交给它就行了。

得到了你需要的图片之后，一键 Cmd + S 就直接覆盖原图保存了（Cmd + Shift + S 就行）。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/3F563FC3-A6CF-4827-AB42-480BC129234E.gif)

拓展阅读：[动动手指，消除照片里的「障碍物」：擦图神器 TouchRetouch](https://sspai.com/post/35816)

### Mockuuups Studio：丰富套壳素材尽在此地

@Log924：在少数派写稿的过程中，常常会遇到如何更好的呈现软件界面的问题，一款趁手的截图套壳工具总是必不可少。

作为一款专业的套壳软件，Mockuuups Studio 提供了超过 500 种的预设模版，并在主界面提供了简洁明了的筛选条件供用户选择。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/B3D8E7E7-3E43-4700-AD31-6E8554227397.png)

当用户点击右下角图标，或者将界面截图拖拽至软件之中后，该截图会显示在所有模板中。用户可以直观地看到套壳后的效果图，并根据自己的实际需求进行选择。

此外，由于手机屏幕大小不同，截图也并不一定能完美匹配模版。点击单张图片后，用户可以在界面中调整截图的裁切区域，或根据不同机型进行裁切。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/0138F639-FAB6-4D2E-AFDF-20044BDC57AF.png)

使用了 Mockuuups Studio 之后，截图就不再是光秃秃的截图，而是成为了融入使用场景的一部分。比如我之前[这篇文章](https://sspai.com/post/60323)的题图，就是用这款软件一键完成的。

### CleanShot X：截图增强再升级

@化学心情下 2：macOS 自带了原生的屏幕截取工具，通过快捷键就可以实现根据不同场景下的截取屏幕 / 窗口 / 区域操作，当然要想让截屏更加顺手或者在此基础上进行增强，那么 CleanShot X 可以一试。

相比之前的版本，增强版本 X 继续基于截图这个根本需求出发，最为基本的功能当然是截屏操作，而功能上涵盖了全屏幕截图、区域截图以及基于窗口的截图，甚至你可以通过快捷键来用它代替系统的截图操作，而完成截图操作之后你可以在左下角的预览窗格中进行进一步的操作 —— 是直接删除、暂时 pin 在预览中、或是编辑甚至上传到 CleanShot cloud 云端中。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/3327FB03-0857-42A2-8E22-FAC63E2499D7.png)

除了截取工具外，CleanShot X 还提供了一个截图的二次处理工具，你可以通过内置的注释工具在图片上进行二次处理或者是局部高亮、或者是加上描边文字、马赛克、线条等等，也就是说，CleanShot X 尽力打造了一个完整的截图工作流，让你在这一个 app 中尽兴完成截图和图片处理操作。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/B56A249E-3FD5-433B-9E21-E3950C70A02B.png)

CleanShot X 另一功能则是长图截取，通过 Scrolling Capture 选择区域然后截屏，然后滚动网页就可以实现网页长截图，对于初级的使用场景可以说是非常方便了。

另一功能则是录屏功能 —— 选中区域，选择录制成什么格式（GIF 还是 MP4），相比其他软件，CleanShot X 这个功能倒是做的很细，比如你可以选择声音拾取、录制帧数等等，而最终输出的效果也是相当不赖，对于有录屏需求的朋友而言可能是一个额外福利。

### MetaImage：批量修改元数据

@張奕源：MetaImage 是一款强大的图片元数据（metadata）处理应用，它可以批量修改和增减图片的元数据，帮你实现隐私保护或社交分享的目的。

修改图片元数据最常见的场景无疑是保护隐私 —— 你可以在把图片发表到社交网站前先清理掉图片里附带的拍照设备、拍照时间、GPS 定位等讯息，避免个人信息泄漏。除此之外，你还可以借助 MetaImage 增加或修改图片的元数据，便于统一存储和管理。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/1DC0149C-1DAC-4700-BC39-A5303BCFA640.png)

作为一款专门处理图片元数据的应用，MetaImage 自然在功能上「该做的都做了」，让你可以方便地修改关于图片的一切。此外，MetaImage 还支持 EXIF、IPTC、XMP 等多种元数据规格，即使使用不同设备拍摄或用于不同场景的图片，依然可以用 MetaImage 编辑。

3 项文字处理利器
---------

接下来出场的则是稍微小众些的需求 —— 写稿必备的文字处理利器（少数派后台编辑器的秘密也隐藏在本节）。本节推荐的工具有： 用于查找特殊字符和 Emoji 的 Glyphfinder、文本自动化处理工具 TextSoap 和 Espresso。

### Glyphfinder：查找特殊字符和 Emoji

@化学心情下 2：在很多设计场景中，通用且基于文本的 Emoji 就可以作为图标的良好替代品，尤其是在开发设计阶段就可以省去相当多的额外工作量。当然真正用起来还是需要有个 Emoji 库来查找筛选会比较的方便，要知道并不是所有人都可以用输入法准确的输入 Emoji 的名称的。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/42095708-2964-4356-BF58-097AF19AC9E6.gif)

而 Glyphfinder 可以算是一个针对这样的 Emoji 查找需求应运而生的查找库，使用上可以分为「关键字搜索」以及直接「分组浏览查找」浏览：前者就是根据一些筛查的关键字检索对应的 Emoji，比如颜色、形状、类别等等都可以作为关键字进行搜索。另一种方式则是非常简单直接的浏览查找 —— 找到合适的 Emoji 或者特殊符号，然后选中再按下 Cmd+C 就可以将 emoji 复制到剪切板中了。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/CC7899E4-F351-4B6D-A8D3-EE86B165A206.gif)

想少做点图标快速实现想法？Glyphfinder 是你的绝对利器。

### TextSoap：给文本格式洗白白

@路中南：我一直都没有分享我的工作流，其实并没有什么特殊的流程，只要能找到符合我需求的工具，用就是了。

平时我还是和文字排版打交道比较多，一直在 macOS 上寻求一款比较便捷的批量文字处理工具。偶然间在 Setapp 里面发现的 TextSoap 令我狂喜，并昵称为「字皂」。它的很多功能其实我都用不到，比如转换大小写互换、句首单词大写，基本都是和中文无大相关的功能。我用得比较多的还是带有正则匹配的字符替换以及加上文章前缀后缀这样的功能，之前都是用 iOS 自带的快捷指令，如今在 macOS 本地跑起来更加便捷。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/4605D41A-8791-4AC0-8CEF-A6F9BEE08DEA.png)

它的亮点功能是可以自定义几组 Cleaners （可以理解为针对文本的一套组合拳）组成一个 Group，我就顺理成章地把我的捷径完整地搬运到字皂上，在发布文章例行排版的时候只需轻轻一点，可以完成以前反反复复好几分钟的文字清理工作。这个 Group 也支持以文件形式分享，我也已经分享给了我的好多位编辑同事。

### Espresso：神清气爽的浓缩模式

@路中南：少数派的后台编辑器其实和前台编辑器一模一样（作者们心里是不是平复了很多？），只是多了一个能够对文章 HTML 代码直接进行编辑的「源码模式」（喂喂喂导演这个是在直播上能说的吗？），但是这个源码模式还是有一点问题，就是它自动转换的源码都不会按标题层级分段，而是糊糊成一团，这时候就会用到 Espresso 的 Re-Indent All 功能，让每个层级都清清爽爽。用前文中的「字皂」清理好的源码在这里一键排版，提升一下可读性，然后粘贴至后台编辑器即可。可读性提升后，同事后续的排查和修改也会更方便，妙。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/B9EC4248-92FD-478E-97AD-F9C0BF4CE85E.png)

3 个影音播放和处理工具
------------

接下来则向大家介绍几款影音相关的工具。它们分别是超强媒体播放软件 Movist Pro、帮你精准踩点的电子相册 Fotomagico Pro 以及本地声音文件管理器 Meta。

### Movist Pro：媒体播放软件

@Log924：Movist Pro 最先吸引到我的是它足够简洁美观的界面，即使是鼠标悬停在界面上方，进度条和控制栏也只会出现在窗口四周，而不会像 QuickTime 一样贸然出现在视频中央。

而除了颜值高，各种强悍的功能它也应有尽有。除了支持播放本地的视频文件（支持 HDR 片源），它还支持直接调取在线视频网站的视频资源进行播放，可以直接通过 Safari 插件或者在软件中输入视频页面地址，对视频链接进行解析和读取，并支持不同清晰度的切换。遗憾的是软件对国内视频网站的支持有限（经过测试，可以播放优酷视频，但现在尚暂不支持腾讯、B 站等）。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/45F4D273-0768-40B2-B347-B1DC82144079.png)

除了基本的章节功能外，它还支持多语种字幕同时显示。在右上角的字幕选项中，按住 Option 键即可选择多种语言的字幕，并可以更改时间轴时间以解决音画不同步的状况。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/52194FFC-EAF5-491A-B2A5-27F83C87CACF.png)

### Fotomagico Pro：电子相册

@Log924：Fotomagico Pro 是一款轻量级的电子相册和视频编辑软件。虽然它和 Premiere、FCPX 等相比功能有限，但术业有专攻。

熟悉视频编辑软件的朋友们一定对于它的界面不会感到陌生，用户可以直接从访达中拖动多媒体文件（照片、视频、音频）加入轨道，而我最喜欢的它的点在于：丰富的预置模版、音频踩点匹配以及动画自动配置。

之前我也通过专业的视频软件制作过基于照片的电子相册视频。最让我头疼的一点在于需要逐个去设置转场、动画等特效。丰富的自定义选项在这时反而给制作带来了麻烦。Fotomagico 在照片导入的时候，就自动完成了转场和动画的设置。通过菜单中的动画助手功能， 还能完成对于动画的批量修改。

如果你想为视频添加一些自己的特色和创意，软件自带的特效库中已经准备好了不少模版，除了普通的转场特效，你还可以通过拼图模版修改照片布局，甚至还可以生成一个大片片头。缺少背景音乐？也没问题，软件自带了不少适合各种场景的背景音乐可以使用。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/08F5B6B9-7AB7-48EA-9705-099E878C0C48.png)

更有趣的一个功能是音频踩点匹配。如果需要制作「踩节拍」的视频，以往需要手工不断的调节照片和视频的长短，保证卡到每一个节奏。在这个软件中，自带了一个音频「关键帧」助手。通过播放音频，并且标记关键帧，软件会按照顺序自动调整照片的位置和时长，也就达到了一键「踩点」的效果。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/F4CF207B-B949-4E70-A1BE-1A554B8CB013.png)

舒爽踩点中

从综合实力上来说，Fotomagico 可能离专业的视频处理软件还有相当一段距离，但对于我制作照片视频的需求而言已是绰绰有余。

### Meta：管理声音元数据

@張奕源：Meta 是一款管理声音文件元数据的应用，最近才刚刚登陆 Setapp。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/B6C53F11-F53F-4E81-9049-F8E5D29A7B5E.png)

如果你是音乐爱好者，需要经常管理本地音乐，那务必要试试 Meta。和同类型的知名应用 Kid3 相比，Meta 的界面更加精美，操作流程也更加直观。在 Meta 内，除了修改声音文件的创作者、艺术家、专辑、年代、音乐类型等基础讯息，Meta 还具备非常实用高效的高阶功能 —— 你可以给音乐批量重命名、批量加标签、在线搜索并匹配专辑封面，甚至一键转入 iTunes Library，便于在 Apple Music 里收听。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/EC16E2BB-B858-40AB-BD32-AEA12E9C5BFD.png)

Meta 的专辑封面自动匹配功能超级方便

总而言之，Meta 身材小小，功能却很强大，也难怪此前这么一款 [ID3](https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=&cad=rja&uact=8&ved=2ahUKEwjb9cO9uKbqAhXYFogKHW0xCjgQFjAAegQIEBAB&url=https://en.wikipedia.org/wiki/ID3&usg=AOvVaw3glsP7rFIrJKpAqT7cdjX3) 编辑器就要卖到 HKD 200。非常推荐需要管理大量本地音乐，或者在不同声音播放器之间捣腾歌曲的朋友尝试。

2 个桌面加成工具
---------

@Vio：我想推荐的这些工具都是给桌面小小加成，能一键隐藏掉菜单栏上部分图标的 Bartender、鼎鼎有名的 [iStat Menus](https://sspai.com/post/41089)、一键完成许多事的「一键夫斯基」[One Switch](https://sspai.com/post/56663)、让 macOS 分屏更好用的 Mosaic，这些都是我想推荐的。不过鉴于它们都已经声名在外，因此在这里我就推荐名气稍小但也相当好用的 SideNotes 和 Wallpaper Wizard。

### SideNotes：就是随手记

@Vio：SideNotes 是一款非常、非常、非常方便的「随手记」的应用。效果如其名，就是在电脑的边边上写笔记。应用场景可以非常广泛，想记什么就记什么：灵感采集、待办事项、便签……

它和其他的笔记类应用最大的区别在于，直接以侧类似「通知中心」这样的方式从侧边滑入，记完之后再滑出。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/9D9D2728-7E30-43E3-A3AD-0923CF9CB222.gif)

它的呼出方式有三种：侧边按钮、侧边热区、菜单栏按钮。

对比其他的随手记工具（例如印象笔记是直接点击菜单栏按钮，弹出用于速记的小窗），它最大的特点在于，整个应用就依附在侧边，使用时弹出。而且它的功能非常完善，支持 Markdown、纯文本、代码等多种输入方式。

我最近用它来做轻度的待办管理，很管用。可以快速把自己接下来几天要做的事情给纸面化，而不用考虑老派任务管理思路中的各种分类、过滤词、精力密度分配、四象限法则等条条框框的东西，就是纯粹的记录这几天要干啥。还有之前想以便签形式暂存的一些信息，也可以放在这里：

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/CC277523-62EF-4729-98EF-EE1C71C65492.png)

总之就是一个字：**方便！**打开方便、记起来方便、看起来也方便。适合像我这样「事情多起来的时候才会做待办管理」的轻度用户。

Setapp 中的同类速记应用还有 [Unclutter](https://sspai.com/post/28798)。如果你不想订阅 Setapp，也可以在少数派商城单独购买 [SideNotes](https://sspai.com/item/71)。

### Wallpaper Wizard：散养壁纸

@Vio：一个壁纸应用，含有各种各样的散养壁纸，并且分门别类。捉到喜欢的壁纸就可以添加到列表，列表可以设置循环、定时切换等等功能，让每次见到的壁纸都不一样。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/AA0AAE9C-8A48-46EC-8FE3-E2C58FC69DB7.png)

### PDF Squeezer：一键压缩 PDF

@張奕源： Squeezer 是我最常用的 PDF 压缩工具。和 JPEG mini 等图片压缩工具类似，这款产品的基本逻辑非常简单：文件丢进去 —— 压缩 —— 保存，接着你就能得到一个体积适中的 PDF 文件了。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/17B4190B-14E2-43C3-B1CD-B19ABAF155D6.jpg)

PDF Squeezer 的主界面，右侧菜单里可以设定细节

实际使用中，你还可以根据个人需要来自定义输出 PDF 的精度数值，譬如 PDF 文件的 dpi、文件压缩比率等。就我的实际体验来看，PDF Squeezer 的压缩率和质量保持都不错，无论是日常的「把文件压小点」，还是特殊状况下「把文件体积精准地控制在某条线上」，它都能胜任。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/AF24D363-9E2C-4760-A2ED-4179CA908BBA.jpg)

PDF Squeezer 支持高级自动化

如果你是高阶用户，PDF Squeezer 甚至还为你准备了 Automator 调用、Command Line 控制等不同玩法，便于你将 PDF 压缩融入工作流，提升效率。

### BetterTouchTool：让触控板更好用

@Vio：一个「自动化」类的应用，同样也是顾名思义，就是一个让触控板更好用的应用。

上面提到，SideNotes 有三种呼出方式，其实我都不喜欢：第一种会在边缘留下一块按钮图形；第二种容易误触；第三种需要占用菜单栏图标，而且鼠标操作本身就是一个降低效率的事情。

于是，我利用 BetterTouchTool，将「双指从触摸板左侧边缘右滑」定义为打开 SideNotes 的快捷键：⌘ ⇧ 空格。这样，我就可以仿照打开「通知中心」的方式，直接用手势打开 SideNotes，非常方便。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/FE5CBFBA-72A6-4927-A256-7155A8144982.png)

当然，我举这个例子的目的是展示 BetterTouchTool 的自动化场景和逻辑。其实它的自动化对象远不止触控板，还包括键盘、鼠标、Touch Bar 等等，相信对自动化有研究的朋友们一定会有更丰富的配方，就不赘述了。

![](.evernote-content/350CA9CE-3FD6-4803-A9B7-609AABBC0F72/04C89466-7149-437A-9A86-DD6098C5A3B4.png)
