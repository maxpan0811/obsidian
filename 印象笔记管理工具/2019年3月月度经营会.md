# 2019年3月月度经营会

---

后续工作方向

1. 持续优化产品板块布局，目标是西欧和非西欧业务的正常比例维持在50%：50%左右，不含包机；西欧产品中持续提高一国、两国和三国的产品比例，Q2的产品布局中一国、两国和三国的团量比例在25%，Q3在40%，Q1实际售卖比例为11%。四国和多国产品关注竞争对手的政策变化，根据众信的导游人头和工资数据做了一次调整，5/1开始执行。
2. 继续优化销售队伍，在1月底按集团要求末位淘汰18人的基础上，4月第一周又做了一次优化，华北大区淘汰4人，西北大区淘汰1人。空余出来的编制哪个销售组能招到优秀销售就给谁，东北欧销售组本周就用了其中一个编制入职了一个新销售。继续加强电商组精细化运营的能力培养；持续投资单团组。
3. 要求有潜力的中层干部积极参与科学管理项目，提报了5个项目，3人参加培训。
