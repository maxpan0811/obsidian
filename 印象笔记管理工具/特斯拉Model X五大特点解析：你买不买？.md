# 特斯拉Model X五大特点解析：你买不买？

---

前不久，特斯拉正式推出了全球第一款全电动豪华 SUV——Model X。从特斯拉宣布要推出这款车型至今已经过了 3 年的时间，现如今，我们终于迎来了这款高端车型。  
  
　根据特斯拉的介绍，这款 Model X 沿用了 Model S 那令人印象深刻的动力系统，其整体性能、行李空间以及搭配的技术都给人一种现代高端 SUV 的感觉。今天，我们来说一说这款车型的 5 个突出特点。

**巧妙的车门**  
  
　特斯拉 Model X 的后车门采用了可以向上翻起的鸥翼式车门，这是 Model X 极具代表性的一项特征。特斯拉在 Model X 的后车门上采用了独特的双铰链设计，乘客只需轻轻往上提起，车门就会自动上翻。

鸥翼式车门除了允许乘客在狭小的空间上下车之外，还能在顶层空间不够的时候采用半开方式，不会出现车门撞到天花板的情况。Model X 的前车门虽然采用的是传统设计，但是能够感应驾驶者的位置，实现自动开关。  
  
　**多功能电池**

作为一家高端电动汽车公司，我们当然会对特斯拉推出的 Model X 的动力装置有所期待。那么，这款车型上使用的电池会实现怎样的效果呢?  
  
　据了解，Model X 采用的是容量为 90 千瓦时的电池组，其单次充电的行驶里程为 250 英里。最关键的一点在于，这一电池组在特斯拉充电桩上只需要短短的 30 分钟就能充满。  
  
　因为特斯拉在 Model X 的电池组上使用了扁平化的设计，所以这款车型的重心要比其他 SUV 低一些，这意味着，它在高速过弯时出现事故的几率将会大大降低，并且增强了车子遭受侧面撞击的能力。  
  
　特斯拉声称，如果 NHTSA 的评级标准不止五颗星的话，那么 Model X 的评级应该达到六星级标准。  
  
　**生化武器防御模式**

任何一个存在过敏现象的人都能感受到这款车子上的空气过滤器实现的正面效果，进入 Model X 的车厢内即可让人从被污染的城市空气中解脱出来，甚至还能抵御花粉的侵扰。  
  
　特斯拉 Model X 在空气过滤方面下了一番大工夫，其空气过滤器的体积是其他竞争对手的 10 倍左右，并且搭载了一个比许多车型上的主要过滤器都要大的二级过滤器。两级过滤器使得车上的乘客可以享受医院手术室级别的空气纯净度。  
  
　另外，Model X 还配有一个疯狂的“生化武器防御模式”，一旦开启，车内将会建立一个比车外气压略高的空气环境，任何有害的气体都别想进入车厢内部。  
  
　**Ludicrous 模式**

Model X 标准版(90D)从静止加速到 96km/h 只需要 4.8 秒，这对于一款 SUV 来说是相当不错的，甚至可以跟奥迪、宝马和阿斯顿马丁等运动跑车放在一起比较。  
  
　而高端版(P90D)只需要按下一个按钮，就能在 3.2 秒的时间里加速到 96km/h。互联网快速搜索之后，我们发现，只有 2014 款雪佛兰 Corvette Z06 V8 和 2011 款法拉利 SA Aperta Pininfarina 6.0 V12 能够实现这样的加速性能。  
  
　很显然，一个现代超级跑车才能拥有的性能出现在了一辆 7 座电动 SUV 上是一件不可思议的事。这也难怪，用户需要进入一个特殊的模式才能获得这样的加速效果，特斯拉称之为 Ludicrous mode(滑稽模式)。  
  
　**手机充电底座**

在谈论了生化武器防御模式和智能鸥翼门之后，我们将以一个充电底座功能作为本文的结尾，不过，Elon Musk 都特别强调了这项功能，想必它还是存在一些特别之处的。  
  
　据介绍，这一充电底座被设置在了 Model X 的中控台附近，乘客只需将手机放进去就行了，完全不用低头查看或者清理烦人的数据线。这项功能不见得有多高科技，但是非常人性化，让车主感受到了特斯拉的贴心程度。
