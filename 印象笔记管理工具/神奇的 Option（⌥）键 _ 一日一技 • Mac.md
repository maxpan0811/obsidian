# 神奇的 Option（⌥）键 | 一日一技 • Mac

---

### *关于栏目*

「一日一技」是少数派的全新栏目，我们将会介绍各种简单又实用的小技巧。这些技巧可能是你知道的，也可能是你还未注意到的；它可能是一个系统的操作技巧，也可能是某个 App 里的细节功能或用法……我们希望通过这个栏目，让你更好了解手中的设备和 App，能更充分去利用它们的特性，以此一点点改善与提升你的数字生活。

如果非要做一个比喻的话，我大概会将 OS X 比作一块可琢可磨，细加「把玩」的美玉。因为在其中实在包含着太多可以「玩味」的东西。

在 OS X 这个漂亮的操作系统中，还隐藏这许许多多的「小秘密」——有的需要通过巧妙的设置去发现，有的需要极富 Geek 范儿的「命令行」去完成，而有的，可能只需要 Option（⌥）辅助就可以了。

今天的「一日一技」我将简要地列举一些 ⌥ 键在桌面、Finder 和部分应用程序中巧妙用法，以展现这个小小按键的神奇之处。

桌面
--

* 依次点击  → 「关于本机」，在这里你可以查看关于电脑的基本信息。但如若需要查看更为详尽的信息，则需要进入「系统报告」中查看。但是如果按住 ⌥ 键并点击 ，你会发现菜单里的「关于本机」变成了「系统信息」，此时你可以快速进入「系统报告」。

![](.evernote-content/643B9C23-4A9D-4B1F-BF50-9D06171F6E7A/C4510477-1987-4C4C-8D12-C2235259990C.jpg)

* 按住 ⌥ 键，依次点击  → 关机/重新启动，可以快速关闭或重新启动计算机。
* 按住 ⌥ 键，再点击 Wi-Fi 小图标，将出现更为丰富的无线网络信息。其中，单击「断开与 XX 的连接」，可暂时断开与该无线网络的连接，但并不会删除对该无线网络的记忆。如果此时关闭 Wi-Fi，再打开，你将发现此前你选择断开的网络又自动连接了，非常方便。
* 按住 ⌥ 键，再点击电池小图标，可查电池「状况」——可能会显示「正常」、「尽快更换」或「修理电池」；点击声音小图标，可快速打开「输出设备」、「输入设备」、「声音偏好设置」。

![](.evernote-content/643B9C23-4A9D-4B1F-BF50-9D06171F6E7A/6F475ABF-8D7B-416B-A8EB-C75B191CD7CC.jpg)

* 按住 ⌥ 键，随后点击右上角通知中心（ Notification Center ）小图标，可快速开启「勿扰模式」。
* 按住 ⌥ 键，分别点按键盘上方 Fn 功能键，将打开对应的偏好设置项 —— 其中 F1 / F2 对应「显示器」；F3 对应「Misiion Control」；F5 / F6 对应「键盘」；F10 / F11 / F12 对应「声音」—「输出」。
* 在 LaunchPad 中，按住 ⌥ 键，通过 Mac App Store 下载的应用程序将呈现「抖动待删除」的效果（和 iOS 类似）。

Finder
------

* 按住 ⌥ 键的同时「清倒废纸篓」，可跳过清倒废纸篓警告和删除已锁定文件警告（注：在 Finder 的偏好设置中，「清倒废纸篓之前显示警告」是默认勾选的）。

![](.evernote-content/643B9C23-4A9D-4B1F-BF50-9D06171F6E7A/63806F65-430A-46FC-8730-4B2AC687951F.jpg)

* 按住 ⌥ 键，并在桌面点击「前往」，将出现显示「资源库」。[1](http://sspai.com/33250#fn1)

![](.evernote-content/643B9C23-4A9D-4B1F-BF50-9D06171F6E7A/562F17D8-B6C8-4C65-8EEE-0D5867F7C94E.jpg)

* 按住 ⌥ 键，右键单击 Dock 栏中的 Finder 图标，将出现「重新开启」选项，点击即可重新启动 Finder；右键单击 Dock 栏中的其他已经打开的应用程序，则将出现「强制退出」选项，点击即可强制退出。

![](.evernote-content/643B9C23-4A9D-4B1F-BF50-9D06171F6E7A/33EBE665-BB1B-4D35-BB72-44B94B167148.jpg)

* 在 Finder 复制一个文件或文件夹，随后使用 Option + Command + V（ ⌥⌘V ） 快捷键可实现「剪切」操作。
* 按住 ⌥ 键，并拖拽文件，可快速复制原文件。

![](.evernote-content/643B9C23-4A9D-4B1F-BF50-9D06171F6E7A/66C2B6EC-03B1-4F33-8C2F-614DE32F1BBD.jpg)

* 选中文件，按住 ⌥ 键，并点击鼠标右键（或双指点按触控板），可以快速显示「始终以此方式打开」，「显示检查器」以及「将 XX 拷贝为路径名称」。其中，「显示检查器」可以对多个文件 / 文件夹汇总，并显示其简介，而不是对每一个文件 / 文件夹都显示一个简介。

![](.evernote-content/643B9C23-4A9D-4B1F-BF50-9D06171F6E7A/08CA48BD-6B2D-4D69-A32C-73AA8EEF129A.jpg)

* 在 Finder 中，将文件浏览切换为「列表视图」，并将「排列方式按」修改为「无」。按住 ⌥ 键，点击文件夹左侧的向右小箭头，即可展开该文件夹下的所有子文件夹和文件。

![](.evernote-content/643B9C23-4A9D-4B1F-BF50-9D06171F6E7A/78586AFF-C2DA-4A44-B56F-6448FDDF5512.jpg)

* 在 Finder 中，「分栏视图」的每栏宽度，在默认情况下是固定不变的。如果你希望让其更宽一点，在拖动分栏线的同时，按住 ⌥ 键即可。

![](.evernote-content/643B9C23-4A9D-4B1F-BF50-9D06171F6E7A/315B4204-B148-44F8-A9BA-53554D18F721.jpg)

* 选中外置磁盘，使用 Command + E（ ⌘E ） 可快速将其退出。若同时按住 ⌥ ，则不会弹出相应的警告窗口。
* 选中任意一个图像文件，按住 ⌥ 的同时再按空格键，可快速进入「全屏预览」。

其他
--

* 按住 ⌥ 键，在 Photo Booth 中拍照时，将不会出现倒计时（默认情况下有 3 秒倒计时动画）。

![](.evernote-content/643B9C23-4A9D-4B1F-BF50-9D06171F6E7A/10D069FA-1095-48D0-8A06-A7CB18E5BACD.jpg)

* 当两个或多个应用程序窗口重叠时，按住 ⌥ 键，点击后层的窗口，该窗口将突出显示，并且，最前的窗口将自动隐藏（注：按住 ⌘ 键，则可对在后的窗口进行任意移动，而不会对在前的窗口产生任何影响）。

鉴于时间和使用经验有限，我暂时只整理了上述关于 ⌥ 键的实用小技巧，如果你有其他好的见解，欢迎在评论区与我们交流。

1. 根据 Apple 对Mac 附带的文件夹的说明，资源库文件夹中包含 Mac 上的所有用户都可用的应用程序所用的字体、互联网插件和其他项目。默认情况下，在 Finder 中是不予显示的。如果你希望让其始终保持显示，可以在终端中输入如下命令：`chflags nohidden ~/Library/`，输入完成后，回车即可；如若希望恢复原状，则需输入如下命令：`chflags hidden ~/Library/` [↩](http://sspai.com/33250#fnref1)
