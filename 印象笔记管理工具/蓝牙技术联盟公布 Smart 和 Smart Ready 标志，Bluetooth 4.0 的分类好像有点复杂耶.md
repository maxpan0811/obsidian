# 蓝牙技术联盟公布 Smart 和 Smart Ready 标志，Bluetooth 4.0 的分类好像有点复杂耶...

---

分类: [外围设备](http://cn.engadget.com/category/peripherals/)

[![](.evernote-content/E40FB409-314E-42DD-B318-F7E0DFFB3E82/2C4EAAA7-8F6E-4041-AC03-87F32EA982AA.jpg)](http://cn.engadget.com/2011/10/25/bluetooth-sig-unveils-smart-marks-explains-v4-0-compatibility-w/)

发展到今天，蓝牙已经有许多版本问世。蓝牙技术联盟（Bluetooth SIG） 最新释出 Smart 标志，让用户在蓝牙 [4.0 版](http://www.engadget.com/2010/07/07/bluetooth-4-0-specification-gets-official-devices-expected-by-q/)可以清楚分辨[不同版本](http://www.engadget.com/2010/04/21/bluetooth-4-0-with-low-energy-almost-finally-ready-to-roll/)。未来蓝牙将会分为三种：Bluetooth Smart Ready、Bluetooth Smart、以及标准 Bluetooth。根据 [Bluetooth SIG](http://www.engadget.com/tag/Bluetooth+SIG/) 的说法，新的印记是为了要分辨装置间的兼容性以及标示各版本的适用传输频率。基本上来说，「Smart Ready」适用于任何双模蓝牙 4.0 的电子产品，如 [iPhone 4S](http://cn.engadget.com/2011/10/13/iphone-4s-claims-title-of-first-bluetooth-4-0-smartphone-ready/)。而「Smart」则是应用在「心率监控器或计步器等使用扭扣式电池并传输单一信息的装置」。Smart Ready 的兼容性会最高，可与 Smart 及标准蓝牙相通。标准蓝牙则无法与 Smart 相通。这样的分类其实有些复杂，还是一定要看上面的那张表才能记住。
