Windows 上也有好看又实用的软件，12 款优秀的 UWP 应用推荐
====================================

在 Windows 10 中，微软首次引入了 UWP（通用 Windows 平台）的概念，让开发者只需一次编写，就能让程序在电脑和手机等多种设备上运行。同时，由于所有 UWP 应用均需要通过应用商店审核，也解决了传统 EXE 软件安装繁琐、权限过高、卸载残留的弊病。今天，少数派就为大家带来了 12 款优秀 UWP 应用，让你的 Windows 更加优雅实用。

Drawboard PDF
-------------

如果你有 Surface，想必对预装的 Drawboard PDF 不会陌生。这款应用专为 Windows 平板打造，可以对 PDF 文件进行批注、高亮、笔记、编辑等操作，与 Surface Pen 配合更是如鱼得水。你还可以将常用书籍作为磁贴固定在开始屏幕，或者创建新文档自由书写，是上课做笔记的不二佳选。

![](.evernote-content/878025FE-A9F2-4802-AC37-9DF77BD71DE9/93218906-9DF3-4A33-95EC-8DFEF23B531B.png)

你可以在 Windows 应用商店以 67 元的价格购买 [Drawboard PDF](https://www.microsoft.com/zh-cn/store/p/drawboard-pdf/9wzdncrfhwqt "Drawboard PDF")，应用还提供专业版订阅，包含PDF 文档合并整理、精确测量、Surface Dial 扩展支持等，每年 169 元人民币。

Sketchable
----------

如果你是为了绘画而入手 Surface 的话，一定不能错过 Sketchable 这款应用。它提供了数种画笔工具及丰富的自定义选项，同时支持通过 Surface Dial 进行调色等操作，还可以将画作导出为包括图层的 PSD 文件，方便进一步处理。

![](.evernote-content/878025FE-A9F2-4802-AC37-9DF77BD71DE9/F8B56E2A-B0A6-4E0A-B127-AB9398C68101.png)

Sketchable 也是 Windows 应用商店首页推荐的常客，事实上，今年 9 月我参加微软 Surface 新品体验会时，现场演示环节所使用的应用就是它。

![](.evernote-content/878025FE-A9F2-4802-AC37-9DF77BD71DE9/7E36D354-3177-40F0-A8BF-2B8B3667D957.jpg)

你可以在 Windows 应用商店免费下载 [Sketchable](https://www.microsoft.com/zh-cn/store/p/sketchable/9wzdncrfjcjw "Sketchable")，部分功能需要通过内购解锁。

泼辣修图
----

作为国产修图应用的翘楚，泼辣修图曾两度获得 App Store 年度最佳应用的殊荣。你可以利用它为照片添加滤镜、文字、蒙版，调整色彩、曝光、畸变等。其操作简单，功能强大，同时提供人脸识别、批量处理及诸多内建教程，是一款不可多得的轻量级专业图片后期处理应用。

![](.evernote-content/878025FE-A9F2-4802-AC37-9DF77BD71DE9/36D32199-3A7C-4E28-88CC-C28427D054FF.png)

你可以在 Windows 应用商店免费下载 [泼辣修图精简版](https://www.microsoft.com/zh-cn/store/p/%E6%B3%BC%E8%BE%A3%E4%BF%AE%E5%9B%BE-%E7%B2%BE%E7%AE%80%E7%89%88/9nblggh6bgx8 "泼辣修图精简版")，或者以 134 元的价格直接购买 [专业版](https://www.microsoft.com/zh-cn/store/p/%E6%B3%BC%E8%BE%A3%E4%BF%AE%E5%9B%BE/9nblggh1npnc "专业版")。如果你之前曾在其他平台上购买了专业版，可以通过登录账户免费解锁全部功能。在 [官网](http://www.polaxiong.com/ "官网") 上，你还可以找到 iOS、Android、macOS 等版本下载链接。

需要注意的是，泼辣修图将于 2018 年全面转向订阅制，但在此之前购买专业版的用户均可以免费获得 248 年会员资格，当成传家宝都没问题，想下手的同学要抓紧时间了。

此外，少数派也曾多次评测过泼辣修图，欢迎 [点此](https://sspai.com/app/%E6%B3%BC%E8%BE%A3%E4%BF%AE%E5%9B%BE "点此") 阅读。

黄油相机
----

同样是一款国产图片处理应用，黄油相机更侧重于多样化的图片加字功能。它具有丰富的中文字体模板，无论是小清新还是大片范，都能轻松满足。此外，黄油相机还提供社区功能，你可以从其他大触的图片中吸取灵感，也可以直接套用，创造属于自己的完美贴图。

![](.evernote-content/878025FE-A9F2-4802-AC37-9DF77BD71DE9/81B9E514-D8ED-46B9-8D10-C0A0755FAFA0.jpg)

你可以在 Windows 应用商店免费下载 [黄油相机](https://www.microsoft.com/zh-cn/store/p/%E9%BB%84%E6%B2%B9%E7%9B%B8%E6%9C%BA/9nblggh5fp0h "黄油相机")，应用同时提供 iOS 和 Android 版，你可以在 [官网](https://www.bybutter.com/ "官网") 下载。

少数派同样对黄油相机做过诸多评测，你可以在 [这里](https://sspai.com/app/%E9%BB%84%E6%B2%B9%E7%9B%B8%E6%9C%BA "这里") 阅读。

Todoist
-------

Todoist 是一款全平台任务管理应用，也是我派编辑部的御用团队协作工具之一。它提供标签、列表、过滤器、优先级等多种任务处理体系，同时支持通过自然语言添加任务，提高使用效率。

![](.evernote-content/878025FE-A9F2-4802-AC37-9DF77BD71DE9/05C39713-19AE-4BF9-88D9-CAF71F4FAFF1.png)

你可以在 Windows 应用商店免费下载 [Todoist](https://www.microsoft.com/zh-cn/store/p/todoist-to-do-list-and-task-manager/9nblggh1rl1k "Todoist")，目前仍处于公测阶段，但功能已足够完善。在其 [官网](http://www.todoist.com/ "官网")，你还可以找到 iOS、Android、macOS、Chrome 等平台下载链接。

此外，少数派也曾对 Todoist 做过许多深度评测，你可以在 [这里](https://sspai.com/app/Todoist "这里") 了解。

如果你想更系统掌握 Todoist 及任务管理技能，可以订阅少数派出品的[「用更现代的方式做任务管理」](https://sspai.com/series/1 "「用更现代的方式做任务管理」")付费栏目。

Microsoft To-Do
---------------

如果你是轻度用户，Todoist 可能就显得有些太复杂了，而 Microsoft To-Do 则可能博得你的青睐。它由 Wunderlist 原班人马打造，保持简洁易用的同时融入微软人工智能技术，可以智能建议待办与提醒事项。由于采用了 Outlook Exchange 接口，只需要登录微软账户，你就可以与 iOS 和 Android 等系统的原生提醒事项协作，不必再安装其它应用。

![](.evernote-content/878025FE-A9F2-4802-AC37-9DF77BD71DE9/B14C806C-1E5B-4242-9FC9-9A408391B7EA.png)

你可以在 Windows 应用商店免费下载 [Microsoft To-Do](https://www.microsoft.com/zh-cn/store/p/microsoft-to-do/9nblggh5r558 "Microsoft To-Do")，应用同时提供 iOS 和 Android 版，可以在 [官网](https://todo.microsoft.com/zh-cn "官网") 下载体验。

8 Zip
-----

8 Zip 是一款设计简约、功能强大的压缩软件。你可以用它解压 ZIP、RAR、7Z 等压缩文件，并在应用内直接预览常见格式文件。美中不足的是，由于 UWP 机制限制，你只能在应用内创建压缩文件，不能通过右键菜单直接访问。

![](.evernote-content/878025FE-A9F2-4802-AC37-9DF77BD71DE9/E6108DE1-12DD-4498-A109-7AC0A096BB2D.png)

你可以在 Windows 应用商店购买 [8 Zip](https://www.microsoft.com/zh-cn/store/p/8-zip/9wzdncrfhwb8# "8 Zip")，或免费下载 [8 Zip Lite](https://www.microsoft.com/zh-cn/store/p/8-zip-lite-unpack-rar-zip-7z-for-free/9wzdncrfjb33 "8 Zip Lite")。

Newsflow
--------

Newsflow 是一款兼顾颜值与功能的 RSS 阅读应用，你可以利用它直接搜索常见订阅源，或手动添加想阅读的内容。你可以通过分类管理所有订阅源，分别设置同步间隔、通知、可读性等选项。对于喜欢的文章，你可以添加至稍后阅读列表或收藏夹，方便在之后找到。你也可以通过 OPML 格式文件导入在其他RSS 应用中的订阅列表，或者备份在 Newsflow 中的相关设置。

![](.evernote-content/878025FE-A9F2-4802-AC37-9DF77BD71DE9/D8A05430-EE94-4EFC-9E37-D10010E13BF2.png)

你可以在 Windows 应用商店下载 [Newsflow](https://www.microsoft.com/zh-cn/store/p/newsflow/9nblggh58s5r "Newsflow")，应用完全免费。如果你同时使用 Feedly，可以购买 [Newsflow + Feedly 扩展](https://www.microsoft.com/zh-cn/store/p/newsflow-feedly-preview/9nh3f6hv57lb "Newsflow + Feedly 扩展")，同步你的相关设置。

Grover Podcast
--------------

作为一款播客应用，Grover Podcast 的设计与 Windows 10 默认音乐播放器 Groove 十分相似，降低了学习成本，给你贴近原生的体验。它采用了 iTunes 的 API，你可以分类浏览热门播客，搜索自己想要的内容，并在多设备间同步订阅列表。当播客有更新时，你可以通过系统提醒及时获知，也可以批量管理内容。

![](.evernote-content/878025FE-A9F2-4802-AC37-9DF77BD71DE9/8BE1EDBB-2FDA-4EA3-AA37-BD8D98953212.png)

你可以在 Windows 应用商店下载 [Grover Podcast](https://www.microsoft.com/zh-cn/store/p/grover-podcast/9nblggh6c4bc "Grover Podcast")，应用完全免费。

少数派此前也曾推荐过 Grover Podcast，你可以点此阅读：[在 Windows 上听播客，现在你有了一个好选择：Grover Podcast | App+1](https://sspai.com/post/41929 "在 Windows 上听播客，现在你有了一个好选择：Grover Podcast | App+1")

必应词典
----

作为微软的亲儿子，必应词典可谓是功能最全面、适用性最广泛的 UWP 词典应用。除了比较基础的单词查询、语段翻译、生词本助记外，必应词典还提供原文赏析、单词挑战、口语练习、词汇量测试等众多进阶功能，涵盖英语学习的方方面面。你还可以通过 Cortana 直接语音唤醒必应词典，并利用微软账户跨设备同步生词本。

![](.evernote-content/878025FE-A9F2-4802-AC37-9DF77BD71DE9/BBA0198A-F3F6-4ED3-A9AC-94356CCB9554.png)

你可以在 Windows 应用商店免费下载 [必应词典](https://www.microsoft.com/zh-cn/store/p/%E5%BF%85%E5%BA%94%E8%AF%8D%E5%85%B8/9wzdncrfhv6h "必应词典")，应用同时提供 [iOS](https://itunes.apple.com/cn/app/%E5%BE%AE%E8%BD%AF%E5%BF%85%E5%BA%94%E8%AF%8D%E5%85%B8/id633888638?mt=8 "iOS") 和 [Android](https://www.coolapk.com/apk/bingdic.android.activity "Android") 版。

网易云音乐
-----

网易云音乐想必不用我多介绍，作为国内最早发布 UWP 版的应用之一，其功能目前已非常完善，不亚于桌面版应用。尽管由于版权等原因，网易云音乐近来受到诸多诟病，但依然是国内综合体验最好的在线音乐服务之一，其用户歌单与推荐算法也是不少人的心头好。此外，网易云音乐也与微软达成了深度合作，让你能够通过 Cortana 直接调用 UWP 版网易云音乐进行听歌识曲，可以说是半个官方应用了。

![](.evernote-content/878025FE-A9F2-4802-AC37-9DF77BD71DE9/A172EB34-7BA4-4DF3-91D0-A8F5D16E4795.png)

你可以在 Windows 应用商店免费下载 [网易云音乐](https://www.microsoft.com/zh-cn/store/p/%E7%BD%91%E6%98%93%E4%BA%91%E9%9F%B3%E4%B9%90uwp/9nblggh6g0jf "网易云音乐")，当然，你也可以在其 [官网](https://music.163.com/ "官网") 下载 Win32 或其它版本应用。

MyerSplash
----------

看腻了一成不变的壁纸，不妨打开 MyerSplash，从 Unsplash 的精选图片里选出自己的最爱。你可以按照分类、赞数、最新等方式筛选壁纸，或者通过随机功能碰碰运气。看上哪一张照片后，你可以直接预览它在桌面或锁屏界面的效果、将其下载至本地或在线查看该摄影师的更多作品。在应用设置中，你还可以设置每天自动更换推荐壁纸，避免视觉疲劳。

![](.evernote-content/878025FE-A9F2-4802-AC37-9DF77BD71DE9/0E49BD0F-D996-4D41-8F26-F421928D19FA.png)

你可以在 Windows 应用商店下载 [MyerSplash](https://www.microsoft.com/zh-cn/store/p/myersplash-%E5%A3%81%E7%BA%B8/9nblggh4vcsn "MyerSplash")，应用售价 7 元，同时提供 [Android 版本](https://play.google.com/store/apps/details?id=com.juniperphoton.myersplash "Android 版本")。

MyerSplash 开发者也曾在少数派分享过他的开发历程，你可以在这里阅读 [MyerSplash UWP 背后的故事](https://sspai.com/post/34328 "MyerSplash UWP 背后的故事")。

Unsplash 近日也推出了官方 macOS 及 Android 客户端，你可以在此了解更多：[免费高质量图片站 Unsplash 终于出了官方客户端，它是你的菜吗？](https://sspai.com/post/41128 "免费高质量图片站 Unsplash 终于出了官方客户端，它是你的菜吗？")
