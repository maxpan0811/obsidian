# 杨东平：上海教育怎么啦？

杨东平：上海教育怎么啦？
------------

*2017-01-21**杨东平*[杨东平](http://mp.weixin.qq.com/s?__biz=MzI2NDU3OTE5Mg==&mid=2247483662&idx=1&sn=c842369383c63f007f9dceceb9620ddd&chksm=eaab3019dddcb90f5d5b05bf8705e440fa4e3a5de62a9b431db90cff7bd62b01658a8be68fc3&mpshare=1&scene=1&srcid=0121PaFtj9j9wQNShreOqRiM##)

今年上海“两会”的最大热点，不是杨雄辞职，也不是反腐、治霾；而是由市委书记韩正自曝家丑，对学生课业负担太重、课外补习乱象丛生的批评，激起市民的强烈共鸣和对上海教育的反思。

![](.evernote-content/26CD221C-8C01-4C0F-A34B-ACF7467F1761/FCFD05FB-B88D-440B-A2ED-99190FE87417.jpg)

韩正在上海两会上表示：孩子们太苦了，要真正减轻课业负担

上海教育由于在OECD组织的PISA测试中，于2009年、2012年两次夺冠，成为中国基础教育亮丽的名片，吸引了全世界的目光。在2014年OECD组织的“教师教学国际调查”（TALIS）中，上海教师又获得优异的表现。然而，在闪闪发光的表征之下，我经常听到来自上海的学生、家长、老师、大学教授令人不安的消息。上海基础教育卓越、优秀的整体评价，与中小学生的日常生活、家长的切身感受似乎存在巨大的反差。这种感觉，从近期的一系列调查中得到了印证。

据2015年11月，网易教育频道发布基于2000万用户数据的《全国中小学生学习压力调查》，中小学生平均每天写作业3小时，是全球平均数的2倍。写作业到23点以后的比例，小学生为18.2%，初中生为46.3%，高中生为87.6%。这一情况是我们所熟悉的，有意思的是它据此做了一个“中小学生熬夜排行榜”，前10名依次为贵州、青海、西藏、广西、甘肃、新疆、云南、重庆、安徽、上海。多为教育资源匮乏、教育水平较落后的地区，而教育最发达的上海居然进入了前十，不能不令人深感震惊。[1]

![](.evernote-content/26CD221C-8C01-4C0F-A34B-ACF7467F1761/A18463DF-373C-4862-B972-BE89B807514F.jpg)

这一现象在2012年的PISA测试中也被揭示。上海事实上是获得了两个世界第一：学业成就和学习负担。上海学生平均每周校内上课时间为28.2小时，在65个国家（地区）中位于第9位，好像不算太多；但平均每周作业时间13.8小时，为全球第一。OECD国家平均为5小时多，日本4小时多，芬兰最低为3小时多。加上校外辅导和私人家教，上海学生每周校外学习时间17小时左右，远远高于OECD的7.8小时的均值。[2]

![](.evernote-content/26CD221C-8C01-4C0F-A34B-ACF7467F1761/20B7C08D-05D8-433F-A070-62EB98232F8A.jpg)

另外一些调查也可作为佐证。

据2017年1月三好网调查，在京沪两地，中小学生参加课外辅导达到74%以上。[3]

据2017年1月10日，360儿童手表发布基于400多万用户的“儿童成长状况大数据”，在全国主要城市中，北京与上海的孩子们最悲催，竞争也最激烈。周末时间被各类培训班所占据的北京孩子为41%，上海为42%。深圳第三，为33%，其它大城市大致在20%上下。这一数据反映的主要是小学生的生存状况。[4]

![](.evernote-content/26CD221C-8C01-4C0F-A34B-ACF7467F1761/84077DC9-2AE7-4C40-8226-55BD7628969C.jpg)

高负担与高成绩相伴，是上海教育的显著特点，这必然影响学生对学校教育的感受。据2016年11月21世纪教育研究院与腾讯教育合作的“家长眼中的教育”调查，总体来看，家长认为孩子目前所在学校应试教育倾向“非常严重”（30.63%）和“比较严重”（29.15%）的合计超过半数，应试教育严重程度总体均值为3.72（满分为5，分值越大表示应试教育越严重），上海为3.67，北京为3.54，均低于均值。家长认为孩子对目前学校的喜欢程度，平均水平为3.61（满分为5，数值越大表示喜欢程度越高），有14个省市高于平均水平，北京为第三。令人惊讶的是上海均值为3.58，居第16位，江苏为3.57，居第17位，均低于总体平均水平。家长对孩子所在班级老师的满意度评价，总体均值为3.43（满分为5，数值越大表示家长越满意）。满意度在平均水平以下的有20个省，北京为3.36，上海为3.33，居于后五位之列。[5]可见在学生和家长眼中，“好的教育”、“人民满意的教育”绝不仅仅意味着考试和分数。

这种调查自然有待深入，更加科学化；但我们已可以据此勾画出中小学应试教育的基本图景：应试教育由来已久，于今为烈。最为严重的，是教育资源相对匮乏的西部地区，如青海、甘肃、宁夏、贵州、云南、山西等等；就学业负担、课外补习热而言，在大城市中以上海、北京最为严重，而上海则为最烈。

正视问题，承认问题，是解决问题的前提。韩正书记已经带了一个好头。上海市教委将成立“减负领导小组”进行专项治理。如何治愈这一中国教育的癌症，当然要复杂和困难的多。这是一场需要全民参与的攻坚战，21世纪教育研究院已将此作为重要任务，参与“作战”。
