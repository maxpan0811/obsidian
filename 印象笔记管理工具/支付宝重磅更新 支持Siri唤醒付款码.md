# 支付宝重磅更新 支持Siri唤醒付款码

---

你 iPhone 上的 Siri 用得多吗？是不是很多锋友都和小编一样只会用 Siri 设闹钟？不过从今天开始，Siri 将会新支持一个很实用的功能：支付宝 App 升级到最新版之后，能够通过 Siri 唤起付款码了！

[![](.evernote-content/4A7460A0-ACAA-4AEA-A0AA-5CAD8C7FCC68/1E853947-B5A3-4E3C-983E-C6819631F284.jpg)](https://resource.feng.com/resource/h062/h29/img201804181532170.jpg)

近日，支付宝更新了10.1.20 版本，在更新日志中，支付宝列出了“出境”和“财富”两个新内容，分别提供了境外旅游推荐、财富管理服务。但是对 iOS 用户来说，还有一个小福利：Siri 已经可以支持语音唤起付款码了。

用户唤醒 Siri，下达“打开支付宝付款”、“打开支付宝支付”、“支付宝付款”、“支付宝支付”、“打开支付宝付款码”或“支付宝付款码”等语音指令，都能够调用出支付宝的付款码。比起之前数次点击，语音操作无疑让这个高频操作变得更加方便快捷了。

[![](.evernote-content/4A7460A0-ACAA-4AEA-A0AA-5CAD8C7FCC68/804DAEDF-39A5-471C-8AE7-7D29C4707B67.gif)](https://resource.feng.com/resource/h062/h29/img201804181532330.gif)

和其他语音助手支持各式各样的第三方 App 不一样，可能出于安全等因素的考量，苹果此前对第三方开发者开放 Siri 的 API 接口非常谨慎，所以 Siri 的应用场景更多只是在日历、闹钟等预装 App 中。

不过从 iOS 10 开始，苹果将 Siri 的 API 向打车、通讯、照片搜索、支付、网络电话以及锻炼这六类 App 开放，并且在 iOS 11 中又进一步开放了一些类别。所以现在，支付类 App 支付宝也开始支持 Siri 的语音操作，相信支付宝在这一过程中也和苹果有过不少的交流。
