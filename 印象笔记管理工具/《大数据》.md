# 《大数据》

---

很久没有想为一本书写读后感的冲动，直到阅读了涂子沛的《大数据》。

8月28号从当当买了《大数据》，9月6号看完，总共花了10天的时间，这种厚度的书对我来说属于神速，因为期间在9月2号和刘文杰（老搜）在深圳银湖组织了[深圳网站分析](http://www.szwebanalytics.com "深圳网站分析")的线下聚会，特别感谢鹏运国旅的董军董先生无私奉献了宝贵的场地，坐下了接近80人。同时9月5号晚上接受了网站分析公会的邀请，参加了多贝网络课堂《[网站分析销售渠道归因的四种模型](http://www.szwebanalytics.com/four-web-analytics-attribution-model.html "网站分析销售归因模型的四种类型")》的演讲。

我想能够这么快看完除了本身接受信息的速度有所提高以外，更重要的是《大数据》足够吸引的内容。吸引点首先出现在引言，引言中复旦大学计算机研究所所长施伯乐说：“**关于数据的知识，将成为个人知识结构中的必备要素和基础。**”看到这句话，感觉这几年没有瞎忙，即使方向有偏，起码打好了根基，于是对接下来的内容充满了期待。

跟本书的划分（上篇帝国风云,中篇法则博弈,下篇公民故事,外篇天下趋势）不同, 我将整本书的框架梳理如下：

一、数据分析与个人
=========

如今的社会，正如美国管理学家、统计学家爱德华戴明所说的，“**我们相信上帝。除了上帝，任何人都必须用数据来说话。**”

很多时候，注重数据说明一个人的理性，理性多了做事就靠谱，不会像胡适讽刺的差不多先生那样，什么事情都不了了之。

在书中很惊讶地看到沃尔玛首席信息官罗林福特所说的一句话“每天早上一醒来，我就要问自己：“**怎么才能让数据流动得更好、管理得更好、分析得更好？**””，不说实践是否真实这样，觉得有这种境界都是值得赞誉的，拿来对比一下自身，惭愧得不行。

二、数据分析与企业
=========

时代发展瞬息万变，在大数据的环境中，数据分析提高了商业运营的效率，能够帮助企业很好地总结过去，并且改善预测未来的能力。决策不再是拍脑袋，而是看了数据后拍桌子：“好，就这么干。”

最近一个企业家的话给我印象很深，他是董军先生，发生在9月2日线下聚会，他说：“**社会发展很快，传统企业要转行，得往互联网走**”，我想互联网除了是一个新的媒介以外，协助企业在互联网中走得越来越远的是的对数据的收集、分析、挖掘和应用，有这样的新时代意识，企业一定会越走越好。

三、数据分析与社会
=========

众所周知，开放是现代社会的主要关键字之一，开放不仅仅体现在文化，经济，意识层面，开放还体现在数据，数据的开放更能看出一个政府是否足够民主，足够坦诚。奥巴马开放了Data.gov，美国人民可以在这个网站查找到很多有价值的信息，我觉得这种举动是很好的。因为很多时候矛盾就是由信息不对称产生的（信息不对称是由斯蒂格勒提出的，掌握信息多的人，在社会竞争中处于有利的地位，而信息贫乏的人，则处于不利的地位），政府数据的开放，意味着让信息自由流动，建立彼此完全的信任。

反观国内，最近陕西安监局长“表哥”事件沸沸扬扬，财务迟迟没有公布，我想如果经得起考验的话，数据不那么惊人的话，早就和网民来了个坦白、深切的互动了。

四、数据分析的未来
=========

沃尔玛的尿布和啤酒是数据分析和挖掘的经典案例，以前总觉得数据挖掘就是一堆的二进制数据在计算机中大量流动，规模吓人。现在觉得其实最高深的数据挖掘应该是和生活走得最近的。包括以下这个案例，美国每次飓风来袭，袋装小食品“Pop-Tarts”的销售都会明显上升，因为飓风常伴随停电，而这种小食品在停电时吃起来很方便，不用煮。

除了数据挖掘，云计算也是一个很热的词，说是数据分析未来的载体一点都不过分，很喜欢昆德拉的比喻：“曾经每一个家庭、村落、农庄都必须有自己的水井。今天，你仅仅打开水龙头，干净的水就通过公共供水管道输送给我们，水龙头可以随时打开或者关上，水就相当于云。”

五、总结
====

这本书除了具体的事例，名人名言，《大数据》带来的更多是视野，融合了技术、政治和商业，值得读。
