# Installing TotalSpaces2 v2.5.4 and above

---

How to install TotalSpaces2
===========================

These instructions apply to [TotalSpaces2 v.2.5.4 and above](https://totalspaces.binaryage.com/changes-beta) on OSX 10.11 and above.

TotalSpaces2 requires temporary changes to your security settings to be installed. [Read more about it here.](https://totalspaces.binaryage.com/sip-details)

* [Step 1 — Turn off System Integrity Protection](https://totalspaces.binaryage.com/sipsettings#step1)
* [Step 2 — Install and run TotalSpaces2](https://totalspaces.binaryage.com/sipsettings#step2)
* [Step 3 — Turn System Integrity Protection back on](https://totalspaces.binaryage.com/sipsettings#step3)

Step 1
======

Turn off System Integrity Protection (SIP)
------------------------------------------

This requires rebooting your maching into recovery mode.

Restart your machine, holding   ![](.evernote-content/4C6A70FD-ED9F-4D5C-AC6A-6BBD1BF42B5C/FF4295B1-88C4-436C-8539-3FE0507C2E75.png)   down until the Apple logo appears.

Then select Terminal from the Utilities menu. It looks like this:

![](.evernote-content/4C6A70FD-ED9F-4D5C-AC6A-6BBD1BF42B5C/2104E693-6DDD-4D89-82F7-BFFD30656FBD.png)

In the window that opens, type
`csrutil disable`
and press return. This turns off System Integrity Protection so that TotalSpaces2 can be installed.

![](.evernote-content/4C6A70FD-ED9F-4D5C-AC6A-6BBD1BF42B5C/B10AEA92-F25D-4651-BC78-10B7236AF0AD.jpg)

Choose restart from apple menu (top left).

Step 2
======

Install and run TotalSpaces2
----------------------------

Double click on the DMG file you downloaded with TotalSpace2 in it. Drag TotalSpaces2 into your Applications folder.

![](.evernote-content/4C6A70FD-ED9F-4D5C-AC6A-6BBD1BF42B5C/93DE48AA-8931-4C42-9BF2-1B9AEF75EAAE.png)

Run TotalSpaces2 from your Applications folder. It will ask for your password in order to install its plugin component.

![](.evernote-content/4C6A70FD-ED9F-4D5C-AC6A-6BBD1BF42B5C/843888E0-F023-407C-B04A-FAA31CF7C09C.png)

When the plugin has been installed successfully, TotalSpaces2 will tell you that you can turn SIP back on.

![](.evernote-content/4C6A70FD-ED9F-4D5C-AC6A-6BBD1BF42B5C/89BEB4AF-11FB-4C64-AA31-016D1B78EA6B.png)

Step 3 (optional)
=================

Turn System Integrity Protection back on
----------------------------------------

Once again restart your machine, holding   ![](.evernote-content/4C6A70FD-ED9F-4D5C-AC6A-6BBD1BF42B5C/FF4295B1-88C4-436C-8539-3FE0507C2E75.png)   down until the Apple logo appears.

Then select Terminal from the Utilities menu.

In the window that opens, type
`csrutil enable`
and press return. This turns on System Integrity Protection so your machine is fully protected.

![](.evernote-content/4C6A70FD-ED9F-4D5C-AC6A-6BBD1BF42B5C/57CFCF93-896E-498E-9E85-B6355EDB10DD.jpg)

Choose restart from apple menu to complete.
