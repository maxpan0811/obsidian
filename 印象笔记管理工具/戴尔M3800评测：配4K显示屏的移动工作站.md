# 戴尔M3800评测：配4K显示屏的移动工作站

　　戴尔发布第一代 Precision M3800 的时候，我们就认为它是苹果 MacBook Pro 的唯一劲敌，而现在，戴尔推出了新款 Precision M3800，除了搭配 4K 显示屏之外，其他硬件配置也得到了升级，当然，它的竞争对手也没有停下脚步，比如 MacBook Pro 新增了 Force Touch 触控板等等。  
  
　　日前，外媒 Trustedreviews 对戴尔新款 Precision M3800 移动工作站进行了一次详细的评测，一起来看看它能带来怎样的使用体验吧。  
  
　**外观设计**  
  

![](.evernote-content/74C1B2E0-2EA6-493B-9B1A-9FDDF8C666EA/634736EF-A7F5-46EB-B250-34BB2A9B3C54.png)

  
　　从造型方面来看，新款 Precision M3800 跟前代产品相比并没有发生任何改变，也许有人会觉得戴尔很难，但是这并不妨碍 Precision M3800 继续保持一流的品质，比如时尚的铝制外壳以及由碳纤材质打造的底座等等。  
  
　　新款 Precision M3800 的机身重量为 1.8 千克，厚度为 19 毫米，这一点跟 MacBook Pro 非常接近，但是比惠普 ZBook 15 要苗条许多，当然，这也跟产品的定位有关，后者定位于一款传统的办公室笔记本。  
  

![](.evernote-content/74C1B2E0-2EA6-493B-9B1A-9FDDF8C666EA/1C40A5C6-1B6E-418E-ACD1-E9B1DC55C8CF.jpg)

  
　　而在键盘设计方面，黑色按键、扁平化的风格跟 MacBook Pro 非常相似，并且实现了非常出色的按键手感和响应速度，同时还配有背光功能。不过，戴尔并没有给它搭配额外的数字键盘。虽然触控板还无法跟苹果的产品相比，但是使用起来也非常方便。  
  
　　另外，除了敲击键盘的时候会出现上盖轻微晃动之外，新款 Precision M3800 并未出现其他的问题。如果有需要的话，你可以拧下 12 颗螺丝，对内存、存储、无线芯片以及电池等部件进行升级，不过，这款产品内部并没有多余的空间用于添加额外组件。  
  
　　**硬件和性能**  
  

![](.evernote-content/74C1B2E0-2EA6-493B-9B1A-9FDDF8C666EA/D6330378-6936-4380-82B8-0F4C7A15F2F6.jpg)

  
　　很显然，15 英寸的 4K 显示屏是新款 Precision M3800 变化最大的地方，其他的配置也有所升级，比如原来的 2.2GHz 酷睿 i7-4702HQ 处理器被 2.3GHz 酷睿 i7-4712HQ 所取代，Lite-On SSD 也换成了三星 SM841n，读写速度达到了 518MB/s 和 452MB/s。  
  
　　虽然戴尔没有采用最新的 Skylake 架构处理器，但是，戴尔新款 Precision M3800 配置了性能相当不错的英伟达 Quadro K1100M 显卡，即使面对高端图形软件也能从容应付。  
  
　　这款移动工作站拥有 3 个 USB 3.0 端口、1 个 Thunderbolt 端口、1 个 HDMI 输出端口、1 个 SD 卡插槽以及 1 个耳机插孔，支持蓝牙 4.0 和千兆以太网，连接性能方面还算全面。  
  
　　外媒 Trustedreviews 的测试人员表示，新款 Precision M3800 的扬声器能够实现非常出色的音质表现，音量、层次感以及低频方面都跟 MacBook Pro 有得一拼。  
  

![](.evernote-content/74C1B2E0-2EA6-493B-9B1A-9FDDF8C666EA/C371F8CD-5BD7-405E-A2E1-9548AAAE5B90.jpg)

  
　　不过，戴尔新款 Precision M3800 的电池续航比苹果 MacBook Pro 要差上不少，前者在 40% 屏幕亮度的情况下进行网页浏览仅坚持了 3 小时 44 分钟，而后者的成绩长达 10 个小时之久。  
  
　　在购买这款产品的时候，消费者可以根据自己的需要配置对应的硬件，还可以选择 Windows 7 专业版、Windows 8.1 专业版以及 Ubuntu 14.04 等系统版本，不过，该产品暂不支持 Windows 10 系统。  
  
　　另外，测试人员发现，酷睿 i7-4712HQ 处理器在压力测试时的温度可以达到 85 摄氏度，当 CPU 和图形核心一起测试时，其温度更是高达 98 摄氏度，虽然其工作性能未受影响，但是散热风扇会造成明显的噪音。  
  
　　**总结**  
  
　　综上所述，戴尔新款 Precision M3800 的配置、性能等方面都较前代产品有了明显的提升，消费者通过它能够从容应对家庭、办公室等场合的日常使用，虽然它在降噪和续航等方面还存在不足，但是综合性能、性价比以及消费者对 Windows 和 OS X 系统的使用习惯等方面来看，新款 Precision M3800 仍可以跟 MacBook Pro 正面对抗。
