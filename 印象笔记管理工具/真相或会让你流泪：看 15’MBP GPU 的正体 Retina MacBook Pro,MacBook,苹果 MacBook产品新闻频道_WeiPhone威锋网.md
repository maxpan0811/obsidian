> 15 英寸 Retina MacBook Pro的显卡是AMD 的 Radeon R9 M370X，它到底是什么鬼？

[![](.evernote-content/42B34206-0E35-4116-9FF4-7BCFBC255AA2/165159AC-398B-42A7-8EB0-5B3AA69BF58A.gif)](http://bbs.feng.com/aodoo3/click.php?n=a6444c89&zoneid=128)

以下为文章全文：

[![](.evernote-content/42B34206-0E35-4116-9FF4-7BCFBC255AA2/05E2953D-6E28-4B5F-9928-D4AE5C2559FD.jpg)](http://resource.feng.com/resource/h055/h16/img201505251318430.jpg "真相或会让你流泪：看 15’MBP GPU 的正体")

  
　　上周苹果惊喜更新 2015 年中 15 英寸 Retina MacBook Pro。在这款新产品中，虽然新设备固态硬盘提升很猛，仍旧使用英特尔 Core i7 2.2GHz 四核处理器，但是在高端机型中苹果更换了设备所使用的 GPU ，将此前使用的英伟达 GeForce GT 750M 换成了 AMD 的 Radeon R9 M370X。因为在 AMD 最近发布的 2015 Radeon M300 系列更新产品线中，AMD 并没有公开过 M370X 零件号，因此很多人疑惑 M370X 到底是什么呢？  
  
　　在新款 Retina MacBook Pro 刚更新上市的时候，业界已经猜测这款 GPU 可能是AMD Cape Verde GPU，因为 M375 也搭配这款 GPU，M375 的性能较前辈 M370X 稍强。目前已经有用户拿到苹果的新款 15 英寸 Retina MacBook Pro，根据这些用户提供的信息我们可以进一步确定新设备中使用的 GPU。  
  

[![](.evernote-content/42B34206-0E35-4116-9FF4-7BCFBC255AA2/8575D2B8-26EE-4B01-B885-25237F1949C6.png)](http://resource.feng.com/resource/h055/h16/img201505251318431.png "真相或会让你流泪：看 15’MBP GPU 的正体")

  
　　根据 Reddit 用户 ootan 公布的 Retina MacBook Pro 的 System Profiler（系统属性）截图，我们可以确定这款设备中使用的 GPU 就是 AMD 的 Cape Verde （佛得角）GPU：显卡设备 ID 是 0x6821——Cape Verde 核心在笔记本上以前就用过这个 ID。  
  
　　Cape Verde 是 AMD 的第一代 GCN 1.0 GPU，早在 2012 年就已经发布了。发现苹果使用的竟然是三年前的Radeon HD 7770，知道真相的用户眼泪流下来，惊呼苹果和AMD真是太凶残。  
  
　　和 GT 750M 中的 GK107 GPU 相比，它是一款更大更强劲（至少在桌面设备中，更耗电）的 GPU。和 GK107 相比它是专门为那些强劲的设备设计的。至于苹果为什么选择使用这款图形处理器，答案大概只有苹果、AMD 以及英伟达知道。而在用了 GK107 这么多年之后，相信他们也是愿意升级到更强的 GPU 上了。  
  
　　虽然目前我们没有掌握 M370X 在笔记本中的表现，但是至少在桌面设备中，GPU的性能不受发热影响，Cape Verde 的表现还是不错的。而在 Retina MacBook Pro 中，因为有发热问题的限制，所以它的性能可能无法与桌面设备中的匹敌，不过我想应该也不会差到哪里去的。那么问题来了：现在苹果在设备中使用性能更强的部件，当该 GPU 活跃时它是否会对 Retina MacBook Pro 的电池续航产生影响呢？这个问题轻易回答不了。  
  

[![](.evernote-content/42B34206-0E35-4116-9FF4-7BCFBC255AA2/A9305B5E-6858-420C-94BF-1716E84E45F6.png)](http://resource.feng.com/resource/h055/h16/img201505251318432.png "真相或会让你流泪：看 15’MBP GPU 的正体")

  
　　另外这里还有一张来自 Boot Camp 下的 GPU-Z 截图，Cape Verde 的代号清晰可见，640 个流处理器的规格更证明了它就是 Radeon HD 7770。同时也确认了其他规格：核心频率为 800MHz，显存容量 2GB GDDR5、有效频率4500MHz。位宽 128-bit、带宽 72GB/s。  
  
　　Cape Verde 是第一批 28nm 工艺、GCN 架构显卡之一，定位主流领域，其中的相对高端型号HD 7770此后就多次改头换面：桌面上先是变成HD 8760，然后又成了 R7 250X，而笔记本上的 R9 M260/M260、R9 M375 其实都是它。即便如此，单从性能上看，R9 M370X 还是可以接受的，至少比之前的 GT 750M 提升巨大，理论上可达两倍。  
  
　　上周更新的15 英寸 Retina MacBook Pro 运行 2.2 GHz 四核 Intel Core i7 处理器，可加频至 3.4 GHz，16GB 内存，256GB 闪存，Intel Iris Pro显卡，起售价为 1,999 美元（RMB 14288）；另外高配版本运行2.5 GHz四核 Intel Core i7 处理器，可加频至 3.7 GHz，16GB 内存，512GB 闪存，AMD Radeon R9 M370X 显卡，售价 2,499（RMB 17988），预装 OS X Yosemite 系统。
