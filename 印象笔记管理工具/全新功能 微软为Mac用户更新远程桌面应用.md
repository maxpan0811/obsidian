# 全新功能 微软为Mac用户更新远程桌面应用

---

[![](.evernote-content/7F577D09-BC83-4401-A0F3-657B517AB913/5F0162F2-41C1-414F-8302-16206E493E2E.png)](http://www.feng.com/iPhone/news/2015-08-12/New-function-Microsoft-update-remote-desktop-applications-for-Mac-users_621477.shtml)

![](.evernote-content/7F577D09-BC83-4401-A0F3-657B517AB913/5F0162F2-41C1-414F-8302-16206E493E2E.png)

  

威锋网讯，微软近日为 Mac 的用户推出了一款新的 Remote Desktop（远程桌面）应用，目前处于 Beta 版阶段。本次的更新为用户带来了全新的用户界面和一些新的功能，用户今天就可以下载试用，并给出你的反馈。

Remote Desktop Beta 版最大的更新就是其全新的用户界面，给人的感觉就是非常简洁易用，用户还注意到了微软对于账户管理方面的改进，并且在遥控过程中可以同时使用 CMD 和 CTRL 的快捷方式。

也就是说，如果你是 Mac 的用户的话，就不再需要担心在进行复制、剪切和粘贴等操作的时候，要思考 CMD 键和 CTRL 键的切换。微软还承诺它将支持多个显示屏，剪贴板跳转、Remote Desktop Gateway。

微软鼓励用户在今天使用这款应用，并给出他们的使用反馈。不过值得一提的是，Beta 版在功能上和正式版相比会有所缺失，所以可能会出现一些 bug 或者是性能方面的问题 。

微软的 Remote Desktop for Mac 可以让用户来控制 Windows 或者 Mac 设备，从而让用户远程完成工作。
