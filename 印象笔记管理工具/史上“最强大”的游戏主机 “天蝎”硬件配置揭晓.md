史上“最强大”的游戏主机 “天蝎”硬件配置揭晓

北京时间2017年4月6日晚上9点，Eurogamer 旗下著名知名硬件技术评测机构 Digital Foundry 独家公布了微软最新主机 Xbox Scorpio，即代号为“天蝎”的新主机最新情报。

![](.evernote-content/CD2B040D-7550-49B4-86D4-E212222B6C73/127A8C42-C679-4D2A-A481-0AF49B829EE3.jpg)

拥有原生 4K 游戏运行能力，最高 6Teraflops 的运算性能，320GB/s 内存带宽，8核 CPU 处理器。被微软称为“史上最强大主机”的天蝎已经揭开了第一层面纱。

![](.evernote-content/CD2B040D-7550-49B4-86D4-E212222B6C73/B0302008-2465-4DE0-B365-1E7ABD869B16.jpg)

**“天蝎”的最终硬件配置！**

* CPU：定制8核心x86 CPU，2.3GHz
* GPU：40运算单元，1172MHz
* 内存：12GB GDDR5
* 内存带宽：326GB/s
* 硬盘：1TB
* 光驱：4K UHD蓝光

### 与Xbox One、Xbox One S的对比：

|  |  |  |  |
| --- | --- | --- | --- |
|  | “天蝎” | Xbox One S | Xbox One |
| CPU | 8个x86核心，2.3GHz | 8核心“美洲虎”，1.75GHz | 8核心“美洲虎”，1.75GHz |
| GPU | 40个处理单元，1172MHz | 12个GCN处理单元，914MHz | 12个GCN处理单元，853MHz |
| 内存 | 12GB GDDR5 | 8GB DDR3/32MB ESRAM | 8GB DDR3/32MB ESRAM |
| 内存带宽 | 326GB/s | 68GB/s，ESRAM为219GB/s | 68GB/s，ESRAM为204GB/s |
| 硬盘 | 1TB | 500GB-2TB | 500GB-2TB |
| 光驱 | 4K UHD蓝光 | 4K UHD蓝光 | 蓝光 |

　　

其中CPU部分仍然基于“美洲虎”（Jaguar架构），但频率进一步提升到2.3GHz。相比此前猜测的多运算单元低频率的设计，“天蝎”的最终GPU单元是40个运算单元，但运行频率高达1172MHz，远高于Xbox One的853MHz和PS4 Pro的911MHz。微软的工程团队还将Direct3D 12的大量指令运算单元直接集成到了GPU的内部，称为“GPU命令处理器”，以更快地处理Direct3D 12的高速API。内存的容量与带宽则与“天蝎”公布时通过概念主板推断的12GB 384bit位宽GDDR5基本相同。

![](.evernote-content/CD2B040D-7550-49B4-86D4-E212222B6C73/719E9904-062F-459B-BA18-15104473324E.jpg)

其他硬件方面，4K高清蓝光和Xbox One S的配置保持一致，音频芯片则原生支持杜比DTS和Atmos音效，最大的进步则是内置的视频编码芯片，它将利用高效的HEVC编码直接原生输出最高4K60fps的视频录像，并且不会造成任何的性能损失。

DF也公布了“天蝎”最终的SoC，其面积高达360平方毫米，晶体管数量达到了70亿，是一个不折不扣的巨无霸：

![](.evernote-content/CD2B040D-7550-49B4-86D4-E212222B6C73/CD0D8E39-2D83-4D17-965D-A6AD185870CF.jpg)

为了控制发热，微软的工程团队在散热部分采用了与如今高端PC显卡相同的封闭抽风式散热结构：

![](.evernote-content/CD2B040D-7550-49B4-86D4-E212222B6C73/E4E90A63-E9E5-4375-B8B6-D00E9BD06F0A.jpg)

![](.evernote-content/CD2B040D-7550-49B4-86D4-E212222B6C73/9AF77B6C-059A-4249-B4A6-C91F1832D2CE.jpg)

　　

整台主机的结构分解概念图：

![](.evernote-content/CD2B040D-7550-49B4-86D4-E212222B6C73/015EBBA4-10D8-49BA-B3D9-8CEEE67F9BC4.jpg)

作为对比，DF展示了Turn 10 Studios将《极限竞速6》移植到“天蝎”上的效果。在1月份开发机抵达工作室之后，Turn 10仅花了两天时间就将《极限竞速6》移植完成，而在开启最高细节层次的情况下，“天蝎”轻松地在游戏中实现了4K60fps，且GPU占用率只有66%：

![](.evernote-content/CD2B040D-7550-49B4-86D4-E212222B6C73/EBC1829A-4B15-4E5B-BAAA-22C73D702A5C.jpg)

而这些额外的运算资源对于现有的Xbox One游戏会有什么样的影响呢？微软要求运行在“天蝎上的所有Xbox One游戏都必须运用到其全部的运算性能，不论主机连接的是4K电视还是1080p的显示器。未获得对应更新支持的游戏也将立即获得更高的（动态）分辨率，更稳定的帧数甚至是更高的帧数（如果原本并不锁帧的话）。此外，运行在“天蝎”上的Xbox One游戏还将额外获得固定16x的各向异性过滤以提升画质，强制锁定垂直同步来消除画面撕裂。

至于“天蝎”的最终定名、价格和有什么样的游戏支持，就要等到微软在E3的大舞台上解开谜题了。
