# 为孩子学编程 这个玩具机器人背后有故事

---

在数字化的时代，编程对现代人来说成了一项越来越重要的技能。为了让下一代愉快的掌握这项技能，不少国外公司都推出了能教孩子编程的玩具。Photon 是这类玩具中较新的的一个，它的差异化之处在于它是一个有故事的机器人，这个故事会激发孩子对 Photon 的兴趣，在解锁 Photon 新能力的过程中，孩子能学到编程的基本知识。

![](.evernote-content/66AA5B78-1FCF-40B3-9F3D-605C2AE087FA/D48FC145-73AE-4B8D-8A08-7C21EF1A9691.jpg)

和其他编程玩具一样，Photon 是用专属的应用控制的。该应用中结合了 Scratch 和 Blockly 两种视觉化编程语言，孩子可以输入指令然后观察 Photon 如何去执行。与别的编程玩具不同的是 Photon 的应用专注于讲故事。为了让孩子集中注意力，保持玩下去的积极性，Photon 的研发人员应用了来自心理学专家和游戏机制专家的研究成果去设计 Photon 的方方面面——从开箱到利用编程知识进行游戏，都遵循着 Photon 叙事主线。

![](.evernote-content/66AA5B78-1FCF-40B3-9F3D-605C2AE087FA/62A2911A-736C-4B25-8D68-1A2EF574AB87.jpg)

总的来说，Photon 的故事讲的是它的飞船在地球上坠毁，它的程序因此损坏；为了恢复功能以便找到飞船的碎片，它需要人类的帮助。每当孩子们完成预定的任务时，他们就可以体验到 Photon 的新技能，例如：回避障碍，按照平面上画的线运动，激活新的感应器以回应不同强度的光照、声音或触碰。

为了激发孩子们的创造力，Photon 眼睛和天线上装有 256 色的 LED 灯，孩子们可以任意变换。此外 Photon 还支持一些磁力配件，如不同的帽子或者背包，孩子们可以把这些配件吸附在 Photon 的头上或者背部，创造看起来独一无二的 Photon。Photon 的开发者向家长们保证它是坚固耐用的，其内部的电子结构已经被紧紧地固定好，就算喜欢拆解东西的孩子也无法破坏。

![](.evernote-content/66AA5B78-1FCF-40B3-9F3D-605C2AE087FA/45075D62-3E29-48DE-9869-74283548E10A.jpg)

驱动 Photon 活动的是两个 200 rpm 的电动马达，Photon 内置 3000 mAh 的电池为马达提供电力。根据孩子们的使用情况，Photon 的续航在 4-6 小时之间，和大多数电子产品一样，它可以用 USB 充电。Photon 应用支持 iOS/Android 和 Windows 三个平台，这三个平台的设备需要通过蓝牙 4.0 与 Photon 连接。

想要给孩子购买这个机器人小伙伴的话可以前往 Kickstarter，单个 Photon 的价格为 199 美元，如果你现在就决定购买的话还能抢到 169 美元/个的优惠名额。按照计划，Photon 将于明年年初发货。
