# 在 Mac 上想玩 Windows 游戏又不想装双系统？试试 Porting Kit

---

![](.evernote-content/067BBAD3-2E5D-430E-8BC5-6CAC7779D7D6/D8F8CC46-2217-48AA-9822-212570055057.jpg)

对于使用 Mac 的人，如果需要运行 Windows 平台的应用程序，一般会有两个选择：

* 安装虚拟机
* 通过 Bootcamp 安装 Windows 系统

那么如果我们想玩 Windows 平台的游戏，我们该选择哪一种方案呢？大多数人会选择第二个，即安装双系统。毕竟虚拟机运行速度总是不够理想，玩游戏的效果更可想而知。

然而，安装双系统也有缺陷，首先是占用硬盘空间，专门腾出一块地方给 Windows ，让本就不够用的硬盘空间更加捉襟见肘。此外，在系统之间切换也是个麻烦事。

其实，还有一个解决方案，就是使用 [Wineskin](http://wineskin.urgesoftware.com/tiki-index.php) 之类的软件，将 Windows 平台的游戏移植到 OS X 上。听起来很酷对不对？但移植游戏的过程还是需要花费不少时间，也有一定的学习成本。

还好，现在有一个更好的解决方案，那就是今天我们要说的 Porting Kit。

Porting Kit 是什么
---------------

简单来说，Porting Kit 让你可以直接在 OS X 上玩 Windows 平台的游戏。稍微深入一点来说，Porting Kit 也是利用上面提到的 Wineskin，先帮你移植好游戏，完成必要文件的配置，而你只需要下载安装即可。

Porting Kit 拒绝盗版游戏，对于付费游戏，你需要提供必要的档案才能完成安装。正因如此，相比于其他一些支持盗版的游戏下载网站，Porting Kit 更加纯净。

![](.evernote-content/067BBAD3-2E5D-430E-8BC5-6CAC7779D7D6/8B4AE36B-5B76-4092-BCDF-EFC6B63C0971.jpg)

Porting Kit 优点
--------------

可以说，Porting Kit 就是一个提供 OS X 平台移植游戏的下载站。这样一个有些特殊的游戏下载应用，它有以下几个优点：

### 一键下载

按下「Download」，其他交给 Porting Kit 就可以了。游戏以及安装游戏需要的文件，都会自动帮你下载，不用担心游戏安装后，由于缺少某些组件而无法运行的问题。

### 安装简单

整个安装过程很傻瓜，免去自己配置的麻烦，这对于不熟悉 Wineskin 之类软件的人来说很重要。安装过程中弹出的选择框，应该点击什么按钮，Porting Kit 都会给出提示。

比如这里我安装「孤胆枪手 2」时，Porting Kit 提示我如果安装时弹出要求安装「Mono」或是「Gecko」，应该选择取消。

安装游戏前，首先阅读一下游戏介绍中的「requirement」和「Notes」，了解游戏运行的要求和注意事项，对顺利安装并运行游戏会很有帮助。

![](.evernote-content/067BBAD3-2E5D-430E-8BC5-6CAC7779D7D6/963F2D93-E96A-4C59-88C5-DF04ACB7C4DB.jpg)

### 有长期维护

Porting Kit 会每天更新游戏库中的游戏，以保障游戏可以游玩。此外，按照 Porting Kit 官网上的说法，每天至少会有五个新游戏添加到游戏库中，Steam 中的游戏也会在近期加入到游戏库中。

### 游戏预览视频

不确定游戏好不好玩？先看看游戏视频吧。Porting Kit 中大部分游戏都提供了游戏的试玩视频，点击「Watch Trailer」就可以观看了。

![](.evernote-content/067BBAD3-2E5D-430E-8BC5-6CAC7779D7D6/454DDE0D-CB9B-4D4A-8084-36EB363975F4.jpg)

### 统一管理游戏

在「Library」目录下，你可以看到已经安装的游戏，还可以将本地的游戏添加进去，方便你对游戏进行管理。

![](.evernote-content/067BBAD3-2E5D-430E-8BC5-6CAC7779D7D6/8543EE7C-9038-4416-A7EA-C8B6765AC1BA.jpg)

小结
--

其实更准确得说，Porting Kit 是一个项目，目的是解决 OS X 系统相对 Windows 而言可玩游戏较少的问题，让 Mac 用户也能玩上 Windows 游戏。

从使用体验上来说，Porting Kit 确实易用，游戏的下载安装简单，我们需要做的事情很少。唯一我想吐槽的是 Porting Kit 游戏库内的新游戏较少，多为经典老游戏，如果你想玩新游戏，那可能还是得另找办法了。

Porting Kit 完全免费并且没有任何广告，你可以在 [官网](http://portingkit.com/en/) 下载。
