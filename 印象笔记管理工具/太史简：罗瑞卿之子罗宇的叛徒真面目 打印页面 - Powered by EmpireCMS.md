| |  | | --- | | 太史简：罗瑞卿之子罗宇的叛徒真面目 | | --- | | 2016-01-15 23:27:02 | |  | | 近日，罗宇的回忆录再一次被境内外敌对势力当作“利器”，拿来向党和社会主义进行猖狂进攻。  罗宇何许人也？老一辈无产阶级革命家罗瑞卿的次子，乳名罗猛猛，现年72岁。清华大学自动控制系毕业，曾任总参谋部航空装备处长（师级）、赴美任战机改装联络组长，1988年授大校军衔。早在上世纪70年代末、80年代初，罗宇就与香港影星狄娜（梁帼馨）官商勾结。当时，狄娜以影星身份混迹于商界，生意涉及内地军方航空、航天工业，频繁出入于大陆，招摇于军界。据知情人士透露，罗宇利用职务之便为狄娜输送了巨额利益，于是二人勾搭成奸。  罗宇因极度害怕所犯罪行受到军纪国法惩治，于1989年利用出席法国航空展的机会叛逃，1990年与狄娜结婚，成为狄娜最后一任丈夫。这一肮脏的“婚姻”，罗宇不以为耻，反以为荣，自我吹嘘为“来自不同世界的姻缘”，“是上天的眷顾”，如此等等，不一而足。婚后，罗宇与狄娜共同经营狄娜名下的贸易公司，共享她们从国内盗窃的军方财产。1992年，罗宇被开除党籍军籍。  2015年秋，他的所谓回忆录《告别总参谋部》在香港出版。这本书秉承西方敌对势力的旨意，以造谣诬蔑、无中生有、肆意歪曲等手法对老一辈无产阶级革命家、中国共产党、中国人民解放军进行恶毒攻击，一直为严肃正派的境内外中国同胞所不齿。  孟子曰：“丹朱之不肖,舜之子亦不肖”，其是之谓乎？ | |  |   http://www.hswh.org.cn/wzzx/llyd/zz/2016-01-14/35911.html |
