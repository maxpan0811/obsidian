# 四大亮点：带你了解Win 10一周年更新包

---

[![](.evernote-content/E627F40A-FBE5-471F-B54C-80A27B295A3E/143D62E7-C476-49CA-9B72-1DFEBFFDF607.jpg)](http://tech.feng.com/2016-03-31/Four-big-window-take-you-know-Win-10-anniversary-update-packages_643107.shtml)

![](.evernote-content/E627F40A-FBE5-471F-B54C-80A27B295A3E/143D62E7-C476-49CA-9B72-1DFEBFFDF607.jpg)

威锋网 3 月 31 日消息 今天凌晨微软在旧金山召开的 BUILD 2016 全球开发者大会上，带来了诸多好消息，包括平台软件的 SDK 开发包、HoloLens 新亮点和人工智能方面的技术发展等等。但更重要的是，针对大家最关心的 Windows 10，微软花了大量时间来介绍今年的最大的更新“Windows 10一周年更新包（Anniversary Update）”，其实也就是大家熟知已久代号为 Redstone 的更新。

在介绍 Windows 10 年度更新包之前，微软首先提到了现在已经有超过 2.7 亿台 Windows 10 设备被激活，今年夏季所有 2.7 亿台 Windows 设备都可以免费获得“Windows 10一周年更新包”，这些设备包括 PC、平板电脑、智能手机、Xbox One 游戏机和 HoloLens 头盔等等。

究竟“Windows 10 年度更新包”有哪些亮点呢？据现场的消息了解，全新的 Windows 10 一周年更新将涉及微软小娜（Cortana）、Windows Hello、Windows Ink和游戏等大量功能创新。下面随我们一同来了解微软在发布会上提到的重点内容。

**第一：Windows Hello 扩展更多用途**

Microsoft Edge 浏览器和更多的应用程序将支持 Windows Hello 生物安全特征。

![](.evernote-content/E627F40A-FBE5-471F-B54C-80A27B295A3E/CF6D96BF-E19A-4BBE-B4DE-16843B64E989.jpg)

很多研究表明，超过 80% 的用户在多个网站使用相同的密码，而且这些账户超过 20 甚至 30 个，一旦其中之一被盗后果将不堪设想。对此微软认为，有助于将设备提升到企业级安全性的 Windows Hello 功能，将在“ Windows 10 一周年更新包”中帮助更多的 Windows 10 用户。

微软称，Windows Hello 便捷简易但很强大，“Windows 10 一周年更新包” 将 Windows 10 的安全性扩展到了多设备和 Microsoft Edge 浏览器上，让用户以企业级安全标准登录设备或者网站。简单的说，你可以刷脸或者通过指纹识别等生物识别特性来登陆所浏览的网站。

与此同时，利用全新公布的 Windows 10 一周年更新的 SDK 开发工具。开发者还能够将 Windows Hello 的最新创新功能整合到自己的应用中。

**第二：Windows Ink**

这是微软为 Windows 10 带来的全新书写体验，除了 Edge、地图、Office 之外，也将支持整合入第三方应用。

微软了解到，超过 70% 的用户每天花 1 个小时左右的时间来使用笔。所以在笔记本电脑或者平板电脑上，少了笔的支持很多体验都将变得不完整。所以，微软推出全新适用于 Windows 10 系统的 Windows Ink 体验，将 Windows 与手写笔完美结合，通过笔尖任何人都可以在设备上拥有像纸上即时创作或白板作画的体验，并且能够轻松将创意分享到今天的社交网络上。

![](.evernote-content/E627F40A-FBE5-471F-B54C-80A27B295A3E/6903A1A8-4C47-46CD-945D-D085A910E724.png)

其实，早期 Windows Ink 已经存在，不过只支持大多数 Microsoft Edge、地图和 Office 等微软自家应用程序。而在“Windows 10 一周年更新包”中，开发者利用 Windows 10 一周年更新的 SDK 开发工具，也可以将 Windows Ink 整合到第三方应用程序上，甚至只需要写两行代码既可。

本次大会上，微软演示了 Windows 10 自家应用针对 Windows Ink 的新特性。比如用户在地图上用手写笔规划线路之后，地图可以准确的识别和分析，并且平时使用手写笔来还可以在便签上识别时间地点，然后与自带的地图应用服务关联。微软承诺，未来将会在商店内提供 Windows Ink 应用分类。

**第三：Cortana 更新更加智能**

在“Windows 10 一周年更新包”中，Cortana 支持锁屏激活，主动建议，支持获取 Windows 和 Android 手机上的通知内容，支持让应用在特定条件下执行任务等。

![](.evernote-content/E627F40A-FBE5-471F-B54C-80A27B295A3E/910AFFF8-A8A4-4EF8-8D45-ACABBAC6D2A6.jpg)

　微软声称，Windows 10 是 Cortana 小娜语音助手的最佳使用场所，而且目前已经有近 1000 个支持 Cortana 的应用程序。不久之后，微软将会在 Windows Sote 应用程序商店提供一个 Cortana 合集，囊括所有支持小娜助手的应用，方便用户更快的查找。

本次周年更新，Cortana 小娜助手“学会了”如何贯穿一整天向你更主动提供有效的指引和建议服务，比如到点了询问你是否需要购买午餐的语音互动。Cortana 还将支持锁屏激活，在未解锁 PC 或手机的状态下，无需登录也能通过 Cortana 直达所有常见类型的应用或任务。更重要的是，Cortana 将可以帮你对 Windows 或 Android 手机上某些基于通知的任务执行相应的特定操作，比如接收或发送短信。

当然了，利用 Windows 10 一周年更新的 SDK 开发工具，开发者也可以将自家的应用加入 Cortana 支持，让其识别应用上的数据和用户的操作，更深层次的帮助使用应用的用户执行任务，使其更像是真正的助手。

**第四： 通用 Windows 平台（UWP）支持  Xbox**

“Windows 10 一周年更新包”发布之后，Xbox One 支持通用应用商店，支持背景音乐，可以快速将零售版变为 UWP 通用版。

游戏玩家将受益于本次周年更新包，届时 Xbox One 将支持 Cortana 和背景音乐功能。其中 Cortana 将成为玩家的游戏助手，可以帮助玩家寻找最棒的游戏，为玩家提供游戏的技巧和窍门。该更新之后，Windows 10 将提供更多玩家请求的功能，比如多 GPU 支持和可关闭垂直同步。此外，Xbox 的开发者模式可以让任何一部 Xbox One 变身为应用开发调试平台，使得每一位开发者为用户客厅开发应用成为可能。

微软表示，在 Windows 10 快速增长的带动下，大量全新的通用应用也在不断涌现，例如 Twitter、Uber、King、迪士尼、星巴克等等。Windows 应用商店的访问量已经超过 50 亿，Windows 用户可以在这里发现更多专为 Windows 10 打造的新应用，包括上千种专为微软小娜（Cortana）打造的应用程序。

**其他内容**

以上四点就是微软针对“Windows 10 一周年更新包”提到了四大重点内容，至于其他可能你还关心的信息还有不少。比如说，微软已经与各家硬件厂商合作推出了超过 500 款的 Windows 10 设备。

微软还针对开发者发布了桌面应用转换工具，可以将基于 Win32、.NET 开发的 exe、msi 等超过 1600 万个桌面老版 Windows 应用直接转换为现代桌面应用，移植扩展到 Win 10 系统的通用平台中并上线到 Windows Store 商店中。转换器能够自动生成大部分代码，开发者只用做出少量修改即可面，同时还能添加更多通用 Windows 平台上的创新功能。

如你也想尝试“Windows 10一周年更新”的新功能，可以加入 Windows inside 预览体验计划（WIP），并安装最新版本即可。微软表示，从今天开始，参与 Windows inside 会员已经可以开始获取 Windows 一周年更新的预览版本。
