**Pretext**

[下载链接](https://itunes.apple.com/app/pretext/id1347707000&at=10lJSw)

#### **应用简介**

Pretext 是一个基于 Files 的 Markdown 编辑器，支持 Markdown 语法高亮、预览、丰富的 iPad 快捷键。

#### **终于有一款支持 Files 的 Markdown 编辑器**

**@Minja：**我刚开始写作的那段时间，主力工具一直是 Byword。相信只是偶尔写点东西的人，也更喜欢这种轻量的编辑器。

但这类写作工具的文档管理一直是个问题，直到我用上了 Pretext。它集成了 iOS 11 的 Files，可以用其他应用修改 Pretext 里的文件，也能在应用内通过 Documents Picker 直接修改别处的文档，就像在 Mac 上的 Finder 里一样自由。

![](.evernote-content/47129775-1D0B-4D7F-A145-E17AAFCF5207/41F1823F-C625-48E0-9E09-6AF60DE48C38.png)

支持 Files

得益于相对开放的文件管理系统，Pretext 足以在 iOS 担任临时的文档修改工具，如果你在 Mac 上使用的是 Typora、Byword 这些没有文件管理系统的编辑器，还可以把文档存进 Pretext 的 iCloud 文件夹，这样就能随时在移动设备上做修改工作了。

至于 Markdown 编辑方面的功能我不多说，这些基本是标配了，但是相对丰富的内置快捷键还是值得夸赞一下。

![](.evernote-content/47129775-1D0B-4D7F-A145-E17AAFCF5207/427F9A5D-F430-49C6-973A-4D2C8A73C7F1.png)

支持快捷键

当然，需要说明的是，这个应用本身不是非常出色 —— 甚至对于中文用户来说实用性没有很强（暂不支持预览中文文章），但仅仅是「基于 Files」这一点，就让我对它爱不释手。接入 Files 的意义可不只是「带来更加 iOS 11 风格的界面」，Drag&Drop、和 macOS 同步的标签系统，这些都非常重要，能保证我们在不同应用里的操作是**连贯**的。

**ScreenFocus**

[下载链接](http://www.apptorium.com/screenfocus?ref=producthunt)

#### **应用简介**

ScreenFocus 是一个窗口管理工具，当你同时使用多个显示器时，它可以让不用的屏幕暗下来，帮你集中注意力。

#### **多屏工作者的好帮手**

**@Minja：**我在同时使用多台显示器，所以一直很关注窗口管理。

以前用过诸如 HazeOver 和其他的窗口管理工具，它们的用途都是高亮显示当前正在使用的窗口，而把其他地方遮上一层半透明的「黑幕」。但事实上，一旦我专心工作起来，有没有这些辅助工具我都能集中注意力在当前窗口，有时连右上角的通知都会「选择性忽视」。

主观上，我是觉得这类工具用途没有它们宣传的那么大，但很长时间里我一直都忍不住在使用它们。直到最近我才找到原因：真正的痛点不在于无关内容的诱惑，而是暂时不用的屏幕**太亮了**，强行把眼睛「拽」过去。如果在夜间关灯使用（不少注重「仪式感」的人应该也像我一样这么做过），简直就像在眼前悬了几颗大灯泡，用久了对眼睛也不好。

最近出的一款窗口管理工具，ScreenFocus，满足了我调教多显示器的需求。它的功能只有一个：**把你不用的屏幕变暗**，具体说，就是光标离开屏幕时，那块显示屏就会变暗。

![](.evernote-content/47129775-1D0B-4D7F-A145-E17AAFCF5207/74F885FE-2A6C-46A1-BE5C-1C88F0F00D22.gif)

自动降低闲置显示器亮度

当然，具体要多暗、延迟时间这些具体参数还是可以调的，我觉得默认的 Medium 亮度是比较合适的，夜间不刺眼，白天不至于瞪着眼睛去看。
