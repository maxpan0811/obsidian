# 在 LINE 上市的前夜，我们试图偷师它的 IP 吸金大法

---

![](.evernote-content/6E24C1D6-C6DA-47ED-9024-A6C6B1881596/D385DC8E-E379-4D7A-A7C7-923D12802DDE.jpg)

千颂伊和都敏俊的调情工具—— LINE ——要上市了。

LINE Corp. 是日本即时消息运用运营商，它计划 7 月在东京和纽约进行首次公开募股。在东京的 IPO 拟发行 3500 万股，指导价每股 2800 日元，总募资超过 1127 亿日元（10.5 亿美元）。纽约的募资情况尚未披露。

在这次 IPO 里， LINE 的估值大概会在 55 亿美元左右。记性好的人会发现，相比于 2014 年，估值已经少了接近一半。那时，它第一次准备 IPO，估值 99 亿美元。

但，这并不妨碍它成为 2016 年日本规模最大的 IPO， 以及 2016 至今全球规模最大的科技企业 IPO。

![](.evernote-content/6E24C1D6-C6DA-47ED-9024-A6C6B1881596/C787C915-5F30-47F2-BA71-D3B6364B9E79.jpg)

产品用不了，形象照样火

即使上市了，也不会改变内地用户如今只能在 LINE 里看到一个个灰色的感叹号，提醒「你的消息无法发出」。

这事要追溯到 2014 年 7 月，那时刚正式进入（自 2012 年 12 月）中国内地市场没多久的 LINE 被彻底「墙」掉。

还好，封锁 LINE 这个事并不只是你国干了，隔壁金正恩最近也发话称「用 LINE 的人都是间谍！」，所有使用该类即时通讯应用的朝鲜用户都将因涉嫌间谍活动被逮捕。

![](.evernote-content/6E24C1D6-C6DA-47ED-9024-A6C6B1881596/A667D464-8A2D-4124-8003-D020443F3502.jpg)

比较尴尬的是——在朝鲜被禁止的通讯应用除了 LINE 和韩国的 Kakao Talk 之外，还有我大微信。

虽然产品用不了了，但产品里的「卡通角色们」继续发光发热，开店赚钱。

去年 7 月底，LINE 在 上海淡水路开了内地第一家 LINE Friend Cafe & Store。开幕当天，人潮汹涌，队伍排到不知哪里去，想起产品在国内的窘境，骤然令人萌生出一种「老夫肉身虽死，但精神永续呐」的壮烈感。

![](.evernote-content/6E24C1D6-C6DA-47ED-9024-A6C6B1881596/598AE8B7-3146-4FAB-B955-9A833BC2BAF5.jpg)

而且，这样的情况早在优衣库与 LINE 推出的合作款 T 恤上便得到了印证。

![](.evernote-content/6E24C1D6-C6DA-47ED-9024-A6C6B1881596/050EA8DF-2D2F-4E12-92B0-307F3C7F3C76.jpg)

收费表情包，卖起来不是事儿

对于大部分内地用户来说，LINE 的意义在于，那些萌萌的卡通形象、贱萌的贴图表情，而不是聊天应用本身。

谈到产品的本质，LINE 的基本功能无非是老两样：发送信息和打网络电话。不过，让 LINE 真正火起来，并做出差异化的，是其五花八门的贴图表情。

![](.evernote-content/6E24C1D6-C6DA-47ED-9024-A6C6B1881596/BFA1F707-4251-4173-91AA-FA6502325E3F.jpg)

除了非常火的 LINE 官方贴图系列中的布朗熊、可妮兔、馒头人以及詹姆士等，LINE 也与各大厂商进行合作，推出了各种吸粉效果拔群的贴图。

无论是三丽鸥的 Hello Kitty、懒蛋蛋，还是迪士尼的「疯狂动物城」、「超能陆战队」，亦或是如「海贼王」、「银魂」、「进击的巨人」这样热门的作品，都在 LINE 上推出了相应的主题贴图。

凡是最新的热门题材，你都能很快在 LINE 上找到对应的贴图；同时 LINE 还在不断地推出经典角色的主题贴图，花式满足情怀粉。

![](.evernote-content/6E24C1D6-C6DA-47ED-9024-A6C6B1881596/4458161F-C2A6-46E1-96DA-35A61239E9B0.jpg)

与微信以免费表情为主的模式不同，LINE 上的表情大多都是收费的：贴图表情包大多为 240 日元（约 15 人民币），而应用主题售价为 360 日元（约 22 人民币）。

这在习惯了免费表情的内地用户消费认知中，并不算便宜。但在日本这个二次元国家，对于忠实粉丝来说，二三百日元根本不是事儿；即使不是御宅族的年轻女生们，也会一边夸张地喊着「好可爱好可爱」然后一边买买买。

IP 吸金：纵横发展，欲速不达

你以为，单单卖贴图就是 LINE 的「生命 Line」？大错特错！

以贴图表情为主的通讯产品仅占了总营收的 22%（数据来源：LINE 2016 Q1 财报）。真正的两个大头是广告和游戏与内容增值服务，它们分别各占了 35%。

![](.evernote-content/6E24C1D6-C6DA-47ED-9024-A6C6B1881596/E0CF2375-41D9-41EE-9B45-3EA9CF04E6F3.jpg)

广告暂且不提。

游戏方面，目前你在 LINE 官网上可以看到近 30 款「LINE 游戏」。这些游戏大多可以通过 LINE 账户进行登录，与 LINE 上的好友进行合作或者竞赛；其中不乏一度成为日本市场内热款的作品，如「LINE Pokopang」及「LINE Brown Farm」等。

有趣的是，在 LINE 的游戏中，接近一半都是以 LINE 那几个原创卡通角色作为主角的。无论是跑酷型游戏，还是农场游戏，或者是三消游戏，在里面你能看到永远都是布朗熊、可妮兔、馒头人、和一头金发的迷之男人詹姆士。

这很符合「日本奸商骗钱大法」的思路：瞄准一个已经火了或者有潜力火的 IP，不断横向、纵向地做拓展，围绕它进行各种五花八门的尝试，从衣食住行乐全方位打击用户，让再难啃的用户也要被攻陷。

而且，这种全方位打击并不需要 IP 本身要像「指环王」或者「魔兽世界」一样有多宏大的世界观做支撑、也不需要有多悠久的历史积淀当靠山。

在 LINE 身上，这几个又贱又萌偶尔还略血腥暴力的角色身上，完全没有所谓的「厚重感」，严格说它们甚至连背景作品都没有——最初，它们唯一的相关作品便是以它们为主题的贴图包。

![](.evernote-content/6E24C1D6-C6DA-47ED-9024-A6C6B1881596/FEF995A9-BA00-4AC3-A68E-1ABD0E2A7997.jpg)

对于以萌为核心卖点的 IP 来说，只要抓住其本质「可爱又有趣」就够了。当年的「轻松熊」是这么火的，后来的「船梨精」也是这么玩的——反正它们一开始肯定不是靠「轻松熊编年史」和「船梨精大电影」火的。

也就是说，当你在「萌」这件事上纠结太多发力太猛，「萌」这件事就不萌了。

每次看到国内厂商们倾尽全力希望打造出的「国民级 IP」时，你难免不产生一种脱力感。以吸金吸到嘴抽筋的爆款手游来说，单看单个作品，的确都是爆款，可一旦脱离游戏这个载体，你却不会有太大冲动去渴望得到它们的相关周边。在线上让你疯狂氪金的，线下却成了你心头尴尬的中二后遗症。

即使你用不了 LINE 聊天，你还是能看到无数少女在 LINE Cafe 里疯狂采购 LINE 的公仔文具马克杯；又或，你十年前在艾泽拉斯刷线上装备，十年后你也愿意穿着法师装去看『魔兽』电影。

![](.evernote-content/6E24C1D6-C6DA-47ED-9024-A6C6B1881596/FA1369AB-8149-4304-9DA8-DA793B5F38C2.jpg)

但是，你难道会想去买『大X神』的马克杯或者去『传奇X业 Cafe』喝杯冰沙吗？对于国内的 IP 们来说，要么是它们还不够好，要么就是一开始野心太大、步子太宽。

卡通形象之下的一站式野心

目前的 LINE 看起来更像又一个巨大的 IP，而不是一个简单的通讯应用。从讲故事的角度上说，其收入来源的多元化，或许比 Facebook 那高达 96% 的广告营收占比，给人以更多的想象空间。

除了贴图和游戏之外，LINE 还搞了名为 LINE Manga 的平台。它号称是「日本最大规模的漫画平台，共有超过 10 万卷漫画」，用户可以通过在 LINE 中购买虚拟漫画币来付费阅读这些漫画。

![](.evernote-content/6E24C1D6-C6DA-47ED-9024-A6C6B1881596/732EC3BB-C514-4865-8A06-6AC3C6CFF2D4.jpg)

未来 LINE 在流媒体音乐领域（LINE MUSIC）和移动支付领域（LINE Pay）上也充满野心。

从这个角度来看，LINE 显然也希望走上微信所走的那条路——成为用户各类服务的集成一站式入口。无论是游戏、音乐、出行打车还是在线支付，LINE 都希望用户能在这个应用上完成。

哦对了，LINE 甚至还丧心病狂地搞了个「LINE 占卜」，让「LINE Friends 和著名的占卜师为你占卜」，同样是在 LINE 上购买虚拟占卜币来付费体验服务。

![](.evernote-content/6E24C1D6-C6DA-47ED-9024-A6C6B1881596/9D2FCB8F-4E45-4D8C-8838-426D8331E91F.jpg)
