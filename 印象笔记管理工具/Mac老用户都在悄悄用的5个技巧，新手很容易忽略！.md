一、还在点来点去开App？太慢了
----------------

说实话，我以前用Mac也老是用鼠标点Dock栏开Safari、邮件、日历之类的。但后来我发现，效率简直低到爆。

### ⌘+空格，直接搜App

![](.evernote-content/9160EDFF-806A-48D8-8750-AB2B9BB00EF4/F86AAAC0-09A7-4EB4-BDEC-E3A0230ED1BA.png)

现在我一律用**Spotlight**启动App，快捷键是`⌘+空格`，输入几个字母按回车，App就开了。别看只是几秒的差别，一天下来能省不少时间。

后来我又装了**Raycast**，这个比Spotlight还猛，不光能开App，还能发邮件、建提醒、跑脚本，甚至还能直接算东西，全键盘搞定。

### 快速切换App：⌘+Tab

![](.evernote-content/9160EDFF-806A-48D8-8750-AB2B9BB00EF4/D9CA48A4-4571-409D-B7E3-9EC9D5BC2119.png)

还有一个必学快捷键是`⌘+Tab`，可以快速在当前打开的App之间切换。我一般是一边按住⌘，另一只手点Tab选想切的App，超顺手。

补充一下：`⌘+Shift+Tab`是反方向切换APP

二、不会调窗口大小？
----------

以前我总是手动拖动窗口大小，左拖右拖，结果越拖越乱。尤其是多屏幕、多任务的时候，效率直接腰斩。

### 拖动窗口 + Option键，超顺滑

![](.evernote-content/9160EDFF-806A-48D8-8750-AB2B9BB00EF4/C1D2C291-5098-44BC-8697-0992E8223E58.png)

新版macOS（比如Sequoia）在窗口管理上已经好多了，只要把窗口拖到屏幕边缘，它就会自动贴边。

但更关键的是，**你按住Option键再拖**，窗口就能自动“吸附”，整个过程非常流畅。

### 或者直接用窗口管理工具

![](.evernote-content/9160EDFF-806A-48D8-8750-AB2B9BB00EF4/39986AF6-470F-468F-8B6F-D8C3EFCCD00C.png)

我自己用的是Raycast里的窗口管理插件，之前还用过Rectangle，都挺好用。

有了它们，就可以用快捷键把窗口直接调成左半、右半、全屏，效率爆炸：

* `Control + Option + ←`：靠左排列
* `Control + Option + →`：靠右排列
* `Control + Option + ↑`：最大化窗口

Raycast还能设置类似“whr”这种指令，一打出来，窗口就自动贴右边，完全不动鼠标。

![](.evernote-content/9160EDFF-806A-48D8-8750-AB2B9BB00EF4/5F11C021-53E4-41E6-A945-6E005E52A1E1.png)

三、通知太多，注意力全被打散
--------------

Mac的通知系统真的很猛……猛到一堆弹窗、声音、图标提醒，搞得我脑袋一团乱。尤其跟iPhone联动的时候，Mac也跟着震不停。

### 学会用“专注模式”

![](.evernote-content/9160EDFF-806A-48D8-8750-AB2B9BB00EF4/9D9EA8DB-1633-41DA-B8F0-C8D0034FCEDF.png)

后来我发现**Focus专注模式**真的太好用了。我给自己设了“工作模式”、“写作模式”，只让少数几个App能打扰我，其他一律闭嘴。

设置方法很简单：

![](.evernote-content/9160EDFF-806A-48D8-8750-AB2B9BB00EF4/6F24D363-695A-45CD-B8A1-3AA61A38E296.png)

**系统设置 → 专注模式**，想开哪个模式，从菜单栏点一下就好了。

你还可以设置：

* 什么时间段自动开启
* 只允许特定联系人来信息
* 特定App才能推送通知

自从用了这个功能，我整个人清净了不少，效率也高了。

四、不会用快捷键？你白用Mac了
----------------

我以前也懒得记快捷键，总想着“点一下又不麻烦”。但后来发现，每次用鼠标点来点去浪费的时间真的不少。

### 必学的几个万能快捷键

这些快捷键，我现在是肌肉记忆了，随手一按就能搞定：

![](.evernote-content/9160EDFF-806A-48D8-8750-AB2B9BB00EF4/F3BB2701-2EB6-4D08-A4DB-08767568DBA4.jpg)

* `⌘ + Tab`：切换App
* `⌘ + ~`：同一App不同窗口之间切换
* `⌘ + H`：隐藏当前App
* `⌘ + ,`：打开当前App的设置
* `Control + ⌘ + Q`：锁屏
* `⌘ + Q`：退出App
* `⌘ + N`：在浏览器开新标签页

五、别盲目相信苹果的默认设置
--------------

macOS的默认设置，有些我觉得真的不太合理。比如下面这几个：

### 5.1 Dock栏老是显示，太占地方

我现在设置成自动隐藏Dock，只有把鼠标移到底部才会显示。干净又清爽，看屏幕心情都好。

![](.evernote-content/9160EDFF-806A-48D8-8750-AB2B9BB00EF4/E6BC40E8-5606-418C-9147-3172B17EE8FA.png)

设置路径：**系统设置 → 桌面与Dock → 自动隐藏Dock**

### 5.2 滚动方向不顺手？马上改回来

Mac默认是“自然滚动”，也就是反的。很多从Windows转来的朋友刚开始很不习惯。

你可以这样改：

![](.evernote-content/9160EDFF-806A-48D8-8750-AB2B9BB00EF4/9EBAE0B7-512C-4214-BEA3-8C56EFADF89F.png)

**系统设置 → 触控板 → 滚动与缩放 → 关闭自然滚动**

这样滚动就变得“正常”了。

### 5.3 Stage Manager可以试试看

Stage Manager是macOS里的“专注工作区”，可以让你只专注一个App，把其他App收在一边。

![](.evernote-content/9160EDFF-806A-48D8-8750-AB2B9BB00EF4/6ABFA603-E8BE-427E-99D7-C372177A8C7D.png)

这个功能有点见仁见智，有人觉得好用，有人觉得鸡肋。你可以在控制中心开关它，用一下再决定要不要用。
