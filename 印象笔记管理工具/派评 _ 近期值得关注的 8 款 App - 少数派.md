### LiquidText 4.0：支持标签，老牌阅读标注工具大更新

**@Tp：**LiquidText 是一款颇具创意的阅读批注工具，你可以像做剪报一样，将文章中的部分内容摘抄到右侧的「Workspace」里，并为其添加批注。最近，LiquidText 发布了自己的 4.0 大版本更新，带来了标签和 Workspace 的搜索功能，能够让你的阅读效率更上一层楼。

**关联阅读**

![](.evernote-content/D91C3CCE-C82B-43E3-9320-32EAAE70213C/1EEFA1F1-71A6-40E7-9431-C9D17B22B9D8.jpg)

在新版本中，你可以为摘抄内容添加标签。 LiquidText 的标签系统稍有不同，在标签名称（Name）前，你还可以添加一个分类（Category）用于筛选。默认情况下，应用只会显示「Name」，但在搜索时，可以通过输入 Category 快速筛选出特定的内容。

![](.evernote-content/D91C3CCE-C82B-43E3-9320-32EAAE70213C/AC8B64A3-9BC2-49D4-9C23-C4523C318365.jpg)

Category 的加入让我们在阅读专业文献时能有更加清晰的逻辑，比如一篇文章引用了多名作家的观点，你就可以在摘抄时将作家名添加至 Category，并在 Name 里输入想要打上的标签。

本次更新还加入了对 Workspace 的搜索功能，你可以直接搜索 Workspace 里的摘抄、评论和标签等信息，帮你更加快速地找到想要的内容。在新版本中，你还能自定义 Workspace 的背景，目前支持背景图案、颜色等内容，不妨根据自己的喜好来装扮 Workspace 吧。

![](.evernote-content/D91C3CCE-C82B-43E3-9320-32EAAE70213C/E9B79603-DF1B-4A65-9B0F-D590AA3753FD.jpg)

你可以在 [App Store](https://apps.apple.com/cn/app/liquidtext/id922765270) 免费下载 LiquidText，应用含高级版内购，售价 68 元。

### Keep：网页书签管理工具

**@Tp：**在网页上遇到喜欢的文章、或是需要稍后阅读的网页时，我们都会想将它收藏起来；Keep 是一款支持分类、标签的网页书签管理工具，不论是网页文章、视频还是 PDF 文件，都可以放到 Keep 进行管理。

Keep 支持多种添加链接的方式，除了常规的分享菜单外，在打开应用时，它还会自动检测你的剪贴板中有没有链接，并询问你是否添加。如果你在用电脑访问网页，还可以通过 Keep 提供的书签将正在浏览的网页收藏起来，无需打开手机。不仅如此，你还可以通过邮件或短信的方式将链接添加到 Keep 中。

![](.evernote-content/D91C3CCE-C82B-43E3-9320-32EAAE70213C/E5D40590-1B6D-4A36-9BDE-C53ED5B67B9F.jpg)

被添加到 Keep 里的网页会自动添加上域名的标签，你可以手动为它们打上标签用作分类筛选。如果你添加的链接里有 Keep 所支持的视频网页（比如 YouTube 或 Vimeo），那么它还会被自动添加到「Video」分类下，这里就相当于一个「稍后观看」的视频列表。音频和 PDF 也同样支持自动分类，不过音频并没有单独的界面，你可以在主界面左上角的「Filters & Tags」里找到「Saved Audio」。

如果你订阅了高级版，还可以解锁语音朗读文章和智能标签等功能，智能标签会通过机器学习你的标签习惯，并自动为你新添加的网页打上标签。

你可以在 [App Store](https://apps.apple.com/cn/app/keep-smart-bookmark-manager/id1359954259) 免费下载 Keep，应用含高级版内购，价格为 22 元 / 月。

### App List：应用收藏和价格提醒工具

**@化学心情下 2：**对于经常在 iOS 上选购 App 的朋友而言，一些付费的游戏或者应用往往都会在某个时间打折，而对于那些并不是急需购买的应用，等待降价或者限免入手或许是不错的选择，但如何快速掌握自己喜欢应用的降价信息并在第一时间获得消息？可能就需要 App List 这个收藏管理工具了。

App list 倒是颇有些购物网站收藏商品的意思，只不过收藏的商品这一次变成了各种 App，而最为主要的功能就是把自己关注的或者希望购买的应用以及游戏放入到 App List 中，之后如果它们的价格发生变动就会通过应用通知提示你。

![](.evernote-content/D91C3CCE-C82B-43E3-9320-32EAAE70213C/1BB44FAC-AC83-4466-8902-E84D8A854061.png)

App List 用起来也非常简单，在 App Store 中找到想要购买的应用，然后点击分享选择「Add to App List」，将关注的应用添加到 App List 中不同的组中，这样你就可以在 App List 统一管理那些心愿单中的 App 了。而当有应用出现价格变动应用就会发送推送提示，这样限免和降价都不会错过。如果你经常购买付费的 App 不妨一试。

![](.evernote-content/D91C3CCE-C82B-43E3-9320-32EAAE70213C/BCE6224E-D13C-4588-88BE-F48F0F6A9139.png)

你可以在 [App Store](https://apps.apple.com/cn/app/the-app-list/id1381108146) 免费下载 App List。

### NightWallpaper：自动切换夜间壁纸

**@Noah\_Choi：**在 Android 平台上，大部分设备会在夜间将壁纸亮度压暗，这样做虽然能为壁纸的展示效果起到一定优化作用，但同时壁纸也会变得灰蒙蒙一片。如果你的手机已经具备夜间模式功能，那么就可以使用 NightWallpaper 这个小工具实现昼 / 夜壁纸自动切换了。

![](.evernote-content/D91C3CCE-C82B-43E3-9320-32EAAE70213C/EBA410FE-251C-41CE-A779-79F5E4B3D9FA.jpg)

NightWallpaper 是一款比较没有存在感的小工具，唯一的界面仅仅是用来配置壁纸而已。首先你需要设置白天壁纸，然后使用 NightWallpaper 「保存」成白天壁纸；夜间壁纸也是同样的操作步骤。完成后你就不用再管它了。只是由于它需要驻留后台，所以建议大家在通知栏将其移动到「静音通知」或禁止它的通知权限。

![](.evernote-content/D91C3CCE-C82B-43E3-9320-32EAAE70213C/82505984-DCD7-4B79-9B7B-286AF02F13CA.gif)

使用 NightWallpaper 不需要 root，也不需要使用 adb 操作，可以说是几乎没有门槛。Android 本身内建了夜间模式接口，使得许多 App 可以能够跟随系统自动切换 App 自己的暗黑主题。NightWallpaper 也正是利用这种方式，只不过是「取之于系统、用之于系统」罢了。

如果你也有区分昼 / 夜设置壁纸的需求，可以在 [酷安](https://www.coolapk.com/apk/com.ypp.nightwallpaper) 免费下载 NightWallpaper。

### YES PDF：轻量级 PDF 阅览器

**@ln - 勇字当头：**

通常情况下，我们极少选择在手机端查看 PDF 文件，一来作为一种常见的文件格式，对 PDF 文件的处理更多地发生在大屏办公设备上；二来，现有的 Android 端 PDF 阅读器大都过于简陋，不适合长时间阅读文件。不过，再小众的需求都值得被满足，由个人开发者开发的这款 YES PDF 便能辅助我们在手机端更好地阅读 PDF 文件。

初次使用 YES PDF，我们可以手动搜索或是通过应用扫描来查找设备内的 PDF 文件，同时 YES PDF 还支持对导入的文件进行分组，方便日后管理。此外，YES PDF 会默认将用户近期阅读的文件展示在「最近」一栏中，进一步省去查找文件的时间。

![](.evernote-content/D91C3CCE-C82B-43E3-9320-32EAAE70213C/512FEA2A-E55A-42A4-8D1F-61A5350A1711.jpg)

相比于手机内置的 PDF 阅读器，YES PDF 添加了一些必要的自定义设置，如横向 / 竖向阅读，日间 / 夜间模式切换等等。除了常见的单击翻页、音量键翻页，开发者还为 YES PDF 加入了自动滚动功能，滚动速度支持自定义调节，如此一来，即使在拥挤的空间中进行阅读，也能轻松单手胜任。

![](.evernote-content/D91C3CCE-C82B-43E3-9320-32EAAE70213C/75505141-1CCD-4130-859D-34CAD92BD7A5.jpg)

作为一款纯粹的 PDF 阅读器，YES PDF 很好地平衡了功能性与简洁度，剔除了一切不必要的设置项，简化了阅读界面，而锁定横屏、导出图片等小功能的加入又显得别出心裁。十分推荐有在手机端进行 PDF 阅读需求的用户使用。

![](.evernote-content/D91C3CCE-C82B-43E3-9320-32EAAE70213C/04AF1B5C-5B72-4CCE-A1E9-249CBD18904E.jpg)

目前你可以在 [酷安](https://www.coolapk.com/apk/com.aaron.yespdf) 免费下载使用 YES PDF。

### Tuck：将应用窗口停靠在屏幕边缘

**@huhuhang：**Tuck 是一款支持将应用窗口快速停靠在屏幕边缘的小工具，能在一定程度上提升窗口切换效率。Tuck 的使用方法非常简单，安装并授予辅助功能权限之后，你就可以把指定窗口停靠在屏幕的上、下、左、右四个边缘。

默认情况下，当鼠标放置到屏幕边缘时，窗口就会自动弹出。如果再次点击边缘，或设置成鼠标移出窗口后，就会自动隐藏。

![](.evernote-content/D91C3CCE-C82B-43E3-9320-32EAAE70213C/19BE04FF-E824-4DE6-A449-3F338C9DA22B.gif)

Tuck 特别适合于必须要频繁在几个窗口之间切换操作的场景。之前，我一般会使用窗口快速布局应用，将多个目标窗口平铺在桌面上。或者使用 Command + Tab 组合键来切换窗口。如今使用 Tuck 之后，既可以兼顾使用较大尺寸的窗口，又可以同时完成快速切换。

Tuck 目前仅支持 Mac，如果你有兴趣，可以访问官网 [免费下载使用](https://www.irradiatedsoftware.com/tuck/)。Tuck 的一次性买断价格为 6.99 美元，不过开发者提供了长期试用模式，仅会在每次启动 Tuck 时弹出购买许可的提示信息。
