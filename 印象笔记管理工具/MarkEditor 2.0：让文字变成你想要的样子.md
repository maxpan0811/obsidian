MarkEditor 2.0：让文字变成你想要的样子
==========================

19小时前

[![](.evernote-content/A28C47A3-57BE-42FB-B01C-0D62557E9ECA/5E7A3DF5-0C82-4697-B1D8-B3C45C27BAD7)](https://sspai.com/user/680043)

#### 责编: [waychane](https://sspai.com/user/703114)

> 好的写作体验，更好的效率利器。

这是 [MarkEditor](https://www.markeditor.com/) 官网的标语，也是开发者[海波](https://sspai.com/user/723660/updates)对它的定义。而站在使用者的角度，**MarkEditor 到底是一款什么样的工具呢？**

对于不熟悉它的用户，会将它视作市面上「又」一款 Markdown 编辑器。但如[海波](https://sspai.com/user/723660/updates)所说：Markdown 在应用中的比重很小，按实体代码量来算，可能不过百分一二。粗暴地将它归为 Markdown 编辑器，似乎太浪费开发者在其他功能上的苦心了。

那些熟知 MarkEditor 的用户，会将它的 Markdown 模版、FilesHook1  和云端同步等[进阶功能](https://sspai.com/post/34656)用得出神入化，甚至自己会去设定主题和预览页面的 CSS。让人感觉它十分「硬核」。

这些都是 MarkEditor，却又展现的不够全面。

MarkEditor 的核心价值在于处理文字。它是一款高效的写作工具，却又不止于文字编辑器。它可以**将文字变成任何你想要的样子，去发挥文字的力量。**

无比舒适的写作工具
---------

就算其他功能说得天花乱坠，作为一款编辑器，能让人安安静静地专心码字，是最基本也是最重要的。而 MarkEditor 的写作体验可以浓缩为一个词：**舒适**。

### 视觉体验：不只是好看

网络上调侃人与人的关系是「始于五官，止于三观」。我们对一款应用的印象，也要看和它的第一眼眼缘。

![](.evernote-content/A28C47A3-57BE-42FB-B01C-0D62557E9ECA/F38D1341-8C8D-4691-BBD8-E4379928BE3F.png)

Dark red 主题

和 [Ulysses](https://sspai.com/search/article?q=ulysses)，[MWeb](https://sspai.com/post/45521) 一样，MarkEditor 也提供了不同配色的主题，来满足不同人的喜好。这些主题素雅耐看，其中的「dark red 」主题和 macOS Mojave 的黑暗模式简直天生一对。

但是和其他编辑器不同的是，MarkEditor 还提供了**主题生成器**，让不懂代码的人，也可以自己尝试定义一套属于自己的主题。

主题生成器里你可以以 HEX 格式**设定编辑器各个界面元素的颜色**。不懂什么是 HEX？没关系，点击输入框旁边的魔法棒，就可以在系统的颜料盘中直接选定颜色。没有耐心一个个去设定？没关系，在主题生成器里点击「尝试随机」，可以尝试出不同的组合。这个功能实用之余，就像个埋好的彩蛋，拿来消遣解压也十分有趣。

![](.evernote-content/A28C47A3-57BE-42FB-B01C-0D62557E9ECA/426D8199-9D22-4A48-A409-3289D33580D0.gif)

随机尝试主题配色

有趣的是，MarkEditor 还有一个**模糊效果**选项，也就是毛玻璃效果。使用 Mac 和 iPhone 用户都不陌生，因为这两个系统的 Dock 栏都使用了毛玻璃效果。打开之后，MarkEditor 会和你精心挑选的桌面壁显得更加和谐。

光有这些「表面功夫」还不够，MarkEditor 的「内功」也没落下 。熟悉 Markdown 的读者都知道，Markdown 文稿要转换为 HTML 页面，才能显示成富文本。而在 MarkEditor 中，它要经过渲染整个页面，分析文章结构，再调整局部的显示效果这三步，最终才让用户看到**实时渲染出的精致排版效果**。

### 写作体验：你想到的，没想到的功能它都有

之所以用「舒适」这个词来形容 MarkEditor，是因为你不仅能拿它来码字。码字之外的事，也可以在 MarkEditor 来完成。

互联网写作，图片为王。这可能是为很多人不屑、但是又不能忽视的一条规律。通常写作都是写好文字稿，再打开浏览器去找配图，下载，最后插入到文稿中。而在 MarkEditor 中，**你可以直接从 Unsplash 中插入图片**。

![](.evernote-content/A28C47A3-57BE-42FB-B01C-0D62557E9ECA/7787B522-FE01-40C5-8DE7-AD578BDB5FEB.gif)

从 Unsplash 中下载插入图片

点击菜单栏中的「写作 - Unsplash」，就可以直接查看最新来自 Unsplash 的图片。Unsplash 是网络上知名的无版权图片库，所以你可以放心的使用图片。如果你需要某个特定主题的图片，使用英文关键词搜索即可。点击一下图片，就可以直接插入到文稿中。

如果是写旅行日历、个人博客之类的文章，配图就多半是自己用手机拍的照。MarkEditor 非常贴心的可以**用手机上传图片**。拿微信（当然也可以是其他扫码工具）扫一下二维码，就会进入上传图片的网页，从照片中挑选一张，立刻就能在 MarkEditor 中插入。

这些虽不是什么新功能，但它确实节约了写作者操作步骤和时间。类似的功能在 MarkEditor 中俯首皆是。

作为起源于美国的 Markdown，使用了很多英语符号作为标记符，可以给中文用户带来不少麻烦。在写 Markdown 文稿的时候，就得不断切换中英输入法。MarkEditor 提供了一项「**Markdown 标记符使用英文**」的选项。例如将它设为「右 shift 键」，在中文输入法状态下按住右 shift 键，可以直接输入英文符号。

这个功能虽小，也不能完全解决中英文输入需要切换的问题，但是每篇文章少按那么十几次输入法切换键，却能让人感到省了不少心。

顺便一提，MarkEditor 有个「**中英文间自动空格**」选项，自动加上[盘古之白](https://sspai.com/post/33549)。

篇幅有限，不能将 MarkEditor 中那些「提升写作体验的功能」一一描述，例如「居中保持」、「禅模式」等等。但是可以看得出，MarkEditor 从来不把自己只当作一款 Markdown 编辑器，而是在思考，**用户写作时需要什么功能**。

让文字去到该去的地方
----------

很多人都误以为「出了高中，我就再也不用写作了」，或者是，写作好似变成了少数人，或者特定时间才会做的事。其实不是我们不用再写了，而是我们写东西的地方变了。

高中时我们只在考卷上写，而现在我们天天要发微博、朋友圈，上张大妈（什么值得买）和小红书写家电和化妆品的分享。写作的地方变了，写作工具就必须与时俱进。

MarkEditor 不仅想让我们写，还**帮我们把文字分享到不同的地方去**。

### 少数派与博客

如果你在少数派上写过文章，又恰巧不习惯在浏览器中写作。现在你可以**用 MarkEditor 直接在少数派发布文章**了。

MarkEditor 在 [1.0 版本](https://sspai.com/post/34656)就可以将文章直接发布到 [Bitcron](https://bitcron.com/)，或者是七牛和自定义的云端后台。现在 MarkEditor 和少数派进行合作，让用户可以**用 MarkEditor 上传提交自己的文章到 Matrix 和首页**，或者将自己在少数派的文章下载到本地。

![](.evernote-content/A28C47A3-57BE-42FB-B01C-0D62557E9ECA/4045ADD5-EBED-4CE2-8C63-23297F14ED90.png)

从少数派下载的文章

有了 MarkEditor 的加持，各位读者可以轻松地在本地写文章、修改文章。等到要发布的时候，不用再拷贝成富文本再贴到少数派的网页编辑器中，而是通过修改元数据，然后一键同步到少数派后台。

这项功能还在内测当中，测试结束之后会面向所有读者开放。

除了新增对少数派的支持，MarkEditor 还增加对了[语雀](https://www.yuque.com/)的支持。

### 微信公众号与朋友圈

在微信中写东西可以分为两类：微信公众号和朋友圈。

喜爱经营微信公众号的朋友，肯定都对排版文章这件事心有余悸。如果不熟悉 CSS 代码或者一些自动化工具，就得将文章贴进浏览器中的微信编辑器中，手动一篇篇的去调整排版。

和在文章中插图的体验一样，MarkEditor 将微信公众号排版集成在编辑器之内，大大缩减了排版所需的工作量。

![](.evernote-content/A28C47A3-57BE-42FB-B01C-0D62557E9ECA/EFA54CCB-4494-4E60-A63E-E60A05C67E0C.png)

调整微信公众号格式

当你写好公众号文章之后，点击「导出 - **复制为微信公众号格式**」，就直接能得到排版好的文章！MarkEditor 内置了三种简单的文章样式。如果不够满意，可以想主题生成器一样，通过可视化的界面，去设置一套自己喜欢的样式。而如果你身边有设计师朋友，懂一些 HTML 和 CSS 样式的编写，向他们请教后，可以直接将 HTML 和 CSS 代码放到「**微信格式自定义**」中，得到设计好的样式。

而对于平日里爱发朋友圈或者微博的用户，MarkEditor 也有妙用。

如果你想写长一点的内容，可以像写文章一样在 MarkEditor 中写好，再导出中选择「**导出为图片 - 通过二维码发送到微信**」。当然，导出为图片的过程中，Markdown 就已经被渲染为适合直接发布的富文本了。

![](.evernote-content/A28C47A3-57BE-42FB-B01C-0D62557E9ECA/0E0D6292-8BDA-4518-B691-4E4C00981B51.gif)

选中图片导出为图片

当然在朋友圈，好看的图片才是主角，而 MarkEditor 有一招会让你发的图片格调更高。

将要发的图插入在 MarkEditor 中，配上表达自己心情的文字，选中图片和文字后右键菜单中，有一项「**将选择文字导出为图片**」。和之前的功能不同，这里的导出为图片可以设置各种样式模版，一张单调的照片，就变成了样式精美的图片。

其实将文章和选中文字导出为图片发送到微信，只是这个功能的冰山一角。在 MarkEditor 中你可以设定图片的排版、尺寸，甚至文字的字体。如果你熟练地使用这个功能，甚至能在 MarkEditor 中就全部完成文章插图的任务。

除了常见的博客和图片、PDF 等方式来分享自己的文章，MarkEditor 还有许多分享方式：

* **MarkEditor URL**：可以直接生成编辑器中一篇文章的网址，将网址分享任意给你的朋友。在分享 MarkEditor URL 时，本地图片会**自动上传**，你不需要额外配置图床用于上传图片；
* **邮件列表**：直接将文稿作为内容群发邮件；
* **Wiki 站点**：将 MarkEditor 中整个工作目录作为一个类似维基百科的网页。

相比 Word 这类传统工具，MarkEditor 更贴近互联网写作的需求；而相比 Ulysses，MarkEditor 则更懂中国用户的需求。Ulysses 中迟迟不加入的图床功能，早就在 MarkEditor 中实现。

好的锤子，能钉更多的钉子
------------

任何工具间的比拼，都要落到对工作效率的提升效果上。MarkEditor 始于 Markdown 编辑器，能做的却不止于文字。

![](.evernote-content/A28C47A3-57BE-42FB-B01C-0D62557E9ECA/948F2BEB-967B-4BEE-9DE7-66F1912739B2.png)

标注出连词和代词

熟悉 [iA Writer](https://ia.net/writer) 的用户，可能都非常羡慕它的语义分析功能。它能标出文章中的动词、名词、副词等不同词性，帮助写作者精炼自己的文章。但只想功能只适用于英文。终于，在 MarkEditor 中可以**对中文进行词性标注**。

Markdown 被用于写文章，这点人尽皆知。不过文章是由图片和文字组成，幻灯片不也是吗，我们为什么不用 Markdwon 快速地创建一些简单的幻灯片呢？

在 Markdown Editor 里，就**可以用 Markdown 创建幻灯片**。

用 Markdown 做幻灯片时，使用的还是基本的语法标记符, 但是展示时需要打开「插件 - Markdown to Slide 」选项。而幻灯片的展示，则是在浏览器中。如果你是连接了投影仪来做演讲，就可以在自己电脑上控制翻页，将浏览器最大化投射出去。

![](.evernote-content/A28C47A3-57BE-42FB-B01C-0D62557E9ECA/054E6C33-558A-4A4F-AB71-F4A7D518A305.gif)

幻灯片在浏览器中的效果

词性标注和幻灯片这两项功能看似属于写作之外，但它们却帮我们节省了切换工具的成本。我们可以不用眼睛一行一行的阅读修改文稿，不用打开 Keynote、PowerPoint 去做幻灯片，在 MarkEditor 中，就顺畅地完成整项工作。

MarkEditor 还有许多提升工作效率的特性，例如**代码笔记，Markdown 模版**还有程序员都懂的 **FilesHook**。这些特性的加入并不是开发者心血来潮，而是 MarkEditor 作为一款工具，它应该能满足用户在使用它时，想到的、没想到的所有需求。

写在最后
----

在和开发者海波的交流中，他说到购买 MarkEditor 的用户中，专业版的用户比例明显高于其他应用。MarkEditor 也一直以「高精尖」的形象流传于用户之间。

但是在我看来，MarkEditor 首先是一款让人觉得**舒适的文字编辑器**；其次能贴合实际的需求，**让文字变成合适的样子**，去到它该去的地方；最后，MarkEditor 不忘它的工具属性，想要**提升用户的工作效率**。

写到这里，又想起官网的那句话：

> 好的写作体验，更好的效率利器。
