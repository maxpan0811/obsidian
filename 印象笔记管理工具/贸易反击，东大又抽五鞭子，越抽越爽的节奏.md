# 贸易反击，东大又抽五鞭子，越抽越爽的节奏

原创学爸蛋总超级学爸

客观地说，懂王（税王）上台后中美关系是有改善的，最起码人家拿出了实际行动示好。就拿中国人最隆重的传统节日——春节来说吧，牢美可是真用心了。

从大年初一开始，每天摔一架飞机给我们看。初一摔了一架F35；初二是对口节目，逗哏的是黑鹰，捧哏的是一架庞巴迪；初三，费城一架医疗包机表演陨石撞地球。

![](.evernote-content/06B0A99B-6AD4-4BE6-A2FB-81DB8880D68A/4CEF2B52-47B9-422F-BDE9-0026A767BC1F.png)

初四，达美航空一架客机表演襟翼故障应对。初五，美国联合航空公司为大家表演机翼喷火。初六，美国达美航空一架客机表演胎压不足如何降落。初七，美国两架飞机表演了吻戏。

从初一到初七整个春节期间，牢美出手极为阔绰，不间断为中国人民表演飞机大冒险，给我们提供了不可估量的情绪价值，这种示好方式我很喜欢。

不过示好归示好，美国并没有放弃对华遏制。就在2月1日，美方以芬太尼等问题为由，宣布对中国输美产品加征10%的关税。

单从这个借口上，我们至少可以获得两点信息：第一，牢美已经美驴计穷了，竟然拿禁毒找东大说事儿。第二，美国的毒品泛滥已经非常严重。

牢美这是典型甩锅行为，把自己的问题推给别人。中国是世界上禁毒力量最大的国家，没有之一。在国际禁毒合作上，也是最积极的国家之一。

去年年底，中美达成一项互换协议，美国释放三名中国特工，以换回三名美国人。其中一名美国人叫斯威丹，他是一名重要的跨国犯罪集团关键人物，负责从加拿大和美国招募毒贩生产毒品。

2012年他抵达中国后，被中国警方逮捕，并于2019年因贩毒被中国司法机关判处死刑。这竟然引起了美国的抗议，美国国务院一直在试图捞人（怀疑有人收了黑钱），说我们抓错了。

你说滑稽不？一个危害美国的大毒枭，被中国抓住之后并被判处死刑，而美国则不惜用中国特工做交换，也要把他捞回去。为此美国众议院以418:0通过一项法案，要求中国尽快释放。

![](.evernote-content/06B0A99B-6AD4-4BE6-A2FB-81DB8880D68A/E16F87B4-1D80-48F1-A718-2F247A63EBD7.png)

看来史密斯专员们都很喜欢他啊。史密斯专员们是喜欢大毒枭吗？错，他们是喜欢自己的小金库罢了。业界早有一句话，CIA才是世界上最大的贩毒集团。

所以牢美拿芬太尼说事儿，分明就是国际政治版《狼和小羊》的故事。拿这个说事，其实是想把一些美国人对毒品问题的愤怒，转移到东大头上而已。

不过美国的毒品问题的确很严重。自2021年起，美国每年因吸毒死亡的人数，超过了十万。需要注意，美国为了照顾瘾君子选民的情绪，不说吸毒而说药物过量。

另一组数据也说明美国毒品泛滥有多严重：美国以5%的世界人口，消费了全球80%的阿片类药物。当然，美国人民也有贡献，以5%的人口，贡献了全球70%的血浆和血液制品。

2023年牢美吸毒死亡的人数，超过了11万。按照前几年的增长速度，2024年的死亡人数应该在12万左右，其中八成是芬太尼等阿片类药物。

这个人数说少不少，说多也不多。说少不少，因为这个人数是美国枪击案死亡人数的两倍；不过说多也不多，光新冠疫情，美国还死了一百多万呢。

芬太尼这玩意太上头了，可以说是毒品王中王，药效是海洛因的50到100倍，特别是芬太尼衍生物卡芬太尼，其药效和毒性是海洛因5000倍，2毫克就足以使一个成年人死亡。

总之，芬太尼只是税王贸易霸凌的一个借口，美国的毒品泛滥，跟东大完全没一毛钱关系。如果东大出手（纵容贩毒），那种美国毒品市场，根本不是现在这个样子。

我们七年前就说过，贸易战是一场互相伤害。但面对贸易霸凌，我们不能妥协。用毛泽东思想来解释，那就是以斗争求和平则和平存，以妥协求和平则和平亡。

七年前我们就套用过伟大教员的话：贸易战要打多久，我看我们不要做决定……让他们去做决定，他们要打多久，就打多久，打到我们完全胜。

再用伟大教员另一句话说，那就是“敌不犯我，我不犯人；敌若犯我，我必犯人”。七年前我们没有妥协，现在我们更不可能妥协了。

![](.evernote-content/06B0A99B-6AD4-4BE6-A2FB-81DB8880D68A/3782A88C-E4EB-40D7-96A9-BEAA950952B5.png)

七年前我们有什么？芯片产业一塌糊涂，操作系统也是一片空白，华为鸿蒙甚至连PPT都没有。那时候有一批公知鼓噪，说现在投降是代价最小的时候。

事实证明，他们说的是对的。只不过他们找错了人，他们应该拿着这个论调去找牢美。贸易反击战七年以来，中国的贸易顺差不断扩大，2024年已经突破了万亿。

2025年的中国，已经不是2018年的中国了。双六代机有了，新型大平头有了，机器狗也已经可以玩二人转了，大模型弯道超车了，纯血鸿蒙有了，高端芯片也攻克了。

2024年底，中国的贸易反击战已经呈现了新的态势，东大越来越损了，反击速度越来越快，反击力度越来越大，反击招数越来越花样繁多。

蛋总观察到的现象，东大的对美贸易反击战，简直变成了内卷——可以说是五部（委）争功。那架势就像是：求求你了，该我了，你就让我打他一拳吧，就一拳。

![](.evernote-content/06B0A99B-6AD4-4BE6-A2FB-81DB8880D68A/B4FFBFED-6D62-4918-BDC5-B9116C733734.png)

比如年前的12月，对美贸易反击（科技反击），中国就祭出了闪电五连鞭，商务部、外交部、四大行业协会、财政部、市场监督总局几乎同时出手，开始了反霸竞赛。

![](.evernote-content/06B0A99B-6AD4-4BE6-A2FB-81DB8880D68A/DE938E52-72C3-4050-B92F-8BCEA7977C9E.png)

而这一次，中国反击似乎晚了，2月4日才开始。这有两方面原因：一方面，特朗普出手是在中国春节放假期间，另一方面，特朗普的加税政令，2月4日才生效。

我们2月4日进行反击，相关部门的工作人员已经算是紧急加班了。而且这次反击依然是组合拳，依然是稳准狠，各部门依然内卷，目前至少有5抽了。

先是国务院关税税则委员会，对于原产于美国的部分商品加征10%~15%的关税。这些商品包括煤炭、石油、天然气、农业机械、皮卡、大排量汽车等。

实话实说，对美关税战我们做不到对等，因为我们根本不需要从美国进口那么多东西。2024年中国对美出口了5246亿美元的商品，来自美国的进口只有1636亿美元。

就这我们也是策略性照顾美国生意。因为2020年1月，中美达成了一个贸易战阶段性缓和协议，我们答应未来几年多购买数千亿美元的商品，其中能源产品占了四分之一。

中国正在为了消化这额外的石油天然气发愁时，俄乌冲突爆发，欧洲石油天然气价格暴涨，中国只好含泪血赚了几百亿。来自中国的运输船从美国装货之后，直接就扑奔了欧洲。

当前，石油天然气是美国对华出口第二大类商品，这次我们加税反击可以说是稳准狠，直接抽特朗普的脸。特朗普一项很重要的目标，那就是多卖石油天然气，伺候好石油利益集团。

当然，这不影响我们含泪血赚美元，因为不入关就不用报税。如果说加税是我们的第一抽，第二抽则来自商务部和海关总署，宣布对钨、碲、铋、钼、铟相关物项实施出口管制。

这几项物资也很要命。比如钨，是制作超硬合金的绝佳材料，尤其是穿甲弹。没有钨合金，那只能用贫铀弹，用贫铀弹，意味着工人和士兵都得接收大剂量的核辐射。

而我国，则是世界上最大的钨生产国。当年，常凯申就知道用钨矿换取美援。现在中国的钨产量占了全球的80%以上，中国的钨储量超过了全球的60%。更为关键的是，中国的钨矿品位高、规模大、数量多。

铟的用途就更广泛了。触摸屏、液晶显示器、有机发光二极管、太阳能发电、激光器、中子检测、医学成像、密封材料、反射镜涂层、红外探测装置都少不了他。

对美国来说有一个好消息，那就是美国也有铟元素。坏消息是这玩意不能单独提炼，必须在铅和锌的生产中慢慢富集。中国的铟储量占了全球的72.6%，产量占了全球的三分之二。

铋这种元素，极具军事价值。用铋可以制造安全系数极高的第四代核反应堆。而且这种反应堆可以做到体型特别小、功率特别大，特别适合用在潜艇和航母等舰艇上。

铋元素还广泛用于高温超导、半导体材料、易熔合金、冶金添加剂、高质量玻璃的制造。铋合金是石油化工行业中常用的催化剂，铋还可以用于制造弹丸、火箭燃料等。

我国已然是全球最大的铋储藏国，也是最大的生产国。我国的储量占了全球的75%，产量占到了全球的70%。铋最先进的生产技术，也在中国。

再说说碲，碲是双色和多色探测装置的重要材料。军用红外探测装置中，主要材料就是碲镉汞。碲这玩意的生产方式跟镓差不多，根本不可能单独生产，因为它是电解铜的炉渣。

总之，这一鞭子可以说是稳准狠。说一千道一万卡住了这些关键原材料，就能把美国为数不多的工业企业卡得半死不活。

说完了第二鞭子，再说说商务部抽的第三鞭子——把两家美国企业纳入到了不可靠实体清单。

![](.evernote-content/06B0A99B-6AD4-4BE6-A2FB-81DB8880D68A/0C9EF5FF-A3FF-4A08-9621-526500E5B891.png)

这两家美国企业分别是PVH集团和因美纳公司。PVH集团是一家美国服装公司，因美纳公司是一家美国生物技术公司。去年11月，张辉瑞还大吹特吹给它站台，没想到打脸来得如此之快。

![](.evernote-content/06B0A99B-6AD4-4BE6-A2FB-81DB8880D68A/03B1317A-8273-4F28-9708-ADDCDAD99F89.png)

还有一鞭子，应该也是商务部抽的，那就是把美国告到了WTO。第五鞭子是市场监管总局抽的，那就是把谷歌端上了砧板——展开反垄断调查。

谷歌搜索因为不愿意遵守中国法律，自行退出了中国市场，但谷歌在中国还是有生意的，比如手机的安卓系统，还有广告业务，Chrome浏览器等。

![](.evernote-content/06B0A99B-6AD4-4BE6-A2FB-81DB8880D68A/C02C92BC-03EB-437B-BF04-ECE5E4121A5F.png)

谷歌遭遇反垄断调查是家常便饭，2017年欧盟给谷歌开出了24.2亿欧元的罚单。2018年欧盟因为手机预装谷歌搜索、Chrome浏览器、谷歌应用商店等，被欧盟罚了43.4亿。

2019年，欧盟因谷歌广告投放，给谷歌开出了14.9亿欧元的罚单。由此可见，欧盟虽然没有互联网公司，但不代表没有互联网收入。

美国司法部也经常收拾谷歌。那既然这样，我们当然不能手软。除了谷歌、英伟达，英特尔、苹果、微软也即将上桌了。

由此可见，中国对美贸易反击，可以说是非常果断。而且是多部门联动，甚至内卷。我希望把对美贸易反击，细化到各省市自治区。

不知不觉，贸易反击战已经打了7年，要知道全面抗战也才八年。七年以来，中国的进步有目共睹，美国的江河日下也显而易见。

对美国来说，最佳的投降时间是2018年3月，其次是现在。特朗普大规模裁撤政府雇员，搞不好要出大事。上一次因裁员酿成的巨祸，发生在崇祯二年。

此时此刻，白宫后边那根歪脖子树，压力很大呀。
