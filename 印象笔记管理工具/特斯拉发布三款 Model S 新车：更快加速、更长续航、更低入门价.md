# 特斯拉发布三款 Model S 新车：更快加速、更长续航、更低入门价

---

[![](.evernote-content/74AFBF2C-E9FA-4B5D-8AE6-72582AA14809/C012D2E5-1518-4B72-80E3-C4F2B5356639.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/tesla-model-s.jpg)

在昨晚的发布会上，特斯拉升级了 Model S 的三款车型。其中最新的 “Ludicrous Mode” 可以让 Model S 在 2.8 秒内从 0 加速到 60mph， 而升级后的 90 kwh的电池组则可以让 Model S 的续航里程达到 526 公里。

### 两大更新

更新后的三款车型分别是旗舰版 Model S P90D、最长里程版 Models S 90D／Model S 90、和入门版 Modle S 70。其中 P90D、90 D 都是双电机四轮驱动的版本，Model S 90 则是单电机后轮启动的版本。

Model S 此次的一大更新是搭载了密度更高的 90 kwh 电池组，比先前的 85 kwh 电池组增加了 15 英里的里程，在 65 mph 的时速下续航里程达 300 英里。

而新推出的 “Ludicrous Mode” 再次刷新了超级轿跑的加速纪录：可以在 2.8 秒内从 0 加速到 60mph（相当于在 2.9 秒内加速到时速 100 公里），与去年推出的 3.2 秒加速到 60mph 的 “Insane Mode” 相比，启动速度提升了 12.5％。

另外，最新入门版的 Model S 70 售价 7 万美元，搭载 70 kwh 的电池组，比先前的入门版便宜了 5000 美元，进一步降低了购买特斯拉的门槛。

### 两个期待

![](.evernote-content/74AFBF2C-E9FA-4B5D-8AE6-72582AA14809/FA447A96-4330-425C-B5F1-CB14FDB83DAD.jpg)

除了更新 Model S 外，马斯克还在发布会上还透露了特斯拉其他的一些计划。

值得期待的是特斯拉会在四年内推出一款新的 Roadster 车型，这款两座的敞篷跑车仅在全球售出 2600 辆，于 2011 年停产。

另一个值得期待的是将在年内推出的 SUV 车型 Model X。[华尔街日报](http://www.wsj.com/articles/tesla-to-offer-three-new-model-s-versions-1437161815)的报道称，从双电机、四轮驱动，到此次推出的更高密度的电池组都可以视为为 Model X 打下的基础，Ludicurous Mode 也不例外。

### 跳票的自动驾驶：还没准备好

Musk [在三月提到](http://www.ifanr.com/news/503738)，Model S 将在三个月内提供自动驾驶的软件更新，然而已经到七月了，自动驾驶还不见踪影。根据 Wired [报道](http://www.wired.com/2015/07/crummy-highways-delaying-self-driving-tesla/)，Musk 在发布会上说，他一周中的一半时间都开在洛杉矶的 405 号公路上，但自动驾驶在这条公路上的测试效果不甚理想，因此这一功能仍然在调试之中。

特斯拉的升级速度堪比手机，这让车主们有喜有忧。不过要享受升级并不意味着每次都要买一辆新车，比如 P85D 的用户可以通过购买 5000 美元升级包升级到 Ludicrous Mode。此前带 85kWh 电池组的车型加 3000 美元可以升级到 90 kWh 的电池。不过 Musk 良心强调，电池组的升级是针对新买家的，已经买了车的用户可以不必急着更新。此外，对于想尝鲜的用户，特斯拉也在今年四月悄悄上线了[二手车](http://www.ifanr.com/517999)销售平台，帮助用户降低购买 Model S 的成本，不过在国内尚未上线。
