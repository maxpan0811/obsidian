# 从第18到第1！Kimi K3暴力登顶前端编程擂台，76%胜率碾压Fable 5和GPT-5.6 Sol

---

来源：[打开原文](https://mp.weixin.qq.com/s/jYW4vtPrDalFDIKqw0rNvQ)

7月16日，AI评测平台Arena.ai更新了前端代码竞技场（Frontend Code Arena）的排行榜，一个名字直接冲到了榜首，Kimi K3，来自月之暗面（Moonshot AI），1679分。

身后是Anthropic的Claude Fable 5（1631分）和OpenAI的GPT-5.6 Sol（1618分）。  
 上一代Kimi-K2.6还蹲在第18名，K3一步跨了17个身位，直接坐上了王座。

Arena.ai官方发帖确认了这一结果：

"Big news: Kimi-K3 by @Kimi\_Moonshot is now #1 in the Frontend Code Arena with 1679 pts, surpassing Claude Fable 5."

“重磅消息：月之暗面的Kimi-K3以1679分登顶前端代码竞技场，超越Claude Fable 5。”

48万票、98个模型参战的擂台上，Kimi K3把两个高价闭源模型甩在了身后。  
 这事到底有多离谱？

76%的时间，人类选了它
------------

Elo分数的差距已经够醒目了，但Arena随后放出的成对胜率数据更有冲击力。

所谓成对胜率，就是给人类投票者看同一个任务下两个模型的输出，让他们选更好的那个。  
 50%意味着打平，超过50%就是赢面。

Kimi K3的数字是76%。

作为对照，Claude Fable 5是63%，GPT-5.6 Sol是58%。  
 换句话说，每四次对局里Kimi K3能拿下三次，而当前最贵的两大旗舰模型连七成都摸不到。

![](.evernote-content/C02E3F90-D70A-412E-8EC8-29BD58DEED4C/B0E82500-E607-4F09-AFF7-D4C7B9CD46AE.jpg)
![](.evernote-content/C02E3F90-D70A-412E-8EC8-29BD58DEED4C/6ECA2E9C-4934-4FFB-BBF4-768EF31503E2.jpg)

▲ 成对胜率排行：Kimi K3以76%断层领先，Fable 5和GPT-5.6 Sol分别以63%和58%居于二三位

更值得注意的是领域拆分。  
 前端竞技场把任务分成了七个领域：品牌与营销、参考设计、数据与分析、消费产品、仿真、内容创作工具、游戏。  
 Kimi K3在其中六个领域拿下第一，唯一丢掉的是游戏，那个位置属于Fable 5。

六冠加身，这已经超出了"某个角度碰巧领先"的范畴。  
 从数据看板到品牌官网，从仿真界面到内容工具，K3几乎把前端生成的主流场景全部拿下了。

但这个怪兽的底牌到底是什么？

2.8万亿参数的怪兽
----------

几乎同一时间，月之暗面正式发布了Kimi K3的技术细节。

标题开门见山，"Kimi K3: Open Frontier Intelligence"（开放的前沿智能）。  
 数字同样干脆：2.8万亿参数，100万token上下文，原生多模态。  
 官方称这是"世界首个开放的3T级模型"。

![](.evernote-content/C02E3F90-D70A-412E-8EC8-29BD58DEED4C/39B1259F-2A30-41D9-A74D-0EAB4544FBC0.jpg)
![](.evernote-content/C02E3F90-D70A-412E-8EC8-29BD58DEED4C/F35C21BC-EA2B-4817-AE00-C3724AAC5448.jpg)

▲ 月之暗面官方发布Kimi K3：2.8T参数、100万上下文、KDA解码加速6.3倍，下方为多项基准对比矩阵

技术层面，K3塞了不少硬货。  
 Kimi Delta Attention（KDA） 能在百万token上下文下把解码速度提升约6.3倍；  
 Attention Residuals（AttnRes） 只增加不到2%的计算成本就换来约25%的训练效率提升；  
 Stable LatentMoE架构在896个专家中激活16个，配合一系列训练稳定性技术，整体scaling效率相比K2提升了约2.5倍。

定价方面，API缓存命中价$0.30/百万token，未命中$3，输出$15，比Fable 5的$10/$50组合便宜一大截，但比许多小型开源模型贵得多。

最劲爆的消息留在了最后：完整模型权重将于7月27日前开放。  
 这意味着，十天之内，全球开发者和企业就可以下载这个2.8万亿参数的庞然大物，自行部署、微调、魔改。

当然，2.8T模型的本地推理通常需要专业级集群。  
 普通开发者的MacBook大概只能用来下载权重文件，然后对着硬盘发呆。

不过，在封神之前，有一盆冷水不得不泼。

别急着封神，总榜只排第9
------------

同一天，Arena.ai还放出了另一张榜：Text Arena（通用文本竞技场）。

Kimi K3在这张榜上的排名是，第9，1486分。

![](.evernote-content/C02E3F90-D70A-412E-8EC8-29BD58DEED4C/2A029A81-F004-4EB2-8511-3D3726FDE7AD.jpg)

▲ Text Arena排行榜：Kimi K3以1486分位列第9（黄框标注），前方仍有Claude Fable 5、Opus 4.6/4.7、Muse Spark等对手

第9名当然算得上第一梯队，相比K2.6的第38名又是一次巨大跃升。  
 但"前端第一"和"什么都第一"之间，隔着一道很深的沟。

月之暗面自己也没回避这一点。  
 官方技术博文明确写道：总体能力仍落后于Claude Fable 5与GPT-5.6 Sol。  
 他们还坦率列出了已知短板：对思考历史（thinking history）极度敏感，agent框架如果不回传完整思考轨迹，输出质量会剧烈抖动；  
 长程任务中容易"过度主动"，在意图模糊时替用户做决定；  
 用户体验的打磨程度仍然落后于头部闭源产品。

独立第三方Artificial Analysis的数据进一步校正了画面：K3的Intelligence Index为57，接近Opus 4.8和GPT-5.5，但仍低于Fable 5和GPT-5.6 Sol；  
 推理速度约62 token/s，低于行业均值；  
 更扎眼的是，在AA的知识准确率测试中，K3的幻觉率从K2.6的39%上升到了51%。

前端榜上的1679分仍然是真金白银的胜利，但把它读成"全面超越闭源前沿"，就像因为某人跑赢了博尔特百米就宣布他是全能运动员，过于慷慨了。

真正的赢家，可能是明天谈价的客户
----------------

社区的反应两极分化得很有意思。

一位开发者直呼离谱："从第18到第1，开源这边真的认真了。"  
  另一位用户更激进，声称已经取消了两份Claude Max订阅，转投Moonshot套餐，理由是K3在前端设计上的表现超过了Fable 5。

但最冷静的那条评论可能也最精准："最大的赢家或许是明天去谈AI账单的客户。"

这话背后的逻辑很简单。  
 当一个能在公开榜单上证明"前端任务我比你强"的模型出现，且API价格还更低，企业采购的议价筹码就直接多了一张。  
 你不一定要真的换模型，但你可以拿着这张榜单走进谈判室。

![](.evernote-content/C02E3F90-D70A-412E-8EC8-29BD58DEED4C/1F76B032-7272-4B8C-9E57-3E3114DFDCDE.jpg)

这种可感知的可替换性，才是这次事件最深层的影响。  
 过去一年里，类似的戏码反复上演：某个实验室发布接近前沿的开放权重模型，在某个公开榜单上猛冲一波，社交媒体上"反超""登顶"的标题满天飞，随后第三方综合评测又把大家拽回多维权衡。  
 DeepSeek干过，GLM干过，现在轮到了Kimi K3。

K3的特殊之处在于：它的突破点落在了前端视觉与可交互产物这个人人都能打开浏览器验证的领域，参数规模冲到了近3T，并且给出了明确的开放权重日程，7月27日。

十天后，这头2.8万亿参数的怪兽是如期出笼，还是跳票延期，将决定这个故事究竟是"开放权重的里程碑"，还是又一次"PPT发布会"。

倒计时已经开始了。
