在 OS X Yosemite 中苹果公司从不同方面对邮件应用进行了更新，可能用户都没有发现这些隐藏的新特性，以下是其中几个鲜为人知的新特性。

![](.evernote-content/FD161A9E-217D-45EC-9396-10276FF6C07D/0624184E-5AC9-45EB-B844-E0030DED8F37.png)

  
　　在 OS X Yosemite 中苹果公司从不同方面对邮件应用进行了更新，可能用户都没有发现这些隐藏的新特性，以下是其中几个鲜为人知的新特性。  
  
**Mail Drop**  
  
　　Mail Drop 支持用户将大型文件（最大可为5GB）以可下载链接的形式在邮件内发送出去。有些用户可能已经习惯在共享大型文件的时候使用第三方服务，比如 Dropbox 或者 Box 等类似能提供下载链接的服务。Mail Drop 不支持用户发送文件夹，而是需要将文件夹压缩之后才可传送。  
  
　　要使用 Mail Drop 还需要有 iCloud 账户，不过用户无需担心存储空间的问题，因为通过该方式共享的文件不会占据用户的存储空间，而且在 30 天之后链接就会失效。  
  
　　如果你想要使用 Mail Drop，首先要检查一下系统是否已经激活这个功能。打开邮件应用的偏好面板，在账户项中查看高级面板，确认已经打开“使用 Mail Drop 发送大型附件”选项。这样发送邮件时，你只需要把压缩包拖到邮件中，点击“发送”按键，文件就会被上传到 iCloud。如果你的收件人也使用邮件应用，他们接收附件的方式将与现在完全一致。如果他们使用的是其他应用或者网页电子邮件，他们将会收到一个链接，用来下载附件。  
  
**Markup**  
  
　　Markup 支持用户在邮件应用中对图片进行标记。用户把图片放入邮件中之后，点击该图片，然后将光标停放在图片上，出现一个向下的箭头。点击该箭头会出现“Markup”和“更多”选项，点击后者可将其他应用扩展添加到邮件应用中，点击前者则能够进入 Markup 菜单。你可以在图片上画画，添加评论、文字或者签名等。  
  
**子地址**  
  
　　邮件应用如何使用子地址？你可以利用“+”，在不建立新邮件账户的情况下创建特别的邮件地址。比如你的邮件地址是 [xiaoming@iCloud.com](mailto:xiaoming@iCloud.com)，你就可以通过 xiaoming+[XXX@iCloud.com](mailto:XXX@iCloud.com) 的方式来分类你收到的邮件。  
  
　　比如你创建一个名为 xiaoming+[haoyou@iCloud.com](mailto:haoyou@iCloud.com) 的子地址，那么你就可以把所有来自好友的邮件都集中到这个子地址之下。在这个邮件泛滥的年代，这难道不是一种很好的邮件过滤方式吗？  
  
**签名**  
  
　　最后再赠送一个小贴士，在Mac的邮件应用中如何创建你自己的签名呢？同样打开邮件应用的偏好面板，找到签名项。然后选择一个邮件账户，点击“+”创建一个该邮件账户可以使用的签名（点击“–”可以删除签名）。你的签名可以是文字，也可以是图片，字体和大小都没有限制。这样当你发送邮件时，你可以通过“选择签名”项来选择你需要使用的合适的签名。
