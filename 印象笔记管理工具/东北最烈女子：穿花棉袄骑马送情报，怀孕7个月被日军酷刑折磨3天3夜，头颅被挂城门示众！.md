# 东北最烈女子：穿花棉袄骑马送情报，怀孕7个月被日军酷刑折磨3天3夜，头颅被挂城门示众！

1936年深冬的东北山林，积雪没到膝盖。一个穿碎花棉袄的女人蜷在囚车里头，肚子已经显怀。日军把刺刀架在她脖子上，问杨靖宇在哪，她啐了口血沫：“狗汉奸才会卖祖宗！”这个女人就是赵玉仙，杨靖宇麾下最出名的交通员，因为总穿件带碎花的棉袄，骑一匹雪白雪白的白马，大伙儿都叫她“梅花鹿”。

![](.evernote-content/1905743F-DC90-45AE-BEEB-4D6B80876974/83261026-A765-4A1E-9589-BDA01A8A6742.jpg)

赵玉仙打小在长白山下长大，爹娘都是猎户。她十五岁就能跟着爹进山，一手枪法准得邪乎——黑夜里点支香，三十步开外，她抬手就能打灭火星子，这本事后来在抗日队伍里出了名。“九一八”那年，她才二十出头，亲眼见着日本兵闯进屯子，把没跑掉的老人孩子往草垛里塞，浇上煤油就点了火。那天她躲在柴火棚里，指甲深深掐进掌心，等鬼子走了，她连夜烧掉了家里的草房，揣着爹留下的那杆老套筒，找到正在招兵的抗联队伍。

刚开始队里的人瞧她是个女的，还穿件花衣裳，都劝她去后方做针线活。赵玉仙不乐意，抓起枪就打靶，三枪全中靶心。杨靖宇正好路过，笑着说：“这姑娘有股野劲儿，是块好料。”没过多久，她就成了交通员，专门在敌占区和根据地之间传递消息。组织上给她配了匹白马，这马是从伪军手里夺来的，通人性，还能认路。赵玉仙总说，这白马比好多大老爷们都可靠。

![](.evernote-content/1905743F-DC90-45AE-BEEB-4D6B80876974/4F39C79E-7DBA-49E8-971C-C6637E135727.jpg)

1935年冬天，她要送一份密码本去濛江。走到蛤蟆塘的时候，突然撞见一小队伪军。那地方光秃秃的，连个躲的林子都没有。赵玉仙急中生智，拍了拍白马的脖子，低声说：“往西边跑，引着他们绕圈子。”白马好像听懂了，撒开蹄子就冲伪军那边奔过去。那些伪军一看有个穿花衣的女的，还骑着这么精神的马，嗷嗷叫着就追。赵玉仙趁机钻进旁边的沟岔，踩着没过小腿的积雪，深一脚浅一脚地往封锁线赶。等甩开敌人的时候，天已经黑透了，她低头一看，棉鞋底早就磨穿了，两只脚冻得又红又肿，像揣了两块冰。可她摸了摸怀里用油布包好的密码本，一点没湿，咬着牙接着走，第二天一早准时把东西送到了联络点。

还有回她在山里执行任务，突然发起高烧，浑身烫得像火炭，迷迷糊糊地从马背上滑下来。等她稍微清醒点，看见白马正用前蹄刨着雪，刨出底下埋着的山楂果，还用嘴叼到她跟前。那果子冻得硬邦邦的，她嚼着嚼着，眼泪就下来了。在这冰天雪地里，人和马早就不是简单的主仆，是能把后背交给对方的战友。

![](.evernote-content/1905743F-DC90-45AE-BEEB-4D6B80876974/006DDC0A-F5C9-4017-9616-E5EADBC9408C.jpg)

1937年开春那次最险。她刚从一个屯子取完情报，就被日军的骑兵盯上了。二十多个鬼子骑着马在后面追，子弹嗖嗖地从耳边飞过。赵玉仙伏在马背上，心里只有一个念头：不能让情报落到鬼子手里。白马好像知道她的心思，四蹄翻飞，跑得比风还快。前面突然出现一道山涧，宽得能并排站开五个人，底下是哗哗的流水。后面的鬼子已经快追上了，赵玉仙闭着眼喊了声“跳”，白马猛地腾空而起，像一道白闪电，居然真的跃过了山涧。等鬼子赶到的时候，只能看着她的背影骂娘。打那以后，日伪那边就传开了，说抗联里有个“神出鬼没的红衣仙姑”，骑着白马能飞。其实哪是什么仙姑，就是个敢拼命的普通女人。

赵玉仙对那身花棉袄有特别的感情。那是她刚嫁人的时候，婆婆给缝的，上面绣着山丹丹花。后来婆家被鬼子害了，她就总穿着这件衣裳，像是在提醒自己，为啥要提着脑袋跟鬼子干。有回杨靖宇见她总穿这件，劝她换件素净的，不容易暴露。她笑着说：“杨司令你放心，鬼子眼里只有花姑娘，哪能想到花姑娘手里有枪？”她确实有办法，有时候把花棉袄反过来穿，里面是灰扑扑的粗布，混在逃难的人群里，谁也认不出来。

![](.evernote-content/1905743F-DC90-45AE-BEEB-4D6B80876974/45227E30-5746-4FC6-B229-729228FACDDE.jpg)

可惜这样一位好同志，最终还是栽在了叛徒手里。1936年冬天，一个被俘虏的联络员没扛住拷打，把赵玉仙的联络点供了出去。那天她刚从外面回来，正给肚子里七个月的孩子做小鞋，门突然被踹开，十几个日军端着枪冲进来。她想摸枕头底下的枪，已经来不及了。

在牢里的那些日子，鬼子用尽了法子。灌辣椒水、坐老虎凳，把烧红的烙铁往她身上按，可她就是咬紧牙关不松口。有个翻译官假惺惺地劝她：“你一个女人家，还怀着娃，何苦呢？说了杨靖宇的下落，保你活命。”赵玉仙瞪着他骂：“我肚子里的娃，生下来就该是中国人，不是亡国奴！要杀要剐随便，想让我当汉奸，做梦！”

![](.evernote-content/1905743F-DC90-45AE-BEEB-4D6B80876974/90963CB5-6F85-4B42-930E-3BCDFFDCF2D3.jpg)

临刑那天，北风跟刀子似的刮。日军把她绑在柱子上，还把她的头砍下来，挂在城门楼上示众，想吓唬老百姓。可他们没想到，老百姓夜里偷偷把她的身子收走埋了，还冒着风险把她的头取下来，跟身子合葬在一片松林里。

后来抗联战士们说，每次过那片山涧，总看见白马在崖边徘徊。有人说它在等穿花衣的主人，有人说那是赵玉仙的魂灵还在守护这片林子。直到新中国成立，当地老乡才敢把她的碎骨收进瓦罐，埋在她当年跃马而过的山脚下。如今那片山坡开满山丹丹，风一吹就像无数件花棉袄在动，远远望去，活像只奔跑的梅花鹿。

![](.evernote-content/1905743F-DC90-45AE-BEEB-4D6B80876974/11A2FCA0-5974-4542-9263-91C17ADBBF45.jpg)

赵玉仙不是什么神仙，就是个普通的东北女人。她会疼，会怕，会在白马递来山楂果时掉眼泪。可在国家遭难的时候，她揣着枪，挺着肚子，把自己活成了一把刺向敌人的刀。这样的人，才是咱中国人的骨头，是咱民族的精气神。她的血洒在长白山的雪地里，长出的不只是山丹丹，还有一代代人心里的勇气。
