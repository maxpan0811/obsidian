# “中华水塔”补水成功，颜值飙升！网友：这也太好看了~

共青团中央

8月15日是全国生态日，二十年前的今天，在浙江安吉余村，一粒思想的种子破土而生——时任浙江省委书记的习近平同志，在余村调研时，首次提出“绿水青山就是金山银山”的重要论述。“两山论”成为习近平生态文明思想的核心理念。党的十八大以来，以习近平同志为核心的党中央，把生态文明建设作为关系中华民族永续发展的根本大计，开展了一系列开创性工作。今天，“两山”论提出已满二十周年。在这一充满前瞻性的思想引领下，神州大地，山河蝶变，生态美、百姓富，处处涌动绿色生机。今天，我们就一起去看一幅人与自然和谐共生的美丽画卷，它叫三江源。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/B90D479A-99B3-4409-9C95-1E69F49FC8D9.jpg)

在中国，有一个地方被称为“中华水塔”，长江、黄河、澜沧江三大河流从这里出发，每年为下游稳定输送近千亿立方米的优质水源，它维系着中国乃至亚洲水生态的安全命脉。它，叫三江源。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/E06599C1-28C0-4282-A17F-7D92E6E59063.jpg)

这片巨大的河湖湿地是三江源国家公园，平均海拔超过4000米。这一片区域有一个非常浪漫的名字——星宿海。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/DAEBFACA-1A1E-463A-852A-6EF401509959.jpg)

从高空俯瞰，无数湖泊沼泽，在阳光的照耀下，星星点点、光彩夺目，如同绽放的花朵，点缀在高原之上。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/E0AE7933-CC93-42B4-99B6-D84B9CD0F9A6.jpg)

星宿海距离真正的黄河源头大约100公里，源头之水蜿蜒至此，造就了大量的湖泊和沼泽。黄河源流汇入星宿海之后，就会进入扎陵湖，再进入鄂陵湖。黄河之所以能日夜不息奔腾而下，离不开这些湖泊湿地强大的蓄水功能。但是，20多年前，这一片区域曾经沙化，大量湖泊消失，扎陵湖和鄂陵湖之间也曾一度断流。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/0D3951ED-FC71-4787-825B-80CC7795A99B.jpg)

三江源国家公园黄河源园区综合部部长 仁青多杰：当时断流的区域就是在这一块。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/5D729260-0381-49CE-918E-EEA97C058660.jpg)

三江源国家公园管理局生态保护处处长 李洪福：我们踩着石头直接就跳过去了，当时我们感觉到触目惊心。

**黄河源头断流

湖泊数量锐减**

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/EF4E15E9-3E45-46EF-9D90-0B9334DEC5D1.jpg)

站在7月的黄河源头极目远眺，扎陵湖和鄂陵湖，就像两块镶嵌在高原上的蓝宝石，美得让人心醉。扎陵湖和鄂陵湖所在的玛多县，曾被称为“千湖之县”，有大小湖泊4000多个。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/89B259E7-DBBF-42B0-B2FC-88A554A34DC7.jpg)

但从上世纪90年代开始，气候变化、鼠害肆虐以及过度放牧等因素，导致草地植被退化。最低谷时，湖泊数量锐减到1800多个，草原黑土滩现象也日益严重。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/4C3D0FC7-183F-43BB-B8E2-3D38B1F399FD.jpg)

三江源国家公园管理局生态保护处处长 李洪福：水源涵养能力降低，水土保持能力也下降，整体河道的断流影响到整个源头的生态系统。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/F379F063-8E1B-4477-AA1B-E439B76A77BE.jpg)

在海拔4300米的三江源黄河源园区，茂密的草场一望无际，但是这片绿其实来之不易。这片区域，黑土滩的情况一度非常严重，黑土滩每扩张一分，草场就退缩一分。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/EC7D1608-10E7-4364-8A89-96C5275EA55D.jpg)

三江源国家公园黄河源园区项目办 王瑞圣：黑土滩一旦形成，就证明它的生态已经恶化了。如果放任不管，那就是连片的沙丘。

**山水林田湖草沙

一体化治理显成效**

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/4DF364C8-B9AB-4B6E-9AF5-D230DAB275BB.jpg)

所谓黑土滩，是高寒海拔地区一种独特的生态恶化现象。草场形成了大片斑秃状土地，远远望去，草原就像得了皮肤病。为了阻断这种生态退化，2005年，三江源的生态治理攻坚战打响了。此后的20年间，沙化治理、退牧还草、消灭鼠害等相继展开。在一望无际的草原上，免耕机日复一日奔忙，对裸露的草场进行生态补植，与草原退化展开赛跑。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/199F7677-1BDB-4DD3-9BF6-1F9D617E92E7.jpg)

三江源国家公园黄河源园区项目干事 王瑞圣：这里治理前就像普通沙石地一样，治理完的成效就是这样子的，我们的草籽就在当时切割的凹槽里面生长，这些草籽好多根部已经进行分裂了，所以会显得非常茂密。

草补种上了，还要防止鼠兔的破坏，那就需要请来它们的天敌——老鹰助力。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/43A278BA-D308-451E-9750-AEB7B76EF403.jpg)

现在的三江源草原上，每隔一段距离就会有这样一根孤零零的水泥杆。它有一个非常形象的名字，叫招鹰灭鼠架，老鹰在观察台上观察目标，瞄准冲击。这也是三江源国家公园保护草场，防治鼠害的一项有力措施。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/21E800CB-273C-4754-9C2E-4DDDB1DF9E31.jpg)

草重新长回来了，三江源也有了新规。管理部门测算出单位面积草场的最高载畜量，指导牧民科学放牧。而为了减少人类活动干扰，让草原得以休养生息，超7万牧民告别了世代相伴的草场，开启了“易地搬迁”。曲俄加增就是其中的一员，去年他在城里安了家。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/D923A579-D21D-466E-BC01-13EA9489C04F.jpg)

青海省果洛藏族自治州玛多县扎陵湖乡擦泽村村民 曲俄加增：在县城住的时候，孩子上学老人就医都很方便，而且房子里面水、燃气、电都有，方便得很。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/E448AF7B-1280-4671-A244-ECDFCE886276.jpg)

2016年国家公园试点成立后，山水林田湖草沙一体化治理工程启动，修复荒漠、保护草地、河湖、雪山冰川以及湿地，均被列入其中。牧民们也有了新身份，区域内每户牧民家庭中，有一人担任生态管护员，从曾经的草原利用者，变身为草原保护人，平均每户牧民每年可领取超2万元的生态补助。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/E3006298-6D11-46E0-AF6E-1ECCD380F827.jpg)

三江源国家公园黄河源园区生态管护员 尼玛才让：一个是看鸟类的情况，再一个就是看有没有人为捕鱼的情况，还有沿线有没有垃圾。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/4F6750FA-7901-427D-AE5B-0BE56F44B8D0.jpg)

生态治理、退牧还草、易地搬迁、山水林田湖草沙一体化治理，二十年的坚持不懈，让三江源旧貌换了新颜。三江源黑土滩治理区域植被盖度，由治理前的不足20%提升至80%以上。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/8C3BEC50-1B21-429F-A8CD-6FEC80A0B6BB.jpg)

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/973FDFDD-A67B-4349-BD49-7F5A32954B56.jpg)

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/F72C023C-1EBB-46BA-B7A2-A15B28B56256.jpg)

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/192E7DCF-FC31-40F0-895C-C6597E2D3C4E.jpg)

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/D37D32A9-BA66-4F3E-B46F-AB7CB0F722EF.jpg)

黄河源头区域再没发生过断流，“千湖之县”玛多的湖泊数量，也增加到5849个，扎陵湖、鄂陵湖面积较2005年分别扩大19平方公里和24.5平方公里，湿地面积增加104平方公里。十年来，三江源区水源涵养量年均增幅6%以上，荒漠化和沙化土地实现“双缩减”。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/523A6287-E1DC-4E14-BA2F-5A96F0893DDA.jpg)

三江源国家公园黄河源园区生态管护员 尼玛才让：现在鸟类鱼类都多了，周边草场的环境也慢慢恢复起来了，水也涨起来了。

**野生动物种群恢复

珍稀动物频频现身**

茂密的草场回来了，丰富的水源回来了，更多的野生动物也回来了。在三江源国家公园可可西里自然保护区，盛夏时节，产仔结束的雌性藏羚羊，正带着它们的幼崽，开启回迁之路。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/11251404-7735-4765-B552-B7CBF2B606CB.jpg)

如今的三江源，有“天地空”一体化网络监测，有划定的核心保护区供野生动物安全栖息，还有迁徙路上和繁殖地的保护站、救助站护航。藏羚羊数量已由不足2万只恢复到7万多只，种群数量较保护初期增长近4倍。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/75A39464-45E2-4086-8683-8B0103A3DDAE.jpg)

藏族小伙子旦增桑周，这位三江源国家公园的特约摄影师，十年间用镜头记录着这里鲜为人知的故事。憨态可掬的藏狐、雄壮的野牦牛，在他的镜头里，各种可爱的动物，成为家乡生态改善最有力的见证。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/30D4C957-B11C-4A1D-8697-6F295F599588.jpg)

三江源国家公园摄影师 旦增桑周：距离很近。因为我们怕惊扰到野生动物，就事先做好掩体，在掩体里面拍摄雪豹的镜头。比如兔狲、猞猁，看到的概率就更高了。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/FFD1D4A6-013D-496F-BBE4-9B49A6617DE6.jpg)

二十年来，三江源的容颜正在被重新描绘，从黄河源头的断流危机到千湖重现，从曾经的沙化荒芜到绿草如茵，从过度放牧到和谐共生，从生命濒危到雪豹潜行、藏羚羊驰骋、白唇鹿成群。这片土地正在用它的变迁，诉说着守护的力量，续写着“中华水塔”的传奇，让美丽中国在这三江之源，绽放出耀眼的光芒。

![](.evernote-content/D5834423-9D26-41E4-AAF4-CD62F952269D/205C2F58-A3E3-4C66-97B7-396CA611405B.png)
