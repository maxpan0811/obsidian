线上教学需要准备什么？ 这里有给老师们的操作建议
========================

受疫情影响，几乎全国的学校都进入了停学的阶段。但是，教育部倡导「停学不停课」，于是「远程教育」这个热炒了许多年的概念终于在这次突发事件中真正走入了寻常百姓家。对很多学生来说，通过网络来远程上课是头一回，而对广大教师来说，这可能也是他们第一次遇到这样的教学要求。

不管是来自网络的发声，还是身边的反馈，我注意到了有不少数量的老师在面对远程授课这个强制性要求时遇到了一些困难，这些困难包括如何更好地呈现板书、讲解试题、互动交流、作业批改、考勤统计等，让他们在远程开展教学工作的时候无法像平时一样从容地传道授业解惑。

其实早在教育部发布和实施相关政策前，我就已经开始关注疫情期间如何解决授课这个问题。因为我爸在数年前离开公立中学之后供职于一家教育培训机构，而在疫情快速扩散后，机构就收到了教育部的停课通知，为了帮助我爸完成剩下的授课任务，我就从软硬件两方面搭建了一套简单易上手的解决方案。在经过了两天的实践摸索后，我爸已经可以脱离我的「监管」自行完成所有的远程授课工作。

因此，我打算以经过我爸验证可行的这套方案为基础，辅以多平台多用途的其它工具选项，为有同样需求的老师们提供一个可参考的思路，以帮助你们可以更快地适应远程授课的工作，并更好地为学生们提供非常时期正常学习的机会。

找一块黑板
-----

在平日的教学情况下，使用投影仪或者电子白板等设备来展示课件已经是一件稀疏平常的事情了，所以把这些电子文档通过远程授课的方式展示给学生也是一项水到渠成的工作，只需在直播或者远程视频会议的时候把相应的应用窗口共享出去就可以，非常简单易操作。

然而，老师们在教室里上课时还可以使用传统的黑板来做辅助的讲解分析，随手一画就可以方便地帮助学生们厘清头绪。特别是对于理工科的老师，非常需要黑板这样的工具来画坐标系、几何图形、化学方程式等，否则教学效果必然大打折扣。那如果换到线上授课，上哪儿去找这一块黑板呢？

### 绘制 App 推荐

#### Microsoft Whiteboard

微软这些年做了不少和教育、企业协作相关的软件，[Microsoft Whiteboard](https://products.office.com/zh-cn/microsoft-whiteboard/digital-whiteboard-app) 就是其中最成功的产品之一，微软甚至在 Surface 系列产品中将它作为 Surface Pen 的默认快速启动应用，替换了之前的 OneNote。

Microsoft Whiteboard 是微软在 2018 年推出一款在线白板工具。其目的是为团队提供一个可无限伸展的智能画布，用于实时构思、创造和协作。相比于其它白板工具，Microsoft Whiteboard 在保证优秀的书写体验的基础上，在防误触和稳定性等方面更胜一筹，再加上智能绘制图形、绘制表格等功能，也能给老师们在远程授课的过程中提供不少的帮助。关于 Microsoft Whiteboard 更加具体的介绍，可以阅读 Power+ 的文章。

关联阅读：[最佳 iOS 在线白板工具 | Best Of](https://sspai.com/post/54922)

以 Windows 平台为例，在实际的线上教育场景中，你可以通过 Windows 自带的分屏功能，将课件或者试题放在左侧，然后将 Microsoft Whiteboard 放在右侧，这样一来就能对照左侧的内容针对性地绘图解析，方便学生更好地理解。

![](.evernote-content/B59ABC5B-3008-4131-B3AE-8EB5005360B7/A55B8408-4E36-40A2-9091-5773FA92128C.png)

如果你使用的电脑有触屏功能，那么你可以直接用手指或者触控笔在 Microsoft Whiteboard 上绘制图案。如果没有，那么你也可以使用鼠标或者触控板在上面绘制图案，虽然效果没有触控笔来得好，但是至少能用。记得先在 Microsoft Whiteboard 的设置里关闭「活动笔」选项，这样就能用鼠标或者触控板正常使用底栏上的触笔、荧光笔等工具了。

![](.evernote-content/B59ABC5B-3008-4131-B3AE-8EB5005360B7/20ED2629-987D-42B3-AFAA-4864CA50F810.png)

#### Explain Everything

如果你不喜欢 Microsoft Whiteboard，那么还可以试试这款来自于波兰创业公司 Wroclaw 的远程白板工具 Explain Everything，支持 Web、iOS、Android、Windows 等多个平台。Explain Everything 的 iPad 版还支持 Apple Pencil，同时体验还很好。在「剪贴画」中，还自带了一些图片，比如地图、几何，还可以添加文字、箭头、浏览器、图片等。

关联阅读：[远程会议很复杂？有了这块「白板」就不一样了：Explain Everything](https://sspai.com/post/47091)

![](.evernote-content/B59ABC5B-3008-4131-B3AE-8EB5005360B7/6E2E7005-9433-425A-8A81-6D6A95702934.png)

#### GeoGebra

比起前面两个可以自由绘制的白板工具，GeoGebra 最大的优势是可以方便地绘制数学科目中的各类图形。在 Mac 或者 PC 上可以使用 MatLab 等专业的工具来绘制图形，但在 iPad 上这类工具就不是很丰富，所以像 GeoGebra 这样功能强大又简单易上手的数学绘图 App 就非常难能可贵了。GeoGebra 能够绘制六种不同类型的图形，通过输入公式或者函数，直接就能得到结果。

![](.evernote-content/B59ABC5B-3008-4131-B3AE-8EB5005360B7/8770AB91-5B59-4722-A992-E55485F00BF1.png)

比如像 3D 图形，输入公式后马上得到相应的图形，非常方便。

![](.evernote-content/B59ABC5B-3008-4131-B3AE-8EB5005360B7/6EB2C975-2204-4009-994F-87CDAE3F9CDC.png)

#### FluidMath

Windows 平台上其实也有不少专门针对数学科目的绘图软件，[FluidMath](https://www.microsoft.com/en-us/p/fluidmath/9wzdncrdxbv4) 就是其实优秀的代表之一。如果你想在电脑上画一个标准的直角坐标系，一般情况下需要几步？FluidMath 的答案是随手画两条垂直的线就行，应用会自动生成一个标准的直角坐标系。

![](.evernote-content/B59ABC5B-3008-4131-B3AE-8EB5005360B7/FD501A83-8242-4AA9-8D0D-61FBF4850DC4.gif)

#### Chirys Draw

除了数学老师，化学老师应该也有绘制专业图形的需求，比方说化学方程式。在 Chriys Draw 中，你可以方便地调用 App 中的素材，高效地绘制出形形色色的化学方程式。

![](.evernote-content/B59ABC5B-3008-4131-B3AE-8EB5005360B7/0EB39F3E-D9A5-4DC5-9C4D-CE03D516C4FE.jpg)

#### 幕布

说完了理工科的老师，其实文科老师也会有不少板书的需求，比如说画一些思维导图来厘清文章中的逻辑结构、故事线或者人物关系。[幕布](https://mubu.com/) 是一款知名的国产应用，可以方便地构建大纲笔记，还能一键切换成思维导图，用来上课展示非常简洁明了。

关联阅读：[自带思维导图和文档管理，这款大纲工具能帮你快速理清思路：幕布](https://sspai.com/post/39134)

![](.evernote-content/B59ABC5B-3008-4131-B3AE-8EB5005360B7/F9A31CA8-270B-48B4-B665-F95135F6FA5A.gif)

#### Drawboard PDF

如果你需要讲解试题或者进行批注，那么在 Windows 上你可以尝试使用 Drawboard PDF。这款应用专为 Windows 二合一平板打造，可以对 PDF 文件进行批注、高亮、笔记、编辑等操作，配合手写笔使用更佳，是 Windows 上体验最优秀的手写批注应用。

### 手写外设推荐

前面推荐的 App 大部分只需鼠标或者触控板也能正常使用，但是为了有更好的绘图效果以及更高效的绘图体验，我还是推荐老师们根据自己的实际情况购置适合自己的手写设备。

#### 带触控屏幕的笔记本

其中，我最推荐的是 [Microsoft Surface Pro](https://www.microsoft.com/zh-cn/surface) 系列，Surface Pen 是必备配件，而键盘盖可以根据自己的需求选配。Surface Pen 在 Surface Pro 笔记本上可以获得相当优秀的书写体验，倒转笔身后能直接用笔帽实现橡皮擦的效果，并且可以一键打开 Microsoft Whiteboard。

#### iPad

如果你不想再买一台 Windows 电脑，并且恰好手边有一台 iPad，那么不用担心，它同样也能帮助你实现手写板书的功能。

如果你采用的是 B 站直播这样的远程授课方式，那么你可以参考一下 Bilibili 官方直播 [新手教程](https://b23.tv/av43424908)，其中从 5:05 开始会介绍一个将 iOS 画面投屏到 PC 端并共享到直播间的方法。所以你可以在 iPad 上使用备忘录 App、GoodNotes、Notability 或者前面介绍过的 Geobra、Chirys Draw 等来展示出更好的板书效果，并通过直播工具直接让学生直接在线上观看到。

![](.evernote-content/B59ABC5B-3008-4131-B3AE-8EB5005360B7/3104C735-8B49-418A-B91E-F0759BC33BB1.png)

如果你使用钉钉或者企业微信中的视频会议来作为授课的方式，那么需要转换一下思路。如果你使用的是 Mac 电脑，那么可以直接将 iPad 通过数据线连接到电脑上，然后打开 QuickTime 直接获取实时的画面，让视频会议软件在共享屏幕的时候选中 QuickTime 就可以了。如果你使用的是 Windows 电脑，那么你可以去下载一个名为 [Airplay Receiver for windows](http://api.blueberry-tek.com/download/wincast/win-airplay.zip) 的应用，通过它就可以非常简单地将 iPad 画面通过 AirPlay 投屏到 Windows 电脑了，然后在视频会议中共享 Airplay Receiver for windows 的窗口即可。根据我的测试，使用这款应用让 iPad 通过 AirPlay 投屏到 Windows 电脑后，同步延迟几乎可以忽略不计。

找一个讲台
-----

在搞定了电脑上的黑板之后，你还需要为自己准备一个讲台，可以让自己从容地来施展拳脚。

第一步，在家里选择一间合适的房间。一个合适的房间需要具备什么样的条件呢？首先，你需要确保室内的绝对安静，除了你自己的讲解声音，不会产生其它的噪音，比如说空调声、电视机声、空气净化器声、人声等，就算是平时很难引起注意的噪音，在直播或者视频会议的过程中都很容易给听众造成不好的收听体验。其次，你选择的房间最好还有不错的隔音效果，因为你控制得了室内的噪音，却无法控制室外的噪音，所以隔音好有助于降低外界因素对收音效果的影响。

第二步，如果你需要真人出镜，那么尽量选择一个浅色系的纯色背景，一面白墙就是不错的选择。如果家里的装潢无法满足这样的要求，那么也尽量选择简单一点的背景，因为这样一来可以让学生在观看的时候减少视觉的干扰并降低眼疲劳出现的概率。与此同时，把房间里所有的灯都打开，确保光线充足，以获得更好的视频效果。如果房间里原来装的灯不够亮，还可以另外准备一些摄影灯，不过这样要求就比较高了，大家按需准备吧。

第三步，测试摄像头和麦克风。如果你的电脑已经有内置的摄像头和麦克风，可以先自己录制一段视频，观察一下摄像头的清晰度与角度是否合适，以及麦克风的收音效果是否合格。如果内置的摄像头和麦克风的录制效果不佳，也可以考虑购置外设来解决。具体的录音外设选购与布置可以参考少数派之前的播客入门文章。

关联阅读：[播客制作入门指南](https://sspai.com/post/56285)

第四步，确保网络环境的通畅。与平时看看视频、浏览网页不同，直播和视频会议需要更大的带宽才可以保证画面的稳定以及声音的清晰，所以我建议家里的带宽可以保证在 100M 以上，当然是越高越好。如果有条件的话，尽量使用网线直接连接电脑；如果没有条件的话，尽量在房间里放置一个路由器，并选择 5G 频段的 Wi-Fi 网络；如果路由器离你的房间较远，那么选择 2.4G 频段的 Wi-Fi 网络虽然不及 5G 频道的极限速度快，但是可以保证更好的信号强度以及更高的稳定性。

找一间教室
-----

既然在疫情期间无法到学校正常开课，那么我们就需要在线上开一个新的教室，让学生们可以在这个教室自由方便地学习知识。

一般来说，学校会和政府教育部门统一采购一部份教育相关的软件或者服务，而这次钉钉和企业微信等平台成为了很多学校的首选。如果是这样的话，老师们自然没有了自由选择「教室」的权利，只能接受使用这些大企业的产品。

如果使用钉钉等团队协作工具，那么远程授课是通过视频会议的形式来展开的，由老师发起会议，学生们加入会议。在这种形式下，你可以用屏幕共享的功能将自己电脑上的课件和白板共享给学生，然后边讲解边画示意图补充。使用视频会议功能来进行远程授课的优势在于老师可以要求学生们都打开摄像头和麦克风，这样就能在第一时间接收到学生们的反馈并做出及时的解答，同时还能起到监督的作用。

![](.evernote-content/B59ABC5B-3008-4131-B3AE-8EB5005360B7/8561B3EF-EE2F-4573-80C1-D5C9A6674A08.jpg)

图片来源：钉钉官网

如果老师们没有受到来自学校的强制性要求，可以自由选择远程授课的形式，那么我更推荐 B 站直播这种更易被年轻群体们接受的形式。使用 Bilibili 官方的直播应用 —— 哔哩哔哩直播姬，老师们可以让授课的形式非常丰富，只需通过简单的部署就可以实现让课件、白板、真人出镜等在同一时间出现在直播画面中，而以弹幕为主的互动形式也可以让学生们以更加轻松的心态参与到远程授课中来。

![](.evernote-content/B59ABC5B-3008-4131-B3AE-8EB5005360B7/17496F5C-C04A-4CFA-8B12-B29AB2B4F695.jpg)

如果老师们需要录制课程视频，然后把视频分发给学生们，那就需要另一套方案了。

如果你是 Mac 用户，那么可以使用系统自带的 QuickTime 来录制课程视频。打开 QuickTime 后，在菜单栏中点击「文件 - 新建屏幕录制」，在选项中点击「麦克风 - MacBook Pro 麦克风」，然后点击屏幕上的任意位置就可以开始录制屏幕了。

关联阅读：[教程：无需第三方软件，在 Yosemite 下给 iOS 设备轻松录屏](https://sspai.com/post/27368)

![](.evernote-content/B59ABC5B-3008-4131-B3AE-8EB5005360B7/91D4EB42-5559-4F9D-A001-4202E55DD746.png)

如果你是 Windows 10 用户，可以使用 Windows 自带的 Xbox Game Bar 来录制屏幕。按下 Win 键 + G，会弹出一个浮动的界面，然后在左上角的录制按钮即可开始录屏。默认状态下录制的视频会被保存在电脑用户文件夹的「视频 - 捕获」中。

![](.evernote-content/B59ABC5B-3008-4131-B3AE-8EB5005360B7/38395E15-871F-489C-AAFE-ABD009047A6E.png)

如果你使用的是更老版本的 Windows，那么可以使用第三方工具 [Bandicam](https://www.bandicam.cn/) 来录制屏幕，开始之前记得在设置中选中自己电脑的麦克风。

![](.evernote-content/B59ABC5B-3008-4131-B3AE-8EB5005360B7/0B0F655B-C2E5-4276-822D-26A9DFC13E74.png)

与学生互动
-----

很多学校或者老师对学生们的**考勤统计**有要求，因为在线上无法督促学生到「教室」来按时上课。如果你采用的是 B 站等直播的形式，那么你可以在授课开始前让学生们提前进入直播间，然后在弹幕中打出自己的名字，你对照着名单过一次就可以。如果你采用的是钉钉等视频会议的形式，那么你可以在会议开始时对好名单点名，会更加方便和直观一点。

不管是什么科目，试卷和课外延伸阅读总是逃不掉的。平时老师们需要将资料印刷出来再分发给学生们，而在线上这个工作就简单了，只需将电子文档共享给学生们就可以了，还可以根据自己的情况选择不同的**资料分发**方案。如果你已经在使用钉钉、企业微信这样平台来授课，那么可以直接将资料上传到云盘，供学生们自由下载；如果你采用的是直播授课方案，那么可以提前组织一个微信群或者 QQ 群，用来沟通课程信息以及分发资料；如果你不想借助于这些平台型 IM 工具，那么也可以选择百度网盘或者奶牛快传这样的云盘工具。

试卷分发下去后，如何解决**作业批改**的问题呢？其实这个不是很好解决，如果你的学校之前没有一套完整的解决方案，那么现在靠自己的力量也很难解决这个问题。最简单的粗暴的方法，是让学生把试题打印出来后在上面直接答题，然后拍照传回给老师，但是这样对老师的工作压力会增大很多。如果试题中客观题较多的话，可以尝试一下金数据这样的表单工具，让学生直接在表单中填写正确答案，就可以直接统计答案的正确率，对老师来说会方便很多。

希望以上建议可以帮助到广大需要远程授课的老师们，但因为我本人不是一线教师，所以无法准确地掌握各位老师的具体需求，如果你们在这个过程中还遇到了其他的问题，欢迎在评论区留言，我和少数派的其他编辑、作者、读者们都会群策群力，尽量帮你们解决问题。
