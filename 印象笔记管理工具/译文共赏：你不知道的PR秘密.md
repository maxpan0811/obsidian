| |  |  |  |  |  |  |  | | --- | --- | --- | --- | --- | --- | --- | | |  | | --- | | 译文共赏：你不知道的PR秘密 |  |  |  | | --- | --- | | 来源：旅业链接 | 2012年10月10日 | |  | |  |  | | --- | | **原文请点击：<http://www.traveldailynews.com/columns/article/50860/five-common-public-relations-myths-hellip>**    在众多企业营销策略中，PR（公共关系）一直是被大家公认为可有可无的一项工作。虽然多数人都知道PR这个名词，但这项工作的真正含义和带来的帮助你是否清楚呢？  **误区一：发新闻稿很简单**  发新闻稿是作为PR最基本也是最为大家所知的推广方式，企业通过阶段性的新闻稿让媒体了解公司的品牌和最新活动。当然，能够被刊登或者是多次转载的新闻稿必须用的恰当。但现在很多的新闻稿只是用来博版面，简单的陈述新闻内容，并不考虑事件发生背后对大众所产生的意义或者媒体从哪些角度来发布。只是一味追求大的图片，并没有告诉报道者该新闻对于整个行业而言会产生什么影响。另外，很多企业的PR部门并没有对自己的媒体名单进行分类，每次发新闻稿都是统一发送，也不管发送的媒体和新闻内容是否对口，这种新闻稿的发送就十分没有效率。编辑/记者一天要看上百上千封的新闻稿邮件，所以PR发送的新闻稿一定要能够脱颖而出。  现在很多PR都开始选择发送Pitch Letter（快讯），内容不仅包含新闻的主要内容，你必须告诉媒体这条新闻为什么值得报道。例如：某公司新到任了一名总经理，那么你就应该考虑：他的上任是否对当地经济有所推动？他过往的经历或者是改革是否会对整个行业产生什么影响？如果稿子里能够涉及到这些内容，那么我相信这条新闻的曝光率会大大提升。  **误区二：PR可以代替销售团队的产出**  我一定要重声一点：PR并不是销售，并不会产生实际的销售量，PR只是提升品牌在市场上的可信度和知名度。当你的销售团队在跟一些大的客户时，可能他们已经听到过你的品牌甚至是很感兴趣，那么陌生拜访就会变成熟客拜访，大大提高了销售的成单率。  **误区三：PR能成就公司一夜成名**  在消费者心中建立品牌的知名度并不是一天两天的事情，多数PR公司做的推广活动都是在近3个月内会看到媒体的曝光。只要推广是一个持续性质的，那么最终得到的反馈也将随着时间推移而产生成倍的增长。  **误区四：PR很便宜**  普遍的概念认为：PR很便宜，当然和做广告相比较，PR确实会便宜一点。PR工作是全日制的，所以一般企业都会自己雇佣PR或者外包给PR公司，全全为工作处理公共关系事件。所以，你的成本并不是上百美金，而是上千美金每个月，当然PR的存在一定会让你劳有所获。  **误区五：我也能够做PR**  这一点恐怕是很多企业老板的想法，我自己也可以写新闻稿，这样就能节约成本，接下去就在办公室里等着记者来采访。但现实可能并不是这样。PR的工作不仅仅只是写新闻稿或者是每月的快讯，PR还需要考虑如何建立品牌知名度，实施营销策略，和媒体建立良好的沟通，跟踪媒体的曝光率，还要非常了解行业知识和动态。  PR的工作非常杂，每天的“to do list”都很满，我不认为企业老板每天还有这么多闲暇时间来顾及这些工作，所以还是让专业的人来做专业的事情。 | | |
