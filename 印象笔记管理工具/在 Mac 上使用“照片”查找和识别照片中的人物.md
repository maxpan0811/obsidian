# 在 Mac 上使用“照片”查找和识别照片中的人物

---

在 Mac 上使用 “照片” 查找和识别照片中的人物
==========================

首次打开 “照片” 时，它会扫描图库以标识照片中的人物。扫描完成后，您可以打开 “人物” 相簿，查看根据照片中的面孔整理的照片精选集，并给 “照片” 找到的人物分配名字。将新照片导入到图库时，“照片” 会扫描它们，并将其中的人物与您已标识的人物相匹配。您还可以手动标识照片中的人物。

*【注】*如果使用 iCloud 照片，照片中识别的人物会在所有的设备上同步。

查看图库中的人物
--------

1. 在 Mac 上的 “照片” 应用 ![](.evernote-content/6B9203D4-88F2-446B-9833-777F729031F9/2AA095D9-B10B-4238-B9C9-942CF4BAFD3C.png) 中，点按边栏中的 “人物”。

   您标记为个人收藏的人物将出现在窗口顶部的大方形中。若要将人物标记为个人收藏，请点按照片上的 “个人收藏” 按钮 ![](.evernote-content/6B9203D4-88F2-446B-9833-777F729031F9/B1FD0A19-DFEA-467D-BDCF-CE4626D02439.png)。您也可以将人物拖入或拖出顶部的个人收藏区域。
2. 连按某个面孔以查看该人物的照片。

   ![](.evernote-content/6B9203D4-88F2-446B-9833-777F729031F9/4367E185-81E6-459F-B4B2-039E5B53DC1F.png)

   若要查看整张照片或仅面孔，请点按工具栏中的 “照片” 或 “面孔” 按钮。若要查看某个人的所有照片或仅一部分，请点按 “照片” 顶部区域的 “展开” 或 “全部显示”。
3. 执行以下任一项操作：

   * *给某人的照片命名：*点按工具栏中的 “添加名字”，键入姓名，然后按下 Return 键（或从键入时显示的姓名中选取一个）。如果照片被错误标识，请按住 Control 键点按它并选取 “[*名字*] 不在此照片中” 或 “这不是 [*名字*]”。
   * *检查并确认包含此人的其他照片：*如果 “照片” 识别出人物的更多照片供您检查，工具栏中会出现一则信息。点按 “检查” 按钮以检查并确认该人物的照片，然后点按 “完成”。如果没有信息出现，请滚动到底部并点按 “确认其他照片”。“照片” 呈现匹配的面孔时，点按 “是” 来确认此人的身份。点按 “完成” 以停止检查匹配的面孔。
   * *将一张照片选为 “主要照片” 以代表某人的全部照片：*按住 Control 键点按照片，然后选取 “设为主要照片”。
   * *在地图上查看照片：*向下滚动到 “地点”，然后连按某张照片缩略图以查看该照片拍摄的位置。
   * *将某个人添加到个人收藏：*滚动到底部，然后点按 “收藏这个人”。点按 “取消收藏这个人” 以将其从个人收藏移除。
   * *从某人的照片创建回忆：*滚动到底部，然后点按 “添加到回忆”。

给 “照片” 识别的人命名
-------------

1. 在 Mac 上的 “照片” 应用 ![](.evernote-content/6B9203D4-88F2-446B-9833-777F729031F9/2AA095D9-B10B-4238-B9C9-942CF4BAFD3C.png) 中，点按边栏中的 “人物”。
2. 点按要命名人物照片上的 “名字” 按钮，键入名字，然后按下 Return 键（或选取在键入时出现的名字）。

   名字将分配给该组中的所有照片。若要给某个人的照片重新命名，请点按名字并键入新名字。

将某人的照片合并到一组
-----------

如果发现多组照片中有同一个人，您可以合并照片组。

1. 在 Mac 上的 “照片” 应用 ![](.evernote-content/6B9203D4-88F2-446B-9833-777F729031F9/2AA095D9-B10B-4238-B9C9-942CF4BAFD3C.png) 中，点按边栏中的 “人物”。
2. 选择要合并的照片组。
3. 按住 Control 键点按一组，然后选取 “合并 [*数字*] 人”。

手动识别照片中的人物并命名
-------------

部分照片可能包含没有自动命名的人物。您可以手动给他们命名。

1. 在 Mac 上的 “照片” 应用 ![](.evernote-content/6B9203D4-88F2-446B-9833-777F729031F9/2AA095D9-B10B-4238-B9C9-942CF4BAFD3C.png) 中，连按照片以打开它。
2. 执行以下一项操作：

   * 选取 “显示”>“显示面孔姓名”。点按面孔下的姓名（或点按 “未命名”），键入姓名，然后按下 Return 键（或从键入时显示的姓名中选取一个）。
   * 点按工具栏中的 “信息” 按钮 ![](.evernote-content/6B9203D4-88F2-446B-9833-777F729031F9/EC2B99D9-2BC3-4979-8595-335777D2942C.png)。点按 “添加” 按钮 ![](.evernote-content/6B9203D4-88F2-446B-9833-777F729031F9/496B4C94-072C-4C90-BA1F-0E405EE86187.png)，必要时拖移圆圈以将其放置在面孔上，点按 “名字” 栏，键入名字，然后按下 Return 键。

     您可以点按 “添加” 按钮 ![](.evernote-content/6B9203D4-88F2-446B-9833-777F729031F9/496B4C94-072C-4C90-BA1F-0E405EE86187.png) 多次，以此来标识一张照片中的多个面孔。若要移除添加到面孔的名字，请点按该人物上的 “移除” 按钮 ![](.evernote-content/6B9203D4-88F2-446B-9833-777F729031F9/F494E8F1-76AF-422D-9988-11A397993403.png)。

若要停止在照片中显示姓名，请选取 “显示”>“隐藏面孔姓名”。

移除 “人物” 相簿中某人的所有照片
------------------

1. 在 Mac 上的 “照片” 应用 ![](.evernote-content/6B9203D4-88F2-446B-9833-777F729031F9/2AA095D9-B10B-4238-B9C9-942CF4BAFD3C.png) 中，点按边栏中的 “人物”。
2. 执行以下一项操作：

   * 按住 Control 键点按人物，然后选取 “移除此人”。点按 “从人物相簿中移除”。
   * 连按该人物，然后向下滚动并点按 “从人物相簿中移除”。

若要从图库中删除照片，请参阅[在 Mac 上的 “照片” 中删除照片或恢复删除的照片](https://support.apple.com/zh-cn/guide/photos/pht3bfab2f96/4.0/mac/10.14)。
