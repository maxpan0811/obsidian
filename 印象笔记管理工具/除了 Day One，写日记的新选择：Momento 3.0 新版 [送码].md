# 除了 Day One，写日记的新选择：Momento 3.0 新版 [送码]

---

![](.evernote-content/9292AF41-25F8-41F1-9226-ADB64076401C/7CFEA5C3-CCC4-4C4A-A2F5-55B8FF2CDB4D.jpg)

如果你坚持在 iPhone 上写日记，那么想必你一定听说过 [Day One](http://sspai.com/tag/Day%20One)、[Momento](http://sspai.com/tag/Momento) 和[格志](http://sspai.com/search?q=%E6%A0%BC%E5%BF%97)吧。事实上这三款应用也是我认为 iOS 平台上最好用、也最各具特色理念的日记应用。

简单介绍一下的话，Day One 以其一直以来的优雅而著称，加上其编辑器对 [Markdown](http://sspai.com/tag/Markdown) 和 [TextExpander](http://sspai.com/tag/TextExpander) 的支持，以及 iOS 和 OS X 的双平台支持，如果你是一个重度的手机日记码字用户，那么绝对是不二之选。

Momento 虽然不像 [Day One](http://sspai.com/tag/Day%20One) 那么出名，但从 2010 年推出以来，其「自动化写日记」的理念一直深得我心。像我这样实在没耐心写太多字，写日记更多是出于记录自己生活轨迹的用户来说，Momento 可以很方便地收集我在各个平台的时间线内容汇总在一起。

格志则更加简单，它把写日记的形式变成了自问自答，通过每天回答几个预设的问题，来记录自己的每一天生活，用来做晨间日记或晚间回顾是个不错的选择，也是国内开发商难得的佳作。

Momento 3：全新的界面设计
-----------------

前面说了这么多，但实际今天的主角是 Momento 3。自 2010 年 Momento 首版发布以后，这已经是第五年了，Momento 3 也是以一款全新的应用重新登场，之前的二代则更名为 Momento Classic 免费向所有人开放。

Momento 3 第一眼最直观的变化，就是焕然一新的设计，从图标到 App 内的风格和布局，都有了一定的调整。

![](.evernote-content/9292AF41-25F8-41F1-9226-ADB64076401C/C9E93096-CA6A-4D86-8B20-C4C25BD85E4A.jpg)Momento 2

![](.evernote-content/9292AF41-25F8-41F1-9226-ADB64076401C/ED3DBB93-3D70-40E7-A799-FD65052612CE.jpg)Day One

可以看到，Momento 3 的主色调从之前的暗红 + 棕色调整成了更加活泼的橙色，重新设计了对图片展示更加友好的时间线视图，并支持按天 / 月 / 年查看。除了标签之外，还加入了按人和地点筛选日记的维度，并推出了「Events」的概念。写日记的按钮也从原来的右上角调整到了更加醒目的界面下方的中央，和微博的设计类似。此外，Momento 3 也去掉了旧版中的日历视图。

相比 [Day One](http://sspai.com/tag/Day%20One)，Momento 3 的主界面直接展示了时间线和日记内容，其它更多的功能则以 Tab 的形式来切换查看，个人认为比 Day One 的首页更加直观高效。随着手机屏幕的越来越大，把写日记的按钮放在屏幕下方的中央，也比 Day One 更友好。当然，最近开始内测的 Day One 2.0 也有很大变化，目前已知的有支持多图和多日记本，届时我们再拿两者好好对比。

改版后的时间线，和朋友圈一样
--------------

除了界面变化，Momento 3 的最大改进就是时间线的展示方式了。

Momento 每条记录中可以插入多张图片，而且不同于之前固定高度的列表排版，现在的 Momento 会自动根据照片张数作不同的布局（就和朋友圈、[墨客](http://sspai.com/tag/%E5%A2%A8%E5%AE%A2)一样）。同时，你在哪里、和谁一起、打了什么标签，都会像一段文字一样自动拼接在一起，而不会显得与日记内容有生硬、割裂的感觉。

作为 Momento 主打的 Feeds 功能也保留下来，你可以从 Facebook、Twitter、Instagram 等平台导入自己的社交网络内容，甚至还可以导入 Uber 的行程记录。现在这些导入的 Feeds 也更加智能了，例如推文可以自动识别出你 @ 的人、所在的位置，以及打上 # 的话题标签。

![](.evernote-content/9292AF41-25F8-41F1-9226-ADB64076401C/4438E249-046F-40C1-B93B-9E1729CCC858.jpg)

细节之处，处处不同
---------

写日记时，Momento 3 更加突出了位置、人、标签和照片这四大元素，直接放置了四个非常醒目的大按钮在下方底栏。

在添加照片时，新版 Momento 比以前更加友好。之前选择照片的界面就像原生照片应用里的「相簿」，而现在则更像「时刻」，会按日期和地点自动将照片分组，你甚至可以很方便地一键选中某一天的所有照片。不过，Momento 3 存储照片的方式有些奇怪。虽然加入了 iCloud 同步，但这些照片并不存在 Momento 3 中，只是指向 iCloud 照片库的照片所在位置。这意味着如果你在 iCloud 中删除了某张照片，那么 Momento 中对应的这张照片也就不在了。官方的说法是为了节省空间，但这样一来，一些想藏进日记的照片就不能同时在系统相册中删除了。

除此之外，Momento 3 还有 **Events** 这个新功能。我们经常有些日记前后是有关联的，比如出国旅游之类很多都是连续好几天，Events 就是可以将一段时间内的内容赋予一个特定的主题，相当于一个个性化的时间范围筛选器。

据官方表示，Momento 3 是一款在设计和代码上都彻底推倒重来的应用，之后会陆续支持更多新功能，包括对 iPad 和 Apple Watch 的支持。但也因此，之前的版本已经维护了五年，开发商为了生存需要，Momento 3 将作为一款新应用[上架 App Store](https://itunes.apple.com/cn/app/apple-store/id980592846?mt=8&uo=4&at=10lJSw)，售价 18 元。

最后，不管你是不是 Day One 用户，我都推荐你试试 Momento 3。两者虽然各有所长，但 Momento 3 对于非文字重度记录者来说，对图片的展示真的更加友好，除了还缺少一款对应的 Mac 版客户端外，一切都很好。
