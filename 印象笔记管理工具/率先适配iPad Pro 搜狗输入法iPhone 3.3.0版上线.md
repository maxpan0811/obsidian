# 率先适配iPad Pro 搜狗输入法iPhone 3.3.0版上线

---

[![](.evernote-content/ACE8EA6B-1014-4AEC-B6FD-CE72C59164C7/2859CF30-E488-4F91-9849-D4FC756C9B7B.png)](http://www.feng.com/iPad/Company_news/2015-11-11/First-to-fit-the-Pro-sogou-input-method-the-iPhone-version-3.3.0-online_629825.shtml)

     近日，搜狗输入法iPhone版在App Store中上线了最新3.3.0版本，一大波创新元素的加入，给这款基础软件带来了更多可玩性，让广大用户获得了更加乐趣无穷的输入体验。值得一提的是，针对即将开售的iPad Pro，搜狗输入法已经提前完成适配，至此，搜狗输入法率先实现了对苹果全线产品的完美支持，果然是好基友一辈子。已经准备入手尝鲜的小伙伴们，也可以同步在更大的iPad上体验搜狗输入法的乐趣了！

![](.evernote-content/ACE8EA6B-1014-4AEC-B6FD-CE72C59164C7/05ECE58F-FD94-464E-B428-E319774D6477.png)

Get表达新姿势 助你登顶朋友圈

如果要问最近朋友圈最刷屏的东东，绝对是“能翻译变文字的问号”和“被收起一半的文字”，由这两种形式衍生出的段子，给小伙伴们带来了无数欢乐。搜狗输入法iPhone 3.3.0版与时俱进，在快捷短语中火速加入了“密文”与“朋友圈”，并精选了大量段子与表白文字，轻松一键即可发送，正苦于get不到这两种表达技巧的网友，分分钟就能登上潮流顶峰，变身朋友圈反转段子王。尤其是今天恰逢双11，如果你还没有发现密文是一种表白的正确姿势，那你就已经OUT了！

![](.evernote-content/ACE8EA6B-1014-4AEC-B6FD-CE72C59164C7/3A607A77-43A5-47CC-961E-870E98C7B650.png)

“密文”“朋友圈”之外，快捷短语中，还加入了另一类流行元素——“弹幕”，各种常见于弹幕中用于插科打诨的经典用语，让你在和朋友们的对话中，可以随机插入更加无厘头的表达方式，绝对是打破沉默、尴尬气氛的救星。

![](.evernote-content/ACE8EA6B-1014-4AEC-B6FD-CE72C59164C7/2859CF30-E488-4F91-9849-D4FC756C9B7B.png)

更多内涵优化 让表达更有内涵

此次更新，搜狗输入法团队除了在快捷短语上脑洞大开，在之前受到大量粉丝追捧的“花漾字” “按键音”“颜文字”等功能上，也给出了诚意十足的加强措施。花漾字新增随机组合模式，不同花色任性组合，给我们的输入走出了一种独特的“飘忽”风情；在新增了“皮卡皮卡丘”“小新与小白”“超级玛丽”等音效之后，按键音让你的输入也能听出一种傻白甜萌的可爱；而种类更丰富的颜文字，也会给我们的表达增添更多“内涵”。

![](.evernote-content/ACE8EA6B-1014-4AEC-B6FD-CE72C59164C7/D645C8D1-B0E1-49A1-A6F8-51863299873C.png)

得到了这么多输入表达的新装备，是不是已经迫不及待想要和朋友们秀一下你的社交力了？别急，搜狗输入法iPhone3.3.0版还有加码，那就是耗电量降低21%！21%！21%！如此良心的优化，是不是一下子戳到了iPhone用户心中的痛点，有种想要给搜狗点赞的冲动。还没下载的小伙伴，那就赶紧动起来吧，一起来感受越来越霸气侧漏的iPhone版搜狗输入法！
