# OS X新手技巧：用终端查看和清除指令历史

---

[![](.evernote-content/938593B1-F80C-404C-A82C-9208BAE7B417/C1FE94FA-ADF4-4E82-A28F-27907013DC5C.jpg)](http://www.feng.com/iPhone/news/2016-02-26/OS-X-new-techniques-using-the-terminal-to-view-and-clear-instruction-of-history_639540.shtml)

　威锋网讯，OS X 上的终端指令能够完成一些系统设置无法完成的任务，可以给喜欢折腾的用户带来更多乐趣。但是你又知不知道如何通过终端来查看自己曾经使用过的指令呢？  
  
　在今天的教程中，我们将一起探讨如何通过终端来查看指令历史，以及清除指令历史，资深用户可忽略。  
  
　**如何在终端中查看指令历史**  
  
　1.运行终端应用并输入下列指令，按下回车  
  
　history

![](.evernote-content/938593B1-F80C-404C-A82C-9208BAE7B417/5F66B747-D96B-4B84-ABE6-ECF511B7B442.png)

2.然后就会显示整个指令行历史（这时候如果你想记录下某个指令，就可以在此进行复制和粘贴等操作

![](.evernote-content/938593B1-F80C-404C-A82C-9208BAE7B417/6496BBA9-7053-43C1-B569-A5CEDEF6A8FA.png)

**如何在终端中清除指令历史**

1.运行终端应用并输入下列指令，按下回车

history -c

![](.evernote-content/938593B1-F80C-404C-A82C-9208BAE7B417/8C78B55E-04AF-4193-8558-7A8E09F9BFB4.png)

2.这样你的指令行历史就被清除了，当然你也可以输入“history”的指令来检查指令历史是否已经被清除。

![](.evernote-content/938593B1-F80C-404C-A82C-9208BAE7B417/396C0668-EFFB-42A5-A21D-D5F5E02D5CD2.png)

由于 OS X 终端指令众多，有时候你可能会忘记记下某个指令，这时候你就可以通过终端来查看指令历史来帮助你重新记录这个指令。如果你不希望别人看到你的指令历史的话，则可以清除这些历史。
