叶檀：这是要跟中国死磕到底的节奏
================

|  |  |  |
| --- | --- | --- |
|  | 时间：2016年12月24日 08:30:44 中财网 |  |

　　请注意川普提名此人担任白宫全国贸易委员会主席这是要跟中国死磕到底的节奏  
 　　这是要跟中国死磕到底的节奏  
 　　12月21日，美国候任总统川普提名经济顾问内瓦罗（PeterNavarro）领导新组建的白宫全国贸易委员会，这给了国内支持川普的人当头又一棒。  
  
 　　内瓦罗最大的特点是，反全球化，对中国持鹰派立场，是川普团队中的珍稀动物“大学教授”“哈佛经济学博士”，热衷社会与媒体，属于社会“闻人”，书名包括《致命中国：美国是如何失去其制造业基础的》(Deathby China: How America Lost its ManufacturingBase);《卧虎：中国军事化对于世界意味着什么》(Crouching Tiger: What China's MilitarismMeans for theWorld)一书，书名与观点一样直接激进，把中国视为美国经济下行的罪魁。  
  
 　　川普在声明中说：（Peter）的论点的清晰程度和研究的透彻程度令我印象深刻。他很有先见之明地记录了全球化给美国工人们造成的伤害，并为未来复兴我们的中产阶级指明了一条道路。川普过渡团队则声明，该委员会的成立进一步证明了川普要让美国制造业再次伟大的决心，要给每一个美国人机会，通过得体的工作赚取让人满意的工资。  
  
 　　来看看川普亮瞎眼的团队。11月30日，川普过渡团队宣布，提名前高盛执行长史蒂文·努钦担任财政部长，没有从政经验，因为在投资领域的实操能力而被相中。另一位华尔街大鳄威尔伯·罗斯将领导商务部，不仅是川普的核心经济智囊，还是破产并购方面的专家，以最低的成本盘活有生存希望的企业。罗斯一定程度上属自由派士，这又有什么呢，罗斯的团队那些钢铁企业主与钢铁律师们，长期身处中美贸易摩擦最前沿，跟川普早就结合在了一起。另一位白宫首席战略顾问班农，曾任职于极右派媒体布赖特巴特新闻网，彭博称之为“美国最危险的政治人物”。  
  
 　　有两三个派别色彩不浓厚的人。住房和城市发展部部长卡森让人大跌眼镜，这位世界上首次成功给脑部连体婴儿实施分离手术的外科医生，低调，热衷社区服务，黑人，虔诚的基督教徒，成为最好的政治象征。名单公布时总是引起一片惊呼，现在，大家对川普公布什么都见怪不怪了。  
  
 　　更让人担心的是军队安全系统的鹰派人物，防长、安全顾问等，主张加强监控，增加军队力量，这与经济领域中主张以实力捍卫国家利益的右派一拍即合。  
  
 　　有些国人继续蒙住眼睛，表示川普的这一动作并不是针对中国，川普是商人，怎么有利怎么来。  
  
 　　这是自欺欺人，要让工作回归美国，恢复美国中产阶级的幸福感，川普针对的最大目标只能是中国，这个有着最多人口、相对完整的产业链、全球最大贸易顺差国。  
  
 　　来看看川普本人，希望国内那些学者们已经有人对川普的各个方面进行了深入分析，美国曾经在二战时对希特勒进行了深入的心理分析，结果与实际发生的情况极端吻合，我们也需要这样务实的团队，那些官样文章除了让自己沉浸在幻想之中，最大的功用就是浪费读者的时间。  
  
 　　站在川普的角度思考，一个成功的商人，不掩饰欲望，有强烈的表演欲，愤怒、粗暴，记仇，可以在人家的专栏上写上潦草的狗脸两字寄回去。动用一切手段达到自己的目标，能低下身段，自己给企业家电话拦下投资，到硅谷去跟扑克脸的库克等人开会。  
  
 　　只要成本可以承受、只要能够达到目的，他会大打规则擦边球，比如在税收问题上。如果情况很糟糕，他会疯狂宣泄自己的情绪，然后像个最精明的人一样，找下一次大赚一笔的机会。这种人有点婴儿人格，某种程度上使用的是流氓社会的准则。  
  
 　　川普带着自己的右翼战团轰轰向前，把美国当作一家大公司来运营，一帮基本没有从政经验的亿万富翁、强硬军人、以及务实派人士成为内阁成员，这些人可以看作是这家“美国公司”的首席财务官、首席风控官、首席智囊。  
  
 　　这家公司的运作逻辑是，用强硬手段让公司运作成功，不惜代价让这家公司成为伟大的公司，底线是不能亏本。  
  
 　　这是里根之后的又一个“重商主义”时代，让财富回流美国，让就业回流美国，这样才能从根本上解决贸易逆差。川普做出让步的惟一前提是，他从对手那儿获得了更多的东西。他要撕毁TPP，不是因为不懂战略，而是遵守一个商人的原则，不得没有得到就给予对方。从这个逻辑出发，川普可以撕毁气候协议，撕毁其他所有的东西，只要国会让他去做，只要他觉得吃亏。  
  
 　　桥水基金的达里奥表示，接下来右派政府的表现会无比有趣，债市牛市到头，低税与重商时代到来，又一个资本时代到来。  
  
 　　中国面临的情况是，或者增加贸易摩擦，或者浪费美元托住人民币汇率，不管你应不应诉，美国都会那么干，让中国让出货物贸易第一大国位置，或者花掉以前积攒的美元。更严重的情况是，美国的军费可能大幅上升，美国用进攻的方式来固守自己的老大位置。中国如果跟着军费大幅上升，后果参照上世纪80年代里根执政后期。  
  
 　　不能再蒙起眼睛，关起门来瞎说八道，甚至取消内部的市场化努力来对抗美国，这是自残，不是对抗。  
  
 　　既然美国要回归里根时代，中国惟一的办法是扩大外部的市场，同时牢牢把握住产业链完善的竞争优势，以打造品质的方法建立庞大的内部市场。如果要重商，大家一起重商，看谁能顶得下来。(叶.檀.财.经.)
