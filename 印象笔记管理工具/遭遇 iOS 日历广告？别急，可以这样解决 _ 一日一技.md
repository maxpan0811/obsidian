# 遭遇 iOS 日历广告？别急，可以这样解决 | 一日一技

---

![](.evernote-content/01BC11DD-27D2-4005-A474-DD1EA1CF8B58/1A014318-BB12-4875-A232-AD145E5D9F63.jpg)

对于 iMessage 中的垃圾短信大家早已经习以为常，苹果也在信息设置中增加了「过滤未知发件人」功能，用来关闭来自非联系人的 iMessage 通知。但是今年，许多人发现手机中的日历居然开始推送广告了，简直是防不胜防，不得不让人佩服这些广告商们的脑洞。

![](.evernote-content/01BC11DD-27D2-4005-A474-DD1EA1CF8B58/C4AB4D67-D46F-4705-8BD8-73B90F722D13.jpg)

许多朋友以为自己的手机中了病毒，其实这只是垃圾广告商利用苹果日历的一个 BUG 对你进行的日历邀请。在搜索引擎中搜索「苹果日历推」等字样，你会发现这早已成为新的营销手段。

![](.evernote-content/01BC11DD-27D2-4005-A474-DD1EA1CF8B58/4B928D03-30DA-4FA8-A190-B57B7370DBEB.jpg)

如果你不幸被加入了这种垃圾日历，请一定要注意了，即使你在收到广告日历邀请时点了「拒绝」选项，它也会一直在日历的收件箱中，无法删除。同时，无论你点击哪个选项，都会给发送者提交一个反馈，意味着他们知道这是一个活跃账户，他们会把这些活跃账户录入到另一个库中，继续进行其他广告的推送。

真的没有解决方法了吗？其实有的。

关闭 iCloud 日历
------------

如果你平时根本就不用 iCloud 的日历功能，你完全可以直接将它关闭，在「设置」里面找到「iCloud」选项，然后将日历同步关闭，同时将本地事件都删除就好了。

![](.evernote-content/01BC11DD-27D2-4005-A474-DD1EA1CF8B58/E27859DA-89DE-43BF-A925-4D86056802A0.jpg)

修改日历邀请的接收方式
-----------

如果你还需要使用 iCloud 日历，那么你可以通过修改日历邀请的接收方式，来阻止日历事件直接推送到你的 iOS 设备上。

在浏览器中登录 [iCloud](https://www.icloud.com)，进入日历管理设置，依次点击进入**右下角的齿轮图标，在「偏好设置 - 高级」中**将「收到事件的邀请方式」更改为**「发送电子邮件到……」**。之后再有类似的邀请，你就不会直接在手机上收到推送通知了，而是会直接发送到邮箱，这样邮箱可以帮你过滤掉这些垃圾内容，同时也避免了将自己是活跃用户的信息暴露给这些垃圾日历发送者。

![](.evernote-content/01BC11DD-27D2-4005-A474-DD1EA1CF8B58/477A3F55-EAB8-466E-B9D4-5CEEF45E632D.jpg)

更换 Apple ID 关联邮箱
----------------

在苹果真正给出解决方案前，最能从根本上解决这种问题的恐怕只有去更换 Apple ID 的邮箱了。

登录 [Apple ID 管理页面](https://appleid.apple.com)，点击「帐户」右上角的「编辑」然后将 Apple ID 邮箱替换为你要修改的邮箱。 不过如果你是使用 `@mac.com / @me.com / @icloud.com` 的邮箱作为 Apple ID，那么是无法进行邮箱更改的。

![](.evernote-content/01BC11DD-27D2-4005-A474-DD1EA1CF8B58/24D588D6-138C-4F83-AC5D-9226F61B9315.jpg)

建议
--

根据网上现有的案例来看，中招者大多使用了 QQ 邮箱作为自己的 Apple ID，因为广告日历的发送者通常使用脚本来一次性生成许多邮箱地址，而 QQ 邮箱是由数字 + @qq.com 的后缀组成，生成这样的邮箱地址无疑是成本最低的。

因此建议使用 QQ 邮箱作为自己 Apple ID 的朋友尽早更换为其他安全性更高、地址可生成性最低的邮箱使用。因为你无法杜绝下一次由其他垃圾广告向你推送的可能。
