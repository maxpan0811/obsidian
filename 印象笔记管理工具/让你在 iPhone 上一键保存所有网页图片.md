# 让你在 iPhone 上一键保存所有网页图片

---

在 Safari 上浏览网页的时候，偶尔会遇上一两张自己特别喜欢的**图片**，希望能**保存**下来。在 iOS 系统的 Safari 中，长按图片，然后在弹出菜单中选择 ⌈储存图像⌋ 是默认的存图方式，适用于大多数网页上的图片元素。然而，当自己希望保存的图片**属于 CSS 背景或是被赋予了链接地址**，长按保存可就不管用了。

其实，只需要一个小小的 **Safari 扩展应用**就能够解决这个问题：[Save Images](https://itunes.apple.com/us/app/save-images-downloader-for-safari/id940246761?mt=8)，一款搜刮网页图片的利器。

简单点，使用的方式简单点
------------

扩展应用好比插件，体积小，功能好。

完成安装，点击进入 **Save Images** 主界面让人一看便知这是一款扩展应用。它的使用方法也**非常简单**：

1. 首次使用时，进入 Safari，打开网页
2. 点击 ⌈共享⌋ 按钮，滑动第三栏**扩展动作**到最右边，选择 ⌈更多⌋
3. 打开 **Save Images** 的开关，点击 ⌈完成⌋
4. 在需要保存网页图片的时候，点击 ⌈共享⌋ 按钮使用 **Save Images**，它会自动提取网页内的图片
5. 选好喜欢的图片，点击 ⌈Save⌋ 保存至相册。

就这么简单！

强大点，提供的功能强大点
------------

能注意很多很多细节的插件才是好插件。

**提取图片快速保存**是 **Save Images** 的基本功能，但它想的非常周到——对网页上的一张图片，你希望知道的、做到的它都可以帮你完成。

### 1. 自定义提取规则

**提取规则**可以在应用的主界面**设置**，有三个**规则开关**：

* <img> tags | 提取网页内的图片元素，一般来说这样的图片也可以长按保存
* CSS background-image's | 从 CSS 样式中提取背景图片，非常实用
* <meta property='og:image'> | 提取 Meta 标签（用户不可见）中的图片

不同的用户可根据自己需求选择合适的提取规则。

![](.evernote-content/3DE4277E-9497-4FFD-9BDD-43B84FAB45F3/488B744E-19A1-4862-8861-C2EEAB02EBC9.png)提取规则用户做主

2. 浏览图片信息

除了提取图片，**Save Images** 也给用户提供了相对完善的**图片信息**，包括图片的名称、格式、分辨率、大小以及图片地址，并且一目了然。

![](.evernote-content/3DE4277E-9497-4FFD-9BDD-43B84FAB45F3/F6BFD3D0-A862-4357-AE29-4733759165DB.png)图片信息一览无余

3. 进一步的扩展动作

针对单张图片，**Save images** 还提供了进一步的**扩展动作**，包括指定给联系人、复制图片地址（是地址不是图片本身）、添加至阅读列表（这一项对图片似乎来说没太大意义）。另外，如果你的 **Workflow** 中有相关的动作，也可以 ⌈Run Workflow⌋。
