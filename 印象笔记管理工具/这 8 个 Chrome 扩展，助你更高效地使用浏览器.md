这 8 个 Chrome 扩展，助你更高效地使用浏览器
---------------------------

原创： 荔枝君 荔枝软荐

![](.evernote-content/B3484A4F-EA5B-44AE-8739-9BFBC277B702/5FAAE000-00F2-47CF-BE29-EE07E472EAAE.png)

为什么 Chrome 浏览器这么受欢迎？除了它强大的性能之外，众多实用的扩展程序也功不可没。

这里，荔枝君为大家推荐 **8 款 Chrome 浏览器扩****展**，可以使大家的工作效率变得更高。

![](.evernote-content/B3484A4F-EA5B-44AE-8739-9BFBC277B702/5374D564-7C5B-462A-89AA-9372FA44D67C.jpg)

关注公众号「**荔枝软荐**」，回复「**018**」即可获得以下 8 个 Chrome 扩展程序的安装包。

**▶︎** **1. 多功能标签页：Infinity 新标签页**

支持添加众多的常用网址，集成待办事项、备忘录、天气、Gmail 通知等功能。

![](.evernote-content/B3484A4F-EA5B-44AE-8739-9BFBC277B702/847F2739-045C-401D-8836-BAEC64147BA8.jpg)

****▶︎**2. 广告屏蔽：Adblock Plus**

屏蔽掉各种网站的烦人广告，避免转移注意力，且节省观看视频时的等待时间。

![](.evernote-content/B3484A4F-EA5B-44AE-8739-9BFBC277B702/3407837B-7CDE-4339-B9EF-E01455B5CE68.jpg)

****▶︎**3. 键盘操作： Vimium**

通过定义快捷键配置，取代所有鼠标操作，熟练掌握各种快捷键之后，能节约很多在鼠标和键盘间来回转换耗费的时间。

![](.evernote-content/B3484A4F-EA5B-44AE-8739-9BFBC277B702/59FD9FFE-62E7-46F4-A5BD-012BDA3724B5.jpg)

****▶︎**4. 网页阅读：简悦**

去掉网页上的干扰元素，让你立刻进入阅读模式，还可以将已优化的网页内容转为 PDF、PNG、Markdown 等格式。

![](.evernote-content/B3484A4F-EA5B-44AE-8739-9BFBC277B702/9F1625A4-5CB4-44DA-A230-A4C58C63F925.jpg)

****▶︎**5. 网页保存：Evernote Web Clipper**

将网页以不同形式保存到印象笔记中，支持完整网页、网址、截图保存，隐藏广告的方式更能去除掉页面中无用的信息。

![](.evernote-content/B3484A4F-EA5B-44AE-8739-9BFBC277B702/EA783FC9-3BA8-4FB2-9F91-BB90840286BB.jpg)

****▶︎**6. 网页截图：注释&录屏**

可以截取部分或整个网页截图，并且支持编辑、标注和模糊敏感信息。

![](.evernote-content/B3484A4F-EA5B-44AE-8739-9BFBC277B702/074BAC43-F245-4932-B368-33CDF26C8AC9.jpg)

****▶︎**7. 网页便签：Note Anywhere**

在任意网页贴上便利贴，记录下重要灵感，即使浏览器重启，便签也不会丢失。

![](.evernote-content/B3484A4F-EA5B-44AE-8739-9BFBC277B702/9B7BA10A-B30B-4525-B2F1-C7D242497250.jpg)

****▶︎**8. 英语学习：翰林英汉双解词典**

可通过必应、有道等词典，查询单词的详细释义与例句；还可翻译整个段落为中文，并保持查词历史记录以便后期复习。

![](.evernote-content/B3484A4F-EA5B-44AE-8739-9BFBC277B702/EDD83D8A-37DD-4DC6-8199-400A7F113F1D.jpg)

关注公众号「**荔枝软荐**」，回复「**018**」即可获得以上 8 个 Chrome 扩展程序的安装包。

最后，已经装了这么多拓展，是否还需要一款「 **管理扩展的扩展**」 ？

实际上，**Infinity 新标签页**就可以办到：添加「 **应用与扩展**」 项目即可实现启用、禁用、卸载所有扩展，管理更方便。

![](.evernote-content/B3484A4F-EA5B-44AE-8739-9BFBC277B702/A75AF962-39D4-45DE-8CDF-50AD7294F79C.jpg)
