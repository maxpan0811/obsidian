# Ulysses for Mac 你不得不知的 3 个实用技巧

---

![](.evernote-content/4BABC700-0698-4D7E-B4EA-77BC93016BEB/B832C2FC-202C-45B8-9AC6-9AD615803611.jpg)

我的所有少数派文章、我的书本，乃至于日记，都是在 Ulysses 中写的。Ulysses for Mac 专注于写作这一行为的每个细节，用户界面简洁无杂乱，是一款不可多得的良心之作，正如创始人 Marcus 所说：Can't beat minimal。

前段时间，我从 [Ulysses 官方博客](http://ulyssesapp.com/blog/2015/09/4-secret-ulysses-tips-for-power-users/) 看到 3 个在写作过程中很实用的技巧，在此编译分享给大家。

1. 使用全键盘流在文章内导航

如果你在写一篇很长很长的文章，你一定会在文章中插入众多的副标题。这时候导航（Navigation）面板就可以成为帮助你快速在文中跳转的工具。

![](.evernote-content/4BABC700-0698-4D7E-B4EA-77BC93016BEB/88330223-CCB9-4A11-9AE1-A4E9DBBCE310.jpg)你可以手动点击 Ulysses for Mac 顶栏右侧的按钮，然而我更喜欢的是直接按下 Command＋8 快捷键呼出之。选择任意一个标题后，使用上下箭头跳转至你想去的部分，然后按下 Return 键即可。

2. 快速跳转至上次离开的位置

在使用了 Ulysses 一段时间后，你的 Ulysses 库内想必已经存在很多篇文章了。但是如果你上一次写作是一周之前的事情了，而且没写完，现在该如何快跳转至上次离开时的编辑状态呢？

![](.evernote-content/4BABC700-0698-4D7E-B4EA-77BC93016BEB/A0BC3451-FCFC-45BC-9C35-A649967540B5.jpg)其实只需在文章列表中找到该文章，然后双击之即可。你会发现你的光标已经定位到上次你离开的地方啦。

3. 筛选视图下快速定位原文章

文章库中有太多文章，对于一个分类狂来说，务必把它们整理的服服帖帖的，给每篇文章添加「关键词」是一个好方法，有了关键字，你可以进行搜索，然而我们使用智能「筛选」（Filter）视图就可以更加方便了。

在 OS X 顶部菜单栏中选择「文件」－「新建筛选」（New Filter）即可创建一个筛选条件，你可以基于关键字，时间，和文中出现的文字等三个方面来创建适合你的筛选条件。完成后，「筛选」视图直接省区了你每次都要进行搜索的操作，因为 Ulysses 的删选视图相当于把搜索的结果放在了一起。

![](.evernote-content/4BABC700-0698-4D7E-B4EA-77BC93016BEB/77B9E6AC-DC9D-424C-AD84-06122321FC38.jpg)筛选视图相当于 OS X 平台的「智能文件夹」，如果你在查看筛选视图条目的时候需要查看原文详情，那么你可以快速定位并跳转它。在删选视图条目上右键弹出选项菜单，然后选择「在群组中显示」（Reveal in Group）项即可，非常方便。
