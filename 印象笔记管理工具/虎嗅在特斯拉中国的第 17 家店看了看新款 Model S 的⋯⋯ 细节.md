# 虎嗅在特斯拉中国的第 17 家店看了看新款 Model S 的⋯⋯ 细节

---

2016年4月19日，特斯拉新 Model S 在位于北京金港汽车公园内的特斯拉金港体验中心首发亮相，该店是特斯拉在中国大陆的第17家体验中心、北京地区的第5家体验中心。除了软件升级之后的一些功能特性之外，新 Model S 的一个亮点是HEPA高科技空气过滤系统，据特斯拉称，该系统能够过滤约99.97%的可吸入颗粒物及过敏源、细菌。

先来看一下外观上的亮点：

![](.evernote-content/B0C7E426-CCF3-479D-85FE-3E063FA015B5/37EAE6CA-4864-40AB-916C-5C3179281A51.jpg)  
放弃了进风口设计，新 Model S 采用了和 Model X 一样的前脸，换上了 LED 大灯  
![](.evernote-content/B0C7E426-CCF3-479D-85FE-3E063FA015B5/836BFAA8-324D-4365-9072-FBBCA3DC1393.jpg)  
底部用于给电池降温的进气口仍旧保留

新 Model S 在性能上的最吸引之处就是“2.9秒破百”：

![](.evernote-content/B0C7E426-CCF3-479D-85FE-3E063FA015B5/59A97934-7405-4737-B06C-B3456FE38A2E.jpg)  
双电机4轮驱动并不是新特性，越来越往标配的方向发展，是特斯拉主推的一个点

![](.evernote-content/B0C7E426-CCF3-479D-85FE-3E063FA015B5/1CC984A4-ABEF-4294-9A39-4C58FB07FCB9.jpg)  
安全方面并没有太多新的东西  
![](.evernote-content/B0C7E426-CCF3-479D-85FE-3E063FA015B5/D1618E71-27F7-4B91-A58A-9EF5A9D25C58.jpg)  
前部溃缩区域没有发动机，新 Model S 的 HEPA 空气过滤系统占据了一些空间

从其他几个角度看看：

![](.evernote-content/B0C7E426-CCF3-479D-85FE-3E063FA015B5/892B84F6-72A2-4AF7-9A33-7B1D7D94D95B.jpg)

新 Model S 从外观来讲，更加美观，性能上更加强劲。但是新 Model S 在做工的细节上还有待改进。

![](.evernote-content/B0C7E426-CCF3-479D-85FE-3E063FA015B5/42F58319-341F-4BD2-8BDA-FF637C8C0AD5.jpg)

除此之外，人们讨论比较多的内饰部分，新 Model S 的变化并不大，但是的确增加了两种木饰花纹，此外全系均配有 Alcantara 车顶内衬。目前特斯拉的确有太多的事情要做，相信内饰和做工的问题，都会在未来得到逐步改进。
