# Windows 10 发布会：在沉默中爆发

---

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/9EA2B6F5-DBF8-4492-A85A-3060F87727D0.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/windows10.jpg)

北京时间 2015 年 1 月 22 日凌晨 1 点，微软十分郑重其事地在雷德蒙德召开发布会，向大家详细介绍下一代拳头产品：Windows 10。此前，微软已经发布了 Windows 10 的技术预览版，但从后来微软有意无意泄漏的信息来看，技术预览版所透露的功能（Modern Apps 能在传统桌面执行，添加开始按钮等等），不过是冰山一角。

实际上，纳德拉履新差不多一年，Windows 10 不但证明该公司是否依然有能力继续创新，也将证明纳德拉是否正确的选择——毕竟过去一年里，纳德拉个人形象扶摇直上，但这些名气是虚的，他需要拿出一款重量级的产品来证明自己。此外，在桌面操作系统市场，Windows 7 的占有率依然高居不下，表明微软后来推出的 Windows 8 对消费者并不构成真正的吸引力；在移动市场，Windows Phone 积弱已久，原本不大的市场份额进一步下滑。

微软不能再分散自己的精力，“One Microsoft”要在一款产品上有所体现。更重要的是，微软要倾尽全力，告诉别人“我能行”，而这正是 Windows 10 推出的背景。

### 不同平台的 Windows，终归是 Windows

发布会上，第一个上台的是微软 Windows 主管 Terry Myerson。他简略地谈了谈技术预览版的参与情况（170 万测试用户），然后进入正题——

Windows 用户（Windows 7、Windows 8.1 以及 Windows 8.1）将免费获得 Windows 10 升级（发布后第一年内），而不管是 PC 上的 Windows 还是手机上的 Windows Phone，都将升级为 Windows 10。从此以后，在不同的设备上，我们不会看到不同版本的 Windows，它们将是统一的。

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/B0409A76-93B1-4BF0-B85B-4F62430390DE.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/1.jpeg)

他说，“一旦设备升级到 Windows 10，我们将持续提供支持，直到设备的生命周期结束——不必额外付费。Windows 10 的体验将是逐步改进的，而且是一直持续提升的。当我们准备好，我们将直接提供功能，而不必等待下一个版本发布。我们将 Windows 视为一个服务。”

紧接着 Terry 开始解释，Windows 10 的“统一”到底怎么回事。“随着通用版 Windows app 可以在整个产品家族当中运行，开发者可以开发一个 app，然后运行数量更多的设备上——包括 PC、平板电脑、手机、Xbox、物联网设备等等。”

### Windows 10 各项小改进

Terry 讲完后，轮到 Joe Belfiore 出场，开始演示新一代 Windows 的各项功能。

首先演示的是“开始按钮”，正如过去所展示的，它糅合了传统开始菜单栏的体验，又将 Modern app 融入其中。但 Joe 演示的版本有所不同——开始按钮可以全屏化，展示更多的内容。一个有趣的改变，这是否表明 Windows 10 将正式放弃“开始界面”？

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/2DEEB7A6-A798-4FEE-9F0C-867C011E73FE.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/win10_windows_startscreen2_Web.jpg)

Windows 10 有了新的“动作中心”（Action Center），可以快速唤出，显示在屏幕的右侧。通过它，可以快速地打开/关闭 Wi-Fi 或蓝牙，还可以直接查看最新的消息，以及系统通知等等。

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/14060B36-92B3-441B-A505-1F7EEBC42908.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/%E5%B1%8F%E5%B9%95%E5%BF%AB%E7%85%A7-2015-01-22-%E4%B8%8A%E5%8D%884.31.jpeg)

当我们拔出键盘、鼠标，Windows 10 将弹出“进入平板模式”，点击之后可以看到 Windows 10 的开始按钮以及任务栏依然保留着，不过桌面上之前打开的应用窗口，都将全屏化，然后可以通过手势来方便切换。Joe 在演示这个模式的时候，特别强调它是一个“无缝衔接、可持续性”的体验。

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/117D0770-A511-4704-B8A3-D44E371A4A0C.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/2.jpeg)

### Cortana 来到 PC，人工智能再进化

微软力推的数字助理 Cortana，有了很大的改进——Joe 演示了它作为数字助理的功能，包括展示日历上的未来事项，查阅职业棒球比赛得分，以及它作为人工智能灵性的一面。Joe 在演示功能之余，一直在“调戏”它，结果是 Cortana 不光可以回应这些玩笑，而且音调也能发生变化，就好象与人在对话一样。

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/CBCC8C9E-BE51-4959-B2C7-9AA17464DA69.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/3.jpeg)

而重要的是，Cortana 现在来到了 PC 上。在任务栏上的搜索栏，可以找到启动它的按钮，启动之后，就可以直接让它做这做那，甚至是开玩笑。Joe 说，由于 Cortana 来到 PC，因此它的体验也要针对 PC。

Joe 尝试了几个不同的指令，包括：

* “给我找关于慈善活动的 PowerPoint 幻灯片”，结果 Cortana 显示了包括 OneDrive 云端在内的搜索结果；
* “给我看 12 月份的照片”

微软明白不见得所有用户都熟悉与某个 app 相关的语音指令，因此只要在搜索栏输入该 app 的名称，Cortana 就会自动展示语音指令。

此外，Cortana 的连续指令识别也还不错，Joe 让它播放音乐，然后又说了一句“Cortana，安静一点”，然后音乐播放就自动暂停——Joe 认为，“PC 上拥有 Cortana 的感觉，就好象拥有家人，可以帮你把事情搞定。”

### Windows 10 在手机、平板

Windows 10 相比 Windows Phone 8.1 又有何改进呢？

可以直接更换桌面背景图片，不再是单调的大色块了。

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/EF7E2145-DC92-4CD2-B840-EF7282127E68.png)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/42.png)

通知中心将与 Windows 10 桌面版保持同步。

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/974F44FF-36C3-43AA-8E71-FF4607D06FE4.png)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/%E5%B1%8F%E5%B9%95%E5%BF%AB%E7%85%A7-2015-01-22-%E4%B8%8A%E5%8D%885.06.26.png)

而且，可以直接在消息通知中回复别人。

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/0F1FB2F7-BAD2-41C4-8307-B560F041DB77.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/%E5%B1%8F%E5%B9%95%E5%BF%AB%E7%85%A7-2015-01-22-%E4%B8%8A%E5%8D%885.06.jpeg)

可以移动键盘到屏幕上任何一个位置。

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/8A902DFA-1B9C-4D72-9E75-14ED8CCC706A.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/2015-01-22.jpeg)

Skype 与短信进行整合，使用不同的服务来发送消息，消息的颜色会有不同。

然后重头戏来了，Joe 开始介绍新的手机版 Office！

在 Word 里，Joe 轻触屏幕下方，唤出了 Word Home 标签页，上面集成了所有与 Word 相关的操作。——在接下来的演示当中，这个标签页成为 Office 系列 app 的重要特征。

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/7B5E8101-5575-4E56-89AB-4ACF80861936.png)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/word.png)

更重要的消息是包括 Outlook 在内，Office 套件全系列，以及照片、人脉、音乐等 app，也都将是 PC、平板电脑、手机三平台通用的。

### Project Spartan

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/9B7D6D86-AD1B-4A6D-9C7C-84942573E8C0.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/win10_windows_spartan_Web1.jpg)

Project Spartan，微软下一代浏览器，专注三项功能式。

第一个是笔记模式。允许人们用笔直接在网页手绘标注（键盘也可以），标注可以分享给别人，也可以保存到 OneNote。

第二个是阅读模式。Project Spartan 会将网页转成适合文本阅读的形式，去掉网页上多余的元素，比如说 Flash 广告之类，而且还支持离线阅读、跨平台同步。

第三个是 Cortana。当人们在浏览器中输入网址，比如说餐厅网站时，Cortana 会提示用户它已经找到的相关信息，“我已知道导航、营业时间等信息”。

### Xbox 与 PC 结合

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/51815F35-2D67-4CAA-A0E7-E8E381BDCC04.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/win10_xbox_devices_Web.jpg)

Joe 漫长的演示，也该休息一下了。现在轮到 Xbox 部门主管 Phil Spencer 上台，讲解 Windows 10 如何与 Xbox 联动。

现在在 Windows 10 上，拥有名为 Xbox 的 app，就好象 SmartGlass 的升级版一样，在里面可以找到 Xbox 好友、查看新消息、相关的游戏，还有新成就之类的。

比较有趣的功能是，在游戏当中按下 WIN+G 快捷键，就可以录下 30 秒的视频，然后发到 Xbox LIVE 中去，与好友分享。

Phil 发布了 DirectX 12，性能上有 50% 的提升，而且更加省电。目前 Unity 已经宣布支持 DirectX 12 了。虽说，现场演示了 DirectX 12 的游戏画面，但从视频上感受并不深。

此外 Xbox One 上的游戏，可以串流到任何 Windows PC 或平板上——也就是说，在电脑上也可以玩 Xbox One 游戏。

### 微软新硬件

**Surface Hub**

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/3BC0751E-208B-4040-A2CF-2FCB6F60E740.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/win10_surfaceHub_meeting1_Web.jpg)

Surface Hub 是支持多点触摸，也支持触摸笔，带两个摄像头的 84 寸 4K 屏，它显然是去年收购的 Perceptive Pixel 的继承者。

它面向的显然是企业市场。一部高效的会议设备。它可以用来开 Skype 视频会议，而且可以在 OneNote 打开白板，手绘图形或记录文字。

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/F3A0F2FB-B291-4B32-BD98-54BB025A27ED.png)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/91.png)

**HoloLens**

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/5E90916B-A605-49E3-8852-325D6F565130.jpg)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/holo.jpeg)

“今天，我们将向虚拟的世界迈出新的一步，你的数字世界将与你的现实世界贴的更近。这是一种新的分享灵感的方式，是一种新的游戏方式，你可以用你的方式改变世界，这就是 Holographic！” ”

Windows Holographic 其实是增强现实技术的落地，它需要用 HoloLens 头戴式显示器相配合，——这个头戴式显示器自己就是个全功能的电脑了，里面有 CPU、GPU，空间立体声技术以及全息处理单元。当人们戴上它之后，就会自然而然地进入到一个虚拟和现实相结合的世界里。

[![](.evernote-content/72A488FC-989B-48F6-98CC-D96204F9E0FE/018579A9-16D7-4069-B3B4-78A89D59C58D.png)](http://cdnzz.ifanr.com/wp-content/uploads/2015/01/101.png)

HoloLens 还属于早期发展阶段，微软承诺在 Windows 10 生命周期内推出。不过，当 HoloLens 和配套软件 Holo Studio 相结合，人就可以通过手势、语音的交互，直接在现实空间当中，创造虚拟的物品。

这会是一个非常有意思的产品和技术。

### 总结

最后纳德拉上台发表演讲，他说，“我们希望能让人们从需要 Windows 转变为选择 Windows 并且喜爱 Windows。这是我们 Windows 坚定的目标。”

这次微软发布会惊喜连连的原因，或许就在于此。

关于 Windows 10，PC 版下周可升级，手机版 2 月份可升级。
