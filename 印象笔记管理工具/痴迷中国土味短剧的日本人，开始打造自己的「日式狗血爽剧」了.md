# 痴迷中国土味短剧的日本人，开始打造自己的「日式狗血爽剧」了

---

原创

爆梗研究所

[爆梗研究所](#) 

*2024年10月18日 17:01*
*北京*

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/EA91284F-483D-457F-B6D4-A5F4D9B8D72E)

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/90A8D251-C9FE-4F64-83D0-16E258A02C13)

相信不少刷短视频的小伙伴，总能看到一些国产短剧推荐，不仅竖屏方便观看，而且扑面袭来的奇葩网络爽文剧情十分抓眼球。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/E48E061C-0642-4CA7-B56F-6AE002CB5AFF)

最初大家肯定都感觉剧情土掉渣，演技太浮夸，心说怎么可能会有人看这种视频，结果再一看表发现过去的这一小时，自己已经把整整30集都给刷完了。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/7F19E791-ECC5-4B20-8A41-4CAAEF930F4E)

在这些剧集中，不乏「霸道总裁爱上我」、「妙计巧撕绿茶婊」、「恶婆婆最终被严惩」等土味题材，让人不禁感叹，2024年的今天也迎来了网络爽文的文艺复兴。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/D765B7D4-2B3E-444A-88DE-878C953C6976)

此外，这些剧集的名字相当炸裂，什么《我的爷爷是大佬》、《我不是神仙》、《重生之我在麦当劳修炼魔法》等，不仅十分接地气，而且直接了当，颇有当年《故事会》的风采。

基本上，一集不到1分钟的内容，起码安排了6个反转，这过山车式的刺激谁看谁迷糊啊。

![](data:image/svg+xml,%3csvg data-v-f5e35cd2='' xmlns='http://www.w3.org/2000/svg' viewBox='0 0 2 2' width='4px' height='4px' class='border_filler border_filler_lefttop js-evernote-checked' data-evernote-id='577'%3e%3cpath data-v-f5e35cd2='' d='M1.85.005A2 2 0 000 2V0h2z' fill='%23ffffff' fill-rule='evenodd' class='js-evernote-checked' data-evernote-id='578'%3e%3c/path%3e%3c/svg%3e)

大家都表示，这些土味短剧一刷简直停不下来啊！我是土狗我爱看！

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/BBFFBF76-B8BE-4780-9669-9D7C9CFACF00)

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/860A1E6A-9440-49CF-8B52-552C11BDA2D6)

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/18F37CEC-3734-4815-9F5D-442CB5FF4010)

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/54691CD3-73B0-484A-BBD0-C3B76B508560)

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/31E550C3-FE64-4CB1-838B-1555A3148C81)

都说文化是共同的，中国土味短剧也直接走出国门，在海外tiktok平台迎来了一波文化输出。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/05FB679A-AB8E-49E3-AD26-EA7D585843B7)

甚至有的短剧片段（仅仅是片段）在海外播放量已超百万，着实恐怖。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/D0854F37-5B6F-4531-AA7B-D25C72D3E005)

看来，老外们也是看腻了好莱坞大片，遇到咱们的土味爽剧也是爱的欲罢不能，纷纷表示赶紧更新最新一集。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/536174BE-A67C-45CD-ABD2-29D9A921442F)

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/0EF05B4D-C55E-40A1-864B-43823FD3FE6C)

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/AB331175-3B21-4DA7-91A3-E845209DD602)

而万万没想到，中国土味短剧最受欢迎的海外国家，居然是日本。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/923F0433-AF37-4431-B592-96D751121DEC)

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/42E11911-20A0-4EB1-A7D5-8DACDCF6EADF)

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/43FB94EC-33CE-42D4-96F1-95B14035D13B)

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/1494F14D-145D-4394-BF28-62F31F9F51B4)

中国土味短剧在日本不仅受众广泛，而且面向海外推出的国产APP「TopShort」在日本 iOS 应用销量排行榜上曾暂时超越 Netflix，累计下载量超过了50万次。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/C09204A4-9D0C-4E0C-9BCE-C9620F898282)

逐渐的，日本人也开始手心做痒，打算做出符合本土口味短剧系列。从职场、家庭各种方面面面俱到。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/90A8D251-C9FE-4F64-83D0-16E258A02C13)

比如有日本人喜欢拍的复仇题材——

![](data:image/svg+xml,%3csvg data-v-f5e35cd2='' xmlns='http://www.w3.org/2000/svg' viewBox='0 0 2 2' width='4px' height='4px' class='border_filler border_filler_lefttop js-evernote-checked' data-evernote-id='875'%3e%3cpath data-v-f5e35cd2='' d='M1.85.005A2 2 0 000 2V0h2z' fill='%23ffffff' fill-rule='evenodd' class='js-evernote-checked' data-evernote-id='876'%3e%3c/path%3e%3c/svg%3e)

还有喜闻乐见的兄弟情深——

![](data:image/svg+xml,%3csvg data-v-f5e35cd2='' xmlns='http://www.w3.org/2000/svg' viewBox='0 0 2 2' width='4px' height='4px' class='border_filler border_filler_lefttop js-evernote-checked' data-evernote-id='1075'%3e%3cpath data-v-f5e35cd2='' d='M1.85.005A2 2 0 000 2V0h2z' fill='%23ffffff' fill-rule='evenodd' class='js-evernote-checked' data-evernote-id='1076'%3e%3c/path%3e%3c/svg%3e)

当然，最普遍的爱情戏也是大多数日本观众的最爱之一——

![](data:image/svg+xml,%3csvg data-v-f5e35cd2='' xmlns='http://www.w3.org/2000/svg' viewBox='0 0 2 2' width='4px' height='4px' class='border_filler border_filler_lefttop js-evernote-checked' data-evernote-id='1274'%3e%3cpath data-v-f5e35cd2='' d='M1.85.005A2 2 0 000 2V0h2z' fill='%23ffffff' fill-rule='evenodd' class='js-evernote-checked' data-evernote-id='1275'%3e%3c/path%3e%3c/svg%3e)

但是，相比于中国短剧，日本人在拍摄的过程中也发现了自己讲述故事的通病——墨迹。要知道，短剧这种快节奏，多反转的特殊内容，一旦走慢节奏，就很容易芭比Q啊。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/F0E04EAC-3BA2-494B-B599-F6896E9BD1D6)

**于是有聪明的日本厂商，想到了最好的解决方法——向中国取经学习！**

就比如日本GOKKO公司就联系到了上海嘉书科技联合打造了一系列日式短剧，凭借其接地气的题材和多反转的剧情套路，博得日本观众喝彩的同时，流量也超级爆炸。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/69479AC9-3D4F-463C-977B-376433CD2424)

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/1BC816F6-71E7-4EDE-A2C5-DBFF6659A90C)

**好家伙，这波中日合拍真的让文体两开花了。**

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/389BB46C-DF93-4708-8E19-C14CE54CD03E)

双方合作主要是嘉书科技提供原作，而日本GOKKO公司则负责拍摄、剪辑和制作等事宜。可谓是各自运用自己的强项输出了。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/9F8D5C87-4312-41AA-8BA5-95D6A88862F8)

他们联合制作的一系列日式狗血短剧在日本市场大受欢迎，其中更是将中国人喜欢看的「霸道总裁」换成了日本人更容易接受的「富家千金」，打造了《大小姐的权利骚扰退治》等热门短剧，一度成为顶流。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/8BDAD881-E008-4315-8E8B-A191B4B18B1E)

《大小姐的权利骚扰退治》单集仅1分钟，月度会员就要约7600日元（约合361元人民币），但是为了追完全部50多集，很多日本观众还是选择了爽快下单。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/DB9F6A33-1B10-4562-91B0-FF6DCF87C608)

不是哥们，你们这米子挣得可真过瘾啊。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/15569756-F7A4-403B-8F76-F64459CFEB01)

而更有趣的是，日本一些大火的短剧厂牌，其大热作品得到瞩目之后，还推广回了中国获得了高人气。

其中代表的就是在日本播放超过11亿的CONECO FILM，他们的搞笑剧集「便利店」系列上线小红书后大火，相信刷过的小伙伴不在少数吧。

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/C004906B-2EDA-4C58-AC46-734AD7D1CA72)

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/DD9F441F-A4E4-4921-8AEA-07BD77175212)

CONECO FILM聪明的将本应连续反转的长剧情，变成了每段看似为独立故事，这样即使在流媒体平台单独刷到，也会被单集剧情吸引，不会因没看过前面剧情而弃坑，更加便于互联网传播。

![](data:image/svg+xml,%3csvg data-v-f5e35cd2='' xmlns='http://www.w3.org/2000/svg' viewBox='0 0 2 2' width='4px' height='4px' class='border_filler border_filler_lefttop js-evernote-checked' data-evernote-id='1564'%3e%3cpath data-v-f5e35cd2='' d='M1.85.005A2 2 0 000 2V0h2z' fill='%23ffffff' fill-rule='evenodd' class='js-evernote-checked' data-evernote-id='1565'%3e%3c/path%3e%3c/svg%3e)

目前短剧风潮仍在席卷日本，在这个快节奏的当下，越来越多的人因为工作家庭的事宜无法好好坐下来欣赏一部2小时的长电影，反而一集不足1分钟的短剧填补了所有碎片时间，无脑的剧情也不会让人思考逻辑，减压效果着实顶满。**总而言之，爽就完事儿了。**

**毕竟，谁不想和短剧里的人生一样，虽为职场牛马，却身家百亿，随时即可手撕垃圾人。**

![](.evernote-content/337E1949-02FF-484C-9F7B-6FC1EDB56C17/BF951A92-6FAC-41A0-AE8D-6BB00C464FC6)
