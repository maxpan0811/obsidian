1. 为什么Web 3很重要

2021/9/26发布

Web 1：（大约1990-2005年）是关于去中心化和社区治理的开放协议。大部分价值都来自于网络边缘——用户和创建者。

Web 2：（大约2005-2020年）是关于大公司运营提供的储存式、中心化服务。大部分价值来自于谷歌、苹果、亚马逊和Facebook等一批公司。

Web 3：是创建者和用户所拥有的互联网，由代币编排而成（感谢@packyM的这个定义）。我们现在正处于Web 3时代的开始，该时代结合了Web 1的去中心化、社区治理精神和Web 2的高级现代功能。

![](.evernote-content/09A460E5-2113-415F-B394-E493D1BF9E66/65C9F8E4-013B-4797-BBD8-B13B3949FCBE.jpg)

为什么Web 3很重要？

首先，让我们看看中心化平台的问题（早在2018年我就在博客里写了更多关于这个话题的文章：为什么去中心化很重要）。中心化平台遵循的是一种可预测的生命周期规律。起初，他们竭尽所能的招募用户和第三方补充，如创作者、开发人员和企业。

他们这样做是为了加强他们的网络效应。随着平台的采用率S曲线不断上移，他们对于用户和第三方的掌控力稳步增长。

![](.evernote-content/09A460E5-2113-415F-B394-E493D1BF9E66/E1252164-8AF5-4DBF-BB73-93CCE22760EF.png)

当他们达到S曲线的顶部时，他们与网络参与者的关系从正和变为零和。要继续增长，需要从用户中提取数据并与（前）合作伙伴竞争。

这方面的著名例子是微软 vs. Netscape，谷歌vs. Yelp，Facebook vs. Zynga，Twitter vs.第三方客户，以及Epic vs. Apple。

对于第三方来说，从合作到竞争的转变，感觉就像是诱饵骗你上钩。随着时间的推移，最好的企业家、开发者和投资者已经学会了不要在中心化平台上构建。这扼杀了创新。

现在我们来谈谈 Web 3。

在Web 3中，所有权和控制权是去中心化的。用户和创建者可以通过拥有非同质化代币（NFTs）和同质化代币的形式来拥有互联网服务。代币赋予用户财产权：即拥有互联网的一部分的能力。

NFT使用户能够拥有对象，这些对象可以是艺术品、照片、代码、音乐、文本、游戏对象、凭据、治理权限、访问通行证，以及人们接下来梦寐以求的任何东西。

NFT存在于像以太坊之类的区块链之上。以太坊是一种去中心化的全球计算机网络，由其用户拥有和运营。

区块链是任何人都可以访问但无人拥有的特殊计算机。

以太坊由同质化代币ETH驱动，用于激励系统背后的实体计算机。ETH也是系统交易的本地货币，如NFT购买。

用户有很多方法可以获得同质化和非同质化代币。你可以买它们，但也有办法赚到它们。

Uniswap追溯性地将其15%的治理代币空投给协议的早期用户。像这样的社区赠款在Web 3中变得很常见，作为建立善意和激励采用率的一种方式。

您还可以通过创意和创业活动赚取代币。例如，人们每天出售NFT赚取价值约1亿美元的ETH。

代币使网络参与者保持一致，共同努力实现一个共同的目标——网络的增长和代币的升值。

这解决了中心化网络的核心问题，即价值由一家公司积累，公司最终将与自己的用户和合作伙伴作斗争。

在Web 3之前，用户和创建者必须在Web 1的有限功能或Web 2的大公司中心化模型之间进行选择。

而Web 3提供了一种结合前两个时代最佳优点的新方式。目前还是这项运动的早期阶段，但也是参与其中的好时机。

2. 软件的可组合性就像金融的复利

2021/10/22发布

世界上非常高比例的软件是开源的。大多数数据中心软件都是开源的，手机操作系统大量使用开源，大部分的下一代设备，如自动驾驶汽车、无人机和嵌入式计算机，也多是运行开源软件。

开源始于80年代的边缘ZZ运动，到90年代也不太常见，但在2000年代及以后开始成倍增长。

为什么呢？因为开源解锁了软件可组合性的力量。

Web 3最令人兴奋的方面之一是协议和代币的可组合性。我们在DeFi中看到这个现象，像Yearn或Element所做的这样，协议建立在互相之上。

人们开始使用NFT探索可组合性，比如LOOT及其周围的生态系统就是一个非常有趣的例子。

我们需要寻找这个世界上各种指数级力量，因为它们是未来快速增长的标志信号。

在硬件层面，最强大的指数级力量是摩尔定律。在金融领域，它是复利。而在软件世界中，它是可组合性。

3. 区块链是新的应用商店

2021/8/16发布

如果你是2009-2012年左右、移动黄金时代的雄心勃勃、寻求风险的创始人，你肯定也开发了一款新的移动App。那正是Uber、WhatsApp、Instagram、Venmo、Snapchat和许多其他顶级App出现的时点。

此后，移动互联网的技术采用率S-曲线已经上移。可能还会有开发伟大移动App机会，但低垂的果实已经被摘完。这个十年，计算的前沿将是在以太坊这类可编程区块链上开发应用程序。

区块链是在物理计算机网络之上运行的虚拟计算机。它们具有以前计算机类型所没有的新属性和新功能。

我们认为，当区块链具有高度表现力、接近图灵完整编程语言时，它们是可编程的。最受欢迎的可编程区块链是以太坊。

可编程区块链很有意思，就像当年移动手机一样，通过应用商店向第三方开放时变得非常有意思。最好的想法往往来自边缘。

去年（2020）在“DeFi夏季”期间，随着第一波加密/区块链杀手级应用的爆发：包括Uniswap、Aave、Compound、Maker等，我们看到这种情况已开始出现。

DeFi表明，构建具有包容性、公平性、透明性和可组合性的金融服务是可能的。在华尔街，价值向内流往中心化机构。而在DeFi中，价值向外流往边缘的人。

> “虽然大多数技术倾向于实现边缘工人执行枯燥任务的自动化，但区块链则让摆脱中心自动化。区块链没有让出租车司机失业，而是让Uber优步失业，让出租车司机直接与客户合作。”
>
> Vitalik Buterin（以太坊创始人）

今年（2021）NFT和游戏会是下一波爆发的区块链应用：包括NBA Top Shot、CryptoPunks、Axie Infinity等。这张图已反映一切，你可以看到最大的NFT交易平台OpenSea上销售额每天都在快速增长：

![](.evernote-content/09A460E5-2113-415F-B394-E493D1BF9E66/A0094EB7-7006-45D6-B061-30B9DB3F0382.png)

对有些人来说，NFT似乎略显轻浮，像是玩具。但NFT确实很重要，因为它为创作者和开发者提供了比现有Web 2平台好太多的经济模型。

下一波区块链杀手级应用将是什么？计算的新浪潮很难预测，但总有一些有意思的新类别出现。

DAO（去中心化自治组织）是一种互联网原生的，共享资源、共建产品、并朝共同目标共同努力的全球性集体。

社交代币是创作者在线赚钱的新方式。他们消除了寻租中介，使得小型经济模型也能够与粉丝基数增长一起发展。

基于区块链的社交网络对用户和开发者都做出了坚定的承诺，因此他们可以在坚实的基础上不断发展，而不必担心API和经济模型的变化。

支持加密的3D世界和元宇宙体验让人们可以在其中社交、赚钱和构建迷你世界，同时保证互操作性和真正的所有权。

这只是一些想法，创业者比起像我这种做预测的人肯定要更擅长于创建未来。最好的想法很可能是，要么还没有被构思出来，要么今天看上去会很奇怪。

如果你在加密领域工作，你估计习惯了局外人把你当成滑稽戏，或者认为你在做的事情是愚蠢或骗局。这些人正努力打造的事物，我们甚至都还没有名字。这一切太新了。

这就是S曲线上一开始时的工作感觉。可编程区块链是计算前沿，就像80年代的PC、90年代的互联网以及过去十年的移动互联网一样。

我们今天回顾计算机时代中的经典时刻，很好奇想知道在那里会是什么感觉。

![](.evernote-content/09A460E5-2113-415F-B394-E493D1BF9E66/7A6E04CE-EBC6-42C8-819B-3359D21F3D0B.png)

但实际上你已经在那了！2021年版的“家庭自制电脑俱乐部”（意指上图中苹果两位创始人在家庭车库的起步阶段）就是DAO或Discord服务器，模式是一样的。热衷者分享想法。而工匠们在晚上和周末努力动手。这些是美好的过去时光——身处最前沿的生活。

4. 代币是新的数字基元

2021/9/21发布

主要的计算浪潮有两个时代：仿真时代（skeuomorphic era）和原生时代（native era）。

在仿真时代，设计思维在很大程度上是来自过往领域。早期的网络主要是对互联网之前时代活动的数字化改造，比如像写信和邮购等。那时的网站大多是只读的。

技术人员花了大约十年的时间才开始认真探索网站可读/可写的想法，用户在上面可以生成内容。而这带来了社交网络、众筹和社交生产力应用等网络原生类别的增长。

这个模式正在加密/Web 3上重演。已经有一些很棒的原生Web 3产品，但总的来说，我们仍处于仿真时代，很多Web 3产品都是从过往领域改造而来。

一些流行的仿真型Web 3想法包括离线票务、供应链管理和离线资产记录保存等。这些可能是好主意，就像只读网站曾经是个好主意一样，但它们只触及了Web 3的表面。

当今许多NFT都是来自线下世界的艺术和收藏品的改造产品。这让人们以为NFT仅限于这些领域，就像人们曾经认为网络仅限于小册子和杂志一样。

代币——同质化和非同质化（NFT）——更被认为是新的数字基元（digital primitives），在灵活性和通用性上与网站等过去的数字基元相似。

代币赋予用户财产权：即拥有互联网的一部分的能力。

Web 2 遗漏了数字产权。当你使用网站（或App）时，它只允许你借用或租赁东西。想象一下，在现实世界中，这就像你每次去新地方时，都必须从头开始购买所有东西。这就是Web 2。

与网站一样，代币是数字基元，可以概括为代表几乎任何东西——金钱、艺术、照片、音乐、文本、代码、游戏项目、控制、访问权限以及人们未来梦想中的任何东西。

用户现在可以在钱包中持续拥有从一个App转到另一个App的物品库存。如果他们的物品价值提升，用户会获得收益。与Web 2相比，这是一个很大的变化，Web 2的收益主要都被科技大公司拿走了。

我们目前仍处于Web 3的仿真时代，但逐渐开始看到一波新的原生应用出现，这些原生应用没有先前的类似物，过往也不存在。

例如，在DeFi企业家开创的机制设计基础上，新一波的DAO正在探索汇集资源、共建物体和自治的新方式。

像LOOT这样的可组合NFT游戏激励了社区围绕一组NFT来构建整个世界——如果没有Web 3赋予的所有权和可移植性，这项活动是不可能的。

同质化代币不与金钱和金融关联的话并没有内在价值（intrinsic），NFT不与艺术品和藏品关联的话也没有内在价值。

这些是伟大的初步应用，可能仍然非常重要，但代币更被认为是一个类似于网站的新的数字基元——围绕其构建新一代互联网的原子单元。

5. 从Web 2到Web 3，“你的take rate就是我的机会”

2021/9/22发布

杰夫·贝佐斯（Jeff Bezos）有句名言：“你的利润就是我的机会”，指的是亚马逊通过降低价格和吞噬竞争对手利润率来获得市场份额的方式。

亚马逊在商业中所做的是互联网更普遍的行为。自90年代以来，降低价格和将价值重新分配给用户一直是互联网的核心经济动态。

Craigslist用分类广告做到了这一点，谷歌和Facebook用媒体做到了这一点，TripAdvisor和Airbnb用旅行做到了这一点，等等。

今天，随着Web 3创业公司开始吞噬Web 2存量玩家的利润，这一趋势仍在继续。Take rate（抽佣率）越高，存量玩家就越脆弱。

电子游戏行业每年的销售额约为1200亿美元，其中很大一部分是虚拟商品。大多数电子游戏的take rate是100%。

Web 3（aka加密）游戏大幅降低了take rate。例如，Axie Infinity过去一年产生了超过10亿美元的总销售额，其中的绝大部分已经返还用户身上。（仅4.25%税率）

![](.evernote-content/09A460E5-2113-415F-B394-E493D1BF9E66/F9889265-424B-46FA-B490-2493881B02DC.png)

在大多数电子游戏中，有些人花钱才能升级，而另一些人则努力练级。Web 3的区别在于，其经济模型是点对点的（peer to peer）：玩家为玩家提供资金，而不仅是游戏开发者。

今天，流媒体服务上有超过800万音乐家，但每年只有不到15,000名音乐家（不到0.2%）能赚到5万美元以上。这是因为大多数收入都被流媒体服务和音乐唱片公司拿走。

而使用NFT，音乐家可以留下销售额中超过90%部分。通过去除中间商，音乐家们只需要1000个真正粉丝就可以有效的支撑自己。

Twitter、Instagram和TikTok等社交媒体平台的take rate为100%——他们根本不与创作者分享任何收入！这对他们来说很棒，但对用户来说却不好。

相比之下，Web 3社交平台像Rally、Mirror和BitClout等的take rate远低于10%。大部分价值都返还给了用户和创作者。

Web 2平台完全依赖于创作者的内容，但只提供些微的回报。这是不可持续的。Web 2的take rate就是Web 3的机会。

6. Web 3实战手册：用代币激励启动新网络

2021/10/1发布

互联网的杀手级应用是网络。网页和电子邮件是网络。Instagram和Twitter等社交应用是网络。Uber和Airbnb等交易平台是网络。

网络的参与者越多，价值就变得越大，这在一定规模时是很棒的，但在开始时却是另一种情况。这是个自启动问题。

在Web 2时代，克服自启动问题意味着英勇的创业努力，而且在许多情况下，还有在市场营销上花费的大量资金。

Web 3引入了一个强大的自启动网络新工具：代币激励。

其基本逻辑是：在启动早期阶段，当网络效应尚未见效时，给用户提供代币奖励作为金融基效用，以弥补其初期匮乏。

![](.evernote-content/09A460E5-2113-415F-B394-E493D1BF9E66/28EDC8E1-3774-4A59-99B9-C10250087550.png)

（上图来自我在2017年写的一篇更长博客文章）

随着时间的推移，伴随网络效应和原生效用的不断增长，代币激励逐渐减少，最终为零，世界只剩下一个新的规模化网络。

关于如何设计代币时间表和阻止作弊者和诈骗者，有很多错综复杂之处，我不会在这里深入讨论，但这是一个非常有趣的话题。

我们来看看一些案例。比如，Helium正试图通过激励个人在家中安装网络设备的方式，给大型电信公司创造草根竞争对手。（顺便说一句，这种用自下而上的草根电信服务提供者，来替代地位牢固的存量巨头的想法是一个永恒的科技幻想，之前已经有不少值得称道的尝试来努力实现，包括早期的Fon和早期的Meraki/Roofnet。而Web 3可能会最终将它实现。）

核心问题是，你如何安装足够多的设备以获得广泛的覆盖范围（供给方），同时也建立起需求方。这是一个经典的“先有鸡还是先有蛋”问题。

Helium借助代币激励的力量来启动供给方。到目前为止，它工作得很好。该网络在全球拥有超过20万个节点。这是一张美国覆盖地图，另一张是湾区的放大地图。

![](.evernote-content/09A460E5-2113-415F-B394-E493D1BF9E66/8F5D336A-FAD8-47E4-99EC-F08905DE52F1.png)

![](.evernote-content/09A460E5-2113-415F-B394-E493D1BF9E66/F30FD560-343A-4140-B09D-E761196DD612.png)

另一个借助代币激励实现启动的网络是Arweave，一个去中心化的存储网络。去年，Arweave网络增长了约12倍。这是过去两年的增长。（注：a16z是Helium和Arweave的投资人）

![](.evernote-content/09A460E5-2113-415F-B394-E493D1BF9E66/72634A91-C521-43BD-BB5D-DEE69ACF43EA.png)

代币激励是有效的，也远比中心化的Web 2模型公平。帮助建立网络的人不应该拥有其中有意义的一部分吗？

自2013年以来，我一直在Web 3和加密领域工作，从未参与过在市场营销上花钱的项目。当用户是真正的所有者、喜欢他们做的事情，喜欢告诉别人这件事时，你不需要花钱在营销上。

Web 3允许创建网络的用户和构建者拥有其中有意义的一部分，同时为企业家解锁强大的新工具。

==增补==

许多Web 3评论家错过了一个要点：代币激励是暂时的，用来渡过创建新网络时最难的阶段：即度过“自启动”阶段。

许多网络的终局将是：一个网络提供任何其他网络的效用（比如Twitter、FB、Lyft、Airbnb）。

Helium是一个很好的案例，因为企业家试图建立“自下而上/草根电信服务”已有20年的历史（可能更多），比如，Meraki团队最初是如何开始的。

还有许多其他尝试。这是雄心勃勃的创始人一直被吸引的一个大想法。但在过去，启动阶段太难克服了。

像Verizon这样的存量巨头是用资本来解决这个问题——他们在需求出现前投资了数百亿美元来构建网络，再转向需求方。

结果是，我们只剩下少数存量巨头，不存在新进入者挑战他们的可能性。

而正如Helium所展示的，Web 3提供了一套强大的新工具克服了启动阶段问题，并为初创企业创造了公平的竞争环境。

7. 七种类型的NFT

2021/12/11发布

NFT是新的数字基元，在灵活性和通用性上与网站等过去的数字基元相似。

NFT是一套简单的区块链标准，以可互操作、可重新混合和安全的方式实现真正的所有权。

第一次以来，用户现在可以真正地拥有数字物品。

我们开始看到不同种类的NFT出现。以下是让我感到兴奋的7种NFT类型：

■ 艺术品

这些可以是朋克（Punks）和无聊猿（Bored Apes）等收藏品的一部分，也可以是你在Foundation和SuperRare等网站上找到的1比1艺术品。

人们喜欢收藏数字艺术品，原因与人们喜欢实体艺术品、时尚、棒球卡等是一样的——一种美学、赞助、地位、收藏和社交的混合体。

■ 音乐

音乐对NFT来说是一个特别重要的领域，因为web2给予音乐家的报酬很少。

最近一系列新的音乐NFT项目上线了，包括http://arpeggi.io、http://Sound.xyz和http://royal.io。

这是使用Arpeggi作曲工具创建的上链歌曲NFT。http://Sound.xyz正在做NFT歌曲下载，在上面NFT是收藏品，也可以让你加入一小部分超级粉丝行列。

![](.evernote-content/09A460E5-2113-415F-B394-E493D1BF9E66/B97D5925-750D-4338-82B7-32CDDCB1274E.png)

■ 访问权

NFT使门票具有低延迟和广泛的互操作性。

一个更原生的用例是数字访问权，像是一个私人Discord服务器或者视频课程。

■ 游戏物品

Web 2游戏玩家每年在虚拟商品上花费约400亿美元。但用户并不真正拥有这些物品，是公司拥有。物品也无法在游戏之间互操作和组合。

正如我们在技术史上一次又一次地看到的那样，互操作性和可组合性极大地加速了创新。

NFT允许用户真正拥有物品，翻转了两端：游戏围绕用户拥有的物品来构建，而不是用户依赖于游戏。

Axie Infinity和Top Shot等开创性游戏开启了新一轮Web 3游戏开发浪潮。其中许多项目会在明年发布。

■ 可赎回品

Unisocks是该模型异想天开的先驱者，在这个模型中，代币可以赎回实物。这种机制可以应用于许多离线商品。

例如，许多高端实物商品收藏者将他们的藏品保存在金库中。你可以创建一个代表藏品的数字代币，让它们以数字方式展示和交易，减少摩擦和去除中间商，并扩大用户体验的可能性。

■ 身份

Web 2处理个人信息的方式可以说是失败了。你的敏感个人信息已被多次黑客入侵。处理密码更是一团糟。冗长的隐私政策和服务条款时不时就会改变。

与你读到新闻时的想法可能正好相反，像比特币和以太坊这类的区块链从未被黑客入侵过。

在Web 3中，用户做主：没有中心化的数据库可供黑客入侵或出售。

今天，这意味着使用ENS这类等系统，以可互操作的方式登录（http://login.xyz）和识别自己是种更好方式。

以后，你估计会使用NFT存储并有选择地披露身份的其他方面，来处理例如记录、凭据和声誉等事情。

■ Web 2数据库

使用加密方法和去中心化的数据存储，你可以将NFT扩展到今天存储在中心化Web 2数据库中的任何东西之上。

这可以是些简单的东西，比如你的观看偏好，或者更高级的东西，比如你的整个社交图表。

这将允许你无缝地从一个服务切换到另一个服务——并完全控制自己的数据，这是Web 3愿景的关键部分。

今年，我们看到了围绕NFT的创新爆发式增长。这可能会持续很多年，因为当下还处于Web 3进化的非常早期阶段。

8. 互联网鄙视糟糕的商业模式并绕道而行

2021/9/26发布

让我们从这张迷人的图表（来自https://matthewball.vc）开始，它提出了一个问题：为什么电子游戏行业与新技术一起得到了成长，音乐行业却没有？

![](.evernote-content/09A460E5-2113-415F-B394-E493D1BF9E66/DFCCBF42-DC0B-4FDF-A779-7C6D5F8908C3.png)

长期以来，电子游戏和音乐有着相同、直截了当的商业模式：对基本内容（游戏或音乐本身）的永久许可收取费用。

当互联网出现时，电子游戏行业的响应方式是拥抱MMO（大型多人在线）和Battle Royale（大逃杀）等新游戏类型、拥抱免费玩游和虚拟商品等新商业模式，还有流媒体等新的互补行为。

而音乐行业的响应方式却是耗费几十年时间来起诉创新者，并最终接受了允许用户通过流媒体平台订阅捆绑内容的增量改进。

与此同时，电子游戏行业还发现了一些更夸张的东西：如果他们把之前收费的东西——游戏本身，该行业的唯一收入来源——完全免费，他们甚至可以赚得更多。

游戏制造商将他们的产品视为一个包含软件、服务和附加组件的捆绑包，然后进行测试，以找到最大化病毒营销效果的免费功能和最大化收入的付费功能之间的最佳平衡。

今天，最复杂的游戏——《堡垒之夜》、《英雄联盟》、《皇室战争》——都是赠送游戏本身而收取虚拟商品费用。

想想看：大约十年前，虚拟商品几乎不存在。而现在，它们是增长最快的媒体类别中最成功产品的主要收入来源。

现在进入主要话题。互联网先驱约翰·吉尔摩有句名言，互联网将审查视为一种缺陷，并绕道而行。互联网对糟糕的商业模式也做了同样的事情。

一个经典的例子是90/00年代的开源vs专有软件。尽管强烈反对，但专有软件供应商仍然坚持使用过时的封闭式源代码模型。而草根开源社区则使用全新的软件堆栈绕道而行。

NFT是互联网绕开糟糕商业模式最近的一个重要例子。与开源一样，NFT是一个草根运动，由创作者领导，将电子游戏行业的虚拟商品模型扩展到自己领域。

即使出版商和工作室与Web 2的存量玩家们一起抵制新模式，个人创作者也可以绕过他们，直接触达他们的粉丝。

音乐家现在可以选择赠送新音乐，以实现最大程度分发，并以稀缺的互补NFT方式收费，比如数字专辑艺术品、幕后通行证和粉丝身份证明等。音乐家3LAU最近以NFT下载方式赚了1160万美元。https://nft.3lau.com

电影制片人或作家可以出售NFT来众筹他们的作品，而不是以传统渠道融资。最近，一位小说家以这种方式筹集了5万美元https://emily.mirror.xyz，一部纪录片也筹集了200万美元https://ethereumfilm.mirror.xyz

电子游戏并没有什么神奇之处，让它们比其他形式的媒体更容易货币化。人们喜欢电子游戏，但人们也喜欢音乐、书籍、电影、播客和数字艺术品。

创作者现在可以选择绕开一直阻碍他们的机构，像电子游戏行业一样，拥抱技术的前进发展，而不是像音乐行业那样，与之抗争。

9. 为什么要在Web 3创业

2022/3/6发布

我最近有机会见了一些不错的Web 2创始人都在考虑进入Web 3。

Web 3现在很火热，因此去除噪音，找出正确的Web3创业长期原因很重要。

以下是创立Web 3项目的一些好理由：

■ 产品

维基百科只是比传统百科全书更好的百科全书，因为它利用了社区的知识和创造力。

而DAO是新的Web 3结构，通过提供更复杂的协调方法以及对互联网原生金融资源的控制，来为线上社区助力。

在过去一年里，数千个DAO被创建出来，涵盖了金融服务、投资、内容创作、游戏、社交俱乐部等众多领域。

也就是说，我们仍处在探索DAO设计空间的早期阶段。

未来可能有机会将DAO应用于全新领域并创造出突破性产品。

■ 经济模型

Web 3通过去除中介层并大幅降低take rate（中介抽佣率）来改变互联网服务的经济模型。

这为初创企业提供了一个机会来吸引和培养社区，通过为创作者、开发者和其他网络参与者提供优秀经济模型的方式。

例如，Instagram和Twitter等社交网络不会与让他们广为流行的创作者分享任何收入。

而在Web 3中，创作者可以通过销售NFT等方式直接触达自己用户，能够赚到他们自己所创造的收入中90%部分甚至更多。

Web 2社交网络是通过说服人们免费提供他们的创意作品而构建形成的。

一旦创作者意识到有可信的替代方案，这一切都将不再可行。

目前NFT总体销售规模平均每月为数十亿美元。

按照现在趋势，Web 3创建者所产生的收入将很快超过Web 2。

■ 市场营销

Web 3解锁了强大的新营销技术。

首先，Web 3使项目能够在原生网络效应较弱的早期阶段为用户提供代币激励。

这有助于早期初创企业克服过去很多有前途的先行者遇到的类似自启动问题。

这方面一个很好的例子是Helium，它利用代币激励措施建立了一个全国性的草根电信服务平台，来挑战电信行业存量巨头。

其次，在Web 3中，代币自身就是一种营销。例如在游戏中，当某人真正拥有某样道具或皮肤时，他们会替你广为传播。

Web 3允许你借助用户的热情来构建服务，而不是建立在现有巨头平台上，或者使用付费广告方式来推广。

这将显著降低你的获客成本，并让你能打造出更高质量的社区。

■ 生态系统

世界上运行的大多数软件都是开源的，它们是由开发者社区创建。

这是因为拥有相关专业知识的聪明人无处不在。

正如比尔·乔伊（Bill Joy）的名言：“无论你是谁，大多数最聪明的人都会为别人工作。”

而Web 3项目可以通过创建和培养开发者生态系统来深挖更大的人才库。

开发者通常需要确认他们是在一个可靠、中立的平台上来做开发，其平台规则不会改变。

Web1时代，类似HTTP和SMTP的协议都做得很好，这带来了创新的大爆发。

Web2时代，试图通过“不做恶”的诺言来向开发者做出承诺。但这并不奏效，就像我们这些年所看到的。Web 2平台经常弃用API接口并更改服务条款，而很少考虑第三方开发者。

Web 3时代，对开发者的承诺是以不变的开放源代码融入到系统中（“做不了恶”）。

这意味着开发者可以信赖它的可靠度和中立性，就像在Web1时代那样。

这将带来新一轮的广泛、无需许可的大创新。

■ ethOS \*

注：ethOS（以太坊操作系统）是一个由社区开发的，立足于以太坊的，免费、开源的移动操作系统，旨在降低开发者实现Web 3的门槛。

最后，也是最重要的，是web3的ethOS。

互联网越来越被少数存量科技巨头主导。

Web 3将我们带回互联网最初的愿景：一个由开源协议治理的去中心化网络，在其上权力和金钱都将流向边缘，而非集中在中间商身上。

10. Web 3将网络转化为经济体系

2021/8/22发布

过去20年我们都在互联网上建立网络。社交媒体平台，像Instagram、Twitter、YouTube和Discord等，都可以被分解成数十亿个更小的网络，由粉丝、好友、订阅用户、支持者等所构成。

这些平台给许多人提供了以前所没有的受众。但由于网络和其所有者的大公司之间底层结构性的错位，网络规则和经济体系之间的关系日益紧张。

例如，控制大型网络的社交媒体公司通常会扼杀有前途的第三方开发者，打击跨网络互操作性，收取过高的抽佣率，并采用侵入性的广告模式。

这并非是因为他们人很坏或者动机不良，而是经济体系的逻辑所致。如果你不这么做，你的竞争对手是会这么做的，然后你就只能关门了。

我们正在进入互联网的新时代——Web 3时代——我们有机会将这些网络升级为经济体系，并在此过程中构建系统，使网络所有者、参与者和第三方开发者的利益完全一致。

经济体系（如此处所用）是指该网络中各种加密资产可以从任一节点自由流动到任何其他节点，该过程由去中心化的参与者所引导，而非中心化的网络所有者。

为什么是Crypto加密？理论上讲，你可以拥有没有加密的数字经济体系，但加密保证了商品（NFT）和货币（token）的真正所有权，从而降低了转换成本，并阻止网络还操弄Web 2时代的那些把戏，比如不时弃用API接口和提高抽佣率。

当Web 2网络允许交易时，它对集中定价、产品和买家-卖家关系都有不少限制。由于锁住了用户，Web 2的抽佣率是相当高的。

真正的数字经济体系就像现实世界的集市。任何人都可以创造新的商品和服务，并且可以与其他人进行自由交易。人们可以充分表达他们的创造力，并享受他们所创造产品的经济收益。

在现有网络之上建立的新经济体系会是什么样子？我们可以通过关注新兴的NFT社区来一睹未来情形。例如，CryptoPunks的经济体系就在Discord、Twitter、Telegram、OpenSea、Larva Labs等平台之间流动。

在这些社区中，CryptoPunks不仅仅是“jpegs”图片。它们是文化产品，通过用户身边社区所共有的神话、网络梗、故事和经历而获得价值。Punk 6529在这里做了很好的解释：

比如说，现在把它应用到一支花了多年时间来吸引粉丝但依然只能从流媒体平台上获得微薄收入的乐队身上。通过使用代币和NFT，这支乐队可以将他们打造的网络升级为有利可图的经济体系。

在这支乐队经济体系中流动的商品可以是社交代币、数字艺术品、数字藏品、门票、游戏道具、独家体验，或者任何其他创作者和技术专家所梦想的东西（我们还为时尚早——将来会有更多的好主意出现）。

这支乐队从首次发行和转售分成中获得收入，可能是通过拥有大量同质化代币的形式来获得，同时也被激励与社区一起把经济体系做得尽可能大。如果你有个好主意，他们会欢迎你在上面添砖加瓦。

经济体系将具有可组合性，意味着新的经济体系可以建立在现有经济体系之上。系统和服务变得像搭乐高积木一样，加速了所有人的发展。就像另一个经济体做了什么？不要与它抗争，与它一起建立你自己的经济体系。

那些自下而上、有机增长的经济体系最终看起来就像城市一样。而那些自上而下、由某家公司拥有控制权的经济体系最终会像主题公园。单一公司的创造力永远无法与一个茂盛的生态系统相媲美。

互联网刚开始时是有如此多的美好愿景，但在过去十年里，却被少数巨大且破败的“主题公园”所把持。我们可以建造拥有欣欣向荣经济体系的“大城市”来取代它们。我们已经有了工具，而且种子也已播撒。
