**Subject:** [Tip] Pick Which Snippets Expand Using Delimiters

As you type, TextExpander will expand your abbreviations by default. If you type "ssig" to insert your signature in an email, your signature snippet will appear immediately. Delimiters give you more control over when a snippet expands. This applies to Windows
and Mac TextExpander desktop apps

No delimiter, no expansion.  Using delimiters means you have more flexibility in your abbreviations - you could use "sig" for signature, without it expanding when you type the words "sign" or "signal". | | | |  | | --- | | How to Set a Delimiter In  Preferences > Expansion > Expand Abbreviations, change the option from **immediately when typed** to **at delimiter** - and you’ll have two options.  **Option 1:** TextExpander keeps the delimiter. For example, use the space bar as a delimiter, and you can keep the space at the end of the snippet.  **Option 2:** TextExpander will "abandon" the delimiter after the expansion. For example, use the Tab key or a square bracket [ ] as delimiter, you probably don’t want that hanging around after your snippet expands. | | |  | | --- | | Set Delimiters | | | Click **Set
