老牌第三方微博客户端「重出江湖」：VVebo 3.0 大版本更新

04 月 20 日

如果你曾经历过 7、8 年前第三方微博客户端「百花齐放」的时期，相信你一定对 [VVebo](https://sspai.com/post/24964) 略有耳闻。VVebo 当年凭借内测时积累的口碑和不俗的素质在上架之初就吸引了大量用户。

但由于微博对第三方客户端的 API 严格限制，VVebo 尝试多次申请高级接口无果，借用 Weico 接口「曲线救国」的方法也被「封杀」后，开发者于 2015 年宣布放弃 VVebo。

![](.evernote-content/AF583590-118F-429A-AD53-0FCD5C2DEE7A/74CD28E6-0D9B-414B-BF13-906B802472C5.jpg)

随后的几年时间里，VVebo 只有寥寥几次维护性更新的记录。去年，开发者在微博上宣布 VVebo 已经解决了接口问题，并且会进行重新设计和开发，让一直坚守 VVebo 的用户们看到了希望。

今天，阔别多年的 VVebo 终于上架 App Store。应用延续了过去「一版一个样」的特点，这次的更新也带来了截然不同的使用体验。

简洁易用的设计和交互
----------

VVebo 从上架初就一直遵循着 iOS 的设计语言，新版本也不例外，原先方正的界面元素都变得圆润起来，更加贴近当前的 iOS 设计思路。在设计上，VVebo 3 的改变可以算得上是「翻天覆地」。

![](.evernote-content/AF583590-118F-429A-AD53-0FCD5C2DEE7A/D23C97D6-212C-42A7-8F79-45F9A33E0A0F.jpg)

左：老版本，右：新版本

在这次更新中，时间线从列表式变成卡片式，每一条微博都是一张卡片，点击即可展开。底栏也从原来的样式变成了浮动在底部的按钮，搜索和发送按钮被放在两侧，中间则分别是消息（@ 与评论）、主页和个人页。

![](.evernote-content/AF583590-118F-429A-AD53-0FCD5C2DEE7A/69F23F28-B40A-434D-ACF8-2561B7211B5D.png)

在时间线上，如果你长时间没有打开 VVebo，它会自动为你收起一部分未读的微博。除此之外， VVebo 在图片的显示上也做了优化，比如超过 9 张图时，会自动缩小部分图片以确保能够完整显示所有图片。

除了界面设计外，新版本也针对当前用户的操作习惯修改了交互逻辑。最显而易见的是 VVebo 新版本修改了滑动逻辑，在非边缘位置左右滑动变为切换列表和返回。

![](.evernote-content/AF583590-118F-429A-AD53-0FCD5C2DEE7A/3EB47411-18F0-4A5A-BCEF-5B21CC295805.gif)

在边缘左右滑动查看转发和评论

从微博边缘向左右滑动可以快速查看评论 / 转发，如果想评论或转发微博，则需手动点击。如果你滑动的是一条转发的微博，从边缘向内滑动并上下移动即可选择查看转发 / 原文的内容。当然，这些操作都能够在设置里进行修改，你可以根据自己的使用习惯进行调整。

新版本 VVebo 大幅修改了滑动逻辑，需要一段时间来适应。相比之下，点按操作的修改则显得友好许多。大部分的互动操作都改成了点按，默认情况下，你可以通过点击微博卡片底部的转发和评论按钮来单独查看该微博的转评情况。

而双击视频则可以快速开启悬浮模式，在看视频的同时也不妨碍你继续刷微博。VVebo 会检测你的音频播放方式，如果在非公放模式下（比如连接着耳机等）则会默认开启视频声音。你也可以在设置里自行调整视频声音的设置。

![](.evernote-content/AF583590-118F-429A-AD53-0FCD5C2DEE7A/7AEDC2D9-9330-45DB-AAE0-5E6881BB2EBB.gif)

双击悬浮视频

对了，在悬浮状态下，关闭视频的方式为向上滑动。你也可以通过向下滑动来进一步缩小悬浮窗。这一点在应用中没有给出明显提示，容易让人不知道如何关闭悬浮视频。

VVebo 在新版本还为「长按」赋予了更多操作，比如长按转发、评论的按钮可以进行对应操作，而长按搜索则可以在不离开当前界面的情况下呼出搜索栏。

「够用」的基础功能
---------

由于微博对第三方客户端 API 的严格限制，VVebo 自然也会有一些功能的缺失，比如私信功能就无法在 VVebo 上使用。但总体来说，VVebo 作为一款微博客户端，该有的功能基本都有，完全够用。

首先，最基础的转评赞、关注和取关，VVebo 都不在话下。在浏览微博时也没有发现明显的功能缺失，无论是超过 140 字的长微博，还是超过 9 张的配图都能正常显示，甚至连微博的热搜榜也有提供。

![](.evernote-content/AF583590-118F-429A-AD53-0FCD5C2DEE7A/8F20209C-F3A4-4E26-9D9A-D3EEBD113640.jpg)

在用户的个人页面，相关的过滤、备注、查看相册等功能也一应俱全。如果你私信功能对你来说不那么重要，VVebo 完全可以满足你对微博的日常需求。另外，开发者也在微博上提到，私信功能会在后续的版本中加入，值得期待。

![](.evernote-content/AF583590-118F-429A-AD53-0FCD5C2DEE7A/CADEA100-66A6-4C6A-9F0B-9BED325A5C49.jpg)

虽然目前 VVebo 支持阅读超过 140 字的微博，但它本身是不能发布长微博的。如果你的微博字数超过了 140，它就会被存到你的草稿箱中。不过超过 9 张图片却可以正常发布，希望应用能在未来支持发布长微博。

实用的「隐藏」功能
---------

VVebo 似乎很少宣传新版本的特色功能，即使是 3.0 大版本的更新日志里也只用「全新设计的版本」来一笔带过。实际上，在 VVebo 3 中有不少实用且隐藏较深的特色功能，这些功能可以大幅增加我们在刷微博时的体验。

当你长按一条微博时，可以在 Context Menu 里找到「大爆炸」的功能。VVebo 会自动为你删去标点和表情，你可以非常方便地复制自己想要的内容。除此之外，你也可以在「大爆炸」中选择不想看到的关键词进行屏蔽。

![](.evernote-content/AF583590-118F-429A-AD53-0FCD5C2DEE7A/F1F96ED7-79BB-4387-8CE5-4E4DAF2031D9.jpg)

除了在「大爆炸」里屏蔽关键词以外，VVebo 还支持在时间线中直接屏蔽某位用户。你所屏蔽的内容会在时间线中被收缩起来，如果你想查看，也可以点击右侧的眼睛形状按钮来展开。屏蔽功能的加入可以让你的时间线变得更加清爽，对于过滤无用信息很有帮助。

![](.evernote-content/AF583590-118F-429A-AD53-0FCD5C2DEE7A/BB3F389F-F61E-4000-9325-841FE5E82482.jpg)

如果你想取消屏蔽，只需要在相应微博右上角的菜单中设置即可。

另外，应用的搜索功能还支持在当前列表内进行搜索。这一功能可以帮你快速筛选时间线中的内容，或者在评论互动里筛选出特定的消息。当你在某一界面长按搜索按钮时，它会默认以「搜索列表」的形式呈现，不需要用户进行二次操作，非常方便。

![](.evernote-content/AF583590-118F-429A-AD53-0FCD5C2DEE7A/8CEC5053-C7B2-42D2-BE01-2D6F2BCD181D.gif)

长按搜索键直接呼出搜索栏

总结
--

总体来说，VVebo 此次更新称得上是一次「推倒重来」，在界面设计和交互逻辑上都与前代有极大的不同。最令人惊喜的，还是 VVebo 在经历了长时间功能缺失后，终于解决了接口问题，弥补了多年的遗憾。

此次回归，VVebo 并没有像其他应用那样改为订阅制收费，而是在应用内加入了打赏的机制。先前购买过 VVebo 的用户可以免费升级，价格则保持为 1 元不变。
