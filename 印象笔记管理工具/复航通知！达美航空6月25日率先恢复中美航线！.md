**▶  达美航空复航告知**

达美航空将于**6月25日恢复西雅图-上海浦东航线**，该航线经停首尔仁川机场，每周运营两班。

**7月起，达美航空将运营西雅图-上海浦东、底特律-上海浦东航线**，该航线同样经停仁川机场，每周运营一班。

您可以通过达美航空官网、Fly Delta应用程序、票务代理、OTA等渠道订票。

搭乘达美航空航班的乘客途径仁川国际机场时，乘客留在机上不进入机场，同时也不需要提供或提前准备韩国签证。

**▶  达美航空温馨提示**

为确保您的航旅安全请您仔细阅读以下流程。同时为了保证您的顺利出行，建议您提前到达机场并合理安排您的时间。

![](.evernote-content/5BE7B152-8F49-4601-AA0F-368F05E0E98A/894EE6C3-B13D-4BCE-8C28-2FAE2C3C1700.jpg)

|  |  |
| --- | --- |
| 1 | **行前准备：**   1. 中国旅客需在登机前14天起连续逐日通过扫描微信／网络二维码填报健康状况 2. 从美国出发的旅客须确保持有有效入境中国的签证 3. 请携带好自己的口罩，并在旅途全程佩戴 |
| 2 | **抵达前准备：**   1. 旅客需填写《健康申报表》 2. 等待检疫人员抵达现场，并允许地服人员打开舱门 |
| 3 | **下机前准备：**   1. 旅客经检疫人员允许分批离开飞机 2. 有疑似症状、特殊需要的旅客将先行离开机舱 3. 下飞机时将进行抵达后的第1次测温，根据机场现场情况及安排，整个过程将持续至少1.5小时 4. 检疫人员将在指定区域对所有旅客进行流行病学调查，同时审核《健康申报表》 5. 所有旅客都将在机场接受新冠肺炎病毒检测，检测结果会在6-8小时内公布 6. 旅客在进行红外测温后凭《健康申报表》办理入境手续 7. 当旅客领取托运行李后，将被送往指定酒店进行检疫隔离14天 |

\* 机场防疫措施可能会有变动，请以官方信息为准。
