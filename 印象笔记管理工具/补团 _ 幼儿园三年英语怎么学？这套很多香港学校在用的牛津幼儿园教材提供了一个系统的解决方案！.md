香港孩子学英语领先我们两三年

长期以来，香港的多数学校以英文授课为主，所以对英语水平的要求也比较高，小朋友从小都得学。

香港课程发展议会审定的《学前教育课程指引》也明确提出了幼儿园阶段的英语学习目标。不过看下来，一点都不“高冷”！

> 1、让孩子多听、多读故事，培养孩子学习英语的兴趣。
>
> 2、让孩子能认识、使用生活中常见的单词。
>
> 3、能听懂简单的对话。
>
> 4、教孩子唱有韵律的儿歌，逐渐培养孩子英语的语感，同时帮助记忆单词。

剑桥大学还专门调查过香港学生英语水平，他们发现一般小学会要求学生入学前达到剑桥少儿英语考试 *Starter*的水平。

![](.evernote-content/FADEEBB0-0BB7-48E6-B4CB-546004C1764A/9EDFA59F-F1F7-40B2-BB5E-135E6A4E456F.jpg)全世界人学英语的水平等级体系，C1级对应的是雅思7-8分，YLE的Starter级别是开始，是香港幼儿园毕业小朋友能达到的水平。

Starter 是剑桥少儿英语考试（“YLE”）的第一个级别，通常来说，能达到这个级别需要掌握四五百词。而国内孩子小学毕业一般能达到第二级 Mover 的水平，也就是基本掌握1000词。

> 剑桥少儿英语考试的三个级别：
>
> * Starters（1级，一般适合小一、二学生）
> * Movers（2级，一般适合小三、四学生）
> * Flyers（3级，一般适合小五、六学生）

总之，香港小朋友英语学习，比我们领先了2-3年。

为什么适合我们小朋友用？

不少花生团的朋友，直接带孩子用 *Wonders* 这样的英美本土教材进行英语启蒙；这个学习难度是蛮大的。毕竟，英语不是我们的母语，我们的孩子，在学书面英语前，在口头英语（Spoken English）上的积累和母语孩子是完全不一样的。

在英语环境上，香港小朋友比我们好一些，更多面对一个双语环境，但英语也不是他们的母语。因此，牛津这样的英语学习专业出版社就专门为他们编写了*Get Set Go* 这样的ESL（English as the Second Language) 教材。我们大多数孩子，跟这样的教材去学，要容易不少，可操作性比较强。

之前，我们介绍过，在英语国家，Phonics的学习周期一般是**从幼儿园一直学到小学二年级**左右：

> * 大约在幼儿园的时候，孩子们开始跟着老师系统学习*Phonics*；老师会结合课本、游戏、练习，教孩子一些简单的发音规律，并锻炼孩子们看到简单词汇自己读出来的能力。
>
> * 进了小学，继续学习1-2年的*Phonics*，掌握更多和更复杂的规则。上完二年级后，孩子们基本上已经会运用大部分*Phonics*规则，一般的拼读，不会有什么障碍（约70%的英文单词是符合 ***Letters-Sounds*****音形对应规律**的）。

这套书对应的就是幼儿园阶段的自然拼读学习。每个年级，每个学期，每一册主课本，都会有对应的Phonics学和练。三个年级，自然拼读学习的起点、过程、终点，列得很清楚，大家可以点开来看 ！

![](.evernote-content/FADEEBB0-0BB7-48E6-B4CB-546004C1764A/4E5974AE-62DD-47A1-9C48-63CB003AA4EB.jpg)

第三步，*Time to Sing,*把单词放在歌曲中，放在常用的句型中，如 *Let's*... 小朋友听歌、看图，结合生活场景，理解语言的实际应用。

![](.evernote-content/FADEEBB0-0BB7-48E6-B4CB-546004C1764A/39D870C9-A8A1-4E70-BBFA-573D39513C98.jpg)

第四步，*Time to Sound,* 练听力，结合图片，把听到的单词写下来。

![](.evernote-content/FADEEBB0-0BB7-48E6-B4CB-546004C1764A/3F3E0321-9CE0-479D-8FAF-AA44C9B7DA7C.jpg)

再到这个单元对应的 Activity Book里，做游戏、做练习 ...

![](.evernote-content/FADEEBB0-0BB7-48E6-B4CB-546004C1764A/60EB35E0-B77C-49D5-B809-E108D1E01B03.jpg)

此外，中班的自然拼读教材里，重点会练一些常用的拼读规则 *Letter-Sound Relationship*，比如 -at, -an, -ot, -un, -ad, -en, -ug, ip  ...

拿-ug来说，学习也分四步：先听故事，注意 -ug 怎么发音；

![](.evernote-content/FADEEBB0-0BB7-48E6-B4CB-546004C1764A/85DE23CE-70B1-4B1A-9F8E-491327C38B0B.jpg)

然后做贴纸游戏、唱歌 ...

![](.evernote-content/FADEEBB0-0BB7-48E6-B4CB-546004C1764A/7BED0713-290B-404F-868A-FE54A8BC14A2.jpg)

做专门的拼读练习，让孩子掌握单词中 -ug的发音规则 ...

![](.evernote-content/FADEEBB0-0BB7-48E6-B4CB-546004C1764A/455B3743-C814-43CC-AC1F-8DB359240203.jpg)

**3、幼儿园大班**

幼儿园大班，在学习结构上和中班是差不多的，也是按照一个个孩子生活中常见的主题来组织语言点和拼读规则的学习。

这是大班第二学期的（6）的目录；在中班的基础上，内容递进，实现学习进阶 ... 涉及到的句型，很多我们这里小学二三年级才开始学！

![](.evernote-content/FADEEBB0-0BB7-48E6-B4CB-546004C1764A/7639CF20-96F5-491D-84DD-1E1DE140051D.jpg)

拿 *Save the Earth*这个单元来说，可以明显看到故事变得完整起来，文字密度在增加 ... 

![](.evernote-content/FADEEBB0-0BB7-48E6-B4CB-546004C1764A/50ACBDC8-776F-413E-8DFA-62A0E15812C3.jpg)

词汇，也从baby words （宝宝词汇）开始逐步过渡到比较专业、复杂的词汇，比如 *dolphin, elephant, rhino*...

![](.evernote-content/FADEEBB0-0BB7-48E6-B4CB-546004C1764A/A5C9AADB-11A0-414E-AD35-16F5AE6B3A29.jpg)

还是遵循四个步骤：*Time to Play, Time to Say, Time to Sing 和 Time to Sound*，完成一个单元的学习。

![](.evernote-content/FADEEBB0-0BB7-48E6-B4CB-546004C1764A/AF5BF0C8-6498-4777-A08F-125FB4C80709.jpg)

完成主体学习后，再去Activity Book 做游戏、做练习，去 Phonics Book 去练相关的拼读规则 ...

这是幼儿园大班第二学期学到的规则，还是有点复杂度的！

![](.evernote-content/FADEEBB0-0BB7-48E6-B4CB-546004C1764A/5E4A9B87-25A4-4639-9053-E674A4CDAC4E.jpg)
