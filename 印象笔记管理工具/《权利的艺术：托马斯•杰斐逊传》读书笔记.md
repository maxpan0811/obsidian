# 《权利的艺术：托马斯•杰斐逊传》读书笔记

《权利的艺术：托马斯•杰斐逊传》读书笔记
--------------------

原创*2018-04-02**京琳-生医83*[无智书斋](http://mp.weixin.qq.com/s?__biz=MzA3NDUxMTgyMg==&mid=2760405726&idx=1&sn=723476ec6f4c893f72efeceb53e8064a&chksm=be02767f8975ff690c3673937a8813a0e0cf6a636ac9df6aca062bd155dcf10c5fd35d301080&mpshare=1&scene=1&srcid=0402FsgyIzEBOEY7IZeK4cdp##)

读完汉密尔顿传，顺理成章就找来他的政敌，托马斯•杰斐逊的传记来读。杰斐逊，美国独立宣言的起草者，自由平权宗教自由的思想家和倡导者，第三任美国总统。他曾是律师、驻法使节、佛吉尼亚州长、国务卿、副总统。他兴趣广泛，博览群书，一手设计建起了自己的庄园，对美食，音乐，科学，哲学人文和历史充满了热爱。他坚信“**人生来平等（Allmen are created equal）**”，同时他却拥有众多黑奴，有生之年没有给任何一位自由之身；他终身担忧权利集中会带来独裁和君主制，然而做总统时，他握有的权利远超过他的前任们。杰斐逊在世83年，身处战争和建国的时代，是个睿智，复杂而矛盾的的伟人，本以为他的传记定是恢弘跌宕。其实不然，本传记重点还是“权利的艺术”。

托马斯•杰斐逊被公认为美国历史上最杰出的总统。常言道：**哲学家思辨，政治家谋略。**杰斐逊二者兼具，哲学家的头脑使他的政治目标更具远见，政治家的手段使他的公众服务成就在历届总统中首屈一指。杰斐逊痛恨对峙冲突，深谙权利和人性的自然本性，他不断从错误中学习，在妥协中坚持，他引领人们实现着他的每一个构想和目标，而每一个目标的实现似乎都是水到渠成，没有跌宕，没有冲突，包括著名的独立宣言起草和通过，包括路易斯安那购地。作家乔恩•米查姆（Jon Meacham）让读者从杰斐逊的角度看杰斐逊身历的世界，杰斐逊一生中无数的成就便这样一一道来，令读者仿佛忘记了实现这些目标的艰难，忘记了他当政的时代年轻共和国是处在怎样的危机之中：政党间激烈分歧，国家经济摇曳脆弱，外在霸权国战争威胁，内在社会认知和人文冲突。

杰斐逊以其权利的艺术，成就了这个国家，也成就了自己的伟大。读完这本书，觉得该书值得一读，知道和学习了不少知识，若有缺憾，便是觉得作者可以在叙述中更故事话些。
