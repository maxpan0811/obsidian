# 在 Chrome 上用划词搜索查维基百科：Qikipedia

---

划词搜索是一个充满实用性的功能，在 iOS 中划词搜索可以得到单词的翻译，而在 Chrome 上当你用光标选中一些词语然后右键打开菜单就可以谷歌搜索，但往往在谷歌搜索显示的众多结果中我只会选择一类，那就是维基百科的词条。

维基百科是一个自由内容、自由编辑的百科全书网站，许多的作者们在写稿时也常常引用。把维基百科当作是一部参考工具书，一点也不过分。

所以为了更快速地获得信息，我们可以使用 [Qikipedia](https://chrome.google.com/webstore/detail/qikipedia/dljenobnbaealaglncgihenfkbkblefd) 这个插件。从名字就能看出，它是一个维基百科增强类的插件，利用它能够让我们在 Chrome 上实现划词搜索。

![](.evernote-content/A4B2E34E-745F-4E03-BF92-7C1A83AB14D1/8BFA6E83-A5FC-41CD-BD95-C87E27C22702.jpg)Qikipedia 划词搜索

安装 [Qikipedia](https://chrome.google.com/webstore/detail/qikipedia/dljenobnbaealaglncgihenfkbkblefd) 的方法很简单，只要在 Chrome 商店中搜索到它，点击添加到扩展程序中即可。然后，点击 Qikipedia 同时右键，打开 Qikipedia 可以更改一些设置选项，让 Qikipedia 的卡片更加个性化。

从大家对 macOS Mojave 新增的黑暗模式的反馈来看，还是有很多人喜欢黑暗模式的，甚至愿意为了体验而去升级 macOS。Qikipedia 也支持 Dark Theme，不过勾选 Dark Theme 后记得点击「SAVE」。这时候，划词搜索看看卡片是不是变成了黑色的了？

![](.evernote-content/A4B2E34E-745F-4E03-BF92-7C1A83AB14D1/82F2094F-E788-4566-9E1D-9C3BA7AAD80D.jpg)Dark Theme 的 Qikipedia

维基百科支持的语言有很多，全部包含 298 个语言版本，其中超过 100 万个条目的语言有 14 种。但对我们来说，常用的无非就是两种：中文、English。所以，在 [Qikipedia](https://chrome.google.com/webstore/detail/qikipedia/dljenobnbaealaglncgihenfkbkblefd) 的语言选项中选择好你常用的一个就可以了。
