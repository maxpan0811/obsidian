# 洛杉矶暴乱，说点你不知道的事情

洛杉矶暴动愈演愈烈，美国亲手掷出的回旋镖飞了六年，终于飞到了自己头上，原来“美丽的风景线”不会消失，只会转移，这一刻，所有移民紧密团结起来，这一刻，我们都是洛杉矶人。

![](.evernote-content/16C82B85-C617-495F-8FF9-2BC9CD9EC5DD/351EA1BF-BC33-4C37-8C90-F765D8B04789.png)

但今天我不打算幸灾乐祸，也不打算落井下石，我将以旁观者的视角从根子上说清楚这次暴动的来龙去脉，前因后果。

这一切都源自美国独特的司法体系，尤其是以联邦移民执法局（ICE）为代表的移民司法体系与美国地方刑事司法体系之间，那理论上存在优先地位、现实中却充满落差的复杂关系。

要理解这场冲突，就必须先理解一个关键机制，当一个人在美国的土地上，因某种行为——可能是酒后驾车，也可能是商店行窃——被地方警察拘捕时，他便同时踏入两套截然不同却又相互交织的“处理流程”。

第一套是地方刑事司法系统，由警察、法院、监狱构成，核心目标是惩治犯罪行为；第二套则由美国移民及海关执法局（ICE）主导的移民司法系统，ICE的目标不在于惩罚，而在于裁决一个根本问题：“你是否还有资格留在美国？” 

由于美国宪法将移民事务明确划归联邦政府专属管辖，而绝大多数日常刑事案件（如抢劫、酒驾）则归属各州管理，联邦权力的层级天然高于地方，ICE作为联邦机构，其执法依据是联邦移民法，在管辖权上拥有先天优势。

这种“优先权”的核心体现，是一种名为“移民拘留令”的工具，它如同一张由ICE发给地方警察局或监狱的“预订通知单”，上面写道：“请注意，贵处现关押的某某人，我方怀疑其违反联邦移民法，可能属于‘无证移民’或存在其他身份问题，待贵方案件处理完毕，请勿将其释放，代我方额外羁押最多48小时，我方将派人接收。”

设想有一个名叫胡安的老墨，因酒驾在洛杉矶被捕，他被关入洛杉矶县监狱，指纹信息随即录入系统，自动与联邦数据库（包含ICE数据）比对，ICE系统发现胡安缺乏合法居留身份，随即向洛杉矶县监狱发出一纸“移民拘留令”。

酒驾在通常情况下，只要不是累犯，法官会裁定“罪行较轻，缴纳5000美元保释金可取保候审。”

这个时候，胡安的家人凑足款项前往监狱接人，却被告知：“很抱歉，虽然刑事法庭允许保释，但我们收到了ICE的‘预订通知’，不能放人。” 于是胡安被径直移交给ICE，投入移民拘留中心。

这里的优先权体现为ICE的移民拘留令直接覆盖了刑事法官的保释令——刑事系统说“可以放”，移民系统说“不，留下”。

另一种情形是，胡安劣迹斑斑未能获得保释，最终被判入狱6个月，刑满之日，当他走出牢房，迎接他的并非自由，而是早已等候在监狱门口的ICE执法人员，他将被直接带走，送入移民拘留中心，等待被驱逐出境。

此处的优先权在于：刑事系统的结论是“服刑完毕，应予释放”，但移民系统在“释放”这个节点上实现了接管，将其导入另一套程序。

这种并行不悖的根源，在于两套系统的本质差异，刑事司法是“刑事”性质的，遵循无罪推定、律师辩护权、公开审判等原则，核心在于“惩罚”，而移民司法则被界定为“民事/行政”性质，其法律程序要求远不如刑事司法严格。

驱逐出境在法律上不被视为一种“惩罚”，而被定位为一种行政管理措施，就好像取消你的会员资格。

因此，两套系统得以并存，刑事系统处理“你做了何种错事”，移民系统则处理“你作为非公民，是否仍具备居留美国的资格”，一个刑事案件的终结，无论是无罪、保释、假释还是刑满释放，恰恰是移民程序的开始。

简言之，ICE主导的移民司法体系之所以拥有“优先权”，源于其联邦属性对地方刑事司法权的层级压制，它通过“移民拘留令”这一工具，实现对在押人员的“提前预订”。

无论刑事案件结果如何，只要此人脱离地方刑事系统的掌控，ICE便能立刻接手，决定其最终是回家还是被驱逐出境，这种“优先”并非指ICE能干预刑事审判，而是指其对个人最终处置权的掌控，拥有地方刑事司法体系无法比拟的优先地位。

![](.evernote-content/16C82B85-C617-495F-8FF9-2BC9CD9EC5DD/238600BF-2482-4654-9295-00C79AE36606.png)

司法制度的逻辑讲完了，接下来是重点，这套制度的矛盾所在。

首先，ICE的移民拘留令是一份行政请求，并非由法官签署的司法逮捕令，很多联邦法院已经裁定，地方警察仅凭此行政请求就在本应释放后继续关押嫌疑人，可能违反了美国宪法第四修正案禁止“无理扣押”的规定，地方政府担忧，若错误关押（如事后发现对方是美国公民），将面临巨额民事赔偿诉讼。

其次，若本地警察同时扮演移民警察角色，辖区内的无证移民将因恐惧被驱逐而不敢报警、不敢作证、不敢求助，这将为真正的犯罪分子提供藏匿的土壤，最终损害整个社区的公共安全，地方警方的目标是建立信任，而非制造恐慌，这是ICE移民拘留令被拒绝最常见的原因。

第三、经济负担。为ICE额外关押一名人员48小时，涉及食物、看管、医疗等成本，ICE通常不会支付这笔费用，很多地方政府认为这是联邦将自身职责的成本转嫁给自己。

最后一点是政治与道德立场，美国许多城市的领导人和居民在政治上倾向于保护移民，认为大规模驱逐不人道，或不认同联邦政府的移民政策，这些城市通常被称为“庇护城市”。

说完了矛盾，接下来是联邦政府和地方政府处理矛盾的方式。

面对地方政府的抗拒，联邦（特别是ICE和司法部）会采取多种手段施压。

第一、法律诉讼。联邦司法部可能起诉这些“庇护城市”或州，指控其违反联邦法律。核心争议在于联邦政府依据宪法主张其在移民事务上的最高权力，地方不得阻挠；而地方政府则援引宪法第十修正案，主张联邦政府无权强迫地方官员执行联邦行政指令。诉讼结果互有胜负，但法院通常倾向于保护地方自主权，认为ICE不能强制地方警察执行非强制性的拘留请求。

第二、财政威胁。这是联邦最常用的武器，威胁切断或扣减给予地方的某些联邦拨款，特别是司法援助拨款（如用于支持地方执法、法庭项目的“伯恩JAG拨款”），然而，这一招在法庭上同样屡屡受挫。法院多次裁定，联邦政府不能以不相关的联邦资金（如社区发展资金）要挟地方执行移民政策，只有当拨款用途与移民执法直接相关时，此类威胁才可能成立，因此，联邦实际能切断的资金非常有限。

第三、舆论施压与政治攻击。总统、司法部长、ICE局长等高层官员公开点名批评庇护城市的领导人，指责其“将罪犯释放到街头”、“威胁国家安全”。当某个被庇护城市释放的无证移民犯下严重罪行时，联邦政府及其支持的媒体会大肆渲染，将其作为地方政策失败的证据，煽动公众情绪，向地方政府施加巨大政治压力。

第四、报复性执法，也就是本轮洛杉矶事件的直接原因。 作为报复，ICE可能宣布在庇护城市进行大规模突击搜捕行动，深入社区、工作场所直接抓捕无证移民，这既可以彰显联邦政府的执法决心，也可以使得地方政府难堪，当联邦的执法遭遇州府的强力抵制，民众的情绪被瞬间点燃，于是，暴动发生了。

回到胡安的例子，洛杉矶县监狱收到了ICE的“移民拘留令”，但狱方认为“ICE的通知单非法官签署的正式逮捕令，我们无法律义务必须遵守，若强行超期关押，可能会面临诉讼及巨额赔偿。” 

因此，当胡安的家人缴纳保释金后，洛杉矶县监狱选择无视ICE的“预订通知”，将其直接释放。

ICE勃然大怒，相关联邦机构公开谴责洛杉矶“放走了潜在的危险人物”，威胁扣减给其警用装备补助金，作为报复，ICE的执法队还在洛杉矶多个社区进行高强度的搜捕行动以“弥补损失”。

地方政府拒绝执行移民拘留令，本质上是美国联邦与地方权力博弈的缩影，地方以宪法为依据进行拒绝，但需承受联邦巨大的财政和政治压力，使得这一领域高度政治化、争议不断且充满变数。

那么，现实博弈中谁占据上风？联邦强迫地方就范的成功率几何？这次洛杉矶暴动就是答案。

大规模社区搜捕使所有移民，无论身份合法与否都陷入恐惧之中，彻底破坏了人们与执法人员的接触意愿，严重动摇了社区警务的根基。

相比于在监狱可控环境中接收一名被羁押人员，深入社区搜捕目标需要更多人力、物力以及情报的支持，执法风险陡增。

更重要的是，这种报复性行动只会加剧地方政府和社区对联邦的反感，进一步坚定其庇护政策的决心，此次洛杉矶面对ICE的非常规操作，选择了正面抗衡，拒绝妥协，最终点燃了街头冲突的导火索。

洛杉矶之乱，是联邦移民权力主张与地方自治原则、执法策略的激烈碰撞，揭示了美国社会中不可调和的巨大矛盾，只要这种结构性矛盾存在，类似“洛杉矶之乱”的冲突就难以避免，就会成为一道持续流血的丑陋伤疤。
