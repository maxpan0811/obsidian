# 体验三天后，他决定卖掉相机买一台 Mavic Pro

---

![](.evernote-content/37B5E2D3-50A2-4B42-8A13-E871F8753DB4/A1876F04-5067-4D7F-8651-CBF4E98A094D.jpg)

大疆在上月底发布了便携无人机 Mavic Pro，因其小巧的机身 + 强大的性能，倍受好评，甚至获得了「史上最强」的称号。少数派此前在第一时间介绍了 Mavic Pro 的[十大亮点功能](http://sspai.com/35650)，但我们还希望进一步分享 Mavic Pro 的体验。  
  
不过这次，我们决定用不太一样的方式来写这篇体验文。动机源于国庆长假期间同事 [@Jackcivlie](http://matrix.sspai.com/user/643397) 的数天 Mavic Pro 体验经历，在放假上班第一天，他见我的第一句话是：

> 我准备把我的相机卖掉，然后买一台 Mavic Pro。

这勾起了我的好奇，这样一个平时相机不离手的摄影爱好者，怎么突然就要买买买了？（虽然他从来也是剁手无数）于是，我们做了一次内部面对面的采访对话，而这篇文章就是我们的对话纪实，你会从中了解到他说这句话背后的原因，也会进一步了解 Mavic Pro 实际上手用起来是怎样的。

遇到 Mavic Pro 之前，我对无人机的看法
------------------------

**Q：除了 Mavic Pro 你用过其它无人机吗？**

没有，虽然在 Phantom 一代和二代推出的时候我就对它们有个大概了解，但是一直没亲自体验过，所以说 Mavic Pro 算是我的首飞。

**Q：在这之前你对无人机的看法是怎样的？**

可能因为我本身就是摄影爱好者的缘故吧，有的人觉得无人机就是个玩具，和航模一样，但是我首先想到的就是航拍，我觉得它可以给我提供一种不同的拍摄视角，而这种视角是我以前很难体验到的，因此我一直都觉得无人机是一种很实用的拍摄工具。

**Q：那之前为什么没有购买无人机的想法呢？**

两方面原因，一方面我觉得无人机还是有些门槛的，操控不好的话不但有可能损坏机器，而且还有可能对人造成伤害，所以我一直比较谨慎。

另一方面是从成本考虑的，虽然一架家用无人机的价格可能没有我的相机加镜头贵，但是考虑到它的便携性、适用范围等原因，使用的次数可能远比相机要低得多，这就无形中增加了成本。当然，你也可以说是因为钱不够（笑）。

三天的体验，让我对无人机有了新的认识
------------------

**Q：十一这段时间你一共试飞了多少次？**

刚拿到机器的时候差不多体验了两三次，当时主要是熟悉一下基本操作，每次飞得时间都很短。十一期间算是比较完整地体验了一下，飞了三四天的时间，中间还喊了另外两位朋友一起飞了几次。

那几天我们去了福田中心公园、龙港大运中心等五六个地方，平均每天都会飞三四次，差不多完成了十几次完整的飞行（每次飞到电量用尽），我自己还尝试了两次夜间飞行。

![](.evernote-content/37B5E2D3-50A2-4B42-8A13-E871F8753DB4/EE60D6E9-0A45-48E1-8B23-9E47D124080A.jpg)

*DJI GO 中详细的飞行数据*

**Q：因为机器只配了一块电池，你们试飞的时候是怎么解决充电问题的？**

因为我们当天是开车出去的，所以就用了车载电源给电池充电。充电速度还算挺快的，一个小时左右就可以充满，所以我们在开车前往下一站的路上基本上就充得差不多了。

不过只有一块电池确实有些头痛，但是 Mavic Pro 在电池容量只有 Phantom 4 一半的情况下续航时间居然只减少了 1 分钟，这已经很不容易了。所以如果真的打算入手的话最好还是要购入一块备用电池，那样才玩得爽，毕竟拍摄这种东西还是讲究时机的，有时候你很难站在那里等一个钟头。

**Q：在经过了这么多天的体验之后，你对它印象深刻的点有哪些？**

刚才我有提到之前我对无人机的印象主要有两点：不方便携带，操作门槛高，而 Mavic Pro 让我在这两方面都有了一个新的认识，所以这也是我印象最深刻的两点。

首先是便携，想想以前身边的朋友出去飞无人机总是要专门背一个包来装机器，而 Mavic Pro 只要放在我平时常用的通勤包里就能出门了，这种体验上的差距还是蛮大的，而且机器也不重。

![](.evernote-content/37B5E2D3-50A2-4B42-8A13-E871F8753DB4/BFDAAE7F-E0E9-46AD-8414-B790DF03BAC0.jpg)

*图： @木头lancer*

其次就是操控上的感受，飞了 Mavic Pro 之后我发现并没有我想象的那么难，第一次飞的时候感觉就像初次用手柄玩游戏一样，感觉有点不适应，不过飞了一两次之后就没有这种感觉了，基本可以控制自如了。

而且它还配备的各种传感器，所以上手难度并不高，也不用过分担心「炸机」的问题。我的两个朋友也是第一次飞无人机，我指导他们一两次之后他们也很快就上手了。

航拍的话，一边控制飞机一边控制镜头稍微有些难度，因为毕竟机器不在你手里，不过对于之前有一些摄影经验的人来说也不难，再加上 Mavic Pro 自带的「指点飞行」「智能环绕」之类的辅助功能，很多时候你只要专心于操作镜头就行了，基本上可以说是只要会飞就会拍摄了。

![](.evernote-content/37B5E2D3-50A2-4B42-8A13-E871F8753DB4/B80C1A43-19BE-41D7-B917-D60C470B7891.jpg)

除了上面两点之外，这次飞 Mavic Pro 还给了我一个新的感受，就是它的配套手机应用 DJI GO 做得也不错，它可以详细地记录你的飞行信息，在飞行过程中还能提供各种反馈信息。应用还提供了「一站式」的视频服务，这对于想要用无人机航拍的我来说非常重要。

就像现在很多人「拍照 + 修图」的过程都是在手机上完成的一样，我在用无人机拍摄视频的时候也希望能够快速地在手机上完成剪辑并且分享出去，这些都可以通过 DJI GO 轻松完成。

因为 Mavic Pro 这次用了新的图传技术，7KM 范围之内无人机拍摄到的 1080p 视频都可以实时缓存到手机里面，省去了你每次都要把内存卡插到电脑上，然后再用电脑剪辑的麻烦。应用的剪辑功能也非常友好，并且提供了各种转场效果和背景音乐，做好之后可以直接发布在 DJI「天空之城」网站上，或者是发布到微博微信上，非常方便。

![](.evernote-content/37B5E2D3-50A2-4B42-8A13-E871F8753DB4/3831164C-82E7-440D-85CA-33E7900F8859.jpg)

*直观的剪辑界面*

这段视频就是我们当天在拍摄之后直接用 DJI GO 剪辑的，效果已经和电脑剪辑差不多了。

**Q：既然你这么看重航拍能力，那你觉得 Mavic Pro 的拍摄效果怎么样？**

之前就知道大疆的拍摄稳定性做得不错，这次虽然机身重量减轻多少会影响一些稳定性，但是实际体验效果还是很不错的，三轴机械云台确实起到了不错的稳定作用。有一次在微风情况下我们飞到了几百米的高空（高空风力会比地面感受更大），虽然 DJI GO 已经提醒我们会受到影响，但是拍摄画面其实还是很稳定的。

![](.evernote-content/37B5E2D3-50A2-4B42-8A13-E871F8753DB4/944539EF-A292-455F-96E8-C19CF56FE0ED.gif)

*百米高空拍摄画面也不抖动*

至于夜拍能力，如果你直接看夜拍视频的话确实会看到一些噪点，不过夜间摄像本身就很考验器材，所以说 Mavic Pro 的表现也算中规中矩。

之后我还测试了夜间长曝光的功能，Mavic Pro 最长支持 8s 长曝光，我当天试了 3s、6s 和 8s，最后效果最好的反而是 6s 和 8s 的效果，我测试的时候也有微风的环境，但是成像效果依然很满意。

![](.evernote-content/37B5E2D3-50A2-4B42-8A13-E871F8753DB4/C242D578-2788-4706-AF18-D9AFEE8CC013.jpg)

Mavic Pro 激发了我的拍摄灵感
-------------------

**Q：为什么产生了卖掉相机去买无人机的想法？**

从大学到现在陆陆续续已经拿相机拍了好几年，说实话那种「脚踏实地」的拍摄视角已经拍得有些厌倦了，以前在假期背上相机包去四处摄影的热情也没有以前那么高了。所以对于拍摄我一直都想做一些不一样的尝试。

无人机航拍正好给我提供了这种不同的拍摄视角，之前一直没有买无人机是因为考虑到的实用性和便携性的问题，现在这些问题在 Mavic Pro 上都得到了很好的解决，所以我觉得是时候入手一台了。

而且现在 iPhone 的摄影效果也越来越好，「随手拍 + 手机修图」基本上可以满足我平时的拍照需求了，所以我现在也逐渐开始转而用手机拍摄。

其实不只是我，我的两位朋友在试飞了几次 Mavic Pro 之后也都有了「卖相机换无人机」的想法，这种不一样的视角确实能够激发出不少新的灵感。
