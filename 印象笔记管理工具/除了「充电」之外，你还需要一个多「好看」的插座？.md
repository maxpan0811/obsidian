# 除了「充电」之外，你还需要一个多「好看」的插座？

---

在单车或是汽车的改装圈子里，一向有那么些被称为「心情件」的改装，即不会带来什么功能或者性能上的提升，但依靠造型风格或者颇佳的质感给驾驶者提供情绪价值的部件。

类似的概念在数码产品中也越来越常见。例如各类 IP 联名设计的手机壳、移动电源、键盘等的推出，都在表达着消费者对除基础功能之外的个性品味的追求。

![](.evernote-content/7ABEE1C7-E169-4674-B50F-8222846DD0A1/D428731F-1578-4AE8-AC60-EBEF78607C57.jpg)

随着居家办公、自由工作者的增多，这两年日益流行的桌面搭建，更是一种高度自我愉悦的活动，因而桌搭类产品在美学追求和个性表达上，相比其他赛道更上了一个高度。只是优秀的产品大概实在有限，在审美相仿的前提下，大家搭来搭去，社媒上的桌搭分享却越来越趋同，似乎各自不可避免的又陷入了不那么个性的困境。

![](.evernote-content/7ABEE1C7-E169-4674-B50F-8222846DD0A1/96FB1C42-8846-4362-8C3E-372E8C5637E4.png)

想要在这将将一平米的桌面上展现尽可能多的个性化，乐高积木无疑是一个绝佳的选择。

![](.evernote-content/7ABEE1C7-E169-4674-B50F-8222846DD0A1/EAE86E6D-465F-48AE-A1FD-271BBBDDBE0E.jpg)

图 / 玩具怪咖毛子哥

当然随着越来越多的移动电子设备如雨后春笋般涌现，以及桌面搭建中对无线化的极致追求，大家对进行更优雅高效的充电行为的需求也在不可避免的日渐提升。

聊到供电，我们这代消费者还是经历了插排这个产品的几度更迭变迁。从丑大黑粗的杂牌，到「颠覆行业」的小米插线板，再到桌搭文化大流行推动桌面充电坞站这个细分品类的发展，我们看到了具有更多设计元素、功能更丰富完善的桌面充电设备。

![](.evernote-content/7ABEE1C7-E169-4674-B50F-8222846DD0A1/DECE4D1A-B1D4-428D-8625-EC34BDAC6A84.jpg)

实际上过去两年也有那么几款桌面充电的「爆品」频繁出现在各位桌搭区 UP 的作品里，然而即便再出色的设计，经过频繁的视觉轰炸，加之一些不合时宜的搭配，也会让人觉得乏味、并逐渐对之祛魅。

所以一款正在逐渐变得不可或缺的产品，面对上日益注重个性化表达的消费者群体，当两个话题相互碰撞，仿佛周星星又跳出来发问：插排+积木有没有搞头？

这就是 Pixel 65 啦。

![](.evernote-content/7ABEE1C7-E169-4674-B50F-8222846DD0A1/1F755235-4045-47CD-9E14-0117857EA600.jpg)

**▍****充电设备的自我修养**

作为一款桌面充电设备，在基础形态上，Pixel 65 算是把前段时间相对成功的两款桌面充电设备的优点结合了起来：方正规则的外观，顶部设计三个国标五孔插座，同时右侧有一个可以组合/拆卸使用的多口充电头。

可能是受限于成本，右侧的充电头拆卸后插排本体并没有设计更多的 TypeC 供电口，算是一个小遗憾。

![](.evernote-content/7ABEE1C7-E169-4674-B50F-8222846DD0A1/0FD081D7-1B40-45D1-8202-EB31ED0EE393.jpg)

右侧可拆卸的是一个 65W/1A1C 的氮化镓充电头，体积中规中矩，插脚不能折叠是个小缺点。65W 是一个非常经典的功率设计，2018 年联想推出第一代 65W 「口红电源」大获成功，2019 年又由倍思带来了第一款多口的 65W 氮化镓充电器并将这个品类的热度延续至今。

![](.evernote-content/7ABEE1C7-E169-4674-B50F-8222846DD0A1/A599BE21-E20B-460F-96EE-33641D85C309.jpg)

接口上 Pixel 65 这只 65W 选择了 1A+1C 的配置方案，A 口最高功率 30W，C 口最高功率 65W，双口同时使用的话则是 45W+12W 的功率分配，A 口没能做到 18W 稍有遗憾。

其实从 1A+1C 的设计也能理解到，这只 65W 充电头的定位更偏向保守，还是考虑了大量老旧 A 口设备的供电需求，而并非是为那些极客考虑，设计成更激进的双 C 口配置。

![](.evernote-content/7ABEE1C7-E169-4674-B50F-8222846DD0A1/5E7EDAFE-23D7-4DCB-9DA1-A94A7D7F8AD6.jpg)

A 口协议测试：支持 FCP、SCP、AFC、QC3.0、DCP

![](.evernote-content/7ABEE1C7-E169-4674-B50F-8222846DD0A1/33E2F76A-5E23-4242-9777-611917962272.jpg)

C 口协议测试：支持 FCP、SCP、AFC、QC3.0、PD3.0、QC4+、DCP、Apple 2.4A、PPS

![](.evernote-content/7ABEE1C7-E169-4674-B50F-8222846DD0A1/6AB4D6D8-8F2E-4FB9-9249-EB264F5D10B1.jpg)

C 口具备 5V3A、9V3A、12V3A、15V3A、20V3.25A 五组固定电压档位，以及一组 PPS 电压档位

当然如果你有更高至百瓦的供电需求，或者在 1A1C 之外还有更多的电子产品需要充电，Pixel 65 也并非不能满足，毕竟还有三个国标五孔可以插上其他的充电设备。

![](.evernote-content/7ABEE1C7-E169-4674-B50F-8222846DD0A1/51D84BFF-BEAC-4721-8C98-EAD593519F54.jpg)

**▍****个性到极致的创意空间**

当然除了基础的供电功能，如你所见，这款桌面插座最大的特色还是遍布全身的「凸凸点点」。像前面所说，Pixel 65 给足了 DIY 的空间，让积木爱好者们可以尽情发挥创造力。只要稍微动点心思就能完成「客制化」的搭配，就像给机械键盘更换键帽一样，没有标准答案，更不会有雷同的尴尬，作为桌搭产品，这大概就是我们期望的个性到极致的创意。

![](.evernote-content/7ABEE1C7-E169-4674-B50F-8222846DD0A1/BA5C67E4-450A-4703-B293-7A28708A492F.jpg)

当然在桌面这个场景下，除了给那些本就摆在桌面上的乐高小人找个展位之外，积木这个概念还是让我想到了很多别的玩法。

比如桌面上像 U 盘、USB Dongle、键帽之类零零碎碎小物件的收纳一向是个棘手问题，那么不如就来 3D 打印一个小收纳盒；又或者如果是客制化键盘玩家，也可以把试轴器和 Pixel 65 结合起来；或者给 65W 的氮化镓充电头做一个收纳数据线的配件，也是当你准备把它带走时更加便携的一个办法。

![](.evernote-content/7ABEE1C7-E169-4674-B50F-8222846DD0A1/A3E79DB2-9759-4910-88FD-0A1443D0F805.jpg)

坦白讲，我个人并不是个乐高爱好者，甚至对这个经典的玩具系列知之甚少，在拿到 Pixel 65 之后我才勉强搞懂了乐高积木的装配原理和关系。但这并没有影响我迅速对这款产品产生了兴趣并打开建模软件，搞了一些奇奇怪怪的配件出来。

这种感觉和我热衷的桌搭颇为类似，本质上都是一些物件的搭配和组合，要和谐，要美感，要高效，要有趣。与我起初看到这款产品的第一印象不同，Pixel 65 并非是乐高爱好者的小众自嗨，反而似乎有引人入积木坑的魅力了。

![](.evernote-content/7ABEE1C7-E169-4674-B50F-8222846DD0A1/983F3A27-ADB7-4E26-B80A-F43E93908D69.jpg)

而即便你并没有丝毫的探索和创作热情，讲实在的，只是把套装里附送的积木零件装配好，他也已经是一个足够漂亮的心情摆件。

![](.evernote-content/7ABEE1C7-E169-4674-B50F-8222846DD0A1/3BB2C75B-7FC1-428E-9F7D-99E55A8697D2.jpg)
