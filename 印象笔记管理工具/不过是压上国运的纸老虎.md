# 不过是压上国运的纸老虎

![](.evernote-content/1A8BB698-9361-4FC3-8F90-D4E888FE933D/55F4B375-20DB-4511-AA81-B781A94980EE)

强硬的以色列，也不过是只纸老虎，正压上国运来一场豪赌。

以色列极度缺乏安全感，作为美国全球三大支点的核心，国家命运与美国深度绑定，当美国强盛之时，大可以对巴勒斯坦人好一点，用怀柔的手段慢慢的扩大实际占据的领土。

但当美国相对实力衰弱，更甚至要从中东撤出，阿拉伯世界开始大和解，以色列就成了阿拉伯世界的对立面，其紧张可想而知，千年来的不安全感立刻激发，就像被一只猛虎追赶一样，必须拼命往前冲，也顾不上吃相了。

但这种拼命挣扎，肆无忌惮的滥用暴力，恰恰加剧了以色列的危机，让以色列变得更加脆弱。

一国对于自我实力的认知，会直接影响到对内对外决策。当苏联解体，美国发动海湾战争打垮了号称世界第三的伊拉克，建立了人类历史上前所未有的强盛的霸权，与美国全球战略、美国犹太人深度绑定的以色列，感到前所未有的安全。

于是才有了对巴勒斯坦人的缓和，推动巴以和解，同意巴勒斯坦建国。但这绝不意味着以色列放弃完全占据巴勒斯坦甚至更广阔领土的想法，只是看到了巴勒斯坦人的力量，想要慢慢来，而且以色列也很需要巴勒斯坦人承担低收入劳作。

比如在加沙地带北部建立的埃雷兹工业园区，就曾有200多家工厂，雇佣了5000多名巴勒斯坦工人，而现在的以色列境内，就有大量没有以色列国籍的巴勒斯坦人从事建筑工、清洁工、洗碗工等体力劳动，这是以色列繁荣的基础。

但利弊一体。

苏联的瓦解令以色列的国际环境大为改善，得以与众多第三世界国家加强经贸往来，更重要的是，迎来了远超几十年前的苏联犹太人移民潮。

1989年底以色列总人口不过456万人，而仅1990-1997年就涌入了71万苏联犹太人，以色列境内来自苏联的犹太移民总人数突破100万人，2019年大选中，来自苏联的移民中有77万选民，占全部选民的12%，对应15-17个议会席位，成了影响以色列政局的关键力量。

苏联移民的人口素质太高了，有一半人在移民前从事学术研究或管理工作。仅1990-1993年来到以色列的苏联移民，就有57400名工程师、12200名医生，而1989年的以色列全国也不过只有30200名工程师和15600名医生。

得益于犹太人注重教育以及苏联对犹太民族的优待，这批人在苏联还从事诸多尖端领域的研究，极大推动了以色列的科技进步和高科技工业发展，奠定了以色列在软件开发、图像处理、半导体等领域的雄厚基础。近年来，在《全球创新指数报告》中，以色列创新总指数居常年稳居全球前20，研发支出强度、创新竞争力、科技企业融资环境等子项经常排名全球第一。

![](.evernote-content/1A8BB698-9361-4FC3-8F90-D4E888FE933D/B3F1035F-DBDB-4C59-B41D-C0B1659A18FC)

现在的以色列14%的公民从事技术和非技术的高科技职业，科技产业占国内生产总值的两成，科技服务业也就是知识密集型服务业明显高于传统服务业。2021年高科技出口占总出口额的54%，其中半导体产业占16%，不足千万人口，就有超过3万名芯片工程师，超过200家芯片公司，也因此吸引了全球高技术企业投资。

![](.evernote-content/1A8BB698-9361-4FC3-8F90-D4E888FE933D/CB3CDF35-D63D-4699-95BC-0C95AE4D1420)

依托强大的人才优势，以色列得以实现彻底的经济转型，从计划经济转为市场经济，从公有制为主体转为私有制，大力促进科技创新和创业，以色列成了“创业的国度”。

![](.evernote-content/1A8BB698-9361-4FC3-8F90-D4E888FE933D/C82034CE-3EE6-4B4A-8463-E304E6329BAC)

大量苏联犹太移民的涌入，不但增加了消费需求，引来了大量外部投资，还大大刺激了以色列的建筑和房地产业，以色列开始大规模修建住房和定居点，很多小城镇迅速发展，以色列内盖夫地区中心贝尔谢巴，之前不过是一个非常贫瘠的小镇，大量苏联犹太人涌入让这座小镇人口迅速增加到25万以上，这才有了现在的繁荣景象。

但是，又要说但是了。

苏联犹太人带来的不仅仅是好处，在美国空前强大震慑四方的时候，还不要紧，但是当地区局势恶化，就不一样了。

苏联犹太人受苏联影响，加上第四次中东战争险胜的记忆，特别担心国家战略纵深不足，而以色列当初把苏联犹太人更多安置在靠近加沙乃至约旦河西岸等巴勒斯坦聚集区，在挤占巴勒斯坦人生存资源和空间的同时，也加剧了巴以矛盾，令深处第一线的苏联犹太移民更加看重安全。

![](.evernote-content/1A8BB698-9361-4FC3-8F90-D4E888FE933D/5D43D439-83C4-42A5-AD56-80CE0972BC98)

种种原因之下，苏联犹太移民格外渴望为以色列建立更加厚实的战略屏障，弥补战略纵深的不足，甚至把“定居点”当成了实现“大以色列梦”的支点。

苏联犹太移民特别重视安全，选民人数足够多，以色列国内没有占据绝对优势的党派，党派众多、力量分散，于是苏联犹太移民就成了左右政局的力量。

随着安全局势恶化，尤其在2000年巴勒斯坦第二次大起义、2006年以黎冲突、2009年针对加沙的“铸铅行动”之后，更强调安全，右翼的家园党开始集中代表苏联犹太移民的意志，在2009年大选时提出的竞选口号最优先的就是解决以色列的安全问题，而不是移民的利益。

强硬的犹太移民选票对利库德集团和内塔尼亚胡越来越重要。2013年就是因为家园党加入利库德集团，才让联盟领先其他党派成功组阁，而2018年家园党不满内塔尼亚胡与哈马斯达成停火协议，党首利伯曼辞去国防部长职务，2019年选举时更是拒绝与利库德集团联盟，导致组阁失败，还导致以色列在2019到2020年一连进行了四次选举。

![](.evernote-content/1A8BB698-9361-4FC3-8F90-D4E888FE933D/3E3395E4-18E6-4095-8AC3-A6F5597DD58F)

或许是为了表达忠诚，移民反而更加极端。就像现在以色列国安部长本·格维尔，本是伊拉克籍的库尔德人，是被犹太夫妇收养才成了以色列国民，但本人却比以色列人还极端，更是极为仇视阿拉伯人，甚至公开呼吁干掉拉宾总理，这次哈马斯发动“阿克萨洪水”行动，直接起因就是本·格维尔以部长身份闯入阿克萨清真寺宣誓主权。

介绍完以色列内部苏联犹太移民，也就更能理解为什么内塔尼亚胡如此强硬。

但强硬是有代价的，看起来强大的以色列不过是纸老虎罢了，如果再不找好机会顺坡下驴，很有可能遭受难以挽回的重创。

以色列被阿拉伯世界包围，安全很重要，但更关键的还是经济，经济才是以色列能够立足的根本，而以色列的经济结构极为脆弱。

![](.evernote-content/1A8BB698-9361-4FC3-8F90-D4E888FE933D/B137B8DF-C323-466C-80F7-C6AF2511360C)

以色列超过7成GDP属于服务业，核心是科技服务业，因为科技含量高、是创业之国，所以高收入的人群多，带动消费能力大幅提升，也就让服务业利润更高，从而带动其他服务业从业者水涨船高，形成了非常良性的循环。

以色列虽然农业发达，但主要生产的是高附加值的农产品，通过出口赚取丰厚利润后再购买粮食，和平时期还好，一旦局势动荡，以色列甚至都无法保证自身粮食安全。

而以色列在苏联解体之后推动经济体制改革，越来越依仗私营和外来资本，好处是以色列成了外国资本投资的乐土，外资仅对以色列高科技产业的投资就从2015年的33.2亿美元增至2021年的256亿美元。

但这也就意味着：以色列的繁荣，建立在外资、人才、科技创新和全球化的基础上，高度依赖一个安全稳定的外部环境。2000年后巴以局势恶化，就导致以色列2001、2002年连续两年负增长，旅游业也出现锐减。

只是好在苏联高素质移民的融入令以色列赶上了信息革命和全球化浪潮，占据优势的半导体融入时代大势，获得了极为丰厚的回报，之后巴以局势稳定，也让以色列渡过难关。

但这次呢？

2000年9月爆发的巴以冲突持续20多个月，总共造成双方2000人左右死亡，1万5千余人受伤，而这一次烈度远高于此前任何一次冲突，一周多时间，双方死伤人数已经超过2000年两年总和，更严重的是以色列全国都面临安全威胁，被称为以色列硅谷的特拉维夫也未能幸免。

![](.evernote-content/1A8BB698-9361-4FC3-8F90-D4E888FE933D/6C058E26-C383-42A4-8544-F0263E96AE2C)

这必然会极大动摇外部投资者的信心，比如英伟达已经宣布取消原定本周在特拉维夫举行的人工智能峰会，高科技人才呢？

以色列现在本土人才供应已经出现不足，非常依赖全球高素质移民，研发投资更是依赖外部和私人投资，2020年以色列政府研发投资占研发总投资的比重竟然只有9.6%，是OECD国家中最低的。

产业结构就更加危险了，以色列本国的科技企业基本从事高利润的人力资本和知识经济，把技术变为现实的本国工业制造相对薄弱，高度依赖跨国公司。

一旦因为安全得不到保障，外部投资、移民大幅下降，甚至存量资产、产业、人才流失，以色列的经济必将遭到重创。而局势的恶化，必然增加军费开支，更是雪上加霜。

而这一次一旦经济受创，再想摆脱危机可就难了。

毕竟再没有上百万高素质的苏联犹太移民，半导体产业全球化被打断，各国都在试图自建体系，以色列本来提出了雄心勃勃的半导体产业计划，一旦冲突长期持续，人才、资本流失，又遭遇智能革命和全球产业大变革，错过关键的几年，再想追上可就难了。

继续强硬？就算清理掉哈马斯又能如何，幸存的巴勒斯坦人中一定会产生第二个、第三个哈马斯，至于搞种族灭绝，就别开玩笑了，那更是自寻死路。

自己制造的恶果，自己解决。
