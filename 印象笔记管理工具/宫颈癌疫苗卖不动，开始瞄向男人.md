# 宫颈癌疫苗卖不动，开始瞄向男人

---

速读摘要

万泰生物是国内布局国产HPV九价疫苗的巨头之一。2023年，其向中国猛增了九价疫苗供应，到了今年第二季度，默沙东CEO罗伯特·戴维斯称，其HPV疫苗的中国代理分销商智飞生物向接种点运输的疫苗量突然大幅下降，库存高于正常水平，预计要到2025年才能恢复正常库存。甚至2019年，世卫组织呼吁，为男孩提供HPV疫苗接种的国家(主要是发达国家)暂缓这一项目，直到所有需要疫苗的女孩先完成接种——2023年，全球女孩第一剂HPV疫苗覆盖率为27%，与WHO设定的90%的战略目标仍有差距。

“‘520’爱她就带她去打HPV疫苗”的宣言，也许很快就要附加一句针对男性的新词：爱她就也给自己打HPV疫苗。

因为疫苗厂商眼下正着手开拓男性市场，拟将原本主要预防女性宫颈癌的HPV疫苗，不限性别地安利给更多人群。

11月19日，最先推出国产二价HPV疫苗的万泰生物公告称，其研发的九价HPV疫苗获批开展针对男性人群适应症的临床试验。

![](.evernote-content/E97B2E62-5480-4E56-990A-44F227C56AA8/0C7C5138-B2A5-415B-B8C7-93944F191A2B.jpg)

万泰生物于近日发布公告表示，其子公司厦门万泰沧海生物技术有限公司已收到国家药监局下发的《药物临床试验通知书》，同意其九价HPV疫苗开展针对男性人群适应证的临床试验

万泰生物是国内布局国产HPV九价疫苗的巨头之一。

而更早之前，默沙东、国产疫苗厂商康乐卫士也获批开展针对男性的临床试验。

疫苗厂商都想另辟蹊径地把蛋糕做大、抢占先机，以对冲女性市场的颓势、减少同质化竞争，让各自年报里的利润，都更体面些。

2024年，HPV疫苗厂商半年报里的营收和HPV疫苗签批量都较同期骤减，日子显然不太好过。有销售对盐财经表示，HPV疫苗推广的确进入了瓶颈期，“好多都爱搭不理的”。

即便男性市场被视为增量和蓝海，但受HPV疫苗先入为主的“女性受众”印象影响，今后要让中国男性买账也显然不易。

![](.evernote-content/E97B2E62-5480-4E56-990A-44F227C56AA8/BABC4F26-DFA7-4A4E-92ED-C814453A8A13.png)

**供需关系逆转，厂商急着降库存**

双十二的最后业绩冲刺，一条约苗营销短信弹窗发来提醒：年末最后一场。约HPV九价疫苗，不需等待，即刻接种，限时补贴1300元现金。

只有经历过2021年前后“九价一针难求”的人，才理解前后的反差。

当年，九价比春运火车票还难抢。不少城市用“摇号”的方式预约接种，深圳2022年第一期九价HPV疫苗摇号，56.2万申请者中仅有1.73万人摇中，“中签率”仅3%，等待期还很长；没有抢苗耐心的，得加价千元才能约上。

但如今，供需关系逆转。盐财经记者注意到，广州各区众多社区医院HPV疫苗预约显示有号，随手选择市区一家社区医院，九价苗预约页面显示，开放的预约接种日当天余号99。

![](.evernote-content/E97B2E62-5480-4E56-990A-44F227C56AA8/D1BFE1D9-09A1-4961-9DD6-726E71E9B264.jpg)

盐财经记者注意到，广州各区众多社区医院HPV疫苗预约显示有号 / 图源：截图自小程序@广州市预防接种服务

种种迹象提示着HPV疫苗进入了供大于求的阶段。

从各疫苗厂商的报告中更能窥得全貌。

先看市场份额最大的默沙东。

2023年，其向中国猛增了九价疫苗供应，到了今年第二季度，默沙东CEO罗伯特·戴维斯称，其HPV疫苗的中国代理分销商智飞生物向接种点运输的疫苗量突然大幅下降，库存高于正常水平，预计要到2025年才能恢复正常库存。

今年三季度，在其他海外市场增长的情况下，默沙东的HPV疫苗销售额下降了10%。

这不只是默沙东的滑铁卢，国产二价疫苗代表万泰生物也急需降库存。其2023年年报披露的疫苗库存量为2983.06万支，和当年产量相当， 大约是当年销量的2倍。

![](.evernote-content/E97B2E62-5480-4E56-990A-44F227C56AA8/58183573-6D38-43B4-9DDC-D9B67B5411E0.png)

2023年万泰疫苗年报披露的库存数据‍‍

今年上半年，万泰生物净利润2.6亿元，同比下降84.7%。另一国产二价HPV疫苗厂商沃森生物，今年上半年净利润1.7亿元，同比下降62.53%；倒是背靠默沙东的智飞生物，还有22.34亿元的净利润，但也同比下降47.55%。

在多部委《加速消除宫颈癌行动计划》的政策利好下，眼下HPV疫苗库存集体积压，来自多方压力传导。

默沙东在二季度电话会议中透露，从去年开始的医疗反腐活动也波及了HPV疫苗领域，“我们确实认为现在开始看到影响，在卫生行业，特别是在中国的疾控中心系统减少了交流，以及接种疫苗人数在减少。”

甘肃一县级基层医院院长对盐财经回忆，约从2020年开始，随着HPV国产疫苗弥补了当地的供应缺口，此前积压多年的需求集中释放，接种人数也最多。

然而，当“想打的都打完了”之后，无论是地方政府面向适龄女童的免费接种项目，还是成人自费，接种新增人数都迅速回落并趋于平稳。

当中，国产HPV疫苗更显不易。

尤其默沙东九价疫苗自2022年8月扩龄至26～45岁人群，并在2023年供应量翻番后，原有年龄限制、供应不足的问题不复存在，国产二价的市场也持续受到挤压，痛感尤为强烈。

![](.evernote-content/E97B2E62-5480-4E56-990A-44F227C56AA8/90148AF2-8A28-435E-89B8-DB54A9F729BC.png)

**男性打HPV，药企乐观，WHO保守**

早在2021年，HPV疫苗就纷纷投注男性市场。

最先行动的是默沙东，其CEO也在今年二季度电话会上再次提及，期待男性适应症的潜在批准，“我们认为这代表了一个重要的机会”。

回到中国厂商，在万泰之前，康乐卫士跑得更快，在2022年12月就开始了其九价HPV疫苗男性适应证三期临床研究。

给男性接种HPV疫苗是门好生意吗？厂商显得很乐观。有测算认为，若以9岁～45岁年龄段推测，中国潜在的接种目标男性人数超过3亿，市场很大。

![](.evernote-content/E97B2E62-5480-4E56-990A-44F227C56AA8/DED00BF4-A527-4A4B-91EC-8BA9712ECCE0.jpg)

国内HPV疫苗男性适应症潜在存量市场空间测算 / 图源：国金证券研究所

医学研究也提供了论据支持，指出感染HPV除了可能导致女性宫颈癌、阴道癌，也可能诱发肛门、生殖器区、口腔、头部、颈部的癌症、生殖器疣等“男女共患”的疾病。同时，性生活是HPV感染的主要方式，而男性是潜在的病毒载体，也可能增加伴侣的感染风险。

再者，2023年，《Lancet Global Health》的一篇研究显示，15岁以上的男性中，近三分之一的人感染了至少一种HPV，约五分之一的男性是高危可致癌型HPV病毒携带者。

因此，一些研究主张，将男性纳入疫苗接种人群，有利于控制HPV感染，降低相关疾病风险。

国际上已有先例。

HPV疫苗接种最早从女性开始，后扩展到男性。目前全球有51个国家实行性别中立的HPV疫苗接种政策，多数位于欧美高收入水平国家。据WHO，到2022年，已有125个国家在其国家女童免疫规划中引入了HPV疫苗，47个国家也包括了男童。即便如此，男童的HPV疫苗接种率仍然低于女童。

此外，一些倡议提出，将男男性行为群体（MSM）纳入HPV疫苗保护。如来自中国医学科学院的一篇研究认为：中国应加快HPV疫苗在男性中使用的适应症研究及审批，尽快为HIV阳性的MSM群体接种HPV疫苗，优先为小于等于26岁的MSM人群接种HPV疫苗，控制性病和艾滋病的传播。这篇研究来自乔友林团队，他是世卫组织全球消除宫颈癌专家组成员，多年来致力于国内的宫颈癌防治。

![](.evernote-content/E97B2E62-5480-4E56-990A-44F227C56AA8/2416E51A-F70F-4720-B902-5D1A70B4BA5A.jpg)

今年9月，一位36岁的男性受访者在新加坡自费接种了九价hpv疫苗，以为自己和伴侣提供保护，图为接种凭证/图源：受访者供图

尽管为男性接种HPV疫苗有其合理性，但因为男性需要更高价次的HPV疫苗，其是否值得纳入公共决策仍有争议。

早在2014年的研究中，挪威国家卫生局宫颈癌筛查咨询委员会的研究人员就对比了男女接种HPV疫苗的成本。他们发现，虽然为男孩接种疫苗有益，但提高女孩的疫苗接种率更有效、更经济，并且可以通过群体免疫帮助男性预防HPV相关癌症，所以应该女孩优先。

甚至2019年，世卫组织呼吁，为男孩提供HPV疫苗接种的国家（主要是发达国家）暂缓这一项目，直到所有需要疫苗的女孩先完成接种——2023年，全球女孩第一剂HPV疫苗覆盖率为27%，与WHO设定的90%的战略目标仍有差距。

![](.evernote-content/E97B2E62-5480-4E56-990A-44F227C56AA8/4BB087F1-16D4-415D-91E8-01BDB9E250F6.jpg)

2023年，全球女孩第一剂HPV疫苗覆盖率为27% / 图源：视觉中国

WHO的立场更为保守审慎。其在最新的HPV疫苗立场文件中明确表示：

在筛查及其他宫颈癌防控措施覆盖面有限、资源匮乏的环境中，按照目前二价和四价疫苗的价格，无论使用何种方案或疫苗，仅为女童接种疫苗都具有成本效益。不分性别的疫苗接种比仅为女童接种疫苗的成本效益要低。

世卫组织建议接种HPV疫苗的主要目标人群为性活跃之前的9-14岁女童。HPV免疫规划应从引入时起就优先考虑高覆盖率，女童覆盖率超过80%也可降低男童感染HPV的风险。

只有当可行和负担得起，并且不会影响主要目标人群疫苗接种或有效宫颈癌筛查规划的资源，才建议为次要目标人群接种疫苗，例如大于等于15岁的女性、男童、成年男性或男男性行为者。

回到国内语境，这些数据和立场也能提供参考。

2020年，中国疾病预防控制中心累估的9～45岁适龄女性接种率为2.24%。随着国产疫苗入局，接种率得以提升。据国金证券估算，2023年底，全国HPV疫苗渗透率约为19%。

![](.evernote-content/E97B2E62-5480-4E56-990A-44F227C56AA8/BD6D3256-7755-4A80-934D-45E99F793BE2.jpg)

2021-2030年国内HPV疫苗市场空间与渗透率提升测算 / 图源：国金证券研究所

今年10月，卫健委妇幼司通报，涉及11个试点省份的HPV疫苗免费接种政策覆盖了全国约40%的适龄女孩（13～14岁）。

仅仅3年，提升可谓显著。参考昆山杜克大学全球健康研究中心《HPV疫苗接种和纳入国家免疫规划的情况》，一方面，中国的HPV接种率已超过亚洲整体（女性第一针剂接种率不及6%）水平，但和高收入国家（女性第一针接种率为50%）仍有差距。

女孩仍需HPV疫苗保护。

![](.evernote-content/E97B2E62-5480-4E56-990A-44F227C56AA8/92564943-D8F6-4299-81E3-07E269119BCF.png)

**赛道拥挤，国产HPV疫苗开打价格战**

在中国，很少有一款疫苗像HPV疫苗这样特殊。如果疫苗有网红，它算得上头部。

其网红特质表现在：即便HPV疫苗尚未纳入国家免疫规划，在中国上市也晚，却是认知度、接种意愿颇高的疫苗。同时，尽管HPV对于防治宫颈癌确有成效，但对接种疫苗的舆论关注，也远高于对宫颈癌筛查的关注，哪怕疫苗无法取代筛查的作用。

2016年，葛兰素史克的二价HPV疫苗才率先获批进入中国，此后默沙东的四价、九价HPV疫苗先后在中国上市，2019年，国产替代正式开始，算起来，HPV疫苗在中国仅有8年历程。

但这8年，竞争可谓激烈。

不仅彼此产品上市时间咬得紧，入局者也众多。

如今，在国家药物临床试验登记与信息公示平台上，HPV疫苗相关登记多达75条，除了热门的九价，也不乏十五价等更高价次的疫苗——尽管目前上市的HPV疫苗仅有5款，但赛道已肉眼可见地拥挤。

![](.evernote-content/E97B2E62-5480-4E56-990A-44F227C56AA8/C40C704D-6CCD-4D3F-8712-01B996109423.png)

国内现已上市HPV疫苗的价目表 / 图源：西南证券

价格战最先在国产二价疫苗中间开打，各地政府集采公告记录下了战况。

今年3月，万泰以86元/支的价格中标，二价HPV疫苗的价格首次降至百元以下。8月，沃森在山东中标，价格27.5元/支，降至“奶茶价”，这一度引发投资者不满——作为对比，万泰的自费定价为329元/支，沃森上市之初定价329元/支。

不同于国产疫苗依赖政府集采，进口疫苗更多面向自费市场，尤其默沙东的四价和九价暂无替代者，九价仍然坚挺在1298元/支的高位，市场主导地位加持，维持了高额利润。

面对市场变化，默沙东作出的一个调整，是继续锐减逐渐失去竞争力的四价，增加“人气王”九价的供应。但随着一众国产九价大力投入，留给默沙东九价的时间也不多了。

国产疫苗仍在一边追赶，一边寻求突围。只是突围路上，一边九价研发仍需重金投入，另一边双双争抢海外市场订单，万泰、沃森仍然难以避免狭路相逢。

未来，国产九价是以价格优势动摇默沙东九价的地位，还是事与愿违地卷入同质化竞争当中，谁将从中突围，拭目以待。
