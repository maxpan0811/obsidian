# 生产力 = 少点几下？来自微软的桌面启动器：Arrow Launcher

---

![](.evernote-content/36F11200-C4A0-4E44-9656-E702B35488FB/AFCE3553-3D4E-4295-BC97-1AF7D216677F.jpg)

微软这个在操作系统和软件服务上的巨头最近在 Android 平台上动作频频，陆陆续续发布了几款 Android 应用。而不得不承认的是，微软的这些 Android 应用质量都相当不错。其中的桌面应用 Arrow Launcher 更是可以称得上是 Android 平台上少有的精品。

主打智能和简单
-------

对于美的定义大概是各有各钟情的美，但是对于实用的定义却总是相似。借用一个产品经理的话：「所谓使用，就是让用户尽可能的少点几下」，这也成了我很久以来鉴定一个应用设计是不是足够简单的标准。Arrow 就是这样一个足够简单和智能的应用。它会自动对应用程序进行归类，将常用的应用排在屏幕的上方方便用户的使用。

![](.evernote-content/36F11200-C4A0-4E44-9656-E702B35488FB/AB5A17A4-6043-4364-8FE7-0DBFBFBC9CCC.jpg)

![](.evernote-content/36F11200-C4A0-4E44-9656-E702B35488FB/012FA343-BCDB-4C60-9B05-13FD90B32449.jpg)

即便是不常用的应用，Arrow Launcher 也提供了首字母排序的功能，点开应用抽屉即可快速找到应用。这也是笔者最喜欢 Arrow Launcher 的地方，随着你使用的越久，它就会变得越智能使用起来就越顺手。

它把别人的工作都干了
----------

Arrow Launcher 总共有五个屏幕，对于 Arrow Launcher 来讲没有增减屏幕的这个概念，不过你可以在「页面管理」中隐藏不需要的屏幕。这五个屏幕分别是应用屏（主屏），小部件、最近使用、联系人以及笔记和提醒。有了这五个屏幕，Arrow Launcher 成功的取代了 Google Keep 和联系人的功能。

笔记和提醒功能
-------

![](.evernote-content/36F11200-C4A0-4E44-9656-E702B35488FB/B0121D16-CF73-4243-9F2D-4FBDB56564AA.jpg)

「笔记和提醒」的功能较为简单，只有最基本的 To-Do List 和提醒功能，不支持分类和归档也不支持上传附件之类的功能。点击已经创建好的笔记即可选择设定提醒，到了时间 Arrow Launcher 就会弹出通知啦。

在此之前我用的是 Google Keep 因为它可以和 PC 、平板进行同步联动。但是 Keep 是一个便签应用，而 Arrow Launcher 的这个笔记提醒就更像是一个 To Do List 的功能。我可以把一天要做的事情写在上面，比如买牙刷之类的，比起 Google Keep 我更喜欢这种嵌入到桌面中的设计。它和小部件的感觉还不太一样，因为点击小部件依然会弹出应用才可以添加内容。

不过也正因为它只是桌面的一个附属功能，所以它只具备最简单的功能，目前还没法进行同步。笔者倒是很希望微软可以将这个「笔记和提醒」功能和 PC 上的 Cortana 进行联动，这样我就不需要手机电脑之间进行切换了。

不得不承认的是，微软的软件开发能力依然是全球数一数二的。就拿它开发的几款 Android应用来说，都是可以称得上是精品的范畴。而这些应用也无不打上了微软生产力至上的标签，这也是它们和其他同类应用最大的不同和优势。总而言之，微软这款 Arrow Launcher 还是做的不错的，虽然不能进行过多的自定义设置，甚至连增添屏幕的功能都没有，但也正是因为这样的设计才让我不会把打把的时间给浪费在收拾桌面上。这也是 Arrow Launcher 的一大特点吧，打开即可使用。

你可以在 [Google Play](https://play.google.com/store/apps/details?id=com.microsoft.launcher) 下载 Arrow 桌面。
