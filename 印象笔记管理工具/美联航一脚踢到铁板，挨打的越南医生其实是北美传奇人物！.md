# 美联航一脚踢到铁板，挨打的越南医生其实是北美传奇人物！

---

挨打的美籍越南裔医生真是当年从南越西贡去美国的。他在约翰·霍普金斯拿到行医执照，妻子和五个孩子都是医生，其中两个孩子在全美最好的医疗机构工作。代他打官司的两个律师都是怼大公司胜率在全美排前5%的牛逼人物。其中一个本就是他的私人律师。请得起这种律师做私人律师，本身也是姓赵的之一了。现在律师表示索赔十个亿……美联航以为不过打一个黄皮猴子而已，没想到人家是美国姓赵的。

每日邮报网站指出，陶大卫是五个孩子的父亲，也已经升格为祖父，他是一名内科医生，而他的五个孩子中的四个也是医生。

陶大卫住在肯塔基州伊丽莎白镇，是一名肺部肿瘤专科医生。

这个被united打伤的越南医生Dr GAO 可不是一般人啊！简直是个北美传奇人物！一个逃难的越南船民，到美国考执照行医二十年，生五个孩子，培养出4个医生，行医生涯中被举报吸毒后性侵男病人，医生说是被冤枉！但是官司打赢了无罪释放，但是他的行医执照还是被吊销，10年后67岁又重新考下医照行医，他尽然还是美国一等一的扑克高手，八次世界联赛拿到一百九十多奖金 据说单在拉斯维加斯的赌场就赢了七百多万。美联航觉得华人都是胆小怕事好欺负 ，看到乘客名单的last name 只要像华裔的名字 座位的random 挑选程序就自动flag！结果这次美联航欺负亚裔不成，结果自己搬起石头砸自己脚！现在就等着打人的机场警察被刑事起诉和美联航对受害人的民事巨额赔偿金吧！

那个被拖的越南大夫家里五个孩子四个医生连老婆也是儿科医生。完了，现在看你美联航怎么洗白都没用了。为了让座打了个有预约病人的医生，如果病人急病在这段时间死了美联航就可以彻底申请破产了 ​

看到医生的家庭背景，我感慨几句；首先，人多力量大，这话虽然朴素，但真是将斗争哲学用最深入浅出的方式表达了出来。小到医生家庭的五个孩子，大到一个多元社会中某个族裔的人口数量，无不都是人多力量大。其次，打铁一定要自身硬。陶医生一家所有人，包括夫妻俩，以及几个孩子都是医生，俩孩子还在Mayo，其经济能力决定了在面对这种不公正时，可以调动相当的资源为自己斗争。六七个医生，保守估计年总收入在两百万美元以上，他们家这样的情况，只会更高。苍天有眼，让头脑里充满了种族偏见的UA遇上了社会中顶尖的0.1%的家庭。

不是很懂 他们家里人是医生不是美国人都已经知道了呢 还说爆出了人家所谓的黑料 这个人不简单在哪里？和一个很厉害的律师是故交？

国外的医生都是高社会地位的精英阶层，一家子七口全是医生可以说牛逼大发了，而且每个都不是普通公立医院混日子的，人家还是能在飞机上【立刻】打电话给律师，而他的律师居然也【立刻】回接电话。在国外医学院是最顶尖的人士才能进的，更不要提学费都是一年几百万人民币，这还只是本地人的价钱

一家子医生本来就很牛逼 美国人是很尊敬医生这类职业的和中国不同 有犯罪记录还有这么屌的私人律师 就更牛逼（并不是什么好事 只是说明踢到铁板） 这种案子本来就有律师抢着接因为抽成天价 现在整个律师事务所都是牛逼认识 ua绝对要大出血了 即使普通律师来打赔偿也不会轻
