# 特斯拉 Model 3 正式发布：这会是年轻人的第一辆电动车吗？

---

[![](.evernote-content/B350D965-9585-4E32-9B92-2844C2B57771/F3CCB3D3-14CC-44B7-8D14-19D8DE228B1E.png)](http://images.ifanr.cn/wp-content/uploads/2016/04/%E5%B1%8F%E5%B9%95%E5%BF%AB%E7%85%A7-2016-04-01-%E4%B8%8B%E5%8D%881.57.50.png)

> 这将是一辆廉价电动车。  
> 这将是一个缩小版本的 Model S。  
> Model 3 将会保持 Model S 的高性能。

这些传闻终于在今天中午尘埃落定，特斯拉召开发布会，正式发布了新款电动车 Tesla Model 3。售价为 3.5 万美元（约合人民币 22.77 万元）。充一次电可以行驶 215 英里（约合 344 公里）。

[![](.evernote-content/B350D965-9585-4E32-9B92-2844C2B57771/A86B3FB5-2DB2-48BD-A94B-A1476FABDADB.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/04/scenic_mountain_1.0.jpg)

作为特斯拉历史上最便宜的一款车，3.5 万美元的售价比起之前的 Model S 更加亲民，便宜的价格也引起了一波特斯拉拥趸的预定潮，中国地区的 16 家特斯拉体验店在昨天已经开始接受预定。从**爱范儿**现在得到的消息来看，在 Model 3 开启预定后的 24 小时内，已经有 15 万辆预定。

[![](.evernote-content/B350D965-9585-4E32-9B92-2844C2B57771/C9143D62-5C55-4E31-9869-A8B5E17CB445.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/04/tesla-model-3-first-ride-028-1020.0.jpg)

[![](.evernote-content/B350D965-9585-4E32-9B92-2844C2B57771/8D2A68BD-7E91-46CA-813B-92779CF8692B.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/04/red_1.jpg)

外观上，特斯拉 Model 3 尺寸比 Model S 小 20%，尺寸与奥迪 A4 相当。配备全景天窗，车门把手采用了和 Model 类似的可伸缩设计。前脸采用无中网格栅设计，这种独特的造型不仅提高了用户对 Model 3 的新鲜感，也令新车的前脸整体性更强。

[![](.evernote-content/B350D965-9585-4E32-9B92-2844C2B57771/474181F0-8186-408F-A460-3ECE2D2C2755.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/04/tesla-model-3-first-ride-010-1020.0.jpg)

车的内饰与 Model S 差别较大，中控采用了一块伸出式横向屏幕，尺寸未知但看起来不小。内饰风格比 Model S 显得更加简洁。0-100Km/h 加速为 6 秒，虽然比不上 Model S 3.3 秒的速度，但仍领先同价位车型不少。

[![](.evernote-content/B350D965-9585-4E32-9B92-2844C2B57771/F88F9C46-1D7F-497F-A878-055651D45B1D.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/04/teslamodel3interior-marked.0.0.jpg)

关于这款车的入华后售价，在发布会中没有透露，但特斯拉在中国定价时往往采取和美国市场同价的策略，我们可以拿 Model S 在中国的售价做一个参考：

> Model S 85 在华售价的计算公式是这样的：81070 美元(在美国的价格)+3600 美元(运输与装卸)+19000 美元(关税和其他税)+17700 美元(增值税)=121370 美元=734000 元人民币(汇率为 6.05 元/美元)。经测算，在华的关税、增值税及其他税共计 36700 美元，占售价的 30%。

这样来看特斯拉 Model 3 入华后的售价很有可能这样计算：

> 新车海外起售价为 35000 美元，约合 22.6 万元人民币，加上 3600 美元（约 23266 元人民币）运输装载费、近30% 的关税+增值税 15671 美元（约 101283 元人民币）左右。

这样算来 Model 3 的起售价将会在 35 万元人民币左右。

### 中级豪华车的克星？

[![](.evernote-content/B350D965-9585-4E32-9B92-2844C2B57771/0788417D-49C4-4254-847E-2D8282222851.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/04/bmw-3er-series-3.jpg)

从特斯拉 Model 3 3.5 万美元的定价来看，此次特斯拉的竞争对手是市场上宝马 3 系、奔驰 C 级、奥迪 A4L 这样的中级豪华车。比起这些传统中端豪华车，特斯拉拥有更年轻前卫的外观，更好的汽车性能，可能会更受年轻人的青睐。

### 更高的产能，让电动车也走进寻常人家

[![](.evernote-content/B350D965-9585-4E32-9B92-2844C2B57771/3C1FC789-1E06-4333-B7D0-2F99A99A519A.png)](http://images.ifanr.cn/wp-content/uploads/2016/04/%E5%B1%8F%E5%B9%95%E5%BF%AB%E7%85%A7-2016-04-01-%E4%B8%8A%E5%8D%8811.52.47.png)

在发布会中特斯拉 CEO 马斯克宣布，Model 3 将陆续投入生产，同时斥资 50 亿美元的超级电池工厂 Giga Factory 的陆续投产，整车最核心的电池部件的成产成本将下降约 30%。而此前特斯拉中国负责人朱晓彤在接受媒体采访时就确认，为了尽快实现特斯拉国产化，管理层正在进行中国工厂选址的考察与调研。

### 写在最后：中国区购车指南

如果你考虑购买一辆售价为三四十万的中级豪华车，特斯拉 Model 3 将会是一个极佳的选择。但**爱范儿**建议你最好能满足以下几条：

> 1、自己有固定停车位（特斯拉将会为你的停车位安装目的地充电站，保证你每天回家后可以方便充电。）
>
> 2、有足够的耐心（虽然今天已经发布，但 Model 3 要到 2017 年底才会正式发货，并且将会遵循北美、欧洲、亚洲的顺序。）
>
> 3、最好已经有一辆燃油车（特斯拉可以作为一款卓越的城市车，以现在的充电桩数量，特斯拉比较难应付长途行驶。）

并且如果你生活在北京，在售的 Model S 已经进了新能源汽车目录，可以参与新能源摇号购车，并且不受限行规定。Model 3 在入华之后不出意外也会加入其中。但作为进口车型，Model 3 将不会享受政府和地方的新能源汽车补贴。不过中国区恐怕要等到 2018 年才能提车，到时政策如何还难以估计。
