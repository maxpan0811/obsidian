**2017**年元旦假期刚过，周末又要来啦！

有在家带娃没有假期的父母吗？

有想过躲到一个没有“熊孩子”的桃花岛上度个假的父母吗？

有想过，终于熬过的漫长的工作日，想好好睡一觉的父母？

 

所有的答案，都是NO。**我要带娃！我要带娃！**

![](.evernote-content/3E944771-BCAE-4301-BCB3-441532300294/33FAF0DE-56DD-4A8B-ACD2-99E6222ECF21.jpg)

好吧，就让我们苦中做乐，看看全世界和我们一样的有“小私心”，想偷懒的父母是怎么做的。每个人都有自己的育儿方法，我们相信每一位父母都在尽自己最大努力让孩子快乐成长。有时候我们会觉得觉得陪娃玩真心有些累，想想要是有什么轻松带娃的秘诀就好了。

**“左手萌娃，右手度假。”**让我们试一试，如何二者兼顾。

**轻松陪娃第一招：**

**迷之《迷宫大侦探皮埃尔》**

用最耗时的迷宫，来打败精力充沛的“熊孩子”。

这本书由世界知名的日本设计工作室**IC4DESIGN**创作，适合一家人一起玩儿。这是一本视觉发现益智迷宫书，精美的迷宫书里包含了**15个**错综复杂的迷宫游戏，**360项**寻找任务，还有无数好玩有好笑的细节，无数外文单词边玩边学！

![](.evernote-content/3E944771-BCAE-4301-BCB3-441532300294/F549B55B-E999-4472-9C95-6267553118DC.jpg)

**这本书的剧情是这样子****的****：**在戏剧城有一块可以把任何地方都变成迷宫的神秘迷宫石，本来这块石头是被锁起来的，但是现在它被神秘大盗X给偷走了，所以迷宫大侦探皮埃尔要把它找回来！整本书需要读者去走迷宫抓捕神秘大盗X，以及寻找各种小玩意儿。

书中最简单的任务就是通过各个迷宫场景，**找，找，找！**

还有你自己的超能力……

画出超级英雄们飞来飞去，和超级坏蛋们打来打去的场景

**《超级英雄的超级手册》**所有绘画步骤都罗列出来，教娃画出终极战士和他们的超级力量、他们的朋友、他们的秘密藏身处、他们的装备以及超级装置（甚至还有他们的强大对手！），可以让小朋友创造出自己的超级英雄宇宙。书中介绍了各种各样的绘画方法、线条的创作以及活动。

书中还有2页超能力贴纸、2页面具和1个立体超级英雄哦，就等你来收集了，看看你有没有能力成为一名超级英雄吧……

**这本书你可以跟娃这么玩**

**·深入浅出的绘画教学**

**![](.evernote-content/3E944771-BCAE-4301-BCB3-441532300294/544398A8-AD0B-46DD-93D9-91F35177B7E4.jpg)**

```
本书作者将复杂东西简单化，手把手教孩子画出超人以及他们的超级世界。这听起来似乎很难，但是在一页页跟随作者的方法动笔作画之后，会发现其实非常简单。作者将每一个步骤都按顺序展示出来，让读者一目了然。完全不会因为深奥的理论打击掉孩子画画的热情。

```
·激发孩子想象力，给图画填上翅膀
