**/** 1、截图软件：Snipaste **/**

Windows 系统有自带的截图工具，但不方便，无论是截图方式，还是标注，也十分的简陋和粗糙。

Snipaste 是一个简单但强大的截图工具，也可以让你将截图贴回到屏幕上！下载并打开 Snipaste，按下 F1 来开始截图，再按 F3，截图就在桌面置顶显示了。就这么简单！

Snipaste 作为一款强大的截图软件，具有丰富的功能和便捷的操作方式。无论是截图功能、标注功能还是贴图功能，都能满足用户在不同场景下的需求。其官网下载速度慢的问题也可以通过多种方法解决。用户可以根据自己的需求和习惯，灵活设置快捷键，充分发挥 Snipaste 的优势，提高工作和学习效率。

如果要进阶版使用，官网中就附带了详细的文档说明，点赞加鸡腿，最后三个平台（Windows、macOS、Linux）均可使用，再加一分！！！

**/** 2、搜索软件：Everything **/**

Everything 是一款由 voidtools 开发的文件搜索工具，主要运行于 Windows 系统。它以其惊人的速度和小巧的体积而备受用户青睐。

Everything 为用户在海量文件中快速定位所需内容提供了便利。同时，通过设置快捷键和排除文件夹搜索等功能，用户可以更加个性化地定制搜索体验，提高工作效率。无论是在日常办公还是个人使用中，Everything 都展现出了卓越的性能和实用性。

![](.evernote-content/AF704C44-310C-423D-A771-B99B5311E90E/1BD49136-39D5-4939-B57E-0D16BB575515.png)

官网地址：https://www.voidtools.com/zh-cn/ 

**/** 3、手势软件：WGestures 2 **/**

WGestures 是一款强大的全局鼠标手势软件，它支持 macOS 与 Windows 平台，具有诸多实用功能。首先，它可以通过绘制不同的图形来触发不同的操作，比如打开网页、关闭窗口、切换程序、搜索文字等，减少了使用键盘和鼠标的次数和距离，极大地提高了工作和娱乐的效率和质量。在日常办公中，用户可以用鼠标代替键盘的一系列快捷键，如复制、剪切、粘贴、前进、后退等基本操作只需简单滑动鼠标即可完成，非常方便快捷。

WGestures 作为一款全局鼠标手势软件，功能强大、定制化程度高、用户界面简洁、对游戏优化效果显著。它为用户提供了更加高效、便捷的操作方式，无论是在日常办公还是游戏娱乐中，都能发挥重要作用。

**/** 4、视频播放：PotPlayer **/**

PotPlayer 作为一款功能强大的多媒体播放器，在视频格式支持、播放设置、音频播放等方面都表现出色。它以其强大的解码能力、丰富的功能和自定义选项，成为了许多用户喜爱的媒体播放器。无论是休闲娱乐还是专业应用，PotPlayer 都能够提供卓越的视听体验。

PotPlayer 几乎可以播放所有主流的音频和视频文件格式。它支持常见的视频格式如 MP4、AVI、MKV 等，对于不太常见的视频格式如 RMVB、TP、TS、FLV 等也能轻松播放。同时，PotPlayer 支持 H.265、VP9 等高清编码格式，确保用户能够播放各种高清视频。

![](.evernote-content/AF704C44-310C-423D-A771-B99B5311E90E/5AC25DB6-1038-40CF-801C-4BBBD90CAAB2.png)

官网地址：https://potplayer.org/ 

**/** 5、解压缩软件：Bandzip **/**

Bandizip 是一款免费压缩文件管理工具。它支持多种压缩格式，包括 ZIP、RAR、7Z 等等，并且具有快速压缩和解压缩速度、稳定性高、用户界面友好等特点。

除了免费版外，Bandizip 还提供了专业版，其中包含了一些额外的高级功能。

当然，对于我们普通用户来说，免费版已经够用了，没有必要搞个专业版。
