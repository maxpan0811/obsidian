入秋：我看叶落的方式丨周末读诗
===============

原创 三书 新京报书评周刊

收录于合集#周末读诗120个

![](.evernote-content/3A9A0D92-D2A3-4C00-B280-37688E9EE79E/6D71DF9A-027E-4988-A18B-61599639447E.gif)

如果我没有为 

有些光秃的枝丫 

转身， 

我将错过 

繁茂的景象 

夏叶缓慢飘落 

满地金黄。 

我不曾拥有 

美丽的事物，感受 

可爱的事物，沉静 

而迷人，心灵麻木。 

更经常地回头看， 如果你想救自己。 

那些从不停步，一往直前，

从不四处顾盼的人，没有看到这一切。

 

*——罗伯特·瓦尔泽《我看叶落的方式》*

*（改自江鑫鑫译文）*

**撰文 | 三书**

人世虽然动荡，但始终有很多大树在静静地守护我们。

![](.evernote-content/3A9A0D92-D2A3-4C00-B280-37688E9EE79E/D80A6CD5-5B64-44B1-9CDF-366469540243.png)

**一叶落，褰朱箔**

/ /

**《一叶落》**

**（后唐）李存勖**

**一叶落，褰朱箔，**

**此时景物正萧索。**

**画楼月影寒，西风吹罗幕。**

**吹罗幕，往事思量着。**

/ /

一叶落，你听见了么？听见叶落，诗人忽然心惊。一片花飞减却春，一叶落知天下秋，谁看见花飞，谁听见叶落，谁就是那个诗人。

 

立秋前，这里的马栗树叶已经黄落，高大的山毛榉仍碧绿，然而树下不知何时也积了好些落叶，其他如槐树、杨树、椴树，也都显出疲惫的样子。

 

“一叶落”，词中说的是梧桐，也是北方的树，挺拔俊茂的大树。这坠地的一声，“啪”“啊”“嗯”，或惊讶，或叹息，像一个音符，奏响秋日的序曲。立秋之后，景物貌似依然，其实已经全变了，灌木萧索，风吹在树上，瑟瑟清响，月光更加皎洁，变得凄冷。

 

词中人听见一叶落，“褰朱箔”，掀起朱红的帘子，走了出去。所见者何？此时景物正萧索，虽然到年底还有三四个月，但你知道，一年盛事已成昨。风改变了方向，阳光白净但有些凉，天空高远，云朵迁徙，树木拆除它们的帐篷，韶华落幕静静散场。

 

当你看见萧索时，一切都远了。你站在空旷的现场，或倚在画楼上，呆望着月亮。“画楼月影寒，西风吹罗幕”，月影有些寒冷，西风吹着，吹着所有事物，远去的，残留的，对于词中人，最关情的是罗幕。

 

“吹罗幕，往事思量着”，罗幕开合，承载着闺中的私密记忆，此时人去楼空，西风吹拂，罗幕低垂，往事思量着。结句用赋体，回味更深，怨耶？情耶？我们也思量着。

![](.evernote-content/3A9A0D92-D2A3-4C00-B280-37688E9EE79E/0EB6286D-4C05-4843-A4DC-F1E47DD674F5.png)

南宋 佚名《胆瓶秋卉图》

 

此词作者是后唐庄宗皇帝李存勖，其为人极有趣，据说这是他的自度曲。存勖小名亚子，原为突厥沙陀族人，赐姓李，为晋王李克用之长子，胆略绝人，骁勇善战，北退契丹，东灭燕，又灭梁，公元923年称帝。李存勖虽武人，但洞晓音律，能自度曲，其词多作于公子时期，而后用兵，前后队伍皆以所撰词授之，使兵士揭声而唱，入阵不论胜负，马头才转，则众歌齐作，故凡所战斗，人忘其死，亦用兵之一奇。可叹庄宗在位仅四年，竟为伶人所杀。

 

初读《一叶落》，词句婉丽，颇觉书卷气，及知作者其人，不觉激赏。吾素爱书卷气中透出草莽气，见草莽气中有书卷气者，更可惊喜！

![](.evernote-content/3A9A0D92-D2A3-4C00-B280-37688E9EE79E/B9474849-BD01-4849-9EA3-23978D570A36.png)北宋 赵佶《池塘秋晚图》

![](.evernote-content/3A9A0D92-D2A3-4C00-B280-37688E9EE79E/7FC771FB-7A48-4B68-BF89-72C07DA4FBB2.png)

**秋风又到人间**

/ /

**《相见欢》**

**（宋）朱敦儒**

**秋风又到人间，叶珊珊。**

**四望烟波无尽、欠青山。**

**浮生事，长江水，几时闲。**

**幸是古来如此、且开颜。**

/ /

《相见欢》，一名《乌夜啼》，唐教坊曲名。南唐后主李煜作两首，均题“乌夜啼”，因其词美，故调名又缘词而称“上西楼”“秋夜月”。据说此调声韵极富变化，音节响亮，似与名相合，五代薛昭蕴与冯延巳词写闺情，语意轻快，后主词则声情悲痛，继之者亦多写哀愁，慢吟轻咏。

 

词牌名相见欢，写得却是愁，也许正因当初欢乐，才有而后的哀愁。反之亦然，正因有聚必有散，有生必有死，故应相见欢。汉乐府相和歌辞瑟调曲《善哉行》曰：“来日大难，口燥唇干。今日相乐，皆当喜欢。”百年修得同船渡，今日相乐，能不喜欢？

 

两宋之际词人朱敦儒，字希真，人称洛川先生，早年隐居不仕，屡辞荐辟，南宋初避乱南雄州（今属广东韶关），后应召至临安，赐进士出身，累迁两浙东路提点刑狱，终因秦桧事坐废，著有词集《樵歌》。

 

人的一生，寸阴何其长，百年何其短，词家朱希真，今人记载不过几十个字，在他生年却长怀千岁忧。倒是“樵歌”之名，比起现实的人生剧情，更觉简净。

 

秋风又到人间，第无数次了，却总是那样的秋风，人间也总是这样的人间。就连这些叶子，珊珊摇落，其实也没有什么，轮回而已，很快还会再来。这么一说，忽焉释怀，秋风也就是春风，无情何尝不是有情，看电影如读佛经，读佛经何尝不是在看电影。

 

朱希真终归回到人间，“四望烟波无尽、欠青山”。烟波无尽，何不小舟从此逝，江海寄余生？青山，那道天空的分界线，绵延舒展，若有情，若无情。

 

“浮生事，长江水，几时闲”，浮生若梦，此处比作江水，长流不息。人身可比一叶扁舟，行于水上，载沉载浮，是水负载着船行走，抑或人行走，负载着船和水？前水复后水，古今相续流，新人非旧人，年年江上游。

 

“幸是古来如此、且开颜”，这句大概在感时伤世，改朝换代，古来如此，就是帝王将相也不过你方唱罢我登场。不过，朱希真的语气有点勉强，“且开颜”的意思，就是难开颜。“幸是”一词，可参商榷，古来如此，有何可幸？古来如此，就不悲哀了吗？

沈周《泛舟图》

![](.evernote-content/3A9A0D92-D2A3-4C00-B280-37688E9EE79E/5C461653-95B9-475A-9D77-D2E5E0D238D2.png)

**众芳芜秽，美人迟暮**

/ /

**《山花子》**

**（南唐中主）李璟**

**菡萏香销翠叶残，西风愁起绿波间。**

**还与韶光共憔悴，不堪看。**

**细雨梦回鸡塞远，小楼吹彻玉笙寒。**

**多少泪珠何限恨，倚阑干。**

/ /

绝妙好诗、好词，无不气象混成，难以句摘，而诗话词话，却爱以句摘，南唐中主这首《山花子》也不例外。中主曾开玩笑地问冯延巳：“风乍起，吹皱一池春水，干卿何事？”延巳曰：“未若陛下‘细雨梦回鸡塞远，小楼吹彻玉笙寒’，不可使闻于邻国。”此则故事人多传之，足以佐证“细雨”二句之妙。宋代王安石亦大为赞赏，认为这是南唐最好的词句。喜欢“金句”的现代读者，不知以为然否？

 

王国维先生颇不屑、颇寂寞地说，明明“菡萏香销翠叶残，西风愁起碧波间”，大有众芳芜秽、美人迟暮之感，古今却独傲“细雨”二句，真是解人难得啊。看来岂只诗人孤独，诗更孤独！

 

我站在静安先生这边，细雨梦回二句，意境清幽，然未若菡萏西风二句，入情入骨。“还与韶光共憔悴，不堪看”，更是沉郁之至，凄然欲绝，世间没有比美人迟暮更难堪的了。

 

英国诗人泰德·休斯的《七愁》，历数了秋天的花园、雉鸡、太阳、池塘、树林和狐狸六种哀愁之后，第七愁便是朱颜的缓慢告别，长满皱纹的脸透过窗户向外眺望，世界像一个粗俗的马戏场，年岁正打点着行装。

 

“多少泪珠何限恨，倚阑干”，这个姿态还算年轻，尚有泪可流，还能悲伤，不像住在我对面公寓里的老妇，每天盛装立于窗前眺望。有什么好看的呢，都看了几十年了，一条什么也不会发生的街，然而她还是看：垃圾车收垃圾，亚马逊送快递，过路小孩或笑或哭，乌鸦在树上叫两声，任何风吹草动，她都跂而望之。一望就是半天，一望又是一年，反正她有的是时间。
