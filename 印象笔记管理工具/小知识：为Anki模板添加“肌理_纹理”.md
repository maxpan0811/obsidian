制作说明
----

还是以 “纸质样式” 为基础，基本的样式没有变，变得只是背景的颜色、图案、字体。

前面已经尝试过[渐变色](https://zhuanlan.zhihu.com/p/69339413)和[花卉平铺](https://zhuanlan.zhihu.com/p/72802281)，而这次带来的是 “**肌理 / 纹理**” 的变化。

肌理素材来源：[TRANSPARENT TEXTURES](https://link.zhihu.com/?target=https%3A//www.transparenttextures.com/)

改变背景颜色代码：

```
background:white;
```

为卡片背景添加图片（比如，纹理）：

```
background-image:url(_shley-tree-2.png );
```

修改模板过程中有什么问题，私信我，或者加我微信。
