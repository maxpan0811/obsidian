上手体验 iPhone X 之后，这些方面让我们印象深刻
============================

距离 iPhone X 正式发售还有 2 天时间，作为今年的绝对主角，使用全新设计的 iPhone X 给用户带来的体验到底如何，一直是大家讨论的焦点。

昨天下午，少数派受邀体验了 iPhone X。在详细评测之前，我们先聊聊在现场体验 iPhone X 时给我们留下深刻印象的几方面。

5.8 英寸的屏幕，4.7 英寸的握持感
--------------------

iPhone Plus 款的 5.5 英寸屏幕阅读体验更好，但从握持舒适度方面来说，不少人还是更喜欢 4.7 英寸 iPhone，它的宽度让你拿在手中感觉更「省力」，单手操作时也能覆盖到更多屏幕宽度。

得益于「全面屏」的设计，iPhone X 足以在将一块 5.8 英寸屏幕塞进和 4.7 英寸 iPhone 宽度差不多的机身内，机身重量也比 iPhone 8 Plus 轻了近 30g。因此 iPhone X 拿在手里的整体感受无论是宽度还是重量，都明显称手很多。

![](.evernote-content/6688D964-64C6-46F9-902C-FB721B139301/16C230D1-EA13-468C-8D41-28F028291561.jpg)

iPhone X 正面统一为黑色，背面则提供了深空灰和白色两种，苹果工作人员告诉我们 iPhone X 背板颜色和 iPhone 8 / 8 Plus 的白灰两色略有不同，但这个差别肉眼很难分辨。

![](.evernote-content/6688D964-64C6-46F9-902C-FB721B139301/FF11F71E-1E61-4772-AE2F-7812CFFEAF4D.jpg)

相比之下，白色版 iPhone X 的不锈钢材质边框比确实比采用铝合金边框的 iPhone 8 更加亮眼，手感也更加圆润。

![](.evernote-content/6688D964-64C6-46F9-902C-FB721B139301/39006E25-AF3E-4F84-B17E-145402BE22E9.jpg)

（单纯从数字对比，5.8 英寸 iPhone X 给人的第一印象似乎应该比 5.5 英寸的 iPhone 8 Plus 屏幕更大，但是 [之前我们曾经分析过](https://sspai.com/post/40893)，iPhone X 的屏幕面积实际是要比 iPhone 8 Plus 小一些的。）

新的手势操作需要适应一下
------------

去掉了标志性的 Home 键，iOS 中很多依赖它才能完成的操作都被重新设计，任务切换功能交给了屏幕下方的横条（Home Indicator）。而激活 Apple Pay、唤醒 Siri、屏幕截图等功能则分配给了机身两侧的电源键和音量键。

![](.evernote-content/6688D964-64C6-46F9-902C-FB721B139301/8EBFA57D-B2AA-447C-AD64-44DE70A45E25.gif)

在应用内，从屏幕底部向上轻扫，即可返回主屏幕。

![](.evernote-content/6688D964-64C6-46F9-902C-FB721B139301/B0E4E17A-2DB7-489B-B00C-0023CAB1F547.gif)

从底部向上轻扫并停顿一下，就能激活多任务界面。

![](.evernote-content/6688D964-64C6-46F9-902C-FB721B139301/BDD1CC60-756F-46AA-9C7F-60DA8EFF1D7D.gif)

iPhone X 还有类似 macOS 的滑动切换应用窗口功能，按住 Home Indicator 并左右滑动，可以在当前应用和上一个应用之间来回切换。

![](.evernote-content/6688D964-64C6-46F9-902C-FB721B139301/5C944150-8005-4C1E-96DD-693AB7BEDEDE.gif)

以往轻触两下 Home 键，屏幕内容会整体向下移动，便于单手操作，这个功能在 iPhone X 中是默认关闭的，需要在系统设置中打开，之后在屏幕底部向下滑动即可把屏幕顶部的内容「拉」下来。

长按 Home 键唤醒 Siri 的操作交给了改名为「侧边按键」的电源键。

![](.evernote-content/6688D964-64C6-46F9-902C-FB721B139301/57FC1D02-A5AB-4258-ACD7-2CD89C572E03.gif)

双击侧边按键，也可以实现以往双击 Home 键激活的 Apple Pay 功能。

![](.evernote-content/6688D964-64C6-46F9-902C-FB721B139301/B4151BCA-4375-4889-9136-3454C6BD4787.gif)

在 iPhone X 上截屏，需要同时按下侧边按钮和音量上键。

![]()

唤出控制中心的方法改成从屏幕右上角向下轻扫，这是现场体验时最容易弄错的功能，好几次想要打开控制中心时都习惯性的从屏幕下面往上滑动，结果发现回到了主屏幕。

![]()

由于控制中心属于高频使用的功能，不少人刚刚用上 iPhone X 时估计都要回几次主屏才能习惯。

Face ID 体验还不错，和钱有关的操作只看一眼是不够的
-----------------------------

取代 Touch ID 的 Face ID 也是最被关注的功能之一，发布会后大家对它的识别速度和准确度都有很多疑问，也衍生出了各种「看一眼就付款」的段子。

现场体验过程中，Face ID 给人的印象还是不错的。解锁时几乎不需要刻意注视手机，在举起手机的同时就可以完成。为了能看清 Face ID 解锁全过程，我们拍摄了慢动作视频，从拿起手机到解锁再到进入主屏，实际时间只有不到 4 秒。

此外苹果还特意将 iPhone X 的锁屏通知设为解锁后可见，在未解锁时，你只能看到发送通知的 App 名，不能查看具体内容，需要解锁后才会显示，这其实是 iOS 11 的新功能，但是在其它 iPhone 上并没有默认开启，这个功能搭配 iPhone X 效果更佳，实现了「这条信息只有机主才能看得到」的效果。

现场体验时苹果向我们演示了 Face ID 的使用场景，除解锁屏幕外，系统中所有以往用到 Touch ID 的地方现在都能用 Face ID 完成，而以往调用 Touch ID 的手机银行、支付宝等应用，适配后也都支持 Face ID 解锁。

至于「看一眼就付款」的问题，苹果的解决方法也很简单，所有包含金钱交易的流程，除了 Face ID 认证，还需要按一下侧边按键才能最终完成支付。

![]()

前置摄像头的更多用法：Animoji 和人像模式
------------------------

iPhone X 在「刘海区」安装的各类传感器是 Face ID 高安全性的保证，利用原深感摄像头记录的面部信息，苹果也开发出更多用法，例如令人印象深刻的 Animoji，以及前置摄像头的「人像模式」。

![]()

现场分别用 iPhone X 的前后摄像头拍摄了「人像模式」，后置摄像头成像效果和 iPhone 8 Plus 基本一致，而前置摄像头的效果由于硬件上的限制，亮度明显不如后置，如果仔细看人脸边缘，也会发现由于识别不清而带来的边缘模糊问题。

![左 / 前置摄像头，右 / 后置摄像头]()

由于时间和场地限制，关于 iPhone X 的更多细节我们无法在现场进行体验和测试，对于大家比较关心的 Face ID 在日常生活中的实际体验，iPhone X 的拍照能力，以及各类应用的适配情况，这些我们都将在之后的深度体验文章中提到。
