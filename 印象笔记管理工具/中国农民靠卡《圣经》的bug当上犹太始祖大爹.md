为何西方传教士对洪秀全恨得牙痒​​？

一起走进今天的故事。

1850年代的南京城内，西方传教士正面临一场信仰崩塌。

他们可以承认太平天国士兵是“迷途的基督羔羊”，甚至称赞忠王李秀成“有基督徒的虔诚”，唯独对洪秀全的“上帝次子”身份如鲠在喉。

但他们再不承认也没办法，因为天王洪秀全已经在《圣经》里找到一个西方人争论千年的漏洞，还让自己成为了漏洞本身的补丁！

**壹**

**撒冷王麦基洗德的千古悬案​​**

洪秀全的神学杀手锏，藏在《创世纪》第14章一位神秘人物身上：

​​“撒冷王麦基洗德带着饼和酒出来，他是至高神的祭司。他为亚伯兰祝福…”（创14:18）​​

这个人就是麦基洗德。他有一个称号叫撒冷王。没错，他就是耶路撒冷最早的王。

![](.evernote-content/3C305F52-A366-468C-B88D-FC1AA06F6BAF/CD4A4D2D-5215-473C-9E5C-08520467FCE0.jpg)

撒冷王这个词翻出来就是太平王／平安王的意思，好了，在这里大家应该提前可以意识到啥了。

毕竟，咱洪秀全搞得就是太平天国！

​​撒冷王的虽然出场不多却有着极其重要的人设。

亚伯拉罕（犹太人和阿拉伯人共同祖先）将战利品的​​十分之一献给他​​——此为“十一奉献”的起源。

他赐福亚伯拉罕获得“应许之地”（迦南，即今耶路撒冷周边）的法理授权。

如果没有麦基洗德，以色列人的在联合国所谓的地契根本无效！​​

​​《希伯来书》的对麦基洗德描述是：​​

“他无父，无母，无族谱，无生之始，无命之终，与神之子相似。”（来7:3）

这个被新约认证“近神”的存在，却留下人类无法解释的终极难题：​​他究竟是谁？​​

**贰**

**为何千年来无人能解解释麦基洗德是谁？**

西方针对麦基洗德的身份主要有以下几种猜测：

|  |  |  |
| --- | --- | --- |
| 理论类型 | 核心观点 | 被推翻关键 |
| 天使论 | 麦基洗德是天使 | 亚伯拉罕对天使从不下跪献祭 |
| 耶稣马甲论 | 他是耶稣化身 | “与神之子相似”证明二者是平行关系 |
| 闪族论 | 诺亚之子闪的后代 | 明确“无族谱”打脸完整闪家族谱 |

从上面这张表格可以看出，西方的三种观点在对应《希伯来书》中的观点时都有硬伤。

所有的观点在解释这个bug的时候都有硬伤和漏洞。

但此人的身份又对耶路撒冷的所有权有着决定性的作用。

**叁**

**洪秀全把自己变成人形补丁包**

1853年，洪秀全在批注《创世记》时悍然宣布：

​​“麦基洗德正是朕穿越时空的化身！”​​

这一补丁刚好完美适用于《希伯来书》中所有的论述。

​​对其叙述的“三无身份”​​

“无父无母无族谱？” → ​​“朕从十九世纪穿越到迦南，对古人而言就是天降之人！”​​

​​对其叙述的“时间悖论”​​

“无生之始无命之终？” → ​​“朕乃上帝亲儿，和上帝一样是超越时间的存在，在没有世界之前就有了我。上帝是自有永有的，我也是。”​​

顺带还​​绑定了“撒冷王”王权​​

“撒冷”（Shalem）希伯来语意为​​平安/太平​​ → ​​“朕创太平天国，翻译成你们的语言就是撒冷。”​​

有了洪天王这一人形补丁，竟很好的填平了神学最深的裂缝！

**肆**

**耶路撒冷地契的原始授权人现身​​**

当洪秀全与麦基洗德画上等号，世界格局便瞬间变得无比奇幻。

​​亚伯拉罕​​向洪秀全献上十一奉献 → 洪是犹大先祖的​​直属上级​​

​​应许之地授权书​​由洪秀全签署 → 以色列建国文件形同废纸

天王在诏书中傲然宣称：

​​“西奈山摩西领十诫，何如朕在昆仑得天命？耶路撒冷金顶辉煌，终是朕太平王故都！”​​

翻译过来就是，这耶路撒冷只不过是我太平天国的旧都罢了，你小以侵占我太平天国的领土迟早得还回来！
