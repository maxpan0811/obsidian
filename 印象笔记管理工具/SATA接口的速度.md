# SATA接口的速度

---

```
SATA 1.0/1.0a (1.5Gb/s)      150MB/s  
SATA 2.0         (3.0Gb/s)      375MB/s  
SATA 3.0         (6.0Gb/s)      750MB/s  
  
以上数值均为理论值，实际应用中还要受到磁盘内部传输速率的影响，例如使用固态硬盘可以完全发挥SATA2.0接口的带宽，而使用传统硬盘将取决于硬盘内部传输速率，主要取决于单碟容量、转速和缓存大小！
```
