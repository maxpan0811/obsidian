### 核心特性

**笔记组织：**

* 支持无限层级的树状结构，方便归类整理
* 单个笔记可以放在多个位置，实现灵活的知识关联
* 强大的标签和属性系统，让笔记管理更加灵活

**编辑功能：**

* 所见即所得的富文本编辑器，支持表格和图片
* 内置 Markdown 自动格式化
* 支持数学公式和代码高亮显示
* 自带 Excalidraw 绘图工具，轻松创建图表

**知识管理：**

* 全文搜索功能，快速定位内容
* 知识图谱可视化，直观展示笔记关系
* 无缝的版本控制，不用担心内容丢失
* 支持笔记加密，保护隐私数据

### 安装指南

Trilium Notes 的安装非常简单。

支持 Windows、macOS、Linux、移动端多个平台：

1. 访问 GitHub Releases 发布页面；
2. 下载对应系统的安装包；
3. 解压后运行程序即可开始使用。

Linux 用户还可以通过 Flatpak 一键安装：

```
flatpak install flathub com.github.zadam.trilium
```
