# 韩复榘跑了，他却留下死战：全国武术冠军窦来庚，带三百弟子打鬼子，最后一颗子弹留给自己

Original史迹寻踪者 鉴史知兴替

抗日神剧看多了，大家总觉得那时候的武林高手就该飞檐走壁、刀枪不入，一个人端一营鬼子不在话下。可真翻档案，像窦来庚这样的全国武术冠军，最后是怎么没的？死在一场根本赢不了的硬仗里，举枪给自己补了最后一颗子弹。他这一生，把"练武的在抗战里到底能干什么"这件事，写得明明白白——人再能打，碰上枪炮，也就是块硬一点的肉。

![](.evernote-content/E4C1A3A6-C22A-46F9-B8FF-8FA4F206C4FB/19197818-D81A-4B19-826A-EF3959422FB1.jpg)

窦来庚是山东临朐窦家洼人，1900年生，字峰山。家里不算宽裕，小子倔，认准了武术这条路就往死里钻。走村串镇拜师傅，从民间拳师一路追到名门正派，最后拜进武当太乙门，成了高凤岭的嫡传大弟子。这不是挂名弟子，是真刀真枪练出来的。那时候的武术还带着功利性——护院、保镖、敢上阵，练不好就是白挨苦。

他厉害到什么程度？1923年4月去上海参加全国武术运动会，拿的是"最优等奖"，也就是第一。五年后1928年10月，南京第一届全国武术国考，这场更狠——少林、武当、八卦、形意各路门派全来了，散打抽签对打，面戴铁丝罩，拳脚无限制，击中一拳一分、一脚两分、击倒四分，三局两胜淘汰制，最后筛出最优等15人，窦来庚排在里头，而且是优等里的头名。中央国术馆馆长张之江、冯玉祥亲自颁奖，冯玉祥送的锦旗上提了四个字——"我武惟扬"，还配了一把龙泉宝剑。搁现在说，就是两届全国武术冠军，公认的第一。

![](.evernote-content/E4C1A3A6-C22A-46F9-B8FF-8FA4F206C4FB/83A77EB7-A334-40A2-8F90-2E52E1B01EB1.jpg)

凭这名气，他本可以留在济南、南京办武馆、收徒弟，安稳过一辈子。可他回了山东。韩复榘当时是山东省主席，也爱武术，把他请去主持山东国术馆，挂副馆长——其实馆长就是韩复榘自己挂个名，真正管事的全是窦来庚。那几年国术馆办得红火，亲自教的弟子上千，半军训半武校，学生大多是揣着"习武保家"念头来的，练拳练刀照着战场去的。

1937年冬天，画风突变。七七事变后山东顶到前线，韩复榘手里有十万兵，按理怎么也得顶一顶。结果韩不战而退，山东拱手让给日本人。窦来庚那时是第三集团军警卫武士队队长，跟着韩复榘，指望的"主心骨"转身跑了。他没多废话，离开韩部，回临朐拉了支"国民军义勇队"，身边先是国术馆的老弟子，一传十十传百，凑了三百多号人，不少是真有功夫的，问题是——枪少。

![](.evernote-content/E4C1A3A6-C22A-46F9-B8FF-8FA4F206C4FB/C6E2003A-FB74-4B00-921C-13BE209C4C03.jpg)

1938年1月临朐城被日伪占了。4月12日夜里，窦来庚亲自带一百多好手摸到城下，趁黑爬墙，先摸掉哨兵，进城贴身近战。这一百多号人练的是实用功夫，刀狠手快，日伪军那套列队射击在近战里使不开，一仗干掉七十多，日伪中队长渡边一郎带着残兵逃出临朐城。这是临朐抗日头一回胜利，靠"半武术半游击"打出来的。5月24日鬼子报复，飞机大炮掩护着七八百日军、三千多伪军反扑临朐城，窦来庚率队守，还是寡不敌众，突围出城，这一仗也毙了日伪七十多，包括一名日军指挥官。

![](.evernote-content/E4C1A3A6-C22A-46F9-B8FF-8FA4F206C4FB/F8A59CFA-9E70-4237-9C2F-B750B9151758.jpg)

1938年秋，义勇队被编进山东省保安体系，改成保安第十七旅，窦来庚任旅长，人数扩到两千多。他这人明白——枪炮年代，武功再高也别跟日军硬拼，所以17旅专干偷袭、扰交通、拔小据点这些活，活跃在沂蒙一带，1938到1942年跟日伪打了二十多仗。值得一提的是，他支持"国共合作、枪口对外"，跟共产党员马保三的部队有联络，还亲自掩护过中共地下党员任铭剑。

![](.evernote-content/E4C1A3A6-C22A-46F9-B8FF-8FA4F206C4FB/D7204561-EBE5-4977-9309-7C92D6EB0309.jpg)

1942年7月18日，日军从十几个县抽了三千多日伪军，分路包抄17旅。窦来庚在北福山开紧急会，部下多半劝避其锋芒、先撤保存实力，他没听，撂下一句"不成功便成仁"——鬼子送上门了，要跑，怎么面对山东父老？

7月19日凌晨开打。西线盘阳方向最烈，日军重炮往17旅阵地上砸，一个多小时西线前沿被突破，部队退到八埠顶、福山一带。东线也吃紧，节节往后缩，打到下午，17旅只剩不足百人，被压在邬家庄南沟里，日伪喊着要活捉窦来庚。最后还剩五十来人，窦来庚已经中弹负伤，弹也快没了。他下令只要还有一个活着的就抵抗到底，自己把最后一颗子弹留给了自己，举枪自戕，42岁。战后国民政府军事委员会追晋他陆军少将，1988年山东省人民政府追认革命烈士，2014年入第一批300名著名抗日英烈和英雄群体名录。

![](.evernote-content/E4C1A3A6-C22A-46F9-B8FF-8FA4F206C4FB/2BB74485-5DD7-4D47-82E6-86C300019DC7.jpg)

神剧里那些飞檐走壁的武林高手，真放到那个年代，大多也就是窦来庚这个路数——个人武艺在贴身近战里能占点便宜，可战场不给一对一的机会，刚冲出去砍倒一个，机枪一转人就没了。他1938年4月那一仗带一百多人干掉七十多日伪，战果已经够亮，可日军回头调三千人来，17旅就给连根拔了。战争是系统工程，情报、补给、火力、战术、组织，一样都不能少，不是哪个高手能一招翻盘的。

可话说回来，窦来庚这种"算不清账"的人，那个年代的中国偏偏不能缺。他本来可以不去送死，可以撤，可以等，可他选了留下。挺过那八年的不是一个两个聪明人，是一批又一批这种"有点不理智"的倔骨头，把命一块一块填进去，才把日子熬出来。神剧把他们的命拍成爽剧，轻飘了。
