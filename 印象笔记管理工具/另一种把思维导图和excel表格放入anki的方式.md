另一种把思维导图和 excel 表格放入 anki 的方式
=============================

[廖俊杰](https://www.zhihu.com/people/liao-jun-jie-11)

不务正业的医学生

由于医学类书籍里面的知识点过于繁多杂乱，本人比较喜欢用思维导图梳理其脉络。

可是思维导图的作用更多在于理解知识而不是保存和传授知识。因此并不适合复习。

所以，我产生了把思维导图放到 anki 里记忆的想法，可是，目前似乎只有 marginnote 可以做到导出导图到 anki，而没有 iPad 的我们不可能为此花上两千大洋

至于使用图片遮挡的办法，也就是通过 Image Occlusion Enhanced 遮盖思维导图，由于思维导图一般都比较大，在背诵，特别是用手机背诵时，要么整个图屏幕装不下，要么缩小图后字小的看不见，比较麻烦。另一方面，anki 在同步媒体时的速度也不可恭维…

所以我想出了另一种办法

将思维导图转变为表格，再通过插件快速导入到 anki 中

这是一张诊断学的思维导图

![](.evernote-content/64271452-3CB6-436E-B6F6-DC7C2C4BC3AD/66D167A9-EB91-4870-B517-607F5C575408.jpg)

通过 xmind7 转成 Excel 格式后（需要 pro 版）

![](.evernote-content/64271452-3CB6-436E-B6F6-DC7C2C4BC3AD/64A631A5-0C43-48AC-83A6-6274D64C3BA5.jpg)

整个思维导图就会变成这样条理清晰的表格

接下来，就是将 excel 放入 anki 了

目前我在网上找到的方案都是

[@余时行](https://www.zhihu.com/people/f6e6eafd757577f7729919c8733f48fe)

先生在早期提供的 [Anki 如何插入表格？](https://link.zhihu.com/?target=https%3A//www.jianshu.com/p/cc507c8ed9d8)

大致思路是将 excel 转换为 html 语言，在通过插件导入 html 语言

我的思路也是如此，不过当我发现了一个不错的插件时，这个过程简单了不少。

我们可以通过 [PasteHTML](https://link.zhihu.com/?target=https%3A//ankiweb.net/shared/info/1963806283) 这个插件比较方便的制作表格卡片

![](.evernote-content/64271452-3CB6-436E-B6F6-DC7C2C4BC3AD/28FFF30F-1234-4F1F-A4FF-EC9598CBD8BF.jpg)

这个插件可以直接复制 html 语言的表格，并以表格形式呈现在 anki 中  
所以我们要做的就是将 excel 转换为 html 格式

然后用浏览器打开文件，就是一份排好版的 html 表格了

�将其复制粘贴到 anki，挖空，就可以开始识记了。

最终效果如下

并不建议把整个思维导图做到一个表格卡片里，因为一张思维导图的内容是十分庞大的，直接显示在手机上会十分难受。可以把一个专题拆分成一个表格，而这个过程也是比较容易的。

注：无法把 html 表格复制到 anki 的朋友可以试着把表格先复制到 word，再从 word 复制到 anki
