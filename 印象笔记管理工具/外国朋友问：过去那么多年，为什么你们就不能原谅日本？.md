# 外国朋友问：过去那么多年，为什么你们就不能原谅日本？

---

原文链接: <https://mp.weixin.qq.com/s/cpR9xUnCMN_dHDXbqR3KIw>

原创 暖暖妈 暖暖妈爱分享
2025年07月31日 22:54 北京

在埃及的时候，有懂中文的埃及朋友对中国近现代历史感兴趣。

他问我当时侵略过中国的国家有很多，英国、法国、意大利、美国、德国、俄国等等，八国联军也干了很多坏事。

为什么过去这么多年了，你们总是抓着日本人不放？

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/5CF27CD9-129F-4B4C-94D0-8C3F895DD0A4)**震惊、愤怒、心痛**

昨天去看了《南京照相馆》，从电影院出来久久不能平静。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/E96EB815-64D1-4880-A103-D048824E9264)

我们中国人都知道一段惨痛的历史：南京大屠杀。

1937年12月13日南京沦陷。

日本人屠城六周以上。

30万同胞惨遭屠杀，2万多妇女被强暴，平均12秒就有一个生命惨死。

我们在历史课本上学到这段历史，在很多地方看到记录当时暴行的照片。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/0583A9DC-9FAF-4BAE-8AB5-573C6F2656E8)

八年级上历史课本

但是在《南京照相馆》里，我第一次好像离历史那么近。 

整个人被震惊、愤怒、心痛所击穿。

（以下略有剧透）

《南京照相馆》讲述的是南京大屠杀发生期间，一家照相馆里普通老百姓互助求生，拼死转移记录了日军滔天罪行照片的故事。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/C7F4AA66-EE0F-44FB-B28E-12E764C05186)

日军闯入南京城时，随军摄影师全程跟拍，记录下了日本侵略者的所作所为。

无差别杀人。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/B02D5175-CA66-4F66-9D18-5717F05676F2)

强奸妇女、女童，甚至老人。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/409AC8AF-1E1D-43B7-A3EF-86CE71A68A81)

尸体堆积如山。

路面、江水被人血染红。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/02155DBA-C123-49A4-8EBC-46C39D0A93D2)![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/58A81F4B-0455-492F-A5F5-4E51EC5B0AAF)

南京城一夜之间如人间地狱。

这些照片登上日本报纸，成了日军邀功的手段。

为了不被国际社会谴责，他们又用照片营造亲善假象。

一对夫妻怀里抱着的，是刚刚被日本兵摔死的婴儿。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/29265EFE-1CB8-4C6B-8440-E65CCA7985CB)

上一秒粉饰太平，下一秒人就会死在刀枪之下。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/3C34D3E5-361B-403C-9943-D62F8ACDD168)

虽然知道这是电影，但所有人都好像置身于那场88年前的苦难历史中。

照相馆老金一家、跑龙套的女演员林毓秀、编号1213的邮差阿昌，还有翻译官王广海，装死才活下来的小兵宋存义……

好像就是普通南京老百姓，特别真实。

《南京照相馆》还用了很多真实的历史照片和素材。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/5A39FEDE-0378-40CA-A239-F03AEA43D233)

臭名昭著的“百人斩竞赛”。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/D2B40A71-AB66-4D76-8243-D54C1E05CDDE)

竟然在日本登报大肆宣扬

两个日本军官打赌谁先杀满100人，谁就先喝红酒，无辜老百姓就此失去生命。

日军搜寻国民党军官的细节：

掀起帽子，看帽檐外的皮肤晒得黑不黑，虎口有没有枪磨出来的老茧，以此判断是不是军人。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/3D11969E-F824-4D49-87E1-1D1E93364882)

这是南京大屠杀幸存者讲述的真实历史细节。

还有转移保护相册也有真实原型。

![](data:image/svg+xml,%3csvg data-v-823ba74e='' xmlns='http://www.w3.org/2000/svg' viewBox='0 0 2 2' width='4px' height='4px' class='border_filler border_filler_lefttop'%3e%3cpath data-v-823ba74e='' d='M1.85.005A2 2 0 000 2V0h2z' fill='%23ffffff' fill-rule='evenodd'%3e%3c/path%3e%3c/svg%3e)

1937年15岁的罗瑾在照相馆当学徒，冲印照片时发现底片上触目惊心的暴行。

他冒着生命危险偷偷加洗了30多张，又选了16张最有代表性的做成相册。

私藏日军暴行证据，是极端危险的。

他秘密保存了两年多，后来去毗卢寺内的交通电讯集训队学习时，把相册藏在厕所土墙里，没想到却丢了。

其实相册意外被他的同学**吴旋**发现了。

吴璇将相册偷偷转移到一座佛像的底座里，秘密藏了6年。

直到日本战败，南京贴出征集战犯军事法庭证据，他才拿出来。

这本相册被编号为南京大屠杀案“京字第一号证据”，**成为审判南京大屠杀主犯谷寿夫的铁证。**

**![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/CC72F4B1-9C6E-4F25-B6A1-F3D790AFC79A)**

《南京照相馆》从普通百姓的视角，让我们看到当年南京究竟发生了什么？

日军犯下了什么样滔天的罪行？

以及这些罪恶是怎样被记录下来的？怎么被今天的我们看到的？

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/A38B742C-6096-487B-83C8-75FE22141EB6)

在隐忍中爆发

网上很多家长问，孩子能不能看？会不会吓坏？

我觉得小学以上的孩子是可以的。

《南京照相馆》拍得非常隐忍。

没有把杀戮当噱头，没有消费苦难，全程没有衣物撕扯，血腥暴力镜头。

也没有强暴女性的场面，没有暴力侵犯的场面。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/E2D6F01C-7F2F-4FCB-9B98-D28FFB367C3C)

但你能清楚知道发生了什么。

普通人的绝地求生，危难存亡时刻的民族血脉觉醒，非常动人。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/148C4925-1BC9-497C-8A63-EBE5EE6A8CFB)

电影里的人物都是小人物，一开始，他们很害怕，为了活下去竭尽所能。

老金一家躲在照相馆的地下室里。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/29BD3F44-A88C-4103-ADD7-F5D0F26183EB)

阿昌四处逃窜，假冒照相馆学徒。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/F85177F1-A835-4918-90E3-ACBA8B3CBB04)

宋班长在死人堆里装死。

林毓秀学日语，给日本人唱戏。

王广海给日军当翻译。

为了活命，他们聚在了照相馆这个暂时的庇护所里。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/877D94EE-51AE-4638-A90D-77298CBAAB24)

但当看到暗房里日军伤害同胞的铁证时，深植于**中国人骨子里的家国情怀被唤醒了**。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/6ADB395C-F28A-4863-8117-69C70E26DDBF)

他们勇敢地保护这些底片，为把这份铁证传出去，让外面的人知道南京发生了什么，义无反顾做出牺牲。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/3D3E2DC3-8AA2-4F12-B980-35FA4AEFC9E7)

没有说教，没有喊话，没有口号。用生命做誓言，完成了民族精神的传递。

大好河山，寸土不让！

我看到了当一个民族面临亡国灭种危险的时候，即便渺小到微不足道的个体是如何在内心燃起团结之火的。

日军的摄影师伊藤，一边喊着仁义理智，艺术追求，一边漠视罪行发生。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/8F4F8462-0EC6-482A-98CC-B53BEFDEC0EF)

一边说，你是朋友，一边暗自设局，目的只为博取军功。

日本军国主义思想是如何把人异化成杀戮机器的，刻画的一览无余。

我想起了张纯如在《南京大屠杀》里写下的两段话：

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/3C3B05DB-49B0-4070-910C-BEBC9FB139C4)

他们（日军士兵）被当成废物一样对待，是低微得不能再低的一群，进入了中国的首府之后，突然间他们拥有了比神还要大的权力。

所以不难理解，在南京，他们把过去几个月，甚至一辈子所受的压抑，以不可遏制的暴力形式爆发出来。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/3C3B05DB-49B0-4070-910C-BEBC9FB139C4)

《南京照相馆》里有一处细节体现了这一点。

伊藤被比他头衔高的军官扇耳光（这个军官以前是农夫），而这个军官又被官衔更高的军报扇耳光。

在军国主义思想下，当人被工具化，战绩便成为唯一目标，罪恶会像恶魔一样滋生。

这样沉痛的教训，不能再次发生！《南京照相馆》发出掷地有声的反思。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/1A6BA150-DE8E-47C9-BD37-101D0ACE90D9)

在以南京大屠杀为题材的影视作品中，我觉得《南京照相馆》是相当好的一部。

值得一刷再刷。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/2F0E64AF-FAFC-4B4C-8800-EB619C768247)

不要因为沉痛而回避

回到埃及朋友那个问题。

其他国家的侵略，它是战胜国对于战败国的剥削。

当时的清政府确实孱弱，我们认了落后就要挨打，但是日本人的作恶是对普通百姓的，是针对手无寸铁的平民的。

我相信很多国人也因为这段历史太过沉重，而不敢或者不愿走进电影院。

很多人还会觉得，南京大屠杀都过去那么多年了，现在是和平年代，为什么还要看这些？为什么还要背那么沉重的历史包袱？

日军在南京大屠杀中这种无底线的残忍，远远超过人类文明的底线！

每个中国人，每个中国的孩子都应该记住这段历史。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/56AA9E00-D172-40A6-9C7E-988C105F6B3C)

日军侵华期间，不只发生过南京大屠杀。

据官方统计，日本侵华战争伤亡人数达3500多万，中国非正常死亡累积人数至少达2.8亿。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/E7AD7FE3-6B96-45FF-9507-E94259D6FEEC)

可以说，今天我们所有活着的人都是幸存者！

日军在中国犯下了如此罪恶，反观日本又是什么态度呢？

自始至终，日本都拒绝承认这段残暴的历史。

日本的教科书将这段内容删得干干净净，用“南京事件”淡化“南京大屠杀”。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/1CD77DC7-1A31-47D7-819E-44370C77DC02)

日本官方到今天为止依然持莫能两可的回避态度。

许多重要的侵华战犯二战后不仅没有接受国际军事法庭的审判，还在日本担任要职。

越来越多日本年轻人在军国主义的教育下认为，那场侵华战争实际上是一场正义的战争。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/E1FA37D3-22E6-497F-A24B-4EDE2D0D73B5)

战火的硝烟褪去，亲历南京大屠杀的幸存者越来越少。

我录了一段视频，呼吁家长让孩子了解、记住南京大屠杀的历史。

当时，登记在册的南京大屠杀幸存者还有38人，但截止到2025年7月份，也就是这个月，南京大屠杀幸存者仅剩26人。

有一天这个数字将会清零。

世界上再也不会有任何一个活着的人能够证明那段历史、战争和屠杀。

忘记历史意味着注定重蹈覆辙。

![](.evernote-content/A677FFC8-957D-4F68-85FD-F932C6CFD00B/BEFF94CE-0145-41BD-BCBE-315F891AD87C)

我们希望我们的孩子永远记住这段历史！并不是希望他们长大了之后去报仇血恨。

而是只有他们知道落后和战争意味着怎样的苦难，他们才能知道和平和正义是有多么难能可贵。

只有不断努力，让国家强大，才能得到真正的和平。

不要因为沉痛而回避，多一人观影，多一人了解。

日本人总说，我们为什么要替我们的祖先说抱歉。

同样，我们又凭什么替我们的同胞说原谅。

勿忘国耻、珍惜和平、吾辈自强！
