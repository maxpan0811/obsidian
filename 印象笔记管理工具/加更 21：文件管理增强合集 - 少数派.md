Setapp 应用订阅服务收集了种类繁多、小巧又实用的小工具，它们有的可以帮助我们提高工作效率，有的是简化版的画图修图工具，有的帮助我们进行任务管理……不挨个试试，你可能都不知道有这么多趁手的小工具。

本文就为大家搜集了 Setapp 中帮助我们整理、处理文件的应用，一起来看看。

Yoink
-----

Yoink 是为 Mac 平台设计的文件暂存工具，我们可以利用它轻松的拖拽功能，把文件快捷高效地拖动到不同的地方。

安装好 Yoink 后，当我们在 Mac 上拖动任意一个文件时，Mac 的右侧底部会自动浮现一个窗口（可以修改窗口位置），我们可以把文件拖拽到此处，随后这份文件就被暂时存放在这里。接下来，我们把暂存窗口里的文件拖拽到目标位置，实现文件的移动或者分享。

你可能会问：想要移动文件或者分享，只要直接拖拽就行了，为什么还要多此一举暂存呢？

对于普通的移动/分享，Yoink 当然排不上用场，但是对于批量移动、多次分享以及跨平台移动，Yoink 都能帮上大忙。

我自己使用 Yoink 主要有以下几种场景：

1. 不同位置来源的文件，要移动同一个目标位置：把多个文件夹里的文件，挨个拖动到Yoink暂存，然后投放到目标文件夹里。
2. 同一个文件要进行多次分享：把文件拖动到Yoink暂存，然后投放到微信、邮件、网盘里去，还可先锁住后隔段时间再分享。
3. 在macOS与虚拟机之间分享：把Mac上的文件拖动到Yoink暂存，然后切换到虚拟机，再投放到公司OA、业务系统里去。

Yoink 几乎支持各种类型的文件，还可以接收剪贴板文字，以及在 iPhone、iPad 之间互相传输，当然它简洁的界面、易用的操作，也是我喜欢使用它的原因。

![](/images/file-generic.png)

Default Folder X
----------------

Default Folder X 是 Mac 平台的快速保存工具。它的作用就是让你在Mac上保存新文档的时候，能够快速定位到目标文件夹里。

它是怎么做的呢？首先，它会在保存文件目录的弹窗外叠加上一圈功能区，右侧有最近使用的文件夹、收藏的地址目录等等，帮助我们非常快速地定位到想要保存的地方。最绝的是，在这个弹窗出现的时候，把鼠标移动到桌面上已经打开的文件夹，这个默认的保存路径直接会被覆盖，变成桌面上打开的那个文件夹，然后点保存就完事了。

对于所有支持文档保存的应用，Default Folder X 都能派上用场。相比之前保存文件要在路径框里点来点去，或者先保存到桌面再移动到文件夹，毫无疑问用 Default Folder X 省事多了。

![](/images/file-generic.png)

Archiver/BetterZip
------------------

看名字就知道，这两个App的作用：压缩/解压缩文件。

Archiver 支持 36 种 压缩文件格式，它在工作时需要我们把待压缩的文件拖动到 Archiver 窗口中去，再选择压缩格式以及压缩程度。

BetterZip 提供了文件窗口方便我们浏览压缩/解压缩文件，它的特点在于，在解压缩前，我们就能预览压缩包里的文件内容。当然，BetterZip 也支持 30 多种压缩格式，以及提取 ISO 文件、加密压缩等等。

![](/images/file-generic.png)

Unclutter
---------

Unclutter 是三项全能工具，它集合了剪贴板历史、文件暂存功能、快速备忘录功能。

Unclutter 窗口会出现在Mac屏幕的顶部，将鼠标光标移动到屏幕顶部，然后下拉就可以出现。在这里从左向右数第一个是剪贴板历史管理，你可以浏览所有的复制内容，并选择相应的条目进行粘贴。随后是文件暂存，我们把文件拖动到屏幕顶部就可以实现暂存，但它与Yoink不同的是，Unclutter 文件暂存会移动原始文件。最后是快速备忘录，我们可以快捷地创建和管理简单的笔记信息。

如果你恰好需要 Paste 剪贴板工具、Yoink 文件暂存工具，又不喜欢多装App，不妨试试 Unclutter 一站式满足需求。

![](/images/file-generic.png)

Filepane
--------

Filepane 在你拖拽文件时玩起了花样，说起来它和 Yoink 和 Unclutter 有点类似。

当我们在 Mac 上拖动文件或文件夹时，鼠标旁边会浮现一个辅助窗口，将文件拖拽进去，就能执行一些快捷操作。目前有添加到邮件、快速移动至文件夹、图片裁剪、压缩/解压缩、共享、删除、转换格式等操作。

如果你在 Mac 上会非常高频地使用这些操作，不妨利用Filepane 来提高操作效率。

![](/images/file-generic.png)

HoudahSpot
----------

HoudahSpot 是 Mac 平台强大的文件搜索工具。用过 Spotlight 或者 Finder 搜索都知道，Mac 系统的文件搜索对于中文不太友好，而且搜索结果也不如人意。

HoudahSpot 支持多维度的搜索条件，包括文件名、创建日期、后缀、标签等元数据，而且搜索结果非常精准。当然它还有这些增强功能：

1. 支持搜索规则：我们可以设置多条搜索条件组成规则，找到我们想要的结果；
2. 搜索结果显示：符合搜索条件的结果，会在 HoudahSpot 中显示，不会像 Spotlight 用过就消失；
3. 保存搜索规则：对于那些常用的搜索条件，我们可以为它们创建模板，方便今后快速使用。

![](/images/file-generic.png)

Path Finder
-----------

Path Finder 可以理解成 Finder 增强版。 Path Finder 和 Finder 长得非常接近，因此使用起来完全无障碍，Path Finder 提供了增强 Finder 的几个功能：

1. 双窗口：Path Finder 可以并排显示两个 Finder 窗口，对于文件夹之间的相互移动非常有用；
2. 投放箱：集成了文件暂存功能，我们可以把文件拖动到左侧上方的投放箱暂存，再移动到目标位置；
3. 查看隐藏文件：点击眼睛图标，就能显示隐藏文件；
4. 高亮标签文件：给带有标签的文件名添加相应的底色，看上去更显眼；
5. 其他功能：Path Finder 还集成了废纸篓、终端、进程处理等等高级功能。

Path Finder 对于 Finder 可以说是全面兼容又进行了全方位的提升，如果你对 Finder 不太满意，不妨试试 Path Finder。

![](/images/file-generic.png)

Renamer
-------

Renamer 可以帮助我们快速地对文件进行批量重命名，这些文件可能是上百张照片，也可能是十几集电视剧集，或者是公司文档。

Renamer 的窗口像极了 Automator，实际上连用法也是。我们可以提取文件名进行批量替换，或者给文件进行重新编号，序号起始值、步进值都可以自定义。对于不同的文件，Renamer 可以提取出多维元数据，我们就根据这些元数据组织重命名操作的规则，再把文件拖到窗口中就能按照我们的要求把文件批量重命名了。目前 Renamer 支持的元数据有歌曲的歌手、年份、类型等，图片的像素、尺寸、相机数据等，文件的创建时间、修改时间等。
