心理学的四大公理
--------

![](.evernote-content/BD47E194-D3BE-491C-B0A7-13A4C442E153/D69773BE-939E-4F1D-A106-F6FC4DB38B87.jpg)

图片来自得到

* **基因一开始给我们设置好了初始参数。**

+ 我说：人是基因的载体。
+ 刘嘉说：这个参数会在后天环境中引导我们的成长。 我们人类其实是基因的载体，我们存在的意义就是让基因得到高效的传递。

* **大脑为我们的心理提供了生理学的约束。**

+ 我说：这里主要说情绪，刘嘉太学术化了，梁宁说过情绪是人的底层系统，后天所学的知识技能都是底层系统上的app或者小程序。
+ 刘嘉说：形神一体的核心所在： 杏仁核 (Amygdala)。跟我们的心血管系统、免疫系统和交感神经系统有着密切的关系。所以说， 当我们调节情绪时，我们其实也是在调节心血管系统、免疫系统等和我们的身体健康相关的人体系统。 只要精神好，身体也会健康。

* **心理是具有扭曲现实的力量的（乔布斯）。**

+ 我说：当能力没法满足现实的预期，需要调整自己的预期。估计两种调整，一种是调低预期，安于现状；一种是成长性思维，相信通过不断努力，满足预期。
+ 刘嘉说：当一个人无法改变物理世界，也无法改变自己情绪的时候，他最好的做法就是扭曲现实，改变自己对这件事情的看法。
+ 禀赋效应 (Endowment Effect)，即当你拥有一件物品之后，你会更加喜欢它。一个真正睿智的人，就是要在这个不完美的物理世界之上，构建出一个完美的心理世界。

* **社会是心理的最终归属**

+ 实际就是环境影响人的心理，因为我们独立的个体，也是社会人。
+ 嫉妒、悲伤、愤怒、害怕、羞愧以及孤独，就是我们被群体质疑、抛弃时产生的负面情绪；爱、欢乐、自豪以及高兴，就是我们被一个群体所接纳时产生的正面情绪。

如何破解囚徒困境：个人理性vs集体理性
-------------------

* 最高的情商：真诚。你不用去琢磨对方在想什么，对方的动机是什么，你只需要把你自己真诚地展现在别人面前就好。在这真诚里，有你的善良，有你的宽恕，更有你的报复。
* 刘嘉的具体策略：

+ 具体而言，这个策略分为两步。
+ 第一步，**在最开始，你要选择合作**。也就是说，在第一轮出手的时候，你总是选择合作。
+ 第二步，在后继的交往中，你采取**一个固定的策略，就是上一次对方怎么对待你的，你也就怎么对他。**

**我的理解：人不犯我我不犯人，人若犯我我必犯人。说到底如果对方背叛，让他付出背叛的成本，如果对方合作，让他获得合作的收益，这样才可以从长远上，推动合作，减少背叛。逻辑跟交友的逻辑、做生意的逻辑一样。**

* 刘嘉的策略解读

+ 这个看似非常简单的策略，其背后包含了非常重要的人与人交往的心理学原则。
+ **第一个原则是友善**。在一开始的时候，**你选择的是合作**，而不是背叛。也就是说你是一颗合作的心，来与人交互。这样，你给了别人一个善良的开始。
+ **第二个原则是报复**。在合作中，**你不能是一个没有原则的老好人，当对方背叛你的时候，你要毫不犹豫的去背叛对方。**
+ **第三个原则是宽恕**。无论对方背叛了你多少次，**只要对方这次选择了合作，那么你的下一次也要选择合作，也就是宽恕对方以前背叛的行为。**为什么？这是因为你的目的是为了合作，因此，你需要用你的善意，对对方的善意做一个及时的鼓励，**鼓励对方的这种合作行为**。
+ **第四个原则是透明**。这个原则也是我认为的最重要的原则。

自尊与成功
-----

* 约拿情结：是这种害怕成功的约拿情结，阻碍着我们获得成功，实际上也阻碍着我们获得真正的高自尊。马斯洛说，“如果你总是想方设法掩盖自己的本有的光辉，那么你的未来肯定黯然无光的”。
* 自尊公式：现代心理学创始人、哈佛大学心理学教授威廉·詹姆斯说：自尊=成功÷抱负，意思是：自尊不仅取决于成功本身，还取决于你对于成功的预期。增大成功和降低抱负，或者说，降低对于成功的预期，都能让你获得高自尊。
* why自尊

+ 自尊之所以重要，是因为它可以**抵抗物质世界的匮乏，让你突破贫穷的束缚，实现自我价值。高自尊是我们每个人都应该去追求的理想状态。**
+ 自尊的过度保护：但是，我们的问题恰恰是，我们常常对于**自尊进行了过度保护，这种过度保护阻碍着我们实现成长**，实现自我突破，最终获得成功。

* 如何获得成功：**成长型思维**会认为，成功并非因为我们的“聪明”，而是因为我们的“努力”和“坚毅”。非固化思维。

**在这里，我只学到一个新词高自尊，但是并不妨碍我一直坚信终身学习与持续性成长。我觉得与其推广高自尊，不如直接传播终身学习、成长型思维，后者比较好理解，知识没必要因为专业术语被阻隔传播。**

战胜拖延症的方法：do
-----------

* 初级方法：

+ **初级的第一步是把大目标化整为零**。把一个大目标分成一个个小目标，然后实现一个个小目标。
+ **第二步是准备**。把工作桌面上所有有诱惑的东西清除掉，只留下工作内容。例如把你的手机关机，放在一个不是触手可及的地方。然后在桌子上放一个便签，用于记下与当前内容不相干的想法。比如说突然想起来今天淘宝上有一个打折活动，先不要做，只是记下来，等你完成了当前的小任务之后再去干这件事情。
+ **第三步就是行动。**请记住：只要行动了，不管干什么，都是好的。比如说，当你坐下来拿起笔之后，发现不知道从何开始怎么办？这时你可以找一项过渡活动，比如抄写歌词或者抄一段书。只要你行动起来了，你就会发现心开始平静下来，焦虑也减弱了，于是你就慢慢进入工作状态了。
+ **第四步就是奖励，也就是及时反馈。**我们是时时刻刻都在等待奖励的。当你干了一件事情之后，大脑处于疲倦状态，就需要点多巴胺来加点油。所以，每当你完成一个小目标的时候，就给自己一个小小的奖励。吃个冰激凌，看个2-3分钟的视频等等。

* 中级方法：

+ **第一点，你要提前进行规划，而不要等到最后一刻。**你每天需要做的第一件事情，应该是你今天最需要完成的工作，越早做这个任务越好。因为在开始的时候，你的压力比较小，焦虑比较少，大脑需要的多巴胺也比较少，所以你可以很快地进入状态。而且越早完成任务，你就能越早做其他喜欢的事情。
+ **第二点，当你有多个工作的时候，怎么办？非常简单，从最难的开始。**什么叫做最难的？非常容易判断，你最不愿意干的事一定是最难的事。你就从最不愿意干的事干起，这是每天工作的第一件事。
+ **第三点，我把它称作一种仪式化的行为。**假设有一顶你非常喜欢的帽子，你在工作的时候把帽子戴上，不工作的时候，就把帽子拿下来。或者你可以把喜欢的小玩偶或者装饰品，当你工作的时候放在桌子前面，工作结束就放起来。这就是心理学中的条件反射，就像巴普洛夫的狗听见了饲养员的脚步声就会自动分泌唾液——当这个小玩偶出现你在面前的时候，你就会自动进入工作的状态。
+ **第四点，那就是一定要破除完美主义。**最现实的办法是，你要先快速完成一个粗糙简陋的版本，然后不断地迭代更新。

* 高级：

+ **养成一个习惯最高效的方法就是把新的习惯嫁接在老的习惯上面，从老习惯里生长出新的习惯。**
