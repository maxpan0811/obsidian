### 1 跳过 Finder 窗口 📁

      如果您对打开 TextEdit 然后单击“新建文档”的额外步骤感到有些烦恼，那么您并不孤单。

       虽然有点不方便，但当你全神贯注时，每次点击都很重要。当你打开 TextEdit 时，你不会看到满是文件的 Finder 窗口，而是可以立即开始在新文档中输入内容，这难道不是很棒吗？

       您可以使用以下简单的终端命令绕过 Finder 窗口，以便 TextEdit 应用程序在每次启动应用程序时自动打开一个新文档：

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/D8F4BFA3-729F-4720-B4FC-2AD5F64F8A7B.png)

→在 Mac 上打开终端应用 → 粘贴以下命令：

```
默认写入 com.apple.TextEdit NSShowAppCentricOpenPanelInsteadOfUntitledFile - bool  false
```

→ 按回车键，然后重新启动 TextEdit。从现在起，每当您打开 TextEdit 应用程序时，它都会直接打开一个新文档，您可以立即开始输入任何您想输入的内容。

### 2 使用密码保护你的文件夹

       我们的 Mac 上都有一些文件需要保密，无论是工作文档、个人照片还是敏感数据。虽然您可能习惯用密码锁定整个 Mac，但您是否知道还可以使用额外的安全层来保护特定文件夹？

      最好的部分是，您不需要任何第三方应用程序来实现此目的，因为 macOS 有一个内置工具“磁盘工具”可以完成这项工作。

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/A51E9B31-BDED-43D9-A57D-1F93885007B4.png)

以下是使用“磁盘工具”对 Mac 上的文件夹进行密码保护的方法：

→ 首先，使用 Spotlight 搜索或从 Launchpad打开“磁盘工具”应用程序。

→ 单击文件 → 新建图像 → 来自文件夹的图像。

→ 选择您想要保护的文件夹，然后单击选择。

→ 在出现的下一个窗口中，您需要命名您的磁盘映像并选择保存它的位置。

→ 对于加密，从下拉菜单中选择128 位 AES 或 256 位 AES。后者提供更强的保护，但速度稍慢。

→ 在图像格式下，选择读/写以允许更改文件夹。

→ 点击保存，系统会提示您创建密码。选择安全且容易记住的密码，或使用内置密码管理器生成密码。

→ 文件夹的映像文件现在将保存在所选位置。双击.dmg 文件时，它将作为驱动器安装，您需要输入密码才能访问内容。

专业提示：使用终端锁定文件夹

如果您更喜欢使用终端，可以按照以下方法使用终端锁定 Mac 上的文件夹。

→ 打开终端并导航到文件夹目录。如果您的文件夹位于下载文件夹中，请输入“ cd Downloads ”

→现在，输入以下命令：

```
hdiutil create -encryption AES -256 -volname “加密文件夹” -srcfolder “您的文件夹名称” -ov -格式UDBZ “EncryptedImageName.dmg”
```

→ 将“您的文件夹名称”替换为实际文件夹名称，将“EncryptedImageName.dmg”替换为加密文件夹的首选名称。

→ 系统将提示您输入并确认密码。完成后，按回车键。

### 3 最大化你的屏幕空间

        如果您是一个重视整洁工作空间的人，您可能已经尝试过自动隐藏 Mac 上的 Dock。

       但如果我告诉你有一种方法可以让它更加高效呢？通过调整几个设置，你可以隐藏 Dock 以最大化你的屏幕空间，并消除悬停时重新出现的动画延迟。

        这个技巧可以将您的 Dock 从占用空间的干扰项转变为时尚、反应灵敏的工具，仅在您需要时立即显示。

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/BE993A13-C274-4681-A621-88A18955CEE1.png)

以下是隐藏 Dock 并消除动画延迟的方法：

→ 首先，进入系统设置 → 桌面和 Dock，然后打开自动隐藏和显示 Dock。这样可以隐藏 Dock，只有将鼠标悬停在其位置上时才会显示。

→ 现在，打开终端并输入以下命令：

```
默认值为com.apple.dock autohide-delay -float 0；默认值为com.apple.dock autohide- time -modifier - int  0；killall Dock
```

→ 之后，点击Return，当你将鼠标悬停在 Dock 上时，它会立即出现，没有任何延迟。

### 4 将文件合并为 PDF

       如果您需要将多个文件、文档或图像合并为一个 PDF，那么您可能经历过打开专用应用程序甚至使用在线服务的麻烦。

       但如果我告诉你，你的 Mac 有一个内置的、闪电般快速的方法，可以直接从 Finder 中执行此操作，你会怎么想？无需额外的应用程序，也无需将文件上传到可疑的服务 - 只需单击几下，你就可以得到一个整齐有序的 PDF 文件。

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/9CD39802-17FF-4892-89BE-2EE0C94B10E3.png)

将多个文件合并为一个 PDF 的方法如下：

→ 打开Finder并导航到您想要合并的文件。

→ 按住“ Command ”键并单击 PDF 中您想要的每个文件。您选择文件的顺序决定了它们在 PDF 中出现的顺序，因此请谨慎选择。

→ 选择文件后，按住 Control 键单击（或右键单击）并选择“快速操作”→“创建 PDF”。您也可以选择文件并使用 Finder 窗口预览窗格中的“创建 PDF”按钮。如果预览窗格不可见，只需转到“视图”→“显示预览”或使用快捷键Command + Shift + P。

→ 将立即创建一个 PDF 文件，以您选择的第一个文件命名。

### 5 立即总结网页内容

       我每天都会阅读大量文章，并且通常会使用某种在线服务来快速总结文章或从中提取要点。直到最近，我才知道 Safari 中隐藏了“总结”选项。

      这项 macOS 内置功能可让您只需点击几下即可将冗长的文章压缩为简短的摘要。无论您是浏览研究的重要细节还是只需要快速概览，此技巧都会节省您的时间和精力，让您的网页浏览更加高效。

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/C40C1119-869F-4C4E-ADEC-945E94A321F9.png)

以下是如何在 Safari 中使用摘要功能：

→ 在任何网页上，单击并拖动光标，突出显示要总结的文本。如果要选择整个页面，也可以使用Command + A快捷键。

→ 右键单击选定的文本并导航至服务 → 总结。

→ 如果您没有看到“摘要”选项，则需要先激活它。为此，请单击顶部菜单栏中的 Safari，然后选择“服务”→“服务偏好设置”。

→ 在首选项窗口中，向下滚动直到找到“总结”并勾选旁边的复选框。

→ 激活后，您可以按照上述步骤操作，新窗口会弹出所选内容的简明摘要。您还可以根据自己的喜好增加或减少摘要的长度。

### 6 自动打开会议相关文件

我们都经历过这种情况（至少我经历过），在会议开始前几分钟手忙脚乱地寻找合适的文件。如果会议开始后您无需动手，就能准备好一切，那不是很棒吗？

使用 macOS 的日历应用，您可以自动执行此过程，确保会议开始时所有相关文件都会自动打开。这个技巧不仅可以节省您的时间，还可以帮助您保持井然有序和专注，确保您始终做好准备。

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/76059131-1BC2-4889-9E31-CFC51C32E3FD.png)

设置会议自动打开文件的方法如下：

→ 在 Mac 上启动日历应用程序，然后单击 + 按钮创建新事件，或双击要修改的现有事件。

→ 在活动详情中，找到“添加附件”选项。单击它，然后导航到会议开始时要打开的文件。您可以附加多个文件 — 文档、电子表格、演示文稿，以及会议所需的任何内容。

→ 为确保文件自动打开，请为事件设置警报。在警报选项下选择自定义，然后选择打开文件。从那里，选择要打开的特定文件。

→ 到达会议时间时，附件将自动打开，以便您立即开始会议。

### 7 自定义文件夹图标

      让我们面对现实：macOS 上带有粉蓝色共享的默认文件夹图标可能有点单调。但您是否知道，您可以通过自定义这些图标轻松个性化您的 Mac？

      无论您是想让桌面或文件夹更具视觉吸引力，还是仅仅需要一种更好的方式来组织文件，自定义文件夹图标都是让您的 Mac 真正属于您的一种有趣且实用的方法。此外，它非常简单。

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/C77CB805-852D-4019-95AD-68592362944B.png)

以下是如何在 Mac 上自定义文件夹图标：

→ 前往网站macOS Icons，这是一个很棒的资源，您可以在其中找到数千个适用于各种用途的精美设计图标。

→ 浏览丰富的图标库并下载您喜欢的图标。每个图标均采用高分辨率格式，让您的文件夹看起来清晰、专业。

→ 导航到 Mac 上您想要更改的文件夹。右键单击该文件夹并选择获取信息。

→ 打开您下载的图标，选择它，然后复制它（Command +C）。

→ 返回文件夹的“获取信息”窗口，单击窗口左上角的小文件夹图标，并将新图标粘贴（Command + V）到位。

→ 就这样，关闭“获取信息”窗口，您的文件夹现在将显示自定义的图标。

### 8 更改文件夹背景

       Mac 的文件夹不仅要实用，还可以反映您的风格和个性。您知道吗，您可以将 Finder 中任何文件夹的背景更改为纯色，甚至是自定义图像？

      无论您是想让特定文件夹脱颖而出，还是只想为文件组织添加一点特色，此技巧都可以让您以实用而有趣的方式自定义工作区。

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/FD21C350-51F0-4638-8348-9131F5C095F1.png)

以下是如何在 Mac 上更改文件夹背景：

→ 在 Finder 中导航到您想要个性化的文件夹并打开它。

→ 通过前往“视图”→“作为图标”或单击 Finder 窗口工具栏中的图标视图按钮，确保您处于图标视图中。

→ 打开文件夹后，单击“视图”→“显示视图选项”（或使用 Command + J）

→ 在视图选项面板中，您将看到背景下的选项：

颜色：选择颜色，然后单击颜色框为文件夹背景选择纯色。

图片：如果您想使用自定义图像，请选择图片。单击选择以浏览您的文件并选择将用作文件夹背景的图像。

→ 选择背景后，它将立即应用于文件夹。关闭“查看选项”面板，然后享受新自定义的文件夹。

### 9 为你的码头添加垫片

       如果您喜欢井然有序的结构，那么您可能会发现，随着应用程序的增多，Mac 的 Dock 会变得有些杂乱。好消息是，有一个简单的方法可以整理您的 Dock，并使其看起来更加简洁、个性化 — — 通过添加垫片。

      分隔符可让你将相似的应用程序分组在一起或在不同类型的应用程序之间创建视觉分隔，从而使你的 Dock 不仅更有条理，而且功能更强大。

以下是如何将 Spacer 添加到 Dock 的方法：

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/0EE14590-B6C9-45BB-BF51-2E19DC559B37.png)

→ 打开终端并输入以下命令：

```
默认写入 com.apple.dock 持久-应用程序-数组-添加 '{“tile-type”="small-spacer-tile";}' ; killall Dock
```

→ 按下Return 键，您将看到 Dock 中出现一个小小的、不可见的空间。

→ 您可以多次重复该命令来添加更多间隔符，从而可以根据需要创建任意数量的分区。

→ 将应用程序拖放到间隔物周围，以按照您喜欢的方式组织 Dock。

### 10 推荐和最近的应用程序

      您的 Mac 非常智能 — — 它非常智能，它可以帮助您在 Dock 中跟踪最常用和最近访问的应用程序。

      通过启用显示建议和最近使用的应用程序的选项，您的 Dock 会变得更加动态，会自动更新以反映您当前正在使用的应用程序。此技巧可节省您的时间并使应用程序之间的切换更加轻松。

以下是在 Dock 中显示推荐应用程序和最近使用的应用程序的方法：

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/65C1DBAD-1512-4A01-9506-E45C4F7D8A01.png)

→ 从 Apple 菜单转到系统设置，然后导航至侧边栏中的桌面和 Dock。

→ 向下滚动直到找到在 Dock 中显示最近的应用程序选项并将其打开。

→ 现在，您的 Dock 将自动显示建议和最近使用的应用程序，让您更轻松地访问所需内容，而不会因太多固定应用程序而弄乱您的 Dock。

### 11 在标签页中打开文档

      如果您的 Mac 桌面经常因打开多个窗口而变得杂乱，那么您并不孤单。但有一个简单的解决方案可以让桌面保持整洁：在选项卡中打开文档，而不是在单独的窗口中打开。

      此技巧在同一个应用程序（如 Pages、Keynote 甚至 Finder）中处理多个文档时特别有用，可以帮助您保持井然有序和专注。

以下是在选项卡中打开文档的方法：

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/AF8CE0BD-06A4-4E6E-9F90-37927367C04B.png)

→ 通过 Apple 菜单进入系统设置，然后导航至侧边栏中的桌面和 Dock。

→ 向下滚动到打开文档时首选选项卡设置，然后从下拉菜单中选择始终，以确保文档尽可能在选项卡中打开。

现在，当您打开新文档时，它们将作为选项卡出现在单个窗口内，而不是多个窗口占据整个屏幕。

### 12 按应用程序分组任务控制窗口

      任务控制是一个用于管理多个打开的窗口和桌面的强大工具，但如果您同时打开多个窗口，它可能会让人难以承受。

      通过在 Mission Control 中按应用程序对窗口进行分组，您可以更轻松地导航打开的应用程序，保持工作空间井然有序并减少视觉混乱。

以下是按应用程序对任务控制窗口进行分组的方法：

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/D120CFD4-4CC4-4CD7-B6B1-21EF5A72BE27.png)

→ 转到“系统设置”，导航至“桌面和 Dock”，然后向下滚动以找到“任务控制”部分。

→ 开启按应用程序分组窗口。

现在，当您激活 Mission Control（使用 F3 键或触控板上的手势）时，来自同一应用程序的窗口将被组合在一起，从而更轻松地找到所需内容并使您的屏幕不那么混乱。

### 13 将应用程序放在最上面

       您是否曾经使用 Apple Notes 应用程序记笔记，同时引用其他文档或应用程序中的信息？我敢肯定，您在处理其他事情时必须不断在应用程序之间切换并试图保持笔记可见，这让您感到很沮丧。

      但你猜怎么着？macOS 有一个内置功能，可以让 Notes 应用处于领先地位 — — 无需第三方应用，无需变通方法，只需一个简单的设置就能带来巨大的改变。

以下是如何让 Notes 应用始终位于最前面：

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/06ABEC12-FF73-47DE-964C-5C69DE559438.png)

→ 首先，启动 Notes 应用程序，然后双击要保留在顶部的笔记。这会将您的笔记弹出到一个单独的窗口中。

→ 在注释窗口处于活动状态时，转到屏幕顶部的窗口菜单，然后选择保持在顶部。

→ 就是这样！无论您在做什么，您的笔记现在都可见。

### 14 将文件夹固定到 Finder 工具栏

       通过 Finder 导航访问您常用的文件夹很快就会变得乏味和烦人。但实际上有一种快捷方式可以让您最重要的文件夹只需单击一下即可访问。

       通过将文件夹固定到 Finder 工具栏，您可以立即访问它们，无论您在文件系统中的哪个位置。这是让您的 Mac 感觉更适合您的工作方式的简单技巧之一。

将文件夹固定到 Finder 工具栏的方法如下：

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/BEA5E66A-D30A-47F0-98B3-C0E6E08F795E.png)

→ 首先，打开 Finder 窗口，然后导航到要固定的文件夹。它可能是你经常使用的东西，比如你的工作项目、下载内容或最喜欢的照片集。

→ 现在，按住 Command 键，然后单击文件夹并将其拖到 Finder 窗口顶部的工具栏上。松开后，您的文件夹将固定在工具栏上，以便快速轻松地访问。

→ 现在，无论您在 Finder 中的什么位置，您都可以单击工具栏中的该文件夹并直接跳转到该文件夹。您还可以将文件和文件夹拖放到其中。

### 15 将应用程序文件夹添加到 Dock

       我想这样做很久了，当我终于遇到这个技巧时，我非常兴奋。这个技巧可以让你在 Mac 的 Dock 中拥有一个包含所需应用程序列表的文件夹，类似于你在 iPhone 或 iPad 上所做的操作。

       我一直想知道为什么 Apple 允许用户在其他设备上这样做，却不允许在 Mac 上这样做；好吧，事实证明有一种方法可以将应用程序分组到文件夹中，并将它们直接放在 Dock 中。

      这样，您就可以让所有内容井然有序，只需单击一下即可，而不会让 Dock 中堆满太多图标。

以下是将应用程序文件夹添加到 Dock 的方法：

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/9B2F9C73-6E97-4F52-8770-617A6B5AA8E3.png)

→ 首先，在 Mac 上的任何位置创建一个新文件夹，然后在此文件夹内，根据您想要分组的应用程序类别创建另一个文件夹 — — 例如，为所有与工作相关的应用程序创建“Office”文件夹。

→ 之后，打开一个新的 Finder 窗口，导航到应用程序文件夹，并找到您想要包含的第一个应用程序。

→ 右键单击该应用程序并选择“创建别名”。这将创建一个可以放置在任何位置的快捷方式。将别名拖到您刚刚创建的“Office”文件夹中。

→ 对您想要包含在此文件夹中的所有其他应用程序重复此过程。

→ 将所有别名收集到文件夹中后，将“Office”文件夹拖放到 Dock 的右侧（靠近垃圾箱和下载文件夹）。

→ 右键单击 Dock 中的文件夹并选择“显示为”部分下的“文件夹”，以确保它显示为文件夹图标而不是堆栈。

→然后，选择“将内容视为”部分下的“网格”，以便在单击文件夹时以整齐的网格布局显示应用程序。

       现在，您可以从 Dock 中删除单个应用程序图标以清理杂乱，同时知道您可以通过刚刚创建的文件夹快速访问所有应用程序图标。

### 16 必备快捷方式快速运行

      有时，在您的日常工作流程中，那些小事会发挥最大的作用，这些小而强大的快捷方式可以帮助您加快常见任务的速度，让您感觉自己像一个真正的 MacOS 高级用户。

![](.evernote-content/4D476DA5-8C07-431B-A189-090D588DC9D7/F4CCD00C-4385-47FE-91DA-F0FC9B44A807.png)

→ 使用“Opt + 8”在任意位置添加要点

→ 使用“Opt + Cmd + 左或右箭头键”在任何应用程序的选项卡之间导航

→ 使用“Cmd + ,”在任何应用中打开“设置”

→ 使用“Cmd + Ctrl + Space”在任何地方访问表情符号键盘

      单独来看，这些快捷方式可能看起来微不足道，但当你开始定期使用它们时，你会注意到你的工作流程变得更加顺畅和快捷。这些技巧让你感觉更能掌控你的 Mac，将日常任务变成无缝操作。
