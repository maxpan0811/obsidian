# 还是iFixit给力 iMac Pro详尽拆解出炉！

---

虽然 iMac Pro 的售价超过4万元，但是依然有不少朋友表达了对这台性能怪兽的兴趣，而此前网络上零零散散的拆解却让人不太尽兴。不过好在专业拆解机构 iFixit 并没有让我们等太久，他们在今天上传了 4999 美元的低配版 iMac Pro 的详细拆解，一起来看看。

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/A8AFE300-515A-422E-BE0A-127B6B7F512F.jpg)](http://resource.feng.com/resource/h062/h24/img201801031427240.jpg)

包装配件方面，为了配合 iMac Pro 深空灰外壳的设计，随机的妙控鼠标、妙控键盘和妙控版都是采用的同样的深空灰配色，限量版哦！同时还提供了一条黑色的 Lightning 至 USB 线缆，供 iPhone/iPad 使用。

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/F1FA87C2-57DD-41C6-B77C-620D1573AD78.jpg)](http://resource.feng.com/resource/h062/h24/img201801031427450.jpg)

接口方面，iMac Pro 提供了3.5mm 耳机插孔、标准 SDXC 读卡器、4个 USB 3 接口（苹果并未标出是3.0还是3.1）、4个支持 Thunderbolt 3 的 USB Type-C 接口和 10Gb 以太网口，接口数量和类别都非常齐全。

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/B8765CEE-97E6-4B6A-BB60-1CEB8F787637.jpg)](http://resource.feng.com/resource/h062/h24/img201801031428150.jpg)

拆开 iMac Pro 依然需要从屏幕下手，热风枪加热后用工具撬开屏幕，吸盘拉开后就能看到内部的景象了，iMac Pro 的内部几乎是完全对称的，强迫症患者会很舒服。

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/EB6F640A-834E-4818-BB1F-99D5C709A5B2.jpg)](http://resource.feng.com/resource/h062/h24/img201801031428520.jpg)  
  
[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/F1FC5500-2EB2-405D-9BE6-17E1DC8957E2.jpg)](http://resource.feng.com/resource/h062/h24/img201801031441390.jpg)  
  
[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/B1C1254D-CC40-4117-8A3D-A33B96A0BBFF.jpg)](http://resource.feng.com/resource/h062/h24/img201801031429490.jpg)

可以看到 iMac Pro 的散热器体积非常大，苹果称这款散热器散热能力提升了80%，这显然是为它强劲性能而准备的。散热器两侧就是体积超大的扬声器了。

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/C6A53986-2DB0-4651-B0FD-2E13AD1C8A8A.jpg)](http://resource.feng.com/resource/h062/h24/img201801031430320.jpg)

不过更强的散热也一定程度上挤占了其他部件的空间，iMac 5K 上深受用户喜爱的内存外部更换槽就没有了，并且苹果也移除了全尺寸桌面硬盘。

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/6155068A-0E4E-4CE7-9732-35A85D06C462.jpg)](http://resource.feng.com/resource/h062/h24/img201801031430590.jpg)

接下来开始拆主板，AirPort 无线网卡被焊在了主板上，而在此前的 iMac 5K 上苹果使用的是模块化设计。

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/59BF902A-914A-47AD-A335-E8225BCAABEE.jpg)](http://resource.feng.com/resource/h062/h24/img201801031431150.jpg)

拆下主板之后，可以看到四根内存都是可以手动卸下来的，iMac Pro 搭载的也是 288-pin DDR4 ECC RAM 内存条，型号为海力士 H5AN8G8NAFR-VKC 2,666 MHz DDR4，能够提供服务器级别的稳定性。

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/B1E41892-609F-4AD0-8A81-AB0843C97CAB.jpg)](http://resource.feng.com/resource/h062/h24/img201801031442050.jpg)

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/5DABEF77-AFBF-44DF-AF3E-167D497148BF.jpg)](http://resource.feng.com/resource/h062/h24/img201801031431460.jpg)

而两块512GB 的 PCI-E SSD 组成了1TB 的硬盘，SSD 来自闪迪，闪存颗粒型号为 SDRQF8DC8-128G，每块 SSD 上四颗闪存，一块512GB，两块一共1024GB。不过因为是苹果定制的 SSD，所以基本上是很难以更换的。

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/5021E103-7F79-4C64-8053-01BC01848FC5.jpg)](http://resource.feng.com/resource/h062/h24/img201801031432210.jpg)  
  
[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/B3E2DD9A-DFC3-4E99-8A39-E1428DB34FAE.jpg)](http://resource.feng.com/resource/h062/h24/img201801031440510.jpg)

接下来要拆散热器了，散热器下方就是来自英特尔的CPU，这块 CPU 据说是英特尔为苹果特供的，所以虽然是标准的 LGA 2066 接口，但是仍不确定是否能够自行升级。

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/1FFE3D78-238B-4F8D-B047-FDF6265B6448.jpg)](http://resource.feng.com/resource/h062/h24/img201801031432400.jpg)  
  
[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/B5ED9F72-6349-4CB9-A404-D8DE55939C78.jpg)](http://resource.feng.com/resource/h062/h24/img201801031432530.jpg)  
  
[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/EA629ED1-A280-4E99-90A4-F7CD344DF16D.jpg)](http://resource.feng.com/resource/h062/h24/img201801031433150.jpg)

下图为主板详细标注：

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/379A1905-E1CD-4DE7-B044-6EFC288D5B7F.jpg)](http://resource.feng.com/resource/h062/h24/img201801031433360.jpg)

红色：英特尔至强 W-2140（Skylake 架构，14nm 制程），标准频率为 3.2GHz，睿频 4.2GHz，

橙色：AMD S5J68 1747 GPEW0333S3 SS63HBN181747US40104 Radeon Pro Vega 56 显卡，配备 8GB HBM2 显存，

黄色：英特尔 X723D733 E1 05780，iFixit 认为可能是平台控制组件

绿色：AQUANTIA AQtion AQC107-B1-C PCIe 转以太网控制器

浅蓝：Pericom Semiconductors PI3PCIE3412AZHE PCIE 3.0 mux/demux switch

蓝色：苹果/USI 339S00428 00012021 Wi-Fi/蓝牙 模块

紫色：Genesys Logic GL3227A SD 4.0 内存卡控制器

下图为主板背面标注，苹果对于主板空间的利用率一如既往的高：

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/67D4433E-ABFD-4FF9-9F91-98FBBA762596.jpg)](http://resource.feng.com/resource/h062/h24/img201801031434060.jpg)

红色：音频/DAC芯片，来自 Cirrus Logic 的 CS42L63

橙色：3个 Primarion PXE1110CDM 1YUS7Q84 H1746 8F4 PMC005 和 Primarion PXE1610CDN 1YUS7Q84 H1745 807 PW005

黄色：2个 英特尔 JHL6540 Thunderbolt 3 控制器

绿色：International Rectifiers IOR 35217-01 C740P GSGK

浅蓝：Macronix MXIC MX25L4006EZNI CMOS 序列闪存

蓝色：恩智浦（NXP）L6524 I/O 扩展器

在下图，iFixit 还为我们标注出了两块重要的芯片：苹果 T2 （红色）和 A10 芯片（黄色），iMac Pro 上市之前的传言也得到了验证。T 系列芯片首次出现在打在了 Touch Bar 的 MacBook Pro 2016款上，T1芯片当时负责 SMC、图像信号处理、音频控制和 SSD 控制、 Secure Enclave 和硬件加密等功能。

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/841E11B5-4E5F-4D3C-A337-F804AE11AED6.jpg)](http://resource.feng.com/resource/h062/h24/img201801031434390.jpg)

电源位于主板下方，所以主板看完之后，我们再来拆电源。 电源生产商为 AcBel Polytec，支持 100-240V 输入，电源上的管理芯片来自意法半导体。

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/BC5B2B73-B9F9-4E1D-9554-C70C4F2AD6AD.jpg)](http://resource.feng.com/resource/h062/h24/img201801031436390.jpg)

最后拆下扬声器，巨大的体积已经足够体现 iMac Pro 的音乐表现，据说 iMac Pro 的扬声器是 Mac 电脑上最出色的。

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/24EBA5CB-21E0-479E-ADC5-2E3E514C433F.jpg)](http://resource.feng.com/resource/h062/h24/img201801031436540.jpg)

主板看完，我们再来看看屏幕，与 iMac 5K 相同，iMac Pro 的屏幕来自 LG Display 的 LM270QQ1，尺寸达到27英寸，分辨率依然为5K，最高亮度500nit。

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/E5FF9893-91A9-4E55-B93F-2964CCAD4DCE.jpg)](http://resource.feng.com/resource/h062/h24/img201801031437590.jpg)

最后，iFixit 为 iMac Pro 打出了3分的可维修性得分（满分10分），因为他们认为 iMac Pro 的显卡是焊在主板上的，而且 SSD 为定制版，且缺少外置内存更换装置，升级需要拆机显得很麻烦。

[![](.evernote-content/6D736D8F-E786-43D0-A0A9-D816426F2A0D/91754D33-8A1B-465C-853D-071224E3A470.jpg)](http://resource.feng.com/resource/h062/h24/img201801031438210.jpg)
