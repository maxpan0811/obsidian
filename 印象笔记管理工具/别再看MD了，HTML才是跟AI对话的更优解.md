Claude Code 团队写文档现在已经不用 Markdown、全改发 HTML 了——Markdown 是作者的草稿，排好版的 HTML 才是给读者看的成品，手搓排版太折腾，这事终于被本地 AI 兜底了。

开源圈有个 7k 星的 html-anything：扔段 Markdown、CSV、甚至没头没尾的草稿，按下 ⌘+Enter，本地 AI 几秒还你一个能直接发的 HTML 单文件。

最戳我的是 0 边际成本——不用配 API Key，直接复用你电脑里已登录的 Claude Code、Cursor、Gemini，等于把已付费的订阅再榨出一个排版设计师。

它自带 75 套模板、9 类场景，小红书卡、PPT、数据图一键出成品，写公众号尤其友好：样式内联，复制进后台 0 调整。

别当它是魔法，它只排版、不替你想内容，最适合有干货却烦透了对齐网格调字号的人。
