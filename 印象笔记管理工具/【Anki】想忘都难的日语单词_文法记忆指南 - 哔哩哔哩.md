获取Anki
======

关于Anki
======

偷懒引用Anki中国 (www.ankichina.net) 的介绍：

> Anki是一个记忆辅助软件，它可以让记忆过程变得更加轻松容易。 Anki只是提供了一种学习方法，与内容无关，它支持图片，音频，视频以及科学符号，并且这种各领域的支持是无止境的。

简单来讲，Anki不仅可以大大提高涉及记忆的学习效率，而且运用领域相当广泛。Anki背后有着两个大量科学文献支撑的简单理念：主动召回测试（Active Recall Testing）和间隔重复（Spaced Repetition）。通俗来讲，主动召回测试就是制作和使用正面问题背面答案的卡片，而间隔重复则是根据记忆的牢固程度来不断拉长复习的时间间隔，用最少的努力去学习东西。关于Anki的原理甚至更多的使用方法，可以参考Anki本身的用户手册（中文2.0版本：www.ankichina.net/anki20.html， 英文2.1版本：apps.ankiweb.net/docs/manual.html） 。

Anki适用对象
========

既然适用范围和领域都如此广泛，Anki是不是就适合任何学习呢？根据我自己的使用经验和网上很多使用案例来看，很多人对Anki的作用存在一定误解——Anki并不是万能的。如果我们把学习的过程简单地分割为获取知识、理解知识和消化知识的话，那么Anki能负责的只有最后一个阶段，也就是记忆。Anki不知道怎么帮你搜索你想学习的东西，也不知道如何塞给你理解方法（共享牌组在一定程度上通过分享他人的知识或是理解提供了一些解决方案，但这终究有些浪费Anki高度灵活的优点，并且影响它在记忆阶段的作用发挥）。放到日语学习这个小框架里来说，Anki不是词典或是语法书（获取知识），不是教材网课（理解知识），它只是一个比较科学智能的随身卡片本。

￼

![](.evernote-content/32895A56-8140-435D-A350-C62FEF842B2D/9FB3A913-9C3D-41ED-A1F0-D0ACC3485D0C.webp)

MUJI聚丙烯单词卡片，我曾经买了一组，但是制作速度太慢。Anki只是增加了一个间隔重复的智能提醒功能。

虽然比起手工制作，用电脑在Anki录卡片快了不少，但无论是制作卡片，还是间隔复习掌握，Anki的学习周期依然是比较长的，因此，比较适合用Anki的日语学习er的定位是：

* 有比较充分的时间（取决于绝对时间长度和需要制作、记忆的卡片数量）
* 有明确的知识来源（单词书，文章阅读，教材，文法手册等）
* 大脑已经初次处理过相关知识（例如，已经理解过某一条文法的解释和例句）  
  而这篇文章则更细化到特定的学习目标：**在有限的时间内积累大量日语词汇/文法。**

Anki类App
========

具备以上优秀功能的记忆辅助软件并不只有Anki一家。可能是开源软件+高度自定义化的原因，Anki在各个平台上的样式都挺……朴实无华的。如果你不欣赏MUJI的性冷淡，而是喜欢国誉等其他牌子封面亮丽的单词卡片的话，另一家UI美得多的quizlet会是你的选择（quizlet有高级版订阅，但不影响免费使用，两者记忆曲线算法也不尽相同，大家可以自己权衡）。

Anki下载
======

Anki可以在浏览器上直接使用（ankiweb.net)，最开始也要先在网站上注册一个账户，注意界面是英文的。  
Anki在各个平台都有对应客户端，下载可以到https://apps.ankiweb.net/拉到最下方，选择对应的平台，包括windows, mac, android, linux等，iOS请到苹果商店搜索，这是唯一一个需要付费的平台（而且还很贵，其中的历史纠葛错综复杂……）。安卓端没有google play的小伙伴可以到github下载，最新版本的地址是：https://github.com/ankidroid/Anki-Android/releases/tag/v2.8.4，下载那个.apk文件就可以。

Anki使用基本原则
==========

如果很熟悉它的理念，你可以根据自己的需求去自由地使用Anki。我在使用中也走了很多弯路，最后总结了自己的一套基本原则，仅供参考，也仅适用于短时间大量积累基本词汇的情况：

* **简洁**。不像纸质卡片，Anki的卡片长度是没有限制的，你可以塞一堆知识点进去，但是在使用时，最好还是当做纸质卡片来存放内容，尽可能去精简，这样才能面对上万张卡片也能轻松复习。有的单词可能有很多很多种意思，但在录进去之前，想想平时用得上那些高级意思吗，而那些简单的意思有必要复习吗，每种意思都有必要放一条例句吗？对初学者而言，一开始积累英语单词就翻着牛津词典背肯定是背不下来的……放进太多信息只会让复习时眼花缭乱，不仅速度下降，而且记忆也未必稳固。另外，网上有很多可以下载的高颜值卡片模板，但我自己因为学习时间比较紧，想尽量避免注意力被卡片样式夺走，所以基本没有调整太多卡片样式设置。
* **快速**。一是要快速录卡片（这也是为什么提倡简洁——否则光是录卡片就要花太多时间，而且打字速度要快，灵活运用简化符号），二是快速复习（简洁的条件下可以很快过掉），我在记忆日语单词时，平均每天复习卡片400多张，但是平均只需要40多分钟，也就是6秒1张，这还包括了中间有时手抄不熟悉单词的时间。人的注意力是有限的，如果每天背单词都要几个小时，除非全职学日语，大概任谁都很难坚持。
* **针对性**。针对性是Anki高自由度的极佳体现，也就是说，你可以跳过一切你知道的东西，只录进从未被记住的知识，我**极力推荐大家自己制作卡片，虽然有些费时，但这是真正完全定制化了的学习卡片组，它100%匹配你的需求。**如果实在没有时间自制卡片，也可以到anki的共享日语牌组库 （ankiweb.net/shared/decks/japanese） 下载符合目标的牌组，共享牌组的质量参差不齐，大小不一（可以看Notes数），使用时需要注意筛选（它毕竟不是真正属于你的牌组，或许还不如直接使用沪江）。我没有用过别人制作的牌组，所以无法推荐，如果有小可爱试过比较好的欢迎分享~另一点就是，有些共享牌组是包括media，也就是机器发音的，第一次使用的话同步速度可能很糟心。

![](.evernote-content/32895A56-8140-435D-A350-C62FEF842B2D/0CA32DD6-A6F1-49E1-8CE3-6013B9B022BF.webp)

* 类型：这里我是直接修改的自带基础样式：Basic(and reversed card)，也就是创建一次卡片就会自动翻转成两张，一张以正面为问题，另一张以反面为问题。如果目标是快速备考，可以直接选Basic（不制作反面卡片），也就是看日文回答中文；时间足够的话，看中文回答日文有助于写作和口语等输出类的语言学习，根据我自己的经验来看，看中文回答日文过卡片的速度急剧下降……但收获还是不少的。
* 区域： 区域指的就是图中的Front（正面）和Back（反面），可以根据自己的需要增加区域，比如添加一个叫做「例句」的字段，只需要点击「区域...」按钮，然后在新的窗口里添加。这里我为了尽最大可能精简卡片，只保留了最基础的正反面区域，区域太多的话录入新卡片不停「Tab」键跳到下一区域还是很绝望的。
* 样式修改：基础样式只是包括了正反面，接下来我们要针对日语的特色进行简单修改，也就是——假名的显示。点击「卡片...」按钮：

![](.evernote-content/32895A56-8140-435D-A350-C62FEF842B2D/FCF4A03A-A471-406D-AE07-6BE5B359C38C.webp)

先来看Card1，即以日文为正面的卡片。没错，没有多余的任何美化样式，只是在正面加了{{kana:Front}}（正面只显示假名），背面在第一行加了{{furigana:Front}}，即显示汉字+对应假名注音。

我是采用的看假名回想汉字写法和意思的方法记忆，如果习惯于看汉字回想假名读音和意思，把正面的{{kana:Front}}改为{{kanji:Front}}即可。

类似地，Card2的设置如下：

![](.evernote-content/32895A56-8140-435D-A350-C62FEF842B2D/05B81AD6-8DAF-411E-9CC1-BB7AE183EC36.webp)

￼  
大功告成！从此以后不用再管这个样式，所有的卡片都会长成这个样子，是不是超级简单呢？

* Front： 只包括汉字（如果有）和假名，记得将假名用英文的中括号 [] 括起来。
* Back: Back区域我会加一切不清楚或是第一次就搞错的东西，包括：

+ 常用的意思（必填）
+ 动词的话，他动词还是自动词（可选）
+ 例句（可选，很少写）：很少抄原句，而是会用波浪线表示，比如动词「明け暮れる」的例句「研究に明け暮れる」会被总结成「〜に〜」
+ 读音（可选）：第一次念的时候降调位置不对的词才会写上，用辞书上的读音数字标注，注ぐ的话就是0或者2，学日语的都懂吧~
+ 联想（可选）：随时想到的容易混淆的近义词，同音词（比如叶う和敵う都读「かなう」）

* 标签：我把标签用来标注词性，这样可以集中复习一些动词。因为用的是标日，所以标注都是动123这样，更规范的表达应该是辞书上的五段动词、サ变之类。不使用标签录入更快。

文法
==

后期我也用Anki背了一些不熟悉的文法，没有多加任何设置，直接使用Basic类型（如果有写作需求也可以制作带翻转的卡片），正面是日文文法条目，背面是各种解释、接续和例句（如果有必要），特别不熟的加蓝高亮。这里为了省时省力我的接续都是「名词+」表示前接名词，「+に」表示后面接「に」。

**快速录入的小tips**

比起复习卡片，或许录入的过程更枯燥费时，要想快速录入，除了打字速度要快，和像之前提到的用一些特殊的简化符号以外，也要灵活运用快捷键。从第一个区域框开始，填完一个按「Tab」键就可以迅速切换到下一个区域，直到标签栏，最后mac用「command+Enter」，win用「ctrl+Enter」即可一键添加，光标自动移到第一个区域框，这样完全不用动鼠标就可以继续下一张卡片了。

复习卡片：巩固记忆
=========

复习强度
====

录完卡片后，接下来我们要做的就是——不断地间隔复习。由于我设置的卡片内容都非常精简，所以很快就能刷过去，也不需要别的纸笔帮忙。唯一的例外是有一段时间因为每天设置了200张新卡片，总复习达到1000+，实在是记不过来，就采用了一边复习Anki一边抄写复习的时候完全想不起的内容，利用睡前绝佳的记忆时间翻看，正常情况下还是没有必要到这个地步的……

复习间隔
====

![](.evernote-content/32895A56-8140-435D-A350-C62FEF842B2D/6360D533-8FB9-4321-80A0-153B233FA462.webp)

复习卡片时，anki提供了几个熟练度选项，每个熟练度选项对应不同的复习间隔。我自己的选择是：

* 「生疏/错误」对应完全想不起来（并加入单词抄写）
* 「困难/模糊」对应很是费劲才想起来
* 「犹豫/想起」对应很快顺利想起
* 「顺利/正确」对应「我已经太熟练了连说梦话都能说出来」  
  为了快速过卡片，如果使用电脑版，可以直接按空格键，默认选择「犹豫/想起」，或者按数字键「2」，对应「困难/模糊」。顺利的话，一路空格键就可以很快过完当日的一组复习。

高级进阶

作为开源软件，Anki有很多插件可以挖掘，但是下载Anki并不是为了好玩，谨记学习目标，切勿本末倒置。目前我自己在使用下面两种插件：日文的yomichan和英文的anki划词助手，这两种都是支持网页划词一键录入Anki卡片的chrome插件，有很详细的意思、例句和发音，进阶阶段推荐大家使用。

实在嫌弃anki过于朴素的样式，也可以到网上下载各种css模板文件，按照教程就可以做出很漂亮的模板，会自己写css代码就更好了。另外anki还支持填空式卡片，有高阶需求的可以自己多多尝试，作为基础教程这里就不再详细展开了。

最后，Anki也支持录入机器发音，对发音实在没有把握可以用<tts>标签引入机器发音，但这个准不准就不一定了……有兴趣的同学可以自行搜索相关设置。

以上就是我在备考N1时的Anki使用的（朴素无华但很有效的）记忆方法，希望对大家有帮助~对于Anki的使用，我自己还在不断摸索中，可能有些想法做法并不合理，也欢迎大家多多提意见，鞠躬！
