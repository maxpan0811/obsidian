| |  | | --- | | 这是原讨论帖的链接，我把它复制过来给大家参考，同时也可以继续在圈子里跟帖交流。<http://bbs.eduu.com/thread-2811619-1-1.html>   学校无论好坏，只找适合的哦！名气和面子工程固然重要，但要是走到活受罪的地步，那就没意思了哦～   分享2014上实的录取过程，给有兴趣的、需要的粑粑麻麻一个参考，算是回馈e度大促销，嘻嘻。不喜欢的就不要来砸场子了，各位兄弟、阿姐口下留情啦～合适的就是最好的～合适的就是最好的～合适滴～滴滴滴。。。  **★网筛：**  4.15-4.25，10天的网上报名时间。登录上海市实验学校的小学部网址：<http://www.pdysh.pudong-edu.sh.cn/sesxxb/default/default.aspx> 对于这个网筛，真的是无心插柳。因为就在2014对口居委出来之前，我还满心欢喜地以为自家对口的是一般的公立，至少不用太担心是菜场。但一下子发现原来对口的就是菜场！郁闷得不得了。之后因为正好是遇到上实招生，于是对上实进行了进一步的了解（以前了解过），决定有机会表放过，试一下是绝对要滴。于是4.15的凌晨3点多，发出了偶滴简历～  ●关于简历：自我评价用的是第一人称。我是从生活中、兴趣点、时事类、文化学习和专攻几个方面来描述的。文化方面就一句话带过：常规的节奏，没有全部发力，程度中等以上。特长描述：逻辑思维和节奏感自编rap2个方面来描述的。没有写证书。我家是公二幼儿园，父母的学历也不是什么博士那么高大上啦，所以说是看幼儿园和父母学历的，其实并不绝对，这方面大家不用太纠结。对了，还有距离，我家不在浦东，有点远的，所以说看距离也不是最绝对。  **★机考：**  4.28上午出网筛结果。开电脑的时候我的手都是抖的，在点击查询前用卡片遮住屏幕，点击后再慢慢地移动，露出一点点，一点点。。。一下子面对屏幕怕自己的小心脏接受不了不通过的视觉冲击。当忽然看到蓝色的字时心里一开心，再拿开卡片，呀！终于送了一口气，已通过！太太太感谢学校给了这次机会。  ●5.2下午，机考开始～！之前，我停止了一切的练习，每天让孩子弄自己兴趣的东西，放松心态，以最天然的状态去机考。当然除了上实要求的模拟练习，熟悉小键盘的使用。机会一共6次，我家用了4次，一天2次，分2天完成，这是儿子自己安排的。最后一天，也就是5.1因为画宇宙图弄得晚了，模拟练习就没来得及。6次都一样的题目，目的在于熟悉操作。  ●机考半个小时，从进去到出来一共是1个小时。儿子说是在一个大计算机教室，位子是自己自由选择的。小朋友们头上贴着不同颜色的贴纸，上楼的时候按颜色开小火车的，到机考教室里就打乱了。考好后出来，粑粑麻麻们接的时候又是按颜色的了：分批进入大门，组织得很有序，1200+的人，一点都不乱，不吵闹。  ●还是带一笔机考吧，可能大家也比较关心。我家的说：95左右的题目，29分××秒完成，做到D，双组合，后面的CD组合很多也很难，他很想做到E，但实在太多太难了，怎么也做不到E。帮助键用过2次。儿子描述的时候边说边笑，他感觉很滑稽，也不知道他在滑稽什么，总之做得很开心就好。  **★二面：**  5.6下午1点45左右，二面出结果。和上次一样，同样是一张卡片遮电脑，慢慢移。还是蓝色的字，不过这次机考把握还是大的，心不是那么紧张，觉得儿子应该没问题，从小对这类比较敏感些。  ●我家在外面长期的兴趣班除了gym和Ris，没有上过思维课，就是中班的时候自己家里弄过一年的逻辑思维，做做题目书和百花12册，学而思的综测也没有参加过。零零碎碎的暑期班，参加过拼音识字、看图说话、街舞、儿童画、小主持人的班，是很随意的那种，那年超高温的暑假，还逃了很多次的课。大班这一年，家里完全放羊了，因为儿子完全沉浸在自己的兴趣里，想想以后读书没什么时间，我也就支持了。我们画过一本简单的介绍宇宙的书，还有无数的上海地铁图、小区示意图、太阳系图、中国和世界地图等等，我挑了一些比较好的装订成册，想以后大了给他看看蛮好玩。除了这个特定的兴趣，其他的我家接触得很散，没有集中的去弄一样，都是一阶段一阶段的，比如有段时间玩彩泥就一直玩玩玩，然后彻底抛弃，偶尔才玩一次。一段时间热衷于自己录音反复的录了听，听了雌笑。曾经有段时间还一直在种花花草草，种了死，死了种，种了再死，无穷无尽～所以感觉铺得太开了，有点不好收场的意思。。。还有点无始无终的味道。  ●5.8下午二面，我们是B组。登记好（查看户口本和孩子的证书、兴趣作品展示，当然我家没有证书）之后孩子在楼上活动，我们就在厅里听校长做报告，非常有启发，对学校也有了更进一步深入的了解。二面之前e度上比较有讨论的是关于原登记号之前的英文字母的含义，这天知道了，是英文单词的第一个字母而已。B就表示是boy组。二面好像共有2批，分8个组，每批4个组。ABCD一批，EFGH一批。是按照报名号顺序排列的，没什么特别的意义，事后证明2批里面都有录取的。  ●二面差不多1个多小时。内容基本和民办的面试差不多。有做运动（每组不一样，我们是双脚夹球跳，有的是乒乓球拍传乒乓球不要掉下来）、搭雪花片、听故事回答问题、知识类的问答（解释：山谷是什么？我家说是2座山之间，老师说用手表示一下）等等。其中还有个内容就是自主学习后做题目：学习线段和射线，A点、B点、C点。。。到家后他还教了我一遍，我突然发现他的表达能力一下子提高了很多。儿子说每个小朋友看一台机器，然后语音总的播放，之后就是做题目（选择题）。结束后，小朋友按组依次进入报告厅，手里拿着礼物：三只荧光笔，由大人接走从另一个门出去，还是安排的很有序的。在出门的路上，儿子问：妈妈，要是考不上怎么办？我说：考不上就说明不合适，表想那么多。说到荧光笔，我们说是让大家回家后努力学习～后来去面民办发的都是吃的，儿子说：是让我吃得饱点，哈哈！  **★录取：** 5.9上午10点之后公布结果。这一个晚上心里不知道乱七八糟什么感觉，又肯定又否定又心定又不安。。。对口菜场伤不起。而且就像很多家长说的，一开始并没有期待，但随着一步步过关斩将的逼近，反而有期待了。凌晨睡的，原本想一下子睡过10点，没想到9点不到就醒了，更烦心。这次最紧张，不敢看，直到11点才登录查看。总算是有惊无险，天地保佑～！真的没想到过，就像做梦一样，无心插柳柳成荫，祖宗的话不无道理。非常感谢上实肯定我家的兴趣教养方法，当然进去上学后自然要以文化知识为主，这是后话，这里就不展开讨论了。   我每次总归最后要说一句的：十全九美，春夏秋冬，阴晴圆缺，悲欢离合，黑夜白昼。。。相对论。。。**世上没有最好的，只有最合适的。**大家淡定～   最后，恭喜2014所有进入心仪学校的小朋友！预祝2015及以后的小朋友，大家都开开心心、顺顺利利升学快乐！ | |
