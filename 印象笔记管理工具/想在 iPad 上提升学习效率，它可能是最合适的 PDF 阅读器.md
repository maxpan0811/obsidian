# 想在 iPad 上提升学习效率，它可能是最合适的 PDF 阅读器

---

去年的考试复习课上，老师「仁慈」地给我们透露了考试范围：一份长达 64 页的 PDF 和包含了 100 道题的问题集1 。如何在短时间内快速消化这份 PDF，就是摆在我眼前的难题。在 PDF Expert for iPad 上阅读这份 64 页课件，尽管在体验上已经不输纸张，但是也仅仅是免去了打印的麻烦。媒介变了，但 iPad 和 PDF Expert 并没有改变学习的流程和效率。

MarginNote Pro 的开发者 Min 也意识到了这个问题，iPad 平台上现有的 PDF 阅读器并没有发挥设备强大的计算能力和交互特性，来「革命」学习者的学习体验。于是他花了两年的业余时间，开发出了 MarginNote Pro。

> 出于一个 Coder 的天性，我尝试打造一款适合自己需求的阅读工具（因为当时市场上完全找不到一款合用的），就这样 MariginNote 开始了。
>
> —— Min，MarginNote Pro 开发者

MarginNote Pro 的外壳是 PDF 阅读器和笔记应用，内核则是串联了**批注、思维导图和记忆卡等学习方法，来帮助你将书本「吃透」的工具。**装有 MarginNote Pro 的 iPad 不再只是 PDF 的载体，而会成为你学习流程中的核心。

初见 MarginNote Pro，它怎么能用来「学习」
----------------------------

学习者有自己的习惯和节奏。传统的 PDF 阅读器并没有从学习流程这个场景出发，而只是复刻了我们对纸质资料的使用习惯。MarginNote Pro 同样作为一款 PDF 阅读器，该如何去连接**学习与 PDF 阅读**这两个场景呢？对「笔记本」和「卡片」这两个概念的提炼是它的答案。

### 笔记本

在传统的学习过程中笔记本是不可或缺的角色。面对完全陌生的资料先把书读薄，将书本上的信息浓缩成自己的知识，这就是笔记本存在的价值。在 MarginNote Pro 的底栏中就鲜明的把笔记本单独摆出来，每导入一份 PDF，应用就会自动给你建立一本笔记本。现实中一本笔记本不会只是针对一份资料，通常是会对应一个科目或者项目主题所有的资料。 MarginNote Pro 中同样鼓励你将多份 PDF 关联到一本笔记本中，分散在 PDF 中知识点就这样被串联了起来。

![](.evernote-content/1D402603-70F8-49E5-AC42-1DC232F03C2E/FE709E05-25B2-46F6-9762-8822CB21B59E.png)每份 PDF 都自动会有一本笔记本，笔记本「学习 exercise3.1」关联了三份 PDF

### 摘录卡片

那笔记本里的内容该是什么呢？纸质时代我们会在书本上直接写批注、划重点，在笔记本上写心得和摘录重要知识点，在 MarginNote 中可以延续这样的习惯。给段落划线，用高光笔涂重点句子，还有直接在 PDF 上手写笔记。对于公式或者图片这种不能用标注方式来选中的内容，方框和套索就显出样式上的便捷。在 MarginNote Pro 中**每一处标注的内容都会变成一张单独的卡片**，显示在笔记本中。开发者为什么要这么设计呢？李敖曾这样介绍自己读书的方法：

> 读书方法好才能把你念过的书主要的部分留存下来。看完这本书就大卸八块、五马分尸……心狠手辣，剪刀美工刀全部剪下来，把这本书分尸掉了。这一页我需要、这一段我需要。把它分出来了，排出来。

MarginNote Pro 中的卡片就像你把书中的内容「剪」了下来，放进你的笔记本。而且不仅仅是简单的「剪」，你还可以进一步对摘录的卡片添加语音、图片、标签甚至手写的备注。书本变成卡片，卡片经过你的消化进入笔记本，一本书就这样被「吃透」。

![](.evernote-content/1D402603-70F8-49E5-AC42-1DC232F03C2E/38DD9975-9B75-46D8-A3EC-3F1EEC0D9382.png)在 PDF 中的高亮、批注都会变成一张卡片

学习纸质书时这些标注通常分散在书本、笔记本和便签中，而在 MarginNote 中统一的以卡片的形式收集在笔记本中。我多次为了找一个小结，在书本和笔记本中翻来翻去，而在 MarginNote 只需要在笔记本中，通过内容或者标签搜索即可。这种安全感也是相比纸质难得的优势。

### iPad 上 PDF 阅读的痛点

但是谈论及在 iPad 上 进行 PDF 学习时，有众所周知的痛点不能回避：没有纸质书独有的「手感」，不能凭肌肉记忆快速定位到想要的页面。而在学习常常需要回看之前的知识点，不解决这个问题，在电子屏上的学习体验就不可能做到和纸质教材同样的效率。

一般的阅读器，例如 PDF Expert，都会把 PDF 结构化来解决这个问题，大纲、收藏页和根据页面跳转这三板斧是最常见的功能。但是大纲太过粗略，收藏页不够有条理，那根据页码跳转？居然指望我们记下几百页 PDF 的内容和对应的页码？

![](.evernote-content/1D402603-70F8-49E5-AC42-1DC232F03C2E/A8B21852-B2D7-4166-99D9-C5406F47A1EF.gif)利用大纲来快速在 PDF 中跳转，找到相关的知识点

MarginNote Pro 的策略则是利用卡片，**卡片既是书本中的一部分，又是你脑中对书本内容的一个「锚」**。回想实际的学习经历，需要回看的肯定是有重要知识点的位置，而在之前的学习中，你就应该已经对这个知识点进行摘录，生成了卡片。相比依靠「手感」来估计想回看的知识点的位置，在 MarginNote 中的「大纲模式2 」寻找和跳转要来的方便和实用的多。

用 MarginNote Pro 来应对一场考试
------------------------

手写标注、思维导图和记忆卡片，无论哪种工具都不是 MarginNote Pro 独有的功能，而且每一类都有功能更强大的专门软件。这些功能糅合在 MarginNote Pro 里，是会变成臃肿的功能堆叠，还是互相作用发生化学反应呢？在刚过去的一次备考中，我只用 MarginNote Pro3 完成了复习任务，也找到了问题的答案。

### 划重点

用 MarginNote Pro 复习的流程其实和用纸质资料非常接近，**只是在每一个环节上都免去了不少与复习本身无关的杂事。**我搜集要复习的 PDF 到分类好的文件夹后，先是阅读资料。将重点知识用高亮或者套索划出来，用 Apple Pencil 直接在 PDF 上批注要比键盘输入自由和快速的多。 初次阅读时，可以将各级标题也划上下划线，使其也成为一张卡片，方便之后在整理思维导图和制作记忆卡片。

### 做导图

不论是老师讲课用的幻灯片，还是 PDF 的资料，其实逻辑是非常清晰，很适合用思维导图来整理的。我利用之前生成的摘录卡片，根据知识点的层级关系稍加整理，就理出了一份 PDF 的知识脉络，梳理出单个章节需要重点复习的知识点。免去了用纸笔将知识点再抄一遍的麻烦。

![](.evernote-content/1D402603-70F8-49E5-AC42-1DC232F03C2E/6FFD668E-FF9B-4B7F-A805-E97A2E5D63B1.png)其中一份 PDF 整理好的导图

### 背背背

当然备考时最重要的还是对知识点的记忆。并不是所有的摘录卡片都需要记忆，有的卡片也还需要额外补充内容。所以记忆卡片前需要再对重点卡片进行加工。之前整理出来的思维导图使卡片有了上下级关系， MarginNote Pro 默认将上级作为问题，下级作为答案。但答案可以再添加录音、批注等，问题可以多选卡片进行组合。和之前一样，这一环省去了重新制作一遍卡片的功夫。除了熟悉的记忆卡片，MarginNote Pro 还提供了「回忆模式」进行复习：回忆模式下，PDF 中被划重点的部分都会打上马赛克。在上下文中进行回忆，也不失一种有效的辅助记忆方法。

![](.evernote-content/1D402603-70F8-49E5-AC42-1DC232F03C2E/A095CE6A-F4B4-4FC9-8B0E-6B9F4BBD20CA.gif)在 MarginNote Pro 里记忆卡片

### 为什么我不导出卡片到更强大的应用中

很多 MarginNote Pro 的用户很推崇将思维导图、记忆卡片导出到 Anki 和 iThoughts 这两个应用中。诚然，在 Anki 中进行记忆卡片的学习，和在 iThoughts 中去整理导图，功能性上都会强大的多。但是，这就失去了 MarginNote Pro 的魅力。

在 MarginNote Pro 里，我不再线性的一遍又一遍重头到尾的啃资料。第一遍阅读完制作好思维导图和卡片后，就利用大纲来跳跃性的去聚焦资料中的关键点（即摘录卡片的位置），将知识放在上下文中反复的熟悉 ，远比单独摘出来去背要有趣的多。而导图则不断加强我对这份资料的条理上的熟悉，能清晰的发现我的薄弱点所在。这种**在纸质资料上无法实现的流程**，才是 MarginNote Pro 的价值所在。

### 在复习中让人温暖的小功能

MarginNote Pro 利用 iPad 交互和计算上的优势，集成了很多让人在学习过程中会用到的小功能，乍看不起眼，实际使用中则感受到了开发者的温暖。

![](.evernote-content/1D402603-70F8-49E5-AC42-1DC232F03C2E/4DF2F451-D46A-4F97-9F47-8B509974242A.gif)查询字典和维基百科

最实用的当属点击单词，即可选择查看词典和在网络中搜索。前者不必多说，后者则可以直接在查询维基百科中搜索被选中的单词。尽管 iPad 有了分屏模式，但是不用跳转到别的应用，就能查询到的体验，实际使用时会比跳出查询便捷的多。类似的小功能还有分割书视图，最多同时看一份资料看三处，让我想起用手指当书签，翻来翻去对比看的读书体验。

对比 PDF Expert 的差异
-----------------

目标人群的不同造就了 PDF Expert 和 MarginNote Pro 在功能设计上的差异，进行让两者在使用场景上有了明确的区分。

MarginNote Pro 的理想用户是那些**要将一部专业书籍进行消化吸收的学习者**，于是有了卡片的概念，融合了大纲和思维导图等等元素。在设计上不可避免的稍显凌乱，功能栏的排布也显得「随心所欲」，不如 PDF Expert 显得条理清晰。PDF Expert 则依然是最通用的 PDF 阅读器，iPad 上的审阅模式和 Mac 上的编辑功能都是为了更商务的场合而设计。

而在软件质量上，MarginNote Pro 作为一款独立开发者的作品，自然很难公平的去和 PDF Expert 进行比较。Apple Pencil 在 MarginNote Pro 上稍显迟钝，不如 PDF Expert 中行云流水。MarginNote Pro 很多地方采用了分栏的设计，但在改变栏宽时，都会显得有些卡顿。

关于是否该尝试 MarginNote Pro
----------------------

不同于那些新奇好玩的应用，MarginNote Pro 设计上并不出彩，操作上也有些古板。但是瑕不掩瑜，这样一款完全为专业学习者而打造的利器，能让我一天中 80% 的专业学习时间提升效率，而不是去填充我 20% 的娱乐时间。从这个角度来看，MarginNote Pro 在学习 PDF 这件事上，理念上的突破远比应用本身来的重要。

或许看完这篇，你还是会本能的觉得长久以来习惯的纸质资料更靠谱。但想想牛顿时代数学世界最尖端的微积分，如今只是每个理工科专业的的基本必修课。知识的广度和深度在高速发展，学习的方法和工具也要相应的得到发展，不能简单的因循守旧，抱着自己已经习惯的学习方法和工具不再加以改进。

MarginNote Pro 当然不是一颗灵丹妙药，不能帮你三天突破英语写作，二十一天学会 C++。但有这样一个工具在，何不尝试将它融入在你的学习流程中呢？
