**点评**

1

模拟题与真题的关系

上实以其第一次面谈题库的神秘而闻名，因此每年的模拟题（通过网筛的小朋友即可得到一次模拟练习的机会）都会被家长拿来反复推敲和演练。我们以去年和今年为例，来看一看究竟模拟题在多大程度上可以代表真题。 

（1）模拟题出题类型 > 真题出题类型

模拟题中未出现的类型，真题中不会出现，如往年必考的ABC规律，今年真题中没有出现。而模拟题中出现的题型，在真题中有可能出现，如分类题；也有可能不会出现，如主题发表题。

（2）模拟题出题内容 < 真题出题内容

题量比较表

|  |  |  |
| --- | --- | --- |
|  | 2017年 | 2016年 |
| 真题 | 166 | 293 |
| 模拟题 | 21 | 52 |

从题量比较中，我们可以清晰地看出内容方面模拟题不能给我们太多的启示。由此我们认为模拟题的作用，主要在于帮助小朋友熟悉出题形式。它可以在一定程度上预告上实真题的动向，但在深度上无法涵盖所有内容。

因此我们建议家长重视模拟题的类型，但在内容上应当跳脱出模拟题的限制。

2

2017年新动态

（1）新增分类题型，所占比重较大

出题内容扩展到了日常生活用品、日常生活作息、表情认知、对错是非常识等方面。

（2）图形找规律是主要考察点

图形找规律类题目涉及的内容仍然很广，有图形旋转、图形重叠、图形分类、图形的色彩变化等。但是没有给规律命名如“A、B、C”等，也不需要一直记忆前面的规律。

（3）模拟题前面有详细的指导说明和作答策略

对于是否使用“放弃”键及正确率与速度哪个优先等热点话题给出了官方诠释：一是如果孩子不会，尽量不要猜测选择，可以选择“放弃”。二是总的作答原则是在正确的前提下，越快越好。

（4）题目数量

今年参加面谈的小朋友有做五十多题的，也有做了一百多题的，他们普遍觉得题目仍然很难。
