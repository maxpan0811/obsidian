# i7-7700K+双路GTX 1080 华硕发布ROG怪兽游戏主机

---

![](.evernote-content/40952D57-BF4D-4B00-A504-B5BB34B3EF8A/874565EF-32F6-4958-9AC3-188EBF63FB49.png)

上周 CES 2017 国际消费电子展会上，PC 还是大会的主角，而且越来越多强大的 PC 陆续亮相。在游戏玩家心中颇有地位的华硕也不甘落后，正式推出了 ROG 玩家国度系列的最新高端台式机 GT51CH，玩家可以根据自己的需求定制一系列高端的组件，包括英特尔第七代 Kaby Lake 微架构的处理器，以及顶级的显卡型号。

华硕 ROG GT51CH 最高的定制配置非常强大，处理器最高可选最新支持超频的七代酷睿产品 Core i7-7700k，显卡最高可以选择两块 GeForce GTX 1080，组成 SLI 的双卡配置，内存最高 64GB DDR4 2800MHz，储存方面可选多种组合，可选 NVMe PCIe 固态硬盘，或者 HDD 机械硬盘，支持组 RAID 0 提升储存性能。

![](.evernote-content/40952D57-BF4D-4B00-A504-B5BB34B3EF8A/8D57E399-FB33-4219-9818-1043569768E1.png)

可能单看配置一般发烧玩家也能自己组装，但华硕提供了一系列 ROG 的技术。例如针对处理器，华为配套的 Aegis III 软件，可以将 i7-7700k 全核超频至 4.8GHz，按下 Turbo 按键就能一键将 CPU 的四个物理核心超频，并且还不需要重新启动系统，确保稳定性的同时，大幅提升游戏体验和多任务处理性能。

另外，GTX 1080 显卡和内存也都可以通过配套工具实现超频，打造更顺畅且无卡顿 4K UHD 游戏体验。对于那些有更高超频需求的玩家，自然也可以自定义微调超频，通过内置的华硕系统级监控软件 Aegis 查看温度、风扇速度、电压阀值和预警界限等大量系统信息进行调整即可。

![](.evernote-content/40952D57-BF4D-4B00-A504-B5BB34B3EF8A/B1EE111D-8BAB-4BF4-AC44-48AF3A31E94B.png)

ROG GT51CH 的外观很酷炫，根据空气动力学灵感设计，有锐利的多边形棱角边缘和光滑的曲面，底部是标识性涡轮状进气口核心设计，并提供 ROG 标志、前面板的两条灯光带动态光效定制。内部的散热系统也很先进，提供 12 路风冷散热通道与水冷散热的搭配。机身上设有 USB Type-C 接口，内置 ESS Sabre 耳机 DAC，华硕称这枚 DAC 可以确保高品质的 Hi-Fi 音质输出，带来高动态声音范围（DNR）和更低的噪音。该主机也有文件加密功能，只有佩戴 ROG 手环才能访问隐藏的文件，保护机密数据。

ROG GT51CH 目前暂时还没正式上架，华硕官方预计 2017 年第二季度发售，价格至少从 2 万元起步。
