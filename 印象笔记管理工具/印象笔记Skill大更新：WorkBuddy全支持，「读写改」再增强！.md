# 印象笔记Skill大更新：WorkBuddy全支持，「读写改」再增强！

---

来源：[打开原文](https://mp.weixin.qq.com/s/AdJiNUBBpmkEosuErKOPog)

![](.evernote-content/9BAB8F38-F025-44BA-B78C-4A69AFD3A362/48A9F4C6-4836-49D1-A618-F7E4DD0B8B5C.png)

这段时间很多老朋友都希望「印象笔记Skill」快点实现创建笔记本、笔记这一类功能。

🐘 帮大家催来了！

以下是印象笔记Skill新版本的「新增」能力：

![](.evernote-content/9BAB8F38-F025-44BA-B78C-4A69AFD3A362/D1C974E7-97E0-480D-B474-8657E5A87600.png)

在已经有的能力基础上，这次新增的是

「给某一条旧笔记更新内容，为笔记创建标签，按照创建时间、标签或笔记本来查找笔记，单独或批量创建笔记本」

，并且优化了搜索笔记的质量体验。

以 WorkBuddy 工作台为例：

![](.evernote-content/9BAB8F38-F025-44BA-B78C-4A69AFD3A362/C5A89248-5271-4318-A04B-7A7543E06DB5.jpg)

一次性输入「创建笔记本等内容+标签管理」的指令，即刻帮你完成记录工作。

![](.evernote-content/9BAB8F38-F025-44BA-B78C-4A69AFD3A362/A6450265-1483-4303-B1BB-5C4817B1BB01.jpg)

此时，打开印象笔记，对应的笔记本、笔记、标题和标签，都已经「准确显示」了。

![](.evernote-content/9BAB8F38-F025-44BA-B78C-4A69AFD3A362/B53B7A3A-5350-4436-B07B-25E9DFDE43DD.jpg)

创建、管理笔记的体验，大家可以在更新印象笔记Skill之后，在自己习惯的 Agent 上实时试用。
