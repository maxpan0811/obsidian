# watchOS 3新特性——医疗

---

watchOS 3这个功能一定要了解 ,因为关键时刻，可能它会救我们一命。它就是 "Emergency SOS"。  
  
　"Emergency SOS"其实很简单，它会在危急时刻帮助您报警和通知亲朋好友，按住 Apple Watch 的侧键，我们将会看到选择出现，其中两个就是 Medical ID 和 SOS 。

![](.evernote-content/061C204C-3430-4A0F-881E-6ADDE1155390/DCFDEA36-43EC-4C21-8B74-C7003F84328D.png)

让我们着重说一下 medical id 。在 watchOS 3 的  medical id 功能里，用户的医疗信息数据将会通过手机上的一个健康应用程序自动与 apple watch 共享，这些信息包括佩戴者的姓名，出生日期，身高体重血型等等，当然，它还会标识你是否是一位器官捐赠者。而医务人员可以在需要的时候访问一些关键的信息。

要知道有时候当我们遇到突发情况时，如果医护人员可以第一时间了解到我们相关健康信息，这也有助于施救者为我们提供帮助，万一真的出现突发情况，对于施救者和被施救者来说，真的必须要争分夺秒！
