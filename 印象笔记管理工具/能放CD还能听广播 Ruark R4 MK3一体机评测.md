# 能放CD还能听广播 Ruark R4 MK3一体机评测

---

[![](.evernote-content/5D52DBA3-9E05-40F3-B94C-7DA29A1C73A2/F91875F7-7985-4F32-B78B-6A5FF75DBB80.jpg)](http://tech.feng.com/2015-11-19/Can-put-CD-can-also-listen-to-the-radio-Ruark-R4-MK3-all-in-one-evaluation_630578.shtml)

　蓝牙音箱的评测我们已经看了不少，“一体化音响系统”这样的东西还真有些少见。事实上如果你愿意多出些钱，一体化音响系统这种集合了 CD、广播、蓝牙音箱的设备并不比单纯的蓝牙音箱差，功能还更加强大。  
  
　来自英国的 Ruark Audio 旗下几乎全部都是这样的多功能音频设备，而最新的 R4 MK3 更是其中的佼佼者。拥有简洁好看的外形，你绝对看不出 R4 MK3 里头有那么些功能。那么我们今天就通过评测，看看这种有趣的一体机都能做些什么吧：  
  
　**设计与控制**

![](.evernote-content/5D52DBA3-9E05-40F3-B94C-7DA29A1C73A2/F91875F7-7985-4F32-B78B-6A5FF75DBB80.jpg)

与前代产品相比，R4 MK3 在外形上有了很大的变化，不过仍旧继承了古典风的设计思路。音频处理部件隐藏在胡桃木制成的外壳中，更别提里头还有一个性能强大的甲乙类功放了。

R4 MK3 外形低调古典，外观应该是致敬了老式的收音机。在外壳之下，一对 3.5 英寸的驱动单元将吸入式 CD 光驱夹在中间。除了光盘吸入口，在机身正面上你还能看到音频接口、电源键和 OLED 显示屏 —— 当然了，显示效果也不会多么华丽就是。

虽然看不到，但 R4 MK3 的底部是藏着一个 4 英寸低音炮的。所以它看似非常文静，真要嗨起来估计也一样给力。

![](.evernote-content/5D52DBA3-9E05-40F3-B94C-7DA29A1C73A2/3B1288B7-D243-432E-84AE-E7154958F1BF.jpg)

被放在机器顶部的是 Ruark Audio 特有的 RotoDial 转盘型控制台。围绕在中央旋钮四周的是诸如播放/暂停，切换曲目这些按键。转动旋钮可以调节音量，也可以通过 OLED 的辅助来选择菜单。

Ruark Audio 为 R4 MK3 配备了遥控器，其实后者完全可以取代 RotoDial，不过多一种操作方式肯定不会是坏事。

总的来说，R4 MK3 的设计和操作非常简单直观。要上手它的话，你并不需要什么复杂的适应过程。

**音源与接口**

![](.evernote-content/5D52DBA3-9E05-40F3-B94C-7DA29A1C73A2/A163B9CC-9654-4DE8-B482-B5286E327A1F.jpg)

R4 MK3 支持普通 CD 光盘，也兼容那些里面刻有 WMA、MP3、AAC、WAV 文件的数据光盘。R4 MK3 支持 FM、DAB、DAB+ 等广播制式，能够适应未来广播的需要。

如果不喜欢光盘，R4 MK3 也支持 USB 播放 MP3、AAC、WMA、WAV 音频文件。非常不幸的是，它并不支持 FLAC、ALAC 等无损文件。

R4 MK3 拥有蓝光 aptX 音频压缩芯片，意味着它能够在保证音质的情况下无线播放音乐。

**音质表现**

![](.evernote-content/5D52DBA3-9E05-40F3-B94C-7DA29A1C73A2/AC779773-49E5-4AEE-9FC9-5A71B6B569FD.jpg)

外观美功能多对于一款播放机来说只能算是加分元素，重头戏还在音质表现上。在这一点上，R4 MK3 表现得还算不错。它总体上算暖声，但通透性足够，声音并没有因暖而变糊。

R4 MK3 的声音细节很好，人声突出但不刺耳。它的声场表现很优秀，内置的 3D 音效模式颇有一种声音充满室内的效果。可惜的是，如果音量太大了，你很容易就能注意到声音的失真。

和传说中的“齐柏林飞艇”比起来，R4 MK3 还有不小差距，特别是其低音的震撼度较之那些优秀产品还差许多。不过，R4 MK3 的声音足够准确，大多数人不会再追求在此之上的音质了。

**总评**

![](.evernote-content/5D52DBA3-9E05-40F3-B94C-7DA29A1C73A2/CE9363E8-5685-4A27-9A2D-3AEEA55B37C5.jpg)

R4 MK3 外形低调，声音均衡，是那种适合放在书架子上的播放机，而非齐柏林这样的张扬性。如果你喜欢收藏 CD，对音质的要求又不到发烧友的那个地步，这款设备确实很适合你。

不过和普通的蓝牙音箱比起来，R4 MK3 要贵上不少，而且也不好携带。并不是所有人都喜欢听 CD，或是专门下 WAV 格式的音频的，所以 R4 MK3 注定只能是一款相对小众的音乐播放设备。
