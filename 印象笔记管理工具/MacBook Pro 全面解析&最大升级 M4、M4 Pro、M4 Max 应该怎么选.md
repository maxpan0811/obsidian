MacBook Pro M4终于发布了

还带来了三款不同的处理器

M4、M4 Pro、M4 Max

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/D888D1A8-9E85-41AB-8137-6E28AD7C0EE1.png)

而且也有14寸和16寸两个尺寸

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/75C5F486-A85A-455C-ADCA-EBEBD83D7617.png)

虽然iPhone16在挤牙膏，但MacBook几乎是放大招了

这次带大家了解下这款MacBook Pro M4、M4 Pro、M4 Max这件的区别

**颜色**

科技以换壳为本

这次的MacBook Pro带来了全新的黑色-太空黑

还有黑色的数据线

黑色的MacBook 看起来专业感十足

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/CDB6B11D-4162-4623-B135-91AD28E3A8D9.png)

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/951123ED-7B9F-45B4-B509-858C98553B31.png)

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/7F35C8A3-1F4F-45AD-A616-B506FAA79267.png)

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/6A2A4E21-66A2-4173-A67D-AAF68E2D41A8.png)

**接口**

这次MacBook Pro M4的接口和MacBook Pro M4Pro 和M4 Max一样多的数量

机身的右侧多出了三个USB-C的接口

使用起来更方便的

不过M4 Pro、M4 Max使用的是雷电5接口

M4仍然使用的是雷电4

但是对于大多数一般用户来说、雷电4已经就足够实用

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/C43E391B-EA1A-4C05-9863-35FBE581B47A.png)

**性能**

性能上M4提升很多

特别是多核CPU和GPU，提升比去年的M2到M3更大

而且还免费升级到了16GB内存

带宽也提升了20%

但价格不变！

其实是非常超值的

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/AF3C42EB-2DC0-4BD1-8868-E9E21639BB6D.png)

**屏幕**

新款的MacBook Pro亮度提升到了1,000尼特

在户外也能够更清晰的看清楚屏幕了

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/A11A12AE-D865-42B5-AEE1-4252B22CD1DC.png)

而且也和iPad Pro一样提供了一个Nano texture（防反光玻璃）

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/9B50CEE4-1738-4F35-B53D-7C7BAA7AE220.png)

外接屏幕的性能也有所提升

M4现在可以外接两个6K、60HZ的屏幕

已经和M4 Pro的版本是完全相同

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/CC76129F-CE3E-4710-9B14-4592CFCD639E.png)

**摄像头**

摄像头也提升到了1,200万像素的超广角镜头

同时支持人物居中和桌上视角这两个功能

人物居中：四处走动时，人物居中功能会始终将你置于画面中间。

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/080DAE39-E64D-4009-BE38-767E2B5B5D51.png)

桌上视角：你能将自己的工作台面展示给对方看

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/F05596EF-08EF-47A3-8351-4F3C46B9C9EB.png)

**续航**

MacBook Pro M4播放影片可以连续播放24个小时

续航和16寸MacBook Pro M4 Pro的版本

已经是差不多的了

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/909DBF3F-65DC-4BD6-9254-A3EF162BAE5A.png)

综合来看

MacBook ProM4的在配置上已经很高了，性价比非常出色的

除了CPU和GPU其他配置和MacBook Pro M4Pro、Max几乎是一样的

M4Pro

M4 Pro的特点有两个：

1、CPU

M4Pro基础版本就比M3Pro和M4提升很多

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/0B4E95B4-5666-43B4-AC70-FB2C5766F26A.png)

M4Pro的满血版=M3 Max和M4Max的基础版本

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/2529C536-4458-49B3-8D3E-2D12B72A1C48.png)

2、内存带宽

M3Pro的内存宽带是150GB/s

M3Max的内存宽带是300GB/s

M4的内存宽带是120GB/s

M4Pro的内存宽带是273GB/s

M4Pro和M3Max已经非常接近了

如果你是从事编程、软件开发、大数据

那这个M4 Pro芯片的MacBook Pro版

更加适合你

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/ECE8B6D0-012B-4AC2-8A0F-FDB5BEFC5E20.png)

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/BAF2642F-84F1-4259-8007-355E516F056C.png)

**M4 Max**

如果你只需要强大的CPU，不需要更好的画质性能，M4 Pro已经就足够了

没必要入手M4 Max了

M4 Max优势在于：

1、GPU性能：

M4的GPU有10个核心

M4 Pro的GPU最多有20个核心

M4 Max的GPU最多有40个核心

如果要从事3D建模、3D渲染、深度学习、游戏开发这一类选择M4Max比M4Pro更加好

2、内存

M4 Max可以选择128GB的内存

而M4 Pro只能选装48GB

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/1852396D-6BB5-49E7-AD61-2F75DFC02942.png)

3、视频剪辑

视频剪辑主要依赖媒体引擎来完成的

M4 Max比M4Pro拥有两倍的媒体引擎

不论是剪辑的流畅度还是导出的速度

都会超过M4 Pro和M4

![](.evernote-content/61D3E929-0BBA-44FE-BA6F-2FB6D4B0B414/BAF2642F-84F1-4259-8007-355E516F056C.png)

如果需要剪辑大型的8K视频

或者是4K 120FPS的视频的朋友

那这个M4和M4 Pro呢

就有一点儿力不从心了

M4 Max才是最好的选择
