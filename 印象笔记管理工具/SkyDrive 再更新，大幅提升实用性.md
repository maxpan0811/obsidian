# SkyDrive 再更新，大幅提升实用性

---

[![](.evernote-content/0D756E8F-349B-4B8E-9BD1-BC8257D6DB80/99944AFE-55D7-4CD7-80F5-83ED57F43592.jpg)](http://www.ifanr.com/wp-content/uploads/2012/04/Skydrive-Logo.jpg?9d7bd4)

去年年末，微软在宣布 Windows 8 要开启“个人云时代”后，对 HTML5 版本的 SkyDrive 进行了[重大升级](http://www.ifanr.com/62932)，随后在 12 月推出了 Windows Phone 及 iOS 版本的 [SkyDrive 手机应用](http://windowsteamblog.com/windows_live/b/windowslive/archive/2011/12/13/introducing-skydrive-for-iphone-and-windows-phone.aspx)，之后又在 2 月份的时候推出了 [Windows 8 版本的 SkyDrive 客户端](http://blogs.msdn.com/b/b8/archive/2012/02/20/connecting-your-apps_2c00_-files_2c00_-pcs-and-devices-to-the-cloud-with-skydrive-and-windows-8.aspx)。

今天，微软又有新动作，发表[博文](http://blogs.msdn.com/b/b8/archive/2012/04/23/the-next-chapter-for-skydrive-personal-cloud-storage-for-windows-available-anywhere.aspx)向我们介绍了更强大易用的 SkyDrive。

### 令人满意的桌面客户端

在 Windows 8 版本的 SkyDrive 推出两个月后，微软终于推出了支持 Windows 7 及 Vista 的桌面客户端，而令人意外的是微软也同时推出了 Mac 版本的客户端。

在 SkyDrive 推出桌面客户端后，它也可以像 Dropbox 一样与桌面系统的文件管理器无缝结合，为用户提供方便快捷的云同步。微软在博文中强调团队在开发桌面客户端的时候重点考虑了三个问题：快捷、安静、稳定。

经过短暂的试用，我认为微软的确做到了这几点。首先整个设置的过程非常快捷，你所需要做的基本就是下载、安装、登录、同步；其次在同步的过程中，SkyDrive 确实很“安静”，所有的同步操作都会在后台完成，你不会看到恼人的通知栏气泡；再次，SkyDrive 的同步速度令人惊喜。

### 储存空间的调整

本次更新过后，微软将 SkyDrive 新用户的免费储存空间从之前的 25GB 降到了 7GB，因为微软经过调查发现，99.94% 的用户真正利用的空间都在 7GB 以下，而本次免费储存空间的调整旨在不影响用户体验的前提下节省开支。不过从用户层面来讲，储存空间自然是越大越好，好在微软为 SkyDrive 的老用户（曾经上传过文件）仍旧保留了使用 25GB 免费储存空间的权利。**需要提醒的是，就算是老用户，也并不是直接开通25G的空间，而是要在限制的时间内从网页端认领。**

微软的桌面客户端还提供了上传大文件（最大 2GB）的特性，此外还推出了付费空间计划，收费如下：

* 20GB，10 美元/年
* 50GB，25 美元/年
* 100GB，50 美元/年

相比 Dropbox 50GB 的付费账户需要 9.99 美元/月或 99 美元/年，100GB 的付费账户需要 19.99 美元/月或 199 美元/年，微软的 SkyDrive 可谓称得上“白菜价”。

### 移动客户端小幅更新

微软还为 Windows Phone 及 iOS 平台的 SkyDrive 带来了小幅更新，本次更新添加了删除、重命名、移动文件的功能，还新增了一套分享选项。此外，iOS 平台的 SkyDrive 开始支持 iPad。

### 缺憾：平台覆盖性不足

* SkyDrive 的桌面客户端并不支持 Windows XP，这对国内的许多用户来说仍旧是个问题。
* Android 客户端仍旧遥遥无期。如果微软真想要打造“个人云时代”，到现在仍旧忽视 Android 这个市场份额最大的智能手机平台就让人有些无法理解。
* Mac 客户端暂时只支持 Lion。Mac OS X 10.6 的用户可能还需要再等等。

### 总结

总的来说，本次更新大大增加了 SkyDrive 的实用性。除了平台覆盖性的缺憾，SkyDrive 的简单易用、相对充裕的储存空间（尽管新用户的有所缩减，但其免费空间比 Dropbox 的 2GB，Box 的 5GB 都要大）、移动客户端的逐渐完善，还有对国内用户来说无需特殊方法就能使用的额外特性，都是 SkyDrive 吸引我的理由。
