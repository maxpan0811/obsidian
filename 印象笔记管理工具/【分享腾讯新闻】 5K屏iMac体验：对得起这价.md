**5K屏iMac体验：对得起这价**腾讯数码: 10-28 11:20当然，这个价格也意味着它并不适合所有的消费者。**点击阅读原文:** http://view.inews.qq.com/a/DIG2014102802999101**新闻发自:** [腾讯新闻客户端](http://inews.qq.com/m)---------------------------腾讯数码讯（编译：唐马克）在十月的发布会举办之前，恐怕大多数人都会认为新款iPad才是这场发布会最大的主角。当然，iPad Air 2也确实足够优秀，超薄的机身和性能的升级依然亮点十足。不过，在更新iPad产品线的同时，苹果也发布配备Retina 5K显示屏的27英寸iMac，抢走了不少iPad Air 2的风头。虽然在外观方面5K Retina iMac与上一代iMac并没有很大的区别，但是分辨率高达5120×2880的27英寸显示屏却足够亮眼，而1470万像素的数量整整是1080p全高清屏幕的7倍之多，甚至比时下流行的4K分辨率还要多67%。如今市面上配备4K屏幕的一体机可谓少之又少，而且5K Retina iMac是一台出自苹果的产品，超高的分辨率意味着用Final Cut Pro剪辑视频是不仅能够以全分辨率播放4K视频，与此同时屏幕上还会有剩余空间留给各种操作面板。毫无疑问，在2500美元的价位上，Retina 5K iMac绝对无出其右。当然，在绝对价格上，2499美元还是很贵的，超过了市面上几乎所有的一体机。不过如果如果你对画质有着极高的要求的话，那么Retina 5K iMac将会是你最好而且可能是唯一的选择。外观体验除了屏幕带来的震撼体验之外，在外观方面5K Retina iMac与2012年发布的上一代27英寸iMac并没有太大的区别。5K Retina iMac的屏幕边缘依然只有5mm厚，如果不是从侧面看的话，甚至会忽略屏幕的厚度。5K Retina iMac整体非常纤细简洁，如果想要在桌子上放一台5K Retina iMac的话，你只需要保证桌面上方的空间能够放下一个超大的27英寸显示器，并不用担心5K Retina iMac会占据更多的桌面空间。同样纤薄简洁的铝合金支架固定在屏幕后方，中间有一个开孔用于固定线缆。5K Retina iMac依然采用了铝合金打造机身，坚实稳固的磨砂手感与MacBook一脉相承，尽显高端。在苹果两年前发布超薄造型的iMac时，或许有人会认为更漂亮的外观不会成为消费者更换iMac的理由，但是如今5K Retina屏幕的加持让人完全无法找到不去购买iMac的理由。5K Retina iMac的各种接口依然位于屏幕的背面，并且整齐的排在一起，包括3.5mm标准耳机插孔、SDXC插槽、四个USB 3.0接口、两个Thunderbolt 2接口以及千兆以太网接口。此外，屏幕背面还有电源线插口、Kensington锁孔以及电源按钮。值得注意的是，5K Retina iMac配备了两个扬声器，分别位于屏幕两侧的边缘，给人感觉声音是从屏幕边缘发出的一样。尽管从外观上并不能看到这两个扬声器，但是音效还是很不错的，比较震撼。除了主机之外，5K Retina iMac的配件与老款iMac并没有区别，用户可以选择Apple Mouse有线鼠标、Magic Mouse触控无线鼠标或Magic Trackpad，同时还会附送一个无线键盘。显示效果“我就想一直盯着它”，这或许就是5K Retina iMac这块屏幕给人的最直接感受。来自Engadget的Dana Wollman表示拆箱时的感觉就像是当初第一次拆箱HDTV。“在最开始的几分钟我什么也不做，只想来回切换自带壁纸。所有的壁纸都要比原来的更加清晰锐利，我想要知道我到底能从屏幕中看到多少细节”，Dana Wollman如是说。虽然OS X Yosemite扁平化的图标看起来很棒，但是播放一些4K视频或者超高清图片才能真正发挥出这块屏幕的功力。如果你用5K Retina iMac打开一些摄影师拍摄的优秀照片的话，你会注意到照片的每个细节你都能看清，甚至是鸟兽的皮毛，整体感觉就像是透过窗户去看外面的景色一样。当然，光凭分辨率这款屏幕还无法实现如此出色的显示效果。苹果使用了Oxide TFT技术来保证有足够的亮度，与此同时，补偿层的加入也使屏幕具有非常优秀的可视角度。从很偏的角度看去，色彩仍然不会失真。本来就非常优秀的显示素质再加上苹果精心的调校，5K Retina iMac才能拥有让人惊异的屏幕表现。此外，苹果还在5K Retina iMac使用了原先在Retina iPad上的有机钝化技术，能够在降低功耗的同时使画面具有更好的清晰度。苹果表示5K Retina iMac的能耗比老款iMac降低了30%，这也意味着更多的像素负担并不会带来过热问题。值得一提的是，苹果自行设计的Timing Controller处理器保证了所有像素都能够正常工作。但与老款iMac相比，5K Retina iMac并不能作为独立的外接显示器来使用，只能用作一体机，这恐怕也是5K Retina iMac最大的缺点之一。而且苹果目前也并没有推出5K分辨率的Cinema Display，因此如果你想要用一台Mac Pro搭配一台显示器的话，就得选择其它品牌的产品了。性能表现虽然5K分辨率的Retina屏幕才是最大的亮点，不过不要忘了iMac还是一台一体机。售价2499美元（中国大陆售价为17988元人民币）的低配版5K Retina iMac配备了3.5GHz酷睿i5四核处理器、8GB内存、1TB Fusion Drive硬盘以及显存为2GB的AMD Radeon R9 M290X显卡。在Geekbench和Xbench等综合跑分测试中，5K Retina iMac的性能表现要比配备3.2GHz酷睿i5四核处理器和英伟达GeForve GT 755M显卡的老款27英寸iMac略有增长。另外，在Cinebench OpenGL测试中，5K Retina iMac的帧数和CPU得分分别为90.54fps和540分，而老款iMac的成绩则为80.18fps和520分。考虑到5K Retina iMac和老款iMac都配备了Fusion Drive混合硬盘（128GB固态硬盘搭配7200转1TB机械硬盘），因此硬盘读写速度和启动时间基本相同。5K Retina iMac的平均读取速度、平均写入速度和启动时间分别为659MB/s、311.5MB/s和15s，而老款iMac的成绩则为668MB/s、318.1MB/s和13s，区别并不大。但5K Retina iMac的性能足以应付4K视频吗？在实际测试中，首先在5K Retina iMac中启动Final Cut Pro视频编辑软件并加载大小为10GB的4K视频，然后即便视频在播放中过程中，5K Retina iMac和去年发布的新款Mac Pro一样都拥有足够的性能使视频显示出各种特效，转换过程非常迅速，几乎毫无延迟。甚至在Final Cut Pro中设定程序让视频在出现掉帧时暂停，从头到位也视频完全没有暂停，这意味着帧数也没有下降。在测试中，5K Retina iMac用了4分53秒将这段10GB的4K视频导出为1080p分辨率的版本，而这也是唯一能够清晰听到风扇高速旋转的声音的时候。即使5K Retina iMac和Mac Pro都足以承担高性能任务，但是需要指出的是，Mac Pro即使在这种情况下也完全不会发出声音。尽管5K Retina iMac不是为游戏而生的，但是在以5120×2880原生分辨率和中等画质中等抗锯齿的设置下，运行《蝙蝠侠：阿卡姆之城》最高帧数仍然能够达到26fps，平均帧数也有22fps。而在4096×2304、3200×1800和2560×1440分辨率下，帧数则会分别增加到34fps、52fps和78fps。虽然5K Retina iMac在游戏上的表现并不是极为出色，但是各种游戏在5K Retina iMac还是可玩的。当然，如果你玩游戏时喜欢特效全开的话，那么你也不会花重金买5K Retina iMac了吧。软件系统5K Retina iMac搭载了在发布会上一同正式开始推送的OS X Yosemite操作系统。而如果过你想要充分发挥OS X Yosemite的全部功能的话，最好搭配一台iOS设备来使用。在升级OS X Yosemite之后，用户可以使用Mac来拨打接听电话、自动连接iPhone作为无线热点、即时同步Pages等软件的操作进程或者远程打开或关闭iOS设备上的Safari标签页并浏览iCloud全部搜索历史。总之，OS X Yosemite让Mac与iOS设备更紧密地结合在了一起。不过，除了必备一台iPhone、iPad或iPod touch之外，用户还必须使用苹果自家的应用，例如Safari浏览器和邮件。而如果你是一位Android手机用户，或者经常使用Google Drive或OneDrive等云服务的话，OS X Yosemite的平台整合功能并不奏效。对于大多数消费者来说，更好用的Spotlight搜索、邮件应用的多项新功能以及向iOS靠拢的扁平化设计风格才是OS X Yosemite最具吸引力的改进。总体来说，OS X Yosemite是一个非常出色的操作系统，而且如果你购买一台5K Retina iMac，OS X Yosemite也是你的最佳选择。配置选择5K Retina iMac的起步价为2499美元（中国大陆售价为17988元人民币），主要配置包括3.5GHz英特尔酷睿i5四核处理器、8GB内存、1TB Fusion Drive硬盘以及配备2GB GDDR5显存的AMD Radeon R9 M290X显卡。当然，苹果也和以往一样为用户提供了丰富的硬件升级方案。处理器方面，用户最高可选配4.0GHz英特尔酷睿i7四核处理器，升级费用为250美元（中国大陆为1840元人民币）。内存方面，16GB和32GB的升级费用分别为200美元和600美元（中国大陆分别为1520元和4560元人民币）。显卡方面，升级配备4GB VRAM显存的AMD R9 M295X独立显卡的价格也为250美元（中国大陆为1840元人民币）。最后，储存方面，2499美元的低配版本可以选择256GB固态硬盘或者1TB Fusion Drive硬盘，而升级为3TB Fusion Drive、512GB固态硬盘和1TB固态硬盘的价格分别为150美元、300美元和800美元（中国大陆分别为1104元、2208元和5888元人民币）。市场竞争不论从出色的Retina 5K屏幕还是高昂的售价来看，目前市面上的一体机似乎都不是5K Retina iMac的直接竞争对手。按照苹果在发布会上的说法，5K Retina iMac的竞争对手其实是售价高达几千美金的高端4K显示器。不过这也存在某种程度的误导，因为5K Retina iMac并不支持作为独立的显示器来使用，所以和4K显示器比较价格并没有太大的意义。如果你已经拥有一台性能强大的主机的话，那么5K Retina iMac的高“性价比”与你毫无关系，你还是得乖乖地去买一台4K显示器。当然，目前市面上的4K显示器的价格也有所差别，价格差价很大。夏普32英寸PNK-321显示器的价格高达4795美元（目前的市场价格在3600美元左右，约合人民币22030元人民币），戴尔即将发售的5K显示器的价格为2500美元（约合人民币15300元），华硕31.5英寸PQ321Q显示器的价格为3000美元（约合人民币18360元），这些4K显示器的确很贵。不过，市面上也有一些相对来说便宜很多的选择，比如戴尔28英寸P2815Q和三星28英寸U28D590D的价格都只有600美元（约合人民币3671元），分辨率同样为3840×2160。如果你想要一台配备4K屏幕的一体机的话，5K Retina iMac显然是最接近的选择。不过，配备1920×1080全高清屏幕的21.5英寸iMac和配备2560×1440 QHD屏幕的27英寸iMac也是不错的选择。而在Windows PC阵营，目前还没有出现能够与5K Retina iMac抗衡的产品。戴尔高端一体机XPS 27配备了QHD屏幕，售价为1600美元（约合人民币9791元）。此外，华硕一体机ET2702IGTH和联想A730也配备了同样分辨率的屏幕。而惠普最高端的一体机Envy Recline 27一体机仅配备了全高清屏幕，售价1400美元（约合人民币8567元）。与以上这些Windows PC一体机相比，虽然5K Retina iMac的售价要贵很多，但是Retina 5K屏幕带来的视觉震撼却无人能及。总结5K Retina iMac目前已经在苹果官方网上商店正式开卖，而考虑到出色的Retina 5K屏幕，2499美元的价格其实很值得。当然，两倍于竞争对手的价格也意味着5K Retina iMac并不适合所有的消费者。如果只是用电脑来上网刷微博的话，起步价1099美元（中国大陆售价为7988元人民币）的老款iMac足以满足你的需求，不过如果你对视频或者照片有着较高追求的话，5K Retina iMac一定会让你心动不已。而且对于没有必要或者没有足够资金购买售价高达2999美元（中国大陆售价元人民21888币）而且没有显示器的Mac Pro专业主机的消费者来说，能够流畅编辑4K视频而且配备Retina 5K显示器的新款iMac简直很难让人拒绝，而且更重要的是，起步价仅为2499美元。当然，如果你已经有一台Mac Pro却对5K显示屏的震撼效果趋之若鹜的话，只能期待苹果快点推出5K Cinema Display显示器了。来源：Engadget
