# 高考志愿填报指南（互联网行业版）

---

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/4B94024D-DB48-4890-BCA2-31194E2F0083.jpg)

知道你们最近要填志愿了。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/5B273048-FFDD-4A45-81C3-BF608B694C89.jpg)

（图片源自：[BI](http://static3.businessinsider.com/image/5756d67c91058424008c7211/students-who-cheat-on-college-entrance-exams-in-china-could-go-to-jail-for-it.jpg)）

其实这些天有很多所谓高考辅导名师在指导考生填报志愿，有些还收费很高。

但是这个填报志愿有一个特点，当时你觉得报得好没用，四年之后毕业还觉得报得好才算好。

没什么专业是四年之间一直很火的，大学的排行榜年年在变，连地域的发展也是动态变化的，所以问题就来了，指导志愿填报，与其为考生着想，不如投其所好。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/FACE6044-301E-43CC-A800-BE9F0B4AB043.jpg)

（图源：[Turner](http://i2.cdn.turner.com/cnnnext/dam/assets/160607113328-gaokao-students-china-study-exam-college.jpg)）

遇到喜欢 211、985 这种名号的，就给你推荐中西部那些号称 “性价比” 高，有强势专业的没落贵族；遇到向往《小时代》里浮华生活的精神上海人，就给你推荐到松江大学城里学一些可以在大城市立足的时髦专业；如果看你有点文艺青年的矜持，就按照中国最美校园排行榜给你塞到一个离家很远，以文科见长的学校。

全都是套路，没有一点真诚。反正四年之后你后悔了，我都不知道人去哪儿了。大不了你取关、拉黑我。

但是老编辑觉得这样是不对的，填志愿是人生大事，我们要为考生负责。别的行业我不了解，对于想进入互联网行业的高考考生，我还是有一些人生经验可以给大家的。

所以我们今天就讲讲想要进入互联网行业的考生如何报志愿。

### 选城市

《死亡诗社》里面的罗宾 · 威廉姆斯，第一节课就教学生撕毁《诗歌鉴赏》课本，因为课本让学生按照笛卡尔坐标系评价诗歌。

我不太懂这些搞艺术的。但是我还是觉得报考志愿是一门技术，还不是艺术，按照地域、学校、专业三个坐标轴来确定自己报考的志愿其实很有效。

我们这个互联网行业，作为热门新兴行业，招人这么难，工资这么高，大学里却没几个对口专业。

而且没有一个国字号的中国互联网大学，重点大学里面都有马克思主义学院却没有互联网学院。甚至连野鸡大学，比如什么湖畔大学，都故弄玄虚，没有在自己名字里加入互联网三个字的。

所以坐标轴中最重要的反而不是专业，而是城市。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/EC5F260B-5994-4A83-8C2D-413127FAB714.jpg)

（图源：[Google](http://www.hdwallpaper.nu/wp-content/uploads/2015/07/China_City_Wallpaper_001.jpg)）

在互联网行业的光谱里，北京和深圳是超一线城市，上海、杭州、广州是一线城市，成都和厦门属于二线城市领头羊，身后是南京、武汉等寥寥几个名字。

所以你想进入互联网行业，就按照这个优先级去选择城市。

特别是分数不是很高的考生，也就是上个普通一本甚至是二本，那就不要任性，就在北上广深杭这五个城市里面选。这几个城市之外，方便进入互联网行业的学校掰着指头都能数出来了。而且都是比较高分的学校。

比如中国科技大学、华中科技大学、西安电子科技大学、电子科技大学，没有超一本线百八十分你想都不要想。

当然凡事都有例外，比如你想当网红，去东北就是个不错的选择。哈尔滨的中央大街，天天有网红端着电脑和麦在直播。你在校四年跟宿舍人学会一口东北话，再走出校门细心观察那些 “社会仁儿” 的套路，然后 YY 走起，快手走起，映客走起，新一代的喜剧新星就冉冉升起。

我相信你爸妈不会让你这么干的，当网红多辛苦啊！看看老编辑，本来 90 后，现在活活给累成 70 后老干部了。

### 选学校

圈定了城市，我们就一个一个说选学校的事情。

北京当然是投身互联网的第一选择。学校多，互联网公司扎堆，实习机会比比皆是，进入互联网可以从大一开始抓起。

但是不同的分数肯定有不同的选择。高分的，清华北大。清华北大当然你出来做什么都对。很多考生从小都在想这个问题，我长大了是上清华还是去北大呢？

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/B7550AAB-CB01-4E11-B45F-C262D22C1FBB.jpg)

（图源 [Wikimedia](https://upload.wikimedia.org/wikipedia/commons/f/f0/Printing_with_a_3D_printer_at_Makers_Party_Bangalore_2013_11.JPG)）

清华北大之间要说有什么区别，传说清华毕业创业成功率最高，主要是校友资源太丰富。我就举个例子，前年有个做 3D 打印的创业者，在校友群里一吆喝，众筹了好几百万，资金问题就解决了。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/4D6A90E5-88F7-46DF-B406-51BEBA4D5E62.jpg)

（百度董事长兼首席执行官 李彦宏，图源：[Quartz](https://qzprod.files.wordpress.com/2016/05/baidu-ceo-e1462860623418.jpg?quality=80&strip=all&w=1600)）

北大创业者，有个说法是，创立的公司累计市值最高。这个主要原因是百度的李彦宏。清华几十个创业者捆一起也比不上。你要按照创业者市值算的话，杭州师范的校友还排第一呢。

上不了清华北大的，第二梯队就是人大和北航，这跟一般专业选学校是一样的。人大的创始人配上北航的技术，这在我们中关村创业大街算是黄金搭档。

分数再低，那就和别的专业不一样了，一般来说会推荐你选北师北理工。但是想进互联网的话还是首选中传和北邮，文科选中传理科选北邮。

北邮有很多互联网创业者，尤其喜欢做给互联网提供技术平台和基础服务的方向，比如极客学院，36 氪，拉勾网，都是北邮系创业者做的，可能是受他们这个校长方滨兴的影响，做个互联网产品不算啥，要做就影响一代互联网人，所以北邮校友人脉特别广。

中传的校友主要在媒体行业，媒体行业也是互联网上一股主要力量。这个我们下面讲专业的时候细细讲。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/D5D8169F-7473-4276-923C-EE212C921E3C.jpg)

（图源：[huffpost](http://i.huffpost.com/gen/2104004/images/o-CHINA-COLLEGE-GRADUATES-facebook.jpg)）

再往下，一般的 211 就主要是选专业了，学校差别都不是特别大，进入互联网就没什么光环和校友资源了。

在上海，其实上海的互联网氛围并不好，传统软件行业氛围倒是很好。上海的知名创业者主要都是复旦和交大的，而且真不是所谓文科选复旦理科选交大，复旦新闻系那都是多少年前的老皇历了。饿了么的创始人张旭豪，携程四君子中除了梁建章，小红书的毛文超，都是交大出身的，但是创业都不是搞技术的。

在上海进复旦交大，和在北京进清华北大一样，都是你做什么都对。但是明显地，同样的分数，想干互联网去上海不是一个 “性价比” 高的选择。去上海旁边的杭州，挑选学校的选择更多。

分数高的去浙大，浙大毕业无论是通过 ACM 各种编程竞赛去大公司还是去创业都是很好的选择。杭州除了浙大之外没有 211 和 985，但是其他学校进互联网公司也很容易。

阿里巴巴创始的十八罗汉，没有一个是浙大的，都是浙江工业大学、工商大学，杭州师范大学、电子科技大学。而且在 07 阿里巴巴第一次上市之后，浙大的学生还是普遍看不起阿里巴巴。所以如果没有 211、985 情节，可以在杭州除了浙大这些学校之间按照自己的分数选一个，毕业之后去创业或者进阿里、网易、蘑菇街什么的，都很方便。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/9DF76458-4D6E-410A-A5AF-9CEBCBB4ED8B.jpg)

（腾讯公司控股董事会主席兼首席执行官 马化腾，图源：[Japantimes](http://www.japantimes.co.jp/wp-content/uploads/2016/03/b-tencent-b-20160319.jpg)）

说到深圳，其实没什么学校好选的，深圳大学是马化腾母校，当然不错，在南山区，一进学校就耳濡目染，附近到处是创客空间。但是深圳大学给外地的名额不多，北方人去了也不一定好适应。

深圳好学校不多，但是几乎要把西安、成都、东北的优秀工科院校毕业生都吸过来了。说到腾讯，肯定要说到华科，说到华为中兴这些通信厂商，肯定要说到电子科技大学、西安电子科技大学。说到大疆，就是造无人机的那个，肯定要提到哈工大。

所以想去深圳发展的，倒是可以先去我刚才说的这些学校。特别是华科，有很多学校自称南方小清华。但是华科在深圳的地位真的可以称得上是清华，因为曾经的华为太子李一男是华科毕业的，如今已经被神化的微信产品经理张小龙也是华科的。华科计算机等相关专业毕业，几乎就等于保送深圳的几家大公司。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/F9C7B211-1AB4-4F40-BCF2-3CC55DFADB31.jpg)

所以想到武汉读书，未来要进互联网的，你就不会在武大和华科之间纠结了。有人说武大毕业的也有雷军，那都是多少年前的事情了。华科有三大互联网社团，联创、冰岩作坊和 Dian，每年都有融资上千万的创业者出现。况且连雷军来武汉替小米招聘，第一站也得到华科。

大家可以在百度中搜索，腾讯校招、百度校招、阿里校招，然后看这个招聘宣讲会的地址。在哪个学校办宣讲会，通常说明这个学校互联网相关专业比较强势，每年有大批毕业生进 BAT。

### 挑专业

刚才说了一大堆学校、地域的事情，估计还有一些考生一头雾水，到底你们互联网行业是干什么的？

互联网行业的职位拆解开来说，最重要的是三块，产品、技术、设计，现在互联网行业竞争进入了下半场，开始从渠道为王转向内容为王。这个会做内容的也是很重要的一块。除此之外、运营、市场、销售、行政也是互联网公司需要的。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/8BC121D2-70A5-4E77-B597-7EA8002C6589.jpg)

我们一个一个说，先说技术。计算机、软件工程这是最相关的专业。但是你会发现很多人搞技术的人是什么通信工程、自动化、地理信息系统毕业的。主要是因为这些专业大量使用编程技术，所以毕业生的编程水平很可能比计算机科班毕业的还高。

所以你要想做技术的话，就看这门专业有多少编程、数据结构、算法相关的课程。

当然，这个大学里面的计算机课程都很落后，比如万年不变的谭浩强的 C 语言，在知乎上被程序员黑得很惨，还有程序员要因此约架，在 github 上比赛写 C 语言，真是吓死宝宝了。

所以很多不相关专业的理科生也通过自学投身程序员。我这里就举个栗子，比如这个生物专业的，搞互联网的比例非常高。曾经有个清华生物学的班级，到博士阶段只有一个人在做生物，剩下一半多去搞互联网、计算机了。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/0EE6B40D-A48A-473F-AF12-D02A3A6A0D7B.jpg)

（丁香园 CTO 冯大辉，图源：[Lifemt](http://pic.lifetmt.com/2014/06/0528%E5%86%AF%E5%A4%A7%E8%BE%891.jpg)）

丁香园 CTO 冯大辉是学生物的，百度的李彦宏自己不是学生物的，但是他老婆马东敏是学生物的，当年他不想回国创业在美国花园里种草，让马东敏都给他薅了。李彦宏的合伙人，百度创始人徐勇，也是学生物的。

主要是因为基础学科的不好找工作，逼得大家只好去搞计算机。

所以选择基础学科，也是进入互联网行业的一条路，特别是学什么数理。做技术给人一种算法基础扎实的感觉，跟那些只会搞语言鄙视的码农相比高下立判。去创业的时候也可以张口闭口 “我是学物理的，我的商业模型是经过数学物理验证的”，逼格不知道高到哪里去了。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/3417714B-12EF-49EB-949F-40D8D1CAAEB1.jpg)

（香港“教父级”设计师 Michael Lau，图源：[Hypebeast](http://cn.hypebeast.com/files/2016/03/michael-lau-8.jpg)）

除了技术，下面就是设计，现在在硅谷的创业公司中，设计出身的创始人比例越来越高，比如 Anrbnb 和 snpchat 的创始人都是设计出身的。

目前国内这个趋势还不明显，但是等你毕业了，应该有所改观，所以选设计相关专业也不错。  
跟设计近似的还有动画这一类视觉相关专业。现在国内最挣钱的互联网方向就是游戏，传说中一年发 50 个月工资的，都是游戏公司或者网易、腾讯的游戏部门。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/FAEA75D7-12A4-4A05-B90F-54C24AD548B1.jpg)

（图源：[Breibart](thttp://cdn.breitbart.com/mediaserver/Breitbart/Breitbart-California/2014/09/17/Steve-Jobs-iPad-AP.jpg)）

产品经理就比较黑了，早年产品经理都是技术出身转做产品。但是最近几年产品经理的需求大涨，很多应届生开始做产品。所以就有一个笑话，“为什么我要做产品经理啊，因为我什么也不会啊？”

相比技术设计，产品经理的职能有点虚，但是非常重要，他要把控用户体验和产品的逻辑。这个岗位需要对运营、技术、设计、商业模式都很敏感，所以是 CEO 的预备役。

想做产品经理的，对专业倒是没有什么限制，很多专业的知识，比如心理学、工业设计、管理学乃至计算机都会对你做一个好的产品经理有所帮助。而且毕业就做产品经理并不是一个很好的选择，从设计、技术、运营这些岗位做起转做产品可能效果更好。

说到做内容和运营，这个就是文科生占优势的职业了。

运营和产品一样，对专业要求不高，但是对学习能力要求很高。运营最重要的是你对自己运营的产品有兴趣，对你的潜在用户有了解。所以你应该选择一个大学里面不是特别忙，有很多空闲时间发展自己兴趣的专业。这也是文科生做运营比理科生有优势的原因。这里真不是在黑文科生，文科生本来就要涉猎更广。

不过对于文科生来说，未来几年最好的方向应该是做内容，去年开始内容创业火起来了，中国电影、娱乐行业也进入了黄金时代。所以做内容是文科生进入互联网的首选。

当高分考生扎堆去金融、国贸、小语种的时候，你应该坚定地选择中文和新闻这种看起来毕业就会很穷酸的专业。

现在几乎所有的投资人都加入了 “关爱前媒体人协会”。央视的主持人一个接一个去做投资，网易的总编一个接一个成功创业。我刚才说对于想进入互联网行业的考生来说，中国传媒大学比北京师范大学更好就是这个原因。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/85F4C94C-2EE9-49C2-9C50-7FA58B2B4EB6.jpg)

（图源：[growmybusinessrocgperth](https://growmybusinessrocgperth.files.wordpress.com/2016/02/insight-consulting-due-diligence-reporting.jpg)）

当然也有说这个互联网媒体不一定要新闻、中文出身。但是既然你认定这条路，与其半道出家，不如本科就打好写作、拍摄和采访基础。然后在媒体辛勤耕耘几年之后就可华丽转身。

现在用户对于内容质量的要求其实越来越高。前有新浪的功勋主编陈彤，去人大补一个新闻学的硕士。后有中戏导演专业研究生 papi 酱，拍五毛钱特效都没有的吐槽视频，真的是降维攻击。

未来是视频的时代，VR（就是虚拟现实）的时代，是全民直播的时代，所以跟影视传媒，动画、视觉相关的专业是很有前途的。互联网过去以程序员为主的人员结构决定了，反而未来几年这些相关专业的毕业生可能出现比较大的缺口。

### 进击的学渣

到这里有些同学可能会说，老编辑你讲的都是些好学校，好专业，大城市。像我们这种三本或者专科水平的学生是不是就没机会进互联网了。

其实不是的，你还有一次 “辍学” 的机会。

辍学生在互联网行业会被高看一眼，地位至少相当于普通 211。如果你担心父母断掉你的生活费，你甚至可以一边上学一边宣称自己辍学，但是绝对不要让校园生活出现在你的朋友圈。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/389124E4-ECDE-43A8-987B-89F70AF7B0CE.jpg)

你买一台二手的 Macbook Air，反正 2013 年以来就没怎么更新过，3K 到 4K 就够了，在宿舍里自学 Sketch 做设计、自学 Axure 做产品、自学 HTML 写前端、自学 Python 写爬虫或者立志学会世界上最好的语言都可以。

如果你对自己的自学能力信心不足，可以去培训机构，但是最好不要说你去过任何培训机构。因为培训机构出身会被歧视。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/AC55FD47-53CB-4427-9062-D5BDFF3CAC21.jpg)

（图源：[Google](http://s3.amazonaws.com/digitaltrends-uploads-prod/2014/11/google-search.jpg)）

很多学习资料都是可以通过搜索引擎、论坛、MOOC 免费获取的。

Github 或者 Dribbble 这些社区一定要早注册，多看别人的作品同时要进入各种技术论坛和知乎上怒刷存在感。你可以从模仿别人的产品、改写别人的代码做起，一步一步来，然后找机会做外包、去小公司实习，一步一步积累经验。

线下也要经常出入创业路演，干货分享、黑客马拉松等场合。互联网是个圈子，你要懂得他们的套路，吃不了猪肉先看看猪走路。

按照目前互联网的市场行情，等到你毕业的时候，你就是一个辍学工作三年的 “圈内人” 了，一份月薪 12K 起的工作就在向你招手。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/5DFF595E-A0FC-4359-9DDC-AC726211DFE0.jpg)

（插图来自：[Globalizanition](http://www.globalization101.org/wp-content/uploads/2011/10/women-computer.jpg)）

看到这里，你是不是又仿佛看了人生的希望，成为互联网精英的冲动让你热血喷张。

但是当你真的按照本文填报了志愿，万一四年之后互联网泡沫破灭了，钱又跑回实体经济了，程序员的工作饱和了，希望你不要翻出来四年前收藏的这篇文章，装作粉丝要给我们寄礼物，其实上门来讨说法。

![](.evernote-content/C7F30488-A8C9-4173-98C1-BEE27957CA39/7FB9A246-E130-4C7A-BA0E-05B0F8BFCF95.jpg)

传统行业和互联网有时候是相辅相成，交替繁荣的。比如如果没有运营商过去十年从 GSM 到 4G 大跃进，今天就没有互联网的高速连接。如果没有智能手机、芯片行业按照摩尔定律大发展，今天的内容、社交这些互联网领域都发展不起来。

你就看美国，根据彭博社去年的一份调查，其实程序员工资水平最高的还不是硅谷，而是西雅图和休斯敦，这些城市或者是航空航天，或者是能源行业很繁荣，对于计算机人才的需求很大。

所以填报志愿还是要选择自己喜欢或者擅长的专业，学校在自己分数够用的情况下选最好的，这样无论这个行业将来如何起起伏伏，你都不会后悔。
