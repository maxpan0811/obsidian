# El Capitan新特性之——自动隐藏菜单栏

---

[![](.evernote-content/282A7F5E-BF90-4451-B5A0-F7157B7A487F/23E41A83-52BF-4591-AD17-0C971B0C240D.jpg)](http://www.feng.com/apple/tutorial/2015-10-22/El-Capitan-of-new-features---automatically-hide-the-menu-bar_627899.shtml)

　El Capitan 虽然没有带来很多新的酷炫功能，但以性能提升为主的它还是带来了很多细微的变化。  
  
　早在 beta 版的时候就有用户发现，位于 OS X 系统上方的菜单栏现在终于可以隐藏起来了。作为系统中的重要组成元素，菜单栏过往在日常的使用中起到应用功能列表的作用，不过随着应用开发者对于应用的改进，有时候菜单栏中的功能都已经全部包含在应用界面中，所以菜单栏的存在感似乎有所降低。

[![](.evernote-content/282A7F5E-BF90-4451-B5A0-F7157B7A487F/23E41A83-52BF-4591-AD17-0C971B0C240D.jpg)](http://resource.feng.com/resource/h060/h59/img201510222326030.png)

自动隐藏菜单栏功能无疑给了用户选择的可能性，如果想要更多显示屏空间，你可选择自动隐藏菜单栏。当你需要菜单栏时，将光标移动至屏幕顶部即可。

有了这个功能，将桌面的文件收起，将 Dock 栏也设置为隐藏，那么就能够让 OS X 的桌面呈现出一张完整的壁纸了。
