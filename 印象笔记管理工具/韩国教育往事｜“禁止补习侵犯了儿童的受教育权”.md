#教育故事

9个

1980年的韩国社会一半是海水，一半是火焰。

这一年的前一年，前总统朴正熙遇刺身亡。朴正熙这位军人出身，靠政变上台的总统，在任18年间把韩国人均GDP从82美元提升到1592美元。这段韩国经济高速发展时期，就是被后人称颂的“汉江奇迹”。

经济高速发展，使得人民生活水平急速提升，购买力大大增强。同时韩国国民的需求在这短短二十年内从马斯洛金字塔最底层的“生理”，大踏步跃迁到顶层的“尊重和自我实现”。

作为一个深受儒家思想影响的国家，这种需求层级的跃迁，在一个领域产生了激烈的结果：教育。

![](.evernote-content/697F85EC-FF06-459B-A451-BEA10B6FF949/FFCF52CD-97C6-4324-B7F2-F6D8164BC907.jpg)

朴正熙，1917 - 1979

汉江奇迹期间，韩国整个社会的资源都被充分调动起来以促成经济的高速发展。而作为一个当时的穷国，社会资源的核心自然是人力资源。

人力资源的调动和发展，背后最大的推手是教育。

前近代朝鲜的教育，是一种深受儒家文化影响，围绕科举展开，依托私塾为核心教育资源的高度精英教育。到日据时代的40年间，朝鲜民族作为二等公民，能够接受中高等教育的就更加是绝对精英（且亲日）的人群。

二战结束后，美国占领军政权在短短三年内强势推行了基于美式体系的教育，并为后来的集权式教育改革提供了土壤。

朝鲜战争结束后，在废墟国土上重建的韩国教育体系，彻底推翻了乡土儒家自治精英教育的传统，转而推行了强力中央集权体制下的普世基础教育。

结果是在5年内，韩国的学龄人口小学入学率从64%快速提升到90%以上。而这批小学生，就是日后汉江奇迹时期支撑起经济发展的那批韩国年轻人。

韩国当时推广普世基础教育的目的是为了国家进一步发展提供人才。而韩国当时发展所需要的人才，包含大量的职业性人才。

因此在普世教育第一阶段任务完成以后，韩国适时得新设了大量职业教育学校，而把学术性中高等学校数量控制在一个比较低的范围内。背后的想法是让一小部分适合的孩子通过考试，进入学术道路，而让更多的孩子进入职业发展道路，并快速成为韩国经济发展的血脉。

![](.evernote-content/697F85EC-FF06-459B-A451-BEA10B6FF949/A5E0F35C-1668-45D8-8873-A30AFF4DE4FE.jpg)

1960年代韩国学生毕业典礼

但是深受儒家思想影响的韩国父母并不这么想。他们的价值观我们很熟悉：万般皆下品，惟有读书高。

我们来复盘下到现在为止那个时代韩国教育出现的三个元素：1. 基础教育普及化 2. 中高等学术教育受控，并大力发展职业教育 3. 受儒家思想影响下对学术性教育的追崇

这三个元素聚集到一起，便召唤出了一条深深影响韩国教育至今的恶龙：韩式教育狂热。这不是G大发明的：英语里面有一个词组叫Education Fever，特指发源于那个时代韩国的教育狂热。你去Google搜索这个词组，就会明白我在说什么。

韩式教育狂热发端于1960年代中期。现象是小升初的考试竞争非常激烈，补课班盛行，考试失利而选择小升初复读的现象非常普遍。

当时的韩国人甚至发明了一个词来形容这种情况：ipsi-jiok（升学地狱）。

升学地狱之下，严重问题开始浮现：

1. 高压的小升初对11岁的学童造成了影响一生的心理生理损伤
2. 整个小学高年级的教学都围绕升学展开，根本无暇顾及这个年龄段孩子长线人生发展所需要学习的课程
3. 家庭要花费高额补课费用
4. 中学教学质量和后续升学结果差别巨大，导致家长/学生对一流学校产生了魔怔式的执念

这种教育狂热牵扯了事业黄金期父母大量的精力，占用了大量社会资源，摧毁了孩童的健康成长，并让大量在当时国策下应该迅速进入职业教育轨道的学生延迟升学。

![](.evernote-content/697F85EC-FF06-459B-A451-BEA10B6FF949/07753DCC-1ADF-4AB0-B444-978872B8FF9E.png)

1969年汉城，可以看到破旧的城市背景中正在建设高架道路

这些显然和朴正熙全力发展经济的既定方针相违背。然而注重教育这样一个本核非常价值观正确的诉求，又不是一个可以用简单粗暴行政手段直接斩断的事情。

于是在万般妥协之下，当时已经完成教育体系中央集权的朴正熙政府，祭出了一个无奈的解决方案：

小升初摇号

这个政策的全称叫做《1969年初中学校平等化法案》。法案从各方面对当时韩国的初中办学设置了规范。而其中最核心的一条便是：取消初中学校的入学考试，改由摇号决定入学名单。

这个政策推出时没有遇到任何阻力。军政府铁腕推行改革时，民众都知道自己不该做什么。同时占绝对多数的考不进好学校孩子的父母，没有任何理由反对这场改革。

改革的效果立竿见影：各初中的生源质量迅速被平均化。同时根据当时的报道，学童的身心健康程度都随着小升初考试的取消而得到大幅改善。

然而这个政策的另外一个核心目标：减少补课，却完全没有达成。不但没有达成，反而加剧了补课现象。

原先经过挑选的生源，相对水平一致，学校和老师可以因材施教，通过校内教学满足大部分学习/升学需求。在摇号政策下，校内/班内生源水平的差别导致老师因材施教的能力大幅下降。

这种负面影响对成绩好的孩子尤其明显。于是校内学不够，校外补课必然更加盛行。

此外，由于高中入学考试依然存在，所以初中学校平等化只是让升学地狱从小升初延后到中考而已。

![](.evernote-content/697F85EC-FF06-459B-A451-BEA10B6FF949/29F4FE39-2984-406F-AAC2-DD53FAE1FD95.jpg)

1970年代韩国中学课堂

小升初摇号没有解决问题怎么办？

初升高继续摇。

于是《1974年高中学校平等化法案》出台了。

这个高中版本的政策除了基本延续了初中版本的核心政策以外，另外加上了两个目标：发展职业教育和发展郊区/乡镇教育。

韩国教育部在这个法案里特别列了一条法则：严禁各级学校或相关单位公布学校教学、考试、升学数据。

这个补丁的效果同样立竿见影，并且和初中版本一摸一样：各高中校内/班内生源变得层次不齐，老师对于差别如此巨大的生源束手无策，教学质量一落千丈。另外由于缺乏竞争以及教学质量及结果数据不透明，学校和教师没有任何动力去实际改善教学，以应对生源变化带来的挑战。

其结果就是有诉求的家长和学生投入更多的时间和金钱到课外补课，来让自己有相对优势去应对韩国当时竞争高度激烈的高考。

到1980年，韩国约30%的高中生参加各种形式的补课。

一系列旨在减少补课的政策，最终加剧了补课。

各位应该可以想象到当时韩国父母对于新政的怨声载道。

MIT博士，韩裔教授Sunwoong Kim在其论文《韩国中等教育平等法案》中，认为韩国课堂内教育的衰败，以及补课现象的愈演愈烈，完全是这两份平等法案的责任。

在朴正熙执政的末期，韩国社会一直处于一种火山口的状态。汉江奇迹带来了社会巨大变化，在短短20年内韩国国民的需求快速提升到了“尊重和自我实现”。当这种需求无法得到满足时，社会不安定便埋下了种子。

包括教育在内的各种怨声载道的集中爆发，便是1979年由朴正熙亲信兼时任韩国情报局首脑的金载圭对朴的成功刺杀。

刺杀的原因众说纷纭，至今没有定论。然而有一派主流观点，是以金载圭为代表的继续改革派认为朴正熙已经从那个挑战恶龙的少年变成了恶龙。

![](.evernote-content/697F85EC-FF06-459B-A451-BEA10B6FF949/28B70762-E4E5-445E-AB79-30EA4160B992.jpg)

刺杀案审理现场，左二为金载圭

在快速解决了政权交接和刺杀事件的调查以后，全斗焕作为新任韩国军政府首脑，全面接管了政军。时间来到了本文开头的1980年，全总统面对的是一个一半海水一半火焰的韩国。

一方面高速经济发展使得韩国在国际社会中声誉雀起，另一方面国内外对于军政府独裁统治的声讨日趋激烈。

这种声讨的高潮，是1980年5月发生的光州事件。群众自发组织的抗议活动遭到了全斗焕派出的军队镇压，造成了100余名韩国国民死亡。这件悲剧几乎是现代韩国政治里最具有标志性意义的事件，并遭到了当时国际社会的普遍严厉谴责。

然而人权没有意识形态重要。光州事件的次年，汉城击败名古屋成为1988年奥运会的主办地。投赞成票的，都是前一年谴责韩国的西方国家。时任的奥委会主席萨马兰奇说过一句话：你如果想让这个国家更民主，那么让奥运会到这个国家，是最好的方式。

光州事件和成功申办奥运会这两件事情，成为了那个时代韩国最鲜明的注脚。

回到教育方面，全斗焕政府上台以后，就把深化教育改革的目标直接指向了补课本身。和前代政府试图通过学校改革来降低补课热度的迂回政策不同，全政府采取了简单粗暴的直接关停手段。

这个政策，就是每次讨论禁止补课话题都会被提起的《1980年教育正常化和过热课外解消方案》。

![](.evernote-content/697F85EC-FF06-459B-A451-BEA10B6FF949/DD98B1A1-BFE7-4616-9CE9-46E1C5807548.jpg)

《方案》听证会现场，当时韩国正式书面语言还是汉语

这个方案的政策中关于禁止补课的条款规定任何在职老师不得提供课外补课。补课只能在有许可的盈利性私营私塾（hagwon）中进行。

后世经常对这个政策的后果进行想象和加工：普通人无法补课，权贵家庭获得秘密上门补课服务“别墅补习”，并获得升学优势，等。

这并非事实。

事实是这个政策完全没有产生任何实质效果。

该补还是补，甚至补得更多。到1997年，高中生的补课率从前文中1980年的30%上升到了接近50%。彼时，韩国家长每年花在补习上的总费用已经到了GDP的3%，和整个国家花在教育上的公共开支几乎持平。

看到这个结果，韩国后续历任政府也逐渐放松了对补课的管制。最终在2000年4月，韩国最高法院宣布禁止补课的法令违反宪法，并予以取消。

在法院的公文中，有这么一句话：

“禁止补习侵犯了儿童的受教育权。”

韩国军政府相信通过的严厉的法案，补课将会遭到重大打击，并逐渐消亡。他们尝试了，失败了。事实是不管有没有“禁补令”，补课业都在持续做大。

而补课业做大的根本原因，恰好是前面初、高中两层平等法案造成的中学教育质量全面、普遍下降。

![](.evernote-content/697F85EC-FF06-459B-A451-BEA10B6FF949/BFA36685-ECE8-408D-BC2D-9448A00B7DED.jpg)

首尔江南区大峙洞补课一条街

补课业的逆势发展，让全斗焕及后续韩国政府开始对高考做改革。想法很单纯：既然竞争压力最终来自高考，那我们就改革高考。

韩国的高考改革尝试过各种方法，最终每一次都造成补课业更加做强：

1. 从只考韩数英三门变为3+n，这种科目数加多的方案，一如预料得让补课变得更广泛
2. 在录取标准中加入50%的高中平时成绩，这使得原本针对高考的补课，变成了前置到高一就开始
3. 针对上一情况，韩国教育部要求高中在给平时成绩时要考虑除了成绩以外的综合测评，这让补课机构又多了综合测评业务
4. 让高考整体变简单，这让补课机构增加了刷题练习手稳的业务

或许这是独裁军政府的问题。屁股决定脑袋时，事情做不好。

因此当第一届真正的民选总统金泳三上台后在广泛听取民众意见的基础上推出《1995年教育改革计划》时，韩国社会对此是寄予厚望的。

那次的教育改革，在短短三年内推出了超过100项措施，并制定了以下核心目标：

1. 拓展多元化教育，推行多元化评价方案
2. 让中小学对家长更多听取家长诉求并采取行动
3. 推行培养创造力而不是记忆力的课程
4. 继续改革高考，以使得补课得到控制
5. 给教师和学校设定新型KPI
6. 继续增加教育开支，到GDP的5%

这个批次的教育改革政策，有一些在金泳三任期内得到执行，有一些延续到继任者金大中时期。

两位总统在任期内把教育改革作为核心施政方向，全力推进相关政策落地。

结果，没有鸟用。

![](.evernote-content/697F85EC-FF06-459B-A451-BEA10B6FF949/B4CB7809-AD2E-41E4-B150-98C608693399.png)

现代韩国补课私塾(Hagwon)课堂场景

上面所列的施政目标，除了教育开支花费到位，以及让学校一定程度上对家长和学生的反馈能采取措施以外，全部失败了。

这次改革的失败，让韩国民众彻底丧失了对于教育改革的信心和兴趣。

此后一直到现任的文再寅的各界政府，一直在教育改革方向陆续推出新政。比如李明博时期建立的35所仿照德国学徒制建立的职业高中，和晚10点以后的补课宵禁令。

这些政策没有对韩国高强度的学习压力和庞大的补课行业产生任何改变。

当韩国目前残酷的教育竞争以一部韩剧《天空之城》进入国人视野，并引起无数共鸣时，有不少人疑问为什么韩国不采取激进的教育改革来改变这个现状。

他们早已尝试过。

太阳底下无新事。
