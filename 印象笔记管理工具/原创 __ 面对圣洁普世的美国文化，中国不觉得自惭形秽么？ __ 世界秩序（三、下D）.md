先简要回顾一下本系列之前的主要内容：

* 过去中国经济的发展，是极大的得益于现行的世界秩序 Pax Americana。但是现在这个世界秩序里大家收益非常不均衡，**我们已经进入世界秩序剧变的危险期**。
* 世界秩序不可持续的根本原因，是我们进入了存量博弈的世界。一方面，我们处在科技大周期的末端，很难有大量的纵向增量蛋糕；另一方面，经济全球化基本见顶，将来横向甚至会是减量博弈。
* 人类社会里，富者愈富穷者愈穷是不可避免的自然规律，所以存量博弈是不稳定的。存量博弈 + 穷者愈穷的结果，秩序的崩溃，是人口和财富的大量消灭，然后系统再一次重启。
* **文明的兴衰是不可避免的，而霸主交替大约间隔百年。新兴霸主的崛起需要尽量积攒力量、避免分裂、广结盟友、自我节制。**
* **在经济、科技、军事力量方面，中国整体上有明显差距，但都在快速追赶并且在个别领域实现了超越。**

那么问题来了：好吧就算你说的对，硬杠起来中国能自保。可是美国代表着普世价值，代表着美好生活，全世界有头脑有良知的人都会支持美国的，他们用不着打仗也能不战而屈人之兵，你们拿什么跟美国斗？

文化，或者说话语权，的力量，的确非常、非常的强大和重要。

双标么？双标。

问题不在于双标。双标是人类的天性，你我皆然（大家可以回想一下，自己写年终总结的时候，和给下属 / 供应商 / 外卖小哥 / 淘宝卖家打分的时候，能不能一碗水端平）。

问题在于，人家双标的自然而然，双标的理直气壮，双标的天经地义。更大的问题在于，被双标的我们这边，还有大把的朋友觉得对方说的对，恨不得拼命的把更多的屎往自己身上抹。一边抹还自我感动觉得自己是良心代表。

这样熟悉的味道和配方，我们以前见识过很多、很多次。比如两年前中兴华为被制裁的时候，有无数的朋友觉得美国老爷一定是公证合理的。

这就是文化，或者说话语权，的力量。

在刘慈欣的《三体》里面，人类为了对抗三体人，发明了思想钢印。接受了思想钢印的人，对战争的胜利都会有着坚定的信念。

“不谈技术细节了，简单说吧，在大脑神经元网络中，我们发现了思维做出判断的机制，并且能够对其产生决定性的影响。把人类思维做出判断的过程与计算机作一个类比：从外界输入数据，计算，最后给出结果。我们可以把计算过程省略，直接给出结果。当某个信息进入大脑时，通过对神经元网络的某一部分施加影响，我们可以使大脑不经思维就做出判断，相信这个信息为真。”

——《三体》

思想钢印虽然是科幻中的技术，但是其原理却是完全真实的。

古代的宗教信仰，就是一种思想钢印。

生活在现代的人类，都知道生活在黑暗中世纪欧洲的人们被基督教会深度洗脑，大多都会有着一种莫名的智商优越感。

![](.evernote-content/D7E42CAD-E04E-4EB5-925B-496EC91526AB/A7CCC4BF-AD91-4CCB-B133-0AC422357A78)

问题是：难道当时的人类的大脑跟现在不一样么？当然一样。当时的人类智商普遍低于现在么？当然不是。当时的人类都知道自己被洗脑么？当然不知道。

问题就在这里。文化，或者说话语权的力量，就是潜移默化，就是润物细无声。用一个已经在中文语境里被高度异化的词，就是 “洗脑”。被文化或者话语权支配的人们，是不会意识到自己被洗脑的。

话语权，是通过对教育和舆论的掌控，尤其是垄断，来实现的。

古时候的很多国家，教育和宗教是合一的，只有祭司阶层，才能识字。印度的种姓制度，其实很传神的反映了咱们一直说的力量四支柱：军事、行政、经济、文化。

现代的祭司阶层，就是教育和大众媒体。

这就好比，在一个市场里，基本只有一家龙头公司在拼命的砸广告（老二在苦苦追赶中）。那么日积月累下来，老大的品牌力量，会得到多大的加成？

“谎言重复一千遍就是真理”，这的的确确是广告业的真理，也是全球文化市场的真理，也是全球思想钢印的实践原则。

面对这样的差距，是不是感到很绝望？是不是觉得，中国的文化力量，永远都追不上美国了？

你们知道，我的答案，肯定是 “不是”。

因为人类社会的文化力量的兴衰，同样有其规律。

美国的文化力量，是天生强大么？这是天大的玩笑。

一百年前的美国，虽然已经是世界最大的经济体，但是仍然是文化荒漠。美国的暴发户们，都以能够与欧洲旧贵族联姻为荣，比如露丝的绿帽未婚夫。

“中国人不像其他民族一样，没有任何迷信，任何可以自责的骗术。至少在四千多年以前，中国政府就已经向人们证明，现在还在向他们证明着，可以支配他们，但不欺骗他们；他们并不是通过谎言来为真理之神服务；迷信不但毫无用处，还会对宗教有害。从不崇拜上帝对中国来说是如此的纯洁和神圣。我不是讲人民的派别，而是讲君王的宗教，讲所有司法人员的宗教，所有非下等之人的宗教。这么多世纪以来，中国那所有诚实的人的宗教是什么呢？这就是：天道、正义。没有哪一个君王不这样。”

面对这样的吹捧，连我们都会脸红的。

而这在当时的欧洲并不是孤例。当时的欧洲，的确把中国当做学习的对象。英国在欧洲第一个建立了近现代的公务员制度，而学习的对象，就是中国的科举制度，所以英文中 “Mandarin” 同时兼有 “公务员” 和 “汉语普通话” 的意思。当然，也学到了官僚系统的各种毛病。

![](.evernote-content/D7E42CAD-E04E-4EB5-925B-496EC91526AB/40E4343F-24C5-4E02-887B-E187C6594DD8)

这是因为，在人类历史上，新兴文明在各项力量上相对老牌霸主的超越，基本都遵循着先从经济或军事超越、再到科技超越、再到文化超越的过程。

道理很简单。一方面，人民都是向往美好生活的，所以有着更高生活水平的社会和群体，就会自带光环。在另一方面，高水平文化产品（哪怕是广告）的生产，是需要金钱和昂贵的人才的。

现在的朋友，可能不太记得，在 80、90 年代，广东话在中国有多高的地位，会说广东话是多么时髦的表现，绝对比现在会英文牛逼多了。那个年代，大陆群众最向往的娱乐是港台剧，全民偶像是港台明星。

这是为什么？难道是大陆明星的演技超越了四大天王那一代的老戏骨么？

![](.evernote-content/D7E42CAD-E04E-4EB5-925B-496EC91526AB/0FD67F0E-AC6B-4DF7-AF77-D97775859C2A)

答案很简单。因为我们富起来了。

因为我们富起来了，所以我们有钱去消费文化产品，所以我们有钱去生产文化产品。

因为我们富起来了，所以我们越来越代表着美好生活，所以我们的国家形象（品牌）越来越好，也开始有了些许光环（至少对于亚非拉人民来说）。

这也是为什么，古时候，生活水平碾压周边所有已知世界居民的老祖宗可以 “中国有礼仪之大故称夏，有服章之美谓之华”。

![](.evernote-content/D7E42CAD-E04E-4EB5-925B-496EC91526AB/9966C546-8134-48CB-BDF3-78142418735F)

所以，就像我们在[前面章节](http://mp.weixin.qq.com/s?__biz=MzU3OTUzNDYyNQ==&mid=2247483965&idx=1&sn=dc86a5bb184c2e08d370b2e5464e1528&chksm=fd65e873ca1261651fdf38a9609c01148ac05d22c0d68bdeffcc601c42af84165293cda99e6b&scene=21#wechat_redirect)里所看到的，随着中国经济的持续发展，中国的文化力量，也终究会起来。

![](.evernote-content/D7E42CAD-E04E-4EB5-925B-496EC91526AB/96F148A1-0C7F-4273-8552-43796F4B21CB)

（2010 年美元不变价，数据来源世界银行）

我们要做的，是讲好我们的故事。不光要说清楚我们不是什么，更要讲清楚我们是什么。要提出我们的价值主张，要建立我们的品牌。

![](.evernote-content/D7E42CAD-E04E-4EB5-925B-496EC91526AB/6E31A03B-0383-4052-99AC-FD78B14E9F8A)

（曾经被国内外无数人嘲笑的 “人类命运共同体” 的概念，在如今肆虐全球的新冠病毒面前，显得如此的重要和真实）

中国的文化内核和现状，同样有着极为强大的优势。

虽然我一直喜欢说人性是相通的，但是民族和群体的共同记忆，却是独特的。

这段时间带孩子，陪着孩子一起读童话。突然发现我们小时候一直以为非常高大上的西方童话，是那么的三观不正。

比如主人不在家，不请自入而且偷吃偷用却觉得天经地义（白雪公主）。比如最大的追求是钓凯子傍土豪（灰姑娘）。比如食物不够吃了，就把孩子抛弃在森林里饿死（森林里的糖果屋）。比如最后能获得美好生活的基本都是贵族（王子和公主从此过上了幸福的生活）。。。

我心头无数头草泥马狂奔而过。

![](.evernote-content/D7E42CAD-E04E-4EB5-925B-496EC91526AB/1669BBDB-1184-4544-B3F1-0DA11B9D9FFE)

这样的价值观，在 21 世纪的今天，不论是中国还是欧美，都非常的不正确。或者至少，绝对不是我们愿意教给孩子的价值观。

但是这样的价值观，与西方的历史记忆，是一致的。那就是在灾难面前，选择逃避（诺亚方舟）。

而现代的神话传说 —— 电影里面，好莱坞片子突出的都是个人主义，都是绝对不能舍小家的个人英雄，是为了逃命或者追逃犯可以随便飙车只要我能活命哪管路人洪水滔天的价值观。

就在昨天，英国政府刚刚承认，英国的策略就是让大多数国民都感染新冠病毒，从而获得群体免疫。这个也是德国，以及可能很多其他欧洲国家，的整体抗疫策略。

![](.evernote-content/D7E42CAD-E04E-4EB5-925B-496EC91526AB/D33285F7-AA63-4BF1-A5A4-13C51E99FEB4)

对于这个策略，我要拷贝一下某不知名朋友的评论：

“这个防疫方法非常尊重科学，体现了发达国家抗疫治疫的科学性。用达尔文的方式，物竞天择的抗疫，体现了现代科学的独特魅力；体现了 “顺其自然无为而治” 的太极神韵；更体现了发达国家队理论自信、科技自信、体魄自信、和制度自信。这种抗疫方法叫 “生死有命富贵在天” 疗法，是一次史诗级的发明创造，一次造福人类的伟大实验。”

这样的价值观，在东西方物质生活水平相差极大的二三十年前，还是有着很大的吸引力的。

但是，随着我们越来越富有，越来越自信，我们的电影，也越来越回归我们共同的历史记忆。那就是（对亲人、对友邻、对众生的）责任和（自我的）奋斗。

面对灾难，我们也会求神，也会拜佛，但更重要的，是依靠自己去补天、去治水、去移山、去填海，是战天斗地，是我命由我不由天。

![](.evernote-content/D7E42CAD-E04E-4EB5-925B-496EC91526AB/A0FAAAA2-3BB2-4FBB-8FC3-1C1DF3826FB0)

即使我们的天真的要塌了（太阳氦闪），我们也不抛弃不放弃，我们也一定要带着家园共同去流浪。

![](.evernote-content/D7E42CAD-E04E-4EB5-925B-496EC91526AB/02F90293-D69C-4D7D-B985-17D2863A14F0)

这样自强不息的精神内核，与宣扬躺赢的个人主义相比，到底哪个更普世？哪个更代表人类的追求和价值？

![](.evernote-content/D7E42CAD-E04E-4EB5-925B-496EC91526AB/3E99744A-13C8-4F72-A1F8-5C04B9CE2AC9)

![](.evernote-content/D7E42CAD-E04E-4EB5-925B-496EC91526AB/042B1E27-7363-442C-8B98-4176D5135960)

![](.evernote-content/D7E42CAD-E04E-4EB5-925B-496EC91526AB/C7E04D17-E976-4F7F-916F-CE27D6F5CA01)

答案是很明显的。

我们所欠缺的，只是需要时间和金钱来打磨的品牌故事，需要时间和金钱来堆砌的品牌知名度和美誉度，以及需要我们生活水平不断提高带来的品牌光环。

时间，在我们这边；未来，在我们这边。

==============================

前面，我们分四个章节，分析了中国和美国力量的对比。那么，整体来看，中国到底处于什么样一个阶段？中国更像什么时候的美国？

|  |  |  |
| --- | --- | --- |
|  | 中国 | 美国 |
| 人均 GDP  （2011 美元不变价） | 12,569 美元  2016 年 | 12,844 美元  1941 年 |
| 城市化率 | 58.5%  2017 年 | 56.5%  1940 年 |
| 第二产业 GDP 占比 | 40.5%  2017 年 | 40.7%  1879 年 |
| 人均能量消费 | 88  MBtu  2017 年 | ~100  MBtu  1850-1885 年 |

上表显示的答案可能出乎绝大多数朋友的预料。身处 21 世纪，能读到这篇文章的朋友们生活中都围绕着很多人类最新的科技，生活在人类历史上基础设施最发达的城市中。

所以，大家可能很难想象，我们现在真实的发展阶段，是将近一百年前的美国。

一百年前的美国是啥样子呢？这是最好的时代，也是最坏的时代。这是贫富差距最大的时代，也是道德沦丧的时代。这是经济快速增长人民生活水平快速提高的时代，也是贪污腐败横行的时代。

二十世纪初的美国，是生机勃勃，潜力无限的国度。二十世纪初的美国，是大英帝国世界霸权体系内的 No.2，是一个净债权国，是科学文化落后的蛮荒之地，是一个每十年就要发生一次经济萧条（注意是 GDP 负增长，不是仅仅增长率下降）的国家，同时美元也还不是主要储备货币。

![](.evernote-content/D7E42CAD-E04E-4EB5-925B-496EC91526AB/249A8629-20AD-4B78-80DF-2F5CC5EBF0FB)

看到这里，有没有很熟悉的既视感？是不是觉得这就是我们伟大的祖国？

后面的故事，大家都知道了。

当然历史不会简单的重复。但是，我希望强调一个我们所拥有的巨大的战略优势，那就是我们相对落后的发展阶段。因为这意味着，我们还有非常非常大的空间去提升我们的经济、科技、文化水平；因为这意味着，我们未来还有着巨大的增长潜力。

时间，在我们这边。

未来，在我们这边。

那么问题来了：就算中国自身的力量在不断增长，可是美国有着那么多盟友，而中国没有一个。你太失道寡助了！怎么破？
