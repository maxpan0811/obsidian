# AirPort Time Capsule 802.11n (第 3 代) - 技术规格

---

AirPort Time Capsule 802.11n (第 3 代) - 技术规格
===========================================

### 尺寸与重量

* 长度：19.7 厘米（7.7 英寸）
* 宽度：19.7 厘米（7.7 英寸）
* 高度：3.63 厘米（1.4 英寸）
* 重量：1.6 千克（3.5 磅）1

### 网络安全

* Wi-Fi Protected Access™（WPA/WPA2）2
* 可为无线安全（WEP）进行 40 位和 128 位加密
* MAC 地址过滤
* NAT 防火墙
* 支持 RADIUS 验证
* 802.1X、PEAP、LEAP、TTLS、TLS、FAST
* 基于时间的访问控制

### 电器性能

* 电压：100 至 240 伏特交流电
* 频率：50 至 60Hz，单相
* 最大持续功率：34 瓦

### 安规认证

* FCC Part 15 Class B、Canada RSS-210、EN 300-328、EN 301-489、EN 301 893、ARIB STD-T66、RCR STD-T33、AS/NZS 4268: 2003、UL 60950、CSA-C22.2 No. 60950

### 存储设备

* 1TB 或 2TB Serial ATA 服务器级硬盘3

### 接口

* 一个千兆广域以太网端口，连接 DSL 或线缆调制解 调器
* 三个千兆局域以太网端口，连接电脑或网络设备
* USB 端口，连接 USB 打印机或 USB 外接硬盘
* 802.11n 无线技术

### 包括软件和附件

* 用于 Mac 与 Windows 的 AirPort Utility
* 用于 Windows 的 AirPort Disk Utility
* 用于 Windows 的 Bonjour
* 电源线
* 印刷及电子文档

### 使用环境

* 操作温度：10° 至 35°C（50° 至 95°F）
* 存放温度：-25° 至 60°C（-13° 至 140°F）
* 相对湿度（操作）：20% 至 80%（非凝结）
* 相对湿度（存放）：10% 至 90%
* 最高操作高度：3,000 米（10,000 英尺）
* 最高储存高度：4,500 米（15,000 英尺）

### 无线协议

* IEEE 802.11a/b/g/n

### 兼容性

* 与符合 Wi-Fi 认证的 802.11a、802.11b、802.11g 或 802.11n 规格的 Mac 机、iPod touch 和基于 Windows 的电脑
* NAT、DHCP、PPPoE、VPN Passthrough (IPSec、PPTP 和 L2TP)、DNS Proxy、SNMP、IPv6 （6to4 和手工配置隧道）

### 频率范围

* 2.4GHz 和 5GHz 可同时使用

### 射频输出功率

* 20 dBm（额定）

### 频道

* 美国和加拿大可用频道 1-11、36-48 和 149-165
* 日本可用频道 1-13、36-64 和 100-140
* 欧洲可用频道 1-13、36-64、100-112 和 132-140
* 澳大利亚、香港和新西兰可用频道 1-13、36-64 和 149-165

### 系统需求

|  |  |  |
| --- | --- | --- |
|  | **Mac 用户** | **PC 用户** |
| **使用 Time Machine 进行备份** | Mac OS X v10.5.7 或更高版本的 Mac 电脑 | - |
| **安装与管理** | 配有 Mac OS X v10.5.7 或更高版本的 Mac 电脑、光驱、以太网或无线网络性能 | 配有 Windows XP (SP3) 或 Windows Vista (SP1) 的 PC、光驱、以太网或无线网络性能 |
| **无线用户访问** | 配有 AirPort 无线性能的 Mac 电脑 | 配有 Wi-Fi 认证的 802.11a/b/g/n 卡的 PC 电脑 |
| **通过 USB 打印机共享打印** | USB 打印机，配有 Mac OS X v10.2.7 或更高版本的 Mac 电脑 | USB 打印机，配有 Windows XP (SP3) 或 Windows Vista (SP1) 的 PC 电脑，Time Capsule 光盘中已包括用于 Windows 的 Bonjour |
| **带 USB 硬盘的 AirPort Disk** | 配有 Mac OS X v10.4.8 或更高版本的 Mac 电脑 | 配有 Windows XP (SP3) 或 Windows Vista (SP1) 的 PC 电脑，Time Capsule 光盘中已包括用于 Windows 的 Bonjour |

### 更多服务和支持

为你的电脑提供的 [AppleCare 维护计划](http://www.apple.com/cn/support/products/mac.html)服务计划涵盖 Time Capsule。

1. 重量会因配置和生产工艺的不同而有变化。
2. Wi-Fi Protected Access (WPA/WPA2) 支持要求 Mac OS X v10.3 或更高版本，或 SP2 或更高版本的 Windows XP
3. 1TB = 1 兆个字节；格式化后的实际容量可能会变小。
4. 该产品需经 CCC 认证，经核准后方可发售。

Wi-Fi Protected Access 是 Wi-Fi 联盟的标志。无线互联网访问需要具有无线上网功能的电脑、基站或其它访问接入点，以及互联网连接（可能需要支付一定费用）。部分 ISP 暂不支持 Time Capsule。产品规格会根据情况变动，恕不另行通知。
