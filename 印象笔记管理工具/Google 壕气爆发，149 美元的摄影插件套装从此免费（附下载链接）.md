# Google 壕气爆发，149 美元的摄影插件套装从此免费（附下载链接）

---

[![](.evernote-content/FBCBE914-C1A8-43B3-B944-E1DEAC469E24/421D64AE-C42B-4650-8856-5F2F7785D14C.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/03/0121.jpg)

知名的摄影后期软件开发商 Nik Software [昨天宣布](https://plus.google.com/+NikCollection/posts/AFGsG2Di7EK)，自家开发的后期插件 Nik Collection 从即日起供所有用户免费下载。

Nik Software 的插件作品可能只在摄影爱好者中流行，但是他们开发的手机应用 Snapseed，还有 Google Photos 内置的图像编辑工具都在大众用户中收获了非常不错的评价。

2012 年，Google 为了 Snapseed 而[收购了 Nik Software](http://www.ifanr.com/156013)，随后不久就把 Nik Collection 总共 7 个优质插件的集合包售价从 499.95 美元降低至 149 美元。就在昨天，这个集合包终于走向了免费。

[![](.evernote-content/FBCBE914-C1A8-43B3-B944-E1DEAC469E24/990F8E02-EA42-44A8-AEC2-710491B087CC.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/03/29.pic_2.jpg)

这 7 个插件分别为：

> Analog Efex Pro 2——模拟胶片相机
>
> Color Efex Pro 4——用于颜色校正、修整图像和提供创意效果，共有 55 款滤镜
>
> Silver Efex Pro 2——可以模仿近 20 种胶片类型，制作黑白照片
>
> Viveza 2——局部调整图片的颜色和色调
>
> HDR Efex Pro 2——制作 HDR 照片
>
> Sharpener Pro——图片锐化工具
>
> Dfine 2——降噪工具

Nik Collection 兼容 [Windows](http://pan.baidu.com/s/1gfPN1RH) 和 [Mac OS](http://pan.baidu.com/s/1pLL3xgR)，点击[这里](https://www.google.com/nikcollection/products/analog-efex-pro/)即可下载。

附简单的安装教程。

首先，打开下载完毕的安装包，根据提示进行安装。7 个插件所需空间为 3.54 GB，安装前准备好充足的硬盘空间。

[![](.evernote-content/FBCBE914-C1A8-43B3-B944-E1DEAC469E24/35357CA3-47C3-49A9-9343-A5420B9CB81A.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/03/17.pic_1.jpg)

[![](.evernote-content/FBCBE914-C1A8-43B3-B944-E1DEAC469E24/137F9DF6-D8BA-492E-AE15-A78760A359FE.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/03/18.pic_.jpg)

[![](.evernote-content/FBCBE914-C1A8-43B3-B944-E1DEAC469E24/599D56E7-C443-4A65-AAE2-6C2E31ED306D.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/03/19.pic_1.jpg)

[![](.evernote-content/FBCBE914-C1A8-43B3-B944-E1DEAC469E24/5D43B516-7D8C-4CCB-92AA-974402D069AB.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/03/20.pic_.jpg)

[![](.evernote-content/FBCBE914-C1A8-43B3-B944-E1DEAC469E24/68D001DD-87EA-4150-8EB2-AFE544E49673.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/03/21.pic_.jpg)

安装完毕后可以在 Launchpad 中看见 7 个插件的图标，点击打开后按快捷键 Command+O 可以打开想要处理的图片。

[![](.evernote-content/FBCBE914-C1A8-43B3-B944-E1DEAC469E24/20817C5E-49C9-4DBB-8FB2-38D6CD3F91E6.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/03/23.pic_1.jpg)

界面左侧是内置的滤镜，也支持自定义预设，大家也可以在右侧手动调整各项参数。我使用 HDR Efex Pro 2 的 07 号“鲜明”滤镜还有 Silver Efex Pro 2 的 005 号“高细节强度（严格）”滤镜处理了一张航拍照片，大家可以在下面这张图片中对比一下和原图的区别。

[![](.evernote-content/FBCBE914-C1A8-43B3-B944-E1DEAC469E24/24A15B13-4050-456A-96DA-5946D6156891.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/03/26.pic_hd1.jpg)

[![](.evernote-content/FBCBE914-C1A8-43B3-B944-E1DEAC469E24/9F79A3E1-675D-4F9D-8D06-7D6BBA5A4196.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/03/25.pic_hd1.jpg)

[![](.evernote-content/FBCBE914-C1A8-43B3-B944-E1DEAC469E24/CD2BB362-5B77-488C-B84B-C03E260141CC.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/03/DJI_0072.jpg)

（原图）

除了在插件内打开照片以外，在照片上单击右键也可以直接选择使用特定的插件进行编辑。

[![](.evernote-content/FBCBE914-C1A8-43B3-B944-E1DEAC469E24/CE61FBA9-5C89-4A12-8E12-27D76CBEBF78.jpg)](http://images.ifanr.cn/wp-content/uploads/2016/03/27.pic_.jpg)

在被 Google 收购之后，Nik Software 有了更强大的资金支持，不再需要依靠出售插件或软件获得收入，此时将自己最受欢迎的产品组合全部免费，不仅不会影响到收入，还能够借此机会让更多摄影爱好者了解自己的产品，Nik Software 这次的做法也算是挺高明的。
