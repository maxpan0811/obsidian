# 享受非凡音效 玩转不同平台——罗技环绕声游戏耳机麦克风

        拥有全息影像眼镜是世界上绝大多数玩家心中的梦想，全体感游戏系统更是异想天开的空中楼阁。抛开镜花水月般的妄想，回归现实，你会发现，身临其境的游戏体验已然触手可及。罗技G633 ARTEMIS SPECTRUM RGB 7.1 环绕声游戏耳机麦克风将携着女神阿尔忒弥斯的凛冽月光而来，实现玩家们真实体验游戏之梦。无论是现代战争的硝烟弥漫枪炮齐鸣，顶级赛道的马达轰鸣人声鼎沸，古战场的热血沸腾杀声震天，还是体育馆内的声嘶力竭摇旗呐喊，都将成为你们的胜利交响曲。玩家们不论是在PC,还是PS4/XBOX One甚至是移动终端都可以享受到非凡音效带来的游戏体验。

![](.evernote-content/8C30CD4B-C04D-42C3-BBD3-16596D306F3A/3CB21591-83BD-49B3-8ED4-74C7CC5858A0.jpg)

来自罗技游戏业务副总裁兼总经理Ujesh Desai表示：“我们发现并没有公司真正致力于为游戏玩家打造高品质的头戴式耳机。为此，我们向我们的音频团队发起了挑战，看他们是否能够做到这一点。作为一名游戏玩家兼音乐家，当看到音频团队的高品质头戴式耳机设计时，我感到十分兴奋。”

**非凡音效，身临其境**

通过脚步声判断敌人的位置以及距离是FPS游戏中必不可少的技能，然而声音来源的真实性与准确性导致了听音辨位的精确度，在瞬息万变的现代战争中错失时机不仅仅只是失去一次击杀的机会，甚至决定了一场游戏的胜利。罗技G633 ARTEMIS SPECTRUM RGB 7.1 环绕声游戏耳机麦克风搭载的Pro-G?音频驱动单元采用了混合网格材料，能够提供游戏迷们想要的发烧友级性能。Pro-G驱动和环绕声技术能够在游戏中创造逼真的空间意识，从而让玩家清晰听到敌人蹑手蹑脚的声音或者是特殊技能线索，让敌人无处遁形。

杀声震天，血肉横飞的古战场是玩家们无法体验的梦幻游戏场景，G633搭载的7.1杜比环绕声能够利用校准和处理让游戏玩家和音频爱好者享受到更加贴近录音棚音质的音频体验，支持高达11个音频声道以及多项个性化选择。玩家们头戴耳机就如处硝烟四起的古战场中，指挥千军万马，东征西讨，享受战争带来的热血沸腾。

![](.evernote-content/8C30CD4B-C04D-42C3-BBD3-16596D306F3A/BAF2EA83-4736-410F-943F-68E4637F182E.jpg)

**多平台兼容，耳听八方**

Plastaytion4和Xbox One强势进入中国市场,大量史诗级独占游戏走进玩家们的视野，单纯的PC游戏已经不能满足玩家们日益庞大的胃口。高端玩家中主机与PC并存的情况已经屡见不鲜，就连手机游戏都要在基数庞大的游戏玩家中分一杯羹。罗技G633 ARTEMIS SPECTRUM RGB 7.1 环绕声游戏耳机麦克风巧妙的解决了主机外设与PC外设不兼容的问题。玩家只需一部G633就可以轻松应对PC，家用主机乃至移动终端的所有游戏平台。

![](.evernote-content/8C30CD4B-C04D-42C3-BBD3-16596D306F3A/C20EF0C9-7CBE-49BA-A304-1D3879CC9744.jpg)

**RGB炫光，个性编程**

RGB灯光已经不知不觉成为高端游戏外设不可或缺的元素，从机械键盘到电竞鼠标都可以见到RGB灯光的影子。罗技G633 ARTEMIS SPECTRUM RGB 7.1 环绕声游戏耳机麦克风亦不可免俗。不仅搭载外设前沿的可调节RGB背光还继承了罗技G系列独家的可编程G键设计。从RGB背光到G键，从自定义标签再到自定义声音配置文件，都使玩家的耳机麦克风成为玩家自身的真实写照。玩家可以使用罗技游戏软件对多达1680万颜色的RGB背光进行调整，并在三个可编程G键上配置自定义游戏宏，从而享受无上游戏体验。

![](.evernote-content/8C30CD4B-C04D-42C3-BBD3-16596D306F3A/931DA3B8-57B6-4C0E-82ED-7AE760941004.jpg)

**多源混音，引爆激情**

游戏音效我所欲也，与战友语音交流亦我所欲也，然现在两者皆可兼得。罗技G633 ARTEMIS SPECTRUM RGB 7.1 环绕声游戏耳机麦克风配备多源音频混音系统，在不影响游戏音效的同时也能清晰的与战友互通信息，不论是语音平台还是视频通话都不再将是你游戏的绊脚石，战友的呼喊声回响在你的耳畔，枪炮齐鸣是为你奏响胜利的凯歌。你，不是一个人在战斗。
