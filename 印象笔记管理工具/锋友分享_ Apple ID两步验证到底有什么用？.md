# 锋友分享: Apple ID两步验证到底有什么用？

---

[![](.evernote-content/443BF064-F14E-4641-8368-47E948D69DF9/14A59C52-1144-42BC-9091-E6071AE0BF3C.png)](http://www.feng.com/iPhone/news/2016-01-19/Feng-friends-sharing-Apple-ID-two-steps-to-verify-what-s-the-use_636506.shtml)

　近日，有不少锋友表示自己遇到了 Apple ID 被盗的问题，而开启两步验证则是防止 Apple ID 被盗的一个非常重要的方法。日前，锋友“边子渊”分享了“两步验证”这一功能的真实作用，一起来了解一下吧。  
  
　在进入主题之前，我们先来熟悉几个概念：

![](.evernote-content/443BF064-F14E-4641-8368-47E948D69DF9/14A59C52-1144-42BC-9091-E6071AE0BF3C.png)

1、什么是“Apple ID”?

Apple ID 是苹果公司为其产品(如 iWork、iTunes Store 和 Apple Store 等等)所引入的认证系统。Apple ID 作为一个全功能于一身的帐户，允许用户访问苹果的各种资源。

2、什么是“两步验证”?

Apple ID 两步验证是一种更加有利于账号安全性的服务，开启该服务后，你在管理自己的 Apple 账号或进行修改密码时都需要进行此项验证，它会给你设置的受信任的 iOS 设备或密保手机发送验证码，相当于一个动态口令。

3、“两步验证”目前的使用场合有哪些?

(1)登录 Apple ID 管理界面时的验证。

(2)修改 Apple ID 密码、重置 Apple ID 密码时的验证。

(3)在新设备上登录 Apple ID 时的验证。

(4)登录网页版 iCloud 时，登录过程不需要两步认证，只需要密码，即可直接访问“查找我的 iPhone”这一核心安全功能，以及在“设置”中看到和此 ID 绑定的所有 Apple 产品。而只有在你需要使用通讯录、邮件、日历、备忘、照片、Pages、Numbers 以及 Keynotes 这些和个人信息与文稿数据相关的功能时，才需要两步验证。如下图所示：

![](.evernote-content/443BF064-F14E-4641-8368-47E948D69DF9/78A3BFC8-4A66-44C1-8EFF-056002C06338.png)

那么，我们执行丢失模式(锁定)、抹除等操作的时候，需不需要密码呢?​​

![](.evernote-content/443BF064-F14E-4641-8368-47E948D69DF9/22C3FC77-D5FA-4301-984D-EE1C137EDD9D.png)

锁定：锁定不需要两步验证。

![](.evernote-content/443BF064-F14E-4641-8368-47E948D69DF9/3FA9FB69-B7A2-4718-8CF2-6C8ED0922485.png)

抹掉：​抹掉也不需要两步验证。

虽然苹果公司这么做的确是方便了大家在丢失设备时的快速找回与锁定，但是只需要一个 Apple ID 和其对应的密码就能快捷锁定和抹掉设备，在这个过程中，两步验证并没有起到保护设备的作用。

因此，锋友“边子渊”表示，从目前的情况来看，设置一个强有力的密码就是对设备最大的保护手段之一。而如果你想要确保个人账户密码安全、个人信息和文稿安全隐私，两步验证则是一个相当不错的选择。

另外，如果你的 iOS 设备被恶意锁定了，可以凭购买凭证到 Apple Store 或授权维修点解锁(可能会产生部分费用)，不要妄图从网络上寻求帮助，谁知道你的设备是偷的还是捡的。最后，锋友们可以点击[这里](http://bbs.feng.com/read-htm-tid-10242644.html)进入原帖，了解更多关于提升 iOS 设备和 Mac 设备安全性的具体操作。
