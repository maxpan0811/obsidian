中央重磅政策背后：谁在阻碍全国统一大市场的建立？
========================

沃顿财经视界
*2022-04-14 16:41*

以下文章来源于大树乡谈
，作者镇长本人

![](.evernote-content/DEBF21D7-D515-4529-85E3-C80C3769A26B/49516C9C-3923-464B-9317-BABD54476C6C)

**- 1 -**

**市场是最稀缺的资源，中国需要打造一个统一的大市场，来获得未来发展的关键动力。**

**中国难道现在还不是统一的全国市场？**

**中国当然是一个整体的市场，全国使用一样的语言、遵守相同的法律、都属于同一个民族“中华民族”等等，相比印度、欧洲又或者美国这样的联邦体系，当然是一个整体，但是整体不意味着“统一”。**

事实上，中国到现在国内市场分割的状况仍然非常突出，不要说地区之间、各省、各市之间往往也存在分割的现象，集中体现在两点：

**第一点：国内各地发展远未形成合力。**

以中国这样的规模体量，比较优势理论对于国内各省也有一定价值。各省、各地之间发展阶段不同，资源禀赋也有不同，理想状态下，各地按照不同的比较优势发展适合的产业，对整个国家来说无疑是最优的，但现实并非如此。

**各地重复建设、产能过剩、债务高筑等问题非常突出。**全国各地非常热衷于上马建设周期短、投资收益大、未来发展好、符合国家政策方向的产业，对于本地是否适合考虑不足，结果导致盲目上马相似项目，就出现了我们经常说的“一哄而上”“一哄而散”的问题。

比如说，连续几年中央一号文都强调发展现代农业，也投入了很大财政资金用来扶持高质量农田建设、智能农业等项目，结果全国光提出实现农业现代化的就高达28个省份。问题在于这里面不少省份农业产业规模连年下降，连基本的耕地红线都守得艰难，投入也不足，提农业现代化有意义吗？

又比如这几年国家强调新型基础设施建设、数字产业、生物医药等，又是一拥而上，小镇梳理了下各省的“十四五”规划，光提出发展“人工智能”的省份就高达25个，想搞生物产业的有29个，医药也是29个，健康产业也有27个。

其他如新能源、新材料、汽车制造等，除了个别实在搞不了的，也基本上每个省都上了，

不得不说，有时候梳理下各省的“十四五”规划，看看各省搞啥不搞啥，如何定位，再对比下“十三五”规划，也是一件挺有意思的事，这个以后再谈，有兴趣的可以先自己了解下。

坦诚地说，这些产业对于资金、技术、人才、供应链等的要求可着实不低，全世界能搞的国家都没多少。现在各省一拥而上，往好处看体现了各省对国家政策的重视，如此大规模的投入也能增加中国在这些方面的基础；但是弊端也很大，宝贵的资金、土地、政策、人力必然被大量消耗。

**第二点：对外贸易发展显著高于国内贸易。**

正如第一点所提到的，中国这样的大国，在一些贸易特征上，各省也有些类似一般的国家，毕竟全世界人口大于2000万人的国家也不过60多个，而中国人口大于2000万的有26个，排前二的广东和山东在全世界都能进前15名。

![](.evernote-content/DEBF21D7-D515-4529-85E3-C80C3769A26B/E7BCFAFE-CFE5-4FD7-B580-E738FC5A8165)

**按理说中国是一个整体市场，内部交易无论如何都更方便一些，但结果是中国的对外贸易发展要明显好于国内贸易尤其是省际贸易。**

是的，省际贸易，这是一个非常重要的指标，背后体现的就是各省之间贸易的自由程度。

改革开放以来，中国出口额年均增速达18%，2021年同比增速更是达到26.1%，然而国内省际贸易的年均增速还不到5%。

这是一个难以想象的情况，中国对外贸易竟然比国内贸易发展还要迅速，这无论如何都不能说是一个统一的大市场。

**这两个还仅仅是一小部分比较突出的点，但已经足以看出，现在的中国国内市场，只能说是一个整体市场，但内部存在大量的市场分割，到处是“小而全”的局部小循环，远远称不上“全国统一大市场”。**

以这样的情况，是不可能充分发挥14亿人大市场的规模优势的，这也是中国对外贸易和国际博弈时，往往感觉有心无力的根本原因。

中国现在仍然是一个颇为臃肿的大胖子，看上去全是肉，倘若拔河的时候确实不容易拔动，但是真要是上了拳击台，碰到同等甚至更强的对手，一时落于下风也是自然的事情。

我们切切不要觉得中国已经很强了。

**- 2 -**

昨天小镇发文之后，有读者朋友转给小镇一篇文章，这篇文章说：中国之所以没能形成全国统一的大市场，是因为西方资本的影响，如果搞统一大市场就会得罪西方资本家。

|  |
| --- |
| [点击此处查看](https://mp.weixin.qq.com/s?__biz=MzI2MDgyNzYwNw==&mid=2247485220&idx=4&sn=38cfb27221ef6784c565a64c66ab974f&scene=21#wechat_redirect) |

不得不说，这样的言论确实猎奇，非常有流量价值。但是如果想要深刻、正确理解中央这个政策，并且帮助自己成长、乃至获得个人发展优势，这种观点一定要摒弃。

确实，中国发展到世界第二大经济体，改革开放以来也确实吸收了大量海外资本，要说国内没有同这些资本联系密切的群体并不现实。但妖魔化海外资本，更是“高级黑、低级红”，是对中国未来发展的严重损害。

中国经济的突飞猛进，就是从2001年加入WTO、融入全球经济开始的，在中国经济发展上，海外资本对中国利大于弊，否则国家也不会一再向海外资本发出积极信号、强化海外资本对中国未来的发展预期。

总不能说国家对海外资本的判断都错了吧？

小镇一向摒弃阴谋论、斗争论，这两点确实是流量密码，特别吸引人，但并不符合现实。需知，真实世界是极为复杂的，越复杂、环环相扣的谋划失败的概率就会激增，看看二战中的几个重大战役，出现了多少阴差阳错，所有的战略家的目标都是尽可能简化环节、以堂堂正正的姿态不断获得优势。

只有个人、组织或者小国，才有可能不顾一切地赌一把，试图以邪道谋得一时收益。

更何况，对中国这样的大国来说，一定要记住最根本、最本质的一定是国内因素，这个国内就包括国内的经济运行机制。

**在建立“全国统一大市场”上，我们面临的最大挑战，恰恰是过去让我们飞速发展的传统机制。**

谈这个问题前，小镇想先介绍一个中国经济发展理论，著名的“**政治锦标赛体制**”理论，这是由周黎安、李宏彬等学者在2004-2007年间一系列研究成果给出的，从地方官员的晋升激励这个角度系统的归纳了中国过去经济发展的体制。

|  |
| --- |
| [点击此处查看](https://mp.weixin.qq.com/s?__biz=MzI2MDgyNzYwNw==&mid=2247485220&idx=5&sn=07e6d83f8e43c249562525ecf9cf5843&scene=21#wechat_redirect) |

**核心观点是：在中国的体制下，上级官员主要依据经济增长这一指标来考核和提拔下级官员，因此下级官员就有极为强烈的动力来发展辖区经济，以获得政治上的升迁。**

这一体制背后由一系列中国独有的体制组成，从而构成了中国独一无二的经济发展模式，通俗说就是我们常说的“唯GDP论”。

这一模式，把各级地方政府打造成了“准市场主体”，各级政府就像一家家集团公司一样，极尽全力发展辖区经济，以搞了多少项目、拉了多少投资、税收增加多少、GDP增长速度如何等指标，直接对全国各级政府官员进行量化和比较。

谁在任期间，经济指标搞得好，晋升的概率就会显著提高。而各级政府下属各部门、各公职人员、国企等等，都要为这一目标而努力，比如每年很多地方会组织宣讲团到经济发达地区宣讲当地招商政策，也有很多地方分派了招商引资任务。

这一体制极大地促进了中国的经济发展，尤其在市场经济发展仍不充分的情况下，中国相当于拥有了数十个世界顶级集团、数百个乃至上千个世界500强级别的次级集团，再叠加中国的体制优势，爆发力极为巨大。

当然有利就有弊，好的一方面我们已经看到了，中国进入WTO后仅仅20年就已经开始向世界第一经济体跃进，这是人类从来没有的奇迹；但另一方面也带来了一系列问题，比如土地财政、地方债，又比如前面讲到的重复投资。

而现在随着中国经济发展达到更高的高度，一个更大的负面作用越来越突出，那就是《中共中央 国务院关于加快建设全国统一大市场的意见》中专门提到的：**打破地方保护和市场分割，打通制约经济循环的关键堵点，促进商品要素资源在更大范围内畅通流动。**

其目标就是打造“**高效规范、公平竞争、充分开放的全国统一大市场**”。

而在整个意见中，地方保护、分割市场，一直是要着力解决的关键。

**- 3 -**

**所以，是什么在阻碍全国统一大市场的建立呢？**

**个体或者部分群体的利益阻碍当然存在，但这绝不是主要的，甚至算不上次要因素，仅仅是表象，真正的阻碍是我们曾经依仗的经济发展机制。**

遇到问题，想找一个所谓地“罪魁祸首”，树一个具体地靶子，这是普通人地朴素思维，总希望有一个具体的、看得到、摸得着的目标，好像只要把这种现实存在地实体“打倒”就可以了。类似于各类历史戏说剧，比如《康熙微服私访记》之类，总有一个作恶多端的坏人，打倒了就行了。

但真的是这样吗？如果是这样，那改革岂不是太容易了吗？

**改革之所以难，难就难在是以一个新的体系、机制去升级、完善乃至替换现有的体系、机制，这是看不见、摸不着的，不仅存在具体的实体，也存在于摸不着地方比如人心。**

所以，这次改革真正的目标，政策原文已经说得很清楚了，就是制度和体制层面，比如：

**四个基础制度规则的统一：**产权保护制度、市场准入制度、公平竞争制度、社会信用制度。

**五个统一的要素和资源市场：**土地和劳动力市场、资本市场、技术和数据市场、能源市场、生态环境市场。

**为什么专门提出这四个作为基础制度规则？**

**这四个之间、政府和市场之间关系如何？**

**为什么要打造统一的要素和资源市场？**

**这些市场又该如何建立、如何运行、个人有什么机会？**

等等，问题很多，但不要急，慢慢来。这个政策非常重要，小镇会花更多精力一点点讲，每篇只讲1到2个问题，尽可能用上切合的例子，更加通俗化，帮助大家理解。

**- 4 -**

**改革绝不是彻底否定过去，一定不能走极端。好的、优势的要尽可能继承，在此基础上把短板延长，如此不断完善、优化，才是良性的改革之路。**

治大国如烹小鲜，就在于此。

**那么如何继承发展过去的“政治锦标赛体制”呢？**

**首先，当然不能废除掉。**

中国经济整体仍然远远落后于先进发达国家，经济总量虽高，但人均不足、发展不均衡问题仍然突出，这些问题需要靠分配进行调节，但是以中国目前的总量，无论如何优化分配也是捉襟见肘，还是要开源，把蛋糕做大。

地方锦标赛的制度，充分发挥地方政府发展经济的积极性，这是中国未来经济仍然保持持续中高速发展的关键，倘若地方政府不去主动发展经济，各地区间竞争弱化，那就走入了另一个极端，我们很难取得对美欧等国的竞争优势。

**其次，保持地方政府发展经济的动力，不代表不进行调整，关键在于调整地方政府的绩效考核体系。**

不单单仅以传统经济增长数据作为考核，而要根据我们的未来发展目标、当前实际情况，对现有的考核体系进行调整，朝着高质量发展、促进全国统一市场的角度适度倾斜。

**这种调整，必然是长期、分阶段、有层次。**

比如从区域上，首先是省内市场统一，然后才是长三角、珠三角、京津冀这样的区域市场统一，最后才逐步过渡到全国。

而在具体领域上，如网络、数字、生态等已经有一定统一特征的，将先行一步，以点带面，不断为撬动更深层改革积蓄力量和支持者。

而最终的目标很简单，就像欧美100年前就已经完成的那样，实现国内基本市场规则的一致，彻底将整个中国国内市场打造为一个整体。一旦完成，中国将是毫无置疑的世界第一经济强国，如果以GDP总量来测算，大概将是美国的150%以上，而以中国为核心将形成一个东方经济圈，足以与整个西方经济体并驾齐驱。

届时，整个国际秩序将发生根本变革。这样的未来，将有着无数改变人生的机遇。
