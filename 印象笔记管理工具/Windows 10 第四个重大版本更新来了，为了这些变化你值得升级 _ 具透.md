昨天，微软正式推送了 Windows 10 发布以来的第四个重大版本更新——秋季创意者更新（Version 1709）。在这里，少数派为大家整理了新版本中比较重要的新特性及功能改进，如果你还不清楚秋季创意者更新有哪些变化，不妨通过这篇文章快速梳理一下。

全新的系统设计语言
---------

在今年 5 月份的 Microsoft Build 2017 上，微软发布了被称为 Fluent Design System 的新设计语言，其由光感、深度、动效、材质、缩放 5 种元素构成，结合了 Metro 的简约高效及 Aero 的高斯模糊，在平面上增加了细腻的光影效果。此后，一些系统应用已经随小版本更新率先采用了这种设计语言，但直到秋季创意者更新才被广泛用于系统的各种界面。

![](.evernote-content/FA8532AF-4F3A-43AA-9D8D-BD71822A194D/0D3F1857-4616-44DE-A45D-842CE24B0628.jpg)

升级至秋季创意者更新后，你会留意到包括开始菜单、通知、操作中心等在内的各种系统界面都有一种有点类似 Windows 7 Aero 的高斯模糊效果，这就是 Fluent Design 的 Acrylic 效果。当你打开一个系统应用，你会发现，随着鼠标移动，各个按钮的光影效果都会发生改变，按下按钮时，光影波纹随之泛起，显得极为精巧而灵动。

![](.evernote-content/FA8532AF-4F3A-43AA-9D8D-BD71822A194D/F722D96E-AFEC-48F2-AB39-9C734F0AF90C.png)

注意 M+ 周围按钮的光影效果

更强大的 Edge 浏览器
-------------

在每次 Windows 10 大版本更新中，微软都会为 Edge 浏览器增加许多新特性。在这次更新中，Edge 终于有了 F11 全屏模式，收藏夹系统也得到了全面改进。你现在可以编辑收藏网站的链接，在添加网站时的目录树也更加直观，便于选择。

![](.evernote-content/FA8532AF-4F3A-43AA-9D8D-BD71822A194D/3BC844E3-4224-4D74-B2DA-A4113F67D5BD.png)

同时，你还可以将网站固定在任务栏上，方便随时访问。

![](.evernote-content/FA8532AF-4F3A-43AA-9D8D-BD71822A194D/6DCA6F1D-0882-4162-9C4C-3CE2FC141A11.png)

![](.evernote-content/FA8532AF-4F3A-43AA-9D8D-BD71822A194D/1D737175-E183-474E-BD0E-9D17B858DB49.png)

你现在还可以直接用 Edge 批注 PDF 和 eBook 电子书，使用笔绘、高亮、注解、填表、布局、旋转等功能，体验不亚于第三方阅读器，又可以少装一个应用了。

![](.evernote-content/FA8532AF-4F3A-43AA-9D8D-BD71822A194D/2B7608F8-614B-437B-9D25-27695546EEEF.png)

全新照片应用
------

在此次更新中，微软为照片应用增加了诸多新功能。你可以为照片和视频添加滤镜、文字、笔墨及 3D 特效，再也不用每次修图都打开 Photoshop 了。在之后的更新中，还会推出重磅新功能 Story Remix，让你轻松用照片制作视频，保留精彩瞬间。

![](.evernote-content/FA8532AF-4F3A-43AA-9D8D-BD71822A194D/8330D512-01DD-4BA8-A1A0-48EB3930DDDD.png)

更智能的小娜
------

在秋季创意者更新后，小娜可以依据情境智能给出操作建议。例如，当你在 Edge 中标注了一个网页后，小娜会询问你是否要将它添加至备忘录；如果你刚拍了一张照片，小娜可以根据它的拍摄时间、位置等信息智能创建提醒，目前该功能仅在美国地区提供，随后会在全球可用。

此外，在中国区，小娜目前已与网易云音乐 UWP 版集成，可以直接调用其播放音乐、听歌识曲等。你现在还可以在锁屏界面通过语音命令直接呼出小娜，还可以让她关机、重启、休眠等。

![](.evernote-content/FA8532AF-4F3A-43AA-9D8D-BD71822A194D/211ED7B9-2494-4AD6-BABD-79F0B3A67326.png)

玩游戏还能反作弊
--------

在本次更新中，微软重新设计了游戏录制工具栏并增加了更多功能选项，让你可以在游戏中进行录制、直播、开启游戏模式和其它更多选项。此外，微软推出了一个近乎实时的游戏直播互动平台 Mixer，让主播和观众更方便的互动交流。

![](.evernote-content/FA8532AF-4F3A-43AA-9D8D-BD71822A194D/34A5C964-7A12-4E5E-8F5F-DD5FC739CAAD.png)

另一个重要改进是新增的 TruePlay 模式，当该选项开启时，所有游戏会运行在特殊隔离的沙箱环境内，进程 ID 被隐藏，使得一般外挂无法发挥功效，让游戏环境更纯净。

![](.evernote-content/FA8532AF-4F3A-43AA-9D8D-BD71822A194D/37AE138B-21A5-4DDC-801B-A3AFAFDC5740.png)

此外，微软还优化了 Windows 10 游戏模式的性能表现，任务管理器也支持显示 GPU 使用情况，让你尽享游戏乐趣。

支持混合现实
------

混合现实是秋季创意者更新的一个核心变化。现在，你可以利用 Windows 10 内置的 Mixed Reality Viewer 和画图 3D 查看、编辑和创建 3D 立体物体模型。你还可以利用设备摄像头将物体放置在现实世界，有点类似 iOS 11 的 ARKit。

![](.evernote-content/FA8532AF-4F3A-43AA-9D8D-BD71822A194D/2A069FB0-6C28-4CC3-BA92-56FE227D3BC6.png)

同时，你还可以将 3D 物体插入到 PowerPoint 和 Word 等其它应用程序，让你做出更酷的演示文档。

作为微软未来布局的重要一环，惠普、戴尔、三星等合作厂商的 Windows MR 混合现实设备也已同步上市，可以给你带来沉浸式的操作体验。目前，Windows 10 的多数系统应用已支持在三维世界中浏览，其它应用则会以 2D 模式运行。应用商店也及时上新了混合现实专题，随着 Windows MR 的普及，相信有更多应用会支持这一特性。

![](.evernote-content/FA8532AF-4F3A-43AA-9D8D-BD71822A194D/70851144-CEBF-4975-945E-85F26ECAD52C.png)

与手机一起协作
-------

现在，你可以利用 Cortana 和 Microsoft Edge 等微软应用将你的 iOS 或 Android 手机与 Windows 连接，享受更统一的操作体验。例如，你可以在电脑上继续浏览之前在 iPhone 上浏览的网页，或者接收 Android 手机上错过的通知。（WP：微软爸爸帮我盖上棺材盖可以吗）

![](.evernote-content/FA8532AF-4F3A-43AA-9D8D-BD71822A194D/9D6E5F47-94CF-4DFB-8E9E-1CE17E993138.png)

此外，原微软 Arrow Launcher Android 启动器将更名为 Microsoft Launcher，给你更无缝的体验。

更好用的手写笔和触控键盘
------------

你现在可以用手写笔控制 PowerPoint 幻灯片播放，做演示时更加高效。此外，设置中「查找我的设备」选项里新增了查找手写笔功能，可以显示最后使用手写笔的时间，帮助你进行回忆。

![](.evernote-content/FA8532AF-4F3A-43AA-9D8D-BD71822A194D/0F664D15-1F84-4D69-8EBD-4E2F097628BE.png)

触摸键盘在秋季创意者更新中也得到改善，可以使用滑行输入。你还可以通过「Win + .」快捷键召唤新的 emoji 面板，聊天更顺畅。

![](.evernote-content/FA8532AF-4F3A-43AA-9D8D-BD71822A194D/6E7C014C-E431-4B0B-A3C1-BF0CEF13E8AB.png)

其它细节改进
------

### One Drive 占位符回归

One Drive 占位符是 Windows 8 上就有的特性，允许你查看未同步至本地的在线文件夹，但在 Windows 10 中被舍弃，引来广泛吐槽。如今，微软终于将这个功能重新带回系统，让你在本地查看 One Drive 中的所有文件，并在需要时将其下载。

### Windows Defender

Windows Defender 新增了勒索软件防护功能，可以帮助防御 WannaCry、Petya 等著名勒索病毒。

### 人脉

你现在可以将常用联系人固定在任务栏上，查看往来信息、邮件等，还可以通过拖动快速分享文件。

### 汇率转换

你现在可以在计算器应用中直接换算货币了。

### Emoji

秋季创意者更新添加了一大波 emoji，再也不怕看不到 iOS 朋友发来的新 emoji 了。

### 辅助功能

秋季创意者更新增加了眼球控制功能，帮助活动不便的残疾人士通过眼球转动控制鼠标、键盘灯，更高效地使用 Windows 10 进行操作。如果你懒得手都不想动，也可以试试它。

此外，Narrator 屏幕阅读器也得到了改进，可以阅读图片描述，更方便了。

以上就是少数派为大家带来的 Windows 10 秋季创意者更新特性速览，如果你还发现了其它变化，欢迎在评论区讨论。
