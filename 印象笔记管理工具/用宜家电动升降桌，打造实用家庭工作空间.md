# 用宜家电动升降桌，打造实用家庭工作空间

---

买了 3D 打印机之后，家里 1.2 米的桌子几乎没有可用空间，只能把要用的东西随机地堆放在椅子四周，把书房搞的特别乱。

该是时候换个大一点的桌子了。

![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/5071118E-61E0-4610-A260-D9EEF0D4B8A1.jpg)

对于下一张书桌，除了有更大的桌面空间，我需要它有足够的稳定性。

3D 打印机比较吃这个。

晚上到宜家的时候，发现没有太多选择。稳定性这一点就淘汰掉大部分宜家书桌了。最终打算买 杰顿 / 芬沃尔德 这款。

![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/6C90FAC4-380D-4997-87C4-1BEB7B555E45.jpg)

两个支撑架设计的足够稳固，支架本身有部分储物功能，还可以手动调节桌面的高度、倾斜度。

我书房最大能放得下 175 的宽度，155×75 厘米的尺寸刚好。桌面材质可以自由搭配，对榉木这个桌面颜色和质感都比较满意。

但是去打折区逛的时候，发现有一把 [贝肯特](http://www.ikea.com/cn/zh/catalog/products/S99187942/http://) 的样桌，160×80 厘米，电动升降。稳定性跟上面那张不相上下。样桌半价，只需要 1999。黑色的风格跟书房基调还比较搭，于是最后买的贝肯特。

外观、材质
-----

因为样桌已经是装好的，所有没有安装环节。拆下来桌面和架子之间的几个快卸扣，运到家里，再装回去就行了。桌面材质就是宜家常见的黑漆，仿了一点木纹效果，手感还可以。

![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/55E959AE-0BA5-4A45-A3C2-98B62BCE2780.jpg)

桌子的支架比较粗壮，焊接的挺扎实。我一开始非常担心它这个结构设计的稳定性，对比了一下杰顿那款，稳定性一样强。

![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/A4546DC8-0478-4B97-BD21-99F28E435FE2.jpg) ![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/4204A758-FF73-4ADA-B7BB-D10C669C89E5.jpg)

高度、升降调节
-------

国内的这款桌子只有升、降两个按钮，没有国外版本那个高度记忆功能，比较遗憾。

![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/4EDB0393-3C14-490B-A969-CFED85C7B427.jpg)

电源做的比较大，桌子下面的网兜收纳了电源，就没什么可利用空间了。顶多放个小插线板，绕个电源线什么的。

![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/5126326E-DE9C-4094-8E77-A53ADB8D9AB0.jpg)

高度最高可以调到 124cm，最低 63cm。我身高 178，站立和坐姿都可以满足。

![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/4676F49B-AEED-4675-A357-7B10CA55BD01.jpg)

收纳
--

升降桌对收纳的挑战比较大。

首先是电源线不能紧贴桌腿收起来，会有一大段电源线暴露出来。如果能像吸尘器那样设计一个可升缩的电源线就好了。

![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/25FB1F3D-3111-4426-A1AB-BEB786B093F0.jpg)

桌面上因为有宜家的收纳盒和 Belkin Dock，线材还比较好处理。正面线材看着不明显。

![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/3C57DE90-EEEC-46BD-A6BC-972DE3FD73E5.jpg) ![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/11FBB397-4F39-4B4C-AA7F-35FA1B88533A.jpg)

然后因为升降功能。要求桌平面上下的垂直空间内都不能有障碍物。比较影响装饰品的悬挂。把最喜欢的章鱼人 amiibo 扯下来换个平面的画了。

![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/AF259EF1-9D3E-4EC1-9088-001248D0683F.jpg)

这样我的桌面可以更新一个大版本了：Neil's Desk v2.0。

![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/4F286A70-AE58-4CCC-AF9A-80A1122FF08D.jpg)

可以从容的放下 iMac + 3D 打印机。

![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/773AD925-A9F6-4FC7-A956-0B69AE6F5008.jpg)

音响可以不用扔桌子底下吃灰了。

![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/09434A09-D19E-42D4-9CCF-AFBA05FA12D2.jpg)

坏处是，桌面宽了，就只够放一把小椅子。

![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/A260E298-AE17-44BA-B3D2-6846C395FBBF.jpg)

我的桌面演变：[Neil's Desk Release Note](http://www.neilonly.com/2016/12/03/neils-desk-release-note/)

下次需要升级的内容
---------

有了升降桌，手臂可以调节到一个非常好的舒适度。但是不同姿势，特别是站立和坐姿，视平线和手臂的距离变化是很大的。就需要显示器的高度也可调。Herman Miller Flo 那个显示器支架看着还不错。

![](.evernote-content/6C17090B-DFCD-487C-A646-21E0C0B77A6F/9BC6F795-7D14-407C-8D17-686A6CAD39E1.jpg)
