# 这款小程序，让你在群聊时不跑题：群里有事

---

经常使用微信群聊的用户肯定会遇到很多烦恼，比如说：

* 微信群聊只有发起群聊的人才能发布**@所有人**的通知，但现实中真正需要发布群通知的往往并不是发起群聊的人，当然，需要发起群通知的往往不是一个人；
* 群聊发布群通知后，被多少人有效阅读不得而知；
* 群聊讨论意见收集时，需要滑动翻阅各成员的留言才能完成收集，并不能直观显示各成员的意见和最终结果；
* 讨论事情时，成员你一句我一句讨论很容易让信息流被切割成很多则零散的信息，导致成员根本无法聚焦到什么内容才是重点；
* 发布群活动时，活动主题、内容、时间、地点通常是以一段文字发送，对信息的获取需要仔细阅读，此外确认参加活动的人数依然需要等待成员的回复后进行统计。

除了朋友圈子里的群聊外，更让人烦恼的是越来越多企业的管理者将日常的管理放在微信上进行，尤其是谈论公事和分配任务时插播一些闲聊时，在混乱的信息流中聚焦重点内容总会让我有一种大海捞针的感觉。

不过现在得益于小程序「群里有事」，微信群聊的这些欠缺终于得到了适度的弥补。

使用介绍
----

打开微信，搜索小程序「群里有事」，点击后便会出现关于它的 4 个主要功能的介绍，分别有：**群通知**、**群活动**、**群投票**以及**求祝福**。

![](.evernote-content/243947D5-7D2E-467A-B923-029A865C0BE6/9E331549-9A61-417D-A4B7-B0803709A961.png)

### 群通知

选择群通知后，出现在你眼前的是排版较为合理的通知样式，除了常规的标题、内容、图片添加外，还包括更多选项功能中的署名、文号、模板选择。

![](.evernote-content/243947D5-7D2E-467A-B923-029A865C0BE6/3D27F41C-112B-49E1-87D3-776D53922320.png)

将编辑完成的通知发布于群聊后，群聊会提示收到来自小程序的信息通知，当成员打开后，「群里有事」便会自动将成员浏览记录下来，时间精确到几时几分。这样一来，每一位成员不仅都可以发起群通知，还可以清楚看到发起的通知被多少成员和哪些成员有效阅读。

![](.evernote-content/243947D5-7D2E-467A-B923-029A865C0BE6/DB580303-1228-4064-A2B2-1B9CB37BA07D.png)

### 群活动

活动其实本身就是一种通知，但相比通知而言，活动需要收集的信息更多。所以在「群里有事」中「群活动」在「群通知」的基础上，丰富了更多选项的功能，如人数限制、活动地址以及报名信息。

编辑完成后的活动通知被成员打开后可以清楚地知道活动的详细内容、地址。值得一提的是，点开地址时，会进入到微信自带的地图功能，不仅可以清楚知道路线，还可以将该地点用手机安装好的地图应用打开。

![](.evernote-content/243947D5-7D2E-467A-B923-029A865C0BE6/C6CE59C6-8EAF-487B-9297-19E77EB339AC.png)

在成员浏览活动后感兴趣时，便可以点击「我要报名」即可完成。如果群活动发起人编辑前选择「需要报名人信息」时，报名的成则需要填写姓名和手机号码。

报名人信息提交后，活动详情界面下方会显示已经报名成员的名字和手机号码。除了方便群活动发起人的核实参加活动人员外，列表还能充当应急的通讯录列表，点击列表中成员头像时就会自动弹出是否拨打成员手机号码的提示框。

![](.evernote-content/243947D5-7D2E-467A-B923-029A865C0BE6/D4B36B5A-9CAC-4E8A-92BD-B09CDBBA8DE9.png)

### 群投票

除了常规的标题、补充内容以及投票选项外，「群里有事」还支持投票是否多选、投票是否匿名以及发起人和截止日期的更多选项功能。

![](.evernote-content/243947D5-7D2E-467A-B923-029A865C0BE6/F34D7F4B-C574-4CE5-85D0-A8B4B0CD056D.jpg)

投票通知被成员打开后，点击投票选项即可完成投票，当选择不支持匿名的时候，投票成员的微信头像将会显示到选项下方，点击头像即可显示该头像的微信昵称。

![](.evernote-content/243947D5-7D2E-467A-B923-029A865C0BE6/6C733484-FE0D-49F9-924E-13213A0F9582.jpg)

此外，「群里有事」为了防止成员相互干扰，未投票时将无法看到投票结果，只能看到参与人数。在投票之后，才能看到当前的投票选项各占比，而最终结果需要在投票时间截止后才能看到。

### 求祝福

当你需要收集群内祝福时，点击后在界面中除了编辑文本外，你还可以选择自带的模板，让通知详情界面变得更加美观。若该通知被成员打开，不仅可以清楚知道内容，还可以留言献上祝福。

![](.evernote-content/243947D5-7D2E-467A-B923-029A865C0BE6/6605F914-3F96-4873-9C2F-7812E9A94006.png)

补充
--

除了以上四个主要功能外，「群里有事」还有着许多小细节值得令人称赞。比如：

* 内容编辑支持 Markdown 排版；
* 自动保存草稿，在编辑时常常遇到微信消息通知，界面跳转后重新打开群里有事时会保留之前的输入内容；
* 所有通知发出后，微信**服务通知**会自动发消息通知目标完成情况提醒；
* 在群里有事列表界面中，左滑即可删除该通知。

在体验中，我觉得仍有着一个很大的不足：显示成员 ID 时并不是成员在群聊中的昵称，而是微信本身的 ID，当群聊成员人数过多时，依靠可变的头像和微信 ID 辨别谁是谁的方法确实比较困难。

想要体验的朋友请直接在微信搜索「群里有事」即可。
