# El Capitan新特性之——提醒事项共享菜单

---

[![](.evernote-content/DCAAD40D-A07E-415B-B75B-8C67ABD42A27/9030A9E5-FC8C-44EF-B545-99B96505CD32.jpg)](http://www.feng.com/apple/tutorial/2015-10-23/El-Capitan-new-features---remind-items-share-menu_627979.shtml)

　在 iOS 上有一个名为“提醒事项”的应用，它可以帮助我们记录生活中大大小小的事情，在设置完项目之后，我们就可以为它添加基于时间或者是地点的提醒。

[![](.evernote-content/DCAAD40D-A07E-415B-B75B-8C67ABD42A27/9030A9E5-FC8C-44EF-B545-99B96505CD32.jpg)](http://resource.feng.com/resource/h060/h60/img201510232115310.png)

其实在 OS X 的 Launchpad 中，也存在“提醒事项”这个应用，不过应用频率相比 iOS 就要低不少了。在 OS X El Capitan 中，这款应用本身尽管并没有得到更新，不过苹果还是对它的跨应用属性进行了扩展。

[![](.evernote-content/DCAAD40D-A07E-415B-B75B-8C67ABD42A27/74C6E985-0C4C-4F72-B0AC-F9280C9F95A5.jpg)](http://resource.feng.com/resource/h060/h60/img201510232115570.png)

现在在 El Capitan 中，借助共享菜单，即可在 OS X 的几乎任何位置创建提醒事项，包括 Safari 网页、联系人或电子邮件文本。利用触控板，向左轻扫即可删除提醒事项列表。你可以直接在通知中将提醒事项标记为 "已完成" 。如果项目还未完成，你可以选择何时再次提醒。

这样 OS X 就能够在全局为用户提供更多提醒，尽量避免用户错过重要事项。
