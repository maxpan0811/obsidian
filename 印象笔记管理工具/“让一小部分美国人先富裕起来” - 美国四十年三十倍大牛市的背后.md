“让一小部分美国人先富裕起来” - 美国四十年三十倍大牛市的背后
--------------------------------

原创西西弗评论J西西弗评论

新中国的发展以1979年分界，一般分为前三十年和后四十年，1979年前和1979年后的经济和政治都有很大区别。但很多人没有意识到，美国的二战后，其实也可以分成前三十年和后四十年。1945-1975年是前三十年，1975-1980年过渡期。1980年以里根上台为代表的后四十年。美国的前三十年和后四十年，经济政策上也是截然不同的。

前三十年，是共同富裕的三十年，是股市萎靡的三十年；后四十年，是股市狂飙的四十年，是工资原地踏步的四十年，让一小部分人先富起来的四十年。

 

**战后七十年的美国经济：前三十年 - 工资翻番，共同富裕**

 

在这次中期选举中当选的美国政界新星，众议员 Alexandria Ocasio-Cortez,（简称AOC）提出向年收入超过1000万美元的人，征收最高边际税率70%的所得税。这个新闻，在中国的互联网舆论场被各种嘲讽。嘲笑AOC天真，疯狂，愚蠢。这些人其实并不了解美国的历史。他们对美国的理解都是后四十年的美国。在前三十年，在苏联的竞争压力之下，那时有另外一个截然不同的美国。

 

我们先看看一张图，这张图是美国1913到2014年的最高边际税率。（就是收入大于某一个数值后部分需要交的税率，中国目前是年收入96万元以上部分 45%，美国是目前单身个人年收入50万美元以上部分缴纳37%的所得税）

 

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/5EEB1CAA-F80A-4D3E-A50C-9A34CE31B6E9.jpg)

这张图的内容很讽刺，现在国人嘲笑70%的最高个人所得税率不可思议，但是，美国从1936年直到1980年，整整45个年头，美国的最高个人所得税率都超过70%。加上一战期间的四年，二十世纪100年中，美国有49年最高个人所得税率超过70%，还有6个年头超过60%。其中，44，45年，51-63年，有整整15年，最高个人所得税超过90%。以1952年为例，一个单身，年收入20万美元以上（相当于现在的300万美元）的部分，需要交92%的个人所得税...... 这个和我们想像中的自由资本主义的美国差别很大吧。

 

再看看遗产税（Estate Tax）从1940-1976年，整整37个年度，遗产税最高税率达到77%。这个税率，基本上真的是富不过三代了。即使不算资产变现时的损失，传承两代后，一个超级富豪也就剩下5%的遗产能传给第三代。

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/EEEBE4F0-9746-47B0-B453-F61320306713.jpg)

 

其他国家也差不多。比如英国，在1950-1974年间，遗产税最高税率高达80%。日本从1950-2002年的最高遗产税税率都超过60%，最高达到90%。

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/440B3402-6156-44A5-A98F-8936545C0CC0.jpg)

 

不是只有我们有改革，资本主义国家的改革力度，远远超过我们的想象。在面临苏联意识形态的竞争，和大萧条的阴影笼罩下，资产阶级可以大幅度向无产阶级让利的。苏联和美国，北约和华约。国与国的矛盾成为了主要矛盾，国内的矛盾变成了次要矛盾。各个西方国家都通过税收和转移支付，大幅度降低贫富差距，安抚无产阶级。二十世纪五六十年代，美国国家朝气蓬勃，各个阶层都收获到了经济增长的果实，技术进步，人民幸福。然而，情况随后发生了变化。美国战后的前三十年共同富裕的黄金时代，结束了。

 

**战后七十年的美国经济：后四十年 - 让一小部分人先富起来**

 

我们看一张新的图：

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/7AB9B330-2451-452F-82AE-12DE800FA943.jpg)

从1945年到1970年，五根线，基本上同步增长，然后，一切都开始变化了。上个世纪70年代后期，美国刚刚从越战的梦魇中走出，又踏入了滞胀的泥潭。滞胀是什么？经济不增长了，而通货膨胀还非常高。直到1980年里根上台。里根上台的经济学就是减税，将最高所得税率从80年的70%降到82年的50%，随后88年里根下台时，只有28%。

 

里根之所以敢如此下狠手改革，有几个原因，第一当时社会所有阶层都急于摆脱滞胀的经济困境。从上图可以看到，美国的真实工资水平从1975-1982年是下降的。其次，是美国无产阶级经过战后的一轮高速增长，普遍生活水准有了很大提高，当时也没有人知道，里根经济学对普通劳动者意味着什么。第三，里根找到了一个秘诀，就是寅吃卯粮。靠大幅增加国债，可以在减税的同时保证福利的下降幅度较小。正好亚洲国家，先是日本，然后是中国还是大量买入美国国债。

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/DB73B73B-60A3-48B6-9AE3-8EDD42F11E71.jpg)

 

里根经济学，深刻的改变了美国的经济形态。A股的老股民们经常都羡慕美国的股市多么欣欣向荣，A股多么不争气。然后，美国股市并不是一直欣欣向荣的。让我们看下面两张图。

 

Inflation adjusted D-Jones 100 years （通胀调整后）

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/D0D15956-6492-4566-A4AF-46255610CF7F.jpg)

S&P 500 （通胀调整后）

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/BDD934ED-8620-456D-86EC-C7C217D51E1A.jpg)

 

如果你是一个生活在1950-1980年间的美国股民，你丝毫不会觉得美国股市多好。经过通胀调整后，1982年7月，道指的指数还不如1950年1月高。**1950-1982年整整32年，美国的道琼斯指数没有跑赢通货膨胀。**标普稍微好一点，但是1982年，也不过是1954-1955年的水平。1991年的经通胀调整后的道指和62年前，1929年大萧条前的最高点是一样的。

 

**美国的大牛市只存在于后四十年**，是从里根开始蓄力，苏联解体后开始腾飞的。从1982年低点到现在，即使经过通胀调整，股指也翻了10倍，从1991年9月苏联解体开始，翻了4倍多。如果不做通胀调整，从82年低点到现在，道指差不多翻了三十多倍......

 

看上去很振奋吧，里根真是伟大的总统，股民的救星。暂时等一下。还记得我们之前的那张图吗？1950-1970年，人均GDP，每个全职工作者的人均GDP，人均工资，中位数工资，基本上同步增长。但是，1975年以后，五条曲线就分道扬镳了。人均GDP继续高歌猛进，中位数工资缺从未抬起了头。让我们再看看两张图。

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/D01E7B17-8F7F-4FFD-B1F6-A79F89938185.jpg)

 

1964年到2018年，美国人平均小时工资，从2.5美元涨到了22.65美元，好像涨了8倍，但是，通胀调整后，几乎完全无增长。2018年的通胀调整后的工资，还不如1970年的水平。股指呢，1964到2018，通胀调整前标普股指涨了接近40倍，调整后，也涨了接近5倍。如果从里根上台的1980年看，就更明显了，通胀调整后的工资勉强涨了10%，通胀调整后的股票涨了接近8倍。

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/8E5B216D-0F7C-48B3-BBD2-AF4697C8A6D5.jpg)

 

 

这张图也很明显，美国制造业工人从1975到2007年的的生产率翻了一番，工资却毫无变化。美国的制造业工人，效率提高了一倍，但是工资丝毫未涨。

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/79CEC563-B38E-4112-9A9E-8A8AEF3BC05C.jpg)

 

如果说所有人都没变化也不对。上面这张图看的更清楚。1970年前，10%的高收入美国人和90%的其他美国人的收入水平在同步增长。1970-1980年里根上台前，经济整体不好，收入最高的10%和剩余90%都是10年没有实际收入的变化。但是里根上台后，90%的人还是收入没变化，但收入最高那10%的的收入开始起飞，翻了一番以上。

 

**里根上台到现在40年了，40年美国的经济增长，是上市公司市值（股市）的增长，****8倍****; 是收入最高的10%阶层的收入增长，****2倍****;  剩下90%的劳动者，他们失去了至少40年。从1967年到现在，已经50年了，50年，90%的美国劳动者，真实收入的增长率是 零！这就是美国过去40年的经济增长，少数人的盛宴！**

 

**战后美国经济发展 1980年为界。1980前，股市几乎没涨多少，实际工资翻倍。1980后，通胀调整后股指涨8倍，调整前30倍，但实际工资原地踏步40年。**

 

比工资原地踏步更糟糕的是贫富差距的扩大。

 

福布斯的富豪榜大家都耳熟能详了，比尔盖茨、巴菲特、贝索斯也成为了中国家喻户晓的人物。福布斯第一次发布美国富豪榜是在1982年。让我们看看下面两张图

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/392C1368-B027-42F5-BB62-5AB2110979CA.jpg)

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/279759FD-90BC-4F19-9B2E-073BFC226073.jpg)

  

福布斯在1982年的美国首富是一个叫Daniel Keith Ludwig的人。他有多少钱呢？当时大概不到20亿美元折算成今天的美元大概50亿而已。今天的首富的财富是1600亿美元。首富的财富，从1982年到2018年，翻了32倍。

 

再看下一张图

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/D3ED958E-CD0E-4799-83DC-9A2A5A961F47.jpg)

 

*（注 gini系数不同来源计算有区别，world bank数据是从1980年34.5涨到 2016年41.6，上图从1980年39到大约47。但不改变结论，美国基尼系数已经到了大萧条之前1928年的水平）*

 

这一个衡量贫富差距的指标，叫基尼系数。具体计算方法不细讲了。简单一个事实，美国的贫富差距，从罗斯福新政时代开始下降，到1970年左右达到最低。里根上台后，开始直线攀升，现在已经达到了大萧条前二十世纪二十年代那个野蛮资本主义时代的贫富差距。世界其他国家，即使G7中除美国外收入最不平等的英国，也没有出现如此剧烈的贫富差距扩大。

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/5A1C24DC-4711-4453-A23A-27EBC98872AE.jpg)

和里根时代一样，英国的基尼系数在撒切尔时代，也出现了剧烈上升，从0.29上升到0.37。但在撒切尔时代之后，又回到了稳定甚至稳中有降的模式。

 

让我们看下一张图

 

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/CFF22798-F59B-4188-93EC-3A9C946F4383.jpg)

 

这一张图可能很多人有印象。从1996年到2016年20年间，美国的整体物价上涨了55%，但是大学学费上涨了200%，医疗费用上涨了100%，照顾孩子的费用上涨了122%。但这个趋势，不是1996年开始的呀。互联网上并不能找到1982年开始的图，可能美国人真正发现这个问题，是在1996年才开始吧。如果从1982年开始算，数据应该是什么样的呢？互联网上没有图，好在原始数据都有，自己做一张吧：

 

![](.evernote-content/402C2B2C-CB89-4AFE-84F2-46720235D61F/6FCC3E6D-E1B5-4C56-BFF0-173AEC025F1D.jpg)

1982-2019年通货膨胀涨了 2.64倍，医院和相关服务涨了9.7倍，大学学费涨了9.5倍。美国的医疗与教育也是民众最忧虑的问题。

 

我们用一个表总结美国的1980年以后的 后四十年吧，时间关系，这里面数字是个大致的估计，但应该没有太大误差

 

|  |  |  |
| --- | --- | --- |
| 1982-2019年涨幅 | 绝对倍数 | 扣除通胀后倍数 |
| 工资 | 不到3倍 | 几乎无增长，实际工资低于1970年水平 |
| 股市（道琼斯指数） | 超过30倍 | 超过10倍 |
| 首富身价 | 超过80倍 | 超过30倍 |
| 通胀 | 2.65倍 | 1倍 |
| 大学学费 | 9.7倍 | 3.7倍 |
| 医院和相关服务费用 | 9.5倍 | 3.6倍 |
| 20城市房价 | 4.2倍 | 1.6倍 |
| 收入最高的10%群体的工资 | 5-6倍 | 2倍 |
| 上市公司利润（S&P 500） | 接近10倍 | 接近4倍 |

从1930年罗斯福新政开始到1975年，资本主义用了四五十年时间自我改良，缩小贫富差距。经过1980年开始四十年的“让一小部分人先富起来的政策”美国的经济形态和贫富差距，基本回到了大萧条之前的状态。

----------------------------------

唯物辩证法认为外因是变化的条件，内因是变化的根据，外因通过内因而起作用。同时，认为经济基础决定上层建筑。美国后四十年经济环境的变化，是美国当前政治生态变化的源头和根据。经济环境的变化是一个结果，是一个事实。但这个变化的原因是什么？不同的政治派别，就有了不同的解释。经济结构出了问题，远见的政治家设法解决问题，短视的政治家去找一个替罪羊推卸责任。美国的选举政治决定了，当选的政治家都要为选举连任服务。

Trump 的当选，是穷人意志的一个愤怒的表达，最支持Trump的群体是没有大学学位的白人。Trump 把经济的变化归因于全球化，归因于中国，设法把美国民众六十年不涨工资的愤怒，归因于中国的竞争。现在为什么美国形成了反华的两党全国一致的反华统一战线？Trump 需要找一个替罪羊宣泄选民的愤怒；莱特希泽这样的有理想的政治精英要遏制中国，维持美国第一的霸权地位；商界精英因为在华利润下降，希望Trump出面谈出更有利的贸易条件。现在这个阶段，美国确实对中国同仇敌忾。但是，打上两三年之后呢？选民会发现，中国并不是他们不涨工资的原因；商界也会发现他们的在华利润会灰飞烟灭，到那时，恐怕美国也很难维持这个统一战线了。美国的政治演化，下一篇文章再探讨吧
