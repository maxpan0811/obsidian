新一年的「怼肉」计划，这些频道你可以参考 - 少数派
==========================

前两天在探讨年度新知的话题下，好多朋友都提到了新的一年一定要重视健身、一定要多运动、身体健康才是最重要的。当然这也是我一直鞭策自己的动力，好的身体真的是各方面的基础。

如果你一时懒得走出家门去运动，其实在家对着电视 / 电脑也一样可以动起来。不管是跟着视频进行运动的教程，还是对力量训练的理论和发力感觉补充。希望大家在新的一年里，都可以在运动方面，完成自己制定的新年计划，将脂肪「怼掉」。

‍♀️打开视频跟着做就 vans 了
------------------

工作后其实没那么多时间天天去健身房，所以我会给自己准备好备用措施，买好瑜伽垫、哑铃、弹力带，下了班回来收拾一下，铺开瑜伽垫、点开 iPad，就能开始一次的锻炼。

虽然可能在家自己练不比健身房的氛围强，但胜在节省时间、简单快捷。跟着运动视频的教程下来，多多少少也会出点汗，活跃了脂肪、震动了肌肉。而对于家庭化自练，就像抄作业一样，有学霸的可抄最好，既省心又保证正确。

在我选择并体验了一些频道后，在健身方面感觉还是国外 up 主的产出比较优质，而且这种「跟着做」的教程频道中 up 主其实不会唠叨太多动作要领，语言不影响动作执行。所以在接下来的内容中，我会放进国外一些优秀 up 主的视频内容，顺带也提供 b 站搬运频道，大家各取所需。

### 周六野 Zoey：甜蜜「宝宝」狂魔

传送门：[主页](https://space.bilibili.com/62540916)（b 站）

活动类型：HIIT、局部自重 / 时长：10~40min / 所需器械：瑜伽垫 / 空间要求：单人身位

周六野小姐姐 b 站和油管都有频道，国内比较出名。她的视频教程非常适合**日常锻炼用**，因为强度适中，基本不需要额外的器械，所以只要你**有时间、有空间**，就能做。针对各部位的标注也很明显，频道里是教程与生活类视频共存的状态。

她的视频还是比较追随「热点」的，比如前段时间因为 Jennie 火起来的直角肩、大腿内侧的相关锻炼教程都逐一发布。而且她还是很宠粉，如果你有一些想看的部位教程，也可以在她视频下留言进行撒娇反馈，没准之后就会出现你想要的视频教程啦。

周六野的 b 站主页

### Fitness Blender：模范生健身夫妻

传送门：[主页](https://www.youtube.com/user/FitnessBlender)（油管）/b 站搬运： [燃脂合集](https://www.bilibili.com/video/av59016655?from=search&seid=12960757872363887807)（共 31 p，无翻译）、 [力量塑形合集](https://www.bilibili.com/video/av68467986?from=search&seid=12960757872363887807)（共 71 p，无翻译）

活动类型：HIIT、局部自重 / 时长：8~40min / 所需器械：瑜伽垫、哑铃、弹力带 / 空间要求：单人身位

频道主为一对夫妻。女生一般负责局部自重的教学部分，男生一般负责 HIIT、Cardio 的教学，所以该频道男女都可以跟着学哦。

他们的视频风格我还蛮喜欢的，背景就是白色，而且录制环境很专业，没有乱七八糟的背景、旁人的说话声等。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/5966B223-0664-43A6-B40D-C820D5621812.jpg)

每个视频开头都会在左上角摆出这节课的训练部位、需要额外准备的器材、是否包含热身 / 拉伸。视频右侧会有一个类似「一炷香」式的倒计时，显示的是整个教程的时间倒数，当然每个动作下也会有专门的计时。同时，每个动作开始前都会在左上角出现小图 gif 来演示该动作，有点像以前我去 Orange Theory 上课的动作指导，方便你充分预备跟上节奏。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/B5DA29BE-1BAB-4B40-8F8E-FCE0ADC6A8E6.jpg)

Blender 频道的教程比较家庭化，要求的空间不大，只要你自己能舒展开即可，适合对运动强度需求中等（想要运动出汗 / 有肌肉酸痛感）的宝宝们去 follow。

### Popsugar Fitness：黑泡欢乐谷

传送门： [主页](https://www.youtube.com/user/popsugartvfit/featured)（油管）/ [b 站搬运视频](https://space.bilibili.com/363702/channel/detail?cid=32150)（2017-2018 部分视频，无翻译）

活动类型：有氧、舞蹈、局部自重 / 时长：20~40min / 所需器械：瑜伽垫、哑铃、弹力带 / 空间需求：能蹦跶开

这个频道比较活泼，光看名字你也能看出来，不止活泼还有一点黑泡的律动，带一点点的南美感觉。所以你可能需要更宽阔一点的空间，毕竟你也不想自己跳得正嗨时，前边的衣柜挡住了你潜力迸发中的律动。

Popsugar 涉及有氧类、POP 风格融入 Tabata 系列的舞蹈、局部自重训练等等，强度适中，时间一般为 20 分钟以上（这个频道的视频基本没有低于 20 分钟的），所以如果你每天的运动时间在 20-30 分钟之间的，你可以考虑跟着他们的视频来一段。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/2F27AAA0-7FD2-46B6-BAA6-256190D33F70.jpg)

除了日常的有氧，Popsugar 的最大特色就是用 Hip-pop/zumba 等融入到高强度间歇训练中，而且动作也不太难。我有上过线下的 Zumba 课，老师们是跳得真好，而大家都跟得手忙脚乱，多少有些不甘心，Popsugar 就不会有这种情况。如果你协调性不好也不必担心，后边的助教分别代表两个程度可参考的人群，如果跟不上，就按照他们的程度做小幅度一点的动作就行。

我一般会在两三天的力量训练中间穿插做一次舞蹈类的训练，既能发散一下自己的柔韧性和协调性，又能愉快地出汗减脂。

所以如果你厌倦了正经的锻炼，如果你不开心了，请点开这个频道的视频跳跳舞，包你开心又畅快，不要害羞！男女都可！

来看看一位不愿透露姓名的朋友的即时反馈，有质有量还开开心心，鱼推荐的肯定行。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/026BC647-F3C8-4AC0-B28B-F5572A8E1829.jpg)

Pamela Reif：人美技术狠
-----------------

传送门：[主页](https://www.youtube.com/user/PamelaRf1/featured)（油管）/b 站搬运 [全身合集](https://www.bilibili.com/video/av78861980)、[核心合集](https://www.bilibili.com/video/av78864444)、[腰腹合集](https://www.bilibili.com/video/av78863986)、[臀腿合集](https://www.bilibili.com/video/av78863376)（无翻译）

活动类型：有氧、局部自重 / 时长：4~20m / 所需器械：瑜伽垫 / 空间需求：单人身位

网友们取名谐音「怕么啦」的小姐姐。频道的视频数量不多，因为小姐姐更新频率不太高，但内容也足够你用。更是好多网友心中的新手友好向的频道，甚至训练效果超越美丽芭蕾的内容。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/99681CB2-FA71-40C7-BA90-7A34A286455F.jpg)

看封面图就能一目了然教程时间和针对的训练部位，我很喜欢这种突出主题的封面。视频内容也跟前几个类似是局部自重训练，而且基本都是徒手。风格和 FB 一样也是健身房私教室背景 + 音乐 + 倒计时。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/DC766664-824C-4BAF-9CCD-25A963C4FBC8.jpg)

另外比较有意思的是，她有一个运动栏目是「song workout」，会根据一首歌的节奏，将那些自重动作融入进去。比如上图里 Dan+Shay 和比伯当时合作的《10000 hours》song workout，我原本以为就是单纯地把这首歌当背景音乐，后来跟着做了一次发现，她真的把节拍和动作融合在了一起，边运动边听歌不要太开心，并且一首歌也就 3-4 分钟的时间，很快就可以做完，完全可以当个小热身，简直太完美。

Yoga with Adriene：治愈系邻家姐姐
-------------------------

传送门：[主页](https://www.youtube.com/user/yogawithadriene)（油管）/b 站搬运：[30 天系列](https://space.bilibili.com/363702/channel/detail?cid=30250)（20~35 分钟不等，无翻译）

活动类型：瑜伽 / 时长：10~60 m / 所需器械：瑜伽垫 / 空间需求：单人身位

如果你不喜欢蹦蹦跳跳，但是钟爱瑜伽这种修身养性但仍旧起到强化核心的运动，非常推荐这个频道。每个视频的封面都会标注好这次课程的训练目的：舒缓酸痛、放松压力等等。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/BC6B77AB-B72E-436B-8C28-65CF773208D5.jpg)

小姐姐本人超级温柔，伴随教学的讲解也是柔声细语，比较适合静修氛围。而且她有一只超可爱的狗狗！看着很有安全感。好多视频中狗狗都会陪在小姐姐旁边，她在垫上做瑜伽，狗狗就在旁边自己玩耍睡觉，画面很治愈。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/76D1B726-C0A2-42D5-8E4F-3FCC41314872.jpg)

她的频道里播放列表规整得比较细致，有根据时间长短划分的，从 10 分钟到一小时，你可以根据自己的时间状态进行选择。也有针对身体部位进行舒缓放松的，还有专门针对初学者的合集，非常到位！如果你想冥想或者静下来思考，可以选择该频道的视频，既能静下心来又能锻炼身体。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/1BDEE42A-7906-4113-ADF8-ECA1077B521E.jpg)

当然健身类的视频，可能中文的讲解会比英文更让我们专注于动作，而不是去思考「哎，他刚才那句话说的是哪个注意事项？」，但是以上这些频道倒也没有太多需要额外注意的部分，不像下边我们要介绍到的力量部分，可能会额外需要去领悟发力感觉以及某些严肃的注意事项。

️硬核老哥让你力量训练更明朗
--------------

虽然当年我在健身房是出了名的好奇宝宝，有问题就问，但是自己私下对健身知识的补充也是必不可少的。最基本的就是：动作的发力点在哪？这个动作锻炼了哪个部分的肌肉？做哪个动作能最大程度地刺激这个部位？等等。

当然理论知识的补充与自己上手的实践都是密不可分的。所以在力量训练的过程中，我们可以根据以下腱子肉猛男们的知识点补充，然后运用到自己的锻炼中，将力量训练成果最高效化（英文视频有提供中文翻译搬运频道）。

### ATHLEAN-X：力量训练扛把子

传送门：[主页](https://www.youtube.com/user/JDCav24/featured)（油管）/b 站搬运：[兔兔姐 28](https://space.bilibili.com/12333557)（中文翻译，实时跟进翻译）

如果你关注健身力量训练的话，基本都会很熟悉我们的 Jeff Cavaliere「姐夫」！毕竟 892 万的关注量摆在那。Jeff 虽然现在是 40 多岁的大叔，但仍旧每天坚持力量训练，体脂率控制在 5% 左右，那个肌肉线条是正正好好的观赏 + 实用最佳呈现，我都羡慕哭了（当然女生不要搞这么低的体脂率啊，对身体不好）。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/86082DF0-FB23-4743-A840-95AB37903872.jpg)

正如网友们的反馈一样，跟着他的频道内容，可以让你真正找到动作的发力感、让你了解到以前那些动作到底哪里错了。作为美国国家体能协会认证的力量和调理专家，他的理论和动作的细节补充，听了绝对让你收获满满。姐夫除了会教你动作外，还会结合人体骨骼结构对动作进行纠正。看过他的视频以后，我已经换掉了以前「悬空飞鸟」的动作，转为有平面支撑的垫上飞鸟。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/65BA5D1B-A050-4364-A7AC-274C182170C3.jpg)

### Jeff Nippard：引经据典学霸哥

传送门：[主页](https://www.youtube.com/user/icecream4PRs/featured)（油管）/b 站搬运：[搬运频道](https://space.bilibili.com/370342489/channel/detail?cid=64337)（只有 11 个视频，但都有中字翻译）、[神仙合集](https://b23.tv/av38538489/p1)（繁体中文翻译）

另一个 Jeff（可能运动里叫 Jeff 的都挺厉害吧），小哥某些角度下有点像雷神，尤其是穿无袖健身衣时，偶尔还会与姐夫进行联动合拍教程，强强联手。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/7FF44BA1-DC4E-4FBD-9015-4BF7D09147B5.png)

他的视频相比姐夫而言，会有更多的来自论文、数据以及期刊的理论支撑，非常专业，仿佛是学霸引经据典进行科普一样。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/247901ED-6C37-47D9-ADBD-818C5DCD1BE2.png)

从封面图的描述上来看，他对肌肉的划分会清晰和细节一些，除了在动作、器械上手示范方面，他还会给大家展示该动作下无实物的发力感觉，去更多的感受动作细节。油管上新晋的「理论帝」，两个 Jeff 都非常推荐大家去看！入股不亏系列。

除了 b 站搬运频道外，我还找到了一个神仙合集，把理论 Jeff 的视频凑了一个整，看分 p 的名字就知道，这是一个[我要一顿爆看的合集](https://b23.tv/av38538489/p1)。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/F2F5C182-7D77-472F-A888-FE4EEB63385B.jpg)

以上推荐的这两个频道是对初级和中级选手都很友好，会学到很多知识。建议大家都可以尝试去看看，下边介绍几个 b 站的老哥。

碎片时间用起来

当然，除了这种大片段式的视频补充，我们也可以把日常刷社交媒体的时间利用起来，进行碎片化的补充，进一步丰富自己的动作库。

关于动作库，我之前有分享过我自己利用 Notion 建立的力量训练动作库，根据部位将动作进行收集，可以随时更新或筛选。在线下分享时，这个 Page 也得到了小伙伴们的肯定，所以在这里也分享出来，如果你有需要，可以直接 copy 这个模版（[鱼的运动库 not 裤](https://www.notion.so/magichouse/not-f71ef366623447219b52a2a837369784)）进行自己动作库的搭建。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/21FD9F5A-F7D4-4CA6-A95E-06A464BFDC0A.jpg)

我的碎片时间主要是日常刷 ins 和微博。我在 ins 上也有关注几个健身博主，比如：**dannibelle、jamieleepurnell（ins id）**。

她们平时都会更新几个片段的动态，就是简短版的教程。一个片段就是一个动作的练习，主要是针对臀部、腹部等局部小动作。如果是我没有练习过的动作，我就会截图记下来放进我的动作库，后期实践后进行筛选。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/8F0D0A84-C6CD-4787-8028-690B11C7DA85.jpg)

最后推荐给喜欢刷微博的宝宝们一个账号：**欧陽忍 Shorio**。

这个博主就是搬运外网上一些健身教程视频，基本都是力量 / 局部训练，全干货，随便扒一条微博就是教程。而且他会把此教程里涉及到的部位和动作重点用文字总结出来，方便大家记忆。偶尔还会穿插一些融入健身梗的图片，很有意思。

![](.evernote-content/A71C6371-6F59-466E-8500-2D082837454C/F7822622-0405-4FD0-947D-98D8DF6E57C2.png)
