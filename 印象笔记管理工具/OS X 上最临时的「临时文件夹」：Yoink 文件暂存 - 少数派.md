# OS X 上最临时的「临时文件夹」：Yoink 文件暂存 - 少数派

---

但凡同一种需求出现多个选择，就会出现大规模的撕（哔-），毕竟每个人都希望自己手里的东西能被承认是最好的，然而「技压群雄」始终只能是平行时空里的幻想，现实生活中更多的还是「各有千秋」甚至「平分秋色」。这是一件好事，因为有竞争才有进步，来，请跟我走一遍以下三个著名事件：

* Symbian 消失；
* iOS 和 Android 互相学习越来越好；
* Windows 10 增加了诸多 OS X 已有的特色功能。

不过要说第一生产力的代表，OS X 自然是略胜一筹，除了近乎苛刻的系统级优化，难能可贵的是它背后的一群优秀开发者，得益于他们的青睐，即使是普通用户，也能轻易的通过一些非常实用的软件来更高效的完成工作。而这些软件往往非常小巧简单，有的开机启动后直接融入到系统里，你甚至感觉不到它们的存在。比如 Yoink。

你不需要我的时候永远看不到我
--------------

在日常使用中，「把文件从一个窗口（应用）添加到另一个窗口（应用）」一直是我们最为频繁的操作和需求，以将浏览器里某张图片分享给聊天应用里的朋友为例，在以往的操作习惯中，我们常用到以下两种解决办法：

* 直接从浏览器里长按图片拖动至聊天应用中，但是大家都知道，这样经常会不小心误操作导致拖动失败然后重复 or 重复数次；
* 打开 Finder 并新建一个临时文件夹，从浏览器将图片拖至文件夹，再打开聊天窗口拖入图片，我都懒得说这样有多复杂。

那如果你恰好（是的，五分钟后你就变成「恰好」了）安装了 Yoink，那么事情就简单多了：拖动图片 - 屏幕边缘浮现 Yoink 窗口 - 拖入 - 拖出。

[![](.evernote-content/61A0D42C-39DF-43A4-844C-5E17D49041F5/A064D44A-5951-465E-AFB4-F887140380EA.gif)](http://sspai.com/attachment/origin/2015/08/21/275636.gif)  
当然，除了图片，Yoink 支持其它任何格式（至少我还没发现拖不进去的东西）的文件，即使选中一段文字拖进去，Yoink 也会为你自动生成一个纯文本文件。

我是你最贴心的临时文件夹
------------

回过头来，似乎 Yoink 就是一个临时文件夹而已，当然事实……就是这样！

然而正如你所见，Yoink 几乎已经融入系统，它只会在你进行「拖动文件」这个动作的时候出现；其次，你不用打开 Finder 去找到这个所谓的「文件夹」；最重要的是，Yoink 「临时」得非常彻底，无论你是从本地还是互联网里拖动文件进去，在你拖出去以后即刻从 Yoink 中消失，界面也随之隐藏，你不用再回过头去删除它，当然如果你是从本地拖入，那么源文件也不会发生任何改动。

值得一提的是，如果你有更多样化的需求，或许你需要知道 Yoink 的以下几个特性：

* 除了一次性即用即删，Yoink 也支持锁定文件便于多次使用，通过文件旁边的锁状按钮即可实现；

[![](.evernote-content/61A0D42C-39DF-43A4-844C-5E17D49041F5/3145E76F-3765-4FAD-9262-409BCB9B1A31.jpg)](http://cdn.sspai.com/attachment/thumbnail/2015/08/23/7709c5aa355af70c7c8a4b6c14229a1b43752_mw_800_wm_1_wmp_3.jpg)

* 如果你拖入的文件过多以至于你忘记了它的摸样，那么你可以通过文件旁边的眼睛状按钮进行预览，预览格式支持图片、文本以及一些设计源文件，文件夹暂不支持；

[![](.evernote-content/61A0D42C-39DF-43A4-844C-5E17D49041F5/8CB36289-0609-4183-BB19-ADAD803C55E1.jpg)](http://cdn.sspai.com/attachment/thumbnail/2015/08/23/469d44d076e5b6b7c37ad6812e8f4bd443753_mw_800_wm_1_wmp_3.jpg)

* Yoink 支持多个文件同时拖入拖出，拖入后呈堆栈状，你也可以在 Yoink 中将其解散成为单个文件；

[![](.evernote-content/61A0D42C-39DF-43A4-844C-5E17D49041F5/20AAA7C8-90F5-42D2-89E3-A510AE0FEE7D.jpg)](http://cdn.sspai.com/attachment/thumbnail/2015/08/23/d371697094a73577074c10fb6688f2ff43754_mw_800_wm_1_wmp_3.jpg)

* Yoink 可以设置为开始拖动文件即自动出现，也可以切换为将文件拖入边缘后才出现；
* Yoink 默认设置是无文件拖入或文件拖出后即隐藏，但是你可以设置快捷键让其「招之则来挥之则去」；
* 除了默认形态，你也可以自由设定 Yoink 在出现在屏幕中的位置及大小，怎么顺手怎么来。不过无论你设置多大，Yoink 都支持拖入无限多的文件，不过想必你也不会一次性拖无限多的文件进去吧。

一曲忠诚的赞歌
-------

事实上，OS X 并不是一个十全十美的操作系统，但是在无数开发者的努力之下，我们总能在 OS X 中找到无数优秀且可以完美解决需求的软件，比如 Yoink。

当然除了 Yoink，相似功能的软件还有其他选择，比如 [Dropshelf](https://itunes.apple.com/cn/app/id540404405?mt=12&uo=4&at=10lJSw)、[Unclutter](https://itunes.apple.com/cn/app/unclutter/id577085396?mt=12&uo=4&at=10lJSw)，不同在于 Dropshelf 功能更简洁（仅有最简单的文件暂存），Unclutter 功能更复杂（还集成了剪贴板管理、便签工具）。而 Yoink 专注于「文件暂存」这一点，并围绕它在功能上作了不少扩展，所以相比之下，我更倾向于选择 Yoink。
