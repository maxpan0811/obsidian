节省 Windows 存储空间，随用随取的「按需同步」网盘你有这 5 个选择
======================================

提到网盘，很多人首先想到的多半是某个以限制下载速度为主要盈利方式的国内网盘服务提供商。在国人眼中，网盘似乎也早已经成为了存储、分享各种资源的必要中转。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/D5AAC5F5-70AD-4B99-8BD2-32135CB9BE14.png)

百度网盘变成了资源分享站

但在工作与效率这个话题下，网盘这种形态更重要的功能其实在于将本地文件目录和云端空间进行同步，进而实现类似 [BYOD](https://en.wikipedia.org/wiki/Bring_your_own_device) 的效果。我们可以随时将重要的工作文件同步到自己的设备上进行办公，由于文件的更新和修改都会被同步回云端，工作设备也不会因此被限制在单个设备，极大地减少了因数据修改、同步而引发的拷贝和备份成本。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/CEE369BE-226F-4260-B23C-F9CA395AD971.png)

OneDrive 官网

和「原始」的复制粘贴相比，网盘的出现从某种程度上来说是对工作方式的一种革新。但传统网盘服务其实也并不适合现代办公需求，如果你打算将网盘引入自己的跨设备工作流，按需同步就是值得关注的核心体验之一。

为什么「按需同步」很重要
------------

早期网盘的同步方式非常机械，为了摸清文件层级，我们往往需要将网盘云端目录中的大量数据统统下载到本地。这种全盘同步的方式不仅会在外出办公的过程中消耗不必要的网络流量，对当前市面上大部分存储空间并不充裕的轻薄办公设备而言也是一种奢侈的空间浪费。

「按需同步」这一同步特性应运而生。

顾名思义，按需同步可以根据我们的实际需要进行同步操作，所有文件修改、增添都会上传到云端，除了文件目录和列表，同步时并不会将所有数据下载到本地。只有需要使用某个文件时才会将其从云端下载到电脑上。这就大大节省了同步需要的网络流量和本地存储空间。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/A38DAEB9-DF8A-4FA3-8836-8A9DAC730B5B.png)

按需同步可以帮助我们很方便地获取文件目录结构

微软自 Windows 10 v1709 开始引入了按需同步特性并首先运用于自家的 OneDrive 之上，使用 OneDrive 同步时系统会首先同步文件列表，之后再根据用户的实际需求有选择性地下载云端文件到本地。配合 Windows 10 的 [存储空间管理机制](https://sspai.com/post/58167)，系统会在一段时间后自动释放已经离线但不再使用的文件，进一步节省本地存储空间。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/15E4F5E5-BE58-496E-B1E4-493A37769CE9.png)

Windows 10 的存储感知功能

可见支持按需同步的网盘有着其他传统网盘不可比拟的优势：一方面文件数据被放在了云端这个高可靠性的「保险柜」中（鸡蛋不要放在一个篮子里的原理），文件目录和层级则能在本地得到较好地保留，管理更加方便；另一方面进行文件同步时我们又可以根据进行下载，加快同步速度，节省本地存储空间。

那我们都有哪些支持这一特性的网盘服务可以选择呢？

支持按需同步的网盘服务
-----------

在为 Windows 10 系统带来 OneDrive 按需同步支持的同时，微软也将这一功能进行了开放，任何第三方的网盘服务都可以在 Windows 10 上实现和 OneDrive 一样的文件同步使用效果。因此在这一部分我们就来对市面上已经支持「按需同步」或类似功能的网盘服务进行盘点，大家可以结合自己的使用场景进行选择。

### 原生的就是香：OneDrive

在 Windows 10 上使用按需同步的网盘服务，我们的首选自然就是上面反复提及的 OneDrive。只需拥有一个 Microsoft ID，我们就可以免费获得 OneDrive 的 5G 在线云空间，对付一般的办公和学习需求绰绰有余。

而 OneDrive 最大的优势自然就是和 Windows 10 的深度整合，上面同样也提到，在最新版 Windows 10 当中，OneDrive 可以配合存储感知功能按照特定的时间周期对网盘同步文件进行自动清理。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/D9E79979-67C6-4F9A-AFDF-71160F3986BE.png)

与此同时，即便我们在本地没有安装可以打开文件的应用，也可以通过 OneDrive 提供的缩略图来预览文件内容。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/E4849A15-593F-44D6-8880-DE2603602A56.png)

即便本地没有安装配套应用也能直接在资源管理器中预览

OneDrive 的按需同步被称作「文件随选」，在文件同步的过程中，OneDrive 会首先下载文件列表，之后我们可以根据文件管理器中图标角标来判断当前的同步状态：

**空心的云朵**表示目前属于联机状态，文件尚未下载到本地。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/6901870F-A96F-45A6-8CE4-6EE84CC73848.png)

联机但未下载

**空心的对勾**表示为本机可用，文件已经下载并且脱机可用，不过在没有进一步操作之前，会根据系统的存储感知设定定时被清理。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/04853B8F-6FA0-4843-A79E-771C7FC42B44.png)

本机可用，但会被清理

**实心的对勾**则表示文件将始终在本机中。通过右键单击文件，我们可以选择是否让文件始终保留在设备中，还是根据存储感知设定定时进行释放。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/E1EDB9B1-55EA-4C79-B63F-9A18CD9E7366.png)

本地始终可用

虽然理想状态下 OneDrive 配合 Windows 10 使用体验较好，但现阶段 OneDrive 在国内网络环境下同步速度比较慢，因而使用场景也大多受限于日常办公、学习；此外如果你觉得免费版提供的 5GB 存储空间不够用，扩充容量还需要通过 Office 365 订阅来进行扩充，其中家庭版订阅因其家庭成员每人 1TB 在线存储空间而广受欢迎。

### 基于私有云的按需同步网盘：群晖 Drive

除了用来存储影音文件，NAS 私有云也是很多人用来作为个人资料同步网盘的首选：它一方面解决了云盘存储的空间问题，云端存储空间大小取决于 NAS 配置选择，另一方面则解决了网盘的访问速度问题，文件同步速度主要取决于 NAS 性能和网络上行速度。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/48240D4D-A148-4FC4-817F-415797D5C04D.png)

群晖 Drive 2.0 的按需同步

我们此前介绍果的 [群晖 Drive 2.0](https://sspai.com/post/55530) 也加入了按需同步功能，并且将其整合到了 Windows 10 的存储感知中，同样支持定时的清理下载内容来释放空间。从这个角度来说，群晖 Drive 2.0 完全可以看作是 OneDrive 的「个人私有云」版本。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/F6D720BF-568C-4066-9FC5-0E3BE2F5D30B.png)

按需同步选项

此外群晖 Drive 的同步状态也和 OneDrive 基本一致，我们同样可以在文件上通过鼠标右键选择永久锁定副本或者释放空间。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/ABC0C2B2-66F9-489A-99BB-582849B2DCB5.png)

链接共享

相比 OneDrive，群晖 Drive 还提供了类似协作编辑的功能，比如说你可以邀请你的朋友来合作编辑文件，更加适合协作办公场景；不过这个编辑功能需要整合群晖 Office 的在线办公套件，同时群晖本身也有着不小的搭设成本，这个相比 OneDrive 更加稳定快速的方案并不适合所有人。

### iPhone 用户的完美搭配：iCloud Drive

iCloud 已经深度整合到了 Apple 的生态系统当中，在 iPhone 上拍摄的照片、保存的邮件、创建的日历等等信息，都可以通过 iCloud 备份并同步到其它 Apple 设备上。

而如果你因为自身或工作原因选择 Windows 作为办公设备，其实也可以通过 iCloud 获得不错的云端同步体验。除了查看拍摄的图片、保存的书签等等，iCloud for Windows 同样也支持按需同步功能。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/EBF9F362-C8C5-4502-A75E-1AAD5820656A.png)

iCloud for Windows

在 Windows Store 中安装 iCloud 并登录后，我们就可以在 Windows 10 的资源管理器中看到 iCloud 的文件存储目录了。和上面介绍的云盘一样，我们可以用同样的角标样式来判断 iCloud 文件在 Windows 上的按需同步状态，因为基于 Windows 10 按需同步的原生支持，在 Windows 系统设置的存储感知功能中也可以看到 iCloud 云盘的相关设置项目。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/8B040D33-1A18-4634-AA16-94314629E369.png)

iCoud 文件目录

值得一提的是使用 iCloud 云盘作为学习和办公文件中转一定得考虑到文档类型的问题，我们在其它 Apple 设备上编辑、保存的 iWork 文件在 Windows 系统上虽然没有原生的应用支持，但依然可以通过 iWork 网页版本进行查看和编辑。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/D83C22C0-30D1-4C7B-BCB9-68D48C8A3752.png)

在 iCloud.com 中打开

总体来看，Windows 版的 iCloud 在按需同步体验上也非常接近 OneDrive，虽然缺少一些原生应用支持，但通过 iCloud 网页版能够解决大都数基于 iWork 的学习和办公需求，非常适合 Apple 产品和服务的重度用户。要说缺点的话，iCloud 云盘免费版只有 5GB 存储空间，如果你有更大的空间需求则需要进行付费购买。

### 兼容性最好的按需同步网盘：Dropbox

相比起前面介绍的同步网盘，Dropbox 或许是其中功能最为全面的一个 —— 除了可以和系统资源管理器深度整合之外，还可以通过插件的形式与第三方服务进行整合，实现高效办公。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/343791C2-4A13-478D-AE6E-132A6E647615.png)

Dropbox 的「智能同步」

Dropbox 中也有着按需同步功能，但该功能仅限付费用户使用，名字叫做「智能同步」。

虽然名称不同，但智能同步在 Windows 上的功能实现和上面提到的几款服务类似，即同步时文件夹以及文件均只有目录和列表，需要本地访问才会下载数据；并且 Dropbox 内置了自动节省硬盘空间的相关设定，几个月没有编辑的文件也会被自动清除存储并转为默认在线访问。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/0F5FCBED-AD4D-4AA4-A0B9-A41C600A54E8.png)

智能同步选项

**因此 Dropbox 的这项智能同步其实可以在更旧的系统上使用**（官方表示该功能可以在 Windows 7 上运行），不过也正是由于这种兼容性，Dropbox 在 Windows 10 上的文件状态提示与上面提到的几款略有差别，在线文件的角标会显示为一个「×」，将其下载到本地后文件图标上则不会有任何状态提示。

唯一的缺点可能是 Dropbox 在国内的访问上，另外就像我们开头就说的那样，「智能同步」这项功能必须购买付费套餐才能使用。

### 国内用户的稳定之选：坚果云

在以上介绍的同步网盘大多为国外网盘服务，而在按需同步这个需求上，大家常常选择的坚果云其实也提供了类似的功能。

坚果云的按需同步为独立开发功能，也叫「云桥服务」。云桥服务并不依赖于 Windows 系统版本，换句话说它同样有着非常不错的系统兼容性。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/63773B87-FCE1-4575-8E7A-2F0E7982552B.gif)

云桥模式同样支持按需取用文件

坚果云的云桥服务和前面提到的按需同步文件索取方式类似，由于并未接入 Windows 的按需同步功能，文件同步状态显示和同类产品略有区别：同步但未下载的文件在资源管理器中以半透明图标显示；用「☁」标记的文件只有双击后才会下载使用，下载后图标左下方的标记则会变成一个「√」；而图标左下方有同步标识则表明该文件正在被同步，或当前没有网络连接，文档改动已经被被挂起。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/5E589867-750E-44EC-AEAA-F4D7ACDEA1DF.png)

坚果云的不同同步状态提示

坚果云为云桥服务提供了缓存清理功能，允许我们自行设定离线缓存文件的最大使用量和缓存周期，我们同样可以让它自动清理掉那些长时间没有使用的文件来节省空间。

![](.evernote-content/0E4E6A98-9DD5-4816-9860-185BFCD9C585/4E122F6B-0D69-48D3-8236-4127134F7805.png)

相比本文提到的大多数网盘服务，使用坚果云进行按需同步最大的优势在于其网络访问的稳定性，同时和 Dropbox 那种必须才能使用「智能同步」的服务相比，坚果云即便是免费版本也可以使用「云桥」，免费版提供 1GB 的上传和 3GB 的下载，功能限制主要集中在文件历史版本和误删恢复上。加上它对 Office 365 和 WebDAV 协议的良好支持，非常适合需要跨设备学习、工作但本地存储空间有限的用户。

结语
--

作为本地重要工作的保险箱，同步网盘的出现着实解决了个人文件如何多端同步的问题，而作为细分的功能，「按需同步」一方面可以完整地将个人文件列表同步到本地，另一方面又可以根据我们的实际需要有选择地下载、存储文件内容，同时节省了本地存储空间、网络数据流量和等待时间，可谓是一举多得。
