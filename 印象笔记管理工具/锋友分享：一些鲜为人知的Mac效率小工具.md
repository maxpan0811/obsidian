# 锋友分享：一些鲜为人知的Mac效率小工具

---

近日，锋友“[Faye法爷](http://bbs.feng.com/read-htm-tid-10264060.html)”分享了几个实用又好用的软件，他表示，工具类软件不在多，能够发挥作用，并且提高效率才是最重要的，下面我们一起来了解一下，希望能够给有需要的锋友提供一些帮助  
  
　**1. Noizio**

Noizio 是一款 OS X 系统下的白噪音应用，可以让自己感觉是坐在一个壁炉前、咖啡馆里、森林里或者是海边，有 10 种模拟音效，可以让你忽略其他噪音，专注工作。这是一种自然而然的感觉，不需要刻意去倾听。Noizio 支持优山美地、支持 Retina，可以随机启动。

**2. Recordit**

Recordit 是一款 Windows、OS X 平台下的屏幕录制工具，非常易用并且可以导出为 GIF 动画。这款软件一共就一个按钮，即开始/结束。视频会自动上传到 Recordit 网站并得到一个 URL 网址，你可以直接分享这个 URL 或者打开后下载 GIF 再分享，支持直接分享到 Twitter。

**3. littlefox**

littlefox(小狐狸加速器)是一款科学上网工具，全平台客户端，简单易用，无需配置，一个按钮即可控制连接/断开。除此之外，这款软件的速度非常快，刷 Facebook 和刷微博一样快，看海外超清视频和看优酷一样流畅。

**4. CmdMarkdown**

Cmd Markdown(作业部落)是一款 Markdown 编辑阅读工具，拥有 Windows、OS X、Linux 以及网页版，除了以 Markdown 格式进行编辑、阅读外，还支持文档管理、代码支持以及针对学术论文的 LaTeX 公式支持。界面简洁，让你专心沉浸在写作中。

**5. Snappy**

Snappy 是一个快速方便的截屏、分享工具。使用 Snappy 截屏后，图片会自动保持顶端停在桌面上。你可以移动图片，或者再次截屏，每次截下来的图片都会保持在桌面最上方。然后，你可以右键点击你要的图片，选择拷贝、保存、分享(Email、Facebook 和 Pinterest 等)或删除。Snappy 免去了参考资料时，不同窗口之间频繁切换的烦恼。

**6. Imagus**

在浏览网页时，将鼠标悬停在缩略图上就可以查看原始图片，不用点开查看大图，使用起来非常方便。

**7. Snap**

Snap 可以给这些 Dock 上的程序添加快捷方式，默认从左向右依次排序，第一个程序 Command + 1，第二个程序 Command + 2 (Finder 与 Trash 默认跳过不排序)。添加或修改 Dock 程序，Snap 会自动重排。第二次按下快捷键则是最小化。如果你不满意 Snap 默认的 1234，还可以自定义全局快捷键。按照自己的习惯排列 Dock 后，用键盘就能控制程序打开/最小化。

**8. HotkeyEVE**

Hotkey EVE 是 Mac 上不可多得的奇葩软件，它会监视你进行什么鼠标操作，只要该操作有对应的快捷键，它就会弹出提示告知快捷键内容。想成为打字如飞的高手？让它来帮助你一段时间吧。
