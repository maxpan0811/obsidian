为了更好地支持 Macbook，我又升级了显示器了！
--------------------------

原创 直树桑 拾书小记

来自专辑

利器安利

![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/A87809E9-C3A2-44EE-8E0E-0BF16162EB53.gif)

看我文章的朋友们应该都知道，去年年底我给自己上了台综合性能比较高的2K 显示器——明基 PD2500Q，当时我在文章中提到了自己的需求：

1、可以实现和自己在用的 Macbook近似的色彩；

2、可以内置拓展坞，减少我 Macbook外置Hub连接过多设备产生的供电不足的问题；

3、可以实现低蓝光，做到轻松护眼；

4、方便多显示器拓展；

5、价格承受范围内；

6、不能丑

大半年过去了，经过了一段时间和线材的折腾后，我又提出了新的需求，那就是第七点，直接可通过 TypeC 接口和 Macbook连接，用 Macbook 的朋友知道这点有多重要——以往只能通过 Hdmi 线或 DP 线来连接，也就是必须要有一个价格动辄三位数的拓展坞来支持才行。

因为长期都在用明基的产品，对这个品牌信任度还是挺高的。加之之前使用明基 PD2500Q 的优质体验，于是在升级设备的时候便第一时间往明基显示器的新品这边看了，这时候刚出来的明基 PD2705Q 则映入了我的眼帘。

![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/DC2A584D-CDCC-45E8-80E7-3F02EF6C57FF.jpg)

进行简单对比后，我发现这款新品**不仅涵盖了前者的所有功能（护眼模式加持、M-book模式 近似 Macbook 的色彩）****，还增加了不少新特性**，其中就包括了我需要的 TypeC，下面我先为大家一一介绍这款新品新增的特性。

**Type C接口、反向供电设计**

明基 PD2705Q 增加了 TypeC 的输出支持，可支持65W的TypeC（官方自带一条线材），这条线一方面可以直接支持最新 Macbook 输出视频、音频、数据等信号，还可以支持对 Macbook 的反向充电。

![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/310A4639-209B-4DDF-9381-A28574418199.jpg)

前阵子去大理出行，我就带上了它开启了实验，过程中发现真的挺便利，反向充电让我解脱了适配器的束缚，让桌面更加简洁。

两条线就解决了你的外接显示器的需求，想想都惬意![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/A23054B6-6E3F-4DA8-AC2D-08DAD1BE8750.png)。

**27 inch大屏的畅快体验**

![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/CD13A513-6BE2-4383-8FB2-4719657921CD.jpg)

明基的新品 PD2705Q 是一款27inch 的显示器，用惯了25inch 的 PD2500Q 后突然感觉视野大了一圈。

大屏的好处显而易见，我们可以在同一桌面上放更多的任务项进行同时处理，提升了效率。

![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/0E7C187F-3DDD-49FB-9FF4-2221FCA0D117.jpg)

上图前排左边的为 PD2500Q，后排右边的为 PD2705Q，由于桌面大小限制，没办法将这两台显示器放在同一个平面进行对比，近大远小导致了大小差距不大，但**实际 PD2705Q 要比 PD2500Q 大了一圈的****，目前已经成为了我现在的办公主力**，而退居二线的 PD2500Q 已成了我的娱乐主力，目前主要是搭配 Switch 使用。

**色彩还原度高+HDR 加成**

明基 PD2705Q 显示器的色彩是非常值得一提的点，每台机子在出厂的时候就进行了校色处理，每台机器随机携带了纸质出厂报告，官方出厂校色报告上给出的△E值仅有0.5817，所以打开显示器你就可以感觉到非常友好的视觉体验。这样也免去了自己购买或者租借校色仪进行自主校色的财力和人力成本，拿到手就可以直接用！

![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/A23CE1DF-CD4F-4E19-A72C-DC7006772E50.jpg)

明基 PD2705Q 在色域上覆盖到了100%sRGB/Rec.709色域的程度，让内容创作者在输出作品时可以有更精准的色彩表达。

除此之外，新加入了 HDR 模式的 PD2705Q 还可以在满足基本观影需求的基础上增加更多的视觉感受。

![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/0D9468C9-18B4-45E2-9295-F3159EFDF24B.jpg)

在该模式下，我们可以在观影、游戏中得到更好的体验，简单来说就是能看得更爽啦。窄边框的设计，无形中让这款产品的颜值得到进一步的提升。

**随机携带的「番茄钟」**

明基为这款设备搭载了新特性，我们可以在机身菜单：人体工程学-智能提醒中打开这个功能。

![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/0FB11FAF-02B0-420E-8046-213A11483C09.jpg)

大家在这里还可以设置提醒的间隔时间和提示时长，时间一到它就会在显示器上提醒你去休息，可以说是显示器自带的贴身番茄钟了。

![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/6927BA56-4AD9-4E88-B1F8-61B9361AE995.gif)

如果你本来就使用番茄工作法工作，如果适应了这个提示或许也是个不错的代替品。

**不可忽视的 Display Pilot**

近期明基显示器为用户提供的 Display Pilot 开始支持 MacOS系统了，并且也支持了这台显示器，于是我就体验了会儿。

直接感觉就是一个字，爽。之所以爽是因为这个软件真的很好使，无形中为我们节省了不少时间，下面给大家谈谈我觉得比较实用的两点：

**①设置区域桌面**

![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/BA6A776E-7BCD-4CC0-ACE1-F8A06BC88F66.png)

通过 Display Pilot下 Tools下的 Desktop Partition 我们就可以给我们的桌面进行快速拆分了，通过这个功能我们可以更好地利用大屏效率低进行协同办公。

**②感应自动旋转**

现在中高端显示器支持旋转已经快成必备的活儿了，明基的主流显示器也都支持这个功能。

![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/EAAECB91-39B2-48DC-971D-BCF0D4BBE753.png)

![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/3C66BADD-7579-480D-BE58-878C6CD0553E.jpg)

在没有 Display Pilot 的日子，我只能手动调节，效率低不成，还总是要歪着头去调出这个功能，久而久之变开始不太想用这个功能了，而 Display Pilot 的出现则很好地解决了这个问题。

**丰富的色彩模式**

丰富的色彩一直是明基产品的主打、也是区别于同类产品的一大亮点，除了和 Macbook 近似色彩的 M-book 模式、上面提到的增加色彩还原的HDR 模式，明基 PD2705Q 还给长时间需要待在电脑前工作的人带来了护眼模式。

同时，也给了设计相关专业的人提供了更多的选择。比如暗房模式、sRGB 模式、CAD/CAM 模式以及动画设计等。可以说有非常广的覆盖面了，作为一个长期使用的显示器完全可以胜任。有了Display Pilot 后，我们还可以在软件端对模式进行轻松控制。

![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/6EBEA50D-7AA2-4DDC-9CA8-B8538F5B556E.gif)

除了这些功能外，明基还为这台设备增加了 KVM、ICC Sync 等功能，前者可以支持一套键鼠控制两个主机，后者对于不会色彩管理的朋友们来说相当方便，可以在切换色彩模式的同时，匹配色彩文件，确保色彩输出一致，同时还有 DP-OUT 功能轻松搭建双屏，有兴趣的朋友不妨一试!

![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/1AC36A2A-064D-4D61-AE7A-7F11186444FE.jpg)

明基 PD2705Q 同样提供了丰富的接口，为我们的日常办公、生活助力。

**写在最后**

通过半个月来的体验，我对这款显示器的性能是十分满意的，TypeC 和方向充电让我简直爱不释手，它对当下的自己来说完全够用了。**不仅可以在家充当很好的工作伴侣，即使在外旅居也变成可以考虑的同行效率伙伴了。**

![](.evernote-content/4363C4A2-D648-44FF-9898-476FBFCFC841/83E6EF5F-4757-4E5B-B7E2-D8A150429543.jpg)

明基 PD2705Q 的上市无疑给了那些对显示器色彩、工作效率有要求的朋友以冲击，仿佛在告诉你：“小伙子，选我准没错。”
