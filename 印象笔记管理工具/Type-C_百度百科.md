# Type-C_百度百科

---

:   Type-C
    *[](#)*编辑

2015年CES大展上，Intel联合USB实施者论坛向公众展示了USB 3.1的威力，具体搭配的接口是USB Type C，能够正反随便插，大小也与[Micro-USB](https://baike.baidu.com/item/Micro-USB)相差无几。理论上，USB 3.1 Type C的传输速度能够达到10Gbps。[1]

Type-C是USB接口的一种连接介面，不分正反两面均可插入，大小约为8.3mm×2.5mm，和其他介面一样支持USB标准的充电、数据传输、显示输出等功能。Type-C由USB Implementers Forum制定，在2014年获得苹果、谷歌、[英特尔](https://baike.baidu.com/item/%E8%8B%B1%E7%89%B9%E5%B0%94)、微软等厂商支持后开始普及。[2] 
2015年8月1日，特科芯推出了全球首款Type-C接口的移动固态硬盘。[3] 
2015年10月10日，铁威马推出全球首款USB 3.1 Type-C接口磁盘阵列。[4] 
2016年1月1日全球首款Type-C指纹加密移动固态硬盘诞生，由特科芯有限公司制造。[5]

Type-C

2015年[CES](https://baike.baidu.com/item/CES/3695360)大展上将会出现首批支持[USB 3.1](https://baike.baidu.com/item/USB%203.1)接口规范的[设备](https://baike.baidu.com/item/%E8%AE%BE%E5%A4%87)。在CES大展上，Intel联合USB实施者论坛向公众展示了USB 3.1的威力，具体搭配的接口是USB Type C，也就是传说中能够正反随便插那种。[1]

运用
--

*[](#)*编辑

从测试视频来看，通过两款三星840 EVO SSD组建RAID 0之后，实测USB 3.0 Type C端口的读写速度都能达到800MB/s以上。也就是说，2015年的大部分固态盘都无法跑满USB 3.1 Type C接口的带宽。[1]

当然，这只是展示，2015年除了[诺基亚N1](https://baike.baidu.com/item/%E8%AF%BA%E5%9F%BA%E4%BA%9AN1)平板、[小米4C](https://baike.baidu.com/item/%E5%B0%8F%E7%B1%B34C)智能手机[6] 
以及新推出的[macbook](https://baike.baidu.com/item/macbook/7817055)的USB-C接口，其余很少有产品配备USB 3.1 Type C接口，想要普及还遥遥无期。[1] 

[![](.evernote-content/3EB33667-FD16-4FB7-9B7F-0C90568439ED/F620F355-AE12-4A51-8493-FDC70219A78D.jpg)](https://baike.baidu.com/pic/Type-C/16565336/0/32fa828ba61ea8d34110aea5900a304e251f5865?fr=lemma&ct=single)

2015年6月3日，USB Type-C端口被Thunderbolt 3采用。英特尔宣布，Thunderbolt 3将不再使用以前的方块式的接口，而是兼容USB3.1的标准，但仍会保留标志性的闪电logo。速度飙升至40Gbps。采用该接口的设备将在2016年上半年投入市场。Type C有望借此得到普及。[7]

2015年8月1日，特科芯推出了全球首款Type-C接口的移动[固态硬盘](https://baike.baidu.com/item/%E5%9B%BA%E6%80%81%E7%A1%AC%E7%9B%98)。该款[SSD](https://baike.baidu.com/item/SSD/5906)提供了最新的Type-C接口，支持USB接口双面插入。特科芯移动固态硬盘也分为旗舰级和标准级两款产品，旗舰级无疑是速度更快可靠度更高，偏向商务或者发烧型用户。而标准级产品则面向主流大众用户，但实际传输速度也快的惊人。[3]

2015年10月10日，铁威马推出全球首款USB 3.1 Type-C接口磁盘阵列，实际测试速度达1.25GB/s。[4]

2016年1月1日，全球首款Type-C指纹加密移动固态硬盘诞生，由特科芯有限公司制造[5] 
。
