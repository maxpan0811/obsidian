**▍****Trancy：沉浸式语言学习扩展**

* 平台：各平台主流浏览器
* 关键词：翻译、双语字幕

@Snow：Trancy 是一款支持多语言翻译的浏览器扩展插件，它不仅支持划词翻译、段落翻译、全文翻译等基础功能，还可以为 YouTube、Netflix、TED、Disney+ 等平台提供双语字幕支持，帮助用户更好地理解视频内容。

Trancy 最为突出的特点，就是提供了不同版式的视频双语字幕。在浏览器中打开视频内容，点击视频下方的紫色 Trancy 图标，即可进入双语字幕播放模式。在该模式下播放视频会默认展示你标记的母语以及想要学习语言的双语字幕，你可以随时调整字幕字体及大小、快速切换前后单句以及收藏需要重点学习的单词和例句。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/171815D8-546A-44AF-9A32-0D8B00F4E857)

Trancy 还提供了「剧场/影院/阅读/混合」共 4 种字幕版式，其中阅读模式和混合模式，都会将视频移至屏幕左侧，字幕则以文章的方式展现，点击字幕前时间点可以快速跳转。在这两种模式下，你可以将注意力集中在字幕上，更好地理解句子上下文关系。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/FE049928-971A-4D3F-A93B-06B5E791680A)

此外，Trancy 还为语言学习者提供了一个独立的练习模式，你可以根据自身需求，有针对性地练习口语、听力、词汇量等薄弱项。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/4EBD209F-24CA-4DCA-9332-0BD53602D567)

Trancy 的翻译引擎除了支持常用的 Google、Microsoft、DeepL 外，还支持接入 OpenAI 或自定义 API。借助 AI 能力，Trancy 不仅可以提供字幕转译和断句的准确率，还可以实现联系上下文的精准释义、AI 总结、AI 语法分析等功能，帮助用户更高效地学习语言。

Trancy 的划词翻译、全局翻译、双语字幕等基础功能只需安装扩展插件即可使用，生词本、词句收藏等需积累化学习功能需注册并登入账号后使用。若想调用 Trancy 内置的 AI 功能，则需解锁付费服务，你可以按 22 元/月 或 198 元/年的价格进行订阅，Trancy 分别提供了 3 天及一周的免费试用服务。

Trancy 支持 Chrome、Edge、Firefox、Safari 等主流桌面端浏览器，以及 iOS 版 Safari 和 Android 版 Firefox 和 Lemur 等移动端浏览器。除扩展外，Trancy 还在 iOS 平台上线了同名应用，你可以在应用中同步 YouTube 订阅，随时随地进行语言学习。

**▍****Joi Daily Planner：日历与提醒的一站式应用**

* 平台：iOS
* 关键词：极简行程、日历、提醒事项

@ElijahLee：越来越多的行程规划应用开始将日历、提醒事项等整合在一起展示。Joi Daily Planner 就是一款这样的聚合应用，它将日历、提醒事项和习惯追踪整合在一个时间线上。

Joi Daily Planner 主打简约设计。应用的主页分为三个区域，顶部是具体的日期以及一周 7 天的周历，它只展示最小化的信息量，比如去掉日期中的 -day 后缀，年份、月份等都以灰色小字挤在右上角。在周历中，当前日期使用红点装饰高亮。中部展示的是具体一天的时间安排，包括日历事件、提醒事项和习惯清单。已完成的事件使用删除线置灰显示，进一步让用户聚焦在未来的事件。底部则是三枚操作按钮，分别是应用设置、添加和月历。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/FA5EDAB3-F2C8-4899-A142-78B8985E526C)

Joi Daily Planner 自带了习惯追踪功能，在初次打开时，应用会收集你起床和晚上休息的时间，并且让你在默认的活动中挑选几个活动，作为每天白天的习惯。这些默认的活动包括锻炼、冥想、瑜伽、读书、喝水等各种各样的项目。这些项目会出现在每天的日程事件中，因为没有设置具体的时间，它们显示在所有日程事件的下方。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/B2F5C475-EF9E-42E2-A800-526ACB7BB322)

应用下方的 + 号，可以选择添加日历、提醒事项和习惯，但功能非常简陋。日历事件只能添加、删除和标记为已完成，不能修改；待办事项只能标记已完成，不能修改也不能删除，更没有复杂的备注、周期等等设置，更像是一个尚未开发完全的功能。从主页往下拉，可以进入总结页面，这里会根据时间与当日事件的完成情况，总结一天完成的任务和习惯。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/59BD3102-8A69-4DF1-9D72-86447172AAB8)

应用集成 Apple 提醒事项、Apple 健康，支持更换图标以及小组件，但这些功能都要求付费订阅。费用是 38 元/月、298 元/年，以及 598 元买断。你可以在 App Store 下载 Joi Daily Planner。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/FB7A07F9-96B2-4E0D-90ED-858F81DFF0E4)

**▍****OnTime：从容面对每次演讲**

* 平台：Android
* 关键词：演讲、计时器

@Peggy\_：由于职业缘故，我经常需要面临一些在给定时间内进行主题汇报的场景，无论是一次学习分享、还是一次读书心得，或者是重要的技能比赛，在特定时间内完成相应的文稿汇报都是一件不容易的事情。影响最大的因素之一便是时间，演讲和比赛经验再丰富的人也会十分注重时间的把控，否则一次干货满满的演讲可能变成惹人困倦的无聊输出，准备好的精彩文稿可能被迫在讲至高潮时戛然而止。

为了避免上述尴尬的场景出现，演讲者和表演者可以反复背诵稿件并多次配合计时锻炼，OnTime 就想要帮助我们解决演讲和表演当中的计时问题。从外表上来看，OnTime 和我们手机内置的计时器似乎没有不同，但是它的独特之处在于将时间通过显眼的背景颜色进行划分，我们可以根据自己的演讲时长或是计时需求将时间划分为三块。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/2A72E22A-D945-47A6-9BD7-AC3BEAD35010)

在第一大块的时间当中，OnTime 和普通计时器一样仅会默默计时，一旦到达我们设置的警告时间，OnTime 的背景便会变为黄色，这也意味着我们的演讲时间或许过半，而等到演讲时间所剩无几时，OnTime 便会切换为红色背景来提醒演讲者加速结尾。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/55FB9EA3-5096-42E3-B502-E3343F50931B)

在 OnTime 中，具体的倒计时时长和颜色切换节点均能自定义，基本上能够满足各类计时场景；只不过我们需要注意设定的警告时间需要小于总时长，否则应用会不工作。尽管只是背景颜色上的改变，却能让演讲者一目了然，对于时间的把控做到心中有数，因此它不仅能用作我们日常的自我训练场景，也可以作为一个裁判员使用的计时器，避免下次单位有类似活动时，还需要人工打断演讲者的尴尬。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/77556A09-5A9C-4300-8D0C-C04CA5BC6279)

如果你需要这样一款小工具，不妨通过 F-Droid 下载体验 OnTime，应用开源且完全免费。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/A1FEB39B-EB02-495D-A4E4-501F5E92C0BC)

**▍****PicGuard Pro：为你的照片添加水印**

* 平台：Android
* 关键词：照片水印

@Peggy\_：尽管当下是一个个人信息泄露严重的时代，但这并不意味着我们可以放任这种情况，任何时候我们都要对自己的隐私信息重视一些，再重视一些。大到办理贷款，小到开具一次简单的证明材料，都可能会泄露我们的重要信息。

PicGuard Pro 是一款能为图片加水印的小工具，相比于直接把自己的身份证照片发送给别人，打上专用水印能够有效避免图片被有心之人挪作他用。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/D6FBC56D-9BB8-4B92-920D-A94F855E56D6)

PicGuard Pro 的使用方法很简单，下载应用后，PicGuard Pro 会给出一些说明，阐述其所需要的权限和用途，当然在这里不要着急全部同意，我们可以选择再想想，尤其是联网权限需要慎重考虑。事实上即使不授予应用联网权限，也不影响其后续的工作。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/C91A19E4-89CB-46C6-86B6-D77BB0D436B9)

进入主界面后，点击加号便能直接挑选需要处理的图片，应用允许我们一次选择多张图片。完成图片选择后，我们可以自行标记想要添加的水印文字，并且对水印文字的颜色，透明度进行调整，不影响使用的同时兼顾美观。PicGuard Pro 也支持对水印文字的大小、行间距、文本间距进行调整，甚至允许我们设置不同的字体。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/07F59CA4-8A67-4535-BF0B-C7E75ADF7E4E)

完成上述步骤后，我们可以随时点击预览来查看图片效果，确认无误后点击「保存」便能得到需要的图片，通常来说，处理的图片越多，所需要的时间也会随之增加。并且稍显不足的是，PicGuard Pro 并不支持自定义水印的位置，水印只能铺满在整幅图片上，所以无法用作个人摄影作品的水印添加。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/3E10AEA2-4810-4B22-BB35-8C82C5035BD8)

PicGuard Pro 的优点是功能足够单一，不会有过多的冗余设计。如果你想要寻找这样一款专一且好用的应用，不妨通过 F-Droid 下载体验，应用开源且完全免费。

**▍****mSecure：跨平台密码管理工具**

* 平台：iOS / iPadOS / macOS / watchOS / Android / Windows
* 关键词：密码管理、密码填充

@化学心情下2：说到密码管理我们往往都会想起 Apple、Google 在其产品中内置的工具，只不过这些内置的密码管理器往往功能和平台支持方面都不如第三方 app，这也就催生了像 1Password 这样的第三方跨平台密码管理工具，以及今天介绍的这款 mSecure。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/CFBCFA0F-E33D-49EF-B4FA-A7C32B5B042C)

除了支持 Apple 平台，mSecure 还加入了对 Android 以及 Windows 系统的支持。相比其它第三方密码管理器，mSecure 支持多种密码库同步方式，有 iCloud、Dropbox、mSecure Cloud 以及局域网内 Wi-Fi 同步，用户可以根据自身的情况调整同步方案。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/213C7650-E5E9-4279-BF68-A2AE8881FD22)

功能方面，mSecure 提供了分类详实的密码管理和标签设计模版，这些模版涵盖了网站登录、银行卡号、安全标签等多种选择。通过这些模版可以非常轻松地创建各种类型的密码信息数据，同时详实的分类也可以更好地帮助我们检索信息。

对于初次使用 mSecure 的用户而言，软件也提供了导入密码数据的入口，支持导入 CSV 格式以及从 1Password 导出的密码数据，这样你就无需手动重新从头开始创建新数据，做到上手就可以快速使用。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/BB7C0E07-5613-40ED-A274-0283AC1E41FD)

除了基本的密码管理功能之外，mSecure 也提供了诸如安全中心等安全功能，帮助你检索库中存在哪些高风险或者已经泄漏的密码，提醒你及时修改更新。此外 mSecure 也包含了密码生成器，可以根据要求生成完全随机的字符密码，从而在一定程度上减少密码被破解的可能性。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/C68BA8DF-2A5E-4904-B4D2-B2F6B756018D)

除了上面说到的管理功能之外，mSecure 也支持的系统层面的密码填充以及通过浏览器扩展来填充密码，不过使用下来似乎 mSecure 对于 macOS 15.x 的支持上存在问题，系统层面上的密码填充似乎不起效。因此能够自动填充的也只有通过 Chrome 或者 Firefox 的扩展实现，但由于系统局限性，目前也仅支持在 macOS 上使用。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/DA29D1C5-C2A5-4AAA-A747-DD0F210232A8)

**▍****图图记账 4.0**

* 平台：iOS、iPadOS
* 关键词：记账

@ElijahLee：图图记账是主打高颜值、简约设计的记账应用，在如今遍地开花的记账应用里可谓独树一帜。从年初开始，开发者投入了 10 个多月的时间重新打磨图图记账，于近日正式推出了 4.0 版本。

首先，新版重点更新了应用的视觉设计，对应用的每一个页面都进行了重新的设计，视觉效果仍旧继承高颜值简约设计的风格，同时视觉体验更加统一、规范。细节之处更加精致，增加了数字货币等多种资产类型、优化账户图标、加入了对深色模式的支持、增加多个专属图标等等。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/9D8FF537-548F-4854-98E1-EE1B0EBAA48C)

在功能更新方面，4.0 设计了全新的预算功能，支持创建长期预算和临时性预算，前者适用于按月度、年度等固定周期作为预算统计的方式，后者则可用于短途旅游、出差或各种活动等的预算控制。预算的剩余额度、花费以及每日限额都进行了视觉语言的重新设计。在创建预算时，还支持选择账本、关联类别等，丰富的自定义选项可以让预算管理更加精细化。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/242EA0FB-0D63-44D3-ACF7-52A1D1E55F17)

其它升级的功能较为零散，但对于特定需求的用户来说都非常实用。例如对于有对账需求的用户，图图记账可以实时展示每一笔金额变动之后的计算结果。账单搜索功能支持丰富的筛选与展示，包括时间、账本以及自定义排序。收支日历可以直接在日期下方显示当天的收支结余，双击日期还可以直接进入当天的记账。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/9BAA6691-5F58-4B3A-8BE3-53FC1023B15B)

图图记账 4.0 版本还有很多细节功能，这些功能的设计与实现背后不止开发者一个人的努力，还有两千多名用户的反馈与建议，可以说图图记账在吸收用户创意方面更加积极，这也使得应用的各种小功能都越来越实用。你可以在 App Store 下载体验新版的图图记账。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/44E58B3B-D527-4F0E-8D3D-1CF9FFF01379)

@Noah\_Choi：以前的 Your Phone、现在的手机连接 app 是我为数不多要为微软扛大旗的产品，因为它可以看作是当前 Android 和 Windows 之间最完美的桥梁——原生、无感、功能实用。而就在近日，微软对「跨设备体验主机」模块进行了更新，进一步增强了手机连接的使用体验。

**▍****iOS 18.2 语音备忘录：更新分层录音功能**

* 平台：iOS
* 关键词：备忘录、分轨录音

@Snow：伴随上周 iOS 18.2 正式推送，语音备忘录 app 正式为 iPhone 16 Pro 和 iPhone 16 Pro Max 用户带来了分层录音功能。

进入语音备忘录，点击展开任意一个录音文件，点击左侧的编辑录音按钮，或从右上角「…」中进入录音编辑页，你可以在音轨右下角看到一个「+」号。点击「+」号，出现 2 号音轨标识，左右拖动时间轴，选择要混音的起始位置，点击底部带数字 2 的录音按钮，应用会自动播放第一层音轨声音，与此同时你可以开始第二轨的录制。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/52D40F84-01EA-4DDA-A547-583A2078BD1C)

录制完成后，你可以在波形图上清晰地看到音轨叠加的效果。进入左上方的选项，除原有的播放速度、跳过静音等功能外，分层录音素材会增加一条音层混音的滑轨，你可以通过左右滑动调节两条音轨的音量强弱。点击右下角的完成按钮，你可以选择直接覆盖原始录音文件，或作为新副本保存。已保存的分层录音素材，还可以借助「分离音层」功能，将两条音轨拆分保存为两个独立的录音文件。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/5F1A30CC-62C6-47C2-9D51-10E62B8441E8)

Apple 当然不只是简单粗暴地混缩或拆解两条音轨而已，依托 A18 Pro 芯片的性能，语音备忘录可以利用机器学习技术分离人声录音。用户可以将伴奏设为第一层音轨，在录制第二轨时，应用虽然会同时收录人声和伴奏，但可以通过算法剥离出一个较为干净的人声，而无需耳机、话筒等其它外设支持。音乐制作人们还可将器乐混音以压缩音频文件的形式直接发送到语音备忘录，以便在灵感到来时，随时随地轻松录制人声音轨。录制生成的分层录音也可以跨设备同步至 Mac 或 iPad，快速导入 Logic Pro 这一类专业 app 中进一步混音和制作。

三位格莱美奖得主：多张白金专辑歌手 Michael Bublé、乡村乐明星 Carly Pearce 与制作人 Greg Wells 联手用 iPhone 16 Pro 上的语音备忘录录制了新歌《Maybe This Christmas》。在幕后揭秘视频中，几位艺人分享了这首动人节日歌曲的灵感，并详细介绍了如何利用语音备忘录的新功能在 iPhone 16 Pro 上单独录制人声。乐迷可以在 Apple Music 通过空间音频聆听《Maybe This Christmas》。

语音备忘录的分层录音功能适用于运行 iOS 18.2 的 iPhone 16 Pro 和 iPhone 16 Pro Max。共享至语音备忘录与导入分层录音功能适用于 Mac 版 Logic Pro 11.1 和 iPad 版 Logic Pro 2.1，分别需要运行 macOS Sequoia 15.2 和 iPadOS 18.2。

**▍****Drafts 46：加入文字转语音功能，加入「运行后台操作」的新快捷指令**

* 平台：iOS / iPadOS / macOS / watchOS / visionOS
* 关键词：灵感记录、文本收集

@化学心情下2：在我看来，Drafts 早已脱胎于单纯的文本编辑工具，而是通过模块化的脚本插件扩展成为一个全面的生产力工具。而在近期的 v46 版本更新中，开发者也引入了两项非常实用的功能。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/1E770F2A-8F6E-42BE-A8C4-7059A1004749)

首先是文字转语音功能，其实在 iOS 和 macOS 的系统能力中就包含了这项辅助功能，而 Drafts 只是将其添加到了应用中更方便调用，使用上也很简单。在 iOS 版本的编辑界面中，只需要长按左上角的文档图标，在弹出的菜单中选择「Read Selection」，当前编辑器就会弹出语音播放控制器，你可以选择暂停、结束或者继续播放，控制语速以及选择不同的语音音色。这项功能个人感觉比较适合在编辑各种演讲稿或发言稿时使用，通过文字转语音来检验文字表达是否通顺连贯。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/33007862-E1E2-4353-A492-850FE845ABCB)

系统默认的文字转语音只会播放当前文档中的所有文本内容，如果你需要播放单独的内容，则需要安装社区 Action 来实现，比如安装这个 Read Selection 动作后，你可以选择编辑器中某一段文本内容，然后点击右上角的侧栏按钮，在侧栏菜单中选择 Read Selection 播放你选择的那段文本语音，而不是文档中的全部文本。

![](.evernote-content/304144E6-B8CE-4415-8607-5E3099A3E9AD/8C53236E-1959-407F-8A96-2F17C8030DE7)

另一项功能则是加入了名为「运行后台操作」的新快捷指令，根据官方的介绍，这个新的快捷指令可以在运行自动化任务时，无需打开 Drafts 确认即可执行某项操作。在创建新快捷指令时，在 Drafts 中选择「Run Background Action on Draft」和「Run Background Action with Text」即可，动手能力强的你不妨试试利用这两项新的指令来创建一个专属的工作流。

Drafts 46 目前已经正式更新上线，你可以在 App Store 和 Mac App Store 中免费更新下载。
