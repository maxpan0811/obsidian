# 王小东：可以不感恩，但别骂祖国

王小东：可以不感恩，但别骂祖国
===============

**2013-12-02 08:27****[环球时报](http://www.huanqiu.com/)***[4346](http://opinion.huanqiu.com/opinion_china/2013-12/4619143.html#comment)****字号：**T**T***

把个人利益与爱国，与国家利益对立起来，甚至把这种对立绝对化，是最近几十年甚嚣尘上的一种理论。这个理论之所以在中国如此强大，如此肆无忌惮，当然与过去的一段历史有关。在那段历史中，中国社会确实有过在国家利益的名义下，不必要地损害个人利益的失误。

　　我们人类是不完美的，在纠正失误时往往会“矫枉过正”。但再怎么“矫枉过正”，也不应该从一个极端跳到另一个极端，也不应该完全迷失真相。这个真相就是，**个人利益固然也有可能与国家利益发生冲突，但对于绝大多数人，在大多数时候，个人利益与国家利益是一致的，甚至可以说，国家利益是个人利益最根本，最强有力的保障。这是人类几千年以来的制度安排。**这才是最普世的“普世价值”。

　　世界银行的《世界发展报告2006》说得很清楚：“一个人能否过上健康和具有高生产力的生活，最大先决因素是这个人的出生国.”这段话是什么意思？说白了，就是一个具有美国国籍的人，很可能就个人素质而言，在各方面都不如你，但还是比你过得更好，比你消费水平高，比你在国际上的社会地位高。只因为美国比中国富裕、强大。**你觉得美国好，想改换门庭，投奔那边？你可以试试，成功的概率不会太高。君不见，一些中国知识分子，将中国毁骂到了极致，将美国颂扬到了极致，凡是中国与美国发生国家利益冲突，他们必站到美国一边，然而，即使他们中少数的佼佼者，拿到美国签证，到美国待了十天半个月，甚至一两年，又如何呢？很多人还是灰溜溜地跑回祖国。**因为，他们对于美国的唯一价值，恰恰在于他们自己祖国的强大，使得美国感觉到有必要遏制，一旦他们离开自己的祖国，这种价值就会大大贬值，于是美国也就不再给其发放口粮了。

　　你们很幸运，如果你们是伊拉克人，抑或是阿富汗人，抑或是上个世纪三四十年代的中国人，恐怕你们就会感到祖国对于你们的个人利益有多么重要了，也许那样，你们会更爱国一点。我只想对你们说最后一句话：你们今天的幸运，是那些爱国者流血、流泪、流汗给你们挣来的，你们可以不感恩，但别老骂他们，行吗？▲(作者是北京知名学者)
