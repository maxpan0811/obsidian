***1***

**为什么选择用战争来做本期国庆营的主题？**

|  |  |  |
| --- | --- | --- |
|  | 《后汉书·张衡传》提到张衡为什么写《二京赋》呢？因为“时天下承平日久，自王侯以下莫不逾侈。”说天下太平的时间太长，所以人们慢慢地习惯了过于奢侈的生活，而忘记了从前经历过的苦难。这句话用在今天的中国，也是适用的。现在的生活过得越来越好，慢慢地从温饱过渡到小康，有的已经近乎奢侈，但是战争真的离我们很遥远吗？第二次世界大战结束之后，其实战争的硝烟依然没有停止。**战争思维是现在孩子缺失的教育，没有战斗力就没有和平。** |  |

***2***

**我们希望用儿童的方式来看待战争**

|  |  |  |
| --- | --- | --- |
|  | 所谓适合儿童的方式一定是站在儿童的角度，一定是要用儿童的视角去看待战争；以儿童的心灵，用儿童的感受来表达对战争的一切；一定是以儿童的思维方式去解读战争。这样才是一个负责任的理性的态度，才能够更好地培养健全的人。  自从有了人类，战争就一直相伴，可是战争的真正起因是什么？人们是为了争夺土地，还是因为财富、资源？也许还有其他原因，导致人类互相残杀，破坏环境和人类文明……我们期待孩子长大了能够面对这个世界，能够客观理性地看待这个世界，一味的回避是不现实，也是不理性的。**“世界上正在发生的许多事，我们都责无旁贷。”教育孩子的社会责任感，战争思维不可缺失。** |  |

***3***

**战争的课程，我们会学到什么？**

|  |  |  |
| --- | --- | --- |
|  | 从古至今，我们带着孩子从历史的长河走到现在的世界，从古代冷兵器战争跨越到现在信息战争，如果战争史存在着分水岭，目前可以认为有两个大的转折：一个是火药的普遍使用，推动历史从冷兵器时代转为热兵器时代；另一个是核武器的发明，通过核威慑重塑了新的国际关系，缔造了脆弱而又稳固的和平。  **在课程中，孩子们会亲历冷兵器时代，感受古人攻与防的智慧。也会从热兵器时代接近核武器战争，对于未来技术的应用，以环境感知、智能控制、系统协同等技术快速发展将使无人化战争成为未来作战的主要样式。无人作战将成为未来信息化战争的主要作战方式。这些都将是课程中导师带着孩子探讨的方向。**  通过所学，所做，所看，带着孩子展望未来，战争力量是和平时代的前提，所以未来去影响和平的战争又会是怎样的？是像《星球大战》一样，在太空的领域争夺主权？还是像《骇客帝国》一样，在二进制的代码中比拼实力？所以课程环节，对于未来战役的畅想，必不可少。 |  |

***4***

**抛开战争，我们还能学到什么？**

|  |  |  |
| --- | --- | --- |
|  | **"机械工程"是一个多领域融合的词汇。同样在战争中也需要的一个关键领域。**对于孩子，他们拥有对于事物的好奇观察，对于未知领域的思考，对于复杂事物渴望创造的心，通过特定主题去理解与认知机械的运行原理，通过工具进行真实的搭建，在使用过程中进行改进和再创造，这无疑是**激发孩子们创造力的不二法门！**    百造国庆创造者营地，带孩子们参与到讨论与制作当中：操作与使用专业的工具和设备，对材料进行二次加工与修正，并以此深度了解战争机器运动和机械原理。在营地的制作环节中，孩子们会投入于团队合作和相互协作之中，真实地感受并理解责任和同理心的含义。 |  |
