今天高考，谈谈感想。

一晃眼高考结束已经20年了，小镇当年高考内卷程度比现在弱得多，但每次梦回高考都能惊出一身冷汗，工作再难，起码比上高中自由。

相比高考，中考才是人生分界的更大难关，尤其在新世纪初的时候，能上高中的比例比后来低得多。以小镇老家山东为例，2000年高中招生人数刚过40万，2001年接近50万，直到2004年达到巅峰近70万人能上高中。只要能进入重点高中，3年后考上大学反而要容易的多。

当时也不搞素质教育那一套，都是早上6点上到晚上22点，每个月放假1天半，自然没有课外辅导的时间和精力，大家就用在校时间拼，只有尖子生有特权，可以跟学校申请晚上继续学习到23点，寒暑假也可以加入尖子班留校学习。大家都这样，也就不觉得辛苦了。

特别重要的是，当年机会太多了，就算没考上高中、没上大学，通过自身努力也可以过上不错的生活。小镇就认识好几位中专都没毕业，自学进入某些高速发展的行业，水涨船高，有的成了上市公司高管，从工厂生产线打工慢慢自己开工厂创业，更是司空见惯。

但是现在太不一样了。

中考仍然是“鬼门关”，但能够参加高考的人大大增加。

一是2022年《职业教育法》正式实施，明确职业学校学生与同层次普通学校学生享有平等机会，可以直接参加高考。

二是近些年高等教育进一步普及、就业压力大增，虽然有大学文凭未必能找到好工作，但如果没有很多工作机会就没有了，于是复读生也越来越多。

从2022年开始参加高考的人数快速增长。

|  |  |
| --- | --- |
| 2020年 | 1071万人 |
| 2021年 | 1078万人 |
| 2022年 | 1193万人 |
| 2023年 | 1291万人 |
| 2024年 | 1342万人 |

2024年复读生已经超过400万人，占比近三分之一，未来复读生比例也不可能降下来，而每年本科招生也就400到500万人。

这就意味着即使以考上最差的本科作为高考成功标准，超过六成的高考生也必然要做好失败的准备。

更严峻的是，新生儿数量直到2017年才过峰，考虑复读生，得再过几年才能缓和，小镇画了一张表，一目了然：

![](.evernote-content/BBBE2F90-866A-443F-8D56-0AB8CAFF4E8B/CC383045-1541-4954-8AB5-2EC064A305BB.png)

高考的性价比越来越低，很多专业就业难度也越来越大。近几年，黄领（工科）专业成了大热门，粉领（文科）就业越来越难，已经到了必须大改的程度。

国家推职业教育，搞分流，被骂的很惨，一说这事就被很多人指责为什么领导的孩子不带头读职业教育。

但这跟自己又有什么关系呢？

不需要关系，只要花钱，有太多渠道可以拿一个本科文凭，北欧随随便便就能搞到本科、硕士，国内也认。更何况，无论读什么，不还是得想办法就业，领导的孩子就算读职业教育，照样不愁。

而对于超过六成考不上本科的，不搞职业教育怎么办？

终究要面对现实。

高考仍然是实现人生跨越最简单、最确定性的道路，全力以赴，考的越高，未来可能性越大。

但高考、大学仅仅是人生第一道关，高考失败了，就不活了吗？高考考上清华北大，就能一辈子躺赢吗？

擅长考试的人终究是少数，考试是竞争性的，多数人在考试这条道路上注定要失败，最重要的是想清楚自己擅长什么，找到考试以外的人生道路。

最重要的是18岁之后的十几年，等到了30岁，学历、从哪里毕业的，都不过是闲聊的话题罢了，对工作和人生的影响越来越小。

因此，成年之后的十几年，直接面对最终用户，找到并满足社会需求，去生产价值，更契合自己的兴趣和特长，这甚至比考上大学更加重要。

用一句话结尾吧：

人生不是既定的铁轨，而是旷野；可以沿着铁轨继续前进，也可以勇敢的离开铁轨，在旷野中走出属于自己的路。
