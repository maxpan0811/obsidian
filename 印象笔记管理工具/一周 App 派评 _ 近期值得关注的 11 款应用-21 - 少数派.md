### Procreate 5：增加动画制作功能，支持悬浮调色盘

iPad 上的专业绘画工具 Procreate 发布了 5.0 大版本更新，本次更新的主要功能是动画制作功能，同时还对调色盘、笔刷等工具进行了增强。

借助全新的动画制作功能，你可以通过 Procreate 直接创作逐帧动画。应用提供了洋葱切片来实现「透写台」的效果，你可以透过洋葱切片查看前后数帧，从而绘出更加自然流畅的动画。

![](.evernote-content/2B4FE699-B3D1-4631-92DA-93997FBA33C2/9CBD972C-B97D-4D05-8AD1-FEA498CA2635.jpg)

除了动画制作外，Procreate 5 还加入了悬浮调色盘，你再也不用反复展开调色界面来更改颜色了。调用悬浮调色盘的操作也非常方便，只要拖动调色界面即可。Procreate 5 还优化了笔刷的创作过程，即使你没有相关经验，也可以在简单的摸索中创造一个全新的笔刷。你也可以将两个现有的笔刷进行组合，实现 1+1>2 的效果。

![](.evernote-content/2B4FE699-B3D1-4631-92DA-93997FBA33C2/2F2FB492-24CF-42A6-B32A-BA4E00F9A8BE.jpg)

除了这些肉眼可见的大改动外，Procreate 5 还更新了绘图引擎。全新的 Valkyrie 引擎基于 Metal 架构开发，能够在 iPad Pro 上以 120fps 的速度运行。得益于全新引擎，Procreate 5 在着色方面也有一定的改进，当你使用 Apple Pencil 进行绘画时，应用会通过压感与倾斜角度实时传输色彩信息。

你可以在 [App Store](https://apps.apple.com/cn/app/procreate/id425073498) 购买 Procreate 5，售价  68 元，老用户可以免费升级新版本。

### 滴答清单：支持 Markdown

上周，任务管理工具滴答清单迎来 5.4 版本更新，新版本首次增加了对 Markdown 的支持，现在你可以利用 Markdown 语言编辑文本样式、调整组织结构，甚至是在清单中撰写长文。

进入任务详情页，再点击编辑任务描述，你就可以看到 Markdown 功能了。与众多 Markdown 编辑器类似，滴答清单在输入法上方提供了一行快捷工具，标题、加粗、斜体、下划线、高亮、链接等常用功能都在此陈列，方便你随时调用。当然，如果你对语法足够熟悉，也可以直接在编辑器中直接撰写，应用也会实时展现。

不过比较可惜的是，任务描述的 Markdown 功能与子任务并不能共存，一旦你开启子任务，Markdown 语法就会自动失效。所以，新增的 Markdown 更像是为滴答清单增加了基础的笔记支持，一些简单的文本编辑工作可以直接在滴答清单中完成。

![](.evernote-content/2B4FE699-B3D1-4631-92DA-93997FBA33C2/99D693B3-3DC0-40EF-982C-6ABB957A8EF8.jpg)

你可以在 [App Store](https://itunes.apple.com/cn/app/ticktick/id626144601) 下载滴答清单。

### Buttercup Password Manager：开源的全平台密码管理器

Buttercup 是一款开源、支持全平台的密码管理器，它没有提供密码存储服务，你需要将它连接至自己的云存储服务来使用它。毕竟对于密码这一类隐私信息来说，只有将它抓在自己手里才是最有安全感的。应用支持 Google Drive、Dropbox 等云存储服务，你也可以将其连接至自己的 WebDAV。

不论是用自建云存储还是网盘，Buttercup 都会将数据通过 AES-256 算法进行加密保存，只有通过 Buttercup 才能解锁并读取里面的内容。在最近的更新中，Buttercup 还加入了对一次性密码的支持，你可以直接通过 Buttercup 获取动态的一次性密码。

![](.evernote-content/2B4FE699-B3D1-4631-92DA-93997FBA33C2/9CF9E32A-AA52-4D9C-98E6-CFD917F36A98.jpg)

在使用体验上，由于 Buttercup 没有操作指引，对新用户不太友好。应用内的几乎所有操作都可以通过屏幕右上角的按钮来实现，如果需要大量添加账户，会显得比较麻烦。不过 Buttercup 在账户设置上给予了用户非常高的自由度，除了账户、密码外，你还可以为其添加各种自定义的信息。

虽然 Buttercup 支持生物识别，但需要用户手动开启。在云存储服务设置界面点击右上角的按钮即可开关生物识别或自动填充。如果你连接了多个云存储服务账户，可以对它们分别进行设置。

你可以在 [App Store](https://apps.apple.com/cn/app/buttercup-password-manager/id1294001514)、[Google Play](https://play.google.com/store/apps/details?id=com.buttercup&hl=en) 或 [官网](https://buttercup.pw/) 下载 Buttercup。

### Anywhere-：多方式调用 Activity 活动的收藏夹

某些时候我们只需要快速使用某个 App 的一个功能（例如签到、扫码），但仍然需要一层一层进入 App。实际上，Android 平台的 App 通过不同「Activity」来展示和实现各种功能，并使用 URL Scheme 进行跳转，理论上我们也可以通过该种方式直接进入某一页面。聪明的你应该能想到，iOS 捷径、定制 ROM 的负一屏、Shortcut 等功能都使用了这种方式为用户提供快速入口。Anywhere- 也是这样一款应用，但它的功能更加强大和自由。

Anywhere- 可以向 Root 用户提供最全面的 Activity 调用，同时面向非 Root 用户提供了 URL Scheme 或 ADB 命令调用两种方式（后者需要 Shizuku 支持）。针对 URL Scheme 方式，我们除了可以直接在活动列表中获得，也可以前往捷径社区查看某 App 的特定 URL Scheme，别担心，虽然捷径社区是面向 iOS 用户的，但绝大部分 URL Scheme 在两个平台上相同，我们直接复制到 Anywhere- 即可实现对应功能。

![](.evernote-content/2B4FE699-B3D1-4631-92DA-93997FBA33C2/CF3F4D63-9963-4BD2-8F9E-B5552BDA585D.png)

根据开发者描述，Anywhere- 最初是想要做一个常用 Activity 的收藏夹，因此获取了 URL Scheme 和 Activity 后，App 主界面会逐项显示用户自定义的项目，这些项目可以添加至 Anywhere- 的 Shortcut 内或直接创建桌面小部件。除此之外，Anywhere- 直接提供了微信和支付宝的扫码合集。

![](.evernote-content/2B4FE699-B3D1-4631-92DA-93997FBA33C2/9D030E6C-CA5D-4588-9329-490C69323AF0.png)

Anywhere- 遵循 Material Design，在设置中允许用户自定义背景图、主题和图标包，它作为一个工具 App 不仅实用，也很养眼。

你可以在 [酷安社区](https://www.coolapk.com/apk/com.absinthe.anywhere_) 下载 Anywhere-，应用免费无内购。

### Chat Journal：随时随地进行对话式日记

学生时期，每晚回到家中拿出藏在抽屉里的日记本记录下一天的大事小情，这一场景俨然成为了一种生活仪式。

而随着智能设备的普及，「记日记」这件事情自然而然地也随之被转移到了线上。Day One、格志日记等应用迅速在这一领域攻城略地，成为许多人记录生活的选择。不过，要在一天结束之时，找到一块完整的时间，找回事件发生时那种独特的心境似乎并非易事。于是，微博、推特这一类社交平台反倒成为了我们在闲暇时刻抒发情感的首选之处。

Chat Journal 通过对话式日记的设计思路，帮助用户将记日记这件事进行了多维度的分割。通过 Chat Journal，我们可以将生活细分至衣食住行等多个维度，一旦有了记录欲，便可以打开任一维度，在对话框里输入想要记录的心情、事件或是照片，无需纠结排版，无需刻意审视文字的流畅度。如此一来，生活当中的无数个细碎时刻都能够被一五一十地还原出来。

![](.evernote-content/2B4FE699-B3D1-4631-92DA-93997FBA33C2/2864BAE3-B6E2-4842-9860-7258CB64A5A4.jpg)

假如打开 Chat Journal 后发现自己想不起来要书写什么，我们还可以让机器人随机问我们一个问题，作答的过程也是对自身经历的一种回忆。

![](.evernote-content/2B4FE699-B3D1-4631-92DA-93997FBA33C2/2AD200A8-152B-44C5-A7A5-BD7C4A83CD5B.png)

当然，作为一款日记类应用，Chat Journal 还提供了详细的数据分析页面，详细地呈现用户记录日记的数量，以及各类标签的使用频率，从而达到帮助用户自我了解的目的。此外，Chat Journal 提供的每日心情记录也是一个不错的自我觉察的途径，通过快速点选心情图标或是回答每日三问（可自行定制问题）便能了解自己的心情变化。

![](.evernote-content/2B4FE699-B3D1-4631-92DA-93997FBA33C2/E93C64E2-BB30-4DC4-A707-788189E02F41.jpg)

凡事都有两面性，Chat Journal 在帮助我们随时随地进行快速记录的同时，也带来了碎片化的问题，希望开发者在后续的开发过程中能够给出一种更具条理性的整理方式，并支持数据的导出。目前，Chat Journal 支持谷歌网盘与本地备份，其他云端备份方式尚在开发中。

![](.evernote-content/2B4FE699-B3D1-4631-92DA-93997FBA33C2/FA1BF72F-8794-4BC4-8C23-9E59D080A660.jpg)

你可以在 [Play Store](https://play.google.com/store/apps/details?id=com.agiletelescope.chatjournal) 免费下载 Chat Journal。

### 小码短链接：支持数据统计的短链接服务

很多平台都有自己的短链接服务，例如微博的 [t.cn](http://t.cn/) 和百度的 [dwz.cn](http://dwz.cn/) 等，短链接相对于原网址更加便于分享和传播。不过，大多数面向于 C 端用户的短链接服务，往往只提供了最基础的缩短网址功能。

小码短链接不仅拥有更简洁漂亮的外观，区别于国内大多数短链接服务，它还提供了针对链接的访问数据统计功能。

![](.evernote-content/2B4FE699-B3D1-4631-92DA-93997FBA33C2/A90486FE-4047-4FA9-B507-E2ABB447BB1E.png)

小码短链接特别适合于线下推广活动、短信营销等场景，你可以通过美观的报表来分析短链接的传播度，从而进一步调整后续的营销方案。除此之外，小码短链接还提供了像批量生成、快速生成二维码、以及修改跳转链接等实用功能。

目前，小马短链接服务完全免费，你可以直接 [访问官网](https://xiaomark.com/) 进行体验。

### 2019 Wrapped by Spotify：盘点你的 Spotify 年度收听数据

到了年底，很多 App 与服务都会给出这一年用户的使用情况，之前 Apple Music 就为用户推出了年度报告；Spotify 作为目前最大音频流媒体平台之一，也像往年一样在上周推出了年终盘点页面：[Spotify 2019 Wrapped](https://www.spotify.com/wrapped)。

![](.evernote-content/2B4FE699-B3D1-4631-92DA-93997FBA33C2/AAAFB6CD-A1B1-4929-A727-23E0B9E1EAC3.png)

今年是 Spotify 上线的第 13 个年头，同时也是 201x 的最后一年，盘点是基于最近的十年，因此你可以从盘点中看到近十年来你的收听数据。首先你将会看到你这一年来不同季节下的音乐收听情况、以及自己的年度最爱音乐人、自己今年收听了哪些音乐人、他们来自哪些国家和地区等等。

![](.evernote-content/2B4FE699-B3D1-4631-92DA-93997FBA33C2/3115C016-D875-4189-BFEC-086DEFD15C14.png)

之后则是你偏爱的音乐类型排行榜、你最喜欢的音乐单曲，以及从收听 Spotify 开始、这十年你每一年的收听时长、以及每一年你最喜欢的歌曲和最喜欢的艺人有哪些。最后你可以将 Spotify 生成的数据分享到你的社交媒体。

![](.evernote-content/2B4FE699-B3D1-4631-92DA-93997FBA33C2/B8FAD567-2235-4C80-9504-955C31F700B9.png)

除了盘点之外，在 Spotify 客户端中你还可以看到生成的播放清单，当然除了你自己的之外还有一系列的综合盘点，例如今年全球用户最喜欢的音乐人、最喜欢的组合等。当然我们还可以看到近十年来的大盘点，或许通过这些音乐榜单你可以对自己这十年来的音乐品味变化也能略知一二。
