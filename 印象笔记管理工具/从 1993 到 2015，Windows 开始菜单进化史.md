# 从 1993 到 2015，Windows 开始菜单进化史

---

[![](.evernote-content/1A9BD2B2-0535-4570-A20A-8A493895790F/4FE12A1B-7A36-4AD8-BCF2-A213997C6DAC.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/human-evolution-Windows-10.jpg)

关于微软最新的 Windows 10 系统，几乎每篇文章必谈的一个话题就是开始菜单的回归。而 Windows 8 的不尽如人意和这个已经存在了 20 余年的开始菜单有着莫大的关系。从某种程度上而言，它的失败可以归咎于开始菜单的缺失，而微软寄予厚望的开始屏幕始终没能取代那一个小小的按钮。

自第一次出现，开始菜单经历了多次改动，但有一些元素却被一直传承和保留了下来，从最久远的 Windows 95，到最新的 Windows 10。 [Ars Technica](http://arstechnica.com/gadgets/2015/07/the-windows-start-menu-saga-from-1993-to-today/) 在虚拟机上回溯了开始菜单这一路的演变历程，让我们有机会重新审视 Windows 最为重要的特性之一。

[![](.evernote-content/1A9BD2B2-0535-4570-A20A-8A493895790F/3C133926-0045-43EE-AB65-52711FDE97F4.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/Windows-Start-Menue-Windows-Chicago.jpg)

### Windows Chicago

Chicago 是一个衍生版的系统，后来衍生出了经典的 Windows 95。在 build 58s 这一个版本中，开始菜单、任务栏、我的电脑的雏形已经开始显现。与后来的开始菜单最大的区别是，当时的开始菜单由三个按钮组成，带有 Windows logo 的按钮负责窗口和应用管理，中间的按钮是一个应用启动器，最右侧的则是帮助和文件检索。

[![](.evernote-content/1A9BD2B2-0535-4570-A20A-8A493895790F/9B2A1A95-4758-4E14-8EDE-5F6A48420DBE.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/Windows-Start-Menue-Windows-95.jpg)

### Windows 95

对于很多人而言，Windows 95 是他们第一次接触到开始菜单，而它在实现比尔盖茨“每个家庭每张桌子都有一台电脑”的愿景的过程中，也扮演着极其重要的角色。

在 Windows 95 上，后来成为经典的开始菜单即便在今天看来并不过时。居于左下角的它，是一个触手可及的应用启动器，也可以方便地查看最近的文件、系统设置、文件检索。在使用中，一个比较有意思的现象是安装的应用程序会被归入开始菜单中的“程序”一栏，它是一个树状结构，可以继续扩展下去。这一特点一直延续到了 Windows 7。

[![](.evernote-content/1A9BD2B2-0535-4570-A20A-8A493895790F/3E415199-7936-493A-AC40-35B497DBE3AA.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/Windows-Start-Menue-Windows-NT-4.0.jpg)

### Windows NT 4.0

Windows NT 是面向企业用户的 Windows 系统版本，更佳稳定。1996 年发布的

Windows NT 4.0 借鉴了 Windows 95 的用户界面，同时也做了其他一些更新。整体而言，NT 上的开始菜单和 Windows 95 上非常相似，最主要的差异是，在默认设置下，“应用程序”被分成了“当前用户安装的应用”和“所有用户安装的应用”两个部分。

[![](.evernote-content/1A9BD2B2-0535-4570-A20A-8A493895790F/322C2BC4-CA71-4864-A83A-4629287BAABF.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/Windows-Start-Menue-Windows-98.jpg)

### Windows 98

Windows 98 最大的变化是系统集成了 IE 浏览器，后来给微软带来了监管上的麻烦，这些都是后话了。

在开始菜单上，一些 Windows 95 上可选的功能在 Windows 98 上变成了默认设置，比如 Windows Update。另外，新增的“收藏”文件夹直接链接到相应的网站，常驻底部的“登出”按钮则宣示着 Windows 98 支持多用户登陆的特性，但实际上你只有在关闭所有窗口和所有应用之后才可以退出登录。这一“缺陷”直到 Windows XP 加入“用户切换”之后才得以改变。

另外的改变包括了支持右键单击，可以用鼠标在开始菜单中拖动应用图标。

[![](.evernote-content/1A9BD2B2-0535-4570-A20A-8A493895790F/6AAD594B-AE13-4FEB-8AC8-150AD4DEC787.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/Windows-Start-Menue-Windows-98-SE.jpg)

### Windows 98 SE

这是微软在 1999 年发布的一个付费升级版本，几乎代表着 Windows 9x 系列的巅峰水准，在稳定性和漏洞控制上有着不错的评价。

相比较上一个版本，Windows 98 SE（Second Edition）在开始菜单上并没有多少改变，主要是在右键单击时增加了“按名称排列”的选项，方便对开始菜单进行整理。

[![](.evernote-content/1A9BD2B2-0535-4570-A20A-8A493895790F/2D7BE98D-9AE3-49F0-A5F7-6FFD5D73533C.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/Windows-Start-Menue-Windows-2000.jpg)

### Windows ME/2000

Windows ME（Millennium Edition）可以看作是 Windows 2000 的低端兼容版本，属于 32 位 16 位混合编程，包含了 Windows 2000 的部分新特性，对于硬件配置的要求更低，类似于微软在 Windows Phone 8 发布之后针对老用户推出的 Windows 7.8。它在稳定性上并不好。而 Windows 2000 则属于 32 位系统，同时也是 Windows NT 5.0。微软希望用它来同时取代 Windows 98 和 Windows NT 4.0，虽然这一美好愿望到了后来的 Windows XP 上才真正实现。

不过，两者在开始菜单上几乎完全一致，功能和选项设置相差无几。不同之处在于 Windows 2000 中，默认设置下并没有“登出”选项。

[![](.evernote-content/1A9BD2B2-0535-4570-A20A-8A493895790F/A40DA599-4CD2-4C44-AAEC-E2EA9B6AF324.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/Windows-Start-Menue-Windows-XP.jpg)

### Windows XP

Windows XP 可以说是微软历史上寿命最长的操作系统，其中 XP 对应着英文“Experience”，意为“体验”。从 2001 年 10 月份正式发布至今已经过去近 14 年的时间了，微软也在 2014 年 4 月 8 日停止对 XP 用户提供服务。即便如此，Windows XP 的[市场占有率仍旧高达 11.98%](http://www.netmarketshare.com/operating-system-market-share.aspx?qprid=10&qpcustomd=0)，远超 Windows 8 的 2.9%，低于 Windows 8.1 的 13.12%。

严格来讲，Windows XP 的开始菜单并没有新颖的功能和特性出现，最主要的变化是你在开始菜单可以看到更多的内容了，从原来的一列变成了两列，固有的功能如程序启动器、设置等等位于左侧，右侧增加了一栏，可以查看最近的应用以及把一些程序固定在某一个位置。

相形之下，Windows 95 时代的开始菜单变得黯然失色，取而代之的是 Windows XP 上更加多彩更加个性化。不过，如果你真的怀念原来的开始菜单，可以通过“经典”主题找回。

[![](.evernote-content/1A9BD2B2-0535-4570-A20A-8A493895790F/AA920ED2-4EF3-4656-9602-005F6B19ED29.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/Windows-Start-Menue-Windows-Vista.jpg)

### Windows Vista

在 Vista 系统中，微软把搜索功能整合进了开始菜单，省去了文件和应用查找上的麻烦，同时还保留了原有的开始菜单的功能。最大的改进是，在之前的 Windows 版本中，所有的检索是单独进行的，耗费的时间很长，在 Vista 中，会对硬盘上的文件建立索引，这样在检索的时候几乎可以立即呈现，这一点和 OS X  10.4 中的 Spotlight 相近。

此外，Windows Vista 的开始菜单在外观上的一个变化是上面没有了“Start”字样。在其他方面，Vista 的开始菜单和 Windows XP 中的两栏设计类似。“应用程序”菜单无限延展的树状结构被替换掉了，改成了一次只展示一栏，通过一个返回按钮回到上一级。

[![](.evernote-content/1A9BD2B2-0535-4570-A20A-8A493895790F/8F91129D-BF97-40DE-90E7-778D79C5F60B.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/Windows-Start-Menue-Windows-7.jpg)

### Windows 7

相比于上一个版本，Windows 7 在开始菜单上并没有多少变化，主要是缓冲了一下 Windows Vista 上过于突兀的改变，更加容易让用户接受，算是一种补救和完善。

[![](.evernote-content/1A9BD2B2-0535-4570-A20A-8A493895790F/51E23560-E765-4FB8-AF85-C3FCF7EBF21F.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/Windows-Start-Menue-Windows-8.jpg)

### Windows 8

微软在 Windows 8 中移除了这个存在了 20 余年的按钮，结果可想而知。严格来说，微软仅仅是移除了这个按钮，然后用一个开始屏幕取代了它，为的是更好地适应触摸屏操作。可是用户并不买账，所以就造成了普及率徘徊不前。

从某种程度上来讲，并不是“开始屏幕”本身的问题，应用瓷贴足够现代化，可以呈现应用的概要信息，等等。主要的问题是它与以往的开始菜单有着太过巨大的鸿沟，在传统界面和 Modern 界面切换时的逻辑不够清晰，在使用体验上存在割裂，而微软也没有做好教育用户的工作。

[![](.evernote-content/1A9BD2B2-0535-4570-A20A-8A493895790F/02304E4E-8394-41E1-B7D1-2B14120FC51B.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/Windows-Start-Menue-Windows-8.1.jpg)

### Windows 8.1

显而易见，这个系统是为了应对用户对 Windows 8 的批评而发布的补救措施，左下角添加了一个 Windows 窗口图标，右键单击可以呼出久违的开始菜单。使用起来不够直观。

不过，在 Windows 8.1 上，开始菜单并没有真正回归，而是承担起了 Modern 页面（开始屏幕）和传统桌面之间切换的按钮。

[![](.evernote-content/1A9BD2B2-0535-4570-A20A-8A493895790F/3AFEA55D-197D-4F94-84F4-717A551D3DFB.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/Windows-Start-Menue-Windows-10.jpg)

### Windows 10

万众期待的 Windows 10 最让人欣喜的是开始菜单的回归，它综合了 Windows 8 上多彩、尺寸多变的瓷贴设计，和 Windows 7 上逻辑清晰的开始按钮。

新的开始菜单重新启用了两栏的排布方式，左侧一栏呈现的是最近的应用程序和文件，另外还有一些传统的元素，比如应用程序、关机、登出和设置。第二栏在固定更多瓷贴后可以进一步衍生出更多子栏目。

更为重要的是，Windows 10 的开始菜单集成了微软的语音助手 Cortana，既可以搜索本机文件完成原有的检索功能，也可以实现更加智能化的语音识别和搜索功能。

Windows 10 中开始菜单的回归来得更加彻底，既保证了原有用户的使用习惯问题，也把更现代化的交互和操作带到了新系统之中。当然关于 Windows 10 的消息远不止这些，更多信息可以查看王飞同学的总结文章。
