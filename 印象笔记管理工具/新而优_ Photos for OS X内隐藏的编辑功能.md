# 新而优: Photos for OS X内隐藏的编辑功能

---

[![](.evernote-content/2A6BAAB1-3641-425E-A486-5A8E237AE557/C30B65B1-88DD-4BC4-9ACB-FB3E3CDA68C2.png)](http://www.feng.com/iPhone/news/2015-08-13/New-and-superior-Photos-for-OS-X-hidden-within-the-editing-functions_621500.shtml)

![](.evernote-content/2A6BAAB1-3641-425E-A486-5A8E237AE557/C30B65B1-88DD-4BC4-9ACB-FB3E3CDA68C2.png)

苹果想要通过 Photos for OS X 来吸引更多用户，这款应用的编辑工具非常简单，任谁都能够轻易上手，当然这款应用还有其他方面的魅力，即使它还是一款 1.0 版本产品，但是在其友好的用户界面背后隐藏着巨大的编辑能力。

**开始编辑**

要进入编辑视图，用户通常可以在查看照片的界面点击编辑按键，当然你还有一种更快的方式，直接按下 Return 即可进入编辑模式。在时刻视图下选择一张照片之后按下 Return 也可进入编辑。

另外即使用户不在编辑模式下，只要按下快捷键还是可以对照片进行编辑的，按 C 打开裁剪工具，F 是滤镜，A 是调整，R 是修整，E 是去红眼工具，M 是对比。用户编辑时按下箭头按键可以在不离开编辑视图的情况下切换到上一张或者下一张图片。

**选择哪个版本来编辑**

如果你的相机拍出来的相片有 Raw+JPEG 格式，Photos 应用会将这两种格式照片分类，默认选择 JPEG 版本来进行编辑。如果你不想使用这个版本来编辑，在编辑视图下选择图像，然后选择使用 RAW 作为原始编辑对象。

**添加更多调整**

![](.evernote-content/2A6BAAB1-3641-425E-A486-5A8E237AE557/21566D11-0002-465C-8CFD-19DB17F097DA.png)

调整工具内都是常见的颜色和黑白调整选择，用户可以点击添加菜单来增加更多选择。如果你常常使用某些工具，比如白平衡、保存为默认等，你都可以在添加菜单中找到。用户每次编辑图片这些选项都会出现。

**扩展编辑范围**

![](.evernote-content/2A6BAAB1-3641-425E-A486-5A8E237AE557/E1F8DFA7-6C7A-4344-98D6-665A67986633.png)

在图片编辑视图下按下Option即可调出很多控制键。这些控制键的范围值都在 –1.00 到 +1.00之间，照片默认是 0。比如曝光控制，当范围值被调到 –1.00 时图片就会很暗，但不是全黑。如果你长按 Option，范围值就更换为 –2.00 到 +2.00，你就可以进一步调整照片。

上图就是通过控制范围值和 Option 键调整出来的不同效果。

**色阶**

直方图虽然能够很好地显示一张图的颜色和色调值，但它仅仅是参考，色阶调整能让用户以更好的方式来控制图片。色阶选项同样是在调整工具的增加菜单中。

**复制和粘贴调整**

在你根据需要编辑了一张相片之后，你其他类似的相片也会一起被调整和编辑，这就不需要你手动进行重复编辑。

在编辑视图下，选择图像>复制调整，然后切换到未编辑的图像视图下，选择图像>粘贴调整即可。

以上就是 Photos for OS X 1.0.1 版本中一些先进的或者隐藏的编辑功能，未来更新还会增加更多新特性。在 OS X El Capitan 版本中，第三方开发者可以开发模块，支持用户在 Photos for OS X 内编辑图片的同时使用开发室提供的工具。目前 iOS 上已经有了这个功能。目前 Photos for OS X 已经证明了自己是一款不错的图片编辑工具，这是一个良好的开始，以后将有更大的发展潜力。
