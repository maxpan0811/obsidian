# 她被日寇剥皮示众3昼夜，活取心脏时突然睁眼，刽子手当场吓疯！这段历史不该被埋没！

1931年的秋天，东北那片黑土地上罩着层愁云。“九一八”事变像把尖刀似的，一下子把这地界的安稳日子戳破了。吉林海龙县五块石村有个庄稼人叫赵玉仙，那天她站在自家田埂上，眼睁睁看着日本兵的铁蹄踩过祖辈种了一辈子的地。那年她34岁，本来过着日出扛锄头、日落烧灶火的农家日子，可打那天起，平静的光景全没了。日本兵到处烧房子、抢东西，村里的男人被拉去当劳工，妇女小孩整天提心吊胆。赵玉仙心里跟扎了冰锥似的，她攥紧拳头，指甲都嵌进掌心肉里，暗下决心：不能就这么看着家乡让人糟践。

![](.evernote-content/D9E2ACD1-683E-4D12-BA9A-3AE330A4C3A7/053AB7AC-6F49-473C-85EB-4918DD87EAE3.jpg)

1933年，赵玉仙听说附近在偷偷组织妇女反日会，专门帮着跟日本人打仗的游击队。她没多琢磨，揣上俩凉窝头就往联络点跑。那时候参加这种组织，脑袋随时可能搬家，可她半点儿不怕。进了反日会，她成了最肯卖力的一个。白天，她还跟往常一样下地干活、照顾家里人；一到晚上，就悄悄叫上村里的妇女们，围着油灯缝衣裳、纳鞋底。油灯昏昏的光里，几十双满是老茧的手飞针走线，把对日本兵的恨、对子弟兵的疼，都缝进了每一针每一线里。有伤员被送到村里藏着治伤，赵玉仙总是头一个凑上去搭把手。她跟着学洗伤口、换草药包扎，看见战士们疼得皱眉头，她心里跟刀剜似的疼。游击队缺药缺得厉害，她把家里仅有的一点积蓄全拿出来，又把自己陪嫁的首饰、家里值点钱的东西都卖了，换成药品偷偷往前线送。后来，她还跟几个妇女一起，在山坳里找了个废弃的山洞，收拾干净当起了地下小医院，用老辈传的土方子给伤员治伤。

![](.evernote-content/D9E2ACD1-683E-4D12-BA9A-3AE330A4C3A7/2E318567-3539-4630-90DC-24508EB6C2F3.jpg)

1934年，因为赵玉仙做事细心又胆大，被推荐去当杨靖宇领导的东北抗日游击队的地下交通员。组织上给了她一匹青鬃花马，这马跑得快，还通人性。打那以后，她就骑着这匹花马，在茫茫的林海雪地里钻来钻去，送情报、买物资。东北的冬天来得早，大雪封山的时候，气温能冻到零下三四十度，哈口气都能结成冰碴子。狂风卷着雪片，刮在脸上跟刀子割似的，三步开外就看不清东西，人和马都快冻僵了。可赵玉仙从没喊过苦，她把情报藏在马鬃里，要么塞在鞋底夹层，有时候还盘在发髻里，回回都能躲开日本兵的搜查。她骑马速度快，行踪又灵便，跟山林里的梅花鹿似的，战友们都亲切地叫她“梅花鹿”。有一回，她带着重要情报赶路，遇上了日本兵的巡逻队，她赶紧策马钻进密林，凭着自己熟地形的本事，在树桩和灌木丛里绕来绕去，总算甩掉了敌人，按时把情报送到了地方。

![](.evernote-content/D9E2ACD1-683E-4D12-BA9A-3AE330A4C3A7/762F96A7-4086-4B1F-84B5-ED27C22FDBB8.jpg)

1936年秋天，赵玉仙已经怀了七个月的娃，走路都费劲。这工夫她接到个急活儿：必须赶紧把日本兵要大规模围剿的情报送到游击队总部。那天偏赶上“大烟炮儿”——就是东北冬天能刮死人的暴风雪，狂风卷着雪片，睁不开眼。雪深得没过膝盖，马走起来都直打趔趄。赵玉仙把棉袄裹得紧紧的，用带子把自己和马肚子捆在一起，怕从马上摔下来。一路上，好几回她都差点被风刮下悬崖，好几回马陷进雪窝里，她就下来用手刨雪，冻得手指又红又肿，几乎没了知觉。肚子里的娃好像也知道妈妈不容易，时不时动一下。她摸了摸肚子，轻声说：“娃啊，再熬会儿，等把信送到了，咱就安全了。”就这么着，她在风雪里走了整整一天一夜，总算把情报送到了。游击队照着这个情报，及时转移了地方，躲开了日本兵的围剿，保住了队伍的力量。

![](.evernote-content/D9E2ACD1-683E-4D12-BA9A-3AE330A4C3A7/B1BE1907-A6CE-4204-B640-C9BA15EA82B8.jpg)

同年11月27日，正赶赵玉仙要再去执行任务的时候，因为出了叛徒，日本兵突然把她的家围了。她想把藏在家里的情报烧了，可已经来不及了。几个日本兵呼啦啦冲进来，揪着她的胳膊就把人拖走了。在监狱里，日本兵知道赵玉仙是交通员，肯定知道不少事儿，就对她动酷刑。用皮鞭抽她，拿竹签扎她的手指，还把烧红的烙铁往她身上按。赵玉仙浑身是伤，血糊糊的，可她牙咬得紧紧的，半个字都没吐。日本兵见硬的不行，又来软的，说只要她说出游击队的下落，就放了她和肚子里的娃。赵玉仙瞪着这些侵略者，骂他们是禽兽不如的东西。气急败坏的日本兵把她的衣服扒光，钉在十字架上，拉到村里示众，想吓唬老百姓。面对围着的乡亲，赵玉仙忍着疼，大声喊：“乡亲们，别怕日本兵！他们早晚得被咱赶出去！”日本兵见她还不服软，残忍地划开了她的孕肚，想逼着她开口，可她直到昏过去，也没吐露一个字。

![](.evernote-content/D9E2ACD1-683E-4D12-BA9A-3AE330A4C3A7/B5EA8C12-01DF-4338-A282-40F13CE8E86F.jpg)

到了1937年1月，有天冷得邪乎。赵玉仙和其他四十多个抗日志士被日本兵押到了靠山屯的刑场。她身子虚得厉害，身上没块好地方，可还是挺着腰杆。日本兵用刺刀一个一个捅这些英雄，血把雪地都染红了。赵玉仙倒下的时候，眼睛还望着远方，好像看见了胜利的样子。她的头被敌人割下来，挂在城门上示众，敌人想靠这个打垮老百姓抗日的心思，可他们想错了。赵玉仙和其他烈士的死，反倒让更多人下定决心反抗。

![](.evernote-content/D9E2ACD1-683E-4D12-BA9A-3AE330A4C3A7/ADC5ACCD-0A29-4861-B79F-5ECE8D7E4D21.jpg)

赵玉仙的事儿，跟好多抗战时牺牲的无名英雄一样，就这么埋在历史的灰堆里了。直到1993年，当地修地方志的时候，才把她的故事写了进去，让更多人知道了这位抗日女英雄的事儿。她没留下一张照片，也没说过啥惊天动地的大话，可她用自己的命，活出了啥叫爱国、啥叫勇敢。在那个黑沉沉的年代，正是因为有千千万万像赵玉仙这样的普通人，用自己的血肉之躯挡在侵略者前面，咱们才能把日本兵赶出去，迎来胜利的日子。东北的林海雪原还记得那匹青鬃马上的身影，黑土地永远记着这位女英雄流的血。她的精神就像雪地里的红梅，在最冷的日子里开得最艳，一辈子都提醒着后人，现在的太平日子来得多不容易，得记住那些为了国家、为了老百姓献出生命的英雄。历史不会忘，老百姓不会忘，那些为了民族能站起来而牺牲的人，他们的名字会永远写在中华民族的史册上。
