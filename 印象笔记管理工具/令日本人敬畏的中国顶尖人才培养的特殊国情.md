# 令日本人敬畏的中国顶尖人才培养的特殊国情

---

令日本人敬畏的中国顶尖人才培养的特殊国情
--------------------

*2017-02-19*[日中资本市场](http://mp.weixin.qq.com/s?__biz=MzI0NjIzMjUzNA==&mid=2653386076&idx=1&sn=08c2ab779fc5b2949de60304bbe1b79e&chksm=f291b1e5c5e638f34d9c3d0d5030b8ff4827a531d1dc0de116df20098ffbd5fe5e80c5ff3430&mpshare=1&scene=1&srcid=0219KzvhqDEYiknUPwFQRPeW##)

![](.evernote-content/F6DD16CD-64EA-4480-ADCF-843E9446180F/AAF76515-D6A8-49F5-A435-51B6657F8847.jpg)

**中村裕：**被称为“史上最强”柯洁年仅19岁就达到九段，是中国围棋界的瑰宝。不光柯洁，近年中国陆续涌现了很多年轻的围棋高手。围棋曾是日本的看家本领，为何如今中国能够超越日本，培养出众多棋艺高超的年轻棋手呢？为了寻找答案，日本经济新闻（中文版：日经中文网）记者走访了柯洁生长的地方——浙江省及北京等地，了解到了中国的特殊国情。（注：本为由日文翻译而成）

|  |
| --- |
|  |
| **田沐沐（左）和周宗强对弈（右）（2016年11月，中国浙江省丽水市）** |

 　　柯洁的家乡是浙江省丽水市。2016年11月，记者来到柯杰曾经学习过的围棋培训班，目睹了那里大力培养新棋手的场景。

　　“在柯洁之后，过了大约10年，我们又发现了一个能力这么强的孩子。他可能跟柯洁一样，两年后为了当职业棋手去北京”，在围棋培训班里，周宗强（45岁）一边跟学生对弈一边对记者这样说。他是这个最初教会柯洁围棋的培训班里的老师，棋盘对面是6岁的田沐沐，念小学一年级。

　　周老师所说的“将来可能会超过柯洁”的田沐沐来围棋培训班还不到1年时间，但现在“他已经能跟45岁的我一争高下了”，周老师认可了田沐沐的才能。每天田沐沐放学后一来到培训班，周老师就陪他下棋。

　　记者拜访培训班那天已经是晚上7点多了，外面已漆黑一片，但同一个楼上的其他培训班还有十几名小学生在跟着一位老师学围棋。这种培训班还有很多，每天有将近100名学生放学后去学围棋到晚上8点半。大家都面带笑容，看上去很开心的样子给记者留下深刻印象。培训班的负责人李守胜（43岁）介绍说：“光我们培训班的孩子就有700人。这附近（丽水市莲都区）学围棋的孩子大约有1000人”。

　　围棋为何如此受欢迎？周老师介绍说：“现在中国竞争激烈，父母们都很拼。为了不让孩子输在起跑线上，一般从小就开发孩子的才能，围棋只是其中之一。不过，认为围棋有助于开发儿童潜能和提高智力的父母越来越多。柯洁的出名也带起了围棋热潮”。

其实，柯洁的父亲柯国凡（49岁）、母亲周柳萍（44岁）从小就开始拼命寻找柯洁的特长。周柳萍称，“试了美术、篮球、奥数等很多科目，最后决定让儿子学他喜欢的围棋”。

|  |
| **柯洁的父母（2016年11月，北京）** |

中国父母拼命为孩子寻找特长还有另一个理由，那就是特长和升学密切相关。

　　虽然各地区多少存在一些差异，但中国孩子上初中一般都要考试，这时候有特长能加分。不管是什么特长，都能加分，提高成绩。

　　加分同样适用于中考和高考。就连北京大学这种中国的代表性高校，高考成绩没过线的学生也能通过体育及文化等特长加分入学，这种情况并不罕见。像世界围棋界的一流选手柯洁一样拥有卓越才能的人会进一步受到优待，无需参加高考，就能进入喜欢的大学，这就是当今中国的实情。

　　柯洁的母亲周柳萍说：“儿子的围棋前辈也在30多岁时进了北大。现在忙于围棋，等稳定下来之后，将来打算让他到北大或清华上学”。正因为如此，除了学习之外，中国父母为让孩子学习特长付出非同寻常的努力。从英语到钢琴、小提琴、绘画，中国到处都有面向幼儿开设的培训班也是正因于此。

　　虽然幼儿教育存在过热问题，但中国长期受到诟病、偏重记忆的“填鸭式”应试教育也因为实行加分制而多少得到一些纠正。为培养柯洁这样的天才提供了沃土，这也是不争的事实。

　　柯洁8岁独自前往北京，11岁成为职业棋手，13岁进入国家队，为实现飞跃奠定了良好基础。在座落于北京市中心的中国棋院，目前共有来自全国各地的约60名国家队队员聚集在此历练棋艺。队员是从约400名职业棋手中选拔出来的精锐，都是16～30岁的年轻一流棋手。

|  |
| --- |
|  |
| **国家队队员切磋棋艺（2016年11月，位于北京的中国棋院）** |

　　虽说是国家队，但并没有报酬，唯一的好处是能出名。为了生活，他们只能作为职业棋手赢得比赛，自己挣钱。年轻职业棋手的年收入为15～20万元人民币左右。虽然与一场比赛就能赚到6万的柯洁等一流职业棋手存在差距，但年轻棋手们仍然怀着对将来的憧憬，每天坚持与队友们一起历练棋艺。

　　特长培训和激烈的竞争改变了全球围棋界的势力格局。日本棋院副理事长山城宏（58岁）介绍说，“感觉现的国际比赛中，进入前8的棋手里有6个是中国人，2个是韩国人。夺冠的几乎都是中国棋手，遗憾的是日本棋手连进入前8都很困难”。

　　据介绍，在日本围棋巅峰期的1986年，围棋人口达到了约1000万，后来因围棋下法多样化，减少到了目前的300～400万。20世纪90年代，韩国棋手崛起，2000年代中国棋手崛起，逐渐取代了日本的地位。

　　中国棋院围棋部部长王谊（56岁）表示，“在围棋方面，我们从日本身上学到了很多东西。现在年轻棋手层出不穷，中国的实力增强，还要感谢日本”。

　　中日之间通过围棋进行交流的传统从二战前一直延续下来，不过2017年将迎来一个重大节点。3月份，大阪将举办由中日韩三国顶尖职业棋手与日本开发的人工智能（AI）软件共同参加的国际围棋赛。日本的参赛选手是在2016年前无古人地夺得7冠的井山裕太九段（27岁）。中国的参赛选手并非柯洁，而是21岁的其他年轻希望之星。三个国家的顶尖棋手实力大比拼，再加上“人类对战AI”的元素，此次比赛必将备受瞩目。
