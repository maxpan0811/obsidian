一直觉得 Windows 托盘区可收纳的功能非常人性化，不像 macOS 的菜单栏势要把顶部的长条填满。

![](.evernote-content/0358A451-FC7B-429D-A4D8-2F935C95A3CD/7388A492-35D1-4F89-88D6-902881F0220F.png)

冬天看着菜单栏这些图标抱团取暖，没准还能让你感觉一丝暖意。但大部分时候都是有些碍眼，甚至在与某些软件菜单栏重合会直接隐藏。

部分软件运行在后台完全可以通过快捷键操作，例如 [Magnet](https://magnet.crowdcafe.com/)，这样看来不是很必要在菜单栏占坑，然而真正支持隐藏菜单栏图标的软件并不多。 Apple 官方也迟迟没有加入菜单栏收纳的功能，只能依赖第三方工具了。

![](.evernote-content/0358A451-FC7B-429D-A4D8-2F935C95A3CD/747318B9-A514-4095-AB7D-8245D59E3CA8.jpg)

此前，大家推荐最多的这类工具是 [Bartender 3](https://www.macbartender.com/)，售价 15 美元。而在 macOS 10.15 上，有用户反馈会索取「全屏录制」权限（官方回应并不会记录用户隐私信息），出于隐私考虑他换用了 [Hidden Bar](https://apps.apple.com/cn/app/hidden-bar/id1452453066?mt=12)。

![](.evernote-content/0358A451-FC7B-429D-A4D8-2F935C95A3CD/7906489E-7E97-47E5-9208-061B26FD9508.jpg)

Hidden Bar 是一款免费开源的工具，已经上架 [Mac App Store](https://apps.apple.com/cn/app/hidden-bar/id1452453066?mt=12)，并且在最新的版本中加入了中文的支持。

![](.evernote-content/0358A451-FC7B-429D-A4D8-2F935C95A3CD/3388E329-E221-4704-BF2F-80936CD3264C.gif)

直接看效果，Hidden Bar 在展开的状态下还是会占用菜单栏空间，打开后菜单栏会出现「｜」以及「>」或者「<」（分别代表展开和折叠两种状态）。

![](.evernote-content/0358A451-FC7B-429D-A4D8-2F935C95A3CD/8DE6A2FE-2350-4D33-9DD9-0A352612E031.gif)

使用起来也比较简单，按住 Command 键，拖动想要折叠的图标移到「｜」符号左侧就可以了，点击「>」就可以直接折叠，菜单栏也瞬间清爽了。

![](.evernote-content/0358A451-FC7B-429D-A4D8-2F935C95A3CD/B0FAA58D-066D-4065-9720-EC48CE55D971.jpg)

「｜」和「>」是可以分开的，你可以自由放置，比如，把「>」拖到最右侧，点击之后折叠的还是「｜」左侧的所有图标。

不过要注意一个逻辑问题，「>」不能放置在「｜」左侧，毕竟它不能把自己也折叠进去。

![](.evernote-content/0358A451-FC7B-429D-A4D8-2F935C95A3CD/B26004DF-A3BE-48E1-8BC0-44BC1F73A370.jpg)

Hidden Bar 可以设置用全局快捷键实现折叠 / 展开，使用更便捷；同时支持延时折叠菜单栏图标。

![](.evernote-content/0358A451-FC7B-429D-A4D8-2F935C95A3CD/EDF17393-E94E-4DC5-8167-AE5ECBA84163.jpg)

略有一丢丢不完美的是，Hiden Bar 中文支持并不完整，不过也基本不影响使用，估计很快会完善了。

如果你也在为 macOS 菜单栏位置紧张烦恼，这款小工具或许可以给你解压，而且还是免费的，甚至会吸引一部分 Bartender 3 付费用户。我也发现 GitHub 上还有一款 [Dozer](https://github.com/Mortennn/Dozer) 的应用，在功能上与 [Hidden Bar](https://apps.apple.com/us/app/hidden-bar/id1452453066?mt=12) 几乎一致，只是显示图标上略有不同，暂且不知道两者是否有关联，大家有兴趣的也可以尝试。
