英国权威揭秘 “美国病毒”，已出现重大变异，美本土将遭二次打击
===============================

作为世界权威的学术科研和教育机构，英国剑桥大学日前给出了最新研究成果，海量数据显示新冠病毒已在全世界蔓延，并且出现了 A、B、C 三种亚型，而且源头都指向了美国本土。

权威消息报道，英国剑桥在经过两个月的严谨而深入的对比后得出结论，全球发病率最高的新冠病毒可以归类为三个亚型，分别用英文字母 A、B、C 表示。

其中，北美大陆，尤其是美国和澳大利亚人群感染最多的是 A 亚型新冠病毒，它的表现是对所有人群都有极强感染特性，上到八十八，下到十七八，无论是什么肤色的人种，A 亚型病毒都能照单全收。

所以科学界目前根据它表现出的特性认为，它是原始病毒变异出的超级型号，杀伤力相当高。

B 亚型病毒主要集中在中国和东亚，东南亚地区，现在可以确定它是由 A 亚型变异而来，现在普遍认为是由感染 A 新病毒的美国人携带而来，对平均体温较低的亚洲人更敏感，也更具传染性，所以才在爆发初期表现出对黄种人更高的致死率。

由于大陆立刻施行了极为严格的防控措施，所以没有让 B 亚型病毒在全世界蔓延，否则全球黄种人和亚洲人都将遭受灭顶之灾。

那么 C 亚型病毒呢？按照剑桥大学研究结果的说法，也是从美国和澳洲的 A 亚型病毒变异而来，与专攻黄种人的 B 亚型不同，C 亚型对平均体温较高的白种人和非洲人种更易感，并且体外存活时间更久，气溶胶传播效率远超 A 型和 B 型。这也就是为何欧洲在不到一个月时间内就彻底沦陷的原因。

经过以上简单介绍，我们可以发现，由于美国是全球人种组成的移民国家，为了更好的传播，原本存在于蝙蝠和穿山甲身上的原始病毒变异为 A 亚型，对所有人展开无差别攻击。与此同时随着美国人的全球活动开始扩散，分别在亚洲地区针对黄种人的基因特性变异出 B 亚型，以及针对欧洲白种人的 C 亚型。这都是病毒为了完成传播和繁殖的特性所决定的，所以在一定程度上来说，没有那么多阴谋论。

但是，病毒来自美国大陆的可能性几乎是毋庸置疑的，按照特朗普的逻辑，这个病毒应该被称为 “美国病毒”。值得一提的是，1918 年致全球至少 5000 万人死亡的 “西班牙流感”，其源头也是来自美国。倒霉的西班牙人背了一百年的黑锅。

那么事情就这样结束了吗？当然不是！

包括英国剑桥在内的全球卫生专家都担心一个迫在眉睫的事实，那就是已经在美国纽约州境内发现了数量越来越多的 C 亚型病毒患者，它直接导致美国的非洲裔黑人死亡率比白人高出至少三倍。

目前的猜测是，实际上，从 2019 年秋季开始的所谓的大流感其实就是新冠病毒原始病毒，它已经在美国本土很多地区造成了群体免疫，很多美国人，尤其是身体素质较好但经济水平很差没有钱治病的黑人族群产生了群体免疫，他们身上都有了抗体。

但是 A 亚型病毒的出现和 C 亚型病毒的反向输入，令已经形成了原始病毒抗体的人群的群体免疫失效，因为他们的抗体是针对原始病毒的，面对 A 型和 C 型病毒，免疫系统会误以为是友军，不但不发起攻击，还会主动带路，令病毒 RNA 顺利进入细胞核心。这种情况，或许可以在一定程度上解释，美国黑人族群的超高死亡率的原因。

不难看出，这些来自自然就的病毒在人体世界中寻找到了施展拳脚的空间，它们不但凶残而且变异速度极快，基本上两个月时间就能完成一次新亚型的替换，致使原来的抗体失效，甚至成为杀死宿主的关键。所以，英国剑桥大学的科研人员认为，由于美国本土集中了全球的所有人种，简直是新冠病毒的天然培养皿，随着 C 亚型病毒的反向输入并开始爆发，美国本土很有可能将迎来病毒的二次，甚至第三次爆发。人类的黑暗时刻或许真的到来了。
