Fast Word Query: Multi-threaded queries for words from local or web dictionaries
================================================================================

0.16MB. Updated 2019-06-23.

The author has shared [1 other item(s)](https://ankiweb.net/shared/byauthor/1807206748).

Description
-----------

**Support for 2.0.x has been discontinued, will be updated add-on for 2.1.x only in the future.
因时间关系，本插件已停止针对 Anki 2.0.x 的更新维护，以后只针对 Anki 2.1.x 进行更新维护，不便之处敬请谅解！**
Home Page: <https://sth2018.github.io/FastWordQuery>
Project Repo: <https://github.com/sth2018/FastWordQuery>
If you have any questions, you can open issues [**HERE**](https://github.com/sth2018/FastWordQuery/issues)
[Supported Dictionaries](https://github.com/sth2018/FastWordQuery/blob/master/docs/services.md)
[为单词添加真人发音（朗文 mdx 词典）](https://github.com/sth2018/FastWordQuery/blob/master/docs/get_mdx_ldoce6_sounds.md)
This addon query words definitions or examples etc. fields from local or online dictionaries to fill into the Anki note.
It forks from [WordQuery](https://ankiweb.net/shared/info/775418273), added **multi-thread** feature, improve stability, and some other features.
**Features**

* Querying Words and Making Cards, IMMEDIATELY!
* Support querying in mdx and stardict dictionaries.
* Support querying in web dictionaries.
* Support **Multi-Thread** to query faster.

**Install**

* Place addons or addons21 folder of this repository to anki addon folder.
***OR**** Use the installation code: 1807206748

**Setting**

1. Click Menu **"Tools -> Add-ons -> FastWQ -> Edit..."**

![](.evernote-content/CA6788A5-C324-4C2D-9072-D62360FEED80/0623C48F-E866-4D2C-8925-D2701D138010.png)

2. Edit the code and click **Save**

![](.evernote-content/CA6788A5-C324-4C2D-9072-D62360FEED80/CE58E930-BC84-4750-8982-A7C165D91D99.png)

**Config**

1. In Browser window click menu **"FastWQ -> Options"**

![](.evernote-content/CA6788A5-C324-4C2D-9072-D62360FEED80/9A6B8551-E462-4108-B026-4EC7C8900397.png)

2. Click **Settings** button in the Options window

![](.evernote-content/CA6788A5-C324-4C2D-9072-D62360FEED80/1DBED879-0791-414E-BEB9-CBD83E444131.png)

* ***Force Updates of all fields** : Update all fields even if it's None*
* ***Ignore Accents** : Ignore accents symbol of word in querying*
* ***Auto check new version** : Check new version at startup*
* ***Number of Threads** : The number of threads running at the same time*

**Usage**

* ***Set the query fields***

1. Click menu **"Tools -> FastWQ"**, or in Browser window click menu **"FastWQ -> Options"**
2. Select note type

![](.evernote-content/CA6788A5-C324-4C2D-9072-D62360FEED80/35132A16-33B5-4A6C-B175-2B19254C8D8E.png)

3. Select Dictionary

![](.evernote-content/CA6788A5-C324-4C2D-9072-D62360FEED80/6429B0E7-0EED-4BFB-A091-AB6E37C62F48.png)

4. Select Fields

![](.evernote-content/CA6788A5-C324-4C2D-9072-D62360FEED80/C8796E0C-9760-4146-A69F-394E65DA3D7F.png)

5. Click **OK** button

* ***'Browser' Window***

1. Select single or multiple words, click menu **"FastWQ -> Query Selected"** or press shortcut Default is **Ctrl+Q**.

![](.evernote-content/CA6788A5-C324-4C2D-9072-D62360FEED80/9E0930E4-2709-4457-80D7-D037A09A112B.png)

2. Waiting query finished

![](.evernote-content/CA6788A5-C324-4C2D-9072-D62360FEED80/469A76C7-0337-4E97-9673-CA791A4CC681.png)

* ***'Add' Window***

1. Click Add button in Browser window, open Add window

![](.evernote-content/CA6788A5-C324-4C2D-9072-D62360FEED80/A44D093E-30BE-43A8-8D72-A0B9D268C5A0.png)

2. Edit key field and click Query button

![](.evernote-content/CA6788A5-C324-4C2D-9072-D62360FEED80/03981FC6-0A0E-41A6-ADE1-7FD9610F1A63.png)
