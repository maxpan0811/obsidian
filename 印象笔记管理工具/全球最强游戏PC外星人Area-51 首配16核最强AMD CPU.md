# 全球最强游戏PC外星人Area-51 首配16核最强AMD CPU

---

![](.evernote-content/CD8D550F-BAFB-47EA-91DC-D83A95D73DE6/1A4CE026-CA09-4585-9DE5-0A5B67A23280.jpg)

Alienware 是戴尔旗下高端游戏 PC 品牌，从某种意义上说，该品牌在游戏玩家心中的地位颇高，当然其笔记本电脑和台式机价格并不低，毕竟其硬件配置也不低，并配以专门的超酷游戏风格机箱设计和高效的散热系统等等，而提供多种高端硬件配置选择也是吸引游戏玩家的因素之一。

在本届 E3 展会上，Alienware 正式推出了全新一代 Alienware Area-51 R3 主机，这是目前世界上最强大的游戏 PC，因为不仅可以选择 AMD Ryzen ThreadRipper + X399 平台，也可以选择和英特尔最新的 Core-X平台， 同时还提供丰富的硬件定制选项，玩家可以根据需求选择想要的版本。

全新的  Alienware Area-51 R3 依然保持此前超酷风格的机箱设计，格给人的印象十分深刻，因为机箱并非传统长方体设计，而是类似于球体的形状，或者其为不规则的六边形。在 Alienware 官方来看，此标志性的创新设计，首先极为符合人体工程学标准，再者重点则在于提供更良好的散热。

![](.evernote-content/CD8D550F-BAFB-47EA-91DC-D83A95D73DE6/BF96A300-2CA3-48A2-8AA4-4B1A92A7D358.jpg)

**配置上绝对是顶级**

Alienware 表示，最高可以选择  Ryzen Threadripper 处理器，而在液冷散热的情况下，最高支持插入三款独立显卡。并且，除了可以使用 Ryzen Threadripper 处理器之外，也可选配英特尔全新的 Core-X 系列处理器，最高可选 12 核心的 Skylake-X 架构芯片。

更具体来说，Ryzen Threadripper 处理器版本的 Area-51 R3 主机，可选 Ryzen Threadripper 12 核 24 线程或者 16 核 32 线程的处理器。而 Core X 系列处理器版，可选 6 核的 Core i7-7800X、8 核的 Core i7-7820X 或 10 核的 Core i9-7900X 三个处理器。

其他硬件定制方面，可选最高 1TB 的 M.2 PCIe 固态硬盘和 2TB 的机械硬盘，最高 64GB 四通道 DDR4 2933 内存，可搭配 32GB 的  Optane内存。独立显卡支持单路、双路和三路交火，最高可选 GeForce GTX 1080 TI 或者 Radeon 580 显卡。网卡是双千兆的 Killer E2500 网卡。电源是 PSU 类型，有 850 瓦或 1500 瓦。

在主机箱上，设有诸多常用的 I/O 接口，包括 1 个 USB 3.1（Gen2）标准的 Type-C 接和 1 个 USB 3.1 Type-A 接口、2 个 USB 2.0 以及 8 个 USB 3.1（Gen1）Type-A 接口。

![](.evernote-content/CD8D550F-BAFB-47EA-91DC-D83A95D73DE6/98843CAF-FC5B-412C-BDFF-49C0C4713D24.jpg)

**全球独占 Threadripper 处理器**

Alienware 与 AMD 签署了排他性协议，前者声称在 2017 年 Alienware 将是 AMD Ryzen Threadripper 独家 OEM 合作伙伴，只有 Alienware 能够推出基于此 16 核心 32 县城的高端桌面 CPU 打造游戏 PC。 所谓独家 OEM 合作伙伴，其意思就是包括惠普、联想，以及 Alienware 的母公司，均不能推出 Threadripper 处理器的品牌 PC。

AMD 此前已经大肆宣传 Threadripper 处理器的性能，至少在竞争上完全不输给 当前所有消费领域的高通桌面处理器。因为 Ryzen 7 和 Ryzen 5 已经可以买组与英特尔 Core i5 和 Core i7 的较量，而英特尔 Core i9 目前只有十核心版，相比  Threadripper 处理器还是差些火候，所以 Alienware 独占这一处理器，相当于在一段时间之内保持了最强游戏 PC 的称号，吸引购买了。不过，在一定程度上会激怒竞争对手和游戏玩家。

**独占，但不是唯一？**

Threadripper 处理器产品可能相当令你期待，毕竟 16 核心的游戏 PC 性能上足够疯狂。不过，现在所面临的有两个消息，一个好消息和一个坏消息。坏消息是，Alienware 独占 Threadripper 处理器的 Area-51 R3 游戏 PC 主机，市面上不会有任何其他品牌的竞争产品，毕竟 AMD 与 Alienware 签署了此高端芯片的独家使用权。

不过，好消息就是，Alienware 虽然声称全球独占 Threadripper 处理器，但其实独占的范围框定在了主流 PC 厂商当中，包括 Alienware 的竞争对手如惠普和联想。而一些小型的 OEM 厂商还是能够推出基于 Threadripper 处理器的 PC 产品，例如 Origin PC、Maingear、iBuypower 和 Cyberpower等等，甚至也可以自主 DIY 的产品。

![](.evernote-content/CD8D550F-BAFB-47EA-91DC-D83A95D73DE6/561DFF70-2EB6-44F1-BBF8-5657B7471C55.jpg)

其实 AMD 官方比较可以回避关于“独占”的问题，也没有十分明确 Alienware 完全独家享用，只是简单的提到：

“我所有的 OME 合作伙伴对 Ryzen 处理器的兴趣都很浓厚，可以看到台北电脑展上都有产品展示。基于这种势头，我们对与 Alienware 合作感到非常兴奋，Alienware 是我们强大 Ryzen Threadripper 处理器的 OEM 合作伙伴，在 OEM 厂商中领导高端桌面处理器市场。我们期待通过与 Alienware 的合作伙伴关心，能够展示出我们 Ryzen Threadripper 处理器和 TR4 平台的全部潜力。”

**那么强大的游戏 PC 何时能够买到呢？**

Alienware 方面提供的实质性消息是，他们会尽快在 7 月 27 日发货 Area-51 Threadripper Edition。这就意味着，不用得到多久就能够体验 16 核心的强大游戏 PC了，具体价格并没有公布，按照前任来说应该不会便宜，七八万块可能会是常态。

至于英特尔粉丝，那就要有耐心等待了，Alienware 方面也没有透露具体的上市日期。因为到目前位置，关于Core X 系列 Core i9 处理器的上市日期还是个未知数，不久前英特尔表示 10 核心或更低核心的版本能够在 6 月 19 日提供预定。不过，目前并没有与 Core X 系列处理器合作的 OEM 厂商，也许 Alienware 就是第一家，只希望不要等太久。

![](.evernote-content/CD8D550F-BAFB-47EA-91DC-D83A95D73DE6/938ACE3F-4413-49D5-9C6D-EFB634671EE5.jpg)

**最后一个问题，你需要它吗？**

相信这也是大多数人的问题， 尤其是普通玩家，不禁会问“我需要如此顶级的 CPU 吗？”

对此，Alienware 方面的给出的回答是，“我们相信，绝对有必要。”Alienware 还进一步解释称，“我们正在研究多方面的功能，包括虚拟桌面，或者在同一主机上实现两个 VR 体验，相信强大的处理器能够满足这些惊人的游戏体验优势。”

当然了，那些只是设想，Alienware 说从实际角度出发，那些边玩游戏边做线上直播或者录制的，多核 CPU 一定能够起到很大的帮助。不过，Alienware 也承认，单纯从游戏角度来看，这是鸡和蛋的情况，因为游戏开发商始终以赚钱为目的，特定性能平台的游戏不会帮他们赚到更多的钱。

简单地说，对于一般游戏玩家，16 核或 12 核的游戏机不是刚性需求，但游戏 PC 能够不断追求极限，这是当今计算平台进步的表现，某些用户也许曾经使用多台 PC 才能满足，可今天单一 PC 就能满足了，这显然是积极的一面。
