狂揽64.5K星 !!! 再见苹果iCloud , 图片视频秒备份 , 告别空间和隐私焦虑 , 太6了
===================================================

Original 开源日记 开源日记

大家好 , 我是开源日记呀 !

你还在为照片和视频的备份发愁吗？

![](.evernote-content/80EE1A06-9053-42A5-AB52-D1CA8130B44E/8B1F89E4-5DF5-4863-892D-0A1240A9E454.png)

你需要一个既强大又方便、还能保护隐私的自托管解决方案！Immich 就是这样一款高性能的开源神器，解决了 Apple iCloud 替代品难找的痛点，同时支持照片、视频的备份与管理，简直太吊了！

#### Immich介绍

![](.evernote-content/80EE1A06-9053-42A5-AB52-D1CA8130B44E/3711D585-A766-41B0-8A45-E5547C68CFD7.png)

> Immich 是一款高性能的开源自托管照片和视频管理工具。它能够自主完成照片备份、多设备管理，并支持共享相册和多用户环境。更重要的是，所有的数据都在你的手中，既安全又可靠——一切尽在本地掌控！再也不用担心隐私被泄露或备份中途断掉。支持 iPhone 和 Android 手机，全方位守护你的珍贵记忆！

#### 开源成就

* **Star 数：** Immich 在 GitHub 上已经获得了 **64.5k+** 的 Star![](.evernote-content/80EE1A06-9053-42A5-AB52-D1CA8130B44E/9723AB85-D3A0-4EF3-918E-C9F3113E5F90.png)
* 开发语言：TypeScript (48.5%), Dart (29.0%),  Svelte (17.9%)

#### 核心功能

![](.evernote-content/80EE1A06-9053-42A5-AB52-D1CA8130B44E/928A2EBB-C2A8-4D02-9826-E48B26291FBA.png)

* **高性能备份**：自动备份照片及视频，并防止重复上传，支持后台自动备份功能。![](.evernote-content/80EE1A06-9053-42A5-AB52-D1CA8130B44E/4CEB1C4F-3BBA-4E20-A19C-5D913637FF23.png)
* **强大的搜索功能**：借助 AI 和 CLIP 技术支持按元数据、对象、脸部、标签等搜索。![](.evernote-content/80EE1A06-9053-42A5-AB52-D1CA8130B44E/41F769AA-6066-4A14-9E8C-1A7F909B65B7.png)
* **高级功能支持**：

+ 支持 RAW 格式文件和 360 度图片显示。
+ 支持 EXIF 元数据、地图展示。
+ 支持 LivePhoto/MotionPhoto 的备份和回放功能。

* **多用户与共享支持**：

+ 支持多用户管理。
+ 支持创建共享相册和公开分享链接。

* **文件夹视图与用户自定义存储结构**：直观而个性化的文件展示方式。![](.evernote-content/80EE1A06-9053-42A5-AB52-D1CA8130B44E/4CAC7E45-8ADC-4AD6-8B1B-3A566E2F4F45.png)
* **附加工具**：集成面部识别、聚类，以及“X 年前的今天回忆”。

#### 安装步骤：

* 确保安装了 **Docker** 和 **Docker Compose**。
* 克隆仓库：

  ```
  git clone https://github.com/immich-app/immich  
  cd immich
  ```
* 启动 Immich：

  ```
  docker-compose up
  ```
* 打开浏览器访问 http://localhost:2283。![](.evernote-content/80EE1A06-9053-42A5-AB52-D1CA8130B44E/62588CA6-1550-4C56-AE87-0F02631F752F.png)

> Immich 是现代数据隐私保护的绝佳选择。它快速、安全，支持多个设备、自动化操作、共享需求，即使是普通用户也能轻松上手。它不仅解决了备份成本高的问题，还带来了极佳的性能体验。
