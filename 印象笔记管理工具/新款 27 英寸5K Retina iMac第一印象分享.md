# 新款 27 英寸5K Retina iMac第一印象分享

---

[![](.evernote-content/F3F996EE-DEF2-43A7-A1BD-461536EF68D9/DDFE09FB-CDB6-4313-9CE6-58526BF053BC.jpg)](http://www.feng.com/iPhone/news/2015-10-28/The-new-5-k-27-inches-Retina-iMac-share-first-impression_628456.shtml)

　日前，苹果更新了 iMac 系列产品，27 英寸版 iMac 配备 5K 分辨率屏幕，新的 iMac 性能更强劲，速度更快。  
  
　苹果表示，新的 5K 屏幕将具备更强大的显示效果，能够增加25%色彩效果。为了提升色彩还原能力，苹果采用了新的显示技术。这种新的显示屏用到了一种“红-绿荧光 LED（red-green phosphor LED）”用于提升红色和绿色的还原能力。

![](.evernote-content/F3F996EE-DEF2-43A7-A1BD-461536EF68D9/DDFE09FB-CDB6-4313-9CE6-58526BF053BC.jpg)

采用了英特尔 Skylake 处理器，最低配的 i5 版本达到了 3.2GHz 主频，最高配的 i7 主频达到了 4GHz，图形核心为 AMD Radeon R9 系列，低配 M380，外加 2GB 显存，高配 M296X 加 4GB 显存。新版 iMac 还配了 2 个 Thunderbolt 2 接口。

27 英寸 iMac 给人的第一印象如何，跟随我们的脚步来看看。

苹果一旦有什么新技术，它都会慢慢的将这一技术在苹果产品中普及。近年来，苹果慢慢的将 Retina 屏幕带到旗下更多产品上。

2010 年，Retina 屏幕技术率先出现在 iPhone 4 上，Retina 屏幕支持更高的像素密度，文字和整体内容更加清晰锐利，易辨认，减轻了眼睛的压力。

此外，Retina 屏幕还拥有更高质量的面板，支持 sRGB 色域，提供宽广的可视范围，更高的亮度和对比度。

虽然在不同的设备上，Retina 屏幕像素密度不尽相同，但都十分优异，在正常距离观看屏幕，用户无法看到像素颗粒的存在，也就是我们说的噪点。

就苹果iPhone来说，Retina屏幕像素密度超过 300ppi；iPad 像素密度最低 264ppi；Retina MacBook像素密度最少 226ppi；Retina iMac 像素密度最少是 218ppi。

虽然苹果没有提升 5K Retina iMac 的像素密度，但是，它支持更广的 P3 色彩空间。

去年苹果发布的 5K Retina 屏幕是目前市场上你能找得到的最优秀5K屏幕之一，支持全 sRGB 色域，拥有出色的锐利度、亮度、对比度以及视角。

正是因为去年 5K Retina 的高度已经如此出色，苹果今年还能够进一步升级，这一点让我们觉得很意外。

![](.evernote-content/F3F996EE-DEF2-43A7-A1BD-461536EF68D9/FFD7F577-33C8-4632-8A1C-BAE02658489D.jpg)

不过，苹果精益求精也不是第一次。我们在 iPhone 6s 上见到了“Touch ID 2”，虽然 Touch ID 技术原本已经非常成熟，但苹果还在继续升级优化。

对普通用户来说，去年的 5K Retina 已经非常优秀。如果把它和新的 27 英寸 5K Retina 摆在一起，显示标准的 sRGB，大多数人其实无法辨认哪一款屏幕是旧的，哪一款屏幕是新的。

也就是说，苹果继续升级 27 英寸 5K Retina iMac，主要考虑的不是普通消费者，而是那些专业的摄影师和视频编辑工作者，他们比普通用户需要更先进的技术，更多的色域。

对这群专业用户来说，苹果发布支持 P3 色域的 iMac 的一个大好消息，这意味着，图片和视频画面都较以往更加真实。

在硬件方面，我们常常对硬件规格的升级很欢迎，尤其是价格不提升的情况下。

除了这块新屏幕之外，27 英寸 5K Retina iMac 还升级了很多可口的内部元件，比如，采用最新款的英特尔 Skylake 第六代 CPU，直接跳过了 Broadwell 系列处理器，上一代 27 英寸 iMac 搭载的是 Haswell 处理器。

可能有人不知道 Skylake 有什么优点，英特尔在第六代处理器上引入了全新的微体系结构，不仅可以减少功耗，更好的提升性能，同时还能大幅提高图形显示的性能。

英特尔表示，最新一代处理器使用了 14nm 工艺，但架构已经更新，单线程、多线程性能可提升 10-20%；在核显图形方面，升级为第九代低功耗版，H 高性能系列搭配顶级 GT4e，U 低压系列也有缓存，号称 3D 游戏性能提升最多 50%。

有人认为，Skylake 肩负拯救 PC 市场的使命。英特尔高管表示，PC 行业正处于风口浪尖，需要一次重大的变化，Skylake 架构针对超过 300 种不同的 PC 设计，预计未来几个季度时间里，基于第六代核心技术的产品将全面登场。

我们体验的这款 5K Retina iMac 属于入门版本，搭载 3.2GHz 四核 Core i5（第六代）处理器。如果你不想要入门版本，你还有其他版本可选。比如，3.3GHz Core i5 处理器；4.0GHz Core i7 处理器（这是 iMac 第一次采用主频超过 4.0 的处理器，如果超频，可超频到 4.2GHz）。

![](.evernote-content/F3F996EE-DEF2-43A7-A1BD-461536EF68D9/1F2DF3FF-CC4C-4F38-9677-90D05595A34E.jpg)

此外，新 iMac 拥有更快的 RAM，时钟速率为1867MHz。苹果官方表示，新iMac最高可支持 32GB RAM。不过，有论坛网友表示，其实可以扩大到 64GB RAM，和 Mac Pro 同一水平。

苹果还为 27 英寸 iMac 换上了最新的 AMD 移动 GPU，不过，苹果没有公布，新 GPU 和旧款相比速度提升有多快。苹果只是表示，在新的 CPU 和 GPU 帮助下，iMac 总的计算能力大约是 3.7 万亿次每秒，再一次接近 Mac Pro 的水平。

闪存空间也得到了升级，速度是之前的 2.5 倍，读取速度达到 1800Mbps。我们购买 iMac 的同时，还购买了 Magic Kerboard 键盘和 Magic Mouse 2 鼠标。在未来的详细评测中，我们会和大家分享它们的速度和实际体验。

虽然我们现在还没有上手这台 iMac 很久， 但它整体的系统架构表明，新 iMac 将会是苹果发布的最快的 iMac。

写到这里，突然想起了 Mac Pro，Mac Pro 产品线从 2013 年到现在都没有升级。现在，27 英寸 iMac 全线都达到了 5K 分辨率，内部规格也大幅提升，不知道什么时候到 Mac Pro。
