换上新内核的 Edge 浏览器更好用了，Chrome 重度用户也可以无痛迁移
======================================

19 小时前

[![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/BF06F425-2548-4C81-B409-B2B8D63B5DC4)](https://sspai.com/user/78/updates) 

#### [化学心情下 2](https://sspai.com/user/78/updates)

虽然此前采用 EdgeHTML 的 Edge 有着令人印象深刻的性能表现，但在 Chromium 这个现代浏览器的「事实标准」面前，缺少足够开发社区支持的 Edge 浏览器在与老对手 Chrome 的竞争中越发显得力不从心（即便大多数时候还是在自己的主场 Windows）。

因而微软在去年年底决定将 Edge 浏览器的内核更换为 Chromium 这一举动也不难理解：对 Edge 开发团队而言，更为成熟的 Chromium 内核可以避免开发过程中的「重复造轮子」，对于用户而言迁移难度也将因此大大减小。

随着最近测试版 Chromium 内核 Edge 浏览器的放出，曾经相互竞争的对手最终变成了同一颗「心」的兄弟，那这个基于新内核的 Edge 浏览器用起来究竟怎么样？它又能否像微软一直以来所期望的那样让 Chrome 用户「倒戈」呢？

平滑的迁移流程，混合的界面风格
---------------

从目前在网络上已经泄露的 Chromium 内核 Edge 的安装过程来看，其安装过程和 Chromium 非常相似，如果不看文件名称只看图标，错认为是 Chromium 的离线安装包也不足为奇，安装过程也是一闪而过，没有那种传统安装软件一步步 next 的过程，可以说几秒钟之后就可以看到 Chromium 内核 Edge 浏览器的界面。

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/0A3188FD-4818-4FD6-A1C2-F3914305CE76.png)

安装时会默认检测系统中是否存在 Chrome 浏览器，之后在首个设置引导页中询问是否将 Chrome 中的数据（书签、历史记录、填充历史）导入到 Edge 中。

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/F8F95152-FDB9-4B2B-8F7F-CCA4BF0ED8FD.png)

当然如果系统中还有其他浏览器的话，你也可以选择其他的浏览器，而导入数据也可以进行选择，最后你可以选择同步数据 —— 是的，你可以像使用 Google 账户同步一样，将这些数据同步到你的 Microsoft 账户中！

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/4FD95212-CCE1-4D5B-BE71-821978BC2EF0.png)

之后的引导页面则是帮助你选择你的新标签页的样式，默认给三种选择：纯搜索栏、常用地址以及资讯瀑布流，此外可以在标签页载入 Bing 壁纸，这个类似与目前的 Bing 搜索网页的设计，只不过现在可以默认设置到新标签页中。

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/856B5BD0-54E6-4861-8935-8131B2E0A846.png)

当然也可以通过新标签页的右上角的菜单来选择展示的模块。从另一个角度来看，这种新标签页的展示和 Chrome 有着异曲同工之妙：都强调了搜索服务，只不过 Chrome 展示的是 Google Doodle，而 Edge 则是 Bing 壁纸。

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/3904ACF3-9CD1-4050-843F-103B6145467E.png)

全部设置完成之后，默认将载入新标签页，仔细观察界面你会发现和 Chrome 相似度极高，尤其是几乎完全沿用了从 Chrome 70 开始上线的新 Material Design UI 设计。

不过点击右上角的汉堡菜单进入的设置页面则更接近 Windows 10 的设计语言，显然目前的版本从视觉上还尚未完全统一。

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/129E8335-5391-46D6-ABE2-637B89874559.png)

其他操作设置上则针对微软的账户体系也进行了微调，比如说地址栏上搜索默认是必应搜索，新标签页下的资讯流来源则是 MSN 新闻资讯等等。

账户与同步已基本完工
----------

相比之前的 Edge，新内核的 Edge 采用了单独的设置页面，这个页面中更接近 Windows 10 目前的用户界面设计风格，可以设置的选项也相当的多，但是基本内容其实和 Chrome 是一样的，只不过针对每一个类目进行了分栏整理。

而原先在 Edge 上的一些原生特色功能则没有移植到新内核的 Edge 上，例如**阅读模式，鼠标悬浮标签页展示网页预览**等等，所以并不清楚未来是否会以扩展的形式加入进来。

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/1F30DF6C-32DB-4782-95AF-ED67B5AB84D4.png)

设置中其他的项目目前基本都是可用状态，唯一目前存在 Bug 的是语言设置，目前仅支持英文界面显示，即便是更改为中文语言，界面依旧还是英文。

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/8564933D-99E8-414D-A231-62D57F5F8EA2.png)

既然换成了 Chromium 内核，Edge 最让我关心的则是微软账户同步体系，这对于习惯于 Google 云同步服务的用户来说不失为一个更容易接受的选择，至少在国内的网络环境中，微软的同步服务更为稳定。而和 Windows 10 的账户体系打通，则可以实现跨平台的游览器账户信息同步。在设置页中针对同步选项也非常详细，同步的数据可以是设置、书签、历史、扩展、表单信息甚至是密码等等。

欢呼！无痛支持 Chrome 扩展
-----------------

**我一直都认为 Edge 最后被迫更改为 Chromium 内核最大的原因在于扩展程序的匮乏** —— 与 Chrome 商店中丰富的扩展应用相比，Edge 扩展需要在封闭的微软应用商店下载，而由于各种限制，以至于上架的扩展只有寥寥几款，而换成 Chromium 内核后的 Edge 终于可以「无痛」兼容 Chrome 扩展了！

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/236AA84D-039E-4077-89B1-706F29AF8B7F.png)

与 Chrome 应用商店类似，新内核的 Edge 浏览器同样也内置了一个扩展应用商店：Microsoft Edge Insider Addons，目前的扩展类目包括密码管理、广告屏蔽、教育、社交、购物、开发者工具等多个类目项。而实际上这些扩展都采用了 **.crx** 格式，也就是说这只是微软自己收集的 Chrome 扩展。当然你如果对目前上架的扩展还不满意的话，在扩展设置中勾选使用「其他商店扩展」就可以无缝从 Google Chrome 商店下载扩展了！

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/E8FBB788-4D55-401B-9FC5-104C794E7A09.png)

我也通过这个办法几乎 100% 将我的 Chrome 扩展移植到了 Edge 上，以至于我自己都分不清我到底用的是 Edge 还是 Chrome（用户体验实在是太像了），唯一要说区别的可能就是目前尚不支持 Chrome 的主题，当然这一点对实际使用并没有太大影响。

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/3419E7C8-1716-4901-9409-D8F104003163.png)

性能优势依然让人印象深刻
------------

Edge 此前在和 Chrome 竞争中多次宣称其优秀的性能，例如说更快的网页渲染速度，以及更低的系统资源占用等等，不过既然新版本将会采用和 Chrome 相同的 Chromium 内核，那么这些性能上优势是否还会保留呢？

有意思的是目前泄露版本的新内核 Edge 其采用的内核时 Chromium 75，也就是说其内核甚至比 Chrome 稳定版还要新（目前是 Chrome 73），客观来说采用新内核的 Edge 还不算稳定（虽然从我的使用来看，浏览体验基本和稳定版 Chrome 无差），不过我们还是可以测试一下目前的性能，其实主要还是打开网页的内存占用情况。

* **测试系统：**64 位 Windows 10 v1809
* **测试浏览器：**Microsoft Edge（EdgeHTML） | Microsoft Edge（Blink）| Google Chrome（稳定版）（浏览器均没有安装第三方扩展，部分浏览器采用隐身模式浏览）

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/C770D09D-3BA7-46D3-9E42-72E9882BB337.png)

Microsoft Edge（EdgeHTML）| Google Chrome | Microsoft Edge（Blink）

测试同样打开少数派的首页，最终的系统占用如下：

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/F5998FF2-CC8A-4430-9503-3BDE358EEDC5.png)

对比之后不难发现，采用新内核 Edge（桌面最右的浏览器）占用了 174.0 MB 内存，虽然比采用旧内核的 Edge 要多一些，但对比 Chrome 的性能怪兽倒是要好上不少，不知道微软是否对此进行了相关的优化。

可玩性：Chrome 怎么用，新内核 Edge 就可以怎么用
------------------------------

除了扩展之外，对于我而言 Chrome 的一大优势则是丰富的实验性功能以及开发者工具，而这也是我最为关心的部分（工作原因导致每天必须要使用）。

既然采用了 Chromium 内核，新内核的 Edge 是否也能胜任开发者的要求呢？

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/70E651F8-939D-40CF-8242-213D94EED79E.png)

先说一下开启实验性功能，在 Chrome 我们可以通过 chrome://flags 来进行开启，Edge 则采用类似的 edge://flags 开启，开启之后界面和 Chrome 上的 flags 基本一致。

那么我们先开启 Windows 10 上开始的暗色主题模式，搜索 Dark，找到设置项「Microsoft Edge Theme」选择 「Enabled」然后重启浏览器即可，之后的效果就是浏览器界面会根据系统主题（深色 / 浅色）自动切换。

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/23E32C9B-AFF0-4DB4-AA6A-CCB74D37EB79.png)

当然你也可以搜索「Desktop PWAs」来开启 PWA 功能，方法和 Chrome 上也是一样的。

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/9C766DEB-ED85-4301-BB99-3E1CB64C0A99.png)

针对很多互联网开发者而言，最长打交道的则是浏览器的开发者工具（F12），在新内核的 Edge 上同样也可以，而且相比 Chrome 的开发者工具，Edge 的开发者工具还有设置页面！

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/03639C20-C207-43C3-A6E0-EFDE6119D067.png)

通过 F1 开启开发者工具的设置页面，比如说可以设置开发者界面主题，添加模拟的地区经纬度坐标、设置工作区甚至可以对开发输出的相关数据进行调整，不得不说这一次微软对开发者支持变得更友好了。

![](.evernote-content/2C416B2D-FB0A-416C-BC86-96DAF79290B1/99ABB066-AA65-4880-9362-2CFF6581CB94.png)

结语：这或许是我未来 Windows 10 的首选默认浏览器
------------------------------

我们曾经诟病 IE 从而诞生了 Firefox ，而之后 Chrome 的出现则让 Chromium 内核成为桌面浏览器的事实标准，几乎市面上所有的第三方浏览器都采用了 Chromium + IE 的解决方案。显然即便是微软放弃 IE ，开发出「更快更强」的 Edge，也最终扛不过市场趋势。

基于 Chromium 内核的 Edge 浏览器其实应该也是微软在技术和市场中进行取舍的结果 —— 顺应市场，而不是强推自己的所谓标准。换用开源的 Chromium 内核的 Edge 由于建立在巨人的肩膀上，因此从具体的使用体验来看，并没有出现想象中的「翻车现场」，整体完善度可以说非常的高。而在部分细节上微软则正在逐步进行调整，而基于微软账户同步体系 + Chromium 内核则对于习惯使用 Chrome 浏览器但苦于国内网络环境的用户可谓是最大的利好。

可以期待的是，至少在未来的 Windows 10 上，你完全都不需要再安装所谓的其他第三方浏览器了，因为 Edge 完全可以满足我的使用需求。
