郑重声明
----

*2017-11-14*[AWS云计算](http://mp.weixin.qq.com/s?__biz=MzA4ODMwMDcxMQ==&mid=2650893601&idx=1&sn=cb7fe7f30fef236dbff2a2732236923c&chksm=8bd980e9bcae09ffccadf0dad320f8ebbab0cc0de074c80787e10d04114ee58cd2262b9acaad&mpshare=1&scene=1&srcid=1114dbluU4Yeh99l5I0JU0XZ##)

![](.evernote-content/13769267-E78B-4D2B-B6F7-3298A2DE3E1B/6E0CEA45-D938-42E9-9612-9D838BAC4A67.gif)

AWS 没有出售中国业务。我们将全心承诺，确保中国客户继续收到 AWS 业界领先的云服务。中国法律不允许外国公司拥有或运营特定技术、提供云服务。为此，为了遵守中国法律， AWS 向光环新网——AWS 的长期中国合作伙伴和 AWS 中国（北京）区域的服务运营商——出售特定的物理基础设施资产。AWS 继续在全球拥有 AWS 服务的知识产权。我们对在中国拥有这一重要的业务，以及它在未来若干年的增长前景都感到非常兴奋。
