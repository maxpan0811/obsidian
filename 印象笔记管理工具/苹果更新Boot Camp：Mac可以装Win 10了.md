# 苹果更新Boot Camp：Mac可以装Win 10了

---

[![](.evernote-content/489436EB-D651-4DC0-A9DE-F141BB383A1B/9D8430BC-3523-4575-8A72-BD9F7EA70798.jpg)](http://www.feng.com/iPhone/news/2015-08-13/Apple-to-update-the-Boot-Camp-Mac-can-Win-10_621555.shtml)

![](.evernote-content/489436EB-D651-4DC0-A9DE-F141BB383A1B/9D8430BC-3523-4575-8A72-BD9F7EA70798.jpg)

威锋网讯 8 月 13 日消息，苹果今天对Boot Camp 进行一次重大更新，主要是带来了对 Windows 10 的支持。这也就意味着，Mac 用户可以放心地安装并使用 Windows 10 了。需要说明的是，最新版本的 Boot Camp 仅支持 64 位版的 Windows 10，目前 OS X 10.10 Yosemite 以及仍处于测试阶段的 OS X 10.11 El Capitan 系统都可以对 Boot Camp 进行更新。

在本次更新之前，有用户尝试使用旧版 Boot Camp 安装了 Windows 10，但是却无奈地发现不兼容，而且还不得不删除了所有分区。

更新之后的 Boot Camp 带来的不仅仅是支持 Windows 10 这么简单，还加入了许多技术支持，比如 USB 3.0 接口、USB Type-C 接口、Thunderbolt 接口、SD 读卡器和 SDXC 读卡器等等，解决了此前的不兼容问题。除此之外，Boot Camp 还对苹果的 Magic Mouse、触控版在 Windows 环境中的性能进行了提升。
