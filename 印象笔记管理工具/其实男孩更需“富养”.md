“女孩要富养，男孩要穷养”，是个较为普遍的认知，这一方面是指物质上的投入不同；另一方面，也因为较之于对女孩的精心呵护，我们对男孩要求更为严格，甚至是苛责和简单、粗暴。

其实，如果了解一下男孩和女孩的先天不同，会发现比起女孩，男孩（尤其是六岁前）更应该被理解，更需要爱的富养。

今天即与您分享**如何在更加理解男孩的基础上，给予他们更多爱和关怀**。本文内容主要摘录于Steve Biddulph的畅销全球400万本的著作《Raising Boys》，他是澳洲最著名的家庭咨询师，拥有心理学、护理、社会工作和科学等专业资历，也曾被评选为澳洲年度模范父亲。

**理解男孩：一张图表读懂男孩**

![](.evernote-content/E7F68F89-5990-44B8-87B5-820A8D76DE38/6BCE8680-AA9D-4D00-AB1D-8C93120F3C84.jpg)

养育了一男一女的父母，多半会深切的体会到养儿育女大不相同，字字珠玑的教养智慧，一移到男孩身上，怎么就失灵了？随着脑科学等的进步，我们了解到男孩和女孩在生理上、大脑发育、心理上的确存在着重大差异。而了解他们，理解他们，才能更好的去爱他们——

|  |  |  |
| --- | --- | --- |
| **年龄** | **生理发展** | **状况** |
| **刚出生** | 需要大量的睾固酮刺激身体发育的男性特征，此时血液里的睾固酮含量和十二岁的男孩一样多 | * 婴儿快乐的时候，生长激素会在体内流动，直达脑部，因而成长茁壮；心情忧郁时，压力荷尔蒙（也就是皮质醇Cortiscol）会导致生长，特别是脑部发展迟缓。 |
| **3个月** | 睾固酮的浓度下降至出生时的五分之一左右，并且后来在整个学步期都维持了很低的浓度 | * 这个时期男孩和女孩在行为举止方面没什么不同。 * 如果一两岁时，妈妈因为沮丧而对孩子不理不睬，孩子会变成“悲伤的大脑”，对妈妈是否爱他感到困惑 |
| **1岁** | 男孩的左右脑连结较女孩差 | * 学习语言、沟通能力较差 |
| **3岁** |  | * 男孩比女孩更容易感到分离焦虑，觉得被遗弃而情绪封闭或暴躁易怒，甚至损害发育 |
| **4岁** | 体内的睾固酮会突然增加两倍（目前还不清楚的原因） | * 变得比较好动、爱逞强、喜欢冒险及大胆的游戏 |
| **5岁** | 睾固酮的浓度减半，刚好可以上学了！ | * 但是体内仍有足够的睾固酮，使他仍然喜欢活动、冒险和探索，但是对女孩并不特别感兴趣 |
| **6—10岁** | 5、6岁左右，男孩的大脑发育比女孩慢6-12个月。 | * 此时尚处于粗略动作协调阶段，所以想动动身体，无法安静坐好 * 而精细动作协调发展尤其迟缓，影响握笔、拿剪刀的能力 * 运用词汇的能力也比女孩迟缓，而无法准确的将自己的需求告诉老师 * 虽然仍然很爱妈妈，但是开始想要“学习当个男生”，越来越期待爸爸的注意与互动，想要和他在一起，向他学习模仿。 * 男孩一旦处于没有组织的状况，就会觉得不安和危险，而开始争斗，建立阶级地位 * 睾固酮较多的男孩精力旺盛，爱竞争、挑衅，不是领导者就是捣蛋鬼 |
| **11-13岁** | 睾固酮浓度再度急速上升；13岁时，大脑随着身体的快速成长而重新整合 | * 突然快速成长，使他们变得迷糊而混乱，容易丢三落四 |
| **14岁** | 此时的发育速度通常很快，体内的睾固酮浓度比起发育前增加了将近8倍，使他突然长高、手脚变长，促使全身的神经系统必须重组。这是开始脱离父母，进入成人世界的阶段 | * 睾固酮的浓度达到了顶点，阴毛、青春痘、强烈的性欲以及焦躁不安的情绪 * 这个年纪的男孩变得有点好辩、左立不安及情绪化，这不表示他们变坏了，而是正在发展全新的自我 * 他们渴望新鲜事物、想扮演英雄，想发光发热 |
| **20岁** | 荷尔蒙会稳定下来，虽然睾固酮浓度仍然很高，但是身体已经适应了，所以不再那么叛逆，也比较能够控制勃起的情况。 | * 充满强烈的创造力、热爱竞争，以及追求成绩和保护别人的欲望 |

备注：睾固酮不仅是一种成长荷尔蒙，也会影响情绪和精力，使男孩精力旺盛打闹不休，它会促使大脑特定区域的发育，并减缓其他区域的成长速度，会使人长出更多肌肉、减少脂肪，也会让人秃头、性情暴躁。据研究，睾固酮会影响大脑，让男孩比较在意地位和竞争。

**尊重男孩：更适当的教养**

在了解了男孩之后，要更有耐心的给予他们更适切的教养——

**1、引导他们释放旺盛的精力**

**![](.evernote-content/E7F68F89-5990-44B8-87B5-820A8D76DE38/D370EDAE-536D-47CA-9463-7436B38489FD.jpg)**

有些男孩他们不是变成班上的英雄，就是变成十足的捣蛋鬼。这些男孩其实具有领导力潜能，如果老师和家长能够和这种男孩成为朋友，引导他们的精力往好的方面发展，他在学校就会表现优异。如果忽略甚至否定这个男孩，那么他的自尊心可能就会建立在攻击成人上，许多问题也接踵而至。

运动无疑是让男孩释放过剩的精力的好办法，运动还能给男孩带来很大好处，例如归属感、增强自信、促进健康；运动可以让男孩与其他男性更亲密；集体运动让男性互相表达关爱；运动也是个学习生命的过程——在运动场上寻找到胜不骄、败不馁、团队精神、全力以赴等人生的价值观。

面对男孩好斗的问题，父母也可以教导男孩在加入游戏或者解决纷争时，以较好的方式取代打架，例如：

*如何从别人脸上的表情判断对方的感觉*

*如何交朋友，以及加入游戏或对话*

*如何判断自己的身体发出的信号，例如知道自己正在生气，必须离开当下的环境*

**2、**花更多的心思提高他们的沟通能力****

![](.evernote-content/E7F68F89-5990-44B8-87B5-820A8D76DE38/23745A88-0BF5-4EE3-BD69-21B0DE7AE326.jpg)

男孩较之女孩落后的语言能力，是很多家长比较发愁的。据神经学家的Jenny·Harssty医生的研究发现，想要避免孩子发生学习和语言问题，可以采取许多方法加以协助。只要孩子在正确的年龄开始学习，大脑就会有非常好的反应。从婴儿时期，就可以开始帮助男孩学习更好的沟通技巧，以让他上学之后拥有更好的阅读、书写和说话能力。

下面这三个方法，可以让男孩和女孩都获益，但对于男孩来说也是一种预防的步骤，因为如果我们没有从旁协助，他们就比较容易在语言方面出现困难：

**循序渐进的教他说话**

在孩子处于某个语言学习的阶段时，对他说下一个阶段的话，比如——

* 对牙牙学语的婴儿，重复他们可能想要表达的单字。当他说“牙、亚”并指着他的玩具鸭子时，你就说“鸭！宝宝的鸭！”，这样他就很快会说鸭了。
* 当幼儿说出一个词汇，例如“牛奶”时，你就可以加上相关字串例如“牛奶瓶”，这样就可以帮助他组合词汇
* 孩子会说出两三个词汇时，就可以开始藉由模仿你而说出整个句子。例如他说“宝宝卡车！”你可以回答“宝宝想要一辆卡车么？宝宝的卡车在这里！”等等。

**把握机会对孩子解释周遭事物**

孩子的大脑藉由对话成长，多利用旅行、做家事、散步、购物等和孩子相处的时间，聊天、指东西给他们看，回答他们的问题。对话对于孩子大脑的影响，比起往后任何昂贵的教育都有效。但切记不要说得太过头，让小孩因为听不懂而觉得无聊。

**孩子还小时，就开始念书给他们听**

即使才一岁，你们也可以一起享受阅读的乐趣，特别是那种有押韵和重复句式的书。孩子有了最喜欢的故事时，你可以玩“预测”游戏：例如“然后小猫会……？”，让你的孩子接着说“喵喵”。给他趣味，进而启发他的思考。

**3、帮他们做好准备**

![](.evernote-content/E7F68F89-5990-44B8-87B5-820A8D76DE38/19D8DA20-A601-49B7-9449-DC046EC9AC86.jpg)

在上学问题上，很多科学家认为，男孩不适宜过早上学。因为在五六岁左右，也就是孩子真正上学的时候，男孩的大脑发育比女孩慢了六到十二个月，其中**所谓的精细动作协调发展的尤其迟缓**，这是因为他们仍然处于粗略动作协调的发展阶段，手臂、腿部和身体肌肉的神经还在发育，总会忍不住想要动一动身体，因此无法安静坐好，在粗略动作协调发展完成之后，他们才会开始学习精细动作协调技巧。女孩则相反，她们的大脑直接发展手指的精细动作，因此在弹簧床垫上蹦跳伸展肢体、打篮球或者游泳方面经常需要协助。

男孩的**另一项发展迟缓的是运用词汇的能力**，导致他们无法准确的将自己的需求告诉老师、在课堂上回答问题等。

当然，每个孩子都是独立的个案，不应该以“几岁了”一概而论，**而是“准备好了么**？”男孩先天的智力发展较女孩慢，现今学校的运作模式容易对男孩造成伤害，作为父母而言，应该让孩子做好准备再入学，同时多鼓励男孩，不要让他觉得自己很差劲。

**爱护男孩：给予他们亲密的爱**

![](.evernote-content/E7F68F89-5990-44B8-87B5-820A8D76DE38/1F7BDD69-DA91-425A-8862-E4B51EA8DF22.jpg)

旧有的观念让大人对待男孩通常比较严厉，更喜欢拥抱女孩，较少和男婴说话，甚至经常因为调皮而打男孩。而男孩的需求完全相反——他们其实需要更多的温柔、笑容以及身体的亲密接触，尤其是在他们出生到六岁的启蒙阶段。**男孩需要学习的第一堂课更是亲密，由信赖、温暖、趣味和善良表现出来的亲密。**

和女孩相比，男孩学的比较慢，不太擅长社交，所以更特别需要非常了解他们的、有时间且能保持心情愉快的家人的帮助。

**男孩从妈妈的身上学习爱人**，他们将妈妈视为第一个亲密和爱的典范与对象，妈妈在教导和谈话时展现的乐趣，有助于男孩的大脑发展语言功能，也能让他更容易与人相处。

除了妈妈的贴身爱护之外，很多研究显示孩子**和爸爸玩耍的方式更有活力**。得到爸爸疼爱，经常和爸爸一起玩的男孩，会和爸爸比较亲密，想要学习模仿，如此一来，和其他男性的相处就更从容自在了。

当男孩抓到蜥蜴或做了一个泥土派时

妈妈赞赏他的成就，爸爸瘙他痒，跟他玩摔跤

在他生病时，读故事给他听，给予他安慰

唯有藉由触摸、眼神交会、笑声和游戏，孩子才能感受到被疼爱

他会了解人是既有趣又温柔的

更进而打开了一个更广阔的世界

> **卢克点评时间**…♥

如今大学入学新生中，女生已多过男生了。也许现在的教育方式，特别是知识教育方面，更适合女生心智发展的节奏，但作为父母，我们依然可以给予男孩子更多的关爱和合适的培养。男孩子更爱冒险，更爱运动，更爱用身体来直接感知周遭世界，更爱机械运动逻辑思维跳跃冥想。男孩要成为坚强，高贵，上进和有爱心的男人，父母一定要跳出侧重知识教育的评价体系，充分给予他们渴望的爱与鼓励，让他们充满信心地走自己的路。关爱我们的男孩，从出生开始。
