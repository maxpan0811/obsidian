# <span>管理会沟通内容</span>
- <span>给客户账期的标准和决策权</span>
- <span>涉及大客户的销售政策决策</span>
- <span>新媒体业务需要的营销支持和产品支持由集团市场销售部来决策</span>
- <span>携程渠道伙伴参加管理会相关议题</span>
  - <span>潘的：</span>
    - <span>海长在单团的售卖支持（自组班/会务团，售卖活动-会销/沙龙，报价效率、与渠道的协作路径）；</span>
    - <span>门店非标品的产品需求（单团&amp;私家团），渠道的推荐办法分享/周三好价日活动的样板分享</span>
  - <span>何总</span>
    - <span>对于携程平台我的意见主要3点</span>
      - <span>1，海长各区域线上整体维持今年水平，低于50%的以50%为目标。</span>
      - <span>2，线上重点加大用携程国旅名义贴牌自营投放，翔龙品牌不做特别运营（roi相对低，正常排序获取自然流量）。</span>
      - <span>3，提升线下渠道份额～特别是贴牌商品。主要目的在重点区域（北广深）门店份额不低于50%，华东不低于60%。</span>
    - <span>携程平台数据（25/01/01-09/13出发）</span>
      - <span>海外跟团（含半自助，含日韩和东南亚），直采占比平台（线上+线下）23%（+华程34%），其中线上46%（+华程51%）、线下13%（+华程26%）</span>
      - <span>海长（海外长线含欧洲），直采占比平台26%（+华程43%），其中线上46%（+华程53%）、线下17%（+华程38%）</span>
      - <span>欧洲，直采占比平台23%（+华程46%），其中线上43%（+华程55%）、线下15%（+华程42%）。</span>
  - <span>郭总</span>
    - <span>郭总补充意见～平衡收益与携程占比的想法我理解。有几点要重点考虑：</span>
      - <span>1、线下占比要提升，包括海长和海短，</span>
      - <span>2、线下不一定要贴牌，可用携程国旅；</span>
      - <span>3、二三四线城市门店，尤其华东；</span>
      - <span>4、欧洲线上占比还要提升，这个我们承诺过，不能食言～后边考虑怎么降低平台佣金率；</span>
      - <span>5、和渠道共推小团（私家团/定制/拼小团）和单团，设定共同目标。</span>
    - <span>这个是我刚回给郭总的</span>
      - <span>1，贴牌自营产品要盈利～靠削减班期库存来提人档是个伪命题，关键点还是得通过加大线下渠道才是可持续的。</span>
      - <span>2，用携程国旅名义线下收客减少贴牌费～已经从定制开始，散拼团后面和张力要达成一致，贴携程国旅的产品也要线上线下同价，不过这段时间可能比较敏感，我想法不说先做；</span>
      - <span>3，在携程渠道欧洲占比持续提升应该是大家共识，不过可能60～65%占比是天花板，再强行提高有点难或不划算。</span>
      - <span>4，拼小团、私家团这次管理会和渠道要达成共识</span>
