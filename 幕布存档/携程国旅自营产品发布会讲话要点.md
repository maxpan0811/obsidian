# <span>尊敬的徐波总、各位携程渠道的小伙伴，下午好（鞠躬），欢迎大家百忙之中拨冗莅临我们此次的携程国旅自营产品发布会现场。</span>
# <span>我先自我介绍一下：我是潘渤，目前分管华程集团的全国销售业务，以及北京、成都、武汉、厦门等分公司的日常业务管理工作。我是老携程了，07年就进入携程集团工作，2014年收购华远之后就到了华远/华程工作，之前在携程时一直在做自营产品的业务，所以对自营产品也是有一定经验，也很有感情。我应该算半个四川人吧，因为我爱人是四川绵阳人，所以今年开始分管成都公司的管理工作，也是很有缘分了。</span>
# <span>此次携程国旅自营产品落地西南，我认为是我们华程集团扩大在西南区域业务布局的一个重要战略机遇。</span>
- <span>首先，携程国旅自营产品一直代表着中国旅游行业领先的产品品质和服务标准，携程最早是从自营产品开始做的，和友商不同，很多友商是从代理产品开始做，是因为当时市场上找不到符合携程品质标准的产品，所以我们就自己做，创造和推广了很多新的产品概念和服务标准，比如自由行、私家团、拼小团、六大服务保障、退一赔一等。</span>
- <span>其次，携程国旅自营产品是为携程渠道（线上+线下）独家生产的产品，在携程渠道以外是买不到的，为携程渠道伙伴建立了稳固的护城河，大的零售商做到一定规模必须要有自己渠道专属的产品，才能有差异化的利润，类似京东的京东京造，携程国旅自营就是携程渠道的“京东京造”。四川、重庆有不少的联盟在做自己的包团，但还是有硬损的风险，携程国旅自营产品相当于携程出面来操盘做包团，是四个省的携程上千家门店一起卖，这个产品丰富度和成团概率就会大大的提升，可以有更多的携程伙伴无风险参与。</span>
- <span>同时，携程国旅自营产品对内是控价的，合同价格必须严格遵守携程规定的统一卖价，保证大家的收益。</span>
- <span>最后，之前自营产品在西南无法快速落地，是受限于我们携程国旅自营团队的人手和资源，但现在并入华程后，华程在西南原有的团队和资源就能保证我们自营产品快速落地，实现本地响应，现在是最好的发展时机。</span>
# <span>祝愿26年携程国旅自营产品在西南的业务能取得长足进步，各位伙伴从中都能收益多多，祝大家生意兴隆，团团大卖。</span>
