# <span>产品布局</span>
- <span>对标蝌蚪和寰宇飞扬</span>
  - <span>产品相仿，成本按照30人定价</span>
  - <span>结算价格持平</span>
  - <span>导游政策相同，利用我们已有导游队伍的优势，要求全国每个有带团导游都为公司每年至少带2个这类高性价比团。</span>
# <span>渠道布局</span>
- <span>各地瞄准蝌蚪和寰宇飞扬现有的大客户，直接用价格去切</span>
- <span>下半年从河南、山东、陕西等我们销售团队和渠道关系比较到位的省份开始打，磨合产品和队伍，明年再全面铺开。</span>
# <span>预期收益</span>
- <span>初期有个人档逐步提高的过程，以半年为限把整个产品链条打通，实现产品毛利不亏。</span>
# <span>人力投入</span>
- <span>按照人均60万/月营收的标准招聘销售，需要多少招多少</span>
# <span>所需政策支持</span>
- <span>2024年销售、销售主管、产品操作、产品主管、机票、签证都按照人头提奖，收一个人按照现有提奖标准打8折，达成每月任务后发100%。相关奖金由集团发放到北京项目团队统一下发。</span>
- <span>前期因人档不足造成的产品毛利损失单独记账，不纳入北京公司今年的考核体系，等后续赚钱后再逐步抵冲。</span>
