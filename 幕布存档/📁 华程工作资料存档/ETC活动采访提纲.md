# <span>Q1</span><span>，经过</span><span>2020</span><span>年的一场疫情，旅行社出境业务停摆已久，原有的出境团队面临着重组或关闭，在此状况下，一旦出境市场复苏</span><span>, </span><span>出境业务又将如何重新出发？ （三位嘉宾）</span>
- <span>疫情后旅行社的兼并集中效应可能会更强。</span>
- <span>在出境游开放初期，政府可能不会马上全面放开出境旅游，短期内可能会要求部分大社先行先试，并以团队游为主，以保证出行游客的安全（疫情、排外）。</span>
- <span>欧美国家因为科技和资本的实力，疫苗普及进度可能会很快，而且因为社会信息化程度相对较高，对旅游过程的透明化管理可能较为有利。排除政治因素的话，有可能是第一批开放旅游的。</span>
# <span>Q2</span><span>， 疫后中国消费者的旅游形态正在朝向小规模、定制化的旅游方式转变，加之疫情加速旅游行业数字化的转型，之前以旅游团为主流的出境生态会有哪些可预见性的蜕变？（例如：市场，采购、渠道、直销，推广、产品等环节，可选择性答）</span><span>               </span><span>（众信、华远）</span>
- <span>产品：</span>
  - <span>从产品形态而言，客人是选择跟团或自己安排出游的根本原因还在于是否需要旅游服务。夫妻两人自己出行可能就自己安排了，但如果要拖家带口出行可能就需要专业机构提供服务，因为不想因为旅游细节疏漏导致假期泡汤、心情很糟糕。携程的客户订单里有很多这种情况。</span>
  - <span>从市场经济的角度而言，大家都是从事自己的专业工作赚钱，然后交换别人的产品或服务，不会每个人自己去种粮食、做鞋，旅游产品也没道理例外。社会上有个说法是旅游行业门槛低，但其实目前市场上客人有好的旅游体验的旅游产品是很稀缺的，很多有支付能力的客人要购买好的旅游产品，可靠的品牌选择还是太少。我观察到很多中高端消费者选择自己安排出行，是因为实在找不到合适的旅游产品，又不愿在旅行过程中受气，所以才选择自己安排。这部分需求完全是可以通过专业的产品设计和服务来予以满足的。</span>
  - <span>所以我认为跟团游的品类会长期存在，只是要真正能够做到物有所值，客人有好的甚至超出预期的旅游体验，而不是仅仅满足于没有投诉。</span>
- <span>推广：旅游是在售卖一种梦想和美好生活，所以在旅游推广中内容的传播我认为是未来工作中需要重点加强的。目前新社交媒体日益发达，为文字、图片、视频、消费者互动提供了很好的技术平台。我们要做的就是尽量把目的地旅游的内容原汁原味地呈现给客人，让大家有身临其境的感受，从而实现“种草”。这样的推广方式投入产出比是更高的。</span>
- <span>渠道：华程还是会坚持B to B的定位，因为中国太大了，很难有一个公司把服务覆盖到所有地区和渠道，华程还是专注于提供优质的产品和服务，服务直接消费者的工作交给我们的渠道伙伴去做。疫情后我们会重点通过IT技术为渠道伙伴赋能，让他们更省力、更有效地销售我们的产品赚钱。华程内部的ERP系统经过5年的建设已经基本成型，目前是在推动基于ERP的各种应用，比如EBK分销系统，让客户能够更准确地掌握华程产品的行程、班期、价格、库存等基本信息，未来还能很简单地一键转发进行推广、一键查询签证进度等等。</span>
- <span>服务：旅游业虽然一直高喊服务第一，但很多时候对旅游服务的细节标准化做得并不够，也不能实时倾听消费者的反馈和意见，这些都是信息技术大有可为的地方。华程的“行程助手”小程序用于获取消费者行程中的实时意见反馈已经用了几年了，未来会进一步加强这方面的尝试。</span>
# <span>Q3</span><span>， 假设市场重启，出境组团业务恢复，出境旅游走向全面恢复达到疫前水平，市场复苏进程应该会经过哪些阶段？（马蜂窝）</span>
# <span>Q4， 出境游复苏的进程中，您认为最大的三个挑战是什么，请排序选择 :  1,未来的不确定性， 2， 原有业务团队的流失， 3，市场营销的蜕变，  4，消费者需求的变化， 5， 业务模式的转化和调整   6， 海外地接生态的改变， 7，传统销售渠道的改变，  8，其他 ， 请补充  （三位嘉宾）  </span>
- <span>原有业务团队的流失：但由于疫情后旅行社兼并的进程可能会加快，飞鹰骨干所以也有机会找到一些更优秀的骨干。</span>
- <span>市场营销的变化：疫情中大家习惯于网上购物或者娱乐，这对后续的营销模式提出了新的要求，如何有效利用新媒体平台与消费者沟通，这对企业经营者的技术敏感度和企业信息化建设提出了很高的要求。</span>
- <span>传统销售渠道的改变：这次疫情对传统门店的打击非常大，疫情结束后应该会有复苏，但肯定会有大的变化。一些新的购买渠道如直播、一些新的异业渠道会快速成长，这就要求我们旅行社能快速把握市场的变化，与这些新的渠道建立联系，把握他们的产品需求，从而占领市场。</span>
# <span>Q5</span><span>， 伴随着全球疫苗接种，您认为欧洲出境业务恢复会比之前的预期提前吗？是什么时候？ （三位嘉宾）</span>
- <span>按照目前打疫苗的进度，乐观估计在2021年Q4就有可能开放欧洲业务。但目前中欧之间的政治关系带来了很大的不确定性。</span>
