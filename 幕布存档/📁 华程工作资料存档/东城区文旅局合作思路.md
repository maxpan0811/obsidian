# 合作目标
- 8月份与东城区文旅局签署战略合作协议。
- [ ] 加入“故宫以东”产品品牌推广伙伴计划，对接东城区文旅局所有的中高端旅游资源，最好能获取东城区文旅局的独家旅游资源支持。
- 与文旅局沟通清楚华程公司的独特优势，以后各类年度旅游推广活动邀请华程一起参与。
# 华程公司的独特优势
- 华程有全国性的分公司和办事处布局，这是其他国内产品批发商所不具备的，因为华程有出境游产品的支撑，整体营收规模较大。
- 华程的销售渠道覆盖全面，在线上渠道和线下渠道都有较大优势。我们在携程、途牛和同程等OTA渠道都是最大的欧洲产品合作伙伴，同时在飞猪、蚂蜂窝和去哪儿平台也有自己的旗舰店。线下我们全国有6000家以上的零售代理伙伴。
- 华程延续携程集团的旅游产品管理基因，有独立的产品研发岗位，以产品创新作为公司业务进步的重要动力。
- 华程通过技术手段全面管理服务品质，行程助手100%记录每个客人的出游意见反馈，后台实时数据分析，改进我们的服务水平。
- [ ] 华程有较强的品牌运作和目的地资源推广能力，熟悉新媒体推广渠道和传统媒体推广渠道，同时品牌推广能够与销售能力相结合，做到叫好又叫座。（以最近的武夷山产品为例，协助武夷山市政府在全国范围推广武夷山目的地，已经开通运营北京——武夷山航线，将要开通深圳——武夷山航线和上海——武夷山专列。在抖抖音、快手上为武夷山产品直播带货。）
- 有携程集团的大力支持，重要项目可以获得携程的战略支持背书和网络平台的流量推广支持。
# 我们的需求和建议
- 建议文旅局能给出全年的营销推广方案，我们可以提早了解、提早规划配合，实现更好的推广效果。
- 希望文旅局这边能把属地的优质旅游资源和商户信息共享给我们，帮我们牵线搭桥，实现更好的异业合作。
- 为了更好地落实推广效果，我们希望和文旅局的市场代理机构直接沟通合作，提高沟通效率，同时也降低文旅局的推广成本。
- 我们愿意参与文旅局的年度主推产品系列的设计与推广，利用华程广泛的代理渠道，为当地招徕更多的优质消费者。
