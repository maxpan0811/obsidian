# <span>众信悠哉零售和优耐德签约是扣4个点</span>
- <span>2025年华程北京和众信门店签约是2%扣点+3万系统使用费，2025年北京众信门店指标是3000万。</span>
- <span>2025年华程华北区域签约是2%扣点+11万系统使用费（按1100万的1%预收，做不到指标同比例退还）</span>
# <span>每个众信门店签约后要签优耐德/竹园的销售指标，任务一年100w起，年底加返1.5%，越高越好，自己选择指标营业额，最高有个门店签到1200w。（举例：这个门店预计自己一年能完成300w，但跟优耐德竹园签约了100w，只有100w是按照1.5%，剩下200w是按照0.5%。如果这个门店签约300w，完成300w，直接就是300w按照1.5%返）</span>
# <span>每周一下午门店必须来人总部开会，不允许请假，请假就撤店（很生硬的语气）。下午门店例会主要总结上周工作和优耐德自营产品推广的事。如果门店上周卖了优耐德和竹园以外的欧洲产品，就要说明原因：是优耐德和竹园没有相应的产品线、团期还是价格不合适，人人过关，每个店长压力都很大，轻易不敢卖别家的产品。</span>
- <span>我在外地渠道了解了一下，外地众信不敢这么做，只有北京这么做。这种做法应该是参考了之前众信直营门店的管理方式，因为在北京优耐德和竹园覆盖全部的出境目的地，市占率也很高，门店靠他们产品能赚钱，所以愿意接受这种管理。</span>
- <span>北京众信这么做的效果很明显，基本北京众信门店90%的欧洲产量都给到了优耐德和竹园。</span>
- <span>众信这种做法也有利于推广他们自有品牌“众信智造”，因为服从总部统一指挥，便于聚量成团。众信门店的包团成团率很高，且价格统一，不内部杀价。</span>
# <span>众信集团内这种生产与零售的关系在北京已经形成了一种良性循环：零售渠道帮助生产端快速打底成团，生产端就敢大量拿机位；而生产端也帮零售端生产一些差异化极大的特色产品，有助于零售端打造自身品牌，且产生独家高利润。</span>
# <span>据我了解，众信在内部管理会上讨论要引进第二家全面合作的出境供应商，认为华程无论在规模、品质和出发口岸上都能匹配众信零售成为一家全国性零售商的远景目标。因为众信零售想独立上市，所以如果只卖优耐德和竹园的产品，那优耐德和竹园的业务上限也将是众信零售的上限，毕竟两家的产品和团期就这么多。所以众信零售北京的老总和负责外阜的老总都多次跟我沟通，希望得到我们的支持。</span>
- <span>从北京公司与众信门店的合作情况来看，2023年做了400万左右（1-8月才做了100万，8月底换人负责后到年底做了300多万），2024年做了1200万（全线），2025年约定目标是3000万，目前已完成860万（全线）。</span>
# <span>个人感想</span>
- <span>我们的产品端（携程自营+华程）与渠道应进一步加强配合，零售端帮助生产端大量成团，生产端帮助零售端生产独家产品，互相之间的占比应达到25%～30%左右较好，太少没有配合，太多互相锁死了上限。</span>
- <span>携程的渠道应该有携程的特点，不是纯粹的平台，门店加盟后要接受携程渠道和产品的管理要求，未来纯粹的平台在市场上很难生存，因为产品缺少差异化，无法保证独家利润。</span>
- <span>对门店的产品销售要更有指导性和强制性，不能大家自愿卖，就像华北的trip专线成团情况极差，根本没有起到独家产品盈利的作用，生产端承担了较大的损失。关键是长此以往面对众信这种产品和渠道高度整合的对手，我们携程体系会很被动，收益也无法提升。</span>
  - <span>携程品牌在上海的影响力很大，完全可以尝试像众信在北京市场一样，对门店进行更严格的管理和指导，可以把上海市场作为试点。</span>
