# 客户分析
- [ ] 从现有数据看我们当前客户群的客户画像是什么？我们现在满足了他们的什么需求痛点？
  - 客户画像：
    - [ ] 45~60岁
    - 对价格比较敏感，有较充足的个人时间，可以避峰出行。
    - 平均单价：10000~15000元
  - 已满足的需求痛点：性价比、合格的专业服务。
- 我们未来期望新增的客户群画像是什么？与现有客户群主要的区别是什么？他们的需求痛点是什么？
  - 客户画像：有孩子的中产阶级家庭
    - 35岁~45岁
    - <span class="bold">孩子在5~15岁之间尚未成年，家中父母健在，对旅游服务有强烈需求。同时愿意为服务付费。</span>
    - 双职工，有钱没时间
    - 对产品设计和品质有自己的要求
    - [ ] 性价比是必要条件
  - 待满足的需求痛点：产品的品质感、服务的细致化。
# 华程的盈利模式：
- [ ] 垄断优势资源
- 在产品设计有独到的经验和想法
- [ ] 导游的专业服务和标准化品质管理
- 和境外旅游局、购物店的独家合作关系和协议折扣
- 把以上环节完全打通的运营能力
# 营销策略
- 渠道策略：
  - <span class="bold">努力建立与维系和大中型公司的直接旅游产品采购关系，比如MICE的客户。</span>
  - 可以考虑通过给这些大中型公司提供旅游券折扣，鼓励其采购我们的旅游券作为员工福利发给员工，然后员工到华程来采购旅游产品。
  - 我们的产品定制与服务能力是我们作为优秀旅行社的核心价值，是我们长久的业务护城河。
- [ ] 营销传播核心内容：
  - 优势资源的细节拍摄，最好能我们自己去拍出独家内容；
  - 对产品设计细节的思路阐述
  - 优秀导游的人设打造
  - 我司与携程的品牌关联，携程品牌对我司的背书
  - <span class="bold">反思：目前的网红拍摄策略是否适合我们希望瞄准的有孩子的中产阶级家庭？他们是否更关注产品和服务的细节？</span>
- 营销传播手段：
  - 传播策略：感觉目前我们是通过网红的影响力来带动我们品牌，应该投入费用也不少，但是否与我们期望传播的核心内容相关，是否具有长期积累竞争力的效果，值得评估。
  - [ ] 针对新媒体渠道（公众号、微博、抖音、Vlog等等）的评估标准
    - 粉丝的数量和增长速度
    - 粉丝的互动数据：阅读、点赞、转发。
  - 目前华程抖音公众号运营的反思：
    - 目前总计有721个关注者，但关注者的第一屏、第二屏看下来全部是公司员工。
    - 最好的两个视频分别有182个和163个“喜欢”，4个和1个“评论”，8个和2个“分享”。“评论”的内容都比较简单。视频围绕的核心是网红，对我们的资源和产品的独家卖点基本没有涉及。
  - 河南批发商品优行导游李欣的抖音号运营情况分析：
    - 有30万+的关注者，有340万个“喜欢”。三个置顶视频浏览量分别是140万次、43.7万次和1300次。内容分别是造型奇特的乞讨者、悬崖上的列车和自拍。
    - “造型奇特的乞讨者”视频有140万个喜欢，20479多个评论，4.6万次分享。同时李欣在评论区有不少互动。
    - [ ] “悬崖上的列车”视频有4.3.7万个喜欢，4359个评论，17000多次分享。李欣也有不少在评论区的互动。
  - 对比思考和建议：
    - 内容采集不一定要是那种大投入、非常专业的视频，重点是要有心、有趣。
    - 和观看者的互动很重要。
    - 李欣大量的关注者估计是从带团客人里来的，我们之前用“行程助手”做点评其实也有这个目的，也积累了大量客人的微信号，但现在都没用上。抖音要增加更多关注者其实也能采取类似做法。比如我们做些旅行注意事项的说明小视频，放在抖音里，每个客人出行前都让他们去关注、自己观看，慢慢就可以积累不少消费者，同时也不容易引起同业反感。
