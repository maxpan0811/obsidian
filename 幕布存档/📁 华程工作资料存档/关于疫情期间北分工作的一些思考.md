# 对于未来一段时间（今年下半年及明年）你公司、事业部业务趋势判断；
- [ ] 2020年Q3：北京疫情会逐步减缓，但估计政府会比较谨慎，采取观望策略。要求北京人只能在北京市内旅游。
- [ ] 2020年Q4：疫情防控常态化，开始开放国内跨省游业务。
- [ ] 2021年Q1：欧洲疫情逐步消失，但在疫苗正式出来之前，政府应该不会冒风险开放出境游，或者小范围开放出境游，但要求出境游客回来后必须隔离14天。
- [ ] 2021年Q2：预计中国新冠疫苗会开始上市，初期因为产量较小，只能小范围注射，政府可能会小范围开放出境游并要求出游的客人必须全部注射疫苗，并在回国后持续观察疫苗的效果。
- [ ] 2021年Q3：中国境内大范围注射疫苗，政府同意已注射疫苗的公民参加出境游，出境游开始逐步复苏。
- [ ] 2021年Q4：出境游逐步恢复正常，但受到疫情心理创伤和经济不景气的影响，我们的出境业务量估计能恢复到2019年同期水平的50%就不错了。
# 基于业务预测，你在接下来的半年内有什么具体的行动考虑（你是老板你会怎么做）。不用务虚大道理，简明扼要。
- [ ] 出于以上预测，由于业务恢复的不确定性，我们应尽快将人员规模压缩到与业务规模相匹配的程度，这才是最安全的做法。目前北京公司人员已经从去年底的168人减到现在的72个（6/30的数字，包含所有职能部门，不含2个残疾人和3个全职导游），我认为还要进一步削减，特别是职能部门的人员。考虑到目前北分与集团都在一处办公，同时基本没有什么业务量，北分的相关职能部门其实都可以与集团的职能部门进行合并。目前HR已经合并工作，其他的行政、财务、市场、技术都可以考虑，一方面是精简了人力，另一方面也可以提高北分的相关职能的工作绩效，这两年看下来北分人员的态度和能力是不如集团的。应考虑进一步精简到50人以下。
- [ ] 在人员精简的同时，2020年Q3以北京周边游和华致酒行的合作为主，但估计周边游空间不大，即使有人头也是没什么毛利的，不考虑大规模投入人力去做周边游。华致酒行的合作有一定想象空间，争取月营业额能做到5万~10万元人民币。
- [ ] 2020年Q4开放跨省游后，将以武夷山产品为主打，带动三亚和桂林产品的售卖，争取月人头能达到1500人左右，月营业额在300万左右。
- [ ] 2021年Q1应该还是以武夷山+三亚/桂林为主，因为三亚有春节旺季，而武夷山机位又是必须消化的，争取月人头能达到2000人左右，月营业额在400万左右。
- [ ] 2021年Q2会有少量欧洲游，但应该还是以国内产品为主，争取月人头能达到2500人左右，月营业额在500万左右。
- [ ] 2021年Q3预计会有一定量的欧洲游，预估欧洲游的月人头数可以达到300人左右，营业额在300万左右；预估国内游（武夷山产品继续经营）的月人头数在3000人左右，营业额达到600万左右。
- [ ] 2021年Q4欧洲游开始开放，但也开始进入欧洲游的淡季，预估平均月人头数在400人左右，营业额在400万左右。国内游（武夷山产品继续经营）的月人头数在3000人左右，营业额达到600万。
- [ ] 以上最重要的步骤是进一步削减人员编制，控制成本，涉及旅游业务的预估都受限于国家的政策。
