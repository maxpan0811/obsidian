# <span>问题</span><span>1</span><span>：通过此次灰度工作坊的培训得到了什么启发和帮助？</span>
- <span>这次跟曹老师沟通最大的收获就是通过做面向亲子的知识类课程以突破国内游的定价上限。</span>
  - <span>曹老师举的是自己家去香港玩的例子：所有去香港的机票、酒店、用车、门票都是曹老师爱人自己去各处预订的，完全不需要旅行社服务。但当他们想带孩子去参观博物馆时，他们就想找一个优秀的导游来给他们讲解，后来花了好几百找了一个导游，这其中就是旅行社提供人员服务的空间。曹老师说自己平时吃穿都很节约，但给孩子报班上万都是不眨眼的，这跟我平时观察到中产阶级家庭在培养孩子上很舍得投资的情况是一致的。</span>
  - <span>我这大半年做国内游最大的痛苦就是产品没有定价权，因为所有资源高度雷同，价格都是基本透明的，销售渠道也是无孔不入的，我们完全没有产品分销的空间。同时上游优质旅游资源也是有限的，而这些优质旅游资源也都已经被各种利益方瓜分完毕了，后来者需要花很多时间成本才能进入，进入后的交易条件也只能以销量累积来逐步改善。</span>
  - <span>通过在旅游产品中增加知识类课程，我们可以把很多旅游资源变废为宝，甚至无中生有创造出产品来。举例来说，上周我和一个清华建筑系校友在聊产品，他自己创业开了一个“百造学堂”，教孩子们通过建筑模型搭建来学习力学、文化和历史，我家儿子在上海百造已经学了三年，很有收获。他就建议我们可以带孩子去潭柘寺上现场课，教孩子对潭柘寺进行现场测绘、临摹，有场地的话还可以自己亲手搭建模型。这种产品对那些有志于孩子素质教育的高层次家长来说是很有吸引力的。一个半天的潭柘寺游玩+课程卖个499元完全是有机会的，因为现在在一线大城市周末出去遛半天娃花个500块是少不了的，但这个课程给孩子开眼界、增长知识带来的收获，肯定远大于出去吃吃喝喝半天的收获。潭柘寺的门票我印象中没几个钱，如果我们只做潭柘寺一日游基本赚不了什么钱，肯定不得不加入购物环节以降低成本与其他国内游批发商竞争，但我们的品牌肯定做不了，这是一条死路。而且如果只做潭柘寺一日游是抓不到什么中高端客户群的，这些客户要去基本都是自驾，不会找旅行社。但通过加上课程环节，只要包装推广到位，这些中高端家庭为了孩子的成长都会趋之若鹜来关注这个产品、传播这个产品。这条路在推广过程中肯定需要付出很多努力，在达到一个经济规模前会亏钱，但这条路会越走越宽，也完全符合我们的品牌定位和调性，这些中高端消费者也将是我们未来出境旅游产品的目标客户。后面建筑类产品可以做成“小小建筑家”系列。</span>
  - <span>再以我们最近做的“小天使外交官”系列产品为例，我们1月份跟波兰大使馆合作了一场，5/9规划和葡萄牙大使馆再合作一场。我认为这有机会成为一个三赢的产品：从华程角度而言，这个产品是能够体现我们华程独特资源优势的，没有哪家国内社有我们这么多的海外资源和关系；从消费者角度来说，一些中高端家庭非常看重孩子开眼界的机会，能够有机会和外国大使馆的大使或大使夫人面对面近距离接触和互动，对他们来说是非常珍贵的机会，收499元还要抢着来，从我们跟身边一些有孩子的中高端家庭沟通下来，家长们都很看重这种开眼界的机会；从大使馆的角度来说，他们也很看重这种与中国中高端家庭接触的机会，有机会面对面去宣传自己国家原汁原味的文化和风土人情，而且是在孩子很小的时候就影响他们，这是大使馆旅游推广部门的重要业绩。事实上这个产品的最初想法就是来自波兰大使馆希望我们组织这样的活动，波兰大使夫人亲自出来接待孩子们，找了专业的摄影师拍照，还发了证书，这是波兰大使馆年度推广计划的一环。我们完全有机会从这个角度去跟各个大使馆沟通这个想法，变成一个每年度的系列保留节目，做入各大使馆的年度预算中。我们从这个产品上可能赚不到太多钱，因为每次只有20个孩子，收费覆盖了场地费用、人员费用、物料费用后可能所剩不多，但这是一个非常好提升我们品牌的产品，获取优质客户对我们品牌的认可。这类产品后面可以做成“小小外交官”系列，主要用于奖励我们的优质VIP客户。</span>
  - <span>建立品牌的铁三角：“产品、渠道、推广”</span>
    - <span>三者中我认为产品是最核心的，有了好的产品，推广起来事半功倍，消费者就会自发变成传播者，特别在目前社交媒体时代，靠广告去卖产品已经很难盈利了，获客成本太高，只有通过消费者的自发传播、重复消费，产品毛利才能cover获客成本。通过和曹老师交流，我觉得我们北分未来的重点就放在研发面向中高端亲子家庭的有知识含量的产品系列，比如“小小外交官”系列、“小小建筑家”系列、“小小艺术家”系列、“小小飞行家”系列。</span>
    - <span>目前旅游专业渠道都是摆在明面上的，合作对华程来说不是问题，产品力本身强才能有更多的转化。未来重点是拓展个人旅游顾问和异业渠道，如果有好的产品，拓展渠道会事半功倍，而且渠道也不会轻易被销售带走。</span>
      - <span>目前小学每个班级都有自己的亲子群，班级家委会的妈妈往往是这些群里的意见领袖，我们可以考虑把这些家委会妈妈发展成我们的小B。我们可以不给返佣，因为返佣会让妈妈们觉得于心不安，但我们可以根据业绩积分，积分高的妈妈可以升级成VIP客户，以后有优质产品资源（例如去大使馆内部参加活动）只邀请VIP客户参加，这些妈妈为了自己的孩子能获得这种机会，肯定是很乐意帮我们进行推广的。</span>
      - <span>在推广亲子研学产品的过程中，我们必须自己亲自做一部分直客，这样才能实际了解中高端家庭对孩子教育 的需求和期望，便于我们培养种子用户，快速迭代产品。可以先从公司的历史优质客户和身边的亲朋好友开始。</span>
    - <span>目前推广的手段非常多，社交媒体如果有好的产品会很快形成“病毒式”传播的效果，关键是把我们产品的价值说清楚，把课程老师的背景包装好，所以推广的核心还是内容制作和传播，通过给客人种草以提高转换率。孩子教育问题是目前中产阶级最关心的话题之一，有很高的话题热度，优质的教育资源分享传播非常快。</span>
# <span>问题</span><span>2</span><span>：大家对工作坊这种培训形式评价如何（意见和建议）？</span>
- <span>这种请异业专家来分享的模式很有价值，可以了解其他行业的一些标杆做法，从外行业专家的角度来观察我们行业的一些问题，开开脑洞。以后有机会可以多搞。</span>
- <span>联想到我们内部的信息化转型工作：这是一个综合了生意模式、业务实操、技术趋势、流程重塑等多方面能力的工程，是没有现成榜样可以抄的。我个人认为携程历史上的很多具体做法，已经不适合目前的市场趋势和技术趋势，我们可以学习携程实事求是的管理思路，而不是照搬具体做法。商业史上照搬成功企业历史做法的案例，鲜有成功的。而我们目前的核心管理干部，或者有业务经验、或者有技术背景、或者熟悉流程运营，但没有也不可能有全才，可以告诉我们具体每一步该怎么做才是对的。在这种情况下，大家应该像这次工作坊一样，定期坐下来比较正式地讨论一下华程实现信息化转型的具体步骤和做法，从各自的角度充分发表看法，互相碰撞，形成一些新的想法和共识，这些想法和共识不一定是对的，但有助于内部快速学习、快速修正，并形成合力。</span>
