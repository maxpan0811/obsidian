# 上半年的历程和复工复产现状
- 1/20~2月底：处理订单善后事宜，业内首家旅行社承诺除签证费外全额退款。
- 3月~4月：上亿的应收款和应付款处理、尝试微商和国内游。
- 5月~7月：武夷山项目、碰到二次疫情。
- 8月：前端销售全员复工卖国内游
- 微商业务主要是给员工个人增加收入
- 国内游主打武夷山、三亚、桂林、丽江、张家界、西北、九寨沟这些主要旅游目的地。
# 未来发展预期
- 形势分析
  - [ ] 国外的疫情必须疫苗出来才能根本性缓解，预期出境游复苏要到明年下半年之后了。
  - 考虑到目前国际形势的不确定性，参与国内经济大循环有较好的前景。
  - 我们面临着一个百年一遇的历史转折时刻，危与机并存，且‘机’的希望大于‘危’。但前提是我们统一思想，全力投入，不给自己留退路，并在过程中不断修正自己。”
- 我们的对策：不等不靠，全力尝试新业务自救
  - [ ] 全面推进国内游业务，国内游业务不是欧洲游的翻版，它的颗粒度更细、业务链更长、竞争性更强。
  - 重点推进武夷山“航旅一体化”项目，与武夷山政府、厦航共同推广武夷山目的地。&nbsp;“武夷山项目组”团队，有效挖掘整合了武夷山当地机票、酒店、景点、民宿、餐饮旅游资源，研发打造特色旅游产品，还特别启用专业摄影摄像团队及自主培养的主播，让游客更直观感受独一无二的武夷山。
  - 加强内部erp系统建设，强身健体，提高效率降低成本。
