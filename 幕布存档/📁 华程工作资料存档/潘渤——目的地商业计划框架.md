# 国内游的SWOT分析
- [ ] 市场的机会：
  - [ ] 国内旅游市场基数庞大，还在逐年增长。同时目前国内跟团产品主要是中低端，中高端的团队产品竞争者较少。
  - [ ] 大量原先参加海外游的中高端客人无法成行，而每年旅游已经变成了他们某种程度上的刚需，这些客人会转向国内游，且对价格不会那么敏感。
  - [ ] 原来大量零售商是没有介入国内游的，因为考虑到国内游利润不如海外游。但现在海外游的开放遥遥无期，这些零售商为了生存会转向国内游业务，他们需要合适的国内游供应商。
  - [ ] 政府从经济复苏扩大内需和稳定就业的角度，都会考虑出台政策鼓励国内旅游的发展。
- [ ] 市场的威胁：
  - [ ] 国内游产品没有语言和签证的限制，进入门槛较低，已经存在很多中小批发商在其中厮杀，产品和服务品质偏低，利润偏低。
  - [ ] 一些原先从事海外游业务的批发商为了生存，也会纷纷转入国内游，会进一步加剧市场的竞争。
  - [ ] 疫情的控制还存在不确定性，出于人民生命安全的考虑，政府放开国内跨省旅游的时间表还不确定，高度依赖于疫情的变化，当中可能会有波折。
  - [ ] 企业员工的年假减少、收入减少。
  - [ ] 国内游目的地进入性较好，自由行比例在逐年提升，对旅行社依赖性较小。
- [ ] 我们的优势：
  - [ ] 有相对充裕的资金、成熟稳定的业务团队和较好的品牌形象，有自己积累多年的产品服务标准和erp系统流程。这些对新业务的开展都是较重要的支持因素。
  - [ ] 可以借力原有的海外产品渠道客户和消费群。
  - [ ] 携程集团有多年的国内游耕耘经验和资源积累，必要时可以借鉴和共享。
- [ ] 我们的劣势：
  - [ ] 整个业务团队不熟悉国内市场，对跨入新的业务领域可能需要思想上的统一和策略上的摸索，前期需要付出一定的努力和成本。
  - [ ] 在国内目的地资源上没有历史沉淀，需要从头开始摸索和累积。
  - [ ] 对原有国内产品的渠道接触不多，对渠道的运作模式和价值需求要从头了解。
