# <span>单项服务合同，那些需要发，那些不需要发？（桔子）</span>
- <span>法律规定不用签那就不签，签的越多越麻烦。</span>
# <span>直客合同的金额可以调整的尺度？目前直客合同的金额=订单应收。如果收的同行价，要按直客价签合同，需要调增应收再签合同，签完合同以后，正规操作的要求是销售按照直客价收款，然后提综合费用返还给客户，但是客户很多不能接受这样操作，希望能按同行价付款按直客价或者高于直客价签合同；</span><span>（桔子）</span>
- <span>我印象中目前直客主要还是同行吧，连小B也很少，为什么不能都把代理协议签上呢？同行用我们的直客合同给我们公司会带来额外的风险。</span>
- <span>针对小B需要签我们的直客合同，那客人的款是不是我们直接收？如果确定小B的操作模式是由小B代收款，我们针对小B类型就要修改我们现有的流程。如果是我们直接收款（通过小程序或者app），那就还是只能用返款的方式。</span>
# <span>什么时候发直客合同？客服接手订单后续跟进之后提的，希望我们能给个明确的发送直客合同的时间。（桔子）</span>
- <span>出发前一个自然天给消费者发直客合同。</span>
# <span>2021年1-3月份销售兄弟公司产品的利润与业绩的计算问题？拿到1-2月份销售业绩报表，销售拼兄弟公司的业务都没有利润，很多销售问什么回事，经查是各地财务都是以团队归属问题计算利润，由于拼团内销结算价到今天为止都还没完全执行到位，那这个时间段之间的业绩计算存在巨大的缺失，怎么解决？（李勇军）</span>
- <span>这个利润要算给销售，只能手工算了。</span>

# <span>产品运营中心和销委会能否合力推出主力售卖的几款产品，可行的情况下如何一起全国合力发团：产品的选定和销售的策略。（傅梦）</span>
