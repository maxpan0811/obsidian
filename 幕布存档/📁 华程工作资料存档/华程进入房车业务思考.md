# <span>房车市场容量</span>
- <span>美国房车保有量1000万辆，营地2万个。</span>
- <span>中国房车保有量10万俩，营地2000个。</span>
# <span>房车业务的价值</span>
- <span>现阶段国内游的利润都在地接和ota平台手里，要全中国各地建地接是很难规模化管理的，而且有很多地方保护主义因素。房车出游本身作为目的地，是可以形成全国规模经营的。</span>
- <span>国内自驾出游是趋势，但利润主要被资源方拿走，旅行社在其中价值贡献不大。而房车消费者买车的目的很明确，就是经常出游，而房车出游现阶段还是比较麻烦的问题，受限于道路可通过性、营地补给、车辆停放、玩法选择等困难，还是需要一部分专业服务的。</span>
- <span>房车的最大客户价值不终结于购买，而在于长期、定期推荐新的玩法，客户要不断开房车出去玩，才能发挥房车最大的价值。这样的客户粘性非常高。</span>
- <span>房车的价值链条非常长：车辆出租和选购、配件升级、托管停放、玩法推荐、教练指导、营地推荐售卖、亲子社交、导航信息。其中有些环节主要是信息分享，适合旅行社目前进入。有些是重资产，可以视发展进度而定。</span>
- <span>华程进入直客领域可以考虑通过房车这类新业务模式吸引高价值客户，而不是进入存量市场与现有供应商进行红海争夺。</span>
- <span>可以充分挖掘原有欧洲游客户资源，通过“翔龙万里行”公众号给客户推送“逛地球”品牌的房车旅游信息，两者还是相对匹配。</span>
# <span>房车价值链分析</span>
- <span>车辆出租和选购</span>
  - <span>在“逛地球”公众号和小程序上，投放各房车厂商整车的产品新闻稿和公关软文，收取广告费。</span>
  - <span>有兴趣体验房车的新客户可以在“逛地球”小程序里租房车出游，参与我们组织的周末活动。</span>
  - <span>长期可以争取与房车零售商合作销售，获得车辆销售返佣。</span>
- <span>配件升级</span>
  - <span>在“逛地球”公众号和小程序上，投放各房车厂商和房车配件商的产品新闻稿和公关软文，收取广告费。</span>
  - <span>收集客户关于配件的各种升级需求，反馈给合作厂商。</span>
- <span>托管停放</span>
  - <span>托管停放服务本质上还是一个近郊营地项目，就是在各大都市的郊区地带建设专门供城市公寓客户日常停放房车的营地，设置于高速出入口处，消费者开自家小车到达托管地后，停下小车再换房车出游</span>
  - <span>有些家庭顾虑购买房车后无处停放，影响采购意愿。汽车厂商可以考虑购买这类托管停放服务，用于赠送给购买房车的客户，提高成交概率。房车厂商的支付能力强，可以快速产生收入。</span>
  - <span>托管停放处可以逐步增加房车零售商的销售和维修升级场地，收取租金。以及车商的品牌展示场地，收取广告费。</span>
  - <span>托管场地可以增加遮阳设施，提供对房车的保护，并覆盖太阳能电池板，获取电费收入，以及提升自身品牌环保形象。</span>
- <span>玩法推荐</span>
  - <span>每周三推出针对周末出游的房车玩法推荐</span>
  - <span>目标出游人群：亲子家庭</span>
    - <span>进入小学之后的孩子周末往往都有很多课外课程，担心出游影响学习。同时父母忙了一周，也需要补觉。如果采取房车出游，孩子可以很方便地在车上写作业、上网课，同时全体家人可以提前一晚住在房车上，第二天只要一人早起开车，其他人继续休息，到了目的地之后再下车游玩。</span>
  - <span>可以考虑的主题有：</span>
    - <span>天文游：</span>
      - <span>选择适合观星的郊外位置，邀请天文专家做讲座和实地观测指导，教孩子们如何观察星星。如果遇特殊的天文现象，可以不局限于周末。因为房车便于孩子就地休息，不需要来回交通，同时载重量大，适合携带较大的专业观星设备，提供充足电能，因此是比较适合的出行方式。</span>
      - <span>临港新城的上海天文馆将在2021年中开业，原定用于游客住宿的房间设施可能不允许对外商业出租，但观星最佳时间往往在半夜，所以孩子住宿会是一个痛点。</span>
    - <span>宠物游：现在很多酒店不允许宠物入住，对带宠物外出的家庭比较不友好，房车出游就不存在这个问题。多个带宠物开房车出游的家庭还可以一起进行社交活动。</span>
- <span>教练指导</span>
  - <span>旅行社可以培养一些教练，类似导游，在营地带整个房车团队一起玩。提供服务和组织营地活动，也可以给有兴趣外出景点游玩的客人提供导游服务 。</span>
- <span>营地推荐售卖：旅行社产品经理体验并推荐售卖各类房车营地。</span>
  - <span>对各个营地建立标准化的描述维度，进行横向比较。积累和更新相关营地真实数据并纳入“逛地球”小程序，供客户实时查询、下单选购。也就是把营地当酒店做。</span>
    - <span>硬件设施：智能水电桩</span>
    - <span>地理优势：靠海、靠山、靠沙漠</span>
    - <span>特色活动：篝火晚会、观星、运动拓展</span>
    - <span>光照污染程度 </span>
    - <span>周边名胜分布</span>
  - <span>对各个营地进行实地考察并写推荐报告。</span>
  - <span>可以考虑采取类似酒店佣金的生意模式。</span>
- <span>亲子社交</span>
  - <span>给愿意参与社交的客户贴标签，如父母教育水平、毕业学校、孩子年龄、兴趣爱好、房车车型等，通过算法帮助客户寻找兴趣相投的孩子玩伴。</span>
- <span>导航信息</span>
  - <span>提供房车可通过道路信息，主要是道路限高评估。</span>
  - <span>提供房车可临时停放的露天停车场信息。</span>
  - <span>沿途可观赏的景点和景区</span>
  - <span>沿途柴油加油点</span>

# <span>资源借力</span>
- <span>营地销售可接入携程酒店板块，作为差异化住宿的选择之一。</span>
- <span>道路限高、露天停车场、柴油加油点信息、周边景区等数据可以与地图供应商合作，作为他们的数据来源。</span>
# <span>初步业务切入的建议</span>
- <span>我们可以先从营地推荐售卖入手，类似销售酒店。定期收集、更新国内营地的真实信息，建立营地评价维度和打分标准、规范数据格式，用“逛地球”小程序面向房车客户销售，获取客户和收入。同时可以拿这些营地真实数据与房车厂商合作，帮助他们更好地推进房车售卖，也帮我们小程序导流客户。</span>
- <span>先挖掘原有“翔龙万里行”的欧洲客源，通过公众号推送玩法信息，鼓励老客户尝试、体验房车生活方式，获取租车收入。</span>
- <span>从房车观星游、宠物游入手，联系天文专家资源，吸引房车亲子客户，扩大新的客户群。</span>
- <span>客户群扩大到一定程度后，可以考虑推进亲子社交，组织兴趣相投的亲子家庭一起出游，形成更大的客户粘性。</span>
- <span>培养房车营地教练队伍，鼓励现有导游队伍转型，建立长期队伍优势。</span>
# <span>资源投入</span>
- <span>初期放1个产品经理、1个内容撰写人员即可。产品经理先把北京周边的100多个房车营地全部跑一遍，据说大多数都已改做它用，从中筛选出仍在经营的营地，撰写推荐软文，就可以开始业务尝试了。</span>
