# <span>针对刚刚过去的中秋国庆长假，出现了“几亿人出行的火爆市场与门庭冷落的旅业现状”，有人说这个冰火两重天景象印证了基于大小交通与住宿门票的旅游产品早已淘汰，您怎么看？</span>
- <span>这个说法要区分不同渠道看：</span>
  - <span>线上ota基本已经恢复8成以上，</span>
  - <span>但线下门店开工率只有3成左右。</span>
- <span>要区分不同批发商看：</span>
  - <span>出境批发商转型国内面临产品、品牌和渠道的重新定位与磨合，确实十一收客有限。</span>
  - <span>原有国内批发商据我们了解复苏很快，已经有8月、9月单月实现盈利的情况。</span>
- <span>大量优质的出境消费者转入国内市场，但因为国内产品单一化、同质化太高，很难满足到这批消费者的需求，消费者只能自己安排。</span>
# <span>除了个性化定制线路、行程规划顾问、个性化导服等定制化服务，您觉得现在消费者对旅游的真正需求是什么？</span>
- <span>还是要想办法去抓消费者的需求痛点。比方说现在大量出行的自由行客人有支付能力，但很多是因为找不到合适的产品和服务，不得不自己花时间去安排，本来这些时间可以去做他的本职工作和休息的，而且过程中客人往往因为有些痛点事先没考虑到而影响了自己的行程体验。</span>
  - <span>三段式一日游</span>
# <span>结合今天“旅游业的供给侧改革”的话题，您觉得旅游产品提供者应该做哪些改变来满足如今消费者的需求？</span>
- <span>国内游的竞争主体发生了变化，一些大的出境企业开始加入到国内游的业务中，他们的企业管理水平、产品设计思路、对品牌和服务的重视、资金实力都有助于提高国内游的整体水准。</span>
- <span>在这种情况下，这些大的旅行社应该、也有资源真正花时间去研究消费者的需求，开发出符合不同客群需求的新产品，并通过自己的实力去长期推广这些优质新产品，还可以深入投资国内游的上游产业链，从根本上改变国内游的竞争环境和格局。</span>
  - <span>国内游的资源获取比较容易，资源上要形成垄断差价很难，还是要从产品设计、营销推广、服务体验角度去提高旅行社自身的竞争能力。</span>
  - <span>国内游的渠道相比出境游更广泛，资源采购更便利，所以要自己主动去寻找合适的渠道配合：电视台和电台、异业伙伴。不能坐等收客。</span>
# <span>疫情影响下，未来产品端在供应链哪些环节上会发生重大变化？</span>
- <span>疫情对整个供应链的影响主要还是体现在提供国内游终端产品的竞争主体（旅行社）发生了较大变化，一些大的品牌出境社加入国内游竞争后，会加剧市场竞争力度，推动新的变化。</span>
- <span>这些出境社资金实力雄厚，融资渠道通畅，很可能会投资国内的上游产业链，整合到自己的价值链中，提高自己的竞争优势和壁垒。</span>
- <span>国内资源方其实基本都在扁平化运营了，因为不存在语言和空间上的距离障碍。疫情后海外的资源方还是很难突破这两点的限制，选择合适的国内大型旅行社作为合作伙伴，负责国内业务运营可能还是比较简便易行的切入中国市场的方式。</span>
# <span>如何平衡这些变化带来的目的地服务与成本控制的关系？</span>
- <span>成本控制当然是企业永恒的话题，但前提是哪些服务形式可以提高消费者体验度和满意度，这是我们需要优先思考的，在满足这个前提下的控制成本才是有意义的。比如“24小时管家”服务，这个服务提高了客人满意度，又会带来客人二次消费的机会，就不简单是成本的问题。</span>
- <span>国内游因为语言和空间上没有太多障碍，各类资源选择丰富，旅行社应该更好地思考如何设计产品、整合资源，减少客人的资源寻找费力度，以更经济的成本提供给客人更多的体验机会（旅拍中的无人机摄影），让客人逐步养成反复消费的习惯。</span>
- <span>哪些服务是自己的核心价值，值得自己投入资源培养能力；哪些资源不是自己的核心价值，与合作伙伴合作以获得更低的成本和更灵活的选择。这都是自己要思考清楚的。</span>
# <span>供应链重构会给旅行社带来哪些新的机会嘛？</span>
- <span>国内内供应链的这种重构，对上游资源方应该是一种契机，但对旅行社本身而言是一个巨大的挑战：原有国内游旅行社需要面临更强的新进对手和更惨烈的竞争；原有的出境游旅行社要面临如何快速适应国内市场的打法，找到一条可持续的国内游发展道路。</span>
- <span>国内游市场的今天也是未来短途出境游的明天，长途出境游的后天。所以能在国内市场打个漂亮仗，就能建立起未来在出境游市场竞争的公司核心能力。</span>
