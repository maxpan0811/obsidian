# <span>2025年业务提升目标</span>
- <span>欧洲35000人，营收65350万，毛利3266万</span>
# <span>业务策略</span>
- <span>营销渠道</span>
  - <span>携程（以携程渠道为抓手，因为有精确数据反馈）</span>
    - <span>携程线上渠道</span>
      - <span>携程线上市占率2024年底做到20%+，2025年中做到40%+。</span>
      - <span>携程线上市占率从2023年底的6%+增长到目前的13%+，但与U-family还是差距巨大</span>
      - <span>携程线上渠道尝试为消费者提供IM服务</span>
        - <span>竹园IM转化率7%</span>
    - <span>携程线下门店</span>
      - <span>每天统计携程门店的询单数据</span>
      - <span>每月根据携程门店实际销售数据反馈，核实头部门店的询单率和市占率</span>
      - <span>华北携程门店市占率2024年底做到30%，2025年中做到50%。</span>
    - <span>携程旅顾</span>
      - <span>携程核心旅顾市占率2024年底做到30%，2025年中做到50%。</span>
        - <span>7月预订口径已经超过优耐德，与竹园持平</span>
  - <span>北京市场主要客户容量及排名</span>
  - <span>华北惠品团渠道覆盖</span>
- <span>产品策略</span>
  - <span>进一步完善产品结构布局</span>
    - <span>布局惠品，增加客户覆盖和黏性，视市场情况完成15000-20000人，约占整体产量比例的40%。</span>
    - <span>常规产品各板块打造华程的专项爆款。选品-高密度班期覆盖-前期偏低价引导-口碑跟进。保持西欧和非西欧各50%的占比，各非西欧板块产量争取都追上优耐德或竹园的其中一家。</span>
    - <span>西欧一国、两国深度产品持续提升，人无我有，人有我优。全年做到西欧产量的50%。</span>
    - <span>研发各板块跨界混搭产品，对标优耐德，弥补空白。</span>
    - <span>配合集团推动欧洲邮轮产品，与优耐德进行本地市场竞争。</span>
    - <span>开始各板块私家团产品的研发，争取在2025年打下有一个好的基础。</span>
  - <span>季节性产品配合集团整体，月月有新花、季季有新花。一定程度赶上获得引领者标签。</span>
  - <span>团队意识和水平提升，怀揣对市场对客人的敬畏之心，出品注重细节，从心认同。</span>
# <span>资源支持</span>
- <span>系统支持</span>
  - <span>希望能每月及时拿到华北区每个携程门店的具体欧洲销售数据，方便评估我们在每个门店的开拓进度</span>
- <span>政策支持</span>
  - <span>人员</span>
    - <span>按照优耐德+竹园的人员总数规划未来两年的人员招聘安排</span>
  - <span>考核（直接下属）</span>
    - <span>销售管理岗（许炜川、陈海兵、薛玖林、李涛）</span>
      - <span>营收任务完成的保底年薪</span>
      - <span>所负责区域有正经营利润后的净利提成</span>
      - <span>单团任务完成后的阶梯提奖</span>
    - <span>产品管理岗（刘圣）</span>
      - <span>全年营收任务完成后的保底年薪</span>
      - <span>全年产品毛利目标完成后的额外提奖</span>
      - <span>全年单团毛利指标完成后额外获得新业务拓展奖励红包</span>
