# <span>竞争对手的选择</span>
- <span>门当户对</span>
  - <span>品牌：一线品牌、二线品牌</span>
    - <span>U-family还是蝌蚪、寰宇飞扬？</span>
  - <span>业务模式：直客、批发、定制</span>
    - <span>U-family还是凯撒？</span>
  - <span>市场定位：大众商品、奢侈商品</span>
    - <span>同一家公司：大众和奥迪</span>
  - <span>做事底线</span>
    - <span>蝌蚪不接受硬件条件之外的投诉</span>
# <span>竞对产品线的选择标准</span>
- <span>目的地</span>
- <span>价格带：价差在10%以内</span>
  - <span>不同价格带代表不同客群</span>
# <span>竞对的维度</span>
- <span>行</span>
  - <span>航司产品力</span>
    - <span>机型</span>
    - <span>时刻</span>
    - <span>起降机场</span>
    - <span>是否转机</span>
- <span>住</span>
  - <span>酒店</span>
    - <span>星级与钻级</span>
    - <span>顶级酒店品牌</span>
    - <span>酒店位置</span>
    - <span>房间朝向</span>
- <span>游</span>
  - <span>行程</span>
    - <span>拉车时间</span>
    - <span>是否配官导</span>
- <span>吃</span>
  - <span>评价</span>
    - <span>米其林</span>
    - <span>黑珍珠</span>
- <span>购</span>
  - <span>枪店</span>
  - <span>对产品成本的影响</span>
  - <span>航班进出点</span>
- <span>娱</span>
  - <span>买人头</span>
# <span>价值量化测算</span>
- <span>把卖点折算成零售价和成本，最好的卖点是成本很低但消费者愿意为此付出高价</span>
  - <span>“欲管理必先测量”——梁建章</span>
