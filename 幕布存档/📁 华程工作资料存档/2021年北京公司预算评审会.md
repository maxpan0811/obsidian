# <span>展示预算评估表</span>
- <span>员工保留55人（含北京日韩部、留职停薪人员）</span>
- <span>最大固定成本104.5万；最大变动费用45.5万。总计150万。</span>
- <span>预算分三种情况</span>
  - <span>预算1——全体趴下留职停薪的成本（含固定成本，变动费用为0）：365万。目标毛利0，预计亏损365万。</span>
  - <span>预算2——保留17名员工全职上班的成本（按最大固定成本+变动费用）：625万。目标毛利200万，预计亏损425万。</span>
  - <span>预算3——44名员工全职上班的成本（按最大固定成本+变动费用）：998万。目标毛利565万，预计亏损433万。</span>
  - <span>结论：先选择预算2，逐步朝预算3努力。</span>

# <span>本人的2021年年度目标、工作任务和时间进度</span>
- <span>面临的问题1：酒店资源是国内业务的重要盈利点。</span>
  - <span>解决方案：学习和掌握酒店套餐产品在各大平台酒店频道的运营和销售技巧。</span>
  - <span>完成时间：2020年Q1</span>
- <span>面临的问题2：部分偏远地区的景点资源有意向让旅游企业承包、运营。</span>
  - <span>解决方案：接触并了解相关资源的具体情况，初步摸索低风险的运营模式。</span>
  - <span>完成时间：2020年Q2</span>
- <span>面临的问题3：从产品端如何支持“逛地球”直客品牌发展，并协助在北京地区推广该品牌。</span>
  - <span>解决方案：参与讨论、确定“逛地球”品牌的品牌调性、价值定位、产品标准、服务标准，以此来指导北京公司产品部门的资源采购和产品设计。</span>
  - <span>完成时间：2020年Q4</span>
