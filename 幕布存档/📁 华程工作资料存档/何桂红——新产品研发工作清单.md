# <span>产品生产前（市场调研）</span>
- <span>目的地市场容量数据</span>
  - <span>现有存量及增长趋势</span>
  - <span>主要客源区域</span>
- <span>市场主流产品TOP5</span>
  - <span>供应商、团型、人次、晚数、钻级、大交通、价格带等</span>
- <span>核心地接社</span>
  - <span>优选至少3家核心供应商，规模优势（批发）、资源优势（定制/入境/mice）</span>
- <span>swot 优劣势分析</span>
- <span>投入&amp;产出计划</span>
  - <span>投入：项目组人工成本</span>
  - <span>产出：年度人头、毛利计划，按照产品线细化到每月</span>
# <span>产品生产中（基于目标客群需求研发、竞比后出产品）</span>
- <span>锁定竞争对手</span>
  - <span>线上：携程自营/牛人专线</span>
- <span>熟记竞争对手产品体系、主要产品及卖点、价格带</span>
- <span>构建我司产品体系：团型/流量占比/季度占比/每款产品使命</span>
- <span>市场主流产品：在对手基础上做差异化、微创和优化（价格or性价比优势是重点！）</span>
- <span>敌无我有：基于大数据生产的新兴产品，如半自助游、研学团、夏冬令营、礼佛、旅拍</span>
- <span>市场当下产品类型，你全覆盖了吗？国内市场竞争环境更加激烈，产品单一很难立足</span>
  - <span>自由行：机+酒，酒店套餐</span>
  - <span>跟团游：出发地、目的地参团</span>
  - <span>私家团：一单一团，不与陌上人同行</span>
  - <span>半自助：大交通+住宿为主</span>
- <span>你学会做产品差异化了吗？差异化必须基于目标客群需求</span>
  - <span>资源差异：酒店/餐食/景点/车/体验活动</span>
  - <span>服务差异：管家服务、导游、产品标准（钻级标准）</span>
- <span>运用工具：</span>
  - <span>产品竞比模板</span>
  - <span>产品定价模板</span>
  - <span>产品思维导图</span>
# <span>产品生产后（产品售卖）</span>
- <span>产品推广</span>
  - <span>坚持定时推送，比如早7-8点推送重点产品，晚10点以前推送产品合集；结合目的地知识、热点、资源等推送效果更佳。</span><span class="tag">@销售效果更佳</span>
- <span>及时赞美</span>
  - <span>有咨询、有反馈、有订单的销售都值得赞美，微信群里要建立与销售的互动</span>
- <span>销售培训+考核+结果公示</span>
- <span>重点销售深入沟通</span>
  - <span>一对一，高频率</span>
  - <span>沟通当地游客偏好，产品改进建议</span>
- <span>产品调整优化</span>
  - <span>除了从销售端获取市场信息，最重要的是产品经理主动获取市场信息，以周为单位关注竞争对手动态、目的地市场动态形成自己相对准确的判断（大方向必须自己把握，别人的意见只是参考）</span>
# <span>总结</span>
- <span>什么样的供应链生产出什么类型的产品，如果产品丰富度不够花时间多找几家供应商</span>
- <span>国内市场竞争环境更加激烈，单一的和同质化产品难以立足，建议产品类型尽可能丰富，重点放在私家团/半自助，重点关注爸妈、亲子客群。</span>
- <span>国内游我们大部分人都没做过，没有成功经验可借鉴，俗话说黑猫白猫抓到老鼠就是好猫，以结果/业绩为导向，想尽一切办法去达成，一般来说，结果不会很差！结果差一般是因为你努力不够！</span>
