# 门店管理模块
- 门店基础信息：
  - 门店基本信息，必填；
  - [ ] 门店地理位置数据，必填；
- <span>以门店为维度录入销售的上门拜访日志或电话沟通记录，并能输出为报表</span>
- 以门店的数量增长作为渠道下沉推进进度的评估维度，能输出报表。
# 销售管理模块
- 销售到门店的拜访次数可以通过门店现场打卡方式记录，并输出报表；
- 销售的门店拜访次数能够与该门店产量做关联分析，分析拜访的有效率；
- 销售能够一键转发系统中的产品彩页到微信群，并能追踪到该销售所发彩页的打开率；
- 客户能够一键转发系统中的产品彩页给消费者，并自动转换成客户的logo和联系人信息，统计客户的转发率；
# 客户服务模块
- 产品服务：客户能够自助查询/下载目的地产品行程、团期、实时余位、彩页、出团通知书、退改签条款；
- 订单服务：客户能够自助下单、提交发票需求、在线付款、查询订单确认状态、查看签证材料完整情况、查看签证出签进度；
- 财务服务：查询在我司的定金余额、返利积累情况；
- 渠道奖励活动服务：查询当前的销售进度、查询渠道销售奖励达成情况、查看自身的客户等级；
# 营销管理模块
- 直接推送产品营销合集/产品简版行程给门店；
- 直接推送产品特价优惠信息给门店；
- 直接推送最新的渠道销售奖励活动内容给门店；
- 整合订单中的客人年龄、性别等信息供客户画像分析；
