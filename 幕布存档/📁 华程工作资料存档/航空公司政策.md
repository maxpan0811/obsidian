# <span>西安MU</span>
- <span>淡季一押15%  申请位置支付，成团率80%   二押50% 提前30天支付  成团率90%  三押100%（含税）提前7天支付  成团率100%</span>
- <span>旺季季一押30%  申请位置支付，成团率90%   二押80% 提前30天支付  成团率100%  三押100%（含税）提前7天支付  成团率100%</span>
# <span>西安HU</span>
- <span>一、报价：全年均价1000 元/人，团队配额内支持同海航其他航线组合，价格按照1000+组合其他航点系列团运价1/2RT计算。</span>
- <span>二、座位数量：70人/航班（包括去程和回程）</span>
- <span>三、系列团追位配额：每个订单允许80%客座率内原价增加5人（不低于散客单程价格）。</span>
- <span>四、出团率：80% 。</span>
- <span>五、合作保证金：100 万元。</span>
- <span>六、票损扣罚标准：第一次考核扣罚：30天外取消按照300元/人扣罚，30天内按月度核算成团率80%，不足使用率的部分按照1000元/人扣罚。。</span>
- <span>                           第二次考核扣罚：半年拉通核算成团率60%，未达60%成团率二次扣罚1000元/人（临时团计入考核）。</span>
- <span>七、国内段联运：米兰航线的联运点和政策享有至少不低于其正常航线的联运政策（根据渠道重点联运地区，区分淡旺季定价）。</span>
- <span>八、临时团价差：所有单团与大押金团队价差淡季不小于600元/人，平季、旺季不小于1000元/人。</span>
