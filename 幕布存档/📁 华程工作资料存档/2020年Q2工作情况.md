# 与骨干一对一谈话次数
- 北分的骨干（副总3人、飞鹰学员6人、高研班学员1人、票务1人）在Q2是要求每天到公司出勤上班，每周有会议，所以一对一沟通较少。
# 本单元活动次数/销售竞赛
- 由于北京疫情限制，Q2未组织团建和外部交流。开过一次武夷山销售动员会。大家积极性很高。
- 针对微商业务有做每天的内部销售排名。Q3开始设定最低达成指标。
# 其他高价值工作
- 应收款工作：从3月底的949万余额降到6月底的446万，减少了500万，收回了54%，还不含途牛的150万小额贷款，算上的话是收回了70%。
- 人员优化：从春节前的135人左右优化到目前的72人，这次又报了7人的职能部门优化名额，在与集团职能部门负责人沟通中。
- 国内游工作推进：北分负责的国内游产品（三亚、桂林、武夷山、北京周边地区）在北京第二波疫情之前就已经做好了各项准备工作，随时可以出街售卖。虽然国内游开放比我个人预期要早，但我们已经做好了准备。
- 微商工作：积极推进微商工作，销售利润排名全国第二，并与华致酒行签订了合作协议，开始售卖酒类产品。
