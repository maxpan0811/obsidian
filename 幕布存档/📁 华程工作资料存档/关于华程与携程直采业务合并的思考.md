# <span>关于合并后的渠道管理和销售管理问题</span>
- <span>上周五根据郭总的要求，我去携程跟老肖沟通了一下关于两边合并后一些渠道管理和销售管理的想法，我理解老肖的想法是把所有在携程线下渠道的直采业务和华程业务都放在现有的携程直采销售团队来管理，非携程渠道的华程线下业务还是由华程销售团队来管理。同时华程的所有线上业务也都交给携程直采的线上团队来负责。</span>
  - <span>但这会带来一个问题：如果华程的产品在携程渠道和非携程渠道发生政策冲突时怎么办，零售渠道间同品牌产品的价格竞争是必然会发生的。那样的话很可能在区域分公司层面这个冲突是无法解决的，因为两边有各自的汇报线。华程这边是通过分公司体系往上报，直采这边是通过自己的管理体系往上报，而在区域层面两边是互不统属的，可能必须要到集团的负责人层面才能解决，这样的处理效率是很低的，很容易引起渠道不满。</span>
  - <span>一种解决方案是直采团队把所有的渠道都接过去管理，比如非欧业务在各地的销售渠道全部由直采团队负责，包括携程渠道和非携程渠道，这也就意味着直采的销售团队要跨出现有携程渠道，重新建立一套目前华程既有的渠道分销能力，这个可能就意味着大量的人力和时间的重复投入，毕竟华程做线下分销已经30年了，整个团队已经具备了完整的线下分销业务管理经验和人才储备。做全国渠道的人才要求和只做单一携程渠道的人才要求是差异很大的，单一零售渠道的渠道政策设计很简单，不需要考虑分销、二批、大零售商、社交媒体等多种渠道业态，也不用考虑这些渠道业态之间的冲突可能性，但做全国渠道分销政策设计是必然要考虑这些情况的，从一开始就要做好政策设计，否则后面这些渠道政策是无法落实的。不是说直采的销售团队伙伴不能学习这些，而是时间成本太高，而且没必要重新发明一遍轮子。</span>
  - <span>从目前直采销售同事与携程渠道的配合来看，我的观察是两边还是处于一种集团内部门间合作的状态，是一种兄弟部门公事公办的态度，而不是我们通常的市场经济条件下甲方和乙方的合作状态，这从上次杭州管理会上的沟通也能明显看出。我觉得这种态度在华东这种我们产品力较强的区域还问题不大，但走出华东后可能就会面临和携程渠道亲密度不够，携程渠道主动配合的意愿不高等困境，长期来看是很难把生意做好的，从目前华东以外区域携程渠道的直采产品市占率不高就可见一斑，所以也并不存在所谓“携程直采销售团队在携程渠道具备独特优势”的说法。</span>
  - <span>另外从人员管理的角度而言，一个销售团队如果只做携程渠道是会有问题的，长期来看晋升通道是有限的，不利于培养操盘能力全面的业务人才。华程的线下渠道耕耘多年，销售可以从小客户做起，逐步负责更大的客户，可以做携程渠道也可以做非携程渠道，可以做线下客户也可以做各类电商客户（非携程），从基层销售可以一直有机会做到区域销售负责人甚至销售分公司老总，这种晋升通道是更有利于吸引、培养和保留好的业务人才的。</span>
  - <span>那线上渠道也是会有类似的问题，面临着与线下渠道（携程门店和非携程门店）的政策冲突协调，以及长期的人才晋升通道问题。如果两边要合并，看起来还是必须以华程为主，原因如下：</span>
    - <span>本来何总和我的设想是把华东携程线上业务全部交给携程直采同事们来负责，因为直采同事更有线上业务运营经验并在内部更容易获取相关业务决策数据。但从目前几个月实际的运营来看，上海华程的线上运营还是由原来华程的电商同事负责，携程直采同事无暇顾及，同时也没办法进入携程内部获取更多的运营数据。这样综合考虑的话，目前华程上海的携程线上运营交给携程直采团队负责并没有达到我们初期的目的，反而把管理体系搞复杂了（携程的团队和华程的电商负责人都在管华程上海的线上业务）。</span>
    - <span>同时线上业务还存在一个同程和途牛的问题，如果以携程直采的线上团队同事来管整体华程集团的线上业务，就会面临一个与同程、途牛业务团队重新建立信任关系的过程（携程直采同事从上到下与同程、途牛从来没有过业务交集）。而且同程、途牛的业务团队如果了解到我们是以携程直采线上团队来统管我们的线上业务后，也会面临一个担心我们新团队无法摆正自己位置从而影响后续配合的问题，毕竟以前是这么多年的竞争对手。</span>
    - <span>从具体的线上业务模式来说，携程直采的运营是属于真正面向C端的运营，而华程的电商其实是在做线上代理业务（虽然会有一些产品上线工作，但本质是靠代理销售），如果携程直采业务的线上运营同事精力顾得过来，把华程在携程线上的具体运营工作交给携程直采同事是最合适的，然后携程线上业务作为一块面向C端的工作，再纳入到华程整体的电商运营管理中来。</span>
  - <span>综上，考虑到全国范围内各类销售渠道的复杂性及政策的多样性、华程原有线上线下大客户对携程直采销售团队的信任问题以及销售人才的晋升通道问题，我建议携程直采业务现有经营的携程销售渠道（线上和线下）都还是并入现有的华程销售体系来，因为携程直采业务的销售团队只是面向某一特定零售渠道，现有经验和能力不足以掌控整个华程销售体系，如果要等携程直采团队来新建整个分销管理能力，这个时间成本就太高了：</span>
    - <span>首先，把所有华程+携程直采的线下销售渠道都放在华程销售团队来管理，充分利用华程30年来线下销售的管理经验和既有的庞大销售团队。</span>
    - <span>线上业务可以一段时间内保留华程和携程直采线上团队的独立运营，如果携程直采同事精力忙得过来就把华程的携程线上业务接过去运营，所有电商业务由集团直管，集团的电商管理团队以华程原有负责人为主，便于和同程、途牛以及其他电商平台保持既有合作关系，未来各区域如果有合适的线上业务管理人才后可以逐步合并当地的携程线上业务和其他非携程的电商业务在一起管理。</span>
# <span>关于非欧产品全国各口岸城市落地的问题</span>
- <span>老肖的意见是把全国非欧业务都作为独立事业部，负责全国的产品研发和各地的销售工作，由目前的上海携程直采同事负责最终决策，我理解就是目前携程直采的总部集中化决策架构。目前携程直采的几位负责人除了邹雨我比较了解，林志卓和胡文煜我接触并不多。但从携程直采业务这么多年的实践来看，始终都走不出华东，华东地区的份额一直居高不下，也就是华北区域还有一些量，这个原因还是值得我们思考的。从我负责北京携程度假部门、做北京“一号项目”的经验来看，这种全国性集中运营管理的决策机制听起来是很有道理的，但最大的问题是在旅游行业很难找到这种有全国操盘能力的事业部负责人。在IT行业要培养一个这样能够负责某个事业部全国操盘的负责人至少10年以上，要从基础销售岗位、产品经理、分公司负责人一路走过来，而且往往要历练过全国好几个主要市场，就有点类似我党培养高级干部一样。而旅游行业放开全国性经营都没多少年，很多干部都只是熟悉一个区域市场、一个目的地板块，也没有全国操盘的概念和能力，你强行让他去负责一个全国事业部，其实是勉为其难的，而且一旦选错了人，过程中又会压抑很多其他干部的积极性，这也就是原来携程度假反复在分公司负责经营和全国集中化运营两种做法之间来回横跳的原因。以我这么多年的实践，我认为在旅游行业里必须走本地化经营、全国协调的模式，也就是目前华程欧洲事业部的模式：各口岸欧洲的经营责任由当地负责，集团的欧洲事业部负责引导产品创新（私家团、环线游、city walk）和资源集中采购，制定产品标准和规则等，这样才能做到既发挥本地人才的积极性、与对手在本地有效竞争，同时又能保证全国产品有相当的品质标准和成本优势。</span>
- <span>那具体到非欧事业板块到某个出发口岸要怎么开始，我是建议参考如下步骤进行：</span>
  - <span>借助我们已有的分公司在当地开始招聘该业务板块所需的产品研发和销售团队人员，华程各地分公司在当地经营多年，在旅游圈内人脉很广，要充分利用当地分公司的人脉资源。</span>
  - <span>根据当地携程零售渠道的历史数据分析该目的地板块在当地市场的主流产品线分布情况：天数晚数、价格带、线路涉及主要城市、必去景点等。当地产品研发团队根据这些数据拿位子、出产品，期间集团事业部可以提供支持。</span>
  - <span>根据数据看携程渠道内哪些门店擅长销售这个目的地，定义出该目的地板块的核心门店，新销售团队根据这个核心门店数据开始按图索骥，一个个核心门店攻克，期间集团事业部可以提供支持。</span>
  - <span>每周根据核心门店的实际询单、开单数据来评估我们产品线与当地市场主流产品线的契合度，如果我们的产品设计方向与实际销售产品不符，且1、2个月没有突破，那就要尽快调整到位，不能孤芳自赏。</span>
  - <span>该目的地产品在当地携程渠道的市占率达到30%以上后，基本就能保证初期的班期成团，后续再进一步拓展到当地市场的其他主要零售渠道。</span>
- <span>综上，我的看法是不能照搬携程的总部事业部全国集中化决策的做法，华程并不具备这样的人才，其实携程也做不到这一点。还是要在主要出发口岸培养当地的业务团队，总部事业部主要起支持作用，这样才能在当地市场找到优秀人才愿意加入公司并长期留在公司服务，团队稳定了才能做好业绩。</span>
# <span>以上是我上周去跟老肖沟通后一些个人的想法，因为前期并没有接触携程直采和华程合并的工作，上周接到郭总指示后第一次去和老肖沟通，可能对很多背景情况都不了解，所以提出的建议可能会脱离实际情况，仅供领导们参考。</span>
