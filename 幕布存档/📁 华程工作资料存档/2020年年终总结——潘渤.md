# <span>全年工作概况总结</span>
- <span>从2020年3月开始积极部署和推进国内游产品，对不适应、不配合国内游业务发展的部门和人员进行了快速调整优化，把工作重点完全转移到国内游上来。疫情中</span><span>北京公司的核心业务骨干基本得到了保留：飞鹰离职1人（陈海兵）、高研班离职1人（冯鑫）。通过2020年的瘦身，把老北分的老员工进行了一次全面清理，保留的都是90后、工作积极主动、愿意承担责任的年轻人。从管理架构上来说，产品管理有何桂红、票务管理有潘莹、散拼销售管理有刘羽娇/孙泽龙、单团销售管理是徐妍/邓勇、HR管理是张建平，都是这2年多来通过了实际业务考验、在工作中表现出了各自的能力和担当的干部，而且都逐步找到了自己的合适定位。整体而言，我认为北分的团队实力通过这次疫情危机，非但没有削弱，反而打磨得更加精干、合理。同时疫情的考验就像长征，留下来的都是对我们的事业具有信念、可以长期稳定跟着公司成长的骨干。</span>
- <span>国内游的产品重点初期放在以武夷山包机为核心的长线团队产品上，但因为北京疫情的反复，以及长线团队市场竞争集中于低端市场，北京的长线产品推进效果不佳。在业务推进过程中发现北京周边游市场需求旺盛，且利润率稳定，就逐步把工作重点转移到北京周边游市场，多开发自带IP的中高端酒+X套餐产品和当地玩乐产品，取得了一定的效果。</span>
- <span>国内游的销售渠道重点初期还是延续以往线下门店+OTA渠道的做法，但疫情后线下门店开工率只有20%，销售效果很不好，人力投入大产出差。考虑到我们所拿到的一手国内产品资源的利润价差也不足以刺激线下门店来主推我们产品（门店往往有自己固定的国内游供应商或很容易找到现有国内批发商），我们确定逐步把销售重点放到面向企业大客户的单团销售和面向直客的线上电商运营（含飞猪平台），以避免门店对我们业务推进的阻碍作用。同时对线下散拼销售团队进行了调整和缩编，并充实、加强了单团销售队伍和电商销售队伍。</span>
# <span>2020年工作的得与失</span>
- <span>得</span>
  - <span>在2020年的国内游转型中，北京公司团队表现出了积极应对危机的心态和果断调整的执行力。虽然分公司历史基础差，但通过笨鸟先飞的做法，并根据分公司当地情况寻找实事求是的业务思路，取得了不逊色于上海和广州公司的国内游业绩。而且这些产品策略和渠道策略应该是具有长期积累性的，可以保证北京公司国内游部门的长期稳定发展。通过这次危机，我们看到当面临新的挑战、大家都处在同一起跑线时，北京团队是完全可以做得跟上海、广州团队一样好的。当疫情过去欧洲游重新恢复时，我们也有信心抓住新的契机，把欧洲游追上去，并相机开展中东非、东南亚等新业务。应该说这个团队信心的获取，是2020年最大的收获。</span>
- <span>失</span>
  - <span>在崇礼滑雪产品的采购上，没有积极做两手准备，而是把宝全部押在了太舞产品上，其实云顶产品当时也已经谈好了政策、出完了产品。当太舞产品出了状况再去重新和云顶合作，虽然对方还是愿意跟我们继续合作，但错过了一定的售卖时间，造成了损失。国内游市场相比出境市场更为复杂，契约精神相对较差，未来在国内资源采购中要吸取这个教训。</span>
# <span>2021年工作安排，制定提升计划，以及需要哪些支持</span>
- <span>考虑到疫情的反复可能性，2021年北京公司的产品策略还是以华北周边（含内蒙）的景酒产品为主，结合西北、西藏、新疆等可进入性较差目的地的长线团队产品，同时配合云南DMC的工作；渠道策略还是坚持面向直客大企业的单团渠道和面向直客散拼的电商渠道（含飞猪）。</span>
- <span>继续严格控制分公司人员和运营成本，保证每个人都能有产出，能尽量养活自己。国内游毛利较低，只有严格控制成本才能尽量接近盈亏平衡点。</span>
- <span>从2020年转型国内游的过程来看，是非常痛苦的。如果欧洲游还正常运行，我个人认为我们的国内游是没有机会做起来的，也没有负责人愿意去承担这个重任。那就带来一个问题：当出境游逐步恢复时，国内游要如何长期保持做下去，国内游部门人员要如何有效激励。旅游行业靠天吃饭，境外市场再出现大规模变动的可能性永远存在，我们如果把国内游做起来了，就不可能放弃，即使利润和人效长期低于出境也不能放弃。这个问题我觉得是需要在2021年充分思考并找到解决方案的。</span>
- <span>鉴于目前欧美国家疫苗的开发速度和采购能力，欧美可能会先于全球其他除中国以外的地区实现市场复苏，可能2021年Q4就会少量开放欧美出境游。在这种情况下，中国政府如果要开放出境游，很可能会选择少数实力领先的大型旅行社进行试点，通过这些大型旅行社的实力来保证出境游客人在国外的防疫安全。同时因为团队游便于控制行程，政府可能会先开放团队游，而不是个人出境旅游。基于这种可能性，我们应保持与旅行社主管部门的沟通，并强调我们在疫情防控、客人安全保障方面的实力和良好历史记录，争取成为出境游开放的试点旅行社之一。我们可能要提前考虑针对疫情防控条件下的产品设计、领队管理、酒店/大巴/餐厅等旅游资源的防疫消毒规范等等，这不仅是集团欧洲产品中心的事情，也是我们分公司要结合当地市场实际、提前考虑的问题。</span>
