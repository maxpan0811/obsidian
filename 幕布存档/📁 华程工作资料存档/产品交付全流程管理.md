# <span>按渠道划分</span>
- <span>华程——OTA——消费者</span>
  - <span>关键时刻1:</span>
    - <span>接触点：华程业务人员与OTA采购经理的沟通环节</span>
    - <span>工作内容</span>
      - <span>了解OTA的采购需求</span>
      - <span>介绍华程产品的卖点</span>
      - <span>与主要竞争对手的产品对比</span>
      - <span>与有主推能力的OTA（途牛和同程）沟通主推机会</span>
    - <span>责任岗位</span>
      - <span>销售人员</span>
      - <span>产品经理</span>
    - <span>评估标准</span>
      - <span>产品线通过OTA审核顺利上线</span>
      - <span>产品线获得OTA全面主推（途牛、同程）</span>
  - <span>关键时刻2</span>
    - <span>接触点：消费者在OTA线上看到华程产品展示页面</span>
    - <span>工作内容</span>
      - <span>购买性价比高的关键字（主要是携程），获得产品呈现给消费者的机会。</span>
      - <span>将华程公司实力通过展示页面体现出来</span>
      - <span>将华程公司产品卖点和优势通过展示页面体现出来</span>
    - <span>责任岗位</span>
      - <span>产品经理</span>
      - <span>设计人员</span>
    - <span>评估标准</span>
      - <span>消费者下单购买</span>
  - <span>关键时刻3</span>
    - <span>接触点（主要针对海外游）：</span>
      - <span>出团前领队发集合通知短信，拉群</span>
      - <span>机场集合讲解旅行相关信息，完成产品经理交办的工作（收取小费等）</span>
    - <span>工作内容：</span>
      - <span>告知消费者出行注意事项和集合信息</span>
      - <span>把全团客人拉群</span>
      - <span>保证客人按时到达机场并顺利登机</span>
    - <span>责任岗位：</span>
      - <span>领队</span>
      - <span>销售</span>
      - <span>产品经理</span>
    - <span>评估标准</span>
      - <span>客人顺利登机</span>
  - <span>关键时刻4</span>
    - <span>接触点（主要针对海外游）</span>
      - <span>领队在海外旅行期间与客人全程相处</span>
      - <span>产品操作在后台通过行程助手获取客人对行程的意见反馈</span>
    - <span>工作内容</span>
      - <span>做好客人全程的旅游接待工作</span>
      - <span>告知客人可以随时通过行程助手反馈行程中的不满意问题</span>
    - <span>责任岗位</span>
      - <span>领队</span>
      - <span>操作</span>
    - <span>评估标准</span>
      - <span>客人在行程助手上的满意度100%</span>
  - <span>关键时刻5</span>
    - <span>接触点：客人在回程后的遗留问题处理，比如丢失东西、不满意投诉等等</span>
    - <span>工作内容</span>
      - <span>客人在海外遗留问题的处理</span>
      - <span>客人投诉的处理</span>
    - <span>责任岗位</span>
      - <span>领队</span>
      - <span>操作</span>
      - <span>QA</span>
    - <span>评估标准</span>
      - <span>客人的问题得到满意的解决，愿意下次继续参加我们的团</span>
