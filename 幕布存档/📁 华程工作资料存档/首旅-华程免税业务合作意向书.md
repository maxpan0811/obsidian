# <span>中国免税市场概况</span>
- 中国现有免税牌照的授予情况
  - 迄今为止国内共授予了8张免税牌照，其中中国国旅持有3张（中免、日上、海免），其他5张分别是：珠免（格力地产）、深免（深圳免税集团）、中出服（中国出国人员服务有限公司）、中侨（吉林中侨免税外汇商品有限公司）、王府井。
  - 2019年，中国国旅通过中免、日上等公司实现免税商品销售业务营收458.18亿元，同比增长37.89%，毛利率为50%。
  - [ ] 海免拥有离岛免税牌照，包括美兰机场店（2012年营业），海口市内免税店（2019年2月营业，店面2.2w平方米），博鳌市内免税店（2019年2月营业，店面4500平米），针对所有离岛旅客，凭借机票、火车票、船票就可离岛前购买。19年营收32.27亿，净利润3.45亿，20年一季度营收8.5亿，净利润1.24亿，由中国国旅负责进货运营，在国旅接手后净利率呈现上升趋势。
- 中国国旅的免税业务经营策略分析
  - 与国内领先的出境旅行社战略合作
    - 2015年中国国旅旗下的中国免税品（集团）有限责任公司与国内五家出境旅行社联手，成立了中国海外旅游投资管理有限公司，共同拓展海外旅游的零售市场。
    - 这五家旅行社分别是：国旅总社（2013年营业额超100亿）、众信旅游（2014年营收42亿、国内上市公司）、竹园国旅（2014年营收20亿）、凯撒同盛（2014年营收26亿、国内上市公司）、凤凰假期（2014年营收22亿）。可以看到，国内欧洲的前5大批发商除了华远国旅（华程国旅），都参与了与中免集团的合作。
    - 在中国海外旅游投资管理有限公司的公司介绍中提到成立该合资公司的考虑：
      - 战略意图吻合：双方对旅游+免税的创新模式一致看好，希望结成更紧密的战略资源合作。
      - 客源优势明显：5家出境旅行社的目的地遍布欧洲、东南亚、日韩，在2014年仅输出到巴黎一地的客人就达到45万人次。
      - 控制客源能力：这5家旅行社为国内市场主要的出境旅游批发商，掌握资源采购、行程设计等关键能力，可以很方便地在行程中植入零售商店，并通过自己的领队执行到位，因此能够有效地引导消费者的购买行为。
      - 持续融资能力：众信和海航凯撒为上市公司，可实现持续融资。
      - 打造新产品优势：此次合作的远期目标是以免税店为切入点控制产业链上游资源，如酒店、餐饮和景区等。合作旅行社多年来在上游与各航司保持密切关系，且经常有包机项目，在下游则具有庞大、成熟的零售分销渠道，以及丰富灵活的推广手段，可以快速提升旅游目的地知名度。
  - <span>与品牌代理商合作获取货源</span>
    - 中免集团在资源采购上采取灵活的合作策略，并不拘泥于向品牌厂商直接采购的做法，部分品牌是通过品牌代理服务商（如：拉格代尔）采购。
- 其他免税集团公司的经营动向
  - 2020年8月DFS入股深免旗下电商公司，占22%股份，为第二大股东。双方目标是借助行业利好政策实现全球与“双区”的资源共享和优势互补，运用新的技术、新的思路、新的模式、新的机制，全方位、深层次地深耕线上线下融合的新消费模式，促进资源、客源、渠道的有效协同与转化，共同为消费者提供多元化、高品质的产品与服务，营造独特舒适的购物环境，创造非同凡响的购物体验，打造“时代尚品质生活平台服务商”。
# <span>华程的优势</span>
- 华程是国内领先的出境旅游产品运营商
  - 华程出境产品遍布欧洲、日韩、东南亚和中东非，其中2019年欧洲发团人数达到35万人（含携程自营欧洲），超越其他4家欧洲批发商，位居全国第一；日本的跟团游产量在上海和北京市场也是位列第一。
  - [ ] 华程在国内有5家一级分公司（北京、上海、广州、成都、武汉），还有十余家二级分公司，服务6000余家线下零售商。同时与携程、途牛、同程等三家OTA客户保持紧密合作，2019年在三家OTA渠道的欧洲产品市占率都位居第一。
  - 华程拥有上百位专业的产品研发和操作人员，同时在20年内积累了上千位资深的欧洲专职领队，能够有效地实现产品行程和零售购物的结合，并引导游客达成销售目标。
  - 华程与海外知名零售商（如DFS、Bucherer、Lafayette）保持长期、互利的合作关系，并与LVMH等知名奢侈品集团有紧密沟通。
- <span>与携程集团的整合效应</span>
  - 华程作为携程的100%全资子公司，在财务上得到携程的大力支持，自身运营资金也非常充裕。
  - [ ] 华程与携程在旅游业务上也保持密切合作，对目的地推广等大项目会在线上和线下同时发力，互相借用对方优势资源。以海南岛为例，2020年7月海南进港航段：三亚，携程近30万、市占约41%；去哪儿13万+、市占19%；三亚合计份额约60%。海口，携程21万、市占30%；去哪儿15万，市占22%；海口合计份额52%+。这些航线资源都可以有效转化为海南离岛免税店的潜在客源。
  - 携程作为旅游网络媒体的宣传引导作用。
# <span>结语</span>
- 据前瞻产业研究院数据，2018年中国免税业销售收入达395亿元，中免以及日上的市场占有率高达82%。  在2019年6月海南省免税品有限公司51%股权宣布完成过户整合入中国国旅麾下后，中国国旅在中国免税行业更加具有垄断性的地位。
- [ ] 首旅集团下属的王府井集团股份有限公司在2020年6/9正式获得第8张免税牌照后，市场抱有极高的预期，希望能打破中国国旅在免税行业的绝对老大地位。通过以上对中国国旅的经营策略分析，我们建议首旅集团应该打通货源多元化采购、出境社客源深度合作、线上线下联合推广等多个环节，形成一个完整的价值链体系，才能实现与中国国旅集团的体系对抗，从而在竞争中实现破局。
- 从免税业务的价值环节来看（品牌采购与商务谈判、海外物流运输与报关、门店管理、营销推广、政府公关等），华程集团和携程集团在海外品牌采购与商务谈判环节有着多年的经验积累和人脉关系，在海外物流运输环节也有多年的包机经验，双方强强互补，可以帮助我们首旅的免税业务快速在海南落地。
- [ ] 华程集团愿意与首旅集团在免税业务上深度战略合作，并全力投入自身的优势资源，实现共赢。
