# <span>易做食物</span>
- <span>安格斯牛肉馅饼</span>
  - <span>200度/双面/各5分钟</span>
- <span>炒空心菜</span>
  - <span>把叶杆和叶子分开</span>
  - <span>小火炒香蒜末和小米椒</span>
  - <span>先炒叶杆，再放入叶子</span>
  - <span>多放油</span>
  - <span>大火爆炒</span>
- <span>腌制鸡翅</span>
  - <span>极美滋复合调味料</span>
  - <span>2斤鸡翅、70g调味料、70克水</span>
  - <span>保证每个鸡翅都沾上腌料</span>
  - <span>盖上保鲜膜整体颠簸一下</span>
  - <span>腌制6～8小时</span>
  - <span>鸡翅烤之前必须解冻</span>
- <span>腌制带鱼</span>
  - <span>带鱼洗净、切断、擦干</span>
  - <span>用调料腌制一天</span>
  - <span>200度12分钟</span>
  - <span>翻面，再200度12分钟</span>
- <span>小米粥</span>
  - <span>小米一盒，大米半盒</span>
  - <span>淘米洗干净</span>
  - <span>放入红薯两个</span>
- <span>油醋面</span>
  - <span>生抽</span>
  - <span>醋</span>
  - <span>花椒油</span>
  - <span>鸡油或猪油</span>
  - <span>盐、味精</span>
  - <span>娃娃菜</span>
  - <span>面汤</span>
- <span>冷冻虾</span>
  - <span>蒸，上汽后再蒸7分钟</span>
- <span>牛排</span>
  - <span>煎牛排</span>
    - <span>早上拿出来放冷藏解冻</span>
    - <span>吸去表面水分，撒上粗盐和黑椒</span>
    - <span>关火后加入黄油，融化后淋到牛排上</span>
    - <span>锅底一层油，加热到手有热度。</span>
    - <span>放入后及时移动、翻动</span>
    - <span>筷子压一下牛排感觉硬度</span>
  - <span>烤牛排</span>
    - <span>空气炸锅在200度预热5分钟</span>
    - <span>放入牛排温度调至180度烤”5分钟</span>
    - <span>将牛排翻个面，再烤5分钟。</span>
- <span>葱油饼</span>
  - <span>葱油饼两面刷油</span>
  - <span>空气炸锅200度第一次单面8分钟，翻面再5分钟</span>
- <span>水煮蛋</span>
  - <span>水开后关火，等不再冒泡</span>
  - <span>勺中放少量水，打入一个鸡蛋，慢慢贴近水面放入水中</span>
  - <span>加盖焖两分钟</span>
  - <span>开小火再煮两分钟</span>
- <span>西红柿炒蛋</span>
  - <span>西红柿</span>
    - <span>底部（无蒂处）划十字</span>
    - <span>开水里浸没泡3分钟，剥皮</span>
    - <span>切成四瓣，去蒂，再切成小块备用</span>
  - <span>鸡蛋</span>
    - <span>鸡蛋打散，加盐半勺</span>
    - <span>鸡蛋液等量食油，加热到冒烟</span>
    - <span>加入鸡蛋液，顶部结块后反转，炒成金黄色。关火，捞出。</span>
  - <span>西红柿下锅前先用葱花、蒜片炝锅。</span>
  - <span>炒西红柿，至汁水大量溢出，加入生抽和番茄汁。</span>
  - <span>加入炒好的鸡蛋一起炒，基本熟了之后最后加入盐、糖再炒两下。</span>
- <span>绿豆汤</span>
  - <span>绿豆200克（8个人），莲子50克，百合25克</span>
  - <span>用水泡1小时</span>
  - <span>用高压锅煮</span>
  - <span>起锅后放几块冰糖</span>
- [ ] 羊蝎子
  - <span>先抄水（用白水煮开）</span>
  - <span>抄完水后用清水将羊蝎子表面残余血水冲干净。</span>
  - <span>加入配好的大料包、花椒、干辣椒、姜、糖、盐、生抽、老抽，用电饭锅煮。</span>
  - <span>煮一小时左右再加入萝卜一起煮。</span>
  - <span>快起锅时放入洋葱</span>
- [ ] 炸薯条
  - <span>削皮后</span><span>切成条状</span>
  - <span>冲干净表面淀粉</span>
  - <span>用水煮开</span>
  - <span>用滤水篮盛出晾干水分</span>
  - <span>放入冰箱冷冻冻硬</span>
- <span>炸春卷</span>
  - <span>空气炸锅版本</span>
    - <span>刷油</span>
    - <span>200度预热5分钟</span>
    - <span>200度加热4分钟</span>
- <span>炸骨肉相连</span>
  - <span>200度预热5分钟</span>
  - <span>200度炸8分钟</span>
- <span>煮饺子</span>
  - <span>买饺子时摸一下袋子，看是否散装，还是冻结成大块。大块不要。</span>
  - <span>第一次煮开后，放一碗冷水再煮开。如此三次。</span>
  - <span>如果需要放蔬菜，就在第二碗冷水煮开后放入，不需要放第三碗水，煮开即可。</span>
- <span>煮汤圆</span>
  - <span>先把冷冻汤圆用冷水过一遍</span>
  - <span>锅里水煮至沸腾再放入汤圆</span>
  - <span>用锅铲推动汤圆避免粘锅</span>
  - <span>煮开后放入少量凉水，反复2～3遍</span>
  - <span>汤圆漂起后即可</span>
- <span>凉拌黄瓜</span>
  - <span>黄瓜去头去尾</span>
  - <span>黄瓜用刀拍一下，然后切成小块放到碗里</span>
  - <span>加入一勺食盐和一勺糖腌制半小时，放入冰箱</span>
  - <span>加盐、白糖、白醋、陈醋、香油、生抽、蒜末</span>
- <span>萝卜炖羊肉</span>
  - <span>羊肉买来先焯水，切成块</span>
  - <span>把羊肉块放入电压力锅，放入料酒、花椒3粒、大葱2段、姜4片、陈皮、枸杞和醋，放水漫过羊肉</span>
  - <span>羊肉在高压锅内炖熟后放汽，打开锅盖放入萝卜块继续炖。</span>
  - <span>高压锅上汽关掉电源。</span>
  - <span>起锅时加入盐和胡椒粉</span>
