# <span>例会制度</span>
- <span>分公司部门周例会</span>
  - <span>销售、产品、机票、签证、领队等五大业务部门的周情况反馈</span>
  - <span>财务和人事部门周情况反馈</span>
- <span>产品操作部周例会</span>
  - <span>复盘上周的投诉、失误</span>
- <span>机位决策周例会</span>
  - <span>二定支付节点机位是否保留及保留数量</span>
- <span>销售部门周例会</span>
  - <span>携程渠道上周未询单订单及有询单未成交订单的原因分析</span>
  - <span>省五核心大客户头部门店在华程的上单数据分析</span>
- <span>销售月度复盘会</span>
  - <span>携程头部门店开单率和市占率</span>
  - <span>省五客户头部门店在华程系统销售数据和排名</span>
- <span>产品月度复盘会</span>
  - <span>携程各产品板块、线路玩法、价格带的市占率</span>
# <span>报表制度</span>
- <span>每周携程渠道销售口径双品牌市占率（累计出发值和上周预订值）</span>
