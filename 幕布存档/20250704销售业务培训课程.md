# <span>销售管理工作的一些基本观念</span>
- <span>我们的工作目标是建立一张当地的销售网络</span>
  - <span>反例：全国到处收人</span>
- <span>渠道的推力（push）和消费者的拉力（pull）</span>
- <span>以客户为单位制定年度任务目标，而不是以销售为单位</span>
  - <span>客户渠道/产品板块容量*目标市占率</span>
    - <span>2019年北京公司英爱市场的指标制定</span>
      - <span>2018年600人+，2019年指标4800人</span>
        - <span>航线运力</span>
        - <span>旅游签证发放量</span>
        - <span>对手产量</span>
        - <span>拆分到三个出发口岸：北京、西安、青岛；拆分到不同的产品档次（高中低端）；</span>
        - <span>考虑到英国签证送签时间在一个月以上，而HU机票交一定押金是在30天之前，所以机损风险不大</span>
        - <span>目标市占率第三名（10%）</span>
    - <span>2025年北京携程线上指标的制定</span>
      - <span>华程北京2024年的携程线上产量1000万，2025年指标6700万</span>
      - <span>对手（众信系）2024年的线上产量5500万，预计对手2025年在我们的攻势下略微增长到6700万</span>
      - <span>目标程信比1：拆解到产品板块、线路玩法、线路ID，评估流量UV和转化率，找出差距加以改善</span>
  - <span>制定合理的市占率目标（经验值）</span>
    - <span>第一名：30%</span>
    - <span>第二名：20%</span>
    - <span>第三名：10%</span>
- <span>怎么评估一个渠道做得好不好：三个指标</span>
  - <span>年度任务达成率</span>
  - <span>渠道市占率</span>
  - <span>同比增长率</span>
- <span>渠道组合（Channel Mix）的概念</span>
  - <span>业务特点：成团之后很快就会收满</span>
  - <span>大客户与小客户的组合</span>
    - <span>大客户跑量，保证基础成团</span>
    - <span>小客户赚钱</span>
  - <span>线上客户与线下客户的组合</span>
    - <span>一些门店转型做线上零售商很成功</span>
  - <span>大客户产量占比控制在合理的比例</span>
- <span>好的销售要有产品意识（产品组合的概念）</span>
  - <span>客户的业绩是由一条一条的产品线产量积累而成</span>
  - <span>产品组合（Product Mix）的概念</span>
    - <span>有的产品线赚钱，有的产品线跑量</span>
  - <span>核心大客户是推动我们新产品、高端产品的主力/要有自己的大客户基本盘</span>
    - <span>西欧深度、中东非、美洲、澳洲、日韩</span>
  - <span>核心大客户的产品组合应该与公司的产品组合类似，否则主要产品的产品力就没有在该渠道充分体现出来</span>
    - <span>华程北京整体产量的客户占比</span>
    - <span>华程北京西欧产品“瑞不可挡”的客户占比</span>
    - <span>凯撒的“瑞不可挡”产品在2023年的产量</span>
# <span>到一个陌生市场怎么从零开始做销售</span>
- <span>销售业务灵魂三问：</span>
  - <span>本省前五大客户是谁？（藏宝图）</span>
  - <span>客户渠道容量（年度产量）是多少？</span>
  - <span>客户主要供应商占比是怎样的？（优耐德和竹园产品为主的客户是优质客户）</span>
- <span>具体做法：</span>
  - <span>选择合作伙伴尽量要门当户对（我们是一线批发商，不抓当地头部客户完不成指标）</span>
  - <span>优先考虑忠诚的客户，我们有实力把小客户培养成大客户</span>
  - <span>陌拜（扫街）、把客户的头部门店都摸清楚（门店开单率）</span>
  - <span>了解大客户的产品结构，找到产品空间</span>
  - <span>特殊的加返政策，与年度任务绑定</span>
  - <span>每个门店/每个大销售都培训到（大型培训会+门店一对一培训）</span>
  - <span>考察团优先邀请核心客户，和产量绑定</span>
  - <span>和客户一起做会销/沙龙，支持客户转化他的客群</span>
  - <span>一句话：有好处时想着大客户，有困难时也想着大客户</span>
# <span>把携程渠道打造为我们业务的基本盘</span>
- <span>线下渠道进一步挖掘业务空间，不满足于程信比大于1</span>
  - <span>分析头部门店差距</span>
    - <span>开单率</span>
    - <span>市占率（程信比）</span>
    - <span>上海携程线下欧洲数据举例</span>
  - <span>核实携程头部门店每周产生的订单数据</span>
    - <span>没咨询到我们销售的，属于客情关系不到位</span>
    - <span>咨询到我们销售但没拿下的，核实产品、团期、价格、航班是否有劣势</span>
    - <span>上海携程渠道日本每周订单举例</span>
- <span>线上业务抓住关键指标，找到问题</span>
  - <span>产品板块（东南西北欧）、线路玩法、主要ID</span>
  - <span>主要关注：程信比、流量UV、转化率</span>
  - <span>上海携程线上欧洲每周数据举例</span>
- <span>欧洲单团业务的提升</span>
- <span>非欧业务提升</span>
  - <span>美洲、澳洲业务不挑出发城市，2025年优先推进，目标市占率30%（2024年携程门店市占率19%）</span>
- <span>其他非欧业务的落地</span>
  - <span>以当地携程渠道的历史非欧产品销售数据作为拿位、制定产品线、定价的参考依据</span>
  - <span>和当地携程渠道紧密合作，实现产品主推，保证打底成团。</span>
  - <span>2025年上半年携程渠道日本业务数据</span>
- <span>携程旅顾的产量提升</span>
