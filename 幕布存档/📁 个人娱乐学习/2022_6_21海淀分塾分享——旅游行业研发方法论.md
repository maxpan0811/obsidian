# <span>专门研发机构的建立</span>
- <span>传统旅游企业只有op岗位</span>
- <span>携程旅游部门从建立之初（2003年）就设立专门的产品经理岗位</span>
- <span>产品经理对产品的经营结果负责，而不是只负责设计产品</span>
# <span>客户需求的确认</span>
- <span class="bold">目的地市场容量的测算</span>
  - <span class="bold">飞往该目的地的航班数量（转机、直飞）/藏宝图</span>
  - <span class="bold">高星级酒店的数量与入住率</span>
- <span>客户玩法的占比分析</span>
  - <span>客人自己夫妻俩出行可能选自由行，带孩子和老人出行会考虑团队游</span>
  - <span>购买人和使用人的不同</span>
# <span>产品设计思路</span>
- <span>行程中航空公司的选择</span>
  - <span>品牌、政策、价格和飞行网点</span>
  - <span class="bold">转机城市的旅游吸引力</span>
    - <span class="bold">暑期多用香港转机的港龙+国泰航空</span>
- <span>行程中城市的选择</span>
  - <span>团队游客人偏爱多城市串游，最好两地进出，不走回头路</span>
  - <span>自由行客人喜欢在超级大都市、海岛做一地待着深度体验</span>
- <span>行程中酒店的选择</span>
  - <span>团队游</span>
    - <span>价格</span>
    - <span>是否顺路</span>
  - <span>自由行</span>
    - <span>离地标景点的距离</span>
    - <span>是否海景房</span>
- <span>行程中景点的选择</span>
  - <span>根据我们产品经理到目的地踩线情况选择（当地人爱玩的地方）</span>
  - <span>参考之前客人的游记和点评数据</span>
  - <span>参考旅游局的年度推广计划</span>
# <span>产品采购成本控制</span>
- <span>包飞机</span>
  - <span>利：控制成本、垄断资源</span>
  - <span class="bold">弊：产品设计走形，根据资源而不是需求做产品行程设计</span>
- <span>包酒店</span>
  - <span>利：控制成本、垄断资源</span>
  - <span>弊：全年淡旺季明显，容易造成旺季大赚、淡季巨亏，对收益管理造成困难。</span>
# <span>产品运营管理</span>
- <span>线上渠道与线下渠道的销量平衡</span>
  - <span>线下渠道成本高：差旅成本、招待成本</span>
  - <span>线上店大欺客：广告投放、促销活动、被控制</span>
- <span>淡季与旺季的库存平衡</span>
  - <span>采购量往往是每天、每班固定数量，但淡旺季销量差异很大</span>
  - <span>如何在旺季有效分配库存以获得高收益、在淡季有效清库存以减少风险</span>
- <span class="bold">旺季（黄金周）的定价</span>
  - <span>不同出发日期产品的价格组合</span>
  - <span>不同进出点和转机点产品的价格组合</span>
# <span>产品的品质与服务管理</span>
- <span>神秘访客：内部人员参团以获得第一手的行程质量反馈。</span>
- <span class="bold">行程助手：客人通过行程助手工具在行程中对旅游的吃住行游购娱等关键要素进行实时点评，产品经理在国内的系统后端实时进行处理，以提高客人出游满意度和获得产品改进数据。</span>
- <span>产品质量的三个境界</span>
  - <span>产品缺陷率考核</span>
  - <span>产品满意度考核</span>
  - <span class="bold">请给我打“非常满意”（AO. Smith）、客人无欢便是过</span>
# <span>结论：</span>
- <span>在旅游行业，产品经理的岗位是对产品的需求调研、产品设计、销售推广、品质服务全流程负责，以产品顺利卖出去、客户满意归来为终点，是产品线的总经理。</span>
- <span>在产品全流程的管理中，要大量借助数据分析和技术工具来获得和判断客户的真实产品需求和满意度，从而进一步开发新产品和提升客户满意度。</span>
- <span>如果以客人从旅行中是否获得欢乐为检验我们工作质量的标准，就会发现大量的工作细节可以改进。</span>
