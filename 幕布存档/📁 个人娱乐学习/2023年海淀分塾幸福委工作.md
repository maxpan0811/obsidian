# <span>海淀分塾幸福委使命、愿景和价值观</span>
- <span>使命：让塾生在盛和塾实现关爱幸福（塾生有幸福体会，把关爱分享出去）</span>
- <span>愿景：一灯照隅,万灯照国（坚持愚直地、诚实地、认真地工作，工作的彼岸是美好的人生！</span>
- <span>价值观：真信 真学 真实践</span>
# <span>2023年工作目标</span>
- <span>对海淀分塾（2023年初为54人）80%的塾生实现走访，包含以下方式。并协助海淀分塾达成整体全年考核目标。</span>
  - <span>线上多对一走访</span>
  - <span>线下登门多对一走访</span>
  - <span>线下聚会多对多走访</span>
# <span>2023年幸福委工作办法</span>
- <span>对现有塾生进行分类</span>
  - <span>身份：经营者、高管</span>
  - <span>地域：京内、京外</span>
  - <span>想学习的知识点，后续考虑按照想学习的内容分组。</span>
    - <span>稻盛哲学</span>
    - <span>稻盛实学：会计七原则、阿米巴经营</span>
  - <span>是否续费：第一年塾生、续费塾生</span>
- <span>走访交流内容</span>
  - <span>需要沟通的问题：</span>
    - <span>塾生入塾希望在盛和塾得到什么</span>
      - <span>第一年新人：以我们介绍现有体系为主，让塾生对盛和塾学习体系有完整的了解</span>
      - <span>续费老塾生：了解他们加入盛和塾想要的是什么，在塾里这两年是否已经得到了自己想要的东西，我们要怎么更好地帮助他达成目标。</span>
    - <span>是否有机会成为幸福企业。</span>
  - <span>全面介绍2023年海淀分塾学习规划</span>
    - <span>提供并讲解2023年一整年的学习规划（纸质文件）</span>
    - <span>现场辅导新塾生如何使用“学在盛和塾”app进行每日读书打卡</span>
  - <span>Q&amp;A</span>
    - <span>为什么我们要一起学稻盛，而不是自己学？</span>
# <span>2023年幸福委考核办法</span>
- <span>主观指标</span>
- <span>客观指标</span>
  - <span>走访率80%</span>
  - <span>打卡率80%</span>
  - <span>续费率70%</span>
  - <span>线下学习参与率48%</span>
