# <span>训练主要部分</span>
- <span>胸</span>
  - <span>胸大肌</span>
    - <span>胸部整体</span>
      - <span>训练动作</span>
        - <span>宽距俯卧撑</span>
        - <span>跪姿俯卧撑</span>
        - <span>击掌俯卧撑</span>
    - <span>胸肌上部</span>
      - <span>训练动作</span>
        - <span>下斜俯卧撑</span>
        - <span>钻石俯卧撑</span>
        - <span>弹力带反手夹胸</span>
        - <span>哑铃上斜卧推</span>
          - <span>注意1：使上臂与水平面平行或者略微下沉，不要过低。</span>
          - <span>注意2：不要使哑铃垂直上下，这样会过多用到肱三头肌的力量，应该使两臂向左右分开成弧形起落。</span>
        - <span>哑铃上斜飞鸟</span>
          - <span>注意1：下降到极限时，肘关节成100～120度角。</span>
    - <span>胸肌中部</span>
      - <span>训练动作</span>
        - <span>弹力带对握夹胸</span>
        - <span>健身球哑铃飞鸟</span>
    - <span>胸肌下部</span>
      - <span>训练动作：上斜俯卧撑</span>
    - <span>胸部中缝</span>
      - <span>训练动作：</span>
        - <span>弹力带屈臂夹胸</span>
        - <span>弹力带交叉夹胸</span>
- <span>背和肩</span>
  - 
  - <span>背部整体</span>
    - <span>训练动作</span>
      - <span>正手宽距</span><span>引体向上</span>
      - <span>正手俯身</span><span>哑铃划船</span>
      - <span>颈后宽距引体向上</span>
  - <span>斜方肌</span>
    - <span>斜方肌上部</span>
      - <span>训练动作</span>
        - <span>弹力带前平举</span>
        - <span>哑铃站姿前平举</span>
        - <span>哑铃耸肩</span>
    - <span>斜方肌中部</span>
      - <span>训练动作</span>
        - <span>哑铃俯身内旋侧平举</span>
        - <span>弹力带对握划船</span>
    - <span>斜方肌中下部</span>
      - <span>训练动作</span>
        - <span>哑铃俯身外旋侧平举</span>
        - <span>正手俯身哑铃划船：3组/间隔90秒</span>
        - <span>弹力带高位下拉</span>
  - <span>菱形肌/小菱形肌</span>
    - <span>训练动作</span>
      - <span>哑铃俯身单臂划船</span>
  - <span>背阔肌</span>
    - <span>训练动作</span>
      - <span>反手窄距引体向上</span>
      - <span>颈后宽距引体向上</span>
      - <span>正手宽距引体向上</span>
      - <span>反手俯身</span><span>哑铃划船</span>
  - <span>大圆肌</span>
  - <span>肩袖四肌</span>
- <span>肩</span>
  - <span>肩部三角肌</span>
    - <span>三角肌整体</span>
      - <span>训练动作</span>
        - <span>哑铃站姿推举</span>
        - <span>哑铃阿诺推举</span>
    - <span>三角肌前束</span>
      - <span>训练动作</span>
        - <span>哑铃站姿前平举</span>
    - <span>三角肌中束</span>
      - <span>训练动作</span>
        - <span>哑铃站姿侧平举</span>
    - <span>三角肌后束</span>
      - <span>训练动作</span>
        - <span>哑铃俯身内旋侧平举</span>
- <span>臀</span>
- <span>腿</span>
