# <span>什么是“复盘”</span>
- <span>围棋里的术语</span>
- <span>复盘与总结不同，复盘很重要的部分是推演（有没有更好的做法）</span>
# <span>为什么要复盘</span>
- <span>生而知之、学而知之、困而知之（不二过）：从自己的经验学习</span>
- <span>越是创新的行业越要从自己的经验中学习</span>
# <span>复盘的方法</span>
- <span>情境重现法</span>
  - <span>信息场：我当时做了什么</span>
  - <span>思维场：我当时是怎么想的</span>
  - <span>情绪场：我当时为什么这么想</span>
- <span>关键点法则</span>
  - <span>正向关键点法：按照时间顺序，围绕事情的关键里程碑事件进行复盘</span>
  - <span>逆向关键点法：凭经验确定事情成功的关键点，围绕关键点复盘，与对手对标。</span>
    - <span>产品线覆盖率</span>
    - <span>班期覆盖率</span>
    - <span>提前上线期</span>
    - <span>成团人档</span>
# <span>复盘的三个问题</span>
- <span>现在情况如何（做得好不好）</span>
  - <span>达成率（和目标比）</span>
  - <span>市占率（和对手比）</span>
  - <span>同比/环比（和自己过往比）</span>
  - <span>用数据说话（固定一些分析维度）</span>
    - <span>营收</span>
    - <span>人头</span>
    - <span>毛利/人均毛利</span>
    - <span>平均卖价</span>
    - <span>利润率</span>
    - <span>质量数据（返修率、投诉率）</span>
    - <span>库存周转率（旅游行业是“机损”）</span>
- <span>当初是怎么决定的</span>
  - <span>目标是如何决定的</span>
    - <span>产品组合</span>
      - <span>每条产品线的产量</span>
    - <span>渠道组合</span>
      - <span>每个客户的产量贡献</span>
  - <span>目标是根据什么决定的（论据）</span>
    - <span>产品</span>
      - <span>市场容量（运力变化）</span>
      - <span>客户需求</span>
      - <span>产品生命周期</span>
    - <span>渠道</span>
      - <span>客户市占率提高</span>
      - <span>客户范围拓展</span>
      - <span>客户类型拓展</span>
  - <span>执行得如何</span>
    - <span>现状与规划的差距</span>
      - <span>一对一的比对</span>
- <span>让我们再审视一下思考的前提</span>
  - <span>市场容量变化</span>
  - <span>客户存续</span>
  - <span>目的地政策变化</span>
# <span>复盘怎么能做得好</span>
- <span>要坚持定期做：每周、每月、季度、年度，这样才能看出趋势和变化。</span>
- <span>要不断追问，不要怕伤面子，丰田5个为什么</span>
- <span>问题的三个层面：事实、逻辑、观点</span>
- <span>有话要直说，但要好好说。多用疑问句，少用反问句。</span>
- <span>分析问题要找到根本原因</span>
  - <span>相关性和因果性</span>
# <span>复盘中可能遇到的困境</span>
- <span>复盘往往意味着对过往自己的否定，要克服自己的面子问题，真正的复盘一定是“无我”的。</span>
  - <span>对待复盘要采取“超脱”、“客观”的态度，“客观”是“超脱”的必然结果。</span>
  - <span>不要用“防卫”心态对待别人的提问</span>
- <span>团队复盘时   大家一团和气不说话（私下告状）</span>
- <span>团队复盘时互相指责，部门间推卸责任，最终变成人身攻击</span>
  - <span>复盘是为了得出规律性的认识</span>
# <span>如何评价复盘的结果是否到位</span>
- <span>复盘的结论是否落实到偶发性因素上</span>
- <span>复盘的结论是指向人还是指向事，指向人可能就没到位</span>
- <span>复盘的结论是否能经得起交叉验证</span>
