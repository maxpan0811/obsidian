# <span>我们怎样阅读一本商业书籍</span>
- <span>一般商业书籍的套路是描写一个成功的公司/个人奋斗的历程，从中总结出一些普遍性的优秀品质</span>
  - <span>艰苦奋斗</span>
  - <span>坚韧不拔</span>
  - <span>团结友爱</span>
  - <span>目标明确</span>
  - <span>……</span>
- <span>但《黑天鹅》这本书告诉我们：</span>
  - <span>你即使看到100万只白天鹅，都不能证明世界上天鹅都是白色的</span>
    - <span>但你只要看到一只黑天鹅，就能证明世界上的天鹅不都是白色的</span>
  - <span>同样地，主人给火鸡喂食1000天，也不能证明主人是爱火鸡的</span>
    - <span>第1001天发生的事情，才能证明主人的真正目的</span>
- <span>很多商业书籍都不值得阅读，特别是证实性的书籍，只有证伪性的值得一读</span>
# <span>现代我们怎么做科学研究</span>
- <span>我们一直认为：通过观察找到规律，得出科学理论。</span>
  - <span>这其实是错的</span>
  - <span>同时观察本身往往就带着固有的思考范式：你看到的往往是你思维中想要看到的。当你换一种思考范式，同样的客观现象可能会带给你完全不同的观点。</span>
- <span>波普尔的科学哲学观</span>
  - <span>猜想+证伪</span>
    - <span>假设一个理论，然后用实验去证伪它，如果没有被证伪，就增加了一定的可信度</span>
  - <span>永远接近真理，但永远无法达到真理的本身</span>
    - <span>现有的科学理论都是目前为止通过了最多证伪性实验的理论，但不能说一定是正确的结论</span>
- <span>科学的三大特征</span>
  - <span>可证实</span>
  - <span>可证伪</span>
  - <span>可复制</span>
