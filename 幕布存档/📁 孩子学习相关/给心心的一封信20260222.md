# <span>心儿见信好，很高兴这次学校的寒假作业让我有这个机会把平时跟你沟通的一些观念系统地梳理一下。</span>
# <span>首先，要坚持自我奋斗。你这次小升初没考上上海实验学校，只能去对口的初中，爸爸知道这对你是一个很大的打击，那段时间看你都闷闷不乐。你从小学三年级的暑假开始准备AMC8、奥数、小托福这些课程的考试，每个周末都没有休息，四年级和五年级的寒暑假课程比平时上课还多、还辛苦，外公和爸爸也是每次接送、陪伴，全家都为这个升学机会付出了很大的努力，但最终还是没有达成目标，你的失落我完全理解，爸爸心里也是有几分同感的。</span>
- <span>但你要慢慢懂得，人生的每一次努力不一定每次都能获得回报，但这不代表这些努力的过程就没有价值。你现在进入初中预备班的第一个学期，期中考试年级第四，期末考试年级第二，学期末获得班级“三好学生”荣誉，这些都是你近三年来努力、积累的结果。所谓“功不唐捐”，你会慢慢懂得人生的努力永远不会白费，现在的积累可能就是你未来某一次机会的敲门砖，爸爸在自己的工作中已经多次碰到这样的例子，所以应该把这次升学不利看作是一次挫折教育，不应因此失落、沮丧。</span>
- <span>好学校的优秀老师、优质教学资源当然会对自身的进步提供更大的助力，但在目前国家“教育平权”的大背景下，竞争已经逐步转化为同区、同校内的竞争，而上海整体的发展水平让优质的老师、优质的教学资源有了更多的获取渠道，把书读好最稀缺的已经不是这些外部客观条件，而是一个有上进心、有自制力的孩子。所以问题的关键还是在于你自己，只要你愿意好好学习，即使是一个普通初中也能让你在同龄人里脱颖而出。</span>
- <span>名校的光环当然可以带给自身更多的荣誉和更好的平台，但你既然来到了对口的学校，你就要通过你自身的努力为学校带来荣誉。在爸爸读书的年代，当时上海的一些高考状元往往来自一些很普通的高中（因为重点高中的一些头部学生都保送了），这些高考状元通过自身的努力为自己的母校带来了“高光时刻”。“你所站立的地方，就是你的中国”，一个有志气的孩子应该通过自身努力让自己所在的母校发光。</span>
# <span>其次，要志存高远。一个人的自我奋斗是成长的基础，但如果没有一个崇高的目标，个人奋斗就会陷入为了自身的利益蝇营狗苟，成为一个精致的利己主义者，最终虚度一生。</span>
- <span>在2009年爸爸选择自己想申请的MBA学校时，清华经管的老师在宣讲会上回答申请者的提问“清华经管MBA项目想要培养什么样的人才”时说：我们就是要为国家和民族培养未来的领导者。这句话在当时那个舆论环境下恍如晴天霹雳般对我当头棒喝，我当时就毫不犹豫选择了清华经管学院。在开学第一课，在大礼堂里我们全体新生学习了邓稼先的事迹，在毕业的最后一课老师给我们讲的是“传承”，是“清华学生的前途要和国家民族的发展结合在一起”。</span>
- <span>所以爸爸还是希望你有机会能够去清华读几年书，不是为了爸妈的面子，或者仅仅是为了你未来能够站在一个更高的起点上，而是希望你有机会能够和一群中国最优秀的年轻人在一起，接受清华爱国传统的熏陶，像校歌里唱的那样，为中国的“自强”而努力学习成长，以后更好地为国效力。</span>
- <span>当然，“读书应志在圣贤，非徒科第”，清华只是让你有机会更好地为国效力，并不是只有读了清华才能为国效力，只要你有这个志向，无论你在什么学校读书，都有机会做出比原来的自己更大的贡献。</span>
- <span>在大学本科期间，如果有机会，爸爸希望你能够去为国服役几年，最好是参加海军。在解放军的大熔炉里，你能学习到一个巨型组织是如何管理的、体会到和不同省份战友的战斗情谊，作为海军还有机会去全世界不同地方看看，增长见识和扩大眼界。在二战期间，有697名哈佛学生为国牺牲，这才是真正的美国精神所在。当然现在和平年代牺牲的风险大大降低，但一个年轻人如果有过为国服役的经历，相信能够大大提升自己的全面能力和思想境界，也是真正意义上的为国效力。</span>
- <span>心儿你要记住：祖国终将选择忠诚于祖国的人，祖国终将记住奉献于祖国的人，只有在祖国的星河里我们优秀的个体才能绽放出更大的光芒。</span>
# <span>要胸怀天下。中国文化中的“天下大同”，是古人对当时已知世界（天下）的一种美好设想，而不仅仅是局限于中国本土，“人类命运共同体”就是“天下大同”的现代版本。目前人类的生产力已经达到了一个极高的境界，理论上满足现有人类的物质需求是没有问题的，但这个世界依然充满着贫穷、饥饿和疾病，而即使发达国家内部也是存在着极大的贫富差距，一些底层人民连基本的生存需求都无法满足，这完全是因为分配制度不公平造成的。中国虽然目前依然还处于发展中阶段，但从近年来世界局势的变化来看，我们要逐步做好为人类社会做出更大贡献、领导全人类走向一个更发达、更公平、更进步的时代的准备，这也是你们这一代年轻人所面临的机遇和挑战。</span>
- <span>“无穷的远方，无数的人们，都与我有关”，我们中国的年轻人，适逢这个中华复兴的伟大时代，既要看到未来的星辰大海，也要看到脚下土地还存在大量的贫穷和不公。人类在宇宙尺度中只是一个极微小的存在，如果我们连地球上的人类都不能全部团结起来，一起过上富裕、平等、幸福的生活，怎么可能集中地球的资源走向宇宙呢？</span>
- <span>中国在明清时代错过了“大航海时代”，错过了“工业革命”和“科学革命”，很大程度是因为处于一个近乎完全封闭的系统内不断熵增导致内部极度沉寂、停滞不前，未来我们要不断探索宇宙的新边界、科学的新前沿，这样我们的国家才能永葆活力，我们人类才能不断进步。霍尔特在20世纪50年代写的《科学：无尽的前沿》为后来美国科技的腾飞指出了明确的方向，爸爸很喜欢这本书的标题，随着中国科技和工业的腾飞，相信你们这一代在科学的前沿将会大有作为。</span>
- <span>当一个年轻人能够胸怀天下，就会感受到巨大的动力和无数的机会，爸爸现在的工作主要还是集中在国内，到你们这一代一定是面对着一张世界地图，中国的影响力遍及全球，你们的事业版图也一定是遍及全世界。</span>
# <span>写了这么多，想起你曾经跟老爸说过：你对我的期望好多啊，有时候觉得好累。其实这些期望都是老爸对未来的一些思考和设想，因为父亲的责任就是要在孩子“不知道自己不知道”的时候尽量为孩子指明道路，不要错过一些机会，因为在中国高强度的竞争环境下，很多机会都是有“时间窗口期”的，一旦错过了就没办法回头了。爸爸只是根据自己的经验和认知，给你指出了一些可能性，但最终的道路还是要靠你自己选择、自己去走，不是非要走到这个高度，如果到时觉得压力太大，大不了就回来“啃老”，老爸的条件养你是肯定没问题的。</span>
- <span>所谓“优秀的孩子是给国家养的，普通的孩子才是给自己养的”，如果以后你能每天在上海上班，正常成家立业，周末有空带孩子来看看爸妈，一起喝两杯，对老爸来说这也是很幸福的晚年了。所以你不要觉得压力很大。</span>
- <span>只是自己的人生就只有一次，现在中国的生活条件即使你失败几次也依然可以过上一个不错的生活，不会像先辈那样必须赌上自己的生命才能实现自己的理想，所以你就把这些目标和挑战当成一个人生的“游戏”，但行好事莫问前程，只管放心大胆地去尝试，爸爸现在的努力就是为你们的未来兜底的。</span>
