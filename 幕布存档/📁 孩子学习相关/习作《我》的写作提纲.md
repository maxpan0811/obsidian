# <span>外貌</span>
- <span>突出身材高大，比同龄孩子高出半个头，体格健壮，教练认为适合打篮球。</span>
# <span>篮球学习经历</span>
- <span>3岁开始上篮球课，从完全不会运球，到学会熟练双手同时运两个球。</span>
- <span>学会了一系列技术动作：</span>
  - <span>左右手交叉运球</span>
  - <span>背后运球</span>
  - <span>胯下运球</span>
  - <span>等等</span>
# <span>篮球锻炼自己的品格</span>
- <span>坚强：从刚开始被撞倒了就哭，到后来比赛中被对手撞倒左手腕骨折还尽力忍住眼泪，整个治疗过程都很坚强。</span>
- <span>心理素质好：比赛中罚球都百发百中</span>
- <span>勇敢拼抢：</span>
  - <span>在决赛中勇猛拼抢，拿球后快速运球进攻跃起投篮，爸爸去拿快餐的几分钟内连中三球得5分。</span>
  - <span>比赛中多次奔跑跃起抢球、空中投篮摔倒在地，腿部蹭破皮还坚持轻伤不下火线，打疯了。</span>
