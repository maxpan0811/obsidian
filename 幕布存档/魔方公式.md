# <span>层先法简介</span>
# <span>第一步：形成底面白色十字，且旁边四面的底层棱块与中心块颜色一致。</span>
- <span>白色中心块为底层，黄色中心块为顶层，先在黄色顶层形成4个白色棱块的小花。</span>
- <span>侧边的棱块颜色与中心块颜色保持一致后，转动180度，将白色棱块转到白色的底层。</span>
# <span>第二步：形成底面全部白色，二层和底层形成同色的T字形</span>
- <span>通过电梯公式将对应颜色块运到底层白色面，注意两边夹面中心块的颜色。</span>
- <span>电梯公式：</span><span class="bold">上顺下逆</span>
