# <span>牌型</span>
- <span>单张</span>
  - <span>口诀：两张小单张，不打不健康</span>
  - <span>对手早早用大王封堵单张，表示对手手里单张较少</span>
- <span>对子</span>
  - <span>口诀：情况不明，对子先行</span>
  - <span>投石问路，了解三家态度和牌力</span>
  - <span>对付对手的三带对</span>
  - <span>对手最后5张牌，控制对手4+1牌型</span>
  - <span>吸引对手的一对级牌、小王和大王下来</span>
- <span>三连对</span>
  - <span>牌局中后程使用，前期对手容易变化牌型来阻击</span>
- <span>三不带</span>
  - <span>主要作用是破坏对手牌型，协助对家</span>
  - <span>往往手里有1、2对大的对子</span>
- <span>三同连张（钢板）</span>
  - <span>大的钢板尽量不要过早打出去</span>
- <span>三带对</span>
  - <span>带走小对子</span>
- <span>顺子</span>
  - <span>小顺子尽量不要在牌局前期打</span>
- <span>炸弹</span>
  - <span>先出炸弹当不了上游的概率超过60%</span>
  - <span>残局先逼对手出炸弹</span>
  - <span>骑马火：比对手炸弹刚好大一点的炸弹，或者最小炸弹炸大王</span>
  - <span>炸七不炸八</span>
  - <span>炸上家放下家</span>
# <span>牌力分析</span>
- <span>手数</span>
  - <span>8～11手算较强牌型</span>
- <span>炸弹</span>
  - <span>人均2.1个</span>
- <span>牌型</span>
  - <span>对子多了可能会有顺子，三带对就比较少</span>
  - <span>三带对多了顺子就比较少，炸弹也少</span>
  - <span>已经打出5张的点数就不会再有炸弹</span>
  - <span>注意自己的断张，外面容易有炸弹。注意各家打出的这个点数牌有几张，如果也很少，估计有大炸弹。</span>
  - <span>注意打出的5和10，所有顺子都要包含5或10</span>
# <span>记牌技巧</span>
- <span>记住三家不要的牌型</span>
- <span>2468-</span>
  - <span>2张红桃级牌</span>
  - <span>4张大小王</span>
  - <span>6张级牌</span>
  - <span>8张A</span>
  - <span>-：用减法不断累计</span>
# <span>出牌技巧</span>
- <span>8张牌一般都是5+2+1的牌型</span>
- <span>让对家守底</span>
- <span>残局：</span>
  - <span>10张打2张，9张打1张，打成8张就明朗；</span>
  - <span>7张或8张，就要打5张</span>
  - <span>6张打3张</span>
  - <span>5张打2张</span>
  - <span>枪不打四，最后4张牌不拦</span>
