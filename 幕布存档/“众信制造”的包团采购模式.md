# <span>和外部供应商合作</span>
- <span>单独设计一条产品线路（前期两家对接人沟通好产品需求，价格，收客需求，产品标准等等）</span>
- <span>众信不会给供应商上定金，供应商给予取消政策，最低成团人数，人挡价格分别返利 。定价完后，返利分为门店返利+总部返利两块构成 </span>
- <span>供应商承诺最低发团人数，到了此人数供应商必须发团并且要承受亏损压力 </span>
- <span>众信针对团期未收客或收了低于发团人数，到放位时限时可双方商量取消或放位拼团散收，但一般众信制造产品都会在散拼产品基础升级一些差异化。所以众信制造对供应商的产品要求比较苛刻 </span>
- <span>众信制造对门店统一外卖价，统一管理，不允许降价，结算价也不允许有任何其他结算优惠。如发现，对于门店处罚是关闭门店系统一个月，对供应商处罚是利润3倍赔偿+停团处理。</span>
# <span>和优耐德、竹园合作</span>
- <span>和优耐德合作众信制造都是以王振玥为主导，他说了算。王振玥直接和王岚及产品经理对接。</span>
- <span>优耐德提供系列机位给王，王挑选各个板块机位计划后来设计产品。</span>
- <span>产品设计好后优耐德后台给予报价，生产 </span>
- <span>王根据优耐德后台给予产品成本，他开始定价，定门店返利 </span>
- <span>取消规则王说了算，但基于优耐德给予国航政策，地接政策等等。</span>
