# 内部链接规范

## 格式

```markdown
[[文件名（不含扩展名）|显示文字]]

示例：
[[products/Claude Code|Claude Code 产品页]]
[[concepts/人效指标|人效指标解释]]
```

## 原则

1. **每次创建/更新页面时，扫描其他相关页面并添加反链**
2. index.md 和 overview.md 应链接到每个主要实体页
3. 术语表中每个术语都应链接到对应页面
4. 避免孤⽴页（无入链的页面）

## 布局建议

每页末尾统一加：

```markdown
## 相关页面

- [[page-A]]
- [[page-B]]
```
