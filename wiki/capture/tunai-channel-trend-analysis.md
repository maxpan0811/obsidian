# 途牛渠道Q2门店趋势分析

## 背景

途牛渠道的门店排名数据分散在月度文件中（3月、4月、5月），需要整合为连续性趋势表，展示门店的月度变化和增长模式。

## 分析方法

### 数据整合流程

1. **提取月度数据**：从各月排名文件中读取门店名称和华程GMV
2. **按门店聚合**：将所有门店数据按门店名称对齐
3. **计算指标**：
   - 各月GMV（3月、4月、5月）
   - Q2合计GMV
   - 活跃月数（有GMV的月份数）
   - 月均GMV = Q2合计 / 活跃月数
   - 环比增长率（3→4月，4→5月）
   - 趋势标签

### 趋势判断逻辑

```python
if active_months == 3:
    if apr > mar * 1.2 and may > apr * 1.2:
        trend = '📈 强劲增长'

    elif apr < mar * 0.8 and may < apr * 0.8:

```

### 趋势标签定义

| 标签 | 含义 | 判断标准 |
|------|------|----------|
| 📈 强劲增长 | 连续3月活跃，每月增长>20% | 3月活跃 + 4月>3月×1.2 + 5月>4月×1.2 |
| 📈 稳定增长 | 连续3月活跃，每月增长>10% | 3月活跃 + 4月>3月×1.1 + 5月>4月×1.1 |
| ⭐ 核心门店 | 连续3月活跃，月均>10万 | 3月活跃 + 月均GMV>100000 |
| ➡️ 平稳 | 连续3月活跃，波动不大 | 3月活跃 + 不满足上述条件 |
| 📉 持续下滑 | 连续3月活跃，每月下降>20% | 3月活跃 + 4月<3月×0.8 + 5月<4月×0.8 |
| 🆕 新晋门店 | 仅2个月有数据 | active_months = 2 |
| ⚠️ 单月开单 | 仅1个月有数据 | active_months = 1 |

## 业务洞察

### 渠道稳定性问题
- **70%门店仅单月开单**：说明途牛渠道门店活跃度较低，大多数门店不稳定
- **仅4.2%连续3月活跃**：核心门店数量极少，渠道依赖少数门店
- **需要建立门店激活机制**：提升单月开单门店的持续产出能力

### 重点门店关注
1. **通州北苑门市部**：5月爆发（¥219万），但3月无数据
   - 需关注：是否为季节性波动？能否持续？
   - 建议：深入了解门店背景，评估可持续性

2. **天津南开奥城门市部**：连续3月增长，是核心门店
   - 表现：3月¥17万 → 4月¥56万 → 5月¥87万
   - 建议：作为标杆案例，推广经验

3. **悦民街营业部**：连续3月活跃，月均¥26万
   - 表现：稳定但增长放缓（3月¥36万 → 4月¥20万 → 5月¥23万）
   - 建议：关注4月下滑原因，评估是否需要支持

### 渠道管理建议
1. **建立门店分级制度**：核心门店、潜力门店、观察门店
2. **制定门店激活计划**：针对单月开单门店的专项支持
3. **加强门店沟通**：了解门店经营情况，及时发现问题
4. **建立预警机制**：对连续下滑的门店及时介入
