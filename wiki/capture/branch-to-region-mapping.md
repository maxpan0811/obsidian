# 分公司与大区对应关系

## 核心规则
使用"订单销售公司"字段映射到大区，**不能用"门店省份"字段**。

**Why:** 因为北京公司和上海公司有线上业务（途牛、同程等），这些线上订单不能按门店所在省份来划分。
- 华北区域的途牛、同程线上业务算在北京公司
- 华东区域的途牛、同程线上业务算在上海公司
- 如果用"门店省份"字段，会把线上业务错误地分配到门店所在省份

**How to apply:**
- 数据聚合时使用"订单销售公司"字段
- 6个分公司对应6个大区：
  - 北京公司 → 华北
  - 上海公司 → 华东
  - 广州公司 → 华南
  - 成都公司 → 华西
  - 武汉公司 → 华中
  - 厦门公司 → 福建

## 订单来源字段
- 携程平台
- 非携-其他渠道
- 非携-途牛
- 非携-同程
- 非携-去哪

携程 = "携程平台"，其他都是非携程渠道。
