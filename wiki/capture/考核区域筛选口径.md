筛选目的地产品时，必须使用「考核区域」字段，而不是「目的地国家_归属业务区域」。

**差异：**
- `考核区域` = 北美洲 → 4,386行（中国大陆出发的常规产品）
- `目的地国家_归属业务区域` = 北美洲 → 11,542行（多出7,156行目的地参团业务部订单）

**Why：** 考核区域是公司考核维度，反映实际业务归属；目的地国家_归属业务区域包含了目的地参团的直飞订单，不适合做业务分析。

❌ 错误示范：
```python
m = df['目的地国家_归属业务区域'] == '北美洲'
```
→ 会把目的地参团（客人飞到北美当地参团）也计入，数据膨胀2.6倍。

✅ 正确写法：
```python
m = df['考核区域'] == '北美洲'
```

**How to apply:** 所有涉及目的地产品筛选的分析脚本，统一用 `df['考核区域']` 过滤。
