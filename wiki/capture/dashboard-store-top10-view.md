## 门店市占率视图（store_top10）

第三张看板视图，展示各省Top10门店的双品牌市占率。

### 数据口径
- **数据源**：`2026年1-12月携程渠道海外跟团游订单明细(自营&代理)20260716.xlsx`
- **品牌分类**：华程(含账号订单) + 自营(国旅订单) = 双品牌
- **排除旅顾**：`门店名称.startswith('旅行顾问')` 的记录不计入门店统计
- **排名依据**：按26年总产量（总营收）降序，取Top10
- **时间维度**：19个时段（全年/Q1-Q4/上半年/下半年/1-12月）

### 表格结构（9列）
| 列 | 说明 |
|---|---|
| # | 排名 |
| 门店名称 | |
| 26年总产量 | fmtRev（万元/亿元） |
| 26年双品牌 | |
| 26年市占 | <40% 标红（.low-share） |
| 25年总产量 | |
| 25年双品牌 | |
| 25年市占 | |
| 市占同比 | pp变化 |

### 交互
- **左侧**：分公司按钮（华北/华东/华西/华南/华中/福建）
- **右侧**：时间按钮（月/季/半年/全年）+ 下拉选择具体时段
- **默认**：Q2

### 省表头
每省显示：`XX家门店 Top10占比XX% | 省双品牌市占XX% 同比±X.X%`

### 关键技术点
1. **fmtRev函数**：营收专用格式化，>=10000万显示"X.XX亿"，>=1显示"X万"
2. **低市占标红**：share26 < 0.4 → .low-share CSS类（红字+浅红背景）
3. **数据结构**：`v.provinces[prov].stores[]`（不是.items）
4. **时间按钮格式**：与产品市占率一致（4类型按钮+下拉），用户确认好用

### 数据文件
- `/tmp/store_view_final2.json`：19个时段的门店数据
- 注入方式：更新DATA.views[2].periods

**Why**：用户需要看各省内头部门店的市占率表现，识别哪些门店双品牌渗透率低（<40%标红），以便针对性提升。排除旅顾是因为旅顾不是实体门店，不应参与门店排名。

**How to apply**：未来更新数据时，重跑数据提取脚本（排除旅顾），更新store_view_final2.json，重新注入DATA。时间按钮格式已成为标准，其他视图也应遵循。
