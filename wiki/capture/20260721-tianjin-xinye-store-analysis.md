天津携程新业广场门市 2026 年收了多少金额？排名第几？

## 查询过程

```sql
SELECT DISTINCT 门店名称
FROM "t_2026年1_12月携程渠道海外跟团游订单明细_自营_代理_20260716"
WHERE 门店名称 LIKE '%新业%';
```

### Step 2: 总产量查询

```sql
SELECT
    ROUND(SUM(CAST(REPLACE(成交收入, ',', '') AS DOUBLE))) as 成交收入,
    COUNT(*) as 订单数,
    SUM(CAST(成交人数 AS INTEGER)) as 出游人数
FROM "t_2026年1_12月携程渠道海外跟团游订单明细_自营_代理_20260716"
WHERE 门店名称 = '天津携程河西区新业广场门市';
```

**结果**：

| 订单数 | 304 |
| 出游人数 | 472 |

### Step 4: 全国排名

```sql
WITH store_revenue AS (
    SELECT
        门店名称,
        ROUND(SUM(CAST(REPLACE(成交收入, ',', '') AS DOUBLE))) as 成交收入
    FROM "t_2026年1_12月携程渠道海外跟团游订单明细_自营_代理_20260716"
    WHERE 门店名称 != 'unkwn'
    GROUP BY 门店名称
)
SELECT
    ROW_NUMBER() OVER (ORDER BY 成交收入 DESC) as 排名,
    门店名称,
    成交收入
FROM store_revenue
ORDER BY 排名
LIMIT 11;
```

## 结论

1. **总产量**：¥10,514,892（304 单，472 人）
2. **全国排名**：第 11 名（排除 unkwn 后）
3. **华程双品牌占比**：33.8%（¥3,553,197）
4. **与 Top 10 差距**：约 600 万（Top 10 门槛 ¥1,647 万）

## 技术要点

- DuckDB 的 `all_varchar=true` 入库导致数值列为 VARCHAR，查询时需 `REPLACE(成交收入, ',', '')` 去逗号后 `CAST AS DOUBLE`
- 排名需排除 `unkwn`（线上聚合门店），否则排名第 1 但无实际意义
- 门店名称模糊搜索用 `LIKE '%关键词%'` 先确认精确名称
