# Claude Code 用户

## 一句话
在终端中用 AI 辅助编程的开发者，关注效率提升和自动化。

## 画像

| 维度 | 描述 |
|------|------|
| 目标 | 减少重复劳动、自动化工作流、少写代码多交成果 |
| 痛点 | 权限控制粒度不够细、Hook 触发难预测、长上下文丢失 |
| 技术程度 | 高（程序员，熟悉 CLI） |
| 文档偏好 | 命令示例 + 配置片段 + Troubleshooting |
