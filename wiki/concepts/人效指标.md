# 人效指标

人效（人力资源效能）衡量单位人员创造的价值。

## 核心公式

```
人均产出 = 总营收 / 月均人数
```

## 解读

- **人均产出增长 < 营收增长** → 增长靠扩招，而非效率提升
- **人均产出增长 > 营收增长** → 效率在提升
- **月均人数下降但营收持平或增长** → 人效显著优化
