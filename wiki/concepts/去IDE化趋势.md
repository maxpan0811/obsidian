# 去 IDE 化趋势

AI 时代软件开发工作流的变化趋势：过去把所有能力塞进 IDE，现在 AI 把能力重新拆出来——编辑归编辑，远程归远程，执行归 CLI，复杂任务交给 Agent。

## 趋势表现

1. **IDE 不再是中心舞台**: 用户更多在 CLI 描述目标，Agent 自动读项目、改代码、跑命令
2. **终端进化**: Warp 整合 SSH 文件树 + 内置 Agent 能力
3. **编辑器轻量化**: 用 Sublime/Zed 代替 VS Code 等 Electron 应用
4. **Agent Command Center**: Windsurf→Devin Desktop，专注云端 Agent 协作
5. **分工清晰**: 编辑/远程/执行/Agent 各司其职
