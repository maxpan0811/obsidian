# 武侠类比系统

> Claude Code 教程系列贯穿的"张无忌学武"隐喻体系。

## 对照表

| 教程 | Claude Code 能力 | 武侠类比 | 含义 |
|------|-----------------|---------|------|
| 一 | 安装配置 | 拜入师门 | 环境就绪 |
| 二 | 基本操作和命令 | 扎马步、练基本功 | 基础操作 |
| 三 | 工作模式和权限管理 | 内功心法 | 底层逻辑 |
| 四 | CLAUDE.md 与上下文管理 | 调息吐纳，稳固根基 | 自我认知 |
| 五 | MCP（外部工具） | 倚天剑屠龙刀（兵器） | 外部能力 |
| 六 | Skills（经验封装） | 太极拳法（招式） | 工作流固化 |
| 七 | Hooks（自动规则） | 护体真气（自动防御） | 规则自动化 |
| 八 | Subagent（分身协作） | 分身术 | 并行能力 |
| 九 | Plugin + Git + 进阶 | 融会贯通，野球拳 | 体系化 |

## 设计逻辑

- **基本功（二）→ 内功（三）→ 调息（四）**：从会用到会用对
- **兵器（五）→ 招式（六）→ 护体真气（七）**：从外部工具到内部规则
- **分身术（八）→ 野球拳（九）**：从单兵到体系
