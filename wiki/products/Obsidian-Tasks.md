# Obsidian Tasks

Obsidian 中最强大的任务管理插件，支持 Markdown 语法和可视化弹窗两种任务创建方式。

## 核心功能

- Markdown 语法任务创建（`- [ ] 任务描述 📅 日期 ⏫ #tag`）
- 可视化弹窗创建（Ctrl+P → Create or edit task）
- 优先级：最低/低/普通/中/高/最高
- 日期属性：截止(📅)、计划、开始(🛫)、完成(✅)
- 任务依赖：前置/后置依赖
- 重复规则：自然语言设置
- Checklist 插件配合：侧栏 #todo 标签管理
