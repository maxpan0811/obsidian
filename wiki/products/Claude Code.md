# Claude Code

> Anthropic 出品的终端 AI 编程助手，也是法律人的 AI 工作台。

## 概述

Claude Code 是一个终端原生的 AI 代理工具。与网页聊天式 AI 不同，它能直接读写电脑文件、执行命令、调用外部工具，像一个真正能帮你干活的"律师助理"。
