# 分享一下怎么用iMac充当显示器的问题，要满足以下几点才行！ - 外设 - Chiphell - 分享与交流用户体验的最佳平台 - Powered by Discuz!

| |  | | --- | | 最近几天都在研究怎样用iMac当显示器。  经过一系列的查找资料，打阿婆400电话，终于弄懂了怎么拿iMac当显示器，现在和大家分享一下。  1、只有27寸的iMac才能有当显示器的资本（上一代带光驱的27寸iMac和最新一代超薄的27寸iMac都可以），21.5寸这款就不行了。  2、保证输入图像的iMac是要支持雷电协议的DP接口，只支持DP协议的miniDP口是不行的（应该只要是27寸的iMac都满足这个条件）。  3、保证输出图像的设备也要是雷电接口的。这点很重要的了！因为有些老一点的Mac设备上的miniDP口只支持DP协议，不支持雷电协议【我有个老款Mac Book（MB466），那个miniDP接口只是DP协议，并不支持雷电协议，导致试了老半天都不行。老一些的Mac mini应该也是这样】。  4、保证连接两端设备的线也要是雷电的线才行，这点也很重要！普通的DP线是无论如何都不能连接成功的（线这个问题也是折腾了半天，换了挺多不同牌子的线，都不行。最后只能买apple的原装雷电线了）。  5、2个设备都开机，在连接成功后，在目标iMac的键盘上按“花+F2”，进入显示器模式。要退出显示器模式，也是按“花+F2”。 | |
