# 这样的2016 MacBook Pro值得你再等上一阵

---

MacBook Pro 上次更新已经是一年前的事情了，而且上次的更新还只是一次小更新：增加了 Force Touch 触控板、更快的闪存、电池续航更长，显卡升级，这么常规的升级实在让很多人提不起兴趣。  
  
　想要买新款 MacBook Pro 的用户可能已经等得不耐烦了，但是最新的消息显示，新款 MacBook Pro 可能要到今年第四季度才发布，也就是说，想要买新设备的用户还得继续等。无疑新一代 MacBook Pro 会更轻更薄，处理器会更强。此外还有其他性能，比如 Touch ID，取消键盘上方的功能键以触控操作取代等。那么下面我们就来看看 2016 MacBook Pro 的这些新特性是否值得你继续等待。  
  
　首先来说说 Touch ID。自从 iPhone 有了 Touch ID，输入密码解锁或者登陆对我来说简直是浪费人生。不少人从 iPad Air 升级到 iPad Air 2 也完全只是因为后者多了 Touch ID。这东西看着虽然不起眼，但是没了它还真不习惯。  
  
　而且最近几年随着苹果对 Touch ID 的完善，它的重要性已经日益突出。比如授权应用购买，用户可以省去输入 Apple ID 密码的麻烦。两年前苹果还对第三方应用开发了这个功能。  
  
　Apple Pay 的上线让 Touch ID 变得更加实用。Touch ID 不仅让购买支付变得更加方便，同时相比实体银行卡/信用卡更安全。  
  
　如果 Touch ID 登陆 Mac，而且其在 iOS 设备上的所有功能都移植到 Mac 上的话，那真的是非常值得消费者等待的一个新特性。  
  
　在 Mac 上登陆网站 Touch ID 的作用就很大了，用户不需要输入登陆密码，只要 Touch ID 验证一下就好，可以帮用户省掉很多麻烦。如果 Touch ID 可以支持 Keychain，那方便就是大大的有。

第二个是Apple Pay。此前已经有消息称Apple Pay将提供网页版，而 MacBook Pro 如果支持 Apple Pay，那你可以想象你在网上购物会有多方便了吗？这不仅仅是简化了输入银行卡/信用卡的操作，同时也能保证用户在线购物的安全。

当然 Mac 要添加 Apple Pay 功能意味着它需要 Secure Enclave 和 Touch ID 传感器。但是在 iOS 设备上这一切是基于 ARM 芯片，这意味着苹果需要找出新的解决方案去兼容英特尔芯片，不过对于苹果来说这应该不是很大的问题。

相信苹果官网在线商店最终会支持 Apple Pay，而其他重要在线购物平台也会紧随苹果的脚步支持Apple Pay。

第三个就是替换实体功能按键的 OLED 触敏面板。目前我们还不知道苹果具体会如何设计这个触敏面板，但是我们可以大胆猜测一下。

如果苹果支持把实体键变成虚拟键，那就是毫无意义的噱头——甚至可以说是倒退。因为使用虚拟按键的话，用户使用时可能还得看着触敏面板操作，而且这种面板会消耗更多电量。不过在这里我们不妨稍微期待一下苹果的创新。

比如其中一种可能就是取消 dock，但不是完全取消，因为旧款 Mac 还需要。所以可能还是从软件入手，OS X 可以检测到支持触敏面板的机型，从而取消 dock 一栏，给用户更多操作空间。

还有另外一种可能就是让应用自定义触敏面板上的按键。比如用户在 Finder 中时，功能键就是其本来的功能，可以控制亮度、音量等。但如果是在应用中，应用就可以自定义这些按键为快捷操作。

Final Cut Pro、Logic Pro、Photoshop 和 Lightroom 等应用就有很多快捷键，如果这个触敏面板支持应用自定义的话，那也更方便用户使用。当然和专用键盘相比，单单这一行功能键的空间也是有限的。但是不管你有多少专业应用，你不可能为这些应用都准备专用键盘，而应用可自定义的虚拟键盘就方便多了。

第三就是设备的连接问题，消息称新 MacBook Pro 将有 USB-C 接口，这个接口支持 Thunderbolt 3。不过在这方面我们还是有一些问题。

新 MacBook Pro 上到底有几个 USB-C 接口？显然不可能像 12 英寸 MacBook 那样只有一个。现有 MacBook Pro 的参考价值也不大，它有两个 USB 3 接口，两个 Thunderbolt 2 接口。既然新 MacBook Pro 的 USB-C 已经内置支持 Thunderbolt，那就不需要独立 Thunderbolt 接口，也就是说四个 USB-C 可能够了。

但是也别忘了，如果苹果也想取消 MacBook Pro 的 MagSafe 充电接口，改用 USB-C 接口的话，那应该需要 5 个 USB-C 接口才能保持和目前一样的连接程度。

第二就是苹果还会留着旧接口吗？如果保留两个 USB 3 接口，那么 USB-C 接口可以有三个。另外现有的 HDMI 和 SDXC 是否会保留？

根据苹果一贯的做法，他们可能真的不会保留。也许有的人对此不满，抱怨不停，但是当年苹果取消光驱等特性的设计时，他们也遭遇了用户狂风暴雨般的“攻击”，后来不还是风平浪静地结束了。

关于 MacBook Pro 的接口还有一个问题：苹果是否会取消 3.5mm 耳机接口？iPhone 7 上苹果很有可能取消这个接口，那 MacBook Pro 上也不是没有这个可能，也许我们会看到它的耳机接口也变成 Lightning 接口。总的来说，新 MacBook Pro 上的接口可能是这样的：4-6 个 USB-C，可能还有一个Lightning接口，其他没了。

如果上述关于 2016 MacBook Pro 的假设都成立，你是否愿意再多等一阵子再购买苹果的笔记本呢？
