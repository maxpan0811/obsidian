# 不想用网页版？试试这些 Mac 上好用的第三方客户端

---

相比于 PC 用户，Mac 在官方客户端这方面上有点像个弱势群体。许多社交服务都不太注重 Mac 客户端的更新，甚至根本没有推出 for Mac 的版本。但不乏有些优秀的第三方软件，让 Mac 用户能够更随心、更方便的使用某些社交服务。

Maipo：更纯净的微博体验
--------------

用桌面版刷微博的人应该不多，新浪推出的官方客户端**微博桌面**上一次更新已经是 2015 年 2 月了，且只支持 Windows 系统。如果你是 Mac 用户，不妨试下 Maipo 这款第三方客户端，它比网页版更加简洁流畅，功能也很丰富，最重要的是不会有糟糕的广告和混乱的时间线。

Maipo 支持多账户登陆、消息通知、多窗口和多边栏，可以设置用户或关键字内容过滤，在保证功能全面的情况下优化了很多体验，增加了各种快捷键和手势以提高操作效率，并且开发者一直在持续更新，在最新的版本中，评论已经可以按热门顺序排列。如果要在 Mac 上使用微博，我相信 Maipo 会是很好的一个选择。

![](.evernote-content/26C089FA-E057-48D9-B107-DAEF97059CDE/82413E44-AC83-4870-9F7E-6E3688455052.png)

点击这里[下载 Maipo](http://weiboformac.sinaapp.com/)。

weweChat：让微信再简单点
----------------

实际上微信官方 Mac 版本身也在强调「极致简单」，但软件的迭代更新总是不可避免地增加越来越多的功能。weweChat 是一款真正极致简单的第三方微信客户端，它只保留了最核心、最基本的功能：收发消息。

![](.evernote-content/26C089FA-E057-48D9-B107-DAEF97059CDE/A6998A50-1F75-4B55-B666-8CF6055E66BB.png)

使用 weweChat 能够便捷地聊天、传送文件、群发消息。但如果你想要实现阅读公众号、使用文件传输助手、视频语音聊天、撤回消息等功能，甚至是发送自己收藏的表情包，怕是只有官方版能够满足了，毕竟 weweChat 使用的是微信网页版的 API。

![](.evernote-content/26C089FA-E057-48D9-B107-DAEF97059CDE/22AADEE9-AC67-42EA-837B-EB91665F0999.png)使用 weweChat 群发消息

Bilibili for Mac：用更优雅的姿势看 B 站
-----------------------------

Bilibili for Mac （简称为 BiliMac）是一款 B 站特供的在线视频播放器。虽然名字起得比较官方，但 B 站本身并没有公布 for Mac 的桌面版，BiliMac 由用户自主开发。相比于普通的 html5 浏览器，它的占用更低，并且支持下载视频等功能。

![](.evernote-content/26C089FA-E057-48D9-B107-DAEF97059CDE/A4E91EB1-4CAB-4369-8793-E1D72C411E52.png)不同方式观看视频的电量消耗，由开发者提供于 github

BiliMac 基于 Safari 浏览器内核（默认设置，可以更改为 Chrome），打开时与普通浏览器方式无异。

![](.evernote-content/26C089FA-E057-48D9-B107-DAEF97059CDE/68D9A740-354B-4ADE-B878-4AD23BB09ADD.png)

进入视频页后，可以选择连接 AirPlay 设备播放、下载弹幕或下载视频，点击播放将跳出一个 MPV 视频播放器，此时按回车键可发送弹幕。

![](.evernote-content/26C089FA-E057-48D9-B107-DAEF97059CDE/EB09B832-1844-4A1F-8526-C48D4F8F879E.png)

如果你对撕逼、无意义弹幕导致的观看体验不佳感到厌倦，可以在软件的偏好设置中选择智能屏蔽（屏蔽掉吵架、日期、剧透和秀优越感弹幕，具体算法不明）或关键词屏蔽，手动营造一个更美好的弹幕环境。

![](.evernote-content/26C089FA-E057-48D9-B107-DAEF97059CDE/E9BEB465-F61A-4AB3-9DD3-1EDBD3827A9A.png)

BiliMac 总的来说像是在 B 站的基础上提供了更多的优化服务，解决发热、占用高的问题，如果你是忠实的 B 站用户，不妨尝试一下，换个更优雅的姿势看视频。

点击这里[下载 Bilibili for Mac](http://bilimac.eqoe.cn/)。

ieaseMusic：来点喜欢的音乐
------------------

ieaseMusic 是一个针对网易云音乐的开源项目，作者自称是最好的网易云播放器。首先从视觉上来说，确实比官方版不知高到哪里去了。并且暂时没有开放评论功能，相比网易云本身的社区氛围，在「听音乐」这件事上会纯粹些，相信这也是很多用户不想要评论功能的原因吧。

![](.evernote-content/26C089FA-E057-48D9-B107-DAEF97059CDE/45C0530B-8214-4862-8556-68F1A7ADC652.png)

打开 ieaseMusic，首页是几个音乐电台，当然也可以通过搜索播放列表、专辑、音乐家或用户的关键词来寻找要听的音乐，ieaseMusic 也支持查看音乐榜单。

![](.evernote-content/26C089FA-E057-48D9-B107-DAEF97059CDE/30C42AB2-6B73-4797-93C6-01E8D822E92E.png)

比较有意思的是，在软件的所有加载界面，都可以看到一条小小的段子，算是开发者的 sense 吧。

![](.evernote-content/26C089FA-E057-48D9-B107-DAEF97059CDE/A339B535-4F3C-405E-B5F0-7789FF70B9CD.png)

作为一款网易云音乐播放器，ieaseMusic 还有许多功能待增加。从 Github 上作者公布的 todo 项来看，很多已经列在日程上了。期待它会变得更好。

点击这里[下载 ieaseMusic](https://github.com/trazyn/ieaseMusic)。

Flume：简洁美观的 instagram 桌面版
-------------------------

2014 年，instagram 不幸也成为了「不存在的网站」之一。作为 Mac 用户，除了直接用浏览器访问之外，使用 Flume 也是一个不错的选择，当然你还是需要备好梯子。

Flume 支持了 instagram 的大部分功能，包括最基本的浏览图片、点赞评论、查看用户主页、热门推荐、搜索等。对那些喜欢收集图片的人来说，Flume 支持以 1080 x 1080 的大小下载图片（instagram 支持的最大分辨率）。

首页按时间线排序，有列表式和网格式两种展示方式。

![](.evernote-content/26C089FA-E057-48D9-B107-DAEF97059CDE/7833C9EE-C496-4ACD-A699-64031B381B6C.png)

上传图片，选择滤镜，简单编辑，添加照片说明、地理位置等信息。

![](.evernote-content/26C089FA-E057-48D9-B107-DAEF97059CDE/110D1780-BDD7-4C2F-B54C-03477C4260B8.png)
