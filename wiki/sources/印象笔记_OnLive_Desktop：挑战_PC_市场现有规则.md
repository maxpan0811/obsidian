# OnLive Desktop：挑战 PC 市场现有规则

---

[![](.evernote-content/838A064E-4198-46D2-8B2A-6CB351778C44/76FE129D-D0FA-4487-AEB5-D574934FE1D4.jpg)](http://www.ifanr.com/wp-content/uploads/2012/02/onliveipad.jpg)

在桌面操作系统领域，谁能够挑战微软的统治地位？Mac OS X 的封闭模式注定它属于小众群体，Linux 的分崩离析或许只能为 Geek 所爱，Chrome OS？或许只有三个人喜欢，其中一个是  Sundar Pichai。

问题并不是 Windows 系统更加华丽、更加自由或者更安全，任何一个用过 Windows 系统的人都有一大堆诅咒它的理由。许多人仍然离不开它的缘由是太多让人依赖的软件，比如 Office。

最近，关于 Office for iPad 的谣言又激起了一股讨论风潮？既然有了 Office for Mac，或许微软也会考虑 Office for iPad？微软完全有理由”担心” iPad，因为 iPad 不是在操作上与 Windows 直接对抗，而是要从根本上改变了游戏的规则，颠覆 Windows 赖以生存的 PC 市场。

微软希望你在拥有 iPad 的同时，仍然购买 PC ，依赖 Windows 系统。它的大赌注则是即将到来的 Windows8 系统：平板即 PC，重新制定游戏规则。

只是，互联网的发展为微软培养了一个很大的敌人。我不是在说 Google，而是在说 OnLive。

爱玩游戏的人可能都知道 OnLive 的流媒体游戏服务。使用 OnLive 的好处在于，你无需担心机器的配置，或者是否是游戏专属的系统，你所需要的只是一个快速的网络。OnLive 的游戏服务支持 PC、Mac 以及 iOS 设备和 Android 设备，真正做到了轻松跨平台。

现在 OnLive 不再满足于游戏，而要把自己的云端服务扩展到桌面应用。他们发布了 OnLive Desktop。如同游戏服务一样，你现在可以在 iPad 上运行 Windows 系统，所需要的只是一个 OnLive Desktop 客户端（4.8MB）。

当你在 iPad 上就能使用 Windows 应用，是否还需要去购买一个运行 Windows 的 PC 或者平板？微软赖以生存的操作系统，如今只是你平板上的一个应用而已。

这完全改变了游戏规则。最关键是，OnLive 能够提供与本地应用相媲美的速度。

OnLive Destop 上的浏览器有多快？[VentureBeat 网站](http://venturebeat.com/2012/02/22/onlive-desktop-plus-launches-the-worlds-fastest-web-browser-on-the-ipad/)在看过演示之后，认为“这是 iPad 上最快的浏览器”。（理论传输速度是 1GB/秒，实际演示中，下载速度是 459 MB/秒，上载速度是 204 MB/秒）。

> “我们整个关于网络可能性的观念将要改变，”OnLive 的 CEO Steve Perlman 告诉 VentureBeat，“重要的是，在计算机发展上，我们有了一个新方向，我们有许多的路径可以选择。其中的一个成果就是速度快的有些荒唐的网络浏览器”。

历史似乎真是螺旋式的上升。当年，微软的 Windows 系统促进了 PC 产业的发展，改变了主机-显示端的计算模式。而如今，随着云端的发展，个人电脑似乎越来越成为一个云端主机的显示端。OnLive Desktop 的火爆或许正是 PC 市场走向衰落的一个信号。

目前，[OnLive 的资费标准](http://desktop.onlive.com/plans)开放了两种：OnLive Desktop Standard，免费，提供 Office 应用（Word、Excel、PowerPoint）和 Adobe Reader，2G 的存储空间。

OnLive Desktop Plus，每月 4.99 美元，除了标准版的应用之外，提供速度超快的浏览器（可运行 Flash），云端存储（比如 Dropbox）和云端邮件（比如 Gmail）。

即将开放的资费还有：OnLive Desktop Pro（9.99 美元/月起价），50GB 存储空间，定制 PC 应用。

OnLive Enterprise，企业版，可以根据用户需要定制，在 iPad/Android/PC/Mac 上运行 Windows 或 Linux，可以通过 MicroConsole 客户端在电视上运行桌面系统。
