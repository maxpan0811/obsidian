# 特斯拉正式发布 Model 3，特斯拉有那么多款跑车，你最喜欢哪一款？

---

**编者按：今天上午11点半，特斯拉在美国加州霍桑正式发布旗下入门级电动汽车Model 3，海外起售价为3.5万美元(约合人民币22.77万元)，单次充电续航215英里，新车将于2017年开始量产，年产量50万辆。**

**Model 3是特斯拉历史上最便宜的一款汽车。**

**造型方面，**Model 3造型上保持了浓重的特斯拉风格，新车前脸与车身采用了同色设计；采用了5座布局，内饰设计即为简洁；Model 3沿用了Model S的很多元素，包括前脸造型、侧面特斯拉三角标识已经隐藏式门把手等；新车还配备了全景天窗。

**动力方面，**Model 3提供了单电机后轮驱动以及双电机四轮驱动的选择，0-60mph（96km/h）加速时间仅需6.0秒，整车续航超过215英里（345km）。

**科技配置方面，**Model 3没有传统的仪表盘，中控台采用了一块方形的15英寸中控触摸屏，拥有特斯拉自动驾驶系统、OTA空中升级、远程诊断、以及APP联接等功能。新车拥有超五星级的安全性能，支持无人驾驶。

**价格方面，**特斯拉公布的**Model 3裸车的售价为3.5万美元**起，用户可根据自己的偏好对部分配置进行定制，但对应的车价也会有所变动。

**销售方面，**按计划，特斯拉将从明年末开始交付，首先是美国、加拿大，然后是欧洲，最后才会是亚太地区。

**预定信息，**根据特斯拉官方公布的信息，中国大陆地区的预订金数额为 RMB 8，000 元，每位车主每次付款限订购2台。目前Model 3接受两种方式的预订：第一，在北京时间3月31日起直接前往[各地特斯拉线下体验中心](http://www.tesla.cn/findus#/?search=store,service,supercharger,destination%20charger)进行预订，。第二，可以登陆[特斯拉中国官方网站](https://model3.tesla.cn)进行线上预订。网上预订的起始时间为北京时间4月1日上午11点30分。目前，Model 3暂不接受电话预订。

> **特斯拉全部车型总结:**

**2008年，特斯拉发布第一辆电动汽车——非常性感的特斯拉跑车Roadst 。**

![](.evernote-content/D5FA2E58-22B7-4A9D-9C86-D9BBE850BE1A/7ED383EC-DF5C-474F-9E42-0E6D3A8B9792.jpg)

Roadster 可续航245英里（约400公里），0-60英里/时加速只用3.7秒。2008年，该款跑车的底价为98,000美元。

据特斯拉公司发布的数据显示，特斯拉一共在30个国家销售超过2,400 辆Roadster跑车。

**2012年，特斯拉发布首款豪华电动汽车——Model S。**

![](.evernote-content/D5FA2E58-22B7-4A9D-9C86-D9BBE850BE1A/1E067EE6-F4C6-406B-8AA5-9868E64B3B25.jpg)

Model S 0-60英里/时加速只用5秒，续航里程达265英里（约426公里）。Model S 被授予“Motor Trend 2013年最佳汽车”。

Model S 免税前价格为106,900美元。

**2014年末，特斯拉发布双电机四驱配置版的Model S，这是世界上第一辆双电机电动汽车。**

![](.evernote-content/D5FA2E58-22B7-4A9D-9C86-D9BBE850BE1A/DF876502-F6B2-431B-8230-B4D37FE64A61.jpg)

同时，这也是特斯拉首次在每一辆车上装载自动驾驶仪。这款Model S有三个版本： 60D、85D 和 P85D，上图是P85D版。

P85D 版最高时速达155英里/时，可以在3.2秒的时间内加速到60英里/时，优于迈凯轮F1超级跑车。

P85D 可以续航285英里，制造商建议零售价为71,200美元。但是该车型已于今年2月份停产。

**2015年初，特斯拉发布了新版本的Model S，分别为70D、90D 和 P90D 版。P90D 版尤其值得关注，因为它具有Ludicrous 模式。**

![](.evernote-content/D5FA2E58-22B7-4A9D-9C86-D9BBE850BE1A/C4455B4A-DC69-4612-865C-CABDFF78198C.jpg)在Ludicrous模式下 Model S  0-60英里/小时加速仅需2.8秒。P90D 版本续航里程为253英里，最高时速可达155英里/时。

这款Model S 起价70,000美元。

**2015年9月，特斯拉发布带有鹰翼车门的Model X。**

![](.evernote-content/D5FA2E58-22B7-4A9D-9C86-D9BBE850BE1A/3998E20F-11BF-47AE-8660-DC1D85C468AF.jpg)

和Model S一样，Model X 有三种不同的版本，132,000美元起价。最高配版本为P90D，续航里程达250英里。在Ludicrous 模式下，0-60英里/小时加速仅需3.2秒，最高时速达155英里/时。

很多人保守估计Model X的发货日期会从2016年初推迟到2016年第二季度。

**1月份，特斯拉将软件更新到7.1版本，为Model S 和 Model X 带来了很多全新的半自动功能。**

![](.evernote-content/D5FA2E58-22B7-4A9D-9C86-D9BBE850BE1A/36E0A599-65BD-492F-809C-3C379C7123F7.png)

软件更新提供了安全功能，如：自动刹车、变更车道、盲点警告。这款车还可以在没有中夹分道线的情况下自动转向，自平行停车，使用交通意识巡航控制管理速度。

这次更新中最酷的功能是可以让司机一键召唤自己的汽车，就像一个私人的机器人管家。自动驾驶套件需要额外支付2,500美元。

**今天，特斯拉发布新车型—— Model 3。**（详细信息请参见文章开头）
