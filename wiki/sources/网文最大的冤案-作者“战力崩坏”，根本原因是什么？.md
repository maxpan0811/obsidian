# 网文最大的冤案：作者“战力崩坏”，根本原因是什么？

骂了这么多年的“战力崩坏”

到底是谁的锅？

其实——

我们一直误会了作者

一、展开世界观的必然走向？

知乎上有个高浏览问题：“**网文战力崩坏的根本原因是什么？**”

**自古评论区出高手**，点娘觉得其中一条评论很妙：

![](.evernote-content/BC6E7F68-E330-433F-95D8-F97ED1A36515/8AB50725-CE55-4E1B-9246-3BCE0E6294CE.jpg)

也就是说，**你觉得独一无二的成就，放在更大的坐标系里，可能只是起点**。

很多读者觉得离谱。

主角九死一生抢来的天材地宝，后面路边小喽啰人手一件。那**主角的努力算什么**？但其实这**不是崩坏，而是视野的扩大**。小说世界从一个小村庄展开到**一国、一界、诸天万界**，**资源的分布**自然越来越**密集**。主角当初的“**神物**”，放在更高维度可能只是**普通物资**。

这不是作者乱写，而是**世界观展开的必然结果**。

二、仙界第一人当矿工？

有读者提到《星辰变》：**仙界第一人**历经千万年艰辛飞升神界，起步是在**黑矿里当矿工**。上界无敌的至尊，飞升后连矿工都打不过。

**读者：逻辑呢？**

**点娘：罗辑在《三体》当执剑人**🐶

言归正传，主角当矿工和上面的评论，其实说的是同一件事。举个例子：你是一名作者，之前在其他平台写作，成绩不错，自觉有**证道之姿**。来到起点后，凭借实力**直发签约，奋斗三年**，终于从**签约境升到了LV3**。可抬头一看，上面还有**LV5大佬、十二天王、大神、白金**……天下英雄如过江之鲫。

你觉得自己“变弱”了，其实只是因为**平台太强**了。

所以**小说里写这种情节，恰恰是对现实的映射**。如果硬要写主角飞升后依然碾压众生，那才是真正的**逻辑崩坏**。凭什么一个**新来的就能吊打苦修亿万年的土著**？

三、  大师们也写不出完美作品

![](.evernote-content/BC6E7F68-E330-433F-95D8-F97ED1A36515/F0E7C3A3-6AA6-4364-9B53-A9A4BE7CFF13.jpg)

这个回答列举了鲁迅、老舍、钱锺书、托尔斯泰、雨果、大仲马等**文学大师的“毛病”**：鲁迅写不动长篇，老舍收尾慌张，托尔斯泰爱在小说里塞历史评论，烂尾程度堪比谏山创……

他想说的是连这些被**写入文学史的大师，作品也远非完美无缺**。

网文也是如此，不少神作中**看似跳脱的战斗情节，恰恰是其生命力所在**。用“战力崩坏”四个字去否定一部作品，**未免太苛刻**了。

四、敦刻尔克要是写成网文，会——？

**二战敦刻尔克大撤退**，德军坦克在距港口十几公里处突然停止前进，33万英法联军得以撤离。

这个情节要是写在网文里，不就是**妥妥的反派强行降智**、**主角光环太假、作者会不会写战争**？评论区早炸了。

![](.evernote-content/BC6E7F68-E330-433F-95D8-F97ED1A36515/2128B188-AC13-43A0-BB90-0AF12B5CFFE3.jpg)

但**历史上它真实发生**了。

现实比小说更不讲逻辑。**以弱胜强、绝境翻盘、不合常理的胜利，恰恰是人类历史上最精彩的部分**。一个故事如果太符合逻辑，每场战斗的结果显而易见，未免也太乏味了。

网文里那些被骂“崩坏”的战斗，有不少都出乎了读者预料。点娘认为，这些**没能预料到的剧情，往往是故事的魅力所在**。

![](.evernote-content/BC6E7F68-E330-433F-95D8-F97ED1A36515/C745F243-9C8D-41E7-A38D-3A690DB47C29.jpg)

所以说，合格的网文里**通常不存在真正的“战力崩坏”**。大家吐槽的，要么是**正常的“战力膨胀”**，要么是**作者写出了过于真实的落差**，而读者一时间没适应过来。

最后，你印象中哪部作品的“战力变化”最让你觉得意外？或者反过来，有哪些作品把“换地图后的战力平衡”处理得非常爽？评论区聊聊叭～
