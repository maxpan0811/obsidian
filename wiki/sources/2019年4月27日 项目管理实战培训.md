# 2019年4月27日 项目管理实战培训

|  |
| --- |
| 2019 年 4 月项目管理培训——吴兆明 |

- Vision - Mission -Strategy - portfolio（项目组合） - Program（项目集）- Project（项目）/Operation（运营）
- IC（个人贡献者/Individual Contributor）
- ERP 系统：高层关注问题能不能解决，有没有收益；中层关注位子能不能保住；基层关注有没有减少工作量。
- OCM：组织变革管理/人员调整
- 上下层的管理鸿沟，同级之间的职能鸿沟。
- 干系人/stake holder：主动影响者、被影响者、自认为受影响者。
- Sponsor（发起人）、PM、BA（业务支持）、SA（技术支持）、Team、竞争对手、供应商、职能部门、项目委员会、政府/法规。
- 发明：从 0 到 1；创新：从 1 到 1.1（加减法、替代法、逆向法、组合法）；
- 项目生命周期：论证、启动、规划、执行、收尾
- 郭总讲话：高性价比、门槛、科学管理降低成本提高效率、时间/质量/成本、提升客户体验/提升效率
- 10%来自书本，20%来自交流，70%来自动手实践。
- 商业分析 - 立项 - 规划（产品研发：用户、需求调研、方案设计、开发/测试、验收/部署；流程优化：现状分析、瓶颈分析、方案设计、任务分解、跟踪实施）
- 戴明博士的SIPOC模型：（Supplier、Information、Process（Tool/Tech）、Output、Customer）

![](attachments/e167b6337791c05a.png)

- 企业的价值评估：营收（增长速度）、利润（运营绩效）、现金（存活能力）
- WBS：FS（A 结束后 B 开始，可以省略）、FF（A完成以后B才能完成）、SS（A 开始后 B 才能开始）、SF（A开始之后B才能完成）。
