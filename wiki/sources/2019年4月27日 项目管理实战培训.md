# 2019年4月27日 项目管理实战培训

|  |
| --- |
| 2019 年 4 月项目管理培训——吴兆明 |

- Vision - Mission -Strategy - portfolio（项目组合） - Program（项目集）- Project（项目）/Operation（运营）
- IC（个人贡献者/Individual Contributor）
- ERP 系统：高层关注问题能不能解决，有没有收益；中层关注位子能不能保住；基层关注有没有减少工作量。
- OCM：组织变革管理/人员调整
- 上下层的管理鸿沟，同级之间的职能鸿沟。
- 干系人/stake holder：主动影响者、被影响者、自认为受影响者。
- Sponsor（发起人）、PM、BA（业务支持）、SA（技术支持）、Team、竞争对手、供应商、职能
