# 杀人偿命，欠债还钱！

---

杀人偿命，欠债还钱！
----------

*2016-11-17*德高国旅

```
今天我南京德高国际旅游有限公司终于决定委托上海律师事务所向上海市人民法院递交了弛誉欢途诉讼状，追讨其欠德高以及各地分公司共计1800万元欠款。我也清楚诉讼一家注册资金只有2千万的有限责任公司的而要回款的难度，我也深知并同情目前700多家供应商要求自救的无奈和热情，但我不得不这么做！  
   好吧，过程是这样的！  
   当驰誉还没完全终止结款的时候，我是最先让人去弛誉要款，要不然欠款远不止这些，当驰誉的供应商聚集在上海的时候，我们也配合着他们商讨自救方案，当弛誉放出有人愿意投资9千万的时候，我们也在拭目以待。当驰誉在供应商西部之旅李平带领下呼喊马老板，梁老板在哪里的时候，我决定和我们德高的投资方马上游科技公司进行了接触，马上游公司股票代码833158，市值13个亿，实力大家可以自行搜索，国内比较有影响的国有上市公司电广传媒（湖南卫视）控股，资产约400亿元的镇江文旅集团注资，自身甩那个什么华拓几万条街。我花了半天时间给董事会看了所有供应商的自救画面，让他们知道所有旅游人的精神，让他们出面能否帮大家走出困境。随后马上游的董事长约见了弛誉欢途的几个所谓能代表广大债主的“高管”。马上游给出的合作条件如下：马上游愿意出资现金4千万，供应商自筹2千万（第三方账户监管，2年以分红形式归还），先期退还供应商10%欠款，马上游带领大家自救海外上市或融资，若18年如果境外不上市或没有进一步融资，公司愿意在一定的条件下全资买断或者以回购形式分期归还所有供应商欠款合计2.7个亿。  
  好吧，过程就是这样，你们曾说资本是令人羡慕的，可以让企业插上腾飞的翅膀，资本也是可恶的，永远做着锦上添花的买卖，不会做雪中送炭的事情，那么请问，这样的条件算是落井下石？算是条件苛刻？看看来找我们所谓资本谈判的人吧，李刚，一个把欢途这么好的平台带向死亡的人；张道顺，欢途最早的创始人，现在还幻想让欢途插上花朵卖个好价格，李平，最大的受害者还在重复着让我们不要起诉。好吧，我只能说夜里说话有点激动了，我甚至开始怀疑你们打着上千家产品供应商，数万个家庭，近十万人幸福的幌子，你们是一伙的吗？当天晚上你们就约见了南京几十家供应商，你们有没有告诉他们下午所谈的内容，看着南京的朋友热血沸腾的在转股协议上面签字，你们就没有一点内疚吗？青岛市场知道马上游谈判的内容，试问青岛45家供应商有一家同意签字了吗？你们在隐瞒什么？  
       好吧，你们告知嫌投资方出资太少，那你们去找个大的，同意承担责任归还我们旅游人所有债务的投资方，3亿元是债务，是债务，是债务，请原谅我重要的说了3遍，是所有旅游人供应商的血汗钱，并不是你们谈判的资本，想卖个好价格，你们早干什么去了，  
     很早就有人和我说，自救其实就是一个骗局，原因在于：欢途其实在许多地区都是交由大的供应商操盘，那么欢途和许多大的供应商是庄头和角头的关系，不过这些角头也下场下注了，没想到做庄的飘没钱了，这个时候角头们也慌了，自己下的注没了不说，到场的赌客们也饶不了角头们啊，于是角头们换为赌客号召其他赌客继续下注，其目的先把自己下的注搞回来再说。难道真的是这样吗？  
   我石家庄公司负责人为100多万愁的整夜未眠，在上海待了半个月；我潍坊分公司负责人200多万，挺个大肚子去上海要钱，待了几天只拿回2万；我青岛分公司好友500多万，夫妻差点离婚，现在老公天天在欢途看帐要钱；请问你们还有没有良知，别拿着我们的勤劳当筹码，这里的每一分都是我们挣出来的；也请你们别拿我们的谦卑当幼稚，我们每一个人不是傻子。  
   好吧，请原谅我用了这么多好吧，现在已经是凌晨2点，窗外又下起了大雨，我心中也是充满了悲愤和无奈，你们会说不和马上游合作，我就在这里乱说乱写，你们驰誉不是有声明吗，你们大可以告我诽谤，可你们别忘了，你们还欠我们1800万，我们和上百家供应商一样，我们也是受害者，我知道在我们之前已经有17家供应商起诉了欢途，可能再加上我们的2千万的金额会让欢途难负其重，但是我不想再保持沉默了，再也不想看着可怜的供应商们被人卖了还帮别人维稳数钱的画面。大家也不用怕驰誉倒闭破产，因为我们有资金也有能力帮助大家走出困境东山再起，为你们也为我们自己，但是此时此刻我只想对驰誉欢途公司说一句：杀人偿命，欠债还钱！  
                              
   
                    德高国旅：张江南
```
