# 2012年Q4携程第四季度净利润1.93亿元 同比下降24%

---

携程第四季度净利润1.93亿元 同比下降24%
=======================

2013年02月01日 07:05  
来源：[凤凰科技](http://tech.ifeng.com/)

|  |  |

凤凰科技讯 北京时间1月31日消息, 携程今日公布其截至2012年12月31日第四季度和2012全年未经审计的财务业绩。财报显示，携程第四季度净收入11亿人民币(1.77亿美元)，同比增长19%；净利润1.93亿人民币(3100万美元)，同比下降24%。

2012年全年净收入42亿元人民币(6亿6800万美元)，同比增长19%；净利润7.14亿元人民币(1.15亿美元)，同比下降34%。

**2012年第四季度业绩概要：**

2012年第四季度的净营业收入为11亿元人民币(1亿7700万美元)，同比增长19%，相比第四季度的预期为同比增长15-20％。

2012年第四季度的毛利率为74%，相比2011年同期为76%。

2012年第四季度的营业利润为1亿2000万元人民币(1900万美元)，同比下降48%。 若不计股权报酬费用，2012年第四季度营业利润为2亿3400万元人民币(3800万美元)，同比下降28%。

2012年第四季度的营业利润率为11%，相比2011年同期为25%。若不计股权报酬费用，2012年第四季度的营业利润率为21%，相比2011年同期为35%。

2012年第四季度归属于携程股东的净利润为1亿9300万元人民币(3100万美元)，同比下降24%。若不计股权报酬费用，2012年第四季度归属于携程股东的净利润为3亿零600万元人民币(4900万美元)，同比下降12%。

2012年第四季度经稀释每存托凭证盈利为1.38元人民币(0.22美元)。若不计股权报酬费用，2012年第四季度经稀释的每存托凭证盈利为2.17元人民币(0.35美元)。

2012年第四季度股权报酬费用共计为1亿1400万元人民币(1800万美元)，占净营业收入的10%，对2012年第四季度每存托凭证盈利的影响为0.80元人民币(0.13美元)。

**2012年全年业绩概要：**

2012年净营业收入为42亿元人民币(6亿6800万美元)，相比2011年增长19%。

2012年毛利率为75%，相比2011年为77％。

2012年营业利润为6亿5500元人民币(1亿零500万美元)，相比2011年下降39%。若不计股权报酬费用，2012年营业利润为11亿元人民币(1亿7400万美元), 相比2011年下降23%。

2012年营业利润率为16%，相比2011年为30%。若不计股权报酬费用，2012年的营业利润率为26%，相比2011年为40%。

2012年归属于携程股东的净利润为7亿1400万元人民币(1亿1500万美元)，相比2011年下降34%。若不计股权报酬费用，2012年归属于携程股东的净利润为11亿元人民币(1亿8400万美元)，相比2011年下降19%。

2012年经稀释每存托凭证盈利为4.98元人民币(0.80美元)，相比2011年为7.08元人民币(1.12美元)。若不计股权报酬费用，2012年经稀释每存托凭证盈利为7.97元人民币(1.28美元), 相比2011年为9.33元人民币(1.48美元)。

2012年股权报酬费用共计为4亿3200万元人民币(6900万美元)，占净营业收入的10%，对2012年每存托凭证盈利的影响为2.99元人民币(0.48美元)。

“我们为携程第四季度及2012年全年取得的稳健业绩感到满意，”携程总裁兼首席执行官范敏先生表示。“携程始终专注于为客人提供最好的产品，最好的服务，最好的价格。2012年是携程的投资年，我们进一步加强了携程在中国在线旅行市场的领先地位。2013年，我们将继续在提升服务水平，提供丰富产品，提高品牌认知和加速技术研发上增加投入。携程已经做好了充分的准备，获取在中国旅行市场中快速发展的机会。”

**2012年第四季度和全年财务业绩**

2012年第四季度，携程总营业收入为11亿7000万元人民币(1亿8700万美元)，同比增长18%。2012年第四季度总营业收入环比下降6%。

截至2012年12月31日，2012年全年总营业收入为44亿元人民币(7亿零800万美元)，相比2011年增长18%。

2012年第四季度宾馆预订营业收入为4亿6800万元人民币(7500万美元)，同比增长17%，增长主要来源于宾馆预订量36％的同比增长，并被每间夜收入14%的同比下降部分抵消。每间夜收入的同比下降主要受促销活动的影响。2012年第四季度宾馆预订营业收入环比增长2%。

截至2012年12月31日，2012年全年宾馆预订营业收入总计17亿元人民币(2亿7300万美元)，相比2011年增长15%。 宾馆预订营业收入占2012年总营业收入39%, 相比2011年为40%。

2012年第四季度机票预订营业收入为4亿4700万元人民币(7200万美元)，同比增长18%，增长主要来源于机票预订量的同比增长。2012年第四季度机票预订营业收入环比下降7%，主要受季节性因素影响。

截至2012年12月31日，2012年全年机票预订营业收入为17亿元人民币(2亿7100万美元), 相比2011年增长18%。机票预订营业收入占2012年总营业收入38%，相比2011年为39％。

2012年第四季度[旅游](http://travel.ifeng.com/)度假业务营业收入为1亿6600万元人民币(2700万美元)，同比增长27%，增长主要来源于休闲旅游预定量的增长。2012年第四季度旅游度假业务营业收入环比下降26%，下降主要受季节性因素影响。

截至2012年12月31日，2012年全年旅游度假业务营业收入为6亿9000万人民币(1亿1100万美元)，相比2011年增长29%。旅游度假业务营业收入占2012年总营业收入16%，相比2011年为14%。

2012年第四季度商旅管理业务营业收入为5800万元人民币(900万美元)，同比增长22%，环比增长6%, 增长主要来源于商旅活动带动的商旅需求的增长。

截至2012年12月31日，2012年全年商旅管理业务营业收入为2亿人民币(3200万美元)，相比2011年增长24%。 商旅管理业务营业收入占2012年总营业收入5%，相比2011年为4%。

2012年第四季度净营业收入为11亿元人民币(1亿7700万美元)，同比增长19%。2012年第四季度净营业收入环比下降6%。

截至2012年12月31日，2012年全年净营业收入为42亿元人民币(6亿6800万美元)，相比2011年增长19%。

2012年第四季度的毛利率为74%，相比2011年同期和上季度为76%。

截至2012年12月31日，2012年全年毛利率为75%，相比2011年为77％。

2012年第四季度产品开发费用为2亿6400万元人民币(4200万美元)，同比增长53%，环比增长9%，增长的主要原因为产品与业务开发人员及股权报酬费用的增加。若不计股权报酬费用，2012年第四季度产品开发费用占净营业额的21%，相比2011年同期16%, 和上季度18%有所增长。

截至2012年12月31日，2012年全年产品开发费用为9亿1200万元人民币(1亿4600万美元)，相比2011年增长52%。若不计股权报酬费用，产品开发费用占净营业额的19%，相比2011年14%有所增长。

2012年第四季度销售与市场[营销](http://auto.ifeng.com/news/marketing/)费用为2亿8100万元人民币(4500万美元)，同比增长52%，增长的主要原因为营销相关活动的增加和人员增加。2012年第四季度销售与市场营销费用环比下降7%，下降的主要原因为营销相关活动的减少。若不计股权报酬费用，2012年第四季度的销售费用占净营业额的24%，相比2011年同期19%有所上升，相比上季度25%有所下降。

截至2012年12月31日，2012年全年销售与市场营销费用为9亿8400万元人民币(1亿5800万美元)，相比2011年增长58%。若不计股权报酬费用，销售与市场营销费用占净营业额的22%，相比2011年16%有所增长。

2012年第四季度的管理费用为1亿5200万元人民币(2400万美元)，同比增长34%, 增长的主要原因为管理人员费用及股权报酬费用的增加，以及增值税改革所产生的流转税的增加。2012年第四季度的管理费用环比增长1%。若不计股权报酬费用，2012年第四季度的管理费用占净营业额的8%，相比2011年同期6%有所增加，相比上季度保持一致。

截至2012年12月31日，2012年全年管理费用为5亿7000万元人民币(9200万美元)，相比2011年增长42%。若不计股权报酬费用，管理费用占净营业额的8%，相比2011年6%有所增长。

2012年第四季度的营业利润为1亿2000万元人民币(1900万美元)，同比下降48%，环比下降37%。若不计股权报酬费用，2012年第四季度的营业利润为2亿3400万元人民币(3800万美元)，同比下降28%，环比下降22%。

截至2012年12月31日，2012年全年营业利润为6亿5500万元人民币(1亿零500万美元)， 相比2011年下降39%。 若不计股权报酬费用，营业利润为11亿元人民币(1亿7400万美元)，相比2011年下降23%。

2012年第四季度的营业利润率为11%，相比2011年同期为25%，相比上季度为16%。若不计股权报酬费用，2012年第四季度的营业利润率为21%，相比2011年同期35%，和上季度25%有所下降。

截至2012年12月31日，2012年全年营业利润率为16%，相比2011年为30%。若不计股权报酬费用，2012年全年营业利润率为26%，相比2011年为40%。

2012年第四季度的所得税有效税率为25%，相比2011年同期23%和上季度24%有所上涨，上涨主要原因是不能税前抵扣的股权报酬费用占营业利润百分比的增加。

截至2012年12月31日，2012年全年的所得税有效税率为31%, 相比2011年为20%, 上涨主要原因是为支付2012年6月宣布的股票回购计划自中国子公司汇往香港直接母公司的股利将缴纳5%的中国预提所得税，和不能税前抵扣的股权报酬费用占营业利润百分比的增加影响，并被部分分公司所享受的税务优惠而部分抵消。

2012年第四季度归属于携程股东的净利润为1亿9300万元人民币(3100万美元)，同比下降24％，环比下降1%。若不计股权报酬费用，归属于携程股东的净利润为3亿零600万元人民币(4900万美元)，同比下降12%，环比增长1％。

截至2012年12月31日，2012年归属于携程股东的全年净利润为7亿1400万元人民币(1亿1500万美元)，相比2011年下降34%。若不计股权报酬费用，2012年全年归属于携程股东的净利润为11亿元人民币(1亿8400万美元)，相比2011年下降19%。

2012年第四季度经稀释每存托凭证盈利为1.38元人民币(0.22美元)。 若不计股权报酬费用，2012年第四季度经稀释每存托凭证盈利为2.17元人民币(0.35美元)。

截至2012年12月31日，2012年全年经稀释每存托凭证盈利为4.98元人民币(0.80美元)，相比2011年为7.08元人民币(1.12美元)。若不计股权报酬费用，2012年全年经稀释每存托凭证盈利为7.97元人民币(1.28美元)，相比2011年为9.33元人民币(1.48美元)。

截至2012年12月31日，现金及短期投资余额为56亿元人民币(8亿9900万美元)。

**业绩展望**

携程预计2013年第一季度净营业收入年增长率约为15-20%。该预测反映了携程根据当前形势所做出的初步判断并有可能发生调整。

**近期发展**

截止2013 年1 月31 日，根据2008 年，2011 年和2012 年分别通过的三个股票回购计划，携程在公开市场中共购买了约1750万携程存托凭证, 价值2亿9900万美元。 [![](.evernote-content/C9800433-D45C-4239-84E1-DABA5FFA7470/2B34F012-79DC-475C-A530-03D7AA36D489.gif)](http://www.ifeng.com/)
