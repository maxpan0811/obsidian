**疫情数据：美国确诊数超过意西总和；英国病死将超中国；欧洲七国病死率超武汉**

**一、****要点**

l  **全球确诊病例即将达到100****万，病死突破5****万**

l  **美国的确诊数已经超过第二名意大利与第三名西班牙的总和**

l  **欧盟的全社会感染率是中国的17.7****倍，全社会病死率是中国的31.8****倍**

l  **意大利伦巴第一个地区病死7,593****人，病死率17.0%****，比SARS****高出77%**

l  **西班牙死亡人数破万，死亡率升至9.1%****，接近SARS****；确诊数在两日内将超过意大利**

l  **德国总病例8.1****万，超过中国；病死率1.2%****，高出湖北省之外中国内地（0.89%****）接近四成**

l  **荷、英病死率分别达到9.1%****和8.6%****，都在接近SARS****。英国病死数明天将超过中国**

l  **意、西、法、荷、比、瑞、英的病死率均超过武汉市（5.1%****），其中英、荷、瑞、法为GHSI****指数中全球对传染病防控“准备最充分的国家”**

l  **俄罗斯单日新增771****例，开始爆发**

l  **今日开始关注其他人口过亿大国的疫情。**

**二、日更信息**

**全球**

l  确诊病例即将达到100万，病死突破5万

**美国**

l  确诊数近23.6万，超过第二名意大利与西班牙的总和

l  病死数5,775例，病死率2.45%。按完结病例口径统计，36.0%的完结病例归于病死

l  美国的全社会感染率是中国12.6倍，全社会病死率是中国的7.6倍

**欧盟**

l  欧盟确诊44.6万人，单日新增2.7万，几个大国（意西德法）病例增速不见放缓

l  病死数3.2万，是中国的9.8倍。病死率7.3%，中国的1.8倍

l  全社会感染率是中国的17.7倍，全社会病死率是中国的31.8倍

**意大利**

l  确诊11.5万，病死1.4万，病死率继续上升，达12.1%。按完结病例统计，病死率43.2%

l  疫情重灾区伦巴第确诊4.48万，病死7,593人，病死率17.0%，比SARS高77%

**西班牙**

l  单日新增确诊6,120例，确诊总数破11万，两日内就可能超过意大利

l  死亡破万，死亡率升至9.1%，接近SARS

l  马德里（660万人）目前确诊约3.2万病例，病死4,175例，病死率达13.0%，

**德国**

l  总病例8.1万，超出中国

l  新增病亡149例，病死总数997例，病死率继续爬升，达到1.2%。高出湖北省以外的中国内地（0.89%）近四成

l  按完结口径计算，德国病死率28.2%

**法国**

l  确诊快速爬升，达5.7万

l  病死4,032例，超过中国。病死率高达7.1%，

**英国**

l  确诊接近3.4万例

l  病死2,925例（单日新增570例），病死率高达8.6%，也在接近SARS

l  英国病死数将在明天超过中国

**荷兰**

l  荷兰继续爆发，病死率达到9.1%

l  荷兰的病死率、每百万人病死数均远超中国湖北省。

**比利时**

l  人口1,160万，疫情持续爆发不见缓。病死超过1000人，病死率6.6%，十分可怕

**病死率超过武汉的欧洲国家。（武汉病死率5.1%****）**

l  包括英、荷、瑞、法（GHSI评选对防控传染病“准备最充分”的13个国家之列）

l  以及意、西、比

**值得关注的人口大国（今天增加了人口大国列表）**

l  **土耳其**：单日新增2,456，总病例1.8万，超过了荷兰、比利时，明天将超过瑞士

l  **印度：**单日新增343例，总病例2,341

l  **巴西：**已经爆发了一段时间，目前病例超过7,000

l  **俄罗斯：**今天有明显的爆发增长（单日771例），总病例3,548

l  **病死率问题：**有几个人口大国的确诊数目不多，但病死率比较高（印度尼西亚9.5%，孟加拉10.7%，埃及6.7%）），暗示检测量可能不够，对人口感染比例可能存在严重低估。后面需要密集观察这些人口大国的疫情
