# 销售人员评奖标准建议

> 来自 DeepSeek 的销售人员评奖建议：从"结果、过程、客户、团队"四维度综合评估。

## 核心原则

**不要只看销售额**，而是从四个维度综合评估，选出真正有价值和可持续的优秀销售。

## 四维评估框架

### 1. 结果维度（权重 50%-60%）

多指标平衡，避免唯销售额论：

| 指标 | 说明 |
|------|------|
| **完成率** | 比增长率更公平，能看出是否达成目标 |
| **利润贡献** | 总利润或毛利率，防止为冲业绩而牺牲利润 |
| **回款率** | 只有回款才算真正完成销售，避免坏账风险 |
| **新客/新产品** | 鼓励开拓新市场和推广战略产品 |

### 2. 过程维度（权重 20%-30%）

适合周期长、靠提成需要积累的行业：

- 销售漏斗健康度（线索量、转化率等过程指标）
- 客户拜访量和有效沟通次数
- 销售工具使用规范性

### 3. 行为指标

规范销售过程，关注：

- 新客户开发数
- 客户满意度
- 团队协作能力

### 4. 回款质量

- 回款周期
- 坏账率控制

## 分组与奖项设计

**分组建议**（保证公平性）：
- 按区域分组
- 按产品线分组
- 新老销售分开评

**奖项分层**：
- 综合业绩奖（核心大奖）
- 单项奖（回款最快、新客最多、进步最大等）
- 季度/年度/特别激励相结合

**考核周期**：
- 季度 + 年度 + 特别激励，避免年底冲刺一刀切

## 注意事项

- **设置软性标准**：合规性、客户反馈作为一票否决项
- **防止副作用**：过度承诺客户、年底压货、老销售靠存量资源躺赢
- **大客户 vs 快消区分**：大客户销售周期长单笔利润高，快消销售单量多单值小
