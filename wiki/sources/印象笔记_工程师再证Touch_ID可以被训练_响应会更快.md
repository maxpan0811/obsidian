> 通过培养 Touch ID 对你指纹的熟悉度，可以减少识别失败的情况和加速 Touch ID 辨认你指纹的时间。

以下为文章全文：

[![](.evernote-content/B4AE0A75-FD04-4FFA-A607-B2A8286A086D/D34207D6-FEBB-467C-BCFF-8C7ED5B1A620.png)](http://resource.feng.com/resource/h046/h51/img201501011219480.png "工程师再证Touch ID可以被训练 响应会更快")

  
　　威锋网讯，根据来自 WordPress 一位名为 Josh Betz 的工程师再次证实，Touch ID 是可以被“训练”的，经过训练它可以变得更“敏感”。这不是什么新的秘密，但经由一名工程师的口中说出，我们还是有必要再次审视这个问题的。  
  
　　通过培养 Touch ID 对你指纹的熟悉度，可以减少识别失败的情况和加速 Touch ID 辨认你指纹的时间。根据 Betz 表示，因为每一次你点按 Touch ID 它都会学习更多关于你的指纹信息，所以更多的指纹数据会被 Touch ID 获取或学习，因而提升了每次解锁成功的机会和解锁速度。  
  
　　要让 Touch ID 学习更多，前往设置应用里的 Touch ID &密码，改变 Touch ID 会要求你输入密码。在这里，触摸一个已经增加了的指纹到 Touch ID 传感器，你会看到指纹会暂时照亮，这意味着 Touch ID 正在记录更多的数据。移动手指，让更多指纹录入，那么你就会发现 Touch ID 在解锁的时候会比之前响应更快。Betz 写道。
