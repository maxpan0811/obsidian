### 做套设计题，解锁一双「像素眼」

**@MakicLin：**「像素眼」是设计师异于常人的观察能力，尽管偏差只有 1 像素，他们也可以迅速发现。而当他们淡定地说出「偏左了一像素」时，路人常常投来瞻仰的眼光。

做好下面这个小测试，你也将解锁「像素眼」。

[Can’t Unsee](https://cantunsee.space/) 是一套出给界面设计师的测试游戏，其中包括 57 道题目（耗时约 15 分钟），你要做的只是从其展示的界面设计中挑出一个更好的。

![](.evernote-content/E74DBD15-BB48-4F8E-A612-22E6B745687B/52C77781-38CB-4394-B78F-9D81045EE339.gif)

Can’t Unsee 基本玩法

其题目分三个难度，由易渐难。简单题目可以一眼看出端倪，困难题目可能要卡很久。其题目涉及到了诸多界面设计中的「错误」：比如说图片比例，元素的对齐，图标风格统一；也有一些极致的细节，比如大小写，拼写错误等。即便用来消磨时间，也是很有趣的项目。当然，若想从中的学到点什么，请记得阅读答案旁的解释文字并点击比较按钮，看看到底哪里有区别。

![](.evernote-content/E74DBD15-BB48-4F8E-A612-22E6B745687B/9AB0AE79-B273-495D-A5AE-503E60EAAED0.gif)

对比正确与错误答案

Can’t Unsee 更是激起了大家晒分数的兴趣，无论是微博还是推特，都可以看到人们在评论区晒出自己的分数。不过当我为了刷分儿仔仔细细玩儿了两遍之后，发现**并不是所有的答案都是绝对正确的**，其高分只意味着你的观察能力更好，并不意味着你的设计力更好。

这是因为 Can’t unsee 的题目**脱离了语境**。比如这道关于「直角搜索框和圆角搜索框谁更好」的题目，就没有什么道理，我们有无数个例子可供反驳……

![](.evernote-content/E74DBD15-BB48-4F8E-A612-22E6B745687B/35CCA457-09E1-4B99-B3DA-163FDB1F439B.jpg)

方形搜索框哪里错了？

其实无论搜索框是方是圆都不是错误的，都是一个可用的搜索框。在这种情况下，语境是判断是否正确的根据，比如为了**统一视觉风格**：界面中有很多圆角元素，那么搜索框就应该是圆角的；如果界面中都是利落的直线，那么圆角的搜索框也就不合理了。去年谷歌把搜索框改为了跑道型，不能说它「更好了」，只能说它更适合当下谷歌的设计语言。

![](.evernote-content/E74DBD15-BB48-4F8E-A612-22E6B745687B/44E38DC6-48EC-4289-9C80-061A06C455E6.jpg)

新旧版谷歌网页对比

Can’t Unsee 中还有几道类似的题目，比如按钮是否有圆角、图片是否应该裁剪、色彩寓意等等。其答案大多是跟随当今的设计潮流而定，并不是绝对正确的答案。往深层讲，界面设计基本是在一个语境下做出合理的选择，这个语境可以是视觉风格，可以是某种目的（比如引导用户购买更多商品），脱离了语境也就没有了正确答案。

### Chrome 将允许用户分享特定句子段落的连接

**@Fairyex：**我们上网看到一些文章里面好句子、好段落，就特别想分享给别人看看。直接复制分享固然是方便，但有时候我们就是想让别人看原文，只是重点在某一段；有时候我们打开某个网页就是为了看特定的一段文字；还有的网站根本就不给复制文字。为了解决这些需求，Chrome 决定在未来一个月内给浏览器添加一个名为 Scroll-to-Text 的新功能，让人们可以**通过链接分享某个网页里面的指定内容**。

![](.evernote-content/E74DBD15-BB48-4F8E-A612-22E6B745687B/99F0C397-9D1B-46A0-BC5B-489E58682B7B.gif)

点开链接后，页面会自动滚动到指定的内容

使用 Scroll-to-Text 的方法十分简单，只需要在原本的网址后面加上 `#targetText=内容开头,内容结尾` 分享就可以跳转到网页包含这段内容的部分。比如上面图片演示的链接就是：

```
https://sspai.com/article/52902?series_id=70#targetText=问起,方便
```

而且除了文本内容，还支持添加：

* 多媒体内容（例如 `#trace = audio＆t = 10,20`）
* PDF（例如 `#page=12`）
* 表格（例如 `#row=4`）

上面这些内容都可以通过 Scroll-to-Text 直接跳转到特定进度。

![](.evernote-content/E74DBD15-BB48-4F8E-A612-22E6B745687B/96558223-8F00-4CC9-8BD8-EDD4AF5E41AE.png)

PDF 跳转

如果你现在使用的是最新开发者版本的 Chrome 浏览器，可以在地址栏输入 `chrome://flags #enable-text-fragment-anchor` 启用这项 Beta 版功能。由于该项功能原理是修改浏览器对于网页的预处理函数，所以即使这项功能正式发布，在短时间内应该只有使用新版本 [Chromium](https://www.chromium.org/) 内核的浏览器支持这项功能。不过谷歌正在努力的将这项功能变为一个标准协议，结合 Windows10 新版 Edge 浏览器将会使用 Chromium 内核的情况 1 ，相信我们很快就可以在所有浏览器上享受到这个功能的便利了。

![](.evernote-content/E74DBD15-BB48-4F8E-A612-22E6B745687B/DDCE3EE0-6B20-481A-9299-BC9860A788B2.png)

开启方法

非常建议所有对这个功能感兴趣且英语不错的朋友读一下[官方的 Github 文档](https://github.com/bokand/ScrollToTextFragment)，写地非常详细有条理。比如里面提到的点击搜索引擎搜索结果可以直接跳转到目标网页关键词部分，就是未来这个功能一个很好的应用。

![](.evernote-content/E74DBD15-BB48-4F8E-A612-22E6B745687B/78E0EC93-EDBB-451C-9343-943D0DCF0FCA.png)

### 火狐推出的密码泄露提醒服务：Firefox Monitor

**@Fairyex：**在数据安全问题上，人们都有一个普遍同意的观点：更强的密码 = 更好的保护，这也是今天介绍的 Firefox Monitor 的观点。但无论多强的密码，在服务器上数据库直接泄露的情况下，都形同如明文，而 Mozilla（就是推出火狐浏览器的那个非盈利组织）最近推出的 [Firefox Monitor](https://monitor.firefox.com/) 就是为了在这种严重服务器数据泄漏事件出现时及时提醒你。

想知道自己常用的账号密码是否已经泄露，只需要打开 Firefox Monitor 主页输入自己注册账户的邮箱就可以了。是的，**只需要输入邮箱，不用注册账户、输入密码等任何其他数据**，担心 Firefox Monitor 本身安全性的朋友可以放心了。

![](.evernote-content/E74DBD15-BB48-4F8E-A612-22E6B745687B/4B6625D4-5D29-49E3-98DA-227A2BFE5746.png)

只需要输入邮箱

输入邮箱之后如果在邮箱曾经出现在某次泄漏事件的列表中，会直接显示该次事件泄露的网站、时间、泄露的内容等数据，方便你及时地了解自己账户数据的泄露情况并进行修改。

![](.evernote-content/E74DBD15-BB48-4F8E-A612-22E6B745687B/7458D7FD-D34D-4E2F-90B9-F9FB85C8E1B0.png)

密码泄露情况

另外针对常用同一个邮箱注册各个网站账户的朋友，Firefox Monitor 提供了邮件提醒功能。点击网站的注册按钮输入想要提醒的邮箱订阅即可（注意电子邮件地址输入法下面有推广勾选框）。

![](.evernote-content/E74DBD15-BB48-4F8E-A612-22E6B745687B/7E94649C-811E-4122-8341-49DE2B616D11.png)

邮件订阅

注册以后，只要发生了影响到注册邮箱的服务器数据泄露事件，目标邮箱就会收到这样一封提醒邮件（由于没有账户，不想再接收提醒的朋友可以在邮件底部点击退订链接退订）：

![](.evernote-content/E74DBD15-BB48-4F8E-A612-22E6B745687B/A6241C9B-4333-422C-B779-9D08B907EFCD.png)

收到的提醒邮件

最后，想要提升账号安全程度，最好的措施还是那几样老调重弹但十分有效的方法：

* 不同地方使用不同的密码；
* 使用密码管理软件生成的随机密码；
* 能开两步验证一定要开两步验证；
* 尽量使用第三方账号登录；
* 收到泄露提醒及时修改密码等。

网站链接：[Firefox Monitor](https://monitor.firefox.com/)

### 如何关闭指定的 Safari 标签页

**@Minja：**前不久有读者问，如何才能批量关闭手机版 Safari 上的指定标签页。有时候打开了太多标签页，又不需要全部关闭，我们就会想**只关闭某些网站的页面**。很凑巧，Twitter 用户 [@Maxwell](https://twitter.com/mxswd/status/1097251939480350720/) 刚好分享了一个批量关闭的技巧：

1. 点击 Safari 右下角的按钮，展示标签视图；
2. 在页面中间下拉几次，出现「搜索标签页」的输入框，输入关键词后就能过滤出指定的标签页；
3. 长按右上角的「取消」按钮，就能批量关闭匹配出来的标签页了。

### 一个简单的 MacBook 省电技巧：快速关闭 Wi-Fi

**@Minja：**关于 MacBook 费电问题的抱怨基本没断过：盖子合上，电脑还在发光发热；一觉醒来，一大截电量不见踪影…… 除去真的「中彩」、硬件损坏的情况不说，很多时候掉电问题是可以通过软件设置来解决的。

以避免盒盖休眠时异常掉电来说，有个屡试不爽的方法就是：**关掉 Wi-Fi**。但是每次用完电脑还要去戳菜单栏上的小图标，很不方便。

MacScripter 论坛间流传着一段 AppleScript，可以快速开 / 关 Wi-Fi，可以和各种自动化工具组合起来使用。这段脚本会自动判断当前 Wi-Fi 是打开还是关闭，如果打开，就关上，反之亦然。放在 LaunchBar 里，运行起来就是这个效果：

![](.evernote-content/E74DBD15-BB48-4F8E-A612-22E6B745687B/45C65CAF-6E76-4304-A5D9-EFC9B3988EF1.gif)

快速开关 Wi-Fi

脚本摘录如下，不需要修改就可以在各种支持 AppleScript 的自动化工具中套用。

```
	set status to do shell script "networksetup -getairportpower en0"
	if status ends with "On" then
		do shell script "networksetup -setairportpower en1 off"
	else
		do shell script "networksetup -setairportpower en1 on"
	end if
```

其实，关掉 Wi-Fi 能省电的主要原因是阻止了电脑和网络交互。在《[旅行电量管理指南](https://sspai.com/article/52539?series_id=70)》里面虽然我介绍过一些比较 Geek 的方式，但有些操作会把省电设置打回原型 —— 使用过防休眠工具、连上过外接显示器灯等等 —— 如果发现休眠时电量又开始莫名其妙地「雪崩」，可以简单粗暴地试试关闭 Wi-Fi。

### 文章：不用写代码，就可以创建网站或应用的工具

**@张嘉夫：**Product Hunt 的创始人 Ryan Hoover 写了一篇[新文章](https://medium.com/@rrhoover/the-rise-of-no-code-e733d7c0944d)，列举了一些不需要写代码就可以创建网站或应用的工具：

* **用 [Webflow](https://webflow.com/) 构建漂亮的、响应式的 CMS（内容管理系统）网站。**

  即使是不会写代码的设计师或企业老板也可以做出强大的动态网站，Webflow 的愿景是让所有人都可以通过直观的视觉界面创建网站，而不是从头学习网站开发，已经有数万名设计师正在使用 Webflow 创建专业的响应式网站。利用 Webflow 可以创建你需要的内容结构，手动添加内容、从 CSV 中导入或是通过 API 导入。
* **用 [Shopify](https://www.shopify.com/) 构建电商网站，启动、运营和增长业务的一站式平台。**

  利用 Shopify 提供的免费工具来创建品牌门面，然后就可以向任何人销售商品，其中内置的工具可以帮你创建、执行和分析 Facebook 和 Google 上的推广活动。Shopify 还提供了一个友好的控制台，用于管理订单、快递和支付。
* **用 [Octane AI](https://octaneai.com/) 为店铺创建 Facebook Messenger 聊天机器人来提高收入。**

  可以自动发送购物车提醒、收据和快递通知，还可以自动回答问题，设置推广活动。与电子邮件相比点击率要高很多。此外 Octane AI 还提供了分析工具，可以追踪收入和客户行为，帮助快速增长的商家基于数据做出决策。GoPro 和华纳音乐集团等企业都在使用这项服务。
* **用 [Bubble](https://bubble.is/) 构建全功能的 Web app。**

  Bubble 是一个 web 开发框架，同时也是一个可视化编程工具，让非技术人员在不用写任何代码的情况下创建优秀的 web 和移动端 app。

  Bubble 的拖拽设计器提供了很高的自由度，可以做到与设计完美像素匹配，同时还支持响应式布局和动画。可以让 Bubble 针对不同的用户操作做出响应，为不同的用户改变文字或外观，发送邮件，处理交易，保存和显示数据等等，这些简单的规则可以组合出非常复杂的功能。此外，Bubble 的云平台还可以作为应用的服务器，在流量增加时自动扩容，安全、可扩展的基础架构让你开箱即用。
* **用 [Thunkable](https://thunkable.com/) 的拖拽工具来构建漂亮的、强大的移动端原生 app**。

  可以添加不同的 UI 组件来创建 UI，例如按钮、文本框、地图等等，同时还可以为 app 添加非可见的组件，例如短信接收器或 NFC 传感器。Thunkable 还提供了基于块的编程语言，让编程新手也可以轻松使用，但也能做出非常复杂的功能（可以制作函数、变量、回调等等）。

  此外，用 Thunkable 做出的每款 app 都支持 iOS 和 Android 双平台。如果你不想从头做一个新项目，还可以在样例 app 基础上进行调整，内置的开源项目也在不断扩充。
* **用 [Voiceflow](https://www.getvoiceflow.com/) 构建语音 app（作者用不到 10 分钟就创建了一个讲笑话机器人）。**

  不需要写任何代码就可以设计并在 Amazon Alexa 和 Google Assistant 商店里发布。通过可视化的拖拽工具来设计端到端的解决方案，设计出的原型可以直接在 Alexa 设备上进行测试并不断改进。
* **用 [Zapier](https://zapier.com/) + [Airtable](https://airtable.com/) 构建复杂的 web app。**

  Zapier 是一个连接不同 app 的自动化工具，可以在不同的 app 间自动移动数据，只要点击几下，就可以创建一个工作流。而 Airtable 则是一个简单易用的在线表格与强大数据库的结合体。

  Zapier 可以将 Airtable 与超过 1000 种 app 连接起来并实现自动化工作，可以自动收集来自其它 app 或工具的信息并存储至 Airtable，例如用 Typeform 来收集用户信息，然后利用 Zapier 自动将新信息填入 Airtable，同时还可以将 Airtable 与邮件连接，在有新信息填入时自动发送邮件通知开发者。亦或者在有新用户注册时自动向其发送欢迎邮件、在 Airtable 里添加新字段时自动创建 Google Calendar 事件等等。
* **用 [Carrd](https://carrd.co/) 构建简单的响应式单页面网站。**

  例如个人简历、可以获取用户邮箱的产品发布页面或其它更复杂的页面。可以从 Carrd 提供的众多模版开始设计，也可以从空白画布开始，完美支持手机、平板和桌面电脑等等屏幕尺寸。每个账号可以构建最多三个免费站点，同时 Carrd 的所有核心功能都是免费的。如果升级到 Pro 版（每年 9 美元），还可以自定义域名、创建更多网站、增加表单（例如注册和支付表单）、添加自定义代码或第三方小组件、添加 Google Analytics 等等。
* **用 [Substack](https://www.substack.com/) 构建邮件通讯。**

  作者可以获得一个 CMS 系统，用于发送电子邮件订阅、通过 Stripe 进行支付等等，此外作者还可以获得一个订阅内容网站供订阅者阅读。邮件通讯设计优雅，适配各种邮件客户端，同时还保证了极快的加载速度。
* **用 [Scapic](https://scapic.com/) 构建浏览器端的 AR/VR/3D 应用。**

  不需要写代码，只要拖拖拽拽即可，就像制作幻灯片一样简单。Scapic 可以用于制作可交互的 AR、Facebook 360 度全景照片、虚拟导览、Facebook 3D 动画、小星球照片、音乐可视化、AR 虚拟门、脸部滤镜等等。Scapic 还提供了可以自定义的模版，不需要事先学习也可以使用，同时还提供了数百万个 3D 模型、照片和插图。
* **用 [Readymag](https://readymag.com/) 创建在线杂志。**

  简单的拖拽界面可以让你专注在生产内容上，自适应的布局完美适配平板和手机。可以从空白页面开始设计，也可以从众多漂亮的模版开始。有来自 Typekit、Webtype、Google 和 TypeToday 的超过 1500 种字体可供选择，也可以上传自己的字体。
* **用 [Sheet2Site](https://www.sheet2site.com/) 把 Google Sheet 变成包含图片、文字、过滤器和链接的全功能网站。**

  Sheet2Site 提供了很多免费的模板，用于制作网格状的网站，例如一列城市、活动、工具、设备等等。只要拷贝 Google Sheet 的链接并粘贴在 Sheet2Site 上，然后进行一些微调，不到 5 分钟就可以完成一个网站。
* **用 [Retool](https://tryretool.com/) 来构建内部控制台和工具。**

  Retool 提供了基本组件，表格、文本框、下拉菜单等等供你定制内部工具。

此外还有专注于「无需写代码」的资源平台如 [Zeroqode](https://zeroqode.com/)、[NoCode](https://www.nocode.tech/) 和 [MakerPad](https://www.makerpad.co/)。

即使不会写代码，也可以使用这些工具在短时间内把创意变成现实，它们赋予了自不同专业、不同背景的人们创建互联网产品的能力。
