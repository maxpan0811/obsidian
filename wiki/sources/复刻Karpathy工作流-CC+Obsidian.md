## 一句话摘要

用 Claude Code + Obsidian 复刻 Karpathy 的 LLM Wiki 模式——核心流程「输入原始数据 → 自动整理 → 建立关系 → 支持查询」，5 分钟搭建，token 消耗降低 95%。

## 核心流程

1. 输入原始数据
2. 自动整理
3. 建立关系
4. 支持查询
5. 发现知识缺口 → 补充研究

## 操作步骤

1. 下载 Obsidian，创建新 vault
2. 在 VS Code 中打开 vault 文件夹
3. 启动 Claude Code
4. 让 Claude Code 读取 Karpathy 的 gist 文档
5. 补充指令让其作为 LLM Wiki agent 构建第二大脑

## 关键文档

[Karpathy 的 LLM Wiki gist](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f)

## 效果数据

有用户将 383 个文件和 100 多份会议记录整理为 Wiki，token 消耗降低了 **95%**。
