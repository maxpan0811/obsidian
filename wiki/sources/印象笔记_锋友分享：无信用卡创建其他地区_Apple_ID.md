# 锋友分享：无信用卡创建其他地区 Apple ID

---

[![](.evernote-content/9202D4C7-3000-49BE-9DFF-2DE739EFC58A/75D8C947-A813-4C94-A31B-DE014CB1B86F.jpg)](http://www.feng.com/iPhone/news/2015-12-03/Feng-friends-sharing-no-credit-card-to-create-other-area-Apple-ID_632088.shtml)

　Apple ID 是苹果公司为其产品(如 iWork、iTunes Store 和 Apple Store)所引入的认证系统。用户可以通过 Apple ID 这个全功能帐户访问苹果的各种资源。因为一个 Apple ID 可以被用于由苹果提供的多个产品和服务，它也可以称为 Apple 账户、iTunes Store 账户等等。  
  
　这个 Apple ID 可用于执行与 Apple 有关的所有操作(包括使用 iCloud 存储内容、从 App Store 下载应用程序以及从 iTunes Store 购买歌曲、影片和电视节目)的登录信息。这个帐户其实是一个 Email 邮箱地址，输入密码之后，用户就可以在 iPhone 上的 App Store 轻松下载安装和更新软件。  
  
　一般情况下，我们注册的 Apple ID 都是国内的帐户，在某些特定的情况下，我们需要使用其他地区的 Apple ID 才能进行操作，比如香港、美国和日本等等。今天，我们就来说说这个创建其他地区 Apple ID 的事。  
  
　日前，锋友 XMLbrother 分享了一个如何在不使用信用卡的情况下创建其他地区的 Apple ID 的图文教程，如果你有这方面的使用需求，不妨来看一下应该怎么操作。

![](.evernote-content/9202D4C7-3000-49BE-9DFF-2DE739EFC58A/EEBCABD0-FD08-4990-9635-88480C855DCC.png)

1. 点击 iPhone 的 App Store 图标，然后在“精品推荐”那一栏的底部点击自己的 Apple ID，然后选择“注销”。

![](.evernote-content/9202D4C7-3000-49BE-9DFF-2DE739EFC58A/2D3F2575-C7F2-4358-99E2-973D77EF0AE1.png)

2. 接下来，我们需要根据自己的需求来注册相应地区的 Apple ID，锋友 XMLbrother 在教程中以香港的 Apple ID 为例子，因此，先登录一个香港地区的 Apple ID。

![](.evernote-content/9202D4C7-3000-49BE-9DFF-2DE739EFC58A/69BF4856-4453-4022-8982-D2B5FAD9CE39.png)

3. 登录香港地区的 Apple ID 之后再次注销，具体操作跟第一步一样。此时，我们就会发现，我们已经进入了香港 App Store 的页面。

![](.evernote-content/9202D4C7-3000-49BE-9DFF-2DE739EFC58A/4F511FA7-08BC-472A-850C-9C9D41975EC3.png)

4. 点击香港 App Store 的排行榜，在免费排行榜中选取任意一个应用程序进行下载。

![](.evernote-content/9202D4C7-3000-49BE-9DFF-2DE739EFC58A/322046D3-5C47-4824-8A0F-BAA2054CDAF6.png)

5. 这时候，iPhone 将会弹出一个要求我们登录 Apple ID 的窗口，我们需要选择创建新 Apple ID。

![](.evernote-content/9202D4C7-3000-49BE-9DFF-2DE739EFC58A/C22495D5-72BA-42B3-8658-EAF2FBAA711A.png)

6. 选择相应的国家或者地区，这里选择的是 Hong Kong/香港。然后点击下一步。

![](.evernote-content/9202D4C7-3000-49BE-9DFF-2DE739EFC58A/9EA0DFD5-F151-467C-8C19-D1EAE45207C1.png)

7. 界面跳转之后我们会看到苹果公司的一些条款和约定，点击“同意”。

![](.evernote-content/9202D4C7-3000-49BE-9DFF-2DE739EFC58A/952363CB-6016-4D31-AE50-658A30B0C3BC.png)

8. 输入邮箱地址和密码。当然，还有密码验证问题、备用邮箱和出生日期等信息。

![](.evernote-content/9202D4C7-3000-49BE-9DFF-2DE739EFC58A/3E7110ED-A21D-426F-A5E3-C716E9B50B3F.png)

9. 在付款方式那一项选择“无”。然后点击下一步。

![](.evernote-content/9202D4C7-3000-49BE-9DFF-2DE739EFC58A/E067617A-D700-4C11-98B6-538846E2DC92.png)

10. 填写账单信息，比如姓名、电话和地址等等，填完之后进入下一步。

![](.evernote-content/9202D4C7-3000-49BE-9DFF-2DE739EFC58A/14AD74EF-9FCC-4A29-862A-2A66345496E2.png)

11. 接下来，我们需要前往第八步所填写的邮箱，进行邮箱验证。苹果会给该邮箱发送一封验证电子邮件，打开之后点击“立即验证”。

![](.evernote-content/9202D4C7-3000-49BE-9DFF-2DE739EFC58A/68A2BAA9-4CC5-412F-89DB-A9720015E939.png)

12. 输入刚刚创建的 Apple ID 的帐号和密码，并选择继续。

![](.evernote-content/9202D4C7-3000-49BE-9DFF-2DE739EFC58A/A04F4ED9-3D11-4417-A6D3-A95D1394A5C9.png)

13. 当页面提示验证通过了之后，我们就可以登录香港的 App Store 下载软件了。

最后，在第二步中，我们需要实现登录一个相应地区的 Apple ID，没有该 Apple ID 的锋友可以点击[这里](http://bbs.feng.com/read-htm-tid-10119004.html)进入原帖获取，包括美区和日区的帐号。
