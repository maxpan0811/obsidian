# ThinkVision发布六款显示器和Type-C一线连接方案

---

10月24日，联想 ThinkVision 在北京召开了新品发布会，发布会期间共发布了ThinkVision P27u、New X24、T22v、T24v、T22m和主打电竞的拯救者系列Y25f这六款新品显示器，此外联想 ThinkVision 还在发布会上带来了全新的全功能Type-C一线连接解决方案。  
  
**ThinkVision P27u**

![](.evernote-content/D8948B97-65BD-4304-8E3F-B34DBF456EFF/2680CD94-5B47-4751-A825-738B5053FB7D.jpg)

ThinkVision P27u是一款为专业人士设计的显示器，同时也是全球首款支持99% AdobeRGB色域的27英寸4K显示器，不论是在画面的色彩表现，还是在细节刻画上都相当有实力。ThinkVision P27u搭载了一块10.7亿逼真色彩的IPS面板，分辨率高达3840x 2160，同时使用了通过德国莱茵TüV眼部舒适度认证的爱眼滤蓝光不闪屏，大幅度减少用户长期使用显示器所造成的疲劳度。

设计方面采用了2mm的超窄边框，让其在多个显示器拼接时所展现的画面更加完整，完全可以满足专业人士对画面完整度的需求。ThinkVision P27u配备了全功能USB Type-C以及4个USB扩展接口，用以满足用户多样化的连接需求。此外该款显示器可以进行前后俯仰-5°~30°，垂直升降110mm的调节，让用户更舒适的使用显示器。

**ThinkVision New X24**

![](.evernote-content/D8948B97-65BD-4304-8E3F-B34DBF456EFF/54FA5166-648B-4B80-9DDC-25149EE58638.jpg)

ThinkVision New X24专门为办公室设计，其设计理念是“追求无边界，线条极其简单的办公环境才是王道”，所以在设计上采用了全金属背板，机身最薄的地方仅为4mm，同时其超窄边框仅有1.1mm。ThinkVision New X24具有96GB高色域，真实还原色彩，其178°广视角屏幕的特性可以你在与其他同事进行讨论时，方便把电脑上的内容以多的角度展现给对方看。ThinkVision New X24同样也使用了爱眼滤蓝光不闪屏，能够减轻用户长期对面屏幕工作时的疲劳度

**联想拯救者系列 Y25f**

![](.evernote-content/D8948B97-65BD-4304-8E3F-B34DBF456EFF/2583CE64-495C-40C7-A5AE-60DFCCBA1F1F.jpg)

联想拯救者系列 Y25f专门为电子竞技设计的显示器，采用了公认的电竞显示器最佳的24.5寸屏幕，同样也是超窄边框的设计。刷新率和响应时间也都参照当前的最佳电竞显示器配置，分别时144Hz刷新率和1ms响应时间。另外Y25f应用了AMD FreeSync显卡同步技术，让显示器与显卡形成完美协同，呈现出逼真纯粹的CG游戏画质。

**ThinkVision T24m**

![](.evernote-content/D8948B97-65BD-4304-8E3F-B34DBF456EFF/A6ECDFC7-F26A-4F69-8186-A5AADF837A00.jpg)

ThinkVision T24m有点像是旗舰ThinkVision P27u的低配版，配备了23.8英寸的显示器，与ThinkVision P27u一样，具有全功能USB Type-C接口，可现实一线连接多个扩展接口，能够实现-5°/30°前后俯仰，110mm垂直升降，以及正负45°的水平旋转和正负90°的垂直旋转。

**ThinkVision T22v和T24v**

![](.evernote-content/D8948B97-65BD-4304-8E3F-B34DBF456EFF/183DAF9A-6447-4CBD-BD7B-155BC62C052B.jpg)

这2款显示器的主要特点是搭载了先进的IRCamera红外摄像头，不但能够保证成像清晰稳定，还能够运用成熟稳定算法，配合WindowsHello实现快速面部识别。而T24v还搭载了全套VoIP网络会议系统，配合T24v本身的高清摄像头、降噪麦克风以及两个音箱，在用于会议时会有一个良好的体验效果。

售价方面，ThinkVision P27u为8999元，ThinkVision New X24为1399元。联想拯救者系列Y25f为1899元，ThinkVision T24m为1699元，ThinkVision T22v为1499元，ThinkVision T24v为1799元。

除了发布这六款新品显示器，ThinkVision 还在发布会上带来了全新的全功能Type-C一线连接解决方案。这个方案只需一条线就能完成充电、一级数据与影音传输，也就是说，在用户在使用电脑、笔记本电脑、手机、平板电脑以及所有外围设备时，都可以共享一条线、一种接口、一种充电器。这样一来，用户可以在使用多种设备情况下，省去了同时连接多条数据线的麻烦，从而提高效率。
