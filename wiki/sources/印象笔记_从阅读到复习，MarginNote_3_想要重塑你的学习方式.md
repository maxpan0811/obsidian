2小时前

[![](.evernote-content/C18C63B3-36FB-4704-8E82-A80B7466A23D/99C00FA6-8BD7-4F04-8ECB-F0930352E8B8.jpg)](https://sspai.com/user/713147)

#### [Umi](https://sspai.com/user/713147)

MarginNote 是由国人开发者 Min 独立完成的一款笔记 App，其将书籍标注与思维导图和抽认卡相结合，使学习过程更加灵活，少数派此前也有过数篇介绍文章。

上周三，MarginNote 迎来了上架以来的第一次付费更新，带来了全新界面和全面增强的笔记功能，我们不妨通过本文快速梳理一遍。

文档模式：你的文献「收集箱」
--------------

打开 MarginNote 3，我的第一反应就是它变「轻」了。二代的主界面更像是一个「文件管理器」，文档按照不同的文件夹和标签进行分类。而三代则重新调整了主界面的逻辑，按照「阅读 - 批注 - 复习」的流程对文件进行分类。

![](.evernote-content/C18C63B3-36FB-4704-8E82-A80B7466A23D/8D1AB016-1EE9-4F8A-8E70-F7E07FD421F3.png)

![](.evernote-content/C18C63B3-36FB-4704-8E82-A80B7466A23D/E326B2C1-7409-4597-ADFD-526DBD89B9F6.png)上：MarginNote 2；下：MarginNote 3

在 MarginNote 3 中，文件从外部导入将首先进入「文档模式」。在这个模式下，MarginNote 3 是一个纯粹的 PDF 阅读器，你在这里可以进行简单的标记和批注，通过快速浏览文档来确定自己是否需要进行「精读」。或者你也可以把它当做你的文献「收件箱」和「中转站」，将自己需要阅读的文献放在一起，方便批量管理。

![](.evernote-content/C18C63B3-36FB-4704-8E82-A80B7466A23D/CCC7B69D-7358-4984-9C40-159AFF0EBD98.jpg)文档模式

而在你进行标注之后，MarginNote 3 会为你标注的内容自动生成学习卡片，放在文献的旁边，并支持自由添加笔记、音频、手绘等，不受页面尺寸制约，就像在纸上的空白边缘处记笔记一样。

![](.evernote-content/C18C63B3-36FB-4704-8E82-A80B7466A23D/1F52F12C-7EE1-4690-B60E-E4869C09F052.jpg)

学习模式：「精读」好帮手
------------

如果你觉得「文档模式」无法满足自己的需求，只需要点击右上角省略号里的「转到学习模式」，就能创建一个包含该文档的脑图笔记本，进入功能更为强大的「学习模式」。在这里你可以使用高亮、圈选、选中、手绘等工具对内容进行标注，而 MarginNote 3 则会将它们剪辑为卡片，按大纲排列于左侧，并生成思维导图。

![](.evernote-content/C18C63B3-36FB-4704-8E82-A80B7466A23D/C2A800C9-BADC-4F40-9E57-93DD91ABC392.jpg)

MarginNote 3 还新增了 OCR 功能，可以将扫描版 PDF 中的图片内容转换为文字，免去了手动输入的麻烦。

![](.evernote-content/C18C63B3-36FB-4704-8E82-A80B7466A23D/7611F36E-723A-4DD9-A656-C2F5D9280986.jpg)

同时 MarginNote 3 也支持 Drag & Drop 操作，可以通过拖拽将外部资料直接添加至笔记内容中，非常方便。

![](.evernote-content/C18C63B3-36FB-4704-8E82-A80B7466A23D/3ADC5ABA-9B54-415B-9B35-B2A27A545B8B.png)

脑图：依旧是 MarginNote 3 的「杀手锏」
--------------------------

作为 MarginNote 一直以来的一个核心卖点，脑图在 MarginNote 3 里变得更加好用了。新版加入了一个非常实用的功能：根据文章结构自动将摘录的内容放入脑图中。

在「学习模式」和「文档模式」下，打开摘录设置，你可以看到「自动添加到脑图」这个选项。开启后，当你添加新的摘录时，MarginNote 3 会自动帮你将卡片放到脑图的相应章节里。比如你在 A 书的第一节的第二章摘录了一段文字，那么这个卡片就会进入到脑图的「A 书 - 第一节 - 第二章」中。这对于阅读文献和专业书籍非常有帮助，能够大大节省你做脑图的时间。

![](.evernote-content/C18C63B3-36FB-4704-8E82-A80B7466A23D/D3516E11-D7BE-4740-A09A-336E09B09D78.jpg)

此外，在学习模式下，MarginNote 3 支持通过左侧边缘滑动来拉出大纲，让你不用再像二代一样来回在大纲和脑图中点击切换。

![](.evernote-content/C18C63B3-36FB-4704-8E82-A80B7466A23D/817026F2-C062-435D-8770-BF19A9EB444F.jpg)从侧边滑动拉出大纲

复习模式：突出重点快速回顾
-------------

做完笔记，是大部分同类应用的终点，但在 MarginNote 3 却是一个新起点。只需要点击学习卡片，选择「添加至卡片组」图标，就可以将其加入复习卡片组内。你可以将多个笔记本中的内容添加至同一卡组，方便考试周时突击复习，只重温重点内容。

![](.evernote-content/C18C63B3-36FB-4704-8E82-A80B7466A23D/C376BC34-498A-4E72-BABB-A6917ED1FBB0.png)

为了应对考试，有时你需要在复习时记忆一些定义。针对这个需求， MarginNote 3 支持隐藏学习卡片中的一部分内容，通过「填空题」来进行提问。

![](.evernote-content/C18C63B3-36FB-4704-8E82-A80B7466A23D/8875B9B4-837C-4743-823E-C0B3F42E30F6.png)

MarginNote 3 还支持在一个笔记本内同时查看多份文档。你可以用它来同时阅读论文及脚注，或者复习时对照知识点与题干，免去上下翻阅的苦恼。

![](.evernote-content/C18C63B3-36FB-4704-8E82-A80B7466A23D/53CEBC25-3326-49E5-BC40-B0B298AF70D9.jpg)

如果你习惯使用其它应用进行复习，MarginNote 3 也支持导出学习卡片至 Anki、导出大纲至 OmniOutliner、导出思维导图至 MindManager 和 iThoughts 等应用。

![](.evernote-content/C18C63B3-36FB-4704-8E82-A80B7466A23D/65CC73E5-0ACA-456C-A958-F4A4F6FA1866.jpg)

最后，如果你需要备份数据，MarginNote 3 支持与印象笔记、iCloud 等服务间进行数据同步。新版还增加了回收站的功能，让你在不慎删除文档后有了一颗「后悔药」。更新至 3.0.1 版本后，你还可以从 MarginNote 2 中迁移自己原有的笔记。

![](.evernote-content/C18C63B3-36FB-4704-8E82-A80B7466A23D/55B340B6-FD0E-47DF-88E7-F16393BC830E.png)

总结
--

阅读文档、批注笔记再到复习回顾， MarginNote 3 希望能够在阅读学习的各个方面带给你更出色的体验。与二代相比，三代在功能方面变得更加强大，同时应用的流畅度也有明显的提升。对于每天都要跟文献打交道的人来说，如果你希望让你自己的阅读体验更有条理，MarginNote 3 是一款真正能够帮助你的工具。
