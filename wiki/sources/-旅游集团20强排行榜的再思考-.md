# 旅游集团20强排行榜的再思考

# 旅游集团20强排行榜的再思考

## 

王兴斌 《 中国青年报 》（ 2013年02月08日   11 版）

|  |
| --- |
|  |
|  |

近日，中国旅游协会与中国旅游研究院联合发布了《2012中国旅游集团20强》榜单，这是从2009年以来的第四个年度榜单。综观这4年榜单，可以看出中国旅游企业集团（以下简称“旅游集团”）的现状与某些发展态势：

    第一，连续4年的排位中，港中旅、携程、锦江国际、华侨城、首旅、国旅、中青旅、海航、上海春秋、南京金陵、广州岭南、杭州旅游、开元、华天等10多家，虽然位次有升降，但一直榜上有名、基本稳定，表明中国旅游集团的基本骨干已在激荡起伏的市场大浪中形成，中国旅游正走上企业集团化、本土化之路，民族自主旅游品牌正在发育之中。中国旅游企业正在步入群雄并起、做大争强的新阶段。

    第二，国有企业在“20强”占大半壁江山，一直是“旅企集团”的主体。在10多家国有企业集团中，港中旅、国旅、中青旅是历史悠久的“央企”，作为“中央主力军”，它们的龙头地位至今巍然不动，并以很快的速度向地方拓展。首旅集团、华天旅游、安徽旅游、黄山旅游、浙江旅游、杭州旅游等国企集团作为“地方集团军”，这些年在地方政府的撮合下，成为各地方国有旅游企业的“大哥大”。无论“央企”还是地方国企，它们都“天生”具有资产优势、资源优势、市场优势和政策优势，在历史形成的全国或地方垄断优势的基础上，只有发展快慢、利大利小之分，没有“全军覆没”的风险。港中旅在整合了香港、内地资源的基础上雄居全国旅游业界之首，并与德国易途旅游成立中旅途易旅游有限公司；国旅总社与美国运通集团组建了国旅运通旅行社有限公司，向会展业深入；中青旅向股份制方向深化改革，都在改革中发展取得新进展。但是，这些依托行政优势而生的企业集团也有其软肋，“行政化”的烙印根深蒂固，缺乏“置于死地而后生”的压力与动力。国有旅游企业集团如何在外部与各类社会资本嫁接，在内部建立规范的现代企业制度，彻底“去行政化”、实现完全市场化的改革还有艰辛而漫长之路。现在不少地方热衷于以行政之手整合国有旅游企业组建集团，“小船绑成轮船”但经不起市场风浪的颠波，成功的不多、无效的不少，个中教训值得反思。

    第三，外资、民营和股份制“旅游集团”后来居上，异军突起，迅速进入中国旅游业界的前沿，深刻地改变了中国旅游业国企独大的格局。在近几年的“20强”榜单中，携程、广之旅等外资企业，春秋、开元、大连海昌等民企，海航等股份制企业，它们中历史最长的不到二十来年，最短的才十余年，大多没有政府背景，也没有现成的市场资源、品牌资源，主要靠拼搏、自主创新，在快速、持续兴起的旅游市场大潮中，乘风破浪中闯出一条生存发展之路，形成了旅游经济多种成分并存的格局。这些旅游集团的崛起显示了民营、合资和股份制企业在旅游经济竞争力中具有强大的生命力，可以更加深刻地理解“充分发挥市场配置资源的基础性作用”的意义。

    第四，行业融合、多种经营是“旅游集团”共同的成长之路。这些“旅游集团”各有业态特色和发展模式，铸就了中国旅坛“百花齐放”的发展局面。港中旅、国旅和中青旅以旅行社为主业，向景区、会展、免税商贸等方向扩展。华侨城以主题公园为主业，融合房地产、社区。首旅集团、安徽旅游、杭州旅游整合地方酒店、旅行社、景区、商贸、餐饮等多种行业，综合发展。黄山旅游以景区为基础向酒店、商贸等多种经营。春秋集团以旅行社为主，自创低价航空。海航集团以民航为主业向度假房产扩展。上海锦江、开元集团、南京金陵等坚持把酒店主业做强做精。

    第五，体制创新、科技支撑是旅游集团壮大的必由之路。20强榜单中最显眼的是携程、去哪儿网、同程网络、景域集团等外资、民营的网络服务企业。去哪儿网从第20名跃升至第10名，同程网络进入20强榜单。景域集团从旅游规划咨询起步，快速向景区门票网上销售（驴妈妈）、旅游营地等多业扩展。这些都显示网络旅行商群雄并起，以不可阻挡之势活跃于中国旅坛。

    近年来，更有业内的老牌企业、业外的网络经营商纷纷进入旅游在线经营，并用不同方式推进线上与线下、网上虚拟服务与网下实体服务相结合，机票＋酒店＋景区、团购、垂直搜索、线路直销等多种模式不断涌现。IT旅游企业的群雄并起，展示了以电子信息新技术为代表的现代科技成果与传统业态相融合，并催生旅游服务业新方式、新业态、新趋势的发展方向。

    企业集团是旅游产业的引领者，是产业发展的核心与骨干，是旅游产业应对国际化竞争的必由之路。但是如果对这些20强进行区域分析，东部地区占19家，中部地区仅有1家，西部地区1家也没有。如果放到全国经济产业中去比较，这20强能进入全国500强企业名单的，只有两三家。如果从全球旅游业来考量，这20强基本都在国内经营，走出国门的很少，真正走向世界、进入国际旅游企业的竞争主战场，还有很长的路要走。

    多次公布的榜单沿用了国内外企业“××强”评判惯例，以营业收入为唯一指标，便于进行排列位次。但是这种“一票定座次”的方法，主要是一种数量、规模的比较，而不是质量、效益的比较，在强调转变发展方式的今天显得不完善。在笔者看来，称“20强”并不确切，称“20大”更为恰当。从企业经济学角度看，利润率、全员劳动生产率、人均实现利税等指标更能反映企业的水平与业绩。从操作层面看，利润与税金更是硬指标。在以“营业收入”为排序标准的同时，能否综合考虑利润额、税金额、人均利税等标准，以全面反映企业集团的不仅“大”而且“强”
