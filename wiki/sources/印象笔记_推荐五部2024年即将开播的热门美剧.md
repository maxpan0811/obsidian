# 推荐五部2024年即将开播的热门美剧

速读摘要

《龙之家族》第一季共10集，于2022年8月首播，在《龙之家族》第一季收获好评后，仅一周后就续订了第二季，为HBO带来了破纪录的收视率。而在《龙之家族》第一季故事的末期，坦格利安王朝中最血腥的内战"血龙狂舞"也拉开了帷幕。所谓的"市长"并非市长，而是一个黑白两道通吃，能够摆平各方面关系，维持城市表面平稳的曾经的重刑犯。但是这个虚拟的位于密歇根州的金斯敦，却是一个黑帮横行、暴力不断的惶恐之城，城市里最发达的产业是监狱业。

**龙之家族 第二季**

主演: 艾玛·达西 / 马特·史密斯 / 奥利维亚·库克

类型: 剧情 / 动作 / 爱情 / 奇幻 / 冒险

首播: 2024-06-16(美国)

![](.evernote-content/00AD6961-D1AA-4F28-AE0E-7831EAA35662/37953013-BA82-44C2-BA01-834959EE97BD.jpg)

在《权力的游戏》电视剧大火之后，改编自乔治·R·R·马丁《火与血：坦格利安王朝史》的衍生剧《龙之家族》，再次将人们带往那个纷争不断的维斯特洛大陆。

《龙之家族》第一季共10集，于2022年8月首播，在《龙之家族》第一季收获好评后，仅一周后就续订了第二季，为HBO带来了破纪录的收视率。

![](.evernote-content/00AD6961-D1AA-4F28-AE0E-7831EAA35662/50639138-D191-494B-803B-DE0ED8EDEC29.png)

近日官方终于放出第二季的宣传预告，并宣布会在2024年夏季播出。在加入更多新角色的同时，还将有一些老面孔回归。

《龙之家族》讲述了《冰与火之歌：权力的游戏》故事发生的200年前，聚焦坦格利安家族的内部纷争。而在《龙之家族》第一季故事的末期，坦格利安王朝中最血腥的内战“血龙狂舞”也拉开了帷幕。

《龙之家族》第二季发布正式预告，而首季豆瓣在评分为8.8！艾玛·达西、马特·史密斯、奥利维亚·库克、瑞斯·伊凡斯、伊芙·贝斯特等主演回归，第二季共8集，6月16日起播出。目前该剧已续订第三季、第四季。

**指环王：力量之戒 第二季**

主演: 莫菲德·克拉克 / 查利·维克斯 / 罗伯特·阿拉马约

类型: 剧情 / 动作 / 奇幻 / 冒险

首播: 2024-08-29(美国)

![](.evernote-content/00AD6961-D1AA-4F28-AE0E-7831EAA35662/C841467E-439F-4664-9185-AE3E83D0BBFD.jpg)

亚马逊斥巨资打造的《指环王》剧版《指环王：力量之戒》第一季于2022年9月2日正式开播，官方宣布，最新统计在首播24小时内同时观看人数最高超过2500万，创下亚马逊史上新纪录。

很多观众表示亚马逊毁了托尔金这个大IP，完全对不起史上最贵剧集的称号，糟糕的表演，糟糕的剧本，糟糕的选角，对原著极其不尊重。还有好多人希望亚马逊立即砍了这部剧，打消制作下一季的想法。

![](.evernote-content/00AD6961-D1AA-4F28-AE0E-7831EAA35662/9CA67A80-9484-4C39-9A07-F2D9A7F47DF8.png)

但2024年5月14日《指环王：力量之戒》第二季预告片发布，同时该剧已续订第三季。不管口碑如何，中土故事继续，“黑暗在身后”。2024年8月29日开播。

《指环王：力量之戒》聚焦中土世界第二纪元故事，在《指环王》故事发生的数千年前，聚焦一群角色——既有我们熟悉的，也有陌生的——直面长期以来的恐惧：邪恶重新降临到中土。从迷雾山脉最黑暗的深处，到精灵之都林顿宏伟的森林，到激动人心的岛国努门诺尔，到地图所及最遥远的地方。这些王国和人物将开辟出的传奇，到他们走后很久很久依然会为人传颂。

**金斯敦市长 第三季**

主演: 杰瑞米·雷纳 / 休·狄伦 / 艾丹·吉伦

类型: 剧情 / 惊悚 / 犯罪

首播: 2024-06-02(美国)

![](.evernote-content/00AD6961-D1AA-4F28-AE0E-7831EAA35662/1C015EAE-0CE8-43FD-B6ED-E836072B4895.jpg)

本剧由《黄石》《猎凶风河谷》主创泰勒·谢里丹和休·狄伦打造，讲述在被外人遗忘的密歇根州金斯顿，政治掮客McLusky家族的故事。这个小镇以监狱事业为主，市长迈克(雷纳)运转的体系维持着权力制衡，他是一个复杂、有缺陷的角色，体系中也包括他的家人们。

![](.evernote-content/00AD6961-D1AA-4F28-AE0E-7831EAA35662/ECE144EA-D280-462C-8411-AC26F1159C3F.png)

片名《金斯敦市长》本身就充满了反讽的意味。所谓的“市长”并非市长，而是一个黑白两道通吃，能够摆平各方面关系，维持城市表面平稳的曾经的重刑犯。而所谓“金斯敦”的原名“Kingstown”，以英文直译就是“国王的城镇”，意味受到国王的保护。但是这个虚拟的位于密歇根州的金斯敦，却是一个黑帮横行、暴力不断的惶恐之城，城市里最发达的产业是监狱业。

**黑袍纠察队 第四季**

主演: 卡尔·厄本 / 杰克·奎德 / 安东尼·斯塔尔

类型: 剧情 / 动作 / 科幻 / 犯罪

首播: 2024-06-13(美国)

![](.evernote-content/00AD6961-D1AA-4F28-AE0E-7831EAA35662/1450D197-46F5-40EB-8963-42B7F0BE1324.jpg)

《黑袍纠察队》第四季中，世界正处于崩溃的边缘。国会女议员维多莉亚·纽曼比以往任何时候都更接近美国总统的位置，并受到正在巩固其权力的祖国人控制。屠夫只剩下几个月的生命，他失去了贝卡的儿子，也失去了黑袍纠察队组织领袖的职位。团队其他成员已经受够了他的谎言。面对前所未有的危险，他们必须想办法齐心协力，在为时已晚之前拯救世界。

目前《黑袍纠察队》第四季还没有开播，官方就宣布第五季确认续订！

**伞学院 第四季**

主演: 艾丹·加拉赫 / 艾利奥特·佩吉 / 罗伯特·席安

类型: 剧情 / 喜剧 / 动作 / 科幻 / 奇幻 / 冒险

首播: 2024-08-08(美国)

![](.evernote-content/00AD6961-D1AA-4F28-AE0E-7831EAA35662/F5779329-5CA5-4C7D-9499-D18C22B7E4AC.jpg)

《伞学院》改编自荣获艾斯纳奖的热门漫画和图画小说，该书由杰拉德·威（《My Chemical Romance》）创作编写、加布里埃尔·巴绘制。第一季于2019年2月15日在美国首播。

尽管网飞已经宣布了将续订《伞学院》的第四季，这将继续哈格里夫斯一家的故事，但下一章将是该系列的最后一章，该剧的所有核心演员都将重新扮演他们的角色。

![](.evernote-content/00AD6961-D1AA-4F28-AE0E-7831EAA35662/9DFE159E-BD28-4E53-8871-076B3265E04E.png)

在给 Variety 的一份声明中，剧集主创 Steve Blackman 表示，在第 3 季让这对兄弟姐妹在新的时间线中被困在没有超能力的情况下，《伞学院》的第 4 季将给哈格里夫一家的旅程带来一个“合适的结局”。

虽然网飞对《伞学院》的改编已经超过了原版漫画，但 Blackman 分享说他知道 Gerard Way 书中为主角一行所创造的未来。
