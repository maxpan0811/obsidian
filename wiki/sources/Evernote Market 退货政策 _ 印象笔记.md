Evernote Market 退货政策
====================

我们承诺：

* 自用户收到商品之日起，7天(含)无理由退货，15天(含)无理由换货。
* 所有商品保修期为1年。

办理退换货及返修申请，请拨打 Evernote Market 客服电话 400-898-1000 ，或提交[用户支持](https://www.yinxiang.com/contact/support/)

具体退换货标准如下：

|  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- |
| 退换类别 | 详情 | 是否要求包装完整 | 是否支持7天无理由退货 | 是否支持15天无理由换货 | 用户是否承担退换货运费 | 备注 |
| 产品质量相关的问题 | 经由生产厂家指定的售后服务中心检测确认，或经售后人员确认属于商品质量问题 | 否 | 是 | 是 | 否 |  |
| 物流相关的问题(配送错误商品、配送过程中损坏、遗失等问题) | 物流配送和商品打包过程中出现的问题，并经售后人员确认 | 否 | 是 |
