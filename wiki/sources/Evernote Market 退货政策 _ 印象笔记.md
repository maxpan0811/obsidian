Evernote Market 退货政策
====================

我们承诺：

* 自用户收到商品之日起，7天(含)无理由退货，15天(含)无理由换货。
* 所有商品保修期为1年。

办理退换货及返修申请，请拨打 Evernote Market 客服电话 400-898-1000 ，或提交[用户支持](https://www.yinxiang.com/contact/support/)

具体退换货标准如下：

|  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- |
| 退换类别 | 详情 | 是否要求包装完整 | 是否支持7天无理由退货 | 是否支持15天无理由换货 | 用户是否承担退换货运费 | 备注 |
| 产品质量相关的问题 | 经由生产厂家指定的售后服务中心检测确认，或经售后人员确认属于商品质量问题 | 否 | 是 | 是 | 否 |  |
| 物流相关的问题(配送错误商品、配送过程中损坏、遗失等问题) | 物流配送和商品打包过程中出现的问题，并经售后人员确认 | 否 | 是 | 是 | 否 | 审核期间可能需要用户提供实物照片，以便售后人员尽快裁定、处理 |
| 其他原因 | 除以上两种情况外，因个人原因导致的退换货，需确保商品包装完好，手册、配件齐全并提供发票(如已开具) | 是 | 是 | 是 | 是 | 退货运费将在退款中扣除 |

注意，以下情况不予办理退换货：

* 非中国 Evernote Market 购买的商品；
* 过保商品(超过三包保修期的商品)；
* 由误用、碰撞、疏忽、进液、改动、未经授权的维修、不正确安装导致的商品质量问题，或撕毁或涂改标贴、防伪标记；
* 根据以上表格，在需要时无法提供商品发票(如已开具)、原包装、保修卡或相关材料的。
