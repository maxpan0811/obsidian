# 让微博变成我想要的样子：5 款 Chrome 扩展让社交网站更加清爽好用

---

![](.evernote-content/4C366BB4-3E9C-4F64-8D5D-84F769255767/8CD95FC4-E349-4C91-BF39-546ECDC3601D.jpg)

不论你是在新浪微博上了解朋友近况还是浏览各种段子，或者是去 Facebook 和 Twitter 上看一看外面的世界 follow 一下自己喜欢的明星，社交网站几乎占据了我们大部分的上网时间。

虽然网站内容每天都在更新，但是天天盯着一模一样的网页 UI 看久了，难免会出现审美疲劳；而新浪微博也因为加入了各种各样用不到的功能，而变得臃肿不堪，每次打开都要加载半天。

既然每天都要花大量时间浏览这些网站，我们当然希望它们可以变得更加符合自己的使用习惯，得益于 Chrome 强大的扩展能力，我们或许可以借助这几个扩展，让这些网站看起来更顺眼，用起来更顺手。

[眼不见心不烦](https://chrome.google.com/webstore/detail/%E7%9C%BC%E4%B8%8D%E8%A7%81%E5%BF%83%E4%B8%8D%E7%83%A6%EF%BC%88%E6%96%B0%E6%B5%AA%E5%BE%AE%E5%8D%9A%EF%BC%89/aognaapdfnnldnjglanfbbklaakbpejm)：高度自定义新浪微博
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

不喜欢新的大图模式？不想看友邻那些古灵精怪的点赞？对于越来越臃肿的微博，我们还是眼不见心不烦吧。「眼不见心不烦」由国人开发，可以高度自定义微博的显示样式。安装成功后在页面右上角的「齿轮」便可打开设置页面。

![](.evernote-content/4C366BB4-3E9C-4F64-8D5D-84F769255767/FBE265F5-DB74-4D98-926A-45ABC0BC6183.jpg)

另外这里还有个适合多个浏览器的脚本 [Yet Another Weibo Filter](https://tiansh.github.io/yawf/zh-cn.html)。

[Mata](https://chrome.google.com/webstore/detail/mata/oakhnpcgdembaopakdnfkejfcaeipeik)：一键切换夜晚模式
------------------------------------------------------------------------------------------------

如果你不想搞那么多设置，一键切换到夜晚模式，其实看着也挺舒服的。

![](.evernote-content/4C366BB4-3E9C-4F64-8D5D-84F769255767/F298F59B-C321-40CC-B52B-911E875BA7F4.jpg)

[YouTube Plus](https://chrome.google.com/webstore/detail/youtube-plus/nkdalpbojfdilmiboaiedicdbigdabpb)：高度自定义 YouTube
---------------------------------------------------------------------------------------------------------------------

YouTube 的网站其实是很良心的，只是它的一些设定并不是那么人性化，比如根据带宽来自动判断画质、自动播放视频等。幸好有 YouTube Plus，除了能解决以上问题，还能高度自定义 YouTube 的排版。在此不建议大家屏蔽 YouTube 的视频内嵌广告，因为真的很良心且大多高质量，很多时候我是心甘情愿看完的。

安装之后点击右上角的三角按钮便可打开设置页面。开发者也十分良心，在每一个选项都附上了详细的解释，点解选项后面的「？」号便可跳转到相应的 Github 页面（英文）。

![](.evernote-content/4C366BB4-3E9C-4F64-8D5D-84F769255767/A3CBF5E6-17B6-4F26-B065-638B0B6BF3EC.jpg)

[Facebook Flat](https://chrome.google.com/webstore/detail/facebook-flat/kadbillinepbjlgenaliokdhejdmmlgp)：给你的脸书换个脸
------------------------------------------------------------------------------------------------------------------

Facebook 的页面千年不变，我早已看厌，而且也有越来越臃肿的趋势。该扩展可以给脸书换脸，瞬间清爽不留痕。安装完成后点击页面左上角那个开关即可，可惜不支持自定义。

![](.evernote-content/4C366BB4-3E9C-4F64-8D5D-84F769255767/594D8BA3-9A82-4DA5-B58B-ADEB67DD1DAD.jpg)

[Twean](https://chrome.google.com/webstore/detail/twean/mgdbopghpkjmgnhjfdnfeihnjgndjnbp)：简洁如 Twitter 也不放过
----------------------------------------------------------------------------------------------------------

其实 Twitter 算是简洁的了，如果你连它也不放过，Twean 也可以帮到你。该扩展可隐藏四个板块：（从上到下）图片/视频的预览、时间线里的广告等、推荐关注，以及热门话题。

![](.evernote-content/4C366BB4-3E9C-4F64-8D5D-84F769255767/7C27A2DB-210F-4919-B327-42AD81F50082.jpg)

如果你也有其它类似的扩展，不妨在评论区与大家分享。
