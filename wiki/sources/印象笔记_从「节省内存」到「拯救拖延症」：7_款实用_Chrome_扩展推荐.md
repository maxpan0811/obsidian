# 从「节省内存」到「拯救拖延症」：7 款实用 Chrome 扩展推荐

---

![](.evernote-content/D3222237-7EE5-43BE-B0A5-CB168503F694/A24185F2-E1EF-4928-B4BA-8946D0512A30.jpg)

相信大部分使用 Chrome 浏览器的人都和我一样，不仅因为它简洁的界面和快速的启动速度，各种强大的扩展程序也是我离不开 Chrome 的一大原因。

曾经有一段时间特别着迷于尝试各种扩展程序，今天就来分享一些我觉得可以帮你提升 Chrome 使用体验的扩展。

[The Great Suspender](https://chrome.google.com/webstore/detail/the-great-suspender/klbibkeccnjlkjkiokjodocebajanakg)：休眠标签
--------------------------------------------------------------------------------------------------------------------------

Chrome 虽好，可它却有个致命的缺点，就是非常的吃内存，打开任务管理器就能看到一大串的 Chrome 进程正在分食你的内存。而这款插件则可以把后台闲置的标签页都休眠，并可以将休眠时间设定为从 20 秒到 3 天。被休眠的标签页在这段时间内就不会占用到你的内存，从而让你的电脑处理速度有所提升。

![](.evernote-content/D3222237-7EE5-43BE-B0A5-CB168503F694/76C4831C-6E6C-4E37-AB58-4C07B2D6A4CB.jpg)

[Web Timer](https://chrome.google.com/webstore/detail/web-timer/ggnjbdfgigejghknieofeahaknkjafim)：网页浏览时间统计
----------------------------------------------------------------------------------------------------------

它可以记录你在每个网页花费的时间，每 3 秒会自动刷新记录时间。通过它你可以直观地看出你是否把大量的时间都花在了浏览「无用」网页上，如果你颇受拖延症困扰，想要改善这种情况，那么这个扩展也许能帮到你。

![](.evernote-content/D3222237-7EE5-43BE-B0A5-CB168503F694/B3FB5885-E285-419E-BC38-D6C2A36E57FF.jpg)

[StayFocused](https://chrome.google.com/webstore/detail/stayfocusd/laankejkbhbdhmipfmgcngdelahlfoji)：拖延者杀手
----------------------------------------------------------------------------------------------------------

时间是金钱，虽说这个说法有点陈词滥调，但却是非常正确的。在互联网这个花花世界我们总是很难集中注意力。微博、豆瓣、Instagram……我们很容易就分心在这些地方不能自拔。而 StayFocused 则可以把你从中解救出来，它可以限制你浏览网页时间，给某一个标签页设定 10 分钟的时间限制后，10 分钟后你就再也无法访问该网站且这个限制会持续到第二天，也就是说，一旦启动，你将一整天都会无法访问该网站，直到第二天。

![](.evernote-content/D3222237-7EE5-43BE-B0A5-CB168503F694/B28EB6C0-36F7-4232-AB98-778A8365B1FF.jpg)

[Sexy Undo Close Tab](https://chrome.google.com/webstore/detail/simpleundoclose/emhohdghchmjepmigjojkehidlielknj)：撤销关闭的标签
-------------------------------------------------------------------------------------------------------------------------

简单粗暴的介绍，它就是一款可以恢复最近关闭的标签页的扩展，如果你像我一样，经常会手抖不小心关闭了某一个页面，甚至更糟糕的关闭了整个浏览器，那它简直就是我等手抖患者的福音。

![](.evernote-content/D3222237-7EE5-43BE-B0A5-CB168503F694/F53D2B8C-5026-4442-8D28-5031221B0DD1.jpg)

[Infinity](https://chrome.google.com/webstore/detail/infinity-new-tab/dbfmnekepjoapopniengjbcpnbljalfg)：更强大的新标签页
----------------------------------------------------------------------------------------------------------------

每次打开新的标签页都会显示不同的摄影作品，比起自带的「白板」更让人赏心悦目。 Infinity 的作用不只是当一个赏心悦目的花瓶这么简单，它自带了一些我们最常用的功能，如天气、待办事项、笔记本等，当然你也可以根据个人使用情况添加你所需的，通过 Infinity 你就可以花样玩遍你的 Chrome 新标签页。最重要的是它支持云备份数据！

![](.evernote-content/D3222237-7EE5-43BE-B0A5-CB168503F694/8609FD7B-9E19-4E9D-8C56-0FE6ACDA7621.jpg)

[远方 New Tab](https://chrome.google.com/webstore/detail/dream-afar-new-tab/henmfoppjjkcencpbjaigfahdjlgpegn)
-----------------------------------------------------------------------------------------------------------

不同于功能强大的 Infinity，远方 New Tab 是一款简单的新标签页扩展，每次打开一个新标签则会显示一张赏心悦目的壁纸带你领略各地的美景，同时还提供可配置的搜索框、天气、时钟等一些简单的基本功能。

![](.evernote-content/D3222237-7EE5-43BE-B0A5-CB168503F694/EFA023E5-90F2-4F8A-A83E-BD7449AFBA20.jpg)

[Panic Button](https://chrome.google.com/webstore/detail/panicbutton/faminaibgiklngmfpfbhmokfmnglamcm)：隐藏打开的标签
--------------------------------------------------------------------------------------------------------------

当你在看着一些「不太优雅的东西」时，老板或者同事突然迷之出现在你的身旁。你只要按下「紧急按钮」所有的标签页都会随之消失，再次点击按钮，就可以恢复先前隐藏的所有标签页。

![](.evernote-content/D3222237-7EE5-43BE-B0A5-CB168503F694/1F801CA5-085C-48BA-826B-4406E17A92A9.gif)

Chrome 扩展虽好，可不要装太多。如果你实在按奈不住想装装装的心，那么请谨记**不需要的扩展就关闭，当要使用时才启动**这条人生真理。
