# 特斯拉宣布于中国建厂 Model Y没有方向盘！？

---

昨天，特斯拉在美国举行了2018年股东大会，其创始人马斯克也在大会上公布了不少有趣的消息，比如特斯拉中国工厂、Model Y ，还有所谓的“SpaceX 选装包”，甚至情到深处，马斯克还哽咽地表示自己是在“用爱造车”。

[![](.evernote-content/D94A02E7-D160-4A2E-8E4B-5F19BAC1A834/2AE2C99F-0FDB-4850-92B2-A6D284ABEEAC.jpg)](https://resource.feng.com/resource/h062/h31/img201806071830000.jpg)

我们先来看看特斯拉中国工厂的消息，在5月10日，特斯拉被发现在国内注册了公司，当时就有人猜测特斯拉接近在国内建厂了。昨天的股东大会上，特斯拉表示公司的中国工厂将会落户上海，成为特斯拉在美国之外的第一家工厂。这家工厂将会被命名为“Dreadnought（无畏）”，和此前传言的一样，电池和汽车都会有组装线，但是前期可能只会生产电池，未来才会慢慢加入 Model Y 车型的生产线。

说到 Model Y，昨天的大会上特斯拉也放出了Model Y 的一个剪影图片，这将会是一款小型 SUV 车型，和 Model 3 一样定位于入门款，所以用户们也非常期待。但是入门款并不代表 Model Y 不会有厉害的技术了，马斯克表示：“Model Y 里面没有任何皮革材质，即使是方向盘上都没有，当然，它也得有方向盘。”

[![](.evernote-content/D94A02E7-D160-4A2E-8E4B-5F19BAC1A834/8C7E51C9-BC8E-401F-A64D-5F7AB41B3E29.jpg)](https://resource.feng.com/resource/h062/h31/img201806071830130.jpg)

很显然，这句话给了外界很大的想象空间，如果 Model Y 没有方向盘，就意味着它会是一辆完完全全的自动驾驶车型，这样的车型现在依然是不符合法规的，不知道等到 Model Y 上市的时候，自动驾驶技术能否发展到可以抛弃方向盘呢？大家也别先嘲笑特斯拉白日做梦，其实通用汽车早就宣布会在2019年推出没有方向盘的汽车了。

除了 Model Y 之外，关注 Roadster 跑车的用户也不少，马斯克表示这会是最快的量产跑车，0到100km/h 加速只要1.9秒，续航里程长达965公里。同时马斯克还表示会为这款车型提供一个“Space X 选装包”，能够让性能更进一步，不知道会不会是氮气加速呢？（开个玩笑）

最后在聊到 Model 3 时，可算是戳到马斯克的痛点了，他表示工厂将会在6月底将每周产量提高到5000辆，以挽回那些等不及而退回定金的用户。并且他还强调：“我们用爱生产汽车，真的很用心，其他很多公司的汽车都是由营销和财务部门打造的，根本没有灵魂，但看看特斯拉，我们有员工在工厂连续工作了60天，最后我不得不把他赶回家。”
