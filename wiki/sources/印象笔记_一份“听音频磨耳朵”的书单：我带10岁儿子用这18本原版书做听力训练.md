一份“听音频磨耳朵”的书单：我带10岁儿子用这18本原版书做听力训练
----------------------------------

原创*2017-10-24**射手冬天*[小花生网](http://mp.weixin.qq.com/s?__biz=MjM5MTgzNDM0MA==&mid=2651293057&idx=3&sn=00f85348b1fdaeb8e88657a78836f78c&chksm=bd5cac0a8a2b251ce689522f80bb21696103f58117949c21b5eb640b597288b641dd2c644952&mpshare=1&scene=1&srcid=1101EqIeGxx6ApfrXItuK2Ld##)

2016年3月，我带儿子开始了原版书阅读。英语培养应该先听后读，听要比阅读高两个级别。今年5月，我们才开始正式加强听力训练。这个书单就是边读原版书、边听音频的训练。 暑假时，儿子去美国博物馆听解说能听懂了，说明进步还是不小的。

**那些听音频磨耳朵的书单**

书单制作人 @射手冬天

1

**What If ?**

|  |  |
| --- | --- |
|  | 作者：Randall Munroe  出版社：Mariner Books  出版时间：2014 |
|

看过了中文版本的 What if，有评论说翻译得不是很准确。趁着当当活动购买了原版一本。Tim非常顺利地开始阅读原版的What if，并且读得津津有味，还给妈妈翻译了一段水作用力的解释。偶尔会拿出中文版把不理解的对照看一下。今天又在市少儿图书馆的中学部里找到了原版小说借阅书架，看来原版阅读又可以继续向前一步了。下载了What if 音频来磨耳朵，第一个听完的音频，记录一下。

2

***I Survived#1: I Survived the Sinking of the Titanic, 1912***

|  |  |
| --- | --- |
|  | 作者：Lauren Tarshis  出版社：Scholastic Paperbacks  出版时间：2010 |
|

最近开启磨耳朵模式，听完 What if，第二部选择了Tim喜欢的灾难题材 -I Survived。泰坦尼克的故事是Tim从小喜欢的，愿意听就会有进步。

3

**60秒科学看宇宙**

|  |  |
| --- | --- |
|  | 作者：*JR Minkel，Scientific American* 著  出版社：人民邮电出版社  出版时间：2013 |
|

喜马拉雅下载了科学60秒，每个音频1分钟左右，很适合喜欢科学的娃来练习精听。目前听到的《科学美国人》是语速最快的音频，比雅思还要难。

4

***The Golden Compass***

|  |  |
| --- | --- |
|  | 作者：*Philip Pullman*  出版社：*Yearling*  出版时间：1994 |
|

作者普尔曼是英国当代著名作家，他的代表作《黑质三部曲》，风靡全球图书界。 分别为《黄金罗盘》、《奥秘匕首》、《琥珀望远镜》。《黄金罗盘》赢得卡内基勋章和英国儿童文学最高奖“卫报小说奖”，作者也因此被称为“继《魔戒》作者托尔金之后最优秀的盎格鲁·撒克逊奇幻小说家”，在2005年赢得了第三届林格伦儿童文学奖。 音频收听中。

5

***Harry Potter and the Sorcerer's Stone***

|  |  |
| --- | --- |
|  | 作者：J. K. Rowling  出版社：Arthur a Levine  出版时间：2015 |
|

《哈利波特》作为原版阅读标志性的读物，非常适合10岁左右的娃阅读。这本是《哈利波特》的第一本彩绘本，插图非常精美，读起来比全是文字的有趣味性，值得哈迷阅读和收藏。同时对比了纯文字版和插图版，内容应该一样。可以结合Jim Dale的音频同时输入。
