# OSX技巧:通过终端将屏保设置为动态壁纸

---

[![](.evernote-content/30A14D1B-7A48-4157-AAAA-0B696408F237/435BABDA-4BAA-417E-8DC1-855E575FF7E9.png)](http://www.feng.com/iPhone/news/2015-10-28/OSX-tips-through-the-terminal-set-screen-saver-to-dynamic-wallpaper_628488.shtml)

　威锋网讯，iOS 系统拥有动态壁纸，但是 OS X 系统没有这个“属性加成”，但这并不代表我们就不能够“曲线救国”。今天的 OS X 小技巧将为你介绍如何在 OS X 中通过终端指令将屏保设置为动态壁纸，资深用户可忽略。

![](.evernote-content/30A14D1B-7A48-4157-AAAA-0B696408F237/435BABDA-4BAA-417E-8DC1-855E575FF7E9.png)

**将屏保设置为动态壁纸**

1.打开终端

2.复制并粘贴下列指令

/System/Library/Frameworks/ScreenSaver.framework/Resources/ScreenSaverEngine.app/Contents/MacOS/ScreenSaverEngine -background &

3.点击返回键，屏保就会立刻变成壁纸了

不过遗憾的是，这种方法也有“致命伤”。唯一的问题就是用户不能够退出终端，一旦退出终端上述的设置便会无效。如果想要一直保持动态壁纸效果的话，就只有隐藏终端并让其在后台运行。

由于 OS X 暂时不支持动态壁纸，所以这也是没有办法中的办法。感兴趣又或者是喜欢折腾的朋友不妨尝试一下，如果觉得影响系统性能的话，退出终端即可。
