## 一句话摘要

Codex 和 Claude Code 的核心差异：Codex 胜在手机控制面板+锁屏执行+Computer Use；Claude Code 胜在代码理解力更优+MCP 生态更成熟+Dynamic Workflows 编排能力。

## 对比表

| 特性 | Codex | Claude Code |
|:-----|:------|:------------|
| 厂商 | OpenAI | Anthropic |
| 手机控制端 | ✅ ChatGPT App 当控制面板 | ❌ |
| 锁屏执行 | ✅ 锁屏继续跑 | ❌ |
| Computer Use | ✅ 直接操控桌面/浏览器 | ❌ |
| Skill 管理 | ✅ 图形界面管理 | ✅ SKILL.md |
| Dynamic Workflows | ❌ | ✅ 数百 subagent 并行 |
| MCP | ❌ 不支持 | ✅ 生态成熟 |
| Hooks | ❌ | ✅ 31 种事件钩子 |
| 模型选后后端 | 内置多模型可选 | 可切换 DeepSeek 等 |
| 价格 | $20/月 或按量 | 按 token 计费 |

## 适合场景

- **Codex 适合**：手机远程监督、想随时看进度、不想守在电脑前
- **Claude Code 适合**：深度编程、大型代码库、复杂工作流编排
