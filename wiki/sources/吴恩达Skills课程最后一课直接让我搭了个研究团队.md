# 吴恩达Skills课程，最后一课直接让我搭了个研究团队

![](attachments/4368360bc0e3fc8c.png)

最近我发现一个规律：无论你用什么 AI 工具——Claude、GPT、Cursor、还是各种 Agent——如果你想提高效率，用到最后，本质都是同一件事：

把流程固化

。

你得把自己脑子里那些精华的经验，提炼成一个可以反复执行、减少差错率的东西。这个东西，就是 Skills。

所以我最近花了大量时间研究 Skills。

市面上讲 Skills 的教程太多了，质量参差不齐，为了不浪费时间，我直接找了吴恩达的课程。

听了之后我发现确实很不错——这门课叫「Agent Skills with Anthropic」，由 DeepLearning.AI 和 Anthropic 官
