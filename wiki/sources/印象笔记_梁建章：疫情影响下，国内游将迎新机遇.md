# 梁建章：疫情影响下，国内游将迎新机遇

---

梁建章：疫情影响下，国内游将迎新机遇
==================

南方日报 2020-04-24 10:24:51

|  |
| --- |
|  |
| 资料图片 |

从贵州 “苗王” 装扮，安吉竹海古风白衣，到苏州唐伯虎造型，多造型人设亮相多场直播，6 场直播带货破亿元，携程集团联合创始人、董事局主席梁建章最近变身 “网红”，成为媒体关注的焦点。在疫情影响下，遭受重创的旅游业正在期待复苏。行业何时才会回暖？旅游业发展将迎来哪些新机遇？日前，梁建章接受南方日报记者采访时表示，出境游停摆的当下，国内游将迎来新机遇。

直播营销或长期存在

南方日报：最近您频繁出现在网络直播中，您亲自来做直播的初衷是什么？直播卖货会成为一种常态吗？

梁建章：我觉得直播要做得好，肯定是是个特别资深的旅游达人，而且一些特别高端的特色酒店，都是需要去亲身体验过才行，这样的人不好找，那就自己来吧。现在对于我来说，当然只能做对公司、对行业比较重要的事。这个事情现在比较重要，是因为现在整个旅游业需要大家去推一把，尤其是中高端的酒店，生意受到很大的影响。我们希望这些酒店业能更多地吸引一些周末周边的休闲度假型旅客，如果这件事做成的话，对整个旅游业加速恢复应该是很有帮助的，当然对我们公司的业绩是非常有帮助。

至于直播会不会长期成为一种消费方式，我觉得也是看不同的行业。现在对于旅游业来说，可能是一个很好的时机，也是比较好的营销手段。尤其是大家晚上没什么事，可以看一看，未来去哪些地方，或者哪些产品是更好的，为家庭多做一些投资。看旅游直播也是一种投资，因为做旅游的计划采购要花很多时间，不如趁现在比较空，先把这个事情做了。

南方日报：现在政府积极推出政策刺激消费，我们注意到携程也与地方政府合作做目的地营销，您认为现在是做目的地营销的好时机吗？

梁建章：我们到各地去，确实相当多的地方政府非常积极地跟我们做这方面的沟通。我觉得这是非常有眼光和前瞻性的一件事。因为，旅游的带动对当地产业是非常重要的。本地人消费，他不买这个买那个，是有一个替代性。外地人消费则是增量，而且能够带动本地人赚钱，本地人赚了外地人旅游的钱，他还会在本地消费其他的东西，这是一个乘数效应。所以拉动外地人来消费，其实是用在刀刃上。而我们正好是能够帮助地方政府，尤其是可以吸引发达地区有消费力的人群来消费。他们把营销或者是促进消费的资源用来鼓励外地人来消费，是非常好的一种方式，我们的合作开展得很好。

长期来说，今年国内旅游还是很有机会的，这个时候也能够更好让国内的高端客户发现国内的有竞争力的、有特色的、创新型的产品。我觉得国内可以做更多营销方面的推动工作。

国内高端创新旅游产品或被关注

南方日报：疫情影响下，旅游业受重创，您预计何时才能复苏？

梁建章：这是大家特别关心的问题，到底是半年、一年还是更长？我觉得早晚会复苏，而且整体复苏以后，休闲度假本来也是一个长期的趋势，也会更加快地到来。一些大的商务会展可能会减少一些，通过网上做也很方便，效果也很好。但我认为这不是特别大的变化，因为整体旅游业还是会以每年 10% 左右的速度增长。

南方日报：古语说 “福祸相依”，受疫情影响，出境游停摆，是否意味着给国内旅游市场带来新机遇？在您来看，整个旅游行业将呈现哪些新的趋势？

梁建章：我觉得国内的一些特别高端、有特色的产品，会在这个时候加速被发现。我现在去的一些地方和酒店，发现它们的品质确实在国际上都是领先的。我们以前可能不太知道，因为它们也是近几年才被开发的。以前大家会慢慢地发现，但通过这次疫情，这些地方会被更多快速发现，国外目的地去不了，这也是一个有利的机会。

另外，旅游的创新可能会处于更重要的位置。现在旅游方面的投资、投入显得更加的兴旺，有不少房地产公司现在都在做旅游地产。但是因为同质化竞争激烈，从而需要很多新的灵感，这是从供给方而言的；另一方面从需求方来看，旅游的人通常喜欢新奇。在舒适基本上保证的情况下，一些主题类的、有创意的旅游体验会加速地发展。这对于整个行业来说，也是一种非常好的健康发展趋势。

南方日报记者 周人果
