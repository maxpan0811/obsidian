这些年，虽然旅游越来越普遍，出行人数也越来越多，但是旅行中出现的各式问题好似也成为了家常便饭。而对于出境旅游来说，“事故”率最高的应该算是东西丢失的问题了。

由于出门在外，很少有人会选择携带大量现金出游，所以财物丢失的情况并不会给人带来太大的困扰。但是，如果护照也顺带丢失，麻烦可就大了，运气不好的直接还会被认为是黑户而导致回国时间延误。难怪，经验丰富的驴友曾戏言：“要像爱护生命一样爱护你的护照！”

**那么，如果在境外真的不慎将护照丢失，该肿么办涅？**

**小龙给你两个“必胜”绝招！！！**

**![](.evernote-content/05C9258A-4309-46BE-B207-2B2DE329BAFB/30583C95-F0CF-448F-8064-7C57FB62C91E)**

**绝招一：**

多数情况下，游客出境游会选择随团。因此，如果发生护照丢失的情况。**切记，**

**要第一时间和领队或导游说明情况，毕竟他们对此类情况经验丰富，可以帮你快速找到应对良策，并完美的解决问题。**

**绝招二：**

如果你是自由行，那么绝招一显然是没有用了。木有办法，那你只能认真记住以下流程了。

**1、****第一时间到当地警局挂失**

发现自己的护照不见时，切记不要慌。最好先回忆护照在哪里被偷的，第一时间回去找！要知道，国外有“职业道德”的小偷，一般不会拿钱和卡之外的东西，你可以在丢失地周边找找，不要放过垃圾桶，还要通知现场的警察和工作人员。如果够幸运，说不定就能从中翻出护照来。

如果真想不起来护照是在哪被偷的，一定要尽快去当地的派出所或警察局报案挂失。如果是自己不幸丢失，也应该向当地警方报警。因为若是警察迅速破案，或者有人捡到，这些证件就会归还原主。

|  |
| --- |
| **特别注意**：    为以防万一，在出行之前，最好能够先对护照、签证、个人身份证、机票、户口本等重要信息进行拍照备份（可以选备份在手机、电邮中）。而如果你之前忘记备份，那可以联系国内亲人，让家人传真到你所在的酒店。  此外，警察出具的报案材料是作为日后补办护照及证明游客在该国合法入境居留的凭证之一，也是向当地移民局申请出境签证时备用的材料，所以报案证明记得要妥善保管好。 |

**2、****申请补办护照或旅行证**

报案后，要携带丢失证明、护照复印件以及本人照片到所在国的中国大使馆或领事馆补办护照或旅行证。补办护照需要3个月至半年的办理周期，因为出外旅行的人是呆不了这么长时间的，所以建议办理旅行证。正常情况下大使馆办理旅行证需要4个工作日，加急只需要2个工作日，即在你提交资料的第二天就能领到。旅行证是替代护照的证件，具有护照的完全等同效力。一般情况下，不做特别说明，旅行证有效期是两年。

|  |
| --- |
| **特别注意：**    补办旅行证是目前最为快捷的办法，所以一定要按照此流程来办理。千万不要听信谣言，为了省事或是快捷而误入陷阱，不仅对事情的解决没有任何帮助，可能还会遭受经济及时间的损失。  此外，旅行证和护照等同大小，除个人信息外都是空白签证页。原则上，空白旅行证只能做回国使用，不允许做跨国旅行。而且，空白旅行证必须和护照复印件及所在国警方出具的护照丢失证明共同使用。只出示空白旅行证，上面并无特殊说明，查证件人会仍然认为你持有的是空白护照，而无警察证明的话仍然会被认为是持空白护照非法入境。  所以，警方出具的护照丢失证明非常重要，千万不可丢失。同时，因为空白旅行证继续旅行情况复杂多变，建议最安全的方式就是从颁发旅行证的国家直接回国。 |

**3****、回国补办护照**

依据相关规定，对护照已经丢失的情况。失主回国后，需要回户口所在地的出入境办证中心，提供证件遗失时在当地的报警纸、旅行社提供的相关证明，或者按照相关部门规定，在指定报纸刊登遗失启事的资料，三项资料中选交其中的一项。其中报警纸无疑是最简便的方法。

如果遗失护照上有可能还拥有某些国家的有效期的签证。那么，按照美国大使馆、加拿大大使馆的规定，丢失签证要重新经过预约到大使馆面谈等环节，才能补贴。而像其他国家则可以选择前往大使馆或驻广州领事馆转帖有效签证到新护照上。
