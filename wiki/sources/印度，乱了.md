近三个月来，印度全国各地集会抗议和示威游行不断。特别是2月23日以来，以穆斯林为主的抗议者与印度教徒和警察之间的冲突愈演愈烈。

恰在此时，美国总统特朗普开始访问印度。冲突过程中，仅印度最大城市，首都德里东北部区域发生的骚乱，就造成至少42人死亡，近360人受伤，参与者高达上万人。有暴徒在学校追杀穆斯林学生，印度警察袖手旁观，甚至印度教徒拿着武器走出校园都无人阻拦，以至于土耳其总统埃尔多安都遥遥惊呼“印度正在对穆斯林进行‘大屠杀’”。

印度新德里抗议街头（2020年1月）

（图片来自：Sunil prajapati / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/34FE124D-F7D4-413F-AE67-D43597994462.jpg)

那么，何种原因导致人数高达2亿的印度穆斯林群体如此愤怒？特朗普的到来是否与冲突更加激烈有密切关系呢？

修正案触发宗派冲突

近期印度爆发的连续示威抗议和不断升级的骚乱，直接诱因是12月11日印度议会通过的《公民身份修正案》。

《公民身份修正法案》(CAB)

以印度东北部为起点，蔓延为全国各地的长期抗议

（2019年12月11日，古瓦哈蒂）

（图片来自：shutterstock@David Talukdar）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/0C922AD5-2A5B-4146-B83F-40E58F5EED9F.jpg)

法案规定“给予2015年之前因受宗教迫害逃离巴基斯坦、孟加拉国、阿富汗三国来到印度的六大族群以申请加入印籍的机会”。这些族群包括印度教、佛教、锡克教、耆那教、拜火教和基督教。表面来看，这是莫迪政府的一项惠民工程，但却唯独将穆斯林群体排除在外。

印度教、佛教、锡克教、耆那教、拜火教和基督教

你们都有机会，但伊斯兰教除外▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/FF672BB4-2C4B-441D-BF08-CB9F42AC68E1.jpg)

然而众所周知，巴基斯坦、孟加拉国、阿富汗三国

都是以穆斯林为主的国家

印度这个政策的宗教倾向性很明显了▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/E5D16B92-EF84-4A15-9B27-BF865F9A6819.jpg)

这样争议颇多的政策，是历史和现实多方面作用的结果。

一方面，穆斯林群体与印度教徒本身就有历史矛盾。两大不同信仰群体在印度次大陆已共存了14个世纪之久，早在伊斯兰世界的倭马亚王朝时期，阿拉伯民族就开始进入印度腹地。随着11世纪崛起的突厥人在印度的武力征伐，伊斯兰教在印度迅速传播。1206年，出身奴隶的突厥人库特布丁·艾伊拜克以德里为中心，在北印度建立了德里苏丹国家。

如果说历史上的德里苏丹国没能统一印度半岛

其实现代印度也无法做到

（半岛东部的孟加拉也是在这一时期伊斯兰化）

（图为德里苏丹国在Tughlaq王朝时期的全盛规模）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/F9A899A4-00B4-4F79-8763-467AB7AD021E.jpg)

在征服过程中，穆斯林对印度教和佛教遗址进行了大肆破坏，并强迫当地民众改宗伊斯兰教，在政治、经济和思想文化方面对非穆斯林群体进行了非公正对待，而这就埋下了此后印度教徒和穆斯林冲突的种子。

比如印度的国家级奇观-泰姬陵

其伊斯兰和波斯风格是非常显著的

（图片来自：Jan Willem van Hofwegen / shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/65B43161-872D-4F58-8D76-3DCA43E34801.jpg)

随着近代以来印度被英国殖民和整个伊斯兰世界的衰落，穆斯林群体的社会地位在印度不断降低。而人口占多数的印度教高种姓阶级，却因为很快掌握外语而迅速进入殖民当局的行政和决策圈，这导致穆斯林群体和印度教低种姓者越来越处于社会底层与边缘地带。穆斯林和印度教徒的社会地位最终发生翻天覆地的变化。

真纳与甘地背后站着的利益群体是有所不同的

所以即使不是英国插手，印巴也是大概率要分治的

（图片@Kulwant Roy）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/5B9A7448-F047-4264-9E4B-E704916E9B80.jpg)

社会身份的显著剥离感让穆斯林群体对社会现状极度不满。二战后发生的印巴分治实际上就是印度教和伊斯兰教的一次大分离。

双方从此是冤家

（图片来自：Hari Mahidhar / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/69434E2C-C6DB-4796-9651-E268BFBBA79A.jpg)

另一方面，印度教等级观念和穆斯林平等主义也有明显的文化冲突。

在两种宗教的训规下，在印度现实社会中，两大群体对民众社会地位的看法是完全不一致的。印度教徒中的很多人认为，人的存在和社会地位是先天注定的，有差别的四大等级（婆罗门、刹帝利、吠舍、首陀罗）是与生俱来的社会规则，低种姓者和贱民没有资格享有正常人应有的社会尊严。穆斯林便被视为外来的“低种姓者”。

而且这种姓身份还能父子相传

真正做到永世不得翻身

（图片来自：R.M. Nunes / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/E6295BC4-F8F9-4783-9AE8-B56A4BF22AA0.jpg)

而伊斯兰教则认为“真主面前信徒皆平等，不因肤色、能力和财富多寡而区别对待”。由此，伊斯兰教吸引了大量的低种姓的印度人改宗。而改宗后的印度新穆斯林便举起理论上的平等大旗，向政府表达就业和参政诉求，向过去看不起自己的印度人吹响反攻号角。

真主面前，木有高低

（印度-加尔各答-斋月）

（图片来自：Saikat Paul / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/F4849314-EA36-4C7D-AFFA-453095C824B2.jpg)

两者的矛盾在多种因素加持下激化也就不足为奇了：印度教徒认为历史上穆斯林的入侵和统治对他们造成过严重伤害，因而对其持有敌视心理。而穆斯林深感权力旁落，当前与其竞争生存和政治资源的正是印度教徒，因此对他们怀有戒备和不满。

而莫迪政府的长期执政及其印度教倾向

也令印度国内不少穆斯林群众较为不安

（印度 -瓦拉纳西-集会上的莫迪支持者们）

（图片来自：arindambanerjee / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/9C4D88A6-B716-4F1F-A55D-CFA7E3B5F21B.jpg)

在这样的背景下，社会上蔓延着印度教与穆斯林的不满情绪，两者随时可能爆发冲突。

莫迪政府的“老谋深算”

颁布如此容易引发宗教冲突的法律文件，印度莫迪政府不可能不知道所能产生的不良影响有多大，但为何执意如此呢？

实际上，莫迪政府的利益盘算决定了其必然要逆水行舟。

一项政策总会有人满意，有人反对

但最重要的是巩固自己的基本盘

（印度-瓦拉纳西-路演中的莫迪车队）

（图片来自：arun sambhu mishra / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/6AF3EB67-4DEC-4743-B251-343196759CF5.jpg)

今年70岁的印度总理莫迪，可以说是深谙处理民族主义和教派主义政治手段的资深政客。由于印度80%的人口是印度教徒，这使莫迪领导的印人党坚持印度教传统，坚决反对穆斯林群体社会地位的提高和权力的增长，以迎合国内印度教徒来提升自身政治地位。

印度教能在国内占据很大的数量优势

主要也得益于建国时的印巴分治

（巴基斯坦+孟加拉人口接近4亿）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/7C432F17-D837-4875-924B-5A5A7B7CEF3B.jpg)

在2019年8月，印人党组成的政府就废除了印控克什米尔的特殊地位，以方便印度教徒前往定居，其考虑的重要因素便是该地穆斯林占人口多数。

虽然中央政府对该地区的掌控越来越牢固

但时不时还是会出些问题

（印控克什米尔-政府军与群众间的对抗）

（图片来自：Umar Fayaz Dhobi / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/63F69CD7-38C9-4189-B93D-84F1804244A8.jpg)

此外，莫迪政府认为穆斯林骚乱的影响范围是可控的。

由于伊斯兰教最早在印度北部展开征服，穆斯林统治的中心也在北方，因此穆斯林群体大多分布在北部，属于偏居一隅的势力。此外，多年来印度还对穆斯林进行了严格控制，使得穆斯林分布十分散落，往往有穆斯林生活的地方，附近就会被印度教徒的居民区包围，这就起到了代替政府进行监督的作用。

看着你，一直在看着你

（图片来自：nisargmediaproductions / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/D3C49C06-F13F-49B5-AA88-02047518CAE6.jpg)

在此次的抗议示威活动中，人口多达2500万的德里地区竟然难以组织起十万人规模的穆斯林集会。

而且，印度穆斯林主要是仰逊尼派中的温和派，再加上印度政府多年的同化，他们抗议政府的目的在于提高自身地位而不是坚决反对政府。这使莫迪政府认为，在人口占绝对多数的印度教徒面前，这些穆斯林不会起多大风浪。

信念护体，有何惧哉

（图片来自：arun sambhu mishra / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/4E0DEFBE-FDD5-4233-8C2C-7629759F10EF.jpg)

最后，这还是反击印度国大党的绝好机会。近年来，印度经济增长放缓、失业率大幅攀升、强奸犯罪难以遏制，人们对莫迪政府颇有怨言。拥有135年历史的主要反对党国大党便成为莫迪人民党强有力的竞争对手，毕竟从印度独立到2014年的大部分时间，国大党一直是执政党。

印人党（BJP）是大莲花

国大党（INC）是大手

（图片来自：arindambanerjee / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/325E4ABF-6624-46A1-BB65-81972A0CD6B3.jpg)

该党主张多元世俗，反对印度教传统，多年来对穆斯林群体和其他少数族群进行支持和特殊照顾，在少数派人民心中地位特殊。而近期该党对腐败和强奸问题中在位政府的不作为进行的强烈谴责，极大地提升了自身的社会形象，这对莫迪的改革是个巨大的干扰因素。

国大党作为老牌大党

有着雄厚的政治背景，但也有着沉重的政治包袱

（最近的两次大选，国大党都是惨败给印人党）

（图片来自：Filip Jedraszak / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/FFA359E1-526A-46CB-8B1B-80DA48B8BAF1.jpg)

早在去年12月颁布修正案引发抗议运动初期，莫迪政府就指责国大党参与策划示威游行，意图推卸责任，并鼓励印度教徒支持者进行举报。果然，很多印度教徒认为国大党为了谋取权力在印度制造混乱。

不得不认，号召力是真的强

（图片来自：Ankur bajpai / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/D87CB4B2-7D6F-41BB-993A-4B3E4C1AFB42.jpg)

特朗普来了激化矛盾

就在2月23日德里爆发血腥冲突之时，美国总统特朗普却选择于25日访问印度，这无疑激化了宗派矛盾，加剧了当前的抗议运动风险。

特朗普访印的最大目的在于为美国即将到来的总统大选助力，通过强化美印关系来转移国内矛盾，维护国际形象，给自己加分。而选择在该关键时间点去印度，在为莫迪政府打气的同时必然激化印度穆斯林的不满，因为特朗普也是反穆斯林的。

来给老朋友打气加油

（图片来自：Madhuram Paliwal / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/1A5469A6-2152-41CB-AE30-2B1F75B1E39A.jpg)

贴心的莫迪政府为了迎接特朗普，做了三件“羞愧之事”：第一是给臭气熏天的德里市河流换水、制造“印度老杭州”；第二是为了展示民主形象，在特朗普车队途径区域粉刷墙壁以掩盖层层叠叠的德里贫民窟；第三便是在能容纳十几万人的板球场举办欢迎特朗普的集会，而不远处正是穆斯林和印度教徒的冲突区域。

不过多少还是会有些不和谐音

（图片来自：Saikat Paul / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/07FAABE2-ABB5-42DA-A13C-0935D14837A9.jpg)

这一切虚假的美丽让携妻带女的特朗普甚为感动，不仅大夸“印度民主、包容、自由，给其他国家树立了榜样”，还对暴力事件采取了完全无视，宣称印度自己去解决，不干涉。

专业的微笑，专业的握手

（图片来自：Madhuram Paliwal / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/A2CF7444-FDAB-4332-929A-02F31779E056.jpg)

叙利亚内战时平民死伤十几人，特朗普就宣称要制裁巴沙尔政权，现在则选择完全忽视印度的国内矛盾，反差之大，以特朗普的人设都有些解释不过去。而当特朗普飞回美国之后，印度的见闻也被他写成“美印关系强劲，与莫迪关系密切”发表在推特上。

推特外交

（图片来自：twitter@Trump）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/957F5A28-2F3B-4949-AD7A-F627B0A26F18.jpg)

不知道美国民众怎么想的，反正印度穆斯林是被激怒惹出了一地鸡毛的。

而这只能由莫迪政府自己扛。

实际上，美印关系并没有得到任何实质性加强，因为此次会晤并无多少贸易和军售新协定签署。尽管长期以来美国希望印度成为其主要防务伙伴，但莫迪往往选择去俄罗斯订购武器。去年莫迪政府就购买了S-400防空导弹防御系统，引发美国强烈不满。

一个老朋友，那肯定是不够的

（图片来自：Matias Lynch / Shutterstock.com）▼

![](.evernote-content/DC52C758-4762-4A5E-B52C-2A546A409466/A376AD94-5B89-475C-A9B7-8ABAB26B4783.jpg)

显然，莫迪政府在俄罗斯和美国两方之间寻求最佳利益平衡点。

在未来应对国内冲突的问题上，成功获得占主导地位的印度教徒支持，莫迪必将收割更多选票。而印度穆斯林的地位问题恐怕难以得到解决，印度社会的稳定和长远发展将受到宗派问题越来越多的阻碍。
