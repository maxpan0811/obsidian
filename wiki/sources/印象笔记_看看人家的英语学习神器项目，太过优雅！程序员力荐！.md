qwerty-learner是啥
----------------

`qwerty-learner`是一款专门为键盘工作者设计的单词记忆与英语肌肉记忆锻炼软件。它能让你在打字的同时学习英语单词，一举两得！这对于像我这种程序员来说简直是福音，不仅提升了英语水平，还让打字变得更有趣啦。

![](.evernote-content/A38C7D80-D248-4630-B809-FB30E52C0DBC/40671B07-9176-4E90-A540-3C371BDA6241.png)

适合人群：

* 学生党：备考四六级、考研、雅思托福，想高效背单词 + 练打字的；
* 程序员：想熟悉 API 拼写、提高代码输入速度的（尤其是刚学编程的新手）；
* 办公族：经常写英文邮件、报表，想提升打字效率的；
* 开源爱好者：想参与开源项目，贡献代码或词库的（星星数这么高，写进简历超加分！）。

已经有 18.6K 开发者给项目送出了 star，非常受欢迎！

![](.evernote-content/A38C7D80-D248-4630-B809-FB30E52C0DBC/F42E1ED1-BC03-4D68-9D6B-7BF28EAF1D90.png)

功能特色
----

* 丰富的词库：这个项目内置了海量的英语单词库，从CET-4、CET-6到GMAT、GRE，再到IELTS、SAT、TOEFL，考研英语、专业四级英语、专业八级英语，应有尽有！更贴心的是，它还为程序员们准备了常见的英语单词和多种编程语言API的练习，让你在编程的同时也能巩固英语技能。![](.evernote-content/A38C7D80-D248-4630-B809-FB30E52C0DBC/961E3C6B-F681-4B45-891B-9A7B40CB7371.png)
* 肌肉记忆锻炼：qwerty-learner的设计初衷就是帮助我们建立英语输入的肌肉记忆。它通过让用户反复输入单词，来加深手指对键盘上英语字母位置的记忆。而且，如果用户单词输入错误，软件会要求重新输入，直到输入正确为止，这样就能有效避免形成错误的肌肉记忆啦！
* 发音与音标显示：在学习单词时，发音和音标可是非常重要的。qwerty-learner贴心地为用户提供了发音功能和音标显示，让你在记忆单词的同时，也能学会正确的发音，真是一举两得！

![](.evernote-content/A38C7D80-D248-4630-B809-FB30E52C0DBC/14FE39B8-0598-476E-B883-DCE2E1E0F05C.png)

* 默写功能：章节练习结束后，qwerty-learner还会提供默写功能，让你进一步巩固学习的成果。听写错误的单词还会被统计到“错题本”里，方便你日后复习。

![](.evernote-content/A38C7D80-D248-4630-B809-FB30E52C0DBC/43CC4048-A5A9-42CD-ABD7-62FED644F011.png)

* 实时数据反馈：软件会实时显示你的输入速度和正确率，让你对自己的进步一目了然。看着数据和排名一路飙升，是不是超有成就感？

![](.evernote-content/A38C7D80-D248-4630-B809-FB30E52C0DBC/93D27775-642B-4034-8E4B-D0945B140C48.png)

安装指南
----

1、懒人版：直接在线玩！

啥都不用装，打开浏览器输入官网地址：`https://qwerty.kaiyi.cool/`，选个词库就能开练，适合想快速体验的宝子们～

2、进阶版：本地部署（以Windows为例）

如果你想深度使用（比如贡献代码），可以本地跑起来，步骤超简单：

先装Node.js、Git、Yarn（官网下载就行，跟着提示下一步）。

打开命令行，输入：

```
git clone https://github.com/RealKai42/qwerty-learner.git  
cd qwerty-learner
```

接着装依赖，输入`yarn install`，喝杯茶等它装完～

最后启动项目，输入`yarn start`，浏览器会自动打开`http://localhost:5173`，搞定！

3、Mac用户专属一键脚本。打开终端，进入项目目录，输入：

```
./scripts/install.sh
```

脚本会自动用Homebrew装环境+启动，简直是“苹果党福音”！

小结
--

`qwerty-learner`是一个非常实用且有趣的英语学习工具。它不仅帮助我们快速记忆英语单词，还能在输入过程中锻炼英语键盘输入的肌肉记忆。无论是学生、上班族还是程序员，都能从这款软件中受益。而且，它还是开源的，大家可以自由使用和贡献代码。如果你也想提升英语水平，不妨试试 Qwerty Learner 吧！

