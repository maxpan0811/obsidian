# OpenAI Codex大更新

> 来源：新智元

## 一句话
Codex推出Appshots（双击Command读全屏）、/goal自主编码毕业、Mac锁屏远程操控，ChatGPT同时杀入PowerPoint。

## 关键更新

### Appshots
- 双击 Command 键，AI 读取当前窗口全部内容（包括未滚动到的隐藏文本）
- 60秒内连续操作自动追加到同一线程
- 企业版即将开放

### /goal 自主编码正式毕业
- 设定里程碑目标→Agent自主推进（小时级到天数级）
- 可中途查看进度、调整方向、暂停
- 从"一次性交互"进化为"长周期自主工作"

### 锁屏远程操控（Locked Use）
- Mac 锁屏/睡眠状态下，手机端 Codex 可安全操控电脑
- 配合 Codex 移动端，7×24h 不下线

### ChatGPT for PowerPoint
- 一句话生成可编辑 PPT
- 从 Gmail/Outlook/SharePoint 拉取实时数据
- 全球 Beta 测试中

### 数据亮点
- 周活跃开发者：300万→400万（两周内）
- 50% 用户用 Codex 不写代码（自动化、跨工具协作、远程操控）
- Business/Enterprise 用户数半年暴增6倍
