新手要适应OS X系统可能需要一点时间，不过不要紧，有各种新手教程为你导航。

　　威锋网讯，OS X 和 Windows 系统在各个方面都存在着不大不小的差异，对于部分由 Windows 转向 OS X 系统的用户来说，他们需要适应的有很多，其中就包括了在 OS X 中完全地卸载应用。  
  
　　如果想让应用完整地从系统中删除的话，可能需要几个步骤，今天我们不妨来分享一下锋友@夏虫不可以语冰为我们带来的教程：  
  
　1.Activity Monitor  
  
　　打开活动监视器（Activity Monitor），在菜单栏中（上方）选择［所有进程］，确保你要卸载的 APP 已经不再运行，如果看到监视器中该 APP 依旧有活动进程，则需手动在活动监视器中结束此应用进程，之后关闭再检查一次。

![](.evernote-content/625B6A04-D781-42C8-9CDA-3F71BBD81DF0/FD13DD39-73EA-480A-99F5-77E1A3F14260.png)

  
　　备注：有些应用程序登陆时会自动打开，则需要先查以下路径删除有关附件。  
  
　　系统偏好设置> 用户组与群组>个人登陆项（如图）  
  

![](.evernote-content/625B6A04-D781-42C8-9CDA-3F71BBD81DF0/9087419D-0AB8-455D-9861-6140780351E9.png)

  
　　打开Finder, 前往 > 前往文件夹  > /Library/LaunchAgents  
　　打开Finder, 前往 > 前往文件夹 > ~/Library/LaunchAgents  
　　打开Finder, 前往 > 前往文件夹  > /Library/StartupItems  
  

![](.evernote-content/625B6A04-D781-42C8-9CDA-3F71BBD81DF0/CDC0D470-71D3-43F2-A5BC-7F5943E77B74.png)

  
　　移除此应用登录相关组件之后，重启设备，继续以下操作。  
  
　　2. 打开 Finder, 开在搜索那里输入要卸载的应用名  
  
　　3. 注意要用列表方式进行搜索文件，搜索范围调整为整个 Mac  
  

![](.evernote-content/625B6A04-D781-42C8-9CDA-3F71BBD81DF0/9E146B67-3365-406A-905F-E99DE7857BC2.png)

  
　　4. 单击搜索选项下面“＋”按钮，然后选择 Name > contents >输入要删除的APP  
  
　　5. 继续单击旁边“＋”按钮添加 >系统文件 >为搜索的附加条件  
  

![](.evernote-content/625B6A04-D781-42C8-9CDA-3F71BBD81DF0/7CF9BABF-4A65-42AA-AC8A-9C08FBED6DE5.png)

  
　　6.  双击不包含转换为包含  
  
　　7.  移除搜索出与要删除 APP，所有的相关此文件（也可全选，拖入废纸篓中）  
  
　　8.  移除后，运行电脑 1 分钟，这是为了确认我们有没有删错文件。确定 MAC 能正常运作后，安全清空废纸篓即可。  
  
　　附：采用此法的原因
