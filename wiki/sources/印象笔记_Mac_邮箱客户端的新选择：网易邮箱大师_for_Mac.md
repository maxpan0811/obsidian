# Mac 邮箱客户端的新选择：网易邮箱大师 for Mac

---

网易邮箱是中国的老牌邮箱了，在 2014 年，网易邮箱推出了客户端品牌「网易邮箱大师」，并逐步完成了针对 Android、iOS、Windows 等各个主流平台的客户端覆盖。

现在，最后一块短板也被补上了，网易邮箱大师正式推出了 Mac 版本，你可以在 macOS 上享受原生的网易邮箱大师体验了。

其实，macOS 上并不乏优秀的邮件客户端，[Spark](https://sspai.com/app/Spark)、[Airmail](https://sspai.com/app/Airmail)、[Newton](https://sspai.com/app/Newton%20Mail) 都笼络了不少人心，然而，它们本地化的不足，往往使一般用户在国内的使用体验大打折扣。比如，国内最主流的 163 和 QQ 邮箱服务，现在都采用了第三方授权码的登录机制，这些客户端并没有针对性地优化和提示，导致用户在无法成功登录时，往往无所适从。

而网易邮箱大师充分考虑了国内主流邮箱服务的情况，并针对一系列中国特有的网络情况和用户习惯，做出了不少改进。下面，我们就一起从设计、添加账号登录、联系人与邮件管理、特色功能四个方面，来看看网易邮箱大师的表现如何。

四栏式设计的网易邮箱大师
------------

和 Windows 平台的交互一致，网易邮箱大师 for Mac 也采用了经典的四栏式设计，让用户在不平操作平台间，保持一致的操作体验：

* 最左侧的黑色功能区：可以切换邮件、联系人、设置三个不同的页面；
* 邮箱账号区：显示已经添加过的账号，并且搜索邮件、写新邮件的入口，也都从这里发起；
* 邮件列表区：显示当前所选择的邮箱账号文件夹内，所有的邮件列表；
* 邮件内容区：显示当前邮件的内容详情，并可以进行回复、转发、移动等操作；

![](.evernote-content/5CE1C9BC-C42F-4B87-9A51-1A3308E696F6/B705C382-0F38-4D71-851C-29C43E204ECF.jpg)

极简而又聪明的添加邮箱流程
-------------

许多人在使用邮件客户端时，常常会卡在添加邮箱账号这一步。一方面，许多邮箱服务商出于安全等考量，增加了双重认证或授权码的机制，常常导致使用网页端的密码登录失败。另一方面，由于不同邮箱服务商对各类邮件协议的支持不尽相同，常常需要用户自行配置邮件的收发协议。

而网易邮箱大师的添加邮箱界面非常简单，默认状态下，只会显示输入邮箱账号与密码的两个文本框。如果你使用的是网易自家的 163 邮箱，输入完成后就能秒登上来看到邮件了。

![](.evernote-content/5CE1C9BC-C42F-4B87-9A51-1A3308E696F6/73A461B6-7F92-4A87-BA9C-EAAE5EF8211A.gif)

不过，不要小看了网易邮箱大师，极致简单的背后，实际上网易邮箱大师的团队对各个不同品牌的邮箱登录，做了大量的优化。例如，当你输入一个 iCloud 邮箱时，界面上就会出现针对苹果「双重认证」机制的提示。同样，在输入 QQ 邮箱后，界面上会显示「使用授权码或独立密码」的提示，而在登录失败时，还会贴心地给出解决方案。

![](.evernote-content/5CE1C9BC-C42F-4B87-9A51-1A3308E696F6/95EB19E2-43B9-416F-A886-8396026FF526.png)

除此之外，在登录不同品牌的邮箱时，网易邮箱大师还会自动从 IMAP、POP3、SMTP 或 Exchange 等邮件协议中选择最合适的收发协议。据网易邮箱大师团队介绍，他们在后台针对十万余种不同的邮箱服务，通过算法自动优化选择出最合适的登录方式，这才使用户的体验变得极致的简单。在大多数情况下，只需要输入账号密码，就完成了邮箱账号的添加。

此外，网易邮箱大师还针对 Gmail 做了相应的优化。在不使用科学上网的前提下，也可以实现 Gmail 的登录与收发邮件。

以联系人为维度，进行邮件管理
--------------

网易邮箱大师除了邮件外，还可以进行联系人的管理，并支持同步网易自有邮箱、Gmail 等邮箱中的联系人。点击左侧边栏的第二个图标，就可以看到跨邮件账号的联系人汇总了。

![](.evernote-content/5CE1C9BC-C42F-4B87-9A51-1A3308E696F6/C21BF85A-6FF6-4FEF-A77B-08EBA5BE6620.png)

虽然联系人这个功能看起来很简单，无非只是显示一些人的通讯信息。不过，网易邮箱大师以联系人为维度，提供了一定程度上的邮件管理能力。

最直观的就是，在联系人、发件人等各处，你可以汇总查看与某一个联系人的所有来往邮件，还可以方便地切换「我收到的」与「我发送的」。

![](.evernote-content/5CE1C9BC-C42F-4B87-9A51-1A3308E696F6/A13140D2-CD5A-494B-B67C-FED4B492B647.png)

同时，针对某一个联系人，你可能一次性批量移动他的邮件到某个文件夹，方便你整理邮件。如果你勾选了「同时应用于新邮件」，还相当于创建了一个收信规则，以后所有的邮件都会自动分类到相应的文件夹。

![](.evernote-content/5CE1C9BC-C42F-4B87-9A51-1A3308E696F6/8DE07D6B-9A0E-4B2A-8893-0F56A05A3D23.png)

不过，网易邮箱大师在移动邮件时，只能在该邮箱账号下的不同文件夹间移动，而不能跨账号移动。

贴心功能：群发单显、来信分类、搜索范围
-------------------

针对使用邮件时的场景，网易邮箱大师还有一些特色的贴心功能。

### 群发单显，让每一个收件人都觉得自己重要

在写邮件的界面上，你可以点击右上角切换到「群发单显」模式。这个模式有什么用呢？常规的情况下，我们应该想像一下，如果你是在向潜在的客户群发邮件，试图推广你的产品时，你希望对方看到你的邮件是发给了一群人吗？当然不，我们希望对方感到自己是最重要的，他是唯一的那个收件人，但这并不意味着我们需要把同一封邮件反反复复地发好几次。

在常规的情况下，我们应该都熟悉了收件人、抄送、密送的规则，即收件人和抄送的所有人之间，可以互相看到彼此都收到了这封邮件。网易邮箱大师提供了「群发单显」模式，可以实现你填写的收件人，每个人收到这封邮件后，看不到其它的收件人，会觉得这封邮件是你一对一发给他的。

「群发单显」模式并不是网易邮箱独有的功能，其实就是我们在其它邮箱服务里常见的「Bcc」，只不过大多数人看到后者，可能一头雾水，网易邮箱大师则在功能介绍和交互上做出了相应的本地化。除了将「Bcc」命名为更通俗易懂的「群发单显」，如果你细心的话，还会注意到一个区别。例如，在 Spark 中你可以同时设定一封邮件的 To、Cc、Bcc 对象，而在网易邮箱大师中，如果切换到了「群发单显」模式，则不再细分为三项，所有的收件人都是 Bcc 对象。

![](.evernote-content/5CE1C9BC-C42F-4B87-9A51-1A3308E696F6/9CD5CD33-64B3-4E04-B449-9FC703C162EB.png)

### 来信分类，自动帮你整理邮件

许多邮箱其实都提供了「收信规则」「过滤器」「智能文件夹」之类的功能，这些功能的本质其实都是相通的，即通过一定的规则，自动化地对邮件进行处理。

网易邮箱大师也支持简单的「来信分类」功能，你可以以发件人、主题或收件人为条件，当新收到的邮箱命中这些规则时，自动将邮件移动至相应的文件夹，或标为已读/红旗，或转发，或删除。

举个例子，比如 [notifications@qq.com](mailto:notifications@qq.com) 经常给你发送一些系统通知类的邮件，对于你来说并没有查看的必要，你可以直接创建来信分类，选择将新收到的此类邮件移动至归档文件夹，并同时标为已读。

![](.evernote-content/5CE1C9BC-C42F-4B87-9A51-1A3308E696F6/CF22D511-9386-4443-828F-2DB56C038949.png)

### 搜索邮件时，支持指定搜索范围

当你的邮箱中有太多邮件时，找到一封历史邮件，有时候是一个非常痛苦的体验。例如，我最近在做一个产品项目，和同事往来的附件中，有相应的开发文档和设计稿。但在搜索时，会显示大量的信用卡账单邮件。

这个时候，网易邮箱大师提供的搜索范围指定，就能派上用场了。你可以指定是搜索整封邮件的全文，也可以只搜索发件人名称，或者只搜索附件名称。在此基础上，下面的文件夹也会相应变动，这样，你可以进一步缩小范围，更快地找到你想搜索的邮件。

![](.evernote-content/5CE1C9BC-C42F-4B87-9A51-1A3308E696F6/E2FBAFAC-2EBD-441E-AABB-D42326460DA1.png)
