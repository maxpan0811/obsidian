# 法律人的 Claude Code 教程（四）：CLAUDE.md 与上下文管理

> 来源：智律积成（微信公众号）

## 记忆系统四层次

1. **用户级** `~/.claude/CLAUDE.md` — 所有项目共享
2. **项目级** `./CLAUDE.md` — 团队共享（提交 git）
3. **本地级** `./CLAUDE.local.md` — 仅个人（自动 gitignore）
4. **Auto Memory** — Claude 自动记录的项目特定信息

## CLAUDE.md 写作四原则

1. 控制在 200 行以内
2. 结构清晰、分点列举
3. 指令要具体、可验证
4. 避免矛盾规则

## 上下文管理命令

`/context` 查看用量 → `/compact` 压缩 → `/clear` 清空 → `/resume` 恢复历史会话

## 实用技巧

- `/init` 快速生成 CLAUDE.md
- `@path` 语法导入外部文件保持主文件简洁
- 提供完整合同审查 CLAUDE.md 配置示例（12 节）
- 分享作者个人 CLAUDE.md（融入可视化和理想类型研究方法）
