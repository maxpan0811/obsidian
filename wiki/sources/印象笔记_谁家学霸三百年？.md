# 谁家学霸三百年？

---

谁家学霸三百年？
========

饶议科学

1970年，诺贝尔医学或生理学奖颁发给研究神经生物学的三位科学家，瑞典的Ulf Svante von Euler，英国犹太生理学家Bernard Katz，美国犹太生物化学家Julius Axelrod。

对于von Euler家族来说，这简直就是小菜一碟。

![](.evernote-content/CF2338C5-8EBE-496B-B347-98E1A8485E03/D0569141-00B0-4BE5-913B-49E652DB5FB5.png)

von Euler是科学世家，其五世祖父为著名数学家欧拉（Leonhard Euler）。那才是他们家族的大科学家。欧拉于1707年出生于瑞士巴塞尔，1783年逝世于俄国圣彼得堡。

![](.evernote-content/CF2338C5-8EBE-496B-B347-98E1A8485E03/8B121BFD-B69F-4815-B679-243E723FB68C.jpg)

​

父亲（Hans von Euler-Chelpin）为1929年化学诺奖得主。

![](.evernote-content/CF2338C5-8EBE-496B-B347-98E1A8485E03/A9D38488-66C7-47AD-9235-BB4EFD4FBD71.jpg)

​

外公（Per Teodor Cleve）发现两个化学元素（钬和铥）。

![](.evernote-content/CF2338C5-8EBE-496B-B347-98E1A8485E03/CD9E21D8-216D-4114-ADB0-F95692CC67F9.jpg)

​

母亲（Astrid Cleve von Euler）是瑞典第一位女科学博士。

![](.evernote-content/CF2338C5-8EBE-496B-B347-98E1A8485E03/4AC43963-2879-4D3F-B43E-38DC9EB8047A.jpg)

​

Ulf Svante von Euler自己在神经生物学有三个主要发现（第一个神经肽P物质、去甲肾上腺素、前列腺素），他1970年获奖是因为确定神经递质是去甲肾上腺素（而非肾上腺素）。

他的学生们(Sune K. Bergström和Bengt I. Samuelsson），因为进一步研究前列腺素而获1982年的诺贝尔奖。

![](.evernote-content/CF2338C5-8EBE-496B-B347-98E1A8485E03/CF6A98D1-4027-48C9-992F-1CB6AB26FB2F.jpg)

​

Bergström的私生子Svante Pääbo，因为用DNA研究人类，而获2022年诺奖。

![](.evernote-content/CF2338C5-8EBE-496B-B347-98E1A8485E03/291BD3A2-1C1E-4154-BFF7-69E73A06E921.jpg)
