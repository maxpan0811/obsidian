# Win 10新10565升级公布 Edge和小娜更智能

---

[![](.evernote-content/F09883EB-5C50-454A-BCB2-9EB49A487532/1560FD06-C7EE-488A-9AEE-FD95011BAB03.png)](http://tech.feng.com/2015-10-13/Win-10-new-10565-released-upgrade-Edge-and-XiaoNa-more-intelligent_626955.shtml)

　威锋网 10 月 13 日消息，今天凌晨，微软针对 Windows 会员计划开启“快速更新”的用户推送了全新的 Windows 10 版本：BUILD 10565。该最新的 Windows 版本带来一系列改进，由于是这几个月来新增功能和改进之处较多的一次，因此据称将是下个月 Threshold 2 升级的预演。

![](.evernote-content/F09883EB-5C50-454A-BCB2-9EB49A487532/1560FD06-C7EE-488A-9AEE-FD95011BAB03.png)

**Skype 深度整合**

![](.evernote-content/F09883EB-5C50-454A-BCB2-9EB49A487532/33942999-A791-47C8-BF14-B58B31BD67A3.png)

首先是针对 Skype 的更新，微软将 Skype 消息、语音和视频应用整合到 Windows 10 系统中了，已激活 Skype 的用户之间可以直接在系统中免费进行文本消息、语音和视频聊天，即将发布的 Windows 10 手机版也将具备相同的功能，使用条件不限 3G/4G 和 WiFi 网络，并且针对“操作中心”的通知提醒功能进行了改进，现在用户可以直接通过“操作系统”快速的回复消息。

毋庸置疑，Skype 通讯体验与 Windows 10 系统的整合，将是微软针对苹果 OSX 系统上 FaceTime 和 Messages 应用的一次回击。

**Cortana 变得更聪明、更智能**

![](.evernote-content/F09883EB-5C50-454A-BCB2-9EB49A487532/67063780-F00E-460C-88F7-D0F94B5481F6.jpg)

新的升级，Cortana 开始支持手写笔记的识别了，即便你的笔记本相对潦草，它也能根据你的笔记设置提醒功能，包括识别你手写的位置信息、日期、电话号码和数字等信息。这个功能有点类似于三星的 Action Memo，利用 S-Pen 也可以做相同的事情。

另外，新 Cortana 也支持电影或其他活动安排的提醒功能了，并可监控和追踪电子邮件中的信息来提前提醒你，在两小时前就向用户发送提醒，提供地图和出行信息，保证不会迟到或遗漏。还有一点，Cortana 也可以实现 Uber快速叫车。

**Edge 浏览器的改进**

![](.evernote-content/F09883EB-5C50-454A-BCB2-9EB49A487532/59B9BC17-BBC1-4070-B48D-5CDCFFE05DFF.png)

很多用户关心的 Microsoft Edge 浏览器这次也迎来了改进和功能性的更新。其中重点在于标签预览功能增加，当用户打开多个浏览器 Tab 标签页时，只要将鼠标放在任意一个选项卡上，就可以快速获取网页的预览，避免实际点击再切换回原视图操作的麻烦。

此外，全新的同步功能终于支持 Edge 浏览器的收藏夹和阅读列表了，同帐户的 Windows 10 设备上，将可以看到相同的收藏夹和阅读列表内容，不用每台电脑上再进行多次收藏，内容会自动迁移。

**视觉上的改进**

![](.evernote-content/F09883EB-5C50-454A-BCB2-9EB49A487532/8AA13F64-A2EF-46DC-A4E4-EB63B9488BA4.png)

在 Windows 10 BULID 10565 版本中，系统界面也发生了明显的变化，首先是所打开窗口的标题栏，现在直接融合主题的配色，视觉上更具活力。当然，这个变化需要手动开启， 在个性化设置里，颜色菜单中选择开启“在任务栏、开始菜单、操作系统和标题栏中显示色彩”即可。与此同时，系统加入了一些新的图标，比如注册表和设备管理器中图标，而开始菜单也已支持在调整尺寸时查看示例，方便快速识别。

![](.evernote-content/F09883EB-5C50-454A-BCB2-9EB49A487532/1806503C-624F-4E6E-87BE-F68CFCF91C76.png)  
  
![](.evernote-content/F09883EB-5C50-454A-BCB2-9EB49A487532/B5433811-EB06-4626-B4F3-1F2F33FBFB19.png)

**其他改进**

新 Windows 10 版本更新还有一些细微的改进，比如激活变得更加容易了。新的 Windows 激活策略保证你在执行 Windows 10 系统全新、干净的安装时，仍可通过输入 Windows 7、8 或 8.1 的密钥进行激活，具体在“设置 > 更新和安全 > 激活中”就有相关选项。另外，现在“右键简单”显示不再过大不适合鼠标操作，支持将人脉中的联系人固定到开始菜单，支持右键更快速的隐藏桌面图标，打印机默认选择最近使用的那台工作等待。 这个版本也有修复之前的已知 BUG，比如 Groove 音乐应用最小化无法背景播放的 BUG。
