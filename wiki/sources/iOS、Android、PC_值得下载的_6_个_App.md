## **不容错过的新 App**

虽然少数派一直在为大家发现和介绍各平台上的优质 App，但仍有不少设计、功能、交互、体验都非常优秀的 App，还没有被我们发掘和介绍。它们可能是一款老 App，也可能是近期上架的新 App，我们会在这里介绍给你。

▍**MarkEdit ：Taio 开发者新作，开源免费的 Markdown 编辑器**

- 平台：macOS
- 关键词：Markdown、编辑器

@化学心情下2：在经历了 Taio 的带来纷争之后，开发者钟颖在近期突然推出了一款基于 macOS 平台的 Markdown 编辑器：MarkEdit ，和很多 Markdown 编辑器寻求商业模式有所不同的是，这款应用完全开源且免费上架了 Mac App Store。

![](attachments/cd5e25612c93bde5.png)

事实上从形态上，这款 Markdown 编辑器又显得有点特立独行：在 Markdown 编辑器寻求跨平台体验的当下，MarkEdit 有且只有 macOS 客户端，并且尽可能遵循了 macOS 原生应用的设计风格。

别的 Markdown 编辑器都在追求「所见即所得」，MarkEdit 更像是 Markdown 代码编辑器，让你无法将其用作写作工具。但相对地则带来了应用功能上的纯粹——像极了系统原生的 TextEdit，不过只适用于写 Markdown 内容；整个应用只有 3MB，却可以快速处理一个 10MB 大小的 Markdown 文档。

![](attachments/b9f96814441fa77a.png)

在使用中你会发现它总是不那么让人觉得顺手，比如习惯了所见即所得的 Markdown 编辑器，你可能只会部分的语法，其他都使用应用内置的便捷功能补全操作，而 MarkEdit 则完全不能，以至于我在添加多级无序列表时有些看不清楚层级结构，甚至你无法实现预览成 HTML，就更不能支持将其转换成其他格式，开发者的意思则是——你可以使用 Pandoc 来转换，而不是通过一个编辑器来实现。

![](attachments/0ad618b818dfe7f3.png)

当然在极度简洁的背后，依旧有一些功能亮点，比如说对于需要进行预览的部分（例如表格、LaTex 公式），MarkEdit 依旧提供了相应的预览方法，只不过采用了弹出窗口的形式进行预览。并且支持语法检查，甚至可以直接使用 Grammarly 进行文本的语法校对。虽然很小巧你还是可以设置多种主题样式，控制行高以及自动换行的形式等等。

MarkEdit 是一款开源的 macOS 平台 Markdown 编辑器，你可以在其 GitHub 主页直接下载使用，也可以在 Mac App Store 上免费下载使用，无任何形式的内购。

![](attachments/3fc6f62186346a5e.png)

▍**Obtainium：更稳定的应用更新追踪工具**

- 平台：Android
- 关键词：应用管理、应用更新器

@Noah\_Choi：开源应用总是我们的首要选择，但它们会分散到商业应用商店、开源市场、托管平台等等各个来源。如何更新它们着实令人头疼。

少数派之前介绍过国内开发者的更新追踪工具 UpgradeAll，不过它运行时依赖后台的软件源，连接稳定性稍显不足。如果你也遇到了此类问题，可以看一下 Obtainium 这款应用，它不依赖于自有软件源，只要托管平台或开源应用商店持续运营，那么我们就可以一站式解决更新追踪的痛点。

Obtainium 使用了 Material You 设计风格，并且适配了动态取色和暗色模式，不得不说 Material You 与开源应用实属绝配。在更新追踪原理上，Obtainium 不需要独立的仓库，而是基于各个应用本身的发布源，其中包括 GitHub、GitLab、Codeberg、F-Droid 等等公开源，还为 ApkMirror 和 SoourceForge 这种下载站提供了支持，支持列表中甚至出现了 Steam，或许我们还可以利用它来追踪喜加一？

![](attachments/1cfe4fd71b8fbbe6.jpg)

由于 Obtainium 本身并不支持自动检测手机中的应用列表，使用时需要用户手动添加应用来源。Obtainium 提供了两种添加方式，其一是搜索，Obtainium 支持搜索 GitHub 和 Codeberg 这两种来源，只需要键入应用关键词即可提供相关库的链接，同时也支持直接导入 GitHub 用户名下的标星仓库，不过这种方式容易导致 Obtainium 无法匹配已安装应用。

所以我建议使用第二种方式，也就是直接填写应用库的主页链接，Obtainium 同样可以解析版本号、架构和下载地址，如果发现有更新，那么它就会在主页给出提醒。

![](attachments/a3436341a3a05a0c.jpg)

除此之外，Obtainium 支持对应用的分组管理、批量更新、筛选过滤等等功能，只是相比于 UpgradeAll，Obtainium 没有类应用市场的探索功能。需要注意的是，由于应用是直接从对应平台进行下载，所以使用时需要进行网络配置。如果你也需要一个能够追踪开源应用更新的工具，可以在 GitHub 或 IzzyOnDroid 免费下载 Obtainium。

![](attachments/c4f5aead30426951.png)

▍**Teleprompter：把手机变成提词器**

- 平台：Android
- 关键词：提词器

@Peggy\_：如果你有准备公开演讲的需要，那么无论是前期的准备还是正式的演讲，提词器既能帮助你在前期快速地熟悉演讲内容也能起到较好的辅助、备忘作用。不过通常来说我们不会为了一次两次的演讲而购入一些专门设备，那么利用已有设备来满足需求就是一个不错的办法。

Teleprompter 就是这样一款应用，通过 Teleprompter 我们可以在手机上直接进行脚本创作，或是导入已经在其他设备上完成的脚本，Teleprompter 支持导入 .doc、.pdf、.txt 等格式的文件，基本能满足需求。开始播放后，Teleprompter 能够按照我们设定的速度逐句滚动，并高亮当前语句，从而达到提词器的效果。

![](attachments/4f87b128c8dabca0.jpg)

除此之外，Teleprompter 还进一步支持了视频录制功能，因此它既是一个提词器，同时也是一款录像机。Teleprompter 允许我们录制高清或是超高清视频，在录制的过程中，还可以在屏幕的上半部分实时查看脚本提示，最终录制好的视频不会包含提词器内容。

![](attachments/dc667daacc100c52.jpg)

通过 Teleprompter 进行录像时，我们还可以同时实时查看脚本内容，如果你需要录制一段相对正式的视频，那么打开 Teleprompter 的脚本镜像模式，再将屏幕对着镜子，便可以利用手机的后置摄像头进行更高质量的录像。

更加方便的是，Teleprompter 支持利用蓝牙键盘、小型遥控器等形式来控制脚本播放，从而达到更好的提示效果，不过相关功能需要付费解锁才能体验。

![](attachments/4d4e610ea5579a30.jpg)

如果你凑巧需要这样一款工具，不妨尝试一下 Teleprompter，应用本体免费且有广告，付费订阅后可以关闭广告并获取更多实用功能。你可以从 Google Play Store 下载使用 Teleprompter。

![](attachments/5d3a582d9fb6acb6.png)

▍**Verse：印象笔记版 Notion？**

- 平台：iOS / iPadOS / macOS / Windows
- 关键词：协作、数据库、Notion

@Snow：继去年推出测试版本之后，由印象笔记团队出品的协作型块编辑器 Verse 由于近期正式在 App Store 上架了移动版本，同时 Andriod 版本也开启内测。

作为一款对标 Notion 的生产力工具，Verse 在应用框架上与 Notion 保持高度一致，如果你有使用 Notion 的经验，Verse 对你来说门槛只会更低，因为它有着完善的中文布局，编辑器甚至支持 3 种中文字体的切换，如果你曾经因为语言障碍放弃 Notion，那可以立刻在 Verse 上尝试起来。

Verse 还提供了一个包含 50 余种全中文文档模板的模板库，除了减肥计划、课堂笔记、番茄工作法等日常使用频率较高的项目，还有「考研复习安排」这类本地化特色鲜明的模板，可以为特定人群提供便捷。

![](attachments/5a425eb0a6647568.jpg)

![](attachments/195eb005ae75f580.jpg)

在功能性上，Verse 也提供了一些具有本地化优势的服务，比如你可以快速添加百度地图、百度网盘分享、B 站视频流、腾讯视频流等模块，你也可以在文档中快速关联同一账号下的印象笔记内容。

开发者还考虑到了你撰写文稿时的专注问题，将白噪音也集成在 Verse 当中，点击右上角的唱片按钮就可以开启小溪潺潺的背景声。

Verse 还做出了一些智能化尝试，例如在此前印象笔记彩色标签的基础上，增加了「智能标签」功能，它会根据文档内容自动生成多个标签，方便用户后续进行管理；应用还支持生成智能摘要并进行智能推荐等。此外，Verse 还内置了一项名为「知识星图」的功能，它与 Obsidian 的关系图谱有些相似，会根据文档数据自动生成结构化图谱，将信息或知识点关联起来，这对于某些课堂笔记来说是非常实用的。

![](attachments/b5f6e421337828a6.png)

尝试完这些独特的功能，就得说说 Verse 的坑了。作为一款基于印象笔记云服务的工作台，Verse 虽然不至于像 Notion 那样间歇性「完全不可访问」，协同效率会更高。

但在正常情况下，Verse 加载效率是肉眼可见地慢于 Notion，这一点在从模板新建文档时尤为明显，如果你在更换模板时未等模板加载就点击右上角的「应用这个模板」，大概率会获得一个模板完全错误的文档。虽然在文档编辑时，我们不会对网速有多么严苛的要求，但时不时需要等加载这样的体验，实在有些干扰创作。

其次，Verse 的模块拖拽灵活度低于 Notion。Verse 主要利用右侧栏中的「分栏」工具来调整页面布局，当前模块行必须先进行分栏操作才可支持多栏显示，你就无法直接拖拽其他模块与当前模块并行，由于我常用拖拽调整文档结构，这一体验差距尤为明显。再次，Verse 的移动版本目前也不是非常成熟，模板、高级模块等都暂未支持，编辑目前主要依靠桌面或网页端完成。

![](attachments/7db26831950522f0.png)

相比于以上体验上的不足，Verse 的订阅规则可能才是最「劝退」我的因素。Verse 的主要亮点，比如之前提到的智能化功能，都需要超级会员才可使用。而目前应用仅支持按年付费，只可以选择 192 元解锁专业账户或 262 元解锁超级会员，且没有免费试用期。不过目前印象笔记与 Verse 会员服务互通，如果你本来就是印象笔记体系的用户，倒是可以借此免费体验一下。

你可以在 App Store 免费下载 Verse，桌面客户端可以在官网下载。

![](attachments/b835fc689c334cc6.png)

除了「新鲜」App，App Store 中的许多老面孔也在不断迭代、更新，增加更多有趣和实用的功能。少数派希望帮你筛选 App Store 中值得关注的 App 动态，让你快速了解 App 和开发者们的最新动态。

▍**RescueTime：全新设计，更加关注数字健康**

- 平台：macOS / Windows
- 关键词：时间记录、时间追踪

@Vanilla：RescueTime 是一款老牌的时间追踪工具，对于大部分热衷于时间管理的效率控来说应该是耳熟能详了。虽然早在前年就推出了全新版本，但是 RescueTime 团队却显得略微低调，不仅没有直接替换旧版 RescueTime，也没有太过大张旗鼓地宣传新版。

RescueTime 的新老版本两者处于并存的状态，但是侧重点有所不同，前者侧重于通过实时的指导功能来提升数字健康，而后者侧重于提供更详细的追踪数据。

![](attachments/2930fa021dafe2f6.png)

图片来源：RescueTime 官网

在旧版 RescueTime 中，统计图表展示的仅仅是所选时间范围内各个应用或者网站各自使用的时间，而在新版 RescueTime 中，应用对所有追踪到的应用或者网站都做了自动分类。

其中 Focus Work 和 Other Work 被归类为工作时间，Personal Activities 则被归类为非工作时间，所有应用或者网站都可以通过箭头在 3 个分类之间自由地重新分配。另外，新版 RescueTime 的统计图表通过横坐标轴来展示 3 个分类各自占用的时间，同时展示工作时间与目标之间的实时差距。

![](attachments/9d2113c567abbd55.png)

在新版 RescueTime 中，菜单栏的本地客户端新增了一个专注功能。在启用专注功能并设定倒计时后，RescueTime 可以屏蔽掉那些干扰你工作的网站，帮助你实现更高效的工作效率，同时也会在菜单栏窗口中显示当天的专注工作时间进度。

![](attachments/2fdfc5248646423e.jpg)

新版 RescueTime 也可以免费下载使用，不过需要订阅 Premium 才可以使用 Overview、Coach、Focus 等功能，免费版本仅限于时间记录，价格为每月 12 美元或者每年 78 美元。

![](attachments/67b431c50fa7947f.png)

▍**GoodNotes 更新：终于支持语音备注功能**

- 平台：iPadOS / iOS / macOS
- 关键词：手写笔记

@黎明前线Alan：近期 GoodNote 更新，加入了语音备注功能：用于将录音内容更好地融入进 GoodNotes 现有的画板系统中，如果你觉得手写以及打字输入不够快速的话，或许这对你来讲是一个更好的选择。

![](attachments/09210567168c19ec.png)

在更新之后，打开 GoodNotes 画板界面，点击顶栏中的麦克风图标，然后正常记录笔记，与此同时你的音频也会一同被 GoodNotes 记录下来，通过录音的方式记录更多信息。

在录音结束之后，用户如果想回溯录音的某一点同时自己在 GoodNote 中书写的内容，在播放音频的同时当时音频时间戳对应的书写内容部分也会高亮显示，方便用户整理当时的记录思绪。

![](attachments/686c781ec63fa1e3.png)

除了直接播放录音音频之外，长按画板中的某个书写的单词或字符，也能快速跳转到与之对应的音频录制内容部分。

你可以在 App Store 免费下载到新版 GoodNotes 5。

![](attachments/ee9aa0855b0eec7a.png)
