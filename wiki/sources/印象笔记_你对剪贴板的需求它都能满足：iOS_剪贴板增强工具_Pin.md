# 你对剪贴板的需求它都能满足：iOS 剪贴板增强工具 Pin

[少数派 你对剪贴板的需求它都能满足：iOS 剪贴板增强工具 Pin](http://sspai.com/32404)

![](.evernote-content/253DC9C1-F696-4F1E-B700-487FEE7C329B/A43BC89E-A4DF-4298-A164-42D91181912E.jpg)

在以下需求或特定使用场景下，你很可能需要一个剪贴板增强工具：

* 需要重复粘贴若干个不同内容，默认的剪贴板只能容纳一个东西；
* 需要查看自己之前复制了什么，默认的剪贴板里的东西是非常易失的；
* 需要快速的对复制的内容进行一些固定操作，比如搜索、拨号等。

在注重效率的 Mac 桌面环境，各种增强剪贴板的小工具数量繁多，比如少数派推荐过的 [Paste](http://sspai.com/tag/Paste)、[CopyClip](http://sspai.com/search/?q=CopyClip)。在 iOS 上，找到这样一个趁手的小工具并不容易，此前有 [Clips](http://sspai.com/tag/Clips)，但它功能很简单，而今天介绍的 Pin，借助 iOS 通知中心小部件，几乎干净利落的解决了关于 iOS 上剪贴板的所有问题。

与诸多复杂的效率工具不同，作为通知小部件的 Pin，可以说是开箱即用——你只需在通知中心「今天」里添加 Widget 即可开始使用。

Pin 将自身的功能分成了 4 个小部件。这样做的好处是，你可以只把自己要用到的部分功能，放到寸土寸金的通知中心里。

剪贴板历史记录
-------

记录剪贴板的历史，是这类剪贴板工具最基本的功能了，[Clips](http://sspai.com/tag/Clips) 就是几乎只有这一个功能。在 Pin 里，除了能让你快捷复制与编辑剪贴板历史记录之外，Pin 还允许你将经常用到的内容置顶，以便需要时迅速选择使用。

分词
--

通过触摸屏操作光标并不是一件让人愉快的事情。相信大多 iOS 用户都有想要复制一段话里的某几个字，然后被 iOS 自作聪明的光标气的想摔手机的经历，比如微信、QQ 等应用，甚至只允许你一次复制一整句话。

为了解决这个痛点，Pin 贴心的「分词」功能就派上用场了——当你复制一整句话，下拉通知中心，Pin 会自动将你复制的长文本智能拆分成若干词组，你可以单选某个词组进行复制、搜索操作，也可以多选若干词组将它们合并起来。

![](.evernote-content/253DC9C1-F696-4F1E-B700-487FEE7C329B/04BAD253-C897-4D72-88FD-AB3C9762AB64.jpg)

![](.evernote-content/253DC9C1-F696-4F1E-B700-487FEE7C329B/DCAB13B4-8B3E-4449-8FD7-A2D0831C81BE.jpg)

动作与预览
-----

大多数情况下，我们对剪贴板里内容的后续操作就那么几种：搜索、翻译、拨号等等。先复制，再进入其他应用粘贴是一件非常低效的事情。

Pin 提供了一种直接在通知中心里解决问题的方式——定义一组快捷操作，点击后直接用剪贴板内容进行操作。

除此之外，Pin 甚至还允许你直接在通知中心里预览搜索页面。

![](.evernote-content/253DC9C1-F696-4F1E-B700-487FEE7C329B/DE4B1BBA-6CA8-4AE2-AFC0-B77FC83F9741.jpg)

扩展与插件
-----

Pin 还提供了丰富的扩展选项，你几乎可以自定义每一个小部件的样式与行为。

值得一提的是，Pin 提供了丰富的自定义动作扩展和插件中心，稍微花点精力设置，你几乎可以把它当成一个简化版的 [Workflow](http://sspai.com/tag/Workflow) 来用。

图片
--

在日常使用中，我们经常会需要将图片发到微信或 QQ 里。Pin 的另一大特色是，它能通过剪贴板或二维码的方式，直接将图片分享至微信或 QQ。你可以利用这一特性，方便的把在微博看到的某个图片发到 QQ。

小结
--

Pin 不是一款新 App，少数派此前在[「一周派评」](http://sspai.com/search?q=%E6%B4%BE%E8%AF%84)栏目里[有过推荐](http://sspai.com/31517)。它在今年不断的迭代更新中，功能越来越丰富完善，诸如「分词」这样的功能更是精准戳中了很多人的痛点，越来越多人喜欢它、每天使用它，所以我们有必要正式推荐它一次，让更多人认识并用上这款出色的工具 App。

如前文所说，Pin 上手无需成本，如想要更符合你的使用习惯，花一点时间配置一下，它就会安静运行在后台，你几乎不用再次点击它。如果你一定要点击它，它还支持 3D Touch。

此外，Pin 对 URL Scheme 的支持还能延伸出更多玩法，少数派近期会通过另一篇文章，从具体的使用场景来介绍 Pin 的使用技巧，敬请期待。
