# 出长直播产品视角阶段性分析（李朋）

> 作者李朋，途牛产品部。从产品视角分析跟团直播业务。

## 三个核心判断

**① 为什么要做直播**
直播可同时实现：品宣（品牌曝光）→ 获客（线索）→ 创收（覆盖成本）三层目标。是按效果付费的流量洼地，优于漫天投流。

**② 为什么是产品部做**
手握机位的玩家无零售能力（且2C与2B基本盘冲突）；大IP主播看不上跟团游（非标、核销低、回报周期长）。产品部有品牌背书+对客体系+客服核销能力，是差异竞争优势。

**③ 播品权重公式**
`100% = 70%播品 + 20%主播 + 10%场景`
播品中：70%看目的地需求热度，30%看产品综合竞争力。

## 分工定位

| 角色 | 定位 |
|------|------|
| 播品 | 进攻性，产品组负责整合资源和孵化 |
| 主播 | 平常心、耐寂寞，建立专业可信人设 |
| 自播 | 为核销而播（不是为支付而播） |
| 南京直播间 | 培养大单品（跑出100单+再切达播） |
| 淮安直播基地 | 灵活用工按效果付费，渠道越多越分散越好 |
| **核销** | **核心优势，决定收入落袋** |
