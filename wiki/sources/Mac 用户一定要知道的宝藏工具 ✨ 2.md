你有没有遇到过这种情况👇  
看教程让你打开终端输入 brew install xxx  
然后你就懵了：这是啥？？？  
  
其实 brew 就是 Mac 上一个「隐藏的 App Store」  
里面有几千个应用，很多 Mac App Store 都没有的  
  
但问题是要敲命令才能用…  
直到我发现了 Applite ![](.evernote-content/4766B6CE-D2A6-4F8B-BE06-B5BE41D800A6/AB18CCE2-9B75-4D02-8DC9-EAC10128FE37.gif)  
  
📌 它是什么  
一个免费开源的 Mac 应用  
把 Homebrew 变成了跟 App Store 一样的图形界面  
搜索 → 点安装，完事！  
  
📌 为什么推荐  
✅ 不用先装 Homebrew，它自动帮你装  
✅ 全程不用碰终端，纯鼠标操作  
✅ 免费开源，不收钱不追踪  
✅ 内置国内镜像，下载不卡（重要❗️）  
  
📌 怎么用  
1️⃣ 官网下载 DMG，拖进应用程序  
2️⃣ 打开后自动配置 Homebrew  
3️⃣ 搜索想要的应用 → 点下载  
4️⃣ 更新/卸载也是点一下的事  
  
📌 国内用户必看  
设置 → 镜像 Tab → 勾选启用 → 选中科大/清华/阿里云  
开了之后从「转圈超时」变成「秒装」🚀  
  
📌 换新电脑也不怕  
支持导出已装应用列表，新电脑一键导入全装回来  
  
官网：aerolite.dev/applite  
系统要求：macOS 13+
