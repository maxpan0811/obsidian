芝加哥疫情随记 2020 年 3 月 21 日
=======================

2020.03.22
 雪瓜籽
阅读 1700

![](.evernote-content/E2060D77-698A-496F-989A-984A5B9B4140/FA299B61-D56C-4533-853F-09BB72C59BAC.jpg-mobile)

今天芝加哥全城戒严，进入避难模式。除了药店，医院，食品店，外卖店，和加油站，其它全部停业。现在大街上非常安静，已经基本没有人了。市政府说已经开放了几家旅店接受无家可归的人，也准备了送餐服务，给那些低收入的群体。我们小区外面人很少，偶尔有少数的人出去，如带宠物散步的，总体来说大家还是非常遵守纪律的。没有看到什么或者听到什么不好的新闻。

现在大家都在家里了，所以今后的一个月内，大家可能会通过网上买东西。快递来的货物是我们跟外界接触的主要渠道，也是我们可能感染病毒的一个可能性。所以在包裹来的时候，能消毒就消毒，不能消毒的话，可以带手套，不能带手套，就要靠好好洗手了。去掉外层包装，及时扔掉。好好洗一次手，再开一层，及时扔掉，再洗一次手。去掉所有包装以后，再洗一次手，然后再用买来的东西。如果是外卖，还可以微波炉加热一下。既然是避难，就什么都不能大意。不是说快递小哥的问题，而是 80% 的感染者是无症或者轻症的。而且，确实有报道说，有快递小哥中招了。

![](.evernote-content/E2060D77-698A-496F-989A-984A5B9B4140/407F719A-B2CE-412D-847F-B2826BBD66CC.jpg-mobile)

今天有时间坐下来看一看关于冠状病毒的治疗方面的文章。最近炒的沸沸扬扬的，就是 Hydroxychloroquine 和 Azythromycin。前者是个治疗疟疾和自身免疫功能紊乱的药，主要功能是定向的免疫抑制剂。后者是抗生素，只要是用来杀细菌的。疟疾对病人的折磨跟免疫紊乱有关，新冠病毒对身体的损伤也跟免疫有关。可能两个病在免疫功能反应上，有些共同的地方。所以这个药这次又有作用了。中国的治疗方案中也有推荐。如果需要，最好先看医生再说，最好不要囤积这些药。如果我们囤积这些药的话，可能会造成短缺，从而对我们的抗击疫情产生巨大的打击。所以不要去找人从药房把这个药搞出来，而是把它留在药房给最需要的人。因为等你需要这个药的时候，你应该已经感染了，所以最好不要在家用，最好还是在医院用。另外，中国有自己用这个药的，由于负作用而出事的。

![](.evernote-content/E2060D77-698A-496F-989A-984A5B9B4140/8354C382-499F-442D-8CDD-8A66163BFF5F.jpg-mobile)

其它的免疫调节的药也很多，最强的是激素。这个不要在家里自己用，因为它的负作用非常大，在医院我们都非常小心，所以不要自己随便用。

有家里常备的药是布洛芬一类，Ibuprofen, Advils, 或者 Motrin。这几个药不要用。虽然他们对普通感冒效果非常好，但是会加重新冠的病情，所以不要用。因为它们会增长免疫因子的存在时间，而使得免疫反应加强，加剧气管和肺的水肿和渗出，使得病情恶化。

如果有发热不舒服，可以用泰诺了，Tylenol。它的作用没有布洛芬的强，可以用。但是如果用的太多，也会和布洛芬类似的作用，所以不要随便加大药量。

没病的时候，最好的方法是锻炼身体。锻炼身体可以调节免疫系统的。锻炼身体的作用是安定免疫系统。我们经常说免疫力强，免疫力弱，这个东西实际上是一个人们自己心里的概念。免疫强弱是个伪定义，并不代表我们的抗病能力强或者抗病能力弱。如果在没有病的情况下，免疫系统过度反应对身体是不好的，会造成免疫失调，甚至紊乱，造成自身免疫疾病。就是说，没有病的情况下，免疫系统越低调，越稳定越好。平时锻炼身体呢，可以让免疫系统稳定。锻炼的时候会产生 IL6。IL6 有两个来源，一个是肌肉，一个是免疫系统。如果是免疫系统产生的，就会激发免疫反应。那相反，如果肌肉产生的，就会抑制免疫系统产生，达到抑制免疫系统的作用。所以免疫系统就相对来说会更低调和稳定。新冠病毒对身体的损失主要是免疫过度，造成肺部水肿和渗出。有一个特别的名词，叫做免疫分子风暴。免疫系统越稳定和低调的，免疫因子风暴越小，对身体损害越少。所以，锻炼身体是有好处的。宅在家里，不要忘记锻炼身体啊！

再一个大家都有的就是维生素 C 了。我们现在的人一般是不缺维生素 C 的。如果我们每天正常吃蔬菜和水果，那末每天的维生素 C 就够用了。维生素 C 在身体是不能储存的，所以平时吃大量的维生素 C 是意义不大的。平时多吃也没有预防作用。但是，如果得病的时候可以多加一些维生素 C，因为这时候身体里面的维生素 C 的要求可能会比平时多一点，。一方面是因为它的抗氧化的作用，我们得病的时候，身体里的氧化剂比较多。另外一方面是它的辅酶作用，在得病的时候，人体内各种生物化学反应活动增加，所以这个时候吃些维生素 C 还是有作用的。

另外一个大家常用的就是锌，Zinc。它被发现对鼻感冒病毒有用。对新冠，可能有一定的作用，但是作用是有限的。因为它的作用是稳定鼻粘膜，防止鼻粘膜水肿。所以，可能有一些的抗病毒作用。但是鼻感冒病毒侵袭的细胞受体和新冠病毒用的受体是不一样的，所以它对新冠病毒的作用还有待进一步的观察和研究。

总结一下，大家在家里时，好好休息，好好锻炼，平衡饮食，多喝水，保证身体不脱水，保证睡眠，保持良好的心情。如果有症状，可以增加维生素 C， Zinc。可以适量吃泰诺，Tylenol。再不行，就给医生打电话，然后尊医嘱。对于食疗，我持开放的态度，留给大家自己发挥了。

![](.evernote-content/E2060D77-698A-496F-989A-984A5B9B4140/92A521B3-FA98-4D0F-A203-BBFAE7FFB774.jpg-mobile)

现在美国的医院还是缺口罩，今天有一个新闻，那就是说 波士顿的一家医院没有 N95 口罩，但是有人捐了一些过期的 N95 口罩。但是口罩已经基本有破损，不能用了，所以很多大学生自愿去帮他们修补这些口罩。这说明就这是一个全国性的一个情况，确实是备战物资不足。说到这儿呢，我要特别感谢各位同学，朋友，和读者。感谢大家积极参与，为美国医院捐口罩。你们的帮助让我非常感动。很多人在行动，用各种各样的渠道找口罩，然后把口罩寄往美国的医院。我们还有个北大拔丝（84）基金会也正在积极筹办。北大拔丝基金会刚刚帮助了中国的医院，现在立刻又反过来帮助美国的医院。马不停蹄，真是太辛苦了。在这场新冠病毒疫情目前，我们大家都是站在一起的，因为这是一场人类和自然的战役。只要大家团结一致，我们一定会胜利的！！！
