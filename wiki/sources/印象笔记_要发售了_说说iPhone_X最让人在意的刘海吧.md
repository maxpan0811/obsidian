# 要发售了 说说iPhone X最让人在意的刘海吧

---

**不可避免的大圆角**

![](.evernote-content/2A51FBD7-961D-4EEF-AD9C-9233F78F7BAC/8B957084-D476-4525-B0BD-0484823D0D47.jpg)

iPhone X 的发布对果粉们来说可谓是一件既惊喜又不惊喜的事情。惊喜的是它真的如大家预期的那样充满吸引力，说不惊喜是因为大多数细节在这之前也都已经知道了。然而即便 iPhone X 确实超棒，人们应该也有了心理准备，某样东西还是让大家特别在意，那就是屏幕顶上的那俩大圆角。

我们现在都知道，iPhone X 之所以能实现面容 ID，全靠机身正面的那个大“刘海” —— 原深感摄像头。苹果对原深感摄像头的实现颇为自豪，称它为“技术集成最密集的区域”。

然而问题在于即使苹果已经把原深感摄像头的区域压缩得很小了，它相对于整个机身正面来说还是太大，特别是在 iPhone X 已经采用了全面屏设计的情况下。因此，原深感摄像头就占用了屏幕顶部中间一块不小的空间，也就在两边形成了一个显眼的大圆角。

不管怎么说，这种不规则的屏幕形状也委实太过反传统了。无论是图片、视频还是应用界面，我们能看到的内容全都是矩形的。iPhone X 这屏幕上平白缺了那么一大块，怎么看都觉得很别扭。可是没有办法，如果要同时实现全面屏和面容 ID，似乎只能这么做。可是苹果究竟对此是怎么想的？它真的是迫于无奈才这么做的吗？

**苹果的理由**

![](.evernote-content/2A51FBD7-961D-4EEF-AD9C-9233F78F7BAC/7438C00B-6A26-4A8D-AF80-D84C5E855F2F.jpg)

其实最关键的问题就在于，苹果事实上是可以避开这种设计的。如果像三星 Galaxy 全面屏系列那样保留上边框，是能够去掉“刘海”的。还有另一种办法，就是用软件手段让两个大圆角部分黑屏，和原深感摄像头融为一体 —— 然而苹果却并不建议开发者这样做，它鼓励后者尽可能去利用这两个大圆角。

值得一提的是，就在不久前苹果的数位高管接受采访，透露 iPhone X 的开发思路，其中也涉及到了新机这其他的屏幕形状。说到这一块时，高管们并没有详细解释他们为什么要这么做，只是强调大圆角的设计并不会给人们带来很大的困扰，人们只要一上手新机，很快就会习惯，并且忽视“刘海”的。至少从这态度里，我们多少能感受到他们的坚决和信心。

在苹果给出的解释中，“忽视”成了一个值得注意的关键词。苹果应该是认为，人们在经过一段时间的使用后，会将更多的注意力放在屏幕本身，而不会过多去在意缺掉的那一块，无论这属于一种习惯，还是基于人的大脑会倾向于补完缺失部分的原理。

更重要的是，苹果想实现真正的“正面屏”的感觉。为此，它不惜要形成两个大圆角，也要消灭掉上边框。目前宣称采用了全面屏设计的新机，绝大多数都不得不保留显眼的上边框或下边框，或上下皆有。iPhone X 没有下边框，顶部也通过两个圆角顶到了边缘。这样，它的正面的屏幕利用率就更高，更能够带给人一种惊艳的感觉。

据了解，苹果在写给开发者的指南中，特意提到开发者要尽量利用上大圆角，不要用黑屏来处理，因为这样会让自己的应用界面显得更局促。很明显，苹果是明白这个道理的。

**困扰会存在吗？**

![](.evernote-content/2A51FBD7-961D-4EEF-AD9C-9233F78F7BAC/1E30FAE0-9141-4CC0-B5ED-63AC18F82389.jpg)

如今苹果已经向媒体、博主和主播们开放 iPhone X 的测试，体验报告大家也看得不少了。如果你注意一下人们的感想，就会发现虽然看图片的时候感觉“刘海”很显眼，但真正对此表示不满的体验者却寥寥无几。从这一点上，苹果高管们的“忽视”的说法是有些道理的。

至于实际的应用，iPhone X 的屏幕形状事实上也不会给我们带来什么困扰。如果应用没有针对 iPhone X 进行专门的优化和适配，那么进入到应用界面后，我们会发现大圆角和底部的手势区域是黑色的。这个时候，应用界面仍然呈矩形，整机的视觉体验倒有点像 iPhone 8。视频播放的时候也是如此。

如果那些应用经过了 iPhone X 适配，那么更多的屏幕空间就会被利用上。怎么去利用多出来的屏幕空间，这个得看开发者们自己发挥创造力。不过肯定肯定的是，我们能够看到的信息量就会更多了。

所以我们无需对这些问题担心太多，开发者们也已经表示，iPhone X 的应用屏幕适配比预想中要简单。因为 iPhone X 的屏幕宽度相比 4.7 英寸 iPhone，实际上并没有变化。在内容显示上用点心，保证关键部分不被原深感摄像头和底部的手势栏遮挡就行了。更重要的是，iPhone X 已经证明了它是很受热捧的新品，开发者没有理由不为它多花点时间。
