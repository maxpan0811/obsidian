Claude Code隐藏功能27招
------------------

多数人使用 Claude Code，只停留在提问、等待、接受第一版结果的层面，因此浪费了大量潜力。真正拉开效率差距的，不是提示词写得多漂亮，而是能否把 Claude 当成一名需要 onboarding 的初级开发者来管理：给它项目上下文、明确约束、设定计划、安排验证，并及时纠偏。文章总结了 27 个 Claude Code 隐藏功能与工作流技巧，从 /init、/statusline、/context、/compact、/clear，到自定义 skills、sub-agents、Git worktrees、DevTools 集成、/loop、自然语言查数据库和 /effort high。底层逻辑可以归纳为三点：管理上下文，而不是堆上下文；把任务拆开并行，让 Claude 协作；能自动化的重复动作，就交给系统完成。最会用 Claude Code 的人，不一定更聪明，只是更系统。
