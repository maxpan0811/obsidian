# 用最简洁的方式，给图片作标注：This

---

![](.evernote-content/FA786D57-B73F-48B6-8F27-119FBAC3E459/6DC08A2B-4887-4690-9A96-2B164A118EB4.gif)

美，很多情况下源自节制。这点在 App 设计上尤为明显。

这就像仍被许多人喜爱的 [Day One](http://sspai.com/tag/Day%20One) 一代的一张图之美，[Instagram](http://sspai.com/tag/Instagram) 早前的照片必为正方形的美，这些限制在最开始，都是令用户感到不满的地方，但它们却为这两款 App 带来了独有的优雅排版与观看体验。

今天我想要介绍的 This 就是一款同样贯彻了精简节制的图片标注应用。它出自制作了把照片变成水彩画的玩图应用 [Waterlogue](http://sspai.com/search/?q=Waterlogue) 的文艺开发商 Tinrocket 之手。

我们对于标注应用的想象是什么样子呢？各种贴纸、字体，支持马赛克、高斯模糊，最好还要有放大镜功能。而 This 把这些统统抛弃，告诉用户，不，你只需要一行字和一个箭头。

去繁就简
----

This 的使用极度的简单，没有工具的选择，没有剪裁滤镜的干扰，你所需要的就只是选择一张要标注的照片，点击需要标注的地方，输入文字，最后分享，一切都一气呵成。

对于初次使用的用户，This 还提供了一个贴心的新手引导与 7 张示例图片，手把手的教你如何「优雅」的使用 This，点开左下角的「相机」符号就可以看到。

点击图片上你想要标注的地方，标记文字就会从标记点「弹射」出来，点击 `（This）` 就可以输入想要标注的文字内容了。而当你拖动标记点的时候线条还会随之摆动，并且根据环境颜色在黑白之间切换。

![](.evernote-content/FA786D57-B73F-48B6-8F27-119FBAC3E459/CF47B6B4-0431-4264-A5E0-A6DA303DD5A4.gif)

![](.evernote-content/FA786D57-B73F-48B6-8F27-119FBAC3E459/DBA776B0-9B63-4D0D-95BC-F6D386B80DD6.gif)

This 提供了六种箭头：箭头、空心圆圈、X、心形、圆点、虚线，通过简单点按屏幕更换。对不同的标注对象，使用不同的箭头，可以使得标注更加有层次感。

而如果你想要删除一个标记，只需要长按那个标记，当出现一个红叉之后放手，标记就会像一个泡泡一样消散，十分有趣。

This 的标记也提供了大中小三种字号选择，同时颜色也可以手动选择为白色或黑色。中文字数较多的话，建议使用小号以规避中文适配换行的缺陷。

![](.evernote-content/FA786D57-B73F-48B6-8F27-119FBAC3E459/C2194BFE-4076-4DEF-81DF-E9299ED07381.gif)

为什么应该用它
-------

做产品最难的就是做减法，给用户加上限制进行美学引导，对于标注类软件来说更是如此。在其他软件都沉迷于自己功能的「多而全」的时候，This 给出了另一种答案 —— 快而简。

使用 This 你并不需要在意标注的形式，也不需要关注标注的颜色，你需要做的就只有找到需要标注的地方，然后进行标注。当我们胸中的表达欲澎湃而起的时候，我们所需要的并不是一个多而全的工具，而是最简单最快速的那个，让我们能抓住灵感的尾巴，在图片上讲一个有趣的故事。

和 [Drafts4](http://sspai.com/search?q=drafts) 相差无几的启动速度，快捷清晰的标注格式，简单的应用逻辑，最后标注出的照片优雅而清爽，*This is This.*

你可以在 [App Store](https://itunes.apple.com/cn/app/this-by-tinrocket/id1067596534?mt=8&uo=4&at=10lJSw) 下载 This ，12 元的售价可能是阻挡更多人喜欢它的一个障碍。
