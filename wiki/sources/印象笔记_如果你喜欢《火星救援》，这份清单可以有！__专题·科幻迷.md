原创 *2015-12-02* *应用推荐第一媒体*

> 我们生存在一个既来不及猎杀恐龙也赶不上和机器人决斗的乏味时代。

作为一个科幻迷，这句话引起我强烈共鸣，并伤透我心。我们无时不刻想穿越侏罗纪，遨游银河系，无奈却困在一颗孤独的行星之上。

前不久 NASA 公布了阿波罗登月计划的底片扫描，那无垠太空的美，实在无与伦比。

向往浩瀚宇宙，因为它充满未知。幸好人类拥有丰富的想象力与创造力，创作出精彩的科幻小说、电影与游戏，来安抚我们对未知的好奇。今天介绍的几款游戏与应用，必定戳中科幻迷的 G 点。

### **Out There: Ω Edition #iOS #Android**

飞船护盾已经破损，我被困在一颗气态行星中。  
此时我内心忐忑，应该冒险突离大气层，还是在这星球安度余日？  
放手一搏……我竟然顺利离开了。

放下手机，仰头叹气，此时我即欣喜又悲伤。庆幸自己旅途平安活了下来，却又悲伤着无法体验真正的星际航行。Out There 这个游戏就是如此硬派，走错一步便无法挽回。随机的遭遇与惊险的旅程让你沉浸在星际航行中。

Out There 讲述的是一名在冷冻长眠中醒来的宇航员，在未知星系中踏上返回母星遥途的故事。游戏通过大量的英文日志反馈与精美的漫画原画来呈现探险过程。

玩家需要指挥飞船探索未知行星，开采元素资源，来维持飞船的正常运转，在这过程中你还将遭遇外星文明，学习先进科技，穿越虫洞黑洞，发现废弃飞船，改造行星生命。怎么听，都有「星际迷航」的味道。

游戏的一大特色是更换飞船，有如储量大但修理麻烦的巨型飞船，又有燃耗低但携带科技弱的迷你飞船。游戏的生命参数为燃料、氧气与护盾值，如果没有合理开采分配资源，那么你将丧命太空。

在粉丝的强烈呼声下，以难度著称的 Out There 推出了简单模式，例如死亡时所留下的飞船资源，也可以重新开局寻回。在最新版本中，游戏还加入了多个结局设定，极大拓展了游戏的耐玩性。

![](.evernote-content/EA4C0BAF-4F28-44FA-9C22-D0BA4130B293/0C50AAD8-C13F-4528-B6F0-2E7B50CE0DB1.png)

Out There: Ω Edition 适用于 iOS 5.1.1+ 的 iPhone 和 iPad，大小 85.9 MB，售价 30 元。同时适用于 Android 2.2+ 设备，大小 61 MB，售价 4.99 美元。

### **Star Command #iOS #Android**

致敬「星际迷航」的不止有 Out There 这一款，Star Command Galaxies 这个游戏企图模拟「企业号」船员的日常，包括吃喝拉撒！不过今天我们介绍的是 Galaxies 的前身 Star Command。

2013 年发布的 Star Command ，是一款相当有料的策略游戏。玩家作为星舰指挥官，负责队员调度与战略指挥，你既要保持飞船最佳的火力输出与护盾升降，又要兼顾舰桥的不速之客与伤队治疗。在非交战状态，大事有船舱建设与设备升级，小事如队员的单位指派与技能配备，惨败之时还要修复船舱招兵买马，作为指挥官的你根本停不下来。

Star Command 采用复古像素风，游戏中的动态效果丰富，从设备运转到拔枪开火，只要玩家细心观察，总会发现不少有趣的细节。静态背景的绘制虽然也很华丽，但不一致的画风却显得些许突兀。

游戏中的操作音效相当丰富，例如沉闷的机器运转声，激活航行地图的启动声，都显得很有氛围。开发者不愧是「星际迷航」的死忠，Star Command 背景音乐毫不马虎，致敬电影采用交响乐作曲，使游戏整体显得大气，更有探索浩瀚宇宙的沉浸感，强烈推荐佩戴耳机体验游戏。

Star Command 的剧情很有「星际迷航」的影子，整体不长，但也值得细细品味。甚至像电影中的桥段，玩家也会面临谈判与开火的抉择，为敌为友，都影响着剧情的发展。

这款策略游戏的可玩性毋宁质疑，系统丰富且不复杂，收集到的代币除了升级设备，还能到不同星系招兵买马，雇佣些外星人，让舰队更加星际化。

不用担心游戏过于复杂，放弃指挥舰队的念头，Star Command 有 4 等难度选择，教学关卡循序渐进，让玩家一步步了解游戏系统机制，从一开始搓几个按钮就把敌舰击垮，到后期制定战略架设移动炮台，上手 Star Command 并不是问题。

![](.evernote-content/EA4C0BAF-4F28-44FA-9C22-D0BA4130B293/566A734A-F59B-4B88-87D0-FECCED0F829F.png)

Star Command 适用于 iOS 4.3+ 的 iPhone 和 iPad，大小 144 MB，售价 18 元。同时适用于 Android 2.3.3+ 设备，大小 109 MB，售价 2.99 美元。

### **Space Age #iOS**

年度大作「辐射 4」，为我们展示了一个「复古未来」的世界。游戏虽然发生在战后未来，但人类文化水平却定格在 50 年代。这种奇特的复古成为了游戏的一大特色。

Space Age 是一款解谜冒险游戏，它同样拥有「复古未来」的背景设定。游戏剧情基本上是黄金年代科幻小说的复刻，让玩家的游戏中怀旧，体验经典的科幻桥段。

故事讲述一队星际宇航员着落于一个看似荒无人烟的行星，Kepler-16。但却发现了一个离奇古怪又似曾相识的外星文明。Space Age 在原声音乐上下足了功夫，配合细腻的像素画面，渲染出一个悬疑色彩浓厚的科幻故事。

在俯视视角下，玩家指挥多个单位，探索地图上的未知区域寻找信标，同时巧用道具解谜通关。游戏中每个关卡都是一张独立的地图，你无需担心迷失在探索之中。Space Age 游戏性偏弱而注重剧情，语速偏快的文字对话，对玩家还是有一定的英语水平要求，如果这不是问题，你肯定会爱上这款游戏。

也许是我的过度解读，游戏的这个桥段让我想起「2001 太空漫游」中的画面，不禁猜想这是开发者对电影的致敬。

![](.evernote-content/EA4C0BAF-4F28-44FA-9C22-D0BA4130B293/A801B4DE-6BF5-4EBB-A63C-EDD8C55C53C6.png)

Space Age 适用于 iOS 6.0+ 的 iPhone 和 iPad，大小 76.4 MB，售价 18 元。

### **Galactic Keep #iOS**

科幻小说给予读者无垠的想象空间，天马行空的文本剧情常常震撼我的心灵。如果你未曾体验过这种快感，我推荐阅读科幻短篇「乡村教师」。

如果说 Space Age 是科幻小说 + 策略解谜的结合，那么科幻小说 + 桌面游戏又能擦出怎样的火花？自从在 Esor 那里发现了这款掌上桌游，我也被它的细节深深着迷。

> 游戏中你将带领一支拥有不同特殊能力队员的宇宙冒险小队，深入坠机地点、地底洞窟、神秘科学实验室等不同地图，面对未知的太空生物种族，甚至是强大的传奇怪物，你必须在探险中生还、战斗，并发现银河里的各种秘密。

简述起 Galactic Keep 的玩法，它似乎毫无亮点——投骰、打怪、收装备，实际上游戏相当硬派，如果你抱着刷图的心态玩游戏，那肯定连教学关卡都过不去。游戏中的每一个敌人，都是 BOSS 级的难度，玩家需要巧用能力与道具，同时攻守配合，才能赢得对战。游戏的站位也相当重要，没准你刚累死累活击败敌人，下一秒便给附近野怪突袭。

游戏拥有的文本量，远超过上文的游戏 Out There 。与传统桌游相似，Galactic Keep 以科幻文学般的表达，生动叙述遭遇事件，把画面留给玩家脑补，这也算是桌游的一大乐趣。游戏过程我总会停下来仔细阅读文字解说，配合自己的想象力，在缓慢进行的回合制里重温老派美式 RPG 的魅力。

无论角色或是道具，内容都以精美的卡牌呈现，每张卡片的文本详细，信息量巨大。这也是我爱上 Galactic Keep 的要因，让我这种卡牌收集控相当满足。

捧着 iPad 玩这个游戏，还真有棋盘的感觉，Galactic Keep 节奏缓慢十分适合午后休闲。如果你对游戏感兴趣，可以在这里阅读 [Esor 的详细测评](http://mp.weixin.qq.com/s?__biz=MjM5MjAyNDUyMA==&mid=210992722&idx=2&sn=431cbfeaeadbdb5db6a93a72375221f6&scene=21#wechat_redirect)。

![](.evernote-content/EA4C0BAF-4F28-44FA-9C22-D0BA4130B293/D36BA929-D882-4B7A-A9E5-98E19C91DFC7.png)

Galactic Keep 适用于 iOS 6.0+ 的 iPhone 和 iPad，大小 430 MB，售价 25 元。

### **四维空间 #iOS**

在「星际穿越」高潮阶段，五维时空的呈现令人着迷。「多维空间」听起来高深莫测，但科幻迷对万千世界的好奇心，使他们总想知道得更多。

「4D」是一款辅助认识四维空间的教育类应用，它利用概念推广的方法，把投影的概念推广到四维空间中，循序渐进让读者认识四维空间。重要的是，它的讲解过程并非教科书般枯燥，不仅通俗易懂还有几分幽默感。

如果给出一个购买它的理由，那就是黑科技「裸眼 3D」，只要按照说明调整眼球视距，读者无需借助工具便可体验 3D 影像，在立体影像中学习动态的四维空间旋转。体验震撼，让人回味无穷。同时应用更有中文支持，绝对值得你购买。

![](.evernote-content/EA4C0BAF-4F28-44FA-9C22-D0BA4130B293/2A6E6E04-130F-4121-8BD0-54D7BC27F29C.jpg)

四维空间适用于 iOS 7.0+ 的 iPhone 和 iPad，大小 6.2 MB，售价 18 元。

### **Action Movies FX #iOS**

科幻电影神作，近有诺兰的「星际穿越」（2014），远有库布里克的「2001 太空漫游」（1968）。

虽然「2001 太空漫游」的节奏很难符合现代观众口味，但影片的道具布景与拍摄手法，让人难以置信它是半个世纪前的作品，实在令人称赞。

现代的的科幻片都以电脑特效为主。不过有了 Action Movie FX，粉丝也能自己拍科幻短片，加特效连电脑都省了。

Action Movie FX 是一款给手机短片加科幻特效的应用，它的开发商 Bad Robot 可是大有来头。Bad Robot 是一家著名的影片制作公司，而它的 CEO 即是导演新版「星球大战」与「星际迷航」的 J. J. Abrams。

所以不难想象，Action Movie FX 拥有不少版权，收录了有如星球大战，星际迷航，使命召唤这些名作的特效，特效滤镜有几十款之多。

使用 Action Movie FX 十分简单，你只需挑选一款特效滤镜，开始录制影片，达到标准长度后便可剪辑特效。

应用不仅能导出标准的视频格式，还可输出 GIF 动图格式，极大拓展了娱乐性。而它近期还更新了「星球大战 7」的滤镜特效包，绝对让星战粉欲罢不能。

![](.evernote-content/EA4C0BAF-4F28-44FA-9C22-D0BA4130B293/ADA0BF72-F9F3-496A-8C45-0680617B5DB8.png)

Action Movie FX 适用于 iOS 7.0+ 的 iPhone 和 iPad，大小 92.4 MB，免费有内购。
