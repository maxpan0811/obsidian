# 华程团预结算差异原因分析及管控方案

> 财务部针对7月回团数据的预结算差异分析，梳理差异原因并制定管控方案。

## 差异原因分类

### 收入调整类

**主观原因收入调整**
| 场景 | 描述 |
|:----|:-----|
| 销售金额未调准 | 销售折让、扣点、产品定价等团前未调整 |
| 无损退团未走完流程 | 含下线团与非下线团 |
| 散拼团综费问题 | 团前未录入综费/出团后调增 |

**客观原因收入调整**
| 场景 | 描述 |
|:----|:-----|
| 有损退团未明确金额 | 团前无法确认损失金额 |
| 单团综费后调 | 回团对账后才能确认综费 |
| 佣金跳档 | 客户销售额累计达标佣金跳档 |
| 团中增减项 | 客诉、客户需求增减等 |

**其他收入调整**
- 携程OEM结算分润
- 考察团收入调整（根据业绩达成确定收费标准）
- 境外小费（导游机场现场收取，团后调整）

### 成本调整类

**主观原因成本调整**
| 场景 | 描述 |
|:----|:-----|
| 成本漏项 | 机票罚金、插座、物料等；导游门票/餐费/停车费等；地接单房差/人档结算价；签证预算高支/个签漏项；币种日期录入错误 |
| 成本错项 | 资源采购部成本计算错误、儿童定义不清、成本单价识别错误、非主流货币汇率变动、付款抹零 |

**客观原因成本调整**
- 团中成本变动（增减项、投诉）
- 航司退款、保险理赔
- 导游好评奖励与导游补亏

## 管控方案（11条）

| # | 措施 | 适用场景 |
|:-:|:-----|:---------|
| 1 | 销售按出团前3天或实占后3天提交预算，系统增加提醒+控制 | 主观收入调整（折让、扣点、无损退团、散拼团综费） |
| 2 | 单团综费出团前3天或实占后3天提交，特殊情况一单一议邮件审批 | 单团出团后调整综费 |
| 3 | 前返/折让/扣点等定义、计算方式规范文件宣贯 | 主观收入调整相关 |
| 4 | 携程OEM团前确认账单，否则按毛利50%调整应收预算 | OEM结算分润 |
| 5 | 考察团系统监控客户达标后提醒 | 考察团收入调整 |
| 6 | 明确境外小费确认规则 | 境外小费收入调整 |
| 7 | OTA客诉在独立模块展现，关联原订单 | 客诉应收调整 |
| 8 | 成本漏项处罚措施；高频小额物料集团统一采购；导游成本预算调增5%；签证流程宣贯 | 主观成本漏项 |
| 9 | 资源采购重定规范：景点表查询单价、小币种固定汇率、付款抹零邮件通知 | 主观成本错项 |
| 10 | 航司退款/保险理赔在独立模块展现 | 航司退款成本变动 |
| 11 | 低频率/小影响/不可预判类型不做管控 | 有损退团、导游补亏 |

## 下一步安排

- **集团市场销售部**：前返/折让/扣点规范性文件宣贯；OTA达标后返规则明确
- **集团流程运营部**：每月拉取团后调整应收数据，对主观调整通报
- **集团财务部**：9-11月持续关注预结算差异和改善情况，每月同步总经理
- **待确认**：境外小费确认规则统一规范

## 关键信息
- 预结算差异分为收入调整和成本调整两大类，各细分为主观/客观/其他
- 11条管控方案覆盖从系统控制、流程优化、处罚措施到机构协调
- 核心逻辑：**能系统管的归系统，能规范的归规范，低频不可控的放弃管控**
