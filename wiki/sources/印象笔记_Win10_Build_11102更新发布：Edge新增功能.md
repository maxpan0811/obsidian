# Win10 Build 11102更新发布：Edge新增功能

---

[![](.evernote-content/44718229-81F4-425E-9A68-123A1740D86A/ADD49987-581B-458B-91E6-1ED4CF5B7D9D.png)](http://tech.feng.com/2016-01-22/Win10-Build-11102-updates-Edge-of-new-functions_636768.shtml)

![](.evernote-content/44718229-81F4-425E-9A68-123A1740D86A/ADD49987-581B-458B-91E6-1ED4CF5B7D9D.png)

威锋网 1 月 22 日消息，就在今天凌晨，微软发布了一个全新的 Windows Redstone 预览版更新：Windows 10 Redstone Build 11102。本次更新主要针对参与 Windows Insider 测试会员的用户，并且要求开启“快速更新（Fast ring）”选项。据称，Build 11102  仅重点针对 Windows 10 默认浏览器 Microsoft Edge 进行了一些重要特征的更新。

微软 Windows Insider 项目负责人  Gabe Aul 在其 Twitter 账户上警告称，无论是游戏玩家还是常规用户，都应该好好地看清楚一些已知的问题列表，因为这个版本涉及到了影响一些游戏体验的问题。

具体来说，微软表示升级该版本将会引起一些 PC 游戏出现崩溃的情况。对此，微软提供了一份相对详细受影响的游戏清单，指出这些游戏在更新最新预览版操作系统之后，可能会无法运行的情况。

Gabe Aul 表示： “当从窗口模式切换到全屏模式的时候，一些 PC 游戏会出现崩溃的情况，原因是 Windows 图形堆栈的 BUG，游戏分辨率发生变化就会出现错误。我们已经注意到了，《巫师3:狂猎(The Witcher 3: Wild Hunt)》《辐射4(Fallout 4)》《古墓丽影（ Tomb Raider）》《刺客信条（ Assassin’s Creed）》和《合金装备5:原爆点(Metal Gear Solid V: Ground Zeroes)》会出现崩溃，但其他游戏也可能会发生。”

似乎此 Windows 10 系统的最新预览版不只是不适合游戏玩家，因为一些应用程序，如系统内置的放大镜（ Magnifier）或第三方类似的辅助工具，也可能会发生崩溃的现象。此外，在更新这个新版本的过程中，某些电脑可能还会出现一些莫名其妙的错误，比如 Wi-Fi 无线网卡不兼容 Windows 10 等等。

对此微软承诺称，这一系列的问题将在未来某个固定的版本中才会修复，关于无线网卡不兼容暂时可以通过安装最新适配该网卡的驱动程序解决。

**新特点又是什么？**

说了一大堆全新的预览版可能会出现的 BUG，那么该升级究竟更新了些什么呢？尽管有  BUG，但是依然还是对之前的 BUG 进行了修复，以及进行性能改进。Gabriel Aul 表示，Build 11102相对于Build 11099 进行了 1200 多处改进，因为大多都是底层的改进，所以是不可见的。

当然，微软在这个版本中重点新添加的功能主要针对 Edge 浏览器，为其返回按钮添加历史访问记录的功能，因此平时浏览网页的时候，通过使用鼠标的右键点击前进和后退按钮，就可以快速地访问最近浏览过的网站了。

微软表示，在 Windows 10 内置的反馈应用程序中，“快速访问历史网站”是最受欢迎支持率最高的功能之一。不过，微软并没有响应用户对广告拦截功能的渴望，实际上这个功能更受欢迎，究竟是否会添加到开发计划中还是未知数。

**视频快速预览 Build 11102 新功能：**

最后需要注意的是，Windows 10 Build 11102 更新仅适用于“开始更新”的 Windows inside 测试用户，如果未参与会员计划或开启该选项，将不会收到此更新。
