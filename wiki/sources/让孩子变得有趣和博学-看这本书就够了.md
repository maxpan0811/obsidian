# 让孩子变得有趣和博学，看这本书就够了

![](attachments/8f8de3ee85ec40c0.jpg)

看点我们都希望培养孩子的问题意识，但如何才能培养孩子提出问题、解决问题的能力？在下文这本书中，传奇作家兰道尔·门罗就以严谨而幽默的话风，回答了一个又一个天马行空的问题，不仅能让孩子增长知识，培养想象力，还能鼓励他们提出问题，做出假设，真正享受回答问题时的思考过程。

有这样一位作家，他的所有书都获得比尔·盖茨推荐，被称为“美国的国宝”。有这样一本书，从出版就收获无数媒体、科学家赞誉，中文版更是获奖无数：大众喜爱的50种图书，第11届文津图书奖科普类推荐图书，豆瓣读书年度好书……几乎横扫各大图书畅销榜。

这位作家就是美国科普作家兰道尔·门罗，而那本收获无数赞誉的书就是他的畅销趣味科普读物《what if？那些古怪又让人忧心的问题》。

现在，在出版第八个年头，它的第二部《what if？2那些古怪又让人忧心的问题又来了》终于在读者和业界的热切期待下出版了。

![](attachments/3eb704e79fccac67.jpg)

据很多成年读者说，《what if？》让他们变得博学、有趣起来，而我最想把它推荐给青少年。

为什么它很适合青少年？在读过那么多科普书后，再看到门罗的书，了解门罗写书的过程，不得不说，在中文语境下，没有人能做这样的事，更没有人能写出这样的书。

兰道尔·门罗其人

![](attachments/85d0be7b61c0f1cf.png)

门罗从克里斯托弗纽波特大学物理系毕业，进入NASA兰利研究中心研究机器人。

在工作之余，他喜欢画火柴人漫画。当他将这些漫画发到网上时，他收获了第一批读者。或许是发现在网上发布漫画比造机器人更有趣，他建立了xkcd.com网站，离开了NASA，成为一名全职网络漫画家。

![](attachments/dc05c2bede053b06.jpg)

在这个网站上，网友们向他提出五花八门的荒诞问题，而他总是试图用科学知识解答。

他的知识储备量巨大，玩转物理、天文、生物、数学、历史、人口学等各种知识。即使是他不懂的问题，也会在全网搜索，找到这个领域的专家，并与之建立联系，最终获得解答方法和答案。

在这样一个充满快节奏和浮躁的时代，能够十年如一日地坐在电脑前查阅、整理科学知识并有条理地回答网友的问题，就非常人所能为。而在奉上严谨答案的同时，门罗还会附赠一份冲走枯燥的趣味，更是极其难得。

![](attachments/6d86b4414760c7e5.png)

这也让他收获一群拥趸，其结果就是，更多荒诞的问题源源不断地向他提出。但门罗似乎并不担心，他说：“再荒诞的问题，都可能有更多符合科学原理的答案。”

![](attachments/1c1da99f532ff079.jpg)

孩子们奇奇怪怪的问题

![](attachments/ce9bf85f75986e53.png)

中藏着想象力和未来

书中的问题有多荒诞？

比如：

如果地球是一只巨大的眼睛，它能看多远？

我想减重20磅，为了实现这个目标，我需要把地球上的多少质量“扔”到太空？

如果所有人同时打开冰箱，能为地球降温吗？

你需要喝多少水才能达到含水量99%的状态？……

而书中最多的问题是孩子问出的，比如：

我五岁的儿子今天问我：如果月球和地球之间有一根消防滑杆，那么从月球滑到地球需要多长时间？

8岁的德克兰问：如果一群蚂蚁突然出现在你的血液中，你会怎么样？

如果稍加留意，我们就会发现，在日常生活中孩子们经常会问这类奇奇怪怪的问题。而我们千万不要忽视这类“奇怪”的“小问题”，有可能他们会因此而喜爱上某个领域或者专业，也会启发更多智慧。在这些问题中，藏着的是孩子的想象力和未来。

![](attachments/a8e986e214addd75.png)

值得注意的是，同样是问答知识书，what if 脑洞问答系列书的问答和“十万个为什么”有很大不同。

“十万个为什么”，是在已有事实上人们发出的疑问，答案通常是固定的，即每个孩子问出一个问题后，获得的答案基本上是一样的。

而门罗的脑洞问答系列是“如果……会怎么样”的模式，也就是，在假设一个条件成立的情况下，会发生什么。

首先，问题都可以是不符合现实的，荒诞的，而确切的答案通常需要一系列的数学计算和论证，或者论证方法不同，答案也会不同。

有人要问了：假设又不会发生，而它也不能给我一个确切的答案，那我读它还有什么用呢？

这正是这本书的奇妙之处，也是最大的价值所在——鼓励提出问题，做出假设，在回答问题的过程中感受思维流淌和碰撞的美妙。最重要的不是答案，而是思考的过程。

![](attachments/2ae128d5b3e5df96.png)

What if 脑洞问答系列书还藏着每个人生存在这个世界上的最重要的能力——解决问题的能力。

网友在不断地向门罗提出问题，而我们的生活中也在不断地出现问题。看看门罗是如何解决问题的。

在《what if？2》的“木星来到我们镇”一章中，一个9岁的小朋友问了门罗这样一个问题：

“亲爱的兰道尔，如果我们把木星缩到一幢房子那么大，然后把它放在街区里，比如说代替一幢房子，会发生什么？”

![](attachments/139ad2953f16e77a.png)

现在，我们开始跟着门罗一起解决这个问题了。

首先，要把木星变小就要知道木星的密度，然后门罗推导出房子大小的木星仅有 2500 吨重。这样重量的一个球放在你家屋子旁边不会发生什么糟糕的事。

但是，木星非常热。木星为什么非常热？它的温度是多少？如果把它压缩到房子大小，会不会爆炸？一个非常热的木星放在你家房子旁边，会把房子点着吗？这条街道会发生什么……

这一系列问题，在门罗严谨而幽默的讲述中被一一解答和展现。整个推理和解决问题的过程中，各种知识和新奇的想法也都包裹在其中了。

![](attachments/22c1e4684c98bb52.png)

![](attachments/9b93fbc8be9d29e0.png)

读what if 系列能够很好地

训练我们的逻辑思维

比尔·盖茨认为：“门罗的方法之所以是学习科学的好方法，是因为他采用了每个人都能理解的想法，然后开始探索当你把这些想法发挥到极致时会发生什么。”

《what if？2》的译者苟利军在译后访谈中说：“读《what if？2》是能够很好地训练我们的逻辑思维的一种方式。他先假设一些条件——尽管不太可能发生——接下来就严格按照推理的方式去验证一些事情，最后得到一些比较好玩的结论。”同时，他也非常想把这本书推荐给青少年。

苟利军是中国科学院国家天文台研究员、《中国国家天文》杂志执行总编、北京天文学会副理事长，也是电影《流浪地球2》的科学顾问。另外两位译者分别是前《第一财经周刊》记者、《中国国家天文》特邀编辑邓舒夏，云南大学中国西南天文研究所教授、文津奖得主尔欣中。

除了一些荒诞的问题，门罗的what if 系列中还有一些与现实联系紧密的问题，比如《what if？2》中就有一个现实中亟需解决的问题——如何用更少的能源发射飞机。

我的朋友是商业航空公司的飞行员，她说飞机起飞会消耗大量燃料。为了节省燃料，我们为什么不能像航空母舰那样使用弹射器系统（调整到普通人能接受的加速度）来发射飞机呢？如果用某种清洁能源来驱动弹射器，是否能节省很多化石燃料？我正在想象一根绳子…… 一头拴在飞机上，另一头绑在悬崖边的一块巨石上，然后把巨石推下悬崖！

按照门罗的推理过程，如果用弹射的方式让飞机起飞，那么会出现两个问题，一个是地面附近稠密空气的阻力会拖拽飞机，让你在进入上层大气层之前损失掉一些速度。另一个是土地问题，即需要一条非常长的跑道。

门罗说：“利用弹射器发射客机的主意并不全然荒唐。虽然节省的燃油可能不多， 但它能让更大的飞机在更短的跑道上起飞。弹射器还能让起飞不那么吵，噪声问题对城市机场来说是老生常谈了。”

……

这样的问题一个接着一个，而门罗也用他丰富的知识储备和强大的逻辑思维一一解答。

![](attachments/d37579b5e2dba7eb.png)

跟着门罗将一个个问题推理下来，通常会有酣畅淋漓之感。从每一个问题进入之后，到你看到他的解答，再出来，你都会经历一场逻辑思维上的奇旅。而这样的思维奇旅在书中有很多。

我常常感叹现在的孩子非常幸运，在儿时就能读到这些有趣、能增长知识和满足想象力的书籍。

但我也强烈建议大人们先读这套书，然后再给孩子读，并且一起讨论，以免像我一样，当孩子问到奇奇怪怪的问题时总是一副手足无措又不好直说“我不知道。”
