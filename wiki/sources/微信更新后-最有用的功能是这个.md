# **大**家好，我是精选君，**一位喜欢Kindle的数码学习博主。**

![](attachments/d91dcef87996c0af.png)

最近微信发布了 8.0.30 版本。比较值得关注的更新点有：

# **1.订阅号取关更容易了。**长按推送的文章，就可以一键取关。PS：不要取关我……

# 

# **↑ 爱范儿，不要担心，不会取关你**

# **2.给好友做备注时，可以添加最多三张照片（以前是一张）。**

# 

# ↑ 最多添加三张照片

# **3.在朋友圈里给朋友评论时，输入框可以延展。**但我一直觉得，如果要输入很多内容，最好还是先在电脑上写好、再复制粘贴吧。

# 

# **↑ 文字少时输入框跟之前一样（左）、文字多时输入框延展（右）**

# **4.「收藏」里的笔记功能，新增了「时间戳」。**这个功能好适合用来记账。

# 

# **↑ 时间戳**

# 这些功能里，个人觉得最有用的更新是：**好友备注可以添加多张图片。**而且为了让这个功能更好用，微信还支持直接从聊天对话中选择。

# 

# 为什么？**因为这解决了添加陌生人为好友后的信息备注问题。**

# 我之前的痛点是：

# 1.加了一个好友，他发来一张名片，自然要保存到备注里。这时候，就不得不先下载图片，然后再打开备注，插入图片。麻烦！现在可以直接点一下**「从聊天中选择」，**即可插入。

# 

# **↑ 从聊天中选择照片**

# 2.以前只能插入一张图片，添加了名片就加不了其他的了。而最好的情况是再增加一张他的生活照，方便日后回忆起他真实的模样。毕竟，**大多数人不用真人照做头像；即使用，也美化过度。**

# 

# 

# ↑ 你说，这是同一个人吗？

# 所以，功能虽小，但是对于经常要对外打交道的朋友来说，非常友好。

# 另外，我们之前也整理了很多微信的实用小技巧，今天一并整理给大家。

![](attachments/34e355e9f339e900.png)

# **01 消息通知横幅**

# 操作路径：点击**按「我」，选择「设置」，再选择「消息通知」，勾选「消息通知横幅」。**

# 通过**「消息通知横幅」**功能，当我们在看公众号文章、视频号直播时，遇到新消息，即可看到一个横幅，点按后即可快捷回复。

# 

# **02 全屏输入**

# 操作路径：**长按对话框，选择「全屏输入」。**

# 有消息要发给领导时，我都会先在一个对话里编好文字，预览发送后的效果。通过**「全屏输入」，**可以更快捷地在手机上完成输入，从而提升编辑效率。

# 

# **03** **建立个人群**

# 操作路径：**发起群聊，选择「面对面建群」，输入一串数字即可。**

# 在日常工作和学习中，临时会有一些想法。

# 这时候，我会建立一个个人群，将思考发进去。

# 

# **04 提取图片中的文字**

# 操作路径：**长按微信中发出（或发来）的图片，选择「提取文字」。**

# 日常看书时，我们会读到一些金句，希望摘录下来。传统的方法就是用手机或电脑码出来，费时费力。微信提供了一个非常简洁的方法**「提取文字」。**多次使用感受是：识别准确率在 99% 以上。

# 

# **05 用微信做卡片**

# 操作路径：**进入「收藏」，点击右上角的「+」，输入文字；点击右上角的「…」，然后「导出为图片」**

# 有了这个功能，大多数便签类 App 都可以删掉了。

# 

# **06 把不常用的群折叠**

# 操作路径：**进入某个群，点击右上角的「…」，选择「消息免打扰」，再选择「折叠该群聊」。**

# 当我们选择将某个群**「折叠」**后，它们就会被放入一个小黑屋**「折叠的群聊」**里，类似于多个公众号被放入**「订阅号消息」**里一样。当你偶然想要查找某个群聊时，便可以打开寻找。

# 这适合于那些行业交流群，想起了，就去看看。

# 

# **07 信息稍后处理**

# 操作路径：**长按一条对话/文件，即可看到「提醒」功能。**

# 有时候需要集中一段时间处理重要事情，不想被过多打扰。中途收到不紧急的消息，设置“提醒”，晚点统一回复。

# 

# **08 利用微信收藏做旅行规划**

# 操作路径**：在「我」里，点击「收藏」，然后点击右上角的「+」，即可创建笔记。**

# 在微信收藏里写规划有个好处：可以插入地址。发送给他人后，点击地址即可定位，并进行导航。

# 

# **09 利用「浮窗」稍后阅读**

# 操作路径：**在微信里打开公众号文章，点击右上角的「…」，选择「浮窗」。**

# 文章读到一半，他人发消息来，这时候可以将文章放入**「浮窗」，**然后稍后阅读。在微信界面，向右滑页面，即可看到浮窗内容。

# 

# **10 置顶群内常用信息**

# 操作路径：**（群主）在群内长按某个信息（文字、链接、文件等），然后选择「置顶」。**

# 使用这个功能后，对应的内容就会出现在群聊的顶端。例如对活动组织者而言：当想统计大家的个人信息时，可以将问卷链接置顶，于是大家都能快速找到链接，完成填写。

#

# 今天重点介绍了微信最近更新的一项实用功能：给好友做备注时，可以添加三张照片。这可以方便日后更快回忆起他是谁。
