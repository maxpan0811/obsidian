6月23日，上海市考试院发布高考各批次控制分数线以及高考成绩分布表。

|  |  |
| --- | --- |
| **批次** | **分数（总分660）** |
| 本科录取控制分数线 | 405  \_ |
| 特殊类型招生控制分数线 | 504  \_ |
| 艺术类本科文化控制分数线 | 304  \_ |
| 体育类本科文化控制分数线 | 284  \_ |

从公布的一分一段表来看，2023年上海高考600分以上的考生共591人，高分段人数比去年（628人）减少了37人，高分段整体变化不大。

其中，619分以上48人，对比去年增加2人；600分以上591人，对比去年减少37人；580分以上2393人，对比去年增加58人。

![](.evernote-content/B95B6BE1-B569-431B-B4CC-E016CF809C9A/86D817B5-B6A5-4C0D-8911-DE02D654B1B2.png)

比对去年，今年高考分数线略有上涨。本科录取控制分数线为405分，比去年的400分上涨了5分，特殊类型招生控制分数线为504分，比去年的503分上涨了1分；艺术类本科文化控制分数线、体育类本科文化控制分数线，均比去年上涨了4分。

据网传，今年上海高考最强的学生来自华师大二附中9班的廖同学，成绩为633分！七宝中学23届5班刘同学，成绩630分！

**华师大二附中 633分！**

**七宝中学 630分！**

上海高考满分是660分，按照考生成绩除以高考总分计算：华二廖同学633分，得分率95.9%！七宝刘同学630分，得分率95.5%！
