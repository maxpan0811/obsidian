# 占领头条的 Windows 10，看这一篇就够了

---

[![](.evernote-content/7C5C30F2-FC6B-4642-ACE8-F97E8EA760C3/CB312A5D-DFFB-4911-A681-0E4E25F6B2BC.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/microsoft2.jpg)

2015 年 7 月 29 日，Winboy 已经等待的太久了，今天微软终于带来了全新的操作系统 Windows 10.

尽管在发布之前做了太多的铺垫，但今天对于微软的粉丝来说仍然是个大日子，微软也将在今日邀请了全球的粉丝一起见证这个伟大的日子。

跟之前的介绍的内容差不多，Windows 10 的更新可以被概括成几大块，首先微软带来的第一个好消息就是免费升级。

### 盗版可以升级么？

微软的 Windows 7，Windows 8 以及 Windows 8.1 的正版用户都可以升级为 Windows 10 对应的系统，例如 Windows 7 家庭版可以升级为 Windows 10 家庭版；专业版则可以升级为专业版。而从未拥有过正版系统的用户需要购买，中文版的 Windows 10 家庭版为 888 元、专业版为 1799 元。

当然此前如果正在使用 Windows 7 以及 Windows 8 的盗版用户如果此前注册了 Windows Insider 的会员，并参与到 Windows 10 Preview 测试的用户，将会陆续收到微软推送的更新提示，微软也曾表示在发布当天会优先将 Windows 10 正版系统推荐给 Windows Insider 的会员，用来感谢他们在测试版本时的付出。

所以说如果你在今天之前没有拿到 Insider 的会员，又处于一个盗版系统的状态，那就只能选择购买正式版了。

此前，微软还扩展了一系列的推广渠道，诸如腾讯电脑管家以及 360 安全卫士，他们将协助微软帮助用户更新到 Windows 10.

简而言之就是希望更多的用户参与到 Windows 10 更新的队列中来，但总体上讲，正版升正版，盗版升盗版，付过费的已经付过了，没付过的还得重新付。

可能有一部分人特别关心的 Surface RT 以及 Surface 2 等设备的更新，而此前在微软的一次研讨会上表示目前并不能升级为 Windows 10，而可能替代它的是 “看起来” 像 Windows 10 的升级版，就像当年 Windows Phone 7.5 升级为 windows Phone 7.8。

### 内置的小娜（Cortana）和小冰能做什么？

小娜语音助手将正式登陆到 Windows 10 操作系统，它将被放在 Windows 10 的开始菜单内。微软的意思是将她打造为了一个依托云计算大数据的全能个人助理，微软说他们很在行大数据运算。

[![](.evernote-content/7C5C30F2-FC6B-4642-ACE8-F97E8EA760C3/450E43AE-997D-491C-8C66-F97F6AFDA6EC.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/cortana.jpg)

微软的介绍说，小娜助手可以完成的更多，小娜不仅限于设置闹钟，发送短信，发送 Email 等固定程序。微软还说，小娜对比苹果的 Siri 则更智能，小娜可以通过用户的使用频率，说话方式以及用户习惯做出统计以及智能分析，比如当你使用浏览器频繁搜索了附近的川菜馆，而后在你询问小娜“附近有什么好餐厅?”的时候，小娜就会优先推荐附近的川菜馆，智能过滤掉一些其他菜馆信息。

而小娜还可以直接回答类似“今天我家的天气如何？”这种问题，当然还有更多的一些类似“模仿周杰伦”，小娜会直接模仿出周杰伦的口头禅；还有“给我讲个笑话”，小娜也会直接给你说一个笑话。注意他并不会打开浏览器去搜索这些话。

微软还强调，小娜的前景是未来可以开放更多的程序接口，可以进行更多程序内的信息统计，整理，操作。例如：帮我在携程上定一张 7 月 30 号飞北京的机票。小娜可能将会直接帮你下单。

而微软介绍小娜对于国内的众多语言做过优化，小娜甚至可以听懂部分方言。

第二部分，微软把小娜助手形容成为一个睿智知性的女性，而小冰则被微软描述为一个萌妹子，它在 Windows 10 中充当与用户聊天的作用。

在打开小娜的工具栏后，输入文字或语音说出小冰就能召唤出这个语音助手了。

在微软内部的分享会上，微软展示了相同的询问句式在小娜助手与小冰助手之间的区别，小冰助手回答问题的方式非常俏皮，微软称小冰开启了一项全新的图像识别技术，小冰注重将图片中识别出的内容转换为一个更拟人化的评论。

比如当你发送一张受伤的照片图片时，小冰可以直接回答出“啊，怎么这么严重啊？”之类的关心的回答；再比如，你发送一张狗的照片的时候，普通的图像识别技术的机器人可能可以识别出这是一只狗，而小冰的表达则类似“如果再瘦点就好了。”

### 为了体验的 Windows Edge 浏览器与为了兼容老程序的 IE 浏览器

微软宣布了全新的 Edge 浏览器，并保留了最早的 IE 浏览器。这更像是微软对于曾经攒下的坑与未来做出的妥协。

[![](.evernote-content/7C5C30F2-FC6B-4642-ACE8-F97E8EA760C3/A05ADFD0-9EF2-4992-8E9F-E30910101602.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/edge.jpg)

微软介绍称，介绍全新的 Edge 浏览器是因为证明微软有实力也有必要更新一款全新的浏览器，这次的 Edge 就是一次尝试。微软还表示，Edge 浏览器内部跑过一些浏览器的测试，Edge 浏览器都跑到了一些其他浏览器的前列。

而且，Windows Edge 整合了微软的 Bing 搜索服务，并可用自家的小娜数字助理进行辅助查询。

当然它还拥有阅读模式，将通篇复杂网页的内容转换成更易于阅读的文字预览，集成 OneNote 以及网页笔记的功能，方便用户进行记录，标注与分享，被标注的网页内容可以被调用到第三方程序，微博、Email、Evernote 等进行分享。

而保留之前的 IE 浏览器则是因为微软认为目前处于 IE 浏览器到 Edge 浏览器的过渡期，IE 浏览器对于一些老网页程序插件的兼容性非常好，Edge 浏览器则用于全新的 Windows 10 体验，相信日后微软会逐渐融合这两个浏览器。

### Continuum 后界面不分平板还是 PC

微软介绍这个功能将提升用户与设备以及屏幕之间的配合程度，更有利于跨平台的体验，简单的来说，当你使用 PC 模式（类似 Surface 连接键盘）的时候，屏幕显示的内容则是基于 PC 的，有桌面，有开始菜单，也就是打开 Modern 程序则是窗口化运行的。

[![](.evernote-content/7C5C30F2-FC6B-4642-ACE8-F97E8EA760C3/342BF28C-2CFF-42B6-AF46-0E1CA1E8739F.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/windows-10.jpg)

当设备检测到你拔掉了键盘，将自动提示你是否切换到平板模式，当点击切换到平板模式后，开始菜单内的所有应用程序将会铺满屏幕进入 Modern 界面，如果目前有打开的应用，也会将其最大化，整个变化过程也是非常流畅的。

当然 Windows 10 还可以在更多的平板、一体机以及笔记本中实现这种功能。这样对于众多 Windows 设备 Windows 10 则将达到整合平台的目的，也就是说当你用一个账户购买的程序将可以在所有设备上支持运行。

### 游戏玩家的福音

Windows 10 这次不光在移动设备、桌面设备之间做着打通平台的事情，就连自己的 Xbox 也被添加到 Windows 10 的应用中。微软称 Xbox 也即将被升级为 Windows 10 的内核。

[![](.evernote-content/7C5C30F2-FC6B-4642-ACE8-F97E8EA760C3/892A21C5-9715-403C-9039-BC0D3907133D.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/xbox.jpg)

Xbox 应用的功能主要是完成用户在 Xbox 社区的问答，分享等操作。比如一款游戏的社区的讨论，发帖，回帖。此外，Windows 10 Xbox 应用还支持录制游戏视频，并可以在 Xbox 应用中直接分享。

而说到录制视频，就得谈到 Windows 10 Xbox 应用的一个新功能，Windows 10 支持 Xbox 的串流功能，就是将 Xbox 运行的游戏画面投射到 PC 的显示器上。

当我们使用 Xbox 进行游戏的时候，使用 Windows 10 上的 Xbox 应用连接 Xbox 主机，并同意串流到 PC 后，PC 显示屏将可以显示 Xbox 游戏内容，当然不光是这样，你的 PC 连接上的手柄也是可以工作的。

微软介绍的使用场景大概是这样的，你的朋友晚上使用电视正在看足球，而你则可以打开 Xbox 主机，使用串流功能在 PC 上玩游戏，这样就不用跟自己的朋友抢电视了。

还有这只是 PC，微软介绍了一个更宏大的游戏场景，你的 Xbox 游戏内容可以投放到任何装有 Windows 10 的设备上去，比如说你的平板，当然如果游戏支持被投射产品的操作方式的时候，甚至你都可以用平板触控操作你的 Xbox 游戏。

想想你一下，你可以用 Surface Pro 3 或者你购买的 Windows 10 寨板上打 Xbox .

### 不知道谁抄谁的 Mission Control 与多桌面支持

微软在 Windows 10 的窗口多任务操作上也做了一些优化，首先要说的就是屏幕区域划分的功能，Windows 10 的桌面支持最多将屏幕划分为四个工作区。

[![](.evernote-content/7C5C30F2-FC6B-4642-ACE8-F97E8EA760C3/8EDB9404-929A-4BF1-985C-1E783FC67F03.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/mission.jpg)

具体要这么做，当你需要两个窗口分列左右操作的时候，就可以将窗口任务栏拖到屏幕的正左方，这时候系统就会自动将系统划分成左右两个区域摆放两个窗口。

当然，当你将窗口任务栏拖拽到左上角的时候，系统就会分列出四个相同大小的窗口让你在四个窗口任务中同时工作。

除去划分一个桌面外，Windows 10 还支持添加多桌面，方便将不同的窗口任务拖到其他的桌面中去。而使用 Alt+Win+左右方向键就可以在多桌面之间快速切换，听起来是不是有点熟悉？

还有微软还添加了对触控板手势操作的支持。

具体的手势是这样的。首先双指同时敲击作用相当于鼠标右键，三指同时向上滑则是调出多任务预览（多任务窗口平铺在桌面），三指同时向下滑则是回到桌面。

### Windows hello 简直有些黑科技

微软介绍说 Windows hello 是 Windwos 10 支持的一个生物识别技术，需要配合配备了 RealSense 摄像头的笔记本使用（当然未来还会有支持单独技术的硬件摄像头卖） ，当一台 PC 配备这个摄像头的时候，Windows 10 可以通过虹膜扫描识别用户进入系统从而免除密码输入。

[![](.evernote-content/7C5C30F2-FC6B-4642-ACE8-F97E8EA760C3/3BA5832F-6B7F-4238-B1F1-771E2428C782.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/07/hello.jpg)

微软演示了这种用户登录的解锁方式，整个解锁时间不到一秒，速度非常快。几乎在用户走到电脑屏幕前的一瞬间，就已经进入桌面了。

微软强调这种扫描能力是支持活体感知的，也就是说当你拿着一张用户的照片去“刷脸”解锁的时候，是不被识别的。

当然微软也介绍了这种硬件技术的开发前景，如果这种技术一旦开放了 API ，将会大大有利于我们“舍掉密码”。

### 其他的一些新玩意

除此外，Windows 10 还将为用户带来 Office 的通用版本，这个 Offcie 版本可以直接在应用商店下载，而这个版本可以帮用户在平板、手机、PC 等客户端跨平台操作。当然，如果你并不是 Offcie 365 的订阅用户，这个版本算的上是一个简版，只拥有“只读”的权限，并不能编辑。

微软也提供了类似图片、地图、音乐以及邮件等内建的应用程序，他们在不同平台的设备上的协同工作以及数据显示方面都做了优化。

而这种跨平台的应用被微软称为 Windows 10 统一平台应用的设定。

微软还新设定了右侧的通知中心，你可以在这里查看备忘录，最新的通知，通知中心的下方还拥有VPN 以及平板模式等开关。

### 结语

微软 Windows 10 带给了我一些惊喜，这些惊喜可比我在 Windows 8 上要更多。

想到了早前微软一名技术高管曾在一次大会上表示：

> 这将是我们最后一版 Windows 。

微软这句话的意思则是希望未来 Windows 软件即服务的模式将会被大幅改变，这句话一出便引起了无限遐想，微软不靠软件还能靠什么呢？

不管了，Windows 10 今天已经正式公布了，从今天开始微软将翻开新的一页。
