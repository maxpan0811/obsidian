## 一句话摘要

YC 总裁 Garry Tan 开源的工具包 gstack（89.7K Star），将工程团队拆成三层架构——决策层、执行层、守门层——共 23 个 AI 角色，一个需求从想法到上线要逐层穿过三层关卡。

## 三层架构

| 层 | 职责 | 角色数 |
|----|------|:------:|
| **决策层** | 该不该做、按什么架构做 | 普通用户最少触发的层 |
| **执行层** | 拆任务、写代码、跑测试 | |
| **守门层** | 审查、安全、质量把关 | |

Karpathy 最近在 No Priors 上说「已经基本没敲过一行代码了」，Garry Tan 晒出 GitHub 密集绿格子——用的都是同一套工具。
