下文是昨天宣讲的2023年北京、上海、广州、厦门、成都、武汉、美洲、中东非、日本、国内游业务单元的KPI考核方案，宣讲时也讲了，由于今年还有些不确定性因素，此方案为今年过渡性方案，请大家知晓，详细内容见下文：

| 各分公司/事业部2023年KPI绩效考核指标（北京、上海、广州、厦门、成都、武汉、美洲、中东非、日本、国内游） | | | | | |
| 序号 | 指标项目 | 权重 | 指标考核计算方式/可衡量的定义 | 考核范围/评分依据 | 数据来源部门 |
| 1 | 费用毛利率 | 40% | 考核方式：  上半年：  实际费用毛利率（α）达成情况  α<1：1.2　卓越  1≤α<1.2：1　优秀  1.2≤α<1.4：0.8  良好  1.4≤α<1.6：0.6   及格  α≥1.6：0  不及格    下半年（累计上半年数据）：  实际费用毛利率（α）达成情况  α<0.8：1.2　卓越  0.8≤α<1：1　优秀  1≤α<1.1：0.8 良好  1.1≤α<1.2  0.6  及格  α≥1.2：0  不及格  一票否决项：  年度营收<3000万，亏损上限-60万；  年度营收≥3000万，亏损上限-100万。  亏损额超过上限，此项分值为0. | 负责业务单元整体考核  费用毛利率费用总额/毛利总额  费用包含人力成本及其他期间费用（含房租，水电，物业费，其他期间费用等）  说明：  1.事业部按分润后财务数据计算；  2.北京公司毛利只算出境业务毛利，锡盟业务不在此考核范围，单独考核。 | 财务部 |
| 2 | 经营管理 | 25% | 1.市占率和关键渠道覆盖：市占率如在现市场业务属于第一梯队即是卓越。关键渠道覆盖：核心关键渠道在现有的渠道业务贡献占比情况进行评估．  ２.产品和资源：产品丰富度;与对手比较，产品性价比及资源优势．  3.应收款管理:具体见财务规定 | α≥100              卓越  85≤α<100         优秀  70≤α<85           良好  60≤α<70            及格  α<60                 不及格 | 分管高管 |
| 3 | 质量 | 好评率－10% | 整体好评率达到98%以上 | α≥99%：1.2　卓越  98%≤α<99%：１　优秀  95%≤α<98%：0.8　良好  90%≤α<95%：0.6  及格  α<90%：0  不及格 | QA |
| QA－15% | QA:100%合格率-缺陷数/开团数(团队数刨除下线团队)  合格率：(α) | α≥99% ：1  98%≤α<99%：0.8  97%≤α<98%：0.5  96%≤α<97%：0.3  α<96%：0 |
| 4 | 团队建设 | 10% | 1.员工离职率，考核占比5%（不含主动淘汰员工）  ２.梯队健康度：关注团队成员成长，骨干成员状态佳，吸纳外部优秀人才，加强团队梯队搭建。 | 1、员工离职率，考核占比10%（不含主动淘汰员工）  α<３%：1.2　卓越  ３%≤α<５%：１　优秀  ５%≤α<１０%：0.8　良好  １０%≤α<１５%：0.6  及格  α≥１５%：0  不及格 | 人力资源部和分管高管 |
