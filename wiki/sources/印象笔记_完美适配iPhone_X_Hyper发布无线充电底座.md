# 完美适配iPhone X Hyper发布无线充电底座

---

[![](.evernote-content/767C1D47-A72D-4B7C-8C30-06E91EC51836/973DD3E8-4299-4845-AA66-59A66D1888C3.png)](http://resource.feng.com/resource/h062/h25/img201801161413050.png)  
  
这款设备名为 HyperDrive，体型迷你携带轻便。它提供 8 个端口的 USB-C 转换，包括 4K HDMI，千兆以太网，microSD，SD，3 x USB 3.1和 USB-C 电源传输。同时还支持最新的 Qi 无线充电标准，可以为 iPhone 8、iPhone X 提供标准的 7.5W 无线充电，另外还支持 5/10/15W Android 设备无线充电。  
  
HyperDrive 附带一个可转换的 iPhone 支架，可以横向或纵向支起 iPhone，不干扰用户解锁或者点击操作设备。内置 3 个无线充电线圈，保证你的手机一放上去就进入充电状态，没有充电盲区。使用内置的可伸缩 USB-C 连接线将 HyperDrive 连接到 MacBook 时，可以转由 MacBook 为手机等设备提供电量。  
  
该设备的优势在于它是目前市面上唯一支持所有 Qi 无线充电规格（7.5W，5W，10W和15W）的无线充电器，尤其是苹果的 7.5W 无线充电，官方宣称 iPhone X 只需要 2 小时 58 分钟就能完全充满，比 mophie 和 Belkin 无线充电器的 3 小时 21 分还要快。使用 Apple 指定的固定频率，不影响 Touch ID 或 Face ID。支持固件升级，可升级固件以支持未来更快的 iPhone 无线充电速度等新功能。

[![](.evernote-content/767C1D47-A72D-4B7C-8C30-06E91EC51836/9C2B88F3-C5D2-4AF3-A599-D1CEDC90044B.gif)](http://resource.feng.com/resource/h062/h25/img201801161413310.gif)

[![](.evernote-content/767C1D47-A72D-4B7C-8C30-06E91EC51836/E80B44A4-EED4-42D9-9C4A-76636FA5951F.gif)](http://resource.feng.com/resource/h062/h25/img201801161413500.gif)

HyperDrive 还做到了良好的控温效果，据称设备使用高能效组件，产生的热量从铝制侧面消散，并通过气隙和绝缘体可转换支架层与 iPhone 隔离，不会因为充电过程中温度过高损害手机以及电池。

HyperDrive 发布后在 kickstarter 进行预售众筹，可以 99 美元（约 630 人民币）的优惠价提前购买，4 月份到货，正常零售的原价为 159 美元。

随着苹果 iPhone 等智能手机采用无线充电，无线充电相关设备会在今年得到更长远发展。上周举行的 CES 展上就有多个厂商展示自家的无线充电产品，包括旅行、家用、车载，并且还推出了公共场所比如酒店、餐厅、机场等的嵌入式无线充电壁挂，方便将来人们的出行充电。当然，各种 HUB 的存在告诉我们距离真正的“无线”还有一点远，也许在 2018 年解决各种线缆的问题会成为大家最关心的问题。
