| |  |  | | --- | --- | | 我们孩子在上实读初一。半个学期过去了。汇报下上实的一些生活感受！  首先说点题外话：感谢E度论坛对我们小升初的帮助。我们到上海不久就赶上了小升初，对上海的教育状况不了解，多亏了这个论坛，才让我有了很多必要的信息的了解，让孩子顺利的通过了小升初     在报考上实之前，我们在网上搜索到关于上实学生学习生活的信息并不多，可能本身招生就少，真正了解的人就相对少一些吧！所以今天我要说说我们在学校里的学习和生活状况！以供小升初学生的家长们参考！      1.学校的课本内知识的教学方式每个老师都有自己的特色。教学模式已脱离填鸭式，以给孩子制定学习目标，鼓励与压力并重的方式为主，达到让孩子自主学习，取得成绩优异的目的！作业不算少，但是也不是太多，作业和考试成绩挂钩，成绩越差，作业越多！毕竟初中四年变三年，学习任务不重，那就是神话！在家的自主学习和预习很重要！最后还是要成绩说话的！（脱离不了大环境）你可以在上课时写其它科作业甚至干别的，只要你会了，成绩好，老师都不太干预。      2.有的老师认为，孩子已经长大，要学习模拟社会生活状态，所以，孩子在学校发生矛盾基本自己解决！      3.很多孩子的家长都是开着宝马的，但是，学校校风还是不错，基本没感觉到攀比现象！      4.老师们对孩子的态度对很和蔼，我们班主任就总是笑眯眯的说话。孩子的初中生活比小学愉快的多，是累并快乐着！      5.学校课外活动形式多样，丰富多彩，孩子只要有一点点特长，都会有机会展示和发展。不怕没机会，就怕没才华！还有，比如看英文原版影片，听科学讲座等等孩子们都非常喜欢！      6.除了学习成绩外，其它方面的才能也有德育分，算在总成绩里面的。所以，孩子的任何特长和兴趣发展都是有鼓励的表示的！      7.我们孩子没住校，抱歉不能提供相关信息！      8.孩子经常会上网收作业，听力作业等等。某些课程作业需要PPT或图文并茂的文档（要打印）来完成。请家长做好准备。    |  | | --- | | 在上实读书，感受上实！(2)  其实小升初的最重要的原则就是：不要最有名，不要最重点，只要最适合！  什么样的孩子适合到上实读书呢？我个人认为：  1.孩子聪慧又伶俐，不能是比较循规蹈矩的孩子。个性比较强一点，有点自己的思想和创意最好。动手能力，交际能力呀最好都能有一点。  2.说话做事彬彬有礼！家教要好！  3.非常重要的一点，要有自主学习能力，领悟力要强，能举一反三。成绩不一定是拔尖的，但一定是有后劲的孩子。自学能力非常重要哦！  4.有自己的爱好和特长！  5.课外活动很多，是某些人想要的，也是某些人不喜欢的。结合孩子的承担能力，请三思！  就说这些吧，不全面，只求能给大家做个参考！ | | |
