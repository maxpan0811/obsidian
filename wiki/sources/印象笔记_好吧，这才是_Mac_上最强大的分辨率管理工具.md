# 好吧，这才是 Mac 上最强大的分辨率管理工具

---

好吧，这才是 Mac 上最强大的分辨率管理工具
-----------------------

原创 *2016-05-11* *5key* Pinapps

![](.evernote-content/0E80363D-C47D-4C6E-8F80-9C48C3D9A3C2/0B25A659-328A-474B-8394-2CCBE0FECC49.jpg)

不知为何，最近市面上出现了好几款分辨率管理工具。比如[昨天我们介绍的 SwitchresX](http://mp.weixin.qq.com/s?__biz=MzA4NTMxOTgxNg==&mid=2649882567&idx=1&sn=8a9f9234a69ed2d4e38df35e3e14a222&scene=21#wechat_redirect) ，能够根据不同的 app 进行分辨率的设置。但如果你日常的工作生活使用双屏，那么今天要介绍的 Displays 才是最强大分辨率管理工具。

与 Switchresx 相比 Displays 的功能完全是另外一个方向，主要包含不同屏幕单独设置分辨率、屏幕画中画、演示辅助以及夜晚模式。

激活后 Displays 会常住在系统菜单栏，面板黄色区域的 icon 从左至右分别是 OS X 系统设置菜单、夜间模式、演示模式、屏幕详细信息和屏幕切换。

![](.evernote-content/0E80363D-C47D-4C6E-8F80-9C48C3D9A3C2/487D4C99-778C-4B22-B2F4-F6744B19BD9D.jpg)

通过呼出的面板可以快速调整不同屏幕的分辨率。试过几台显示器，Displays 也是对于屏幕设备识别最准确的应用。

![](.evernote-content/0E80363D-C47D-4C6E-8F80-9C48C3D9A3C2/2C1B829D-6BC5-408D-A904-30067ADB76CE.jpg)

**01.画中画**

使用「画中画」功能可以将第二屏幕放置在第一屏中。应用可以设置默认展示在屏幕的左上、左下、右上、右下四个位置，也可以在屏幕上随意拖动。唯一遗憾的是「画中画」的尺寸无法自行设置。

![](.evernote-content/0E80363D-C47D-4C6E-8F80-9C48C3D9A3C2/44FF7808-E725-4C69-82E0-C98E0265C076.jpg)

画中画效果：

![](.evernote-content/0E80363D-C47D-4C6E-8F80-9C48C3D9A3C2/4BF54942-E557-4AF0-B500-6D522767C900.jpg)

**02.分辨率标识**

正如昨天文章中所提到的，还是有不少人（特别是使用 rMbp）想要针对不同 apps 使用不同的分辨率。但记住不同 apps 的不同分辨率是一件麻烦的事情，在 Displays 中我们可以将1600分辨率设置标识为「Excel」，方便日后的使用。

![](.evernote-content/0E80363D-C47D-4C6E-8F80-9C48C3D9A3C2/F45ED2AF-3F54-460E-A594-DCEB62AED500.jpg)

分辨率标识展示：

![](.evernote-content/0E80363D-C47D-4C6E-8F80-9C48C3D9A3C2/F49B49CE-F098-49B4-BB63-8D8EBBC20F66.jpg)

**03.夜间模式**

之前也介绍过多款应用（比如 [ControlPlane](http://mp.weixin.qq.com/s?__biz=MzA4NTMxOTgxNg==&mid=2649882567&idx=1&sn=8a9f9234a69ed2d4e38df35e3e14a222&scene=21#wechat_redirect)）可以实现类似的效果。Displays 的方式则比较简单，为夜间模式设置好一个固定的亮度，通过面板或快捷键来激活触发。

![](.evernote-content/0E80363D-C47D-4C6E-8F80-9C48C3D9A3C2/8DB368FE-E3D1-4DCC-B986-45A47AD9BAAD.jpg)

**04.演示模式**

演示模式可以让我们通过功能面板或快捷键呼出一个聚焦蒙版，快速定位鼠标所指的位置。这在 PPT 演示或是视频录制中都非常的有用，这也是我最喜欢的一个功能。

![](.evernote-content/0E80363D-C47D-4C6E-8F80-9C48C3D9A3C2/D4F43624-8E50-4A7C-8BBC-FD88560182D2.jpg)

至于蒙版的样式，我们可以在设置面板中按照自己的喜欢自行设置。

![](.evernote-content/0E80363D-C47D-4C6E-8F80-9C48C3D9A3C2/42788CC2-9562-4B6B-949C-5D64216809DA.jpg)

Displays 的快捷键面板提供的功能也相对比较强大，可以为多个常用操作和分辨率分别进行快捷键的设置。可问题是现在键盘的快捷键键位严重不足，实在很难找出空闲的键位了。

![](.evernote-content/0E80363D-C47D-4C6E-8F80-9C48C3D9A3C2/AB053392-0E80-4879-8972-5EB790CDF09E.jpg)

相比昨天的 SwitchresX，Displays 提供的手动控制分辨率和演示模式等功能更为实用。如果你是 rMBP 用户而且经常外接显示器实用，Displays 确实是一个值得入手的工具。

Displays 刚刚上架，目前促销售价 2.99美金。有兴趣的同学可以到下方 Mac AppStore 地址进行购买：

> *https://itunes.apple.com/app/displays/id1107272470?l=fr&ls=1&mt=12*
