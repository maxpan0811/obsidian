熟悉小花生的朋友，大概都知道英国阅读体系 - *Collins Big Cat，*简称**“大猫”系列**。目前国内市场上，这是唯一的全面引进的英国学校主流阅读体系；目前，越来越多国内的学校和机构在开始使用。

英国学校是没有正式课本的，大猫这类阅读体系（Reading Program），事实上起到了英语课教材的作用。**大猫是一个**完整的阅读进阶体系，****从最初的英语启蒙到小学毕业，有20个级别，分级细致、严谨、科学，每个级别都包含数十本书，“战线”长、体量大。

今天，花生团推出的是**《大猫》系列3-11级的书目**，这9个级别，可以构成**一个适合中国小学生1-6年级的完整英语课外阅读体系**，小步走，但步步进阶，从而攻克小学阶段阅读大关。

如果能坚持下来，在阅读过程中**会获得对英语能力的全方位提升**：**词汇、语法、阅读、写作、口语表达。**此外，作为一套英国本土孩子用的阅读教材，**语言学习和知识学习是“融为一体”的**，在主题和内容上，“同步”了英国小孩的学科学习内容，比如科学艺术、社会人文等。

目前，像“大猫”这样适合小学生的完整阅读书目体系，市场上非常稀少。如果你有心让孩子借助原汁原味的阅读资源，在小学阶段在英语学习上获得突破，这是一个值得信赖的选择。

*1*

这是英国孩子广泛在用的阅读教材

在英国，主流的 Reading Program 有好几个，其中，有历史悠久的 *Oxford Reading Tree*，也有近十几年发展势头很不错的*Collins Big Cat*，其它还有*Project X*, *Rigby Star* 等。

这些Program体系都很庞大。比如，Oxford Reading Tree 就有800多本书。（注：国内大家用的比较多的叫 *Read with Biff, Chip& Kipper*，也就是牛津树的家庭阅读系列，主要的学校教材系列尚未原版引进。）

Colins Big Cat 也够有规模，从幼儿园到小学毕业，有近700百本之多。下面这张图，是整个 Program的结构（大家点开可放大看），右上角的绿色字块说的是：

> *Collins Big Cat has been aligned to reflect **the higer reading standards**outlined in the 2014 **NATIONAL CURRICULUM**.*
>
> 柯林斯大猫系列体现了英国2014最新教纲中定义的**较高阅读水准**。

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/A7768822-2529-4738-8DEC-0CC1983F6D23.jpg)

来源：Collins官网，建议大家把大图下载，可以作为整体学习结构参考；

大家看到这里五颜六色的，是这么回事 ...

Reading Program 的进阶是遵照英国统一的*Band*（分级系统）来设定的。简单来说，在一个 Reading Program 里，英国本土孩子从开始接受教育，一般是4岁左右开始，一直到11岁小学毕业，**学英语被分成了近20个阶段（Band），每个阶段用一种颜色代表。**各个出版社推出的 Reading Program 里的书，**都会按照统一的标准去对应这些Band**，每本书的书脊上也会标注相应的颜色。

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/CD6D4370-4899-43E1-B71B-4D7DEDD90A2E.jpg)

右上角标注本书属于Band Green，即第5级

对于小学生来说，这是正式和规范的阅读体系，跟着这套书，孩子可以一点点把英语“读”上去。

*2*

为什么这套书适合中国小学生？

孩子进了小学，功课越来越忙，留给课外英语学习和阅读的时间非常有限。比较理想的状态是：一周有3-5天，每天在30分钟左右的时间内，能基本完成一册书的阅读，通过长久的坚持，争取从量变达到质变。

很多亲身实践过的朋友表示，《大猫》最打动他们的有两点：进阶非常合理、每册书很薄很好读，非常适合咱们的孩子利用“碎片”时间实现循序渐进的进步。

**1、****利用碎片时间就能完成的课外阅读，孩子比较容易坚持**

看惯了美国小学生像“砖头”一样的教材，《大猫》这套英国孩子的阅读教材，我拿到手的第一感受就是：**好轻薄，好没有压力，好容易读完！**

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/36864536-C93B-474D-9448-009398E7F183.jpg)

从第3级到第11级，单册书的厚度，逐渐从16页过渡到24页，最多到32页；每册书的学习点，简单而清晰。孩子完全不会有要啃下“大部头”的感觉，心理压力小，轻松读完一本书的“成就感”，反而会很强。兴趣、自信、坚持，就是这样一点点来的。

孩子如果功课紧，每周可以读三次，每次抽出半个小时，一周就能完成1-3册书的阅读。这就是积累的力量。

**2、进阶细致科学，对应小学阶段的英语水平**

《大猫》是英国孩子在学校里使用的 Reading Program，也就是**“教材型”分级读物，**和一般书店里卖的普通分级读物有很大区别。

你会发现，这种教材型分级读物的一大特色，就是**分级特别细致**，整个学习被分为19个级别！分级这么细致的好处，就是进阶坡度特别缓和，孩子“爬上去”比较容易。这也是这类教材型分级读物的好处。

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/D733563A-5D35-431F-A581-499873D97C67.jpg)

根据《大猫》每个级别的核心词汇量，参考国内孩子的普遍英语水平，大致来讲，《大猫》和咱们的学习级别，可以这样对应...

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/55DC0602-0BE3-4343-B906-DCB130C6446D.jpg)

从英语启蒙开始，到能够独立类似初级、中级难度的英文章节小说，我们开团的9个级别，基本上覆盖了整个小学阶段的阅读进阶。

此外，作为**教材型分级读物，**《大猫》系列每册书最后都有“学习指引”，关于这册书的 “学习目标-Learning objectives”，有很清晰的说明。爸爸妈妈可以根据这些学习目标，对孩子的阅读效果做一个简单的检测：比如，重点词汇是否掌握了？文章是否都理解了？阅读方法是否学会了？等等。

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/9BDEAD63-D483-4800-A706-2E2DC6D2222B.jpg)

对怎么带孩子去读、去理解、去分析思考，也有详细的使用指导。老师或家长都可以根据孩子的实际情况，参考这些指示，把抽象的“阅读理解”化为具体的步骤和行动。

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/6F432D81-FE34-4A88-8964-778F92B31F6A.jpg)

**3、涉及知识面特别广，同步接触英美孩子的学科知识**

根据我们接触到的英美两国英语教学情况，**把语言学习和知识学习紧密结合起来，是一个总的趋势。**

《大猫》每册书背后的“阅读指导”里，都会指出这本书的 *Curriculum Links*，对应的是什么学科、什么学习内容。比如，这册书联系到了Social Study中有关“情商教育”的学习内容 ...

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/86927EB1-9C31-40DE-B75C-D355D44036FD.jpg)

根据英国英语教纲，《**大猫》系列里的阅读材料，主题是和孩子的科学、艺术、社会、人文科目学习紧密对应的。**比如2年级的孩子会学科学、艺术、体育等，而这一级的《大猫》读物里就会有对应的科学、艺术、体育等内容。也就是说，每个级别读下来，也就基本接触了英美孩子的主要学科知识。

*3*

跟着《大猫》如何学英语？

分级细致，进阶合理，是《大猫》系列非常核心的特色。具体来说，跟着《大猫》如何提升英语能力？下面来仔细说说...

我们本次开团的级别是**从第3级到第11级**（Band Yellow-Band Lime）：

**起点**是初学英语的孩子，比如小学1、2年级，掌握了一定量的词汇、短语和句型；

**终点**是可以自主阅读多数初级到中级难度的英文原版作品，比如小学5、6年级，具备了坚实的英语阅读和表达基础。

具体来说，一步步，是这么走的 ...

**1、一年级-二年级（3-4级，黄+蓝）**

**黄色（第3级）**

> * 单词量：50-100
> * 文字量：每句话6-10个词，一页2-4行字
> * 词汇：大量高频使用的词汇
> * 句型和语法：句型有一定变化，短语丰富
> * 图文内容：故事有一定情节，图文结合紧密

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/87434246-70F0-427C-9F55-33D731D34AD6.jpg)

《大猫》体系中，初级阶段，大量而有序的输入是关键。给一年级孩子的读本，黄色有29本，蓝色有30本，加起来59本的体量，目标就是让孩子通过阅读，把常见的词汇、句型和基本语法点熟悉起来，打好基础。

虽然有一定的启蒙基础，但这时候孩子的词汇量很有限。所以这个阶段的文本难度并不高，每页两行字左右，每句话10来个词，但句型上会讲求一些变化，让孩子接触到不同的语法使用。

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/13D6D5A8-A11D-4628-AEAF-FB1DE7AAB839.jpg)

每本十几页，篇幅虽短，但故事都有一定的情节，读起来会比较有趣。

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/30763649-7BA7-4E63-A36B-8B8BB274012D.jpg)

**蓝色（第4级）**

> * 单词量：90-170
> * 文字量：每句话8-15个词，3-5行字
> * 词汇：语言更具色彩，会接触到各种专业名词
> * 句型和语法：各种句型都会接触到，语法形式多样
> * 图文内容：多体裁，图片配合故事，但不是一一对应

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/608EE620-5D22-48FD-80F7-019796C1EACA.jpg)

小学低年级孩子就可以开始接触各种“专业词汇”了，为了便于记忆，特别强调词汇的重复。

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/169A6B08-7857-415B-AA9F-38FECC82F8BB.jpg)

孩子会读到各种体裁的内容，比如故事、诗歌、非虚构类短文等等。语言描写更具色彩，加入了更多形容词和副词，读起来生动多了。

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/AEEE4F6C-893E-4E24-9EB8-4CF9FB0C81B3.jpg)

**2、二年级-三年级（第5-6级，绿+橙）**

**绿色（第5级）**

> * 单词量：170-250
> * 文字量：每句话8-15个词，一页3-5行字
> * 词汇：会接触到多个主题的专业词汇和表达方式
> * 句型和语法：句子更长，句型变化更多
> * 图文内容：故事人物增多，图片在一定程度上能对应文字

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/B438ADF6-453C-4C8F-8C23-8DCB7A3808DF.jpg)

给二三年级孩子的读物，主题上的多样性越来越明显。比如，这本说的是雕塑是怎么回事儿？讲解这样的主题，用词用句都会较之前复杂，孩子可以接触到更多专业词汇和表达方式。

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/ED14E809-43F7-4EDE-8E02-2FF0B683B7B6.jpg)

这个阶段读本的表现形式也更加多样，比如用漫画的形式展现电是怎么来了？这样看下来，阅读的难度好像都降低了呢...

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/275607D8-4FC8-4C82-A170-A43D7168EDB1.jpg)

**橙色（第6级）**

> * 单词量：250-300
> * 文字量：每句话8-20个词，一页3-5行字
> * 词汇：会接触到更多专业名词
> * 句型和语法：接触更多复杂的句型，非虚构类文章表达更正式
> * 图文内容：情节增多，体裁增多

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/FFC49077-7954-4ED9-8CF8-3E0C44C2A37E.jpg)

有前面几个级别的积累，孩子已经掌握一定的专业词汇，也接触过不少学科知识了，这个阶段读物有一个明显变化，就是知识类读物的话题深度和难度都有所上升。比如，庞贝古城为什么会毁灭？其中的表达方式更加正式，用词更加专业和准确。

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/2AB77B74-3AD1-4A7D-9DDB-1A86FEA31EB5.jpg)

比起一二年级，这个阶段（第5-6级）每本书的篇幅增长至24页，故事的容量大了，人物和情节更丰富，孩子的阅读持久力也逐渐增强。

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/43231C39-4517-47B1-924B-A9437738C7C6.jpg)

**3、三年级-四年级（蓝绿+紫+金）**

**蓝绿色（第7级）**

> * 单词量：300-500
> * 文字量：每句话10-25个词，一页3-6行字
> * 词汇：词汇更难
> * 句型和语法：接触更多复杂的长句
> * 图文内容：体裁增多，图片的信息量减少，学着独立理解文字

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/9B4BC972-2105-4725-B5FA-56B428D58213.jpg)

进入小学中年级阶段，孩子开始慢慢练习减低阅读中对图片的依赖。故事中图片的信息量逐渐减少，强调对文字的理解能力。比如，从读一篇校园生活故事开始试试看...

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/28040CBD-D866-4791-9A37-0405207215EA.jpg)

科普读物中，开始接触到生活中比较少见的词汇，图片和生词的对应关系比较明显，帮助孩子理解。

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/3DCCBFA0-A236-4F49-BC23-9432659B082E.jpg)

**紫色（第8级）**

> * 单词量：500-700
> * 文字量：每句话10-25个词，一页3-7行字
> * 词汇：词汇更长、更难
> * 句型和语法：句子更长，句型更复杂
> * 图文内容：开始阅读简单的章节书，非虚构类文章体裁更多

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/922C7022-0F5C-4A6F-B3CA-877213E88B80.jpg)

这个阶段的故事开始分章节讲述，也就是说，孩子正式进入“章节书”的阅读阶段。对人物性格、心理的描写更加细腻和生动，故事情节和线索更加复杂，开始有点读“短篇小说”的感觉了...

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/353D37D1-A039-41BC-9E08-818FF3A60EDF.jpg)

科普读物的体裁更广泛，比如说明文、分析报告、信息类文章等等，话题与当代社会的结合非常紧密。

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/7EFBF039-A5FE-4BF5-9922-FCFF5A958262.jpg)

**金色（第9级）**

> * 单词量：700-900
> * 文字量：每句话10-25个词，一页3-10行字
> * 词汇：词汇更长、更难
> * 句型和语法：句子更长，句型更复杂
> * 图文内容：继续阅读简单的章节书，非虚构类文章体裁更多

![](.evernote-content/EE76517F-D8ED-45AC-A52B-DF3077C67B5B/6BA96627-8C14-4DF2-BE57-5924B4946B30.jpg)
