# 2026年5月月度工作小结——欧洲事业部 肖鸣

> 华程国旅集团 2026年5月工作小结，欧洲事业部肖鸣。

## 核心内容

### 业务数据（纵横定制）
| 指标 | 数值 |
|------|:----:|
| 本月毛利 | 21 万元 |
| 本月营收 | 180 万元 |
| 本月人数 | 74 人 |
| 毛利率 | 11.7% |
| 亮点团 | 5月7日意大利+希腊 17人团，利润13万元 |

### 导游管理工作（6项）
1. **导游基金**：开支项目分析和列举
2. **报账流程**：规范和细化
3. **导游大赛**：初赛启动
4. **NPS 跟进**：持续关注
5. **商户协同**：与商户组同步年度目标，划分分公司指标
6. **培训计划**：课题已定，导师筛选中

### 其他项目
- **品味项目**：已上线，全国推广，第一波产品出炉
- **导游行中社交价值提升项目**：两个方向已确定，内容设计中
- **品梦项目**：基本方向已定——DMC全直采、酒店品牌筛选、特色筛选、周边环境规定、餐标准、增值服务等

### 下月计划
- 重点推进品梦项目和导游项目的推进
- 持续跟进 NPS 相关工作

## 关键信息
- 欧洲事业部定制业务5月营收180万、毛利21万
- 三大项目并行：品味（已上线）、品梦（方向确定）、导游社交（设计中）
- 意大利+希腊单团贡献13万利润（占毛利62%），显示高端定制利润空间大
