用户在浏览器地址栏输入访问的网址域名时，需要转换成 IP 地址才能让计算机理解要找到哪个服务器，而 DNS 服务器的作用就是将网址转换为 IP 地址。在查询 DNS 服务器时，可能需要消耗一定的时间，或者由于某些原因遭到 DNS 污染或者劫持，导致网络访问出现问题。
