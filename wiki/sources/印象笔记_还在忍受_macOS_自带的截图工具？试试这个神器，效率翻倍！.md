还在用 macOS 自带的截图工具？
------------------

**直到我换了这个工具，我才意识到自己浪费了多少时间**

你不会意识到 macOS 自带的截图工具有多糟糕，直到你尝试做一点点额外的操作。

我以前也是这样。

![](.evernote-content/1F012967-29D1-405E-9D34-29C5951A69ED/08DBC4F8-CAB0-424D-AFF9-88BCAEDB0110.png)

我长期使用 Mac 自带的快捷键 `⌘+Shift+5` 来截图，以为已经够用了。每次想要隐藏截图中的某些内容，我都会依赖预览（Preview）或照片（Photos）应用。甚至当我需要从图片中提取文字时，我会手动打出来。虽然能用，但实在是太麻烦了。

然后，我尝试了 **CleanShot X**。

一开始，我只是用它来做一些基础操作，比如给图片加更好的标注，或者截取更清晰、更专业的截图。但很快，我发现自己已经离不开它了。

现在，它已经成为我日常工作流程的核心部分，无论是写作、管理产品文档，还是只是随手给别人发个截图，都变得非常高效。

即使你不像我一样每周截上百张图（是的，我是个疯子），这个应用也能帮你节省比你想象中更多的时间。

**读完这篇文章，你会学会如何使用 CleanShot X 来提升工作效率，减少不必要的步骤，并将一个简单的截图工具变成你 Mac 上最实用的应用之一。**

**CleanShot X 是 Mac 上最好的截图和屏幕录制工具，内置标注功能和云端上传功能……**

我每天用 CleanShot X 做内容创作的 3 大功能
-----------------------------

最开始，我只是用 CleanShot X 来截普通的截图，因为我已经习惯了那种操作方式。但很快，它就成为了我写博客、做演示，甚至只是向别人演示如何操作时的第一选择。

以下三个功能，几乎承包了我内容创作的大部分工作。

### 1. 录制 GIF 并即时分享

有时候，**截图无法准确表达你的意思，而录制完整视频又显得过于繁琐**。这时候，GIF 就是完美的解决方案。我经常用它来展示应用演示、UI 操作流程，以及各种 macOS 小技巧。

CleanShot 允许我用快捷键快速启动 GIF 录制，选定需要的区域，并在几秒内完成录制。**一旦停止录制，屏幕左下角就会弹出一个窗口，里面有可复制的链接，方便直接分享给别人。**

**![](.evernote-content/1F012967-29D1-405E-9D34-29C5951A69ED/4EC37EE9-3BEA-41D3-80C7-47AD80164243.png)**

现在，我不再需要写长篇大论或打字解释，而是直接甩个 GIF，省时又高效。这一个小小的改动，就让我避免了重复解释相同问题的时间浪费。

### 2. 让截图更精致的「边框」功能

如果你用过 macOS 的自带截图工具，你会发现**很多截图看起来都很「平淡」**，尤其是网页截图或者聊天记录截图，直接放到文章或演示里会很突兀。

CleanShot X 的「边框」功能，**可以一键给截图添加漂亮的背景，使其更具层次感、更专业、更容易阅读**。

![](.evernote-content/1F012967-29D1-405E-9D34-29C5951A69ED/D738FA0A-9429-4EB5-9A52-ACD22414CE5E.png)

这听起来可能是个小功能，但如果你经常需要在博客、PPT 或产品文档里插入截图，你会发现它的价值。我以前会花时间在 Canva 里调整截图，现在**只需一键**，省去了大量调整和设计的时间。

### 3. 一秒隐藏敏感信息

在用 macOS 自带工具隐藏敏感信息时，我以前的做法是：

1. 打开预览（Preview）
2. 添加一个矩形形状
3. 手动调整透明度

虽然能用，但又慢又不直观。

![](.evernote-content/1F012967-29D1-405E-9D34-29C5951A69ED/AB81C6E4-93C4-4EAD-AC42-2942CBFC5AC5.png)

现在，我只需要在 CleanShot 里**直接拖拽要隐藏的内容**，就能用「模糊」或「像素化」方式快速遮挡。

* **模糊（Blur）** 适用于遮挡邮件、用户名等信息
* **像素化（Redact）** 用于彻底隐藏密码或机密信息

对于像我这样经常在博客和新闻通讯里分享截图的人来说，这个功能实在是太方便了。

CleanShot X 如何完美融入我的 macOS 工作流
------------------------------

CleanShot X 不只是我 Mac 里的一款普通应用，而是彻底改变了我的截图体验。

### **1. 自定义快捷键，加快截图速度**

安装 CleanShot X 后，我做的第一件事就是\*\*把默认的 `⌘+Shift+5` 改成了 `⌘+Shift+S`，这样无论在哪台 Mac 上，我的肌肉记忆都能自动生效。

![](.evernote-content/1F012967-29D1-405E-9D34-29C5951A69ED/D399A0A6-8E72-4373-BB3E-ABD7DC389930.png)

这意味着：**不再需要手动打开应用，也不用找菜单，直接一键截图**。

### **2. 所有截图自动存入专属文件夹**

macOS 默认把截图存到桌面上，时间久了就会变得一团糟。**我设定了 CleanShot X 把所有截图自动存入「Screenshots」文件夹**，这样桌面始终保持整洁。

![](.evernote-content/1F012967-29D1-405E-9D34-29C5951A69ED/AB2F3E6D-DBA1-4188-A9EF-8673D56781F7.png)

同时，我还设置了**截图自动按照时间戳命名**，这样在查找旧截图时更方便。

### **3. 与 Paste 应用完美配合**

我经常**连续截图**，以前需要不断去 Finder 找文件，现在直接用 **Paste**，所有截图都会自动进入剪贴板历史，**无需手动整理**。

![](.evernote-content/1F012967-29D1-405E-9D34-29C5951A69ED/3AF02DEA-8C08-4354-A0E3-6AE8F24594D2.png)

但有时候，我不想让所有截图都混在 Paste 里，而是想单独管理截图历史。这时候，我就用 **CleanShot 自带的截图历史功能**，能按类型（截图、GIF、录屏）筛选，还能快速删除不需要的内容。

![](.evernote-content/1F012967-29D1-405E-9D34-29C5951A69ED/95A8E303-E001-4B52-9437-EAB1E8497D7A.png)

CleanShot X 解决了 macOS 自带截图工具的痛点
-------------------------------

多年来，我一直忍受 macOS 自带截图工具的各种限制。比如：

* **滚动截图**：macOS 不能截取长网页，需要拼接多个截图

![](.evernote-content/1F012967-29D1-405E-9D34-29C5951A69ED/28843A0E-9CEB-4D99-A23A-EB7A0F79B032.png)

* **OCR 文字提取**：想复制图片中的文字？只能手动打出来

![](.evernote-content/1F012967-29D1-405E-9D34-29C5951A69ED/3BB7CC9F-9232-4369-B056-06B6D9FBC779.png)

* **屏幕录制**：macOS 自带的录屏没有鼠标点击效果，看不清操作步骤

![](.evernote-content/1F012967-29D1-405E-9D34-29C5951A69ED/1473BE6F-9C89-4F5F-A9A4-E9BD917CB3D0.png)

CleanShot 直接解决了这些问题。

### **1. 一键滚动截图**

以前，我截图网页或聊天记录时，要拼接多张图片，现在只需拖动滚动截图功能，一次性搞定。

### **2. OCR 文字识别**

如果你曾尝试从图片中复制文字，你就会明白手动输入有多麻烦。

**CleanShot X 的 OCR 功能** 可以直接**选取图片上的文字并复制**，无需额外工具。

我主要用它来：

* 复制视频中的文字
* 提取截图里的错误信息
* 从 PDF 或 UI 界面中获取不可选的文本

总结
--

如果你还在用 macOS 自带的截图工具，你可能正在浪费时间。CleanShot X 让截图、标注、录制、分享变得无比简单，彻底改变了我的工作方式。

现在，你只需要安装 CleanShot X，就能像我一样，**用更少的时间，完成更多的事情**。本期的分享到这告一段落了，下期我们再见！！！
