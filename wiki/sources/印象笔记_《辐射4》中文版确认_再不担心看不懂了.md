# 《辐射4》中文版确认 再不担心看不懂了

---

[![](.evernote-content/4A4EF892-7CD0-4389-B850-49625FD6140D/7404F46E-219F-4714-93C2-6144EB3575D7.jpg)](http://tech.feng.com/2015-10-09/The-radiation-4-Chinese-version-confirmation-Don-t-worry-about-don-t-understand_626663.shtml)

　继《合金装备 5：幻痛》后，玩家们接下来的期待点估计就要数《辐射 4》了。看不懂怎么办？没问题，这次我们有官方中文版了。

![](.evernote-content/4A4EF892-7CD0-4389-B850-49625FD6140D/7404F46E-219F-4714-93C2-6144EB3575D7.jpg)

Bethesda 不久前正式在官方的 FaQ 里确认，《辐射 4》在亚洲地区有繁体中文版。虽然早前已有迹象暗示《辐射 4》繁体中文版的存在，但看到官方的正式确认还是让人更放心。

繁中版《辐射 4》采用的是中文文本+英文配音，同地区的日本本地化程度更高，为日文文本+日语配音。不过，中文区的玩家一般来说更喜欢听原配，所以这不是问题。

除了中文版确认外，《辐射 4》PC 版的配置要求也终于被公开。本作最低要求英特尔 Core i5-2300 2.8 GHz 或 AMD Phenom II X4 945 3.0 GHz，8GB 内存，NVIDIA GTX 550 Ti 2GB 或 AMD Radeon HD 7870 2GB。

推荐配置方面，则是英特尔 Core i7 4790 3.6 GHz 或 AMD FX-9590 4.7 GHz，8GB 内存，NVIDIA GTX 780 3GB 或 AMD Radeon R9 290X 4GB。硬盘空间要求 30GB。

《辐射》系列地图广阔，支线故事众多，文本量自然也大得可怕。《辐射 4》整个开发历时五年，开发商野心勃勃，可以想见文字量比起前作只多不少。现在，国内玩家终于能毫无障碍地去玩到神作了。
