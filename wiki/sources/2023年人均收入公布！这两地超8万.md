# 2023年人均收入公布！这两地超8万

> 全国居民人均可支配收入39,218元，同比+6.3%。上海、北京首次突破8万元。

## 全国数据

| 指标 | 数值 |
|------|:----:|
| 全国人均可支配收入 | **39,218元**（+6.3%名义 / +6.1%实际） |
| 人均工资性收入 | **22,053元**（+7.1%，占56.2%） |
| 国内生产总值 | 126.06万亿元（+5.2%） |
| 人均消费支出（上海） | 52,508元（全国第一） |

## 省份收入梯队

| 梯队 | 省份 | 关键数据 |
|:----|------|---------|
| 8万+ | 上海、北京 | 首次突破8万元 |
| 6万+ | 浙江 | 63,830元，唯一6万+ |
| 5万+ | 江苏、天津 | 新晋5万元俱乐部 |
| 4万+ | 广东、福建 | 广东距5万仅一步之遥 |
| 3万+ | 山东、内蒙古等14省 | 含新晋宁夏、山西 |
| 2万+ | 河南、吉林等10省 | — |

## 增速

所有省份均正增长。**西藏**以+8.65%领跑；甘肃、海南、新疆等11地增速跑赢全国均值（6.3%）。

| 省份 | 人均消费支出 |
|:----|:-----------:|
| 上海 | 52,508元 |
| 北京 | 47,586元 |
| 浙江 | 42,194元 |
| 江苏/天津/广东/福建 | 3万+梯队 |
