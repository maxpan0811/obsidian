# ProcessOn思维导图Skill上线：一句话生成专业脑图！附详细操作教程～

![](.evernote-content/FD4B2750-56F0-4EDA-AB84-40CF283847F3/BBD83FC9-409C-4CE3-BC5C-81CB91E8C526.png)

很多人每天都在画思维导图，记录会议、整理读书笔记、拆解项目方案、规划学习路径……但画完之后，往往还要继续搜索资料、分析问题、制定计划、推进执行。

于是一个问题出现了：如果思维导图不仅能帮你整理思路，还能直接帮你完成思考和执行，会怎么样？

最近，ProcessOn上线了全新的思维导图 Skill 功能，通过接入不同的AI能力，导图中的每一个节点都可以成为任务入口，自动生成内容、分析问题、制定方案，甚至帮助推进后续工作。思维导图直接从思考工具升级为了行动工具，也让知识管理真正进入了AI协作时代。

新功能到底如何用呢？接下来附上完整操作教程～

01

ProcessOn思维导图Skill能做什么？

ProcessOn思维导图Skill（ProcessOn Mindmap Generator）是 ProcessOn官方推出的AI思维导图生成能力。

你只需要提供文章、文档、网页链接、会议纪要、学习资料，甚至图片中的文字内容，它就能自动完成阅读、理解、提炼、归纳和分类，并生成一张结构清晰、层级分明、可持续编辑的专业思维导图。

简单来说：输入一段内容，AI 帮你生成一张高质量思维导图。

它特别适用于以下场景：

• 文章总结与内容提炼

• 文档拆解与重点梳理

• 读书笔记整理

• 学习路径规划

• 会议纪要结构化整理

• 工作报告总结

• 项目任务拆解

• 问题分析与根因定位

• 组织架构梳理

• 知识体系搭建

• 论文与文献阅读整理

• 头脑风暴成果归纳

生成完成后，系统会返回：

✅ 思维导图预览图——快速查看生成效果；

✅ ProcessOn可编辑链接——一键打开并继续编辑、分享或导出。

需要注意的是，思维导图Skill专注于知识梳理、信息归纳和结构化表达，更适合生成各类思维导图和知识脑图。

如果需要生成流程图、泳道图、时序图、系统架构图、ER 图、Mermaid 图等专业图表，建议使用ProcessOn Diagram Generator，之前公众号也分享过：OpenClaw与ProcessOn Skill深度整合的完整实践〉〉

02

思维导图Skill详细使用流程

整个操作过程并不复杂，即使是第一次接触AI Skill的用户，也可以快速完成从内容输入到思维导图生成的全过程。

Step1：选择支持Skill的AI工具

ProcessOn思维导图Skill并不是独立运行的软件，而是一项可以被AI调用的专业能力。因此，在使用之前，需要先准备一个支持Skill的AI工具。

目前常见的选择包括：

• Claw 类工具：比如OpenClaw、QClaw、WorkBuddy、JVSClaw等，面向普通用户的 AI Agent 工具，通过聊天即可安装和调用各种Skill；

• 开发者 Agent 工具：Codex、Claude Code、Hermes Agent，这类工具通常可以在项目目录、技能目录或工作流中使用 Skill，适合批量处理文件、整理资料、生成结构化内容。

• AI 智能体搭建平台：比如Coze、其他支持插件、工具或Skill能力的智能体平台，可以根据实际情况使用。

完成安装后，只需输入文章、网页、文档、会议纪要等内容，AI 就会自动分析信息并生成对应的思维导图。

Step2：安装ProcessOn思维导图Skill

安装好支持Skill的AI工具后，就可以安装ProcessOn思维导图Skill了，常见安装方式有以下几种：

方法1：通过对话指令安装

这是最适合小白的方式，你只需要把安装指令复制到 AI 工具的对话框里，让 AI 帮你完成安装。

![](.evernote-content/FD4B2750-56F0-4EDA-AB84-40CF283847F3/BF58090E-3CFA-40C4-BA60-81C232AD40FF.png)

操作步骤
----

1. 打开你的 AI 工具；

2. 进入 AI 助手聊天界面；

3. 复制安装指令；

4. 粘贴到输入框；

5. 点击发送；

6. 等待提示安装成功。

以下安装指令任选其一即可

安装指令模板1:
--------

请使用下面的命令安装 ProcessOn 思维导图 Skill：openclaw skills install processon-mindmap-generator

安装指令模板2:
--------

请先检查是否已安装 SkillHub 商店，若未安装，请根据 https://skillhub.cn/install/skillhub.md 安装SkillHub商店，但是只安装CLI，然后安装processon-mindmap-generator技能。若已安装，则直接安装processon-mindmap-generator技能。

安装指令模板3:
--------

请使用下面的命令安装 ProcessOn 思维导图 Skill：npx skills add https://github.com/processonai/processon-skills.git --skill processon-mindmap-generator --yes

方法2：到skillhub或clawhub技能平台，搜索ProcessOn思维导图技能，探索更多安装方式

Step3：输入内容和提示词

安装完成后，就可以正式使用 ProcessOn思维导图 Skill 生成脑图，可以参考下方提示词模版。

1. 最简单的提示词
----------

请使用 ProcessOn 思维导图 Skill，帮我把下面内容整理成结构清晰、层级分明、配色美观的思维导图，并返回预览图和 ProcessOn 可编辑链接，方便我登录后继续编辑。

内容如下：请输入你的内容···

2. 文章总结类提示词：适合文章、公众号、报告、长文本

请使用 ProcessOn 思维导图 Skill，帮我提炼下面这篇文章的核心内容，拆解文章结构，生成一份完整思维导图。

要求：

1. 保留文章主要观点和逻辑层级；

2. 每个一级节点概括一个核心主题；

3. 二级、三级节点提炼关键内容；

4. 语言简洁，不要大段复制原文；

5. 配色美观，结构清晰；

6. 输出预览图和ProcessOn可编辑链接。

文章内容如下：请输入你的内容···

3. 文档拆解类提示词：适合工作方案、项目文档、制度文件、汇报材料
---------------------------------

请使用ProcessOn思维导图Skill，帮我快速阅读并拆解这份文档，提炼关键信息，生成一份方便理解和推进工作的思维导图。

要求：

1. 按照文档原有结构进行拆解；

2. 提炼核心目标、关键内容、执行步骤、注意事项；

3. 层级清晰，重点突出；

4. 不要遗漏重要信息；

5. 生成 ProcessOn 可编辑链接，方便我后续修改。

文档内容如下：请输入你的内容···

4. 学习路径类提示词：适合某个新领域、新技能、新课程的学习路线
--------------------------------

请使用 ProcessOn 思维导图 Skill，帮我生成一份【重点填写｜学习主题】学习路径思维导图。

要求：

1. 从入门到进阶分阶段梳理；

2. 每个阶段列出学习目标、核心知识点、练习方法；

3. 适合零基础用户理解；

4. 结构清晰，层级完整；

5. 配色美观；

6. 输出预览图和 ProcessOn 可编辑链接。

学习主题是：请输入你的内容···

5. 问题分析类提示词：适合项目复盘、产品问题、运营问题、故障排查
---------------------------------

请使用 ProcessOn 思维导图 Skill，帮我分析【重点填写｜问题名称】的根本原因，并生成一份结构清晰的思维导图或鱼骨图。

要求：

1. 从多个维度分析问题原因；

2. 每个维度继续拆解二级、三级原因；

3. 不只停留在表面原因，要尽量挖掘深层原因；

4. 最后补充可执行的优化建议；

5. 输出预览图和 ProcessOn 可编辑链接。

需要分析的问题是：请输入你的内容···

Step4：获取预览图和编辑链接

生成完成后，AI 通常会返回一张脑图预览图和一个ProcessOn 可编辑链接，点击链接登录ProcessOn后，即可继续编辑、分享或导出。

💡 使用建议
------

• 优先通过官方渠道安装 ProcessOn Skill；

• 避免上传涉密或敏感文件；

• 执行文件修改类操作前，建议先确认内容；

• 首次体验可优先在测试环境中使用。

养成良好的使用习惯，既能保障数据安全，也能获得更稳定、高效的AI体验。

操作案例分享

接下来结合几个场景帮大家分享一些实用案例，给大家做一下展示。

1、梳理读书笔记

读书、整理文档或梳理图片资料时，手动拆解内容往往费时费力，零散信息也难以快速形成体系。ProcessOn AI 支持上传书籍、文档和图片，自动按内容结构进行拆解、归纳与总结，帮助大家快速梳理资料脉络，高效提炼核心信息。

提示语：

帮我提炼这本书的核心内容，拆解逻辑结构，生成完整思维导图，配色美观、结构清晰。

2、搭建学习路线

学习新知识时，很多人都会遇到同样的问题：不知道学什么、先学什么、如何系统学习。ProcessOn AI 只需一句自然语言指令，即可自动生成结构化、分阶段的学习路径思维导图，帮助你快速搭建知识框架，明确学习方向。

提示语：

3、问题分析

项目复盘、故障排查时，很多团队的问题分析往往停留在表面，难以找到真正原因。ProcessOn AI 支持从多个维度逐层拆解问题，一键生成思维导图或鱼骨图，帮助团队快速定位根因，制定优化方案。

提示语：

帮我分析社交APP用户流失的问题，可以从产品功能、用户体验、技术性能、运营策略、竞品市场五维度逐层解析，生成美观、结构适配的思维导图。

搭建组织架构时，很多团队会面临权责不清、汇报关系混乱、调整维护成本高等问题。ProcessOn AI 只需一句自然语言指令，即可自动生成层级清晰的组织结构图，明确部门职责与汇报关系，并支持在线编辑，轻松完成后续调整。

提示语：

帮我生成一个AI创业公司组织结构图，包含决策层、技术研发部、商业化部、人力行政部、财务部、安全合规部六大核心部门，明确层级与汇报关系。
