# 微星Aegis/Nightblade/Trident游戏主机新品

---

紧随英特尔更新七代酷睿处理器的步伐，微星（MSI）也推出了 Aegis、Nightblade、Trident 台式游戏主机新品。  
  
　首先介绍下配置升级后的 Aegis 3，其迎来了酷睿 i7-7700 处理器、英特尔 B250 芯片组主板、可选 Nvidia GeForce GTX 1060 / 1070 独显；追求发烧性能的玩家还可以选择非锁频的酷睿 i7-7700k 处理器，配套的 Z270 芯片组主板，甚至 GeForce GTX 1080 独显。

![](.evernote-content/BE7C060D-05BB-4415-AA09-241F42220D89/E9C28E59-D1C0-4B3A-BD5C-42D26288A523.jpg)  
  
Aegis 3（via：MSI）

Aegis Ti3 是微型最强性能的台式游戏主机，其配备了英特尔酷睿 i7-7700k 处理器、最高 64GB DDR4 2400 MT/s 内存、可选 GTX 1070 / 1080 双卡交火。

![](.evernote-content/BE7C060D-05BB-4415-AA09-241F42220D89/AD24F40D-265D-4575-BBCC-27A98694254C.jpg)  
  
Nightblade 3 / MI3 / Nightblade MIB

Nightblade 3 机箱稍小一些，但性能没有妥协。其可选酷睿 i7-7700 处理器、GTX 1070 独显，前板上还有个方便搬动的提手。

Nightblade MI3 和 Nightblade MIB 体型更小、性能也有所下调，消费者可选酷睿 i5-7400 处理器 和 GTX 1060 独显。

![](.evernote-content/BE7C060D-05BB-4415-AA09-241F42220D89/8050E135-913B-4B6D-B869-AFB5E473F6AB.jpg)  
  
Trident 3

最后介绍的是 Trident 3，它的体型与一台家用游戏主机相当，仅 14×10×3.8 英寸（35.6×25.4×9.7 CM）。用户可选酷睿 i7-7700 处理器、最高 32GB DDR4 2400 内存、以及 GTX 1060 独显。

微星表示，上述机型都将于本月在全球开售，价格依配置而不尽相同，感兴趣的网友可以洽询当地经销商。
