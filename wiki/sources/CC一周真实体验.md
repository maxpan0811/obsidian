## 一句话摘要

一周深度使用 Claude Code 的真实体验——最震撼的是项目理解能力（接手老项目几分钟理清认证链路），改 bug 的排查思路比很多初中级工程师靠谱，重构能自动追踪所有调用方一次性改完。

## 三大亮点

1. **项目理解能力**：无文档的老项目，说「帮我理清认证流程」，自己去翻十几个文件，画完整调用链路（自己干要半天，它几分钟）
2. **改 bug 思路**：偶现并发问题，先去读锁机制代码 → 看测试用例 → 定位到事务隔离级别——排查思路比初中级工程师靠谱
3. **重构能力**：自动追踪所有调用方，一次性改完
