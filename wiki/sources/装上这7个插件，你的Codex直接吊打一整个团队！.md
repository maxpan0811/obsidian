# 装上这7个插件，你的Codex直接吊打一整个团队！

你好，我是元小二，专注分享 AI 提效、一人公司实践和个人成长。这里有 OpenClaw、Claude Code、自动化流程、虚拟产品，也有理财、思考和生活系统。

朋友们，这是一个喜大普奔的好消息：
我作为一个 Codex 的元老级用户，今天要把我压箱底的插件清单一次性全掏出来，狠狠地分享给你们。

...不夸张地说，自从我把这套插件配齐之后，我对市面上那些花里胡哨的工具的评价是：可以集体退休了。
废话不多说，直接上干货。我的朋友，准备好你的小本本。
一、神级功能，独一份的存在1. Computer Use：Codex 独一家，给我夯爆了这玩意儿是真的绝了。
直接让 Codex 操控你的电脑，你只要口头描述一下，剩下的它自己搞定。我之前为了自动化一个流程，请人写脚本弄了快两周，各种报错，最后还是没跑通（用rpa的话，元素会经常变）。
换成 Computer Use 之后，我嘴一张，它手一动，一次性跑通。
我对它的评价是：这是目前 Codex 最值得吹的功能，没有之一。十多种玩法我专门写过一篇，建议你去翻一翻，看完保证你也想原地起飞。
2. Playwright：Computer Use 搞不定的，它来兜底有些网页 Computer Use 操控不了，比如电商后台、公众号后台这种刁钻的场景。
别慌，Playwright 直接给你开一个新浏览器，模拟真人操作，丝滑得很。
我的建议是：Computer Use 当主力，Playwright 当备胎。两个一起上，几乎没有它俩搞不定的网页。
二、打通工作流，让 Codex 帮你卷起来1. Gmail：给自己配一个全自动邮箱秘书装上这个，你就能让 Codex 定时给你发邮箱日报，每天早上打开邮箱，信息已经帮你整理好了。
更骚的操作是：给开源仓库做贡献的时候，让 Codex 定时去查管理者的修改意见，然后自己继续改代码。
等等党大胜利，躺着就把活干了。
2. Github：AI 时代人人必装，不解释这个好处真不用我多说了。Codex 连上 Github 仓库，就是标配，就是基本盘。
你要是还没装，我只能说：朋友，你已经落后了。
3. Google Drive：内容创作和资料归档的命根子安装了谷歌云盘插件，就跟装了个飞书 CLI 差不多。
你存在云盘里的谷歌文档、各种资料，Codex 都能随手读取，方便到离谱。做内容、搞归档的朋友，这个必装。
三、进阶玩法，让你的产出直接上一个台阶1. Remotion / Hyperframes：给视频加特效，动感拉满做动画视频特别方便。
尤其适合给你已经录好的口播视频加特效。最近抖音上特别火的柱子哥，你仔细看，他的视频特效全是 Remotion 做的，那个动感，一看就高级。如果你对做视频感兴趣，可以看看这篇：视频剪辑门槛打下来了！装上这6个 Codex Skills，你一句话就能出片！
2. Vercel：一键部署，比 Cloudflare 还省心Codex 不像 Google AI Studio、扣子那些自带部署功能。
你做好一个网站想分享给朋友或客户，得先部署上线。这时候免费扔到 Vercel 上，是真香，比 Cloudflare 还方便。
