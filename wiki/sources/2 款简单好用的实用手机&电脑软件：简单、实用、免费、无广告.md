▮**MP3转换器**

MP3 转换器是来自 App Store 的一款专业、免费、无广告的多功能音频格式转换器，目前仅支持 iOS、iPad OS、macOS 等苹果操作系统。

![](.evernote-content/73DCE21E-81F7-40DD-B059-E795B0D24C58/FD00EC6E-99DD-4813-A465-6FFCEEBCE972.jpg)

软件界面非常简洁，但功能丰富且强大。它支持从本地、iCloud 导入音频，或从电脑导入音频、导入自己现场录制的录音。除此之外，它还支持从视频中提取音频进行导入。

![](.evernote-content/73DCE21E-81F7-40DD-B059-E795B0D24C58/2ED1ADCC-61EC-4216-B045-9E59FEE2F843.jpg)

它支持对 aac、m4a、mp3、wav、amr、ac3、wma、flac、ogg、caf、opus 等常见格式的音频文件进行格式转换、裁剪、拼接、调速、降噪、编辑元数据等操作，并且支持无损处理音频文件（转换/编辑前后音质保持不变）。

![](.evernote-content/73DCE21E-81F7-40DD-B059-E795B0D24C58/47A26659-0190-4F4B-8DAC-E107885CD5FD.jpg)

音频文件处理完成后，进入【我的音乐】页面可以查看处理好的音频。点击文件右侧的三个点，点击【分享】可以将处理好的文件保存到本地。

![](.evernote-content/73DCE21E-81F7-40DD-B059-E795B0D24C58/B085C4D3-F2E1-4369-B980-C2D56008C0AC.jpg)

▮**Markdown.UWP 编辑器**

Markdown.UWP 编辑器，顾名思义，是一款简单、免费、无广告的 Markdown 编辑器。它支持自动检测文件编码，支持自动保存内容，目前仅支持 Windows 平台。

![](.evernote-content/73DCE21E-81F7-40DD-B059-E795B0D24C58/E6578482-CA67-4D45-9D5F-49B22C2ABA10.png)

相比其他同类软件，它显得尤为简单：没有复杂的设置、没有大量快捷键，全部输入都依赖纯粹的 Markdown 语法和符号。

点击【实时预览】软件窗口会被分成两个区域。左侧区域用于编辑 Markdown，支持 Markdown 语法高亮显示。右侧区域用于实时预览 Markdown 语法，渲染速度很快。

![](.evernote-content/73DCE21E-81F7-40DD-B059-E795B0D24C58/9EB72A6A-4B1B-4D33-B246-5ED179DFD867.png)

点击【设置】，可以切换主题颜色、软件字体：
