王宝强、马蓉、宋喆。

这个周末的夜晚，因为王宝强的一个离婚声明，整个微博、微信都疯了。

三人成虎，网络一边倒的口诛笔伐：

“马蓉该死，谁让她出轨，臭婊子”

“宋喆该杀，拿着宝宝的钱，睡宝宝的老婆”

网络上充斥着这样的戾气，网友们冲到他们的微信下面铺天盖地、口无遮拦的骂，每个人都站在道德的制高点上，挥舞着道德的大棒，每个人都是正义的化身。

▼

**马蓉有没有错**，显然，她一定错了，错在没有离婚就再去寻找新的选择。

出轨在我们国家是一个千夫所指的词汇。马蓉嫁了一个明星，享受更多的社会资源，因此，她的错误被放大了无数倍，成了我们所有人茶余饭后的谈资。这好像是很理所当然的事。

历史总是惊人的相似，几年前，有同样的一幕在上演：女方出轨，男明星被绿，男三从此结束职业生涯，天翻地覆，无人不知，无人不晓……

是的，美美说的是陈冠希始作俑者的艳照门。

当时，谢霆锋对记者说：“**我希望这件事能早点过去，大家都不要再提了。我在跟我太太结婚之前，就很清楚她过去的事，所以我们现在还会一起出去逛街、看电影、吃饭。我们的生活和以前并没有什么不同，不一样的是，我们开始更加懂得，有些事需要我们去珍惜。**”

风波过去后，又有一段时日，谢张离婚，张柏芝独自带大两个孩子。谢霆锋究竟是不是因为艳照门和张柏芝离婚，我们都不得知，但是，现在回头看，谢霆锋在张柏芝最危险的时候保护了自己的太太，保护了自己孩子的母亲，用最隐忍最绅士的方式处理这一风波，从那时候开始，我觉得谢霆锋真的很绅士风范和优雅成熟。

即便后来离婚，他们离婚的声明是这样的：

|  |
| --- |
| 本人张柏芝、谢霆锋正式宣布我们因性格不合而共同决定离婚。有关离婚事宜，我们已和平地达成协定。    我们将共同管养我们的两个孩子，一切皆会以两小孩的幸福成长为主要考虑因素。（部分） |
|
|
|
|

傻根王宝强的离婚声明，充满了快意恩仇的基督山伯爵味道，所有的错误都是这对狗男女造成的，我王宝强一点没有错，剑剑封喉，置马蓉——他的前妻，他孩子的母亲于死路，不给她留一丝情面和活口，字里行间看的我心惊胆颤，**光天化日，活活吊打！**

即便她有错，即使她出轨不值得同情和原谅，但**王宝强，你真的太不绅士了**。或许你会聚的美美是岁月静好的圣母婊，事情不发生在你的身上，你当然可以云淡风轻，但身为一个女人，我最真实的感受是，王宝强这样的男人真的太过决绝。**他完全有能力私下解决，但是他却选择了将一个女性放在高台上被众人谴责和谩骂。**

或许有人会告诉我，上面这些感情都不如王宝强的来的狗血，扪心自问，我们真的知道真相么？我们看到的只是他们想给我们看到的，背后的伤口、屈辱、辛酸，或许我们都不知，但这些明星最少给我们看到的方式都是成熟的爱，成熟的分手，很绅士。

明星是掌握社会资源的，是粉丝们模仿的对方，他们有更多的话语权。如果每个人都和王宝强一样不留余地，铩羽而归，置之死地，我想象不到，这个社会究竟会戾气成怎么样？

▼

明星是个好光环，有钱有名有人爱。

**可我觉得，明星是个放大镜**，你身上有一丝一毫的缺点，下面都会有千千万万双眼睛。宝强今天这篇声明，是直接置曾经亲密爱人于死地了，是动了真爱、真怒才能如此决绝吗？我不懂。是一定要让自己曾经爱过的人，千夫所指才痛快吗？我也不懂。

但这里有份**财产变更记录**，我想，大家看看，也许，懂。

|  |
| --- |
| 北京宝亿嵘影业有限公司成立于2010年8月，该公司的法人是马蓉，出资人分别为王宝强妻子马蓉和王宝强现任经纪人、宝亿嵘影业CEO任晓妍。    据悉，当初马蓉是出资了1500万，任晓妍出资了500万。          **不过，在今年3月和4月，该公司发生了两次股东变更****：**    **3月25日**，公司股东由马蓉、任晓妍变更为王宝强、任晓妍、宋喆；    **4月19日**，公司股东由上述三人变更为共青城宝亿嵘投资管理合伙企业（有限合伙）法人股东、王宝强。          至此，**马蓉、宋喆早在今年4月19日之前，就已被踢出公司股东层。** |

**脑子是个好东西，好希望我有你有大家有。**

就是啊，何必搞得这么慌张呢？

从时间上看，宝强单方面宣告“解除”婚姻关系，“解除”经纪人职务的声明不是应该准备好久了吗？

既然，王宝强你能如此理性的处理好这些关系，微博上又何必如此义愤填膺。

**把一个女性放在高台之上，供万人辱骂唾弃！供万人宰割！这个人曾是同床共枕的枕边人，是自己孩子的母亲，当我再看看王宝强的照片，美美忽然觉得寒毛直竖和毛骨悚然！相爱相杀，原来人心可以那么狠？**

美美宁愿看到的一幕是：王宝强第一时间拿着刀去了结了对方。当然这是不对的。但至少比发表这份置之死地的声明，更像一个男人。

**▼**

娱乐圈感情纷纷扰扰，分手、离婚事件不要太多，作为像宝强一样的公众人物，也不乏劲爆狗血剧情：什么黄奕撕逼黄毅清，董洁拉锯潘粤明，姚晨伏击凌潇肃，贾静雯夺子孙志浩……

### 这些年作为观众，我都麻木了。所有陈芝麻烂谷子的私密往事，一件一件往外抖，争先恐后到生怕自己落了下风！

### 

### 就好像农村赶集时坐地打滚的泼妇呀，她们一手掐着脚踝，一手拍打着大腿，一边大声哭诉：你们都来看看呀，大家都来评评理……

### **我不歧视，但我不忍心看下去。**

### 

### 虽然当事人可以把自己的委屈，自己的伤疤揭开来，挥洒的漫天都是，供全天下人评论、观瞻，但当我遇见这种事，还是会**感觉像是无意间撞破了别人隐私一样，局促，且尴尬。**

### 

即便是此去经年，大家好像都各自又拥有了幸福的生活，可以像翻书一样把往事翻过，但双方的孩子、家人、朋友、圈子，他们该如何面对那些伤痛？

▼

其实我想说，两个人此生能走在一起，是缘分，但这种缘分和平常不同，它是伴随着任意一方的退出，戛然而止的。是不以任何撕逼为转移的。

这种退出，你可以选择激烈，你可以选择撕扯，你可以选择吊打，**当然你也可以选****择更具绅士的做法。而不是这样你死我活。**

请别再说，我这个人就是直，我是性情中人，我真淳朴，真憨厚，激烈就我的表达方式……

毕竟，谁都有过去，谁都有余生，无论把对方撕成什么样的逼，我们都还要在这颗蓝色星球上把这一生给相伴度过去……

**把女方放在了火上猛烤，将自己引导成一个没有丝毫错误的男人，他正被兄弟夺妻，被妻子背叛，全国人民都在同情王宝强。**

任何事情都有原因和理由，婚姻是缘分和经历，无论幸福还是痛苦，都是双方行为的结果，谁又敢说，王宝强丝毫没有错？！但这封声明里，他没有说自己丝毫的错

**王宝强曾经说过：同情比羞辱更可怕，不知道被全国人民同情着的宝宝究竟感受如何。**

**▼**

对不起，王宝强，

从离婚声明开始，你不再是我的偶像

你和成熟的男人之间，隔着一万个谢霆锋。
