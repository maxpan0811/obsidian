> > 经过几轮筛选和答辩，2014年上半年CEO杰出贡献奖及公司杰出贡献奖出炉了，项目介绍及奖金情况如下，各位先知晓下。
> >
> >  
> >
> > 本次总共申报参评项目14个，QA复核满足参评标准的7个，最终答辩通过并获奖项目4个，其余3个留待继续观察效果参与年底评选。另外，门票事业部和投资团队共同获得CEO特别贡献奖。
> >
> >  
> >
> > 1.       **华南批发商反击战** **大住宿事业部****CEO****杰出贡献奖**  **奖金** **300****万**
> >
> > 项目简介：
> >
> > 酒店平台资源整合试点项目，拟通过在华南地区，联合内外采购资源集中攻破一点，从而完成了预付国内采购端的战略布局，领跑同行！项目历时7个月，涉及公司内外多战略合作方的资源整合，是携程酒店平台与业务紧密结合的最佳实践。公司2013年底独立预付板块之前，[华南地区预付市场](x-apple-data-detectors://0/)本地势力盘踞一方，导致包含批发商在内的整体预付采购能力处于弱势，高端酒店劣势明显，蚕食着携程华南整体市场份额。随着项目的结束，广深一线城市上半年增长率接近50%，高于去年同期15个百分点，在保持现付近30%的高速增长外，预付产量同期翻了6倍，online劣势率大幅下降37个百分点，为公司挽回近3亿的客户价值。以此同时，项目期间研究出一套平台定价策略和收益管理工具，总结出了可复制可推广的多方联合作战经验，具有标杆意义，项目成果已用于港澳地区，并计划在国内其他区域推广。
> >
> >  
> >
> > 2.       **国际机票****ATPCO****主运价** **大交通事业部****CEO****杰出贡献奖** **奖金** **150****万**
> >
> > 项目简介：
> >
> > ATPCO主运价项目，是跳过中航信直接接入ATPCO系统中的国际机票原子运价数据。我们是国内首家利用ATPCO数据的公司，也是国际上直接利用ATPCO数据屈指可数的OTA之一。该项目进一步扩大了公司在国内领先的优势。本项目实现了规则的自动处理，新运价进入系统的时间从原来一天缩短到了2小时，同时提高了运价的准确性，降低对航信的依赖，而且带来了运价及航线的大幅度增长。项目直接带动了国际机票8%的增长。
> >
> > 项目过程中，项目组对引擎进行了大的技术创新改造，在数据成倍增长的情况下，速度仍然有大幅提升。
> >
> > 1)       基于内存映射文件技术，把原本从数据库载入的数据通过文件中直接载入内存，实现了数据高速获取。
> >
> > 2)       基于点对点的文件传输协议，利用局域网文件传输系统实现了数据的快速更新。
> >
> > 3)       整个引擎集群共享一台服务器生成缓存数据，简化了数据更新流程。既保证了所有引擎服务器的数据一致性，又摆脱了对数据库的强依赖。
> >
> > 4)       实现了引擎高速并行计算和全流程流式计算， 提升了服务器计算资源的利用。
> >
> > 3.       **门票事业部****+****投资团队****CEO****特别贡献奖**  **奖金****240****万**
> >
> > 项目不用介绍了吧，大家都晓得的噢J
> >
> >  
> >
> > 4.       **巡航项目** **商业智能部** **公司杰出贡献奖** **奖金****50****万**
> >
> > 通过提升酒店Noshow模型和延住模型的精准性，直接节省人员成本并提升酒店收益近4300万

> > 5.       **天机项目** **大住宿事业部** **公司杰出贡献奖** **奖金** **12.4****万**
