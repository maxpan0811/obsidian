![](.evernote-content/FB911F9A-7219-4E8D-BE0E-CE57166032F1/5A0FC491-C524-4C21-9398-6844DD8297C3.gif)

以前我从没理解过为什么有人会选择 11 英寸的 iPad Pro。iPad Air 功能几乎能做到 iPad Pro 的所有事情，除了稍微好点的屏幕和相机系统，所以当时对我来说，选择 11 英寸的 iPad Pro 感觉就是个多余的选择。

![](.evernote-content/FB911F9A-7219-4E8D-BE0E-CE57166032F1/4912ED09-68EA-4DFB-A24F-4B5398132AC7.png)

直到最近几周，我也体验了下 11 英寸 iPad Pro，现在我已经不太想回去用大屏的那款了。

1. 尺寸刚好
-------

选择 11 英寸的 iPad Pro，最直接的原因就是它的尺寸。

![](.evernote-content/FB911F9A-7219-4E8D-BE0E-CE57166032F1/75BA8421-6228-41F8-810A-B362141FA096.png)

它轻便到我几乎忘了背包里还有它。你完全不觉得它重，带着它去办公，毫不费力。

2. 写作体验无敌
---------

我现在正在用 11 英寸的 M4 iPad Pro 写这篇文章。说实话，写作体验比我手头上的任何设备都要好。

这不仅仅是因为它小巧方便。这个尺寸让我可以轻松地把它放进任何包里，随时随地拿出来写作，特别是如果你选了支持蜂窝网络的版本，就能更自由地处理文档和灵感。

![](.evernote-content/FB911F9A-7219-4E8D-BE0E-CE57166032F1/96D93B83-40DB-4983-BDBF-A9A64EC5F8AA.png)

而且，iPad 的操作系统和缺乏多任务的限制，反而让我在写作时更加专注。iPadOS 不支持同时运行多个应用，强制让我专注于一个任务。这样的纯粹体验非常高效，干扰少。

3. 相机太实用了！
----------

说实话，之前我一直觉得 iPad 上的相机完全没必要，谁会拿 iPad 拍照呢？

但最近我发现了一个场景，iPad 上的相机真的是太方便了。每次参加活动，讲者的 PPT 页码一页一页刷过来，内容密密麻麻，完全看不清楚。

![](.evernote-content/FB911F9A-7219-4E8D-BE0E-CE57166032F1/76735970-469D-4731-BBF4-33B589B24674.png)

以前我会用手机拍下这些幻灯片，然后 AirDrop 给 Mac 再去做笔记。现在呢？我只需要拿下 iPad 键盘，拍个照，瞬间就能在 Notes 里看到那张图片，完美地融入笔记中，效率爆表。

![](.evernote-content/FB911F9A-7219-4E8D-BE0E-CE57166032F1/518BF3F1-939C-4175-A535-F6E2DFF71AD9.png)

虽然我平时不怎么用相机，但这个功能真的让我省了好多时间。

4. iPadOS 18 的魅力
----------------

这可能是我用过最稳定的苹果系统之一，感觉比我预期的稳定多了。现在体验也很棒，忍不住想和大家分享下几个让我特别兴奋的新功能：

![](.evernote-content/FB911F9A-7219-4E8D-BE0E-CE57166032F1/45ED9B41-170B-450A-BDE2-AAA9CEF07E36.jpg)

**智能手写校正**：对于我这种字写得一塌糊涂的人来说，iPadOS 18 上的智能手写功能简直就是天赐良机！它会根据我的字迹不断学习，自动调整、校正，让我的手写更加清晰易读。虽然目前功能还有些小bug，但潜力巨大。

**计算器应用**：我终于等到了一个原生的、没有广告的计算器应用！不再需要转到网页上搜索计算器了。

**应用自定义**：iPadOS 18 终于支持了应用图标自由摆放，可以不再按照原有的网格排列。甚至还可以为图标染上自己喜欢的颜色，虽然有时搞得看起来像一团乱，但如果调配得好，还是挺好看的。

5. Face ID 真的很方便
----------------

以前我还觉得 Touch ID 和 Face ID 有什么区别，反正都是用来解锁的。可是用了 Face ID 后，我发现它比 Touch ID 更方便多了。

![](.evernote-content/FB911F9A-7219-4E8D-BE0E-CE57166032F1/B76331D9-F73A-4251-8AA8-306159F94AA2.png)

几乎每次解锁，我不需要动手指，只要对着屏幕一扫就能解锁。唯一的麻烦是，我老习惯性把手指放到电源键上，得重新训练自己。

6. 它的寿命超长
---------

M4 芯片的强大性能让人感到不可思议。很多人可能会觉得它的性能过剩，但我倒是觉得它能让这款 iPad Pro持续使用好多年。

![](.evernote-content/FB911F9A-7219-4E8D-BE0E-CE57166032F1/6E34B5EB-D1DF-4AEF-AFD5-BA4ACD0DDA45.png)

其实如果你花了这么多钱买一台 iPad Pro，最重要的不是“超强”性能，而是它的耐用性。

最后
--
