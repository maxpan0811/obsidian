# 清明感怀

# ——追忆父亲跟随的首长们

# 作者 思目

# （老红军陈目海之子）

# 2025.4.5

昨日是清明节，许多人自发地涌入烈士陵园，纪念为革命牺牲的先烈们！笔者扫墓归来，禁不住回想起二十年前的一个晚上，老父亲一段感情真挚的回忆。

那是2005年晚秋，在组织的殷切关怀下，刚度过九十岁生日的老父亲，去医院做了起搏器植入手术。大概是植入后有些不适，那天晚上老父亲久久难以入睡。平时不怎么爱说话的老父亲，突然对在一旁陪护的笔者十分感慨地说道：参加红军后，一共挨了七枪，仅胸部中弹造成贯通伤就两次——四次反围剿打草鞋岗一次，长征打黄平一次，侥幸没死。从来也没有想到，居然还能活到九十岁！

说着说着，不知怎的就回忆起他参加朱毛红军后，牺牲的那些首长们，……

1929年6月20日，朱毛红军再次攻打龙岩城，我就是在这一天，加入的红军——当时我患疟疾正在陈国辉旅的医院治疗，同一个姓黄的医官等一起被红军俘虏。红军想留下这个黄医官，他借机提出要我做他的勤务兵（我原是二营营长的勤务兵）。后来才明白，黄医官非要我做他的勤务兵是有意图的，没过多久，他就抽冷子跑了！

大概是因为我不懂医，卫生队就把我退回总部，这样我就从第二纵队被调到第三纵队去了。第三纵队是原来秋收起义部队，红四军下井冈后由红三十一团改番号为第三纵队，记的其时的纵队司令是伍中豪、党代表好像是蔡会文。伍中豪很注意仪表，每天都穿戴整齐，一副标准的军人派头。大概是他已有一个勤务兵，所以我在他身边转了大约一个星期后，就被调到第七支队支队部去继续当勤务兵。

![](attachments/6307ae1f63628e24.jpg)

截取自1956年给上级组织部门《自述》的底稿。

这个七支队是刚由五支队改的，因为红四军进入闽西后队伍扩大了，第一和第二纵队各增加了一个支队。这个七支队的前身，源自武汉国民政府警卫团，由于没赶上南昌起义转而参加毛主席领导的秋收起义，失败后余部在三湾缩编为中国工农红军第一师第一团第一营。1928年4月，朱德率领的南昌起义余部及湘南起义部队与毛主席领导的部队在井冈山会师后，合并组建红四军时，改番号为红三十一团（秋收起义余部）第一营。

我到支队部的日期大约是在7月中旬，记得当时第七支队的首长有支队长、党代表和副支队长等三人，我被分配给一个姓冯的副支队长当勤务兵。因为那时我还不满十四岁又不太懂事儿，他们的名字基本上都记不太清了——只记得支队长姓曾、党代表姓欧阳，两人全都是湖南人；副支队长姓冯，个子高高的是北方人。之所以记不住他们的姓名还有一个重要原因，那就是与三位首长待在一起的时间太短，尤其是与曾支队长和欧阳党代表。

曾支队长当年9月下旬打上杭时一支胳膊被打残，伤愈后升任第二纵队司令员，之后在1931年打高兴圩时壮烈牺牲！虽然我与他相处的时间仅有两个多月，但却因我也姓曾和一件刻骨铭心的事儿，留下较为深刻的印象！那是我被冯副支队长因故打了一顿后，他先是把我留在他身边，之后又分配我跟着欧阳党代表。多年后，在给上级组织部门写《自述》时，我忽然想起支队长的名字似乎叫“曾石福”。后来看到资料等才晓得，支队长的大名应该是“曾士峨”！因当时仅听到别人叫他时的口音，故在《自述》中误写为“曾石福”。

![](attachments/07718a2d9191fecb.png)

出处同上

没有记住欧阳党代表的名字，也是因为待在一起的时间过短，曾支队长离开后周昆被调来接任支队长，我被分配给他当勤务兵。当年10月底二打梅县时，欧阳党代表带领战士们发起冲锋时，不幸腹部中弹负重伤！伤愈后，他被提升为第三纵队政治部主任，第七支队政委（古田会议后党代表改为政委，笔者注）由朱良才接任。解放后我错以为欧阳钦就是他，写了封信聊表思念之情，没收到回信还有些埋怨。直到九十年代才恍然大悟他叫欧阳健，由于消息闭塞不晓得，早在1932年的赣州战役中他就已英勇牺牲！

我给冯副支队长当勤务兵不久，大约在7月底，部队离开龙岩向东北方向开拔。三、四天以后，来到“溪南”、“杨美墟”、“上林坪”一带，一天晚上在一个不知名小镇的街上宿营。第二天拂晓行军时，我不小心把冯副支队长的包裹给弄丢了——可能是走的时候忘了拿。得知包裹丢了，他立刻就火了，用马鞭子狠狠抽了我一顿！结果，我左小腿的皮肤都给打烂了，直到现在还能看到留下的那块伤疤。

可能是支队长曾士峨看见冯副支队长打我的情况以后，红军毕竟不是旧军队，临时把我分配给欧阳政委当勤务兵。1930年初，我记得是在打过永丰县城以后，在江西东固附近的“坡头”和“直夏”一带，同国民党军唐云山的独立十五旅作战时，冯副支队长不幸牺牲了。

我被分配给周昆当勤务兵后，他一直待我挺好——他不是旧军官出身，直到1930年的10月我才被调离。古田会议之后，闽赣苏区的红军在毛主席领导下发展很快，第三纵队与闽西的地方武装于6月组建红十二军。军长由伍中豪担任，政委是谭震林；随后，周昆升任第一纵队司令员——我跟着他去纵队部上任，蔡会文回来担任纵队政委——原七支队政委朱良才被调走，好像是去了黄公略的红三军。10月，红军将纵队改为师，周昆为红三十四师师长，师政委换成高自立。大约就在这时，我从周昆身边被调离。

说到这里，老父亲忍不住长叹了一口气，然后继续说道：你看，我早期跟随的三位领导——冯副支队长于1930年初第一个牺牲；之后是支队长曾士峨，1931年9月在打高兴圩时第二个牺牲；支队党代表欧阳健，据说后来调到宁都起义的红五军团任师政委，1932年2月打赣州时第三个光荣牺牲了。如果只跟过几天的伍中豪也算，他是在1930年下半年牺牲的。比起他们，我不知有多幸运！比他们多活了七十多年，还有什么不满足的！？

![](attachments/f84c6795d5118dc4.jpg)

右为伍中豪（图片取自网络）

考虑到父亲的回忆过于简单，笔者认为有必要依据查证到的信息资料，将上述三位革命烈士的生平做一较为完整的简要介绍：

**曾士峨烈士**根据目前可见的资料信息，曾士峨字迪勋，号广泽，湖南益阳人。1926年夏北伐军入湘时，考入国民革命军第八军独立一师学生队。经指导员苏先骏（浏阳县人，共产党员，黄埔四期生，后叛变）介绍，加入中国共产党。

1927年初，进入黄埔军校长沙三分校学习。之后在国民革命军第四集团军总司令部警卫团任连长，北伐到武汉后，转入我党控制的武汉国民政府警卫团继续担任连长。

武汉警卫团未能赶上南昌起义，留在江西扩编为“江西省防军暂编第一师”，由原任第一营营长的黄埔二期生余洒度任师长。为加强对已编入该师第三团苏先俊部的控制，余洒度特将分别由张子清、伍中豪、曾士峨为连长的三个建制连调入该团为第三营，张子清和伍中豪分任第二营和第三营营长，曾士峨任第三营参谋长。秋收起义失败后，毛泽东在三湾对秋收起义余部进行改编，曾士峨被任命为特务连连长。

![](attachments/245b2ca62bc21a27.jpg)

曾士峨遗照（图片取自网络）

上井冈山后，曾士峨一直跟随在毛泽东身边，之后在作战中负伤。1928年6月，三连连长资秉谦在龙源口战斗牺牲，曾士峨立即接任三连连长。周舫牺牲后，曾士峨接任第一营营长。1929年3月，红四军将红二十八团及红三十一团改为第一和第三纵队，曾士峨任第三纵队参谋长兼第五支队（后改为七支队）支队长。

古田会议结束后，伤愈复出的曾士峨接任第二纵队司令员。1930年6月，升任红四军参谋长兼第二纵队司令员，10月部队改纵队为师，曾士峨任军参谋长兼红十一师师长。1931年9月，红一方面军围攻高兴圩的粤军蔡廷锴部，战斗最激烈时，曾士峨组织军官队带头冲锋——腹部中弹英勇牺牲！

对于曾士峨的牺牲，毛泽东十分悲痛！据有的文章介绍，在红军粉碎敌人第三次“围剿”的祝捷大会上，毛泽东首先叫全体同志起立向曾士峨烈士默哀，并指示红军总政治部发出通令，号召红军指战员向曾士峨学习。

1950年曾士峨的三哥曾叔彬写信给中央询问曾士峨的下落，军委总干部部在回信告之曾士峨牺牲之情状称赞道：“曾士峨同志为我党我军之优秀干部，为革命牺牲，实为我党我军之重大损失。”并向家属颁发烈士证书，给予最高抚恤金500万元(相当于500元)。

**欧阳健烈士**依据李立娥著《红军将领欧阳健评传》，欧阳健系湖南华容县人，出生时的名讳为欧阳曙，长大后为自励将“曙”改为“熙”。1926年经著名共产党人蔡协民等人介绍，加入中国共产党。

1927年“马日事变”后，他与同是华容人的何坤听从毛泽东的建议，分别改名为欧阳健与何长工。随后，与何长工一起加入武汉警卫团，被任命为扩编的“江西省防军暂编第一师”第一团连指导员。

秋收起义失败后部队在三湾进行改编，依据时任七连文书之一的宋任穷回忆，欧阳健被任命为七连党代表。但经笔者多方查证，七连第一任党代表应为李克如、第二任为孔繁树，欧阳健似为1928年4月后方接任。

1929年3月，红四军将团改为纵队，欧阳健仍任七连改编的十六大队党代表。8月接替蔡会文，被任命为第七支队党代表。10月底二打梅县时，欧阳健身负重伤。古田会议结束后，欧阳健升任第三纵队政治部主任。

1930年6月组建红十二军时，欧阳健被任命为该军第二纵队政治委员。10月纵队改为师时，欧阳健被任命为红三十五师政委，其时粟裕为该师参谋长兼第一零四团团长。

闽赣苏区第一次反围剿时，红三十五师在代军长罗炳辉指挥下，成功将敌主力第十八师引入龙冈。随后红一方面军围攻龙冈，全歼敌第十八师师部及两个旅近万人，敌中将师长张辉瓒被时任红三十四师第一百团团长的寻淮洲率部活捉！

![](attachments/52f454949ff9668f.jpg)

此为网上搜到的欧阳健画像

1931年6月，红三十五师师长龙普霖调离，师长一度由欧阳健兼任。9月7日，欧阳健正指挥红三十五师向高兴圩西北方的长迳村迂回时，突然与粤军戴戟的第六十一师毛维寿部遭遇！随即双方展开激烈战斗。战至8日，原已撤往兴国的敌蒋鼎文第九师突然杀回，措手不及的红一方面军只好撤离。

此战双方均伤亡惨重，战后红四军进行改编，欧阳健调至由红三十四师改编的红十二师担任政委一职。是年底，发生震惊中外的宁都起义！欧阳健被调往起义部队整编的红五军团，担任第十三军第三十七师政委。

1932年3月，红十三军赶到赣州援助红三军团时，欧阳健身先士卒率领红三十七师奋勇拼杀，不幸被一颗子弹击中头部壮烈牺牲！时年尚不满三十岁。

由于欧阳健参加红军后已改名，家里人始终不清楚他的下落，直到1950年接到他的老战友何长工写来信，方知道欧阳健早已为革命献出了宝贵的生命！

**冯明忠烈士，**他就是老父亲念念不忘的冯副支队长！笔者是从曾任红三十一团组织干事邓华将军的回忆中，查到他的名讳。笔者依据老父亲所言的“他是军校生”反复进行查证，推断他应该是与曾士峨一起于1927年，毕业于黄埔军校长沙三分校。之所以如此笃定，一是查遍广州黄埔军校名录未见其大名；二是只有长沙（黄埔）分校第一期召有北方学生。后来，他与曾士峨一起，被编入武汉警卫团。依据三湾改编时，冯明忠被任命为第三连党代表推断，他应该也是1926年的党员。大约上井冈山后，被调任机枪连连长。

1929年红四军下井冈前，冯明忠似应被任命为红三十一团第一营副营长，之后为第五支队、第七支队副支队长。按照《朱毛红军历史追踪》一书中的记叙，冯明忠担任过第七支队的支队长，但依据曾士峨负伤没几天周昆即于10月初接任支队长推断，冯明忠至多代理过几天而已。

另一个明确的佐证是，冯明忠直到1930年初在江西东固附近的“坡头”和“直夏”一带，同国民党军唐云山的独立十五旅作战牺牲时仍为第七支队副支队长。

令人惋惜的是，冯明忠的革命经历不仅《中国红军人物志》中没有记载，而且连张照片也没留下。笔者的父亲曾遗憾地说过，搞不好，解放后他的家人可能背负国民党军官的骂名！

父亲最后跟随的首长，是时任红一百团团长的寻淮洲烈士，虽然只给他当了不到三个月的勤务兵，却有着极深厚的感情——打龙岗负伤全愈之后，被寻淮洲安排去学吹号。之后父亲没有再回到红三十四师，而是追着寻淮洲去了红三十五师，一直到1932年底寻淮洲被提升为红二十一军军长才与他彻底分开。

老父亲晚年伤感地说，他是1939年跟着一一五师到山东后才得知寻淮洲早已牺牲的消息，那天晚上他跑到村外没人的地方默默流泪。

今年10月是红军长征胜利九十周年，我们今天的大好河山，是无数革命志士流血牺牲才换来的。正如毛主席他老人家所言：为有牺牲多壮志，敢教日月换新天！
