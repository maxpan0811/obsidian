邀请你来体验全新的少数派 Web 3.0 ᵇᵉᵗᵃ
=========================

2 小时前

距离少数派上一次改版已经两年之久，这两年时间里少数派在内容类型上做了很多探索和改变，在此基础上拓宽了渠道和业务，也有赖于各位的支持，少数派在这两年间一直朝着更好的方向发展。

今年，我们有了一定的余力在产品上做一些改变和优化，要使少数派能够更好地满足现阶段以及未来规划中官方、作者及读者的需求。

在产品改变的第一阶段中，我们对少数派 Web 进行了一次大规模更新，从设计、结构到功能都作出了不小的改变。经过前段时间紧张地开发，少数派 Web 3.0 ᵇᵉᵗᵃ 终于上线，快来体验一下吧！

```
在登录少数派后，点击右上方头像在下拉菜单中点击「体验新版」，并打开「开关」（该开关可以随时打开/关闭）。接下来你就可以体验到少数派 Web 3.0 ᵇᵉᵗᵃ目前在开放测试的几个功能模块了。
```
