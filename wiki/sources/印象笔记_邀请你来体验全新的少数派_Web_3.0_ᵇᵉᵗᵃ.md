邀请你来体验全新的少数派 Web 3.0 ᵇᵉᵗᵃ
=========================

2 小时前

距离少数派上一次改版已经两年之久，这两年时间里少数派在内容类型上做了很多探索和改变，在此基础上拓宽了渠道和业务，也有赖于各位的支持，少数派在这两年间一直朝着更好的方向发展。

今年，我们有了一定的余力在产品上做一些改变和优化，要使少数派能够更好地满足现阶段以及未来规划中官方、作者及读者的需求。

在产品改变的第一阶段中，我们对少数派 Web 进行了一次大规模更新，从设计、结构到功能都作出了不小的改变。经过前段时间紧张地开发，少数派 Web 3.0 ᵇᵉᵗᵃ 终于上线，快来体验一下吧！

```
在登录少数派后，点击右上方头像在下拉菜单中点击「体验新版」，并打开「开关」（该开关可以随时打开/关闭）。接下来你就可以体验到少数派 Web 3.0 ᵇᵉᵗᵃ目前在开放测试的几个功能模块了。
```

![](.evernote-content/41B82CBD-4785-4882-9EFB-6BE76FCEA7CB/EA973DF7-B647-4CF2-9A93-C0D5D7070EA6.png)

全新的设计风格
-------

少数派 Web 3.0 由《比身边人多一点设计》的作者 MakicLin 设计，本次少数派 Web 3.0 在设计上完成了交互和视觉设计层面上丰富和统一。在风格上，少数派 Web 3.0 采用了更加直接、精确的棱角。

对于本次改版 MakicLin 说：

> 新的少数派在设计语言上做了更高层次的统一：页面区块分布明了，色彩系统更加完善，阅读体验更加舒适。从设计上为少数派做好准备，进化为一个更完整的数字生活消费指南。你也会发现新的少数派在视觉上多了些棱角分明。因为我相信少数派内容里透出的态度和独特感，像一个尖锐且精确的直角，是值得读者和作者炫耀和喜欢的东西，希望你也喜欢新的少数派。

### 文章卡片更加多样

在少数派 Web 3.0 中我们重新设计了文章卡片，并为其新增了一套标准的「分类标识」，文章的来源和属性一目了然。

![](.evernote-content/41B82CBD-4785-4882-9EFB-6BE76FCEA7CB/C5BF405D-CE56-4C56-A1C3-0F137CE93A70.png)

除此之外，我们在基本文章卡片的基础上新增了「派早报」「重要文章」的卡片类型，值得特别关注的内容不会轻易错过。

![](.evernote-content/41B82CBD-4785-4882-9EFB-6BE76FCEA7CB/697B3C90-79B0-4F4E-A1B1-97FF5BA55B41.png)

### 文章页适应不同场景

本次更新对文章页也做了全新的设计，加入了侧边快捷按钮以及目录，对文章的操作更加便捷。你会发现原来「点赞」按钮被替换成了「充电」，如果你觉得这篇文章还不错记得点击「充电」哦。

![](.evernote-content/41B82CBD-4785-4882-9EFB-6BE76FCEA7CB/42C7D138-B08C-4BD5-976B-914A405C960C.png)

和文章卡片一样，在基础文章页之外也针对不同来源和属性的文章提供了不同的样式。

比如你每天都在看的「派早报」，在新的文章页面下，你可以更快速地了解今天有哪些值得关注的科技消息，同时快速跳转查看详情。

![](.evernote-content/41B82CBD-4785-4882-9EFB-6BE76FCEA7CB/40BC48CD-269C-4746-AC8E-F2DAB67F3588.png)

还有出现频率越来越多的视频内容，现在也有了观看体验更佳的专属样式。

![](.evernote-content/41B82CBD-4785-4882-9EFB-6BE76FCEA7CB/C6AC6876-7C6A-4791-884A-09071A50F8C5.jpg)

结构功能调整优化
--------

本次更新，对于之前网站结构和功能上的问题作出一定的调整。菜单导航及站内通知尤为人诟病的几个地方已经得到了解决，我们也带来了一些新的特性。

### 菜单导航分层更清晰

旧版网站导航栏中可以说是「一锅烩」，不管是独立模块还是文章分类都堆叠在了一起，信息层级混乱不清晰。在 3.0 我们对菜单导航进行了分离，更清晰直观，在有跳转模块或切换文章分类时能够准确找到操作区域。

![](.evernote-content/41B82CBD-4785-4882-9EFB-6BE76FCEA7CB/784E82B3-0F82-46EF-BDD7-9202B722A0D3.png)

### 通知中心优化

在旧版本中，由于所有的通知都是堆叠在一起的，你很难在通知列表中找到相对重要的一条。对于一些作者来说，经常会遇见这种情况：

* 写了一篇文章；
* 反响很好，获得了读者关注和热烈讨论；
* 通知中心出现了几十甚至上百条对于这篇文章的操作提醒（点赞、评论）；
* 打开通知中心，发现评论被淹没在了点赞的海洋。

新版通知中，我们将通知中心分为了三个部分，分别展示**喜欢及关注**（频次高、不太重要）、**评论及通知**（频次一般、重要）以及**系统及订单**（频次低、重要）这三种类别的通知信息，解决了之前屡被诟病的问题，极大的提高了通知中心的效率。

![](.evernote-content/41B82CBD-4785-4882-9EFB-6BE76FCEA7CB/ADF1F512-ECAB-48D2-ABB4-2FABE18944A2.png)

### 账号设置

这次更新中，「账户设置」部分新增了「个性域名」的设置。设置「个性域名」，有助于帮助你更好地记忆及分发你在少数派的「个人主页」。

你可以在少数派新版登入你的帐号后，依次点击右上角的「头像」-「帐号设置」-「帐号和密码」，在「个性域名」中设置你的个性域名。

![](.evernote-content/41B82CBD-4785-4882-9EFB-6BE76FCEA7CB/01F05029-DF39-4BC1-A876-81D59FE074A0.png)

当设置完成后，就可以通过 `https://sspai.com/u/username` 形式的链接访问到你的少数派主页了。

### 个性域名设置须知

* 个性域名创建后将无法修改，请在设置前仔细斟酌并检查个性域名输入无误
* 个性域名支持包含 5-16 个字符的英文字母、数字及下划线
* 在新版正式上线前，主页链接将包含 beta 前缀
* 原 ID 形式的链接在新版正式上线后将无法访问
* 设置个性域名后可能导致在第三方编辑器上使用的 Token 失效，若出现同步问题，请前往 Token 设置页「新建」或联系工作人员
