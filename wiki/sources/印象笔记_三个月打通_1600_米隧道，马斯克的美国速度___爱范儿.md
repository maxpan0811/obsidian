三个月打通 1600 米隧道，马斯克的美国速度
=======================

[董车会](https://www.ifanr.com/category/intelligentcar)

02-17 20:21

7

评论

为了避免在拥堵的路上浪费宝贵的时间，科比选择直升机出行，但世事无常，这位伟大的运动员不幸事故遇难，惨痛的教训再次警告富人们，飞机并不一定是最保险的出行方式。

如何尽大可能保证安全又确保通勤的高效，马斯克在三年前大开脑洞计划在城市底下挖一条隧道，仅供自动驾驶的特斯拉电动车出行。

这家专职挖洞的公司起名 “Boring”，一可译为钻孔，二为无聊，不谙中文的马斯克同样是玩一语双关的好手。

![](.evernote-content/213F0EE0-2B9A-4A52-A9C7-8ED04DBE1616/325E9C4B-336D-4D2E-97D8-8852C57A23F2.jpg)

2019 年 8 月，拉斯维加斯会议和旅游局董事会批准了与 Boring 公司签订的合同，为拉斯维加斯会议中心设计和建造一个 Loop 系统，连通拉斯维加斯的会展区与大学校园区，马斯克因此拿下了这笔价值 4870 万美金的订单，在竞标中落败的是当地的经典交通方案 —— 单轨列车。

继去年实验性 “试车” 之后，马斯克一时间的 “无聊之举” 终于取得了阶段性成果。在赌城拉斯维加斯会展园区的地底下，一条 1600 米的隧道被成功打通，而距离这一区域正式开工仅过去短短三个月。其开掘速度丝毫不逊色上海特斯拉工厂的建造，堪称隧道挖掘史上的又一项纪录。

![](.evernote-content/213F0EE0-2B9A-4A52-A9C7-8ED04DBE1616/338F2AE5-101A-4680-8432-330D3CC688C5.png)

和地铁庞大的轨道不同，根据图片和视频材料显示，Boring 隧道的直径并不大，几乎仅一台 Model X 的宽度有余，可想而知挖掘难度是不能和地铁相提并论的。

![](.evernote-content/213F0EE0-2B9A-4A52-A9C7-8ED04DBE1616/BE07289A-3B71-4DBD-8F88-3FC2B894E654.jpg)

官方也表示，隧道的挖掘成本不会很高，同样长度的线路下，它仅为地铁隧道挖掘成本的十分之一，甚至是三十分之一左右。

待隧道通车之后，来到拉斯维加斯参加会展的人即可免费乘坐特斯拉，在 1 分多钟的时间内穿过拉斯维加斯会展园区。按照特斯拉的计划，到 2021 年 1 月，Loop 系统可以每小时运送 4400 名与会者。

简单说明一下 Loop 系统的工作逻辑：乘客通过地面站点通过吊舱垂直下降到隧道之中，这个载具目前还是特斯拉的量产车型，未来可能会是一款基于 Model 3 底盘定制的 “胶囊电动车”，单车可以乘载 8-16 人。虽然运载能力仍不及地铁列车，但其最高时速可达 250 公里，完全可以在速度上弥补运力。

![](.evernote-content/213F0EE0-2B9A-4A52-A9C7-8ED04DBE1616/6B801C39-16B7-4E77-A0E0-8B4D323B124B.gif)

2018 年，特斯拉曾在加州 SpaceX 公司后院挖了一条试验性隧道并成功进行了实车试验，他们为一台 Model X 的车头车尾装上了类似四驱车的四个导轮，没有之前构想的 “托盘”，车辆在隧道中行驶全靠自己提供驱动力。

首先车辆需要具备自动驾驶能力，能够自行控制刹车油门和方向，导轮介于车身和路肩之间，为车辆提供转向支撑。我们目前能看到隧道内的道路为水泥地，但官方表示后期会让地面极为光滑，较小的摩擦力能让车行驶得足够快。测试现场，车辆的最高速度达到了 64 公里每小时。

![](.evernote-content/213F0EE0-2B9A-4A52-A9C7-8ED04DBE1616/DA9B8642-D63E-4D70-8C89-F63EA53E505B.gif)

而在 2019 年的一场地面与地下的比武中，隧道内的 Model 3 跑出了最高 204.39 公里的时速，连上下传送加隧道行驶，地下这台车用 1 分 36 秒跑完了 1.83 公里的路段，而地面车辆则是花了 3 分钟 8 秒。

虽然测试令人兴奋，但坊间的质疑声不断。比如说，上下传送会不会导致地面拥堵？一台车的运力太小；安全性有待考证等等……

不管怎么说，马斯克的疯狂脑洞又再一次走进现实，并实现了商业化及阶段性成果，而这一方案究竟能否缓解交通压力造福民生，我们明年 CES 就能看到答案。
