Tesla Model X 下周见，千兆工厂已破土
=========================

2014-8-1, 16:26 | [钟舒婷](http://www.ifanr.com/author/neffy "Posts by 钟舒婷")

[![](.evernote-content/BD94D272-A448-4D99-9F1B-D682D4368281/4AD43A9A-9C46-4B17-B7D3-E01BC409FD3E.jpg)](http://cdn.ifanr.cn/wp-content/uploads/2014/08/dsc01107.jpg)

今天，是特斯拉的[财报日](http://www.ifanr.com/news/438480)，第二季度特斯拉营收达到 7.69 亿美元，但净亏损为 6200 万美元，而在随后的[电话会议](http://www.forbes.com/sites/markrogowsky/2014/07/31/tesla-q2-earnings-live-eps-blows-past-estimates-on-record-production-confirms-gigafactory-groundbreaking/)中，CEO 马斯克（Elon Musk）更是透露了一些关于 Model X 的消息。

在特斯拉的[中国官网](http://www.teslamotors.com/cn/modelx)上，我们可以看到 [Model X](http://www.ifanr.com/400549) 计划在 2015 年夏季进行交付，因此对 Model X Alpha 版的测试已经被提上了日程。从下周开始，特斯拉将对 Model X 的 Alpha 版本原型车进行测试，从而对之前的设计方案进行验证。测试的目标包括电池、汽车材质的耐用性，甚至会进行碰撞试验。

[![](.evernote-content/BD94D272-A448-4D99-9F1B-D682D4368281/5CA29F78-A4A8-4CAE-A3DE-A79DEAE7C6D2.jpg)](http://cdn.ifanr.cn/wp-content/uploads/2014/08/121.jpg)

特斯拉的第三代电动车 Model X 采用 SUV 车型，配备 60kWh 或 85kWh 电池组，从零加速到 96 km/h 只需要 5 秒钟。而它最大的特点是采用了“猎鹰门”（Falcon-Wing Doors），后者会搭载传感器，用于调节开门顺序，以避免撞击附近的物体，以及 Model X 打算用镜头[取代](http://www.ifanr.com/411725)传统汽车的后视镜，这些大胆的设计是特斯拉是否能如期进行生产计划所需克服的问题之一。

目前，特斯拉正在加紧准备 Model X 的生产，上个月特斯拉就暂时[关闭](http://www.ifanr.com/news/434954)了位于佛蒙特的工厂，花费 1 亿美元用于建设新的组装线。在下周这条组装线就会投入使用，每周可以生产 1000 辆 Model S，届时 Model X 也会在这里同时投入生产。在进行 Alpha 版的测试后，特斯拉会尽快推出 Beta 版，并期待在启动生产后的三个月内实现每周数百辆的产能，比当初 Model S 所需的时间减少一半。

[根据](%20http://www.forbes.com/sites/markrogowsky/2014/07/31/tesla-q2-earnings-live-eps-blows-past-estimates-on-record-production-confirms-gigafactory-groundbreaking/)今早的财报，特斯拉期望在第三季度可以生产 9000 辆 Model S，交付 7800 辆，今年预计可以交付 3.5 万辆 Model S。而马斯克也放出豪言，称在明年年底要实现 10 万辆的年产量。

特斯拉的图景描绘得如此美好，进军亚太市场，大幅度提高产能，但如果想要应付大量 Model S、Model X 的订单，特斯拉首先要解决的是电池的供给问题。周四，特斯拉宣布与松下就建造千兆工厂（ Gigafactory） [达成了合作协议](http://www.ifanr.com/news/437211)。Gigafactory 是一次性将电池完全整合的电池工厂，特斯拉打算在同一家工厂生产并组装所有的零部件，从电池元件到电池成品。这样每千瓦时的电池组的成本将降低 30 % 以上，毫无疑问，这对接下来 Model X 和平民车型 [Model 3](http://www.ifanr.com/417922) 的生产具有重要的战略意义。

在电话会议中马斯克还提到， 第一家千兆工厂在内华达州的选址已经在 6 月[破土动工](http://www.ifanr.com/418480)，同时另外两个州的选址也正在跟进。

千兆工厂造价将达 50 亿美元，根据特斯拉和松下的协议，特斯拉将出资其中的 40% 至 50%，而松下出资 30% 至 40%，工厂所在的州出资 10%。到 2020 年，Gigafactory 每年所提供的电池最多将满足 50 万辆特斯拉电动车的需求。
