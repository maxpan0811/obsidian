我从 Windows 转到了 macOS，用了挺久的，总体来说体验还是挺不错的。

多年来，我基本离不开 Windows，不管是日常用的家用电脑，还是学校、工作里的服务器管理，全靠它支持。期间虽然也接触过 macOS，用过几台 MacBook，但那会儿一直没把它当主力机。

![](.evernote-content/5D0B9A8D-34F5-4FBF-BF59-92DE5FED3145/096A2FBF-61D5-40FD-9F6F-CFA9BA75B024.png)

转用 macOS 后，我确实体验到了很多新鲜感，但也必须说，有些东西是 Windows 才有的，我还挺想念的。今天就聊聊这几个让我最怀念的 Windows 功能。

1. 更方便的窗口管理
-----------

**Windows 11 的 Snap Layouts 真心好用**

在 Windows 11 上，Snap Layouts 是一个让我特别喜欢的功能，能非常方便地把多个窗口整齐地排列在屏幕上。比如需要对比文档、处理多任务的时候，它特别省心。

![](.evernote-content/5D0B9A8D-34F5-4FBF-BF59-92DE5FED3145/5D38A966-1632-4D7B-BA8E-2DE396BC36C7.png)

macOS 也有类似功能，比如按住 Option 键再点绿色按钮就能选择分屏，但说实话，感觉没有 Windows 那么直观和高效。 希望 Apple 在未来的 macOS 版本里加强这方面的功能吧！

2. Windows 的文件管理器
-----------------

**用 Finder 八个月了，我还是更喜欢 File Explorer**

![](.evernote-content/5D0B9A8D-34F5-4FBF-BF59-92DE5FED3145/7010432D-B8D5-4912-A93C-FDCB8858737C.png)

虽然我不太喜欢 Windows 11 的文件管理器，但我觉得 Windows 10 的 File Explorer 确实比 Finder 好用。它界面更直观，特别是那个“详细信息视图”，展示文件的方式很清晰。

另外，Windows 传输大文件的时候，会有传输速度的实时图表，这点也是 Finder 比不了的。

3. 硬件升级的自由
----------

**怀念能动手折腾的日子**

老实说，现在的笔记本普遍都开始变得越来越不可升级了，这不仅仅是 Apple 的问题，很多品牌都这样。

![](.evernote-content/5D0B9A8D-34F5-4FBF-BF59-92DE5FED3145/937B946E-A32F-455C-A819-6126B61D4034.png)

不过，以前用 Windows 台式机的时候，那种自由度还是挺让人怀念的。换硬盘、加内存，甚至换机箱，按照自己的风格把电脑内部线缆整理得整整齐齐，那种 DIY 的乐趣是 MacBook  无法给我的。

4. 游戏体验
-------

**用 Mac 玩游戏？想想就算了**

最后一个不得不提的就是游戏体验。虽然我很久没正儿八经用 PC 玩游戏了，但偶尔还是会想念那些在电脑上畅玩的日子。

![](.evernote-content/5D0B9A8D-34F5-4FBF-BF59-92DE5FED3145/69672523-A159-454A-8C78-A6E627FD70C7.png)

我喜欢玩一些大型游戏，用键盘操控的时候，那种沉浸感比用主机强太多了。而 Mac 的硬件和生态显然不是为游戏而设计的，这方面还是 PC 更胜一筹。

最后
--

换到 macOS 后，我确实感受到了更简洁的界面、更流畅的系统体验，以及和其他 Apple 设备之间的无缝协作，让我工作效率提升了不少。但不可否认，Windows 依然是个非常不错的系统，它的一些功能，比如 Snap Layouts、File Explorer、硬件升级自由度和游戏支持，依旧让我怀念。

![](.evernote-content/5D0B9A8D-34F5-4FBF-BF59-92DE5FED3145/5B013A12-419B-4212-B20A-3674FFFA47A7.png)

当然，现在 macOS 更符合我的实际需求，所以也不后悔这个选择。
