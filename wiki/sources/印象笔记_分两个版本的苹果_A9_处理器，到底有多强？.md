# 分两个版本的苹果 A9 处理器，到底有多强？

---

[![](.evernote-content/E857DAB4-650D-4451-9838-417FC43FF9C8/C7703413-35FE-4243-AC05-884DCA242FC6.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/09/death_note_ryuk_yagami_light_artwork_anime_apples_Wallpaper_2560x1600_www.wallpaperswa.com-1.jpg)

前两天，我们刚看了新 iPhone 的拆解。拆解中有一个细节，iPhone 6s 上的 A9 处理器型号为 APL0898 ，封装的是三星 2GB LPDDR4 RAM；而 iPhone 6s Plus 的 A9 处理器型号为 APL1022 ，封装的是海力士 2GB LPDDR4 RAM。

这两款处理器的区别并不止于型号，实际上，这两款有很大不同。根据 iFixit 的说法，APL0898 由三星代工，采用了三星 14nm FinFET 工艺制造；APL1022 由台积电代工，采用 16nm FinFET 工艺制造。

而 [Chipworks](https://www.chipworks.com/about-chipworks/overview/blog/inside-the-iphone-6s) 在后续的芯片级拆解中也验证了这一说法。

[![](.evernote-content/E857DAB4-650D-4451-9838-417FC43FF9C8/8C89B1EC-C37F-493C-B2FD-AF5FE4B1F9F4.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/09/A9-1.jpg)

从 Chipworks 给出的这张示意图来看，16nm 版 A9 处理器封装面积要大于 14nm 版 A9 处理器的封装面积。不过，搭载两个不同版本的新 iPhone 在实际使用中、甚至性能跑分中差异都不大，难以察觉。

### A9 处理器包含了什么？

苹果鲜有在发布会上公布处理器技术细节的习惯，所以当时大家对于 A9 的细节其实并不清楚。Chipworks 的这次拆解对于探究 A9 的细节有很大帮助。

[![](.evernote-content/E857DAB4-650D-4451-9838-417FC43FF9C8/F210983A-9799-4331-8B1A-3393FE997F8D.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/09/151421n8qn5n6zxgnjeg5q.jpg)

这里以 14nm 版 A9 为例。这张示意图可以让我们清楚的看到 A9 处理器的细节。在这块芯片中一共封装了双核心 CPU、六核心 GPU 和两个 SRAM，以及协处理器 M9。

### CPU 部分

与 A8 处理器的 CPU 部分相似，A9 处理器的 CPU 部分依旧是双核心。不过苹果的处理器单核性能一向非常强劲，这次的 A9 处理器更是如此。

[![](.evernote-content/E857DAB4-650D-4451-9838-417FC43FF9C8/ACE8B9C9-397D-4C18-9142-3F24FDF7FB6A.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/09/CPU.jpg)

Geekbench 3 检测显示 A9 处理器的频率为 1.83 GHz，相比 A8 的 1.4 GHz 提升不小。最后的跑分结果也说明了这一点，单核 2526 分，多核 4404 分。其单核性能在历代 iOS 设备中遥遥领先，多核性能则略逊色三核心的 A8X 处理器，位居第二。如果横比的话，在该项测试中，Exynos 7420 取得了单核 1520 分、多核 5478 分的成绩；骁龙 810 取得的成绩为单核 1331、多核 4294。由此来看，A9 处理器的 CPU 单核性能确实了得。

如果拿这个成绩用来和今年的 New Macbook 对比一下结果应该有趣。[Daring Firebal](http://daringfireball.net/) 就干了这样的对比。1.1 GHz 版的 Macbook （搭载双核心四线程的 Core M-5Y31 处理器）在同样的 Geekbench 3 中，单核得分 2295、多核得分 4464。从这个角度来说，苹果在发布会上的宣传并没有“违反广告法”。不过苹果 A 系列处理器在 Geekbench 测试一向表现较好，所以跑分超过 Core M 处理器的结果还是理性对待比较好。

### GPU 部分

由于 A9 处理器的 GPU 部分为六核心，从苹果御用的 GPU 提供商 Imagination Technologies 的产品线来看，这款 GPU 很可能是去年 A8X 上大方异彩的 Power VR GX6650，也有可能是 Power VR GT7600。A9 搭配的 GPU 具体型号究竟是什么，我们先按下不表，直接来看看简单粗暴的图形处理跑分。

GFXBench 结果如下。

[![](.evernote-content/E857DAB4-650D-4451-9838-417FC43FF9C8/F267B8B0-C5E4-44B6-8A35-636BBB959AA8.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/09/7.pic_2.jpg)

A9 的图形处理测试成绩在各个场景和模式下都非常不错。在和采用了 Tegra K1 的 Shield Tablet 进行对比后，我们发现 A9 的图形处理能力几乎全方位超越了 Tegra K1 这个去年的图形王者。

### 总结

A9 处理器有着很典型的苹果处理器风格。

CPU 部分走着大核心的路子，核心数在某些八核、十核处理器处理器面前完全不够看，但胜在单核心性能出众，双核心性能一流。虽然有三星和台积电代工之分，但两者在实际使用中暂时体验不出来。有一点需要指出的是，三星和台积电版 A9 在新 iPhone 中的使用规律并不清晰，或许两者都在 iPhone 6s 和 6s Plus 中混用，也可能是三星版 A9 在 iPhone 6s 中使用，而 6s Plus 使用的是台积电版 A9。

至于 GPU，这是苹果一直以来的优势部分，图形性能在 A9 之上的 GPU 目前来看，还是凤毛麟角，或许几乎没有手机使用的 Tegra X1 能与之一战。

总的来说，苹果 A9 在 Exynos 7420 的下一代产品和骁龙 820 铺货前，作为市面最强的 SoC 当之无愧。
