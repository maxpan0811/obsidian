什么是项目式学习？如何帮助学生规划？
------------------

原创*2016-12-29**Robert老师*[YouthCareer](http://mp.weixin.qq.com/s?__biz=MzIxMjA1NjcyMA==&mid=2456256738&idx=1&sn=3ed3d6fc0a1a203c9811a4d99301b387&chksm=80d8cf1db7af460b521509aa51c2566e477637866ea4992a01b50ab8ec0942e53a1fa934ce84&mpshare=1&scene=1&srcid=1229C3u2LzivcJaw3Pjq7EtG##)

（本文是《教育迷思》系列文章第十六篇，编号3.4“项目式学习”。详细目录参见公众号菜单里的“公众号导读”。）

项目式学习（project based learning，简称PBL）是一股正在兴起中的教育趋势，顾名思义，它强调的是在做项目的同时边做边学，强调知行合一。究竟什么是项目式学习呢？我们先对比一下项目式学习和传统学习的区别就会更一目了然。

传统的上课是这样的：

![](.evernote-content/0A7FDBF1-0DD3-4675-9AB3-8C843F733828/BF4D42A4-3ADB-4D32-AE7F-5D3F6C1CFB74.jpg)

老师写完板书说，我们今天讲电路的组成，分为电源、开关……，先讲讲什么是电源，blablabla。

少数同学会质问老师：我为什么要学电路？

老师的回答无非是：

a，考试分数高才能上好大学，以后才有前途。这些都是物理考试的重要内容。

b，这是要打好基础，以后才有机会学习其他更高深的科学。

c，物理可以锻炼你的批判性思维。

d，总之，这是为你好！

可惜听完老师的解释，学生并不领情，学生的反应是这样的：

![](.evernote-content/0A7FDBF1-0DD3-4675-9AB3-8C843F733828/E03A8A7B-E50A-496C-A3BB-9C6F6B1067CD.jpg)

台下其实已经睡着了一片，稍微给老师点面子无非画个眼睛充充门面……

这是因为学生感觉不到学习的意义，似乎和我没关系，都是被家长和老师逼着学。

同样是物理，我们看看特斯拉和SpaceX的CEO Elon Musk怎么教孩子。

Musk因为对教育不满，直接办了一个幼儿园，主要服务于员工子弟。

![](.evernote-content/0A7FDBF1-0DD3-4675-9AB3-8C843F733828/06414C76-1F9F-4346-BD43-33A944D7E63E.jpg)

以汽车中的机械知识为例，传统的做法是“我们将教会你有关螺丝刀和扳手的所有知识。”但Musk觉得这非常枯燥乏味，他认为正确的做法是给孩子一辆汽车，先开始拆吧。一旦开始拆卸，第一个遇到的关卡就是螺帽要松开，这时需要一把螺丝刀，这时孩子会主动来学习如何用各种类型的螺丝刀以及各种相关知识，老师不必再告诉他为什么要学习螺丝刀，这是不言自明的。

不仅是如何使用，更重要的是学生知道为什么要用螺丝刀？其目的是拧螺帽。那有没有别的拧螺帽的方法和工具？从一个角度看，手工螺丝刀和电动螺丝起子有什么不同？从另一个角度看，螺丝螺帽也有非常多的种类，为什么汽车的这个部位要用六角螺母？

在Mush看来：“理解问题、寻找解决方案是最重要的”。哪怕学生一时没有答案都无所谓，这种探索能力的练习远比直接告诉他们答案更重要。而且这些项目似浅实深，真正深挖下去可能性是无限的，不仅蕴含着大量的基础物理学原理，也有跨学科的化学、材料、设计、数学等大量知识。

我们先小结一下项目式学习和传统学习有哪些不同：

|  |  |  |
| --- | --- | --- |
|  | 传统学习 | 项目式学习 |
| 以谁为中心？ | 老师上课，以老师为中心（要我学） | 自学，以学生为中心（我要学） |
| 动机 | 被逼着学，缺乏动力 | 我要学，主动性非常强，项目导向、问题导向 |
| 学习目标 | 目标模糊，哪怕为了考试分数也会因为目标遥远而缺乏学习动机 | 有清晰的中短期目标（如拆车，或解答一个问题），非常务实、接地气。且注重学习能力、过程，知识不是学习目标。 |
| 如何学？ | 上大课，板书、PPT、讲解，老师规划学习进度，布置作业 | 个性化，学生自主规划、探索、实践，老师变成顾问从旁辅导 |
| 学科覆盖 | 单一学科 | 跨学科 |

接下来我们剖析一个经典的项目式学习案例，来看看项目式学习的威力。

TED上有一个高中生Jack，初中的时候他很喜欢的一个叔叔因为胰腺癌去世了，从那一天起他被胰腺癌问题吸引了（这是项目和动机的起源）。

![](.evernote-content/0A7FDBF1-0DD3-4675-9AB3-8C843F733828/2FECB2BF-2928-48A6-BA64-E04B23516522.jpg)

Jack通过互联网的初步科普学习发现，胰腺癌晚期基本没治，但如果中早期就发现则治愈率很高。只可惜早期测试的成本高、准确率低，于是他开始致力于如何改进中早期胰腺癌测试。（项目方向收窄，不涉及治疗，只做检测。不同的方向会导向不同的学习领域，比如治疗显然是医疗领域，而检测则可能涉及物理、生物、化学等多学科，取决于最后的方案。）

为了降低测试代价，一方面是降低单次检测成本，另一方面也要和最主流的体检相结合以覆盖更多的人，因为很少有人会想到早期去专门做某个病的测试。于是Jack把检测方向定位在血液中的蛋白质，并自学了相关的论文，了解了8000多种胰腺癌患者血液中的常见蛋白质。（这已经是研究生和教授级别的专业论文了，但并不妨碍目标清晰的Jack在中学就开始自学。）

接着Jack的项目子目标变成了在8000多种蛋白质中寻找最合适用于检测的蛋白质，他定出了科学的筛选标准：

a，该蛋白质应该适用于各种类型的胰腺癌；

b，早期胰腺癌患者含量显著的高；

c，非患者不含此蛋白质

并开始了机械而重复的排查筛选工作，最终在第四千多次筛选时，发现了一个符合标准的蛋白质。（这里主要涉及化学、生物、基因方面的知识学习，Jack在学习过程中已经远远超过了高中的生物和化学知识）

紧接着项目目标就转向如何测试这个具体的蛋白质。Jack的突破来自于一个只有项目制学习者才会拥有的灵感瞬间。在一节高中生物课实验课上，Jack在开小差，因为他觉得老师讲的很无聊（下图是Jack戏称生物实验课“绝对是扼杀创新的地方”）

![](.evernote-content/0A7FDBF1-0DD3-4675-9AB3-8C843F733828/8EA6A0A9-1039-4C45-8D11-A0220765C51D.jpg)

当时老师在实验一个只对特定蛋白质起反应的抗体酶，而Jack则在底下偷看物理材料学方面的课外书，主题是讲一种新型的纳米材料。突然，他头脑中电光火石闪过，他意识到可以将这两种特定的纳米材料和酶结合起来用于构建胰腺癌早期测试的微观结构。

所谓念念不忘，必有回响，如果不是Jack已经思考这个问题这么久，他绝不可能想到两者会存在这样一种联系，不会闪过这样一种解决方案。而恰恰只有项目式学习者，因为他们目标明确，他们持续在思考，他们为一些问题而深深的着迷，日思夜想，他们才最有机会捕捉这些突破的瞬间，去进行真正的创新。传统的学习模式在这种突破上是无能为力的，因为老师都不懂，生物老师懂些酶，物理老师懂些纳米，但在这个特定的交叉领域，老师心里并没有揣着那个精彩的问题，所以他们也永远给不出精彩的答案。

当然这还没完，突破性的解决方案之后还要落地。这种纳米结构很脆弱，必须有载体，Jack选择了纸，要制成胰腺癌特殊蛋白试纸。但这时他遇到了新的麻烦，他在家里没有实验环境，这些器材只有一流的大学和企业实验室里才有。于是项目的子目标转化为寻找到一个合适的、愿意接受高中生的实验室。（从这个阶段开始，Jack的学习开始升级，从纸面上、屏幕上转向了社会和人，普通的在线知识已经不能满足他，他需要拜师学艺，学习更细分的特定知识。但找导师的路和阅读文献截然不同，开始需要练习到沟通、说服、演讲、人际关系等软性能力。）

Jack写了200多封email发给各大学的相关领域教授，解释了其实验思路，申请去他们的实验室进行实验，他非常为自己骄傲，用他的话说：“我坐在那儿期待赞美的email涌入，说，你真是个天才！”。

![](.evernote-content/0A7FDBF1-0DD3-4675-9AB3-8C843F733828/8BB8F846-1AA1-4DC4-B51F-C5E0F93BD823.jpg)但猜猜事实上有多少教授愿意帮他呢？

只有一个。其他邮件基本都石沉大海，甚至还遇到闲着蛋疼来泼冷水的教授，长篇大论的反驳Jack的实验设想，逐条批驳每一个实验步骤，说这里错误，那里不可行……（但哪怕一个人都没有回，Jack也可以去大学当面堵那些教授，当面提出请求可以大幅降低被拒绝的概率。）

于是Jack来到了唯一给他机会的教授的实验室，在经受了教授和20个phd的轮番拷问后，他得到了他的实验机会。并经历了一个phd漫长而完整的实验、改进、再实验、再改进的重复过程，最终制造出了3美分、测试仅需5分钟、准确率接近100%的早期胰腺癌验血试纸，并受邀到TED来讲他的经历。

这是项目式学习的经典胜利，Jack在微观生物、医疗领域的积累其实远远不如同实验室的那些phd深，用大白话说，基础不够扎实，他承认面对不少质问他的回答甚至是靠猜的。如果换了普通的父母和老师很可能会打击Jack说，先好好学习，等读到phd甚至毕业到医药公司工作以后，再去挑战这样的难题吧。但其实真正的教育绝非如此，Jack展示的才是真正的学习，单点突破，虽然不如phd们扎实，但有选择的补充了部分基础知识，其实在项目中也就够用了，遇到问题再去学就好。知识是可以快速学习的，真正难以学到的是科学的思维模式、学习能力、想象力、沟通能力，真正难以坚持的是日复一日的思考一个难题并耐心的等待答案出现的那一天。

故事讲完了（想看视频的参见文末链接）。可惜项目式学习还远不普及，别说中国，即便在美国也不多见（Jack的学校显然也没有做到很好，以至于他不满学校的生物实验课）。所以具体到学生每个学期的学习，我们能够做些什么呢？

其实第一个思路已经隐藏在前面的故事里。

Jack和教授以及phd探讨的其实已经是一个博士毕业设计级别的作品。我们的教育虽然从幼儿园到大学都充斥着僵化的讲课式教育，但毕竟在最高端的博士领域，我们有一种截然不同的教育模式叫做写论文。博士头一两年还上一些课，但整个四五年的攻读期间，其最主要的精力和时间都花在了论文上。论文就是一种项目式学习，每个博士的选题都不一样，高度个性化。这种模式就是我们主流教育值得模仿的。

事实上，本科乃至中学，学生已经有越来越多的科目要求以一篇小论文为大作业，并计入学科成绩，但可惜其强度不够高、学生花的精力还不够多。所以最容易落地的项目式学习其实是每个学期精选一个科目的论文，尽量用大学研究生的高规格来对待。

这里的第一步最重要的其实是选题。phd的选题可能要数易其稿，花上一两年，换三五个方向，最后才定下。这个选题为什么重要？前人有过什么成果？我从哪个角度切入？容不容易冒出来，还是会泯然众人？这些选题过程对人的批判性思维、战略思维是最好的训练，而有时学生的高度和眼光还不够，这是家长可以协助学生一起来做头脑风暴的地方。比如这个学期有个初三的学生要写一篇历史小论文，自由选题，而他对船舶专业特别感兴趣，所以我们就讨论如何在这个领域找题材，比如某一时期或国家的沉船史、造船史。大学里甚至有教授为了研究船舶历史真的和学生们一起像古人一样重新手工造船，这样才能真切的考核为什么选某一种木料，为什么舱位这么布置，帆要多大……我一向主张极高明而道中庸，要敢想，立意要高，然后再结合学生的时间、实际条件，打些折扣来执行。

如果每个学期都用这样的高规格来对待一篇论文，整个中学阶段下来成果会非常可观。不仅能力上肯定会远超同侪，而且作为履历用作大学的申请也是非常有特色和个性化的一种素材。尤其是美国大学新启用的CAAS申请系统，鼓励学生从初中就开始把履历填写到系统中，把个人资料变成一种动态的成长，这会进一步防止学生作假，并且让真正有特色的学生变得更显眼。

除了上述的论文思路外，另外一个项目式学习常见的思路就是才艺重新规划。才艺是学生本身就会花非常多时间的一个方向，但以钢琴为首的很多才艺其实竞争过于激烈，算是企业里所谓的红海竞争，无法凸现学生的个人价值，并非上策。在多项才艺中应该做减法，至少省出一项的时间交给学生自己来规划一个长期的项目。比如同样是喜欢生物方向的学生，哪怕做不到胰腺癌测试这样级别的项目，至少可以先学一些国外的大学生物在线课程，读一些感兴趣的话题的论文，比如转基因、雾霾成分，即便一时无法达到做项目的级别，也可以先做主题演讲、辩论。如果在一线大城市，还可以趁着一流大学没放假时抽空去旁听一些课程、讲座。在这个过程中，允许孩子自己换主题，并引导和陪伴孩子去找学习资源、寻找导师。
