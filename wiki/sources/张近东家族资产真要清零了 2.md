# 张近东家族资产真要清零了

---

“眼看他起高楼，眼看他宴宾客，眼看他楼塌了。”

曾几何时，张近东是江苏首富，是中国零售业的教父级人物，苏宁的蓝色标志遍布中国每一座城市的繁华角落。然而，时至今日，当“张近东被列为失信被执行人”、“其所持全部苏宁易购股份被司法冻结”等新闻接踵而至时，一个沉重且残酷的问题浮现在所有人面前：张近东家族的资产，这次真的要清零了吗？

这并非危言耸听，而是对一个商业传奇可能走向终点的深度叩问。

从“服务为王”到“债台高筑”：帝国的裂痕

张近东的商业哲学，起点是朴素的——“服务是苏宁的唯一产品”。靠着这个理念，他从南京宁海路一家小小的空调专卖店，一步步击败了所有对手，与国美上演了长达十年的“美苏争霸”，最终成为中国家电零售的霸主。那是苏宁的黄金时代，现金流充裕，模式清晰，张近东当时也意气风发。

然而，时代的洪流从不会为任何人停留。电商的崛起，如同一场突如其来的海啸，冲击着所有实体商业的根基。面对阿里巴巴、京东的凌厉攻势，张近东并非没有警觉。他开启了苏宁的“互联网化”转型，线上线下同价的“O2O模式”在当时也颇具前瞻性。

但真正的转折点，也是悲剧的起点，是苏宁那场令人眼花缭乱的“大跃进”式扩张。张近东似乎不再满足于只做一个“卖家电的”，他渴望构建一个无所不包的“苏宁生态圈”。于是，我们看到了：

苏宁体育：豪掷巨资收购国际米兰俱乐部，拿下国内中超、英超等顶级赛事版权。PP体育一度风光无两，但体育版权是公认的“烧钱无底洞”，每年数十亿的投入，换来的却是微薄的回报和持续的失血。

![](.evernote-content/C8CCA9B3-86E8-4214-8C61-A86B4E790659/55B74D6B-01B6-417C-A7BD-A212B37CF106.jpg)

恒大客场挑战苏宁图片

苏宁金融：布局支付、供应链金融、消费金融等领域。金融虽是利润奶牛，但同样需要巨大的资本金和风控能力，在强监管和激烈竞争下，苏宁金融并未能成为坚实的后盾。

![](.evernote-content/C8CCA9B3-86E8-4214-8C61-A86B4E790659/E642DC3D-F52A-4757-B160-5F5093139EBE.jpg)

苏宁置业：大举进军商业地产和文旅项目。这看似与零售主业协同，实则将苏宁深度绑定了中国房地产的黄金周期。当恒大、融创等巨头一个个陷入困境，苏宁置业的重资产模式也变成了压垮骆驼的最后一捆稻草。

![](.evernote-content/C8CCA9B3-86E8-4214-8C61-A86B4E790659/701F9C73-27B1-4498-963D-821E7856729C.jpg)

这种“零售+体育+金融+地产”的多元帝国，听起来很美，但致命的缺陷在于，除了零售，其他所有板块都是“吞金兽”。它们不仅没能为主业输送稳定的血液，反而像无数根吸管，疯狂地抽取着零售业务辛苦积攒的现金流。苏宁这艘大船，本该加固核心引擎，却选择挂上了太多沉重的风帆，最终在风暴中失去了平衡。

股权质押的“死亡螺旋”：财富归零的加速器

当主营业务增长乏力，新业务持续失血，债务危机便如约而至。为了填补巨大的资金黑洞，张近东和苏宁系最常用的工具，就是“股权质押”。

这本质上是一场高杠杆的赌博。张近东将其持有的苏宁易购股票作为抵押物，向金融机构换取贷款。他相信只要股价稳定，就能不断获得资金，维持帝国的运转。然而，这却是一个典型的“死亡螺旋”设计：

1. 基本面恶化 → 2. 股价下跌 → 3. 质押平仓线逼近 → 4. 被迫补充质押或现金 → 5. 市场信心崩溃，股价进一步下跌 → 6. 触发强制平仓 → 7. 股价崩盘，控制权丧失。

![](.evernote-content/C8CCA9B3-86E8-4214-8C61-A86B4E790659/E0A9C4D8-C986-457D-AAFF-20F60FB3C6F6.jpg)

苏宁的走势完美印证了这一路径。从2020年底开始，苏宁易购的股价一路断崖式下跌。张近东不断补充质押，但终究敌不过市场的恐慌和基本面的坍塌。最终，大规模的强制平仓出现，他不仅失去了对上市公司的控制权，其质押的股票市值也灰飞烟灭。这部分财富，是张近东家族资产的核心大头，它的“归零”，几乎是定局。

更致命的是，股权质押的爆仓，往往伴随着个人连带责任的担保。当苏宁系公司无法偿还债务时，作为担保人的张近东，其个人名下的其他资产——房产、存款、投资等，都将被列入追偿清单。此次被列为“老赖”，正是因为他未能履行相关的法律给付义务。这意味着，债务的追索已经从公司层面，彻底蔓延到了他的个人生活。

“清零”的真相：是数字归零，还是帝国归零？

现在，让我们回到最初的问题：张近东家族资产真要清零了吗？

从数字层面看，绝对的“清零”可能并不存在。作为一个纵横商界三十余年的顶级富豪，其资产配置的复杂程度远超外人想象。我们无法断言他是否在危机爆发前，通过某种合法或灰色的方式，为家族转移了一部分“压箱底”的财富。或许在海外的某个信托里，或许在子女的名下，仍保留着一部分火种。但对于一个曾经拥有千亿身家的家族而言，即便剩下百分之一、千分之一，与其巅峰相比，也形同“清零”。

从帝国层面看，这无疑是一场彻底的“清零”。张近东失去的，不仅仅是股票市值，更是他一手建立的商业帝国、行业话语权、社会声誉以及未来东山再起的资本。他从一个规则的制定者，变成了一个被规则清算的对象。这种精神层面和权力层面的“清零”，远比数字上的清零更为彻底和痛苦。

![](.evernote-content/C8CCA9B3-86E8-4214-8C61-A86B4E790659/5608BE67-6722-408D-B6DE-844CA4B12338.jpg)

苏宁的案例，是中国民营企业发展史上一堂代价高昂的公开课。它警示所有企业家：

第一，永远要对现金流保持敬畏。任何脱离了现金流的“生态梦”和“帝国梦”，都只是沙滩上的城堡。

　　第二，警惕多元化的陷阱。盲目进入不熟悉的领域，尤其是资本密集型行业，极易引发系统性风险。

　　第三，高杠杆是双刃剑。在顺周期时，它能让你一飞冲天；在逆周期时，它只会让你摔得更惨。

如今，张近东的名字更多地与“被执行人”、“限制高消费”联系在一起。昔日的零售大王，如今却连乘坐飞机、入住星级酒店都成了奢望。这巨大的落差，令人唏嘘不已。

或许，对于张近东个人而言，财富的清零是一种解脱，让他从一场持续多年的豪赌中终局。但对于中国商业史而言，一个时代的符号以如此惨淡的方式落幕，终究是一场悲剧。他的故事，最终将成为商学院教科书里关于战略、贪婪与周期的一个冰冷注脚。

至于那座曾经辉煌的苏宁总部大楼，依然矗立在南京的徐庄软件园，只是楼里的人，和楼外的天，都已换了人间。
