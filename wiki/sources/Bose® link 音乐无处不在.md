# Bose® link 音乐无处不在

---

|  |
| --- |
| **Bose® link 音乐无处不在** |
| Bose® link能将你的悠闲家庭娱乐系统的功能扩展到家居周围最多14块收听区域——甚至是户外。  Bose® link扩展功能是内置在Bose 悠闲系统内的，还能将uMusic智能播放系统的性能扩展到最多14个聆听区域，甚至是户外。Bose的产品，比如赞誉不绝的妙韵音乐系统和Bose 151SE户外扬声器，可以轻松地结合在一起，让屋内和屋外都充满高音质的声音。用户可使用遥控进行独立的静音和音量控制，在两种可选的音频流中进行选择。  由于系统扩展到了多个房间，SA-3悠闲立体声功放与悠闲系统共同运作，增强本地的输入能力，使你的CD播放机或电视通过本地的接口就能加入到系统中来欣赏音乐。  现在，Bose® link AL8家庭无线音频连接使你比以前更加容易扩展自己的悠闲系统。 有了Bose® link AL8，你无需在房间之间架设线路就可以在另一个房间内享受自己的娱乐系统。  Personal® Music Center II遥控器是新增的配件，用于多个房间的安装。 双向交互式音频控制器拥有背光液晶屏幕，使用户可以用uMusic智能播放系统或通过浏览收藏夹和选择歌曲来播放。   * 无论你是在聚会还是只不过在家里走一走——音乐、谈话节目或是球赛的声音都能随着你走动，无论是在房间之间走动，还是在从房间进进出出。 * 由于你的悠闲系统可以在不同的房间内同时播放两套独立的声音，当你在露台、厨房或室卧室内欣赏音乐时，你的孩子能同时在起居室内观看电影。 * 悠闲系统控制器通过无线电频率进行运作，所以可以穿越大多数墙面和地板。 * 你可以通过悠闲系统添加Bose扬声器以及其他Bose的家庭娱乐系统，让整个房间都充满了Bose带给你的娱乐体验。 |
