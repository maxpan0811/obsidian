# 读文章、剪藏网页、新标签页增强…… 这 10 个扩展让你的 Chrome 更好用

6小时前

[![](attachments/80e3f05da75c19c2.jpg)](https://sspai.com/user/713147)

#### [Umi](https://sspai.com/user/713147)

目录

你有没有想过，在浏览器的新标签页看一眼就能知道今天要做的任务、发生的新闻；或者上网查资料时，浏览器能够自动帮你屏蔽无关网站，保持专注；甚至是在 YouTube 看视频时，也能像国内网站一样发送弹幕……

本篇文章我就为大家收集了 10 款 Chrome 插件，它们有的能够提升网页阅读体验，有的能够能够帮你剪藏网页，还有的能够帮你增强新标签页功能。下面一起来看看。

## 简 Tab

Chrome 上的新标签页扩展不胜枚举，如让你沉浸于美景的 [Momenteum](https://momentumdash.com/ "Momenteum")、纯净无干扰的 [Currently](https://chrome.google.com/webstore/detail/currently/ojhmphdkpgbibohbnpbfiefkgieacjmh "Currently") 和主推功能性的 [Infinity New Tab Pro](https://sspai.com/post/42282 "Infinity New Tab Pro") 等。而本次要推荐的 [简 Tab](http://ksria.com/simptab/ "简 Tab")，则在美观与实用之间做出取舍，成为了一款简约而不简单的佳作。

为了更纯粹地展示壁纸，简 Tab 砍掉了搜索框、天气等有碍观感的元素。但你仍可以将鼠标移至边缘以查看书签、常访问网站及工具栏等。简 Tab 支持 Unsplash、Bing、Wallheaven 等多个壁纸源，还能用向日葵 8 号卫星纵览地球实时影像。

除了展示壁纸外，简 Tab 还提供了纯色 + 模块化时钟的禅模式，以及白噪音、快捷键、自定义样式等多种特性。你可以通过下图了解更多。

你可以在 [Chrome 网上应用店](https://chrome.google.com/webstore/detail/%E7%AE%80-tab-simptab-%E6%96%B0%E6%A0%87%E7%AD%BE%E9%A1%B5/kbgmbmkhepchmmcnbdbclpkpegbgikjc "Chrome 网上应用店") 或 [官网](http://ksria.com/simptab/ "官网") 获取简 Tab。

![](attachments/9065f1bd28fc1120.png)

## ctrl tab

在 macOS 中，`⌘Command-↹Tab` 快捷键可以让你在多个窗口间快速切换，而 ctrl tab 的名字也源自于此。你可以利用它将 RSS、GitHub、Hacker News、BBC、Reddit 等数十个订阅源及 Todoist 等效率工具汇聚在一起，在新标签页纵览时事。

你可以在 [Chrome 网上应用店](https://chrome.google.com/webstore/detail/ctrl-tab/ijfnjhlohbehhlhhokgokabonpnlldon "Chrome 网上应用店") 或 [Firefox 附加组件](https://addons.mozilla.org/zh-CN/firefox/addon/ctrl-tab/ "Firefox 附加组件") 获取 ctrl tab。

![](attachments/e69e62addb8f8b70.png)

## Astrolabe

Chrome 对于多标签页的处理实在不够友好，稍微多一点，标签页就都挤成一团，完全无法分辨。为了解决这个问题，不妨试试 Astrolabe，它可以生成当前打开的所有标签页的缩略图，点击即可切换，一目了然。

你可以在 [Chrome 网上应用店](https://chrome.google.com/webstore/detail/astrolabe/lienonopcegannlgdkgonpofbhgmclcc "Chrome 网上应用店") 或 [Firefox 附加组件](https://addons.mozilla.org/zh-CN/firefox/addon/astrolabe/ "Firefox 附加组件") 获取 Astrolabe。

![](attachments/4a8f1be51e85e9af.png)

## Tab Manager Plus for Chrome

[Tab Manager Plus](https://chrome.google.com/webstore/detail/tab-manager-plus-for-chro/cnkdjjdmfiffagllbiiilooaoofcoeff "Tab Manager Plus") 同样是一款标签页管理扩展，它可以将所有标签页以图标或列表的形式排列，并允许你快捷固定、新建、关闭、搜索等。你还可以用颜色区分不同窗口，更加直观。

你可以在 [Chrome 网上应用店](https://chrome.google.com/webstore/detail/tab-manager-plus-for-chro/cnkdjjdmfiffagllbiiilooaoofcoeff "Chrome 网上应用店") 获取 Tab Manager Plus。

![](attachments/f39f154e084b14fa.png)

## Reggy

总有一些网站，要求我们必须登录才能查看内容。如果你不想将自己的隐私乖乖交出去，不妨试试 [Reggy](https://chrome.google.com/webstore/detail/reggy/pnneajlgffpejnabhionnhdfhibijihe "Reggy")。

Reggy 内置了 [Mailinator](https://www.mailinator.com/ "Mailinator")、[MailDrop](https://maildrop.cc/ "MailDrop") 等数个临时邮箱服务，并可以随机生成用户名、密码、手机号、住址等个人信息，还支持一键填充。接收验证码后，你就可以把邮箱抛在一边，再也不用跟垃圾邮件做斗争了。

![](attachments/8993cf85ad859deb.png)

需要注意的是，尽管 Reggy 能生成电话号码，但并不能用来接收短信，因此对付需要实名验证的国内网站就有心无力了。

你可以在 [Chrome 网上应用店](https://chrome.google.com/webstore/detail/reggy/pnneajlgffpejnabhionnhdfhibijihe "Chrome 网上应用店") 获取 Reggy。

## Just Focus

有时，我只想上网查查资料，双手却不受控制地打开了其它网站，甚至会有工作 5 分钟，摸鱼 2 小时的情况发生。而 [Just Focus](https://justfocus.co/ "Just Focus") 则可以为你临时屏蔽这些网站，并记录专心时间。如果你工作期间尝试访问黑名单网站，就会被跳转至 Just Focus 官网，欣赏名人名言。

你可以在 [Chrome 网上应用店](https://chrome.google.com/webstore/detail/just-focus/gefaddaengbodpiobpbgblajdboalmgc "Chrome 网上应用店") 获取 Just Focus。

![](attachments/845961a90d540197.png)

## Clearly

[Clearly](https://chrome.google.com/webstore/detail/clearly/odfonlkabodgbolnmmkdijkaeggofoop "Clearly") 是一款 Chrome 阅读扩展，可以将当前网页重新排版为易于阅读的界面。除了常规的主题、字体、字号等功能外，Clearly 还支持朗读网页内容，工作时也不怕错过喜爱的文章了。

你可以在 [Chrome 网上应用店](https://chrome.google.com/webstore/detail/clearly/odfonlkabodgbolnmmkdijkaeggofoop "Chrome 网上应用店") 获取 Clearly。

![](attachments/6b36b5c96a7f9502.png)

## MaoXian Web Clipper

若想将网页永久保存，你可能会首先想到印象笔记剪藏，但与其寄希望于在线服务，不如直接存到本地来得安心。而 [MaoXian Web Clipper](https://mika-cn.github.io/maoxian-web-clipper/index-zh-CN.html "MaoXian Web Clipper") 就可以将任意网页完整保存至本地，并支持 HTML 和 Markdown 两种格式。

MaoXian Web Clipper 的操作方式与印象笔记剪藏类似，只需要点击图标，选中想要保存的区域，就可以将网页下载至本地。你还可以自定义子目录及标签等，并在扩展页面搜索曾经剪藏过的网页。

你可以在 [Chrome 网上应用店](https://chrome.google.com/webstore/detail/maoxian-web-clipper/kjahokgdcbohofgdidndeiaigkehdjdc "Chrome 网上应用店")、[Firefox 附加组件](https://addons.mozilla.org/zh-CN/firefox/addon/maoxian-web-clipper/ "Firefox 附加组件") 或 [官网](https://mika-cn.github.io/maoxian-web-clipper/chrome-install-by-crx.html "官网") 获取 MaoXian Web Clipper。

![](attachments/f66776127c4becf9.png)

## Color by Fardos

无论是平面设计，还是幻灯片演示，一套优秀的色彩方案都可以起到锦上添花的作用。而 [Color by Fardos](https://chrome.google.com/webstore/detail/color-by-fardos/iibpgpkhpfggipbacjfeijkloidhmiei "Color by Fardos")，就可以承包从取色到配色的全部流程。

安装扩展后，只需输入名称，你就能找到相关颜色及其互补色、三色调等方案，你还可以点击屏幕上的任意颜色，并实时预览调色效果。你还可以依照 RGB 色值匹配任意颜色，并保存至调色盘备用。此外，Color by Fardos 还支持屏幕取色及渐变配色，可谓设计师必备了。

你可以在 [Chrome 网上应用店](https://chrome.google.com/webstore/detail/color-by-fardos/iibpgpkhpfggipbacjfeijkloidhmiei "Chrome 网上应用店") 获取 Color by Fardos。

![](attachments/72eb6fcbeb892a06.gif)

## Dmooji 弹幕君

随着 B 站的异军突起，优酷、爱奇艺等传统视频网站都相继实装了弹幕功能。不过，行业老大哥 YouTube 并不为之所动，让不少小伙伴有槽难吐。而 [Dmooji 弹幕君](https://www.dmooji.com/ "Dmooji 弹幕君") 这款扩展，就可以让你在 YouTube 上找回熟悉的感觉。

安装 Dmooji 后，你会发现 YouTube 视频进度条下方多了一块似曾相识的区域。只需用 Google 或 Facebook 账户登录，你就能够自由发送弹幕了。除了基础样式外，Dmooji 还支持设置弹幕位置，并发送彩色、渐变和反向弹幕。

![](attachments/31aaa8ba639118f3.png)

观看时，你也可以设置弹幕展示区域、防遮挡、透明度及字体大小等，并查看全部弹幕列表。为了避免用户数量不足带来的尴尬，Dmooji 还会自动提取评论，并作为弹幕展示，看视频再也不会寂寞了。

你可以在 [Chrome 网上应用店](https://chrome.google.com/webstore/detail/dmooji-live-comments-danm/dcacgbaadlgfnmcpjncoobionpjnbnih/ "Chrome 网上应用店") 或 [官网](http://dmooji.com/ "官网") 获取 Dmooji 弹幕君。
