最佳答案

```
是USB接口规范吧。  
USB Tpye-C 是USB的接口标准 “USB Type-C”，USB PD 是“USB Power Delivery Specification（USB PD）”供电规格。  
其中USB PD2.0供电规格是USB Type-c接口支持的一个重要标准。  
TYPE-C与USB PD2.0，将从根本上解决电子产品供电的混乱（不一致）局面。  
形象的说，就是适配器全部做成一个产品，用电器通过USB Type-C接口通知适配器输出需要的电流电压。  
从而实现充电器和充电/通讯接口的统一/通用。
```
