# 微软SB和SP4芯片细节公布 详细分析该如何买

---

[![](.evernote-content/573DB0D2-7F4E-48FB-891A-653DEBA55DF7/6591572C-C257-4D91-A612-C4A22B7E9084.jpg)](http://tech.feng.com/2015-10-12/Microsoft-is-SB-and-SP4-chip-details-Detailed-analysis-of-how-to-buy_626865.shtml)

![](.evernote-content/573DB0D2-7F4E-48FB-891A-653DEBA55DF7/6591572C-C257-4D91-A612-C4A22B7E9084.jpg)

威锋网 10 月 12 日消息 关于之前很多人对微软两款新 Surface 产品的规格疑问，今天终于得到了官方的明确确认。微软表示，顶配版 Surface Pro 4 将搭载 Skylake 架构的 Core i7 双核处理器，而 Surface Book 也采用了几乎相同型号的双核心版本，但是频率更高，速度更快。

具体而言，顶配版 Surface Book 搭载了英特尔第六代双核心的 Core i7-6600U 处理器，集成 HD Graphics 520 核显。而 Surface Pro 顶配版则采用了 Core i7-6650U 双核处理器，集成 Iris 540 核显。Surface Book 和 Surface Pro 4 其他低配版本的处理器也得到了披露，不过暂时没有公布频率。

微软还提供了一些有关 Surface Book 独立显卡的某些细节，不过型号未公布。微软在声明中称，“Surface Book 内部 Nvidia GeForce GPU 基于 Maxwell 架构设计，拥有 1GB 的 GDDR5 高速显存。这枚 GPU 经过定制，以确保现实中拥有更好的图形处理性能。”

**疑问得到解答，Surface 新品都是新“Intel inside”处理器**

上周二 Surface Book 的发布会上，微软 Surface Book 负责人 Panos Panay 声称，“得益于独立显卡和两个额外的处理器，从根本上 Surface Book 比 MacBook Pro 快两倍”。

那时候听到“两个额外的处理器”这样的语句很多人完全搞不清楚是什么意思，并认为微软可以神奇的将四核处理器塞进如此轻薄的平板电脑上，以发挥更强大的性能，但现在已经可以确定说的就是双核心处理器。

![](.evernote-content/573DB0D2-7F4E-48FB-891A-653DEBA55DF7/CCBCB75E-7921-425D-8286-726848505A0A.png)

关于 Surface Book 和 Surface Pro 4，虽然微软没有处理器的参数，但既然考虑到全都是英特尔六代 Skylake 架构新品，那么我们从英特尔官方网站也能了解到更详细细节，下面是基本上不会有误的处理器参数。

**Surface Book：**

- 英特尔 Core i5-6300U 默频 2.4 GHz（睿频 3.0GHz），集成 HD graphics 520 核显

- 英特尔 Core i7-6600U  默频 2.6 GHz（睿频 3.4GHz），集成 HD graphics 520 核显

**Surface Pro 4：**

- 英特尔 Core m3-6Y30 默频 0.9 GHz（睿频 2.2GHz），集成 HD graphics 515 核显

- 英特尔 Core i5-6300U 默频 2.4 GHz（睿频 3.0GHz），集成 HD graphics 520 核显

- 英特尔 Core i7-6650U  默频 2.2 GHz（睿频 3.4GHz），集成 HD graphics 540 核显

**Surface Book，一定要花 2700 美元买顶配版才值得吗？**

可以看到，Core i5-6300U 和 Core i7-6600U 两者其实主要的差别还是在于处理频率的高低。CPU 部分自然后者性能更强，而 GPU 部分虽然都是相同的 HD graphics 520 核心，但是 Core i5 型号的 GPU 最高频为 1000MHz，而 Core i7 款则为 1050MHz，实际区别不大。另外， Core i7，拥有过 4MB 的三级缓存，Core i5 为 3MB。

![](.evernote-content/573DB0D2-7F4E-48FB-891A-653DEBA55DF7/F6B2A41A-4EBF-494B-B81A-6A95DF195482.jpg)

那么处理器的特性上有何差异呢？几乎没有，两者都支持 Intel vPro 技术，支持 VT-d 和 VT-x，也支持 AES-NI 指令集，这些功能大多都与企业 IT 设备管理和数据加密安全相关，非企业用户的话，其实不必考虑其中的差别。因此，在处理器的选择上，普通消费者只要遵循性能上差异即可。

纯性能自然还是 Core i7 最强，不过两款处理器的 TDP 都是 15W，所以在负载压力性能测试上差异不会超过 15%，而对于轻量或中度的日常使用差别不大。因此，我们建议不是时常处理复杂任务的用户，直接购买“Core i5 + 256GB + GeForce GPU”型号即可，不必多花 200 美元买同配置的 Core i7 型号，毕竟这多花的钱在平时使用你很难注意到其中的差异，除非真的用于大型办公和专业工作。

**那 Surface Pro 4 该如何选呢？**

与 Surface Book 不同的是，这一次微软没有再提供 Core i3 的版本，而且不同的处理器集成的核显型号也完全不同。

我们找到了一张英特尔 GEN 9 第九代集显的差异图。你可以看到，顶配的 Surface Pro 4 所集成的 HD graphics 540 核显拥有多达 48 个 执行单元，而且还附带 64MB 的 eDRAM，在所有型号中图形性能表现上自然最为不俗。

![](.evernote-content/573DB0D2-7F4E-48FB-891A-653DEBA55DF7/9F88653D-4CCB-45A5-BAED-996016487374.jpg)

为何呢？大家可以翻看很多对桌面级 Core i7 -6700k 核显性能的评测，该处理器集成的 HD graphics 530 核显，在分辨率为 1366×768 分辨率时，运行新《古墓丽影（Tomb Raider）》的帧数仍然有 54fps。不过我们并不要指望 Surface Pro 4 的 HD graphics 540 能够超越这个成绩，毕竟 TDP 限制，而且还要考虑到非桌面平台的内存带宽差异影响性。当然，应付轻量级的游戏绝对不是什么大问题。

Surface Pro 4 中档的 Core i5-6300U，相比之下速度就要慢些，主要体现在图形处理熊能上，比如运行 Photoshop 程序 Core i7 款就比较有优势了。但是应付一般的图形处理任务和轻量级游戏 Core i5 款同样与 Core i7 款无明显差异。

真正有意思的是入门级的 Surface Pro 3， 搭载 TDP 仅为 4.5W 的 Core m3-6Y30 处理器，它的速度与 TDP 15W 的 Core i5-6300U 和 Core i7-6650U 相比就差很多了，并且它集成的 HD graphics 515 核显最高仅 850 MHz 的频率，差距非常明显。当然，功耗的显著降低，说明在续航方面肯定比高配的两款更加出色，而且性能也比低端搭载 Atom X7 的 Surface 3 强不少 。

![](.evernote-content/573DB0D2-7F4E-48FB-891A-653DEBA55DF7/6226DBE4-F230-4786-ADF6-6D9040A55D0F.jpg)

该怎么买呢？说实话，预算充足直接购买“Core i5 + 8GB + 265GB”是最好的选择，只是价格已经与入门级的 Surface Book 相当，心理上可能很痛苦。不过究竟如何选还是根据个人实际情况最好：

- Core M3 款。你比较追求的是电池的续航，而且你看不上 Atom 产品，那么拥有更高的 CPU 和 GPU 性能的 M3 型号会是不错的选择。但是如果你是企业用户，M3 版缺乏很多企业特点，比如 vPro，那么请放弃。

- Core i5 款。你有些时候还用来处理一些工作任务，使用到 Photoshop、Lightroom 或进行视频短片编辑，直接买 8GB 内存版已经可以满足了。

- Core i7 款。 你长期使用处理器较为繁重的任务，而且真正需要更快的 CPU 和 GPU 图形处理器性能，那么 Core i7 款整体应该会提供比 Core i5 款将近 20 % 左右的性能提升（还无实际测试）。

**关于 Surface Book 的 GPU**

最后回头来看一下猜测一下 Surface Book 的 GPU。微软表示，这枚 Nvidia GeForce GPU 内核就藏在 Surface Book 配备的超薄外形的键盘上，支持热拔插操作，无需重启，拆开平板电脑部分时，只要按下键盘上的一个解锁按键即可，但是微软始终没有透露更多的详细细节。

![](.evernote-content/573DB0D2-7F4E-48FB-891A-653DEBA55DF7/FD4FD482-EC94-4301-B31A-7586B34414EC.png)

其实 Nvidia 在英文官方博客上透露了有关这枚 GeForce GPU 的部分细节，不过很多内容已经被修改过了，不知是否微软不满意。在 10 月 6 日的原版博文中，Nvidia 发言人确认该 GPU 基于 Maxwell 架构设计，但鉴于是在超薄外形的键盘中的定制款，而且独特的外形限制，它的性能无法与传统的 GeForce 900M 显卡匹敌。另外，它配 1GB GDDR5 显存，归类于 800M 系列。

总之，Surface Book 这枚 GeForce GPU 有很多无法知晓其中的细节，还有待上市之后查看更深入的评测。
