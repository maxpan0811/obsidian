### 01 dexter

这是一款专为金融研究打造的自主智能代理工具，能够自主思考、规划并完成复杂的金融分析任务。

![](attachments/958ec075628d76f4.jpg)

它可以把复杂的金融问题拆解成清晰的研究步骤，借助实时市场数据完成自动化分析，还能自我校验结果，直到得出可靠结论。

![](attachments/53828a61f5acb8e7.jpg)

适合金融分析师、投资者以及需要快速处理财报、资产负债表、现金流报表等数据的专业人士，不用手动整理数据，就能获得数据支撑的分析结果，效率提升十分明显。

https://github.com/virattt/dexter

### 02 DocuSeal

DocuSeal是一个开源的电子签名平台，可以让你像用DocuSign一样在线签署PDF文件，但完全由自己掌控数据。

![](attachments/5a6d99dbeeac2ff5.jpg)

支持拖拽签名、日期、复选框等表单字段，并且生成的签名具有法律效力的审计追踪。

这个项目最大的亮点是“开箱即用”，提供了Web界面和API接口。相比商业软件，它不仅省了钱，还不用担心第三方服务的数据泄露风险。

你可以把它部署在自己的服务器上，用于公司合同签署、员工入职文件确认等场景。

https://github.com/docusealco/docuseal

### 03 DeepSeek-TUI

DeepSeek-TUI是一个类似Claude Code的终端代码智能代理工具，基于 DeepSeek V4 模型开发。

![](attachments/4268b187231efc2f.jpg)

支持文件读写、执行 shell 命令、代码编辑、Git 管理等完整开发操作。

它提供三种工作模式，可自动选择模型与思考强度，还能实时查看推理过程。

适合习惯终端操作的开发者、运维人员，在无图形界面的服务器环境中也能完成代码编写、调试与项目管理，支持私有化部署与多平台运行，轻量化且功能全面。

https://github.com/Hmbown/DeepSeek-TUI

### 04 html2rss-web

html2rss-web是一个能把任意网页转换成RSS订阅源的工具。

![](attachments/1d99ed9c4814a935.jpg)

现在很多网站已经不再提供RSS功能了，但你又不想频繁点开一个个网页去刷更新，这时候就可以用它来“变”出一个RSS订阅地址。

你只需要配置一下网页里的标题和内容对应的CSS选择器，它就会定期抓取并生成标准的RSS格式。

配合Reeder、Inoreader这类阅读器使用，所有没有RSS的网站都能被统一管理起来。适合RSS重度用户和希望减少信息噪音的读者。

https://github.com/html2rss/html2rss-web

### 05 DockFlare

DockFlare是面向 Cloudflare Tunnel 的自动化管理工具，属于自托管的入口控制器。

![](attachments/94ecbf05a84ab0f5.jpg)

它通过 Docker 标签自动配置隧道规则与 DNS 记录，提供可视化 Web 界面管理服务与访问策略，支持状态持久化与实时日志监控。

![](attachments/0bed4b52df12ec49.jpg)

适合那些自己运维了好几台服务器、跑了一堆容器的小团队或个人开发者，帮你第一时间发现“容器挂了”或者“内存泄漏”的问题。

https://github.com/ChrispyBacon-dev/DockFlare
