# 别再机械地「复制粘贴」了，你的剪贴板可以更好用

别再机械地「复制粘贴」了，你的剪贴板可以更好用
-----------------------

数码荔枝

复制粘贴可能是我们在电脑上最常用的操作之一，尤其是文本内容。但对于这样一个高频操作，系统原生的剪贴板功能，却并不好用。

**清除文本格式**

原生的剪贴板默认保留文本格式，如字号、粗体等，粘贴后经常需要重新调整，非常烦人。虽然个别软件（如 Chrome、Word）支持清除格式后粘贴，但并不是每个软件都标配这个功能。

Paste Plain Text 正是一款为解决这一痛点而开发、免费的 Mac 小工具。软件体积小巧、占用极低，启动后可隐藏在后台运行，自动清除复制文本的格式，完全不需要任何操作。

![](.evernote-content/AF841688-82C2-408E-A160-020223AD06CF/9E5DF42D-28D6-4A94-A83D-44337A8A4AAF.png)

**这些剪贴板工具更好用**

不过，类似 Paste Plain Text 这样的小工具，只能解决某些小痛点，当我们对复制粘贴有更多需求时，仍然会遇到各种麻烦。

比如我们每个人都会遇到的一个常见场景：由于系统剪贴板只能保存一条剪贴记录，复制新内容就会覆盖上一条记录，当你需要粘贴多条内容时，就只能反复地手动复制粘贴、复制粘贴、复制粘贴……

相信很多人还认为，电脑本来就只能这样，机械地重复「复制粘贴」虽然有点费时费力，但好像是一件正常的事情。

像这样的例子还有很多。事实上，你完全可以使用功能更为全面的剪贴板工具，一次性解决所有繁琐的操作。

这里给大家介绍 3 款好用的剪贴板工具。

**Ditto**

![](.evernote-content/AF841688-82C2-408E-A160-020223AD06CF/817F7211-016D-4890-B05E-1AB29CC1BB1C.png)

Ditto 是一款开源的免费 Windows 剪贴板工具，除了基本的剪贴历史记录，还支持分组复制粘贴，快捷键快速粘贴等诸多实用功能。对于只使用 Windows 的用户来说，Ditto 是一个十分不错的选择。

以下两款软件在数码荔枝正版软件商城有售，点击文末「阅读原文」前往了解详情。

**ClipboardFusion**

![](.evernote-content/AF841688-82C2-408E-A160-020223AD06CF/14164CED-C06F-4E9B-951B-63EAFBF66532.jpg)

ClipboardFusion 是一款功能强大的剪贴板工具，支持 Windows 和 macOS 双平台。它可以记录文本、图像、文件路径等信息，支持多种触发器来擦除空格、换行、定位符等你不需要的内容，还可以编写宏来完成复杂的工作流。

ClipboardFusion 还能多平台同步，也有 iOS 和 Android 版本。

**Paste**

![](.evernote-content/AF841688-82C2-408E-A160-020223AD06CF/541C9741-94B1-4851-AF90-013B96348727.jpg)

Paste 是一款颜值高、体验出色的剪贴板工具，只有 macOS 和 iOS 版本。Paste 界面美观，动画效果流畅细腻，宛如苹果原生应用。功能方面，Paste 支持保存无限剪贴板记录，还可以保存常用邮箱、编程代码等固定文本片段，随时取用。

Paste 支持 iCloud，你可以在 Mac 和 iOS 设备间无缝同步剪贴板记录。
