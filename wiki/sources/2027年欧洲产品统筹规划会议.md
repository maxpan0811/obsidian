## 摘要

2027年欧洲产品统筹规划会议（遵义），涵盖产品现状分析（成团率55.6%、产品池过大）、竞对分析（飞扬/盛行/国旅环球）、客户需求升级（东犁退休俱乐部）、产品标准化八段式命名、导游分层、直采酒店资源、招投标规则等10个议题。核心结论：成团率偏低是核心问题，根因是产品+班期供给过度分散，必须建立集团母盘产品体系。

## 正文

### 一、产品现状核心数据

**数据口径**：出团日期 2025.10.1-2026.6.30 | 标品 | 欧洲 | 出发地散拼 | 排除俄罗斯、机票团

| 指标 | 数值 |
|------|------|
| 有效团期总数 | 4,350 |
| 实际成行/截止团 | 2,417 |
| 下线团 | 1,933 |
| 整体成团率 | **55.6%** |
| 下线原因"收客不佳" | 1,886团（占下线97.6%） |
| 实际出行总人数 | 58,356人 |
| 成行团平均人档 | 24.1人 |
| 不足20人的成行团占比 | 26.6% |

**Q4 2025 vs Q2 2026 对比**：

| 区间 | 成团率 | 团均人数 | 截止团 | 下线团 |
|------|--------|---------|--------|--------|
| 2025Q4-2026Q1 | 54.97% | 25.10 | 1,372 | 1,124 |

**区域表现**：
- 🟢 基准以上（Q4-Q1）：南欧、北欧
- 🟡 基准以下（Q4-Q1）：西欧、东欧+巴尔干、英爱、多国联线
- 🟢 基准以上（Q2）：英爱、西欧、南欧
- 🟡 基准以下（Q2）：东欧巴尔干、北欧

### 二、核心问题——产品池+班期池过大

9个月涉及约 **1,167个"公司+线路产品ID"组合**：
- 约 290个：一个团都没成行
- 约 723个：最多只成行1团

**恶性循环**：
> 不断开发新产品 → 创建新产品ID → 排多个班期 → 客流被分流 → 单个产品收客不足 → 大量班期下线 → 认为产品不好卖 → 再开发新产品

### 三、ERP录入规范问题

产品管理混乱实例：意大利西西里+马耳他
- 北京录入"西欧"、上海录入"多国联游"、成都录入"南欧"
- 线路简称：一堆花名/空白/笼统写"英爱""西欧包团"

产品名称混合了：产品等级 + 营销词 + 国家组合 + 景点 + 航司 + 机场代码 + 天数 + 钻级

### 四、2027年产品改进方向

1. **欧惠系列**：集团统筹卖点价格，各地生产（已有西欧板块，待完善北欧、英爱）
2. **中端系列**：各公司统一卖点，多团期多出发地
3. **高端系列**（旅行家·世界志）：对标竞对集团统一生产

**必须建立的四级结构**：
> 产品家族（东南西北欧）→ 标准母产品（法意瑞/奥捷匈）→ 地区版本 → 出发班期

### 五、竞对分析

核心竞对：飞扬、盛行、国旅环球
- 标准化产品模板 + 全国统一采购
- 产品、营销物料高度统一
- 对区域各自为战构成直接竞争压力

### 六、东犁退休俱乐部供应商大会要点

- 供应商被批评"十年未更新线路设计"
- 分餐制：2024每条线至少3顿分餐，2025新增2-3顿
- 督查：全程拍餐、暗访酒店，品质不达标"旅游费赔付+罚款"

### 八、招投标规则

| 规则项 | 具体内容 |
|--------|---------|
| 原标供应商优先 | 同等价格下优先中标 |
| 后返政策优先 | 同等条件下后返更优者优先 |
| 同线路并标 | 相同产品同一标段招标 |
| 二次报价机制 | 新供应商低于原标价5%以上触发 |
| APPLE对APPLE | 司机餐补/进城费/酒店等标准统一 |
| 淡旺季支持 | 旺季优先支持我司的供应商加分 |

### 九、导游分层

- 北京"推一罚十"纯玩产品线（无自费，违约赔10倍）可全集团复用
- 导游按服务水平分级匹配对应产品线（惠品/质品/精品/臻品）

### 十、产研运营核心痛点

> 销端与本地产品经理业绩未挂钩，产品经理缺乏主推跨区域产品的动力
