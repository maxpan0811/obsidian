前段时间，腾讯公司楼下，官方装虾的活动直接冲上了全球热搜，让无数网友哀嚎在深圳出差错过了薅鹅毛的机会。

但别焦虑，在 AI 时代，只要你热点追的够慢，就完全不用再学了！

这不，悲伤还没酝酿开，**腾讯转头就把这只火出圈的虾，直接塞进了人人都用的微信里。**

![](attachments/25cee4d9c0d68d50.jpg)

最近，微信上线 ClawBot 插件，作为腾讯推出的全场景职场 AI 智能体桌面工作台，**腾讯版小龙虾 WorkBuddy 第一时间接入 ClawBot 插件完成了适配。**

用户只需在微信里用一句话描述需求，桌面端的 WorkBuddy 就能像同事一样自主规划和执行复杂任务、操作本地文件，并交付可验收的结果。

不止如此，考虑到大家办公软件的多样性，WorkBuddy 的远程控制功能 Claw，支持接入企业微信、QQ、钉钉、飞书等即时通讯工具，来远程控制电脑上的 WorkBuddy 执行任务。

我们抱着看看鹅厂这次又整了什么好活的心态，实测了整整一周，最终结论只有一个：

鹅厂养虾，手拿把掐！

***01***

**WorkBuddy 装虾，0 代码一分钟搞定**

在 WorkBuddy 出来之前，想玩明白 AI Agent、用上爆火的 OpenClaw，门槛高到能劝退九成普通人。

先不说去 GitHub 找资源、写配置文件，光是科学上网、把界面改成中文，就能让一大批人直接放弃。更别说想把它连到微信、飞书这些日常工具里，没点代码功底根本搞不定。

**但腾讯 WorkBuddy，整个安装配置流程，全程不用写一行代码，会点图标，就能一分钟搞定：**

只要在网页上点击下载 https://www.codebuddy.cn/work/，连续点击下一步，然后在应用中登录微信，桌面端的 WorkBuddy 就安装按成。

接下来，想接入微信？点一下页面左侧的 Claw 设置按钮，选微信客服号集成，再点一下右侧的配置按钮，用微信扫一下弹出的二维码就大功告成。

然后，我们就可以在微信里，和虾聊天了。

![](attachments/4ab2a9a426c68bbb.png)![](attachments/c54e1469c019bb8c.png)

连接飞书、钉钉、QQ，也是一样的流程，点一下配置再扫码就能全平台无缝打通。

整个过程，非常丝滑，不需要一行代码，公司保洁阿姨来了，也能一分钟上手。

不止是连接简单，它直接把**12 大行业、140 + 位细分领域的专家 Agent**，全给你内置好了。我们可以在微信中将其一句话唤起，让 AI 成为我们 24 小时工作的电子牛马。

![](attachments/8ac17766bb147f48.png)

比如，我让它帮我找到小红书最合适的旅行类博主，只需要输入：调用小红书运营专家 agent，搜索小红书"旅行小窍门"相关的热门笔记，从中找出 5 位不同的作者，逐一查看他们的用户主页和历史笔记，然后帮我整理一份达人评估表，包含：昵称、粉丝量级、近期笔记互动数据均值、内容风格描述、适合合作的内容方向，并按性价比推荐优先级排序。

![](attachments/e10ec596c59a9734.png)

它就能自动调用专家，思考需要调用的工具，并给出非常完整的交付结果，连合作优先级都给直接排好了。

![](attachments/ab304646c6133767.png)![](attachments/baea21acb2421e0b.png)

想做内容、搞设计、算数据、做运营，也是一样，从 UI 设计师、增长黑客到招聘专家、销售教练，各行各业的电子同事，借助 WorkBuddy，我们在微信一句话就能召唤。

除了已有的 agent，我们也可以直接安排一些随机任务，WorkBuddy 会自行思考应该如何完成以及如何调用不同的工具。我们来做一个简单的测试，让它查询了最新的金价波动。

![](attachments/9700555b7614bd1f.png)

可以看到，AI 不仅给了文字版结果，还给了我一个 html 文件，从而可视化的看到具体的交易情况，价格走势、最高价、最低价，每日涨跌幅。（每一个点都是心碎的声音）

![](attachments/7dea0e1b1ce628df.png)

**当然，这些功能只是小试牛刀，没有完全发挥出 WorkBuddy 的全部实力。**

接下来测试中，我决定将难度升级。

***02***

**WorkBuddy 自动搭建 LOF 套利 agent，**

**再也不用手动盘数据**

如果只是做个信息整理、内容生成，那未免太小看腾讯的养虾能力了。

我们直接上了硬核难度：用一句话，让它搭一个能自动盯盘、定时推送的 LOF 基金套利 Agent，低风险赚钱。

先给小白科普下这个任务的逻辑：LOF 基金同时支持场外申购（如银行、基金公司等）和场内交易（证券交易所像股票一样实时买卖），当场内价格与场外净值产生差值，就出现了溢价或者折价套利的机会。比如场外更便宜时，我们就场外按净值申购，转到场内高价卖出，差价扣掉手续费就是纯收益。

**（提醒：由于场外交易采用的是 T+2 交易，场内是实时交易，会产生一个至少两天的时间差，期间可能发生暴涨暴跌。因此，需要我们精准筛选出场内外差值够大，安全边际够高的 LOF。股市有风险，且大模型还没有百分百消除幻觉，涉及重要信息大家注意核实，谨慎操作。）**

但问题是，国内有 300 多只可套利的 LOF 基金，每天手动盘点场内外差价、算溢价率、分析行情，费时又费力，普通人根本盯不过来。

于是我们给它提了需求：「帮我建立一个 ETF 薅羊毛 agent，每天晚上九点，以表格形式，告诉我场内外基金有哪些差价超过 10% 了」。

![](attachments/b0f6872ee8542de1.png)

注意，这里我们故意埋了个坑：能做场内外套利的是 LOF 基金，不是 ETF（ETF 有另一套套利机制，但根本不适合普通人操作）。

但 WorkBuddy 直接精准识别了这个错误，不仅没顺着我们的错误走，反而直接交付了一套完整的 LOF 基金差价分析 Agent，连细节都拉满了：能自动筛选标的外，区分溢价 / 折价两套套利策略；并且以表格形式输出基金名称、代码、场内价格、场外净值、溢价率、套利方向，连操作建议都给写明白了。

![](attachments/c58a0deb0d7d3e90.png)

整个过程，比较惊喜的一点是相比原版 OpenClaw 需要自己写大量**配置 skills 文件，WorkBuddy 直接把常用的 skills 放进了广场里，只要点一下，就能自动配置好了。**

![](attachments/3ffe47aaf36692f3.png)

甚至，在创建 agent 或者自动任务的时候，模型会自动分析完成任务需要哪些 skills，然后自动调用金融数据技能、安装所需工具，不用手动配置任何环境。

更绝的是，考虑到 agent 搭建通常很难一次性达成想要的效果，你还能无限迭代这个 Agent。

我们追加了需求：给每个基金加成分股分析、市场行情解读、券商观点，并且把时间改成每天晚上九点 25，还要每天把结果推送到微信。

![](attachments/b6f969e7e405c1ad.png)![](attachments/78caebeb1ca3edf0.png)

它二话不说，直接读取现有脚本，对接了基金持仓、券商研报的 API，加上了微信推送功能，把定时任务改了时间。

此外，比较值得称道的一点是，**WorkBuddy 几乎集成了国内所有主流模型。**

之前，不少模型企业也推出了自己的 claw 类产品，的确也好用，但是一来不支持微信，二来只能选择某一家的模型产品，不够灵活。

而在 WorkBuddy，我们可以基于丰富的模型选择，这根据任务不同，选择更适合的模型，避免套利不成，token 消耗先爆炸了。

![](attachments/c7fe858cd1c7e6b5.png)

**而且，WorkBuddy 针对国内版用户，无论新老，都一次性赠送 5,000 Credits**https://www.codebuddy.cn/profile/usage

我算了一下，在 WorkBuddy，即使选用最先进的模型，简单任务只消耗个位数 Credits，一些复杂任务消耗也基本控制在十几 Credits，5000Credits 可以够用个把月了，性价相当比不错。

![](attachments/2c5f639fb8809c19.png)

只是，设计定时任务当然还不够，我还需要随时随地可视化的知道最新的 lof 基金溢价情况，操作也很简单，微信上说一声就好。

![](attachments/d33e9f1b351339c0.png)![](attachments/8286fe003221a111.png)

最终，AI 不仅以文字的格式告诉了我具体的套利产品与细节，同时附带了可视化 HTML，如何选择一目了然。

在最后一列，AI 还贴心给出了分析结果，比如哪些产品适合溢价套利，哪些产品适合折价套利，这一点可以说非常贴心了。

过去，搭建这样一套系统，几乎是量化公司或者代码高手才能完成的操作，现在，借助 WorkBuddy，几乎人人都能上手。

***03***

**让 AI 帮我统计发票，**

**月底不再为报销发愁**

如果说基金套利是少数人的进阶玩法，那发票整理这个功能，就是能让所有职场人直接喊救命的刚需。

作为内容从业者，我们同事要大量出差、跑采访，一个月下来，打车、吃饭、高铁、住宿的发票能攒几十甚至上百张，PDF、图片格式混着来，高铁票和普通发票格式完全不一样，要挨个打开、录信息、做表格、核对金额，光是整理完就要花大半天，一个小数点错了就要被财务打回返工。

现在，这些脏活累活，全交给 WorkBuddy 就行。

我们直接甩给它一个装了 45 张各类发票的文件夹，提了一句：整理桌面上发票文件夹中的所有发票，按照开票方、销售方、发票类别、金额等信息，整理到一个表格中发给我。

![](attachments/ed1dda8bd7605c48.png)

接下来就是全程无需插手的自动化处理：

它先自动读取了文件夹里的所有文件，识别出 45 个 PDF 发票；然后自动编写 Python 脚本批量提取信息，过程中发现铁路电子客票格式不匹配、正则匹配出错的问题，全程自主迭代、自主修复；

可以看到，其中，还混入了大量图片格式、以及与传统发票格式天差地别的高铁票，但 AI 依然精准的发现了其中问题，并自动提出改进方式。

最终交付的文件如下，文件名、销售方、发票类别、金额、开票日期、发票号码，所有信息分毫不差，连混在里面的图片格式发票、高铁票都实现了精准识别。

![](attachments/c2a0343a39b27b9b.png)

以前要花一整天搞定的报销，现在一句话，几分钟就完事了。

再也不用对着一堆发票熬大夜，再也不用因为小数点后两位的错误跟财务反复 battle，职场人的幸福感，直接被这只虾拉满了。

***04***

**结尾**

实测下来，WorkBuddy 如果满分十分，我的个人评价可以给到 8.5 分。

之所以有这么高分，核心是它精准解决了目前市面上 Claw 类产品的两大痛点：

**第一，全平台零门槛打通，微信原生适配**。这是最杀的一点。别的同类产品，要么不支持微信，要么接入流程极其复杂，而 WorkBuddy 不仅第一时间适配了微信刚上线的 ClawBot 插件，还做到了扫码 1 分钟绑定，语音指令、传文件、远程操控，微信里直接就能完成，真正做到了 随时随地用 AI 干活。

对比原版 OpenClaw 需要自己写大量 skills 配置文件，WorkBuddy 直接把常用技能做成了开放广场，GIF 搜索、GitHub 管理、PDF 编辑、AI 绘图、邮件管理…… 上百个技能，点一下就自动配置，甚至模型会自动分析任务需要什么技能，自主完成配齐，小白也能零门槛上手。

第二，**全主流模型适配，性价比拉满**。很多同类产品，只能绑定自家的大模型，灵活性极差，而 WorkBuddy 不仅有免费的 5000 Credits，还几乎集成了国内所有主流大模型，GLM、Kimi、MiniMax、DeepSeek 全覆盖，你可以根据任务难度选最合适的模型，把成本控到最低。

再加上鹅厂官方出品，对比各种第三方工具，在数据安全、本地文件读取的权限管控上，又多了一层保障，尤其是职场人处理公司文件、财务数据，安全感直接拉满。

当然，实测下来也有一些美中不足：比如，偶尔会出现本地文件连续读取失败的情况；跨应用授权的流程还有优化空间，初次跨应用调用的时候，需要不断手动确认授权，不然可能任务失败。

但这些小问题，在它带来的效率提升面前，几乎可以忽略不计。毕竟回顾过去一个季度**，你就会知道，科技进步的速度，尤其是国内厂商的产品化能力，强的令人发指，眼前的小问题，可能不到一礼拜，就能被迎刃而解。**

一个多月前的春节，身边还有不少同事，为了装个 OpenClaw，从注册 GitHub 账号开始，折腾了七八天都没搞定；而现在，只需要点几下下载、扫个码，就算是完全不懂代码的财务、前台，都能轻松搭出自动化办公应用，搞定办公用品管理、报销统计这些重复工作。

曾经只属于技术极客的 AI Agent，现在被腾讯用极致的产品化能力，送到了每一个普通职场人的手里。

同事之间少了推诿扯皮，每天还会互相案例好用的 skills 与搭建小窍门，同事之间关系，都比过去更亲密了。科技万岁，其实就是从这些微小的细节中体现的。
