Windows 10 更新十月版，你要的云剪贴板、截图标注、防勒索都来了
====================================

22小时前

[![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/7C5D0C9C-DEE1-4F11-B214-FFACB8040836.jpg)](https://sspai.com/user/706656)

#### [沨沄极客](https://sspai.com/user/706656)

目录

微软在 10 月 3 日举行了新品发布会，在发布 Surface Pro 6 等新品的同时还正式发布了 Windows 10 更新十月版（October 2018 Update），版本号是 Build 17763，Windows 10 版本 1809。

此次更新现已正式上线，自动更新推送将于 10 月 9 日开始。如果你现在就想升级到 1809，只需手动到 Windows 的更新中检查一下，今天就可以升级到最新的 Windows 10 更新十月版了。

值得关注的 Windows 重大更新
------------------

这次更新新增了全新云剪贴板、157 个 Emoji、新屏幕截图工具，带来了黑暗主题的文件资源管理器、支持防勒索的 Windows 安全中心等一系列重磅更新，以及记事本更新、蓝牙设备电量显示、磁贴文件夹命名等令人惊喜的小更新。下面就来详细讲讲更新的内容吧。

### 黑色主题的文件管理器来了

在这个新版本中，Windows 10 为文件资源管理器增加了一个全新的黑色主题。包括右键菜单在内，整个文件资源管理器都变成了黑色风格。喜欢晚上使用电脑的你，不用再面对刺眼的白色界面了。

你可以在「设置」 → 「个性化」 → 「颜色」中启用黑色主题的文件管理器。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/A9DC1176-C246-4AC0-A780-E7B8EC69FD7E.png)

### 属于 Windows 的全新云剪贴板

本次更新中的另一个大更新是剪贴板功能。剪贴板更新可以分为两个方面，一是**本地剪贴板的历史记录功能**，另一个是**云剪贴板**。

你现在可以用 `Win + V` 组合键打开**剪贴板历史记录**，最近复制到剪贴板的内容都会显示出来，还支持固定常用的文字。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/A1C1D94E-FF59-49EB-AE8F-F765431DBF16.png)

云剪贴板和字面意思一样，可以在两台设备上同步显示剪贴板文字。这需要在两台 Windows 设备上登录同一微软账号，并开启云剪贴板功能。

Windows 10 在「设置」 → 「系统」中新增了一个「剪贴板」设置界面，在这里可以选择是否启用历史记录、是否启用云剪贴板、手动或自动同步、清除历史记录等。

「在家按 `Ctrl + C`，再到公司按 `Ctrl + V`」终于不再是一个段子了。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/44113F0C-A314-4FD5-89CC-0275558921C0.png)

### 全新的屏幕截图工具「截图和草图」

这次更新还解决了另一个饱受人诟病的问题，那就是 Windows 的截图功能。

以往的 Windows 截图功能羸弱，只能用 PrintScreen 键截取全屏，或是在 Windows 附件中找到截图工具。相信有不少人都习惯用 QQ 的 `Ctrl + Alt + A` 进行截图。如今微软终于开始解决这个问题了，截图功能在这次更新中如获新生。

这次更新新增了一个名为「**截图和草图**」（Screen Sketch）的截图编辑工具。实际上这个功能以前内置在 Windows Ink Workspace 中，但这个针对平板用户开发的功能，对于 PC 和笔记本用户而言实在是太隐蔽了。这个「截图和草图」把以往的截图工具和标注工具结合在了一起。

现在按下 `Win + Shift + S` 组合键，可以进入自由截图模式。支持框选截图、全屏截图、自由截图，截取完毕后自动存到剪贴板内。右下角会同步显示通知，如果点击通知还可以对它进行标注和编辑。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/787DE920-4E31-4B2C-8D94-0EE7331B410D.gif)

### 更强大的搜索功能

开始菜单中的「搜索」更新了**全新界面**、**应用搜索**功能。

搜索方法跟以前一样，不过现在不用打开浏览器，就可以在更大的扩展栏侧边栏中看到搜索结果了。

如果搜索结果中包含有应用，还会显示来自 Microsoft Store 的应用下载链接。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/22C2DEC5-5F8B-4C76-8BC4-58C67F5BE346.png)

### 系统内置中文输入法更新

相信有不少人在使用 Windows 10 系统自带的输入法。这一次，中文输入法也获得了重磅更新。

现在 Windows 的自带中文输入法增加了**新的 IME 工具栏**、**特殊表情面板**、**黑色主题**。

输入法 IME 工具栏相信大家并不陌生，可以在工具栏上用鼠标修改中英文、全角半角、中英文符号、简体繁体切换等诸多实用功能。

在 Windows 中如何快捷的输入 Emoji 一直是个不小的难题。本次更新中新增的特殊表情面板，让自带输入法也支持**直接输入 Emoji、颜文字、特殊字符**，甚至可以直接搜索**网络表情**。你可以用 `Win + .` 或 `Win + ;` 组合键来打开特殊表情面板。

输入法工具栏需要在「设置 」→ 「输入法」 → 「中文（中国）」→「 外观 」中开启。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/1AF75ACA-C5F7-480C-AC13-0D5680D52E17.png)

### 新增了 157 个 Emoji

不仅中文输入法获得了更新，1809 中新增了对 Unicode 11 的支持，其中最为人们熟知的可能就是 Emoji 表情了，这次新增了包括「袋鼠、龙虾、月饼、肥皂」等 157 个 Emoji。

你可以用 `Win + .` 组合键在表情面板看到它们。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/52711915-482E-4F1C-A944-FF5801ADB13E.png)

### 更棒的 OneDrive 体验

如今越来越多的电脑装备上了 SSD，硬盘读写速度得到了明显提升。不过相对较低的硬盘容量也带来了不少问题。把文件放到云端上是个不错的选择。不过对于 OneDrive 这类主打同步而非存储功能的网盘而言，很多时候并不能解决**本地存储空间紧张**的问题。

不过好在微软发现了这个需求，在新版本的 OneDrive 中，你可以把某个文件夹设置为「仅在线可用」，来实现释放本地空间的效果。现在你可以把文件转移到 OneDrive 目录里，然后 OneDrive 会把这些文件上传到云端。你在本地依然可以看到文件的列表。但是只有在联网后才能打开文件。

当然，你也可以像以前一样设置为「始终保存在此设备上」，那文件就会始终存储在本地。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/A6D0E14C-8E0F-413C-9F85-E0FC1BCADE54.png)

### 任务管理器新增电池视图

对笔记本用户而言，续航时间是非常重要的。但 Windows 有很多服务会在后台运行，有些服务静默运行但是耗电量巨大。更新之后你就不用担心这个问题了，新版本 Windows 可以在任务管理器中看到**电池使用**、**电池使用趋势**两个新增列。

* 电源使用：显示应用和服务的即时使用电量。
* 电源使用趋势：根据两分钟内的电量使用情况来推算电源使用趋势。

增加了这个功能后，Chrome 这种耗电量大户的日子恐怕是不好过了……

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/0CA88D74-B93A-49F6-AE4C-C052E5278E5F.png)

### 支持系统字体大小调整

字体方面的更新，这次更新依然没有允许用户直接更换系统字体，但是增加了**系统文本字体大小调整**功能。以前想做到这一点，通常需要调整系统的缩放分辨率，或是采用第三方工具才能做到。现在可以直接在设置中统一调整系统字体大小，这项设置同时适用于 UWP 应用和 Win 32 程序。

### 界面更美，更多的 Fluent Design UI

微软在 Redstone 4 开始尝试采用 Fluent Design 之后，收获了大量的好评。现在 Windows 开始为更多系统应用添加 Fluent 的效果。这次适配了标题栏、时间线视图、Windows Defender 界面等多个界面。

相比最早的 Windows 10，加上了 Fluent Design 之后的效果确实惊艳不少。希望微软能继续把这一设计标准发扬光大。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/E9C13EEC-D14C-4028-B2AC-CF69C2144815.gif)

更安全的 Windows 安全中心
-----------------

从这个版本开始，「Windows Defender 安全中心」将被重新命名为「Windows 安全中心」。

### 勒索软件防护

几次勒索软件风波中，只要你及时更新到最新的 Windows 系统，基本上就不会有中招的风险。这一次，Windows 安全中心中增加了一项勒索软件防护（Windows Application Guard）设置，可以设置一部分文件夹为重要文件夹，阻止勒索软件和病毒木马对重要文件进行攻击。即便被攻击也可以将文件恢复。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/CF79A0D3-2AD1-4649-9EBD-75DE4744030C.png)

### 严格管理第三方安全软件

更新后的 Windows 安全中心将会更严格地管理第三方杀毒软件（包括杀毒、防火墙、Web 保护软件），保障电脑安全。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/DE87032F-AB06-487A-8EC5-EFEB53D50D57.png)

### WSL 也能用的防火墙

现在防火墙还支持对 Windows 子系统 Linux （WSL）进程进行管理。当 Linux 应用想从外部访问端口时，Windows Defender 防火墙会发出提示。

Edge 获得小幅更新
-----------

随着 Windows 推出新版本，Edge 也获得了不少更新。现在 Edge 有了**新的设置菜单**、**增加对 PDF 的支持**、**增强安全工具**。

### 新的设置菜单，允许自定义工具条

设置界面经过了重新设计，分为「常规」、「隐私和安全」、「密码自动填充」、「高级」四个部分。高级设置中允许用户关闭网站自动播放、调整网站权限。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/314EEBF6-2CCC-4DA3-99C7-983F3075B9C4.png)

### 增强版的 PDF 支持

现在 Edge 中可以对 PDF 进行更多操作了。PDF 工具栏得到了更新。

如果将 Edge 设为打开 PDF 的默认应用时，文件管理器中有了一个新的 PDF 图标。如果用 Edge 打开本地 PDF，也会一起显示在时间轴和 Edge 的历史记录中。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/6B6DECB0-8C7E-4162-B546-0718DED4769E.png)

### Edge 安全性更新

现在 Edge 整合了用户验证支持，允许用户使用 Windows Hello、外部验证工具，来完成自动登录等操作。说简单一点，如果你的电脑配有指纹识别、面部识别等功能，在 Edge 中登录会变得更加方便。

小功能更新
-----

这次的重磅更新很多，小功能改进也不少，这就来看看 1809 有那些小功能的更新和调整吧。

### 蓝牙设备终于可以显示电量了

在 iOS 设备上，我们可以看到与 iPhone 连接的**蓝牙设备剩余电量**。如今在 Windows 上也终于提供了这项功能。

在「设置」 →「设备」→「 蓝牙与其他设备」中可以查看。（查看剩余电量功能需要蓝牙设备支持）

### 新增查看流量功能

在「网络」的设置界面中，现在提供了详细的「数据使用情况」，当你使用 SIM 卡时会自动记录流量使用量。妈妈再也不用担心我用超流量了。

### 「开始」菜单允许对磁贴文件夹命名

在几个版本以前，「开始」界面就已经允许用户创建磁贴文件夹了。现在 Windows 允许用户对这些文件夹进行命名了。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/7ECF2893-9FF0-41C5-967D-EC3D4680EC79.gif)

### 新增 HEIF 格式支持

本次更新中引入了对 HEIF 图片格式的支持，现在可以在文件管理器中直接对 HEIF 格式的图片进行旋转、编辑元数据等操作，而不需要安装第三方软件了。

如果想要编辑，在文件资源管理器中右键点击 HEIF 文件，菜单中会显示「向右旋转」「向左旋转」等操作。「右键 」→「 属性」→「详细信息」可以编辑元数据。

### 游戏栏和游戏模式新设计

新版本的**游戏栏**，变得更大、更漂亮、更易于使用了。

这个界面上增加了对声音的操作。玩家可以快速完成「更改音频输出设备」「静音」等操作。终于不用担心戴耳机玩游戏的时候，声音却从音响里出来的尴尬了。

你可以用 `Win + G` 打开游戏栏。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/416A3161-793B-46D7-92DA-A839F1F0A3F8.png)

### 记事本小幅更新

记事本原先仅支持 Windows 的行结尾（CRLF），现在支持 Unix / Linux 行结尾（LF）和 Mac 行结尾（CR），并会将其显示在右下角。这项更新对进行跨系统编程的程序员朋友很有意义。此次更新还新增了记事本内 Bing 搜索功能，总体而言更新幅度并不大。

如果想了解更多，可以在 [微软官网 Blog](https://aka.ms/notepadeol) 找到更多关于记事本更新的详细信息。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/E126EF5E-EE1C-4D42-B585-B6E44754229D.png)

### 「投影到这台电脑」功能更新

投影功能可以让我们把一台电脑的屏幕分享到另一个设备上。现在这个模式允许用户按需求修改投影质量了。

1. 游戏模式：尽可能地减少屏幕延迟，方便玩家通过无线网进行游戏。
2. 视频模式：确保大屏幕上的视频播放顺畅，可能会少量增加屏幕延迟。
3. 生产力模式：是上面两个模式的平衡模式，保证屏幕延迟正常，打字更自然。

### 显示模式中可调整 HDR 模式

在「显示」设置中，有一个新的 **Windows HD Color** 页面，可以控制内容的高动态范围（HDR）和宽色域（WCG）设置。例如支持的显示器上的照片，视频，游戏和应用程序。（需要支持 HDR 的外部设备支持）

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/A285F8FD-C5B7-4122-B49E-C82A24996774.png)

### 地区和语言设置界面拆分为两个

原先合并在一起的「地区和语言」界面，现在分为两个界面了。在语言设置中新增了 Microsoft Store 的语言包下载链接，可以下载不同地区的语言包。

### 可以分用户安装字体了

现在 Windows 安装字体默认只为当前用户安装（以往是为整台电脑的所有用户安装字体）。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/57292C41-B393-447B-82EA-70A69A76034C.png)

### Windows 更新策略改变

如果谈到系统更新，Windows 的更新策略总是被人吐槽的最多的。

这一次 Windows 也做了一些改变。升级了更新策略，利用机器学习来预测什么时候安装、重启设备更合理。

其他有意思的小更新
---------

* 在 Windows 10 专业工作站版（Windows 10 Pro For Workstations）中，新增了一项名为「终极性能模式」电源计划方案，不支持电池供电的设备。
* Windows Ink 体验优化，可以直接在部分应用的搜索框中手写输入。
* 日历应用更新搜索功能，可以搜索活动名称、地点、人物。目前仅限微软账户。
* 控制面板中取消亮度调整。不过不用担心，你可以在「设置」中调整亮度。
* 针对英文输入的 Typing insights（输入预测）与自动更正功能。
* 时间线视图增加了对闰秒的支持。
* Windows MR（混合现实）获得小幅度更新。
* 安装系统时的流程有部分变化，主要帮助用户启用时间线等功能。

![](.evernote-content/E0E4E3F4-26DC-4912-805C-EE1009274F12/4635DEB4-F3DD-499F-84A2-CE88660BA96D.png)

总结
--

总体而言，此次 Windows 10 更新十月版的亮点不少，用户呼声已久的「云剪贴板、截图功能、字体大小」等问题都得到了重点解决。诸如「蓝牙设备电量、流量监控、磁贴文件夹命名」这一类的升级也很切痛点。还加入了输入法表情包这种有趣的小更新。

在我看来，微软在彻底放弃 Windows Phone 之后，已经将工作重心转移到 Windows 的优化整合上，原先积累的许多问题都在积极地解决，在十月更新之后还有可能添加全局标签页系统。从春季更新到十月更新，只不过短短 6 个月的时间，微软就做出了更新幅度不小的调整。很期待 Windows 的后续更新。
