ERP供应商预付功能上线

各位好，ERP供应商预付款功能已上线。本次预付款功能可代替原先必须**新建虚拟团**来完成供应商付款的所有场景

对应场景（包含但不限于以下场景）

| 资源类型 | 付款类型 |
| 机票 | 定金、押金、退票款 |
| 电商 | 佣金 |
| 备选资源 | 预付款 |
| 保险 | 预付款 |
| 门票 | 预付款 |
| 地接 | 酒店包房预付 |
| 签证 | 代送签 |

从今日起（2020.3.6），请各相关单位使用该功能，同时虚拟团也即将关闭。
