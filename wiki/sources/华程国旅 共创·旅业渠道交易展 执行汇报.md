# 华程国旅共创·旅业渠道交易展执行汇报

> 旅业资讯主办，华程是连续多年冠名/特约方。展会已办十二届，专注旅行社行业获客与分销。

## 展会概况

- 前身：全国旅行社渠道运营商年会（2013年起）
- 定位：国内专注旅行社行业获客、深度分销的专项展会
- 形式：论坛+展会联动
- 连接：全国优质组团社、门店、定制机构、细分服务商

## 历年主题

| 年份 | 主题 |
|:----:|------|
| 2013 | 渠道运营商年会（首次） |
| 2015 | 全网营销下获客新战役 |
| 2017 | 掌控旅游分销新局面 |
| 2023-2025 | 共创·旅业渠道交易展 |
