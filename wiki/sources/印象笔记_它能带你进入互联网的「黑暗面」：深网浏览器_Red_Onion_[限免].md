# 它能带你进入互联网的「黑暗面」：深网浏览器 Red Onion [限免]

---

![](.evernote-content/556E7640-4AE0-45C6-844A-292108F74105/01CAED32-EE99-4B2F-9D12-587AC3DAEC06.jpg)

这两年来，Netflix 的《纸牌屋》在让大家知道了不少美国政治幕后的同时，也把「Deep Web」也即[「深网」](https://zh.wikipedia.org/zh-sg/%E6%B7%B1%E7%BD%91)的概念带入了大家的视线里。

> 96%的互联网数据无法通过标准搜索引擎访问，其中大部分属于无用信息，但那上面有一切东西，比特币洗钱、LSD、赏金黑客…… —— 《纸牌屋》

对大多数人来说，「深网」可能是他们一辈子都不会接触到的东西，但这个世界却又真实存在，就像是我们日常所见的互联网的影子一样。

很多文章都把基于 Tor[1](http://sspai.com/33384#fn1) 技术的「深网」形容的像是一个可怕而又怪诞的世界，有着大量突破日常的东西，但其实当你真正的去了解它，你会发现，**它其实更像是 90 年代的早期互联网 ——**有趣的内容与不合时宜的内容混在一起，需要你自行去分辨。

在这里你可以找到各种音乐共享、轻小说分享、甚至是 BT 资源，当然也会踩到各种「雷」。而其匿名的特性则更是实现了那句早期互联网的名言 —— 「**你永远不知道和你聊天的是不是一只狗**」。

相信有不少人已经在自己的电脑上尝试过通过「洋葱路由」来探索这个奇妙的世界，但安装「洋葱路由」终归是一件麻烦的事情，且只能在电脑上进行。而现在，我们有了一个更傻瓜式的选择来一窥「深网」的究竟 —— iOS 深网浏览器 Red Onion。

简单上手
----

打开 Red Onion 之前，我们还需要准备一样东西才能让我们正常的使用它。是的，由于 451 代码在我们于国内使用国外服务时太过常见，而 Red Onion 不幸又是把部分服务代码放在自己的外国服务器上的，所以我们需要准备一把结实的「梯子」。

在架好「梯子」之后，我们就可以正式开始使用 Red Onion 了。初次打开 Red Onion 会弹出一个密码设置界面，要求你设置一个4位的数字密码，作为一个**以匿名与隐私为首要目标**的浏览器，它会在你每次重新打开或从后台切换回 Red Onion 时弹出要求你进行解锁，对于有 ID Touch 的 iPhone 则会在解锁时默认优先使用指纹解锁。

![](.evernote-content/556E7640-4AE0-45C6-844A-292108F74105/58FAF500-214C-4741-ABDB-99714D3F42A3.jpg)

![](.evernote-content/556E7640-4AE0-45C6-844A-292108F74105/DCFB4A7B-0691-4F87-9C30-4AA86B4D5A89.jpg)

在设置完密码之后，便会自动进入「Tor」也即是洋葱网络的配置进程中，耐心等待一段时间之后你就会看到一个配置成功的反馈界面。恭喜你，这下你就可以正式开始「深网」的探索之旅了。

![](.evernote-content/556E7640-4AE0-45C6-844A-292108F74105/B476EE56-3360-4EFB-ACA9-750B8253A9CF.jpg)

![](.evernote-content/556E7640-4AE0-45C6-844A-292108F74105/B092C139-B14B-439D-A3F6-A03712EA3FBD.jpg)

Red Onion 采用的是类似 [iCab 浏览器](http://sspai.com/32954) 的界面设计，非常简陋，包含菜单、下载文件、标签页三个按钮。不过令人意外的是，它竟然支持左右滑动切换标签页，在这点上出人意料的人性化。

OK，对于基础功能你已经全部了解完毕，Let's party like it's 1999。

探索 Dark Side
------------

「深网」之所以被称为「深网」就在于它的隐蔽性，那么，我们想要找到一些这样的网站应该从哪里开始呢？

对大多数人来说，「The Hidden wiki」是个不错的开始，它就像是「深网」的 hao123 ，你在上面能找到不少的网站来开始自己的探索之旅。它在表层网络就有一个可直接访问的页面，你可以通过搜索引擎直接找到它。

对于部分人来说，这样简陋的导航网站已经无法满足，搜索引擎或许更合适，那么，这里同样也有一个支持中文搜索的「深网」搜索引擎 —— [Not Evil](http://hss3uro2hsxfogfq.onion/)[2](http://sspai.com/33384#fn2)（不作恶）。

无论是我们平时的表层互联网，还是「深网」，有一类网站是绝对不会绝迹的，那就是钓鱼网站，而「深网」中缺乏监管，所以更要小心，**不要让自己成了被好奇心害死的猫**。

其他作用
----

对于大多数人来说，访问「深网」网站也只是一时为了满足自己的好奇心，除此之外，Red Onion 还有什么其他的用途吗？

### **隐藏你的 IP 地址，保护你的隐私**

其实这才是 Red Onion 最核心的功能，作为一个基于「Tor」技术的浏览器，它可以极大的保护你的 IP 地址不被锁定，保证你在网络访问时的匿名性，减少你被恶意网站追踪的可能性。

### **屏蔽广告**

Red Onion 考虑到由于「Tor」技术会导致网页访问速度回相对较慢，故而内置了网页广告过滤功能，节省流量的同时还你一个纯净的阅读环境。

从 Red Onion 近期的版本迭代来看，它有着向 [iCab](http://sspai.com/32954) 靠拢的趋势，功能有很多值得把玩一番的地方，再加上它的「深网」入口的身份，为何不来试一试它呢？祝你在「深网」找到探索早期互联网的愉悦。
