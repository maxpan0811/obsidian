# 买买买 ！值得购买的五款iPad Pro外接键盘

---

[![](.evernote-content/0A25CD17-6EAD-4180-9390-90C617D6D7D7/0F49A6FB-287C-4B3F-A293-E91D0EB8BC93.jpg)](http://www.feng.com/iPhone/news/2015-12-28/Buy-buy-buy-_634388.shtml)

　三个月前，苹果推出了旗下首款大屏平板电脑产品 iPad Pro，其屏幕尺寸为 12.9 英寸，分辨率为 2732×2048，像素数量超过 560 万。库克曾经说过，iPad Pro 是可以取代传统的 PC 的生产力设备。  
  
　为了配合新 iPad Pro，除了内置高精度传感器的 Apple Pencil 之外，苹果还给这款产品量身打造了一款外接键盘 Smart Keyboard。当然，我们也可以选择第三方厂商推出的外接键盘产品，比如外媒 PhoneArena 推荐的这几款。  
  
　**1. Zagg Messenger Universal**

![](.evernote-content/0A25CD17-6EAD-4180-9390-90C617D6D7D7/812CBDD7-B511-46B9-848B-F8C3F9902A60.jpg)

外媒 PhoneArena 首先介绍的第一款 iPad Pro 外接键盘是非常经典的 Zagg Messenger Universal 系列，这一系列的 iPad Pro 版本采用了类似 Smart Case 的造型设计，拥有非常出色的便携性，不过，键盘设计方面也因此变得紧凑，其市场售价为 70 美元(约合人民币 447 元)。

**2. 罗技 Create Backlit**

![](.evernote-content/0A25CD17-6EAD-4180-9390-90C617D6D7D7/0F49A6FB-287C-4B3F-A293-E91D0EB8BC93.jpg)

Create Backlit 是罗技公司为 iPad Pro 专门打造的一款外接键盘，采用了磁力链接的设计，其最大的亮点在于它允许 iPad Pro 开机就能自动配对，完全不需要考虑连接方面的操作。虽然这款产品使用的是全尺寸的按键设计，但是便携性并未受到影响。另外，这款罗技 Create Backlit 还拥有防水性能。它的售价为 149 美元(约合人民币 950 元)。

**3. Zagg Slim Book Pro**

![](.evernote-content/0A25CD17-6EAD-4180-9390-90C617D6D7D7/3B29FDF7-5EA5-44EA-9722-BC0DF3F367EB.jpg)

Slim Book Pro 跟 Messenger Universal 一样是 Zagg 推出的外接键盘产品，它采用的是可调整的连接转轴设计，最大角度为 135 度。最重要的是，这款产品的内部配有电池，能够大大提升产品的续航能力。Slim Book Pro 在尺寸大小和手感方面给人的感觉非常像是笔记本的键盘，用户不必担心用起来不习惯。它的市场售价比 Messenger Universal 要贵一些，为 139 美元(约合人民币 888 元)。

**4. IVSO iPad Pro 保护壳**

![](.evernote-content/0A25CD17-6EAD-4180-9390-90C617D6D7D7/E6DEA890-106B-4410-9884-5E22EF9D68E6.jpg)

IVSO iPad Pro 保护壳是外媒 PhoneArena 推荐的最后一款 iPad Pro 第三方外接键盘产品，因为聚氨酯皮革材质较为便宜的关系，这款产品的售价也是最便宜的，仅为 19 美元(约合人民币 121 元)。

虽然成本不高，但是这款 IVSO iPad Pro 保护壳的手感非常柔软，它对 iPad Pro 的保护性一点也没有打折扣。设计方面，这款产品没有覆盖 iPad Pro 本身的任何一个接口，并向用户提供支架、蓝牙 3.0 和非常独特的手提杆功能。另外，内置的可充电电池能够提供 3 个小时的续航时间，将 iPad Pro 的工作时间延长。

**5. 苹果官方 Smart Keyboard**

![](.evernote-content/0A25CD17-6EAD-4180-9390-90C617D6D7D7/BA39F726-ACD8-49C5-8D25-1DA32C5568E1.jpg)

最后，我们来介绍一下苹果官方推出的 Smart Keyboard 外接键盘，以便 iPad Pro 在选择第三方外接键盘的时候能够有个参考标准。总的来说，这款售价为 169 元(约合人民币 1080 元)的外接键盘让 iPad Pro 更添灵活性。

苹果表示，为了打造像 Smart Keyboard 这样纤薄耐用的设备，他们只能在一张纤薄如纸的尼龙上采用金属蚀刻的方式来创造出独特的可传导材质，让电能与数据可以双向传输。而位于 Smart Keyboard 的连接边缘的 Smart Conector 让键盘与 iPad Pro 紧密相连，运行顺畅无阻。
