# Part 1: What Is Claude Code?

# 第一部分：什么是 Claude Code？

Claude Code is a tool that runs on your computer. Don’t be intimidated by the word “code.” You don’t need any coding or programming skills to use it.Claude Code 是一款在你的电脑上运行的工具。不要被“Code（代码）”这个词吓到，你完全不需要任何编程技能就能使用它。You install Claude Code on your computer just like you would an app like Zoom or Zotero.你可以像安装 Zoom 或 Zotero 这样的应用程序一样，将 Claude Code 安装在你的电脑上。Once you have it on your computer, you open your project folder (dissertation, paper, etc.) and let Claude work inside the folder.安装完成后，打开你的项目文件夹（如学位论文、论文草稿等），让 Claude 在这个文件夹里工作。Claude can read every file in the folder, edit existing files, and create new ones. It can also remember what you were working on after you end a session.Claude 可以读取文件夹里的每一个文件，编辑已有文件，并创建新文件。它还能在你结束会话后，记住你之前的工作内容。Unlike browser-based apps like ChatGPT and Gemini that only “talk,” Claude Code can actually “do” things for you.与 ChatGPT 和 Gemini 这类只能在浏览器里“对话”的应用不同，Claude Code 真的可以为你“做事”。

---

## 1.1 Why You Should Care About It?

## 1.1 为什么你应该关注它？

You are a researcher and have a folder on your computer with several PDFs, drafts of your papers, spreadsheets, datasets, and a few transcripts of interviews.假设你是一名研究人员，电脑里有一个文件夹，里面存放着若干 PDF、论文草稿、电子表格、数据集，以及几份访谈转录稿。All these documents are relevant to your project, but you need to make connections across published literature, gathered data, and your own notes.这些文档都与你的项目有关，但你需要将已发表的文献、收集到的数据以及自己的笔记联系起来。This is the kind of use-case that Claud Code was built for.这正是 Claude Code 被设计用来处理的典型场景。Most academics and researchers have used AI apps in their browsers (ChatGPT, Claude, Gemini, etc.) You open a tab and type in a prompt and the AI replies.大多数学者和研究人员都使用过浏览器中的 AI 应用（如 ChatGPT、Claude、Gemini 等）。你打开一个标签页，输入提示词，AI 就会回复。If you want to ask questions about your papers or drafts, you add them to ChatGPT or Claude.如果你想针对自己的论文或草稿提问，需要把它们上传到 ChatGPT 或 Claude 中。Claude Code is different because instead of you bringing your files to an AI app in the browser, you bring the AI into the folder containing all your data.Claude Code 的不同之处在于：不是你把文件带到浏览器的 AI 应用中，而是把 AI 带进存放所有数据的文件夹里。

---

## 1.2 Why Claude Code Matters for Academic Work

## 1.2 Claude Code 对学术工作的意义

There are two fundamental ways in which Claude Code can work as your reliable and powerful research assistant.Claude Code 能以两种基本方式成为你可靠且强大的研究助手。First, Claude Code remembers you, your research, your writing style, and your ever-evolving research requirements.首先，Claude Code 会记住你、你的研究、你的写作风格，以及你不断变化的研究需求。You won’t have to explain your requirements repeatedly.你无需反复解释你的要求。Second, since Claude Code works within a folder on your computer, it can process multiple files at a time.其次，由于 Claude Code 在你电脑的文件夹中工作，它可以同时处理多个文件。For example, it can go through all forty-five PDFs in your folder and extract relevant information (objectives, methodology, etc.) from all of them.例如，它可以遍历文件夹中的所有 45 篇 PDF，并从中提取相关信息（如研究目标、方法论等）。You can even ask it to create a new file based on extracted information.你甚至可以要求它根据提取的信息创建一个新文件。Claude Code can handle all types of files from Word files to Excel sheets, PDFs, etc.Claude Code 可以处理各种类型的文件，从 Word 文档到 Excel 表格，再到 PDF 等。

---

## 1.3 What Does It Mean in Practical Terms?

## 1.3 在实际操作层面这意味着什么？

If you are a qualitative researcher, load a folder full of interview transcripts and ask Claude Code to give you every instance of a particular utterance, e.g. how each participant talked about a given subject.如果你是一名质性研究者，加载一个装满访谈转录稿的文件夹，让 Claude Code 找出某个特定表述的所有出现位置，比如每位参与者是如何谈论某一给定主题的。Then ask it to look for overarching themes across interviews.然后，让它寻找贯穿所有访谈的宏观主题。If you are a quantitative researcher, add a messy CSV or an Excel sheet in your folder and ask Claude Code to clean it up for you.如果你是一名量化研究者，把一个杂乱的 CSV 文件或 Excel 表格放进文件夹，让 Claude Code 帮你清理数据。You can ask it to run descriptive statistics or ask it to explain a critical comment made by a reviewer.你可以让它运行描述性统计，或者让它解释审稿人提出的一条关键意见。

---

## 1.4 What Claude Code is Not

## 1.4 Claude Code 不是什么

Claude Code is an incredibly powerful tool, but it is not a replacement for your expert judgment.Claude Code 是一个极其强大的工具，但它不能替代你的专家判断。It can draft, summarize, code, but what counts as argument or evidence is your responsibility.它可以起草、总结、编码，但什么构成论证或证据，这仍然是你的责任。Academics who are likely to succeed using Claude Code are the ones who treat it as a research assistant and not those who outsource all their thinking and judgment to it.能够成功使用 Claude Code 的学者，是那些把它当作研究助手的人，而不是把所有思考和判断都外包给它的人。

# Part 2: Installing Claude Code App, Your First Session

# 第二部分：安装 Claude Code 应用及首次会话

If you’re new to Claude Code, set aside 15-20 minutes to install it and set it up. You don’t need any coding skills to use Claude Code.如果你是 Claude Code 的新手，请预留 15–20 分钟来安装和设置它。使用 Claude Code 不需要任何编程技能。You will need a Claude Pro or Max subscription to use Claude Code on your computer.你需要在电脑上使用 Claude Pro 或 Max 订阅才能使用 Claude Code。

---

## 2.1 Installation

## 2.1 安装

Claude Code is available for both Windows and Mac.Claude Code 支持 Windows 和 Mac 系统。Go to and download the version compatible with your computer.前往官网下载与你电脑兼容的版本。Run it the way you would any other installer (e.g. Zoom, Zotero, etc.).像运行其他安装程序（如 Zoom、Zotero 等）一样安装它。On Windows, it appears in your Start Menu. On Mac, it appears in your Applications folder.在 Windows 上，它会显示在“开始”菜单中；在 Mac 上，它会出现在“应用程序”文件夹中。Open the app once you have installed in it.安装完成后打开该应用。The first time you open it, Claude will ask you to sign in through browser.首次打开时，Claude 会要求你通过浏览器登录。Once you have signed in, you will see a column on the left and a chat panel in the middle.登录后，你会看到左侧有一列历史记录，中间是对话面板。The left column will show you a record of all your conversations just like in Claude or ChatGPT in browser.左侧栏会显示你所有的对话记录，就像浏览器版 Claude 或 ChatGPT 一样。

---

## 2.2 Opening Your First Folder

## 2.2 打开你的第一个文件夹

Click “Open Folder.” It usually on top of the chat bar in the main panel.点击“Open Folder（打开文件夹）”，它通常位于主面板聊天栏的上方。Its position keeps changing slightly with every update, but you will mostly find it in the vicinity of the chat bar.它的位置会随着每次更新略有变化，但你基本上可以在聊天栏附近找到它。Navigate to any folder on your computer in which you have stored PDFs or your drafts.导航到你电脑中存放了 PDF 或草稿的任何文件夹。If you want to be extra careful, create a dedicated “Claude Research Assistant folder” and navigate to that.如果你想更加谨慎一点，可以专门创建一个“Claude Research Assistant（Claude 研究助手）”文件夹并打开它。Now Claude Code is inside that folder. It has access to every file and subfolder inside the main folder.现在 Claude Code 就在该文件夹中，它可以访问主文件夹中的每一个文件和子文件夹。

---

## 2.3 What a Session Looks Like

## 2.3 会话是什么样的？

A session is the conversation you have with Claude in the main chat panel just like you do it in browser apps like ChatGPT.会话就是你在主聊天面板中与 Claude 进行的对话，就像在 ChatGPT 等浏览器应用中一样。The interaction is like browser-based apps like ChatGPT. The only difference is that now Claude Code has access to your files on your computer.交互方式与基于浏览器的应用类似，唯一的区别是现在的 Claude Code 可以访问你电脑上的文件。It also has the ability to create files on your computer.它还能在你电脑上创建文件。Once you are in the folder, give Claude a simple prompt like: Read all the papers in the folder and give me their main arguments as a separate file.进入文件夹后，给 Claude 一个简单的提示，例如：“阅读文件夹中的所有论文，并将它们的主要论点整理成一个单独的文件。”That’s it. Most of your interaction with Claude Code would be writing instructions like this.就是这样。你与 Claude Code 的大部分互动都是编写这样的指令。Claude Code will go ahead and retrieve relevant information from the papers and start creating a new file.Claude Code 会从论文中提取相关信息，并开始创建新文件。This file will show up in the folder that you are working in.这个新文件会出现在你正在工作的文件夹中。As it proceeds, it will ask you for permission to do so.在执行过程中，它会请求你的许可。You can make Claude Code bypass permissions, but in the beginning, it’s a good idea to make Claude Code ask for permissions.你可以让 Claude Code 绕过权限请求，但在初期，建议让它询问你的许可。You don’t want Claude Code to end up deleting any important files without asking you.你肯定不希望 Claude Code 在未经询问的情况下删除了重要文件。Every session you start in Claude Code is automatically saved in the panel on the left.你在 Claude Code 中开启的每个会话都会自动保存在左侧面板中。

# Part 3: Claude Code as Your Research Assistant

# 第三部分：Claude Code 作为你的研究助手

Every time Claude Code starts, it immediately looks for a file called CLAUDE.md for a set on of instructions to follow.每次启动 Claude Code 时，它会立即查找一个名为 CLAUDE.md的文件，以获取一组需要遵循的指令。Notice the block letters, this is important.注意是大写字母，这一点很重要。There are two ways you can create a CLAUDE md file: automatic and manual.创建 CLAUDE.md文件有两种方式：自动和手动。

---

## 3.1 Manually

## 3.1 手动创建

If you want to do it manually, open Notepad on Windows or TextEdit on a Mac and write down the kind of instructions you want Claude Code to follow.如果你想手动创建，请在 Windows 上打开记事本，或在 Mac 上打开文本编辑（TextEdit），写下你希望 Claude Code 遵循的指令。Please note that you only need to write these instructions in plain language. You don’t need to use any coding or programming language.请注意，你只需要用自然语言写这些指令，不需要使用任何编程语言。You can divide CLAUDE md file into following sections:你可以将 CLAUDE.md文件分为以下几个部分：#Role(the hash sign here indicates that it’s a heading)#Role（这里的井号表示这是一个标题）Describe the kind of role you would like Claude Code to assume as your research assistant. Give it a few details about your research field and current project.描述你希望 Claude Code 作为研究助手扮演的角色。提供一些关于你的研究领域和当前项目的细节。#Standards#StandardsDescribe academic standards relevant to your field of research. For example, you can tell Claude Code to follow a given citation style or a specific structure of a research paper.描述与你研究领域相关的学术标准。例如，你可以告诉 Claude Code 遵循特定的引用格式或研究论文的具体结构。#WritingStyle#WritingStyleTell Claude Code the kind of writing style you would like it to follow, whether you want it to respond in academic style or in an informal style.告诉 Claude Code 你希望它遵循的写作风格，无论是学术风格还是非正式风格。#CritiqueStyle#CritiqueStyleDescribe how you would like Claude Code to critique your work, whether you would like to focus on your argument, evidence, or methodology.描述你希望 Claude Code 如何批评你的工作，是关注论点、证据，还是方法论。

![](attachments/e675dba8a82d14a5.png)

Save the file as CLAUDE.md in your main Claude folder on your computer and you are done.将该文件以 CLAUDE.md为名保存在你电脑的主 Claude 文件夹中即可。You don’t need to have a perfectly crafted CLAUDE.md file. You can always go back and edit it as your project evolves.你不需要一开始就打造一个完美的 CLAUDE.md文件，随着项目推进，你可以随时回去编辑它。

---

## 3.2 Automatically

## 3.2 自动创建

If you don’t want to create a CLAUDE.md file manually, simply start a chat session in Claude Code and give it details about its role, standard, writing style, and critique style.如果你不想手动创建 CLAUDE.md文件，只需在 Claude Code 中开启聊天会话，提供关于其角色、标准、写作风格和批评风格的详细信息。Then ask it to create a CLAUDE.md file using this information.然后要求它利用这些信息创建一个 CLAUDE.md文件。Claude Code will create the file and save it to your Claude folder.Claude Code 会创建该文件并将其保存到你的 Claude 文件夹中。

![](attachments/afa4f86358444622.png)

---

## 3.3 Auto-Memory

## 3.3 自动记忆（Auto-Memory）

As you work through your project, Claude Code will write short notes about your work and save them for its own use.在你开展项目的过程中，Claude Code 会写下关于你工作的简短笔记并保存下来供自己使用。You won’t see them and you don’t need to bother about them too much.你看不到这些内容，也不需要太在意它们。Claude Code puts these notes in a folder and manages it on its own.Claude Code 将这些笔记放入一个文件夹中并自行管理。Each time you start a session, Claude Code reads CLAUDE.md and its own notes and uses them to answer your questions.每次启动会话时，Claude Code 都会读取 CLAUDE.md和它自己的笔记，并用这些信息来回答你的问题。Over time, your instructions and Claude Code’s notes gives it enough context about you and your work that it becomes a reliable research assistant.随着时间的推移，你的指令和 Claude Code 的笔记会为它提供足够的关于你和你的工作的背景信息，使其成为一个可靠的研究助手。You can ask Claude Code about its memory by typing “Tell me what you have stored in your memory.”你可以输入“Tell me what you have stored in your memory（告诉我你记住了什么）”来询问 Claude Code 的记忆内容。If a certain piece of information is outdated (e.g. a citation style), simply ask it update its memory with the new information.如果某些信息过时了（例如引用格式），只需要求它用新信息更新记忆即可。

---

## 3.4 What Not to Put in CLAUDE.md

## 3.4 不要放入 CLAUDE.md 的内容

Don’t put any confidential information in CLAUDE.md or anything that you don’t the AI to use.不要在 CLAUDE.md中放入任何机密信息，也不要放入你不希望 AI 使用的内容。Don’t let CLAUDE.md become a set of outdated information because that will have considerable impact on its output.不要让 CLAUDE.md变成一堆过时的信息，因为这将严重影响它的输出质量。You may want to update the file every few weeks.你可能每隔几周就需要更新一次该文件。

---

# Part 4: Working with Your Research Documents

# 第四部分：处理你的研究文档

A Claude folder does not have to be very neatly organized. This doesn’t mean you start putting irrelevant files in this folder.Claude 文件夹不需要组织得非常整齐。这并不意味着你可以往里面放无关的文件。What it means is that you should not spend any time organizing it. Claude can do that itself.它的意思是你不应该花时间去整理它，Claude 自己可以做到。The only thing you may want to keep in mind is not to give your files confusing names like Dissertation (final) (final2) (use this one).唯一需要注意的是，不要给你的文件起令人困惑的名字，比如“Dissertation (final) (final2) (use this one)”。Add a bunch of PDFs, Word files, datasets, interview transcripts, etc. related to your project.添加一批与你的项目相关的 PDF、Word 文件、数据集、访谈转录稿等。Suppose you have twenty research papers related to your research topic and you want to find a common theme that runs across all of them.假设你有二十篇与研究主题相关的论文，你想找出贯穿所有这些论文的共同主题。Or maybe you have a certain claim or an argument, and you are looking for papers that present evidence supporting or contrasting that claim.或者你有一个特定的主张或论点，想找一些提供支持或相反证据的论文。Open the folder in Claude Code and write a prompt like: Read every PDF in this folder and tell me which articles disagree with the following argument: [paste the argument here]在 Claude Code 中打开该文件夹，并输入提示词，例如：“阅读此文件夹中的每一篇 PDF，告诉我哪些文章不同意以下论点：[在此处粘贴论点]”。Claude will read all the PDFs in the folder and will give you the relevant information from these articles. Claude’s answer may be in the form of a table.Claude 会阅读文件夹中的所有 PDF，并给出这些文章中的相关信息。Claude 的回答可能会以表格的形式呈现。

---

## 4.1 Research Assistant for Literature Review

## 4.1 文献综述的研究助手

If you are a biomedical researcher or a social scientist who frequently runs systematic reviews, download fifty research articles and put them in a folder titled “Systematic Review with Claude.”如果你是一名经常进行系统性综述的生物医学研究者或社会科学家，下载五十篇研究文章，并将它们放入名为“Systematic Review with Claude（用 Claude 做系统综述）”的文件夹中。Open the folder in Claude Code and give it your screening (inclusion/exclusion) criteria and ask it to screen all the papers in the folder.在 Claude Code 中打开该文件夹，给出你的筛选（纳入/排除）标准，并要求它对文件夹中的所有论文进行筛选。Claude Code will screen the papers according to your criteria and give you the results in the form of a table.Claude Code 会根据你的标准筛选论文，并以表格形式给出结果。

---

## 4.2 Working with Transcripts

## 4.2 处理转录稿

If you are qualitative researcher, you can add interview transcripts in the Claude folder and ask it extract information related to a given topic.如果你是一名质性研究者，可以将访谈转录稿添加到 Claude 文件夹中，并要求它提取与给定主题相关的信息。For example, you can ask it to extract how each respondent answer a given question.例如，你可以要求它提取每位受访者是如何回答某个给定问题的。

---

## 4.3 Give Claude a Tedious Task

## 4.3 交给 Claude 一项繁琐的任务

Put fifty-odd PDFs in a folder and open it in Claude Code. Then ask it to go through them all and rename them using their titles.将一个包含约 50 个 PDF 的文件夹在 Claude Code 中打开，然后要求它遍历所有文件并根据标题重命名。You will see that Claude Code has done the needful in a couple of minutes.你会发现 Claude Code 在几分钟内就完成了这项必要工作。

---

## 4.4 Asking Claude Code to Create Files

## 4.4 要求 Claude Code 创建文件

Since your project is going to evolve, you would want Claude Code to create certain files for you too so that can you go back to them if needed.由于你的项目会不断发展，你可能希望 Claude Code 也为你创建一些文件，以便需要时回头查看。It will also help Claude Code with easy retrieval as the project develops.这也有助于 Claude Code 在项目推进时更容易地检索信息。Anytime you ask Claude Code to do a significant task like screening papers or extracting information from transcripts, ask it to save your answer in folder as a file.每当你让 Claude Code 执行诸如筛选论文或从转录稿中提取信息等重要任务时，要求它将答案作为文件保存在文件夹中。Generally, Claude Code will save these files as markdown (md), which take up very less space and are very easy for Claude Code to retrieve information from.通常，Claude Code 会将这些文件保存为 Markdown（md）格式，这种格式占用空间很小，且 Claude Code 很容易从中检索信息。But you can also ask it to create a Word file or an Excel sheet.但你也可以要求它创建 Word 文件或 Excel 表格。

# Part 5: Claude Code “Skills”

# 第五部分：Claude Code 的“技能”（Skills）

A Skill is a set of instructions that makes Claude Code a specialist for one task.“Skill（技能）”是一组指令，使 Claude Code 成为某项任务的专家。Just like CLAUDE.md file, a Skill is also a markdown file written in plain English.就像 CLAUDE.md文件一样，Skill 也是一个用通俗英语编写的 Markdown 文件。And just like CLAUDE.md, you can create a Skill file both manually and automatically.而且和 CLAUDE.md一样，你也可以手动或自动创建 Skill 文件。Once you create a Skill, Claude Code will remember when to use it. You do not have to remember yourself.一旦创建了 Skill，Claude Code 会记住何时使用它，你无需自己记。But you can always invoke a Skill if want to, by using the forward-slash command.但如果你愿意，也可以通过斜杠命令（/）随时调用某个 Skill。The easiest way to create a Skill is to ask Claude Code to do it for you.创建 Skill 最简单的方法是让 Claude Code 替你完成。Let’s say, you have regular Zoom calls, and you have transcripts of those calls. For every call or transcript, you want to extract actionable items or things to do.假设你有定期的 Zoom 通话，并且有这些通话的转录稿。对于每一次通话或转录稿，你都想提取出可执行的行动项或待办事项。To create a Skill, start a Claude Code session and simply ask it to create a Skill for extracting actionable items from Zoom transcripts.要创建一个 Skill，启动 Claude Code 会话，直接要求它为“从 Zoom 转录稿中提取可执行行动项”创建一个 Skill。Claude Code will get to work. It will ask you follow-up questions and create a Skill file that you can edit if you want to.Claude Code 会开始工作，它会问你一些后续问题，并创建一个 Skill 文件，你可以根据需要对其进行编辑。Maybe you want Claude Code to follow a specific structure while responding.也许你希望 Claude Code 在回答时遵循特定的结构。Once the Skill is created, restart Claude Code and the Skill will be ready to use.Skill 创建完成后，重启 Claude Code，该 Skill 就可以使用了。

## 5.1 Difference between CLAUDE.md and Skills

## 5.1 CLAUDE.md 与 Skills 的区别

CLAUDE.md contains global instructions about you and your project. It gives Claude Code the big picture about you and your research.CLAUDE.md包含关于你和你的项目的全局指令。它让 Claude Code 了解关于你和你的研究的整体情况。Skills, on the other hand, are meant for particular tasks. These files contain information that are much more specific and granular.而 Skills 则是针对特定任务的。这些文件包含的信息更加具体和细化。CLAUDE.md, Skill, and auto-memory work in concert to give you the best possible response.CLAUDE.md、Skill 和自动记忆协同工作，为你提供尽可能优质的回答。

# What Not to Delegate to Claude Code

# 不要委派给 Claude Code 的事项

Claude Code is great at labor intensive, time consuming, and repetitive tasks. Outsource these tasks to Claude Code.Claude Code 非常擅长劳动密集型、耗时且重复性的任务。把这些任务外包给 Claude Code。But Claud Code will not be able to create what actually counts as scholarship because it won’t be able to give you new and original arguments.但是，Claude Code 无法创造出真正算作学术成果的东西，因为它无法提供新颖且原创的论点。It can synthesize information that you can then use in service of your argument, but the job of coming up with an original argument remains yours as a researcher.它可以综合信息，供你用于支撑自己的论点，但提出原创论点的工作仍然属于你——作为一名研究者的职责。————感谢您能读到这里，这篇文章对你应该有用，你可以试着收藏，转发，或者点赞推荐。
