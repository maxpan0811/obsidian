# 北京大兴⇌济州定期往返航线盛大起航，华程国旅集团携手济州航空，打造高性价比赴韩旅游！

速读摘要

4月24日，济州航空北京大兴--韩国济州定期往返航线迎来盛大首航。济州航空航班7C8135抵达北京大兴国际机场，机场给予了最高规格的"水门礼遇"。以今天为起点，济州航空北京大兴--济州航线将会每周运行四班，不仅为韩国游客提供了更加便利的海外旅行，也给中国游客带来便利旅行，使中国游客了解更多的韩国旅游目的地。济州岛是韩国最大的度假胜地与蜜月目的地，也是中国游客一直以来都较为青睐的韩国代表性旅游胜地，全年气候宜人，大量的韩剧和韩综曾在此取景拍摄。

**4月24日，济州航空北京大兴--韩国济州定期往返航线迎来盛大首航。这是**华程国旅集团携手济州航空打造的首个战略合作航旅项目，也是**韩国航空公司正班航线首次走进北京大兴国际机场，标志着中韩旅游进入全新的发展阶段。**

![](.evernote-content/0F5222EC-73EA-4670-93F8-A355E28484B9/A435C48B-EB46-4B55-8E15-45E832A8FE70.jpg)![](.evernote-content/0F5222EC-73EA-4670-93F8-A355E28484B9/5D2F4160-588E-4CA8-B38C-D17A537F6430.jpg)

**当天下午13:00，济州航空**航班****7C8135抵达**北京大兴国际机场****，机场给予了最高规格的“水门礼遇”。14:00，**首航仪式隆重举行，****济州航空、北京大兴国际机场、华程国旅集团、北京东航地面服务公司、韩国旅游发展局领导现场出席。

![](.evernote-content/0F5222EC-73EA-4670-93F8-A355E28484B9/5E61E929-29B2-4BFE-973B-19C7898135DD.png)![](.evernote-content/0F5222EC-73EA-4670-93F8-A355E28484B9/AB0A0C08-7E14-4E26-B116-F53FDEE411BF.jpg)

****14:30，**华程国旅集团首航团贵宾100余人，乘坐7C8136航班从大兴国际机场出发，17:55航班落地济州岛，贵宾们将正式开启“享在济州5天”之旅。**

![](.evernote-content/0F5222EC-73EA-4670-93F8-A355E28484B9/F2424B09-0213-433D-9EF7-639219D1FCA5.jpg)![](.evernote-content/0F5222EC-73EA-4670-93F8-A355E28484B9/A5DA3CEC-F829-41DC-8A6B-D81EA428B03B.jpg)

济州航空安保部部长洪準模在首航仪式现场表示，“济州航空目前是韩国低成本航空公司中运营中国航线最多的。未来，济州航空将努力吸引更多的韩国游客到中国北京，为北京旅游业做出贡献。同时，我们将尽最大努力让更多的中国乘客以合理的价格，安全、舒适地享受韩国旅行”。

![](.evernote-content/0F5222EC-73EA-4670-93F8-A355E28484B9/4F550D2C-6945-443E-8E38-AD42E8CE6BA2.jpg)

北京大兴国际机场航空业务部总经理王亚琦现场致辞，“此次首航意义非凡，济州航空是首家在大兴机场运行的韩国航司，首家在北京两场运行的韩国航司”。大兴机场是京津冀协同发展的新引擎，2023年全年旅客量达到3941万人次，2024年大兴机场旅客量预计突破4800万人次，其中国际及地区旅客量有望达到400万人次。

![](.evernote-content/0F5222EC-73EA-4670-93F8-A355E28484B9/CF52A050-EFB3-4085-BD44-166C3FD552A2.jpg)

华程国旅集团副总裁兼华程航服公司总经理匡世盛表示，作为中国领先的出境旅游及会展服务运营商，华程国旅集团将以此次北京济州--大兴定期往返航旅项目为起点，带领更多中国人特别是京津冀地区年轻人前往济州岛，并与济州航空展开更深度的战略合作，探索航旅融合的新模式，开发更多航线和目的地。

![](.evernote-content/0F5222EC-73EA-4670-93F8-A355E28484B9/17D90499-5233-4072-805A-1B42E068D5C3.jpg)

北京东方航空地面服务有限公司副总经理何旭迎表示，东方航空将为各家航司提供优质服务，快速响应航司运行过程中的困难与问题。

![](.evernote-content/0F5222EC-73EA-4670-93F8-A355E28484B9/B41778F8-1810-4019-89B8-B2F704AB4CC5.jpg)

韩国旅游发展局中国区总部首席代表陈钟和表示，“相信大兴与济州之间的定期航班，会更加方便京津冀地区的中国游客前往济州岛，更好地体验韩国独特的自然人文风情。韩旅局也会为了中韩旅游的持续友好发展，竭尽全力提供所需要的帮助。”

![](.evernote-content/0F5222EC-73EA-4670-93F8-A355E28484B9/B2EA3969-9105-4FD4-961C-4A03803FA7DE.jpg)

济州航空中国代表朴钟润向所有为了济州航空新航线顺利开航给予支持和协助的相关人士表示衷心感谢。“以今天为起点，济州航空北京大兴--济州航线将会每周运行四班，不仅为韩国游客提供了更加便利的海外旅行，也给中国游客带来便利旅行，使中国游客了解更多的韩国旅游目的地。”

![](.evernote-content/0F5222EC-73EA-4670-93F8-A355E28484B9/891C1BDE-8146-432D-902E-486BC4D3A594.jpg)

济州岛是韩国最大的度假胜地与蜜月目的地，也是中国游客一直以来都较为青睐的韩国代表性旅游胜地，全年气候宜人，大量的韩剧和韩综曾在此取景拍摄。持有中国大陆护照（有效期大于6个月）的游客可免签进入韩国济州岛30天。

![](.evernote-content/0F5222EC-73EA-4670-93F8-A355E28484B9/F7413533-668E-4AB1-A70A-29C5D381C7C3.jpg)![](.evernote-content/0F5222EC-73EA-4670-93F8-A355E28484B9/45137AF9-E1B5-4F6D-9686-D178362D7B61.jpg)

**针对不同赴韩国游客客群画像，华程国旅集团推出了丰富多元的韩国旅游线路，产品形态涵盖纯玩高端定制、机+酒FIT自由行、随团游、4+1半自助、目的地参团一日游等10余条线路，主题涵盖度假、美食、研学、旅拍、奖励旅游等。**
