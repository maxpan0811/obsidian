用了那么久 Chrome，这些版本区别你都了解吗？
-------------------------

数码荔枝

全新的 Chrome 69 浏览器将在 9 月 4 日正式发布，新版最大的改变是采用了全新的 Material Design 风格。
==================================================================

![](.evernote-content/EFA54610-E8D5-46C5-9CFC-5123272A5FB0/A02042CA-0FB8-4FBC-AAB5-53BD6C0FAE85.jpg)

虽然正式版目前还是 68，但其实，你现在就可以马上体验包含各种新功能和全新界面的 Chrome 69、甚至 70——那就是安装 Beta 版、Canary 版。

可能有些朋友模糊知道，Chrome 有多个不同版本在并行更新，但它们各自分别是什么版本、以及这些版本之间有哪些区别呢？这篇文章带你进一步了解 Chrome。

**一切要从 Chromium 说起**
--------------------

![](.evernote-content/EFA54610-E8D5-46C5-9CFC-5123272A5FB0/15F26161-7949-4DE0-9AF3-AB58E2D3F48A.png)

正如之前我们的文章中提到，Chromium 是谷歌发起的一个开源浏览器项目，最初使用 WebKit 内核，之后又在 WebKit 基础上分支开发出 Blink 内核。

开源的力量是强大的，Chromium 结合谷歌和众多第三方开发者的力量，已经是当今最先进的浏览器项目。

Chromium 更新速度非常快，几小时更新一次，因此不能自动更新。由于代码非常新且没有测试，因此很多功能极不稳定。

相比 Chrome，Chromium 没有那么多谷歌自家的服务，同时也缺少了部分商业闭源插件（H.264 解码器、Flash 等），是一个非常「纯净」的浏览器，不太适合普通用户直接使用。而包括 Chrome 在内的众多浏览器，会在它的基础上增加许多自家独有的服务与功能。

**Chrome 各版本究竟有哪些不同**
---------------------

除去开源的 Chromium，Chrome 一共有这么几个版本：Canary、Dev、Beta 以及正式版 Stable。这么多版本，具体又有哪些不同呢？

下面将为大家做一个简单的介绍。

首先，虽然 Chrome 这几个版本名称各不相同，但都沿用了相同的版本号，只是更新早晚的区别。就好比 iOS 等系统，Beta 版可以率先更新到 iOS 12 并进行测试，不断改进稳定后，正式版才升级到 12 版本。

Chrome 也是如此，更新最快的 Canary 会领先正式版 1-2 个版本。

![](.evernote-content/EFA54610-E8D5-46C5-9CFC-5123272A5FB0/E1C7A48D-7897-4852-B0A5-31F611E5B30C.jpg)

Chrome Canary 是更新速度最快的 Chrome 版本，几乎每天更新。它相当于支持自动更新、并添加了谷歌自家服务与商业闭源插件（Flash 等）的 Chromium，更加强大好用。

Canary 是功能、代码最先进的 Chrome 版本，一方面软件本身没有足够时间测试，另一方面网页也不一定支持这些全新的功能，因此极不稳定。好在，谷歌将其设定为可独立安装、与其他版本的 Chrome 程序共存，因此适合进阶用户安装备用，尝鲜最新功能。

![](.evernote-content/EFA54610-E8D5-46C5-9CFC-5123272A5FB0/2000E55A-9DB2-4AF2-B48B-2DF729AC6372.jpg)

Chrome Dev 最初是以 Chromium 为基础、更新最快的 Chrome，后来则被 Canary 取代。Dev 版每周更新一次，虽然仍不太稳定，但已经可以勉强满足日常使用，适合 Web 开发者用来测试新功能和网页。

![](.evernote-content/EFA54610-E8D5-46C5-9CFC-5123272A5FB0/3452059E-8EA6-43BC-80CF-8BF695CF71C1.jpg)

Chrome Beta 以 Dev 为基础，每月更新一次。它是正式发布前的最后测试版本，所有功能都已在前面几个版本中得到测试并改进，因此已经十分稳定，普通用户也可以用来日常使用。

![](.evernote-content/EFA54610-E8D5-46C5-9CFC-5123272A5FB0/E7FCEC6F-4C29-4B0A-A618-E2015E1F36A1.jpg)

最后的 Chrome Stable 就是我们熟知的正式版，它以 Beta 为基础，几个月更新一次。由于所有的功能都已经过数个月反复测试，是稳定性最高的 Chrome 版本。

**那我要用什么版本更好？**
---------------

其实对于绝大多数人来说，正式版可能就是最合适的版本，它经历了长时间、无数人的测试，足够稳定、成熟。但如果你想要更早体验到全新的功能、界面，那就可以考虑其他几个版本。

如果你只是希望能提前试用已经比较稳定的功能，不希望浏览器频繁崩溃，那么 Beta 版毫无疑问最为适合，它基本可以取代正式版使用。

如果你希望尝鲜最新的技术，并且不介意多占用一点存储空间，不放试试漂亮的「金丝雀」——Canary 吧。使用它，你可以在第一时间接触最新的浏览器技术。由于它可以和正式版共存，即便某个更新无法正常使用，你也可以随时打开备用的正式版，而不会影响手头的工作。
