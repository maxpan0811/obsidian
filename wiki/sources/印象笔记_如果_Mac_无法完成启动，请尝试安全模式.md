如果 Mac 无法完成启动，请尝试安全模式
=====================

您可以使用安全模式来帮助您解决可能阻止 Mac 完成启动的问题。

什么是安全模式？
--------

安全模式（有时称为安全启动）是一种 Mac 启动方式，它会执行某些检查并防止某些软件自动加载或打开。以安全模式启动 Mac 会执行以下操作：

* 验证启动磁盘，并尝试根据需要修复目录问题
* 只载入必需的内核扩展
* 防止启动项和登录项自动打开
* 停用用户安装的字体
* 删除字体缓存、内核缓存和其他系统缓存文件

所有这些变化有助于解决或查明与启动磁盘有关的问题。

![](.evernote-content/C6C3C82D-25FC-41C0-8108-7308453388AD/B8194F95-25F2-4BE3-98CA-E691C129D538.png)

以安全模式启动
-------

请执行以下步骤以启动进入安全模式。

1. 启动或重新启动 Mac。
2. 听到启动声后，立即按住 Shift 键。
3. 看到屏幕上出现 Apple 标志后松开 Shift 键。

Apple 标志出现后，进入登录屏幕或桌面的时间可能比平时要长。这是因为作为安全模式的一部分，Mac 会对启动磁盘执行目录检查。

要退出安全模式，请重新启动电脑，在启动时不要按下任何键。

如果按下电源键后，您听不到启动声，请参阅“[如果 Mac 无法开机应如何处理](https://support.apple.com/kb/HT204267?viewlocale=zh_CN)”。

### 如果您在使用 FileVault

如果您的启动磁盘通过 [FileVault](http://support.apple.com/kb/HT204837?viewlocale=zh_CN) 进行了加密，您仍可在开启 Mac 的电源后立即按下 Shift 键来以安全模式启动。作为此过程的一部分，系统可能会提示您登录两次：第一次为解锁启动磁盘，第二次则为登录 Finder。在您看到第一个登录屏幕后，可松开 Shift 键。

![](.evernote-content/C6C3C82D-25FC-41C0-8108-7308453388AD/B8194F95-25F2-4BE3-98CA-E691C129D538.png)

如果在安全模式下未出现问题
-------------

如果以安全模式启动时 Mac 未出现问题，请再次尝试重新启动，并在启动时不要按下任何键。如果正常启动时问题似乎已解决，则问题可能是由启动磁盘存在的缓存或目录问题所致，并且这些问题已通过安全模式得到解决。

如果您以正常方式重新启动 Mac 后，在到达桌面时问题重新出现，请尝试停用在您登录时自动打开的任何[登录项](https://support.apple.com/kb/PH18759?viewlocale=zh_CN)。

如果 Mac 在安全模式下重新启动或关闭
--------------------

如果 Mac 在安全模式下自动重新启动，则 OS X 可能发现了问题并且正在尝试解决。

安全模式会对启动磁盘执行目录检查，这与您选择使用[磁盘工具](http://support.apple.com/kb/HT201639?viewlocale=zh_CN)来验证或修复磁盘的情况类似。如果 OS X 发现问题，启动磁盘上的目录将被修复，并且 Mac 会重新启动。

如果 Mac 在以安全模式启动时自动关机，请检查 Mac 上的电源连接以确保 Mac 端和电源插座端的插接均良好。如果您使用了 MagSafe 电源适配器，请确保[电源适配器上的 LED 指示灯为黄色或绿色](https://support.apple.com/kb/HT203207?viewlocale=zh_CN)。然后，再次尝试以安全模式启动。

如果 Mac 重复重新启动或在安全模式下关机，请联系 Apple 支持，或咨询 Apple 授权服务提供商或 Apple Genius 来寻求更多帮助。

![](.evernote-content/C6C3C82D-25FC-41C0-8108-7308453388AD/B8194F95-25F2-4BE3-98CA-E691C129D538.png)

如何判断 Mac 是否是以安全模式启动
-------------------

如果您不确定 Mac 是否是以安全模式启动，可使用“[系统信息](http://support.apple.com/kb/HT203001?viewlocale=zh_CN)”来进行检查。当以安全模式启动时，“系统信息”的“软件”部分会将启动模式列为“安全”而非“正常”。

![](.evernote-content/C6C3C82D-25FC-41C0-8108-7308453388AD/28E52828-7FC7-4D12-93B5-63F739CB9789.png)

当 Mac 以安全模式启动时，登录窗口也会在屏幕的右上角中显示“安全启动”。

### 某些功能在安全模式下不可用

当以安全模式启动时，Mac 的某些功能不可用，因此，使用安全模式完成启动后，您应以正常方式重新启动电脑。

例如，以下功能在安全模式下不可用：

* 您无法在 DVD 播放器中播放影片。
* 您无法在 iMovie 和某些其他视频 app 中捕捉视频。
* 某些音频输入或输出设备可能不工作。
* 某些 USB、FireWire 和 Thunderbolt 设备可能不可用。
* Wi-Fi 网络可能受限或不可用，具体取决于您所用的 Mac 和 OS X 版本。

在 OS X v10.5 或更高版本中，安全模式还会停用图形加速功能。这让 OS X 菜单栏显示为实心，即便在“系统偏好设置”中选择了“半透明菜单栏”也是如此。当 Mac 以安全模式启动时，您的屏幕还可能在登录期间“闪烁”或“分割”。

在 OS X v10.6 或更高版本中，安全模式还会停用“文件共享”功能。

![](.evernote-content/C6C3C82D-25FC-41C0-8108-7308453388AD/B8194F95-25F2-4BE3-98CA-E691C129D538.png)

在没有键盘的情况下以安全模式启动
----------------

如果 Mac 没有键盘用来以安全模式启动，但可以远程访问电脑，则可以使用命令行将 Mac 配置为以安全模式启动。

1. 远程打开“终端”或通过 SSH 登录到该电脑以访问命令行。
2. 请使用以下“终端”命令：  
   `sudo nvram boot-args="-x"`  
   如果您希望同时以详细模式启动，则改为使用以下命令：  
   `sudo nvram boot-args="-x -v"`
3. 使用安全模式后，请使用以下“终端”命令返回正常启动模式：  
   `sudo nvram boot-args=""`
