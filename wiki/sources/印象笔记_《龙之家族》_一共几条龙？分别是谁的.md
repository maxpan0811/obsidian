# 《龙之家族》 一共几条龙？分别是谁的

---

速读摘要

雷妮拉从7岁起开始骑乘叙拉克斯，叙拉克斯也成为了"黑党"的重要标志。它是一条红色巨龙，体型庞大到恐怖，脖颈超长，拥有丰富的战斗经历，也是"黑党"最重要的战斗力。伊耿三世出生时，其襁褓里就放了一颗龙蛋，这颗龙蛋孵化出了"暴云"。血龙狂舞"时期，伊耿三世只有9岁，他带着暴云在躲避战乱时，暴云因重伤而死。它全身闪亮金鳞，翼膜为淡粉色，宛如金色黎明，是维斯特洛大陆乃至整个世界有史以来最雄伟最漂亮的龙。

在电视剧《龙之家族》中将会出现17条龙。以下是部分龙及其所属者的介绍（涉及剧透）：

![](.evernote-content/E4D59565-F2B7-478A-85E9-EA2DB316F3C1/80AE9EC8-4244-49CC-BEBD-406CD1F93002.png)

* 黑党阵营：

+ 叙拉克斯：驭龙者为雷妮拉公主。它有一身黄色的鳞片，体型庞大且十分凶悍，其名字据说来自古瓦雷利亚的一位女神。雷妮拉从7岁起开始骑乘叙拉克斯，叙拉克斯也成为了“黑党”的重要标志。在君临暴乱中，雷妮拉的第三子乔佛里试图骑乘叙拉克斯，但叙拉克斯无法适应新骑手，导致乔佛里滑下龙背摔死。叙拉克斯继续飞行着陆在龙穴外，最终被暴民所杀。
+ 科拉克休：外号“嗜血巨虫”，驭龙者是戴蒙王子。它是一条红色巨龙，体型庞大到恐怖，脖颈超长，拥有丰富的战斗经历，也是“黑党”最重要的战斗力。在石阶列岛战争期间，戴蒙骑乘着科拉克休参与多场战斗，经常往返于石阶列岛和龙石岛之间。“血龙狂舞”末期，科拉克休对“绿党”巨龙瓦格哈尔发起挑战，最终元气耗尽倒在赫伦堡的城墙下。
+ 梅丽亚斯：被称为“红女王”，驭龙者是雷妮丝公主。它是一头成年母龙，通体火红。在鸦栖堡之战中，雷妮丝公主骑着“红女王”大战“绿党”的两条巨龙——伊耿二世的阳炎和伊蒙德王子的瓦格哈尔，最终被杀。
+ 沃马克斯：驭龙者是雷妮拉的长子杰卡里斯。它是一头年轻公龙。在“血龙狂舞”内战打响时，杰卡里斯只有15岁，他骑上沃马克斯飞往鹰巢城和临冬城等地，为他的母亲雷妮拉寻求支持，成功地为黑党赢得了北境和谷地。在喉道海战中，沃马克斯飞得太低，结果一头扎入了海里。
+ 阿拉克斯：雷妮拉的次子路斯里斯的坐骑。为争取拜拉席恩家族的支持，路斯里斯王子骑着阿拉克斯飞往风息堡，与伊蒙德的巨龙瓦格哈尔搏斗后，被撕成碎片落入大海。
+ 暴云：它的驭龙者是雷妮拉和戴蒙的儿子伊耿三世。伊耿三世出生时，其襁褓里就放了一颗龙蛋，这颗龙蛋孵化出了“暴云”。“血龙狂舞”时期，伊耿三世只有9岁，他带着暴云在躲避战乱时，暴云因重伤而死。
+ 泰雷克休：雷妮拉的第三子乔佛里的坐骑。在血龙狂舞时期它还是一条幼龙，乔佛里被母亲留在身边，泰雷克休被锁在龙穴里，后被冲击龙穴的暴民所杀。
+ 月舞：驭者是戴蒙和兰娜尔的女儿贝妮拉公主。它被誉为历史上最优雅的龙，有着浅绿色的鳞片，犄角散发着珍珠般的光泽，是叙拉克斯的后代。内战末期，贝妮拉骑上月舞迎战伊耿二世和他的阳炎，最终不敌被阳炎吞噬。
+ 海烟：最初是海蛇和雷妮丝的独子兰尼诺的坐骑，兰尼诺死后，海烟成为龙石岛上的无主野龙。后来亚当成为海烟的骑手，他们在第二次腾石镇之战中与戴伦王子骑乘的特赛里恩交战，海烟最终被沃米索尔咬断脖子而死。

* 绿党阵营：

+ 瓦格哈尔：伊蒙德王子的坐骑。它是坦格利安家族最早的三条巨龙之一，也是维斯特洛大陆体型最庞大的巨龙。伊蒙德以一只眼睛为代价得到了瓦格哈尔。在破船湾的狂风中，伊蒙德凭借瓦格哈尔击杀了黑党公主雷妮拉的二儿子路斯里斯与他的小龙阿拉克斯。
+ 阳炎：伊耿二世的坐骑，是一头金光灿灿的公龙。它全身闪亮金鳞，翼膜为淡粉色，宛如金色黎明，是维斯特洛大陆乃至整个世界有史以来最雄伟最漂亮的龙。
+ 梦火：海伦娜王后的坐骑，是一头完全成熟的母龙。《权力的游戏》中龙妈丹妮莉丝的三颗龙蛋就是梦火产下的。
+ 特赛里恩：也被称作“蓝女王”，是戴伦王子的坐骑。它是一头年轻的母龙，翅膀呈深钴蓝色，利爪、龙冠和肚子上的鳞片亮如铜箔，在空中飞起来更轻盈更敏捷。

剧中还有其他一些龙，如果你想了解更多关于这些龙的信息，可以继续提问。需要注意的是，具体的龙的数量和归属可能会随着剧情的发展而有所变化，如果你想全面了解剧中所有龙的情况，建议观看电视剧或查阅相关的剧情资料。
