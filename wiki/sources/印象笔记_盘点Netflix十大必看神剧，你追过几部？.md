# 盘点Netflix十大必看神剧，你追过几部？

---

速读摘要

《毒枭》在经过三季于哥伦比亚的反毒战争后将告一段落。就在沃克即将宣布就职的时候，白宫幕僚长却告诉安德伍德，总统将不会提名他为国务卿。故事讲述了坐镇在地狱的冥界之主——路西法，厌倦了冥界无聊的生活的他下定决心抛弃他的地狱王国，他决定放弃王位，前往纸醉金迷的洛杉矶寻求刺激，成了位酒吧老板。但粉丝们在推特上纷纷发帖想要"拯救路西法"，于是Netflix接盘《路西法》，并于2019年回归第四季，Netflix接手后的该剧制作更加精良。

在Netflix的丰富影视库中，挑选出十大必看神剧，仿佛是在繁星点点的夜空中挑选出最耀眼的十颗星星。这些剧集各具魅力，有的如深邃的夜空引人深思，有的如璀璨的流星划破黑暗带来希望。

**毒枭 Narcos**

主演: 瓦格纳·马拉 / 乔安娜 克里斯蒂 / 莫里斯·孔特

类型: 剧情 / 动作 / 传记 / 犯罪

首播: 2015-08-28

![](.evernote-content/0F87163E-B02F-4415-A52C-7ADEEA001197/9CE3E741-8C5B-4779-8B04-E72DCAAC8194.jpg)

故事围绕着二十世纪八十年代哥伦比亚大毒枭巴勃罗·埃斯科巴的传奇人生，讲述了哥伦比亚毒品集团的崛起与覆灭。

走私贩出身的埃斯科巴，凭借着过人的机敏和凶残的手段，建立起了超级毒品帝国：“麦德林集团”——传记类的叙事手法和写实的拍摄风格，让观众们更有代入感，仿佛真的身处南美洲见证了那个血腥的年代，围观着一代毒枭的人生起落。

《毒枭》在经过三季于哥伦比亚的反毒战争后将告一段落。Netflix将原来的第四季独立出来作为衍生剧《毒枭：墨西哥篇》。

本作将聚焦于反毒战争的起源 – 墨西哥的瓜达拉哈拉贩毒集团，时间也将移前至80年代，讲述冷静﹑大胆﹑头脑清晰﹑忠于 亲友的Felix Gallardo（Diego Luna饰）如何把散乱的贩毒者统一起来，成为被称为「Godfather」的大毒枭。

而缉毒局探员Kiki Camarena（Michael Pena饰）带着妻儿从加州搬到瓜达拉哈拉，但他没意料到此反毒战的危险程度，并引发了惨剧。在《毒枭》第一季中曾提到，Kiki被贩毒集团抓获后，死前曾被剥了两腿的皮﹑对着手脚开枪﹑用电钻钻头等。不过他的死也令缉毒局开始对毒枭穷追猛打。

**纸牌屋 House of Cards**

主演: 凯文·史派西 / 罗宾·怀特 / 凯特·玛拉

类型: 剧情

首播: 2013-02-01

![](.evernote-content/0F87163E-B02F-4415-A52C-7ADEEA001197/464A0F5A-D19B-43D4-AED5-A7DC510FBCB5.jpg)

纸牌屋是一部在2013年播出的经典政治题材的美剧，改编自同名小说。

故事的主角弗兰西斯安德伍德是一个非常有政治野心的南卡罗纳 州美国民主党众议员。他帮助加勒特沃克赢得了第45 任美国总统大选，加勒特沃克为了回报他，就将他任命为国务卿。然而，就在沃克即将宣布就职的时候，白宫幕僚长却告诉安德伍德，总统将不会提名他为国务卿。

**浴血黑帮 Peaky Blinders**

主演: 基里安·墨菲 / 山姆·尼尔 / 海伦·麦克洛瑞

类型: 剧情 / 犯罪

首播: 2013-09-13

![](.evernote-content/0F87163E-B02F-4415-A52C-7ADEEA001197/A8C2BC25-7EAF-4741-956D-6032C7D8BC57.jpg)

《浴血黑帮》是奥托巴瑟斯特、柯尔姆麦卡锡等执导的一部电视剧。

《浴血黑帮》讲述了战后伯明翰地区传奇黑帮家族Peaky Blinders的故事。时间要追溯到1919年，家族成员有一大嗜好，就是将剃刀刀片缝进他们帽子的帽檐之间，这也是“剃刀党”的名称由来。

斯里安·墨菲将饰演一名残酷的黑帮份子Tommy Shelby ，是家族兄弟的领袖，嗜血无情。在那个时代，退伍军人、革命者和罪犯，都在社会底层挣扎生存。而当贝尔法斯特的警方负责人开始介入时，Tommy和他的黑帮势力制造出的恐怖统治开始了倾斜。

该剧有剧情，有演技，有质感，有品位，是一部值得一看的电视剧。

**怪奇物语 Stranger Things**

主演: 米莉·波比·布朗 / 薇诺娜·瑞德 / 大卫·哈伯

类型: 剧情 / 恐怖 / 奇幻

首播: 2016-07-15

![](.evernote-content/0F87163E-B02F-4415-A52C-7ADEEA001197/1D5387AD-B579-4274-81CA-7887D573C4A6.jpg)

怪奇物语是由美国NETFLIX公司发行，且于2016年首播的科幻惊悚的美剧。

《怪奇物语》：向征服了一代人的 80 年代经典作品致敬。

《怪奇物语》以 1983 年的印第安纳州为背景，一个小男孩莫名其妙地消失了。当朋友、家人和当地警察寻找答案时，他们都被卷入一个非同寻常的谜团，其中涉及绝密的政府实验、恐怖的超自然力量，还有一个非常奇怪的小女孩。

**黑镜 第一季 Black Mirror**

主演: 罗里·金尼尔 / 鲁伯特·艾弗雷特 / 丹尼尔·卡卢亚 

类型: 剧情 / 科幻 / 惊悚

首播: 2011-12-04

![](.evernote-content/0F87163E-B02F-4415-A52C-7ADEEA001197/9AD8EF3D-3CF8-4306-B30C-72C35A29660E.jpg)

这部剧是由英国电视4台和Netflix联合制作出品的一部迷你电视剧，每集都是一个独立的故事，不同的演员、不同的故事背景、甚至是不同的现实社会，但都围绕当今的生活展开。

在过去10年间，科技就已经改变了生活中的方方面面，每个家庭、每张桌子、每个手掌之间都有一个屏幕、一个监视器、一部智能手机都是一面反映时下现实的黑镜子。

整部剧中都透露着科技进步带来的反思，一些讽刺现实的镜头也同样值得我们每个人去深思。“黑镜”本身指的就是电子设备锁屏状态下的黑色屏幕，打开这面“镜子”你能看到世界的一切，锁上这面“镜子”才是现实中自己的脸。

**路西法 Lucifer**

主演: 汤姆·艾利斯 / D·B·伍德塞德 / 劳伦·日尔曼

类型: 奇幻

首播: 2016-01-25

![](.evernote-content/0F87163E-B02F-4415-A52C-7ADEEA001197/661C1189-E3F8-4ECE-BA99-005A2BA1D27C.jpg)

在《哥谭》作为蝙蝠侠的衍生剧播出后，DC继续决定将旗下的Lucifer也改编成电视剧。这位无聊的地狱之王在王座上过得可是一点也不开心。他下定决心抛弃他的地狱王国，来到洛杉矶成了位酒吧老板，追寻娱乐之都的刺激生活并协助洛杉矶的警察一起惩治罪犯。

故事讲述了坐镇在地狱的冥界之主——路西法，厌倦了冥界无聊的生活的他下定决心抛弃他的地狱王国，他决定放弃王位，前往纸醉金迷的洛杉矶寻求刺激，成了位酒吧老板。直到他因一起凶杀案遇到了克洛伊，引起了路西法的兴趣，从此开始协助洛杉矶的警察一起惩治罪犯，乐此不疲。路西法也感觉到了自身的转变，变得更加正义，更有人性。

该剧的前三季原本是FOX电视台的一部超英剧，三季完结后即被砍。但粉丝们在推特上纷纷发帖想要“拯救路西法”，于是Netflix接盘《路西法》，并于2019年回归第四季，Netflix接手后的该剧制作更加精良。

**绝命毒师 Breaking Bad**

主演: 布莱恩·克兰斯顿 / 亚伦·保尔 / 迪恩·诺里斯

类型: 剧情 / 犯罪

首播: 2008-01-20

![](.evernote-content/0F87163E-B02F-4415-A52C-7ADEEA001197/9A6FD821-DCAF-4048-97F7-AFFD2146575F.jpg)

《绝命毒师》讲述了一位普通的高中化学老师在得知自己身患绝症之后，为了给家人留下财产，而利用自己超凡的化学知识制造毒品，并成为世界顶级毒王的传奇犯罪故事。从第一次制毒开始，作为普通人沃尔特的人生就已经开始逐渐失控，并逐渐坠落至万劫不复的深渊。

作为史上评分最高的美剧，这部剧自2008年开播到2013年剧终，共5季62集，至今依旧拥有居高不下的人气，更是被无数观众奉为经典。

**超感猎杀 Sense8**

主演: 米格尔·安赫尔·西尔维斯特 / 杰米·克莱顿

类型: 剧情 / 科幻

首播: 2015-06-05

![](.evernote-content/0F87163E-B02F-4415-A52C-7ADEEA001197/6247EDCE-2526-4C17-9B0A-3ACD46300E99.jpg)

本剧由《黑客帝国》系列和《云图》导演沃卓斯基姐妹与《巴比伦5号》编剧、著名制作人J·迈克尔·斯特拉辛斯基共同监制此剧。

讲述来自不同城市的八位陌生人，不同身份，不同取向，因为某个神秘事件实现通感，不仅共享情感和思想，还能共享语言和技能。一方面，有个神秘人想将这八个人聚集在 一起；另一方面，一个神秘组织又使尽一切手段猎杀他们。他们必须齐心协力，杀出一条生路，找寻超感真相。

影片八人组的成员包括《自由坠落》的德意志美男马克思·雷迈特，《男孩战争》的美利坚帅哥布莱恩·J·史密斯，《空乘情人》的西班牙型男米格尔·安格·希尔维斯特，《木星上行》和《星图》的韩国女演员裴斗娜等实力演员。

剧集于2015年6月5日在Netflix放出整季后，三天之内就有超过30万Netflix用户观看了该剧，并引发讨论热潮，成为现象级的热门科幻剧集。

**纸钞屋 La casa de papel**

主演: 乌苏拉·科尔维罗 / 伊西娅尔·伊图诺 / 阿尔瓦罗·莫奇

类型: 剧情 / 动作 / 悬疑

首播: 2017-05-02

![](.evernote-content/0F87163E-B02F-4415-A52C-7ADEEA001197/2A89F1B8-E887-4780-94DE-90A564297F36.jpg)

自称教授的天才策划者，召集了一个由八名强盗组成的抢盗团伙，他们每个人都有自己的才能。他们将自己的团队与人质反锁在西班牙皇家造币厂内，通过操纵人质、警察以实现自己的计划。但不管是身为劫匪团队的主角阵容、敌对的警方单位，又或者是受困其中的人质等等，几乎人人都有着各自的立场及想法，因此情况总会因为他们的行动而突然改变。

**王国 Kingdom**

主演: 朱智勋 / 裴斗娜 / 柳承龙 / 金相镐

类型: 剧情 / 恐怖 / 古装

首播: 2019-01-25

![](.evernote-content/0F87163E-B02F-4415-A52C-7ADEEA001197/EE40D901-818A-4DAE-85B4-3C3D0CB91BFB.jpg)

日暮黄昏，妖氛弥漫。昏暗的朝鲜王宫深处，老国王行踪鬼魅，举止若兽。一时间杀气弥漫，朝野上下谣言四起。而在这一切背后，议政府领议政赵学洙（柳承龙 饰）及其女儿——王的继妃赵氏（金慧埈 饰）却在暗暗窥视，图谋不轨。

自然而然，质疑父王生死的世子李苍（朱智勋 饰）便成为赵学洙一党急于除去的眼中钉。遭到构陷的李苍被迫和得力手下武英（金尚浩 饰）出逃，在投奔老师的途中，邂逅了名叫舒非的医女（裴斗娜 饰）。从舒非口中，世子得知一种能令死者复活的神秘草药的存在。

而随着夜幕降临，一群满身创痕甚至腐烂不堪的僵尸突然复活，疯狂扑向每一个活着的人类。

![](.evernote-content/0F87163E-B02F-4415-A52C-7ADEEA001197/0CBC82BB-B229-4126-A60C-CCDAF69BFA06.jpg)

每一部神剧，都有其独特的魅力。它们或让你陷入深深的思考，或让你在欢笑中感受生活的美好，或让你在泪水中体验人性的温暖。这些剧集不仅是视觉的盛宴，更是心灵的洗礼。

那么，这十大必看神剧中，你追过几部呢？是否也有那么一部剧，让你至今仍难以忘怀，深深烙印在你的心中？
