## 摘要

华程高端旅行产品品牌方案（2026.07），对标众信"我的世界拼图"。建立"旅行家·世界志"统一品牌体系，四项硬门槛（16席以内/连住≥30%/一程多章/内容门槛），首批四卷（意大利/哈布斯堡/伊比利亚/荷比卢）。核心差异化：众信做"版图广度"，华程做"主题深度"。

## 正文

### 一、华程现状诊断

已有高端产品基础（御览星辰/御享西葡/御鉴哈布斯堡），但：
- 基本都是10晚，酒店钻级高但无法全板块复制
- 命名/标准/视觉/销售语言各自为战
- **结论：还没有形成高端品牌效应，需从单品升级为统一系列**

### 二、竞对分析——众信"我的世界拼图"

| 指标 | 数值 |
|------|------|
| 独立产品线总数 | ~34条 |
| 欧洲板块占比 | ~48% |
| 欧洲团期 | 34团/16条线 |
| 满员GMV | **3,209万元** |
| 已验证GMV（满团/成团） | 2,163万（67%） |
| 客单中枢 | 5.92万元/人 |

**16条欧洲产品线覆盖**：北极极光/欧洲外三岛/西葡/希腊地中海/荷比卢花期/意大利南北/东欧/英爱/法国一地/波罗的海/德国一地/瑞士一地/大巴尔干等

**众信核心壁垒**：
- 统一母品牌（"我的世界拼图"概念）
- 固定产品语言（16人团/固定模板/金牌领队）
- 可持续复购（完成版图=收藏驱动）
- 定期发售（提前一年公布计划/仅众信渠道/零售价统一）

### 三、华程差异化定位

**众信：完成版图** → 用"去过哪里"推动收藏和复购
**华程：读懂一地** → 用主题/节奏/非标内容建立认知

**Slogan**：一程一卷，读懂世界

### 四、四项硬门槛

| 门槛 | 标准 |
|------|------|
| 16席以内 | 全系列原则上≤16位客人 |
| 住宿 | 连住≥2晚占比≥30%；全程至少1处连住3晚；核心城市入住市中心 |
| 一程多章 | 每条线多条文化/艺术/历史/自然主题，最多3条故事轴线 |
| 内容门槛 | ≥1位当地知己+3项非标体验+2处特色住宿+2次连住体验+集团独家资源 |

### 五、关键定义

**非标体验**：必须同时满足主题相关/稀缺进入/真实互动。常规VIP通道/普通品酒/拍照打卡不计入。

**当地知己**：长期生活或工作于目的地的专业人士（学者/策展人/建筑师/音乐家等），提供≥60分钟有主题的真实交流。不是普通地陪/讲解员。

### 六、首批四卷

| 编号 | 卷 | 主题 | 核心能力验证 |
|------|-----|------|-------------|
| 世界志01 | 意大利卷 | 文明双轴：古罗马与文艺复兴 | 艺术/建筑/生活方式 |
| 世界志02 | 哈布斯堡卷 | 帝国回响 | 历史/音乐/文明（发挥中欧资源优势） |
| 世界志03 | 伊比利亚卷 | 大航海之路 | 长线主题叙事能力 |
| 世界志04 | 荷比卢卷 | 花与城市文明 | 季节限定/首发传播爆点（众信已有3团全满/48300元） |

**意大利示例**（对标众信23天66300元）：
- 华程21天，只讲意大利，不分散到撒丁岛/马耳他
- 罗马/佛罗伦萨3晚市区连住
- 佛罗伦萨2个主题：文艺复兴诞生与发展/美第奇家族

### 七、命名原则

```
旅行家·世界志〔编号〕
目的地卷｜主题名称
天数＋旅程说明
```

减少"御/臻/奢/尊享"等泛化词。编号体现收藏，"卷"体现系列，主题回答为什么去。

### 八、客户旅程设计

| 阶段 | 内容 |
|------|------|
| 看见产品 | 统一封面/章节编号/主题句/四项硬指标 |
| 报名之后 | 需求采集/行前阅读/线上沙龙/领队见面会 |
| 旅程之中 | 当地知己/非标体验/特色住宿/留白时间 |
| 旅程归来 | 旅行手记/章节印章/线上档案/下一卷推荐 |

### 九、实施时间表

| 时间 | 动作 |
|------|------|
| 7-8月 | 确认品牌体系与准入标准 |
| 8-9月 | 完成首批四卷产品打样 |
| 9月 | 完成销售工具（海报/长图/手记/话术/培训） |
| 10月起 | 统一发布并按月复盘 |
