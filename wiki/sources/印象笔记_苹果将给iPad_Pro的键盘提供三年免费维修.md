# 苹果将给iPad Pro的键盘提供三年免费维修

---

![](.evernote-content/DEE9D0B9-9E28-4134-B26A-2CB8C524BE1C/31A3F21E-3AF9-464A-BE61-F3C661A52A08.jpg)

昨天我们报道，有消息称自购买日起 3 年内，如果用户的 iPad Pro Smart Keyboard 出现功能方面问题，苹果会提供免费维修服务。今天国外媒体已经确认了这条消息真实可信。

用户的 9.7 英寸和 12.9 英寸 iPad Pro 的 Smart Keyboard 如果出现了功能问题，比如错误的 Smart Connector、按键重复输入、按键生硬、不响应等，苹果方面将提供免费维修。外媒的报道称，目前苹果零售店和授权维修点已经了解到这项新的政策。

如果你的 9.7 英寸和 12.9 英寸 iPad Pro 的 Smart Keyboard 键盘出现了问题，首先确认是否还在规定的时间内，然后在通过[天才吧](https://gb.feng.com/)、授权服务供应商或者苹果的电话和在线支持等官方渠道来获得帮助。毕竟这是一项新政策，不排除接待你的工作人员可能还没有了解到，这种情况下你或许需要对方向更高一层领 导确认。

苹果认为Smart Keyboard 是iPad Pro不可或缺的一部分，这款键盘能够缩小iPad Pro和笔记本电脑之间的差别。Smart Keyboard 确实支持很多iOS 快捷方式，不需要特别进行匹配或者充电。

今年苹果应该会更新 12.9 英寸 iPad Pro，Smart Keyboard 应该也会迎来新的版本。另外苹果可能还会发布一款 10.5 英寸 iPad 产品。
