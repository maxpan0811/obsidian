下面这些颜色代表 USB 协议版本

|  |  |
| --- | --- |
| **颜色** | **含义** |
|  | 白色  USB1.x，传输速度 1.2Mbps ~12 Mbps |
|  | 黑色 USB2.0，传输速度 480Mbps |
|  | 蓝色 USB3.0，传输速度 5Gbps |
|  | 青蓝色 USB3.1，传输速度 10Gbps |

![](.evernote-content/1C07DE7B-F7F8-42BF-8009-96E6437E4E6D/99A4DBD7-5C66-4CCA-AE32-69A1355ECABE.png)

下面这些颜色代表充电相关

|  |  |
| --- | --- |
| **颜色** | **含义** |
|  | 紫色  代表支持华为的快充 |
|  | 绿色  代表支持高通的快充 |
|  | 黄色、橙色、红色  代表支持大电流或者支持电脑关闭状态下充电 |

**USB  3.x 的进化和关系**

* USB 3.2 包含了以前的 USB 3.0 和 USB 3.1。
* USB 3.0 = USB 3.1 Gen 1 = USB 3.2 Gen 1x1
* USB 3.1 Gen 2 = USB 3.2 Gen 2x1

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 版本 | 名称 | 带宽 | 编码方式 | 备注 |
| USB 3.0 | SuperSpeed | 5Gbit/s | 8b/10b | Type-A、Type-B 接口 |
| USB 3.1 Gen 1 | SuperSpeed | 5Gbit/s | 8b/10b | USB3.0 更名而来，后改名为 USB 3.2 Gen 1x1 |
| USB 3.1 Gen 2 | SuperSpeed+ | 10Gbit/s | 128b/132b | 后改名 USB 3.2 Gen 2x1 |
| USB 3.2 Gen 1x1 | SuperSpeed 5Gbps | 5Gbit/s | 8b/10b | USB 3.0 更名而来 |
| USB 3.2 Gen 1x2 | SuperSpeed 10Gbps | 10Gbit/s | 8b/10b | Type-C 接口 |
| USB 3.2 Gen 2x1 | SuperSpeed+ 10Gbps | 10Gbit/s | 128b/132b | USB 3.1 Gen 2 更名而来 |
| USB 3.2 Gen 2x2 | SuperSpeed+ 20Gbps | 20Gbit/s | 128b/132b |  |

但是，这个并不严格。

大厂商更符合规范，会和上面颜色基本一致。

不过大量的 USB 口颜色没有按照规范来，比如绿色的雷蛇鼠标

因此，这个知识并不一定 100% 正确，仅供参考。
