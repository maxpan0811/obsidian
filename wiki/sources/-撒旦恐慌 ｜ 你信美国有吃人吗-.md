# 最近这几天，中国主流媒体以及网民们很热闹地传说美国社会有吃人现象，关于揭秘爱泼斯坦的“鸡肉婴儿”照片，以及一个十多年前的墨西哥模特儿控诉吃人的视频在中国互联网上疯传，甚至连观察者网也在传播这样的视频：

# 我可以很肯定地说：这些都是在大白天闹鬼，人吓人。

先从观察者网的这个视频说起。这个访谈节目的主持人叫Alexandra Clark，通常被称为Real Alex Clark，是当前美国的保守派网红，Z世代（就是国内说的“95后”）保守主义女性的代表人物之一。在进入政治评论界之前，她曾是一个流行音乐电台主持人。2019年，她被美国年轻人右翼组织Turning Point USA（TPUSA）的创始人Charlie Kirk 发掘——对，就是前段时间被暗杀，举办国葬追悼那位——在TPUSA支持下创办了一个保守派流行文化节目；这在由民主党和左翼人士主导文化界的美国，非常标新立异。![](attachments/d5e22dafd621e872.jpg)Real Alex Clark的观点结合了传统主义、民粹主义以及新兴的食品健康主张，是我前段时间写的《[吃不吃猪油，这是个政治问题](https://mp.weixin.qq.com/s?__biz=MzI5MzMzODE5MQ==&mid=2247499663&idx=1&sn=0ef784d84f876bd7eb67621bc9bc372d&scene=21#wechat_redirect)》MAHA运动的重要媒体代言人，她的节目常为非主流观点提供平台，包括对撒旦崇拜阴谋、秘密社会以及非常规医疗手段的探讨。2024年10月31日在Alex Clark主持的播客节目中，她采访了一位名叫 Kibbi Linga 的嘉宾。Kibbi 声称自己出生在一个由她母亲领导的撒旦邪教家庭，并指控在童年时期被迫目睹并参与了包括活人祭祀、乱伦以及吃人肉等极端罪行。观察网的这个切片就是来自被精选出来的节目预告片片段。不过，原节目明确标注了“听众需谨慎（Listener Discretion Advised）”，并声明内容不适合儿童。但是观察者网在发视频时，没有显示这段提示语。这个指控涉及极其严重的刑事犯罪，但目前并没有公开的法律文件或官方调查，证实Kibbi Linga 所描述的具体案件。实际上不仅是Alex Clark的这次访谈，这位Kibbi Linga在美国多个访谈节目中，都曾经讲过这段耸人听闻的经历，但她从未提供过关键的核实信息，例如她父母的姓名、事发地点的确切地址，或者任何官方的调查记录。Kibbi Linda的公开身份是一名网红画家，她将“撒旦教仪式虐待（SRA）幸存者”作为网络传播的个人标签，在自己运营的播客上专门讨论摆脱虐待、信仰重建以及心理健康话题，分享自己在逃离邪教家庭后，重新建立了对上帝的信仰——这就是为何她被邀请上Real Alex Clark访谈节目的原因。Kibbi在社交媒体上出售自己的艺术作品，这些作品通过鲜艳的色彩，表达痛苦与重生的主题。![](attachments/a8642e22c2a402a3.png)Kibbi在美国社会有不少支持者，也有很多人怀疑她可能是为了社交媒体流量而编造故事，或者确实处于某种精神不稳定的状态。心理学研究表明，一些声称经历过“撒旦仪式虐待”（SRA）的个体可能患有解离性身份障碍（DID）、复杂性创伤后应激障碍（C-PTSD）或受到“恢复记忆疗法”的影响。这些疾病可能导致个体产生极度真实但并非事实的记忆幻觉。而实际上，Kibbi的指控在美国社会由来已久了，她的这些行为可以认为是20世纪80年代至90年代席卷欧美社会的“撒旦恐慌”（Satanic Panic）的延续。当时，欧美社会广泛相信存在一个庞大的地下撒旦教网络，正在有组织地绑架、虐待甚至吃掉儿童。尽管涉及数万起指控和数次耗资巨大的侦查、审判，但最终没有任何证据证明这些地下邪教网络的存在。1980年出版的畅销书《米歇尔的回忆》（Michelle Remembers）被公认为“撒旦恐慌”的导火索。这本书由加拿大精神科医生劳伦斯·帕兹德（Dr. Lawrence Pazder）和他当时的病人（后来成为他妻子的）米歇尔·史密斯（Michelle Smith）合著。![](attachments/972f80f23c5e90ae.png)书中声称记录了米歇尔通过治疗“恢复”的童年记忆。她指控在1954年至1955年间，她的母亲将她交给了一个位于不列颠哥伦比亚省维多利亚市的撒旦邪教。书中描述了各种恐怖细节，包括长期囚禁、婴儿祭祀、吃人肉等，最终，圣母玛利亚和耶稣亲自显灵，帮助她在一场大规模仪式中击败了撒旦。观察者网发的那条Kibbi指控视频，跟四十五年前的这本书的内容几乎如出一辙。该书出版的目的其实是推广帕兹德医生的“恢复记忆疗法”（Recovered Memory Therapy），他宣称通过数小时的催眠和诱导性谈话，解锁了米歇尔压抑多年的痛苦记忆。那么问题来了：各种果总的读者，你觉得这个心理学疗法靠谱吗？还是正好起到了反作用——心理医生通过诱导对话将幻觉植入了这个女人脑中？几年后，这本书诱发了是美国历史上耗时最长、开销最高的司法闹剧之一 ——麦克马丁幼儿园案(McMartin Preschool Trial)。1983年，加州曼哈顿海滩的一位母亲（后被证实患有偏执型精神分裂症）向警方举报，称她儿子在麦克马丁幼儿园遭到创始人孙子的性侵。案件初审时，司法调查人员对孩子说：“如果你不告诉我们那些坏事，你妈妈就会生病。”在诱导下，数百名孩子描述了极其荒诞的情节：老师在学校里杀掉大象和乌龟、通过秘密地下隧道运送孩子、强迫他们喝血和吃人肉、甚至坐飞机去外地参加邪教仪式。帕兹德医生书中推广的那套压抑记忆理论，让当时的检察官相信：即便孩子说不出细节，或者描述得极其荒谬，那也是因为创伤太重导致记忆扭曲，从而无视了证据的缺失。而那本书在社会上畅销，让公众相信撒旦教就在身边。当麦克马丁案爆发时，人们不再认为这是一家幼儿园的问题，而是认为《米歇尔的回忆》里写的邪教终于被曝光了！这像不像我们现在互联网上反转基因、反预制菜的现象？经过7年的司法调查、诉讼和1500万美元的支出，法庭最终承认所谓的地下隧道根本不存在，那些被杀死的动物尸体也从未被找到。然而，幼儿园方麦克马丁家族的声誉和生活被彻底摧毁，被告在监狱里被关押多年。就像我在《[西贝现象观察（26）县城消费裁判一线城市生活](https://mp.weixin.qq.com/s?__biz=MzI5MzMzODE5MQ==&mid=2247499740&idx=1&sn=8556183e7941820247446b9b6a0aebd5&scene=21#wechat_redirect)》文中所写，预制菜恐慌是中国社会的县城文化和社会现代化的冲突，撒旦恐慌是西方社会的保守主义和现代社会的冲突。在这场冲突中，看起来有点邪门的角色扮演桌游《龙与地下城》(D&D)以及重金属摇滚乐躺枪了，这也推进了撒旦恐慌的蔓延。![](attachments/2d9a429197a79545.jpg)1979年8月，密歇根州立大学一名十六岁学生，智商高达180的詹姆斯·达拉斯·艾格伯特三世（James Dallas Egbert III）突然从宿舍消失，留下了一封疑似遗书。他的父母的雇佣了私家侦探在得知艾格伯特是《龙与地下城》游戏玩家后，向媒体提出了一个耸人听闻的假说：艾格伯特因为沉迷游戏，在大学地下的蒸汽隧道里玩“真人版D&D”时迷失了方向，或者产生了幻觉。这一假说迅速被媒体放大，导致这个游戏被和撒旦邪教联系起来。尽管后来侦探找到了艾格伯特，一年后这个孩子还是枪击自杀身亡。后来各种调查显示，他失踪的真正原因是巨大的学业压力、药物成瘾，以及在那个保守时代对自己同性恋身份的痛苦挣扎。然而美国社会并没有接受真相，将少年失踪归咎于这个撒旦化桌游的类似事件多次发生，一些保守派家长成立了“反D&D家长联盟”。1982年，一部基于艾格伯特事件改编，实际上是利用社会偏见制造恐慌的低成本电视电影《迷离迷宫》(Mazes and Monsters)上映，片中扮演受妖魔化游戏诱惑的学生的，就是今天大家都知道的汤姆·汉克斯（Tom Hanks）——你们应该看过他演的《拯救大兵瑞恩》、《阿甘正传》（参见《[四看阿甘正传 ｜ 自由不是追求来的](https://mp.weixin.qq.com/s?__biz=MzI5MzMzODE5MQ==&mid=2247489397&idx=1&sn=dbc381fa183d88caa662c00b00707901&scene=21#wechat_redirect)》《[阿甘正传影评（二）](https://mp.weixin.qq.com/s?__biz=MzI5MzMzODE5MQ==&mid=2247489404&idx=1&sn=5770da8e3df86e6ce34b463326af3ad5&scene=21#wechat_redirect)》），而这是他的荧幕处女作；他成名后一直不愿提起这部让人尴尬的电影。![](attachments/59cf1bb8b2e4f647.jpg)80年代新兴的重金属摇滚因其叛逆的形象和黑暗的歌词，被教会视为“撒旦的颂歌”。当时广为流传的一种阴谋论称，如果将重金属音乐的磁带倒序播放，就能听到隐藏的撒旦指令（如“为撒旦杀人”）——90年代中期，一位美国朋友给我讲过这个事情，不过他是我那时的摇滚乐密友，他说起来是戏谑美国社会荒唐的。重金属常用的“摇滚手势”🤘被解读为恶魔之角；专辑封面的骷髅和五角星等被视为召灵符号。1990年，重金属乐队Judas Priest（犹大圣徒）被起诉，称其音乐中的潜意识信息导致两名少年自杀。而Ozzy Osbourne的歌曲《Suicide Solution》也曾多次面临类似的法律诉讼。为此，美国副总统戈尔的妻子提派·戈尔成立了家长音乐资源中心(PMRC)，推动了大家熟悉的“家长指引：露骨内容” (Parental Advisory)标签的诞生。![](attachments/a978e62f3ca6f004.png)![](attachments/308c8bf23a3df88a.jpg)当年这种现象在西方社会生成，是多种社会焦虑交织的产物：1、家庭结构变革：20 世纪 70-80 年代，更多女性进入职场，托儿所和幼儿园需求激增。家长们对于将孩子交给陌生人照顾感到焦虑和内疚，这种心理被“撒旦教徒潜伏在幼儿园”的谣言放大。2、宗教与政治：“新右翼”和原教旨主义基督教在 80 年代兴起。他们利用撒旦作为绝对邪恶的象征，攻击性解放、摇滚乐等世俗文化，以强化传统价值观。3、媒体推波助澜：当时的脱口秀和报纸为了收视率，频繁邀请所谓的“专家”和“幸存者”讲述耸人听闻的故事，将个案渲染成全球阴谋。4、心理学诊断潮流：人们以为心理学手段能揭开被压抑的真相，心理医生和社工当时广泛使用诱导性提问或催眠，导致儿童和患者产生“虚假记忆”。虽然到90年代中期，这场恐慌就逐渐消退了，但恐慌的核心谣言，如“精英阶层秘密虐待儿童”，在欧美社会中一直存在，近年来典型表现就是QAnon阴谋论。QAnon是一个起源于美国、在欧美社会极具影响力的极右翼阴谋论体系。它的逻辑建立在对全球精英的极度不信任之上。QAnon追随者相信，世界被一个由政客（主要是民主党）、亿万富翁、好莱坞明星和官僚组成的秘密集团（Cabal）统治。他们指控这些精英不仅是腐败的，还是崇拜撒旦的食人恋童癖。QAnon宣称精英阶层通过折磨儿童来提取一种名为“肾上腺素红”（Adrenochrome）的物质，以达到长生不老的目的。2017年10月，一个自称“Q”的匿名用户在4chan论坛上发帖。该用户声称自己是拥有美国最高国家安全权限（Q级权限）的高级官员。“Q”经常发布一些含糊不清、像密码一样的暗语，引导追随者去解读和拼凑所谓的“真相”。QAnon利用4chan、Reddit和各类社交媒体的算法推荐，形成了一个全球性的信息茧房；支持者通过参与解析Q发布的暗语，获得了一种类似解谜游戏的参与快感。![](attachments/3284449ba4d7ca8a.png)80年代撒旦恐慌时期，受攻击目标通常是随机的、具有社会焦虑象征的对象，如幼儿园老师、重金属歌手或亚文化群体。而今天的QAnon时期，敌人被具象化为特定的政治和文化精英（如民主党领导人、好莱坞明星和全球主义者），这种转变将道德恐慌直接转化为政治动员工具。QAnon的终极预言是，唐纳德·特朗普是被选中的救世主，他正在进行一场秘密战争摧毁这个邪恶集团。他们期待着一场名为“大风暴”(The Storm)的清算日，届时那些邪恶的精英们将被大规模逮捕、审判甚至处决。![](attachments/5ad30f9a70e04b91.jpg)这样的社会背景就是爱泼斯坦事件导致萝莉岛、吃人等各种稀奇古怪的阴谋论在美国社交媒体上发酵的原因，然后那些美国段子就蔓延到中国网络上来，让一些中国网民们津津乐道了。![](attachments/5c5a7615b07ef86e.jpg)
