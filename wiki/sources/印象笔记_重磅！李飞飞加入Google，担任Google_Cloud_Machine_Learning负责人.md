重磅！李飞飞加入Google，担任Google Cloud Machine Learning负责人
-------------------------------------------------

*2016-11-16**老尼姑*AI那点事

![](.evernote-content/9BD2BD2E-FAB7-46BF-9277-74DE35D648B2/4CDF8D8B-3FE2-4398-9E8B-20147EFB285A.jpg)

学术界大牛投身工业界的浪潮还在继续，现在又多了斯坦福AI实验室负责人Feifei Li这位圈内最知名的女性代表。

虽然加入了Google，但是据她实验室的同学说，“It's sabbatical，not long term full time.”不过，这种事情很难界定，比方说Hinton一直都说学界业界一半一半，其实基本上都是泡在Google。

**附简介：**

李飞飞出生于北京，长在四川，16岁随父母移居美国，斯坦福大学计算机系终身教授，人工智能实验室与视觉实验室主任。

李飞飞教授主要研究方向为机器学习、计算机视觉、认知计算神经学
