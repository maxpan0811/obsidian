## 一句话摘要

5.9K Star 的 Claude Code 实战教程 v2.2.0——不是功能列表，是带图、带模板、带学习路径的实战教程，11-13 小时学完直接用。

## 核心亮点

- **结构化学习路径**：10 个模块，循序渐进
- **即用模板**：slash 命令、CLAUDE.md、技能、子代理、MCP 配置、Hooks 全部写好
- **Mermaid 流程图**：讲明白原理
- **自测 quizzes**：定位知识盲区，个性化推荐
- **保持更新**：紧跟 Claude Code 版本

## 解决的问题

官方文档只列功能，不说怎么组合起来干活。claude-howto 填补了这个空白——带图、带模板、带学习路径。
