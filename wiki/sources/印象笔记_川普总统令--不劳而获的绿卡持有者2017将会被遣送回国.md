川普总统令--不劳而获的绿卡持有者2017将会被遣送回国
----------------------------

*2017-04-29*[TheHouseClub](http://mp.weixin.qq.com/s?__biz=MzA3MTY2ODAyNA==&mid=2651144418&idx=6&sn=e634ab4c29e996773ce077e268ef011c&chksm=84dbf8b5b3ac71a3218586a24a84bd2f9b0d5655b93b0da8bbd615528dea1c49291e52c5afb8&mpshare=1&scene=1&srcid=0506fjkQD9nbhi3ZPwtRJirk##)

![](.evernote-content/94E9A394-6F62-4EB6-8254-001257C4F019/01189B51-579B-4C6F-ACB9-390932878696.jpg)

新官上任三把火，川普才上任不久就签署多项总统令。

```
以下是关于移民政策的：新总统严打非法移民，甚至波及50万绿卡持有者。
```

![](.evernote-content/94E9A394-6F62-4EB6-8254-001257C4F019/B18CB89E-8FD2-4456-BB0A-0CCE51DD66B3.jpg)

> **第一：合法移民不得再享受美国福利。**

```
如果移民之前就是低收入需要领政府救助的，就停止发放绿卡，如果已经进入美国，一但领了政府资助，就马上遣返。  
  
也就是如果你有绿卡但失业了，一但领了政府低保，绿卡就会被吊销。
```

![](.evernote-content/94E9A394-6F62-4EB6-8254-001257C4F019/C8F642F6-C710-4415-9225-2E4FD6F7C60F.jpg)

```
比如以前父母移民在加州纽约麻省一些州可以享受medicaid免费医疗，因为不是现金，但现在马上不允许了。类似的还有老年公寓也不准绿卡去住了。用了也属于deportable。如是亲属移民的sponsor,  政府将让绿卡本人赔偿。 文件规定OMB必须出台一个报告，写明移民（不是非法移民）如何占有了美国100 billion美元的资源。这其实是对合法移民的一个先遣令，随后法令会让OMB去论证。同时，联邦政府将出台一个定期报告，对比非美国出生的移民家庭（注意不是非法移民）和美国本土出生家庭（特别写的native born)对美国资源的占有如何不同，目的是让美国人（本土人）认定移民家庭如何浪费了美国资源。
```

> ```
> 第二： 老年公寓，免费医疗，现在都不允许。如果去住了，也会被遣返。
> ```

![](.evernote-content/94E9A394-6F62-4EB6-8254-001257C4F019/5E0EEF76-1743-4574-A83B-80BBD5762AC2.jpg)

> ```
> 第三： 合法劳工的问题，将改革H-1B分配方式，不允许移民抢本国人的工作，甚至实习，OPT, CPT 也会被改革。
> ```

```
具体内容为：   
取消H-1B抽签制度，只为毕业美国高校的学生优先提供H-1B；  
对于H-1B的雇主要求： 员工人数超过50，且半数员工持有H-1B 和L1工作签的公司不得申请新的H-1B，也提高雇佣外籍员工的薪水标准。  
雇佣外籍员工的美国外包公司将首当其冲。  
  
这条政策只允许移民中最聪明的人留下。重新审查合法移民的排期，重新检查L1。  
  
重新修改工作移民EB, 不准抢美国人工作，可以预见以后排期会越来越长，多半还有很多security check。  
  
移民热衷的国家利益豁免和EB1也将非常难了。重新检讨OPT、CPT，不准外国学生抢美国学生的实习，只能回国。  
重现检讨L1，重新审核合法移民的排期，规范485身份调整。
```

![](.evernote-content/94E9A394-6F62-4EB6-8254-001257C4F019/C766D0FE-CED1-4953-8C91-694638B2A3ED.jpg)![](.evernote-content/94E9A394-6F62-4EB6-8254-001257C4F019/7509FCDC-F7A2-4718-B50E-954AFB018F7F.jpg)

> ```
> 第四： 严厉打击在美国旅游期间就非法滞留美国，变成美国身份的人。
> ```

```
严打tourism citizenship ，要求联邦政府发表发表定期报告，展示合法和非法移民对美国就业的抢占，让多少美国人没有工作，特别用了legal and illegal immigrant workers。此结果导致本土美国白人更加憎恨外国来的移民了。
```

![](.evernote-content/94E9A394-6F62-4EB6-8254-001257C4F019/04D24CDE-E87F-454B-9734-A68D92B91D27.jpg)

```
美国新闻媒体资深评论员Thomas Friedman说道，美国对移民政策的重大调整，将会对欧洲及其它移民国家的政策产生深远影响。
```

![](.evernote-content/94E9A394-6F62-4EB6-8254-001257C4F019/4BB02815-198C-42F1-809F-FE64D26AE9E8.jpg)

> ```
> 第五： 川普新政实施全面禁穆，七个主流的穆斯林国家全面施行禁止发放签证和绿卡，对于已经持有绿卡的穆斯林也施行最严格的审查。
> ```

```
以上政策可以开出，川普是动真格要优先保护本国人的利益，这个政策不但影响非法移民，也影响50万绿卡持有者。也会对欧洲的移民政策产生深远影响。
```

![](.evernote-content/94E9A394-6F62-4EB6-8254-001257C4F019/1737A6D0-033E-4698-867B-87838BAA8E08.jpg)

**专家建议**

```
： 如果办十年签证去美国，留美期间，用旅游签证不要去读书，也不要生孩子，也不要相亲结婚。
```

![](.evernote-content/94E9A394-6F62-4EB6-8254-001257C4F019/C4191C76-8067-4806-9841-0D5513A619D7.jpg)

```
第二：  招生录取并不受这个政策的影响。留学容易，只是留下来难。因此，来美留学的学生要想清楚毕业去向，回国还是留美？如果留美，是否有足够的实力留美？
```
