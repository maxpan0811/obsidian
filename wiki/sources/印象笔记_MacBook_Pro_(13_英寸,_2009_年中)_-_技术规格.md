MacBook Pro (13 英寸, 2009 年中) - 技术规格
===================================

### 尺寸和重量

高度:
:   2.41 厘米 (0.95 英寸)

宽度:
:   32.5 厘米 (12.78 英寸)

深度:
:   22.7 厘米 (8.94 英寸)

重量:
:   2.04 千克 (4.5 磅)1

### 外设连接和扩展

* MagSafe 电源适配器端口
* 千兆以太网端口
* 一个 FireWire 800 端口 (速率高达 800 Mbps)
* Mini DisplayPort 端口
* 两个 USB 2.0 端口 (速率高达 480 Mbps)
* SD 卡槽
* 音频输入/输出
* Kensington 线缆锁孔

### 通讯功能

* 内置 AirPort Extreme Wi-Fi 无线网络2 (基于 IEEE 802.11n 草案规范）；与 IEEE 802.11a/b/g 兼容
* 内置蓝牙 2.1 + EDR (提升的数据速率)
* 内置 10/100/1000BASE-T 千兆以太网卡 (RJ-45 接头)

### 音频

* 内置立体声扬声器
* 内置全向麦克风
* 光纤数字输出/耳机输出组合 (可选音频模拟输入)
* 支持附带麦克风的苹果立体声耳机

### 环境状况报告

* 高度可回收的铝合金和玻璃材质外壳
* 不含汞的 LED 背光显示屏
* 不含砷的显示屏玻璃
* 不含溴化阻燃剂（BFR）
* 不含聚氯乙烯（PVC）的内部线缆
* 符合 ENERGY STAR 能源之星 5.0 版要求
* 获PP EPEAT 金级认证

### 声音性能

噪声发射标示值符合 ISO 9296 标准

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
|  | 声功率级  LWAd (B)  1B = 10dB | | 声压级  操作员位置  LpAm (dB) | |
| 2.26 GHz | 2.53 GHz | 2.26 GHz | 2.53 GHz |
| 闲置 | 2.6 | 2.6 | 12 | 12 |
| 硬盘驱动器访问 | 2.6 | 2.6 | 11 | 11 |
| CD 驱动器访问 | 3.2 | 3.2 | 23 | 24 |

1. LWAd 表示 A 计权声功率级的统计上限值（四舍五入到最接近的 0.1B）。
2. LpAm 表示在操作员位置处测得的 A 计权声压级（四舍五入到最接近的 dB）。
3. 1B (bel) = 10dB（分贝）。

### 显示器

* 13.3 英寸 (对角线）LED 背光光面宽显示屏，支持百万色彩
* **支持分辨率**: 16:10 宽高比可显示 1280 x 800（初始）、1152 x 720、  
  1024 x 640、800 x 500 像素；4:3 宽高比可显示 1024 x 768、800 x 600 和 640 x 480 像素；4:3 拉伸宽高比可显示 1024 x 768、800 x 600 和 640 x 480 像素；3:2 宽高比可显示 720 x 480 像素；3:2 拉伸宽高比可显示 720 x 480 像素

### 图形处理器和视频支持

* NVIDIA GeForce 9400M  图形处理器，与主内存共享 256MB DDR3 SDRAM 显存3
* 双显示与视频同步：同时支持初始分辨率下的内置显示屏和高达 2560 x 1600 像素的外接显示屏，两台显示屏都支持百万色彩
* 内置 iSight 摄像头
* Mini DisplayPort 端口

### 纯数字视频输出

**DVI 输出** 使用 Mini DisplayPort 端口至 DVI 转换器 (需单独购买)

**VGA 输出** 使用 Mini DisplayPort 端口至 VGA 转换器 (需单独购买)

**Dual-link DVI 输出** 使用 Mini DisplayPort 端口至 Dual-Link DVI 转换器，支持 30英寸 Apple Cinema HD 显示器 (需单独购买)

### 输入

* 内置 78（U.S.）键或 79（ISO）键的标准尺寸背光键盘，包括 12 个功能键，4 个方向键（倒"T"形排列）
* Multi-Touch 触控板实现精准游标控制；支持双指滚读、开合、旋转、轻扫、三指轻扫、四指轻扫、单击、双击和拖曳操作

### 处理器和内存

* 2.26GHz 或 2.53GHz Intel Core 2 Duo 处理器，3MB 片上共享二级缓存，以处理器速度全速运行
* 1066MHz 前端总线
* 2GB (两个 1GB SO-DIMM) 或 4GB (两个 2GB SO-DIMM) 1066MHz DDR3 内存; 两个 SO-DIMM 插槽，最高支持 8GB

### 存储设备

* 160GB 或 250GB 5400-rpm Serial ATA 硬盘4
* 8 倍速吸入式 SuperDrive（DVD±R DL/DVD±RW/CD-RW）光驱
* 最大刻录速度：8x DVD-R 和 DVD+R；4x DVD-R DL（双层）、DVD+R DL（双层）、DVD-RW 和 DVD+RW；24x CD-R；10x CD-RW
* 最大读取速度：8x DVD-R、DVD+R、DVD-ROM；6x DVD-ROM（双层 DVD-9）、DVD-R DL（双层）、DVD+R DL（双层）、DVD-RW、DVD+RW；24x CD

### 电池和电量5

* 60 瓦时内置锂电池
* 60W MagSafe 电源适配器，具有线缆管理系统
* MagSafe 电源适配器端口

### 电器性能和操作环境

* 电压：100 至 240 伏特交流电
* 频率：50 至 60Hz
* 操作温度：10℃ 至 35℃（50°F 至 95°F ）
* 存放温度：-24℃ 至 45℃（-13°F 至 113° F ）
* 相对湿度：0% 至 90% 间，非冷凝状态
* 最高操作高度：3,000 米（10,000 英尺）
* 最高储存高度：4,500 米（15,000 英尺）
* 最高装运高度：10,667 米（35,000 英尺）

### 随机装箱

* MacBook Pro
* 显示屏清洁布
* 60W MagSafe 电源适配器, AC 插头和电源线
* 安装/恢复 DVD
* 印刷和电子版手册

### 软件

* Mac OS X v10.5 Leopard（Time Machine, Quick Look, Spaces, Spotlight, Dashboard, Mail, iChat, Safari, Address Book, QuickTime, iCal, DVD Player, Photo Booth, Front Row, Xcode 开发者工具）
* iLife ’09（包括 iTunes, iPhoto, iMovie, iDVD,  
  iWeb, GarageBand）

### 可选配件

### Time Capsule

Time Capsule 是拥有内置硬盘的全功能 AirPort Extreme 基站。通过单一设备即可无线备份并建立无线网络连接。
