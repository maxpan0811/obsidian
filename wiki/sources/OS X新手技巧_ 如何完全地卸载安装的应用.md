新手要适应OS X系统可能需要一点时间，不过不要紧，有各种新手教程为你导航。

　　威锋网讯，OS X 和 Windows 系统在各个方面都存在着不大不小的差异，对于部分由 Windows 转向 OS X 系统的用户来说，他们需要适应的有很多，其中就包括了在 OS X 中完全地卸载应用。  
  
　　如果想让应用完整地从系统中删除的话，可能需要几个步骤，今天我们不妨来分享一下锋友@夏虫不可以语冰为我们带来的教程：  
  
　1.Activity Monitor  
  
　　打开活动监视器（Activity Monitor），在菜单栏中（上方）选择［所有进程］，确保你要卸载的 APP 已经不再运行，如果看到监视器中该 APP 依旧有活动进程，则需手动在活动监视器中结束此应用进程，之后关闭再检查一次。
