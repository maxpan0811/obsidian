# 加个外置SSD就能让你的旧款iMac满血复活

---

[![](.evernote-content/F1D5D9F2-E826-4348-9C35-78B961378256/98C1EA52-1F4E-4973-B2CF-147958C721B0.jpg)](http://www.feng.com/iPhone/news/2016-03-28/Add-a-external-SSD-can-let-your-older-iMac-resurrection-full-of-blood_642786.shtml)

　我有一台 2011 年的 iMac，使用几年之后，它的速度已经变得有点慢。于是，我选择为硬盘减负，之后，它的速度稍稍提高，但是和 2015 年的 iMac 还是有不少差距。  
  
　我可以增加更多内存，但是我不确定它能不能达到我的预期。我可以为硬盘腾出一点空间，但这似乎也不是最好的办法。我真正需要做的是，将 OS X 操作系统安装到 SSD 上。  
  
　我有一台 2010 年的 MacBook Air，它的速度比 2011 年的 iMac 还快，这是因为我为 MacBook Air 配备了外置固态硬盘，它的速度比标准硬盘驱动快。MacBook Air 的操作系统就被我安装在固态硬盘上，而 iMac 没有。  
  
　由于照片库转移的关系，我当前在用的三星 SSD 已经快被装满。然后，我就决定去 LaCie 那儿走走，看看能不能买点什么，让我的旧款 iMac 速度变快一点。  
  
　我看到 LaCie 有一款 Rugged Thunderbolt 500GB SSD ，它是一个银色为主色的硬盘，还配有橘色，看起来让人联想起小型的救生艇。  
  
　虽然 Rugged Thunderbolt 500GB SSD 不像三星 SSD 那样便携，但它具备防水、防尘以及防震功能，生存技能 Max，所以我将其纳入考虑范围。

![](.evernote-content/F1D5D9F2-E826-4348-9C35-78B961378256/98C1EA52-1F4E-4973-B2CF-147958C721B0.jpg)

Rugged Thunderbolt 500GB SSD 使用 Thunderbolt 接口插入，这使得我 2011 年款 iMac 拥有了最快的数据传输速度。

Rugged Thunderbolt 500GB SSD 可以让你运行安装程序，它允许你划分硬盘（如果你打算在 Mac 和 Windows PC 之间来回切换的话）。因为我主要是打算用它来安装 OS X 操作系统，所以我直接用了 HFS+。

在外置硬盘上安装 OS X 操作系统易如反掌。即使你已经在 Mac 上安装 El Capitan 操作系统，你也能够从 App Store 免费下载安装程序。

下载完毕之后，将应用指定到目标硬盘（比如我的目标硬盘就是 LaCie），然后跟着指示操作，你就能够顺利安装。

![](.evernote-content/F1D5D9F2-E826-4348-9C35-78B961378256/2CC753F8-39A7-4D75-AAED-AE81906A4324.jpg)

如果你使用 Migration Assistant 软件帮助，那请在确保你的新硬盘有足够空间的时候才选择你当前账户。由于我没有将 Photo Library 从硬盘上删除，空间不够，于是我取消了安装。

在 OS X 系统正在安装时，你可以去吃个午餐，或者去看一部电影，等你回来，你会发现系统已经安装完毕。

接下来，你就要确定一切就位。比如，假设你使用 Dropbox，那你就得重新启动这个程序，检查看看你的用户账号是不是也在。这是一个安全问题。

OS X 会自动安装，不过你还是在系统偏好下面检查一下 Startup Disk 设置，确保你添加的新硬盘是系统选择用于启动电脑的硬盘。

如果你在初次安装的时候没有将用户账号迁移过来，那你就需要使用 Migration Assistant 来完成账号迁移任务了。当然，在迁移的过程中，你要做的也是等待。

在将 OS X 操作系统安装到外置硬盘之后，我的旧款 iMac 终于快到飞起。

如果我选择了更大的 SSD，比如达到 1TB 的硬盘，那我就可以将整个系统都搬过去。可是，相比让硬盘挂在电脑后背，我更喜欢将这些快速的硬盘塞到 iMac 中。不过，大家也要明白，升级 iMac 是一个不容易的任务，要和很多杂乱的线缆打交道。

今天这种自行升级 iMac 的办法只适合极客使用，而且这种升级办法还有一定的压力，生怕哪一步出错，如果你抗压能力不够，手动能力也不够，那就不要随便尝试，如果你资金不紧张的话，其实也可以考虑 2015 年的 iMac 或者 MacBook。

![](.evernote-content/F1D5D9F2-E826-4348-9C35-78B961378256/EB62F67C-8D01-460F-B11D-4F72200E7B05.jpg)

去年 10 月份，苹果发布全新 4K 和 5K iMac，以及 Magic Keyboard 键盘、Magic Mouse 2 鼠标和 Magic Trackpad 2 触控板。全新的 iMac 一体机搭载了更快的处理器和显卡，并加入了两个 Thunderbolt 2 接口。

21.5 英寸的 iMac 机型价格适中，拥有轻薄漂亮的外观，用户可选 1.4GHz、2.7GHz 及 2.9GHz 三款不同 Haswell 处理器型号，以及 500GB、1TB 存储，HD 5000、Iris Pro 以及 NVIDIA GeForce GT 750M 显卡，能够适用于办公、娱乐、影音处理等不同需求。

如果你对视频编辑有超高要求，你可以考虑 5K iMac。配备 Retina 5K 显示屏的 iMac 可以说是目前影音性能最强大的一体机电脑，27 英寸、高达 5K 分辨率的屏幕，可以获得极致清晰的显示效果；值得一提的是，5K 27 寸 iMac 升级到了六代酷睿 Skylake，是目前唯一一款采用英特尔第六代处理器的 iMac，可以选择 Core i5-6500/6600、Core i7-6700K。

另外 3.3GHz、3.5GHz 酷睿 i5 或 4Ghz 酷睿 i7 处理器及 AMD Radeon R9 M290X 显卡提供的强劲综合性能，再加上 5K 屏幕出色的显示效果，非常适合用于编辑超高清视频。

根据苹果的介绍，在 2015 年 27 英寸 5K Retina iMac上，因为有了更为宽广的全新色域，使 Retina 显示屏可表现颜色的数量提升了 25%，因此，你浏览的照片和视频，都更加栩栩如生。

2015 年的 5K Retina iMac 中，苹果已经不满足于使用 sRGB 标准，所以他们将目光放在了 P3 色彩标准上。P3 是由美国电影电视工程师协会 2007 年的时候提出的一种新标准，也是 RGB 的一种。

采用了 P3 标准的 5K iMac 得到业界盛赞。华尔街日报指出：看着新款 iMac 的屏幕，你会发现它显示的绿色和红色更加明亮，好像有人把饱和度调高。因为显示器不会删除某些色调，所以新屏幕可以显示的色彩和细节更多。
