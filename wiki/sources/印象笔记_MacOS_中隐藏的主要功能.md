最有用的隐藏功能可改善数字体验。

### 介绍

MacOS 有很多隐藏的宝藏，但很多人却没有注意到。有些是数字便利；有些可以改变游戏规则，提高生产力并简化工作流程。

以下是将改变 Mac 体验的顶级隐藏功能！

### 快捷方式应用程序 + Raycast

想象一下工作时间到了，您可以：

![](.evernote-content/B0E2F0D3-9C84-4FA9-A063-8EFE09C33ABD/9F5C7455-A145-495A-9D4A-56043B04551F.gif)

这是使用快捷方式应用程序创建的工作流程。

它是一款用于创建事件链并将其转换为快捷方式的工具。它可在所有 Mac 以及 iPhone 和 iPad 上原生使用。

在这里，我设置了编码时使用的打开和调整应用程序大小的链。

![](.evernote-content/B0E2F0D3-9C84-4FA9-A063-8EFE09C33ABD/BA4E5262-B7E7-4E0D-B709-81E9E8F44F42.png)

每次我想要工作时，这可以节省我大约 15 秒的时间，并且看到所有应用程序都打开并定位到正确的位置，我感到很满意。

以下是完整工作流程的细分：

![](.evernote-content/B0E2F0D3-9C84-4FA9-A063-8EFE09C33ABD/7A67F26E-5950-4B76-8D75-8914CC2A27D8.jpg)

开启工作专注模式后，通知会静音，然后打开四个应用程序，所有应用程序都具有调整大小的功能，可以将它们放到适当的位置。

### 无限可能

打开和关闭应用程序非常简单。此应用程序集成了第三方软件，能够操作日历和待办事项列表、使用链接拨打电话、运行 shell 命令，甚至可以将其他快捷方式串联在一起。

这款应用的变革性非常强，值得专门写一篇文章，因为不可能在一个章节中介绍它的所有功能。刚开始使用时可能有点复杂，但从长远来看绝对值得。

仍有一个问题：如何有效地激活这些快捷方式。

### 选择

当我们需要时，在快捷方式应用程序中手动单击它们。将快捷方式直接作为小部件添加到桌面，并且由于小部件是交互式的，我们可以从那里单击它们。使用 Spotlight，但是快捷方式太多就会很混乱，或者 Raycast。

### 光线投射

我几乎在每一篇我撰写的 macOS 文章中都大力推崇 Raycast，这是有原因的。没有一个应用程序能像 Raycast 一样改变我的工作流程。

![](.evernote-content/B0E2F0D3-9C84-4FA9-A063-8EFE09C33ABD/88671D7E-4C8E-4471-A3D3-512C83DCD805.jpg)

它就像聚光灯，但功能更强大。它也是免费的。

如果 Spotlight 只是一个带有附加功能的搜索栏，Raycast 就是一个控制面板。它将设备的控制范围缩小到单个提示。

考虑阅读我的文章；它将彻底改变你的 Mac 体验，就像它改变我的一样。

### 快捷方式如下：

我们可以直接访问所有快捷方式并按顺序查看它们，而在 Spotlight 中，除非我们知道确切的名称，否则我们无法访问任何快捷方式。

![](.evernote-content/B0E2F0D3-9C84-4FA9-A063-8EFE09C33ABD/FEFFE68E-970B-457F-82AC-1B29BA86C5DA.jpg)

Raycast 使得使用快捷方式并在需要时激活它们变得非常方便。

但在其简洁的设计背后实际上隐藏了很多实用的功能。

### 热点角

Hot Corners 允许通过将光标移动到屏幕角落来执行所选操作。如果您想查看桌面、快速记笔记或锁定设备，此功能非常有用。

![](.evernote-content/B0E2F0D3-9C84-4FA9-A063-8EFE09C33ABD/8957180F-06DF-48CF-83E0-CFB39A8805E0.gif)

要打开热点角：系统设置→桌面和Dock→热点角。

![](.evernote-content/B0E2F0D3-9C84-4FA9-A063-8EFE09C33ABD/F83F3652-B610-4F8E-925F-257B27386506.png)

接下来，点击每个角落选择一个操作，例如打开 Mission Control、查看桌面或访问通知中心。然后，通过将光标滑向所选角落来激活这些操作。

![](.evernote-content/B0E2F0D3-9C84-4FA9-A063-8EFE09C33ABD/FC8CE7E2-A99D-4E71-A994-EFDCBE2B1989.jpg)

文本替换和键盘快捷键 学习键盘快捷键很难，但最终是值得的，因为它可以加快整个设备的工作流程。

另一方面，文本替换是快捷方式。例如，将“ttyl”改为“稍后再聊”。

这些都是定制的，我们可以以任何方式进行设置。

### 设置

系统设置>键盘>键盘快捷键用于设置自定义键盘快捷键。系统设置>键盘>文本替换，用于文本快捷方式。这里我整理了一个我日常使用的快捷方式列表。

![](.evernote-content/B0E2F0D3-9C84-4FA9-A063-8EFE09C33ABD/44FB5E65-0757-417E-B0E0-B21F154F346C.png)

### 我不能没有的常规快捷方式

Command + Space / Command + Return — 激活 Spotlight / Alfred。

C ommand + Q — 退出应用程序。Command + Tab — 在上一个应用程序之间切换。

Command（按住）+ Tab，然后 Tab Tab Tab Tab — 在任何打开的应用程序之间切换。

Command + H——隐藏当前应用程序。

Command + Option + Q — 强制退出应用程序（当出现滞后时）。

Command + ,（逗号） ——在大多数应用程序中打开设置。

### 浏览器快捷方式

Command + T — 打开一个新标签页以准备输入。

Command + L — 打开当前选项卡的链接窗口以搜索新内容

Command + Shift + T — 打开最后一个关闭的标签页

Command + W — 关闭当前选项卡

Command + 1 / 2 / 3 / 4 — 在不同选项卡之间切换

Command + 箭头键（右和左）—后退 /前进

### 文本替换

文本替换主要用于电子邮件中的简单短语，例如“此致”或“感谢您的时间”。

这些可以在文本输入部分进行自定义。

![](.evernote-content/B0E2F0D3-9C84-4FA9-A063-8EFE09C33ABD/2859E8D5-AB32-4D28-AFA4-1003AD6D00D1.png)

我们只需点击 + 图标，然后选择要替换的文本即可。它非常直观，每次使用时都可以节省一些时间。

![](.evernote-content/B0E2F0D3-9C84-4FA9-A063-8EFE09C33ABD/692891D7-E69F-4AA3-A2F1-1628919A4E63.jpg)

### 结论

这些是我个人日常使用最多的功能。其中一些功能增加了一些功能；另一些功能则改变了 MacOS 的体验。

我希望您发现它们很有用，并且它们能够提高您的工作效率。
