四元数是简单的[超复数](https://baike.baidu.com/item/%E8%B6%85%E5%A4%8D%E6%95%B0/10724562)。 [复数](https://baike.baidu.com/item/%E5%A4%8D%E6%95%B0/254365)是由[实数](https://baike.baidu.com/item/%E5%AE%9E%E6%95%B0/296419)加上[虚数单位](https://baike.baidu.com/item/%E8%99%9A%E6%95%B0%E5%8D%95%E4%BD%8D/6803458) i 组成，其中i^2 = -1。 相似地，四元数都是由实数加上三个虚数单位 i、j、k 组成，而且它们有如下的关系： i^2 = j^2 = k^2 = -1， i^0 = j^0 = k^0 = 1 , 每个四元数都是 1、i、j 和 k 的线性组合，即是四元数一般可表示为a + bi+ cj + dk，其中a、b、c 、d是实数。

对于i、j、k本身的几何意义可以理解为一种旋转，其中i旋转代表X轴与Y轴相交平面中X轴正向向Y轴正向的旋转，j旋转代表Z轴与X轴相交平面中Z轴正向向X轴正向的旋转，k旋转代表Y轴与Z轴相交平面中Y轴正向向Z轴正向的旋转，-i、-j、-k分别代表i、j、k旋转的反向旋转。（见右图）

基本信息
----

编辑

四元数（Quaternions）是由爱尔兰数学家[哈密顿](https://baike.baidu.com/item/%E5%93%88%E5%AF%86%E9%A1%BF/28890)(William Rowan Hamilton,1805-1865）在1843年发明的数学概念。四元数的[乘法](https://baike.baidu.com/item/%E4%B9%98%E6%B3%95)不符合[交换律](https://baike.baidu.com/item/%E4%BA%A4%E6%8D%A2%E5%BE%8B)（commutative law）。

[![](.evernote-content/6B9D2602-867C-4EC8-B46D-711BE5EEEB54/CB2E0FD7-1818-4D7F-B757-AAC4D3FCB1B7)](https://baike.baidu.com/pic/%E5%9B%9B%E5%85%83%E6%95%B0/5795379/0/73ca5910526400bec3ce799e?fr=lemma&ct=single "威廉·卢云·哈密顿")
威廉·卢云·哈密顿

明确地说，四元数是复数的不可交换延伸。如把四元数的集合考虑成多维实数空间的话，四元数就代表着一个[四维](https://baike.baidu.com/item/%E5%9B%9B%E7%BB%B4/13864441)空间，相对于复数为[二维空间](https://baike.baidu.com/item/%E4%BA%8C%E7%BB%B4%E7%A9%BA%E9%97%B4)。

四元数是[除环](https://baike.baidu.com/item/%E9%99%A4%E7%8E%AF)(除法环）的一个例子。除了没有[乘法](https://baike.baidu.com/item/%E4%B9%98%E6%B3%95)的交换律外，[除法](https://baike.baidu.com/item/%E9%99%A4%E6%B3%95)环与域是相类的。特别地，乘法的[结合律](https://baike.baidu.com/item/%E7%BB%93%E5%90%88%E5%BE%8B)仍旧存在、非零元素仍有[逆元素](https://baike.baidu.com/item/%E9%80%86%E5%85%83%E7%B4%A0/6307126)。

四元数形成一个在实数上的四维[结合代数](https://baike.baidu.com/item/%E7%BB%93%E5%90%88%E4%BB%A3%E6%95%B0)（事实上是除法代数），并包括复数，但不与复数组成结合代数。四元数（以及实数和复数）都只是有限维的实数结合[除法](https://baike.baidu.com/item/%E9%99%A4%E6%B3%95/6280598)代数。

四元数的不可交换性往往导致一些令人意外的[结果](https://baike.baidu.com/item/%E7%BB%93%E6%9E%9C/29394)，例如四元数的 n-阶多项式能有多于 n 个不同的根。

详见参考资料《关于四元数的几何意义和物理应用》

基本性质
----

编辑

四元数就是形如 ai+bj+ck+d 的数，a、b、c、d是[实数](https://baike.baidu.com/item/%E5%AE%9E%E6%95%B0/296419)。

i^2=j^2=k^2=-1

ij=k、ji=-k、jk=i、kj=-i、ki=j、ik=-j

（a^2+b^2+c^2+d^2）的[平方根](https://baike.baidu.com/item/%E5%B9%B3%E6%96%B9%E6%A0%B9/79171)，称为四元数的[模](https://baike.baidu.com/item/%E6%A8%A1/13332719)。

例子
--

编辑

假设：

x = 3 + i

y = 5i + j - 2k

那么：

x + y = 3 + 6i + j - 2k

xy =( {3 + i} )( {5i + j - 2k} ) = 15i + 3j - 6k + 5i^2 + ij - 2ik

= 15i + 3j - 6k - 5 + k + 2j = - 5 + 15i + 5j - 5k

群旋转
---

编辑

像在四元数和[空间](https://baike.baidu.com/item/%E7%A9%BA%E9%97%B4)转动条目中详细[解释](https://baike.baidu.com/item/%E8%A7%A3%E9%87%8A/33037)的那样，非零四元数的乘法群在R3的取实部为零的[拷贝](https://baike.baidu.com/item/%E6%8B%B7%E8%B4%9D)上以共轭作用可以实现[转动](https://baike.baidu.com/item/%E8%BD%AC%E5%8A%A8/18600)。单位四元数（绝对值为1的四元数）的[共轭](https://baike.baidu.com/item/%E5%85%B1%E8%BD%AD/31802)作用，若实部为cos(t)，是一个角度为2t的转动，转轴为[虚部](https://baike.baidu.com/item/%E8%99%9A%E9%83%A8/5231815)的方向。四元数的优点是：

非奇异表达（和例如[欧拉角](https://baike.baidu.com/item/%E6%AC%A7%E6%8B%89%E8%A7%92)之类的表示相比）

比[矩阵](https://baike.baidu.com/item/%E7%9F%A9%E9%98%B5)更紧凑（更快速）

单位四元数的对可以表示[四维](https://baike.baidu.com/item/%E5%9B%9B%E7%BB%B4/13864441)空间中的一个转动。

所有单位四元数的集合组成一个[三维](https://baike.baidu.com/item/%E4%B8%89%E7%BB%B4/9517577)球S3和在乘法下的一个群（一个李群）。S3是行列式为1的实正交3×3正交矩阵的群SO（3,R）的双面覆盖，因为每两个单位四元数通过上述关系对应于一个转动。群S3和SU（2）同构，SU（2）是行列式为1的复酉2×2[矩阵](https://baike.baidu.com/item/%E7%9F%A9%E9%98%B5/18069)的群。令A为形为a + bi + cj + dk的四元数的集合，其中a,b,c和d或者都是整数或者都是分子为奇数分母为2的[有理数](https://baike.baidu.com/item/%E6%9C%89%E7%90%86%E6%95%B0/105546)。集合A是一个环，并且是一个格。该环中存在24个四元数，而它们是[施莱夫利符号](https://baike.baidu.com/item/%E6%96%BD%E8%8E%B1%E5%A4%AB%E5%88%A9%E7%AC%A6%E5%8F%B7)为{3,4,3}的正二十四胞体的顶点。

矩阵表示
----

编辑

有两种方法能以[矩阵](https://baike.baidu.com/item/%E7%9F%A9%E9%98%B5/18069)表示四元数，并以[矩阵](https://baike.baidu.com/item/%E7%9F%A9%E9%98%B5/18069)之加法、乘法应用于四元数之加法、乘法。

第一种是以二阶复数矩阵表示。若 h = a + bi + cj + dk 则它的[复数](https://baike.baidu.com/item/%E5%A4%8D%E6%95%B0/254365)形式为：

![](.evernote-content/6B9D2602-867C-4EC8-B46D-711BE5EEEB54/003D65D5-422D-467A-9FD7-5D2976E2D44E.png)

这种表示法有如下优点：

所有复数 (c = d = 0) 就相应于一个实矩阵。

四元数的绝对值的平方就等于矩阵的[行列式](https://baike.baidu.com/item/%E8%A1%8C%E5%88%97%E5%BC%8F)。

四元数的共轭值就等于矩阵的[共轭转置](https://baike.baidu.com/item/%E5%85%B1%E8%BD%AD%E8%BD%AC%E7%BD%AE)。

对于单位四元数 (|h| = 1）而言，这种表示方式给了四维球体和SU（2）之间的一个同型，而后者对于量子力学中的[自旋](https://baike.baidu.com/item/%E8%87%AA%E6%97%8B)的研究十分重要。（请另见[泡利矩阵](https://baike.baidu.com/item/%E6%B3%A1%E5%88%A9%E7%9F%A9%E9%98%B5)）

第二种则是以四阶实数矩阵表示：

![](.evernote-content/6B9D2602-867C-4EC8-B46D-711BE5EEEB54/B01C6A67-2D95-40CF-9C4C-80657D55987F.png)

其中四元数的共轭等于矩阵的[转置](https://baike.baidu.com/item/%E8%BD%AC%E7%BD%AE/11034603)。

历史
--

编辑

四元数是由[哈密顿](https://baike.baidu.com/item/%E5%93%88%E5%AF%86%E9%A1%BF)在1843年爱尔兰发现的。当时他正研究扩展复数到更高的维次（复数可视为平面上的点）。他不能做到[三维空间](https://baike.baidu.com/item/%E4%B8%89%E7%BB%B4%E7%A9%BA%E9%97%B4)的例子，但四维则造出四元数。根据哈密顿记述，他于10月16日跟他的妻子在[都柏林](https://baike.baidu.com/item/%E9%83%BD%E6%9F%8F%E6%9E%97)的皇家运河（Royal Canal）上散步时突然想到

![](.evernote-content/6B9D2602-867C-4EC8-B46D-711BE5EEEB54/75C3D9DB-1165-40B6-9814-D606A858F7A4.png)

的方程解。之后哈密顿立刻将此方程刻在附近布鲁穆桥（Brougham Bridge，现称为金雀花桥 Broom Bridge）。这条方程放弃了[交换律](https://baike.baidu.com/item/%E4%BA%A4%E6%8D%A2%E5%BE%8B/2173703)，是当时一个极端的想法（那时还未发展出[向量](https://baike.baidu.com/item/%E5%90%91%E9%87%8F/1396519)和矩阵）。

不只如此，哈密顿还创造了[向量](https://baike.baidu.com/item/%E5%90%91%E9%87%8F/1396519)的内外积。他亦把四元数描绘成一个有序的四重实数：一个纯量（a）和向量（bi + cj + dk）的组合。若两个[纯量](https://baike.baidu.com/item/%E7%BA%AF%E9%87%8F/3567411)部为零的四元数相乘，所得的纯量部便是原来的两个向量部的纯量积的负值，而向量部则为[向量积](https://baike.baidu.com/item/%E5%90%91%E9%87%8F%E7%A7%AF)的值，但它们的重要性仍有待发掘。

哈密顿之后继续推广四元数，并出了几本书。最后一本《四元数的原理》（Elements of Quaternions）于他死后不久出版，长达八百多页。

用途争辩
----

编辑

四元数的用途仍在争辩之中。一些哈密顿的支持者非常反对[奥利弗·亥维赛](https://baike.baidu.com/item/%E5%A5%A5%E5%88%A9%E5%BC%97%C2%B7%E4%BA%A5%E7%BB%B4%E8%B5%9B)（Oliver Heaviside）的向量[代数学](https://baike.baidu.com/item/%E4%BB%A3%E6%95%B0%E5%AD%A6)和[约西亚·威拉德·吉布斯](https://baike.baidu.com/item/%E7%BA%A6%E8%A5%BF%E4%BA%9A%C2%B7%E5%A8%81%E6%8B%89%E5%BE%B7%C2%B7%E5%90%89%E5%B8%83%E6%96%AF)（Josiah Willard Gibbs）的[向量微积分](https://baike.baidu.com/item/%E5%90%91%E9%87%8F%E5%BE%AE%E7%A7%AF%E5%88%86)的发展，以维持四元数的超然地位。对于三维空间这可以讨论，但对于更高维四元数就失效了（但可用延伸如[八元数](https://baike.baidu.com/item/%E5%85%AB%E5%85%83%E6%95%B0)和柯利弗德代数学）。而事实上，在二十世纪中叶的科学和工程界中，[向量](https://baike.baidu.com/item/%E5%90%91%E9%87%8F)几乎已完全取代四元数的位置[1]
。

[詹姆斯·克拉克·麦克斯韦](https://baike.baidu.com/item/%E8%A9%B9%E5%A7%86%E6%96%AF%C2%B7%E5%85%8B%E6%8B%89%E5%85%8B%C2%B7%E9%BA%A6%E5%85%8B%E6%96%AF%E9%9F%A6)（James Clerk Maxwell）曾经在他的《电磁场动力理论》（A Dynamical Theory of Electromagnetic Field）直接以20条有20个变量的微分方程组来解释[电力](https://baike.baidu.com/item/%E7%94%B5%E5%8A%9B/1250605)、磁力和电磁场之间的关系。某些早期的[麦克斯韦方程组](https://baike.baidu.com/item/%E9%BA%A6%E5%85%8B%E6%96%AF%E9%9F%A6%E6%96%B9%E7%A8%8B%E7%BB%84)使用了四元数来表述，但与后来亥维赛使用四条以[向量](https://baike.baidu.com/item/%E5%90%91%E9%87%8F/1396519)为基础的麦克斯韦方程组表述相比较，使用四元数的表述并没有流行起来[2]
。
