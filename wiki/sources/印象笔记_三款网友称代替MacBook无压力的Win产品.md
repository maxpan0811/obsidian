# 三款网友称代替MacBook无压力的Win产品

![](.evernote-content/427F1973-6618-4302-93BF-A57905AD67AF/C2BB52CE-720F-435E-8081-C1479A36BD67.jpg)

  
　　前两天，苹果发布了全新配备 Retina 显示屏幕的 MacBook，该产品是目前为止最轻、最薄的 MacBook 家族一员。虽然苹果公布了很多革命性的改变，突出其便携性，尤其是完全无风扇的设计，不过当谈到新 MacBook 的价格，很多用户表示不太能够接受。  
  
　　苹果的 MacBook 的起始价格为 9288 元人民币，考虑到目前 Windows 生态系统中很多 Core M 笔记本电脑发布有一段时间了，且价格趋于稳定甚至有不少打折活动，从中或许有一些可以替换 MacBook。下面推荐三款，相信不会太令人失望。  
  
　　**戴尔XPS 13**  
  
　　Dell XPS 13 是 MacBook 推出之后，很多用户认为最能匹敌的产品。该笔记本电脑搭载的并非 Core M 处理器，而是性能更优越的第五代 14nm 英特尔专为笔记本电脑打造的低电压移动 CPU，屏幕大小为 13.3 英寸，分辨率提供两种版本。  
  
　　与 MacBook 价格相近的 8999 元版，分辨率为 1920×1080 像素，非触屏，不过搭载了i7-5500U 处理器（最高 3.0GHz）。另一版 10999 元的版本分辨率提升至 3200×1800 像素，搭载 i5-5200U 处理器（最高 2.7GHz），两款产品都配备了 256GB 的固态硬盘和 8GB 内存。  
  

![](.evernote-content/427F1973-6618-4302-93BF-A57905AD67AF/F9FED826-85C7-4E67-B65B-CF3E3AEAF836.jpg)

  
　　看上 XPS 13 的用户相信对其华丽的“边到边”的微边框屏幕印象十分深刻，屏幕边框仅 5.2cm 厚度，覆盖有康宁的 Gorilla 强化玻璃。同时，该笔记本电脑还提供了背光键盘、阳极化铝外壳和碳纤维材质的掌托等等。  
  
　　毫无疑问，根据英特尔的定位，Broadwell-U 的性能自然比 Broadwell-Y Core M 在性能上更胜一筹，即便集成的 HD 5500 GPU 性能也更为强劲 。与此同时，戴尔声称其续航可以达到 15 个小时。不过其机身尺寸数字略微增加了些，厚度为 9-15mm，重量提升到 1.18kg。  
  
　　对于预算不足的用户，戴尔也提供了仅 6999 元的 Core i5 5010U款式，不过配置上固态硬盘容量降低至 128GB，内存简配到 4GB。  
　  
　　**联想 Yoga 3 Pro**  
  
　　Yoga 3 Pro 是第一批搭载 Core M 处理器的 Windows 8.1 平板电脑，主打的特色在于极致的铝合金转轴工艺设计。联想声称该转轴由 813 个零件组成，拥有高达 25000 次的翻转寿命，一天翻转 360 度 10 次也能顶 6 年使用。得益于转轴 Yoga 3 Pro 成为了最多模式笔记本电脑，可变形的模式包括笔记本电脑、平板电脑、敞篷和站立。  
  

![](.evernote-content/427F1973-6618-4302-93BF-A57905AD67AF/018190C3-D574-4D26-A595-AE502BD82046.jpg)

  
　　配置上，该笔记本电脑仅 12.8mm 厚度，1.19kg 重，采用了一块 13.3 英寸分辨率高达 3200×1800 像素的屏幕，内部处理器为 Core M-5Y70，最高可选 8GB 的内存和 512GB 的 SSD 固态硬盘。与 MacBook 同容量的 256GB 和 8GB 内存版售价为 7999 元人民币。  
  
　　**华硕 Zenbook U305FA**  
  
　　ASUS Zenbook U305FA 这一款是三款中最便宜的了，因为它虽然采用了 13.3 英寸的雾面防眩光广角 IPS 屏幕，但分辨率只有 1920×1080 像素。不过笔记本电脑的机身设计上用料还不错，采用了铝合金一体成型的机身，厚度只有 12.7mm，重量也仅 1.13kg。  
  
　　在其机身内部，搭载了来自 Broadwell 架构的 Core M-5Y10 处理器，Turbo 动态加速后主频可达到 2GHz，标配 256GB 固态硬盘，支持双频 Wi-Fi 和蓝牙 4.0，价格 6690 元起售。
