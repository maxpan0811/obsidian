# iPhone X 用刷脸的 Face ID 取代了指纹，睡觉时真的会被解锁吗？

---

今天发布的 iPhone X 取消了指纹解锁 Touch ID，而带来了全新的 Face ID。没错，在 iPhone X 上，只需要刷脸，你就可以解锁手机、购买应用、支付 Apple Pay 了。

苹果敢将 Face ID 作为 iPhone X 的主打特性，自然有了十足的底气，能保证 Face ID 在速度体验与安全性方面，做到比 Touch ID 持平，甚至更好的水平。所以现在，请大家跟随少数派一起来进一步了解 Face ID 到底怎么样，以及大家关心的一些诸如「睡觉会不会被人解锁」的问题，到底会不会发生。

Face ID 是什么？
------------

**Face ID 是基于 3D 的面部识别。**不同于三星手机或支付宝刷脸登录，使用的其实是一张 2D 的照片，而 Face ID 会使用你的面部深度信息。据苹果在发布会上的透露，iPhone X 一共会投射 3 万个红外点光源，作为特征点，这大大丰富了识别的特征元素。

Face ID 在硬件上，依赖于 iPhone X 前置刘海区的「TrueDepth Camera」，它一共由下图中的八个传感器或元件构成。其中，与 Face ID 识别过程直接相关的主要有红外镜头、泛光感应元件、距离传感器、点阵投影器。

其中，点阵投影器向外投射出 3 万个肉眼不可见的红外点光源，由红外镜头拍摄一张红外照片，根据照片上的位移变形情况，分析出被探测物的景深信息，并由此合成相应的 3D 模型。同时，距离传感器会探测在一定范围内，是否存在被探测物，以决定是否开启点阵投影器与红外镜头的工作。而在夜晚等暗光环境下，泛光感应元件会向外投射不可见的红外光源，以帮助 Face ID 更好地工作。

Face ID 安全吗？
------------

Face ID 可以用于购买应用以及支付 Apple Pay，证明苹果有足够的信心，**Face ID 有足够的安全性，而不像别家的面部识别，仅靠一张照片就能欺骗。**

苹果之所以有这样的底气，在开发过程中，为了避免不良意图的人，使用照片、头像模具来解锁，苹果一共使用了 1,000,000,000 张照片来训练 Face ID 所依赖的神经网络，让 Face ID 不被照片或模具所欺骗。据苹果在发布会的介绍，原先的 Touch ID，任意 50,000 人中，可能会有一人能用他的指纹解锁你的手机。而使用 Face ID，在 1,000,000 人中，才会有一个人可能会被误判解锁成功。不过如果是双胞胎，目前的确会大大减弱 Face ID 的安全性。

同时，大家很关心的问题是，**如果有人拿着我的手机对着我，手机就会解锁吗？或者在睡觉的时候，把手机拿到我面前，能成功解锁吗？**苹果显然也考虑到了这一点，所以，Face ID 的面部解锁，是基于用户正在注视着手机的。也就是说，Face ID 足够聪明到，能分辨出你正在盯着手机，才会触发解锁（但目前还不能确定具体需要注视多长时间才能解锁），单纯地把手机放在面前，是没有用的。

在保证安全的前提下，Face ID 利用人工智能，还可以持续地学习一个用户的长相。在完成首次对 Face ID 的录入后，随着时间的推移，**即使你戴眼镜了、开始长了胡子、喜欢戴帽子了等，Face ID 仍然能继续认出你，并不断根据新的数据来校准识别模型，以保证识别率。**

### Face ID 怎么用？

Face ID 在 iPhone X 的设置中，替代了原先的 Touch ID，但像原来录指纹一样，你可以录入多张面孔。在录入的过程中，设置会引导你随着周围一圈光环，360 轻轻转动头部，确保 Face ID 能捕获到你的整张面部信息。

![](.evernote-content/24CC1070-5E05-4534-BDC7-88EFE413AEA4/10E6E7CE-21A8-4C2B-9B9F-598B16F82177.jpg)

而在解锁手机时，则非常简单。你拿起 iPhone X 时，屏幕会自动亮起，这时候只要你看着屏幕，Face ID 会自动识别到你的脸，并解锁手机。整过过程，基本上就是拿起手机，屏幕亮起，出现解锁动画。

![](.evernote-content/24CC1070-5E05-4534-BDC7-88EFE413AEA4/7269C378-C5B2-45AD-84A5-65FAC66D657D.png)

使用 Apple Pay 支付的流程也很简单，首先双击侧边键唤起 Apple Pay 后，将手机对准自己的脸，即可完成授权。然后，再将 iPhone 靠近相应的付款 NFC 设备，整个流程就完成了。

![](.evernote-content/24CC1070-5E05-4534-BDC7-88EFE413AEA4/004789F9-0228-4B2E-A9F4-9BA8ADEBBF04.png)

除此之外，得益于 Face ID，你的手机不仅能认出这张脸是不是你，还能知道，你有没有在盯着屏幕，因此，可以实现一系列实用的小特性。例如，**当 Face ID 发现是你时，才在锁屏上显示通知内容详情；当发现你一直在盯着屏幕时，保持屏幕的常亮；发现你在使用手机时，降低或静音通知 / 闹钟的声音。**这些功能，在 iOS 11 的系统设置中均可以找到对应的选项，少数派会在随后的 iOS 11 技巧文章一一提到这些。
