这本书第1版出版于2012年，当时基于 Python 的开源数据分析库（例如 pandas）仍然是一个发展迅速的新事物，本书也成为该领域排名No.1的经典畅销书，前两版中文版累计销售近30万册。

而如今这本书已经发布到第三版了，它针对Python3.10 和 pandas1.4 进行了更新，并通过实操讲解和实际案例向读者展示了如何高效地解决一系列数据分析问题。读者将在阅读过程中学习新版本的 pandas、NumPy、IPython 和 Jupyter。

在这本书中你可以学到以下内容：

* 使用 Jupyter 笔记本和 IPython shell 进行探索性计算。
* 了解 NumPy 中的基本和高级功能。
* 开始使用 pandas 库中的数据分析工具。
* 使用灵活的工具加载、清理、转换、合并和重塑数据。
* 使用 matplotlib 创建信息丰富的可视化。
* 应用 pandas groupby 工具对数据集进行切片、切块和汇总。
* 分析和操作规则和不规则时间序列数据。
* 了解如何通过全面、详细的示例解决实际的数据分析问题。
