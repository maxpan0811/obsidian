# 美国华裔究竟有多少反对黑人抗议运动？

![](attachments/239596770cfd9c8f.jpg)

弗洛伊德之死在美国乃至世界引起震动，也在华人中文社交媒体引起轩然大波。

根据维基百科最新统计，全美50个州2000多个城镇爆发抗议示威运动，甚至全球有60多个国家示威游行，支持为弗洛伊德讨回公道，抗议种族主义和警察暴行。

![](attachments/68872d4f44d81c00.jpg)

北美超过百人参加的抗议地图，弗洛伊德死亡所在州明州标志为红色（图源：维基百科）

![](attachments/01a08fa57e6259d0.jpg)

全球超过百人参加的抗议地图（图源：维基百科）

**华人社交媒体的怪现象**

**来自大陆的美国华一代都是逆美国主流文明而上的反弗洛伊德抗议者吗？**

但是华人社交媒体出现了一个怪现象，充满了以“客观”“学术”之名对黑人个人与群体进行污蔑与贬低的种族主义语言，甚至连美国前总统奥巴马也未能幸免。

在弗洛伊德之死事件上，关注焦点不是警察滥用暴力，不是体系性针对黑人的执法司法等权力不公，不是抗议者的正当诉求，**而是对准弗洛伊德曾经的犯罪过往（忽略出狱后的改变）和抗议中出现的打砸抢烧等骚乱**。

在骚乱事件中，他们不区分和平抗议者和骚乱者，也不看骚乱的根源和骚乱者的多方来源，一概归咎于“黑人”，支持警察甚至军队“平暴”，对于抢劫者“格杀勿论”，至于警察以暴力驱散和殴打和平抗议者他们就视而不见了。即便是一向表现为自由派的知识分子，所惊讶瞩目的也是雕像斩首和沉默权的有无之类问题，担忧弗洛伊德抗议或者BLM运动过了头。

华人社交媒体对弗洛伊德之死表现出的种种怪现象，恐怕如果川普能读懂中文的话都会骇然。有些话他都不敢说甚至说不出口的，更不要说做了，比如“黑人是动物”和“格杀勿论”之类。

但此类怪现象在中文社交媒体甚嚣尘上，发帖者和发言者众，而辟谣者反对者少，微信群为重灾区。某日夜半，我在某个以“自由宽容仁爱”命名的小群中提出质疑并发表相反看法，一觉醒来，被素常还算有些私交的群主踢出群，无论在群中还是私下，都没得到任何解释。一个以哈佛及周边高校华人学生学者（包括访问学者）为主体的所谓学术群，也充斥了此类文章与观点，反对者极少，管理者也无动于衷。至于当地一些华人群，活跃分子们也在不遗余力地转发抹黑黑人个人与群体的文章、反弗洛伊德抗议并热心张罗为支持警察签名和捐款。只有极少数的小群与美国主流媒体一致，而这些小群，大多都是在川普上台之后以“反川”之名建立的抱团取暖群。

这给人一种印象，好像美国华人都不同情黑人的遭遇，都反对弗洛伊德抗议，认为他们无论个人还是群体天然存在问题才导致他们现在的处境。一句话：罪有应得。

与朋友言及此类怪现象，朋友说，中国人天然右翼，美国大陆华一代狼奶未尽，崇拜强者（川普/白人）蔑视弱者（黑人群体）是自然之理，二者一体，手心手背而已。

虽然他给出一个好像比较令人信服的理由，但是美国华人，或者更准确点说，美国大陆华一代真的都是天然右翼，都崇拜强者（川普/白人）蔑视弱者（黑人群体）吗？来自大陆的美国华一代都是逆美国主流文明而上的反弗洛伊德抗议者吗？

华人社交媒体的反弗洛伊德抗议浪潮让我想起2016年大选前狂热的川粉潮。当时也是整个中文社交媒体，确切说，整个简体中文社交媒体，尤其是微信群，都掀起了狂热的挺川浪潮，包括一些平时谨言慎行的各种美国华人基督教团契群。至今难以忘记当时有人把川普称为上帝派来拯救美国的伟大救星，还有人把川普比作大卫王。

可是最后的投票结果呢？根据Edison Research为BBC等媒体做的研究报告，亚裔将选票投给川普的只有29%，65%的选票投给了希拉里，6%没有作答。

![](attachments/d0757e9e2f2de18d.jpg)

各族裔大选投票给川普和希拉里的比例。图片来自BBC，版权归原作者所有

**与预期不同的民意调查**

**亚裔没有做“哑”裔，更没有背叛曾为亚裔平权作出历史贡献的黑人**

![](attachments/cefc60b6f45ac507.jpg)

标语牌上写着亚裔支持“黑人命也是命”

鉴于2016年川粉潮的教训，我搜索了一下亚裔支持弗洛伊德抗议的比率，发现一向权威的皮尤调查竟然真的做了此类民意调查。**全美成年人支持BLM运动的平均值67%，亚裔75%**，拉丁裔77%，黑人86%，白人60%。亚裔75%支持BLM，高于平均值8个百分点，仅低于拉丁裔2个百分点。图标如下：

![](attachments/5a6fdbf8b70c5ef5.jpg)

67%的美国人表示支持BLM运动，亚裔的支持率为75%。图片来自皮尤研究，版权归原作者所有

这个调查结果与2016年大选投票的结果基本相符，甚至略高。

时隔将近4年，美国华人对川普的印象是否有所不同？或许，从美国亚裔对川普在族裔关系中所起的作用的看法当中可见一斑。皮尤调查显示，将近一半（48%）的美国人认为川普令美国族裔关系更紧张，其中持此观点的黑人比率最高68%，其次为亚裔62%，拉丁裔为55%，白人为42%。持此观点的亚裔甚至比拉丁裔高7个百分点，仅次于黑人。这个结果也与2016年大选投票的结果基本相符。

![](attachments/6983fb16e6390676.jpg)

几乎一半美国人认为川普让族裔关系更紧张，其中持此观点的亚裔62%。图片来自皮尤研究，版权归原作者所有

在此次反种族主义反警察暴行的弗洛伊德抗议活动中，亚裔的表现如何呢？华人中文社交媒体充斥的反黑人反抗议是否能真实反映美国华人参与抗议的真实状况？

皮尤调查发现，10%的黑人、9%的拉丁裔、10%的亚裔和5%的白人承认上个月参加以种族或种族平等为主题的集会或抗议活动。**美国亚裔参加者的比率和黑人参加者的比率最高**。在上个月，21%的亚裔、13%的黑人、11%的拉丁裔和7%的白人通过捐款支持抗议活动。**亚裔虽然人数少，但捐款者的比例远远高于黑人和拉丁裔**，在少数族裔中，如果犹太人也单列的话，恐怕只有犹太人可比。社交媒体上分享有关种族平等为主题的内容方面，亚裔的比例也仅次于黑人，与亲友交谈此类话题的比例也比拉丁裔高。

![](attachments/e07f1700b898f7fe.jpg)

由此可知，美国亚裔在上个月反种族主义的游行示威中非但非常活跃，而且表现相当突出，亲身参与抗议现场的比例与黑人持平、社交媒体发声仅次于黑人，捐款支持大大高于黑人，当然这也与亚裔自身的经济优势有关。

在此次抗议中，同为少数族裔的亚裔，并没有像往常一样习惯于做“哑”裔，更没有背叛曾为亚裔平权作出历史贡献的黑人和其他少数族裔。在这次有关种族与社会公义的重大历史关头，亚裔正在美国政治舞台上积极发声，并利用自己的优势发挥自己的力量。

![](attachments/f487fee28e3b3e3a.jpg)

1/10黑人、拉丁裔和亚裔说参加过以种族或种族平等为主体的集会。图片来自皮尤研究，版权归原作者所有

可惜皮尤调查没有对支持抗议的亚裔人进行细分。有朋友指出，参加皮尤调查的都是会说英语的，可能以ABC（出生在美国的华人）居多，没多少华一代，更没多少大陆华一代，即便有，也属于那25%不支持BLM的人。

这个猜测有可能，但是根据皮尤调查，亚裔74%的人口出生在非美国本土，其中包括73%的成年人。也就是说，美国亚裔一代成年人占亚裔总人口73%。而亚裔55%的人口英文熟练。由此可见，皮尤调查专门找出生在美国本土的亚裔或者华裔（ABC）概率很低。

![](attachments/ef3b679dfbc8dc70.jpg)

但是美国大陆华一代（特指改开后三四十年来美的华人，包括公民与非公民）究竟有多少人呢？他们对美国的政治走向影响究竟可能多大？

为此，我查了一下美国亚裔的构成及比例。

根据莱茵数据在线一篇文章（我不确定它的原始来源，根据上下文或许来源于美国人口普查局），2015年数据显示，本土出生的华人（ABC）占37%，大约63% 的华人出生在美国本土以外。出生于美国境外的华人（移民）中， 其中 59.9%来自于中国大陆，15.9%来自中国台湾， 9.4% 来自中国香港，15.3%来自其他国家和地区。

皮尤调查根据美国人口普查局的数据显示，美国华人是美国亚裔中最大的族裔群体，截至2015年，美国华裔人口总数为494.8万人，占美国人口总数的1.5%，占美国亚裔人口的24.3%。

![](attachments/4fa064343714bfd2.jpg)

美国亚裔内部各族裔人口分布。图片来源于皮尤调查，版权归原作者

如果莱茵数据在线的数据为实，那么2015年美国大陆华一代的人口应该是：494.8 x 63% x 59.9%=186.72（万人）。

美国国土安全部的统计数据，从1980年至2012年，中国人（包括大陆和香港）获得美国绿卡的人数总计为163.0938万人，截止到2010年，共有40.058万名移民美国的中国人加入美国国籍，入籍人口占绿卡人口的24.56%，不足1/4。

我们计算的2015年美国大陆华一代人口与美国国土安全部的中国人绿卡总数基本相符。如果入籍人口比率在这5年中基本不变，那么加入2015年持美国国籍的大陆华一代应该大约：186.72 x 24.56% = 45.858 （万人）。

2015年，亚裔总人口为20416808人，公民占57%，即1163.758万人。

公民当中，对美国政局起决定作用的是符合投票资格的选民。

皮尤2020年5月份最新研究报告发现，美国亚裔有1100多万合格选民，其中华裔选民占257.6万人，是亚裔合格选民的23.4%。

**假如拥有美国国籍的大陆华一代都是合格选民（其中的未成年人本应不算），那么他们仅占华裔合格选民的17.8%左右，占亚裔合格选民的4.2%左右**。

![](attachments/2d553a3d7ff8a349.jpg)

亚裔内部各族裔合格选民数量。图片来自皮尤调查，版权归原作者所有

**挺川群体参照**

**美国华人有沉默的大多数，高声叫喊的大多数人也许并没有投票权**

因为目前没有发现细分亚裔对弗洛伊德抗议的态度，又因为反弗洛伊德抗议人群与挺川群体的高度相似性，我以美国华人对川普工作的态度以及美国华人对民主党共和党的认同作为一个参照系。

根据APIAVOTE和AAPI DATA做的调查数据，美国亚裔注册选民中，华裔选民对川普工作满意的只有24%，不满意的高达70%，6%的表示不知道。在构成亚裔85%人口的6大亚裔少数族裔中，华裔选民不满意川普工作的数据仅次于日裔选民，远远高于越南和菲律宾裔选民以及美国亚裔选民的平均值。甚至比朝鲜裔和印度裔选民的都高4个百分点。

因此，25%不支持BLM运动的亚裔中，可以推断不支持BLM的华裔比例偏低，至少低于于平均值。亚裔的其他族裔中，不支持BLM的不见得比华裔少，但平均占绝对少数。

![](attachments/bb0d85f531e9ddd6.jpg)

在弗洛伊德抗议之争中，民主党普遍比共和党对民众抗议更有同情的理解和更主张进行体系性变革，以改变体制性的种族主义和警察滥用暴力现象。在亚裔当中，华人选民在民主党、共和党以及独立选民中的认同如何呢？

APIAVOTE和AAPI DATA对此也做了调查，结果是华人选民选择共和党的比例是12%，也仅次于日裔选民，选择民主党的比例34%，更多的选择了独立选民44%，选择独立选民的比例在亚裔族裔中比例最高。

![](attachments/a56a25740a9f2277.jpg)

所以，美国华人川粉多？大选之后，我们已经知道这是一个伪概念。**之所以有这个错觉，也或许是大家走错了地方（微信），也或许是美国华人有一个沉默的大多数，也或许高声叫喊的大多数人其实并没有投票权。**至少在亚裔6大族裔中，华川粉比例几乎是最少的，仅次于日本。

也或许像某位朋友指出的，美国华人中的大陆华一代川粉多？难不成这24%认可川普工作的和12%认同共和党的都是大陆华一代？首先，100%都是大陆华一代不可能，这违背我们的生活常识，因为我们在生活中遇到很多大陆华一代认同民主党，反川普。我们也知道一些来自其他地区的华一代也有挺川的。至于跟亚裔当中的其他族裔相比，华裔挺川的比例并不是最高的。其次，即便这12%的人都是大陆华一代，那么至少还有5.8%的大陆华一代不认同共和党。这是一个有趣的发现。

现在回到弗洛伊德抗议：美国华人真的都不同情黑人遭遇、都反弗洛伊德抗议吗？我想大家应该很容易就得出答案：肯定是No。

美国华人选民中，不同情黑人遭遇反弗洛伊德抗议的人是多数还是少数？我想这个答案应该也不难了：肯定占少数。

那么，美国大陆华一代选民都是天然右，都不同情黑人遭遇、都反弗洛伊德抗议吗？我想这个问题也不是很难得出答案：No。

美国大陆华一代选民大多是天然右，大多不同情黑人遭遇、反弗洛伊德抗议吗？因为没有更近一步细分的统计数据，这个问题比较难回答，但是根据目前对川普工作的满意度比例和对共和党的认同来看，用“大多”或许不合适。虽然不能用“大多”，恐怕有相当高的比例是可能的。

最后，如果美国大陆华一代有比较高的右倾倾向，会影响美国政局吗？影响力多大？答案已经包含在我们前面的计算当中了。即便全部大陆华一代选民都是右翼，也只不过占华裔合格选民的17.8%（关于民主党共和党认同的调查数据显示不超过12%，何况17.8%中还包括没有选举资格的未成年人），占亚裔合格选民的4.2%，而整个亚裔合格选民才占全美合格选民的5%。

所以，在华人社交媒体被各种反黑人反弗洛伊德抗议信息与言论吓到的朋友们，没必要愤怒与难过。你们并不孤单，不要被华人社交媒体的表象所迷惑，美国华人圈其实还有一个更广阔的、更坚信公平与博爱的世界。
