选择新 Mac 时，最重要的决定之一就是配置多少内存。

我们可以根据自己实际需求选择。

随着 2020 年 Apple 芯片及其统一内存架构的出现，对 RAM 的传统理解发生了变化，导致用户在选择内存选项时需要考虑新的因素。

传统 PC RAM 中的 CPU 和 GPU 等独立组件都有自己专用的内存池，而 Apple 的统一内存架构允许所有处理单元访问单个共享的高带宽、低延迟内存池。

因此，在苹果芯片系统中，数据不需要在多个存储区域之间复制，从而显着提高了速度和功效。通过整合内存，采用该技术的 Apple 设备理论上可以在多任务处理、创意工作流程和要求苛刻的应用程序中提供卓越的性能，同时最大限度地减少通常与不同内存类型之间的数据传输相关的延迟。

随着 macOS 的优化，这意味着对于大多数 Mac 用户来说，更少的内存可以比以前走得更远。

现在，所有 Mac 型号的内存均从 8GB 起增至 16GB，并且可以选择通过按订单生产的型号升级到更高配置。然而，选择这样做可能会使新 Mac 的基本价格增加相当大的成本。

为了帮助针对工作流程和预算做出最佳选择，以下详细说明了谁应该选择整个 Mac 系列中当前可用的每个内存层，以及为什么它可能最适合您：

16GB：一般用户、学生和具有标准工作负载的专业人士应选择 16GB。此配置非常适合主要浏览网页、流式传输内容、使用生产力应用程序和执行多任务处理的用户。得益于 Apple 芯片的性能和 macOS 的效率，16GB 足以满足日常任务的流畅体验。其实日常开发也可行。

24GB：需要更多空间来进行多任务处理和中等创造性工作的用户应考虑 24GB。此层非常适合那些处理稍微繁重的应用程序（例如视频编辑、游戏、图形设计或编码）但又不会将这些任务推向极端的用户。它为同时运行多个密集型应用程序并执行 16GB 无法轻松处理的多任务处理的用户提供了额外的缓冲区。

32GB 或 36GB：从事密集型创意工作流程（例如 4K 视频编辑、多轨音乐制作或大型编码项目）的专业人士应选择 32GB 或 36GB。这些配置适合经常使用占用大量内存的应用程序（例如 Adobe Creative Cloud 应用程序、Xcode 或虚拟机）的用户。即使在大量工作负载下，它也能提供足够的能力来实现一致的性能，而无需频繁依赖磁盘交换。

64GB：管理高分辨率媒体和大量多任务处理的高级用户和创意专业人士应选择 64GB。它非常适合从事大型 3D 建模项目、编辑 8K 视频或处理复杂模拟和数据分析的人员。此内存量应支持运行多个虚拟机或高级软件开发环境而不会出现任何速度减慢。

96GB：处理涉及大型数据集或媒体的密集项目的高端创意专业人士和开发人员应考虑 96GB。该层为那些需要超过 64GB 的用户提供了显着的开销，确保在编辑多个高分辨率视频流、详细的 3D 渲染或处理需要保存在活动内存中的大量数据集时的无缝性能。

128GB：电影剪辑、数据科学或工程等专业领域的专业人士应该考虑 128GB。此内存配置专为涉及超大型项目并需要将大量数据保存在活动内存中的工作流程而设计。它非常适合那些在电影、基于模拟的软件或高级科学计算中从事高分辨率视觉效果工作的人员，可在要求苛刻的任务中提供强大的多任务处理和最佳性能。

192GB：处理关键任务、内存密集型应用程序的企业级用户、研究人员和行业专业人士应选择 192GB。该选项适用于大规模数据处理、机器学习或人工智能开发，确保最大效率并减少 RAM 和存储之间的数据交换需求。对于从事需要强大的单机测试的复杂服务器端项目的开发人员来说，它也是理想的选择。

凭借 Apple Silicon 的统一内存架构和 macOS 的高效内存管理，基础 16GB 配置现在可以满足大多数用户的需求，为日常计算任务和适度的创意工作提供成本和性能的良好平衡。
