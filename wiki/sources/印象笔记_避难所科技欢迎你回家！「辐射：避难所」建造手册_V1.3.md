# 避难所科技欢迎你回家！「辐射：避难所」建造手册 V1.3

---

![](.evernote-content/50A9C42E-A14A-4F19-9B2B-D55136538971/67B97097-4C59-40B0-8958-3533FF106D51.jpg)

> 公元 207X 年，地球核爆危机四起，作为地球第一科技集团，「避难所科技」设置在全球各地的地下避难所成为人们的救命稻草。作为避难所科技的一位高级工程师，你的上级分配给你一项艰巨任务：建设一个全新的地下避难所。你要保证这一座避难所能够自给自足、防火灭火、防范变异害虫和掠夺者的侵袭等，这个地区的智慧生命能否延续下去，全靠你的一己之力了。
>
> 各位工程师，我们的避难所已经升级到 V1.3，而你也即将启程，打造庇护之地。这本全新的「避难所建造手册」已经内置在哔哔小子中，希望它能够助你一臂之力。
>
> ——避难所科技 版权所有

第一章：生命支持
--------

由于废土世界污染严重，避难所不允许居民随便外出，因此地下居民的自给自足非常重要。

避难所科技提供的发电室、餐厅和净水器是居民生存的必要条件，这三种房间直接影响到避难所的三项生存指数，任何一个参数变红都意味着避难所出现危机，作为工程师的你可通过升级房间、调配人员来提高产量；如果发展前期人手实在不够，可以申请房间加速生产以解决燃眉之急，不过此举成功则已，失败便会导致灾难，例如变异昆虫侵害和火灾，如果出现上述问题，请参考第二章关于灾难处理的说明。

![](.evernote-content/50A9C42E-A14A-4F19-9B2B-D55136538971/818BEA46-D6AF-4AB7-9A25-1AD80F4EBE6A.jpg)

房间规划也关乎避难所兴衰，避难所科技的高效率房间允许合并，2 到 3 个同类、同级且相邻的房间可合并成一个中型或大型房间，容纳人数也依次增加，最多可容纳 6 人，避难所总容纳量也跟随设施建造而增加，如果各位工程师发现避难所人数止步不前，那就该考虑建设新的寝室或生产房间了。建造房间需要使用核子可乐瓶盖来购买材料，核子可乐瓶盖是废土世界中的货币，可由任务升级、任务奖励、外派探索、杀敌防御等形式获得。

![](.evernote-content/50A9C42E-A14A-4F19-9B2B-D55136538971/ADC30ADB-B759-47F4-9DA3-A5CC76662E48.jpg)

避难所科技考虑到设施的生产效率，每个避难所房间都有自己的属性，分别对应避难所居民评价标准，即 S.P.E.C.I.A.L.系统，想要让房间生产更加高效，就必须让小人的属性与房间属性相匹配。有的小人过于平庸怎么办？无需担心，冰冻三尺非一日之寒，后期可以通过解锁训练室和教室锻炼居民属性；如果你极其厌恶某人，也可以将他流放至废土，任其自生自灭。不过各位工程师要搞清楚「流放」与「外派」的区别，关于外派，请参考第五章相关部分。

![](.evernote-content/50A9C42E-A14A-4F19-9B2B-D55136538971/2643B850-6470-4512-828E-7D30B6B8B1D9.jpg)

第二章：防灾防害
--------

废土世界不比和平时期那样安全，由于辐射污染，许多蟑螂、老鼠变异成更具有攻击性的野兽，甚至会出现死亡爪这样的毁灭性生物；除了禽兽，废土世界外的掠夺者也是危害避难所安全的一大隐患，因此防灾减灾也是重中之重。

避难所内出现灾难分为三种情况：资源短缺、内生虫害、外来灾害。资源短缺应该归咎于人口膨胀导致生产力不足，进而影响居民状态、生产力再次骤减，形成恶性循环；解决此问题需要停止人口增加，必要时申请房间生产加速。

![](.evernote-content/50A9C42E-A14A-4F19-9B2B-D55136538971/745D8360-8D5B-4AEF-B117-D6844D72BEE9.jpg)

内生灾害与外来灾害属于不可抗力，因此一旦出现这两种问题便属于紧急情况。内生灾害可能由于生产加速失败而导致火灾和变异昆虫爆发，不过在正常生产情况下也会随机出现上述灾难；外来灾害一般是废土掠夺者入侵。火灾问题无需太过担心，避难所居民会主动拿起灭火器灭火，必要时可拖动其他房间小人参与灭火；变异昆虫和废土掠夺住出现时，我们唯有领导居民去战斗方可取胜（针对掠夺者，我们可以升级大门来拖延掠夺者的入侵），因此在平日里我们就必须时刻更新居民们的武器。关于武器部分，请参考第五章的说明。

![](.evernote-content/50A9C42E-A14A-4F19-9B2B-D55136538971/8D97C995-1E88-4995-8E9B-CB03652F7CF6.jpg)

各位工程师请注意：废土世界资源紧缺，如果你击杀了掠夺者，请及时点击他们的尸体，你会有意想不到的收获。

第三章：人丁兴旺
--------

避难所科技提供了高品质的生产环境，但他们终究离不开避难所居民的操作，因此充足的人口是避难所正常运营的充分且必要条件。

避难所科技建议您主要通过居民自我繁衍来增加人口，尤其是在发展前期，切不可依赖人员招募。居民自我繁衍的前提条件是具有至少一个寝室，然后命令孤男寡女同处寝室，只要两者的魅力值（即 S.P.E.C.I.A.L.系统中的 C）足够即可互生情愫，大约几分钟后你就能看到女方换上了黄色的孕妇毛衣，孕妇在大约 3 小时左右才会产子，孩童在至少 3 小时之后才会长大成人。需要注意的是，孕妇具有生产能力，但不具有防御能力；孩童不具有任何能力，只是四处游荡等待长大，因此当避难所中存在孕妇与孩童时，一定要注意他们的安全。

![](.evernote-content/50A9C42E-A14A-4F19-9B2B-D55136538971/1BC689DB-42FD-419B-8927-11E98516B425.jpg)

外部人员招募是另一种扩张人口手段。在发展前期，我们不能过于依靠招募手段，不过在避难所资源充足之后，各位工程师可以建设招募电台增大招募成功比例。招募电台最好由高魅力值的居民来运营；更大的招募电台、更多的运营人员也会相应增加招募成功比例。

![](.evernote-content/50A9C42E-A14A-4F19-9B2B-D55136538971/E0813072-F196-466C-99FC-443CAA2D982D.jpg)

虽说内部繁衍能够更快地增长人口，但谁都不希望避难所里的居民最后都成为了近亲，所以外部招募也同样重要。两种招募手段没有孰轻孰重，只有适时运用，请各位工程师注意。

第四章：生活秩序
--------

由于辐射危害，废土世界对正常人类依旧有威胁，所以各位工程师在建设避难所的同时需要安抚各位居民，保持良好的生活秩序，做好长时间地下生活的准备。

避难所科技建议各位工程师，请时刻注意各位居民的满意程度，高满意度反映了避难所内具有良好的生活秩序，因此我们获得更高的生产效率、更容易招募外部人员；如果各位工程师发现避难所满意度下降，有可能是由加速失败或资源不足导致。如果因加速失败而导致房间失火、害虫爆发的情况，在平息灾难后有可能会导致居民满意度下降；如果食物短缺、水资源不足，居民忍饥挨饿也会导致满意度下降。在避难所 V1.3 中，避难所科技为各位居民提供了宠物可供陪伴，关于宠物请参考第五章关于收集方面的说明。

![](.evernote-content/50A9C42E-A14A-4F19-9B2B-D55136538971/5F59F85E-3BA5-45DF-B9DA-C821DB5AF503.jpg)

在防御掠夺者、击杀变异害虫、救火灭火的过程中，各位居民难免会受伤；在外派探索过程中，防御废土辐射也至关重要；这些因素同样影响着避难所的生活秩序，因此在避难所建设过程中一定要建造医疗室和实验室以便生产治疗针和抗辐宁。需要注意的是，受伤的居民不会自动恢复血量，避难所科技建议各位工程师及时治疗受伤居民；外派居民请务必携带足够的治疗针和抗辐宁，否则此人有可能一去不回。

![](.evernote-content/50A9C42E-A14A-4F19-9B2B-D55136538971/2643B850-6470-4512-828E-7D30B6B8B1D9.jpg)

避难所科技为各位居民免费提供了避难所统一着装，但这件衣服不具备任何属性，如果各位工程师收集到其他衣服，请及时为避难所居民更换，因为除了默认服装外，其他所以衣服都具有不同的属性加成，因而影响到居民的个人属性和相关生产效率。考虑到良好的生活秩序包括高效生产，请各位工程师注意更换服装。有关于服装收集，请参考第五章相关部分。

![](.evernote-content/50A9C42E-A14A-4F19-9B2B-D55136538971/CD31BB38-7026-4AA8-9D73-E9FA684567DC.jpg)

第五章：废土收集
--------

在避难所居民正常生活的同时，各位工程师要打点好周边事物，例如物品配备。作为一个地下城堡，物品主要来自于外面的废土世界。我们可以通过探索废土世界的目的主要是为了获得武器和服装。在探索废土世界之前，避难所科技为各位工程师提供以下建议。

首先是外派人选。避难所科技的 S.P.E.C.I.A.L.系统能够充分评价一个人的综合素质，如果你需要一个优秀的外派人员，我们建议这个人的S.P.E.C.I.A.L.数值较高，如此才能够应对废土世界的复杂环境。其次，合适的装备能够让探索过程无比轻松，对人物属性有加成的服装都能够帮助居民探索世界，我们建议各位工程师首先考虑服装的力量加成；高攻击力的武器能够让小人走得更远、换来更好的武器，所以不要吝啬，放手一搏。

![](.evernote-content/50A9C42E-A14A-4F19-9B2B-D55136538971/2E0D496F-43D8-421B-95C9-B1047B7EAA76.jpg)

值得一提的是，避难所科技曾制造出无数动力装甲，但由于灾难他们散落在世界各地，如果你的居民有幸得到一套动力装甲，那么无论是避难所防御或是探索废土世界，你都能事半功倍。

除了服装和武器之外，核子可乐瓶盖也可以通过探索来收集，不过这一切的一切，前提是他能赶得回来。废土世界危险重重，如果你发现探索人员血量不足、药品用尽，请务必马上召回，否则这次探索必会以失败告终。另外，请保证你有足够的储藏室空间。

![](.evernote-content/50A9C42E-A14A-4F19-9B2B-D55136538971/5FF803BB-68CE-4EC3-A5C2-F83BC91D33C3.jpg)

如果你看到了这里，祝贺你，你已经成为了避难所科技认证的合格工程师，如果你在建设过程中遇到了资源不足、装备不足或人员紧缺等情况，可以通过商店直接向避难所科技总部购买救急箱、噶抓助手或者可爱的小宠物。

最后，请微笑面对每一位避难所居民。祝你顺利，工程师。
