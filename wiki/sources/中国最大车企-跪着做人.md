# 中国最大车企，“跪着做人”

连续18年产销冠军上汽集团，决定“跪着做人”。

10月中旬，上汽集团总裁贾健旭在8月年中干部会议的“下跪”言论突遭曝光。他提出，“要学会跪着做人，低调做人，高调做事”。

不破不立，这一“跪”本是必然。新能源巨头比亚迪挤掉上汽，登上中国汽车销量第一的宝座。

今年前三季度，上汽不仅销量下跌，营收下跌，净利润也下跌。第三季度连续三个月销量跌幅超35%。实际上，近三年上汽也是少数销量跌幅持续扩大的汽车企业。

![](attachments/9332e8a6d88e2ff5.png)

上汽集团2024年9月产销快报 / 图源：上汽集团官网

燃油车、新能源汽车两条战线失速。两大合资王牌上汽通用、上汽大众折戟沉沙，前者更遭到“凯迪拉克别克化，别克雪佛兰化，雪佛兰边缘化”的嘲讽；大众ID.3指导价已不到13万元。新能源车智己、飞凡同室操戈，一起在20万元价位厮杀。

四年前，上汽董事长陈虹的“灵魂论”备受质疑。彼时，上汽不愿考虑华为、百度提供的整体解决方案，“这样会变成他是灵魂，我是躯体”。四年后，就任总裁1个月的贾健旭在“年中讲话”喊出“学会跪着做人”。

![](attachments/52a5f867784a3546.png)

上汽集团总裁贾健旭 / 图源：上汽大众‍‍‍‍

从“灵魂”到“下跪”，并不能以“自大”到“谦卑”的逻辑粗暴理解。作为实力最强的国有车企，上汽在中国汽车电动化转型之路上“起了个大早，赶了个晚集”，如今市场格局变为新能源主导，合资品牌失去了竞争力，上汽集团陷入低谷。

从辉煌的过去走入低迷的现状，“跪着做人”只是一种精神准备。如何在新的市场环境下找到新的增长点，是上汽必须解决的难题。

![](attachments/db12fb98c25ea5f0.png)

内忧外困

2024年下半年，中国汽车市场发生了历史性变化：燃油车时代与新能源时代更迭，中国第一车厂的名头也随之更替。

连续18年的“销冠”头衔，上汽只能拱手让给比亚迪。前三季度，上汽累计销量为264.9万辆，同比下滑21.6%。而比亚迪累计销量达274.8万辆，同比上涨32.1%，比上汽多卖了10万辆；并将保持增长势头挺进第四季度。

上汽的汽车，卖不动了。

上汽两大合资王牌上汽大众、上汽通用，今年销量大幅下滑。曾经的销量支柱上汽大众前三季度销量77.21万辆，同比下跌7.24%。上汽通用前三季度销售72.42万辆车，同比下跌61.55%。

![](attachments/4a36d9c504eb3b1b.png)

上汽集团2021-2024年6月的每股收益情况 / 图源：上汽集团官网

上汽乘用车分公司荣威、名爵，前三季度累计销量48.13万辆，同比下跌26.01%。上汽通用五菱和上汽大通，9月销量同比下滑均接近20%。

销量唯一没有下滑的是智己，9月销量6263辆，同比增长246.98%，主要是因为L6车型上市。

其余品牌中，上汽正大、上汽通用五菱印尼公司、名爵印度公司等，销量也都有不同程度的下滑。

同时，外部环境愈发恶劣。

2024年9月13日，美国宣布从中国进口的电动汽车的关税将从25%升至100%。

![](attachments/a08daea9ae672855.png)

9月13日，美国宣布中国进口的电动汽车的关税升至100%

根据欧盟10月4日投票通过的对中国产电动车加征关税的政策，上汽集团被加征的关税最高，高达35.3%，受到的不利影响也最大。吉利和比亚迪被加征的关税分别为18.8%和17%；特斯拉受到的影响最低，被加征7.8%。

一方面，除了智己，目前上汽所有品牌的汽车销量均难挽颓势。一方面，靠出口“堤内损失堤外补”愈发行不通，发达国家和地区均实行严厉的贸易保护主义政策，去发展中国家和地区则要面对来自中国新能源汽车的激烈竞争。

然而，在上汽一枝独秀的智己，也缺乏拯救集团的回天之力。

智己成立于2020年，含着三把金汤匙——上汽集团、张江高科和阿里巴巴，一举叠满老牌国企、高新技术、互联网三大buff。同时期还诞生了脱胎于荣威的新能源汽车飞凡。

![](attachments/d9a521fd2498dcfa.jpg)

智己汽车成立于2020年 / 图源：IM智己汽车

彼时，智己和飞凡成了上汽“品牌向上”的两大主力军，肩负上汽集团转型升级的重任。飞凡主攻20万元级市场，智己上攻30万元级别市场。这两年，新能源汽车价格战如火如荼，智己和飞凡被逼进同一个“斗兽场”，都在20万元上下厮杀，互相掣肘。

而且，月销量破千的只有智己L6和LS6，但破千也是从今年才开始的。其数据横向与其他友商销量一比，相差极大。智己L6对标特斯拉Model3和小米SU7，Model3今年月销量都在2万辆上下，SU7的月销量超过1万辆。

![](attachments/eac17dd08786f6ee.png)

合资车的黄昏

今年7月，上汽集团核心高管更迭。董事长陈虹到龄退休，原总裁王晓秋接任。此前担任副总裁兼上海大众总经理的贾健旭出任上汽集团总裁。

上汽已经到了必须改变的时刻。

从2024年6月开始，比亚迪的月销量超过上汽，并有“越超越远”之势。

新的霸主已经诞生，上汽的高光时刻还停留在2018年。那一年，上汽销量达到705.2万辆的历史高点，此后逐年下跌，到2023年跌至502.1万辆。

反观比亚迪，2018年至2020年，年销量只有50万辆。从2021年开始，比亚迪销量跃升，2023年达到302.4万辆。2024年前三季度，比亚迪已经“反超”上汽。

![](attachments/e3397b1fbf4f457b.png)

2024年全国汽车销量排行榜 / 图源：大搜车智云

过去太成功，往往成为企业最大的包袱。一旦外部环境变化，原有优势迅速消失。

上汽集团过去的成功，大部分靠合资汽车。上汽大众成立于1984年，是中国最早的合资车企之一，引入的桑塔纳、帕萨特、途观车型，不仅成为中国消费者眼中“好车”的代表，也为中国打下现代化汽车供应链的基础。上汽通用（泛亚）成立于1997年，具备当时大多数合资公司没有的产品研发能力。

另一家三方合资车企是上汽通用五菱，多款微型小汽车被誉为“神车”。

合资车的黄金年代，上汽大众、上汽通用与一汽大众轮流坐庄，争夺乘用车销冠宝座。2018年黄金年代的顶峰，上汽大众、上汽通用和上汽通用五菱的销量占上汽集团的86.6%。到2023年，三家公司总销量占比仍高达72.1%。

![](attachments/b499307e6fb32c37.jpg)

上汽通用五菱奔赴埃及实地考察 / 图源：SGMW上汽通用五菱

也就是说，新能源汽车狂飙突进这几年，上汽的新能源汽车表现十分有限，还是燃油车扛起集团的大半壁江山。

起了个大早，赶了个晚集。上汽本来是最早在电动、智能、网联和共享“新四化”方面全面布局的企业。早在2006年，上汽就领先比亚迪两年推出第一代插电混动技术和车型。到了2024年5月，比亚迪董事长王传福说，国内每卖出两辆插混车型，就有一辆是比亚迪。

上汽通用的衰落呈一路下滑的趋势。三大品牌凯迪拉克、别克、雪佛兰，凯迪拉克“以价换量”，还带上了“车主最爱去洗浴中心”的标签；别克靠MPV车型GL8打天下，但在“三缸”英朗栽了大跟头；雪佛兰的命运和美国本土“国民车”截然相反，中美双方定位也出现分歧，通用汽车希望提升形象、走高端路线，上汽更看重销量，结果雪佛兰不上不下，沦为和北京现代、上汽大众斯柯达一样的“小透明”。

![](attachments/7f8d28849b6df593.jpg)

雪佛兰沦为“小透明”

2020年新冠疫情暴发、国内新能源汽车崛起，上汽大势已去，凯迪拉克和别克都推出电动车型，几乎无人问津。

而且通用汽车作为美国公司，首当其冲受到中美贸易战的影响。2018年开始就承受25%的关税。如今美国即将对中国电动汽车加征100%关税，对动力电池加征25%的关税，上汽通用的生存更加艰难。

上汽大众也遭遇困难，但没有上汽通用那么严重。中国市场对德国大众汽车集团非常重要，这也是欧洲最初推出“双反调查”时德国不愿参与的重要原因。

贾健旭接手上汽大众后，推动大众集团与上汽集团先后达成大众、奥迪多款新能源车型联合开发计划——相当于为合资公司重新寻找到了技术和产品来源，改变了之前“等靠要”的惯例。

![](attachments/a9dc9d5c1622781b.png)

不破不立

2024年8月，46岁的贾健旭在接任上汽集团一个月后的年中会议喊出“跪着做人”，凸显了其作为“救火队长”的急迫心情，某种程度也突显了改革的决心。

原董事长陈虹的“灵魂论”饱受诟病，但拒绝华为其实是上汽集团的理性选择。作为大型国有企业，拥有合资车的辉煌历史，家底丰厚的上汽有实力自研自动驾驶和智能化技术。在其任上，上汽在自动驾驶、固态电池方面都有较大投入，但是一直没有形成鲜明的智能化优势。

随着集团换帅，主要子公司的管理层也先后“换防”。

8月9日，上汽通用宣布，泛亚原执行副总经理卢晓担任上汽通用汽车总经理。8月14日，上汽大众和上汽乘用车也宣告管理层更迭。上汽大众原销售与市场执行副总经理俞经民调至上汽乘用车，转任常务副总经理。上汽乘用车原业务规划和项目管理部动力驱动平台执行总监祝勇升任副总经理。

![](attachments/af2d16a4103f8867.png)

卢晓担任上汽通用汽车总经理‍

随着高层人事调整告一段落，上汽正准备“背水一战”。

“跪着做人”是上汽改革的第一步，打破内部保守的文化惯性。所谓“破山中贼易，破心中贼难”，上汽内部不乏傲慢自大的氛围，一些干部“头抬得比天都高”，令企业无法适应市场的快速变化，贾健旭要求整个企业放下架子，低调务实面对市场，以便促成更深层次的企业变革。

当然，对于曾经的行业老大上汽来说，“触及灵魂”分外艰难。

上汽改革的第二步也是核心步骤，则是贾健旭提到的“零整同”。把丰富的自研自产的零部件企业和整车企业配合起来，不能各自为战。

![](attachments/01a1664e8f61e76a.png)

贾健旭在媒体交流会上透露，大众品牌的所有混动车型将与上汽联合开发 / 图源：@汽车大公司‍‍

上汽除了八大整车业务，零部件业务也极其强悍。像零束、延锋、联合电子、华域汽车……都是中国零部件产业的佼佼者。其业务范围涵盖智能驾驶、智能座舱、内外饰、座椅、门板、安全系统、座舱和车身电子、车载光学元器件、发动机、变速器、电驱动、混合动力系统、底盘、热管理系统等……

比亚迪除了不做玻璃和轮胎，其他都能自给自足。其实上汽早就达到了这一水平，而且超越了这一水平——中国每一款豪华畅销车上，都有上汽系的零件。比亚迪目前还做不到这一点。

只是零部件公司先考虑利润，后考虑集团和整车利益，因此都在“外面”赚钱。但现在整车业务不盈利，盈利的零部件就要“回归”，一是“肥水不流外人田”，零部件的利润也要收回来；二是借大平台优势全栈自研，保留自己的技术和产能优势；三是兜底，一些赚不到钱的项目，至少有自家供应商承接。

必须要指出的是，“零部件和整车一条心”，特别难。

像比亚迪和长城，整车厂和零部件也不是百分百“同心同德”。长城旗下的曼德光电有空调产品，整车厂还是会大量采购弗迪的热泵。比亚迪在自身整车冲焊产能不足的前提下，也会去大量采购长城精工生产的“五门一盖”。

![](attachments/33083a4454dfbf61.jpg)

实现“零部件和整车一条心”，特别难 / 图源：上汽集团

由此可见，不管“关系”如何，双方的选择还是“本能”地遵循商业法则——谁性价比高，就买谁。

“跪着做人”引发了行业内外、甚至全社会的关注，对上汽来说是好事。

所有人都看到了上汽全方位改革的决心和措施，走出了很多头部车企都不敢走的一步。很多时候，自我变革都是痛苦的，但学会“跪着做人，才有站起来的那天”。
