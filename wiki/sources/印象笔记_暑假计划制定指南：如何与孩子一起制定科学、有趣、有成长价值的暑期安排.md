# 暑假计划制定指南：如何与孩子一起制定科学、有趣、有成长价值的暑期安排

---

来源：[打开原文](https://mp.weixin.qq.com/s/kYzZwOEO_LJfTIcCPtONQQ)

![](.evernote-content/6713F8A3-DA52-41C3-ADBE-C1E48E2AEEE8/BEEB9D65-4A3E-4507-95DE-98207F3C7D74.png)

作者说明：这是一份关于"怎么和孩子一起过好暑假"的笔记。它想解决两件事：一是心法——暑假首先要摆正的是大人的"态度"，把孩子当成一个独立的人来尊重、倾听，而不是当作被管理的对象；二是技法——在心法之上，怎么把学科、作息、兴趣、屏幕时间安排得既科学又能执行下去。文中既有儿童发展心理学的依据，也有我自己正在陪孩子试的几个真实做法。后面的脑图、作息表、会议模板都是工具，请按你家孩子的情况取用；但请先读"零、写在前面的心法"那一节——工具用得再熟练，态度错了，暑假也过不好。

零、写在前面的心法：先有态度，再有计划
-------------------

先把一句话放在最前面：暑假过得好不好，七分看大人的态度，三分看方法。 工具、模板、作息表都是末节；真正决定这两个月温度的，是大人愿不愿意把孩子当成一个"独立的人"来对待。

吴军在《态度》里写给两个女儿的家书，开篇就把这件事说透了——子女不是家长的私有财物，而是上天给父母带来的最好的礼物。 这不是一句鸡汤，它会实实在在地改变你怎么开口。当你真把孩子当成"礼物"，而不是"我的作品""我的任务"，很多话术都不用刻意背：你会自然而然地先听他讲完，再表达自己的担心；你会允许他说"我不想做"，而不是立刻顶回去"你怎么能不想做"。

吴军把"和子女沟通"总结成四条心得，我把它们翻译成暑假里能直接用的几条：

1. 先承认他是个独立的人，需要被尊重。 计划是他的暑假，不是你的项目。所以这份计划必须是"商定"的，而不是"下达"的——后面第二章所有的会议流程，都是在为这一条服务。你说出"你有权利拒绝我的想法"的那一刻，态度就对了。

2. 承认他和你不一样。 他热了会自己跑去把风扇调大、挪到风大那一头接着睡；他喜欢把事情安排得清清楚楚再动手，没有确定性的话会急得直哭——这些"不一样"不是要被纠正的毛病，而是你要顺着走的纹理。顺着他的条理性去做脑图、做作息表，阻力最小、效果最好。

3. 沟通是双向的，倾听比表达重要。 我有一个真实的体会：我跟孩子说"只要能认真做好一件事，就能做好很多件事"，他当场纠正我——"是做好每一件事，不是很多件事。"那一刻我特别受触动。一个能听懂大人在说什么、还能给出更准确建议的孩子，你要做的就不再是"教"，而是"听"。倾听，常常比引导更有力量。

4. 重要的约定，写下来，必要时甚至写信。 吴军发现孩子进入青春期后，文字沟通比面对面更有效——它避开了话赶话的争执，还顺带练了表达。暑假的约定（写作业的标准、游戏时间的规则），不妨让孩子自己写在脑图或一张纸上、贴在墙上。白纸黑字，是他的承诺，也是大人的克制——免得我们一着急就随口改口。

还有一句也要放在心里：乐观的态度，比什么都重要。 吴军说，起点差不多的人，会因为做事态度不同而不断分化，慢慢拉开差距。暑假正是练这个的时候——遇到没做好的地方，不说"你看，我就说你不行吧"，而是"我们一起想想，是哪里卡住了"。你保持什么态度，孩子就会学到什么态度。

关于"达标奖励"，我自己正在试的一个做法： 我跟孩子约定——写字一笔一划、有错必改、当天达标，就可以和小伙伴一起玩半小时"我的世界"。试行第一天，他写得格外认真。我后来想明白它为什么管用：因为它正好踩中了三件事——这个标准是他自己认的（他做得了主）、认真写字他确实做得到（他有成就感）、达标后那半小时，是他和好朋友一起挖地道、挖到彼此家里去的快乐时光（有人陪着、有人懂他）。奖励之所以有效，不是因为它"诱人"，而是因为它尊重了他的努力和他的喜好。

如果你只有时间记住一节，就记住这一节。下面所有的"技法"，都建立在这个"态度"之上；态度错了，再漂亮的计划表也只是一张纸。

一、为什么暑假计划需要"被设计"
----------------

1.1 暑期学习损失：被忽视的成长陷阱
-------------------

教育研究早就发现一个现象：暑假两个月，孩子如果完全不碰书本，开学后会"退步"大约一个月的量，数学比语文退得更明显。这个"一个月"不用太较真具体数字，但趋势是实在的——暑假完全放任不管，知识会回撤。

所以设计暑假计划的目标，不是把孩子变成学习机器，而是在放松和维持之间找个平衡。

1.2 暑假独有的教育价值：学校做不到的事
---------------------

暑假的价值远不止"不落后"。它是极少数没有同伴比较压力的时间窗口，孩子可以：

• • 深入探索一个感兴趣的领域（哪怕开学后会放弃）

• • 大量阅读没有"任务属性"的书

• • 尝试学校不会教的技能（做饭、骑行、编程、乐器）

• • 发展非学业维度的能力：自主性、时间感、抗挫折力

计划的目的不是填满时间，而是创造有质量的时间体验。

1.3 三条朴素的原则
-----------

后面所有方法，背后就三条很朴素的原则。记牢它们，比记任何技巧都管用：

① 让孩子觉得"这是我自己想做的"。 一份计划如果只是"被要求的"，孩子心里会一直别扭，执行的阻力会磨掉所有效果。怎么让他"想做"？让他参与定、让他选、让他有发言权——能自己做主、觉得自己做得到、还跟家人有连接，这三样就够。

② 目标定在"踮踮脚够得着"的高度。 太简单，他无聊；太难，他挫败。最好的难度，是比他现在的水平稍微高一点点、努努力能完成的那种。

③ 管计划、管冲动的大脑，要慢慢长。 负责做计划、忍住冲动、切换注意力的那部分能力，要到十二岁左右才基本长好。暑假让孩子自己参与定计划、守时间、回头看进展，其实就是在帮这部分大脑"练力气"——这本身就是暑假最大的收获之一。

二、沟通先行：如何与孩子"开会"制定计划
--------------------

2.1 一个重要的认知前提：计划必须是被"商定"的
-------------------------

很多家长的失败不是执行不力，而是从一开始就错了——父母单方面制定了一份"完美计划"，然后告诉孩子"这是你的暑假安排"。

孩子不会认同一份不属于他的计划。认同感是执行的前提，不是结果。

正确的起点是：把孩子当作平等的参与者，而非被管理对象。以下沟通框架基于此前提构建。

2.2 第一次家庭会议：愿景与边界（建议时长 30–45 分钟）
--------------------------------

时间选择：周末或假期前一天的放松时段，避免在孩子疲劳或情绪不佳时进行。

参与成员：父母 + 孩子，祖辈如需参与提前单独沟通（避免当着孩子面产生分歧）。

会议流程：

【议程一】开场仪式（5 分钟） — 父母开场白，建议话术：

"这个暑假我们有大概 60 天，这是一年里唯一一段可以完全自己安排的时间。我想和你一起做个计划，让这 60 天过得有意思、有收获。这个计划我想听你的想法，你也有权利拒绝我的想法，我们一起找一个两个人都觉得合理的方案。"

【议程二】暑假愿景讨论（10 分钟） — 问孩子："你觉得暑假最好是什么样子的？"（开放式，不评判）。记录孩子的回答，再用孩子的语言而非大人的框架重述：

"所以你喜欢的是：可以睡懒觉、能打篮球、有时间玩你喜欢的游戏，还有……对吗？"

【议程三】父母的期望分享（5 分钟） — 用"我希望"而非"你应该"：

"我希望暑假里你能保持数学练习，这样下学期不会太吃力；也希望你有时间多看书，不是那种一定要写读后感的书，是你自己想看的书。我还有些担心你的作息，假期你经常凌晨才睡……"

【议程四】共同列出"要事清单"（10 分钟） — 把双方期望合并成一张认可的清单。规则：每方各列 5 项，想加新的就得先删一个旧的——这是在练优先级。

【议程五】约定边界（5 分钟） — 父母明确表达不可商量的底线：

"这些是我和你妈觉得重要、不能没有的：每天阅读 30 分钟，数学每天 10 道题，户外至少 1 小时，屏幕时间每天不超过 X 小时。其他的，我都可以和你商量。"

【议程六】约定下一次会议（2 分钟） — "我们下周一再坐在一起，把这份清单变成具体的时间表，好不好？"

注意：第一次会议的目标不是出方案，而是建立共识氛围。如果孩子拒绝发言，不要强迫，可以换一个更轻松的问题（比如"你最怀念去年暑假哪件事？"）。

2.3 第二次家庭会议：结构化时间表（建议时长 45–60 分钟）
---------------------------------

前置准备：父母根据第一次会议的清单，准备好一份"空白时间表"草稿，展示结构而非填满内容。

会议流程：

【议程一】回顾与感受确认（5 分钟） — "上周我们说了这些（展示清单），你觉得都还合理吗？有没有想加或想减的？"

【议程二】确定每日时间模块（15 分钟） — 展示模块选项，和孩子一起讨论每天怎么分配：

• • 晨间（起床后）：适合需要专注的学习（数学、语言）

• • 上午：适合户外 / 运动

• • 午后：适合兴趣探索、阅读、动手实践

• • 傍晚：适合轻度任务或自由时间

• • 晚间：亲子时间、阅读、准备第二天

分配原则：① 学习任务放在孩子精力最充沛的时段；② 同一模块每天尽量固定（建立节奏，减少决策疲劳）；③ 周末和周中可以有区别。

【议程三】分配学科任务（15 分钟） —

• • 语文：阅读 + 表达（口头复述 / 日记，不强制写作文）

• • 数学：每日计算练习（10–15 分钟）+ 每周一道思维挑战题

• • 英语：听说为主，读英文绘本或看动画（无字幕 → 有字幕 → 尝试跟读）

• • 体育：每天至少 1 小时户外，运动项目自选

【议程四】用脑图呈现（10 分钟） — 打开思维导图工具（见第五章），当着孩子的面把讨论内容画出来，颜色、图标由孩子选——这是建立"所有权"感的关键一步。

【议程五】约定检查点（5 分钟） — "我们每周末开一次 15 分钟的短会，看看执行得怎么样、哪里要调整。"——明确检查频率，避免"计划制定后无人问津"或"每天被追问进度"两个极端。

2.4 沟通话术参考
----------

• • 孩子不想做作业　✗"你必须做，不做不行" → ✓"我理解不想做。我们看看有没有办法让它变得没那么痛苦？"

• • 计划执行不理想　✗"你看，我就说你不行吧" → ✓"这周有好几天没按计划来，是计划太满了，还是有别的困难？"

• • 孩子想加游戏时间　✗"游戏有什么好玩的" → ✓"你想多玩，说明它真的有意思。我们看看能不能在完成任务后多玩一会儿？"

• • 周末想睡懒觉　✗"作息必须规律" → ✓"平时上学起得早，周末可以多睡会儿。我们定一个最晚起床时间好不好？"

• • 孩子抱怨计划太难　✗"别人家的孩子都能做到" → ✓"哪里觉得难？是太多了，还是某个科目特别不想做？"

核心原则：先连接，再引导。先承认情绪的合理性，再讨论解决方案。

三、学科规划：四科的具体安排原则
----------------

3.1 语文：阅读是核心，写作是副产品
-------------------

暑假语文的核心任务不是"提前学下学期课本"，而是大量自由阅读。

道理也简单：有研究发现，暑假读得多的孩子，开学后阅读理解明显更好；而且"想读"比"读多少"更要紧——被逼着读的，效果差一大截。

具体建议：

• • 每日阅读时间：30–45 分钟，类型不限（漫画、小说、科普、杂志均可）

• • 读后互动：睡前或餐桌上问"今天看了什么有意思的？"——无需写读后感，口头复述同样有效

• • 表达训练：每天选一个新闻或书中情节，用自己的话讲给家人听（训练口头表达能力）

• • 写作：不强制作文，但鼓励每天记日记（见下面"日记"部分）

日记：练"把一件事讲清楚"（暑假最有性价比的写作练习）

作家马伯庸有个观点我很认同：小学生不需要学复杂的抒情技巧，他只需要学会一件事——把一件事客观地、有条理地讲清楚。 这听起来简单，其实很难，得长期练。而每天记日记，就是最好的练习场。

怎么练？让孩子每天用一两句话，把当天的一件事讲清楚，重点练两种最基本的叙事：

• • 线性叙事（按时间顺序）：什么时间、什么地点、和谁一起、做了什么。比如"下午三点，我和小明在楼下踢了半小时球，我进了两个。"

• • 总分结构（结论先行）：先说结果，再补细节。比如"今天最高兴的事是学会了自己煎蛋——先打蛋，再开小火，最后翻面，成了。"

习惯了这种"说清楚一件事"的叙事，到了写作文，结构自然就有了。先要求他"写得清楚"，再谈"写得有感情"。

我的做法：让孩子用语音助手（我们用的是 Agent 龙虾）口述当天的日记，他只要说，agent 帮他记成文字。一来他不用为"写字累"发愁，愿意每天记；二来暑假结束想回顾"这两个月都发生了什么"，翻开记录就全在——比凭记忆靠谱得多。这也正好呼应《态度》说的"用文字沟通"：让他从小习惯用文字留下自己的想法。

推荐暑假书目类型（不限于纯文字书）：

• • 冒险/奇幻小说（建立阅读习惯）

• • 科普图文书（满足好奇心）

• • 人物传记（榜样的力量）

• • 短篇故事集（降低阅读压力）

3.2 数学：防遗忘优先，挑战题其次
------------------

暑假数学的首要任务是防止遗忘，而非超前学习。

为什么数学最要紧？前面说过，数学是暑假最容易"回撤"的科目，丢的主要是计算速度和基本概念。至于"提前学下学期内容"，往往得不偿失——暑假学一遍，开学老师再讲一遍，孩子容易觉得"我已经会了"，上课反而不听了。

具体建议：

• • 每日计算练习：10–15 分钟（推荐使用 app 如「数学天天练」或购买计算练习册）

• • 内容选择：以本学期已学内容为主，确保基础扎实

• • 每周一道思维挑战题：选一道略高于当前水平的趣味题，允许查阅答案、允许讨论

• • 生活中的数学：购物算账、烹饪用量、时间计算——这些"隐性练习"比做题更有意义

数学暑假不要做的事：

• • 提前把下学期课本过一遍（除非孩子强烈要求且学有余力）

• • 大量刷题（磨灭兴趣，浪费时间）

• • 强制背公式（理解比记忆重要）

3.3 英语：沉浸优先，碎片时间利用
------------------

语言学习的关键是暴露频率，而非单次时长。暑假是英语沉浸的好时机。

具体建议：

• • 每日听力输入：15–20 分钟（选择比当前水平略难的动画或 podcast，不强制跟读）

• • 推荐资源：「小猪佩奇」英文版→进阶「糟糕历史」→自然过渡到「老友记」等生活剧

• • 阅读：英文绘本或分级读物（RAZ、牛津树系列），每天 10–15 分钟

• • 口语：如果有机会，参加暑期英语夏令营或线上口语课；没有条件则让孩子尝试"配音秀"

• • 单词：使用闪卡 App（如 Anki 儿童版）每日 5–10 个新词，不强制默写，看见认识即可

核心原则：暑假英语的目标不是"学多少"，而是"保持接触英语的习惯"。

3.4 体育：每天都要动一动
--------------

体育不仅关乎身体，更关乎大脑和学习状态。

运动不光长身体，也长脑子——动起来之后大脑供血更足，记忆力和专注力都会跟着好起来，这是有研究反复证实过的。

具体建议：

• • 每日户外运动：至少 1 小时，形式不限（骑车、游泳、篮球、踢球、跑步、跳绳均可）

• • 避免两种极端：完全不运动，或每周只运动一次但时间极长（后者伤害大于前者）

• • 建议结构：每周选 1–2 项孩子喜欢的运动作为"主项"，其余时间自由活动

• • 高温天气：将户外运动调整到早晨或傍晚，中午时段改为室内运动（跳舞、游戏等）

暑假运动目标：不是培养运动员，而是让孩子体验"运动是生活自然组成部分"。

四、作息规划：建立节奏，而非机械填表
------------------

4.1 暑假作息的核心原则
-------------

暑假作息与上学时最大的区别是：可以放松，但要有底线；可以灵活，但不能混乱。

建议暑假作息框架（可根据家庭实际情况调整）：

工作日参考作息（按家庭情况调整）：

• • 07:30–08:00　起床 + 早餐（不晚于 8 点）

• • 08:00–09:00　晨间：数学 / 英语（精力最充沛，适合高强度任务）

• • 09:00–09:15　休息 + 零食

• • 09:15–10:00　语文阅读 / 写作 / 英语

• • 10:00–11:30　户外运动（乒乓球、篮球、骑行、游泳等）

• • 11:30–12:00　自由活动 / 家务（整理房间、帮忙做饭）

• • 12:00–13:30　午餐 + 午休（建议午睡 30–45 分钟，不宜过长）

• • 13:30–15:00　兴趣探索：画画、乐器、编程、实验等

• • 15:00–17:00　户外活动或社交（找邻居小朋友玩）

• • 17:00–18:30　自由时间（屏幕时间可安排在此）

• • 18:30–19:30　晚餐 + 家务

• • 19:30–20:30　亲子时光 / 阅读

• • 20:30–21:00　洗漱 + 准备睡觉

• • 21:00　就寝（建议不晚于 21:30，暑假可比平时晚 30–60 分钟）

周末参考作息：

• • 09:00 前　起床自由时间

• • 09:00–10:00　周末家庭早餐

• • 10:00–12:00　户外活动（全家参与：爬山、公园、自行车等）

• • 12:00–13:30　午餐

• • 13:30–15:30　自由活动 / 兴趣班

• • 15:30–17:00　家庭项目（一起做饭、手工、参观博物馆等）

• • 17:00 后　自由时间

4.2 让孩子参与制定作息表
--------------

方法：在脑图或纸质表格上，用不同颜色标注不同类型活动。让孩子自己决定颜色的对应关系。

关键：留白比填满更重要。 每天至少留 2–3 小时完全由孩子自由支配，这部分时间不安排、不评价、不追问——这是培养自主性的核心机制。

五、脑图工具：从想法到可视化的操作指南
-------------------

5.1 为什么用脑图
----------

脑图（思维导图）的好处，是它顺着大脑"从一个中心往外发散"的思考方式，一眼能看到整体，也能看清细节，还好看好记。

对孩子来说，脑图比一张密密麻麻的文字计划表友好得多——它把"做计划"这件事的心理门槛降低了，让定计划本身变成一件有意思的亲子活动。

小提醒：脑图的真正价值，不在"提升多少效率"这种说法，而在"和孩子一起画"这个过程——一起商量、一起选颜色、一起贴上墙，本身就值得。

5.2 暑假计划脑图结构示例
--------------

暑假计划脑图的主干——中心是"暑假计划"，向外五个主分支：

• • 语文：阅读、表达

• • 数学：每日计算、每周挑战题

• • 英语：听力、阅读

• • 体育：每日户外、主项运动

• • 作息：平日、周末（含自由时间）

每日细化版（中心节点写上孩子姓名 + 第几周）：

• • 晨间 → 数学 30 分钟 + 英语 15 分钟

• • 上午 → 户外运动 60 分钟

• • 午后 → 语文阅读 30 分钟

• • 傍晚 → 兴趣时间（孩子选择）

• • 晚间 → 家庭阅读 / 亲子游戏

5.3 工具推荐
--------

• • XMind（全平台）：功能完整、模板丰富，家长和孩子可共用。

• • 幕布（App + 网页）：大纲与脑图一键切换——把大纲直接粘进去就能转脑图，适合高年级。

• • 百度脑图（浏览器）：免费、免安装，随时可用。

• • Canva（App + 网页）：模板美观、支持图片，适合有设计兴趣的孩子。

• • 手绘脑图（纸 + 彩笔）：最直接，亲子一起画，适合小学低年级。

小提醒：工具不是重点，"和孩子一起画"这个过程才是。哪个顺手先用哪个。

5.4 亲子绘制脑图的操作步骤
---------------

1. 1. 准备阶段（5 分钟）：父母打印或打开空白脑图画布，准备彩色笔

2. 2. 中心节点（3 分钟）：孩子写下自己的名字和暑假起止日期

3. 3. 第一层分支（10 分钟）：一起讨论确定主分支（学科、作息、兴趣、社交等）

4. 4. 第二层展开（15 分钟）：每个分支往下延伸具体内容，由孩子口述，父母或孩子手写/打字

5. 5. 美化阶段（10 分钟）：选择颜色、添加图标或小表情

6. 6. 拍照存档（1 分钟）：拍一张保存到手机，作为纸质版备份

重要提示：脑图完成后，一定要打印出来贴在家里显眼位置（冰箱门、书房墙），让它成为家庭生活的"参照物"而非"存档文件"。

六、常见问题与应对策略
-----------

问题一：计划制定了，但孩子根本不执行
------------------

诊断：计划缺乏孩子的参与感，或目标太高。

应对：

• • 召开计划回顾会，不是追责，而是共同调整

• • 问孩子："你觉得哪里最难做到？"——答案往往是调整方向的线索

• • 降低门槛：比如数学从每天 20 题降到 10 题，完成率比数量更重要

问题二：孩子玩手机/游戏停不下来
----------------

诊断：缺乏有吸引力的替代活动，或屏幕时间是唯一的奖励。

应对：

• • 不完全禁止，而是结构化管理：每天在完成任务后有固定屏幕时间（如 1 小时）

• • 和孩子商量规则，孩子参与制定的规则比父母强加的更容易遵守

• • 丰富非屏幕活动选项：运动、社交、户外活动

问题三：计划过于死板，一有意外就全盘崩溃
--------------------

诊断：计划缺乏弹性。

应对：

• • 在计划中预留"弹性时间"（每天 1–2 小时空白，不安排任何任务）

• • 允许周末与周中有差异

• • 记住" 80% 规则"：完成了 80% 就是成功，不需要 100%

问题四：孩子对某些学科完全抗拒
---------------

应对：

• • 先区分"不擅长"和"不喜欢"——孩子往往因为不擅长而不喜欢

• • 降低难度起点（倒退到基础题），重建胜任感

• • 寻找学科与兴趣的交叉点：喜欢历史的孩子可以先从历史故事书开始语文阅读

问题五：家长工作繁忙，无法全程陪伴
-----------------

应对：

• • 借助工具降低监督成本：任务打卡 App（如「樵夫宝」）、自动计时器

• • 与孩子约定"自主时间块"，在这段时间内孩子独立完成，父母只需检查结果

• • 周末弥补：即使工作日陪伴有限，周末全心投入也能建立安全感

七、每周家庭会议的迷你模板
-------------

每周末花 15 分钟，做一个轻量级的计划检视，按顺序聊四件事：

1. 1. 这周做得好的三件事？（双方各说一件）

2. 2. 这周哪里觉得比较难 / 没做到？

3. 3. 下周有什么需要调整的？（只改 1–2 项，不要大幅重写）

4. 4. 有什么开心的事想分享？

规则：

• • 这不是批评会，是调整会

• • 父母先说自己做得好和需要调整的地方（示范）

• • 孩子的反馈无论多幼稚都要认真回应

结语
--

暑假是孩子成长里为数不多"可以慢下来"的两个月。一个好的暑假计划，不是把孩子变成自律机器，而是陪他在放松中保持节奏，在自主里学会为自己负责。

但请记住一句话：计划是手段，人才是目的。 这两个月你真正在做的，是陪一个独立的小人儿，慢慢长成他自己。当他纠正你"是做好每一件事，不是做好很多件事"的那一刻，你该庆幸——他在认真地对待生活里的每一件事，比很多大人都认真。

所以，最终极的目标是这样一种感觉：暑假结束那天，孩子心里是"这个假期是我的，我过得很不错"，而不是"终于开学了，可算熬过去了"。而你自己呢，是轻轻松松说一句"这两个月，我也和孩子好好相处了一段"，而不是"总算盯完了"。

态度对了，方法都是顺手的事。
