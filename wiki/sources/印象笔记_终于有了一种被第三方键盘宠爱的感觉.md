最早在十年前我就开始尝试给 Mac 搭配各种机械键盘来提升打字手感。但是早些年的键盘基本都是针对 Windows 设计，用在 Mac 上需要一番折腾。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/E90F5427-9D2F-421B-A86B-CAD1E610C804.jpg)

2015年我的第一把机械键盘

这几年，针对 Mac 优化设计的键盘越来越多，比如我以前介绍过的 [Keychron K3](https://mp.weixin.qq.com/s?__biz=MzI2MTAzMzcyNQ==&mid=2648562855&idx=1&sn=a4a113b2b597093014d09ee97632f85d&scene=21#wechat_redirect)、[iQunix MG65](https://mp.weixin.qq.com/s?__biz=MzI2MTAzMzcyNQ==&mid=2648563346&idx=1&sn=81f162011d4e6445f474566b2e76557d&scene=21#wechat_redirect)。

最近收到厂家寄的一只新键盘——MelGeek O2，从外观设计到键帽布局都处处体现为 Mac 量身打造。现在它已经是我的 Mac 主力键盘了。

先看颜值
----

这只键盘的外观延续了苹果经典圆润 R 角透明边框设计，如果你对苹果当年的 iMac G4 有所了解，应该对这种有一点复古，又有一点未来感的设计风格不陌生。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/B30A5162-EDEB-4CF5-9F50-A7C353DC1512.jpg)

MelGeek O2

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/61267044-4A0B-4F78-884B-48A95AD3C0E4.jpg)

iMac G4

按键布局尽可能跟原生键盘一样
--------------

它的键帽布局都是尽可能按照苹果原生键盘来排布的，比如最上面一排的 F 区，功能跟苹果原生键盘完全一致。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/B59ACADF-7519-4B7F-8E68-7A83597DE97B.jpg)

有些第三方键盘的 F 区也匹配了苹果原生键盘，但匹配的是老版本。比如下面这张图上的 Keychron K3，留意 F4、F5、F6 这几个键能看出区别。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/929BC831-57AD-4983-9C30-29E40D4805EF.jpg)

MelGeek O2（上）和Keychron K3 Max（下）

MelGeek O2 甚至为了让左上角 Esc 键的比例跟原生键盘一致，也设计得宽一些。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/1CDFA0EA-0848-40D6-8F74-F71582354FE6.jpg)

左下角是咱们 Mac 用户很熟悉的 control、option、command 三个键。出厂就是这三个，不需扣下来替换增补帽，也不需要软件设置改键位。 作为 Mac 用户，这次终于有一种被第三方键盘宠爱的感觉。

当然了，如果你想给 Windows PC 用，它也附赠了适配 Windows 的增补键帽，替换好之后，按 Fn + S 就可以切换成 Windows 系统模式。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/182F1624-FB94-4475-9567-FDDB0C6B3E50.jpg)

可能有对 Mac 比较熟悉的朋友会说，它怎么没有原生键盘左下角的“地球键”？因为地球键苹果是有专利的，任何第三方键盘都不能复刻这个按键功能。就像 Touch ID 也并没有开放给第三方，想用指纹验证的还是只能选择原生键盘。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/B889D905-785B-48EC-AD0F-B16852230DC0.jpg)

总体来说，它算得上是一只 Mac 用户开箱即用，可以迅速习惯的三方键盘。

想怎么连，就怎么连
---------

MelGeek O2 是一款三模键盘，也就是说它支持*有线、蓝牙和 2.4Ghz*三种连接模式。

用 2.4Ghz 接收器很适合搭配像 Mac mini、Mac Studio 这样自带 USB-A 口的 Mac 主机，既能保持桌面尽可能简洁，又能获得非常低的延迟。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/79EC7065-2489-483F-B70B-246F05A0E924.jpg)

说到延迟，O2 使用它们自研 O₂Link 传输协议，输入延迟超低，有线和 2.4Ghz 连接能达到大约 1ms，蓝牙连接也能达到 2 - 3ms，都低于行业的平均值。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/26CC48E6-7857-45AA-A292-8074660101B0.jpg)

MelGeek自己开发的Hive驱动软件

Mac 笔记本用蓝牙模式更方便，也不会占用 MacBook 宝贵的接口。对了， *O2 支持最多配对 8 个 蓝牙设备*，按住 Fn 加上数字 1到 8 就可以快速切换连接的设备。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/B0C221E5-0A85-4723-BB2E-495639841E79.gif)

关于电池续航，官方数据是*低功耗模式下可以用半年*。用官方的 Hive 驱动软件可以设置键盘的模式。如果想要极致续航，可以选 2.4G 省电模式或蓝牙连接。

敲击手感
----

手感也是选一款键盘非常重要的考量。MelGeek O2 用的是凯华独家定制的矮轴，采用软弹的 GASKET 结构，敲起来轻盈干脆，声音也没有那么吵，手感自然是比苹果妙控键盘好的多得多。

其他一些细节
------

键盘的底下也做了一些巧思的设计。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/F71BAE2C-F768-42EE-B052-80E001AB623B.jpg)

这两块蓝色方形其实是可拆卸脚撑，磁吸上去的，按一下就可以翘起来取出，然后把它“啪”的一下吸到旁边的蓝色脚垫上，就可以增加键盘的倾斜角度。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/E81B466C-3F26-4BC5-8722-161BC128B7A2.gif)

这款键盘带 RGB 背光灯，适合晚上在黑暗环境下打字，我最喜欢浅蓝光，亮度调到适中，有一种冰蓝的感觉。但我更喜欢它两侧的这个小小氛围灯，可以常亮，也可以作为呼吸灯，它没有那么炫酷，但我喜欢这种小心机。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/CAB96E4A-EC60-4A0B-A5DF-1325BB9FBAAE.jpg)

它有两颗不一样颜色的键帽，左上角的橙色 ESC 键和蓝色的回车键，视觉上没有那么沉闷，多了一些跳跃感。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/8FFD3066-2072-4A12-B0E0-B89A1DFF4B34.jpg)

或许有些朋友会更想要纯白色设计，搭配 Mac 的简约风格。不知道你会更喜欢哪一种呢？

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/AB97586B-3538-4A5B-8C6C-DFDB08AB1C8D.jpg)

白色 ESC、Enter按键为非附赠键帽

除了适配了 Mac 原生键位，作为一款客制化键盘，也可以通过他们自家的驱动软件，灵活修改每一个按键的功能，以及灯光效果。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/A5F2F5CF-6DCA-461D-A1BC-433CF90B3D83.jpg)

MelGeek？Mel + Geek
------------------

我以前从来没注意过 MelGeek 这个品牌，查了一下原来是来自深圳，研发团队有来自大疆、华为这些大厂的，他们团队在差不多十年前就开始设计研发客制化键盘。

MelGeek 是 Mel + Geek 这两个词的组合。Mel 代表感性、美学，Geek 就是理性、专业。MelGeek O2 也确实是一只集设计美感和专业性于一身的键盘。

![](.evernote-content/EF2D04B6-5949-41B2-AB1F-B01EA56C4D04/1B620374-B8BB-41CF-9BD3-D678B144BCB7.jpg)

如果你也是 Mac 重度使用者，并且想找一只 Mac 友好、手感不错的第三方键盘，相信你也会喜欢这只 MelGeek O2 三模矮轴机械键盘。
