印象笔记 Windows 版重磅更新，让你的工作空间更全能、更强大。

1、全新印象笔记•剪藏 IE 版
----------------

重新设计的剪藏 IE 版更小巧精炼，功能却比以前翻倍，让你在电脑上调研时，快速把重要的网页保存到印象笔记。(安装 Windows 最新版，会自动在 IE 浏览器安装剪藏插件。)

除了直接保存网页外，你还可以用剪藏来截屏和标注保存的内容。 剪藏后，为笔记设置提醒，到期时就会收到邮件，提醒你跟进笔记里的事项。剪藏的机票信息、酒店订单都能这样管理。

![](.evernote-content/EDEE826F-9A98-4DF3-BA5C-07853E765765/8796120C-18E7-4F90-9389-56715F0A73E9.jpg)

不要忘了，你可以随时在印象笔记里搜索剪藏的内容，图片内的文字都能搜得到。

使用时，记得要右键单击浏览器顶部，勾选“命令栏”显示命令栏，才能用剪藏 IE 最新版~

注意：尽管剪藏 IE 最新版支持大部分版本的 IE 浏览器，截屏与标注功能目前仅支持 IE 10 和 11。

2、标注 PDF，重点一目了然
---------------

收到同事发来的 PDF 文档，尤其是信息量较大的那种，是否要花很多时间苦苦寻找里面的重点？

印象笔记帮你在一处保存 PDF 和其他工作要用的文档，里面的内容统统可以搜索到。

这次 Windows 版更新也支持 PDF 标注了，使用高亮、箭头、图形、图章和注释迅速标出重点，或是在上面给反馈，然后分享给同事，沟通直观又高效。

![](.evernote-content/EDEE826F-9A98-4DF3-BA5C-07853E765765/57BEE0D1-96E8-47BF-A433-212861E2B4FB.jpg)

标注后，文档第一页会自动生成“标注一览”，方便自己和他人一眼看清所有重点。

注意：高级帐户和企业版用户可以无限量使用 PDF 标注功能。但！是！免费用户不要担心，现在每位用户有 10 次体验机会，只要点击笔记界面里一个像“圈起来的 A” 的图标，就可以开始标注 PDF ~

值得一提的是，如果你喜欢用笔记本分类整理笔记，一定会喜欢 Windows 最新版更清晰的笔记本列表视图：

![](.evernote-content/EDEE826F-9A98-4DF3-BA5C-07853E765765/213ED0F9-8B2B-45F0-B6E9-AA7557461025.jpg)

企业版用户现在可以直接选择查看企业笔记本或私人笔记本，寻找笔记本更轻松。
