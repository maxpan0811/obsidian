# 用好你的 iPhone，帮你大幅提升搜索效率

## 用好你的 iPhone，帮你大幅提升搜索效率

原创 *2016-03-15* *5key*

![](attachments/28b233a3eab6dc77.jpg)

之前给大家介绍了 Workflow 的划词搜索 ，利用 Workflow 进行快速搜索。这是一个基于划词的操作方法，需要通过复制信息来进行完成。于是有同学问到如果想要主动搜索一个词，如何才能效率更高呢？

这是一个好问题。就像昨天给大家介绍 BrowserHub 的情况一样，在 iOS 上浏览器同样只能默认一个搜索引擎，当需要在不同网站进行搜索的时候我们得首先进入这个网站才能进行操作。

这显然不符合我们尽可能“榨取” iOS 设备价值的观点，好在这个需求效率神器 Workflow 也能帮我们满足。今天就借此话题我们再来了解一下 Workflow 的强大之处。

首先看看成品，在激活这个自定义搜索插件后我们可以选择一个搜索引擎（可自行配置）然后输入搜索关键词，Workflow 会自动将我们带到搜索结果页面。只要我们事先将常用的搜索工具配置好，搜索的效率将会得到大幅的提升。

![](attachments/c4f1800b2f5bbcc5.jpg)

其实 Workflow 自带有一个「Search Web」的操作，只是内置的搜索引擎大多在国内“不存在”，所以我们还是需要自己弄。

这个 Flow 我已经上传到线上，大家可以在底部找到链接直接安装使用。下面我还会以「微信搜索」为例示例一个创建的过程，大家如果有特殊需求可以安装这个方式来自行添加。

整个操作其实并不复杂，主要涉及到四个操作：

1. Choose from Menu
2. Ask for Input
3. Text
4. Open URLs

首先从左侧的列表中找到「Choose form Menu」将其拖入右侧 flow 中，在「Add new item」中添加一个搜索标题（以微信搜索为例）。

完成后再从左侧搜索「Ask for Input」，将其拖到右侧 Flow 的「微信搜索」子模块中。在「Question」中填入搜素前的问题，比如「您的搜索词是？」。

![](attachments/995849ad7fb839f0.jpg)

接下来在左侧列表中搜索找到「Text」，将其拖入右侧 Flow 「Ask for Input」的下方。在其中填入微信网站搜索的 URL，将其中 Query 词后的字段替换成「Input」字段。应用会自动将上一步的搜索词带入拼接成将要执行的搜索 URL。

![](attachments/e65fb63c447682db.jpg)

最后从左侧列表找到「Open URLs」，拖入右侧 Flow 「Text」的下方，整个「微信搜索」的 Workflow 就完成了。点击上方的播放案例测试一下，它依次会询问你要收缩的关键词，然后将关键词带入拼接完整的 URL，并在 Safari 中的打开进入搜索结果页。

![](attachments/42c0bd2f090f927d.jpg)

测试无误后我们可以在「Today Widget」中将其激活，这样我们可以在屏幕任何时刻下滑唤出 Workflow 的自定义搜索，选择搜素引擎进行搜索操作。

![](attachments/8bf582138d76f166.jpg)

原则上来说，只要你能找到网站对应的搜索结果页 URL 的规则，都可以通过 Workflow 来简化搜索。今天介绍的这个只是通过「Menu」操作将多个搜索整合在了一起，提升整体的操作效率。

如果你并不需要太复杂的操作，也可以直接使用我制作好的这个 Workflow。我在这里设定了微信、知乎、百度、淘宝、爱词霸 5个常用搜索。
