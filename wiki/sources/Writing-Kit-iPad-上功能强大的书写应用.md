# Writing Kit：iPad 上功能强大的书写应用

[![](attachments/0bd9883ffd3d1389.jpg)](http://www.ifanr.com/wp-content/uploads/2012/04/w9.jpg?9d7bd4)

在 iPad 已经不缺乏优秀的书写应用，比如可以和电脑进行同步的 iA Writer 或是以简洁优雅获胜的 Byword，然而它们的功能都不及 [Writing Kit](http://getwritingkit.com/) 丰富。

它支持 MarkDown 轻量级标记语法、支持 Dropbox 同步、支持 CloudApp 插入图片、内置浏览器、拥有极为完备的分享功能。此外，它还支持 TextExpander，只要打出相应的短语输入长长的一段文字，能够有效的提升打字的效率。

打开应用，首先显示的是应用的简介，在编辑区域上点击一下，就会弹出虚拟键盘，可以马上输入文字。在虚拟键盘上，有几个功能按键，分别可以插入一级二级三级标题、有序无序列表、给字体加粗、添加引用段落等等相对应的 MarkDown 语法标记。假如拥有 CloudApp 账号，还能够在文章中插入图片。

[![](attachments/34d1a3df07a65b1c.jpg)](http://www.ifanr.com/wp-content/uploads/2012/04/IMG_0062.jpg?9d7bd4)

应用内置了预览功能。当输入一段话之后，点击导航栏右上方的看着像是眼睛的按钮，便能够预览当前的内容，方便检查文章里面的错误。

[![](attachments/d0b9b290bb1c52cb.jpg)](http://www.ifanr.com/wp-content/uploads/2012/04/w2.jpg?9d7bd4)

Writing Kit 声称自己是一个“用于研究的工具（research tool）”，而不仅仅是一个写作工具，是因为它内置了浏览器，写文章时可以直接上网查阅资料。这个浏览器的功能比较完整，能够将网页连接保存到 Delicious、Zootool、Pinboard 等社交化书签服务中去，同时还能够将当前的页面保存到“队列”里面，供以后集中阅读，当然也支持 Instapaper、Read it Later 等待阅服务。而且浏览器下方的还提供了 Text Only 按钮，能够直接优化当前的网页，提供更舒适的阅读体验。

[![](attachments/3f7df8b1b3f656af.png)](http://www.ifanr.com/wp-content/uploads/2012/04/w6.png?9d7bd4)

应用内置的浏览器其实还有一个更重要的功能，那就是能够直接把正在浏览的网页的链接插入到正在编辑的文章中。这样，若文章需要引用某个网页中的资料，就能够直接将它的网页链接插入到文章中，不必来回切换应用，又是复制又是粘贴了。

[![](attachments/d5a0108914ac1123.jpg)](http://www.ifanr.com/wp-content/uploads/2012/04/w11.jpg?9d7bd4)

应用还具备“智能搜索”功能，虽然比不得 Wolfram Alpha 那般智能，但在 iPad 应用当中，已属强大。比如它能够直接对数字进行加减乘除运算、能够换算各种单位、搜索某只股票的表现、查询地理位置坐标等等。

[![](attachments/602e00dfcb3ef888.jpg)](http://www.ifanr.com/wp-content/uploads/2012/04/w.jpg?9d7bd4)

Writing Kit 吸引我目光的，还有它那丰富完整的分享功能。比如文章写完之后，不但能够保存在本地，还能能够直接保存到 Evernote 中。

[![](attachments/97a988deac8ee028.jpg)](http://www.ifanr.com/wp-content/uploads/2012/04/IMG_0069.jpg?9d7bd4)

只不过，应用内置的“自动拼写检查”比较令人讨厌，这个功能会自动把开头输入的第一个英文字母变为大写，在输入中文的时候，时常会被打断。不过，当关闭这个功能之后，中文输入就变得正常起来。

就我个人的体验而言，Writing Kit 是一个比较完美的 iPad 写作应用，也推荐给大家用一下。它的售价是 4.99 美元。如果考虑到它是一个 Universal 版应用，只要购买一次便拥有 iPad 版和 iPhone 版应用，这个价格不算贵。
