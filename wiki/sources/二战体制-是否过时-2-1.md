# 戳上方蓝字[李子旸]关注公众号，不错过精彩文章~

![](attachments/9f1f063f28aa7c20.jpg)

![](attachments/e8a67134e82acd0f.jpg)

战争改变世界的能力，往往令人惊叹不已。一场大战，会让世界面目全非，彻底颠覆战前的“旧秩序”，打造出战后的“新秩序”。第二次世界大战就是典型的例子。1945年结束的二战，确定了今日世界的基本政治格局。这个政治格局，可称之为“二战体制”。

我国是二战体制的最大受益者之一。首先，我国是二战的战胜国。这一举扭转了1840年以来我国对外战争屡战屡败的悲惨局面。第二，二战以后，联合国成为国际秩序的主导和核心，而我国是联合国“五常”之一，在重大国际问题上有一票否决权。这大大改变了我国“半殖民地半封建”的贫弱形象。

因为我国在二战体制中地位主动，得到很多好处。所以，我国在外交上一直努力维护二战体制。

可是，最近三四十年，世界变化很大——我国的变化尤其大。因为这种大变化，二战体制渐渐显露出“不合时宜”之处，也就是越来越不适应、不符合现实情况。对我国来说，也是如此。表面上，二战体制有利于我国，但实际上，二战体制在很多方面阻碍、约束了我国的发展。

二战体制是否过时了呢？

二战体制的要素一：领土神圣不可侵犯

二战以前，世界是个弱肉强食的丛林。“列强”凭借实力，到处扩张领土和势力范围。这种互相争夺激烈到了一定程度，先是爆发一战，然后是更惨烈的二战。

两次世界大战打下来，“领土神圣不可侵犯”成为全世界的共识。“列强”的手脚被束缚住了，不再能随意入侵其他国家。如果有人违反，联合国就发动各国共同阻止“侵略者”。

这个变化看上去很好，好得很。世界因此保持了长期的整体和平——这都快一百年了。

但凡事有利就有弊。“领土神圣不可侵犯”固然制止了“列强”肆意欺负小国弱国，但却也阻止了人类的政治整合，刺激和催生了政治分裂甚至碎片化。一些根本不应该成为国家的地方，堂而皇之地成为国家。

天下大势，合久必分，分久必合。

18、19世纪时，世界的主旋律是“合”。德意志诸邦整合成为现代德国。意大利曾经只是个地理名词，就和今天的中东、拉美一样。经过统一战争，意大利成为统一的民族国家。美国的南北战争，结局也是促进和强化了美国的统一。当然，还有英法俄等殖民帝国。殖民主义很不好，但客观上，殖民主义把亚非拉很多分散的地方，初步整合为单一政治体。

二战以后，世界的主旋律变成“分”。第一波，主要是被剥削、被压迫的殖民地国家实现独立解放。殖民帝国纷纷解体。这是好事。冷战以后，“分”渐渐离谱。一些“蕞尔小国”，根本没有条件政治独立，就因为二战体制的“领土神圣不可侵犯”，纷纷成了独立国家。

这些所谓“独立国家”，在很多方面都不足以成为一个国家，比如，完全没有自保能力，只能依靠美国“世界警察”的保护；再有，国家过小，经济上势必依赖大国，实际上成为大国的附庸。这就是“经济区”和“政治区”的不一致乃至内部扭曲。这不但造成很多不必要的纠纷和冲突，而且阻碍了当地的发展。

在“领土神圣不可侵犯”的原则下，“小国林立”虽然有很多不利之处，但没人敢提出质疑，更没人敢“强行改变现状”。随着时间的推移，世界“碎片化”的弊端日益显露。人类整合的历史大势，被迟滞甚至逆转了。

“领土神圣不可侵犯”的另一个问题是：这个“神圣原则”已经和现实高度不符，已经成了自欺欺人。

曾经有一段时期，国界的划分，和自然地理、经济格局等等大致相吻合。山这边，是甲国，山那边，是乙国。海这边，是甲国，海那边，是乙国。这种国界，比较自然，比较稳固，确实起到了“划分你我”的作用。对这种国界，“领土神圣不可侵犯”比较适用。

“蕞尔小国”纷纷独立建国，让越来越多的国界不再是自然而然的“划分”，而是人为的强行分割。这样一来，所谓“领土神圣不可侵犯”就显得无聊甚至滑稽了。

比如，对互联网的管理，小国几乎完全无能为力。国界可拦不住无线电波。结果就是，互联网的收益被网络技术强大的大国拿走，而各种负面影响却留给了小国。

再比如，某些环境污染事件，国界同样拦不住。某国土壤沙化严重，动辄刮起沙尘暴，吹到邻近国家。边防军再厉害，也拦不住漂浮的沙尘。

更重要的是安全利益。现在，“长程”武器越来越多。某国把长程武器部署到边境，就可以轻易打击邻国。邻国的安全利益受到直接威胁。

俄乌战争打起来以后，我国在外交场合提出“合理的安全关切”，就是这个意思。乌克兰一心加入北约，可北约很敌视俄罗斯。如果俄罗斯放任乌克兰加入北约，那么北约就可以把各种武器直接部署到俄罗斯边境线，且距离莫斯科很近。

因此，俄罗斯进攻乌克兰，表面上看是“侵略”，违反了“领土神圣不可侵犯”原则。其实，俄罗斯是在主动防御。俄罗斯对自身的安全关切，是合理的。真正的侵略者，是北约和乌克兰。可是，俄罗斯合理的安全关切，却和二战体系的“领土神圣不可侵犯”原则相冲突。

未来，我国统一台湾时，如果被迫使用武力，也必将被人攻击为“侵略者”。二战体系客观上阻碍了我国的统一事业。

有趣的是，海洋法和民法，其实都早就不像二战体制的“领土神圣不可侵犯”那样受到“一条线”的束缚。海洋法规定，各国有领海，但同时还规定各国有专属经济区、防空识别区、大陆架等等“扩展”区域，权益各不相同。各国在海洋上的权益划分，不是僵硬地划一条绝对的钱。线以内，就合法。线以外，就非法。如果是这样“一条线”僵硬划分，很多海洋争议就无法解决。

民法中也有“相邻权”的规定，两位邻居，并不是严格按照家门内外来划分的。你在家里并不可以任意做事。比如音响声音开太大，吵到邻居。邻居有权要求你降低音量、停止侵害。也就是说，并不是你在自己家里，就有“神圣不可侵犯的主权”。

从这个角度来看，二战体制确立的“领土神圣不可侵犯”，就很僵硬很笨拙了，越来越不符合现实，越来越不能解决现实问题。

二战体制的弊端，还有其他。篇幅所限，今天写到这里，下篇继续。
