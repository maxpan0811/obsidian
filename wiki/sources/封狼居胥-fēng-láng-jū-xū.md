# 封狼居胥[fēng láng jū xū]

#

#### 百科名片

封狼居胥指西汉大将[霍去病](http://baike.baidu.com/view/3129.htm)登狼居胥山筑坛祭天以告成功之事，[辛弃疾](http://baike.baidu.com/view/2982.htm)借此事言当时南宋近事，指宋孝宗隆兴元年（1163年）[张浚](http://baike.baidu.com/view/81900.htm)北伐，在符离（现安徽省宿县符离集）兵败的事。
