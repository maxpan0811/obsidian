# AirPort Express 802.11n (第 1 代) - 技术规格

---

AirPort Express 802.11n (第 1 代) - 技术规格
======================================

**尺寸与重量**

* 尺寸：94.01 毫米（3.7 英寸） x 75.00 毫米（2.95 英寸） x 28.5 毫米（1.12 英寸）
* 重量：188 克（6.7 盎司）1

**无线协议**

* 802.11n 草案规范2
* 802.11a/b/g

**兼容性**

* 与 Wi-Fi 认证的 802.11a、 802.11b 和 802.11g 以及 802.11n 草案2.0 的 Mac 电脑、基于 Windows 的 PC 电脑互通，同时兼容其他的 Wi-Fi 设备
* NAT、DHCP、PPPoE、VPN Passthrough （IPSec、PPTP 和 L2TP）、DNS Proxy、SNMP 和 IPv6 (6 到 4 及手动信道)

**安全性**

* Wi-Fi Protected Access™ (WPA/WPA2) 3
* 可配置 40 位或 128 位加密的无线网络安全防护(WEP)
* MAC 地址过滤
* NAT 防火墙
* 支持 RADIUS 认证
* 802.1X, PEAP, LEAP, TTLS, TLS, FAST
* 基于时间的访问控制

**附带软件和配件**

* 用于 Mac 和 Windows 的 AirPort Utility
* 用于 Mac 的 802.11n Enabler
* 用于 Windows 的 Bonjour
* 印刷和电子文档

**接口**

* 802.11n 无线接口
* 10/100BASE-T 广域以太网端口 (RJ-45 接口），连接 DSL 、线缆调制解调器或以太网
* USB 接口用来连接 USB 打印机 5
* 3.5 毫米迷你音频接口用于输出模拟或光纤数字声音 4
* 内置电源供应接口

**电源供应和环境需求**

* 100-240V 交流电，50-60Hz，输入电流：0.2 Amps
* 操作温度：0° 至 35° C（32° 至 95° F）
* 储存温度：-25° 至 60° C（-13° 至 140° F）
* 操作相对湿度：20% 至 90%，非冷凝状态
* 最大操作高度：3000 米 （10,000 英尺）
* 最大存储高度：4500 米 （15,000 英尺）

**频宽**

* 2.4GHz 或 5GHz

**无线电输出功率**

* 20 dBm （额定）

**安全规格**

* FCC Part 15 Class B, Canada RSS-210, EN 300-328, EN 301-489, EN 301 893, ARIB STD-T66, RCR STD-T33, AS/NZS 4268: 2003, UL 60950, CSA-C22.2 No. 60950

**频道**

* 美国、加拿大、拉丁美洲和台湾，可用频道 1-11、36-48 和 149-165
* 日本可用频道 1-14 和 36-64
* 欧洲可用频道 1-13 和 36-140
* 澳大利亚、香港和新西兰可用频道 1-13, 36-64 和 149-165

**系统需求**

|  |  |  |
| --- | --- | --- |
|  | **Mac 用户** | **PC 用户** |
| **无线客户端访问** | 配有 AirPort 或 AirPort Extreme 无线网络性能的 Mac | 配有 802.11a/b/g 或 802.11n 2.0 草案规范的无线网卡的 PC |
| **通过 USB 打印机共享打印** | USB 打印机，配有 Mac OS X v10.2.7 或更新版本的 Mac | USB 打印机，配有 Windows XP (SP2) 或 Windows Vista 的 PC；AirPort Extreme CD 中已包括用于 Windows 的 Bonjour |
| **软件** | 安装了 Mac OS X v10.5.7 或更新版本的用户可以:   * 访问 AirPort Express 无线网络 * 设置 AirPort Express 基站 * 使用 AirTunes 播放音乐 * 共享一台 USB 打印机 * 使用 Wi-Fi Protected Access (WPA/WPA2) | * Windows XP 或 Windows Vista * WPA 支持需要 Wi-Fi 认证的 PC 解决方案 |

**延伸服务和技术支持**  
为你电脑提供的 [AppleCare 维护计划](http://www.apple.com/support/products/mac.html)涵盖 AirPort Express 基站。

---

1. 重量会因配置和生产工艺的不同而有变化。
2. AirPort Express 基站基于 IEEE 802.11n 草案规格，与 IEEE 802.11a、IEEE 802.11b 和 IEEE 802.11g 兼容。
3. 对 WPA 的支持需要 Mac OS X 10.3 版或更新版本。使用 Wi-Fi Protected Access (WPA) 会减少网络用户的最大数量。当加入已存在的无线网络时，AirPort Express 仅支持 WPA-Personal。WPA2 需要配备有 AirPort Extreme 无线网卡和 Mac OS v10.3 或更新版本的 Mac 电脑。
4. 需要相容的立体声系统或有源扬声器。
5. 通过 USB 无线打印必须使用兼容的打印机，并在 Mac OS X v10.2.7 或更新版本，或者 Windows XP，Windows Vista 或 Windows 2000 系统下进行。
6. 该产品需经 CCC 认证，经核准后方可发售。

无线互联网访问需要具有无线上网功能的电脑、基站或其它访问接入点，以及互联网连接 （可能需要支付一定费用）。部分ISP 暂不支持 AirPort Express
