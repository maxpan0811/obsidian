“筑梦航天·遨游太空”2021大型航空主题活动
=======================

北京天骄快乐家族 北京天骄快乐家族幼儿运动课程

![](.evernote-content/BBFC156B-E732-477B-A29F-EB654CDD0C79/4329B7D7-0463-4F0B-875E-08E462251C23.gif)

**筑梦航天·遨游太空**

—— 2021大型航天主题活动 ——

![](.evernote-content/BBFC156B-E732-477B-A29F-EB654CDD0C79/88CEF09C-D17F-4127-B989-DB2979A2A191.jpg)

**活动背景**

**北京时间2021年10月16日0时23分，搭载神舟十三号载人飞船的长征二号F遥十三运载火箭，在酒泉卫星发射中心按照预定时间精准点火发射。**约582秒后，神舟十三号载人飞船与火箭成功分离，进入预定轨道，顺利将翟志刚、王亚平、叶光富3名航天员送入太空，飞行乘组状态良好，发射取得圆满成功。我国的航天事业进入了新的篇章，作为教育者的我们有义务让幼儿感受伟大的航天精神，特此举办本次以航天为主题的活动。

![](.evernote-content/BBFC156B-E732-477B-A29F-EB654CDD0C79/0A729FD6-602C-4062-B8DC-E7C6DE6D4B26.jpg)

**活动目标**

本次活动是带孩子了解宇宙的奥妙，做一些以宇宙为题材的游戏，不但在游戏中学习了宇宙的知识，也在游戏中增加了孩子与家长之间的亲密感。

![](.evernote-content/BBFC156B-E732-477B-A29F-EB654CDD0C79/1ED24236-E625-4835-B86E-05B74683826A.jpg)

**活动亮点**

浩瀚的宇宙，往往给予人们神秘的色彩，

一望无际的尽头，

是否还有人们期待的第二个地球呢?

极美的星夜，天上没有一朵浮云，深蓝色的天上，满缀着钻石般的繁星。

宇宙的秘密，还要我们继续挖掘。

**活动流程**

|  |  |
| --- | --- |
| 1 | 主持人开场前主持：（活动背景讲解） |
| 2 | 教师开场舞蹈 |
| 3 | 主持人开场 |

|  |  |
| --- | --- |
| 4 | 小航天员入场（国旗班在前，小航天员在后家长在每个班级后，活动场地落位，家长等候线等待） |
| 5 | 升国旗，奏国歌   所有小宇航员入场完毕。升国旗，奏国歌，行注目礼。 |
| 6 | 园长讲中国航天发展史、未来展望、以及现在对幼儿的期许 |

|  |  |
| --- | --- |
| 7 | 教师诗朗诵《我有一个航天梦》 |
| 8 | 航天主题体能游戏 |
| 9 | 合影留念 |

![](.evernote-content/BBFC156B-E732-477B-A29F-EB654CDD0C79/6F585AF1-1B8F-428A-AF14-10757E76472E.jpg)

**活动准备**

|  |  |
| --- | --- |
| 1 | 假设活动9点整开始 幼儿园提前1.5——2个小时到达 |
| 2 | 舞台、及背景布 |
| 3 | 各班级：负责签到班级人员、组织家长小朋友到各自班级的候场区排队 |

|  |  |
| --- | --- |
| 4 | 活动开始之前半小时、各班级提前通知家长带着幼儿上厕所。 |
| 5 | 主班配班及保育员在活动中随机应变。 |
| 6 | 活动结束：主持人宣布结束、合影留念 |

![](.evernote-content/BBFC156B-E732-477B-A29F-EB654CDD0C79/BC0037C0-77EA-4E0B-9288-8C0AC50E80AE.jpg)

**活动保障**

|  |  |
| --- | --- |
| 1 | **教练系统：**  师生配比1:30；主训教练均经严格选拔及培训，全场把控，反复演练；应急团队：专人负责，随时在线。 |
| 2 | **组织管理系统：**  严格按照方案流程组织活动,每小队选举责任心强的人担任队长、副队长以及安全员，明确责任义务以达到有效管理； |
| 3 | **装备品质系统：**  活动配备的所有装备，都是经过国际品质体系认证过的名牌产品，以保证活动安全万无一失； |

|  |  |
| --- | --- |
| 4 | **后勤保障系统：**  为每位学员购买人身意外及医疗保险，备有急救箱应付突发事件，并配备专业后勤教练精心服务。 |

**温馨提示**

1.本活动经过实地详尽考察，独家策划活动环节，并对活动项目作出安全评估；

2.建议给孩子穿宽松的秋季衣物、运动鞋；

3.随身物品：背包/单肩包、水杯、个人专用药品（如有医疗史，报名时请提前告知）；

4.请勿携带大量现金、贵重物品、手机等电子产品；

5.活动中请听从老师的安排，严禁私自离队，参观时请爱护公物，未经允许不得随意触碰。
