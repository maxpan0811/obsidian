# Commander One: 最好的双栏文件管理器 | 领客专栏·Mac 玩儿法

---

Commander One: 最好的双栏文件管理器 | 领客专栏·Mac 玩儿法
----------------------------------------

原创 *2016-01-07* *应用推荐第一媒体*

Commander One 是 Eltima 家（Folx）在今年采用 Swift 后重新开发的一款双栏文件管理软件，重新设计开发后无论是功能还是性能都有了进一步提高，个人认为 Commander One 将成为用户桌面上继 Path Finder、TotalFinder、Fileloupe 等大牌文件管理工具后的又一选择。

无论是在官网下载或是 MAS，Commander One 都提供了免费版与 Pro 版，免费版的功能可以说是厂商的表现实在吝啬之极，基本上偿不到「甜头」，有了 Pro 版，你才能享受以下功能：

* FTP 管理
* Dropbox、Google Drive、外接磁盘管理（iOS 设备、MTP、USB Flash 等设备）
* 进程管理
* Amazon S3
* 窗口内调用终端模拟器（热键 Ctrl+O）
* 压缩与解压缩
* 主题

Commander One 在界面上给予了用户很大自由度，可以选择三种文件排列样式（列表、带详细信息列表以及缩略图模式），在窗口顶部提供了两组功能按钮，分别是「隐藏文件显示」、「文件简介查看」、「文件预览」、「文件查找」、「文件压缩」、「FTP 连接管理」以及「Amazon S3 连接管理」。

在后台设置里你可以为软件选择不同样式的主题配色，同时也可单独针对背景、文本、光标等元素的颜色进行自定义。  
Commander One 的基本功能就是方便用户对两个目录下的文件进行复制、移动、比对等操作。

因此，开发者提供了不计数量的创建操作标签（Tab）的功能，这样用户可以在左右两栏开启多个目录。

在窗口底部，提供了我最喜爱的快速处理工具栏，分别是「查看」、「编辑」、「复制」、「移动」、「新文件夹」与「删除」这六个操作，同时也可用 F 键直接触发。

上面我们介绍了 Commander One 的一级功能，下面就来谈谈二级功能，我也更愿意叫「细化功能」，比如上图中，我要压缩文件，可以选择 Zip、7z 等 7 种压缩包格式，同时也可对压缩质量进行选择。

另外大家也能看到熟悉的 FTP & Amazon S3 连接管理功能，而对于文件移动来说则采用了直接输入目录路径的方法，但这种操作对于小白用户来说还是有些门槛。

软件细化功能中的精华就是文件搜索了，只有搜索功能做到位，这款软件才能实现「File Manager」所应有的价值。搜索功能分「文件搜索」与「文本搜索」两种，都可用正则表达式，区分大小写肯定是必备，但这款软件还能在压缩包中直接搜索文件，还真让我有些意外呢。

总的来说，Commander One 是一款非常有竞争力的文件管理软件，我可以将 Dropbox、U 盘挂载到上面进行实时管理。

但美中不足之处也有，像文件管理界面中缩略图模式下，图标缩略图尺寸不可随意调整，Pro 版自选 UI 主题糟糕的配色设计，都是需要厂商改进的地方。

相信随着功能不断迭代，这款软件在未来版本中会给用户更多的惊喜！
