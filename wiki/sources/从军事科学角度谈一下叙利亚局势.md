# 从军事科学角度谈一下叙利亚局势

速读摘要

之前阻止反政府武装并稳定战局后，叙利亚并没有犒赏三军，而是解散了大量政府武装。但这次反政府武装经过长期准备，在正面交火后，并没有一直纠缠在正面，而是利用摩托车、民用车辆等，沿着高速公路等交通线大纵深机动到政府军防线的背后，让刚开始还算是在英勇抵抗的政府军士气瞬间就崩溃了。民用车辆的速度很容易超过现在大多数民用无人机，因此这次反政府武装可以用"民用无人机加常规炸弹"去轻松对付政府军的坦克等重装备，而他们自己采用的民用摩托和其他车辆机动能力却是民用无人机追不上的。

**一、军事问题最终毕竟还是要从军事角度来看待**

这次叙利亚局势本身其实并不出乎人们的意料之外，因为这地方仗打得太多了，再多打一些稀松平常。太出人们意料之外的只是这次怎么反政府军如此之快地就一路狂飙突进，从开战到打垮政府军仅用了10天时间，并推翻巴沙尔·阿萨德政权。

人们给出了很多解释，例如：

- 俄乌战争导致俄罗斯军力大量撤出叙利亚，极大削弱了支持政府军的力量。
- 之前阻止反政府武装并稳定战局后，叙利亚并没有犒赏三军，而是解散了大量政府武装。
- 经济崩溃，没有军费支持。
- 反政府武装选择以色列与真主党停火的同时发起进攻，这样真主党的支持会很弱，有利于减少政府军一方的支持。
- 其他

这些当然都很有道理，并且也是非常重要的。但仅仅靠这些来解释并不足以让人满意。例如，中国解放战争结束后，也开始大量复员军人，因为当时国家很穷、打了那么多年仗，已经准备开始搞建设了。所以复员了大量军人大家也能理解，但并没因此影响后来抗美援朝时的士气。叙利亚长年内战，民不聊生，战后军人复员也是正常。开战后阿萨德马上提升军人的工资也作了弥补努力。

俄罗斯军力撤出的确是一个重要原因，但战争发起后俄罗斯也迅速重新调回军事人员，尤其大量战机可以快速调动，并且也对反政府武装造成了极大杀伤。

尤其关键的是：叙利亚政府军刚开始还是与反政府武装进行了激烈交战的，问题是反政府武装推进得太快了，导致了政府军士气迅速崩溃。

军事问题，毕竟还是要从军事科学的角度来深入进行讨论，只有外行的媒体才会从外部经济和外援等因素来发一下泛泛的议论。

我们可以做一个假设，如果俄罗斯没有撤出军事人员，政府军人员工资待遇正常，是否就能改变这一次的战局。我给出的答案是：可能还是这个战果，虽然反政府武装损失可能会更大一些。为什么？因为最根本的原因是这一次反政府武装所采用的正确的战略战术。因为此次反政府武装从反攻到成功的时间太短，以至于人们都没有时间来认真消化此次战争中从单纯军事角度的很多经验、教益和教训。这就是：

**快速机动能力，大穿插战术**。

这也再次验证我在《超越战争论》中所提出的一种武器运用模式：“民用运载工具加常规战斗部”模式。

**二、快速机动**

只要简单分析一下这次反政府武装的进军过程，会马上联想到一大堆历史上的战例以及战略战术：二战时德军的大钳形攻势，苏军的大纵深理论。中国人民解放军经常采用的穿插战术。巴顿经常采用的正面牵制、利用坦克机动迂回敌后方补给线的战法。

之前反政府武装之所以与政府军和俄罗斯等援军打成焦灼状态，是以城市巷战为主的战略战术。这种战法必然结果是代价高昂，牺牲巨大，进展极为缓慢。双方都是这种战法，最终结果就取决于谁更耐受消耗。而由于政府军得到俄罗斯与伊朗就近的大力支持，所以，反政府武装经不起与政府军的长期消耗而失败。

但这次反政府武装经过长期准备，在正面交火后，并没有一直纠缠在正面，而是利用摩托车、民用车辆等，沿着高速公路等交通线大纵深机动到政府军防线的背后，让刚开始还算是在英勇抵抗的政府军士气瞬间就崩溃了。这在历史上是有无数战例反复验证过的，一旦自己防线背后、尤其是补给线出现敌军并被切断退路，除了极少数最为善战的军队外，绝大多数军队是士气瞬间就彻底崩溃了。

反政府武装就是这样开着民用车辆一路狂奔，从北向南一直打到大马士革。在这个过程中整个政府军的士气兵败如山倒，迅速地土崩瓦解，根本就没来得及消化和理解反政府军的新战术并寻找破解之法，整个政府军控制的区域就全被反政府武装攻下了。

**三、坦克快过时了**

最近我在发表一个系列的文章，讨论美国航母对中国已经越来越无用的问题（参见[对**中国来说，美国航母无用论已经成为事实（三）**](https://mp.weixin.qq.com/s?__biz=MzA5ODYxMDE2Mg==&mid=2650507228&idx=1&sn=41fc38ded363d758289ebd38a6afcea5&scene=21#wechat_redirect)，[**对中国来说，美国航母无用论已经成为事实（二）**](https://mp.weixin.qq.com/s?__biz=MzA5ODYxMDE2Mg==&mid=2650507193&idx=1&sn=46e5564bccfa97528825dce6ee8b3cc9&scene=21#wechat_redirect)，[**对中国来说，美国航母无用论已经成为事实（一）**](https://mp.weixin.qq.com/s?__biz=MzA5ODYxMDE2Mg==&mid=2650507189&idx=1&sn=b3e2fc942f56fed452acad0cf6e67764&scene=21#wechat_redirect)）。其实，由于最近一系列军事变革，坦克也越来越无用了。

![](attachments/ac4bbadd0c57bb52.png)

《超越战争论》中提出的通用武器理论

人们经常是遇到一个案例或新武器，就仅仅为解释这个案例或新武器来建立理论，这种理论是不具有通用性的。上面是我在《超越战争论》中给出的，可以应用于一切战争形态和技术时代的武器理论。一切武器本质上是由战斗系统和运载系统组成，人们常常是以运载系统来给军队分类。但战争中最后起作用的是战斗系统。我们来分析一下坦克的发展历史以及它为什么在今天遭遇越来越大的困境。

要全面理解现代战争，请参见我集古今中外军事战略理论以及几千个战例研究之大成，以完全科学化的工具完成的当代创新军事理论《超越战争论》，使军事理论研究上升到空前高度。该书在众多主流军校、军事单位及企业等以“毛泽东军事思想的数学原理”为题授课，好评如潮。过去、现在和未来发生的一切战争都可以从中找到最为精确和完美的答案。

![](attachments/e26def956e1e34af.png)

坦克是在一战中因应打破堑壕战的泥潭而出现的。堑壕是热兵器出现以后，最有效获得防御能力的手段。直到今天，看似简单的堑壕防御能力都是非常优异的。一个极为便宜的沙袋装上土堆在堑壕前面，其防御能力很容易达到一米均质装甲。如果一袋不够就再多加一袋。一旦形成这样的堑壕，进攻一方就会遭遇太大的伤亡却难以有效突破敌方的防线。坦克简单来说就是一个把堑壕装在履带上可以前进至敌方堑壕的装备，从而可以抵消掉敌方堑壕的防御能力。

在抗美援朝战争中，志愿军简单地通过在有足够坡度、坦克上不去的小山包上建立防线，又抵消掉了坦克的优势（参见“[《长津湖》及抗美援朝的军事科学解读](https://mp.weixin.qq.com/s?__biz=MzA5ODYxMDE2Mg==&mid=2650504549&idx=2&sn=0770daa753f28783bd52e3be0ed564c9&scene=21#wechat_redirect)”）。

坦克的另一大优势是机动能力。二战中德军、巴顿将军以及后期的苏联红军都是充分利用坦克的机动能力迂回到敌后，从而实现有效突防的。

人民解放军特别善长于即使靠人的两腿也能实现超高的机动能力进行穿插迂回。只要迂回到敌人后方，切断敌方的补给线，敌方的防线就很容易崩溃。

但是最近发生的俄乌战争以及中东的战争，无人机的出现让坦克的优势近乎消失殆尽。

坦克的机动能力不如无人机。因为坦克过于笨重，最高时速很难超过70公里。大部分情况下只能在30-50公里/小时行驶，尤其在城市环境更慢。而现在大量使用的民用无人机速度一般都可达到50公里/小时。所以，无人机追上并击毁坦克是很轻松的。因此，在最近发生的几场战争中，无人机几乎可以轻松吊打一切坦克。

此次叙利亚战争中，反政府武装同样大量使用了无人机，使处于一线的政府军重装备迅速被摧毁，这也是导致政府军士气快速崩溃的关键原因所在。这种民用无人机加装常规战斗部的结合所带来的威力，已经一再地被近期的多个战争所充分证明。

**四、民用运输工具加常规战斗部**

我在《超越战争论》第8章中系统讨论过武器系统的成本分析。目前武器系统的发展是成本倒置的：就是因为太过考虑防御能力和隐身能力，把过多的、极度高昂的成本消耗在运载系统上。在“第四节 寓军于民”中就提出过“民用运输工具加高效战斗系统”的理论，并且预测了今天民用无人机加常规战斗部的极大威力。

因为专门军用的坦克考虑到防御能力，装甲非常厚，而且还装备了主动反应装甲等技术，这导致重量相比民用车辆急剧增加，由此发动机和油箱都得极大增加。因此，坦克极为笨重，并且速度很难增加。油耗极高，200升/百公里是常态，一辆坦克消耗的燃油相当于20-30辆轿车。这又使后勤补给极为困难。

但是，民用车辆在高速公路上达到120公里时速甚至更高是稀松平常的。这使民用车辆的机动能力远远超过坦克。同时，民用车辆的速度很容易超过现在大多数民用无人机，因此这次反政府武装可以用“民用无人机加常规炸弹”去轻松对付政府军的坦克等重装备，而他们自己采用的民用摩托和其他车辆机动能力却是民用无人机追不上的。

随着民用车辆智能化程度的提升，军用车辆的演进和进步速度肯定是远远跟不上的。想象一下，类似特斯拉皮卡这样已经具有防弹功能，未来高度智能化，可以用太阳能充电，由此可以使后勤支持需求极大地降低。这样的车辆在战场上相对极度笨重的，需要庞大油料后勤支持的车辆，谁优谁劣？

即使在今天，民用运载系统成本极低、加上军用战斗部，其杀伤能力完全一样。油耗极少使得补给远比坦克和运兵车容易。车辆的损耗补充也很容易。成本远远比专业的军用运载系统低几个数量级。

以下是摘自我在《超越战争论》中讨论的案例和相关论述：

有一场战争很能体现“民用运载工具加战斗系统”模式的威力，这就是“丰田战争”。乍得军队有400多辆配备反坦克导弹的越野车、皮卡车，它们击败了利比亚全部为军事专用的装甲部队。最著名的一次战斗，1987年1月2日，乍得军队和利比亚军队在法达地区发生交火，乍得军装备大量的民用丰田以及陆虎吉普，车上配备美法提供的米兰反坦克导弹及机关炮，在短暂的冲突中， 784名利比亚士兵阵亡，92辆坦克和33辆步兵战车被毁，81人被俘，13辆坦克和18辆战车被缴获，而乍得军仅18人阵亡，三辆丰田车被毁。

因战斗部击毁能力的增强，一辆价格1000万人民币的高成本专用军事车辆被导弹击中，与一辆成本20万的极低成本民用车辆被相同的导弹击中，结果都是被击毁。但它们装载了相同作战单元后，其进攻能力却是一样的。甚至于，因为装备极便宜的民用运载工具省下了大量的钱，就可以花更多的钱用于战斗部上，因此其击毁能力反倒更强。既然如此，为什么不用1000万生产50辆民用作战运载系统工具，而要生产一辆军事专用运载工具呢？并且，这50辆民用运载工具在平时完全可用于一般生产过程而产生经济效益，但那个1000万的军事专用车辆在平时不会产生任何经济效益。根据战争循环因果序列，50个作战单元形成的击毁能力，是1个作战单元的2500倍！或者可以只生产25台运载工具，但把省下的500万用于成倍增强真正有最终战术价值的战斗部。

一般来说，每辆军用坦克价值为300万-1000万美元，而一辆民用车辆的价值一般都在2、3万美元水平。即使去掉坦克的武器系统，两者成本依然相差100倍以上。以100辆民用越野车加装击毁能力更强的导弹和火箭弹组成的战斗部队，与现役和未来几十年内最先进的任何1辆专用坦克相比，前者的作战效能绝对都是后者的千倍以上。

能够最快速地消灭敌人，就是最好的装甲防护。

现在的民用无人机成本更是与坦克相差上万倍。无人机在目前的战争中大出风头，更确切的说是“民用无人机加常规战斗部”大出风头。但很多人，包括很多专业的军事家还没看清楚当前时代的巨大变革。可以说，现在民用无人机加常规战斗部的大杀四方，就是对《超越战争论》理论的完美展示和应用。这次反政府武装更是把摩托车、皮卡、普通轿车等民用运载系统加常规战斗部的模式进行了全方位的展示。

如果有一百万架民用无人机携带两百万枚普通的60毫米、82毫米、120毫米迫击炮弹，它所具有的杀伤力是何等恐怖？现在的军事家们可能还没想清楚这个问题。它们的全部成本加起来仅仅是一艘航母
