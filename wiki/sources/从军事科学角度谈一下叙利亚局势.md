# 从军事科学角度谈一下叙利亚局势

速读摘要
之前阻止反政府武装并稳定战局后，叙利亚并没有犒赏三军，而是解散了大量政府武装。但这次反政府武装经过长期准备，在正面交火后，并没有一直纠缠在正面，而是利用摩托车、民用车辆等，沿着高速公路等交通线大纵深机动到政府军防线的背后，让刚开始还算是在英勇抵抗的政府军士气瞬间就崩溃了。民用车辆的速度很容易超过现在大多数民用无人机，因此这次反政府武装可以用"民用无人机加常规炸弹"去轻松对付政府军的坦克等重装备，而他们自己采用的民用摩托和其他车辆机动能力却是民用无人机追不上的。

一、军事问题最终毕竟还是要从军事角度来看待

这次叙利亚局势本身其实并不出乎人们的意料之外，因为这地方仗打得太多了，再多打一些稀松平常。太出人们意料之外的只是这次怎么反政府军如此之快地就一路狂飙突进，从开战到打垮政府军仅用了10天时间，并推翻巴沙尔·阿萨德政权。
人们给出了很多解释，例如：
俄乌战争导致俄罗斯军力大量撤出叙利亚，极大削弱了支持政府军的力量。
之前阻止反政府武装并稳定战局后，叙利亚并没有犒赏三军，而是解散了大量政府武装。
经济崩溃，没有军费支持。
反政府武装选择以色列与真主党停火的同时发起进攻，这样真主党的支持会很弱，有利于减少政府军一方的支持。
其他
这些当然都很有道理，并且也是非常重要的。但仅仅靠这些来解释并不足以让人满意。例如，中国解放战争结束后，也开始大量复员军人，因为当时国家很穷、打了那么多年仗，已经准备开始搞建设了。所以复员了大量军人大家也能理解，但并没因此影响后来抗美援朝时的士气。叙利亚长年内战，民不聊生，战后军人复员也是正常。开战后阿萨德马上提升军人的工资也作了弥补努力。
俄罗斯军力撤出的确是一个重要原因，但战争发起后俄罗斯也迅速重新调回军事人员，尤其大量战机可以快速调动，并且也对反政府武装造成了极大杀伤。
尤其关键的是：叙利亚政府军刚开始还是与反政府武装进行了激烈交战的，问题是反政府武装推进得太快了，导致了政府军士气迅速崩溃。
军事问题，毕竟还是要从军事科学的角度来深入进行讨论，只有外行的媒体才会从外部经济和外援等因素来发一下泛泛的议论。
我们可以做一个假设，如果俄罗斯没有撤出军事人员，政府军人员工资待遇正常，是否就能改变这一次的战局。我给出的答案是：可能还是这个战果，虽然反政府武装损失可能会更大一些。为什么？因为最根本的原因是这一次反政府武装所采用的正确的战略战术。因为此次反政府武装从反攻到成功的时间太短，以至于人们都没有时间来认真消化此次战争中从单纯军事角度的很多经验、教益和教训。这就是：
快速机动能力，大穿插战术。
这也再次验证我在《超越战争论》中所提出的一种武器运用模式：“民用运载工具加常规战斗部”模式。
二、快速机动

...只要简单分析一下这次反政府武装的进军过程，会马上联想到一大堆历史上的战例以及战略战术：二战时德军的大钳形攻势，苏军的大纵深理论。中国人民解放军经常采用的穿插战术。巴顿经常采用的正面牵制、利用坦克机动迂回敌后方补给线的战法。
之前反政府武装之所以与政府军和俄罗斯等援军打成焦灼状态，是以城市巷战为主的战略战术。这种战法必然结果是代价高昂，牺牲巨大，进展极为缓慢。双方都是这种战法，最终结果就取决于谁更耐受消耗。而由于政府军得到俄罗斯与伊朗就近的大力支持，所以，反政府武装经不起与政府军的长期消耗而失败。
