开放的力量—破解GitHub的发展密码
===================

GitHub ：

> 一个共享虚拟主机服务，用于存放使用Git版本控制的软件代码和内容项目。它由Chris Wanstrath、PJ Hyett和Tom Preston-Werne用Ruby on Rails编写而成。

—维基百科
