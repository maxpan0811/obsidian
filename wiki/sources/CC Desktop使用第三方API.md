## 一句话摘要

在官方 Claude Desktop 界面里直接调用其他厂商模型（如 GLM），保留所有界面交互和体验——无需付费订阅，不用考虑 IP，更不用担心封号。

## 操作步骤

1. **启用开发者模式**：完全退出 Claude Desktop → Help → Troubleshooting → Enable Developer Mode
2. **配置第三方 API**：Developer → Configure third-party inference
3. **填写接口信息**：输入适配的接口地址、API Key、模型名称（如 `glm-5-turbo`）
4. **Apply 应用**，即可使用第三方模型

## 支持范围

支持 Anthropic 协议兼容的第三方模型或中转服务。国产模型走 Gateway 模式。
