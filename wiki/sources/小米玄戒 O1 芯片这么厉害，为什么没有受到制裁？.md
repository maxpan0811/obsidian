小米玄戒 O1 芯片这么厉害，为什么没有受到制裁？
=========================

**一句话版**：它既强到能拼苹果与高通，又巧妙地“躲过”美国对**先进计算芯片**的红线——晶体管不到 200 亿、定位手机消费级，暂时不在 BIS 的“黑名单”里。

---

![](.evernote-content/5528BC7A-41D0-4606-81E7-2A82DC901C7E/ADC7D22E-7091-4B87-899A-5EFBFFB26CB5.png)

明天19:00 小米将正式发布这块芯片，在此之前我们看看这块芯片为啥厉害，以及为啥不受美帝的芯片制裁

1 | 玄戒 O1 到底有多强？
----------------

|  |  |
| --- | --- |
| 关键规格 | 数据 |
| 制程 | **TSMC 第二代 3 nm（N3E）** |
| 晶体管 | **≈ 19 0亿个** |
| CPU | 10 核（2×X925 3.9 GHz + 4×A725/X925 3.4 GHz + 2×A720 1.89 GHz + 2×A520 1.8 GHz） |
| GPU | 16 核 Mali-G925 / IMG CXT-48-1536（爆料不一） |
| 基带 | **外挂 MediaTek 5G Modem**  （并未集成到 SoC） |
| Geekbench 6（泄露） | 单核 ≈ 3119， 多核 ≈ 9673 —— 接近 Dimensity 9400，略超骁龙 8 Gen 3 |

仅用四年，小米 **从 0 到 3 nm**，性能追平联发科、逼近高通——这在手机芯片史上几乎是“开挂速度”。

---

2 | 玄戒 O1 的工程信息与成本猜想
--------------------

* • **堆料一眼可见**：

+ • 2 颗超大核 X925 + 16 MB L3；
+ • 3 nm N3E 晶圆价格 ≈ 22 000 美元/片；
+ • 估算晶圆利用率 ≈ 65%，成品良率 ≈ 70%。
+ • 业内人士推算 **单颗裸片成本 ≥ 1700 元**（不含封装与测试）。

* • **自研／外采拆分**

  模块

  来源

  CPU 指令集

  ARM 授权

  CPU/GPU 微架构

  ARM 公版 + Imagination 授权

  NPU/ISP

  小米自研（Surge/IPU 团队延伸）

  基带

  MediaTek T90 系列外挂

  封装

  日月光 / Amkor（推测）

  这更像一次 **“高端拼装 + 局部自研”**：自己写 glue 代码、做 block-level 优化，把外部最好的 IP 和工艺整合起来，先打响品牌。

4 | 玄戒 O1 量产靠谁？
---------------

* • **国内有芯片厂可以帮小米生产这块芯片吗？**
* • **不能**，国内**没这个能力**生产3nm芯片
* • 国内现有的生产芯片最高工艺是：中芯国际的7nm (目前主要给华为等被制裁的厂商用)

+ • **中芯国际**：

- 中芯国际能用到的最先进的光刻机能生产14nm的芯片
- 中芯采用了特殊的多重曝光技术，勉强可以达到7nm的工艺
- 无 EUV，7 nm 仍靠多重曝光；3 nm 至少 2027+。
- （主要给华为等国内厂商生产）

+ • 结论：**短期只能台积电**。

5 | 玄戒 O1 为何暂时“躲过”美国制裁？
-----------------------

### 5.1 制裁规则在卡什么？

BIS 2025 年 1 月的 **《先进计算 IC 中期规则》** 把“3 nm 及以下、面向数据中心”IC 列入 ECCN 3A090.a，并设**三条豁免线**：

1. 1. 晶体管总数 **< 300 亿**；
2. 2. 或（不含 HBM）< 35 亿（2027） / < 40 亿（2029）；
3. 3. 或经 BIS 认可的可靠封测厂／设计商。

### 5.2 玄戒 O1 目前还在“安全区”

|  |  |  |  |
| --- | --- | --- | --- |
| 指标 | 玄戒 O1 | BIS 触发值 | 结论 |
| 晶体管 | **≈ 190 亿** | 300 亿起 | **未超线** |
| HBM | 无 | 一旦含 HBM → 管制 | **规避** |
| 应用场景 | 手机 SoC | 数据中心 / HPC | **消费级** |
| CPU/GPU 峰算力 | < 20 TFLOPS | HPC 控制值远高 | **未触线** |

**一句话**：O1 是一颗高端手机芯片，但不足以被视作“先进计算 GPU/AI 加速器”，因此 **台积电帮助小米量产并不违规**。

### 5.3 未来会不会遭“加码”？

|  |  |
| --- | --- |
| 潜在变动 | 影响 |
| BIS 下调阈值（例如 < 20 亿） | O1 下一代被迫回退 5 nm 或找 Samsung |
| 把所有“3 nm 及以下”一刀切 | 同上，且手机 SOC 也需单独申请许可证 |
| 增列小米实体清单 | 台积电需 **Case-by-case** 许可，交付周期拉长 |

6 | 玄戒 O1 的“自研”成色到底几分？
----------------------

* • **同行自研对比**：

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| 维度 | 小米 O1 | 苹果 A18 | 高通 S8G4 | 华为 Kirin 9010 |
| CPU 微架构 | ARM | **自研** | 过渡**自研** | **自研** |
| GPU | MaliIMG 授权 | **自研** | **自研** | **自研** |
| NPU / ISP | **自研** | **自研** | **自研** | **自研** |
| 基带 | 外挂 MediaTek | **自研**  + 高通授权 | **自研** | **自研** |
| 制程/代工 | 台积电 | 台积电 | 台积电 | **中芯国际** |

**结论**：玄戒 O1 属于“半自研 + 高度整合”路线——在全球分工框架下，这是**现实且高效**的突围方案。

---

7 | 华为 vs. 小米：双路线都值得期待
----------------------

* • **华为**：自建 **全栈 自研**，哪怕制程落后，也能掌控 CPU、NPU、基带；真正的“去美化”。
* • **小米**：借助台积电 + ARM + Imagination 等**可用资源**，快速进入 3 nm 俱乐部；侧重市场迭代速度。

二者 **并非你死我活**：一个填补高端自主全链空白，一个用最快速度把国产品牌拉回“第一梯队”。只要能缩短“被卡脖子”阵线，都是好事。都值得我们尊敬和支持。

---

8 | 写在最后
--------

玄戒 O1 的诞生，是 **中国手机芯片第一次直面 3 nm 高地**。它暂未受制裁，不是因为美国“放水”，而是**规则阈值尚未涉及**高端手机 SoC。但地缘和技术的博弈仍在上演——**今天能跑 3 nm，明天能不能继续，还要看政策微妙变化**。

对普通消费者而言，先享受性能红利再说；对产业链，则是一次关于“设计能力”与“制造壁垒”的大考——你觉得，下一张考卷还会这么“友好”吗？
