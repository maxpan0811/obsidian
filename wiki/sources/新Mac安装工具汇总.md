对于 Mac 下的应用程序，本着简单、实用的原则，可以最小化安装后满足日常工作和生活。

以下是工具列表：

- Homebrew - macOS 包管理工具，可以通过命令方式安装系统依赖和一些应用程序，后续只要使用`brew update && brew upgrade`即可升级，简单方便，类似于 ubuntu 下的 `apt`。
- The Unarchiver - 多格式解压缩工具，至少我遇到的压缩包没有它解不出来的
- Raycast - 替代mac 本身的Spotlight使用的，可以添加插件，可以集成 AI 等等
- bitwarden- 跨平台密码管理器，mac 版、手机版都有，这个感觉比 1password 啥的好用多了，关键是免费-
- Cryptomator - 开源文件加密工具，免费，对于自己的一些敏感信息文件可以通过该工具加密后传到云盘上面，更加安全
- Obsidian - 笔记工具，这个本地存储，可以安装插件实现 git 的同步
- Warp- 终端工具，轻量内置 AI 提示
- OrbStack - 轻量级容器运行时（ Docker 替代方案），轻量、好用，不会占用你过多的空间，同样免费
- IINA - 现代媒体播放器，谁用谁知道
- Folo- 万物皆可 RSS，还有一些特殊能力，自己摸索吧
- windsurf- 若是你喜欢 VScode，那么你可以用这个，集成 AI，对话式编程
- 阿里云盘 - 简单、轻量、空间大
- 网易爆米花 - 配合上面阿里云盘实现在线播放，并且可以记录观看历史，豆瓣评分等等
- Maipo - 第三方微博客户端，时间线正常
- 腾讯柠檬 - Mac 下的管家工具，卸载，垃圾清理，免费
- 文颜 - markdown 下面转化成微信公众号、知乎文章格式的小工具

如果你还有好用、免费的工具可以留言。
