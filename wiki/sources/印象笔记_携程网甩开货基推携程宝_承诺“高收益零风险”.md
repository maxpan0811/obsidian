携程网甩开货基推携程宝 承诺“高收益零风险”
======================

[证券新闻](http://finance.qq.com/l/stock/index.htm "更多证券新闻相关内容列表")[证券时报网](http://kuaixun.stcn.com/2013/1204/10975144.shtml)[[微博](http://t.qq.com/stcn_com?pref=qqcom.dp.titleinfo)]2013-12-04 15:35

证券时报网（www.stcn.com）12月04日讯

|  |
| --- |
|  |

  
　　2013年，互联网[金融](http://finance.qq.com/l/financenews/jinrongshichang/jinrong.htm)如火如荼，各种类“余额宝”产品也层数不穷，各路互联网英雄豪杰蜂拥而上，谁也不愿错过这场互联网搭载下的“金融盛宴”。面对当下[理财](http://finance.qq.com/money/)市场充斥的各种“宝”，携程坐不住了。日前，其推出了一款名为“携程宝”的理财产品。与其它“宝”不同的是，该产品并不挂钩货币基金，而是属于具有理财性质的礼品卡产品。用户在携程购买携程宝，由携程直接承诺到期收益，以“携程礼品卡”的形式直接充入携程帐户。

　　携程官网介绍，携程宝是携程独家推出的“旅游理财”系列礼品卡产品。携程宝首期将上线“90天旅游基金”、“180天旅游基金”及“月月返”三款产品，后续还将陆续推出更多特色旅游理财产品。

　　其中，90天旅游基金到期返4%礼品卡，180天旅游基金到期返6%礼品卡，而月月返产品则是每月返100元的礼品卡。

　　根据销售协议，90天旅游基金与180天旅游基金，均为1万元/份起售，1个订单最多可购买99份；月月返产品为3万元/份起售，1个订单最多可购买99份。

　　需要指出的是，订单中购买及赠送的均为“礼品卡”而非现金，可用于在携程预订预付费酒店、惠选酒店、旅游度假产品、团购产品等。根据国家相关规定，携程宝作为一款具备理财性质的预付费礼品卡产品，不能够兑换现金。

　　因此，从其产品介绍可以看出，携程宝并非目前普遍存在的挂钩货基的理财产品，而是针对未来有明确出游计划的携程用户设计的旅游基金。

　　尤其值得注意的是，其官网宣称：“……携程宝安全零风险。携程宝用户只要遵守携程宝产品的期限和规则，就能获得约定的到期收益，全程无后顾之忧。”

　　公开宣称“无风险高收益”，这让那些受制于不得承诺无风险保底的[银行](http://finance.qq.com/money/bank/)、基金理财产品“艳羡不已”。分析人士表示，理财产品遵循风险收益匹配原则，因此传统上理财产品多不能承诺无风险保底。

　　据中国证券网报道，对此携程相关人士对此回应称，携程宝的定位是一款具备理财性质的预付费礼品卡产品，并非理财产品。“主要是为了回馈礼品卡用户，并倡导消费者制定旅游计划”。

　　上述分析人士指出，携程宝从性质上看，并非真正意义上的理财产品，与“充话费按期返还”有相似之处，某种程度上是打了理财的擦边球；而对此携程人士表示不予置评。

　　事实上，随着互联网平台对于“理财”市场的日益介入，“网上营销”的话术规范成为相关人士的关注点：宣传得保守，吸引不到投资者；说得抢眼球，又有违规风险。

　　此前，有媒体质疑基金公司在淘宝店面销售基金时宣传业绩不当，更是引发了市场关于理财产品借助互联网平台销售过程中如何规范的话题。

　　（证券时报网快讯中心）
