# Apple - OS X Server - 专为团队合作而设计的功能

---

OS X Server 的先进功能为你的企业带来了更强的动力，以及更好的控制和协作。因此，你的整个团队都能更高效地紧密合作了。

现在，在 Mac、iPad 和 PC 上共享文件夹，交换文件和访问文档，变得比以往更为简单。不但如此，用于 OS X Server 中共享  
文件的 SMB2 全新超快协议，更增强了安全性和 Windows 兼容性。由于 OS X Server 支持 UNIX 文件权限以及访问控制列表，因此你可以对共享文件及共享对象进行更好的控制。通过文件共享连接监视功能，你可以查看谁在访问你的服务器，谁在连接你的服务器及其连接的时长。智能的 Spotlight 搜索可以帮助用户在服务器上快速、轻松地查找内容。

* [客户端](http://www.apple.com/cn/osx/server/features/#gallery-sharing-client)
* [服务器](http://www.apple.com/cn/osx/server/includes/gallery-sharing-server.html#gallery-sharing-server)

![](.evernote-content/BF6194F2-BA5F-4A1A-BF38-9C34F38C90F3/AE6C7B0B-0F99-44A9-8A59-0EEE05E1D090.jpg)

你可直接从 Mac、iPad 或 PC 上访问文档、共享文件夹，并与任何群组成员交换文件。

.

.

.

![](.evernote-content/BF6194F2-BA5F-4A1A-BF38-9C34F38C90F3/A31DBDF0-3FF2-4A44-B831-0E6EE1485A2C.jpg)

借助描述文件管理器，在你的组织中部署、配置和管理 Mac 电脑及 iOS 设备都变得更简易。你可以在这里控制一切：创建描述  
文件，设置邮件、日历、通讯录和信息的用户账户，配置系统设置，启用限制，设置 PIN 和密码规则等。凭借描述文件管理器，分发通过 App Store 批量购买计划所购买的机构授权 app 比以往更为简便。它还能让用户访问带有自助服务的门户网站，并从那里下载和安装新的配置描述文件，清除密码，远程锁定或重置丢失或被盗的 Mac、iPhone 或 iPad。

![](.evernote-content/BF6194F2-BA5F-4A1A-BF38-9C34F38C90F3/260F6C19-FC57-41BE-8A99-99FA3048935E.jpg)

.

![](.evernote-content/BF6194F2-BA5F-4A1A-BF38-9C34F38C90F3/EB4561B2-C23F-4BFB-9881-F3A4EC3DDA7B.jpg)

Xcode 服务器可让 Mac 或 iOS 开发团队更轻松地创制稳定、可靠的软件。软件开发者可直接在 Xcode 内部使用 OS X Server，用以自动整合项目并将其存档，自动在多部 iOS 设备上运行测试，托管 Git 资源库，并查看构建与测试结果的历史记录。

![](.evernote-content/BF6194F2-BA5F-4A1A-BF38-9C34F38C90F3/FF1A86E0-4AB5-4A26-840A-1C8220FB5B3C.jpg)

.

![](.evernote-content/BF6194F2-BA5F-4A1A-BF38-9C34F38C90F3/9ABDD885-0806-4308-954E-B950E2C582F8.jpg)

OS X Server 可以作为指定 Time Machine 备份点对所有联网的 Mac 电脑进行备份。集中备份存储有助于保护重要数据，并释放个人硬盘的磁盘空间，完全不再需要各自独立的备份硬盘。OS X Server 还可以充分利用 Time Machine，将你的共享文件、日历、邮件、wiki 等服务器数据备份到其他硬盘上。因此，你可以轻松恢复到之前的配置。现在，你还能查看你的网络上每台 Mac 的备份状态。看看哪些电脑和用户正向你的服务器进行备份，了解他们上次备份的时间，以及每次备份的总文件大小。

![](.evernote-content/BF6194F2-BA5F-4A1A-BF38-9C34F38C90F3/F1228EEC-CA49-4212-8CD6-5683CD63B91C.jpg)

借助 Time Machine，你可将自己网络上所有的 Mac 电脑数据备份到 OS X Server，  
并保护重要的数据。

.

![](.evernote-content/BF6194F2-BA5F-4A1A-BF38-9C34F38C90F3/DF11A65C-17E1-404A-83F5-E9DF09907D92.jpg)

OS X Server 可让多个群组通过他们自己的 wiki 所支持的网站，轻松实现协作与沟通。通过简单而强大的工具，用户可以创建并编辑 wiki 和单个页面，标记及互相参阅资料，上传文件和图片，还可以创建图表并添加评论。Wiki 支持浏览器访问，因此任何人都可以随时进行访问。如此，用户们可以轻松搜索新页面和内容，展示近期活动，并使用 Quick Look 查看附件。而且只要有页面更新或有新评论，每个人都会收到自动电子邮件的通知。Wiki 服务器还专为 iPad 进行了优化。通过 WebDAV 支持，iPad 用户可以在 Pages、Numbers 和 Keynote 中使用 wiki 打开并保存附件。

![](.evernote-content/BF6194F2-BA5F-4A1A-BF38-9C34F38C90F3/1E0AD3DF-2C44-4A18-967F-E1446A97A8C1.jpg)

.

![](.evernote-content/BF6194F2-BA5F-4A1A-BF38-9C34F38C90F3/4C5EE702-CF9B-4DCA-9C12-CDFB7D538732.jpg)

日历服务器可确保网络中的每个人都清楚时间和地点。而且每个人都可以分享日历，安排会议，预订会议室和协调事件。通讯录服务器可让你轻松共享通讯录，并在 Mac、iPad 和 iPhone 上保持同步。OS X Server 的邮件服务器可与 Mac、iPad、 iPhone 和 PC 的电子邮件客户端协作。当新邮件到达时，会通过推送通知即时提醒用户。有了内置的病毒检测和垃圾邮件过滤，不需要的邮件不会再打扰你。

* [日历](http://www.apple.com/cn/osx/server/features/#gallery-communication-calendar)
* [通讯录](http://www.apple.com/cn/osx/server/includes/gallery-communication-contacts.html#gallery-communication-contacts)
* [邮件](http://www.apple.com/cn/osx/server/includes/gallery-communication-mail.html#gallery-communication-mail)

![](.evernote-content/BF6194F2-BA5F-4A1A-BF38-9C34F38C90F3/6F7E77F7-A6D2-4D36-A7C8-A451818F89F5.jpg)

![](.evernote-content/BF6194F2-BA5F-4A1A-BF38-9C34F38C90F3/7BE657B9-B0A9-44DA-96E9-CC9CC29F9930.jpg)

Xsan 是一款强大的可扩充的存储整合解决方案。现在，你企业中的每个人都可以快速地同时访问 TB 级的集中数据。Xsan 内置于 OS X 之中，可以让任意一台支持 Thunderbolt to Fibre Channel 转换器的 Mac 访问 Xsan 卷宗。OS X Server 包含 Xsan Admin 应用软件，可以对 Xsan 卷宗进行管理和配置。

.

![](.evernote-content/BF6194F2-BA5F-4A1A-BF38-9C34F38C90F3/AB040CA1-C60D-44ED-9A22-5F9BFE5EA6D3.jpg)

凭借虚拟专用网络 (VPN) 接入，你的异地用户可安全地连接你的网络并获取服务，同时阻止未经授权者进行访问。OS X Server 包含 VPN On Demand，让每台 Mac 都可以自动建立安全的连接。VPN 服务器支持 L2TP/IPSec 和 PPTP 隧道协议，可以为你的 Mac 电脑、iPad、iPhone 和 Windows PC 提供加密和远程网络接入。

![](.evernote-content/BF6194F2-BA5F-4A1A-BF38-9C34F38C90F3/04996839-9D5C-4561-8C87-D397C6FCD0EB.jpg)

有了信息服务器，你的企业就拥有了即时通讯所赋予的协作动力。现在，多个工作组可以安全地传送文件，共享群组信息，召开音频会议，甚至还可以为其他运行在 OS X 上的信息用户播放演示文稿、影片或照片幻灯片。

![](.evernote-content/BF6194F2-BA5F-4A1A-BF38-9C34F38C90F3/6693F081-2A85-4980-A3FC-A7E43C0E4C27.jpg)

NetInstall 专为在企业中负责操作系统安装和软件升级的管理员而设计，是一个理想的 IT 助理。它可以自动安装软件，无论是新版的 OS X、适用于工作组的应用软件套件，还是两者兼而有之。同时，它还避免了分发  
软件时可能造成的浪费。而且有了它，管理员也不必再亲自去分别配置每个系统了。

.

.
