# 3.3 用链接，建立信息通路 - 少数派

3.3 用链接，建立信息通路
==============

你好，我是 Louiscard，这是你打造信息管理系统的第 12 天，今天我来跟你聊聊「链接」这件事。

在互联网的时代，我们对链接的概念都非常熟悉，在页面点击链接，跳转到另一个页面、弹出一个视频、打开一个程序。

大名鼎鼎的维基百科，不仅持续补充各种知识，还在信息之间建立愈发智能且密集的链接，构建起一张相互关联的知识地图，让我们得以在不同页面间灵活跳转。

和维基百科一样，要想建立我们自己的信息管理系统，当然也不能忽略强大的链接功能。一旦掌握了链接的正确用法，简直瞬间变成能够驾驭信息的魔术师，可以在笔记页面、网页、文档、各种 App 中来回穿梭跳转。

这个过程的重点在于创造信息的网络，让信息之间建立更丰富的联系，你创造的联系越多，它们被你理解得越深刻，你能索引到这些信息的路径就越丰富。学习并不是整理一个个独立的盒子，学习就像编织一张大网，而链接就是打破黑盒的最好的方式之一。

我特别喜欢一个概念叫做「虫洞」，恰当的使用链接可以帮助你打破时空的限制，瞬间从一个次元穿越到另外一个次元，这也是链接最大的意义。它就像任意门，构建出一道道神秘的通道，让你通往知识体系的任意一个角落。

下面我们就以印象笔记中链接为例，介绍下链接的主要使用场景及操作方法。

1. 笔记链接
-------

笔记链接是使用最多的一种链接。印象笔记中每个笔记都有一个固定的链接地址，类似网页的网址，我们将其复制粘贴到其他笔记里，就能点击链接跳转回这个笔记页面。

下面这个例子是我们的一次活动排期表，这个表格左侧是活动时间，右侧是具体任务，笔记在团队内部共享，方便大家了解活动每天的工作安排。

![](.evernote-content/BE2320D0-4F16-48E4-B2D7-394843C82E39/5EAC0F4C-FC41-4608-8C04-1766744F2A31.0)

你可能注意到，有些内容是绿色的，在印象笔记里面，默认带有笔记链接的文字会显示成绿色，所以都是带有链接的内容。添加了链接之后，点击会直接跳转到相应的笔记页面，起到了导航的作用。

具体方法是：

1. 在笔记上右键；
2. 点击复制笔记链接；
3. 新建或打开另外一条笔记，粘贴到正文中。

印象笔记的目录功能实质上也是利用的链接。我们在「笔记的列表视图」选中多条笔记，然后点击「创建目录」，就会把笔记本当中的笔记生成一个目录，点击目录可以跳转到相应的笔记页面。

除了印象笔记之外，OneNote、DEVONthink、Notion、Quip、有道云笔记等绝大多数云笔记都有这个功能，不管你用的哪个工具都赶快去试试吧。

2. 笔记内部链接
---------

很多朋友会使用像 OmniFocus 或 Todoist 这样的任务管理工具，很多任务也和印象笔记链接起来。比如，你剪藏了一篇非常重要的文章在印象笔记，想要今天下午 4 点之前认真读一遍。因此，在任务管理工具中创建了一条任务叫做：「阅读文章：关于 Day One 的介绍」。我们能否像第一部分介绍的一样，把笔记的链接复制粘贴到任务管理的备注中，方便任务推送给我们的时候可以直接点击链接跳转到笔记里呢？

![](.evernote-content/BE2320D0-4F16-48E4-B2D7-394843C82E39/45EF3572-ACA5-4616-B645-761B3E45AAE6.0)

大家可以尝试一下，看看是否可行？

你会发现，点击之后会跳转到笔记的网页版，不仅需要额外登陆，还不能编辑。那么如何才能在其他软件点击链接，直接跳转到印象笔记的客户端打开笔记呢？这种情况下就需要用到「笔记内部链接」功能。

方法非常简单，和复制笔记链接唯一的区别在于，在笔记上右键之后，按住 ⌥Option，菜单中原来的「复制笔记链接」变成了「复制笔记内部链接」，然后点击复制即可。如果是 Win 系统，在复制的时候需要同时按住 Ctrl。

所以，如果你想要在其他 App 也能点击链接跳转到印象笔记，别忘了使用神奇的「内部链接」。需要说明的是并非所有工具都需要使用「内部链接」的概念，绝大多数工具只有一种链接形式，可以点击跳转到 App 中打开相应的笔记，你可以试试自己在用的工具是否支持，欢迎在留言区告诉我。

3. 网页链接
-------

我有一个坚持了挺久的习惯是记录好吃的小店，给每个小店评分，添加菜系等信息。当需要招待一些朋友吃饭，只需要打开「孤独的美食家」这个笔记本，根据朋友的喜好和位置挑选一个实地考察过的馆子，通常不会出什么大问题。

![](.evernote-content/BE2320D0-4F16-48E4-B2D7-394843C82E39/9D30971C-C391-4937-B423-A44B224B899E.0)

为了方便使用，我不仅记录名字，同时希望能够点击跳转到大众点评的页面，这样可以直接把位置分享给朋友或者能进一步看看有哪些推荐的招牌菜。想要实现这样的效果，就需要添加网页的链接。

实际上，这和添加笔记链接的方法没什么本质的区别，只不过如果我们直接复制链接地址，链接会是一长串，看起来不太美观。

![](.evernote-content/BE2320D0-4F16-48E4-B2D7-394843C82E39/0CC8C34C-F381-4552-BC3A-97161C9AC378.0)

下面我来介绍下添加网页链接的方法。比如我想要给笔记中的「便宜坊」添加大众点评的链接。

第一步：复制网页地址；

第二步：选中「便宜坊」这几个字，右键，点击「链接 - 添加…」；

第三步：粘贴链接到文本框，点击「好」。

能够添加网页链接之后，会解锁非常多的玩法。这等于让你的笔记内容和互联网的内容建立了一个直接的链接，链接到各种内容。

比如，我有一个笔记，专门记录感人的视频，如果我在网上看到的一些视频让我涕泗横流，失声痛哭，我一般会在擦干眼流之后记录到这个笔记里。

![](.evernote-content/BE2320D0-4F16-48E4-B2D7-394843C82E39/4BA2ACC3-F5C1-4A77-AF55-61F0C989895D.0)

不仅仅添加截图，当然也会把视频链接贴上去，当哪一天心情不好，想要大哭一场，打开这个笔记，逐个点击链接，畅快淋漓的泪目一番。

4. 添加本地文件链接
-----------

印象笔记的信息承载能力非常强悍，可以在一篇笔记里放文字、图片、Word 文档、PPT 等，放文件的方法也非常简单，直接拖拽。这种方式相当于把文件复制一份并内嵌上传到了笔记中。

有时候我们并不想复制一份放上去或者本地的文件特别大，是一个超过 200M 的视频，没办法放到印象笔记（每条笔记附件最大 200M）。有没有什么办法，在不上传到笔记，只是添加链接到情况下，能够在印象笔记里通过点击链接直接打开本地的各种文件呢？

答案是：可以，不过稍微有点麻烦，我先来说说在 Mac 上如何实现，方法如下：

第一步：选中要添加链接的文字，按快捷键 ⌘Command-K

第二步：将要添加链接的本地文件拖拽到地址文本框中。但这时候，你发现，没有办法点击「好」。

第三步：在地址最前面输入 file://，然后点击「好」。

这时候点击链接就会直接打开文件夹或者文件本身，而这个文件并没有上传到印象笔记，只是通过路径打开了本地的文件。

如果是 Win 可以试试下面这个方法：新建一个文件的快捷方式，将只有 1KB 的快捷方式作为附件放到印象笔记里，点击快捷方式就会打开这个文件，效果也非常好。（如果有更好的方法，欢迎留言分享给我。）

DEVONthink 中支持「索引」的方式导入文件或文件夹，不改变本地文件的位置，但能够在工具中搜索、编辑，非常方便。

![](.evernote-content/BE2320D0-4F16-48E4-B2D7-394843C82E39/2FE5893A-0D25-4354-B408-495000596575.0)

5. 其它 App 的链接
-------------

除了笔记、网页、文件路径等，绝大多数工具都提供链接支持，像 OmniFocus、OmniOutliner、DEVONthink、Trello、Day One 等等，这些链接也可以放到印象笔记里面，或者打通它们之间的通路。

比如下面的视频中，我们点击链接，可以直接跳转定位到 OmniOutliner 的某一行内容，这就给了我们知识管理更加巨大的发挥空间，让这些工具搭配使用，形成一整套流畅完整的信息管理系统，非常酷。

6. 自动化的链接
---------

链接可以指向一个页面，甚至可以触发一组动作。比如点击一个链接可以播放音乐、打开 Ulysses 在写文章、启动免打扰模式、启动暗夜模式、启动一个番茄钟。有了 URL Schemes 和快捷指令，以上的这一组动作只需要一个链接即可启动。感兴趣的朋友可以了解 iOS 的便捷指令和 Mac 上 keyboard Maestro 的相关信息。

最后的话
----

我一直觉得对链接的理解是信息管理水平的试金石，非常反映段位高低。当熟练使用链接之后，特别是对 URL Schemes 和自动化有基本的认知之后，你会慢慢成为一个信息管理的魔法师，精心准备着各种机关，在别人眼里，只看到你在不同软件和页面来回跳跃，你想要的信息就像藏在帽子里小白兔，一伸手想要的东西就能自动飞来，让人叹为观止。

想达到这个效果，按照这一篇文章的方法，快去亲自感受下链接功能的神奇吧。
