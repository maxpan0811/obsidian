## 一句话摘要

0 基础四步走：Obsidian → Claudian 插件 → Claude Code + DeepSeek V4 后端 → 搭建个人知识库，让 Claude Code 直接读笔记、搜文件、跨文件关联回答问题。

## 搭建步骤

1. **安装 Obsidian** — 本地 Markdown 笔记软件
2. **安装 Claudian 插件** — 通过 BRAT 安装，把 Claude Code 嵌入 Obsidian
3. **安装 Claude Code + 配置 DeepSeek V4** — CC Switch 桌面工具，选 DeepSeek 供应商
4. **搭建知识库** — AI 直接读文件、搜笔记、跨文件关联

## 核心卖点

DeepSeek V4 文字能力对标 Claude Opus 4.6，百万 token 价格不到 Claude 的十分之一。
