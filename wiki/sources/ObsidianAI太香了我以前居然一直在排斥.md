很多人用 Obsidian 用了很久，却一直没摸到它真正的灵魂，比如之前的我。

直到我把 AI 装进 Obsidian，才明白`笔记+AI`，才是这款工具的完全体。

![](attachments/fcb28ae1ef479bbf.png)

之前我一直很排斥将 AI 引入 Obsidian 中，甚至是明确拒绝的态度，担心自己的笔记数据、文件什么的被读取。

但真正体验一段时间后，我才后知后觉，`笔记+AI`才是大势所趋，而且能帮你省下超多繁琐操作，只需要选中文本直接问 AI，不用再复制来复制去，体验非常好。

而且在 Obsidian 里接入 AI，操作也特别方便，一点都不复杂。

下面我就以接入 DeepSeek 为例，给大家一步步展示完整的安装和使用教程。

### 安装 Copilot 插件

1. 打开 Obsidian 设置（`Ctrl/Cmd + ,`）。
2. 进入 **`社区插件`** → \*\*`浏览`\*\*。
3. 在搜索框输入 **`Copilot`**
4. 点击安装 → 安装完成后点击启用。

![](attachments/b26e89f00ca2e3d5.png)

### 获取 DeepSeek API 密钥

![](attachments/14da1c65cefffb2d.png)

访问 DeepSeek 开放平台[1]，毕竟目前 Deepseek 算是主流大模型里，使用费率相对较低的一个平台。

别的平台虽然也有免费额度，但一旦用完，后续收费会比 Deepseek 高不少。所以我这边就以 Deepseek 为例，给大家实操演示。

![](attachments/14da1c65cefffb2d.png)

### 配置 Copilot

Basic 中配置 API Keys -> Set Keys

![](attachments/8ba0f1845df048e8.png)

按照我图片里圈出来的地方照着设置就行，跟着图片一步一步操作，依次填写、设置好就可以。

![](attachments/32a13543384d5745.png)设置好之后回到 Obsidian 的编辑页面，在左侧菜单栏找到`OpenCopilot Chat`图标，直接点开。

右边就会弹出一个聊天窗口。

![](attachments/79a533f5622d8cb5.png)

设置好之后，我们在这个聊天窗口里就可以选择自己的模型，选择自己的文件和 Deepseek 对话，输入内容回车，直接在 Obsidian 中和 Deepseek 进行对话，非常方便，一个工具里面操作。

这样就不用再切换浏览器、再去编辑。而且在 Obsidian 里面，你甚至可以让它分析你的对话，分析你的某段文字、某段代码。

![](attachments/a6fa24fb65a02fb4.png)

所以，咱也不知道自己之前过的什么日子，有这么好用的 AI 加持，还有什么理由不拥抱呢。

### 写在最后

当然市面上主流的笔记软件也都有 AI 功能，比如 Notion、语雀等，早已没什么稀奇的了，但是 Obsidian 这款笔记的最大魅力，就是高度的自定义特性，可以不受厂商的限制，通过插件进行高度自定义配置。

所以对于玩笔记的我们来说，你在用哪家的笔记软件，或者你对将笔记接入AI有什么看法，都可以在评论区聊聊。
