Michael Cohen 是一名杰出的苹果教育家，Michael Cohen 在担任一个科技栏目嘉宾时，专门着重谈及了苹果 Swift Playgrounds 为教育所带来的贡献和作用。

![](.evernote-content/C6D53D7A-E3D4-4323-AEF3-C71B73438D24/9BFE7113-8E90-444E-90EB-BA77EB3F9495.jpg)

苹果公司在 2014 年带来 Swift Programming 的时候，当时他们的目标是那些开发人员。Swift Programming 作为一种更具有弹性和灵活性的语言，它被采用，并且在大多数的情况下被众多的开发者所接受。而不到 3 年之后，苹果又重新定义和改变了针对开发者们的方案，而这次除了开发者以外，苹果更是希望打开那些 9-16 岁学生编程的大门。

很显然，通过 Swift Playgrounds ，苹果有机会重新定义并且重新定位一股不可阻挡的教育技术力量。
