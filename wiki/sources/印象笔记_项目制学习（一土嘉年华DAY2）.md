项目制学习（一土嘉年华DAY2）
----------------

*2016-12-28**陆琼琼*[一土上海社区](http://mp.weixin.qq.com/s?__biz=MzI3MzU5MDA2Ng==&mid=2247483670&idx=1&sn=5c932be8155732798e6889794be8c4ba&chksm=eb21b34bdc563a5dfe4d59153fbda5d05078850cb041ec9069950b5209bb13d64601915ba778&mpshare=1&scene=1&srcid=1228uAAondhpjzLkJlPEMVdV##)

文|陆琼琼 12-27

编|林祥毅 12-28

知识很容易获得，但更容易遗忘。学了忘，忘了再继续学，不断地重复是一种让人痛苦的低效。用一种什么样的方式，可以让学习的过程充满了愉快体验，让成果验证和加固知识的掌握，变成一种自发的行为？

![](.evernote-content/86CE4FB7-6FBA-46C1-BAC1-F352E52D2109/D392DDCD-475D-4F8C-89F6-C49FFCCAE437.jpg)

图：HTH学校走廊里挂满学生设计的艺术和科技作品

High TechHigh（简称HTH）是在美国加州圣地亚哥的学校系统，成立于2000年，目前包含了13所特许学校（小学、初中、高中）。比尔和梅琳达.盖茨基金会是学校的资助方。

学校最大的不同是颠覆传统课堂：所有课程都是基于项目的学习。将各学科知识渗透在项目制学习的过程中，以问题为核心，学生通过学习、讨论、动手实践、老师引导、专家咨询等方式将自己的学习与真实的世界相连，在解决问题的过程中学到真正有意义的知识。

在学期结束的时候，所有的学生都会走到讲台上做一次期末展示，展示的内容就是他们这学期学到了什么，做了什么项目，有什么成果，这些成果又怎么运用到生活，这些学习又是怎样改变和重构了他们的思维方式。

![](.evernote-content/86CE4FB7-6FBA-46C1-BAC1-F352E52D2109/CA14A30F-00C4-4B55-B9B6-38DF20948E38.jpg)  
图：Tina老师，一土学校教学顾问，来自于HTH

今天由Tina老师主讲项目制学习工作坊。

出于项目制学习的教学习惯，Tina老师首先抛出了一个问题：用5分钟回忆学校之外真正学到东西的时刻。我们开始在自己的核心记忆库里面寻找那些永远不会忘记的时刻。

![](.evernote-content/86CE4FB7-6FBA-46C1-BAC1-F352E52D2109/27AE09E2-B80F-4E10-AB7B-C17EA973CCCA.jpg)

**每个人都能找到，那些真正学到东西的时刻。**

分享一个苏州伙伴讲述的经历：鱼塘主爷爷，为胆小怕动物的小孙女，带来的一堂关于捕鱼的项目制学习课。

一鱼塘的鱼，怎么捕捞？方法有多少种？

选择放水捕鱼，基于什么原因？

放掉的水可以有哪些去处？

捞上来的鱼要做哪些分类？

哪些鱼还要继续放回去养，作为可持续的生态？

是一个引发思考，提出问题，解决问题，反思的过程。

小姑娘不一定成为一名捕鱼能手，但是这样一次学习的经历，必然塑造和影响了她的思维方式，并且终身受用。

**大家终身难忘的学习时刻，有哪些特点？**

这样的学习是和真实世界相连的，基于问题的解决，需要沟通、合作、借力，回答一连串开放式的问题，遇到各种挑战甚至风险和挫折，需要有老师指导，需要专家和权威的引导，更多的问题不是一个人能完成的，需要团队合作，有榜样可以参考。学习的过程不是枯燥的，甚至是在游戏中学，可以做各种实验。

这就是HTH一直在探索和实践的项目制学习（PBL）。

项目制学习（Project-Based Learning）是一种教和学的模式，旨在把学生融入整个项目制任务完成的过程中，让学生积极地学习跨学科的知识和技能，自主地进行知识的建构，主动地进行小组协作和人际交往。以与现实链接的、学生自发掌握的知识、培养能力为教学目标，并以成果展示作为评估。

![](.evernote-content/86CE4FB7-6FBA-46C1-BAC1-F352E52D2109/E39FEBE5-E19D-4B49-A0E6-7D382ECCE93A.jpg)  
图：HTH学校一个造船项目课的成果展示

造船项目是一个持续十周的项目，以某一整天的学习举例，了解项目制学习是如何进行的。

|  |  |
| --- | --- |
| 8:00-8:30 | 晨会 |
| 8:30-9:30 | 数学实验室，学习数学知识，测量，构建模型 |
| 9:30-10:30 | 写作工作坊，老师会指导学生如何写记叙文，包括怎么引用别人说的话，成果展示一篇记叙文 |
| 10:30-12:00 | 科学探索：比如密度的学习，老师提供十种液体，学生进行猜测、实验和记录结果 |
| 12:00-12:15 | 阅读 |
| 12:15-1:00 | 午餐 |
| 1:00-2:00 | 阅读，为下周做准备，比如阅读关于船只的历史/种类等小组讨论 |
| 2:00-2:15 | 简短的反思，写下过程中的想法和反思 |
| 2:15-2:30 | 打扫和规整 |

不难发现，这是一个集合了数学、科学、语文等多学科知识的实践活动，多种科目知识渗透在一个项目中，通过解决问题，自发学习和掌握各种知识，并用成果验证和体验学习的乐趣。

![](.evernote-content/86CE4FB7-6FBA-46C1-BAC1-F352E52D2109/EF8392EB-8D49-4575-A792-87F5DB31EC18.jpg)

图：教师为项目和课程体系的设计者

**老师是项目制学习，个性化教育实践的关键角色。**

HTH的老师都是项目和课程体系的设计者。他们跨学科协作设计自己的课程，主导学校的教工会议探讨学校的议题。他们参与关于课程体系、评价、职业发展、招聘等学校最为重要的方方面面的决策。

工作期间支持他们开展小组教学、有足够的时间计划和创造综合的项目，为学生直到毕业打造出评价和展示学习成果的连贯标准。

HTH的老师非常了解自己的学生，采用以学生为中心的教育理念，支持和挑战每一位学生。在具体项目进行的过程中，项目组老师和各科专家团队老师一起辅助和引导学生。

每位学生都有自己的导师，以小组的形式经常与导师碰面，导师还会家访，成为学校与学生家庭的联系人。

![](.evernote-content/86CE4FB7-6FBA-46C1-BAC1-F352E52D2109/F72AD6B6-BA57-4154-B229-A8336D71F3E9.jpg)  
图：HTH的老师和学生正在商讨如何改进实验

 

**源于内在的动力，学生学到的远比知识多很多**，这样的学生毕业之后的发展到底什么样？

HTH的高中毕业生中96%升入大学，其中66%升入四年制大学。34%的毕业生选择STEM领域深造学习，相比之下，美国全国的平均比例是17%。

Tina说他们在刚进入大学的时候，需要一个适应长时间坐着听课的过程。之后，他们的毕业生在提问、与老师讨论的积极性上普遍比较高，较强的沟通能力也为他们最后步入社会工作提升了竞争力。
