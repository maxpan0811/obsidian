**好书就像项链上的珍珠，一般都是一串。捧起一颗，就带出一条。**

在给孩子挑选读物时，有两个出版社的书必不可少，一个是**Usborne**，另一个就是响当当的**DK出版社**。

英国DK公司，全名**Dorling Kindersley**，始建于1974年，后被培生收购。

**在英国，差不多每个孩子都是读着DK的图书长大的。**

英国DK出版社的图书向来以**印刷精美，图文并茂**闻名世界，作为儿童科普类图书，**DK将主要的科学知识通过生动活泼的手法呈现出来**，重在培养出一种思维方式，从而激发出孩子的求知欲和想象力。

DK的愿景就是创作既有**视觉冲击力**又包含**信息丰富**的图书，使读者能够理解每一个主题。

![](.evernote-content/02E04FF4-324A-4504-B3C6-7B8F29148DCC/86F24624-3F12-482E-9F57-8481A309F704.jpg)

**DK自有其坚持的理念。在营销为王的时代，DK坚持做内容为王。**

Dk的每一本书意味着一个浩大的工程，例如他们的《自然历史百科》就是和美国史密逊国家自然历史博物馆及特约为这本画册撰稿的众多知名专家，整整耗时五年才得以完成和出版的。 

制作如此精良，质量如此高的书，如果能读原版书，当然是最好了，但是在为孩子选书时，如果难度太高，孩子难免产生抵触心理，所以不如从中文版本先开始看起。

但是面对琳琅满目的中文版DK，很多家长还是犯怵：“究竟选哪本？”

赖赖妈比较推荐**中国大百科全书出版社**和**中信出版社**，**电子工业出版社**这几家的书。

以下就介绍一些比较经典的适合孩子的DK中文版丛书：

**《DK儿童百科全书》**

英国DK公司 / 杨寅辉 等 / 中国大百科全书出版社 / 2010-6-1 

![](.evernote-content/02E04FF4-324A-4504-B3C6-7B8F29148DCC/3D51FC4E-069E-4438-ACEA-89DBB097F101.jpg)

        “打开这本配以精美插图的百科全书，就如同踏上一场惊奇的旅程，开始探索我们周围的奥秘。遨游在太空之中，发现各星球上不可思议的环境；深入皮肤之下，了解人体运转的各项原理。”

**《DK动物生活大百科》**

Charlotte Uhlenbroek 等 / 张劲硕 / 电子工业出版社 / 2014-6-1

![](.evernote-content/02E04FF4-324A-4504-B3C6-7B8F29148DCC/934DB234-7491-4336-A832-415A381E9840.jpg)

        由**英国著名动物学家和心理学家夏洛特·阿兰布鲁伊克博士**带领顶级专家队伍撰写，获得美国自然历史博物馆鉴定认可。

        这套好于中国大百科全书出版社的《DK儿童动物百科全书》。

《DK动物生活大百科》是由中国科学院动物研究所高级工程师、国家动物博物馆科普策划总监，中国科普作家协会会员，科学松鼠会会员**张劲硕博士**翻译，而《DK儿童动物百科全书》只是由张劲硕博士审校的。

**《DK儿童太空百科全书》**

美国DK公司 / 王佳 / 中国大百科全书出版社 / 2011-1 

![](.evernote-content/02E04FF4-324A-4504-B3C6-7B8F29148DCC/EA97B32E-6212-444D-976F-3B55C033FF32.jpg)

        《DK儿童太空百科全书》几乎涵盖了所有普通读者感兴趣的和想要了解的太空知识。从距离我们最近的行星到遥远的星系，以及我们所在的整个宇宙。

**《DK儿童目击者》**

![](.evernote-content/02E04FF4-324A-4504-B3C6-7B8F29148DCC/CB27148A-3560-4A30-9266-8EC0D5069BA2.jpg)

《儿童目击者》是英国DK出版公司为激发孩子们的阅读兴趣而推出的一套丛书。

        这套书分别为：**预备级（0-3岁）、第1级（2-5岁）、第2级（4-7岁）**。

**在实际操作中，家长可以不必严格按年龄局限，其实哪一级别的书都可以帮助孩子学到他们不懂的知识。**

**《DK儿童自然大百科》**

中信出版社 / 2014-3-1 

![](.evernote-content/02E04FF4-324A-4504-B3C6-7B8F29148DCC/1AB4721E-53AC-4579-AA6C-8787D3F15B99.jpg)

        《DK儿童自然大百科》在编排上特别考虑了孩子的认知规律，图片精美，编排合理，符合儿童认知习惯，易于拉近儿童与动植物的距离，是儿童了解自然、亲近自然不可多得的科普读物。

**《DK地球大百科》**

克里斯廷·利宾考特 / 王纯纯 / 电子工业出版社 / 2013-1 

![](.evernote-content/02E04FF4-324A-4504-B3C6-7B8F29148DCC/A1CCD5FD-E91B-4A1A-9C89-F99F13989685.jpg)

**克里斯廷·利宾考特编著的《DK地球大百科》是DK顶级品牌“目击者”系列的精华版。**

        《DK地球大百科》从“目击者”选取最精彩的五个自然主题：**海洋、火山、天文、晶体和宝石、自然灾害**，集结而成精华版。

**中国科普研究所李元（太阳系（6741）号小行星被命名为“李元星”）、中国科学院地质和地球物理研究所、中国地质科学院地质力学研究所、中国地震局地质研究所等专家审定。**

**《万物运转的秘密 : 给青少年的物理世界入门书》**

大卫·麦考利 (David Macaulay)、尼尔·阿德利 (Neil Ardley) / 赵耀康、韦坤华 / 电子工业出版社 / 2014-5 

![](.evernote-content/02E04FF4-324A-4504-B3C6-7B8F29148DCC/CEA2CA91-10A6-4427-8924-E54BAB6A197E.jpg)

**凯迪克金奖得主大卫·麦考利经典科普绘本**，这本充满创意的物理世界入门书自出版以来畅销半个世纪，让看似毫不相关的事物有了绝妙的联系。

**《时代》曾评论道：“麦考利画得比世界上任何一位钢笔墨水插画家都好。”**

**《DK宇宙大百科》**

马丁·里斯 / 余恒、张博、王靓、王燕平 / 电子工业出版社 / 2014-11-1

![](.evernote-content/02E04FF4-324A-4504-B3C6-7B8F29148DCC/AFA6D3A6-ACB1-4806-BCA2-2D8FDF7CCB5C.jpg)

        宇宙是什么？它从何而始？又将如何而终？地球、太阳系乃至银河系在浩瀚的宇宙中扮演了什么角色？…… 

**四位译者历时两年多时间打造出精品，中国科学院北京天文台教授，中国天文学会常务理事，天文学名词审定委员会主任—卞毓麟老师担任顾问。**

**《DK儿童历史大百科》**

英国DK公司 / 中信出版社 / 2015-3-1 

![](.evernote-content/02E04FF4-324A-4504-B3C6-7B8F29148DCC/E1FA4515-9C25-40AE-99B1-D211013BE851.jpg)

        《DK儿童历史大百科》从**史前文明一直讲到21世纪的世界**，按人类文明的发展顺序，分别介绍了**五大洲的历史人文风情、重要历史事件**、将每一个历史时期的地图、绘画、珍贵文物照片和富有趣味的文字解读相结合，为孩子搭建了一座了解世界的历史殿堂。

**《DK儿童百科百问百答》**

英国DK公司 / 王晓宁 等 / 中国大百科全书出版社 / 2011-5 

![](.evernote-content/02E04FF4-324A-4504-B3C6-7B8F29148DCC/10644332-25DD-4B73-BE1C-C10593435D18.jpg)

        《DK儿童百科百问百答》改变了以往的阅读和思维模式，溶入了全新的编辑理念，将200个互动的科学常识，300个不可思议的奇闻怪事，500个纵横交错的知识连接通过简洁的语言和生动的图片，直观快速的展现在眼前。

**《温迪嬷嬷讲述绘画的故事》**

![](.evernote-content/02E04FF4-324A-4504-B3C6-7B8F29148DCC/5A7CB622-F764-4329-BE62-54E872F14B03.jpg)

        《温迪嬷嬷讲述绘画的故事》一书深入分析三十余幅巨作，包括波提切利的《维纳斯的诞生》、伦勃朗的《犹太新娘》、莫奈的《莲池》、凡·高的《自画像》等。由画面放大的局部深入大师的精神世界，揭示作品底蕴，解释绘画技巧。

        简明扼要，娓娓道来，温雅隽永，耐人寻味，温迪嬤嬤的《绘画的故事》将你引上一段奇妙的文化旅程。帮你理解、欣赏的艺术。

后记：

DK的书因为内容太过丰富，对于太小的宝宝接受起来会有些难度，但是家长们可以用**一点点蚕食**的方法慢慢引入，让孩子在不知不觉间喜欢上经典。

一些选书tips和购书渠道：

1. 一个人编译的比一群人编译的好。

2. 尽量选名家编译的，品质有保障。

3. 首选英国DK和美国DK。

4. 亚马逊DK的书比较全，京东上的DK比较杂，推荐亚马逊有活动时收。
