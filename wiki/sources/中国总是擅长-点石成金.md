# 今天“人民日报”“新华社”先后发了一篇文章，题为《[为什么一只活鸭上百元，一只烤鸭只要](https://mp.weixin.qq.com/s?__biz=MjM5MjAxNDM4MA==&mid=2666951400&idx=1&sn=85b24f2ef3e86ac1cc9fe7afdfac0774&scene=21#wechat_redirect)[25元？](https://mp.weixin.qq.com/s?__biz=MjM5MjAxNDM4MA==&mid=2666951400&idx=1&sn=85b24f2ef3e86ac1cc9fe7afdfac0774&scene=21#wechat_redirect)》。这篇文章的结构大家应该很熟悉，抓住一只烤鸭去挖掘背后的大规模工业之美，也就是小镇一个月前写的《[一只十几元的烤鸭，是美国无法翻越的高山](https://mp.weixin.qq.com/s?__biz=MzI2MzY3NjAwOQ==&mid=2247501535&idx=1&sn=d4d41a9b7d2ede93ea9e07230074874f&scene=21#wechat_redirect)》。

# 这篇文章后来得到陈力丹教授的认可，之后有媒体针对这一现象进行了报道，还成了一个小热点。（参见《[有幸得到陈力丹教授的肯定，分享三个感想](https://mp.weixin.qq.com/s?__biz=MzI2MzY3NjAwOQ==&mid=2247501884&idx=1&sn=e52e5d1c8d2b291b0ceba031e3d38859&scene=21#wechat_redirect)》）

# 这是好事，传统媒体受新媒体冲击很深，蹭流量和搞大新闻，成了很多记者的核心追求，不管报道是否属实，只要能成为社会热点、引起普遍关注，那就成了，记者群里会纷纷发“喜报”，恭喜踩准了流量。

于是挖掘社会阴暗面和奇闻轶事，甚至制造矛盾和冲突，成了非常普遍的现象。但真的没有好的题材可以挖掘吗？不是的，中国作为人类历史上空前规模的工业体，工业与普通人的生活之间距离太远，仅仅抓住生活的小细节，挖掘背后的工业之美、时代进步，也可以创作同时具备流量和社会价值的好文章。

# 小镇上个月初写《[一只十几元的烤鸭，是美国无法翻越的高山](https://mp.weixin.qq.com/s?__biz=MzI2MzY3NjAwOQ==&mid=2247501535&idx=1&sn=d4d41a9b7d2ede93ea9e07230074874f&scene=21#wechat_redirect)》，从一只烤鸭入题，列举了包括垃圾处理、地沟油、纺织业、生物质处理、矿物提炼、绿色循环等在内的例子，就是要阐述在一个超大规模的工业体系内，如何通过“物尽其用”，获得无与伦比的竞争优势。

# 但“物尽其用”仅仅是超大规模经济体的一个方面，还有很多方面尚缺乏公众关注，比如今天要谈的就是“点石成金”，中国总是擅长把公认的成本转化为红利。

最典型的就是人口。在中国崛起之前，西方普遍认为，一个发展中国家，必须控制人口增长，否则一定会严重限制一个国家的发展，甚至导致国家危机。

![](attachments/51beacd64e57ed36.jpg)

比如1994年美国农业部政策顾问莱斯特·布朗发表《谁来养活中国》一文，代表了当时西方普遍的观点。在这篇文章中，莱斯特·布朗基对中国从1990年到2030年的粮食产能进行了估算，认为中国人口高速增长、淡水资源分布不均、工业化导致耕地减少，预计2030年中国人口增至16亿人，人均粮食消费量上涨至400公斤，而耕地面积将减少到7.2亿亩，最终认为届时中国本土粮食产能只能满足自身四成的需求，其余六成必须依赖进口。

基于这种判断，认为中国庞大的粮食进口需求，将加剧全球粮食危机，破坏全球稳定，最终得出了“谁来养活中国”的疑问。虽然次年中国邀请莱斯特·布朗访华，也进行了充分交流，但莱斯特·布朗没有改变自己的判断，还进一步把文章扩充为一本书，引起世界普遍关注。

当时国内对莱斯特·布朗的批判，大多是从感性角度，基本没有说服力。真正从理性角度反驳的代表，是袁隆平和林毅夫，他们从数据和事实出发，认为在科技进步和有效治理的共同作用下，“中国人可以养活自己”。

事实也确实如此，2005年底，联合国世界粮食计划署宣布，自2006年起停止对中国的粮食援助，中国成为全球粮食的捐赠国。

粮食问题解决，人口才具备了从成本向红利转变的基础，这个过程六分靠科技、四分靠治理。

在粮食减产压力下，自2004年起，中央一号文恢复连续发布，每年的重点都是粮食增产。根本的红线是18亿亩耕地，这几年借助技术进步，加强了卫星遥感、无人机监测、图斑调查等，一些常年耕地萎缩的南方省份，也开始出现耕地增加，而北部东三省、内蒙古、新疆，开垦了超过2亿亩耕地，大规模的高标准农田建设，极大改善了农业种植条件，还有很多管理手段，比如普遍推广农业保险、推行省际横向补贴、重点农产品保护性收购等等。

而在科技方面，今年1月国新办举行新闻发布会，提到了一个关键的自我认定“经过多方努力，我国农业科技创新整体迈进世界第一方阵”，注意，目前国家仍然认为在工业领域仍然处于追赶阶段。农业科技进步贡献率从2012年的54.5%上升到2023年的63.2%，农作物综合机械化率达到73%，良种覆盖率超过96%，高标准农田超过10亿亩。

有很多人质疑中国的粮食连年增收，总是拿自己的身边说事。真应该去北方好好看看什么叫大规模、机械化农业生产，以工业化的思维，从各个环节提高效率，绝非小农经济能够想象的。其实最简单的道理，如果中国粮食不是连年增收，国际粮价就不可能如此稳定，国际四大粮商不是傻子，他们非常关注全球尤其中国的粮食生产情况。

![](attachments/9835224c83a6ea12.png)

在解决了粮食供应的基础上，通过教育和产业的联动升级，虽然劳动力负增长，但得益于劳动力素质的提升，中国有资格从人口红利转变为人才红利。由于全民健康水平提升，银发经济也带来了非同一般的新红利，老人不是负担，而是老有所为，这必将是新的增长点。

根本不用担心劳动力不足，中国在有意克制效率的提升，还要注意到，中国农业从业人口太多了。2024年农业就业人口占比高达22.2%，虽然多数是兼业而非全职农民，但劳动力数量远超实际需求，这也是农民收入低的重要原因，就那些产值，分的人多了，自然人均少。

只是从社会稳定等更高维度出发，选择保持历史耐心，慢慢减少农业人口，目前大概每年减少1个百分点，也就是大概700万人。

预计到2050年，农业就业人口最多3千万就足够了，仅这一项全社会对劳动力的需求就可以减少1亿人，届时，也将真正实现城乡的均衡发展，市民、村民，仅仅是不同生活方式，而非经济社会地位的差别。

而随着中国在数字和智能领域的广泛应用，庞大的高素质人口和制造产业，催生出了“数据”这一新资源，海量的生产和消费行为数据，能够为AI训练提供充足的燃料，那么未来中国还将从人才红利，进一步升级为数据红利。

![](attachments/78f140708f22bd13.png)

而上述，在全球其他国家，包括发达国家，往往是成本而非红利。

人口之外，“点石成金”的突出的案例是绿色转型。

西方搞绿色转型，就是把减碳视为负担，全球其他国家也是这么看的，每年气候大会，发达国家和发展中国家吵得都很凶，核心就是谁来承担绿色转型的成本。

然而中国却把绿色转型打造成了产业优势，已经实现了换道超越。核心就是实现制造业的绿色化、低碳化，比如把火电发电更多转换为风光等可再生能源发电，把燃油汽车转换为电动车，推进船舶的绿色化，而为了支持绿色化改造，还同步推进电网的升级，国家还关注建筑外立面的光伏化，在技术和产业革新的同时，也在进行社会治理和制度的改革。

甚至就算没有革命性的技术突破，在现有制造水平下，通过转换生产方式，照样可以提高全要素生产率。而这很大程度上依赖中国的人才红利和大规模生产优势，快速的升级迭代，迅速降低了绿色能源的成本，2024年，中国可再生能源装机占比超过50%，提前完成2030年目标，而绿色能源也从十年前的奢侈品，变成了更加普惠的选择。

还给了广大欠发达地区，越过火电，直接进入绿色能源时代的新选择。分布式的光伏，大大降低了对电网的要求，贫瘠的非洲，在解决了能源问题之后，也可以开始搞自己的工业。

只要选择拥抱中国，一切应有尽有。

![](attachments/0d98dc8bbf62d5fc.png)

不仅是绿色能源的转型，还有产业的绿色升级。比如宁波慈溪市桥头镇废塑料再生利用行业很发达，但是也带来了严重的生态污染，后来实施“塑料再生绿岛”项目改造，全面升级了废气收集和处理设施，周边环境得到立竿见影的改善。

而潍坊港在去年建成全国首个“零碳港口”，就是综合利用分布式“风电+光伏”绿电供应系统、绿氢制储全产业链应用，年产绿电7000万度，足够港口所需，多余电力用于制备氢气储存他用，从而实现了零碳排放。

![](attachments/14d86796a49ac165.png)

还有前面提到的内蒙古、新疆的大规模农田开垦，尤其是新疆耕地面积已超过1亿亩，仅2022年到2024年就增产粮食百亿斤，正在成为中国的“西北粮仓”。得益于两件事：一是几十年长期坚持的治沙，二是光伏等产业的发展。

通常觉得新疆干旱缺水，所以不适合发展农业，其实不是的。新疆光照极为充足、水源稳定、土地广阔，天然适合搞大农业，只不过要想挖掘这三大资源，需要很高的门槛。比如水，新疆主要依赖冰川融水，而缺乏植被导致蒸发量极大，必须人为干预。这几年防沙治沙加上推广光伏，就让新疆变得适合搞农业。

![](attachments/94d1a72a44d6e8bf.png)

如果说到这，似乎绿色转型仍然是一种政府主导的模式，似乎不具备效率优势。

如果是这样，小镇也不会说绿色转型，是中国实现了成本转变为红利的典型。

关键要看不同能源背后的本质差异。小镇之前在《[被妖魔化的](https://mp.weixin.qq.com/s?__biz=MzI2MzY3NjAwOQ==&mid=2247502010&idx=1&sn=dfeb9abed529a6775968b2c79b4f29e3&scene=21#wechat_redirect)[“内卷”，恰恰是中国崛起的关键](https://mp.weixin.qq.com/s?__biz=MzI2MzY3NjAwOQ==&mid=2247502010&idx=1&sn=dfeb9abed529a6775968b2c79b4f29e3&scene=21#wechat_redirect)》已经提到了，传统化石能源和绿色新能源，在固定成本和变动成本上差异很大。

一个火电厂，固定成本也就是厂房、设备等占比大概就是40%，而用于发电的煤炭以及因此产生的物流等变动成本，才是大头。随着技术的不断进步，变动成本占比越来越高，这就导致火电规模效应存在上限，超过一定规模之后，继续扩张，边际成本反而会增加。

风光等绿色可再生能源就不一样了，固定成本占比极高，目前在中国大概是70%左右，而光、风等资源对当前人类而言，几乎是无限且免费的，于是经营的变动成本占比很低，随着规模的扩大，变动成本占比可以持续下降，这就意味着目前风光能源的规模扩充几乎没有上限，可以通过扩大规模，进一步摊薄固定成本，继续提升利润。而且固定成本也是可以靠技术和规模化持续降低。

源头能源的价格持续下降，一定会带动全社会生产成本下降，提高全要素生产率。

正是因为绿色新能源不同于传统能源的近乎无限扩充的特性，才使得中国的新能源产业在短短几年内就实现飞跃，才让中国把全球公认的绿色转型是成本，变成了发展红利。

人类历史上最庞大、最完善的工业体系，也更容易发挥协同效应。比如前面提到的光伏发电与农业生产之间的联动，新疆、青海、内蒙古等大量兴建的光伏电站，不仅把不值钱的荒漠戈壁变成了金山银山，还大大改善了生态植被，一开始种草，等到沙子土壤化，慢慢就可以种其他农作物。

![](attachments/2dd4391b01138b05.jpg)

风光发电不稳定，放在如西班牙这样的国家，很可能导致全国性的大停电。但在中国就不一样了，中国有同样庞大的耗能产业，比如电解铝产业，未来通过更灵活的弹性生产，完全可以成为消纳风光发电、稳定电网的新型储能。类似的还有算力产业，国家推广“东数西算”，就是要更高效的利用西北的风光发电。

类似的新能源汽车，未来随着电网技术的进步，几千万、数亿辆电动乘用车，也可以成为中国庞大的新能源系统的一个个组件。

现在网上有些竞争的失败者，顺着西方编造的“产能过剩”，矮化中国的新能源产业，说什么过剩。这就有些过分了，竞争不过那就想办法提高竞争力，而不是搞这些歪门邪道。

尤其是汽车，相比仍然庞大的传统燃油汽车，新能源汽车的产能是不足而不是过剩，未来两三年是中国汽车开疆拓土的关键窗口期，需要的是扩张而不是收缩。

最简单的道理，美国会那么好心，帮中国解决问题吗？

按照欧洲和日本的汽车市场规律，第一大车企在国内市场的占比应该在35%到40%，2024年中国排名第一的比亚迪在国内才只卖了383.7万辆，占国内销售的12.75%，还有非常大的上升空间。国内目前汽车品牌还是太多了，由于很多企业背后是地方政府，淘汰的进程会慢一点，但不可能逆转。

而从全球看，未来乘用车的新能源化是必然，无非早点晚点，而2024年全球新能源汽车渗透率才只有23.2%，也就是说还有80%的燃油汽车有待淘汰。

考虑中国本土汽车市场未来有望突破4千万甚至5千万辆，即使全球其他国家不增长，那么全球未来年需求量也将达到1亿辆，中国汽车完全应该占据一半的份额，对比2024年中国新能源汽车不过1200万辆产销量，起码要再增长300%才够，又怎么能说过剩呢？

还是要强调，不同于化石能源边际成本高、扩张边界有限，新能源的扩张边界极大，规模越大、边际成本越低、效率越高，如果想不清楚不同能源背后的根本逻辑，出局已是必然。
