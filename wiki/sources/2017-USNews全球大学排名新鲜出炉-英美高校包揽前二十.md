# 2017 USNews全球大学排名新鲜出炉，英美高校包揽前二十

2017 USNews全球大学排名新鲜出炉，英美高校包揽前二十 2017 USNews全球大学排名新鲜出炉，英美高校包揽前二十 2016-10-25美国留学快报今天，USNews发布了2017全球最佳大学排名，这也是USNews第三次发布全球大学排名。2017年US News的全球最佳大学排名致力于让人们了解那些全球顶尖学府在全球范围中的定位。它不仅对每个大学的本科及研究生教育水平进行整合，并对学校的学术研究与业界声誉做出准确且综合的评价。 全球最佳大学排名重点关注机构的研究水平。特别是，排名用指标测量大学的全球和地区的声誉、学术研究的表现、包括文献引用和发表频度的文献使用指标、有关教工和博士
