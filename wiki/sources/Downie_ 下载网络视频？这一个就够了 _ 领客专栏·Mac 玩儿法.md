# Downie: 下载网络视频？这一个就够了 | 领客专栏·Mac 玩儿法

---

Downie: 下载网络视频？这一个就够了 | 领客专栏·Mac 玩儿法
------------------------------------

*2016-04-14* *让 Mac 更好用的*

Downie 这款老牌视频下载器如今也焕发了第二春，它可不是一般的视频下载软件。不仅仅局限于 YouTube / Vimeo Downloader 的定位，现在这个老家伙可是支持近 1000 个网站的视频下载，其中包括了像优酷，土豆、爱奇艺、网易云音乐、ACFUN、Bilibili 这样的国内大站哦！

![](.evernote-content/03EB49BB-91FA-4F1E-8094-85D291F63F0C/9E8AEBA9-29AD-4135-897E-32AEB6924A57.jpg)

如果你想下载的视频不在 Downie 的网站列表支持范围内，可点击右上角的「支持」给开发者提 Request。视频下载的操作极为方便，只需将 URL 从浏览器拖入 Downie 主窗口即可开始下载视频，另外还有另外三种方法也可下载视频：

* 拷贝 URL 直接在 Downie 主窗口上 ⌘ +V）
* 通过支持 Chrome、Safari、Opera 三大浏览器的官方插件一键导入 Downie （软件设置里可直接找到插件下载链接）
* 将 URL 直接拖入 Dock 上的 Downie 图标

Downie 下载的速度非常快，支持 4K 级别的 Youtube 视频下载。视频可自动导入 iTunes 管理、可下载字幕、支持使用同门软件 Permute 对视频进行拼接。

![](.evernote-content/03EB49BB-91FA-4F1E-8094-85D291F63F0C/74681464-9E74-463F-B322-D0445706495A.jpg)

有些网站的视频无法正常下载，可能是需要用户登录账号。如果登录后还是无法下载。此时你可以通过 Downie Menubar 上的 User-Guided Extraction 窗口对网页里的视频资源进行扫描，右侧列表会列出所有的视频链接，找到你的目标视频点击「+」号即可正常下载。如果这样还不行，可给开发者提 Request。

2.0 版本更新了许多实用功能，适配了 OS X 10.9 及以上的系统版本（OS X 10.8 不再支持）。支持视频下载的暂停和恢复，如果下载进程由于网络障碍而中断，可进行自动恢复、HTTP 下载可自动将视频分解成片段，提高下载速度。

![](.evernote-content/03EB49BB-91FA-4F1E-8094-85D291F63F0C/DBD8B3D9-8F35-48FF-B0D6-07A2D96A193C.jpg)

历史列表方便用户对已下载的视频进行查找

![](.evernote-content/03EB49BB-91FA-4F1E-8094-85D291F63F0C/296E13D8-ED6A-4B1F-A6CD-C733897CB28C.jpg)

推荐当下 Downie 最热门的下载视频

Downie 在操作上对细节的雕琢非常深入，比如：

* 当你粘入一段包含目标视频链接的文本片段时，Downie 可自动识别链接并下载；
* 内置了四个搜索引擎（YouTube、Vimeo、TED、SoundCloud），用户可直接在主窗口右上角的搜索框搜索目标视频；
* 支持下载的网页视频都提供视频截图预览，体验会更好；
* DMG、ZIP、RAR 这样的非视频格式也在逐步融入进来；
* 未完成下载的文件会自动改为 .part 后缀格式；
* 可自动选择最佳清晰度，也可自定义清晰度级别；

![](.evernote-content/03EB49BB-91FA-4F1E-8094-85D291F63F0C/F0BC4F47-9403-46FB-BA7F-935B6226B109.jpg)

搜霉霉的视频更方便了。

Downie 目前可在官网下载试用，适用于 OS X 10.10+ 的设备，售价 $19.99。
