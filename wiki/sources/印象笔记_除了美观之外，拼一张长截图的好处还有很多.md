# 除了美观之外，拼一张长截图的好处还有很多

---

不论是 iOS 还是 Android 平台，都有很多优秀的长截图工具。截图这种形式不仅很方便，而且发送给别人时还能保留信息原有的样子。在我们的生活中，长截图有许多非常实用的用途，让我们一起来看看长截图究竟能做哪些有趣的事。

制作歌单
----

「你平时跑步时都听哪些歌，把歌单发给我吧。」  
「好啊，我用网易云音乐分享给你。」  
「不好意思，我用 Apple Music……」

把自己喜欢的歌单分享给朋友，长截图是一个很方便的方法。在 iOS 的长截图 App 中，[Tailor](https://sspai.com/app/Tailor) 绝对是懒癌患者的福音，它能通过对截图的自动识别，将同一应用中的每屏截图自动合并拼接成长图。

使用时，首先将需要分享的歌单用截图的方式保存下来，然后打开 Tailor，Tailor 会自动识别截图进行拼接，帮你隐藏重复的部分。点击分享按钮，一张属于你的歌单长截图就制作好了。

![](.evernote-content/3603B62E-843A-4128-8FDF-8E20743CBCFA/B6DCC811-01F9-4B30-B058-5DF128D9C5A0.jpg)

保存网页内容
------

「我看到了一个特别有意思的文章，我把文章链接发给你。」  
「你截个图发给我吧，上面显示注册才能看全文……」

看到一篇有趣的文章，不妨用长截图保存下来发给好友。[Picsew](https://sspai.com/app/Picsew) 是一款功能全面的 iOS 长截图工具，它支持将 Safari 中的网页保存为一张截图。

使用方法非常简单，在 Safari 中打开需要分享的网页，点击下方的分享按钮，找到带有 Picsew 图标的「网页快照」（如果没找到，可以在「更多」中开启），再点击右上角的保存即可将网页保存为长截图。

![](.evernote-content/3603B62E-843A-4128-8FDF-8E20743CBCFA/0E14F5C4-9603-4C4B-838E-E4947C9288BC.png)

分享电影对白
------

「昨晚又看了一遍《一代宗师》，里面有一段台词真好。」

追剧看电影时，碰到喜欢的台词，不妨用长截图保存下来。[PPIICC](https://sspai.com/app/PPIICC) 是 Android 平台上一款小巧实用的长截图工具，它可以帮助我们制作电影对白长截图。

将电影的对白截图保存下来，在 PPIICC 中导入截图，选择「拼接设置」，在拼接模式中选「电影台词拼接」，然后点击「拼接」，一张电影对白长截图就制作好了。

![](.evernote-content/3603B62E-843A-4128-8FDF-8E20743CBCFA/E49ADEC1-D8AF-4E3D-829B-4E76DAA26435.png)

发布长微博
-----

「我想发一个长微博，但微博的排版太不方便了，有什么办法可以解决吗？」

虽然微博开放了 140 字的限制，但当你真的希望发布一个内容很长的微博，可能还是长截图会比较方便。如果你想用图片制作一条长微博，可以试一下 [锤子便签](https://sspai.com/app/%E9%94%A4%E5%AD%90%E4%BE%BF%E7%AD%BE)。它是一款简单易用的便签工具，支持 Android、iOS 和 macOS 等多个平台。

制作长微博时，将文字和照片先输入到锤子便签中进行排版，然后点击右上角保存便签。再点击右上角的分享，选择「分享至新浪长微博」，锤子便签会帮你生成一个长截图，分享到微博即可。

![](.evernote-content/3603B62E-843A-4128-8FDF-8E20743CBCFA/D56AB06E-61F2-450A-B65D-A9A172D4B0A5.png)

值得一提的是，如果你习惯使用 Markdown 格式，你可以在设置中选择「默认排版格式 - Markdown 模式」来开启 Markdown 模式进行编辑。

长截图的用途当然不只上面这些，如果你想了解更多实用的长截图工具，快来看看少数派为你准备的「[妙用长截图](https://sspai.com/topic/167)」专题。
