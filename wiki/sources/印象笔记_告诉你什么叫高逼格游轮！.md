“海洋量子号”
=======

乘坐拥有最先进智能科技的游轮享受着加勒比海岛的阳光沙滩，霸道总裁的节奏有木有哇？！下面菜菜就带大家详细的八一八这海洋量子号游轮到底有何牛X之处？

**炫酷高科技让人眼花缭乱**

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/596CCAEC-AA1F-43E8-A3EA-8F2220C3D2AB.png)

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/65DF02FF-887F-4E7E-91F6-3FE2E835C02B.jpg)

图：互联网 via Internet

皇家加勒比游轮公司历时8年设计，5年倾力打造的“海洋量子号”（Quantum of Seas）于去年11月2日在英国南部开启处女航，终点站为美国，之后将在加勒比巡航。这艘游轮号称“世界上第一艘智能邮轮”，船上大规模的使用最先进的高科技，智能程度令人叹为观止。

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/C9DBC23A-7D14-4BF7-ACC1-0E371917EBA8.png)

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/E3110EA5-FC97-491A-9129-97E54576F85B.jpg)

图：互联网 via Internet

海洋量子号总共设有2110间客房，可容纳4180名游客。海洋量子号堪称全球游轮史上的又一次重大飞跃，船上大规模的使用智能科技，是高科技与奢华的创新结合之典范。游轮上每个房间都是“阳台房”，内舱房装有虚拟阳台，由大型落地窗式LED屏幕呈现大海或目的地的实景。土豪们还可以尝试复式套房：厨房，客厅，海景大阳台一应俱全。无论是待在套房还是内舱房，都会有比其他游轮房间更多的空间，智能的客房设计让居住变得更为舒坦。

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/69C7A812-57AE-4538-81F1-41D7893438CD.jpg)

图：互联网 via Internet

游轮的房门全部装有智能感应器，游客需用智能腕带轻轻“哔”一下，房门即刻开启。同时无论房间内还是甲板上都能享受海上最快的上网速度。在酒吧里（Bionic Bar）可以品尝机器人调制的美酒，而且机器人调酒师可不是照本宣科设置好的调酒程序，TA可以接受客人的制定呢。

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/431F2325-1251-4073-A9BC-43BBEF22E6BE.jpg)

图：互联网 via Internet

最跨时代的设计是名为“北极星”的360度观景舱，吊在离海面近300米处，在无障碍的观景舱内屏息静气，尽情欣赏游轮和大海的壮观景象吧。每次观景体验时间约15分钟。日出、日落这种浪漫时刻可以付费特别定制，想求婚的、给惊喜的，知道怎么做了吧？

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/E21D6E16-5360-44DC-BF92-67392F372D7D.png)

图：互联网 via Internet

船上还首创真空飞行体验，通过垂直风洞，产生空气流，游客可体验飞翔的感觉，尽享垂直降落的刺激和跳伞的快感。

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/5149ED59-5163-4080-A081-43C7B9878B0A.jpg)

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/310F9A2B-F6F8-4C64-A2F6-0469670F154F.jpg)

图：互联网 via Internet

Two 70是船上的一处大厅，白天这里是观景休闲厅，透过巨大的落地窗可将海上美景尽收眼底。而到华灯初上之时，落地窗摇身一变换成闪烁的LED大屏幕，炫目的灯光秀成为大厅内的表演最华丽的背景。

**18家餐厅吃货的天堂享受**

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/BAFD38E1-2DA7-4B81-B809-92A645EB052A.jpg)

图：互联网 via Internet

享受美食是游轮上的一大乐事。海洋量子号首推动态用餐(Dynamic Seating)概念，改变传统游轮用餐方式。船上拥有多达18家餐厅，日式，美式，意式，欧式应有尽有，没有你吃不到的，只有你没有想到的。船上詹姆斯意式餐厅由明星主厨詹姆斯 • 奥利弗主理，餐单也全部由他一人制定，菜肴全部选用当季最新鲜的时令食材精心烹饪而成。

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/7D5DF8BB-9625-4CEE-8BBD-215AF8798E22.png)

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/09659A30-F171-48BE-98DC-DEE5352EAC7D.png)

图：互联网 via Internet

喜欢猎奇的童鞋可以去装修奇幻，颇具Tim Burton电影搞怪风格的“仙境坊”餐厅，进入餐厅放佛进入了《爱丽丝梦游仙境》中的兔子洞中，各种穿越。餐厅的烹饪理念也非常前卫，运用风、冰、火、水、土壤和梦幻 这些元素，创意打造令人耳目一新的高分子菜肴。

**购物中心奢侈大牌免税购**

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/FDDF1A90-ACA2-45D8-B750-92DF5C6B3BBA.jpg)

图：互联网 via Internet

在海洋量子号的中心区域还建了一座购物中心Royal Esplanade！是的，人家是正儿八经的购物中心，里面各种商店，酒吧，餐厅，休闲设施神马都配备齐全。而且购物中心入驻的都是大牌哦，最棒的是这里买东西是免税的，非常划算！

**各种活动玩到停不下来**

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/076EDD58-535B-4A4F-B213-755E677D86D3.jpg)

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/665B3D23-AC23-4E03-8172-0141798C0DFB.jpg)

为了确保游人在船上的时光不会无聊，海洋量子号于是将能想到的所有娱乐活动统统都搬上了船！夜总会，卡拉OK，现场乐队表演，海滩主题表演，身临其境的3D电影院，百老汇经典歌剧，迪士尼卡通人物面对面，赌场，攀岩，SPA美容护理这些传统游轮有的海洋量子号统统都有。

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/D6BEC7A4-C587-4F7E-B261-491D6335AAB8.jpg)

![](.evernote-content/88C9A8E1-7646-4390-9733-3482D467E6E8/C2DCBACC-4BC1-4DA9-A882-BACD4A41A874.jpg)

图：互联网 via Internet

而是其他游轮没有的活动，海洋量子号也要有！船上设有户外赛道，冲浪模拟器，SeaPlex多功能运动场可以溜冰，欣赏冰上表演，开碰碰车等等。SeaPlex是海上最大的室内运动空间，在这里重温儿时流行的游戏，和体验从未玩过的项目。
