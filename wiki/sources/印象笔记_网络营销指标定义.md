指标定义
----

创建[自定义报告](http://support.google.com/analytics/bin/answer.py?answer=1033013)时，您可以使用下表作为指标参考。

您也可以访问下列链接，了解[维度定义](http://support.google.com/analytics/bin/answer.py?answer=1033861)和[有效的维度-指标对](http://support.google.com/analytics/bin/answer.py?answer=1033062)的相关信息。

|  |  |  |
| --- | --- | --- |
| 指标名称 | 类别 | 定义 |
| 跳出次数 | 网站使用情况 | 此字段标识通过所选维度对您的网站的单页访问次数。例如，如果您将此指标应用到“广告系列”维度，此字段会显示用户通过特定广告系列到达您网站的单页访问次数。 |
| 跳出率 | 网站使用情况 | 只浏览了一页的访问次数（即访问者从进入网页离开网站的访问次数）所占的百分比。 |
| 点击次数 | 网站使用情况 | 此字段
