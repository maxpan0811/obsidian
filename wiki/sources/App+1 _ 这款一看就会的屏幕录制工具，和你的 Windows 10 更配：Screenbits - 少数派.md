这款一看就会的屏幕录制工具，和你的 Windows 10 更配：Screenbits
==========================================

![](.evernote-content/FA83F996-0B2A-43EE-ABA8-FABFF194A069/375F02EC-6A20-46CA-A0A3-B63B593734D1)

App+1 | 这款一看就会的屏幕录制工具，和你的 Windows 10 更配：Screenbits

07月29日

你打开了一个上手有些复杂的软件，想给不熟悉的新人录制一个视频教程，按下 `Win+G` 想要调出 Windows 10 自带的 Xbox Game Bar，却没想到它依然不支持桌面环境的录制。

不想在后台开个游戏再进行桌面录制的你，只得转向第三方——OBS 很多主播都爱用，但是复杂如飞机仪表盘的界面实在是劝退；Camtasia 功能强大、界面设计更现代，但「贵」在价格……

好在微软商店里还有不少兼具美观与实用性的选择，比如 Screenbits。

和注重参数调教、恨不得将所有自定义设置都一并塞到我们手里的同类工具不同，Screenbits 从简洁的主界面、清爽的设置菜单到纯色、符合 Windows 系统应用设计风格的托盘图标，每一处细节都更像是一款 Windows 10 系统应用而非第三方录屏工具。

![](.evernote-content/FA83F996-0B2A-43EE-ABA8-FABFF194A069/FC47603B-D5C2-4776-9387-A99DF4D8259B)

![](.evernote-content/FA83F996-0B2A-43EE-ABA8-FABFF194A069/05C1EEF7-29DC-491F-AFDC-4423F49A307E)

可能不是最好看，但绝对不会让你觉得违和

这种与 Windows 系统风格高度一致的和谐感贯穿于 Screenbits 的每一个细节，甚至选择录屏区域时的控件样式都与 Windows 10 自带截图工具如出一辙。

![](.evernote-content/FA83F996-0B2A-43EE-ABA8-FABFF194A069/EB25B129-F202-4D62-A84A-B7D1CAF037FA)

Windows 10 自带截图控件

![](.evernote-content/FA83F996-0B2A-43EE-ABA8-FABFF194A069/7CB4C8B1-F8A0-4590-B211-5565648CF664)

Screenbits 的录制区域选择控件，第一次用的时候有种「宾至如归」的感觉

跟随系统设置的明、暗主题设计得也是十分自然，不管在哪种场景下使用都能以最纯粹的工具应用属性做到「不打扰」。

![](.evernote-content/FA83F996-0B2A-43EE-ABA8-FABFF194A069/E49A5B69-296D-48EE-B67D-68D1D6D6E8C3)

亮色与暗色主题

在宛如系统原生应用这一点上，Screenbits 做到的可不仅仅是皮毛。和 OBS 这类「巨人」产品相比，Screenbits 在功能和界面布局设计上也像是一款更接近普通用户的系统工具。主界面没有多余的嵌套菜单，设置界面与 PowerToys 类似、左右层级一一对应；实际使用时，大部分人的常见需求，比如分辨率、帧率、录制模式、音频来源等等，都能通过主界面进行临时调整或在设置菜单中快速定位并进行设定，如果你是视频博主、主播这类拥有专业收音设备的人群，Screenbits 主界面那个无需翻查、调试，方便得像系统音量管理面板的音频选择设置应该也能让你省时省心。

![](.evernote-content/FA83F996-0B2A-43EE-ABA8-FABFF194A069/0EB7B628-2A3A-46AC-9AF5-9444748F3F8D)

这种照顾到专业创作者的功能还有不少，包括分辨率缩放设置、水印、摄像头录制等等，在录制恐怖游戏流程的过程中，游戏玩家可以选择同时启用摄像头录制并选择摄像头窗口的所在位置。

![](.evernote-content/FA83F996-0B2A-43EE-ABA8-FABFF194A069/3F950821-4E2A-492E-8E6A-FDD7EE313F85)

对一般用户来说，Screenbits 细致到左、右键区分的点按视觉反馈，视频文件命名模板以及与 Xbox Game Bar 相似的快捷录屏文件管理功能应该也都能在制作视频教程和视频素材的过程中发挥不小的作用。

![](.evernote-content/FA83F996-0B2A-43EE-ABA8-FABFF194A069/A4C3C957-C97C-4F77-B654-1735B5B3E143)

Screenbits 直接将录屏文件管理整合在了主界面

![](.evernote-content/FA83F996-0B2A-43EE-ABA8-FABFF194A069/C0989CC9-898E-4C78-8301-7ACF5377E7CA)

录制教程时非常实用的点选、左右键颜色区分

总体来说 Screenbits 在功能和实际体验上都做到了「系统级」，在 Windows 平台录屏工具中也是学习成本最低的选择之一。不过考虑到它依然有完善空间的功能性和汉化，104 元的定价还是有些偏贵，推荐在最近这种折扣时期入手。
