用这 9 个写作小功能，提升你的码字幸福感
=====================

[![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/4AC09DCB-9C73-40B8-A99F-9424888FB514.jpg)](https://sspai.com/user/734351)

#### [北斗神锤](https://sspai.com/user/734351)

5小时前

33  [12](#)

从最开始微软的 Word 到后来流行的 Markdown 编辑器，再到具有协作功能的 Google Docs 等，我们已经接触了数量不少的写作工具，在少数派你可以找到 [几十篇关于它们的文章](https://sspai.com/search/article?q=markdown)。它们都是非常优秀的写作工具，也是很本分的写作工具，没有做出什么出格的功能。

这些工具多为国外的产品，其实国内也有不少的编辑器产品。有 WPS、石墨文档这样的，也有不少独立开发者的 MD 产品，还有诸多与微信公众号平台相关的看起来有些奇怪的产品。正如国内手机厂商对 Android 系统的个性化定制一样，国产软件总有一些小功能会让你为之动心。

下面我们就来介绍下这些有点特别的功能，主要是和写作相关的功能，不包括管理和发布类功能。

白噪音
---

有的人喜欢在写字的时候听音乐，也有的人喜欢听风声雨声。[字里行间](http://zi.com/)（网页版）就提供了播放白噪音的功能，而 [MarkEditor](http://zrey.com/app/markeditor) 更是直接内置了一个音乐播放器，可以播放本地音频文件。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/ED717BB9-FA00-47DE-9DAF-2962BEC6431C.png)

字里行间

阅读时间
----

字数统计几乎是文本编辑器必备的功能，有的还区分了词数和字数（比如 [TextNut](https://sspai.com/app/TextNut)）。[Bear](https://sspai.com/app/Bear) 更进一步，可以估计文章的阅读时间。而 [Ulysses](https://sspai.com/app/Ulysses) 是这个功能的王者，字数统计分为字符、不计空格、字、句、段落、页面六种，阅读时间分为慢、平均、快、朗读四种。

阅读时间可以说是一个很贴心的小功能，适合那些会在意读者阅读时间或者有领导的用户。试想一下，领导让你写一段 5 分钟的讲话，那么 5 分钟是多少字呢？

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/67F1FCB5-988F-4F08-9E65-F59D5C513E35.png)

Ulysses

专注模式
----

这是我本人很喜欢的一个功能，它也可能叫做「禅模式」「打字机模式」，主要作用是让你当前书写的内容显示更突出并且永远位于屏幕中间，现在大多数的写作工具都支持了。Ulysses 也是其中做的最好的，突出显示的内容可选择句、行、段，滚动方式可选择上、中、下、可变。功能丰富，设置起来也简单。相比之下，TextNut 就有些让人摸不着头脑。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/8D1A5063-7541-49B9-9D34-DB8DF652291F.png)

Ulysses

调整内容顺序
------

在我们写文章的时候，经常会碰到需要调整段落顺序的情况，一般只能选择剪切粘贴来完成。不少写作工具都提供了大纲视图，而 [Yu Writer](https://sspai.com/post/40449) 可以像在 Word 里一样用鼠标拖动来调整内容顺序。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/5B16C609-0DFB-422E-B4C4-90AE6A8E8E7F.png)

Yu Writer

也有的写作工具会在输入内容的时候就按照区块来输入，之后便可以随意调整顺序，如 [机核网](http://www.g-cores.com/) 编辑器、[美篇](https://www.meipian.cn/)、[Press](https://itunes.apple.com/cn/app/press-%E6%9C%8B%E5%8F%8B%E5%9C%88%E5%85%AC%E4%BC%97%E5%8F%B7%E5%9B%BE%E6%96%87%E6%B7%B7%E6%8E%92-%E9%95%BF%E5%BE%AE%E5%8D%9A%E7%BC%96%E8%BE%91%E5%99%A8/id1135351148?mt=8)。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/5018C684-4BB4-4BDD-9B0D-0F2C90E70C0E.png)

机械网，美篇

首行缩进
----

上学时候，我们写作文都被要求每一段的开头空两格，不知怎么到了电子屏上就不流行这种排版了。如果你喜欢首行缩进，可以考虑使用 MarkEditor 或 Ulysses，当然还有 Word 。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/12906BB8-4881-4BCA-8B6A-0AFE90776E32.png)

MarkEditor

插入图片
----

最早 Markdown 开始流行的时候，如果你的编辑器支持直接拖入和粘贴图片就能甩开其它编辑器一大截。不过到了今天仍然有个别反人类的家伙不支持这种插入图片的方式，而 MarkEditor 都支持直接从手机插图到电脑上了，后面要提到的微信公众平台编辑器插件也支持这个功能。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/F45424F8-9DBC-4191-BFA1-FECB635CC935.png)

图片编辑
----

Markdown 语法本身支持调整图片大小，也有编辑器为此做了图形化的界面。传统的非 Markdown 编辑器大多也支持调整图片大小，不足为奇，我们对图片编辑的需求也不仅限于此。

当下流行的图文混排方式是单张图片居中并与文字隔开，多数的编辑器也是这样做的。有些时候就很不方便了，逼迫我们把多张图片合并成一张。少数派网站的编辑器就支持了最多 3 张图片的合并图片功能，机核网编辑器提供的是多张图片轮播的图库功能。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/65A78324-1061-40D1-BA0E-1929DB548E12.png)

少数派，机核网

对插入的图片进行标注也是很常见的需求，国产小软件 Press 就支持了最简单的图片标注功能。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/EA2712FB-7125-43CE-9B31-349E84430066.png)

Press

图片素材库
-----

微信公众号平台的确是一个很好的平台，但也有不少让人诟病的地方，比如必须有题图才能发布。当下的风气也是这样，即使不是微信公众号平台，你也不好意思发布一篇没有图片的文章。

那么图片哪里找呢？少数派上介绍过不少相关的网站和软件，写作工具字里行间（手机版）则是内置了一个图片中心，你可以直接从里面挑选并插入文章里。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/9E422307-63B0-4A53-AD47-420CEBA5E389.png)

字里行间

微信公众号平台编辑器
----------

微信公众号平台本身的编辑器没什么好说的，要说的话就是难用！但由于其平台的优势，打它主意的厂商可不少。因为它们基本上只适用于微信公众号平台，所以这里我们把这些增强插件和第三方编辑器放在一起说。

[新媒体](http://xmt.cn/)、[壹伴](https://yiban.io/) 等增强插件在功能上几乎没有差别，通常是增加了一些流行的格式设置，比如两端缩进、文字间距，也有方便的手机传图和生成二维码功能，壹伴还有照顾工程师的插入代码功能。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/FDB4066D-AEE9-4896-89D3-751B8D6AD1C9.png)

壹伴

在文章输入区域左侧会有一个丰富的样式列表，点击即可生效，以此来实现一些更复杂的样式。当然有的样式过于花哨，这也取决于使用者的你。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/3762B52C-AC94-4FB0-9877-B56336FFB493.png)

它们自然也集成了图片中心，表情包、gif 动图、无版权图应有尽有。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/1D16B802-7AFD-4DD8-A878-5B47BBD87805.png)

壹伴也支持预计阅读时间

有了图片之后，对其进行编辑自然也不在话下，这些插件拥有完备的图片处理加设计功能。壹伴会打开一个新的网页进行编辑，新媒体则直接打开了由创客贴提供技术支持的弹窗，但只支持题图的制作。如果你在微信公众号里看到了两张相似的题图，很可能他们用了同一个插件。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/19085467-086C-41E7-B255-6BA45E4B9597.png)

壹伴

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/D1587251-27AF-411D-BD2B-14C1F79573DD.png)

新媒体

壹伴和新媒体都支持对图片进行裁剪、调整尺寸、设置圆角、添加边框和阴影。这是一个很讨喜的功能，给文章中的图片添加边框或者阴影呈现效果会更好，特别是对于各种截图来说（本文作者就偷懒没有做处理）。关于给图片添加阴影也可以看看这篇 [在 Windows 上为图片添加阴影，让截图更好看](https://sspai.com/post/41443)。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/03AF4F79-575D-4F75-B0A5-EF373A31756B.png)

壹伴

这里给大家提个醒，有些手机截图其实本身是有圆角的，贴到文章之后你会发现图片四个角是一小点的黑色或白色，这个时候给图片设置下圆角就能解决问题。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/3896FC0F-ECB5-4C74-B779-300F02D8DCB3.png)

新媒体支持一键给所有图片添加阴影和圆角，非常棒了。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/191B95F7-641E-4AC7-8951-F5C5B6D342D2.png)

新媒体

再如秀米、135 编辑器这样的第三方编辑器，功能上差不多但用起来就很糟糕了，有兴趣的读者可以自行试用。糟糕的原因是，这些工具可能更偏向于一个排版工具而不是写作工具。善加利用的话，它们可以让你的公众号写作体验好很多。

说到公众号写作，我派就有 [使用 Typora 一次性搞定公众号写作与排版](https://sspai.com/post/40524) 的经验分享。另外，MarkEditor 本身就支持输出为微信公众号格式（需要将图片放在图床上）。

![](.evernote-content/CE14E4A9-A6B8-4677-92AE-5D293AF9F480/B856DD80-E4DB-46AD-85C6-0533553D3290.png)

MarkEditor

结语
--

上面介绍的多为细小零碎的功能，对于见多识广又要求高的少数派读者来说，这些功能可能就是把同类软件比下去的重要理由。

这里说点题外话，你会不会觉得上面有些功能超过了一个文本编辑器的边界？在这里，我们可以笼统地将软件分为专而精、大而全两类，国外多是专而精，国内多是大而全。我们似乎是更加偏爱那些小而美、简洁优雅的软件，但用久了你会不会希望它能多加一点功能？

越来越多你常用的软件随着更新开始在功能上有交叉，你是「不用就好了」还是会感到困扰？欢迎在评论里说说你的看法。
