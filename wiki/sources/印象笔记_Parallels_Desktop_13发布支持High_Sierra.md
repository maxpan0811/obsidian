# Parallels Desktop 13发布支持High Sierra

---

威锋网讯，Parallels Desktop 13 for Mac 在本周二已经正式发布，这一 Windows 虚拟软件的第 13 个版本带来了许多的新特性，包括了对 macOS High Sierra 和 Windows 10 Fall Creators Update 的支持。

![](.evernote-content/6D10D3E3-23EF-4EA6-945C-BDAD92085AA8/D2CC9725-A365-49EF-A591-5CC2D2C3AC28.jpg)

本次升级还带来了 Touch Bar 的支持，这使兼容的 MacBook Pro 用户能够使用 Microsoft Outlook、Excel 和 PowerPoint 的 Touch Bar 功能，以及 Windows 开始菜单和桌面的其它整合，包括 Cortana，任务视图和任务栏固定元素。Touch Bar Wizard 还允许用户自定义 Touch Bar 并添加自己最喜欢的 Windows 应用。

![](.evernote-content/6D10D3E3-23EF-4EA6-945C-BDAD92085AA8/387543E3-DE8F-4309-B985-6CF1278F02BF.jpg)

新的动态分辨率支持意味着用户可以更改其 Windows 会话的窗口大小，启动和重启的速度也因此变得更快更流畅。同时，增强对 Retina 屏幕的支持或能够在 Mac 屏幕上看到 Windows 应用有着更好的拉伸效果。

新的画中画模式旨在让用户轻松地监视虚拟机，同时支持即将推出的 Windows 10 People Bar，这一功能允许用户在 Windows 任务栏或 Mac Dock 中查看最近的联系人。

具体来说，与之前的版本相比，Parallels 声称新版在 Windows 文件和文档访问速度方面快 47%，通过 USB 3.0 和 Thunderbolt SSD 设备会有着更快的文件传输速度，以及使用 Parallels Desktop for Mac Pro Edition 的每个虚拟机可拥有高达 32 vCPU 和 128GB vRAM ，更具有为 macOS High Sierra（10.13）和 Windows 10 Fall Creators Update 优化的所有新版本的代码。

Parallels Desktop 13 for Mac 售价 79.99 美元，现有用户可以以 49.99 美元的价格升级到最新版本。
