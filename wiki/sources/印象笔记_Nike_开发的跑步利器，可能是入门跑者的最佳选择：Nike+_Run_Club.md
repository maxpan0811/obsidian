跑步已然成为大多数人首选的运动方式，除了拥有一双合适的跑鞋，找到一款好用的跑步 App 也成了许多跑者的需求。从最基本的跑步记录功能出发，出身运动名门的 [Nike+ Run Club](https://itunes.apple.com/cn/app/nike-run-club/id387771637?mt=8&uo=4&at=10lJSw) 因其功能上的纯粹和恰到好处，可以成为入门跑者的首选。

核心功能：跑步记录
---------

作为一个跑步 App，最基本也是最核心的功能便是对跑步的记录。一打开 NRC（即 Nike+ Run Club），默认的初始位置便是位于 App 功能栏最中心的「即刻开跑」界面。
