七宝外国语PK协和双语尚音学校
---------------

*2014-11-21*[学而思上海幼升小指导](http://mp.weixin.qq.com/s?__biz=MzA5MDExMzIzNg==&mid=201554810&idx=1&sn=76f827eb1eaddee03621d62d753ef5b7&mpshare=1&scene=1&srcid=0111T3WOFzaA59Wk77Pwf6OK##)

![](.evernote-content/801FAA34-18CB-478D-B16E-991E45AF7BEF/1274584D-5B3B-43C3-B68F-AEFC34853914.jpg)

梦想要实现，首先要敢想！2015幼升小的家长们，你还在纠结为孩子选择哪所学校迟迟定不下目标吗？坚定目标，勇往直前。今天我们来看看上海七宝外国语小学PK上海协和双语尚音学校吧！！

|  |  |  |
| --- | --- | --- |
| **学校** | 上海市七宝外国语小学 | 上海协和双语尚音学校 |
| **地理位置** | 闵行区新镇路79号 | 闵行区龙茗路185号 |
| **招生人数** | 优先闵行区 | 全市 |
| **招生人数** | 6个班 | 240人左右，4个国内班，每班约33人； 国际教育课程实验班90人左右； |
| **收费情况** | 8500元/学期 | 精英班：7400元/学期+必修名家课程1600元/学期=9000元/学期 国际教育课程实验班：40500元/学期 |
| **硬件设施** | 学校设施齐全，配置计算机房、图书馆、语音室、综合实验室、音乐室、舞蹈室、多功能厅。 有校车12条路线； | 有住宿，有食堂，伙食很好； 电脑房、音乐室、美术室、舞蹈房、多媒体教室专用教室十多间，音乐厅、琴房二十多间；学生公寓4人一间，套间内卫生间设施齐全，备有空调； |
| **课程特色** | 特色英语：  一年级：  魔盒+佳音+牛津 kid's story 牛津，佳音kid's story一二年级针对口语。  三年级后：  自编教材，定期发故事，牛津作为辅助教材，重点学佳音，四年级后会加SBS的内容；另外还有外阅读课。  二年级用3L考试分班，超前的孩子会上3L。  四年级：上了解英语世界各国文化风俗的课。 有很多需要家长一起和孩子完成的活动，比如做节日卡、阅读等等 | 国内班： 特色按名家课程分班：书法芭蕾、画画、二胡古筝、京剧 英语：牛津、朗文 英语教授自然、美术 数学：自编数学习题集 每周有月考，一张试卷 国际班：  加拿大小学BC课程，美国story towm双语教授数学 |
| **师资力量** | 师资优质（聘请优秀的外语教师和称职的外籍教师） | 上海籍教师占75%以上，外籍教师5人，教师中拥有大学本科50%，大专48%，70%拥有中高级职称，小学教师平均年龄35岁，实行全天坐班制，住宿学生有生活老师负责管理 |
| **面试题** | 表达、理解（英文朗读、阅读短文） 肢体（跳槽） 计算（20以内的加减法） 思维、记忆力（判断正误、回忆图片） | **家长：**  看宣传片，回答问题。 例如：为什么选择尚音？对学习不好是老师的问题怎么看待？课堂知识与兴趣之间什么更关心？是否有亲子阅读？如何培养阅读习惯？爸妈在家是怎么分工的？ **小朋友：** 肢体（跑400米） 计算（规定时间20以内的加减法口算） 动手、思维（搭积木） |
| **小升初** | 14年：233个学生，180个重点，上外附中5个、华育20，上宝48，文莱46，世外25个； | 直升初中部，少部分华育、上宝 |
| **其他评价** | 老师非常的严格，别想素质教育，但是活动也是很多的，小朋友很开心，也有压力大不适应而转学的 | 学校比较注重家庭的整体素质，老师进度都蛮快的，家长需要在平时多督促孩子才行，小朋友每天上学都很开心 |

爸爸妈妈们，为了孩子的升学之路，赶快准备起来吧，笨鸟先飞，快人一步！
