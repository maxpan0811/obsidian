我所认识使用 Evernote 的朋友中，绝大多数人都喜欢 Evernote 的 Web Clipper 浏览器外挂。

也就是那个可以帮我们在电脑浏览器、手机浏览器中自动截取网页图文全文的便利功能。尤其是职场上常常需要研究新知识、新资料的工作者，或是乐爱生活与玩乐的旅行家、美食家，对他们来说，这就是收集资料最快速简便的一个方式。

前两天，Evernote 进一步更新了他的 Evernote Web Clipper 浏览器外挂，其实基本上功能没有太大的改变。但是我想趁着这个机会，分享一下我自己，以及整理我所认识、采访过的 Evernote 使用者们 —— 我们如何利用 Evernote 这个好用功能来收集线上资料的经验，包含资料收集的心法，以及重点功能的操作方法。

**1. 用来剪贴什么？**

在你的 Google Chrome 、Safari 等浏览器中安装［Evernote Web Clipper］套件，接下来只要看到需要的线上资料，按下浏览器上的［Evernote］按钮，就能截取网页全文。

我自己实际使用上遇过的剪贴情境有下面几个：

* 网页全文（利用简化文章与荧光笔画重点）
* 邮件与附加档案（可挑选段落）
* 订票（利用截取文章）
* 购物清单（利用截取文章）
* YouTube 、Linkedin 、Amazon（截取特殊格式网页）
* 线上图片、PDF、画面（利用屏幕截取与注解功能）

大家最常用的应该就是单纯的想要截取网页全文吧？我也是，这时候我访问过的大多数 Evernote 使用者，都会使用［简化文章］功能，把网页的外框去掉，保留文章的标题与内文，甚至先把看到的重点画上萤光笔，这样在笔记中更好阅读。

![](.evernote-content/16F4B1BC-CEA6-4E61-9D2B-34E10F62AE2B/3B931A43-6AD1-4648-BCFE-63DD3C72A488)

而其实在很久以前的 Evernote Web Clipper 中，就支援可以截取 Gmail 网页邮件中的「会话段落」、「附加档案」，这是我常常使用的功能，在邮件页面按下 Evernote 截取，就可以勾选要剪贴哪一段的邮件段落。

我通常会把邮件中最重要的任务、资料段落截取，附带邮件的附加档案，放入 Evernote 中变成参考资料库，或是加上提醒，变成待办任务。

![](.evernote-content/16F4B1BC-CEA6-4E61-9D2B-34E10F62AE2B/A30AD99A-24BC-4D07-A687-0FBE6B469439)

有时候我也会用 Evernote Web Clipper 来剪贴自己订票成功的火车票、旅馆、电影票页面，我访问过的许多使用者中，也有许多人喜欢用来剪贴自己的购物清单，并且还有不少朋友采用我分享过的「欲望清单管理方法」。

这时候，我们会选择用［截取文章］这个功能来截取，因为简化文章会让订票、购物页面的格式跑掉，但格式对这类票劵网页来说是很重要的，而［截取文章］功能可以保留网页内这个指定段落的完整格式。

![](.evernote-content/16F4B1BC-CEA6-4E61-9D2B-34E10F62AE2B/54F89091-B67B-4003-A3D7-412D09B47B2C)

另外如果想要抓取网页中的 PDF、图片，或是把某个活动、某个网页「证据」抓取下来时，有图有真相，这时候我会使用［屏幕截图］的功能，直接把网页的画面抓成图片。

而且如果这时候是要跟工作伙伴沟通，还能在图片上即时做注解说明，传给伙伴看，就更加清楚。

![](.evernote-content/16F4B1BC-CEA6-4E61-9D2B-34E10F62AE2B/A68EA6F8-62C1-4565-A5AF-3A7D796A5B3E)

**2. 移动时在手机上自动截取网页全文**

我自己最近常常使用的情境则是在手机上用 Evernote 截取从 Facebook 、Zite 等新闻服务中看到的需要的网页全文，方便我事后回到电脑上进行整理。

从 2014 年 9 月更新的 Evernote Android 版开始，以及后来支持 iOS 8 的 Evernote iPhone 版，都可以在手机上直接进行网页全文的截取。

方法很简单，以我最常使用的 Evernote Android 版为例，只要在手机浏览器右上方选单选择把网页分享到 Evernote ，这时候 Evernote 就会自动帮我们抓取网页的图文内容。

![](.evernote-content/16F4B1BC-CEA6-4E61-9D2B-34E10F62AE2B/24C9689E-C2C9-4ECC-B7E4-D2F64AE1934C)

**3. 暂存区的收集概念**

无论是电脑上或手机上的 Evernote 网页截取功能，都可以在截取时选择要放到哪一个记事本分类中。这当然很方便，但是有时候临时没有时间去管怎么分类，或是当下还没想清楚这个资料要怎么分类时，要怎么做？

我访问了许多朋友后，发现大家都跟我一样有一个很有效的做法，就是建立一个［暂存区］的记事本（或者你想怎么命名都可以），大多时候先把不确定但觉得有用的资料先收集到暂存区，等到有空时，再到暂存区进行整理。

或者有的朋友也说，就算不整理也没关系，反正收集来的资料都是存在 Evernote 这个统一资料库，以后统一搜寻去找回来即可。

**4. 收集时可以怎么加标签？**

Evernote 中的分类方法不只有记事本分类，还有「标签分类」，一则记事上可以加上多个标签，比记事本分类更有弹性。

那么，在利用网页截取工具做资料收集时，标签可以派上什么用场呢？关于这一点，每个人的作法可能都不一样，而当大家询问我会怎么做时，我通常回答：「我的一个基本的观念是，标签是为了自己未来实际的行动而加，而非为了资料的分类而加」。

例如我会把之后真的想要立即使用、需要使用的资料，用 GTD 的概念加上像是［今日事］、［明日事］的标签，这样我每天或每周回顾自己收集的资料时，可以从这两个标签找到那些我随手收集，但可能忘记去执行的资料。

或者就像我前面提到的收集购物网页清单时，用欲望管理的概念做标签分类，之后就能用标签去找回想要买的、需要买的东西。

我的一个很重要的想法是：「笔记是为了未来的使用而记录，如果收集的资料没有指向后续行动，那么这样的资料收集意义不大。」所以加上这些有行动意义的标签，就是让我记得去对这些资料产生行动。

![](.evernote-content/16F4B1BC-CEA6-4E61-9D2B-34E10F62AE2B/AA5260F6-F962-4024-9193-E7CA9803A0EE)

**5. 收集当下写出重要的心得笔记或使用目的**

另外我觉得电脑版的 Evernote Web Clipper 工具有一个很棒的特色，就是截取网页时有一个「注解」的文字输入栏位，可以让我们在收集时立刻在笔记前面加上自己的想法。

例如我读完一篇文章觉得很棒想要收集，用 Evernote 截取网页时我会立刻在注解中写在我读完后的简单心得。或者例如我看到一篇觉得以后工作用得到的文章，我用 Evernote 截取网页时也会在注解中写下我想要用在哪个工作上？怎么用？

我的经验告诉我，这个简单的「多写下几句自己的想法」的步骤，是资料收集法的窍门，会让以后这则资料更有用，他会带来几个具体影响：「多写下一些自己的想法，就多增加一个关键字，以后更容易用不同关键字找到这则资料；写下为什么自己想要收集它的用途，以后看到这则资料时不会忘记它要用来干嘛（相信我，这件事真的很容易忘记），可以回忆起来并引发具体行动。」

**6. 帮收集的资料加上时间提醒的使用时机**

Evernote 有时间提醒功能，这个功能也可以跟我们收集的资料做有效的搭配。

我自己最常使用的两个时机是「提醒订票时间」、「延迟邮件任务」。当我在网页上完成订票，并且用 Evernote 截取订票页面后，我会立刻加上一个时间提醒，通常是这个票劵要出发的前一天，让我自己记得。

而且当这则订票笔记弹出提醒后，我会让它留在 Evernote 的待办清单最上方，这样实际到达旅馆、机场、演唱会现场时，我打开 Evernote 就能立刻看到它。

**7. 相关笔记的搜寻与链接**

当为了一个专案、一个学习目标在收集资料时，一定会剪贴许多相关的资料，这时候要怎么处理比较好呢？

其实我访问过的许多使用 Evernote 的朋友都说：虽然会想整理相关笔记，但现实面一个人很难有那么多时间花在整理资料，所以我们大多数时候都是「就放着吧！」虽然可能相关的资料会散落在笔记不同地方，但是以后只要用关键字搜寻，就能一起找回来，也不一定要特别花时间整理。

但是如果是「特别重要或专业的资料」，我自己的一个做法是，一边找一边把自己笔记里的「小维基百科」建立起来，以在研究一个专案为例，我会在找到非常重要的资料时，在截取网页的同时，把这一则笔记用「记事链接」的方式链接到这个专案的一则总目录笔记中。

这样无论资料收集到什么程度，我只要优先看这个专案的总目录笔记，起码都能看到最新状态的最重要资料清单。

**8. 打开 Google 搜寻辅助**

要研究资料总免不了上 Google 搜寻，为了避免「其实我之前收集过相关资料，但自己忘记了又再找一次」，所以我会打开 Evernote Web Clipper 浏览器套件里的设定，开启［在搜寻时显示 Evernote 相关结果］的功能。

这样当我傻傻地在 Google 搜寻时，或许就会注意到原来笔记里有收集过相关的资料。

也因为这样的情形发生了很多次，当我的资料库已经有六千多则笔记时，我早就已经不记得自己收集过什么，这时候我会运用「第二大脑」的概念，在需要资料时先去 Evernote 搜寻，如果找不到，再到 Google 去补充弹药，然后把有用的弹药用网页截取收集到 Evernote，让第二大脑之后愈来愈有用。

![](.evernote-content/16F4B1BC-CEA6-4E61-9D2B-34E10F62AE2B/91AB83FF-438E-4DE3-A6B5-64A36D81CFC1)

**9. 复习与阅读**

最后，我想分享的是在我采访许多善于用 Evernote 收集资料的朋友的经验后，大家都有一个共同的信念，就是收集的目的是为了使用资料，如果没有好的方法去活用收集到的资料，那么还不如不要花时间收集。

而这个好的方法除了上述提到的几点外，我也补充我从 Bingje Yang 这位使用者身上听到的宝贵经验。他是一位收集了上万则笔记的 Evernote 使用者，但是他有一个独特的习惯是每一天会花两个小时去阅读自己在 Evernote 中截取的文章，把 Evernote 当作他的睡前读物。

我询问他这样做对他来说的意义是什么？他说因为收集这些他自己感兴趣的网络好文章，对他来说就是让自己学习的好管道，每一个人生阶段他透过这样的收集与阅读，来把一篇篇文章真正内化成自己的思考与知识。阅读本身是一种乐趣，也是让自己真正吸收，这是他使用 Evernote Web Clipper 的目的。

这也是在一路探访不同 Evernote 收集资料达人的经验时，最让我震撼的心法之一，希望也对大家有所启发。
