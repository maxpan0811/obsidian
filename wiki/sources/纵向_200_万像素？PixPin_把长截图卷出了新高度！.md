# 纵向 200 万像素？PixPin 把长截图卷出了新高度！

原文链接: https://mp.weixin.qq.com/s?__biz=MzUxNzk1NjU0MQ==&mid=224757...

软件界面内容太长，截一张放不下；几张参考图要对比，手动拖来拖去总是对不齐；给同事讲操作流程，单纯录屏又很难表达清楚。

显然，真正影响效率的，不只是截图 / 录屏本身，更是之后的事情：怎么拼、怎么标、怎么贴在桌面上对照；在录屏时，怎么把操作过程展示明白也很关键。

最近 PixPin 发布了一版重磅更新，几个新功能都挺实用，今天就简单聊聊。

PixPin 是一款适用于 Windows / Mac 的截图贴图工具。

它支持自定义区域截图、窗口截图，也能做垂直 / 水平长截图。截完图以后，可以直接进行箭头、文字、马赛克、图形等标注。

如果只是临时参考某张图，也可以把截图变成贴图，置顶放在屏幕上，对照表格、网页、聊天记录、设计稿时都很方便。

另外，PixPin 也支持 OCR 文本识别和翻译，图片里的文字不用再手动敲一遍。

以前截长网页、长表格、后台页面，经常要分成好几张，再手动拼接，发出去还得确认排序，十分麻烦。

这次更新中，PixPin 带来了超长截图模式，更适合整理超长网页、长表格、聊天记录和文档页面。面对超长内容时，可以减少分段截图与手动拼接的操作。

▲ 实测可达数十万像素，更长的界面没时间测了……

需要注意的是，超长截图生成的图片高度可能很夸张，部分图片编辑工具存在高度限制，不一定能正常打开。

但对于产品后台、订单列表、网页教程、聊天记录这类内容来说，能一次讲清楚，就少很多来回解释。

如果你经常把几张截图贴在桌面上对比，这个功能会很有用。

新版 PixPin 支持多贴图自动对齐。多选贴图后，可以直接通过右键菜单使用自动对齐，不需要一张张手动拖到差不多的位置。

对齐时，会以当前右键选中的贴图为基准，移动其他贴图的位置。

整理参考图、摆放流程图、对比改版前后、查看几张素材细节时，画面会更整齐，也更容易看出差异。

做教程、录问题反馈、给同事演示操作时，可以一边录屏，一边把摄像头画面放在角落里。

摄像头内容支持矩形或圆形显示。切换为圆形显示时，还可以按住 Shift + 鼠标左键拖动，调整圆形内的显示区域。

这个功能不一定适合所有人，但对于老师、产品经理、运营、客服、做教程视频的人来说，会比单纯录屏更好地表达你的想法。

新版还支持 30FPS 录制，帧率选项达到了 5 种，充分满足不同需求。

除了上面几个明显功能，PixPin 这次也补了不少细节。

比如保存为 PDF 时，现在可自定义页边距和分页选项；文字标注可附带箭头，指示方向；识别二维码时，可以直接显示内容。
