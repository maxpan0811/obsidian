前言
--

Karabiner 全称 Karabiner-Elements，是 macOS 上的一款功能强劲的改键工具。它可以将键盘上的按键映射成其他一个或多个按键组合。然而，Karabiner 使用 JSON 作为配置文件，配置繁琐且不易理解，给上手带来了很高的门槛。

于是，有人专门为其制作了一个工具「Goku」，借助 Goku 可以方便地为 Karabiner 进行配置。
