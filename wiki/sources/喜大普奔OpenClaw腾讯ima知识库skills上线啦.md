# 喜大普奔！OpenClaw🦞腾讯ima知识库skills上线啦！

OriginalDracoVibeCoding Draco正在VibeCoding

 

打开手机或者Mac/PC端ima首页，单击“灵感随时claw”：

![](attachments/cfe59c1f510ee9c3.png)

进入ima skills专区：

![](attachments/5b129a55db226d4f.png)

单击“安装引导”第1步中的“一键复制”按钮，将其中的内容复制出来；

请安装 ima 技能 下载地址：https://app-dl.ima.qq.com/skills/ima-skills-1.1.2.zip API Key 获取：https://ima.qq.com/agent-interface

粘贴给你的OpenClaw🦞 (我的🦞是腾讯云轻量应用服务器OpenClaw镜像版+企微)：

![](attachments/f95a32aa1341ffdb.png)

然后，单击第2步中的“获取API Key”，会弹出一个API Key的窗口（请将内容复制出来并粘贴给🦞，这个弹窗只会出现一次，如果你没记住，就只能删掉这个API Key并重新生成了）：

![](attachments/ac3087391ab76254.png)

打通API后，🦞向我汇报如下：

我直接帮你把 IMA 凭证写到本机配置文件里，然后做一次最小可用性检查。
配好了，而且已经验通。
我做了两件事：
 ⦁已将凭证写入本机配置文件
 ⦁成功调用了 IMA OpenAPI
 ⦁能正常返回你的知识库列表
我验证到的结果里，前几个知识库包括：
 ⦁个人知识库
 ⦁Vide Coder 自习室
 ⦁Brain Food
 ⦁一人公司
 ⦁Projects
所以现在  ⁠ima-skill⁠  已经是可用状态了。
你接下来可以直接让我做这类事情：
 ⦁搜一下你的 IMA 知识库里有没有某个主题
 ⦁把一个网页/微信文章加入某个知识库
 ⦁把文件上传到某个知识库
 ⦁新建或追加一篇笔记
比如你现在就可以直接说：
 ⦁“帮我看看 IMA 里有哪些知识库”
 ⦁“把这篇网页加入个人知识库”
 ⦁“帮我记一篇笔记到 IMA”
 ⦁“搜索 IMA 里有没有关于 OpenClaw 的内容”

官方宣称的功能和场景如下：

![](attachments/81a2ed68a8f03dcf.png)![](attachments/9c52f68cb5ff201e.png)

有了OpenClaw🦞这个通路，似乎可以考虑在很多场景中把ima当Obsidian来用了...

请帮我调研“汗青堂”历史系列丛书究竟有哪些，然后在IMA“书荒终结者”这个知识库中创建一个文件，并且按照“汗青堂”丛书的序号顺序，将序号和对应的书名写入
