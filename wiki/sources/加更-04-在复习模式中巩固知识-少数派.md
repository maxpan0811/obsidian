在系列的 [第一篇](https://sspai.com/post/65532) 讨论过，学习资料要经过**阅读、理解和内化**三个过程，才能被我们的大脑真正吸收，成为所谓的「知识」。

前两篇连载中，介绍了通过文档模式来阅读资料，通过学习模式来理解知识点。想要完成知识的「内化」，就需要在 MarginNote 的复习模式中，用抽认卡来巩固知识。

## 认识抽认卡

在正式开始进入 MarginNote 中的复习模式前，我们先来认识下抽认卡。

抽认卡是广为世界各地学习者所认同的一种记忆方法。它其实一点也不神秘：将问题和提示写在卡片的正面，将答案写在反面。学习时首先将所有的卡片都循环记忆几次，然后开始以自我检测的方式随机抽出卡片，根据卡片上的题目进行回答。将错误和正确的进行分组，下一个轮次就只记忆错误过的卡片。如此循环，直到所有卡片记忆完成。

![](attachments/6b1e8103e8ab9178.png)

遗忘曲线 / 图片来自 [Wikipedia](https://sspai.com/link?target=https%3A%2F%2Fwww.wikipedia.org%2F)

看似简单的方法背后，其实恰好符合人脑的记忆规律。[遗忘曲线](https://sspai.com/link?target=https%3A%2F%2Fzh.wikipedia.org%2Fwiki%2F%25E9%2581%2597%25E5%25BF%2598%25E6%259B%25B2%25E7%25BA%25BF) 显示，仅仅二十分钟后你就会遗忘大约一半的内容。所以通过抽认卡的学习模式，可以逐渐将短时记忆，逐渐强化后成为长期记忆。

而 MarginNote 复习模式中的功能，就是专门为抽认卡学习法而准备的。

## 准备卡片组

```
本节所涉及的功能：

* 添加到复习
* 新建卡片组
```

实践抽认卡学习法的第一步当然是将需要记忆的内容，制作成卡片。

在文档模式和学习模式中，我们已经积累了不少笔记卡片。在 MarginNote 中，这些笔记卡片其实就可以直接当做抽认卡使用。

![](attachments/8e1c5c4cca103125.gif)

添加卡片

在学习模式的文档或者大纲视图中，选择笔记卡片，在工具条中点击「添加到复习」按钮，就会出现窗口，选择抽认卡片组。如果还没有创建过卡片组，就需要新建一个。

需要注意的是，卡片组与文档模式中的文件夹，学习模式中的脑图笔记本都没有任何关联，所以任何文档中的任意笔记卡片，都可以添加到任意卡片组中。

添加卡片到卡片组之后，就可以进入复习模式查看卡片。可以看到，笔记卡片的标题被当做卡片正面，即标题；而笔记卡片的评论，就成为了卡片反面，即笔记内容。

![](attachments/1b8e0a083edaca58.gif)

在组内添加卡片

在复习模式的界面中，也可以继续将笔记卡片添加进卡片组。点击「+」，就会罗列出所有还未进入到卡片组的笔记卡片，可以通过标签或颜色筛选，当然也可以直接通过搜索找到卡片。

![](attachments/d75483bc17e69f2f.jpg)

编辑卡片的选项

作为抽认卡的笔记卡片，还可以针对性地做一些编辑。点击「编辑」，即可添加卡片背面上的内容。以我自己的使用经验为例，我会给不熟悉的单词添加上释义，免得学习的时候反复去查字典。

## 开始复习

做好准备工作后，我们就可以真正开始使用抽认卡复习！

点击工具栏上的「开始复习」，便会进入到复习界面，以排好的顺序开始展示卡片。抽认卡的学习通常分为学习和复习两个阶段，在第一阶段时，我会直接点击「翻转」来显示答案，尝试记忆答案的内容。在记忆的过程中，可以借助卡片在脑图和文档中的上下文，帮助我们进行联想记忆，相比直接死记硬背，这样无疑是更高效的。

![](attachments/84829df1a630fbaf.gif)

第一次复习

在复习界面的下方有三个选项，是为了让我们自己对卡片内容的熟悉度进行一个评估。前文中我们介绍过，抽认卡的原理就是通过有规律的复习，将短期记忆强化为长期记忆。在 MarginNote 中，对于几乎无法复述的卡片，我们标识它为「难」，经过一轮学习后，两个小时后再次尝试复述内容；对于记得住大部分内容的卡片，我们可以标为「中」，明天再来一轮复习；对于已经完全掌握的卡片，就可以标为「易」，两天之后再次检查。

在第一遍学习卡片时，其实我们是在记忆卡片内容，想必绝大部分卡片是会被标为「难」，此时也不必气馁，MarginNote 会在一轮过后，提醒你是否要从头再来一遍。在这一遍中，可以重新尝试一次复述卡片内容，得出来的结果自然也会比第一遍更符合实际掌握的程度。

![](attachments/6683dfe767dab47f.jpg)

复习完成提示窗口

在经过学习阶段后，就是利用抽认卡来复习。在复习的时候，我们就不用再针对每一张卡片都花时间，而是把时间集中在自己尚未记忆的卡片上。在复习模式的界面上，会显示每一次复习的日期和所花的时间，以及下一次建议复习的时间。

当然你可以用卡片筛选的功能，直接挑出「今天到期」的卡片，开始复习，也可以根据上次的评分来选择卡片进行复习。

![](attachments/46df88948e29b686.jpg)

根据卡片属性筛选

经过一段时间反复的复习，你的卡组里被标记为「难」的卡片会越来越少，原来需要靠意志力硬背的内容，也像是在打游戏过关一样，先解决简单的卡片，再重点攻克难的，直到完全掌握，这正是抽认卡的魅力。

MarginNote 提供的抽认卡已经能满足大部分学习者的需求，如果你想调整复习间隔，或是丰富更多卡片的内容，MarginNote 还支持将卡片导出到 Anki。Anki 是一款免费、强大的抽认卡工具，几乎可以满足你对抽认卡的一切需求（如果你想了解 Anki 的使用，可以阅读 [这篇文章](https://sspai.com/post/39951)）。在卡组界面中，点击「导出备份」，选择「导出到 Anki」，在用 Anki 打开导出的文件，就可以在 Anki 中编辑使用卡组。

## 小结

经过复习模式的锤炼，我们终于走完了「阅读 - 理解 - 内化」三步，将印刷在书本上的文字，转换成了我们脑中宝贵的知识。至此，MarginNote 的连载也要告一段落。

当然，在这个系列没有介绍完 MarginNote 内所有的功能，例如视频的播放和批注，以及在学习模式中的「回忆」功能。不过对于 MarginNote 的主要使用流程，相信你已经有所了解，知道该如何用 MarginNote 去消化吸收海量的学习资料。

希望 MarginNote 能对你的学习之旅中有所帮助！

## One more thing：插件的安装与使用

MarginNote 是一款专为高效学习者准备的利器，它的高效不仅体现在「文档-学习-复习」三种模式的递进，提高了你吸收知识的效率，在工具操作上，它也希望是高效的。为此，MarginNote 开发了插件接口，让使用者可以安装各种第三方插件，提高使用效率。

点击菜单中的「插件」，就会弹出插件的管理界面。如果你之前没有装过插件，可以点击下方的链接，在 MarginNote 论坛中寻找插件。

![](attachments/eae44e74a03c3eb6.gif)

安装插件

MarginNote 的插件安装非常简单，以 DeepL 动态翻译器为例，在论坛上找到下载链接，下载后缀为 mnaddon 的插件文件，双击文件即可自动启动 MarginNote 进行安装。安装完成后在插件管理器中，就能进行关闭或者卸载的操作。

在 MarginNote 上已有非常丰富的第三方插件资源，这里推荐三个笔者觉得比较实用的插件，其他的插件各位可以根据自己的需求进行安装。

### DeepL 实时选句动态翻译器

虽然 MarginNote 的研究视图中已经提供了谷歌翻译，但是 [DeepL 翻译器插件](https://sspai.com/link?target=https%3A%2F%2Fbbs.marginnote.cn%2Ft%2Ftopic%2F6113%2F3) 还是非常值得安装。相比谷歌翻译，（以下仅是我的主观感受）DeepL 的英译中结果更加符合中文语言习惯，例如一些倒装句或者主从句，都能翻译成我们熟悉的语序，读起来没有明显的翻译感。

![](attachments/aaa548d15539a4dd.jpg)

DeepL 翻译效果 / 图片来自 [MarginNote](https://sspai.com/link?target=https%3A%2F%2Fbbs.marginnote.cn%2Ft%2Ftopic%2F6113)

### SwitchTitle 摘录标题手动转化

在学习模式中进行摘录时，默认情况下摘录的内容会作为笔记的评论，而不是标题。而在一些场景中，例如摘录专业名词词组时，我们会希望把它直接当作笔记的标题，[SwitchTitle](https://sspai.com/link?target=https%3A%2F%2Fbbs.marginnote.cn%2Ft%2Ftopic%2F6764) 这个插件就能解决这个非常细节的需求。

![](attachments/a6d891f13146b940.gif)

摘录标题手动转化

SwitchTitle 提供三种模式：

- 选择文本作为标题；
- 选择文本作为笔记评论；
- 选择文本同时作为标题和笔记评论。

这样无论你希望摘录的文本作为标题或者内容，都可以灵活地切换。

### MarkDown

MarginNote 的笔记卡片是所有模式的核心要素，对于卡片内容的编辑，MarginNote 现有的方式已经十分丰富。但是对于 Markdown 爱好者来说，不支持 Markdown 格式可能是一种遗憾。

[MarkDown](https://sspai.com/link?target=https%3A%2F%2Fbbs.marginnote.cn%2Ft%2Ftopic%2F7280) 插件不仅可以让你以 Markdown 格式编辑笔记卡片，它甚至支持 Latex 语法，可以在卡片内显示数学公式，还有 UML 中的流程图和序列图，以及表格，都可以用 Markdown 文本实现。对于习惯 Markdown 语法的人来说，这是一款不能不装的插件。
