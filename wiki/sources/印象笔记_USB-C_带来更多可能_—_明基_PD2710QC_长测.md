两年之前，我第一次评测带有 USB-C 接口的显示器，在那时我就已经认定，[USB-C 会成为接下来的几年里，显示器发展和更新的重要一环](https://zhuanlan.zhihu.com/p/27331593) ，也是区别新一代显示器和老旧显示器的分水岭。

![](.evernote-content/F3A13C44-FF63-4017-9CA9-AB1DE996E6C8/805AA213-33A0-4EEA-B32B-8EEFA9CC1B30)

2018 年，明基也推出了其首款搭载 USB-C 接口的显示器：PD2710QC，我派之前也有 [关于它的简单介绍和优惠价格](https://sspai.com/post/44588)，那么它的 USB-C 接口在具体使用体验上感受怎么样，屏幕素质究竟如何？这期 MDT 长测将会带你详细了解 PD2710QC 的表现。

\*全文基于 PD2710QC 国行量产版本，27 吋 2560×1440 分辨率，100% sRGB / Rec.709 色域，搭载全功能 USB-C 接口，支持升降旋转。

设计
--

虽然也是采用了 USB-C 扩展的设计，但明基 PD2710QC 的扩展和其他我们之前见过的 USB-C 显示器并不太一样。最明显的区别就是 PD2710QC 上明显大过其他显示器的底座。

![](.evernote-content/F3A13C44-FF63-4017-9CA9-AB1DE996E6C8/5ADCE12D-ADEB-4E2D-B5DB-B4C8A7F2D35B)

这块底座也正是 PD2710QC 扩展功能的所在，在底座上我们能发现底座一侧则有 DP 接口，USB-C接口, 四个 USB-A 接口、3.5mm 耳机插孔和一个网线接口。

![](.evernote-content/F3A13C44-FF63-4017-9CA9-AB1DE996E6C8/6B60D6AD-5AAC-49B1-B9D3-7FF183618F77)

而底座的另一侧有开关、电源接口和一个通用锁孔，底座的电源为 120W 输入，这也是为了支持 USB-C 和 4 个 USB-A 的供电而设计的。

在通过 USB-C 与笔记本电脑相连接之后，PD2710QC 可以为笔记本电脑提供 61W 的供电，然后用 DP 接口给屏幕输出来自笔记本电脑的视频信号，同时还能为笔记本电脑扩展 4 个 USB-A 接口（USB-C 速度为 10GBps），网线接口和 3.5mm 耳机接口。

![](.evernote-content/F3A13C44-FF63-4017-9CA9-AB1DE996E6C8/B40B99EB-09C3-4CCD-83A6-4F09D45CB121)

PD2710QC 支持旋转升降和前后俯仰，避免了长期低着头看笔记本电脑屏幕导致的颈椎问题。其中显示器可升高最大高度也比一般显示器要高一些：达到了 180mm, 这也是为了配合可以放置笔记本的底座而特别设计的。

![](.evernote-content/F3A13C44-FF63-4017-9CA9-AB1DE996E6C8/D9C7BC2F-F82A-4246-AECB-3C5C01326F29)

MateBook X Pro 为 13.9 寸 3:2 比例屏幕，屏幕的纵向高度大于一般的 15.6 吋 16:9 屏幕

180mm 意味着可以支持一个 15.6 寸笔记本电脑屏幕完全打开，也就是说在连接 PD2710QC 之后不仅可以将笔记本电脑合盖放在底座上仅使用显示器，也可以利用底座把笔记本电脑垫高从而同时打开双屏。

![](.evernote-content/F3A13C44-FF63-4017-9CA9-AB1DE996E6C8/77AC5247-F82B-4B6A-979A-5135548A2F9D)

回到屏幕本身，屏幕背后的下方提供了一排接口，其中有 HDMI 接口，两个 DisplayPort 输入接口（如果使用 USB-C 连接笔记本，那么有一个 DP 接口是一直被占用的），一个 DisplayPort 输出接口。

![](.evernote-content/F3A13C44-FF63-4017-9CA9-AB1DE996E6C8/A2BD8805-634B-46DD-BEA3-8597B1F69709)

另一侧则是显示本身的电源接口和电源开关，显示器 OSD 控制按钮全都在机身背面，这一竖列最下方还有一个显示器开关按钮。

![](.evernote-content/F3A13C44-FF63-4017-9CA9-AB1DE996E6C8/68569F36-3FE3-44C3-A801-B0CCC530478B)

因此在没有 USB-C 连接的情况下，PD2710QC 也可以直接用 DP 或者 HDMI 连接（当然这样也会导致 USB-C 扩展底座被浪费），就像任何普通的显示器一样，由于有 DP 输出接口，因此 PD2710QC 无论是否在 USB-C 模式下都可以支持多流传输（MST）技术，把最多四个显示器通过菊花链连接在一起。

屏幕素质
----

PD2710QC 为 27 吋 2560×1440 分辨率，100% sRGB / Rec.709 色域，拥有出厂校色，我们也对这块屏幕进行了专业的测试：

![](.evernote-content/F3A13C44-FF63-4017-9CA9-AB1DE996E6C8/76CB6F13-1531-4E81-AAD2-DAC018B8D8F5)

按 [《当评价一款产品屏幕时，我们关注的是什么？》](https://zhuanlan.zhihu.com/p/22411193)中提到的标准，sRGB、P3、Adobe RGB 才是对消费者更为直观有效的色域，而 PD2710QC 拥有出厂校色，所以并不需要担心色域超出 sRGB, 实测的结果也表明这块屏幕覆盖了 96.9% sRGB 色域，忽略测量误差后已属于能全覆盖 sRGB 的范畴，并且覆盖相当精准，没有超出的部分，只是在紫色有一点点缺失, 能满足日常家用和绝大部分的 UI 设计、影像处理工作。

![](.evernote-content/F3A13C44-FF63-4017-9CA9-AB1DE996E6C8/C55FCFA8-976C-422A-BE37-EF9055AF0A23)

在我们的测试中，PD2710QC 的屏幕最高亮度为 293.31cd/m², 对比度最高达到了 889.42:1, 属于桌面级显示器的标准水平。

![](.evernote-content/F3A13C44-FF63-4017-9CA9-AB1DE996E6C8/49976CFA-84B8-441C-A653-EF1CBF11A4DE)

色准部分，由于 PD2710QC 经过了出场厂校色，所以屏幕色偏都很低，平均 ΔE 只有 0.54，最大 ΔE 为 2.86, 分别低于平均 ΔE 小于 1 和最大 ΔE 小于 3 的标准，屏幕色为 6532K, 与 sRGB 的标准 6500K 非常接近，**得益于出场厂校色，这块屏幕的色偏非常小，已不需要额外校色就可以直接使用。**

当然，经过再次校色之后的表现也会更好，BenQ PD2710QC 的平均 ΔE 和最大 ΔE 经过再次校色之后可以分别达到 0.31 和 1.48 的水平，不过肉眼已经比较难看出后期再校色和出场厂校色的区别了。

综合来看，PD2710QC 的屏幕素质不错，300 cd/m² 级别的亮度在室内使用已经足够。对于专业的影像处理和 UI 设计等工作，因其几乎完美覆盖 sRGB 色域，且有出场校色所以色偏很小，可以不需自行校色直接满足除了 P3 或者 AdobeRGB 色域之外的所有需求。

另外，**PD2710QC 还在显示器控制中提供了一个 MacBook 模式**，明基提到这种模式能更好得模拟 Mac 产品显示器的颜色，因此我们也在这个模式下再次进行了测试：

![](.evernote-content/F3A13C44-FF63-4017-9CA9-AB1DE996E6C8/C1330D65-92E0-41CA-B0C0-D3FD29F1D30A)

通过实测，MacBook 模式其实是把 sRGB 的出场校色改为 P3 色域校正，因此 P3 色域的覆盖也会从 sRGB 模式的约 70% 提升至 80% 左右，但是由于这块面板并不是 P3 色域，因此并不能和 MacBook Pro 一样覆盖全部 P3 色域。

其他功能
----

显示器与我们的健康息息相关，除了之前提到过的旋转升降保护颈椎之外，**PD2710QC 也使用的是 DC 直流调光，因此不会出现闪屏现象，加之屏幕表面为雾面处理，避免了反射眩光，显示更柔和**，长时间使用不容易导致疲劳。

![](.evernote-content/F3A13C44-FF63-4017-9CA9-AB1DE996E6C8/108409E1-4FD1-4292-9C96-B1DD45FFD890)

除此之外 PD2710QC 还提供了低蓝光模式，在硬件层面减少蓝光发出，为此我们也新增了一个光谱测试项目，后方为标准模式下显示器显示白色时的光谱，前方橙色图形为低蓝光模式下的光谱。通过对比我们不难发现，在低蓝光模式下的蓝光更少，可以减少蓝光对人体分泌褪黑激素（melatonin）的影响，帮助使用者获得更规律的更好的睡眠。

总结
--

相较于普通显示器，通过 PD2710QC 最大的优势还是其拥有 USB-C 接口，只需要一根连接线，就可以为笔记本电脑提供供电、数据扩展、视频传输三种功能，极大程度的提高了日常使用的便利性。

![](.evernote-content/F3A13C44-FF63-4017-9CA9-AB1DE996E6C8/3AE8426C-2B1B-4AF5-ACC6-124A7A506549)

在一个显示器应该有的「本分」：屏幕素质上， PD2710QC 也因为自带出场校色能精准覆盖 sRGB 且色偏很小，能满足除了 P3 或者 AdobeRGB 色域之外的所有需求。

总体来说，PD2710QC 可以被理解成为一个 27 吋 QHD 分辨率显示器 + 100% sRGB + 出厂校色 + 扩展坞 + 支持菊花链连接的综合体，所以直接拿一个普通 27 吋 QHD 显示器对比价格意义不大，考虑到 PD7210QC 相对于普通显示器带来的额外功能，最低活动价 3999 并不算太贵。

**优点：**

* 拥有 10Gbps 的 USB-C 接口，支持扩展
* 出厂校色，色准很不错，能完美覆盖 sRGB
* 27 吋 QHD 分辨率，四面窄边框，屏占比较高
* 雾面处理，抗眩光能力强
* 支持显示器旋转升降

**缺点：**

* 27 吋 QHD 分辨率
* 屏幕和扩展坞为分体

**购买建议与适用人群：**

从 MacBook Pro 到 ThinkPad X1、MateBook X Pro 或者是 XPS 系列，不论 Mac 或者 PC 都已经在高端和中高端产品上普及了 USB-C 接口，甚至有的产品如 MacBook Pro 和 XPS 13 2018 机身上只有 USB-C 接口，因此一个具有 USB-C 接口的显示器就成为了理想的搭配选择。

PD2710QC 主要的问题还是在 2560×1440 分辨率上，目前购买只有 USB-C 接口笔记本的消费者更需要的是一个高端定位显示器，因此在 27 吋这个级别 QHD 分辨率显然不完美，如果未来能推出 4K 分辨率版本，无疑会更受欢迎。
