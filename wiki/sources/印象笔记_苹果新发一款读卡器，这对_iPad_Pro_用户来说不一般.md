# 苹果新发一款读卡器，这对 iPad Pro 用户来说不一般

---

[![](.evernote-content/A4451EC3-20FA-4E80-B7A7-235AF03825E5/86A5D9B2-FF0D-4DB5-A5BA-BF1EED3303CF.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/12/img_3548.jpg)

这两天苹果的小动作不少。昨天推出了可以让 [iPhone 6/6s 变身复古 HTC 手机的手机套](http://www.ifanr.com/594689)，今天又接连推出了 iOS 9.2、watchOS 2.1 等软件更新。

不过这些还没有完，在 2012 年推出一款 Lightning to SD 卡相机读卡器后，今天，苹果还新推出了一款 [Lightning to SD 卡相机读卡器](http://www.apple.com/cn/shop/product/MJYT2FE/A/lightning-to-sd-card-%E7%9B%B8%E6%9C%BA%E8%AF%BB%E5%8D%A1%E5%99%A8?fnode=91)。

[![](.evernote-content/A4451EC3-20FA-4E80-B7A7-235AF03825E5/ED212688-7364-49AB-BF40-D2E2725B24DF.png)](http://images.ifanr.cn/wp-content/uploads/2015/12/USB3.0.png)

这款读卡器整体看起来像一个普通的读卡器加上一根带 Lightning 接口的数据线，通体白色，设计上还是很有 Apple Style 的。

这款产品使用上与普通的读卡器无异，即插即用。iPad 会自动打开照片 app，并将你的照片整理到 “时刻”、”精选” 和 “年度” 中。结合着 iCloud，你所有的相关设备都能同步照片。

改读卡器主要针对 iPad 推出，不过更新到 iOS 9.2 的手机应该也能用得上（待考证）。

这里要着重讲一下 iPad Pro。在 iPad Pro 的拆解中，iFixit 指出 iPad Pro 的 Lightning 接口实际是支持 USB 3.0 标准的，不过当时还用不了。

而当这款读卡器用在 iPad Pro 上时，iPad Pro 用户即可体会到 USB 3.0 的速度。USB 3.0 的速度最高为 640MB/S 是 USB 2.0 标准 60MB/S 的十倍以上。而目前支持 USB 3.0 的手机实际上屈指可数，三星在 Note 3 等机型后，后续机型也回归了 USB 2.0。

至于价格，这款 Lightning to SD 卡相机读卡器为 228 元。
