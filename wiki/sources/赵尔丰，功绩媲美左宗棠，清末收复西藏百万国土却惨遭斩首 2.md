# 赵尔丰，功绩媲美左宗棠，清末收复西藏百万国土却惨遭斩首

* 赵尔丰何许人也？
  ========

1845年，赵尔丰出生于辽东官宦世家，祖籍襄平（辽宁省辽阳市），正蓝旗人。

他的父亲是道光年间进士，曾在山东担任知县，他的三个兄弟赵尔震、赵尔巽、赵尔萃均考取进士，其中，二哥赵尔巽还是《清史稿》的编纂者。

按说出生在这样的书香门第，赵尔丰考取个功名应该也不是难事，可偏偏屡试不中，最后没办法靠纳捐在广东当了个小官。

在广东的很长一段时间里，赵尔丰碌碌无为，到1884年，年近不惑的赵尔丰才调任山西，先后在静乐、永济当个县令。

![](.evernote-content/11C01E61-B8CA-487E-8E88-450312719C8A/8233AA04-D56B-4A6A-845F-F6A758BD9B52.jpg)

照此下去，赵尔丰大多会像他的父亲一样平淡无奇的走完自己的仕途。

**但所谓“千里马常有，伯乐不常有”，当时的山西巡抚锡良就认为赵尔丰是千里马，即便未曾考过功名又怎样？**当年，他们的前辈左宗棠不也是个落第秀才嘛。

就这样，赵尔丰得到锡良提携，随侍身边，并因此开拓见识和眼界。

**1903年锡良调任四川总督，赵随之入川，从此开启了他的大展宏图之旅。**

* 青藏高原上的分裂危机
  ==========

众所周知，1900年八国联军发动侵华战争，京津、华北及东北地区都被列强的铁蹄蹂躏，广大百姓正处在水深火热之中。

而此时的藏区也同样不安宁，因为这里实在是与英属印度的距离太近了。

当时的大英帝国号称日不落帝国，有着全世界最强大的海军，殖民地遍布世界各地，印度因地域广阔，原材料丰富，更是被称为英国女王头上的明珠。

**英国在拥有印度后，开始觊觎这片与其相邻的雪域高原，即通过占领西藏，将势力伸向中国长江腹地，与东南沿海遥相呼应，并把整个长江流域据为己有。**

![](.evernote-content/11C01E61-B8CA-487E-8E88-450312719C8A/2635C144-9528-49C3-B5BB-CA9A3BD0ECAF.jpg)

1888年，英国从印度派出两千人出兵西藏，藏区人民组织上万人奋起反抗，结果遭到清朝驻藏大臣的阻挠，因为清政府已经被打怕了，不愿节外生枝。

在清朝的妥协退让下，双方于1890年达成《中英藏印条约》不平等条约，**将喜马拉雅山南坡的哲孟雄谷地划给英国，哲孟雄就是后来的锡金王国，1975年被印度吞并。**

经此事件后，从藏区僧侣贵族到平民便产生离心倾向，他们认为大清已经衰落，已无法实施有力保护。

1903年，英军第二次入侵西藏，并于次年进入拉萨，与藏区代表签订《拉萨条约》，除了赔款，还让渡通商、贸易主权。

![](.evernote-content/11C01E61-B8CA-487E-8E88-450312719C8A/6A73C63D-D8A8-4BB2-AD11-B9F28781CA13.jpg)

**同时在英国人的威逼利诱下，一些藏区贵族被成功拉拢，这样一来，英国人迅速在藏区扶植一批代理人，企图使西藏脱离中国统治。**

![](.evernote-content/11C01E61-B8CA-487E-8E88-450312719C8A/33F22C68-F682-42FB-BC1D-7F5ACF8D53D3.jpg)

当时的清政府拒绝承认该条约，驻藏大臣有泰被革职。

1905年派遣凤全任驻藏大臣，凤全意识到西藏潜在的分裂危机，于是通过改革削弱喇嘛和土司势力，但因用力过猛、措施失当，引起暴动，凤全本人在巴塘镇压时被杀。

**此时的西藏已是危机四伏，一边是英军的武力威慑，一边是清廷藏区改革失败，西藏已濒临分裂边缘，赵尔丰就是在这种棘手且危急的时刻入藏，力挽狂澜。**

* 从川边改土归流到平定西藏叛乱，赵尔丰铁腕粉碎西藏叛乱阴谋
  ============================

1905年5月，凤全遇害后，赵尔丰受命平定土司叛乱，这虽然是他第一次处理川藏边区事务，但却表现出超强的魄力和手腕。

对于武装叛乱，赵尔丰主张武力镇压，当时造反的理塘土司逃至桑披寺，赵尔丰率军围攻长达数月，并切断水源，攻破该寺，但理塘土司却侥幸逃脱至西藏。

**1906年8月，清政府设立省级建制的川滇边特别行政区，赵尔丰因平定叛乱有功被任命为川滇边务大臣，至此赵尔丰从幕后转到前台，开始独当一面。**

![](.evernote-content/11C01E61-B8CA-487E-8E88-450312719C8A/5DFDE807-0EF9-4EE9-A392-7FF9F8535914.jpg)

清政府认为藏区能有今天的局势完全是因为驻藏官员管理不力及土司权力过大。

因此决定在川边藏区实行改土归流，通过稳定川边局势，继而进军西藏，这一任务自然地落到赵尔丰头上。

赵尔丰主持川滇边区事务后，开始大刀阔斧的改革，在西康地区设立府、州、县，以朝廷派遣的流官取代当地土司。

由于土司的权力被削弱，自然遭到他们的武力反抗。

在此后的六年里，赵尔丰几乎在不停的打仗，先后铲除巴塘、理塘等大小土司，并取消昌都等地的活佛政治地位，极大稳固川滇地区的局势。

同时赵尔丰将西藏和川滇藏区划分界限，防止两大藏区势力勾结，并开田引水，试验种植稻米。

**由于赵尔丰在川边的改革取得成效，清政府于1908年2月任命赵尔丰为驻藏大臣兼任川滇边务大臣，推行改土归流和易俗改革。**

![](.evernote-content/11C01E61-B8CA-487E-8E88-450312719C8A/29433EC7-C11A-4852-AD05-BF5C8FF5A66A.jpg)

此时的西藏高层已成为亲英势力，再加上赵尔丰的改革严重损害土司利益，导致僧侣贵族与土司联合起兵阻挠赵尔丰进藏。

就在这个关键时刻，赵尔丰的哥哥赵尔巽被任命为四川总督，这给赵尔丰平定藏区叛乱提供有力后勤保障。

1909年，64岁的赵尔丰不顾高寒缺氧和严寒风雪的恶劣气候，率2万大军入藏平叛，先是击垮巴塘叛军，后入藏收复江卡、贡觉等部落，后越过丹达山。

一路上清军在赵尔丰的感染下，奋勇杀敌，势如破竹，藏军被打的丢盔弃甲，大败而逃。

不久，赵尔丰兵锋向西直抵江达宗，拿下江达、工布，拉萨门户洞开，为首的达赖喇嘛逃往印度避难。

![](.evernote-content/11C01E61-B8CA-487E-8E88-450312719C8A/E8CB757E-E7FB-4376-BE0D-B9FC6849E90F.jpg)

1910年2月，赵尔丰进驻拉萨，4个月后又引兵南下波密。

波密位于藏南地区，分裂势力强大，当地土司与英国人眉来眼去，一旦这里被英国占领，将极大影响藏区边疆安全。

赵尔丰不敢大意，马不停蹄向这里进发，最终历尽千辛万苦，成功平定波密。

**至此藏区被全部平定，粉碎了英国打通印缅、穿插藏地、俯瞰川滇的阴谋，巩固中国西南百万领土安全。**

赵尔丰主政西藏后，立即实行改土归流，对所到之处的土司进行招抚，说服他们建立郡县，对于反抗的，用武力剿灭。

![](.evernote-content/11C01E61-B8CA-487E-8E88-450312719C8A/8A5F4DF1-85C0-43C4-AC1E-40293DB4EFE4.jpg)

在他的铁腕推动下，土司特权被彻底废除，所有行政事务归政府管理，同时，抑制寺庙势力的恶性膨胀，禁止土司、寺庙放高利贷，严禁炒作买卖土地。

另外，赵尔丰开办学堂，普及藏区的汉语和汉文化知识，打破寺庙的教育垄断。

他还鼓励垦荒，兴厂开矿，发展邮政，统一度量衡，甚至还修建从藏区到四川的公路，加强朝廷对当地的管理，以及内地和藏区的联系。

**赵尔丰经营川边藏区数年时间里，不仅粉碎分裂势力，维护国家统一，而且极大推动藏区社会经济发展，以至于后世评价赵平定西藏之功绩堪比左宗棠收复新疆。**

![](.evernote-content/11C01E61-B8CA-487E-8E88-450312719C8A/0BFEC489-E789-402D-BAC7-5E46FCC8FE08.jpg)

**然而就是这样一位有功疆臣下场竟十分凄惨。**

* 平藏功臣惨死成都街头
  ==========

1911年4月，赵尔巽调任东三省总督，赵尔丰接任四川总督，几个月后清政府宣布铁路国有，借此把铁路利权出卖给西方帝国主义，一场声势浩大的保路运动迅速在四川爆发。

起初赵尔丰上疏，为群众运动据理力争，但无济于事，朝廷反而要求其武力镇压。

**赵尔丰无奈只能执行，最终造成成都血案，赵尔丰的名声跌落谷底，此后被免去四川总督，留任边务大臣。**

1911年10月，武昌起义爆发，很快革命浪潮席卷南方各地，四川于11月22日宣布脱离清朝独立，成立军政府。军政府成立的第一件事自然要拿前清官员祭旗。

**赵尔丰作为前四川总督自然首当其冲，很快都督尹昌衡率部将赵尔丰擒获，并在成都召开公审大会，尽斥其罪，下令将赵尔丰当场处决。**

![](.evernote-content/11C01E61-B8CA-487E-8E88-450312719C8A/2FED002D-D8FB-4916-983A-34736B8B8DD6.jpg)

**就此晚清一代枭雄以如此不堪的方式结束了自己的一生。左宗棠收复新疆可以流芳千古，而赵尔丰收复西藏却遗臭当时。**
