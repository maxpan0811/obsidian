究竟是哪些不一样的地方，值得我们为 Model X 等上三年之久？
=================================

[周 欣](http://www.ifanr.com/author/zhouxin "Posts by 周 欣")| 昨天 04:39

标签：[D 系列](http://www.ifanr.com/tags/d-%e7%b3%bb%e5%88%97) [Elon Musk](http://www.ifanr.com/tags/elon-musk) [Model 3](http://www.ifanr.com/tags/model-3) [Model S](http://www.ifanr.com/tags/model-s) [Model S P85D](http://www.ifanr.com/tags/model-s-p85d) [Model X](http://www.ifanr.com/tags/model-x) [Tesla](http://www.ifanr.com/tags/tesla) [全轮驱动](http://www.ifanr.com/tags/%e5%85%a8%e8%bd%ae%e9%a9%b1%e5%8a%a8) [双电机](http://www.ifanr.com/tags/%e5%8f%8c%e7%94%b5%e6%9c%ba) [发布会](http://www.ifanr.com/tags/%e5%8f%91%e5%b8%83%e4%bc%9a) [特斯拉](http://www.ifanr.com/tags/%e7%89%b9%e6%96%af%e6%8b%89) [电动汽车](http://www.ifanr.com/tags/%e7%94%b5%e5%8a%a8%e6%b1%bd%e8%bd%a6) [鹰翼门](http://www.ifanr.com/tags/%e9%b9%b0%e7%bf%bc%e9%97%a8)

由于设计和制造方面的原因，原本计划在 2013 年推出的 Model X 一直搁置到了今天，才“千呼万唤始出来”。

美国太平洋时间 9 月 29 日晚上 8 点（即今天上午 11 点），特斯拉正式召开了第一批 Model X 的发布会。

[![](.evernote-content/8DDF2157-C2B6-4B61-AF55-A45BC1FA3369/053D7719-9C06-4FF8-A6EF-5A1ACF533DB7.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/09/launch1.jpg)

从 2012 年 2 月模型展示到 4 月接受预定、再到现在的正式交付，特斯拉的死忠粉们用了整整三年时间，才迎来了第一批 Model X 的发布。据特斯拉员工透露，期间也有不少车主因为受不了这种漫长的等待而选择了退订。

和 Model S 一样，早期购买者可以得到签名版的 Model X（拥有独特的徽标和 Signature Red 车漆）——当然，它早就被一抢而光了。

超过 130,000 美元的不菲价格足以使这款车型位列 Model S 的高端梯队，那么，相比较 Model S，特斯拉的首款 SUV 车型——Model X 到底有哪些特点呢？我们今天就来具体分析一下。

### 更大：拥有三排座椅，可容纳七名成人

虽然 Model S 也可以容纳七名成人，但它的方式是在尾箱增加儿童座椅，使整部车的乘坐格局变为 5＋2 模式。

[![](.evernote-content/8DDF2157-C2B6-4B61-AF55-A45BC1FA3369/EC251D41-118D-47F3-8335-DF36BF3369ED.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/09/child2.jpg)

对此，[The verge](http://www.theverge.com/2015/9/29/9414415/tesla-model-x-suv-launch-date) 表示：

> it’s definitely not the sort of thing you’d want to have to use on a regular basis
>
> （正常情况下，这肯定不是你想要的。）

但 Model X 是直接通过增加车内座椅排数来扩大容量，两者有着本质的区别。

[![](.evernote-content/8DDF2157-C2B6-4B61-AF55-A45BC1FA3369/3936B27D-79D2-43A9-9EB2-D6D3CF685F11.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/09/rows.jpg)

对于家庭而言，Model X 应该是个不错的选择。

### 更快、更强：“双电机，全轮驱动”

Model X 和 S 之间有着很多相似之处。

还记得特斯拉在去年发布的以 D 为后缀的电动车系列（60D、85D 和 P85D）吗？如图所示。

[![](.evernote-content/8DDF2157-C2B6-4B61-AF55-A45BC1FA3369/145237FD-271A-4EC9-9D0C-61F112E2CC1C.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/09/p85d.jpg)

这一系列的最大的特点就是 “双电机，全轮驱动”（dual-motor, all-wheel-drive），而同样的动力系统也被用在了 Model X 系列上。

[![](.evernote-content/8DDF2157-C2B6-4B61-AF55-A45BC1FA3369/748E1336-F161-473B-BBA2-1C8C0A51400E.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/09/dual.jpg)

在介绍这种动力系统的优势时，特斯拉的[官网](http://www.tesla.cn/%E5%AF%B9%E5%A4%96%E5%8F%91%E5%B8%83/%E7%89%B9%E6%96%AF%E6%8B%89%E5%8F%91%E5%B8%83%E5%8F%8C%E7%94%B5%E6%9C%BA%E5%85%A8%E8%BD%AE%E9%A9%B1%E5%8A%A8model-s-%E5%A4%A7%E5%B9%85%E6%8F%90%E5%8D%87%E5%90%84%E9%A1%B9%E6%80%A7%E8%83%BD)上是这样说的：

> 依靠这种技术，特斯拉 Model S 可以独立数字控制各个电机，使四个轮子都能分配到扭矩，在毫秒间将动力分配给前后轴。这种实时精确控制让四个车轮都能分配到扭矩，使得Model S 的牵引力、稳定性、操控性和安全性都有革命性的提升。
>
> 双电机全轮驱动 Model S 另外把原本惊人的加速能力进一步提高。Model S P85D 从 0 到96 km/h 加速仅需 3.2 秒，足以和跑车媲美。

而以上这些特点，也同样适用于 Model X 系列。

### 续航稍弱，但也弱不到哪里去

由于 Model X 的车身要比 Model S 重 10%，所以它在续航里程上也有所缩水。

根据美国国家环境保护局（EPA）给出的数字：在充满电的情况下，性能版 Model X 的里程数要比 Model S 少 3 公里；非性能版要少 13 公里。

不考虑配置，仅以 90 kWh 的电池为例，Model X 在续航上至少可以达到 250 英里。如果将其与超级充电站之类相连，表现上也能达到 Model S 那样的水平。

### 鹰翼门设计，让 Model X 更拉风

鹰翼门以向上折叠的方式打开，以减少占用空间，使乘客在狭窄的停车场内也可比较容易地进入第三排入座。

[![](.evernote-content/8DDF2157-C2B6-4B61-AF55-A45BC1FA3369/74F57A92-1B74-4982-8103-8768EB450AF1.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/09/door1.jpg)

（看起来像是长了翅膀）

向上开启的方式保证了充裕的上下车空间，而开启过程本身也非常炫酷。

### 一段时间内，特斯拉不会再出比它贵的车了

两个消息：特斯拉 Model S 据称已经进入了北京新能源汽车摇号目录；另一方面，Model X 的新订单将于 2016 年下半年开始交付。

由于 Model X 价格在 132,000 美元以上，一般人承受不起，Elon Musk 的下一步计划是发布 Model 3，使其成为市场上的主流机型。

[![](.evernote-content/8DDF2157-C2B6-4B61-AF55-A45BC1FA3369/F1A7B82F-9603-4F50-B520-CD8BF57014E5.jpg)](http://images.ifanr.cn/wp-content/uploads/2015/09/model1.jpg)

35,000 美元的价格相对来说比较平价，也会使人们暂时忘记特斯拉 Model X 或 S 的昂贵。

### 结语

综合以上几点，我们可以得出 Model X 的特点：

> 外形更大更拉风；功能更快也更强；“续航稍弱”，但也弱不到哪里去。

总而言之就是越来越完美，唯一不变的或许是——一般人还是消费不起。
