[摘要] 本文档记录了 NeXTSTEP 操作系统设计规范的研究过程，目的是为桌面看板的 Platinum 风格优化提供参考。搜索过程：从印象笔记找到 11 篇相关文章（包括 Designing the TEXTSTEP user interface 和 NeXTSTEP Wikipedia），同时使用 Tavily 搜索。NeXTSTEP 核心设计特征包括：3D 凿刻控件（凸起/凹陷立体效果）、灰色系配色方案（浅灰背景 #C0C0C0、深灰条纹标题栏、深蓝选中态）、右侧垂直 Dock（非 macOS 底部横排）、Miller Columns 多级列浏览、Inspector 属性检查器面板。与现代 macOS 对比：按钮从 3D 凿刻变为扁平/微圆角、Dock 从右侧变为底部、字体从 Helvetica 变为 San Francisco。对当前看板的参考价值：3D 凹凸感已在用、条纹标题栏已在用、可考虑把分公司按钮改成右侧竖排、把筛选条件做成右侧浮动面板、考虑切换到 Helvetica 字体。后续补充了 5 张 NeXTSTEP 截图获取过程（Medium 博客超时无法访问，从 Wikipedia/ToastyTech/System Folder Blog 获取替代来源），发现印象笔记 MCP API 不支持图片嵌入，走"本地保存+手动拖入"流程。原文链接：/Users/panbo/Obsidian/程序开发/20260719-NeXTSTEP设计规范研究.md
