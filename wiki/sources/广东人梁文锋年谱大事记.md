# 广东人梁文锋年谱大事记

> 量化投资和AI大模型领域的创业家。来源：微信公众号文章。

## 教育背景

| 时间 | 事件 |
|:----|------|
| 1985年 | 出生于广东湛江，父亲为小学教师 |
| 2002年 | 考入浙江大学电子信息工程专业（吴川一中高考状元） |
| 2007年 | 考上浙江大学信息与通信工程研究生，研究机器视觉 |
| 2010年 | 浙大硕士毕业 |

## 量化投资阶段

| 时间 | 里程碑 |
|:----|--------|
| 2008年 | 开始探索全自动量化交易 |
| 2013年 | 创办杭州雅克比投资 |
| 2015年6月 | 创办幻方量化 |
| 2016年10月 | 推出第一个AI模型，管理规模达10亿元 |
| 2017年 | 管理规模扩至30亿元，AI算法团队扩张 |
| **2018年** | **幻方量化获私募金牛奖**（中国私募证券最高奖项） |
| 2019年 | 投资2亿元自研"萤火一号"训练平台 |
| 2021年 | 管理规模突破千亿，跻身量化"四大天王" |

## AI大模型阶段

| 时间 | 里程碑 |
|:----|--------|
| 2023年5月 | 宣布进军AGI |
| 2023年7月 | 创办DeepSeek |
| 2024年5月 | 发布DeepSeek-V2 |
| **2024年12月26日** | **发布DeepSeek-V3，训练成本仅557.6万美元，轰动全球** |
| **2025年1月20日** | **发布DeepSeek-R1**；受邀参加总理主持的专家座谈会并发言 |

## 主要荣誉

- **私募金牛奖**（2018）— 中国私募证券领域最高奖项
- 幻方量化管理规模突破千亿（2021），量化"四大天王"之一
- DeepSeek-V3训练成本仅557.6万美元，性能比肩GPT-4o，引发全球关注
- DeepSeek-R1性能与OpenAI o1正式版持平，且开源权重
- 受邀参加总理主持的专家座谈会并发言
