在 macOS 上搞开发，选择对的工具能让你的工作事半功倍。好在 macOS 生态下有很多优秀的软件可以提升生产力、减少繁琐的手动操作，让整个开发流程更加顺畅。

![](.evernote-content/891F62D5-9C9F-4786-B1D0-429560C4138B/DCB1C114-4E7C-4F39-AAAE-8304FD6E1D27.png)

今天我写了下面这份清单包含了一些精选的免费或开源工具，能帮你更高效地写代码、管理项目、调试问题，让你的开发体验更加丝滑。

基础工具
----

### Git - 代码管理的基石

Git 是现代开发的必备工具，无论是个人项目还是团队协作，都离不开它。在 macOS 上，你可以用 Homebrew 安装 Git，也可以直接在终端尝试 `git` 命令，系统会提示安装 Xcode 命令行工具。如果你不喜欢命令行，还可以用 GUI 版本，比如 SourceTree 或 GitHub Desktop，让 Git 操作更加直观。

![](.evernote-content/891F62D5-9C9F-4786-B1D0-429560C4138B/CB879045-7E0B-4B26-B1DD-A179CEC06E92.png)

**为什么推荐？**

* **版本控制**：记录代码的所有变更，随时回溯历史版本。
* **团队协作**：多人开发，推送、拉取代码都很方便。
* **代码备份**：不会因为误操作丢失重要代码。

### ServBay - 一站式开发环境管理

在 macOS 上搭建开发环境，有时候会遇到各种依赖冲突，ServBay 可以帮你一键安装 Node.js、Python、PHP 等开发工具，并且可以自动更新，免去手动配置的麻烦。

![](.evernote-content/891F62D5-9C9F-4786-B1D0-429560C4138B/3FC3EE43-4F4B-4807-AF2B-A99AC7E688E9.png)

**为什么推荐？**

* **环境管理**：告别手动安装、配置各种工具，让一切变简单。
* **团队统一**：所有人使用相同的开发环境，减少“在我电脑上没问题”的情况。
* **自动更新**：不用担心用的工具版本太旧，随时保持最新状态。

---

终端工具
----

### iTerm2 + Oh-My-Zsh - 超强终端组合

如果你经常在终端里敲命令，那 macOS 自带的 Terminal.app 可能会让你觉得不够用。iTerm2 提供了分屏、标签页、自定义快捷键等高级功能，而 Oh-My-Zsh 则让你的终端更加智能，比如自动补全、Git 状态显示等，能大幅提升效率。

![](.evernote-content/891F62D5-9C9F-4786-B1D0-429560C4138B/FC861EBA-DCBF-4A29-8596-228326522199.png)

**为什么推荐？**

* **高度可定制**：你可以按照自己的习惯调整终端，让工作更顺手。
* **提升效率**：插件系统能让很多重复操作变得更简单。
* **美观舒适**：可以自定义终端外观，让它更符合你的风格。

### Terminus - 跨平台终端工具

如果你经常需要远程连接服务器，比如 AWS、DigitalOcean，那么 Terminus 可能是个不错的选择。它界面简洁，跨平台支持 macOS、Linux 和 Windows，适合需要在多个环境下工作的开发者。

![](.evernote-content/891F62D5-9C9F-4786-B1D0-429560C4138B/0F7BB54A-55B9-4F5A-975F-7348369B9784.png)

**为什么推荐？**

* **简洁高效**：没有花里胡哨的功能，专注于终端体验。
* **跨平台支持**：在不同系统间切换时，使用体验一致。
* **云端友好**：适合远程开发，管理多个云服务器也很方便。

调试工具
----

### Bruno - 轻量级 API 测试工具

以前大家测试 API 都用 Postman，但现在它收费了，所以很多人开始用 Bruno。Bruno 的优势是所有测试用例都是 JSON 文件，可以用 Git 进行版本管理，团队协作也很方便。

![](.evernote-content/891F62D5-9C9F-4786-B1D0-429560C4138B/73232506-E7DE-41D4-B36A-6059DD6F1536.png)

**为什么推荐？**

* **支持 Git 版本管理**：API 测试用例可以像代码一样管理。
* **适合团队协作**：可以共享测试用例，提高效率。
* **界面简洁**：比大部分 API 测试工具更好用，易上手。

### SwitchHosts - 轻松管理 hosts 文件

如果你经常在本地调试多个项目，并且需要频繁修改 hosts 文件，SwitchHosts 绝对是必备工具。它能帮你快速切换 hosts 配置，避免手动修改的麻烦。

![](.evernote-content/891F62D5-9C9F-4786-B1D0-429560C4138B/36E32396-7EEC-4586-82BF-A8E5DCF0E79B.png)

**为什么推荐？**

* **自动管理 hosts 文件**，无需手动修改。
* **一键切换不同的环境**，比如开发、测试、生产环境。
* **避免冲突**，让不同项目的 hosts 配置保持独立。

### AnotherRedisDesktopManager - Redis 可视化工具

如果你开发过程中需要操作 Redis，这款 GUI 工具能让你的数据管理更加直观，支持查看字符串、哈希、列表、集合等数据结构。

![](.evernote-content/891F62D5-9C9F-4786-B1D0-429560C4138B/9035E5DE-F652-47B1-81EF-BF38FD200D21.png)

**为什么推荐？**

* **可视化管理**：比命令行操作更直观，数据查询更方便。
* **跨平台支持**：macOS、Windows、Linux 都能用。
* **轻量级**：不会占用太多系统资源，但功能很全。

轻量编辑器
-----

### CotEditor - 快速文本编辑神器

虽然 VS Code 功能强大，但有时候我们只是想快速编辑一个文件，CotEditor 就是个不错的选择。它启动速度快，不会像大编辑器一样占用大量内存。

![](.evernote-content/891F62D5-9C9F-4786-B1D0-429560C4138B/DBD15A4D-4DFC-4541-844D-920803942469.png)

**为什么推荐？**

* **启动快**：比 VS Code 之类的大型编辑器更轻便。
* **可自定义**：可以调整快捷键、颜色等，符合个人习惯。
* **稳定可靠**：不会因为开太多文件而崩溃或卡顿。

如果想更方便地使用 CotEditor，可以在终端里加个别名：

```
# 让 'cot' 变成打开 CotEditor 的快捷命令  
sudo ln -s /Applications/CotEditor.app/Contents/SharedSupport/bin/cot /usr/local/bin/cot  
alias cot='open -a /Applications/CotEditor.app'
```

提高生产力的工具
--------

### Hidden Bar - 隐藏菜单栏图标

如果你觉得 macOS 的菜单栏图标太多太乱，Hidden Bar 可以帮你隐藏不常用的图标，让桌面更整洁。

![](.evernote-content/891F62D5-9C9F-4786-B1D0-429560C4138B/6C1EE6D2-DB94-440E-B131-3223263C393C.png)

**为什么推荐？**

* **保持桌面整洁**，只显示需要的图标。
* **快捷访问常用应用**，提升工作效率。
* **开源免费**，使用简单无广告。

### Maccy - 超好用的剪贴板管理器

macOS 的剪贴板只能存储最近复制的一项内容，而 Maccy 能帮你保存多个历史记录，随时搜索和粘贴。

![](.evernote-content/891F62D5-9C9F-4786-B1D0-429560C4138B/8C9F7574-8F64-460E-9F04-F826DA304F9C.png)

**为什么推荐？**

* **保存剪贴板历史**，不用担心复制的内容丢失。
* **支持搜索**，快速找到之前复制的内容。
* **提升效率**，减少重复输入，提高工作流畅度。

结语
--

这些 macOS 开发工具能极大地提升你的开发效率，让你专注于写代码，而不是花时间折腾环境、调试问题。

![](.evernote-content/891F62D5-9C9F-4786-B1D0-429560C4138B/B4F6C549-14B3-4C81-AB88-A5DD4D7822EF.jpg)

每个工具都有各自的特色，可以根据自己的需求选择最合适的工具，相信它们一定能让你的工作更加顺畅！
