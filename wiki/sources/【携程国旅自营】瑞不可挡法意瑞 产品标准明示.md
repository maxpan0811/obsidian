# 瑞不可挡法意瑞·产品标准明示

> 【睿】系列精选级产品接待标准明示。4-5钻，20-26人团。

## 核心标准

| 项目 | 规格 |
|------|:----:|
| 团队人数 | 20-26人 |
| 住宿 | 4钻+携程评分≥4分 |
| 酒店披露 | 6家（入住范围明确），含热早 |
| 交通空座率 | ≥45% |
| 餐饮标准 | ≥15欧/人/餐 |
| 特色餐 | ≥3次 |
| 导游经验 | ≥8年 |
| 含票景点 | 5-8个含深度体验 |
| 特色活动 | ≥2项特色活动+≥2项VIP通道 |
| 购物/自费 | **无购物、无自费** |
