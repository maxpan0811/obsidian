RSSHub
------

![](.evernote-content/9A18884A-8E48-48DF-9070-EBEA2B1A2966/6DC18797-7021-4E3F-AA47-B191A234113D.png)

**RSSHub**

[下载链接](https://github.com/DIYgod/RSSHub)

#### **工具简介**

[RSSHub](https://rsshub.js.org/) 是国人开发者 [@DIYgod](https://twitter.com/DIYgod) 制作的 RSS 生成工具（[Github 地址](https://github.com/DIYgod/RSSHub)），可以将微博、Twitter、知乎、即刻、豆瓣中的一些工具生成为 RSS，通过 RSS 服务进行订阅。

#### **一些可以考虑的使用方法**

很多人可能和我一样，会觉得 Twitter 这样的社交服务里的零碎内容，有什么放进 RSS 的必要？但实际上，这个东西确实有它存在的价值。

越来越多的传统媒体甚至产生于互联网的媒体，都出于各种各样的理由关闭了它们的 RSS，但是它们仍然会运营它们的 Twitter 等账户。也就是说，我们没办法通过 RSS 订阅的那些媒体，可以通过订阅它们的 Twitter 账户曲线救国。

通过 RSSHub 这个服务，我们可以非常简单地为任何一个 Twitter 用户生成一个 RSS 链接，比如 Bloomberg 的 Technology 新闻，它的 Twitter 帐号是：[@technology](https://twitter.com/technology)（我也觉得这个 ID 很 diao。），那么它的 RSS 链接就是：`https://rsshub.app/twitter/user/technology`。

我会通过 RSS 扫 Bloomberg Technology 中我感兴趣的新闻，如果想细看，就会打开 Twitter 的链接保存到稍后读（Instapaper 抓 Bloomberg 的效果更好一些）。

![](.evernote-content/9A18884A-8E48-48DF-9070-EBEA2B1A2966/805E34A4-0322-4993-884E-9FF9FE17DF6D.jpg)

把 Bloomberg 文章从 Twitter 抓到 Instapaper

RSSHub 里有着大量内置的 RSS 选择可供大家使用。其中有一个我觉得比较有意思的是「正在上映的高分电影」：`https://rsshub.app/douban/movie/playing/7.5`。

这段链接代表的是抓取**豆瓣**数据库中**正在上映**的**高分**电影，最后的 `7.5` 表示这个分数以上的电影才会显示。同时你还可以设定地区，比如如果你是在北京，你可以在链接后加一个 `/beijing`，也就是 `https://rsshub.app/douban/movie/playing/7.5/beijing`，这样你就可以从你的 RSS 里刷出最近在北京上映的高分电影了。

![](.evernote-content/9A18884A-8E48-48DF-9070-EBEA2B1A2966/450880E6-AE86-4FC7-A486-845AA5B98524.jpg)

在 RSS 客户端中查看高分电影

这些自定义功能对于高玩来说不是什么难事，但这种服务的价值就是让定制成本降低。其实哪怕是高玩也并不愿意每次都做最复杂的事，能简单还是简单得好。

TweetBot 3
----------

![](.evernote-content/9A18884A-8E48-48DF-9070-EBEA2B1A2966/630D74FB-6DE3-440C-AF53-DA60B0B020F9.png)

**Tweetbot 3 for Mac（2018–05–16）**

[下载链接](https://itunes.apple.com/cn/app/tweetbot-4-for-twitter/id1018355599?l=en&mt=8&at=10IJSw)

#### **应用简介**

Tweetbot 是苹果生态里非常知名的一款 Twitter 第三方客户端，开发团队叫 Tapbots，目前在开发的其它产品还有 Calcbot 和 Pastebot。他们的产品以精美的设计和值得玩味的细节闻名，并且每款作品都跟机器人主题相关。

#### **值得注意的更新内容**

**@文刀漢三：**Tweetbot for iOS 是一款明星产品，初代就以精致的拟物设计和强大的过滤功能打响名声。即使到了 Twitter 官方慢慢缩紧第三方权限的今天，也仍有一大批忠实的用户。

然而 Tweetbot for Mac 却是一款命运多舛的产品，初代公测期间就遇上 Twitter API 紧缩，限制第三方客户端的使用人数不能超过 10 万。很快 Tweetbot 就超过了这个限制，后来与 Twitter 官方协商，似乎给予了它们这批最老的第三方客户端一些特权，稍微放开了这个限制。

这次 Tweetbot 3 for Mac 发布，又是几乎顶着官方 API 收紧发布的。Tweetbot 3 发布当天，Twitter 就宣布新的 API 政策，第三方客户端将会延迟一两分钟收到新消息、评论、私信等内容。如果想及时收到新消息，第三方开发者须每月支付 2899 美元订阅费来使用新的 API，并且最高只有 250 名用户可以使用。如果想要覆盖更多用户，则需要咨询 Twitter 获得企业报价。新政策将于 8 月 19 号开始执行。

不能及时收到新消息对用户主要有两个影响：

* 第三方客户端的推送消息将会延迟收到；
* 其他用户发布的新推，也会延迟一两分钟出现在我们的时间线上。

前者可能稍微容易接受，后者在看苹果发布会这种场景下，可能就不太容易接受了。

说回 Tweetbot。这次新版本更像是设计语言的一次进化，但看得出来有些赶工，很多元素都打磨得不够精细，不是我们熟悉的 Tapbots 风格，比如元素杂乱个人简介页面、无法显示转推信息的单条 Tweet 界面、把「转推、评论」等按钮显示出来，却不显示转推和评论数字的时间流。

我个人比较不能理解的是，大家一直在吹捧 Tweetbot 3 的多栏功能，但其实这个功能自从 Tweetbot 1 开始就有了。甚至到了 Tweetbot 3，多栏功能还有退化，变成只能显示当前帐户的其它分栏，而无法同时其它账号，这对于需要同时管理多个 Twitter 账号的人来说是不方便的。

不过值得称赞的是，千呼万唤的夜间模式终于出来了，还有方便的多图翻页功能。

我其实理解这次的仓促发布，Tapbots 团队甚至没有进行任何预告，上架得非常突然，可能就是为了赶在 Twitter 宣布前上架。但我个人也比较信任他们团队对于软件品质的追求，相信他们会慢慢完善这款应用。

如果你不是 Twitter 重度用户，我建议不用着急购买，可以先等它继续完善自身的功能和细节，然后再看 Twitter 新政策对第三方客户端的影响有多大，最后再决定购买。如果这个过程中能等到降价活动，或许再入手也不迟。

Price Tag
---------

![](.evernote-content/9A18884A-8E48-48DF-9070-EBEA2B1A2966/021E612F-8B52-482A-9C58-375C6645F669.png)

**Price Tag 2.5**

[下载链接](https://itunes.apple.com/cn/app/price-tag-find-best-apps/id1166819590?mt=8&at=10lJSw)

#### **应用简介**

Price Tag 是一个应用信息聚合工具，具有应用降价信息查询、应用推荐、分享应用以及换区等功能。

#### **新支持了画中画视频播放**

Price Tag 给我的影响一直以来都是一个更接地气的「App Store」，它的界面样式颇有 iOS 11 风格，而内容则选自国内科技媒体。

这一次更新，它在时间线里加入了视频播放功能，并且支持画中画，可以把视频放在屏幕一角，继续刷别的资讯。

![](.evernote-content/9A18884A-8E48-48DF-9070-EBEA2B1A2966/79A21BA5-E5DC-4096-A5DB-ED639F11A725.gif)

支持画中画视频播放

另外的升级主要在于细节的优化，包括时间线内卡片的样式和应用内 Price Tag 商城的浏览体验等。
