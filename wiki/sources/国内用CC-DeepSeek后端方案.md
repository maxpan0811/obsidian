## 一句话摘要

Claude Code 和 Codex 都支持切换模型，用国内大模型（如 DeepSeek）做后端——无需国外订阅、没有封号风险、额度更便宜。

## 核心方案

用 **CC Switch** 等桌面工具配置 DeepSeek V4 做后端：
- 供应商选 DeepSeek
- 粘贴 API Key
- 填模型名称
- 保存启动

## 解决的问题

| 问题 | 解决方式 |
|------|---------|
| 没有国外订阅账号 | 用国内大模型 API |
| 官方额度贵 | DeepSeek 价格低至 1/10 |
| 封号风险 | 不走官方 API，风险清零 |
