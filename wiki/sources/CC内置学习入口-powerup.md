## 一句话摘要

Claude Code 内置 `/powerup` 命令 — 一组互动演示，展示平时容易忽略的能力，帮你打破旧习惯，发现 Claude Code 的真正工作方式。

## 核心观点

> 很多时候我们学一个新工具，很容易带着旧习惯去用它。表面上是在用新工具，实际还是老一套工作方式。`/powerup` 会让我们重新看看：Claude Code 到底还支持哪些工作方式。

## 哪些能力容易被忽略

- 任务规划
- 上下文管理
- 工作流拆解
- 更完整的开发过程

## 适合谁

- 刚开始用 Claude Code 的人—花几分钟体验
- 已经用了很久的人—也值得花几分钟看一看
