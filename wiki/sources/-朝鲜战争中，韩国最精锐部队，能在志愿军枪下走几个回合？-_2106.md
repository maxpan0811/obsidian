# 朝鲜战争中，韩国最精锐部队，能在志愿军枪下走几个回合？

原创 这才是战争 2019-12-10 17:23:14

上了些年纪的人，都对《奇袭白虎团》印象深刻。这个真实的战例用现代的眼光看，无疑是场经典的斩首行动。不过，文艺作品不可能面面俱到地将细节完整地呈现给观众。但志愿军各级指战员战术谋划的匠心、战场上临机应变、有勇有谋，恰恰就体现在那些细节中。

# 一、“白虎”辨析

“白虎团”其实是志愿军的习惯称呼，它的真正番号是首都师团第1联队。

韩国军队是一支在美国的扶植下，以在旧日军、伪满军、伪警察中服过役的朝鲜人为骨干建立的军队。由于朝鲜曾沦为日本殖民地长久50年之久，因此韩军的组织架构、编制体制和武器装备虽均为美式，但在部队编制的称谓上，却沿袭了旧日军军团、师团、联队、大队、中队、小队的叫法。

![](attachments/e26cd90493b1fef4.jpg)

首都师团是韩军中排名第一的精锐部队，在朝鲜战争之前的战斗中号称“从来没败过”。其成立于1949年6月，由所谓的“首都警备司令部”改编而成。成立时下辖第1、第2、第17、第18联队。朝鲜战争中，首都师团下辖部队多次发生变化，但第1联队始终都在建制内。作为1946年李伪军首批组建的8个步兵联队之一，历史比首都师团还悠久的第1联队毫无疑问是该师团主力。就素来瞧不起韩军战斗力的美军将领，对首都师团却另眼相看，称其为“韩国第一荣誉师”。该师团自称“无敌猛虎”，并以一只张着血盆大口、露着獠牙的黑底金纹虎头作为师徽。而第1联队官兵们不仅臂章上均有虎头师徽，而且联队部里有一面写有“优胜”二字的虎头旗。由于臂章常年暴露在外，日晒雨淋，容易褪色发白，这也许是志愿军称其为“白虎团”的由来。关于这面虎头旗的来历，主流观点认为是李承晚亲授的联队旗。也有观点认为，它其实是韩军内部军事竞赛的优胜旗。但无论如何，都不能改变其为韩国最精锐部队的事实。

阵地战初期，首都师团在东海岸与朝鲜人民军反复拉锯。后调至战线中段，金城以西、上甘岭以东地段与志愿军对峙。在19个月的时间里，该师团构筑了坑道、盖沟、环形战壕和各种明暗火力发射点相结合的半永久性的坚固防御阵地，还得到美军5个榴弹炮营和大量坦克、航空兵的支援。该师团夸口自已的阵地是“坚不可摧”的防线。

1953年，正当朝鲜停战谈判将要达成协议之际，李承晚集团蓄意破坏战俘遣返协议，疯狂叫嚣要“单独干”。为打击敌人的嚣张气陷，志愿军总部决定在金城以南地区发起夏季反击战役，由第20兵团负责具体实施。该兵团将兵力分为左、中、右三个集团。其中西集团右翼68军当面之敌，就是首都师团。

此时的首都师团在一线并肩展开第1、第26两个联队，部署在浦幕的机甲联队为师团预备队。整个师团防御正面10公里、左纵深5公里、右纵深8公里。第1联队将下属3个大队成1个梯队展开于522.1高地、552.8高地及其以北诸高地，防御正面宽约3.5公里，联队部位于二青洞。68军要想楔入首都师团防御纵深，首先得啃下这块“硬骨头”。

# 二、精心谋划

68军决心展开第203、204师实施并肩突击，由203师负责主攻第1联队。

为摸透敌情，志愿军203师师长杨栋梁组织战斗骨干现地侦察，营、连、排干部对各自进攻目标进行摸查，师、团干部也对主突方向的要点进行勘察。在部队占领进攻出发阵地的第4天，203师侦察兵以伏击手段歼灭了韩军第1联队搜索队，捕获5名俘虏。经过讯问，进一步查明了敌军兵力部署和工事构筑情况。

于是203师决心集中力量，在韩军第1联队右翼与第26联队的结合部实施突破，之后第二梯队从突破口沿山谷直插敌纵深，在向纵深发展中从东向西攻击其左翼阵地，一举将其整个防御部署打乱。鉴于韩军第1联队阵地无暴露的翼侧，杨栋梁决心派出4路穿插分队：第1路为班级分队，化妆成李伪军，沿直木洞沟直插二青洞，以奇袭手段端掉第1联队联队部。第2路为营级分队，紧随第1路，直插北亭岭、下榛岘、梨实洞和下枫洞地区，坚守阵地断敌退路。第3路为连级分队，沿上甘岭下边直插敌后方，断敌退路。第4路沿粟洞沟直插朱罗池，偷袭支援第1联队作战的美军第555炮兵营。

![](attachments/07c3ce621560a475.jpg)

方案确定后，各分队都积极请战。师党委研究决定，派全师战绩最佳的607团侦察排1班执行第1路穿插任务。这个由10人组成的侦察班成员均是立过战功的中共党员，在调入侦察班之前都担任过班长、副班长职务。为加强领导，杨栋梁让607团侦察排代理排长杨育才带队，另外从全师挑出2名军事素质好、有战斗经验的朝鲜族战士韩淡年、金大柱当联络员，归杨育才指挥，共同组成一个13人的化袭班，执行端掉敌联队部的艰巨任务。

另外几路穿插分队的人选也尘埃落定：609团副团长赵仁虎率该团2营为第2路；师侦察连和608团侦察排为第3路。第4路由607团侦察排2、3班共22人组成。

杨育才接到任务时，离进攻发起时间只有3天。化袭班在短短3天时间里，又是捕俘审俘，又是研究穿插途中可能会遇到哪些问题，该如何应对，又是忙着搞演练，忙得不亦乐乎。因为事关重大，师、团首长也抽空参加他们的集体讨论。最后，大家决定由身材高大的杨育才化妆成美军顾问，韩淡年化妆成护送顾问的首都师团搜索队小队长，其他战士化妆成队员。路上遇到盘问，由韩淡年、金大柱出面应付。他们共携带8支M1卡宾枪．2支M1伽兰德步枪和3支M3冲锋枪。每人除携带300发子弹外，还配有1支51式手枪、8枚美制手榴弹、1把匕首和3天的干粮。

在1951年夏秋防御战中，203师曾在二青洞一带战斗过，对地形并不陌生。通过审俘获知韩军第1联队联队部外围防卫和内部结构后，杨育才将化袭班分成3个战斗组。第1组负责歼灭敌警卫排，并支援其他各组的攻击。第2组负责歼灭敌炮兵指挥部，第3组负责打作战室。

为给化袭班渗透敌后创造有利条件，203师专门指定了1个炮兵连，负责沿化袭班预定渗透路线逐段炮击护送。

# 三、敌后穿插

7月13日晚，乌云密布，大雨欲来。化袭班隐蔽在我前沿阵地的1条小坑道内待机。609团2营的600余名指战员反穿雨衣，潜伏在白杨里的沟壑里。

21时，志愿军20兵团所属各部和第9兵团24军在千余门大口径火炮支援下，经过7至28分钟火力准备，同时向金城突出部内李伪军4个师团阵地发起猛攻。

203师609团1营猛攻522.1高地。14日零时，杨育才率领化袭班出击，从522.1高地东侧与415高地之间的鞍部渗透入敌阵地。4分钟以后，赵仁虎率609团2营也从这里向敌后穿插。其中，5连由三里南直插上枫洞东南的421.2高地，断敌退路；4连直插二青洞，配合化袭班打掉第1联队部及附近炮兵群，然后攻占下枫洞以南地区，阻敌增；6连为营预备队。

化袭班跑步通过500米开阔地带不久，尖兵赵顺合突然低声报告踩着了地雷。行进在队伍中间的杨育才让别人隐蔽，自己跑过去小心翼翼地把地雷两边的土扒开，发现是枚反坦克雷，虚惊一场。为避开雷区，化袭班此后踩弹坑、淌水沟，一路跑步前进。炮弹在附近爆炸，子弹在身边掠过，他们不得不常常卧倒在泥水里。

![](attachments/68c5afb1ae83a6f5.jpg)

紧随其后的609团2营遇到雷区后，营长石广志判断，这应该是个反坦克和反步兵混合雷场。反步兵雷应该被我军炮火准备摧毁得差不多了，但也难保有漏网之鱼。此时，敌阵地上的残余火力点仍在拼命喷吐着火舌，敌纵深炮火也一阵紧似一阵地袭来，在雷区多耽搁一分钟就多一分危险。于是，石广志命令4连排成几路前进。战士们挽起裤脚和袖管，用皮肉去触查地雷，很快就为全营蹚出了一条安全通道。

在415高地前，化袭班发现两边山头守敌不停地向山下倾泻弹雨，但有个小山包却毫无动静。赵顺合摸上去一看，发现阵地上只有2具敌尸，其他人已不知去向。于是化袭班从这个小山包迅速通过了敌主阵地，钻到了敌后。

1时10分左右，当队伍行至415高地南侧约500米处时，借着敌机投下的照明弹亮光，杨育才清点了一下人员，结果居然点出了14个人头。他生怕自己看花了，反复数了4遍，确信队伍里多了个人。于是，他用暗号通知化袭班停下，一把揪出了1名跟在队伍后面，听不懂暗号的韩军。一审问，原来此人今晚在415高地执勤，志愿军的炮火击毙了他的2个同伴，失魂落魄的他吓得趴在草丛里。见杨育才一行人匆匆而过，他以为是前沿阵地下来的溃兵，于是便起身跟着跑。问出了当晚第1联队的口令“云雹”，化袭班给他嘴里塞上毛巾，捆牢四肢，丢在路旁草丛中。

处理完俘虏，化袭班继续赶路。刚一拐弯，又碰到2个韩军，大喊“干什么的？”联络员金大柱回答：“自己人”，接着立即反问：“你们是干什么的？”敌人回答：“我们是站岗的。”见眼前这10余人继续赶路，其中1名韩军一边喊“站住”，一边走过来想看个仔细。金大柱立即迎上去骂道：“你狗眼瞎了，连老子都不认识了。你看不到我们是护送美军顾问到团部去吗？误了事，要你的脑袋。”这一通骂，把敌人给镇住了。韩淡年冲金大柱嚷：“还不快走，跟他啰嗦什么！误了事，枪毙你”。金大柱假装害怕地答应：“是，是，是”。就这样，化袭班顺利通过岗哨。

此后，化袭班一路上多次遇到敌人。韩淡年通过抢先问口令的方式，核实了口令的真实性。之后凭借口令一路过关，通过勇进大桥后，接近了二青洞，迎面遇上了负责韩军联队部外围警戒的巡逻队。韩军见是“自己人”快速奔过来，大声喝问“哪部分的”。韩淡年回答：“师部搜索队，奉命护送美军顾问到联队部”。边说边不停歇的快速前进。敌巡逻队眼睁睁看着他们从眼前通过。

化袭班一路顺风顺水，紧随其后的609团2营就没这么幸运了。因为不知道口令，2营在通过韩军一处警戒阵地时，5连9班以突袭的方式解决了敌人。此后， 4连发现迎面开来约1个中队的李伪军。考虑到天降大雨，能见度很低，为不影响穿插速度，4连依旧步履匆匆，全连保持肃静前行，希望能蒙混过关。两支队伍交错时，一些韩军起了疑心。4连抢先开火扫射，当场歼敌大部，少数残敌乘着黑夜落荒而逃。

# 四、大打出手

![](attachments/75835fc4c314a56c.jpg)

志愿军发起攻击后，首都师团的防御态势很快便出现了摇摇欲坠的趋势。师团长崔昌颜少将急令机甲联队增援阵地多处被突破的第1联队，并派副师团长林益淳准将前往二青洞第1联队部协调。

第1联队联队长崔喜寅在14日零时过后，将联队部所有能抽出的人员，都派出把守二青洞周围的几个山口。自己与率部赶来的机甲联队联队长陆根洙上校商量反击事宜。二人为确定反攻方向争吵起来，互不相让，最后由随后赶到的林益淳拍板定案。陆根珠匆匆离去，与他所部前锋第2大队会合。

1时55分左右，化袭班在二青洞附近公路上，与机甲联队第2大队50余辆运兵车组成的车队相遇。他们在二青洞村北迅速隐蔽，准备放过这批敌人后再越过公路。当驶过30余辆汽车后，车队前方忽然枪声大作，并伴随着一串串爆炸声。609团2营和这股敌人接上火了。

原来，远远看到敌车队开着大灯源源驶来，带队的赵仁虎和石广志紧急商量了一下，命令6连出击。6连指导员率领3排冲上去。当接近到距汽车200多米时，郭树昌令9班拦头、8班斩腰、7班断尾。3个班迅速散开，各自抢占有利地形。

9班班长刚要开火，被郭树昌一把按住。他发现了第3辆汽车后跟着一辆坦克，上面还搭着10余步兵。“先干掉敌人的坦克！”郭树昌抓起两颗反坦克手雷，悄悄绕到坦克侧面。待坦克接近时，他突然跃起，投出反坦克手雷。“轰”的一声巨响，坦克被炸瘫了。几乎与此同时，3个班一齐打响，手榴弹接二连三地在敌群中开花。毫无心理准备的韩军们被打得抱头鼠窜。

战斗一打响，整个车队都停了下来，把公路堵得死死的。韩军们纷纷从车上跳下来，哇哇乱叫。如果他们集结起来，堵在二青洞口，化袭班就没法完成任务了。杨育才当机立断，令部下2人一组打1辆车，趁乱迅速冲过公路，到二青洞口白杨树下集结。

韩军们没弄清楚车队前锋到底遇到了啥情况，更没想到车队中部拦腰杀出一支奇兵。在冲锋枪、卡宾枪扫射，手榴弹接二连三的爆炸声中，机甲联队官兵还以为和第1联队的溃兵发生了误会。化袭班并不恋战，冲过公路会合后，第1联队部门口的哨兵正待询问，大家一拥而上，以迅雷不及掩耳之势扭断了他们的脖子，冲进联队部大院。

![](attachments/c95c515360a2885e.jpg)

此刻，敌联队部大院四周的探照灯都开着，照得如同白天。院内停满了吉普车和汽车，人来人往，喧闹异常。见敌人远比预想的多，杨育才临时将原来3个战斗组拆分成4个：第1组4人打警卫小队；第2组3人打炮兵指挥部，得手后再攻联队部。第3组3人打作战室。第4组3个人由杨育才率领，负责歼灭混乱之敌，炸毁汽车，并支援第2、3组战斗。以第1组打响为令。

第1组首先打掉了警卫小队住处门口的2辆吉普车，然后冲进屋扫射。残敌跳窗逃跑，该组追了5、6里路，到达了609团2营4连预定穿插终点下枫洞。此地是敌人退却，增援的必经之路，该组索性留在当地与敌人激战。打退敌3次进攻后， 4位孤胆英雄弹药用尽，正在敌尸体上搜子弹时，又冲过来6个韩军。在4比6的肉搏中，他们以1人轻伤的代价，毙敌2人，俘敌2名。靠缴获的敌武器弹药，他们又打退了敌人2次反击，击毁不少敌汽车，将三岔路口堵得死死的。直到4连1个班赶来，4勇士才交接阵地，赶回二青洞。

第1组打响后，第2组往敌炮兵指挥部屋内扔了几颗手榴弹，不等硝烟散尽，2人冲进屋扫射。残敌从后门逃跑，正撞在被隐蔽在后门的第3名侦察兵枪口上。

第3组先往作战室里投弹，再冲进去扫射，打得满屋的韩军军官哭爹喊妈。林益淳和崔喜寅见势不妙，趁乱逃窗逃跑。不过，天亮之后，林益淳还是被204师俘获。只有崔喜寅带着2个参谋最终逃脱。

第4组在杨育才率领下，先打灭院里的探照灯，然后在院子里来回冲杀。完成任务的2、3组也随即加入战团，李伪军们不知道来了多少志愿军，组织不起抵抗，战斗仅持续了14分钟便宣告结束。打扫战场时，化袭班在床底下和草丛中搜出了浑身发抖的第1联队人事科科长、副官、炮兵副大队长等8名军官，总计俘虏了16人，现场清点出54具敌尸，缴获枪支76支（挺）、电台2部、总机2部、发电机2部、报话机、电话机117部、吉普车27辆、卡车11辆、火箭炮1门，还有两个大仓库和其它很多器材。作战室墙上那面写有“优胜”字样的虎头旗，也被化袭班扯下来当收缴战利品的包袱皮用。

![](attachments/585ad0c3495f1022.jpg)

# 五、扩大战果

化袭班圆满完成任务之时，609团2营也以坚决果断的行动，消灭了机甲联队第2大队，并在混战中击毙了联队长陆根洙上校。

继续向穿插终点赶路的6连7班刚翻下公路，就发现了1个正在开火的美军火箭炮连。该班摸到近前，发现阵地四周围着铁丝网，网上挂满了地雷和照明雷，只有较远的地方有条臭水沟通向网外。这时排长扛着1挺重机枪赶来，在重机枪的掩护下，班长黄在渔带着3名战士悄悄地从水沟中钻进了铁丝网内。正当他们一步一步靠近敌人时，重机枪突然出了故障。就在我方火力减弱时，敌人发现了他们。顿时，几十支卡宾枪疯狂地扫射过来。战士马头保当场牺牲，战士张树勤和另一名战士腿部中弹伤势严重，黄在渔左肩中弹，鲜血直流。但他仍然大喊：“同志们，坚决拿下炮阵地！”张树勤用左胳膊支地，用右手单臂举枪向敌人射击，黄在渔和另一名战士边射击边向敌人逼近，并投出了几排手榴弹。敌人开始动摇，他们趁机又是一阵猛扫，毙敌20余名，夺取了4门火箭炮。

2时40分，2营一路经过大小11次战斗，赶到了穿插终点梨实洞和上枫洞公路的三叉路口。该部正欲攻占421.2高地时，发现前方有数处灯光，并传来汽车轰鸣声。石广志命令部队分兵迂回，包围设伏。待敌近到眼前时，发现是正在向前运动的美军榴弹炮部队。2营以锐不可当之势迅速将美军纵队分割包围。顷刻之间，猝不及防的美军大部被歼灭，19门榴弹炮都成了2营的战利品。清扫战场时，志愿军战士发现了一面绣有“555”字样的军旗，这才知道他们消灭的是美军555炮兵营。

因为被打掉了指挥中枢，第1联队的防线很快便崩溃了。其中第1大队被全歼，第2大队溃败后，原1000人中只收容到280人，遭重创的第3大队因为溜得快，侥幸逃脱了覆没的命运。此外，该联队建制内和临时配属作战的炮兵部队基本被歼。68军经一夜激战，在10公里宽的正面上前进了6至8公里，顺利完成了上级赋予的第一阶段任务。到7月14日下午，战前员额1.2万人的首都师团仅收拢了3000残兵败将。号称“不败”的韩国最精锐部队，只1天就被志愿军打垮了。

![](attachments/ecaad00e480224c7.jpg)

战后，609团2营被志愿军总部授予“一等功臣营”，该营4连荣立特等功，许多人被评为战斗英雄。杨育才被记特等功一次，并授予他“一级英雄”称号。

至于美军第555炮兵营营旗，以及从李伪军第1联队联队部缴获的“优胜”虎头旗，则由中国人民革命军事博物馆收藏至今，成为那段传奇战史的历史见证。
