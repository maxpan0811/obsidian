特斯拉 Model Y 发布，3.9 万美元起，明年交付
============================

![](.evernote-content/0B48214F-785F-44C4-A471-D85C8A5F0E65/012D2BD0-98FA-4B51-A58E-45143336CD79)

S-E-X 产品线坚持了三年，特斯拉补上了最后一个 Y—— 从 Sex，到 Sexy。

你或许有所耳闻，2016 年发布、2017 年交付、2018 年入华的 Model 3，马斯克最初给它起名 “Model E”，与 Model S、X 和刚刚发布的 Model Y，一道组成特斯拉 SEXY 天团。然而老前辈福特公司不作美，坚持自己对 “Model E” 商标的所有权。特斯拉只得抱憾退而求次，将 “E” 掉转化为 “3”。S3XY 的表象，藏着 SEXY 的心。

Model Y，四巧板拼图的最后一块。

Model 3 它亲哥哥

没有任何惊喜，也不出任何意外，Model Y 就是我们以为的那样：它与 Model 3 的关系，正如 Model X 之于 Model S；它与 Model X 的关系，正如 Model 3 之于 Model S。简而言之，它是一辆 “Model X 化” 的 Model 3。

相对而言，**Model Y 更像是一辆 “Model 3 SUV”**，而不是一辆 “Model X Mini”。

![](.evernote-content/0B48214F-785F-44C4-A471-D85C8A5F0E65/B39641FA-0E15-4D98-B4BC-B45C745F3D51)

内外设计体现得很明显。Model Y 车头几乎是照着 Model 3 原样刻画，车头全封闭、日行灯位于灯组上部内侧、下层保险杠结构都与 Model 3 如出一辙；车身后半部包括车顶线条，则是还原了老大哥 Model X 的 fastback 溜背风格，即如今流行的所谓 “轿跑 SUV”。如果把 Model 3/Y 与 Model S/X 摆在一起，你很容易区分特斯拉的新旧两种设计语言。

车内同样是分外眼熟，Model 3 标志性的无仪表盘、15 英寸全触屏、轻体量的中控台、隐藏式空调出风口。马斯克曾在推特上公开表示 Model Y 有 75% 部件与 Model 3 通用，由此可见一斑。

令人意外的是，体型没比 Model 3 大多少的 Model Y 竟然可以选择七座版本。不过由于车顶下溜曲线，第三排的头部空间很是局促，恐怕只能用来临时应急，想必七座版也不会成为多数人的首选。

续航、性能有降也有升

因为（同等车长尺寸下）SUV 会比轿车更重、更高、风阻更大，Model Y 的续航理应比同电池的 Model 3 稍短。这在同电池的 Model X 与 Model S 续航对比中已经有所体现。其实即便对于汽油车，配备相同动力系统的前提下，同尺寸的 SUV 也会比轿车跑得更慢、油耗更高。

![](.evernote-content/0B48214F-785F-44C4-A471-D85C8A5F0E65/5259E059-C5ED-4B09-9212-485E39344424)

然而令人意外的是，Model Y 的几个版本相比 Model 3 并非一致性的下降。考虑到 Model Y 的底盘基础（尤其是电池组）很大可能与 Model 3 通用，这或许是由于特斯拉对于 Model Y 的软件优化所致。就在 Model Y 发布前夕，特斯拉刚刚表示 Model 3 可经软件优化稍稍提升性能。

根据续航不同，Model Y 分为三个版本：

* Model Y：370km（标准续航）、480km（长续航）、450km（双电机、性能版）

对比一下 Model 3 的几个对应版本：

* Model 3：354km（标准续航）、523km（长续航）、499km（双电机、性能版）

需要注意，以上续航数字均为美国 EPA 循环标准，与欧洲和国内常用的 NEDC 标准无可比性。EPA 续航 523km 版本的 Model 3，在国内 NEDC 标准下的综合续航可达 590km。**以此差距估计，Model Y 进入国内后按照 NEDC 标准公布的综合续航，大致可能在 418、542、508km（估计仅供参考）**。

![](.evernote-content/0B48214F-785F-44C4-A471-D85C8A5F0E65/C180654F-32AC-4193-A5D2-C869586A8AAD)

虽然是 SUV 身份，Model Y 并非全系皆为全轮驱动，只有双电机、性能版为双电机全驱，而标准续航、长续航版为单电机后驱。特斯拉按照惯例并未公布功率和扭矩，不过搭载的很可能就是 Model 3 所用电机。各版本的 0-96km/h 加速成绩分别为 5.9（标准续航）、5.5（长续航）、4.8（双电机全驱）、3.5（性能版）秒，如前文所说，相比 Model 3 有微小幅度下降。

今年年初，马斯克就曾经放出豪言，要在今年底前实现全自动驾驶。Model Y 同样搭载着目前特斯拉最新的 2.0 版 AutoPilot 硬件，具备在将来 OTA 升级至全自动驾驶能力。

Model Y 各版本售价：

* **标准续航：370km、5.9s、3.9 万美元（26.2 万元人民币）；**
* **长续航：480km、5.5s、4.7 万美元（31.6 万元人民币）；**
* **双电机全驱：450km、4.8s、5.1 万美元（34.3 万元人民币）；**
* **性能版：450km、3.5s、6 万美元（40.3 万元人民币）**

四部曲后续

S-E (3)-X-Y 四部曲画上句号，特斯拉的下一步会朝那儿走？除了这四大核心车型，特斯拉还有（以及即将有）三条 “支线”：超级跑车 Roadster 2、电动卡车 Semi 和确定正在开发的电动皮卡（未定名）。

![](.evernote-content/0B48214F-785F-44C4-A471-D85C8A5F0E65/981C1DD8-A6C6-452C-BB70-3E9912EF15D1)

这其中 Roadster 2 和 Semi 都已经完成亮相，前者计划在 2020 年交付，虽然售价超过百万人民币，但其实做超跑能不亏钱就是福音；后者交付时间同样定于 2020 年，发布伊始订单就纷至沓来。Roadster 2 是特斯拉炫技的顶点，Semi 则有望开辟商用车阵线。尚未露面的特斯拉电动皮卡，应该不会让我们等太久，2020 年前到来的可能性不小。

另一大重要工作是老车型的迭代更新，眼下 Model S 已经卖了足足 7 年，期间经历了一次外观改款和几次电池升级，按照惯例十分接近换代窗口。Model X 自 2015 年首次亮相，至今尚未获得大幅度的改款和升级。

考虑到特斯拉上海工厂会在今年底投产，中国消费者有很大希望，第一次与美国市场同步喜提特斯拉新车 Model Y。
