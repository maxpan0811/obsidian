# 第2088期：We don’t have to always agree with each other

---

速读摘要

难过的不是我们观点不同，而是这么多年了，仅仅因为观点不同仍然会瞬间反目。我写这个公众号七八年了吧，老读者有过好几批，也走了好几批。尤其前几年我被困在家那段日子经常忍不住大骂讽刺揶揄那些蠢猪们，引起了一些读者对我的不满，就取关了。我们平时所说的"谁谁谁三观很正"其实真实的意思是"谁谁谁三观跟我一致"——这个世界已进入全方位二元割裂世纪，环球同此操蛋，加上人人手里的手机和社媒对"同温层、信息茧房"效应的无限放大和加持，双方各执一词，彼此都觉得自己掌握着宇宙真理，别人都是白痴。

每天都有新读者关注，也经常有人取关。取关的原因很多，我不在意，只是每次有读者（尤其多年的老读者）因为我的某句话与他观点不同而愤然离开时，我心里仍然有些难过。

难过的不是我们观点不同，而是这么多年了，仅仅因为观点不同仍然会瞬间反目。

![](.evernote-content/09650BE2-F1C6-4E15-BB7A-011F2FE2391D/F11E1944-3144-4C7E-82A3-05BCB5B7F4C9.jpg)

我平时不太关注时事，即便偶尔关注，也尽量不直接评论，有时候忍不住揶揄两句，稍稍出口恶气，也改变不了什么。

有句话糙理不糙的话——**观点这东西，就像屁眼儿，人人都有一个**。

人人都有的东西，网上也不多我一个屁眼儿，所以我干脆少说话或者尽量不说。

但谁都有忍不住的时候，有些话说了就容易引发争议。

我写这个公众号七八年了吧，老读者有过好几批，也走了好几批。尤其前几年我被困在家那段日子经常忍不住大骂讽刺揶揄那些蠢猪们，引起了一些读者对我的不满，就取关了。我能明显感觉到哪些事情上哪些人是站在哪边的，也一直提醒自己看破不说破，能笑而不语就不语，最好面无表情做只模范僵尸。

我们平时所说的“谁谁谁三观很正”其实真实的意思是“谁谁谁三观跟我一致”——这个世界已进入全方位二元割裂世纪，环球同此操蛋，加上人人手里的手机和社媒对“同温层、信息茧房”效应的无限放大和加持，双方各执一词，彼此都觉得自己掌握着宇宙真理，别人都是白痴。

如今任何事情都会立即分为两派观众，哪怕只是围观一泡屎，也会有一半人支持稀屎，另一半人支持硬屎。

人人都有观点，人人都有发表自己观点的权利，我只希望如果有一天你在我的文章里看到了让你火冒三丈的观点或语言，请勿生气，也不要给我留言说一句气哼哼的告别语，直接取关，不再看我的文字即可。

![](.evernote-content/09650BE2-F1C6-4E15-BB7A-011F2FE2391D/C65C3761-8C0B-437A-88A3-4A6E75205939.jpg)

We don't always need to agree with each other—we really don't.

I enjoy writing, so I write every day.

You find my writing readable, so you read it. If you don't enjoy it anymore, then don't read it lah. Unfriend me and leave. There is no need to tell me why you don't like me anymore.

As easy as that. Let's keep it simple, shall we?
