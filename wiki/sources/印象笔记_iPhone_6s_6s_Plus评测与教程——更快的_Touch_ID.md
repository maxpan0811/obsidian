刚用 iPhone 6s 和 iPhone 6s Plus 的用户将发现一个非常明显的变化，那就是 Touch ID 变快了许多，这是因为苹果给 iPhone 6s 和 iPhone 6s Plus 设计了全新的 Touch ID 传感器。其中最能够证明 Touch ID 改进的一个例子是：汗手的识别率提高了。  
  
　如今即使手指上有粘糊的汗，Touch ID 的识别率已达到 100%。但是在 iPhone 6，汗手的识别率是没有 100% 的。而且不知道大家有没有这样的情况，越是识别不了，越紧张，手指的汗液就会越多，最后只好重新输入密码解锁。这种情况在 iPhone 6s 和 iPhone 6s Plus 已经完全消失。当然，如果手指不是粘糊的汗液，而是湿漉漉的汗水或者水迹，依然会影响 Touch ID 的识别率，需要擦干才能识别。  
  
　在正常使用中，Touch ID 的识别速度快到令人难以习惯，还以为没有锁屏呢。手指轻碰就能识别，用户看到了实实在在的提升。
