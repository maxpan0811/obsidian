# 征服帕米尔高原

---

| |  |  | | |  | | --- | | □冯铁兵  新疆和平解放后，人民解放军二军进驻南疆，根据中央军委和第一野战军的指示，派出部队开往边境第一线，接管边界防务，保卫祖国领土完整和边防安全。  当时，帝国主义分子和边境内外的反动势力勾结在一起，不断派遣间谍、特务，策划武装叛乱。因此，进疆部队迅速进驻边防一线，守卫边卡，这是一项极为迫切的任务。  南疆地处祖国的西陲边境，除中苏边界外，与阿富汗、巴基斯坦、印度等国家接壤的边境线长达１０００　多公里。而中阿、中巴、中印边境线大都在海拔五六千米以上的帕米尔高原和喀喇昆仑山脉的崇山峻岭之间，空气稀薄，终年积雪，荒无人烟，许多边卡根本没有道路。二军根据情况，决定以第四师、第五师和第十三师（　民族军改编）　一部分接管各边卡防务。  当时，南疆各边卡的国民党部队已参加了起义，但时值严冬，有的边卡道路被冰雪封闭，不能通行。１９４９　年１２月，四师十一团副参谋长李鹿鸣和一营教导员石明率两个连进驻地处帕米尔高原的蒲犁（今塔什库尔干）接管边防。  帕米尔高原平均海拔４０００　多米以上，冰山雪岭，高寒缺氧，暴风雪肆虐，生活环境之恶劣，已超过人类所能承受的极限。但是，为了保卫边防，捍卫国家主权，我英勇的指战员毅然向帕米尔高原前进，向生命的极限挑战。  １２　月１０　日，李鹿鸣和石明带领着两个连的战士，从英吉沙出发，向帕米尔高原进发。５　天后，部队爬上了海拔４０００　多米的雪山。忽然间，天气骤变，狂风大作，大雪纷飞，风卷着雪粒打得人脸都麻木了，根本睁不开眼睛，看不清路，部队只好停下来，原地休息。战士们又冷又饿，大家坐在雪地里，一把雪一把干粮当做午餐，一个个满身白雪，眉毛、胡须也结了冰霜。大家乐呵呵地说：“我们都是高原上的雪人了。”  大雪过后，天气更加寒冷，很多战士的手脚出现了冻伤，行走非常艰难。经过７　天的高原行军，这支顽强的队伍终于登上了海拔６０００　多米的帕米尔高原最高峰。从此，五星红旗飘扬在了帕米尔高原上。  １９４９　年１２　月，五师十五团特务连进驻皮山县桑珠镇。１９５０　年５　月，特务连以一个加强排驻皮山县通往印度的要塞赛图拉，并设立了指挥所。同年５　月，四师十一团一个加强排进驻叶城县通往印度、巴基斯坦的要塞英扎大拉。到此为止，二军已经完成了南疆境内的中阿、中巴、中印边防线上所有哨卡的接防工作，将祖国的边界线一寸不少地从旧政府的手中接管了过来。  １９５０　年９　月，第十三师一个加强骑兵连也进驻蒲犁。骑兵连以蒲犁为指挥所，分驻于通往阿富汗、巴基斯坦、印度各要塞哨卡，每个哨卡相隔上百公里，有的哨卡只有几个人，他们必须忍受着长年的严寒、寂寞和经常断粮造成的饥饿。  进入蒲犁各边卡后，隐藏在内部的反革命分子便在暗地里开始积极活动。窃据副排长职务的原国民党边卡人员张同兴，煽动驻明铁克的原国民党边卡人员叛变。他们凶残地杀死了３名老战士，并绑架我边防人员６　人，劫走军马１９　匹、轻机枪两挺、冲锋枪一支，逃入巴基斯坦所属的坎巨提。此事发生后，我军又发现隐藏的反革命分子王荣、韩文等准备煽动原国民党边卡人员叛变。二军郭鹏军长得知后，大怒：  “这些反革命分子太猖狂，对他们绝不能手软！”他急令四师十团组织一个加强连进驻蒲犁各边防哨卡，对残余的叛乱分子进行了清查。  １９５２　年３　月，为了统一边防哨卡防务，二军教导团三营组成边卡营，接替四师、五师部队所担负的边境哨卡防务，专门负责边防守卫任务。  １９５３　年，教导团边卡营整编为四师十一团三营，三营十连进驻蒲犁，十一连进驻皮山，十二连进驻乌恰。守卫边卡的部队常年执勤在海拔５０００　米的崇山峻岭之间，那里终年积雪，空气稀薄，冬季最冷时，气温经常在零下４０　摄氏度，冻得枪栓都拉不开。夏天高原气候则变化无常，时雪时雨。哨卡的营房已年久失修，常常是外面下大雪，屋里下小雪；外面刮大风，屋里刮小风。战士们开玩笑说：“我们的营房像座碉堡，到处都是‘机枪眼’。”  从山下往山上运送物资，是一项最困难的工作，运输线长，根本就没有路可走，只能靠耐力很强的骆驼、牦牛、毛驴当运输工具。每年１０　月开始到第二年５　月上旬，大雪封山，不能通行，一切物资运输都停止了，山上的指战员们只能靠储存的食物维持生存。他们常年吃不上青菜和新鲜食物，几乎都出现了营养不良的症状，身体十分虚弱。但是，指战员们为了完成守卫边防的任务，仍然保持着乐观的精神面貌。他们顽强地开展爬山竞赛和马术训练，学会了骑马、爬雪山、过冰河、马上劈刺和骑马射击，日日夜夜守卫和巡逻在边防线上。  英雄的边防战士面对如此艰苦的环境，靠着坚定的革命信念，始终保持高昂的革命斗志，战胜了各种艰难困苦，谱写了一曲战天斗地、保卫边防的壮歌。 | | | | | |
