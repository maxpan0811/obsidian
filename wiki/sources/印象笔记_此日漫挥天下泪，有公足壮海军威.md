# 此日漫挥天下泪，有公足壮海军威

---

```
邓世昌  
1895年，黄海大东沟海域爆发了世界上第一次铁甲舰与铁甲舰对决的大海战，中国北洋舰队被日本联合舰队重创。撞击日舰吉野号失败后，邓世昌与致远舰共存亡，和爱犬一起自沉殉国。
```

光绪皇帝闻讯，悲痛中写下挽联：“此日漫挥天下泪，有公足壮海军威”。并赐予邓世昌“壮节公”谥号,追封“太子少保”,御笔亲撰祭文、碑文各一篇。
