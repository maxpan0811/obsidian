好用的 Chrome 扩展，我们又为你找到了这 10 款
============================

2小时前

[![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/B201785B-4951-4652-837E-2D9CFE7EC186)](https://sspai.com/user/713147)

#### [Umi](https://sspai.com/user/713147)

目录

Chrome Web Store 每天都会上架新扩展，但由于无法按时间检索，往往只能仰赖开发者自行推广。正因如此，少数派会从各渠道持续关注优质扩展的发布动态，并定期集中推荐给大家。

今天，少数派就为大家精选了 10 款近期上架的优秀 Chrome 扩展，希望它们能让你的浏览体验更加完美。

Goalhacker
----------

总是忘了今天要干什么？不如试试 [Goalhacker](https://chrome.google.com/webstore/detail/goalhacker/oemenifjoenjjoelohniboopoimfkncb "Goalhacker") 这款扩展，将待办、日历和笔记都搬到 Chrome 的新标签页上面吧。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/713F2E94-CB97-43EB-BBF5-E4ACD8CF0521)

安装并使用 Google 账户登录后，Goalhacker 就会自动同步你的日历信息，并显示在侧栏中。你也可以在新标签页直接创建笔记和待办，并指定截止期限，杜绝遗忘。除此之外，Goalhacker 还支持团队协作功能，你可以通过发送电子邮件邀请同事加入当前 Space，共同编辑任务。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/D9D6E817-D89F-4B4A-BEAB-528DBA05F5D9)

你可以在 [Chrome Web Store](https://chrome.google.com/webstore/detail/goalhacker/oemenifjoenjjoelohniboopoimfkncb "Chrome Web Store") 免费获取 Goalhacker。

GITHUBER
--------

同样是为提高效率而生的新标签页扩展，[GITHUBER](https://github.com/zhuowenli/githuber "GITHUBER") 将大部分精力都集中在了程序员们的身上。通过抓取 GitHub Trending 列表，你可以在首页浏览任意编程语言的热门 Repo 列表，提高工作效率。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/EA51E62C-94AD-4730-B4CD-5459604E0ABA)

此外，GITHUBER 还内置了 Google、百度、必应、雅虎四种搜索引擎。你也可以在左侧边栏添加常用网站，实现一键访问。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/53DFEA21-8296-4C4C-8928-BFD74CD26867)

GITHUBER 在 [GitHub](https://github.com/zhuowenli/githuber "GitHub") 免费开源，你也可以从 [Chrome Web Store](https://chrome.google.com/webstore/detail/githuber-%E5%BC%80%E5%8F%91%E8%80%85%E7%9A%84%E6%96%B0%E6%A0%87%E7%AD%BE%E9%A1%B5/janmcneaglgklfljjcpihkkomeghljnf "Chrome Web Store") 直接获取。

DuckDuckGo Privacy Essentials
-----------------------------

随着互联网产业的高速发展，个人隐私问题已经成为人们顾虑的焦点。如果你不想让自己的私密数据被明码标价出售，不妨试试 [DuckDuckGo Privacy Essentials](https://duckduckgo.com/app "DuckDuckGo Privacy Essentials") 这款扩展。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/19BD0FE3-E102-4EF8-8BC7-CE7A3FF77047)

你很可能已经听说过 [DuckDuckGo](https://sspai.com/post/40117 "DuckDuckGo") 的大名，这是一家不会收集隐私、完全依靠社区运营的搜索引擎提供商。而作为其官方浏览器扩展，DuckDuckGo Privacy Essentials 可以帮助你发掘和阻止网页上不为人知的跟踪代码，并对整体安全性进行评估。

你可以在 Chrome Web Store 下载安装 [DuckDuckGo Privacy Essentials](https://chrome.google.com/webstore/detail/duckduckgo-privacy-essent/bkdgflcldnnnapblkhphbgpggdiikppg "DuckDuckGo Privacy Essentials")，也可以从 [官网](https://duckduckgo.com/app "官网") 获取更多平台应用。

达达划词翻译
------

生词翻译扩展可能是竞争压力最大的领域之一，少数派此前也曾推出过一篇 [专题介绍](https://sspai.com/post/44139 "专题介绍")。而 [达达划词翻译](https://chrome.google.com/webstore/detail/%E8%BE%BE%E8%BE%BE%E5%88%92%E8%AF%8D%E7%BF%BB%E8%AF%91/cajhcjfcodjoalmhjekljnfkgjlkeajl "达达划词翻译")，则是在此之后诞生的又一位高颜值的实力派。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/304208FA-463D-483A-8EB7-ECE80C0E3F85)

与大部分同类扩展相似，安装完成后，只需要用鼠标选中你想翻译的词组或段落，再点击浮现的达达图标，就能查询到翻译后的文本了。其中，语段翻译由搜狗提供支持，而单词翻译则来自权威牛津词典。此外，你还可以直接点击查看词根词缀，辅助记忆。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/9D48E437-B2D0-4F27-96CD-885ED00868D7)

查完单词，如果不加整理记忆，下次看到时，很可能已经忘记了。因此，达达划词翻译还内置了生词本功能，基于艾宾浩斯记忆曲线进行温习提醒，帮助你科学掌握外文词汇。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/9B1D1B40-83A4-45DC-8480-A99D7A99AB40)

此外，达达划词翻译还支持在多个设备间同步单词本，或者同步至扇贝单词和有道词典。你还可以在设置中选择自动翻译或更换字体等，功能丰富。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/CDDC3546-1AE6-48C1-B47F-81B7493C246D)

你可以在 [Chrome Web Store](https://chrome.google.com/webstore/detail/%E8%BE%BE%E8%BE%BE%E5%88%92%E8%AF%8D%E7%BF%BB%E8%AF%91/cajhcjfcodjoalmhjekljnfkgjlkeajl "Chrome Web Store") 或 [GitHub](https://github.com/waynecz/dadda-translate-crx "GitHub") 免费获取达达划词翻译扩展。

百度截图翻译插件
--------

达达划词翻译已经足以满足大部分时候的需求，不过有时，我们会遇到生词被镶嵌在图片中的情况。这时，[百度截图翻译插件](http://fanyi.baidu.com/static/webpage/ocrPluginPage.html "百度截图翻译插件") 就可以帮到你。其调用了百度 OCR 和翻译的相关 API，只需点击图标后拖拽选区，就可以完成从文字识别到成段翻译的全过程。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/85404274-E6B3-4C42-9B48-6A3FB37D7902)

翻译完成后，你可以跳转至百度翻译官网，深入了解文段中包含的重点词组；也可以重新选区，查询更多释义。比较遗憾的是，百度截图翻译插件目前仅支持中英互译，更多语言类型仍待开发完善。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/81E55905-AA82-428B-AEB8-F1292B811F9E)

你可以在 [Chrome Web Store](https://chrome.google.com/webstore/detail/baidu-screenshot-translat/obhlofkljphbenhcjnimfcdlknkdilcm?hl=zh-CN "Chrome Web Store") 或 [官网](http://fanyi.baidu.com/static/webpage/ocrPluginPage.html "官网") 获取百度截图翻译插件。

书签侧边栏
-----

为了追求更大的有效显示面积，不少人会选择隐藏 Chrome 书签栏，但这也会带来一个新问题，那就是如何方便快捷的点击书签。而 [书签侧边栏](https://chrome.google.com/webstore/detail/bookmark-sidebar/jdbnofccmhefkmjbkkdkfiicjkgofkdh "书签侧边栏") 则可以将 Chrome 的书签栏移动至浏览器两侧，并通过滑动或点击等方式唤出。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/B4760D65-8844-4971-AF70-230DD76610C6)

在书签侧边栏的设置中，你还可以自定义其外观、位置、行为等，自由度较高。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/0146501E-56AD-48CD-B637-F295BA4688B9)

你可以在 [Chrome Web Store](https://chrome.google.com/webstore/detail/bookmark-sidebar/jdbnofccmhefkmjbkkdkfiicjkgofkdh "Chrome Web Store") 免费获取书签侧边栏。

Dribble Color Generator
-----------------------

Dribble 是世界著名的设计师交流网站，可以从上面发掘出许多优秀作品。而 [Dribble Color Generator](https://chrome.google.com/webstore/detail/dribbble-color-generator/cineoljoghlkpmnpcjcmbbpedgjbmpac "Dribble Color Generator") 则可以帮助你下载任意作品的调色板，并保存为 .aco 或 .sketchpalette 文件，让你从此不再为配色担忧。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/E33E233A-FBE9-463B-86BE-D2F178C0B6E5)

你可以在 [Chrome Web Store](https://chrome.google.com/webstore/detail/dribbble-color-generator/cineoljoghlkpmnpcjcmbbpedgjbmpac "Chrome Web Store") 或 [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/dribbble-color-generator/ "Firefox Add-ons") 免费获取 Dribble Color Generator。

Site Palette
------------

同样是一款配色辅助扩展，[Site Palette](https://chrome.google.com/webstore/detail/site-palette/pekhihjiehdafocefoimckjpbkegknoh "Site Palette") 可以一键生成任意网站的调色板，并按照 Material Design 等设计语言给出近似颜色方案，方便后续设计工作。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/907868EB-0F32-4836-B414-A3E2CD2093F3)

你可以在 [Chrome Web Store](https://chrome.google.com/webstore/detail/site-palette/pekhihjiehdafocefoimckjpbkegknoh "Chrome Web Store") 免费获取 Site Palette，其 [专业版](http://palette.site/pro/ "专业版") 支持通过终端调用 API，但需按请求次数付费。

AHA Music
---------

用浏览器观看视频时，你的注意力可能会被背景音乐所吸引。这时，除了在评论区发一句「求 BGM」并被复读一千遍「爱的供养再问自杀」外，你还可以试试 [AHA Music](https://chrome.google.com/webstore/detail/aha-music-music-identifie/dpacanjfikmhoddligfbehkpomnbgblf "AHA Music") 这款扩展。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/1BFA475D-88B6-4FF2-8241-89FBF5DB21A6)

如上图所示，只需点击 AHA Music 的图标，它就会自动识别当前正在播放的背景音乐，并给出 Spotify、YouTube、Deezer 等流媒体服务链接。由于采用了 [ACRCloud](https://www.acrcloud.com/ "ACRCloud") 作为曲库提供方，AHA Music 基本覆盖了所有时下流行歌曲，让你无后顾之忧。

你可以在 [Chrome Web Store](https://chrome.google.com/webstore/detail/aha-music-music-identifie/dpacanjfikmhoddligfbehkpomnbgblf "Chrome Web Store") 获取 AHA Music，或访问其 [官网](https://www.aha-music.com/ "官网") 了解更多信息。

Dark Reader
-----------

近年来，黑暗模式越来越成为人们的宠儿，就连苹果都在 macOS Mojave 中对其添加了原生支持。而 [Dark Reader](https://chrome.google.com/webstore/detail/dark-reader/eimadpbcbfnmbkopoojfekhnkhdbieeh "Dark Reader")，顾名思义，就可以为那些不支持黑暗模式的网站自动生成颜色翻转版本。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/25D9D17C-DFFF-431E-801A-0486B619F18B)

除了智能翻转外，在 Dark Reader 的设置中，你还可以手动调整颜色、对比度、滤镜等。如果你是高级玩家，还可以在开发者工具中直接编写 CSS，可玩性极高。

![](.evernote-content/36BEBA86-9435-4AE6-A453-A263C7B8994F/E1A6F7E4-2CD6-4D15-8C06-5F0365F629EE)

你可以在 [Chrome Web Store](https://chrome.google.com/webstore/detail/dark-reader/eimadpbcbfnmbkopoojfekhnkhdbieeh "Chrome Web Store") 和 [Firefox Add-Ons](https://addons.mozilla.org/en-US/firefox/addon/darkreader/ "Firefox Add-Ons") 免费获取 Dark Reader，扩展同时在 [GitHub](https://github.com/darkreader/darkreader "GitHub") 开源。
