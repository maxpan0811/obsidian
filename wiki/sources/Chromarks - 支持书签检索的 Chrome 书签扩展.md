# Chromarks - 支持书签检索的 Chrome 书签扩展

---

Chromarks - 支持书签检索的 Chrome 书签扩展
-------------------------------

原创 *2016-01-23* *青小蛙*

Chromarks 是一款 Chrome 扩展，用来管理和检索书签，让你更容易的找到需要的链接。@Appinn

![](.evernote-content/C9A2031F-07E2-4F6C-BF2B-8B996306AA34/78052F0A-290A-461E-B9BA-1A9EB40DC7C4.jpg)

感谢 @手哥爱吃鱼 同学的推荐：

Chrome 的书签管理扩展最有名的是 Neater Bookmarks，今天在浏览 Chrome 应用商店的时候，无意中发现一款新的Chrome 书签管理扩展 **Chromarks**，基本上 Neater Bookmarks 有的功能她都有，它比 Neater Bookmarks 强大的地方在于 可以实现书签和文件夹的检索，

![](.evernote-content/C9A2031F-07E2-4F6C-BF2B-8B996306AA34/49942BE4-47AE-425D-856E-90346CCAFCBD.jpg)

Neater Bookmarks 也有此功能，但只能检索英文书签，当然你在 Chrome 搜索框中也可以检索到想要的书签，但是不能检索文件夹；过去我的解决办法是将 chrome://bookmarks/#q= 保存为书签，书签名称为 书签搜索 这样当在搜索框中输入 \*\*书签搜索\*\* 和 检索词就可以检索所有的 书签和文件夹。

![](.evernote-content/C9A2031F-07E2-4F6C-BF2B-8B996306AA34/C581FDFF-9122-4139-9F8E-5156B4FC8AD1.jpg)

当然现在有了 Chromarks 要方便很多。Chromarks 的缺点在于不是很美观
