印象笔记是新一代办公应用，是全球 1 亿人的工作空间，所有手机、电脑和平板都能用。关注我们，记忆更好，效率更高，工作就会更牛，升职加薪不是事儿。客观地说：爱用印象笔记的人，90% 都是聪明人，剩下 10% 是更聪明的人。

![](.evernote-content/C344C34D-B806-434D-A4CB-4BE4D0E69EA1/5166FB1D-670C-434A-83B8-D174520A5FFC.jpg)

我是中科院上海分院的一名生物信息学的研究生。Coursera 上最先接触的课程，便是 Andrew Ng 的 Machine Learning。作为一个一心想要学习数理知识和计算机技能的我，当初无法想象，面对一张不断闪烁的屏幕，**如何能够沉下心来，在线上课程中有所收获呢？**

但是当我遇到印象笔记后，一切就不一样了。他将我的疑惑变成喜爱，他将我的喜爱变成了动力。

### **网页神器：剪藏**

说到印象笔记与 Coursera 的联系，我可能最乐意提到的、也最想向大家分享的，就是印象笔记在 Chrome上的插件——「**剪藏**」，我对它真是爱不释手。市场上也有许多其他的在线笔记软件，很少有像「剪藏」一般，能够完全保留下网页的格式。这也许对一般用户不重要，但是对于我来说，一个对网页上数学公式排布有着强烈执着的人，这简直就是一件「神器」。

![](.evernote-content/C344C34D-B806-434D-A4CB-4BE4D0E69EA1/04ECE54B-9472-4772-9D59-0B676A285066.jpg)

学过那门 Machine Learning 的同学应该知道，课程讲义的网页上布满了数学式子，有时甚至有成串的代码。**「剪藏」这个插件，不仅能保留网页的格式，甚至能完整保留下网页内的数学公式的排布、还有代码的语法高亮。**加上印象笔记独有的简洁却不简单的界面，我便爱上了在印象笔记中复习、查看课程网页剪辑的内容。

这样，即使在没有机会登录 Coursera 的情况下，我也能配合印象笔记的手机客户端，时时复习课程的内容，不会因保存网页的凌乱格式，而影响到阅读。不过，这也多亏了 Machine Learning 这门课的特殊之处——课程视频、讲义、测试、作业一直在网上保留，随时可以注册学习，不需要像有些课程一样等待开课，注册晚了还会遗漏前几节课。这门课成为了我在线学习的里程碑。

***★****「剪藏」支持所有主流浏览器**，无论你用什么看网页，都可以一键剪辑并永存好内容~*

**进阶技巧：**如果你安装了「剪藏」，你一定会发现，当你百度一下时，网页中会嵌入一个框，推荐和你搜索内容相似的笔记，这无疑是一种帮助加强记忆、巩固知识的方法，用户在学习过程中方方面面的需求，印象笔记都考虑到了。

![](.evernote-content/C344C34D-B806-434D-A4CB-4BE4D0E69EA1/5D503C73-8A19-4929-ACC6-D1C7EAA943A3.jpg)

### **Markdown 神器：马克飞象**

作为一个有一点 Geek 的人，我希望能够在网上学习的过程中，同时在线记录课程笔记。由于我选的大多为数理、信息方面的课程，在线笔记是否能完整保留公式、图片、一些简要的代码格式，便是我的「**刚需**」。印象笔记的扩展应用「**马克飞象**」就成为了我做笔记、复习、保存代码的最佳工具。

![](.evernote-content/C344C34D-B806-434D-A4CB-4BE4D0E69EA1/395B7CED-C267-4437-AA89-4AB6B5976CD4.jpg)

「马克飞象」拥有简洁而又友好的界面，是一个绝佳的 Markdown 在线记录和同步的工具。我可以很方便地利用 Markdown 语言在印象笔记中建立这样的课程笔记，就犹如记录博客一般，但是却只要短短几行代码。

「马克飞象」可以输入数学公式，甚至可以画流程图、支持多种代码的语法高亮，它成为了我私人的代码版本管理器，记录着我学习课程的点点滴滴。

![](.evernote-content/C344C34D-B806-434D-A4CB-4BE4D0E69EA1/27365544-7855-4DE9-ADC4-D2D58EB83BFF.jpg)

当学完一门课程，回头头看看自己刚开课时写的代码，既为以前代码的幼稚而好笑、又为自己的成长而欣慰。

### **收藏小神器**

再说几个印象笔记很实用的小功能吧。很多手机应用都可以直接绑定印象笔记帐户，比如微信、微博、知乎日报，一旦看到和课程相关的内容，我会立刻收藏到印象笔记中。

![](.evernote-content/C344C34D-B806-434D-A4CB-4BE4D0E69EA1/0D5599B8-19EB-4B6D-B904-95CB8B9801BC.jpg)

不得不说，印象笔记对于保留网页格式真的做到了极致，即使是微信、微博上的内容，仍然可以保留网页的格式，保持原汁原味。**对于一个 Geek 来说，混乱的格式也会带来信息的损失。**

### **学习成果**

在印象笔记的帮助下，我已经完成了 Coursera 上 Stanford University 的 Machine learning ，John Hopkins 的 Data Analysis 等 4 门「**硬课**」。这些课程，无一不需要学习者对于数理知识的掌握和对于操作的反复实践。能够成功攻克这些课程，印象笔记真是给了我莫大的帮助。

感谢Coursera，感谢印象笔记的团队和众多插件的开发者。如果你是 Geek，如果你想尝试在线学习，来试试我文章中推荐的工具吧。
