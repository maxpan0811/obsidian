# 报告分享：2011年中国人休闲消费达28568亿 微博已成主流平台

| |  |  |  |  |  |  |  | | --- | --- | --- | --- | --- | --- | --- | | |  | | --- | | 报告分享：2011年中国人休闲消费达28568亿 微博已成主流平台 |  |  |  | | --- | --- | | 来源：旅业链接 | 2012年07月02日 | |  | |  |  | | --- | | 日前，由国家旅游局综合协调司委托中国社会科学院财经战略研究院编制的《2012年中国休闲发展报告》显示，  Ø 2011年我国居民休闲消费规模大致在28568亿元，相当于社会消费品零售总额的15.53%， GDP的6.05%。  Ø 休闲领域成为投资热点，原本分属于不同领域的港中旅、恒大集团、海航集团、万达集团、保利文化、华谊兄弟、盛大网络等休闲供应商正越来越走向业务综合化，围绕休闲消费构筑全产业链。  Ø 微博上的核心消费是休闲消费，微博已成为我国居民的主流休闲消费平台之一。  据新浪数据，55.4%的用户在用微博记录心情，53.4%进行娱乐休闲，29.3%关注感兴趣的名人，26.5%消磨时间，35.2%与老朋友保持沟通，这都属于典型的休闲消费范畴。此外，36.8%了解最新发生的事情，46.8%结交新朋友，属于与休闲相关的延伸性休闲消费。这说明，微博上的核心消费是**休闲消费**。  　　目前，微博已成为休闲产业的主流信息发布媒介，以旅游休闲为例，截止到2012年2月29日，共有1348个旅游机构在新浪开设微博，包括640家境内旅游局、84家境外旅游局、24个旅游协会、600个景区。在开设微博的旅游局中，不仅省市一级的旅游局重视微博的作用，县一级的旅游局同样高度认可微博的平台效应。新浪微博目前共有24家省一级的旅游局，230家市一级的旅游局，375家县一级的旅游局和84个境外旅游局和办事处开设了微博，各旅游局微博的总发博数为275880条，总被转发数4081873次，总被评论数1819185次。 微博平台上休闲需求呈现鲜明特征，一方面微博休闲已经嵌入到工作、休息的每一个环节中，微博休闲已经非常地碎片化，另一方面微博休闲还具有裂变性、随身性、互动性、社交性的特征，在为我国公民提供新形式的休闲方式的同时，还具有潜力巨大的商业价值。  按新浪的统计，男性新浪用户平均日访问微博时间为25.31分钟，女性为36.58分钟，高于普通SNS网站男性24.21分钟，女性19.83分钟的水平。高频度的交流带来高强度的互动和传播。微博通过促进信息流动的方式促进了休闲需求，微博通过快节奏、碎片化的信息交流方式，有力地促进了休闲信息的流动。 | | |
