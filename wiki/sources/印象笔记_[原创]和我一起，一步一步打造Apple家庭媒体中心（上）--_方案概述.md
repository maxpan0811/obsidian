Apple TV 2G的越狱 3. XBMC的安装与配置 4. Mac Mini的升级 5. iPhone与iPad的设置   觉得图片被打水印真是很影响图片的整体效果，所以把图片都上传到国外的网盘上，如果FY们看不到，请多刷新几次，一定可以的。   一、总体介绍：   1. 方案设计：     这是我的家庭媒体中心的线路图，大家可以参考：      这套方案的核心就是把各类视频音频等媒体存储在NAS里，然后通过Apple的Airplay技术，将媒体流到各种支持Airplay的设备里去播放。 客厅里，Time Capsule加上组Raid 0 的磁盘阵列，组成了家庭网络的NAS。Time Capsule打开5Ghz的802.11n后，无线已经基本上取代了家里的有线。信号也非常的不错。但是家里楼上的几个房间，还是需要Airport Express进行补充。关于详细的Time Capsule的设置，在下面的设置篇里有介绍。     2. 客厅：        Apple TV 2G越狱后，配上XBMC，真是完美的家庭媒体中心。通过Apple TV 2后面的光纤口，配上XBMC支持的解码，输出DTS, AC3都没有问题。联上B&W的Panorama，可以看几乎所有格式的720p的高清电影，听音乐，效果真是超赞。(1080p的电影，其实XBMC也支持软解，但是由于apple tv 2，只支持720p的输出，所以目前只能看720p,不过，XBMC的开发人员据说正在努力，让我们给他们加加油！现在油价又在涨，我们给他们加93号的吧。)      饭厅的边柜上是B&W新出的Zeppelin Air，是B&W在老的Zeppelin的基础上，加上了Airplay功能，并且对音箱也做了升级。这款音响，B&W号称是世界上最棒的iPod, iPhone音响，听过以后，才知道B&W绝没有自夸。音色真是棒！特别是支持了Airplay后，再也不需要把iPod或者iPhone插在它的底座上，你只需要利用iPhone或者iPad上的Remote程序，就可以播放你的iTunes资料库里的音乐。吃饭的时候，或者是家里打扫卫生的时候，听听背景音乐，真是享受。强烈强烈推荐！        3. 书房      书房里，Mac Mini加上27’ Cinema Display，是在家里的PC中心。13’ Air是我平时移动办公的。IBM是我的windows下载机。很多美国人喜欢直接把Mac Mini放在客厅，接电视当HTPC。咱们没有那么奢侈，咱们缴着全世界最高的税，住着全世界最贵的房，用着全世界最贵的电，开着全世界最贵的车，烧着全世界最贵的油，买了不知道是不是全世界最贵的Mac。。。咱们要时时节约。 之所以没有用iMac 27，而是用Mac Mini加27’ Cinema Display的原因，一是因为看到太多关于iMac 27进灰的报道贴，发现是不是进灰，全靠人品，我又有点怀疑自己的人品，所以。。。二是因为，看了27’ Cinema Display的效果，套一句歌词“就这样被你征服。。。”。三是自己平时不玩游戏不做3D，Mac Mini升级到8G内存与Seagate Momentus XT 混合硬盘后，表现已经完美。   4. 卧室      卧室里也是一台Apple TV，我另外加上了一套Bose Companion 5。这样在卧室里，看电影，听音乐，效果也是非常不错。通过Airplay，也可以将iTunes资料库里的音乐，流到Bose C5上播放。     5 小卧室       小卧室里由于和TC隔了几堵q1an9，由于无线信号相当的不行。于是就补了一个Airport Express，而且把原来的一个不支持Airplay的Altec Lansing的音响，接了上去。这样在小卧室里，也可以享受到Airplay的乐趣。    6. 设置 Time Capsule   我的NAS主要是由Time Capsule来组建。应该说，Time Capsule本身的功能，在同类产品里并不算强，不过，打着苹果的logo，而且放家里也够用，所以我也认了。      Time Capsule本身内置的硬盘，用来做两部Mac的Time Machine备份，以及我其它重要数据的备份，比如照片与工作文档。利用它的USB口，接一个Belkin的一转四的USB Hub，然后再接上外置的磁盘阵列。目前我接了两个磁盘盒，每个盒内是两个2T的西数的绿盘，5400转的。不过在利用了磁盘盒提供的Raid 0后，读写速度已经非常令我满意了。这样8T的存贮，已经可以放接近2000部的720p的高清片子。反正现在Apple TV 2G也不支持1080p的输出。（唉，其实买Time Capsule，不如买Airport Extreme再外加几个硬盘呢。。。当时手贱啊！）  7. 关于Airplay   我的这套Apple家庭媒体中心，核心技术就是Apple的Airplay，简单说，就是通过无线或有线的方式，将iTunes资料库中的音视频媒体，流到各个支持Airplay的播放器上进行播放。我就是通过Airplay，将视频流到各个Apple TV上看视频，将音频流到Zeppelin Air。因为我还有一台老的Altec Lansing的音箱，它不支持Airplay，所以我就通过Airport Express的音频功能，将音频流过去播放。 点击iTunes右下角的Airplay的图标，就会弹出一个选择菜单：     我们就可以选择将音频视频流到哪个设备去播放了。 并且，你在家里的任何地方，都可以用你的iPhone或者iPad来控制Airplay。这是多么的方便啊！   更有趣的是，我们可以选择多个设备，这些设备，可以互不干扰的播放不同的媒体，也可以播放同一个媒体。我试了将同一首歌曲，流到不同房间的设备上进行播放，整个家里，充满了美妙的音乐。。。。   这样美好的生活，让我们首先感谢伟大的党，感谢伟大的国家！    然后让我们感谢这些伟大的人：     Steve Jobs Steve Wozniak Tim Cook Jonathan Ive Phil Schiller Burrell Smith Jef Raskin 。。。 喜宝       由于文章很长，担心FY们看的心烦，所以就先发上半部分，关于具体的各项设置，将在下半部分中详细讲述。   花了一个星期时间，终于把下半部分写完了，大家可以参考：    |  | | --- | |  | |
