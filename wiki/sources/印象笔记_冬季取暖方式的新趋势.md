| |  | | --- | | **冬季取暖方式的新趋势**  中国目前很多家庭冬季都采用空调取暖，这是一个节能和健康的大误区。早在十几年前，西方发达国家的冬季取暖都首选健康、快速、舒适的燃油、燃气速暖机，特别是在非常注重节能和安全的日本，70%的家庭都使用燃油速暖机。专家呼吁，中国在倡导建设节约型社会的大背景下，应该尽快改变冬季取暖方式，使用更节能、高效、健康的燃油、燃气型速暖机。    据介绍，燃油速暖机和空调相比在以下几个方面具有明显优势：  一、能效比  空调取暖能效比低、耗电量大、升温速度慢。空调受其制热原理的限制，取暖时有诸多不足：当电能转化为热能时，其中损失较大，热效率低，制热能效比（制热量与总输入功率的比值）通常小于2.5，当室外温度低于5摄氏度时，热交换能力就大幅下降，制热量就会随之下降，当室外温度低于0摄氏度时，其效能比很快降为1，只相当于一台电热暖风机，空调的制热能力只能达到它平时的1/3，根本无法为我们提供充足的热量。如果室外温度再低的话，空调启动都会非常困难。  据能源专家介绍，从能源利用的角度来看：
