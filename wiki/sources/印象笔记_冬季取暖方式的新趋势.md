# 冬季取暖方式的新趋势

---

|  |  |
| --- | --- |
| |  | | --- | | **冬季取暖方式的新趋势**  中国目前很多家庭冬季都采用空调取暖，这是一个节能和健康的大误区。早在十几年前，西方发达国家的冬季取暖都首选健康、快速、舒适的燃油、燃气速暖机，特别是在非常注重节能和安全的日本，70%的家庭都使用燃油速暖机。专家呼吁，中国在倡导建设节约型社会的大背景下，应该尽快改变冬季取暖方式，使用更节能、高效、健康的燃油、燃气型速暖机。    据介绍，燃油速暖机和空调相比在以下几个方面具有明显优势：  一、能效比  空调取暖能效比低、耗电量大、升温速度慢。空调受其制热原理的限制，取暖时有诸多不足：当电能转化为热能时，其中损失较大，热效率低，制热能效比（制热量与总输入功率的比值）通常小于2.5，当室外温度低于5摄氏度时，热交换能力就大幅下降，制热量就会随之下降，当室外温度低于0摄氏度时，其效能比很快降为1，只相当于一台电热暖风机，空调的制热能力只能达到它平时的1/3，根本无法为我们提供充足的热量。如果室外温度再低的话，空调启动都会非常困难。  据能源专家介绍，从能源利用的角度来看： 能源到发电的使用效率为37%，通过电流输送再到空调制热，能源的使用效率约为35%；而能源通过精制、输送到燃油速暖机制热，使用效率约为90%。 中国的教科书都已经将宣传了几十年的地大物博修改掉了，目的就是希望国人具备能源危机意识，从而建设节约型社会。使用空调制冷效果很好，但空调用电制热就明显与节能相悖了。  二、空气和水分的平衡  空调取暖通风不畅，头晕胸闷的“空调病”接踵而来。而燃油速暖机在燃烧的过程中会释放出少许水蒸气，防止空气干燥，同时对空气还有一定的消毒作用。此外燃油速暖机本身带有缺氧保护装置，可以促使在取暖时每隔一两个小时开窗通一次风，保持空气的平衡。  事实上，我国空调业目前出现“重空气冷暖，轻空气品质”状况，致使我国的空调业长期游离在“健康”之外。普通空调有75%的空气被反复利用，如果增加新风量就意味着要提高能源的消耗。而我国的能源储备不足，这使得我国在发展空调业方面忽略了“新风”。  三、制暖速度  空调在冬季开机时，为防止冷空气在室内循环，在空调充分加热之前，空调运转后5分钟内风扇不会转动，5分钟后风扇开始工作，但由于空调是采用热泵从外界空气吸取热量，并释放到室内，所以室外空气温度大大影响制热效率，加热整个房间需要花费很长的时间，因此在下班回家以及寒冷的清晨，打开空调后要等很长的时间才会使房间有温暖的感觉。此外，当室外机交换器上有霜形成，自动除霜功能将除霜，在除霜时内外风扇又将停止运转，空调又不制热了。  燃油速暖机一般在开机两分钟左右就会排出近80摄氏度的热风，不受室外温度的影响不间断地输出热量，很快就能高效均匀地烘暖整个房间。从日本原装进口的“BLUE HEATER”燃油速暖机甚至能够在开机后的45秒内就送出舒适的热风，是在所有的燃油速暖机中启动速度最快的。因此该品牌燃油速暖机在日本的350万台全年销量的市场份额中，年销售170万台，在日本的市场占有率历年来始终是排第一位的。  四、取暖效果  空调的整体设计更偏重于制冷，因此，它的出风口一般都在房间的上方，夏天冷风从上方吹出来，冷空气会自然下沉，使整个房间能够很快的凉爽。而在冬季，暖风从上方吹出后，由于热空气是往上的，使得整个房间很难均匀的烘暖。很多人都有这样的切身体会，冬季在客厅里坐在沙发上看电视，往往会觉得脚特别的冷，这是因为用空调取暖的房间，天花板和地板会有明显的温差。  燃油速暖机一般直接放在地板上，热风直接吹向地板，然后随着热空气的自然上升，均匀烘暖整个房间。这是主要用于制冷功能的空调无法兼顾到的。  五、空调受能源和环境的限制  目前，国家电力资源缺乏，很多省级电网拉闸限电，冬季空调取暖的普遍使用给国家带来了很大的电力负担。对家庭来讲，由于各种家用电器的普及，冬季很多家庭在使用空调时都要顾及电表的负荷，有部分家庭因使用空调还会经常碰到保险丝熔断等烦恼。  燃油速暖机是将精制专用煤油经气化100%燃烧后将油转化成热量后释放出来。所耗用的电量十分小。  欧美及日本等发达国家冬季取暖方式选择燃气、燃油速暖机和他们重户外运动，享受“新风”有关，他们注重节能、健康的因素值得我们重视。 | |
