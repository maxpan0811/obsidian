# 技嘉发布P57系列17.3英寸游戏笔记本新品

[![](.evernote-content/B049B0B2-C447-47BB-B734-7EFAD2D14405/8E897893-88D8-4336-A8FD-91F6B61361BE.jpg)](http://tech.feng.com/2016-02-23/Gigabyte-release-P57-series-17.3--inch-notebook-product-game_639220.shtml)

![](.evernote-content/B049B0B2-C447-47BB-B734-7EFAD2D14405/8E897893-88D8-4336-A8FD-91F6B61361BE.jpg)

  
　说到游戏笔记本，很多人的第一印象是角度奇特的外形和彪悍的硬件配置。不过技嘉（Gigabyte）最新推出的P57系列（P57W和P57K），却没有按照既定的路线去走。其采用了富有美感的黑色外壳，同时加上了橙色线条作为点缀。两款机型均采用了英特尔第6代酷睿i7-6700HQ处理器、17.3英寸@1920x1080 IPS显示屏、8GB/16GB DDR4 2133内存（双槽/最高32GB）、全尺寸30键无冲背光键盘。  
  
　技嘉P57系列游戏本采用了Intel HM170移动芯片组，i7-6700HQ处理器中集成了Intel HD Graphics 530核芯显卡，可选蓝光或Super Multi DVD刻录光驱，提供了2W\*2的扬声器和高清摄像头。  
  
　存储方面，其支持“3存储系统”，包括1个2.5英寸HDD、1个M.2 SSD（2280规格/支持NVMe PCIe X4和SATA）、以及1个可置换驱动器位。  
  
　P57K和P57W的最大不同，主要体现在独立显卡上。其中P57K配备了拥有2GB GDDR5显存的Nvidia GeForce GTX 965M，而P57K则是拥有3GB GDDR5显存的GTX 970M。  
  
　网络方面，P57系列游戏本支持10/100/1000Mbps以太网、802.11ac Wi-Fi（兼容a/b/g/n）、以及低功耗蓝牙4.1。  
  
　接口方面，P57提供了3个USB 3.0、1个USB 3.1、HDMI 2.0 + D-Sub + mini-DP视频输出、RJ-45以太网口、音频输入/输出、SD读卡器、以及直流电源输入。  
  
　此外，该还机拥有Kensington锁孔，配备了60.8Wh（P57K）和75.81Wh（P57W）的锂电池，充电器分别为150W和180W。  
  
　两款机型的大小均为421x290x24.9 mm，重约2.9千克。目前8GB RAM + 128GB SSD的P57W的售价为$1499，16GB RAM + 256GB SSD的价格则是$1699。
