在学与用之间给孩子留出空白去思考|百造年卡课程发布
=========================

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/E556E7BF-7812-495E-887D-A1280B2E6ACD)

我们不是通过经历来学习，而是通过经历进行反思来学习，学生必须在经历中有所体悟。

—— 教育家杜威

百造学堂近期在北京上海发布了 Lab 拓展课程年卡，课程包含八大主题的 32 个课题，通过预约的方式选择孩子感兴趣的内容。2020 年初可预约的课程陆陆续续出炉啦，为咱家娃锁定席位吧！

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/B6FF0C20-75F6-44E2-97F3-EF81E7AEF2B1)

**本年度开放的主题有**

**技能类**

木工精工、艺术设计

**工具类**

基础编程、建模与 3D 打印

**内容类**

趣味数学、神奇科学、人文博物、

建筑与城市

**2020 年 1 月 lab 课程发布**

**神奇科学在哪里？**

**奇妙数理化 + 脑洞科技 + 趣玩实验**

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/0BAA6B27-834C-48F2-8DA5-F9FD7AD2C06C)

**当伯努利原理遇到飞机**

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/211E4BCB-22D6-4E56-B11B-7922EBDCD68D)

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/5EA800AE-9F3E-472F-AC0A-B92AB4F2C3CF)

很久以前，在没有飞机的时代，那些年代的孩子对天空充满向往。因为，他们无法飞上天空俯览大地，而我们今天穿梭在云层里的飞机，成了他们一生的遗憾。关于飞机的物理学，飞机是如何起飞的？由你来揭秘。

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/F7B13B2F-EC4F-4626-BB51-0AC71C07D37B)

飞机滚转是副翼控制的

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/D3331505-F65A-4028-89CF-07E4BCB06259)

飞机偏航靠方向舵控制

伯努利原理是流体机械能守恒的基本表现形式，是水动力学的基础，也是流速与压强的特殊关系。本次课程通过动手实验，重现生活中伯努利原理的应用。

**通过该课程可以：**

1. 重新认识生活中的诸多现象，学会用所学的知识去解释现象和发散思维。

2. 学习流体的相关基础知识，认识气态流体和液态流体。

4. 利用木条、纸张等常见物品进行趣味实验，提高动手能力及实验规律总结，锻炼逻辑思考的意识。

5. 通过对比实验和即兴比赛，激发学习兴趣的同时，掌握科学的学习方法。

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/0BAA6B27-834C-48F2-8DA5-F9FD7AD2C06C)

**液体的表面张力**

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/211E4BCB-22D6-4E56-B11B-7922EBDCD68D)

把硬币浮在水上，看着彩色液滴在玻璃片上跳舞，这些其实都是表面张力带来的现象～表面张力有很多好玩的小实验，本次课程以生活中的细节出发，了解液体的内聚性和吸附性，体现分子引力的外在表现形式。

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/5661B2A4-5A2F-4F34-9957-A3D12478ABB0)

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/DA80EB0E-EC7C-4AB6-ACE7-F8526138FDFA)

**通过该课程可以：**

1. 认识并验证表面张力的存在，了解表面张力的基本作用形式。

2. 学会实验记录的基本原则以及自我设计对比实验过程和实验结果，树立科学思维。

3. 认识表面活性剂及其作用，联系生活实际，掌握表面张力在实际生活中的影响。

4. 用所学的科学方法，制作出美丽的水拓画，体验水韵画卷中的科学原理，实现科学与艺术的统一，兴趣与知识的共存。

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/0BAA6B27-834C-48F2-8DA5-F9FD7AD2C06C)

**热胀冷缩**

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/211E4BCB-22D6-4E56-B11B-7922EBDCD68D)

自然界中的大部分物体存在受热时会膨胀，遇冷时会收缩的特性。热胀冷缩是微观粒子运动的外在表现形式，也是理解分子动理论的基础。

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/59ADF772-8CC3-4568-95EC-60BDE986A865)

**通过该课程可以：**

1. 了解温度的由来，以及温度计的发展历史

2. 学习主要的温度计类型以及材料特性。

3. 认识热胀冷缩的不同表现形式，并能够运用其解释生活中的诸多现象。

4. 通过动手实验，了解气体、固体和液体各自的热胀冷缩现象。

5. 通过对比实验，了解不同物质的热胀冷缩程度。

6. 通过学习制作和实验，初步了解线性关系，并用其简单测定温度。

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/0BAA6B27-834C-48F2-8DA5-F9FD7AD2C06C)

**光的基本性质**

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/211E4BCB-22D6-4E56-B11B-7922EBDCD68D)

彩虹是怎么形成的？我们又是如何看到这色彩斑斓的世界？左右眼看到的世界一样吗？在百造工坊中，了解属于我们眼睛的秘密，认识光的基本性质。

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/5DCAEF60-8CA4-44B8-B67C-D361B91C28C7)

**通过该课程可以：**

1. 认识光的色散现象及光的三原色。

2. 了解光源及其分类，探究人眼的视力秘密。

3. 通过影子实验和路径实验，认识光的直线传播，并呈现光在空气、水和固体中的传播路径。

4. 通过不同介质下的对比，体验类比方法和发散思维。

5. 了解人眼的构造和近视、远视的原因。

6. 认识光反射的基本现象，利用反射知识制作 3D 效果的全息金字塔，提高动手能力。

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/7A82E2A9-EB39-443C-BC5D-8545D1F2C42F)

**在介绍百造 Lab 之前，**

**不得不提及大洋彼岸都一所神奇的高中。**

17 年前在美国加州圣地亚哥出现了一所特殊的学校 High Tech High，在这所学校里，没有教科书、没有上下课铃声、没有考试，孩子们每天都忙于一些自主选择的特定项目，为了一年一度的大型展览废寝忘食。随后的 17 年间，High Tech High 成功复制了 13 所学校，并且将教育延伸到了初中和小学。

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/CEE41AE0-3655-4098-8E3F-63F601BCC36B)

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/0900B32D-EAF0-40A7-A70E-A67E7DF092A1)

HTH 的办学特色很多，最让人钦佩的就是它敢于打破传统的按学科分类的教学模式，完全采用 PBL-Project Based Learning 基于项目的学习方法，让学生们通过完成一个个真实的项目获得他们这个年纪该掌握的且实用的知识和技能。

**而作为一所美国公立学校，其所获得的成绩也令人惊讶。**

**·**100% 达到了加州大学 A-G 高中课程标准，加州平均水平是 36%

**·** 95% 的学生参加了 SAT 考试，加州平均水平 45%

**什么是 PBL-Project Based Learning 学习？**

**它又有怎样的魅力呢？**

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/E62A0C65-948B-421F-928A-8BB91C841BDA)

项目制学习在西方是一种从幼儿园到大学被广泛应用的教学方法。在我看来这是两半球教学方法的较大区别之一。在国内和不少亚洲国家，孩子从小到大的学习都是别人设题目，学生解题，最后有一个标准答案决定对错。而项目制学习是先设定一个目标，这个目标可以由别人定，也可以由自己定，可以是清楚的，也可以只是一个大概方向，学习的目的是达到目标，而途径可以有很多钟。目标的大小、难易程度、根据孩子的能力而定，但它会覆盖几个关键点：**设定目标，制定计划，获取信息，团队协作，测量数据，分析结果，演示结果。**

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/995FE722-E27B-4F60-B200-F135B343ABFA)

**运用 PBL 的方式学习有两大好处：**

**第一点是**，它能训练孩子解决问题的能力。面对问题，分析问题，运用学到的知识，以往的经验，找到合适的方法去解决问题，最后再验证解决方案是否正确，以及有没有更好的方案。

**第二点是**，它能极大地激发孩子的好奇心和求知欲。关于这一点，TED 上有一个特别有名的演讲，叫做 “Bring on the learning revolution”（学习的革命），它就特别提到好奇心对于孩子学习的巨大帮助。有好奇心的孩子会特别有求知欲，他们会将自己学到的知识和生活结合起来，想到去解释或者解决生活中遇到的问题，这种孩子往往将来的学业或者事业都会很成功！

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/20F99FFA-6344-439A-8D89-AD2923774DAB)

**在美国衡量一个学校、老师的好坏，有一个很重要的指标就是看她们的 Project 教得怎么样，越是好的学校、好的老师，她们组织的 Project 就越多、也越有趣。**

**百造工作坊的基础课程**

运用 PBL-Project Based Learning 的方式每月一个主题，带领孩子们深入体验项目制学习法，节节相扣，层层深入，从个人观点到集体创作探究深挖孩子们解决问题的能力，这个过程和将来工作里完成项目的过程比较类似，孩子在实践中培养能力。在学习的过程中主动思考，甚至是被期待去犯错，因此他们变成了主动的学习者。当然，要实现这一切，需要有完善的计划、足够的灵活度和一个可以展开团队协作的环境。

![](.evernote-content/D9A1269E-A3A8-4871-B97D-00D8FE759264/7C9D3D48-83F7-45A9-BBC2-7E87F3EB4318)

然而所有的创造力都必须扎根于基础知识，才有可能产生。否则，让孩子随意地发挥想象力，只能是空中楼阁、纸上谈兵。
