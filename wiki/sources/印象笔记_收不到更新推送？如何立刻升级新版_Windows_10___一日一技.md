收不到更新推送？如何立刻升级新版 Windows 10 | 一日一技
==================================

#### [Umi](https://sspai.com/user/713147)

1小时前

4  [1](#)

本月 18 号，微软正式面向全球推送了 [Windows 10 秋季创意者更新](https://sspai.com/post/41340 "Windows 10 秋季创意者更新")，这也是自其发布以来的第四次重大版本更新，包含诸多新特性及功能改进。但由于全球 Windows 用户众多，微软通常会分批次推送系统更新，时间可能长达两三个月之久。

如果你已经等不及尝鲜新功能，却始终无法在设置中检查到系统更新，下面这些方法可能会帮到你。

使用 Windows 10 易升
----------------

Windows 10 易升是微软为了帮助低版本用户升级至最新版系统而开发的傻瓜式一键安装工具。它可以自动检测当前系统版本信息及电脑配置情况，并选择最适合你的系统更新进行下载。

你只需要启动程序，点击「立即更新」，就可以接着做其它事情了，它会帮你完成接下来的步骤。

![](.evernote-content/1D41F0DB-FE9B-4FCD-80E6-8221075ED6E4/9F20F673-35E4-47C1-A416-7FA5D794192D.png)

你可以在这里下载 [Windows 10 易升](https://www.microsoft.com/zh-cn/software-download/windows10 "Windows 10 易升")，直接点击最上面的「立即更新」按钮就可以了。

使用 Windows Media Creation Tool
------------------------------

有时，Windows 10 易升仍会存在一定的延迟，这时，你可以选择 Media Creation Tool。它是微软推出的官方介质创建工具，可以帮助你全新安装 Windows 系统、为其它电脑创建安装介质等。

你可以在刚才 Windows 10 易升网页靠下面的位置找到「立即下载工具」按钮，之后，按提示选择「立即升级这台电脑」，它就会开始下载最新的系统镜像。

![](.evernote-content/1D41F0DB-FE9B-4FCD-80E6-8221075ED6E4/08086B1B-0A9F-4250-89CA-102E17CA904C.png)

下载完成后，你可以选择保留之前的个人文件和应用，或者直接执行全新安装，以获得更纯净的系统体验。

通过镜像文件安装
--------

如果你想选择要升级的系统版本，或者要为其它电脑创建安装介质，就需要手动下载系统镜像文件了。

我依然推荐使用 Media Creation Tool 进行下载，只需要选择「为另一台电脑创建安装介质」，你就能够决定想要下载的系统信息，并选择直接下载 ISO 文件或创建 U 盘启动盘。

![](.evernote-content/1D41F0DB-FE9B-4FCD-80E6-8221075ED6E4/A5347E9B-66C1-4314-BDA6-C8BB0B156DB8.png)

成为 Windows Insider
------------------

如果你想比大多数人都先体验到新功能，Windows 预览体验计划就是为你准备的。你可以在「设置-更新和安全」中找到该选项，点击「开始」，然后按提示操作，就能够收到最新的预览测试版本，同时为 Windows 改善做出贡献。

![](.evernote-content/1D41F0DB-FE9B-4FCD-80E6-8221075ED6E4/F0FD6819-4F58-4B7F-BB82-380F4E40D6DC.png)

但同时，预览版本往往比正式版更不稳定，有较大可能会出现各种 bug 等，甚至可能导致你的文件丢失或系统损坏等严重错误。如果这是你的主力工作电脑，我强烈不建议你更新到测试版本。

还不确定要不要升级至秋季创意者更新？快来阅读少数派为你准备的 [Windows 10 秋季创意者更新具透](https://sspai.com/post/41340 "Windows 10 秋季创意者更新具透")，深入了解新系统的各种新特性。
