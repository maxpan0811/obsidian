![](.evernote-content/E1884F21-0644-4697-9762-3AC96225720B/154D05A6-29FD-461C-A850-B2C8241E3D06.webp)

前段时间 Aflred 给大家推送了邮件介绍即将到来的 V3 版，同时还放出了几张新版Workflow 的截图“引诱”大家。

虽然还没有具体的细节透露，但从下图中我们也还是能发现新版的 Workflow 将变得更加的灵活、强大。这让大家对 Alfred V3 充满了无限的期待，看来 Alfred 在 Mac 电脑装机必备 list 中地位将更加无法撼动。

![](.evernote-content/E1884F21-0644-4697-9762-3AC96225720B/A850C0EE-F947-438B-9365-3D148677757C.webp)

离新版发布应该还有一些日子，我们还无法得知新版中 Workflow 更多的细节。不过我还是照例还是要为介绍一些近期发现不错的 workflow（基于 V2）。

**1. Network**

与之前介绍过的 Wi-Fi workflow 不同，通过 Network 我们可以展示现有可用的 Wi-Fi 列表并选择快速切换，无需在手动点击顶部菜单栏的 Wi-Fi 图标进行切换。

![](.evernote-content/E1884F21-0644-4697-9762-3AC96225720B/69F939A6-20EB-40DF-AC28-51324BFC518B.webp)

**2. Restart App**

这款 Workflow 可以帮助我们快速重启某个应用。比如输入「restart sketch」，它将帮我们自动重启 Sketch。

![](.evernote-content/E1884F21-0644-4697-9762-3AC96225720B/116D3064-C323-4DBD-A7F6-FBA3C6AAB5B6.webp)

**3. Copy URL**

Copy URL 可以获取当前浏览器 Tab 中的 url 地址并将其转换为 HTMl、Markdown 语言的格式。特别是 Markdown，对于长期书写 Markdown 的同学们来说可以说是一大利器。

![](.evernote-content/E1884F21-0644-4697-9762-3AC96225720B/2D46BDAC-7F22-457A-81C7-A0DCC425694F.webp)

**4. Password Generator**

自从去年的几次网站密码泄露之后，我已经对自己的密码体系重新进行了梳理。对于“占坑”的产品或应用全部随机生成密码，然后再保存到 1Password 中。这款 Workflow 正好帮我简化了操作，不必先去 1PW 中生成密码了。

![](.evernote-content/E1884F21-0644-4697-9762-3AC96225720B/451FD3D8-CBE6-4259-A7FE-84A381A05EFF.webp)
