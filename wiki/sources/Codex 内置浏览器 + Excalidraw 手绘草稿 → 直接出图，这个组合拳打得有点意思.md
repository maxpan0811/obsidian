Codex 内置浏览器 + Excalidraw 手绘草稿 → 直接出图，这个组合拳打得有点意思。

具体操作不复杂：在 Codex CLI 里打开内置浏览器，进入 Excalidraw 画布，随手画个草图、写几行标注，Codex 就能基于你的手绘直接生成图片。说白了，把 Excalidraw 当成 Codex 图像生成的 Prompt 画板——画布上涂什么，它就照着出什么，所见即所得。

更香的是，Codex 的生图几乎不限制次数。对比市面上那些按张收费的 AI 绘图工具，这等于白捡了一个免费无限画板，想试多少种风格就试多少种。再搭上 Codex 本身的编程能力，批量生图、统一风格、自动化后期处理一套全搞定。比如同一张草图，让 Codex 一次性出十几个配色方案，或者批量套用不同的画风滤镜，以前要开好几个工具来回倒腾的事，现在一个终端里全链路跑通。

创意门槛直接打下来了。以后轻度出图的场景——做 PPT 配图、社交媒体的封面、甚至产品原型的概念图——可能真不用再开别的工具了。
