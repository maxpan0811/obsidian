# 华程集团月度会会议纪要（4月 Review）

> 会议时间：2026-05-19，13:30-18:32。主持：何勇。

## 总结（何勇讲话）

> 四五月份经营利润虽达标，但盈利多受**汇率利好**带动，并非依托机损下降、人档提升实现，盈利稳定性不足，整体营收规模未达预期。若三季度无改善，运营效率将低于预算目标。

**三点部署**：
1. **严控费用 + 加大前端销售投入**：同步清退业绩不佳、渠道份额下滑的人员
2. **后端严控编制**：原则上只减不增，系统搬迁完成后进一步调整
3. **欧洲业务**：客单价逆势走高，加快三段特惠项目落地，Q3提前启动Q4招标对接

## 各部门关键信息

**电商/市场（钱俊超）**：三月贵州/湖南膨胀金券核销成效突出；四月天津场售券后核销为零，表现不佳。

**人力/效率（于小禾 + 郭东杰）**：资深老销售坐拥优质资源但产出偏弱，携程等优质渠道合作不足，区域业务占比下滑。郭东杰：参考携程加权模式，结合交易额+毛利多维评定效率指标。

**数据口径（刘圣/郭东杰）**：各业务单元数据模板与统计口径须与财务/市场保持一致，避免数据结论偏差。潘渤牵头联合各单元总经理研讨优化销售管理模式，依托CRM系统做好跟踪。

**欧洲环线游（唐一波）**：对接携程给纵横开发的 GYG 及 TripAdvisor API 接口使用事宜；与 Mapa 洽谈合作（互推产品，开拓中西客源市场）。

**欧洲事业部（田园）**：在奇士霍夫基础上增加老佛爷、宝齐莱等门店数据，统一算法做整体数据分析。

**美洲中亚（匡世胜）**：Q4毛利差额100万（业务端与财务端数据不符）；国旅产品在携程渠道下单意愿更高，非携程渠道收客下滑；南美市场贴牌问题待推进。

**导游/领队（肖鸣/任霞）**：探讨导游沟通由邮箱改企微；需配合惠品团项目完成竞对人头政策收集；重测导游管理与收入情况；评估满意度数据用于非携程渠道NPS评估。

## 关键信号

- **汇率是当前利润的核心变量**：利润达标但质量存疑，去掉汇率因素则经营效率不及预算
- **欧洲业务是唯一增长亮点**：客单价逆势走高，三段特惠项目是Q3重点
- **非携程渠道收客持续下滑**：美洲中亚已明确，预计其他业务线类似
- **数据口径问题积累**：毛利差额百万级，需系统性治理
