相比 Windows，Mac 在很早之前就从系统层级提供了颇为完善的截图/截屏工具，这使得我们可以非常轻松的通过快捷键来对整个桌面、应用窗口、自定义区域进行截图操作。虽然功能上还算完备，但在实际使用中还是显得不够方便，例如默认截图后会保存到桌面，而复制到剪切板则需要通过另外一组键盘快捷键，操作起来也是繁杂。而通过第三方工具则可以对其进行「增强」，比如这款在 Setapp 上最为热门的截屏类工具 App：CleanShot X。

![](/images/file-generic.png)

相比系统原生的截屏功能，CleanShot X 在此基础上又进行了「升级」：比如支持滚动截图功能，可以通过辅助工具来实现精细化的截图操作，而截图后的素材内容会先「暂存」到这个「快速访问」小窗口中，并以此实现更进一步的处理操作，甚至你还可以通过 CleanShot 实现录屏操作——显然 CleanShot 早已跳脱出功能单一的 Mac 端第三方截图工具，而成为一款极为实用的 Mac 端截屏/录屏扩展工具。

那么对于这样一款集截图处理，录屏为一身的 Mac 上的工具软件，到底有哪些实用的功能呢？是否可以真的充当起系统原生截屏工具的完全替代呢？

**适用于各种场景的截图**
--------------

熟悉 Mac 端原生截屏工具的朋友一定非常熟悉原生截屏的几种形式——捕捉整个屏幕、捕捉某个 App 窗口进行截屏以及区域截屏，通过快捷键完成之后，你可以在屏幕的角落中看到缩略图，点击缩略图可以进行编辑或者存储到桌面的操作。

实际上作为 Mac 原生截屏的替代，CleanShot X 的主要场景也跟这个类似，同样也有全屏的捕捉、App 窗口的捕捉以及区域选择截屏，除此之外还有一个非常实用的滚动截屏，这个功能其实非常适合对浏览器的网页进行截屏。

![](/images/file-generic.png)

![](/images/file-generic.png)

十字准线和放大镜

而仔细对比过原生截屏和 CleanShot 之后，你会发现 CleanShot 更像是原生截图工具的细节增强，这主要体现在 CleanShot X 中任意截屏模式下都有着细节上的优化。比如在区域截屏工具上，往往我们需要对区域进行更精准的操作，在 CleanShot 的区域截图模式下，鼠标截屏光标上会出现十字准线，并且在移动时还可以出现放大镜，来放大对其需要截屏的像素边缘。同时对于处于动态的桌面，为了方便更准确的截屏，还可以采取「临时冻结」的形式来截取当前帧下面的静态画面。

![](/images/file-generic.png)

![](/images/file-generic.png)

隐去当前截屏中的桌面背景

而**窗口截屏模式**下，CleanShot X 在功能细节上也颇具考量，比如可以选择是否让截取的窗口边缘带有阴影（虽然这算是 Mac 上截应用窗口的特色之一），另外对于一些场景下需要隐去当前截屏中的桌面背景，在 CleanShot X 中也可以对其进行单独设置，这样截出的窗口画面就不会被桌面壁纸所干扰了。

至于**滚动截屏功能**也非常实用：选定好需要滚动截屏的范围，然后滚动滚轮就可以自动开始滚动截屏。即便你操作滚动的速度不均匀，也不会出现画面撕裂的情况。而且因为它并非浏览器的扩展，所以除了支持在浏览器中对网页滚动截屏，也支持通过滚动文档进行截屏来制作信息长图。

![](.evernote-content/8EA12FB5-171F-4B31-AA9C-6D3EFE8115F5/F7026B97-ADE0-4C43-BC26-53A6BD03DE9A.gif)

所以盘点完 CleanShot X「截图」功能，就不难发现它几乎就是系统原生截图工具的增强，所以我就**直接用它来替换掉系统原生截屏工具**。当然如果每次使用截屏，只能在系统状态栏调用就显得不够方便，所以在快捷键上，我直接将其映射到系统原生截屏工具的快捷键中，一方面可以不用再记住新设置的快捷键，另一方面 CleanShot X 也可以用更少的快捷键实现更多的操作。

**用「快速访问」处理截图**
---------------

Mac 原生的截图处理其实分为两个小步骤：先截屏，截取的画面会变成缩略图悬浮到 Mac 桌面的角落，再点击这个悬浮的缩略图选择是复制到剪切板中还是将其保存为文件。而 CleanShot X 则将这个「缩略图」扩展成了功能更为丰富的截图快速处理中心，被称作是「快速访问」。

![](/images/file-generic.png)

相比原生截图的「缩略图」，在 CleanShot X 的「快速访问」中则在这个「缩略图」上加入多个按钮来让你选择执行下一步操作，最为主要的当然是「复制到剪切板」以及「保存为文件」，而四个角也有特定的功能动作：打开注释编辑工具来处理截图，或者是将图片暂时 「钉」在桌面上 —— 这非常适合一次性处理多张截图，并逐步进行预处理。

![](/images/file-generic.png)

除此之外，右键点击这个「快速访问」面板还可以实现更多快捷操作，比如说无需保存为图片就可以实现截图的方向调整，将截图上传到 CleanShot X 云中或者是快速将截图进行分享、使用第三方 App 打开当前截图等。这些快捷操作**都无需首先将截图保存为图片文件在进行**，因此单就操作而言**更容易实现便捷的工作流**。

**将截图钉在桌面上**
------------

虽然说「快速访问」功能可以有效处理截图工作流，但是如果需要同时处理多张截图的话就需要用到 CleanShot X 另一项实用的功能——**浮动截图**。你可以将它理解成将截图的相关数据以「缩略图」的形式暂时「钉」在桌面上，这样你可以将大量的截图先暂时「放起来」，然后再进行进一步地处理。

![](/images/file-generic.png)

你可以对这些「浮动截图」调整尺寸比例、调整透明度，从而更好地在桌面上对其进行区分。而浮动截图的后续操作其实只有一种：打开注释编辑器对截图进行注释编辑。

**全面的注释编辑器**
------------

有时候简单的截图并不能精准地表达。所以我们往往都会在截图上加上一些注释来让截图信息更为丰富，CleanShot X 同样内置了一个**功能强大的注释编辑器**，比如对截图画面进行裁剪，在截图上创建各种箭头以及形状，使用笔迹工具来标注，为截图上打上马赛克或者对某个部分突出显示等等。

![](/images/file-generic.png)

注释完成之后，你可以将其保存为图片文件，通过分享菜单发送到其他应用中，也可以上传到 CleanShot X 云服务中或者干脆直接复制到你的剪切板中。

**不仅仅是截屏，还有录屏**
---------------

除了这些功能丰富的截屏操作，CleanShot X 还「赠送」了一个极其强大的录屏功能，要知道那些有类似功能的独立 App 售价可相当不菲。

和截屏一样，CleanShot X 的录屏同样可以选择捕获画面的区域：全屏、App 窗口或者是屏幕中自定义的某个区域，选择完成之后会弹出录屏的工具条，来对即将录制的内容进行相关设置，这里你可以选择录制麦克风的声音或者是内录系统声音，录制为 MP4 视频还是 GIF 动画，并且可以选择是否录制时显示鼠标的光标。

![](/images/file-generic.png)

而在工具条中还可能对录制中更多的细节项目进行调整，比如说录制的视频帧数，画面质量以及分辨率等等。而如果是录制教学视频，我们往往需要展示键盘按键以及光标位移，而在录屏的过程中都可以对其进行捕获和展示，比如光标部分可以更换颜色、尺寸、轮廓形状等，而按键部分可以对其记录下来并展示在画面中，在画面的位置以及尺寸可以调整，也可以设置显示按键的类型 —— 是全部的按键都进行记录，还是仅记录下快捷键输入。

![](/images/file-generic.png)

如果录制的是视频，在结束录制之后我们可以通过「快速访问」来打开简单的**视频编辑器**，这里可以调整画面的尺寸、画面的质量、声音（改成单声道或者静音），当然也可以对其进行简单的剪辑操作 —— 剪掉某些不需要的帧。调整完之后选择合成就可以出片了。

**给截图多一份保险**
------------

对于一些重要的截图以及视频，只是保存到本地其实并不算稳妥，更为方便的做法是上传到云端，而 CleanShot X 也为这部分有着更高要求的用户提供了**云服务**，只要有浏览器就可以访问管理这些上传的截图或录屏。

![](/images/file-generic.png)

上传云端后可以通过分享链接来发送给指定人员，通过团队账号来进行授权、使用标签来标记截图，甚至通过自己的域名来访问这些上传到云端的图片库。

**结语**
------

作为 Setapp 商店中最受欢迎的截屏工具，CleanShot X 与其说是系统截图的增强替代，不如可以称作是重**塑了在 Mac 上的截屏/录屏的工作流**，从而使得原本繁琐的操作变得简单且高效。

如果你希望可以优化 Mac 原本的截图操作，或者是希望可以找到一个操作方便的 Mac 端录屏工具，那么 CleanShot X 绝对值得一试。
