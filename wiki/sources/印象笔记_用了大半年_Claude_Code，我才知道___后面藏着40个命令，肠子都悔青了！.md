朋友们，今天要跟大家坦白一件挺丢人的事。我当初用 Claude Code 写代码，一直就是干巴巴地在对话框里打字，让它改这改那。改完了什么样，我就看它嘴上说的。它说改好了，我就信了。结果有一次，一个登录流程的重构，它顺手把我三个不相关的文件也动了。我压根不知道，直到上线前测试的同事跟我说，你这接口怎么返回值变了。

我当场就懵了。翻了半天对话记录才搞明白怎么回事。

后来才知道，Claude Code 里有一个 /diff 命令，一敲就能看到它到底改了哪些文件、改了哪些行。我这大半年，等于是闭着眼睛开车。

我对自己这种用法的评价是：纯纯的原始人。
