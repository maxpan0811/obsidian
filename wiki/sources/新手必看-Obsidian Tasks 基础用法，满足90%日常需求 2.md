新手必看：Obsidian Tasks 基础用法，满足90%日常需求
==================================

Original赫点茶 赫点茶

虽然有段时间没写 Obsidian 的文章了，但是近期依然有粉丝客户找我问些关于 Obsidian Tasks 插件用法问题，于是便有了今天的这篇文章

Obsidian Tasks [^1]插件是干嘛用的？
---------------------------

**Tasks 插件** 是 Obsidian 中**最强大、最流行的任务管理插件**。

我在早期的文章中也推荐过这款插件，你如果使用 Obsidian 的话 Tasks 插件则是装机必备的。

![](.evernote-content/920C55B0-6449-4849-8C04-E16966CAF3C7/5D8D8B29-5BBE-4264-BB07-00133A8D0BFD.png)

Tasks 功能图，图片来源：Obsidian  插件社区

如何安装 Tasks 插件
-------------

打开 Obsidian 的选项，点击`第三方插件`，在社区插件市场点击右侧的`浏览`，然后在搜索框里输入 `tasks` 搜索，安装并启用，如下图所示。

![](.evernote-content/920C55B0-6449-4849-8C04-E16966CAF3C7/4C3F6BE5-4FEE-4183-BA8B-60BAC604B781.png)

针对不同版本的 Obsidian 及操作系统，你安装启用这款插件之后，可以尝试重启一下你的 Obsidian。

Tasks 的使用方式
-----------

Tasks 这款第三方插件的设置项目前不需要我们设置。如果是你第一次用的话，目前不需要我们设置，你只需要熟悉一下它的语法。然后接着跟着我的步骤去实际使用这款插件感受一下。

读者朋友可以在自己的 Obsidian 中去 copy 复制粘贴下方的这个语法，看一下实际显示出来的效果，就很一目了然了。

`方式1：`

- [ ] 这是一个未完成的任务
- [x] 这是一个已完成的任务
- [ ] 交报告 ✅ 2026-04-03
- [ ] 读书 #todo/project

我们还可以通过命令行的形式，创建或编辑任务，Windows 用户输入 `Ctrl + P` 进入命令行。

![](.evernote-content/920C55B0-6449-4849-8C04-E16966CAF3C7/2ACC52A1-3CF1-4B88-BD13-49D7A7AFC1A7.png)

然后在弹出的窗口输入`tasks` 选择 `Create or ediit task` 创建一个简单的任务。

![](.evernote-content/920C55B0-6449-4849-8C04-E16966CAF3C7/93FB8E17-044A-4C77-91FD-8E33F51396A7.png)

在弹出的窗口中填写今天的`任务事项`，比如`任务描述`，`开始时间`、`截止时间`等。

![](.evernote-content/920C55B0-6449-4849-8C04-E16966CAF3C7/62D72050-CCE7-413F-BFE1-5D17729EC9A5.png)

看不懂各选项设置没关系，我将中英文对照给你们整理好了，直接上手用就行。

至此，我们完成通过`方式2` 使用 Tasks 创建一个任务。

给任务添加属性
-------

而且 Tasks 里还支持非常丰富的属性，通过符号标记，比如优先级、完成、开始，还有标签分类等。

![](.evernote-content/920C55B0-6449-4849-8C04-E16966CAF3C7/E3674738-DABB-46F6-B534-9AB3B47D5A51.png)

实际用法参考下方的语法，

- [ ] 写公众号 📅 2026-04-02 
- [ ] 录制视频 ⏫ 优先级高 
- [ ] 下班记得买菜 🛫 2026-04-01 
- [ ] 交报告 ✅ 2026-04-03 
- [ ] 读书 #todo/project

读者朋友可直接在自己的 Obsidian 中输入 `- [ ]` 符号体验一下 Tasks 的功能。第一次用可能会不习惯，甚至会因为`嫌用起来太麻烦`直接否定它。

千万别！怎么用，你看到这里就已经会了，尝试用几天体验一下，习惯之后你会喜欢上这款软件的。

简单的创建跟完成任务，都已经能够满足大部分的读者朋友了。当然它还有一些非常高阶的用法，比如说去通过语法进行查询，当然，这是有学习成本的，但是对于绝大部分的读者朋友，对他们来说可能就没有任何用，所以我这篇文章也不继续罗列官方的一些语法，感兴趣的可以自行研究[^2]。

Tasks 与 Check List 插件的配合使用
--------------------------

之前有粉丝客户找到我进行 Obsiidian 的相关指导，反馈说他没有图示中的这块功能，让我帮他弄一下。

![](.evernote-content/920C55B0-6449-4849-8C04-E16966CAF3C7/97A75048-0943-442D-8304-1DDAAAF644F2.png)

然后我发现他是没有装 `Checklist` 这款插件。

![](.evernote-content/920C55B0-6449-4849-8C04-E16966CAF3C7/C2CF12CA-4F2F-40E6-9C38-F0C1FD6B3AED.jpg)

在任务中添加

`#todo` 标签，然后展开右侧功能栏，滑动鼠标找到`Todo List` 就可以搜索并管理我们的任务了。

![](.evernote-content/920C55B0-6449-4849-8C04-E16966CAF3C7/00AE0508-6E12-4376-929D-BA6C23E1A9DB.png)

当然 Tasks 的功能还有很多，但是对于大多数普通用户来说，以上内容其实就已经能够满足 80% 甚至 90% 的使用场景。至于更多的用法，可以参考下方参考文献 `脚注2` 关于 Tasks 官方插件的功能介绍用法。

祝各位读者朋友 Obsidian 越用越六！
