# 短文：从1984年的《终结者》想到的—很多事情我们在有生之年可以亲眼见到其实现

原创深圳宁南山宁南山

从1984年的《终结者》想到的—很多事情我们在有生之年可以亲眼见到其实现

有时候我们觉得现在技术进步很慢，像以前十九世纪，二十世纪那种划时代的发明现在似乎很少了，比如说石油开采技术，火车，汽车，飞机，电力，钢铁，计算机，互联网等等。但是我最近思考了下，觉得未必是这样。

几十年前有个电影，确切的说首部在1984年上映，叫做《终结者》，施瓦辛格主演的，讲的是2029年以天网（SkyNet）为首的人工智能机器正试图消灭所有幸存的人类。天网将一名T-800终结者送回1984年的洛杉矶，并将其程序目标设定为杀死人类反抗军领袖约翰康纳的母亲莎拉·康纳。而人类抵抗军也将一位战士凯尔·瑞斯送回1984年去保护莎拉。当时看了之后觉得很震撼，觉得太科幻了，有生之年不可能看到这么高级的场景出现。

![](attachments/8d643ee8cbc0473d.png)

但是在2021年的今天看来，除了后续的终结者系列中出现的液态金属战斗型机器人，还有时空穿越等这些过于高端的技术之外，其他的技术未必就不可能在有生之年看到了，毕竟我还活个几十年应该没问题。

首先在《终结者》里面背后的反派“天网”是一台无处不在的超级计算机，它是一个软件程序，并不是存储在某一台物理计算机上，而是可以在网络间无处不在，这不就是现在兴起的云计算技术么？大规模的服务器集群使得软件程序并不依赖于某一台服务器，这台服务器坏了，故障了，被人类反抗军破坏了，它还可以在其他服务器上继续运行。

目前全球最大的云计算公司亚马逊旗下的云计算公司Amazon Web Services, Inc.（AWS）是2006年成立的，也就是在终结者上映的22年后，在过去的十几年，全球已经出现了亚马逊，谷歌，微软，阿里，华为，腾讯这样的巨头云计算公司，投入大量的资金推广和研发。

然后是人工智能技术，天网自己产生了意识，可以进行对话，数据分析和智能决策，控制各种武器对人类发动攻击。

人工智能在二十年前看起来还是很高级的技术，但是尤其是最近十年，由于消费电子产品这个庞大的市场的存在，给予全球人工智能公司提供了源源不断的研发资金，因为这是一个每年全球销售15亿部的庞大市场，人脸识别和语音识别都在手机上得到大量应用。

我们熟知的语音识别，苹果的Siri是2011年才推向市场的，到今天语音识别已经非常成熟了，你打开手机上的各种软件，对普通话语音的识别程度已经非常高，我现在使用百度地图，高德地图，都是直接说想去的地点，识别准确率非常高。

同样的还有人脸识别功能，现在刷脸支付已经普及到了路边的自动售货机，而刷脸开门已经普及到了小区门口，我现在住的小区就是人脸识别开门，已经是非常成熟的技术了。

而实际上以苹果手机为例，苹果的Face ID面部识别功能在2017年9月12日发布的iphone X上才首次发布，而当时的媒体有的还持有怀疑的态度，例如担心手机识别不了人，或者用一张照片就能够骗过手机解锁。

![](attachments/a865a1ccbfa7dcab.png)

在《终结者》系列中有一幕场景，就是大楼里面的无人机自动攻击武器追杀主角们，在2020年的纳卡战争中，阿塞拜疆军队凭借无人机对亚美尼亚军队造成了巨大的杀伤和破坏，有评论认为这是人类第一场无人机决定胜负的战争，这场战争中大量使用了土耳其研发制造的TB-2无人机，该款飞机2014年7月首飞，研发者Selçuk Bayraktar毕业于美国麻省理工学院，他的老婆是土耳其总统埃尔多安第四个女儿。

而实际上就以中国为例，航天科技集团旗下公司开发的彩虹-3A无人机2008年才公开亮相，而更为知名，水平更高的彩虹-4无人机则是2012年公开亮相。

这里再多说一句，无人机为啥发展快，因为它便宜且攻击力强，因此各国在局部冲突中均大量购买和使用，这等于就产生了很大的市场，市场源源不断的给研发人员提供工资，并且供给资金进行更新技术的研发，因此技术进步会很快。

无人机现在还需要有人操控进行攻击，那么未来能实现完全自动运行，发现和攻击目标吗？我们可以说一定能够实现，而且很可能就在未来一二十年，甚至都不用说到我们有生之年。

类似的还有汽车的自动驾驶功能，2014 年 10 月，特斯拉发布首款自动驾驶系统Autopilot，并率先搭载于 2015 年上市的 Model X 车型，今天我们已经可以看到中国用户在国产电动车上高速路时躺下，手离开方向盘作死的视频，以至于国产电动车品牌厂家发声明目前自动驾驶还无法做到无人工干预，这其实也才六七年的时间。

目前市面上的电动车自动驾驶已经普遍可以做到L2了，正在向L3进化，到2025年可能会出现L4级别的量产车型，到L4就已经是自动驾驶了，到后面法律层面的制约将会大于技术进步层面的制约，如果法律层面的问题也解决了，可以说我们有生之年看到L5完全自动驾驶普及一点问题都没有，二三十年后中国的马路将是另外一番场景。

由此可见，语音识别也好，面部识别也好，无人机也好，自动驾驶也好，这些人工智能技术的应用都在过去十年因为消费电子产品，汽车或者战争而得到了巨大发展。

另外就是人工智能的智能决策，现在的战争的确是人类决策的，但是未来能不能根据战场态势，使用超级计算机+AI调用战场的无人作战设备进行作战呢，这难道不是一种决策吗？举个例子，卫星发现了敌人的地面装甲部队，立即自动召唤最近的无人攻击机前往进行攻击，这个过程其实没有人类干预我认为也是可以做到的。

再比如在战斗机群进攻航母的过程中，超级计算机实时的接收从卫星，地面雷达和预警机侦察到的战场态势，实时的控制我方无人战斗机和导弹的运动轨迹，根据敌我在某区域的战机和导弹的数量和位置，实时的自动计算和决策是进行饱和攻击还是进行机动攻击，先攻击哪个目标对己方损失最小，还是说在该区域先行撤退，哪种方式最优。这些《终结者》中天网具备的智能决策功能，目前人类也在不断发展中。

《终结者》第一部中的T-800型机器人，能够自己开车，开枪射击，根据预先输入的指令跟踪人类并且进行攻击，但是还不能奔跑，同时看起来动力系统续航能力很强，无需充电，我觉得这样等级的机器人，我们有生之年未必就不会看到。看下波士顿动力发布的机器人视频就知道了，他们做了很多款不同的机器人，巅峰之作是Atlas仿人形机器人，十年前他们的机器人甚至站立都困难，下图是Atlas的前身petman，这是2009年的视频，还需要用线缆牵引着站立。

![](attachments/283c854f04736a78.png)

这是2013年初发布的Atlas原型机视频，初步可以单腿站立了，还可以侧面接受摆球撞击。

![](attachments/7a940d3d18654541.png)

2017年发布的Atlas 2已经可以做后空翻了，下图是我截取自2017年11月发布的视频。

![](attachments/b22037a85a965b77.png)

而根据波士顿动力2021年8月发布的视频，现在Atlas已经可以实现做一些跑酷的动作了，甚至能够单手支撑越过横杆，而根据波士顿动力工程师透露，这一代的Atlas并非像以前一样通过事先编程来完成整个路线，而是靠它自身的传感器对环境进行感知，从而自主决策。

![](attachments/0d7844f3268ab055.png)

或许对我们个人来讲，十年的时间很长，因此我们并没有感受到那种一夜之间天翻地覆的变化，而是在不知不觉中习惯了新技术对日常生活的渗入，从而因此感觉变化并不太大，但是从历史的维度来看，十年也只是弹指一挥间，我认为对于我们的日常生活来说，人工智能技术的应用将是未来几十年的最重要变化，包括在科幻电影中经常出现的在城市中运行的飞行汽车，现在都已经有公司在开发了。

仅仅是十年左右的时间，人类就在云计算和人工智能方面取得这样的进步，如果技术持续进步下去，三十年之后呢，四十年之后呢，五十年之后呢？我们只要想办法活到2050年，2060年，2070年，那么就能够亲眼见证到很多科幻电影里面的场景在一点点的，不知不觉中实现。
