# 他才是这个时代真正的国民男神，堪称国士！ - 周小平 - 草根网

---

|  |
| --- |
| 他才是这个时代真正的国民男神，堪称国士！  2017-06-05  字号： 小中大     1776年，美国制造出世界第一艘可以用于实战的“海龟号”潜艇。1962年新中国在依靠苏联全套W潜艇资料下才仿制出第一艘国产03型潜艇。技术差距太大了。        1954年，美国制造出世界第一艘“鹦鹉螺号”核潜艇，该艇很快投入使用，并且打破了世界各种潜艇纪录。1970年中国第一艘核潜艇“长征一号”（091汉级）才刚刚开始十多年漫长的边试验边摸索的过程。技术差距更大了。        1981年，美国第四代战略核潜艇“俄亥俄级”下水，该艇具有超越时代的隐蔽力和恐怖的核武器携带量，一艘就可以轻松毁灭一个中等规模的国家。同年，中国第一代战略核潜艇（092汉级）也刚刚开始艰难试航，但由于技术指标落后和武器配套进度滞后，在很长一段时间里中国的海基核威慑力几乎都只是摆设。甚至直到2004年，中国第二代战略核潜艇（094晋级）的性能指标几乎仍然全方位落后于“俄亥俄级”，更别说美国最新一代的战略核潜艇了。技术差距更更更大了。        差距是如此巨大，以至外国专家曾如此嘲讽过我们：“中国核潜艇从广州出港，在越南西贡就可听到。” 中国潜艇无法解决噪音巨大的问题，这是一条几乎看不到希望的道路，这条道路黑暗得足以令很多人灰心丧气……        而然，在2017年5月30日，一条央视“焦点访谈”曝光的消息则如一道刺破黑暗的曙光。由一位叫马伟明的院士带队研制的“潜艇无轴泵喷推进器”已经获得实质性进展，或许已运用于中国最新一代核潜艇建造。这将是中国潜艇史上足以载入时代的进步，它使得中国人第一次可能拥有世界先进水平的潜艇技术。        潜艇推进技术是当今核潜艇最核心的技术之一，它既决定了潜艇的机动能力，也决定了潜艇的隐蔽性。只有机动性和隐蔽性两个指标均达到要求，核潜艇才能真正成为大洋中沉默的杀手，否则就是反潜武器集火的目标，难以发挥出作用。        从出现潜艇以来，潜艇推进技术先后经历螺旋桨式和机械式泵喷推进器两个阶段，其中，冷战时期的苏联凭借着螺旋桨技术上的优势曾建立起一支令西方战栗的水下红色海军，而上个世纪80年西方凭借着新一代机械式泵喷推进器技术则在这两个性能指标上超越苏联潜艇。        但不管这两种技术的哪一种，都是依赖潜艇体内的驱动轴提供动力，并没有根本解决传统“螺旋桨-轴系-艇体”耦合振动有道的潜艇低谱噪声，这使得现如今所有潜艇降噪技术都成了“治标不治本”的办法。        而无轴泵喷推进器则是解决这一问题的最好出路，其将传统螺旋桨和机械式泵喷推进器的轴系驱动转化为布置在泵喷导流管内的电机驱动，从而取消了传统动力系统的驱动轴，既能从根本上消除轴系所带来的各种噪声，同时也能极大地提高潜艇前进速度。        一旦中国新一代核潜艇运用这一技术，将实现技术上巨大的跨越，使得中国核潜艇缩短与美国最新一代核潜艇技术水平的距离。无论是日本苍龙美国海狼，还是越南印度的基洛，都将在中国装备潜艇无轴泵喷推进器的新一代潜艇面前黯然失色。届时，人们一定会记起那个敢于带队在黑暗去追赶的人——马伟明。        1960年出生的马伟明无疑是新中国后心路历程最沧桑的一代人，他们曾贫穷过无知过自以为是过，但到改革开放打开国门后，却突然发现自己的国家已被世界远远的甩开。差距似乎巨大到无法追赶。许多人当时就陷入了绝望，他们觉得中国肯定快崩溃了，于是就自卑地跪倒在地；而另外一些人在经历最初诧异后，则默默地陷入了如何赶超的思考，他们愿意用毕生的心血去改变现状。马伟明无疑属于后一种人。        马伟明研究电机领域，这是个看起并不起眼的领域，但却对军事装备至关重要。在他之前，国内这一行业技术水平惨不忍睹，但马伟明却愿意默默为之奉献了30余年，无论寒暑，不分昼夜。终于，他和团队制造出世界上首台交直流双绕组发电机系统、攻克了第三代集成化发电系统、无轴泵喷推进器、甚至未来“电磁炮”“全电战舰”也初现倪端……        他几乎以一己之力将中国的相关领域追赶上来，其意义不亚于两弹一星。何为国士？他即为国士。我们不否认一个国家的发展是全社会共同推动的结果，但也不能否认这个民族最杰出精英的巨大作用。中华民族何其幸运，诞生了钱学森、邓稼先……马伟明这样的无双国士，因此中国没有如外国人一再预言的那样崩溃，而是从落魄中一步步挺了过来，并在今天实现了赶超和飞跃。        而关于马伟明笔者还曾听过坊间这么一个传闻：在一个国防重大项目科研立项时，曾有过两种技术路线的争议。马伟明坚持的方向是美国最新在研技术路线，而另一个研究所则选择了保守技术路线。有关部门把数亿经费全部拨给保守技术路线，使得马伟明团队只好自筹资金研制。结果，有经费拨款的保守技术进展迟迟，马伟明直接瞄准世界前沿反而提前试验成功。        马伟明所研发的这一技术叫“电磁弹射技术”，是新一代航空母舰标准匹配。用不了几年，我们就能够看到中国舰载机随着电磁弹射而迎风起飞，相信这一场面会使得无数中国人热泪盈眶。 |
