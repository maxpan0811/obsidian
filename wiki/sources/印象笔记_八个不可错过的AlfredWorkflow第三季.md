我推荐的不仅是apps，更是一种态度!

![](.evernote-content/65EE5FC8-4907-45A4-96EB-B00C8D14226A/F55E17FE-531F-424F-9167-F28CB26969A4.jpg)

之前的两期 Workflow 推荐大家似乎都很喜欢，看样值得继续做一下。这次又凑满八个，第三季又来了。

**1. 计时器**

输入「 timers」激活。 选择 New Timer，输入如 「1 关洗衣机」，一分钟后会系统通知你关洗衣机。

![](.evernote-content/65EE5FC8-4907-45A4-96EB-B00C8D14226A/D434FF60-88DD-4C1A-9A56-0B9470E78222.jpg)

**2. 隐藏文件**

输入 「Show」 或 「Hide」 显示或隐藏系统中的隐藏文件。

![](.evernote-content/65EE5FC8-4907-45A4-96EB-B00C8D14226A/490C329B-7A5F-4DE7-BBA5-DE6CE5DD06AA.jpg)

**3. 生成二维码**

输入「 qr」加 URL 地址，帮你生产二维码图片并在浏览器中打开。

![](.evernote-content/65EE5FC8-4907-45A4-96EB-B00C8D14226A/5272CD3F-EE27-4988-8B9B-2703D3B63E0D.jpg)

**4. IP 地址查询**

输入「 ip 138」加 IP 地址，显示IP 归属地。

![](.evernote-content/65EE5FC8-4907-45A4-96EB-B00C8D14226A/0DD2F3A3-23B5-4CE5-ABA2-B42F75A3A4CC.jpg)

**5. 网易新闻**

输入「wy headline」 显示最新头条新闻，回车后在浏览器中打开。

![](.evernote-content/65EE5FC8-4907-45A4-96EB-B00C8D14226A/0F5E5C9B-004C-4197-95F3-C10DEA938155.jpg)

**6. 电池信息查询**

输入 「mybattery」 显示电池使用情况。

![](.evernote-content/65EE5FC8-4907-45A4-96EB-B00C8D14226A/78DB6C54-4BD6-4F1E-B7DF-5743D91F08C1.jpg)

**7. 发送URL 地址**

输入 「saa」 将当前浏览器 tab 的 URL 发送到相应应用中打开。我最常用的是将 Chrome 浏览器当前页面发送到 Safari 中打开。

![](.evernote-content/65EE5FC8-4907-45A4-96EB-B00C8D14226A/DC6C43FA-C1BF-4CB8-BD42-C09DC0AD164E.jpg)

**8. 美股实时价格查询**

送给炒美股的同学们，回车后在 Yahoo finance中打开。

![](.evernote-content/65EE5FC8-4907-45A4-96EB-B00C8D14226A/E8DC6F75-79EE-4645-BF28-2D0EAF685254.jpg)
