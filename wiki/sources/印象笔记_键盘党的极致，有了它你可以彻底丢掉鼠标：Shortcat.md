# 键盘党的极致，有了它你可以彻底丢掉鼠标：Shortcat

---

![](.evernote-content/736101CE-E1B7-4150-8A16-B22316FDE8B7/A353001B-D19D-499F-8205-5B3690426982.jpg)

使用电脑的时候，我们经常重复着下面的动作：手放到鼠标上、拖动鼠标、点击、手放回键盘上。据粗略估计，完成这一过程平均需要三秒钟时间，更关键是，重复得多了，手累。

对于键盘党来说，打开新窗口、打开新标签页，这些操作直接有快捷键可以搞定。可是点击网页上的链接、打开一个文件夹，这些好像就只能用鼠标了？

当然不是。Mac 上这款图标是「喵星人叼着鼠标」、名字叫 **Shortcat** 的软件，能完美解决你的需求，它的目标是——**没有鼠标，所有操作通通用键盘搞定**。

Shortcat 能做些什么？
---------------

先看下面这张图，在任意应用程序中按下（默认）快捷键「Command + Shift + Space」即可呼出 Shortcat。

Shortcat 会为应用中所有可点击的内容分配一个快捷键（一到两个英文字母），比如下图「通用」的快捷键是 R，「桌面与屏幕保护程序」的快捷键就是 S，你只需输入对应快捷键，就可以直接选中对应内容。

![](.evernote-content/736101CE-E1B7-4150-8A16-B22316FDE8B7/F44D720E-34AE-48E4-8ADE-6BDCD7DEBD3D.jpg)

Shortcat 怎么操作？
--------------

### 基本操作

简单地说，Shortcat 的基本操作分为以下几个步骤：

1. 按下快捷键「Command + Shift + Space」后，当前应用（或窗口）底部会出现一块蓝色输入框；
2. 在输入框中输入英文符号「.」，可显示所有可点击项；
3. 按 Control + 对应字母，即可选中对应选项；
4. 按下回车键，即为鼠标左键「点击」该选项。

### 模拟鼠标双击

看到第四点，你会发现 Shortcat 可以用「Enter 回车键」模拟鼠标的点按操作，按「回车键」对应的是「鼠标左键单击」，那么双击呢？很简单，**快速按两次回车键**就是双击。

如果你只想把光标移动到某个项目上，而不作点击操作，连按两下 Control 键即可。

### 点击操作附加修饰键

修饰键，是指键盘上 Command、Option、Control、Shift 这类按键。

电脑上有一些操作需要使用到「按住修饰键再点击」，以实现一些特殊的功能。一个很常见的例子是：在 Safari 中，直接点击链接是在当前标签页打开，而按住 Command 再点击链接，则会在新标签页中打开。

Shortcat 同样支持修饰键的使用，按下修饰键，再按回车即可。比如 Safari 中想用 Shortcat 在新标签页打开某个链接，按住 Command 再回车就行了。

速度还不够快？试试直接搜索
-------------

用键盘操作就是为了「快」，但上面提到的 Shortcat 基础用法里，需要先找对应字母再输入的方式并不自然，实际操作也不太流畅。

没关系，Shortcat 也有更自然流畅的方法：**直接搜索你想点击的项目**。

先看下图这个例子。我想在网页里点击试听 TFBOYS 的单曲《不完美小孩》，操作步骤是：

1. 呼出 Shortcat 输入关键词「buwanmei」
2. 按下 Control+D，将光标移动选中到期望位置
3. 回车

![](.evernote-content/736101CE-E1B7-4150-8A16-B22316FDE8B7/2D62E21B-F725-4A77-9D13-1CB6FB4601F8.jpg)

从这个例子可以发现，**Shortcat 支持中文汉语拼音搜索，并支持模糊匹配。**我输入「buwanmei」「wanmei」「不完美」「完美」等关键词，都可以匹配成功。

但如果在这里我想点击专辑 《不完美小孩》，而不是单曲《不完美小孩》，怎么办呢？

回看上面操作的第二步，将 Control+D 改为按 Control+E 即可。

### 只显示应用程序按钮

在 Shortcat 搜索框中，输入 「.button」还可以只显示应用中的可点击按钮。

比如，当我想点击 Finder 顶栏中的共享按钮时，输入「.」显示的是所有可选项，结果是下图这样的：

![](.evernote-content/736101CE-E1B7-4150-8A16-B22316FDE8B7/A019F5CD-1EDF-4977-97BC-5DDE09E00719.jpg)

够乱吧？我们试试输入「.button」，界面则会是下图这样。一下子整洁干净很多，能方便我们很快定位想要的选项。

![](.evernote-content/736101CE-E1B7-4150-8A16-B22316FDE8B7/77D893C9-F784-4B5D-B5EB-F3C8BEC4C314.jpg)

Shortcat 支持的应用
--------------

Shortcat 支持所有 OS X 系统自带的应用程序（比如 Safari、Finder、Xcode），也支持绝大部分第三方应用程序。下面，再演示几个例子：

1. 用键盘完成在**迅雷**中添加新下载任务、打开已完成项目等操作：

![](.evernote-content/736101CE-E1B7-4150-8A16-B22316FDE8B7/B93B527E-83E9-4DB3-A8B7-71A4D00DA446.gif)

2. 在 **Safari** 中浏览网页、切换标签页：

![](.evernote-content/736101CE-E1B7-4150-8A16-B22316FDE8B7/E87BFB46-6374-4391-810D-3A4517FA0110.gif)

3. 在 **QQ音乐** 中切换歌单、播放音乐：

![](.evernote-content/736101CE-E1B7-4150-8A16-B22316FDE8B7/7B902EDC-BEDB-4805-B5A5-EB2F278DAFA2.gif)
