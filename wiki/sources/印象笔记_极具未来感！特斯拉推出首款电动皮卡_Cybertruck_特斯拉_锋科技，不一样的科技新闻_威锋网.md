# 极具未来感！特斯拉推出首款电动皮卡 Cybertruck 特斯拉 锋科技，不一样的科技新闻_威锋网

极具未来感！特斯拉推出首款电动皮卡 Cybertruck 特斯拉 锋科技，不一样的科技新闻\_威锋网
==================================================

当赛博朋克（cyberpunk）和卡车（truck）相遇，会有什么情况发生？  
  
特斯拉给出的答案是旗下首款电动皮卡 Cybertruck。

[![](.evernote-content/DEB58D4B-D83C-4F21-B567-84B60DF12943/FAE05F71-A71E-4D40-9F35-BC3A3D12F96B.jpg)](https://resource.feng.com/resource/h072/h09/img201911221559360.jpg)

  
  
在 11 月 22 日中午，特斯拉首席执行官埃隆・马斯克在美国洛杉矶正式推出了首款电动皮卡 Cybertruck，最大的亮点是外形采用了赛博朋克风格，硬朗的直线棱角设计有别于其他汽车的圆润造型，再加上贯穿式车灯的设计，整体极具未来感。  
  
[![](.evernote-content/DEB58D4B-D83C-4F21-B567-84B60DF12943/B1C9AE46-C43C-47E8-AD55-BA71341F132E.jpg)](https://resource.feng.com/resource/h072/h09/img201911221559190.jpg)  
  
  
[![](.evernote-content/DEB58D4B-D83C-4F21-B567-84B60DF12943/EE14E1BC-510E-46E5-89ED-59D6FBC91660.jpg)](https://resource.feng.com/resource/h072/h09/img201911221559540.jpg)  
  
  
[![](.evernote-content/DEB58D4B-D83C-4F21-B567-84B60DF12943/1FF39A36-B21D-4899-9383-ADC164BEFB8C.jpg)](https://resource.feng.com/resource/h072/h09/img201911221559450.jpg)  
  
  
官方信息显示，Cybertruck 车长 231.7 英寸（约 5.8 米），宽 79.8 英寸（约 2 米），最多可载 6 人，并且在第二排座位下还有额外的储物空间，中控屏幕尺寸为 17 英寸，空气悬架及以上到离地间隙 16 英寸（约 40 厘米），配置标准自动驾驶仪。  
  
[![](.evernote-content/DEB58D4B-D83C-4F21-B567-84B60DF12943/3D87BCB2-52AD-473C-A3B6-1933AD361F52.jpg)](https://resource.feng.com/resource/h072/h09/img201911221600060.jpg)  
  
特斯拉表示，Cybertruck 使用了 Ultra-Hard 30X 冷轧不锈钢结构表皮以及装甲玻璃，能够实现极强的耐用性和抗撞击能力。

![](.evernote-content/DEB58D4B-D83C-4F21-B567-84B60DF12943/6276C0FC-0B58-411E-9170-C5EB84CF855E.gif)

[![](.evernote-content/DEB58D4B-D83C-4F21-B567-84B60DF12943/4BB4510D-AA10-4B34-9D8E-D6EFB955961F.gif)](https://resource.feng.com/resource/h072/h09/img201911221604070.gif)  
  
作为一辆电动皮卡，官方表示 Cybertruck 拥有高达 3500 磅的有效负载能力和可调节的空气悬架，以及最高 14000 磅的牵引力，能够适应多种不同强度的工作环境。  
  
[![](.evernote-content/DEB58D4B-D83C-4F21-B567-84B60DF12943/246F27AB-3041-425B-A2C4-0142667DF34B.jpg)](https://resource.feng.com/resource/h072/h09/img201911221602490.jpg)  
  
Cybertruck 分为单电机后轮驱动版、双电机全轮驱动版和三电机全轮驱动版，三款不同版本的具体数据如下：  
  
1. 单电机后轮驱动版：充满电后的单次续航里程分别为 **250 英里（402.3 公里）**、从 0 加速到 60 英里（96.56 公里）所需时间**小于 6.5 秒；**  
  
2. 双电机全轮驱动版：充满电后的单次续航里程分别为 **300 英里（482.8 公里）**、从 0 加速到 60 英里（96.56 公里）所需时间**小于 4.5 秒；**  
  
3. 三电机全轮驱动版：充满电后的单次续航里程分别为 **500 英里（804.7 公里）**、从 0 加速到 60 英里（96.56 公里）所需时间**小于 2.9 秒。**  
  
价格方面，Cybertruck 单电机后轮驱动版、双电机全轮驱动版和三电机全轮驱动版的价格分别为 **3.99 万美元、4.99 万美元、6.99 万美元，**换算为人民币的话分别为 **28 万、35 万、49.1 万**，目前已经可以在特斯拉官网预订，**预计 2021 年下半年开始生产**。
