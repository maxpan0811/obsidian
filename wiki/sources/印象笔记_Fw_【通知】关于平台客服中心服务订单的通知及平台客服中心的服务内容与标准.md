Dear all：

自2021年4月6日起，集团平台客服中心将对接全集团除定制包团以外订单的所有除下单以外的服务工作，

1、平台客服中心服务的产品类型范围：除定制包团以外的产品类型，含出发地自由行、出发地散拼团、出发地私家团、目的地散拼团、目的地自由行，及各种单项服务

2、销售的工作内容：负责对接客户沟通需求并在EBK中完成下单工作

3、订单的后续工作将由平台客服中心对接客户完成，

平台客服中心的服务内容包含且不限于：订单认领、订单信息的核对、订单调价、客人需求变更的跟进包括取消或增加资源、提取综合费用、开发票、认领水单、发送合同、回访等等

具体的服务内容请详见下方邮件，同时请销售留意下单注意事项部分。

主题：关于平台客服服务内容及标准  

如会议所述，提供客服平台服务内容及标准如下：同时需渠道商拓下单时，对于客户的特殊需求，需要在ERP订单中进行跟单备注，谢谢。

**渠道商拓下单注意事项：**

| 渠道商拓下单注意事项 | 1、所有EBK下单，客服平台默认订单信息为准； 2、下单时，已知所需让价时，在EBK下单后，需至ERP订单备注中注明相关内容； 3、客户增减项目时，客服平台询问产品端后，会将信息同步给渠道商拓，由商拓定价； 4、其他特殊事项跟单备注(联系包括且不限于邮件、企业微信等)。 |

**客服平台工作内容：**

| 名称 | 工作内容 |
| 咨询下单 | 1、登录在线客户端进行客户咨询  2、登录EBK平台进行下单 |
| 400电话/企业微信 | 电话呼入30秒接通，企业微信及时回复120秒。 |
| 订单认领 | 0.5小时刷一次，认领新订单，保证认领界面没有未认领的，当天订单当天处理。 |
| 订单管理 | 订单信息核对及确认客户  1、查看客户订单是否是测试单，如是，直接取消  2、非测试订单，检查客户信息，出行人信息是否完整：  A、客户信息完整：邮件/QQ/企业微信，联系操作部确认订单资源；  B、客户信息缺失：联系客户确定客户及出行人信息，确认是否有效订单，确认后邮件/QQ/企业微信，联系操作部确认订单资源；  C、确认资源后，联系客户核对客户，出行人信息，并确定付款时间；  D、与客户确定付款时间后，订单内勾选“确认客户”，备注客户付款时间  E、客户信息错误，有电话不通，空号，不全等情况，视为无效订单，备注清楚后等自动清位。  F、客户电话占线或未接，2小时内再次联系，仍未接通，次日联系客户，仍旧未接通的，视为无效订单。 |
| 订单调价，收款/实收  EBK订单，客户可以通过在线扫码支付，线下汇款，账户支付等方式付款：  A.在线扫码支付全款，订单会显示支付成功，系统自动修改为定金占位；  B.在线扫码支付订金（订金根据产品设置比例自动生成），订单会显示部分支付，系统自动修改为定金占位；  C.线下汇款，每日需去水单界面查看有无到账水单，认领后后关联并支付订单；  D.若线上支付或汇款金额为订金金额，需联系客户确认余款支付时间；  E.若为公司存量客户，有账期结算规则，在上单后为客户修改为担保占位；  F.若客户支付费用高于应收金额，调整销售让价，并做综合杂费；  G.其他特殊项目调整（涉及需定价的，则通知销售定价） |
| 订单后续维护  A.增减调整机票、酒店、景点等，确认开票及订单金额调整，确认分房；  B.根据确认的资源，在订单中添加备选资源等相关信息，并确认价格； |
| 订单收款，合同提交，收款确认书，出团通知书操作  A.订单需要再出行前完成订单全额收款，上单后即与客户确认付款时间，未全额付款订单3天内催收一次；  B.直客订单无需提供收款确认书，同行订单担保占位的，出行前需要提供收款确认书给财务，订单内上传收款确认书，客户订单管理下载敲章后再次上传，至订单下载。  C.直客订单实占占位后即可制作直客合同，填写合同人信息，合同金额以实际应收金额为准。若有同行下单，不方便发合同给实际出行客人的情况，直接填写同行信息，将合同下发给同行，对方手机或邮箱收到确认信件，进行确认，直客合同需要再订单出行前完成确认。  D.操作制作完成出团通知书后，订单客服通知客人在网页上及时下载，也可以在订单上下载后发送给客户。(最迟出团前一天通知客人下载） |
| 客人取消  1.客户在线发起退订后，订单客服收到ERP通知，点开通知处理  2.订单客服在订单中点击“退团”，操作审核时会注明该团损失金额和明细，客服联系客户告知损失，客户确认后告知相应款项2周左右到账；  3.订单客服再次回到订单界面点击“确认”，并进行订单应收调整（无需审核）。  4.完成后，在订单管理界面点击“超收回退”，输入金额后，待财务审核；  5.财务审核完成后，订单客服在销售列表“客户账务管理”界面点击相应款项的退款。 |
| 跟单备注  针对每一次咨询确认，客人或同事协调间，及时于订单下方跟单备注，详细记录时间节点，以备复查。 |
| 水单认领 | 当天完成水单认领的支付。 |
| 发票 | 根据客户类型，及时处理并填写发票信息。 |
| 查看通知 | 频刷，处理客户在平台发起的订单取消 退订 退款 线下汇款，处理完成后结束通知 |
| 团队点评 | 在团结束的14天内做好点评工作。为防止外网BUG造成点评不显示，建议导游留好客人点评截图。若飞猪平台可以团上点评，则点评工作最好在团上完成。中差评/未点评，采取罚款机制，金额需商议，惩罚金额当团结算中扣除。所有处罚情况应该第一时间汇总至操作主管处。对中差评所提内容进行汇总，后续针对问题进行改进。 |
| 回访 | 回团一周内，对团上客人进行随机回访，询问客人对整个旅程的意见和建议，及时反馈改进。 |
| 服务时间 | 9:00-18:00 |
| 服务标准 | 电话接通率95%，在线接通率100% |
| 服务速度 | 电话30秒内应答，微信120秒回复，订单提交后3小时内联系客户（其中资源确认产品端时间为2小时），紧急问题2小时内解决，非紧急订单问题8小时内解决。 |
