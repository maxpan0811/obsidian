低密度脂蛋白胆固醇
=========

编辑
[锁定](https://baike.baidu.com/view/10812319.htm "锁定")

本词条由[“科普中国”科学百科词条编写与应用工作项目](https://baike.baidu.com/science)
审核
。

胆固醇在血液中常以[脂蛋白](https://baike.baidu.com/item/%E8%84%82%E8%9B%8B%E7%99%BD/2405269)的形式存在[1]
，而血浆中低密度脂蛋白（Low Density Lipoprotein，LDL）是运输内源性胆固醇的主要载体，其通过结合其[细胞膜](https://baike.baidu.com/item/%E7%BB%86%E8%83%9E%E8%86%9C/601737)上的低密度脂蛋白受体（LDL-R） 被降解和转化。

低密度脂蛋白胆固醇（Low-Density Lipoprotein Cholesterol，LDL-C）是由LDL-C是空腹血浆中的主要脂蛋白，约占血浆脂蛋的2/3，是运输胆固醇到肝外组织的主要运载工具[2]
。

LDL-R功能缺陷会造成血浆LDL-C的清除能力降低， 最终导致动脉内膜粥样斑块形成。因此，LDL-C的含量与心血管疾病的发病率及病变程度相关，被认为是动脉粥样硬化的主要致病因子，其浓度与[冠心病](https://baike.baidu.com/item/%E5%86%A0%E5%BF%83%E7%97%85/547914)的发病率有明显正相关，也是评价个体[冠心病](https://baike.baidu.com/item/%E5%86%A0%E5%BF%83%E7%97%85/547914)发生的危险因素的一个重要指标[1-2]
。

中文名
:   低密度脂蛋白胆固醇

外文名
:   Low-Density Lipoprotein Cholesterol

简    称
:   LDL-C

正常值
:   105-130 mg/dl

氧化性
:   极易被氧化

病理因素
:   动脉粥样硬化、心血管疾病等

降解机制
:   被低密度脂蛋白受体(LDL-R)降解或转化

相关简介
----

低密度脂蛋白胆固醇（Low-Density Lipoprotein Cholesterol，LDL-C）是在血浆中由极低密度脂蛋白胆固醇（ VLDL - C）转变而来，其合成部位主要在血管内，降解部位在肝脏[2]
。

### 低密度脂蛋白

[低密度脂蛋白](https://baike.baidu.com/item/%E4%BD%8E%E5%AF%86%E5%BA%A6%E8%84%82%E8%9B%8B%E7%99%BD/3854428)是富含胆固醇的脂蛋白，是一种运载[胆固醇](https://baike.baidu.com/item/%E8%83%86%E5%9B%BA%E9%86%87)进入外周组织细胞的脂蛋白颗粒。正常人空腹时，血浆中胆固醇的2/3会与LDL结合，其余的则由极低密度脂蛋白（VLDL）携带[3]
。低密度脂蛋白可被氧化成氧化低密度脂蛋白，被氧化的低密度脂蛋白胆固醇可通俗地理解为“坏”胆固醇，因为LDL-C水平升高会增加患冠状动脉心脏病的危险性。

高浓度LDL-C是明确独立的[动脉粥样硬化](https://baike.baidu.com/item/%E5%8A%A8%E8%84%89%E7%B2%A5%E6%A0%B7%E7%A1%AC%E5%8C%96)的危险因素，其中起作用的是氧化修饰的低密度脂蛋白胆固醇（Ox-LDL-C）。防止LDL-C被氧化是关键性的因素。当氧化修饰的低密度脂蛋白过量时， 其携带的胆固醇便积存在动脉壁上，久了容易引起动脉硬化。流行病学证据均显示低密度脂蛋白升高与心肌梗死和血管疾病死亡风险的增加密切相关[2]
。

### 高密度脂蛋白胆固醇

低密度脂蛋白胆固醇与体内另一种与胆固醇转运相关的脂蛋白——[高密度脂蛋白胆固醇](https://baike.baidu.com/item/%E9%AB%98%E5%AF%86%E5%BA%A6%E8%84%82%E8%9B%8B%E7%99%BD%E8%83%86%E5%9B%BA%E9%86%87/4857924)（High Density Lipoprotein Cholesterol，HDL-C）相对应。二者均与动脉粥样硬化及心血管疾病相关，前者（低密度脂蛋白胆固醇）与疾病的发生成正相关，后者是动脉粥样硬化的主要危险因素之一，属于致动脉粥样硬化脂蛋白，与疾病发生呈负相关。血清LDL-C水平越高，动脉粥样硬化的危险性越大。与HDL-C测定类似，LDL-C也是测定LDL中胆同醇量以表示LDL水平[2]
。

参考范围
----

成人低密度脂蛋白胆固醇的范围为2.07 ~ 3.37 mmol/L（80 ~ 130 mg/d）。我国新近修订的《中国成人血脂异常防治指南》建议：LDL-C < 3.37mmol（130 mg/dL）为合适范围；3.37 ~ 4.12 mmo/L（130 ~ 159 mg/L）为边缘升高；≥ 4.14 mmol/L（160 mg/d）为升高[3]
。

美国 NCEP ATPⅢ报告将LDL-C分成5个水平用于血脂异常的防治：< 2.59 mmoL（100 mg/dL）为合适水平；2.59 ~ 3.34 mmoL（100 ~ 129 mg/dL）为近乎合适水平；3.38 ~ 4.12 mmo/L（130 ~ 159 mg/dL）为临界高水平；4.14 ~ 4.89 mmol/L（160 ~ 189 mg/dL）为高水平；≥ 4.92 mmol/L（190 mg/d）为极高水平[3]
。

临床意义
----

### 病理性升高

肝功能异常、肝炎、动脉粥样硬化、高血压、心血管疾病、高脂蛋白血症、[急性心肌梗死](https://baike.baidu.com/item/%E6%80%A5%E6%80%A7%E5%BF%83%E8%82%8C%E6%A2%97%E6%AD%BB)、[冠心病](https://baike.baidu.com/item/%E5%86%A0%E5%BF%83%E7%97%85)、[肾病综合征](https://baike.baidu.com/item/%E8%82%BE%E7%97%85%E7%BB%BC%E5%90%88%E5%BE%81)、[慢性肾功能衰竭](https://baike.baidu.com/item/%E6%85%A2%E6%80%A7%E8%82%BE%E5%8A%9F%E8%83%BD%E8%A1%B0%E7%AB%AD)、肝病和[糖尿病](https://baike.baidu.com/item/%E7%B3%96%E5%B0%BF%E7%97%85)等，也可见于[神经性厌食](https://baike.baidu.com/item/%E7%A5%9E%E7%BB%8F%E6%80%A7%E5%8E%8C%E9%A3%9F)及怀孕妇女[4]
。

### 非病理性升高

1. 饮食不均衡，摄人的脂肪量过高；
2. 吸烟、饮酒引起的偏高；
3. 运动量过大、剧烈运动时显著升高；
4. 超重和肥胖的人容易引起低密度脂蛋白偏高[4]
   。

### 病理性减低

低密度脂蛋白胆固醇降低常见于营养不良、慢性贫血、骨髓瘤、创伤和严重肝病等[4]
。

影响因素
----

很多种因素都能够引起低密度脂蛋白的升高，主要可分为非病理性因素和病理性因素两类[5]
：

### 非病理性因素

（1）饮食不均衡，摄入的脂肪过高；

（2）吸烟饮酒引起的偏高；

（3）超重和肥胖的人容易引起低密度脂蛋白偏高[5]
。

### 病理性因素

（2）动脉粥样硬化，高血压、心血管疾病。低密度脂蛋白可以携带胆固醇进入人体动脉壁细胞，低密度脂蛋白高于正常值时，过量的胆固醇在血管壁堆积，形成动脉粥样硬化，甚至是冠心病的发生[5]
。

主要危害
----

### 形成动脉粥样硬化

如果血液中LDL-C浓度升高，它将沉积于心脑等部位血管的动脉壁内，逐渐形成动脉粥样硬化性斑块，阻塞相应的血管，最后可以引起脑卒中和外周动脉病等致死、致残的严重性疾病[6]
。

### 引起冠心病

LDL及其所携带的胆固醇（LDL-C）偏高常会引起冠心病等心脑血管疾病，所以有人称LDL-C为“坏胆固醇”。如果血液中LDL-C水平如果超出正常范围时，就会使心脏病发生的危险性增加[6]
。

错误认识
----

**1、机体胆固醇含量越低越好**

胆固醇主要分为[低密度脂蛋白](https://baike.baidu.com/item/%E4%BD%8E%E5%AF%86%E5%BA%A6%E8%84%82%E8%9B%8B%E7%99%BD)胆固醇（LDL-C）和[高密度脂蛋白](https://baike.baidu.com/item/%E9%AB%98%E5%AF%86%E5%BA%A6%E8%84%82%E8%9B%8B%E7%99%BD)胆固醇（HDL-C），LDL-C高于正常水平是“坏事”，但HDL-C高于3.0则是“好事”。高密度脂蛋白胆固醇可将血液中的多余的胆固醇转运到肝脏，处理分解成[胆酸](https://baike.baidu.com/item/%E8%83%86%E9%85%B8)盐，通过胆道排泄出去，从而形成一条血脂代谢的专门途径，也称“逆转运途径”[7]
。

**2、LDL-C是导致动脉粥样硬化的主要原因**

正常情况下LDL-C是以非氧化状态存在，非氧化的LDL-C并不容易引起动脉粥样硬化（小动脉壁像稀粥样的改变），最新版的《内科学》已明确阐述LDL-C被氧化成了氧化低密度脂蛋白胆固醇（Ox-LDL-C），这些氧化了的LDL-C才会沉积在血管内壁，导致粥样硬化。如果把LDL-C看成是奔驰在公路上的汽车，氧化了的LDL-C就是生锈有故障的汽车，HDL-C好比是清除路障的拖车。如果都车况良好的话，哪怕汽车多一点，交通照样顺畅，但有问题、生锈的汽车上路抛锚就肯定很容易导致交通堵塞了[7]
。
