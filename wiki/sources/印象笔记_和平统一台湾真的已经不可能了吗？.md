#国际政治与中美关系

![](.evernote-content/0523EA98-2FE2-4F9E-85B5-658BC677184F/F54DFF95-ED2A-4B1E-BF2B-8A69072E8E8A.png)

 

台湾问题的解决，不取决于所谓的台湾的人心向背，而取决于中美实力的对比。中国大陆实力够强了，台湾人最终会在武力统一和和平统一之间做出明智的选择。

 

 

1

 

台海两岸的现状是什么呢？是不统不独不和不武。

 

几年前，曾经有个台湾民调。如果中国大陆不使用武力，台湾民意压倒性的多数支持台湾独立；如果中国大陆使用武力，美国又不干预，压倒性的多数反对台湾独立；如果中国使用武力，美国干预，大概一半一半。

 

为什么中国中央政府永远不能放弃“武统”选项？因为放弃武统之日，就是台湾独立之时。大陆放弃“武统”，或者台湾人相信大陆不会动用武力，台湾马上就会独立。当前的台湾，无论是民意还是政客的意愿，都是压倒性的希望独立。

 

在上个世纪九十年代，台湾的GDP一度接近大陆的四分之一，台湾的军费和整个大陆的军费差不多。中美军事实力差距巨大。那时的两岸关系，可以用一个简单的博弈论模型来描述。台湾是主动方，可以先选独立或维持现状。

 

**这个模型下，维持现状的支柱是中国大陆的“威胁是否可信”。也就是你敢独，我必动武。**

 

中国大陆对台湾的武力优势和威胁，是台湾不敢独立的最核心原因。大陆为了防止台独，维持台海两岸的和平，就必须对台湾进行“可信”的武力威胁，让台湾相信，你敢独，我必动武。我们制定《反分裂国家法》就是用法律的形式，制定可信的武力威吓。台湾越相信大陆会动武，台海两岸越能维持和平。

 

台湾先在独立和维持现状之间选择。想达到右下角（0,0) 的均衡的方法就是，让大陆让台湾相信，红字部分的数值低于-10）。如果大陆无法让台湾相信这一点，台湾就会宣布独立。

|  |  |  |
| --- | --- | --- |
| （中国，台湾） | 武统 | 不武统 |
| 独立 | （-10，-100） | （-？,100) |
| 维持现状 | （-30，-100） | （0，0） |

这个博弈，和美苏冷战时的恐怖平衡类似。双方都让对方相信，你射核导弹，我也必定射核导弹，全力报复。冷战中，美苏双方强硬的威胁，反而是和平的保证。

 

对台独问题上，中国大陆没有一点缓和的余地，你敢独，我不惜一切代价也要武统。只有这样才能保住两岸的和平。

 

这是关于台湾的第一个博弈：**台海过去二十年之所以可以维持现状，是依靠中国大陆的可信威胁：你敢独，我就必然动武。**

 

 

2

 

二十世纪九十年代，在那个中国还是靠旅大级和七爷的时代，台湾的军费和整个大陆差不多。那时，中国大陆和台湾的海空军实力相差不大。今天，海峡两岸的实力对比，已经完全不一样了。中国大陆的军费远远大于台湾。如果美国不动手干预，台湾军队完全不是中国大陆的对手。

**今天的台海博弈，实际上台湾已经不是棋手，只是棋子了。**

很多人没有注意到，台湾的军费占GDP的比例，过去30-40年一直在下降。从最高的6%，下降到今天的不足2%。这个比例远低于美国，也低于美国军事保护下的韩国。

**今天的台湾其实已经自暴自弃，完全放弃了靠自己抵抗的幻想，完全指望美国干涉了。**

 

中国大陆不动手武统的主要原因是美国的可能干涉，次要原因是国际制裁和对经济的影响。

 

未来的台海博弈，将会是一个新的模型。中美之间博弈的模型。

 

中国是首先选择方。如果中国相信美国不会报复（介入武统战争），会放任中国武统，中国肯定就会武统。而美国想维持现状的核心是美国一方**“可信的威胁”。**

 

美国要让中国相信，红字的部分，低于-10。美国不干预的损失，大于美国干预，否则中国就会选择武统。

|  |  |  |
| --- | --- | --- |
| 中国，美国 | 干预 | 不干预 |
| 武统 | （-10，-10） | （10, -? ) |
| 维持现状 | （0，0） | （0，0） |

这个情况下，美国会用各种方法威胁中国，包括对台湾继续各种安全承诺，包括制定法案，要保卫台湾。用这个来让中国大陆相信，如果武统，美国必定介入。

 

这就是关于台湾的第二个，未来的博弈：维持和平现状，依靠**美国的可信威胁：你敢动武，我就干涉。**

 

 

3

 

在陈水扁还在台上的时代，李敖曾经把台湾比做中国的睾丸，攥在美国手里。当要打压中国时，就时不时就捏一下。搞个啥台湾政客访美或者美国政客访台，卖个武器，刺激恶心一下中国，来换取中国的让步。

 

为什么台湾会变成中国的睾丸？对于中国中央政府来说，台湾独立是永远无法接受的。一旦台湾宣布独立，或者美国正式承认台湾，中国中央政府是不可能无动于衷的，必须采取果断措施。

 

而美国，直到现在，都认为中国没有在美国介入下武力统一的决心和能力。只要美国干预，中国就会退缩。

 

美国政府认为，虽然大陆不断威胁，但一旦台湾宣布独立或美国承认台湾，大陆就会面临两难。不打，政治上交代不过去，打，在美国干预下打不赢。中国中央政府为了维持台海两岸的现状，为了维持美国对“一个中国”政策的承认，会对美国让步。所以，每次中美关系不好时，美国就会打台湾牌。

 

美国时不时捏一下，但也不敢使劲一下子捏爆了。最近美国拜登政府国务卿打破惯例，公然把台湾成为一个“Country”（国家），也就是再捏一下，试探试探。

 

美国认为自己可以时不时打台湾牌，时不时捏一下的理由就是：**美国认为，只要台湾不宣布独立，大陆就不会考虑武统。为了维持和平现状，不让台湾独立，大陆在台湾问题上就有求于美国。**

 

 

4

 

台海目前的“不统不独不和不武”就是这两层相互嵌套的博弈。

 

在过去，第一个博弈起主导作用。大陆希望避免台湾独立。这个情况下，美国就可以时不时打台湾牌，用台湾问题来恶心大陆。美国觉得大陆在台湾有求于美国。

 

那个时候，我们对台湾的态度也是友好为主，惠台助台，加强两岸关系，希望台湾不要独立。当然，同时也要继续武力威胁。

 

但在未来，也许，第二个博弈会逐渐变成主导。大陆的想法将不再是：台湾台湾你千万不要独立呀，你独立我就很难办，我又吓又拉，你千万不要独立；而变成：你独立，我100%要武统，你不独立，我也说不定要武统。

 

在这个情况下，第二个博弈就起了主导作用。那时，台湾就不再是大陆的睾丸，相反说不定会变成美国的小指。（说台湾是美国的睾丸，太抬举台湾了。台湾对美国没那么重要）

 

那时，中国的态度就是，台湾爱独立不独立。你独立我立马武统。你不独立，还得看你的表现，我不爽了我一样武统。

 

中国啥时候不爽，就掰一下台湾这个小指头。说我要打台湾。那时候两难的就是美国了。

 

美国干预，是双输，毕竟台湾在中国边上，中国离得近有优势。不干预，美国的老大地位就一去不复返了。那时，**美国就只能气急败坏的天天抗议中国，通过各种各样的“保卫台湾法”，来制造美国一方“可信的威胁”。**

 

那时，主动权就握在我们一边。

 

 

5

 

中美在东亚，又三个潜在局部冲突区域。朝鲜、台海和南海。其中，在朝鲜发生冲突对我们最有利，在南海最不利。

 

南海虽然修建了三个岛屿机场，但人工岛的防御能力是很差的。最近的陆地机场在海南。美国可以充分发挥海空优势。我们目前在南海打赢美国海空军，非常难。

 

朝鲜是独立主权国家，金三胖也不是傻瓜，时不时闹一闹，但他主动去打韩国的几率很低。

 

与南海相比，台湾，离大陆距离很近，陆基的超长距离火箭炮都可以打到台湾。在福建的远程防空导弹阵地，理论上都可以威胁台湾上空的飞机。在台湾和美国打一下，比南海优势大得多。

 

如果中美之间发生冲突，宁可在台海，也不要在南海。

 

一旦中国大陆有实力在美国介入的情况下，也能武统台湾，台湾局势就会完全不同了。

 

那时，美国就成了被动方，台湾就变成了美国的小指，被中国攥在手里，时不时掰一下。

 

**啥时候能这样？我也不知道，双方实力对比是军事机密，但我觉得再过10-20年就差不多了。**

 

 

6

 

我对海峡两岸和平统一是有信心的。

 

**但所谓和平统一，不是台湾人归心大陆，接受一国两制，主动统一。这个想都别想。**

 

在维持不统不独的现状和两岸和平统一两者做选择，甭管老一代台湾人还是台湾年青人，都一定会选择维持现状。

 

**但是，如果是在被武力统一，还是和平统一两者之间做选择呢？那时，我相信那时，台湾人会做出明智的，符合双方利益的选择。**

 

1945年8月，狂热高傲的日本人，也在“一亿玉碎”和“无条件投降”之间，选择了无条件投降。其后日本人对美国的恭顺态度，大家也都看见了。

 

再举个不太恰当的比喻：二战前，捷克是军工大国，军事实力不弱。为了防御德国，捷克投入巨资修了极其牛B的的防线，抵抗意志也很强。但慕尼黑协定，英法绥靖放弃保护捷克后，捷克做出的选择也是立马躺倒。为啥？实力差距太大，国家也没有纵深可以打持久战。

 

美国放弃南越后，南越又撑了多久呢？

 

所以，现在的台湾人有多少比例希望独立，根本无关紧要。

 

等台湾必须在被武力统一和和平统一之间做选择时，那时大概率就可以和平统一了。

其实，台湾当局已经早就放弃了军事抵抗大陆武统的念头。面对大陆这个实力远超过自己的威胁，台湾军费比例越来越低，连韩国和欧洲国家都不如。

 

**今天，因为中国大陆想保持和平，维持现状，台湾就变成了大陆的弱点，美国时不时就捏一下。如果有一天，中国大陆一方觉得和不和平无所谓了，该武统就武统吧。台湾就会变成美国的弱点，形势就会逆转。**

 

**那时，大陆压根儿就不怕台独。越快独越好，你不独，我还缺乏武统动手的理由呢。**

 

**那时，为难的就是美国。是放弃台湾，还是牺牲自己，保护台湾？**

 

**大陆军事力量已经形成对台湾压倒性优势，美国放弃台湾之日，就是两岸和平统一之时。**

 

毕竟，台湾岛没有脚，没法游到太平洋彼岸。台湾人，在武统与和统之间，在生存和毁灭之间，会选和平统一的。
