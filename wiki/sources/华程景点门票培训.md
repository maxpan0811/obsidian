# 华程景点门票业务培训

> 华程景点门票系统培训材料，涵盖背景、业务流程、系统操作规则。欧洲板块先行落地，2026年6月15日新功能切换节点。

## 项目背景与目标
- **背景**：实际成本与支出成本存在差异，成本数据失真；财务单据全人工操作；线下预订沟通工作量大
- **目标**：
  - 成本数据自动采集，实际成本与支出成本100%吻合
  - 财务单据100%自动化生成
  - 预订流程线上化，提升对接效率与预订准确性
- **范围**：欧洲板块

## 各岗位职责
| 岗位 | 职责 |
|------|------|
| 资源部 | 维护景点&门票信息；采购门票、申请支付；处理OP预订/修改/取消申请 |
| OP/研发 | 行程添加景点、勾选门票；提交门票集采/预订需求 |
| 财务 | 纸质票收票审核；支付审核&打款；制单 |

## 系统操作规则
- **景点管理**：资源部维护景点&门票 → 研发/OP才能在行程上添加景点、勾选门票
- **门票政策**：年龄限制（未维护则默认成人）、付款政策（现付/其他）、退改政策（允许取消/修改）
- **采购方式**：预订（支持预订）、集采（集团采购）、自采（分公司自行采购）、现买（导游现场购买）
- **价格规则**：必须维护，否则无法提交预订；低于最低人数按最低人数计算成本

## 切换节点
- 2026年6月15日之前出团的团沿用老功能
- 6月15日及之后出团均使用新功能
