# 《毁灭战士：黑暗纪元》为啥值得高分？FPS玩出动作游戏的爽感！

---

![](.evernote-content/F88EFD0A-C3F9-40A4-B854-39AEDA521EC1/99F61833-93AF-4270-9AB1-B0BA252F5983.jpg)

  
最近《毁灭战士：黑暗纪元》媒体评分出炉，9 分、8 分比比皆是，评语更是一片称赞之声。不少第一次接触 DOOM 的朋友可能有点疑惑，这看起来就是“无脑突突突”啊？为啥大家会给出这么高的评价？

即便抛开辉煌的当年勇不谈，最新作“黑暗纪元”也呈现出了极强可玩性！穿行于枪林弹雨的射击爽游作为基酒，开发商 id Software 又往其中灌入了盾牌、弹反等颇具动作性的新元素。什么？你说还是想不出来到底咋玩？那不妨跟着果娘继续往下看！

![](.evernote-content/F88EFD0A-C3F9-40A4-B854-39AEDA521EC1/BC831166-63C6-49A9-A09F-1438E057E2D3.jpg)

  
>>>时间线来到“起源”时代，新玩家不必担心看不懂故事

《毁灭战士》这个 IP 已经有三十多年历史，没玩过前作担心看不懂剧情也是人之常情。所以聊玩法之前，果娘先给各位新朋友吃颗定心丸，“黑暗纪元”时间线设定在整个 DOOM 宇宙的初期阶段，主人公尚未成为恶魔之敌时的远古时代。

流程中对于 DOOM“前世”身份的阐述极其值得玩味，尽管完全可以当做起源故事来看，但老玩家应该也能与前作“永恒”DLC 的故事联系起来。而且这代剧情演出观感很好，宗教意味与各种高维度的天神设定，让电影式的“大片”并没有俗气感，或许真能更广泛传播和流行。

![](.evernote-content/F88EFD0A-C3F9-40A4-B854-39AEDA521EC1/1433DA23-5814-40DE-A49B-F729EC0D7078.jpg)

  
>>>DOOM永远是最爽快的FPS，而且这代还有动作元素

如果说以前的 DOOM 讲究以快打快，端起枪往前冲，高速跑图消灭眼前所有恶魔就完事了。那这次《毁灭战士：黑暗纪元》最大改变就是把“冲”改成了“走”，主人公机动性远不像前作“永恒”那般迅捷，因此必须思考敌人站位、资源配置与控制节奏。而且敌人 AI 也有所升级，恶魔们明显更有脑子，直接莽上去很容易被密集火力打败。

所幸由于盾牌的加入，主人公性能绝对能匹配上敌人的进步，尽管整体节奏变慢，但射击爽感并未减弱。同时我甚至觉得各种弹反、格挡、近战时机的把握极为上瘾，游戏调性简直是“DOOM+鬼泣+只狼”的混合体，太有打击感啦！

![](.evernote-content/F88EFD0A-C3F9-40A4-B854-39AEDA521EC1/01B38B30-2D89-470C-A6FA-5EAC6D98DFD8.jpg)

  
详细来说，战斗系统的最大改变莫过于“盾锯”，前作中的电锯只是获取弹药简单机制，而现在则被彻底重塑为集格挡、防反、突进、投掷于一体的核心要素。

盾锯可以简单概括为 3 种用法：首先格挡敌人攻击之后，可以为近战武器充能，而近战攻击则可以回复枪械弹药，从而达成最简单的输出循环；精准格挡还可以触发反击动作和短暂的无敌时间，以此应对四面八方的流弹，等于是在 FPS 之上又加入了“打铁”要素。

其次，想要快速移动也少不了盾锯加持，格挡时可以触发“锁定”效果，随后便能向敌人派生冲刺动作，不仅属于无成本的快速位移，而且也可以当成起手抢招打断敌人的攻击动作。

除此之外，盾锯甚至还能投掷出去，不管直接砸向敌人，还是触发关卡中的各项机关都极为实用。毫不夸张地说，它比各种枪械更像核心机制，甚至还能通过技能加点决定盾锯的成长方向，防御反击、主动攻击、位移控制，多种对应技能树保障了每位玩家都能构架出顺手的打法。

![](.evernote-content/F88EFD0A-C3F9-40A4-B854-39AEDA521EC1/5D9756C1-D581-4C2F-BFB6-82ACE5FB10A5.jpg)

  
当然作为硬派 FPS 的代名词，各色枪械依然是《毁灭战士：黑暗纪元》的重要输出手段。这代枪械系统进行了一定程度简化，玩家能同时装备两把远程主武器，战斗中快速切换也保证上手就能玩懂，基本没有什么刁难人的门槛。

十余种远程主武器会根据成长解锁不同的专精方向，举个例子：霰弹枪可以走爆炸伤害流，也支持变成吸血枪保证生存。火箭筒既能玩单发高伤害，也能变成对群宝具。

最后也不能忘了拳套、连枷和战锤 3 种近战武器，经典的近战击杀回血机制也是堂堂回归。保证可观回血收益之后，暴力处决镜头更是主观上就让人很愿意近战开干。

种种这些综合下来，我们后期可以根据关卡的具体情况及时更换武器，偏线性的时候可以不考虑补给输出最大化，偏沙盒的关卡则要顾及回血和生存，总之难度曲线和游戏节奏的把控都很好。随着各项枪械逐渐成型，爽感也是指数级上升。

![](.evernote-content/F88EFD0A-C3F9-40A4-B854-39AEDA521EC1/3C96F95A-E401-42B4-B204-51F37C279C88.jpg)

  
>>>线性战斗提供了射击爽，飞龙、机甲关卡则带来了新鲜感

《毁灭战士：黑暗纪元》包含经典的线性、半沙盒关卡，哪怕是偏沙盒的地图也谈不上复杂。不过流程中进一步加强了解谜要素，基本每个区域都有独立机关与平台谜题，坦白说它们都不算难，但由于还要思考敌人火力分布等要素，再加上本作地图 y 轴上明显有很多设计，使得纵深之间脑力过载也就成了常见情况。

当然，过载绝对不代表批评，相反，我觉得浮空平台、爬墙机制以及盾锯投掷与冲撞构成的解密真挺好玩的，而且地图界面与 UI 界面也适配解谜进行了优化，玩家可以更清晰地标记目标、规划路径。如果各位是前作或者 DOOM 老粉，那上手后肯定能直接爽玩，这里也就不再展开。

![](.evernote-content/F88EFD0A-C3F9-40A4-B854-39AEDA521EC1/0D3350D3-FCC2-4358-B46D-6A6D452CA0C5.jpg)

  
更为难得的是，除了常规关卡之外，流程中还有两种特殊关卡可供挑战。首先我们可以操控巨型机甲 Atlan 与恶魔军团正面交锋，超级大场面某种意义上让我想起了前几年《最终幻想16》的召唤兽大战；另外，部分关卡还能骑乘飞龙在空中穿梭，飞机大战一般对抗飞行的怪物，玩法虽不复杂，但呈现的视觉效果极为靠谱~

尽管这两种关卡占比较少，不过几张图而已，但各项机制都几乎独立，完全打破了主线流程的单调感。说句像另外一款游戏也不算夸张，作为射击大战的调剂绝对合格！

![](.evernote-content/F88EFD0A-C3F9-40A4-B854-39AEDA521EC1/73F82612-A44E-49F1-AD8F-9C1F9419ED52.jpg)

  
>>>结语

总的来说，《毁灭战士：黑暗纪元》延续了丰富的枪械种类，以及市面上独一份的射击爽感，复古 FPS 第一人的称号依然适用。与此同时，盾锯带来了极为靠谱的动作元素，格挡、弹反并非单独存在，而是切实融合进输出循环之中。再搭配视觉冲击力拉满的近战武器，整体战斗节奏张弛有度，无论新老玩家都能快速上手，并且瞬间感知到 DOOM 究竟是一款怎样的游戏！

此外，线性、沙盒、机甲、飞龙等多样关卡，则带来了丰富玩法，搭配大片质感的剧情故事，你总会期待下一关究竟是什么样子，直到通关都会保持新鲜感。

对于老粉而言，《毁灭战士：黑暗纪元》仍然是市面上最好的 FPS 爽游，而且这次新加入的盾锯、技能树与枪械升级带来的 RPG 养成感也很足！如果你是第一次接触 DOOM，本作更是极为适合入坑的起源故事，新时代画质加持下绝对能让你肾上腺素飙升，爽杀恶魔根本停不下来！
