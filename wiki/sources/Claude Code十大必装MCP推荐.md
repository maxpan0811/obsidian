很多人装了Claude Code之后就直接开始用了，不知道它还能装扩展。

MCP就是Claude Code的扩展系统。原始状态下它只能读写你电脑上的文件、跟你对话。

装上MCP之后，它能操作浏览器、能抓网页、能连数据库、能发飞书消息——能力边界一下子被打开了。

![](attachments/6b6d92bb4ffcba62.png)

---

我自己装过不下几十个，下面是真正留下来一直在用的10个👇

1️⃣ Playwright — 自动操作浏览器

让AI帮你打开网页、点按钮、填表单、截图。自动发帖、批量截图、抢票思路都靠它。我小红书的配图截图和自动发布全是Playwright干的。装了它，浏览器里能手动做的事，AI都能帮你自动做。

2️⃣ Desktop Commander — AI接管你的电脑

比自带的文件读写强太多。它能跑终端命令、管理进程、搜索文件内容。相当于AI不光能看你的文件，还能像你一样操作整台电脑。装了它之后你会觉得“这才是完全体”。

3️⃣ GitHub — 操作代码仓库

直接在对话里提PR、查Issue、做代码审查、管理仓库。用Claude Code写代码的人迟早要和GitHub打交道，这个MCP让整个流程丝滑到不用离开终端。

![](attachments/50f831ebc8ea4d48.png)

4️⃣ Context7 — 让AI不再瞎编代码

目前最火的MCP，热度榜第一。它会自动把你用的技术框架的最新文档注入给AI，这样AI写代码时参考的是真实文档而不是它自己的记忆。装之前AI经常给你写过时的API，装了之后这个问题基本消失。

5️⃣ Firecrawl — 抓网页内容

给它一个网址，它把整个页面内容抓下来变成干净的文本给AI分析。做竞品调研、批量采集数据、分析某个网站的内容结构，都靠这个。比自己写爬虫简单太多。

6️⃣ Fetch — 轻量版网页读取

和Firecrawl类似但更轻。给一个URL，直接读取内容。适合快速看一篇文章、拉一个API的返回结果、读一个在线文档。不需要复杂抓取的场景用它就够了，简单直接。

![](attachments/d03e5a72bbdc26b7.png)

7️⃣ PostgreSQL/SQLite — 直接操作数据库

AI能连上你的数据库，帮你查数据、建表、跑SQL、做数据分析。以前要自己写SQL语句，现在说"帮我查上个月销售额前10的产品"它就直接查了。对有数据库的项目来说是刚需。

8️⃣ Sequential Thinking — 让AI学会分步思考

Anthropic官方出的。遇到复杂问题时，它会让AI把任务拆解成多个步骤，一步一步想清楚再动手。没装的时候AI容易东一榔头西一棒子，装了之后处理大任务明显更靠谱。

9️⃣ 飞书 MCP — 连接你的办公工具

能读写飞书文档、操作多维表格。意味着AI可以帮你写周报、更新项目进度表、整理会议纪要，直接写进飞书里。打工人的效率神器，省掉大量复制粘贴。

🔟 Excel MCP — AI直接帮你做表

读写Excel文件、写公式、操作工作表。丢一个表给它说"按月汇总销售额，加个同比增长列"，它直接改好给你。每天跟Excel打交道的人装了这个，下班时间至少提前一小时。

![](attachments/983a1d5d5828f865.png)

💡 怎么安装MCP？

去mcp.so或者GitHub搜 awesome-mcp-servers，上面收录了上万个MCP。找到想装的之后，在终端里输入：

claude mcp add 名称 -- 启动命令

比如装Playwright：

claude mcp add playwright -- npx @playwright/mcp@latest

一行命令就搞定，不用改配置文件。装完重启Claude Code就能用了。

 Claude本身是一个大脑，MCP是它的手和脚。光有脑子不能动手，装上了才是一个完整的人。

先从Playwright装起，体验一下AI帮你操作浏览器的感觉。你会上瘾的。
