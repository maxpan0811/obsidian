MacBook Pro (15 英寸，2017) - 技术规格
===============================

### 

### Multi-Touch Bar

* 配备集成式 Touch ID 传感器的 Multi-Touch Bar

### 外观

* 银色
* 深空灰色

### 显示屏

* Retina 显示屏
* 15.4 英寸 (对角线) LED 背光显示屏 (采用 IPS 技术)；初始分辨率 2880 x 1800 (220 ppi)，支持数百万色彩
* 支持的扩展分辨率：
  + 1920 x 1200
  + 1680 x 1050
  + 1280 x 800
  + 1024 x 640
* 500 尼特亮度
* 广色域 (P3)

### 处理器

* **2.8GHz**  
  2.8GHz 四核 Intel Core i7 处理器 (Turbo Boost 最高可达 3.8GHz)，配备 6MB 共享三级缓存  
  *可选配 3.1GHz 四核 Intel Core i7 处理器 (Turbo Boost 最高可达 4.1GHz
