“争做小英雄·铸就强国梦”2021大型军事主题活动
=========================

北京天骄快乐家族 北京天骄快乐家族幼儿运动课程

![](.evernote-content/C12D9C9B-C589-4904-BB0D-13F233BFFD92/92DDA347-F013-426E-A333-67BF5E1FAEF0.gif)

**争做小英雄 铸就强国梦**

—— 2021大型军事主题活动 ——

![](.evernote-content/C12D9C9B-C589-4904-BB0D-13F233BFFD92/9B0A1FD3-7802-4246-B94E-EE70C2EE45FC.jpg)

**活动背景**

1.幼儿国防课是对幼儿进行国防教育的重要课程，同时也是锻炼幼儿刚强意志和强健体魄的重要途径，**从小培养幼儿的爱国思想！**

2.幼儿国防教育为了**培养孩子们积极进取，勇敢坚强的优良品质**，让孩子们了解中国的强国、强军梦，激发爱国主义情怀！

3.了解中国的军事文化，真正认识到少年智则国智，**少年富则国富，少年强国强**，促使孩子萌发热爱祖国之情！

![](.evernote-content/C12D9C9B-C589-4904-BB0D-13F233BFFD92/7372036A-9F88-41BB-B8D7-EB1FABE07448.jpg)

**活动目标**

**1.军事训练：**通过军事化的训练，培养幼儿服从纪律，听从指挥的良好品质。改变幼儿不良习惯，培养军人气质。

**2.体能训练：**通过军事体能训练，综合锻炼幼儿走，跑，跳，钻，爬，投等技能，提高幼儿身体综合素质。

**3.安全教育：**通过活动，让幼儿学习安全常识，提高安全意识，提高自我保护能力。

**4.国防教育：**树立良好的爱国主义情怀，学习国防知识，增长幼儿见识。

**5.心理教育：**培养幼儿坚韧不拔，勇敢顽强的作风，打造幼儿不怕困难，勇往直前的精神，挑战极限，激发潜能。

![](.evernote-content/C12D9C9B-C589-4904-BB0D-13F233BFFD92/75C13137-D1DB-4FCC-9C9A-569EBA86603A.jpg)

**活动准备**

|  |  |
| --- | --- |
| 1 | 假设活动9点整开始 幼儿园提前1.5——2个小时到达 |
| 2 | 舞台、及背景布 |
| 3 | 各班级：负责签到班级人员、组织家长小朋友到各自班级的候场区排队 |

|  |  |
| --- | --- |
| 4 | 活动开始之前半小时、各班级提前通知家长带着幼儿上厕所。 |
| 5 | 主班配班及保育员在活动中随机应变。 |
| 6 | 活动结束：主持人宣布结束、合影留念 |

![](.evernote-content/C12D9C9B-C589-4904-BB0D-13F233BFFD92/16DED177-AFFA-43DD-AF47-8E1A1688CEE7.jpg)

**活动保障**

|  |  |
| --- | --- |
| 1 | **教练系统：**  师生配比1:30；主训教练均经严格选拔及培训，全场把控，反复演练；应急团队：专人负责，随时在线。 |
| 2 | **组织管理系统：**  严格按照方案流程组织活动,每小队选举责任心强的人担任队长、副队长以及安全员，明确责任义务以达到有效管理； |
| 3 | **装备品质系统：**  活动配备的所有装备，都是经过国际品质体系认证过的名牌产品，以保证活动安全万无一失； |

|  |  |
| --- | --- |
| 4 | **后勤保障系统：**  为每位学员购买人身意外及医疗保险，备有急救箱应付突发事件，并配备专业后勤教练精心服务。 |

**温馨提示**

1.本活动经过实地详尽考察，独家策划活动环节，并对活动项目作出安全评估；

2.建议给孩子穿宽松的秋季衣物、运动鞋；

3.随身物品：背包/单肩包、水杯、个人专用药品（如有医疗史，报名时请提前告知）；

4.请勿携带大量现金、贵重物品、手机等电子产品；

5.活动中请听从老师的安排，严禁私自离队，参观时请爱护公物，未经允许不得随意触碰。
