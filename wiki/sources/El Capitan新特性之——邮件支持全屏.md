# El Capitan新特性之——邮件支持全屏

---

[![](.evernote-content/6885DB4E-6744-4423-810B-27AD9095FE16/042436BB-93D4-4E71-BA01-2EFE18248051.jpg)](http://www.feng.com/apple/tutorial/2015-10-22/El-Capitan-of-new-features---email-support-full-screen_627877.shtml)

每一年的 OS X 系统升级，苹果都没有忘记对原生的邮件应用进行升级。但每一次的升级，似乎又都会让用户不甚满意。在 El Capitan 中，不知道邮件应用又会为我们带来什么新的改进呢？

[![](.evernote-content/6885DB4E-6744-4423-810B-27AD9095FE16/042436BB-93D4-4E71-BA01-2EFE18248051.jpg)](http://resource.feng.com/resource/h060/h59/img201510222103470.jpg)

在 OS X El Capitan 系统中邮件应用的基本应用体验和以前一样，但是苹果还对其进行了大量更新和完善，用户处理自己的邮件信息更加容易，最明显的视觉变化就是增加了对全屏的支持。

[![](.evernote-content/6885DB4E-6744-4423-810B-27AD9095FE16/EA4A1D30-E2E9-4482-B8F2-4F6FED06312D.gif)](http://resource.feng.com/resource/h060/h59/img201510222104080.gif)

邮件应用在支持全屏之后，用户处理多条邮件信息就更加容易了，假设你当前正在编写一封邮件，如果你想从其他邮件中查看或者复制某些内容，那么当你点击其它区域的时候，系统就可以把当前的编辑窗口移动到屏幕底部，从而访问其它邮件，并让用户复制其它邮件内的文本或者附件。

另外用户同时管理多个邮件对话也更容易。用户通过标签就可以在不同对话间切换，与 Safari 切换标签相同。
