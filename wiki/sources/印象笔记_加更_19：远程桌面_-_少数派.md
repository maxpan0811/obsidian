远程桌面，即可通过网络访问其他电脑的桌面。在生活中的应用场景有：

1. 在 iPad 或者 Mac 终端上访问局域网内其他电脑，进行一些图形界面上的操作。
2. 连接远程服务器，例如偶尔要使用 Windows 系统，就可以使用远程桌面连接 Windows 服务器，在上面进行相关操作，而无需另外安装虚拟机或者安装双系统。

接下来，就以 Setapp 中的两款远程桌面软件为例，介绍它们各自的使用方法，以及对比使用的总体体验。

Jump Desktop
------------

正如其名，Jump Desktop 是一款远程桌面控制软件，支持目前的四大主流平台。软件使用方法较为简单，在安装完成后，点击软件工具栏上的「+」号，即可添加远程桌面。

![](.evernote-content/0A1241CA-54FC-49CC-B310-4298CDE8EF59/4AF7050E-B0A7-4323-A8E5-A008B9FECA97)

此处可以看出，Jump Desktop 支持两种协议：RDP 和 VNC。RDP 是微软的私有协议，属于 Windows Server 和 Professional 的内建软件，Windows Home 需要额外下载方可使用。

VNC 则是开源的远程桌面协议，在所有平台都有对应的软件，在本地局域网中使用较为方便。但是如果在外网想要访问内网的机器，就需要会使用 SSH 进行端口转发。在 macOS 上自带了 VNC 协议，如果需要远程控制 Mac，需要在「系统偏好设置」> 「共享」的页面内，勾选「屏幕共享」。

![](.evernote-content/0A1241CA-54FC-49CC-B310-4298CDE8EF59/DCBA2D38-CEAB-4E7E-9115-8EA6A3D8A286)

以局域网内的一台 MacBook Air 为例，需要先查询该设备的 IP 地址，在另一台电脑上打开「系统偏好设置」>「网络」，可以看到此处的 IP 地址为：192.168.0.105。还有另一种方法，可以将鼠标移动到状态栏 Wi-Fi 图标上，按住 option 键，就可以显示当前网络的 IP 地址。

![](.evernote-content/0A1241CA-54FC-49CC-B310-4298CDE8EF59/68158108-473D-42DB-9DDD-4957002C4EEC)

打开 Jump Desktop 主界面，点击「+」号，因为是连接 MacBook Air，所以协议要选择 VNC 协议，IP 地址如上，Name 可以设置成需要的备注，最后点击 Add 即可。

![](.evernote-content/0A1241CA-54FC-49CC-B310-4298CDE8EF59/55FF4048-F10D-4CA5-97EC-CD57D77DB3B8)

但是，细心的读者可能会想到，那么在哪里输入用户名和密码呢？将鼠标移动到添加好多远程桌面上，显示出一个带齿轮的图标，点击「Edit settings」。

![](.evernote-content/0A1241CA-54FC-49CC-B310-4298CDE8EF59/408A67DD-39D7-4E27-B94A-E3D166EF3DEE)

点击第二行的「Auto Logon」，输入用户名和密码，最后点击保存，即可连接。但是要注意的是，如果设备的用户名是中文，用户名则是所对应的拼音。

![](.evernote-content/0A1241CA-54FC-49CC-B310-4298CDE8EF59/59DA37EE-84E6-4ED1-B9D2-C8059A5CBF65)

双击连接远程桌面，之后就可以像平时操作一样，直接远程控制 Mac。

![](.evernote-content/0A1241CA-54FC-49CC-B310-4298CDE8EF59/5415B572-C40C-4C40-9A05-43C4EF39B3CA)

Screens 4
---------

Screens 4 同样是一款远程桌面控制软件，它的用法与 Jump Desktop 别无二致，但界面上更美观一些。

安装好 Screens 4 之后，打开主界面点击左上角的「+」号，添加远程桌面。会弹出二级菜单，询问添加「新屏幕」或是「新建群组」。新建群组一般是在多台机器的情况下，才会使用分组功能。对于普通用户来说，直接选择「新屏幕」即可。

![](.evernote-content/0A1241CA-54FC-49CC-B310-4298CDE8EF59/A366BF8C-9F4A-4B3A-8649-6D344B60F965)

接下来，会弹出如下界面，Screens 4 此处的设计和 Jump Desktop 不尽相同。它会优先显示出局域网内的设备，如果需要连接局域网内的设备，可以从此处直接点选。如果不是局域网内的设备，可以点击左下角的「自定义...」，添加外网远程桌面。

此处依旧使用内网的 MacBook Air 为例，选中局域网的设备，点击「继续」配置设备。上面两个部分的信息，都是由软件自动生成，用户只需要填写设备的用户名和密码即可，填写好之后回车「保存」。

![](.evernote-content/0A1241CA-54FC-49CC-B310-4298CDE8EF59/F762FC2E-2A47-4515-A91E-1AE4534FFFB7)

此时，保存好相应的信息之后，软件回到主界面，双击设备即可连接远程桌面。连接远程之后，可以像日常一样，对电脑执行操作。但是需要注意的是，在远程桌面的顶部，有一排工具栏按钮。由于普通用户在日常比较少使用此类软件，可以在工具栏的位置点击右键，点击「自定工具栏...」，将显示设置修改为显示「图标和文本」。

![](.evernote-content/0A1241CA-54FC-49CC-B310-4298CDE8EF59/1FC0F3C4-02A7-4F40-95EA-C5DF0B70C619)

Which is better?
----------------

界面美观和易用性上，Screens 4 更加简洁明了，在工具栏上有完整的编辑按钮。例如链接、编辑和删除桌面。而 Jump Desktop 则需要点击右键才能显示功能。

在初次设置远程桌面时，Jump Desktop 需要在两个不同的界面，分别设置 IP 和账号密码，但是 Screens 4 会自动侦测主机，自动匹配对应的系统与主机名信息，只需要在单个界面直接配置即可。所以在配置的简易程度上，Screens 4 也是比 Jump Desktop 更加方便。

对于中文的支持，Screens 4 支持中文，而 Jump Desktop 还是英文界面，且没有更改语言的选项。

综上三点来看，如果读者需要使用远程桌面，建议直接使用 Screens 4，能够带来更好的体验，避免浪费时间精力在挑选软件上。

Remote Mouse
------------

Remote Mouse 是一款可以将你的手机或平板，变成一套好用的电脑遥控器的程序。

在电脑端和移动端安装好了之后，进入 iOS 客户端，如果在同一个局域网下，就会直接显示出设备名字，轻点即可连接控制电脑。如果找不到设备，点击状态栏上的图标 > 「显示二维码」，并使用 iOS 客户端右上角的扫描二维码，也可以添加设备。

![](.evernote-content/0A1241CA-54FC-49CC-B310-4298CDE8EF59/27C295BE-8E7A-48D8-B72C-3950BCB61F04)

进入主界面，可以看到一块淡绿色的界面。点击工具栏最左侧的更多按钮，滑到设置界面底部，可以自定义壁纸。在这个操作界面，通过手势操作完成如下操作：

* 单指轻按 = 点击鼠标左键
* 双指轻按 = 点击鼠标右键
* 双指上下左右滚动 = 在窗口中上下左右滚动
* 双指捏合展开 = 放大或缩小
* 三指拖动 = 选择并移动

![](.evernote-content/0A1241CA-54FC-49CC-B310-4298CDE8EF59/D8DCC89C-FF9C-44DA-8CE2-6069BC6F6A33)

在界面的右边，还有上下滚动条，可以在使用一只手指的情况下拖动。当然，这些都是它的基础功能，无须多言。比较有意思的，还是它的附加功能。但是要注意，附加功能需要另外付费购买。付费功能包括：多媒体遥控器、网页遥控器、程序遥控器、快捷键、电源选项。在未来，远程鼠标还会支持：局域网唤醒、自定义快捷键和文件浏览器功能。

在付费功能中，比较值得一提的是音乐控制功能。它可以用于控制电脑上音乐播放，点击工具栏上的音乐图标，即可显示控制界面。但是经过实际使用，目前媒体控制功能，只能控制音量的加减，对于切歌暂停等功能可能无法使用，有可能是软件的 bug。

而网页遥控器、程序遥控器和快捷键等功能，没有比较合理的使用场景，属于比较鸡肋的功能。所以建议把该软件作为临时应急之用，不需要再付费购买专业版，让它当一个简简单单的无线鼠标即可。

至于电源选项功能，如果有实际需求，建议使用捷径的 SSH 运行脚本，通过命令来实现电脑的休眠、重启、注销和关机等功能。

![](.evernote-content/0A1241CA-54FC-49CC-B310-4298CDE8EF59/4C774D21-2FA9-4C96-BEA5-8104DFF01DAD)

综上所述，对于 Remote Mouse 这款软件，只需要下载基本功能版即可，把它当成一个简单的无线鼠标使用。对于其他的功能，稍微把玩一下即可。
