# MacBook Pro (Retina, 13 英寸, 2013 年中) - 技术规格

---

MacBook Pro (Retina, 13 英寸, 2013 年中) - 技术规格
===========================================

![](.evernote-content/A232A738-FA2B-49BF-B07F-F1D98FB77D9F/03F604DE-E3EE-43F0-AB7B-E9E12F72371F.png)

### 显示屏

* Retina 显示屏：13.3 英寸 (对角线) LED 背光显示屏 (采用 IPS 技术)；分辨率 2560 x 1600 (227 ppi)，支持数百万色彩
* 初始分辨率：2560 x 1600 像素 (Retina)；扩展分辨率：1680 x 1050、1440 x 900 和 1024 x 640 像素

### 处理器

* **2.4GHz**  
  2.4GHz 双核 Intel Core i5 处理器 (Turbo Boost 高达 2.9GHz)，配备 3MB 共享三级缓存  
  *可选配具备 3MB 共享三级缓存的 2.6GHz 双核 Intel Core i5 处理器 (Turbo Boost 高达 3.1GHz)；或可选配具备 4MB 共享三级缓存的 2.8GHz 双核 Intel Core i7 处理器 (Turbo Boost 高达 3.3GHz)。*
* **2.6GHz**  
  2.6GHz 双核 Intel Core i5 处理器 (Turbo Boost 高达 3.1GHz)，配备 3MB 共享三级缓存  
  *可选配具备 4MB 共享三级缓存的 2.8GHz 双核 Intel Core i7 处理器 (Turbo Boost 高达 3.3GHz)*

### 内存

* **2.4GHz - 128GB**  
  主板集成 4GB 1600MHz DDR3L 内存  
  *可选配 8GB 或 16GB*
* **2.4GHz - 256GB**  
  主板集成 8GB 1600MHz DDR3L 内存  
  *可选配 16GB*
* **2.6GHz - 512GB**  
  主板集成 8GB 1600MHz DDR3L 内存  
  *可选配 16GB*

### 存储设备1

* **2.4GHz**  
  + 128GB
  + 256GB
* **2.6GHz**  
  + 512GB  
    *可选配 1TB 闪存*

### 尺寸与重量

* 高度：1.8 厘米 (0.71 英寸)
* 宽度：31.4 厘米 (12.35 英寸)
* 深度：21.9 厘米 (8.62 英寸)
* 重量：1.57 千克 (3.46 磅)2

### 图形处理器和视频支持

* Intel Iris Graphics
* 双屏显示和视频镜像：同时支持初始分辨率下的内置显示屏和高达 2560 x 1600 像素的两台外接显示屏，均支持数百万色彩
* **Thunderbolt 数字视频输出**
  + 原生 Mini DisplayPort 输出
  + DVI 输出使用 Mini DisplayPort to DVI Adapter 转换器 (需单独购买)
  + VGA 输出使用 Mini DisplayPort to VGA Adapter 转换器 (需单独购买)
  + Dual-link DVI 输出使用 Mini DisplayPort to Dual-Link DVI Adapter 转换器 (需单独购买)
* **HDMI 视频输出**
  + 支持 1080p 分辨率 (高达 60Hz)
  + 支持 3840 x 2160 分辨率 (30Hz)
  + 支持 4096 x 2160 分辨率 (24Hz)

### 摄像头

* 720p FaceTime HD 高清摄像头

### 外设连接和扩展

![](.evernote-content/A232A738-FA2B-49BF-B07F-F1D98FB77D9F/8BAAE32A-3A47-4F04-AFC0-B4C61FAF8B7A.png)

* MagSafe 2 电源适配器端口
* 两个 Thunderbolt 2 端口 (速率高达 20 Gbps)
* 两个 USB 3 端口 (速率高达 5 Gbps)
* HDMI 端口
* 耳机插孔
* SDXC 卡插槽
* Apple Thunderbolt to FireWire Adapter 转换器 (需单独购买)
* Apple Thunderbolt to Gigabit Ethernet Adapter 转换器 (需单独购买)

### 无线

* **Wi‑Fi 网络**802.11ac Wi-Fi 无线网络3；兼容 IEEE 802.11a/b/g/n
* **蓝牙**蓝牙 4.0 无线技术

### 音频

* 立体声扬声器
* 双麦克风
* 耳机插孔
  + 支持具有线控功能和麦克风的 Apple iPhone 耳机
  + 支持音频输出 (数字/模拟)

### 键盘和触控板

* 内置 78 (U.S.) 键或 79 (ISO) 键的标准尺寸背光键盘，包括 12 个功能键和 4 个方向键 (呈倒 T 形排列)，并配有环境光传感器。
* Multi-Touch 触控板可实现精准光标控制；支持惯性滚动、开合、旋转、轻扫、三指轻扫、四指轻扫、轻点、连续轻点两次和拖拽操作。

### 电池和电源4

* 长达 9 小时无线网络浏览
* 长达 9 小时视频播放
* 长达 30 天待机时间
* 内置 71.8 瓦时锂聚合物电池
* 60 瓦 MagSafe 2 电源适配器 (带线缆管理系统)；MagSafe 2 电源适配器端口

### 电气性能和操作环境

* 电压：100 至 240 伏交流电
* 频率：50Hz 至 60Hz
* 操作温度：10° 至 35°C (50° 至 95°F)
* 存放温度：-25° 至 45°C (-13° 至 113°F)
* 相对湿度：非凝结状态下 0% 至 90%
* 最高操作高度：3,048 米 (10,000 英尺)
* 最高储存高度：4,572 米 (15,000 英尺)
* 最高装运高度：10,668 米 (35,000 英尺)

### 有限保修与服务

中国的消费者权益保护法包括 ''三包'' 规定，赋予了消费者自购机之日起 2 年的主要部件质量问题保修服务。在此基础上，配备 Retina 显示屏的 MacBook Pro 也附带 90 天的免费电话技术支持以及来自 Apple 的 1 年有限保修服务。通过购买 [AppleCare Protection Plan 全方位服务计划](http://www.apple.com/cn/support/products/mac.html)，将你的服务和支持期限延长至购买之日起 3 年。只有 AppleCare Protection Plan 全方位服务计划可以为你提供直接来自 Apple 技术专家的电话支持，并保证所有维修都是经 Apple 授权的技术人员使用 Apple 原厂配件进行维修。要了解更多信息，请访问 Apple 技术支持或致电 (86) 4006-272273。

### 声音性能

噪声发射标示值符合 ISO 9296 标准

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
|  | **声功率级  LWAd (B)  1 B = 10 dB** | | **声压级  操作员位置  LpAm (dB)** | |
| **2.4GHz** | **2.6GHz** | **2.4GHz** | **2.6GHz** |
| **闲置** | 2.5 | 2.5 | 6.0 | 5.0 |
| **硬盘驱动器访问** | N/A | N/A | N/A | N/A |
| **CD 驱动器访问** | N/A | N/A | N/A | N/A |

1. LWAd 表示 A 计权声功率级的统计上限值（四舍五入到最接近的 0.1B）。
2. LpAm 表示在操作员位置处测得的 A 计权声压级（四舍五入到最接近的 dB）。
3. 1B (bel) = 10dB （分贝）。
