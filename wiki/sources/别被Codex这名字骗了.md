# 别被 Codex 这名字骗了

> 来源：APPSO

## 一句话
Codex已从"编程IDE"进化为全能Agent产品——不只是写代码，日常任务/生图/翻译/远程控制全能用，且手机端ChatGPT App可直接操控电脑端Codex。

## 核心功能

- **不只会写代码**：日常问答、GPT Image 2生图、深度研究、翻译全部支持
- **Skill/插件系统**：Chrome扩展、Computer Use、斜线指令
- **手机端控制**：iOS/Android ChatGPT App内完整Codex体验（不运行在手机上，运行在电脑上）
- **远程SSH**：直接连接公司远程开发环境

### 手机端三大能力
1. 随时查看任务进度（终端输出/截图/测试结果/文件差异）
2. 关键节点审批/指令（期待上灵动岛弹窗）
3. 主动发起新任务

### 竞品格局
- Codex vs Claude Code vs 龙虾（OpenClaw的开源前身）
- 商业化产品更稳定易用
- 作者调侃：Claude Code号被封，Codex就赢了
