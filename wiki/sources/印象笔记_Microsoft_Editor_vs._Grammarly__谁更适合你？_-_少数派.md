前言
--

微软在今年三月底上线了一个全新的服务，名叫 Microsoft Editor。这是一款由人工智能驱动的写作助手，将帮助你自信地撰写 20 多种语言。Microsoft Editor 不仅能提供基础的语法建议和拼写检测，如果你订购了 Office 365（不论个人还是家庭版），你还能享受到诸如清晰度，简洁性，一致性，词汇建议等高级语法润色功能。Microsoft Editor 不仅内嵌到 Office 全家桶中（Word, Outlook 等），也提供了浏览器插件供你在多平台使用。

不得不说，Microsoft Editor 和老牌语法辅助工具 Grammarly 实在是太像了。Microsoft Editor 和 Grammarly 都是为了解决一个非常简单的任务：改善你的写作。两种工具都旨在在你编写时修正拼写和语法错误，充当你写作工作中的校对官。二者都可以在多平台使用，也都提供 Word 等软件的内嵌辅助功能。

虽然 Grammarly 一直在语法检测辅助领域占据领先地位，但是此次搭载 Microsoft 365 更新推出的 Microsoft Editor 也是来势汹汹。面对这样两个内核几乎一致的语法工具，到底哪一个更适合你呢？我会先简单介绍下 Microsoft Editor 这款新工具，然后我将会从交互界面、语法检测功能，使用效果、语言支持、价格多个方面来详细对比两者的特点。

本文仅根据我个人使用体验进行横向对比，选用的版本都是高级付费版本，语法辅助语言均为美式英语。欢迎各位在讨论区继续补充和指正。

Microsoft Editor 简介
-------------------

正如前言所提到的，Microsoft Editor 提供免费版和 Microsoft 365 订阅版。免费版能提供基础语法和单词拼写错误检测，订阅版则能带来更多高级文法修饰和语法加强功能。

Microsoft Editor 同样对文笔风格（包括清晰度，简洁性，形式性和包容性等）进行润色和修正。比如包容性，它会提示作者使用更中性普遍的词汇来代替具有偏见的文本（policeman ->police officer)。

交互界面
----

交互界面是产品最直观的感受，是最直接影响用户体验的部分。界面排布是否清晰准确，使用逻辑是否符合大多数人平常的习惯，都会在很大程度上决定用户的使用体验和学习成本。

两者的交互界面基本没有差异，都是在文本右端提供一个文本校对显示。两者都可以在工具栏上直接点击图标进行启动。Grammarly 是将各项分类放在了工具栏上，而 Editor 是直接显示在校对框中。

高级功能比较
------

两款工具的基础功能几乎一致，主要的差别还是体现在高级版的功能。Microsoft Editor 拥有三大块主要的功能模块：文法修正 (Corrections)、精益求精 (Refinements) 以及行文优化 (Enhancement)。

Grammarly 不仅仅提供 Microsoft Editor 以及具备的以上功能，还有一些有趣的特点。Grammarly 提供了写作目标 (Writing Goal) 这个功能选项，用户可以选择他们的目标受众 (Audience)，熟悉程度 (Familiarity)、文章领域 (Field)、行文语调 (Tone) 以及文章目的 (Intent)。

例如，我在写少数派这类文章时，可以设定我的写作目标是以非正式的语气为知识渊博的读者写一些关于技术主题的东西。然后，Grammarly 将不断分析我的写作，并就如何调整从词汇到标点到句子长度的所有内容提出建议，以实现我的写作目标。

![](.evernote-content/8AF588D7-EE36-4923-98D0-DED4D699EC12/6D774D27-3503-4164-A1AF-7DCC7D628C11.gif)

Grammarly 的文章目标功能

Grammarly 还提供对于专业写作来说更有针对性的辅助功能，诸如词汇加强 (Vocabulary)、被动语态选择 (Passive voice)、行文流畅 (Fluency)、分解难读句型 (Split hard-to-read sentences) 以及学术写作中的人称代词选择 (Personal pronoun in academic writing) 等。

此外，虽然 Microsoft Editor 会在近期上架文献查重功能（Word 版已经可以查重，浏览器插件还无此功能），但是现在 Grammarly 已经能提供基础的查重。虽然还是比不上 [ithenticate](http://www.ithenticate.com/) 这种专业查重工具，但是基础的文献查重已经能完美实现。

![](.evernote-content/8AF588D7-EE36-4923-98D0-DED4D699EC12/976A9B3E-9866-4CE4-BC5A-B84D6D78692B.png)

基本的查重功能

使用效果
----

我测试了两种文档（学术论文以及普通邮件），由于 Grammarly 拥有更多功能，它的检测时间肉眼可见的比 Microsoft Editor 要慢。

以学术论文作为例子，我对比了自己一篇论文的前两个部分 (引言和理论)，总计 1616 字。两款工具都给文章打了一个总分，Grammarly 给出了 98/100 的分数，而 Microsoft Editor 的分数为 89/100。

### 拼写准确性

两者找出的问题数量几乎一致（Grammarly 提供 11 处修改意见，Microsoft Editor 指出 14 处）。Grammarly 提供的 11 处语法单词错误修改意见中，有 1 个是作者名字错误提醒，其他均为有效提示，准确率为 10/11=91%。相反，Microsoft Editor 指出 14 处的错误有 12 处是作者名字错误，2 处是标点符号规范，有效提示不足 15%。

Grammarly 的交互体验很不错，点击修正 (Correctness) 可以一目了然地看到所有的语法和单词的错误。错误建议，诸如删除空格，在 and 前加逗号以及重写成分缺失的句子都是正确并且需要进行修改的部分。

![](.evernote-content/8AF588D7-EE36-4923-98D0-DED4D699EC12/D26DE477-3168-4431-AD36-5449118B071A.gif)

Grammarly 的修正模式

然而，Microsoft Editor 对这块就稍显笨拙。需要自己手动点击每一个红色下划线的地方或者点击下一页符号，并不能直接显示所有的修改点。

![](.evernote-content/8AF588D7-EE36-4923-98D0-DED4D699EC12/86733091-D270-4E01-A975-3DA4B785F651.gif)

Editor 的修正模式

### 文章清晰度

对于文章清晰度 (Clarity), Grammarly 提供 16 处修改意见，Microsoft Editor 指出 27 处。Grammarly 提供的修改意见中，有 9 处被动语态提示 （被动语态并不应完全禁止），其他提示诸如指代词不清 (Unclear antecedent)、删除重复词汇 (Tauotology) 以及复杂句型简单化 (Rewrite hard-to-read) 都可以算作有效修改。清晰度有效提示率至少大于 7/11=64%。

![](.evernote-content/8AF588D7-EE36-4923-98D0-DED4D699EC12/F6549CD6-4D21-40FC-8EC0-FABE58E1D615.png)

Grammarly 的清晰度模式

在 Microsoft Editor 指出 27 处中，错误主要集中在三个方面：被动语态、使用具象形容词以及避免使用大词改用小词。除开四个被动语态的错误，Microsoft Editor 的有效错误提示率为 23/27=85%。

![](.evernote-content/8AF588D7-EE36-4923-98D0-DED4D699EC12/93AD11E6-F56C-43C8-B4D7-025CC634BC8B.png)

Microsoft Editor 的清晰度模式

### 文章契合度

对于文章契合度 (Engagement), Grammarly 提供 10 处修改意见，Microsoft Editor 虽没有特定模块，但 (Acronyms (3) + Conciseness (8) + Vocabulary (5)) 这 16 点也能算作文章契合度。Grammarly 主要集中在词汇的选择上，Microsoft Editor 除开词汇本身 (与清晰度有一定重合)，还对词汇简明和首字母缩写也有检测。两者的有效检测度都是 100%，Microsoft Editor 涵盖的模块更多一些。

![](.evernote-content/8AF588D7-EE36-4923-98D0-DED4D699EC12/2A6BD386-FA65-49AE-91D5-C046EA85EAA6.png)

Grammarly 的文章契合度模式

![](.evernote-content/8AF588D7-EE36-4923-98D0-DED4D699EC12/A2D73790-F5C5-4BAD-B972-FADE542BBE02.png)

Microsoft Editor 的文章契合度模式

### 文章交付度

交付度 (Delivery) 这一块，主要体现在文章是否符合目标群体。Grammarly 提供 3 处修改意见很好地契合了之前选择的文章预期目标，Microsoft Editor 中的文字形式 (Formality) 以及文字包容性 (Inclusiveness) 共 3 项可以算作这一类。两者在这一项都能做到完美的有效提示度，主要错误提示集中在短语缩写以及中性词的选择。

### 小结

整体来讲，两者在文章契合度和交付度的有效提示难分伯仲。在拼写准确性上，Grammarly 要更胜一筹，而在文章清晰度上，Microsoft Editor 做的更好。

平台适性
----

Microsoft Editor 和 Grammarly 都可以在 Word，Word Online 和 Outlook 中完美运行。浏览器插件方面，Microsoft Editor 目前仅支持 Edge 和 Chrome。然而，Edge，Chrome 和 Firefox 中都提供了 Grammarly。

语言选择
----

Grammarly 目前只支持四种类型的英语，Microsoft Editor 则能支持多达 20 多种不同语言。至少，你在写中文的时候，Microsoft Editor 能助你一臂之力，而 Grammarly 只能爱莫能助。

![](.evernote-content/8AF588D7-EE36-4923-98D0-DED4D699EC12/DDFE9D10-F427-4CEA-9621-B687252AA34D.png)

订阅价格
----

毫无疑问，尽管两者都提供免费版本，但 Microsoft Editor 是更便宜，更具「性价比」的选择。（虽然性价比本身也与个体差异存在很大关系）

只需要支付 Microsoft 365 订阅费用就能体验 Microsoft Editor 高级版，同样，如果想使用 Grammarly 的高级功能，也得需要订阅 Grammarly Premium 或 Business。

![](.evernote-content/8AF588D7-EE36-4923-98D0-DED4D699EC12/62B72936-14C8-43FC-B2E8-F2D837B170E2.png)

Microsoft 365 订阅费用

Grammarly Premium 的现在每月最低费用为 29.95 美元，或者你可以提前支付每年约 140 美元的年度订阅费用。即便它能提供众多先进文法检测功能，这个价格依然不便宜。值得欣慰的一点是，Grammarly 平时会提供大量的折扣或者学生折扣。如果你的学校恰巧购买了它，恭喜你，节省了一大笔钱。

![](.evernote-content/8AF588D7-EE36-4923-98D0-DED4D699EC12/91328F02-BE0D-48F6-ABE9-FC2F6035CB44.png)

Grammarly 订阅费用

相比之下，Microsoft 365 可以让你访问完整的 Microsoft Office 套件以及 1TB 的云存储，每月需支付 9.99 美元，如果您按年付费，则需要每年支付 99.99 美元。当然，这个费用是包含了 Microsoft Editor 高级版使用的。

微软已经承诺未来会为 Microsoft Editor 加入更多功能，在目前的订阅价格之下，Microsoft Editor 在价格上无悬念碾压 Grammarly。

综合选择
----

在急于订阅之前，值得尝试使用 Microsoft Editor 和 Grammarly 的免费版。选择适合自己的，比选择功能看似强大的，更为重要。

对于付费版的选择上，如果你只是一位普通用户，那么 Microsoft Editor 无疑是一个更好的选择。尽管缺少一些语法方面的高级功能，但也足够应对大多数写作场景。此外，它的价格有明显的优势，搭配 Microsoft 365 套餐以及微软良好的生态圈，让 Microsoft Editor 显得更不错。此外，多语言的支持也是一大亮点。

如果你是一位专业写作者或者有意改善自己的行文风格，Grammarly 势必是你的最佳选择。丰富强大的文法修饰以及精进功能，能在你写作的各个方面做出大量有益的建议。此外，久经考验的 Grammarly 具备良好的跨平台优势以及可靠稳定的插件服务。

如果你坚持使用免费版本，那么建议你使用 Grammarly。它带有 Microsoft Editor 所没有的少量附加功能，并且也更加稳定。
