# 人工智能：阶级斗争的归来？｜ AI: The Return Of Marxist Class Struggle？

人工智能：阶级斗争的归来？｜ AI: The Return Of Marxist Class Struggle? 智能摘要两位德国思想家，卡尔·马克思和弗里德里希·恩格斯，强调了阶级斗争在社会历史中的重要性。我们最近在18个国家进行的一项研究涵盖了13，000名受访者，突显出围绕人工智能，在领导者和员工之间，或者正如两位哲学家所说的"统治阶级"和"无产阶级"之间，存在着日益加剧的分歧。我们面临的挑战是创造条件，使得"人工智能中产阶级"出现。在从工业革命过渡到数字革命的过程中，通过全面的教育和培训实现人工智能技术的民主化，可能会导致新的均衡，使每个人，无论其阶级如何，都能平等地获得人工智能。原文
