特斯拉 Model 3 不到 30 万了，快来看春节前购买指南
===============================

​​盼望着，盼望着，国产 Model 3 的降价终于来了。

切记，特斯拉的一切销售原则都坚持着「为了当下快卖车」的终极目的，之前国产版价格定高是因为当时进口版车型仍然承担主力走量的任务，必须要通过国产版价格定高来体现进口版价格的合理；如今刺刀见红，是国产版降价冲刺的时候了。

![](.evernote-content/DDD8961F-7A17-4CA0-B925-AF1F25539F85/D188D5EE-6353-4D4F-A668-8C2E39A714B1.jpg)

国产版价格落入 30 万之内，和我在两个月之前的判断完全一致，这将让 Model 3 在入门豪华车和品质电动车两个消费群体中的吸引力无比增强。

来，一起看看怎么买比较合适。

补贴
--

补贴的影响最普世，也最容易理解。

国产版 Model 3 享受 2019 年度国家补贴。

如果是私人购买，可以享受 24750 元补贴，如果是商用，补贴 17325 元。

唯一需要注意的是补贴什么时候会因政策发生变化，一般每年补贴大调整是在六月底，但特斯拉也提示在今年一月底到二月初，补贴有退坡的可能。

售价及选配
-----

对国产版而言，直接降价，车价调整为 32.38 万，如果私人购买，则去除补贴后为 29.905 万元。

![](.evernote-content/DDD8961F-7A17-4CA0-B925-AF1F25539F85/6AAA7388-A24D-4B80-A96B-0B8AA4D5BDC3.jpg)

但对于特斯拉车主来说，这才是刚刚开始，几乎没有哪辆特斯拉不带有任何选装。

这次的选装价格调整，与其说是由于 Model 3 的制造工艺不断成熟导致的成本下调，不如说是为了让选配价格在更为低廉的基础价格面前显得过高而影响车主下单的热情。

Model 3 在不同的阶段会提供一款免费车漆，你可以理解成这是当季的流行色，此次降价之前一段时间免费的是珍珠白（多涂层）车漆，这款车漆在刚开始进口版起售的时候可是需要 13000 块钱的，可以说是天大的优惠了，而此次降价之后免费色被改为相对质感有限的黑色车漆。

![](.evernote-content/DDD8961F-7A17-4CA0-B925-AF1F25539F85/8CC9B961-09E1-48C4-B7A0-68EDD238CDEE.jpg)

利好的方面是其它所有车漆一并降价，并且统一变成 8000 元，这里我最推荐的还是我自己车选用的中国红（多涂层）车漆，工艺更好，气质上也更年轻运动，最重要的是，这款车漆在之前的价格一直近两万，此番相当于降价了一万有余。

相信街上红色的 Model 3 会逐渐多起来。

![](.evernote-content/DDD8961F-7A17-4CA0-B925-AF1F25539F85/D0F6CB1D-0729-46C3-A722-0F5E5BAA86A9.jpg)

轮毂上，又能给你省 1 万，作为之前唯一的官方选装项，19 寸的动力轮毂从 14200 降价为 6000 元，我的建议是，除非预算特别吃紧，一定要选。

理由非常简单，Model 3 还是一款很有驾驶乐趣的车，19 寸轮毂能让你的车在操控、能耗和舒适度上实现完美统一。另外，如果你看过 Fremont 工厂量产的第一辆 Model 3 的照片的话，就会发现当时使用的也是这款 19 寸轮毂，可以说它是 Model 3 最标准最原汁原味的轮毂设计。

![](.evernote-content/DDD8961F-7A17-4CA0-B925-AF1F25539F85/2EE6B9A2-679C-4A59-B437-CD0F104743E6.jpg)

另外，按照我国现行车辆审核政策，车辆可以改装与行驶证上同尺寸的轮毂，选装一个 19 寸轮毂也会让你的玩车空间增大。

![](.evernote-content/DDD8961F-7A17-4CA0-B925-AF1F25539F85/66D5F2A7-0DC1-4479-BB4B-86B211FF1052.jpg)

内饰方面，白色内饰不再对标准续航增强版开放，这是区分车辆档次的重要手段之一，也能更好地照顾进口版车主的内心感受，不过我觉得对国产版用户来说略微有点遗憾，毕竟白色内饰可以让你整车的质感提升一个档次。

![](.evernote-content/DDD8961F-7A17-4CA0-B925-AF1F25539F85/D347EF57-1BF3-41B2-96A3-7FA0706FD631.jpg)

其实真正会让人纠结的，是完全自动驾驶能力选项，5.6 万的价格相对车价着实不低，但 FSD 最近的进步也非常可喜，感觉价值会越来越高。这就要根据您是长期在市内短途使用还是偶尔也要长途使用来判断了。

充电
--

比较意外的是，本次调整之后全系 Model 3 不再包含充电桩和免费安装服务，如果有家用桩的使用需求的话，可以额外购买家庭充电服务包，价格为 8000 块钱。

我认为这是特斯拉中国机敏调整成本的优势体现，之前其实并不是所有车主都能安装充电桩，导致大家在拿到车之后部分车主只能将自己的充电桩再二次出售。

![](.evernote-content/DDD8961F-7A17-4CA0-B925-AF1F25539F85/57E04CBE-D47C-4F97-94B4-FD8BA3D50914.jpg)

而如今减少一个充电桩的成本只后，特斯拉可以把更多的利润让渡给选装项目上，让车价更便宜。

Model S、X 作为豪华旗舰，继续享有充电桩和安装服务。

我刚查了一下二手交易平台，目前的官方充电桩还处于较低的价格区间，需要的车主可以在价格波动之前提前收购一根备用。

几点建议
----

很多朋友都很关心怎么能尽快拿到自己的车，以我在 11 月初和上周两次到访工厂的感受，大胆猜上几点。

我们不妨以 30 日员工交付的车辆为样本，当天交付的 15 辆车，全部为蓝、白、冷光银三个颜色，而这段时间以来的大量航拍照片也证明，基本上生产的存车以这些颜色为主，所以它们肯定会被首先交付。

![](.evernote-content/DDD8961F-7A17-4CA0-B925-AF1F25539F85/CF1CD1AF-802B-4187-93A8-A56EC2E976FA.jpg)

而 15 辆车当中，19 寸轮毂占多数，这也说明特斯拉比较看好 19 寸轮毂的受欢迎程度。

最后，那天我在视频中着重指出供拍摄的新车 VIN 码超过 3000 了，说明可供交付的车辆还是比较充足的。

所以，如果你的车选型基本在上面的车漆范围里，应该会在近期收到交付的通知，如果是还没下订单的朋友，如果希望尽快提车，也可以参照上述范围选择自己的配置，

谈谈意义
----

2020 年的中国电动车市场，在销量构成上会与之前几年出现结构性的变化。

随着产品的逐步成熟和价格的逐渐下降，市场将逐渐从前几年的政策性导向转变为市场性导向。

而这个转变的利好，就在于真正能够在体验上与传统燃油车产生质变，具备过人品牌价值，同时价格不至于拒人于千里之外的产品将成为爆款。如下图，这是 2018 年 Model 3 在美国的表现。

![](.evernote-content/DDD8961F-7A17-4CA0-B925-AF1F25539F85/A220BBF9-BD62-43D6-9FCC-6E104D568068.jpg)

细分下来，第一是政策性市场，Model 3 在北京这样消费能力惊人的地区已经在去年取得了惊人的份额，而对于购买力相对次级，同时不限制购买混动车型的地区来讲，Model 3 能够利用的政策红利有限。

第二是需求型市场，即便是不限购城市，也有很多喜欢全新体验的高端用户，Model 3 的此次降价，将充分激发他们的购买欲。

第三是升级市场，之前大量保有的低价劣质电动车用户一直在等待合理的产品进行升级，他们对电动车的使用痛点已经充分掌握，在这里，Model 3 国产版的吸引力同样巨大。

而如同我开头所说，这次调价将对豪华入门级市场产生新的震动，很多之前固守燃油车的用户，也可能会因为价格合适来选择特斯拉，同样现在那些打算花 20 万 + 来选购自主品牌电动车的车主，特斯拉品牌的号召力真的不是几万块钱的差价能够阻止的。

我认为特斯拉中国会在本次调价之后尽可能地保持价格的稳定，而后续的大招就是跟进国产长续航 Model 3 和 Model Y 了。

祝大家尽早尽情尽美地开上属于自己的 Model 3。
