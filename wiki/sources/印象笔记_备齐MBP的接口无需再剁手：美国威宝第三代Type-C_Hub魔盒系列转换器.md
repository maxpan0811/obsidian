# 备齐MBP的接口无需再剁手：美国威宝第三代Type-C Hub魔盒系列转换器

---

新款MacBook Pro是有史以来最轻薄的，舍弃惯常的接口就意味着某些情况下，你必须使用一些额外的转接器来享受苹果带给你的“简单”及“便捷”的科技生活，若要在苹果官方买齐适配新款MBP的整套接口大约需要2000人民币，那么问题来了，面对林林总总的转接器选择哪一款比较好呢，我想超过45年历史的美国品牌Verbatim威宝旗下的Type-C Hub 魔盒系列转换器还是非常值得一看的。

![](.evernote-content/E9F034A0-C575-4DB4-BA2E-FE6ADD53DD47/250B2C33-35FB-4AEB-B5B0-5DD6B008FFC4.png)

可能在大家的印象中，Verbatim 威宝做的刻录碟非常不错，实际上近半个世纪的发展使威宝成为数据存储媒体行业中最被认可的品牌，紧跟时代步伐，Type-C接口的数据线、转接器一直以来也都有不错的销量和口碑，如今这款Type-C Hub 魔盒系列转换器集合了多种接口，使用设计上也更为方便。

![](.evernote-content/E9F034A0-C575-4DB4-BA2E-FE6ADD53DD47/4176EC73-F973-478B-B66A-FF0C8AF94F77.png)

首先可以看到Type-C Hub魔盒系列转换器的包装非常简单，这也很符合Verbatim 威宝一直以来的务实风格，在包装盒上我们可以大致了解该产品的功能特点，比如提供 USB3.0扩展口、支持4K的HDMI 扩展等等，包装背面则给出了该产品更为详细的参数。

![](.evernote-content/E9F034A0-C575-4DB4-BA2E-FE6ADD53DD47/7AFEE1C7-F65B-4EA2-B181-9F2CF9CED4B5.png)

Type-C Hub 魔盒系列转换器身材相当小巧，重20克，84\*28\*8.4mm的三围与 New Macbook 的轻薄相呼应，转换器采用阳极氧化铝外壳，在其一角印有 Verbatim Logo，而转换器伸出的Type-C接头以及所使用的线材都经过严格筛选和制作，Type-C接头下方水滴式设计可以避免重复插拔对卡尾造成的损害，另一端也对线材采取了保护措施，看得出这款产品对用户还是很负责的。

![](.evernote-content/E9F034A0-C575-4DB4-BA2E-FE6ADD53DD47/0AF9501F-9A40-411E-8D66-D57E2894DE3F.png)

![](.evernote-content/E9F034A0-C575-4DB4-BA2E-FE6ADD53DD47/3D8DA0EC-5B31-4B95-A533-53020AB417EF.png)

该转换器提供了玫瑰金、土豪金、太空会和银白四种颜色，不管是手感还是外观都与New Macbook 以及最新的MacBook Pro等苹果产品保持一致，此外该外壳还具备耐刮耐磨不留指纹等优点。

![](.evernote-content/E9F034A0-C575-4DB4-BA2E-FE6ADD53DD47/0D789E92-3B1D-4B64-8EA2-475106E3D698.png)

![](.evernote-content/E9F034A0-C575-4DB4-BA2E-FE6ADD53DD47/B8D96CDD-D725-4740-8032-D8CCD164444F.png)

Type-C Hub 魔盒系列转换器配备了一个Type-C电源接口、一个Type-C接头、一个 HDMI 接口、两个 USB 3.0 接口，以及一盏工作指示灯，从接口类型扩展来说并不输苹果官方推出的Type-C转接器，而且外观风格上与苹果产品更具一致性。

![](.evernote-content/E9F034A0-C575-4DB4-BA2E-FE6ADD53DD47/7D873164-5F68-4D2E-A29C-0A3DD7B65B82.png)

![](.evernote-content/E9F034A0-C575-4DB4-BA2E-FE6ADD53DD47/5E07E5FA-344C-4A57-AB3D-7A8FA314BFC8.png)

该转换器的使用非常简单，只要将Type-C接头插入New Macbook的Type-C接口即可实现多类型接口扩展，此时工作指示灯亮起。在实际体验中，小编同时将 U 盘、移动硬盘、iPhone、iPad分别两两同时插入，发现所有接入的设备其功能都可以完美使用，由于是 USB 3.0 接口，因此接入支持 USB 3.0 的 U 盘或存储设备时，传输速度都相当快，大体积文件的传输也相当稳定，没有发生文件损坏和无法传输的情况。

![](.evernote-content/E9F034A0-C575-4DB4-BA2E-FE6ADD53DD47/8A751296-DF5E-41E2-B4C7-0BBE48581F92.png)

![](.evernote-content/E9F034A0-C575-4DB4-BA2E-FE6ADD53DD47/948CDC4A-BB7F-41A3-85A6-4E025072C04F.png)

对于 iPhone和其他手机，利用这款转换器也可以实现给手机充电和数据传输，然后小编利用电源适配器和Type-C线接入转换器的Type-C接口进而给 New Macbook 充电，其反应迅速，充电效果稳定，与苹果官方的转换器在功能效果上并没有什么不同，就是温度方面个人感觉略高了一些，但也都能够控制在50度以下，大多数时候就40度左右，当然体感温度的多少还得结合室温来判断。值得一提的是，还可兼容最新的MacBook Pro，无论是充电还是传输都相当稳定。

![](.evernote-content/E9F034A0-C575-4DB4-BA2E-FE6ADD53DD47/4399843C-EA97-46BB-81A1-344C10C2E47C.png)

一款优秀的Type-C转换器一定可以实现所有接口同时工作，即满负荷工作，为此小编也进行了测试。在连接电源的情况下，再用 iPhone、U盘、投影仪接入对应的接口，然后进行数据传输、手机充电、文件投影等工作，发现均能够正常使用，当然了，外接 USB 鼠标、键盘或是具备Type-C接口的手机也都没有问题。当设备以及电源插入转换器之后，咬合度不会太紧也不会太松，插拔体验还是比较舒适的，而转换器接入New Macbook之后也没有松动或是接入困难等情况，可见Type-C Hub 魔盒系列转换器的接头和接口规格和工艺都相当严谨和扎实。

![](.evernote-content/E9F034A0-C575-4DB4-BA2E-FE6ADD53DD47/72D7F7F8-DB1D-45FE-B3AC-F1AAB998BCEC.png)

另外，Type-C Hub 魔盒系列转换器上各个接口之间的距离设置还是很合理的，两个USB 设备接入时，其接头都不会贴合或是接触，而 HDMI 接口设置在转换器的一端也相当明智，做到了各个设备之间的运行互不影响。

其实要说接口类型设置，市面上同样也有同时具备了 USB 3.0、HDMI 的Type-C 转换器，但是从外观设计、材质用料、人性化接口位置设计等方面来说，都有这样那样的缺点，Verbatim 在收集了以往用户反馈之后，推出的这款Type-C Hub 魔盒系列转换器不仅让其外观与苹果产品同色同质，而且还针对薄弱部位做足了保护措施。另外，扩展接口的功能性和位置的合理性都得到了进一步提升，在使用过程中，每一个接口功能都得到了应有的发挥，数据传输、充电稳定安全，并且能够兼容手机、平板、鼠标、键盘等市面上绝大多数数码设备，如果你还在烦恼如何将更多的设备同时接入New MacBook / MacBook Pro，那么我想是时候准备一个Verbatim Type-C Hub 魔盒系列转换器了。
