宝子们😎，今天来教大家把超火的智能助手DeepSeek - R1装进自己电脑哦。不用联网就能随时和AI互动，写文案、聊天、做翻译都不在话下，学生党和上班族都很适用呢🧐。

💻Windows用户图文教程来啦：

1. 下载Ollama（超详细）：打开浏览器搜ollama（无需特殊网络）登录，点右上角Download按钮，按电脑系统版本下载安装包，双击安装包一直点“下一步”就好啦。安装完右下角会有提示框。验证安装成功小技巧：在终端输入【ollama - v】出现版本号就成功了（不会用终端？windows键+R输入cmd回车就行）。

2. 【使用DeepSeek R1】：在终端输入【ollama run deepseek - r1】（默认下载7B参数规格，静静等模型下载进度条加载完）。备注：模型有不同参数可选，像1.5b，7b，8b，14b（适合PC本地且推理能力较强）等，下载对应参数规格的方法是【:加参数量】，比如下载14b模型：【ollama run deepseek - r1:14b】。

3. 【开启AI聊天】：出现【>>>】符号就可以开始啦。试玩示例：“用甄嬛体帮我写请假条”“模仿李佳琦推荐我的化妆水”“帮我想10个情人节礼物创意”。

⚠注意事项：

▫ 模型下载至少20分钟（不支持断点续传，下载时要保持网络连接）。

▫ 7b模型文件约4.7G，14b约9G，要根据磁盘空间选择模型。
