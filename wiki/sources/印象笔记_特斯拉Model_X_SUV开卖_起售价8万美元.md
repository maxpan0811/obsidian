# 特斯拉Model X SUV开卖 起售价8万美元

---

[![](.evernote-content/96601B49-2719-487A-9A30-03170A804BC6/59BBDC38-E55F-4D62-A12F-121E494A7F38.jpg)](http://tech.feng.com/2015-11-24/Tesla-Model-X-SUV-on-sale-for-80000_631072.shtml)

![](.evernote-content/96601B49-2719-487A-9A30-03170A804BC6/59BBDC38-E55F-4D62-A12F-121E494A7F38.jpg)

北京时间11月24日消息，据彭博社报道，特斯拉在周一开始销售其Model X SUV，邀请已预订该车型的客户向公司提供他们的个性化配置。不计入联邦税收抵免或国家退税，Model X的起售价为8万美元。

首批从特斯拉费利蒙工厂生产线下线的Model X，是限量版的创始人系列车型，一般分配给了特斯拉董事会成员和亲密朋友，比如谷歌联合创始人谢尔盖·布林(Sergey Brin)。在创始人系列之后，特斯拉又推出了签名版系列车型。签名版Model X是全功能车型，起售价为13.2万美元，客户需要支付4万美元订金。

随着Model X的上市，特斯拉目前在售两款车型，可以借此进一步扩大自身的吸引力。增加更多车型对于特斯拉来说至关重要，该公司已经获得了大量关注，在电动汽车上的销量也超过了主要汽车制造商，但目前仍未实现盈利。

入门版Model X的起售价为8万美元，较最低价版的Model S贵出5000美元，客户需要支付5000美元订金。它配备70千瓦时电池，一次充电的续航里程为220英里(约合354公里)。

2012年2月，特斯拉在洛杉矶举行的活动上发布了Model X的概念版，原本计划在次年开始销售，随后推迟到了2014年年底。特斯拉的电邮显示，Model X标配双电机，全轮驱动，提供三个版本：70D、90D以及P90D。

70D版本属于入门级车型，最高时速为140英里(约合225公里)；90D版本配备90千瓦时电池，一次充电的续航里程为267英里(约合429公里)，最高时速为155英里(约合249公里)；P90D为高配版，从0加速到时速60英里(约合96公里)不到4秒，入门级70D需要6秒。

不过，8万美元只是Model X的起售价，许多客户很可能会增加其它选择，比如售价2500美元的高保真音响系统。用户在预订Model X时，可以选择5座、6座或7座配置。
