# 嗯，严格意义上来说 Windows 10 已经正式发布一个小时了

---

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/6507A488-53C9-4489-B6FD-EF47FAB70BCA.png)

在新西兰，现在已经是 2015 年 7 月 29 日凌晨一点 —— 这个全球时间过得最快的地方已经在 Windows 10 官方发布日过去一个小时。美国时间现在大致在早上八点到九点之间，根据报道，在此之前微软已经开始为预定了 Windows 10 升级的 Windows 7 和 8.1 用户预加载升级需要的文件。但事实是，目前为止还看不到多少人在讨论 Windows 这个市占率数年来离 100% 没多远的桌面系统大更新。

不过作为一篇新闻文章，还是来谈谈 Windows 10 引入的新功能吧。

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/9D8F140F-60B6-4300-8DD9-2C8B9189ED80.png)

首先是“开始菜单”的回归。用户不再需要像 Windows 8 被强制启动到一个满是功能不完整的 Modern 应用同时功能陌生、导航生硬的 Modern Start 界面；也不用在像 Windows 8.1 之后依然纠结没有一个可以快速访问应用和文件的趁手入口。但是，这个开始菜单依然和 post-WIndows 7 时代的那个不同，它仍然没法让普通用户按照熟悉的多栏、多级菜单中快速定位到任何一个已经安装的应用，单列应用试图查找传统应用依然很不直观，它更突出的使命还是让固守原有习惯的用户尽可能多地看到 Modern 应用们。所幸，Windows 10 内置了一个很方便的全局搜索功能，这个功能可以在未建立索引（在以前的系统内，无论有意无意，用户的文件索引会被数字、企鹅等各种“电脑管家”删除，让搜索变成一个很鸡肋的功能）的情况下提供更快更好的搜索结果，而且可以同时查找应用、文件，和网络关键词。

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/05D2956E-2783-436C-A28D-7600C4E7EB85.png)

虽然在微软即将正式推送给所有升级用户的版本就是 Windows build 10240，而我在全新安装了这个版本的 Surface Pro 3 上至今仍保持了“在未重启资源管理器情况下”低于 30% 成功打开 Cortana 几率，还是应该说说这个微软特别重视的功能。对于中国用户而言，Windows 10 会同时内置小娜和小冰，都是微软的新一代智能语音助理。入口也都一样，就是图上开始菜单右侧的圆圈图标（在任务栏属性里取消勾选“使用小任务栏按钮”后会是一个常驻的输入框）。小娜更注重做“有用”的事，其中包括帮用户追踪天气、股价、关键词新闻、影视剧更新、体育赛事结果、机票酒店信息和重要日程提醒等；而小冰则主要以插科打诨为主。

嗯，这个白纸黑字的“简单” UI 就是正式版 Windows 10 Cortana 的形象。

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/994A27C6-D878-47A7-8B6D-9498194E19C8.png)

接着是全新设计的“控制面板”。再也不是让人头晕目眩的密集小图和晦涩的功能说明，新的设置入口把功能整理成几个分类。把这个窗口收窄到一定程度后，它就会变成和 Windows 10 Mobile 一样的设置菜单样式，不过这并没有 ……

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/A9591C35-3727-46AE-827B-F9449092539C.png)

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/7CAC3519-6618-4FC1-8C8D-5DEB4D783E6E.png)

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/EB753383-B947-4A37-B8FE-366038C490B8.png)

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/A5D0BAD1-CCF9-4491-99A4-49E4F65F9FCC.png)

系统托盘里的电源管理、Wi-Fi 面板、音量控制和输入法列表都被更新成 Modern 设计的版本。（需要提到的一点是，这里可以看到笔者没有使用 Windows 10 官方提供的中文输入法，因为目前为止微软仍未解决官方中文输入法频繁导致第三方应用假死或崩溃的问题）但是：

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/45A7E1C0-D217-47EB-A849-D03D3F0045B1.png)

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/C37D2330-A373-4E26-A0CA-9CCE7D8B2CFA.png)

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/298FCE32-36F9-4879-BC1F-2FCE29FA46B6.png)

这些全数都还存在。Windows 10 的设计呈现出一种非常割裂的状态。表面的一层都改成 Modern 风格的新设计，但内部，包括资源管理器都还是传统的风格。系统托盘的几个系统指示图标尺寸到现在也尚未统一。目前为止还不确定这种风格割裂会不会给普通用户带来迷失感。

不仅如此。

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/941B8A09-6054-49BB-A714-F759E41A9E48.png)

目前为止 Windows 10 的右键菜单样式至少有这么多种。（图片来自 Winbeta 主编 [Zac Bowden](https://twitter.com/ZacB_)）

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/1541F19F-79A9-4445-A63C-251B9715ABD9.png)

操作中心是一项由 Windows Phone 引入 Windows 桌面系统的新功能。用户点击系统托盘音量控制右侧的图标或者由屏幕有边缘向内划就能呼出这个操作中心。这里统一展示了 Modern 应用和系统通知信息，同时还能让用户快速开关常用功能。在支持的应用中，用户可以直接对通知做出回复等操作。

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/3833B1A9-82BF-4E63-94E5-C7227D091339.png)

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/396AEE35-D182-473F-A146-509FE70E77EB.png)

Microsoft Edge 是微软在 Windows 10 中引入，用于替代 Internet Explorer 的新浏览器。它的交互和界面相比 IE 都要简洁得多，但功能，如设置局域网选项、代理信息、安全等级等方面比前任都存在很大不足。发布之初承诺的插件支持至今也没能实现。可取的几点是：Edge 支持从 Chrome、Firefox 等第三方浏览器导入书签，能让用户直接通过触控笔在网页上标注涂鸦并直接分享，还可以使用 Cortana 直接查找页面内某一关键词的信息。兼容性方面，目前为止 Edge 相比 IE 和 Chrome 都要稍差，但网页渲染速率和多线程占用内存后的流畅性都要比 Chrome 明显更理想。

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/67EBFF8D-E5C8-443E-BEE9-809EAAEAEAA1.png)

和 Windows 8.x 一样，Windows 10 内置了一个受官方管控的应用商店。这里可以下载 Win32 （传统桌面）和 Modern 应用。除此之外，微软在 Windows 10 时代提供了快速将 Win32、Android 和 iOS 应用移植为 Modern 应用的工具。目前已知会在 Windows 10 发布后短期内上线适配 Modern 版本的主流应用有有道词典、QQ 和微博等，曾经在由 Terry Myerson 展示的新版微信并没有明确消息。

不过，微软自家的 Skype 已经停止了 Modern 应用，后续更新都转向传统 Win32 程序模式。微软即将推出的内置消息应用同样不是 Modern 版。以及，我们可以提前看一下微软和 Twitter 合作的新版 Windows 10 客户端设计 ……

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/DA1A64E8-3552-46E9-AB40-0F88C4D6C83D.jpg)

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/B2CA1838-22FC-42A7-820E-F108CAFA200E.png)

生产力方面，Windows 10 最值得为人称道的就是极强了多窗口并排功能。用户可以很便捷地通过 Win + 方向键让窗口变为全屏、半屏和四分之一屏尺寸，与此同时 Aero Snap 鼠标手势也得以保留。当用户把一个应用贴靠为半屏时，系统会自动在另一半呈现待选的其它窗口。

![](.evernote-content/EE1139C8-51CB-4950-B155-080304D89AF9/80DD9F98-6DFD-425A-A667-87AA0C42FC71.png)

同样有意义的还有 Windows XP 之后终于再次得到强化的虚拟桌面管理功能。用户可以通过 Ctrl + Win + D 创建新的虚拟桌面，并使用 Ctrl + Win + 方向键来切换。某个虚拟桌面内的任务可以通过上图的界面直接拖拽至另一个桌面中，被关闭虚拟桌面中承载的任务会默认向前合并。

Windows 10 是微软在 Windows 8 大跃进之后的一次紧急修补，也是微软走向跨平台和全平台统一的第一步，但因为短期内的人事变动和转型，这个“正式版”依然处在半成品状态，更不用提迟迟谈不上完善的 Windows 10 Mobile。期待看到微软围绕新的桌面系统，横向拓展以达到他们多次宣称的“十五亿设备”目标，但以现在的进展来看，路还很远。
