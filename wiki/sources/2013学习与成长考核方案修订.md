Dear all，

 

今年公司考核部门KPI方案中干部梯队丰满度做了调整，因此部门考核各位负责人的梯队丰满度也做相应调整，如下表标红的部分。同时，学习与成长占各位KPI的权重调整到15%，如蓝色字体。

 

2013学习与成长考核方案：

|  |  |  |  |  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| **项次** | **考核内容** | **占比** | **考核内容解释** | **数据来源** | **计算方法** | **等　　级** | | | | |
| **卓越5-****10%** | **优良20%-30%** | **中等50%** | **合格10-20%** | **不合格****0-5%** |
| 1 | **干部梯队丰满度** | **5%** | 部门内部干部是否在各个级别上科学分布，梯度完整性以及各岗位储备干部情况。（根据公司考核部门的维度修订。公司按10人以上以下区分，部门按5人以上以下区分） | 由HR组根据季末各组的各职级的人数计算。 | 1）分5人以下部门和5人以上(含5)部门。5人以上（含5）部门：1<=每下级/上级人数比例<2，计缺陷分0.5分；小于1以下计缺陷分1分。  4）5人以下部门:最高不超过4分 | 缺陷分<=0 | 缺陷分<=1.5 | 缺陷分<=3.5 | 缺陷分<=4.5 | 缺陷分>4.5分 |
| 2 | **优秀人员流失率** | **5%** | 优秀员工（指3级及以上员工）流失人数除以该组总人数，以此记分（同公司考核部门的维度） | 由HR组计算各组季度流失率（流失优秀员工数/组总人数），按人数及比例考核；有3级以上人才输送给其他组或部门，每输出1人加0.5分。 | 季度流失率＝流失优秀员工数/组人数 组人数＝（期初数+期末数）/2 | （有人才输出才有可能卓越，每输出1人加0.5分） | 无流失 | 季流失率<=15％ | 15%<季流失率<30% | 季流失率>=30% |
| 3 | **人均受训课时** | **3%** | 全组人员参加培训的人均课时（包括主管本人），（同公司考核部门的维度）。计入培训课时的培训是指：  1、公司层面组织的培训。  2、部门层面组织的培训（包括上海旅游业务部或者各分公司旅游业务组织的培训）  3、上海旅游业务各组的组内培训（不包括新人八周培训）  4、读书会也算培训的一种形式，计入培训课时。 | 公司层面组织的培训：根据培训系统中的数据； 部门层面组织培训：根据培训系统中的数据，或者分公司统计的数据； 总部各组组内培训：由各组的培训联络人记录并提供给部门HR | 人均受训课时＝全组人员的总培训课时/组人数 组人数＝（期初数+期末数）/2 | ＞＝12小时， | ＞＝8且   　＜12小时 | ＞＝6且   　＜8小时 | ＞＝3且   　＜6小时 | 小于3小时 |
| 4 | **授课课时** | **2%** | 1、上海各主管：主管本人在部门层面和组内培训课时（包括由部门HR邀课给新进经理人或者给外部门人员的培训）  2、分公司各主管：主管本人在分公司或者旅游业务部门层面的培训。  3、部门层面的读书会导读老师的课时折半计算。 | 上海的由部门HR统计授课课时； 分公司的由各分公司记录并提供给部门HR | 授课课时汇总。 | ＞＝6小时， | ＞＝4且   　＜6小时 | ＞＝3且   　＜4小时 | ＞＝2且   　＜3小时 | 小于2小时 |
