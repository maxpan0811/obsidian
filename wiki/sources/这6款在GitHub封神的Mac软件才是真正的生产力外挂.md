哈喽朋友们，我是小猫。

这会儿是周六晚上六点半，刚合上电脑，趁着晚饭前的空档跟你们聊五毛钱的。

说实在的，用 Mac 搞数字化业务这么多年，我真的是被 App Store 里各种花式“订阅制”给恶心到了。一个小小的状态栏整理工具，敢收你一年大几十；一个简单的剪切板增强，又要按月扣费。真把咱们当满地乱窜的韭菜割呢？

作为天天在技术堆里打滚的产品人，我现在的软件库基本全靠 GitHub 这个开源祖师爷兜底。今天不拽那些假大空的词，直接把我压箱底的 6 款万星（Star）免费神器掏出来。装上它们，你的 Mac 才配叫真正的生产力。

![](attachments/b558b12a7be475ac.png)

# 01. Ice：别花冤枉钱买 Bartender 了

第一个必须干掉的，就是右上角那个乱成一锅粥的菜单栏。

平时我后台挂着 RSS 监控海外汽车零售数据，又跑着各种自动化的 webhook，菜单栏那点地方早爆了。很多人去花钱买 Bartender，其实大可不必，直接上Ice。

这玩意儿完全原生开发，不仅免费，逻辑还极度舒适。你直接把图标分三档：常用的留着，偶尔用的（比如蓝牙）设置为鼠标划过去才显示，八百年不用的直接塞进永久隐藏区。一分钱不花，立马把苹果没做好的极简美学给你找回来。

下载地址：https://github.com/jordanbaird/Ice

![](attachments/c90c69b96b5290ad.png)

# 02. AltTab：拯救反人类的窗口切换

苹果原生的Command + Tab切换器，不知道你们受不受得了，反正我是快被折磨疯了。它只认“软件”，不认“窗口”！

有时候我在搞多工厂的 OMS 订单调度逻辑，开了好几个浏览器窗口比对数据，切的时候全靠盲猜。AltTab直接把 Windows 上最好用的窗口实时预览，原封不动给你搬过来了。快捷键一按，屏幕上直接弹出所有窗口的大图缩略图。哪个是数据看板，哪个是需求文档，一目了然。对于多线并行的打工人来说，这就叫系统级的续命神药。

下载地址：https://github.com/lwouis/alt-tab-macos

![](attachments/fd3044ad81260d98.png)

# 03. Pearcleaner：这才是物理级的斩草除根

别天真地以为把 App 拖进废纸篓就算卸载干净了。那些藏在系统深处的配置文件，能悄悄把你寸土寸金的硬盘吃掉好几个 G。

别去下那些臃肿还带广告的清理大师，去装个开源的Pearcleaner。最近我在本地频繁测试各种开源的 AI 自动化部署工具（比如 OpenClaw），装了删、删了装。只要我把软件往废纸篓一扔，Pearcleaner 会瞬间跳出来，精准揪出所有隐藏的依赖文件、甚至冗余的多语言包，一键物理超度。你的 Mac 永远能保持刚开机时的清爽。

下载地址：https://github.com/alienator88/Pearcleaner

![](attachments/de15da6ac5f5dd91.gif)

# 04. KeyCastr：暗箱操作的“透明外挂”

这个是给内容创作者和经常需要线上视频开会的朋友必装的。

平时我在 X 和公众号上发一些 AI 部署的实操视频，观众老是留言问“小猫你刚刚按了什么键弹出来的”。用上KeyCastr，你在键盘上敲的每一个快捷组合键（比如Cmd+Shift+V），都会像电影字幕一样在屏幕角落实时弹出来。不管是给业务团队录制内部 App 的培训视频，还是做技术演示，这种把暗箱操作透明化的外挂，能省下你无数的沟通口水。

下载地址：https://github.com/keycastr/keycastr

![](attachments/f7a2be909568d310.png)

# 05. LosslessCut：一秒都不渲染的“重构魔法”

剪视频这事儿，很多时候真不需要杀鸡用牛刀。

比如录了一段十几分钟的系统操作演示，我只是想掐头去尾截取中间那一段。要是打开剪辑软件，建工程、拖素材，最后还要等它像个大爷一样重新渲染导出，人都要急碎了。

LosslessCut简直是神作。它从底层就不进行视频重新编码。你切好不要的片段，点个保存，几秒钟完事，画质绝对 100% 零损耗。这种不走传统的重型剪辑，纯靠“无损数据搬运”带来的极速爽感，谁用谁懂。

下载地址：https://github.com/mifi/lossless-cut

![](attachments/3d10911a1c28a72b.png)

# 06. Handbrake：转码界深藏功与名的老炮儿

最后压轴的，是视频转码界祖师爷级别的Handbrake。

经常录一些高清的实操物料，动不动就好几个 G，放进微信根本发不出去。把它扔进 Handbrake 里，批量排个队，在肉眼几乎看不出画质损耗的情况下，它能把体积硬生生压到原来的十分之一。连最新的 AV1 编码和电脑的硬件加速都能吃透。开源界十几年磨出来的一把快刀，极其纯粹，没有任何短板。

下载地址：https://github.com/HandBrake/HandBrake

真正的极客精神，从来不是比谁买的付费软件多，而是懂不懂用最硬核、最纯粹的底层工具去重构自己的工作流。去 GitHub 搜一圈这几个名字，把那些常年吃灰的订阅全停了吧。
