## 一句话摘要

Claude Code 定义了当前 AI 编程工具的上限，但它不对中国开发者开放。DeepSeek 公开招兵买马组建 Harness 团队，从零构建对标 Claude Code 的代码智能体产品。

## 核心信息

- **动作**：DeepSeek 官网、小红书、X 同步招聘「Agent Harness 产品经理」和「Agent Harness 研发」
- **目标**：从零构建对标 Claude Code 的代码智能体产品
- **逻辑**：模型之外的 Harness 层才是真护城河——谁控制了开发工具链，谁就控制了 AI 编程生态
