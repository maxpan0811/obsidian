老警察与新警察

>> 上班刚一周,刚穿上警服的小五决定犒劳犒劳自己,到剧院看电影。买票的队伍排得
长
>> 长的,小五舒口气,排到最后。
>> 〉 　　“新警察吧？”旁边一个人问。
>> 〉 　　小五纳闷：“你咋知道？
>> 〉 　　“咳,老警察哪有排队买票的。”
>> 〉 　　“哦。”小五明白了,径直走到售票口前,递上钱说：“我买一张票。”
>> 〉 　　“新警察吧？”窗口里的人笑了。
>> 〉 　　“你咋知道？”
>> 〉 　　“老警察哪有掏钱买票的,你直接进吧,没人敢拦。”
>> 〉 　　“哦。”小五又长了见识,一试,果然没人拦。
>> 〉 　　进了剧场,小五到楼下随便找了个位子坐下。屁股还没坐稳,旁边就有人
问：
>> “新警察吧？”
