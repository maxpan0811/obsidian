# OS X新手技巧：打造个性签名并插入到PDF

---

[![](.evernote-content/527FE32A-5E81-4AAC-9E99-787C0531BAE2/B2C37172-A6B8-45BA-B0F9-BC6133FC847D.jpg)](http://www.feng.com/iPhone/news/2016-03-14/OS-X-new-skills-to-build-character-signature-and-inserted-into-the-PDF_641281.shtml)

　威锋网讯，OS X 其实隐藏着很多神秘且实用的功能，即便你是一个使用 Mac 多年的用户，可能也不是那么容易发现这些功能。今天要介绍的是预览应用的一个功能，它可以让用户增加签名到 Mac，然后插入到 PDF 文档中。

![](.evernote-content/527FE32A-5E81-4AAC-9E99-787C0531BAE2/B2C37172-A6B8-45BA-B0F9-BC6133FC847D.jpg)

本文旨在为新手提供更多使用 OS X 的便捷功能，资深用户可略过。在开始之前，你需要准备一支笔和一张白纸。

**制作签名**

1.拿出准备好的笔和白纸，签下你的大名

2.运行预览应用，从菜单中找到工具-注解-签名-管理签名，并打开

![](.evernote-content/527FE32A-5E81-4AAC-9E99-787C0531BAE2/1FECCF11-EBB3-4961-ADE9-675366A3E69A.jpg)

3.打开“管理签名”的窗口你会发现两个选项：触控板和摄像头。你可以选择使用触控板来输入签名，不过本文建议使用摄像头来获得更好的效果。点击“摄像头”后，将你刚刚在白纸写好的签名放置在 iSight 摄像头前。确保房间有充足的光线进入摄像头，以免摄像头无法顺利读取签名。如果光线不充足的话，可以试试增加屏幕的亮度。

![](.evernote-content/527FE32A-5E81-4AAC-9E99-787C0531BAE2/5F2788FB-B4B3-4AC7-AB9C-2A40611C1209.jpg)

4.签名扫描后便会出现在屏幕上，满意的话点击“完成”即可。

**插入签名到PDF**

现在你的签名就已经添加到 Mac 上了，要将签名插入到已经在预览中打开的 PDF，点击工具箱（下图）。

![](.evernote-content/527FE32A-5E81-4AAC-9E99-787C0531BAE2/0378BB91-B494-4262-BB06-0E922DD7CC03.png)

然后从菜单栏中点击“签名”按钮。系统就会将电脑中的签名列出，此时选择要插入的签名即可，你还可以对签名的大小和位置进行调整。

![](.evernote-content/527FE32A-5E81-4AAC-9E99-787C0531BAE2/93A45EC2-73E7-4518-B8D5-F7D6ECEA3827.jpg)
