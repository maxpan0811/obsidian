这8个技能是我最近用下来最实用的，

在这里分享给你们，直接复制命令就能装：

1️⃣ agent-browser — 让龙虾帮你操作网页

想爬数据不用写代码，直接说："去某某网站，把列表抓下来"。龙虾自己开浏览器、操作页面、返回结构化数据。

npx skills add vercel-labs/agent-browser@agent-browser

2️⃣bb-browser — 免登录神器

以前让龙虾去小红书发帖，得先扫码登录，过期了还要重新扫。这个技能直接调用你本地浏览器的状态——你已经登录过了，龙虾直接用你的身份操作。

npx skills add epiral/bb-browser@bb-browser

3️⃣OpenCLI — 18个平台一句话操作

评论、关注、搜索这些手动操作，全部做成命令行。覆盖B站、知乎、小红书、X、Reddit、GitHub等18个平台。

npx skills add joeseesun/opencli-skill@opencli

4️⃣skill-vetter — 装技能前先安检

技能市场越来越丰富，风险也在增加。这个技能会在安装前扫描：有没有网络外发、有没有读取敏感环境变量、有没有可疑代码。

npx skills add useai-pro/openclaw-skills-security@skill-vetter

5️⃣self-improving-agent — 让龙虾能自我进化

自动记录你纠正过的错误、踩过的坑、反复提到的工作习惯，整理成知识卡片，后续任务中主动调用。

npx skills add charon-fan/agent-playbook@self-improving-agent

- Skill名称: self-improving-agent-skill
- 来源: ClawHub (initail维护)
- 版本: 0.2.0
- 安装时间: 2026-04-20 23:25
- 安装位置: ~/.openclaw/workspace/skills/self-improving-agent-skill/

### 🌟 核心功能：

#### 🤖 自我改进系统

- 持续学习: 从每次交互中学习经验
- 多记忆架构: 语义记忆 + 情景记忆 + 工作记忆
- 自我纠正: 自动检测和修复错误
- 自我验证: 定期验证skill准确性

#### 🔒 安全特性

- 用户确认门: 所有skill修改都需要用户明确批准
- 人在循环中: 用户参与改进决策过程
- 可追溯更改: 所有更改都有来源归属
- 置信度跟踪: 测量模式可靠性随时间变化

#### 📊 研究基础

基于最新AI研究设计：

- SimpleMem: 高效终身记忆系统
- Multi-Memory Survey: 语义+情景记忆架构
- Lifelong Learning: 连续任务流学习
- Evo-Memory: 测试时终身学习

### 🎯 触发时机：

#### 自动触发：

1. 任务完成时 - 提取模式，提出skill更新
2. 发生错误时 - 捕获错误上下文，触发自我纠正
3. 会话结束时 - 整合工作记忆到长期记忆

#### 手动触发：

- 用户说"自我进化"、"self-improve"、"从经验中学习 "
- 用户说"分析今天的经验"、"总结教训"、"总结经验"
- 用户要求改进特定skill或工作流

6️⃣lossless-claw — 给龙虾长记忆

OpenClaw作者推荐的方案。对话持久化存到数据库，旧对话打包成树状摘要，控制Token消耗，龙虾不会再"失忆"。

npx skills add Engineering/lossless-claw@lossless-claw

7️⃣control-center — 查看龙虾状态

养了七八只龙虾，不知道它们死没死。这个面板能看每个任务消耗多少Token、每只龙虾健不健康、有没有卡住，还能查看和修改记忆人设。

npx skills add aradotso/trending-skills@openclaw-control-center

8️⃣openclaw-backup — 养龙虾的保险

定时备份整个配置文件和记忆库。不小心把龙虾养死了也能恢复，配置、记忆、历史对话一键回档。

npx skills add

theagentservice/skills@openclaw-backup
