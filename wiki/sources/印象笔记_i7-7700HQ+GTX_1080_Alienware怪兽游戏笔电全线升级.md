# i7-7700HQ+GTX 1080 Alienware怪兽游戏笔电全线升级

---

![](.evernote-content/0AFB1457-41EF-49F9-97C1-7BF7E2AB0395/D88FDD04-F998-4F33-ADFF-9F532499918F.png)

在游戏笔记本电脑领域，惠普似乎进展得并不顺利，但戴尔的情况却正好相反，因为戴尔拥有一条长期备受玩家关注的高端游戏产品线 Alienware。在 CES 2017 召开之前，戴尔正式更新旗下 Alienware 品牌的游戏笔电，重点升级了处理器和独立显卡，使其足以应对 4K 和 VR 主流趋势。

具体来说，戴尔 Alienware 产品线三个尺寸的游戏笔记本电脑都获得了更新，包括 Alienware 13、15 和 17，所有机型都换上了英特尔最新第七代基于 Kaby Lake 微架构的酷睿处理器，并且还搭配了 Nvidia 最新 Pascal 架构的独立显卡，甚至不只是 GeForce GTX 1050 或 GTX 1050 Ti，还包括 GTX 1060、GTX 1070 和 GTX 1080。

![](.evernote-content/0AFB1457-41EF-49F9-97C1-7BF7E2AB0395/8F2AFB0A-6AA7-492D-B874-17F5B20F50B9.png)

戴尔表示，2017 款 Alienware 游戏笔记本电脑，更新了工业设计，其外壳运用到了镁合金、钢和铜等元素。笔记本电脑的底部的散热设计也经过了改良，增加通风量，使得整体散热更高效，更多热量从背部释放。另外，Alienware TactX 键盘区也重新设计，减少烦人的指纹收集情况。

**Alienware 13**

![](.evernote-content/0AFB1457-41EF-49F9-97C1-7BF7E2AB0395/E531B12F-BE7D-40CE-A49B-9F03C3C2C2FE.png)

13.3 英寸的 Alienware 13 提供了多种型号，屏幕最低分辨率 1366×768，最高可选 2560×1440 像素的 OLED 显示屏。处理器有两种档次，分别是 Core i5-7300HQ 和 Core i7-7700HQ。内存最高可选 32GB DDR4 2400MHz。显卡选择范围有五档，最基本的是使用核显 HD Graphics 630，独显最低 GTX 1050 2GB，最高配可选 GTX 1060 6GB。

![](.evernote-content/0AFB1457-41EF-49F9-97C1-7BF7E2AB0395/3789FF35-DDA1-4679-AC81-719C67BC5ADF.png)

储存范围可选同样非常广泛，最高可选 2TB 的 PCIe SSD。接口方面提供有 1 个千兆网口、2 个 USB 3.0 标准的 USB Type-C 接口、1 个 Thunderbolt 规范的 USB Type-C 接口、1 个 HDMI 2.0 接口、1 个 mini DisplayPort 1.2 接口。机身内置杀手网卡和蓝牙 4.1，标配 76 瓦时的电池和 180W 功耗的电源适配器，提供支持外红检测的摄像头和背光键盘，售价从 999 美元起步。

**Alienware 15**

![](.evernote-content/0AFB1457-41EF-49F9-97C1-7BF7E2AB0395/2FE480AD-0215-4184-859D-9991FA24FBBA.png)

与 Alienware 13 不同的是，15.6 英寸的 Alienware 15 屏幕最低从 1080p 起步，并且可选 3840×2160 像素的 IGZO 屏幕。处理器可选项增加到四款，最高配为 Core i7-7700HQ，内存最高 32GB，储存最高除了 2TB 的 PCIe SS 之外还可再增加 1TB 的 7200 转 HDD 硬盘。可以塞进那么多硬盘主要得益于更大的机身，并且因此还额外增加了 5 个用户可编程的按键。

![](.evernote-content/0AFB1457-41EF-49F9-97C1-7BF7E2AB0395/A0F1240C-D697-4866-B67A-0DE0D398569E.png)

至于显卡，最低选项也是核显，新增 GTX 1060 6GB、GTX 1070 8GB和 RX 470 8GB三个选项，性能上明显比 Alienware 13 提升一个等级。接口基本上与 13.3 英寸款相同，不过 Alienware 15 可以选择标配电池的容量和电源适配器的功率，电池有 68 Wh 或 99Wh 两种，电源适配器可选 180W 或 240 W，售价从 1199 美元起步。

![](.evernote-content/0AFB1457-41EF-49F9-97C1-7BF7E2AB0395/316008CC-5313-4188-86B7-864B04BC19AD.png)

**Alienware 17**

![](.evernote-content/0AFB1457-41EF-49F9-97C1-7BF7E2AB0395/FEDCFC99-DC70-443B-8B17-2649CFA40A74.png)

17.3 英寸的 Alienware 17 是三款机型中最为野兽派的一款，如果选择最高配置的话，屏幕最高为 340× 2160 分辨率的 IPS 屏幕，支持 Tobii 眼部追踪解决方案，处理器为英特尔 Core i7-7820HK，内存最高 32GB DDR4 2400，储存最高 2TV PCIe SSD + 1TB HDD，显卡相比 Alienware 15.6 多了一个 GTX 1080 8GB 的选择。

![](.evernote-content/0AFB1457-41EF-49F9-97C1-7BF7E2AB0395/C6DC6D73-3D4D-4AA8-840D-EAAB4E827502.png)

另外，Alienware 17 的扬声器从 2.0 升级到了 2.1，提供更出众的低音效果，电源适配器增加 330W 选项。根据戴尔官方报价，Alienware 17 的售价也是 1199 美元起步。

![](.evernote-content/0AFB1457-41EF-49F9-97C1-7BF7E2AB0395/3FB31433-B0F7-4DB7-8F21-8AE6210584E5.png)
