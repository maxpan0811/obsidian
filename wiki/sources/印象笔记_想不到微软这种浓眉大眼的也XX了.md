# 想不到微软这种浓眉大眼的也XX了

---

​想不到微软这种浓眉大眼的也XX了
=================

原创任易任易

话说国内的主要互联网APP，包括微信、微博、抖音都开始显示IP所属省份，这是个好事，因为可以显著提高1450网军的运营成本，毕竟成本的些微提升，都是巨大的阻碍，如同古代的长城一样。

然后我突然发现，微软这种浓眉大眼的标准外企，也开始XX了，因为他们开始提供免费的VPN服务器，而且还是跟Cloudfare这种CDN服务器合作，可以有效的隐藏留言者的IP。

这TMD应该不是故意的。国内互联网公司刚刚宣布「我会显示文章来源」，微信甚至刚刚在4月29号上线评论和公众号显示IP来源，然后同样是4月29日，微软就在台湾地区宣布Edge浏览器提供免费VPN代理。

这绝对不是故意的。我相信微软绝对没有恶意，因为微软在中国大陆有巨大的知识产权收入，也有公有云业务，作为一家企业，完全没必要跟中国大陆的政策对着干。

什么是VPN代理？我们正常访问互联网，是在网址栏输入域名，例如www.tiktok.com经过DNS服务器解析到IP，然后终端获得相关数据，就可以看外国小姐姐了；

但我们作为大陆IP，无法直接访问tiktok，那就需要架设一个国外代理，我们所有的网络请求，都会发送到国外代理上，然后通过国外代理的IP地址，去访问tiktok，这样就能无障碍的看外国小姐姐了（具体操作更复杂，我仅介绍原理）。

然而，架设国外代理是需要成本的，而且用户体验非常差。以一台达拉斯的VPS代理为例，2M带宽，独立IP一年的用户使用成本在300元左右，而这个IP还很有可能被互联网公司标识为不正常流量，就像下图这样，每次用Google搜索，都需要先进行一次人机识别。

![](.evernote-content/34292ACB-94A7-4B95-87AE-D5DC0882AA68/B6219845-307F-4090-8208-2C613380F57C.png)

但是一个IP地址，能不能让几百人同时使用呢？原则上可以，因为不同的终端（PC、浏览器、手机、平板）会有不同的Cookie；但是互联网公司可以非常轻松的发现某一个IP上登录了几百个用户（User），直接暴露了。

比如推特在2020年，也就是红脖子围攻国会山的时候，就封禁了大量的风险IP，原因就在于这些IP都是代理IP，同一个IP登录了几百个推特账号。

这个从技术上太容易识别了，只要一个数据库语句 Select user\_id, count(\*) from user group by user.ip 就可以直接查询出来代理IP了，接下来的工作就是封号，推特连美国总统的账号都能封，封一批使用代理IP的账号有什么难的？

推特能干的事，中国互联网公司像微博、腾讯、百度自然也会干。现在只是显示IP来源，相对推特勒来说，真的是太讲武德了。

但是显示IP来源的事，这对于1450网军同样是一个阻碍。因为1450网军需要翻越GFW，并且把自己伪装成一个中国大陆IP；

那就会直接面对二选一的困境：要么花费一年50-200元左右的成本架设代理服务器（可以视为一人一管的核酸检测，但需要一定的技术背景），要么跟几百人共用一个代理IP地址（可以视为一百人一管的核酸检测，成本显著下降）。

但是一百人一管核酸检测的话，如果有一个阳性，那么其他99人都会被视为密接，直接被管控起来（临时封号、发帖校验等），暴露一个就等于暴露一群。

一人一管的核酸检测，优点是不会被他人影响，缺点是成本比混检贵十倍以上。

这就必须提到一条了，如果想低成本发动广大无技术背景的4V群众时，一定要尽量把流程简单化；购买一个VPS并且架设一个通往大陆的VPN，虽然很简单，但是一大批1450网军同样是不会的。

这就是**中国古代长城的价值**——虽然大部分长城只有一米高，一个人很容易翻越过去——但是一万名骑兵想要带着战马一起翻越长城，就变的非常困难。

毕竟游牧民族进攻中原之后，想带着财帛女子回到草原的时候，区区一米高的长城，就会变成一道鸿沟；后面的追兵尾随不舍，而一万多人带着战马和战利品翻越城墙还需要大量时间。

微软提供免费VPN的行为，就像是直接给拱卫中原的长城上开了几万个缺口，让游牧民族可以毫无压力的长驱直入，然后从容遁走。

微软当然可以说他是商业行为，不是为了给1450网军提供便利；但是，**我信你个鬼**。

中文互联网的评论区，这才刚刚清朗几天啊？想不到浓眉大眼的微软，又要变相帮助1450网军隐藏IP来源？

我虽然不知道**微软图的是啥**？但是制造舆论，从国家政策层面阻止微软这种行为就好了。

顺便评价一下王思聪被封号的事。

粉丝过五百万的人都应该有一种公众人物的自觉，就是务必把自己的ID当做一个大型媒体来自我约束。

像方方就不懂这个道理，自己在2020年期间已经比大部分媒体影响力都高了，还当自己是个自然人，连造谣的内容都敢对几百万人瞎说（殡仪馆满地手机）。而马化腾、马云、刘强东都是先后认识到了这一点，减少了直接面向公众的发言，以便于规避风险。

公众人物都提着双刃剑：实力与责任共存，福利与义务共存，公众人物的一呼百应与高道德要求共存，公众人物无隐私与个人隐私共存，永远没法只吃肉不挨打。

王思聪如果只是想以一个自然人而不是公众人物的角色说话，大可以注册一个小号，爱说什么说什么，who care？

但是王思聪用一个4000万粉丝的ID进行发言，而且言论直接与动态清零方针相左，有极强的煽动性，这是一个大媒体该做的事情么？？？

所以王思聪发而不中其节，才有今日之咎。

扯远了，微软正式面向台湾地区提供免费VPN之日，估计就是国家WXB出手之时。
