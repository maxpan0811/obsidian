appsolution

专注于推荐新酷精华应用。主打 iOS/Android 平台，是国内最早、用户最多的 App 推荐认证公众号。支持自定义菜单及机器人查询。

![](.evernote-content/3B1782C8-7F80-40CB-BE3C-176FB556D422/B03C6383-21C8-4968-A133-DEBFB4C831F5.png)

前面几周分别给大家介绍了我如何使用 OmniFocus 和 Alfred 来提升日常的工作效率。今天再来给大家介绍另一个在我日常工作中必不可少的效率工具 —— PopClip。

![](.evernote-content/3B1782C8-7F80-40CB-BE3C-176FB556D422/0E611C11-F99A-4DA4-86C4-13FCC8B3DBFF.jpg)

看上图大家应该发现这是一个划词工具。当初 iOS 出现划词是为了解决对文本扩展操作的需求，而 PopClip 在 OS X 上对划词这个功能做了极大增强，依附于各类小插件，我们能获得更多强大的功能来提升我们的操作效率。

PopClip 的核心其实在于扩展，上线快三年官方已经累计 130 多个常用的扩展。这还不包含 Github 上那些非官方的扩展们。接下来以我常用的一些扩展来给大家介绍一下 PopClip。

注：PopClip 可以在 OS X 下几乎所有应用下划词使用，以下仅以 Mou 为例。

**淘宝搜索**

当初在 Alfred 自定义搜索那篇也介绍过淘宝搜索，我更多用 Alfred 来输入想要搜索的商品。而当我逛论坛的时候发现有兴趣的商品，直接鼠标划词点击图标也能直接进入淘宝搜索结果页。

![](.evernote-content/3B1782C8-7F80-40CB-BE3C-176FB556D422/51EDE48E-4096-4C3C-AF16-620BD6F63E78.jpg)

**AppShopper搜索**

写专栏难免要经常去查询一下 app 的价格走势图，划词直接进入搜索结果页。这个估计是我使用次数仅次于 Google 搜索的扩展了。

![](.evernote-content/3B1782C8-7F80-40CB-BE3C-176FB556D422/97311043-0004-46BA-A37F-EBC45D97B730.jpg)

**Google 搜索**

这个就不用过多介绍了，直接 Google 搜索关键词。

![](.evernote-content/3B1782C8-7F80-40CB-BE3C-176FB556D422/97311043-0004-46BA-A37F-EBC45D97B730.jpg)

**Translate Tab**

划词直接调用 Translate Tab 进行翻译。

![](.evernote-content/3B1782C8-7F80-40CB-BE3C-176FB556D422/14C7BB51-497A-4212-B9F9-6340E2D8391F.jpg)

**Dash Search**

对代码划词调用 Dash 查看文档，这个工程师们应该会喜欢。

![](.evernote-content/3B1782C8-7F80-40CB-BE3C-176FB556D422/81A9CA0D-083F-494C-A1E2-FEB3D7C6B3F1.jpg)

**Alfred**

划词同样可以激活 Alfred，我主要用它在邮件中划词查找本机文件。

![](.evernote-content/3B1782C8-7F80-40CB-BE3C-176FB556D422/85C7553D-47A9-4137-A040-F244352BCFEC.jpg)

**OmniFocus**

划词调用 action 添加面板。

![](.evernote-content/3B1782C8-7F80-40CB-BE3C-176FB556D422/D955C28A-A7B3-4F20-97C4-E6BC7917193A.jpg)

**直角引号**

划词后为其添加直角引号，这个对经常码字的人会非常实用。

![](.evernote-content/3B1782C8-7F80-40CB-BE3C-176FB556D422/1F7249E0-7999-4F6B-A718-4C460A28E30F.jpg)

**Google 地图搜索**

对地址划词后调用地图搜索。

![](.evernote-content/3B1782C8-7F80-40CB-BE3C-176FB556D422/74D2A55B-890E-453D-BA82-F66859D89D23.jpg)

**Fantastical 添加**

划词添加到日历。

![](.evernote-content/3B1782C8-7F80-40CB-BE3C-176FB556D422/7CD68B08-EFF1-409C-8946-1316E470DB28.jpg)

如果想要发现更多有用的扩展，你有三个办法：

1. 官方还有很多不错的扩展，大家可以在下面的地址中去找寻适合自己：https://pilotmoon.com/popclip/extensions/
2. 还有很多优秀的扩展并不在官网，大家可以直接去搜索「Popclip Extension」。
3. 用扩展制作工具 PopMaker 自行制作。

关于 PopMaker，简单介绍一下。它主要的功能是创建一些基于自定义搜索的 PopClip 扩展，其功能和 Alfred 的自定义搜索基本相似。

如果你还记得上次在 Alfred 之自定义搜索中给大家介绍的方法，获取网站搜索 URL 之后添加到 PopMaker 中再导出安装即可。

![](.evernote-content/3B1782C8-7F80-40CB-BE3C-176FB556D422/931CC89C-F024-4077-91E0-A9871CB1451A.jpg)

Alfred 和 PopClip，是我日常操作 Mac 最重要的两个效率大杀器。所有基于信息浏览的操作我都会交给 PopClip 来搞定。这款应用的售价 4.99 美元，时有降价促销，感兴趣的朋友可以关注。

关于我在 Mac 上的效率篇基本已经结束了，不知道对大家是否有帮助。如果你有什么想要了解的，欢迎大家给 A 君直接回复。我会根据大家的需求来确定之后的选题。
