在 Obsidian 终于可以直接**浏览网页**和**剪藏**了！

Obsidian 新版本更新了 web viewer 核心插件，安装或者更新 1.8.4 版本，即可以在「设置」-> 「核心插件」 -> 「web viewer」开启该插件。

![](attachments/617571eb48fec010.png)

开启该插件后，你将会得到这些功能：

1. 网页浏览：web viewer 的核心功能。
2. 网址收藏：和「书签」插件联动。
3. 剪藏全文：没错，可以直接在 obsidian 中边看边剪藏了！

下文将逐渐给大家详细介绍这些功能。

## 浏览网页

开启插件后，我们新建任意一个标签页，都会多车了一个「open web viewer」的新命令，当然，「cmd/ctrl + p」呼出「命令面板」，也会查到关于 web view 的相关命令。

![](attachments/181bff86857c774c.png)![](attachments/63ed046fb7427e8c.png)

打开一个 web viewer 页面后，出现的就是经典的浏览器的布局，我们在地址栏中输入网页地址，就可以直接浏览了。

![](attachments/74bccc6cfc872e34.png)

另外，自己笔记文档中的外部链接，未来都会直接在在 web viewer 标签页中被打开，这非常方便追溯原文，让自己的思绪延续下去。

web viewer 内置了阅读模式，点击右上角的眼睛图标，就能进入阅读模式，我拿知乎的问答试了一下，真的是清理的干干净净，perfect！这下 obsidian 也能变成一个干干净净的阅读器了。

![](attachments/83cd03413ca05f31.png)

## 收藏网页

网址可以直接通过「收藏」功能，直接添加到书签面板中。

![](attachments/bd8fe79aabd3c774.png)

我还优化了操作的便捷性，通过「commander」插件，把「收藏」这个命令直接添加到页首这个位置。

![](attachments/3cf7abf300977330.png)

在浏览网页的时候，就能直接点击收藏，保存好的资料出处，或者 read it later。

![](attachments/659c92b01910ab12.png)

## 剪藏网页

web viewer 的重磅级功能来了，当我们阅读网页时，觉得这个资料非常值得保存，我们可以用 web viewer 内置的剪藏功能，使用剪藏前，建议先进入阅读模式，清理无关重要的画面。

然后，点击页面右上角的「···」，然后点击「save to vault」，就能直接将当前浏览的内容保存为本地的!笔记了！

![](attachments/7de18155ac363e5d.png)

剪藏笔记的文件夹可以在设置中，找到 web viewer 这一个项目，然后自定义保存的位置。

![](attachments/73d77c28ea2adca8.png)

## 总结

更新了 web viewer 后，obsidian 完善了它整个系统中重要的一环，我们通过 web view 浏览资讯，然后捕捉自己感兴趣的东西，然后在本地进行推演和迭代，然后还可以通过 web viewer 去发布，全程在一个软件中完成所有流程。

通过 web viewer，我们还可以集成各种工具，例如看板、思维导图、读书网站、网盘、AI 问答等等。

只能说，obsidian 越来越让人惊喜！
