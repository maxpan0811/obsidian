# macOS 有哪些冷门但是一但发现就无法自拔的软件？

> ✍️ 张翼ZYi · 📅 2026-06-27 · [原文](https://www.zhihu.com/question/35050387/answer/1953762598668444186)

---

**1、[Cap](https://zhida.zhihu.com/search?content_id=748491056&amp;content_type=Answer&amp;match_order=1&amp;q=Cap&amp;zhida_source=entity)（好用的录屏软件）**

一个录屏软件，开源、免费。

如果你有演示的时候，需要在鼠标位置添加缩放的功能，一定要试试 Cap。

这个一定是 Screen Studio 的平替。

**2、[LocalSend](https://zhida.zhihu.com/search?content_id=748491056&amp;content_type=Answer&amp;match_order=1&amp;q=LocalSend&amp;zhida_source=entity)**

局域网文件互传工具，轻量快速，非常好用，多平台都支持，再也不用打开WX去和文件助手互传了。

**3、[uTools](https://zhida.zhihu.com/search?content_id=748491056&amp;content_type=Answer&amp;match_order=1&amp;q=uTools&amp;zhida_source=entity)（好用的启动工具）**

神器级效率工具，文件搜索、翻译、OCR、截图、剪贴板管理……一堆插件装完根本停不下来。 算是 Alfred 的平替。

虽然也有会员，但是基础功能就足够用了。

**4、[NoTab](https://zhida.zhihu.com/search?content_id=748491056&amp;content_type=Answer&amp;match_order=1&amp;q=NoTab&amp;zhida_source=entity)（我自己开发的浏览器插件）**

这条是夹带私货（逃，是我自己开发的浏览器插件，挺好用的，我现在每天在用。

NoTab 可以在当前页面预览链接内容，不用一堆标签页乱飞，特别适合在知乎、GitHub、Reddit、Google 搜索时快速预览内容。

还支持快速翻译、快速搜索等功能。

能让你做到一个页面完成所有事情。

用着挺爽的，欢迎知乎的朋友们试试 [https://notab.pro/](https://link.zhihu.com/?target=https%3A//notab.pro/)

这个插件虽然有pro版，但是轻度使用，免费版足够了。

**5、[Typora](https://zhida.zhihu.com/search?content_id=748491056&amp;content_type=Answer&amp;match_order=1&amp;q=Typora&amp;zhida_source=entity)**

支持实时预览的 Markdown 文本编辑器，界面极简却功能强大，提供了智能标点、图片插入、专注模式、代码高亮等大量有关 Markdown 偏好的设置。

我以前挺喜欢用的，不过现在不怎么用了，现在看到 Markdown 原文就能脑补一下预览后的样式，还是更喜欢看原文。

**6、[iStat Menus](https://zhida.zhihu.com/search?content_id=748491056&amp;content_type=Answer&amp;match_order=1&amp;q=iStat+Menus&amp;zhida_source=entity)**

菜单栏常驻的系统监控工具，实时显示 CPU、内存、磁盘、网络等核心指标，支持 GPU 监控和历史图表分析。

还有天气模块和通知功能（如电池阈值提醒），非常全面。

**7、[Synergy](https://zhida.zhihu.com/search?content_id=748491056&amp;content_type=Answer&amp;match_order=1&amp;q=Synergy&amp;zhida_source=entity)**

通过一套键鼠控制多台跨平台设备（Mac/Windows/Linux），支持剪贴板共享和文件拖拽。

特别适合多设备工作流。

**8、[Maccy](https://zhida.zhihu.com/search?content_id=748491056&amp;content_type=Answer&amp;match_order=1&amp;q=Maccy&amp;zhida_source=entity)**

轻量级剪贴板管理器，支持无限历史记录、快速搜索和格式转换（如将图片 URL 转为 Markdown 链接）。

仅存储文本内容，避免隐私泄露。
