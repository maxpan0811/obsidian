## 一句话摘要

Anthropic 的 Claude Code 团队宣布内部文档不再写 Markdown，直接写 HTML——理由是 Markdown 是写给作者看的，HTML 才是给读者看的。`html-anything` 项目随之爆火（250+ Star）。

## 核心观点

> Markdown 是写给作者看的，HTML 才是给读者看的。
