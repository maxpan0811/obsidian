# 华程国旅集团 LOGO 使用规范

> 规范集团及旗下品牌视觉形象管理文件。涵盖华程国旅集团、翔龙万里行、携程国旅、TRIP2EU四个品牌的组合与单一标识规范。

## 品牌定位
| 品牌 | 定位 |
|------|------|
| 华程国旅集团 | 中国领先出境旅游运营服务商，携程全资子公司 |
| 翔龙万里行 | 出境旅行专家，B2B出境旅游批发品牌 |
| 携程国旅 | 携程旗下C端综合旅游服务品牌 |
| TRIP2EU | 欧洲循环巴士游服务品牌，自由高性价比 |

## 组合标识及适用场景
1. **华程+翔龙+携程国旅+TRIP2EU**：内部会议、综合性对外场景
2. **华程+翔龙+携程国旅**：携程渠道贴牌产品宣发
3. **华程+携程国旅**：携程渠道轻量化产品宣传
4. **华程+翔龙+TRIP2EU**：欧洲业务宣传、发布会、展会
5. **华程+翔龙**：常规品牌宣传

## 单一标识
- **翔龙符号**：授权客户翔龙系列产品宣传
- **携程国旅出品**：授权客户携程国旅贴牌产品宣传

## 通用要求
- 比例规范：不得拉伸压缩变形
- 色彩规范：使用统一标准色值
- 留白要求：周围预留足够空白
- 版权声明：未经授权不得修改复制
