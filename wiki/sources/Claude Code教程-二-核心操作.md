# 法律人的 Claude Code 教程（二）：十分钟掌握核心操作

> 来源：智律积成（微信公众号）
> 原文链接：https://mp.weixin.qq.com/s/...

## 关键概念

- **工作目录** = 案件档案袋。在哪个文件夹启动 Claude Code，它就只能处理那个文件夹的材料
- 三种操作符：`/` 调用内置命令、`@` 引用文件、`!` 执行 Bash 命令

## 快捷键

| 快捷键 | 作用 |
|--------|------|
| Enter | 发送消息 |
| `\` + Enter | 多行输入 |
| Esc | 中断生成 |
| ↑ / ↓ | 查看历史 |
| Ctrl+L | 清屏 |

## 核心命令

/help, /clear, /compact, /status, /model, /resume, /init, /export, /plan, /rewind, /stats, /exit, /config, /doctor

## 实用技巧

- 粘贴图片（截图或扫描件）让 Claude Code 分析
- 拖拽文件到窗口 = `@` 引用
- /export 导出对话记录到文件或剪贴板
- /model 切换模型：Haiku（快问快答）/ Sonnet（日常推荐）/ Opus（复杂分析）
