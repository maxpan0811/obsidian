**疫情数据日更：美国确诊超中意总和；法国病死超中国。关注人口大国隐形疫情**

**一、****要点**

l  **美国的确诊数超过中国与意大利总和，明后天将超过意大利与西班牙的总和**

l  **美国病死率2.2%****，是2009-10****年美国新型H1N1****流感病死率的100****倍。可见Trump****将COVID-19****流感化错误的程度**

l  **欧盟（4.4****亿人）确诊超过40****万，病死近3****万**

l  **意大利疫区伦巴第确诊4.3****万，病死7,199****人，病死率16.6%****，较SARS****高出73%**

l  **西班牙确诊数破10****万，未来几日可能超过意大利，病死破9****千，未来两日可能破万。马德里病死3,865****人，病死率13%**

l  **德国的病死率爬升至1.1%****，超过除湖北以外的中国内地（0.9%****）**

l  **法国病死3,524****人，超过中国。病死率高达6.7%**

l  **英国确诊近3****万，病死2,352****人，病死数几日内会超过中国**

l  **土耳其疫情继续爆发，已超过比、荷、奥，几日内将超过瑞士**

l  **关注其他人口大国（印度、印尼、巴西、巴基斯坦、孟加拉、尼日利亚、俄国、日本）等的疫情。这些国家因为不同的原因，数据都不可靠，可能存在隐形疫情。他们代表COVID-19****抗疫的第三阶段，将进一步摧毁人口流动的全球化。**

**二、日更信息**

**美国**

l  根据GHSI评选，美国为“传染病防控准备最充分的国家”的全球第一名

l  确诊数破20万，超过中国与意大利确诊数的总和。照此速度，美国确诊数明后天将超过意大利与西班牙的总和

l  病死数4,516例，病死率升至2.2%

l  2009-10年新型H1N1流感，美国全国感染人数6000万，病死约12,469例，病死率为0.02%（万分之二）。Trump先前将COVID-19与流感相比，目前COVID-19在美国的病死率是新型H1N1的100倍

l  美国的全社会感染率是中国10.9倍，全社会病死率是中国的5.9倍

l  疫情继续落后欧盟约一周左右，与欧盟发展速度相当

l  纽约（人口约860万）确诊4.5万，人数约等于2月18日的武汉，按全社会感染率计算为今日武汉的1.3倍

l  继韩国、伊朗、意大利、西班牙之后，COVID-19在美国增长速度又刷新了人们对这个病毒的认知。另外，看完美国的案例似乎就能明白为什么这里每年流感疫苗接种率如此之高，仍有数千万感染及数万人死亡

**欧盟**

l  欧盟确诊41.7万人，单日新增3.4万，几个大国（意西德法）病例增速不见放缓

l  病死数2.96万，是中国的8.9倍。病死率7.1%，中国的1.75倍

l  全社会感染率是中国的16.6倍，死亡人数8.9倍，全社会病死率29倍

**意大利**

l  确诊破11万，病死1.3万，病死率继续上升，达11.9%

l  疫情重灾区伦巴第确诊4.3万，病死7,199人，病死率16.6%，比SARS高73%。伦巴第的全社会死亡率（每百万人死亡人数）是武汉的6倍，COVID-19病死率是5倍

**西班牙**

l  单日新增确诊7,719例，确诊总数如期破10万

l  西班牙的全社会感染率超出意大利20%，很有可能在未来几日超过意大利

l  死亡9,053例，死亡率升至8.9%。未来两日内病死人数可能破万

l  疫情最严重的马德里（660万人）     目前确诊约3.0万病例，病死3,865例，病死率达13.0%，可比意大利伦巴第地区。这里是另一个人间地狱

**德国**

l  总病例7.5万

l  新增病亡166例，病死总数848例，病死率继续爬升，达到1.1%。德国的病死率已经超过湖北省以外的中国内地（0.89%）

**法国**

l  病例快速爬升，达4.5万

l  病死3,524例，超过中国。病死率高达6.7%，全社会死亡率超过中国湖北省

**英国**

l  根据GHSI评选，英国为“传染病防控准备最充分的国家”的全球第二名

l  确诊接近3万例

l  病死2,335例（单日新增563例），病死率攀升至7.9%。按照这个速度，英国病死数量在几天内就会超过中国

**荷兰**

l  根据GHSI评选，英国为“传染病防控准备最充分的国家”的全球第三名

l  荷兰继续爆发，病死率8.6%

l  荷兰的病死率、每百万人病死数均远超中国湖北省。

**值得关注的其他国家**

l  **土耳其**：单日新增2,148，总病例15,679。预计明天将超过荷兰、比利时，几日内超过瑞士

l  **印度：**单日新增240例，总病例1,637。印度的数字也是一个谜。从印度的“佛系”及政府封城的高压手段看，印度的疫情可能很严重了，只是无法通过全面检测呈现出来。这是一个13.8亿人口的大国。印度这样的国家爆发就是笔者说的COVID-19“第三阶段”（第一阶段中国，第二阶段发达国家，第三阶段广大发展中国家）

l  **印度尼西亚：**单日新增149例，总确诊1,677，病死率高达9.4%。这是一个人口2.7亿的大国。从这么高的病死率看疫情可能已经严重爆发。笔者之前谈到疫情“第三阶段”时即用印度尼西亚做过比较。

l  **其他值得关注的人口大国**：巴西（2.1亿人，5,923例），巴基斯坦（2.2亿人，2，071例），孟加拉（1.6亿人，54例），尼日利亚（2.1亿人，151例），墨西哥（1.3亿人，1,215例），日本（1.26亿，2,178例），俄罗斯（1.5亿，2,777例）。这些人口大国家都是COVID-19高风险国，同时出于不同的原因，疫情数字准确度都非常值得怀疑，可能存在隐形疫情。他们代表COVID-19抗疫的第三阶段，将进一步摧毁人口流动的全球化
