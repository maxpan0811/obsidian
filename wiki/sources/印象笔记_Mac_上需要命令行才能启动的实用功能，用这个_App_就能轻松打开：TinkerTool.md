关于电脑系统永不过时的话题之一，就是对其进行调整配置，使之更加符合自己的使用习惯或满足自己的需求。macOS 系统以用户友好、使用方便而著称，但「众口难调」难免有些不到位之处。今天介绍的这一软件 [TinkerTool](https://www.bresink.com/osx/TinkerTool.html "TinkerTool") 则将 macOS 中额外的隐藏特性配置选项以安全、便捷的方式展现出来，或许可以让你的系统更加贴心好用。

![](.evernote-content/E5FF45A4-1A5C-4255-A955-006E6DA3A46B/126CF315-5473-4E6D-B356-AC803B615CFC)

如上图所示，TinkerTool 界面上方为各个配置页，下方则是相应的配置选项。需要注意的是最下方的配置生效方法与按钮，一般为重启软件或重新登录系统。

常用配置
----

TinkerTool 中配置选项多达 62 个，分别作用于这些方面：访达（Finder）、程序坞（Dock）、启动台（Launchpad）、桌面、Safari 浏览器、iTunes 等。这里就不一一介绍了。下面简要介绍几个在我看来比较有用的特性。

### Dock 配置页

**显示最近使用的项目（Add stack for recent items）：**开启后，可方便快捷地访问最近使用的应用程序、文稿、收藏项目等，通过右键菜单可以进行选择。

![](.evernote-content/E5FF45A4-1A5C-4255-A955-006E6DA3A46B/86C17ACF-D4DF-4562-9861-D11609C56A1F)

最近使用的项目

![](.evernote-content/E5FF45A4-1A5C-4255-A955-006E6DA3A46B/624ED523-D7C1-4E57-B734-AFD3463116F2)

右键菜单

**插入空白分隔符（Insert spacer after Dock tile）：**该项可以在 Dock 栏中指定图标后，加入一个或半个图标大小的空白图标，方便用户对 Dock 栏手动分隔为不同的区域。如果想要删除该空白分隔，则可以右键单击它，选择移除。

![](.evernote-content/E5FF45A4-1A5C-4255-A955-006E6DA3A46B/EE773065-F57C-4E16-B024-2F95FFDD895F)

### Finder 配置页

**禁用桌面（Disable Desktop features）：** 开启后，在桌面上不显示任何文件或文件夹图标，给你一个好看、不杂乱的桌面。这个功能不会删除桌面上的文件，可以在 Finder 中打开桌面文件夹访问。

**显示隐藏和系统文件（Show hidden and system files）：** 有时候我们需要对系统中的隐藏文件进行管理，这里可以让它们在 Finder 中显示出来。**如果你不清楚隐藏文件的具体作用，请不要做任何操作**。10.12 以上的系统中已加入快捷键支持：⇧Shift + ⌘Command + 英文句点。

### General 配置页

macOS 系统自带的截图快捷键（⇧Shift + ⌘Command + 3/4）十分方便，这里可以对其做进一步的配置。

**截图格式（Screenshot file format）**默认为 PNG 格式，可根据你的需要，设置为 PNG、JPEG、GIF、PDF 等9种格式之一。

![](.evernote-content/E5FF45A4-1A5C-4255-A955-006E6DA3A46B/40869A73-3511-4176-A6F4-92B302A00D5F)

**截图文件保存位置（Destination folder）**默认为桌面，这里也可以将其改为其它文件夹，方便管理或同步。

### iTunes 配置页

**允许以半颗星进行评分（Allow ratings with half stars）**，相信使用 iTunes 管理歌曲并追求精确的用户会比较喜欢这个特性。

### 其它配置页

在 Launchpad 配置页中，可更改网格行列数目。

![](.evernote-content/E5FF45A4-1A5C-4255-A955-006E6DA3A46B/619F116B-E3DA-4B10-BFE9-5524E36F67EC)

在 Safari 配置页中，可将退格键（Backspace）设置为返回上一页的快捷键，还可以设置默认的网页缩放大小、字体。

在 Fonts 配置页中，可设置 macOS 中软件的默认字体，但由于可能引起界面问题，故不建议使用。

### Reset 配置页

**Reset to pre-TinkerTool state：**还原为使用 TinkerTool 配置之前的状态。

**Reset to defaults：**还原为 macOS 系统默认的状态。

补充说明
----

[TinkerTool](https://www.bresink.com/osx/TinkerTool.html "TinkerTool") 中的所有配置只对当前用户生效，不需要管理员权限，对系统组件没有更改，不影响系统完整性与升级。如果不喜欢配置后的效果，可以恢复至配置前或默认状态。所以，可以放心配置、试用。  
最新版的 TinkerTool 只适用于 macOS 10.12 Sierra 与 macOS 10.13 High Sierra。其它版本的系统可以在 [官网的这个页面](https://www.bresink.com/osx/TinkerToolOverview.html "官网的这个页面") 中下载对应的可用版本。由于 TinkerTool 的原理是呈现 macOS 系统自带的隐藏配置项，所以不同版本中配置项目可能有所不同。

关于 TinkerTool 的其它疑问可以访问 [官网的 FAQ 页面](https://www.bresink.com/osx/0TinkerTool/issues.html#FAQ "官网的 FAQ 页面")（英文）进行查看。
