# 11款相见恨晚的Mac效率神器

原文链接: https://mp.weixin.qq.com/s?__biz=MzA4MzM1NjI4OA==&mid=265056...

...Original小猫 猫果乐园 
没有星标就看不到小猫了😭 你的星标⭐是我的最大动力
说句大实话，咱们天天夸 macOS 优雅，但真到了高强度干活、多线并行的时候，原生的交互逻辑经常让人抓狂：找个临时文件得在十几个窗口里来回切；好不容易排队等到的客服回复，通知一闪就没了；分享个 PDF 还得忍受层级极深的右键菜单。
真正的极客，绝不会被原生系统“绑架”。只要花上几杯咖啡的钱（甚至完全免费），你完全可以把手里这台昂贵的 Mac 爆改成一台高度定制化的生产力怪兽。
今天不藏私，直接把我压箱底的 11 款效率工具全盘托出。装上它们，你的工作流摩擦力至少能降一半。
01. Docside：把 Dock 栏的“死角”榨干（$5.99 买断）
绝大多数人从来没意识到，屏幕下方 Dock 栏左右两侧的空白区，是巨大的空间浪费。Docside简直是个天才，它直接把这两个废弃死角变成了全能中转站。
左侧（临时货舱）：你可以随手把文件、链接、甚至一段文本直接甩进去。它比原生的暂存逻辑强悍得多，甚至支持自定义动作，比如扔进去直接触发脚本或者一键 AirDrop。
右侧（预览中心）：这里会自动同步你的下载文件夹和最新截图。鼠标滑过去直接展开大图预览，根本不需要你再去反反复复地打开 Finder 翻找。
下载地址：https://apps.apple.com/us/app/outpost-launcher/id6749519388?mt=12
02. Pipiri：万物皆可“画中画”（$12 买断）
网页视频能画中画不稀奇，但Pipiri能让你的 Mac 上任何窗口实现悬浮监控。
这个功能在特定场景下绝对能救命。比如我经常在码字写评测的时候，后台还挂着飞书，死死盯着重庆工厂那边的发车确认消息。切窗口怕错过，并排分屏又挤占写作空间。用 Piperi 把聊天框变成一个全局悬浮的小窗，有动静随时看。最绝的是它的“自动隐藏”——光标一靠近，小窗瞬间透明化，绝对不挡你正常干活。
下载地址：https://pipiri.subsol.one/AFFILIATE_20?affiliate_id=475579491
03. Hyperkey：拯救吃灰的大写键（完全免费）
键盘上的 Caps Lock（大写锁定键）占着最核心的黄金 C 位，但 99% 的时间都在吃灰。
Hyperkey的逻辑极其粗暴且有效：它把大写键变成了一个集成了Command + Option + Control + Shift的超级修饰键。以后再也不用担心快捷键冲突了，按下大写键 + D，直接触发一套极其复杂的组合指令。一分钱不花，凭空多出几百个不冲突的快捷键位。
下载地址：https://hyperkey.app/
04. Outpost Launcher：摇一摇，干掉繁琐的分享菜单（$3.99 买断）
macOS 那个反人类的分享菜单我吐槽过无数次了。Outpost Launcher给出了一个极其性感的解法：物理摇晃。
用鼠标拖住一个文件，轻轻一晃，高度自定义的控制面板就弹出来了。你可以一键把文件丢进特定项目组，或者发给指定联系人。最王炸的是它吃透了端侧的 Apple Intelligence——摇晃一份英文 PDF，直接触发 AI 生成摘要。这种极简的交互，爽感拉满。
下载地址：https://apps.apple.com/us/app/outpost-launch
