# 2026年3月北京公司奖金计提发放说明

> 北京公司月度奖金计提发放的 **操作说明**。2019年11月起生效，按8条注意事项规范奖金核算流程。

## 核心规则

| # | 规则 |
|:-:|------|
| ① | 奖金核算明细页只提供当月数据 |
| ② | 标黄列全部为必填；占用奖金池的奖金类别填写"其他" |
| ③ | 奖金池页中签证及票务部分需自行计算及补充 |
| ④ | 已发未发统计表需与奖金明细页及奖金池页核对一致 |
| ⑤ | 往期未回款奖金回款后填写负数冲销，做已发和年底预留分配 |
| ⑥ | 占用奖金池的福利津贴、已发奖金数等填写发放期；其余为空 |
| ⑦ | 代出票奖金由出票人所在公司填写，费用承担方填写对应金额 |
| ⑧ | 非欧单元自产他销订单销售奖金，由产品端计算并填写 |
