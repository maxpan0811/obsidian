# Office 订阅服务改名了，花同样的钱还能享受更多功能

![](attachments/71a5278e72e118a2.png)

Office 订阅服务改名了，花同样的钱还能享受更多功能

3 月 30 日晚，微软针对自家软件服务宣布了一系列新动态，众多更新内容中最引人瞩目的依然要归功于微软「改名部」——Office 365 办公套件将在 4 月下旬正式改名为 Microsoft 365。借此机会，原有的 Office 套件不仅将迎来一小波功能更新，一些新的应用和服务也将加入其中。

如果你对升级后的 Microsoft 365 感兴趣或是已经订阅了 Office 365，不妨与我们一起来看看接下来它将迎来哪些新变化。

## Office 的新功能

作为本次更新的核心内容，Office 系列的传统三件套首先将会通过这次「改名」获得一系列生产力升级，如果你的日常工作离不开 PowerPoint、Word 和 Excel，微软接下来要带来的新功能将让它们变得更加顺手。

以 PowerPoint 为例，作为办公族最为常用的工具，除了做好幻灯片，如何做好幻灯片演示同样也非常重要。今后 PowerPoint 将通过量化工具来帮助我们克服演讲时的紧张感 —— 这个名为 Presenter Coach 的 AI 工具可以在我们彩排的过程中检测语速、表达逻辑等等问题并提供有针对性的优化建议，但目前并不清楚中文支持是否也会第一时间上线。

![](attachments/b8555505501c5817.gif)

Presenter Coach

同时，「设计理念」功能在新版中也将变得更加好用，除了「一键」生成图文排版建议，新版 PowerPoint 的「设计理念」功能还会直接向用户提供来自 Getty Images 的 8000 多张图片素材和 175 段视频素材的使用权限，此外还有 300 多种新字体以及 2800 多个新图标。

![](attachments/9286aa79dc1c4e4c.gif)

「设计理念」功能更新

至于 Excel，在升级至 Microsoft 365 之后，这款工具将加入一项财务管理功能：Money，用于追踪、分析资金支出状况，支持与银行账户、信用卡信息关联，关联后可以自动导入交易和账户余额等信息并借助财务分析来提供理财建议。和上述功能不同，这项功能确定只会首先向美国地区的用户开放。

![](attachments/9d10dad7204ff9d3.gif)

用 Excel 进行财务管理

另外，Excel 中还将加入新的、具备「语义性」的数据类型支持，语义性数据类型可以分析填写的数据并提供准确的类型分类，配合智能模板能够在进行数据整理的过程中大幅缩短时间并提高效率。该功能将在未来几个月内率先以英语支持的形式提供给 Microsoft 365 订阅用户。

最后，Microsoft 365 还会为针对 PowerPoint、Excel 和 Word 三件套向订阅用户提供 200 多个新增高级模板，微软希望这些模板配合上述功能能够解决各种工作场景和任务需要。和 Adobe CC 类似的，Microsoft 365 订阅显然想为文档办公提供一站式的素材和创意解决方案。

## Microsoft Editor：来自微软的 AI 写作辅助

Office 365 被扩展为 Microsoft 365 当然不可能仅仅只是换个名字，围绕办公与协作，Microsoft 365 还扩展了原先的订阅服务并进一步整合了一系列新的生产力工具，比如 Microsoft Editor。

在 Microsoft 365 中 Microsoft Editor 被进一步扩展到 Word 和 Outlook 网页版中，并支持通过 Microsoft Edge 浏览器或者 Google Chrome 浏览器扩展在其他网络内容编写的场景下使用。作为订阅附加功能，用户可以使用 Microsoft Editor 的高级语法功能，也就是说在进行文字创作时可以更专注于文章的行文逻辑，至于某些精细的词组表达则可以通过 Microsoft Editor 来提供更为专业的建议。

![](attachments/6eaf28b59e23a877.png)

与 Word 整合的微软编辑器

其实这种类似的功能很早就已经出现在 Word 上，只不过 Microsoft Editor 这种基于 AI 的写作助手工具更为智能，也扩展了写作的场景。虽然根据微软的说法 Microsoft Editor 前期仅支持 20 种语言且不包括中文，但在今天早些时候上线的 [Chrome](https://chrome.google.com/webstore/detail/microsoft-editor/gpaiobkfhnonedkhhfjpmhdalgeoebfa) 和 [Edge](https://microsoftedge.microsoft.com/addons/detail/hokifickgkhplphjiodbggjmoafhignh) 浏览器插件中我们还是可以选择开启中文语言的文本纠错，但功能十分有限，推测针对更多语言的辅助写作特性并未全量上线。

![](attachments/2526627b649d8abf.png)

浏览器扩展

## Family Safety：跨平台的全能家庭成员管理工具

虽然应用名称直译过来比较奇怪，但看得出来，Family Safety 这款应用的确是想替 Microsoft 365 用户在「如何防止孩子过于沉迷数字世界」这件事情上出一份力：除了支持在未成年人使用的设备上过滤掉有害内容之外，还能自动统计孩子的各种设备间的各种应用使用时长。

![](attachments/c5f0a23a597271ff.gif)

跨设备家庭成员管理

得益于微软生态下的账号同步体系，Family Safety 甚至还支持同时查看并管理 Windows、Android、Xbox 多平台下的使用时长，在家人在上班或是其他外出通勤的路途中还能共享实时位置、发送详细的沿途数据，并且保存最常去的一些地址，当家人下班离开公司或是上学到学校等场景下自动在手机上推送提醒，这些功能对于日常工作忙碌但同时也要兼顾家庭的上班族来说可谓是贴心无比。

另外，这项功能还可以将驾驶路途中的最高速度与驾驶里程通过手机传感器记录下来并生成可视化数据，甚至还可以生成一份驾驶历史记录统计来提醒驾驶员采取哪些改进才能变得更加安全。

![](attachments/c23a65f3d97a3aa8.png)

可用于驾驶记录

不过目前这项很有未来科幻风的功能由于要收集本地路况等大量数据，所以在推出初期仅仅在美国、加拿大与澳大利亚三国部分地区可用。微软宣布 Family Safety 将在 4 月放出预览版给部分 iOS 与 Android 用户下载，用户也可以通过填写 [表单](https://www.microsoft.com/en-us/microsoft-365/family-safety) 的方式来获取预览版使用资格；和 Microsoft Editor 类似的，Family Safety 也并非一款独立的付费服务 —— 正式上线后它将作为整个 Microsoft 365 套件的一部分，提供给订阅用户免费下载使用。

## Microsoft Teams：团队工具也能用来管理个人琐事

Microsoft Teams 是全球范围内都相当受欢迎的一款办公协作应用，而且相比于同类的 Slack，Teams 在文件存储与视频会议上有更多的优势，且深度整合了 Word、Excel 、OneDrive 以及 Skype 等微软办公全家桶。

![](attachments/7e953a38da3fcdb4.gif)

创建会议

在微软昨日发布的 [Blog 文章](https://www.microsoft.com/en-us/microsoft-365/blog/2020/03/30/introducing-new-microsoft-365-personal-family-subscriptions/) 中，微软同时也宣布了在过去两个月中， Skype 与 Teams 的日活跃用户数均大幅增加，其中 Skype 更是月同比增长激增 70% ，目前总用户数达到了 4000 万，微软为了更好地应对这种因疫情而使用 Skype 的临时用户，也在早先为 Skype 引入了「[Meet Now](https://support.skype.com/en/faq/FA34926/what-is-meet-now-and-how-do-i-use-it-in-skype)」功能：允许非 Skype 用户创建一个视频通话 / 会议，然后将生成的分享链接保存并发送给其他参加会议的人； **用户甚至可以直接在 Skype 的 Web 页面中迅速创建并使用**。这种「开箱即用」的设计让许多非 Skype 用户也能上手即时使用 Skype 的服务来完成视频沟通需求。

除了 Skype 的更新之外，微软也宣布了未来 Microsoft Teams 这款办公协作应用的新方向：让 Teams 来帮助 Microsoft 365 用户更加高效地完成生活中的日常琐事。具体来说，在本次宣布的更新之后，用户可以使用 Microsoft Teams 来建立待办事项清单、以及创建一个家庭共享日历或是购物清单；存储 Wi-Fi 密码等重要的账户信息，甚至是为你所在的社区举办一个社区活动或是线下读书会。

总而言之，微软希望配合现有的 Skype 与「[微软待办](https://sspai.com/post/58375)」等应用生态，继续拓展 Teams 在非办公场景时的使用界限，不仅提高你工作的效率，还想帮你提高「生活的效率」，毕竟效率工具的使用场景并不局限于办公室之中，同样也能让私人生活变得更加井井有条。

![](attachments/8bf418b3594e7898.png)

Teams 同样也可用于个人生活管理

这些更新内容，会在接下来的数月中陆续更新整合进 Microsoft Teams 的各个平台的应用中，为所有的 Teams 用户推送。

## 内容更多但价格不变

虽然在功能上 Microsoft 365 基于 Office 365 加入了多项新功能，但对于之前的 Office 365 个人用户和家庭用户而言，订阅价格几乎没有任何变化。

其中 Microsoft 365 个人版对应的是此前的 Office 365 个人版，而 Microsoft 365 家庭版对应的是此前 Office 365 家庭版，均包含相同的 Office 应用和 OneDrive 云空间，而 Microsoft 365 即将加入的新应用和功能均适用于这两种订阅模式，两者唯一的区别在于家庭版支持最多六个家庭成员，而个人版仅限单人使用。

![](attachments/c4ed15917643f059.png)

新的订阅服务对比 | 图：微软

目前 Office 365 个人版官方售价 398 元 / 年，家庭版为 498 元 / 年。
