新年伊始，很多软件都发布了版本更新，今天汇总了一些，来和大家分享一下。

![](.evernote-content/4BD73B09-6006-43A2-913E-6FE9F2414972/CCE1BE8E-D2B2-4D52-B8A1-F4DC7F8F5C89.png)**BetterDisplay 3****[Mac]**

BetterDisplay 3 是一款 Mac 显示器管理工具，可通过 DDC 调整内建 / 外接显示器的多种参数。

同时提供虚拟屏、串流、画中画等丰富的高级功能，支持大多数现代显示器。

![](.evernote-content/4BD73B09-6006-43A2-913E-6FE9F2414972/FCDD228D-3F47-4A30-871C-C09601BAC053.png)

两天前，BetterDisplay 开发者发布了 3.3.0 版本，新增了：显示器按组管理、快捷提示对应屏幕、禁止外接显示器休眠等丰富功能。

**显示器按组管理**

更新到新版本后，就能在 BetterDisplay 内创建显示组，并对组内的显示器进行批量管理啦！

比如同步调整亮度、音量、对比度等，以及开启布局保护、分辨率配置保护等。

![](.evernote-content/4BD73B09-6006-43A2-913E-6FE9F2414972/89AC696A-FE64-4CD8-8FD8-B065DBA4376F.png)

**快捷提示对应屏幕**

对于拥有多个显示器的用户，有时候可能搞不清楚显示器名称和显示器本身的对应关系。

现在，在 BetterDisplay 内，按下 Option 键的同时，再点击显示器名称，屏幕就会短暂发出半透明红光来提醒。

![](.evernote-content/4BD73B09-6006-43A2-913E-6FE9F2414972/344BC5C5-9A62-4DEE-BAE6-218CD1BB7E30.png)

做了个示意图，大致效果如上

荔枝君测试了下，通过 Option + 点击，很多地方都能触发，用起来挺方便的。

![](.evernote-content/4BD73B09-6006-43A2-913E-6FE9F2414972/936DF8E1-2102-4BE9-A038-FBBA6DC572B3.png)

Option + 点击箭头所示位置

![](.evernote-content/4BD73B09-6006-43A2-913E-6FE9F2414972/B4BCCC91-0DA1-4579-8643-86594F16667F.png)

Option + 点击箭头所示位置

**禁止外接显示器休眠**

在新版的菜单栏面板中，新增了「Manage Display」选项，点进去就能自定义显示器的多项高级属性，其中包括「Prevent Sleep While Connected」。

![](.evernote-content/4BD73B09-6006-43A2-913E-6FE9F2414972/8A79D1A3-7E4E-4D80-AB09-BC0734078D1D.png)

打开它，对应显示器在被连接后，就会被禁止进入休眠状态，让我们能够长久的安心工作。

由于是新功能，目前这部分设置项**未完全汉化**。

Wins 是一款专为 Mac 设计的多窗口管理工具，通过拖动或快捷键就能实现快速分屏，并提供悬浮分屏及深色模式。

![](.evernote-content/4BD73B09-6006-43A2-913E-6FE9F2414972/6AD6BCED-62FA-4360-A250-C207DE2512B7.png)

1 月 5 日，Wins 迎来了 2.5.1 版本，针对近期新增的调度中心 Pro 功能作出了多项改进，其中包括：

* 修复该功能与 Adobe 和 Excel 等 App 的兼容性问题；
* 优化该功能的能耗表现，现在更省电省资源；
* 另外还修复了悬浮分屏的动画花屏问题。

![](.evernote-content/4BD73B09-6006-43A2-913E-6FE9F2414972/5DD0A675-7D32-44CF-8D74-52C7D31DAE1B.png)**Input Source Pro****[Mac]**

最后要介绍一款免费的输入法自动切换工具 Input Source Pro，软件近期发布了 2.5.0-beta 版本更新。

它的基础功能就是为自定义应用，设定一个默认的输入法。

![](.evernote-content/4BD73B09-6006-43A2-913E-6FE9F2414972/8A9E8EFF-D577-4864-A95F-4882495C1D44.png)

当我们在一边编辑中文文档，一边使用终端输入命令行的时候，往往需要频繁地手动切换输入法。

Input Source Pro 就能帮助我们实现：光标点击文档窗口时，用中文输入法；光标点击终端窗口时，用英文输入法。

![](.evernote-content/4BD73B09-6006-43A2-913E-6FE9F2414972/DC36AE67-6226-4B35-BE9B-D70C933B36B6.gif)

根据应用自动切换

![](.evernote-content/4BD73B09-6006-43A2-913E-6FE9F2414972/4D1A36E7-CCF3-4E34-AF8E-AD277E8730EA.gif)

根据网站自动切换

在新版中，软件还更新支持「使用同一个快捷键，在多个输入法之间切换」。

在系统默认情况下，按下快捷键，会在所有输入法之间轮转。

现在利用 Input Source Pro，就能实现用一个快捷键，调用并在自定义的数个输入法之间切换。

![](.evernote-content/4BD73B09-6006-43A2-913E-6FE9F2414972/9AF64B32-04DA-4A91-8B44-F7796EAADBF9.png)
