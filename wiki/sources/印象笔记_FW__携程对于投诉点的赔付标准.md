主题：携程对于投诉点的赔付标准  

各位好，为提高旅游服务质量，规范旅行社经营，保护旅游者合法权益，我司根据《旅行社服务质量赔偿标准》相关条款制定出了几个投诉点的赔付上下限标准，今后在实际操作中会严格按照此标准对客人进行赔付，请知晓，谢谢！

| **投诉点** | **赔付下限** | **赔付上限** | **赔付依据** |
| 领队态度不佳 | 订单金额×1% | 订单金额×5% | 《旅行社服务质量赔偿标准》第九条　导游或领队未按照国家或旅游行业对旅游者服务标准提供导游或者领队服务，影响旅游服务质量的，旅行社应向旅游者支付旅游费用总额1%至5%的违约金，本赔偿标准另有规定的除外。 |
| 领队语言/行为暴力 | 订单金额×1% | 订单金额×5% |
| 领队本职操作问题 | 订单金额×1% | 订单金额×5% |
| 司机与导游领队争执 | 订单金额×1% | 订单金额×5% |
| 司机安全驾驶相关 | 订单金额×1% | 订单金额×5% |
| 司机态度不佳 | 订单金额×1% | 订单金额×5% |
| 导游态度不佳 | 订单金额×1% | 订单金额×5% |
| 导游服务不佳 | 订单金额×1% | 订单金额×5% |
| 导游不讲解 | 订单金额×1% | 订单金额×5% |
| 领队/导游未协助办理入住 | 订单金额×1% | 订单金额×5% |
| 增加自费（客人未签字） | 退还自费费用 | 自费费用退一赔一 | 《旅行社服务质量赔偿标准》第十条　旅行社及导游或领队违反旅行社与旅游者的合同约定，损害旅游者合法权益的，旅行社按下述标准承担赔偿责任：（二）未经旅游者签字确认，擅自安排合同约定以外的用餐、娱乐、医疗保健、参观等另行付费项目的，旅行社应承担另行付费项目的费用。 |
