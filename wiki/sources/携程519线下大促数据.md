# 携程 519 线下大促数据

> 数据口径：2026-05-19 携程线下大促，跟团游整体。

## 核心数据

| 指标 | 数值 |
|------|------|
| 跟团游整体产量 | **3 亿** |
| 国内跟团 | 1.3 亿（43%） |
| 海外跟团 | 1.7 亿（57%） |
| 跟团**自营**产量 | **5,154 万**，占比 **17%**（创新高） |
| 国内自营 | 1,400 万，占比 11% |
| 海外自营 | 3,700 万，占比 22% |

## 海外自营份额结构

| 来源 | 金额 | 占比 |
|------|------|------|
| 直采 | 2,679,006 元（约 268 万） | 58% |
| **华程贴牌** | — | **24%** |
| 其它海外贴牌 | — | 18% |

**华程贴牌占携程海外自营的 24%**，是第二大贡献方，仅次于直采。

## 独立出游数据

| 类型 | 金额 |
|------|------|
| 独立出游总计 | 5,700 万 |
| 私家团 | 2,679 万 |
| 定制 | 2,085 万 |

## 关键信号

- 自营占比 17% 创新高，携程在加大自营掌控力
- 华程贴牌是携程海外自营的重要组成，但直采（58%）占主导，说明携程持续压缩对供应商的依赖
- 私家团（2,679万）是独立出游最大单项，高于定制（2,085万），个性化产品已超过传统定制规模
