# 国内学者“大饥荒”问题研究述评

---

|  |
| --- |
| 关于60年代初死亡人数 |

2000、2010年两次人口普查的数据，横坐标是出生时间，纵坐标是人口数目，从出生看，50、60、61是异常年份，62年恢复正常，63年是“报复性”的补偿回升，粗率估计了一下，59-63年，人口少出生的数量大概在2000万左右。

人口普查的数据一来是2000年之后的数据，精确度很好，两次普查数据重合度非常高，而且对应的年龄段正好是60岁以下的，夭折的比例低，总数不会有太大的差异；二来，人口普查数据没有造假的动力，可信度很高。所以，在考虑59-61这三年的非正常死亡总数时候，先扣除2000万，再来计算会比较精确。当然饿死100万也是天大的错误
