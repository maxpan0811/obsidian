# 小米同时上架三款插线板 用途更加场景化 米家,小米,插线板 锋科技,不一样的科技新闻_威锋网

  [家电](http://tech.feng.com/appliances/index.shtml)
  *作者：*[拾欢伯](http://bbs.feng.com/u.php?uid=15628516) *来源：*威锋网**

  [评论 *17*](http://tech.feng.com/2018-01-03/_695630.shtml#comments)

  昨天，MIJIA米家品牌在小米各大渠道上架了三款全新的米家插线板，除了基础版和带 USB 接口与独立开关的版本之外，还新增了一款“二位转换器”，看来小米是打算在小米百货的路上越走越远了。

  [![](.evernote-content/2DA1DB1A-483B-4577-B2A5-5C516682A151/C4D4EAA6-A89A-48DC-A5C4-FFE40D47A26C.jpg)](http://resource.feng.com/resource/h062/h24/img201801031728440.jpg "小米同时上架三款插线板 用途更加场景化")

    
  我们先来看看这三款产品，第一款是售价24.9元的米家插线板3位基础版。既然是基础版，那么肯定不会有花里胡哨的功能，这款插线板只有一个总开关和三个新国际插孔。所以对于只需要基本需求的朋友来说无疑是很划算了，而且3米和5米两个版本应该能满足大多数使用环境。  
    

  [![](.evernote-content/2DA1DB1A-483B-4577-B2A5-5C516682A151/EF2FA3E2-9886-453F-9458-26E44C29486F.jpg)](http://resource.feng.com/resource/h062/h24/img201801031728590.jpg "小米同时上架三款插线板 用途更加场景化")

    
  第二款是米家四位四控插线板，售价高达89元，而贵也是有贵的道理，除了每个新国际标准插孔都单独配备开关之外，3个 USB 接口都支持高达 2A 的快充，小米称这些 USB 插孔能够提供和原装充电器一样安全的充电功率，不损伤电池寿命。考虑到灵活的开关和方便的 USB，这款产品算得上是办公桌必备利器。  
    

  [![](.evernote-content/2DA1DB1A-483B-4577-B2A5-5C516682A151/44FFEAD9-43CE-4773-B434-518221F0F01A.jpg)](http://resource.feng.com/resource/h062/h24/img201801031729130.jpg "小米同时上架三款插线板 用途更加场景化")

    
  最后一款售价44.9元的“二位转换器”之前很少见过，但是用途其实非常广泛，当我们只需要拓展一下插孔，又不想有丑陋的电线时，二位转换器的优势就凸显出来了，该产品包含2个新国际插孔与2个USB接口，提供过载保护等功能，应该是床头必备单品了。  
    

  [![](.evernote-content/2DA1DB1A-483B-4577-B2A5-5C516682A151/8491C155-152C-46A6-BE80-10B6901942C7.jpg)](http://resource.feng.com/resource/h062/h24/img201801031729280.jpg "小米同时上架三款插线板 用途更加场景化")

    
  其实小米旗下的周边产品一直都是非常值得推荐的，就拿米家插线板来说，小米在2015年进入该市场，现在旗下已经有了十余款产品，每个都称得上爆款。至于大家担心的安全问题，这些插线板内部上都运用了青米核心的集成电路技术以及铜带一体化技术，避免虚接、发热、着火，并且每个插孔都有儿童独立安全门。  
    
  不过小编自己最看重的，还是像 USB 快充、米家智能插线板的联网体验等创新功能和超高的颜值，大家都怎么看呢？
