年光如激浪，休怅望，青春都一晌丨周末读诗
====================

原创 三书 新京报书评周刊

收录于合集#周末读诗138个

![](.evernote-content/12CDE82F-E1F5-459B-B43B-45174286A4EE/5C8EE0B7-B771-409C-A252-A20F41072224.gif)

平安大街很宽，双向六车道，从街这边望向对面，中间像隔着一条汪洋，车辆川流不息。我走在人行道上，与路边的槐树一样卑微，一样不合时宜。

 

走着走着，我看见前方有一匹马，套着木板车，站在槐树荫下。接着就闻到牲畜的气味，混着尘土、草料、汗水以及马粪，让我想家的气味，渐近渐浓。

 

快到跟前，我看清了，真的是一匹马，我的意思是一匹真的马：枣红马，眼睛会眨，蹄子会动。我驻足观看，像观看一个奇迹，高大、俊美的枣红马，简直不像是真的。

 

板车上是一堆香瓜，车辕旁坐着一个大爷，泛旧蓝布衣裤，黑绒布鞋，手里松松地握着马鞭。我买了几个香瓜，和大爷说了几句话，然后慢慢靠近马。

 

马的气味、体温和鼻息全都活生生的，黑亮的眼睛温驯安静，我看见我在其中的形象：一个莫名的存在。我想抱住它的颈，想伸手摸摸它，但是不敢。我想对它说点什么，也不知该说什么该怎么说。我感到自己的笨拙，呆呆地站了一会儿，然后便走了，重新做回一个人。

 

槐树，马，香瓜，尘土，农民，我，我们同在一个世界，一个被六车道上的车流冲到边缘、甩在后面的世界。

 

天快黑时，我原路返回，街边没有了那匹马。走到十字路口，不期然地看见它：夹在汽车中间，站着等红灯，与它的主人相依为命。夜色如海，点点尾灯闪烁，摇曳成一片繁华幻梦，枣红马消失其中。

**淡淡阳光，桂枝薄霜**

/ /

**《十二月》**

**（唐）李贺**

**日脚淡光红洒洒，薄霜不销桂枝下。**

**依稀和气解冬严，已就长日辞长夜。**

/ /

农历十二月，也就是腊月，日光淡薄，太阳温润地走过，洒下微红的光芒。时值隆冬，桂枝结着薄霜，即便有太阳，枝下霜一整天也不会化。

 

鬼才李贺总是透出清冷气息，“日脚淡光红洒洒，薄霜不销桂枝下。”这两句虽然写实，但他的感受方式以及用词，创造出不一样的诗。日脚，太阳像长了脚，是说太阳走得很快，如同“雨脚”，也是说雨下个不停。红洒洒，腊月的阳光不强，故呈现淡红，尤其是上午和向晚的时候，柔和舒缓可以直视。

 

桂树常青，与松树一样，青青枝叶，白雪严霜，更觉新鲜辉映。岑参在《天山雪歌送萧治归京》诗末曰：“雪中何以赠君别，惟有青青松树枝。”青青松树枝，在雪中真是美丽。李贺句中的桂枝，别有一种仙意，仙人之山多桂树，又《楚辞·招隐士》诗中，桂树丛生兮山之幽，桂乃山谷之士攀援之物，可见其高洁孤芳。

 

冬至已过，阳气发动，虽然还是很冷，但依稀可以觉察和气发萌，天气在趋于大寒时，严冬也在悄悄消解。“依稀和气解冬严，已就长日辞长夜。”末句看似平淡无奇，然而我们能切身感受到其中的惊喜，尽管大寒将至，但已辞长夜就长日，白天越来越长，昼光越来越强，万物正在复苏，天地又是新的天地。

 

读这首诗，想起小时候，那时没有暖气，热水也不常有，冬天寒冷且漫长，但是一进腊月，因为快要过年了，一切都变得明亮。晌午坐在院子里，阳光温煦，屋上炊烟袅袅，村中鸡鸣狗吠，梧桐日影静静移转，饭菜冒着热气，收音机大声地播放地方戏，天气虽冷，腊月的回忆却热气腾腾。

清 明俭《四季山水屏》（局部）

**十二月一日，在云安县江边**

/ /

**《十二月一日三首》其一**

**（唐）杜甫**

**今朝腊月春意动，云安县前江可怜。**

**一声何处送书雁，百丈谁家上水船。**

**未将梅蕊惊愁眼，要取楸花媚远天。**

**明光起草人所羡，肺病几时朝日边。**

/ /

公元765年，离开成都草堂后，杜甫辗转去了嘉州、渝州、忠州，是年秋天，他来到云安，本来计划下江东归，然而这时肺病发作，不得已而栖泊于此。看看已至腊月，十二月一日这天，他来到长江边，一气呵成写了三首连章律诗。

腊月在北方是数九寒天，南方气暖，此时已可见“春意动”。寒冬时节的春意，触发了杜甫的诗情。云安县地处峡中，大江横前，水碧山青，风景壮丽，聊可乐亦正可哀也。天涯羁旅，老病缠身，杜甫的心情很难不沉重。“云安县前江可怜”，可怜是可爱的意思，可爱中又掺杂着伤心。

一声雁鸣，令他举头怅望，低头凝想，“一声何处送书雁，百丈谁家上水船。”战乱以来，颠沛流离，有弟皆分散，无家问死生，入蜀途中在秦州，也曾闻雁声而动了乡愁。时值蜀中大乱，岁之将暮，倍增伤怀，不知大雁传书“何处”。放眼江上，又见船夫们正在用百丈大绳牵挽大船，冲涛逆流而上急滩，亦不知是谁家的船。看见船只，更加归心似箭。

“未将梅蕊惊愁眼”，杜甫是爱梅花的，但自入蜀以来，怕见梅花，裴迪曾登蜀州东亭逢早梅相忆而寄给杜甫一首诗，杜甫在和诗中说，“幸不折来伤岁暮，若为看去乱乡愁。”愁绪萦怀，见梅辄惊，方进腊月，梅蕊未吐，幸未撩起他的忧愁。“要取楸花媚远天”，这里是“楸花”，不是“椒花”。花楸性喜寒，唐代多植于道旁，花楸子秋冬英垂如豆，红果累累，比花椒要艳丽得多，杜甫用“媚远天”写其姿态，实则也是写他的心酸。

最后，他想到自己有过的高光时刻，才要高兴，却又顿生怅惘，贫病交加，惟恐终老峡中。“明光起草人所羡，肺病几时朝日边。”古代有朝正之礼，即正月群臣要朝见天子，杜甫并没有为朝廷起草过文诰，但因将近正月，而想起自己在长安献《三大礼赋》的盛况，“集贤学土如堵墙，观我落笔中书堂”，而此时肺病累年，江湖转徙，于是感慨不知何年何日才能再朝觐天子。

杜甫在峡中的诗作，以七言律诗最为放荡，尤以写当地风土民情者最脍炙人口，例如“负盐出井此溪女，打鼓发船何郡郎”，“春花不愁不烂漫，楚客唯听棹相将”，文情参差，清新堪比竹枝词。

**腊月年光如激浪**

/ /

**《渔家傲》**

**（宋）欧阳修**

**腊月年光如激浪，冻云欲折寒根向。**

**谢女雪诗真绝唱，无比况，长堤柳絮飞来往。**

**便好开尊夸酒量，酒阑莫遣笙歌放。**

**此去青春都一晌，休怅望，瑶林即日堪寻访。**

/ /

欧阳修曾创作《渔家傲十二首》，以联章鼓子词的形式，分咏一年十二个月的自然物候与民情风俗。这首《渔家傲》不在其中，但也写十二月，写的是他个人对腊月的感觉。

 

“腊月年光如激浪”，到了腊月，年光不仅飞逝，更如激浪汹涌，一浪拍一浪，一天催一天，所谓“急景凋年”。“激浪”，这不仅是欧阳修的感觉，也是我们对腊月的体验，随着年关将至，日子忽然急迫，赶着完成工作，腊月二十以后，扫灰、祭灶、办年货、蒸馍……怎一个“忙”字了得！

 

冻云暗淡，枯树凝寒，欲雪的天气。“谢女雪诗真绝唱，无比况，长堤柳絮飞来往。”是日，果然降雪，欧阳修与朋友走在长堤上，大雪纷飞，他想起东晋谢安与子侄辈咏雪的典故，并盛赞谢道韫将白雪比作柳絮，称之为“绝唱”、“无比况”。

 

当此雪景，欧阳公大发豪兴，“便好开尊夸酒量”，可是为什么“酒阑莫遣笙歌放”？对酒当歌，人生几何，这不是该有的旷达态度吗？他说：“此去青春都一晌，休怅望，瑶林即日堪寻访。”因为每首诗都是哀歌，每首歌都是关于失落。

 

青春都一晌，休怅望过去，休忧虑将来。过去还在，未来会来，不如做“现在”的臣仆，也是皇帝。

元 檀芝瑞《雪竹图》

还有两周就要过年，太久没有回家了，最近老是梦见父母。前天晚上睡不着，黎明时分渐入朦胧，梦见妹妹对我说：“咱妈死了。”我拒绝相信，心里十分震惊，怎么可能？同时似乎又很确定这是真的，只是以拒绝的姿态对抗这个噩耗。我从震惊中醒来，原来是个梦，这样的梦做过好几次了，听说梦见父母去世预示会有好事发生，打电话回去，父母一切安好，足以安慰。不过我在想，这是个梦，但将来的某天也会有此情景，那时便不再是梦，也许是个更大的梦，那么所谓活着，就是在梦境之间不停地穿梭。

 

晚上出去散步，从山坡上俯瞰，夜色中的湖面映着天光，清亮如镜，四围湖岸，迤逦万家灯火，“这样的人间，多么久远。”我听见自己说，声音好像来自另一个星球。
