**Taskade**

Mac/Win/iOS/Andriod/Chrome/Firefox

免费

Taskade 是一款**多平台的待办清单应用，**能将你的各种任务有序组织起来。此外，软件的图标和界面都非常美观。

![](.evernote-content/B4489F86-1753-4248-B121-E03C60311C53/BA92F919-880A-4C62-B4AA-1F853C6F896B.jpg)

在 Taskade 中，你可以为不同类型的任务分别创建**分区**，互不干扰。在每个分区中，通过新建**列表**来记录你的所有待办事项，列表还能转换为**看****板**和**表格**这两种更直观的形式。

![](.evernote-content/B4489F86-1753-4248-B121-E03C60311C53/8E0545B1-35CB-4FB0-8659-697FB3E0DD6A.jpg)

△ 表格模式

Taskade 支持修改文字样式，如设置标题级别、缩进和删除线等，所有待办事项的层级与重要程度都可以清楚展示出来。

**CleanMyMac X**

Mac

免费+付费升级；荔枝有售

CleanMyMac 3 是 **Mac 用户清理、优化电脑的不二之选**，现在，升级版的 CleanMyMac X 也上架「数码荔枝」了！

相比 3，CleanMyMac X 最显而易见的改变就是颜值更高了。不管是优雅的渐变色搭配，还是细致的动画效果，都能秒杀无数同类软件。

在功能上，CleanMyMac X 也有很大的提升：

**菜单栏工具：**在 Mac 菜单栏中添加了更多实用小组件；

**Bear**

Mac/iOS

免费+付费升级

Bear 是一款**基于 Markdown 语法的笔记软件**，曾获得过 2016 年 App Store 年度最佳应用奖和 2017 年 Apple 设计奖。

![](.evernote-content/B4489F86-1753-4248-B121-E03C60311C53/52D68473-D1AC-49CE-A406-4DCDD32BC271.jpg)

Bear 的界面简洁优雅，让写作的过程更专注、流畅；Markdown 语法与文字融洽相处，让文章内容层次分明。

初见 Bear，它可能没你想象中那么惊艳，但长时间使用后，你会发现它的各种功能设计与排版样式都做到了「**恰到好处**」。

![](.evernote-content/B4489F86-1753-4248-B121-E03C60311C53/244DD87D-F47F-4E7B-A8AF-D16F39E83407.png)

**XMind: ZEN**

Mac/Win/iOS/Andriod

免费+付费升级；荔枝有售

思维导图可以帮助我们**整理思路、发散思维**，是一种科学且高效的学习方式，而说到制作导图的工具，很多人第一反应就是 XMind。

最近，全新的 XMind: ZEN 上架「数码荔枝」，它在 XMind 8 Pro 的基础上精简部分专业功能，新增了大纲模式、深色主题等多种实用特性，还实现了导图与 Markdown 文档的相互转换。

![](.evernote-content/B4489F86-1753-4248-B121-E03C60311C53/5AD71055-8207-4174-83FE-E054392172F3.jpg)

更值得一提的是，XMind: ZEN 大幅度美化了软件界面，让制作导图的过程更专注、轻松和愉悦。

![](.evernote-content/B4489F86-1753-4248-B121-E03C60311C53/154AE564-0A50-4E50-951A-5BFB237FA987.jpg)

**Spark**

Mac/iOS

免费+付费升级

Spark 是一款**内外兼修的邮件客户端**，不仅界面美观，操作起来也很便捷、高效。

![](.evernote-content/B4489F86-1753-4248-B121-E03C60311C53/8ABB7C90-5BBD-4917-8C43-32C78A513C4E.jpg)

它可对邮件进行智能分类、快速回复和智能搜索等操作，内置的日历支持自然语言输入，让你在一个地方就能轻松查看、创建待办事项。

Spark 与 PDF Expert 等知名软件都出自于开发商 Readdle，是不少 Mac 用户的首选。
