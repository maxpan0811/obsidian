# 天道下济而光明，地道卑而上行

---

```
原文应是“天道下济而光明”（出自《易•谦卦》）  
“下济”，“谓降下济生万物”（《周易正义》）。“天道下济而光明”，意思是上天的体性是降下济物，而日月辉光因此愈加鲜明。此句以天道说明谦则亨之理。”
```
