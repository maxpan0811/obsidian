| |  | | --- | | “年产20亿片IC卡载带”项目落户淄博 |     |  | | --- | | 发布时间： 2010-06-04 信息来源： 电子信息处 字体：大 中 小 |     |  | | --- | | 9月4日，山东凯胜电子股份有限公司与新加坡德新科技有限公司合作的“年产20亿片IC卡载带”项目在山东省淄博市签约。该项目总投资5亿元人民币，占地13万平方米，将于2010年上半年建成投产，形成年产20亿片IC卡载带能力，预计年实现销售额14亿元人民币，年创利税5亿元人民币。这是淄博市为完善产业链，加强为当地IC卡、RFID封装配套，积极寻求国际合作签署的又一重大项目。  载带指的是广泛应用于IC卡、RFID、电阻、电感等电子元件自动化封装的金属质或塑胶载体，是IC卡封装的主要原材料，载带价格占封装成本的70%左右，目前国内还不能生产，主要靠进口。这一项目将填补国内空白，替代进口，成为淄博市新的经济增长点。 | |
