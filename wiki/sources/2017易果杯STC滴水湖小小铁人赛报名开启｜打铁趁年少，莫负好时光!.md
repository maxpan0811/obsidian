Kid's Master组（精英组，有计时，不分年龄组）：10:00-11:30

Kid's Challenger组（欢乐组，无计时，分为三个年龄组）：14:00-17:30

**二、比赛地点：**滴水湖 西岛

**三、比赛类型：**小小铁人三项赛

**四、参加组别：**本次比赛限5-12岁小朋友报名参加，共分为2个组别（Kid's Master组、Kid’s Challenger组）

赛制条件、比赛距离及报名费各有不同。具体如下![](.evernote-content/FCCD7808-69A9-45B3-A816-88CE6994C32B/4F62BFC5-88DC-4995-91A8-EF3ED6BBF45A.jpg)Q&A：

Q：如何计算小朋友的年龄

A：小朋友年龄=2017-出生年份。

例：小明2010年12月31日出生，年龄=2017-2010=7岁

Q：如何区分不同组别的选手

A：年龄组以颜色区分，有相对应的泳帽、号码布及赛道标识。请小朋友们记住自己的颜色组别，这将是你在赛场上最亮眼的组别标识。

1、报名条件：5-12岁身体健康的小朋友都可以报名参加（要求一定运动基础）

2、报名费：Kid's Master组：300元/人    Kid's Challenger组：200元/人

\*报名费包含赛事包、小小铁人纪念T恤，泳帽，号码布，号码带，比赛补给，奖牌、赛事运动安全险等等。

3、退费规则：在9月1日以前申请取消报名的，已缴纳部分扣除50%参赛费用后退还；在此之后取消报名的，无论任何原因不能参赛都不予退还。且仅限选手本人进行名额转移和参赛，名额不允许转让给他人。

**\*报名方式：点击阅读原文报名即可**

骑车：位于西岛内环形，单圈为1km。宽约3.5米。

跑步：位于西岛内部西侧道路，单圈折返为500米。

**八、 参赛须知**

1. 比赛装备组委会将按年龄与组别统一发放泳帽。

2. 主动确保自身状况良好，没有感冒、发热、病毒性或细菌性-扁桃体炎、其他急慢性伤病等

3. 8岁以下小朋友可以允许使用浮板、臂漂，帮助完成游泳赛段。

4. 我们要求小选手们自行携带好：自行车，头盔，非裸露脚趾的运动鞋，泳镜，铁三服或者泳衣（一件式），骑行衣物，跑步衣物，毛巾、湿衣物置物袋及替换衣物。

5. 建议携带装备：防晒用品，遮阳帽，太阳镜，补给饮料及食品。

2. 我们配备有和大铁113比赛同等专业的应急救援团队，确保每位小小铁人的安全。

3. 组委会同样为小小铁人安排了自行车技师，为小小铁人们的赛车进行赛前检查，包括头盔、刹车性能等的检查，拆除挡泥板等易造成安全隐患的自行车零部件。

4. 赛前准备时，家长可以进入换项区，协助小小铁人进行比赛准备。请家长和小小铁人们注意看好各自的装备，避免混淆或干扰其他参赛小选手。

5. 比赛过程中，家长可以全程在赛道周围观看比赛，但是不能进入赛道，否则可能造成安全隐患，或影响比赛正常进行。

6. 比赛现场设有专门的赛事医疗点，为小小铁人提供应急医疗处理。

7.比赛活动期间草坪上人员众多，请家长们注意看顾小朋友的安全。

**九、 参赛流程**

// 赛前篇 //

**住宿****（![](.evernote-content/FCCD7808-69A9-45B3-A816-88CE6994C32B/2CDA4909-8609-4867-BAB5-B2297F20E604.png)详见今天微信公众平台第二条）**

 

**交通及停车**

（一）交通：

自驾：可导航至上海市浦东新区临港滴水湖西岛。

地铁：可乘地铁16号线至滴水湖站（2号口出），出站后约1公里左右路程至西岛。

 

（二）停车：

比赛现场停车场有4个区域： 

1、环湖一路观光车道停车位（西岛南侧）。9月17日西岛北侧引桥至水上运动俱乐部路段禁止停车。西岛南侧引桥到水上运动俱乐部，经海港大道段观光车道，停车位约600个。

2、水芸路地面免费停车场（龙竹路）：入口在水芸路龙竹路上，约200个车位。

3、水芸路地下路付费停车场（水芸路）：入口在水芸路瑞木路上，约100个车位。

4、水芸路地面免费停车场（石松路）：在山兰路石松路入口，约200个车位（近滴水湖地铁站）。

![](.evernote-content/FCCD7808-69A9-45B3-A816-88CE6994C32B/128A392D-0152-44BC-BB82-F301CD28B733.jpg)

\*自行车运输：本次组委会不设自行车运输接驳。请家长自行将所需参赛物品带至滴水湖西岛。

**检录：**

1.记得带上家长和小朋友报名登记时提供的证件哦！

**赛事包领取时**，每个小朋友将得到：号码布1张，号码带个；泳帽一个，小小铁人赛事包1个；body mark1张，免责申明1份（家长现场签字后交还组委会），赛事规程1份。自行车检录单1张。头盔贴3张、自行车贴1张。加油棒2个。

**下一步**，将自行车推至换项区，接受自行车检查通过后，将自行车和头盔及其他换项物品（号码布，自行车水壶、鞋子，骑行服、跑步服装等），放置在自己号码对应的换项位置上（会用参赛号码标识）。做好游泳前的准备。

 

**换项区是差时开放的**。请各位家长合理安排时间。

Kid's Master换项区的开放时间是8：00-9:45

Kid’s Challenger换项区的开放时间是8：00-9:45和12：00-13：45

**号码布使用规范：**检录时，先将号码布置于换项区；自行车阶段需位于背后腰部位置，跑步阶段需位于前身腰部位置。搭配号码带使用。

\*领到赛事包的小朋友请到body mark的工作人员处进行body mark，然后根据自己批次挨个按照工作人员指挥排好队等待比赛。

// 赛中篇 //

**（一）游泳**：

1、每个批次8人，按照号码编排排队下水出发，需遵照裁判指令。（出发名单会在后续报名完成后，在“上海铁人STC”公众号上发布）

2、泳姿不限，但是需要游在自己泳道，切勿占据其他选手泳道。

3、如果小朋友游泳不熟练，可以使用浮板，但是不得借助外力行进。

4、小朋友不可在比赛全程赤裸上身。

5、必须佩带组委会提供的泳帽。泳镜、泳衣需自己准备（建议穿着铁三服）。

6、游泳比赛结束后，工作人员会协助小朋友离开泳池。上岸后小朋友要一路光脚跑到换项区（有地毯保护）

 

**（二）换项区：**

1、游泳上岸之，小朋友会光脚跑到换项区（一路都有地毯），到自己自行车跟前穿戴好头盔、鞋、及其他必要物品之后，推着自行车跑出换项区，一直到“上车线”之后才能开始骑行。请注意，此时号码布应放在身体后腰位。

2、因为换项也是比赛的一部分，我们鼓励小朋友自行更换衣服、调整号码布的位置。Kid's Challenger 5-7岁组选手可允许一名家长凭家属证明手环提前进入换项区协助小朋友换项。但请不要影响换项区内正常秩序，否则工作人员可能会将您请出换项区。

3、我们准备了一男一女两个更衣帐篷，可供小选手使用。也可以在泳衣外直接套上T恤。

4、自行车离开换项区时需推行自行车离开，在“上车线”后方可骑车。完成自行车骑行后，选手同样须在“下车线”前下车，然后推行至换项区内，找到自己的位置，把自行车放好，头盔放好，换上跑步衣物然后进入跑步赛道。

**（三）自行车：**

1、所有选手必须自行携带自行车，组委会不提供自行车。

2、自行车的型号及款式不限，但是必须通过安全检查，不符合安全检查的自行车将不允许参赛。不允许使用共享单车。Kid’s Challenger 5-7岁组选手可以辅助轮。

3、离开换项区之前必须戴好头盔，且不可以在头盔内佩戴头饰硬物，安检人员也将会检查头盔，不符合安全规定的头盔也将不允许参赛。

4、我们将在换项区外设置自行车维修区，提供简易自行车维修打气服务。

5、自行车赛道一律靠右骑行，如果要超车则必须在左边，而且要摇铃或者大喊引起前面的参赛选手注意，以免造成交通事故。

6、自行车骑行过程中必须全程佩戴头盔。

7、自行车赛段必须全程佩戴号码布。号码布置于身体后侧。

8、根据现场标志的指引完成自行车赛段。遇到问题请及时求助赛道旁的志愿者。

 

**（四）跑步：**

1、跑步赛段必须全程佩戴号码布。号码布置于身体前侧。

2、跑步运动服最好选择速干的材质，千万不要选择纯棉的衣服。

3、每个小朋友都将获得一次冲线机会及留下照片。

4、每个小朋友都将获得一枚小小铁人奖牌。

**（五）等候：**

由于泳道数量所限，选手需要分批下水。可能会等候较长时间。

可前往比赛休闲区，参与现场的趣味亲子活动。

// 赛后篇 //

1、完成跑步比赛后每位小朋友都将获得一次冲线的机会，家长可以在终点处帮小朋友们拍照留念。

2、每位小朋友都会获得一枚完赛奖牌。

3、休息区有免费的蛋糕和水果供小朋友们享用。

4、比赛现场也会有很多主题活动、小游戏可以免费参与。

5、现场不设置冲淋区域。

 

**十、致家长的一封信**

1、装备推荐

![](.evernote-content/FCCD7808-69A9-45B3-A816-88CE6994C32B/29B2D58C-368D-487B-B4CD-2EDBE8F021B5.jpg)

注：我们非常建议小选手穿着铁三服进行比赛。铁三服紧身快干，可以同时满足游泳、骑车和跑步的需求，也节省了换项的时间。如果家长有需要，欢迎随时联系小小铁人赛咨询群内客服人员咨询。

2、建议饮食

孩子们应在运动前2－3小时进食一些易消化吸收的食物或流质食物，吃些高纤维饼干或是优酪乳、葡萄干、新鲜的水果等，可以减少运动时的肠胃不适。此外，运动前40分钟，可以喝300－400ml的水补充水分，不至于在运动中丢失过多的水分，而使身体疲乏。

3、当天如有小到中雨，比赛将如期进行。独立完成，不畏挑战同样是铁人精神的一部分。

4、观看比赛时，不得随意横穿赛道，以免影响运动员比赛，或对运动员和自身造成伤害事故。需要穿越赛道时，应从指定的穿行口通过，并听从管理人员指示；

5、检查脚趾甲（过长的脚趾甲容易再比赛中导致翻起、黑指甲等情况。同时不宜过短，防止指甲履盖保护下的嫩肉在比赛中被磨出血）

6、检查鞋内是否藏有沙砾碎石。

7、赛前如发现孩子有紧张情绪，可多和孩子沟通，调节呼吸放松情绪。缓解赛前焦虑。

 

**十一、不可抗力**

遇到天气或其他不可抗拒原因，组委会将根据安全风险临时决定是否延期或终止赛事活动。
