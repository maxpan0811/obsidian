# 从林彪主持会议批判邓小平说起……

1966年8月1日至12日，中共八届十一中全会在北京举行。图为毛主席主持会议

《毛泽东年谱（1949—1976）》第五卷中有这样一个条目：“同日（1966年9月5日）上午，在中南海游泳池住处召集林彪、周恩来、王任重等开会。毛泽东说：关于刘少奇同志的生活会，不要再开下去了。越开调子越高，一定要说成反党反社会主义，再讲下去就是彭罗陆杨一帮了。……”

所谓“刘少奇同志的生活会”，是从什么时候举行的？由谁主持召开的？年谱未作说明，也没有加注。其他党史资料也无涉及。其实，这是一个需要弄清楚的重要问题。

众所周知，在1966年8月12日闭幕的中共八届十一中全会上，林彪在中央政治局常委中的排名上升到第二位，代替刘少奇成为党的接班人，而刘少奇则从原来的第二位下降至第八位，不再主管具体工作。因为他犯了“路线错误”。所谓“刘少奇同志的生活会”，无疑是中央高层批评刘少奇的会议，这个会由谁来主持？笔者推测是刚刚成为接班人的林彪，是毛泽东让林彪主持这个批评刘少奇的中央政治局常委会或中央政治局常委生活会，大概会议开得不合毛泽东的意，所以被他叫停了，即“不要再开下去了。越开调子越高，一定要说成反党反社会主义，再讲下去就是彭罗陆杨一帮了”。这表明当时毛泽东认为刘少奇的“错误”性质跟已经被打成反党反社会主义的“彭罗陆杨”有所不同；更重要的是会议把批评刘少奇变成了主要批判邓小平，这违背了毛泽东的原意，所以他不让开了。这次会议，不仅影响当时的政治局势，而且涉及“文革”以后的政治生态。迄今已出版的几部关于“文革”史的著作，包括“文革”在内的党史著作，都没有专门的讨论，有关林彪在“文革”中情况的研究论著，也没有谈到这次会议。

鉴于此，笔者专门访问过一些人，也搜集了相关的材料，虽有收获，但仍有许多问题不敢定论。现仅把搜集和了解到的若干情况整理出来，抛砖引玉，期望引起党史专家的关注和进一步研究。

# 林彪确实主持过批判邓小平的中央政治局常委扩大会

关于这个问题，笔者看到的有下面几则材料：

1967年1月13日，周恩来在接见解放军参加文化部“文化大革命”工作队造反联络站代表时的讲话中说：十一中全会（中共八届十一中全会）的时间很短，而且又是集中批判刘少奇的错误，刘少奇作为毛主席的接班人，二十多年了，有许多的错误。对邓小平没有多少批评，会后常委开了一些生活会，才揭得较深，但是少数同志参加的。

《周恩来年谱（1949—1976）》下卷中记载：“（1966年）8月15日在林彪处参加中共中央政治局常委会议。”在林彪处召开政治局常委会议，当然由林彪主持；而内容也就是周恩来1967年1月13日所说的。

在毛毛著的《我的父亲邓小平“文革”岁月》一书中有这样一段话：“十一中全会结束后，毛泽东决定，由林彪主持召开一次政治局常委扩大会。会议原定继续批判刘少奇，但林彪、江青等人认为，刘少奇已实际被打倒，目前的主要危险和最大障碍是邓小平，于是将会议的矛头指向邓小平。他们不但刻意组织人员批邓，林彪还亲自出马，将邓的问题说成是敌我矛盾。”

上述这几条材料证明，在1966年中共八届十一中全会后，林彪确实主持过中央政治局常委扩大会，内容主要是批判邓小平。至于是常委扩大会，还是常委生活会，会议称呼不影响问题实质。

# 林彪主持这次常委扩大会的缘起

“十一中全会结束后，毛泽东决定，由林彪主持召开一次政治局常委扩大会。”而毛泽东为什么决定“由林彪主持召开一次政治局常委扩大会”呢？没看到有人、有材料解释这个问题。笔者曾听中央文革小组成员关锋不止一次说，这次会议是陶铸建议召开的。陶铸之所以建议召开这次会议，是因为他在1966年5月政治局扩大会议上被决定调中央书记处任常务书记，中共八届十一中全会批准了这个决定。在其位就要谋其政。作为中央书记处常务书记的陶铸，为了马上开展中央书记处的工作，向毛泽东提出了这个建议。关锋说，陶铸这个建议不一定符合毛泽东的心愿。当时，毛泽东对邓小平不满，认为邓小平主持的中央书记处成了“独立王国”，他成立隶属中央政治局常委会的中央文革小组，就是要代替书记处。

1967年2月12日，毛泽东就曾找中央文革小组的张春桥、王力、姚文元、戚本禹谈话，批评中央文革小组成了“独立王国”，说：“小组成立以来，没有一次提出要我主持开会，去年6月（中央文革小组是1966年5月28日中共中央发通知宣告成立的）以来，代替了书记处，也是独立王国。骂别人独立王国，自己独断专行，否认政治局常委存在。”在毛泽东要求中央文革小组代替中央书记处的情况下，陶铸也许不了解毛泽东的意图，还要中央书记处开展工作，提出了召开常委会的建议，毛泽东便让林彪主持召开了常委扩大会。关锋说，中央文革小组成员参加了这次会议，他本人也是参加者之一。

关锋关于这次常委会缘起的说法是否可信，我们见不到别人和别的材料说明召开这次常委扩大会的理由，关锋的说法姑且算作一说——其实际情况并不一定是这样的。

《毛泽东年谱（1949—1976）》第五卷中，“（1966年）8月下旬”的条目中说：“经毛泽东同意，组成中共中央政治局常委（扩大）碰头会。吸收中央文革小组全体成员参加，讨论党政业务和文化大革命运动。”该条目的最后一句话说：碰头会一般由周恩来主持。在这个地方，下面加注说：“1956年中共八大以来处理中央日常事务的中央书记处，在八届十一中全会以后基本停止活动。”这说明中央书记处在当时的中央决策层中已经没有什么地位了。因此，林彪主持的这次中央政治局常委扩大会根本不是讨论中央书记处的问题。

# 林彪主持这次常委扩大会的内容和影响

关于这次中央政治局常委扩大会的内容，参加会议的另一原中央文革小组成员王力作过比较详细的回忆。他说：

十一中全会后开了政治局扩大会（中央文革小组成员参加），也叫“生活会”，批评刘少奇。新常委都到了，陈云没到。文革小组成员中部队的几个人没参加，包括刘志坚、谢镗忠，加了叶群等人。这个会本来没有说批邓，可是江青发起，林彪响应，说主要危险不是刘，刘已被毛主席一张大字报打倒了，现在主要危险是邓，因此这个会变成了刘邓并列。据我了解，这同毛泽东原来的设想不一样。会上，对刘少奇进行了批评，刘少奇作了检讨。林彪对刘少奇的批评也并不那么尖锐，除一般的话外，只是说历史上在苏区刘少奇眼里根本就没有林彪这些人。（那时刘是红三军团政治部主任，林是红一军团司令员。但刘当时就是中央政治局委员，林还是年轻干部。）

林彪对刘少奇的批评不怎么尖锐，但对邓就讲得很重，特别是把邓同吴晗、彭真及北京市委联系起来。吴晗是社会主义的敌人，邓小平的问题性质是很严重的，是政治性的。意思就是敌我矛盾，总之说得很凶。而且林彪公然提到野战军问题。……这次会毛主席要批刘少奇，江青要把刘邓并列，以邓为重点。在江青看来，邓是掌握军队的，是她掌权的障碍。这点江青同林彪一致。林彪认为能同他争权的不是刘而是邓。……我觉得在毛主席心目中，一直是刘邓要分开。……邓在常委中还是有分工的，他管中联部、中调部，还有一个重要的分工是“彭罗陆杨”的专案审查委员会。可是这个会上林彪把邓说成那样，邓没法工作了。

王力的这段回忆，有一处可能不准确。他说“政治局扩大会”，即不同于笔者引用的其他材料，也和他自己的说法相矛盾。他说这次会议“新常委都到了”，没说政治局委员都到了，事实上不是常委的政治局委员，如陈毅、谭震林、叶剑英等都没到。所以，这次会议不是政治局扩大会，而是常委扩大会。

王力回忆说：这个会前，叶群跟有些人谈话，张春桥跟王力等人谈话，动员批邓。江青说刘少奇没什么，一篇大字报就打倒了，最危险的是邓。……关锋、戚本禹、张春桥、姚文元对邓都不熟悉。江青说，王力等人是党的优秀干部，在这个问题上你们要发言。张春桥还对我说：主席起初不同意你为文革小组成员，说你是彭真的人。他说这个话是压我批邓。但会上我还是没有发言。为此江青一直恨我。有人发了言，我认为他是很勉强的，是应付之辞。他发言时满头大汗，说明他不是出自内心的。有些人按照林彪的指示，讲邓讲得多。从邓讲到李井泉，讲到西南局，讲到贺龙，这样毛主席就把这个会停了。

王力在另一处回忆时又说：“十一中全会后开常委生活会，毛主席指示要批刘，林彪、江青他们以批邓为主，而且把邓提到敌我矛盾，邓小平不能工作了，把他管的工作交给康生。这个会牵连到贺龙的时候毛主席把这个会给停掉了。”

关锋也说：在这次会议上，有些人对邓小平批评很厉害、很尖锐。会后，据说向毛主席汇报这次会议的情况时，毛主席很不满，十分生气地说：“不开了！再开就是彭罗陆杨了！”关锋说，他当时也不懂这是什么意思，大概是毛泽东不同意这样批邓，当时刘和邓都在中共八届十一中全会上重新当选为中央政治局常委，同已经被打倒了的“彭罗陆杨”还有所不同。

关于这次会议，邓小平的反应和对他的影响，在毛毛著的《我的父亲邓小平“文革”岁月》中有这样的描述：

“父亲在会上受到不公正的甚至是污蔑性的批判，心理一定是不平衡的。回家后他虽什么也没说，却夜不能寐。母亲看见他卧室的灯深夜不熄，便去问他：‘三点多了，怎么还不睡’父亲告诉母亲：‘今天晚上开会已经从批刘少奇转向给我提意见了’母亲问：‘谁批你’父亲只说了一句话：‘军队的人’父亲不再多言，母亲也不敢多问，只是安慰地说：‘快睡觉吧，不然明天开会起不来了’父亲知道，这次批判后，他的‘错误’的性质，不再只是派工作组‘镇压’群众，而是历史在内新账老账一起算了。

“会后，父亲被迫停止了工作。他将一部分原来由他分管的中联部、中调部等工作交代给康生，说：我的工作交给你，我不能工作了。”

这次常委扩大会，毛泽东不让开了，林彪主持的这次会议也就这样结束了。《周恩来年谱（1949—1976）》下卷在注中说：“中共八届十一中全会后，原按毛泽东意图中央工作由林彪主持，但林彪主持过几次会后就不再管，中央日常工作由周恩来负责。”

综上所述，大略可以看出林彪主持的这次批判邓小平的政治局常委扩大会的轮廓。这次会议的召开和停止，导致了三个重要的后果：接过了刘少奇在中央政治局常委中排名第二位次的林彪，此后又重新退居幕后，不再继续主持中央日常会议；此后由周恩来主持中央日常会议；邓小平在中共八届十一中全会后还分管一部分工作，此后则不再分管工作。当然，还有很多情况不清楚，诸如，毛泽东究竟为什么决定开这次会？关于这次会的内容和开法，毛泽东有怎样的指示和布置？在会上有些人批邓究竟讲了些什么？为什么一涉及贺龙，毛泽东就把这个会停了？为什么这个会“再讲下去就是彭罗陆杨一帮了”？为什么这个会不继续开了？……这些问题还需要进一步探索。

(来源《党史博览》）
