# 精益求精更完美 大疆Phantom 3飞行器评测

随着无人四轴飞行器技术的逐渐成熟，它们已经逐渐从少数人手中的玩物变成了非常具有实用性的工具了。四轴飞行器体积轻小，动作灵活，还可以稳定悬停在空中，可以说是递送物品、侦察、航拍的神器。  
  
　　在市面上海量的四轴飞行器中，我们的大疆毫无疑问是最受全世界爱好者喜爱的品牌之一，因为它旗下的飞行器产品都具有廉价、易用、性能强劲这些优点。大疆最新的 Phantom 3 和前辈们相比或许没有革命性的变化，但许多新的改进却带来了更好的体验。那么就让我们今天来看一看：  
  

![](.evernote-content/3F137EE6-276A-492C-978D-99A3FF91E986/0A4B71D7-DE52-4200-9BED-936AFB3877D8.jpg)

  
　　对于一款四轴飞行器来说，最重要的一项特性是它在空中是否能听你的指挥，毕竟一台电脑和一部飞行器都可能会出问题，不同的是后者会直接从天上掉下来。在这点上 Phantom 3 表现很好，经过将近一个月的试玩，还没有遇到不听指令的现象。最值得一说的是，市面上绝大多数四轴飞行器都只依靠 GPS 来定位，但 Phantom 3 除此之外还支持俄罗斯的 GLONASS，这意味着它能够找到更多卫星信号，搜索的速度当然也就更快。  
  
　　当你实在找不到卫星信号（比如在室内玩）的时候，Phantom 3 利用前置摄像头的视觉定位系统来稳定自己。这个技术最早出现在大疆的“黑科技”Inspire 1 上，但 Inspire 1 太贵，所以能在 Phantom 3 上找到它真真是极好的。即便 Phantom 3 相对前者更廉价，但视觉定位的表现没有打折，被推一下以后还能复位。不过，视觉定位技术好用的前提是周围的景象对比度要高，如果附近的表面颜色单调，那它的表现就不太好了。  
  
　　每到一处新地点，放飞它后首先要做的第一件事就是调校指南针，也就是让它在 X 轴和 Y 轴上分别转一圈。如果调校失败，那就表示周围的电磁辐射较强，需要再来一次。一旦调校完成，Phantom 3 的飞行就会非常稳定，你甚至可以尝试让它着陆在自己手上，不需要什么高超的操作技巧。  
  
　　Phantom 3 总有有三种型号，主要的区别在安置的摄像头上。评测所使用的型号是最好的 Professional 版，使用它能够得到绝佳的航拍体验。Phantom 3 Professional 能够拍摄 4K 视频，而且能以 1200 万像素的标准拍摄静态照片。如果你让飞行器悬停，它拍摄的图像稳定得就好像是在三脚架上一样。不过超高的稳定性带来了一个诡异的缺点：当你水平旋转飞行器时，拍出来的视频在运镜上会给人一种僵硬呆板的感觉。  
  

![](.evernote-content/3F137EE6-276A-492C-978D-99A3FF91E986/17BCFE47-48B3-4F94-9525-37093AEDDB77.jpg)

  
　　说到非常重要的续航，Phantom 3 相比 Phantom 2 更有进步。经过计时，前者的滞空时间能达到 20-25 分钟，相比原有的 15-20 分钟稍有提升。然而，新的电池意味着它不兼容旧款产品。如果你为以前的 Phantom 飞行器购买过电池，那很遗憾现在用不上了。  
  
　　Phantom 3 的遥控器内置名为 LightBridge 的视频下行系统，意味着这个原本是额外配件的设备现在终于被整合在遥控器里了，这对便携性来说是个大进步。新的设计意味着信号传输范围被大大增强，即便已经飞到了 100 多米高，手机上仍能接收它传送来的视频信号。凭借 LightBridge 的强大性能，你甚至能在飞行器仍在空中时对视频进行编辑和分享，甚至是直播 —— 虽然直播稳定性并不太好。  
  

![](.evernote-content/3F137EE6-276A-492C-978D-99A3FF91E986/517ECCC2-815A-40C9-8FCC-ABB3530481EF.jpg)

  
　　对于那些想烧四轴飞行器但却犹豫不决的玩家们来说，他们最大的担忧是莫过于“我那么手残，不小心玩坏了怎么办”，毕竟大疆飞行器再怎样一部都要好几千块钱。这一点完全不用担心，因为一键起飞和一键降落都已经是很成熟的技术了。Phantom 3 的应用里有回家功能，能够帮助它寻找遥控器所在位置，并自动飞回来降好。如果你已经看不到它或是有点心慌，使用这个功能它就会马上回来。Phantom 3 自动降落的精度能达到 2 米以内，即便附近是高高的草丛它也能安全降落。如果你突然改变主意，可以在任何时候切换到手动操作。  
  
　　**总评**  
  

![](.evernote-content/3F137EE6-276A-492C-978D-99A3FF91E986/EE79ABCF-CA34-4D65-8247-AD8157A4F3A6.jpg)

  
　　大疆 Phantom 3 虽然不像 Inspire 1 那样亮点满满，但过硬的素质让它充满了极高的实用性。它飞行稳定，操作品质好，内置先进的视频传输技术，对于一款消费级的飞行器产品来说这些已经足够。  
  
　　对于专业摄影师或者记者来说，这样的飞行器可以打开新的可能性，拍摄下以前无法拍摄的影像；对于普通的爱好者来说，扎实的素质自然是可玩性的体现；而对于那些初学者，易于上手的特性就是一切。
