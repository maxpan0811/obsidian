# 引言：看不清力量结构，就无法制定战略

一个组织最常犯的错误，不是判断失误，而是**根本不知道谁在影响局势**。

- 有时你以为客户在主导，其实是政府；
- 有时你以为组织已达成共识，其实是表面统一、内部各怀心思；
- 有时你推了正确的战略，最后败在一个你没考虑到的“沉默角色”。

换句话说：**你不是输在了战略上，而是输在了“对力量分布的误判”上。**

毛泽东早在1938年批注李达《社会学大纲》时，就高度关注这一点。他提出：

“社会中各阶级力量不等，不可同视。事成与否，系于力量分布。”

他用“阶级分析法”建立了一种“力量识别模型”——用以判断谁是敌人、谁是朋友，谁能争取、谁该斗争，谁在关键时刻能左右胜负。

![](attachments/7d8218cc7880252e.png)

这一分析逻辑，后来成为中共战略部署的底层方法，也为今天的组织战略提供了重要启示：**不要盲目上战略，先看清谁在场、谁关键、谁沉默、谁转向。**

---

# “阶级分析”不是意识形态，是战略识别系统

在阅读《社会学大纲》时，毛泽东并不满足于接受“社会结构三分法”，他提出：

- 阶级不是身份，而是**基于利益而形成的动态结构关系**；
- 同一角色在不同情境下可能转换；
- **矛盾的实质，是不同力量对生存空间与主导权的争夺**。

毛泽东强调，不是看一个人的“出身”，而是看其在**当下结构中起到什么作用**。

例如：

- 富农在土地革命中是主要斗争对象；
- 但在抗战时期则可能成为可以争取的力量；
- 而地主阶级内部也可能分裂，有的妥协、有的顽固。

这种分析背后，是一种类似今天“Stakeholder Mapping”（利益相关者映射）的战略框架。

我们可以对比一下：

|  |  |  |
| --- | --- | --- |
| 概念 | 毛泽东的阶级分析 | 企业的战略识别 |
| 分析对象 | 各阶级群体 | 客户、员工、股东、监管、舆论、竞争者等 |
| 判断维度 | 利益冲突 + 动态位置 + 政治属性 | 影响力 + 立场 + 可转换性 |
| 应用结果 | 联合/斗争/中立策略 | 协同/谈判/防御/转化策略 |

这不是简单的“站队分析”，而是一种看清力量结构的战略起点。

---

# 毛泽东的“力量地图”怎么画？

毛在一篇又一篇的批注与讲话中，逐步形成一套“力量识别+策略匹配”的分析路径，核心有三步：

# 1.谁在场？（Stakeholder Identification）

不是泛泛地讲“群众”“上级”“资本”，而是：

- 谁是当前局势中的关键变量？
- 哪些角色长期沉默却可能成为支点？
- 有没有被忽视但能起决定性作用的“中间层”？

# 2.谁关键？（Power vs. Position Mapping）

毛泽东极其重视\*\*“决定性的少数”\*\*。他认为有些阶层虽小，但其位置和转向会带动结构剧变，如小资产阶级知识分子、士绅、基层干部。

# 3.策略匹配（Align or Split or Push）

- 可以联合的要主动靠拢（统一战线）；
- 可分可争的要设法转化（瓦解敌军）；
- 明确对抗的要准备斗争（正面交锋）；

这三步合起来，就是一个**力量识别+动态博弈+资源调动**的战略地图构建方式。

---

# 从“群众路线”到“组织战略图谱”

毛泽东提出的“群众路线”，其本质不是“靠人多”，而是**通过结构性识别与合力重构来实现系统逆转**。

这一逻辑，其实可直接类比于企业中的“组织战略图谱”构建。

例如，一个企业若要推动AI系统落地，不能只看项目本身，还要问：

- 谁是核心推动者？谁是沉默阻力？谁是影响舆论的关键小团体？
- 谁的支持最容易“撬动氛围”？谁的否定会产生“连锁动摇”？
- 谁在表态？谁在观望？谁可能“变脸”？

你必须像毛泽东分析“地主、小资产阶级、民族资产阶级、农民、工人”一样，逐个画像、分析位置、判断可转化性，最终制定出适配的路径——这才是真正的战略，而不是通用模板的复制。

---

# 战略不是资源配置，是结构重构

毛泽东曾说：

“一切战略问题的核心，是力量配置问题。不能掌握力量，就无战略。”

这句话，放在今天，仍然适用。

企业战略不是“搞点增长方案”，而是对力量结构的重新编排：

- 重新组织产品线背后的资源权重；
- 重构客户结构中的主导人群；
- 重建组织内部支持变革的行动联盟。

因此，真正有穿透力的战略往往不是靠“大投入”完成，而是靠对“谁在推动、谁在拦路、谁在动摇”的精准识别与策略组合完成。

毛泽东通过阶级分析训练出来的，不仅是政治判断力，更是一种适用于复杂系统中的**结构操盘术**。

---

# 结语：战略的第一步，不是找机会，而是识别力量

我们今天太容易被“趋势”、“赛道”、“增长点”所吸引，而忽略了一个最本质的问题：

**你站在谁的对面？谁站在你身后？而你真正能动用的力量有哪些？**

这不仅适用于国家、政党，也适用于一个正在转型、创新、甚至挣扎的组织。

毛泽东留下的“阶级分析法”，远不是过时的历史语言，它是一个今日企业战略仍可借鉴的“力量识别工具”。

它提醒我们：

- 不要幻想组织中每个人都是一条心；
- 不要把所有反对声音都当成破坏；
- 不要在不知道“谁真正关键”的情况下贸然部署；

在变革时代，战略不是计划，而是**对结构的洞察，对人心的排列，对力量的精准调配。**
