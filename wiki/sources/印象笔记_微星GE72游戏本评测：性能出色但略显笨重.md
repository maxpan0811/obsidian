# 微星GE72游戏本评测：性能出色但略显笨重

---

[![](.evernote-content/766B42A2-0824-4AD3-BF77-C5CE83A7705F/0B73BC2C-FFBA-4FBF-9F95-03DC1F713E32.jpg)](http://tech.feng.com/2015-12-30/Msi-GE72-game-this-review-excellent-performance-but-slightly-bulky_634595.shtml)

　过去，人们看到游戏笔记本电脑的普遍反应是“你为什么买这货?”，而现在，人们通常会问“你在哪儿买的?”。随着游戏本性能的不断提升和价格的不断下降，游戏本正在缩小和台式机之间的差距。  
  
　近日，外媒 TechRadar 为我们带来了微星 GE72 Apache Pro 游戏本的评测，这款产品的市场售价为 1299 美元(约合人民币 8364 元)，我们来看看它到底值不值这个价。  
  
　**外观设计**

![](.evernote-content/766B42A2-0824-4AD3-BF77-C5CE83A7705F/0B73BC2C-FFBA-4FBF-9F95-03DC1F713E32.jpg)

微星 GE72 Apache Pro 是一款 17 英寸的游戏笔记本电脑，其机身尺寸为 419×280×29 毫米，比华硕 ROG 要厚一些。在造型方面，微星 GE72 Apache Pro 的设计灵感来自现代跑车，这似乎已经成为了游戏笔记本电脑的一个趋势。它的边角设计显得非常霸气，当盖子合上的时候，整个笔记本电脑看起来有点像某款跑车的引擎盖。

另外，这款游戏本以黑色为主色调，拉丝铝材外壳能够避免沾染指纹和油渍。虽然触控板由塑料材质打造，但是仍然模仿着铝材的外观，并不会给人一种不协调的感觉。因为尺寸够大的原因，支持背光的键盘能给人非常不错的手感，打起字来也非常舒服。

**硬件规格**

![](.evernote-content/766B42A2-0824-4AD3-BF77-C5CE83A7705F/0AC8A548-8FFD-44D4-B897-95B8D1796949.jpg)

这款 GE72 Apache Pro 游戏本拥有一块 17.3 英寸 1920×1080 全高清显示屏，微星公司还给它搭载了主频为 2.6GHz 的第六代英特尔 i7-6700HQ 四核处理器、英伟达 GeForce GTX 960M 显卡以及 16GB 的运行内存，并且还配有一块 1TB 的机械硬盘和一块 128GB 的固态硬盘。

![](.evernote-content/766B42A2-0824-4AD3-BF77-C5CE83A7705F/638FF5B5-51CE-4644-9549-FCCC8D94FFA8.jpg)

接口方面，这款游戏本拥有两个 USB 3.0 接口、一个 USB 2.0 接口、一个 HDMI 端口、耳机以及麦克风插孔、迷你 DisplayPort 以及 USB-C 接口等，用户完全不需要担心连接性方面的问题。

**性能体验**

![](.evernote-content/766B42A2-0824-4AD3-BF77-C5CE83A7705F/059CB022-1EDE-4033-8BB2-341354B0B662.jpg)

TechRadar 的测试人员表示，这款游戏本完全能够满足游戏玩家的需求，得益于英伟达 GeForce GTX 960M 显卡的出色表现，玩家能够在游戏过程中将游戏的画面等级调得比想象中要高，不仅如此，17.3 英寸显示屏还能带来清晰、色彩丰富的画面效果。

散热方面，测试人员称，即使将微星 GE72 Apache Pro 放在腿上玩《辐射 4》也不会觉得发烫，不过，重达 2.69kg 的机身并不适合长时间放在腿上玩。另外，这款游戏本还配有一个可以手动打开的散热风扇，不过，测试人员除了启动它进行测试之外，其他时候并不需要用到它。

需要提到的是，微星 GE72 Apache Pro 的扬声器能够给玩家带来相当不错的音效体验，即使将音量调到最大也不会出现失真的情况。

**总结**

![](.evernote-content/766B42A2-0824-4AD3-BF77-C5CE83A7705F/5C4E355D-E763-4A44-9997-32902C511CE8.jpg)

综上所述，微星 GE72 Apache Pro 的性能算不上顶尖，但是在性能和便携性之间找到了一个不错的平衡，它比技嘉 P55K V4 要大一些，但是却比华硕 ROG G751JY 要轻。

如果你想要摆脱巨大的台式 PC，但是又想要获得出色的音质和画面效果的话，微星 GE72 Apache Pro 会是一个不错的选择。此外，第六代英特尔处理器和英伟达 960M 显卡完全能够应付接下来几年内推出的游戏。
