# 《仙剑6》PC配置详解：32位和XP都悲剧

![](.evernote-content/9F74DA63-F3C7-4AE2-8EA2-0793B190E5FC/167A20F5-B9E8-43A7-AB80-765BBD0C3677.jpg)

  
　　随着7月8日上市日期的临近，《仙剑6》近日陆续曝光了加入“FPS”的游戏模式、女BOSS等，而且最经典的一代也刚刚上线iOS，售价60元。  
  
　　当然，玩家们肯定还是关心PC版能不能爽完，对此，仙剑官网更新了配置答疑——  
  
　　近日收到部分玩家反馈“32位操作系统，运行市面上部分游戏，游戏中出现中途退出等异常状况”，也有很多玩家对于自身电脑配置并不了解，陆续接到很多玩家向我们询问未来《仙剑6》是否会出现类似状况。我们在与开发组充份沟通后，**确认如下：**  
  
　　1：由于32位操作系统在内存使用方面的限制（不支持高内存），在运行高画质游戏时无法充分发挥内存效率，确实更易出现一些问题。  
  
　　2：每个人的电脑硬件配置和软件设置不同，使用32位操作系统体验游戏出现问题的概率更高一些。  
  
　　综上，如果您正在使用32位操作系统，曾经在部分游戏中出现“中途退出、无法开启”等情况，那么我们强烈建议您升级至win7以上、64位操作系统。  
  
　　**建议XP系统升级到WIN7或以上系统：**  
  
　　由于微软已经停止对XP系统的正常服务，同时考虑到个人电脑软硬件环境的多样性，经过与开发团队的慎重沟通后确认，要想100%保证XP系统的玩家顺畅、高质量的体验游戏，必然导致要向下兼容而牺牲掉一部分游戏画面质量和游戏玩法，这对于一款游戏来说显然是得不偿失的。因此我们建议还在使用XP操作系统的玩家在游戏上市前进行系统升级，升级至64位的win7或win8系统。  
  

![](.evernote-content/9F74DA63-F3C7-4AE2-8EA2-0793B190E5FC/2108D9E9-075F-4B01-9BC5-08C4D5263B95.jpg)

  
　　**标准配置**  
  
　　- 操作系统：Windows XP/7/8/8.1；  
  
　　- 处理器：Intel Core 2 Duo 3.0GHz或AMD同性能产品；  
  
　　- 内存：4GB；  
  
　　- 显卡：512MB显存，NVIDIA GeForce 9800或AMD同性能产品；  
  
　　- 硬盘：16GB  
  
　　- 辅助工具：一段安静的时光、风扇、西瓜  
  
　　**推荐配置**  
  
　　- 操作系统：Windows XP/7/8/8.1；  
  
　　- 处理器：Intel Core i5 2.8GHz或AMD同性能产品；  
  
　　- 内存：4GB以上；  
  
　　- 显卡：1GB显存，NVIDIA GeForce GTX560或AMD Radeon HD6870及以上；  
  
　　- 硬盘：16GB  
  
　　- 辅助工具：一段安静的时光、空调、冰镇西瓜
