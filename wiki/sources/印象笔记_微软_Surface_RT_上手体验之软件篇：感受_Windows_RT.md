今天爱范儿在上海全程直播了微软 Windows 8 中国发布会。在发布会结束之后的 Demo 环节中，我们对 Surface RT 进行了充分的体验。这次中国发布会也是今年 6 月 19 日 Surface 首次亮相之后的第一次正式展示。

发布会上，微软全球 Windows 与 Windows Live 事业部总裁 Steven Sinosky 强调，笔记本电脑和平板电脑同样都是 PC，除了操作方式不一样，微软要为他们提供同样的体验。对于微软自家推出的平板电脑 Surface 系列来说，统一 Surface RT 和 Surface Pro 上的体验，就需要在完全不同的硬件上，统一 Windows 8 与 Windows RT 的操作感受。
