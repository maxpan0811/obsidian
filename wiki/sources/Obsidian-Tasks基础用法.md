**摘要**: Obsidian Tasks 插件的基础用法教程。涵盖安装、两种创建方式（Markdown 语法和命令行弹窗）、任务属性（优先级/截止/计划/依赖）、Checklist 插件配合使用。文中提供中英文对照的创建弹窗字段说明，适合新手入门。

## 核心观点

- Tasks 是 Obsidian 最强大的任务管理插件，支持 Markdown 语法和可视化弹窗两种创建方式
- 基础用法可满足 80-90% 日常需求，无需学全部语法
- 配合 Checklist 插件可通过 #todo 标签在侧栏搜索管理任务
- 支持优先级（最低/低/普通/中/高/最高）、截止日期、计划日期、依赖任务等属性

## 关键细节

- **安装**: 社区插件市场搜索 tasks → 安装启用 → 重启 Obsidian
- **方式1**: Markdown 语法 `- [ ] 任务描述 📅 2026-04-02 ⏫ #todo/project`
- **方式2**: Ctrl+P → 输入 tasks → Create or edit task → 可视化弹窗填写
- **常用符号**: 📅截止 ⏫优先级高 🛫开始 ✅完成 #todo标签
- **Checklist 插件**: 配合管理带 `#todo` 标签的任务
