> 年度出版物，基于ITB买家俱乐部中国区1,400+买家代表调研。合作伙伴：牛津经济研究院旅游经济学、携程集团。

## 买家构成

| 业务类型 | 占比 |
|---------|:---:|
| 主题及定制游 | 20% |
| 团队旅游/批发商 | 16% |
| 会议和活动策划方 | 12% |
| 奢华游 | 10% |
| 奖励旅游旅行社 | 9% |
| 在线旅行社 | 8% |
| 企业买家 | 5% |
| 差旅管理公司 | 3% |
| 其他 | 2% |

## 报告内容

含目的地旅游局、行业协会、知名旅行社、MICE买家和企业买家等多方高管访谈。涵盖市场趋势、消费变化、行业挑战等。
