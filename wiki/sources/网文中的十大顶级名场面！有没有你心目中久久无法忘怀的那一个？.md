热衷网文的少年们，有没有在想起看过的网文中那些热血沸腾的名场面时，在深夜辗转反侧，久久无法忘怀呢？今天一起重温十个网文最震撼人心的片段，有不同见解的书友可以在评论区留下你的心头好，大家一起交流讨论，说不定就能遇到你喜欢的呢！

# 第十名 《武动乾坤》 请林动师兄，壮我道宗！

那时的土豆尚在神坛，依旧年少热血！这句话一出，众多书友热泪盈眶！

# 第九名 《遮天》 叶凡圣体大成了！

当年众多书友奔走相告，人族终于出了出了一位能比肩大帝的人物，甚至打出了现实伤害，当天贴吧爆炸，论坛瘫痪，点娘维护，简直恐怖如斯！

# 第八名 《凡人修仙传》 韩立飞升仙界

作为凡人流的开山鼻祖，这个场面主打的就是一个功成名就。毕竟，谁不想成为韩老祖呢！

# 第七名 《第一序列》 七次感谢自己

绝境时刻，任小粟还差七次感谢才能解锁武器，于是他七次感谢自己。“我要七次感谢我自己。”“第一次，我感谢自己面对机会时，从不怯弱。”“第二次，我感谢自己面对危险时，从不畏惧。”“第三次，我感谢自己面对磨难时，从不妥协。”“第四次，我感谢自己面对诱惑时，总有底线。”“第五次，我感谢自己从不虚伪。”“第六次，我感谢自己清醒如初，从不迟疑。”“第七次，我感谢自己在生活的泥潭里，一路高歌，披荆斩棘！”

# 第六名 《仙逆》 王林三次化凡

第一次感悟凡人生老病死，第二次陪伴王平一世轮回，而第三次则是以梦道之术，开启大梦一场！特别是第三次，麻子哥一生杀戮无数，反倒是平凡时弥足珍贵。而也正是这三次化凡，成为了本书的点睛之笔！

# 第五名 《雪中悍刀行》 小地瓜，我找到你了

雪中的名行面何其多，且不提十八宗师齐聚拒北城，慷慨赴死的悲壮，还有徐骁嫡长子徐凤年，在此求死的意气！只说徐凤年听到面前的小乞丐说“我的名字是徐念凉，绰号叫小地瓜”时，真真让人泪奔！  
![](attachments/be4c10def89cb26e.jpg)

# 第四名 《诡秘之主》 七神默许，愚者反对

这是克莱恩的巅峰一战！亚当蜕变为“空想家”需要一场席卷世界的战争，会导致贝克兰德大雾霾事件和无数平民死亡。乔治三世晋升为“黑皇帝” 要驱使大量民众建造陵寝，将导致民众生活困苦。面对七神的默许态度，克莱恩问自己，这样的安排，这样的结局，你接受吗？不，我不接受！就算没有意义，有些事情还是得去做！于是克莱恩召集老弱病残天使，以凡人之躯硬撼神明，破坏了晋升仪式，强行抹杀了黑皇帝！赞美愚者！

# 第三名 《剑来》 愿挽天倾倒者，请起身！

这句话是绣虎崔瀺在大骊御书房召集三十六位修道之人和山水神祇，商议如何阻挡妖族大军北上宝瓶洲时所说。“一座浩然天下，竟然需要一个最小的宝瓶洲，来帮忙阻滞妖族大军，是不是个天大的笑话？我倒是想要让那浩然天下七洲，就这么活活笑死。”“偌大一座桐叶洲，都挡不住妖族大军，那就交由我们小小宝瓶洲，来将此事做成了。诸位，大势倾轧在即，愿挽天倾者，请起身。”年轻皇帝率先起身，在座所有人，皆站起身。国师壮哉！  
![](attachments/0c42bfc658c8b70b.png)

# 第二名 《诛仙》 碧瑶挡剑

“九幽阴灵，诸天神魔，以我身躯，奉为牺牲。三生七世，永坠阎罗，只为情故，虽死不悔”。世人都在说正邪有别，只有碧瑶念诵着痴情咒，没有犹豫，没有后悔，义无反顾的推开了诛仙剑阵下的张小凡，把自己推给了死亡。终是碧瑶挡了剑，世上再无张小凡！

# 第一名《斗破苍穹》 三年之约

三十年河东，三十年河西，莫欺少年穷！网文20年，这句话现在看来已经没有多大的震撼了，但是当年初次读到这里的时候，多少懵懂少年因为这一句话走上了网文路？退婚+戒指老爷爷的设定，你就说有没有看的热血沸腾？小白归小白，可这是小白文的鼻祖啊！  
随着新网文的不断更新，更多的名场面还会接连出现，再过五年、十年，现在看的热血沸腾的名场面会不会变成未来的老梗？接下来，评论区交给你们了~![](attachments/2698dcc5a20877ce.gif)
