如何评价特斯拉 Model 3？
================

本文来自微信公众号：[王铁根](https://mp.weixin.qq.com/s/ypA8MQTgd5Y8_BZJOm4_NQ)

国内哪一款电动车最好，一直令电动车爱好者们争论不休。由于所有电动车都有公关投放，所以就连蔚来 ES8 这种匪夷所思的产品都有人吹，可见电动吹的底线有多低。不过，**有一款车却是所有电动吹共同的精神图腾，电动吹对它的好没有争议，它就是特斯拉 Model 3。**

这令我感到困扰。因为当时 Model 3 这个车还没在中国开售，没有人开过这个车。这种情况下他们非要云吹 Model 3 很厉害，我也没办法。这让我回想起蔚来 ES8 还没上市的时候，电动吹们也是狂吹 ES8，给人感觉 ES8 无所不能，会飞，充一次电能飞到阿斯加德。后来的事情大家都知道了，ES8 一交车，就被热心网友扒了个底掉，堪称近代工业史上的耻辱产品。然而当时吹 ES8 的那些电动吹们，早就吹别的车去了，仿佛蔚来从未存在过一样。

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/93C6864E-7BD0-4FFD-AED3-375951EE0240)

图为死亡女神海拉在阿斯加德

不难看出，电动吹的优势是没开过一个车就敢吹，但我只有开过了才敢说好坏，要脸令我先天处在劣势。所幸，我的一位朋友买一辆特斯拉 Model 3，而且是国内最早一批提车的。我作为知名电动车爱好者，第一时间扑向了这辆 Model 3 Performance 高性能全轮驱动版。

在谈这辆 Model 3 之前，希望每位读者先放下对特斯拉品牌的幻想。过去，网络宣传让特斯拉显得神秘而高贵，很多人甚至分不清马一龙（埃隆・马斯克）和托尼史塔克的区别。加上真正能接触到特斯拉的人也不多，所以特斯拉神秘的面纱让大家竞相联想。大多数人没有能力自己去验证一下，只好听得风是得雨。正如我现在对一位印第安老人说潘长江是中国的时尚教父一样，他又没法证伪，他就只好认为真的是这样。

特斯拉不神秘，它就是一个造电动车的，同时可能还是自动驾驶害死车主最多记录的保持者。

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/CB1C6353-81DF-4F5F-A259-6C1CA3164A63)

model 3 初体验

刚刚开到 Model 3，我用了一个词：一言难尽。因为如果我们仅仅拿所有市面上的纯电动车做比较的话，Model 3 的表现是最好的。和其他令人发指的电动车们相比，Model3 居然显得较为正常，这对电动车来说相当难得。

它不仅秒杀蔚来，而且比同门的特斯拉 Model S、Model X 也都好得多。是不是对 Model 3 充满希望？别急。如果拿 Model 3 和正儿八经的汽车去对比的话，它又一泡污。马一龙那种为了装腔不惜一切代价的特质在 Model 3 上也有很强的体现。

为什么说 Model 3 和正经汽车比是一泡污呢，我慢慢来讲。

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/2F312F58-68DA-4B3B-A13A-865CCBCE091A)

Model 3 的车尾极短

Model 3 的车身尺寸并不小，数据上甚至比思域略大一些。但大轮圈、超长轴距和短到几乎不存在的车尾让 Model 3 看上去比想象中小。其实如果你尚未见过 Model 3，不妨在街上找找思域、朗逸等车，Model 3 的尺寸和它们差不多。

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/92E8DF77-BD6C-400D-9391-F29884C67BA8)

设计柔和，没有夸张的设计

我讲外形的意思是，不要因为这台 Model 3 超过 50 万的售价就期望它在外形尺寸上给人很虎或者很酷炫的感觉，没有的。它看上去比实际尺寸更小一点，而且外观上没有张牙舞爪的设计，一切都安分、柔和。

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/44CC0A5E-7C11-421C-81D6-4B4E4116756C)

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/5C6265CF-36A3-4991-8512-3085803CF1D8)

你无法找到一个舒适的开门手势

车的门把手是很愚蠢的。现在流行一种风气，就是为了装腔我非要做得和别人不一样。为什么这么设计，不为什么，就为了和别人不一样。很多新造车势力可能觉得门把手影响了逼格，于是把门把手镶嵌到车门里了，对外宣称这种设计降低风阻。

蔚来门把手好歹还会自动翘起来，让你去抠，虽然体验不如传统门把手得劲，但也凑合。Model 3 这个门把手要求你伸出大拇指，抬起其他四指（不翘起来的话就把门把手挡住了），用大拇指去用力按门把手一端，这样门把手另一端就翘起来了。这时候另外四指趁机抠住翘起来的部分，这样才可以把门拉开。手小的人需要双手才能操作。

当然，在特斯拉粉丝眼中，这是特立独行、酷炫的体现。你们的门把手都弱爆了。

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/CF2BE72F-5BB8-4855-B339-0731B47B5E1A)

中控只有一块屏，除此之外什么都没有

经历了门把手的考验，来到车内。Model 3 另一个标志性的设计是没有仪表盘，这让中控看上去像一个朴素的写字台。

自从大屏幕的不良风气流行以来，不少网络车企赚足了便宜，其中一个重要的好处就是省了大量的设计费。以往，老牌车厂要花大价钱，想方设法让车里的按键布局合理、方便耐用，还得美观。现在，垃圾大屏中控正在侵蚀这些优秀的传统。

我也开过不少有中控大屏幕的车。这些车虽各有特点，但有一个共同点就是大屏幕都很傻逼。经过简单的观察你就会发现，**任何一个装有大屏幕的车，在装了大屏幕之后都显得有些不知所措，屏幕是装上了，很大，很受土锤喜爱，但大问题来了，屏幕上显示啥好呢？**

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/45B7679C-99ED-4883-A39C-7AF9FD0E2485)

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/406DD9CC-934F-4BD1-88BA-4DBC2BD64696)

纳智捷和众泰的中控大屏（图自汽车之家）

显然，他们在装上大屏幕之后也发现了这个不知道显示啥好的问题，所以比较流行的解决方案是在大屏幕上放一个车的照片，或者放一个地球的背景，总之就是随便弄点啥东西给填满。不尴尬么？

你坐在车里，歪着身子探着头，伸出一个食指在屏幕上划啊划。屏幕的角度并不舒服，你脖子有些酸痛。屏幕反应并不快，你想调低空调温度结果第一下没按准，第二下才按到空调。你每按一下下调 0.5 度，你按了 10 下于是下调了 3 度。屏幕上布满了你的手指印。

于是你开始怀念那些布局合理、美观耐用、没有大屏幕，充满质感的汽车中控了。但没办法，现在土锤太多了，我觉得这个世界最终恐怕要被土锤占领。

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/8B6CBD12-A4F4-4C83-ADFC-1D9AC9FEDB88)

宝马 3 系的中控，优雅、漂亮、实用

宝马这个优雅漂亮的中控，没有大型屏幕。怎么，让你感觉没面子了吗？让你感觉缺乏 “科技感” 了吗？

**没有物理按键的控制区域是反人类的，特斯拉 Model 3 把这种反人类的设计做到了极致。**和纳智捷、众泰一样，特斯拉的大屏幕也只好留出大片的空白，这是没办法的事，屏幕太大了。所幸，它是特斯拉，可以引起很多人的盲目崇拜，就像贾斯汀比伯把内裤露出来一大截却成为了时尚一样。坦率地讲，大屏幕代表了廉价、土锤审美、设计上明目张胆的偷懒，以及对安全的极度不负责。

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/D28A195D-86DB-482B-A6D5-524D7D9F3B89)

您看看，这个大屏幕左上角显示车速信息。相信如果车主之前开过滴滴，就会比其他人更快习惯这个设计，因为很多滴滴司机师傅接单用的手机就固定在类似的位置。

大屏幕左下角显示什么不必在意，因为只要你用右手扶住方向盘，右手就会把这一部分挡住。你如果在行驶期间把右手拿开，低头观察被挡住的区域，会发现这里显示的信息果然没用，这算一个令人惊喜的贴心小设计吧。

**最令人无语凝噎的当属 Model 3 没有雨刷拨杆，需要在屏幕里进入二级菜单调整这一神迹一样的设定。**让我们设想一下，Model 3 驾驶员在高速上开车，此时突降骤雨，但自动雨刷没反应过来。此时车一秒向前移动 30 米，前挡风玻璃已经糊了，驾驶员要去大屏幕里翻找二级菜单？刹车也行，就由你追尾别人变成别人追尾你，你选吧。

据信，没有雨刷拨杆的设计是为了节省成本。马一龙，成本是这么省的吗？你和车主生命有仇么？

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/396986AB-71DE-47F3-9EB4-C0B9E0C77F91)

腿部空间也并不理想

Model 3 的前排容易找到一个较为舒适的坐姿，后排就惨了。以 Model 3 的尺寸来看，后排小得有些匪夷所思。虽然 Model 3 的轴距比雅阁还长，后排却比飞度还小，再一次证明了轴距和后排空间没有直接关系。不仅腿部空间捉襟见肘，头部空间也小得可怜，头顶上两三厘米就是玻璃。

显然，特斯拉已经无力在这么小的空间里再装上遮阳隔板了，所以这是一个迫不得已的设计，夏天应该难逃被晒的命运。

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/2746D09C-E091-48A1-97B6-3FD03BF551CF)

注意看头部与玻璃的距离，只有一两厘米

这么大面积的玻璃车顶，非常适合费玉清在演唱歌曲的同时观察天空。

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/F485A318-1118-4B9D-AB6C-7F8C644F0A27)

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/DB14D947-D3A3-4CB4-B2AF-BC5D054080A9)

车顶没有遮阳帘

说了这么多反人类设计，我想再夸夸 Model 3。它的驾驶感受是目前所有电动车里最好的，因为它几乎做到了 “正常”。尽管动能回收力度还是很强，调到最轻微的模式仍然很强，但它的加速居然调校得像汽油车一样舒适，这是很难得的。

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/8591DAE9-16E2-4A14-A02D-3729ECE4038B)

**加速想要舒适，最主要要看是否柔和**（不是快慢）。Model 3 的极限加速能力非常强大，但无论你深踩还是浅踩油门，它的加速都是柔和的。很多比 Model 3 加速慢的电动车，加速的时候都是一股愣头青的劲头，呼的一下，乘客就容易晕。

其中最奇葩的当属蔚来 ES8，踩下油门后车先装死，驾驶员正在奇怪为什么踩油门没反应，这时候又突然来一个猛烈的加速。正常人一定会晕，也许只有极个别的二手法拉利车主才会觉得蔚来好开吧。

同样让人惊喜的是 Model 3 有这副极舒适的底盘，极韧，高级感极强，这一点要明确给好评。

巨大的噪音让底盘赚来的分数又如数丢掉。Model 3 的路噪和风噪都很大，轮胎很挑路面，京沪高速这种路面，时速超过 90，车里就轰隆隆响了，表现不如思域。风噪也是一样，在路面良好的情况下只要时速达到 60，就能听到呼呼的风声。

Model 3 的续航能力也是电动车中一骑绝尘的存在。普通快充桩，想要从几乎没电充到满，2 小时不太够。但如果在特斯拉的超级充电桩，用不了 30 分钟就可以从几乎没电充到 50%，也就是 250 公里 + 的续航里程，30 分钟 250 公里，这么看似乎还行。但前提也不要忘了，只有当 Model 3 的电量极低时，才能达到这样的充电效率，而当你的 Model 3 还有 60% 的电，你想充到 100%，一个小时都充不满。

**客观地讲，Model 3 这样的表现已经基本上可以胜任城市通勤了。**它不仅电池容量大，而且极其省电（和蔚来相比），就算开着空调豁着开，城市里也就是百公里 20 度电出头的电耗；而蔚来 ES8 这么开，百公里 50 度电都打不住。Model 3 开起来续航焦虑仍然会有，但已经小了很多，这和北汽 EU5 类似：续航能达到或者超过 400 公里，充电还算快。如果说开蔚来 ES8 是时时刻刻都在焦虑，那么开 Model 3 如果不出城的话，几乎可以几天焦虑一次。

**做工一如既往地差，但这是特斯拉的传统，我也不好大力批判。**

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/30B031E5-06B1-4BB0-889B-47024615EA28)

新车，胶条已经裂开一大块

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/DE827018-C102-45B6-85C7-8C26EC7DCA7E)

尾门的液压杆在某一角度会蹭到胶条，据说很快会把胶条蹭坏

所以，特斯拉 Model 3 能不能取代燃油车？

取代个屁。**Model 3 的所有优点，都不是电动车特有的优点；但电动车的缺点，Model 3 却一个不落。**

我说 Model 3 还不错，是以电动车的标准来衡量的。倒不是我偏心，如果你也像我一样开过什么北汽 EX200，什么众泰 E200，什么欧拉什么哪吒，什么蔚来 ES8，再开 Model 3 的话，也会得出一样的结论：Model 3 好开。

Model 3 的加速快、平顺，都是汽油车一样可以做到的，没什么稀奇的。只要一台车是通过轮胎与地面接触的摩擦力加速，那就都有一个极限，无论是汽油还是电动，都不难达到这个极限。

Model 3 的续航长，是和电动车比的。和汽油车比续航，怕是要按在地上摩擦。

千万不要说安静，Model 3 一点都不安静。Model 3 的风噪和胎噪让它在高速行驶时比汽油车更吵。低速的时候，Model 3 又不得不播放噪音来提醒周围的路人，说 Model 3 的优势是安静，实在站不住脚。事实上，汽油车现在也极其安静，如果你在小区里行驶，前面有两个人在路中间走，你不轻轻鸣笛，路人是不会知道后面有车的。

**所以总的来说，Model 3 是一台大大进步了的电动车，但它无法突破纯电动的局限。**它加速迅猛，开起来不错，悬挂极为舒适，续航是纯电动车里最长的，但也只有这些优点了，剩下的都是缺点：

1. 价格昂贵。拿 Model 3 后驱长续航版来说，这个价格明明已经可以买到如宝马 530 这样的中大尺寸豪华轿车了。这么高的价格，它比宝马 530 强在哪儿呢？

2. 做工粗糙，各类异响、装配瑕疵层出不穷。

3. 内饰简陋，一块大屏幕打天下，毫无豪华感、设计感，也毫无便利性可言。

4. 充电在电动车里算快的，但比汽油车仍然慢了好几个数量级。

5. 噪音大，动能回收令人不适。

6. 雨刷的设定非常非常危险，为了安全起见，不建议雨雪天气驾驶 Model 3。

7. 后排腿部空间、头部空间都非常小，车顶没有遮阳板或遮阳帘，后排头顶就是玻璃。

8. 门把手反人类。

9. 噢，还有很重要的一点，Model 3 的 “自动驾驶” 是一个极其危险的魔盒，小心这玩意要了你的命。

这台 Model 3 的车主在北京有十几台车，居住条件也完全可以安装充电桩，所以按理来说这是电动吹们口中 “最适合开电动车” 的条件。但即使如此，车主仍然表示，Model 3 令他感到焦虑。虽然 Model 3 充电足够快，但在汽油车那 2 分钟 600 公里的充能速度面前，还是慢得像蜗牛。

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/67E93E05-F45F-488D-AFCE-985FD209F44D)

宝马 5 系这样舒适豪华的中大型轿车，也只要 40~50 万。

如果你喜欢电动车，你不介意在充电站等，你享受续航焦虑，你觉得用手抠门把手很酷，你对一边开车一边从大屏幕二级菜单里找雨刷这样的危险操作也觉得没问题，那我觉得你应该说的是：“我喜欢电动车。”

而不是 “电动车要把你们汽油车取代了！”

![](.evernote-content/3D5CD5F3-3289-4F16-97A1-909919F65C0C/0E8C4CB6-E938-42FD-9AAC-9087DAAE40F0)

马一龙的《硅谷钢铁侠》里透露，他当年做电动车，就是为了 “让美国摆脱石油依赖”。好家伙，现在美国确实摆脱石油依赖了，还他妈成了石油出口国！但原因是页岩油开采技术的发展，并不是因为马斯克把一大堆电池装在汽车底盘上。

Model 3 值不值得买？如果非常喜欢且不差钱，现在就可以买。如果仅仅做市区代步使用，并且就想买电动车，那我觉得 Model 3 如果价格能降到 25 万以内，仍然是值得买的。但请千万记住，第一不要雨雪天驾驶，自动雨刷可能会要了你的命。第二不要使用 Autopilot 功能，这玩意能给人一个它很强大的错觉，然后在你对它产生信任的时候要了你的命。

别迷信特斯拉了，它只是一个电动车，不是宇宙飞船。纯电动的技术路线已经快走到头了，夕阳无限好，只是近黄昏。
