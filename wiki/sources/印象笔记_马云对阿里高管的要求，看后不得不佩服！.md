企业家阅读

**老板们睡觉之前应考虑的几个问题：**

**1 我的企业下一步应该怎么走？**

**2  按照目前的状况，要把公司做到什么程度？**

**3  为什么这两年公司越来越难做？**

**4  公司现在的办公环境？员工的工作氛围？学习力？对公司的发展起着什么作用？**

**5 业主除了每个月关注你有没有按时交房租！你公司的运营状况她关注过吗？有关系吗？**

**6 公司发展过程中“人，财，物”起着关键性作用！运营过程中，这几点出现问题，是否有人跟你一起面对？解决？老板们，你们还在孤军作战吗？**

《资本大智慧与总裁峰会》

ENTER TITLE

**峰会主题**

**【对接资本】****：投融资方法，投资人投资标准，如何梳理计划书，资本运作方法**

```
```
【融资方法】：掌握多种融资方法，通过众筹、资产融资、债权融资、股权融资等方式，打通获取资金的路径。

【商业模式】：打造商业模式，树立核心竞争力，去中间化轻资产运营，细分市场、精准定位，上市标准化运营。

【全网营销】：打造全网营销系统，精准营销，流量倍增，提高转化率，增加客户粘性,品牌打造。

【跨界众筹】：跨界资源整合，免费商业模式打造。筹钱、筹人、筹资源、筹客户。  
【股权激励】：对内激励，对外融资，如何用股权整合上下游资源。股权如何合理分配，制定安全的进入退出机制。

【团队打造】：总裁领导力智慧，企业内训，人才培养。

```
```
【上市操作】：上市落地一站式操作，上市前融资、上市后定向增发。

【资源整合】：高端人脉圈整合交流，盛世游艇不定期组织游艇出海活动，为企业家提供互动与商务交流机会。
```
```
