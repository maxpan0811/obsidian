# iPad Pro键盘是模仿还是创新? 一看便知晓

---

[![](.evernote-content/7A2FF5C7-D5B8-429D-859E-F02D6B82DF50/F3A686C7-DDA1-4A93-B50A-E4676C17DAE4.jpg)](http://www.feng.com/iPhone/news/2015-09-10/The-Pro-keyboard-is-imitation-or-innovation_624054.shtml)

![](.evernote-content/7A2FF5C7-D5B8-429D-859E-F02D6B82DF50/F3A686C7-DDA1-4A93-B50A-E4676C17DAE4.jpg)

从台下观众的反应来看，iPad Pro 在今天凌晨发布会上的人气似乎比 iPhone 6s 还要高。12.9 英寸屏幕、A9X 处理器、Apple Pencil，当然还有和苹果无线键盘几乎一模一样的 Smartkeyboard 。这不是苹果第一次为自家平板电脑配备外设，但是此前无论是 Smart Cover 还是Smart Case，都没有像今天的Smart Keyboard 这样给力。

![](.evernote-content/7A2FF5C7-D5B8-429D-859E-F02D6B82DF50/D97A615A-43B5-4715-929E-A9E99FE423F9.jpg)

“对很多人来说，要完成工作或记录想法，键盘仍是便捷有效的工具。全新的 Smart Keyboard 为这一历史悠久的发明赋予新的生命力，让 iPad Pro 更添灵活性。而且，Smart Keyboard 的众多创新技术，让你可以摆脱开关和插拔，甚至再也无需配对，出色融合了全尺寸键盘的实用性和高度的便携性。”

![](.evernote-content/7A2FF5C7-D5B8-429D-859E-F02D6B82DF50/6DC7B771-FBE8-40CA-86D3-3423EFDBF718.jpg)

由于之前已经有微软为自家的 Surface 推出过外接键盘，因此苹果这一次尝试也是冒有一定的风险。不过从今天的发布会来看，Smart Keyboard 的出现仍然证明了苹果的创新能力。

Smart Keyboard 的第一个创新，就是它的连接方式，这种方式可以为 iPad Pro 带来全新的供电模式。苹果表示，要打造像 Smart Keyboard 这样纤薄耐用的设备，传统的线缆或部件已经无法再依赖，因此他们只能在一张纤薄如纸的尼龙上采用金属蚀刻的方式来创造出独特的可传导材质，让电能与数据可以双向传输。

![](.evernote-content/7A2FF5C7-D5B8-429D-859E-F02D6B82DF50/297ED320-73E6-4ED0-90F7-BE5693697074.jpg)

第二个创新则是 Smart Conector，这是由苹果设计的全新连接方式，这种方式可以让键盘与 iPad Pro 紧密相连，运行顺畅无阻。它位于 Smart Keyboard 的连接边缘，以及 iPad Pro 的侧面。Smart Connector 与 Smart Keyboard 内的可传导织物携手合作，实现电能与数据的双向传输，让你享受简单却又精妙的打字方式。

![](.evernote-content/7A2FF5C7-D5B8-429D-859E-F02D6B82DF50/4C3A26EC-32C1-498D-9592-3D5DEA89F653.jpg)
