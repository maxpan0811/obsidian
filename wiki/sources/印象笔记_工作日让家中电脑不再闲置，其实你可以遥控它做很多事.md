工作日让家中电脑不再闲置，其实你可以遥控它做很多事
=========================

22小时前

[![](.evernote-content/7A5F3693-CE16-42D8-99C0-A96664A50720/0626E1B2-CFF5-4D1D-BA31-9CD8201A1168)](https://sspai.com/user/78)

#### [化学心情下2](https://sspai.com/user/78)

即便是在这个人人都有手机的年代，我们依旧会在家中配上一台电脑，原因其实很简单，那些年下载的剧集和电影、Steam 上的那些等着我畅玩的游戏大作，还有那些需要处理的工作文件以及资料——手边一台的电脑的重要可想而知。

但对于大部分人来说，到了公司有公司配备的办公电脑，因此家中电脑会在你每天工作日八小时的时间里处于空闲状态——动则几千块钱的电脑在吃灰显然不符合我们「勤俭节约」的本意，如果打个小算盘你就会发现家中电脑的利用率低到令人发指的地步，那么怎么才能在工作日的时候让家中的电脑也能被充分利用起来呢？

远程控制家中电脑，让他给我干点活
----------------

作为一个勤（ji）俭（du）节（kou）约（men）的人，早在我刚刚开始工作时，就开始琢磨怎么给工作日白天无所事事的家中电脑找点事情做，在那个手机还是诺基亚、宽带叫做 ADSL 的年代，我利用家中电脑的方式更像是一台 P2P 下载机——在上班前打开电脑，然后打开 BitTorrent 或者电驴，将最新的剧集和软件镜像的下载任务挂上去，然后等着下班之后打开电脑看看下载进度。只不过这个看似最简单的利用方法有个致命缺陷：如果下载的资源不够热，下班后你会发现下载的资源进度丝毫未动，只能看着白天哗啦啦的电费欲哭无泪。

很显然这种完全不管，只等着八小时「收货」的利用电脑空余资源的方式效率实在太低，最好的方法当然是可以在上班的时候随时控制家中电脑，如何远程控制？对于刚刚工作的我而言，似乎是个不小的难题。

![](.evernote-content/7A5F3693-CE16-42D8-99C0-A96664A50720/BA91493A-4302-429D-BB1C-7EF9495E82E5)

Windows 本身自带远程控制，看上去似乎是最完美的解决方案，这样也不用安装第三方软件省时省力，但糟糕的是忙活了半天才知道原来无论是家中的电脑还是公司的电脑都没有公网 IP、也不在同一个内网网段内——而 Windows 远程控制只能使用公网的 IP 地址或者两者在同一个内网内，显然这种看似简单的方法是极不实际的。

![](.evernote-content/7A5F3693-CE16-42D8-99C0-A96664A50720/FFA9AF5E-E768-4D40-BFC5-F2D07AA37679)

QQ 好像有远程协助功能，比如说我朋友的电脑出了点问题，远程协助都是用 QQ 发起，思路可以是这样，家里电脑的 QQ 登录 A 账号，公司登录 B 账号，然后发起邀请不就控制了么——等下，家中电脑接收到请求远程控制邀请，那谁来点击同意？QQ 远程协助是不能「无人值守」进行远程控制的。

显然适合这么操作的远程控制软件，必须既可以「无人值守」又不需要依赖有两台设备都配有公网的 IP 地址，直到我找到了「向日葵远程控制」。

向日葵远程控制：最简单实用的远程控制软件
--------------------

向日葵的好处在于完全不需要主机和宿主机的 IP 地址，安装被控端之后绑定账户名就可以被控制；此外「无人值守」也非常方便，被控端软件安装好之后，只要被控端处于开机状态，控制端只需要输入被控端的电脑账户登录名和密码就可以控制操作。

![](.evernote-content/7A5F3693-CE16-42D8-99C0-A96664A50720/15060634-1838-4D04-AD11-524C16A3513A)

而实操起来也很简单，我在家中的电脑上安装了向日葵被控端，然后打开被控端后登录向日葵账号，之后就可以不用做任何的额外操作了，因为这时候被控系统服务已经在后台默默运行了。当我在公司电脑上安装控制端，并且登录相同的向日葵账号，就会在主机列表中看到家中电脑，点击「远程控制」然后输入家中电脑的账户和密码，就可以看到家中电脑的电脑桌面了。

![](.evernote-content/7A5F3693-CE16-42D8-99C0-A96664A50720/8392052B-5C82-43B9-8767-55C6D882EF97)

当然，对于一些特殊需求，向日葵当设计也很人性化，比如说被控的设备是一台笔记本电脑，如果直接远控会让电脑屏幕亮起，这里可以在进行远控的时候选择直接黑屏操作，这样你在控制端仍然可以看到电脑桌面并且执行操作，但被控电脑但屏幕实际上是黑屏的，所以外人看不到你的实际操作，安全性也得到了保证。

远程控制：远控下一部剧集试试
--------------

其他的操作其实和 Windows 自带的远程控制挺像，基本上就在一个应用窗口里面操作一台电脑的系统，根据向日葵的说法，只要电脑不关机不断网，你可以随时通过控制端来操作，那么最为实用的使用家中电脑方案，作为下载工具实在太合适了。

![](.evernote-content/7A5F3693-CE16-42D8-99C0-A96664A50720/93675874-2FE3-4E7C-8EE7-0831C320D093)

向日葵支持远程剪贴板操作，这么一来即便是在办公室复制的下载地址也可以通过向日葵直接贴到迅雷上，比如我最近想下载一些纪录片，就可以直接把下载地址复制到家中电脑的迅雷上，然后让家中电脑默默下载即可，这样就不会出现开机一整天下载纹丝不动浪费时间的情况出现了——即便是下载出现问题，也可以通过远程控制的方式来选择其他的下载内容，充分利用空闲的家中宽带和电脑。而到了家就可以直接享受白天下载好的剧集和电影，再也不用为晚上不时缓冲卡顿的在线视频观看体验烦恼了。

![](.evernote-content/7A5F3693-CE16-42D8-99C0-A96664A50720/6BB6B68D-EE27-461F-911F-F8245BA34FF7)

另一种玩法：家中电脑做后期渲染
---------------

空闲的家中电脑只是用来作为远控的下载机实在有些浪费，毕竟家中电脑的性能都相当不错，考虑到吃个鸡都是绰绰有余的性能，其实作为工作的辅助倒是真的事半功倍。

对于从事影视后期的朋友来说，最为浪费时间的其实是后期渲染——实时预览依赖 GPU，但到了后期合成渲染，那就是仰赖 CPU 和 内存进行处理的事情，所以当你点下渲染合成的那一刻，你的电脑别的其他工作就什么也做不了，如果同时有多个内容需要渲染，那么今天就只能「被迫加班」了。

![](.evernote-content/7A5F3693-CE16-42D8-99C0-A96664A50720/8721C5FE-2AEF-4170-AC5B-7F4A24C52BC6)

如果用向日葵远程控制能不能将这部分渲染工作分给家中电脑来完成呢？其实完全没问题，只需要将最后的工程文件导入到家中的电脑中，然后再用向日葵远程控制家中电脑打开后期软件，直接进行渲染。这样你在公司的电脑可以不用做工程渲染这样极度浪费时间的工作，而将这项需要大量计算力的事情交给家中性能极佳的电脑来完成，让家中的电脑成为你的「迷你渲染农场」。

当你需要后期渲染的输出文件时，可以通过向日葵的文件传输，检查输出目录中是否已经有对应的文件出现，然后将文件传到公司电脑就可以用了。这样异步的操作可以大幅度的利用资源，减少渲染造成的时间浪费，工作不加班不是梦。

我不在家，喵主子你在干什么！
--------------

虽说现在网络摄像头很是流行，但从物尽其用的角度来说，既然远程控制家中电脑已经成为可能，那么查看家中电脑打开摄像头也就很简单了，尤其是但家中有那些捣蛋的猫主子，正好可以看看他们每天白天都在干什么。

![](.evernote-content/7A5F3693-CE16-42D8-99C0-A96664A50720/FD7E09BD-A21A-4EFE-8F6C-D2581DA12B83)

用向日葵远程控制登录家中的电脑，然后打开电脑上的「相机」应用，这时候就可以调用电脑上的摄像头设备进行实时预览啦。这样一个极度简陋的家中视频监控已经初见雏形，后面的操作就完全看我们对监控的需求，你可以选择直接录制视频来监控当前角度家中的环境，也可以随时给猫主子们拍上几张倩影，至于他们捣没捣蛋，都藏在你远程默默录制的那段视频中~晚上再找这几个主子算账！

![](.evernote-content/7A5F3693-CE16-42D8-99C0-A96664A50720/BBC84A21-03B7-4B5D-BC69-5E958DA9E65E)

对于需要挂机的游戏，我用家里电脑练级
------------------

家中电脑白天空闲，如果正好也玩各种网游页游，那么用来给自己玩的 DNF 或者页游挂机攒经验似乎是个相当不错的主意，这样在公司电脑只要用向日葵远程操作家中电脑，打开游戏的网页输入页游账户密码，放上去挂机即可；即便是网游例如 DNF 也可以远程操作登录客户端挂机赚经验。总之就是不能让家中的电脑空闲在那里。

![](.evernote-content/7A5F3693-CE16-42D8-99C0-A96664A50720/94DF6839-B83D-41F9-9D3C-0C4CC4E0FF30)

通过远程的方式控制家中电脑挂机练级的好处在于，一方面不耽误在公司正常上班也不耽误自己游戏涨经验。另一方面随时可以控制游戏进度，上线下线完全可控，比起所谓的游戏托管要靠谱许多。

家中电脑运维：坚决不让更新重启出现在晚上吃鸡过程中
-------------------------

没有什么比起 Windows 更新的强制重启更令人讨厌的了。

晚上回家打开电脑开局吃鸡，玩到一半弹出 Windows 更新需要重新启动计算机，蛮长的系统更新中消磨掉宝贵的游戏时间，除了满脸的 what 之外，你恨不得砸掉面前的这台正在更新的电脑。

![](.evernote-content/7A5F3693-CE16-42D8-99C0-A96664A50720/85FA863E-5A6F-42F3-9B8D-1260221F4C52)

晚上宝贵的业余时间显然不能被各种软件、游戏还有系统更新浪费，要知道现在 Windows 一个大版本更新下载更新文件都是 3GB 起，至于暴雪的游戏更新更不用说，更何况晚上那糟糕的网速，更新到能玩每个一两个小时根本下不来。显然最好的办法当然是白天电脑空闲时就把软件游戏更新好，系统补丁更新好。

在这一点上我的做法就是利用白天空余时间用向日葵远程控制打开家中电脑，点开 Windows 更新查找更新，以及打开战网、Steam 等平台更新游戏，这样至少我到家的时候，就再也不需要为了更新各种系统、游戏、软件而等待电脑更新重启。到手就用，不耽误任何一分钟的业余时间。

![](.evernote-content/7A5F3693-CE16-42D8-99C0-A96664A50720/94C08E7C-5662-4A8C-AC2B-EE6F14E64717)

向日葵远程控制的一个好处在于，即便是更新后电脑重启，只要电脑进入系统的登录页面就可以进行远程控制，期间无需人工干预操作，可谓是相当省时省力。

结语：通过向日葵远控充分利用家中电脑
------------------

有了向日葵远程控制工具，我在公司工作间隙，可以充分利用家中的电脑资源来做我想做的事情，家中电脑可以帮我下载剧集、作为后期渲染的渲染电脑、帮我游戏挂机、监控家中环境等等，当然我的朋友还有使用远控家中电脑来完成家中智能家居的控制，利用峰谷电价进行挖矿作业等「高端操作」，总之我的目的就是让工作日我不在家时，家中电脑也可以充分利用。如果是你，你会如何利用手里的这台空闲的电脑？欢迎在文后留言，谈谈你的利用闲置电脑的大法吧！
