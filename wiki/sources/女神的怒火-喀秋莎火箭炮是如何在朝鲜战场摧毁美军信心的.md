# 女神的怒火：喀秋莎火箭炮是如何在朝鲜战场摧毁美军信心的？

# “喀秋莎”火箭炮部队的组建

炮兵第21师是我军入朝参战的惟一一个喀秋莎火箭炮兵师，其前身是四野第48军143师。这是一支英勇善战、作风顽强、英雄辈出的部队。建国初期电影《董存瑞》、《翠岗红旗》、《战火中的青春》三部战斗故事片的原型均出自这个师。鉴于该师战绩突出，出了包括董存瑞等4位全国战斗英雄，中央军委决定将其改编为火箭炮兵师，1950年10月底，该师奉军委命令，立即北上，在东北接受改编训练，准备赴朝参战。

![](attachments/bb7349ff1fd61098.jpg)

21师吴荣正师长

炮兵21师装备的火炮是苏联造M-13火箭炮，俗称“喀秋莎”火箭炮。炮兵连为4门制，每营12门，每团24门，全师120门。M-13火箭炮是一种多轨式轮式自行火炮，弹径为132毫米，每辆炮车有一座8联装轨式火箭发射架，每条滑轨上下各装挂1枚火箭弹，每炮共装16枚，最大射程8500米，发射为电气点火，数秒钟内即可全部发射完毕。这种火箭炮弹径大、弹群密集、发射速度快、杀伤破坏力大、战场机动性能好。

![](attachments/07a4a82320995d13.jpg)

![](attachments/8be89cb1a503312c.jpg)

苏联喀秋莎火箭炮

虽然21师只有很少一部分人懂得炮兵操作，再加上朝鲜战场战况激烈，留给21师的改编训练时间相当有限。为此，21师迅速投入各项准备。全师从紧急抽调了由文化基础的300多人前往沈阳炮校学习基础的炮兵知识和理论40天。当时由于中苏关系还比较密切，军委派出了一个苏联的专家组，帮助炮21师进行改建。全师采用”一个会教全班、一个班教全连的辐射教学方法“，全师官兵都不分白天黑夜的学习，通常一个炮兵部队形成战斗力至少要1年时间，但是炮21师只用了1个月时间。在剩下的10多天中进行了战前动员和开进准备，从接受新装备到开赴前线炮21师只用了40天的时间。

# 入朝第一战告捷:后洞迎战美军第七师，美军损失700多人

1951年9月1日，21师203团奉命支援第27军79师235团，反击进占后洞里阵地的美七师的两个营，这是21师入朝的第一战。由于保密工作到位，此时美军还不知道我军拥有火箭炮，在无所顾忌的情况下，美军向我军阵地发动了猛攻。

此时，203团的24辆‘’喀秋莎‘’战车，突然对进攻我军阵地的美军发动了齐射。在很短时间内，将全部的384枚火箭弹打向了敌军阵地，密集的爆炸声覆盖了美军的两个营的阵地，美军损失700多人，无力再战退出阵地。他们惊呼志愿军使用了原子炮，此时美军还不知道志愿军炮兵部队已秘密装备了苏制致命武器喀秋莎火箭炮。

# 次战美军骑兵一师:采取“引蛇出洞”战术，给予对手沉重打击

1950年10月，21师乘胜追击，这是面对的是志愿军的老对手，在长津湖撤退的美国骑兵一师。这次出动的是21师202团，针对针对火箭炮射速快、覆盖范围广的特点，志愿军决定使用诱敌深入的战术。

当晚，202团于连川以北之青木润等地，构筑了发射阵地，首先按照战术安排，由我军步兵对美军阵地发动了进攻，美军见志愿军前来进攻，急忙钻出工事与我军激烈交锋。当美军刚露头，我军的300多枚火箭弹，极速落到美军阵地，美骑兵一师付出了800多人的伤亡，最后被迫退出阵地。

# 上甘岭之战:喀秋莎火箭炮的巅峰时刻

1952年10月24日，著名的上甘岭战役爆发。在这场血战中，喀秋莎火箭炮再显神威。五圣山作为上甘岭战役的重要战略要点，其南麓的597.9高地和537.7高地就是五圣山的门户，在上甘岭第一阶段中，美军在强大火力掩护下夺取了这两个高地，直接威胁五圣山。10月14日至20日，是上甘岭战役的第一阶段。7天7夜的鏖战中，双方反复争夺地表阵地，战斗异常残酷。在这种情况下，志愿军总部果断决定，将21师209团配属给参战的15军，归15军军长秦基伟统一指挥。

10月19日，209团接到命令：今晚全团投入上甘岭战斗。当天下午，全团24门“喀秋莎”火箭炮，分头从预备阵地出发，奔向发射阵地，并做好了准备。16时30分，15军的104门火炮首先发出怒吼，炮弹像雨点一样飞向597.9高地。一阵火力急袭后，15军炮火向敌后方延伸。在地堡里躲避弹雨的美军钻出来，准备抗击15军步兵的冲锋。这时，209团突然实施了齐射。在完成两次齐射后，209团迅速撤出阵地，从进入阵地到打完转移，前后只用了13分钟。美韩军绝大部分防御工事被摧毁。

第二阶段战役打响后，志愿军以209团24门火箭炮和70门各类野战炮火力支援，在1952年11月11日16时发动进攻，当晚夺回阵地，全歼守敌。第二天，美军以一个团的兵力企图反攻，在炮兵的火力支援下一举击溃美军。战役期间团、营级的齐放发射达到了100多次，有力的支援了步兵进攻和防御作战。在43天的上甘岭战役中，我“喀秋莎”二〇九炮团，前后参加10次战斗，给敌人沉重的打击，有力地支援了步兵，因而受到步兵战士的热情欢迎。上甘岭战役美韩军与志愿军伤亡比例为2.3∶1，美韩军的伤亡有70%是由志愿军炮火杀伤造成的，其中很大一部分来自“喀秋莎”火箭炮。当时，炮兵二十一师的车号是“84”，部队一见“84”车号，就主动让路，并鼓掌欢迎。

# 金城战役:喀秋莎火箭炮的谢幕之战

1953年五月中旬到六月中旬，志愿军和人民军为了配合停战谈判，对美军和韩军先后发起两次夏季进攻。到了六月中旬，停战谈判各项协议均已达成，正准备签字，事情却遇到一个严重阻碍——韩军李氏集团贼心不死，企图破坏停战的实现。

为了再次给韩军以沉重打击，志愿军决定实施金城战役。抗美援朝战争期间，志愿军装备了大批苏式“喀秋莎”火箭炮。在金城战役中，志愿军集中五个“喀秋莎”炮团和其他炮兵部队的上千门火炮，此次战役由志愿军20兵团担任主角，整个20兵团有四个军外加一个师，82毫米迫击炮以上的火炮就超过1000门，打算在战役开始时，首先实施猛烈的火力突击。7月13日晚21点整，20兵团对韩军展开了势如山崩的炮击。在这场战役中，喀秋莎火箭炮一共发动了69次齐射，帮助步兵取得了金城反击战的胜利，促使美国不得不尽快在谈判桌上妥协，为板门店停战协定的最终签订扫清了障碍。

# 结语

火箭炮兵第21师入朝作战历时两年半，先后配合12个军，在兄弟炮兵部队的支援下，协同步兵进行大、小战斗30余次，歼敌10万余人，击毁敌坦克56辆、汽车230余台、火炮30余门、击落击伤敌机24架，有力地支援步兵作战，在抗美援朝战场上发挥了重要作用。但是，恩格斯曾经说过：“枪是不会自己射击的，需要有勇敢的心和强有力的手去使用它们”“赢得战斗胜利的是人而不是枪”。志愿军能赢得抗美援朝战争的胜利，绝不仅仅是依靠“喀秋莎”火箭炮等先进装备，而是因为有着英勇无畏的战斗意志，不怕牺牲的战斗精神、灵活机动的战略战术和保家卫国的正义立场。联合国军“钢多气少”，志愿军从最初的“钢少气多”，战争后期在保持昂扬斗志的同时，大力换装苏式制式装备，特别是自动火器和喀秋莎火箭炮等先进装备，更好地完成了抗美援朝的历史使命。随着装备的更新换代，喀秋莎火箭炮逐渐被更先进的火箭炮所替代，但它在朝鲜战场上的优异表现仍然值得怀念和称道。
