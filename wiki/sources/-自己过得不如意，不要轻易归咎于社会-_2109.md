# 自己过得不如意，不要轻易归咎于社会

速读摘要

有两个比较深的感受，一是现在太卷了，二是对社会结构的认知存在严重错误。人类还远没有适应技术的突飞猛进，现在的自媒体过于发达，信息多而无用，反而充斥大量负面情绪。有人逃税漏税、有钱人避税、个税只是工资收入等等。而这2000万人还可以进一步细分，考虑专项附加扣除，如果有房贷、有父母、有子女的情况下，一年扣除额度拉满是7.2万元，加上基础的6万元免税额度，年入16.8万以下就是3%的个税税率，但实际上很难拉满到7.2万元，找一个整数大概年入15万以下对应3%税率。

10月是编制类考试公告密集发布的高峰期，也是明年毕业的应届毕业生求职的高峰期。最近跟几位在校生有所沟通，主要是咨询工作和职业规划的事。有两个比较深的感受，一是现在太卷了，二是对社会结构的认知存在严重错误。

前一个之前聊过好几次了，十几年前上大学的时候，几乎没有就业的概念，说得好听叫顺势而为，说得难听叫随波逐流，最多寒暑假实习一两次而已；而现在大学生需要从大一就开始计划几年后的就业，容错率低得多，导致非常焦虑。

而对社会结构认知的严重错误，进一步加大了焦虑。这在一些年轻读者提问中体现得特别直接，多数提问者存在不同程度的焦虑，对进入社会存在恐惧感，甚至觉得活着没有意义。

而觉得没有意义的原因在于，感觉未来没有出路，然后列举很多社会负面，比如月入3000元、底层都是牛马、再怎么奋斗也改变不了人生以及官富二代的嚣张等等，归结为一点：**快乐是高层的，作为普通老百姓的后代，未来的人生没有希望，还不如早早去死。**

有这种想法完全可以理解。最直接的压力来源还是就业困难，倘若回到十几年前、30年前，大学生还是有些含金量，机会也多得很，能够更加顺利的找到工作，自然就少了很多迷茫和恐惧，更何况人类还远没有适应技术的突飞猛进，现在的自媒体过于发达，信息多而无用，反而充斥大量负面情绪。

人类进化还停留在前工业时代，习惯方圆10里的生活，小范围的世界不会有这么多坏消息，然而发达的信息技术让任何一个人都能随时随地看到全国乃至全球的负面信息。于是，一个人越是关注网络，越会容易悲观。但正如小镇在《[千年机遇，就在眼前](http://mp.weixin.qq.com/s?__biz=MzI2MzY3NjAwOQ==&mid=2247498642&idx=2&sn=d078dd4e5abea02589b390ff0070a7bf&chksm=eabaf294ddcd7b825f9ff586d8622cc168c34ef264c986fc9249e9165caaf7db92bbb3646803&scene=21#wechat_redirect)》分享的，现实的中国机遇有很多，只是在当前舆论环境下，过得好的闷声发大财，就导致网络上充斥着悲观情绪。

需要正确面对这种技术超越人类进化的现实。

而更深层次的还有一个关键原因：**大学生本质是不事生产的伪脱产者**。大多数大学生由家庭供养，在校期间几乎不需要考虑赚钱谋生，因此别管大学生们家境差异有多大，对于社会的认知都是从小资产阶级的视角去展开，离开大学之后，才回归真实的阶层。

![](attachments/625f09f7140838c8.png)

正因此，才会将工作等同于牛马，又因此产生厌恶和恐惧。

这一点在留学生群体中表现得更为严重。有不少留学生回国后特别怀念留学的生活，因为多数乃至全部生活费用都是由他人提供，每年十几万、几十万甚至上百万的开支，大概率远超留学生工作后的薪资。

而相比国内的大学生，留学生在国外更加脱离社会生活，与当地的社会是脱节的，既然脱节就不需要考虑很多现实的烦心事，甚至因为回国后很容易就能跟留学生活完全切割，就更容易放纵欲望。

所以国外当地人生活面临的困难、社会经济的问题等等，留学生是不在意的，当地人也只把留学生当成过客，不会用当地社会的规矩去要求留学生。于是留学生就会觉得生活优渥、没有压力、人际关系简单，但如果真的选择移民留在这个国家，才会发现曾经觉得很美好的社会，竟然有这么多问题。

但大多数留学生还是要回国，回国后需要工作，需要回归真实的社会，这种反差就更加强烈了。正因此作为学生，就会对自己未来长期生活国家的一切问题更加敏感。

**无论国内大学生还是留学生，从校园走入社会，都像是脱离“楚门的世界”，不适应是正常的。**

![](attachments/0023ca92ea1481f0.jpg)

“楚门的世界”与真实世界最大的区别，就是前者过于简单，这就会导致学生们对社会的认知变得简单粗暴，典型就是开头提到的社会结构认知。

很多学生对社会的认知是两极分化的，也就是“二极管思维”，将社会简单分为高层和底层，没有中间状态，进而认为快乐的高层和痛苦的底层之间有一根线，只有突破这根线才算改变人生。

于是当读了大学、拿到了优秀、找到了工作、付出努力等等之后，还是没有过上想象中没有烦恼的上层生活，就会认为自己仍然是底层，觉得之前上大学、找工作、努力等等一切都是没有意义的，于是就觉得人生没有意义，不如一死了之。

一个代表是网络上一旦国家有什么进步或者好消息，一定会有人问“这跟我月薪3000有什么关系呢”？

但是如果能够明白，0和1之间有无数个数字，从0到1每一点点的进步都可以改变人生，就会少很多恐惧。

不要被网络上的虚假信息给骗了，网络上瞧不起月薪过万，觉得这不过是底层，但真实情况如何呢？

10月15日，国税总局发布了2023年度个税汇算清缴数据，这个数据体现的是2022年的整体收入情况，才过去两年，收入变化并不大，按照2023年、2024年前三季度人均可支配收入增长情况，在国税总局公布数据的基础上增加一成就行了。

当然，肯定有人提出种种反例：有人逃税漏税、有钱人避税、个税只是工资收入等等。

需要注意经过这几年的税收征缴强化，个税覆盖范围越来越大，已经不仅仅是工资收入，比如今年在“收入纳税明细”中新增了五项个人所得，包括经营所得、利息股息红利所得、财产租赁所得、财产转让所得、偶然所得，加上此前的工资薪金、劳务报酬、稿酬、特许权使用费等4项，一共有9项。

![](attachments/1b192faac2edd469.jpg)

逃税漏税难度很大，覆盖范围也越来越广，至于有钱人，由于资产多、结构复杂，而且还可以跨国置业，所以征税难度确实大，但现在也开始准备对巨富海外投资收益征税。最重要的是，相比十几亿人口，这类巨富人口占比很低。

花些笔墨解释了下，直接说结论。

国税总局公布了几个关键信息：

1. 取得综合所得的人员中，无需缴纳个税的人员占比超过七成；

2. 剩余不到三成的实际缴税人员中，60%以上仅适用3%的最低档税率；

3. 年收入100万元以上的高收入者约占申报人数的1%，但这部分人申报缴纳的个税占全部个税的五成以上；

4. 申报收入位居全国前10%的个人缴纳的个税占全部个税的九成以上；

5. 2022年起征点从3500元提升到5千元后，个人纳税人从1.57亿下降到6512万人。

综合以上信息，可以将全国7.4亿就业人口按照2022年收入分层：

月入3500元以下：约6亿人（至少七成在3000以上）

月入3500-5000元：约1亿人

年入6万-10万：约4千万人

年入10万以上，仅2000万人

而这2000万人还可以进一步细分，考虑专项附加扣除，如果有房贷、有父母、有子女的情况下，一年扣除额度拉满是7.2万元，加上基础的6万元免税额度，年入16.8万以下就是3%的个税税率，但实际上很难拉满到7.2万元，找一个整数大概年入15万以下对应3%税率。

因此约2000万年入10万以上的人群中，约1200万在年入10-15万之间，大概对应月薪一万；而超过月薪一万的才仅仅不过千万人。

剩下的近千万人，大多数在年入百万以内，只有大概70万人左右超过年入百万。

此外结合其他与收入有关的统计，比如统计的住户调查、人社的企业薪酬调查，还可以把全国就业人口（包括全职、兼职、灵活就业等等）收入划分得更加清楚。

有几个比较值得注意的点位：月入3000元大概在15分位左右，50分位（中位数）大概是5000元，90分位才是1万元。

这才是真实的中国社会，收入太低、收入太高都不是一般大学生的常规收入范围，而哪怕从月入3000开始，每多赚一点，生活都会发生改变，而到了月入一万以后，剔除掉生活必需开支，每多赚一点对生活的提升更加明显。

需知月入1万和月入2万，从生活体验角度，收入差距不是2倍，可能是5倍甚至更多。就算收入一样，换一份工作时间更短、更能平衡生活、更喜欢的工作，不也是改变？

怎么能说努力无法改变人生呢？

大学生当然应该关心社会，但前提是关注的是真实的社会，而不是想象中高度抽象的虚假幻象。大学生刚刚离开应试教育，教科书以外没读多少书，社会阅历几近于无，没见过几个校园以外的人，也没走过几个地方。正因为没有具体的经验、阅历太浅，更容易用非常宏大的概念去解读自己遭遇的困难。

动不动就把自己遇到的具体困难，给概括化成年轻人如何、社会如何、中国如何、世界如何，就好像自己找到一份月入5000的工作，都需要世界和平作为前提，真的需要吗？

具体问题具体分析，遇到困难了，把困难具体化，想想自己通过什么努力能够改变，把改变的时间尺度以年为单位筹划，不要总妄想几天就跃升。更不要轻易把自己遇到的具体困难上升到国家、社会，改变不了大环境，只会让自己更加痛苦。

更何况，中国的进步肉眼可见，改变自己难道比一个国家的进步更难吗？
