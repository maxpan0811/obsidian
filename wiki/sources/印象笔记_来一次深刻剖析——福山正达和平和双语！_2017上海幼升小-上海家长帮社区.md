| |  | | --- | | 谈到浦东民办，很多家长经常会把福山正达和平和双语放在天平上比较，问问东，问问西，想知道这两个热门的民办小学到底该选哪家？到底哪家更适合自己的孩子？到底福山正达好在哪里？平和双语强在何处？……今天小帮就找到一些资深家长来谈谈这两所学校的全比较。    我从开始帮儿子择校起，一直没有真正注意过平和双语这个好学校。    因为印象里，这个学校学费比较贵，走读班很难考，偏向体制外，更重要的是有点高大上的姿态对不上我家工薪家庭的条件。    所以，也就没有好好用心去分析和研究过这个学校。平时也不知道该怎么准确地回答家长的这个问题，因为孩子的小学只有一次，慎重起见，今天花了点时间，整理了下平和双语的方方面面。    来一次深刻的剖析，给后来的家长们一点择校时的启示和思路，谈谈我自己对两个学校的看法。特别谢谢很多家长指出我的有些错误。我都会一一改正错误。 **学费**   **福山正达：**12000元/学期    书杂费0元，预收款100元，伙食费每月250元左右，校服费1496元  **平和走读：**19000元/学期，书杂费400元，预收款300元，伙食费2074元/学期，校服费2050元    一目了然，在基本费用方面，福山正达比起平和双语还是实惠许多的。    至于兴趣班费用，两个学校都有非常丰富的兴趣班，也都是收费的。福山正达的费用我比较清楚，平和我没找到。因为都是自愿报名，所以兴趣班费用，校车，保险费用就不讨论了。 **教学理念**   两个学校都是英语为特色的小学，，都是把孩子的品德教育和综合素质培养看得很重的，都重视孩子的良好习惯和礼仪的培养，都有非常丰富的校园活动，又都有着快乐的校园氛围。 **课程教学**   **福山正达**是高质量课堂，语数并重，英语特色。  **平和双语**是提倡轻负担，高质量的课程教学。    在教材上，两家学校也是几乎一样。语数都是上海市统编教材，都设有数学思维课。英文教材，福山正达是《牛津英语》和《香港朗文》。    平和也是《香港朗文》，加《牛津故事乐园》拓展阅读。2014年，平和首开的英语实验班是用的《Treasure Chest》。而福山正达的2个LA兴趣班是用的是同属加州语文懂的教材《Treasures》 。虽然同属加州语文系列教材，内容也接近，但还是有点区别的。    不过，我顿时感觉，这两家真是针尖对麦芒了。太有趣了！    **《Treasures》和《Treasure Chest》**    相同点：这两套教材每周的阅读技能，高频词汇，自然拼读，语法和写作部分都是一样的。    不同点：Treasures的目标学生是以英语为母语的学生， Treasure Chest 是针对英语为非母语的学生 **特色课程**   **福山正达：**英语探究课，国际理解，武术，信息课程   **平和双语：**双语教学，IT课程，游泳课程  两个学校都有常设全英语的特色课程，IT课程和体育健身的特色课程，都是比较中肯的。    **平和的优势：**  学校相当有国际范。平和的各种社团，各种活动都相当专业。平和的优势是12年制的和ib高中，初中高中的孩子能提供的资源可不一样，从小孩子们能看到是个很好的引导方式。    IB高中的含金量高，平和学校12年制后考入有名的国外大学的孩子很多。   **毕业班去向**   这是问题，我想是这两个很多方面非常相似的学校，最为家长们看重和想要了解的关键了吧。那我就好好展开来讲。    **福山正达：择校&对口**    福山正达今年第一届毕业生。详细情况还不得而知。所以，我们可以暂时参考公办福山的历年择校情况来看，公办福山大体为50%。那么福山正达的择校要求对于福山教育而言，第一届毕业生最底标准也该是50%，不过，从福山对福山正达的重视程度和创品牌的因素来看，肯定会高于这个数字的。    也能从现在得知的，今年五个班，已经录取华育中学16个同学，据说上外附中录取人数也不错。我翻了下，去年华育中学外区录取人数表，福山三个公办校区一共32位，也就是说，今年福山正达五个班将去年一半的名额收入了囊下。    去年外区民办里，七外录取25人，上外附小22人，那么福山正达基本可以确定，今年外区民办里华育录取人数排名应该第三不会错。    这个成绩对于第一届毕业班而言绝对够得上非常耀眼了。至于进入浦东本区招生主场的重点初中，我相信人数肯定更不在话下了。(对于初中择校，如果摊开讲，会很复杂，这里不多说了）   **平和双语：直升&择校&对口** 家长帮因为没有平和家长的详细介绍学校的资料，我从网上找到的资讯是：平和双语是50%直升，也就是一个班通过考试考察排名后，十几人直升入平和初中。    当然也有少部分不选择直升，出来择校。这也是出乎我意料的一点，我一直以为平和是全部直升初中部，九年制的。    我相信有很多家长都是这么认为的吧。（如果百分比有错，希望家长指出改正）    **平和的初中部到底强不强？**  平和的初中部除了小学部直升的一部分孩子外，每年还会有预备班面试外招。一般总人数为110个左右吧。直升平和高中部我们先放一放，后面讲，先从中考说起。我查阅了浦东一系列年来，浦东初中中考的数据排名，对平和初中的实力，我就有个一个非常清晰的认识。    在浦东还没有张江集团学校，新竹园中学，上海中学东校等这些优质民办和优质准民办的初中前，平和的初中部每年都是浦东中考的前一二名。实力之强，可见一斑。    但是，很快，张江集团第一届就坐稳了浦东第一的位置，2007开办的新竹园中学也只用了二年时间，就站住了浦东第二的位置，上中东随后来到第三。平和则年年退后一位，去年中考排在了第四。    今年的后起之秀，交中，建平远翔都将纷纷强势杀到，平和很极大可能在今年再退位。数据说明什么？说明，平和在浦东出现越来越多优质民办初中，优质公办之后，优势已经慢慢减弱，竞争力一直在下降。  **平和双语到底是不是偏体制外？**  我可以斩钉截铁地说，平和双语就是偏体制外的学校。首先，平和的初中部每年也是50%的孩子直升入平和的高中部。进入平和的高中部，也就意味着，就是出国读书。但每年，还是有部分平和孩子参加中考。我查了下数据，基本上，平和每年参加中考的人数都在48--58人之间。    再从2014年的一模详细表单科排名来分析。    平和的语文第 27名， 数学第8名，英语第4名，物理第5 名，化学第5名。可见，平和初中的语文和数学的基础课程都不强的，和新竹园比起来都不是差一点两点。    甚至语文水平属于非常难看的。有平和的家长纠正，平和的直升高中的孩子，在中考前就分开了。 那么除去直升的优秀学生外，参加中考的也都是小学的直升尖子和预备班招进的优等生，中考数学的成绩还是比地段生为主的几个公办初中低了，语文更是有差距，也侧面地看出，平和的语文，数学基本功的确不是很强。    所以，不能直升高中部的话，孩子去中考的优势并不明显。也说明了平和双语的确是个偏体制外的学校。    **平和的九年制到底该怎么评价？**    平和的九年制，会给孩子一个宽松的环境和综合发展的机遇。    但也必须努力才行。因为有一半的孩子是没有直升的。单拿平和快乐教育来讲，那么同为九年制的上海实验东校的性价比不可谓太高了。所以，小学的快乐素质教育，并不需要那么高大上，公办是一样的可以做到的，只要父母愿意。    **平和的小学部怎么看？**    就和平和早期初中部没有对手一样，之前平和一直是浦东民办最好的选择，所以，从生源到声势都是非常好的，没有对手的。但2010年福山正达的创办，让平和双语一家独大的局面，已经开始慢慢改变。    从今年幼升小报名情况来看，福山正达已经发起了强力的挑战。并且开始略占上风。毕竟走体制内的福山，也本该是更多平常家庭的选择。并且这种趋势一定会愈演愈烈下去。    明年福山正达的南校区又要开始招生了，新的一个校区的崛起，也代表着浦东每个民办学校的生源选择会再进一步压缩。竞争带来机遇，竞争带来活力，竞争带来动力，竞争带来优质，对于浦东民办而言就是好事。 **总结**   选择平和双语，还是福山正达这两个优秀的民办小学，就没有什么好纠结的。    就问家长自己，**准备给孩子怎样的路线。**    偏体制内，那么福山正达语数并重，英语特色，孩子综合发展，快乐的校园氛围，择校多样化就非常适合你。    准备孩子出国，或者给孩子一个快乐的，轻松的学习环境，综合发展孩子兴趣和特长，那么选**平和双语就是最好的。**    希望这篇文章能帮助家长们，对于择校有个新的启示和思考。    择校，要看着脚下，望着远方，揣着梦想，那么你为孩子的选择就对了！都是我自己找的资料和我的一些观点。大家参考。    学校之间不是单单讲一个好和不好，也不是别人说好就好，说不好就不好，而是要搞个明白到底合适我的地方在哪里？那么你都能看到每个学校属于自己独有的亮点。 | |
