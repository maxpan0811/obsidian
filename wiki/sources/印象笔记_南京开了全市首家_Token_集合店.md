PART 01

一件容易被看漏的事

全市首家 Token 集合店，到底在卖什么

7 月 1 号，南京干了一件事。可能很多人刷到了这条新闻但没当回事，觉得又是哪个地方政府搞的产业活动。但我仔细看完之后觉得，这件事可能是今年 AI 基础设施领域里最有标志性意义的事件之一。

麒麟科创园，**南京 Token 集合店**正式上线。**全市首个**。江宁区政府、南京城市算力网、南京移动联合运营。政府亲自下场签约共建。

![](.evernote-content/5789F2A2-5CBE-468B-BA6D-2307B9D91852/AD33A66D-FF94-449D-BC2A-2718F12BA70D.png)

你第一反应可能跟我一样。Token 集合店？这是什么新型零售？

不是零售。这个东西的分量比你想的大得多。

先用最通俗的话说清楚它在干什么。如果你平时在用 ChatGPT、DeepSeek、豆包这些 AI 工具，那你每天都在消耗 Token，只是你自己不知道而已。Token 是大模型处理信息的基本计量单位。你每次问 DeepSeek**帮我写个方案**，模型会把你的话拆成一个个 Token 来理解，再一个 Token 一个 Token 地吐出答案。1 个 Token 大约等于 1.5 个汉字。你用 AI 干的所有事情，聊天、写代码、做 PPT、分析数据，底下消耗的都是 Token。

过去南京的企业想用 AI 算力做业务，要自己去找算力供应商一家一家谈价格、一个一个搞技术对接。信息不透明，价格不统一，流程繁琐。大企业还能扛，中小企业根本就搞不定。现在这个**Token 集合店**做的事情，是把碎片化的算力资源整合成标准化的 Token 服务，企业按需采购就行。大模型微调、智能体开发、工业仿真、数据分析，需要多少算力买多少 Token。

![](.evernote-content/5789F2A2-5CBE-468B-BA6D-2307B9D91852/CAF09990-CD3B-4C6A-921C-54D62BA07A0D.png)

打个比方。过去你要用电，得自己买发电机，请人安装，自己维护，出了故障自己修。现在直接去供电局开户，插上插头就能用了。你不用关心电是哪个发电站发的，你只管用，按度数付费。

Token 集合店做的就是这件事。企业不用关心算力是哪台服务器、哪个数据中心提供的，它只管调用 AI 服务，按 Token 用量结算。

这个比方不是我随便打的。你往下看就知道，国家正在做的事情，跟这个比方说的一模一样。

![](.evernote-content/5789F2A2-5CBE-468B-BA6D-2307B9D91852/4811BC58-821A-4398-9928-A7508EFF6819.png)

PART 02

为什么是南京先开出来

政策、区位、算力、产业、网络，五个条件同时成熟

说到这里有一个问题我觉得值得想一想。全国那么多城市，为什么南京先开出来了？

其实不是偶然的。南京是国家人工智能创新应用先导区，江宁区是南京 AI 和软件产业的核心承载区，智能算力规模领跑全市。你看它手上的牌，腾讯华东云计算基地在这儿，中科逆熵在这儿，中科南京信息高铁研究院在这儿。算力供给有了，产业场景有了，市场需求也有了。缺的只是一个把供给和需求高效对接起来的平台。而且南京还有一个很多人不知道的优势，江苏省通信管理局在今年 3 月就联合多部门出台了方案，要在 2026 年底建成**城域 1 毫秒、省内 3 毫秒、长三角 5 毫秒**的算力时延圈。什么概念？你在南京调用算力的延迟，跟你打开手机 App 的延迟差不多。这个网络基础是很多城市还不具备的。

![](.evernote-content/5789F2A2-5CBE-468B-BA6D-2307B9D91852/5753E4C4-1690-4A2C-8F7A-04CFAE6AFBCB.png)

Token 集合店就是这个平台。

PART 03

一家集合店，串起五股力量

政府搭台，银行进场是最意外的一步

但真正让我觉得这件事分量不一样的，是签约仪式上的四个维度。

第一批是政企共建。江宁区政府跟南京城市算力网、南京移动签约，政府亲自下场搭台子。这件事的信号就很明确了，Token 这个生意不是民间自由生长的野草，是政府要来浇水施肥的正规作物。第二批是算力供给，Token 调度平台跟算力供应商签约，打通多元算力通道。第三批是产业需求，平台跟东南大学、五星控股、南京鑫合通感科技签约，把大学和企业的真实需求接进来。

![](.evernote-content/5789F2A2-5CBE-468B-BA6D-2307B9D91852/75A25E4B-FBFD-4300-926E-6F67652FC31F.png)

第四批是最让我意外的。

江苏银行南京分行跟 Token 调度平台签了金服合作协议，推出了**算力专项金融产品**，降低企业购买 Token 的资金门槛。

银行都进来了。

你想想看这件事。银行愿意给 Token 做金融配套，这件事你细想一下。在银行的风控体系里，Token 已经被当作一种**可量化、可交易、可定价**的标准化商品。它不再是一个技术概念，它是一个金融标的。银行愿意围绕它设计信贷产品，说明它在银行眼里跟原材料、设备、厂房这些传统资产一样，有明确的价值锚点。

企业买 Token 做 AI 开发，跟制造企业买钢材做产品、跟餐饮企业买食材做菜，在金融逻辑上已经是同一件事了。银行给 Token 做金融配套，就像银行给小微企业做原材料贷款一样自然。这说明 Token 在金融机构的眼里已经完成了从技术术语到**生产要素**的身份转换。

![](.evernote-content/5789F2A2-5CBE-468B-BA6D-2307B9D91852/0863E830-A538-43F5-9F62-A7753F6861ED.png)

银行进场意味着什么

银行是最保守的机构。它愿意围绕 Token 设计信贷产品，本身就是市场确定性的证明——如果不相信这是可持续、有真实需求的市场，银行不会冒这个风险。

反过来想，如果银行不相信 Token 是一个可持续的、有真实需求的市场，它不会冒这个风险去设计金融产品。银行是最保守的机构，它进场了，说明这个市场的确定性已经很高了。

这个信号非常关键。

PART 04

算力正在变成新的水电煤

六张网框架下，算力被定性为国家级基础设施

回到更大的图景。你可能最近刷新闻刷到过一个词，**六张网**。今年 4 月 28 号，中央政治局会议明确提出，加强水网、新型电网、算力网、新一代通信网、城市地下管网、物流网等规划建设。国家发改委主任郑栅洁说，今年这些领域的投资将超过 7 万亿元。

注意看第三张网。算力网。

算力网被正式纳入了国家**六张网**基础设施建设框架。跟水网、电网、通信网并列。

这件事的信号再清楚不过了。国家已经把算力定性为**跟水、电、燃气同一级别的基础设施**了。不是行业概念，不是技术名词，不是创业圈自嗨的 buzzword，是写进中央政治局会议纪要里的国家级基础设施。跟你家里的自来水管和电线是同一个级别。十五五规划纲要里，109 项重大工程的第一项就是**全国一体化算力网**。

![](.evernote-content/5789F2A2-5CBE-468B-BA6D-2307B9D91852/3DC062E7-84E4-41F7-BD0D-AAF9AF9763B7.png)

官方的说法是，算力网就是一张**算力版的国家电网**。用电不需要每家每户装发电机，因为大电网统一建设统一调配。算力网也一样，把全国各地的数据中心、超算中心连成一张网，统一调度，按需付费。

你看，前面我打的那个比方，**自己买发电机**变成到供电局开户，不是修辞，是国家正在做的事情。

而南京的 Token 集合店，就是这张**算力国家电网**在城市层面落地的一个商业实体。

从城市到国家

南京这一家店，本质上是国家级算力网络在城市层面的第一批商业实体。模式一旦跑通，就是全国复制的起点，不是南京一地的孤例。

PART 05

这件事的加速度

两年增长超千倍，需求正在倒逼供给

有一组数据可以帮你感受这件事的加速度。2024 年初，国内 AI 日均 Token 消耗量是千亿级。到 2026 年 3 月，这个数字飙升到了**140 万亿**。两年时间，增长超过**1000 倍**。一个 AI 视频镜头需要消耗 20 万 Token，一次自动驾驶测试需要消耗百万级 Token，全球开发者每调用 10 个 Token 就有 6 个以上由中国算力完成推理。当 AI 从聊天工具变成生产工具，Token 的消耗量就不是线性增长了，是指数级的爆发。这就是为什么**Token 工厂****Token 超市****Token 集合店**这些概念在半年之内密集出现。需求倒逼供给，供给催生基础设施。

所以你看到中国电信在 5 月份就已经推出了全国试商用的 Token 套餐，面向开发者、企业、个人和家庭提供全场景服务。中国电信宁夏分公司的**Token 工厂**集采项目规模达到**174 亿元**，这是三大运营商中首个以 Token 工厂命名的百亿级集采项目。无锡在 5 月也搞了城市级的**Token 超市**。弘信电子在无锡建大规模的**Token 工厂**。然后是 7 月 1 号南京的 Token 集合店。

这些事情密集发生在 2026 年上半年，不是巧合。是一个从中央到地方、从政策到市场、从运营商到银行的完整体系正在快速成型。算力这张网，正在以你感受不到的速度铺开。就像十年前 4G 网络铺开的时候你没什么感觉，但等你回过神来，移动支付、短视频、外卖平台已经彻底改变了你的生活方式。

算力网铺开之后会发生什么？说实话我也不敢下结论。但我觉得可以确定的是，AI 的使用门槛会大幅降低，降低到普通人和小微企业都能像用水用电一样使用 AI。

再想一步。南京这个 Token 集合店，政府搭台、运营商运营、银行配套金融、大学和企业接入需求。这个四方协作的模式一旦跑通，其他城市一定会复制。就像当年一个城市建了高铁站，周围的城市马上也要建。未来每个有产业基础的城市，都可能有自己的 Token 交易平台。

而且你注意一个细节。这个平台解决的是企业采购算力时成本高、资源调度难、供需匹配不畅的痛点。但平台是平台，企业拿到 Token 之后真正用起来，还需要有人帮它选模型、做方案、优化用量。这个服务层的工作，平台是不做的。这就给了大量中间服务商、渠道商、AI 解决方案公司新的机会。

对于普通人来说，这件事最直接的影响是什么？

AI 的使用成本会持续降低。过去你用 AI 要么自己买 API 按量付费，要么买订阅套餐。未来你可能像用水用电一样用 AI，按量计费、一键调用、政策补贴。中小企业不用再自己搭算力集群了，就像你不用自己建发电厂一样。算力变成了公共服务，跟自来水和市政用电一样。

这就是南京这家 Token 集合店背后真正的意义。它不是一个商业项目，它是一个时代信号。

PART 06

写在最后

从珍珠街到麒麟科创园，144 年后同一个逻辑

1882 年 9 月 4 号，一个叫爱迪生的人在纽约曼哈顿珍珠街建了全世界第一座中央发电站。在那之前，电是只有大工厂用得起的奢侈品。工厂要用电得自己买发电机，购买安装维护全套自理，成本极高。珍珠街发电站改变了这一切。它第一次把电从少数大工厂的专属资源，变成了普通人家里插上插头就能用的公共服务。接上线，按表计费，人人平等。

从那天起，电不再属于少数人。

**144 年后**的 2026 年 7 月 1 号，南京在麒麟科创园开了一家 Token 集合店。政府搭台，运营商供电，银行配套金融，大学和企业接入需求。做的事情跟 144 年前爱迪生在珍珠街做的，没有什么不同。

把算力接上全国一体化的网络，按 Token 精确计费，让所有人用得起 AI。

![](.evernote-content/5789F2A2-5CBE-468B-BA6D-2307B9D91852/D47EA46B-7E67-4D84-BF21-E15AB187D841.png)

写在最后

这可能是 2026 年 Token 赛道里最有标志性意义的事件之一。

不是因为它的规模最大，而是因为它第一次在城市层面把政府、运营商、银行、大学、企业五方力量完整地串了起来。模式一旦跑通了，复制就是时间问题。
