有一个官方的 Anthropic 插件，叫做 claude-code-setup。

它会扫描你的整个项目，并精确告诉你需要激活什么。

→ 需要设置哪些钩子

→ 需要安装哪些技能

→ 哪些 MCP 服务器适合你的技术栈

→ 需要启动哪些子代理

→ 每个项目的逐步配置

你只需运行一条命令：

/plugin install claude-code-setup@claude-plugins-official

它就会像资深工程师审查你的代码库一样，对你的设置进行审核。
