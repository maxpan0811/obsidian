# Aerial，让你的每一幅屏保都是 Apple 制作的航拍影片

---

### 多少人还在使用屏保

屏保 - 全称「**屏幕保护程序**」，是上个世纪遗留下来的时代产物。彼时，主流的电脑显示器还是 CRT 阴极射线管屏幕，也就是是我们熟悉的大屁股显示器（是时候暴露一波年龄了）。CRT 通过电子束撞击屏幕来显示图像，长期静止的画面会使得电子束持续的轰击屏幕的某一处，这样屏幕就会出现不可恢复的亮斑和阴影，久而久之就没法再用了。所以，当时的操作系统设计了屏幕保护程序来确保当用户停止操作电脑一段时间后有动态的画面出来保护屏幕。Win98 和 WinXP 时代最出名的屏保当属那个漫天飞舞的 Windows 徽标了。

然而，随着时间的推移，CRT 显示器逐渐被液晶显示器替代，如今，想找到一个大屁股已经不再是一件容易的事情。主流的液晶显示器使用背光技术或自发光技术，已经没有了当年 CRT 显示器的那个缺陷，屏保就再也不用肩负当年的使命，而人们赋予它的更多的就是观赏的价值了。纵观 Windows 发展历史上，从来都不乏漂亮到令人窒息的屏保，当年我也钟情过「**3D 水族馆**」这样美轮美奂的屏保，曾经也放着几首优美的轻音乐，就这么静静地看着各种热带鱼悠闲的在屏幕上畅游。

后来很长的一段时间，我一直没有使用屏保，换到了 Mac 平台以后，甚至连那些常见的屏保应用都没有了，我和屏保在一起的路越走越远。

### 重新点燃对屏保的热爱

其实我从来没有放弃过去寻找漂亮的屏保。一开始的时候，我逐个地尝试 Mac 自带的屏保：

![](.evernote-content/9ED1BE7E-355B-40D6-B98B-796AA50C7361/425A5744-FE70-409E-B42E-B2C7EB2D2BF3.jpg)macOS 默认的屏保

Mac 默认的屏保美则美矣，但是翻来覆去都是照片实在是有些乏味。最后选择了自带的 **iTunes 专辑封面屏保**，一方面我觉得大部分唱片专辑自带的封面都挺漂亮，密密麻麻的专辑封面变成动态磁贴在屏幕上随机翻转的效果还是蛮酷炫的，另一方面，这个封面还可以**直接点击播放**，可以交互的东西总是让人觉得很有意思。

可是没多久我觉得这样的屏保太不极客了，于是开始寻找第三方的屏保，终于在 Instagram 上面我发现了一个好东西——**Filco**，搜索这个标签你可以发现大量的美图：

![](.evernote-content/9ED1BE7E-355B-40D6-B98B-796AA50C7361/C977874E-D6A8-42E8-A3F1-2CDBCD00D7C1.jpg)Instagram 上的 #fliqlo 标签

Fliqlo 成为了我的宠儿，瞬间霸占了我的所有显示屏，黑底白字儿的翻页计时钟，有着浓浓的科技范儿以及和苹果 100% 匹配的简约风。很长一段时间，我都觉得，这就是我永远的屏保了！直到我遇到了它——Aerial。

### 众里寻他千百度

Aerial 是 JohnCoates 托管在 GitHub 上的一个开源项目，正如名字所说，该屏保视频取材自苹果零售店 Apple TV 的专用屏保——全部由苹果亲自制作的航拍影片。里面的每一个航拍都是不可多得的精美作品！配合 Mac 无可挑剔的 Retina 屏幕，用着 Aerial 我总有一种「看，这是朕给自己打下的江山」的错觉。

![](.evernote-content/9ED1BE7E-355B-40D6-B98B-796AA50C7361/5BC76149-7B3A-42F9-87A6-E75CDEF7DCB5.gif)薄雾中的金门大桥

*去金门大桥这么多次，从来没有在那么美的角度看过她，好想就这么呆呆地看着这个屏保，让时间静止下去。*

屏保的安装非常的简单，只需要 [下载 Aerial.zip](https://github.com/JohnCoates/Aerial/releases/download/v1.2/Aerial.zip) 文件，解压缩并双击安装就可以了，文件中只有屏保的描述文件，并不包含视频资源，预览和播放的时候才会自动下载。

> 使用 `Homebrew` 的用户可以直接通过 `brew cask install aerial` 命令进行安装（需先安装 `cask`）

初次使用之前，建议大家对屏保进行一些基本的配置。

打开 **系统偏好设置->桌面与屏幕保护程序->屏幕保护程序**，选择 **Aerial 屏保**并点击**屏幕保护程序选项**：

![](.evernote-content/9ED1BE7E-355B-40D6-B98B-796AA50C7361/6ABB1DB7-4244-4126-B5C3-AE00D7298706.jpg)屏保选项设置

在这里，可以勾选和预览自己喜欢的影片，航拍横跨了中国、拉斯维加斯、旧金山、夏威夷、迪拜等等地方的白天和夜晚的精美景象。因为每一个航拍体积都很大（200~400MB），所以不建议大家把 30 几个全部选中了，推荐通过预览来选择自己喜欢的场景。

影片会被缓存到 `~/Library/Caches/Aerial` 目录下，你可以到该目录下浏览或者删除相应的影片。

另外，值得一提的是，Aerial 在播放时会自动从苹果官方拉取资源下载影片，所以你的屏保会实时和官方保持更新。**Windows 平台和 Linux 平台也可以在 GitHub 仓库中找到对应平台的相应项目**。

最后，希望你们因此爱上屏保，享受屏保！

### 补充：

1. 评论中有人提到自己使用 macOS 10.11+ 在预览屏保时出现「你不能在当前版本的 macOS 中使用 Aerial 屏幕保护程序」错误提示。作者在原仓库中已经给出了解决办法：

**选中 Aerial 屏保，在保持选中的情况下关闭「系统偏好设置」，重新打开「系统偏好设置」，Aerial 即可恢复正常工作。**（这是一个已向 Apple 官方提交的关于 macOS/OS X 下使用 Swift 屏保的 Bug：[rdar://25569037](http://www.openradar.me/25569037)，如仍未正常工作，请参阅[该文档](http://alistairmcmillan.github.io/2016/04/25/OSX-Swift-screensaver-bug.html)使用另外的稳定版本进行安装。）
