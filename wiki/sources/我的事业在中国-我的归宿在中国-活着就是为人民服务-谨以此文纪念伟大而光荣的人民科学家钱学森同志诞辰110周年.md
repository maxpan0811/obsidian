# “我的事业在中国，我的归宿在中国。活着就是为人民服务”——谨以此文纪念伟大而光荣的人民科学家钱学森同志诞辰110周年

# “我的事业在中国，我的归宿在中国。活着就是为人民服务”——谨以此文纪念伟大而光荣的人民科学家钱学森同志诞辰110周年

原创王长安江湖歌行

![](attachments/7580d13d3ca86cf7.jpg)![](attachments/69b3e446e9b169db.jpg)

文/王长安

图源/网络侵删

钱学森，世界著名科学家，空气动力学家，新中国载人航天事业奠基人，中国科学院及中国工程院院士，中国两弹一星功勋奖章获得者，被誉为“中国航天之父”、“中国导弹之父”、“中国自动化控制之父”以及“火箭之王”。

对我国两弹一星工程的研究发展和整个国防事业都有着不可磨灭的重大贡献和深远影响。是二十世纪应用科学领域中最杰出的科学家之一，其40年代便和自己的恩师冯•卡门并驾齐驱成为航空航天领域最杰出的代表人物。

正如2007年感动中国颁奖词里的真切评价：“在他心里，国为重家为轻。科学最重，名利最轻。他是知识的宝藏，是科学的旗帜，是中华民族知识分子的典范”。

历史上的今天，也就是110年前的1911年12月11日，钱学森出生于江浙地区的一个知识分子家庭。

钱学森的父亲钱均夫曾经留学日本，学习历史、地理和教育，并且民国成立后在教育部里任职。

因此，钱学森三岁时便随父母北上进京，接受过“时髦”的西式教育的钱均夫从小便没让儿子读私塾，而是让他接受正规的现代教育。

钱学森到了北京后，陆续上过蒙养院，也就是类似今天的幼儿园，女师大附小（今北京市第二实验小学）、师大附小（今北京市第一实验小学）和师大附中。

出生在辛亥革命浪潮声中的钱学森，他的童年也就在这光怪陆离的新新民国的首善之地脚下成长起来。

得益于家庭环境和当时的学校的先进风潮影响。

钱学森小时候便立志，长大学有所成后要效法早期留美学人詹天佑，为中国的铁路工程做出贡献。

在中学教育这般一个人的人生重要时期，此时于师大附中学习理科的钱学森有幸遇到了一群很好的老师和当时中国算得上难得可贵的先进教育资源。

后来回忆这段人生时光的钱学森，对这些老师们的教诲更是感激不尽，深怀感情地说道：“我能为国家为人民做点事，也是与中小学老师的教育分不开的。”

1929年，中学毕业的钱学森因为受到当时孙中山先生所提出的“建国方略”的启发，而考入了上海交通大学机械系，立志以后要为着宏伟的铁路蓝图所贡献自己的力量。

可是，在步入大学后，苦心于学习的钱学森在历史的风潮席卷下，令饱受时代考验的他最后改变了初衷想法。

因为此时，狼子野心的日本帝国主义悍然发动了九一八事变，继而又发动了与一二八事变。

当时正在上海读书的钱学森，更是亲身经历目睹了由蔡廷锴将军率领的十九路军在上海慷慨悲壮抗击日本侵略者的激烈残酷的战争场面。

而面临国家赤裸裸地惨遭外敌侵略和肆意侮辱横行时，年轻的钱学森当然和其他所有进步青年一样怒发冲冠、满腔愤慨，从而也使他最终转变志趣投入到军事科学的研究何学习。

亲眼目睹日本侵略者的强大军事力量后，他认为中国要战胜日本侵略者，如果只靠军人们的英勇血性是远远不够的，还应当有现代化的军事武器，特别是强大的空军力量。

于是，他开始常常到图书馆里深刻钻研有关航空工程的书籍，他的人生志趣也从此由从小便期待未来为国家设计火车头逐渐转向了为国家航空事业发展所努力。

在交大毕业后，立志在航空事业里取得成就的钱学森，在认真思考了世界上航空领域发展的情况后，考取了清华大学留美研究生，决定赴美攻读飞机设计。

1935年8月，钱学森正式赴美，进入了美国麻省理工学院航空系继续学习，并且仅仅一年后便取得麻省理工学院的硕士学位。

可接下来的一个“麻烦”却阻碍了他继续学习、深耕研究。

因为，当时外国学生在美国航空工厂实习很不受欢迎，几乎没有机会。从而迫使钱学森的研究方向只好从航空工程渐渐转为航空理论。

虽然是受制于美国人的限制和“歧视”造成的客观条件不允许，但此时的钱学森还是有些懊恼的。

因为，先不说自己这么久的努力和满心期待，更因为在这个国弱势危的时代，相比对理论的探索研究而言，当前更急迫地可能还是航空工程的“实在”。

不过因为一个人的点拨，他马上便重新振奋起来，并且更加斗志昂扬、精神百倍。

这要提及到钱家的世交了，钱学森的父亲有位知交好友，是的没错，正是钱学森未来的岳父——民国时期的军事理论家蒋百里。

当时，蒋百里正在欧洲进行考察，他认为德国空军之所以能够得到迅速的发展，是因为得益于航空飞行理论扎实的深厚基础研究。

所以，他在得知钱学森在美学习的事情后，也很是鼓励钱学森钻研航空理论，学成以后可以更好为国家发展做出贡献。

于是，1936年重振斗志的钱学森再次千里迢迢从美国东海岸的波士顿赶往位于美国西海岸的加州理工学院；并且因为优异的成绩和谦虚和善的良好表现，师从于著名的空气动力学教授冯•卡门攻读博士。

后来，冯·卡门曾回忆两个人的相遇的时刻，说道：“我抬头看到一位个子不高，仪表严肃的年轻人，他异常准确地回答了我所有问题。他思维的敏捷和富于智慧，顿时给我以深刻的印象。”

![](attachments/5723221f68a7348b.jpg)

也正是在此期间，有心人天不负。年轻的钱学森正式开启了自己伟大而光辉的科学生涯。

在当时的世界上，新兴的航空工程工业面临着两大难题亟待科学工作者们来解决:

1.当飞机的速度接近音速时，空气的压缩对飞行器的性能有什么影响？

2.寻求超音速飞行器的理论指导与技术设计。

而这两个课题正是钱学森博士论文的研究方向。

1939年，钱学森以四篇博士论文获得了加州理工学院航空和数学的双博士学位，而这四篇论文则正是解决了上面所述的两大难题。

这便是科学史上著名的“热障”理论和卡门—钱学森公式。

它们是所有航空飞行器的指导理论，并且在二战期间乃至在二战以后很长一段时间，都被世界各国广泛应用于超音速飞机的设计与制造，直到电子计算器被广泛应用。

而此时，二次世界大战已经到了关键时刻。

为了赢得战争，表现突出的钱学森和恩师卡门在这一非常时间里，在美国政府的要求和邀请下加入了一项新武器的研究，这是一个人类军事科技史上具有革命性的武器——“飞弹”，随后更是成为了五角大厦导弹技术核心顾问组的成员，为美国未来的军事发展规划蓝图，乃至他参与美国建立的JPL的如今还在运转。

1945年，战争虽还为结束，其结果却似乎已经摊开。

盟军的胜利已是时间问题，为此美国罗斯福总统要求迅速派遣专家前往德国的军事工业重地探索和研究对手的先进技术隐秘，以图收缴德国掌握的军事科技和最新装备，并且争取优秀的德国军事科技人才来美国效力。

在当年四月底，就以冯•卡门为首组建了一个由三十六位专家组成的「科学咨询团」前往德国。

而钱学森正身处其间，受聘担任为其中火箭组的主任。其中钱学森参与讯问的第一位德国火箭专家便是二战中著名的德国 V1 与 V2 飞弹的总设计师，更是后来美国国家航空暨太空总署也就是NASA空间研究设计师，主持设计了阿波罗４号太空计划师的大名鼎鼎的冯．布劳恩。

并且在这次讯问中，钱学森让布劳恩写出的《德国液态燃料火箭发展及未来展望概论》，更是受到美国军方的重视兵将其和其的研究小组带往美国。

等这个所谓的“科学咨询团”从欧洲回到华盛顿后，便立即开始整理资料，最后向美国国防部提出了著名的“迈向新高度”的研究报告。

而这份远景规划的国防报告一共有九卷，其中的第３、４、６、７、８卷和技术情报附录是出自钱学森之手。

此时的钱学森更是为此走访了美国各处的研究中心和实验室，对美国航空事业的发展现况和未来潜力几乎可谓是有了全盘了解。

对于钱学森的贡献，美国专栏作家米尔顿．维奥斯特曾真切地写道：“钱（学森）是帮助美国成为世界第一流军事强国的科学家银河中的一颗明亮的星。”

![](attachments/db07000ac6bd985b.jpg)

在美国这些年的求学和实践的时光里，钱学森已然从当初身怀志向而远渡重洋的年轻人成长为了优秀的青年才俊、著名科学家。

直到1947年夏天，第二次世界大战早已落幕，此时已经功成名就的钱学森归国探亲，并拜访了他的母校上海交通大学，并且在交通大学、浙江大学、和清华大学作学术演讲。

也正是在这段探亲期间，钱学森在上海与父亲挚友蒋百里的女儿，青梅竹马的蒋英女士正式结婚。

不过不久，因为在美的科研事业和工作的忙碌和催促，以及在当时国内正处特殊情形下，不愿回来为国民党“装点门面”的他重新踏上了出国之路。

新中国成立后，呼吁海外学子归国建设，在这国家和人民真正且急切需要的时候，钱学森提出了回国申请。

可直到1955年，在历经五年磨难和美国政府无端刁难和恐吓的钱学森与夫人蒋英以及一双子女终于搭乘轮船，毅然决然地踏上了回国的路。

关于钱学森回国的故事，我想在这里应该不需要小白作过多赘叙了，相信大家都早已耳熟能详了。犹记得小时候语文课文中就有一篇有关钱学森回国的文章，还有更不用说历史课了。

那句经典名言:“一个钱学森，抵得上美国五个海军陆战师，不能放他走！”更是从此成为这件事件的见证和注脚。

但这些年不少西方媒体都将钱学森回国的原因简单归于当时麦卡锡主义的盛行就罢了，网上有些人不知何意？还时常阴阳怪气跳出来不说是附和了，那更是借题发挥，更胜一筹啊。

其实，这些人的那些薄见之薄见，小白都不想多说什么。因为不说完全是小人之心吧，起码这些人就没明白一点，主次不分、本末倒置。

在钱学森当年踏出国门时，他的父亲就曾寄语他，“人，生当有品：如哲、如仁、如义、如智、如忠、如悌、如孝”。吾儿此次西行，非其夙志，当轻轻然而归，灿灿然而返。”

终于如愿以偿地回到祖国后，钱学森被安排在了中国科学院工作，并且负责筹备建立力学研究所。

不过，因为对国内的工业建设和技术水平还不太了解，中央和科学院的领导们决定先安排他去当时中国的重工业中心东北参观考察。

也正是这次东北之行不仅改变了钱学森以后的命运，更是彻底改变了我国的国防事业。

钱学森在参观完哈尔滨军事工程学院后，经过深思熟虑和调查研究向党中央郑重提出了《建立中国国防航空工业的意见》。

在次月，经过党中央的严谨讨论后，由国务院制订《一九五六年至一九六七年科学技术发展远景规划纲要(草案)》，其中提出要在十二年内使中国喷气和火箭技术走上独立发展的道路。

当年4月，成立了中华人民共和国航空工业委员会，统一领导中国的航空和火箭事业，由聂荣臻任主任，黄克诚、赵尔陆任副主任。

而也正是在这次参观结束后，某一次晚宴上，因为在朝鲜战场上见识过了美国人的高端先进武器，对于未来国防建设的考虑，国家领导们对此很是重视。

陈赓大将便借此机会曾问钱学森道：“钱先生，中国人自己搞导弹行不行？”

钱学森却不假思索地回答道：“有什么不能的？外国人能造出来的，我们中国人同样能造出来。难道中国人比外国人矮一截不成？”

陈赓大将听了这句话后，更是非常高兴，说：“我就等着你这句话呢！”

毛主席更是对他大加赞赏和期待，你就是咱们民族发展的希望，不要觉得不好意思，咱们国家的火箭发展离不开你这样的科学家！并且鼓励他要多为新中国培养大量的年轻科技人才。

“学森同志，我们中国的科学工作者，一定要立足于自力更生，敢于走前人没有走过的路！”

深知自主掌握研发尖端武器重要性和必要性的毛主席，对钱学森来说可谓是英雄所见略同。

正如，钱学森后来在文章中所写道，毛主席的教导，拨正了我做科技工作的方向。

并且留下那句名言:“手中有没有剑，和用不用剑，那是两回事。”

所以，从10月开始，钱学森在科研工作中，又多了一个重要工作。

那便是为国家培养年轻人才，给年轻的科研人员教授有关导弹的基础理论，他为此更是专门编撰了一部基础教材《导弹概论》。

而正是这部经典著作，启蒙了新中国第一代从事导弹和火箭研制的中国人。

正因如此重要性，钱学森成为了国内外敌人优先除掉的目标。

![](attachments/a44108d0c332cd5f.jpg)

在这种背景下，新中国的科研事业开始轰鸣起来，日渐蒸蒸日上。

不过，好景不长。因为中苏关系恶化后，苏联不仅单方面撕毁了大量的已有的科技合作条约，更是直接撤走了所有的技术专家。

苏联领导人更是放言，“离开了苏联援助，中国的导弹永远上不了天！”

可面对这样突如其来的打击，钱学森却毅然决然地站了出来，对党和国家表示：没有专家也不怕，我们自己也能拼出一番新天地来！

为此，他身心更加彻底、百分之一百二的投入在新中国的导弹事业上，三天两头要出差，在酒泉基地一待就是十天半个月，甚至一个月。而且行踪更是严格保密，连妻子蒋英都不知道，一度非常担心“失踪”的他。

就是在这种各类艰难险阻和艰苦奋斗下，钱学森等科学家们，带着不服输的精神，埋头苦干、克服万难。

1961年11月5日，我国第1枚近程导弹“东风1号”顺利完成飞行试验。

1964年10月16日，我国第一枚原子弹爆炸成功，震惊世界。

在同年的12月26日，毛主席的生日这天，钱学森受邀一起到家吃饭。

在餐桌上，毛主席一如既往的风趣却饱含感情，对钱学森说：“你是导弹专家，依靠你，使我国的原子弹早日爆炸。”

正是在这样无私奉献、艰苦奋斗、坚持自力更生的精神下，新中国的强大国防从无到有、再到更好，更是完成了从零到一的这种伟大转变。

1966年10月27日，中国首次用中近程导弹运载原子弹的飞行实验成功；

1967年6月17日，中国第一颗氢弹空爆试验成功；

1970年4月24日，“东方红一号”闪耀于太空之上，中国第一颗人造卫星发射成功；

2003年10月15日，神舟五号飞船顺利升空，中国首次载人航天飞行成功。

……

中国航天建设事业从一片空白，到第一枚导弹，第一颗卫星，第一艘载人飞船等来回报中国人民的坚贞和奋斗。航天这条长征路上，中国人民也是越来越自信，从中国制造到中国创造，而所谓“航天超级大国”们有的，我们都有；他们所没有的，我们也必将会有的。

1976年9月9日，毛主席在北京与世长辞，而视毛主席为良师益友的钱学森悲痛万分，他在《人民日报》撰写了这篇《终身不忘毛主席的亲切教诲》的文章，回忆二十余年来，毛主席对他的教导与鼓励。

1982年，此时已经从国防科研领导岗位退下来的钱学森已经71岁了。不过他并没有真正退休，而是再次回到学术理论研究当中。

他非常看重自己晚年的科研成果，曾对自己的堂妹钱学敏说：“我这些年来和你们一起研究和探讨的这些问题与设想，才是我回国以后开创性的、全新的观点和理念。它的社会意义和对现代科学技术发展的重要性，可能要远远超过我对中国‘两弹一星’的贡献。”

对于此，今天我们可以简单探讨一下。比如这阵子很热的“元宇宙”，20世纪90年代初，钱学森开始了解到“Virtual Reality”（虚拟现实技术），便立刻想到将之应用于人机结合和人脑开发的层面上，并给其取名为“灵境”。

钱学森曾在给友人的信中写道：“我特别喜欢‘灵境’，中国味特浓。”

这种中国科研工作者的人文浪漫，在钱老身上更是体现的淋漓至尽，不仅有严谨的科学精神，更有丰富的科学思想。

其中如，他在1987年9月全国科技名词委工作会议上说：“space这个词有的译为‘空间’，给人感觉是空空洞洞的。我根据毛主席‘巡天遥看一千河’诗句，建议把它订为‘航天’。‘航天’连同‘航天飞机’，一并被接受了。我考虑在太阳系飞行叫‘航天’，在空气中飞行叫‘航空’，将来在银河星系以及大宇宙中飞行可叫‘航宇’。”

还有，当时他还给国务院副总理写信建议发展电动车。

当然，钱老的视角肯定不只是为了环保，相对而言更多是为了换道超车，他曾预测到将来汽车保有量将超过2000万（实际上现在超2亿辆了），因为只有减少对原油的依赖，才能确保能源安全。

诸类这些还有很多，有的现在正在进行、实现和丰富完善，有的等待人们继续探讨研究等等。

无论是在从事导弹航天期间，还是从国防科研一线领导岗位退下来以后，钱老为了及时跟踪世界科技和时政前沿，仍在时刻注意着世界科研发展动态，购买何订阅各种最新出版的书籍、不同领域的杂志。

乃至，直到2009年10月31日去世的前两天，他依然坚持在读书看报。

今天的我们来回顾钱老的光辉而璀璨的一生，其实最好不过的也正如他晚年时自己所说的，一生中三个最激动的时刻。

第一个便是学有所成、来自良师益友的认可，即当年恩师冯·卡门告别前的一番赠言：

“他跟我说，‘现在在学术上你已经超过我了’，我一听他这句话，激动极了”。

而师徒两人之感情更是情深意重，冯·卡门在回忆录里，曾深深感慨道，“他是一个无可置疑的天才，他的工作大大促进了高速空气动力学和喷气推进科学的发展。他的这种天资是我不常遇到的。我发现他非常富有想象力，他具有天赋的数学才智。人们都这样说，似乎是我发现了钱学森，其实，是钱学森发现了我。”

第二次，则是加入中国共产党，为共产主义事业奋斗终生。

“在建国十周年的时候，我被接纳为中国共产党的一员。这个时候我心情非常激动，我是一名中国共产党党员了！我简直激动得睡不着觉。”

第三次，则是真正成为光荣、优秀的广大劳动人民中的一员。

1991年，钱学森在被授予国家杰出科学家奖章时，恰逢《史来贺传》出版，他读到《史来贺传》的序中写道：“雷锋、焦裕禄、王进喜、史来贺跟钱学森作为四十年来在群众中，享有崇高威望的共产党员的优秀代表”。

钱老对此曾激动道，“简直心情激动极了”。在他讲到这里时，眼中更是泛起了泪花，声音也几度哽咽。

此时的钱老，应该会时常想起当年与毛主席初次见面时，毛主席突然问他的那个问题吧。

“除了工程控制论的研究外，你最大的特长是什么？”

“主席，我作为一名科级工作者，活着的目的就是为人民服务。如果人民最后对我一生所做的各种工作表示满意的话，那才是最高的奖赏。”

所以，中国人民到底行不行？不是靠那些只会嗡嗡叫的苍蝇们可以论定的，而是靠我们一步步稳重的脚印和理论实践来证明的。

如今我们的天宫，熠熠生辉于星河间，为探索星空奋斗的科研人员提供服务；

我们的北斗，笼罩这颗小小寰球，为苍穹盖下辛勤的劳动人民指引方向；

我们的嫦娥，奔向九天拥揽月盘，为广寒宫里千年的民族梦萦实现理想。

而那些在奋力于跳梁之间的流辈，此时又能如何？

谨以以下几段主席诗句送于其流和所有同胞:

红军不怕远征难，万水千山只等闲。

天若有情天亦老，人间正道是沧桑。

独有英雄驱虎豹，更无豪杰怕熊罴。

梅花欢喜漫天雪，冻死苍蝇未足奇。

“我们的事业是正义的。正义的事业是任何敌人也攻不破的。我们正在做我们的前人从来没有做过的极其光荣伟大的事业。我们的目的一定要达到。我们的目的一定能够达到。”

“让那些内外反动派在我们面前发抖罢，让他们去说我们这也不行那也不行罢，中国人民的不屈不挠的努力必将稳步地达到自己的目的。”

谨以此文纪念伟大而光荣的人民科学家钱老钱学森同志诞辰110周年。
