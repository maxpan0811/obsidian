# 飞鹰2组——山东渠道管理数据分析

> 飞鹰2组（汇报人薛玖琳）于2025年3月7日做的山东区域渠道管理数据分析报告。

## 2024年山东渠道数据
- 山东全年任务完成约**3345万**
- 全年渠道客户欧洲**2740万**，达成率**96%**
- 欧洲占84%（西欧占欧洲53%），非欧占16%（中东非占非欧53%）

## 五大客户分析
| 客户 | 策略 |
|------|------|
| 携程渠道 | 目标2100万（培训+沙龙+加返政策） |
| 青岛国旅 | 目标1500万（加大包团会销） |
| 银座旅游（88家门店） | 增1名专职销售，目标增长300万 |
| 众信 | 目标签约250万 |
| 永安国旅（32家门店） | 已邀老板参加考察团，达成合作 |

## 竞对对比
| 供应商 | 营业额 | 占比 |
|--------|--------|------|
| 华程 | 896万 | 26% |
| 竹园 | 743万 | 21% |
| 优耐德 | 420万 | 12% |

### 供应商产品对比
- **竹园**：非常意大利德法意瑞占比西欧36%，客单价20,300元
- **优耐德**：4国占比41%，法瑞意+瑞士4晚
- **华程**：第一西欧，第二英国
- **山东市场特点**：西欧平均客单价1.6-2万的流量产品为主

## 济南门店数据
- 39家门店，开单30家，开单率77%
- 华程开单20家，占66%
- 华城青岛携程程信比0.53（济南0.81）
- **青岛竞对劣势来源**：竹园深度合作、中腰部门店渗透率不足

## 改进方案
1. 针对竹园主流产品打造有竞争力的流量产品（前期降低毛利，培育市场）
2. 加大对携程门店拜访、培训、沙龙、更新加返政策
3. 提升程信比到1（济南目标1.2）

## 银座旅游分析
- 88家门店，24年欧洲容量900万
- 华程占比6% vs 众信系24%、凯撒20%
- 策略：首单立减活动、联合总部产品培训会
