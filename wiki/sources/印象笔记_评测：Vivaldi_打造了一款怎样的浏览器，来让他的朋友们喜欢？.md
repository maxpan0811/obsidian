# 评测：Vivaldi 打造了一款怎样的浏览器，来让他的朋友们喜欢？

---

![](.evernote-content/E99D5638-F3DA-427F-B69F-F5E432688D61/7152D6F3-9CF6-4893-865A-1C0495956C73.jpg)

> 答案是，打造一款任何人都喜欢的浏览器。

在说一款浏览器适合任何人，可能有一些夸张，但这款叫做 Vivaldi 的浏览器，至少说能适合很多人。

可能许多人还是第一次听说这一款浏览器，因为它刚刚发布了 1.0 正式版本，但实际上它已经经历了漫长的内测和公测。没听说过不要紧，但你肯定会欣赏它的「血统」——这款浏览器由 Opera 联合创始人、前 CEO 谭咏文打造。由于 Opera 越来越偏离他所梦想的浏览器（被某数字收购？），所以他离开 Opera 开始制作了这样一款「为我们的朋友们打造」的 Vivaldi 浏览器，希望让那些真正重视浏览器体验的深度用户能够喜欢。

不过，怎样打造一款让不同要求和偏好的朋友们都喜欢的浏览器呢？**也许 Vivaldi 浏览器高自由度的自定义项和十八般武艺，就是它给出的答卷。**

什么都喜欢，什么都会
----------

在 Vivaldi 的官网上白纸黑字地写着「无所不能，无所不包」。已经很久没有人想要打造一款「什么都会」的 web 浏览器了。由 Chrome 引领的风潮开始，许多浏览器开始选择做一个更加纯粹和简单的浏览器，做好浏览器的本职工作，而把相关的扩展功能交给插件或扩展程序。但显然，Vivaldi 与 Chrome 的思路是截然不同的。

他们不希望打造一款简洁的浏览器，但将功能交给扩展程序。他们希望打造的是一款能适应不同用户，了解用户喜好，能让他们充分自定义的浏览器。除了普通用户，Vivaldi 希望能获得深度用户的喜欢，因此在打造这款浏览器的时候，他们还为浏览器添加了许多提高效率的功能，以配合那些喜欢使用键盘快捷键、对界面设计有强迫症、或甚至是一次性打开好几十个标签页的 Power user。

这就是最好用的标签栏，没有其他可能了
------------------

深度用户的浏览器标签栏可能是什么样子？曾经看过朋友在使用浏览器时打开几十个网页，以随时查看和参考不同网页上的内容。而我在阅读新闻网站时的习惯也是一次性在后台打开多篇新闻，这样在阅读一篇新闻的时候，后续的新闻就已经加载完毕。显然，打开多个标签页会让标签栏变得窄小拥挤，因此浏览器对标签栏有着良好的管理才能让你更得心应手地浏览网页。

Vivaldi 拥有强大管理功能的标签栏，为你解决了这个棘手的问题。

在普通的情况下，Vivaldi 的标签栏和其他浏览器没有什么两样。不过，**标签栏预览**功能可以让你更简单地找到你要切换的标签页。如果你懒得把鼠标放到标签页上预览，你甚至可以向下拖动标签栏，**让标签栏始终以预览图的形式显示**。当打开的页面过多的极端情况下，Vivaldi 的标签页会缩小成一个图标，但只要你把鼠标放到标签栏上查看预览图，也不会搞错你要切换的标签页。

![](.evernote-content/E99D5638-F3DA-427F-B69F-F5E432688D61/755C7F64-7A3C-4EF4-8D35-41A2058D8236.jpg) *▲ 悬停鼠标来查看标签页预览，或直接向下拖动来扩大标签栏面积*

即便可以查看预览图，太窄小的标签页按钮依然会让切换标签变得困难。在这样的情况下，Vivaldi 的**标签页分组**功能可以有效提高你的效率。你只需要把一个标签页拖动到另一个标签页就可以将他们归为一个标签组，当然，你也可以在右键菜单中随意解散标签组。更简单地使用标签组的方法是让 Vivaldi 为你自动归类相似的标签，你只需要在某个标签的邮件菜单中选择**归类相似标签**，Vivaldi 就会自动把相似内容的网页归为一个标签组，还你一个干净的标签栏。你不用担心如何点选标签组中的标签，别忘了当你把鼠标放上去所有网页的预览图都可以展开，非常容易点选。

![](.evernote-content/E99D5638-F3DA-427F-B69F-F5E432688D61/38E001F9-F563-4DD5-A601-7DEFFF13B929.jpg)

*▲ 将相似标签归组，并通过标签页预览来方便地切换*

如果你有时候需要打开多个网页，并对比查看不同网页中的内容，那你一定爱死那些可以平铺双窗口的浏览器了。别急，我既然提到了这个功能，那 Vialdi 就一定会有，甚至做得更好。

当你把任何标签归为了标签组，你就可以在右键菜单中选择**平铺标签组**。无论你有两个标签页或是五个、十个，都可以平铺起来让你可以同时查看。如果你的显示器足够大，大概你想平铺几十个标签页也没有问题……希望你没有被吓到。

![](.evernote-content/E99D5638-F3DA-427F-B69F-F5E432688D61/B06CADF4-2A14-4978-9EC8-255697C203C9.jpg)

*▲ 平铺标签组里的所有标签页*

话说回来，打开太多的标签页可能会占用你的电脑许多内存，但如果你就是不想关闭它们该怎么办呢？你可以在标签上右键点击选择「**标签休眠**」功能，或是「休眠后台标签」。Vivaldi 将会把这些标签页从内存中移除但不关闭，以节省内存。你可以在需要的时候随时重新加载它们。

不用费心设置的快速拨号
-----------

大多数的浏览行为都从标签页开始。为了让这个页面变得有意义，大部分浏览器都会添加一个快速拨号的功能，让你从新标签页快速访问常用网站。

Vivaldi 浏览器也拥有一个与 Opera 类似的快速拨号页面。不过与其他浏览器最大的不同是，Vivaldi 的快速拨号页面让你可以**把任意一个书签文件夹作为快速拨号**。你甚至可以放置多个书签文件夹到快速拨号页面以随时切换。此外，你还可以在新标签页直接管理书签和历史记录。

![](.evernote-content/E99D5638-F3DA-427F-B69F-F5E432688D61/D89EB78C-3ECD-4D94-ABB9-6737CCE9851D.jpg)

让侧边栏更具实用意义
----------

我以前非常讨厌某些浏览器中的侧边栏，因为实际上它和效率这个词的关系不大，那鸡肋的几个功能完全可以在功能栏上实现，还何必要打开一个侧边栏。不过，Vialdi 浏览器的侧边栏让我的想法有所改变。

按下 F4 就可以打开 Vivaldi 的侧边栏。Vivaldi 的侧边栏除了自带的查看书签、下载和笔记的功能，你还能在侧边栏中自由添加**web 面板**。这意味着你可以将任何网站添加到侧边栏中，并且 Vivaldi 会将侧边栏中的网站自动转换为适合更细长视图的移动网站视图，这非常适合添加一些需要常常打开的社交网站。比起其他浏览器中匮乏的侧边栏应用，Vivaldi 的 web 面板更具灵活和多样性。

![](.evernote-content/E99D5638-F3DA-427F-B69F-F5E432688D61/3B878DA5-092A-428B-8281-41DEBFD5BE5C.jpg)

![](.evernote-content/E99D5638-F3DA-427F-B69F-F5E432688D61/92321DA4-3FF8-4408-9E30-8BD485CECADF.jpg)

![](.evernote-content/E99D5638-F3DA-427F-B69F-F5E432688D61/179182C1-5713-4735-A446-D29CDBA9532E.jpg)

*▲ Vivaldi 侧边栏中的笔记可在记录笔记时记录网站和截图。**Web 面板功能让你可在侧边栏自由添加网站。*

键盘党福音：快捷指令与完全键盘操作
-----------------

如果你习惯了 Spotlight 或是 Cotana 的搜索框，你应该会觉得某些时候搜索的效率会非常高。正因为如此，Vivaldi 为你提供了一个*\*快捷指令*的面板，让你可以用搜索的方式快速在 Vivaldi 中查找内容。你可以按下 F2 键快速打开快捷指令，它会像 Spotlight 那样在屏幕中央显示一个搜索框，你可以直接搜索书签、历史、命令或者设置项，而不需要去菜单里寻找它们。

![](.evernote-content/E99D5638-F3DA-427F-B69F-F5E432688D61/6F98156F-28A6-4EAE-81A1-A698D2CE689C.jpg)

许多深度用户都是键盘控，除了快捷指令可以很大程度上方便键盘党，Vivaldi 还对**完全键盘操作**提供了友好的支持。你可以在「设置 - 键盘」中启用「完全键盘操作」，之后你就可以使用 Tab 键循环聚焦整个网页中的按钮或控件，完全使用键盘来浏览和操作网页。

一千个用户眼中就有一千个 Vivaldi
--------------------

我们对浏览器的要求各不相同，既然这样，与其为用户选择，不如让用户自由选择他们想要的设计。Vivaldi 浏览器有着非常高自由度的自定义功能，你可以自由更改配色，以及安排各种工具栏的位置，来组合你自己喜欢的视觉效果和工作空间。

比如，Vivaldi 浏览器默认可以根据不同网页的配色来对标签栏进行着色，你可以在设置里关闭这个功能，也可以选择是活动标签着色还是标签栏背景着色。你甚至可以调换地址栏和标签栏的位置，放在底部或是左边，甚至关闭全部用户界面（虽然我不知道为什么要这样做），总之你可以把它们安排成你最顺手的样子。

![](.evernote-content/E99D5638-F3DA-427F-B69F-F5E432688D61/3B723E01-61B0-46C6-9E57-2EC74007C1A0.jpg)

![](.evernote-content/E99D5638-F3DA-427F-B69F-F5E432688D61/DC0F63E8-BF56-4008-919E-1C37B35DE15E.jpg)

*▲ 自由安排你的 Vivaldi 界面*

Vivaldi 的设置项与 Chrome 相比非常之多，除了以上提到的一些功能，Vivaldi 也支持鼠标手势，支持 Chrome 扩展程序，你可以好好研究一番，以组合成最适合你的状态。不过比较遗憾的是，可能由于刚刚起步，Vivaldi 浏览器目前还没有添加同步功能，不过这也没有太大的关系，因为我一直使用 xMarks 和 Lastpass （1Password）来同步书签和密码，也推荐给你。

![](.evernote-content/E99D5638-F3DA-427F-B69F-F5E432688D61/56680772-6A6A-485E-B88C-1780B98F0B5E.jpg)

「这是为我们的朋友打造的浏览器」
----------------

> 1994 年，两名程序员开始研发能够在有限的硬件性能下运行得足够快的网页浏览器，他们谨记用户的需求和愿望。Opera 从此诞生。
>
> 我们的小软件长大了，我们的小组扩编了，我们组成了一个社区。我们与用户紧密结合，基于你们的反馈以及我们认定的造就一个伟大浏览器的想法，不断完善着我们的软件。我们始终创新并追求卓越。
>
> 时间推进到 2015 年。我们曾经钟爱的浏览器已经转变了前进的方向。我们遗憾地看到，它不再服务于用户和贡献者社区——那些从一开始就帮助建立了这个浏览器的人们。所以我们得出了一个自然而然的结论：为我们自己和我们的朋友们，我们必须要打造一个全新的浏览器；一个快速、多能又高效的，把用户需求放在首位的浏览器；一个为你而生的浏览器。
>
> 于是，Vivaldi 诞生了。
>
> —Jon

Vivaldi CEO Jon 认为，目前大多数浏览器都在使自己变得简洁，但仍然有很多人想做一些与众不同的事，他们希望浏览器也能这样。这样的想法让 Vivaldi 浏览器尽力去了解用户，并且让用户意识到他们是独特的。因此，对于 Vivaldi 来说，**他们首要考虑的不是去教育用户，改变用户的习惯，而是去尽力满足用户的需求**。

如果你厌倦了那些简洁的浏览器，并希望掌控自己的浏览体验，别怀疑，你可能就是这些独特的用户中的其中一员，你就是 Vivaldi 口中的朋友。

Vivaldi 浏览器提供了 Windows / Mac / Linux 版本，你可以前往官网下载，软件本身对中文有良好的支持。另外，由于 Vivaldi 官网 设置了谷歌的人机检测，下载时 *有可能*需要你自备梯子。
