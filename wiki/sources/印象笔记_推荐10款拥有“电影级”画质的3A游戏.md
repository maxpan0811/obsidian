### 二、[荒野大镖客](https://zhida.zhihu.com/search?content_id=222693861&content_type=Article&match_order=1&q=%E8%8D%92%E9%87%8E%E5%A4%A7%E9%95%96%E5%AE%A2&zhida_source=entity)2

　　荒野大镖客2的画面在以往乃至现在看来，已经达到了很高的水平。其游戏画面不仅为我们展现了一个真实震撼的西部世界，而且还带给我们一种全新的体验。本作中将会有超过30种不同种类的武器，60多个不同场景，50多个任务和支线以及其他内容。这是一款开放世界生存游戏——它将会把你带到一个真实存在的游戏世界。游戏中的每一寸土地、建筑物，甚至是每个细节都可以被玩家以自由探索相结合的方式进行制作，使之成为独一无二的游戏体验。在画质方面它可以说是顶级水准了——从建模到场景和武器都非常真实;细节纹理也非常丰富，这使得整个游戏环境中的光线非常漂亮;同时也能让玩家有一种身临其境感觉，比如玩家可以在一个广阔、开放世界里自由走动等等。除此之外，在游玩《荒野大镖客2》中时还能体验到各种不同于以往任何游戏作品的新玩法：包括骑马、驾驶载具、射击等多种不同类型。并且《荒野大镖客2》还拥有众多强大的DLC和全新角色与故事模式，而且游戏还能用VR设备游玩哦。

### 三、刺客信条：大革命

　　这是一款高画质的动作冒险游戏。玩家可以在游戏中体验到巴黎的真实风貌，还可以选择不同的职业来解锁不同的能力。玩家在游戏中需要通过探索获得资源，来解锁技能点或是升级武器装备来提升能力，并且可以学习新技能或者是进行战斗。游戏整体的画面效果非常不错，同时画质还有着极高的水平。《刺客信条：大革命》不仅拥有非常优秀的3D视觉体验，而且它还有着极其强大的开放世界。这个世界里充满了刺激和惊险，但是也不缺乏人文关怀，是一款不错的游戏。

### 四、[艾尔登法环](https://zhida.zhihu.com/search?content_id=222693861&content_type=Article&match_order=1&q=%E8%89%BE%E5%B0%94%E7%99%BB%E6%B3%95%E7%8E%AF&zhida_source=entity)

### 五、巫师3

　　在游戏中，玩家将会扮演一个名叫杰洛特的巫师，它是一名充满传奇色彩的赏金猎人。虽然他的故事可能不太广为人知，但他确实是一名非常出色的猎人，并且在所有猎人中有着最好的声誉，这使得他成为玩家最想要去探索的目标之一。这款游戏给人一种身临其境感觉——它不仅能给玩家带来前所未有的游戏体验，而且还能给我们带来全新的RPG体验。而且由于巫师3有着非常高水平的画质表现，所以其画面水平在同类游戏中也是一流水准，而它在游戏里所展现出来的那种逼真感觉会让人爱不释手!该作包含众多[RPG元素![](data:image/svg+xml,%3csvg xmlns='http://www.w3.org/2000/svg' width='10px' height='10px' viewBox='0 0 16 16' class='ZDI ZDI--FourPointedStar16 css-1dvsrp js-evernote-checked' fill='currentColor' data-evernote-id='121'%3e%3cpath d='m5.068 9.267-3.08-.77a.512.512 0 0 1 0-.994l3.08-.77a2.289 2.289 0 0 0 1.665-1.665l.77-3.08a.512.512 0 0 1 .994 0l.77 3.08c.205.82.845 1.46 1.665 1.665l3.08.77a.512.512 0 0 1 0 .994l-3.08.77a2.29 2.29 0 0 0-1.665 1.665l-.77 3.08a.512.512 0 0 1-.994 0l-.77-3.08a2.289 2.289 0 0 0-1.665-1.665Z'%3e%3c/path%3e%3c/svg%3e)](https://zhida.zhihu.com/search?content_id=222693861&content_type=Article&match_order=1&q=RPG%E5%85%83%E7%B4%A0&zhida_source=entity)和开放世界要素，同时也拥有庞大世界地图和故事剧情。而它又因为其开放级程度极高而备受好评，成为一款不可多得大作。另外在该作中还有一个叫做“TheLastofEternity”DLC——它是由巫师3里一些有趣角色组成——其中包括多位全新人物和支线任务。其游戏中还拥有很多高质量武器模型——其中一些武器甚至会出现在电影当中!同时还能让玩家体验到很多不同种类的物品和任务奖励以及支线任务——比如在剧情中我们就会发现一位新人物或者是一件新装备。

### 六、使命召唤系列

　　《使命召唤》的画质在3A大作里是比较顶尖的。这是一款高质量的3D游戏，的玩法很简单，就是玩家控制一名特种兵在战场上收集资源和弹药，并用各种不同类型武器来击败敌人。我们可以在多个地图上收集资源，然后去各种不同场景来让玩家在不同位置完成任务。游戏画面也非常出色——不仅有精美的世界风景和华丽的光影效果，还有各种逼真的战斗场景和武器效果，这些都将会让玩家们有一种身临其境感觉，而且作为一款3A大作而言，《使命召唤》游戏内的每一个小细节都将会非常精美、细致。此外它所包含的丰富内容不仅有单人战役，还有合作战役、多人合作战役以及其他类型战役玩法。《使命召唤》中还拥有非常丰富的地图场景、多种地形、武器弹药和其他资源等众多要素为玩家们提供了更丰富内容，更多游戏玩法等!

### 九、地铁：离去

　　《地铁：离去》是一款由索尼游戏发行的第一人称射击类游戏，于2017年10月22日在PS4平台发售。《地铁：离去》是一款太空题材RPG类游戏。本作的背景设定在俄克拉荷马州的一个地下城市，在游戏中，玩家将可以自由选择武器和人物、建筑与环境，可以选择单人游玩也可以选择与朋友联机游玩。除了武器装备外，还有各种道具和技能，还能通过支线任务获取资源。游戏中拥有超过100个地点供玩家探索，地图大小超过200公里。最重要的是这款游戏是索尼在3A大作中首次使用全3D技术进行打造的作品!它将会让你体会到一个完全不同的世界!我相信你一定会喜欢这款佳作。
