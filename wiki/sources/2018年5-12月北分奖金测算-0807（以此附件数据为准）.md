潘总：

结合测算数据（先拋除包机部分），分析如下：

一、毛利-奖金同比

| 18年5-12月预估奖金 | 2299448 |
| 17年5-12月实发奖金 | 3630273 |
| 增幅 | -37% |

|  | 18年5-12月（预估） | 17年5-12月（实际） | 增幅（18比17） |
| 欧洲发团人数 | 19835 | 16836 | 18% |
| 欧洲毛利含50SH | 15346710 | 20991320 | -27% |
| 欧洲单客毛利 | 774 | 1247 | -38% |

1）18年预估奖金中，未含管理人员（徐妍、朱莹、金春、张博媛、范鑫、冯鑫等人）、财务口径中要计入奖金池的福利费、以及非体系内的特殊奖金等情况。

2）当前看，发团人数与毛利呈反向关系，可能处于完全竞争市场，故提高发团人数必会降低毛利，即市占率与毛利率成反比。

3）当前看，奖金与单客毛利同向，且降幅一致。

二、员工收入

|  | 17年 | 占比 |
| 固定人均月收入 | 3861 | 60% |
| 奖金人均月收入 | 2611 | 40% |
| 人均月收入 | 6472 |  |

北京公司评级D，原则上无上调固定收入资格；奖金增幅按-37%同比例预估，则18年收入预估：

|  | 18年 | 占比 |
| 固定人均收入 | 3861 | 70% |
| 奖金人均收入 | 1654 | 30% |
| 人均月收入 | 5515 |  |

|  | 人均月收入 | 相较当地劳动力市场 | 备注 |
| 17年北京市 | 8467 | - | 以17年北京市对标，18年次年出，一般逐年递增，幅度不一 |
| 17年北京公司 | 6472 | -24% |
| 18年北京公司 | 5515 | -35% |

1）人均收入逐年降低， 且低于当地劳动力市场幅度较大，故在人才的招、留处于劣势；

2）人均收入的降低主要体现在奖金收入的降低，从员工角度来讲，员工（尤其是销售）只会看发团人数增长了（18%），但奖金却在大幅萎缩（-37%），激励性减弱；

3）建议对策：若仍在现有体系规则下，则
1、抓大放小；2、在固定工资、或补贴、或长期激励上，或几项的组合作文章。3、一段时期内，舍弃经营利润的追求。
