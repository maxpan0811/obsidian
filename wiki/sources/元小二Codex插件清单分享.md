# 元小二：Codex插件清单分享

用户"元小二"分享的 Codex 插件/工具清单及使用心得。

## 推荐清单

### 神级功能
- **Computer Use** — Codex 独有，直接操控电脑，口头描述即可自动化流程。适合 RPA 场景（元素变化时优于传统 RPA）
- **Playwright** — Computer Use 搞不定的网页（电商后台、公众号后台等），开新浏览器模拟真人操作。建议 Computer Use 主力 + Playwright 备胎

### 工作流整合
- **Gmail** — 自动邮箱秘书，定时发日报、查开源仓库修改意见
- **GitHub** — AI 时代标配，连代码仓库
- **Google Drive** — 内容创作和资料归档，类似飞书 CLI

### 进阶产出
- **Remotion / Hyperframes** — 视频特效，适合口播视频加动感
- **Vercel** — 一键部署，比 Cloudflare 方便

### 踩坑提醒
- Codex 内置四件套插件不太好用，建议用社区 Skills 替代（如 PPT 用藏师傅/zara老师的 PPT Skills）
- Chrome 插件几乎没用，优先 Computer Use，再 Playwright
- Build iOS app / Figma / Notion / product design / Supabase 在各自垂直领域能打，按需安装
