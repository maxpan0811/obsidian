在使用 Mac 时，查词典可能是最基本的功能之一了。在 macOS 上，我们也有很多种查单词的姿势：自带的词典应用可以满足基本的需求，第三方应用则提供了专业又丰富的功能，各类快捷启动器还可以实现一步查词，可谓选择多多。那么在 Mac 上查单词都有哪些选择？我们一起来看看。

本地查词应用
------

### 原生词典

macOS 的原生词典凭借「亲儿子」的地位，在 macOS 的多个入口都占据有利地位。同时它还内置了维基百科、美式英文、英式英文、德语、法语等多种语言词典，可以满足简单的查词需求。

![](.evernote-content/D38245D2-7BA6-460E-84B5-4257B29CE9BC/7D1EBF66-5FB7-411B-901D-E8B56A276D92.png)

在查词方式的多样性上，原生词典是第三方应用无法超越的。目前在 macOS 上，我们可以利用以下途径调用原生词典查单词。

1. 触控板：选中单词后，用力点按（可在 系统偏好设置 - 触控板 中修改）；
2. 右键：选中单词后右键，选择「查询……」；
3. 聚焦搜索：唤醒聚焦搜索后，输入或粘贴单词即可查询；

当然作为原生应用，词典还有很多方面无法满足用户的需求。比如缺少典型例句、没有单词本收藏夹等功能。

关联阅读：[把查词做到极致的 macOS 原生词典，其实很好用](https://beta.sspai.com/post/43155)

### 第三方词典

为了弥补原生词典功能上的不足，我相信很多 Mac 用户都会安装一个第三方词典，并且在纠结安装有道词典还是欧路词典之后，最后不如索性两个都装上。

有道词典和欧路词典都可以看例句、听发音、查翻译。对于外语语言学习者来说，单词本、生词标记、多平台同步等功能都非常实用。

当然，它们还有超越原生词典的鼠标取词功能：把鼠标光标移动到屏幕显示的单词上，一个悬浮小窗马上显示单词的意思。

![](.evernote-content/D38245D2-7BA6-460E-84B5-4257B29CE9BC/DC204A1C-7DF3-425B-94C5-E999C071FB28.png)

如果一定要在有道词典和欧路词典中选一个，我更推荐欧路词典。它支持词库管理，可自行选择安装语言词库、发音库等；背单词功能可以给考研、考托福雅思的同学们提供背单词的新方式。

关联阅读：[欧路词典：史上最强词典 APP？](https://www.sspai.com/post/24224)

![](.evernote-content/D38245D2-7BA6-460E-84B5-4257B29CE9BC/9FD9AB80-4888-4A7A-B80C-B9D03CC0536F.png)

快速启动器
-----

除了聚焦搜索外，快速启动器也都内置了查词功能，它们有的调用 macOS 原生的词典，也有的可以自行集成第三方词典工具。

### Spotlight 聚焦搜索

聚焦搜索除了可以搜索文稿外，还可以进行例如查汇率等简单的操作，当然查单词对它来说也不在话下。唤醒聚焦搜索后，直接输入单词，搜索结果中就会出现单词的含义。聚焦搜索调用的正是 macOS 原生的词典，因此便捷迅速是它的特点，而支持语言少、释义简单等等都是它的不足之处。

![](.evernote-content/D38245D2-7BA6-460E-84B5-4257B29CE9BC/BA18B889-0773-4B29-AE79-4E457C8063BD.png)

### Alfred

Alfred 是 macOS 系统上的一款老牌效率应用，通过快速启动器方便快捷地实现复杂的功能，可以极大地提升 Mac 使用效率。

关联阅读：

* [从零开始学习 Alfred：基础功能及设置](https://www.sspai.com/post/32979)
* [高效使用 Mac，从 Alfred 开始](https://sspai.com/topic/237)

Alfred 默认可以调用 macOS 原生词典的查词功能。唤醒 Alfred 后，输入 define + 空格 + 单词，就能快速查阅单词含义。当然，Alfred 的修饰词可以随意更改，比如改为更加简短的字母来实现快速查词。

![](.evernote-content/D38245D2-7BA6-460E-84B5-4257B29CE9BC/6746FDD7-965E-4331-8A3E-B2A5E202B80E.png)

饭制的 [Alfred workflow](https://github.com/samqiu/Youdao.alfredworkflow) 还可以实现在 Alfred 搜索框中调用有道词典网页版接口查询，来实现查询单词含义。

![](.evernote-content/D38245D2-7BA6-460E-84B5-4257B29CE9BC/8F8A42C0-6FF0-4C2F-9232-6036DD368282.png)

Chrome 插件
---------

如果你在用 Chrome 浏览器，那么丰富的插件还能让你多一种查词姿势。

### Saladict 沙拉查词插件

Saladict 的特色是内置了许多个查词引擎。它会把你选中的词汇在必应词典、柯林斯词典、有道词典谷歌翻译、搜狗翻译等等词典中，都搜索一遍，并给出缩略的页面。

使用 Saladict 插件，没有自动取词功能，你需要把单词选中，然后点击 Saladict 的图标，在弹出的悬浮窗口中选择词典和查看释义。

![](.evernote-content/D38245D2-7BA6-460E-84B5-4257B29CE9BC/E5F09ED3-743C-484C-ACDE-E630DD51C3BC.gif)

你可以在  [Chrome Web Store](https://chrome.google.com/webstore/detail/%E6%B2%99%E6%8B%89%E6%9F%A5%E8%AF%8D-%E7%BD%91%E9%A1%B5%E5%88%92%E8%AF%8D%E7%BF%BB%E8%AF%91/cdonnmffkdaoajfknoeeecmchibpmkmg) 免费获取沙拉查词。如果你的上网方式不怎么科学，可以访问  [Saladict 的 GitHub 主页](https://github.com/crimx/ext-saladict) 下载 .crx 文件安装。

关联阅读：[如何在浏览英文网页时获得更好的阅读体验？](https://www.sspai.com/post/54697)

### 有道划词插件

有道词典也为 Chrome 开发了查词插件，它的特点是我们滑动鼠标选中单词后，有道词典的小窗自动弹出，显示单词释义，不需要我们再点击操作，非常方便。

![](.evernote-content/D38245D2-7BA6-460E-84B5-4257B29CE9BC/5F6616A0-8B6F-4367-838A-01E194E90AA0.png)

网页版词典
-----

最后的最后，在 Mac 上查单词，当然可以使用网页版词典。你可以直接使用 [柯林斯词典网页版](https://www.collinsdictionary.com/zh/)、[剑桥词典网页版](https://dictionary.cambridge.org/zhs/)、[牛津词典网页版](https://languages.oup.com/) 等等。当然还可以选择 Wikipedia 等百科查询单词释义。

[WordReference](https://www.wordreference.com/) 是外语学习者的利器。它提供了超过十多种语言的相互两两释义，并且还在不断增长中。WordReference 论坛也是宝藏，用户可以在论坛自由地发帖，请教词汇和语法方面的问题。

![](.evernote-content/D38245D2-7BA6-460E-84B5-4257B29CE9BC/AADD41B2-23DD-4FDC-B2E5-B269DD5A6C2D.png)

既然用了网页版词典，那么除了大众词典之外，还有很多「术业有专攻」的词典可以选择。在医学领域，[Dorlands.com 道兰医学词典](https://www.dorlands.com/dorlands/wsearch.jsp) 、[Taber’s Medical Dictionary Online 泰伯医学辞典](https://www.tabers.com/tabersonline/)、和 [Merriam-Webster Medical Dictionary 韦氏医学词典](https://www.merriam-webster.com/medical) 算是三个殿堂级别的英文版医学词典。当然他们都只有英英互译。

[OneDict 一本词典](http://onedict.com/) 很有意思，它专门收集了学术论文、专利、学术期刊里的翻译，结合语境来为单词提供高质量又权威的释义。[Tmxmall](https://www.tmxmall.com/qsearch) 也做了类似的工作，但它的翻译数据来源没有明确的局限。

![](.evernote-content/D38245D2-7BA6-460E-84B5-4257B29CE9BC/D2660941-3637-4B81-BB7E-0F9823DA5BCF.png)
