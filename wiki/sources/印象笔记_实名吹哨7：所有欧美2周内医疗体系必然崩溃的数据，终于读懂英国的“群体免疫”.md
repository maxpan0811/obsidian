**光看R0是不够的！日复增长率（每天复利增长倍数）在爆发期要重要得多！重要得多！重要得多！**

* 大家常常只关注"R0"，就是每一个传染者会感染几个人。我copy 百科上一段解释：

**基本传染数**（**Basic reproduction number**）是在流行病学上，指在没有外力介入，同时所有人都没有免疫力的情况下，一个感染到某种传染病的人，会把疾病传染给其他多少个人的平均数。基本传染数通常被写成为*R*0。 *R*0的数字愈大，代表流行病的控制愈难。在没有防疫的情况下，

* 若*R*0 < 1，传染病将会逐渐消失。
* 若*R*0 > 1，传染病会以指数方式散布，成为流行病（epidemic）。但是一般不会永远持续，因为可能被感染的人口会慢慢减少。部分人口可能死于该传染病，部分则可能病愈后产生免疫力。
* 若*R*0 = 1，传染病会变成人口中的地方性流行病。

* 关注R0，核心意义是关注他是比1大还是比1小！

+ 为什么呢？病毒传播的累计总人数就是一个等比数列。R0就是那个比。举例子：一开始有1个病人，然后R0是2，那么累计病人的总数，就是这个等比数列的求和：1，1\*2^1,  1\*2^2, 1\*2^3, 1\*2^4...... ，即 1，2，4，8，16.......这个大家应该初中还是高中就学过。
+ 那么求和的公式是什么呢？总和= [a1(1-q^n)] / (1-q)。在没有疫苗、没有其他因素会阻断传播时，会一直传播下去，即n无穷大。求极限时，总和=a1/(1-q)。当q比1小，但无限接近于1时，就会发现这个总和是无穷大，意思如果人口无穷大，他就会永远感染下去。如果人口有限，他会不断地传播，直到总人口每个人都被感染。当然>1时也是同样结果，数字越大，传染的速度越快。只是终局仍然是所有人感染
+ 但如果<1，比如0.5。那么我们举个例子，有100人感染了某种病毒，那么求极限总和=a1/(1-q)=100/（1-0.5）=200人。200人而已。不是无穷大。
+ 所以传染病里其实更关心是R0比1大还是比1小。当然R0越大，传染性显然是更强的。

* 互联网产品，也常讲社交传播，逻辑一摸一样。

+ 比如每个用的这个产品的用户，其中有20%会传播给别人，总共传给4个人。那4个人收到推荐后，50%会下载下来开始使用，那么这个对应的R0=20% X 4 X 50% =0.4。R0等于0.4会有什么结果呢？会更好地获得用户。
+ 比如你花钱买广告获得了1个下载用户，平均成本是10元。那么你想获得100万用户，就得花1000万的广告成本。
+ 但如果有传播，且R0=0.4时，那么初期再没有重复刷用户的情况下，你第一个用户会带来额外0.4个第一层传播用户。而这0.4个用户会再传播，获得0.4\*0.4=0.16个第二层用户，然后持续传播。那么这1个起始用户，将获得总共=a1/(1-q)=1/（1-0.4）=1.67个终局用户量。那么你初期花的10元，平摊到1.67个用户上，就是5.99元/用户。你的获客成本更低了。
+ 但如果你更厉害，R0能做到>1，比如初期的Facebook，他们是没钱打广告获客的。他们的新用户增长来自于老用户推荐。比如每个活跃用户，有50%不只是自己偷偷用，而是他会推荐他10个好友/同学去用。这10个人6个被说动了，用起来后有2个觉得没意思，不用了，4个继续用下去，成为新的活跃用户。则R0=50%\*4=2。但R0一旦大于0，就会一直增长一直增长，直到目标用户人群都被覆盖，获得过10亿用户，成为世界互联网公司巨头，现在市值5000亿美金。腾讯同理、微信同理、病毒传播也同理。

* 但是，在当R0比1大很多时，R0具体是几就已经不是那么重要了！在病毒快速爆发期，日复增长率（每天复利增长倍数）要重要得多！重要得多！重要得多！

+ 因为我们并不在意到底1个人感染的是3个还是4个，反正是远大于1的，就是病毒会持续传染很多人。我们更在意的是，他到底是1天就传染3个人，还是1周传染3个人，还是1个月传染3个人！这结果天差地别！！！我们称呼这个天数叫DAY吧
+ 日复增长率 = （R0+1） ^ (1/DAY）-1

* 比如R0=1，DAY=2，则日复增长率 = (1+1) ^ (1/2) -1 = 0.414 = 41.4%。即每天累计病历增加41.4%。
* 所以R0高也可以增长很快，传播日期短也可以增长很快。但到底是哪个导致的快，我们暂时不关系。我们只关心到底日复增长率等于几。因为：

+ 10%的日复增长率，传播一个月，结果是17倍（100个病人变1700个病人）
+ 20%的日复增长率，传播一个月，结果是237倍（100个病人变2.3万个病人）
+ 30%的日复增长率，传播一个月，结果是2620倍（100个病人变26万个病人）
+ 40%的日复增长率，传播一个月，结果是24201倍（100个病人变242万个病人）

* 而过去10多天到现在，欧洲和美国的日复增长率，大概在30%~40%间！

* 大家可能会问，那现在欧美大家都恐慌了，躲回家不上班了，停学了，戴口罩了。日复增长率肯定就下来了吧？是的，的确是会下降，但是可能是从40%，降到10%，已经很显著很显著了！为什么不是降到0%，因为你还是要和家里人接触，如果你要上班、买东西还是得聚集。即使戴口罩，并不代表就不会传染。要不中国早就复工了，全员戴口罩就行，反正不传染呗。
