在编写理工科文本的时候，有一大痛点是如何处理各种公式，虽然大部分 Markdown 编辑器支持通过 `$ $` 行内或 `$$ $$` 独自一行插入 LaTeX 公式，但是编写公式本身依旧是一个枯燥的工作。

后来我发现了一个服务 / 软件叫 [Mathpix OCR](https://mathpix.com/ocr)，其首先是一个 OCR 服务，不同之处是它可以将打印和手写的公式转换成 LaTeX：
