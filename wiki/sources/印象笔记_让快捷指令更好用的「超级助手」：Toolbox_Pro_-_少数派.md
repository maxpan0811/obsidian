让快捷指令更好用的「超级助手」：Toolbox Pro
===========================

[快捷指令](https://sspai.com/app/%E6%8D%B7%E5%BE%84%EF%BC%88Workflow%EF%BC%89) 是提升 iOS 使用效率的一大利器：简洁易懂的操作界面和大量第三方 App 的支持让它为用户提供了高效的自动化服务。最近新推出的 Toolbox Pro（下文简称 Toolbox，涉及专业版功能时则用原名）能够为快捷指令带来诸多实用功能，堪称快捷指令的「超级助手」，外媒 MacStories 更是将其称为「Power User 的必备工具」。

本文将通过 7 个实例向你介绍 Toolbox 究竟为什么能够在面世之初就得到这样的褒奖，如无特殊说明，本文提到的实例均可在 Toolbox 免费版下使用。

简单了解 Toolbox Pro
----------------

我之所以称 Toolbox 为快捷指令的「超级助手」，是因为它与快捷指令是一种相互依存的关系。Toolbox 需要依靠快捷指令才能调用其中的功能，而快捷指令则需要第三方工具来实现更多功能。最令我感到意外的是，Toolbox 中的绝大部分功能都可以直接在快截指令中运行，不需要跳转。

Toolbox 目前有 58 种功能，其中包括了全局变量、图像识别等快捷指令尚未内置的功能。利用这些功能，你可以为快捷指令注入更强大的力量，实现像应用锁等快捷指令原来不能实现的功能。

Toolbox 值得夸赞的地方不止于此。它与快捷指令一样兼顾了易用与功能，如果你想调用 Toolbox 的功能，就和在快捷指令中调用原生功能一样简单。对于一些复杂的功能，Toolbox 应用本身就是一个详细的使用手册，你可以查看所有功能的详细介绍、下载范例。

![](.evernote-content/729CB8F1-01BD-4C63-A3DD-6CC66D781D2C/0593C549-0F8A-42B9-94C5-6F0B37FC8F3C.png)

值得一提的是，Toolbox 针对一些需要进行特殊设置的功能，不仅在应用中给出了使用介绍，也将其放在了快捷指令动作的简介中。比如生物认证功能，就需要通过剪贴板进行内容传递，因此你需要在这个动作后使用「获取剪贴板」来得到认证结果。

更丰富的文本编辑功能
----------

文字编辑是我日常生活中不可或缺的一部分。快捷指令本身的文字编辑功能就已经能够满足我大部分的需求，有了 Toolbox 的加持，在文字编辑上，快捷指令甚至要比一些专门的文本编辑器还方便。

### 文本编辑增强器

[点击下载](https://www.icloud.com/shortcuts/e66ce5c0dbf84547a8ba27c84c3e702c)

Toolbox 仅免费提供了两种文本编辑工具：**移除 Emoji** 和**裁剪文本**。虽然相比于 Pro 版功能来说稍显鸡肋，但对于快捷指令本身的文本编辑动作来说，也算是填补了部分空白。

「移除 Emoji」可以用来应对那些特别爱在文字中添加 Emoji 的人，帮助你提取出有用的文本信息。「裁剪文本」则可以帮你去除文字的某一部分。比如有些人习惯在段落开头输入两个空格，你可以用它来将每段文字的首行两个字符（即两个空格）给删除掉。

![](.evernote-content/729CB8F1-01BD-4C63-A3DD-6CC66D781D2C/9DAE9DDA-5196-4CE8-9A47-EDCBAD5CC344.gif)

裁剪文字

购买 Pro 版以后，Toolbox 的文本编辑就上了一个台阶。不但有「快速匹配」「移除重复内容」等便利的功能，更有「获取文本详情」这样的语义识别工具。

### 模型价格管理（仅 Pro 版）

[点击下载](https://www.icloud.com/shortcuts/d17f0a651e36413fb4d55a23dc353524)

我最近迷上了制作高达模型，随着购入的模型不断增多，记录、统计它们的价格成了一件麻烦事。于是我用到了 Toolbox Pro 的「全局变量」和「快速匹配」两个功能，制作了一个用于记录、查询模型价格信息的快捷指令。

通过简单的选择和输入，我就能在全局变量中添加一个固定格式的模型价格信息。查询时，我可以根据机体型号或尺寸来进行选择，快捷指令会自动替我计算总价。

其中的价格提取就用到了「快速匹配」，这个动作可以提取**关键词之前**、**之后**或是**之间**的文本信息。在这个快捷指令中，它被设置为提取「售价：」与「元」之间的信息，也就是价格数字。这样一来，他就可以直接将数字进行叠加运算。

因此，这个快捷指令也可以用于管理产品价格。当有顾客向你咨询下单时，你可以用它来生成带有总价的价目列表。

![](.evernote-content/729CB8F1-01BD-4C63-A3DD-6CC66D781D2C/08F00D84-5155-449E-A117-92E6190AE55B.gif)

### 筛选姓名

[点击下载](https://www.icloud.com/shortcuts/38b973e1d1164d03a3f75139d2fa6d07)（仅 Pro 版）

Toolbox Pro 的「获取文本详情」是许多专业级文本编辑工具都不具备的功能。它可以替你分析当前文本，并输出文本中包含的人名、地名、组织名，甚至是文本情感和所用的语言。这个动作在设定上非常简单，你只需要设定获取的文本和要输出的信息即可。

我根据自己的需求做出了「筛选姓名」的快捷指令，它支持读取分享菜单或剪贴板的内容，并将里面的人名提取出来供我选择。在我选择好需要的人名后，它就会自动粘贴至我的剪贴板。如果你想用它来整理名单，还可以加上「去除重复项目」来避免内容赘余。

![](.evernote-content/729CB8F1-01BD-4C63-A3DD-6CC66D781D2C/93B0DC23-5825-4622-ACC6-F327B1418486.gif)

检测姓名

除此之外，你还可以使用「获取文本详情」来打造一个简单的自然语义识别工具。在你与他人沟通工作时所提到的人名、地点等信息都可以借助这一功能进行筛选，甚至直接添加到你的任务管理工具中去。

为快捷指令添加生物认证
-----------

[范例下载](https://www.icloud.com/shortcuts/ec06045b799e4cc7b8ceaa907ef44ee0)（仅 Pro 版）

Toolbox Pro 还为快捷指令带来了生物验证的功能，这也算是为快捷指令「加了一把锁」。生物验证的应用范围极广，特别是在你设置了一些自动化快捷指令的情况下，你肯定不希望别人拿你的手机碰一下 NFC 标签就自动触发快捷指令吧。

由于快捷指令的限制，生物认证功能需要跳转至 Toolbox Pro 进行验证，并将是否通过的信息复制到剪贴板。此外，在首次使用此功能时，需要启用生物认证的权限，或许会显示认证失败，这是正常现象，不影响后续的正常使用。

![](.evernote-content/729CB8F1-01BD-4C63-A3DD-6CC66D781D2C/B5066C62-816A-4228-AF21-C3C807383BF0.jpg)

首次运行报错是正常现象

要为快捷指令加上认证的方法非常简单，只需要在快捷指令的开头添加「Authenticate」动作并加上对剪贴板内容的识别即可。为快捷指令添加验证不仅可以保护快捷指令的隐私，更可以在自动化指令中实现「二次验证」。

举例来说，我在床头柜贴了一张 NFC 标签用于触发「为我推送天气信息」的自动化指令。一般来说我只会在起床后使用它，但睡前将手机随手放在床头柜上充电时，难免会不小心触发 NFC 标签。这时候，生物认证的「二次验证」功能就显得非常有用了。

强大的图像识别
-------

[点击下载](https://www.icloud.com/shortcuts/e6c6570c7e2542619eb756c3444457e9)（仅 Pro 版）

Toolbox Pro 的图像识别功能或许会是快捷指令往专业级迈进的一大步。快捷指令本身的图像处理已经足够满足大部分需求，配合上 Toolbox Pro 强大的图像识别功能更是锦上添花。

我将 Toolbox Pro 中的图像识别功能做了一个合集，方便各位了解 Toolbox 的功能。其中包括了**热点图生成**、**物体识别**和**面部检测**。由于 Toolbox 的 OCR 识别目前仅支持英文，本文不进行赘述。

这三个功能的本质都是对照片内容进行识别，并进行不同的操作。在我体验来看，Toolbox 在热点图的识别准确率还算不错，但在物体识别方面稍有不足。

从热点图中可以看出它能够识别出图像主体，并计算出观众的注意力会集中在哪，不过它对于一些复杂场景下的物体识别只能说是差强人意，不时会出现一些令人啼笑皆非的错误。另外，我在两台设备上分别尝试使用面部识别，都会显示「与 App 通信出现问题」，希望应用能在后续更新中解决这一问题。

![](.evernote-content/729CB8F1-01BD-4C63-A3DD-6CC66D781D2C/7582F0BC-392B-486D-AFA2-8A3FCF6B6247.png)

这一功能需要用到「获取剪贴板」来将导出生成的热点图。对于影像工作者来说，你可以依靠这一功能来查看自己的图片中，观众的注意力会聚焦在哪个位置，从而进一步优化自己的作品。

像保存文件一样使用变量
-----------

快捷指令仅支持局部变量（只能在特定快捷指令的某一次运行使用），如果你要想生成一个全局变量（可以被多个快捷指令，在多次运行中调用），只能依靠在 iCloud Drive 写入文本等方式来实现，非常麻烦。Toolbox 正好弥补了快捷指令在这方面的不足。

### 运动计时

[点击下载](https://www.icloud.com/shortcuts/d7e8248d837c424490e78d6f1f5eb13c)

如果你有运动的习惯，并且想记录自己的运动用时，可以试试我写的这个「运动计时」。运动前点击「开始计时」，它会向全局变量中写入当前的时间，结束后点击「结束计时」，它就会向你推送这次的运动时间。不仅如此，你还可以根据自己的实际情况对其进行优化，比如在每次开始运动前向你推送上一次运动的记录等等。

![](.evernote-content/729CB8F1-01BD-4C63-A3DD-6CC66D781D2C/6E982868-1F22-4EF9-86FA-BAA2B8023771.gif)

在过去，这样的操作几乎只能把数据存储在 iCloud Drive 或者备忘录应用来「曲线救国」。但有了全局变量后，你甚至可以为其添砖加瓦，在运动完成后给你推送运动时的心率等信息，非常方便。

全局变量能做到的不止于此，它甚至还能为你打通快捷指令之间的屏障，让一个变量可以在多个快捷指令中使用。

### 聚合搜索

[点击下载](https://www.icloud.com/shortcuts/724b0691c92848a4819d45030f320adc)

在之前，我为了提高搜索效率，做了一个「聚合搜索」的快捷指令。它的原理是通过「词典」来筛选出不同的搜索引擎，随之而来的问题就是**如何管理词典**。由于快捷指令本身不支持全局变量，所以我只能用「文本」或「词典」动作来存储搜索引擎。但这样的结果就是这些词典能且仅能在当前快捷指令中使用。

有了全局变量，我就可以将词典信息保存在 Toolbox 中实现「一劳永逸」。把搜索引擎词典存入 Toolbox 还有一个好处就是可以在多个快捷指令中调用它，比如你可以通过调用词典数据在现有的快捷指令中加入聚合搜索的功能，而不用进行大面积的重写。

![](.evernote-content/729CB8F1-01BD-4C63-A3DD-6CC66D781D2C/587794C2-CA83-422A-98E4-72DB50A3FD31.jpg)

使用全局变量存储词典

总结
--

除了上面提到的功能外，Toolbox 还支持检测设备状态（比如是否在运动、系统语言等信息）、扫描文档，甚至是自定义菜单、计算数学公式（可以配合 OCR 使用）、整理 JSON 词典等功能。对于普通用户来说，Toolbox 是一个扩展式的工具，可以弥补快捷指令的诸多不足；到了 Power User 的手里，Toolbox 就更像是一个期待已久的百宝箱，说它是快捷指令的「超级助手」毫不为过。

快捷指令更新缓慢的情况下，Toolbox 的出现无疑是让 iOS 自动化更上一个台阶的工具。它不仅对快捷指令「查漏补缺」，更是带来了基于机器学习的图像识别等功能。在使用上，Toolbox 和快捷指令本身的操作逻辑相同，用户在快捷指令中可以「无缝」调用，并且绝大多数的动作也不需要进行二次跳转即可实现。

如果目前的快捷指令无法满足你天马行空的想象和对自动化的高追求，那么 Toolbox 绝对是你不容错过的工具。你可以在 [App Store](https://apps.apple.com/cn/app/toolbox-pro-for-shortcuts/id1476205977) 免费下载 Toolbox Pro，应用含高级版内购，售价 40 元。
