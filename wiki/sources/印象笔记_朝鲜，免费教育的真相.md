# 朝鲜，免费教育的真相

---

速读摘要

朝鲜自1975年以来实行11年制强制义务教育，包括学前1年、小学4年、中学(初中+高中)6年。从幼儿园高年级开始，7岁的学龄儿童将正式接受常规课程，12年义务教育从这时开始。能不能获取申请资格，除了学生自身的努力之外还要看学校和出身，不同地域不同学校给的资格名额也不同。学生没有选择大学的权利，由学校单方面通知学生到被推荐的大学去参加校招考试。这所学校的学生可以直接进入大学，不需要与全国考生竞争。

NO.1758-朝鲜免费教育

作者：那日苏

校稿：猫斯图 / 编辑：养乐多

对于现代职场人来说，努力赚钱储蓄的目的大多是为了应对医疗、子女教育和住房三项生活上的大事件。

而在与我们一水之隔的朝鲜，这三项基础民生工程都是免费的。在很多被三大件压得喘不过气的人看来，这似乎是主体思想优越性的绝佳展示，甚至会吸引一些东亚年轻人，比如韩国人“脱南”去做朝鲜人。

五彩缤纷的，都是国家分的房子

（图片：Andreas.Mattsson  / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/AC15ABE9-62F6-40C4-A1DA-50F55BD9DCF5.jpg)

然而，这些福利政策的真相究竟如何？今天我们就先来看看朝鲜的免费教育真相，看完了再决定到底要不要去羡慕他们。

十二年义务教育

朝鲜自1975年以来实行11年制强制义务教育，包括学前1年、小学4年、中学（初中+高中）6年。现今绝大多数朝鲜中青年都是在这种义务教育制度下成长起来的。

朝鲜的青少年文化普及率是很高的

在国家的强制要求下，都得接受义务教育

（图片：PetraKub / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/93F3A80F-4397-43B4-81AD-056B48C66F34.jpg)

2014年，朝鲜内阁教育省提出新的改革方案，将小学延长至5年，这样义务教育的年限就增加到了12年。不过大部分朝鲜学生的校园生活都要超过12年，这与严格的模块化管理体系是密切相关的。

上了学才能参加文艺汇演呐

（图片：junrong / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/966A1108-6B76-4990-AF42-9E48EEF96052.jpg)

一个朝鲜孩子成长到3~4岁的时候，就会被送到父母工作单位或者所居住社区下辖的托儿所托管。在朝鲜几乎每一个“洞”一级的行政单位（相当于我国的街道办事处）都有至少一个托儿所。

好好学习，天天向上▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/FD902060-2FC6-4218-9946-C67192F04351.jpg)

小孩子将在托儿所成长到6岁，然后被送去相对应的幼儿园。和中国分小班大班一样，朝鲜幼儿园也根据入园年份分为高年级和低年级。从幼儿园高年级开始，7岁的学龄儿童将正式接受常规课程，12年义务教育从这时开始。

模范幼儿园

（图片：Oleg Znamenskiy / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/6F18BBC6-B89F-4E49-BF70-AEACB0CEE2B5.jpg)

不管是托儿所还是前期幼儿园，都被视为是孩子父母单位的员工福利，只象征性收一点管理费用，据说还可以报销。相比于韩国小朋友上私立幼儿园的年均费用540多万韩元（约合人民币3万元），至少在价格方面是很有优势的。

都算作社会主义基础公共服务的一种

家长只要搞好社会建设就好了

（图片：Truba7113 / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/6C59AE38-3AD7-406B-BF97-5C36048C5CEA.jpg)

朝鲜的幼升小也是比较轻松的。

首先是在学校的选择方面不需要家长过多费心，朝鲜曾短暂设立了部分“重点小学”，但随后又因违背了公民师资均等原则而被取消。因此在小学择校阶段，父母只需要选择离家最近的即可。朝鲜的小学资源也比较丰富，为了照顾边远地区的孩子，全国还有数百所10到20人的“微型”学校。

在一个民主的社会主义国家，怎么能搞阶级差异呢？

（图片：christiantimeless / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/2E388C97-0754-48E7-B36F-5C4F3C6B8241.jpg)

这一时期的孩子在学校学习的内容除了数学、朝鲜语和艺术之外，还有专门的课程学习金日成和金正日两代领导人的童年故事，相当于思想品德教育，为培养优秀的组织性人才打好基础。由于升学的轨迹是被安排好的，所以一直到初中都没有什么太大的变化。

朝鲜的艺术氛围还是很浓的

朝鲜族本就是能歌善舞的民族

（图片：Torsten Pursche / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/8332A383-54BD-4A5A-80A8-1D2BB9BBAF0B.jpg)

初中毕业的时候，有些孩子会进入到重点高中或者特殊学院。

不过这个选择与大多数人无关，基本上朝鲜教育系统中每个初中都有一所相对应可以直接升学的高中，学生们大都继续在高中学习文化知识。朝鲜高中的课程有常规的朝鲜语、数学、英语（没看错就是英语）、物理、化学，也有一些在我们看来十分具有特色的课程。

例如射箭课，传统艺能不能丢

（图片：Oleg Znamenskiy / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/5CC421CE-F75D-4C12-936F-421B2CAE1707.jpg)

比如在历史教育方面，朝鲜孩子一直到高中都不学习古代史，取而代之的是金日成革命史、金正日革命史、金正淑（金日成的妻子）革命史。另外和韩国一样，朝鲜高中也有汉字课程，只是朝鲜已在日常生活和媒体出版中全面废除汉字，所以学生平均掌握汉字水平不高。

去革命老区学习伟大领导人的精神也是学生的必修课

（图片：Oleg Znamenskiy / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/2BED73C4-6097-49C2-9C68-19214861ED0F.jpg)

最受男孩子们欢迎的课程名为“军事政治课”，这是朝鲜先军政治的体现。在这节课上学生们能学习到军事地图、射击等技能，还会实地模拟山地作战，打真人CS，军队未来的优秀接班人可是要多多培养。

人人争当小战士

（图片：Oleg Znamenskiy / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/E07A6177-BDF8-4ED2-8583-018B563D2BAE.jpg)

免费的才是最贵的

升学途径顺畅、学费全免，从这个角度上来看，朝鲜的教育制度几乎是自普鲁士发明义务教育制度以来，人类官办教育的最巅峰。

每一种制度都褒贬不一的评价

即使是朝鲜的义务教育制度，亦有可学习之处

（图片：terimma / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/EB6A1344-8636-4A9B-9C87-5681681DD267.jpg)

朝鲜方面也是如此宣传的，朝鲜和平统一祖国委员会管理的宣传网站最近发表了一篇名为《通过下一代教育研究两种系统》的文章，文章中称：“我们共和国是一个‘学习之国’，一个‘教育之国’，人民从免费的教育当中收益。与朝鲜不同，南朝鲜则是一个实行反人民教育政策的地方，学校已成为庞大的赚钱机器。”

文章结尾强调“免费教育是通过减轻教育费用负担来保障受教育权的基本前提。”

初衷是好的，但是在实践的过程中难免会有差错

以朝鲜的经济状况，很难做到高等教育资源均等分配

（图片：Oleg Znamenskiy / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/B839321F-FAFF-497A-A570-7C75BC64B68F.jpg)

然而有句话叫，“免费的才是最贵的”，每一件看似无偿的馈赠其实在暗中早已标好了价格。

不收学费的朝鲜教育也并非全无代价，一对想让孩子受到更好教育的父母，总会在这里或那里付出很多。

在好学校见到伟大领袖的机会可是比较多的

（图片：Torsten Pursche / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/432207B7-05A3-420B-AC6D-E662F59AD210.jpg)

根据大量从朝鲜逃亡南韩的脱北者的描述，朝鲜所谓的“免费教育”其实十分矛盾。尽管并不收取学费，但从小学阶段学生们就需要为学校捐助柴火、帮助学校建设、维修设备，以及为人民军和其他官方组织捐款。

没有强制的收费，但有强制的捐助

可以不收钱，但是可以收物品

（图片：Mikhail Priakhin / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/526BB8F4-4F53-4869-A05F-2055A24451D7.jpg)

据描述，这些捐款捐物并非自愿，而是每一个学生家庭必须提供的。而且捐助的物品的总量之多、品类之杂令人咋舌。每人平均每年要上交学校几十公斤的铁、铜、锌等金属，除此之外常见的捐物还有兔毛、木材、节庆庆祝时要用的花卉，甚至粪便制成的堆肥等等。

出来放牛的同时，还可以捡点牛粪当学费

（图片：Oleg Znamenskiy / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/CA83ADDA-E417-4AF7-83E9-08A741FC026F.jpg)

为了能凑集这些财物，有时甚至需要一个学生家庭全家辛苦搜集。如果不能如期提供这些物资也不能算了，学生家长必须用现金补偿。根据折算，一个朝鲜学生每月上缴的物资和钱约价值人民币240元，对于朝鲜普通家庭来说这是一个不小的负担。

其实有一种攒够各种素材才能升级的感觉

（图片：Saeschie Wagner / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/67EDF690-D820-41F8-9D71-DCC7010A033A.jpg)

如果说这些明面上的花费已经让学生家庭背负了沉重的负担，那暗地里的索贿则更是给普通家庭的孩子关上了大门。在朝鲜，向老师赠送礼物是一种普遍行为，几乎被认为是官方程序的一部分。

根据一些脱北者的证词，学业成绩和入学资格也都可以通过贿赂来取得。

前朝鲜老师金哲明（化名，于2008年叛逃）透露：进入平壤的一所大学要花费1000美元，进入一所省级大学也要花费300至500美元。另一位名叫赵贤美的脱北者表示：当她想要获取高考资格时，市教育局直接告诉她需要捐献一台电脑，她的家庭为此不得不卖掉电视机买了一台笔记本电脑，这才让赵贤美有机会参加高考。

虽然21世纪都过了20年了，电脑也仍然是稀缺物资

（图片：Torsten Pursche  / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/578CE5A6-3529-4CE6-A3D5-E064AC77B289.jpg)

开始疯狂内卷

即使行贿获得高考资格，由于朝鲜严酷的高考制度，大多数学生也无法进入大学。

即使拜的都是同一家领导

这时候也要分分前后排了

（图片：LMspencer / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/0C33D880-E594-4C05-AF63-76383BE83270.jpg)

朝鲜大学考试包括预备考试和校招考试。预备考试和我们的高考非常相似，由朝鲜内阁教育省选定的教授出题，进行全国统一的科目考试，没有文理分科也没有特招、加分等等选择余地。考试所有题目都是论述题，也没有能让人浑水摸鱼的选择填空。预备考试总共历时两天，一天考3门，各45分钟。

它和高考最大的不同就在于，预备考试只是选出了那些有机会进入大学的人，他们还要根据这个成绩来申请适合自己的大学，然后再通过这所大学自己的校招考试来决定能否被录取。

能不能获取申请资格，除了学生自身的努力之外还要看学校和出身，不同地域不同学校给的资格名额也不同。首都学校就比地方学校名额多，地方重点比地方普通名额多。学生没有选择大学的权利，由学校单方面通知学生到被推荐的大学去参加校招考试。

朝鲜学生梦寐以求的大学—金日成综合大学

也是北朝第一所正规大学▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/DC8F2F75-9B83-4CAF-855A-B81EDBAD7319.jpg)

校招考试由三部分组成：专业考试、体测、面试。这一套程序下来，能进入大学的考生已经为数不多了，他们也就成为了朝鲜社会的“人上人”。

朝鲜也没有复读制度，那些没考上大学的学生们，就只能去服兵役了。

朝鲜是世界上服兵役时间最长的国家，大多数男学生们在高中毕业之后面临的是约10年的军旅生涯；而女学生就会被分配到相应的工厂或者事业单位工作，几年后和退伍回来的中年男性相亲组成家庭，生儿育女开始新一轮的循环。

这个国家虽然内部相对稳定，但外部风险太大了

为了维持庞大的常备军，不多的人口要尽量供给兵源

（图片：Maxim Tupikov / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/1AC1CC4B-B2DF-4FFE-81DC-1DC2D1D51E59.jpg)

但有些家庭也可以跳出这个系统去读一些“特殊高中”或者学院。

最出名的特殊高中是1984年在平壤设立的第一科学技术高中。该学校是金正日英才教育政策的产物，在这个政策的影响下，每个地区都有一所“第一中学”，保证教育质量远超同级中学，一中和一般中学之间的差距较大，道级一中和中央一中之间更有天壤之别。

也有专门面向军二代的革命学院

这更是精英中的精英了

学生毕业后也基本会进入朝鲜军政界

（图片：Oleg Znamenskiy / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/8E7042DD-8706-4DC3-BAEF-184CD3686783.jpg)

平壤第一科学技术高中的教师几乎都是金日成综合大学、金策工业综合大学和金亨稷师范大学三所朝鲜顶尖学校的毕业生，使用的教材都与普通高中不一样。

这里的学生分为两类，一种是从初中部直升上来的干部子女、在日同胞等权贵阶层，另一种是从全国各重点中学掐尖的最资优生。这所学校的学生可以直接进入大学，不需要与全国考生竞争。

除了重点高中和普通高中，朝鲜高中系统里还有一些专门服务于对外事务的学院。朝鲜每个市都有一所外语学院，平壤外语学院是其中的佼佼者，学院里有8种外语供学生选择，包括英语、汉语、俄语和日语等。

朝鲜封闭但不封锁

在外语学院或者少年宫的外语组也会有外教的身影▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/34D2B642-FBA9-4017-997C-0623810D7D00.jpg)

毕业后学生们可以去从事导游和对外服务的行业，也可参加高考继续深造，寻求进入平壤外国语大学的机会。外语学院在朝鲜定位相当于职业高中，但门槛却高得多。能进入外语学院的学生必须成分要好，要么是抗日战争烈士后代，要么是表现优异思想进步的少年干部。

就算是分配到接待外国游客的酒店也算一个好工作了

（图片：Chintung Lee / shutterstock）▼

![](.evernote-content/79BD8708-6114-43D2-AFD7-AB47C6FC9EE4/8C6A3A35-35E2-467B-ACA7-374CEC135EAE.jpg)

如果说普通高中和普通大学还是一般家庭勒紧裤腰带能让孩子上得起的学校，那这些按身份安排的学校就完全是普通人可望而不可及的存在了。
