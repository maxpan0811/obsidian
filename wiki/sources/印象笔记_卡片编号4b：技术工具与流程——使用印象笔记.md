### 悬浮笔记目录

对笔记内目录这个特性的支持也是呼声已久，因为在 2019 版网页编辑器和此前印象笔记客户端都没有笔记内目录这一功能（某段时间借助 Markdown 可以短暂实现）。

![](.evernote-content/AC7E3339-FE6F-4002-9393-C2E01FB3EAE9/046F1767-D80A-4F62-A7DA-5A6C9B0302C4.gif)

悬浮目录

超级笔记中新增的悬浮笔记目录平时折叠隐藏在编辑器的右上方，最大化显示笔记内容；需要时则可以通过点击右上角的目录 icon 进行呼出。

设计上，悬浮目录左侧的竖线能够帮助我们清楚地看到相关目录之间的层级关系。有了它的加持也可以快速定位到笔记内部的相应内容，写作长文时非常方便。

### 跨笔记模块拖拽

微软在 Fluid Framework 中设想过的最令人印象深刻的场景，莫过于把文档模块从一个文件中拖到另一个文件。

在超级笔记中我们也能看到这样一个有些类似的特性。我们可以把一个模块从一条笔记拖到另一条笔记之中，这样笔记与笔记之间的内容迁移就变得简单了许多。

![](.evernote-content/AC7E3339-FE6F-4002-9393-C2E01FB3EAE9/C3E4E761-2473-4F11-BE03-5C2B37582ADA.gif)

这一点，飞书也好、Notion 也罢，都是没有的。

写文章时，平常我们通过剪藏、微信公众号、微博等方式积累在印象笔记当中的案例、故事、名家语句等等素材顿时就能「活用」起来了，只需要拖拽这些素材就可以快速地合体形成一份文章框架。

因为基于客户端，只要屏幕足够大，我们甚至也可以同时打开好几条笔记的窗口进行拖拽。

最后，超级笔记也为新版客户端带来了智能表格支持。此前印象笔记曾多次对表格功能进行过升级，但整体而言印象笔记的表格功能一直停留在简单的文本内容阶段，旧版客户端也一样把表格当成文字来处理，甚至不能像 Excel 那样对表格内容进行运算，进度条什么的就更不用说了……

![](.evernote-content/AC7E3339-FE6F-4002-9393-C2E01FB3EAE9/ED37B7A1-4CE8-4BDF-8433-88ED86C2473E.gif)

智能表格还是很实用的

超级笔记的智能表格则完全支持表格内公式、进度条等功能，借助于超级笔记进行简单的项目管理自然也可以更方便一点。

![](.evernote-content/AC7E3339-FE6F-4002-9393-C2E01FB3EAE9/81E692AB-20C0-44B6-B607-341406930605.gif)

智能表格中直接插入进度条

超级笔记在功能上要比 2019 版网页编辑器更加强大，以之前文章中提到的网页版编辑器的万能「+」号为例，超级笔记在 2019 版网页编辑器的基础之上进行了升级，除了通过「+」号添加的各种文档模块之外，还新增了可以选中进行快速格式调整、复制粘贴的模块化按钮；同时我们也可以使用快捷键 / 来打开模块化扩展功能，这是网页版编辑器所不支持的。

![](.evernote-content/AC7E3339-FE6F-4002-9393-C2E01FB3EAE9/994834AD-8CFD-481E-9EF1-EFFB6806EAA6.gif)

万能的「+」

[印象笔记全新网页版编辑器公测——资深用户告诉你，怎样才能用好它？ - 少数派](evernote:///view/207087/s3/7a780653-7d35-4ea0-84b5-37a0ac54869a/7a780653-7d35-4ea0-84b5-37a0ac54869a/)

万物可「 + 」
--------

虽然新版的编辑器有着一个十分高效的工具栏，里面有着文字高亮、上下标等功能，但是我这里最想说的是新版的编辑器所提供的「 + 」的功能。

![](.evernote-content/AC7E3339-FE6F-4002-9393-C2E01FB3EAE9/E52AD59E-3B12-45A4-82A4-7C5732BAF476.png)

标题栏

（这里需要说一下的是，在 2017 版编辑器中「最大化的」按钮是在右边，而 2019 版是在左边。如果你适应了 2017 版，这一点需要注意一下。）

通过使用「 + 」能够大幅度提高信息录入和排版效率，甚至不用顶部的工具栏照样操作无虞，效率高到你会觉得工具栏是一种束缚。

![](.evernote-content/AC7E3339-FE6F-4002-9393-C2E01FB3EAE9/168FD7A3-0DC9-4D38-A80C-6E1540564042.gif)

万能的加号

「 + 」是万能，没有不能「 + 」的。标题、引用、图片、附件、表格、待办事项、无序列表、数字列表、分隔线、代码块，通通可「 + 」。这番操作是如此流畅，使用中有一种一气呵成的感觉，而不用担心会影响到自己的写作的思路。

### 更优雅的表格操作

表格，是一件非常厉害的排版利器。在新版的编辑器中再一次对表格的功能进行优化提升。

这一次，在操控性上，通过使用新版的编辑器可以获得不输于 word、excel 的使用体验的同时省却了 office 套件那样的笨重感觉。

制表的速度更快，表格与文字的混同编辑的效率更高了。

![](.evernote-content/AC7E3339-FE6F-4002-9393-C2E01FB3EAE9/C6AA774A-797C-4AB8-98D4-9D04FEA598C9.gif)

模块化表格操作随心所欲

同样的借助于模块，借助于表格，我们可以通过在表格中嵌入附件，让整体的美观度更高。

![](.evernote-content/AC7E3339-FE6F-4002-9393-C2E01FB3EAE9/E022BA4C-BE14-47BB-8AC1-F4B5E46CA4D8.png)

表格混同排版
