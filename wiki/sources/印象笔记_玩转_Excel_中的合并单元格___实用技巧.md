让我们来设想这样一个场景：假设你现在是某银行总部的业务分析员，领导发给你了一张 Excel 表格，列举了各营业网点的客户经理 2018 年的新增客户量。领导希望你在这个基础上做一些分析，得出发展新客户最多的员工、发展新客户最多的营业部，以及各营业部的人均新客户量排名等情况。

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/5B486013-9DF5-4956-84ED-4F5D1A8FFD73.png)

领导发给你的 Excel 表格

我们工作中遇到的类似情况并不少见，其实这个任务也并不难，对 Excel 稍微熟悉点的，运算排序、数据透视表等功能，分分钟就能搞定。麻烦的地方在于，由于「营业网点」这一列被合并单元格了，导致这些功能无法正常运作。麻烦的是，一共有 300 多个营业网点，要能顺利地开展数据分析，就不得不先把他们取消合并，难道要一个个操作？

这样的场景，在许多工作中都并不陌生。但凡要和 Excel 打交道的人，都为**合并**和**取消合并**单元格这件事头疼过。由于 Excel 表格往往在不同时间、不同的人手里承担着不同的角色：

* 在用于**数据分析**时，由于单元格合并后，无法进行常规的排序、筛选以及进行公式计算，我们自然希望不要合并任何单元格；
* 在用于**数据展示**时，出于美观和减少冗余信息的考虑，尤其是需要将 Excel 表格粘贴到 PPT 中进行展示时，我们又需要把一些单元格合并起来；

在这一分一合之中，如果只是少量的几行数据，那么手动一个个操作下也就罢了。然而，一旦数据量规模大了，再一个个合并或取消合并，就变成了一件机械、重复而又费劲的事情。这篇文章，我们就来谈谈如何能够得心应手地搞定 Excel 中合并单元格这件事。

本文中的举例和操作，都会用到上面案例中的 Excel：[**⏬ 示例文档.xlsx（点此下载）**](https://cdn.sspai.com/ee/%E7%A4%BA%E4%BE%8B%E6%96%87%E6%A1%A3.xlsx)

你可以下载后，自行对照本文操作和学习。

行单元格：你需要的，不一定是合并单元格
-------------------

我们先从最简单的地方说起：行单元格的合并。

在制作 Excel 表格的标题栏时，为了实现标题居中的效果，类似上图中的「银行客户经理 2018 年新增客户量」，我们往往会把表格的首行选中，合并单元格。然而，如果这时候你想在现有的表格后追加一列新的数据内容，例如除了列示客户经理的新增客户量外，再添加一列总客户量后，你会发现标题就不再居中对齐了。

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/DE6791E9-F6DF-4B0D-B0DA-B68CD38D64B6.png)

在现有表格后追加一列，原合并的标题单元格，就不再居中对齐了

这时候，你不得不先取消合并标题的单元格，再次重新选中首行的前五个单元格，重新合并一次。

虽然这个操作也并不烦琐，但是，Excel 中为了实现标题的排版，提供了更好更简单的方法，不需要每次都操作合并单元格。这个方法就是使用单元格的对齐设置。用这种方法，每次在插入新的列后，首行标题都能自动按新的表格宽度居中。

第一步：先选中你希望标题居中显示的行区域，如图中的 A1:D1；

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/8739B994-97A1-4440-B432-BA5CE29C5088.png)

选中表格的 A1:D1 区域

第二步：右键选中区域，选择「设置单元格格式」，进入到「对齐」选项卡，将「水平对齐方式」设置为「跨列居中」；

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/57E19815-1910-4453-9F4A-2E41BF546B8A.png)

在「对齐 - 水平对齐方式」中选择「跨列居中」

第三步：你可以看到，这时候标题已经处于居中的位置，而且在后面追加新的列时，首行标题会自动按新的表格宽度进行居中。

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/7054A8E4-4D3C-4D9F-A5FD-23D198A8BBCB.gif)

新增一列后，首行标题自动按新的宽度居中了

总结一下，这种方法相较于合并单元格，在表格内插入新的一列时，两者都能实现标题的自动居中对齐。它的主要优势体现在，当你在现有表格内容后追加新的列时，同样能实现标题自动居中对齐。

| 方法 | 在现有表格内插入列 | 在现有表格后插入列 |
| --- | --- | --- |
| 合并单元格 | 自动居中对齐 | 需要手动取消合并，重新选定区域合并单元格 |
| 使用跨列居中对齐 | 自动居中对齐 | 自动居中对齐 |

显然，设置「跨列居中」并不比合并单元格的操作复杂，但对排版调整的适应性更强。因此，在针对行单元格的合并时，主要建议就是：你不一定真的需要合并行单元格，不妨试试这种「跨列居中」的对齐方式。

列单元格合并
------

说完了行单元格的合并问题，接下来我们说说更常见的列单元格合并。如本文最初提到的「营业网点」，就是典型的列单元格合并。这里面最常见的两个问题就是，**如何快速批量合并单元格**，以及**如何批量取消合并**。

不过，在此之前，我们先来试着了解一下合并单元格这个操作，是如何影响单元格的值的。

### 理解合并单元格，是如何影响单元格的值的

以图中的「安庆集贤南路营业部」为例，这个合并后的单元格，占据了 A3:A7，共 5 个单元格的位置。那么，你有没有想过， A3、A4、A5、A6、A7 这五个单元格，分别是什么值呢？五个都是「安庆集贤南路营业部」吗？

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/316C64AD-1477-4751-8772-CBD6B1EDCF63.png)

「安庆集贤南路营业部」占据了 A3:A7，共 5 个单元格的位置

要回答这个问题，其实很简单。从三个方法来看：

1. 当你取消合并 A3:A7 该区域时，会发现只有 A3 单元格内有值，剩下的四个是空白。

   ![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/22124157-FB0C-441F-8A73-1DA368E9B86F.png)

   取消合并后，A4:A7 区域的单元格是空白的
2. 当你试着将 A3:A5 重新合并为一个单元格时，尤其是在 A4:A7 区域填入内容时，这时使用工具栏中的合并按钮，Excel 会提示如下，它明确地提示你，仅保留左上角的值，而放弃其他值。

   ![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/148154A8-17F5-43A0-A9F7-DE430525102B.png)

   使用工具栏的合并按钮时，会告警仅保留左上角的值
3. 更进一步地，虽然合并后的单元格在视觉上是一个格子，但我们依然可以试着引用这五个单元格内的值。不难发现，只有 A2 的值是「安庆集贤南路营业部」，其余 A3:A6 的引用值都是 0。在 Excel 中，引用一个空单元格的值，会得到 0。

   ![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/3217A598-5FD9-4914-A960-1D25B3E1AA3D.png)

   仅 A2 中有值，其余值为空。在 Excel 中，引用空单元格会得到 0

好了，到这一步我们可以确信，对于合并单元格这个操作来说，只有第一个格子保留了最终显示出来的值，其余值都为空。理解这一点非常关键，对于继续理解下面批量操作中的内容，有着不小的帮助。那么我们接下去看批量合并和批量取消合并的操作。

### 批量取消合并单元格并补全空白值

还记得我们最初的目标吗？为了完成表格的分析任务，我们需要将 300 多个被合并的营业网点，取消合并。

其实，批量取消合并单元格并不困难，只要选中整个 A 列，在工具栏中取消合并单元格就完成了。麻烦的是，按照我们刚刚说的，被取消后的单元格，只有第一行是有值的，其余行都是空白的。

为了能够形成一个有效的分析数据源，显然我们需要这些空白行，也能填上和第一行相同的内容。

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/A9533C13-FB04-4687-9D13-6EC5BCA801B4.png)

取消合并后，只有第一行有值

#### （1）IF 函数补全法

既然我们已经理解了一个单元格被合并后，值是如何变化的，我们可以利用这个特性，使用 IF 函数补全空白行。对于每一行来说，要不它有值，即代表这一行是被合并单元格的第一行；要不它是空白的，那么我们只需要让它和单元格的第一行一样即可。

如何构造这个函数呢？其实非常简单：

第一步：在 A 列边上插入一个辅助列；

第二步：在 B2 中输入公式如下，并双击填充本列后，即完成了补全，这时候可以删除 A 列，仅留下 B 列即可；

```
=IF(A2<>0,A2,B1)
```

要理解这个公式并不困难，只是对上述思路做了一些微小的调整。

* 判断是否为合并单元格的首行，即 A2 不为空，也就是不等于 0：A2<>0；
* 如果 A2<>0 为真，即 A2 为被合并单元格的首行，则使 B2=A2，即合并单元格的首行值；
* 如果 A2<>0 为假，即 A2 并不是首行，则让它等于上一行的值；这也是思路调整的地方，它并不直接等于合并单元格的首行值，而是通过循环的方式，只需要等于上一行即可；

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/6FB42683-BEA0-4F41-ACC5-ACB98ECCD45C.png)

在辅助列中用 IF 函数还原出被合并前的所有单元格

#### （2）空白值定位法

从我的经验来看，在身边许多人向我求助如何快速填充取消合并单元格后的空白值时，一部分人在知道 IF 函数补全法之后，自己再遇到这样的情况，就能得心应手了。不过的确也有些人，对函数公式不是很在行，这次搞定了，下次又忘了。因此，面对这部分人群，我后来还发现了一个空白值定位法，不需要用到函数公式，虽然操作烦琐了些，但更容易掌握。

空白值定位法，其实是利用 Excel 的一些相关功能，快速批量地模拟我们手工填充补全空白值的过程。想像一下，如果纯手工操作，无非我们就是一格一格地往下选中空白单元格，将首行内容粘贴进去。那么，如果利用 Excel 的已有功能，批量快速地代替手工的操作呢？

第一步：选中整个 A 列，在工具栏中取消合并单元格；

第二步：点击 Excel 菜单中的「编辑 - 查找 - 转到…」（可以通过 Ctrl + G 快捷键调出），找到这个定位功能，并在左下角进入「定位条件…」。在选中 A 列的情况下，定位「空值」。可以看到，空白处如 A3:A6、A8:A10、A16:A19 等区域，就被批量选中了；

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/7EB5B8C0-3826-4628-BC28-A6AE9ECA2F40.gif)

使用定位空白值的方法，批量选中空白处

第三步：保持空白单元格的选中状态，在首个单元格中输入 =A2，并使用 Ctrl+Enter 批量填充，就完成了空白值补全；

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/9485F16A-0F2B-4A39-885D-DC56A5E33789.gif)

输入 =A2 后，使用 Ctrl+Enter 批量填充

空白值定位法，相比 IF 函数法，的确更直观一些。虽然步骤和操作多了一些，但的确对于一些人来说，不需要记忆和理解 IF 函数的逻辑，只需要记住操作步骤，也不失为一个好方法。

需要注意的是，无论是 IF 函数法还是空白值定位法，完成上述操作后，你留在原先空白单元格内的值，还是公式引用的相对值。为了使表格能进行排序等操作，最终需要你选中整个 A 列，复制并选择「粘贴值」，从而只保留绝对值，这样最为稳妥。

### 逆向工程：快速批量合并单元格

现在，取消合并并快速填充空白单元格的小技巧，你已经掌握了。不过，在日常工作中，有不少情况下，是需要我们把单元格合并起来。那么，如何能快速批量地实现单元格合并呢？

这里我们也介绍两种最简单的方法，即「**数据透视表法**」和「**分类汇总法**」。

#### （1）数据透视表法

使用数据透视表法，其实是利用了数据透视表的排版功能。你只需要按照如下的操作步骤，就能快速地实现批量单元格合并。

第一步：选中所有数据，在「插入」选项卡中，选择「数据透视表」；你可以用示例文件中的「取消合并」Sheet 页，里面包含了可以操作的示例数据；

第二步：在数据透视表字段中，按原表格中的顺序，依次将字段拉入显示在「行」中；

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/D077348F-8057-4A53-890C-562A20E4A1C2.png)

将字段按顺序，显示在「行」中

第三步：选择「设计」选项卡，依次将「分类汇总」设置为「不显示分类汇总」，以及「报表布局」设置为「以表格形式显示」，得到初步的表格样式；

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/24405065-E058-41FB-919A-F7F4E33EB39E.png)

依次设置「分类汇总」和「报表布局」

第四步：右键透视表数据区域，选择「数据透视表选项」，打开「布局」选项卡，选中「合并且使带标签的单元格居中」；

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/3DD7C8DF-DEEB-409E-979C-E009E1FE751C.png)

选中「合并且使带标签的单元格居中」

第五步：确定后，数据透视步中的「营业网点」便合并单元格了。这时候，你可以复制整个透视表，获得并调整合并单元格后的样式；

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/499C8DC2-C5E8-40A8-94A5-87B32B97BCD0.png)

「营业网点」合并单元格完成

#### （2）分类汇总法

使用「数据透视表法」得到的结果，有一个不好的地方，因为这个方法相当于用数据透视表生成的结果，完全替代了原始数据，而 Excel 有一套默认的数据透视表样式，这意味着你需要手工重新调整表格样式，回到你需要的样子。

而分类汇总法，则可以**基于原表格做修改，几乎不需要调整样式**。而且它还有一个额外的好处，使用分类汇总法得到的表格，**即使被合并了单元格后，也不影响数据的筛选操作**。

好，我们先来看看分类汇总法要如何操作。

第一步：在「数据」选项卡中，打开「分类汇总」，选定汇总项为「营业网点」，并确定。

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/97E6BFFA-2C54-4E8B-9726-0B78A441E4B7.png)

使用「分类汇总」功能

第二步：表格中最前方被插入了一列。同样，利用之前空白值定位法，选中 A 列后，定位出空白值。你可能需要按住 Command 键手动反选 A1、A2 两个单元格；

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/4338CB26-925F-4C5D-8BB5-03140F26DF5B.gif)

使用定位空白值的方法，选中需要合并的区域

第三步：针对这些被选中的空白区域，点击工具栏中的合并单元格；

第四步：选中整个表格，再次打开「分类汇总」对话框，点击左下角的全部删除；

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/94296D1A-D651-46F1-8A98-F3C16198D9FF.png)

在「分类汇总」的左下角，选择全部删除

第五步：利用格式刷功能，选中 A 列已经合并的区域，将其格式应用到 B 列营业网点。可以看到，营业网点就按我们的预期被合并了。最后，再删除 A 列就完成了。

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/598F27FC-9D31-414F-B086-285F658C9A7E.gif)

利用格式刷功能，将 A 列合并单元格格式，应用到 B 列

可以看到，分类汇总法，虽然操作步骤比数据透视表法多，但是整个过程是在原表格的基础上完成的，不需要再花费很多时间，像数据透视表法那样调整最终得到的表格样式。

除此之外，我们前面提到了，这种方法得到的表格还有一个优势，它能够**正常筛选**。

像使用数据透视表法得到的表格，当你试图筛选「营业网点」这一列时，Excel 显示的筛选内容会出现问题。要理解这个原因也很简单，前文提到过，被合并单元格只有第一行是有值的，其余行其实是空白的，自然无法正常筛选。那么，为什么用分类汇总法得到的表格可以呢？这是怎么一回事呢？

为了一探究竟，我们来看看，使用分类汇总法、数据透视表法、以及手工合并这三种方式，最终得到的合并后单元格，其内的每个单元格的值，有没有什么不一样：

![](.evernote-content/56D2A516-4B32-4593-B9C6-0D836BFD4E8D/7D17546F-4282-407C-AF63-A5A08291DA87.png)

三种方法的对比

可以看到，神奇的是，数据透视表法和手工合并这两者，和我们在本节最开始讲到的一模一样：被合并的单元格，只有首行有值，其余行都是空白的。然而，使用分类汇总法得到的合并单元格，却保留了原先每一行的值，自然也就可以正常使用 Excel 的筛选功能了。

也就是说，Excel 在合并单元格时，默认都是会只保留左上角单元格的值，而清空其它剩下的单元格，只有一种情况存在例外：当使用格式刷来创建合并单元格时，所有原始数据都会被保留下来。

这样看来，「分类汇总法」的确是最佳批量合并单元格的方法。这样得到的表格，除了能够正常筛选之外，有朝一日发到他人手中，如果对方需要取消单元格的合并，直接取消合并单元格后，所有的原始数据都会直接显示出来，而不需要前文中提到的 IF 函数法或空白值定位法，自己去花时间恢复空白值。可以说是「功在当代，利在千秋」了。

---

好了，关于 Excel 中的单元格合并，就写到这里。再次复习一下要点：

* 行单元格合并：试试用「跨列居中」来替代合并单元格，实现标题居中等需求。
* 列单元格合并：
  + 批量取消合并，并填充空白值：
    - IF 函数法：操作简单，但需要一定逻辑理解。
    - 空白值定位法：步骤稍多，但容易记忆和掌握。
  + 批量合并单元格：
    - 数据表透视法：简单易学，缺点在于要重新还原表格样式。
    - 分类汇总法：推荐方法！步骤稍多，但不需要额外调整样式，且生成的合并单元格能够保留所有原始值。
