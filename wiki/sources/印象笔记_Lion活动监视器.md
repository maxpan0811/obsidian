点按“活动监视器”窗口底部的“系统内存”。

以下信息会显示：

|  |  |
| --- | --- |
| **可用空间：** | 可用内存是指尚未使用并且立即可用的内存。 |
| **联动：** | 联动内存包含必须总是驻留内存的信息。 |
| **活跃：** | 活跃内存包含当前正在使用的信息。 |
| **非活跃：** | 非活跃内存包含当前未在使用的信息。将此信息保留在内存中为您（或电脑的客户端）稍后回访该信息提供方便。 |
| **已使用：** | 已使用内存是指正被进程或系统使用的内存。它是联动内存、活跃内存及非活跃内存的总和。如果系统要求内存，则它会获取可用内存，再获取已使用内存。 |
| **虚拟内存大小：** | 虚拟内存大小是 Mac OS X 和应用程序所保留虚拟内存空间的总数量。所使用的实际虚拟内存的总数量可能要小得多，因为 Mac OS X 和应用程序经常会保留它们不使用的虚拟内存空间。 |
| **入页面：** | 从硬盘拷贝到内存的信息数量。圆括号中的数字是当前拷贝信息的速率。 |
| **出页面：** | 从内存拷贝到硬盘的信息数量。圆括号中的数字
