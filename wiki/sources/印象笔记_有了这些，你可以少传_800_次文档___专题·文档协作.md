原创 *2016-01-18* *应用推荐第一媒体*

![](.evernote-content/182449C6-D078-455C-9325-96563DD5D655/AEAB9A53-4BDA-426B-9DAC-53337F8F4A86.jpg)

团队间如何高效协作已经越来越成为大家关注的话题，不仅仅是通过网络的「远程办公」，甚至是近在咫尺的同事想要高效协作，也不是那么容易的事情。现在市场上也有越来越多的 App 和解决方案来帮助我们高效地分配工作，如 Trello，Teambition 等等。

今天我就想来聊一聊「协同写作」这个话题，协同写作指的是一个文档可能由多个成员来一起完成。

比如项目的策划，可能在写完之后就需要发给其他人看并收集反馈意见。这个时候最好的办法就是通过「协同写作」的 App 来和伙伴分享，它们可以让你及时看到伙伴的修改意见并做出反馈。

### **1. 印象笔记 #iOS #Android**

印象笔记是一款比较老牌的「合作」笔记本，但是严格来说它并不能算作「协同写作」软件。后者的一个特点就是会实时保存和同步正在编辑的文档，而印象笔记则采用一次同步的方式。

不过这并不会阻止印象笔记成为一款非常棒的团队协作应用，它可以在多个平台上使用，可以在其中插入图片、上传附件。印象笔记的亮点不在于它的写作功能上（虽然设计之初确实是这样的），而在于它能很轻松地拍下纸质文档，并转换成一个阅读性较强的图片。除此之外，强大的搜索功能可以让你搜索 PDF、图片、MS Office 文档中的特定文字。

另一方面，印象笔记可以很方便地保存有用的网页。可以通过 Chrome 上的插件，也可以通过手机的 App 并随时分享给自己的伙伴，有需要还可以使用「圈点」来添加批注。只可惜印象笔记不支持 Markdown 语法，只能使用第三方的服务如「马克飞象」等等。

![](.evernote-content/182449C6-D078-455C-9325-96563DD5D655/4646D51C-B1F9-458B-947E-4AE1C583B8CC.jpg)

### **2. Quip #iOS #Android**

Quip 是一个一开始就带有光环的 App，它的创始人是 Facebook 前 CTO Bret Taylor 和 Google Engine 之父 Kevin Gibbs。他们想在移动设备上取代 MS Office 的地位，强调跨平台尤其是移动端的服务。确实，很多时候我们打开文档不再是在电脑上打开，而是在手机、平板之类的移动设备上打开。

Quip 除了是一款「协同写作」工具之外，还是一款「即时通讯」工具，可以让你一边写作一边获得反馈。它还会记录下任何人对于文档所做的修改，并同步到其他人的设备。同时它也会在左侧的聊天框中显示出来修改的部分，让伙伴可以很轻松地对文档进行对照。

「协同写作」工具在排版方面都不是特别强大，提供的排版功能都极为有限。通常一个典型的协同写作 App 只会提供几个排版功能：加粗、斜体、下划线、删除线、代码块、待办事项、列表、标题和表格等等。

不过这样做的好处就在于，它可以让你在写作的过程中，更多地关注于写作的内容而非排版。不过对我来说，这些排版功能已经足够了，何况大多数时候我都不会用工具栏而是直接通过 Markdown 语法进行排版。

![](.evernote-content/182449C6-D078-455C-9325-96563DD5D655/82D8B612-E975-4D44-8953-AD9F713D6E40.jpg)

### **3. 石墨文档 #iOS #Web**

石墨可以看作一个中国版的 Quip 文档，从功能上和设计上都和 Quip 有些相似。不过和 Quip 的不接地气相比，石墨无论是稳定性还是外观设计都更加符合国人的使用习惯。

石墨对于 Markdown 的支持比较有限，仅支持用「#」来进行标题格式的修改。其实在我看来，石墨这么优雅的东西更适合做一些文艺小清新的事情。比如官方提到的那个故事，一个设计师发了一页「什么都没有」的白纸给女朋友，然而几秒钟之后页面上神奇地出现一个接一个的字，是设计师给女友的告白心声。

扫一扫分享到微信我觉得是石墨最棒的设计，因为对方不需要再去下载一个 App （何况也没有 App 给他下载）就可以查看文档。但是不得不说，手机上打开网页版真的使用体验很差，查看足以，但是想编辑还是差了不少。

目前石墨推出了 iOS 的客户端，然而 Android 用户却只能苦逼地打开网页版了。

![](.evernote-content/182449C6-D078-455C-9325-96563DD5D655/36DCECC8-CD17-40AA-A4B8-F3FC38B09E40.jpg)

### **4. Google Docs #iOS #Android**

Google 出品，必属精品。如果说前面几款 App 主打的都是「小而美」，那么 Google Docs 主打的就是「大而全」。你可以把它理解成一个被搬到网页上的 MS Office，当然，因为是 Web App 同时是针对移动端优化的产品，所以功能上比 MS Office 稍逊一筹。

除了不接地气、不符合国情之外，Google Docs 还是强大得没话说的。一些桌面时代常用的排版功能都被搬到了网页上，更换字体、插入公式和绘图、调整行间距、自定义页眉页脚等等。只可惜 Google Docs 不支持 Markdown，排版上还属于最让人无奈的那种选中然后才能调整格式的情况。

不过如果仅仅是作为一个跨平台的协同写作工具的话，Google Docs 的功能和体验还是不错的。这也是我将 Google Drive 而非 OneDrive 当作主力的一个重要原因。

![](.evernote-content/182449C6-D078-455C-9325-96563DD5D655/B3FA26D0-47C7-4DCB-B47F-0BA101828AC6.jpg)

除了文中提到的这四款 App，还有很多不错的协同写作 App，比如来自 Dropbox 家的 Paper、Hackpad、Teambiton 的分享功能等等。

那么，跟「我刚把最新版发你了，你改完再发过来」道个别吧？
