# 美 + 整理 + 分享 + 稍后读 = 下一代书签服务 Raindrop.io

---

![](.evernote-content/72ED416F-0F8F-4B6B-A3C8-AA6C9D2E1A08/4838F133-EA8D-42FC-8379-7C9CD1554815.jpg)

我一度特别不明白，为什么有的人需要用专门的第三方书签服务，你可能喜欢 [Stache](http://sspai.com/search?q=stache) 与众不同的美，或是需要 [Delicious](https://delicious.com) 来分享与发现更多有意思的网站，然而对我来说，这些都不足以打动我，毕竟浏览器自带的书签收藏功能已经足够好用，不管是 Safari 还是 Chrome 的书签功能，满足了基本管理 + 同步的需求，为什么我还需要额外的第三方书签服务？

一开始使用 [Raindrop.io](http://Raindrop.io) 时，我也是带着这样的困惑。一段时间使用下来，我已经把书签都转移到了 [Raindrop.io](http://Raindrop.io) 上面，如果你只是打开[它的官网](https://raindrop.io/)，它对自己的介绍是「你喜欢的网站，触手可及」，然而这句话实际上并没有表现出 [Raindrop.io](http://Raindrop.io) 的真正过人之处。在我看来，它代表了下一代书签，或者说书签本来应该有的样子：美 + 整理 + 分享 + 稍后阅读 = Raindrop.io。

为什么说 [Raindrop.io](http://Raindrop.io) 代表了下一代书签？

让我们回到源头，想想到底什么是书签服务？本质上，它是对一个个 URL 的处理，不同的 URL 对应了不同的网站和内容，你可以通过书签的方式整理归类这些网址。

然而，传统的浏览器书签仅仅提供了书签的命名、文件夹功能，可是，当我们收藏一个网页时，我们不仅仅只是想记录下这个单调的网站名称，或简单的把它放到某一个文件夹下面。我们还想要把某个网站分享给别人，或者网页上有什么重要的内容，我们想保存下来稍后再看。从这一点上来看，Safari 的书签+阅读列表+共享链接」其实不就是照着这个思路做的吗？

[Raindrop.io](http://Raindrop.io) 也是，它的整个理念，不仅仅只是对书签的简单整理和分类，更在此基础上添加了分享和稍后阅读的功能，让一串网址背后对应的实际内容更加突显出来，在人与人之间传播。

![](.evernote-content/72ED416F-0F8F-4B6B-A3C8-AA6C9D2E1A08/373840F5-AA69-4852-8622-514879F6995C.jpg)

跨平台地管理书签

[作为一款书签软件，跨平台自然是少不了的基本功。Raindrop.io](http://%E4%BD%9C%E4%B8%BA%E4%B8%80%E6%AC%BE%E4%B9%A6%E7%AD%BE%E8%BD%AF%E4%BB%B6%EF%BC%8C%E8%B7%A8%E5%B9%B3%E5%8F%B0%E8%87%AA%E7%84%B6%E6%98%AF%E5%B0%91%E4%B8%8D%E4%BA%86%E7%9A%84%E5%9F%BA%E6%9C%AC%E5%8A%9F%E3%80%82Raindrop.io) 支持的平台十分广泛，从各种主流浏览器的插件，到手机上的 App，再到 Mac 上的客户端，都一应俱全：

浏览器插件：[Chrome](https://chrome.google.com/webstore/detail/raindropio/ldgfbffkinooeloadekpmfoklnobpien) / [Opera](https://addons.opera.com/extensions/details/raindropio-smart-bookmarks) / [Firefox](https://raindrop.io/releases/raindropio.xpi) / [Safari](https://safari-extensions.apple.com/details/?id=io.raindrop-7459JWM5TY) / [Yandex Browser](https://addons.opera.com/extensions/details/raindropio-smart-bookmarks) 新的标签页：[Chrome](https://chrome.google.com/webstore/detail/raindropio-new-tab-speed/knifgjkgmgdinjeecneiphoniamhgbof) / [Firefox](https://raindrop.io/releases/newtab.xpi) App：[iOS](https://itunes.apple.com/us/app/raindrop.io-keep-your-favorites/id1021913807?uo=4&at=10lJSw?uo=4&at=10lJSw) / [Android](https://play.google.com/store/apps/details?id=io.raindrop.raindropio&hl=en) 桌面软件：[OS X](https://itunes.apple.com/app/raindrop.io-keep-your-favorites/id957810159?uo=4&at=10lJSw?uo=4&at=10lJSw)

特别是新的标签页插件，可以直接替代 Chrome 或 Firefox 的默认新标签页，这样就能完美替换掉浏览器自身的书签服务了。你只需要把书签加入到「最喜爱的网站」这个分类，就会默认出现在新标签页上了。

![](.evernote-content/72ED416F-0F8F-4B6B-A3C8-AA6C9D2E1A08/B7B0C517-E53C-4A85-9F1F-58A7932969E4.jpg)

[另外，你可能已经注意到了，Raindrop.io](http://%E5%8F%A6%E5%A4%96%EF%BC%8C%E4%BD%A0%E5%8F%AF%E8%83%BD%E5%B7%B2%E7%BB%8F%E6%B3%A8%E6%84%8F%E5%88%B0%E4%BA%86%EF%BC%8CRaindrop.io) 在管理书签时，引入了标签的概念，除了可以将网页放入不同的分类和文件夹中，你也可以直接给网页打上多个标签，写上描述文字，并且这些信息都可以直接被搜索到。另一方面，如果你不满意网页原来自带的图标，你可以直接在封面中搜索关键词，会找到相应的高清图标，替换网页的原来标识。

![](.evernote-content/72ED416F-0F8F-4B6B-A3C8-AA6C9D2E1A08/05296F87-FC89-456D-9451-305A6A739360.jpg) ![](.evernote-content/72ED416F-0F8F-4B6B-A3C8-AA6C9D2E1A08/CFC5847C-A41A-428B-9CFB-0B3240E2C3EE.jpg) 如果是首次开始使用 [Raindrop.io](http://Raindrop.io) 的话，完成以下操作会更方便日后的使用： 安装各个使用的浏览器插件； 在 iOS 的 Safari 分享菜单中启用 [Raindrop.io](http://Raindrop.io) 的插件； 导入已有的书签。

![](.evernote-content/72ED416F-0F8F-4B6B-A3C8-AA6C9D2E1A08/5AA7F617-AEDF-4D19-B82C-EC61A4A1B161.jpg)

用 [Raindrop.io](http://Raindrop.io) 替代 Pocket

我们很多人都在使用 Pocket 来做稍后阅读，然而本质上，Pocket 和书签保存的都是一个网址，为什么不让这两件事情放在一起完成呢？就像 Safari 中的「书签 + [阅读列表」一样，Raindrop.io](http://%E9%98%85%E8%AF%BB%E5%88%97%E8%A1%A8%E3%80%8D%E4%B8%80%E6%A0%B7%EF%BC%8CRaindrop.io) 给出了更加优雅的解决方案：它可以自动识别你保存的网页内容类型，并放到相应的分类里。

当你在任何一个平台上保存网页时，都会默认进入到「Inbox」这个分组下，或者你可以直接选中一张图片，一段文字，也都能够直接保存。这时「Inbox」这个列表就和稍后阅读很像，你并不一定需要长期收藏这些链接，只是需要一个地方把来不及看完的网页放在一起。

[另外，Raindrop.io](http://%E5%8F%A6%E5%A4%96%EF%BC%8CRaindrop.io) 还会智能识别网页内容，如果发现是一篇文章，或照片和视频，还会自动进入到「Articles」和「Photos & content」的分类，想看文章就更方便了。

![](.evernote-content/72ED416F-0F8F-4B6B-A3C8-AA6C9D2E1A08/93F1EF71-DDB8-4B2A-80BF-B1ABF26BB177.jpg) ![](.evernote-content/72ED416F-0F8F-4B6B-A3C8-AA6C9D2E1A08/892E4DE7-C708-43C2-A68D-796ADAFC70AF.jpg) ![](.evernote-content/72ED416F-0F8F-4B6B-A3C8-AA6C9D2E1A08/BC592631-95C7-4578-8C8A-C51975C60060.jpg)

在阅读体验上，如果 [Raindrop.io](http://Raindrop.io) 识别到当前页面其实是一篇文章，会自动开启类似 Pocket [中的阅读模式，对网页上的内容进行格式化排版以便于阅读。不过在不同的平台上，Raindrop.io](http://%E4%B8%AD%E7%9A%84%E9%98%85%E8%AF%BB%E6%A8%A1%E5%BC%8F%EF%BC%8C%E5%AF%B9%E7%BD%91%E9%A1%B5%E4%B8%8A%E7%9A%84%E5%86%85%E5%AE%B9%E8%BF%9B%E8%A1%8C%E6%A0%BC%E5%BC%8F%E5%8C%96%E6%8E%92%E7%89%88%E4%BB%A5%E4%BE%BF%E4%BA%8E%E9%98%85%E8%AF%BB%E3%80%82%E4%B8%8D%E8%BF%87%E5%9C%A8%E4%B8%8D%E5%90%8C%E7%9A%84%E5%B9%B3%E5%8F%B0%E4%B8%8A%EF%BC%8CRaindrop.io) 采取的机制稍有不同。在 iOS [设备上，Raindrop.io](http://%E8%AE%BE%E5%A4%87%E4%B8%8A%EF%BC%8CRaindrop.io) 会直接调用 Safari View Controller，并直接开启 Safari 自带的阅读模式。而在 Mac 和浏览器插件中，则自带了一套排版引擎，可以选择不同的配色、字体及大小。

![](.evernote-content/72ED416F-0F8F-4B6B-A3C8-AA6C9D2E1A08/430E2C11-D1D6-457E-8BB5-7749EC97D325.jpg)当然想要 [Raindrop.io](http://Raindrop.io) 完全替代 Pocket [还是有点代价的，Raindrop.io](http://%E8%BF%98%E6%98%AF%E6%9C%89%E7%82%B9%E4%BB%A3%E4%BB%B7%E7%9A%84%EF%BC%8CRaindrop.io) 不支持离线保存网页内容，需要保持联网的状态才能拉取网页上的内容进行格式化。

[Raindrop.io](http://Raindrop.io) 甚至还能生成你的个人 RSS

[Raindrop.io](http://Raindrop.io) 还支持你共享某个文件夹，你可以通过邮件的形式邀请他人，或者直接将某个文件夹生成对应的网页链接分享出去，别人只要访问该链接就能查看文件夹内的所有书签收藏。例如我文件了「TV」这个文件夹，任何人都能够通过[该网址](https://raindrop.io/collection/805730)访问到文件夹中的书签内容。

![](.evernote-content/72ED416F-0F8F-4B6B-A3C8-AA6C9D2E1A08/58CDA06B-47C0-4196-A099-0D45D733729A.jpg) ![](.evernote-content/72ED416F-0F8F-4B6B-A3C8-AA6C9D2E1A08/BF50E5B1-9732-4E27-8574-890AF123BD97.jpg)

当然这还不是最酷的，通过 Raindrop.io，你甚至可以以文件夹为单位，生成不同的 RSS 订阅源。开发者还非常贴心地考虑了大家的隐私安全问题，订阅源默认是分为私人和公开的。公开的订阅源只有分享了该文件夹才可以被访问，而私人的订阅源则会生成一串随机的链接，仅自己能够使用，不过这也意味着不要把私人的订阅源直接发给别人，否则就意味着即使你没有分享自己的书签文件夹，别人也能看到你的书签了，仅仅只是为了方便自己使用哦！

![](.evernote-content/72ED416F-0F8F-4B6B-A3C8-AA6C9D2E1A08/251F40AF-7AAA-4646-A48B-3D2C03D49C3B.jpg)[怎么样？Raindrop.io](http://%E6%80%8E%E4%B9%88%E6%A0%B7%EF%BC%9FRaindrop.io) 称得上最完美的下一代书签服务吗？如果一定要说美中不足的事情，那就是 Safari 上的浏览器插件风格是浓浓的 Material Design 风格，和整体显得有些违和。以上介绍到的这些功能都是完全免费的，如果你愿意每个月花费 2 刀升级到专业版，还可以享受到 Dropbox 备份、多层级文件夹、无广告（虽然我并没有发现哪里有广告）和优先服务支持等特权。
