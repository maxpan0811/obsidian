# 微软发布全新无线通用折叠键盘：售699元

9月21日消息，微软公司今日正式在中国发布了支持跨平台使用的微软无线通用折叠键盘。该产品专为 iPad、iPhone、Android 设备，以及 Windows 平板而设计，售价699元。  
  
　　**兼容多种不同的操作系统**  
  
　　微软无线通用折叠键盘可在平板电脑和智能手机之间任意切换，能够兼容多种不同的操作系统，专为 iPad、iPhone、Android 设备，以及 Windows 平板而设计，使用键盘右上角的“OS”切换键开关即可快速实现不同操作系统间的自由切换。  
  
　　它能同时与两款不同操作系统的移动设备进行配对，当你需要跨设备进行文件编辑时，无需频繁地断开连接并配对设备。长按左上角的“设备1”或“设备2”按键几秒钟，待其指示灯开始闪烁，键盘就能够快速的与您的设备相连接。如使用完毕，只需合上键盘就能将其关闭。  
  
　**仅5毫米厚可折叠放入口袋**  
  
　　无线通用折叠键盘仅5毫米厚，并采用创新的可折叠设计，用户能够轻松的将其折叠放入钱包或口袋中，同时无线通用折叠键盘还采用了防泼溅设计，专门的防水材料制作而成。  
  
　　微软无线通用折叠键盘使用最新的智能蓝牙4.0技术，采用内置可充电电池，一次充电能够使用长达3个月的时间。  
  
　　据悉，微软无线通用折叠键盘在国内各大线上线下渠道开始发售，零售价为699元。
