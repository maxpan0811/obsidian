# Nvidia发布高端显卡Titan Xp：支持Mac平台

---

威锋网讯，Nvidia 今天宣布其最新的超高端显卡 Titan Xp 正式发布，这款被 Nvidia 称为“全球最强劲性能“的显卡配备 12GB 的 GDDR5X 显存（11.4 Gb/s），3840 CUDA 核心（1.6GHz）和 12 FLOPS 的单精确度浮点数运算性能。

![](.evernote-content/37DB4033-CCB2-4AE1-B398-089728D72E10/7133AA16-9593-4D7A-956E-142BD9A0C430.jpg)

售价 1200 美元，今年的 Titan 显卡非常特别，因为 Nvidia 还提供了新版本的驱动程序，让基于 Pascal 的 Nvidia 显卡第一次支持 Mac 平台。本周早些时候，苹果公布了未来高端 Mac Pro 的计划，它将会支持更好的显卡，因此像 Titan Xp 这样的超高端显卡已经提供支持将是一个好的兆头。另外，Titan Xp也可能支持旧款 Mac Pro 和黑苹果设备。

Titan Xp 采用了和 Titan X 几乎完全相同的整体设计，定位也依然是面向高性能计算、专业图形、VR 开发等领域。据Nvidia表示，这款显卡的性能比上一代显卡高出三倍。Titan Xp 从今天开始就可以从 Nvidia 的网站进行购买，各位“土豪”不妨前往了解更多详情。
