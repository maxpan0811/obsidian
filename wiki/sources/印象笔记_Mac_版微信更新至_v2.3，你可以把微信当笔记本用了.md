# Mac 版微信更新至 v2.3，你可以把微信当笔记本用了

---

自从在去年经历了 364 天的更新间隔后，Mac 版微信的更新终于变得频繁起来，不但在功能上赶上了 Windows 版，有时甚至会优先加入一些 Windows 都还没有到新功能。虽然在测试阶段就已经有不少人都知道这次更新的新功能了，不过直到这周一，微信才正式在 Mac App Store 发布 v2.3 更新，除了对现有功能进行优化之外，还加入了笔记功能，让你可以把微信当成笔记本来用。

把微信变成笔记本，不但能写字还能同步文件
--------------------

如果你经常使用微信的收藏功能，那你应该知道这个收藏功能其实并不是 mark 一下公众号文章那么简单，[少数派此前就向大家介绍过](https://sspai.com/post/33185)，它其实还是一个小型记事本，不但能记纯文本，语音、图片、位置信息也不在话下，完全满足了「随手记」的需求。

![](.evernote-content/41159043-DEAD-44C5-BBA7-CA53434A32A1/698F5774-0185-49E3-BA5B-6E1B0174CA23.jpg)在手机微信上记笔记

这次 v2.3 更新，微信把这个功能带到了 Mac 上，在 Mac 版的「收藏」选项内，可以看到「新建笔记」的选项，点击后会弹出一个独立的笔记窗口供你记录内容，笔记提供了粗体、斜体、下划线等基本的文字格式，还支持拖拽上传文件。写好的笔记可以同步到云端，也可以直接发送给好友。

![](.evernote-content/41159043-DEAD-44C5-BBA7-CA53434A32A1/63C3CB0E-1773-4903-B09F-66F574E62B67.png)

虽然 Mac 版现在还不能插入语音和位置信息，但是在手机上记录的语音和位置信息在电脑上都可以直接查阅。

![](.evernote-content/41159043-DEAD-44C5-BBA7-CA53434A32A1/8A46FE8D-ABCE-4EE7-9C97-5E1D03086AD3.png)点击位置信息直接打开系统地图

找人、找聊天信息更方便
-----------

除了记笔记外，Mac 版微信还增加了搜索群成员和查找聊天记录的功能，在群聊信息的侧边栏内，可以直接在搜索框内搜索群内成员。

![](.evernote-content/41159043-DEAD-44C5-BBA7-CA53434A32A1/989A1BCF-2072-4E8B-BACF-B07B35E18BB2.png)

而聊天记录查找功能则像手机一样把聊天记录分为「全部」「文件」「图片与视频」「链接」四类，和手机相比少了「日期」和「音乐」分类，不过每一类信息都会直接呈现在对应选项下，看起来非常直观，已经方便了不少。

![](.evernote-content/41159043-DEAD-44C5-BBA7-CA53434A32A1/248B4837-687A-4A25-BFE4-FA1676461B7D.png)

备份聊天记录可以只备份文字部分了
----------------

在热门聊天工具中，微信恐怕是少数几个还需要手动备份聊天记录的，而它又在很多人的生活和工作中都担任重要角色。

所以微信的聊天记录如果真的对你很重要，定期备份很有必要，在 v2.3 中备份聊天记录时，可以在手机端选择仅备份文字部分了。这无疑加快了备份的速度，前提是那些图片你已经都备份好了。

![](.evernote-content/41159043-DEAD-44C5-BBA7-CA53434A32A1/E42C61B5-A4CA-4C9D-8651-D4F4D12F894D.jpg)
