# OS X 新手技巧：如何自定义Mac的应用图标

---

威锋网讯，相比于 iOS，苹果的桌面系统 OS X（现在已更名为 macOS）毫无疑问有着更加丰富的自定义属性。在 macOS Sierra 正式版还没到来之前，不少朋友依然使用的是 OS X El Capitan，今天我们就来一起学习如何在这个系统中对应用的图标进行自定义吧。

![](.evernote-content/7D6B32CC-7FA5-418B-A881-8ACCD2B77BF1/BB24D9FD-FB27-4C9F-8F90-C302278FFB69.png)

1.在 Finder 或桌面中找到一个应用，右键点击它，然后选择“显示简介”

![](.evernote-content/7D6B32CC-7FA5-418B-A881-8ACCD2B77BF1/0B38947F-4CF1-447B-9FF7-343D55BC679C.png)

2.打开该应用的“显示简介”菜单

![](.evernote-content/7D6B32CC-7FA5-418B-A881-8ACCD2B77BF1/AC233154-D3CE-4C0A-A248-19FB2531B962.png)

3.接下来找到你要使用的应用图标图片，使用“预览”打开

![](.evernote-content/7D6B32CC-7FA5-418B-A881-8ACCD2B77BF1/9EA01967-B490-4378-8E27-CCD9A8C5E000.png)

4.使用“预览”打开后，点击菜单栏的编辑-全选来选择整幅图像

![](.evernote-content/7D6B32CC-7FA5-418B-A881-8ACCD2B77BF1/4814696C-311B-4FC2-8053-92A6936981B9.png)

5.在图片被选择的情况下，找到菜单栏的编辑-复制，将图片内容复制到 Mac 的粘贴板上

![](.evernote-content/7D6B32CC-7FA5-418B-A881-8ACCD2B77BF1/5B614908-734D-4538-92A4-CABB2ADE1823.png)

6.返回到“显示简介”窗口，选择点击窗口左上角的按钮，然后找到菜单栏中的编辑-粘贴

![](.evernote-content/7D6B32CC-7FA5-418B-A881-8ACCD2B77BF1/05941FF2-CD3C-4206-A51C-E0104D69873D.png)

7.根据更改的应用图标的不同，你可能会被要求输入密码

![](.evernote-content/7D6B32CC-7FA5-418B-A881-8ACCD2B77BF1/B6B7222D-5020-4893-856C-3FCD23095C35.png)

8.现在图片应该已经应用到应用中了，但要使其生效，最好还是重启一下 Mac

9.重启后现在可以再次打开该应用的“显示简介”窗口，看一下更改后的变化

![](.evernote-content/7D6B32CC-7FA5-418B-A881-8ACCD2B77BF1/C899A404-44C0-48D2-A5FB-15230F2060B7.png)
