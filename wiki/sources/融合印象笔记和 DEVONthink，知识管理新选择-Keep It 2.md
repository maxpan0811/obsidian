# 融合印象笔记和 DEVONthink，知识管理新选择：Keep It

融合印象笔记和 DEVONthink，知识管理新选择：Keep It
==================================

说起知识管理类 app，大家对 [Evernote（印象笔记）](https://sspai.com/topic/259) 和 DEVONthink 会比较熟悉，这两款产品也代表了这个领域入门和进阶的最佳选择，但是它们显然也不能满足所有用户的需求。这个时候，很多处于中间地带的产品就孕育而生了，Keep It 就是其中之一。和 DEVONthink 一样，Keep It 也是一款 iOS 和 macOS 平台上的文档管理 app，但是拥有更明朗清晰的功能、更现代美观的界面和更统一的多平台体验。

按照知名效率网站 [The Sweet Setup](https://thesweetsetup.com/) 对 Keep It 的介绍，这是一款 Evernote 的最佳替代品，而 [MacStories](https://www.macstories.net/) 的主编 Federico Viticci 对它的评价是「一款融合了 DEVONthink 与 Evernote 的产品，不像后者这么臃肿，比前者相比又更多地与 iOS 系统集成，同时比两者都更好看。」

如果你和我一样，没有特别冗杂繁重的知识管理需求，又不满足于 Evernote 孱弱的文件格式支持和结构层级，那么你也应该试试 Keep It，也许它正好能击中你的痛点。

资料搜集
----

一般来说，「资料搜集」是文档管理的第一步，我们需要将各种各样的资料从不同的来源搜集到同一个 app 中，而其中最重要的两个部分是搜集方式和格式支持。

与 DEVONthink for Mac 和 Evernote 的桌面端相比，Keep It 的搜集方式有点简陋，不像前两者拥有各自的浏览器插件，Keep It 只在应用内提供了一个「添加」的功能，你可以通过这个按钮添加本地文件和网页链接。值得一提的是，Keep It 的「添加」功能在 iOS 客户端和 macOS 客户端上功能和体验几乎一致，不会带来使用上的割裂感。

![](.evernote-content/12CDBCB1-8F60-4F29-9CAA-5800E3C1E20C/A789D615-DCF0-4675-8FC8-5CBBEC5A76E5.jpg)

「添加本地文件」很好理解，你可以将所有类型的文件都添加到 Keep It，都可以通过 iCloud 在多平台之间同步，不过只支持对文本、图片等类型的文件进行预览或者编辑。

![](.evernote-content/12CDBCB1-8F60-4F29-9CAA-5800E3C1E20C/CA34A2C4-46FB-40CA-A4C6-B8AFD9708FB1.jpg)

「添加网页链接」则支持了 3 种格式，分别为 Web Link（后缀为 webloc）、web archive、PDF（分为单页和分页两种），其中 Web Link 是一种在线格式，打开时需要联网重新加载，而 web archive 和 PDF 是离线格式，可以在离线情况下正常使用。

除了添加现成的文件和网页链接，Keep It 还支持新建 Markdown、富文本、纯文本等格式的笔记，并且可以通过模版来快速添加。

![](.evernote-content/12CDBCB1-8F60-4F29-9CAA-5800E3C1E20C/709168D7-03CF-43B7-A6A6-4BC2D541B4F3.jpg)

显而易见，Keep It 在桌面端上的搜集方式肯定没有 DEVONthink 和 Evernote 的浏览器插件来得方便，不过也勉强够用了，而在 iOS 平台上，这三者就几乎没什么区别了，都可以通过分享表单或者 app 本身来添加。至于格式支持，Keep It 可以说是完爆 Evernote 了，它对文件格式的覆盖程度甚至可以媲美 DEVONthink。

资料整理
----

将资料搜集进来后，接下来的一步就是整理了。

Evernote 采用的是「笔记本组 - 笔记本」的结构，所以只能实现二级层级；而 DEVONthink 就复杂了很多，它通过「Sync Store-Database - 文件夹 / 标签」来构成自己的层级。DEVONthink 和 Evernote 的标签都是可以无限嵌套的，但只有 DEVONthink 的文件夹支持无限嵌套，Evernote 并不支持。

是不是已经看晕了？我们再来看一看 Keep It 的结构层级。Keep It 的文件整理形式主要有两种：Folder（文件夹）和 Bundle（捆绑包）。文件夹可以嵌套文件夹和捆绑包，其中文件夹可以无限嵌套，而捆绑包不行，捆绑包只能单独存在或者包含于一个文件夹之下。

![](.evernote-content/12CDBCB1-8F60-4F29-9CAA-5800E3C1E20C/3A99275A-8045-4721-ADB2-1F6D362A6BC2.jpg)

文件夹与捆绑包的另一个区别是文件只能放置在一个文件夹下，但是能放置于多个不同的捆绑包下。这样一来，我们就能利用捆绑包来整合不同文件夹中的文件。如下图所示，我将「ufunc 函数」这个文件放到不同的捆绑包后，我可以在「Show in」菜单下看到它的不同位置。

![](.evernote-content/12CDBCB1-8F60-4F29-9CAA-5800E3C1E20C/60F53A34-7FAC-46BB-9CC7-D34305F09FF1.png)

除了文件夹和捆绑包，Keep It 还加入了 Tag（标签）和 Label（标记）。对于标签，大家应该都很熟悉了，在大多数的知识管理 / 文本编辑器 app 中都有这个功能，你可以使用标签来为文件添加分类信息，同时一个文件可以添加多个标签。不过需要注意的是，在 Keep It 中标签是不可以嵌套的，所有的标签都属于同一层级。

![](.evernote-content/12CDBCB1-8F60-4F29-9CAA-5800E3C1E20C/910D8024-9842-46BF-9C4E-427111035B19.png)

标记这个功能对于 Evernote 用户可能有点陌生，不过 DEVONthink 也是有这个功能的。虽然意思上 Label 和 Tag 很接近，但是两者在用法上有很大的区别。针对一个文件，你可以为它打上多个不同的标签，但是你只能将它打上一个标记。除此之外，Tag 只能在文件夹 / 标记的详情列表以及文件详情页看到，而标记直接出现在侧边栏中，通过分配的颜色可以非常方便地找到对应的文件。

![](.evernote-content/12CDBCB1-8F60-4F29-9CAA-5800E3C1E20C/7BD5CFAE-9827-46C1-B266-4310B23F5CF4.png)

除了基础的分类和归纳，Keep It 也提供了过滤的功能。你可以通过文件名、文件类型、来源、标签、标记、创建时间等条件进行筛选，匹配的所有文件都可以通过保存在侧边栏的「已保存的搜索」来快速调取。Keep It 的筛选条件虽然不及 DEVONthink 来得及那么丰富，但也远远超过了 Evernote 的数量。

![](.evernote-content/12CDBCB1-8F60-4F29-9CAA-5800E3C1E20C/C503D22D-CB41-40BC-AB03-EE0FFF5DB914.jpg)

资料搜索
----

当搜集的资料都已经整理好之后，我们就需要通过合理的方式来调用它们。除了用按图索骥的方式从文件夹或者捆绑包中去找文件，我们更倾向于用搜索的方式更快地找到目标文件。

Keep It 除了支持对文本文件进行搜索，还支持对 PDF 和图片中的文字进行搜索。除了系统默认支持的英文，你还可以在设置中为索引选择不同的语言或者字母体系，甚至还可以自己手动导入语言文件。

![](.evernote-content/12CDBCB1-8F60-4F29-9CAA-5800E3C1E20C/FC655627-0C62-495A-92D7-1132B05B6CE5.jpg)

根据我这段时间的测试，Keep It 对文本文件、离线网页文件（web archive）和 PDF 都有较好的中文 / 英文支持，但是对在线网页文件（webloc）和图片中的中文 / 英文都有点无可奈何。不过我想这对大多数人来说，已经可以满足基本的需求了。以下图为例，我对同一个文件（web archive 格式）中的中 / 英文内容进行了搜索，结果都可以成功获取到结果。

![](.evernote-content/12CDBCB1-8F60-4F29-9CAA-5800E3C1E20C/6D763C73-9D97-44F3-ABA8-4359C07152C4.jpg)

为什么选择 Keep It？
--------------

花钱的事，不谈需求都是扯淡。虽然本文的目的是向大家推荐 Keep It 这款 app，但是我还是希望大家能先理清自己的需求。

如果你是一名热衷于搞学术的学生，或者科研工作者，那么我更加推荐你去购买 DEVONthink。首先，DEVONthink 拥有更多的结构层级和更丰富的过滤参数，这也就意味着它可以满足你更深层的知识管理需求；其次，DEVONthink 拥有更强大的搜索功能，可以对搜索结果按相关性排序，并且在更新 3.0 版本后更好地支持了中文搜索；最后，DEVONthink 除了 macOS 和 iOS 版本，还提供了网页版，对于日常学习或者工作中要频繁使用 Windows 电脑的人来说无疑更加友好。

关联阅读：[5 个实用脚本，把 DEVONthink 打造成个人知识库](https://sspai.com/post/44109)

如果你不想花钱，又极度依赖 Evernote 的网页剪藏工具，以及它与第三方工具 / 服务的联动性，那么不妨安心地留在 Evernote。至于印象笔记（Evernote 中国特供版），在它完成整合一连串的新功能之前，个人并不推荐新用户入坑。

关联阅读：[用平凡的工具创造精彩，这些印象笔记的实用技巧值得一试](https://sspai.com/post/55712)

剩下的人，那么应该就和我一样了，既用不上 DEVONthink 如此强大又复杂的功能（当然还有高昂的价格），又不满足于 Evernote 多年的碌碌无为，Keep It 就是为我们准备的，功能开箱即用，上手门槛极低，多端体验一致，iCloud 无缝同步……
