# 携程产品排序规则

---

![](.evernote-content/B3CE6B4B-563E-4865-8011-3E81851F62FC/C3ACBBED-1752-4624-962D-0418A3E5A591.jpg)

  

|  |  |  |
| --- | --- | --- |
| 排序因子 | 定义 | 权重  （动态调整中） |
| 搜索相关度 | 产品与搜索内容的相关性和匹配度 | 中 |
| 转化率  （分售卖站） | 近30天预订浏览比（预定量/浏览量） | 高 |
| 新品指数 | 产生新产品id时，给每个新产品一个推广时间（推广时间详见FAQ） | 低 |
| 点评指数 | 该产品的用户点评数量及点评分数 | 高 |
| 关注度  （分售卖站） | 该产品的近30天总预订量 | 高 |
| 产品缺陷  （9月18号上线） | 供应商行中质量分（由业务提供一个季度更新一次） | 中 |
| 毛利指数  （分售卖站） | 该产品近两个月的毛利额 | 高 |
| 动销率 | 产品供应商有效量产品占比 | 低 |
| 预订成功率 | 近60天产品满位或不成团的产品缺陷分 | 中 |

4高3中2低，基本上高权重就决定了排名，中低权重忽略不计

Q： 新品指数怎么算的？

系统只判断新ID（复制老产品到新产品id，系统则判断为新产品）；

根据产品类型预设推广期，推广到期加权分衰减到零；

a. 产品类型=出境旅游、邮轮、签证、境外N日游或者境外1日游，则根据目的地判断天数。具体规则见下：

目的地属于香港、澳门，则新品推广时间=45天

目的地属于欧洲、美洲、中东非洲、澳大利亚、新西兰，新品推广时间=90天

其他目的地，则新品推广时间=60天

b. 产品类型=国内旅游，新品推广时间=30天

c. 产品类型=境内N日游，新品推广时间=14天

注： 老产品换马甲：转化、点评等数据归0，仅得到新品指标，并且把销量和点评数据从一条产品ID转移到另一条也被禁止了，所以不建议老产品换马甲
