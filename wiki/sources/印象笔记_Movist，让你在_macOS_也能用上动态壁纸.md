# Movist，让你在 macOS 也能用上动态壁纸

---

写在前面
----

在 Steam 上面有一款名叫 [Wallpaper Engine](http://store.steampowered.com/app/431960/Wallpaper_Engine/) 的软件非常厉害，可以把桌面设置成各种各样的动态壁纸，不仅如此，甚至还可以交互，创意工坊里面也有大量好玩的创意。售价也非常的良心。

无奈 Wallpaper Engine 并不支持 Mac。但是，如果只是想要动态壁纸，并不是想要把游戏当作桌面的话，这里可以提供一个方法。

原理及方案选择

在 Windows 上无论是 Wallpaper Engine 也好，还是火萤也罢，动态壁纸本质就是视频播放（Wallpaper Engine 支持很多更高级的功能，这里就忽略掉），那么，我们可不可以用播放器把一段视频置底循环播放呢？

少罗嗦，先看东西：

![](.evernote-content/9DA6ECEF-74EE-49A4-88D9-B8B5F0CDA387/3262A1A1-9C14-457F-AAC5-369408A7DA1C.gif)

Saber

![](.evernote-content/9DA6ECEF-74EE-49A4-88D9-B8B5F0CDA387/69817E35-FF46-4AD2-9EFE-397978802D8A.gif)灵梦

P.S. 并不是我桌面什么都没有，只是太渣乱我隐藏掉了。

P.S.2 实际效果很流畅的，卡顿或者顿挫感只是在制作 GIF 之后帧数不足导致的。

如果你对这个样子还满意的话，咱们接着往下看。

实现方法就是使用 Movist 这款强大的播放器实现的，这不是它的主要功能，只是附加功能。它是收费软件，在 App Store 中卖 30CNY，如果你用电脑看视频还算多的话，那么购买这个播放器还是非常划算的。

操作方法
----

首先用 Movist 打开你想要设为壁纸的视频，如果不想要声音，可以如下方法关掉音轨即可（如果界面不一样别慌，只是我换了主题罢了，该有的每个主题都有，仔细看看）

![](.evernote-content/9DA6ECEF-74EE-49A4-88D9-B8B5F0CDA387/4181AB28-E9BD-406F-8710-CFF1572C7F08.png)

关闭音轨

然后在菜单栏找到视频，勾选桌面背景即可（或者直接 command＋D）。

![](.evernote-content/9DA6ECEF-74EE-49A4-88D9-B8B5F0CDA387/E8FC316D-4939-46CE-A0F2-D8A6AA723C5D.png)桌面背景模式

好了，接下来就可以去干别的了。

尾巴
--

其实这个功能也是我无意发现的＝ ＝，好像没什么人提到过，就写写发出来了。当然在 Mac 上面还有其他一些方法，只是我这个使用起来很简单。我平时有习惯开着演唱会听歌工作，这个功能算是一种更优雅的解决方案吧，听到有意思的地方，直接切到桌面瞟一眼。如果你只是想要动态桌面的话，打开循环就好。

那么，下次见。
