# ESBA广州、上海城市赛，九位MVP强势来袭

---

ESBA广州、上海城市赛，九位MVP强势来袭
----------------------

在全国开赛的 东方启明星

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/158DD357-51AE-4966-9671-248A3219B31B.gif)![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/C36D8CC5-5F4A-47FA-A923-999D5AFFF3CD.gif)

近日，2019-20赛季ESBA全国篮球联赛冬季联赛，广州城市赛&上海城市赛相继打响！此次城市赛依然依据各组别设置小组奖及MVP个人奖。

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/E012955E-81D5-42DE-98C0-961FB8E9A72E.jpg)

此次广州赛参赛队伍46支队伍363人，本次也是深圳U8队伍首次来广州参加比赛，且特制了深圳组的奖杯。最终获得U6、U8、U10、U12组MVP分别是：

|  |  |
| --- | --- |
| U06MVP 陈家誉 | U08MVP 简嘉浩 |
| U10MVP 谢朗臣 | U12MVP 黄继文 |

年龄最小的U6组别，获得前三名的分别是：

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/2FE39CB0-FCEE-4D40-A9EE-E4AA6C76667C.jpg)

?亚军：三中校区   带队教练：林汉东

虽然是ESBA的兴趣组，在比赛中可谓是气场全开。

躲过左隔右挡，这位24号小球员才终于与队友汇合，为自己争得了一点进攻机会。对方球员整个身体都尽力限制了24号的动作，尤其是身着黑衣的8号球员，虽然是女孩子耐力一点都没有落下，紧追全场。

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/69F33AE5-F9D6-49C4-BE5C-31361C8D1EE6.jpg)

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/CD6713B0-7F52-44AA-8C33-601F65D4464D.jpg)![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/17D328C5-D501-4D3E-A417-BC6B79A86508.jpg)

蹦起来投、飞起来投，击掌校区在面对不同对手时依旧保持了投篮的最佳状态，一路高歌猛进，表现极佳的9号和17号球员疯狂得分，刚开始紧张的气氛逐渐变得有节奏，最终也夺得了U6组的冠军。

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/D2546ACD-FAEB-40E0-9E60-DDC66EFD7041.jpg)

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/5EB177F9-387D-407A-82CC-92A7A30F51D0.jpg)

获得胜利后开心的宝贝们以及表面装作很严肃的教练。

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/149B16E9-46EB-4460-B21F-026794931BC9.jpg)

由于东方启明星一直都鼓励多积攒比赛经验，所以U8组别的小球员虽然与兴趣小组相邻，多数却已是赛场上的“老手”了。这一对抗起来就明显“来劲”了。

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/84538F3A-CCC1-4B92-94E6-36FE8EE90BD8.jpg)

?冠军：骏景校区   带队教练：伍梓杰

?亚军：华师校区   带队教练：朱俊昌

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/85809C69-3350-4799-839F-7FAA54215E12.jpg)

第一轮刚开始，不过是发球却已困难重重，想把球传出去，面前却屹立着一面“墙”。

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/BC08F984-302D-455D-9BF8-53C990CC9099.jpg)

好不容易逮住机会全力投篮，一起跳却被对手“盖”个正着。在球场最痛苦的事情大概就是一个完美的起跳投篮，被比你跳得更高的对手一把拦下。

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/0D1F7BDD-D765-401C-A901-F6931E062AC1.jpg)

而那些成功投篮的小球员呢？身边也都堆满了防守，家长紧张得直呼“太难了！”

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/D028432F-E786-471D-A301-0849E301330A.jpg)

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/E8B0F1EC-9D5D-4D6A-9B92-812B55C952E2.jpg)

不过，若说最难怕是这位“最佳队友”了。身着黑衣的小球员，一己之力阻挡三名对方球员，为自己的队友争取进攻机会。可见团队的力量与精神，就是在这一场一场的实战中磨砺出来的。

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/E31110C1-CDE0-46BF-A167-9B3E8FBC282B.jpg)

U10组别的小伙伴上场便多了一点大哥哥的气质，又透着一点顽皮。没有那么硬碰硬显然在技巧上多了一点灵活。U10组前三名：

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/7BF9CAAB-A5AC-499C-9A94-41AD96D26157.jpg)

?冠军：金碧校区   带队教练：邓油业

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/61749791-3614-498A-94BD-5EC85B847333.jpg)

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/531A984D-9306-4EBB-896E-9FB81E257D19.jpg)

?季军：时代外滩校区   带队教练：李燊标

拿投篮动作来说，U10组的假动作明显多一些，进攻方总是变着样儿的投篮、传球让对方的防守摸不着头脑。明明是要投篮的动作，却用手指控制住球，转手传了出去，让对方一个措手不及。

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/097B2A4E-F3B6-4814-8D4B-6D77A5FAC37A.jpg)![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/E075B08F-651D-45F4-A12B-7143BC236F96.jpg)

但是光有技巧显然是不够的，实力较量还是得面对面碰一碰，两队球员当仁不让，这个强行突破十分nice！

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/309E1A36-C818-4098-A1CF-CE0B95F371D6.jpg)

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/BF4165CB-FF06-4019-BA2C-DC18B594339D.jpg)

这个盖帽，这个投篮，在U12的场上每个球员的动作都明显像加了快进，时不时的超远投，都是对力量和技巧上的展示。

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/5BC5BD0B-DEFD-4703-8483-0CB947400B63.jpg)

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/496E2412-4D19-4E84-B10D-B1CDD0B74096.jpg)

小小的5号球，在不大的手上传过，虽然是兴趣组却一样把篮球打得火热。拼抢、远投，看到小球员们用尽全力跳起，不惧对手的阻拦。

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/3BB9AFD6-4422-4354-8C8F-73E0D1B0B054.jpg)

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/558FFB3D-A3F5-4C5A-99EF-028B0A3B9A5E.jpg)

前两场有些紧张，双方比分紧咬不放，现场球鞋摩擦地面的声音、教练紧张地指挥声、孩子们的现场指挥、还有家长的加油声沸腾不断，终于第三场迎来了翻盘局，家长们都围着场边丝毫不敢放松。

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/B0586F3C-0458-4F6D-B51F-58F7A6CC4CE1.jpg)

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/5F14BE8A-8444-4D93-9EC1-7C207C6DE430.jpg)

特别有趣的是比赛现场的一对小姐妹，一远一近相互配合，可让对面的男孩子摸不到头脑，可见女孩子默契的可爱之处。

![](.evernote-content/574BB7E1-0870-41B9-BA8B-7AB92B9DC025/EE1B4176-E1C7-444A-B455-911978BB1962.jpg)

U10组别冠军：七位，亚军：三林队；U12组别冠军：王者之鹰队，亚军浦育联队；U15组别冠军：七位数队。

大孩子的对抗就明显更激烈一些，双方起跳高度不分上下，这一球能进与否，连裁判都忍不住停下了步伐。

你以为这就结束了？比投篮更紧张的是抢篮板，比抢篮板更紧张的怕就是连续抢篮板，请欣赏这一段紧张的高光时刻：

投篮猛进一路向前，有失意有顺意，有得到也有失去，汗水与笑声在球场回荡，是赛场也是青春。

最好的年华里，为热爱而倾尽努力。为每一个热爱篮球的球员，提供更多的实战机会。

支持你的热爱，也要呵护你的成长。
