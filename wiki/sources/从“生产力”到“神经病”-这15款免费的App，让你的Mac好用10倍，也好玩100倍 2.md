准备好了吗？让我们从“极致效率”开始，一路滑向“极致快乐”。

01 AppLite：告别“流氓软件”的最后净土
------------------------

你在 Mac 上怎么装软件？大部分新手的噩梦始于此：App Store 里的软件少得可怜，还都要钱。

很多技术大牛会推荐 Homebrew，那是 Mac 软件管理的终极神器。但当你打开它，看到那黑漆漆的终端窗口和一行行代码时，恐怕只想直接关机。

**AppLite 就是你的救星，它是给普通人用的 Homebrew。**![](.evernote-content/D8F356A1-9986-447D-866B-33D1DCAB3313/36F924E7-07AA-4BE6-A854-3DA62793873B.png)它给冰冷的代码穿上了一层现代化的 GUI（图形界面）。它的体验就像逛超市一样：

* **搜索即得**：想装什么，搜一下，点个“下载”，全自动安装。
* **干净纯粹**：这里没有捆绑，没有广告，只有纯净的官方正版软件。
* **一键管理**：它能统一管理你电脑里的软件，不仅能一键更新所有应用，还能在卸载时帮你把那些藏在系统深处的残留文件连根拔起。

02 腾讯柠檬清理：Mac 上的“全能管家”
----------------------

从 Windows 转过来的用户，最不习惯的一点就是 **“失控感”**。

我的网速现在是多少？内存是不是爆了？风扇转这么响，CPU 是不是烧了？macOS 原生系统崇尚“极简”，把这些信息通通藏了起来。

但在高强度工作时，这种“盲盒”体验让人极其焦虑。 虽然我们常吐槽“企鹅”家的软件臃肿，但这款 Mac 版的柠檬清理（Lemon），却是出奇的“小而美”。![](.evernote-content/D8F356A1-9986-447D-866B-33D1DCAB3313/3AB394A0-B27E-44F0-A493-FF2938265FA5.png)它是专为解决这种焦虑而生的：

1. **状态栏显微镜**：它直接驻扎在顶部状态栏，实时显示网速、CPU 温度和风扇转速。你的电脑状态，抬眼便知。
2. **隐私粉碎机**：浏览器留下的痕迹、系统产生的垃圾，一键扫描清理。
3. **卸载监工**：这是最爽的功能。当你把软件拖进垃圾篓时，它会自动跳出来提示：“检测到残留文件，是否清理？”

它就像一个隐形的管家，平时不打扰，关键时刻帮你把家打扫得干干净净。有了它，你的 Mac 才能始终保持“刚出厂”般的清爽。

03 LocalSend：打破苹果的“围墙花园”
------------------------

AirDrop (隔空投送) 确实好用，被誉为苹果生态的护城河。

但如果你是“双机党”呢？一旦你想把 Mac 里的 4K 视频素材传给安卓手机，或者发给 Windows 电脑上的同事，瞬间抓瞎。用微信？画质被压缩成渣，超过 1GB 的文件直接拒收。用网盘？上传下载都要等，如果不买会员，那几十 KB 的速度能让你怀疑人生。

**LocalSend 是真正的跨平台之光，是打通生态壁垒的锤子。**![](.evernote-content/D8F356A1-9986-447D-866B-33D1DCAB3313/92A81793-DE9E-414A-A33C-C363130540AB.png)

* **真正的全平台**：Mac、Windows、iOS、安卓、Linux，众生平等，想怎么传就怎么传。
* **极致的隐私**：不需要联网，不需要注册账号，不需要上传到任何服务器。所有数据只在你的局域网内点对点传输。
* **满速狂飙**：速度仅取决于你的路由器，跑满带宽是常态，且绝不压缩画质。

只要在同一个 Wi-Fi 下，设备间传文件就像左手倒右手一样自然。从此，你再也不用为了传个文件，在这个 App 和那个 App 之间反复横跳了。

04 IINA：把“现代影院”装进电脑
-------------------

macOS 自带的 QuickTime Player，界面设计确实优雅，符合苹果的审美。但作为播放器，它太“娇气”了：下载的高清电影是 MKV 格式？打不开。想挂载个中文字幕？找不到入口。想倍速追剧？操作繁琐得让人想放弃。

**IINA 被称为 macOS 上最好的现代播放器，没有之一。**![](.evernote-content/D8F356A1-9986-447D-866B-33D1DCAB3313/E373CB11-9527-4E79-B13F-CA7451B8C342.png)它生于开源，长于颜值，强于内核。

* **通吃一切**：几乎没有它解不了的格式，把它设为默认播放器，你的世界就清净了。
* **在线字幕**：看生肉资源也不怕，它内置了字幕搜索功能，一键匹配中文字幕。
* **交互之王**：它完美适配了 MacBook 的触控板。双指滑动调节进度，双指缩放调节画面大小，甚至支持画中画模式，让你一边写文档一边摸鱼看球。

好用的工具就像空气，你感觉不到它的存在，但离了它你就无法呼吸。当你习惯了 IINA 的丝滑，你会惊讶地发现，原来的 QuickTime 真的只能算是个“图标预览器”。

05 The Unarchiver：解压缩，本该如此简单
----------------------------

这是每一个职场新人的“社死”时刻：老板发来一个 RAR 格式的加急文件，你自信满满地双击，结果 Mac 毫无反应——因为它不支持。或者，你把 Mac 压缩的 ZIP 发给用 Windows 的客户，对方打开一看，文件名全是乱码，以为你发了病毒。别让系统的“傲慢”，成为你沟通的壁垒。 The Unarchiver 就是为了解决这个问题而生的。![](.evernote-content/D8F356A1-9986-447D-866B-33D1DCAB3313/F10C873A-2648-4162-AC9B-BA94137579E4.png)它的设计哲学极度克制：安装，然后忘记它。你不需要打开它的主界面，不需要学习怎么用。当你装上它的那一刻，问题就已经解决了。

* **格式全能**：RAR, 7z, Tar, Zip... 只要是压缩包，双击就能解压。
* **乱码终结者**：它会自动识别文件名编码，完美解决了 Windows 和 Mac 之间互传压缩包的乱码顽疾。

06 Maccy：给剪贴板装上“海马体”
--------------------

很多人每天 90% 的工作，本质上就是 Ctrl+C 和 Ctrl+V。 但 macOS 的原生剪贴板像金鱼一样，只有 7 秒记忆——复制了新的，旧的就丢了。 你一定经历过这种崩溃时刻：**辛辛苦苦复制了一段文案，结果手滑又复制了一个无关紧要的网址，上一条内容瞬间灰飞烟灭。**

市面上有收费昂贵的 Paste，界面华丽但略显臃肿。**对于追求极致效率的人来说，开源免费的 Maccy 才是终极答案。**![](.evernote-content/D8F356A1-9986-447D-866B-33D1DCAB3313/918A2804-382A-40BA-BF51-48AE7D8149E7.png)它没有任何花哨的动画，只有极致的速度：

* **瞬间呼出**：按下 `Command+Shift+C`，它会在鼠标旁瞬间弹出一个极简列表，列出你过去复制过的所有文字、图片、甚至是文件路径。
* **搜索即得**：不需要在历史记录里翻找，直接打字就能模糊搜索。早上复制的会议链接，下午三秒钟就能找回来。
* **格式净化**：从网页复制的文字带着乱七八糟的粗体和背景色？Maccy 支持直接粘贴为纯文本，瞬间治愈排版强迫症。

它是你大脑的延伸，给了你无限次“后悔”的机会，让你不再为误操作而懊恼。

07 Mac Mouse Fix：让几十块的鼠标“起飞”
----------------------------

当你给 Mac 插上普通鼠标，第一感觉往往是：**难用**。 滚动生涩、没有惯性、滚轮方向反直觉……苹果似乎在用这种糟糕的体验，逼你买那一千块的妙控板（Magic Trackpad）。

**Mac Mouse Fix 就是来“打脸”的，它能让几十块钱的普通鼠标瞬间“起飞”。**![](.evernote-content/D8F356A1-9986-447D-866B-33D1DCAB3313/F0634F5C-64C1-4843-B01E-10708B55027D.png)

* **拟真触控**：它能让普通滚轮拥有触控板般丝滑流畅的惯性滚动，甚至还有触顶/触底时的回弹动画（Duang 的一下），手感极佳。
* **超级中键**：它解放了鼠标中键。按住中键移动鼠标，就能触发“任务视图”或“左右切换桌面”，完全复刻触控板的手势操作。
* **良心免费**：最感人的是，作者因为支付接口问题，直接宣布让中国区用户**免费使用**。

不需要昂贵的硬件，只需要优秀的软件，体验就能发生质的飞跃。

08 ICE：治愈强迫症的“收纳师”
------------------

随着软件越装越多，你的菜单栏是不是已经挤满了各种图标？ 尤其是新款 MacBook 的“刘海屏”用户，图标稍微多一点就被刘海“吃”掉了，想点都点不到，这种“视而不见”的焦虑感非常搞心态。

收费的 Bartender 很强，但 **ICE 是完美的免费平替，它是你菜单栏的“极简收纳师”。**![](.evernote-content/D8F356A1-9986-447D-866B-33D1DCAB3313/2F6CABBC-8894-48D8-9AD7-236BE0AB515A.png)它引入了巧妙的分区管理逻辑：

* **常用区**：Wi-Fi、电量等核心信息，始终显示。
* **隐藏区**：微信、输入法等后台软件，折叠收起。鼠标移过去，自动展开；移走，自动收起。
* **永久隐藏区**：像 Mac Mouse Fix 这种配置一次就不再动的软件，直接彻底隐藏，眼不见心不烦。

对于刘海屏用户，它甚至能在菜单栏下方单独显示一行图标栏。 

**整洁的桌面，是清晰思路的开始。** 把干扰项藏起来，把专注力还给自己。

09 AirBattery：上帝视角的电量管家
-----------------------

作为苹果全家桶用户，最焦虑的时刻莫过于： 正准备开会，发现 AirPods 没电了； 正写稿写得嗨，iPad 或妙控键盘突然弹窗电量不足。 以前你得把设备一个个拿起来看，或者在系统里点好几层菜单才能确认电量。

**AirBattery 把这种信息直接“喂”到你嘴边。**![](.evernote-content/D8F356A1-9986-447D-866B-33D1DCAB3313/385E45DA-3A30-4E37-8466-E437CFF06FB2.png)它在 Mac 的菜单栏或小组件里，实时显示你所有苹果设备的电量。 

iPhone、iPad、AirPods、Apple Watch，甚至是妙控鼠标和键盘，所有设备的剩余电量一目了然。

**真正的掌控感，就是不需要你刻意去查，信息就在那里。**

它是治疗“电量焦虑症”的一剂良药，让你永远告别设备突然罢工的尴尬。

10-11 Loop + AltTab：把 Windows 的灵魂“偷”过来
--------------------------------------

这其实是一套**“窗口管理组合拳”**。 平心而论，Mac 的窗口管理真的很弱：没有拖拽分屏，切换窗口没有预览图。这方面，Windows 确实遥遥领先。

既然打不过，那就加入（抄过来）。

* **Loop**：呼出一个优雅的圆环菜单，或者直接把窗口拖到屏幕边缘，瞬间实现 1/2、1/4 分屏。再也不用把鼠标移到那个绿色的缩放按钮上傻等了，你的屏幕空间从此井井有条。![](.evernote-content/D8F356A1-9986-447D-866B-33D1DCAB3313/6A08BE6D-4986-469B-AB22-6AAF5A2545A4.png)
* **AltTab**：它把 Windows 最经典的 `Alt+Tab` 带到了 Mac 上。原生 Mac 的 `Command+Tab` 只能切换软件，不能切换具体窗口。而 AltTab 不仅能切换窗口，还能看到高清的**实时预览图**，而不是一个个冰冷的图标。![](.evernote-content/D8F356A1-9986-447D-866B-33D1DCAB3313/380287F3-C9BE-47F9-AFFF-717F790DB103.png)

12 RunCat：在这里，你可以合法“虐猫”
-----------------------

想知道你的电脑卡不卡，通常得打开活动监视器，看那堆看不懂的 CPU 曲线和内存数据。哪怕看懂了，也只是一堆冰冷的数字。  
RunCat 把这些枯燥的数据，变成了一只奔跑的猫。它住在你的菜单栏上，是你最忠实的系统监视员。你的 CPU 占用越低，它就慵懒地散步，甚至看起来想睡一觉；![](.evernote-content/D8F356A1-9986-447D-866B-33D1DCAB3313/1D9943E8-C1A4-402A-9D79-3E04C2C93721.png)一旦你开了几十个 Chrome 网页，或者在渲染 4K 视频，CPU 负载飙升，它就会开始疯狂冲刺，那频率快到让你担心它下一秒就会猝死。看着它跑断腿的样子，你甚至会生出一种变态的快感：“跑！给我跑！这就是资本家的感觉吗？”  
除了这只经典的猫，它还内置了“动物园”：

* **疯狂的鹦鹉（Party Parrot）**：随着 CPU 升高疯狂甩头；
* **仰卧起坐的小人**：CPU 越忙，他练得越猛；
* 甚至还有在火锅里翻滚的肉片……

12 Daktilo：把 MacBook 变成复古打字机
----------------------------

现代 MacBook 的蝴蝶键盘或者剪刀脚键盘，虽然静音高效，但总少了点“创作的仪式感”。尤其是当你文思枯竭，或者写不出方案的时候，那种轻飘飘的手感让人更烦躁。

**Daktilo 能让你的 Mac 瞬间穿越回 1980 年代，找回海明威写作时的感觉。**![](.evernote-content/D8F356A1-9986-447D-866B-33D1DCAB3313/465D6B53-483A-4093-ABCA-F92AE83E6B9B.png)它模拟了老式机械打字机的全套声效：

* **敲击声**：每一次按下字母键，都是清脆、有厚度的机械撞击声，仿佛指尖敲打的不是塑料，而是铸铁。
* **回车声**：最灵魂的是每一次换行，它会发出那声经典的“叮——”！那种“完成了一行”的成就感瞬间拉满。

这哪是在写枯燥的周报，这分明是在写传世名著。这种听觉反馈极其解压，让你觉得每一个字都掷地有声。 

哪怕只是在跟老板在钉钉上吵架，打出来的字都显得更有气势，仿佛你在用打字机敲击出一份宣战书。

13 Eyeballs：你永远不孤单
------------------

独自加班的深夜，办公室空无一人，会不会觉得屏幕冷冰冰的？Eyeballs 送你一双时刻关注你的“眼睛”。这双大眼睛会趴在菜单栏或屏幕角落，死死地盯着你的鼠标指针。![](.evernote-content/D8F356A1-9986-447D-866B-33D1DCAB3313/55A0CC22-C0AA-4D80-9EC7-AEF21DA4BF5C.png)它的视线完全跟随你的操作：

* 鼠标移到左边，它就向左看；
* 鼠标移到右边，它就向右看；
* 鼠标停在中间不动，它还会和你对视，甚至眨眼卖萌，或者做出斗鸡眼的搞怪表情。

它没有任何实际功能，既不监控数据也不提醒消息，主打一个 **“陪伴”**。

它就像一个住在屏幕里的电子宠物，虽然不会说话，但它的目光始终追随着你。在这个虚拟的世界里，至少还有一双眼睛，满眼都是你。对于社恐来说，这或许是比真人更完美的陪伴。

14 Ball：纯粹的快乐，只需要一个球
--------------------

如果前面几款还带点功能性或观赏性，那最后这款 **Ball**，就是纯粹的**“行为艺术”**，是 Mac 里的指尖陀螺。![](.evernote-content/D8F356A1-9986-447D-866B-33D1DCAB3313/3EA853E9-4A67-44FE-98EE-07DCAD41A888.png)它就是一个红色的球，拥有真实的物理引擎。

* **重力感应**：你可以用鼠标拖着它到处甩，把它狠狠地扔到屏幕边缘，它会像真球一样弹回来。
* **发泄解压**：你可以把它挂在菜单栏上晃悠，也可以让它受重力影响自由落体。
* **无脑发呆**：当你大脑宕机时，只需要拨弄两下这个球，看它在屏幕框里弹来弹去。

它有什么意义？它没有任何意义。

但在成年人的世界里，“没有意义”本身就是一种顶级的奢侈品。我们做任何事都要讲究 ROI（投入产出比），都要讲究复盘。

唯独在玩这个球的时候，你不需要对任何人负责，不需要产出任何价值。当你被复杂的报表搞得头昏脑涨时，玩两分钟球，大脑瞬间重启。 

快乐，有时候就是这么简单且枯燥。

---

写到最后，突然发现这 15 款软件其实构成了我们生活的两个侧面。

前面的 AppLite、LocalSend 甚至 Maccy，是为了对抗系统的“傲慢”与“固执”。它们填补了原生 macOS 和我们实际需求之间的那道鸿沟，帮我们从繁琐的操作里把时间抢回来 。

而后面的 RunCat、Eyeballs 和那个红球，则是对“极致效率主义”的一次小小反叛。

我们总是试图把电脑变成最锋利的生产力工具，装满了各种 GTD 软件，讲究 ROI，讲究复盘，却忘了它也可以只是一个简单的玩具 。

**在这个时刻紧绷的世界里，“没有意义”本身就是一种顶级的奢侈品 。**

你可以用 Lemon 清理垃圾，也可以看着 CPU 上的猫疯狂奔跑；你可以用 IINA 倍速刷课，也可以拨弄那个红球发呆两分钟 。

这些软件或许不能直接帮你升职加薪，但如果它们能在你焦头烂额的工作间隙，博你一笑，或者帮你在这个赛博世界里找回一点点“掌控感”，那就足够了 。

毕竟，我们努力提高生产力，是为了更快地搞定工作，好腾出时间去好好生活，而不是把生活也变成工作 。

给自己的 Mac 装点“没用”的东西吧，毕竟，谁还不是个长不大的孩子呢 ？
