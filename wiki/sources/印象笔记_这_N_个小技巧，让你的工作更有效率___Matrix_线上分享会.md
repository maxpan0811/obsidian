我提倡用快捷键，一来因为它不需要鼠标指针定位，按下正确的键就会百分百生效；二来比起抬手去操作鼠标或触摸板，用快捷键会和打字的状态之间切换起来更无缝。我提倡记住那些日常最常用到的、对自己的使用习惯有最大改善的一批键位，而不用强行去记大量的快捷键。

运营岗事情杂是众所周知的，又加上是在创业团队，那么一般要参与的事就会比较多。我经常觉得难以抓住主线，做事容易被打断，注意力溃散。

上一年年尾看一些时间管理的资料，有提倡记录自己在每件事上面花费了多少时间，来培养时间感。那时候恰好买了 [OmniOutliner](http://sspai.com/tag/OmniOutliner)，它可以用快捷键自动在文中插入时间戳，我就开始随手记录每个工作时段我都在做什么，必要的时候简短附上完成情况或者历史遗留问题。

这个记录的过程会一直提示我眼下的主要任务，不被分散。每天这个文档未完成的时候，看起来会像是待办事项，到了下班，它就成了一份简短的总结。做运营必须重视数据，这记录有点像是我个人的数据分析「后台」，一天或者一周结束的时候，我可以很明晰地查看事情进度和自己的工作状态，以便调整。

最早我用印象笔记做这样的记录，后来买了 OmniOutliner ，转移到上面去了，它好处在于更加结构化，而且容易调整笔记顺序。实际上我觉得工具不是重点，时间记录、任务管理有很多自动化的的工具，Yves 最近还在用一个设计感很强的个人数据收集统计服务 [Gyroscope](https://gyrosco.pe)，我选择了最原始的方式，反而因为都是我亲手记录的而更大程度影响了我的行动。

我来说说我以前提高自己效率的方法：

1. 逼自己习惯一种新的工作方式，最简单的是用日历和提醒事项把要做的事情都整理好，然后严格执行。
2. 在习惯了日程规划以后，提前把要做的事情的细节整理出来。比如用 OmniFocus 代替提醒事项，像记录会议一样，用 123、笔记之类的把一件事分成几个小事情，日历要一直用。
3. 采用个性化工作方式，比如用番茄土豆工作法让自己思想集中。我反对所有带有 Todo 功能的番茄土豆工作法 App，有 Todo 还要 OmniFocus 干嘛？集中精神力的时候还要看手机了解任务干嘛？在执行番茄钟前就要了解任务了呀。

相同功能的 App 不要重复，也不要重叠，这种情况是最理想的。日历就只用一个日历 App，任务打勾勾就只用提醒事项或者 OmniFocus，不同的 App 跳来跳去反而浪费时间。

[相关文章](http://sspai.com/tag/TextExpander)

文字输入的效率往往体现在键盘的敲击次数上。比如说此前我们介绍过的双拼输入法，所以字都可以用按两次按键打完，相对全拼提高了不少输入效率。除了输入法之外，还有一种情况是重复文本。回想一下日常的输入情况，有些短语甚至是句子可能你一天都要打很多次，如果按两下键盘就能快速地打出一句话来，那这输入效率自然也是提升了很多，对于这种重复长文本的输入，我通常是用 [TextExpander](http://sspai.com/26513) 解决的。

TextExpander 简单来说有点像 iPhone 和 Mac 自带的用户词典，对于经常用到的句子或者短语，可以提前设定好对应的缩略词，以后每次只要在键盘上按两三个键就可以输入一段完整的话了。不过和用户词典相比，它的功能就强大多了，不但有短语分组功能，而且还支持富文本，插入图片和设置变量，灵活性要高很多。

我工作的时候经常要发很多邮件，很多时候邮件的内容都是模板化的。我会在 TextExpander 里面提前设置好邮件的格式，并且在应用中预留出需要手动填写的部分。之后每次发邮件，只要输入快捷键就可以调用邮件模版，完善好内容之后便可插入相应的程序中，连格式都不用改，直接就能发送了，发一封邮件可能都不到一分钟。当然除了发邮件它也可以用在类似 Markdown 写作之类的很多地方，就看你的个人需求啦。

不过如果你还没有用用户词典的习惯，推荐先去尝试一下，因为 TextExpander 是收费的，所以希望大家可以在确定自己真的有类似的需求之后再考虑要不要买。其实我个人比较倾向于在弄清楚自己的需求之后，再考虑找对应的应用去解决问题，我不太会先买应用再去考虑需求。

**关联阅读：**[《利用 TextExpander 提高在 Mac 上撰写 Markdown 的效率》](http://sspai.com/27479)

### 2. 双拼和注音输入法

*分享者：SamWanng*

懒得背五笔字根又烦于全拼的拖沓，只能选择了「双拼」这个折中的方案。

双拼的学习曲线还算平缓，只要小学拼音学的不错，记忆一下键盘对应的声韵母稍加练习，待你熟练掌握之后你就会觉得为什么没早一点学习双拼。

汉字在全拼平均码长 3.13，而双拼是 2，这一点就可以提高 50% 效率。双拼的优点需要切身体会，绝对不是少敲几个键那么简单。节奏感和整齐性的吸引力一般只有使用了才能充分体会认知。

当然如果有精力和时间，你可以尝试一下注音输入法，无论是输入速度还是选词的准确性都要强于双拼。
