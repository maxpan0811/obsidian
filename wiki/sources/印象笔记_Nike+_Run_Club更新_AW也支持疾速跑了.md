# Nike+ Run Club更新 AW也支持疾速跑了

---

![](.evernote-content/9E04B523-2B6E-449A-B40F-E0051FF4B1CE/2B90C48E-117F-4421-9BF2-CEE40F6890FE.jpg)

威锋网 6 月 20 日消息，今天 Nike+ Run Club 更新，为 Apple Watch 带来全新的“疾速跑（Speed Run）”功能，用户无需 iPhone 即可使用这个新的功能。

疾速跑此前是 Nike+ Run Club（NRC）在 Apple Watch 上的一种跑步类型，但却不可用，它会提示用户可以使用相应的 iPhone 应用来替代，如今更新之后你只需在手表上点击开始就可以追踪你的跑步了。

与 Apple Watch 的 Workout 应用相似，用户可以在跑步过程中双击主屏幕来标记间隔。另外，更新还带来一些针对其他错误的修复和功能改进。

本次更新的是 5.7 版本，具体如下：

Apple Watch——疾速跑上线。 现在用户可以使用手表上的疾速跑功能标记间隔和圈数。特别提示：在跑步过程中双击主屏幕可以标记间隔。

重要注意事项：用户需要先授权 NRC 访问运动和健身活动，然后才能对跑步进行追踪。

为实现理想的 Apple Watch 版使用效果，建议安装使用 watchOS 3.2.2 或更高版本。

iPhone——奖杯和成就徽章回归。解锁新成就，庆祝你取得的成绩，助你保持运动激情。

轻松查找朋友！ 点击 Club 选项卡中的 添加朋友图标，体验升级功能。

其他错误修复及功能改进。
