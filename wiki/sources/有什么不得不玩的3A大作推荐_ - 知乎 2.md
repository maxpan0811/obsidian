**《黑神话：悟空》**
------------

顶级的画面表现与游戏性，瑕不掩瑜的国单启程之做。

按照惯例，我要先说这款游戏的缺点。

其一，是大家都有提到的空气墙，这点真的没法洗，纵然我游戏玩得多，对线性游戏的空气墙绝大多数情况下都能做到忽视（就是遇到空气墙我就会明白此路不通，能瞬间理解，而不会有更多的感受），但《黑神话：悟空》的空气墙确实多到了能让我感受到的程度，以我对空气墙的极不敏感，在一周目未全探索的情况下都能对几处空气墙留有印象，这确是《黑神话：悟空》所存在的痛点。

其二，没有HDR，这是影响我体验的最大点，我认为《黑神话：悟空》没有HDR的主要原因是lumen加了HDR或会使全景光追像个小丑，因为就算是超高的全景追踪，体验下来也就比[lumen![](data:image/svg+xml,%3csvg xmlns='http://www.w3.org/2000/svg' width='10px' height='10px' viewBox='0 0 16 16' class='ZDI ZDI--FourPointedStar16 css-1dvsrp js-evernote-checked' fill='currentColor' data-evernote-id='645'%3e%3cpath d='m5.068 9.267-3.08-.77a.512.512 0 0 1 0-.994l3.08-.77a2.289 2.289 0 0 0 1.665-1.665l.77-3.08a.512.512 0 0 1 .994 0l.77 3.08c.205.82.845 1.46 1.665 1.665l3.08.77a.512.512 0 0 1 0 .994l-3.08.77a2.29 2.29 0 0 0-1.665 1.665l-.77 3.08a.512.512 0 0 1-.994 0l-.77-3.08a2.289 2.289 0 0 0-1.665-1.665Z'%3e%3c/path%3e%3c/svg%3e)](https://zhida.zhihu.com/search?content_id=362792871&content_type=Answer&match_order=2&q=lumen&zhida_source=entity)的全局光照亮那么一点，除此之外再无很大区别，该有的反射lumen有，该有的氛围感lumen也有，甚至中档全景追踪还会起到负面提升，镜面反射都糊成一团，简评就是，超高值得一开，但中低还不如lumen。回到HDR，如果你的屏幕支持HDR，而且亮度还不错，又或者一些赋歌用的miniLED或者OLED 显示器or电视，HDR能在不损耗性能的情况下极大的提升观感（色彩明度和更好的动态范围），HDR应当是当今游戏大作的标配，特别是《黑神话：悟空》在画面上下了极大功夫的背景下，没有HDR着实让人无法理解。

其三，UE5的画面解算算法存在bug，部分过场明显能看到不在画面中心的贴图未加载，期待后期修复。

接下来是优点。

其一，顶级的美术。“分不清，真的分不清。”放在某些游戏或许是嘲讽，但放到《黑神话：悟空》上绝对毫无违和感。UE5可怕的画面表现力在游戏科学的调教下，着实给了我一场长达五十小时的视觉盛宴，从[黑风山![](data:image/svg+xml,%3csvg xmlns='http://www.w3.org/2000/svg' width='10px' height='10px' viewBox='0 0 16 16' class='ZDI ZDI--FourPointedStar16 css-1dvsrp js-evernote-checked' fill='currentColor' data-evernote-id='646'%3e%3cpath d='m5.068 9.267-3.08-.77a.512.512 0 0 1 0-.994l3.08-.77a2.289 2.289 0 0 0 1.665-1.665l.77-3.08a.512.512 0 0 1 .994 0l.77 3.08c.205.82.845 1.46 1.665 1.665l3.08.77a.512.512 0 0 1 0 .994l-3.08.77a2.29 2.29 0 0 0-1.665 1.665l-.77 3.08a.512.512 0 0 1-.994 0l-.77-3.08a2.289 2.289 0 0 0-1.665-1.665Z'%3e%3c/path%3e%3c/svg%3e)](https://zhida.zhihu.com/search?content_id=362792871&content_type=Answer&match_order=1&q=%E9%BB%91%E9%A3%8E%E5%B1%B1&zhida_source=entity)的体积光，到小西天的精美建筑，游戏科学在画面上下的功夫着实没得说。仅268的售价带你逛完丛林逛沙漠，逛完沙漠还有雪地、洞穴和火山。线性游戏相较于开放世界来说，因为线路相对固定，所以可以有更考究的构图，也就是一条路走到某个点，你所看到的场景中的树或建筑，在屏幕上所呈现的画面，是可以精心构造的，所以即便没有很宏伟的建筑，《黑神话：悟空》在视觉上所带来震撼依旧强烈，这点在小西天尤为明显（因为我记路能力超强，所以从头到位都没迷过路，diss小西天摸不清路的玩家的痛我体会不到哈哈哈），例如打完[赤尻马猴![](data:image/svg+xml,%3csvg xmlns='http://www.w3.org/2000/svg' width='10px' height='10px' viewBox='0 0 16 16' class='ZDI ZDI--FourPointedStar16 css-1dvsrp js-evernote-checked' fill='currentColor' data-evernote-id='647'%3e%3cpath d='m5.068 9.267-3.08-.77a.512.512 0 0 1 0-.994l3.08-.77a2.289 2.289 0 0 0 1.665-1.665l.77-3.08a.512.512 0 0 1 .994 0l.77 3.08c.205.82.845 1.46 1.665 1.665l3.08.77a.512.512 0 0 1 0 .994l-3.08.77a2.29 2.29 0 0 0-1.665 1.665l-.77 3.08a.512.512 0 0 1-.994 0l-.77-3.08a2.289 2.289 0 0 0-1.665-1.665Z'%3e%3c/path%3e%3c/svg%3e)](https://zhida.zhihu.com/search?content_id=362792871&content_type=Answer&match_order=1&q=%E8%B5%A4%E5%B0%BB%E9%A9%AC%E7%8C%B4&zhida_source=entity)和初见小西天土地的那段路。再例如黑风山的几个土地庙旁边。

截止3.4主线通完，因为对强度需求不高，抽卡需求也不是很强，算是进入长草期，地图上剩下的资源就当作以后的小小惊喜吧。原神的优点十分明显，人物塑造力度很大，毕竟人设的成败会直接影响卡池流水，同样他的缺点也在于这款游戏似乎只塑造了人物，在世界人文方面的功夫尚有欠缺，在t2开放世界游戏中说3.4版本原神是世界塑造最烂的游戏也不为过，他有细节，但不够。3.5 3.6毫无亮点的两个版本，终于在3.7的宵宫任务拉回点口碑，但是核心剧情的几个关键点处理着实糟糕，为来剧情质量画大大的问号。

### **《极限国度》**

自行车、滑雪、滑翔、瘠薄的痛，我是没想到骑自行车也能这么爽，这游戏可以玩的很花，可以在空中单车切滑翔，竞速类的讲真爽就完事儿了，而且不用加速器也能和别人连上，地图里很多其它玩家和玩家影子，挺热闹的。

极限国度真的很爽，第一人称的单车速降，还有雨夜公路赛，这两个真的爆赞。

### 《**刺客信条：奥德赛**》

这款游戏除了开篇那个‘育碧巨献’给我留下点印象，整体游玩下来体验不如起源，在斯巴达刺杀也不好使，该莽还是莽，莽不过就寄（等级差多了踢不动都）。

### 《**刺客信条：[英灵殿](https://zhida.zhihu.com/search?content_id=362792871&content_type=Answer&match_order=1&q=%E8%8B%B1%E7%81%B5%E6%AE%BF&zhida_source=entity)**》

### 《**[看门狗2](https://zhida.zhihu.com/search?content_id=362792871&content_type=Answer&match_order=1&q=%E7%9C%8B%E9%97%A8%E7%8B%972&zhida_source=entity)**》

### 《**看门狗：军团**》

这个游戏能说道的是伦敦街道超真实的画面，在我看来就是为光追做了一款游戏，画面真实，游戏玩法上相对一般，如果是画面党，比较推荐购买。

简评，这作是探员收集+探员暖暖，可惜只有嗨丝没柏丝。

### 《**[幽灵行动](https://zhida.zhihu.com/search?content_id=362792871&content_type=Answer&match_order=1&q=%E5%B9%BD%E7%81%B5%E8%A1%8C%E5%8A%A8&zhida_source=entity)：断点**》

### **《[文明6](https://zhida.zhihu.com/search?content_id=362792871&content_type=Answer&match_order=1&q=%E6%96%87%E6%98%8E6&zhida_source=entity)》**

阿育的纪元我没玩过，但文明6这个游戏真的是三天开一把一把玩三天，玩的很上瘾，而且很多地方都尽可能的展现了国际形势下各种选择可能导致世界格局的变化，非常值得一试的战棋游戏

hitman系列，会有那种，自己设计关卡的感觉，游戏的玩法探索度很高，不过人物动作差评，感觉很僵硬。

### **《逃生系列》《[港诡实录](https://zhida.zhihu.com/search?content_id=362792871&content_type=Answer&match_order=1&q=%E6%B8%AF%E8%AF%A1%E5%AE%9E%E5%BD%95&zhida_source=entity)》**

### **《[超凡双生](https://zhida.zhihu.com/search?content_id=362792871&content_type=Answer&match_order=1&q=%E8%B6%85%E5%87%A1%E5%8F%8C%E7%94%9F&zhida_source=entity)》**

哪怕在几次更新优化后，我仍然认为这是一款剧情乏味，可玩性并不高的游戏作品，画面不符合我的审美，但是不可否认未来感科技感是十足的，在一些小设计上面也是很有用心的，但我仍认为这款游戏在目前三百这个价位不推荐各位脑子一热入手，在后期更新、dlc等情况下，或者有不错的折扣，建议入手。

22 11 10，又打MOD玩了一会，比之前强不少（也许上一次玩还是1.2或者1.3？）他还是有很明显的不足，整体而言是不如巫师三的，但是我很期待他的dlc和后续更新，因为当时赶工有很多东西被阉割掉，这是导致很多人差评的主要原因，现在的2077是一个略显残破的游戏，但我选择相信CDPR。

如果2077和宣发的内容一样，这游戏或许会是这几年同类型游戏的天花板。

2.0版本更新可以说让这款游戏真正完整，达到了对他应有的预期，**但这是不应该的**，如果玩家在首发的时候就能体验到目前这个质量的游戏，那他必然会是神作。

这个游戏很吵，五个人从游戏开始到游戏结束就没闭过嘴，你站着不动也能一直找话说。但这正是游戏优点所在，整体而言能当个喜剧片看，就是剧情后面有点俗套无语，但在喜剧合家欢的大前提下，整体的剧情表现还是很不错的，关键在于，因为角色对话量很大所以每个主要人物塑造得都很立体。战斗方面的安排我倒是不太满意，这个游戏的战斗似乎只是为了告诉玩家：你不是在看一场交互电影，这是个游戏，你要准备打怪了。如果结尾剧情不那么俗套，战斗安排精简一些，则该游戏属T1范畴。

### 《**地狱之刃：塞纳的传说**》

如果说一代的美工夸张，那二代真的是做到了虚实难分的地步，截至目前24年7月，我单方面宣布塞纳2是目前游戏领域最强画质，路径追踪的2077和心灵杀手都比不上。但是这款游戏除了画面、音乐、艺术设计之外，游戏性比一代还要缩水不少，中间的解密相当枯燥，战斗内容相比一代少了不少可玩性（或许是为了更加真实？）不过本作的战斗衔接设计的相当出色，前两场BOSS战的设计也相当出彩，总体而言二代就是在一代的基础上砍了游戏性，但画面、音乐与战斗的融合、真实感都有了明显的进步。目前在XGP，非常值得一试的作品。

6月13日，加在后面。我想这篇回答至此已经没有多少人坚持看完。我已经忘记这篇回答是什么时候写的了，但我能查到我20年9月，组装了人生第一台专属于我自己的电脑，回看这篇回答，通篇除了《原神》再没有一款中国的游戏，这三年多时间，游戏玩了很多，不上一百也有大几十款，但在大作这个领域，或许原神能算其一，除此之外想不出其二，在T0个人极端推荐和T1建议入手两个板块，更是一款没有。这几年时间，我们聊的、推荐的几乎都是欧美、日本的游戏。

想来有趣，黑神话首个PV是20年8月20号，正是我装机前不久，这些年几乎所有消息都有关注，每条PV都看了不知道多少遍，我太想玩到我们自己的大型作品了。而截至今日根据黑神话所爆出的相关信息，我无数次想在玩都没玩的情况下把黑神话加入T0，但终究还是忍住了，其一是确不知其质量，其二是即便是主观榜，也该稍微掺杂点理性。我很期待8月20日游戏放出后（或者过几天放出个demo我也算是有理可依），将这款游戏直接列入T0，向所有看到这篇回答想涉足大型游戏的朋友，**去推荐属于我们自己的游戏作品**。
