# Windows 上的良心国产应用，我们又帮你找了这 5 款

---

提到 Windows 上的国产软件，大家首先想到的是不是某某安全卫士，某某管家和某某全家桶？虽然当下绝大数人使用的国产 Windows 软件中大部分可以用「广告」「捆绑」「会员」这几个用户最讨厌的关键词来形容（虽然在其他系统上也差不多），不过中国的野生程序员们从不缺少技术能力，千千万万家公司中也不乏良心之辈，这就来推荐几个。

Listen1 音乐播放器：听音乐有它就够了
----------------------

无论你平时使用的是哪个在线音乐软件，不管是网易云音乐，QQ 音乐，还是虾米音乐，估计最头疼的就是版权问题，我想听的歌一部分 A 音乐平台有版权而 B 音乐平台没有，另外一部分则是 B 音乐平台有版权而 A 音乐平台没有，难道除了同时装这几个软件以外就没有更加优雅的解决方法了吗？

当然有，[**Listen1**](http://listen1.github.io/listen1/) 就号称 「One for all free music in China」：装一个软件，听所有免费音乐。

Listen1 是一个开源的音乐播放器，可以搜索和播放来自网易云音乐，虾米，QQ 音乐三个主流音乐网站的歌曲，Windows 版本不用安装，解压即用。它拥有所有 Windows 良心软件都具有的特性：界面简洁，没有广告、没有会员、没有捆绑、没有推荐，轻量级占用少反应灵敏，最高效地实现一个功能。

![](.evernote-content/301A66B4-1FD5-45A0-8539-F72E215232F3/132F3074-F51E-4BC5-8F49-794FEE215434.png)

在 Listen1 上只能收听各平台的免费音乐，对于需要付费收听的音乐还是需要使用官方 App 购买后收听。

即使你不想为此安装一个软件也没关系，Listen1 支持包括 Chrome 插件在内的你能想到的所有平台，而且有能力的派友可以直接去 [Github 地址](https://github.com/listen1) 为它锦上添花。

GifCam：最简单获取 Gif 截屏的方式
----------------------

GifCam 属于那种专注于实现一个功能的极简软件，它的功能只有一个，那就是直接获取截屏的 Gif，而不用我们先录屏再从视频转成 Gif。

![](.evernote-content/301A66B4-1FD5-45A0-8539-F72E215232F3/66737B51-D94A-46E6-8F51-FD649CD72C3D.png)

无论从 1.47M 的体积还是那只有四个按钮的界面都能体现出 GifCam 的极简，不过在保持这样子体积的同时它专注的功能却做得非常好，随意取景（当然也支持全屏，藏在了 Rec 下拉菜单中）、帧率调节、 隐藏鼠标、保存质量选择等必须的功能应有尽有，最好玩的是在录制时还能随意拖动取景窗。

![](.evernote-content/301A66B4-1FD5-45A0-8539-F72E215232F3/9E31188A-7A76-4004-808A-CF10D25FDFDF.gif)

如果你不需要那些功能繁多安装缓慢的复杂截图工具，只是想要简单快速地录制 Gif 截图，强烈推荐 GifCam。

UPUPOO：软萌的动态壁纸
--------------

很多使用 Steam 的派友都应该听过或者用过 Wallpaper Engine 吧，它可以将图片、视频、网页、程序或小游戏设置成壁纸，除了可以自定义以外创意工坊上还有源源不断更新的动态壁纸可以选择。不过，对于不玩游戏的派友来讲，安装 Wallpaper Engine 还需要多装一个 Steam  比较麻烦。而且 Steam 对于钱包的危害实在是太大了。

这时免费的 UPUPOO 是另外一个好选择，同样是动态壁纸软件，不过从[**官网**](http://www.upupoo.com/)和 LOGO 就可以看出它是一款偏向二次元的动态壁纸软件。

![](.evernote-content/301A66B4-1FD5-45A0-8539-F72E215232F3/8CB5F18E-0A3F-4E6B-B2A0-DF68AE5E1672.gif)

除了基础功能与 Wallpaper Engine 差不多，UPUPOO 也提供像 Steam 创意工坊类似的壁纸库，壁纸数量很多且以二次元为主，当然其他风格的动态壁纸也不少。你也可以很轻松地上传自己喜欢的视频或者网页到壁纸库上去。比起 Wallpaper Engine，UPUPOO 启动速度快上不少，性能上稍有优势，还多了壁纸控制条和管理桌面的功能，双显示器动态壁纸也不在话下。

![](.evernote-content/301A66B4-1FD5-45A0-8539-F72E215232F3/7AF1A156-4E58-4DD5-81B9-35AFD06695EE.png)

总的来说，UPUPOO 是一款优秀的动态壁纸软件，比起同类更加「萌」也更加接地气。喜欢在桌面上放上软萌妹子或者二次元帅哥的派友不要错过哦。

桌面日历：直接在桌面做任务管理
---------------

任务管理 App 我们都用的很多，不过你有没有试过直接在桌面上进行任务管理呢？**[桌面日历](http://chs.desktopcal.com/chs/)**就是这样一款从名字到功能都简单直接的国产软件。

它的功能十分专一 —— 在桌面上放置一个表格日历，可以在上面添加记事和任务 —— 就像你以前用笔在墙上的挂历记事一样。

![](.evernote-content/301A66B4-1FD5-45A0-8539-F72E215232F3/721346AD-A686-4A17-9617-95D86550206C.png)

外观上桌面日历支持自由调整整个日历位置和大小，默认颜色和透明度也可以随意调整，避免出现遮挡你心爱的壁纸。

每一个格子双击即可编辑内容，不仅支持普通的文字还支持有序和无序列表，能实现简单的 Todo 功能。每一个格子还支持自定义背景颜色用以和其他格子区分开增强辨识度，友情提示不要设置对比太强烈的颜色，否则眼睛可就要受罪了。

桌面日历也给强迫症和完美主义者提供了详细的调整设置。这里有个小提示，大家用的时候经常找不到的字体大小设置藏在了字体设置里面，推荐安装完之后就改，我用默认的「小五」就小了点。

![](.evernote-content/301A66B4-1FD5-45A0-8539-F72E215232F3/02ADBF99-BD84-4503-968E-A3DDCEAD0481.png)

很多派友做任务管理时遇到的一个困难就是经常忘记看，设置时间提醒又经常推迟然后还是忘记看（这该死的拖延症）。不如试试用桌面日历把要做的事情都放在桌面，每次返回桌面都能清晰知道自己干了啥，要干啥，既能时刻提醒又不入侵生活。

QRcode：艺术二维码生成器
---------------

最后再给有公众号，做自媒体或者需要好看二维码的派友推荐一个简单实用的艺术二维码生成器，除了普通二维码以为还能生成这种在公众号常见的动态二维码——

![](.evernote-content/301A66B4-1FD5-45A0-8539-F72E215232F3/1E3AD4E3-7CB8-4DA7-A5BE-04B0C82244B8.gif)

下载地址和超详细的使用说明都在这里：[Github 链接](https://github.com/sylnsfar/qrcode/blob/master/README-cn.md)，它原本是由 Python 编写的，有动手能力想要高度定制的朋友也可以直接扫描上面任意二维码获取它的 Python 版本。
