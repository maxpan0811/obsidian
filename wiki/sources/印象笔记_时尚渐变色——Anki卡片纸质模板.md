### 笔记类型：①填空 + 拼写 ②基础

都是基于基本的 “笔记类型” 进行修改的。也都是 Anki 官方提供的技巧。

在基于 cloze 笔记类型的基础上，正反两面各自添加了

```
{{type:cloze:文字}}
```

使得在 cloze 笔记类型的基础上，还能进行拼写操作。（小知识点）

### CSS 方面:

使用 CSS3 border-radius 属性对元素进行圆角处理。

```
	/*控制卡片四角*/
	border-radius: 15px 50px 30px 5px;	
```

使用 CSS3 渐变 (gradients)，为头部添加时尚流行的渐变色。

```
	/*设置渐变色*/
	background: linear-gradient(to bottom right, #bbb5ac, #f7f4ed);	
```

给头部文字添加阴影，使得字体能在更多的背景下清晰浮现。

```
	text-shadow:black 3px 1px 5px;
```

通过 CSS 添加了过度的动画效果，让 Anki 卡片根据活力，提供了一定的反馈。

```
/* ----------
	动画效果代码来源Leaflyer的模板，其个人博客：http://www.lofter.com/
---------- */
.slide { 
position: relative; 
	-webkit-animation: slide 1s 0s;
	-webkit-animation-fill-mode: forwards; 
}

@-webkit-keyframes slide {
	0% { opacity: 0 ; top: 100px; }
	50% { opacity: 0.8; top: -10px; }
	100% { opacity: 1.0; top: -3px; }
}
```
