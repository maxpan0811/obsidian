# 2026Q1中国旅游消费趋势洞察报告：开局之变（附件版）

> 来源：RAW/PDF/（5.7MB，含详细数据图表和分析框架）
> 本报告为该系列第9期，首次引入"灵感激发→信息搜集→决策促成→预订执行"四段决策框架

## 五大核心发现

1. **短视频与社交平台主导种草** — 合计超半数，AI种草仅2.1%（但知晓率90.2%）
2. **天气首次超过价格成为第一决策因子** — 53.8% vs 47.4%
3. **实时/实拍内容成为第一促单因素** — 23.0%超过限时优惠和灵活退改
4. **AI是信息源而非决策者** — 仅15.2%高度信任直接购买，66.2%需二次核实
5. **补偿性消费退潮，常态化增长接棒** — 大幅增支人群扩至5.80%

## 决策链条全景数据

**灵感激发**：用户信息源 — 短视频32.6% / 社交平台20.5% / OTA 17.6% / AI 2.1%
种草内容偏好：风景展示62.5% / 美食探店58.6% / 攻略避坑14.6%
触发动因：他人分享52.3% / 固定假期规划49.6% / 释放压力38.7%

**信息搜集**：平均2.9个信息源，81.6%跨App比价
决策因子权重：天气53.8% > 价格47.4% > 交通40.8% > 人流38.7%
最信任来源：用户真实分享38.8% > 熟人推荐22.6%，算法推荐仅4.3%

**决策促成**：促单因素排名 — 实时实拍23.0% / 灵活退改19.5% / 限时优惠19.4%
退改政策前移：39.9%用户将退改灵活性作为选择平台的核心标准
AI信任度：高度信任15.2% / 需要二次核实66.2% / 半信半疑15.4%

**预订执行**：预订窗口 — 8-14天38.1% / 3-7天31.0% / 提前15天+23.9%
预订渠道 — OTA 34.9% / 比价平台27.0% / 官方渠道23.0%
预订形式 — 分开预订（自主组合）48.1% / 一站式打包37.3% / 先订核心14.6%
AI在交易环节渗透几乎为零

## 酒店选择决策
- 地理位置54.9% > 卫生53.5% > 房间设施36.6% > 价格35.8% > 品牌知名度34.2%
- 收入分层：低收入（干净的便宜）→ 中收入（位置优先）→ 高收入（餐饮设计+严格卫生）
- 独自出行重价格(39%)；家庭出游重位置(57%)；搭子出行重氛围/设计(25%)

## 航司选择
- 价格49.4% > 安全记录36.4% > 座位舒适度34.3%
- 18-22岁：价格62%优先；50岁+：安全53%优先
- 带父母出行：安全41%+舒适42% > 价格39%

## 旅行社选择
- 无购物/无隐形消费40.9% > 品牌37.5% > 价格36.5%

## 新兴热点
- **演出旅游**：60.4%感兴趣，实际仅22.8%；F1中国大奖赛票房+30%，武汉马拉松经济效应10.15亿
- **无人酒店**：25.6%体验过，科技体验62.4%为核心吸引力，安全顾虑为最大阻碍
