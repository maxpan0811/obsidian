朋友们，这是一个喜大普奔的好消息：

NotebookLM 很强大，我们都知道。但它有个让人抓狂的问题——**每次都要手动添加资料、手动点击、手动等待生成**。如果你在研究一个新主题，光是找影片、找来源、等它跑完，就要花掉一大堆时间。

![](attachments/1534416d2e8114ad.png)

今天我要告诉你一个组合拳：**Claude Code + NotebookLM**，把一切变成一段对话！

用 Claude 搜索影片、添加资料、生成简报，全部在 Claude 里面完成。从此之后，你再也不用打开 NotebookLM 的界面了。

## 一、这个组合到底有多神？先看 Demo！

我直接在 Claude 里输入：

“帮我搜索关于 Claude Code 最新热门的影片，给我一个总结分析，然后在 NotebookLM 里帮我生成一个 Infographic。”

然后，我就去喝了杯咖啡。

几分钟后回来，**一张完整的信息图已经生成好了**——从新手入门到进阶应用，四个阶段清清楚楚。同时，Claude 还给我写了一份总结分析，包括核心主题、趋势和完整教学脉络。

![](attachments/ea7995b785940c68.png)

最关键的是什么？

我回到 NotebookLM，刷新一下页面，**那 5 个视频来源已经自动添加进去了，Infographic 也在资料夹里静静躺着**。

整个过程，我**没有打开过 NotebookLM 的界面，没有手动点过任何按钮**。

这就是我说的：**如果你用过这个组合一周，你会觉得手动操作都是傻子。**

## 二、为什么这个组合这么强？

让我先简单介绍一下这两个工具：

### NotebookLM：Google 的 AI 研究助手

你可以把 PDF、影片、网页丢进去，然后：

- 问它问题
- 让它划重点
- 生成简报
- 甚至做成 Podcast

它最厉害的地方是：**只会从你给的资料里找答案，不会乱编。**

### Claude Code：Anthropic 的 AI 编程工具

它可以：

- 帮你写程序
- 操作文件
- 通过 MCP（Model Context Protocol）连接外部服务

**MCP 你可以把它想象成一个插头**——让 Claude 插上各种外部工具，然后操作它们。

### 组合起来呢？

把这两个连在一起，就可以**用 Claude 来操作 NotebookLM**。

不用自己点击，不用自己记忆步骤，**直接用自然语言说你要做什么就好了**。

### 第二步：安装 NotebookLM MCP

打开终端器（Mac 的 Terminal 或 Windows 的命令提示符），输入安装指令。

安装之后，需要账号认证。因为 NotebookLM 是 Google 的服务，需要你登录账号才能操作。它会跳出一个登录窗口，你登录之后就完成了——**跟平常登录 Google 服务一样简单**。

### 第三步：用 Claude 帮你完成所有设置

这是我最喜欢的部分。

很多人看到安装教程就头大——要下载这个、再下载那个、然后配置一堆东西。

**但我的做法是：直接把 GitHub 的下载链接复制起来，丢给 Claude。**

然后跟它说：

“请帮我下载这个 GitHub repo（https://github.com/PleasePrompto/notebooklm-mcp），帮我设置好，让我可以开始使用 Claude + NotebookLM。记得，帮我安装到全局（globally），这样我在任何资料夹里都可以用。”

![](attachments/70fbc22face9e179.png)

按下 Enter，Claude 就开始干活了。

**我甚至不用打开终端器，Claude 会自动帮我执行命令。**

你会发现，其实很多步骤你可以直接跟 Claude 说”你可不可以帮我做”，**大概 88% 的时候它都会帮你做**。这是我发现的小技巧。

### 第四步：登录 NotebookLM

Claude 会提示你需要登录。这时候，直接跟它说：

“帮我执行登录指令。”

它会打开一个 Google 登录窗口。你登录完之后，**按一下 Enter**，它就会把你的登录信息存起来。

之后你就再也不用登录了。

## 四、 开始使用：把重复性工作变成一句话

设置完之后，你就可以开始玩了。

### 测试一下：生成 Study Guide

假设我的 NotebookLM 里已经有一个叫”Demo Claude”的 Notebook，里面有几个资料来源。

我直接跟 Claude 说：

“Hi Claude，我想要在我的 NotebookLM 里，那个叫 Demo Claude 的 Notebook 里面，把我所有的 source 做成一个 Study Guide。”

然后，我就去忙别的事了。

几分钟后，Claude 告诉我：

“Study Guide 已经生成完成，并已下载。”

我回到 NotebookLM，刷新一下，**Study Guide 已经静静躺在那里了**。

### 我常用的几个应用场景

#### 1. 打造 PDF 知识库

如果你在研究一个主题，有一堆 PDF 要看，你可以叫 Claude 把它们全部丢进 NotebookLM，然后自动生成摘要。

**再也不用一篇一篇读了。**

#### 2. Podcast 脚本生成

找几个你喜欢的 YouTube 频道，把最新影片加进去，然后叫 Claude 生成 Podcast 脚本或 Audio Overview 的脚本。

这很适合做内容创作或学习总结。

#### 3. 竞品和市场研究报告

假如你想了解某个竞品或市场，你就叫 Claude 去找相关的文章、影片，然后统一加进 NotebookLM，生成一份分析报告。

**比你自己一篇一篇读快太多了。**

#### 4. 学习笔记自动化

每次看完一个课程，你就叫 Claude 帮你整理一份学习笔记和重点卡片。

长期下来，你的知识库会越来越完整。

#### 5. 个人 AI 知识库

把你自己的文件、笔记、资料定期同步丢进 NotebookLM，然后随时可以用自然语言去查询。

![](attachments/f8e01ff5c5fe1a79.png)

**这是你自己的 AI 助理，因为它只会根据你的资料回答。**

## 五、更进阶的玩法：技能叠加技能

我刚刚还用了一个叫”YouTube Search”的 MCP 技能。

它基本上就是帮我们去抓 YouTube 影片的资源、标题等等。

我把 YouTube Search 这个技能跟 NotebookLM 的技能结合起来，**技能跟技能之间互相沟通**。

所以刚刚就是：

1. 搜索”Claude Code 教学”影片
2. 把搜索结果传到 NotebookLM
3. 生成 Infographic

这就是 Claude Code 的核心：**技能加技能，完成复杂任务。**
