我的 2010 年代清单 | 一位设计师的十本好书

> 阅读一本书，就好比跟一位有经验的长者在对话，阅读一本设计专业的书籍，就更如同是在跟一位设计专家在交流。

要说 2010 年代的设计，最热门的当数用户界面设计，也就是 UI 设计（User Interface Design）。互联网公司工资高、福利好、弹性工作…… 等等，使得 「UI 设计师」职位成从众多设计工种里脱颖而出，成为了最被关注的对象。

2011 年，也就是我进入大学的那一年，买了第一部大屏智能手机，也就这个时代，移动互联网产品群雄逐鹿，老牌新秀的互联网团队，纷纷设计了各种各样的 APP。从 2020 年回望，现在的互联网大梯队的，也正是从那个时间点开始的。我们这批人，可以称之为第一代移动互联网的「原著民」。

我本科的专业是工业设计，与此同时，国内大部分的设计院校，并没有「UI 设计」相关的专业。工业设计专业里有一个分支是「人机交互」方向，「人机交互」方向研究的是人与机器之间的关系，人与机器的交互，通过用户界面的媒介进行交流。因此，学习了人机交互方向，是最有可能毕业后从事 UI 设计工作的。

由于我有意识地关注和学习「人机交互」相关的知识，在大学四年级时，获得了一个 UI 实习设计师的工作机会，由此进入了互联产品设计行业。毕业后，进入一家明星级互联网创业团队工作，参与了互联网产品设计的「从 0 到 1 」的整个过程，以及从桌面端到移动端的整体使用场景的设计，在实践和不断学习中，从事 UI 设计工作至今。回顾自己的专业知识，大部分还是靠阅读书籍所获得的。

下面是我从事设计师工作以来，对我影响深刻的十本好书。

《在你身边，为你设计》
-----------

* 作者：腾讯公司用户研究设计与设计体验部
* [详细介绍](https://book.douban.com/subject/20471302/)

![](.evernote-content/E370BD9F-8FC4-436D-A099-DFB5954A9332/9E3A40FD-6DFF-4EAE-89BC-3E8DBBD74273.png)

大学三年级阅读一本设计书，是关于互联网产品设计的，也是直接影响了我毕业后就业方向的书。本书包含了互联网产品设计规过程中的：设计理念、设计方法、用户研究、体验设计、设计流程和工具，以及团队成长与管理等方面的知识与经验分享。

对「用户体验设计」产生了深厚的兴趣，正是来源看完这本书。毕业校招季的时候，当然也参加了腾讯设计的校招，虽然止步于最后一轮的面试，但是我在其他的互联网公司中获得了工作几乎，并一直从事着「用户体验设计」相关的工作。

《About Face 4: 交互设计精髓》
----------------------

* 作者：[美] 艾伦・库伯 / [美] 罗伯特・莱曼 / [美] 戴维・克罗宁 / [美] 克里斯托弗・诺埃塞尔
* [详细介绍](https://book.douban.com/subject/26642302/)

![](.evernote-content/E370BD9F-8FC4-436D-A099-DFB5954A9332/F4413542-9196-49F5-90B0-0884478DF40E.png)

《About Face 4: 交互设计精髓》系统地讲述了交互设计的过程、原理和方法，涉及的产品和系统有个人计算机上的个人和商务软件、Web 应用、手持设备、信息亭、数字医疗系统、数字工业系统等。

这是我花最多精力去研读的一本设计书，反复阅读过两次，一次是大学时期，硬着头皮「死啃」了一遍，但由于太过深奥的理论，我基本没有读懂书中讲的内容；另一次是在工作后的第三年，我再次阅读了这本书，花了近半年的业余时间，结合实际的工作实践和思考，对这本书又有了新的认识。

这是一本从事「用户体验设计」相关工作的人员必读的一本经典，有交互设计界「字典级」书的美誉。

《史蒂夫・乔布斯传》
----------

* 作者：[美] 沃尔特・艾萨克森
* [详细介绍](https://book.douban.com/subject/6798611/)

![](.evernote-content/E370BD9F-8FC4-436D-A099-DFB5954A9332/EF0B2AD6-6897-4F55-AE7A-8266CB606382.png)

史蒂夫・乔布斯一生创造了个人电脑、动画电影、音乐、手机、平板电脑以及数字出版等六大领域的颠覆性变革，他不仅是一位极具创造力的企业家，更是一位充满追求完美的设计师。

我们今天互联网产品设计，很大程度地继承和延续了史蒂夫・乔布斯的设计思想。阅读这一本书，了解到史蒂夫・乔布斯做当时当地的历史环境下，个人的成长的过程，以及对于产品设计的思考，具有非常大的指导和借鉴意义。

《启示录：打造用户喜爱的产品》
---------------

* 作者：[美] Marty Cagan
* [详细介绍](https://book.douban.com/subject/5914587/)

![](.evernote-content/E370BD9F-8FC4-436D-A099-DFB5954A9332/A035FED5-FA3B-45DA-AECD-3C109591DFF6.png)

人员、流程、产品是本书中讨论的三大部分，这本书系统地描述互联网产品的诞生过程，并对产品从业人员提出了一系列的提问。

一本产品经理必看的书目，但事实上，设计师也更应该了解产品，设计师的工作需要从「编程」到「产品」的知识面进行延伸，才能更好的回到「设计」的本身，完成更出色的设计。

《硅谷设计之道》
--------

* 作者：王欣
* [详细介绍](https://book.douban.com/subject/30774766/)

![](.evernote-content/E370BD9F-8FC4-436D-A099-DFB5954A9332/DEFF95D8-1C1C-47CE-85C7-28F3E78C0B39.png)

作者较为通俗的描述了自己对「用户体验设计」的理解，没有高深的理论，但却清晰地表达了创意的流程，全面剖析了硅谷先进的产品设计流程。列出多种设计工具或者思维工具，以此深化用户体验策略的学习和应用。

《原田进：设计品牌》
----------

* 作者：[日] 原田进
* [详细介绍](https://book.douban.com/subject/27008753/)

![](.evernote-content/E370BD9F-8FC4-436D-A099-DFB5954A9332/30D1D3C7-1B09-4723-9C08-3192276D8FFF.png)

《原田进：设计品牌》系统地讲述了品牌设计的整个内容，可以当作工具书，制定品牌设计手册。不管是不是从事「品牌设计」的相关工作，都应该去全面地了解一下「品牌」，商业中涉及的设计，最终目的都是给消费者灌输「品牌」，为他最后买单的时候建立决策意识。

《设计中的设计》
--------

* 作者：[日] 原研哉
* [详细介绍](https://book.douban.com/subject/1941558/)

![](.evernote-content/E370BD9F-8FC4-436D-A099-DFB5954A9332/06E71830-2132-465C-BC59-2300EBE03384.png)

设计到底是什么？

如果要对设计的本质，进行更深层次的思路，回答这个问题，就不妨阅读原研哉的《设计中的设计》。原研哉作为一名从业二十余年并且具有世界影响的设计师，他在书中描述了自己的探索，对一系列的产品进行了再设计，并策划了「RE-DESIGN—21 世纪的日常用品再设计」展对「设计到底是什么？」这个问题给出了答案。

> 如果要将某一种素材当成优秀的素材，第一步就是要把这种素材的特性无限纯化。对于雕塑而言，黏土是隐藏着无限可能性的素材，但这种可能性的实现是与黏土作为雕塑素材的纯粹分不开的。如果黏土中藏着钉子或者金属片的话，人们就不能随心所欲地塑造它。我们能满手是血的和泥吗？很难想象在这种情况下生产出来的东西能给我们的生活带来富足。
>
> —— 引自《设计中的设计》第一章「设计到底是什么？」

《造型的诞生：图像宇宙论》
-------------

* 作者：[日] 杉浦康平
* [详细介绍](https://book.douban.com/subject/23782092/)

![](.evernote-content/E370BD9F-8FC4-436D-A099-DFB5954A9332/0EF39648-D8FF-47E0-B3E3-67298E0F3D35.png)

讲造型的诞生，如果早一点能看到这本书，我应该毕业的时候会更坚持做工业设计。设计师最主要的产出，仍是将信息图形化，用更合适的解决方案，将产品的意图传达给用户。人类的起源，伴随着图像的产生，通过去挖掘深植于人们心中的形象和概念，概括了整个「图像宇宙世界」。这本书也真是带着我们去寻找」图形「的起源，以及」图形「是如何被人类理解的。

《观看之道》
------

* 作者：[英] 约翰・伯格
* [详细介绍](https://book.douban.com/subject/26427579/)

![](.evernote-content/E370BD9F-8FC4-436D-A099-DFB5954A9332/78C3DB55-0D90-470C-8959-32AAACB20E8A.png)

这是我看的第二本约翰・伯格写的书，起初知道这本书，是「看理想」梁文道老师的节目中推荐的，是与电视系列片《观看之道》在 BBC 播出，同时出版配套的图文册，遂成艺术批评的经典之作。

> 观看先于言语。观看确立了我们在周围世界的地位。他人的视线与我们的相结合，使我们确信自己置身于这可观看的世界之中。而当图像终于可以取代文字，充斥于视线所及的一切空间，我们又该如何观看而不至于迷失？
>
> —— 引自《观看之道》

《三体》系列丛书
--------

* 作者：刘慈欣
* [详细介绍](https://book.douban.com/subject/6518605/)

![](.evernote-content/E370BD9F-8FC4-436D-A099-DFB5954A9332/8447DACD-85DD-45A8-8CB8-E3A58875F923.png)

最后一本是非「设计」标签的书，但是对我影响也同样深远。看完《三体》系列丛书之后，在我有限的人生长度和视野范围内，我选择相信宇宙就是大刘书中所描述的样子，那就是我对宇宙的向往。世界观甚至宇宙观的建造，同样也是设计师思想重要的部分。眼光朝着未来，未来才会慢慢变成我们设计的样子。

> 宇宙就是一座黑暗森林，每个文明都是带枪的猎人，像幽灵般潜行于林间，轻轻拨开挡路的树枝，竭力不让脚步发出一点儿声音，连呼吸都小心翼翼…… 他必须小心，因为林中到处都有与他样潜行的猎人。如果他发现了别的生命，不管是不是猎人，不管是天使还是魔鬼，不管是娇嫩的婴儿还是步履蹒跚的老人，也不管是天仙般的少女还是天神般的男孩，能做的只有一件：开枪消灭之？在这片森林中，他人就是地狱，就是永恒的威胁，任何暴露自己存在的生命都将很快被消灭。
>
> —— 引自《三体》黑暗森林第 446 页

以上十本书，是从我设计师职业成长中的「设计启蒙、追寻先驱、系统学习、深入思考、延伸宽度和前瞻未来」等版块挑选出来的，也就是我的 2010 年代的十本设计好书。

别熬夜，多运动，保持阅读。
