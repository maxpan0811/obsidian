# 5款真正必备的 Mac 软件，不装根本用不了！

引言

如果以个人需求当作是否“必备”那实在有失公允，比如我经常使用 Apple 图书 App 看书写读书笔记，需要将笔记导出只能使用 Kilb 这款软件，所以这是我 Mac 必装的，那是否称得上 Mac 必备软件呢？

我想肯定不行。

所以本期“真正必备的 Mac 软件”一定是 Mac 买回来不装这些软件，大概率都无法正常使用的。虽然我这么说有些绝对了，但还是那句话，基于我近二十年的使用经验，错不了太多。

解压缩软件

Mac 自带的解压工具无法解压缩 .rar 格式的压缩文件。而目前网上大多数资源要么是 .rar 格式的，要么是加密码的 .rar 格式的。不安装解压缩软件基本是半残的，所以这类软件是“必备”的。

![](.evernote-content/C7F30633-DBED-4547-82D1-E2B95BE1B7FF/EF295FF9-3FCD-45BC-BC7A-4B424CA6A55A.png)

这里推荐 The Unarchiver 这款解压软件。因为可以直接在 App Store（Mac） 中直接下载（后面我再解释为啥要提这点），并且是完全免费的。下载后只需运行一次，之后解压缩几乎和 Mac 原生解压工具体验一致。

广告屏蔽软件

为啥上面要强调在 App Store（Mac） 下载呢？这是因为 Safari 浏览器无法屏蔽网页广告——通常 Windows 系统我们安装第三方浏览器，大多会自带广告屏蔽功能，虽然不全面，但至少在搜索和下载时，不会被广告劫持去莫名下载一些 .exe 文件来。

![](.evernote-content/C7F30633-DBED-4547-82D1-E2B95BE1B7FF/0900C65A-7ABD-4AF9-97CA-894E018F9CED.png)

这里首先推荐安装 Safari 浏览器扩展 1Blocker，我在对比了很多类似软件，这个应该是价格比较便宜的，功能比较强的那种了。并且 iPhone 和 iPad 也可以使用，如果正巧你还有 Apple Vision 的话也可以。

其次我推荐直接安装第三方浏览器。因为 Safari 浏览器上的广告屏蔽扩展都是付费的，所以最好的方式就是使用别的浏览器来安装免费的扩展。

![](.evernote-content/C7F30633-DBED-4547-82D1-E2B95BE1B7FF/891FC9F8-48A6-4D25-B4CB-AB41BA00CC9C.png)

我比较推荐的是安装 Microsoft Edge 浏览器，主要是不花点精力 Chrome 浏览器这种是下载不了的，更别说安装了。

![](.evernote-content/C7F30633-DBED-4547-82D1-E2B95BE1B7FF/83116970-AD2E-46D9-B8D6-CE16E8E3094F.png)

然后在 Microsoft Edge 浏览器扩展功能中搜索一下 uBlock Origin 这个广告屏蔽扩展就可以了，我是为了屏蔽更多内容还装了 uBlacklist 这个扩展，好用不止一点点。

视频播放软件

在线播放视频，配合上面的广告屏蔽软件应该是很舒服了。但是下载的视频，如果是 .mp4 格式的还好，像是 .rmvb 或是 .mkv 格式等，Mac 基本就无力播放了。必须安装视频格式解码器或第三方播放器。

![](.evernote-content/C7F30633-DBED-4547-82D1-E2B95BE1B7FF/BA6C3373-F19E-4ACD-B2EF-2FBA33C6D219.png)

解码器安装不适合新手，从“必备”的角度来讲，还是第三方播放器比较合适。这里推荐 IINA 视频播放器。

![](.evernote-content/C7F30633-DBED-4547-82D1-E2B95BE1B7FF/7D668C39-3F11-495B-AEE0-BEAD35F1356D.png)

界面美观，解码能力强，几乎能播放所有视频文件，并且色彩配置很稳，不会出现看着看着画面里的红色变灰色这种奇葩问题。

硬盘读写软件

写到这里我几乎想不出来还有什么必备的了，甚至广告屏蔽软件也只是勉强入围，所以算是找了个凑数的。

![](.evernote-content/C7F30633-DBED-4547-82D1-E2B95BE1B7FF/87A7C88E-8537-45FD-9515-AAA393684E4B.jpg)

外接的大容量移动硬盘比较常见使用 NTFS 格式，在 Windows 系统下非常通用和方便，但连接到 Mac 上只能从硬盘中复制文件，无法往里传输文件。因为 NTFS 本质是 Microsoft 为 Windows 系统开发的文件系统，所以 Mac 必需使用第三方软件才能实现写入功能。

![](.evernote-content/C7F30633-DBED-4547-82D1-E2B95BE1B7FF/E69CCB85-A96F-4650-892F-42179BD94A13.jpg)

此类软件比较推荐的是 iBoysoft NTFS for Mac 和 Paragon NTFS for Mac，这两款界面比较美观，使用简单易上手，几乎是即装即用，读写 NTFS 的外接硬盘速度也非常理想。

附加

软件卸载软件

虽然不能称之为“必备”，但是对于有需要卸载软件的 Mac 用户还是很有帮助的。

![](.evernote-content/C7F30633-DBED-4547-82D1-E2B95BE1B7FF/DE390449-1626-497D-802B-0C069C2DDDB7.png)

常规的 Mac 软件卸载主要是删除了软件本体，一些配置文件还保留在设备中，如想一并移除，可以使用 AppCleaner 这款软件，同样免费轻量且易用。这里还有一篇介绍的文章：[软荐更新｜AppCleaner - 让 Mac 软件卸载得更彻底](https://mp.weixin.qq.com/s?__biz=MzU3MjYwMTA2MQ==&mid=2247486693&idx=1&sn=89f29ed69a9f72ce0d7fad1fe8454829&scene=21#wechat_redirect)

结尾

希望这五款软件能有一两个是你需要且能长期必备的，如有更好的推荐，欢迎在评论区讨论～我们下期再见啦！
