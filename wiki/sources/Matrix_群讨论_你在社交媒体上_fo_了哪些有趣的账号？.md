# Matrix 群讨论 | 你在社交媒体上 fo 了哪些有趣的账号？

Matrix 群讨论 | 你在社交媒体上 fo 了哪些有趣的账号？

本周，在第十四场「和派聊聊」Matrix 作者群讨论活动中，作者们各自分享了自己在社交媒体上 fo 了哪些有趣的账号，并说了推荐的理由。机器人为大家整理了讨论中的精选内容，一起来看看吧。

> 大家也可以到「和派聊聊」讨论文章 的评论区中留下你在「涨姿势」这个话题上的经历和看法，参与到同一话题的讨论中，更有机会获得我们送出的 Google 官方扑克。

@朱岳：后浪文学的编辑，好奇心日报「星期六小说」专栏的特约编辑，在原创文学优秀读者的发掘和华文文学的引进中有着举足轻重的作用。说人话就是，他标记的书都很棒，动态是出版社编辑部的经典日常。

@不流：星期六小说专栏第四篇文章《对一次暴力的描述》的作者，合肥书店「保罗的口袋」创始人，他的动态是买书看书看电影，他是我书讯的重要来源。 
@重版出不来：编辑的专用树洞，动态就是编辑部劝退的那些事儿，有一颗做编辑的心的我往往越看越勇（哈哈

@记录生活的蛋黄派：这位长得像白客，举止神态像张全蛋的 00 后小伙，凭借着五毛钱的特效，真实不做作的体验反馈，让我在看完他第一支视频后便关注了他，就连我妈，也挪开了注视《非诚勿扰》的目光，对着 iPad 后的 Up 主笑不拢嘴。蛋黄派吃的东西不贵，一顿肯德基新品套餐，一次网购的口水鸡，都能成为他视频的主题，4-7 分钟一支视频，刚好填满零碎时间，获得快乐。

@锦堂生活空间：来自马来西亚的清华大学留学生，喜欢打 DOTA 2，有创业打天下的梦想，分享清华大学的学习和自律日常，在不计其数的「高材生 + 校园」的 vlog 里面凭借着乐观的心态和塑料但可爱的普通话，让我持续关注着。

@魔都食鉴局：经常对一些热门或者有趣的食物进行鉴赏和分享，并根据多位鉴赏人员的评价进行综合评分，有理有据，方便排雷。

@分镜 bot：接受私信投稿的账号，主要是分享一些漫画的有趣分镜，并且要求投稿都附上推荐的理由，经常会看到一些让人眼前一亮的漫画。

微博在我看来是一个严肃（指杠人撕 x 以及给爱豆打 call 等等方面）的社交平台，自己也一直不时清理关注人的习惯。所以到最后剩下有趣的博主不多了，但 @氪爆 绝对是能够让人无脑「哈哈哈哈哈哈」的那一个。

每次首页时间流负能量太多的时候我就会想起这个账号、点开他主页、然后看看他置顶微博的「每日沙雕推送」。

「哈哈哈哈哈哈哈哈哈」，正义不一定会到，氪迟但到。

魔都食鉴局：非常擅长横评的吃货小团体，每次推文都有堪称神来之笔的动图来挑动读者的味蕾…… 以横评、探店（街）为主，大众点评种草餐厅的重要来源。

RUC 新闻坊：运营主体很独特的「独立媒体」（来自中国人民大学新闻学院），可以看到不少高质量的策划、编译等内容…… 爱了，每篇基本必刷。

不会画出版社：用漫画的形式讲述每个人身边都可能发生的故事，在排版上充分利用了微信公众号图文的特点进行创新…… 还真是头一回在微信里见到有多结局的漫画。

共青团中央：唔…… 虽然皮皮团（划掉）在几乎全部主流平台都开通了账号…… 但还是觉得 QQ 空间最有趣，可能还是因为大部分空间网友足够积极和理智，评论区偶尔很好玩（再加上皮皮团大晚上动不动放毒，以及每个早上的早安也很能打动人的样子～～）。

空间还有另一个用处：团团偶尔会把一些放不上台面的时评发在空间小秘密里头…… 如果看到有一条小秘密发出来没几分钟就有几十条评论…… 那肯定是皮皮团了 [捂脸]
