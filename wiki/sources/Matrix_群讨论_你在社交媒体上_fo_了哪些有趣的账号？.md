# Matrix 群讨论 | 你在社交媒体上 fo 了哪些有趣的账号？

本周，在第十四场「和派聊聊」Matrix 作者群讨论活动中，作者们各自分享了自己在社交媒体上 fo 了哪些有趣的账号，并说了推荐的理由。机器人为大家整理了讨论中的精选内容，一起来看看吧。

> 大家也可以到「和派聊聊」[讨论文章](https://sspai.com/post/56207) 的评论区中留下你在「涨姿势」这个话题上的经历和看法，参与到同一话题的讨论中，更有机会获得我们送出的 Google 官方扑克。

## 蛋挞西瓜

**豆瓣：**

@[朱岳](https://www.douban.com/people/1867648/)：后浪文学的编辑，好奇心日报「星期六小说」专栏的特约编辑，在原创文学优秀读者的发掘和华文文学的引进中有着举足轻重的作用。说人话就是，他标记的书都很棒，动态是出版社编辑部的经典日常。

@[不流](https://www.douban.com/people/tuiblio/)：星期六小说专栏第四篇文章《对一次暴力的描述》的作者，合肥书店「保罗的口袋」创始人，他的动态是买书看书看电影，他是我书讯的重要来源。

@[重版出不来](https://www.douban.com/people/172657707/)：编辑的专用树洞，动态就是编辑部劝退的那些事儿，有一颗做编辑的心的我往往越看越勇（哈哈

![](attachments/758f9b0e82581e05.png)

**B 站:**

@[记录生活的蛋黄派](https://space.bilibili.com/337521240?from=search&seid=1332272008282353650)：这位长得像白客，举止神态像张全蛋的 00 后小伙，凭借着五毛钱的特效，真实不做作的体验反馈，让我在看完他第一支视频后便关注了他，就连我妈，也挪开了注视《非诚勿扰》的目光，对着 iPad 后的 Up 主笑不拢嘴。蛋黄派吃的东西不贵，一顿肯德基新品套餐，一次网购的口水鸡，都能成为他视频的主题，4-7 分钟一支视频，刚好填满零碎时间，获得快乐。

@[锦堂生活空间](https://space.bilibili.com/383038901?from=search&seid=9871654283089019461)：来自马来西亚的清华大学留学生，喜欢打 DOTA 2，有创业打天下的梦想，分享清华大学的学习和自律日常，在不计其数的「高材生 + 校园」的 vlog 里面凭借着乐观的心态和塑料但可爱的普通话，让我持续关注着。

「哈哈哈哈哈哈哈哈哈」，正义不一定会到，氪迟但到。

## 小菜子

**YouTube：**

[Ali Abdaal](https://www.youtube.com/user/Sepharoth64)：一位剑桥毕业的医学生，目前是一名驻院医生。或许大家都熟知，学习区博主的专业通常就是医学和生物。所以 Ali 的业余 YouTube 就是有关学习的，他经常使用最新的科技产品，将科技的手段融入工作学习的流程中，非常具有参考意义。另外有网站和播客。

[Thomas Frank](https://www.youtube.com/user/electrickeye91)：一位知识管理专家。主要是学习生活的一些技巧，比如如何保持良好习惯、如何做笔记等等，非常适合在读的大学生。

[Yale Chen](https://www.youtube.com/user/Giojwychen)：台湾的一位理财博主，专注于美股理财。虽然相比他的理财内容，我更喜欢他个性生活化的一面，觉得他很有幽默感。他的性格会 导致粉丝两极化，喜欢的人会很喜欢，讨厌的人也会很讨厌。

[Morvan](https://www.youtube.com/user/MorvanZhou)：主攻 Python，内容浅显易懂，有 B 站账号：莫烦，也有网站。

![](attachments/a6be729beed191ec.png)

**Twitter：**

[Wes Bos](https://twitter.com/wesbos)：加拿大的一位前端技术大拿。我觉得他的 Twitter 很有趣，贴近生活，同时也有很多专业的编程技巧。我是因为他的 JavaScript 30、CSS Grid 粉上他的。对于前端爱好者或者程序员，强烈推荐关注。另外他和另外一个程序员创建了 Syntax 播客，现在已经成为排名相当靠前的编程技术播客（英文），值得一听。

**B 站：**

[小浩美食记](https://space.bilibili.com/298289539?from=search&seid=2784216533023964638)：日常做菜，方便面是他的拿手好菜，千奇百怪的做法，非常让人有做饭的欲望。尽管他是我昨天关注的，但是我已经迷上了他的雪碧冷面，迫不及待就要动手了。

[WillTV](https://space.bilibili.com/205795482?from=search&seid=18319240342570077904)：有着最简单的 Mac 入手教程，有 10 个视频。在我眼里，排名第一，我找不到比它更好的中文教程了。

**Others：**

[Future Learn](https://www.futurelearn.com/)：广大学习爱好者不容错过的公开课学习平台，虽然它的内容相比 Coursea、edX 更为小众，但是它竟然有西班牙语、意大利语、德语的专项学习课程，还有 Haskell 函数编程、weka 数据挖掘、Python 图形化界面开发，有全球时尚变化、能源革命、生活中的化学物质、冥想与正念、领导力……

[Product Hunt](https://www.producthunt.com/)：对于时刻追随潮流、具有广阔好奇心的我来说，它简直就是宝藏。每次都可以发现最新的 App 或者网站。

[看理想](http://www.vistopia.com.cn/)：强烈推荐新推出的《100 个职业告白》。100 种职业，100 种人生，从视频里我可以体验到他们都在认真努力的生活，由此产生了一种热爱。（感动是一种玄学，有兴趣的可以看一看）

## nerd\_du

之前在 YouTube 上因为漫威的相关内容关注了「[超粒方](https://www.youtube.com/channel/UC0Q-fBheHysYWz9ObSEzMdA)」和「[加點吉拿棒](https://www.youtube.com/channel/UCec6QE03CMLgtf0f1-uDdbw)」（中文，有字幕），超粒方做的影评视频都挺有意思，不只是简单地讲剧情或彩蛋，其中会穿插很多电影专业知识和文化背景，总能看到一些意想不到的新知识点。

包括现在已经停更的 YouTube 频道「[Every Frame a Painting](https://www.youtube.com/user/everyframeapainting)」（英文，有中文字幕），也是讲电影相关知识，博主是两位电影相关专业的学生（现在应该工作了），很多人应该在微博上看到过他讲「成龙的电影为什么和其它动作电影与众不同」。

其实一些媒体的官方 YouTube 频道也会有不少有趣的系列内容，例如 [Vox](https://www.youtube.com/user/voxdotcom) 的 Borders 系列（英文字幕），带你了解各地的风土人情和历史背景，看看也很有意思。
