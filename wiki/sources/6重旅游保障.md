# 6重旅游保障

|  |  |  |
| --- | --- | --- |
| |  |  | | --- | --- | | 6重旅游保障 |  | |
| 携程目前已通过100%透明团、特殊原因退订保障、自然灾害保障金、旅游预警机制、应急援助体系、一站式旅游保险6重保障服务，形成了完整的客户权益保障和关怀机制。 |
|  |
| 100%透明团  为保证客人明明白白去旅游，携程针对团队游涉及的食、住、行、游、购、娱、导等细节，率先推出“100%透明团”，明确规定所有透明团产品均需做到：  · 大小景点介绍全，行程透明不走样；  · 酒店资讯全披露所见即所得；  · 购物店信息先知晓，看得清楚买得明白；  · 航班信息提前公示，行车里程清楚说明；  · 推荐项目早公布，一路心中有数；  · 团餐比例早知道，美食计划更自由。游玩时间明确定，行程不匆忙无
