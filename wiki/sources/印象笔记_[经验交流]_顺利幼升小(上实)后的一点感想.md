| |  | | --- | | 昨天看了上海台1/7节目关于小升初的专题，才发觉全上海每个角落的竞争都很激烈，庆幸女儿早早考进上实，少了很多后顾之忧。  作为爸爸，名牌国内本科加名牌国外硕士，中学阶段物理、数学、化学都是省和市奥数队的（虽然自己最后社会上混得很一般），但总觉得小朋友学习应该也不差，所以一直懒于给小朋友报各种学习班，唯独热衷于跟着女儿两个人到处旅行，体验各种吃喝玩乐的腐败生活。  反倒是孩子妈，早早开始准备，时不时警示我幼升小的惨烈。因为我坚决反对报培训班，孩子妈就自己泡论坛，专研教材，自己在家里带孩子学习英语、数学和逻辑。当然，小朋友在外面学习舞蹈、滑冰这类的体育类班级我还是赞同的。  因为房子对口菜场，我们也把目光投到了上实和几大民办小学。具体过程就不多说，情况和大多数家庭没有差别。在上实机考和面试中，我有几次都会怀疑女儿能否顺利突围，因为小朋友的不可预知性太强，而且和别的爸妈交谈中，感觉女儿并不出类拔萃。但是女儿可以很自信和清晰地和我讲述考试的各类题目，也很确定自己正确与否，让我对她还是保留着一点信心和期待。  5月9日那天，第一时间上网刷到女儿被上实录取的通知，立刻放弃5月10日所有民办小学的面试计划。一个选择足矣，我才不要像其它人家，手握多个offer然后纠结。于是一家人周末开开心心地出去玩了两天。  有很多朋友问我经验，其实我有些羞愧，实在没法总结出很多，但我还是想说自己的一点感想，尤其是看了电视的专题节目之后。  第一、小朋友气质最重要。自信、阳光、机灵的气质，远比你会做无数题目更出挑，在面试的时候更有辨识度。每个小朋友在爸妈眼里都是最棒的，但是在别人眼里就会有区别。长相没法改变，但是气质可以培养。在小学面试的时候，人多时间紧，老师必然认定那些看的对眼的孩子。  第二、培训班未必要上，但家庭教育必不可缺。我很赞同论坛里一个帖子，家在是最牛的学校。百花和昊品这类的培训班我们没上过，甚至可以说是没听过。但是我们平时会在潜移默化中训练孩子的逻辑思维，事实证明下来，并不比上过班的孩子差。在机考前，我专门请假了一天，给孩子总结了可能的各类题型及变化，孩子也觉得爸爸上课非常好。  第三、注重孩子的学习习惯培养。回想起来，我和孩子妈分工明确，我负责带孩子动，妈妈负责静。孩子玩起来很疯，读书起来很安静，这样的习惯非常好。  第四、按照自己的节奏培养，不要盲目跟风。考上了理想的学校，我反而才开始关注起论坛，才发现竞争的激烈超乎我想象，还好我很晚知道这个论坛，否则我肯定紧张到自乱阵脚。我在想，即使进了菜场小学，我们按照自己的节奏培养，虽然辛苦，但也是可以培养一个出色的孩子。  无论如何，上了名校也不是一劳永逸，学习是个持续的过程，人生的路很长，保持一个谦虚好学的心更重要。此外，大多数孩子今后都是普通人，我觉得也应该让孩子学会接受平庸。我跟女儿说了，在上实保持不掉队就行，到了高三，如果应试成绩好，就去高考，如果不行，那咱就出国去读书吧。  最后感恩一下，孩子的出色表现让我们这条路走的很顺利，提前结束战斗。但无论成功失败，孩子健康快乐胜过一切。 | |
