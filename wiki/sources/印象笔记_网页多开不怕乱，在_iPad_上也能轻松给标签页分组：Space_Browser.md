网页多开不怕乱，在 iPad 上也能轻松给标签页分组：Space Browser
========================================

我们每天花费在浏览器上的时间占据了我们数字生活的大半部分，越来越多的应用和服务专门为在浏览器上运行而优化。但是很多时候，随着我们的使用，我们往往会无意识地打开越来越多的标签页。

![](.evernote-content/B094B9DD-85F8-44C2-9D12-04BB80378FBB/F75A82A7-A363-49F0-ADE2-D95FCC33DA46.png)

在十余个凌乱的标签页中很难找到我们想要的那个

面对着数十个标签页，我们第一反应往往都是去一个一个把没有用的标签页关闭。不耐烦的时候，我们甚至可能会直接打开新的窗口重新访问我们想要的网站。实际上管理标签页非常简单，在桌面端浏览器，比如 Chrome 上，我们通常会有诸如 [Toby](https://sspai.com/post/44105) 的插件来协助我们管理众多凌乱的标签页。但是在 iPad 等移动端上，这些方法往往就行不通了。

今天我要介绍的是最近一个浏览器界的新起之秀 —— Space Browser。主打「工作空间」设计概念的 Space Browser 能够帮助我们更为优雅地管理我们凌乱的浏览器标签页。

工作空间，告别凌乱标签页
------------

Space Browser 由两位年仅 20 岁的开发者利用一年时间开发，是一款为了方便我们整理 iPad 浏览器中繁多的标签页而设计的浏览器工具。

Space Browser 巧妙地在 iPad 等移动设备上面提出了「工作空间」的概念 —— 一个「工作空间」对应一组标签页、一页个人收藏以及一段浏览历史等等，在两个不同的「工作空间」里面，我们就像是在用两个浏览器一样，互不干扰，却能沟通必要的信息。

![](.evernote-content/B094B9DD-85F8-44C2-9D12-04BB80378FBB/7729B099-A8F9-4A00-A4F2-E51029C6C54D.png)

Space Browser 主页面

Space Browser 的主界面主要有以下三个部分组成：

* History：历史记录
* Tabs：标签页集合（即当前工作空间概览）
* New Tab：新建标签页

### Tabs 分区：查看和管理当前空间的所有标签页

在 Tabs 分区，我们可以看到当前工作空间打开的全部标签页，这里是 Space Browser 的主界面，我们点击左上角「工作空间」名称或者直接单指右划即可看到我们的全部工作空间。

![](.evernote-content/B094B9DD-85F8-44C2-9D12-04BB80378FBB/04FB7DCA-3C7B-4872-971C-812BD94C77E2.png)

工作空间 - Workspace

在一个工作空间里面，我们可以任意地打开标签页，这些标签页全部归属于这一工作空间。与此同时，工作空间中的所有的浏览记录、书签、打开的标签页等等全部都被收纳在这一空间里面，不会影响到其它空间。

在这里，我将默认的两个工作空间设置为蓝色的 Home、红色的 Dev 并增加了一个黄色的 Design 作为我的三个工作空间。这样，我就可以将不同目的与功能的标签页加入不同的工作空间进行整理。

![](.evernote-content/B094B9DD-85F8-44C2-9D12-04BB80378FBB/81B68159-9AF3-4351-94F7-48A7ECC78353.jpg)

对工作空间的颜色、属性等配置进行设置

对于一个工作空间，我们也可以设置其名称、颜色以及浏览属性等等。比如，我们可以设定某一工作空间打开的标签页默认全部为隐身模式、或将工作空间固定在列表顶端方便找到等等。

### New Tab 分区：打开和浏览网页

在 New Tab 分区，我们可以打开新的标签页，进行正常的网页浏览。在这里 Space Browser 和我们默认的 Safari 浏览器并无太大区别，点击右上角就可以回到「Tabs - 当前工作空间」分区。

![](.evernote-content/B094B9DD-85F8-44C2-9D12-04BB80378FBB/5A0CF067-658D-4A41-ADB0-2E2AB2E61683.png)

新标签页 - New Tab

Space Browser 在一个工作空间里面浏览网页的时候，也可以非常方便地直接切换到另外一个工作空间里面。我们只需要点击左上角的「工作空间」图标即可选择需要切换到的工作空间。另外，你也可以长按「工作空间」图标并直接滑动来切换工作空间。

![](.evernote-content/B094B9DD-85F8-44C2-9D12-04BB80378FBB/71CE6A23-8829-4994-B844-A60685B4A975.jpg)

快速切换工作空间

有了 Space Browser「工作空间」的概念，我们就可以非常轻松地对我们几十个标签页进行归类管理，再也不需要担心找不到我们想要的网站了。

时间线：回溯浏览记录
----------

Space Browser 的另外一个非常独特的功能，就是它「时间线」形式的历史记录展现。

![](.evernote-content/B094B9DD-85F8-44C2-9D12-04BB80378FBB/FF164A20-B4C1-4B88-9576-4E6BD0C38B24.png)

时间线形式的历史记录

在 History 界面，我们可以看到当前工作空间里面的全部浏览历史，并且浏览历史是按照网站进行分类的。我们可以通过双指放大、收合的手势操作来更改时间线的时间间隔单位，从而以更细粒度的方式查看历史记录时间线。

Space Browser「时间线」形式的历史记录功能非常强大，我们可以在一个工作空间里面的浏览历史时间线里进行任意的「时间穿梭」。History 界面有一个蓝色的时间轴竖线，我们通过滑动这一时间轴，能够让 Space Browser 回到浏览历史里面的任意时间点。

专为 Power User 设计的实用功能
---------------------

除了上面提到的「工作空间」、「历史记录时间线」等等亮眼功能，Space Browser 还有许许多多非常有用的、专为 Power User 设计的小功能。

![](.evernote-content/B094B9DD-85F8-44C2-9D12-04BB80378FBB/F9DED23E-32FA-4764-8446-C6EF902D19A6.png)

额外的功能和设置项目

例如，在设置里面，Space Browser 为我们提供了下列几项功能：

* 自动请求桌面版本网页
* 内置 Adblocker 过滤网页广告
* 支持通过 [一些手段](https://medium.com/spacebrowser/how-to-set-the-default-browser-on-ios-6d8ebc9cc9b) 在 iOS 上将默认浏览器设置为 Space Browser

与此同时，Space Browser 还：

* 支持诸多手势操作、适配外接键盘多项快捷键
* 支持 Dark Mode 暗色模式

小结
--

Space Browser 着实为我们单调的 iOS 浏览器环境增添了一种全新的使用理念。「工作空间」的设计让我们能够自如地在不同的浏览场景下进行切换，完全不用担心被海量的标签页淹没；「时间线形式的历史记录」让我们能够更加清晰地回溯浏览历史。

如果你在 iPad 上面缺少一个生产力浏览器，或者你觉得默认的 Safari 浏览器不足以满足你对多页面管理的需求，那就试一试 Space Browser，看看它能否让你的工作效率更上一层楼。
