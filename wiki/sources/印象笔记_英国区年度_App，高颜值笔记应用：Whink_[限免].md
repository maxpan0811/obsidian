# 英国区年度 App，高颜值笔记应用：Whink [限免]

---

[少数派 英国区年度 App，高颜值笔记应用：Whink [限免]](http://sspai.com/32255)

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/0C868129-639C-4553-93DE-CAB1A9007B0F.jpg)

作为效率工具，大多数笔记应用在设计上都采用了相对简单直观的设计风格，把主要精力放在了提高用户效率，完善实用功能上。但对部分用户而言，**高颜值**才是提升生产力的第一要素。

作为笔记应用中的「新人」，Whink 以精致的设计、灵动的风格，在众多笔记应用中宛若一道清风，成为了令人耳目一新的存在。

设计风格
----

> Create Beautiful Notes Effortlessly （美丽笔记，轻松创作）

这是 Whink 官网上的第一句介绍。 「美丽」和「轻松」是 Whink 最大的亮点。不同于其他笔记应用，Whink 没有使用相对沉稳的界面颜色，而是使用活泼明亮的颜色，再加上扁平化的风格，希望给用户带来一种欢快的感觉。 

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/E4A4B786-5E41-4CBE-9A3C-5366846E2426.jpg)

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/6B4CB754-1706-4CC6-9A8B-AC4AC538A51F.jpg)

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/234EE6E6-42EC-45F3-9E45-5AC29AC230EB.jpg)

Whink 在界面设计上大量使用曲线和圆角矩形等几何元素，希望避免沉闷单调的感觉，在设计上让用户能够在使用时感到轻松和愉悦。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/2B27F7AB-5311-441A-98F8-6052E5AA53A9.jpg)

Whink 的独树一帜还体现在一些小细节上。例如，在需要切换模式时，只需要在工具栏上左右滑动便可，无需点击按键。又如，在进入笔记页面时，仔细观察可以发现工具栏有一个「弹」出来的动画效果。这些精致的细节，配合上漂亮的界面让用户在使用 Whink 时有一种与众不同的体验。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/70E5ED03-128C-4469-9899-FF79ED233D39.gif)

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/D4C66B4D-E399-4D1D-AA5A-6F44E7B230E5.gif)

书写模式
----

进入「书写」模式后，通过屏幕上方的功能菜单可以对一些常用的参数如笔迹粗细，颜色进行调整。橡皮擦，高亮工具，圈选等常见的功能也都一应俱全。通过最右边的「插入」键可以插入页面、图片、文本和标签。「书写」模式支持双指手势缩放和移动，用户在放大到合适的大小后便可开始手写输入了。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/B304FB5F-06AD-4EC9-AFBC-23CABCBBD92D.jpg)

Whink 同时也具备「缩放模式」，在此模式下写出的字比较整齐美观，适合大量文字的输入。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/80019E22-CBB4-4AB1-BC19-3736F9D24022.jpg)

购买内购后（18元），便能解锁形状识别和录音功能。前者能够帮助用户快速地在笔记中插入圆形，矩形和三角型等几何图形，后者则方便用户在笔记内的任何位置插入录音。建议有需求的用户购买。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/94948E0F-3AB2-46F6-AB80-85666256DA4A.jpg)

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/6354FF7E-197C-4523-855B-71EEE0781D1D.jpg)

不足的是， Whink 的防误触功能和同类优秀应用还有差距。手掌放在屏幕上时，有一定机率会发生误触。建议用户通过点击屏幕左下方的按键打开掌托，并在设置中调整「书写风格」来降低误触概率。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/48F64952-B37F-43F9-8E34-E4EF1BCD4D0A.jpg)

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/5C5693F7-1845-45E6-8B60-FBA534A29BAD.jpg)

**需要注意的是**， 除 Apple Pencil 外，Whink 暂时没有支持其他的蓝牙手写笔。Adonit Jot Script 无法使用，而 Pencil by 53 和一般的手写笔效果没有区别。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/7D8EAA27-F735-45D2-AF87-E05C92199E0A.jpg)

输入模式
----

「书写」模式固然灵活，但有时我们还是需要使用键盘录入文字。Whink 的「输入」模式为用户提前设定好了几种不同的字体风格，利用这些预设，再配合上图片和文本框，用户就能轻松地制作出排版简洁，样式精致的图文混排笔记。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/116BF556-81F4-4711-BB2D-CC2791927129.jpg)

进入「输入」模式后，点击页面内任意位置便能进行键盘录入，并支持插入有序列表，无序列表和代办事项。  
![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/3A983825-55DF-44F0-BA5F-8D6F0FDF2A05.jpg)遗憾的是，Whink 的样式选择目前比较有限，用户不能对大小，字体，字号等进行自定义调整。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/6494DA8F-1433-4988-9D21-1FC8F4FE6BDE.jpg)

同时，一些功能体验上的细节还有待完善。例如，在插入图片后，键盘输入的内容会被图片盖住而手写输入的内容会出现在图片上方，需要用户自己按住图片打开「启用文字自动换行」。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/12745F94-F39F-4A52-A8A8-0E60C53A6C97.jpg)

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/94ECF795-14D4-4AF5-A1C3-5A30110E5BCD.jpg)

阅读模式
----

除了记笔记，Whink 还加入了「阅读」模式方便用户进行简单地阅读和标注工作。用户可以从其他应用中导入 PDF 文件到 Whink 笔记页面中然后进入「阅读」模式来阅览。和大部分同类应用一样，导入的 PDF 文件将会被压平变成图片，无法再进行内容复制或是搜索。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/E9D1362A-8734-4CCE-9CA7-5562B42CDB3D.jpg)

Whink 在「阅读」模式中配备了「高亮」工具，如果用户需要其他的功能，则需要进入「书写」模式。目前的「阅读」模式并不能完全代替 PDF 工具，更适合作为阅读器来使用。如果你需要静心阅读或是复习笔记，或许这样精简的功能更有助于集中精力，减少干扰，提高效率。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/B44A70E0-8021-4F16-A177-BFA3220286C1.jpg)

通用性支持，文件管理和同步
-------------

Whink 支持最新的 iPad Pro 和 Apple Pencil，并适配了 iOS 9 中新加入的「Split View」和「Slide Over」功能。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/0C3821DD-C090-4F30-8ECB-9647D4DFDD63.jpg)

Whink 另有 iPhone 版本。但由于屏幕大小原因，iPhone 版本更适合用来查看笔记或进行简单的键盘文字输入。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/04852E1F-EA2F-45E7-A63E-A786695DE841.jpg)「输入」模式

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/B2D6D9AD-C5C4-461E-B1AA-6120B68A05E6.jpg)「书写」模式

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/93DDE403-CD8B-4E41-8FF0-CF22954041E0.jpg)「阅读」模式

在文件管理方面，Whink 没有使用列表界面，而是采用了更加直观的方式。用户在主界面上可以看到所有的笔记本封面，在空白封面上双击或者点击「新建」便可建立新笔记本，点击单个封面便可进入笔记本，按住后可以将一个笔记本拖到另一个笔记本上建立文件夹来进行管理。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/C50420C2-01FA-4A94-B0D4-570B4C787792.jpg)

文件导出方面，Whink 支持 PDF 格式（无法保留录音文件）或 Whink 笔记格式（能保留录音文件，只能使用 Whink 打开）。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/8EF9F06B-7648-4E62-97E6-DC24F8272F0F.jpg)

Whink 支持 iCloud 同步。在第一次同步时，我遇到了笔记无法打开的 BUG（之后没有再遇到）。建议用户在第一次使用时，确认能够连接 iCloud 后再进行同步。

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/0CF974DB-016E-4159-860F-B2EE33C7C890.jpg)

![](.evernote-content/65E22E66-F80C-40FA-AE78-2E76D709AC50/251ECB74-6CFE-4884-9CBC-8B0A7B9E4797.jpg)

*第一次同步时遇到 BUG，打不开笔记*

总结
--

### 优点

* 清新的界面风格
* 精致的细节设计
* 轻松制作出漂亮的图文笔记
* 较为全面的功能

### 缺点

* 防误触功能和同类优秀应用相比还存在差距
* 字体样式选择有限，自定义功能较少
* 功能细节有待进一步优化

总的来看，如果你是个「效率党」，对功能有很高的要求，希望现在就将 Whink 当作主力笔记应用来使用，我想它可能在有些地方会让你不够满意。但是，如果你是个坚定不移的「颜值党」，想找一款笔记应用来简单地记录生活，制作漂亮的手账或是游记，或许 Whink 正是你所需要的那一个。
