# OS X新手技巧：手动同步Safari iCloud历史

---

[威锋网-首页- 最新RSS订阅 OS X新手技巧：手动同步Safari iCloud历史](http://www.feng.com/iPhone/news/2016-01-05/OS-X-novice-skills-manual-sync-Safari-up-college-history_635067.shtml)

[![](.evernote-content/17F6A5B6-973F-4FB9-8074-DA21B34FAC1E/3AD3CC97-F710-4DDE-B3D0-028F16FADFAE.jpg)](http://www.feng.com/iPhone/news/2016-01-05/OS-X-novice-skills-manual-sync-Safari-up-college-history_635067.shtml)

　威锋网讯，只要 Mac 设备和 iOS 设备使用同一个 Apple ID，iCloud 就可以帮你在这些设备之间同步 Safari 历史。不过这一切均为自动发生，有时候 Mac 或者 iOS 任一设备可能会处于离线状态，此时 Safari 历史就会处于不同步的状态，在这样的情况下，你可以等待 iCloud 自动同步，也可以通过今天向你介绍的这个方法：强制手动同步 Safari（iCloud）历史。

![](.evernote-content/17F6A5B6-973F-4FB9-8074-DA21B34FAC1E/4574B556-A40B-4C8C-97B3-737DD0AEE3FB.jpg)

  
　本文针对新手用户，资深用户可忽略。想要强制手动同步 Safari（iCloud）历史，首先要开启 Mac 上 Safari 的隐藏 Debug 菜单。  
  
　要开启 Debug 菜单，首先退出 Safari，然后运行终端，输入：  
  
　defaults write com.apple.Safari IncludeInternalDebugMenu 1  
  
　最后点击回车，并重新运行 Safari 即可开启 Debug 菜单。  
  

![](.evernote-content/17F6A5B6-973F-4FB9-8074-DA21B34FAC1E/1774DD2A-456B-4891-AE42-4A34D82A911A.png)

  
　然后在 Safari 的菜单栏中找到 Debug 一栏，并选择“Sync iCloud History”。  
  
　完成上述的设置后，等待 1-2 分钟后所有和同一 iCloud 关联的设备便会同步 Safari 的历史。本教程的重点在于开启 Safari 的 Debug 菜单，请大家多加注意。
