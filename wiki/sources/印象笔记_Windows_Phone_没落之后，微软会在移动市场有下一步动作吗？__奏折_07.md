### 从 CES 2019 看今年科技行业发展

**@Fairyex：**就在本月，[CES（国际消费类电子产品展览会 International Consumer Electronics Show） 2019](https://www.ces.tech/) 正式在美国拉斯维加斯圆满落幕。作为全球最大的消费级电子技术展示与交流展会，CES 几乎每年都会吸引全行业顶尖的公司和组织参展，在这里我们不仅可以看到各种创新性与突破性的技术，还能一窥今年科技行业的发展趋势（这也是 CES 都选择在年初举办的原因）。

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/FB6CFEE0-B4C3-4B47-83D8-A1713DB5EF55.jpg)

在 CES 上宣布的重大创新（CES 官网）

#### 各家公司的无人自动驾驶技术

今年最大的主角非近几年风头无两的自动驾驶技术莫属了，即使没去到会场，也能体会到它的魅力 —— 俄罗斯搜索引擎巨头 Yandex 研发的自动驾驶汽车成功地拿到了在内华达州的驾驶许可证，在 CES 期间作为出租者为游客提供出行服务 1 。

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/A3FBB220-8B5C-4D66-9E9B-243B0D6127AE.gif)

Yandex 自动驾驶出租车（官方介绍）

Yandex 很巧妙地将自动驾驶作为出租车使用，驾驶座完全无人，只在副驾驶配置了应对紧急情况的安全员。并且中控台上的新 iPad Pro 会实时显示车辆看到的信息，还可以将其拿下来给任意乘客观看。从各大媒体的试乘体验来看，Yandex 自动驾驶出租车已经足以应付路线上的所有情况，实时显示的计算机视觉信息也显得车辆更为可靠。最重要的是 Yandex 给自动驾驶技术快速普及做了一个很好的示范 —— 既然现在的硬件和算法还不足以应付所有的道路情况，那么先做出租车或者公交车这类路线固定的公共车辆自动驾驶无疑是一条捷径：既能大大减少技术落地的复杂度，又能在最短时间内给大众带来对自动驾驶的信心和信任。

除此之外各家厂商，包括我们熟悉的宝马、奔驰等传统公司与高通、英特尔、百度等互联网公司都纷纷拿出了包括脑电波控制汽车，自动驾驶货车在内的自动驾驶产品。按照去年与今年展会的发展速度，只要解决了相关法规问题，L3 以上级别的自动驾驶将会在几年内落地到千家万户。

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/A934A668-8354-42D3-87F9-6EC2A5375704.jpg)

自动驾驶等级（SAE International）

#### 民用级 8K 分辨率与可折叠屏幕

会场内，最「抢眼的 」莫过于各家形态各异的显示产品了。其中 **8K、****可折叠**是今年的两个重头戏。

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/EACCE484-6F03-495B-B45F-54206C85E6A2.gif)

LG 8K OLED TV（Digital Trends）

LG，三星，索尼，TCL 等电视厂商纷纷发布了旗舰级 8K 电视，四倍于 4K 分辨率的每帧 3300 万像素以每秒 60 帧的帧率让所有图像的内容都不在像是显示出来的 —— 而是天然存在于那里一般。虽然现在购买 8K 电视的用户能看到的最佳内容可能只有 MKBHD2  的视频，但是可以预见这一大波厂商的发力会在今年引发主流内容商的跟进，很快内容匮乏也不再是 8K 设备的短板了，要知道 **4K 分辨率的普及也是在 2015 年才开始的**。

另外两个非常具有未来感的产品分别是 LG 发布的卷轴电视与[柔宇科技](https://www.royole.com/cn/index.php?route=product/royole/shijue)发布的折叠式手机屏幕。

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/137A2699-2CB4-4153-9BF4-5E2FCEA4A211.gif)

LG Rollable TV（MKBHD）

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/E30C26C9-A063-49D6-883A-87544C7CF573.gif)

Royole 柔宇科技（CNET）

除了炫酷的未来科技感，这项技术给我们日常生活带来的改变也是颠覆性的。比如柔宇科技就展示了他们给各种日常用品都加上了 OLED 屏幕 —— 皮包，台灯，水杯甚至开关。 OLED 轻薄、可变形、黑色不发光的特性与物联网设备简直就是绝配，利用这项技术，用不了多久我们身边就会「无物不可显示」了。最重要的是，**以上技术全部都是消费者现在就可以购买到的，不是普通人可望不可及的概念性技术。**

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/9DE5A7C1-4EAF-4BC8-874D-4CE9C0AD92AA.jpg)

#### 人工智能与 AR 结合

每一年 CES 上都会有各家厂商在 AR 和 VR 领域上各显神通，不过硬件的局限导致这两项技术年年都处于「要火不火」的状态。今年，各家厂商不约而同地都将自家地技术和产品同大热的人工智能（机器学习）联系起来，希望通过机器学习算法来降低实现 AR/VR 效果的硬件要求，在 CES 上我们能看到各家努力的成果。

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/F93ED517-247D-4D7F-8071-1283EA6E5409.gif)

超级自然的头发染色（[Syarif Abdullah](https://www.youtube.com/channel/UCSRMsLYcprtdXOcU2v34ylw)）

AR 涉及到的图像处理是机器学习的强项，所以我们看到即使在上一代 iPad 上也能实现如此自然精细的图像处理效果。软件与算法类的研究成果下放速度比硬件要快上许多，相信 2019 年更多达到完美标准的炫酷实用的 AR 应用 / 游戏会很快和我们见面。

### CleanShot：Setapp 新工具

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/05575E9C-0239-4766-8F07-591BA51B1495.png)

**CleanShot**

[官网链接](https://getcleanshot.com/)

#### 应用简介

CleanShot 是一款功能丰富的截图工具，常规的截图功能之外，支持屏幕录制、快速预览、图片标注、自定义壁纸、更好用的窗口截图等功能。[近期加入了 Setapp](https://setapp.com/apps/cleanshot)。

#### 它想帮你完成整套截图处理

**@沨沄极客：**我们先看一段 CleanShot 官网的 1 分钟小视频，效果做的很棒。

初次使用 CleanShot 时，它会请你去设置里关闭两个系统自带的截图快捷键，避免发生冲突，CleanShot 为了保持截图习惯，仍然采用 `Command + Shift + 3` 截全屏、`Command + Shift + 4` 截区域的快捷键设定（允许自定义）。

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/027C85CD-7269-4F43-8ACE-208C503BD47C.png)  

初次进入 CleanShot

有时候我们想在一个干净的桌面下截图，但是桌面上有很多图标、而且来回切换壁纸很麻烦。CleanShot 就解决了这个烦恼，按下快捷键时，桌面上的图标会被临时清空，让你在一个干净的桌面环境下截图。截图完毕后，屏幕的角落会出现快速预览界面，可以选择浏览、复制、保存、销毁。

屏幕录制方面，CleanShot 支持录制 MP4 和 GIF 两种格式，MP4 可以同时录制麦克风的声音（不支持录制电脑发出的声音），视频录制时也会临时隐藏桌面图标。不过在我实际测试时，直接录制 21 秒的全屏幕视频（3360 x 2100）文件有 84 MB，显然未经过压缩，后期使用时还需要做一些处理。

CleanShot 还支持简单的图片标注功能，把截图拖动到菜单栏的图标上就会出现标注界面。标注有自带阴影的标注、带描边的文字、马赛克、高亮等功能。

可以说，CleanShot 很好的补充了系统自带截图的一些缺点。

### Pixelmator Pro 支持分类管理图层

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/207F009A-02C1-4132-AF8A-C390FF6ED884.png)

**Pixelmator Pro 1.3（2019–01–18）**

[下载链接](https://www.pixelmator.com/pro/)

#### 应用简介

Pixelmator Pro 是一款 macOS 上的图片处理应用，完全为 Mac 电脑打造，先人一步引入机器学习、支持 Metal 引擎，把常用修图功能做得更为简单易用，并且在拥有强大功能的同时保持轻量的体积。

#### macOS 风格的图层管理

**@Minja**：我一直觉得 Pixelmator Pro 不适合做重度工作，一个原因就是它的图层管理曾经很基础。不过这个情况已经成为历史了，1.3 版本的 Pixelmator Pro 在图层管理上下了狠劲：

1. 支持为图层打标签
2. 支持按标签过滤图层
3. 支持按图片、形状、文字、RAW 或者组来过滤图层

第三点想必 Photoshop 用户很熟悉，可以说 Pixelmator Pro 这次是追平了，不再会因为图层过多而找不着北；前两项更新就更有 macOS 风格，允许我们像在 Finder 里打标签一样管理图层，可以让我们把一些重点图层挑出来。

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/6F61B7E6-8C62-4F61-A428-C57A08FC0491.png)

给图层打标签

一些图层较多的图，以前我会放到 Sketch 里面画（有个批量重命名插件比较好用），现在用 Pixelmator Pro 感觉会更方便，至少在图层管理上已经没什么好挑剔的了。

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/15E410B6-7C58-46DE-8677-7CADDDE47AE4.png)

过滤图层

同样在图层方面，Pixelmator Pro 这回还强化了蒙板（Mask）的功能。蒙板对于很多新手来说都很神秘，特别是 Photoshop 里那一串「图层蒙板」「矢量蒙板」「剪切蒙板」…… 其实蒙板就是个镂空层，把一张带空的纸叠在另一张上面，部分图案会透出来，这就是最直观的蒙板了。

高阶用法当然避不开复杂蒙板，但日常使用中，我们更需要这种「两张纸叠一块儿」的简单蒙板。在 Pixelmator Pro 这里，蒙板又回归了「所见即所得」，任何图层都可以当模板，不用管它是图片还是组，看到什么，「透出来」的就是什么。

最后，Pixelmator Pro 还加入了**快速图层样式**。这个操作有点进阶，这里就用一个例子带过：画插画时可以画两个图层，一个涂彩色，一个用黑白抹出浓淡，两张通过特殊的图层样式叠一起，就混合出一张调好颜色深浅的插画。

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/820A3958-9FAF-4C6F-BE81-A861D36D2583.png)

图层样式的用法示例 插画 / Minja

### Google 发布 Play 商店应用新禁令

**@Fairyex：**就在几天前，Google 更新了其 Play 商店开发者政策中心的[受限内容部分](https://play.google.com/intl/zh-CN_ALL/about/restricted-content/)，添加了几项针对所有已上架和新上架应用的新限制并完善了现有限制的内容。

新增的项目为不允许应用宣传或销售未获批准的药物成分，也就是打击我们常说的「卖假药」。该项受限内容的说明部分基本覆盖了除 FDA （食品药品监督管理局 Food and Drug Administration）认证之外的所有中西药、保健品、各种宣称有治疗效果的安慰剂。如果 Google 严格执行该项政策的话，在打击「卖假药」力度上可以说是十分强力的了（对比同类搜索引擎和平台）。

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/7FB90EEC-7EB3-4F03-A852-72F8FB123E7F.jpg)

未获批准的药物成分

另外，Google 对受限内容中的「敏感事件」与「欺凌与骚扰」做出了更为准确全面的定义修改。添加「**否认发生过重大悲剧事件的客观事实**」和「**欺凌国籍冲突或宗教冲突的受害者**」等规定。

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/CA3BEA3B-4A26-4FB3-9A3B-23C1F1B03273.jpg)

敏感事件

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/7949FB40-0849-4308-A1B8-2E71FF222260.jpg)

欺凌与骚扰

而对于检测到有违反上面 Play 商店开发者政策的已上架应用，Google 采取的方法是**先下架再通知**，这也保证了用户能够接触到有害应用的概率更加地小。最近 Google 正在加速收紧对自家平台上的内容管控，上周奏折提到的下架申请短信与通信录权限应用、本周修改的受限内容以及未来即将实行的几个政策都不是无的放矢，背后的推动力是针对 Google 不断提升的政府阻力。

在关于这次修改开发者政策文章下的网民评论分成了两种声音：一种是觉得 Google 「管的有点太过了」，离自家提倡的自由、开放、开源的风格越来越远了；另一种自然是为之叫好，认为此举可以更好地提升 Play 商店上的应用质量，也能更好地保护用户。不过，以 Google 另一个重要的内容平台 YouTube 此前 [Elsagate（儿童邪典视频） 事件](https://en.wikipedia.org/wiki/Elsagate)中官方的回应 3 作为参考，在事件后也没有修改自己几乎相当于没有的审核机制，与 Play 商店受限内容相似的情况在 [YouTube 帮助中心](https://support.google.com/youtube/answer/2802268?hl=zh-Hans&ref_topic=2803176)中也几乎是针对观看者的「提示和建议」。

### 微软宣布中止对 Windows Phone 10 的支持

**@沨沄极客：**近日，[微软宣布将在今年底终止对 Windows Phone 10 的支持](https://cn.engadget.com/2019/01/18/microsoft-windows-10-mobile-support-ending/)，在 12 月 10 日之后，再也不会有任何软件或安全性更新了。

Windows Phone 10 从推出至今也才满 3 年时间，但如大家所见，几年间除了诺基亚以外，实在没有什么设备搭载这个系统。去年微软的 Joe Belfiore 在 Twitter 上回应用户时就曾明确表示过「Windows Phone 不再是微软关心的重点」，只会提供最低限度的安全更新。现在是时候说再见了。

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/CD83F938-D949-43CE-9812-DE51A76B793F.png)  

Joe Belfiore

其实导致这一情况的原因有许多：Windows Phone 进入市场太晚、更新节奏跟不上 iOS 和 Android 的脚步、没有吸引足够多的用户使用、没有开发者愿意开发 App…… 这些都是客观的外界因素，是导致 WP 失败的直接原因。

目前已经有不少人详细分析了 Windows Phone 没落的内外原因，这里就展望一下微软在移动市场的下一步动作。

在 WP 彻底停止支持之后，可以预见**微软会把更多的资源投入到 iOS 和 Android 应用的开发上**，因为随着 PC 市场的日益饱和，手机端的重要性不言而喻。微软一边说着 SurFace 才是生产力，一边又积极地提供 iPad 版本的 Office。这是因为微软清晰地看到了移动市场的未来，不可能彻底放弃。既然已经在 Windows Phone 上折戟，占了手机市场半边天的 iOS 更是一块丢不起的重要阵地。

而且**微软在 Windows 上也会做出更多与云端、手机端配合的功能**。云端已经有了 Cortana 和 OneDrive，手机端则在上一次 Windows 10 更新中新增了「我的手机」功能，可以将手机关联到电脑上。目前这项功能还不怎么完善，但将来一定会将诸如电话同步、短信同步之类的功能整合进 Windows。目前第三方已经推出了 [Dell Mobile Connect](https://sspai.com/post/43331) 这样的工具让 Windows 接到来自 iPhone 的电话和短信。微软也一定不会放弃这块巨大的蛋糕。

### 3 个简单实用的 Windows 的 CMD 命令

**@沨沄极客：**在图形界面（GUI）出现之前，人们只能通过命令行来完成任务，命令行的作用是「通过特定的命令和电脑进行沟通」。如今有了图形界面，我们可以在简单易懂的界面下操作，但并不意味着命令行就一无是处了。高效、快捷通常是命令行的代名词，一些特定的信息也只能通过命令行来查询。我选取了一些在 Windows 下的 CMD 命令，希望能在日常工作中给大家带来帮助。

按下 `Win + R` 快捷键呼出「运行」窗口，输入 cmd，就可以进入命令行窗口了。也可以在 Cortana 里直接搜索 cmd 来打开命令行窗口。

#### 1. 本机网络 IP 查询：`ipconfig`

这条命令可以查询电脑的 IP 地址，目前大多数电脑会连接到路由器上再转到公网，所以用 `ipconfig` 通常会返回路由器的内网 IP 地址，如果网络连接出现了错误，则会显示「已断开连接」。当你需要在内网使用文件传输协议时，都可以通过这条命令查询，快速找到本机 IP 。

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/689239E5-3C90-4BCD-8B11-E7458483D56B.png)  

ipconfig

除了查询之外，ipconfig 还有一些扩展功能，先输入 `ipconfig /release` 后，再输入 `ipconfig /renew`，就会强制让 Windows 请求一个新的内网 IP。也可以通过 `ipconfig /flushdns` 来刷新 DNS 地址。

当电脑出现一些难以解决的网络问题时，这些命令多少可以帮到你一点。

#### 2. 路由节点追踪：`pathping`

大家都知道我们可以通过 `ping` 命令来查询能否连接某个网站。这里介绍一个 `ping` 的高级版 `pathping`。

`pathping` 后输入一个 IP，可以查询到从本机访问这个 IP 的过程中经过了哪些节点。

当你访问一些国内不存在的网站时，你可以用它来查询是哪个节点出了问题。

![](.evernote-content/A25228E0-993F-4D37-8F82-72C7CB3373D1/9867B210-BC3D-4093-8327-452EABE39801.png)  

pathping

#### 3. 扫描系统文件：`sfc /scannow`

`SFC` 是 `System File Checker` 的简称，这条命令需要在管理员模式下执行。它可以用来修复系统中损坏和丢失的文件。这条命令会扫描所有的系统文件，运行时间会比较长。
