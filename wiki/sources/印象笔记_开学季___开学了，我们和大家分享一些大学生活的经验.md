开学季 | 开学了，我们和大家分享一些大学生活的经验
==========================

进入大学，你将迎来一段不同以往的校园生活。你可能要一个人来到新的城市，住进宿舍，认识新的朋友。你会有更多时间安排自己的学习、生活。你可以发展一门新的爱好，充实自己的生活，也可以努力提升自己，为未来做打算。

暑假结束，新学期开始，我们整理了一些大学生活可以用到的经验技巧，有学习计划和自我提升，也有宿舍好物推荐和生活技巧，希望你的大学生活可以过得更好更充实。

我们也整理了 30+ 款提高学习效率大学生可以用到的 App，涵盖学习生活各个方面  [上课了，我们给大家推荐一些提高学习效率的 App](https://sspai.com/post/56500)

学习的方式，并非只有去教室上课一种
-----------------

大学所学专业可能并不是你的兴趣所在，好在大学时间相对宽裕，可以自己安排学习计划，如果愿意，试试自学其它感兴趣的专业课程。通过 MOOC（幕课），你可以足不出户学习到全球各个大学的课程内容，不少 MOOC 还可以跟随课程内容提交作业，最终获得结业证书，比较有名的 MOOC 包括国外的 Coursera、Udacity 还有国内的网易公开课等等，大家可以根据自己的兴趣选择。

![](.evernote-content/ADC19892-FDB8-4899-851B-3F0F6BD5FDC6/B72D5EB5-1C76-4ADE-9C06-1C8A90AA262D.jpg)

知名的 MOOC 平台

其实，自学本身也是一项重要技能，没有了高中时学校和老师严格安排的学习计划，如何规划时间通过自学获得知识是非常值得思考的问题。如果你真的能在大学期间自学一门课程，相信获得的经验将远超课程本身。

养成良好的生活习惯，从第一天开始
----------------

好的习惯不是一天养成的，大学四年的时间正好提供了一个提升自我的好机会。如果你想比别人先行一步，不妨先从尝试培养这些好习惯开始：

**理财，从生活费开始**

* 建立独立核算思维，独立进行生活费用开支的核算，不管是学费、住宿费，还是电脑手机等大件，都应该一律视作自己的开支；
* 按照固定周期和固定金额向父母领取生活费，提高自己消费的计划意识；
* 培养记账的习惯，充分了解自己的每一分钱都花在哪里。

![](.evernote-content/ADC19892-FDB8-4899-851B-3F0F6BD5FDC6/45DEFDF1-8A15-4ADC-9760-701005C35667.jpg)

**固定作息，提高睡眠质量**

* 所有与睡眠无关的物品、活动，都不要带到床上；
* 控制光线，使用遮光的窗帘遮挡住自然光线；
* 睡觉前建立一套例行程序，建立良好的睡眠前心理暗示；
* 通过调整呼吸让自己更快入睡。

![](.evernote-content/ADC19892-FDB8-4899-851B-3F0F6BD5FDC6/60E9B83C-5DA8-49F7-B814-AF4E426552D9.jpg)

**经常运动，控制糖分摄入**

* 寻找自己感兴趣，并愿意长期坚持的运动方式；
* 跑步是一个不错的选择，但前提是掌握科学的跑法；
* 饮食和运动一样重要，可以先从控制饮食中的糖分开始；
* 想喝饮料的时候，不妨试试含有代糖的产品（比如零度可乐）。

![](.evernote-content/ADC19892-FDB8-4899-851B-3F0F6BD5FDC6/4A0A34FE-E8C3-4188-80A0-6ABA0764DE77.png)

在宿舍过得好一点
--------

宿舍是大学生最主要活动场所之一，作为每天生活、学习、打游戏的地方，你的生活质量和它息息相关。为了能让你过得更「爽」，我们特意为你整理了一些值得购买的「宿舍神器」：

**舒适整洁的宿舍书桌**

* 电源收纳盒、束线器、文档整理架是桌面收纳必备的三件套；
* 一台可以调节色温的台灯能保护双眼；
* 一只好用的笔，活页本和手写板则是你学习的好帮手。

**提升生活质量的小物**

* 一副隔音耳塞是睡眠质量的有效保障；
* 一瓶人工泪液可以缓解眼部疲劳；
* 一个烘鞋器则让你轻松应对潮湿天气。

**值得一试的数码产品**

* 机械键盘和显示器是玩游戏和经常敲字同学的必买好物；
* 一台好用路由器，记得选要通过校园拨号认证的型号；
* 一把电竞椅是一份对自己健康的合理投入；
* 一台打印机让你复习时不再需要去打印店排长龙。

![](.evernote-content/ADC19892-FDB8-4899-851B-3F0F6BD5FDC6/B07BFA76-E180-4C5A-BDA0-9857DFE8C38D.jpg)

选一台适合自己的笔记本电脑
-------------

对大学生来说，笔记本电脑是必备的大件之一。面对铺天盖地的安利和头晕目眩的参数，如何选购一台称心如意的好电脑？

简单来说，需求、配置和购买渠道是购买一台笔记本电脑最重要的三点。第一点是明确需求，「给我一台能打游戏的电脑」和「来一台玫瑰金的超薄本」所需要的电脑完全不同，既要满足专业需要，又得能胜任休闲娱乐需求，明确自己的需要才能选择合适的电脑。

第二点是了解主要配置。配置是最让人头疼的，各种参数是新手的噩梦，也是贩子圈钱的机会。其实你没必要摸清每一个细节，了解那些最主要的部件（比如硬盘、CPU 和显卡），知道它们的作用就够了。另外，接口是一个容易被忽视的点，Type-C 虽然好，但考虑到国内的实际情况，建议选择兼有传统 USB 接口的电脑。

第三点是选择正规购买渠道。最理想的渠道是官网，直营店和授权店也是质量靠谱的选择。另外，在电商网站你能找到更漂亮的价格。京东最为可靠，其次是天猫。而更便宜的某宝，水也更深，最不推荐的则是电脑城，玩弄术语、哄抬价格，这样的地方还是远离为好。

别错过只有大学生才能享受的福利
---------------

许多商家都会提供专门针对大学生的活动，除了「吃饭打折」和「景点门票优惠」这种常规优惠外，一些商家还会专门提供「教育优惠」服务。

每年秋季开学前后，很多硬件厂商都会推出学生福利，只需凭学生证就能用很便宜的价格买到笔记本电脑、平板等各类数码产品。其中比较有名的是苹果每年的 [新学期优惠](https://apple.pvxt.net/e5XMz) 和微软的 [开学季](https://www.microsoftstore.com.cn/student) 活动。

![](.evernote-content/ADC19892-FDB8-4899-851B-3F0F6BD5FDC6/82A154FD-9197-48E5-8A8F-23ED123DA09B.png)

除了数码硬件产品之外，不少软件应用服务也有专门提供给学生的折扣，例如，

* 苹果的 [Pro App 教育套装](https://www.apple.com/cn-k12/shop/product/BMGE2CH/A)
* 苹果的 [Apple Music](https://sspai.com/post/37121) 学生优惠
* 微软的 [Office 365](https://www.microsoftstore.com.cn/student) 优惠
* 阿里云「[云翼计划](https://promotion.aliyun.com/ntms/act/campus2018.html)」主机优惠
* 腾讯云「[云 + 校园](https://cloud.tencent.com/act/campus)」主机优惠
* GitHub 的 [学生开发工具包](https://education.github.com/pack)（内含 GitHub 高级账号、AWS 亚马逊云服务、优达学城等等）
* Sketch [学生优惠](https://www.sketch.com/store/edu/)
* Setapp [教育优惠](https://setapp.com/educational-discount)

另外，一些日常生活中经常使用的服务也会提供针对学生的福利。例如爱奇艺提供了包年 99 元的优惠价格购买原价 198 元的 VIP 会员服务，淘宝和京东都为通过认证的学生提供一些学生特价和专属优惠券，网易严选则为学生提供全场 9.5 折优惠等。
