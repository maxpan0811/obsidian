今日份的“活久见”：

微信读书（腾讯旗下的哦），竟然推出（代表数据开放的）api，推出了面向 agent 的官方 skills！！！

![](.evernote-content/356417E8-7F45-4DD5-99F4-3718A55629FB/BE9CC0D3-2DCF-43C3-8397-873D1C1D9DC2.png)

链接：https://weread.qq.com/r/weread-skills

安装方法很简单：在 codex 里面新建一个项目“weread-mcp”，复制粘贴下面这句话，回车发送即可。

![](.evernote-content/356417E8-7F45-4DD5-99F4-3718A55629FB/D2B9E2CD-0CEB-45B8-87B1-551A510C1AC4.png)

虽然但是，我尝试获取 api key，试了十几次，换了 chrome 和 safari，都拿不到 api key🤣

![](.evernote-content/356417E8-7F45-4DD5-99F4-3718A55629FB/3BEE1DC7-EEFC-4C50-A9A3-3B6E706414D4.png)

换到了 arc 浏览器，终于拿到 api key 了，可以玩起来啦。

（也就是说，如果你获取 api key 有问题的话，多试试不同的浏览器～）

### 天下大势，浩浩汤汤

ai 大势，浩浩汤汤，破天荒事情，竟然真的发生了。这些封闭产品，有一天真的开放的，真的让人不习惯啊。

（现在，我已经有一些奇怪的idea，准备跟codex聊一聊了🤣）
