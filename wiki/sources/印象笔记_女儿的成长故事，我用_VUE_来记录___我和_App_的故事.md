# 女儿的成长故事，我用 VUE 来记录 | 我和 App 的故事

---

我总感觉等女儿长大以后，她妈妈一定会一半幸福一半埋怨的对她说：“你出生的时候，你爸就知道拿着个手机对你拍拍拍！”

最初认识 VUE 是因为老婆怀孕的时候，为了不让远方的父母担心，我想把老婆每天的日常生活用短视频记录下来，然后分享在家庭微信群里，好让家人放心。我现在还记得初次使用 VUE 的情景，设置好视频时长和分段，选上春节特供的温暖节日滤镜，然后拍下老婆在厨房里忙碌的画面，配好片头贴纸和背景音乐，第一个短视频就以极其流畅的操作拍摄了下来！VUE 将一个视频的拍摄分为几个等长段落的拍摄大大简化了短视频即时剪辑的难度，我可以轻松地用画面的跳跃构造蒙太奇，拍出品质还不错的短视频。于是，初尝摄影乐趣的我开始有了拍摄宝宝出生的想法！

![](.evernote-content/91742A44-8564-4190-9423-214E0C6FEBA8/4AB2730A-B964-4F86-B0D8-662831443FB6.png)

女儿是在今年 3 月出生的，那时广州还有些许寒凉。也不知道是因为天气寒冷还是在妈妈肚子里待得太久的缘故，刚出产房时，她的小手小脚都已经变成了紫色。奶奶小心翼翼地把她从产房里抱出来，我就亦步亦趋跟在奶奶后面，同时轻轻地拍下她与这个世界第一次相遇的时刻。女儿很可爱，不管天气多冷，在奶奶怀里总是举着一只小手，让人既可笑又可气。在照顾家人的间隙，我用 VUE 拍下了她吃奶、睡觉、啼哭的画面，看着视频里还不能完全睁开眼睛的女儿，仿佛能感受到生命的热量！

接下来的几个月，伴随着女儿的成长，我拍摄的主题变得越发有趣味，从奶奶给她洗澡到妈妈带她去游泳，或者是一家人带她去游乐园 。朋友说我生命中最重要的三个女人总是出现在画面中！拍摄时，推拉摇移的基本技巧必不可少，但我也喜欢给女儿一个大大的特写，偶尔还会学习美国大片的拍摄技巧。比如《魔戒》中佛罗多和山姆总是画面的左边走向右边，而我在拍摄女儿去游园的时候， 一家人会从片头画面的左边进入代表到来， 从片尾画面的左边出去代表离开。有时，她似乎知道我在拍摄她，眼睛会直勾勾的盯着镜头，也会做出一些吐舌头的俏皮表情。最让人内心融化的瞬间，是她刚洗完澡，妈妈给她按摩的片段。画面中，在 VUE 附赠钢琴曲的伴奏下，妈妈的手温柔的划过她的小脸，女儿认真的看着母亲，嘴角轻轻在笑！

![](.evernote-content/91742A44-8564-4190-9423-214E0C6FEBA8/64A34638-3E9C-450E-A2A9-7A3C3989A15A.jpg)

我用 VUE 记录了女儿从出生到现在的十几段视频，在女儿一周岁的时候，我打算把这一年来所有的短视频全部剪辑在一起，做成一个特辑，然后每年重复做这件事。我想如果没有 VUE，我就只能拿着 DV 到处拍了，但我哪是这么勤劳的一个爸爸呀。也许等女儿长大以后，当她妈妈说完文章开头的那句话时，我也会对女儿说：“爸爸的镜头里，有你，有我，有我们一家人！”
