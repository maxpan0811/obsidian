# Linux 携手 edX 再开一门MOOC，帮用户成为Linux 系统管理员

---

非盈利开源操作系统Linux 和非盈利MOOC 平台edX 携手，为用户再开一门[Linux 系统管理课程](https://www.edx.org/course/linux-system-administration-essentials-linuxfoundationx-lfs201x)，售价499美元，在6月17号之前注册选课可获100美元折扣，这也是edX 上首门收费的Linux 课程。之前edX 曾上线免费[《Linux 导论》](https://www.edx.org/course/introduction-linux-linuxfoundationx-lfs101x-2)，至今已经吸引了超过40万学习者选课。

这门新课适用于已经完成《Linux 导论》学习的用户，和那些已经在IT领域工作但没有直接Linux 相关知识的用户。499美元还包括了“Linux系统管理员认证考试”的费用，以及一次重考机会。课程会于今年8月3号开始，开课后用户可以按照自己的时间安排随时上课和参加考试。在考试合格之后，用户应已具备了应征初级IT 职位的能力。

科技公司和在线教育平台结盟已经成为趋势。今年3月，[edX还与微软合作推出7门IT课程](http://36kr.com/p/220457.html)；上个星期，Google 在I/O大会上[宣布和另一大MOOC平台Udacity 合作](http://www.forbes.com/sites/ellenhuet/2015/05/28/google-udacity-android-nanodegree-developer-online-course/)，推出培养安卓开发者的6门课程。用户花费6-9个月，每星期10小时左右，即可学完全套课程。课程的内容全部免费，但用户可以选择支付200美元来获得增值服务，包括项目评分、反馈、导师指导、和结业证书。成绩前50%的学生会受邀访问Google 总部并和负责招聘的工程师会面，并有机会得到年薪10-18万美元的初级工程师工作。

Udacity 的定位一直是为职场人士提供更大上升空间，而[Coursera 也曾多次表示](http://36kr.com/p/532372.html?utm_source=site_search)，接下来会重点探索[职业教育中的商业可能性](http://36kr.com/p/531553.html?utm_source=site_search)。MOOC平台想走职业教育的道路，和科技巨头们合作确实是不错的切入点，前者需要给用户提供更强的学习驱动力，后者正面临巨大的技术人才缺口，双方都有很强的需求，且在线的模式，本身就很适合进行IT类课程教学。
