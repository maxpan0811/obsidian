# 微信读书Skill接入Agent

## 一句话
微信读书官方Skill可接入OpenClaw/Claude Code/Codex等Agent，两步安装后Agent能读取书架、划线、笔记，生成阅读画像。

## 安装方式
1. 复制安装指令到Agent
2. 获取API Key
3. Agent即可读取书架/划线/笔记/阅读统计

## 可用场景
- 分析阅读习惯和偏好主题
- 生成阅读画像信息图
- 将读书数据变成AI可调用的素材
