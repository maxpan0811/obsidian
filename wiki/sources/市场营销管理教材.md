> 2001年版市场营销管理PPT教材，作者wt，最后保存于2002年10月。聚焦渠道策略与分销管理。

## 核心主题：渠道策略与分销管理

### 产品——渠道适应性衡量尺度
| 维度 | 说明 |
|------|------|
| 产品定义 | 高定义（如铅笔）→ 最低成本渠道；低定义（如专业服务）→ 需解释的渠道 |
| 聚合性 | 独立出售物 vs 解决方案中的组成部分 |
| 排他性 | 排他性产品需排他性渠道（Gucci案例：扩大渠道→形象受损→削减后利润回升64%） |
| 客户教育 | 需要多大程度的客户教育和解释 |
| 替代性 | 产品的可替代程度 |
| 成熟度 | 产品生命周期阶段 |
| 客户风险 | 客户购买决策的风险等级 |
| 谈判 | 交易所需的谈判程度 |

### 渠道选择过程的次序
1. 分析谁是我们目前的客户（购买准则、购买状态、客户类型差异）
2. 选择和产品匹配的渠道（产品—渠道适应性衡量）
3. 选择赢利能力强的渠道
