# 《市场营销管理》教材

> 2001年版市场营销管理PPT教材，作者wt，最后保存于2002年10月。聚焦渠道策略与分销管理。

## 核心主题：渠道策略与分销管理

### 产品——渠道适应性衡量尺度
| 维度 | 说明 |
|------|------|
| 产品定义 | 高定义（如铅笔）→ 最低成本渠道；低定义（如专业服务）→ 需解释的渠道 |
| 聚合性 | 独立出售物 vs 解决方案中的组成部分 |
| 排他性 | 排他性产品需排他性渠道（Gucci案例：扩大渠道→形象受损→削减后利润回升64%） |
| 客户教育 | 需要多大程度的客户教育和解释 |
| 替代性 | 产品的可替代程度 |
| 成熟度 | 产品生命周期阶段 |
| 客户风险 | 客户购买决策的风险等级 |
| 谈判 | 交易所需的谈判程度 |

### 渠道选择过程的次序
1. 分析谁是我们目前的客户（购买准则、购买状态、客户类型差异）
2. 选择和产品匹配的渠道（产品—渠道适应性衡量）
3. 选择赢利能力强的渠道

### 分销密度选择
| 策略 | 描述 | 适用范围 |
|------|------|---------|
| 密集分销 | 通过尽可能多的批发零售商 | 便利品 |
| 选择性分销 | 少数经过精心挑选的中间商 | 选购品 |
| 独家分销 | 一定区域只选一家中间商 | 特殊品 |

### 渠道管理决策
- 渠道成员的选择
- 渠道成员的激励
- 渠道成员的评估

### 渠道整合
- 混合型销售渠道模式：多种渠道紧密结合，统一渠道体系
- 案例：Staples（直邮+呼叫中心）、IBM（SMB解决方案定制）
- 渠道移交管理：任务完成点的精确定义、移交程序定义

### 案例研究
- **海尔自建营销网络**：特许经营模式，利用品牌价值，产品多元化降低渠道成本
- **Gucci渠道策略**：扩大渠道→形象受损→削减渠道→利润回升64%
- **本田雅阁**：新建高端销售网络匹配豪华品牌定位
- **春兰/创维/美的/TCL**：自建网络 vs 利用现有商业系统的权衡
