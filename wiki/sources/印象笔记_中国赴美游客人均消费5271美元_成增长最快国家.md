# 中国赴美游客人均消费5271美元 成增长最快国家

---

|  |  |  |
| --- | --- | --- |
| |  |  | | --- | --- | | |  | | --- | | **中国赴美游客人均消费5271美元 成增长最快国家**  据美国《侨报》编译报道，美国商务部旅游观光办公室6月5日发布了2011年国际游客消费报告。报告的初步数据显示，中国大陆赴美游客去年消费总额57.43亿美元，排名第7，根据上述数据，中国大陆赴美游客人均消费约为5271美元，在前10个旅游市场中是人均消费最高的地区，其他国家的人均消费能力依次为日本、巴西、澳大利亚、德国、英国、韩国、法国、加拿大、墨西哥。 中国大陆从 2011年起已成为美国前10个旅游客源市场之一。上述机构3月份发布的国际游客报告显示，去年赴美大陆游客有1089405人，排名第9。这前10个市场依次为：加拿大、墨西哥、英国、日本、德国、巴西、法国、韩国、中国大陆、澳大利亚。中国大陆以36%的同比增速，成为美国旅游市场中增速最快的地区。 近两年来，中国大陆成为美国增长最快的旅游市场，也是美国最为看好的旅游市场。美国商务部称，在未来5年，中国大陆赴美人数预计增长率将达200%，在 2016年超过法国和德国赴美游客人数，这意味着对美国经济复苏贡献巨大，因此争夺中国出境旅游市场，成为奥巴马主推的“国家出口计划”的重要组成部分，该计划的目标是，在2015年将美国出口额翻倍。  在这一政策的指导下，美国商务部中国商务信息中心将于美东时间6月21日下午1:00-2:00召开网络研讨会，主题为如何吸引中国游客。 | | |
