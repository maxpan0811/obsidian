# 还在用 Excel 排项目计划？试试专业项目管理软件：OmniPlan

---

![](.evernote-content/1115824E-C152-4580-9422-50F49854ED26/7EE7185F-B781-44EB-8A78-B04520D7544F.jpg)

还记得在上学时每天会把要完成的功课写在便签纸的经历吗？随着智能设备的普及，这样的便签纸逐渐被高效智能的 [GTD](http://sspai.com/tag/gtd) 应用所取代。当涉及到管理调控多人合作的大型项目工程是，简化的 GTD 不再适用，你可能需要更加全面高效的工具。

本文将介绍 Mac 平台著名的专业项目管理软件 OmniPlan。

The Omni Group
--------------

[The Omni Group](https://www.omnigroup.com) 是一家从事于效率与生产力软件开发的美国公司，该公司共50多人却开发了多款 Mac & iOS 平台应用产品，本身就是一个高效能运作公司的代表。

![](.evernote-content/1115824E-C152-4580-9422-50F49854ED26/D6E9CE30-AE1A-4527-89DF-4C1E2152CAAA.jpg)

The Omni Group 主要开发了四款效率软件，不少读者应该对其旗下的 GTD 应用 [OmniFocus](http://sspai.com/tag/OmniFocus) 不会感到陌生，少数派也有 [OmniFocus 文章](http://sspai.com/tag/omnifocus)介绍。其实 OmniFocus 与 OmniPlan 是侧重点不同的同类产品，前者针对于个人用户的日常计划清单管理，后者应用于中小型多人合作的项目管理控制。

![](.evernote-content/1115824E-C152-4580-9422-50F49854ED26/4C821BC0-FA3F-47C7-B3EF-51D1DCC6E087.jpg)

与 Microsoft Project 相比，OmniPlan 的优缺点
------------------------------------

微软早在 1995 年就在 Windows 平台开发了项目管理软件 [Microsoft Project](https://products.office.com/zh-cn/Project/project-professional-desktop-software)，现已更新至 2016 版本。OmniPlan 开发较晚，与 Project 功能相近，却略有不同。

### 1. 美观友好的界面风格

列表清单是传统任务记录的方式，而甘特图则是项目管理专业的展示方式。Microsoft Project 采用左任务列表＋右甘特图的展示模式，非常直观地将与任务相关的所有信息展示出来。OmniPlan 也采用了这样的界面，同时提供了资源列表、日历展示等多种视图。从设计语言的角度上看，两者都继承了各自其他产品的风格。

![](.evernote-content/1115824E-C152-4580-9422-50F49854ED26/97B59EF6-1D13-4EC0-970F-7D8399D4F4D5.jpg)

*Microsoft Project*

![](.evernote-content/1115824E-C152-4580-9422-50F49854ED26/A50ABB2C-0334-4C49-BF21-0E550CEBF098.jpg)

*OmniPlan*

### 2. 高度的兼容性

作为后辈，OmniPlan 为了吸引用户，尽了最大的努力兼容 Microsoft Project，它可以直接打开 .mpp 文件（.mpp 文件为 Microsoft Project 生成的文件），也支持导出多种格式为文件，包括 .mpp 文件，但其 v3.2.1 版本导出的文件在 2013 版的 Project 中会出现任务完成度丢失、颜色配置消失的 bug，而 3 月初更新的 v3.3 版本则出现了导出失败的问题。

OmniPlan 的技术支持表示会修复这些问题，其官网展示页面的介绍表明应用已经支持 2016 版的 Project。

![](.evernote-content/1115824E-C152-4580-9422-50F49854ED26/1139D261-853A-4DAE-A25C-B0FC652FD7E2.jpg)

### 3. 高质量报表的生成与分享

项目管理的重要功能是生成报表用于项目控制分析。依赖于自家产品的支持，Project 生成的报表继承了 Excel 风格，图形化的数据显示非常专业。

![](.evernote-content/1115824E-C152-4580-9422-50F49854ED26/95A3B6F2-BD0E-4AFF-8035-78ADF860A451.jpg)

OmniPlan 则支持便捷生成列表化的数据报告，甚至将 Gantt Chart 也一并带出，并支持转为 .html 格式，**可以提供给未安装软件的上司或同事打开阅览**，非常直观方便。

![](.evernote-content/1115824E-C152-4580-9422-50F49854ED26/3AE2972C-2E53-4801-99A9-A2E34D8921DE.jpg)

对于多个项目的同时管理，OmniPlan 还推出仪表盘（Dashboard）功能，可以快速查看项目进展情况。但从目前效果上看，该功能仍有很大的改进空间。

![](.evernote-content/1115824E-C152-4580-9422-50F49854ED26/D776B2B9-54C8-4397-81A8-4D8FD7146111.jpg)

### 4. 多平台支持与同步协作

Omni Group 为自己的主打产品都做了移动便携设备的软件开发，OmniPlan 也不例外。但高昂的价格（¥488）和移动操作的局限，使得 OmniPlan iOS 版的用户体验并不太好，毕竟像这样专业生产力软件，还是应该在专业的操作系统上才能发挥优势。

OmniPlan 还为用户提供了**免费**的 [Omni Sync Server](https://manage.sync.omnigroup.com) 同步服务，任何用户都可以将项目管理文件上传至 Omni 服务器，便于团队成员间互相共同编辑与操作。

![](.evernote-content/1115824E-C152-4580-9422-50F49854ED26/84331064-71FA-4CDE-8F16-147A4BACD81C.jpg)

Microsoft Project 虽没有移动版本的加持，但是结合自家 Office 365 服务，同样开发了多人协作的 Web 简化版 [Project Online](https://products.office.com/zh-cn/Project/project-online-portfolio-management)。

OmniPlan 涉及的项目管理术语
------------------

### **1. 任务、摘要任务****、里程碑任务**

* **任务（Task）**是最项目管理中基本的单元，是具体可执行的内容；
* **摘要（Summary）**可以看作多个任务的集合、群组；
* **里程碑（Milestone）**是任务列表中重要的任务结点，如方案的验收通过、产品研发完成等。

![](.evernote-content/1115824E-C152-4580-9422-50F49854ED26/50073E7D-FACA-40AC-BBC7-E56A8B1FD03E.jpg)

在 OmniPlan 的任务列表可新建任务，并在右侧信息栏修改与该任务相关的所有信息。

![](.evernote-content/1115824E-C152-4580-9422-50F49854ED26/58D1AFD5-4224-4FCF-9C3C-795B37E18BA7.jpg)

### 2. 资源 Resources

**资源**主要是任务执行的相关人员，也包含相关设备、工具、素材等。

OmniPlan 提供了资源列表，可供用户编辑修改资源名称、调用成本等信息。

![](.evernote-content/1115824E-C152-4580-9422-50F49854ED26/535B3115-EB8F-47B1-A10F-8F1FCA86F4A6.jpg)

### 3. 任务相关性 Dependency

**任务相关性**指的是任务之间，开始 Start 与完成 Finish 的关系逻辑，如A任务完成后才可开始B任务。任务逻辑的存在，大大简化了调整任务节奏与排序的工作。

![](.evernote-content/1115824E-C152-4580-9422-50F49854ED26/78CF35AC-E0A6-4804-A58A-9D34657D9EBD.jpg)

### **4. 甘特图 Gantt Chart**

**甘特图** 是专业项目管理非常重要的部分，它按照时间、逻辑与进程这三大关系，将所有任务作图形化展示。

![](.evernote-content/1115824E-C152-4580-9422-50F49854ED26/C1D237E8-C557-4D14-AF7D-15099EBAA105.jpg)

### **5. 报表 Report**

**报表**会直观反映任务进度、资源、成本控制等内容。

小结
--

如果你的工作或学习涉及到进度管理、成本控制或风险评估等，小到科研课题，大到建筑工程，都可以尝试借助 OmniPlan 进行高效有序地管理。

最后要强调的是，**项目管理软件只不过是提升效率的工具，能否发挥出工具的效用，还是要取决于项目相关人员的管理能力与执行力**。

**限于文章篇幅和工具的专业性，**本文未对 OmniPlan 的使用作过多介绍，有兴趣和需求的读者可以在 OmniPlan 官网查阅 [使用手册](https://support.omnigroup.com/documentation/omniplan/mac/3.0/zh/) 和 [新功能视频介绍](https://www.omnigroup.com/video/omniplan/)。

本文内容适用于 OmniPlan v3.3.1，你可以在 App Store 和 Mac App Store 或 [官网](https://www.omnigroup.com/omniplan) 下载 OmniPlan。
