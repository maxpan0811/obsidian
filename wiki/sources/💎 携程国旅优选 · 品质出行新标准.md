# 携程国旅优选 · 品质出行新标准

> 携程国旅品牌升级方案。从"好产品"到"可信赖品牌"的系统升级。

## 品牌定位

| 项目 | 内容 |
|------|------|
| 品牌名称 | 携程国旅优选 |
| 品牌口号 | 「国旅认证 · 品质无忧」 |
| 品牌使命 | 让每一次出行，都值得信赖 |
| 目标客群 | 中高端家庭、品质出行客户、重体验型旅客 |
| 核心价值 | 权威背书 + 确定服务 + 甄选体验 |

## 三大升级体系

### 1. 产品升级：标准化 + 独家化
- **三大产品系列**：国旅臻选（高端小团）、国旅经典（热销明星）、国旅焕新（新兴目的地）
- **五统一标准**：统一定价、统一成团、统一服务、统一宣传、统一保障
- **一价全含**：明确无强制消费
- **透明化**：明示航司、酒店、餐饮标准

### 2. 服务升级：专业化 + 确定化
- 金牌导游 / 专属旅行管家
- 24小时行中服务团队
- 标准化行前行后服务流程

### 3. 品牌升级：口碑化 + 差异化
- 品牌视觉系统升级
- 携程+国旅双品牌背书
