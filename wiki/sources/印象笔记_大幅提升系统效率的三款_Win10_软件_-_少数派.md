大幅提升系统效率的三款 Win10 软件
====================

之前的视频中跟大家介绍了几个在 Win10 平台复现 Mac 操作习惯的几个应用。不得不说，在最近一段时间的体验下来，无论是操作习惯还是使用的效率上面。都给我带来了一定程度的的方便和提升。这期视频中，我再跟大家介绍几个，可以进一步大幅提升效率的软件。

>> 延伸阅读：[如何在 Windows 上还原 macOS 的使用习惯](https://sspai.com/post/58112)

全能截屏工具：Snipaste
---------------

在 Win10 平台本身内置了全局截图的功能，通过组合键就能快速完成全屏截图或窗口截图。但是这样的操作无疑的效率略显低下。更高效的截图，我建议使用 Snipaste 这个软件，完全免费的，在 [Snipaste 官网](https://zh.snipaste.com/) 就可以直接下载。那除了截屏它还能干什么呢？

>> 延伸阅读：

[Windows 上最好的免费截图标注工具：Snipaste](https://sspai.com/post/34962)

[他做了最强免费「截图」工具 Snipaste 后，还有上万字的话想说](https://sspai.com/post/35097)

### 快速抓取颜色信息

平时我们使用 PhotoShop 做图，色板中可能一下找不到对应的对于有些参考颜色。可以直接通过快捷键激活 Snipaste，然后将鼠标移动到需要获取的色彩。这时就能看到「按 C 复制颜色值」，再将色彩信息粘贴到 PS 拾色器中，就完成了色彩的选择。

![](.evernote-content/22BC9CA1-3780-4E7D-AB08-83404E202E40/1D0ED943-37D0-4498-BB44-8D387C9D0102.png)

快速截取窗口

Snipaste 的检测识别能力很强，开启「自动检测窗口」之后可以快速截取制定程序窗口，而无需手动拖拽。

![](.evernote-content/22BC9CA1-3780-4E7D-AB08-83404E202E40/4FE86868-6FC5-4290-9AE1-8EEC017E8A0B.png)

快速编辑

截图成功之后，我们可以在下方的操作栏快速选择想要进行的操作，而无需开启另外的程序。比如可以加边框和箭头，还能通过鼠标的滚轮调整线条粗细程度。

![](.evernote-content/22BC9CA1-3780-4E7D-AB08-83404E202E40/4B69A69A-94FD-4D3D-A62D-676F7BB2F938.png)

多线程下载管理工具：XDM
-------------

XDM 是一个非常强大的下载工具，通过配合浏览器插件可以非常轻松的下载各种网页中的内容，包括一些可以播放的视频、音频。打开 [XDM 的官网](http://xdman.sourceforge.net/)，点击下载，就可以选择适合你电脑的版本下载使用了。软件自带中文，我们可以在软件上方的「Tools」选择 language 切换，然后重启软件。

![](.evernote-content/22BC9CA1-3780-4E7D-AB08-83404E202E40/B45D48CF-F12A-499F-B60A-F9D785ED7B4E.png)

### 安装浏览器扩展

点击设置按钮可以开启「浏览器监视」。我使用的是 Chrome 浏览器，所以在 Google Chrome 这里选择安装扩展。然后返回设置主页，在下方更多设置中，还可以勾选「下载中防止休眠和睡眠」，这样就可以安心挂机下载，不用担心电脑休眠而暂停了。

![](.evernote-content/22BC9CA1-3780-4E7D-AB08-83404E202E40/16329DEA-C7FC-47C5-94FC-8F6AC803F397.png)

### 下载网页视频

XDM 还能自动识别正在播放的网页视频文件。以 YouTube 为例，我们打开一个视频，就会发现屏幕右下角弹出了一个「DOWNLOAD VIDEO」的按钮，点击就可以看到目前正在播放的视频的各种分辨率。你可以选择想要的分辨率，还可以进一步选择文件格式，然后确认下载就可以了。

![](.evernote-content/22BC9CA1-3780-4E7D-AB08-83404E202E40/DEB28096-DB69-4C45-BA93-B3E5EDE4E85D.png)

这款软件对经常制作视频的我来说是非常有用的，它可以帮我快捷地下载好想要的音视频素材，让我的效率变的更高。

终极效率工具：Quicker
--------------

接下来要给大家推荐的一款软件，可以说是一款终极的效率工具了，那就是 Quicker。在我看来 Quicker 可以说是 Win10 平台效率工具的大集合，因为他可以实现的操作真的是太多了，比如快速启动、文件处理、脚本运行等等。这款软件可以说是我目前为止遇到过最好用的效率软件了，好用到让我足以舍弃其他很多很好用的效率软件。

### 全局控制区域

软件主界面上方为全局控制区域。不管你在什么界面，什么窗口下激活面板，上方的这些功能按钮都不会进行更换。所以在这里我就放置了一些我会经常使用到的功能，比如「打开我的电脑」、「在 Everything 搜索文件」、「翻译」等等。尤其是搜索文件和翻译，都不需要复制完文本再到相应的软件中粘贴，而是直接帮你进行了后两步的操作。

![](.evernote-content/22BC9CA1-3780-4E7D-AB08-83404E202E40/A96A0238-F141-4391-8B54-855B50FC7E0C.png)

### 自定义功能区域

下方则是针对不同软件进行自定义的功能区域，他会根据当前处于激活状态的程序窗口加载出不同的功能。比如我的浏览器就已经进行了自定义设置，当我在使用浏览器时中调出面板，就能看到「一键打开主页」或「下载内容」、「对当前网页进行长截图」等按钮。

![](.evernote-content/22BC9CA1-3780-4E7D-AB08-83404E202E40/9CACE9E2-F1A2-4F54-8D26-A004B3989404.png)

在 YouTube 视频界面，我也可以一键对正在播放的视频进行下载

![](.evernote-content/22BC9CA1-3780-4E7D-AB08-83404E202E40/5D2A7BDA-053F-4E8F-8C94-079FEF1D29A1.png)

它还有一个强大的工具箱，可以为不同的应用设置一系列的快捷动作，就像 iOS 的「捷径」。比如在 Word 中，你可以一键设置常用的字体样式，也可以设置一键输出为 PDF。

![](.evernote-content/22BC9CA1-3780-4E7D-AB08-83404E202E40/00C954DF-F3A0-4A2A-8BE2-8102017D2BCC.png)

### 自定义功能

在大多数情况下，下方区域显示的都是默认的界面，我们可以像对上方全局区域一样添加一些常用的功能在里面。这里就可以打开它的共享插件库，添加你想要的功能，不得不说，这个软件的共享插件库非常的丰富，足足有十几页，足以应付大多数使用场景了。

![](.evernote-content/22BC9CA1-3780-4E7D-AB08-83404E202E40/828F403A-4677-4DAF-8CC6-6150BBF63162.png)

当然，如果当你遇到想要的操作在工具箱里没有的，我们还可以自己创建一些常用的动作。比如我每次发邮箱都会在后边跟一句「来自电动派大星」我就可以选择键入文本，输入需要的内容然后点击保存就可以用了。

![](.evernote-content/22BC9CA1-3780-4E7D-AB08-83404E202E40/1BDBC617-FD8C-4978-9199-98B420CD3376.png)

### 手机控制

除了在电脑端直接操作，你还可以通过 Quicker 的手机客户端对电脑进行操作。只需要在官网下载手机版本的软件，扫描电脑端的二维码配对，就可以用手机控制电脑了。像是播放、暂停音乐，控制音量，向电脑发送图片等等。基本上只要是电脑上能实现的功能，通过手机都同样能够实现。

![](.evernote-content/22BC9CA1-3780-4E7D-AB08-83404E202E40/98023498-F5A1-4BFD-A23F-A646BB189A7A.png)

上面所说到的这些操作，都只是 Quicker 的一小部分功能。它的强大之处并不仅限于此，至于更多的玩法，还是需要你们自己去尝试。如果你对 Quicker 还有更多好玩的用法。欢迎在下方留言与我交流。我是电动派大星，我们下期视频再见。
