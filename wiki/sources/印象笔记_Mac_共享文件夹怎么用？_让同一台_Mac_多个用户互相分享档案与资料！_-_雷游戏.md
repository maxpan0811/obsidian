Mac 共享文件夹怎么用？ 让同一台 Mac 多个用户互相分享档案与资料！
=====================================

[IGN](https://www.leiyouxi.com/user/31)
•

2023年4月28日 14:17

•[投稿](https://www.leiyouxi.com/y/sort/contribution)

如果你家里有 Mac 电脑，而且会有两人（或以上）共享，那我们通常会在 Mac 里新增多位用户账户，家人要使用时都能各自登入自己的用户帐户，里面的档案与文件也不会混在一起。 但在某些情况下，你可能会想将档案共享给其他 Mac 用户，那我们是否能将 Mac 共享文件夹给同一台 Mac 的多位用户账户呢？

当然是可以的！ 目前有三个比较常见的方法，第一个是将想互相分享的档案放到「**共享**」这个 macOS 系统内建的文件夹里让所有用户都能存取，第二个方法则是放到「**公用**」文件夹里，第三个方法则是放到「投递箱」里，三者稍微有点不同，但都能让同一台 Mac 的多个用户共同存取档案与资料。

[![](.evernote-content/5A72DB60-7817-45F0-8D0B-3E9163C05844/A2EF93A3-6AC4-4FD1-964F-03C950DBA255.png)](https://www.leiyouxi.com/wp-content/uploads/2023/04/853y0.png)

那我们要如何让同一台 Mac 上的所有用户之间共享文件夹呢？ 请参考下方两种方法。

### 方法一：使用 Mac「共享」文件夹

每一个 macOS 系统里都有内置一个「**共享**」文件夹，让同一台 Mac 上的所有用户帐户都能存取，假设我的 Mac 上有 A 和 B 两个用户帐户，那 A 和 B 都能任意将资料放到「共享」文件夹里，也都有权限可以查看、编辑或删除「共享」文件夹里的内容。

那我们要怎么找到 macOS 系统内建的“**共享**”文件夹呢？ 很简单，步骤如下：

1. 开启 Finder 应用程序后，点击菜单列顶端的「**前往**」>「**前往档案夹**」。[![](.evernote-content/5A72DB60-7817-45F0-8D0B-3E9163C05844/871D0BFB-6D3D-457F-A54E-FDBDF1D6A157.png)](https://www.leiyouxi.com/wp-content/uploads/2023/04/gmksq.png)
2. 在空白搜索框里输入「**/用户/共享**」。[![](.evernote-content/5A72DB60-7817-45F0-8D0B-3E9163C05844/6E1661EA-C3BE-4327-8210-0D9EC0A874D4.png)](https://www.leiyouxi.com/wp-content/uploads/2023/04/nc0oy.png)
3. 双击或按 Enter 打开此文件夹，在「**共享**」文件夹中，就是这台 Mac 上的所有用户之间可以互相共享的内容，任何一位用户都能查看、储存、删除或编辑资料，所有变动也都会自动同步。 例如我在这里新增一个HELLO文件夹，那么此 Mac 上的所有用户登录其帐号时都能在这里看到 HELLO 文件夹。[![](.evernote-content/5A72DB60-7817-45F0-8D0B-3E9163C05844/ACC45B38-2BEC-4C20-8770-8C92765B0431.png)](https://www.leiyouxi.com/wp-content/uploads/2023/04/cm3l1.png)

此外，如果想更快速地存取「**共享**」文件夹，那也能在 Finder 里，点选左侧「**Macintosh HD**」，点选「**用户**」，这里也会出现 Mac 共享文件夹。

[![](.evernote-content/5A72DB60-7817-45F0-8D0B-3E9163C05844/D3F8B91C-927E-4489-9375-932246074D39.png)](https://www.leiyouxi.com/wp-content/uploads/2023/04/lwsck.png)

[![](.evernote-content/5A72DB60-7817-45F0-8D0B-3E9163C05844/8239AAE5-7BE1-4A98-80E2-183C1CA9AC96.png)](https://www.leiyouxi.com/wp-content/uploads/2023/04/3oauk.png)

### 方法二：使用 Mac「公用」文件夹

而第二个 Mac 共享文件夹方法则是使用 macOS 内建的「公用」文件夹，「公用」文件夹跟上述介绍的「共享」文件夹其实功能很像，但不同的地方在于，**此台 Mac 上的每一个 Mac 用户帐户都各自有一个「公用」文件夹**，因此我们可以将自己想分享给其他本机 Mac 用户查看或使用的档案与资料放在「公用」 文件夹里，供其他人取用。

使用 Mac 公用文件夹的方法如下：

1. 开启 Finder 应用程序后，点击菜单列顶端的「**前往**」>「**前往档案夹**」。[![](.evernote-content/5A72DB60-7817-45F0-8D0B-3E9163C05844/871D0BFB-6D3D-457F-A54E-FDBDF1D6A157.png)](https://www.leiyouxi.com/wp-content/uploads/2023/04/gmksq.png)
2. 在空白搜索框里输入「**/用户**」。[![](.evernote-content/5A72DB60-7817-45F0-8D0B-3E9163C05844/A149CB4C-74B9-4D19-BB17-4BE6125B9F25.png)](https://www.leiyouxi.com/wp-content/uploads/2023/04/c1sj7.png)
3. 在这边点进你自己的用户账户名称。[![](.evernote-content/5A72DB60-7817-45F0-8D0B-3E9163C05844/A79FB176-3918-4E53-B4EE-282F2364B60E.png)](https://www.leiyouxi.com/wp-content/uploads/2023/04/vcm9g.png)
4. 点进去「**公用**」文件夹。[![](.evernote-content/5A72DB60-7817-45F0-8D0B-3E9163C05844/1751AD52-47D2-4707-8EDC-DA44DDAAA746.png)](https://www.leiyouxi.com/wp-content/uploads/2023/04/c4zl3.png)
5. 把你想共享给其他 Mac 本机用户自由取用的文件或资料放在这里面，并告知他们可以到你的用户帐户里的「公用」文件夹访问。[![](.evernote-content/5A72DB60-7817-45F0-8D0B-3E9163C05844/840C2A5C-8A09-4FE5-8CFF-C4EAE5344708.png)](https://www.leiyouxi.com/wp-content/uploads/2023/04/1682691476.png)

### 方法三：使用 Mac 公用文件夹里的「投递箱」功能

第三个方法，如果你只想和此台 Mac 上的某一用户分享某些档案或资料，例如这台 Mac 本机上有 A、B、C 三个用户帐户，但 A 只想和 C 共享某些档案，那就很适合使用 Mac 公用文件夹里的「**投递箱**」功能，A 可以将任何文件拖曳到 C 的「投递箱」里，而只有 C 自己可以看到投递箱里的文件有哪些。

Mac 投递箱的使用方法很简单，步骤如下：

1. 开启 Finder 应用程序后，点击菜单列顶端的「**前往**」>「**前往档案夹**」。[![](.evernote-content/5A72DB60-7817-45F0-8D0B-3E9163C05844/871D0BFB-6D3D-457F-A54E-FDBDF1D6A157.png)](https://www.leiyouxi.com/wp-content/uploads/2023/04/gmksq.png)
2. 在空白搜索框里输入「**/用户**」。[![](.evernote-content/5A72DB60-7817-45F0-8D0B-3E9163C05844/A149CB4C-74B9-4D19-BB17-4BE6125B9F25.png)](https://www.leiyouxi.com/wp-content/uploads/2023/04/c1sj7.png)
3. 点进你想分享文件给哪一位 Mac 用户的该帐户，例如我想将文件分享给 Esther，那我就点进 Esther 这个用户里，点进「**公用**」文件夹。[![](.evernote-content/5A72DB60-7817-45F0-8D0B-3E9163C05844/1751AD52-47D2-4707-8EDC-DA44DDAAA746.png)](https://www.leiyouxi.com/wp-content/uploads/2023/04/c4zl3.png)
4. 将你想分享或传给他的档案直接拖曳到他的「**投递箱**」里面，你会看到它跳出窗口跟你说「你可以将项目放入投递箱，但你看不到这些项目」的提示，这是因为只有自己才能打开投递箱，你无法打开其他 Mac 用户帐号的投递箱唷！[![](.evernote-content/5A72DB60-7817-45F0-8D0B-3E9163C05844/BBED7249-2B42-47C9-A7FB-66DF1F7CD0C5.png)](https://www.leiyouxi.com/wp-content/uploads/2023/04/1682691493.png)
5. 一旦对方在此台 Mac 登入他的用户帐号后，它只要依序在 Finder 里点进「用户」>「公用」>「投递箱」，就能看到你给他的文件，这样就成功透过「投递箱」实现比较隐密的 Mac 共享文件夹需求。
