OpenCode是什么？
------------

OpenCode是个开源的AI编码助手，官网：https://opencode.ai/zh

几个特点挺吸引人：

* 开源，不用担心突然关闭
* 不用梯子，直接用
* 可以接所有模型，包括国内模型
* 使用习惯上和Claude Code几乎一样
* ![](.evernote-content/C91CE3A0-376E-4A45-A04F-12D02FA14039/6F2FA871-F216-4506-BC61-2660027A647B.png)

用起来怎么样？
-------

今天用它帮项目组做实际工作：操作浏览器，下载大量系统附件。

体验上和Claude Code差不多，习惯不用改。

配置上稍微麻烦点，但如果只是想让它变强大，其实也简单：

**两步就够了：**

1. 添加自己用的模型
2. 安装两个插件：

* oh-my-opencode-slim  
    
  https://github.com/alvinunreal/oh-my-opencode-slim
* superpowers  
    
  https://github.com/obra/superpowers

装完这两个插件，OpenCode就很强大了，对普通人来说够用。

迁移成本
----

基本没有。

Claude Code之前的所有配置，OpenCode都直接兼容。

不用重新学习，不用重新配置，直接能用。

我会继续测试几天，看够不够用。

如果没问题，就全力用OpenCode。
