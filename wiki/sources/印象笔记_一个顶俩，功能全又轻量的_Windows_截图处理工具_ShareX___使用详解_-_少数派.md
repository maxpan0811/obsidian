一个顶俩，功能全又轻量的 Windows 截图处理工具 ShareX
==================================

Minja 曾在 macOS 的图片处理方面下了不少功夫，利用 LaunchBar、Automator 等工具实现了许多有意思的功能——《[6 个小工具，打造图片批处理工作流 | 工作日志](https://sspai.com/post/56037)》、《[macOS 图片批处理工作流（第二弹）](https://sspai.com/post/53079)》。其中提到了拼接图片、给图片加阴影、批量调整大小、制作圆角卡片等操作，能让图片的观感上一个台阶。

我们发现，像是加阴影、拼图片这些操作在 macOS 和 iOS 上已经有不少现有工具可以实现，那么这些做法在 Windows 上有没有方便的解决方案呢？显然是有的。

但如何选用工具长期存在争议，比如画图、QQ 截图、Snipaste 都很轻便，能承担一些图片编辑工作，但像拼接图片、标注序号等工作并不容易完成；再比如 Ashampoo、Photoshop 这些重量级工具，功能是全了，但每次想批量加一个水印又大动干戈，未免过于臃肿。**想恰到好处地完成图片截取、轻量编辑这件小事，并不简单**。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/D3F0DEFF-2156-4BB6-97F9-A46EF6387F6D.png)

ShareX

因此，[ShareX](https://getsharex.com/) 出现在了我的桌面上。它能搞定截图，也能处理图片；足够轻量，同时功能不少，还是一款免费无广告的开源工具。本文会提到一些 ShareX 中的基础操作，也会提到自动上传图床、生成截图阴影、多功能串联等更强大的功能。

基础功能：截图及标注
----------

在 ShareX 的[首页](https://getsharex.com/)上，放出了所有功能一览。ShareX 将它的功能特点大致分为了五个部分，分别是「截图、标注、上传、第三方图床、生产力工具」，其中前三项也是它的主要功能。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/4EB9047C-CF24-4F27-9FD4-CA1450F80511.png)

ShareX 官网

我粗略地给截图工具划分了三个等级——

1. **基础工具**：具备基础的方框截图功能，提供简单的标记功能，支持有限的编辑功能（可以撤销但无法二次编辑）。稍好一些的工具可能会提供线条粗细调整、颜色调整。它能满足普通人对截图工具的需求。代表工具是 QQ 截图、微信截图等社交软件自带的截图工具。

   ![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/B559FA2D-A1FF-49F5-8D54-F45B7184FB9D.png)

   QQ 截图
2. **专用截图工具**：具备各种针对截图优化的功能（延时截图、截图区域自适应、多形状截图、WinSnap 的截图阴影等），提供较全面的标记功能（马赛克、高亮、文字阴影等），具备一些特定的附加功能（Snipaste 的贴图功能）。

   ![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/543581A7-7BC9-41D1-9CD1-E8A0D51AF58D.png)

   Snipaste 截图
3. **全功能工具**：提供全面的截图功能（OCR、滚动截图、自定义形状截图等），提供全面的标记功能（数字标记、放大镜等），支持撤销、反复拖动等更强大的编辑功能。

   ![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/731C155C-6A95-4DCF-A8DC-F823BD6CFB18.png)

   ShareX 截图

当然，在这三个级别之外，还有 Photoshop、Affinity Photos 等**专业图片编辑工具**。它们主要提供的是图片修改功能，当需要对图片中的内容进行修改时，小工具的功能再强大也很难超过专业工具的范畴，因此不在此处讨论。

在这三个级别中，功能数量逐级递增，能做的事情越来越多。如果按这一标准划分，**ShareX 已经达到了截图功能中的「全功能工具」这一级别，能够实现上面提到的所有功能**，照理来说它的体积应该相当的大，结果完整安装包也只有不到 6 MB，只有当你需要 GIF 录制等额外功能时才需要下载 FFmpeg 等扩展包。

### 修改快捷键

ShareX 的基础功能就是截图，可以通过快捷键和编辑界面触发，相信大家可以很快上手各种截图方式的基本操作方式。

在默认设置下的快捷键是 `Ctrl + PrintScreen` ，按键比较远。建议将最常用的「捕捉矩形区域」快捷键设置为你更习惯的截图方式，比如替换掉 `Ctrl + Alt + A`。另一个值得一用的是「捕捉活动窗口」快捷键，按下就可以快速截取当前窗口。录制 GIF 如果不常用，可以将其快捷键取消掉。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/9D81A30F-6DF0-4638-AC42-FD0514EC4811.png)

修改快捷键

在功能方面，ShareX 的汉化工作做得很不错，很多小功能和字面意思是一致的，下面只提几个值得一提的特殊操作。

### 滚动截图

我很少在截图工具中看到滚动截图功能，因为在滚动截图期间会**因用户的操作出现问题**、会**因 GIF 视频无法完美截图**、会**因为某些网页有固定的按钮而多次截图**，这项功能很难保证最终呈现的效果。

正因如此，ShareX 提供了多种方法来保证截图的最终效果，最终呈现的效果通常都很不错。究其原因，ShareX 在不使用管理员权限的情况下，采用了多种方法实现滚动截图操作——默认方式是「模拟按下 Home」 然后**分别用「模拟滑动、Page Down、鼠标滚轮」这三种方式尝试截图**，为了防止过长的图片使软件崩溃，默认最多一次滚动 20 页。在自动完成滚动的同时自动截图，并在完成后自动删除重复的图像，根据偏移来合并图像。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/23B9A362-F2F7-42AB-8A0B-A629A0078556.png)

滚动截图

在实际操作时，只要启用「滚动截图」，选择位置，就能获得整页截图了。可见，虽然想做好「滚动截图」这个操作并不容易，但 ShareX 考虑到了尽可能多的情况，极大提升了这一功能的可用性。

下面是滚动截图的演示视频，最终效果可以点击[这张图片](https://cdn.sspai.com/video/mp4/%E6%BB%9A%E5%8A%A8%E6%88%AA%E5%9B%BE%E6%95%88%E6%9E%9C%E5%B1%95%E7%A4%BA.jpg)查看。

### 图片标注

图片标注已经成为了截图工具中的一项标准功能，就连 Windows 10 都开始为截图 `Win + Shift + S` 增加了图片标注功能。但功能的多寡，各家并不统一。

和我们习惯的操作方式不同的是，当你按下截图快捷键后，ShareX 就会进入标注界面，和 Snipaste 不同，ShareX 让你**先对全屏进行标注**，然后再对需要的部分进行框选或其他操作。完成框选后就会将图片截取下来，进行下一步处理。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/69AC03C4-DB92-459D-A8C8-41B123A847F4.png)

截图工具栏

最常见的标注功能是简单的线条、形状、文字等，而且一旦使用下一工具，就无法再次编辑了；还有一些截图工具过分追求大而全，提供五角星、菱形等几乎不会使用的图形、提供画笔和复杂贴图等等。这两种体验显然都不好。参照 iOS 上两款很受欢迎的图片标注工具 Annotable、我的标记，它们在提供功能时都有各自的取舍。

我在两年前写付费教程时使用的是 Ashampoo 这款截图工具，它的常规功能非常全面，但唯独缺少「放大镜」，这也不得不让我再次进入 Photoshop 对这些图片进行二次处理。

而 ShareX 可能是图片标注功能最全面的截图工具之一了，上面提到的这些功能几乎都被涵盖在内。光是几个细节就让人感到舒适——

* **自定义圆角截图**：Windows 几乎所有的截图都是矩形的，这项功能让我们在截图时就让边缘变得更平滑。
* **自动识别窗口元素**：这一点和 Snipaste 类似，我现在已经很难适应那些无法识别窗口元素的截图工具了。
* **曲线箭头**：在拖拽箭头时，你可以拖拽箭头中部，让它形成一个曲线箭头，绕过其他文字，也能提升美感。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/F131E64E-A88B-4DE6-AEF2-004B4387473D.png)

曲线箭头和标号

* **放大镜和坐标**：在截图时如果觉得对不准某个点，滚动鼠标滚轮，放大镜就会变得更大。还可以选择方形放大镜、是否开启宽十字线等。
* **操作序号**：一个简简单单的数字序号，对图片标识度的提升是巨大的。在 PS 中完成这一操作你需要依次修改图标上的数字。
* **二次编辑**：我们无法保证每一个标注操作都是精准的，因此可以反复修改的二次编辑功能让标注成为了真正的**编辑器**而不仅仅是**截图窗口**。

这些功能在其他各种截图工具中都是不常见的，有了这些功能，就可以很轻松地对桌面上的内容进行指示、修改。如果你喜欢它的编辑器，还可以在设置中启用「在 Windows 上下文菜单中显示用 ShareX 编辑」的功能。

### 图片拼接

多张图片的拼接，这一点在手机上比较常见，我也一直认为这是一个简单的操作。而到了 Windows 上我惊讶的发现，居然少有可以对多张图片进行拼接的**小工具**。ShareX 也是我常用的拼图小工具。

在 ShareX 的「工具 - 合并图片」功能中，选择图片、选择拼接方向（Horizontal 水平拼接、Vertical 垂直拼接）和图片之间的间隙，保存就可以完成拼接了。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/B54F1A2A-A3CA-4625-9DEE-16FC8A10E2AB.png)

合并图片

ShareX 最初吸引我的地方，也正是这些完善的基础功能，**ShareX 出色地完成了一款截图工具需要完成的所有工作**。还能在「学有余力」的同时，完成一些常用的小操作。

进阶功能
----

当一款轻量工具实现了专业工具的部分功能，并且体验足够良好，我会更愿意选择轻量工具。

这里主要讲讲 ShareX 在加水印、截图阴影、上传到图床这三个功能上的表现。其实基础功能方面还有一项上传功能，但由于缺少国内图床，不便直接使用，因此也作为进阶功能介绍。

### 如何创建特效

ShareX 中有许多高级功能，这项功能在 ShareX 中被称为「图片特效」，都是通过它的特效编辑器实现的。在默认情况下，这项功能会为图片添加一个文字水印。

特效编辑器一共支持 42 种常见的图像效果，勾选就能使用。图中标记为绿色的是我认为常用的动作。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/4DD707AE-D86A-4AFD-AF3D-00DEF4DF97E2.png)

ShareX 图像效果一览

ShareX 还自带了自定义截图阴影的功能，开发者早在 2017 年就[在 Github 上提供了方案](https://github.com/ShareX/ShareX/issues/2728)，根据这一方案，我们可以定制出属于自己的功能，比如调整图片大小，调成黑白色、添加镜面反射效果等等。

不过 ShareX 没有为每种特效添加一个默认效果，导致在使用时不够直观。这里提供一个配置文件，导入后能让其中的每个动作勾选时立即显示相应的效果。默认是「带阴影截图 + 圆角 + 水印」的效果，可以自行取消。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/7FB7AAAF-430C-48C4-95CF-237E9F4CD058.png)

ShareX 所有效果展示

[点击此处下载 ShareX 图像效果配置文件](https://cdn.sspai.com/file/json/ShareX%20%E5%9B%BE%E5%83%8F%E6%95%88%E6%9E%9C%E4%B8%80%E8%A7%88%20%E4%BF%AE%E6%AD%A3.json)

#### 自定义特效

这里就以截图阴影为例，来讲讲如何创建一个新的特效。

**具体操作：**

1. 在「动作设置 - 效果 - 图像效果配置」进入配置界面（也可以点击「工具 - 图像效果」，随便挑一张图片，双击打开，就会进入编辑特效的界面）。
2. 「新预设 - 添加 - 过滤器 - Shadow」，这就是阴影的设置。右侧就是阴影的具体配置。依次是「不透明度、大小、深度、颜色、偏移」。
3. 这里我们可以将不透明度设为 `0.3`，大小设为 `20~30`，深度设为 `0.1` ，颜色可以设置为稍浅一些的灰色（比如 HEX `#666666`），偏移为向下 5 像素，设为 `0, 5` 就行。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/94F277B7-7849-48EF-B187-85DC9349B08B.png)

阴影截图

1. 保存图片即可。

这些 ShareX 中的动作被称为「预设」，可以将其导出为文本，你只需要复制下面这段代码，在相同的位置按下「导入 - 从剪贴板」，就可以导入截图阴影的预设。

```
{
  "Name": "截图阴影",
  "Effects": [
    {
      "$type": "ShareX.ImageEffectsLib.Shadow, ShareX.ImageEffectsLib",
      "Opacity": 0.3,
      "Size": 25,
      "Darkness": 0.1,
      "Color": "102, 102, 102",
      "Offset": "0, 5",
      "Enabled": true
    }
  ]
}
```

#### 启用特效

但此时我们截图还是没有阴影，如何启用这一预设呢？

这是因为 ShareX 同时只能使用一个特效预设，**因此必须要设置启用一项操作才能自动执行添加阴影**。需要在「动作设置 - 效果 - 图像效果配置」这里，将「截图阴影」选中。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/DCC21189-DEF1-4846-B969-A67FF64533ED.png)

启用特效

然后退到主界面，勾选「截图后的动作 - 添加图片水印/特效」启用效果，将来的截图就都会被加上阴影了。

### 如何将图片保存至云端

ShareX 默认提供了大量的服务和协议，足以让你把图片和文件存到各种服务之中。包含图片、文本、文件、短链接等多种服务。其中允许匿名上传的会被标记为黑色，无法直接使用的则为红色，红色说明需要授权。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/56863E2B-7CE8-45E7-B66E-4521ECE4846B.png)

选择上传服务

在「上传至 - 上传目标设置」中可以对服务进行登录和授权。这里的每一种工具配置方法不尽相同。比如 OneDrive 需要授权并获取验证码，允许在特定目录中上传文件并生成共享链接。而一些公开图床则只能直传并获取连接。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/125FCF53-CFF5-41C5-ACC2-4C297EF63E1F.png)

上传至服务器

其中有许多是国内无法直接访问的服务，比如 Flickr、Google Photos、Mega 等，因此如果你想让所有人都能看到你上传的图片，还是需要在众多服务中进行筛选。

目前提供的文本上传器中，GitHub Gist 值得一用。而图片上传器都不是很推荐，下面就如何自定义上传这个问题来详细讲讲。

### 如何自定义上传服务

关于如何用 ShareX 上传图片、文字、文件到图床，ShareX 官方给出了一份 [自定义上传服务指南](https://getsharex.com/docs/custom-uploader)，配置时需要根据不同的图床配置一一对应。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/0F2C1DF0-3400-42D3-9D9C-6ABE657D5735.png)

自定义上传

你需要在「上传至 - 自定义上传目标 - 新增」中进行配置。免费且提供 API 的服务配置起来是最简单的，因为它们已经把所有的内容准备好了。最容易配置的是 [sm.ms 图床](https://sm.ms/)，你只需要根据[sm.ms 图床上传 API](https://sm.ms/doc/v2) 中的说明，将正确的内容填到 ShareX 中，然后点击左下角的「测试」，如果成功返回链接，就说明已经配置完成了。

下面提供 [sm.ms](http://sm.ms/) 图床、[Pastr.io](http://pastr.io/) 文字上传服务、七牛云上传文件的配置文件，你也可以找到更多允许 API 上传免费和付费服务，将其添加到 ShareX 中（比如少数派不公开提供图床 API，因此普通用户无法通过 ShareX 上传图片）。

#### 图床配置：sm.ms

```
{
  "Version": "13.0.1",
  "Name": "sm.ms 图床",
  "DestinationType": "ImageUploader",
  "RequestMethod": "POST",
  "RequestURL": "https://sm.ms/api/upload",
  "Body": "MultipartFormData",
  "Arguments": {
    "ssl": "false",
    "format": "json"
  },
  "FileFormName": "smfile",
  "URL": "$json:data.url$",
  "DeletionURL": "$json:data.delete$"
}
```

**导入方法**：先复制一段代码，然后在 ShareX 的「上传至 - 自定义上传目标 - 新增」中，点击「导入 - 从剪贴板」。

#### 文字上传服务配置：Pastr.io

```
{
  "Version": "13.0.1",
  "Name": "Pastr.io 文字上传",
  "DestinationType": "TextUploader",
  "RequestMethod": "POST",
  "RequestURL": "https://pastr.io/api/create/",
  "Body": "MultipartFormData",
  "Arguments": {
    "paste": "$input$",
    "syntax": "nohighlight",
    "destruct": "none"
  },
  "URL": "https://pastr.io/view/$json:data.unique_id$"
}
```

导入方法和上面相同，导入完毕后需要在「上传至 - 文字上传至」修改为 [Pastr.io](http://pastr.io/)，将其修改为默认上传方式。

#### 文件上传服务配置：七牛云

七牛云的配置方案复杂一些，七牛云是付费的存储服务，需要用到每个用户私有的 API，出于安全原因，七牛不建议由本地生成 token（谨慎使用第三方工具），因此无法直接获取 token 填入 ShareX。

token 获取方法在七牛云的[官方说明](https://developer.qiniu.com/kodo/manual/1208/upload-token)中，这个页面同时提供了[上传凭证生成器](http://jsfiddle.net/gh/get/extjs/4.2/icattlecoder/jsfiddle/tree/master/uptoken) 可以在线获得 UploadToken。ShareX 配置文件的格式大致如下：

```
{
  "Version": "13.0.1",
  "Name": "七牛云",
  "DestinationType": "ImageUploader, FileUploader",
  "RequestMethod": "POST",
  "RequestURL": "http://up-z2.qiniup.com",
  "Body": "MultipartFormData",
  "Arguments": {
    "token": "【个人token】",
    "key": "%y%mo%d%h%mi%s-%ra.png",
    "file": "$input$"
  },
  "FileFormName": "file",
  "URL": "【个人域名】/$json:key$"
}
```

其中个人 token 和个人域名部分需要修改为你自己的，测试成功后修改默认上传工具，就可以将七牛云作为图床、文件存储服务使用了。

截图工作流
-----

ShareX 最强大的还不仅仅是上面这些功能，而是可以将这些功能串联起来，形成工作流。

如果你已经上手使用 ShareX 了，那你应该会发现它的使用逻辑并非「点一次运行一个功能」，而是预先设置好你想要的工作流，然后在每次执行时都按这个工作流执行。

**截图时需要做什么？** 这个问题 ShareX 替我们想得很清楚，最常用的截图流程是「截图 - 编辑 - 复制到剪贴板 / 保存到本地 - 上传 - 复制链接」。因此 ShareX 的功能设计也是围绕这一流程展开的，你可以定义「截图后」和「上传后」这两个关键时间点的步骤。这里提供几个使用思路——

* **截图后编辑并复制到剪贴板**：分别勾选「打开图像编辑器、复制到剪贴板、保存图像文件」，就能在每次截图时执行这些动作。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/AFD024F6-CC10-4E4E-8F32-88149D544463.png)

剪切、复制到剪贴板、保存

* **自动添加截图阴影**：按上文方法导入「图片阴影」特效，然后勾选「添加图片特效/水印」。
* **上传图片获得图片短链**：勾选「截图后 - 上传图片」，然后勾选「上传后 - 缩短 URL」和「URL 复制到剪贴板」。

![](.evernote-content/033C73B2-9E7A-4232-8F82-930D7DC4E65C/028062B8-EB6B-4FC3-BF7E-D5437797A1F2.png)

* **监视文件夹自动上传**：在「动作设置 - 监视文件夹」中添加特定文件夹，其中每次有新增图片时都会自动上传。这一功能可以用在 QQ 或微信的默认保存目录。

总结
--

虽然 ShareX 有足够多的亮点，称得上是一款优秀的截图工具，但我不认为 ShareX 能成为截图工具中的 Best Of。

在文章开头提到的三个等级中，可以说每个等级都有能打的工具。ShareX 并没有在**所有的领域**超过其他工具。像是它的 OCR 识别功能中，中文识别率并不高；Snipaste 的特色功能「贴图」也没有在 ShareX 中出现；使用逻辑上的不同让 ShareX 与普通截图工具相比更不容易上手。

ShareX 的适用范围很明确，如果你是一位经常需要截取图片、标记图片、打水印、上传图片的用户，需要针对图片执行一套完整的工作流。那么 ShareX 一定是你的不二之选。如果你需要其他特色功能或者只想简单易用，那么 ShareX 则需要经过一些配置才能更好用。
