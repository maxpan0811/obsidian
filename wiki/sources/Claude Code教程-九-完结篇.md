# 法律人的 Claude Code 教程（九）：九篇学完，真正的练功才刚开始

> 来源：智律积成（微信公众号）

## 核心内容

### Plugin
- 将 Skill、Hook、Agent、MCP 配置打包成 Plugin，团队安装后立即可用
- 从个人能力变成组织能力

### Git 版本控制
- 文档的"时光机"，修改留痕、随时回滚、分支实验
- **"不 Git，不 CC"**

### 隐私安全三层
1. 绝对不碰：密码、密钥、身份证号、商业秘密
2. 脱敏后处理：客户姓名→甲方乙方，金额区间化
3. AI 输出不能直接外发：必须人工复核

### 七条实践心法
1. 给验证标准
2. 先了解再动手（复杂任务用 Plan Mode）
3. 给具体别给模糊
4. 配置即记忆
5. 像助手一样沟通
6. 及时纠偏（两次以上→/clear 重开）
7. 扩大产能（多开窗口）

### Obsidian + Claude Code
- Obsidian 负责沉淀，Claude Code 负责加工
- 第二大脑接上 AI 助手，复利明显

### 开源工具推荐
CodePilot、SuitAgent（杨律诉讼场景）、OpenCode、Codex CLI、Gemini CLI、Cline

## 系列主旨

"愿 Claude Code 成为你的野球拳"——初看不起眼，真带进日常工作反复练，后劲很大。
