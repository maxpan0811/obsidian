==================================

作者: [ArielMeng](https://www.meiwen.com.cn/user/waodlttx.html) | 来源：发表于 2017-06-28 08:43 被阅读 2977 次

前面两篇文章介绍了 Anki 强大的功能，[国外流行的一款大大提高记忆效率的免费软件](https://www.jianshu.com/p/5ab591006650)，[利用 Kindle，Kindle Mate 与 Anki 批量建立自己的生词记忆库](https://www.jianshu.com/p/074b67b4f616)。

今天再介绍 Anki 的一个插件 Word Query，利用这个插件，我们可以轻松打造自己的百词斩词库，包含百词斩词条中的趣味图片，象形，以及单词视频，还可以添加牛津英汉词典等权威词典的释义。

(1). 首先，安装插件 Word Query

点击 工具 → 插件 → 浏览 & 安装，安装代码为：775418273，安装后需重启 Anki。

![](.evernote-content/F10CBA3F-161C-4491-8BFE-25A641B3C52F/A23E57F6-8B89-4BFC-A4AA-093EB5DC4095.png)

插件代码

(2). 新建笔记类型  
[利用 Kindle，Kindle Mate 与 Anki 批量建立自己的生词记忆库](https://www.jianshu.com/p/074b67b4f616)一文中，有新建笔记类型的详细步骤，本文不再赘述。

给新建的笔记类型 words 的区域添加字段：单词，图片，例句，发音，音标，释义，象形，TV

![](.evernote-content/F10CBA3F-161C-4491-8BFE-25A641B3C52F/B80D7204-7C38-4DDE-BA63-C8D40ADF9E19.png)

(3). 设置 Word Query 参数  
点击 工具 → Word Query → 字典文件夹 → 添加 (+)  
选择字典所在的文件夹

![](.evernote-content/F10CBA3F-161C-4491-8BFE-25A641B3C52F/E89AF79E-4B1B-4925-9BEB-2821AB46B8AA.png)

选择笔记类型，选择新建好的笔记类型 words

选择笔记类型

选择好笔记类型后，会出现设置字典字段的界面

![](.evernote-content/F10CBA3F-161C-4491-8BFE-25A641B3C52F/0A172D9E-A093-4E63-B18C-068FDB9BECA3.png)

设置字典字段

设置字典字段  
图片：字典选择百词斩，字典字段选择图片  
发音：字典选择 Bing xtk，字典字段根据个人喜好选择美式发音或英式发音  
音标：字典选择 Bing，字典字段选择美式音标或英式音标  
释义：如果安装了字典，可以选择自己喜欢的字典，如牛津高阶 8，字典字段为 default。如果没有安装字典，采用百词斩，或者 Bing 的都可以，看个人喜好  
象形：字典选择百词斩，字典字段选择象形  
TV：字典选择百词斩，字典字段选择单词 TV

![](.evernote-content/F10CBA3F-161C-4491-8BFE-25A641B3C52F/0CFA1A8B-FE19-4CF7-9709-EBDFEE021395.png)

设置字典字段

(4). 批量导入生词  
[利用 Kindle，Kindle Mate 与 Anki 批量建立自己的生词记忆库](https://www.jianshu.com/p/074b67b4f616)一文中，详述了如何将 Kindle 中的生词本导入 Anki，KindleMate 中的导出字段选择了序号，原形，用法，释义。本文不再采用 KindleMate 中的释义，而是采用百词斩和牛津词典的释义，所以 KindleMate 中的导出字段不需要再选释义，只选原形，用法即可。

![](.evernote-content/F10CBA3F-161C-4491-8BFE-25A641B3C52F/1EA6DFB6-E004-440F-BECE-2F33DFA6A410.jpg)

生词本导出设置

导出 Kindle 生词本的方法可以查看[利用 Kindle，Kindle Mate 与 Anki 批量建立自己的生词记忆库](https://www.jianshu.com/p/074b67b4f616)一文。

在 Anki 界面 点击 导入文件，设置导入 选项。  
类型选择新建好的笔记类型 words  
**允许在字段中使用 HTML，重要！！！**  
文件的第 1 字段对应到单词  
文件的第 2 字段对应到例句  
设置好后选择导入

![](.evernote-content/F10CBA3F-161C-4491-8BFE-25A641B3C52F/FAB850DE-CF01-4DA6-B072-5FA58D8B3AC4.png)

设置导入选项

(5). 批量获取生词释义  
在 Anki 界面，点击 浏览 → words，全选所有单词，点击 Word Query → Query Selected  
如果采用的是百词斩或者 Bing 的释义，那么到这里就已经完成了单词库的建立。

![](.evernote-content/F10CBA3F-161C-4491-8BFE-25A641B3C52F/EAD01757-7D0E-47FB-920E-31B6F7A2E1F9.png)

批量获取生词释义

如果采用的是牛津词典等自己添加的词典，可能会需要添加字典的 CSS 文件，如下图所示：

![](.evernote-content/F10CBA3F-161C-4491-8BFE-25A641B3C52F/55CFD9D2-C8A9-40EB-A7D7-677D04BE1206.png)

需要字典 CSS 文件

点击确定后，弹出文件选择弹窗，找到 CSS 所在的文件夹，选中 CSS 文件后，点击打开。再重复点击 浏览 → words，全选所有单词，点击 Word Query → Query Selected 即可。

![](.evernote-content/F10CBA3F-161C-4491-8BFE-25A641B3C52F/812378FD-0A21-4346-97EA-9CAFDBDC7B20.png)

选择字典 CSS 文件

(6). 美化单词卡片格式  
简单的美化一下样式。例句和释义文字居中看着不怎么舒服。  
选择工具 → 管理笔记类型 → 当前笔记名称 → 编辑卡片，在 style 的单引号里，输入 text-align:left; 使文字居左显示。单词字段可以将文字大小改为： font-size:26px; 或者 28px，等等都可以，改变颜色：color:#aa0000;， 或者其他 16 进制的颜色都可以。  
单词和音标字段，在 style 的单引号里，输入 display:inline-block; 单词和音标就会在同一行

![](.evernote-content/F10CBA3F-161C-4491-8BFE-25A641B3C52F/E41F4DF0-77DB-4354-8DF8-3D3C8BD13309.png)
