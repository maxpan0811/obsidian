# 特斯拉Model 3就要交付了！我们来了解它吧

---

什么是特斯拉 Model 3？

![](.evernote-content/8014EEDB-355A-437F-81C1-E4EF7F0A5677/7BCDC45A-F1E3-4E64-9E00-D338B6101DE3.jpg)

  
　Model 3 可以说是第一款走亲民价位路线的特斯拉电动汽车，所以它受到了相当的关注。35000 美元的税前售价，相当于人民币约 23.8 万元。与此相比，特斯拉之前的两款车型 Model S 和 Model X 在国内发布时的官方指导价分别超过 70 万元和 90 万元，差距可谓十分明显。  
  
　据了解，官方称入门级的特斯拉 Model 3 单次充电可以行驶达到 346 公里，相比同样是电动汽车的 2015 款日产聆风单次充电只能走 135 公里，它能够应对绝大多数时候的日常驾驶需求，省下很多寻找充电桩的麻烦。  
  
　值得一提的是，特斯拉 Model 3 虽然和它的前辈们一样也有各种选配方案，但只有不到 100 种选择。相较之下，Model S 有超过 1500 种选配方案。另外和 Model S 相比，Model 3 能够乘坐 5 名成人，但少了两个为儿童准备的后排座椅。  
  
　有一点值得注意，那就是 Model 3 的车主是不能享受特斯拉的超充服务的，只有 Model S 和 Model X 的车主能够享受该服务。据了解目前的政策是 Model 3 车主每年有 400 千瓦时的超充免费额度，超过的部分就要按需付费了 —— 这也是不少消费者对 Model 3 颇为犹豫的主要原因之一。

Model 3 的性能如何？

![](.evernote-content/8014EEDB-355A-437F-81C1-E4EF7F0A5677/C33E8851-9FAA-48F7-B584-E3854814146F.jpg)

  
　特斯拉 Model 3 的百公里加速用时大概是 5.3 秒，这个成绩要比 2015 年的宝马 BMW 228i 敞篷版要稍微好一点，但较之普通款又稍微慢一点。  
  
　Model 3 采用的是 2170 型锂电池，这是特斯拉和松下公司联合打造的。特斯拉承诺 Model 3 在电池和性能方面都能有更多的选择，但具体细节却还未真正公开。  
  
　在动力方面，Model 3 有单电机后轮驱动和双电机四轮驱动两种选择。虽然具体的细节我们仍未知晓，但 Model S 的四轮驱动型要比默认贵 5000 美元，即 34000 元人民币，可以作为参考。另外，Model 3 也支持疯狂加速模式。该模式下，Model S 的百公里加速时间可以达到吓人的 2.389 秒，不知道 Model 3 在该模式下会有怎么样的表现。  
  
　另外再说说自动驾驶。每一辆特斯拉 Model 3 上都配备了自动驾驶技术相关的硬件，可以实现半自动驾驶，即使是入门款也是如此。这是特斯拉 Model 3 一个很有吸引力的地方。因为和它相比，日产聆风没有现金的驾驶辅助，而雪佛兰 Chevy Bolt 只有最基本的防撞辅助。  
  
　不过理所当然的，很多便利性的自动驾驶功能，比如自适应巡航控制，自主停车和自动变道辅助这些都只是可选项，而不是标配。就目前来说 Model S 和 Model X 的相关功能升级需要 3000 美元，折合人民币约 20400 元，可以作为一个参考。  
  
　那么 Model 3 可以和哪些车型进行一个差不多同级别的对比呢？汽油动力方面，它的价位大概是宝马 3 系、奔驰 C 级、奥迪 A4 和雷克萨斯 IS 在同一个级别。另外还有全电型的宝马 i3 和混合动力的奥迪 A3，但它们的基础价位更高，纯电动时的可走的里程却更短。  
  
　至于电动车这边，同一级别的则有福特 Focus Electric、起亚 Soul EV、大众 e-Golf 和日产聆风。如果将这个对比范围扩展到那些插电式混合动力车型的话，竞争就更加激烈了。在 CES 2016 上，雪佛兰公布了 Bolt EV 全电动力汽车，单次充电也能走到 322 公里。未来日产也会推出更新、里程更长的车型来替换掉现有的产品。特斯拉未来要面对的挑战并不小。

产能够不够？

![](.evernote-content/8014EEDB-355A-437F-81C1-E4EF7F0A5677/F2161E8C-E35F-4B98-8FC0-0C75A40F34E1.jpg)

  
　提到特斯拉经常要说的就是产能问题，这关系到焦急的车主们什么时候能够拿到期待已久的新车。按照 CEO 马斯克本人的说法，Model 3 的正式交付将在本月的 28 日开始首批仅为 30 辆。不过他承诺到今年 12 月时，月产量将达到两万，以应付现在超过 40 万辆的订单数。  
  
　至于具体的产能规划，马斯克透露说，从这个月月底的 30 辆开始，8 月份的产量为 100 辆，到 9 月份提升至 1500 辆，最后直至计划中的 2 万辆。至于中国部分，按照特斯拉官方之前的说法是今年年底或明年年初开始交付工作。  
  
　另外首批交付的仅为单电机后轮驱动车型，预订了双电机四轮驱动的车主那就要继续等了。  
  
　当然了，考虑到特斯拉在产能和交付的问题上可以说是“劣迹斑斑”，很多人对它实际上是否能够达到预期的产能表示十分怀疑。据了解，国内第一批的车主可能会是在去年 4 月 1 日交付 8000 元订金的那批人。拥有 Model S 或 Model X 的车主或有优先权。不过根据来自体验店的消息，因为预订人数太多，排到 2019 年以后也是有可能的。

Model 3 最后的结论

![](.evernote-content/8014EEDB-355A-437F-81C1-E4EF7F0A5677/DB10EF22-1B19-4D47-85DB-C268EE73BEFE.jpg)

  
　特斯拉 Model S 可以说是一款极具革命性的车型，因为它彻底改变了大众对电动汽车的想法，让电动汽车从一个人们眼中还很概念的技术，变成值得考虑的一个选项。在许多主流的电动汽车还未能突破 100 英里行程时，Model S 就已经达到了 250 英里之多。  
  
　如果说过去的两款车型还让人望而却步，那么 Model 3 的出现就改变了这一切，相对亲民得多的价位让更多的人可以去选择电动汽车，感受一下新能源的未来。尽管只是入门级车型，但目前来看体验也不会差。  
  
　然而另一方面，Model 3 车主没有特斯拉免费超充资格，以及切实存在的人们对其产能的怀疑，这些都是阻碍不少消费者选择它的重要原因。接下来会如何发展呢？只有时间才能告诉我们答案了。
