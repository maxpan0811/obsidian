**Alfred**

OS X 上绝对的大杀器，配合 workflow 能极大的提升工作效率。

关于 Alfred 的介绍之前写过很多，每当身边朋友在使用过一段时间 OS X 之后我都会给他推荐。17英镑的售价确实不便宜，但一旦你上手之后你就再也离不开它了。

**Bartender**

系统顶部菜单栏管理工具，允许用户根据喜好来整理菜单栏。特别是对于小屏幕用户非常有用，从此工具栏不再凌乱。就国内应用代购平台的销售量来看 Bartender 深受大家喜爱的，是代购平台上单品销量最高的应用。

![](.evernote-content/84E1E596-6ABE-420F-A35E-56446B1CD986/F8E4D7C4-8369-418E-B1D1-7DEF65C019F7)

**iStat Menus 5**

OS X 上又一个必备系统工具。通过 iStat 可以快速查看CPU、内存、网络、硬盘、电池等系统信息，全面掌握 Mac 运行状况。

![](.evernote-content/84E1E596-6ABE-420F-A35E-56446B1CD986/8AD13744-0024-4F75-80B8-BEEF6DC31AED)

**Clean My Mac 3**

垃圾清理及应用卸载工具。虽然 OS X 系统很先进，但时间长了还是避免不了产生大量垃圾文件占用硬盘空间。Clean My Mac3简单易用，扫除垃圾和卸载应用都非常方便。如果家里 Mac机器较多，推荐购买家庭套装会更加的划算。

![](.evernote-content/84E1E596-6ABE-420F-A35E-56446B1CD986/FEEE1310-C4E1-44F1-A0C1-DFEFBE7D3145)

**DaisyDisk**

如果 Clean My Mac 还是不能帮你省出更多的硬盘空间，那你应该试试 DaisyDisk。这款应用可以全面扫描硬盘并给出使用详情报告。你可以清晰的查看到底是什么占用了过多的空间。我的一位同事昨天刚刚用 DaisyDisk 清理出了80多G 空间。

顺便说一句，如果你使用 Sketch 频率非常高且近期硬盘空间消耗得非常快，那是因为 Sketch 的版本缓存文件在占用空间。建议备份好文档清理一下缓存。

![](.evernote-content/84E1E596-6ABE-420F-A35E-56446B1CD986/9FAA796B-C008-4038-A56A-09B6CB284B85)

**Caffeine**

系统防休眠应用，通过这款应用可以灵活控制系统的休眠时间。防止在会议或工作时Mac 进入休眠而带来的不便。如果你对系统休眠时间设置有更高的要求，建议尝试更为强大的Amphetamine。

![](.evernote-content/84E1E596-6ABE-420F-A35E-56446B1CD986/F6976941-DE7E-47BB-9B51-A6B3DB0E7A53)

**1Password 5**

OS X 上最为强大的密码管理工具，与OS X 和 iOS（8以上）深入整合。从此不用再记住那么多不同的密码，只需要记住一个1Password 的主密码即可。

需要注意的是1Password 不保存用户设置的主密码，一旦忘记密码应用无法提供找回功能。主密码一定要妥善记牢。

![](.evernote-content/84E1E596-6ABE-420F-A35E-56446B1CD986/A46A06D7-76DC-404F-BB39-4BDB011E8069)

**TextExpander**

OS X 上强大的输入增强工具，能够通过短语输入来调用设置好的文本段。比如输入「;em」回填我的邮箱地址，输入「;dz」 回填我的收货地址。按照自己的习惯设置好后，输入效率将能得到巨大的提升。

![](.evernote-content/84E1E596-6ABE-420F-A35E-56446B1CD986/A0A43FC5-9771-48BF-87A6-C4FA59EE5020)

**Popclip**

划词搜索增强工具，通过 Popclip 你可以直接划词带入搜索（比如淘宝）、调用应用，大幅提升操作效率。

![](.evernote-content/84E1E596-6ABE-420F-A35E-56446B1CD986/095CE5EB-650F-4CDA-B649-1136848C7F7D)
