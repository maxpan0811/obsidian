最受欢迎的 macOS 播放器，终于迎来正式版：IINA 1.0.0
==================================

01月04日

[![](.evernote-content/28B3BC32-0E26-478A-B9F7-BE0AEC85C20F/4B0A41E2-28C1-4796-A856-5D10DAAFB9D7)](https://sspai.com/user/713147)

#### [Umi](https://sspai.com/user/713147)

希望给 macOS 生产力添砖加瓦的开发者前赴后继，着眼于基础工具类 App 的却比较少见。拿视频播放器来说，好看的往往功能少，好用的又可能有点儿丑。免费开源的 [IINA](https://iina.io/ "IINA")，却证明了鱼与熊掌亦可兼得。

少数派此前已介绍过一次 IINA，而赶在 2018 年的最后一天，公测近 2 年的它终于迎来了第一个正式版本。这期间发生了哪些变化，让我们通过本文快速了解一下。

关联阅读：[macOS 上最好的现代视频播放器：IINA](https://sspai.com/post/40870 "macOS 上最好的现代视频播放器：IINA")

从内而外，焕然一新
---------

在 macOS Mojave 中，Apple 加入了原生的黑暗模式。完全由 Swift 编写的 IINA 自然不会遗漏这一点，提供了亮 / 暗两种主题，并可以按照系统外观自动切换。你还可以自定义播放控制栏的位置及工具栏上所展示的按钮，让 IINA 符合你的操作习惯。

![](.evernote-content/28B3BC32-0E26-478A-B9F7-BE0AEC85C20F/BBD3D3B9-DC4C-4941-910F-71FDC55AF38B.png)

除了调教视频外，IINA 正式版还是一款美观又强大的音乐播放器。IINA 提供了完整视图和 mini 小窗两种模式，你还可以选择是否显示专辑封面，并用通知中心小组件控制进度。IINA 正式版还加入了对 AirPods 等蓝牙耳机的播放控制支持，不用碰电脑也能畅快地享受音乐。

![](.evernote-content/28B3BC32-0E26-478A-B9F7-BE0AEC85C20F/BBB127BA-B068-4623-81BB-86B05AD7C0A0.png)

此外，IINA 正式版的偏好设置界面也经过了重新设计，更加直观，所有选项一览无余。即使是小白，也能够轻松上手。

![](.evernote-content/28B3BC32-0E26-478A-B9F7-BE0AEC85C20F/71C15CA1-0967-45DA-9B3E-2E72A7B42576.png)

金玉其外，IINA 的内在也丝毫不含糊。在 1.0.0 正式版中，IINA 封装了最新版本的 mpv 0.29.1、FFmpeg 4.0.1 和 youtube-dl，让视频渲染更加高效。

![](.evernote-content/28B3BC32-0E26-478A-B9F7-BE0AEC85C20F/8376F2A1-74EB-4E2B-8085-78C276D56032.jpg)

贴心功能，为你而生
---------

播放视频时，你可以利用 IINA 实现实时裁切、旋转、调节速度等功能。

![](.evernote-content/28B3BC32-0E26-478A-B9F7-BE0AEC85C20F/30990343-E990-4C6E-8946-E1B879B30696.png)

在正式版中， IINA 还加入了丰富的视频及音频滤镜，包括锐化、模糊、负片、3D LUT 等。你还可以将一组滤镜添加为模板，在下次播放时通过快捷键直接调用。

![](.evernote-content/28B3BC32-0E26-478A-B9F7-BE0AEC85C20F/B384D78A-F079-4BB7-8C1B-B7991B190342.png)

IINA 正式版的另一项实用技能，当属在线字幕查找了。即使不慎下载到生肉，你也可以从 OpenSubtitles.org 或 Assrt.net 自动匹配字幕，免去折腾的烦恼。

![](.evernote-content/28B3BC32-0E26-478A-B9F7-BE0AEC85C20F/BE077C9F-1880-43B9-B3FE-B85D773AF9C5.png)

为了不占用系统拷贝粘贴等快捷键，IINA 正式版加入了「编辑」菜单，并重新设计了键位（例：`⌘Command-C → ⇧Shift-⌘Command-C`）。你也可以从几款预设中选择最熟悉的那一个，或完全自定义。

![](.evernote-content/28B3BC32-0E26-478A-B9F7-BE0AEC85C20F/DEB2A4C1-4AA5-4BBF-AE8E-B1A663FEFFE4.png)

进阶设置，更加强大
---------

内置了 [youtube-dl](https://github.com/rg3/youtube-dl "youtube-dl") 后，IINA 正式版支持直接打开在线视频，你也可以利用 [Chrome](https://chrome.google.com/webstore/detail/open-in-iina/pdnojahnhpgmdhjdhgphgdcecehkbhfo "Chrome") 或 [Firefox](https://addons.mozilla.org/zh-CN/firefox/addon/open-in-iina-x/ "Firefox") 扩展直接发送网页至 IINA 播放。少数派此前也介绍过一种 [借助 LaunchBar 利用 IINA 打开在线视频](https://sspai.com/post/41172 "借助 LaunchBar 利用 IINA 打开在线视频") 的方法，欢迎继续阅读。

![](.evernote-content/28B3BC32-0E26-478A-B9F7-BE0AEC85C20F/0C248E0C-D042-4AB2-A0EC-23BBD12E90AF.png)

IINA 正式版还加入了对命令行的支持，只需要打开终端，执行 `ln -s /Applications/IINA.app/Contents/MacOS/iina-cli /usr/local/bin/iina` 命令，就可以愉快使用了。例如，`iina --pip` 就可以在画中画模式下打开视频文件。你可以通过 `iina --help` 查看所有可选命令。

![](.evernote-content/28B3BC32-0E26-478A-B9F7-BE0AEC85C20F/3C15AA0F-38A3-4463-833D-4325040CE727.png)

你可以在 [IINA 官网](https://iina.io/ "IINA 官网") 或 [GitHub](https://github.com/iina/iina/releases "GitHub") 下载其最新版本，或使用 `brew cask install iina` 直接安装。
