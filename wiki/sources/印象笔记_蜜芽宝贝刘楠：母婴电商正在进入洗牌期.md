# 蜜芽宝贝刘楠：母婴电商正在进入洗牌期

---

![](.evernote-content/33BA13F6-C6EA-4063-A5CC-6F6A24CDDA92/AF548C90-A3DE-41EC-B780-A1FD14EDD7F9.jpg)

在电商领域，如何快速确立行业地位？蜜芽宝贝刘楠认为，最有效的方式仍非价格战莫属。成立刚刚一年、既已完成三轮融资的蜜芽宝贝近日宣布将在今年3月中旬举办进口纸尿裤节，以价格战的方式高调向同行拉开今年的竞争序幕。

第一战为什么选择纸尿裤？这也涉及到另外一个问题，即蜜芽宝贝为何要开打价格战？在母婴领域，纸尿裤属于典型的标准品，在母婴电商平台中销售占比高、利润率却很低，据刘楠透露，即使是一款国外售价等同于人民币68元的纸尿裤，在国内的电商平台售价达到148元左右，毛利率仍只有3%。这是因为在纸尿裤的海外采购过程中，从货源开始，经过出口商、进口商、批发商、再到中间商的层层加价，留给电商平台这样零售商的利润空间就已经非常低了，因此在国内的电商平台上，纸尿裤的类目和价格都已经做到相当透明。

这也就意味着，纸尿裤其实是相对教难打价格战的一类商品。在这次所发起的促销活动中，刘楠的决心则在于“还原纸尿裤的价格”：在蜜芽宝贝所提供的新闻通稿里，：花王、尤妮佳等品牌纸尿裤的数个型号都将降至“两位数”，“其中尤妮佳更是做到电商界纸尿裤品类史无前例的68元低价”。

在刘楠看来，价格战的背后不仅仅是资金补贴，而是以销售这种前端方式强调自己的后端实力。这种信心和蜜芽宝贝的供应链有关，根据《创业邦》之前对刘楠的采访，在过去一年，蜜芽宝贝“修炼”的重点都在于自己的海外供应链，通过自己做贸易的方式直接接触海外品牌的一手货源，尽可能砍掉中间环节，来增强自己对货品和定价的控制能力。

通过价格战这种低价拉拢消费者的形式来“强迫”竞争对手加入，进一步把幕后的实力摆到台面上，这才是蜜芽宝贝要打这轮价格战的主要目的：不仅能彰显实力，还能淘汰相对弱小的玩家，“一打价格战谁在裸泳一目了然”。

母婴电商是去年以来飞速崛起的一个领域，仅在创业公司领域，除了蜜芽宝贝在2014年12月宣布完成6000万美元的 C 轮融资；上线不到一年的贝贝网也于2015年1月宣布获得了1亿美元的 C 轮融资，并上线了海外购业务；今年3月，由女性社区发展而来的辣妈帮也对外宣布完成了1亿美元的 C 轮融资。创业公司之外，平台型电商也对这块紧咬不放，如天猫海外购的母婴频道和京东的母婴业务等。可以说玩家和资金都已到位，接下来要上演的则是彼此之间的残酷厮杀。

在采访中，刘楠也提到，母婴电商领域今年一定是价格战的一年，而价格战会让行业迅速进入洗牌期，让这个已成红海的创业领域尽快退潮。
