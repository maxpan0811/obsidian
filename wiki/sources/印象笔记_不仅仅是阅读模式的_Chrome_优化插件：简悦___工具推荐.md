不仅仅是阅读模式的 Chrome 优化插件：简悦 | 工具推荐
===============================

| 本文为付费栏目文章，您已订阅，可阅读全文 |

在阅读网页文章时，每个网站都有自己的文章样式、千奇百怪的论坛样式。在阅读时给我们造成了很多困难。Safari 和 Edge 都有各自的阅读模式，而 Chrome 则没有内置类似功能，需要我们自己安装插件实现。

我自己经常需要长时间浏览网页文章，在一众阅读模式插件中发现了一款颇有亮点的阅读模式优化插件：简悦，它不仅**能够提供沉浸式的阅读体验**，还能为网页文章提供不少加持，在对文章的处理上跟上了时代的脚步，**搞定了保存到稍后读、保存为图片、导出 Markdown 等许多小功能**，为我省下了不少安装零碎插件的麻烦。

为何浏览器需要阅读模式？
------------

对于浏览器是不是需要一个「阅读模式」的问题，很多人有自己的看法。

我认为，阅读模式的最大价值在于 **能够对文章进行统一的格式化处理**，从而带来更好的阅读体验。以广告为例，如果不开去广告插件，一些国内网页的两侧通常会显示一些定向推荐广告、页游广告。如果打开去广告插件，国外网站又会显示「建议将网页加入 Adblock 的白名单」的悬浮窗，更有甚者不加 Adblock 就不给看文章内容。而在阅读模式下就能做到一视同仁，去除广告的同时也不会受到悬浮窗的干扰。

![](.evernote-content/59660541-4296-462C-A376-A20ACA529396/D4325C7E-5FA1-46DF-8C59-E017FAC4C2CC.png)

阅读模式下的文章

阅读模式的另一个意义，在于**优化对文章的二次处理**，比如一些网站不允许选中内容、不允许复制内容，通过阅读模式格式化文字后，就可以进行选中和复制。论坛形式的文章通常也是格式一团糟，一些阅读模式插件能够正确识别论坛帖子的形式进行处理。使用自定义 CSS 也是一大亮点。

阅读体验还能提供 **页面的自定义、自动化处理**，对我而言最实用的就有「自动进入阅读模式」「自动生成目录」等功能，不需要我多做设置、不多点一下鼠标，就能自动帮我完善网页显示效果。再比如文章导出功能，Chrome 仅支持导出为 PDF，而一些插件能够将文章以 Markdown、Epub 格式导出到本地。

![](.evernote-content/59660541-4296-462C-A376-A20ACA529396/2621C771-57A3-4CE2-A1A8-926D935A76BE.png)

自动生成目录、进度条

但要做到以上这些事，找一款合格的阅读模式插件是免不了的。

一站式解决网页阅读难题
-----------

「[简悦](http://ksria.com/simpread/)」是我目前在 Chrome 上长期使用的一款**阅读模式插件**。它提供的功能很好地实现了「文章格式化、二次处理、定制化」等功能。

简悦官方提供的功能列表大大小小列了一堆。它在阅读、聚焦、站点适配等本职功能之外，提供了插件管理、导出到云端服务、导出到本地等功能。其中不乏一些有亮点的功能。

![](.evernote-content/59660541-4296-462C-A376-A20ACA529396/33301D49-0AB5-40BD-9C43-F20536C993D6.png)

简悦的功能列表

### 定制化的阅读体验

作为一款阅读模式插件，实现更好的阅读体验是它必须完成的本职任务。我曾经使用过一些阅读插件，声称能够自定义页面，结果实际只能对字体样式、字体颜色进行统一设置，显得有些不伦不类。

从阅读体验的角度来讲，**入门用户需要有不同风格的主题可直接选择、深度用户则需要自己定制行间距、样式、缩进，乃至完全自定义的 CSS。** 这些功能在手机端的阅读 App 中就体现的很好，在多看阅读、Kybook、Marvin 等阅读工具中就有现成的主题，也有可供自定义的主题。而在桌面端，理应可以实现更深度的定制。

简悦提供了九种全局主题，有明有暗有彩色。也提供对页面细节做单独调整。在页面上启动阅读模式后就能调整字体、调整字号、版面宽度、行间距、字间距…… 这些常见的设置。

![](.evernote-content/59660541-4296-462C-A376-A20ACA529396/1F6B9687-4264-4F0D-B4D9-66EEE69BC7E5.png)

定制页面样式

如果你想做更深度的定制，干脆粘贴一个现成的 CSS 进去，都是不错的选择。CSS 文件使用起来也非常简单，直接将 CSS 代码粘贴到「简悦设置 - 高级设定 - 自定义样式 - 自定义 CSS」中即可。

**小技巧：如何找到合适的 CSS 文件**

在 @Minja 的这篇首页文章 [《简单又好看，你的 Markdown 文稿也能加上个性化主题》](https://sspai.com/post/43873) 中也提供了 Typore、Markdownhere、MWeb、Ulysses、少数派等多种 CSS 的下载途径。

上期奏折的最后，我推荐了一个支持在线预览 CSS 效果的模板库网站：[Marked Custom Style Gallery](https://ttscoff.github.io/MarkedCustomStyles/generate_examples/styles.html#default_styles/antique)。它提供了一些现成的主题，你可以在网页中快速预览效果并下载。

经过阅读模式排版的网页，通常有着更统一的视觉效果，各种网页的阅读体验趋近一致。

阅读模式也可能出现不适配的情况，无法启用阅读模式。此时可以**通过聚焦模式高亮部分段落**，聚焦模式适合在任何页面上使用。

![](.evernote-content/59660541-4296-462C-A376-A20ACA529396/CFCB3684-95DC-4E4E-9D22-0DB3A5B89CAD.png)

聚焦模式

### 导出为 Epub、PNG、Markdown

简悦最初吸引我的功能其实并非「阅读模式」，而是它的附加功能。导出为 Epub 和 Markdown 格式是我最喜欢的功能。

如果想把一个网页保存到本地，Chrome 只能实现保存 HTML 和 PDF 格式，前者会多创建一个文件夹用于缓存网页中的图片和脚本，PDF 则很难进行二次编辑。

每当我想在手机上阅读、分享给别人、二次编辑时。我都会希望导出为更多的格式，比如 Epub、PNG、Markdown 等更合适的格式。简悦则正好做到了这些功能。

![](.evernote-content/59660541-4296-462C-A376-A20ACA529396/5CEA5660-F198-4412-A459-B3DB05C50AFA.png)

导出为 Markdown 格式

在我的实际过程中导出 PNG 有一定的几率失败，所以暂且不能代替网页生成图片插件。但 Markdown 和 Epub 的体验都很不错。Markdown 的样式识别比较精准，图片也是直接引用在线链接，不会在本地创建图片缓存。尤其是导出为 Epub 格式时还用上了自定义的 CSS。总的来说体验非常不错。

### 字数统计、划词搜索等额外插件

简悦提供了一个[插件中心](https://simpread.ksria.cn/plugins/)，目前插件数量不多，已有的插件还是比较实用的：我主要在用的是字数统计、划词搜索、代码增强这几款插件。

* 字数统计：能够在左下角显示字数、阅读进度，统计阅读时间等。
* 划词搜索：选中文本后出现弹窗，能够实现 Google、Bing、Baidu、Wikipedia 的一键搜索。
* 代码增强：CSDN 和 Stack Overflow 的代码框原本是有代码高亮的，但部分自定义 CSS 可能无法显示。代码增强插件可以在阅读模式下实现代码高亮。

![](.evernote-content/59660541-4296-462C-A376-A20ACA529396/AF89CD44-97BB-4368-A72E-E3ED96143259.png)

字数统计、划词搜索、代码增强

简悦插件的适用范围往往是限于文章本身的，字数统计、代码增强都是基于文章的优化。像是划词翻译等插件，可以通过 Chrome 插件来实现。不过简悦插件的启用和停用更方便一些，算是一个潜在的优势吧。

在桌面端实现最好的分享体验
-------------

在阅读之外，最痛苦的莫过于想发一篇好文章给朋友，但对方却读不到。

每当我自己的朋友需要一些建议时，我会想到 Power+ 里有哪些文章恰好可以解决他的问题。但每当遇到这种情况时，我都会发愁。直接发文章链接是行不通了，有时只是帮个小忙，让对方「付费后才能阅读」显得小题大做。

而一些笔记应用的分享功能也不够实用，比如用印象笔记分享文章，由于无法创建公开链接，双方必须都要注册账号。这种情况就出现在了我们组内，Hum 想通过 Evernote 分享一些批注过的文章给我们，但我正在使用印象笔记，数据不互通且无法访问公开链接…… 最后 Hum 还是把文章放在了印象笔记里，把整个笔记本分享给了我们。

如果导出为文件，PDF 在手机上的阅读体验差、Epub 没有阅读器也不一定能打开…… 难道就没有一个好办法可以畅快的分享一篇文章吗？简悦做到两种好用的分享方法。

### 生成临时页面

对于上面这种情况，简悦有一项功能相当切中痛点，它能够**生成临时页面**。进入这个模式时，简悦会将经过优化的文章主体内容上传到自己的服务器上，生成一个可供直接分享的链接，链接会在服务器上保留 5 个小时的时间。

如果遇到「需要临时发一篇文章」的情况，就可以通过简悦生成临时页面并将链接发给朋友，这样就无需账号、无需考虑格式问题，对朋友而言收到的仍然是一个可以直接打开阅读的链接，文章仍然是经过优化的版本。收发双方的体验都相当良好。

![](.evernote-content/59660541-4296-462C-A376-A20ACA529396/6FC6CED2-1766-4069-85B4-F367BD94412D.png)

创建临时页面

临时页面的作用也不止是分享，由于页面本身就是经过优化的版本，所以当你通过 Chrome 打印 PDF、通过简悦不支持的有道云笔记剪藏时，也能剪藏优化过的版本了。

### 划词生成分享卡片

不知从什么时候开始，一些应用开始做图片式的分享卡片了。我经常能在朋友圈看到这种类型的卡片图片，卡片的内容往往是一小段文字、带有网页链接的二维码。

简悦也支持生成分享卡片功能，把这项细节功能放在阅读优化工具中也丝毫不显违和。不过卡片的样式目前不支持定制，希望将来能有更好看的卡片定制插件出现吧。

![](.evernote-content/59660541-4296-462C-A376-A20ACA529396/0B201500-48E7-4FFD-BD17-009743F85021.png)

卡片分享

### 还有一些亮点功能

简悦在功能上也在不断进步，近期的一次更新支持了 LaTeX 公式识别（简悦可能是第一个支持 LaTeX 的阅读模式）。它还支持处理本地的 TXT 文件、Markdown 文件，这些优化让 TXT、Markdown 文件和其中的公式获得更好的显示效果。

简悦还能够将页面直接发送到 Google Drive、Dropbox、印象笔记、Onenote、Pocket、Instapaper 等服务。

更多的功能你可以在 [简悦官方的帮助中心](http://ksria.com/simpread/docs/#/) 查看，简悦还在 Notion 中做了一个 [官方信息汇总页](https://www.notion.so/c8386288be1642c6bee7a29cb46fb7a8) 。

仍不完美的阅读模式
---------

不过简悦还有一些不足之处，就我自己使用时也有一些不畅快之处。首先是加载速度稍慢，如果开启了「自动进入阅读模式」，Chrome 会先加载网页，然后一闪而过进入简悦的阅读模式，就像网站加载了两次一样。

其次是炫技甚于实用的 iOS 版本简悦。简悦没有提供 iOS App 版本，而是做了一款依赖 JSBox 运行的插件。你需要在浏览器中「分享页面 - 进入 JSBox - 选择阅读模式」才能进入，而且可能会提示站点不支持，无法进入阅读模式。说实话，很难与 Safari 原生的阅读模式竞争。

最后就是一些网站、文章本身的不规范，尤其是一些论坛帖子，文字排版混乱不说，一些插在文中的图标、表情会单独显示一行，这些都会破坏阅读模式的体验。虽然这不是简悦的错，但这些是可以改进的。

总结
--

可以说，[简悦](http://ksria.com/simpread/)把 Chrome 缺失的阅读模式给补全了。而且简悦最强大的反而不是阅读功能本身，而是在阅读相关功能上也提供了许多细节优化，省下了字体修改、中英文之间的空格、导出文章、划词搜索这些「装了嫌多，不装嫌少」的插件。

简悦的支持非常广泛，提供 [Chrome](https://chrome.google.com/webstore/detail/%E7%AE%80%E6%82%A6-simpread/ijllcpnolfcooahcekpamkbidhejabll)、[Firefox](https://addons.mozilla.org/zh-CN/firefox/addon/simpread/) 两大巨头的插件版，提供搜狗、QQ、360 极速等国内浏览器插件版，为 Safari、Edge 提供了 [Tampermonkey 油猴插件版：简悦 轻阅版](https://greasyfork.org/zh-CN/scripts/39998-%E7%AE%80%E6%82%A6-simpread-%E8%BD%BB%E9%98%85%E7%89%88)。大有一统天下的气势。

简悦能在多个功能上切中用户的痛点，着实不易。
