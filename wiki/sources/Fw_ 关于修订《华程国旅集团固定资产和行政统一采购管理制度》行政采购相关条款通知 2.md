Dear all，

为进一步优化集团行政统一采购管理流程，明确各类行政物资采购责任部门，提升采购工作效率，现对集团财字〔2026〕第 001 号《华程国旅集团固定资产和行政统一采购管理制度》中行政统一采购管理板块的采购部门划分条款进行更正修订，具体更正内容依据集团最新核定标准执行。

本次更正仅针对行政统一采购的物品分类及对应采购部门进行调整，制度中其他条款内容保持不变，仍严格遵照执行。更正后的行政统一采购物品分类、具体采购物品及对应采购部门详见下表：

|  |  |  |
| --- | --- | --- |
| **分类** | **具体采购物品** | **采购部门** |
| 员工福利相关 | 不带logo：日常福利 | 属地公司相关负责部门 |
|  | 带logo：员工福利 | 集团总裁办 |
| 团队物资相关 | 转换插头/旅行五宝/导游旗帜等 | 集团总裁办 |
|  | 出团横幅等 | 属地公司相关负责部门 |
| 市场宣传相关 | 笔记本/龙仔/LOGO包雨伞/挂历/公司宣传手册/帆布袋等 | 集团总裁办 |
|  | 名片印制/宣传单页制作/易拉宝/KT板等 | 属地公司相关负责部门 |
| 日常办公相关 | 饮用水/办公文具/绿植租赁/保洁用品等 | 各区域行政 |

请各单位接此通知后，及时组织相关人员学习更正内容，严格按照修订后的采购部门划分开展行政统一采购工作，确保采购流程规范、权责清晰。执行过程中如有疑问，请及时与集团行政部 / 总裁办沟通对接。

本通知自发布之日起生效，原制度中相关条款与本通知不一致的，以本通知为准。
