本文内容仅代表作者本人观点，文章对标题和排版略作修改，[原文链接](http://matrix.sspai.com/p/dbb56c20)。

作为 b 站重度用户，「收藏夹」曾是个让我非常头疼的整理死角。随手一「马」的视频在里面，看过「存档」的视频不舍得删，和同一主题相关的视频都塞在一个文件夹里，要找什么视频都得翻上好久、还不如直接去主站搜索。

「那要这收藏夹有何用。」

在第三次尝试彻底整理而以失败告终之后，我终于决定放弃  b  站收藏夹。但是收藏夹可以放弃，收藏的视频不能放弃啊。

好消息是这个世上真的存在「一劳永逸」这件事，只要匀出一个小时，你就可以真正「收藏」好 b 站的视频（当然也适用于 a 站 n 站所有收藏夹非常鸡肋的站）。

同样是为了解救我的小伙伴，以及希望能帮助到有同样困扰的人。

Workflowy 是个好东西，希望人人都爱它
-----------------------

这个能让你们把收藏的视频好好安放的工具叫做：[Workflowy](http://www.workflowy.com)。它是把牛刀，但是用来杀鸡也非常好使。

曾经几次试图写安利 Workflowy 的文章，也有一次成稿，但从未成功完全表达我对它的爱。既然一次表达不了，那就多表达几次。一篇文章写不完，就多写几篇。

如果非要给 Workflowy 下个定义的话，它是一个：可以无限展开、有无限子目录的笔记本。

我的 Workflowy 主界面中只有四个 item，少于我其他所有笔记应用的分类。但一层层展开这四个 item，你会看到我的毕业论文大纲和资料采集、专业课的知识点整理、在少数派及其他平台写作的大纲、还有今天想重点说说的对读过看过的东西的归档。

![](.evernote-content/60E4D2C8-082D-4013-9E96-F95C1BD037B0/77719A37-1873-44E8-89D8-00227EE2A938.jpg)

五组快捷键
-----

Workflowy 的快捷键非常非常强大和顺手，我可以用它整理完一整门功课的资料而手不碰触控板，不过今天只介绍其中五组最常用也最好用的：

### 1. tab

在进入 Workflowy 主界面的时候你会发现，你只能敲出一列点（让我把这些点叫做 item），「完全 get 不到这个笔记本的用法，倒是再给我第二页啊」，当时我的心里是这样咆哮的。

![](.evernote-content/60E4D2C8-082D-4013-9E96-F95C1BD037B0/13CD19BA-63BF-4F0E-98D4-5F3ADF6CB52E.jpg)

不要怕，让光标移到你敲出的纵向第二个 item，摁一下 tab 键。

![](.evernote-content/60E4D2C8-082D-4013-9E96-F95C1BD037B0/E798B3C6-0069-4CC7-B891-4D671386A1B7.jpg)

对，奇迹发生了。你敲出的第二个 item，向右移动了一格，这代表它附属于第一个 item。这时候在第三个 item 上摁两下 tab，它会向右移动两格，代表它附属于第二个 item。

![](.evernote-content/60E4D2C8-082D-4013-9E96-F95C1BD037B0/365A79E4-C322-47B6-9873-A7FBF1C319EC.jpg)

 相信聪明的你已经明白了使用这个笔记本的真谛。它可以无限展开，你想有多少个子层级，就有多少个子层级。

![](.evernote-content/60E4D2C8-082D-4013-9E96-F95C1BD037B0/D2BD9866-F33E-4711-B123-F0B66AAEF51D.jpg)

注：提高层级的按键是 shift + tab

### 2. cmd(ctrl for windows) + shift + >

你用纸笔做笔记的时候，如果 1.1.1.1.1 得过多，最后一个层级写字的空间就会非常逼仄。

但是由于这个快捷键，Workflowy 不会。它的名字叫「zoom in」，它可以让你「钻」到当前光标停留的 item 去，让你永远拥有一张从左上开始的白纸，不管你已经敲到了第几百层。

![](.evernote-content/60E4D2C8-082D-4013-9E96-F95C1BD037B0/160AA319-90D7-49D9-835C-7A04EDDCF6D9.jpg)

注：反向「钻出」的快捷键是 cmd(ctrl) + shift + <

### 3. shift + enter

这个快捷键的功能是给当前光标所在的 item 添加附注，整理时就把视频对应的网址添加在这里。

![](.evernote-content/60E4D2C8-082D-4013-9E96-F95C1BD037B0/A00FD43A-6931-403C-9B97-CCFCFEC99265.jpg)

### 4. cmd(ctrl) + up（方向键）

让光标停留在当前 item 上，并按下这个快捷键，比当前 item 层级低的 item都会被收起，让你在完成一个主题的归类后能够专注于下一个主题。

![](.evernote-content/60E4D2C8-082D-4013-9E96-F95C1BD037B0/AAABCE85-031A-4574-B9D5-3A7A4E346485.jpg)

注：反向「展开」的快捷键是 cmd(ctrl) + down

### 5. #

用你的功能键打出「# 」，并和 item 前面的文字保持一个空格，你接下来输入的文字就会变成 tag，直接点击 tag 就可以跳转到所有打上该 tag 的 item。你可以给一个 item 打上无数个 tag，当然原则也是够用就好。

![](.evernote-content/60E4D2C8-082D-4013-9E96-F95C1BD037B0/873AB749-EEE9-489C-AD25-43829ADBAEB5.jpg)（点击「BJ」tag 后 的搜索结果）

**剩余快捷键请摁 「cmd/ctrl + ?」 召唤。**

那么如何整理 b 站收藏夹呢？
---------------

其实文章到这儿就可以了，不过敬业如我还是给你们举个栗子。

你有二次元的墙头，有三次元的爱豆，有化妆教程，有游戏实况，还有什么我也举不出例子了。

![](.evernote-content/60E4D2C8-082D-4013-9E96-F95C1BD037B0/337E3B3D-106B-4328-B133-318CB6E26E23.jpg)

想怎么分类就怎么分类，想分多细就分多细，然后把你的视频链接复制过来，用上面说的第三个快捷键，添加为当前 item 的附注。

当你想找某一个视频的时候，只要在上面的「search」栏搜索关键词或者「#你自己写的tag」，秒秒钟出结果，然后直接点击附注中的链接就能跳转到 b 站视频。

最后，Workflowy 还有一个格外贴心的分享（安利）功能。把光标放在你想分享的最高层级的 item 上，就能看到一个叫做「share」的按钮，点击就可以生成链接分享给别人。

![](.evernote-content/60E4D2C8-082D-4013-9E96-F95C1BD037B0/9BE5986F-CCA0-4568-94F2-89943E49896C.jpg)

看着我的眼睛告诉我你们已经按捺不住了。

最后真诚地放上我的安利：[EITO 赛高](https://workflowy.com/s/FOE6.69NMyJYpfi)。

最后的注：Workflowy 的免费版每月只能添加 250 条 item，但如果别人使用你的链接注册，就可以给你多赚取 250 条 item 的额度，点击网页版右上角的「Get more space」即可获得链接。只要好好卖安利，每月 2000 条不是梦。
