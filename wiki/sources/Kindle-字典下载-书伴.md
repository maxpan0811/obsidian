## 一、官方内置 Kindle 字典：

Kindle 默认内置了一些亚马逊官方提供的字典，包含各英法德意日等不同语种。不同区域的亚马逊账号下字典个数和类型可能有所不同，不过**均可通过云端免费获取**。如果 Kindle 本机没有这些字典，可从 Kindle 图书馆的 “**字典**” 收藏夹中获取（注意，切换到 “全部” 需要翻到最后一页才能看到 “字典” 收藏夹，你可点击右下角页码输入一个大数快速转到最后一页）。以下是中亚账号提供的 12 种字典：

## 五、Kindle 字典安装及设置步骤

1、在本页挑选自己需要的字典，并将其网盘到到电脑中；  
2、用 USB 数据线将 Kindle 连接到电脑，直到出现 Kindle 盘符；  
3、将网盘到的字典文件复制粘贴到 `documents\dictionaries\` 中；  
4、弹出或安全断开 Kindle 的 USB 连接，并拔除 USB 数据线；  
5、打开 Kindle，按照相应的型号完成下面的默认字典设置步骤。

- **Kindle Paperwhite 2/3/4 代、Kindle Oasis 1/2/3 代、Kindle Voyage、Kindle 7/8 代**  
   `主页 -> 菜单键 -> 设置-> 语言和字典 -> 字典`，点击对应语言，然后选择一本字典，点击确定后即可讲该字典设为默认字典。
- **Kindle Paperwhite 1 代**  
   `主页 -> 菜单键 -> 设置-> 设备选项 -> 语言和字典 -> 字典`，点击对应语言，然后选择一本字典，点击确定后即可将该字典设为默认字典。
- **Kindle 3**  
   `Home -> Menu -> Settings -> Menu -> Change Primary Dictionary`，点击后系统列出所有词典，选择你已经传到 documents 内的词典即可。
- **Kindle 4**  
   `Home -> Menu -> Settings -> Dictionaries（第二页）`，点击后系统列出所有词典，选择你已经传到 documents 内的词典即可。
- **Kindle touch**  
   `Home -> Menu -> Settings -> Dictionaries`，点击后系统列出所有词典，选择你已经传到 documents 内的词典即可。
- **Kindle DXG**  
   `Home -> Menu -> Settings -> Menu -> Change Primary Dictionary`, 点击后系统列出所有词典，选择你已经传到 documents 内的词典即可。
