没想到微信读书还能这么玩！

微信读书最近推出了自己的 Skill，它不是在微信读书里用，而是可以放到各种 Agent 工具里，比如 OpenClaw、Claude Code、Codex 等。

安装也很简单，两步就够：复制安装指令，再获取 API Key。

装好以后，Agent 就能读取你的书架、划线、笔记和阅读统计。

我试了一下，直接让它分析我的阅读习惯、常读内容、偏好的主题，还顺手生成了一张阅读画像信息图。

以前这些数据只是躺在 App 里，现在变成了可以被 AI 调用、整理、分析、再创作的素材。

如果你平时也在微信读书里划线很多，这个 Skill 强烈推荐！
