等了三年的 Things 3，也许是你入门 GTD 最好的选择
===============================

2017 年 05 月 18 日

[![](.evernote-content/29E6C157-FE49-40D2-B8A9-BF9E71FB8AC6/38203675-424A-4A7B-B175-45A6C8D79EBB)](https://sspai.com/user/703114/updates) 

#### [waychane](https://sspai.com/user/703114/updates)

你期待的「有生之年」App 是什么？除了四月底上架的少数派 iOS 客户端，肯定有很多人的答案，是跳票近三年终于上架的 [Things 3](https://culturedcode.com/ "Things 3")。

经过三年的开发周期，Things 3 在保持了 Cultured Code 团队高设计水准的同时，也依旧保持了软件的易用性。相比复杂、厚重的 OmniFocus、2Do 等 GTD 应用，有更友好的交互界面以及更低上手成本的 Things，或许是你入门 GTD 任务管理更好的选择。

设计更精致了
------

由 Cultured Code 开发的 Things 曾于 2009 年获得 Apple 设计大奖，此次大版本更新，Things 3 依旧保持了开发团队高水平的设计功底，无论是在界面还是交互上，无处不给人惊喜。

### 更现代化的 UI

Things 2 虽然之前更新了「扁平化」设计，但界面依旧延续了「拟物」风格，显得有些老气。全新的 Things 3 不仅采用了视觉上更清爽的白色背景作为主色调，应用内也更换了更加现代化的字体和图标。

![](.evernote-content/29E6C157-FE49-40D2-B8A9-BF9E71FB8AC6/6F65B815-1972-447E-9A67-BBF1F28CC13E.png)

左边是 Things 3，右边是 Things 2

![](.evernote-content/29E6C157-FE49-40D2-B8A9-BF9E71FB8AC6/C62D48D0-C4C9-44E2-B5B8-EAD0E170DC65.png)

Things 3 for Mac 界面

### 令人惊喜的交互

除了界面上的变化，Thing 3 还采用了大量精美而不失稳重的交互动效，这里通过动图直观地感受一下吧。

![](.evernote-content/29E6C157-FE49-40D2-B8A9-BF9E71FB8AC6/2493FAE5-6DFB-4CF1-A45A-391B30CBB875.gif)

值得一提的是，通过截图可以看出，Things 3 终于带来了对于中文语言的支持，对于 GTD 入门用户理解相关概念也会更有帮助。

操作更方便了
------

Things 3 的整体使用逻辑与上一代没有很大的区别，同样采用了「责任区域」、「项目」以及「待办事项」三大层级来管理你的事务，却在细节上让任务管理变得更方便、更高效。

### 更方便地创建任务

在 Things 2 里，通过点击首页的添加按钮我们只能创建一个新的待办事项，如果要创建新的责任区域或者项目，我们还得点击设置图标或项目图标，再进行创建，尤其是创建「责任区域」时步骤很繁琐。

![](.evernote-content/29E6C157-FE49-40D2-B8A9-BF9E71FB8AC6/359405A2-35E7-48D2-BAD0-CB9E7C56CD51.png)

Things 2 创建「责任区域」的步骤

Things 3 将创建责任区域和项目的操作同样放到了首页的添加按钮中并直接在首页平铺展示，这样不仅操作更直观，也更利于查看。

![](.evernote-content/29E6C157-FE49-40D2-B8A9-BF9E71FB8AC6/A9B92653-9C94-4722-89B9-FA102E2CD658.png)

Things 3 创建「责任区域」的步骤

### 更快捷的搜索

在 Things 2 里，如果要搜索任务或标签等，你只能在应用**首页**下拉，并点击最上方的搜索框搜索，在移动设备尺寸越来越大的今天，面对如此高频的操作，这样的设计无疑很「反人类」。

Things 3 能让你在**任意界面**下拉唤出搜索框并弹出键盘直接输入，对于在多层级项目内快速切换以及搜索操作的效率提升非常明显，也很方便。

![](.evernote-content/29E6C157-FE49-40D2-B8A9-BF9E71FB8AC6/3868C8BF-E96E-4EC5-BA40-D9C5A1B3D97A.gif)

在任意界面下拉唤出搜索框

### 你可以为任务设置提醒时间了

Things 2 被人诟病最多的就是只能为任务设置截止日期而不能设置截止时间，你只能通过更改软件的全局提醒时间在一天中的固定时刻收到当天任务提醒，非常不灵活，也容易受干扰。

在 Things 3 里，你终于可以为任务分配具体的截止时间了。

![](.evernote-content/29E6C157-FE49-40D2-B8A9-BF9E71FB8AC6/27CE0DFF-B32A-4D91-A516-CAB66066A094.png)

### 支持「日历事件」

除了 Things 2 支持连接系统自带提醒事项应用以外，Things 3 新增的「日历事件」还能让你访问日历日程，通过专门的入口查看日历事件，也能直接在项目界面看到接下来要到来的日程，根据日程来合理安排待办事项的执行计划似乎是一个不错的做法。

![](.evernote-content/29E6C157-FE49-40D2-B8A9-BF9E71FB8AC6/C0BAB052-553C-4C30-9D69-7D8EEC64206C.png)

过滤器
---

> 过滤，是现代任务管理工具的先进之处，意义所在。

这是 [JailbreakHum](https://sspai.com/user/681230 "JailbreakHum") 在知乎 Live 《[个人任务管理进阶：任务过滤和进阶工具](https://www.zhihu.com/lives/823240334342492160 "个人任务管理进阶：任务过滤和进阶工具")》说的一句话，而 iOS 与 macOS 平台最出名的 GTD 应用 OmniFocus 和 2Do 也因为「透视」和「智能列表」强大的过滤功能与其它应用拉开差距，走上「神坛」。由此可见过滤功能在 GTD 应用中的地位之高。那么，Things 3 对过滤功能的支持又怎么样呢？

### 有哪些过滤器？

Things 3 同样不支持像 OmniFocus 或 2Do 那样通过语法自定义复杂而强大的过滤器，你只能像 Things 2 一样通过两种过滤器来过滤任务：

* 时间：仅支持通过应用默认的「今天」、「未来」、「随时」、「某天」四个过滤器查看任务；
* 标签：允许设置多标签，但是没有入口，只能通过「快速搜索」搜索相应的标签查看。

![](.evernote-content/29E6C157-FE49-40D2-B8A9-BF9E71FB8AC6/E188D472-DE4D-464A-8E86-3B6F470E8D19.png)

另外，Things 3 的「快速查找」虽然是搜索功能，我把它理解成一个简化版的 2Do Smart List，前者只能通过单维度搜索过滤出单一的项目或标签等，搜索结果也不能保存，不过在某些场景下用它来定位任务会是很方便高效的做法。

### 够用吗？买不买？

是的，你没看错，经过将近 3 年的开发，Things 3 的过滤功能依然与 Things 2 一样简单甚至可以说是简陋，它甚至连基本的结合多个标签或结合标签与时间来过滤查看任务都做不到，与 2Do、Todoist 等支持复杂而强大的过滤语法的 GTD App 更是没法比，为什么 Things 依然能在这么多强大的 GTD App 中保持良好的口碑和用户群呢？

也许，Things 2 没有中文、不能设截止时间、没有强大的过滤功能等是你没用它的原因，Things 3 的出现，恰好补上了应用本身对个人任务管理基础功能的缺失，在能够遵循 GTD 思路的基础上它又做到了功能足够轻量、界面也足够友好，反而能让事务没那么多或正在入门个人任务管理的你更专注、更愉悦。

不过，3 年磨一剑的 Things 3，却更像是对 Things 2 版本细节方面的完善，除了设计之外没有让人眼前一亮的新功能，在各家 GTD App 都在谈都准备做「自动化」的今天，Things 3 似乎有些力不从心。对于一款大版本升级的应用而言，Things 3 也多少令人失望。

如果你已经有一套个人任务管理体系和流程，我不建议你轻易迁移到 Things 3 来，如果你觉得强大的 OmniFocus、2Do 等过于复杂，对过滤没有很强的需求，或者你正在入门 GTD 知识，我会推荐你试试 Things，它足够漂亮、足够简单，在不给你太多压力的同时能帮你管理好任务。毕竟，能帮你把事做好的工具，就是好工具。
