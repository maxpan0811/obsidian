| |  | | --- | | *本帖最后由 sk2341 于 2013-12-10 11:37 编辑*    更新1210 ：  之后楼主又仔细研究了一下Proceate，发现在ipad上画画的方法并不是就在那么大一点的画纸上直接作画。要活用双指缩放，当你把画纸放大了去慢慢得画你想要的细节时，原来画得精确就是一件很容易的事情了，所以楼主驳回jot script 不适合作画的说法，放大之后的世界，电容笔就可以画的很从容，适应这支笔后偏移已不是问题，也没有人会那么慢得画斜线。这点跟楼主倾向于用放大框来手写笔记也是一个道理。具体体验可以参考一下楼主引用大神用这支笔作画的视频。    更新1208 ：  目前发现的问题，当画对角线的斜线时，如果慢慢得画，画出来的就是起伏很小的波浪线，画快是很直。  所以，楼主目前观点，此笔现在存在Y轴偏移，和慢慢画斜线不直的问题，不适合用来画画，只是用来写字倒是很好。  如果想画画，又要有压感，可以了解一下 Jot touch 4.   更新1206 ：  1.笔迹偏移问题：竖着放ipad时，画水平线完美，不论怎么改变笔的角度，线都与笔头轨迹重合。     画垂直线时，若把笔向左或右倾斜严重，线会向左或向右偏移，向下或向上倾斜笔不影响。    目前不知道是笔的设计缺陷，还是软件问题，解决方法就是握笔姿势的适应，正常握笔时尽量不要向右倾斜太多，向下倾斜的握笔姿势就完全不会发生偏移现象，楼主目前已适应（才拿到笔那几天就觉得写字很怪，原来是左右会偏移）  2.画画体验，介于楼主拙劣的画画水平，特别从youtube搬运来一个视频来看看Adonit jot script& Proceate      关注电容笔的小伙伴最近应该都了解过这款 Adonit 最新出的电容笔，它完全是一次技术的革新，它不同于最早笨重粗大的橡皮头笔，也升级了自家 jot pro 透明吸盘的累赘设计，拿到手之后感觉就是一根圆珠笔无异。    首先就来说说这个笔头，笔头是塑料材质，硬的，理论上是不会刮花屏幕，楼主用了一年多的 jot pro（笔头是塑料圆盘）也没有留下任何划痕。    笔头是可以按下去一点的，感觉只能压下去1毫米左右，就像键盘的键程一样，笔头里面应该有弹簧，按下弹起很快，所以写字会咯噔咯噔，抬起来咯噔一下，写字时与屏幕接触会响一下，画长线一直hold就不响，有些同学可能会觉得吵，但比起它纤细的笔头这个优点，这点楼主是可以忍受。    这支笔是需要电池供电的，安一节7号电池（新买的赠送一节），据国外网友使用的经验，一节电池他用了2周半，具体还要看每天使用时间。如果不开电源，这支笔是完全不可用的，这和传统的电容笔是不一样的。    值得特别提的一点就是，不匹配设备的蓝牙，只打开笔的开关，它也是可以工作的，就像普通的电容笔一样。开启ipad的蓝牙，与一些支持 jot script 的笔记软件（比如goodnotes)连接，就可以支持手腕保护的功能：就是ipad 只识别你用笔写的东西，手指手腕划的都是自动忽略的，这点很cool 。    Jot script 是一个好的硬件，但好的笔记体验，自然也要结合优秀的软件支持了。    楼主选择使用的就是GoodNotes，也是之前就一直在用的。    至于什么这支笔跟 penultimate（墨笔）量身定做啊，虽然现在是支持，但就目前的软件优化情况，用墨笔做笔记？那完全是闹着玩。    还有noteshelf，你说它也支持 jot script，然后类？在楼主看来，它只是一个华而不实的笔记本软件，记几句话一张纸就满了，这你受得了。    所以，要认真做笔记的同学请往下面看，我们可都是认真的，所以做笔记，要的就是生产力。    要写出完美的笔记，首先就要有方便的放大框，然后就是能灵活的双指缩放页面，另外能支持pdf那当然是最好了，再配合上 jot script 的蓝牙手腕保护，就可以做到想写哪就写哪了。    回答几个问题：    这支笔写字有没有延迟？    就楼主目前使用没有延迟，完全跟普通电容笔一样，延迟一般也都是软件优化问题，不可能是笔的问题。      笔尖很小，写小字方便吗？    楼主目前还是比较依赖放大框和双指缩放，mini2本来屏幕就比较小，直接写在屏幕上效果不是很理想，ipad air也许可以。    小结：楼主使用这支笔时间也才不久，感受也不是足够具体充分，只是暂时跟大家分享一些，更新正在路上。。。 |
