# 黑神话悟空中的一周目二周目什么意思？有几个周目和结局？

[Q学长研习社](#) 

*2024年08月23日 18:11*
*江西*

![](attachments/8343a6a9215e8180.jpg)

**《黑神话：悟空》目前已知有三周目 。**

**一周目结局有两种 ：**

**结局一：**天命人与大圣残躯展开对决并战胜后，大圣残骸消散，金箍掉落，老猴子为天命人戴上金箍，天命人化身悟空继续踏上西游之路。这意味着天命人继承了悟空名号和命运，陷入了命运的轮回。

![](attachments/5a8fb39ff17c3eb3.jpg)

**结局二：**天命人同样战胜大圣残躯，但在老猴子试图为其戴金箍时选择拒绝，象征着他拒绝悟空的命运，对命运发起挑战。

![](attachments/8ad589721e7f8d03.jpg)

二周目开启方法是在完成一周目结局后，返回标题界面，找到 “再入轮回” 选项即可重新开始游戏，并继承部分一周目的武器等 。

![](attachments/7154b7d7d42a613a.jpg)

二周目整体怪物有所增加，剧情方面也有更多可能性，自由度不变，玩家仍可随意探索地图支线等玩法，但隐藏结局等仍需自行解锁 。二周目结局的达成条件之一是完成每个章节的隐藏地图，并拿到对应的奖励（如定风珠、辟火罩等）；

![](attachments/030bd871a9bd1a5c.jpg)

然后正常打完结局，通关之后继续游戏，去小西天浮屠塔壁画的房间触发第二结局，在该结局中玩家需要与四大天王和二郎神战斗，击败二郎神后就能达成结局，玩家能够获得二郎神的武器，二郎神还会告知玩家一些隐藏信息，最后的片尾会多出一段隐藏动画倒放西行路 。

![](attachments/32012ac8bfcdb909.jpg)

三周目开启方法为通关第一周目后，在游戏的主菜单会出现一个 “再入轮回”，点击后会出现弹窗 “是否踏入新的轮回，重走西游之路”，点击确定即可开启三周目，玩家前两个周目的装备武器都会继承 。不过，目前有玩家拿到了隐藏的大圣套装，猜测只是一个小彩蛋 。

![](attachments/bdaaffcb0a9d2d89.jpg)

关于是否存在三周目以上的周目以及是否有第三结局等，目前还没有明确的官方确认信息 。有传闻存在东海龙宫，并且可能是花果山的隐藏地图，玩家需要找到东海龙宫并击败某位 boss，之后获得直面天命的机会，这可能是第三结局，但也有说法认为这可能是一张废案地图，且全网还没有任何一个玩家找到或确认这个第三结局的具体内容，仍需等待更多的游戏内容更新或官方消息来揭晓 。
