# macOS Sierra如何让你养成清理桌面的习惯

---

威锋网讯，去年苹果为 Mac 平台带来了全新的系统-macOS Sierra，更名后的 OS X 系统不仅仅只是名字上的变化，更重要的是它带来了一系列的新功能和体验上的提升，相信不少 Mac 用户体验这个系统都已经有一段时间了，那么你认为这个新系统有哪些地方给你留下了深刻的印象呢？  
  
　近日来自 iMore 的编辑 Lory Gil 就给出了自己对于 macOS Sierra 的体验分享，我们接下来不妨听听看。（注：下文的第一人称均指代 Lory Gil）  
  
　我的电脑桌面总是特别乱，我创建了一排文件夹来方便我快速访问和使用，也许其中的一个或两个文件夹装着的都是那些不知道用来干嘛的文件和一些担心丢失的文件。不过，在大多数时候，我还是希望一切东西都整整齐齐，例如将无用的文件马上扔到废纸篓、将文件整理好放进文件夹。  
  
　自从我下载 macOS Sierra 后，要让一切都变得整齐就更加容易了，我甚至设法清理那些难看的垃圾文件夹，这在以往是很难想象的。  
  
　**Siri让搜索变得简单**

![](.evernote-content/89F1078F-6188-44A1-9998-C0DC1F5A2557/212F635D-23ED-4CAA-93CE-3EE87F246803.jpg)

正如之前提到，我以往会在桌面放一到两个文件夹来放置一些东西，例如朋友给我的漫画书文件，不再用的乐队的练习课程，又或者是一些通过云服务分享的小电影。

当中的很多文件我都不想丢失，但我又经常想不起它们放在哪里，我只是记得有这些文件存在。macOS Sierra 支持 Siri，有了 Siri 后，这个问题就不再是问题了。我可以让 Siri 帮忙搜索，例如“显示我的乐队文件”，一系列结果就会出现在查询结果中。

这让我在拖拽文件到后台文件夹的时候更加无后顾之忧，因为即使我忘记自己将这些文件拖到哪里了，也可以让 Siri 帮我寻找。而多余的文件夹也没有存在在桌面的必要了，这样桌面就会显得简洁多了。

**可以重新安排菜单栏的工具了**

![](.evernote-content/89F1078F-6188-44A1-9998-C0DC1F5A2557/7E2EB216-4565-44A6-BE71-F7EFB3AD6654.jpg)

这个功能对别人来说并不是什么大功能，但我倾向于经常使用菜单栏的小工具，我喜欢将这些小工具以特定的顺序排列。我每天使用 1Password 和 Dropbox 很多次，但 iStats Menus 就用得比较少。所以我希望能够快速打开 1Password，但因为小工具太多，我经常找不到。

自从用上 macOS Sierra 之后，我就能够轻松拖拽这些第三方的小工具以及苹果内置的小工具，排列这些工具变得更加简单。现在 1Password 就放在菜单栏的最右边，另一侧是搜索和 Siri。

怎样才能移动菜单栏的小工具？长按 Command 键，然后点击并拖拽菜单栏上的图标即可。你还可以删除内置的菜单栏工具，例如 Wi-Fi 或电池状态，将它们拖出菜单栏就可以了。

**桌面和文稿文件夹同步迫使我定期清理**

![](.evernote-content/89F1078F-6188-44A1-9998-C0DC1F5A2557/F2E41BE0-154F-492F-A3B5-DDE61358D731.jpg)

我在 Sierra 上用得较多的新功能就是桌面和文稿文件夹的 iCloud 同步了，当我在桌面短暂放置一些文件的时候，很快相关（同一个 iCloud 账户）的电脑也会出现这些文件。

如果我不注意的话，我的桌面就会变得一团糟。如果我在我的 iMac 上保存截图、收据、文稿和音乐文件，这些东西在 27 英寸的屏幕上看起来还不会太乱，但当我转过头用 13 英寸的 MacBook Pro 后，桌面就会显得非常密集且拥挤。所以为了让这些电脑的桌面都保持整齐，我养成了定期清理桌面文件的习惯。是的，清理桌面已经成为我日常生活的一部分了。

**整齐能提高效率**

当我的办公室很乱的时候，我总感觉到我的工作生活杂乱无章。有时，我会完全避开办公室，选择在客厅工作，直到我清理办公室。

当电脑桌面变得杂乱的时候同样如此，如果文件和文件夹到处都是，我就不想工作了，又或者完成一个项目就需要更多的时间。

以上提到的一些 macOS Sierra 的新功能，无论是好是坏，都让我养成了清理桌面的好习惯。而养成清理桌面的习惯能让我提高工作效率，心情也变得好了。

macOS Sierra 是 2016 年 6 月苹果在全球开发者大会(WWDC 2016)发布的新一代 Mac 操作系统，该版本延续了前代扁平化的设计风格，并在功能上进行了诸多改进与优化。此次更新的焦点主要集中在与 iPhone 和 Apple Watch 集成以提高用户体验方面，它可能不是最完美的 Mac 系统，但却是最成熟的一个系统。
