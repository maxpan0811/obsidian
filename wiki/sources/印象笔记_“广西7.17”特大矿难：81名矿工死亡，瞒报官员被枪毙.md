# “广西7.17”特大矿难：81名矿工死亡，瞒报官员被枪毙

速读摘要

2001年7月17日凌晨3点多，广西壮族自治区南丹县龙泉矿冶总厂下属的拉甲坡矿发生了特大透水事故，这场事故导致拉甲坡矿、龙山矿、田角锌矿共有81名矿工被埋。黎东明早上7点就知道发生了矿难，可是直到下午5点，他才把这次事故汇报给了当时分管南当矿业的县委副书记莫壮龙。记者在龙泉矿经过多方面走访和采访，得到的统一回复都是龙泉矿只发生了小规模的渗水事件，没有出现人员伤亡。记者又采访了几位遇难者的家属，才知道龙泉矿渗水事故的真相。

2001年7月17日，广西自治区南丹县发生了一起特大矿难，这起矿难中共有81名矿工遇难，造成直接经济损失8000多万。这场矿难对南丹县、甚至整个中国而言，都是一场灾难性事故。

让人心寒的是，“7.17矿难”发生后，县政府领导为了逃避责任，与涉案公司统一口径整体瞒报，就连新闻都被封锁，这起矿难隐瞒了整整十几天才公之于众，严重耽误了救援，最终导致81名矿工死亡。

![](.evernote-content/0DE2742B-F406-42C0-B931-A3F0D710BBC9/1551EDC8-C1BE-44D4-A2B2-147676ACC053.png)

煤矿发生特大渗水事故

2001年7月17日凌晨3点多，广西壮族自治区南丹县龙泉矿冶总厂下属的拉甲坡矿发生了特大透水事故，这场事故导致拉甲坡矿、龙山矿、田角锌矿共有81名矿工被埋。

矿难发生的前一天下午4点左右，龙山矿、拉甲坡矿，田角锌矿一共有500多名工人下井作业，凌晨3点多，工人对拉甲坡矿9号矿井实施了两次爆破。巨大的爆炸让矿井的隔水岩体发生破坏，岩体损坏后大量的高压地下水从矿洞中涌出。矿井发生严重的渗水事故，对拉甲坡矿、龙山矿和田角锌矿造成了重大影响。

![](.evernote-content/0DE2742B-F406-42C0-B931-A3F0D710BBC9/16874A71-A9EC-43BE-9A94-AF41FA1862E2.png)

想方设法瞒报

7月17日早上7点多，拉甲坡矿的矿长黎家西发现矿井下发生了严重的渗水事故，黎家西将这起涉水事故汇报给了龙泉矿业的矿主黎东明。黎东明早上7点就知道发生了矿难，可是直到下午5点，他才把这次事故汇报给了当时分管南当矿业的县委副书记莫壮龙。

81名矿工被埋在井下，迫切需要救援，黎东明却在事故发生14个小时之后才将情况汇报给了县委副书记。县委副书记莫壮龙得知发生困矿难后，向县委书记万瑞忠汇报。万瑞忠得知发生矿难后，没有第一时间安排救援，而是让下级们先了解矿难的具体状况，然后跟县长唐毓盛商量。

万瑞忠和唐毓盛知道这起事故的死亡状况特别严重，如果消息泄露，两个人的乌纱帽就保不住了。万瑞忠还在考虑，如果龙泉矿业总厂被查封，整个南丹县的财政收入将会受到影响，两个人的政绩也不好看。万瑞忠和唐毓盛商量后没有第一时间采取救援，也没有将这起事故汇报给上级领导，而是决定将这起矿难瞒报。

万瑞忠跟唐毓盛说，你先了解清楚龙泉矿有没有把握做好内部封锁的消息，如果能将消息封锁下来就不要汇报了。7月17日晚上10点多，唐毓盛给莫壮龙打电话，再次叮嘱这起事故千万不要闹大了。

此时矿难已经发生了20多个小时，上级领导充分了解到这起矿难的严重性，却没有采取救援，而是在想方设法为了保住政绩和乌纱帽，将这起事件一层层瞒报下来。

![](.evernote-content/0DE2742B-F406-42C0-B931-A3F0D710BBC9/199B9E3F-FB83-435A-A948-759617CE0B75.jpg)

层层掩埋的真相

7月18日早上，万瑞忠和唐毓盛决定和莫壮龙、韦光学碰头召开秘密会议，商量一下矿难的事情到底该怎么处理。几个人商量过后决定，这起事故由龙泉矿自己处理，只要县里不往上汇报，市级领导和省委领导就不可能知道。莫壮龙和韦光学的任务就是与龙泉矿的领导协调，将这起事件瞒报下来，防止消息泄露。

四个人还达成了“攻守联盟”，一旦上级领导知道这起重大矿难事故，四个人就一口咬定说，龙泉矿从来没有正式对这起事故进行书面汇报，只要四个人将责任推给龙泉矿的负责人黎东明，自己的乌纱帽就能保住。

7月22日晚上8点，四个人又聚在一起商量这起矿难的处理办法，莫壮龙声称龙泉矿已经在安顿死者家属，新闻媒体方面的消息也已经被封锁了。这次小会议上万瑞忠再三交代，大家一定要保持一致的口径，这起矿难千万不能泄露出去，一旦上级严查这件事，四个人就装作什么也不知道，没有在一起开过会，只要这件事情没有被抖出来，就要继续隐瞒。

7月23日，万瑞忠主持召开广西自治区南丹矿业秩序整顿通报会，跟矿业秩序工作组的副组长赵桂华说，矿山治理期间千万不要出什么事情，有人传言龙泉矿发生了大事故，你们调查出什么结果了吗？赵桂华也和几个人统一好了口径，赵桂华说：经过调查，矿井只是发生了严重透水事件，没有发现人员伤亡。

![](.evernote-content/0DE2742B-F406-42C0-B931-A3F0D710BBC9/ED835B20-2013-403F-962E-C4E4BC3D9047.jpg)

81名矿工被矿上“除名”

矿难发生以来，万瑞忠四人想方设法将这起矿难瞒报下来，几人多次向上级汇报调查结果，只说发生了严重透水事故，均不承认有人员伤亡。7月28日，南丹县传出越来越多的矿难传闻，此时81名矿工被埋在矿井下11天，尸体已经腐烂了。龙泉矿的老板黎东明却对调查人员信誓旦旦地说，我们龙泉矿一共碰到过10多次透水事故，从来都没有出现人员伤亡，这次事件中我也没有听说有人员受伤或者失踪的消息。

龙泉矿的矿难消息虽然被全面封锁，这起事故依旧引起了广西自治区政府的关注。万瑞忠等领导干部欺上瞒下，矿主黎东明更是想尽一切办法封锁消息。龙泉矿的各级负责人、班长以及工头多次召开会议，要求工作人员千万不能泄露真相，将这起矿难事故一瞒到底。

为了安抚受害者的家属，黎东明拿出400多万现金对死者进行赔偿，黎东明还找人将遇难者家属召集起来，先用金钱收买，再威胁遇难者家属，不许将死人的事情说出去。为了彻底隐藏死人的消息，龙泉矿矿工的花名册都被篡改了，死亡的81名矿工就像从来没有在矿上存在过一样。

![](.evernote-content/0DE2742B-F406-42C0-B931-A3F0D710BBC9/29BBB1A7-C848-4EDF-8ED4-05B7BA4D754D.jpg)

记者揭开真相
======

这起案件在层层瞒报下终于还是传了出去，7月22日，《南国早报》接到群众举报，龙泉矿发生了重大透水事故，多名矿工死亡。报社领导立刻派记者奔赴南丹县，对龙泉矿进行采访。记者在龙泉矿经过多方面走访和采访，得到的统一回复都是龙泉矿只发生了小规模的渗水事件，没有出现人员伤亡。

为了得到真实有效的信息，两名记者脱掉上衣，伪装成矿上干活的工人，趁机进入矿区寻找答案。两名记者进入矿区后采访依旧困难重重，矿区里遍布打手，记者和矿工说不了几句话就被人打断了。记者乔装打扮多次进入矿区，终于得到了一部分真相，况下有几十人遇难。记者又采访了几位遇难者的家属，才知道龙泉矿渗水事故的真相。

通过记者的不懈努力，这起特大矿难被曝光，时任广西自治区党委书记的曹伯纯知道这起事件，他拍桌而起，要求南丹县委负责人查明真相，活要见人，死要见尸，所有阻碍查案的人员一律严肃处理。“7.17矿难事故”不断发酵，引起了党中央和国务院领导的高度重视，有关部门组织专案组进入南丹县对矿难事故进行彻查，广西“717南丹矿难”事件的真相才被揭开。

![](.evernote-content/0DE2742B-F406-42C0-B931-A3F0D710BBC9/1F94BD1F-2ECF-4557-B191-798FE369F1FC.png)

处理结果
====

8月6日，纪检监察部门对南丹县县委书记万瑞忠，县长唐毓胜等人采取措施，在几位领导干部的交代下，龙泉矿业总经理黎东明也被捕。涉及这起案件的主要负责人万瑞忠、唐毓胜、莫壮龙、罗绍章、韦学光等人因为故意隐瞒事故，滥用职权和涉嫌受贿罪被逮捕。

经过有关部门的彻查，涉嫌瞒报的几位官员都出现了贪污受贿等重大违纪现象。南丹县原县委书记万瑞忠利用职务便利，非法收受他人财物300多万，涉嫌受贿罪。南丹县原县长唐毓盛利用职务便利，非法收受他人财物44万元，挪用公款95万元。

2002年6月5日，广西南宁市中级人民法院对“南丹717特大透水事故”进行审判，万瑞忠犯受贿罪，滥用职权罪被判处死刑，唐毓盛犯受贿罪，挪用公款罪，滥用职权罪，被判处有期徒刑20年。莫壮龙犯受贿罪，滥用职权罪被判处有期徒刑10年，韦学光犯受贿罪，滥用职权罪被判处有期徒刑13年。

龙泉矿总经理黎东明犯非法采矿罪，重大责任事故罪，妨害作证罪，单位行贿罪，被判处有期徒刑20年，处罚金20万。龙泉矿的各级负责人以及管理人员也分别被判处有期徒刑，此次案件中共有120多名涉案人员被捕。

==
