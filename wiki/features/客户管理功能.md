# 客户管理功能

所属产品：[[products/ERP系统|ERP 系统]]

涉及模块：客户公司/门市管理、合同信息、审核、应收统计、客户指标

关注岗位：销售、渠道、财务、产研

## 功能列表

| 功能 | 版本 | 说明 |
|------|:----:|------|
| 客户总产量年汇总 | v1 | 客户详情页总产量按目的地增加年汇总值，不可编辑，自动求和 |
| 广告费字段更新 | v1 | "平台使用费/入场费"更名为"平台使用费/入场费/广告费" |
| 审核页应收汇总 | v1,v4 | v1 审核页展示近2年应收数据；v4 增加目的地板块统计和指标达成率 |
| 临时合同负责人修改 | v2 | 申请临时合同时允许修改合同负责人 |
| 签约公司主体化 | v3 | 签约公司由内部组织名称变更为公司主体名称，选择错误将走错审批流 |
| 合同查询条件细化 | v4 | 合同筛选（全选/有/无）；合同公司可选择主体公司和内部业务单元 |
| 审批类型扩展 | v4 | 客户&门市单据类型增加客户公司指标和产量类型 |
