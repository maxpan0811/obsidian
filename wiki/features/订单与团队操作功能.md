# 订单与团队操作功能

所属产品：[[products/ERP系统|ERP 系统]]

涉及模块：订单管理、团处理、成本管控

关注岗位：销售、销售负责人、操作、财务经理

## 功能列表

| 功能 | 版本 | 说明 |
|------|:----:|------|
| 团成本解锁销售申请入口 | v1 | 成本锁定团显示锁图标和"成本已锁"提示，销售可提交解锁申请；财务经理审核时必选是否记罚；审批结果通过企微通知 |
| 订单管理线路ID查询 | v3 | 订单列表可按线路ID查询订单 |
| 批量下载护首扫描件 | v3 | 团队操作页勾选客人后批量下载护照首页图片 |
