# 财务审批功能

所属产品：[[products/ERP系统|ERP 系统]]

涉及模块：实付管理、集成凭证、订单应收列表、出纳审批

关注岗位：财务、出纳

## 功能列表

| 功能 | 版本 | 说明 |
|------|:----:|------|
| 实付附件/发票删除规则 | v1 | 编辑/撤回状态可删除附件和发票，其他状态仅替换。机票产品支付支持暂存 |
| 凭证勾选合计 | v1 | 同团号多单据勾选后上方显示成本和收入汇总值 |
| 出纳制单标签 | v2 | 审批页增加"已制单/未制单"标签，标记网银汇款状态 |
| 应收列表查询优化 | v2 | 销售公司改为多选；增加合同状态、下单渠道、操作字段 |
| 结算完成时间展示 | v4 | 跟团游审批页显示该团首次结算完成时间（仅26年后出团数据） |
