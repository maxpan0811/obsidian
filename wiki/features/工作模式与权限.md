# 工作模式与权限

所属产品：[[products/Claude Code|Claude Code]]

## 三种工作模式

| 模式 | 行为 | 适用场景 |
|------|------|---------|
| Default | 首次使用某类工具时弹出确认 | 刚开始使用、处理敏感文件 |
| Plan (`/plan`) | 只读分析，不动手 | 复杂任务前期规划 |
| Auto Accept | 自动接受文件编辑 | 批量处理、模板化任务 |

切换方式：`Shift+Tab` 循环切换。

## 权限确认

在 Default 模式下，Claude Code 首次执行操作时弹出确认：
- **Yes**：允许一次
- **Yes, don't ask again**：永久允许
- **No**：拒绝

三类工具的确认机制：读取文件（免确认）、Bash 命令（永久记忆）、文件修改（仅当次会话）

## 配置文件层级

| 层级 | 路径 | 作用域 |
|------|------|--------|
| 全局设置 | `~/.claude/settings.json` | 所有项目 |
| 全局本地 | `~/.claude/settings.local.json` | 自动积累的权限 |
| 项目共享 | `.claude/settings.json` | 团队共享 |
| 项目本地 | `.claude/settings.local.json` | 仅个人 |

## allow/deny 配置

```json
{
  "permissions": {
    "allow": ["Read", "Write", "Grep", "Glob"],
    "deny": ["Bash(rm *)", "Bash(curl *)"]
  }
}
```
