# 核心操作

所属产品：[[products/Claude Code|Claude Code]]

## 工作目录

在哪个文件夹启动 Claude Code，它就只能处理那个文件夹的材料。每个案件应有独立文件夹。

## 三种操作符

- `/` 调用内置命令（如 `/help`、`/clear`）
- `@` 引用文件（如 `@合同.docx`）
- `!` 执行 Bash 命令（如 `!ls`）

## 快捷键

| 操作 | 快捷键 |
|------|--------|
| 发送消息 | Enter |
| 多行输入 | `\` + Enter |
| 中断生成 | Esc |
| 查看历史 | ↑ / ↓ |
| 清屏 | Ctrl+L |
| 切换模式 | Shift+Tab |
| 粘贴图片（Mac） | Ctrl+V / Cmd+V |

## 常用命令

| 命令 | 功能 |
|------|------|
| `/help` | 查看帮助 |
| `/clear` | 清空对话 |
| `/compact` | 压缩上下文 |
| `/status` | 查看状态 |
| `/model` | 切换模型 |
| `/resume` | 恢复历史会话 |
| `/init` | 初始化 CLAUDE.md |
| `/export` | 导出对话 |
| `/plan` | 进入规划模式 |
| `/rewind` | 回退 |
| `/config` | 配置设置 |
| `/doctor` | 诊断问题 |

## 文件导出

- 直接说"保存为文件"，Claude Code 自动创建
- `/export` 导出完整对话记录
